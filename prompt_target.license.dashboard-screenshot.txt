### Navigation
| Niveau | Valeur | Position |
|--------|--------|----------|
| NAV1 | Licences | Header bleu (50px) |
| NAV2 | Tableau de bord | Sous-header blanc (40px) - actif |
| NAV3 | Général | Sidecar gauche |
| Filtre | Région | Dropdown sous NAV3 |

### Comportement
| Action | Résultat |
|--------|----------|
| Clic → ligne | Navigation vers NAV2 du type |
| Clic NAV2 tab | Navigation vers listing du type |
| Clic [Commander ▼] | Menu dropdown Commander par type |
| Change Région | Filtre les données du tableau |

### APIs
| Endpoint | Description |
|----------|-------------|
| GET /license/summary | Récapitulatif par type |
| GET /license/summary?region={region} | Filtré par région |
