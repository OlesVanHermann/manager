# 11. APIs DOMAINS
##############################################################################

### 11.1 APIs Domain
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain                               | Liste domaines        |
| GET     | /domain/{domain}                      | Details domaine       |
| GET     | /domain/{domain}/serviceInfos         | Infos facturation     |
| PUT     | /domain/{domain}                      | Modifier domaine      |
| GET     | /domain/{domain}/authInfo             | Obtenir AUTH/INFO     |

### 11.2 APIs Transfer Lock
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/rules/lock           | Etat verrou           |
| POST    | /domain/{domain}/lock                 | Activer verrou        |
| DELETE  | /domain/{domain}/lock                 | Desactiver verrou     |

### 11.3 APIs Zone DNS
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/zone                          | Liste zones           |
| GET     | /domain/zone/{zone}                   | Details zone          |
| GET     | /domain/zone/{zone}/record            | Liste records         |
| GET     | /domain/zone/{zone}/record/{id}       | Details record        |
| POST    | /domain/zone/{zone}/record            | Creer record          |
| PUT     | /domain/zone/{zone}/record/{id}       | Modifier record       |
| DELETE  | /domain/zone/{zone}/record/{id}       | Supprimer record      |
| POST    | /domain/zone/{zone}/refresh           | Rafraichir zone       |
| POST    | /domain/zone/{zone}/reset             | Reinitialiser zone    |
| GET     | /domain/zone/{zone}/export            | Exporter zone         |
| POST    | /domain/zone/{zone}/import            | Importer zone         |
| GET     | /domain/zone/{zone}/history           | Historique zone       |
| POST    | /domain/zone/{zone}/history/{id}/restore | Restaurer version  |

### 11.4 APIs DNS Servers
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/nameServer           | Liste serveurs DNS    |
| POST    | /domain/{domain}/nameServer           | Modifier serveurs     |
| GET     | /domain/{domain}/nameServer/status    | Statut serveurs       |

### 11.5 APIs DynHost
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/zone/{zone}/dynHost/record    | Liste entrees DynHost |
| GET     | /domain/zone/{zone}/dynHost/record/{id}| Details entree       |
| POST    | /domain/zone/{zone}/dynHost/record    | Creer entree          |
| PUT     | /domain/zone/{zone}/dynHost/record/{id}| Modifier IP          |
| DELETE  | /domain/zone/{zone}/dynHost/record/{id}| Supprimer entree     |
| GET     | /domain/zone/{zone}/dynHost/login     | Liste logins          |
| POST    | /domain/zone/{zone}/dynHost/login     | Creer login           |
| PUT     | /domain/zone/{zone}/dynHost/login/{login}| Modifier login     |
| POST    | /domain/zone/{zone}/dynHost/login/{login}/changePassword | Changer MDP |
| DELETE  | /domain/zone/{zone}/dynHost/login/{login}| Supprimer login    |

### 11.6 APIs GLUE
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/glueRecord           | Liste GLUE            |
| GET     | /domain/{domain}/glueRecord/{host}    | Details GLUE          |
| POST    | /domain/{domain}/glueRecord           | Creer GLUE            |
| PUT     | /domain/{domain}/glueRecord/{host}    | Modifier GLUE         |
| DELETE  | /domain/{domain}/glueRecord/{host}    | Supprimer GLUE        |

### 11.7 APIs DNSSEC
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/dsRecord             | Liste DS records      |
| POST    | /domain/{domain}/dsRecord             | Ajouter DS record     |
| DELETE  | /domain/{domain}/dsRecord/{id}        | Supprimer DS record   |
| POST    | /domain/{domain}/activateDnssec       | Activer DNSSEC        |
| POST    | /domain/{domain}/deactivateDnssec     | Desactiver DNSSEC     |

### 11.8 APIs Contacts
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/serviceInfos         | Contacts actuels      |
| GET     | /domain/{domain}/owo                  | Etat OWO              |
| POST    | /domain/{domain}/owo                  | Configurer OWO        |

### 11.9 APIs Tasks
| Methode | Endpoint                              | Description           |
|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/task                 | Taches domaine        |
| GET     | /domain/zone/{zone}/task              | Taches zone           |
| GET     | /domain/{domain}/task/{id}            | Details tache         |

### 11.10 APIs DNS Anycast

|---------|---------------------------------------|-----------------------|
| GET     | /domain/{domain}/option/dnsAnycast    | Etat Anycast          |
