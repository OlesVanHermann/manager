### 2.2 Clé de produit (SPÉCIFIQUE WINDOWS)
| Élément | Description |
|---------|-------------|
| Format | XXXXX-XXXXX-XXXXX-XXXXX-XXXXX (25 caractères, 5 groupes) |
| Affichage | Zone fond sombre (#1F2937), monospace |
| Bouton | [Copier] |
| Info | Message "Clé activée automatiquement" |

### 2.3 Carte Spécifications licence (SPÉCIFIQUE WINDOWS)
| Champ | Description |
|-------|-------------|
| Type | SPLA (Services Provider License) |
| Cores autorisés | Nombre de cores CPU licenciés |
| CALs incluses | "Illimité" pour SPLA |
| Activation | Badge "● Activée" ou "○ Non activée" |

### 2.4 Guide d'activation PowerShell (SPÉCIFIQUE WINDOWS)
NAV4 dédié avec étapes d'activation.

#### Étape 1 : Récupérer la clé
- Affichage clé de produit
- Bouton [Copier]

#### Étape 2 : Activer via PowerShell
```powershell
# Installer la clé
slmgr.vbs /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

# Activer Windows
slmgr.vbs /ato
```
- Note: "Exécuter PowerShell en tant qu'administrateur"

#### Étape 3 : Vérifier l'activation
```powershell
# Vérifier le statut
slmgr.vbs /dli
```
- Résultat attendu: "License Status: Licensed"
- Bouton [Besoin d'aide ?]
