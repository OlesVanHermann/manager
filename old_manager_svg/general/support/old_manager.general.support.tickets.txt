# Cahier des Charges : support - Liste des tickets

## 1. VUE D'ENSEMBLE
- **Page** : Mes tickets de support
- **Type** : Datagrid avec filtres
- **Objectif** : Gérer les demandes de support

## 2. STRUCTURE PAGE
- Header avec titre + bouton créer
- Filtres (statut, date)
- Datagrid tickets
- Pagination

## 3. COMPOSANTS UI
- **OdsButton** : Créer un ticket
- **OdsSelect** : Filtres statut
- **OdsDatepicker** : Filtre période
- **OdsDatagrid** : Liste tickets
- **OdsBadge** : Statut ticket
- **OdsPagination** : Navigation

## 4. HEADER
| Élément | Type | Position |
|---------|------|----------|
| Titre | H1 | Gauche |
| Créer un ticket | Primary Button | Droite |

## 5. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Statut | Select | Tous, Ouvert, En cours, Résolu, Fermé |
| Période | Datepicker | Du... au... |
| Recherche | Input | Numéro ou sujet |

## 6. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| N° Ticket | Text link | Oui | Numéro unique |
| Sujet | Text | Oui | Titre du ticket |
| Service | Text | Oui | Produit concerné |
| Statut | Badge | Oui | État du ticket |
| Priorité | Badge | Oui | Basse/Moyenne/Haute |
| Créé le | Date | Oui | Date création |
| Mis à jour | Date | Oui | Dernière activité |

## 7. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Ouvert | Bleu |
| En cours | Orange |
| En attente client | Jaune |
| Résolu | Vert |
| Fermé | Gris |

## 8. BADGES PRIORITÉ
| Priorité | Couleur |
|----------|---------|
| Basse | Gris |
| Moyenne | Orange |
| Haute | Rouge |

## 9. APIS & ENDPOINTS
- GET /support/tickets - Liste tickets
- GET /support/tickets?status={status} - Filtrer par statut
- POST /support/tickets - Créer ticket
