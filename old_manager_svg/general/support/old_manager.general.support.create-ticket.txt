# Cahier des Charges : support - Créer un ticket

## 1. VUE D'ENSEMBLE
- **Page** : Créer un ticket de support
- **Type** : Formulaire multi-étapes
- **Objectif** : Soumettre une demande de support

## 2. STRUCTURE PAGE
- Stepper (3 étapes)
- Formulaire étape courante
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsStepper** : Progression
- **OdsSelect** : Catégorie, service
- **OdsInput** : Sujet
- **OdsTextarea** : Description
- **OdsFileUpload** : Pièces jointes
- **OdsButton** : Précédent, Suivant, Envoyer

## 4. STEPPER
| Étape | Nom | Description |
|-------|-----|-------------|
| 1 | Catégorie | Choix du service |
| 2 | Détails | Description du problème |
| 3 | Confirmation | Récapitulatif |

## 5. ÉTAPE 1 - CATÉGORIE
| Champ | Type | Requis | Options |
|-------|------|--------|---------|
| Type de demande | Radio | Oui | Technique, Commercial, Administratif |
| Catégorie | Select | Oui | Cloud, Web, Réseau, Bare Metal |
| Service | Select | Oui | Liste services utilisateur |
| Urgence | Select | Non | Basse, Moyenne, Haute |

## 6. ÉTAPE 2 - DÉTAILS
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Sujet | Input | Oui | Max 100 car. |
| Description | Textarea | Oui | Min 50 car. |
| Pièces jointes | FileUpload | Non | Max 5 fichiers, 10Mo chacun |

## 7. ÉTAPE 3 - CONFIRMATION
| Section | Contenu |
|---------|---------|
| Récapitulatif | Service, Sujet, Description |
| Pièces jointes | Liste fichiers |
| Temps réponse | Estimation selon urgence |

## 8. BOUTONS
| Bouton | Position | Action |
|--------|----------|--------|
| Précédent | Gauche | Étape -1 |
| Suivant | Droite | Étape +1 |
| Envoyer | Droite (étape 3) | Soumet ticket |

## 9. APIS & ENDPOINTS
- GET /me/services - Services utilisateur
- GET /support/categories - Catégories support
- POST /support/tickets - Créer ticket
- POST /support/tickets/{id}/attachments - Ajouter PJ
