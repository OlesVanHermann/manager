## 8. APIS & ENDPOINTS

### 8.1 Obtenir le profil

```
GET /me
```

### 8.2 Modifier le profil

```
PUT /me
{
  "firstname": "Jean",
  "name": "Dupont",
  "email": "jean.dupont@email.com",
  "phone": "+33123456789",
  "address": "123 rue de la République",
  "zip": "75001",
  "city": "Paris",
  "country": "FR",
  "language": "fr_FR"
}
```

### 8.3 Valider le changement d'email

```
POST /me/task/emailChange/{id}/accept
{
  "token": "TOKEN"
}
```
