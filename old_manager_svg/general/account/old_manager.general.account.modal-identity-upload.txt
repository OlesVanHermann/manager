## 6. APIS & ENDPOINTS

### 6.1 Envoyer un document d'identité

```
POST /me/document
{
  "type": "identity",
  "file": "<base64>"
}
```

### 6.2 Vérifier le statut

```
GET /me/document?type=identity
```
