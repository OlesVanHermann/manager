## 6. APIS & ENDPOINTS

### 6.1 Ajouter un numéro SMS

```
POST /me/accessRestriction/sms
{
  "phone": "+33612345678",
  "description": "Téléphone personnel"
}
```

### 6.2 Envoyer le code de validation

```
POST /me/accessRestriction/sms/{id}/sendCode
```

### 6.3 Valider le code

```
POST /me/accessRestriction/sms/{id}/validate
{
  "code": "123456"
}
```
