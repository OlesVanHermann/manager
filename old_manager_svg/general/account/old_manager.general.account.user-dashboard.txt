## 8. APIS & ENDPOINTS

### 8.1 Récupérer le profil utilisateur

```
GET /me
```

### 8.2 Récupérer l'historique de connexions

```
GET /me/accessRestriction/ipDefaultRule/log
```

### 8.3 Récupérer le statut 2FA

```
GET /me/accessRestriction
GET /me/accessRestriction/backupCode
```
