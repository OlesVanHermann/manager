# Cahier des Charges : home/account - Historique Emails

## 1. VUE D'ENSEMBLE
- **Type** : Liste des emails reçus
- **URL** : /account/emails
- **Objectif** : Consulter l'historique des emails OVHcloud

## 2. STRUCTURE PAGE
- Sidebar avec Mon compte actif
- Breadcrumb
- Titre avec description
- Barre de recherche + refresh
- Datagrid emails
- Pagination

## 3. COMPOSANTS UI
- **OdsInput** : Recherche
- **OdsButton** : Refresh (icône ↻)
- **OdsDatagrid** : Liste emails
- **OdsPagination** : Navigation pages
- **OdsActionMenu** : Menu contextuel

## 4. ACTIONS HEADER
| Élément | Style | Action |
|---------|-------|--------|
| Recherche | Input 220px | Filtrer |
| Refresh | Icon button | Recharger |
