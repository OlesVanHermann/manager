## 7. APIS & ENDPOINTS

### 7.1 Récupérer les préférences

```
GET /me
```

### 7.2 Modifier les préférences

```
PUT /me
{
  "timezone": "Europe/Paris",
  "currency": "EUR"
}
```

### 7.3 Lister les tokens API

```
GET /me/api/application
GET /me/api/credential
```

### 7.4 Demander la suppression du compte

```
POST /me/terminate
```
