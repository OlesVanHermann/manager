# Cahier des Charges : home/account - Édition Profil

## 1. VUE D'ENSEMBLE
- **Type** : Formulaire d'édition
- **URL** : /account/profile/edit
- **Objectif** : Modifier les informations du profil client

## 2. STRUCTURE PAGE
- Breadcrumb avec lien retour
- Titre "Mon compte"
- Formulaire en 2 sections
- Boutons d'action

## 3. COMPOSANTS UI
- **OdsInput** : Champs de saisie
- **OdsSelect** : Type de compte, Pays
- **OdsButton** : Enregistrer (primary), Annuler (secondary)

## 4. SECTION 1 : IDENTIFICATION CLIENT
| Champ | Type | État |
|-------|------|------|
| Type de compte | Select | Particulier/Entreprise |
| Prénom | Input | Éditable |
| Nom | Input | Éditable |
| E-mail | Input | Éditable |

## 5. SECTION 2 : INFORMATIONS DE CONTACT
| Champ | Type | État |
|-------|------|------|
| Adresse | Input | Éditable |
| Code postal | Input | Éditable (180px) |
| Ville | Input | Éditable (180px) |
| Pays | Select | Non modifiable (grisé) |
| Téléphone | Input | Éditable |

## 6. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Enregistrer | Primary bleu | Sauvegarder |
| Annuler | Secondary | Retour |

## 7. CHAMPS DIMENSIONS
| Élément | Largeur |
|---------|---------|
| Input standard | 400px |
| Code postal | 180px |
| Ville | 180px |
| Bouton Enregistrer | 180px |
| Bouton Annuler | 120px |

## 8. APIS & ENDPOINTS
- GET /me - Charger profil
- PUT /me - Mettre à jour profil
