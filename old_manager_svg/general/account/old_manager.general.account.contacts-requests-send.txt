## 9. APIS & ENDPOINTS

### 9.1 Lister les demandes envoyées

```
GET /me/task/contactChange?toAccount=false
```

### 9.2 Obtenir le détail d'une demande

```
GET /me/task/contactChange/{id}
```

### 9.3 Annuler une demande (si en attente)

```
POST /me/task/contactChange/{id}/abort
```
