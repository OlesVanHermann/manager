# Cahier des Charges : home/account - Informations personnelles

## 1. VUE D'ENSEMBLE
- **Type** : Page complète avec formulaire
- **URL** : /account/user-infos
- **Objectif** : Modifier les informations personnelles du compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 900px |

## 3. LAYOUT
| Zone | Largeur | Contenu |
|------|---------|---------|
| Formulaire principal | 700px | Informations éditables |
| Sidebar droite | 380px | Infos compte (lecture seule) |

## 4. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte › Informations personnelles
- **OdsCard** : Containers (formulaire + sidebar)
- **OdsInput** : Champs texte
- **OdsSelect** : Dropdowns (Civilité, Pays, Langue)
- **OdsMessage** : Alerte changement email (bleu)
- **OdsBadge** : Niveau support
- **OdsButton** : Enregistrer, Annuler

## 5. ALERTE EMAIL
| Type | Contenu |

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Enregistrer | Primary | Bas gauche | Sauvegarder modifications |
| Annuler | Secondary | À droite de Enregistrer | Annuler modifications |

## 9. VALIDATION
| Champ | Règles |
|-------|--------|
| Email | Format email valide |
| Téléphone | Format international (+XX) |
| Code postal | Format selon pays |
| Champs requis (*) | Non vides |

## 10. APIS & ENDPOINTS
- GET /me - Récupérer infos utilisateur
- PUT /me - Mettre à jour infos utilisateur
- POST /me/changeEmail - Demander changement email
- POST /me/confirmEmail - Confirmer avec code
