## 8. APIS & ENDPOINTS

### 8.1 Lister les services

```
GET /me/api/credentials
```

### 8.2 Obtenir les contacts d'un service

```
GET /service/{serviceName}/serviceInfos
```

### 8.3 Modifier les contacts

```
POST /service/{serviceName}/changeContact
{
  "contactAdmin": "nic-handle",
  "contactTech": "nic-handle",
  "contactBilling": "nic-handle"
}
```
