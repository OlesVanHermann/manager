# Cahier des Charges : home/account - Paramètres Avancés

## 1. VUE D'ENSEMBLE
- **Type** : Page paramètres avancés
- **URL** : /account/advanced
- **Objectif** : Gérer SSH, API, restrictions, sessions

## 2. STRUCTURE PAGE
- Tabs avec Paramètres avancés actif
- 4 cartes de fonctionnalités
- Zone danger (suppression compte)

## 3. COMPOSANTS UI
- **OdsCard** : Cartes fonctionnalités
- **OdsButton** : Actions
- **OdsMessage** : Zone danger (rouge)
