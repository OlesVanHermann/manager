## 6. APIS & ENDPOINTS

### 6.1 Vérifier l'éligibilité au paiement fractionné

```
GET /me/order/{orderId}/splitPayment/availability
```

### 6.2 Obtenir les options de fractionnement

```
GET /me/order/{orderId}/splitPayment/options
```

### 6.3 Activer le paiement fractionné

```
POST /me/order/{orderId}/splitPayment
{
  "installments": 4
}
```

### 6.4 Obtenir l'échéancier prévisionnel

```
GET /me/order/{orderId}/splitPayment/schedule?installments=4
```
