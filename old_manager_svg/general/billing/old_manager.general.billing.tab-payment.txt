# Cahier des Charges : home/billing - Moyens de Paiement

## 1. VUE D'ENSEMBLE
- **Type** : Gestion moyens de paiement
- **URL** : /billing/payment-methods
- **Objectif** : Gérer les moyens de paiement et compte prépayé

## 2. STRUCTURE PAGE
- Sidebar avec Mes moyens de paiement actif
- Tabs (Moyens de paiement, Crédits & bons, Fidélité, Compte prépayé)
- Bouton Ajouter
- Cartes moyens de paiement
- Section compte prépayé

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation sections
- **OdsButton** : Ajouter un moyen de paiement
- **OdsCard** : Cartes paiement
- **OdsBadge** : "Par défaut" (vert)
- **OdsLink** : Modifier, Supprimer, Définir par défaut

## 4. CARTES MOYENS DE PAIEMENT
| Carte | Header | Contenu |
|-------|--------|---------|
| Carte bancaire | Bleu #1E40AF | Type, N° masqué, Expiration, Titulaire |
| SEPA | Gris #6B7280 | IBAN masqué, Banque, Titulaire |
| Ajouter | Dashed | Zone empty state |

## 5. STRUCTURE CARTE
| Élément | Position |
|---------|----------|
| Dimensions | 350x200px |
| Header | 50px, couleur par type |
| Titre | Icône + Type |
| Badge par défaut | Header droite (si défaut) |
| N° masqué | Corps |
| Détails | Corps |
| Actions | Footer (liens) |

## 6. SECTION COMPTE PRÉPAYÉ
| Élément | Contenu |
|---------|---------|
| Titre | "Compte prépayé OVH" |
| Solde | 0,00 € |
| CTA | "Créditer mon compte" (outline) |

## 7. APIS & ENDPOINTS
- GET /me/paymentMethod - Liste moyens
- POST /me/paymentMethod - Ajouter
- DELETE /me/paymentMethod/{id} - Supprimer
- PUT /me/paymentMethod/{id}/default - Définir défaut
- GET /me/ovhAccount - Compte prépayé
