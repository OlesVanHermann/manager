# Cahier des Charges : billing - Dashboard complet

## 1. VUE D'ENSEMBLE
- **Page** : Tableau de bord facturation détaillé
- **Type** : Dashboard multi-sections
- **Objectif** : Vue complète facturation avec widgets

## 2. STRUCTURE PAGE
- Header avec titre
- Cartes statistiques (4 colonnes)
- Section dernières factures
- Section moyens de paiement
- Section prochains renouvellements

## 3. COMPOSANTS UI
- **OdsCard** : Tuiles statistiques
- **OdsDatagrid** : Tableaux
- **OdsBadge** : Statuts
- **OdsLink** : Navigation
- **OdsPagination** : Pagination
