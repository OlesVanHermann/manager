# Cahier des Charges : billing - Historique des factures

## 1. VUE D'ENSEMBLE
- **Page** : Historique des factures
- **Type** : Datagrid avec filtres date
- **Objectif** : Consulter l'historique complet des factures

## 2. STRUCTURE PAGE
- Header avec titre
- Filtres (période, statut)
- Bouton export
- Tableau factures
- Pagination

## 3. COMPOSANTS UI
- **OdsDatepicker** : Période (du/au)
- **OdsSelect** : Filtre statut
- **OdsButton** : Exporter CSV
- **OdsDatagrid** : Tableau factures
- **OdsBadge** : Statut facture
- **OdsPagination** : Navigation pages

## 4. FILTRES PÉRIODE
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Du | Datepicker | 01/01/2025 |
| Au | Datepicker | 27/12/2025 |

## 5. FILTRE STATUT
| Option | Description |
|--------|-------------|
| Tous | Toutes les factures |
| Payée | Factures réglées |
| En attente | En cours de paiement |
| Impayée | Factures non réglées |

## 6. COLONNES DATAGRID
| Colonne | Type | Tri |
|---------|------|-----|
| N° Facture | Link | Oui |
| Date | Date | Oui |
| Référence | Text | Oui |
| Montant HT | Montant | Oui |
| Montant TTC | Montant bold | Oui |
| Statut | Badge | Oui |
| PDF | Icon link | Non |

## 7. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Payée | Vert |
| En attente | Orange |
| Impayée | Rouge |

## 8. PAGINATION
| Élément | Valeur |
|---------|--------|
| Info | Affichage de 1 à 6 sur 48 factures |
| Navigation | < 1 2 > |

## 9. APIS & ENDPOINTS
- GET /me/bill - Liste factures
- GET /me/bill?date.from={}&date.to={} - Filtrer période
- GET /me/bill?status={status} - Filtrer statut
- GET /me/bill/export - Export CSV
- GET /me/bill/{id}/download - Télécharger PDF
