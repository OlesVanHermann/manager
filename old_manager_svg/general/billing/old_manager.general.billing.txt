# Cahier des Charges : billing - Vue d'ensemble facturation

## 1. VUE D'ENSEMBLE
- **Page** : Tableau de bord facturation
- **Type** : Dashboard avec tuiles et tableau
- **Objectif** : Vue globale de la facturation du compte

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar navigation
- Onglets (Vue d'ensemble, Factures, Services, Paiements)
- Tuiles statistiques
- Tableau dernières factures

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tuiles statistiques
- **OdsDatagrid** : Tableau factures
- **OdsBadge** : Statut facture
- **OdsButton** : Actions

## 4. TUILES STATISTIQUES
| Tuile | Contenu | Action |
|-------|---------|--------|
| Solde du compte | 0,00 € | Ajouter du crédit |
| Prochaine facture | 124,80 € estimée | Voir le détail estimé |
| Moyen de paiement | Visa **** 4242 | Gérer |

## 5. TABLEAU DERNIÈRES FACTURES
| Colonne | Type | Tri |
|---------|------|-----|
| Référence | Link | Oui |
| Date | Date | Oui |
| Montant HT | Montant | Oui |
| Montant TTC | Montant | Oui |
| État | Badge | Oui |
| Actions | Link | Non |

## 6. BADGES ÉTAT
| État | Couleur |
|------|---------|
| Payée | Vert |
| En attente | Orange |
| Impayée | Rouge |

## 7. APIS & ENDPOINTS
- GET /me/bill - Liste factures
- GET /me/balance - Solde compte
- GET /me/paymentMethod - Moyens de paiement
- GET /me/bill/estimate - Estimation prochaine facture
