## 6. APIS & ENDPOINTS

### 6.1 Lister les domaines du pack AllDom

```
GET /allDom/{serviceName}/domains
```

### 6.2 Désactiver le renouvellement en masse

```
PUT /allDom/{serviceName}
{
  "renew": {
    "automatic": false
  }
}
```
