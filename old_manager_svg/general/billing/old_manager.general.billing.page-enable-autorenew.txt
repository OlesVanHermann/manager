# Cahier des Charges : billing - Page activer le renouvellement automatique

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **Contexte** : Facturation
- **Objectif** : Activer le renouvellement automatique pour plusieurs services

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 900px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, informations
- **OdsMessage** : Bannière d'information moyen de paiement
- **OdsDatagrid** : Liste des services
- **OdsCheckbox** : Sélection des services
- **OdsButton** : Annuler, Activer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le renouvellement automatique" |

## 5. BANNIÈRE MOYEN DE PAIEMENT
| Type | Contenu |
|------|---------|
| Info (bleu) | "Moyen de paiement : Carte bancaire •••• 1234" |
| | "Ce moyen de paiement sera utilisé par défaut pour les renouvellements automatiques." |
| | "La facturation s'effectue selon la fréquence de renouvellement de chaque service." |

## 6. TABLEAU DES SERVICES

### Colonnes
| Colonne | Description |
|---------|-------------|
| ☑ | Checkbox de sélection |
| SERVICE | Nom du service |
| DATE D'EXPIRATION | Date de fin |
| FRÉQUENCE | Mensuel, Annuel, etc. |

### Exemples de données
| ☑ | Service | Expiration | Fréquence |
|---|---------|------------|-----------|
| ✓ | ns123456.ip-1-2-3.eu | 15/01/2025 | Mensuel |
| ✓ | mydomain.com | 01/03/2025 | Annuel |
| ☐ | vps-abc123 | 20/02/2025 | Mensuel |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Activer | Primary (bleu) | Active le renouvellement auto |

## 8. APIS & ENDPOINTS
- GET /me/service - Liste des services
- PUT /me/service/{serviceId}/autorenew - Activer/désactiver le renouvellement
- GET /me/paymentMean - Moyens de paiement
