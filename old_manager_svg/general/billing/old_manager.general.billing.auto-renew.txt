# Cahier des Charges : billing - Renouvellement automatique

## 1. VUE D'ENSEMBLE
- **Page** : Renouvellement automatique
- **Type** : Datagrid avec toggles
- **Objectif** : Gérer le renouvellement automatique des services

## 2. STRUCTURE PAGE
- Header avec titre et description
- Filtres (type, statut)
- Tableau services avec toggles
- Pagination

## 3. COMPOSANTS UI
- **OdsSelect** : Filtres
- **OdsDatagrid** : Tableau services
- **OdsBadge** : Statut renouvellement
- **OdsToggle** : Activation/désactivation

## 4. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Type | Select | Hébergement, Domaine, VPS, Email... |
| Statut | Select | Automatique, Manuel, Désactivé |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Service | Link | Nom du service |
| Type | Text | Hébergement Web, Domaine, VPS |
| Expiration | Date | Date d'expiration |
| Renouvellement | Text | Mensuel, Annuel |
| Statut | Badge | Automatique/Manuel/Désactivé |
| Action | Toggle | On/Off |

## 6. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Automatique | Vert |
| Manuel | Orange |
| Désactivé | Rouge |

## 7. TOGGLE COMPORTEMENT
| État | Position | Action |
|------|----------|--------|
| On | Droite (bleu) | Renouvellement auto activé |
| Off | Gauche (gris) | Renouvellement manuel |

## 8. APIS & ENDPOINTS
- GET /billing/autoRenew - Liste services
- PUT /billing/autoRenew/{serviceId} - Modifier renouvellement
- GET /service/{type}/{serviceName} - Détail service
