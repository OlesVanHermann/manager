## 6. APIS & ENDPOINTS

### 6.1 Obtenir le solde fidélité

```
GET /me/fidelityAccount
```

### 6.2 Créditer des points

```
POST /me/fidelityAccount/creditOrder
{
  "amount": 1000
}
```

### 6.3 Obtenir les options de crédit

```
GET /me/fidelityAccount/creditOptions
```
