## 11. APIS & ENDPOINTS

### 11.1 Obtenir les options d'engagement

```
GET /services/{serviceId}/billing/engagement
```

### 11.2 Souscrire un engagement

```
POST /services/{serviceId}/billing/engagement
{
  "duration": 24,
  "paymentMethod": "monthly|upfront"
}
```
