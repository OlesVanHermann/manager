# Cahier des Charges : home - Hub Widget Facturation

## 1. VUE D'ENSEMBLE
- **Widget** : Montant total des factures
- **Type** : Carte de résumé
- **Objectif** : Afficher le montant facturé

## 2. STRUCTURE WIDGET
- Fond bleu primaire (#0050D7)
- Titre blanc
- Sélecteur de période
- Montant en grand format
- Indicateur de statut de paiement
- Lien vers factures

## 3. COMPOSANTS UI
- **OdsCard** : Container fond bleu
- **OdsSelect** : Sélecteur période
- **OdsText** : Montant, titre
- **OdsLink** : Voir les factures

## 4. SÉLECTEUR PÉRIODE
| Option | Description |
|--------|-------------|
| Le mois dernier | Par défaut |
| Ce mois-ci | Mois en cours |
| Les 3 derniers mois | Trimestre |
| Cette année | Année courante |

## 5. CONTENU
| Élément | Style |
|---------|-------|
| Titre | "Montant total des factures" |
| Période | Dropdown fond semi-transparent |
| Montant | 36px, bold, blanc |
| Indicateur | Cercle vert + "Payé" ou "En attente" |
| Lien | Underline "Voir les factures →" |

## 6. INDICATEURS STATUT
| Statut | Couleur cercle | Label |
|--------|----------------|-------|
| Payé | Vert (#10B981) | Payé |
| En attente | Orange (#F59E0B) | En attente |
| Impayé | Rouge (#EF4444) | Impayé |

## 7. APIS & ENDPOINTS
- GET /billing/invoices?period={period} - Montant par période
