# Cahier des Charges : signup - Paramètres du Compte

## 1. VUE D'ENSEMBLE
- **Page** : Paramètres initiaux du compte
- **Type** : Étape wizard (1/3)
- **Objectif** : Choisir pays, devise et langue

## 2. STRUCTURE PAGE
- Header simple avec logo OVHcloud
- Indicateur d'étape "Étape 1 sur 3"
- Carte centrale (600x620px)
- Footer RGPD

## 3. COMPOSANTS UI
- **OdsCard** : Container formulaire
- **OdsSelect** : 3 sélecteurs
- **OdsMessage** : Zone explicative
- **OdsButton** : Valider
- **OdsLink** : Connexion existante
