# Cahier des Charges : hub - IAM Tag Manager

## 1. VUE D'ENSEMBLE
- **Page** : Tag Manager
- **Type** : Liste de tags avec filtres
- **Objectif** : Gérer les tags pour catégoriser les ressources

## 2. STRUCTURE PAGE
- Sidebar avec navigation Hub
- Header avec titre et description
- Filtres rapides (checkboxes)
- Bouton action principal
- Datagrid des tags

## 3. COMPOSANTS UI
- **OdsDatagrid** : Tableau des tags
- **OdsButton** : Assigner des tags
- **OdsCheckbox** : Filtres rapides
- **OdsBadge** : Types de tags
- **OdsActionMenu** : Actions par ligne

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Titre | Tag manager |
| Description ligne 1 | Créez des tags en vous basant sur des couples clé:valeur pour catégoriser |
| Description ligne 2 | vos ressources OVHcloud et appliquer des politiques |

## 5. FILTRES RAPIDES
| Checkbox | Description |
|----------|-------------|
| Afficher les tags système | Tags OVH automatiques |
| Afficher les tags non assignés | Tags sans ressources |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Tag | Link | Format clé:valeur (ex: environment:production) |
| Ressources assignées | Number | Compteur de ressources |
| Type de Tag | Badge | Custom / Prédéfini / System |
| Actions | ActionMenu | ⋮ |

## 7. TYPES DE TAGS (Badges)
| Type | Couleur fond | Couleur texte |
|------|--------------|---------------|
| Tag custom | Bleu clair (#DBEAFE) | Bleu foncé (#1E40AF) |
| Tag prédéfini | Indigo clair (#E0E7FF) | Indigo (#3730A3) |
| Tag system | Gris (#F3F4F6) | Gris foncé (#374151) |

## 8. ACTION MENU
| Action | Description |
|--------|-------------|
| Assigner à des ressources | Ouvre modal assignation |
| Désassigner le tag | Ouvre modal confirmation |

## 9. PAGINATION
- Format : "Affichés X - Y sur Z éléments"

## 10. APIS & ENDPOINTS
- GET /iam/tag - Liste des tags
- GET /iam/tag/{tagId}/resource - Ressources d'un tag
- POST /iam/tag - Créer un tag
- DELETE /iam/tag/{tagId} - Supprimer un tag
