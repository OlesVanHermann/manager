# Cahier des Charges : home/hub - Bannière Email Injoignable

## 1. VUE D'ENSEMBLE
- **Type** : Bannière warning pleine largeur
- **Déclencheur** : Emails bounced détectés
- **Objectif** : Alerter sur problème réception emails

## 2. STRUCTURE BANNIÈRE
- Icône warning gauche
- Message multi-lignes
- Lien action intégré

## 3. COMPOSANTS UI
- **OdsMessage** : Conteneur warning
- **OdsIcon** : Triangle warning
- **OdsLink** : Lien inline

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% (1200px max) |
| Hauteur | 120px |
| Border radius | 8px |
| Padding | 24px |

## 5. STYLE WARNING
| Propriété | Valeur |
|-----------|--------|
| Background | #FEF3C7 |
| Border | 1px #F59E0B |
| Icône couleur | #F59E0B |

## 6. ICÔNE
| Propriété | Valeur |
|-----------|--------|
| Type | Triangle warning |
| Position | Gauche, centré vertical |
| Taille | 40px |
| Opacity fond | 0.3 |

## 7. MESSAGE TEXTE
| Ligne | Contenu |
|-------|---------|
| 1 | "Il semble que nos derniers e-mails ne soient pas arrivés dans votre boîte de réception." |
| 2 | "Vous pouvez les consulter dans votre espace client, rubrique « Mon compte », onglet « E-mails reçus »." |
| 3 | "Nous vous invitons également à vérifier l'[adresse e-mail de contact de votre compte.](#)" |

## 8. LIEN ACTION
| Label | Destination |
|-------|-------------|
| "adresse e-mail de contact de votre compte" | /account/infos |

## 9. CONDITION AFFICHAGE
- Dernier email envoyé bounced
- Flag me.emailBounced = true

## 10. ACTIONS MASQUAGE
- Mise à jour email réussie
- Consultation onglet emails

## 11. APIS & ENDPOINTS
- GET /me/emailBounced - Statut bounce
- GET /me/email - Liste emails envoyés
- PUT /me - Mettre à jour email contact
