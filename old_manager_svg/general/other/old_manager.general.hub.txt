# Cahier des Charges : home - Hub Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard principal OVHcloud
- **Type** : Hub de vue d'ensemble
- **Objectif** : Vue consolidée de l'activité client

## 2. STRUCTURE PAGE
- Header avec bienvenue personnalisé (nom client)
- Zone de confiance indiquée
- Section "Vue d'ensemble de mon activité"
- 4 widgets principaux en grille 2x2
- Section "Mes produits" avec cartes

## 3. COMPOSANTS UI
- **OdsCard** : Widgets et cartes produits
- **OdsBadge** : Statuts (Actif, À renouveler)
- **OdsChip** : Compteur tickets
- **OdsProgress** : Progression commande
- **OdsText** : Montants, labels
- **OdsLink** : Navigation

## 4. WIDGET STATUTS PAIEMENT
| Colonne | Type | Description |
|---------|------|-------------|
| Service | Text | Nom du service |
| Statut | Badge | Actif (vert) / À renouveler (orange) |
| Échéance | Date | Date limite paiement |

### Lien header
- "Voir tout" → /billing/services

## 5. WIDGET FACTURATION
| Info | Type |
|------|------|
| Titre | Montant total des factures |
| Sélecteur période | Le mois dernier (dropdown) |
| Montant | Grande valeur (ex: 1 234,56 EUR) |
| Statut | Cercle vert + "Payé" |
| Lien | Voir les factures |

## 9. APIS & ENDPOINTS
- GET /me - Infos compte (nom client)
- GET /services - Liste services
- GET /billing/invoices - Factures
- GET /support/tickets - Tickets support
- GET /order - Commandes en cours
