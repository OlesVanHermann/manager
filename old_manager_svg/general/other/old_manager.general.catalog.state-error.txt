# Cahier des Charges : home/catalog - État Erreur

## 1. VUE D'ENSEMBLE
- **Type** : Error state
- **URL** : /catalog (en cas d'erreur API)
- **Objectif** : Afficher une erreur de chargement

## 2. STRUCTURE PAGE
- Page centrée
- Icône erreur
- Message
- Bouton réessayer
- Lien support

## 3. COMPOSANTS UI
- **OdsEmptyState** : Conteneur erreur
- **OdsIcon** : Triangle warning rouge
- **OdsText** : Messages d'erreur
- **OdsButton** : Réessayer
- **OdsLink** : Contacter le support

## 4. CONTENU ERREUR
| Élément | Valeur |
|---------|--------|
| Icône | Triangle ! (fond rouge clair, icône rouge) |
| Titre | "Une erreur est survenue" |
| L1 | "Impossible de charger le catalogue." |
| L2 | "Veuillez réessayer ultérieurement." |

## 5. ACTIONS
| Élément | Style | Action |
|---------|-------|--------|
| Réessayer | Primary bleu 140x44 | Reload page |
| Contacter le support → | Link bleu | /support |

## 6. STYLE ICÔNE
| Propriété | Valeur |
|-----------|--------|
| Cercle fond | #FEE2E2 |
| Triangle | #EF4444 |
| Point d'exclamation | Blanc |
