# Cahier des Charges : home/procedures - Modal Session Warning

## 1. VUE D'ENSEMBLE
- **Type** : Modal avertissement session
- **URL** : N/A (overlay automatique)
- **Objectif** : Prévenir expiration imminente de session

## 2. STRUCTURE MODAL
- Overlay semi-transparent
- Carte modale compacte
- Icône timer orange
- Message warning
- Bouton OK
