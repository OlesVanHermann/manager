# Cahier des Charges : home - Hub Widget Support

## 1. VUE D'ENSEMBLE
- **Widget** : Dernières demandes d'assistance
- **Type** : Liste de tickets
- **Objectif** : Suivi rapide des tickets support

## 2. STRUCTURE WIDGET
- Carte blanche avec bordure
- Header avec titre, compteur et actions
- Liste de tickets avec lignes alternées

## 3. COMPOSANTS UI
- **OdsCard** : Container
- **OdsChip** : Compteur tickets ouverts
- **OdsBadge** : Statut ticket
- **OdsLink** : Actions header

## 4. HEADER
| Élément | Type |
|---------|------|
| Titre | "Dernières demandes d'assistance" |
| Compteur | Chip bleu (nombre ouverts) |
| Recharger | Link action |
| Voir tout | Link navigation |

## 5. COLONNES TICKETS
| Colonne | Type | Description |
|---------|------|-------------|
| N° Ticket | Link bold | #12345678 |
| Sujet | Text muted | Description courte |
| Date/Heure | DateTime | 27/12/2024 14:30 |
| Statut | Badge | Ouvert / Fermé |

## 6. BADGES STATUT
| Statut | Couleur | Background |
|--------|---------|------------|
| Ouvert | Vert | #DEF7EC |
| Fermé | Gris | #F3F4F6 |

## 7. EXEMPLES DONNÉES
| Ticket | Sujet | Statut |
|--------|-------|--------|
| #12345678 | Problème de connexion VPS | Ouvert |
| #12345677 | Question facturation mensuelle | Fermé |
| #12345676 | Demande de migration serveur | Ouvert |

## 8. ACTIONS
- **Recharger** : Rafraîchir la liste
- **Voir tout** : Navigation vers /support/tickets

## 9. APIS & ENDPOINTS
- GET /support/tickets?status=open - Tickets ouverts
- GET /support/tickets?limit=3 - 3 derniers tickets
