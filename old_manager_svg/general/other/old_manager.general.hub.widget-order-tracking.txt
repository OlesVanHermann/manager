# Cahier des Charges : home - Hub Widget Suivi Commande

## 1. VUE D'ENSEMBLE
- **Widget** : Suivi de commande
- **Type** : Carte de progression
- **Objectif** : Suivre l'avancement d'une commande

## 2. STRUCTURE WIDGET
- Carte blanche avec bordure
- Titre "Suivi de commande"
- Infos commande
- Barre de progression
- Statut et estimation

## 3. COMPOSANTS UI
- **OdsCard** : Container
- **OdsProgress** : Barre de progression
- **OdsBadge** : Statut livraison
- **OdsText** : Infos commande

## 4. INFOS COMMANDE
| Info | Type |
|------|------|
| N° Commande | Text bold (#98765432) |
| Produit | Text (VPS Starter SSD 1) |
| Date commande | Date (Commandée le 27/12/2024) |

## 5. BARRE DE PROGRESSION
| Propriété | Valeur |
|-----------|--------|
| Style | Barre horizontale |
| Fond | Gris (#E5E7EB) |
| Remplissage | Bleu primaire |
| Pourcentage | Affiché en texte (75%) |

## 6. STATUTS COMMANDE
| Statut | Badge | Description |
|--------|-------|-------------|
| Validée | Bleu | Commande validée |
| En préparation | Bleu | Préparation en cours |
| Livraison en cours | Bleu | Déploiement en cours |
| Livrée | Vert | Terminé |

## 7. ESTIMATION
- Format : "Estimation: DD/MM/YYYY HH:MM"
- Style : Text muted, petite taille

## 8. APIS & ENDPOINTS
- GET /order?status=inProgress - Commandes en cours
- GET /order/{orderId} - Détail commande
