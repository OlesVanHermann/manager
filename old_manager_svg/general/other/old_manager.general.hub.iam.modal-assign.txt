# Cahier des Charges : hub - IAM Modal Assigner Tag

## 1. VUE D'ENSEMBLE
- **Modal** : Assigner un tag à des ressources
- **Type** : Sélection multiple
- **Objectif** : Ajouter un tag à plusieurs ressources

## 2. STRUCTURE MODAL
- Lien retour vers détail tag
- Titre avec nom du tag
- Badge tag en cours d'assignation
- Description
- Datagrid ressources sélectionnables
- Compteur sélection
- Boutons footer

## 3. COMPOSANTS UI
- **OdsModal** : Container (ou page pleine)
- **OdsLink** : Retour navigation
- **OdsBadge** : Tag à assigner
- **OdsDatagrid** : Sélection ressources
- **OdsCheckbox** : Sélection lignes
- **OdsButton** : Annuler, Assigner

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Lien retour | "← Retour au détail du tag" |
| Titre | "Assigner '[tag]' à des ressources" |

## 5. SECTION TAG
| Élément | Style |
|---------|-------|
| Label | "Tags en cours d'assignation :" |
| Badge | Tag format clé:valeur (bleu) |

## 6. SECTION RESSOURCES
| Élément | Valeur |
|---------|--------|
| Titre | "Ressources en cours d'assignation :" |
| Description | "Sélectionner le/les ressource(s) que vous souhaitez tagguer dans la liste ci-dessous" |

## 7. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Checkbox | Checkbox | Sélection |
| Nom | Link | Nom ressource |
| Type de ressource | Text | Type (dedicatedServer, vps...) |
| Tags | Badges ou "-" | Tags existants |

## 8. COMPTEUR SÉLECTION
- Format : "X ressource(s) sélectionnée(s)"
- Position : Sous le datagrid

## 9. BOUTONS FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Default (outline) | Fermer modal |
| Assigner le tag | Primary | Confirmer assignation |

## 10. ÉTATS SÉLECTION
| État | Style |
|------|-------|
| Non sélectionné | Fond blanc |
| Sélectionné | Fond bleu clair (#E8F0FE), checkbox cochée |

## 11. APIS & ENDPOINTS
- GET /iam/resource - Liste toutes les ressources
- POST /iam/tag/{tagKey}/{tagValue}/resource - Assigner tag
- Body : { resourceIds: [...] }
