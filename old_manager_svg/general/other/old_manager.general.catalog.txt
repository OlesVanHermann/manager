# Cahier des Charges : home/catalog - Catalogue Produits

## 1. VUE D'ENSEMBLE
- **Page** : Catalogue produits OVHcloud
- **URL** : /catalog
- **Objectif** : Explorer et découvrir les produits OVHcloud

## 2. STRUCTURE PAGE
- Pas de sidebar (pleine page)
- Breadcrumb
- Titre avec compteur
- Barre de recherche + filtres
- Chips filtres actifs
- Grille de produits (3 colonnes)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsInput** : Recherche produit
- **OdsSelect** : Filtres Univers, Catégorie
- **OdsChip** : Filtres actifs (removable)
- **OdsCard** : Cartes produits

## 4. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Univers | Select | Public Cloud, Bare Metal, Web Cloud... |
| Catégorie | Select | Compute, Storage, Network... |

## 5. CHIPS FILTRES ACTIFS
| État | Style |
|------|-------|
| Actif | Fond bleu #0050D7, texte blanc |
| Normal | Fond bleu clair #E8F0FE, texte bleu |
| Action | ✕ pour supprimer |

## 6. GRILLE PRODUITS

### Structure carte
| Élément | Style |
|---------|-------|
| Dimensions | 420x140px |
| Titre | 16px bleu bold |
| Catégorie | 12px gris muted |
| Description | 13px, 2 lignes |
| Lien | "En savoir plus →" bleu |

### Produits exemple
| Produit | Catégorie |
|---------|-----------|
| Instances Public Cloud | Compute |
| Managed Kubernetes Service | Containers & Orchestration |
| Object Storage | Storage |
| Managed Databases | Databases |
| Load Balancer | Network |
| AI Training | AI & Machine Learning |
| Private Registry | Containers & Orchestration |
| Block Storage | Storage |
| Data Processing | Big Data |

## 7. APIS & ENDPOINTS
- GET /products - Liste catalogue
- GET /products?universe=xxx - Filtre par univers
- GET /products?category=xxx - Filtre par catégorie
