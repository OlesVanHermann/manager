# Cahier des Charges : home/procedures - Modal Session expirée

## 1. VUE D'ENSEMBLE
- **Page** : Modal session expirée
- **Type** : Modal warning
- **Objectif** : Informer que la session a expiré et rediriger vers la connexion

## 2. STRUCTURE MODAL
- Icône warning (!)
- Titre
- Message explicatif
- Bouton reconnexion

## 3. COMPOSANTS UI
- **OdsModal** : Container modal
- **OdsIcon** : Warning (!)
- **OdsText** : Titre, messages
- **OdsButton** : Se reconnecter

## 4. ICÔNE WARNING
| Propriété | Valeur |
|-----------|--------|
| Cercle externe | Jaune clair (#FEF3C7) |
| Cercle interne | Orange (#F59E0B) |
| Icône | ! blanc |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Titre | Session expirée |
| Message | Votre session a expiré pour des raisons de sécurité. |
| Sous-message | Veuillez vous reconnecter pour continuer. |

## 6. BOUTON
| Bouton | Style | Action |
|--------|-------|--------|
| Se reconnecter | Primary | Redirection login |

## 7. COMPORTEMENT
| Propriété | Valeur |
|-----------|--------|
| Fermeture X | Non (pas de croix) |
| Click outside | Non |
| Échap | Non |
| Auto-redirect | Après 30s |

## 8. APIS & ENDPOINTS
- Aucun (redirection vers /auth/login)
