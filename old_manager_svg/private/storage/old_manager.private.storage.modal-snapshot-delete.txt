# Cahier des Charges : private/storage - Modal Snapshot Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation destructive
**Fonction:** Supprimer un snapshot NAS-HA
**Contexte:** Suppression définitive d'un point de restauration

## 2. DIMENSIONS

- Modal: 500x340px
- Background page: #F9FAFB avec overlay rgba(0,0,0,0.4)

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #FEE2E2 (danger light)
- Titre: "Supprimer le snapshot"
- Couleur titre: #DC2626
- Bouton fermer: × (rouge)
- Height: 60px

## 4. CONTENU

### Texte introduction
- "Vous êtes sur le point de supprimer le snapshot suivant :"

### Info box snapshot
- Background: #F9FAFB
- Border-radius: 4px
- Label: "Nom du snapshot" (text-muted, 12px)
- Valeur: nom complet (bold) ex: "snap-backup-2024-01-15"
- Dimensions: 450x50px

### Warning box
- Background: #FEF3C7
- Border-left: 4px solid #F59E0B
- Titre: "Attention" (bold, #92400E)
- Texte: "Cette action est irréversible. Toutes les données contenues dans ce snapshot seront perdues."
- Dimensions: 450x70px

## 5. COMPOSANTS UI

### Info box
- Padding: 15px
- Label: font-size 12px, text-muted
- Value: font-size 14px, font-weight 600

### Warning box
- Border-radius: 4px
- Titre: font-weight 600
- Texte: font-size 13px

## 6. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Border: #E5E7EB
- Action: Ferme le modal

### Bouton Supprimer
- Style: btn-danger (110x36px)
- Background: #DC2626
- Action: Supprime le snapshot

### Conséquences
- Suppression définitive du snapshot
- Perte de toutes les données du point de restauration
- Action irréversible

## 7. APIS & ENDPOINTS

```
GET /dedicated/nasha/{serviceName}/partition/{partitionName}/snapshot/{snapshotName}
→ Informations du snapshot

DELETE /dedicated/nasha/{serviceName}/partition/{partitionName}/snapshot/{snapshotName}
→ Supprime le snapshot
```
