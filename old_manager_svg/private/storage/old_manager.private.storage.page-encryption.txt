# Cahier des Charges : private/storage - Page Encryption

## 1. VUE D'ENSEMBLE
- **Type** : Page chiffrement stockage
- **Contexte** : Storage - Sécurité
- **Objectif** : Gérer le chiffrement des données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Cartes KMS et stats
- **OdsButton** : Générer clé, Configurer
- **OdsDatagrid** : Liste des clés
- **OdsBadge** : État clé

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Chiffrement du stockage" |
| Description | "Gérez le chiffrement des données au repos." |

## 5. CARTE KMS
| Propriété | Valeur |
|-----------|--------|
| Titre | "Serveur de clés (KMS)" |
| État | Dot vert + "Connecté" |
| Serveur | kms.ovh.net:5696 |
| Algorithme | AES-256-XTS |
| Bouton | "Configurer" (outline) |

## 6. CARTE STATISTIQUES
| Métrique | Format |
|----------|--------|
| VMs chiffrées | X / Y |
| Datastores chiffrés | X / Y |
| Couverture | XX% (vert) |

## 7. COLONNES CLÉS
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| ID de la clé | 280px | Monospace |
| Créée le | 150px | Date |
| Utilisée par | 200px | X VMs, Y datastores |
| État | 150px | Badge |
| Rotation | 150px | Dans Xj |
| Actions | 100px | Menu ⋮ |

## 8. BADGE ÉTAT CLÉ
| État | Fond | Texte |
|------|------|-------|
| Active | #DCFCE7 | #16A34A |
| Expirée | #FEE2E2 | #DC2626 |
| Révoquée | #F9FAFB | #6B7280 |

## 9. APIS & ENDPOINTS
- GET /storage/encryption/kms - État KMS
- GET /storage/encryption/keys - Liste clés
- POST /storage/encryption/keys - Générer clé
- POST /storage/encryption/keys/{id}/rotate - Rotation clé
