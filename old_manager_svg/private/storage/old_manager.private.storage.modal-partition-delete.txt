# Cahier des Charges : private/storage - Modal Partition Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation destructive
**Fonction:** Supprimer une partition NAS-HA
**Contexte:** Suppression définitive de données

## 2. DIMENSIONS

- Modal: 450x270px
- Background page: #F9FAFB

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #FEE2E2 (danger light)
- Titre: "Supprimer la partition"
- Couleur titre: #DC2626
- Bouton fermer: × (rouge)
- Height: 55px

## 4. CONTENU

### Warning box
- Background: #FEF3C7
- Border: #FDE68A
- Icône: ⚠️
- Titre: "Cette action est irréversible"
- Texte: "Toutes les données et snapshots seront définitivement supprimés."

### Informations partition
- Label: "Partition :"
- Valeur: nom de la partition (bold)
- Détails: "Taille : 2 To • Utilisation : 50% • 14 snapshots"

## 5. COMPOSANTS UI

### Warning box
- Border-radius: 4px
- Dimensions: 410x70px

### Texte info partition
- Label: regular
- Valeur: font-weight 600
- Détails: text-muted

## 6. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x40px)
- Action: Ferme le modal

### Bouton Supprimer
- Style: btn-danger (120x40px)
- Background: #DC2626
- Action: Supprime la partition

### Conséquences
- Suppression de toutes les données
- Suppression de tous les snapshots
- Suppression de tous les accès configurés
- Action irréversible

## 7. APIS & ENDPOINTS

```
GET /dedicated/nasha/{serviceName}/partition/{partitionName}
→ Informations partition (taille, utilisation)

GET /dedicated/nasha/{serviceName}/partition/{partitionName}/snapshot
→ Liste des snapshots (pour comptage)

DELETE /dedicated/nasha/{serviceName}/partition/{partitionName}
→ Supprime la partition
```
