# Cahier des Charges : private/storage - Page Replication

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion réplication stockage
- **Contexte** : Private Cloud Storage
- **Objectif** : Configurer la réplication entre datastores

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et valeurs
- **OdsDatagrid** : Tableau réplications
- **OdsBadge** : État synchronisation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Réplication de stockage" |
| Description | "Configurez la réplication entre datastores." |

## 5. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Nom | 230px | Nom réplication (bold) |
| Source | 200px | Datastore source |
| Destination | 200px | Datastore destination |
| RPO | 100px | Recovery Point Objective |
| Dernier sync | 170px | Heure dernière sync |
| État | 100px | Badge état |

## 6. FORMAT RPO
| Valeur | Affichage |
|--------|-----------|
| Minutes | "X min" |
| Heures | "X heure(s)" |
| Jours | "X jour(s)" |

## 7. FORMAT DERNIER SYNC
| Propriété | Valeur |
|-----------|--------|
| Format | "HH:mm:ss" |
| Style | 13px #1F2937 |

## 8. BADGES ÉTAT
| État | Fond | Texte |
|------|------|-------|
| Sync | #DCFCE7 | #16A34A |
| En cours | #EFF6FF | #0050D7 |
| Erreur | #FEE2E2 | #DC2626 |
| Pause | #FEF3C7 | #92400E |

## 9. EXEMPLES RÉPLICATION
| Nom | Source | Destination | RPO |
|-----|--------|-------------|-----|
| repl-prod-dr | datastore-ssd-prod | datastore-dr-remote | 15 min |
| repl-db-backup | datastore-ssd-prod | datastore-nfs-backup | 1 heure |

## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/replication - Liste réplications
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/replication - Créer réplication
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/replication/{replId} - Modifier
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/replication/{replId} - Supprimer
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/replication/{replId}/sync - Forcer sync
