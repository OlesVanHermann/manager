# Cahier des Charges : private/storage - Modal Policy Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création politique stockage
- **Contexte** : Storage - vSAN/vSphere
- **Objectif** : Définir une politique de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 450x420px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom, IOPS, Latence
- **OdsSelect** : Type stockage, Réplication
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Créer une politique de stockage" |
| Bouton fermer | × |

## 5. CHAMP NOM
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom de la politique" |
| Placeholder | "Ex: policy-production-ssd" |
| Largeur | 400px |

## 6. SELECT TYPE STOCKAGE
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de stockage" |
| Options | SSD NVMe, SSD SATA, HDD, Hybrid |

## 7. SELECT RÉPLICATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Réplication" |
| Options | RAID-0, RAID-1 (Mirroring), RAID-5, RAID-6 |

## 8. CHAMPS PERFORMANCE
| Champ | Largeur | Exemple |
|-------|---------|---------|
| IOPS minimum | 190px | 1000 |
| Latence max (ms) | 190px | 10 |

## 9. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline | 90px |
| Créer | Primary #0050D7 | 90px |

## 10. APIS & ENDPOINTS
- GET /storage/policy/options - Options disponibles
- POST /storage/policy - Créer politique
