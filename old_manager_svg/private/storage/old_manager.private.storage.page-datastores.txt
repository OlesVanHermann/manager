# Cahier des Charges : private/storage - Page Datastores

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion datastores
- **Contexte** : Private Cloud Storage
- **Objectif** : Gérer les datastores de l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et valeurs
- **OdsButton** : Ajouter datastore
- **OdsCard** : Cartes datastores, résumé
- **OdsBadge** : Type stockage
- **OdsProgressBar** : Utilisation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Datastores" |
| Description | "Gérez les datastores de votre infrastructure." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter datastore" |
| Style | Fond #0050D7, texte blanc |

## 6. LAYOUT CARTES DATASTORES
| Propriété | Valeur |
|-----------|--------|
| Disposition | 3 colonnes |
| Dimensions | 340x160px |

## 7. CARTE DATASTORE
| Élément | Contenu |
|---------|---------|
| Badge type | SSD/HDD/NFS |
| Nom | Nom datastore (bold) |
| Progress bar | Utilisation + pourcentage |
| Stats | IOPS, Latence |
| Info | VMs count, Hôtes |
| Bouton | "Gérer" |

## 8. BADGES TYPE
| Type | Fond | Texte |
|------|------|-------|
| SSD | #EFF6FF | #0050D7 |
| HDD | #FEF3C7 | #92400E |
| NFS | #F3E8FF | #7C3AED |

## 9. COULEURS PROGRESS BAR
| Niveau | Couleur |
|--------|---------|
| 0-50% | #0050D7 ou #16A34A |
| 50-70% | #F59E0B |
| 70-100% | #DC2626 |

## 10. SECTION RÉSUMÉ CAPACITÉ
| Métrique | Style |
|----------|-------|
| Espace total | 20px, bold |
| Utilisé | 20px, bold |
| Disponible | 20px, bold, #16A34A |

## 11. BARRE GLOBALE
| Propriété | Valeur |
|-----------|--------|
| Fond | #E5E7EB |
| Remplissage | #0050D7 |
| Texte | "X% utilisé" |

## 12. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/datastore - Liste datastores
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/orderNewDatastore - Commander
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/datastore/{datastoreId} - Détail
