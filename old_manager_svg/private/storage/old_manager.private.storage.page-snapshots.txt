# Cahier des Charges : private/storage - Page Snapshots

## 1. VUE D'ENSEMBLE
- **Type** : Page listing snapshots
- **Contexte** : Private Cloud Storage
- **Objectif** : Gérer les points de restauration des VMs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsDatagrid** : Tableau snapshots
- **OdsPopover** : Menu actions
- **OdsCard** : Résumé
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Snapshots" |
| Description | "Gérez les points de restauration de vos VMs." |

## 5. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Nom | 200px | Nom snapshot (bold) |
| VM | 150px | Nom VM source |
| Date | 140px | Date/heure création |
| Taille | 100px | Taille (Go) |
| Description | 220px | Description (#6B7280) |
| Actions | 80px | Menu ⋮ |

## 6. FORMAT DATE
| Propriété | Valeur |
|-----------|--------|
| Format | "DD/MM/YYYY HH:mm" |

| Action | Style |
|--------|-------|

| --- | Séparateur |

| Propriété | Valeur |
|-----------|--------|

## 9. STATS RÉSUMÉ
| Métrique | Format |
|----------|--------|
| Espace utilisé | "Espace total utilisé par les snapshots: X Go" |

## 10. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "X - Y sur Z snapshots" |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/snapshot - Liste snapshots
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/snapshot - Créer snapshot
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/snapshot/{snapshotId}/restore - Restaurer
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/snapshot/{snapshotId} - Supprimer
