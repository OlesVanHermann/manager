# Cahier des Charges : private/storage - Page Dashboard

## 1. VUE D'ENSEMBLE

**Type:** Page dashboard
**Fonction:** Vue d'ensemble d'un NAS-HA
**Contexte:** Tableau de bord principal avec KPIs et partitions

## 2. DIMENSIONS

- Page: 1100x700px
- Cards KPI: 330x130px
- Card infos: 500x200px
- Card actions: 480x200px
- Tableau partitions: 1040x180px

## 3. LAYOUT & STRUCTURE

### Header page
- Titre: "NAS-HA zpool-xxxxx"

- Sous-titre: "10 To • Gravelines (GRA)"

### Lien
"Voir toutes les partitions →" (#0050D7)

## 8. APIS & ENDPOINTS

```
GET /dedicated/nasha/{serviceName}
→ Informations NAS-HA (nom, capacité, datacenter)

GET /dedicated/nasha/{serviceName}/partition
→ Liste des partitions

GET /dedicated/nasha/{serviceName}/use
→ Utilisation de l'espace

GET /dedicated/nasha/{serviceName}/task
→ Tâches en cours
```
