# Cahier des Charges : private/storage - Page Policies

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion politiques stockage
- **Contexte** : Private Cloud Storage - vSAN
- **Objectif** : Configurer les politiques de stockage VM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

- **OdsButton** : Créer politique

- **OdsPopover** : Menu actions

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/storagePolicy - Liste politiques
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/storagePolicy - Créer politique
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/storagePolicy/{policyId} - Modifier
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/storagePolicy/{policyId} - Supprimer
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/storagePolicy/{policyId}/vm - VMs associées
