- **Objectif** : Organiser et limiter les ressources par groupe

## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/resourcePool - Liste pools
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/resourcePool - Créer pool
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/resourcePool/{poolId} - Modifier
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/resourcePool/{poolId} - Supprimer
