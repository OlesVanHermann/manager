# Cahier des Charges : private/cloud - Modal Backup Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation destructive
**Fonction:** Désactiver le backup Veeam d'un datacenter
**Contexte:** Suppression définitive des sauvegardes

## 2. DIMENSIONS

- Modal: 450x270px
- Background page: #F9FAFB

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #FEE2E2 (danger light)
- Titre: "Désactiver le backup"
- Couleur titre: #DC2626
- Bouton fermer: × (rouge)
- Height: 55px

### Warning box
- Background: #FEF3C7
- Border: #FDE68A
- Dimensions: 410x80px
- Icône: ⚠️

## 4. CONTENU

### Message warning
- Titre: "Attention"
- Ligne 1: "La désactivation du backup supprimera définitivement"
- Ligne 2: "tous les points de restauration existants."

### Question confirmation
"Êtes-vous sûr de vouloir désactiver le backup pour ce datacenter ?"

### Note irréversibilité
- Texte: "Cette action est irréversible."
- Color: text-muted

## 5. COMPOSANTS UI

### Warning box
- Border-radius: 4px
- Icon + titre en gras
- Texte explicatif

### Style danger
- Header rouge clair
- Texte danger #DC2626

## 6. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x40px)
- Action: Ferme le modal

### Bouton Désactiver
- Style: btn-danger (120x40px)
- Background: #DC2626
- Action: Désactive le backup

### Conséquences
- Suppression de tous les restore points
- Arrêt des backups quotidiens
- Données non récupérables

## 7. APIS & ENDPOINTS

```
DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup
→ Désactive le backup

GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup
→ Statut actuel du backup
```
