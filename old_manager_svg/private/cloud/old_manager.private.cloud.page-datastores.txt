# Cahier des Charges : private/cloud - Page Datastores

## 1. VUE D'ENSEMBLE

**Type:** Page listing avec datagrid
**Fonction:** Gérer les datastores vSAN et NFS
**Contexte:** Vue des espaces de stockage VMware

## 2. DIMENSIONS

- Page: 1100x700px
- Datagrid: 1040x450px

## 3. LAYOUT & STRUCTURE

### Header page
- Titre: "Datastores"
- Sous-titre: "Gérez vos datastores vSAN et NFS"

### Bouton action
"Commander un datastore" (btn-primary, 180x40px)

### Info box
- Background: #EFF6FF
- Border: #BFDBFE
- Message: guide facturation et vMotion

## 4. DATAGRID DATASTORES

### Colonnes
| Colonne | Largeur | Description |
|---------|---------|-------------|
| Nom | 135px | Nom du datastore (link) |
| Profil | 130px | vSAN Premium, NFS Standard |
| Taille | 90px | Capacité totale |
| Utilisation | 160px | Progress bar avec % |
| Nœud actif | 120px | Primary/Secondary (host) |
| Facturation | 110px | Horaire/Mensuel |
| Statut | 100px | Badge Livré |
| Portée | 100px | Global/Local |
| Rack | 50px | R01, R02, etc. |
| Actions | 30px | Menu ⋮ |

### Progress bar utilisation
- Normal (<80%): #0050D7
- Warning (≥80%): #F59E0B
- Text: pourcentage au centre

### Données exemple
| Nom | Profil | Taille | Utilisation |

## 5. MENU ACTIONS

### Dropdown (180x80px)
- "Passer en mensuel" (si horaire)
- "Convertir en global" (si local)
- "Supprimer" (text-danger)

## 6. INFO BOX

### Contenu
"ℹ️ Pour modifier la facturation ou la portée d'un datastore, utilisez le menu d'actions."

### Liens
- "Guide facturation"
- "Guide vMotion"

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer
→ Liste des datastores

GET /dedicatedCloud/{serviceName}/filer/{filerId}
→ Détails d'un datastore

PUT /dedicatedCloud/{serviceName}/filer/{filerId}
Body: { billingType: "monthly" }
→ Change la facturation

DELETE /dedicatedCloud/{serviceName}/filer/{filerId}
→ Supprime le datastore

GET /dedicatedCloud/{serviceName}/filer/{filerId}/hourlyConsumption
→ Consommation si facturation horaire
```
