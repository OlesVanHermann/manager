# Cahier des Charges : private/cloud - Page Licenses

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion licences
- **Contexte** : Private Cloud - VMware
- **Objectif** : Visualiser et gérer les licences VMware vSphere

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Cartes licences
- **OdsProgressBar** : Utilisation licence

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Licences VMware" |
| Description | "Gérez vos licences VMware vSphere." |

## 5. LAYOUT CARTES LICENCES
| Propriété | Valeur |
|-----------|--------|
| Disposition | 3 colonnes |
| Dimensions carte | 340x140px |
| Gap | 20px |

## 6. CARTE LICENCE
| Zone | Contenu |
|------|---------|
| Titre | Nom licence (bold 14px) |
| Sous-titre | Nombre/type (12px #6B7280) |
| Progress bar | Utilisation |
| Stats | Pourcentage utilisé |
| Expiration | "Exp: DD/MM/YYYY" |

## 7. TYPES LICENCES
| Licence | Description |
|---------|-------------|
| vSphere Enterprise Plus | Licences CPU |
| vCenter Server Standard | Instance vCenter |
| NSX-T Data Center | Réseau software-defined |

## 8. COULEURS UTILISATION

| 0-50% | #0050D7 (bleu) |
| 50-90% | #F59E0B (orange) |

## 9. PROGRESS BAR
| Propriété | Valeur |

- GET /dedicatedCloud/{serviceName}/license - Liste licences
- GET /dedicatedCloud/{serviceName}/license/{licenseId} - Détail licence
- GET /dedicatedCloud/{serviceName}/license/history - Historique
