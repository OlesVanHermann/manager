# Cahier des Charges : private/cloud - Page Templates

## 1. VUE D'ENSEMBLE
- **Type** : Page listing templates VM
- **Contexte** : Private Cloud
- **Objectif** : Visualiser les modèles de VMs disponibles

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsSelect** : Filtre OS
- **OdsInput** : Recherche
- **OdsCard** : Cartes templates
- **OdsButton** : Créer VM, Importer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Templates VM" |
| Description | "Modèles de machines virtuelles disponibles." |

## 5. FILTRES
| Élément | Style |
|---------|-------|
| Select OS | Bordure bleue si actif |
| Recherche | Fond blanc, placeholder "Rechercher..." |

## 6. LAYOUT GRILLE
| Propriété | Valeur |
|-----------|--------|
| Disposition | 4 colonnes |
| Dimensions carte | 250x180px |
| Gap | 20px |

## 7. CARTE TEMPLATE
| Zone | Contenu |
|------|---------|
| Header | Fond couleur OS (10% opacité), icône, nom |
| Body | Type OS, bits, specs minimales |
| Footer | Bouton "Créer une VM" |

## 8. COULEURS OS
| OS | Couleur |
|----|---------|
| Ubuntu | #F97316 (orange) |
| Debian | #DC2626 (rouge) |
| Windows Server | #7C3AED (violet) |
| CentOS/Rocky | #0050D7 (bleu) |
| Windows Desktop | #0EA5E9 (cyan) |

## 9. SPECS MINIMALES
| Format | Exemple |
|--------|---------|
| Template | "Min: X vCPU, Y Go RAM, Z Go" |

## 10. BOUTON CRÉER VM
| Propriété | Valeur |
|-----------|--------|
| Texte | "Créer une VM" |
| Style | Fond #0050D7, texte blanc |
| Largeur | 210px |

## 11. CARTE IMPORT TEMPLATE
| Propriété | Valeur |
|-----------|--------|
| Bordure | Pointillés #E5E7EB |
| Icône | + (gros) |
| Texte | "Importer un template" |
| Formats | "OVA, OVF, VMDK" |

## 12. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "{count} templates disponibles" |

## 13. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/template - Liste templates
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/template/import - Importer
