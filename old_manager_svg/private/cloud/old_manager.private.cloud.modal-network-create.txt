# Cahier des Charges : private/cloud - Modal Network Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création réseau virtuel
- **Contexte** : Private Cloud
- **Objectif** : Créer un nouveau réseau virtuel/VLAN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Overlay | 550x450px |
| Modal | 450x390px |
| Background modal | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres et labels
- **OdsInput** : Champs formulaire
- **OdsCheckbox** : Option DHCP
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un réseau virtuel" |
| Fond header | #F9FAFB |
| Bouton fermer | × en #6B7280 |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nom du réseau | Input texte | Oui (*) | "Ex: Production-VLAN" |
| VLAN ID | Input numérique | Oui (*) | "100" |
| Sous-réseau | Input texte | Non | "192.168.1.0/24" |
| Passerelle | Input texte | Non | "192.168.1.1" |

## 6. CONTRAINTES VLAN ID
| Propriété | Valeur |
|-----------|--------|
| Minimum | 1 |
| Maximum | 4094 |
| Texte aide | "(1-4094)" |
| Style aide | 11px #6B7280 |

## 7. OPTION DHCP
| Propriété | Valeur |
|-----------|--------|
| Label | "Activer DHCP" |
| Type | Checkbox |
| État par défaut | Non coché |

## 8. STYLE INPUTS
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc |
| Bordure | #E5E7EB |
| Border radius | 4px |
| Hauteur | 40px |
| Input VLAN largeur | 150px |
| Inputs IP largeur | 250px |

## 9. BOUTONS FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline #E5E7EB, texte #6B7280 | Gauche |
| Créer | Fond #0050D7, texte blanc | Droite |

## 10. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/network - Créer réseau virtuel
