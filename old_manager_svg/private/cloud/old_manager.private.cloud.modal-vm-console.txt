# Cahier des Charges : private/cloud - Modal VM Console

## 1. VUE D'ENSEMBLE
- **Type** : Modal sélection type console
- **Contexte** : Private Cloud
- **Objectif** : Choisir le mode d'accès console à une VM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Overlay | 500x300px |
| Modal | 400x240px |
| Background modal | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres et labels
- **OdsRadio** : Options console (visuellement cartes)
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Accès console" |
| Fond header | #F9FAFB |
| Bouton fermer | × en #6B7280 |

## 5. TEXTE INSTRUCTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Choisissez le type de console pour {nom_vm}" |
| Style | 13px #1F2937 bold |

## 6. OPTIONS CONSOLE
| Option | Description |
|--------|-------------|
| Console HTML5 | "Console HTML5 (dans le navigateur)" |
| Console VMRC | "Console VMRC (application externe)" |

## 7. STYLE CARTE OPTION
| État | Bordure | Épaisseur |
|------|---------|-----------|
| Sélectionné | #0050D7 | 2px |
| Non sélectionné | #E5E7EB | 1px |

## 8. DIMENSIONS CARTES
| Propriété | Valeur |
|-----------|--------|
| Largeur | 350px |
| Hauteur | 50px |
| Border radius | 4px |
| Fond | Blanc |
| Texte | 13px #1F2937 centré vertical |

## 9. BOUTONS FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline #E5E7EB, texte #6B7280 | Gauche |
| Ouvrir | Fond #0050D7, texte blanc | Droite |

## 10. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/console/html5 - Console HTML5
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/console/vmrc - Console VMRC
