# Cahier des Charges : private/cloud - Page Hosts

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion hôtes ESXi
- **Contexte** : Private Cloud
- **Objectif** : Gérer les serveurs physiques de l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajouter hôte
- **OdsCard** : Cartes hôtes
- **OdsProgressBar** : Utilisation CPU/RAM
- **OdsSwitch** : Toggles DRS/HA

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Hôtes ESXi" |
| Description | "Gérez les serveurs physiques de votre infrastructure." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter hôte" |
| Style | Fond #0050D7, texte blanc |

## 6. LAYOUT CARTES HÔTES
| Propriété | Valeur |
|-----------|--------|
| Disposition | 3 colonnes |
| Dimensions | 340x170px |

## 7. CARTE HÔTE
| Élément | Style |
|---------|-------|
| Indicateur | Cercle couleur (vert/orange) |
| Nom | 14px, bold |
| Specs | 12px, #6B7280 |
| Barres | CPU et RAM progress bars |
| Stats | VMs count, Uptime |
| Bouton | "Gérer" |

## 8. INDICATEURS HÔTE
| Charge | Couleur |
|--------|---------|
| < 70% | #16A34A (vert) |
| 70-90% | #F59E0B (orange) |
| > 90% | #DC2626 (rouge) |

## 9. BADGE SURCHARGE
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Texte | "Surcharge" (#92400E) |
| Visible | Si CPU ou RAM > 90% |

## 10. HÔTE EN SURCHARGE
| Propriété | Valeur |
|-----------|--------|
| Bordure carte | #F59E0B |
| Valeurs | Texte #DC2626 |

## 11. SECTION RÉSUMÉ CLUSTER
| Métrique | Format |
|----------|--------|
| Total vCPU | "{used} / {total}" |
| Total RAM | "{used} Go / {total} Go" |
| Total VMs | Nombre |
| DRS | Toggle ON/OFF |
| HA | Toggle ON/OFF |

## 12. TOGGLE STYLE
| État | Style |
|------|-------|
| ON | Fond #0050D7, cercle droite |
| OFF | Fond #E5E7EB, cercle gauche |

## 13. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/host - Liste hôtes
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/host - Ajouter hôte
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/host/{hostId} - Détail hôte
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/cluster - Config DRS/HA
