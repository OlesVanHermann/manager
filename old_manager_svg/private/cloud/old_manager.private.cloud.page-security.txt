# Cahier des Charges : private/cloud - Page Security

## 1. VUE D'ENSEMBLE
- **Type** : Page paramètres sécurité
- **Contexte** : Private Cloud
- **Objectif** : Configurer la sécurité de l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsCard** : Sections paramètres
- **OdsSwitch** : Toggles on/off
- **OdsBadge** : État certificats
- **OdsButton** : Renouveler certificats
- **OdsSelect** : Session timeout

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Sécurité" |
| Description | "Paramètres de sécurité de votre cloud privé." |

## 5. SECTION POLITIQUE D'ACCÈS
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Politique d'accès" |
| Fond | Blanc |
| Hauteur | 180px |

## 6. PARAMÈTRES ACCÈS
| Paramètre | Type | État exemple |
|-----------|------|--------------|
| Authentification 2FA | Switch | Activé (bleu) |
| Restrictions IP | Switch | Désactivé (gris) |
| Session timeout | Select | "30 min" |

## 7. STYLE SWITCH
| État | Fond | Position cercle |
|------|------|-----------------|
| Activé | #0050D7 | Droite |

| Propriété | Valeur |
|-----------|--------|
| Texte | "Renouveler certificats" |
| Style | Fond #0050D7, texte blanc |
| Largeur | 180px |

## 13. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/security - Paramètres sécurité
- PUT /dedicatedCloud/{serviceName}/security - Modifier paramètres
- GET /dedicatedCloud/{serviceName}/certificate - Liste certificats
- POST /dedicatedCloud/{serviceName}/certificate/renew - Renouveler certificats
