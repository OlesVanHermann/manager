# Cahier des Charges : private/cloud - Page HA

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration haute disponibilité
- **Contexte** : Private Cloud - VMware vSphere HA
- **Objectif** : Configurer la haute disponibilité du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsCard** : Sections configuration et état
- **OdsSwitch** : Toggles activation
- **OdsBadge** : État cluster

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Haute Disponibilité (HA)" |
| Description | "Configuration de la haute disponibilité du cluster." |

## 5. SECTION CONFIGURATION
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Configuration HA" |
| Fond | Blanc |
| Hauteur | 180px |

## 6. PARAMÈTRES HA
| Paramètre | Type | Valeur exemple |
|-----------|------|----------------|
| État HA | Switch | Activé |
| Contrôle d'admission | Switch | Activé - Réserve 1 hôte |
| Heartbeat datastores | Texte | Liste datastores |
| Réponse isolation hôte | Texte | "Shutdown and restart VMs" |

## 7. STYLE SWITCH
| État | Fond | Position cercle |
|------|------|-----------------|
| Activé | #0050D7 | Droite |
| Désactivé | #E5E7EB | Gauche |

## 8. SECTION ÉTAT CLUSTER
| Propriété | Valeur |
|-----------|--------|
| Titre section | "État du cluster HA" |
| Fond | Blanc |
| Hauteur | 150px |

## 9. CARTES ÉTAT
| Carte | Contenu |
|-------|---------|
| Protection | "Cluster protégé", "X hôtes disponibles, Y en réserve" |
| VMs protégées | "X VMs avec restart priority" |
| Dernier failover | Date/heure ou "Jamais (aucun incident)" |

## 10. STYLE CARTE PROTECTION
| Propriété | Valeur |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/ha - Configuration HA
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/ha - Modifier HA
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/ha/status - État cluster
