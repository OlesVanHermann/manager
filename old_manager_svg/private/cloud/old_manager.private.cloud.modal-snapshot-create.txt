# Cahier des Charges : private/cloud - Modal Snapshot Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création snapshot
- **Contexte** : Private Cloud
- **Objectif** : Créer un point de restauration pour une VM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Overlay | 550x380px |
| Modal | 450x320px |
| Background modal | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres et labels
- **OdsInput** : Nom du snapshot
- **OdsTextarea** : Description
- **OdsCheckbox** : Option mémoire
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un snapshot" |
| Fond header | #F9FAFB |
| Bouton fermer | × en #6B7280 |

## 5. INFO VM
| Propriété | Valeur |
|-----------|--------|
| Label | "Machine virtuelle" |
| Style label | 11px #6B7280 |
| Valeur | Nom VM (ex: "web-server-01") |
| Style valeur | 13px #1F2937 bold |
| Fond | #F9FAFB, bordure #E5E7EB |

## 6. CHAMPS FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nom | Input texte | Oui (*) | - |
| Description | Textarea | Non | - |

## 7. STYLE CHAMPS
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc |
| Bordure | #E5E7EB |
| Border radius | 4px |
| Input hauteur | 40px |
| Textarea hauteur | 60px |

## 8. OPTION MÉMOIRE
| Propriété | Valeur |
|-----------|--------|
| Label | "Inclure la mémoire de la VM" |
| Type | Checkbox |
| État par défaut | Coché (bleu #0050D7) |
| Icône check | ✓ blanc |

## 9. BOUTONS FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline #E5E7EB, texte #6B7280 | Gauche |
| Créer | Fond #0050D7, texte blanc | Droite |

## 10. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/snapshot - Créer snapshot
