# Cahier des Charges : private/cloud - Page VM Listing

## 1. VUE D'ENSEMBLE
- **Type** : Page listing machines virtuelles
- **Contexte** : Private Cloud
- **Objectif** : Gérer les VMs du cloud privé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer VM
- **OdsInput** : Recherche
- **OdsDatagrid** : Tableau VMs
- **OdsBadge** : État VM
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm - Liste VMs
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm - Créer VM
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId} - Détail VM
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/start - Démarrer
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/stop - Arrêter
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/reboot - Redémarrer
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/migrate - Migrer
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId} - Supprimer
