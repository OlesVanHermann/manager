# Cahier des Charges : private/cloud - Page Console

## 1. VUE D'ENSEMBLE
- **Type** : Page console VM
- **Contexte** : Private Cloud
- **Objectif** : Accès direct à la console d'une machine virtuelle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #1F2937 (dark) |

## 3. COMPOSANTS UI
- **OdsText** : Titre VM
- **OdsButton** : Actions console
- **Console** : Zone terminal (noVNC/VMRC)

## 4. HEADER CONSOLE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Console VM: {nom_vm}" |
| Style | 14px blanc bold |
| Fond header | #1F2937 |

## 5. BOUTONS ACTIONS
| Bouton | Position | Style |
|--------|----------|-------|
| Ctrl+Alt+Del | Droite | Fond #374151, texte blanc |
| Plein écran | Droite | Fond #374151, texte blanc |

## 6. STYLE BOUTONS
| Propriété | Valeur |
|-----------|--------|
| Hauteur | 30px |
| Border radius | 4px |
| Fond | #374151 |
| Texte | 11px blanc |

## 7. ZONE CONSOLE
| Propriété | Valeur |
|-----------|--------|
| Dimensions | 1040x500px |
| Fond | #0a0a0a |
| Bordure | #374151 |
| Border radius | 4px |

## 8. TERMINAL
| Propriété | Valeur |
|-----------|--------|
| Police | monospace |
| Taille | 14px |
| Couleur texte | #22c55e (vert terminal) |
| Curseur | Block █ clignotant |

## 9. RACCOURCIS CLAVIER
| Raccourci | Action |
|-----------|--------|
| Ctrl+Alt+Del | Envoi séquence reboot |
| F11 | Plein écran |
| Échap | Quitter plein écran |

## 10. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/console/access - Obtenir URL console
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/console/ticket - Ticket VMRC
