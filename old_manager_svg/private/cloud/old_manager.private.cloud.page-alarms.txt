# Cahier des Charges : private/cloud - Page Alarms

## 1. VUE D'ENSEMBLE
- **Type** : Page listing alarmes
- **Contexte** : Private Cloud
- **Objectif** : Surveiller les alertes de l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

- **OdsText** : Titres et descriptions
- **OdsCard** : Cartes résumé
- **OdsBadge** : Niveaux alarme
