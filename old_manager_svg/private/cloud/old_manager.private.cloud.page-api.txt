- **Type** : Page accès API
- **Contexte** : Hosted Private Cloud (VMware)
- **Objectif** : Gérer l'accès programmatique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/api - Endpoints API
- GET /dedicatedCloud/{serviceName}/api/token - Liste tokens
- POST /dedicatedCloud/{serviceName}/api/token - Créer token
- DELETE /dedicatedCloud/{serviceName}/api/token/{tokenId} - Révoquer token
