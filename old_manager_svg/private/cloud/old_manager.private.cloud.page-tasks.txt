# Cahier des Charges : private/cloud - Page Tasks

## 1. VUE D'ENSEMBLE
- **Type** : Page listing tâches
- **Contexte** : Private Cloud
- **Objectif** : Historique des opérations sur l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsChip** : Filtres
- **OdsDatagrid** : Tableau tâches
- **OdsBadge** : État tâche
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Tâches" |
| Description | "Historique des opérations sur votre infrastructure." |

## 5. FILTRES
| Filtre | État |
|--------|------|
| Toutes | Actif (bordure bleue) |
| En cours | Inactif |
| Terminées | Inactif |
| Échouées | Inactif |

## 6. STYLE FILTRE
| État | Style |
|------|-------|
| Actif | Fond blanc, bordure #0050D7, texte #0050D7 |
| Inactif | Fond blanc, bordure #E5E7EB, texte #6B7280 |

## 7. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Tâche | 260px | Type opération |
| Cible | 160px | VM/Ressource |
| Utilisateur | 140px | Email |
| Début | 140px | Heure |
| Durée | 100px | Temps écoulé |
| État | 120px | Badge état |

## 8. BADGES ÉTAT
| État | Fond | Texte | Format |
|------|------|-------|--------|
| En cours | #EFF6FF | #0050D7 | "En cours X%" |
| Succès | #DCFCE7 | #16A34A | "Succès" |
| Échec | #FEE2E2 | #DC2626 | "Échec" |

## 9. FORMAT DURÉE
| Exemples | Format |
|----------|--------|
| < 1 min | "X sec" |
| 1-60 min | "X min" ou "X min Ys" |
| > 1h | "Xh Ymin" |

## 10. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "X - Y sur Z tâches" |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/task - Liste tâches
- GET /dedicatedCloud/{serviceName}/task/{taskId} - Détail tâche
