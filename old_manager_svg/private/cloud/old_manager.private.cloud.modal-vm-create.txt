# Cahier des Charges : private/cloud - Modal VM Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création VM
- **Contexte** : Private Cloud
- **Objectif** : Créer une nouvelle machine virtuelle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 540px |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Titres et labels
- **OdsInput** : Nom VM
- **OdsSelect** : Template, ressources, réseau, hôte
- **OdsButton** : Actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Créer une machine virtuelle" |
| Bouton fermer | × |

## 5. CHAMP NOM VM
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom de la VM *" |
| Type | Input text |
| Placeholder | "new-vm-prod" |
| Obligatoire | Oui |

## 6. CHAMP TEMPLATE OS
| Propriété | Valeur |
|-----------|--------|
| Label | "Template OS *" |
| Type | Select |
| Exemple | "Ubuntu Server 22.04 LTS" |
| Obligatoire | Oui |

## 7. SECTION RESSOURCES
| Champ | Type | Options |
|-------|------|---------|
| vCPU | Select | 1, 2, 4, 8, 16, 32 |
| RAM | Select | 2 Go, 4 Go, 8 Go, 16 Go, 32 Go, 64 Go |
| Disque | Select x2 | Taille (Go) + Type (SSD/HDD) |

## 8. LAYOUT RESSOURCES
| Propriété | Valeur |
|-----------|--------|
| Label principal | "Ressources" |
| Sous-labels | "vCPU", "RAM", "Disque" |

## 9. CHAMP RÉSEAU
| Propriété | Valeur |
|-----------|--------|
| Label | "Réseau" |
| Type | Select |
| Format option | "{vlanName} ({subnet})" |

## 10. CHAMP HÔTE CIBLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Hôte cible" |
| Type | Select |
| Option par défaut | "Automatique (DRS)" |

## 11. FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Fond blanc, bordure grise |
| Créer | Fond #0050D7, texte blanc |

## 12. VALIDATION
| Condition | Comportement |
|-----------|--------------|
| Nom vide | Bouton désactivé |
| Template non sélectionné | Bouton désactivé |
| Tous champs valides | Bouton actif |

## 13. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/template - Templates
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm - Créer VM
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host - Hôtes disponibles
