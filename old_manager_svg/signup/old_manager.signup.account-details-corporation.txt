# Cahier des Charges : signup - Détails Compte Entreprise

## 1. VUE D'ENSEMBLE
- **Page** : Informations entreprise (Corporation)
- **Type** : Étape wizard (3/3)
- **Objectif** : Valider/compléter les infos entreprise

## 2. STRUCTURE PAGE
- Header simple avec logo OVHcloud
- Indicateur d'étape "Étape 3 sur 3"
- Layout 2 colonnes : Formulaire (720px) + Réassurance (480px)
- Badge "Pré-rempli" sur le titre
- Footer RGPD

## 3. COMPOSANTS UI
- **OdsCard** : Container formulaire + réassurance
- **OdsMessage** : Bannière info
- **OdsInput** : Champs éditables
- **OdsInput disabled** : Champs pré-remplis (grisés)
- **OdsBadge** : "Pré-rempli" (vert)
- **OdsSelect** : Langue
- **OdsPhoneNumber** : Numéro téléphone
- **OdsCheckbox** : Confirmations
- **OdsButton** : Valider

## 4. BADGE PRÉ-REMPLI
| Propriété | Valeur |
|-----------|--------|
| Style | Vert (#DEF7EC) |
| Texte | "Pré-rempli" |
| Position | À droite du titre |

## 6. CHAMPS DISABLED
- Fond gris (#F3F4F6)
- Texte gris (#6B7280)
- Non modifiables (données INSEE)
