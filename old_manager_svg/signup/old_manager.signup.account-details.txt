# Cahier des Charges : signup - Détails Compte Particulier

## 1. VUE D'ENSEMBLE
- **Page** : Informations personnelles (Particulier)
- **Type** : Étape wizard (3/3)
- **Objectif** : Saisir les informations du compte particulier

## 2. STRUCTURE PAGE
- Header simple avec logo OVHcloud
- Indicateur d'étape "Étape 3 sur 3"
- Layout 2 colonnes : Formulaire (720px) + Réassurance (480px)
- Footer RGPD

## 3. COMPOSANTS UI
- **OdsCard** : Container formulaire + réassurance
- **OdsMessage** : Bannière info
- **OdsInput** : Champs texte
- **OdsSelect** : Pays, Type téléphone, Langue
- **OdsPhoneNumber** : Numéro téléphone
- **OdsCheckbox** : Confirmations
- **OdsButton** : Valider

## 4. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Style | Info (bleu #EFF6FF) |
| Icône | ℹ️ |
| Texte | "Assurez-vous que les informations que vous avez renseignées sont exactes." |

## 5. SECTIONS FORMULAIRE
### Section 1 : Informations personnelles
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Prénom | Input | ✓ | "Jean" |
| Nom | Input | ✓ | "Dupont" |

### Section 2 : Adresse postale
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Pays de résidence | Select | ✓ | "France" |
| Adresse | Input | ✓ | "2 rue Kellermann" |
| Code postal | Input | ✓ | "59100" |
| Ville | Input | ✓ | "Roubaix" |

### Section 3 : Contact
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Téléphone | PhoneNumber | ✓ | "+33 6 12 34 56 78" |
| Type de téléphone | Select | Non | "Téléphone portable" |
| Langue préférée | Select | Non | "Français" |

## 9. APIS & ENDPOINTS
- POST /me/signup - Créer le compte
- GET /me/geolocation/country - Liste des pays
- POST /me/contact - Créer contact
