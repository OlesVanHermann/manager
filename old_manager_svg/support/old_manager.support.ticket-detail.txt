# Cahier des Charges : support - Détail ticket

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'un ticket
- **Type** : Conversation avec timeline
- **Objectif** : Suivre et répondre à un ticket

## 2. STRUCTURE PAGE
- Header avec infos ticket
- Timeline messages
- Zone réponse
- Sidebar infos

## 3. COMPOSANTS UI
- **OdsBadge** : Statut, priorité
- **OdsCard** : Messages
- **OdsTextarea** : Nouvelle réponse
- **OdsFileUpload** : Pièces jointes
- **OdsButton** : Envoyer, Fermer ticket

## 4. HEADER TICKET
| Élément | Type | Description |
|---------|------|-------------|
| N° Ticket | Text | #123456 |
| Sujet | H1 | Titre du ticket |
| Statut | Badge | État courant |
| Priorité | Badge | Niveau urgence |

## 5. SIDEBAR INFOS
| Info | Valeur |
|------|--------|
| Service | Nom du service |
| Créé le | Date création |
| Mis à jour | Dernière activité |
| Catégorie | Type demande |
| Assigné à | Agent support |

## 6. TIMELINE MESSAGES
| Type message | Style |
|--------------|-------|
| Client | Aligné droite, fond bleu |
| Support | Aligné gauche, fond gris |

### Structure message
| Élément | Type |
|---------|------|
| Avatar | Icône utilisateur/support |
| Auteur | Nom |
| Date | Horodatage |
| Contenu | Text |
| Pièces jointes | Liste liens |

## 7. ZONE RÉPONSE
| Champ | Type | Description |
|-------|------|-------------|
| Message | Textarea | Nouvelle réponse |
| Pièces jointes | FileUpload | Fichiers à joindre |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Envoyer | Primary | Poster réponse |
| Fermer le ticket | Outline | Clôturer |
| Rouvrir | Outline | Si fermé |

## 9. APIS & ENDPOINTS
- GET /support/tickets/{id} - Détail ticket
- GET /support/tickets/{id}/messages - Messages
- POST /support/tickets/{id}/messages - Répondre
- PUT /support/tickets/{id}/close - Fermer
- PUT /support/tickets/{id}/reopen - Rouvrir
