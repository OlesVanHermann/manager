# Cahier des Charges : support - Status des services

## 1. VUE D'ENSEMBLE
- **Page** : Status des services OVHcloud
- **Type** : Dashboard de monitoring
- **Objectif** : Consulter la disponibilité des services

## 2. STRUCTURE PAGE
- Header avec statut global
- Filtres par région/produit
- Liste services avec barres uptime
- Historique incidents

## 3. COMPOSANTS UI
- **OdsBadge** : Statut global
- **OdsSelect** : Filtres
- **OdsCard** : Carte service
- **OdsProgress** : Barre uptime
- **OdsTimeline** : Historique incidents

## 4. STATUT GLOBAL
| Élément | Type | Description |
|---------|------|-------------|
| Icône | ✓ ou ⚠ | État global |
| Texte | Text | Tous les services opérationnels / Incident en cours |
| Badge | Badge | Operational / Degraded / Outage |

## 5. FILTRES
| Filtre | Options |
|--------|---------|
| Région | Europe, Amérique, Asie-Pacifique |
| Produit | Cloud, Web, Network, Bare Metal |
| Période | 24h, 7j, 30j, 90j |

## 6. CARTE SERVICE
| Élément | Type | Description |
|---------|------|-------------|
| Nom | H3 | Nom du service |
| Statut | Badge | Operational/Degraded/Outage |
| Uptime | Progress | Barre 99.9% |
| Détails | Link | Voir incidents |

## 7. BARRE UPTIME
| Couleur | Signification |
|---------|---------------|
| Vert | Opérationnel |
| Orange | Dégradé |
| Rouge | Panne |
| Gris | Maintenance |

## 8. HISTORIQUE INCIDENTS
| Colonne | Type | Description |
|---------|------|-------------|
| Date | Date | Horodatage |
| Service | Text | Service impacté |
| Type | Badge | Incident/Maintenance |
| Durée | Text | Temps d'indisponibilité |
| Statut | Badge | Résolu/En cours |

## 9. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Operational | Vert |
| Degraded Performance | Orange |
| Partial Outage | Orange |
| Major Outage | Rouge |
| Maintenance | Bleu |

## 10. APIS & ENDPOINTS
- GET /status - Statut global
- GET /status/services - Liste services
- GET /status/services/{id} - Détail service
- GET /status/incidents - Historique incidents
- GET /status/maintenance - Maintenances planifiées
