================================================================================
CAHIER DES CHARGES - old_manager.telecom.fax.voicemail-activation
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/fax
Composant  : Activation repondeur fax
Type       : Page toggle activation
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Page permettant d activer ou desactiver la fonction repondeur sur un service fax.
Affiche l etat actuel et propose un bouton pour basculer l etat.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| < Retour au fax                                                  |
+------------------------------------------------------------------+
| Repondeur                                                        |
|                                                                  |
| +--------------------------------------------------------------+ |
| | ETAT ACTIF                                                   | |
| | La fonction "Repondeur" est active sur votre Fax.            | |
| |                                                              | |
| | [Desactiver la fonction "Repondeur"]                         | |
| +--------------------------------------------------------------+ |
|                                                                  |
| OU                                                               |
|                                                                  |
| +--------------------------------------------------------------+ |
| | ETAT INACTIF                                                 | |
| | La fonction "Repondeur" n est pas active sur votre Fax.      | |
| |                                                              | |
| | [Activer la fonction "Repondeur"]                            | |
| +--------------------------------------------------------------+ |
|                                                                  |
+------------------------------------------------------------------+

4. ETATS
--------
- Chargement : Spinner pendant recuperation config
- Actif : Message + bouton desactiver
- Inactif : Message + bouton activer
- Sauvegarde : Spinner pendant mise a jour
- Succes : Toast "Mise a jour effectuee avec succes"
- Erreur : Toast erreur

5. ENDPOINTS API
----------------
| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /telephony/{billingAccount}/fax/{serviceName}/settings | Recuperer config |
| PUT | /telephony/{billingAccount}/fax/{serviceName}/settings | Modifier config |

6. COMPOSANTS UTILISES
----------------------
- tuc-section-back-link : Lien retour
- oui-spinner : Chargement
- tuc-toast-message : Notifications

7. TRADUCTIONS FR
-----------------

8. ACTIONS UTILISATEUR
----------------------
- Activer : Active le repondeur
- Desactiver : Desactive le repondeur
