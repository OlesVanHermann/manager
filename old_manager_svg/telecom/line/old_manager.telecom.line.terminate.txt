# Cahier des Charges : telecom/line - Resiliation de la ligne

## 1. RESUME EXECUTIF

### Perimetre
Demande de resiliation d'une ligne telephonique avec procedure de retour materiel.

### Objectif fonctionnel
- Demander la resiliation de la ligne
- Selectionner la raison de resiliation
- Informer sur la procedure de retour materiel

### Users cibles
- Administrateurs telephonie
- Gestionnaires de comptes

---

## 2. LAYOUT

| Colonne | Contenu |
|---------|---------|
| Gauche | Informations et procedure retour |
| Droite | Formulaire de resiliation |

## 4. APIS & DONNEES

### Endpoints

```
GET /telephony/{billingAccount}/line/{serviceName}
    Response: { canTerminate: boolean, ... }

POST /telephony/{billingAccount}/line/{serviceName}/terminate
    Body: { reason: string }

DELETE /telephony/{billingAccount}/line/{serviceName}/terminate
    â†’ Annule la demande de resiliation
```

### Enum raisons
```
telephony.TerminationReasonEnum:
  - moving
  - billingIssue
  - cessationOfActivity
  - lackOfFeatures
  - technicalIssues
```

## 6. REGLES METIER

- Resiliation effective a la prochaine facturation
- Annulation possible jusqu'a la date effective
- Retour materiel sous 15 jours
- Caution restituee sous 10 jours apres reception
- Raison obligatoire pour statistiques
