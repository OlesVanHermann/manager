# Cahier des Charges : telecom/line - Redemarrer le telephone

## 1. RESUME EXECUTIF

### Perimetre
Redemarrage a distance d'un telephone Plug & Phone.

### Objectif fonctionnel
- Afficher les informations du telephone
- Declencher un redemarrage a distance
- Appliquer les configurations en attente

---

## 2. LAYOUT

### Section unique avec informations et bouton

---

## 3. COMPOSANTS UI

### Zone informations
| Champ | Type | Exemple |
|-------|------|---------|
| Marque / Modele | Label + Value | Cisco SPA504G |
| Adresse MAC | Label + Value | 00:1A:2B:3C:4D:5E |

### Bouton action
| Bouton | Icone | Texte |
|--------|-------|-------|
| Default | Refresh | Redemarrer le telephone |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone
    Response: { brand, model, macAddress }

POST /telephony/{billingAccount}/line/{serviceName}/phone/reboot
```
