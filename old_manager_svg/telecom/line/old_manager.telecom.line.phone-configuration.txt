# Cahier des Charges : telecom/line - Configuration du telephone

## 1. RESUME EXECUTIF

### Perimetre
Configuration avancee des parametres du telephone IP (Plug & Phone).

### Objectif fonctionnel
- Modifier les parametres d'interface utilisateur
- Configurer les options reseau
- Ajuster les parametres audio
- Gerer le redemarrage apres sauvegarde

---

## 2. LAYOUT

### Structure
- Switch mode Standard/Expert
- 4 sections en grille 2x2

---

## 3. COMPOSANTS UI

### Barre de mode
| Element | Description |
|---------|-------------|
| Switch | Standard / Expert |
| Bouton | "Modifier la configuration" |

### Section Interface utilisateur
| Parametre | Valeur exemple |
|-----------|----------------|
| Langue de l'afficheur | Francais |
| Format de l'heure | 24h |
| Format de la date | JJ/MM/AAAA |
| Retroeclairage | Auto |
| Volume sonnerie | 8 |
| Volume ecouteur | 6 |

### Section Reseau
| Parametre | Valeur exemple |
|-----------|----------------|
| Mode IP | DHCP |
| VLAN active | Non |
| QoS active | Oui |
| Serveur NTP | ntp.ovh.net |

### Section Audio
| Parametre | Valeur exemple |
|-----------|----------------|
| Codec prefere | G.722 |
| Suppression echo | Active |
| VAD | Desactive |

### Section Redemarrage
| Element | Description |
|---------|-------------|
| Checkbox | Redemarrer le telephone apres la sauvegarde |

### Boutons (mode edition)
| Bouton | Type | Action |
|--------|------|--------|
| Valider | Primary | Sauvegarde |
| Valeurs par defaut | Default | Reset |
| Annuler | Default | Abandon |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone
GET /telephony/{billingAccount}/line/{serviceName}/phone/functionKey
PUT /telephony/{billingAccount}/line/{serviceName}/phone
POST /telephony/{billingAccount}/line/{serviceName}/phone/reboot
```
