# Cahier des Charges : telecom/line - Softphone

## 1. RESUME EXECUTIF

### Perimetre
Configuration du softphone OVH pour utiliser la ligne depuis une application mobile.

### Objectif fonctionnel
- Activer/desactiver le softphone
- Gerer les appareils autorises
- Personnaliser le theme et le logo

---

## 2. LAYOUT

### 4 sections verticales
| Section | Contenu |
|---------|---------|
| 1 | Activation + liens telechargement |
| 2 | Configuration des appareils |
| 3 | Personnalisation du theme |
| 4 | Logo personnalise |

---

## 3. COMPOSANTS UI

### Section Activation
| Element | Description |
|---------|-------------|
| Toggle switch | Activer le softphone |
| Texte info | Description du service |

### Section Telechargement
| Element | Description |
|---------|-------------|
| Bouton App Store | Lien iOS |
| Bouton Google Play | Lien Android |
| Lien direct | URL telechargement |

### Section Configuration appareils
| Element | Description |
|---------|-------------|
| Texte | Maximum 3 appareils |
| Bouton | + Generer un jeton |
| Bouton | Tout supprimer |
| Tableau | Liste des appareils |

### Tableau appareils
| Colonne | Description |
|---------|-------------|
| ID | Identifiant jeton |
| Nom | Nom appareil |
| OS | iOS / Android |
| User Agent | Version app |

### Section Theme
| Element | Description |
|---------|-------------|
| Cercles couleurs | Bleu, Vert, Violet, Orange, Rouge |
| Selection | Bordure sur couleur active |
| Bouton | Appliquer |

### Section Logo
| Element | Description |
|---------|-------------|
| Zone drag & drop | Glisser un fichier |
| Preview | Apercu logo actuel |
| Info | Format PNG/JPG, max 512KB |
| Bouton | Appliquer |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/options
    Response: { softphoneEnabled: boolean }

PUT /telephony/{billingAccount}/line/{serviceName}/options
    Body: { softphoneEnabled: boolean }

GET /telephony/softphone/storeLinks
    Response: { ios, android, direct }

GET /telephony/softphone/themes
    Response: [{ id, color }]

GET /telephony/{billingAccount}/line/{serviceName}/softphone/devices
POST /telephony/{billingAccount}/line/{serviceName}/softphone/devices
DELETE /telephony/{billingAccount}/line/{serviceName}/softphone/devices/{id}
```
