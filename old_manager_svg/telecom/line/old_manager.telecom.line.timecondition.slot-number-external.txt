# Cahier des Charges : telecom/line - Creneau numero externe

## 1. RESUME EXECUTIF

### Perimetre
Configuration d'un creneau avec redirection vers un numero externe.

### Objectif fonctionnel
- Selectionner le type "numero externe"
- Saisir le numero de destination
- Valider le format international

## 4. APIS & DONNEES

### Endpoints
```
PUT /telephony/{billingAccount}/line/{serviceName}/timeCondition/slots/{slotId}
    Body: {
      type: "external",
      number: "+33612345678"
    }
```

### Validation numero
| Regle | Description |
|-------|-------------|
| Format | E.164 international |
| Prefixe | Doit commencer par + |
| Longueur | 8-15 chiffres apres indicatif |
