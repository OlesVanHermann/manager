# Cahier des Charges : telecom/line - Repetition plage horaire

## 1. RESUME EXECUTIF

### Perimetre
Modal de selection des jours de repetition d'une plage horaire.

### Objectif fonctionnel
- Selectionner les jours individuels
- Utiliser des raccourcis (semaine, week-end)
- Visualiser la selection

## 4. APIS & DONNEES

### Structure repetition
```javascript
{
  repeat: {
    monday: true,
    tuesday: false,
    wednesday: false,
    thursday: false,
    friday: false,
    saturday: false,
    sunday: false
  }
}
```

### Raccourcis
| Raccourci | Jours |
|-----------|-------|
| weekdays | monday-friday |
| weekend | saturday-sunday |
| everyday | monday-sunday |

- Resume mis a jour dynamiquement
