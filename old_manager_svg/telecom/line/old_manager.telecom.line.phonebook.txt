# Cahier des Charges : telecom/line - Carnet d'adresses

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion du carnet d'adresses d'une ligne telephonique.

### Objectif fonctionnel
- Gerer les contacts du telephone
- Ajouter/modifier/supprimer des contacts
- Importer/exporter le carnet
- Supprimer en masse

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone/phonebook
POST /telephony/{billingAccount}/line/{serviceName}/phone/phonebook
PUT /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/{id}
DELETE /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/{id}
POST /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/import
GET /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/export
```

### Modele Contact
```typescript
interface PhonebookContact {
  id: string;
  surname: string;
  name: string;
  group: string;
  homePhone?: string;
  homeMobile?: string;
  workPhone?: string;
  workMobile?: string;
}
```

## 6. REGLES METIER

- Maximum 200 contacts par carnet
- Groupes: Travail, Famille, Amis
- Au moins un numero requis par contact
- Import CSV ou vCard
- Synchronisation avec telephone
