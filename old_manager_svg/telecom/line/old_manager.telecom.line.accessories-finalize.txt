# Cahier des Charges : telecom/line - Confirmation accessoires

## 1. RESUME EXECUTIF

### Perimetre
Page de confirmation apres validation de la commande d'accessoires.

### Objectif fonctionnel
- Confirmer la validation de la commande
- Afficher le numero de commande
- Proposer le paiement

## 4. APIS & DONNEES

### Endpoints
```
GET /me/order/{orderId}
POST /me/order/{orderId}/pay
```

### Modele
```typescript
interface OrderConfirmation {
  orderId: string;
  orderNumber: string;
  totalHT: number;
  totalTTC: number;
  status: 'pending' | 'paid' | 'shipped';
  paymentUrl: string;
}
```
