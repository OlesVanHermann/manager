# Cahier des Charges : telecom/line - Configuration codec

## 1. RESUME EXECUTIF

### Perimetre
Page de configuration du codec audio d'une ligne.

### Objectif fonctionnel
- Choisir le codec audio prioritaire
- Activer le mode automatique
- Appliquer a plusieurs lignes

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone/codec
PUT /telephony/{billingAccount}/line/{serviceName}/phone/codec
```

### Modele
```typescript
interface CodecConfig {
  currentCodecs: string[];
  autoMode: boolean;
  priority: 'quality' | 'bandwidth';
}
```

## 6. REGLES METIER

- G711A = 64 kbit/s, qualite CD
- G729 = 8 kbit/s, qualite telephone
- Mode auto = negociation automatique
- Changement effectif immediat
