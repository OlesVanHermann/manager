## 1. RESUME EXECUTIF

### Perimetre fonctionnel
Module de gestion des lignes telephoniques SIP/MGCP OVHcloud.

### Types de lignes
| Type | Description |
|------|-------------|
| sip | Ligne SIP standard |
| mgcp | Ligne MGCP (deprecie) |
| trunk | Trunk SIP entreprise |
