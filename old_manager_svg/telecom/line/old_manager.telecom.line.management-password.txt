## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}
POST /telephony/{billingAccount}/line/{serviceName}/changePassword
```

### Modele
```typescript
interface SIPCredentials {
  sipLogin: string;
  sipServer: string;
}

interface ChangePasswordRequest {
  password: string;
}

interface PasswordStrength {
  score: number; // 0-4
  level: 'weak' | 'fair' | 'good' | 'strong';
}
```

- Majuscule + minuscule + chiffre recommandes
- Telephone doit etre reconfigure apres changement
- Generation auto = mot de passe fort aleatoire
