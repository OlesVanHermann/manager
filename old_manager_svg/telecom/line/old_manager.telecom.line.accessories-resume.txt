# Cahier des Charges : telecom/line - Recapitulatif accessoires

## 1. RESUME EXECUTIF

### Perimetre
Page de recapitulatif avant validation de la commande d'accessoires.

### Objectif fonctionnel
- Voir le recapitulatif commande
- Verifier l'adresse de livraison
- Accepter les conditions

## 4. APIS & DONNEES

### Endpoints
```
GET /me/order/cart/{cartId}
POST /me/order/cart/{cartId}/checkout
```

### Modele
```typescript
interface OrderSummary {
  items: OrderItem[];
  totalHT: number;
  vat: number;
  totalTTC: number;
  shippingAddress: Contact;
}

interface OrderItem {
  description: string;
  unitPrice: number;
  quantity: number;
  total: number;
}
```
