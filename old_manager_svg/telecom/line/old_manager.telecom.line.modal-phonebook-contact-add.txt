# Cahier des Charges : telecom/line - Modal ajout contact

## 1. RESUME EXECUTIF

### Perimetre
Modal d'ajout d'un nouveau contact au carnet d'adresses.

### Objectif fonctionnel
- Creer un nouveau contact
- Definir les numeros de telephone
- Assigner a un groupe

## 3. COMPOSANTS UI

### Modal
| Element | Description |
|---------|-------------|
| Titre | Ajouter un contact |
| Bouton fermer | X en haut droite |
| Overlay | Fond sombre |

### Champs identite
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Text input | Oui |
| Prenom | Text input | Oui |

### Champ groupe
| Element | Description |
|---------|-------------|
| Input | Text avec valeur |
| Bouton | Choisir (dropdown) |

### Info box
| Element | Description |
|---------|-------------|
| Fond | Bleu clair |
| Icone | i |

| Bouton | Type | Position |
|--------|------|----------|

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/line/{serviceName}/phone/phonebook
```

### Modele
```typescript
interface PhonebookContactCreate {
  surname: string;
  name: string;
  group?: string;
  homePhone?: string;
  homeMobile?: string;
  workPhone?: string;
  workMobile?: string;
}
```

## 6. REGLES METIER

- Nom et prenom obligatoires
- Au moins un numero telephone requis
- Format numeros: E.164 ou national
- Groupe optionnel (defaut: sans groupe)
