1. DESCRIPTION FONCTIONNELLE
================================================================================
Bandeau d'information pour rappeler à l'utilisateur de valider son identité
pour la téléphonie. Affiché conditionellement si la procédure n'est pas
complétée.

Obligation légale : Vérification d'identité pour les services télécom.

Cas d'usage :
- Dashboard téléphonie
- Page de commande de ligne
- Hub manager

================================================================================

|---------------------------|---------|-----------|--------------------------|

| identityCheckFormLink     | string  | @         | URL du formulaire        |

4. CONDITIONS D'AFFICHAGE
================================================================================
