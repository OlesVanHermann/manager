================================================================================
CAHIER DES CHARGES - old_manager.telecom.common.success-drawing-check
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/common
Composant  : Animation succes (check anime)
Type       : Composant animation SVG
Framework  : AngularJS 1.7.x
Module     : telecom-universe-components

2. DESCRIPTION FONCTIONNELLE
----------------------------
Composant affichant une animation de coche (checkmark) pour indiquer
le succes d une operation. L animation dessine progressivement le check.
Utilise dans les modals de confirmation apres une action reussie.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
|                                                                  |
|                         ___                                      |
|                        /   \                                     |
|                       |  v  |  <-- Check anime                   |
|                        \___/                                     |
|                        ====   <-- Ombre                          |
|                                                                  |
|              Operation effectuee avec succes !                   |
|                                                                  |
+------------------------------------------------------------------+

4. ELEMENTS SVG
---------------
| Element   | Description                               |
|-----------|-------------------------------------------|
| polyline  | Trace du checkmark (points: 0,20 20,45 45,0)|
| ::before  | Pseudo-element pour l ombre               |

5. ATTRIBUTS COMPOSANT
----------------------
| Attribut         | Type    | Description                      |
|------------------|---------|----------------------------------|
| drawSuccessCheck | boolean | Declenche l animation            |

6. ANIMATION CSS
----------------
- Nom : succesDrawingCheckAnim
- Duree : 0.7s
- Effet : stroke-dashoffset de 80 a 0
- Resultat : Le check se dessine progressivement

7. STYLES LESS
--------------
@success-drawing-check-size: 50px
@success-drawing-check-stroke-color: #32acff (bleu OVH)
@success-drawing-check-stroke-width: 2
@success-drawing-check-animation-duration: 0.7s

8. STRUCTURE HTML GENEREE
-------------------------
<div class="success-drawing-check__icon draw">
  <svg width="50" height="50" viewBox="0 0 50 50">
    <polyline points="0,20 20,45 45,0" />
  </svg>
</div>
<div class="success-drawing-check__text">
  <!-- Contenu transclude -->
  Operation effectuee avec succes !
</div>

9. EXEMPLE D UTILISATION
------------------------
<tuc-success-drawing-check draw-success-check="$ctrl.showSuccess">
  Votre fichier a ete envoye avec succes !
</tuc-success-drawing-check>

// Dans le controleur
$ctrl.showSuccess = true; // Declenche l animation

10. CAS D USAGE
---------------
- Confirmation creation campagne
- Succes envoi de fichier
- Validation formulaire
- Fin de processus multi-etapes

11. ACCESSIBILITE
-----------------
- Texte transclude pour lecteurs d ecran
- Animation non bloquante
