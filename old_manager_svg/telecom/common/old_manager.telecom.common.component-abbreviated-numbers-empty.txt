# Cahier des Charges : telecom/common - Abbreviated Numbers Empty (Confirmation suppression)

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation
**Fonction:** Confirmer la suppression de tous les numéros abrégés
**Contexte:** Accessible depuis le menu Actions du datagrid des numéros abrégés

## 2. DIMENSIONS

- Modal: 500x220px
- Border radius: 8px
- Ombre: drop-shadow

## 3. HEADER MODAL

- Bouton fermer: "×" (position droite, 15px du bord)

## 4. CONTENU

### Titre
"Nettoyage de la liste des numéros abrégés" (16px, bold)

### Message
- Ligne 1: "L'ensemble de vos numéros abrégés va être supprimé."
- Ligne 2: "Voulez-vous continuer ?"

## 5. FOOTER MODAL

Position: translate(25, 160)

### Boutons

| Bouton | Style | Dimensions | Position |
|--------|-------|------------|----------|
| Annuler | btn-outline (bordure grise) | 90x36px | Gauche |
| Valider | btn-danger (#DC2626) | 90x36px | Droite (x=360) |

## 6. APIS & ENDPOINTS

```
# Supprimer tous les numéros abrégés
DELETE /telephony/{billingAccount}/abbreviatedNumber
→ { success: true, deletedCount: 15 }
```
