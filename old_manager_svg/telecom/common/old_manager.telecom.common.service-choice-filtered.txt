# Cahier des Charges : telecom/common - Service Choice Etat filtre

## 1. RESUME EXECUTIF

### Perimetre
Etat filtre de la modal de choix de service avec resultats.

### Objectif fonctionnel
- Afficher les resultats filtres
- Mettre en evidence les correspondances
- Permettre de filtrer par type

---

## 2. LAYOUT

### Structure modal avec resultats filtres + badges

---

## 3. COMPOSANTS UI

### Recherche active
| Element | Description |
|---------|-------------|
| Input | Avec terme recherche |
| Focus | Bordure bleue |
| Clear | Bouton X pour effacer |

### Info resultats
| Element | Description |
|---------|-------------|
| Texte | "3 resultats pour \"commercial\"" |
| Terme | En gras |

### Liste filtree
| Element | Description |
|---------|-------------|
| Groupes | Seulement ceux avec resultats |
| Highlight | Terme en surbrillance jaune |

### Service Item (filtre)
| Element | Description |
|---------|-------------|
| Radio | Selection |
| Titre | Avec terme surligne |
| Sous-titre | Type + numero |

### Astuce
| Element | Description |
|---------|-------------|
| Icone | Ampoule |
| Texte | Rechercher par nom ou numero |

### Filtres par type
| Element | Description |
|---------|-------------|
| Label | FILTRER PAR TYPE |
| Badges | Chips selectionnables |

### Badges de type
| Badge | Etat selectionne |
|-------|------------------|
| Lignes SIP | Fond bleu, bordure bleue |
| Numeros | Fond gris |
| Fax | Fond gris |
| Trunk | Fond gris |

### Resume selection
| Element | Description |
|---------|-------------|
| Fond | Vert clair |
| Titre | Service selectionne : |
| Info | Nom + numero |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Valider | Primary | Confirme |
| Annuler | Default | Ferme |

---

## 4. APIS & DONNEES

### Props
| Prop | Type | Description |
|------|------|-------------|
| searchTerm | string | Terme recherche |
| results | Service[] | Resultats filtres |
| selectedType | string | Type filtre |
| selected | Service | Service selectionne |
