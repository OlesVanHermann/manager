## 1. RESUME EXECUTIF

Modal ODS pour confirmation avant action importante ou destructive.

--------------------------------------------------------------------------------
## 2. UTILISATION
--------------------------------------------------------------------------------

Template :
<oui-modal
    data-heading="Confirmation"
    data-primary-action="$ctrl.confirm()"
    data-primary-label="Confirmer"
    data-secondary-action="$ctrl.cancel()"
    data-secondary-label="Annuler"
    data-on-dismiss="$ctrl.cancel()"
    data-type="warning">
    <p>Message de confirmation</p>
</oui-modal>

Controller :
confirm() { this.$uibModalInstance.close(true); }
cancel() { this.$uibModalInstance.dismiss('cancel'); }

Ouverture :
this.$uibModal.open({ template, controller, resolve: { data } })
    .result.then(() => { /* confirme */ })
    .catch(() => { /* annule */ });

--------------------------------------------------------------------------------
## 3. ATTRIBUTS
--------------------------------------------------------------------------------

data-heading : Titre modal
data-type : info, warning (defaut confirmations), error, success
data-primary-action/label : Bouton principal
data-secondary-action/label : Bouton secondaire
data-on-dismiss : Callback fermeture X/Echap
data-loading : Affiche spinner

--------------------------------------------------------------------------------
## 4. BONNES PRATIQUES
--------------------------------------------------------------------------------

Type warning pour actions destructives/irreversibles
Label explicite (Supprimer, Resilier, pas OK)
Message clair mentionnant element concerne
