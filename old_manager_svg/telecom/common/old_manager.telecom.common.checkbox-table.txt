## 1. RESUME EXECUTIF

Directive AngularJS pour selection multiple par checkbox dans tableau.
Gere selection par ligne, page, et globale pour actions groupees.

--------------------------------------------------------------------------------
## 2. UTILISATION
--------------------------------------------------------------------------------

Header (selection globale) :
<th>
    <input type="checkbox"
        data-ovh-checkbox-table
        data-ovh-checkbox-table-ids-all="$ctrl.items.raw"
        data-ovh-checkbox-table-ids-page="$ctrl.items.paginated"
        data-ovh-checkbox-table-ids-selected="$ctrl.items.selected"
        data-ovh-checkbox-table-id="id">
</th>

Body (selection ligne) :
<td><input type="checkbox" ng-model="$ctrl.items.selected[item.id]"></td>

--------------------------------------------------------------------------------
## 3. ATTRIBUTS
--------------------------------------------------------------------------------

data-ovh-checkbox-table : Active directive
data-ovh-checkbox-table-ids-all : Tous les elements
data-ovh-checkbox-table-ids-page : Elements page courante
data-ovh-checkbox-table-ids-selected : Objet selection { id: boolean }
data-ovh-checkbox-table-id : Propriete identifiant

--------------------------------------------------------------------------------
## 4. STRUCTURE DONNEES
--------------------------------------------------------------------------------

self.items = {
    raw: [],           // Source de verite
    paginated: null,   // Page courante
    selected: {},      // { 'id-123': true, ... }
};

Recuperation selection :
filter(raw, item => selected[item.id])

Reset apres action :
self.items.selected = {};
