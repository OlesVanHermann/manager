1. DESCRIPTION FONCTIONNELLE
================================================================================
Double champ mot de passe avec :
- Vérification de la force du mot de passe (popover)
- Confirmation du mot de passe
- Validation en temps réel

Cas d'usage :
- Création/modification mot de passe voicemail
- Configuration mot de passe SIP
- Changement mot de passe FTP

3. PROPRIETES (BINDINGS)
================================================================================
| Propriété      | Type   | Direction | Description                         |
|----------------|--------|-----------|-------------------------------------|
| ovhPwd         | string | =         | Valeur du mot de passe              |
| ovhPwdConfirm  | string | =         | Valeur de la confirmation           |

================================================================================
4. VALIDATION
================================================================================
Mot de passe :
- Minimum 8 caractères
- Au moins 1 chiffre
- Au moins 1 minuscule
- Au moins 1 majuscule
- Maximum 32 caractères
- Directive : tuc-ovh-password-strength-check

Confirmation :
- Doit correspondre au mot de passe (directive match)
- Maximum 32 caractères

Sécurité :
- Coller (paste) désactivé sur les deux champs
