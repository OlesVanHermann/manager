## 1. RESUME EXECUTIF

Composant ODS pour afficher un menu d'actions contextuel. Utilise dans
tableaux (datagrid) et pages de liste pour actions sur ligne ou globales.

--------------------------------------------------------------------------------
## 2. UTILISATION
--------------------------------------------------------------------------------

Dans datagrid (compact) :
<oui-action-menu data-compact data-placement="left-start">
    <oui-action-menu-item data-on-click="$ctrl.edit($row)">Modifier</oui-action-menu-item>
    <oui-action-menu-divider></oui-action-menu-divider>
    <oui-action-menu-item data-variant="error" data-on-click="$ctrl.delete($row)">Supprimer</oui-action-menu-item>
</oui-action-menu>

Dans topbar (avec texte) :
<oui-action-menu text="Actions">
    <oui-action-menu-item data-on-click="$ctrl.order()">Commander</oui-action-menu-item>
</oui-action-menu>

--------------------------------------------------------------------------------
## 3. COMPOSANTS
--------------------------------------------------------------------------------

oui-action-menu : Conteneur
  - text : Label bouton (si non compact)
  - data-compact : Mode icone seule
  - data-placement : Position menu

oui-action-menu-item : Action
  - data-on-click : Callback
  - data-variant : Style (default, error)
  - data-disabled : Desactive

oui-action-menu-divider : Separateur

--------------------------------------------------------------------------------
## 4. BONNES PRATIQUES
--------------------------------------------------------------------------------

Ordre : Actions principales > Separateur > Actions secondaires > Separateur > Actions destructives (error)
Conditions : ng-if pour masquer selon contexte
Placement datagrid : compact + left-start
