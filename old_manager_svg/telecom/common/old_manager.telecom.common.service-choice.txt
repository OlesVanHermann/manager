# Cahier des Charges : telecom/common - Composant Choix de service

## 1. RESUME EXECUTIF

### Perimetre
Modal de selection d'un service VoIP (ligne, numero, fax).

### Objectif fonctionnel
- Lister les services disponibles par groupe
- Permettre la recherche
- Selectionner un service

---

## 2. LAYOUT

### Structure modal avec recherche + liste groupee

---

## 3. COMPOSANTS UI

### Modal Header
| Element | Description |
|---------|-------------|
| Titre | Choix du service |

### Barre de recherche
| Element | Description |
|---------|-------------|
| Icone | Loupe |
| Placeholder | Rechercher... |
| Effacer | Bouton X pour vider |

### Liste des services
| Element | Description |
|---------|-------------|
| Groupes | Services regroupes par billingAccount |
| Radio | Selection exclusive |
| Nom | Nom/description du service |
| Detail | Type + numero |

### Groupes
| Element | Description |
|---------|-------------|
| Header | Nom du groupe (uppercase, gris) |
| Separateur | Ligne entre groupes |

### Service Item
| Element | Description |
|---------|-------------|
| Radio | Circle checked/unchecked |
| Titre | Nom du service |
| Sous-titre | Type + numero (italique) |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Valider | Primary | Confirme la selection |
| Annuler | Default | Ferme la modal |

---

## 4. APIS & DONNEES

### Types de service
| Type | Label |
|------|-------|
| line | Ligne SIP |
| number | Numero |
| fax | Fax |
| trunk | Ligne SIP Trunk |

### Endpoints
```
GET /telephony
    Response: [billingAccount1, billingAccount2, ...]

GET /telephony/{billingAccount}/line
GET /telephony/{billingAccount}/number
GET /telephony/{billingAccount}/fax
```
