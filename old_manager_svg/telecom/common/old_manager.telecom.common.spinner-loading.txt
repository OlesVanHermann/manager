# Cahier des Charges : telecom/common - Modal Spinner de chargement

## 1. RESUME EXECUTIF

### Perimetre
Modal de chargement avec spinner anime.

### Objectif fonctionnel
- Indiquer un chargement en cours
- Bloquer l'interaction utilisateur
- Afficher un message contextuel

---

## 2. LAYOUT

### Structure modal centree avec spinner

---

## 3. COMPOSANTS UI

### Modal Container
| Element | Description |
|---------|-------------|
| Overlay | Fond semi-transparent |
| Modal | Boite blanche centree |
| Dimensions | 400x300px |

### Spinner
| Element | Description |
|---------|-------------|
| Type | Cercle anime |
| Track | Gris clair (#E5E7EB) |
| Fill | Bleu primaire (#0050D7) |
| Taille | 80px diametre |

### Texte de chargement
| Element | Description |
|---------|-------------|
| Titre | Chargement en cours... |
| Detail | Message contextuel |

### Indicateur de progression
| Element | Description |
|---------|-------------|
| Type | Points animes (3 dots) |
| Animation | Opacite decroissante |

---

## 4. APIS & DONNEES

### Props
| Prop | Type | Description |
|------|------|-------------|
| message | string | Message principal |
| detail | string | Message secondaire |
| visible | boolean | Affiche/masque le loader |

## 6. REGLES METIER

- Bloque toute interaction
- Animation CSS fluide
- Timeout de securite (30s)
- Message adapte au contexte
