# Cahier des Charges : telecom/common - Abbreviated Numbers Modal (Ajout/Edition)

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter ou modifier un numéro abrégé
**Contexte:** Accessible depuis le bouton Ajouter ou le menu Actions d'une ligne

## 2. DIMENSIONS

- Modal: 500x380px
- Border radius: 8px

## 3. HEADER MODAL

- Titre: "Ajouter un numéro abrégé" (16px, bold)
- Variante édition: "Modifier le numéro abrégé"

## 4. FORMULAIRE

### Champs

| Champ | Type | Obligatoire | Validation |
|-------|------|-------------|------------|
| Numéro abrégé | Input text | Oui * | 3-4 chiffres, commence par 2 |
| Numéro cible | Input text | Oui * | Format international 00xx... |
| Nom | Input text | Oui * | - |
| Prénom | Input text | Oui * | - |

### Dimensions inputs
- Tous: 450x38px

### Messages d'aide (sous les champs)
- Numéro abrégé: "Doit être un nombre à 3 ou 4 chiffres, débutant par un 2."
- Numéro cible: "Doit être un numéro de format international. Ex. 00xx..."

### Style champ actif
- Bordure: #0050D7 (2px)

## 5. FOOTER MODAL

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 90x36px |
| Ajouter | btn-primary | 90x36px |

Variante édition: "Enregistrer" au lieu de "Ajouter"

## 6. APIS & ENDPOINTS

```
# Ajouter un numéro abrégé
POST /telephony/{billingAccount}/abbreviatedNumber
Body: {
  abbreviatedNumber: "200",
  destinationNumber: "0033123456789",
  surname: "Dupont",
  name: "Marie"
}
→ {
  abbreviatedNumber: "200",
  destinationNumber: "0033123456789",
  surname: "Dupont",
  name: "Marie"
}

# Modifier un numéro abrégé
PUT /telephony/{billingAccount}/abbreviatedNumber/{abbreviatedNumber}
Body: {
  destinationNumber: "0033987654321",
  surname: "Martin",
  name: "Pierre"
}

# Supprimer un numéro abrégé
DELETE /telephony/{billingAccount}/abbreviatedNumber/{abbreviatedNumber}
```
