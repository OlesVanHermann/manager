================================================================================
CAHIER DES CHARGES - old_manager.telecom.common.gauge
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/common
Composant  : Jauge analogique
Type       : Composant visualisation SVG
Framework  : AngularJS 1.7.x
Module     : telecom-universe-components

2. DESCRIPTION FONCTIONNELLE
----------------------------
Composant SVG affichant une jauge analogique avec aiguille.
Utilisee pour visualiser des valeurs comme le debit, la qualite
de signal ou tout indicateur sur une echelle 0-100%.

3. STRUCTURE UI
---------------
        ___________
       /   |   |   \
      /  0 | 50| 100\
     |     |   |     |
     |    \|/        |
      \   (o)       /
       \___________/

Representation semi-circulaire avec :
- Cadran gradue (0, 45, 90, 135, 180 degres)
- Aiguille rotative indiquant la valeur
- Marques de graduation

4. ELEMENTS SVG
---------------
| Element      | Classe        | Description                    |
|--------------|---------------|--------------------------------|
| path frame   | gauge frame   | Arc du cadran (fond)           |
| path mk0     | gauge mark    | Graduation 0%                  |
| path mk45    | gauge mark    | Graduation 25%                 |
| path mk90    | gauge mark    | Graduation 50%                 |
| path mk135   | gauge mark    | Graduation 75%                 |
| path mk180   | gauge mark    | Graduation 100%                |
| path cursor  | gauge-cursor  | Aiguille de la jauge           |

5. ATTRIBUTS COMPOSANT
----------------------
| Attribut | Type   | Description                          |
|----------|--------|--------------------------------------|
| value    | number | Valeur a afficher (0-100)            |
| min      | number | Valeur minimale (defaut: 0)          |
| max      | number | Valeur maximale (defaut: 100)        |

6. TRANSFORMATION AIGUILLE
--------------------------
- Rotation de l aiguille basee sur la valeur
- Angle 0 = valeur min (gauche)
- Angle 180 = valeur max (droite)
- transform="rotate(X)" ou X = valeur * 180 / max

7. DIMENSIONS
-------------
- Viewbox : echelle 0.2
- Centre : translate(220, 220)
- Rayon cadran : 200px
- Longueur graduations : 40px

8. STYLES CSS
-------------
.gauge { }
.gauge.frame { stroke: #ccc; fill: none; }
.gauge.mark { stroke: #999; }
.gauge-cursor { fill: #333; }

9. EXEMPLE D UTILISATION
------------------------
<tuc-gauge value="$ctrl.signalStrength"></tuc-gauge>

<!-- Avec min/max personnalises -->
<tuc-gauge
    value="$ctrl.bandwidth"
    min="0"
    max="1000"
></tuc-gauge>

10. CAS D USAGE
---------------
- Qualite de signal telephonique
- Debit internet
- Niveau de batterie
- Indicateur de performance
