NAV1 : telecom
NAV2 : common
================================================================================

## 1. RESUME EXECUTIF

Systeme pagination integre au datagrid ODS. Navigation pages, controle
elements par page, lazy loading.

--------------------------------------------------------------------------------
## 2. MODES
--------------------------------------------------------------------------------

Backend (lazy loading) :
<oui-datagrid
    data-page="{{ $ctrl.page }}"
    data-page-size="{{ $ctrl.pageSize }}"
    data-rows-loader="$ctrl.loadPage($config)"
    data-on-page-change="$ctrl.onPageChange($pagination)">
</oui-datagrid>

Frontend (donnees en memoire) :
<oui-datagrid
    data-rows="$ctrl.allData"
    data-pagination-front
    data-page-size="25">
</oui-datagrid>

--------------------------------------------------------------------------------
## 3. ATTRIBUTS
--------------------------------------------------------------------------------

data-page : Page courante (1-indexed)
data-page-size : Elements par page
data-pagination-front : Active pagination client
data-on-page-change : Callback ($pagination.page, $pagination.pageSize)

--------------------------------------------------------------------------------
## 4. LOADER BACKEND
--------------------------------------------------------------------------------

$config : { offset, pageSize, sort, criteria }
Retour attendu : { data: [...], meta: { totalCount } }

--------------------------------------------------------------------------------
## 5. BONNES PRATIQUES
--------------------------------------------------------------------------------

Backend : >100 elements
Frontend : <100 elements
Tailles recommandees : 10, 25, 50, 100
