================================================================================
CAHIER DES CHARGES - old_manager.telecom.common.resiliation
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/common
Composant  : Formulaire resiliation
Type       : Composant formulaire
Framework  : AngularJS 1.7.x
Module     : telecom-universe-components

2. DESCRIPTION FONCTIONNELLE
----------------------------
Composant generique de resiliation d un service telecom.
Affiche les avertissements, demande la raison et confirmation.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| Resiliation de mon abonnement                                    |
|                                                                  |
| [!] Je comprends que la resiliation de mon offre entrainera      |
|     la coupure de mon acces internet, ainsi que celle de tous    |
|     les produits et services dont je n ai pas explicitement      |
|     demande la conservation.                                     |
|                                                                  |
| [!] Je comprends que le cas echeant, je serai redevable de la    |
|     somme correspondante au prorata des mensualites dues         |
|     jusqu a la fin de ma periode d engagement.                   |
|                                                                  |
| Pour quelle raison voulez-vous resilier ?                        |
| [v Selectionnez une raison___________________]                   |
|                                                                  |
| (Si "Autre" selectionne)                                         |
| Precisions :                                                     |
| [________________________________________________________]       |
|                                                                  |
+------------------------------------------------------------------+
| [Annuler]                                    [Je resilie]        |
+------------------------------------------------------------------+

4. RAISONS DE RESILIATION
-------------------------
| Code               | Label FR                                   |
|--------------------|-------------------------------------------|
| addressMove        | Demenagement                               |
| billingProblems    | Systeme de facturation non adapte         |
| cessationOfActivity| Cessation d activite                       |
| changeOfTerms      | Changement des CGU                         |
| ftth               | Passage a la fibre (FTTH)                  |
| goToCompetitor     | Passage a une offre concurrente            |
| technicalProblems  | Problemes techniques                       |
| other              | Autre                                      |

5. CHAMPS FORMULAIRE
--------------------
| Champ   | Type     | Requis | Description                       |
|---------|----------|--------|-----------------------------------|
| reason  | select   | Oui    | Raison de resiliation             |
| details | textarea | Cond.  | Details si reason=other           |
| confirm | checkbox | Oui    | Confirmation lecture avertissements|

6. ATTRIBUTS COMPOSANT
----------------------
| Attribut  | Type     | Description                         |
|-----------|----------|-------------------------------------|
| typeName  | string   | Type de service (ex: "Pack")        |
| name      | string   | Nom du service                      |
| onSubmit  | function | Callback lors de la resiliation     |
| onCancel  | function | Callback annulation                 |

7. ETATS
--------
- Formulaire : Affichage des avertissements et selection raison
- Validation : Verification checkbox et raison
- Soumission : Appel callback onSubmit

8. TRADUCTIONS FR
-----------------
resiliation_confirm_title: "Resiliation de mon abonnement"
resiliation_message1: "Je comprends que la resiliation entrainera la coupure..."
resiliation_message2: "Je comprends que je serai redevable du prorata..."
resiliation_reason: "Pour quelle raison voulez-vous resilier ?"
resiliation_confirm: "Je resilie"
resiliation_common_cancel: "Annuler"
resiliation_cancel: "Annuler la resiliation en cours"

9. ACTIONS UTILISATEUR
----------------------
- Annuler : Ferme sans resilier
- Je resilie : Soumet la demande de resiliation
