1. DESCRIPTION FONCTIONNELLE
================================================================================
Animation SVG de validation (checkmark) avec effet de dessin progressif.
Affiche une coche verte animée pour confirmer le succès d'une action.

Cas d'usage :
- Confirmation de commande validée
- Succès d'une opération de sauvegarde
- Fin d'un processus en plusieurs étapes
