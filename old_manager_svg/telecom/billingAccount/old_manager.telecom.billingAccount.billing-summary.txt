# Cahier des Charges : telecom/billingAccount - Récapitulatif de consommation

## 1. VUE D'ENSEMBLE

**Écran** : Récapitulatif de consommation du groupe
**Objectif** : Visualiser la consommation détaillée par service
**Contexte** : Accessible depuis le menu Facturation

## 3. COMPOSANTS UI

### Lien retour
- Texte bleu avec flèche ←
- Navigation vers page facturation

### Card Résumé
- 4 métriques en colonnes
- Label muted en haut
- Valeur grande (18px) en gras

### Tableau Détail
- Header gris (#F9FAFB)
- Colonnes : Service, Total, Entrants, Sortants, Transferts
- Ligne service avec nom + numéro de téléphone
- Pagination en bas à droite

## 4. DONNÉES & COLONNES

### Résumé
| Métrique | Type | Description |
|----------|------|-------------|
| Total des appels | number | Somme de tous les appels |
| Appels entrants | number | Appels reçus |
| Appels sortants | number | Appels émis |
| Transferts | number | Appels transférés |

### Tableau Détail
| Colonne | Type | Description |
|---------|------|-------------|
| Service | composite | Nom + numéro (2 lignes) |
| Total | number | Total appels du service |
| Entrants | number | Appels reçus |
| Sortants | number | Appels émis |
| Transferts | number | Appels transférés |

## 5. ACTIONS & INTERACTIONS

### Navigation
- **← Retour** → `/telecom/{billingAccount}/billing`

### Pagination
- Boutons ‹ › pour navigation

## 6. APIS & ENDPOINTS

### GET (lecture)
```
GET /telephony/{billingAccount}/service
GET /telephony/{billingAccount}/service/{serviceName}/voiceConsumption
GET /telephony/{billingAccount}/service/{serviceName}/voiceConsumption/summary
```

### Paramètres
```
?dateFrom=YYYY-MM-DD
?dateTo=YYYY-MM-DD
?wayType=incoming|outgoing|transfer
```
