================================================================================
CAHIER DES CHARGES - old_manager.telecom.sms.batches-history
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/sms
Composant  : Historique campagnes SMS
Type       : Page liste historique
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Page affichant l historique complet des campagnes SMS envoyees.
Permet de consulter les anciennes campagnes avec filtres par date.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| < Retour                                                         |
| Historique des campagnes                                         |
+------------------------------------------------------------------+
|                                                                  |
| Filtres :                                                        |
| Date debut [__/__/____] Date fin [__/__/____] [Filtrer]          |
|                                                                  |
| +--------------------------------------------------------------+ |
| | Date       | Nom         | Expediteur | Envoyes | Delivres   | |
| |------------|-------------|------------|---------|------------| |
| | 15/12/2024 | Promo Noel  | OVH        | 450     | 445        | |
| | 10/12/2024 | Newsletter  | NEWS       | 320     | 315        | |
| | 05/12/2024 | Rappel      | CABINET    | 180     | 178        | |
| +--------------------------------------------------------------+ |
|                                                                  |
| Actions : [Supprimer]                                            |
|                                                                  |
+------------------------------------------------------------------+

4. COLONNES TABLEAU
-------------------
| Colonne    | Description                         |
|------------|-------------------------------------|
| Date       | Date d envoi de la campagne         |
| Nom        | Nom de la campagne                  |
| Expediteur | Expediteur utilise                  |
| Envoyes    | Nombre de SMS envoyes               |
| Delivres   | Nombre de SMS livres                |

5. FILTRES DISPONIBLES
----------------------
- Date debut : Filtrer depuis une date
- Date fin : Filtrer jusqu a une date

6. ETATS
--------
- Chargement : Spinner pendant recuperation
- Affichage : Tableau historique
- Vide : "Aucune campagne"

7. ENDPOINTS API
----------------
| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName}/batches?status=COMPLETED | Historique |
| DELETE | /sms/{serviceName}/batches/{id} | Supprimer |

8. COMPOSANTS UTILISES
----------------------
- oui-datagrid : Tableau de donnees
- oui-datepicker : Filtres date
- oui-spinner : Chargement

9. TRADUCTIONS FR
-----------------
sms_batches_history_title: "Historique des campagnes"
sms_batches_none: "Aucune campagne"

10. ACTIONS UTILISATEUR
-----------------------
- Filtrer : Applique les filtres de date
- Supprimer : Supprime une campagne de l historique
