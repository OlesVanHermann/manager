## 1. INFORMATIONS GENERALES

| Champ | Valeur |
|-------|--------|
| Titre page | Ajouter un expediteur |
| Breadcrumb | SMS > Expediteurs > Ajouter |
| URL state | sms.service.senders.add |

--------------------------------------------------------------------------------
## 2. MODES D AJOUT
--------------------------------------------------------------------------------

Trois modes disponibles via onglets/choix :

1. **Ajouter manuellement des expediteurs**
2. **Ajouter a partir de donnees personnelles** (NIC handle)
3. **Ajouter a partir de vos noms de domaine OVH**

--------------------------------------------------------------------------------
## 3. INFORMATIONS TARIFAIRES
--------------------------------------------------------------------------------

- Tarification : Validation d un nom = Gratuit
- Horaires validation manuelle : Du lundi au vendredi de 8h a 20h
- Delai : 1 a 2 jours ouvres

--------------------------------------------------------------------------------
## 4. AVERTISSEMENTS
--------------------------------------------------------------------------------

- Les operateurs mobiles interdisent les numeros de telephone comme expediteur
- Alternative : expediteur alphabetique ou numero court
- Domaines : seuls les domaines de 11 caracteres ou moins sont compatibles
- NIC handle : certains telephones n affichent pas la ponctuation

--------------------------------------------------------------------------------
## 5. FORMULAIRE MANUEL
--------------------------------------------------------------------------------

| Champ | Type | Validation |
|-------|------|------------|
| Expediteur | Input | Requis, alphabetique, max 11 car |
| Description | Input | Optionnel, max 40 car |
| Courte justification | Textarea | Requis, max 250 car |
| Pieces justificatives | File upload | Max 10 fichiers, 10Mo, pdf/jpg/jpeg/png |

**Note pieces justificatives:**
- Papier en-tete societe/marque
- Autorisation responsable avec signature et tampon
- Papier identite ou extrait Kbis

--------------------------------------------------------------------------------
## 6. FORMULAIRE DONNEES PERSONNELLES
--------------------------------------------------------------------------------

- Liste des donnees personnelles disponibles (NIC)
- Selection multiple possible
- Etat vide : "Aucune donnee personnelle disponible."

--------------------------------------------------------------------------------
## 7. FORMULAIRE DOMAINES OVH
--------------------------------------------------------------------------------

- Liste des domaines OVH du compte
- Filtre : domaines <= 11 caracteres
- Selection multiple possible
- Etat vide : "Aucun nom de domaine disponible."

--------------------------------------------------------------------------------
## 8. APIS UTILISEES
--------------------------------------------------------------------------------

| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName}/sendersAvailableForValidation | Expediteurs disponibles |
| POST | /sms/{serviceName}/senders | Creer expediteur |
| GET | /domain | Liste domaines OVH |
| GET | /me | Donnees personnelles |

--------------------------------------------------------------------------------
## 9. ACTIONS
--------------------------------------------------------------------------------

- Ajouter l expediteur selectionne
- Ajouter les X expediteurs selectionnes (selection multiple)
