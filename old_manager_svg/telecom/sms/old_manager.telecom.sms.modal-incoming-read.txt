# Cahier des Charges : telecom/sms - Modal détails SMS reçu

## 1. VUE D'ENSEMBLE
- **Type** : Modal de lecture
- **Contexte** : Page SMS reçus
- **Objectif** : Afficher les détails d'un SMS reçu

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Informations
- **OdsCard** : Message
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détails du SMS reçu" |

## 5. INFORMATIONS AFFICHÉES
| Champ | Format |
|-------|--------|
| Date de réception | DD/MM/YYYY HH:mm:ss |
| Expéditeur | Numéro international |
| Crédits utilisés | "X SMS (Y crédit)" |

## 6. ZONE MESSAGE
| Propriété | Valeur |
|-----------|--------|
| Background | Gris clair (#F9FAFB) |
| Border | Bordure grise |
| Contenu | Texte complet du SMS |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary | Ferme modal |

## 8. APIS & ENDPOINTS
- GET /sms/{serviceName}/incoming/{id} - Détail SMS reçu
