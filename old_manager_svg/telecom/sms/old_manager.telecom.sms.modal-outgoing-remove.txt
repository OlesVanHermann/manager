# Cahier des Charges : telecom/sms - Modal supprimer SMS envoyé

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page SMS envoyés
- **Objectif** : Supprimer un SMS de l'historique des envois

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Détails du SMS
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le SMS envoyé" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Confirmez-vous la suppression de ce SMS ?" |

## 6. DÉTAILS SMS
| Champ | Format |
|-------|--------|
| Date | DD/MM/YYYY HH:mm:ss |
| Expéditeur | Sender ID |
| Destinataire | Numéro international |
| Code tarifaire | Zone tarifaire |
| Message | Texte tronqué |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le SMS |

## 8. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/outgoing/{id} - Supprimer SMS envoyé
