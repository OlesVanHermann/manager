## 3. FORMULAIRE
--------------------------------------------------------------------------------

| Champ | Type | Validation |
|-------|------|------------|
| Activer l alerte ? | Toggle/Checkbox | - |
| Seuil d alerte | Input number | Requis si active, min 1 |
| Notification | Select | both, mail, sms |
| E-mail | Input email | Requis si mail/both |
| Numero | Input tel | Requis si sms/both, format international |

**Options Notification:**
- Mail et SMS (both)
- Mail (mail)
- SMS (sms)

**Erreurs:**
- Seuil requis : "Le champ seuil d alerte est obligatoire."
- Seuil min : "La valeur minimum du seuil d alerte doit etre de 1."
- Email requis : "Le champ adresse e-mail est obligatoire."
- Email invalide : "Adresse e-mail non valide."
- Numero requis : "Le champ numero est obligatoire."
- Numero invalide : "Veuillez saisir un numero valide."
