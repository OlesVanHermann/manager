# Cahier des Charges : telecom/sms - Modal supprimer historique

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page historique des envois
- **Objectif** : Supprimer un envoi de l'historique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Nom de la campagne
- **OdsMessage** : Info
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer de l'historique" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir supprimer cet envoi de l'historique ?" |
| Nom campagne | Affiché dans une card grise |

## 6. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Cet envoi sera supprimé de la liste mais les SMS ont déjà été envoyés." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime de l'historique |

## 8. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/outgoing/{outgoingId} - Supprimer de l'historique
