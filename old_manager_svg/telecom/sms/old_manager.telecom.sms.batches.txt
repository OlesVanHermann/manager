================================================================================
CAHIER DES CHARGES - old_manager.telecom.sms.batches
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/sms
Composant  : Page gestion des campagnes SMS
Type       : Page liste avec actions
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Page principale de gestion des campagnes SMS. Affiche la liste des campagnes
avec leurs statuts et permet d acceder aux sous-sections : creation, statistiques,
historique, campagnes planifiees.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| Gestion des campagnes                                            |
+------------------------------------------------------------------+
| [Creer une campagne] [Statistiques et historique]                |
| [SMS recus] [Campagnes planifiees]                               |
+------------------------------------------------------------------+
|                                                                  |
| +--------------------------------------------------------------+ |
| | Nom            | Message  | Date/heure | Expediteur | Statut | |
| |----------------|----------|------------|------------|--------| |
| | Promo Noel     | -20% ... | 15/12 10h  | OVH        | [v]    | |
| | Rappel RDV     | Votre... | 16/12 09h  | CABINET    | [v]    | |
| | Newsletter     | Decouvr..| 17/12 14h  | NEWS       | [!]    | |
| +--------------------------------------------------------------+ |
|                                                                  |
| Actions par campagne : [Details] [Telecharger] [Supprimer]       |
|                                                                  |
+------------------------------------------------------------------+

4. COLONNES TABLEAU
-------------------
| Colonne     | Description                        |
|-------------|------------------------------------|
| Nom         | Nom de la campagne                 |
| Message     | Apercu du message SMS              |
| Date/heure  | Date et heure d envoi              |
| Expediteur  | Nom ou numero expediteur           |
| Statut      | Etat de la campagne                |

5. STATUTS CAMPAGNE
-------------------
| Status     | Label FR                              |
|------------|---------------------------------------|
| PENDING    | Planifiee                             |
| INSERTING  | Ajout des SMS dans la base de donnees |
| INSERTED   | SMS inseres dans la base de donnees   |
| COMPLETED  | Terminee                              |
| TO_CANCEL  | A supprimer                           |
| CANCELING  | En cours de suppression               |
| CANCELED   | Supprimee                             |
| FAILED     | Erreur                                |

6. NAVIGATION
-------------
| Action                    | Destination                    |
|---------------------------|--------------------------------|
| Creer une campagne        | /batches/create                |
| Statistiques et historique| /batches/statistics            |
| SMS recus                 | /sms/incoming                  |
| Campagnes planifiees      | /batches/pending               |
| Details                   | /batches/{id}                  |

7. ENDPOINTS API
----------------
| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName}/batches | Liste campagnes |
| DELETE | /sms/{serviceName}/batches/{id} | Supprimer campagne |
| GET | /sms/{serviceName}/batches/{id}/statistics | Stats campagne |

8. COMPOSANTS UTILISES
----------------------
- oui-datagrid : Tableau de donnees
- oui-action-menu : Menu actions par ligne
- oui-spinner : Chargement

10. ACTIONS UTILISATEUR
-----------------------
- Creer une campagne : Ouvre formulaire creation
- Voir le detail : Affiche details campagne
- Telecharger : Export CSV des resultats
- Supprimer : Supprime la campagne
