# Cahier des Charges : telecom/sms - Modal modifier URLs callback

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Page options SMS
- **Objectif** : Configurer les URLs de callback pour les événements SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : URLs callback
- **OdsMessage** : Info utilisation
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier les URLs de callback" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Description |
|-------|------|-------------|
| URL de callback | Input URL | Notification envoi/réception |
| URL de callback STOP | Input URL | Notification demande STOP |

## 6. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Ces URLs seront appelées pour notifier votre serveur des événements SMS (accusé de réception, demande STOP)." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre les URLs |

## 8. APIS & ENDPOINTS
- GET /sms/{serviceName} - Configuration actuelle
- PUT /sms/{serviceName} - Modifier configuration (callBack, stopCallBack)
