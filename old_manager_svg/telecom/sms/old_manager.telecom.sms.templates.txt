## 2. CONTEXTE FONCTIONNEL
--------------------------------------------------------------------------------

Les modeles sont OBLIGATOIRES pour envoyer des SMS vers les Etats-Unis.
Seuls les messages transactionnels, alerte et 2FA sont autorises.
Aucun SMS publicitaire accepte.

Les variables dans les modeles sont delimitees par # (ex: #CODE#)

Modeles de demonstration disponibles sans validation :
- Your security code is #CODE#.
- The #OBJECT# of ID #ID# is in fail state: #ERROR#.
- Your package number #PACKAGENUMBER# has been delivered to your address #ADDRESS#.
