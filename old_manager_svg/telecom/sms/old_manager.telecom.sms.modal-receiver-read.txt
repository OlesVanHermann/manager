# Cahier des Charges : telecom/sms - Modal consulter liste destinataires

## 1. VUE D'ENSEMBLE
- **Type** : Modal de consultation
- **Contexte** : Page listes de destinataires
- **Objectif** : Consulter les contacts d'une liste de destinataires

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Info troncature
- **OdsDatagrid** : Tableau des contacts
- **OdsButton** : Exporter CSV, Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Consulter la liste de destinataires" |

## 5. INFO LISTE
| Élément | Format |
|---------|--------|
| Nom liste | "Liste : {nom}" en bold |

## 6. MESSAGE TRONCATURE
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Affichage des 10 premiers contacts sur X au total" |

## 7. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Numéro | Téléphone | Format international |
| Nom | Texte | Nom complet |
| Données | Texte | Données personnalisées |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Exporter CSV | Secondary | Télécharge fichier CSV |
| Fermer | Primary | Ferme modal |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/receivers/{slotId} - Détail liste
- GET /sms/{serviceName}/receivers/{slotId}/csv - Export CSV
