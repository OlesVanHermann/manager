# Cahier des Charges : telecom/sms - Modal ajouter listes destinataires

## 1. VUE D'ENSEMBLE
- **Type** : Modal de sélection
- **Contexte** : Page composition SMS
- **Objectif** : Ajouter des listes de destinataires prédéfinies

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCheckbox** : Sélection des listes
- **OdsMessage** : Résumé sélection
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter des listes de destinataires" |

## 5. DESCRIPTION
| Élément | Texte |
|---------|-------|
| Texte | "Sélectionnez les listes à ajouter au message :" |

## 6. LISTE DES LISTES
| Élément | Description |
|---------|-------------|
| Checkbox | Sélection |
| Nom de la liste | Texte |
| Compteur | "(X destinataires)" en gris |

## 7. MESSAGE RÉSUMÉ
| Propriété | Valeur |
|-----------|--------|
| Type | Success (vert) |
| Icône | ✓ |
| Format | "X liste(s) sélectionnée(s) - Y destinataires au total" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Ajoute listes sélectionnées |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/receivers - Liste des listes de destinataires
- GET /sms/{serviceName}/receivers/{slotId} - Détail d'une liste
