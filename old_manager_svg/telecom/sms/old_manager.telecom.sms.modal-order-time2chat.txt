# Cahier des Charges : telecom/sms - Modal commander Time2Chat

## 1. VUE D'ENSEMBLE
- **Type** : Modal de commande
- **Contexte** : Page expéditeurs
- **Objectif** : Commander un expéditeur Time2Chat pour recevoir des réponses

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 640px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs texte
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander Time2Chat" |

## 5. DESCRIPTION
| Élément | Texte |
|---------|-------|
| Texte | "Configurez votre expéditeur Time2Chat pour recevoir des réponses SMS." |

## 6. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom commercial | Input | Oui * |
| Numéro de téléphone | Input tel | Oui * |
| Site web | Input URL | Oui * |
| Nom de l'entreprise | Input | Oui * |
| Nom de la marque | Input | Oui * |
| Cas d'usage principal | Input | Oui * |
| Nom du contact opérationnel | Input | Oui * |
| E-mail du contact | Input email | Oui * |

## 7. LAYOUT
| Zone | Position |
|------|----------|
| Nom commercial | Pleine largeur |
| Numéro de téléphone | Pleine largeur |
| Site web | Pleine largeur |
| Nom entreprise | Pleine largeur |
| Nom marque | Pleine largeur |
| Cas d'usage | Pleine largeur |
| Contact | Deux colonnes (nom | email) |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Commander | Primary | Soumet la commande |

## 9. APIS & ENDPOINTS
- POST /sms/{serviceName}/sender/time2chat - Commander Time2Chat
