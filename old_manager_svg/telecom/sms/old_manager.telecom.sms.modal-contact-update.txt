# Cahier des Charges : telecom/sms - Modal modifier contact

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification
- **Contexte** : Page carnet d'adresses
- **Objectif** : Modifier les informations d'un contact

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 540px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs texte
- **OdsSelect** : Groupe
- **OdsMessage** : Info format numéro
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le contact" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Position |
|-------|------|--------|----------|
| Nom | Input | Oui * | Gauche |
| Prénom | Input | Oui * | Droite |
| Groupe | Select | Non | Pleine largeur |

## 6. MESSAGE INFO NUMÉROS
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Au moins un numéro de téléphone doit être renseigné au format international (+33...)" |

## 7. CHAMPS TÉLÉPHONE
| Champ | Type | Position |
|-------|------|----------|
| Téléphone domicile | Input | Gauche |
| Mobile personnel | Input | Droite |
| Téléphone travail | Input | Gauche |
| Mobile professionnel | Input | Droite |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde modifications |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/phonebook/{bookId}/contact/{contactId} - Détail contact
- PUT /sms/{serviceName}/phonebook/{bookId}/contact/{contactId} - Modifier contact
