================================================================================
CAHIER DES CHARGES - old_manager.telecom.sms.modal-sender-remove
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/sms
Composant  : Modal suppression expediteur
Type       : Modal confirmation
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Modal de confirmation pour supprimer un expediteur SMS.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| [X] Fermer                                                       |
+------------------------------------------------------------------+
| Supprimer l expediteur                                           |
|                                                                  |
| Voulez-vous vraiment supprimer l expediteur                      |
| "MONENTREPRISE" ?                                                |
|                                                                  |
+------------------------------------------------------------------+
| [Annuler]                                    [Supprimer]         |
+------------------------------------------------------------------+

Etat suppression :
+------------------------------------------------------------------+
|                        [Spinner]                                 |
|           Suppression de l expediteur en cours...                |
+------------------------------------------------------------------+

Etat succes :
+------------------------------------------------------------------+
|                     [Check anime]                                |
|                  Expediteur supprime                             |
+------------------------------------------------------------------+

4. ETATS
--------
1. Confirmation : Message de confirmation
2. Suppression : oui-spinner + message
3. Succes : tuc-success-drawing-check + confirmation

5. ENDPOINTS API
----------------
| Methode | Endpoint | Description |
|---------|----------|-------------|
| DELETE | /sms/{serviceName}/senders/{sender} | Supprimer expediteur |

6. COMPOSANTS UTILISES
----------------------
- oui-spinner : Indicateur suppression
- tuc-success-drawing-check : Animation succes

7. TRADUCTIONS FR
-----------------
sms_senders_remove_sender_title: "Supprimer l expediteur"
sms_senders_remove_sender_confirm: "Voulez-vous vraiment supprimer l expediteur {{sender}} ?"
sms_senders_remove_sender_removing: "Suppression de l expediteur en cours..."
sms_senders_remove_sender_removed: "Expediteur supprime"

8. ACTIONS UTILISATEUR
----------------------
- Annuler : Ferme modal sans action
- Supprimer : Supprime l expediteur
