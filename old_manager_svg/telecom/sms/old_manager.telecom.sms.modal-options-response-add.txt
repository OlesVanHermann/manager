# Cahier des Charges : telecom/sms - Modal ajouter suivi réponses

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page options SMS
- **Objectif** : Ajouter une option de suivi des réponses SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type de média
- **OdsInput** : Email expéditeur, Email destinataire
- **OdsMessage** : Info
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une option de suivi des réponses" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Options |
|-------|------|--------|---------|
| Type de média | Select | Oui | E-mail, SMS, HTTP |
| E-mail expéditeur | Input email | Oui * | noreply@... |
| E-mail destinataire | Input email | Oui * | support@... |

## 6. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Les réponses SMS seront transférées à l'adresse e-mail spécifiée." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Crée l'option de suivi |

## 8. APIS & ENDPOINTS
- POST /sms/{serviceName}/smsResponse - Créer suivi réponses
