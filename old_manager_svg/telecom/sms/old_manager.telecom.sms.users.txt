## 1. RÉSUMÉ EXÉCUTIF

### Périmètre
Page de gestion des utilisateurs API d'un compte SMS.

## 5. APIs & DONNÉES

### Endpoints probables
| Méthode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName}/users | Liste utilisateurs |
| POST | /sms/{serviceName}/users | Création |
| DELETE | /sms/{serviceName}/users/{login} | Suppression |
| POST | /sms/{serviceName}/users/{login}/password | Reset password |
