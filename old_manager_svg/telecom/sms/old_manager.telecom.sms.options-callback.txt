## 2. CONTEXTE FONCTIONNEL
--------------------------------------------------------------------------------

Permet de configurer des URLs de callback pour etre averti lors d evenements SMS.

Deux types de callbacks :
1. **Callback changement d etat** : Quand un SMS change d etat (envoye -> remis)
2. **Callback STOP** : Quand un destinataire repond "STOP" pour se blacklister
