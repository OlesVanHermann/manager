# Cahier des Charges : telecom/sms - Modal transférer crédits

## 1. VUE D'ENSEMBLE
- **Type** : Modal de transfert
- **Contexte** : Page gestion crédits
- **Objectif** : Transférer des crédits entre comptes SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Comptes source et destination
- **OdsInput** : Nombre de crédits
- **OdsMessage** : Info transfert
- **OdsButton** : Annuler, Transférer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Transférer des crédits SMS" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Votre compte SMS | Select | Oui * | Compte source avec solde |
| Compte à créditer | Select | Oui * | Compte destination avec solde |
| Nombre de crédits | Input numérique | Oui * | Max affiché |

## 6. FORMAT COMPTE
| Format |
|--------|
| "{serviceName} ({credits} crédits)" |

## 7. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Le transfert est immédiat et ne peut pas être annulé." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Transférer | Primary | Effectue le transfert |

## 9. APIS & ENDPOINTS
- GET /sms - Liste des comptes SMS
- POST /sms/{serviceName}/transferCredits - Transférer crédits
