# Cahier des Charges : telecom/xdsl - Acces Internet xDSL

## 1. RESUME EXECUTIF

### Perimetre
Page detaillee de l'acces Internet xDSL avec toutes les informations techniques.

### Objectif fonctionnel
- Afficher l'etat complet de la ligne
- Configurer les parametres de connexion
- Gerer les IPs
- Voir les caracteristiques de la ligne

## 3. COMPOSANTS UI

### Breadcrumb
| Element | Description |
|---------|-------------|
| Chemin | Telecom > Pack Internet > pack-xdsl-abc123 > xdsl-abc123-1 |

### Titre
| Element | Description |
|---------|-------------|
| Titre | Acces Internet xDSL |
| Identifiant | xdsl-abc123-1 |

### Tuile Informations generales
| Element | Description |
|---------|-------------|
| Etat | Actif - Ligne synchronisee (point vert) |
| Lien details | Plus de details sur l'etat |
| Diagnostic | Bouton lancer l'outil diagnostic |
| Envoi emails | Autorise (SMTP active) |
| Type | VDSL |
| Degroupage | Total + bouton |

### Tuile Connexion
| Element | Description |
|---------|-------------|
| Synchronisation | Down/Up Kbps |
| Type connexion | PPPoE |
| Profil connexion | Standard + bouton config |
| Port DSLAM | Port X - Slot Y + bouton reset |

### Tuile IPs
| Element | Description |
|---------|-------------|
| IPv4 | Adresse + Reverse DNS |
| IPv6 | Adresse + Config |
| Commander | Bouton commander des IPs |

### Tuile Caracteristiques de la ligne
| Champ | Exemple |
|-------|---------|
| NRA | PAR75-ABC |
| Distance NRA | 1 250 metres |
| Attenuation | 18.5 dB |
| Type cable | 4/10mm : 850m, 6/10mm : 400m |
| LNS | lns-rbx1.ovh.net |
| GTR | 4 heures ouvrees (4HO) |

### Tuile Alertes et notifications
| Element | Description |
|---------|-------------|
| Texte | Configurez des alertes pour etre notifie... |
| Bouton | Gerer les alertes |

## 4. APIS & DONNEES

### Endpoints
```
GET /xdsl/{serviceName}
GET /xdsl/{serviceName}/lines/{number}
GET /xdsl/{serviceName}/lines/{number}/dslamPort
GET /xdsl/{serviceName}/ips
GET /xdsl/{serviceName}/eligibility

POST /xdsl/{serviceName}/lines/{number}/dslamPort/reset
PUT /xdsl/{serviceName}/lines/{number}/profile
```

### Profils disponibles
| Profil | Description |
|--------|-------------|
| auto | Auto-negociation |
| safe | Mode securise |
| performance | Performance max |
