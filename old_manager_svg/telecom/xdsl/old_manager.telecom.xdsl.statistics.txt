# Cahier des Charges : telecom/xdsl - Statistiques

## 1. RESUME EXECUTIF

### Perimetre
Page de visualisation des statistiques de la ligne xDSL.

### Objectif fonctionnel
- Afficher les graphiques de performance
- Selectionner la periodicite
- Analyser synchronisation, trafic, ping, SNR, attenuation

## 4. APIS & DONNEES

### Endpoints
```
GET /xdsl/{serviceName}/statistics
    Query: { period: "daily" | "weekly" | "monthly" | "yearly" }
    Response: {
      sync: [{ timestamp, down, up }],
      traffic: [{ timestamp, down, up }],
      ping: [{ timestamp, value }],
      snr: [{ timestamp, down, up }],
      attenuation: [{ timestamp, value }]
    }
```

### Periodicites
| Periode | Points de donnees |
|---------|-------------------|
| daily | 24 (1/heure) |
| weekly | 7 (1/jour) |
| monthly | 30 (1/jour) |
| yearly | 12 (1/mois) |
