# Cahier des Charges : telecom/telephony - Détails de l'offre

## 1. VUE D'ENSEMBLE
- **Type** : Page d'information
- **URL** : /telephony/{billingAccount}/line/{serviceName}/details
- **Objectif** : Afficher les informations détaillées de la ligne VoIP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne
- **OdsCard** : Informations de l'offre
- **OdsBadge** : Statut
- **OdsText** : Détails divers

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | "Détails de l'offre" |

## 5. CARD INFORMATIONS SUR L'OFFRE
| Propriété | Valeur |

| Titre | "Informations sur l'offre" |
