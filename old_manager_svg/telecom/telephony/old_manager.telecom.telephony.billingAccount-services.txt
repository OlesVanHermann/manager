# Cahier des Charges : telecom/telephony - Services du groupe

## 1. RESUME EXECUTIF

### Perimetre
Page de liste des services (lignes, alias, fax) d'un groupe de facturation.

### Objectif fonctionnel
- Lister tous les services du groupe
- Rechercher un service
- Acceder aux details de chaque service

## 3. COMPOSANTS UI

### Header page
| Element | Description |
|---------|-------------|
| Breadcrumb | Telecom / VoIP / billing-X / Lignes |
| Titre | Lignes |
| Recherche | Input avec icone loupe |

### Tableau services
| Colonne | Description |
|---------|-------------|
| Service | Numero (lien) |
| Description | Texte libre |
| Type | Ligne / Alias / Fax |
| Type de service | SIP / Redirection / File / Conference / Trunk |
| Actions | Detail → |

### Types de service
| Type | Description |
|------|-------------|
| Ligne | Ligne telephonique |
| Alias | Numero avec configuration |
| Fax | Service fax |

### Sous-types ligne
| Sous-type | Description |
|-----------|-------------|
| Ligne SIP | Ligne VoIP standard |
| Trunk | Ligne SIP trunk |

### Sous-types alias
| Sous-type | Description |
|-----------|-------------|
| Redirection | Renvoi simple |
| File d'appels | Queue/EasyHunting |
| Conference | Conference telephonique |

### Sous-types fax
| Sous-type | Description |
|-----------|-------------|
| Fax Entreprise | Fax virtual |

### Pagination
| Element | Description |
|---------|-------------|
| Bouton < | Page precedente |
| Numero | Page actuelle (bleu) |
| Bouton > | Page suivante |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/service
GET /telephony/{billingAccount}/line
GET /telephony/{billingAccount}/number
GET /telephony/{billingAccount}/fax
```

### Modele Service
```typescript
interface TelephonyService {
  serviceName: string;
  description: string;
  serviceType: 'line' | 'alias' | 'fax';
  featureType: string;
}
```
