# Cahier des Charges : telecom/telephony - Supprimer le groupe

## 1. RESUME EXECUTIF

### Perimetre
Confirmation de suppression d'un groupe VoIP.

### Objectif fonctionnel
- Informer des consequences
- Confirmer la suppression definitive
- Supprimer le groupe et ses services

| Element | Description |
|---------|-------------|
| Link | Retour a l'administration |

### Titre
| Element | Description |
|---------|-------------|
| Titre | Supprimer le groupe |

### Encart warning
| Element | Description |
|---------|-------------|
| Type | Warning (jaune) |
| Titre | Attention |
| Message | La suppression du groupe est definitive... |

### Carte Consequences
| Element | Description |
|---------|-------------|
| Texte | La suppression du groupe entrainera |
| Liste | Consequences detaillees |

### Liste des consequences
| Consequence |
|-------------|
| La resiliation de toutes les lignes telephoniques |
| La perte de tous les numeros associes |
| La suppression de la configuration (renvois, messagerie...) |
| Le remboursement du depot de garantie sous 30 jours |

### Bouton
| Element | Description |
|---------|-------------|

| Style | Danger (rouge) |
| Icone | Poubelle |
