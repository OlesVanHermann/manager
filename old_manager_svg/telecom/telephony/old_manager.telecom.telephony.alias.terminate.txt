# Cahier des Charges : telecom/telephony - Résilier le numéro alias

## 1. VUE D'ENSEMBLE
- **Type** : Page de résiliation
- **URL** : /telephony/{billingAccount}/alias/{serviceName}/terminate
- **Objectif** : Résilier un numéro alias du groupe de téléphonie

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Alias
- **OdsLink** : Lien retour
- **OdsMessage** : Infos résiliation, Warning SVA
- **OdsSelect** : Motif de résiliation
- **OdsTextarea** : Précisions
- **OdsButton** : Résilier (danger)

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Résilier le numéro alias" |
| Titre section | "Informations" |

## 5. MESSAGES INFORMATIFS
| Type | Contenu |
|------|---------|
| Info (bleu) | "La résiliation de votre numéro alias sera effective à la fin de votre période d'engagement en cours. Une fois la demande validée, vous recevrez un email de confirmation." |
| Warning (jaune) | "Si vous utilisez ce numéro en tant que numéro SVA (numéro surtaxé), assurez-vous d'avoir récupéré les fonds de votre portefeuille SVA." |
