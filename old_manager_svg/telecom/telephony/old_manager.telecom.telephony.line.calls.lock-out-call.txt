# Cahier des Charges : telecom/telephony - Verrouillage des appels sortants

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/calls/lockOutCall
- **Objectif** : Protéger la ligne avec un code PIN pour les appels sortants

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne / Appels
- **OdsLink** : Retour aux appels
- **OdsCard** : Container principal
- **OdsMessage** : Info verrouillage, Warning PIN
- **OdsCheckbox** : Activer verrouillage
- **OdsInput** : Code PIN (password)
- **OdsButton** : Enregistrer, Annuler

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour aux appels" |
| Titre | "Verrouillage des appels sortants" |
