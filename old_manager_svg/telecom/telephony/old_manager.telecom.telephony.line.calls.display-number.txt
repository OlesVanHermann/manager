# Cahier des Charges : telecom/telephony - Numéro affiché (CLIP)

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/calls/displayNumber
- **Objectif** : Configurer le numéro affiché lors des appels sortants (CLIP)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne / Appels
- **OdsLink** : Retour aux appels
- **OdsCard** : Configuration principale + Liste numéros
- **OdsMessage** : Info CLIP
- **OdsCheckbox** : Masquer numéro (anonyme)
- **OdsList** : Liste des numéros disponibles
- **OdsButton** : Changer, Enregistrer, Annuler

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour aux appels" |
| Titre | "Numéro affiché (CLIP)" |

## 5. LAYOUT 2 COLONNES
| Colonne | Contenu |
|---------|---------|
| Gauche (600px) | Configuration du numéro |
| Droite (480px) | Liste des numéros disponibles |
