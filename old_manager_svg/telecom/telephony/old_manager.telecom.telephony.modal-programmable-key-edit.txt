# Cahier des Charges : telecom/telephony - Modal configurer touche programmable

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Page touches programmables du téléphone
- **Objectif** : Configurer une touche programmable (function key)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning touche L1
- **OdsSelect** : Fonction
- **OdsInput** : Paramètre
- **OdsMessage** : Info fonctions disponibles
- **OdsButton** : Annuler, Valider

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configurer la touche" |
| Bouton fermer | × |

## 5. MESSAGE WARNING (si touche L1)
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ La touche L1 est utilisée pour la ligne principale. Modifier avec précaution." |

## 6. CHAMPS FORMULAIRE
| Champ | Type | Description |
|-------|------|-------------|
| Fonction | Select | Type de fonction |
| Paramètre | Input | Valeur associée (numéro, URL...) |

## 7. FONCTIONS DISPONIBLES
| Fonction | Paramètre requis |
|----------|------------------|
| Ligne | Numéro de ligne |
| BLF (Busy Lamp Field) | Numéro à superviser |
| Appel rapide | Numéro |
| Transfert | Numéro destination |
| Renvoi | Numéro destination |
| Interception | Numéro/groupe |
| Mise en attente | - |
| Conférence | - |
| URL | Adresse URL |
| DND (Do Not Disturb) | - |

## 8. MESSAGE INFO FONCTIONS
| Type | Contenu |
|------|---------|
| Info (bleu) | "Fonctions disponibles : Ligne, BLF, Appel rapide, Transfert, Renvoi, Interception, Mise en attente, Conférence, URL, DND..." |

## 9. AIDE PARAMÈTRE
| Texte |
|-------|
| "Entrez le numéro de téléphone à superviser" (pour BLF) |

## 10. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Valider | Primary | Enregistre la configuration |

## 11. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/line/{serviceName}/phone/functionKey - Liste touches
- PUT /telephony/{billingAccount}/line/{serviceName}/phone/functionKey/{keyNum} - Modifier touche
