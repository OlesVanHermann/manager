# Cahier des Charges : telecom/telephony - Modal numéro affiché (CLIP)

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Configuration d'appels
- **Objectif** : Modifier le numéro affiché lors des appels sortants

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection numéro
- **OdsMessage** : Info disponibilité
- **OdsButton** : Annuler, Valider

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Numéro affiché (CLIP)" |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Sélectionnez le numéro qui sera affiché lors des appels sortants depuis cette ligne." |

## 6. CHAMP FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Numéro à afficher | Select | Oui * |

### Options
| Source |
|--------|
| Numéros du groupe de facturation |

### Exemple
| Valeur |
|--------|
| +33 9 12 34 56 78 (Ligne principale) |

## 7. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info (bleu) | "Seuls les numéros de votre groupe de facturation sont disponibles." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Valider | Primary | Enregistre le choix |

## 9. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/number - Liste numéros disponibles
- PUT /telephony/{billingAccount}/line/{serviceName}/options - Modifier CLIP
