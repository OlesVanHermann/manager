# Cahier des Charges : telecom/telephony - Line Terminate Cancel

## 1. VUE D'ENSEMBLE

**Type:** Page confirmation
**Fonction:** Annuler une demande de résiliation en cours
**Contexte:** Accessible quand une résiliation est programmée

## 2. DIMENSIONS

- Container: 600x350px
- Border radius: 8px

## 3. NAVIGATION

### Lien retour
"← Retour" (text-primary)

## 4. CONTENU

### Titre
"Annuler la résiliation" (18px, bold)

### Bannière warning (560x80px)
- Background: #FEF3C7
- Bordure: #FDE68A
- Icône: ⚠
- Titre: "Résiliation programmée" (12px, bold)
- Contenu:
  - "La résiliation de la ligne \"{description}\" ({numéro}) est"
  - "programmée pour le {date}."

### Texte explicatif
- "Vous pouvez annuler cette demande de résiliation tant que la date"
- "effective n'est pas atteinte."
- (espace)
- "Une fois annulée, votre service continuera à fonctionner normalement"
- "et vous serez facturé lors du prochain cycle de facturation."

## 5. BOUTONS

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler la résiliation | btn-primary | 180x40px |
| Retour | btn-outline | 120x40px |

## 6. APIS & ENDPOINTS

```
# Annuler la résiliation
DELETE /telephony/{billingAccount}/line/{serviceName}/terminate
→ {
  success: true,
  message: "Demande de résiliation annulée"
}

# Vérifier statut résiliation
GET /telephony/{billingAccount}/line/{serviceName}/terminate
→ {
  terminationId: "term-123",
  effectiveDate: "2025-01-31",
  status: "pending",
  canCancel: true
}
```
