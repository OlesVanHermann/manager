# Cahier des Charges : telecom/telephony - Gestion des contacts

## 1. RESUME EXECUTIF

### Perimetre
Gestion des contacts administratif, technique et facturation.

### Objectif fonctionnel
- Visualiser les contacts par service
- Modifier les contacts
- Gerer les contacts du groupe et des lignes

## 3. COMPOSANTS UI

### Titre
| Element | Description |
|---------|-------------|
| Titre | Gestion des contacts |
| Description | Modifiez les contacts administratif, technique... |

### Encart info
| Element | Description |
|---------|-------------|
| Type | Info (bleu) |
| Message | Certaines modifications necessitent... |
| Lien | Creer un ticket de support |

### Tableau contacts
| Colonne | Description |
|---------|-------------|
| Service | Nom/numero du service |
| Contact Admin | NIC admin |
| Contact Tech | NIC technique |
| Contact Facturation | NIC facturation |
| Actions | Bouton modifier |

### Sections du tableau
| Section | Contenu |
|---------|---------|
| Groupe | Contacts du groupe |
| Lignes | Contacts par ligne |
| Alias | Contacts par numero |

### Bouton action
| Element | Description |
|---------|-------------|
| Icone | Crayon |
| Action | Ouvre modal modification |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/serviceInfos
    Response: { contactAdmin, contactTech, contactBilling }

GET /telephony/{billingAccount}/line/{serviceName}/serviceInfos
GET /telephony/{billingAccount}/number/{serviceName}/serviceInfos

PUT /telephony/{billingAccount}/serviceInfos
    Body: { contactAdmin, contactTech, contactBilling }
```
