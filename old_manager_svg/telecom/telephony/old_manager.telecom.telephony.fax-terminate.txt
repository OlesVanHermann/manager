# Cahier des Charges : telecom/telephony - Fax Terminate

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire
**Fonction:** Demander la résiliation d'un service fax
**Contexte:** Accessible depuis la gestion d'un service fax

### Lien retour
"← Gestion du fax" (text-primary)

## 3. LAYOUT

- Titre: "Résiliation du service Fax" (22px, bold)
- 2 colonnes: Informations (530px) + Résiliation (510px)

## 5. COLONNE RÉSILIATION (510x280px)

### Titre section
"Résiliation du service Fax" (14px, bold)

### Formulaire

| Champ | Type | Dimensions |
|-------|------|------------|
| Raison de la résiliation | Select | 450x40px |
| Précisions (si "Autres") | Textarea | 450x60px |

### Options raison
- Choisissez une cause
- Déménagement
- Facturation non adaptée
- Cessation d'activité
- Manque de fonctionnalités
- Problèmes techniques
- Départ vers la concurrence
- Autres

### Lien bulk
"Appliquer à plusieurs fax" (text-primary, 11px)

### Bouton
- Texte: "Résilier"
- Style: btn-danger (#DC2626)
- Dimensions: 100x36px

## 6. APIS & ENDPOINTS

```
# Demander résiliation fax
POST /telephony/{billingAccount}/fax/{serviceName}/terminate
Body: {
  reason: "cessation_activity",
  details: "Fermeture de l'entreprise"
}
→ {
  terminationId: "term-123",
  effectiveDate: "2025-01-31",
  status: "pending"
}

# Raisons prédéfinies
GET /telephony/termination/reasons
→ [
  { code: "moving", label: "Déménagement" },
  { code: "pricing", label: "Facturation non adaptée" },
  { code: "cessation_activity", label: "Cessation d'activité" },
  { code: "missing_features", label: "Manque de fonctionnalités" },
  { code: "technical_issues", label: "Problèmes techniques" },
  { code: "competitor", label: "Départ vers la concurrence" },
  { code: "other", label: "Autres" }
]
```
