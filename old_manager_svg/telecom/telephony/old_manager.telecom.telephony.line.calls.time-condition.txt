# Cahier des Charges : telecom/telephony - Plages horaires conditionnelles

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration avancée
- **URL** : /telephony/{billingAccount}/line/{serviceName}/calls/timeCondition
- **Objectif** : Définir des comportements différents selon les plages horaires

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 850px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne / Appels
- **OdsLink** : Retour aux appels, Aide
- **OdsMessage** : Info conditions temporelles
- **OdsCard** : Paramètres + Créneaux + Planning
- **OdsToggleGroup** : Oui/Non
- **OdsSelect** : Délai
- **OdsCalendar** : Planning hebdomadaire
- **OdsButton** : Actions, Enregistrer, Annuler

## 12. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/timeCondition - Configuration
- PUT /telephony/{billingAccount}/timeCondition - Modifier
- GET /telephony/{billingAccount}/timeCondition/{serviceName}/condition - Liste conditions
- POST /telephony/{billingAccount}/timeCondition/{serviceName}/condition - Ajouter
- DELETE /telephony/{billingAccount}/timeCondition/{serviceName}/condition/{id} - Supprimer
