# Cahier des Charges : telecom/telephony - Documents SVA

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion des documents justificatifs KYC pour SVA.

### Objectif fonctionnel
- Telecharger les documents requis
- Suivre le statut de verification
- Gerer les documents existants

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/svaWallet/documents
POST /telephony/svaWallet/documents
DELETE /telephony/svaWallet/documents/{id}
GET /telephony/svaWallet/documents/{id}/download
```

### Modele Document
```typescript
interface SvaDocument {
  id: string;
  type: 'kbis' | 'id' | 'rib' | 'address_proof';
  relatedTo?: string; // beneficiary name
  fileName: string;
  status: 'valid' | 'required' | 'pending' | 'rejected';
  uploadDate?: string;
}
```

## 6. REGLES METIER

- Formats: PDF, JPG, PNG
- Taille max: 5 Mo
- Verification: 48h ouvrees
- Documents requis selon type structure
- Piece ID pour chaque beneficiaire
