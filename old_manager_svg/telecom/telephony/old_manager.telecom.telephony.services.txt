# Cahier des Charges : telecom/telephony - Services

## 1. RESUME EXECUTIF

### Perimetre
Liste de tous les services du groupe VoIP.

### Objectif fonctionnel
- Visualiser tous les services (lignes, numeros, fax)
- Filtrer par type et fonctionnalite
- Acceder au detail d'un service

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line
GET /telephony/{billingAccount}/number
GET /telephony/{billingAccount}/fax
```

### Types de service
| Type | API |
|------|-----|
| Ligne | /line |
| Numero | /number |
| Fax | /fax |

### Fonctionnalites
| Fonctionnalite | Description |
|----------------|-------------|
| SIP | Ligne SIP standard |
| Trunk | Ligne trunk multi-canaux |
| File d'appels | Numero avec queue |
| SVI | Serveur vocal interactif |
| EcoFax Pro | Service fax |
