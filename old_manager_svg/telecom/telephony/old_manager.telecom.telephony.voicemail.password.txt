# Cahier des Charges : telecom/telephony - Mot de passe messagerie vocale

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/service/{serviceName}/voicemail/password
- **Objectif** : Modifier le code PIN de la messagerie vocale

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Service
- **OdsCard** : Modifier PIN + Numéros d'accès
- **OdsMessage** : Info code PIN
- **OdsInput** : Nouveau PIN, Confirmation
- **OdsCheckbox** : Forcer changement
- **OdsButton** : Appliquer

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | "Messagerie vocale" |

## 5. LAYOUT 2 COLONNES
| Colonne | Contenu |
|---------|---------|
| Gauche (520px) | Modifier le code PIN |
| Droite (400px) | Numéros d'accès |

## 6. CARD MODIFIER LE CODE PIN
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le code PIN" |

### Message Info
| Type | Contenu |
|------|---------|
| Info (bleu) | "Le code PIN est utilisé pour accéder à votre messagerie vocale. Il doit contenir exactement 4 chiffres." |

### Champs
| Champ | Type | Requis |
|-------|------|--------|
| Nouveau code PIN | Input password (4 digits) | Oui * |
| Confirmer le code PIN | Input password (4 digits) | Oui * |

### Option
| Checkbox | Label |
|----------|-------|
| Non coché | "Forcer le changement de mot de passe au prochain accès" |

### Action Groupée
| Lien | Action |
|------|--------|
| "Appliquer à plusieurs lignes..." | Multi-sélection |

### Bouton
| Bouton | Style | Action |
|--------|-------|--------|
| Appliquer | Primary | Modifie le code PIN |

## 7. CARD NUMÉROS D'ACCÈS
| Propriété | Valeur |
|-----------|--------|
| Titre | "Numéros d'accès" |

### Informations
| Champ | Exemple |
|-------|---------|
| Numéro interne | 123 |
| Numéro externe | +33 1 84 00 00 99 |

## 8. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/voicemail/{serviceName} - Infos messagerie
- PUT /telephony/{billingAccount}/voicemail/{serviceName}/settings/password - Modifier PIN
