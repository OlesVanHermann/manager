# Cahier des Charges : telecom/telephony - Composant Coordonnees du titulaire

## 1. RESUME EXECUTIF

### Perimetre
Composant reutilisable pour les coordonnees (identite + adresse) dans les formulaires de commande.

### Objectif fonctionnel
- Collecter les coordonnees completes du titulaire
- Gerer le type de titulaire (particulier/entreprise)
- Proposer l'inscription a l'annuaire universel

---

## 2. LAYOUT

### Structure carte avec sections identite + adresse

## 3. COMPOSANTS UI

### Note obligatoire
| Element | Description |
|---------|-------------|
| Note | * Champs obligatoires |

### Type de titulaire
| Champ | Type | Options |
|-------|------|---------|
| Type de titulaire | Radio | Particulier, Entreprise |

### Section Identite
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Nom | Input | Oui |
| Prenom | Input | Oui |
| Email | Input | Non |
| Telephone | Input | Non |

### Section Adresse
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Adresse | Input | Oui |
| Code postal | Input | Oui |
| Ville | Input | Oui |
| Pays | Select | Oui |

### Annuaire universel
| Element | Type | Description |
|---------|------|-------------|
| Checkbox | Option | Apparaitre dans l'annuaire universel |
| Note | Info | Vos coordonnees seront publiees dans les annuaires telephoniques |

---

## 4. APIS & DONNEES

### Validation
| Champ | Regle |
|-------|-------|
| Adresse | 5-200 caracteres |
| Code postal | Format selon pays |
| Ville | 2-100 caracteres |
| Pays | Code ISO 2 lettres |

### Structure donnees
```json
{
  "holderType": "individual",
  "identity": {
    "lastName": "Dupont",
    "firstName": "Jean",
    "email": "jean.dupont@email.com",
    "phone": "+33612345678"
  },
  "address": {
    "street": "123 rue de la Paix",
    "zipCode": "75001",
    "city": "PARIS",
    "country": "FR"
  },
  "directory": false
}
```

### Validation code postal par pays
| Pays | Format |
|------|--------|
| FR | 5 chiffres |
| BE | 4 chiffres |
| CH | 4 chiffres |
| DE | 5 chiffres |
| GB | Alphanum |
