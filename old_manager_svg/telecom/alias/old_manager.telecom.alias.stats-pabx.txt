## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/records
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/queue/{queueId}/liveCalls
```

### Modele Record
```typescript
interface OvhPabxRecord {
  id: number;
  callStart: string;
  callEnd: string;
  callingNumber: string;
  agent: string;
  duration: number;
  status: 'answered' | 'missed' | 'abandoned';
}
```
