## 1. RESUME EXECUTIF

Permet de modifier le type de configuration d'un numero alias telephonique.
Deux modes disponibles : Simple (redirect, CCS, cloudIvr, conference) et
Expert (svi, CCS Expert). Le changement de type entraine la perte de la
configuration actuelle.

Utilisateurs cibles : Administrateurs telephonie VoIP
Impact : Perte configuration actuelle si changement de type

## 5. APIs & DONNEES
--------------------------------------------------------------------------------

  -> Recupere featureType actuel

### API Ecriture

POST /telephony/{billingAccount}/number/{serviceName}/changeFeatureType
  Body : { featureType: "redirect|contactCenterSolution|cloudIvr|conference|svi|contactCenterSolutionExpert" }
