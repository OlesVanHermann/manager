# Cahier des Charges : telecom/alias - CCS Filtrage des appels

## 1. RESUME EXECUTIF

### Perimetre
Configuration du filtrage d'appels du Contact Center Solution.

### Objectif fonctionnel
- Rejeter les appels anonymes
- Gerer les listes noires/blanches
- Configurer les regles de filtrage

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/easyHunting/{serviceName}/screenListConditions
POST /telephony/{billingAccount}/easyHunting/{serviceName}/screenListConditions
DELETE /telephony/{billingAccount}/easyHunting/{serviceName}/screenListConditions/{conditionId}
PUT /telephony/{billingAccount}/easyHunting/{serviceName}
```

### Modele ScreenListCondition
```typescript
interface ScreenListCondition {
  conditionId: number;
  callerIdNumber: string;
  screenListType: 'incomingBlackList' | 'incomingWhiteList';
}
```
