# Cahier des Charges : telecom/alias - Gestion des membres

## 1. RESUME EXECUTIF

### Perimetre
Gestion des membres d'une file d'attente/groupe.

### Objectif fonctionnel
- Ajouter/supprimer des membres
- Configurer les parametres par membre
- Reordonner les membres

| Element | Description |

| Element | Description |
|---------|-------------|

| Statut | Badge | Couleur |
|--------|-------|---------|

### Edition inline
| Champ | Type | Description |
|-------|------|-------------|
| Statut | Select | Disponible/Inactif |

### Confirmation suppression
| Element | Description |
|---------|-------------|
| Fond | Rouge clair |
| Question | Confirmer la suppression ? |
| Boutons | Supprimer / Annuler |

### Boutons edition
| Bouton | Type | Action |
|--------|------|--------|
| Enregistrer | Primary | Valide modification |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/agent
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/agent/{agentId}
POST /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/agent
PUT /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/agent/{agentId}
DELETE /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/agent/{agentId}
```

### Modele Agent
```typescript
interface EasyHuntingAgent {
  agentId: number;
  number: string;
  description: string;
  status: 'available' | 'loggedOut';
  timeout: number;
  wrapUpTime: number;
  simultaneousLines: number;
  position: number;
}
```
