# Cahier des Charges : telecom/alias - Redirection simple

## 1. RESUME EXECUTIF

### Perimetre
Configuration d'une redirection simple d'appels.

### Objectif fonctionnel
- Rediriger les appels vers une ligne
- Configurer la presentation du numero
- Option d'appels sortants

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/redirect/{serviceName}
PUT /telephony/{billingAccount}/redirect/{serviceName}
PUT /telephony/{billingAccount}/redirect/{serviceName}/changeDestination
```

### Modele Redirect
```typescript
interface TelephonyRedirect {
  serviceName: string;
  destination: string;
  featureType: string;
}
```
