### Perimetre
Gestion des agents du PABX (operateurs).

### Objectif fonctionnel
- Ajouter des agents a une file

### Header

### Liste agents
| Colonne | Description |
|---------|-------------|
| Ordre | Position dans la file |
| Numero | Numero de l'agent |
| Statut | Actif/Inactif |

### Statut agent
| Statut | Badge | Description |
|--------|-------|-------------|
| available | Vert | Disponible |
| inAQueue | Bleu | En appel |
| loggedOut | Gris | Deconnecte |

### Formulaire agent
| Champ | Type | Description |
|-------|------|-------------|
| Numero | Input + recherche | Selection ligne |
| Timeout | Number (s) | Delai sonnerie |
| Wrap-up | Number (s) | Temps repos |
| Simultanes | Number | Appels simultanes |

| Element | Description |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent
PUT /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}
DELETE /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}
```

### Modele Agent
```typescript
interface OvhPabxHuntingAgent {
  agentId: number;
  number: string;
  position: number;
  status: 'available' | 'inAQueue' | 'loggedOut';
  timeout: number;
  wrapUpTime: number;
  simultaneousLines: number;
}
```

- Agent doit etre une ligne VoIP valide
