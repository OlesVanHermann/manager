# Cahier des Charges : telecom/alias - Calendrier/Scheduler

## 1. RESUME EXECUTIF

### Perimetre
Gestion du calendrier pour conditions temporelles.

### Objectif fonctionnel
- Definir les periodes d'ouverture/fermeture
- Planifier des evenements exceptionnels
- Gerer les jours feries

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/timeCondition/{serviceName}/condition
POST /telephony/{billingAccount}/timeCondition/{serviceName}/condition
PUT /telephony/{billingAccount}/timeCondition/{serviceName}/condition/{id}
DELETE /telephony/{billingAccount}/timeCondition/{serviceName}/condition/{id}
GET /telephony/{billingAccount}/scheduler/{serviceName}/events
POST /telephony/{billingAccount}/scheduler/{serviceName}/events
```

### Modele Event
```typescript
interface TelephonySchedulerEvent {
  uid: string;
  title: string;
  dateStart: string;
  dateEnd: string;
  categories: string;
  description: string;
}
```
