## 1. RESUME EXECUTIF

### Perimetre fonctionnel
Configuration des numeros alias selon leur type : redirection, conference,
SVI, file d'attente (easyHunting), centre d'appels (CCS).

### Objectif
- Configurer les redirections d'appels
- Gerer les files d'attente et agents
- Configurer les SVI (menus vocaux)
- Parametrer les conferences

--------------------------------------------------------------------------------

## 4. COMPOSANTS UI

### Configuration easyHunting
- oui-datagrid : Liste agents/files
- oui-calendar : Plages horaires
- oui-stepper : Wizard configuration

--------------------------------------------------------------------------------

## 5. APIs & DONNEES

### Changement de type
POST /telephony/:billingAccount/number/:serviceName/changeFeatureType
Body: { featureType: string }

### Hunting
GET /telephony/:billingAccount/easyHunting/:serviceName/hunting/queue
GET /telephony/:billingAccount/easyHunting/:serviceName/hunting/agent

### Strategies hunting
- ringAll : Sonne tous les agents
- random : Agent aleatoire
- sequential : Un par un

## 7. REGLES METIER

### Limites easyHunting
- Max 50 agents par file
- Max 10 files par alias
- Strategies : ringAll, random, sequential, cumulationByAgentOrder
