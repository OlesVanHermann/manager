# Cahier des Charges : telecom/alias - Modal Intercepter l'appel

## 1. RESUME EXECUTIF

### Perimetre
Modal d'interception d'un appel en attente.

### Objectif fonctionnel
- Prendre un appel de la file
- Transferer vers soi-meme
- Intervenir rapidement

### Endpoints
```
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/queue/{queueId}/liveCalls/{id}/intercept
```
