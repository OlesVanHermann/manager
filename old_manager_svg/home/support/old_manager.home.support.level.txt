# Cahier des Charges : home/support - Niveau de support actuel

## 1. VUE D'ENSEMBLE
- **Type** : Page dashboard support
- **URL** : /support/level
- **Objectif** : Afficher le niveau de support actuel et les statistiques

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Support
- **OdsCard** : Cards niveau et fonctionnalités
- **OdsBadge** : Badge "Actuel"
- **OdsButton** : Voir les offres

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | "Mon niveau de support" |

## 5. CARD NIVEAU ACTUEL
| Élément | Description |
|---------|-------------|
| Badge | "Actuel" (bleu) |
| Bordure | Bleue 2px |
| Titre | Nom du niveau (ex: "Standard") |
| Description | Description du niveau |
| Détail | Liste des fonctionnalités incluses |

## 6. CARD FONCTIONNALITÉS
| Type | Icône | Couleur | Exemples |
|------|-------|---------|----------|
| Inclus | ✓ | Vert | Tickets email, Documentation, Forum |
| Non inclus | ✕ | Rouge/Gris | Support téléphonique, SLA, Account manager |

## 7. BOUTON CTA
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voir les offres →" |
| Style | Primary |
| Action | Navigation vers comparatif offres |

## 8. CARD STATISTIQUES
| Statistique | Format | Couleur |
|-------------|--------|---------|
| Tickets ouverts | Nombre | Normal |
| Tickets résolus (30j) | Nombre | Vert |
| Temps de réponse moyen | Durée (ex: 24h) | Normal |
| Satisfaction | Note X/5 | Vert |

## 9. LAYOUT
| Zone | Position | Largeur |
|------|----------|---------|
| Card niveau actuel | Gauche | 400px |
| Card fonctionnalités | Droite | 700px |
| Card statistiques | Bas pleine largeur | 1100px |

## 10. APIS & ENDPOINTS
- GET /me/supportLevel - Niveau actuel
- GET /support/tickets/stats - Statistiques
