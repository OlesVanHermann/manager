## 7. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic × | Fermer modal |
| Clic Annuler | Fermer modal sans sauvegarder |
| Clic toggle | Activer/désactiver l'alerte |
| Modifier seuil | Mettre à jour la valeur |
| Clic Enregistrer | Sauvegarder les préférences |

---

## 8. APIS & ENDPOINTS

### 8.1 Récupérer les préférences d'alertes

```
GET /me/billing/alertPreferences
```

Response:
```json
{
  "newInvoice": true,
  "paymentCompleted": true,
  "serviceExpiring": true,
  "creditThreshold": false,
  "creditThresholdValue": 50
}
```

### 8.2 Mettre à jour les préférences

```
PUT /me/billing/alertPreferences
```

Body:
```json
{
  "newInvoice": true,
  "paymentCompleted": true,
  "serviceExpiring": true,
  "creditThreshold": false,
  "creditThresholdValue": 50
}
```
