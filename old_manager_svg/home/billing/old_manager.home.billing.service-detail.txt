## 8. APIS & ENDPOINTS

### 8.1 Détail du service

```
GET /me/billing/service/{serviceName}
```

Response:
```json
{
  "serviceName": "domain.example.com",
  "serviceType": "domain",
  "status": "active",
  "creationDate": "2023-03-15",
  "expirationDate": "2026-03-15",
  "renewalType": "automatic",
  "price": 9.99,
  "period": "year"
}
```
