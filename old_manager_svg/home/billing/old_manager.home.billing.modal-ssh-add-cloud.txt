# Cahier des Charges : home/billing - Modal ajouter clé SSH Cloud

## 1. VUE D'ENSEMBLE
- **Type** : Modal de sélection
- **Contexte** : Page clés SSH
- **Objectif** : Ajouter une clé SSH existante à un projet Public Cloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Projet Public Cloud
- **OdsText** : Instructions
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une clé SSH" |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Titre section
| Texte |
|-------|
| "Sélectionner un projet Cloud" |

### Champ
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Projet Public Cloud * | Select | "Mon projet PCI - abc123" |

### Aide
| Texte |
|-------|
| "La clé SSH sera ajoutée au projet sélectionné et pourra être utilisée lors de la création de nouvelles instances." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Ajoute la clé au projet |

## 7. APIS & ENDPOINTS
- GET /cloud/project - Liste des projets PCI
- POST /cloud/project/{projectId}/sshkey - Ajouter clé SSH au projet
