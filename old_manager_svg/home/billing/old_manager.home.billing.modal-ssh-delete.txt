# Cahier des Charges : home/billing - Modal supprimer clé SSH

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page clés SSH
- **Objectif** : Supprimer une clé SSH

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 190px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la clé SSH" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Message
| Texte |
|-------|
| "Êtes-vous sûr de vouloir supprimer la clé SSH" |
| "**"my-server-key"** ?" |

### Information
| Texte |
|-------|
| "Cette action est irréversible." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime la clé |

## 7. APIS & ENDPOINTS
- DELETE /me/sshKey/{keyName} - Supprimer clé SSH
