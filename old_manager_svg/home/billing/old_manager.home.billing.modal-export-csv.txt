## 8. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic × | Fermer modal |
| Clic Annuler | Fermer modal |
| Sélection format | Changer le format actif |
| Modifier période | Mettre à jour les dates |
| Cocher/décocher colonne | Inclure/exclure de l'export |
| Clic Exporter | Télécharger le fichier |

---

## 9. APIS & ENDPOINTS

### 9.1 Exporter les factures

```
GET /me/bill/export?format=csv&from=2025-01-01&to=2025-12-31&columns=reference,date,amountHT,vat,amountTTC
```

Response: Fichier téléchargeable (CSV, PDF ou Excel)
