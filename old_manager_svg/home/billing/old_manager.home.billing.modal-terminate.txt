## 11. APIS & ENDPOINTS

### 11.1 Résilier un service

```
POST /me/service/{serviceName}/terminate
```

Body:
```json
{
  "reason": "not_needed",
  "comment": "string (optional)"
}
```

Response:
```json
{
  "terminationDate": "2026-03-15",
  "status": "pending_termination"
}
```
