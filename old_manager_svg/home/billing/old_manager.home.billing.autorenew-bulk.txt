# Cahier des Charges : home/billing - Actions groupées renouvellement

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation groupée
- **URL** : /billing/autorenew/bulk
- **Objectif** : Activer le renouvellement automatique pour plusieurs services

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Facturation / Renouvellement / Actions groupées
- **OdsLink** : Retour
- **OdsMessage** : Info moyen de paiement, Warning services indisponibles
- **OdsText** : Compteur services
- **OdsButton** : Annuler, Confirmer

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Activer le renouvellement automatique" |

| Type | Contenu |
|------|---------|
| Info (bleu) | Moyen de paiement : "Carte bancaire - •••• 4242" |
| | "Il s'agit de votre moyen de paiement par défaut." |
| | "Le renouvellement automatique sera effectué tous les mois selon la fréquence de chaque service." |

## 6. COMPTEUR SERVICES
| Élément | Description |
|---------|-------------|
| Titre | "5 services sélectionnés" |
| Description | "Le renouvellement automatique sera activé pour tous ces services." |

| Élément | Description |
|---------|-------------|
| Lien toggle | "Voir les 5 services disponibles" |
| Liste | ns1234567.ip-123-45-67.eu, example-domain.com, vps-abc123.vps.ovh.net, ... |

## 9. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Confirmer | Primary | Active le renouvellement |

## 10. APIS & ENDPOINTS
- PUT /services/{serviceId}/autoRenew - Activer renouvellement (par service)
- GET /me/paymentMean/default - Moyen de paiement par défaut
