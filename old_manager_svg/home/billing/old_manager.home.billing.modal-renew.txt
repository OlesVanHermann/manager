# Cahier des Charges : home/billing - Modal Renouvellement

## 1. VUE D'ENSEMBLE
- **Type** : Modal activation renouvellement auto
- **Déclencheur** : Action menu "Modifier le renouvellement"
- **Objectif** : Configurer le renouvellement automatique

## 2. STRUCTURE MODAL
- Header avec titre
- Service concerné (readonly)
- Select durée renouvellement
- Select moyen de paiement
- Footer actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x420px
- **OdsInput** : Service (readonly)
- **OdsSelect** : Durée, Moyen de paiement
- **OdsButton** : Annuler, Activer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le renouvellement automatique" |
| Background | #F3F4F6 |

## 5. SERVICE CONCERNÉ
| Propriété | Valeur |
|-----------|--------|
| Label | "Service concerné" |
| Input | Readonly, fond gris |
| Contenu | Nom service + Type |

## 6. DURÉE RENOUVELLEMENT
| Propriété | Valeur |
|-----------|--------|
| Label | "Durée de renouvellement" |
| Options | 1 mois, 3 mois, 6 mois, 12 mois |
| Format | "XX mois - €€,€€ €/mois" |

## 7. MOYEN DE PAIEMENT
| Propriété | Valeur |
|-----------|--------|
| Label | "Moyen de paiement pour le renouvellement" |
| Défaut | Carte par défaut |
| Options | Liste moyens enregistrés |

## 8. ACTIONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Activer | Primary bleu | "Activer" |

## 9. APIS & ENDPOINTS
- PUT /services/{id}/renew/automatic - Activer auto
- GET /services/{id}/renew/options - Options durée
