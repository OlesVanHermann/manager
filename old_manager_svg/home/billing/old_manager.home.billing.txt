# Cahier des Charges : home/billing - Factures

## 1. VUE D'ENSEMBLE
- **Type** : Liste des factures
- **URL** : /billing/invoices
- **Objectif** : Consulter et gérer les factures

## 2. STRUCTURE PAGE
- Sidebar avec Mes factures actif
- Breadcrumb
- Titre avec description
- Tabs de navigation
- Actions de masse + Recherche
- Datagrid des factures
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Factures | Pay As You Go | Suivi des paiements | Mes avoirs
- **OdsSelect** : Actions de masse
- **OdsInput** : Recherche
- **OdsCheckbox** : Sélection + Factures papier
- **OdsDatagrid** : Liste factures
- **OdsBadge** : Statuts (Payée, À régler, Impayée)
- **OdsActionMenu** : Menu contextuel
- **OdsPagination** : Navigation

| Option | Action |
|--------|--------|
| Voir la version HTML | Ouvre HTML |

| Voir les détails pour cette dette | (si dette) |
| Régler mon solde immédiatement | (si à régler) |

## 7. ACTIONS MASSE
- Télécharger en PDF
- Exporter en CSV

## 8. APIS & ENDPOINTS
- GET /me/bill - Liste factures
- GET /me/bill/{id} - Détail facture
- GET /me/bill/{id}/pdf - PDF facture
- POST /me/bill/export - Export CSV
