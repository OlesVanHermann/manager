# Cahier des Charges : home/billing - Modal Résiliation Échéance

## 1. VUE D'ENSEMBLE
- **Type** : Modal résiliation à échéance
- **Déclencheur** : Action menu "Résilier à échéance"
- **Objectif** : Programmer la résiliation à la fin de la période

## 2. STRUCTURE MODAL
- Header warning (rouge)
- Service concerné
- Box conséquences
- Select raison (optionnel)
- Checkbox confirmation
- Note annulation possible
- Footer actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x520px
- **OdsInput** : Service (readonly)
- **OdsMessage** : Conséquences (rouge)
- **OdsSelect** : Raison
- **OdsCheckbox** : Confirmation
- **OdsButton** : Annuler, Confirmer (rouge)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "⚠️ Résilier à échéance" |
| Background | #FEE2E2 |
| Couleur texte | #991B1B |

## 5. CONSÉQUENCES
| Propriété | Valeur |
|-----------|--------|
| Type | Error (rouge) |
| Titre | "Conséquences de la résiliation" |
| Points | Date désactivation, Suppression données, Libération domaine |

## 6. RAISON RÉSILIATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Raison de la résiliation (facultatif)" |
| Options | Prix trop élevé, Service non utilisé, Migration concurrent, Autre |

## 7. CHECKBOX CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Je comprends que cette action est irréversible" |
| Requis | Oui pour confirmer |

## 8. NOTE
"Vous pouvez annuler cette demande jusqu'à la date d'échéance."

## 9. ACTIONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Confirmer | Danger rouge | "Confirmer" |

## 10. APIS & ENDPOINTS
- POST /services/{id}/terminate - Résilier à échéance
- DELETE /services/{id}/terminate - Annuler résiliation
