## 7. APIS & ENDPOINTS

### 7.1 Lister les commandes éligibles

```
GET /me/order?retractable=true
```

Response:
```json
[
  {
    "orderId": "ORD-2026-001234",
    "date": "2026-01-02T10:00:00Z",
    "amount": 9.99,
    "remainingDays": 12,
    "canRetract": true
  }
]
```

### 7.2 Exercer la rétractation

```
POST /me/order/{orderId}/retract
```
