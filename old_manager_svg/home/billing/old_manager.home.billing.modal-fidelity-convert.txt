# Cahier des Charges : home/billing - Modal convertir points fidélité

## 1. VUE D'ENSEMBLE
- **Type** : Modal de conversion
- **Contexte** : Page points fidélité
- **Objectif** : Convertir des points fidélité en crédit compte prépayé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : Indicateur d'étapes (1-2)
- **OdsText** : Description, taux conversion
- **OdsInput** : Montant à convertir
- **OdsMessage** : Aperçu crédit
- **OdsButton** : Annuler, Convertir

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Convertir mes points fidélité" |
| Bouton fermer | × |

## 5. STEPPER
| Étape | État |
|-------|------|
| 1 | Actif (bleu) |
| 2 | Inactif (gris) |

## 6. CONTENU MODAL

### Description
| Texte |
|-------|
| "Convertissez vos points fidélité en crédit sur votre compte prépayé." |
| "1 point = 0,01 €" |

### Champ de saisie
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Montant à convertir (points) | Input number | 1000 |

### Validation
| Règle | Message erreur |
|-------|----------------|
| Minimum | "Minimum : 100 points" |
| Maximum | "Maximum : 50000 points" |

### Aperçu conversion
| Type | Contenu |
|------|---------|
| Success (vert) | "Crédit qui sera ajouté : **10,00 €**" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Convertir | Primary | Effectue la conversion |

## 8. APIS & ENDPOINTS
- GET /me/fidelityAccount - Solde points fidélité
- POST /me/fidelityAccount/creditOrder - Convertir en crédit
