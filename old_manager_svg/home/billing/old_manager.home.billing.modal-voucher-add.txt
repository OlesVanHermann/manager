## 9. APIS & ENDPOINTS

### 9.1 Ajouter un bon de réduction

```
POST /me/voucher
```

Body:
```json
{
  "code": "WELCOME2026"
}
```

Response:
```json
{
  "code": "WELCOME2026",
  "description": "-10% sur domaines",
  "expirationDate": "2026-03-31",
  "status": "active"
}
```
