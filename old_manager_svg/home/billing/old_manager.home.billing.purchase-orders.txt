## 7. APIS & ENDPOINTS

### 7.1 Lister les bons de commande

```
GET /me/purchaseOrder
```

Response:
```json
[
  {
    "purchaseOrderId": "PO-2026-001234",
    "creationDate": "2025-12-15T10:00:00Z",
    "maxAmount": 5000.00,
    "usedAmount": 1250.00,
    "expirationDate": "2026-12-31",
    "status": "active"
  }
]
```

### 7.2 Créer un bon de commande

```
POST /me/purchaseOrder
```

Body:
```json
{
  "maxAmount": 5000.00,
  "expirationDate": "2026-12-31"
}
```
