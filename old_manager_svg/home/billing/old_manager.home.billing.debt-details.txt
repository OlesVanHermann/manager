## 7. APIS & ENDPOINTS

### 7.1 Lister les impayés

```
GET /me/debtAccount
```

Response:
```json
{
  "totalAmount": 89.99,
  "currency": "EUR",
  "debts": [
    {
      "debtId": "debt-001",
      "billId": "FR2026010001",
      "date": "2026-01-01",
      "amount": 89.99,
      "dueDate": "2026-01-15",
      "status": "late"
    }
  ]
}
```

### 7.2 Payer un impayé

```
POST /me/debtAccount/pay
```

Body:
```json
{
  "debtIds": ["debt-001"],
  "paymentMethodId": "pm-card-001"
}
```
