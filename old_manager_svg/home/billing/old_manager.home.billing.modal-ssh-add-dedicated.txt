# Cahier des Charges : home/billing - Modal ajouter clé SSH Dedicated

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page clés SSH
- **Objectif** : Créer une nouvelle clé SSH pour serveurs dédiés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom de la clé
- **OdsTextarea** : Contenu clé SSH
- **OdsBadge** : Validité de la clé
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une clé SSH" |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Titre section
| Texte |
|-------|
| "Ajouter une nouvelle clé SSH Dedicated" |

### Champs
| Champ | Type | Placeholder | Validation |
|-------|------|-------------|------------|
| Nom de la clé * | Input | "ex: my-server-key" | Badge ✓ vert si valide |
| Clé SSH * | Textarea (130px) | Clé publique SSH | Badge "✓ Clé valide" si format correct |

### Aperçu clé
| Contenu exemple |
|-----------------|
| ssh-rsa AAAAB3NzaC1yc2EAAAADAQABAAAB... |
| ...user@hostname |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Crée la clé SSH |

## 7. APIS & ENDPOINTS
- POST /me/sshKey - Créer clé SSH Dedicated
