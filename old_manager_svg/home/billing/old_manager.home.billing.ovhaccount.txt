## 8. APIS & ENDPOINTS

### 8.1 Solde du compte prépayé

```
GET /me/ovhAccount
```

Response:
```json
{
  "balance": 125.50,
  "currency": "EUR"
}
```

### 8.2 Historique des mouvements

```
GET /me/ovhAccount/movements
```

Response:
```json
[
  {
    "date": "2026-01-02T10:30:00Z",
    "type": "credit",
    "description": "Rechargement par CB",
    "amount": 50.00,
    "balanceAfter": 125.50
  }
]
```
