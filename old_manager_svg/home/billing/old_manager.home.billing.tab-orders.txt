# Cahier des Charges : home/billing - Commandes

## 1. VUE D'ENSEMBLE
- **Type** : Liste des commandes
- **URL** : /billing/orders
- **Objectif** : Consulter l'historique des commandes

## 2. STRUCTURE PAGE
- Sidebar avec Mes commandes actif
- Breadcrumb
- Titre avec description
- Recherche
- Datagrid des commandes
- Pagination

## 3. COMPOSANTS UI
- **OdsInput** : Recherche commande
- **OdsDatagrid** : Liste commandes
- **OdsBadge** : Statuts
- **OdsActionMenu** : Menu contextuel
- **OdsPagination** : Navigation
