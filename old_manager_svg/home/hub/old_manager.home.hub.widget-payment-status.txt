# Cahier des Charges : home/hub - Widget Statuts Paiement

## 1. VUE D'ENSEMBLE
- **Type** : Widget tableau statuts renouvellement
- **Parent** : Dashboard hub
- **Objectif** : Vue rapide échéances services

## 2. STRUCTURE WIDGET
- Header avec titre et compteur
- Tableau 4 colonnes
- Statuts colorés
- ActionMenu par ligne

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur blanc
- **OdsDatagrid** : Tableau services
- **OdsBadge** : Statuts colorés
- **OdsChip** : Compteur total
- **OdsActionMenu** : Actions ligne
- **OdsLink** : Navigation

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 800px |
| Hauteur | 400px |
| Border radius | 12px |
| Padding | 20px |

## 5. HEADER
| Élément | Style |
|---------|-------|
| Titre | "Derniers statuts de paiement des services" |
| Compteur | Chip bleu avec nombre |
| Lien | "Voir tout →" |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| SERVICE | 40% | Nom + type |
| STATUT | 25% | Badge coloré |
| ÉCHÉANCE | 20% | Date |
| ACTIONS | 15% | Menu ⋮ |

## 7. STATUTS RENOUVELLEMENT
| Statut | Label | Background | Texte |
|--------|-------|------------|-------|
| automatic | Renouvellement auto | #DEF7EC | #03543F |
| manual | Renouvellement manuel | #FEF3C7 | #92400E |
| unpaid | Impayé | #FEE2E2 | #991B1B |

## 8. LIGNE SERVICE
| Élément | Style |
|---------|-------|
| Nom service | Lien bleu #0050D7 |
| Type service | 12px muted sous le nom |
| Hauteur ligne | 60px |
| Hover | Background #F9FAFB |

## 9. ACTIONMENU OPTIONS
| Option | Action |
|--------|--------|
| Renouveler | Ouvre modal renouvellement |
| Résilier | Ouvre modal résiliation |

## 10. APIS & ENDPOINTS
- GET /services - Liste services
- GET /services/{id}/renew - Infos renouvellement
- PUT /services/{id}/renew/automatic - Activer auto
- POST /services/{id}/terminate - Résilier
