# Cahier des Charges : home/hub - Bannière KYC Fraude

- **Type** : Bannière warning blocante
- **Déclencheur** : Vérification identité requise (fraude)
- **Objectif** : Obliger soumission documents KYC

## 2. STRUCTURE BANNIÈRE
- Icône warning gauche
- Message alerte

- Bouton fermer (optionnel)
