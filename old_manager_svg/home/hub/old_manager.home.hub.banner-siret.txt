# Cahier des Charges : home/hub - Bannière SIRET

## 1. VUE D'ENSEMBLE
- **Type** : Bannière info mise à jour données
- **Déclencheur** : SIRET manquant ou obsolète
- **Objectif** : Demander mise à jour numéro SIRET

## 2. STRUCTURE BANNIÈRE
- Icône info gauche
- Message explicatif multi-lignes
- Lien action externe
- Bouton fermer

## 3. COMPOSANTS UI
- **OdsMessage** : Conteneur info
- **OdsIcon** : Cercle info
- **OdsLink** : Lien mise à jour
- **OdsButton** : Fermer (×)

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% (1200px max) |
| Hauteur | 120px |
| Border radius | 8px |
| Padding | 24px |

## 5. STYLE INFO
| Propriété | Valeur |
|-----------|--------|
| Background | #E8F4FD |
| Border | 1px #0050D7 |
| Icône couleur | #0050D7 |

## 6. MESSAGE TEXTE
| Ligne | Contenu |
|-------|---------|
| 1 | "Dans le cadre de l'amélioration de la qualité de vos données, nous avons besoin de mettre à jour" |
| 2 | "le numéro SIRET de votre entreprise. Cette mise à jour ne prendra que quelques secondes," |
| 3 | "munissez-vous de votre KBIS et [cliquez-ici pour la mise à jour. ↗](#)" |

## 7. LIEN ACTION
| Label | Style | Destination |
|-------|-------|-------------|
| "cliquez-ici pour la mise à jour. ↗" | Bleu underline | /account/company |
| Icône | ↗ externe | Modal ou page dédiée |

## 8. BOUTON FERMER
| Propriété | Valeur |
|-----------|--------|
| Position | Top right |
| Icône | × |
| Couleur | #6B7280 |
| Comportement | Masque temporairement (7j) |

## 9. CONDITION AFFICHAGE
- Type compte = corporation/professional
- Pays = FR
- SIRET vide ou invalide

## 10. VALIDATION SIRET
| Règle | Valeur |
|-------|--------|
| Format | 14 chiffres |
| Algorithme | Luhn |
| Vérification | Base SIRENE INSEE |

## 11. APIS & ENDPOINTS
- GET /me - Vérifier type compte et pays
- GET /me/company - Données entreprise
- PUT /me/company - Mettre à jour SIRET
- GET /me/company/siret/validate - Valider SIRET
