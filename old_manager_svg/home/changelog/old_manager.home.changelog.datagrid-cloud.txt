- **Type** : Détail datagrid Cloud
- **URL** : /changelog?tab=cloud
- **Objectif** : Afficher le datagrid des changements Cloud

## 2. STRUCTURE DATAGRID
- En-têtes de colonnes triables
- Lignes avec alternance de couleur
