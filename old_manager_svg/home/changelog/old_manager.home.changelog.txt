# Cahier des Charges : home/changelog - Page Principale

## 1. VUE D'ENSEMBLE
- **Type** : Page changelog/roadmap
- **URL** : /changelog
- **Objectif** : Afficher les nouveautés et la roadmap produits OVHcloud

## 2. STRUCTURE PAGE
- Sidebar avec menu principal
- Header avec titre "Changelog & Roadmap"
- Tabs Cloud / Hosting
- Datagrid des changements
- Section Roadmap avec tuiles
- Info box GitHub

## 3. COMPOSANTS UI
- **OdsTabs** : Cloud | Hosting
- **OdsDatagrid** : Liste des changements
- **OdsBadge** : Statut (New, Updated, Coming Soon)
- **OdsCard** : Tuiles roadmap
- **OdsMessage** : Info GitHub

## 4. SIDEBAR MENU
| Item | État |
|------|------|

| Colonne | Type | Description |
|---------|------|-------------|
