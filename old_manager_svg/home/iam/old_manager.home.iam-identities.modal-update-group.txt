## 9. APIS & ENDPOINTS

### 9.1 Modifier un groupe

```
PUT /me/identity/group/{groupId}
```

Body:
```json
{
  "name": "Developers",
  "description": "Équipe de développement - Accès aux environnements dev/staging",
  "policies": ["ReadWrite"]
}
```

### 9.2 Ajouter un membre

```
POST /me/identity/group/{groupId}/member
```

Body:
```json
{
  "userId": "user-123"
}
```

### 9.3 Retirer un membre

```
DELETE /me/identity/group/{groupId}/member/{userId}
```
