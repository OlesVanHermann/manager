## 8. APIS & ENDPOINTS

### 8.1 Lister les groupes

```
GET /me/identity/group
```

Response:
```json
[
  {
    "groupId": "group-001",
    "name": "Admin",
    "description": "Administrateurs avec accès complet",
    "membersCount": 2,
    "policies": ["FullAccess"],
    "creationDate": "2025-01-15T10:30:00Z"
  },
  {
    "groupId": "group-002",
    "name": "Developers",
    "description": "Équipe de développement",
    "membersCount": 4,
    "policies": ["ReadWrite"],
    "creationDate": "2025-02-20T14:00:00Z"
  }
]
```

### 8.2 Créer un groupe

```
POST /me/identity/group
```

Body:
```json
{
  "name": "NewGroup",
  "description": "Description du groupe",
  "policies": ["ReadOnly"]
}
```
