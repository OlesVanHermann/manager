## 8. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic onglet Identités | Afficher dashboard (cette page) |
| Clic onglet Politiques | Naviguer vers iam-policies |
| Clic onglet Clés API | Naviguer vers iam-api-keys |
| Clic onglet Logs | Naviguer vers iam-logs |
| Clic Gérer (Utilisateurs) | Naviguer vers iam-identities.users |
| Clic Gérer (Groupes) | Naviguer vers iam-identities.groups |
| Clic Configurer (SSO) | Naviguer vers iam-identities.sso |
| Clic Voir tous les logs | Naviguer vers iam-logs |

## 9. APIS & ENDPOINTS

### 9.1 Résumé IAM

```
GET /me/iam/summary
```

Response:
```json
{
  "users": {
    "total": 8,
    "admins": 2,
    "standard": 6
  },
  "groups": {
    "total": 3,
    "details": [
      { "name": "Admin", "members": 2 },
      { "name": "Developers", "members": 4 }
    ]
  },
  "sso": {
    "providers": 1,
    "active": true,
    "name": "Azure AD"
  },
  "policies": {
    "total": 12,
    "ovh": 3,
    "custom": 9
  },
  "recentActivity": [
    { "date": "2026-01-02T10:30:00Z", "action": "login", "user": "admin@company.com" }

```
