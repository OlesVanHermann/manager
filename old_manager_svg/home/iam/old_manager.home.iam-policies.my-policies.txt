## 6. APIS & ENDPOINTS

### 6.1 Lister mes politiques

```
GET /me/iam/policy?owner=me
```

Response:
```json
[
  {
    "policyId": "policy-001",
    "name": "FullAccess",
    "description": "Accès complet",
    "permissions": ["*"],
    "groups": ["Admin"],
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```
