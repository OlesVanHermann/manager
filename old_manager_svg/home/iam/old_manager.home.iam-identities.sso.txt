# Cahier des Charges : home/iam - SSO Identity Provider

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration SSO
- **URL** : /iam/identities/sso
- **Objectif** : Gérer connexion SSO externe

## 2. STRUCTURE PAGE
- Onglets (actif: SSO)
- Bannière info connexion
- Tableau SSO provider
- Détails identity provider

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsMessage** : Info SSO
- **OdsDatagrid** : Liste SSO
- **OdsCard** : Détails provider
- **OdsActionMenu** : Actions

## 4. ONGLETS
| Onglet | Active |
|--------|--------|
| Utilisateurs locaux | Non |
| Groupes d'utilisateurs | Non |
| SSO | Oui |

## 5. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Background | #E8F0FE |
| L1 | "Un identity provider externe est actuellement lié à ce compte. L'utilisateur devra" |
| L2 | "renseigner votre identifiant {NIC}-ovh/idp sans son mot de passe." |

## 6. COLONNES TABLEAU
| Colonne | Type |
|---------|------|
| SSO service URL | Lien bleu |
| Création | Date |
| Dernière modification | Date |
| Actions | Menu ⋮ |

## 7. ACTIONMENU OPTIONS
| Option | Couleur |
|--------|---------|
| Détails du SSO | Normal |
| Modifier le SSO | Normal |
| Supprimer le SSO | Rouge |

## 8. DÉTAILS IDENTITY PROVIDER
| Champ | Exemple |
|-------|---------|
| URL du service SSO | https://idp.mycompany.com/saml2/sso |
| Nom d'attribut de groupe | http://schemas.microsoft.com/ws/2008/06/identity/claims/groups |
| Certificats de signature | "2 certificats actifs" (lien) |

## 9. APIS & ENDPOINTS
- GET /me/identity/provider - Liste providers
- POST /me/identity/provider - Ajouter SSO
- PUT /me/identity/provider/{id} - Modifier
- DELETE /me/identity/provider/{id} - Supprimer
- GET /me/identity/provider/{id}/certificate - Certificats
