# Cahier des Charges : home/iam - Modal ajouter fournisseur SSO

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page SSO/SAML IAM
- **Objectif** : Configurer un nouveau fournisseur d'identité SAML

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCheckbox** : Utilisateurs locaux
- **OdsInput** : Attributs SAML
- **OdsTextarea** : Métadonnées XML
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un fournisseur SSO" |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Configurez un fournisseur d'identité SAML pour activer l'authentification SSO." |

## 6. SECTION TITRE
| Texte |
|-------|
| "Configuration du fournisseur d'identité" |

## 7. FORMULAIRE

### Champs
| Champ | Type | Placeholder |
|-------|------|-------------|
| Activer également les utilisateurs locaux | Checkbox | - |
| Attribut utilisateur | Input | ex: http://schemas.xmlsoap.org/ws/2005/identity |
| Attribut groupe | Input | ex: http://schemas.xmlsoap.org/claims/Group |
| Métadonnées XML SAML * | Textarea (120px) | XML EntityDescriptor |

### Exemple métadonnées
```xml
<?xml version="1.0"?>
<EntityDescriptor xmlns="urn:oasis:names:tc:SAML:2.0:metadata"
  entityID="https://idp.example.com">
  ...
</EntityDescriptor>
```

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Crée le fournisseur SSO |

## 9. APIS & ENDPOINTS
- POST /me/identity/provider/saml2 - Créer fournisseur SAML
