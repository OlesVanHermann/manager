# Cahier des Charges : home/iam - Modal configurer condition

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Création/édition de politique IAM
- **Objectif** : Configurer une condition d'accès (plage horaire, IP, etc.)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type de condition, Opérateur
- **OdsInput** : Valeurs (heures)
- **OdsMessage** : Info condition
- **OdsButton** : Annuler, Configurer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration de la condition" |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champs
| Champ | Type | Options/Valeur |
|-------|------|----------------|
| Type de condition | Select | Plage horaire, Adresse IP, Jour de semaine... |
| Opérateur | Select | Entre, Égal à, Différent de... |

### Valeurs (selon type)
| Type | Champs |
|------|--------|
| Plage horaire | Heure de début (09:00), Heure de fin (18:00) |
| Adresse IP | CIDR (192.168.1.0/24) |
| Jour de semaine | Multi-select jours |

## 6. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info (bleu) | "ℹ️ Cette condition restreint l'accès aux heures de bureau." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Configurer | Primary | Enregistre la condition |

## 8. APIS & ENDPOINTS
- Pas d'API directe - la condition est intégrée à la politique
- POST /me/iam/policy - Créer politique avec conditions
- PUT /me/iam/policy/{policyId} - Modifier politique
