## 8. APIS & ENDPOINTS

### 8.1 Lister les politiques OVH

```
GET /me/iam/policy?owner=ovh
```

Response:
```json
[
  {
    "policyId": "ovh:iam:admin",
    "name": "ovh:iam:admin",
    "description": "Accès administrateur complet à IAM",
    "owner": "ovh",
    "permissions": [
      "iam:*",
      "identity:*"
    ]
  }
]
```
