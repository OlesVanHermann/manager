## 6. APIS & ENDPOINTS

### 6.1 Obtenir les contacts d'un service

```
GET /service/{serviceId}/contacts
```

### 6.2 Modifier les contacts

```
POST /me/contact/change
{
  "serviceId": "string",
  "contactAdmin": "string",
  "contactTech": "string",
  "contactBilling": "string"
}
```
