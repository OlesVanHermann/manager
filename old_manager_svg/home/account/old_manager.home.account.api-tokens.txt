## 5. APIS & ENDPOINTS

### 5.1 Lister les tokens

```
GET /me/api/application
```

### 5.2 Créer un token

```
POST /me/api/credential
{
  "applicationId": "string",
  "redirection": "string",
  "allowedIPs": ["string"]
}
```

### 5.3 Révoquer un token

```
DELETE /me/api/credential/{credentialId}
```
