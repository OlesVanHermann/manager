# Cahier des Charges : home/account - Modal Ajouter SMS 2FA

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 2 étapes
- **URL** : N/A (composant modal)
- **Objectif** : Configurer l'authentification à deux facteurs par SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Indicateur 2 étapes (cercles)
- **OdsText** : Instructions
- **OdsInput** : Numéro téléphone
- **OdsButton** : Envoyer code, Navigation

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Numéro | Active (bleu) |
| 2 | Vérification | Inactive |

## 5. FORMULAIRE (ÉTAPE 1)
| Champ | Type | Requis | Placeholder | Helper |
|-------|------|--------|-------------|--------|
| Numéro de téléphone | Input tel | Oui | +33 6 12 34 56 78 | Format international requis |

## 6. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Envoyer le code SMS | Outlined Primary | Envoyer SMS vérification |
| Annuler | Secondary | Fermer modal |
| Suivant → | Primary | Passer à l'étape 2 |

## 7. APIS & ENDPOINTS
- POST /me/accessRestriction/sms - Ajouter SMS 2FA
- POST /me/accessRestriction/sms/{id}/sendCode - Envoyer code SMS
