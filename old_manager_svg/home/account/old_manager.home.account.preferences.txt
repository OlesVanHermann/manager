## 6. APIS & ENDPOINTS

### 6.1 Obtenir les préférences

```
GET /me/preferences
```

### 6.2 Mettre à jour les préférences

```
PUT /me
{
  "language": "fr_FR",
  "timezone": "Europe/Paris"
}
```

### 6.3 Mettre à jour les consentements

```
PUT /me/consent/{key}
{
  "value": true
}
```
