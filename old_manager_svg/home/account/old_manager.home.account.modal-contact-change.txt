# Cahier des Charges : home/account - Modal Modifier contacts

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification contacts
- **URL** : N/A (composant modal)
- **Objectif** : Modifier les contacts d'un service (admin, tech, billing)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Nom du service
- **OdsInput** : Champs NIC handle (3 champs)
- **OdsMessage** : Info validation par email (bleu)
- **OdsButton** : Actions

## 4. INFO SERVICE
| Champ | Exemple |
|-------|---------|
| Service | ns12345.ip-12-34-56.eu |

## 5. FORMULAIRE
| Champ | Type | Requis | Valeur exemple |
|-------|------|--------|----------------|
| Contact administrateur | Input text | Oui | ab12345-ovh |
| Contact technique | Input text | Oui | cd67890-ovh |
| Contact facturation | Input text | Oui | ab12345-ovh |

## 6. INFO
| Type | Message |
|------|---------|
| Info | "ℹ Le nouveau contact recevra un email pour accepter ou refuser le transfert." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Enregistrer | Primary | Envoyer demande de changement |

## 8. APIS & ENDPOINTS
- POST /me/contact/request - Créer demande de changement de contact
