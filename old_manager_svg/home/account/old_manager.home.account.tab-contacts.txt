# Cahier des Charges : home/account - Gestion des Contacts

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion contacts services
- **URL** : /contacts
- **Objectif** : Gérer les contacts admin/technique/facturation des services

## 2. STRUCTURE PAGE
- Sidebar avec "Mes contacts" actif
- Breadcrumb
- Titre "Gestion des contacts"
- Tabs (Mes services, Demandes envoyées, Demandes reçues)
- Filtres et recherche
- Datagrid des services

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation contacts
- **OdsButton** : Nouvelle demande
- **OdsSelect** : Filtre catégorie
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste services
- **OdsActionMenu** : Menu contextuel
