# Cahier des Charges : home/account - Modal ajout clé SSH

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page clés SSH
- **Objectif** : Ajouter une nouvelle clé SSH au compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom de la clé
- **OdsTextarea** : Clé publique
- **OdsMessage** : Info box
- **OdsButton** : Annuler, Ajouter la clé

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | Gris clair (#F9FAFB) |
| Titre | "Ajouter une clé SSH" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom de la clé | Input | Oui * | Identifiant de la clé |
| Clé publique | Textarea | Oui * | Contenu de la clé SSH |

## 6. TEXTAREA CLÉ PUBLIQUE
| Propriété | Valeur |
|-----------|--------|
| Hauteur | 120px |
| Police | Monospace |
| Placeholder | ssh-rsa AAAAB3NzaC1yc2EAAA... |
| Helper | "Collez votre clé publique SSH (commence par ssh-rsa, ssh-ed25519, etc.)" |

## 7. VALIDATION CLÉ
| État | Indicateur |
|------|------------|
| Format valide | ✓ Vert "Format de clé valide" |
| Format invalide | ✗ Rouge "Format de clé invalide" |

## 8. INFO BOX
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Cette clé sera disponible pour tous vos services." |

## 9. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter la clé | Primary | Enregistre la clé |

## 10. APIS & ENDPOINTS
- POST /me/sshKey - Ajouter clé SSH
- Paramètres : keyName, key
