# Cahier des Charges : home/account - Modal Changer Mot de Passe

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation
- **Déclencheur** : Clic "Changer mon mot de passe"
- **Objectif** : Confirmer demande de changement de mot de passe

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer
- Message explicatif
- Info box email
- Checkbox confirmation
- Footer avec actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x380px
- **OdsMessage** : Info box bleu
- **OdsCheckbox** : Confirmation
- **OdsButton** : Annuler, Confirmer
