# Cahier des Charges : public-cloud/ai-endpoints - Modal Supprimer clé API

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'une clé API
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer définitivement une clé

## 2. STRUCTURE MODAL
- Bande rouge en haut
- Header avec titre et bouton fermer (×)
- Message de confirmation
- Note irréversibilité
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x250)
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Supprimer (danger)

## 4. TITRE
"Supprimer la clé API"

## 5. BANDE COULEUR
- Bande rouge (#DC2626) en haut de la modal (6px)

## 6. MESSAGE
"Êtes-vous sûr de vouloir supprimer la clé API"
"{key-name}" (bold) ?
"Cette action est irréversible."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Supprimer | Danger rouge | Supprime la clé |

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/endpoints/token/{tokenId}
