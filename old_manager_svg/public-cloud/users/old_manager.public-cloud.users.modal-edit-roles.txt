# Cahier des Charges : public-cloud/users - Modal Éditer Rôles

## 1. VUE D'ENSEMBLE
- **Modal** : Éditer les rôles de l'utilisateur
- **Type** : Modal sélection multiple
- **Objectif** : Modifier les rôles d'un utilisateur existant

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x540px)
- Header avec titre
- Nom utilisateur
- Grille de rôles (checkboxes)
- Compteur rôles sélectionnés
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsCheckbox** : Sélection rôles
- **OdsCard** : Encarts rôles
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER
"Éditer les rôles de l'utilisateur"

## 5. MESSAGE
"Modifier les rôles de **user-dev-def456**"

## 6. RÔLES (avec état actuel)

| Rôle | État |
|------|------|
| Administrator | non sélectionné |
| Compute Operator | ✓ déjà sélectionné |

| Volume Operator | ✓ déjà sélectionné |
