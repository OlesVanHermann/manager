# Cahier des Charges : public-cloud/users - Liste des Utilisateurs

## 1. VUE D'ENSEMBLE
- **Page** : Users & Roles - Gestion des utilisateurs OpenStack
- **Type** : Dashboard avec datagrid
- **Objectif** : Gérer les utilisateurs et leurs rôles

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Users & Roles actif)
- Breadcrumb
- Titre
- Bouton ajouter + recherche + filtre
- Datagrid utilisateurs
- Lien matrice des rôles

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsButton** : Ajouter utilisateur, Filtrer
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste utilisateurs
- **OdsChip** : Rôles (bleu)
- **OdsBadge** : Statut (Activé)
- **OdsActionMenu** : Menu contextuel (ouvert)

## 4. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Nom d'utilisateur | Text | Username OpenStack |
| Description | Text muted | Description de l'utilisateur |
| Rôles | Chips | Liste des rôles + compteur "+N" |
| Date de création | Date | Format JJ/MM/AAAA |
| Status | Badge vert | Activé |
| Actions | ActionMenu | ⋮ |

- GET /cloud/project/{projectId}/user
