**Type:** Page dashboard projet
**Fonction:** Vue d'ensemble d'un projet Public Cloud
**Contexte:** Tableau de bord principal avec ressources et facturation

- Sidebar: 250x500px

## 9. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}
→ Informations projet

GET /cloud/project/{projectId}/usage/current
→ Facturation en cours

GET /cloud/project/{projectId}/instance
→ Compte des instances

GET /cloud/project/{projectId}/volume
→ Liste volumes (pour comptage)

GET /cloud/project/{projectId}/kube
→ Liste clusters Kubernetes

GET /cloud/project/{projectId}/region
→ Régions disponibles
```
