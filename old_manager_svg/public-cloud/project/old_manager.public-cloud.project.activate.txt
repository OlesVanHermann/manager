## 5. APIS & ENDPOINTS

### 5.1 Activer le projet

```
POST /cloud/project/{projectId}/activate
```

Body:
```json
{
  "paymentMean": "bankAccount|creditCard|paypal"
}
```

### 5.2 Lister les moyens de paiement

```
GET /me/payment/method
```

### 5.3 Ajouter un moyen de paiement

```
POST /me/payment/method
```
