## 7. APIS & ENDPOINTS

### 7.1 Mettre à jour le projet

```
PUT /cloud/project/{projectId}
```

Body:
```json
{
  "description": "mon-projet-production",
  "default": false
}
```

### 7.2 Activer la certification HDS

```
POST /cloud/project/{projectId}/hds
```

### 7.3 Supprimer le projet

```
DELETE /cloud/project/{projectId}
```

### 7.4 Obtenir les informations du projet

```
GET /cloud/project/{projectId}
```
