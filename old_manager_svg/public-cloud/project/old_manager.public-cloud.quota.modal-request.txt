## 6. APIS & ENDPOINTS

### 6.1 Envoyer la demande de quota

```
POST /cloud/project/{projectId}/quota/request
```

Body:
```json
{
  "region": "GRA",
  "resourceType": "instance",
  "currentQuota": 20,
  "requestedQuota": 50,
  "justification": "string"
}
```

### 6.2 Obtenir les quotas par région et type

```
GET /cloud/project/{projectId}/quota?region=GRA
```
