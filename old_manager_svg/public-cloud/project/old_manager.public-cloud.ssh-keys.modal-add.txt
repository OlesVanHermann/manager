# Cahier des Charges : public-cloud/ssh-keys - Modal Ajouter

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter une clé SSH
- **Type** : Modal formulaire
- **Objectif** : Ajouter une nouvelle clé SSH au projet

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x500px)
- Header avec titre et close
- Formulaire 2 champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom de la clé
- **OdsTextarea** : Clé publique SSH
- **OdsButton** : Annuler (secondary), Ajouter (primary)

## 4. HEADER
"Ajouter une clé SSH" + bouton ×

## 5. CHAMPS

### Nom de la clé *
- **Input** : Texte
- **Placeholder** : "Ex: macbook-pro-dev"

### Clé publique SSH *
- **Textarea** : Grande zone (180px hauteur)
- **Placeholder** : "Collez ici le contenu de votre fichier .pub"
- **Exemple** : "ssh-rsa AAAAB3NzaC1yc2E..."
- **Helper** : "Généralement situé dans ~/.ssh/id_rsa.pub ou ~/.ssh/id_ed25519.pub"

## 6. BOUTONS
- Annuler (secondary)
- Ajouter (primary)

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/sshkey
  - Body: { name, publicKey }
  - Crée une nouvelle clé SSH
