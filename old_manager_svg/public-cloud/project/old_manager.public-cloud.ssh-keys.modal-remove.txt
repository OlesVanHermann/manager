# Cahier des Charges : public-cloud/ssh-keys - Modal Supprimer (Variante)

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer la clé SSH
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'une clé SSH

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x250px)
- Barre rouge en haut
- Titre avec close
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsButton** : Annuler (secondary), Supprimer (danger)

## 4. HEADER
Barre rouge (6px) + "Supprimer la clé SSH" + bouton ×

## 5. MESSAGE
- "Êtes-vous sûr de vouloir supprimer la clé SSH **server-deploy-key** ?"
- "Cette action est irréversible. Les instances utilisant cette clé ne seront pas affectées."

## 6. BOUTONS
- Annuler (secondary)
- Supprimer (danger/rouge)

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/sshkey/{keyId}
  - Supprime la clé SSH
