## 6. APIS & ENDPOINTS

### 6.1 Créer un projet

```
POST /cloud/createProject
```

Body:
```json
{
  "description": "Mon nouveau projet",
  "voucher": "CODE_PROMO"
}
```

### 6.2 Valider un voucher

```
POST /cloud/createProjectInfo
```

Body:
```json
{
  "voucher": "CODE_PROMO"
}
```

### 6.3 Lister les moyens de paiement

```
GET /me/payment/method
```
