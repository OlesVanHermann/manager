# Cahier des Charges : public/cloud - Page Quota

## 1. VUE D'ENSEMBLE
- **Type** : Page quotas projet
- **Contexte** : Public Cloud
- **Objectif** : Visualiser les limites et utilisation des ressources

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsDatagrid** : Tableau quotas
- **OdsProgress** : Barres d'utilisation
- **OdsBadge** : État (OK, Critique)
- **OdsButton** : Demande augmentation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Quotas du projet" |
| Description | "Limites et utilisation des ressources." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Demander augmentation" |
| Style | Outline bleu (#EFF6FF) |
| Largeur | 150px |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Ressource | 280px | Nom ressource |
| Utilisé | 150px | Valeur utilisée |
| Quota | 130px | Limite max |
| Utilisation | 260px | Barre progression |
| % | 60px | Pourcentage |
| État | 80px | Badge état |

## 7. RESSOURCES SUIVIES
| Ressource | Utilisé | Quota | % |
|-----------|---------|-------|---|
| Instances | 12 | 50 | 24% |
| vCPU | 48 | 200 | 24% |
| RAM (GB) | 192 | 512 | 37% |
| Block Storage (TB) | 1.2 | 10 | 12% |
| Floating IPs | 8 | 10 | 80% |
| GPU (T4) | 2 | 8 | 25% |

## 8. COULEURS SEUILS
| Seuil | Couleur barre | Badge |
|-------|---------------|-------|
| 0-50% | #16A34A | OK (vert) |
| 50-75% | #F59E0B | OK (vert) |
| 75-100% | #DC2626 | Critique (rouge) |

| État | Fond | Texte |
|------|------|-------|

| Critique | #FEE2E2 | #DC2626 |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/quota - Quotas du projet
- POST /cloud/project/{id}/quota - Demander augmentation
