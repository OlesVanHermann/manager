## 8. APIS & ENDPOINTS

### 8.1 Lister tous les projets

```
GET /cloud/project
```

Response:
```json
[
  {
    "project_id": "abc123",
    "description": "mon-projet-production",
    "status": "ok",
    "unleash": true
  }
]
```

### 8.2 Créer un projet

```
POST /cloud/createProject
```

### 8.3 Supprimer un projet

```
DELETE /cloud/project/{projectId}
```
