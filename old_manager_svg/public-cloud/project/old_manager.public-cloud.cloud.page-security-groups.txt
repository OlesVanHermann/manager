# Cahier des Charges : public/cloud - Page Security Groups

## 1. VUE D'ENSEMBLE
- **Type** : Page Security Groups
- **Contexte** : Public Cloud
- **Objectif** : Gérer les règles de pare-feu pour les instances

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer groupe
- **OdsCard** : Cartes groupes (2x2 grille)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Security Groups" |
| Description | "Règles de pare-feu pour vos instances." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Créer groupe" |
| Style | Primary #0050D7 |
| Largeur | 180px |

## 6. DISPOSITION CARTES
| Propriété | Valeur |
|-----------|--------|
| Layout | Grille 2x2 |
| Largeur carte | 500px |
| Hauteur carte | 170px |
| Gap | 20px |

## 7. CONTENU CARTE
| Élément | Description |
|---------|-------------|
| Titre | Nom du groupe (bold, 15px) |
| Description | Description (gris, 12px) |
| Règles | Entrantes: X, Sortantes: Y |
| Instances | Nombre attachées |
| Boutons | Éditer, Règles |

## 8. EXEMPLES GROUPES
| Nom | Description | Entrantes | Sortantes | Instances |
|-----|-------------|-----------|-----------|-----------|
| default | Groupe par défaut | 2 | 1 | 12 |
| web-servers | HTTP/HTTPS | 4 | 1 | 4 |
| database | MySQL, PostgreSQL | 2 | 1 | 3 |
| monitoring | Outils monitoring | 3 | 2 | 2 |

## 9. BOUTONS CARTE
| Bouton | Style |
|--------|-------|
| Éditer | Outline bleu (#EFF6FF) |
| Règles | Outline gris |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/securityGroup - Liste groupes
- POST /cloud/project/{id}/securityGroup - Créer groupe
- GET /cloud/project/{id}/securityGroup/{sgId}/rule - Règles
- POST /cloud/project/{id}/securityGroup/{sgId}/rule - Ajouter règle
- DELETE /cloud/project/{id}/securityGroup/{sgId} - Supprimer
