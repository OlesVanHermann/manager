# Cahier des Charges : public-cloud/dashboard - Modal Configurer Datastore S3

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer un Datastore S3
- **Type** : Modal formulaire
- **Objectif** : Ajouter un datastore S3 pour les AI Tools

## 2. STRUCTURE MODAL
- Overlay sombre (40% opacité)
- Modal blanche centrée
- Header avec titre + fermeture
- Formulaire 5 champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsInput** : 5 champs
- **OdsSelect** : Région
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
- **Titre** : "Configurer un Datastore S3"
- **Fermeture** : × (coin droit)

## 5. CHAMPS FORMULAIRE

| Champ | Type | Placeholder/Valeur | Requis |
|-------|------|-------------------|--------|
| Alias | Input text | "alias" | oui |
| Endpoint | Input text | "https://s3.bhs.io.cloud.ovh.net/" | oui |
| Région | Select | "bhs" | oui |
| Access Key | Input text | "AKIAIOSFODNN7EXAMPLE" | oui |
| Secret Key | Input password | "••••••••••••••••••••••••" | oui |

## 6. BOUTONS FOOTER

| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Ajouter | Primary bleu | Droite |

## 7. VALIDATION
- Alias : format alphanumérique
- Endpoint : URL valide commençant par https://
- Access Key : format AWS standard
- Secret Key : non vide

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/data/region/{region}/alias
  - Body: { alias, endpoint, accessKey, secretKey }
