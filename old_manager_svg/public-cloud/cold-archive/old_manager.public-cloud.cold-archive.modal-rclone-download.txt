# Cahier des Charges : public-cloud/cold-archive - Modal Télécharger rClone

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal télécharger config rclone
- **Type** : Modal formulaire simple
- **Objectif** : Télécharger un fichier de configuration rclone

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 450x280px centrée
- Header avec titre et bouton fermer
- Description avec lien documentation
- Sélecteur de région
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (450x280px)
- **OdsText** : Description
- **OdsLink** : "En savoir plus ↗"
- **OdsSelect** : Sélection région
- **OdsButton** : Annuler (ghost/outline)
- **OdsButton** : Télécharger (primaire)

## 4. TITRE
"Télécharger un fichier de configuration rclone"

## 5. DESCRIPTION
"Vous allez télécharger un fichier de configuration permettant de vous connecter avec rclone."

### Lien documentation
"En savoir plus ↗" (bleu)

## 6. SÉLECTEUR RÉGION
- Label: "Sélectionner la région :"
- Type: OdsSelect dropdown
- Valeur: "Roubaix (RBX)"
- Icône dropdown: ▼

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Ghost/Outline bleu | Gauche |
| Télécharger | Primaire bleu | Droite |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/user/{userId}/rclone
  - Params: { region: string }
  - Télécharge le fichier de configuration rclone
