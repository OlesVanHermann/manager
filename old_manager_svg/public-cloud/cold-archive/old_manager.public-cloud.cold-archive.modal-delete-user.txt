# Cahier des Charges : public-cloud/cold-archive - Modal Supprimer Utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal supprimer credentials utilisateur
- **Type** : Modal danger/confirmation
- **Objectif** : Supprimer les credentials S3 d'un utilisateur

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 600x280px centrée
- Header avec titre et bouton fermer
- Bannière danger avec avertissement
- Question de confirmation
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (600x280px)
- **OdsMessage** : Danger banner (rouge clair)
- **OdsText** : Question de confirmation
- **OdsButton** : Annuler (outline)
- **OdsButton** : Confirmer (danger rouge)

## 4. TITRE
"Supprimer les credentials d'un utilisateur"

## 5. BANNIÈRE DANGER
- Fond: Rouge clair (#FEF2F2)
- Bordure: Rouge (#FCA5A5)
- Icône: ⚠ (rouge)
- Texte ligne 1: "Cette action est irréversible."
- Texte ligne 2: "Les credentials seront définitivement supprimés."
- Couleur texte: Rouge foncé (#991B1B)

## 6. QUESTION
"Êtes-vous sûr·e de vouloir supprimer cet utilisateur des Object Storage Policies?"

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Confirmer | Danger rouge (#DC2626) | Droite |

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/user/{userId}/s3Credentials
  - Supprime les credentials S3 de l'utilisateur
- DELETE /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/user/{userId}
  - Retire l'utilisateur du conteneur
