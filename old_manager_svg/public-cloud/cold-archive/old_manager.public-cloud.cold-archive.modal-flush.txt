# Cahier des Charges : public-cloud/cold-archive - Modal Vider Archive

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal vider archive
- **Type** : Modal danger/confirmation
- **Objectif** : Vider tous les objets d'une archive

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 600x400px centrée
- Header avec titre et bouton fermer
- Bannière warning avec détails archive
- Champ de confirmation texte
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (600x400px)
- **OdsMessage** : Warning banner (jaune)
- **OdsText** : Instructions de confirmation
- **OdsInput** : Champ de saisie "TERMINATE"
- **OdsButton** : Annuler (outline)
- **OdsButton** : Confirmer (danger rouge)

## 4. TITRE
"Voulez-vous vider votre archive ?"

## 5. BANNIÈRE WARNING
- Fond: Jaune (#FEF3C7)
- Bordure: Orange (#F59E0B)
- Titre: "⚠ Attention" (bold)
- Contenu:
  - "Votre archive **media-assets-archive** contient **4,521** objet(s)."
  - "Vous êtes sur le point de vider l'archive de tous ses objets."

## 6. CONFIRMATION
- Instruction: "Entrez \"TERMINATE\" dans le champ ci-dessous pour confirmer"
- Input: Champ texte avec placeholder "TERMINATE" (monospace)
- Validation: Doit correspondre exactement à "TERMINATE"

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Confirmer | Danger rouge (#DC2626) | Droite |

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/objects
  - Vide tous les objets du conteneur
  - Nécessite confirmation côté serveur
