# Cahier des Charges : public-cloud/cold-archive - Modal Modifier Rétention

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal modifier durée de rétention
- **Type** : Modal formulaire
- **Objectif** : Modifier la durée de rétention d'un conteneur archivé

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 700x480px centrée
- Header avec titre et bouton fermer
- Description avec nom conteneur
- Explication sur le verrouillage
- Input durée de rétention
- Message warning avec nouvelle date fin
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (700x480px)
- **OdsText** : Description et explications
- **OdsQuantity** : Nombre de jours (−/+)
- **OdsMessage** : Warning avec nouvelle date (jaune)
- **OdsButton** : Annuler (outline)
- **OdsButton** : Modifier (primaire)

## 4. TITRE
"Modifier la durée de rétention"

## 5. DESCRIPTION
"Vous êtes sur le point de modifier la durée de rétention de votre conteneur **compliance-data**."

"En configurant une durée de rétention, votre archive sera verrouillée et vous ne pourrez plus supprimer vos données pendant toute la période de rétention."

## 6. CHAMP DURÉE
- Label: "Ajouter le nombre de jours de rétention"
- Type: OdsQuantity
- Boutons: − / +
- Valeur: 365 jours

## 7. MESSAGE WARNING
- Fond: Jaune (#FEF3C7)
- Bordure: Orange (#F59E0B)
- Contenu:
  - "Votre nouvelle durée de rétention prendra fin le **27/12/2025**."
  - "Vous ne pourrez procéder à la suppression de vos données qu'une fois cette date passée."

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Modifier | Primaire bleu | Droite |

## 9. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/retention
  - Body: { retentionDays: number }
  - Modifie la durée de rétention (ne peut qu'être étendue)
- GET /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}
  - Récupère la rétention actuelle
