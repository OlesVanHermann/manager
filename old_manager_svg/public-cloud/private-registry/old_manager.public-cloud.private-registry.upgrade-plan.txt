# Cahier des Charges : public-cloud/private-registry - Changer de Plan

## 1. VUE D'ENSEMBLE
- **Page** : Managed Private Registry - Changer de plan
- **Type** : Page sélection de plan
- **Objectif** : Permettre l'upgrade/downgrade du plan du registre

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Private Registry actif)
- Breadcrumb
- Titre et description
- Cartes plans avec radio selection
- Note prorata
- Boutons action

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre, description
- **OdsCard** : Cartes plans (3)
- **OdsRadio** : Sélection plan (implicite dans carte)
- **OdsButton** : Annuler (outline), Changer de plan (primary)

## 4. HEADER
- **Titre** : "Changer de plan"
- **Description** : "Sélectionnez le plan adapté à vos besoins de stockage et d'images."

## 7. BOUTONS
- **Annuler** : Outline, retour page précédente
- **Changer de plan** : Primary, soumet le changement

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/containerRegistry/{registryId}
  - Récupère le registre et son plan actuel
- GET /cloud/project/{projectId}/containerRegistry/plan
  - Liste des plans disponibles
- PUT /cloud/project/{projectId}/containerRegistry/{registryId}/plan
  - Body: { planId: "medium" }
  - Change le plan du registre
