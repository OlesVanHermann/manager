# Cahier des Charges : public-cloud/private-registry - Onglet CIDR

## 1. VUE D'ENSEMBLE
- **Page** : Private Registry - Restrictions IP
- **Type** : Page liste des restrictions CIDR
- **Objectif** : Gérer les IPs autorisées à accéder au registre

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Private Registry actif)
- Breadcrumb
- Header registre (nom, description)
- Bouton ajouter
- Onglets (Informations, Restrictions IP actif)
- Tableau restrictions
- Compteur

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre, description
- **OdsButton** : "+ Ajouter une restriction" (primary)
- **OdsTabs** : Informations | Restrictions IP
- **OdsDatagrid** : Liste restrictions CIDR
- **OdsText** : Bloc IP (monospace)
- **OdsActionMenu** : Actions (⋮)

## 4. HEADER
- **Titre** : "my-registry"
- **Description** : "Gérez les restrictions d'accès IP à votre registry."
- **Action** : "+ Ajouter une restriction" (primary, 200px)

## 5. ONGLETS
| Onglet | Actif |
|--------|-------|
| Informations | - |
| Restrictions IP | ✅ (souligné bleu) |

## 6. TABLEAU RESTRICTIONS

### Colonnes
| Colonne | Description |
|---------|-------------|
| BLOC IP (CIDR) | Notation CIDR (monospace) |
| TYPE | Management, Registry, ou les deux |
| DESCRIPTION | Description libre |
| ACTIONS | Menu ⋮ |

### Données exemple
| CIDR | Type | Description |
|------|------|-------------|
| 192.168.1.0/24 | Management + Registry | Réseau interne bureau |
| 10.0.0.0/8 | Registry | VPN entreprise |
| 172.16.0.0/12 | Management | Cluster Kubernetes |
| 203.0.113.50/32 | Registry | CI/CD Runner |

## 7. ACTION MENU
Options (menu ouvert) :
- Modifier
- ---
- Supprimer (rouge)

## 8. COMPTEUR
"4 restrictions IP configurées"

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/containerRegistry/{registryId}/ipRestriction
  - Liste des restrictions IP
- POST /cloud/project/{projectId}/containerRegistry/{registryId}/ipRestriction
  - Ajoute une restriction
- PUT /cloud/project/{projectId}/containerRegistry/{registryId}/ipRestriction/{cidrId}
  - Modifie une restriction
- DELETE /cloud/project/{projectId}/containerRegistry/{registryId}/ipRestriction/{cidrId}
  - Supprime une restriction
