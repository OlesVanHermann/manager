# Cahier des Charges : public-cloud/private-registry - Modal Créer

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un registre privé
- **Type** : Modal formulaire complet
- **Objectif** : Créer un nouveau registre Harbor

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (800x680px)
- Titre + bouton fermer
- Formulaire multi-sections
- Récapitulatif prix
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsTile** : Cartes sélection (région, plan) avec radio
- **OdsRadio** : Sélection unique
- **OdsInput** : Nom du registre
- **OdsCheckbox** : Option IAM
- **OdsCard** : Récapitulatif estimation
- **OdsButton** : Annuler (secondary), Créer (primary)

## 4. HEADER
- **Titre** : "Créer un registre privé"
- **Bouton fermer** : ×
