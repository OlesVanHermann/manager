# Cahier des Charges : public-cloud/quota - Page Demande Support

## 1. VUE D'ENSEMBLE
- **Page** : Augmenter mes quotas (via support)
- **Type** : Page formulaire
- **Objectif** : Soumettre une demande d'augmentation de quotas au support

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Quota et Régions actif)
- Breadcrumb
- Titre "Augmenter mes quotas"
- Description
- Lien retour
- Info banner
- Formulaire 3 questions
- Boutons

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsLink** : Retour à la page précédente
- **OdsMessage** : Info banner (bleu)
- **OdsTextarea** : 3 champs textuels
- **OdsButton** : Annuler (outline), Confirmer (primary)

## 4. TITRE & DESCRIPTION
- **Titre** : "Augmenter mes quotas"
- **Description** : "Créez et suivez vos demandes d'assistance en fonction de vos besoins. Vous pouvez les consulter, les trier et accéder à celles déjà archivées."

## 5. LIEN RETOUR
"← Retour à la page précédente"

## 6. INFO BANNER
Fond bleu :
- **Titre** : "ℹ Information importante"
- **Message** : "Pour que votre demande d'augmentation soit prise en compte, veuillez remplir tous les champs de ce formulaire. Sans un remplissage complet, nous ne serons pas en mesure de répondre favorablement à votre demande."

## 7. FORMULAIRE

### Question 1
- **Label** : "Décrivez votre projet et l'utilisation prévue des ressources"
- **Placeholder exemple** : "Notre projet consiste à déployer une infrastructure de machine learning... Nous avons besoin de 20 instances GPU pour l'entraînement de modèles..."

### Question 2
- **Label** : "Quels types de ressources souhaitez-vous augmenter ?"
- **Placeholder exemple** :
  - "- Instances : de 10 à 50"
  - "- vCPU : de 40 à 200"
  - "- RAM : de 160 Go à 800 Go"

### Question 3
- **Label** : "Quelle est la date souhaitée pour cette augmentation ?"
- **Placeholder exemple** : "Nous souhaitons que cette augmentation soit effective d'ici le 15 janvier 2025."

## 8. BOUTONS
- Annuler (outline)
- Confirmer (primary)

## 9. APIS & ENDPOINTS
- POST /support/tickets
  - Body: { subject, description, category: "quota_increase" }
  - Crée un ticket support pour augmentation de quotas
