# Cahier des Charges : public-cloud/ai-tools - Modal Supprimer Token

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation suppression token
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'un token AI

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal centrée (500x220px)
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, message
- **OdsButton** : Annuler (outline), Confirmer (danger rouge)

## 4. HEADER
- **Titre** : "Supprimer un token"
- **Bouton fermer** : ✕

## 5. CONTENU
"Êtes-vous sûr de vouloir supprimer le token **my-training-token** ?"

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer modal |
| Confirmer | Danger rouge | DELETE token |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/token/{tokenId}
  - Supprimer le token
