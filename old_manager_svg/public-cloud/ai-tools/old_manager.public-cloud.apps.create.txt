# Cahier des Charges : public-cloud/apps - Déployer une App

## 1. VUE D'ENSEMBLE
- **Page** : Déployer une nouvelle application AI
- **Type** : Formulaire wizard avec résumé
- **Objectif** : Configurer et commander une app AI Deploy

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Deploy actif)
- Breadcrumb
- Titre page
- Formulaire (gauche)
- Résumé commande (droite)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsInput** : Champs texte
- **OdsSelect** : Sélection région/ressources
- **OdsTile** : Sélection visuelle
- **OdsCard** : Résumé commande
- **OdsButton** : Commander

## 4. BREADCRUMB
"Public Cloud › my-project › AI Deploy › Déployer une app"

## 5. CHAMPS FORMULAIRE

### Nom de l'app
- Type : Input text
- Placeholder : "my-inference-app"
- Requis : oui

### Localisation
- Type : Tile selector
- Options :
  - **GRA** - Gravelines (sélectionné)
  - **BHS** - Beauharnois
  - **DE** - Francfort

### Ressources
- Type : Tile selector
- Options :
  - **ai1-1-gpu** : 1 GPU · 13 vCores · 40GB RAM · 375GB SSD (sélectionné)
  - **ai1-1-cpu** : 4 vCores · 16GB RAM · 50GB SSD

### Application à déployer
- Type : Input text
- Placeholder : "registry.example.com/my-model:v1.0"
- Description : Image Docker à déployer

### Port Http
- Type : Input number
- Valeur : 8080
- Description : Port exposé par l'application

## 6. RÉSUMÉ COMMANDE
- Fond bleu clair (#F0F5FF)
- **Titre** : "Votre commande"

| Champ | Valeur |
|-------|--------|
| Nom | my-inference-app |
| Région | GRA |
| Ressources | ai1-1-gpu |
|  | 1 GPU · 13 vCores · 40GB RAM |

| Port Http | 8080 |
| Confidentialité | Public |

**Prix estimé** : 1.23 €/h

## 7. BOUTONS
- **Commander** : Primary bleu, dans le résumé

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/app
  - Body: { name, region, flavor, imageUrl, httpPort, ... }
- GET /cloud/project/{projectId}/ai/capabilities
  - Récupère les flavors et régions disponibles
