# Cahier des Charges : public-cloud/qpu - Liste des QPUs

## 1. VUE D'ENSEMBLE
- **Page** : Quantum (QPU) - Liste des notebooks quantiques
- **Type** : Dashboard avec datagrid
- **Objectif** : Gérer les notebooks quantiques

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Quantum QPU actif)
- Breadcrumb
- Titre + bouton créer
- Datagrid notebooks

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsButton** : Créer notebook
- **OdsDatagrid** : Liste notebooks
- **OdsBadge** : Statuts
- **OdsActionMenu** : Menu contextuel

## 4. BREADCRUMB
"Public Cloud › my-project › AI & Machine Learning › Quantum (QPU)"

## 5. TITRE ET BOUTON
- **Titre** : "QPUs"
- **Bouton** : "+ Créer un Notebook" (primary bleu)

## 6. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Name | Text | Nom du notebook |
| Région | Text | GRA, BHS |
| Framework | Text | Perceval, Qiskit |
| Ressources | Text | Flavor (ai1-1-cpu) |
| Durée | Text | Temps d'exécution |
| Statut | Badge | RUNNING/STOPPED |
| Actions | ActionMenu | ⋮ |

## 9. BADGES STATUT

| Statut | Couleur |
|--------|---------|
| RUNNING | Vert (#DEF7EC) |
| STOPPED | Gris (#F3F4F6) |

## 10. ACTION MENU

| Option | Style |
|--------|-------|
| Gérer | Normal |
| Arrêter | Normal |
| Supprimer | Danger rouge |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/notebook?framework=quantum
  - Liste des notebooks quantiques
- POST /cloud/project/{projectId}/ai/notebook
  - Créer un notebook quantique
  - Body: { name, region, framework, flavor }
- PUT /cloud/project/{projectId}/ai/notebook/{notebookId}/stop
  - Arrêter un notebook
- DELETE /cloud/project/{projectId}/ai/notebook/{notebookId}
  - Supprimer un notebook
