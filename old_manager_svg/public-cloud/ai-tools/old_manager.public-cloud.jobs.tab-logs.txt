- Fond sombre (#1F2937)
- Police monospace

| Niveau | Couleur |

| timestamp | Gris (#9CA3AF) |
| [INFO] | Vert (#10B981) |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/job/{jobId}/log
  - Récupère les logs
  - Query: { tail?, since?, until? }
- WebSocket /cloud/project/{projectId}/ai/job/{jobId}/log/stream
  - Streaming temps réel
