# Cahier des Charges : public-cloud/instances - Modal Start

## 1. VUE D'ENSEMBLE
- **Type** : Modal démarrage instance Public Cloud
- **Contexte** : Public Cloud Instances
- **Objectif** : Démarrer une instance arrêtée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 240px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Icône | ▶ sur cercle vert #DEF7EC |
| Titre | "Démarrage de votre instance" |
| Bouton fermer | × |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Texte | "Vous allez démarrer votre instance {nom}." |

## 6. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #D1D5DB | 100px |
| Confirmer | Primary #0050D7 | 100px |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/start - Démarrer
