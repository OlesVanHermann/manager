# Cahier des Charges : public-cloud/instances - Modal Démarrer

## 1. VUE D'ENSEMBLE
- **Modal** : Démarrage de votre instance
- **Type** : Modal confirmation simple
- **Objectif** : Confirmer le démarrage d'une instance arrêtée

## 2. STRUCTURE MODAL
- Overlay sombre avec fond page visible
- Conteneur blanc 600x240px centré
- Icône play verte
- Titre
- Message de confirmation avec nom instance
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Play vert (cercle vert clair avec ▶)
- **OdsText** : Titre et message
- **OdsButton** : Annuler (outline), Confirmer (primary)

## 4. TITRE
"Démarrage de votre instance"

## 5. MESSAGE
"Vous allez démarrer votre instance **my-instance-01**."

## 6. ICÔNE
- Cercle vert clair (#DEF7EC)
- Icône play ▶ en vert foncé (#03543F)

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Confirmer | Primary bleu | Droite | POST start |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/start
  - Démarrer l'instance
