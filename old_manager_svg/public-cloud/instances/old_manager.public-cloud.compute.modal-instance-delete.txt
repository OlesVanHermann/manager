# Cahier des Charges : public/compute - Modal Delete Instance

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression instance
- **Contexte** : Public Cloud Compute
- **Objectif** : Confirmer la suppression d'une instance

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 260px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et messages
- **OdsMessage** : Avertissement danger
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #FEE2E2 |
| Titre | "Supprimer l'instance" |
| Couleur titre | #DC2626 |
| Bouton fermer | × en #DC2626 |

## 5. CORPS MODAL
| Élément | Contenu |
|---------|---------|
| Question | "Êtes-vous sûr de vouloir supprimer cette instance ?" |

## 6. INFO INSTANCE
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Bordure | #E5E7EB |
| Label | "Instance" |
| Format | "nom (flavor • région)" |
| Exemple | "web-server-01 (b2-15 • GRA11)" |

## 7. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Background | #FEE2E2 |
| Icône | ⚠ |
| Titre | "Attention" |
| Couleur | #DC2626 |
| Message | "Les volumes attachés ne seront pas supprimés." |

## 8. BOUTONS FOOTER
| Bouton | Largeur | Style | Action |
|--------|---------|-------|--------|
| Annuler | 90px | Outline #E5E7EB | Fermer |
| Supprimer | 90px | Danger #DC2626 | DELETE |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{id}/instance/{instanceId} - Supprimer instance
