## 6. APIS & ENDPOINTS

### 6.1 Obtenir les informations d'accès

```
GET /cloud/project/{projectId}/instance/{instanceId}/applicationAccess
```

Response:
```json
{
  "accesses": [
    {
      "type": "http",
      "url": "http://192.168.1.50:8080",
      "login": "admin",
      "password": "xxxxx"
    },
    {
      "type": "ssh",
      "url": "ssh://ubuntu@192.168.1.50:22",
      "login": "ubuntu",
      "password": "xxxxx"
    }
  ]
}
```
