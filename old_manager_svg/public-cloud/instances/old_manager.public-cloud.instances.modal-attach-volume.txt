# Cahier des Charges : public-cloud/instances - Modal Attacher Volume

## 1. VUE D'ENSEMBLE
- **Modal** : Attacher un volume à votre instance
- **Type** : Modal sélection avec dropdown
- **Objectif** : Attacher un volume de stockage à une instance

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x340px
- Titre + bouton fermer
- Select de volumes avec dropdown ouvert
- Texte informatif avec lien création
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsSelect** : Sélection volume (dropdown)
- **OdsLink** : "Créer un volume →"
- **OdsButton** : Annuler, Attacher

## 4. TITRE
"Attacher un volume à votre instance"

## 5. LABEL
"Attacher une ressource"

## 6. SÉLECTEUR VOLUME
- Placeholder : "Sélectionner un volume existant"
- Dropdown ouvert avec options :

| Volume | Taille | État |
|--------|--------|------|
| volume-data-01 (sélectionné) | 100 GB | - |
| volume-backup-02 | 250 GB | - |
| volume-logs-03 | 50 GB | - |

## 7. TEXTE INFORMATIF
"Vous pouvez attacher un volume existant ou créer un nouveau volume à attacher à votre instance."

Lien : "Créer un volume →"

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Attacher | Primary bleu | Droite | POST attach volume |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/volume
  - Liste des volumes disponibles (non attachés)
- POST /cloud/project/{projectId}/volume/{volumeId}/attach
  - Body: { instanceId }
  - Attacher le volume à l'instance
