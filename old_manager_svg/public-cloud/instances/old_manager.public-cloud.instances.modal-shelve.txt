# Cahier des Charges : public-cloud/instances - Modal Suspendre

## 1. VUE D'ENSEMBLE
- **Modal** : Suspension de votre instance
- **Type** : Modal confirmation avec informations détaillées
- **Objectif** : Suspendre une instance pour réduire les coûts

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 700x520px
- Titre + bouton fermer
- Message de confirmation
- Banner info bleu (ressources libérées)
- Note sur la durée
- Banner warning jaune (facturation mensuelle)
- Liste des avantages
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre et messages
- **OdsMessage** : Info (bleu) et Warning (jaune)
- **OdsButton** : Annuler, Suspendre

## 4. TITRE
"Suspension de votre instance"

## 5. MESSAGE
"Vous allez suspendre votre instance **web-server-prod**."

## 6. BANNER INFO (BLEU)
- "Les ressources dédiées à votre instance Public Cloud seront relâchées mais son IP sera conservée."
- "Les données de votre disque local seront stockées, et seul le stockage utilisé pour cela sera facturé (1)."
- "Vous pourrez réactiver votre instance à tout moment."

## 7. NOTE DURÉE
"La durée de l'opération est en fonction de la taille de votre disque local."

## 8. BANNER WARNING (JAUNE)
"⚠️ (1) Attention : Pour les instances passées au mois, la facturation classique continuera tout de même, quelque soit le statut du service."

## 9. AVANTAGES
Liste à puces :
- Les ressources (CPU, RAM) sont libérées
- L'adresse IP est conservée
- Seul le snapshot est facturé (instances horaires)

## 10. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Suspendre | Primary bleu | Droite | POST shelve |

## 11. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/shelve
  - Suspendre l'instance (libère les ressources)
