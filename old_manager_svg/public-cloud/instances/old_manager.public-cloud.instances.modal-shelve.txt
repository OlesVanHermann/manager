# Cahier des Charges : public-cloud/instances - Modal Shelve

## 1. VUE D'ENSEMBLE
- **Type** : Modal suspension instance Public Cloud
- **Contexte** : Public Cloud Instances
- **Objectif** : Suspendre une instance pour arrêter la facturation

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 700px |
| Modal hauteur | 520px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 12px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et infos
- **OdsMessage** : Bannières info et warning
- **OdsButton** : Annuler, Suspendre

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Suspension de votre instance" |
| Bouton fermer | × |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Intro | "Vous allez suspendre votre instance {nom}." |

### Bannière info
| Propriété | Valeur |
|-----------|--------|
| Fond | #DBEAFE |
| Bordure | #3B82F6 |
| Point 1 | "Les ressources dédiées seront relâchées mais l'IP conservée" |
| Point 2 | "Les données du disque local seront stockées" |
| Point 3 | "Seul le stockage sera facturé (1)" |
| Point 4 | "Vous pourrez réactiver à tout moment" |

### Note durée
| Propriété | Valeur |
|-----------|--------|
| Texte | "La durée de l'opération dépend de la taille du disque local." |
| Couleur | #6B7280 |

### Bannière warning (facturation mensuelle)
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Icône | ⚠️ |
| Point 1 | "(1) Attention : Pour les instances au mois," |
| Point 2 | "la facturation classique continuera." |
| Couleur texte | #92400E |

### Avantages suspension
| Point | Valeur |
|-------|--------|
| 1 | Les ressources (CPU, RAM) sont libérées |
| 2 | L'adresse IP est conservée |
| 3 | Seul le snapshot est facturé (instances horaires) |

## 6. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Suspendre | Primary #0050D7 | 110px |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/shelve - Suspendre
