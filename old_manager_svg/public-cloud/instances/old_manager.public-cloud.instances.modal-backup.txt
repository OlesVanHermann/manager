# Cahier des Charges : public-cloud/instances - Modal Créer Backup

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un backup
- **Type** : Modal formulaire
- **Objectif** : Créer un snapshot/backup de l'instance

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x520px
- Titre + bouton fermer
- Info instance
- Champ nom du backup
- Info prix
- Banner warning jaune (impact performance)
- Option backup distant (checkbox)
- Champs backup distant (grisés par défaut)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom du backup
- **OdsMessage** : Warning (jaune)
- **OdsCheckbox** : Option backup distant
- **OdsSelect** : Localisation (conditionnel)
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Création d'un backup"

## 5. INFO INSTANCE
"Instance : **web-server-prod**"

## 6. CHAMP NOM
| Champ | Type | Label | Valeur exemple |
|-------|------|-------|----------------|
| Nom backup | Input | Saisissez le nom de votre backup | backup-web-server-prod-2024-01-15 |

## 7. INFO PRIX
"Prix : ~0,01 €/mois/Go"

## 8. BANNER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- "⚠️ Votre instance sera suspendue pendant quelques secondes, puis les performances en lecture/écriture du disque pourront être affectées jusqu'à la finalisation du backup"

## 9. OPTION BACKUP DISTANT
- Checkbox : "Ajouter un backup distant (Option)"
- Active les champs supplémentaires quand cochée

## 10. CHAMPS BACKUP DISTANT (conditionnels)
| Champ | Type | Label |
|-------|------|-------|
| Nom distant | Input | Nom du backup distant |
| Localisation | Select | Sélectionnez une localisation |

## 11. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Confirmer | Primary bleu | Droite | POST snapshot |

## 12. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/snapshot
  - Body: { snapshotName }
  - Créer un snapshot de l'instance
