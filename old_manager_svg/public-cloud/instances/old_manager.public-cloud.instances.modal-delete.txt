# Cahier des Charges : public-cloud/instances - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer une instance
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression définitive d'une instance

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x280px centré
- Titre + bouton fermer
- Icône warning orange centrale
- Question de confirmation avec nom instance
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Warning ⚠️ (cercle jaune bordure orange)
- **OdsText** : Titre et question
- **OdsButton** : Annuler, Supprimer (danger)

## 4. TITRE
"Supprimer une instance"

## 5. ICÔNE
- Cercle jaune clair (#FEF3C7) avec bordure orange (#F59E0B)
- Emoji ⚠️ au centre

## 6. QUESTION
"Êtes-vous sûr de vouloir supprimer l'instance **web-server-prod** ?"

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE instance |

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/instance/{instanceId}
  - Supprimer définitivement l'instance
