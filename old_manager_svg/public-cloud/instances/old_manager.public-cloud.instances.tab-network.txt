# Cahier des Charges : public-cloud/instances - Onglet Réseau

## 1. VUE D'ENSEMBLE
- **Page** : Onglet Réseau du détail instance
- **Type** : Vue avec tableaux IP publiques et privées
- **Objectif** : Afficher la configuration réseau de l'instance

## 2. STRUCTURE PAGE
- Header avec breadcrumb et infos instance
- Onglets : General, Réseau (actif), Snapshots, Console
- Titre "Configuration réseau"
- Deux cartes côte à côte :
  - IP Publiques
  - IP Privées

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes de tableaux
- **OdsTable** : Tableaux d'adresses IP
- **OdsButton** : "+ Attacher" (IP privées)

## 4. EN-TÊTE INSTANCE
- Breadcrumb : Public Cloud > Mon Projet Prod > {nom}
- Titre : {nom_instance}
- Info : {région} | {modèle} | ACTIVE (vert)

## 5. ONGLETS
- General
- **Réseau** (actif, souligné bleu)
- Snapshots
- Console

## 6. CARTE IP PUBLIQUES
Titre : "IP Publiques"

### Colonnes
| Colonne | Description |
|---------|-------------|
| Adresse IP | Adresse (monospace) |
| Version | IPv4 ou IPv6 |

### Données
| Adresse IP | Version |
|------------|---------|
| 51.210.10.25 | IPv4 |
| 2001:41d0:305:1000::1 | IPv6 |

## 7. CARTE IP PRIVÉES
Titre : "IP Privées"
Bouton : "+ Attacher" (bleu clair #E8F0FE)

### Colonnes
| Colonne | Description |
|---------|-------------|
| Adresse IP | Adresse (monospace) |
| Version | IPv4 ou IPv6 |

### Données
| Adresse IP | Version |
|------------|---------|
| 192.168.1.10 | IPv4 |
| 192.168.1.11 | IPv4 |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance/{instanceId}
  - Récupérer les IPs de l'instance (dans ipAddresses)
- GET /cloud/project/{projectId}/instance/{instanceId}/interface
  - Interfaces réseau attachées
- POST /cloud/project/{projectId}/instance/{instanceId}/interface
  - Attacher un réseau privé
