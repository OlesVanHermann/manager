## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance - Liste instances
- POST /cloud/project/{projectId}/instance - Créer
- DELETE /cloud/project/{projectId}/instance/{id} - Supprimer
- POST /cloud/project/{projectId}/instance/{id}/start - Démarrer
- POST /cloud/project/{projectId}/instance/{id}/stop - Arrêter
- POST /cloud/project/{projectId}/instance/{id}/reboot - Redémarrer
