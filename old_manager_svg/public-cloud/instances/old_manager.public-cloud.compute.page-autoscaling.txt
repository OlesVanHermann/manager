# Cahier des Charges : public/compute - Page Autoscaling

## 1. VUE D'ENSEMBLE
- **Type** : Page groupes autoscaling
- **Contexte** : Public Cloud Compute
- **Objectif** : Gérer les groupes d'instances auto-scalés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer groupe
- **OdsCard** : Cartes groupes
- **OdsBadge** : États des groupes

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Autoscaling Groups" |
| Description | "Groupes d'instances avec mise à l'échelle automatique." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Créer groupe" |
| Style | Primary #0050D7 |
| Largeur | 160px |

## 6. CARTE GROUPE AUTOSCALING
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 200px |
| Background | white |
| Bordure | #E5E7EB |

### Contenu carte
| Élément | Description |
|---------|-------------|
| Nom | Nom du groupe (bold) |
| Badge état | Active, Scaling, Idle |
| Info | Instances: X/min-max, Flavor, Région |
| Politique | Règles de scaling |

## 7. BADGES ÉTAT
| État | Fond | Texte |
|------|------|-------|
| Active | #DCFCE7 | #16A34A |
| Scaling | #FEF3C7 | #92400E |
| Idle | #F9FAFB | #6B7280 |

## 8. POLITIQUES DE SCALING
### Types de métriques
| Métrique | Exemple règle |
|----------|---------------|
| CPU | CPU > 70% → +1 instance |
| Queue | Queue > 1000 → +2 instances |
| Scheduled | 08:00-20:00 → 10 instances |

## 9. ACTIONS PAR CARTE
| Bouton | Style | Action |
|--------|-------|--------|
| Configurer | Primary outline | Ouvrir config |
| Instances | Outline | Voir liste instances |
| Métriques | Outline | Voir graphiques |

## 10. EXEMPLES GROUPES
| Nom | Instances | Flavor | Politique |
|-----|-----------|--------|-----------|
| web-frontend-asg | 4/2-8 | b2-15 | CPU-based |
| api-workers-asg | 6/3-10 | b2-30 | Queue-based |
| batch-processors-asg | 1/1-20 | c2-60 | Scheduled |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/autoscaling - Liste groupes
- POST /cloud/project/{id}/autoscaling - Créer groupe
- PUT /cloud/project/{id}/autoscaling/{groupId} - Modifier
- GET /cloud/project/{id}/autoscaling/{groupId}/instance - Instances
- DELETE /cloud/project/{id}/autoscaling/{groupId} - Supprimer
