- **Type** : Page liste snapshots
- **Contexte** : Public Cloud Compute
- **Objectif** : Gérer les instantanés d'instances

| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/snapshot - Liste snapshots
- POST /cloud/project/{id}/instance/{instanceId}/snapshot - Créer
- POST /cloud/project/{id}/snapshot/{snapshotId}/instance - Restaurer
- DELETE /cloud/project/{id}/snapshot/{snapshotId} - Supprimer
