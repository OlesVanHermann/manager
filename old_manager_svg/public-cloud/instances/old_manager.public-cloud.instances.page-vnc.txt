## 7. APIS & ENDPOINTS

### 7.1 Obtenir l'URL VNC

```
POST /cloud/project/{projectId}/instance/{instanceId}/vnc
```

Response:
```json
{
  "url": "wss://vnc.cloud.ovh.net/..."
}
```

### 7.2 Obtenir les informations de l'instance

```
GET /cloud/project/{projectId}/instance/{instanceId}
```
