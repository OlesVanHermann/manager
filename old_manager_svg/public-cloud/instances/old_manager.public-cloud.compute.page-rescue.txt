# Cahier des Charges : public/compute - Page Rescue

## 1. VUE D'ENSEMBLE
- **Type** : Page mode rescue
- **Contexte** : Public Cloud Compute
- **Objectif** : Activer le mode récupération sur les instances

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsMessage** : Avertissement warning
- **OdsDatagrid** : Liste instances
- **OdsButton** : Actions rescue
- **OdsCheckbox** : Sélection instances

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mode Rescue" |
| Description | "Démarrer une instance en mode de récupération." |

## 5. BANNIÈRE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Background | #FEF3C7 |
| Bordure | #F59E0B |
| Couleur texte | #92400E |
| Icône | ⚠️ |
| Titre | "Avertissement" |

### Messages d'avertissement
- "Le mode rescue redémarre votre instance avec un système temporaire."
- "Vos données restent accessibles via le montage du disque principal."
- "Un mot de passe temporaire sera envoyé par email."

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Checkbox | 30px | Sélection |
| Instance | 250px | Nom instance |
| IP | 180px | Adresse IP publique |
| État | 170px | Dot + texte |
| Image rescue | 180px | rescue-ovh |
| Action | 140px | Bouton action |

## 7. ÉTATS INSTANCE
| État | Couleur dot | Texte |
|------|-------------|-------|
| Running | #16A34A | "Running" |
| Rescue | #F59E0B | "Rescue" |
| Stopped | #6B7280 | "Stopped" |

## 8. BOUTONS ACTION
| Contexte | Label | Style |
|----------|-------|-------|
| Normal → Rescue | "Activer rescue" | Primary #0050D7 |
| Rescue → Normal | "Quitter rescue" | Outline #E5E7EB |
| Stopped | "Indisponible" | Disabled #E5E7EB |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{id}/instance - Liste instances
- POST /cloud/project/{id}/instance/{instanceId}/rescueMode - Activer rescue
- POST /cloud/project/{id}/instance/{instanceId}/reboot - Quitter rescue
