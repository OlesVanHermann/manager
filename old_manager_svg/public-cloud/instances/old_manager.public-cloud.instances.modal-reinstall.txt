# Cahier des Charges : public-cloud/instances - Modal Réinstaller

## 1. VUE D'ENSEMBLE
- **Modal** : Réinstaller une instance
- **Type** : Modal formulaire danger
- **Objectif** : Réinstaller l'OS de l'instance (supprime les données)

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x420px
- Titre + bouton fermer
- Question de confirmation
- Banner warning jaune
- Sélecteur nouvelle image
- Sélecteur clé SSH
- Message danger rouge
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsSelect** : Image, Clé SSH
- **OdsText** : Labels et messages danger
- **OdsButton** : Annuler, Réinstaller (danger)

## 4. TITRE
"Réinstaller une instance"

## 5. QUESTION
"Êtes-vous sûr de vouloir réinstaller l'instance **web-server-prod** ?"

## 6. BANNER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- "⚠️ Toutes les données seront supprimées."
- "Cette action est irréversible."

## 7. CHAMPS
| Champ | Type | Label | Valeur |
|-------|------|-------|--------|
| Image | Select | Sélectionnez une nouvelle image | Ubuntu 22.04 |
| Clé SSH | Select | Clé SSH | my-ssh-key |

## 8. MESSAGE DANGER
- Couleur : Rouge (#DC2626)
- "La réinstallation formatera le disque et supprimera toutes les données."

## 9. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Réinstaller | Danger rouge | Droite | POST reinstall |

## 10. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/reinstall
  - Body: { imageId, sshKeyId }
  - Réinstaller l'instance avec nouvelle image
