# Cahier des Charges : public-cloud/instances - Modal Mode Rescue

## 1. VUE D'ENSEMBLE
- **Modal** : Redémarrer en mode rescue
- **Type** : Modal formulaire avec options
- **Objectif** : Démarrer l'instance en mode rescue pour dépannage

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 700x540px
- Titre + bouton fermer
- Banner info bleu (explication)
- Sélection image rescue (radio)
- Sélecteur image custom (désactivé par défaut)
- Note sur le disque
- Info instance
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (bleu)
- **OdsRadio** : Choix type image
- **OdsSelect** : Sélecteur image (conditionnel)
- **OdsText** : Labels et messages
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Redémarrer en mode rescue"

## 5. BANNER INFO
"ℹ️ Si vous sélectionnez 'Distribution Rescue Made-in-OVH', vous recevrez un mot de passe temporaire avec lequel vous pourrez vous connecter à votre instance en mode rescue. Sinon, votre instance sera accessible avec votre clé SSH habituelle mais l'empreinte SSH de l'instance aura changé."

## 6. OPTIONS IMAGE
| Option | Sélection | Description |
|--------|-----------|-------------|
| Distribution Rescue Made-in-OVH | Sélectionné | Image OVH avec accès temporaire |
| Utiliser mon image | Non sélectionné | Image personnalisée |

## 7. SÉLECTEUR IMAGE CUSTOM
- Désactivé si "Rescue Made-in-OVH" sélectionné
- Placeholder : "Sélectionner une image..."
- Liste déroulante des images disponibles

## 8. NOTE
"Le disque principal sera disponible en tant que second disque non monté."

## 9. INFO INSTANCE
- Label : "Instance concernée :"
- Valeur : **web-server-prod**

## 10. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Confirmer | Primary bleu | Droite | POST rescue |

## 11. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/rescue
  - Body: { imageId?, rescue: true }
  - Démarrer en mode rescue
