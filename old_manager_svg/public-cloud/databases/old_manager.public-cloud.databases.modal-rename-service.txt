# Cahier des Charges : public-cloud/databases - Modal Modifier le nom

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le nom
- **Type** : Modal formulaire
- **Objectif** : Changer le nom d'affichage du service

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x220px
- Header avec titre et description
- Formulaire avec 1 champ
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le nom"

## 5. DESCRIPTION
"Modifie le nom affiché dans l'en-tête"

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Valeur actuelle | Requis |
|-------|------|-----------------|--------|
| Nom | Input | my-postgresql-db | Oui |

### Notes
- Ce nom est uniquement pour l'affichage
- N'affecte pas l'identifiant technique du service
- Caractères alphanumériques et tirets autorisés

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Modifier | Primary | Droite | PUT description |

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/database/service/{serviceId}
  - Body: { description }
  - Modifier le nom d'affichage
