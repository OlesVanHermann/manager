# Cahier des Charges : public-cloud/databases - Modal Ajouter pool de connexion

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un pool de connexion
- **Type** : Modal formulaire
- **Objectif** : Créer un pool de connexion PostgreSQL (pgBouncer)

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x460px
- Header avec titre et description
- Formulaire avec 5 champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom, Taille
- **OdsSelect** : Base de données, Mode, Utilisateur
- **OdsButton** : Annuler, Créer un pool de connexion

## 4. TITRE
"Ajouter un pool de connexion"

## 5. DESCRIPTION
"Créer un nouveau pool de connexion pour une base de données de votre service."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder/Valeur | Requis |
|-------|------|-------------------|--------|
| Nom du pool de connexion | Input | my_pool | Oui |
| Base de données | Select | defaultdb | Oui |
| Mode | Select | session | Oui |
| Taille | Input | 10 | Oui |
| Nom d'utilisateur | Select | Aucun utilisateur | Non |

### Options Mode
| Mode | Description |
|------|-------------|
| session | Une connexion par session client |
| transaction | Une connexion par transaction |
| statement | Une connexion par statement |

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Créer un pool de connexion | Primary | Droite | POST pool |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/postgresql/{serviceId}/database
  - Liste des bases de données
- GET /cloud/project/{projectId}/database/postgresql/{serviceId}/user
  - Liste des utilisateurs
- POST /cloud/project/{projectId}/database/postgresql/{serviceId}/connectionPool
  - Body: { name, database, mode, size, username? }
  - Créer un pool de connexion
