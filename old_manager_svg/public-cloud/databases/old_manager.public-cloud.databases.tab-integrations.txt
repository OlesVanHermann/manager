# Cahier des Charges : public-cloud/databases - Onglet Intégrations

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Intégrations (Kafka)
- **Type** : Page onglet listing
- **Objectif** : Gérer les intégrations entre services (Kafka, OpenSearch, Grafana)

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec service actif
- Breadcrumb avec "Intégrations"
- Onglets (Intégrations actif)
- Titre "Intégrations"
- Bouton ajouter intégration
- Tableau des intégrations
- ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Intégrations actif
- **OdsButton** : "+ Ajouter une intégration" (outline)
- **OdsTable** : Liste des intégrations
- **OdsBadge** : Statut (READY)
- **OdsActionMenu** : Actions

## 4. TITRE
"Intégrations"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | Inactif |
| Utilisateurs | Inactif |
| Logs | Inactif |
| Métriques | Inactif |

## 6. TABLEAU INTÉGRATIONS
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Source | Texte | Service source |
| Destination | Texte | Service destination |
| Type | Texte | Type d'intégration |
| Statut | Badge pill | READY |
| Actions | Menu ⋮ | ActionMenu |

### Types d'intégration
- opensearchLogs
- grafanaDatasource
- kafkaMirrormaker

### Données exemple
| Source | Destination | Type | Statut |
|--------|-------------|------|--------|
| my-kafka-cluster | my-opensearch | opensearchLogs | READY |
| my-kafka-cluster | my-grafana | grafanaDatasource | READY |
| my-external-kafka | my-kafka-cluster | kafkaMirrormaker | READY |

## 7. ACTION MENU (ouvert)
| Action | Type |
|--------|------|
| Supprimer | Danger (rouge) |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/service/{serviceId}/integration
  - Liste des intégrations
- POST /cloud/project/{projectId}/database/service/{serviceId}/integration
  - Body: { type, destinationServiceId }
  - Créer une intégration
- DELETE /cloud/project/{projectId}/database/service/{serviceId}/integration/{integrationId}
  - Supprimer une intégration
