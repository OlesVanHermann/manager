# Cahier des Charges : public-cloud/databases - Modal Supprimer intégration

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer une intégration
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'une intégration

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Supprimer une intégration"

## 5. MESSAGE
"Vous allez supprimer votre intégration {integration_name}. Souhaitez-vous continuer ?"

### Variables
- `{integration_name}` : Nom de l'intégration (ex: kafkaLogs)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Confirmer | Primary | Droite | DELETE integration |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/service/{serviceId}/integration/{integrationId}
  - Supprimer une intégration
