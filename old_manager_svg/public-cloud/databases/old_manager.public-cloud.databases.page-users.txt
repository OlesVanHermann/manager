|--------|-------|
| Modifier permissions | Normal |

| --- | Séparateur |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/database/{dbId}/user - Liste users
- POST /cloud/project/{id}/database/{dbId}/user - Créer user
- PUT /cloud/project/{id}/database/{dbId}/user/{userId} - Modifier
- DELETE /cloud/project/{id}/database/{dbId}/user/{userId} - Supprimer
