# Cahier des Charges : public-cloud/databases - Onglet Configuration

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Configuration
- **Type** : Page onglet formulaire
- **Objectif** : Configurer les paramètres d'un service DB

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec service actif
- Breadcrumb avec "Configuration"
- Onglets (Configuration actif)
- Titre "Configuration"
- Carte "Service" avec formulaire
- Zone de danger

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Configuration actif
- **OdsCard** : Carte configuration
- **OdsInput** : Nom du service
- **OdsInput** : Heure de maintenance
- **OdsInput** : Heure de backup
- **OdsToggle** : Protection suppression
- **OdsCard** : Zone de danger (fond rouge)
- **OdsButton** : "Supprimer le service" (danger)

## 4. TITRE
"Configuration"

## 5. CARTE "SERVICE"
### Champs
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Nom | OdsInput text | my-postgresql-db |
| Heure de maintenance (UTC) | OdsInput time | 03:00 |
| Heure de backup (UTC) | OdsInput time | 04:00 |
| Protection contre la suppression | OdsToggle | Activée (on) |

### Toggle Protection
- État: Activé (fond bleu #0050D7)
- Label: "Activée"

## 6. ZONE DE DANGER
- Fond: Rouge clair (#FEF2F2)
- Bordure: Rouge (#FCA5A5)
- Titre: "Zone de danger" (rouge #991B1B)
- Bouton: "Supprimer le service"
  - Fond: #FEE2E2
  - Bordure: #991B1B
  - Texte: Rouge (#991B1B)

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/database/service/{serviceId}
  - Body: { name, maintenanceTime, backupTime }
  - Modifier la configuration
- PUT /cloud/project/{projectId}/database/service/{serviceId}/maintenance
  - Paramètres de maintenance
- DELETE /cloud/project/{projectId}/database/service/{serviceId}
  - Supprimer le service (si protection désactivée)
