# Cahier des Charges : public-cloud/databases - Modal Informations pool de connexion

## 1. VUE D'ENSEMBLE
- **Modal** : Informations sur le pool de connexion
- **Type** : Modal information (lecture seule)
- **Objectif** : Afficher les détails de connexion d'un pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x360px
- Header avec titre
- Tableau d'informations (7 lignes)
- Pas de footer (modal informative)

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsTable** : Liste des propriétés (key-value)
- **OdsClipboard** : Copier URI/certificat
- **OdsButton** : Télécharger certificat

## 4. TITRE
"Informations sur le pool de connexion"

## 5. TABLEAU D'INFORMATIONS
| Propriété | Valeur | Actions |
|-----------|--------|---------|
| Base de données | defaultdb | - |
| Port | 20184 | - |
| Mode SSL | require | - |
| Certificat CA | -----BEGIN CERTIFICATE-----... | Copier, Télécharger |
| Service URI | postgresql://user:pass@host... | Copier |
| Mode | session | - |
| Taille | 10 | - |

### Style du tableau
- Alternance de couleur de fond (gris clair / blanc)
- Valeurs tronquées pour URI et certificat

## 6. ACTIONS SUR LES VALEURS
| Valeur | Actions disponibles |
|--------|---------------------|
| Certificat CA | Copier dans le presse-papier, Télécharger fichier |
| Service URI | Copier dans le presse-papier |

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/postgresql/{serviceId}/connectionPool/{poolId}
  - Détails du pool de connexion
- GET /cloud/project/{projectId}/database/service/{serviceId}/certificates
  - Certificat CA
