# Cahier des Charges : public-cloud/databases - Modal Modifier les ressources (Flavor)

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier les ressources (Flavor)
- **Type** : Modal sélection tableau
- **Objectif** : Changer la configuration des ressources (vCores, RAM, stockage)

## 2. STRUCTURE MODAL
- Overlay sombre avec fond page visible
- Conteneur blanc 900x770px
- Header avec titre et bouton fermer
- Description
- Configuration actuelle (card)
- Tableau des flavors disponibles
- Warning box impacts
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur large
- **OdsCard** : Configuration actuelle
- **OdsTable** : Sélection des flavors
- **OdsBadge** : "Actuel", "✓ Sélectionné"
- **OdsMessage** : Warning impacts
- **OdsButton** : Annuler, Appliquer

## 4. TITRE
"Modifier les ressources (Flavor)"

## 5. DESCRIPTION
"Sélectionnez les ressources adaptées à vos besoins. Le changement sera appliqué lors de la prochaine fenêtre de maintenance ou immédiatement si vous le souhaitez."

## 6. CONFIGURATION ACTUELLE
- Fond: Gris (#F3F4F6)
- Label: "Configuration actuelle"
- Valeur: "db1-7 • 2 vCores • 7 Go RAM • 80 Go SSD"
- Prix: "~45,00 € HT/mois"

## 7. TABLEAU FLAVORS
### Colonnes
| Colonne | Description |
|---------|-------------|
| Type | Nom du flavor (db1-4, db1-7, etc.) |
| vCores | Nombre de cœurs virtuels |
| Mémoire | RAM en Go |
| Stockage | Stockage SSD en Go |
| Prix estimé | Prix mensuel HT |
| Statut | Actuel ou Sélectionné |

### Données exemple
| Type | vCores | Mémoire | Stockage | Prix | Statut |
|------|--------|---------|----------|------|--------|
| db1-4 | 2 | 4 Go | 50 Go SSD | ~30,00 € | - |
| db1-7 | 2 | 7 Go | 80 Go SSD | ~45,00 € | Actuel |
| db1-15 | 4 | 15 Go | 160 Go SSD | ~90,00 € | ✓ Sélectionné |
| db1-30 | 8 | 30 Go | 320 Go SSD | ~180,00 € | - |
| db1-60 | 16 | 60 Go | 640 Go SSD | ~360,00 € | - |

### Ligne sélectionnée
- Fond: Bleu clair (#E8F0FE)
- Bordure: Bleue (#0050D7)
- Texte: Bleu

## 8. WARNING BOX
- Fond: Jaune (#FEF3C7)
- Titre: "⚠️ Impact de la modification"
- Points:
  - "Le service sera brièvement indisponible pendant la migration (~2-5 minutes)."
  - "Différence de prix: +45,00 € HT/mois"

## 9. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Appliquer | Primary bleu | Droite | PUT update flavor |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/capabilities/flavor
  - Liste des flavors disponibles
- PUT /cloud/project/{projectId}/database/service/{serviceId}
  - Body: { flavor }
  - Changer le flavor
