# Cahier des Charges : public-cloud/databases - Modal Certificat

## 1. VUE D'ENSEMBLE
- **Modal** : Certificat
- **Type** : Modal information (lecture seule)
- **Objectif** : Afficher et permettre le téléchargement du certificat CA

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x300px
- Header avec titre
- Bloc de code avec certificat
- Footer avec bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsCode** : Bloc de code sombre
- **OdsButton** : Télécharger (dans le bloc), Fermer

## 4. TITRE
"Certificat"

## 5. BLOC DE CODE
### Structure
- Header gris foncé (#374151) avec label "Certificat"
- Bouton "Télécharger" dans le header
- Corps noir (#1F2937) avec le certificat

### Contenu
```
-----BEGIN CERTIFICATE-----
MIIDrzCCApegAwIBAgIQCDvgVpBCRr
GZnFcBbBSD3Vy4PNBf5J9Y3LBXhZ9O
iHBKh+v/1ORqJTmLOJLBpJ3iCgYEAw
...
-----END CERTIFICATE-----
```

## 6. ACTIONS
| Action | Description |
|--------|-------------|
| Télécharger | Télécharger le certificat en fichier .crt ou .pem |
| Fermer | Fermer la modal |

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Télécharger | Small gris | Header du bloc | Download fichier |
| Fermer | Outline | Footer droite | Fermer modal |

## 8. USAGE
- Requis pour les connexions SSL/TLS
- À installer côté client pour vérifier le serveur

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/service/{serviceId}/certificates
  - Récupérer le certificat CA
