# Cahier des Charges : public-cloud/databases - Onglet ACL (Kafka)

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet ACL (Kafka)
- **Type** : Page onglet listing
- **Objectif** : Gérer les Access Control Lists Kafka

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec service Kafka actif
- Breadcrumb avec "ACL"
- Onglets (ACL actif)
- Titre "ACL"
- Bouton ajouter ACL
- Tableau des ACL
- ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec ACL actif
- **OdsButton** : "+ Ajouter une ACL" (outline)
- **OdsTable** : Liste des ACL
- **OdsBadge** : Permission (read, write, admin)
- **OdsActionMenu** : Actions

## 4. TITRE
"ACL"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | Inactif |
| Utilisateurs | Inactif |
| Topics | Inactif |
| ACL | Actif (souligné bleu) |
| Logs | Inactif |
| Connectors | Inactif |
| Paramètres | Inactif |

## 6. TABLEAU ACL
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Utilisateur | Texte | Nom utilisateur |
| Topic | Texte | Pattern topic (* ou spécifique) |
| Permission | Badge pill | read / write / admin |
| Actions | Menu ⋮ | ActionMenu |

### Badges Permission
| Permission | Couleur |
|------------|---------|
| read | Bleu (#DBEAFE, #1E40AF) |
| write | Jaune (#FEF3C7, #92400E) |
| admin | Vert (#DEF7EC, #03543F) |

### Données exemple
| Utilisateur | Topic | Permission |
|-------------|-------|------------|
| admin | * | admin |
| app_producer | orders-* | write |
| app_consumer | orders-events | read |

## 7. ACTION MENU (ouvert)
| Action | Type |
|--------|------|
| Supprimer | Danger (rouge) |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/kafka/{serviceId}/acl
  - Liste des ACL
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/acl
  - Body: { username, topic, permission }
  - Créer une ACL
- DELETE /cloud/project/{projectId}/database/kafka/{serviceId}/acl/{aclId}
  - Supprimer une ACL
