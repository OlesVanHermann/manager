# Cahier des Charges : public-cloud/databases - Modal Supprimer la réplication

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer la réplication
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'une configuration de réplication

## 2. STRUCTURE MODAL
- Overlay sombre avec fond page visible
- Conteneur blanc 600x380px
- Header avec titre et bouton fermer
- Icône warning + message
- Détails de la réplication (card)
- Warning box rouge
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Warning rouge
- **OdsText** : Message de confirmation
- **OdsCard** : Détails source/cible
- **OdsMessage** : Conséquences (danger)
- **OdsButton** : Annuler, Supprimer (danger)

## 4. TITRE
"Supprimer la réplication"

## 5. MESSAGE
"Êtes-vous sûr de vouloir supprimer cette réplication ?"

## 6. DÉTAILS DE LA RÉPLICATION
- Fond: Gris (#F3F4F6)
- Colonnes:
  - Source: mysql-prod-db-abc123
  - Cible: mysql-replica-xyz789
  - Info: "Type: Asynchrone • Tables: * (toutes)"

## 7. WARNING BOX
- Fond: Rouge clair (#FEE2E2)
- Titre: "⚠️ Conséquences de la suppression"
- Points:
  - "La synchronisation des données sera immédiatement arrêtée"
  - "Les données existantes sur la cible seront conservées"
  - "Cette action ne peut pas être annulée"

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE replication |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/service/{serviceId}/replication/{replicationId}
  - Supprimer la configuration de réplication
