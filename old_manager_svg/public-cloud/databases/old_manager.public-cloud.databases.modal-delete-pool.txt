# Cahier des Charges : public-cloud/databases - Modal Supprimer pool de connexion

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un pool de connexion
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'un pool de connexion

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Supprimer un pool de connexion"

## 5. MESSAGE
"Voulez-vous vraiment supprimer le pool de connexion {pool_name} ?"

### Variables
- `{pool_name}` : Nom du pool (ex: my_pool)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Confirmer | Primary | Droite | DELETE pool |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/postgresql/{serviceId}/connectionPool/{poolId}
  - Supprimer un pool de connexion
