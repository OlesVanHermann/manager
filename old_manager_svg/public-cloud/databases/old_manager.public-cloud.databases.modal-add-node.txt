# Cahier des Charges : public-cloud/databases - Modal Ajouter nœud

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un nœud
- **Type** : Modal confirmation avec estimation
- **Objectif** : Ajouter un nœud au cluster avec estimation de coût

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x340px
- Header avec titre et bouton fermer
- Toggle affichage prix (Heure/Mois)
- Message confirmation avec estimation
- Info box
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsToggle** : Affichage prix (Heure/Mois)
- **OdsText** : Message et estimation
- **OdsMessage** : Info box grise
- **OdsButton** : Annuler, Valider

## 4. TITRE
"Ajouter un nœud"

## 5. TOGGLE AFFICHAGE PRIX
| Option | État |
|--------|------|
| Heure | Inactif |
| Mois | Actif (bleu) |

## 6. MESSAGE PRINCIPAL
"Vous disposez actuellement de **{count} nœud(s)**.
Voulez-vous vraiment en ajouter un ?

Votre facture sera augmentée de **{price} € HT** par mois."

### Variables
- `{count}` : Nombre actuel de nœuds (ex: 3)
- `{price}` : Prix mensuel additionnel (ex: 45,00)

## 7. INFO BOX
- Fond: Gris (#F3F4F6)
- Texte:
  - "La modification sera effective dans quelques minutes."
  - "Vous serez notifié une fois l'opération terminée."

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Valider | Primary bleu | Droite | POST add node |

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/service/{serviceId}/node
  - Ajouter un nœud
- GET /cloud/project/{projectId}/database/capabilities/flavor/{flavorId}
  - Prix du flavor pour estimation
