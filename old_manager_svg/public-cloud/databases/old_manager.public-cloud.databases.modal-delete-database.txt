# Cahier des Charges : public-cloud/databases - Modal Supprimer base de données

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer la base de données
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'une base de données

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer la base de données

## 4. TITRE
"Supprimer la base de données"

## 5. MESSAGE
"Voulez-vous vraiment supprimer la base de données {database_name} ?"

### Variables
- `{database_name}` : Nom de la base à supprimer (ex: my_database)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Supprimer la base de données | Primary | Droite | DELETE database |

## 7. AVERTISSEMENT
- Action irréversible
- Toutes les données seront perdues

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/{engine}/{serviceId}/database/{databaseId}
  - Supprimer une base de données
