# Cahier des Charges : public-cloud/databases - Modal Regénérer mot de passe

## 1. VUE D'ENSEMBLE
- **Modal** : Regénérer le mot de passe
- **Type** : Modal confirmation
- **Objectif** : Générer un nouveau mot de passe pour un utilisateur

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x180px
- Header avec titre
- Message de confirmation sur 2 lignes
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Regénérer le mot de passe"

## 5. MESSAGE
"Êtes-vous sûr de vouloir regénérer le mot de passe
de l'utilisateur {username} ?"

### Variables
- `{username}` : Nom de l'utilisateur (ex: admin_user)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Confirmer | Primary | Droite | POST reset password |

## 7. COMPORTEMENT POST-ACTION
- Après confirmation, afficher le nouveau mot de passe
- Le mot de passe n'est visible qu'une seule fois
- Proposer copie dans le presse-papier

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/{engine}/{serviceId}/user/{userId}/credentials/reset
  - Regénérer le mot de passe
  - Retourne: { password }
