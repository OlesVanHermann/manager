# Cahier des Charges : public-cloud/databases - Modal Supprimer topic (Kafka)

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le topic
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'un topic Kafka

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 420x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer le topic

## 4. TITRE
"Supprimer le topic"

## 5. MESSAGE
"Voulez-vous vraiment supprimer le topic {topic_name} ?"

### Variables
- `{topic_name}` : Nom du topic (ex: my_topic)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Supprimer le topic | Primary | Droite | DELETE topic |

## 7. AVERTISSEMENT
- Tous les messages du topic seront perdus
- Les consumers ne pourront plus lire ce topic

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/kafka/{serviceId}/topic/{topicId}
  - Supprimer un topic
