# Cahier des Charges : public-cloud/databases - Modal Ajouter utilisateur

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un utilisateur
- **Type** : Modal formulaire
- **Objectif** : Créer un nouvel utilisateur pour le service

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x440px
- Header avec titre et description
- Formulaire avec 4 champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom d'utilisateur
- **OdsInput** : Groupe (optionnel selon engine)
- **OdsSelect** : Rôles
- **OdsInput** : Clés (optionnel selon engine)
- **OdsButton** : Annuler, Créer un utilisateur

## 4. TITRE
"Ajouter un utilisateur"

## 5. DESCRIPTION
"Créer et configurer un nouvel utilisateur pour votre service."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nom d'utilisateur | Input | name | Oui |
| Groupe | Input | group | Selon engine |
| Rôles | Select | Sélectionnez un rôle | Selon engine |
| Clés | Input | - | Selon engine (Redis) |

### Notes par engine
- **PostgreSQL/MySQL** : username, roles (readWrite, read, admin)
- **MongoDB** : username, group, roles
- **Redis** : username, keys, categories, commands

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Créer un utilisateur | Primary | Droite | POST user |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/{engine}/{serviceId}/user
  - Body: { name, group?, roles?, keys? }
  - Créer un utilisateur
