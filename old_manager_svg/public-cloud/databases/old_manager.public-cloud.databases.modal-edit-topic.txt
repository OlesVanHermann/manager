# Cahier des Charges : public-cloud/databases - Modal Modifier topic (Kafka)

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier un topic
- **Type** : Modal formulaire
- **Objectif** : Modifier les propriétés d'un topic Kafka existant

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x420px
- Header avec titre et description
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom (disabled), Réplication, Partitions
- **OdsAccordion** : Configuration avancée
- **OdsButton** : Annuler, Modifier le topic

## 4. TITRE
"Modifier un topic"

## 5. DESCRIPTION
"Modifiez un topic Kafka existant."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur |
|-------|------|------|--------|
| Nom du topic | Input | Disabled | my_topic |
| Réplication | Input | Éditable | 3 |
| Partitions | Input | Éditable | 1 |

### Notes
- Le nom du topic n'est pas modifiable
- Les partitions ne peuvent qu'augmenter (pas diminuer)
- Fond grisé pour le champ disabled

### Configuration avancée (accordéon)
- Paramètres Kafka modifiables

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Modifier le topic | Primary | Droite | PUT topic |

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/database/kafka/{serviceId}/topic/{topicId}
  - Body: { replication?, partitions?, config? }
  - Modifier un topic
