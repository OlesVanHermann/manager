# Cahier des Charges : public-cloud/databases - Onglet Namespaces (M3DB)

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Namespaces (M3DB)
- **Type** : Page onglet listing
- **Objectif** : Gérer les namespaces M3DB

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec service M3DB actif
- Breadcrumb avec "Namespaces"
- Onglets (Namespaces actif)
- Titre "Namespaces"
- Description
- Bouton ajouter namespace
- Tableau des namespaces
- ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Namespaces actif
- **OdsText** : Description
- **OdsButton** : "+ Ajouter un namespace" (outline)
- **OdsTable** : Liste des namespaces
- **OdsBadge** : Type (Agrégé, Non agrégé)
- **OdsActionMenu** : Actions

## 4. TITRE
"Namespaces"

## 5. DESCRIPTION
"Collections de points. Vous pouvez avoir maximum 5 namespaces sur votre service. Seul le namespace par default est non agrégé. Il inclut tous les nuages de points au fur et à mesure de leur arrivée."

## 6. TABLEAU NAMESPACES
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Texte | Nom du namespace |
| Type | Badge | Agrégé / Non agrégé |
| Durée de rétention | Texte | 48H, 30D, 365D |
| Résolution | Texte | - / 1H / 1D |
| Actions | Menu ⋮ | ActionMenu |

### Badges Type
| Type | Couleur |
|------|---------|
| Non agrégé | Bleu (#DBEAFE, #1E40AF) |
| Agrégé | Bleu (#DBEAFE, #1E40AF) |

### Données exemple
| Nom | Type | Rétention | Résolution |
|-----|------|-----------|------------|
| default | Non agrégé | 48H | - |
| metrics_1h | Agrégé | 30D | 1H |
| metrics_1d | Agrégé | 365D | 1D |

## 7. ACTION MENU (ouvert)
| Action | Type |
|--------|------|
| Editer | Standard |
| Supprimer | Danger (rouge) |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/m3db/{serviceId}/namespace
  - Liste des namespaces
- POST /cloud/project/{projectId}/database/m3db/{serviceId}/namespace
  - Body: { name, type, retention, resolution }
  - Créer un namespace
- PUT /cloud/project/{projectId}/database/m3db/{serviceId}/namespace/{namespaceId}
  - Modifier un namespace
- DELETE /cloud/project/{projectId}/database/m3db/{serviceId}/namespace/{namespaceId}
  - Supprimer un namespace
