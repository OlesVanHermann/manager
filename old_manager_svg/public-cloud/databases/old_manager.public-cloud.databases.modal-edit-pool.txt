# Cahier des Charges : public-cloud/databases - Modal Modifier pool de connexion

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier un pool de connexion
- **Type** : Modal formulaire
- **Objectif** : Modifier les propriétés d'un pool de connexion existant

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x420px
- Header avec titre
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom (disabled), Taille
- **OdsSelect** : Base de données, Mode
- **OdsButton** : Annuler, Modifier ce pool de connexion

## 4. TITRE
"Modifier un pool de connexion"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur |
|-------|------|------|--------|
| Nom du pool de connexion | Input | Disabled | my_pool |
| Base de données | Select | Éditable | defaultdb |
| Mode | Select | Éditable | session |
| Taille | Input | Éditable | 10 |

### Notes
- Le nom du pool n'est pas modifiable après création
- Fond grisé pour le champ disabled

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Modifier ce pool de connexion | Primary | Droite | PUT pool |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/database/postgresql/{serviceId}/connectionPool/{poolId}
  - Body: { database, mode, size }
  - Modifier un pool de connexion
