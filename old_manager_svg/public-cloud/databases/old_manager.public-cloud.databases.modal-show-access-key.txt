# Cahier des Charges : public-cloud/databases - Modal Clé d'accès

## 1. VUE D'ENSEMBLE
- **Modal** : Clé d'accès
- **Type** : Modal information (lecture seule)
- **Objectif** : Afficher et permettre le téléchargement de la clé RSA

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x260px
- Header avec titre
- Bloc de code avec clé RSA
- Footer avec bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsCode** : Bloc de code sombre
- **OdsButton** : Télécharger (dans le bloc), Fermer

## 4. TITRE
"Clé d'accès"

## 5. BLOC DE CODE
### Structure
- Header gris foncé (#374151) avec label "Clé d'accès"
- Bouton "Télécharger" dans le header
- Corps noir (#1F2937) avec la clé

### Contenu
```
-----BEGIN RSA PRIVATE KEY-----
MIIEpAIBAAKCAQEA2Z3qX2BTLS4e
...
-----END RSA PRIVATE KEY-----
```

## 6. ACTIONS
| Action | Description |
|--------|-------------|
| Télécharger | Télécharger la clé en fichier .pem |
| Fermer | Fermer la modal |

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Télécharger | Small gris | Header du bloc | Download fichier |
| Fermer | Outline | Footer droite | Fermer modal |

## 8. SÉCURITÉ
- La clé privée n'est visible qu'une fois
- Recommandation de stocker dans un endroit sécurisé

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/service/{serviceId}/user/{userId}/key
  - Récupérer la clé d'accès
