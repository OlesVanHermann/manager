# Cahier des Charges : public-cloud/load-balancer - Modal Créer Pool

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un pool
- **Type** : Modal formulaire
- **Objectif** : Créer un pool de backends pour le Load Balancer

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 700x560px avec coins arrondis
- Titre + bouton fermer
- Banner info
- Formulaire avec champs
- Section Health Monitor optionnelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (bleu clair)
- **OdsInput** : Nom, URL Path, Code attendu
- **OdsSelect** : Protocole, Algorithme, Persistence, Type HM
- **OdsText** : Labels
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un pool"

## 5. BANNER INFO
- Fond bleu clair (#DBEAFE)
- Bordure bleue (#3B82F6)
- "ℹ️ Un pool regroupe un ensemble d'instances backend sur lesquelles le trafic sera distribué."

## 6. CHAMPS PRINCIPAUX

| Champ | Type | Valeur | Position |
|-------|------|--------|----------|
| Nom du pool | Input | pool-web-servers | Full width (640px) |
| Protocole | Select | HTTP | Gauche (300px) |
| Algorithme de répartition | Select | ROUND_ROBIN | Droite (300px) |
| Persistence de session | Select (optionnel) | Sélectionner... | Full width |

## 7. ALGORITHMES DE RÉPARTITION
- ROUND_ROBIN
- LEAST_CONNECTIONS
- SOURCE_IP

## 8. SECTION HEALTH MONITOR (OPTIONNEL)

| Champ | Type | Valeur | Position |
|-------|------|--------|----------|
| Type | Select | HTTP | Gauche (200px) |
| URL Path | Input | /health | Centre (200px) |
| Code attendu | Input | 200 | Droite (200px) |

## 9. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Créer | Primary bleu | Droite | POST pool |

## 10. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool
  - Body: { name, protocol, loadbalancerId, algorithm, sessionPersistence?, healthMonitor? }
  - Créer le pool
