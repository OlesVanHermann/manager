# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer L7 Policy

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer une L7 policy
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer une politique L7

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x180px
- Header avec icône warning + titre
- Question de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsIcon** : Warning (⚠) orange
- **OdsText** : Question
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
- Icône ⚠ orange dans cercle jaune
- Titre : "Supprimer une L7 policy"

## 5. QUESTION
"Supprimer la policy "policy-redirect-https" ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE L7 policy |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy/{l7PolicyId}
  - Supprimer la L7 policy
