# Cahier des Charges : public-cloud/load-balancer - Onglet L7 Policies

## 1. VUE D'ENSEMBLE
- **Page** : Liste des L7 Policies d'un listener
- **Type** : Page datagrid avec actions
- **Objectif** : Gérer les politiques L7 d'un listener

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Titre "L7 Policies"
- Bouton ajouter
- Datagrid

## 3. COMPOSANTS UI
- **OdsText** : Titre
- **OdsButton** : Ajouter une L7 policy
- **OdsDatagrid** : Liste des policies
- **OdsBadge** : Statut (ACTIVE)
- **OdsLink** : Nom (vers détail)

## 4. TITRE
"L7 Policies"

## 5. BOUTON AJOUTER
- **Label** : "+ Ajouter une L7 policy"
- **Style** : Primary bleu

## 6. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Position | Number | Ordre d'exécution |
| Nom | Link bleu | Lien vers détail |
| Action | Text | REDIRECT_TO_URL, REDIRECT_TO_POOL... |
| Statut | Badge success | ACTIVE |

## 7. DONNÉES EXEMPLE

| Position | Nom | Action | Statut |
|----------|-----|--------|--------|
| 1 | policy-redirect-https | REDIRECT_TO_URL | ACTIVE |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy?listenerId={listenerId}
  - Liste des L7 policies
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy
  - Créer une L7 policy
