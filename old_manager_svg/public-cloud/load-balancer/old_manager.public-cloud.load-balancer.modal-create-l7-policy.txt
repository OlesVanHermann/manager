# Cahier des Charges : public-cloud/load-balancer - Modal Créer L7 Policy

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter une L7 policy
- **Type** : Modal formulaire
- **Objectif** : Créer une politique L7 pour un listener

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x480px
- Titre
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, labels
- **OdsInput** : Nom, URL redirection
- **OdsSelect** : Position, Action, Code redirection
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter une L7 policy"

## 5. CHAMPS

| Champ | Type | Placeholder |
|-------|------|-------------|
| Nom | Input | - |
| Position | Select | - |
| Action | Select | - |
| URL de redirection | Input | - |
| Code de redirection | Select | - |

## 6. ACTIONS DISPONIBLES
- REDIRECT_TO_POOL
- REDIRECT_TO_URL
- REDIRECT_PREFIX
- REJECT

## 7. CODES DE REDIRECTION
- 301 (Moved Permanently)
- 302 (Found)
- 303 (See Other)
- 307 (Temporary Redirect)
- 308 (Permanent Redirect)

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST L7 policy |

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy
  - Body: { name, listenerId, action, position, redirectUrl?, redirectHttpCode?, redirectPoolId? }
  - Créer la L7 policy
