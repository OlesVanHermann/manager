# Cahier des Charges : public-cloud/load-balancer - Modal Éditer Health Monitor

## 1. VUE D'ENSEMBLE
- **Modal** : Editer le health monitor
- **Type** : Modal formulaire
- **Objectif** : Modifier les paramètres d'un health monitor

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x540px
- Titre
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom, URL Path, Code, Périodicité
- **OdsSelect** : Type (disabled)
- **OdsText** : Labels
- **OdsButton** : Annuler, Editer

## 4. TITRE
"Editer le health monitor"

## 5. CHAMPS

| Champ | Type | Valeur | Modifiable |
|-------|------|--------|------------|
| Nom | Input | health-monitor-http | Oui |
| Type | Input (disabled, fond gris) | HTTP | Non |
| URL Path | Input | /health | Oui |
| Code attendu | Input | 200 | Oui |
| Périodicité | Input | 5 | Oui |

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Editer | Primary bleu | Droite | PUT health monitor |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor/{healthMonitorId}
  - Body: { name, urlPath?, expectedCodes?, delay? }
  - Modifier le health monitor
