# Cahier des Charges : public-cloud/load-balancer - Modal Renommer Load Balancer

## 1. VUE D'ENSEMBLE
- **Modal** : Renommer le Load Balancer
- **Type** : Modal formulaire simple
- **Objectif** : Modifier le nom d'un Load Balancer

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x220px
- Titre
- Champ nom
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsInput** : Nom
- **OdsButton** : Annuler, Renommer

## 4. TITRE
"Renommer le Load Balancer"

## 5. CHAMP
| Champ | Type | Valeur |
|-------|------|--------|
| Nom | Input | lb-prod-01 |

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Renommer | Primary bleu | Droite | PUT LB |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/loadbalancer/{loadBalancerId}
  - Body: { name }
  - Renommer le Load Balancer
