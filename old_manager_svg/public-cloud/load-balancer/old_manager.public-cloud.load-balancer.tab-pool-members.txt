# Cahier des Charges : public-cloud/load-balancer - Onglet Membres Pool

## 1. VUE D'ENSEMBLE
- **Page** : Liste des membres d'un pool
- **Type** : Datagrid avec actions
- **Objectif** : Gérer les membres backend d'un pool

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Lien retour vers liste des pools
- Titre (nom du pool)
- Barre d'onglets (Membres actif)
- Boutons d'ajout
- Datagrid

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre
- **OdsTabs** : Informations générales, Health monitor, Membres
- **OdsButton** : Ajouter manuellement, Ajouter des IPs (outline)
- **OdsDatagrid** : Liste des membres
- **OdsBadge** : Statut (ACTIVE)

## 4. LIEN RETOUR
"‹ liste des pools"

## 5. TITRE
"pool-backend-api"

## 6. ONGLETS
| Onglet | Actif |
|--------|-------|
| Informations générales | - |
| Health monitor | - |
| Membres | ✓ (souligné bleu) |

## 7. BOUTONS AJOUTER
| Bouton | Style | Action |
|--------|-------|--------|
| + Ajouter manuellement | Outline bleu | Open add member modal |
| + Ajouter des IPs | Outline bleu | Open add IPs modal |

## 8. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Text | Nom du membre |
| Adresse IP | Text | IP privée |
| Port | Number | Port backend |
| Statut | Badge success | ACTIVE |

## 9. DONNÉES EXEMPLE

| Nom | Adresse IP | Port | Statut |
|-----|------------|------|--------|
| member-web-01 | 192.168.1.10 | 8080 | ACTIVE |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member
  - Liste des membres
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member
  - Ajouter un membre
