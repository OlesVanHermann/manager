# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer Listener

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un listener
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer un listener du Load Balancer

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x180px
- Header avec icône warning + titre
- Question de confirmation (2 lignes)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsIcon** : Warning (⚠) orange
- **OdsText** : Question, nom listener (bold)
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
- Icône ⚠ orange dans cercle jaune
- Titre : "Supprimer un listener"

## 5. QUESTION
- Ligne 1 : "Voulez-vous supprimer le listener"
- Ligne 2 (bold) : "listener-http-01"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE listener |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/listener/{listenerId}
  - Supprimer le listener
