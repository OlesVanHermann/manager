# Cahier des Charges : public-cloud/load-balancer - Modal Ajouter Membre

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un membre
- **Type** : Modal formulaire
- **Objectif** : Ajouter un membre (serveur backend) à un pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x440px avec barre bleue gauche
- Icône + titre + bouton fermer
- Banner info
- Formulaire (nom, IP, port)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec accent bleu
- **OdsIcon** : Plus (+)
- **OdsMessage** : Info (bleu clair)
- **OdsInput** : Nom, Adresse IP, Port
- **OdsText** : Labels, helper text
- **OdsButton** : Annuler, Ajouter

## 4. HEADER
- Icône + dans cercle bleu clair
- Titre : "Ajouter un membre"
- Bouton fermer (×)

## 5. BANNER INFO
- Fond bleu clair (#DBEAFE)
- Icône ℹ
- "Un membre représente un serveur backend du pool."

## 6. CHAMPS

| Champ | Type | Placeholder | Helper | Required |
|-------|------|-------------|--------|----------|
| Nom | Input | web-server-01 | - | Non (optionnel) |
| Adresse IP | Input | 10.0.1.15 | Adresse IP privée du serveur backend | * |
| Port | Input | 8080 | - | * |

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST membre |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member
  - Body: { name?, address, protocolPort }
  - Ajouter un membre au pool
