# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer L7 Rule

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer une L7 rule
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer une règle L7

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x160px
- Header avec icône warning + titre
- Question de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsIcon** : Warning (⚠) orange
- **OdsText** : Question
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
- Icône ⚠ orange dans cercle jaune
- Titre : "Supprimer une L7 rule"

## 5. QUESTION
"Supprimer cette L7 rule ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE L7 rule |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy/{l7PolicyId}/l7Rule/{l7RuleId}
  - Supprimer la L7 rule
