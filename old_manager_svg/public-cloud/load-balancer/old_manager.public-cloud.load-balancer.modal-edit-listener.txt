# Cahier des Charges : public-cloud/load-balancer - Modal Éditer Listener

## 1. VUE D'ENSEMBLE
- **Modal** : Editer le listener
- **Type** : Modal formulaire
- **Objectif** : Modifier un listener existant

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x360px
- Titre
- Formulaire avec champs (certains disabled)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsInput** : Nom, Pool
- **OdsInput (disabled)** : Protocole, Port (fond gris)
- **OdsSelect** : Pool par défaut
- **OdsButton** : Annuler, Editer

## 4. TITRE
"Editer le listener"

## 5. CHAMPS

| Champ | Type | Valeur | Modifiable |
|-------|------|--------|------------|
| Nom | Input | listener-http-01 | Oui |
| Protocole | Input (disabled) | HTTP | Non |
| Port | Input (disabled) | 80 | Non |
| Pool par défaut | Select | - | Oui |

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Editer | Primary bleu | Droite | PUT listener |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/listener/{listenerId}
  - Body: { name, defaultPoolId? }
  - Modifier le listener
