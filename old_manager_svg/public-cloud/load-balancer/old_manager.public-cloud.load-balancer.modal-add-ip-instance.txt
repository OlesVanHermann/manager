# Cahier des Charges : public-cloud/load-balancer - Modal Ajouter IPs Instances

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter des IPs d'instances
- **Type** : Modal sélection multiple avec tableau
- **Objectif** : Ajouter des membres depuis les instances existantes

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x460px
- Titre
- Champ de recherche
- Tableau avec checkboxes
- Compteur de sélection
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Recherche
- **OdsTable** : Liste des instances
- **OdsCheckbox** : Sélection multiple
- **OdsText** : Compteur, labels
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter des IPs d'instances"

## 5. CHAMP RECHERCHE
- **Placeholder** : "Rechercher une instance"
- **Largeur** : 300px

## 6. TABLEAU INSTANCES

### Colonnes
| Colonne | Largeur |
|---------|---------|
| Checkbox | 50px |
| Instance | 190px |
| Adresse IP | 160px |
| Port | 100px |

### Données exemple
| ☑ | Instance | Adresse IP | Port |
|---|----------|------------|------|
| ✓ (fond bleu) | instance-web-01 | 192.168.1.100 | 8080 |
| ☐ | instance-api-01 | 192.168.1.101 | 3000 |

## 7. COMPTEUR SÉLECTION
"1 instance selectionnée"

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST membres |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance
  - Liste des instances
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member
  - Body: [{ name, address, protocolPort }]
  - Ajouter les membres
