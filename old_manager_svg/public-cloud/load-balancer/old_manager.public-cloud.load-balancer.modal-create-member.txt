# Cahier des Charges : public-cloud/load-balancer - Modal Créer Membre Manuellement

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter Manuellement
- **Type** : Modal formulaire simple
- **Objectif** : Ajouter un membre manuellement à un pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x340px
- Titre + description
- Formulaire (nom, IP, port)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsText** : Titre, description, labels
- **OdsInput** : Nom, Adresse IP, Port
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter Manuellement"

## 5. DESCRIPTION
"Ajouter un membre manuellement"

## 6. CHAMPS

| Champ | Type | Width | Required |
|-------|------|-------|----------|
| Nom (optionnel) | Input | 390px | Non |
| Adresse IP | Input | 390px | * |
| Port | Input | 390px | * |

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST membre |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member
  - Body: { name?, address, protocolPort }
  - Ajouter un membre au pool
