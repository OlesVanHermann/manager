# Cahier des Charges : public-cloud/load-balancer - Modal Créer Health Monitor

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un health monitor
- **Type** : Modal formulaire
- **Objectif** : Créer un health monitor pour vérifier l'état des membres

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x540px
- Titre + description
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, description, labels
- **OdsInput** : Tous les champs
- **OdsSelect** : Type de health monitor
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un health monitor"

## 5. DESCRIPTION
"Le health monitor détermine l'intégrité des membres"

## 6. CHAMPS

| Champ | Type | Valeur | Position |
|-------|------|--------|----------|
| Nom | Input | - | Full width |
| Type | Select | HTTP | Full width |
| URL Path | Input | /health | Full width |
| Code attendu | Input | 200 | Gauche (260px) |
| Périodicité | Input | 5 | Droite (260px) |
| Timeout | Input | 5 | Gauche (260px) |
| Max tentatives | Input | 3 | Droite (260px) |

## 7. TYPES DISPONIBLES
- HTTP
- HTTPS
- TCP
- PING

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST health monitor |

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor
  - Body: { name, poolId, type, urlPath?, expectedCodes?, delay, timeout, maxRetries }
  - Créer le health monitor
