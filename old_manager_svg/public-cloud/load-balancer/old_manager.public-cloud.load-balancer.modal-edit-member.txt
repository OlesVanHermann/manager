# Cahier des Charges : public-cloud/load-balancer - Modal Modifier Membre

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le membre
- **Type** : Modal formulaire simple
- **Objectif** : Modifier le nom d'un membre de pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x220px
- Titre
- Champ nom
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsInput** : Nom
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le membre"

## 5. CHAMP
| Champ | Type | Valeur |
|-------|------|--------|
| Nom | Input | member-web-01 |

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Modifier | Primary bleu | Droite | PUT membre |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member/{memberId}
  - Body: { name }
  - Modifier le membre
