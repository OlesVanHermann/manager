# Cahier des Charges : public-cloud/public-ip - Modal Modifier Instance

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier l'instance associée
- **Type** : Modal formulaire
- **Objectif** : Associer une IP publique à une instance

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x400px)
- Titre avec close
- Description avec IP
- Sélection instance
- Sélection IP privée
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsSelect** : Instance, IP privée
- **OdsButton** : Annuler (secondary), Confirmer (primary)

## 4. HEADER
"Modifier l'instance associée" + bouton ×

## 5. DESCRIPTION
"Associez l'adresse IP **57.128.42.124** à une instance."

## 6. CHAMPS

### Instance
- **Select** : Dropdown liste des instances
- **Valeur** : "web-server-staging"

### Adresse IP privée
- **Select** : Dropdown liste des IPs privées de l'instance
- **Valeur** : "192.168.0.15"

## 7. BOUTONS
- Annuler (secondary)
- Confirmer (primary)

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance
  - Liste des instances disponibles
- GET /cloud/project/{projectId}/instance/{instanceId}/interface
  - IPs privées de l'instance
- PUT /cloud/project/{projectId}/ip/floating/{ipId}
  - Body: { instanceId, ip }
  - Associe l'IP à l'instance
