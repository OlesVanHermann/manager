# Cahier des Charges : public-cloud/public-ip - Modal Résilier

## 1. VUE D'ENSEMBLE
- **Modal** : Résilier l'adresse IP
- **Type** : Modal confirmation danger avec saisie
- **Objectif** : Confirmer la résiliation définitive d'une IP

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x380px)
- Barre rouge en haut
- Titre avec close
- Message d'avertissement
- Warning message
- Champ de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsInput** : Champ de confirmation TERMINATE
- **OdsButton** : Annuler (secondary), Résilier (danger)

## 4. HEADER
Barre rouge (6px) + "Résilier l'adresse IP" + bouton ×

## 5. MESSAGE
"Vous êtes sur le point de résilier l'adresse IP **57.128.42.124**."

## 6. WARNING
Fond jaune : "⚠️ Cette action est irréversible. L'adresse IP sera définitivement supprimée et ne pourra plus être récupérée."

## 7. CONFIRMATION
- **Label** : "Entrez « TERMINATE » pour confirmer"
- **Input** : Champ texte

## 8. BOUTONS
- Annuler (secondary)
- Résilier (danger/rouge)

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ip/floating/{ipId}
  - Résilie la Floating IP
