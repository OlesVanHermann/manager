# Cahier des Charges : public-cloud/ai-jobs - Modal Arrêter le job

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation d'arrêt d'un job
- **Type** : Modal confirmation warning
- **Objectif** : Demander confirmation avant d'arrêter un job en cours

## 2. STRUCTURE MODAL
- Overlay sombre sur la page
- Modal centrée (500x280)
- Header avec titre et bouton fermer
- Icône warning
- Message de confirmation
- Boutons Annuler / Arrêter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Message et nom du job
- **OdsButton** : Annuler (outline), Arrêter (danger)

## 4. TITRE
"Arrêter le job"

## 5. CONTENU
### Icône
- Cercle jaune avec "!" (warning)

### Message
"Êtes-vous sûr de vouloir arrêter le job"
"**{job-name}**" (en gras)
"?"

### Avertissement
"Cette action va interrompre l'exécution du job."
"Les données non synchronisées seront perdues."

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline gris | Ferme la modal |
| Arrêter | Danger rouge (#DC2626) | Arrête le job |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/job/{jobId}/kill
  - Ou: DELETE /cloud/project/{projectId}/ai/job/{jobId}/stop
