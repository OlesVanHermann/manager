# Cahier des Charges : public-cloud/notebooks - Onglet Dépôt Git Public

## 1. VUE D'ENSEMBLE
- **Page** : Dépôts Git publics attachés
- **Type** : Onglet avec datagrid
- **Objectif** : Gérer les dépôts Git publics synchronisés

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Notebooks actif)
- Breadcrumb
- Onglets navigation
- Titre + description
- Bouton ajouter
- Datagrid dépôts

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 5 onglets
- **OdsButton** : Ajouter Git public
- **OdsDatagrid** : Liste dépôts
- **OdsBadge** : Permission
- **OdsActionMenu** : Menu contextuel

## 12. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/notebook/{notebookId}/git
  - Liste des dépôts Git attachés
- POST /cloud/project/{projectId}/ai/notebook/{notebookId}/git
  - Attacher un dépôt Git
  - Body: { url, mountPath }
- DELETE /cloud/project/{projectId}/ai/notebook/{notebookId}/git/{gitId}
  - Détacher un dépôt
