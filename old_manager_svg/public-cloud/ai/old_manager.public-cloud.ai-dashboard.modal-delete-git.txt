# Cahier des Charges : public-cloud/ai-dashboard - Modal Supprimer dépot Git

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un dépôt Git privé
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un dépôt Git privé

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Icône warning
- Message de confirmation avec nom
- Description impact
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x260)
- **OdsIcon** : ! (warning, cercle rouge)
- **OdsText** : Message + description
- **OdsButton** : Annuler, Confirmer (danger)

## 4. TITRE
"Supprimer un dépot Git privé"

## 5. ICÔNE
- Cercle fond rose (#FEE2E2)
- Point d'exclamation ! rouge (#DC2626)

## 6. MESSAGE
"Êtes-vous sûr de vouloir supprimer le dépot Git"
"{git-repo-name}" (bold) ?

## 7. DESCRIPTION
"Cette action est irréversible. Vos notebooks, jobs et apps"
"ne pourront plus accéder à ce dépot Git privé."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Confirmer | Danger rouge | Supprime le dépôt |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/git/{gitId}
