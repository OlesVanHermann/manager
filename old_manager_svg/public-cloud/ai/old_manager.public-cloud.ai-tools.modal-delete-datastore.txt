# Cahier des Charges : public-cloud/ai-tools - Modal Supprimer Datastore

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation suppression datastore
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'un datastore S3

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal centrée (500x220px)
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, message
- **OdsButton** : Annuler (outline), Confirmer (danger rouge)

## 4. HEADER
- **Titre** : "Supprimer un datastore S3"
- **Bouton fermer** : ✕

## 5. CONTENU
"Êtes-vous sûr de vouloir supprimer le datastore **my-training-data** ?"

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer modal |
| Confirmer | Danger rouge | DELETE datastore |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/data/region/{region}/alias/{alias}
  - Supprimer le datastore
