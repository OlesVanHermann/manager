## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/endpoints/metrics
  - period: "30d" | "7d" | "24h"
- Retourne:
  - totalRequests: number
  - totalTokens: number
  - estimatedCost: number
  - avgLatency: number
  - p95Latency: number
  - tokensTimeSeries: [{date, value}]
