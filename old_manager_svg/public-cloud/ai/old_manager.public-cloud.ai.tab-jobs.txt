# Cahier des Charges : public-cloud/ai - Onglet Jobs

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard AI Platform - Onglet Jobs
- **Type** : Page tableau de bord avec tableau
- **Objectif** : Afficher les jobs d'entraînement AI

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar
- Breadcrumb
- Titre "AI Platform"
- Onglets (Notebooks, Jobs actif, Apps)
- Tableau des jobs

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 3 onglets
- **OdsButton** : Nouveau job (primaire)
- **OdsTable** : Liste des jobs
- **OdsBadge** : Statuts (RUNNING, SUCCEEDED, FAILED, QUEUED)

## 4. TITRE
"AI Platform"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Notebooks | Inactif |
| Jobs | Actif (souligné bleu) |
| Apps | Inactif |

## 6. SECTION JOBS
- Titre: "AI Training Jobs" avec badge compteur (4)
- Bouton: "Nouveau job" (primaire bleu)

## 7. TABLEAU JOBS
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Texte | Nom du job |
| Image | Texte mono | Image Docker |
| Ressources | Texte | Type de ressources |
| Statut | Badge | État du job |
| Durée | Texte | Temps d'exécution |
| Actions | Bouton | Logs / Annuler |

### Badges statuts
| Statut | Couleur | Texte |
|--------|---------|-------|
| RUNNING | Vert | #03543F |
| SUCCEEDED | Bleu | #0050D7 |
| FAILED | Rouge | #991B1B |
| QUEUED | Jaune | #92400E |

### Données exemple
| Nom | Image | Ressources | Statut | Durée | Action |

| train-bert-v3 | pytorch/pytorch:2.0 | 4x GPU A100 | RUNNING | 4h 32m | Logs |

| data-preprocess | python:3.11 | CPU only | QUEUED | - | Annuler |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/job
- POST /cloud/project/{projectId}/ai/job/{jobId}/cancel
- GET /cloud/project/{projectId}/ai/job/{jobId}/logs
