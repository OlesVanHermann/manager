# Cahier des Charges : public-cloud/ai-dashboard - Modal Supprimer Docker Registry

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un registre Docker privé
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un registre Docker

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Icône warning
- Message de confirmation avec nom
- Description impact
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x260)
- **OdsIcon** : ! (warning, cercle rouge)
- **OdsText** : Message + description
- **OdsButton** : Annuler, Confirmer (danger)

## 4. TITRE
"Supprimer un registre docker privé"

## 5. ICÔNE
- Cercle fond rose (#FEE2E2)
- Point d'exclamation ! rouge (#DC2626)

## 6. MESSAGE
"Êtes-vous sûr de vouloir supprimer le docker"
"{registry-name}" (bold) ?

## 7. DESCRIPTION
"Cette action est irréversible. Les images stockées dans ce"
"registre ne seront plus accessibles par vos jobs et apps."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Confirmer | Danger rouge | Supprime le registry |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/registry/{registryId}
