## 6. APIS & ENDPOINTS

### 6.1 Créer une clé API

```
POST /cloud/project/{projectId}/ai/token
```

Body:
```json
{
  "name": "string",
  "description": "string",
  "expirationDate": "2024-12-31T23:59:59Z"
}
```

Response:
```json
{
  "id": "token-id",
  "name": "ma-cle-api",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "createdAt": "2024-01-15T10:30:00Z"
}
```
