# Cahier des Charges : public-cloud/ai-training - Liste des Jobs

## 1. VUE D'ENSEMBLE
- **Page** : AI Training - Liste des jobs d'entraînement
- **Type** : Dashboard avec datagrid
- **Objectif** : Gérer les jobs d'entraînement ML

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Training actif)
- Breadcrumb
- Titre + bouton créer
- Barre de recherche
- Datagrid jobs

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsButton** : Créer job
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste jobs
- **OdsBadge** : Statuts
- **OdsProgressBar** : Progression
- **OdsActionMenu** : Menu contextuel

## 4. BREADCRUMB
"Public Cloud / Mon Projet Cloud / AI Training"

## 5. TITRE ET BOUTON
- **Titre** : "AI Training"
- **Bouton** : "+ Créer un job" (primary bleu)

## 6. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Job | Text + ID | Nom du job + identifiant tronqué |
| Image | Text mono | Image Docker avec tag |
| Ressources | Text | GPU type + VRAM |
| Progression | ProgressBar | Barre + pourcentage |
| Durée | Text | Temps d'exécution |
| Coût | Text | Coût estimé en € |
| Statut | Badge | RUNNING/QUEUED/COMPLETED |
| Actions | ActionMenu | ⋮ |

## 8. BADGES STATUT

| Statut | Couleur | Code |
|--------|---------|------|
| RUNNING | Vert (#DEF7EC) | success |
| QUEUED | Orange (#FEF3C7) | warning |
| COMPLETED | Bleu (#DBEAFE) | info |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/job
  - Liste des jobs
- POST /cloud/project/{projectId}/ai/job
  - Créer un job
- GET /cloud/project/{projectId}/ai/job/{jobId}
  - Détail d'un job
- DELETE /cloud/project/{projectId}/ai/job/{jobId}
  - Supprimer un job
