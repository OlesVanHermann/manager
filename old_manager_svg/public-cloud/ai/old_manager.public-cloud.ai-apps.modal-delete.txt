# Cahier des Charges : public-cloud/ai-apps - Modal Supprimer une app

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'une application AI Deploy
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer définitivement une application

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Message de confirmation avec nom app
- Bandeau warning rouge
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x220)
- **OdsText** : Message confirmation
- **OdsMessage** : Warning rouge
- **OdsButton** : Annuler, Supprimer (danger)

## 4. TITRE
"Supprimer une app"

## 5. MESSAGE
"Êtes-vous sûr de vouloir supprimer l'application"
"{app-name}" (bold) ?

## 6. BANDEAU WARNING
Fond rouge (#FEF2F2) :
"⚠ Cette action est irréversible. Toutes les données seront perdues."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Supprimer | Danger rouge | Supprime l'app |

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/app/{appId}
