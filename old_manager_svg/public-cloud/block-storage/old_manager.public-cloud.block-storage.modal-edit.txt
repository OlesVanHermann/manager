# Cahier des Charges : public-cloud/block-storage - Éditer un Volume

## 1. VUE D'ENSEMBLE
- **Page** : Éditer un volume Block Storage
- **Type** : Page formulaire
- **Objectif** : Modifier les propriétés d'un volume existant

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Block Storage actif)
- Breadcrumb + lien retour
- Titre page
- Formulaire édition
- Boutons actions

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsLink** : Retour
- **OdsChip** : Localisation, Type (read-only)
- **OdsInput** : Nom
- **OdsQuantity** : Capacité
- **OdsToggle** : Amorçable
- **OdsButton** : Annuler, Modifier

## 4. CHAMPS FORMULAIRE
- Localisation : GRA11 (chip, read-only)
- Type de volume : classic (chip, read-only)
- Nom du volume : Input editable
- Capacité : Quantity input (-, valeur, +) en Go
- Amorçable : Toggle switch (Oui/Non)

## 5. BOUTONS
- Annuler (outline)
- Modifier le volume (primary)

## 6. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/volume/{volumeId}
- PUT /cloud/project/{projectId}/volume/{volumeId}
- PUT /cloud/project/{projectId}/volume/{volumeId}/upsize
