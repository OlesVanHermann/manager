## 6. APIS & ENDPOINTS

### 6.1 Obtenir les détails du snapshot

```
GET /cloud/project/{projectId}/volume/snapshot/{snapshotId}
```

### 6.2 Créer un volume depuis snapshot

```
POST /cloud/project/{projectId}/volume
{
  "name": "vol-name",
  "snapshotId": "snapshot-uuid",
  "size": 100,
  "region": "GRA11",
  "type": "high-speed"
}
```

### 6.3 Obtenir le prix

```
GET /order/catalog/public/cloud?ovhSubsidiary=FR
```
