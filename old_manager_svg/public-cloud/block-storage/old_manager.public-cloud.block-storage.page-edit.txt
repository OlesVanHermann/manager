## 6. APIS & ENDPOINTS

### 6.1 Obtenir les détails du volume

```
GET /cloud/project/{projectId}/volume/{volumeId}
```

### 6.2 Modifier le volume

```
PUT /cloud/project/{projectId}/volume/{volumeId}
{
  "name": "new-name",
  "size": 150,
  "bootable": true
}
```

### 6.3 Redimensionner le volume (upsize)

```
POST /cloud/project/{projectId}/volume/{volumeId}/upsize
{
  "size": 150
}
```

### 6.4 Obtenir le prix

```
GET /order/catalog/public/cloud?ovhSubsidiary=FR
```
