# Cahier des Charges : public-cloud/block-storage - Modal Attacher

## 1. VUE D'ENSEMBLE
- **Modal** : Attacher le volume à une instance
- **Type** : Modal formulaire avec select
- **Objectif** : Attacher un volume à une instance

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal blanche centrée (600x380px)
- Header avec titre + fermeture
- Info banner bleu
- Infos volume
- Select instance
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info banner
- **OdsSelect** : Sélection instance
- **OdsButton** : Annuler, Attacher

## 4. INFO BANNER
"Le volume sera attaché à l'instance sélectionnée.
Assurez-vous que l'instance est dans la même région (GRA9)."

## 5. CHAMPS
- Volume : volume-data-prod (500 Go, High-speed)
- Instance à attacher : Select dropdown
- Note : Instances disponibles dans GRA9 : 3

## 6. BOUTONS
- Annuler (outline)
- Attacher (primary)

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance?region={region}
- POST /cloud/project/{projectId}/volume/{volumeId}/attach
