# Cahier des Charges : public-cloud/volume-backup - Modal Restaurer

## 1. VUE D'ENSEMBLE
- **Modal** : Restaurer un volume
- **Type** : Modal confirmation warning
- **Objectif** : Confirmer la restauration d'un volume depuis une sauvegarde

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x320px)
- Barre orange en haut
- Titre avec close
- Message de confirmation
- Warning message
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler (secondary), Restaurer (primary)

## 4. HEADER
Barre orange (6px) + "Restaurer un volume" + bouton ×

## 5. MESSAGE
"Vous êtes sur le point de restaurer le volume **volume-prod-01** à partir de la sauvegarde **backup-prod-2024**."

## 6. WARNING
Fond jaune : "⚠️ Les données actuelles du volume seront remplacées par celles de la sauvegarde."

## 7. BOUTONS
- Annuler (secondary)
- Restaurer (primary)

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/volume/backup/{backupId}/restore
  - Body: { volumeId }
  - Restaure le volume depuis la sauvegarde
