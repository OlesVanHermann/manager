## 7. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}/containerRegistry
→ Liste des registries

GET /cloud/project/{projectId}/containerRegistry/{registryId}
→ Détails registry (images, stockage)

GET /cloud/project/{projectId}/containerRegistry/{registryId}/image
→ Liste des images

GET /cloud/project/{projectId}/containerRegistry/{registryId}/user
→ Utilisateurs du registry

DELETE /cloud/project/{projectId}/containerRegistry/{registryId}
→ Supprime le registry
```
