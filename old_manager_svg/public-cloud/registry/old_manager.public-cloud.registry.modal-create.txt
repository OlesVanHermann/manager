# Cahier des Charges : public/registry - Modal Create

## 1. VUE D'ENSEMBLE

**Type:** Modal wizard 2 étapes
**Fonction:** Créer un Private Registry Docker
**Contexte:** Configuration d'un container registry

## 2. DIMENSIONS

- Modal: 700x660px
- Background page: #F9FAFB avec overlay rgba(0,0,0,0.4)

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #F9FAFB
- Titre: "Créer un Private Registry"
- Bouton fermer: ×
- Height: 60px

## 4. STEPPER

### Étapes
| Étape | Numéro | État | Label |
|-------|--------|------|-------|
| Configuration | 1 | Actif (cercle bleu) | Configuration |
| Résumé | 2 | Inactif (cercle gris) | Résumé |

### Style
- Cercle actif: #0050D7, text blanc
- Cercle inactif: border #E5E7EB, text gris
- Ligne de connexion: #E5E7EB

## 5. FORMULAIRE (ÉTAPE 1)

### Champ Nom du registry
- Label: "Nom du registry *"
- Type: Input text
- Placeholder: "ex: production-registry"
- Largeur: 650px

### Champ Région
- Label: "Région *"
- Type: Select dropdown
- Valeur exemple: "Gravelines (GRA)"
- Largeur: 650px

### Champ Plan
- Label: "Plan *"
- Type: Radio cards (3 options)

#### Option S - Small (sélectionnée)
- Border: #0050D7, 2px
- Titre: "S - Small" (font-weight 600)
- Features:
  - 200 Go de stockage
  - Illimité pull/push
- Prix: "4.99 EUR/mois"

#### Option M - Medium
- Border: #E5E7EB
- Titre: "M - Medium"
- Features:
  - 600 Go de stockage
  - Illimité pull/push
- Prix: "14.99 EUR/mois"

#### Option L - Large
- Border: #E5E7EB
- Titre: "L - Large"
- Features:
  - 1 To de stockage
  - Illimité pull/push
- Prix: "29.99 EUR/mois"

### Dimensions radio cards
- 200x100px chacune
- 3 en ligne

## 6. INFO BOX URL

### Style
- Background: #EFF6FF
- Border-left: 4px solid #3B82F6
- Text: #1E40AF

### Contenu
- Texte: "Le registry sera accessible via l'URL :"
- URL: "[nom].gra7.container-registry.ovh.net"

## 7. FOOTER MODAL

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Position: droite

### Bouton Suivant
- Style: btn-primary (110x36px)
- Position: droite

## 8. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}/containerRegistry/plan
→ Liste des plans disponibles

GET /cloud/project/{projectId}/region
→ Régions disponibles

POST /cloud/project/{projectId}/containerRegistry
Body: {
  name: "production-registry",
  region: "GRA7",
  planId: "small"
}
→ Crée le registry
```
