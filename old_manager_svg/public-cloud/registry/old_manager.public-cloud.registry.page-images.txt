# Cahier des Charges : public/registry - Page Images

## 1. VUE D'ENSEMBLE
- **Type** : Page liste images Docker
- **Contexte** : Public Cloud Container Registry
- **Objectif** : Gérer les images du registre privé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Push image
- **OdsDatagrid** : Liste images
- **OdsBadge** : Visibilité
- **ActionMenu** : Menu ⋮

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Container Registry" |
| Description | "Registre d'images Docker privé." |

| Propriété | Valeur |
|-----------|--------|

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/containerRegistry - Registre info
- GET /cloud/project/{id}/containerRegistry/{registryId}/repository - Images
- DELETE /cloud/project/{id}/containerRegistry/{registryId}/repository/{repoId} - Supprimer
