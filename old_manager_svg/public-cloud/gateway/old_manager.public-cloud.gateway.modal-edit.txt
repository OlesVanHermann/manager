# Cahier des Charges : public-cloud/gateway - Modifier Gateway

## 1. VUE D'ENSEMBLE
- **Page** : Modifier votre Gateway
- **Type** : Page modification (plein écran)
- **Objectif** : Modifier la taille et le nom d'une Gateway existante

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation Public Cloud
- Sidebar avec menu NETWORK (Gateway actif)
- Breadcrumb : Mon Projet / Gateway / Modifier votre Gateway
- Lien retour "← Revenir à la page précédente"
- Titre principal
- Description
- Info gateway actuelle
- Formulaire de modification (taille, nom)
- Bouton de soumission

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation hiérarchique
- **OdsLink** : Retour
- **OdsText** : Titre, descriptions
- **OdsCard** : Cartes de sélection taille (radio cards)
- **OdsRadio** : Sélection nouvelle taille
- **OdsInput** : Nom du gateway
- **OdsBadge** : "(actuel)" sur la taille courante
- **OdsButton** : Suivant

## 4. TITRE
"Modifier votre Gateway"

## 5. DESCRIPTION
"Sélectionnez la taille de votre Gateway en fonction de vos besoins en matière de bande passante. * Les prix n'incluent pas la TVA."

## 6. GATEWAY ACTUELLE
- Label : "Gateway actuelle :"
- Valeur : "gateway-gra-001 (Région: GRA11)"

## 7. ÉTAPE 1 : MODIFIER LA TAILLE
Cartes de sélection avec radio buttons :

| Taille | Bande passante | Prix HT/mois | État |
|--------|----------------|--------------|------|
| Taille S | Jusqu'à 200 Mbps | 9,99 € | - |
| Taille M | Jusqu'à 500 Mbps | 24,99 € | (actuel) - sélectionné |
| Taille L | Jusqu'à 2 Gbps | 49,99 € | - |

## 8. CHAMP NOM
| Champ | Type | Valeur actuelle |
|-------|------|-----------------|
| Nom du gateway | Input | gateway-gra-001 |

## 9. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Suivant | Primary bleu | PUT modification gateway |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/gateway/{gatewayId}
  - Détails de la Gateway actuelle
- PUT /cloud/project/{projectId}/gateway/{gatewayId}
  - Body: { name, size }
  - Modifier la Gateway
