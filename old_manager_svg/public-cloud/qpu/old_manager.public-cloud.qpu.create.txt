## 7. BOUTONS

- **Équivalent CLI** : Secondary, affiche la commande CLI

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/quantum/region
  - Liste des régions disponibles
- GET /cloud/project/{projectId}/ai/quantum/flavor
  - Liste des flavors notebook
- GET /cloud/project/{projectId}/ai/quantum/qpu
  - Liste des QPU disponibles (Quandela, IBM)
- GET /cloud/project/{projectId}/ai/quantum/framework
  - Liste des frameworks (Perceval, Qiskit, Cirq)
- POST /cloud/project/{projectId}/ai/quantum/notebook
  - Body: { name, region, flavor, qpuId, framework, editor, privacy }
  - Crée le notebook quantum
