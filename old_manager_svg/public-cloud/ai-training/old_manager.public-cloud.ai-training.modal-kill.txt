# Cahier des Charges : public-cloud/ai-training - Modal Arrêter job

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation d'arrêt d'un job
- **Type** : Modal confirmation warning
- **Objectif** : Demander confirmation avant d'arrêter un job en cours

## 2. STRUCTURE MODAL
- Overlay sombre sur la page
- Modal centrée (500x280)
- Header avec titre et bouton fermer
- Message de confirmation
- Alerte warning
- Boutons Annuler / Arrêter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Message et nom du job
- **OdsMessage** : Alerte warning (jaune)
- **OdsButton** : Annuler (outline), Arrêter (warning orange)

## 4. TITRE
"Arrêter le job"

## 5. CONTENU
### Message
"Êtes-vous sûr de vouloir arrêter le job"
"**train-model-v2**" (en gras)
"?"

### Alerte warning
- Fond: Jaune clair (#FEF3C7)
- Bordure: Orange (#F59E0B)
- Icône: ⚠️
- Message: "Cette action interrompra l'exécution du job."
- Sous-message: "Les données non sauvegardées seront perdues."

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline gris | Ferme la modal |
| Arrêter | Warning orange (#F59E0B) | Arrête le job |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/job/{jobId}/kill
