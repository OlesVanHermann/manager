## 7. APIS & ENDPOINTS

### 7.1 Lister les workflows

```
GET /cloud/project/{projectId}/workflow/backup
```

### 7.2 Obtenir les détails d'un workflow

```
GET /cloud/project/{projectId}/workflow/backup/{workflowId}
```

### 7.3 Obtenir les exécutions

```
GET /cloud/project/{projectId}/workflow/backup/{workflowId}/execution
```
