## 4. MENU ACTIONS (dropdown ouvert)

| Dimensions | 220×100px |

### 4.1 Options du menu

| Action | Description |
|--------|-------------|
| Modifier le nom | Ouvre la modal d'édition du nom |
| Activer le renouvellement automatique | Active/désactive le renouvellement |
| Créer une instance | Crée une instance du type du plan |

---

## 5. APIS & ENDPOINTS

### 5.1 Lister les Savings Plans

```
GET /cloud/project/{projectId}/savingsplan
```

### 5.2 Modifier le nom

```
PUT /cloud/project/{projectId}/savingsplan/{savingsPlanId}
{
  "name": "new-name"
}
```

### 5.3 Activer le renouvellement

```
POST /cloud/project/{projectId}/savingsplan/{savingsPlanId}/renew
{
  "autoRenew": true
}
```
