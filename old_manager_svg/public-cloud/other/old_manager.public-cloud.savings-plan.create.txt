## 8. APIS & ENDPOINTS

### 8.1 Lister les modèles d'instance

```
GET /cloud/project/{projectId}/flavor
```

### 8.2 Obtenir les offres Savings Plan

```
GET /cloud/project/{projectId}/savingsplan/offerings
```

### 8.3 Créer un Savings Plan

```
POST /cloud/project/{projectId}/savingsplan
{
  "service": "instance",
  "deploymentZones": 1,
  "model": "b3-8",
  "quantity": 5
}
```
