## 6. APIS & ENDPOINTS

### 6.1 Activer le renouvellement automatique

```
PUT /cloud/project/{projectId}/savingsplan/{savingsPlanId}/renewal
{
  "autoRenew": true
}
```

### 6.2 Désactiver le renouvellement automatique

```
PUT /cloud/project/{projectId}/savingsplan/{savingsPlanId}/renewal
{
  "autoRenew": false
}
```
