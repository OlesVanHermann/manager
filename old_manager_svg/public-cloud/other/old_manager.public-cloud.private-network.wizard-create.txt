## 9. APIS & ENDPOINTS

### 9.1 Créer un réseau privé

```
POST /cloud/project/{projectId}/network/private
{
  "name": "network-name",
  "vlanId": null,
  "regions": ["GRA11"]
}
```

### 9.2 Créer un sous-réseau avec DHCP

```
POST /cloud/project/{projectId}/network/private/{networkId}/subnet
{
  "dhcp": true,
  "network": "192.168.0.0/24",
  "region": "GRA11",
  "noGateway": false
}
```
