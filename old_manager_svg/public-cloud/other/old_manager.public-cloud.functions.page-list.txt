## 12. APIS & ENDPOINTS
- GET /cloud/project/{id}/function - Liste fonctions
- POST /cloud/project/{id}/function - Créer fonction
- PUT /cloud/project/{id}/function/{functionId} - Modifier
- POST /cloud/project/{id}/function/{functionId}/deploy - Déployer
- DELETE /cloud/project/{id}/function/{functionId} - Supprimer
