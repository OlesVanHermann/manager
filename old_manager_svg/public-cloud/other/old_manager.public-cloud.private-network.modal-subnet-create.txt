## 6. APIS & ENDPOINTS

### 6.1 Lister les régions du réseau privé

```
GET /cloud/project/{projectId}/network/private/{networkId}/region
```

### 6.2 Créer un sous-réseau

```
POST /cloud/project/{projectId}/network/private/{networkId}/subnet
{
  "region": "GRA11",
  "cidr": "192.168.1.0/24",
  "ipVersion": 4,
  "gatewayIp": "192.168.1.1",
  "dhcp": true
}
```
