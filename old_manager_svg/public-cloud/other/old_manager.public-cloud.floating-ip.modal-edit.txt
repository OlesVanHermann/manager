## 6. APIS & ENDPOINTS

### 6.1 Lister les instances

```
GET /cloud/project/{projectId}/instance
```

### 6.2 Lister les interfaces réseau d'une instance

```
GET /cloud/project/{projectId}/instance/{instanceId}/interface
```

### 6.3 Modifier l'association Floating IP

```
POST /cloud/project/{projectId}/region/{region}/floatingip/{floatingIpId}/attach
{
  "instanceId": "instance-id",
  "ip": "10.0.0.5"
}
```
