## 8. APIS & ENDPOINTS

### 8.1 Lister les contacts

```
GET /cloud/project/{projectId}/acl
```

### 8.2 Obtenir le contact admin

```
GET /cloud/project/{projectId}
```

### 8.3 Modifier le contact admin

```
POST /me/contact/project/{projectId}/changeContact
{
  "contactAdmin": "newNic-ovh"
}
```
