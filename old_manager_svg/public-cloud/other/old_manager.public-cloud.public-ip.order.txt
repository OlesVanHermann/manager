# Cahier des Charges : public-cloud/public-ip - Commander

## 1. VUE D'ENSEMBLE
- **Page** : Commander une IP publique
- **Type** : Page formulaire wizard 4 étapes
- **Objectif** : Commander une nouvelle IP publique (Floating IP)

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Public IPs actif)
- Breadcrumb
- Titre
- Stepper horizontal (4 étapes)
- Contenu étape actuelle
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsStepper** : 4 étapes (Type, Région, Instance, Résumé)
- **OdsTile** : Sélection région avec radio
- **OdsButton** : Précédent (secondary), Suivant (primary)

## 4. STEPPER
| # | Étape | État |
|---|-------|------|
| ✓ | Type d'IP | Complete (vert) |
| 2 | Région | Active (bleu) |
| 3 | Instance | Inactive (gris) |
| 4 | Résumé | Inactive (gris) |

## 5. ÉTAPE 2 : SÉLECTION RÉGION
- **Titre** : "Sélectionnez une région"
- **Description** : "Choisissez la région où vous souhaitez créer votre Floating IP."

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region
  - Liste des régions disponibles
- POST /cloud/project/{projectId}/ip/floating
  - Body: { region, instanceId? }
  - Crée la Floating IP
