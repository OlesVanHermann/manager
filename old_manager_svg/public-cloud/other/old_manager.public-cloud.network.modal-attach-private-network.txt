## 6. APIS & ENDPOINTS

### 6.1 Lister les réseaux privés disponibles

```
GET /cloud/project/{projectId}/network/private
```

### 6.2 Attacher un réseau privé à l'instance

```
POST /cloud/project/{projectId}/instance/{instanceId}/interface
{
  "networkId": "network-id"
}
```
