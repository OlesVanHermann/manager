# Cahier des Charges : public-cloud/ssh-keys - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer la clé SSH
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'une clé SSH

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (500x280px)
- Header rouge avec warning
- Warning message
- Info clé
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. HEADER
Fond rouge clair : "Supprimer la clé SSH" + bouton ×

## 5. WARNING
Fond jaune : "⚠ Cette clé ne pourra plus être utilisée pour créer de nouvelles instances."

## 6. INFO CLÉ
- **Label** : "Clé à supprimer :"
- **Valeur** : "my-laptop (RSA 4096)"

## 7. BOUTONS
- Annuler (outline)
- Supprimer (danger/rouge)

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/sshkey/{keyId}
  - Supprime la clé SSH
