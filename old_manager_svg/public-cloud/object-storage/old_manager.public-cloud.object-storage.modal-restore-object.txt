# Cahier des Charges : public-cloud/object-storage - Modal Restaurer Objet

## 1. VUE D'ENSEMBLE
- **Modal** : Restaurer l'objet (Glacier)
- **Type** : Modal formulaire
- **Objectif** : Restaurer un objet archivé depuis Glacier

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x400px)
- Barre bleue à gauche
- Icône refresh
- Titre
- Bandeau info
- Formulaire (durée, type)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : ↻ dans cercle bleu
- **OdsMessage** : Info (bleu)
- **OdsText** : Nom objet
- **OdsInput** : Durée en jours
- **OdsSelect** : Type de restauration
- **OdsButton** : Annuler (secondary), Restaurer (primary)

## 4. HEADER
- **Icône** : ↻ dans cercle bleu
- **Titre** : "Restaurer l'objet (Glacier)"

## 5. BANDEAU INFO
Fond bleu clair :
"ℹ L'objet sera accessible pendant la durée spécifiée après la restauration depuis Glacier."

## 6. FORMULAIRE

### Objet
- **Label** : "Objet"
- **Valeur** : "archive-2023-data.tar.gz" (non modifiable)

### Durée de disponibilité (jours)
- **Input** : Champ numérique
- **Valeur** : 7
- **Helper** : "Nombre de jours pendant lesquels l'objet sera accessible (1-365)"

### Type de restauration
- **Select** : Dropdown
- **Options** :
  - Standard (3-5 heures) (sélectionné)
  - Expedited (1-5 minutes)
  - Bulk (5-12 heures)

## 7. BOUTONS
- Annuler (secondary)
- Restaurer (primary)

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/storage/{containerId}/object/{objectKey}/restore
  - Body: { days: number, tier: "Standard" | "Expedited" | "Bulk" }
  - Initie la restauration depuis Glacier
