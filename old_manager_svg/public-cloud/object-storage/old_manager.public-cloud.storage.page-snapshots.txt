## 8. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}/volume/snapshot
→ Liste des snapshots

GET /cloud/project/{projectId}/volume/snapshot/{snapshotId}
→ Détails snapshot

POST /cloud/project/{projectId}/volume/{volumeId}/snapshot
Body: { name: "snap-name" }
→ Crée un snapshot

POST /cloud/project/{projectId}/volume/snapshot/{snapshotId}/volume
Body: { name: "new-volume" }
→ Crée un volume depuis snapshot

POST /cloud/project/{projectId}/volume/snapshot/{snapshotId}/image
Body: { name: "new-image" }
→ Crée une image depuis snapshot

DELETE /cloud/project/{projectId}/volume/snapshot/{snapshotId}
→ Supprime le snapshot
```
