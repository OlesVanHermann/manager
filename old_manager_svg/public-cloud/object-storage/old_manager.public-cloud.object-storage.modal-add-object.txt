# Cahier des Charges : public-cloud/object-storage - Modal Ajouter Objet

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un objet
- **Type** : Modal formulaire avec upload
- **Objectif** : Uploader des fichiers vers un conteneur

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x550px)
- Barre bleue à gauche
- Icône upload
- Titre + bouton fermer
- Formulaire (préfixe, classe stockage, zone upload)
- Liste fichiers sélectionnés
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : ↑ dans cercle bleu
- **OdsInput** : Préfixe optionnel
- **OdsTile** : Sélection classe stockage (radio)
- **OdsFileUpload** : Zone drag & drop
- **OdsButton** : Parcourir (outline)
- **OdsChip** : Fichier ajouté avec taille et bouton supprimer
- **OdsButton** : Annuler (secondary), Uploader (primary)

## 4. HEADER
- **Icône** : ↑ (upload) dans cercle bleu
- **Titre** : "Ajouter un objet"
- **Bouton fermer** : ✕

## 5. FORMULAIRE

### Préfixe (optionnel)
- **Input** : Champ texte
- **Placeholder** : "dossier/sous-dossier/"

### Classe de stockage
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Standard | ✅ | Par défaut |
| Infrequent Access | - | Moins fréquent |
| High Performance | - | Haute performance |

### Zone Upload
- Zone drag & drop avec bordure pointillée
- Texte : "Glissez vos fichiers ici"
- Texte : "ou"
- Bouton : "Parcourir..."

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/storage/{containerId}/object
  - Body: FormData (fichier)
  - Headers: Content-Type, X-Object-Meta-*, X-Storage-Class
  - Upload un objet vers le conteneur
