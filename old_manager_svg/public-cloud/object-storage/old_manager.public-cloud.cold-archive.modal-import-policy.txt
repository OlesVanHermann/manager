# Cahier des Charges : public-cloud/cold-archive - Modal Importer Policy

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal importer fichier policy
- **Type** : Modal upload fichier
- **Objectif** : Importer une policy S3 au format JSON

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 450x280px centrée
- Header avec titre et bouton fermer
- Description
- Zone de drop fichier
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (450x280px)
- **OdsText** : Description
- **OdsDropzone** : Zone d'upload fichier
- **OdsButton** : Annuler (ghost/outline)
- **OdsButton** : Importer (primaire)

## 4. TITRE
"Importer un fichier"

## 5. DESCRIPTION
"Vous pouvez importer un fichier JSON pour gérer les droits de votre utilisateur."

## 6. DROPZONE
- Fond: Gris clair (#F9FAFB)
- Bordure: Pointillée grise (#E5E7EB)
- Dimensions: 402x80px
- Texte: "Glissez vos fichiers ici ou"
- Lien: "Importer des fichiers" (bleu)
- Formats acceptés: JSON

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Ghost/Outline bleu | Gauche |
| Importer | Primaire bleu | Droite |

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/user/{userId}/policy
  - Body: Fichier JSON de policy S3
  - Importe la policy pour l'utilisateur
