# Cahier des Charges : public-cloud/object-storage - Créer Conteneur

## 1. VUE D'ENSEMBLE
- **Page** : Créer un conteneur d'objets
- **Type** : Formulaire wizard avec récapitulatif
- **Objectif** : Créer un nouveau conteneur Object Storage

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Object Storage actif)
- Breadcrumb
- Titre
- Formulaire (colonne gauche)
- Récapitulatif (colonne droite)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTile** : Cartes type/région avec radio
- **OdsRadio** : Sélections multiples
- **OdsTabs** : Tabs région (1-AZ, 3-AZ, Local Zone)
- **OdsInput** : Nom conteneur
- **OdsSelect** : Options (versioning, chiffrement, utilisateur)
- **OdsCard** : Récapitulatif
- **OdsButton** : Créer (primary)
