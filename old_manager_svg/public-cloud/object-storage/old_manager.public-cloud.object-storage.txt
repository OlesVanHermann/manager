# Cahier des Charges : public-cloud/object-storage - Liste Principale

## 1. VUE D'ENSEMBLE
- **Page** : Object Storage - Liste des conteneurs
- **Type** : Page liste principale avec datagrid
- **Objectif** : Vue d'ensemble et gestion des conteneurs

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Object Storage actif avec indicateur)
- Breadcrumb
- Header page (titre)
- Onglets (Mes conteneurs, Utilisateurs, Configurations)
- Bouton créer
- Barre de recherche

- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre "Object Storage"

- **OdsButton** : "+ Créer un conteneur" (primary)
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste conteneurs
- **OdsBadge** : Type (S3 bleu, Swift jaune), Statut (Actif vert)
- **OdsActionMenu** : Actions (⋮)

## 6. TABLEAU CONTENEURS

### Colonnes
| Colonne | Description |
|---------|-------------|
| Nom | Lien vers détail |
| Région | Code région (GRA, SBG) |
| Type | Badge S3/Swift |
| Classe de stockage | Standard, Standard IA |
| Objets | Nombre |
| Taille | Taille totale |
| Statut | Badge Actif |
| Actions | Menu ⋮ |

## 7. ACTION MENU
Options (menu ouvert) :
- Voir les objets
- Ajouter un objet
- Gérer le versioning
- ---
- Supprimer (rouge)

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/storage
  - Liste des conteneurs
- POST /cloud/project/{projectId}/storage
  - Crée un conteneur
- DELETE /cloud/project/{projectId}/storage/{containerId}
  - Supprime un conteneur
- PUT /cloud/project/{projectId}/storage/{containerId}/versioning
  - Configure le versioning
