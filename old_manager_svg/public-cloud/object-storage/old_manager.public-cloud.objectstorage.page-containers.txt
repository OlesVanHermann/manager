- **Type** : Page liste containers S3
- **Contexte** : Public Cloud Object Storage
- **Objectif** : Gérer les buckets de stockage objet

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/storage/container - Liste containers
- POST /cloud/project/{id}/storage/container - Créer container
- PUT /cloud/project/{id}/storage/container/{name} - Modifier
- DELETE /cloud/project/{id}/storage/container/{name} - Supprimer
