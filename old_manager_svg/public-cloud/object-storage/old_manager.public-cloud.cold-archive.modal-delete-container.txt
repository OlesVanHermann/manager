# Cahier des Charges : public-cloud/cold-archive - Modal Supprimer Conteneur

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal suppression conteneur
- **Type** : Modal danger/confirmation
- **Objectif** : Supprimer définitivement un conteneur Cold Archive

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 600x400px centrée
- Header avec titre et bouton fermer
- Bannière danger avec avertissement
- Nom du conteneur à supprimer
- Champ de confirmation (saisir le nom)
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (600x400px)
- **OdsMessage** : Danger banner (rouge clair)
- **OdsText** : Nom du conteneur (monospace)
- **OdsInput** : Confirmation nom conteneur
- **OdsButton** : Annuler (outline)
- **OdsButton** : Supprimer (danger rouge)

## 4. TITRE
"Supprimer le conteneur"

## 5. BANNIÈRE DANGER
- Fond: Rouge clair (#FEF2F2)
- Titre: "⚠ Attention" (rouge, bold)
- Texte: "Cette action est irréversible. Toutes les données contenues dans le conteneur seront définitivement supprimées."

## 6. CONTENEUR À SUPPRIMER
- Label: "Vous êtes sur le point de supprimer le conteneur :"
- Nom: "logs-production" (monospace, bold)

## 7. CONFIRMATION
- Instruction: "Pour confirmer, tapez le nom du conteneur :"
- Input: Champ texte (préremplit avec placeholder du nom)
- Validation: Doit correspondre exactement au nom du conteneur

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Supprimer | Danger rouge (#DC2626) | Droite |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}
  - Supprime le conteneur et toutes ses données
  - Nécessite que le conteneur soit vide ou force=true
