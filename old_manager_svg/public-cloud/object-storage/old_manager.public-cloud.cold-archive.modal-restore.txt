# Cahier des Charges : public-cloud/cold-archive - Modal Restaurer

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal restaurer archive
- **Type** : Modal confirmation
- **Objectif** : Restaurer une archive Cold Archive

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 600x420px centrée
- Header avec titre et bouton fermer
- Description avec nom archive
- Info sur durée minimale stockage
- Détails sur frais supplémentaires
- Bannière info temps de restauration
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (600x420px)
- **OdsText** : Description et avertissements
- **OdsMessage** : Info banner (bleu)
- **OdsButton** : Annuler (outline)
- **OdsButton** : Confirmer (primaire)

## 4. TITRE
"Restaurer votre archive"

## 5. CONTENU

### Description
"Vous êtes sur le point de restaurer vos données de l'archive **compliance-data**."

### Durée minimale
"Cold Archive a une durée de stockage minimale de 180 jours."

### Frais supplémentaires
"La restauration des données avant 180 jours entraîne des frais supplémentaires égaux aux coûts de restauration plus les frais de stockage actif pour les objets restaurés plus les coûts de stockage archivé actuels."

## 6. BANNIÈRE INFO
- Fond: Bleu clair (#E0F2FE)
- Bordure: Bleu (#0284C7)
- Icône: ℹ
- Texte: "Le temps de restauration dépend du volume de données. Quelques minutes à plusieurs heures selon la taille."

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Confirmer | Primaire bleu | Droite |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/restore
  - Déclenche la restauration de l'archive
- GET /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/restore/status
  - Statut de la restauration en cours
