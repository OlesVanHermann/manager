# Cahier des Charges : public-cloud/object-storage - Modal Activer Chiffrement

## 1. VUE D'ENSEMBLE
- **Modal** : Activer le chiffrement (SSE-OMK)
- **Type** : Modal confirmation avec info
- **Objectif** : Activer le chiffrement côté serveur

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x340px)
- Barre bleue à gauche
- Icône info
- Titre
- Description
- Warning
- Lien documentation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : ℹ dans cercle bleu
- **OdsMessage** : Warning (jaune)
- **OdsLink** : "En savoir plus →"
- **OdsButton** : Annuler (secondary), Confirmer (primary)

## 4. HEADER
- **Icône** : ℹ dans cercle bleu
- **Titre** : "Activer le chiffrement (SSE-OMK)"

## 5. DESCRIPTION
"Active le chiffrement côté serveur avec des clés gérées par OVHcloud pour sécuriser vos données au repos."

## 6. WARNING
Fond jaune :
"⚠ Une fois activé, le chiffrement s'appliquera uniquement sur les nouveaux objets uploadés, et ne pourra plus être désactivé."

## 7. LIEN
"En savoir plus →"

## 8. BOUTONS
- Annuler (secondary)
- Confirmer (primary)

## 9. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/storage/{containerId}/encryption
  - Body: { enabled: true }
  - Active le chiffrement SSE-OMK
