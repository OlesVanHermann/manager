# Cahier des Charges : public-cloud/object-storage - Modal Supprimer Conteneur

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le conteneur
- **Type** : Modal confirmation destructive avec saisie TERMINATE
- **Objectif** : Supprimer un conteneur et tous ses objets

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x350px)
- Barre orange à gauche (warning)
- Icône warning
- Titre
- Message de confirmation
- Champ saisie TERMINATE
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : ⚠ dans cercle orange
- **OdsText** : Message avec nom conteneur en rouge
- **OdsInput** : Champ saisie confirmation
- **OdsButton** : Annuler (secondary), Supprimer (danger)

## 4. HEADER
- **Icône** : ⚠ dans cercle orange
- **Titre** : "Supprimer le conteneur"

## 5. CONTENU

### Message
"Êtes-vous sûr de vouloir supprimer le conteneur"
**backup-prod-2024** (rouge) "?"

"Cette action est irréversible. Tous les objets contenus seront définitivement supprimés."

### Confirmation
- **Label** : "Tapez TERMINATE pour confirmer"
- **Input** : Champ texte
- **Placeholder** : "TERMINATE"

## 6. BOUTONS
- Annuler (secondary)
- Supprimer (danger/rouge)

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/storage/{containerId}
  - Supprime le conteneur et tous ses objets
