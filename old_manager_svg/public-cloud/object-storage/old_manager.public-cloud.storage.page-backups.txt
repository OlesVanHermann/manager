# Cahier des Charges : public/storage - Page Backups

## 1. VUE D'ENSEMBLE
- **Type** : Page liste backups volumes
- **Contexte** : Public Cloud Block Storage
- **Objectif** : Gérer les sauvegardes de volumes

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer backup
- **OdsDatagrid** : Liste backups
- **OdsBadge** : État
- **ActionMenu** : Menu ⋮

## 4. HEADER
| Propriété | Valeur |

| Action | Style |
|--------|-------|
| Créer volume | Normal |
| Télécharger | Normal |
| --- | Séparateur |
| Supprimer | Danger (#DC2626) |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/volume/backup - Liste backups
- POST /cloud/project/{id}/volume/{volumeId}/backup - Créer backup
- POST /cloud/project/{id}/volume/backup/{backupId}/volume - Restaurer
- DELETE /cloud/project/{id}/volume/backup/{backupId} - Supprimer
