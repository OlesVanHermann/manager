# Cahier des Charges : public-cloud/object-storage - Onglet Utilisateurs

## 1. VUE D'ENSEMBLE
- **Page** : Conteneur Object Storage - Onglet Utilisateurs
- **Type** : Page liste des utilisateurs S3 du conteneur
- **Objectif** : Gérer les accès utilisateurs au conteneur

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Breadcrumb
- Header conteneur (nom, résumé)
- Onglets (General, Objets, Utilisateurs actif)
- Titre section avec compteur
- Bouton créer
- Tableau utilisateurs

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre, résumé
- **OdsTabs** : General | Objets | Utilisateurs
- **OdsChip** : Compteur utilisateurs (2)
- **OdsButton** : "Créer un utilisateur" (primary)
- **OdsDatagrid** : Liste utilisateurs
- **OdsBadge** : Permissions (readwrite vert, readonly bleu)
- **OdsButton** : "Credentials" par ligne (outline)

## 4. HEADER CONTENEUR
- **Titre** : "assets-production"
- **Résumé** : "GRA | S3 | 1,234 objets | 45.2 GB"

## 5. ONGLETS
| Onglet | Actif |
|--------|-------|
| General | - |
| Objets | - |
| Utilisateurs | ✅ (souligné bleu) |

## 6. HEADER SECTION
- **Titre** : "Utilisateurs S3"
- **Compteur** : (2) dans chip bleu
- **Action** : "Créer un utilisateur" (primary)

## 7. TABLEAU UTILISATEURS

### Colonnes
| Colonne | Description |
|---------|-------------|
| Nom | Nom utilisateur |
| Access Key | Clé d'accès (monospace) |
| Permissions | Badge rôle |
| Créé le | Date création |
| Actions | Bouton Credentials |

### Données exemple
| Nom | Access Key | Permissions | Créé le | Actions |
|-----|------------|-------------|---------|---------|
| app-backend | AKIAIOSFODNN7EXAMPLE | readwrite (vert) | 15/12/2024 | [Credentials] |
| cdn-readonly | AKIAI44QH8DHBEXAMPLE | readonly (bleu) | 20/12/2024 | [Credentials] |

### Badges permissions
| Permission | Couleur |
|------------|---------|
| readwrite | Vert |
| readonly | Bleu |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/storage/{containerId}/user
  - Liste des utilisateurs du conteneur
- POST /cloud/project/{projectId}/storage/{containerId}/user
  - Crée un utilisateur avec accès au conteneur
- GET /cloud/project/{projectId}/storage/{containerId}/user/{userId}/credentials
  - Affiche les credentials
