# Cahier des Charges : public-cloud/rancher - Modal Mettre à Jour Version

## 1. VUE D'ENSEMBLE
- **Modal** : Mettre à jour la version
- **Type** : Modal formulaire avec sélection
- **Objectif** : Mettre à jour la version de Rancher

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x420px)
- Titre avec close
- Description
- Version actuelle
- Sélection nouvelle version
- Warning interruption
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Version actuelle
- **OdsSelect** : Sélection nouvelle version
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler (secondary), Mettre à jour (primary)

## 4. HEADER
"Mettre à jour la version" + bouton ×

## 5. DESCRIPTION
"Choisissez la nouvelle version de Rancher pour votre service."

## 6. VERSIONS

### Version actuelle
- **Label** : "Version actuelle"
- **Valeur** : v2.8.4

### Nouvelle version
- **Label** : "Nouvelle version"
- **Select** : Dropdown avec versions
- **Valeur** : "v2.8.5 (recommandée)"

## 7. WARNING
Fond jaune :
"⚠️ La mise à jour entraînera une interruption de service de quelques minutes. Assurez-vous que vos clusters downstream sont préparés."

## 8. BOUTONS
- Annuler (secondary)
- Mettre à jour (primary)

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/rancher/{rancherId}/version
  - Liste des versions disponibles
- PUT /cloud/project/{projectId}/rancher/{rancherId}/version
  - Body: { version }
  - Lance la mise à jour
