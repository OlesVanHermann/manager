# Cahier des Charges : public-cloud/ai-notebooks - Créer un notebook

## 1. VUE D'ENSEMBLE
- **Page** : Formulaire de création d'un notebook
- **Type** : Page wizard avec récapitulatif
- **Objectif** : Configurer et lancer un nouveau notebook

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec AI Notebooks actif
- Breadcrumb
- Titre "Créer un notebook"
- Colonne gauche: Formulaire (sections)
- Colonne droite: Récapitulatif commande

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsCard** : Sections formulaire
- **OdsInput** : Nom du notebook
- **OdsRadio/OdsTile** : Région, Ressources, Framework, Éditeur, Confidentialité
- **OdsCard** : Récapitulatif (droite)
- **OdsButton** : Commander, Équivalent CLI
- **OdsLink** : Configuration avancée

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/capabilities
  - Régions, flavors, frameworks, éditeurs
- POST /cloud/project/{projectId}/ai/notebook
  - name: string
  - region: string
  - flavorId: string
  - framework: string
  - frameworkVersion: string
  - editor: "jupyterlab" | "vscode"
  - privacy: "private" | "public"
  - volumes: array (optionnel)
  - sshKeys: array (optionnel)
