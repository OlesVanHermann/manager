# Cahier des Charges : public-cloud/ai-notebooks - Modal Arrêter

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation d'arrêt d'un notebook
- **Type** : Modal confirmation warning
- **Objectif** : Demander confirmation avant d'arrêter un notebook

## 2. STRUCTURE MODAL
- Overlay sombre sur la page
- Modal centrée (500x280)
- Header avec titre et bouton fermer
- Message de confirmation
- Alerte warning (perte de données)
- Boutons Annuler / Arrêter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Message et nom du notebook
- **OdsMessage** : Alerte warning (jaune)
- **OdsButton** : Annuler (outline), Arrêter (warning orange)

## 4. TITRE
"Arrêter le notebook"

## 5. CONTENU
### Message
"Êtes-vous sûr de vouloir arrêter le notebook"
"**my-ml-notebook**" (en gras)
"?"

### Alerte warning
- Fond: Jaune clair (#FEF3C7)
- Bordure: Orange (#F59E0B)
- Icône: ⚠️
- Message: "Attention : Toutes les données non sauvegardées dans le stockage persistant seront perdues."

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline gris | Ferme la modal |
| Arrêter | Warning orange (#F59E0B) | Arrête le notebook |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/notebook/{notebookId}/stop
