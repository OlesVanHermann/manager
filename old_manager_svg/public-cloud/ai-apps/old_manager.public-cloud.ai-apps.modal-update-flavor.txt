# Cahier des Charges : public-cloud/ai-apps - Modal Modifier les ressources

## 1. VUE D'ENSEMBLE
- **Modal** : Modification des ressources (flavor) d'une app
- **Type** : Modal formulaire avec cartes sélectionnables
- **Objectif** : Changer le type de ressources GPU/CPU

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Section sélection flavor (cartes radio)
- Section quantité (stepper)
- Récapitulatif des ressources
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x500)
- **OdsCard** : Cartes flavor (radio style)
- **OdsStepper** : Quantité
- **OdsCard** : Récapitulatif (fond gris)
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Modifier les ressources"

## 5. SÉLECTION FLAVOR
| Flavor | Specs | Prix | État |
|--------|-------|------|------|
| ai1-1-gpu | 1 GPU NVIDIA V100S, 32 GB VRAM | 1,23 €/h | Sélectionné |
| ai1-4-gpu | 4 GPU NVIDIA V100S, 128 GB VRAM | 4,92 €/h | Non sélectionné |
| cpu-medium | 4 vCores, 16 GB RAM | 0,08 €/h | Non sélectionné |

## 6. QUANTITÉ
- Label: "Sélectionner la quantité de ressources"
- Info: "Quantité maximale pour {flavor} : 4"
- Stepper: Input avec boutons +/-
- Valeur: 1

## 7. RÉCAPITULATIF
Fond gris (#F9FAFB) :
| Spec | Valeur |
|------|--------|
| CPU | 8 vCores |
| RAM | 64 GB |
| Stockage local | 340 GB SSD |
| Réseau | 3 Gbps |
| GPU | 1 × NVIDIA V100S (32 GB) |
| Prix | 1,23 € HT/h |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Confirmer | Primaire bleu | Applique les changements |

## 9. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/ai/app/{appId}/flavor
  - flavorId: string
  - quantity: number
