# Cahier des Charges : public-cloud/kubernetes - Modal Réinitialiser Cluster

## 1. VUE D'ENSEMBLE
- **Modal** : Réinitialiser la configuration du cluster
- **Type** : Modal danger avec confirmation
- **Objectif** : Réinitialiser le cluster à son état original

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 700x540px
- Titre + bouton fermer
- Banner warning (conséquences)
- Options radio (supprimer/réinstaller nœuds)
- Sélecteur version
- Champ confirmation "RESET"
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsRadio** : Options nœuds
- **OdsSelect** : Version Kubernetes
- **OdsInput** : Confirmation
- **OdsButton** : Annuler, Réinitialiser (danger)

## 4. TITRE
"Réinitialiser la configuration du cluster"

## 5. BANNER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- Texte :
  - "⚠️ Votre cluster Kubernetes sera réinstallé dans son état original."
  - "Cela supprimera l'intégralité de vos services, leur configuration, etc."
  - "**Cette action est irréversible.**" (rouge)

## 6. OPTIONS NŒUDS
| Option | Description | Sélection |
|--------|-------------|-----------|
| Supprimer les nœuds | Les nœuds seront supprimés et devront être recréés | Sélectionné |
| Réinstaller les nœuds | Les nœuds seront réinstallés avec leur configuration actuelle | - |

## 7. VERSION KUBERNETES
- Label : "Version mineure de Kubernetes"
- Select : "1.28 (version actuelle)"

## 8. CONFIRMATION
- Label : "Entrez 'RESET' dans le champ ci-dessous pour confirmer"
- Input : RESET

## 9. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Réinitialiser | Danger rouge | Droite | POST reset |

## 10. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/kube/{kubeId}/reset
  - Body: { workerNodesPolicy: "delete"|"reinstall", version }
