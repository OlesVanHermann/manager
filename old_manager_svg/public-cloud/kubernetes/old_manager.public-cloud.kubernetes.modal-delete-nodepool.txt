# Cahier des Charges : public-cloud/kubernetes - Modal Supprimer Node Pool

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le pool de nœuds
- **Type** : Modal danger avec confirmation DELETE
- **Objectif** : Supprimer un pool de nœuds

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x380px
- Header jaune (warning)
- Titre warning + bouton fermer
- Question de confirmation
- Banner danger rouge
- Champ confirmation "DELETE"
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Danger (rouge clair)
- **OdsInput** : Confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- Titre orange : "Supprimer le pool de nœuds"

## 5. QUESTION
"Êtes-vous sûr de supprimer **worker-pool-1** ?"

## 6. BANNER DANGER
- Fond rouge clair (#FEF2F2)
- Bordure rouge (#FCA5A5)
- Icône ⚠ rouge
- "Cette action est irréversible."
- "Tous les nœuds du pool seront supprimés définitivement."

## 7. CHAMP CONFIRMATION
- Label : "Entrez 'DELETE' dans le champ ci-dessous pour confirmer"
- Input : DELETE

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Primary bleu | Droite | DELETE pool |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/kube/{kubeId}/nodepool/{poolId}
  - Supprimer le pool de nœuds
