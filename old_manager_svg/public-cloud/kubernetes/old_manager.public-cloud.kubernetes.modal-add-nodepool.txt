# Cahier des Charges : public-cloud/kubernetes - Modal Ajouter Node Pool

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un pool de nœuds
- **Type** : Modal formulaire
- **Objectif** : Créer un nouveau pool de nœuds dans le cluster

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 800x650px
- Icône + titre + bouton fermer
- Formulaire complet
- Estimation prix
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur large
- **OdsIcon** : Plus (+) cercle bleu
- **OdsInput** : Nom du pool
- **OdsCard** : Cartes sélection flavor
- **OdsQuantity** : Nombre de nœuds (−/+)
- **OdsToggle** : Autoscaling
- **OdsCard** : Type facturation (Horaire/Mensuelle)
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un pool de nœuds"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Valeur |
|-------|------|--------|
| Nom du pool de nœuds | Input | nodepool-2 |
| Type de nœuds | Cards | b2-7 / b2-15 / b2-30 |
| Nombre de nœuds | Quantity | 3 |
| Autoscaling | Toggle | Désactivé |
| Type de facturation | Cards | Horaire / Mensuelle |

## 6. CARTES FLAVORS
| Flavor | Specs | Prix |
|--------|-------|------|
| b2-7 (sélectionné) | 2 vCores, 7 Go | ~18 €/mois |
| b2-15 | 4 vCores, 15 Go | ~35 €/mois |
| b2-30 | 8 vCores, 30 Go | ~70 €/mois |

## 7. ESTIMATION
- Fond gris (#F3F4F6)
- Titre : "Estimation mensuelle"
- Pools de nœuds : 54,00 €
- Prix total estimé : **54,00 €**

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST nodepool |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/flavor
  - Flavors disponibles
- POST /cloud/project/{projectId}/kube/{kubeId}/nodepool
  - Body: { name, flavorName, desiredNodes, autoscale, monthlyBilled }
