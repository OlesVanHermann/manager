# Cahier des Charges : public-cloud/kubernetes - Onglet Kubeconfig

## 1. VUE D'ENSEMBLE
- **Page** : Onglet Kubeconfig du cluster
- **Type** : Vue gestion kubeconfig
- **Objectif** : Télécharger/générer le fichier kubeconfig

## 2. STRUCTURE PAGE
- Header avec nom cluster et infos
- Onglets : General, Node Pools, Kubeconfig (actif)
- Titre "Kubeconfig"
- Card principale avec boutons
- Card utilisation avec commande

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes d'informations
- **OdsButton** : Générer, Copier, Télécharger, Regénérer
- **OdsClipboard** : Commande copiable

## 4. EN-TÊTE
- Breadcrumb : Public Cloud > Mon Projet Prod > Kubernetes

- Info : GRA11 | v1.28 | 6 nodes | READY (vert)

## 5. ONGLETS
- General
- Node Pools
- **Kubeconfig** (actif, souligné bleu)

## 6. CARD PRINCIPALE
- Titre : "Kubeconfig"
- Description : "Téléchargez le fichier kubeconfig pour accéder à votre cluster."
- Bouton : **"Générer kubeconfig"** (Primary bleu)

Après génération, boutons supplémentaires :
| Bouton | Style | Action |
|--------|-------|--------|
| Copier | Outline | Copier contenu |
| Télécharger | Outline | Download fichier |
| Regénérer | Outline | Nouveau kubeconfig |

## 7. CARD UTILISATION
- Titre : "Utilisation"
- Description : "Utilisez cette commande pour accéder à votre cluster:"
- Code : `export KUBECONFIG=./kubeconfig.yaml && kubectl get nodes`
- Fond gris (#F3F4F6), police monospace

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/kube/{kubeId}/kubeconfig
  - Télécharger le kubeconfig
- POST /cloud/project/{projectId}/kube/{kubeId}/kubeconfig/reset
  - Regénérer le kubeconfig
