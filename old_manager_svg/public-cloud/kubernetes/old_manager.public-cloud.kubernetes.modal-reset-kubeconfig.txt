# Cahier des Charges : public-cloud/kubernetes - Modal Reset Kubeconfig

## 1. VUE D'ENSEMBLE
- **Modal** : Réinitialiser le kubeconfig
- **Type** : Modal warning avec confirmation
- **Objectif** : Générer un nouveau fichier kubeconfig (révoque les anciens accès)

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x300px avec barre orange gauche
- Icône warning orange + titre
- Question de confirmation
- Banner warning jaune (révocation accès)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec accent orange
- **OdsIcon** : Warning orange
- **OdsText** : Question
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler, Réinitialiser

## 4. TITRE
"Réinitialiser le kubeconfig"

## 5. QUESTION
"Êtes-vous sûr de vouloir réinitialiser le fichier kubeconfig de votre cluster ?"

## 6. BANNER WARNING
- Fond jaune (#FEF3C7)
- "⚠ Les accès existants seront révoqués. Vous devrez télécharger un nouveau fichier kubeconfig."

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Réinitialiser | Primary bleu | Droite | POST reset kubeconfig |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/kube/{kubeId}/kubeconfig/reset
  - Réinitialiser le kubeconfig
