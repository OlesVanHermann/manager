| Clic hors menu | Fermer ActionMenu |

## 8. APIS & ENDPOINTS

### 8.1 Lister les instances SAP HANA

```
GET /sap-hana
```

Response:
```json
[
  {
    "serviceName": "sap-hana-abc123",
    "displayName": "SAP HANA Production",
    "offer": "SAP HANA Large",
    "serverCount": 2,
    "datacenter": "rbx7",
    "datacenterName": "RBX",
    "status": "active"
  },
  {
    "serviceName": "sap-hana-def456",
    "displayName": "SAP HANA Dev",
    "offer": "SAP HANA Medium",
    "serverCount": 1,
    "datacenter": "gra3",

]
```
