## 8. APIS & ENDPOINTS

### 8.1 Activer une licence Veeam

```
POST /veeam/enterprise/{serviceName}/activate
```

Body:
```json
{
  "ip": "192.0.2.100",
  "username": "admin",
  "password": "********"
}
```

Response:
```json
{
  "serviceName": "veeam-ent-abc123",
  "status": "active",
  "activationDate": "2025-01-02"
}
```
