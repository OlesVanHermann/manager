## 7. APIS & ENDPOINTS

### 7.1 Obtenir détails licence Veeam

```
GET /veeam/enterprise/{serviceName}
```

Response:
```json
{
  "serviceName": "veeam-ent-abc123",
  "licenseType": "enterprise_plus",
  "sourceIp": "192.0.2.50",
  "destinationIp": "192.0.2.100",
  "version": "Veeam Enterprise 12",
  "vmsProtected": 50,
  "vmsMax": 100,
  "status": "active",
  "renewalDate": "2026-01-15",
  "creationDate": "2024-01-15",
  "contacts": {
    "admin": "admin@example.com",
    "billing": "billing@example.com",
    "tech": "tech@example.com"
  }
}
```
