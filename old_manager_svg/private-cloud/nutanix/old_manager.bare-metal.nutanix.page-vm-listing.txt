# Cahier des Charges : bare-metal/nutanix - Page VM Listing

## 1. VUE D'ENSEMBLE
- **Type** : Page listing machines virtuelles
- **Contexte** : Cluster Nutanix
- **Objectif** : Gérer les VMs du cluster Nutanix

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer VM
- **OdsInput** : Recherche
- **OdsDatagrid** : Tableau VMs
- **OdsBadge** : État VM
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |

## 11. APIS & ENDPOINTS
- GET /nutanix/{serviceName}/cluster/{clusterId}/vm - Liste VMs
- POST /nutanix/{serviceName}/cluster/{clusterId}/vm - Créer VM
- GET /nutanix/{serviceName}/cluster/{clusterId}/vm/{vmId} - Détail VM
- POST /nutanix/{serviceName}/cluster/{clusterId}/vm/{vmId}/start - Démarrer
- POST /nutanix/{serviceName}/cluster/{clusterId}/vm/{vmId}/stop - Arrêter
- POST /nutanix/{serviceName}/cluster/{clusterId}/vm/{vmId}/reboot - Redémarrer
- POST /nutanix/{serviceName}/cluster/{clusterId}/vm/{vmId}/snapshot - Créer snapshot
- DELETE /nutanix/{serviceName}/cluster/{clusterId}/vm/{vmId} - Supprimer
