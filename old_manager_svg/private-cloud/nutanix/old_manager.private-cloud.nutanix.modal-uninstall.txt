## 6. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Saisie != "UNINSTALL" | Bouton Desinstaller désactivé |
| Saisie == "UNINSTALL" | Bouton Desinstaller activé |
| Clic Annuler | Fermer modal |
| Clic × | Fermer modal |
| Clic Desinstaller | Exécuter désinstallation nœud |

---

## 7. APIS & ENDPOINTS

### 7.1 Désinstaller un nœud

```
POST /nutanix/{serviceName}/nodes/{nodeId}/uninstall
```

Body:
```json
{
  "confirmation": "UNINSTALL"
}
```

Response:
```json
{
  "taskId": "task-xyz-789",
  "status": "pending",
  "estimatedDuration": "30 minutes"
}
```
