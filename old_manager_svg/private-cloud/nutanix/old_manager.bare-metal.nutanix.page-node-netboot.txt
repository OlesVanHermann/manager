## 8. APIS & ENDPOINTS

### 8.1 Obtenir configuration netboot

```
GET /nutanix/{serviceName}/nodes/{nodeId}/boot
```

Response:
```json
{
  "bootMode": "harddisk",
  "kernel": null,
  "rootDevice": null,
  "availableModes": ["harddisk", "rescue", "network"],
  "availableKernels": ["rescue64-pro", "rescue-customer"],
  "availableRootDevices": ["hd0", "hd1", "hd2"]
}
```

### 8.2 Configurer netboot

```
PUT /nutanix/{serviceName}/nodes/{nodeId}/boot
```

Body:
```json
{
  "bootMode": "rescue",
  "kernel": "rescue64-pro",
  "rootDevice": "hd0"
}
```
