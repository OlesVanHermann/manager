## 8. APIS & ENDPOINTS

### 8.1 Lister les nœuds

```
GET /nutanix/{serviceName}/nodes
```

Response:
```json
{
  "nodes": [
    {
      "nodeId": "node-nutanix-01",
      "displayName": "node-nutanix-01",
      "serverType": "scale-l",
      "status": "deployed",
      "datacenter": "rbx7",
      "osStatus": "ahv-installed",
      "nodeIp": "10.0.0.11",
      "cvmIp": "10.0.0.21"
    }
  ],
  "total": 4
}
```

### 8.2 Éteindre un nœud

```
POST /nutanix/{serviceName}/nodes/{nodeId}/powerOff
```

### 8.3 Résilier un nœud

```
DELETE /nutanix/{serviceName}/nodes/{nodeId}
```
