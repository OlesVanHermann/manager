## 7. COMPORTEMENT

| Action | Résultat |
|--------|----------|

| Clic hors menu | Fermer menu |

```
GET /nutanix/{serviceName}/nodes/{nodeId}
```

Response (nœud résilié):
```json
{
  "nodeId": "node-03",
  "displayName": "node-03",
  "serviceName": "ns12345.ip-51-123-45.eu",
  "status": "terminated",
  "terminationDate": "2025-12-15",
  "availableActions": ["contact_support"]
}
```
