## 7. VALIDATION

| Règle | Message erreur |
|-------|----------------|
| IP AHV obligatoire | L'adresse IP de l'hyperviseur est requise |
| IP CVM obligatoire | L'adresse IP de la CVM est requise |
| Format IP valide | L'adresse IP n'est pas valide |
| IP dans sous-réseau | L'adresse IP doit appartenir au sous-réseau du cluster |

## 8. APIS & ENDPOINTS

### 8.1 Obtenir versions AOS disponibles

```
GET /nutanix/{serviceName}/aosVersions
```

Response:
```json
{
  "versions": [
    {"version": "6.5.2.7", "type": "LTS", "recommended": true},
    {"version": "6.5.2.5", "type": "LTS", "recommended": false},
    {"version": "6.5.1.8", "type": "LTS", "recommended": false}
  ]
}
```

### 8.2 Installer un nœud

```
POST /nutanix/{serviceName}/nodes/{nodeId}/install
```

Body:
```json
{
  "ahvIp": "192.168.1.101",
  "cvmIp": "192.168.1.102",
  "aosVersion": "6.5.2.7"
}
```

Response:
```json
{
  "taskId": "task-install-123",
  "status": "pending",
  "estimatedDuration": "45 minutes"
}
```
