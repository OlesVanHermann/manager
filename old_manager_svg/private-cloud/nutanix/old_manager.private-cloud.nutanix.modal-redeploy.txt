## 6. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic Annuler | Fermer modal |
| Clic × | Fermer modal |
| Clic Continuer | Ouvrir modal de confirmation avec saisie |

**Note:** Cette modal est la première étape. Un clic sur "Continuer" devrait ouvrir une seconde modal demandant une saisie de confirmation (ex: "REDEPLOY").

---

## 7. APIS & ENDPOINTS

### 7.1 Redéployer un cluster

```
POST /nutanix/{serviceName}/redeploy
```

Body:
```json
{
  "confirmation": "REDEPLOY"
}
```

Response:
```json
{
  "taskId": "task-redeploy-456",
  "status": "pending",
  "estimatedDuration": "120 minutes"
}
```
