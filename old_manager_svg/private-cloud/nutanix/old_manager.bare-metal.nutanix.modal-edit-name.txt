# Cahier des Charges : bare-metal/nutanix - Modal Modifier Nom

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire simple
- **Contexte** : Gestion cluster Nutanix
- **Objectif** : Modifier le nom d'affichage du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champ nom
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le nom d'affichage" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. CONTENU

### Champ nom
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom d'affichage" |
| Type | OdsInput |
| Largeur | 400px |
| Exemple | Mon cluster Nutanix |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Enregistrer | Primary (#0050D7) | Enregistre le nouveau nom |

## 7. APIS & ENDPOINTS
- PUT /nutanix/{serviceName} - Modifier nom cluster
