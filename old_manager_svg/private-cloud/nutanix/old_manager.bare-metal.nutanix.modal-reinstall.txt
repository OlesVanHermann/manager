# Cahier des Charges : bare-metal/nutanix - Modal réinstaller le noeud

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec sélection
- **Contexte** : Dashboard Nutanix
- **Objectif** : Réinstaller AOS et AHV sur un noeud Nutanix

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description
- **OdsMessage** : Info API
- **OdsSelect** : Sélection version AOS
- **OdsButton** : Annuler, Ré-installer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ré-installer le noeud" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Vous allez ré-installer AOS et AHV sur votre noeud." |

## 6. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info (bleu) | "Pour une ré-installation simultanée sur plusieurs noeuds, utilisez l'API." |

## 7. SÉLECTION VERSION
| Propriété | Valeur |
|-----------|--------|
| Label | "Version AOS" |
| Type | OdsSelect |
| Exemple | "6.5.2.5" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ré-installer | Primary (bleu) | Lance la réinstallation |

## 9. APIS & ENDPOINTS
- GET /nutanix/{serviceName}/availableVersions - Versions disponibles
- POST /nutanix/{serviceName}/node/{nodeId}/reinstall - Réinstaller le noeud
