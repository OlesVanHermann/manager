# Cahier des Charges : bare-metal/nutanix - Liste des nœuds

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec datagrid
- **Contexte** : Gestion cluster Nutanix
- **Objectif** : Afficher la liste des nœuds d'un cluster Nutanix

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : Ajouter un nœud
- **OdsDatagrid** : Liste des nœuds
- **OdsBadge** : Statut nœud
- **OdsLink** : Nom du nœud (cliquable)
- **ActionMenu** : Menu contextuel (⋮)

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Nutanix - {clusterId}" |

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | Non sélectionné |
| Nœuds | Sélectionné (souligné bleu) |

## 6. BOUTON AJOUTER
| Propriété | Valeur |
|-----------|--------|
| Label | "+ Ajouter un nœud" |
| Style | Primary (#0050D7) |
| Largeur | 140px |
| Hauteur | 36px |

## 7. DATAGRID
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1040px |
| Hauteur | 300px |

### Colonnes
| Colonne | Largeur | Type |
|---------|---------|------|
| NOM | 150px | OdsLink (bleu #0050D7) |
| TYPE | 120px | Texte |
| STATUT | 100px | OdsBadge |
| LOCALISATION | 120px | Texte (datacenter) |
| OS | 120px | Texte (version AHV) |
| IP NŒUD (AHV) | 120px | Adresse IP |
| IP CVM | 120px | Adresse IP |
| ACTIONS | 50px | Menu ⋮ |

### Badges Statut
| État | Fond | Couleur texte |
|------|------|---------------|
| Déployé | #DCFCE7 | #16A34A |
| À déployer | #FEF3C7 | #92400E |
| En erreur | #FEE2E2 | #DC2626 |

### Types de nœuds
- HGR-HCI-2
- HGR-HCI-4
- HGR-HCI-8

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "{n} nœuds" |

## 9. APIS & ENDPOINTS
- GET /nutanix/{serviceName}/node - Liste des nœuds
- GET /nutanix/{serviceName}/node/{nodeId} - Détails nœud
- POST /nutanix/{serviceName}/node - Ajouter nœud
