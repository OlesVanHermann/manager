# Cahier des Charges : bare-metal/nutanix - Redéploiement

## 1. VUE D'ENSEMBLE
- **Page** : Redéployer le cluster
- **Type** : Modal confirmation warning
- **Objectif** : Réinitialiser le cluster Nutanix

## 2. STRUCTURE MODAL
- Bande orange en haut
- Header warning avec icône
- Liste des avertissements
- Options de redéploiement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal warning
- **OdsIcon** : Warning (cercle orange)
- **OdsText** : Avertissements
- **OdsRadio** : Options redéploiement
- **OdsButton** : Annuler, Redéployer

## 4. HEADER WARNING
| Propriété | Valeur |
|-----------|--------|
| Bande | Orange en haut |
| Icône | ⚠ dans cercle orange clair |
| Titre | Redéploiement du cluster |

## 5. MESSAGE PRINCIPAL
Vous conservez vos nœuds physiques et votre facturation reste inchangée.

## 6. AVERTISSEMENTS (liste)
- toutes les données présentes sur votre cluster seront effacées ;
- vos codes d'accès Prism Central seront regénérés ;
- vos configurations dans Prism Central seront effacées ;
- la configuration réseau sera réinitialisée aux valeurs par défaut (IP, vRack, etc.).

## 7. OPTIONS REDÉPLOIEMENT
| Option | Sélection | Description |
|--------|-----------|-------------|
| Configuration initiale | Sélectionné | Redéployer la configuration initiale à l'identique |
| Personnaliser | - | Personnaliser la configuration |

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Redéployer | Primary | Droite |

## 9. APIS & ENDPOINTS
- POST /nutanix/{serviceName}/cluster/redeploy
  - Paramètres : customConfig (boolean)
