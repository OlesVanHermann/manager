## 6. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic - | Décrémenter (min: 1) |
| Clic + | Incrémenter (max: 4, car 8-4=4) |
| Clic Annuler | Fermer modal |
| Clic × | Fermer modal |
| Clic Ajouter | Soumettre commande ajout nœuds |

---

## 7. APIS & ENDPOINTS

### 7.1 Obtenir informations cluster

```
GET /nutanix/{serviceName}
```

Response:
```json
{
  "serviceName": "nutanix-abc123",
  "displayName": "Cluster Nutanix Production",
  "currentNodes": 4,
  "maxNodes": 8,
  "pricePerNode": 500.00
}
```

### 7.2 Commander nœuds supplémentaires

```
POST /order/cart/{cartId}/nutanix/{serviceName}/nodes
```

Body:
```json
{
  "quantity": 1
}
```
