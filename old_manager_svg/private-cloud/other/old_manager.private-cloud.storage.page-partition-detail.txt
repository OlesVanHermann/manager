# Cahier des Charges : private/storage - Page Partition Detail

## 1. VUE D'ENSEMBLE

**Type:** Page détail avec onglets
**Fonction:** Vue complète d'une partition NAS-HA
**Contexte:** Gestion d'une partition spécifique

## 2. DIMENSIONS

- Page: 1440x900px
- Card Information: 400x400px
- Card Actions rapides: 400x200px
- Card Options ZFS: 400x180px

## 3. LAYOUT & STRUCTURE

### Header OVHcloud
- Background: #000E9C

## 8. APIS & ENDPOINTS

```
GET /dedicated/nasha/{serviceName}/partition/{partitionName}
→ Informations partition

PUT /dedicated/nasha/{serviceName}/partition/{partitionName}
Body: { partitionDescription: "..." }
→ Modifie la description

PUT /dedicated/nasha/{serviceName}/partition/{partitionName}
Body: { size: 1000 }
→ Modifie la taille

POST /dedicated/nasha/{serviceName}/partition/{partitionName}/customSnapshot
→ Crée un snapshot

GET /dedicated/nasha/{serviceName}/partition/{partitionName}/options
→ Options ZFS actuelles
```
