# Cahier des Charges : private/storage - Page Volumes

## 1. VUE D'ENSEMBLE
- **Type** : Page listing volumes de stockage
- **Contexte** : Private Cloud Storage
- **Objectif** : Gérer les volumes de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer volume
- **OdsDatagrid** : Tableau volumes
- **OdsBadge** : Type, état
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Volumes de stockage" |
| Description | "Gérez les volumes de stockage de votre cloud privé." |

## 5. BOUTON CRÉATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Créer volume" |
| Style | Fond #0050D7, texte blanc |

## 6. COLONNES TABLEAU

## 12. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume - Liste volumes
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume - Créer volume
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume/{volumeId} - Modifier
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume/{volumeId}/attach - Attacher
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume/{volumeId}/detach - Détacher
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume/{volumeId} - Supprimer
