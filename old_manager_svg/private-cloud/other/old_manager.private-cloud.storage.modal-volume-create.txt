# Cahier des Charges : private/storage - Modal Volume Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création volume
- **Contexte** : Private Cloud Storage
- **Objectif** : Créer un nouveau volume de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Overlay | 550x420px |
| Modal | 450x360px |
| Background modal | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres et labels
- **OdsInput** : Nom, taille
- **OdsSelect** : Type stockage, VM
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un volume" |
| Fond header | #F9FAFB |
| Bouton fermer | × en #6B7280 |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nom du volume | Input texte | Oui (*) | "vol-" |
| Taille | Input numérique + Select unité | Oui (*) | "100" + "Go" |
| Type de stockage | Select | Non | - |
| Attacher à | Select | Non | "Sélectionner une VM..." |

## 6. OPTIONS TYPE STOCKAGE
| Type | Description |
|------|-------------|
| SSD NVMe | "SSD NVMe (10 000 IOPS)" |
| SSD Standard | "SSD Standard (5 000 IOPS)" |
| HDD | "HDD (1 000 IOPS)" |

## 7. OPTIONS UNITÉ TAILLE
| Unité | Valeur |
|-------|--------|
| Go | Gigaoctets |
| To | Téraoctets |

## 8. STYLE INPUTS
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc |
| Bordure | #E5E7EB |
| Border radius | 4px |
| Hauteur | 40px |
| Input taille largeur | 150px |
| Select unité largeur | 80px |

## 9. BOUTONS FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline #E5E7EB, texte #6B7280 | Gauche |
| Créer | Fond #0050D7, texte blanc | Droite |

## 10. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume - Créer volume
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm - Liste VMs pour attachement
