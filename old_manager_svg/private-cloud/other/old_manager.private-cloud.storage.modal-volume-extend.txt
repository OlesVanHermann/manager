# Cahier des Charges : private/storage - Modal Volume Extend

## 1. VUE D'ENSEMBLE
- **Type** : Modal extension volume
- **Contexte** : Private Cloud Storage
- **Objectif** : Augmenter la taille d'un volume de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Overlay | 500x350px |
| Modal | 400x290px |
| Background modal | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres et labels
- **OdsInput** : Nouvelle taille
- **OdsMessage** : Info (pas d'arrêt requis)
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Étendre le volume" |
| Fond header | #F9FAFB |
| Bouton fermer | × en #6B7280 |

## 5. INFO VOLUME
| Propriété | Valeur |
|-----------|--------|
| Label | "Volume" |
| Style label | 11px #6B7280 |
| Valeur | "{nom} ({taille} actuellement)" |
| Style valeur | 13px #1F2937 bold |
| Fond | #F9FAFB, bordure #E5E7EB |

## 6. CHAMP NOUVELLE TAILLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Nouvelle taille *" |
| Type | Input numérique |
| Unité | "Go" affiché à droite |
| Largeur input | 200px |

## 7. CONTRAINTES TAILLE
| Propriété | Valeur |
|-----------|--------|
| Minimum | Taille actuelle + 1 Go |
| Maximum | "16 To" |
| Texte aide | "Minimum: XXX Go • Maximum: 16 To" |
| Style aide | 11px #6B7280 |

| Bouton | Style | Position |
|--------|-------|----------|

## 10. APIS & ENDPOINTS
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/volume/{volumeId}/extend - Étendre volume
