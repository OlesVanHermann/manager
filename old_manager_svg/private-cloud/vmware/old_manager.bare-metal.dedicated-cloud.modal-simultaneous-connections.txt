# Cahier des Charges : bare-metal/dedicated-cloud - Modal Simultaneous Connections

## 1. VUE D'ENSEMBLE

**Type:** Wizard modal (2 étapes)
**Fonction:** Configurer le nombre maximum de connexions simultanées
**Contexte:** Sécurité VMware Private Cloud

## 2. DIMENSIONS

- Container: 570x290px
- Background: #F9FAFB
- Border-radius: 8px

## 3. STEPPER

| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Active (#0050D7) |
| 2 | Confirmation | Inactive (#E5E7EB) |

## 4. ÉTAPE 1 - CONFIGURATION

### Header
- Titre: "Modifier les connexions simultanées" (18px, font-weight: 600)

### Champs
| Champ | Type | Dimensions |
|-------|------|------------|
| Nombre maximum de connexions simultanées | Input number | 200x40px |

### Valeur par défaut
- Connexions: 10

### Helper text
- "Entrez 0 pour désactiver la limite." (text-muted, 12px)

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (border) | Fermer modal |
| Suivant | Primary (#0050D7) | Étape 2 |

## 6. APIS & ENDPOINTS
