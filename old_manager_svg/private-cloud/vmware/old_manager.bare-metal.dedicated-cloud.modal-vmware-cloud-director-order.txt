# Cahier des Charges : bare-metal/dedicated-cloud - Modal VMware Cloud Director Order

## 1. VUE D'ENSEMBLE

**Type:** Modal de commande/activation
**Fonction:** Commander et activer VMware Cloud Director sur l'infrastructure
**Contexte:** Accessible depuis le dashboard ou les options VMware

## 2. DIMENSIONS

- Modal: 700x540px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal (Info)
- Background: #EFF6FF
- Icône: Cercle bleu #1E40AF avec "i"
- Titre: "Commander VMware Cloud Director" (color: #1E40AF)
- Close button: × (color: #1E40AF)

### Bloc info
- Background: #EFF6FF
- Border: 1px solid #BFDBFE
- Texte: "Cette option permet d'activer VMware Cloud Director sur votre infrastructure Hosted Private Cloud."

### Liste des hôtes
- Titre: "Liste des hôtes pour pcc-192-0-2-1 :"
- Items (liste à puces):
  - Host 1 : Standard-L (64 CPU, 256 Go RAM)
  - Host 2 : Standard-L (64 CPU, 256 Go RAM)
  - Host 3 : Standard-XL (128 CPU, 512 Go RAM)

### Lien CGU

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/host
→ Liste les hôtes de l'infrastructure

GET /dedicatedCloud/{serviceName}/vcd/eligibility
→ Vérifie l'éligibilité à VCD

POST /dedicatedCloud/{serviceName}/vcd/order
→ Passe la commande VCD

GET /dedicatedCloud/{serviceName}/vcd/contract
→ Récupère l'URL des CGU
```
