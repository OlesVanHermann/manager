- **Type** : Page configuration simple
- **Contexte** : Dedicated Cloud
- **Objectif** : Gérer l'activation SPLA (licences Microsoft)

| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Carte largeur | 900px |
