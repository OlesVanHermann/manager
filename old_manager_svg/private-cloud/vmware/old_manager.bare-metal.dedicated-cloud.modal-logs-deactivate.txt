# Cahier des Charges : bare-metal/dedicated-cloud - Modal Logs Deactivate

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation warning
- **Contexte** : Gestion Dedicated Cloud
- **Objectif** : Désactiver la collecte des logs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message avertissement
- **OdsButton** : Fermer, Désactiver

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Attention, vous allez désactiver la collecte des logs" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |

## 5. CONTENU
| Ligne | Texte |
|-------|-------|
| 1 | "La désactivation de ce service entrainera l'arrêt de la collecte" |
| 2 | "et de la visualisation des logs." |
| 3 | "Voulez-vous continuer ?" (gras) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary | Ferme modal |
| Désactiver | Danger #DC2626 | Désactive logs |

## 7. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/log
