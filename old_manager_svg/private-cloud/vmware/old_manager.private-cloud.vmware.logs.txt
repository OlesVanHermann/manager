## 10. APIS & ENDPOINTS

### 10.1 Obtenir les logs

```
GET /dedicatedCloud/{serviceName}/logs
```

Query params:
- `type` (optionnel): login, vm, security, host, network
- `startDate` (optionnel): ISO date
- `endDate` (optionnel): ISO date
- `page`: numéro de page
- `pageSize`: nombre de résultats par page

Response:
```json
{
  "data": [
    {
      "id": "log-001",
      "timestamp": "2025-12-28T14:32:15Z",
      "type": "login",
      "user": "admin@pcc-192-0-2-1",
      "description": "Connexion réussie depuis 192.0.2.50",
      "ipAddress": "192.0.2.50"
    }
  ],
  "pagination": {
    "currentPage": 1,
    "totalPages": 15,
    "totalResults": 148
  }
}
```
