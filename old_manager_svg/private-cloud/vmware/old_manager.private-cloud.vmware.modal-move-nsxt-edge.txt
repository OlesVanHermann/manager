## 6. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic Annuler | Fermer modal sans action |
| Clic × | Fermer modal |
| Clic dropdown NSX Edge | Afficher liste des NSX Edges |
| Clic dropdown Datastore | Afficher liste des datastores disponibles |
| Sélection datastore | Activer le bouton Relocaliser |
| Clic Relocaliser | Lancer la relocalisation |

## 8. APIS & ENDPOINTS

### 8.1 Lister les NSX Edges

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsxt/edges
```

Response:
```json
[
  {
    "id": "edge-001",
    "name": "edge-gateway-01",
    "currentDatastore": "ds-001",
    "status": "running"
  },
  {
    "id": "edge-002",
    "name": "edge-gateway-02",
    "currentDatastore": "ds-002",
    "status": "running"
  }
]
```

### 8.2 Lister les datastores disponibles

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/datastores
```

Response:
```json
[
  {
    "id": "ds-001",
    "name": "datastore-ssd-01",
    "freeSpace": "500GB",
    "type": "ssd"
  },
  {
    "id": "ds-002",
    "name": "datastore-ssd-02",
    "freeSpace": "1TB",
    "type": "ssd"
  }
]
```

### 8.3 Relocaliser un NSX Edge

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsxt/edges/{edgeId}/move
```

Body:
```json
{
  "targetDatastoreId": "ds-002"
}
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "pending",
  "estimatedDuration": "5 minutes"
}
```
