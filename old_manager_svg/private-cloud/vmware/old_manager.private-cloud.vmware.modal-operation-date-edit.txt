## 6. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic jour | Sélectionner cette date |
| Clic ‹ | Afficher mois précédent |
| Clic › | Afficher mois suivant |
| Modification heure | Mettre à jour l'heure de traitement |
| Clic Annuler | Fermer modal sans sauvegarder |
| Clic × | Fermer modal |
| Clic Modifier | Sauvegarder nouvelle date + fermer modal |

---

## 7. APIS & ENDPOINTS

### 7.1 Modifier la date d'une opération

```
PUT /vmware/{serviceName}/operations/{operationId}/schedule
```

Body:
```json
{
  "scheduledDate": "2026-01-02T14:30:00Z"
}
```

Response:
```json
{
  "operationId": "op-123",
  "scheduledDate": "2026-01-02T14:30:00Z",
  "status": "scheduled"
}
```
