# Cahier des Charges : bare-metal/dedicated-cloud - Zerto Alerts

## 1. VUE D'ENSEMBLE

**Type:** Composant alertes empilées
**Fonction:** Afficher les statuts d'activation Zerto
**Contexte:** Feedback utilisateur post-installation DRP

## 2. LAYOUT

- Conteneur: 900x200px
- Background: #FFFFFF
- Spacing entre alertes: 10px
