## 7. APIS & ENDPOINTS

### 7.1 Obtenir détails Private Cloud

```
GET /dedicatedCloud/{serviceName}
```

Response:
```json
{
  "serviceName": "pcc-192-0-2-1",
  "solution": "vSphere",
  "commercialRange": "Premier",
  "location": "Gravelines",
  "accessPolicy": "open",
  "options": {
    "zerto": true,
    "veeam": true,
    "nsxt": true,
    "logs": true
  },
  "securityOptions": {
    "pciDss": false,
    "hds": false,
    "hipaa": false
  }
}
```

### 7.2 Obtenir informations abonnement

```
GET /dedicatedCloud/{serviceName}/serviceInfos
```

Response:
```json
{
  "mailingList": "admin@example.com",
  "expirationDate": "2026-12-31",
  "renewalType": "automaticRenewal"
}
```
