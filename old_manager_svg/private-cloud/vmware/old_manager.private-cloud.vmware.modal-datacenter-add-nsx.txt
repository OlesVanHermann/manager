## 8. APIS & ENDPOINTS

### 8.1 Commander des nœuds NSX Edge

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx/edges
```

Body:
```json
{
  "size": "medium",
  "quantity": 2
}
```

Response:
```json
{
  "orderId": "order-456",
  "price": {
    "value": 0.16,
    "currency": "EUR",
    "text": "0,16€ HT/mois"
  }
}
```

### 8.2 Obtenir les tailles disponibles

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx/edges/sizes
```

Response:
```json
[
  {
    "size": "medium",
    "vcpu": 4,
    "ram": "8Go",
    "storage": "200Go",
    "pricePerVcpu": 0.02,
    "description": "Convient pour les fonctionnalités L2-L4"
  },
  {
    "size": "large",
    "vcpu": 8,
    "ram": "16Go",
    "storage": "400Go",
    "pricePerVcpu": 0.02,
    "description": "Pour charges plus importantes"
  }
]
```
