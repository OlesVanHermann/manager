# Cahier des Charges : private/cloud - Modal VM Delete

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation suppression VM
- **Contexte** : Private Cloud
- **Objectif** : Confirmer la suppression d'une machine virtuelle

|-----------|--------|
| Overlay | 500x320px |
