---

## 7. APIS & ENDPOINTS

### 7.1 Obtenir les gammes disponibles

```
GET /dedicatedCloud/{serviceName}/commercialRanges
```

Response:
```json
[
  {
    "name": "dedicatedCloud",
    "label": "Dedicated Cloud",
    "description": "Gamme premium avec ressources dédiées",
    "startingPrice": { "value": 500, "text": "A partir de 500.00 EUR HT/mois" }
  }
]
```

### 7.2 Ajouter un datacenter

```
POST /dedicatedCloud/{serviceName}/datacenter
```

Body:
```json
{
  "commercialRange": "dedicatedCloud"
}
```
