## 7. APIS & ENDPOINTS

### 7.1 Désactiver le backup Veeam

```
DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup
```

Response:
```json
{
  "taskId": "task-456",
  "status": "todo",
  "message": "Disabling Veeam backup"
}
```
