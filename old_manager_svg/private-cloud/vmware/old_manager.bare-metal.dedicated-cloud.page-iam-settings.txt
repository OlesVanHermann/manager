# Cahier des Charges : bare-metal/dedicated-cloud - Page IAM Settings

## 1. VUE D'ENSEMBLE

**Type:** Page paramètres avec tabs
**Fonction:** Configuration IAM OVHcloud pour le Private Cloud
**Contexte:** Gestion accès via IAM OVHcloud

## 2. LAYOUT

- Page: 1200x450px
- Background: #F9FAFB

## 3. TABS

| Tab | État |
|-----|------|
| Utilisateurs vSphere | Inactif (muted) |
| IAM OVHcloud | Actif (primary + underline) |

## 4. CONTENU

### Description
- "L'activation de l'IAM OVHcloud délègue la gestion des accès au service IAM OVHcloud."
- "La gestion des rôles associés et leurs autorisations dans vSphere s'effectue depuis cette page."
- "La gestion des politiques et accès s'effectue depuis l'IAM OVHcloud."

### Warning Banner
- Background: #FEF3C7
- Border: #FDE68A
- Message: "Les infrastructures bénéficiant d'options de sécurité renforcée ou d'un service certifié (HDS, PCI-DSS ou SNC) ne peuvent actuellement pas utiliser l'IAM OVHcloud."

### Statut
- Label: "Statut"
- Badge: "Activé" (vert, #DCFCE7/#16A34A)

### Boutons actions (colonne droite)
| Bouton | Style |
|--------|-------|
| Désactiver l'IAM OVHcloud | Secondary (border) |
| Gérer les politiques et accès ↗ | Secondary + primary text |

## 5. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/iam/status
POST /dedicatedCloud/{serviceName}/iam/disable
GET /dedicatedCloud/{serviceName}/iam/compatible
```
