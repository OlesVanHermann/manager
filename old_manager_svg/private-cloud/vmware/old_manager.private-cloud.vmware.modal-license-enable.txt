## 9. APIS & ENDPOINTS

### 9.1 Obtenir les licences disponibles

```
GET /dedicatedCloud/{serviceName}/spla/availableLicenses
```

Response:
```json
[
  {
    "id": "win-2022-std",
    "name": "Windows Server 2022 Standard",
    "pricePerCore": { "value": 25, "text": "25,00 € HT/mois" }
  },
  {
    "id": "win-2022-dc",
    "name": "Windows Server 2022 Datacenter",
    "pricePerCore": { "value": 85, "text": "85,00 € HT/mois" }
  }
]
```

### 9.2 Activer une licence SPLA

```
POST /dedicatedCloud/{serviceName}/spla/enable
```

Body:
```json
{
  "licenseType": "win-2022-std",
  "acceptedTerms": true
}
```
