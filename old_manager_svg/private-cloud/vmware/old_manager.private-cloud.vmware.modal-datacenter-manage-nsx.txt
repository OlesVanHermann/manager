## 10. APIS & ENDPOINTS

### 10.1 Obtenir la configuration NSX Edge actuelle

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx/edges
```

Response:
```json
{
  "currentSize": "medium",
  "availableSizes": ["medium", "large", "extraLarge"],
  "pricing": {
    "perVcpu": 0.02,
    "currency": "EUR"
  }
}
```

### 10.2 Redimensionner les NSX Edge

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx/edges/resize
```

Body:
```json
{
  "targetSize": "large"
}
```
