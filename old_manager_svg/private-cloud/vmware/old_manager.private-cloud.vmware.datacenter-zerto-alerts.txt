## 7. APIS & ENDPOINTS

### 7.1 Obtenir statut Zerto

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/zerto/status
```

Response:
```json
{
  "enabled": true,
  "operational": true,
  "drpType": "ovhcloud_to_ovhcloud",
  "primarySite": "dc-rbx-01",
  "secondarySite": "dc-sbg-02",
  "vpnConfigured": false,
  "notifications": {
    "email": "admin@example.com",
    "confirmationSent": true
  }
}
```
