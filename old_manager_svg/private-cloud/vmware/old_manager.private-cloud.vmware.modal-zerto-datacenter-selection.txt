| Champ | Règles |

## 8. APIS & ENDPOINTS

### 8.1 Lister les datacenters éligibles

```
GET /dedicatedCloud/{serviceName}/datacenters/zerto/eligible
```

Response:
```json
[
  {
    "id": "dc-001",
    "name": "dc-rbx-01",
    "location": "Roubaix, France",
    "availableHosts": 4,
    "zertoStatus": "notConfigured"
  },
  {
    "id": "dc-002",
    "name": "dc-gra-01",
    "location": "Gravelines, France",
    "availableHosts": 2,
    "zertoStatus": "active"
  }
]
```

### 8.2 Obtenir les détails d'un datacenter

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}
```

Response:
```json
{
  "id": "dc-001",
  "name": "dc-rbx-01",
  "location": "Roubaix, France",
  "hosts": {
    "total": 4,
    "active": 4
  },
  "zerto": {
    "status": "notConfigured",
    "eligible": true
  }
}
```
