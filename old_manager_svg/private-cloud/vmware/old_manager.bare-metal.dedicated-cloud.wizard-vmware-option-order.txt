# Cahier des Charges : bare-metal/dedicated-cloud - Wizard VMware Option Order

## 1. VUE D'ENSEMBLE

**Type:** Wizard single-step
**Fonction:** Commander/Activer une option VMware (ex: vRealize Operations)
**Contexte:** Activation d'options VMware sur le Dedicated Cloud

- Container principal: 1440x750px
- Zone wizard: 1100x600px (centrée)

## 6. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x40px)
- Position: gauche
- Action: Retour à la page options

### Bouton Activer
- Style: btn-primary (120x40px)
- Position: droite
- Action: Déclenche l'activation de l'option

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/vmwareOption/{optionId}
→ Récupère les détails de l'option

GET /dedicatedCloud/{serviceName}/vmwareOption/{optionId}/priceImpact
→ Calcule l'impact sur les prix par host

POST /dedicatedCloud/{serviceName}/vmwareOption/{optionId}/enable
→ Active l'option VMware
```
