# Cahier des Charges : private/cloud - Page Maintenance

## 1. VUE D'ENSEMBLE
- **Type** : Page maintenance planifiée
- **Contexte** : Hosted Private Cloud (VMware)
- **Objectif** : Consulter les maintenances

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsMessage** : Bannière alerte
- **OdsCard** : Historique maintenances
- **OdsBadge** : Type, Statut
- **OdsButton** : Voir détails

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Maintenance" |
| Description | "Planification et historique des maintenances." |

## 5. BANNIÈRE ALERTE
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Icône | ⚠ fond orange |
| Titre | "Maintenance planifiée" |

## 6. CONTENU BANNIÈRE
| Ligne | Contenu |
|-------|---------|
| 1 | Date et heure maintenance |
| 2 | Impact (ex: "Redémarrage des hôtes ESXi") |
| Bouton | "Voir détails" (outline) |

## 7. COLONNES HISTORIQUE
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Date | 180px | DD/MM/YYYY |
| Type | 200px | Badge type |
| Description | 350px | Texte description |
| Durée | 130px | Xh ou Xmin |
| Statut | 150px | Badge statut |

## 8. BADGES TYPE
| Type | Fond | Texte |

## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/maintenance - Liste maintenances
- GET /dedicatedCloud/{serviceName}/maintenance/{id} - Détail maintenance
