## 7. APIS & ENDPOINTS

### 7.1 Obtenir les licences disponibles

```
GET /dedicatedCloud/{serviceName}/spla/licenses
```

Response:
```json
[
  {
    "id": "win-2019-std",
    "name": "Windows Server 2019 Standard"
  },
  {
    "id": "win-2022-dc",
    "name": "Windows Server 2022 Datacenter"
  },
  {
    "id": "sql-2019-std",
    "name": "SQL Server 2019 Standard"
  }
]
```

### 7.2 Attribuer une licence à une VM

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId}/license
```

Body:
```json
{
  "licenseId": "win-2022-dc"
}
```
