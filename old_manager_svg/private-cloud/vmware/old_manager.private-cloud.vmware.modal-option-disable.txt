## 10. APIS & ENDPOINTS

### 10.1 Obtenir les informations d'impact

```
GET /dedicatedCloud/{serviceName}/option/{optionName}/disableImpact
```

Response:
```json
{
  "option": {
    "name": "vROps",
    "label": "VMware vRealize Operations",
    "description": "Surveillance et optimisation des performances"
  },
  "billingImpact": [
    {
      "resource": "vROps - Licence par VM",
      "currentPrice": "50,00 €/mois",
      "newPrice": "0,00 €/mois"
    }
  ],
  "totalCurrent": "50,00 €/mois",
  "totalNew": "0,00 €/mois"
}
```

### 10.2 Désactiver l'option

```
POST /dedicatedCloud/{serviceName}/option/{optionName}/disable
```
