## 7. APIS & ENDPOINTS

### 7.1 Activer un utilisateur

```
POST /dedicatedCloud/{serviceName}/user/{userId}/enable
```

Response:
```json
{
  "taskId": "task-456",
  "status": "todo",
  "message": "Enabling user"
}
```
