## 11. APIS & ENDPOINTS

### 11.1 Obtenir droits utilisateur par datacenter

```
GET /dedicatedCloud/{serviceName}/user/{userId}/right/{datacenterId}
```

Response:
```json
{
  "vsphereAccess": "readWrite",
  "vmNetworkAccess": "operator",
  "vxlanAccess": "readOnly",
  "hostStorageManagement": true
}
```

### 11.2 Modifier les droits

```
PUT /dedicatedCloud/{serviceName}/user/{userId}/right/{datacenterId}
```

Body:
```json
{
  "vsphereAccess": "readWrite",
  "vmNetworkAccess": "operator",
  "vxlanAccess": "readOnly",
  "hostStorageManagement": true
}
```
