## 6. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic Annuler | Fermer modal sans action |
| Clic × | Fermer modal |
| Clic checkbox | Toggle état + activer/désactiver bouton Confirmer |
| Clic lien contrat | Ouvrir contrat (nouvel onglet) |
| Clic Confirmer | Activer les licences SPLA (si checkbox coché) |

---

## 7. APIS & ENDPOINTS

### 7.1 Obtenir les contrats à accepter

```
GET /dedicatedCloud/{serviceName}/spla/contracts
```

Response:
```json
[
  {
    "name": "Conditions Générales SPLA Microsoft",
    "url": "https://contracts.ovh.com/spla-microsoft"
  },
  {
    "name": "Conditions Particulières Licences OVHcloud",
    "url": "https://contracts.ovh.com/licenses"
  },
  {
    "name": "Annexe Tarification SPLA",
    "url": "https://contracts.ovh.com/spla-pricing"
  }
]
```

### 7.2 Activer les licences SPLA

```
POST /dedicatedCloud/{serviceName}/spla/enable
```

Body:
```json
{
  "contractsAccepted": true
}
```
