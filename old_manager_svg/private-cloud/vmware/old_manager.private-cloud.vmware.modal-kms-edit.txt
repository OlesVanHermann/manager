## 8. APIS & ENDPOINTS

### 8.1 Modifier le serveur KMS

```
PUT /dedicatedCloud/{serviceName}/kms/{kmsId}
```

Body:
```json
{
  "description": "Serveur KMS production",
  "sslThumbprint": "AB:CD:EF:12:34:56:78:90:AB:CD:EF:12:34:56:78:90"
}
```

Response:
```json
{
  "taskId": "task-456",
  "status": "todo",
  "message": "KMS update requested"
}
```
