# Cahier des Charges : bare-metal/dedicated-cloud - Page User Management

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion utilisateurs
- **Contexte** : Dedicated Cloud (VMware)
- **Objectif** : Gérer les accès utilisateurs à l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajouter utilisateur
- **OdsDatagrid** : Tableau utilisateurs
- **OdsBadge** : Rôle et accès NSX
- **OdsSwitch** : Toggle activation
- **OdsPopover** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Gestion des utilisateurs" |
| Description | "Gérez les accès à votre infrastructure VMware." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter un utilisateur" |
| Style | Fond #0050D7, texte blanc |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Nom d'utilisateur | 200px | Username |
| Email | 230px | Adresse email |
| Rôle | 150px | Badge rôle |
| NSX | 100px | Badge Oui/Non |
| Activé | 100px | Toggle switch |
| Actions | 100px | Menu ⋮ |

## 7. BADGES RÔLE
| Rôle | Fond | Texte |
|------|------|-------|
| Administrator | #F3E8FF | #7C3AED |
| Read-only | #EFF6FF | #0050D7 |

## 8. BADGES NSX
| État | Fond | Texte |
|------|------|-------|
| Oui | #DCFCE7 | #16A34A |
| Non | #FEE2E2 | #DC2626 |

## 9. TOGGLE SWITCH
| État | Style |
|------|-------|
| ON | Fond #0050D7, cercle droite |
| OFF | Fond #E5E7EB, cercle gauche |

## 10. MENU ACTIONS
| Action | Style |
|--------|-------|
| Modifier les droits | Texte #1F2937 |
| Réinitialiser mot de passe | Texte #1F2937 |
| Activer accès NSX | Texte #1F2937 |
| Supprimer | Texte #DC2626 |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/user - Liste utilisateurs
- POST /dedicatedCloud/{serviceName}/user - Créer utilisateur
- PUT /dedicatedCloud/{serviceName}/user/{userId} - Modifier
- DELETE /dedicatedCloud/{serviceName}/user/{userId} - Supprimer
- POST /dedicatedCloud/{serviceName}/user/{userId}/changePassword - Reset password
