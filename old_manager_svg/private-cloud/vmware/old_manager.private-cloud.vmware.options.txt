## 8. APIS & ENDPOINTS

### 8.1 Obtenir les options du service

```
GET /dedicatedCloud/{serviceName}/options
```

Response:
```json
{
  "compliance": {
    "current": null,
    "available": ["HDS", "PCI-DSS", "SNC"]
  },
  "options": {
    "zerto": {
      "enabled": true,
      "status": "active"
    },
    "veeam": {
      "enabled": true,
      "status": "active"
    },
    "nsxt": {
      "enabled": true,
      "datacenters": [
        { "name": "dc-gra1", "status": "active" },
        { "name": "dc-sbg3", "status": "active" }
      ]
    },
    "logs": {
      "enabled": true,
      "status": "active"
    }
  }
}
```

### 8.2 Désactiver une option

```
POST /dedicatedCloud/{serviceName}/options/{optionName}/disable
```
