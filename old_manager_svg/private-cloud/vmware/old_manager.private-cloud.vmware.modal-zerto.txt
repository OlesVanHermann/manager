## 10. APIS & ENDPOINTS

### 10.1 Lister les sites disponibles pour réplication

```
GET /dedicatedCloud/{serviceName}/zerto/availableTargets
```

Response:
```json
[
  {
    "serviceName": "pcc-192-0-2-2",
    "datacenter": "dc-sbg1",
    "location": "Strasbourg"
  },
  {
    "serviceName": "pcc-192-0-2-3",
    "datacenter": "dc-rbx2",
    "location": "Roubaix"
  }
]
```

### 10.2 Activer Zerto

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/zerto/enable
```

Body:
```json
{
  "replicationType": "ovhcloud_to_ovhcloud",
  "targetServiceName": "pcc-192-0-2-2",
  "targetDatacenter": "dc-sbg1"
}
```
