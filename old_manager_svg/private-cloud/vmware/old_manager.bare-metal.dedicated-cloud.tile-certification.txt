# Cahier des Charges : bare-metal/dedicated-cloud - Tile Certification

## 1. VUE D'ENSEMBLE
- **Type** : Tuile options
- **Contexte** : Dashboard Dedicated Cloud
- **Objectif** : Afficher l'état des certifications (HDS, PCI-DSS)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsBadge** : États activation
- **OdsChip** : Badges datacenter
- **OdsLink** : Lien interface gestion

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Certifications" |
