# Cahier des Charges : bare-metal/dedicated-cloud - Banner VMware Cloud Director Migration

## 1. VUE D'ENSEMBLE

**Type:** Banner (Bannière d'information)
**Fonction:** Informer l'utilisateur de l'éligibilité à la migration VMware Cloud Director
**Contexte:** Affiché sur le dashboard Dedicated Cloud pour les infrastructures éligibles

## 2. DIMENSIONS

- Largeur: 1360px (pleine largeur avec marges 40px)
- Hauteur: 140px
- Border-radius: 8px
- Position: Haut de page, avant le contenu principal

## 3. COMPOSANTS UI

### Container
- Background: #EFF6FF (bg-info bleu clair)
- Border: 1px solid #BFDBFE
- Border-radius: 8px

### Icône
- Cercle: 50px diamètre
- Background: #1E40AF (bleu foncé)
- Contenu: Point d'exclamation "!" blanc

### Contenu texte
- Titre: "Migration vers VMware Cloud Director disponible"
  - Font-size: 16px
  - Font-weight: 600
  - Color: #1E40AF
- Description: 2 lignes
  - "Votre infrastructure est éligible à la migration vers VMware Cloud Director."
  - "Profitez des fonctionnalités avancées de gestion multi-tenant et d'orchestration cloud."
  - Font-size: 14px
  - Color: #1E40AF

## 4. ACTIONS & INTERACTIONS

### Bouton principal
- Label: "En savoir plus"
- Style: btn-primary (#0050D7)
- Dimensions: 180x36px
- Action: Ouvre documentation/guide migration

### Lien secondaire
- Label: "Planifier ma migration →"
- Style: Lien souligné text-primary
- Action: Redirige vers wizard de migration

### Bouton fermeture
- Icône: × (croix)
- Position: Coin supérieur droit
- Color: #1E40AF
- Action: Masque la bannière (stocke préférence utilisateur)

## 5. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/migration/eligibility
→ Vérifie éligibilité à la migration VCD

POST /dedicatedCloud/{serviceName}/banner/dismiss
→ Enregistre le masquage de la bannière
```

## 6. LOGIQUE D'AFFICHAGE

- Afficher si: infrastructure éligible ET bannière non masquée
- Masquer si: utilisateur a cliqué sur ×
- Persistance: Cookie ou préférence utilisateur
