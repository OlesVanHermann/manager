# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Ajouter Utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (3 étapes)
- **Contexte** : Gestion des utilisateurs vSphere
- **Objectif** : Créer un nouvel utilisateur avec mot de passe et droits

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 900px |
| Padding | 60px |

## 3. COMPOSANTS UI
- **OdsStepper** : Navigation 3 étapes
- **OdsInput** : Champs du formulaire
- **OdsButton** : Navigation

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un utilisateur" |

## 5. STEPPER
| Étape | Label | État exemple |
|-------|-------|--------------|
| 1 | Informations | Actif (bleu #0050D7) |
| 2 | Droits | À venir (gris) |
| 3 | Confirmation | À venir (gris) |

## 6. ÉTAPE 1 : INFORMATIONS

### Formulaire
| Champ | Type | Obligatoire | Validation |
|-------|------|-------------|------------|
| Nom d'utilisateur | OdsInput | Oui | 2-11 caractères alphanumériques |
| Mot de passe | OdsInput (password) | Oui | Voir règles ci-dessous |
| Email | OdsInput (email) | Non | Format email valide |

### Règles mot de passe
| Règle |
|-------|
| Au moins un chiffre |
| Au moins une lettre |
| Au moins une majuscule |
| Au moins une minuscule |
| Au moins un caractère spécial |
| 8 à 30 caractères |

### Style champ focus
| Propriété | Valeur |
|-----------|--------|
| Bordure | Bleue (#0050D7) 2px |

### Note
| Propriété | Valeur |
|-----------|--------|
| Texte | "* Champ obligatoire" |
| Couleur | Gris (#6B7280) |

## 7. NAVIGATION
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary | Ferme wizard |
| Suivant | Droite | Primary (bleu #0050D7) | Étape suivante |

## 8. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/user - Créer utilisateur
