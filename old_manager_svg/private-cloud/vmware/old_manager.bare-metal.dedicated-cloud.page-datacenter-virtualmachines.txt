# Cahier des Charges : bare-metal/dedicated-cloud - Page Datacenter Virtual Machines

**Type:** Page listing avec datagrid
**Fonction:** Lister et gérer les VMs d'un datacenter
**Contexte:** Gestion licences et sauvegarde VMs
