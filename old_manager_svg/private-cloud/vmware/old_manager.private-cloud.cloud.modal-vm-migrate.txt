# Cahier des Charges : private/cloud - Modal VM Migrate

## 1. VUE D'ENSEMBLE
- **Type** : Modal migration VM
- **Contexte** : Private Cloud
- **Objectif** : Migrer une VM vers un autre hôte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Titres et labels
- **OdsCard** : Info VM
- **OdsSelect** : Hôte destination
- **OdsRadio** : Type migration
- **OdsButton** : Actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Migrer la VM" |
| Bouton fermer | × |

## 5. INFO VM
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Bordure | #E5E7EB |
| Label | "VM à migrer" |
| Valeur | "{vmName} ({ipAddress})" (bold) |

## 6. HÔTE SOURCE
| Propriété | Valeur |
|-----------|--------|
| Label | "Hôte source" |
| Valeur | "{sourceHost}" (readonly) |

## 7. CHAMP HÔTE DESTINATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Hôte de destination *" |
| Type | Select |
| Format option | "{hostname} (CPU: X%, RAM: Y%)" |
| Obligatoire | Oui |

## 8. OPTIONS TYPE MIGRATION
| Option | État | Description |
|--------|------|-------------|
| Migration à chaud (vMotion) | Selected | Sans interruption de service |
| Migration à froid | Default | Arrêt requis |

## 9. STYLE RADIO
| État | Style |
|------|-------|
| Selected | Cercle #0050D7 avec point interne |
| Default | Cercle blanc, bordure grise |

## 10. FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Fond blanc, bordure grise |
| Migrer | Fond #0050D7, texte blanc |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/host - Hôtes disponibles
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/migrate - Migrer VM
