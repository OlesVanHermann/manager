## 8. APIS & ENDPOINTS

### 8.1 Obtenir statut migration VCF

```
GET /dedicatedCloud/{serviceName}/migration/vcf
```

Response:
```json
{
  "status": "approved",
  "approvalDate": "2025-12-15",
  "termsAccepted": true,
  "migration": {
    "state": "pending_scheduling",
    "targetOffer": "VMware Cloud Foundation"
  }
}
```

### 8.2 Accepter conditions migration

```
POST /dedicatedCloud/{serviceName}/migration/vcf/accept
```
