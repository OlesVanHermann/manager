# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Modifier Droits Utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de configuration droits
- **Contexte** : Gestion utilisateurs Dedicated Cloud (VMware)
- **Objectif** : Modifier les droits d'un utilisateur sur un datacenter spécifique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 900px |
| Page hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsSelect** : Droits vSphere, réseau VM, VLAN/vRack
- **OdsCheckbox** : Peut ajouter des ressources
- **OdsButton** : Annuler, Confirmer
- **OdsText** : Descriptions droits

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier les droits de l'utilisateur" |

## 5. CONTENU

### Contexte datacenter
| Propriété | Valeur |
|-----------|--------|
| Titre | "Droits sur le datacenter {datacenter_name}" |
| Exemple | DC-RBX-1 |

### Droits vSphere
| Propriété | Valeur |
|-----------|--------|
| Label | "Droits vSphere" |
| Type | OdsSelect |
| Largeur | 350px |
| Description | "Permet l'accès complet aux ressources du datacenter dans vSphere." |
| Note | "L'utilisateur peut créer, modifier et supprimer des VMs." (italique) |

### Options Droits vSphere
| Option | Description |
|--------|-------------|
| Aucun accès | Pas d'accès au datacenter |
| Lecture seule | Visualisation uniquement |
| Lecture/Écriture | Accès complet |

### Droits réseau VM
| Propriété | Valeur |
|-----------|--------|
| Label | "Droits réseau VM" |
| Type | OdsSelect |
| Largeur | 350px |
| Description | "Permet de visualiser la configuration réseau des VMs." |

### Options Droits réseau VM
| Option | Description |
|--------|-------------|
| Aucun accès | Pas d'accès réseau |
| Lecture seule | Visualisation configuration |
| Lecture/Écriture | Modification réseau |

### Droits VLAN/vRack
| Propriété | Valeur |
|-----------|--------|
| Label | "Droits VLAN/vRack" |
| Type | OdsSelect |
| Largeur | 350px |

### Options Droits VLAN
| Option | Description |
|--------|-------------|
| Aucun accès | Pas d'accès aux VLANs |
| Lecture seule | Visualisation VLANs |
| Lecture/Écriture | Gestion VLANs |

### Checkbox ressources
| Propriété | Valeur |
|-----------|--------|
| Label | "Peut ajouter des ressources" |
| Description | "Permet d'ajouter des hosts et datastores au datacenter." |
| Coché par défaut | Non |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Confirmer | Droite | Primary (#0050D7) | Enregistre les droits |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/user/{userId}/right/{datacenterId} - Droits actuels
- PUT /dedicatedCloud/{serviceName}/user/{userId}/right/{datacenterId} - Modifier droits
