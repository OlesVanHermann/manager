# Cahier des Charges : bare-metal/dedicated-cloud - Tab DC VMs

## 1. VUE D'ENSEMBLE
- **Type** : Page listing avec tableau
- **Contexte** : Datacenter Dedicated Cloud
- **Objectif** : Lister les machines virtuelles du datacenter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Tableau largeur | 900px |

## 3. COMPOSANTS UI
- **OdsText** : Titre
- **OdsMessage** : Info SPLA
- **OdsDatagrid** : Liste VMs
- **OdsBadge** : Licence, Backup
- **OdsPopover** : Menu actions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Machines virtuelles" |
