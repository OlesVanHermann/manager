## 10. APIS & ENDPOINTS

### 10.1 Obtenir les profils datastore

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/profiles
```

Response:
```json
[
  {
    "name": "ds-nfs-01",
    "type": "NFS - 3 To",
    "monthlyPrice": { "value": 85, "text": "85,00 € HT" }
  }
]
```
