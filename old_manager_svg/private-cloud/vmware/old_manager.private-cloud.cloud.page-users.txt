## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/user - Liste utilisateurs
- POST /dedicatedCloud/{serviceName}/user - Créer utilisateur
- PUT /dedicatedCloud/{serviceName}/user/{userId} - Modifier utilisateur
- DELETE /dedicatedCloud/{serviceName}/user/{userId} - Supprimer utilisateur
- POST /dedicatedCloud/{serviceName}/user/{userId}/resetPassword - Reset mot de passe
