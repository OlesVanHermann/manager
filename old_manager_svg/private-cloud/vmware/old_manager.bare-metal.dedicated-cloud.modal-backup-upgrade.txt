# Cahier des Charges : bare-metal/dedicated-cloud - Modal Backup Upgrade

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation
**Fonction:** Confirmer le changement d'offre de sauvegarde Veeam
**Contexte:** Upgrade de l'offre backup (Advanced → Premium)

## 2. DIMENSIONS

- Modal: 500x430px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. HEADER

- Titre: "Changement de l'offre de sauvegarde" (18px, font-weight: 600)
- Bouton fermer: × (20px, muted)
- Séparateur: #E5E7EB

## 4. WARNING BANNER

- Dimensions: 450x50px
- Background: #FEF3C7
- Border: #FDE68A
- Message: "Si vous avez activé la sauvegarde sur d'autres datacentres..."

## 5. CONTENU

### Description
- "Vous souhaitez changer votre offre de sauvegarde de l'offre **Advanced** à l'offre **Premium**."
- "Vous aurez désormais les paramètres suivants :"

### Liste des fonctionnalités (bullets)
- Rapport de sauvegarde personnalisable
- Sauvegarde chiffrée
- Minimum 28 points de rétention
- Minimum 2 sauvegardes complètes
- Réplication hors site
- 1 rétention par mois conservée pendant 1 an

### Billing Notice
- "Vous serez facturé(e) selon le nombre et la taille des machines virtuelles sauvegardées."
- Font-size: 12px, muted

### Confirmation
- Question: "Êtes-vous sûr(e) de vouloir changer d'offre ?" (bold)
- Checkbox: "Je valide la réplication des données" (checked)

## 6. FOOTER

### Boutons
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | Secondary (border) | 90x36px |
| Passer à Premium | Primary (#0050D7) | 120x36px |

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/offerCapabilities
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/changeOffer
```
