## 10. APIS & ENDPOINTS

### 10.1 Lister les datastores

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer
```

Response:
```json
[
  {
    "filerId": "ds-001",
    "profile": "SSD-3000",
    "size": { "value": 3, "unit": "To" },
    "usagePercent": 40,
    "activeNode": "A",
    "activeHost": "esxi-01",
    "billing": "monthly",
    "state": "delivered",
    "scope": "global",
    "datacenter": "RBX7",
    "rack": "A23"
  }
]
```

### 10.2 Commander un datastore

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/orderNewFilerHourly
```

### 10.3 Convertir en mensuel

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/{filerId}/convertToMonthly
```

### 10.4 Changer la portée

```
PUT /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/{filerId}
```

Body:
```json
{
  "scope": "global"
}
```
