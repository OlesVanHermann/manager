## 10. APIS & ENDPOINTS

### 10.1 Lister les hosts

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host
```

Response:
```json
[
  {
    "hostId": 123456,
    "name": "esxi-01.pcc.ovh.net",
    "profile": "HOST-128-L+",
    "billing": "monthly",
    "state": "delivered",
    "datacenter": "RBX7",
    "room": "5R30",
    "rack": "A23",
    "hourlyConsumption": null
  }
]
```

### 10.2 Convertir en mensuel

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/{hostId}/convertToMonthly
```
