## 9. APIS & ENDPOINTS

### 9.1 Obtenir les offres de mise à niveau

```
GET /dedicatedCloud/{serviceName}/upgradeOptions
```

Response:
```json
[
  {
    "name": "Premier",
    "features": {
      "hosts": "Accès à la gamme complète d'hôtes",
      "datastores": "Stockage NVMe et haute disponibilité",
      "bandwidth": "Jusqu'à 25 Gbps"
    },
    "additionalPrice": {
      "value": 150,
      "currency": "EUR",
      "text": "+ 150,00 € HT/mois"
    }
  }
]
```

### 9.2 Initier la mise à niveau

```
POST /dedicatedCloud/{serviceName}/upgrade
```

Body:
```json
{
  "targetOffer": "Premier"
}
```
