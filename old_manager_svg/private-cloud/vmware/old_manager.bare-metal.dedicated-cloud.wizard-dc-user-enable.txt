# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Activer Utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de confirmation
- **Contexte** : Gestion utilisateurs Dedicated Cloud (VMware)
- **Objectif** : Réactiver un utilisateur vSphere désactivé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 700px |
| Page hauteur | 350px |

## 3. COMPOSANTS UI
- **OdsButton** : Annuler, Activer
- **OdsText** : Warning, question et description

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer l'utilisateur" |

## 5. CONTENU

### Warning
| Propriété | Valeur |
|-----------|--------|
| Texte | "Attention : cette action réactivera l'accès de l'utilisateur." |
| Couleur | Rouge (#DC2626) |
| Style | Gras (font-weight 500) |

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|
| Container | Blanc, border #E5E7EB, radius 8px |
| Question | "Voulez-vous activer l'utilisateur {username} ?" |
| Username | Bleu (#0050D7), gras |

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "L'utilisateur pourra à nouveau se connecter à vSphere." |
| Couleur | Gris (#6B7280) |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Activer | Droite | Success (#16A34A) | Active l'utilisateur |

## 7. APIS & ENDPOINTS
- PUT /dedicatedCloud/{serviceName}/user/{userId} - Activer utilisateur (state: enabled)
