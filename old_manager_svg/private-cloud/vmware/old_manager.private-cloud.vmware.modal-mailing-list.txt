## 9. APIS & ENDPOINTS

### 9.1 S'abonner à la mailing list

```
POST /dedicatedCloud/{serviceName}/mailingList/subscribe
```

Body:
```json
{
  "email": "admin@example.com",
  "consent": true
}
```

Response:
```json
{
  "status": "subscribed",
  "email": "admin@example.com"
}
```
