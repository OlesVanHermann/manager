## 6. ACTIONS & INTERACTIONS

### Sélection datacenter
- Click sur carte → sélectionne le datacenter
- Un seul datacenter sélectionnable

### Contraintes
- Datacenter secondaire ≠ datacenter principal
- Infrastructure différente recommandée

### Bouton Précédent
- Style: btn-secondary (100x36px)
- Position: gauche
- Action: Retour à l'étape 1

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Position: centre-droite
- Action: Annule le wizard

### Bouton Suivant
- Style: btn-primary (100x36px)
- Position: droite
- Action: Valide la configuration et lance la création

## 8. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter
→ Liste les datacenters du PCC

GET /dedicatedCloud/{serviceName}/zerto/eligibleDatacenters
→ Liste les datacenters éligibles comme secondaire

POST /dedicatedCloud/{serviceName}/datacenter/{primaryDcId}/zerto/ovh
Body: {
  secondaryDatacenterId: "dc-london-02",
  secondaryInfrastructure: "pcc-192-0-2-2"
}
→ Configure la réplication OVH vers OVH
```
