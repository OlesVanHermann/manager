# Cahier des Charges : bare-metal/dedicated-cloud - Component Backup Offers Picker

## 1. VUE D'ENSEMBLE

**Type:** Composant de sélection multiple (Picker)
**Fonction:** Permettre la sélection d'une offre Veeam Backup
**Contexte:** Wizard de configuration backup

## 2. DIMENSIONS

- Container: 1100x400px
- Cards: 340x320px chacune
- Espacement entre cards: 20px

## 3. HEADER

- Titre: "Sélection de l'offre Veeam Backup" (16px, font-weight: 600)

## 4. STRUCTURE DES CARDS

### Radio button

| Sélectionné | Cercle plein #0050D7, centre blanc |

### Badge "Offre actuelle"

## 5. OFFRES DISPONIBLES

### Standard (Current)
- 14 points de rétention
- 1 sauvegarde complète
- 1 backup proxy par hôte
- Rapport de sauvegarde
- **Prix**: 5,00 € HT/VM/mois

### Advanced (Selected)
- Minimum 14 points de rétention
- Minimum 1 sauvegarde complète
- 1 backup proxy par hôte
- Rapport personnalisable
- Sauvegarde chiffrée
- Choix des jours de sauvegarde
- **Prix**: 10,00 € HT/VM/mois

### Premium
- Toutes les fonctionnalités Advanced
- Réplication hors site
- Rétention 1 an (1/mois)
- **Prix**: 20,00 € HT/VM/mois
