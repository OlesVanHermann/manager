## 10. APIS & ENDPOINTS

### 10.1 Obtenir les offres datastore

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/profiles
```

Response:
```json
[
  {
    "name": "ds-ssd-256g",
    "description": "SSD 256 GB - High Performance",
    "price": { "value": 29.99, "text": "29,99 € HT/mois" },
    "commitment": "monthly"
  }
]
```

### 10.2 Commander un datastore

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/orderNewFiler
```

Body:
```json
{
  "profile": "ds-ssd-256g",
  "quantity": 1
}
```
