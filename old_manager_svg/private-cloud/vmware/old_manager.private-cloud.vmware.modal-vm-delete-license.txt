---

## 7. APIS & ENDPOINTS

### 7.1 Supprimer la licence VM

```
DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId}/license
```

Response:
```json
{
  "taskId": "task-123",
  "status": "todo",
  "message": "License removal requested"
}
```
