# Cahier des Charges : bare-metal/dedicated-cloud - Modal Zerto Delete Site

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation danger
**Fonction:** Supprimer un site Zerto de la configuration
**Contexte:** Accessible depuis le menu actions d'un site Zerto

## 2. DIMENSIONS

- Modal: 500x300px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal (Danger)
- Background: #FEE2E2
- Icône: Cercle rouge #DC2626 avec "!"
- Titre: "Supprimer un site Zerto" (color: #991B1B)
- Close button: × (color: #991B1B)

### Contenu
- Question: "Souhaitez-vous vraiment supprimer le site Zerto"
- Nom site: "site-paris-01" (font-weight: 600)
- Avertissement: "Cette action est irréversible. Le site sera définitivement supprimé de votre configuration Zerto."
