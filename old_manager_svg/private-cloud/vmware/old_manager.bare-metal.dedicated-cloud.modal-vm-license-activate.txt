# Cahier des Charges : bare-metal/dedicated-cloud - Modal VM License Activate

## 1. VUE D'ENSEMBLE

**Type:** Modal de formulaire simple
**Fonction:** Activer une licence sur une machine virtuelle
**Contexte:** Accessible depuis le menu actions d'une VM dans l'onglet Licences

## 2. DIMENSIONS

- Modal: 600x290px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal
- Background: #F9FAFB
- Titre: "Activer la licence"
- Close button: ×

### Info VM
- Label: "Machine virtuelle :"
- Valeur: "vm-webapp-prod-01" (font-weight: 600)

### Formulaire

#### Champ Licence
- Label: "Licence *"
- Type: Select dropdown
- Dimensions: 520x44px
- Valeur: "Windows Server 2022 Datacenter"

## 4. HEADER MODAL

- Hauteur: 60px
- Titre: "Activer la licence"

## 5. CONTENU

### Sélection de licence
| Option | Description |
|--------|-------------|
| Windows Server 2022 Datacenter | Licence datacenter complète |
| Windows Server 2022 Standard | Licence standard |
| SQL Server 2022 Enterprise | Licence SQL Enterprise |
| SQL Server 2022 Standard | Licence SQL Standard |

## 6. FOOTER MODAL

- Séparateur: ligne 1px
- Bouton Annuler: btn-secondary (100x36px)
- Bouton Activer: btn-primary (100x36px)

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/license/available
→ Liste les licences disponibles

POST /dedicatedCloud/{serviceName}/vm/{vmId}/license
Body: {
  licenseType: "windows-server-2022-datacenter"
}
→ Active la licence sur la VM
```
