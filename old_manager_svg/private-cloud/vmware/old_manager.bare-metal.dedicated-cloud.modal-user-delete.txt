# Cahier des Charges : bare-metal/dedicated-cloud - Modal User Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation danger
**Fonction:** Supprimer un utilisateur de l'infrastructure vSphere
**Contexte:** Accessible depuis le menu actions d'un utilisateur dans l'onglet Utilisateurs

## 2. DIMENSIONS

- Modal: 500x260px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal (Danger)
- Background: #FEE2E2
- Icône: Cercle rouge #DC2626 avec "!"
- Titre: "Supprimer l'utilisateur" (color: #991B1B)
- Close button: × (color: #991B1B)

### Contenu
- Question: "Êtes-vous sûr de vouloir supprimer l'utilisateur"
- Nom utilisateur: "user-backup" (font-weight: 600)
- Avertissement: "Cette action est irréversible. L'utilisateur perdra tous ses accès à l'infrastructure vSphere."
- Font-size avertissement: 13px, color: text-muted

## 4. HEADER MODAL

- Hauteur: 60px
- Background: #FEE2E2
- Icône warning: cercle 30px avec "!"

## 5. CONTENU

### Message de confirmation
```
Êtes-vous sûr de vouloir supprimer l'utilisateur
user-backup ?

Cette action est irréversible. L'utilisateur perdra tous
ses accès à l'infrastructure vSphere.
```

## 6. FOOTER MODAL

- Séparateur: ligne 1px #E5E7EB
- Bouton Annuler: btn-secondary (100x36px)
- Bouton Supprimer: btn-danger #DC2626 (120x36px)

## 7. APIS & ENDPOINTS

```
DELETE /dedicatedCloud/{serviceName}/user/{userId}
→ Supprime l'utilisateur de l'infrastructure

GET /dedicatedCloud/{serviceName}/user/{userId}
→ Récupère les infos de l'utilisateur pour affichage
```
