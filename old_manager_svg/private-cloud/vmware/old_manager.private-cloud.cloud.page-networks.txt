# Cahier des Charges : private/cloud - Page Networks

## 1. VUE D'ENSEMBLE
- **Type** : Page listing réseaux virtuels
- **Contexte** : Private Cloud
- **Objectif** : Gérer les réseaux du cloud privé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer réseau
- **OdsDatagrid** : Tableau réseaux
- **OdsBadge** : VLAN ID, DHCP, état
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation

| Propriété | Valeur |

## 12. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/network - Liste réseaux
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/network - Créer réseau
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/network/{networkId} - Modifier
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/network/{networkId} - Supprimer
