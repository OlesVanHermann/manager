# Cahier des Charges : bare-metal/dedicated-cloud - Component Activation Status

## 1. VUE D'ENSEMBLE
- **Type** : Composant réutilisable (badge status)
- **Contexte** : Dedicated Cloud - Affichage états
- **Objectif** : Afficher l'état d'activation d'un composant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Badge hauteur | 24px |
| Border radius | 4px |

## 3. COMPOSANTS UI
- **OdsBadge** : Badge de statut
- **OdsChip** : Badges datacenter

## 4. ÉTATS POSSIBLES

### Activé
| Propriété | Valeur |
|-----------|--------|
| Fond | #DCFCE7 |
| Texte | #16A34A "Activé" |

### En cours
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Texte | #92400E "En cours..." |

### Désactivé
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEE2E2 |
| Texte | #DC2626 "Désactivé" |

## 5. BADGES DATACENTER (optionnel)
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | #0050D7 "Hébergé sur {dc-name}" |

## 6. USAGE
Composant utilisé dans les tuiles pour afficher l'état d'activation des options (NSX-T, vCD, logs, etc.)
