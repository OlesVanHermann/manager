| Règle | Message erreur |
|-------|----------------|
| Valeur >= 0 | La valeur doit être positive ou nulle |
| Valeur entière | La valeur doit être un nombre entier |

---

## 9. APIS & ENDPOINTS

### 9.1 Modifier le nombre max de connexions

```
PUT /dedicatedCloud/{serviceName}/security/simultaneousConnections
```

Body:
```json
{
  "maxConnections": 5
}
```

Response:
```json
{
  "maxConnections": 5,
  "status": "updated"
}
```
