# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Désactiver Utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de confirmation warning
- **Contexte** : Gestion utilisateurs Dedicated Cloud (VMware)
- **Objectif** : Désactiver temporairement un utilisateur vSphere

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 700px |
| Page hauteur | 350px |

## 3. COMPOSANTS UI
- **OdsButton** : Annuler, Désactiver
- **OdsText** : Question et description

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Désactiver l'utilisateur" |

## 5. CONTENU

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|
| Container | Blanc, border #E5E7EB, radius 8px |
| Question | "Voulez-vous désactiver l'utilisateur {username} ?" |
| Username | Bleu (#0050D7), gras |

### Description
| Ligne | Contenu |
|-------|---------|
| 1 | "L'utilisateur ne pourra plus se connecter à vSphere." |
| 2 | "Cette action peut être annulée en réactivant l'utilisateur." |
| Couleur | Gris (#6B7280) |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Désactiver | Droite | Warning (#F59E0B) | Désactive l'utilisateur |

## 7. APIS & ENDPOINTS
- PUT /dedicatedCloud/{serviceName}/user/{userId} - Désactiver utilisateur (state: disabled)
