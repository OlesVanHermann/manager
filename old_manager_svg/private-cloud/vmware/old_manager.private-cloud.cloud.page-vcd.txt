## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/vcd
→ Statut VCD

GET /dedicatedCloud/{serviceName}/vcd/organization
→ Liste des organisations

GET /dedicatedCloud/{serviceName}/vcd/organization/{organizationId}
→ Détails organisation

POST /dedicatedCloud/{serviceName}/vcd/organization
Body: { name: "org-new" }
→ Crée une organisation

DELETE /dedicatedCloud/{serviceName}/vcd/organization/{organizationId}
→ Supprime l'organisation

GET /dedicatedCloud/{serviceName}/vcd/organization/{organizationId}/vdc
→ vDCs de l'organisation
```
