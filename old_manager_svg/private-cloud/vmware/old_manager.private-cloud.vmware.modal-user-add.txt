## 8. APIS & ENDPOINTS

### 8.1 Créer un utilisateur

```
POST /dedicatedCloud/{serviceName}/user
```

Body:
```json
{
  "login": "newuser",
  "email": "newuser@example.com",
  "password": "SecureP@ss123",
  "role": "standard"
}
```

Response:
```json
{
  "taskId": "task-123",
  "userId": 456,
  "status": "todo",
  "message": "User creation requested"
}
```

### 8.2 Obtenir les rôles disponibles

```
GET /dedicatedCloud/{serviceName}/roles
```

Response:
```json
[
  { "id": "standard", "label": "Utilisateur standard" },
  { "id": "admin", "label": "Administrateur" },
  { "id": "readonly", "label": "Lecture seule" }
]
```
