# Cahier des Charges : bare-metal/dedicated-cloud - Datacenter Hosts

## 1. VUE D'ENSEMBLE

**Type:** Page listing avec datagrid
**Fonction:** Lister et gérer les hosts d'un datacenter
**Contexte:** Configuration datacenter VMware Private Cloud

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/{hostId}
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/orderNewHostHourly
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/{hostId}/changeMonthlyBilling
```
