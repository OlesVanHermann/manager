## 8. APIS & ENDPOINTS

### 8.1 Obtenir le statut de la licence

```
GET /dedicatedCloud/{serviceName}/spla/status
```

Response:
```json
{
  "enabled": true,
  "licenseType": "SPLA",
  "totalVMs": 12,
  "licensedVMs": 8,
  "estimatedMonthlyCost": {
    "value": 96,
    "text": "96.00 EUR HT"
  }
}
```
