## 11. APIS & ENDPOINTS

### 11.1 Obtenir les profils hôtes

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/profiles
```

Response:
```json
[
  {
    "name": "SDDC-128",
    "frequency": "2.1 GHz",
    "memory": "128 Go",
    "cores": 12,
    "monthlyPrice": { "value": 450, "text": "450,00 € HT" }
  },
  {
    "name": "SDDC-256",
    "frequency": "2.3 GHz",
    "memory": "256 Go",
    "cores": 18,
    "monthlyPrice": { "value": 750, "text": "750,00 € HT" }
  }
]
```
