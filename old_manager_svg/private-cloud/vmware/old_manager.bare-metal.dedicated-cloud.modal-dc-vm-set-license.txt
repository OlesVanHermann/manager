# Cahier des Charges : bare-metal/dedicated-cloud - Modal Définir Licence VM

## 1. VUE D'ENSEMBLE
- **Type** : Modal de sélection avec dropdown
- **Contexte** : Gestion des licences VM
- **Objectif** : Attribuer une licence SPLA à une machine virtuelle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélecteur de licence
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Définir la licence de la VM" |
| Fond | Gris clair (#F9FAFB) |
| Couleur texte | Gris foncé (#1F2937) |
| Bouton fermer | × |

## 5. CONTENU

### Sélecteur de licence
| Propriété | Valeur |
|-----------|--------|
| Label | "Sélectionnez une licence" |
| Type | OdsSelect (dropdown) |
| Style sélectionné | Bordure bleue (#0050D7) 2px |

### Options disponibles
| Licence |
|---------|
| Windows Server Standard 2022 |
| Windows Server Datacenter 2022 |
| SQL Server Standard 2022 |
| ... |

### Style dropdown ouvert
| Propriété | Valeur |
|-----------|--------|
| Fond option sélectionnée | Bleu clair (#EFF6FF) |
| Fond autres options | Blanc |
| Bordure | Gris (#E5E7EB) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Attribue la licence |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId}/license - Licences disponibles
- PUT /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId} - Attribuer licence
