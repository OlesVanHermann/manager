## 9. APIS & ENDPOINTS

### 9.1 Activer IAM OVHcloud

```
POST /dedicatedCloud/{serviceName}/iam/enable
```

Response:
```json
{
  "taskId": "task-123",
  "status": "todo",
  "message": "Enabling IAM OVHcloud integration"
}
```

### 9.2 Désactiver IAM OVHcloud

```
POST /dedicatedCloud/{serviceName}/iam/disable
```

### 9.3 Obtenir statut IAM

```
GET /dedicatedCloud/{serviceName}/iam
```

Response:
```json
{
  "enabled": false,
  "compatible": true,
  "restrictions": []
}
```
