# Cahier des Charges : private/cloud - Page Affinity

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion règles d'affinité
- **Contexte** : Private Cloud - VMware DRS
- **Objectif** : Configurer les règles de placement des VMs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajouter règle
- **OdsDatagrid** : Tableau règles
- **OdsBadge** : Type règle, état
- **OdsPopover** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Règles d'affinité" |
| Description | "Configurez les règles de placement des VMs." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter règle" |
| Style | Fond #0050D7, texte blanc |
| Largeur | 130px |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Nom | 230px | Nom règle (bold) |
| Type | 150px | Badge affinité/anti-affinité |
| VMs | 300px | Liste VMs concernées |
| État | 150px | Badge actif/inactif |
| Actions | 80px | Menu ⋮ |

## 7. BADGES TYPE
| Type | Fond | Texte |
|------|------|-------|
| Affinité | #DCFCE7 | #16A34A |
| Anti-affinité | #FEE2E2 | #DC2626 |

## 8. BADGES ÉTAT
| État | Fond | Texte |
|------|------|-------|
| Actif | #DCFCE7 | #16A34A |
| Inactif | #E5E7EB | #6B7280 |

## 9. MENU ACTIONS
| Action | Style |
|--------|-------|
| Modifier | Texte #1F2937 |
| Activer/Désactiver | Texte #1F2937 |
| --- | Séparateur |
| Supprimer | Texte #DC2626 |

## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/affinityRule - Liste règles
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/affinityRule - Créer règle
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/affinityRule/{ruleId} - Modifier
- DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/affinityRule/{ruleId} - Supprimer
