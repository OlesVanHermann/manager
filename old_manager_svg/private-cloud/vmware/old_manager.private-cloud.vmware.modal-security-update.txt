## 8. APIS & ENDPOINTS

### 8.1 Modifier une plage IP

```
PUT /dedicatedCloud/{serviceName}/security/ipRange/{ipRangeId}
```

Body:
```json
{
  "ip": "192.168.1.0/24",
  "description": "Réseau local bureau Paris"
}
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "pending"
}
```
