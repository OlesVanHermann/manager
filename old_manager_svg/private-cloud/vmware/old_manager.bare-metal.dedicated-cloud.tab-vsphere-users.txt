**Type:** Page Listing / Datagrid avec menu actions
**Fonction:** Gérer les utilisateurs vSphere et rôles IAM

## 7. PAGINATION

## 8. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/user
→ Liste les utilisateurs vSphere

GET /dedicatedCloud/{serviceName}/iam/role
→ Liste les rôles IAM

POST /dedicatedCloud/{serviceName}/user
→ Crée un utilisateur

DELETE /dedicatedCloud/{serviceName}/user/{userId}
→ Supprime un utilisateur

DELETE /dedicatedCloud/{serviceName}/iam/role/{roleId}
→ Supprime un rôle IAM
```
