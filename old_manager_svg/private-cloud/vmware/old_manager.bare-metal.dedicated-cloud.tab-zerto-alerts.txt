# Cahier des Charges : bare-metal/dedicated-cloud - Tab Zerto Alerts

## 1. VUE D'ENSEMBLE

**Type:** Page d'alertes
**Fonction:** Afficher les alertes et notifications Zerto
**Contexte:** Sous-section Alertes de l'onglet Zerto

## 2. DIMENSIONS

- Container principal: 1360x350px

## 3. LAYOUT & STRUCTURE

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > Datacenter dc-123 > **Zerto - Alertes**

### Zone principale
1. Titre
2. Alertes empilées (success, info, warning)
3. Note explicative

## 4. COMPOSANTS UI

### Titre
- Text: "Alertes Zerto"
- Font-size: 18px

### Types d'alertes

#### Alerte Success (PRA créé)
- Background: #DCFCE7
- Border: 1px solid #BBF7D0
- Icône: ✓ (cercle vert)
- Titre: "Votre PRA Zerto a été créé avec succès !"
- Messages:
  - "Le plan de reprise d'activité est maintenant actif avec pcc-192-0-2-2 en datacenter secondaire."
  - "Vous avez reçu votre mot de passe ZSSP sur l'adresse email user@example.com"
- Bouton fermer: × (vert)
- Hauteur: 70px

#### Alerte Info (Activation en cours)
- Background: #EFF6FF
- Border: 1px solid #BFDBFE
- Icône: i (cercle bleu)
- Messages:
  - "Votre PRA Zerto est en cours d'activation..."
  - "N'hésitez pas à vérifier la tâche enableDisasterRecovery..."
- Hauteur: 55px

#### Alerte Warning (VPN non configuré)
- Background: #FEF3C7
- Border: 1px solid #FDE68A
- Icône: ⚠
- Messages:
  - "La configuration VPN n'est pas finalisée."
  - "Veuillez configurer le VPN pour finaliser l'activation..."
- Lien: "configurer le VPN" (text-primary souligné)
- Hauteur: 55px

### Note
- Position: Bas
- Texte: "Les alertes sont affichées en fonction de l'état actuel de votre configuration Zerto."
- Font-size: 12px, color: text-muted

## 5. ACTIONS & INTERACTIONS

### Bouton fermer (×)
- Masque l'alerte
- Disponible uniquement sur certaines alertes

### Lien "configurer le VPN"
- Redirige vers la page de configuration VPN

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/zerto/alerts
→ Récupère les alertes actives

POST /dedicatedCloud/{serviceName}/zerto/alert/{alertId}/dismiss
→ Masque une alerte
```
