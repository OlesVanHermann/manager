# Cahier des Charges : bare-metal/dedicated-cloud - Onglet Utilisateurs vSphere

## 1. VUE D'ENSEMBLE
- **Type** : Onglet de gestion utilisateurs
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Gérer les utilisateurs vSphere et leurs droits

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste des utilisateurs
- **OdsToggle** : Gestion IAM, Validateur token, NSX
- **OdsButton** : Ajouter utilisateur
- **OdsBadge** : États utilisateur
- **OdsActionMenu** : Actions par ligne

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Utilisateurs vSphere" |

## 5. CONTENU

### Toggle Gestion IAM
| Propriété | Valeur |
|-----------|--------|
| Label | "Gestion IAM" |
| État | Désactivé |
| Description | Permet l'intégration avec OVHcloud IAM |

### Bouton d'action
| Bouton | Style | Action |
|--------|-------|--------|
| + Ajouter utilisateur | Primary (#0050D7) | Ouvre wizard création |

### Datagrid Utilisateurs
| Colonne | Type | Largeur | Triable |
|---------|------|---------|---------|
| Nom | Texte + sous-titre | 170px | Oui |
| Email | Texte | 200px | Oui |
| État | Badge | 100px | Oui |
| Validateur token | Toggle | 120px | Non |
| NSX | Toggle | 100px | Non |
| Actions | ActionMenu | 80px | Non |

### Format colonne Nom
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | Nom utilisateur (gras) |
| Ligne 2 | @pcc-xxx-xxx.ovh.com (gris, petit) |

### Badges État
| État | Fond | Couleur texte |
|------|------|---------------|
| Actif | #DCFCE7 | #16A34A |
| Désactivé | #FEE2E2 | #DC2626 |

### Style Toggles
| État | Apparence |
|------|-----------|
| Activé | Fond #0050D7, cercle à droite |
| Désactivé | Fond #E5E7EB, cercle à gauche |

### Exemple données
| Nom | Email | État | Token | NSX |
|-----|-------|------|-------|-----|
| admin | admin@example.com | Actif | ✓ | ✓ |
| developer01 | dev@example.com | Actif | ✗ | ✗ |
| readonly_user | viewer@example.com | Désactivé | ✗ | ✗ |

### ActionMenu options
| Action | Couleur |
|--------|---------|
| Modifier | Noir |
| Changer mot de passe | Noir |
| Modifier les droits | Noir |
| Désactiver/Activer | Noir |
| Supprimer | Rouge (#DC2626) |

### Pagination
| Propriété | Valeur |
|-----------|--------|
| Texte | "{n} utilisateurs" |

## 6. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/user - Liste utilisateurs
- POST /dedicatedCloud/{serviceName}/user - Créer utilisateur
- PUT /dedicatedCloud/{serviceName}/user/{userId} - Modifier utilisateur
- DELETE /dedicatedCloud/{serviceName}/user/{userId} - Supprimer utilisateur
- POST /dedicatedCloud/{serviceName}/user/{userId}/changePassword - Changer mot de passe
