## 8. APIS & ENDPOINTS

### 8.1 Créer un rôle IAM

```
POST /dedicatedCloud/{serviceName}/iam/roles
```

Body:
```json
{
  "name": "admin-role",
  "vsphereUser": "admin@vsphere.local"
}
```

Response:
```json
{
  "roleId": "pcc-XXX-XXX-XXX-XXX/admin-role",
  "status": "creating"
}
```
