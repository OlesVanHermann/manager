## 10. APIS & ENDPOINTS

### 10.1 Obtenir les packs disponibles

```
GET /dedicatedCloud/{serviceName}/servicePacks
```

Response:
```json
[
  {
    "name": "Premier",
    "features": ["NSX", "vROps", "Support Premium 24/7"],
    "price": { "value": 500, "currency": "EUR", "text": "À partir de 500 €/mois" }
  },
  {
    "name": "Essentials",
    "features": ["Configuration de base", "Support Standard"],
    "price": { "value": 200, "currency": "EUR", "text": "À partir de 200 €/mois" }
  }
]
```

### 10.2 Commander une mise à niveau

```
POST /dedicatedCloud/{serviceName}/servicePacks/upgrade
```

Body:
```json
{
  "servicePack": "Premier"
}
```
