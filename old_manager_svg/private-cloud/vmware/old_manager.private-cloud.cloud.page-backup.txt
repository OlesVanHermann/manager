# Cahier des Charges : private/cloud - Page Backup

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion sauvegardes
- **Contexte** : Private Cloud - Veeam
- **Objectif** : Gérer les sauvegardes des machines virtuelles

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et valeurs
- **OdsCard** : Cartes stats, tableau jobs
- **OdsBadge** : État exécution
- **OdsDatagrid** : Liste jobs

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Sauvegardes Veeam" |
| Description | "Gérez les sauvegardes de vos machines virtuelles." |

## 5. CARTES STATISTIQUES
| Carte | Label | Valeur exemple |
|-------|-------|----------------|
| 1 | VMs protégées | "18 / 24" |
| 2 | Dernière sauvegarde | "Succès" (vert) |
| 3 | Stockage utilisé | "2.4 To" |
| 4 | Prochaine exécution | "30/12 03:00" |

## 6. STYLE CARTES
| Propriété | Valeur |

## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/veeam/backup/job - Liste jobs
- GET /dedicatedCloud/{serviceName}/veeam/backup/job/{jobId} - Détail job
- POST /dedicatedCloud/{serviceName}/veeam/backup/job/{jobId}/start - Lancer backup
- GET /dedicatedCloud/{serviceName}/veeam/backup/stats - Statistiques
