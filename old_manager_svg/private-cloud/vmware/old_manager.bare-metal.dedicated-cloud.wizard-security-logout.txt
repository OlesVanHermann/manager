# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Security Logout

## 1. VUE D'ENSEMBLE

**Type:** Wizard 2 étapes
**Fonction:** Configurer la politique de déconnexion en cas de double connexion
**Contexte:** Paramètres de sécurité du Dedicated Cloud

## 2. DIMENSIONS

- Container principal: 1440x600px
- Zone wizard: 1000x450px (centrée)

## 3. LAYOUT & STRUCTURE

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > Sécurité > **Politique de déconnexion**

### Header wizard
- Background: #F9FAFB
- Titre: "Modifier la politique de déconnexion"
- Hauteur: 70px

### Stepper (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Active (bleu) |
| 2 | Confirmation | Inactive (gris) |

## 4. ÉTAPE 1 : CONFIGURATION

### Label select
- Texte: "Politique de déconnexion en cas de double connexion"

### Select dropdown
- Dimensions: 500x45px
- Placeholder: "Sélectionnez"
- Icône: ▼ (dropdown)

### Options dropdown (affichées)
| Option | Description |
|--------|-------------|
| Première connexion | Déconnecte la première session (sélectionnée, bg-info) |
| Dernière connexion | Déconnecte la nouvelle session |
| Ne pas déconnecter | Autorise les connexions multiples |

## 5. COMPOSANTS UI

### Select dropdown ouvert
- Background options: bg-white
- Option sélectionnée: bg-info #EFF6FF avec text-primary
- Bordure: #E5E7EB
- Border-radius: 4px
- Shadow: drop-shadow(0 4px 6px rgba(0,0,0,0.1))

### Stepper
- Step active: cercle bleu #0050D7 avec numéro blanc
- Step inactive: cercle gris #E5E7EB
- Label active: text-body font-weight: 500

## 6. ACTIONS & INTERACTIONS

### Sélection option
- Click sur option → sélectionne et ferme le dropdown
- Option sélectionnée mise en surbrillance

### Bouton Annuler
- Style: btn-secondary (100x40px)
- Position: gauche
- Action: Retour à la page sécurité

### Bouton Suivant
- Style: btn-primary (120x40px)
- Position: droite
- Action: Passe à l'étape 2 (Confirmation)
- Disabled si aucune option sélectionnée

## 7. ÉTAPE 2 : CONFIRMATION

### Contenu (non visible dans le SVG)
- Résumé de la politique sélectionnée
- Bouton "Confirmer" pour appliquer le changement
- Bouton "Précédent" pour modifier

## 8. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/security
→ Récupère les paramètres de sécurité actuels

PUT /dedicatedCloud/{serviceName}/security
Body: {
  logoutPolicy: "first" | "last" | "none"
}
→ Met à jour la politique de déconnexion
```
