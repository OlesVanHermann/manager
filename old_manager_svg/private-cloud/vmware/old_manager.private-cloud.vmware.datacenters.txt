## 10. APIS & ENDPOINTS

### 10.1 Lister les datacenters

```
GET /dedicatedCloud/{serviceName}/datacenter
```

Response:
```json
[
  {
    "datacenterId": "dc-1",
    "name": "dc-1",
    "commercialRange": "Dedicated Cloud",
    "totalCpu": "96 GHz",
    "totalRam": "768 Go",
    "totalDisk": "24 To",
    "vmCount": 45,
    "windowsVmCount": 12,
    "licensedWindowsVmCount": 8
  },
  {
    "datacenterId": "dc-2",
    "name": "dc-2",
    "commercialRange": "Dedicated Cloud avec NSX",
    "totalCpu": "192 GHz",
    "totalRam": "1.5 To",
    "totalDisk": "48 To",
    "vmCount": 78,
    "windowsVmCount": 25,
    "licensedWindowsVmCount": 18

]
```
