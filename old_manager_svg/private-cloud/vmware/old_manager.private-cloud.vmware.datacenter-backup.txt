## 9. APIS & ENDPOINTS

### 9.1 Obtenir configuration backup

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup
```

Response:
```json
{
  "enabled": true,
  "offer": "advanced",
  "restorePoints": 28,
  "reportOptions": {
    "duration": true,
    "size": true,
    "diskSize": false,
    "fullDayIncluded": true,
    "restorePoints": true
  }
}
```

### 9.2 Changer d'offre

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/changeOffer
```

Body:
```json
{
  "offer": "premium"
}
```

### 9.3 Mettre à jour options rapport

```
PUT /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/report
```

Body:
```json
{
  "duration": true,
  "size": true,
  "diskSize": true,
  "fullDayIncluded": true,
  "restorePoints": true
}
```

### 9.4 Activer/Désactiver backup

```
POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/enable
DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup
```
