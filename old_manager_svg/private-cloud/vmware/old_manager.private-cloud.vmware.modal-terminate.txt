|--------|----------|

| Clic × | Fermer modal |
| Clic select | Ouvrir dropdown options résiliation |
| Clic Confirmer | Continuer vers modal-terminate-confirm |

---

## 7. OPTIONS DE RÉSILIATION

| Option | Description |
|--------|-------------|
| Résilier à la date d'expiration | Service résilié à fin d'engagement, accès conservé |
| Résilier immédiatement | Résiliation immédiate avec perte de données |

---

## 8. APIS & ENDPOINTS

### 8.1 Initier la résiliation

```
POST /dedicatedCloud/{serviceName}/terminate/prepare
```

Body:
```json
{
  "terminationType": "atExpiration"
}
```

Response:
```json
{
  "serviceEndDate": "2025-03-31",
  "terminationType": "atExpiration"
}
```
