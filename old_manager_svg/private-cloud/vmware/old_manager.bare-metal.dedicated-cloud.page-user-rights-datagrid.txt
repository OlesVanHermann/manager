# Cahier des Charges : bare-metal/dedicated-cloud - Page User Rights Datagrid

## 1. VUE D'ENSEMBLE

**Type:** Page listing avec datagrid
**Fonction:** Afficher et gérer les droits d'un utilisateur par datacentre
**Contexte:** Gestion droits vSphere par DC

## 2. LAYOUT

- Page: 1300x500px
- Background: #F9FAFB

## 3. HEADER

### Retour
- "← Retour" (lien primary)
- Fil: "Gestion des droits de l'utilisateur **{username}** par datacentre"

## 4. DATAGRID

### Colonnes
| Colonne | Largeur | Type |
|---------|---------|------|
| Datacentre | 150px | Texte |
| Accès vSphere | 180px | Texte |
| Gestion hôtes/stockages | 180px | Texte (Oui/Non) |
| Accès vmNetwork | 150px | Texte |
| Accès V(x)Lans | 150px | Texte |
| Actions | 80px | Menu ⋮ |

### Valeurs accès vSphere
| Valeur | Description |
|--------|-------------|
| Lecture/Écriture | Accès complet |
| Lecture seule | Consultation uniquement |
| Aucun | Pas d'accès |

### Valeurs Accès réseau
| Valeur | Description |
|--------|-------------|
| Administrateur | Droits admin |
| Lecture seule | Consultation |
| Aucun | Pas d'accès |

### Menu Actions
| Option |
|--------|
| Modifier les droits |

## 5. PAGINATION

- Format: "X datacentres"

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/user/{userId}/right
PUT /dedicatedCloud/{serviceName}/user/{userId}/right/{rightId}
```
