# Cahier des Charges : web - Liste des domaines

## 1. VUE D'ENSEMBLE
- **Page** : Liste des noms de domaine
- **Type** : Datagrid avec recherche et actions
- **Objectif** : Voir et gérer tous les domaines du compte

## 2. STRUCTURE PAGE
- Sidebar gauche avec menu "NOMS DE DOMAINE"
- Breadcrumb : "Noms de domaine"
- Titre principal "Mes domaines"
- Barre recherche + bouton commande
- Datagrid des domaines
- Pagination

## 3. COMPOSANTS UI
- **OdsInput** : Recherche domaine
- **OdsButton** : Commander un domaine (primary)
- **OdsDatagrid** : Liste domaines
- **OdsBadge** : Statut (Actif, Expire bientôt)
- **OdsPagination** : Navigation

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Titre | "Mes domaines" |
| Description | "Gérez tous vos noms de domaine OVHcloud." |
| Recherche | Placeholder "Rechercher un domaine..." |
| Bouton | "+ Commander un domaine" (primary) |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Link bleu | Nom de domaine cliquable |
| Expiration | Date | Date d'expiration (orange si proche) |
| Renouvellement | Text | Automatique / Manuel |
| DNSSEC | Text | Actif (vert) / Inactif |
| Status | Badge | Actif / Expire bientôt |
| Actions | Menu ⋮ | Actions contextuelles |

## 6. BADGES STATUT
| Statut | Couleur | Condition |
|--------|---------|-----------|
| Actif | Vert | Domaine OK |
| Expire bientôt | Orange | < 30 jours |

## 7. EXEMPLES DONNÉES
| Domaine | Expiration | Renouvellement | DNSSEC | Status |

| example-shop.com | 01/02/2025 | Automatique | Inactif | Actif |
| mon-blog.net | 10/01/2025 | Manuel | Inactif | Expire bientôt |
| mycompany.eu | 20/06/2025 | Automatique | Actif | Actif |

## 8. PAGINATION
| Élément | Valeur |
|---------|--------|
| Compteur | "4 domaines" |

## 9. APIS & ENDPOINTS
- GET /domain - Liste des domaines
- GET /domain/{domain} - Détails domaine
- POST /order/domain - Commander un domaine
