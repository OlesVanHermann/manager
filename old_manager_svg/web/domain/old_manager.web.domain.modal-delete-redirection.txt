# Cahier des Charges : web/domain - Modal supprimer redirection

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation avec options
- **Contexte** : Gestion domaine - Redirections
- **Objectif** : Supprimer une redirection email ou web

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question
- **OdsCheckbox** : Option www
- **OdsMessage** : Warning irréversible
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la redirection" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Voulez-vous supprimer la redirection {source} → {destination} ?" |

## 6. OPTION WWW
| Propriété | Valeur |
|-----------|--------|
| Checkbox | Coché par défaut |
| Label | "Supprimer également www.{source}" |

## 7. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ Cette action est irréversible." |

## 8. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Supprimer | Danger (rouge) | Supprime la redirection |

## 9. APIS & ENDPOINTS
- DELETE /email/domain/{domain}/redirection/{id} - Supprimer redirection email
- DELETE /domain/zone/{zoneName}/redirection/{id} - Supprimer redirection web
