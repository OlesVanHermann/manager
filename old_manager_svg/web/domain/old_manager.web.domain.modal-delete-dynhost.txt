# Cahier des Charges : web/domain - Modal supprimer DynHost

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation avec détails
- **Contexte** : Gestion DNS - DynHost
- **Objectif** : Supprimer une entrée DynHost

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 250px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question
- **OdsCard** : Détails de l'entrée
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le DynHost" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Voulez-vous supprimer l'entrée DynHost suivante ?" |

## 6. CARTE DÉTAILS
| Champ | Exemple |
|-------|---------|
| Sous-domaine | home.example.com |
| IP actuelle | 192.168.1.100 |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Supprimer | Danger (rouge) | Supprime l'entrée |

## 8. APIS & ENDPOINTS
- DELETE /domain/zone/{zoneName}/dynHost/record/{id} - Supprimer entrée DynHost
