# Cahier des Charges : web - Modal Ajout DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter une entrée DNS
- **Type** : Formulaire de création
- **Objectif** : Créer un nouvel enregistrement DNS

## 2. STRUCTURE MODAL
- Dimensions : 600x520px
- Header avec titre
- Formulaire 4 champs
- Bannière info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsSelect** : Type d'enregistrement
- **OdsInput** : Sous-domaine, Cible, TTL
- **OdsMessage** : Info propagation
- **OdsButton** : Annuler, Ajouter

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Titre | "Ajouter une entrée DNS" |
| Fond | Gris clair (#F9FAFB) |
| Bouton fermer | ✕ |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Type d'enregistrement | Select | ✓ | "A - Adresse IPv4" |
| Sous-domaine | Input | Non | "@, www, blog, api..." |
| Cible | Input | ✓ | "123.45.67.89" |
| TTL | Input | Non | "3600" |

### Options Type
- A - Adresse IPv4
- AAAA - Adresse IPv6
- CNAME - Alias
- MX - Serveur mail
- TXT - Texte
- NS - Serveur de noms
- SRV - Service
- ...

### Valeur par défaut TTL
- 3600 secondes (1 heure)

## 6. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Style | Info (bleu #DBEAFE) |
| Icône | ℹ️ |
| Texte 1 | "La propagation DNS peut prendre jusqu'à 24 heures." |
| Texte 2 | "Vérifiez vos enregistrements avec un outil de diagnostic DNS." |

## 7. FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 8. VALIDATION
- Type obligatoire
- Cible obligatoire
- Format cible selon type (IP pour A, domaine pour CNAME...)

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record - Créer enregistrement
- POST /domain/zone/{zoneName}/refresh - Appliquer changements
