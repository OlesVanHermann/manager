# Cahier des Charges : web/email - Wizard créer un répondeur automatique

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire avec options de durée
- **Contexte** : Gestion email - Répondeurs
- **Objectif** : Créer une réponse automatique pour absences

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, labels, notes
- **OdsRadio** : Type de répondeur, durée
- **OdsSelect** : Compte email
- **OdsDatepicker** : Date début, date fin
- **OdsTextarea** : Message
- **OdsCheckbox** : Conserver copie
- **OdsButton** : Annuler, Créer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un répondeur automatique" |
| Description | "Configurez une réponse automatique pour vos absences." |
| Note | "Un seul email sera envoyé par expéditeur pendant la période active." |

## 5. TYPE DE RÉPONDEUR
| Option | Description |
|--------|-------------|
| Lié à un compte existant | Réponse liée à une boîte mail (sélectionné par défaut) |
| Adresse libre | Sans boîte mail associée |

## 6. SÉLECTION COMPTE
| Propriété | Valeur |
|-----------|--------|
| Label | "Compte *" |
| Type | OdsSelect + domaine fixe |
| Format | {compte} @ {domaine} |

## 7. DURÉE
| Option | Description |
|--------|-------------|
| Temporaire | Avec dates de début et fin (sélectionné par défaut) |
| Permanent | Toujours actif |

## 8. DATES (si temporaire)

### Date de début
| Propriété | Valeur |
|-----------|--------|
| Label | "Date de début *" |
| Type | OdsDatepicker avec heure |
| Format | YYYY-MM-DD HH:mm |

### Date de fin
| Propriété | Valeur |
|-----------|--------|
| Label | "Date de fin *" |
| Type | OdsDatepicker avec heure |
| Format | YYYY-MM-DD HH:mm |

## 9. OPTIONS

### Conserver copie
| Propriété | Valeur |
|-----------|--------|
| Checkbox | Coché par défaut |
| Label | "Conserver une copie des messages reçus" |

## 10. MESSAGE
| Propriété | Valeur |
|-----------|--------|
| Label | "Message *" |
| Type | OdsTextarea |
| Hauteur | 100px |
| Note | "Les balises HTML ne sont pas autorisées dans le message." |

### Exemple de message
```
Bonjour,
Je suis actuellement absent du bureau.
Je vous répondrai dès mon retour.
Cordialement.
```

## 11. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Créer | Primary (bleu) | Crée le répondeur |

## 12. APIS & ENDPOINTS
- GET /email/domain/{domain}/account - Liste des comptes
- GET /email/domain/{domain}/responder - Liste des répondeurs
- POST /email/domain/{domain}/responder - Créer répondeur
