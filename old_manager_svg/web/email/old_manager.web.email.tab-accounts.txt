# Cahier des Charges : web - Onglet Comptes Email

## 1. VUE D'ENSEMBLE
- **Page** : Liste des comptes email
- **Type** : Onglet avec datagrid
- **Objectif** : Gérer les comptes Email Pro

## 2. STRUCTURE PAGE
- Onglet "Comptes email" actif
- Titre + description
- Bouton création
- Datagrid des comptes

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Créer un compte
- **OdsDatagrid** : Liste comptes
- **OdsProgress** : Quota par compte
- **OdsBadge** : Statut
- **OdsActionMenu** : Actions

## 4. HEADER SECTION
| Élément | Valeur |
|---------|--------|
| Titre | "Comptes email" |
| Description | "Gérez les comptes email de votre service." |
| Bouton | "+ Créer un compte" (primary) |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Adresse email | Link bleu | Email complet |
| Nom affiché | Text | Prénom Nom |
| Quota utilisé | Progress + Text | Barre + "X Go / 10 Go" |
| État | Badge | Actif (vert) |
| Actions | Menu ⋮ | ActionMenu |

## 9. APIS & ENDPOINTS
- GET /email/pro/{service}/account - Liste comptes
- POST /email/pro/{service}/account - Créer compte
- GET /email/pro/{service}/account/{email} - Détails
- PUT /email/pro/{service}/account/{email} - Modifier
- DELETE /email/pro/{service}/account/{email} - Supprimer
- POST /email/pro/{service}/account/{email}/changePassword - Changer mdp
- GET /email/pro/{service}/account/{email}/alias - Liste alias
- GET /email/pro/{service}/account/{email}/fullAccess - Accès partagés
