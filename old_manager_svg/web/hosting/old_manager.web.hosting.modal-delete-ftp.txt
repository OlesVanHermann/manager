# Cahier des Charges : web/hosting - Modal supprimer utilisateur FTP

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation
- **Contexte** : Hébergement web - Utilisateurs FTP
- **Objectif** : Supprimer un utilisateur FTP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 250px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question, info
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'utilisateur FTP" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Êtes-vous sûr de vouloir supprimer l'utilisateur {ftpUser} ?" |

## 6. INFORMATION
| Texte |
|-------|
| "Cette action ne supprime pas les fichiers du répertoire associé." |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Supprimer | Danger (rouge) | Supprime l'utilisateur |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/user/{login} - Supprimer utilisateur FTP
