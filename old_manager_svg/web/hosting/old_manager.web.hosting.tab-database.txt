# Cahier des Charges : web - Onglet Bases de données

## 1. VUE D'ENSEMBLE
- **Page** : Gestion bases de données
- **Type** : Onglet avec quota et datagrid
- **Objectif** : Gérer les bases MySQL

## 2. STRUCTURE PAGE
- Onglet "Bases de données" actif
- Titre + description
- Carte quota
- Bouton création
- Datagrid des bases

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Quota utilisé
- **OdsProgress** : Barres quota
- **OdsButton** : Créer une base
- **OdsDatagrid** : Liste bases
- **OdsActionMenu** : Actions

## 4. CARTE QUOTA
| Élément | Valeur |
|---------|--------|
| Label | "Bases utilisées" |
| Valeur | "2 / 5" (bold) |
| Barre | 40% remplie |

## 5. BOUTON HEADER
| Propriété | Valeur |
|-----------|--------|
| Label | "+ Créer une base" |
| Style | Primary |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link bleu | Nom de la base |
| Serveur | Text | Adresse serveur MySQL |
| Type | Text | MySQL 8.0, MariaDB... |
| Taille | Text | Taille en Mo/Go |
| Quota | Progress + Text | Barre + pourcentage |
| Actions | Menu ⋮ | ActionMenu |

## 7. EXEMPLES DONNÉES
| Nom | Serveur | Type | Taille | Quota |
|-----|---------|------|--------|-------|
| monsite_db | mysql123.hosting.ovh.net | MySQL 8.0 | 156 Mo | 31% (500 Mo) |
| monsite_blog | mysql123.hosting.ovh.net | MySQL 8.0 | 89 Mo | 18% (500 Mo) |

## 8. ACTIONMENU OPTIONS
| Option | Description |
|--------|-------------|
| Accéder à phpMyAdmin | Ouvre phpMyAdmin |
| Modifier le mot de passe | Change le mot de passe |
| Exporter la base | Télécharge dump SQL |
| Importer une base | Upload dump SQL |
| --- | Séparateur |
| Supprimer | Supprimer la base |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/database - Liste bases
- POST /hosting/web/{serviceName}/database - Créer base
- GET /hosting/web/{serviceName}/database/{name} - Détails
- DELETE /hosting/web/{serviceName}/database/{name} - Supprimer
- POST /hosting/web/{serviceName}/database/{name}/dump - Exporter
- POST /hosting/web/{serviceName}/database/{name}/restore - Importer
- POST /hosting/web/{serviceName}/database/{name}/changePassword - Changer mdp
