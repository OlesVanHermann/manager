# Cahier des Charges : web/hosting - Modal activer le Boost

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire avec acceptation CGV
- **Contexte** : Hébergement web
- **Objectif** : Activer temporairement le Boost pour améliorer les performances

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, descriptions, prix
- **OdsSelect** : Choix offre Boost
- **OdsCheckbox** : Acceptation CGV
- **OdsButton** : Annuler, Activer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le Boost" |

## 5. DESCRIPTION
| Texte |
|-------|
| "Sélectionnez l'offre Boost pour améliorer les performances de votre hébergement." |

## 6. SÉLECTION OFFRE
| Propriété | Valeur |
|-----------|--------|
| Label | "Offre Boost" |
| Type | OdsSelect |
| Exemple | "Performance - 9,99 € HT/mois" |

## 7. INFORMATIONS BOOST
| Texte |
|-------|
| "Le Boost permet d'augmenter temporairement les ressources de votre hébergement pour faire face à un pic de trafic." |
| "Facturation : 0,33 € HT/jour" |
| "Le montant sera prélevé sur votre prochaine facture de renouvellement." |

## 8. ACCEPTATION CGV
| Propriété | Valeur |
|-----------|--------|
| Checkbox | Doit être coché pour activer |
| Label | "J'accepte les conditions générales de vente" |

## 9. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Activer | Primary (bleu) | Active le Boost |

## 10. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/boostHistory - Historique Boost
- POST /hosting/web/{serviceName}/requestBoost - Activer le Boost
