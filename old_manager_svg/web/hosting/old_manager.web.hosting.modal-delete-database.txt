# Cahier des Charges : web/hosting - Modal supprimer base de données

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation avec warning
- **Contexte** : Hébergement web - Bases de données
- **Objectif** : Supprimer une base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question
- **OdsMessage** : Warning irréversible
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la base de données" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Voulez-vous supprimer la base de données {databaseName} ?" |

## 6. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ Attention" |
| | "Cette action est irréversible. Toutes les données contenues dans cette base de données seront définitivement supprimées." |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Supprimer | Danger (rouge) | Supprime la base |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/database/{name} - Supprimer base de données
