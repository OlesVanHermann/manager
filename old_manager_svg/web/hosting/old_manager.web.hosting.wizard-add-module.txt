# Cahier des Charges : web/hosting - Wizard installer module en 1 clic

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard multi-étapes
- **Contexte** : Hébergement web - Modules
- **Objectif** : Installer un CMS (WordPress, PrestaShop, etc.) en 1 clic

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 750px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, labels, descriptions
- **OdsStepper** : Indicateur 4 étapes
- **OdsCard** : Cartes de sélection module
- **OdsSelect** : Domaine
- **OdsCheckbox** : Mode avancé
- **OdsMessage** : Info mode avancé
- **OdsButton** : Annuler, Installer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Installer un module en 1 clic" |

## 5. INDICATEUR D'ÉTAPES
| Étape | Description |
|-------|-------------|
| 1 | Choix du module (active) |
| 2 | Configuration |
| 3 | Base de données |
| 4 | Récapitulatif |

## 6. ÉTAPE 1 : CHOIX DU MODULE

### Question
| Texte |
|-------|
| "Quel type de module souhaitez-vous installer sur {domain} ?" |

### Grille de modules
| Module | Version | État |
|--------|---------|------|
| WordPress | 6.4.2 | Sélectionné (bordure bleue) |
| PrestaShop | 8.1.2 | Non sélectionné |
| Joomla | 5.0.1 | Non sélectionné |
| Drupal | 10.1.6 | Non sélectionné |

### Style cartes
| Propriété | Valeur |
|-----------|--------|
| Sélectionnée | Bordure bleue 2px |
| Non sélectionnée | Bordure grise |
| Dimensions | 160px × 100px |

## 7. SÉLECTION DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Label | "Domaine" |
| Type | OdsSelect |
| Exemple | www.example.com |

## 8. CHEMIN D'INSTALLATION
| Texte |
|-------|
| "Chemin d'installation par défaut : /www" |

## 9. MODE AVANCÉ

### Checkbox
| Propriété | Valeur |
|-----------|--------|
| Label | "Mode avancé" |
| État | Non coché par défaut |

### Message info (si coché)
| Type | Contenu |
|------|---------|
| Info (bleu) | "Le mode avancé vous permet de :" |
| | "• Choisir ou créer votre base de données" |
| | "• Définir le chemin d'installation" |
| | "• Personnaliser les identifiants administrateur" |

## 10. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Installer | Primary (bleu) | Lance l'installation |

## 11. APIS & ENDPOINTS
- GET /hosting/web/moduleList - Liste des modules disponibles
- GET /hosting/web/{serviceName}/attachedDomain - Domaines disponibles
- POST /hosting/web/{serviceName}/module - Installer module
