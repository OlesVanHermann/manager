# Cahier des Charges : web - Modal Ajout Multisite

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un domaine multisite
- **Type** : Formulaire création
- **Objectif** : Attacher un domaine à l'hébergement

## 2. STRUCTURE MODAL
- Dimensions : 600x600px
- Header avec titre
- Bannière info
- Formulaire avec radio/select/input
- Checkboxes options
- Footer boutons

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsMessage** : Bannière info
- **OdsRadio** : Type de domaine
- **OdsSelect** : Sélection domaine
- **OdsInput** : Sous-domaine, Dossier racine
- **OdsCheckbox** : Options SSL, Firewall
- **OdsButton** : Annuler, Ajouter

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Titre | "Ajouter un domaine" |
| Fond | Gris clair |
| Bouton fermer | ✕ |

## 5. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Style | Info (bleu) |
| Icône | ℹ️ |
| Texte 1 | "Attachez un domaine ou sous-domaine à votre hébergement" |

| Option | Coché | Description |
|--------|-------|-------------|

## 8. FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 9. APIS & ENDPOINTS
- GET /domain - Liste domaines du compte
- POST /hosting/web/{serviceName}/attachedDomain - Créer multisite
- POST /hosting/web/{serviceName}/attachedDomain/{domain}/regenerateSsl - Activer SSL
