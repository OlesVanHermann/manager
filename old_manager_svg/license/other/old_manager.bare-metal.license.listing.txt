# Cahier des Charges : bare-metal/license - Listing

## 1. VUE D'ENSEMBLE
- **Type** : Page listing licences
- **Contexte** : Gestion des licences logicielles
- **Objectif** : Vue d'ensemble et gestion des licences

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres
- **OdsButton** : Commander, Guides
- **OdsDatagrid** : Tableau licences
- **OdsBadge** : Statuts
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation pages

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Licences" |
| Menu Guides | Bouton dropdown "Guides ▼" |

## 5. BOUTON COMMANDER
| Propriété | Valeur |
|-----------|--------|
| Texte | "Commander une licence" |
| Style | Fond #0050D7, texte blanc |
| Dimensions | 180x40px |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Nom | 180px | Lien bleu vers détails |
| Type | 180px | Type licence (Plesk, Windows, etc.) |
| IP | 170px | Adresse IP associée |
| Expire le | 170px | Date (rouge si proche) |
| Statut | 200px | Badge statut |
| Actions | 100px | Menu ⋮ |

## 7. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Actif | #DCFCE7 | #16A34A |
| Expire bientôt | #FEF3C7 | #92400E |

## 8. MENU ACTIONS (ouvert)
| Action | Style |
|--------|-------|
| Voir détails | Texte #1F2937 |
| Migrer | Texte #1F2937 |
| Upgrade | Texte #1F2937 |
| Résilier | Texte #DC2626 |

## 9. EXEMPLES LICENCES
| Type | Description |
|------|-------------|
| Plesk Web Pro | Panel administration web |
| Windows Server 2022 Standard | OS serveur Microsoft |
| cPanel Premier | Panel hébergement |
| SQL Server Standard | Base de données Microsoft |

## 10. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "1 - 4 sur 12" |

## 11. APIS & ENDPOINTS
- GET /license/plesk - Liste licences Plesk
- GET /license/cpanel - Liste licences cPanel
- GET /license/windows - Liste licences Windows
- GET /license/sqlserver - Liste licences SQL Server
- POST /license/{type}/{serviceName}/changeIp - Migrer licence
- DELETE /license/{type}/{serviceName}/terminate - Résilier
