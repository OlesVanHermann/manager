## 9. APIS & ENDPOINTS

### 9.1 Obtenir les options d'upgrade

```
GET /license/sqlserver/{serviceName}/upgrade/options
```

Response:
```json
{
  "currentLicense": {
    "version": "2019",
    "edition": "standard",
    "coreNumber": 4
  },
  "availableUpgrades": [
    {
      "version": "2022",
      "edition": "standard",
      "coreNumber": 8,
      "priceDifference": 45.00
    }
  ]
}
```

### 9.2 Effectuer l'upgrade

```
POST /license/sqlserver/{serviceName}/upgrade
```

Body:
```json
{
  "version": "2022",
  "edition": "standard",
  "coreNumber": 8
}
```
