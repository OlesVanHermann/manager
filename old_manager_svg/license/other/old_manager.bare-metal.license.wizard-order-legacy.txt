## 8. APIS & ENDPOINTS

### 8.1 Obtenir les types de licence disponibles

```
GET /license/available
```

Response:
```json
{
  "licenseTypes": [
    {
      "type": "cpanel",
      "available": true
    },
    {
      "type": "plesk",
      "available": true
    },
    {
      "type": "directadmin",
      "available": false
    }
  ]
}
```

### 8.2 Commander une licence

```
POST /order/cart/{cartId}/license
```

Body:
```json
{
  "licenseType": "plesk",
  "version": "webpro",
  "duration": "P1M"
}
```
