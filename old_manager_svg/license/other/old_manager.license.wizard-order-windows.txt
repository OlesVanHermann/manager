## 8. APIS & ENDPOINTS

### 8.1 Obtenir les options Windows

```
GET /license/windows/options
```

Response:
```json
{
  "versions": [
    {"id": "2019-standard", "name": "Windows Server 2019 Standard"},
    {"id": "2019-datacenter", "name": "Windows Server 2019 Datacenter"},
    {"id": "2022-standard", "name": "Windows Server 2022 Standard"},
    {"id": "2022-datacenter", "name": "Windows Server 2022 Datacenter"}
  ],
  "coreNumbers": [8, 16, 32, 64]
}
```

### 8.2 Commander une licence Windows

```
POST /order/cart/{cartId}/windows
```

Body:
```json
{
  "ip": "192.168.1.105",
  "version": "2022-datacenter",
  "coreNumber": 32,
  "duration": "P1M"
}
```
