---

## 7. APIS & ENDPOINTS

### 7.1 Déplacer une licence

```
POST /license/{licenseType}/{serviceName}/changeIp
```

Body:
```json
{
  "destinationIp": "192.168.1.200"
}
```

Response:
```json
{
  "taskId": "12345",
  "status": "pending",
  "message": "License migration initiated"
}
```
