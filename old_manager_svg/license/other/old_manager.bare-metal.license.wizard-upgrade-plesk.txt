## 9. APIS & ENDPOINTS

### 9.1 Obtenir options Plesk

```
GET /license/plesk/{serviceName}/upgrade/options
```

Response:
```json
{
  "currentLicense": {
    "version": "web-admin",
    "domains": 10,
    "languagePacks": 0,
    "antivirus": false
  },
  "versions": ["web-admin", "web-pro", "web-host"],
  "domainOptions": [10, 30, 100, "unlimited"],
  "addons": {
    "languagePacks": [0, 1, 5],
    "antivirus": ["none", "kaspersky"],
    "powerPack": true
  }
}
```

### 9.2 Mettre à niveau Plesk

```
POST /license/plesk/{serviceName}/upgrade
```

Body:
```json
{
  "version": "web-pro",
  "domains": "unlimited",
  "languagePacks": 0,
  "antivirus": "none",
  "powerPack": false
}
```
