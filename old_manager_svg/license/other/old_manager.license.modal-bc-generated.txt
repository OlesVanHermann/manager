## 7. APIS & ENDPOINTS

### 7.1 Obtenir détails BC

```
GET /me/order/{orderId}
```

Response:
```json
{
  "orderId": "987654321",
  "date": "2025-01-15",
  "expirationDate": "2025-01-22",
  "status": "notPaid",
  "amount": {
    "value": 99.99,
    "currency": "EUR"
  }
}
```

### 7.2 URL de paiement

```
GET /me/order/{orderId}/pay
```

Response:
```json
{
  "url": "https://www.ovh.com/cgi-bin/order/pay.cgi?orderId=987654321"
}
```
