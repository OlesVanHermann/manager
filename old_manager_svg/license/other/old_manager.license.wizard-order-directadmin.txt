## 8. APIS & ENDPOINTS

### 8.1 Commander une licence DirectAdmin

```
POST /order/cart/{cartId}/directadmin
```

Body:
```json
{
  "ip": "192.168.1.103",
  "version": "pro",
  "os": "linux",
  "duration": "P1M"
}
```
