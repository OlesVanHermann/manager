## 7. APIS & ENDPOINTS

### 7.1 Obtenir historique SPLA

```
GET /license/spla/{serviceName}/history
```

Query params:
- `offset`: pagination
- `limit`: pagination

Response:
```json
{
  "licenseId": "SPLA-555666",
  "serviceName": "ns5432109.ip-192-168-1.eu",
  "history": [
    {
      "period": "2025-12",
      "product": "Windows Server 2022",
      "quantity": 3,
      "declarationDate": "2026-01-01",
      "amount": 89.99
    }
  ],
  "total": 4
}
```
