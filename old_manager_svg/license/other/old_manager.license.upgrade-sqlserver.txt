## 8. APIS & ENDPOINTS

### 8.1 Obtenir les options d'upgrade SQL Server

```
GET /license/sqlserver/{serviceName}/upgrade/options
```

Response:
```json
{
  "currentLicense": {
    "version": "2019",
    "edition": "standard",
    "cores": 4,
    "ip": "192.168.1.102"
  },
  "availableUpgrades": [
    {"version": "2019", "edition": "standard", "cores": 8, "price": 129.99},
    {"version": "2019", "edition": "enterprise", "cores": 8, "price": 299.99}
  ]
}
```
