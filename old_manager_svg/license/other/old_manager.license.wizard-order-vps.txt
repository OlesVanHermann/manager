## 9. APIS & ENDPOINTS

### 9.1 Obtenir les VPS disponibles

```
GET /license/vps/available
```

Response:
```json
{
  "vpsBlocks": [
    {
      "serviceName": "vps-abc123.vps.ovh.net",
      "ips": ["192.168.1.50", "192.168.1.51"]
    }
  ]
}
```

### 9.2 Obtenir les types de licence pour VPS

```
GET /license/vps/{serviceName}/available
```

Response:
```json
{
  "licenseTypes": [
    {
      "type": "cpanel",
      "versions": [
        {"id": "admin-vps", "name": "cPanel - Edition Admin pour VPS"}
      ]
    },
    {
      "type": "plesk",
      "versions": [
        {"id": "web-admin-vps", "name": "Plesk - Edition Web Admin pour VPS"}
      ]
    }
  ]
}
```

### 9.3 Commander une licence VPS

```
POST /order/cart/{cartId}/license/vps
```

Body:
```json
{
  "vpsServiceName": "vps-abc123.vps.ovh.net",
  "ip": "192.168.1.50",
  "licenseType": "plesk",
  "version": "web-admin-vps",
  "duration": "P1M"
}
```
