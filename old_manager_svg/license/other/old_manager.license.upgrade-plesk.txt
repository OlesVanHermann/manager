## 7. APIS & ENDPOINTS

### 7.1 Obtenir options upgrade Plesk

```
GET /license/plesk/{serviceName}/upgrade/options
```

Response:
```json
{
  "currentLicense": {
    "version": "web-pro",
    "domains": 30,
    "languagePacks": 0,
    "antivirus": false
  },
  "availableVersions": [
    {"id": "web-admin", "name": "Web Admin", "domains": 10},
    {"id": "web-pro", "name": "Web Pro", "domains": 30},
    {"id": "web-host", "name": "Web Host", "domains": "unlimited"}
  ],
  "addons": {
    "languagePacks": [0, 1, 5, 10],
    "antivirus": ["none", "kaspersky", "drweb"],
    "powerPack": true
  },
  "durations": ["P1M", "P12M", "P24M"]
}
```

### 7.2 Commander upgrade Plesk

```
POST /order/cart/{cartId}/license/plesk/{serviceName}/upgrade
```

Body:
```json
{
  "version": "web-host",
  "domains": "unlimited",
  "languagePacks": 0,
  "antivirus": "none",
  "powerPack": false,
  "duration": "P12M"
}
```
