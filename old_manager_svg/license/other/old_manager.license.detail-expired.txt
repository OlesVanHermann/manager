## 9. APIS & ENDPOINTS

### 9.1 Obtenir détails licence expirée

```
GET /license/{licenseType}/{serviceName}
```

Response:
```json
{
  "name": "cpanel-789012",
  "type": "cpanel",
  "version": "pro-cloud",
  "licenseId": "cpanel-789012-fr",
  "ip": "192.168.1.100",
  "serviceName": "ns1234567.ip-192-168-1.eu",
  "status": "expired",
  "creationDate": "2024-12-15",
  "expirationDate": "2025-12-15",
  "lastRenewal": "2024-12-15",
  "price": 9.99,
  "renewalMode": "manual",
  "history": [
    {"date": "2025-12-16T00:00:00Z", "event": "expired", "details": "Non renouvelée"}
  ]
}
```

### 9.2 Renouveler licence expirée

```
POST /order/cart/{cartId}/license/{licenseType}/{serviceName}/renew
```
