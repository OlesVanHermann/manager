## 10. APIS & ENDPOINTS

### 10.1 Valider l'upgrade

```
POST /license/{licenseType}/{serviceName}/upgrade/confirm
```

Body:
```json
{
  "version": "pro-cloud",
  "duration": "P12M",
  "acceptCgv": true,
  "acceptConditions": true
}
```
