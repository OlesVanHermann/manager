# Cahier des Charges : bare-metal/license - Changer OS DirectAdmin

## 1. VUE D'ENSEMBLE
- **Page** : Changer le système d'exploitation
- **Type** : Page formulaire
- **Objectif** : Changer l'OS associé à une licence DirectAdmin

## 2. STRUCTURE PAGE
- Titre + lien retour
- Banner info
- OS actuel (lecture seule)
- Sélection nouvel OS
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour à la gestion des licenses
- **OdsMessage** : Info (bleu)
- **OdsText** : OS actuel
- **OdsSelect** : Nouveau système d'exploitation
- **OdsButton** : Annuler (outline), Changer l'OS (primary)

## 4. INFO BANNER
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte | Sélectionnez le nouveau système d'exploitation pour votre licence DirectAdmin. |

## 5. OS ACTUEL
| Propriété | Valeur |
|-----------|--------|
| Label | Système d'exploitation actuel |
| Type | Texte lecture seule |
| Exemple | CentOS 7 |

## 6. SÉLECTION NOUVEL OS
| Propriété | Valeur |
|-----------|--------|
| Label | Nouveau système d'exploitation |
| Type | Select |
| Options | AlmaLinux 8, AlmaLinux 9, Rocky Linux 8, Rocky Linux 9 |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Changer l'OS | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /license/directadmin/{serviceName}/changeOs
  - Paramètre : newOs (string)
