# Cahier des Charges : bare-metal/license - Ajouter licence SPLA

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter une licence SPLA
- **Type** : Formulaire étapes
- **Objectif** : Ajouter une licence SPLA Microsoft sur un serveur

## 2. STRUCTURE PAGE
- Titre + lien retour
- Banner info explicatif
- Étape 1 : Choix serveur
- Étape 2 : Type et numéro de série
- Étape 3 : Confirmation
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour à la gestion des licenses
- **OdsMessage** : Info (bleu)
- **OdsSelect** : Serveur, Type
- **OdsInput** : Numéro de série
- **OdsText** : Récapitulatif
- **OdsButton** : Annuler (outline), Ajouter la licence SPLA (primary)

## 4. INFO BANNER
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte | Les licences SPLA permettent d'utiliser des logiciels Microsoft sur vos serveurs dédiés. |

| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter la licence SPLA | Primary | Droite |

## 9. APIS & ENDPOINTS
- GET /dedicated/server
  - Liste des serveurs
- POST /license/spla
  - Paramètres : serverServiceName, type, serialNumber
