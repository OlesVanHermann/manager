## 7. APIS & ENDPOINTS

### 7.1 Obtenir les détails de la licence

```
GET /license/directadmin/{serviceName}
```

Response:
```json
{
  "type": "directadmin",
  "version": "pro",
  "os": "linux",
  "ip": "192.168.1.103",
  "licenseId": "DIRECTADMIN-111222",
  "serviceName": "ns7654321.ip-192-168-1.eu",
  "expiration": "2025-08-01",
  "status": "active"
}
```

### 7.2 Obtenir les informations de renouvellement

```
GET /license/directadmin/{serviceName}/serviceInfos
```

Response:
```json
{
  "renew": "automatic",
  "nextBillingDate": "2025-08-01"
}
```

### 7.3 Changer l'OS

```
PUT /license/directadmin/{serviceName}/changeOs
```

Body:
```json
{
  "os": "linux"
}
```

### 7.4 Déplacer la licence (changer IP)

```
POST /license/directadmin/{serviceName}/changeIp
```

Body:
```json
{
  "destinationIp": "192.168.1.104"
}
```
