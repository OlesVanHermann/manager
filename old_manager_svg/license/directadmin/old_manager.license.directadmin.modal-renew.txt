# Cahier des Charges : licence/directadmin - Modal Renew

## 1. VUE D'ENSEMBLE
- **Type** : Modal renouvellement licence DirectAdmin
- **Contexte** : Licences DirectAdmin
- **Objectif** : Renouveler manuellement une licence

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 300px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsRadio** : Sélection durée
- **OdsButton** : Annuler, Renouveler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Renouveler DirectAdmin" |
| Bouton fermer | × |
