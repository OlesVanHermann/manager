## 7. APIS & ENDPOINTS

### 7.1 Obtenir les détails de la licence

```
GET /license/sqlserver/{serviceName}
```

Response:
```json
{
  "type": "sqlserver",
  "version": "2019",
  "edition": "standard",
  "coreNumber": 8,
  "ip": "192.168.1.102",
  "licenseId": "SQLSERVER-456789",
  "serviceName": "ns9876543.ip-192-168-1.eu",
  "expiration": "2025-12-10",
  "status": "active"
}
```

### 7.2 Upgrader la licence

```
POST /license/sqlserver/{serviceName}/upgrade
```

Body:
```json
{
  "edition": "enterprise",
  "coreNumber": 16
}
```

### 7.3 Déplacer la licence

```
POST /license/sqlserver/{serviceName}/changeIp
```

Body:
```json
{
  "destinationIp": "192.168.1.108"
}
```
