# Cahier des Charges : licence/sqlserver - Modal Renew

## 1. VUE D'ENSEMBLE
- **Type** : Modal renouvellement licence SQL Server
- **Contexte** : Licences SQL Server
- **Objectif** : Renouveler manuellement une licence SQL Server

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 320px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsRadio** : Sélection durée
- **OdsButton** : Annuler, Renouveler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Renouveler SQL Server" |
| Bouton fermer | × |

## 5. LICENCE ACTUELLE
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Label | "Licence actuelle" |
| Nom | "SQL Server {version} {édition}" |
| Détails | "{cores} cores • {serviceName}" |
| Prix | "€ XXX.XX" à droite |

## 6. SÉLECTION DURÉE
| Durée | Description |
|-------|-------------|
| 1 mois | Sélectionné par défaut |
| 12 mois | "Économisez 15%" en vert |

| État | Style |

| Label | "Montant à payer" gris |

| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Renouveler | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- GET /license/sqlserver/{serviceName} - Détail licence
- POST /order/license/sqlserver/{serviceName}/renew - Commander renouvellement
