## 7. APIS & ENDPOINTS

### 7.1 Obtenir détails licence Windows

```
GET /license/windows/{serviceName}
```

Response:
```json
{
  "type": "windows",
  "version": "2022-datacenter",
  "cores": 32,
  "ip": "192.168.1.105",
  "licenseId": "WINDOWS-654321",
  "serviceName": "ns9999999.ip-192-168-1.eu",
  "expirationDate": "2026-01-05",
  "status": "active",
  "renewal": {
    "mode": "automatic",
    "nextRenewal": "2026-01-05"
  },
  "configuration": {
    "edition": "datacenter",
    "coresAllocated": 32
  }
}
```
