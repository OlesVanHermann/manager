# Cahier des Charges : licence/windows - Modal Renew

## 1. VUE D'ENSEMBLE
- **Type** : Modal renouvellement licence Windows Server
- **Contexte** : Licences Windows Server
- **Objectif** : Renouveler manuellement une licence Windows

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 340px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsRadio** : Sélection durée
- **OdsMessage** : Info renouvellement
- **OdsButton** : Annuler, Renouveler

## 4. HEADER MODAL
