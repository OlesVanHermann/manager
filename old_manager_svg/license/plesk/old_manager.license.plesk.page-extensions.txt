# Cahier des Charges : licence/plesk - Page Extensions

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion des extensions Plesk
- **Contexte** : Licences Plesk
- **Objectif** : Gérer les extensions actives et en ajouter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour au détail
- **OdsCard** : Cartes extension
- **OdsBadge** : Badge "Actif"
- **OdsButton** : Actions extension

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour au détail de la licence" |
| Titre | "Extensions Plesk" |
| Sous-titre | Service name (monospace) |
| Bouton | "+ Ajouter une extension" |

## 5. EXTENSIONS ACTIVES
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Extensions actives (N)" |
| Layout | 3 colonnes |
| Largeur carte | 330px |
| Hauteur carte | 140px |

### Structure carte extension active
| Zone | Contenu |
|------|---------|
| Icône | 40×40px, couleur thématique |
| Nom | Titre de l'extension |
| Badge | "Actif" vert |
| Description | Courte description |
| Prix | "€ XX.XX HT/mois" ou "Inclus avec licence" |
| Expiration | "Expire: DD MMM YYYY" |
| Actions | "Désactiver" ou "Configurer" |

### Extensions affichées (exemples)
| Extension | Icône | Prix |
|-----------|-------|------|
| WordPress Toolkit | WP violet | € 8.00/mois |
| SSL It! | SSL bleu | Inclus |
| Imunify360 | IM rouge | € 12.00/mois |

## 8. APIS & ENDPOINTS
- GET /license/plesk/{serviceName} - Détail licence
- GET /license/plesk/{serviceName}/option - Extensions actives
- GET /order/catalog/formatted - Extensions disponibles
- POST /license/plesk/{serviceName}/option - Activer extension
- DELETE /license/plesk/{serviceName}/option/{option} - Désactiver
