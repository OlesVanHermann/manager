## 7. APIS & ENDPOINTS

### 7.1 Obtenir détails licence cPanel

```
GET /license/cpanel/{serviceName}
```

Response:
```json
{
  "type": "cpanel",
  "version": "pro-cloud",
  "ip": "192.168.1.100",
  "licenseId": "CPANEL-123456",
  "serviceName": "ns1234567.ip-192-168-1.eu",
  "creationDate": "2024-03-15",
  "expirationDate": "2025-03-15",
  "status": "active",
  "renewal": {
    "mode": "automatic",
    "nextRenewal": "2025-03-15"
  },
  "options": {
    "accounts": 100,
    "linkedHosting": "ns1234567.ip-192-168-1.eu"
  }
}
```
