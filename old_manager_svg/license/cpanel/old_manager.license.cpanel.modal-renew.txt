# Cahier des Charges : licence/cPanel - Modal Renew

## 1. VUE D'ENSEMBLE
- **Type** : Modal renouvellement licence cPanel
- **Contexte** : Licences cPanel
- **Objectif** : Renouveler manuellement une licence cPanel

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 320px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsRadio** : Sélection durée
- **OdsButton** : Annuler, Renouveler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Renouveler la licence cPanel" |
| Bouton fermer | × |

## 5. LICENCE SÉLECTIONNÉE
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Label | "Licence" |
| Format | "{Édition} - {serviceName}" |

## 6. SÉLECTION DURÉE
| Durée | Prix | Remise |
|-------|------|--------|
| 1 mois | Prix mensuel | - |
| 6 mois | Prix × 6 | - |
| 12 mois | Prix × 12 | -10% (vert) |

### Style sélection
| État | Style |
|------|-------|

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Renouveler | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- GET /license/cpanel/{serviceName} - Détail licence
- POST /order/license/cpanel/{serviceName}/renew - Commander renouvellement
