# Cahier des Charges : billing - Mes commandes

## 1. VUE D'ENSEMBLE
- **Page** : Mes commandes
- **Type** : Datagrid avec filtres
- **Objectif** : Consulter l'historique des commandes

## 2. STRUCTURE PAGE
- Header avec titre et description
- Filtres (année, statut)
- Tableau commandes
- Pagination

## 3. COMPOSANTS UI
- **OdsSelect** : Filtres année, statut
- **OdsDatagrid** : Tableau commandes
- **OdsBadge** : Statut commande
- **OdsLink** : Actions

## 4. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Année | Select | 2025, 2024, 2023... |
| Statut | Select | Tous, Livrée, En cours, Annulée |

## 5. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| N° Commande | Link | Oui | BC-2025-987654 |
| Date | Date | Oui | 26/12/2025 |
| Produits | Text | Non | VPS Starter (1 mois) |
| Montant TTC | Montant | Oui | 8,39 € |
| Statut | Badge | Oui | Livrée/En cours |
| Actions | Link | Non | Détails |

## 6. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| Livrée | Vert | Commande terminée |
| En cours | Bleu | En traitement |
| En attente | Orange | Attente paiement |
| Annulée | Rouge | Commande annulée |

## 7. PAGINATION
| Info | Valeur |
|------|--------|
| Affichage | 1-4 sur 32 commandes |

## 8. APIS & ENDPOINTS
- GET /me/order - Liste commandes
- GET /me/order?date.from={year} - Filtrer par année
- GET /me/order/{id} - Détail commande
- GET /me/order/{id}/status - Statut commande
