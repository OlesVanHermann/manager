# Cahier des Charges : billing - Moyens de paiement

## 1. VUE D'ENSEMBLE
- **Page** : Moyens de paiement
- **Type** : Cartes avec actions
- **Objectif** : Gérer les moyens de paiement du compte

## 2. STRUCTURE PAGE
- Header avec titre
- Bouton ajouter
- Cartes bancaires (grille)
- Section PayPal
- Section SEPA
- Compte prépayé OVHcloud

## 3. COMPOSANTS UI
- **OdsButton** : Ajouter moyen de paiement
- **OdsCard** : Carte paiement
- **OdsBadge** : Par défaut, Vérifié
- **OdsLink** : Actions

## 4. BOUTON HEADER
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter un moyen de paiement | Primary | Ouvre modal ajout |

## 9. APIS & ENDPOINTS
- GET /me/paymentMethod - Liste moyens de paiement
- POST /me/paymentMethod - Ajouter moyen paiement
- PUT /me/paymentMethod/{id}/default - Définir par défaut
- DELETE /me/paymentMethod/{id} - Supprimer
- GET /me/balance - Solde prépayé
