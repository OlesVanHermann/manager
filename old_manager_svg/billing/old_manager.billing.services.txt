# Cahier des Charges : billing - Mes services

## 1. VUE D'ENSEMBLE
- **Page** : Mes services
- **Type** : Datagrid avec ActionMenu
- **Objectif** : Gérer tous les services actifs

## 2. STRUCTURE PAGE
- Header avec titre
- Filtres (recherche, catégorie, statut)
- Tableau services
- ActionMenu par ligne
- Total mensuel estimé

## 3. COMPOSANTS UI
- **OdsInput** : Recherche
- **OdsSelect** : Filtres catégorie, statut
- **OdsDatagrid** : Tableau services
- **OdsBadge** : Renouvellement, Statut
- **OdsActionMenu** : Actions par service

## 4. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Recherche | Input | Rechercher un service... |
| Catégorie | Select | Toutes catégories, Hébergement, VPS... |
| Statut | Select | Tous statuts, Actif, Expiré |

## 5. COLONNES DATAGRID
| Colonne | Type | Tri |
|---------|------|-----|
| Service | Link | Oui |
| Catégorie | Text | Oui |
| Renouvellement | Badge | Oui |
| Expire le | Date (rouge si proche) | Oui |
| Statut | Badge | Oui |
| Prix | Montant/période | Oui |
| Actions | Menu ⋮ | Non |

## 6. BADGES
| Type | Couleur |
|------|---------|
| Automatique | Bleu |
| Manuel | Orange |
| Actif | Vert |

## 7. ACTION MENU
| Action | Description |
|--------|-------------|
| Voir le service | Ouvre le service |
| Gérer le renouvellement | Paramètres renouvellement |
| Anticiper le paiement | Payer en avance |
| Résilier à échéance | Résiliation programmée |
| Supprimer | Rouge, supprime le service |

## 8. FOOTER
| Info | Valeur |
|------|--------|
| Compteur | 5 services actifs |
| Total | Total mensuel estimé: 45,96 € |

## 9. APIS & ENDPOINTS
- GET /services - Liste tous les services
- GET /services?category={type} - Filtrer par catégorie
- PUT /services/{id}/renew - Modifier renouvellement
- DELETE /services/{id} - Résilier service
