# Cahier des Charges : billing - Modal résiliation de service

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information
- **Contexte** : Facturation
- **Objectif** : Informer l'utilisateur de contacter le support pour résilier

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message d'information
- **OdsButton** : Compris

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résiliation du service" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Pour résilier ce service, veuillez contacter le support au 1007 (appel gratuit)." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Compris | Danger (rouge) | Ferme modal |

## 7. COMPORTEMENT
- Ce modal s'affiche pour les services qui nécessitent un contact téléphonique
- Aucune action de résiliation directe possible
