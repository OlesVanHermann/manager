# Cahier des Charges : billing - Modal demande de remboursement

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Facturation
- **Objectif** : Demander un remboursement sur le moyen de paiement d'origine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description et montant
- **OdsButton** : Annuler, Valider la demande

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Demander un remboursement" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Vous êtes sur le point de procéder au remboursement de {montant}." |
| Info | "Vous serez remboursé sur le moyen de paiement d'origine." |
| Question | "Souhaitez-vous procéder au remboursement ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Valider la demande | Primary (bleu) | Demande le remboursement |

## 7. APIS & ENDPOINTS
- POST /me/refund - Demander un remboursement
- GET /me/payment/availableBalance - Solde disponible
