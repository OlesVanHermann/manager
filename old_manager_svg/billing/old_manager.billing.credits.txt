# Cahier des Charges : billing - Crédits & Avoirs

## 1. VUE D'ENSEMBLE
- **Page** : Crédits & Avoirs
- **Type** : Dashboard avec historique
- **Objectif** : Gérer les crédits et avoirs du compte

## 2. STRUCTURE PAGE
- Header avec titre
- 3 cartes (Solde, Recharger, Code promo)
- Historique des mouvements

## 3. COMPOSANTS UI
- **OdsCard** : Tuiles actions
- **OdsInput** : Code promo
- **OdsButton** : Actions
- **OdsDatagrid** : Historique
- **OdsBadge** : Type mouvement

## 4. CARTE SOLDE DISPONIBLE
| Élément | Valeur |
|---------|--------|
| Titre | Solde disponible |
| Montant | 45,00 € (vert, grand) |
| Description | Crédit utilisable sur vos prochaines factures |

## 5. CARTE AJOUTER CRÉDIT
| Élément | Type | Action |
|---------|------|--------|
| Titre | Text | Ajouter du crédit |
| Description | Text | Rechargez votre compte... |
| Bouton | Primary | Recharger |

## 6. CARTE CODE PROMO
| Élément | Type |
|---------|------|
| Titre | Text |
| Input | Text field |
| Bouton | Primary Valider |

## 7. HISTORIQUE DES MOUVEMENTS
| Colonne | Type | Description |
|---------|------|-------------|
| Date | Date | JJ/MM/AAAA |
| Description | Text | Avoir, Paiement, Code promo |
| Type | Badge | Crédit (vert) / Débit (rouge) |
| Montant | Montant coloré | + 25,00 € / - 50,00 € |
| Solde après | Montant | Solde résultant |

## 8. BADGES TYPE
| Type | Couleur | Exemple |
|------|---------|---------|
| Crédit | Vert | Avoir, Code promo |
| Débit | Rouge | Paiement facture |

## 9. APIS & ENDPOINTS
- GET /me/balance - Solde disponible
- POST /me/credit - Recharger compte
- POST /me/voucherCode - Appliquer code promo
- GET /me/credit/movements - Historique mouvements
