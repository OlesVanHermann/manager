# Cahier des Charges : billing - Liste des factures

## 1. VUE D'ENSEMBLE
- **Page** : Mes factures
- **Type** : Datagrid avec filtres
- **Objectif** : Consulter et télécharger les factures

## 2. STRUCTURE PAGE
- Header avec titre
- Zone filtres (période, statut, recherche)
- Bouton export CSV
- Tableau factures
- Pagination

## 3. COMPOSANTS UI
- **OdsDatepicker** : Filtre période
- **OdsSelect** : Filtre statut
- **OdsInput** : Recherche
- **OdsButton** : Export CSV
- **OdsDatagrid** : Tableau factures
- **OdsBadge** : Statut
- **OdsPagination** : Navigation

## 4. FILTRES
| Filtre | Type | Options/Placeholder |
|--------|------|---------------------|
| Période | Datepicker | 01/01/2025 - 31/12/2025 |
| État | Select | Tous, Payée, En attente, Impayée |
| Recherche | Input | N° facture... |

## 5. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| N° Facture | Link | Oui | FR12345678 |
| Date | Date | Oui | JJ/MM/AAAA |
| Période | Text | Oui | 01/12 - 31/12/2025 |
| Montant HT | Montant | Oui | 204,99 € |
| Montant TTC | Montant bold | Oui | 245,99 € |
| État | Badge | Oui | Payée/En attente/Impayée |
| Actions | Links | Non | PDF | Détails |

## 6. BADGES ÉTAT
| État | Couleur |
|------|---------|
| Payée | Vert |
| En attente | Orange |
| Impayée | Rouge |

## 7. PAGINATION
| Info | Format |
|------|--------|
| Total | Affichage 1-7 sur 24 factures |
| Pages | 1, 2, 3, 4 |

## 8. EXPORT CSV
| Colonnes exportées |
|-------------------|
| N° Facture, Date, Période, Montant HT, Montant TTC, État |

## 9. APIS & ENDPOINTS
- GET /me/bill - Liste factures
- GET /me/bill?date.from={}&date.to={} - Filtrer par période
- GET /me/bill/{id}/download - Télécharger PDF
- GET /me/bill/export - Export CSV
