# Cahier des Charges : bare-metal/cda - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Cloud Disk Array - Sommaire
- **Type** : Dashboard détail
- **Objectif** : Afficher les informations et santé d'un cluster CDA (CEPH)

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb
- Titre (nom du cluster)
- Onglets (Sommaire, Utilisateurs, Contrôle d'accès IP, Pools)
- 3 cartes en ligne (Détails, Santé, Utilisation)
- Section tâches actives

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud / Cloud Disk Array
- **OdsTabs** : 4 onglets
- **OdsCard** : Détails, Santé, Utilisation
- **OdsBadge** : Prêt (vert), En santé (vert)
- **OdsProgressBar** : Utilisation espace
- **OdsDatagrid** : Tableau tâches actives
- **OdsButton** : Modifier (icône ✎)

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Sommaire | Vue d'ensemble du cluster |
| Utilisateurs | Gestion des utilisateurs |
| Contrôle d'accès IP | Restrictions réseau |
| Pools | Gestion des pools de stockage |
