# Cahier des Charges : bare-metal/cda - Utilisateurs

## 1. VUE D'ENSEMBLE
- **Page** : Cloud Disk Array - Utilisateurs
- **Type** : Page liste onglet
- **Objectif** : Gérer les utilisateurs du cluster CEPH

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb + titre
- Onglets (Utilisateurs actif)
- Encart info
- Bouton ajouter
- Tableau des utilisateurs

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud / Cloud Disk Array
- **OdsTabs** : Utilisateurs actif
- **OdsMessage** : Info explicatif (bleu)
- **OdsButton** : Ajouter un utilisateur (primary)
- **OdsDatagrid** : Tableau des utilisateurs
- **OdsActionMenu** : Menu actions par ligne

## 4. ENCART INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | Vos utilisateurs peuvent utiliser le cluster avec des permissions spécifiques par pool. |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Text | Nom de l'utilisateur |
| Clé | Text (masked) | Clé d'accès tronquée |
| Pools | Text | Liste des pools autorisés |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTION MENU
| Option | Style |
|--------|-------|
| Voir les permissions | Normal (bleu) |
| Supprimer | Rouge (danger) |

## 7. BOUTON
| Bouton | Style |
|--------|-------|
| Ajouter un utilisateur | Primary |

## 8. APIS & ENDPOINTS
- GET /dedicated/ceph/{serviceName}/user
  - Liste des utilisateurs
- POST /dedicated/ceph/{serviceName}/user
  - Créer un utilisateur
- DELETE /dedicated/ceph/{serviceName}/user/{userName}
  - Supprimer un utilisateur
- GET /dedicated/ceph/{serviceName}/user/{userName}/pool
  - Permissions par pool
