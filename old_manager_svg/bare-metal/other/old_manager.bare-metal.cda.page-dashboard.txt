# Cahier des Charges : bare-metal/cda - Page Dashboard

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard principal Cloud Disk Array
- **Contexte** : Stockage Ceph distribué
- **Objectif** : Vue d'ensemble du cluster CDA (santé, espace, tâches)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 700px |
| Background | #F9FAFB |

- **OdsCard** : Tuiles dashboard
