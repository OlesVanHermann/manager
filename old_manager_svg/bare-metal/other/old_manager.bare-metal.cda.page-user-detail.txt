# Cahier des Charges : bare-metal/cda - Page User Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail utilisateur CDA
- **Contexte** : Cloud Disk Array - Gestion utilisateurs
- **Objectif** : Gérer les permissions par pool d'un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Lien retour
- **OdsText** : Titres et labels
- **OdsDatagrid** : Tableau permissions
- **OdsCheckbox** : Cases à cocher permissions
- **OdsButton** : Actions

## 11. APIS & ENDPOINTS
- GET /dedicated/ceph/{serviceName}/user/{userName} - Détails utilisateur
- GET /dedicated/ceph/{serviceName}/user/{userName}/pool - Permissions par pool
- PUT /dedicated/ceph/{serviceName}/user/{userName}/pool/{poolName} - Modifier permissions
