# Cahier des Charges : bare-metal/load-balancer - Page Frontends

## 1. VUE D'ENSEMBLE
- **Type** : Page listing frontends
- **Contexte** : Load Balancer
- **Objectif** : Gérer les frontends (points d'entrée)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajout frontend
- **OdsDatagrid** : Tableau frontends
- **OdsSwitch** : Toggle activation
- **OdsPopover** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Frontends" |
| Description | "Les frontends définissent les points d'entrée de votre Load Balancer. Configurez les ports et protocoles d'écoute." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter un frontend" |
| Style | Fond blanc, bordure #E5E7EB |
| Dimensions | 160x36px |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Frontend | 230px | Nom + (ID) |
| Protocole | 150px | HTTPS : port |
| Datacenter | 200px | Zone (code) |
| Ferme par défaut | 250px | Lien ferme (bleu) |
| Actif | 100px | Toggle switch |
| Actions | 60px | Menu ⋮ |

## 7. TOGGLE SWITCH
| État | Style |
|------|-------|
| ON | Fond #16A34A, cercle droite |
| OFF | Fond #E5E7EB, cercle gauche |

## 8. MENU ACTIONS (ouvert)
| Action | Style |
|--------|-------|
| Voir les détails | Texte #1F2937 |
| Modifier | Texte #1F2937 |
| Supprimer | Texte #DC2626 |

## 9. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "1 - 4 sur 4" |

## 10. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/http/frontend - Liste frontends HTTP
- GET /ipLoadbalancing/{serviceName}/tcp/frontend - Liste frontends TCP
- POST /ipLoadbalancing/{serviceName}/http/frontend - Créer frontend
- PUT /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Modifier
- DELETE /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Supprimer
