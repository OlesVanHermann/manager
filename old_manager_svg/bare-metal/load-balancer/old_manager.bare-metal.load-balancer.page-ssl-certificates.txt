# Cahier des Charges : bare-metal/load-balancer - Page SSL Certificates

## 1. VUE D'ENSEMBLE
- **Type** : Page listing certificats SSL
- **Contexte** : Load Balancer
- **Objectif** : Gérer les certificats SSL/TLS pour HTTPS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajouter certificat
- **OdsDatagrid** : Tableau certificats
- **OdsBadge** : État certificat
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|

|-----------|--------|

| Style | Fond #0050D7, texte blanc |

| Colonne | Largeur | Contenu |

## 11. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/ssl - Liste certificats
- POST /ipLoadbalancing/{serviceName}/ssl - Ajouter certificat
