## 11. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/vrack/network - Liste réseaux
- POST /ipLoadbalancing/{serviceName}/vrack/network - Ajouter réseau
- PUT /ipLoadbalancing/{serviceName}/vrack/network/{vrackNetworkId} - Modifier
- DELETE /ipLoadbalancing/{serviceName}/vrack/network/{vrackNetworkId} - Supprimer
- POST /ipLoadbalancing/{serviceName}/vrack/status - Activer/Désactiver vRack
