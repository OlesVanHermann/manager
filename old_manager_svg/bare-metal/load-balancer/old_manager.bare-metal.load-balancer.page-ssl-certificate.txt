# Cahier des Charges : bare-metal/load-balancer - Page SSL Certificates

## 1. VUE D'ENSEMBLE
- **Type** : Page listing certificats SSL
- **Contexte** : Load Balancer
- **Objectif** : Gérer les certificats SSL pour les frontends HTTPS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 450px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Actions (ajouter, commander)
- **OdsDatagrid** : Tableau certificats
- **OdsBadge** : Types certificats
- **OdsPopover** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Certificats SSL" |
| Description | "Gérez vos certificats SSL pour sécuriser les connexions HTTPS de vos frontends." |

## 5. BOUTONS ACTIONS
| Bouton | Texte |
|--------|-------|
| Ajouter | "+ Ajouter un certificat" |
| Commander | "Commander un certificat" |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Certificat | 280px | Nom + (ID) |
| Domaine (Subject) | 350px | CN du certificat |
| Type | 200px | Badge type |
| Expiration | 200px | Date (rouge si proche) |
| Actions | 60px | Menu ⋮ |

## 7. BADGES TYPE
| Type | Fond | Texte |
|------|------|-------|
| Let's Encrypt | #EFF6FF | #0050D7 |
| Custom | #DCFCE7 | #16A34A |

## 8. EXPIRATION
| État | Couleur |
|------|---------|
| Normal | #1F2937 |
| Proche expiration | #DC2626 + ⚠ |

## 9. MENU ACTIONS (ouvert)
| Action | Style |
|--------|-------|
| Voir les détails | Texte #1F2937 |
| Modifier | Texte #1F2937 |
| Supprimer | Texte #DC2626 |

## 10. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "1 - 3 sur 3" |

## 11. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/ssl - Liste certificats
- POST /ipLoadbalancing/{serviceName}/ssl - Ajouter certificat
- DELETE /ipLoadbalancing/{serviceName}/ssl/{id} - Supprimer
- POST /ipLoadbalancing/{serviceName}/ssl/letsencrypt/order - Commander Let's Encrypt
