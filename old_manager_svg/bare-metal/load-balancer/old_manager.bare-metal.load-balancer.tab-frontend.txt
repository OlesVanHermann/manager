# Cahier des Charges : bare-metal/load-balancer - Onglet Frontends

## 1. VUE D'ENSEMBLE
- **Page** : Liste des frontends
- **Type** : Onglet tableau
- **Objectif** : Gérer les points d'entrée du LB

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Frontends"
- Onglets navigation
- Bouton création
- Tableau frontends
- Bandeau warning configuration

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Créer un frontend
- **OdsDatagrid** : Tableau frontends
- **OdsBadge** : État, SSL
- **OdsActionMenu** : Menu par ligne
- **OdsMessage** : Warning configuration

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link | Nom du frontend |
| Port | Number | Port d'écoute |
| Protocole | Text | HTTP, HTTPS, TCP |
| Farm par défaut | Link | Farm associée ou Redirection |
| Certificat SSL | Text | Nom du certificat |
| Règles | Text | Nombre de règles |
| État | Badge | Actif (vert) |
| Actions | ActionMenu | Menu ⋮ |

## 5. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Modifier | Normal | Édite le frontend |
| Gérer les règles | Normal | Règles de routage |
| Configurer SSL | Normal | Certificat SSL |
| Désactiver | Normal | Désactive le frontend |
| Supprimer | Danger | Supprime le frontend |

## 6. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠ |
| Titre | Configuration en attente |
| Texte | Des modifications sont en attente d'application. Cliquez sur "Appliquer la configuration" pour les activer. |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/http/frontend
- POST /ipLoadbalancing/{serviceName}/http/frontend
- DELETE /ipLoadbalancing/{serviceName}/http/frontend/{frontendId}
