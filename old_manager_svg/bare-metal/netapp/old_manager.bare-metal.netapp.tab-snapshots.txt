# Cahier des Charges : bare-metal/netapp - Onglet Snapshots

## 1. VUE D'ENSEMBLE
- **Page** : Snapshots d'un volume
- **Type** : Onglet avec tableau et sélection politique
- **Objectif** : Gérer les snapshots du volume

## 2. STRUCTURE PAGE
- Titre "Snapshots"
- Toolbar avec menu Actions
- Tableau des snapshots
- Section sélection politique
- Bouton appliquer

## 3. COMPOSANTS UI
- **OdsButton** : Actions dropdown
- **OdsDatagrid** : Tableau snapshots
- **OdsBadge** : Statut snapshot
- **OdsTile** : Cartes politiques
- **OdsActionMenu** : Menu actions ligne

## 4. COLONNES DATAGRID SNAPSHOTS
| Colonne | Type | Description |
|---------|------|-------------|
| ID | Text | Identifiant snapshot |
| Nom | Text | Nom du snapshot |
| Description | Text | Description |
| Créé le | Date | Date création |
| Chemin | Text | Chemin d'accès |
| Taille | Text | Taille du snapshot |
| Type | Text | Automatique/Manuel |
| Statut | Badge | available |
| Actions | ActionMenu | Menu ⋮ |

## 5. ACTIONMENU SNAPSHOT (⋮)
| Action | Description |
|--------|-------------|
| Restaurer | Restaure le snapshot |
| Supprimer | Supprime le snapshot |

## 6. SECTION POLITIQUES DE SNAPSHOT
Affiche les politiques disponibles en cartes cliquables.

### Carte politique sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Background | Bleu clair |
| Border | Bleu |
| Contenu | Nom + rétention (Quotidien, Hebdo, Mensuel) |

### Carte politique non sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Background | Blanc |
| Border | Gris |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Actions ▼ | Secondary | Dropdown |
| Appliquer la politique | Primary | Applique politique sélectionnée |

## 8. ACTIONS DROPDOWN
| Action | Description |
|--------|-------------|
| Créer un snapshot | Crée snapshot manuel |
| Supprimer les snapshots | Supprime snapshots sélectionnés |

## 9. APIS & ENDPOINTS
- GET /storage/netapp/{serviceName}/share/{shareId}/snapshot
  - Liste des snapshots
- POST /storage/netapp/{serviceName}/share/{shareId}/snapshot
  - Créer snapshot manuel
- DELETE /storage/netapp/{serviceName}/share/{shareId}/snapshot/{snapshotId}
  - Supprimer snapshot
- PUT /storage/netapp/{serviceName}/share/{shareId}
  - Modifier politique snapshot
