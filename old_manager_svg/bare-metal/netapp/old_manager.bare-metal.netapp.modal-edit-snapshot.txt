## 6. APIS & ENDPOINTS

### 6.1 Modifier un snapshot

```
PUT /storage/netapp/{serviceName}/share/{shareId}/snapshot/{snapshotId}
{
  "name": "snapshot-2026-01-02",
  "description": "Backup quotidien avant mise à jour"
}
```
