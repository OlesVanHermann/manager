# Cahier des Charges : bare-metal/netapp - Modal Volume Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création volume
- **Contexte** : Stockage NetApp
- **Objectif** : Créer un nouveau volume de stockage
