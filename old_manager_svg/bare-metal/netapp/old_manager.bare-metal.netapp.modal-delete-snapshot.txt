# Cahier des Charges : bare-metal/netapp - Modal Supprimer Snapshot

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Gestion volumes NetApp
- **Objectif** : Supprimer un snapshot existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsDefinitionList** : Détails du snapshot
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer un snapshot" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |
| Bouton fermer | × |

## 5. CONTENU

### Question
| Propriété | Valeur |
|-----------|--------|
| Texte | "Êtes-vous sûr de vouloir supprimer le snapshot suivant ?" |

### Détails du snapshot
| Champ | Valeur exemple |
|-------|----------------|
| Snapshot ID | snap-abc123def456 |
| Nom | backup-2024-12-29 |
| Description | Sauvegarde avant migration |

### Style labels
| Propriété | Valeur |
|-----------|--------|
| Label | 12px, #6B7280, font-weight 500 |
| Valeur | 13px, #1F2937 |

## 6. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Supprimer | Primary danger | #DC2626 | Supprime le snapshot |

## 7. APIS & ENDPOINTS
- DELETE /storage/netapp/{serviceName}/volume/{volumeId}/snapshot/{snapshotId} - Supprimer snapshot
