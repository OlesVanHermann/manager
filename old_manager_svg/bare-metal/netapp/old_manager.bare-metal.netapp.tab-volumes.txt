# Cahier des Charges : bare-metal/netapp - Onglet Volumes

## 1. VUE D'ENSEMBLE
- **Page** : Liste des volumes d'un service NetApp
- **Type** : Onglet tableau
- **Objectif** : Gérer les volumes du service

## 2. STRUCTURE PAGE
- Titre "Volumes"
- Toolbar avec bouton création
- Tableau des volumes
- Pagination

## 3. COMPOSANTS UI
- **OdsButton** : Créer un volume
- **OdsDatagrid** : Tableau volumes
- **OdsBadge** : Statut volume
- **OdsProgressBar** : Utilisation
- **OdsLink** : Liens vers détail
- **OdsActionMenu** : Menu actions
- **OdsPagination** : Navigation

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link | Nom du volume (cliquable) |
| Description | Text | Description du volume |
| Taille | Text | Capacité du volume |
| Utilisation | ProgressBar | Pourcentage utilisé |
| Protocole | Text | NFS version |
| Statut | Badge | État du volume |
| Actions | ActionMenu | Menu contextuel |

## 5. ACTIONMENU (⋮)
| Action | Description |
|--------|-------------|
| Voir le détail | Ouvre page détail volume |
| Créer un snapshot | Crée snapshot manuel |
| Gérer les ACL | Ouvre onglet ACL |
| Redimensionner | Ouvre modal resize |
| Supprimer | Ouvre modal suppression |

## 6. BOUTON TOOLBAR
| Bouton | Style | Position |
|--------|-------|----------|
| + Créer un volume | Primary | Haut droite |

## 7. APIS & ENDPOINTS
- GET /storage/netapp/{serviceName}/share
  - Liste des volumes
- POST /storage/netapp/{serviceName}/share
  - Créer un volume
- DELETE /storage/netapp/{serviceName}/share/{shareId}
  - Supprimer un volume
