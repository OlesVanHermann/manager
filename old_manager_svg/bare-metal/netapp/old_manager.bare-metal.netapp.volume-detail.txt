# Cahier des Charges : bare-metal/netapp - Détail volume

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'un volume NetApp
- **Type** : Page détail avec onglets
- **Objectif** : Voir et gérer un volume spécifique

## 2. STRUCTURE PAGE
- Breadcrumb retour aux volumes
- Titre du volume avec description
- Onglets : Informations générales | Snapshots | ACL | NFS Actifs
- Contenu selon onglet actif

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes informations
- **OdsBadge** : Statut volume
- **OdsProgressBar** : Utilisation
- **OdsButton** : Actions

## 4. ONGLET INFORMATIONS GÉNÉRALES

### Carte Volume
| Propriété | Valeur |
|-----------|--------|
| Nom | prod-data |
| Description | Données production |
| ID | vol-abc123 |
| Statut | Actif (badge vert) |

### Carte Stockage
| Propriété | Valeur |
|-----------|--------|
| Taille | 2 To |
| Utilisé | 1.5 To (75%) |
| Disponible | 500 Go |

### Carte Montage
| Propriété | Valeur |
|-----------|--------|
| Chemin | /vol/prod-data |
| Point de montage NFS | 10.0.0.1:/vol/prod-data |
| Protocole | NFS v4.1 |

### Carte Snapshot Policy
| Propriété | Valeur |
|-----------|--------|
| Politique | default |
| Quotidien | 7 snapshots |
| Hebdomadaire | 4 snapshots |
| Mensuel | 6 snapshots |

## 5. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Créer un snapshot | Primary | Crée snapshot manuel |
| Modifier | Outline | Ouvre modal édition |
| Supprimer | Danger | Ouvre modal suppression |

## 6. APIS & ENDPOINTS
- GET /storage/netapp/{serviceName}/share/{shareId}
  - Détails du volume
- GET /storage/netapp/{serviceName}/share/{shareId}/snapshot
  - Liste des snapshots
- GET /storage/netapp/{serviceName}/share/{shareId}/acl
  - Liste des ACL
