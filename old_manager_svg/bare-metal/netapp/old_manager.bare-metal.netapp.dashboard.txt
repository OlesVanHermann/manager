# Cahier des Charges : bare-metal/netapp - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard d'un service NetApp
- **Type** : Page détail avec onglets
- **Objectif** : Voir les informations générales et gérer le service

## 2. STRUCTURE PAGE
- Breadcrumb avec retour liste
- Titre du service avec description
- Onglets : Informations générales | Volumes | Snapshot Policies
- Contenu selon onglet actif

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes informations
- **OdsBadge** : Statut service
- **OdsProgressBar** : Utilisation stockage
- **OdsButton** : Actions

## 5. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Créer un volume | Primary | Ouvre modal création |
| Modifier | Outline | Modifier description |
| Résilier | Danger | Ouvre modal résiliation |

## 6. APIS & ENDPOINTS
- GET /storage/netapp/{serviceName}
  - Détails du service
- GET /storage/netapp/{serviceName}/share
  - Liste des volumes
