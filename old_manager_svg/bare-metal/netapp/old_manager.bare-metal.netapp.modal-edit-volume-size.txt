# Cahier des Charges : bare-metal/netapp - Modal modifier taille volume

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : NetApp ONTAP Storage
- **Objectif** : Modifier la taille d'un volume NetApp

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, aide, info
- **OdsInput** : Champ numérique avec stepper
- **OdsButton** : -, +, Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la taille du volume" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champ Nouvelle taille
| Propriété | Valeur |
|-----------|--------|
| Label | "Nouvelle taille pour \"{volumeName}\" (Go)" |
| Exemple | "vol-production" |
| Aide | "Min: 100 Go - Max: 10000 Go" |
| Type | Input numérique avec stepper |

### Composant Stepper
| Élément | Description |
|---------|-------------|
| Bouton - | Décrémente la valeur |
| Input | Valeur numérique (ex: 2000) |
| Bouton + | Incrémente la valeur |
| Unité | "Go" |
| Style input actif | Bordure bleue (#0050D7) |

### Information taille actuelle
| Propriété | Valeur |
|-----------|--------|
| Texte | "Taille actuelle: {currentSize} Go" |
| Exemple | "Taille actuelle: 1000 Go" |
| Style | Gris (#6B7280), petit |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary (bleu) | Applique la nouvelle taille |

## 7. APIS & ENDPOINTS
- PUT /storage/netapp/{serviceName}/share/{shareId} - Modifier taille volume
