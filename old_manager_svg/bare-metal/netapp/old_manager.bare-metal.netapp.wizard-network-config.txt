# Cahier des Charges : bare-metal/netapp - Wizard Network Config

## 1. VUE D'ENSEMBLE
- **Type** : Wizard configuration réseau
- **Contexte** : Stockage NetApp
- **Objectif** : Configurer la connexion vRack pour le stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Lien retour
- **OdsText** : Titres et descriptions
- **OdsStepper** : Indicateur étapes
- **OdsSelect** : Dropdowns
- **OdsButton** : Actions

## 4. NAVIGATION
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour" |
| Couleur | #0050D7 |

## 5. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration réseau" |
| Description | "Configurez la connexion vRack pour votre stockage NetApp." |

## 6. STEPPER
| Étape | État | Style |
|-------|------|-------|
| 1. Sélection du vRack | Completed | Cercle vert + coche blanche |
| 2. vRack Services et Sous-réseau | Active | Cercle bleu #0050D7 + numéro |

## 7. STYLE STEPPER
| Propriété | Valeur |
|-----------|--------|
| Cercle taille | 40px |
| Connecteur | Ligne 2px |
| Completed | #16A34A |

| Pending | #E5E7EB |

## 8. CHAMPS FORMULAIRE

### vRack Services
| Propriété | Valeur |
|-----------|--------|
| Label | "vRack Services" |
| Type | Select dropdown |
| Focus | Bordure #0050D7 |
| Helper | "Pas de vRack Services ?" + Bouton "Créer" |

### Sous-réseau
| Propriété | Valeur |
|-----------|--------|
| Label | "Sous-réseau" |
| Type | Select dropdown |
| Format | "{CIDR} - {nom}" |

## 9. FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Fond blanc, bordure grise | Gauche |
| Configurer | Fond #0050D7, texte blanc | Droite |

## 10. APIS & ENDPOINTS
- GET /vrackServices - Liste vRack Services disponibles
- GET /vrackServices/{serviceName}/subnet - Liste sous-réseaux
- POST /storage/netapp/{serviceName}/vrackServices - Associer vRack Services
