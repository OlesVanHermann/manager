# Cahier des Charges : bare-metal/cdn - Wizard Modifier Backend

## 1. VUE D'ENSEMBLE
- **Type** : Wizard 2 étapes
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Modifier le backend associé à un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 800px |
| Page hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsStepper** : 2 étapes
- **OdsSelect** : Backend existant
- **OdsInput** : Nouveau backend (IPv4)
- **OdsText** : Backend actuel
- **OdsButton** : Annuler, Suivant

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le backend" |

## 5. STEPPER (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Sélection | Active |
| 2 | Confirmation | À venir |

## 6. CONTENU ÉTAPE 1 - SÉLECTION
| Propriété | Valeur |
|-----------|--------|
| Largeur container | 680px |
| Hauteur | 260px |

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sélectionnez le backend pour le domaine {domainName}" |
| Style domaine | Bleu #0050D7, font-weight 500 |

### Backend actuel
| Propriété | Valeur |
|-----------|--------|
| Label | "Backend actuel :" |
| Valeur | IPv4 actuelle (ex: 192.168.1.10) |

### Backend existant (select)
| Propriété | Valeur |
|-----------|--------|
| Label | "Backend existant" |
| Type | OdsSelect |
| Largeur | 300px |
| Placeholder | "Sélectionner un backend..." |

### Nouveau backend (input)
| Propriété | Valeur |
|-----------|--------|
| Label | "Nouveau backend" |
| Type | OdsInput |
| Largeur | 300px |
| Exemple | 10.0.0.50 |

## 7. FOOTER
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Secondary | Gauche | Annule et ferme |
| Suivant | Primary (#0050D7) | Droite | Passe à la confirmation |

## 8. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/backends - Liste des backends
- PUT /cdn/dedicated/{serviceName}/domains/{domainName} - Modifier backend domaine
