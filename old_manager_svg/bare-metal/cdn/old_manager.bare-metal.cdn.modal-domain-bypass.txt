# Cahier des Charges : bare-metal/cdn - Modal Bypass Domaine

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation (warning)
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Confirmer le bypass temporaire du CDN pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 600px |
| Page hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Bypass

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bypass CDN pour le domaine" |

## 5. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Titre | "Attention" |
| Texte | "Le domaine {domainName} sera temporairement bypassé du CDN." |
| Couleur texte | #92400E |

## 6. FOOTER
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Bypass | Primary warning | #F59E0B | Active le bypass |

## 7. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domainName}/bypass - Active le bypass
