# Cahier des Charges : bare-metal/cdn - Wizard ajouter règle de cache

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes
- **Contexte** : CDN Infrastructure
- **Objectif** : Créer une nouvelle règle de cache pour un domaine CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étapes
- **OdsText** : Labels, descriptions
- **OdsSelect** : Type de fichier
- **OdsInput** : Pattern, TTL
- **OdsRadio** : Type de cache
- **OdsButton** : Annuler, Suivant

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une règle de cache" |

## 5. STEPPER

### Étapes
| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Active (bleu) |
| 2 | Confirmation | Inactive (gris) |

## 6. FORMULAIRE - ÉTAPE 1 (CONFIGURATION)

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Configurez les paramètres de la règle de cache." |

### Champ Type de fichier
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de fichier" |
| Type | OdsSelect |
| Options | Extension, URI, Header |
| Défaut | "Extension" |

### Champ Pattern de fichier
| Propriété | Valeur |
|-----------|--------|
| Label | "Pattern de fichier" |
| Type | OdsInput |
| Placeholder | "*.jpg, *.png, *.gif" |

### Champ Type de cache
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de cache" |
| Type | OdsRadio group |
| Options | Cache (sélectionné), Pas de cache |

### Champ TTL
| Propriété | Valeur |
|-----------|--------|
| Label | "TTL (durée de vie)" |
| Type | OdsInput (number) + unité |
| Exemple | "3600" |
| Unité | "secondes" |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Suivant | Primary (bleu) | Passe à l'étape 2 |

## 8. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules - Créer règle
- GET /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules - Liste règles existantes
