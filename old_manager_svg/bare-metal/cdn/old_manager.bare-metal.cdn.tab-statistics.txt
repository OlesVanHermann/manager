# Cahier des Charges : bare-metal/cdn - Onglet Statistiques

## 1. VUE D'ENSEMBLE
- **Page** : Statistiques du CDN
- **Type** : Onglet dashboard graphiques
- **Objectif** : Visualiser les métriques de performance

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Statistiques"
- Onglets navigation
- Filtres période/domaine
- Cartes KPI
- Graphique requêtes
- Graphiques répartition

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsSelect** : Filtres période, domaine
- **OdsCard** : Cartes KPI
- **OdsChart** : Graphiques (line, pie, bar)
- **OdsMessage** : Comparaison vs hier

## 4. FILTRES
| Filtre | Options |
|--------|---------|
| Période | Dernières 24h, 7 jours, 30 jours, Personnalisé |
| Domaine | Tous les domaines, domaine spécifique |

## 5. CARTES KPI
| Métrique | Valeur | Comparaison |
|----------|--------|-------------|
| Requêtes totales | 1.2M | ↑ +15% vs hier |
| Bande passante | 456 GB | ↑ +8% vs hier |
| Cache Hit Ratio | 94.2% | ↑ +2% vs hier |
| Latence moyenne | 23 ms | ↓ -5ms vs hier |

## 6. GRAPHIQUE REQUÊTES PAR HEURE
| Propriété | Valeur |
|-----------|--------|
