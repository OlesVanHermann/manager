# Cahier des Charges : bare-metal/cdn - Modal Ajouter Certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Importer un certificat SSL personnalisé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 570px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom du certificat
- **OdsTextarea** : Certificat, Clé privée, Chaîne
- **OdsButton** : Fermer, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un certificat SSL" |
| Fond | #F9FAFB |
| Bouton fermer | × |
