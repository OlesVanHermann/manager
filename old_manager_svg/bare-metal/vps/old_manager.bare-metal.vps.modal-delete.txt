# Cahier des Charges : bare-metal/vps - Modal Delete

## 1. VUE D'ENSEMBLE
- **Modal** : Résilier le VPS
- **Type** : Confirmation dangereuse avec saisie
- **Objectif** : Confirmer la suppression définitive

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bannière danger (rouge)
- Info VPS concerné
- Champ confirmation DELETE
- Boutons Annuler / Résilier

## 3. COMPOSANTS UI
- **OdsModal** : Container (600x400px)
- **OdsMessage** : Bannière danger (fond rouge)
- **OdsText** : Info VPS
- **OdsInput** : Saisie "DELETE"
- **OdsButton** : Annuler (outline), Résilier le VPS (danger)

## 4. BANNIÈRE DANGER
| Style | Contenu |
|-------|---------|
| Fond rouge | ⚠️ Action irréversible |
| Bordure rouge | Cette action supprimera définitivement votre VPS et toutes ses données. |
| | Les snapshots et sauvegardes seront également supprimés. |

## 5. VPS CONCERNÉ
| Élément | Valeur |
|---------|--------|
| Label | VPS concerné |
| Fond gris | vps-abc12345.vps.ovh.net |

## 6. CONFIRMATION
| Élément | Valeur |
|---------|--------|
| Label | Confirmation |
| Aide | Tapez DELETE pour confirmer la résiliation |
| Valeur | DELETE (rouge) |

## 7. BOUTONS
| Bouton | Style | Couleur |
|--------|-------|---------|
| Annuler | Outline | Gris |
| Résilier le VPS | Primary | Rouge (danger) |

## 8. APIS & ENDPOINTS
- DELETE /vps/{serviceName}
  - Suppression définitive du VPS
