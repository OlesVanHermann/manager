# Cahier des Charges : bare-metal/vps - Modal Mot de passe

## 1. VUE D'ENSEMBLE
- **Modal** : Réinitialiser le mot de passe
- **Type** : Confirmation avec avertissement
- **Objectif** : Réinitialiser le mot de passe root

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message confirmation
- Avertissement (texte rouge)
- Information envoi email
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Container (600x340px)
- **OdsText** : Messages
- **OdsMessage** : Avertissement danger
- **OdsButton** : Annuler (outline), Confirmer (primary)

## 4. CONTENU
### Message principal
"Êtes-vous sûr de vouloir réinitialiser le mot de passe root de votre VPS ?"

### Avertissement (rouge)
"⚠️ Attention : votre VPS sera redémarré pour appliquer le changement de mot de passe."

### Information
"Le nouveau mot de passe vous sera envoyé par e-mail à l'adresse associée à votre compte OVHcloud."

## 5. BOUTONS
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Confirmer | Primary |

## 6. APIS & ENDPOINTS
- POST /vps/{serviceName}/setPassword
  - Génère un nouveau mot de passe
- POST /vps/{serviceName}/reboot
  - Redémarre le VPS
