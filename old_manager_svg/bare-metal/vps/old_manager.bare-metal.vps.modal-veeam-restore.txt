---

## 6. APIS & ENDPOINTS

### 6.1 Restaurer le VPS depuis backup

```
POST /vps/{serviceName}/automatedBackup/restore
{
  "restorePoint": "2026-01-01T14:30:00Z",
  "changePassword": true|false
}
```

### 6.2 Lister les points de restauration

```
GET /vps/{serviceName}/automatedBackup/restorePoints
```
