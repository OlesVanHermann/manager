# Cahier des Charges : bare-metal/vps - Upscale (Ajouter ressources)

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter des ressources
- **Type** : Page avec slider de configuration
- **Objectif** : Augmenter RAM/CPU sans changer de gamme

## 2. STRUCTURE PAGE
- Titre "Ajouter des ressources"
- Sous-titre explicatif
- Bandeau warning interruption
- Comparaison configuration (avant/après)
- Slider mémoire
- Résumé prix
- Boutons action

## 3. COMPOSANTS UI
- **OdsMessage** : Bandeau warning
- **OdsCard** : Configuration actuelle
- **OdsCard** : Nouvelle configuration (highlight)
- **OdsSlider** : Sélection mémoire
- **OdsButton** : Annuler, Valider et payer
