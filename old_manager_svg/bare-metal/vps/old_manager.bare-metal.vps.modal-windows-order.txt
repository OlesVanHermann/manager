## 7. APIS & ENDPOINTS

### 7.1 Commander l'option Windows

```
POST /vps/{serviceName}/option
{
  "option": "windows"
}
```

### 7.2 Obtenir le prix de l'option

```
GET /order/cart/{cartId}/vps/options
```

### 7.3 Vérifier la compatibilité

```
GET /vps/{serviceName}/availableUpgrade
```
