# Cahier des Charges : vps - Dashboard VPS

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard VPS avec tabs
- **Type** : Vue détaillée avec onglets
- **Objectif** : Gérer un VPS spécifique

## 2. STRUCTURE PAGE
- Badge gamme (Starter/Value/Essential)
- Titre : nom DNS du VPS
- Sous-titre : VPS • Datacenter
- Bouton Actions (dropdown)
- Onglets de navigation
- 3 cartes de configuration
- Zone actions rapides

## 3. COMPOSANTS UI
- **OdsBadge** : Gamme, Statut
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tuiles info
- **OdsProgress** : Utilisation ressources
- **OdsButton** : Actions rapides

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Accueil | Dashboard principal |
| Snapshots | Gestion snapshots |
| Backup automatique | Sauvegardes |
| Reinstallation | Réinstaller OS |
| Monitoring | Métriques |
| DNS secondaire | Config DNS |

## 7. CARTE RESSOURCES
| Métrique | Valeur | Progress |
|----------|--------|----------|
| CPU | 30% | Barre bleue |
| RAM | 50% (2 Go / 4 Go) | Barre bleue |
| Disque | 40% (32 Go / 80 Go) | Barre bleue |
| Uptime | 12 jours, 5 heures | - |

## 8. ACTIONS RAPIDES
| Bouton | Style | Action |
|--------|-------|--------|
| Redémarrer | Primary | Reboot soft |
| Créer un snapshot | Default | Snapshot manuel |
| Réinstaller | Default | Modal réinstall |
| Console VNC | Default | Accès KVM |
| Éteindre | Danger | Arrêt VPS |

## 9. APIS & ENDPOINTS
- GET /vps/{serviceName} - Infos VPS
- GET /vps/{serviceName}/monitoring - Métriques
- POST /vps/{serviceName}/reboot - Redémarrer
- POST /vps/{serviceName}/stop - Arrêter
