## 4. CONTENU

### 4.1 Description

| Texte |
|-------|
| Veuillez choisir le block d'ip que vous souhaitez autoriser ainsi que ses options d'accès. |

### 4.2 Sélection Block IP

| Champ | Type | Description |
|-------|------|-------------|
| Block IP | Multi-select | Liste des blocks IP disponibles |

| Aide |
|------|
| Sélectionnez un ou plusieurs blocks IP dans la liste. |

### 4.3 Autorisations

| Protocole | État par défaut |
|-----------|-----------------|
| CIFS | Coché |
| FTP | Coché |
| NFS | Non coché |

---

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline primary | Ferme la modal |
| Confirmer | Primary | Ajoute l'accès |

---

## 6. APIS & ENDPOINTS

### 6.1 Ajouter un accès

```
POST /vps/{serviceName}/backupftp/access
{
  "ipBlock": "192.168.1.0/24",
  "cifs": true,
  "ftp": true,
  "nfs": false
}
```
