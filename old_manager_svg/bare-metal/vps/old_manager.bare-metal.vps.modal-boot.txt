# Cahier des Charges : bare-metal/vps - Modal Boot

## 1. VUE D'ENSEMBLE
- **Modal** : Mode de démarrage
- **Type** : Sélection par tiles
- **Objectif** : Choisir le mode de boot du VPS

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bannière info
- 3 tiles de sélection
- Boutons Annuler / Redémarrer

## 3. COMPOSANTS UI
- **OdsModal** : Container (700x390px)
- **OdsMessage** : Info banner (fond bleu)
- **OdsTile** : 3 options avec radio
- **OdsRadio** : Sélection mode
- **OdsButton** : Annuler (outline), Redémarrer (primary)

## 4. BANNIÈRE INFO
"ℹ️ Le VPS redémarrera avec le mode sélectionné."

## 5. OPTIONS DE DÉMARRAGE

### Disque local (sélectionné)
| Élément | Valeur |
|---------|--------|
| Titre | Disque local |
| Description | Démarrage normal depuis le disque |
| État | ✅ Sélectionné (bordure bleue) |

### Mode rescue
| Élément | Valeur |
|---------|--------|
| Titre | Mode rescue |
| Description | Système temporaire pour maintenance |
| État | Non sélectionné |

### Réseau (PXE)
| Élément | Valeur |
|---------|--------|
| Titre | Réseau (PXE) |
| Description | Démarrage via le réseau |
| État | Non sélectionné |

## 6. BOUTONS
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline | 100px |
| Redémarrer | Primary | 160px |

## 7. APIS & ENDPOINTS
- GET /vps/{serviceName}/boot
  - Mode de boot actuel
- PUT /vps/{serviceName}/boot
  - Body: { bootType: "local" | "rescue" | "network" }
- POST /vps/{serviceName}/reboot
  - Redémarre avec nouveau mode
