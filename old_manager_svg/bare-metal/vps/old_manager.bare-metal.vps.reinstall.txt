# Cahier des Charges : vps - Réinstallation

## 1. VUE D'ENSEMBLE
- **Page** : Réinstallation OS VPS
- **Type** : Onglet avec wizard
- **Objectif** : Réinstaller le système d'exploitation

## 2. STRUCTURE PAGE
- Onglet "Reinstallation" actif
- Bandeau warning danger
- Étape 1 : Sélection OS
- Étape 2 : Options
- Bouton réinstaller

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsMessage** : Warning danger
- **OdsRadio** : Sélection OS (cards)
- **OdsSelect** : Clé SSH, Script
- **OdsButton** : Réinstaller

## 4. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Style | Danger (rouge) |
| Icône | ⚠️ |
| Titre | "Attention : la réinstallation effacera toutes les données" |
| Message | "Pensez à sauvegarder vos données importantes avant de procéder." |

## 5. ÉTAPE 1 : CHOIX OS
### Cartes sélection (radio)
| OS | Version | Sélectionné |
|----|---------|-------------|
| Ubuntu | 24.04 LTS | ✓ |
| Debian | 12 Bookworm | |

### Style carte
- Dimensions : 210x100px
- Radio button à gauche
- Nom OS (bold)
- Version (muted)
- Bordure bleue si sélectionné

## 6. ÉTAPE 2 : OPTIONS
| Champ | Type | Exemple |
|-------|------|---------|
| Clé SSH | Select | "Ma clé SSH principale" |
| Script post-installation | Select | "Aucun (optionnel)" |
| Lien | Link | "+ Ajouter une nouvelle clé SSH" |

## 7. BOUTON RÉINSTALLER
| Propriété | Valeur |
|-----------|--------|
| Label | "Réinstaller" |
| Style | Primary (200x44px) |
| Action | Ouvre modal confirmation |

## 8. APIS & ENDPOINTS
- GET /vps/{serviceName}/templates - Liste des OS disponibles
- GET /me/sshKey - Clés SSH du compte
- POST /vps/{serviceName}/reinstall - Lancer réinstallation
