# Cahier des Charges : vps - Monitoring

## 1. VUE D'ENSEMBLE
- **Page** : Monitoring VPS
- **Type** : Onglet métriques et graphiques
- **Objectif** : Surveiller les performances du VPS

## 2. STRUCTURE PAGE
- Onglet "Monitoring" actif
- 4 cartes métriques en ligne
- Graphique CPU 24h
- Graphique trafic réseau 24h
- Carte configuration alertes

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Cartes métriques
- **OdsProgress** : Barres utilisation
- **OdsBadge** : Statut
- **OdsChart** : Graphiques ligne
- **OdsSelect** : Période

## 4. CARTES MÉTRIQUES
| Métrique | Valeur | Style barre |
|----------|--------|-------------|
| CPU | 30% | Bleu |

## 9. APIS & ENDPOINTS
- GET /vps/{serviceName}/monitoring - Métriques actuelles
- GET /vps/{serviceName}/monitoring/statistics - Historique
- GET /vps/{serviceName}/serviceMonitoring - Config alertes
- PUT /vps/{serviceName}/serviceMonitoring - Modifier alertes
