# Cahier des Charges : bare-metal/vps - Onglet Backup

## 1. VUE D'ENSEMBLE
- **Onglet** : Backup
- **Type** : Page gestion sauvegardes automatiques
- **Objectif** : Gérer les backups automatiques du VPS

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre
- Onglets navigation
- Carte état backup
- Tableau sauvegardes
- Info box

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 6 onglets
- **OdsCard** : État backup, Liste sauvegardes
- **OdsToggle** : Backup automatique
- **OdsBadge** : Actif (success), Complet (success)
- **OdsDatagrid** : Tableau sauvegardes
- **OdsActionMenu** : Menu ⋮
- **OdsMessage** : Info box
- **OdsPagination** : Navigation

## 9. INFO BOX
"ℹ️ Le backup automatique effectue une sauvegarde complète chaque nuit à 03:00. Les sauvegardes sont conservées pendant 7 jours. Vous pouvez restaurer ou monter un backup à tout moment."

## 10. APIS & ENDPOINTS
- GET /vps/{serviceName}/automatedBackup
  - État backup automatique
- GET /vps/{serviceName}/automatedBackup/attachedBackup
  - Liste des backups
- POST /vps/{serviceName}/automatedBackup/restore
  - Restaurer un backup
- POST /vps/{serviceName}/automatedBackup/attachedBackup/{backupId}/mount
  - Monter en lecture
