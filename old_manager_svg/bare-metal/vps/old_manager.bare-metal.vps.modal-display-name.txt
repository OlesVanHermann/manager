# Cahier des Charges : bare-metal/vps - Modal Nom affichage

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le nom d'affichage
- **Type** : Formulaire simple
- **Objectif** : Changer le nom affiché du VPS

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Info banner
- Nom technique (lecture seule)
- Champ nom d'affichage
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Container (500x300px)
- **OdsMessage** : Info banner
- **OdsText** : Nom technique
- **OdsInput** : Nom d'affichage
- **OdsButton** : Annuler (outline), Enregistrer (primary)

## 4. INFO BANNER
"ℹ️ Ce nom sera affiché dans votre espace client uniquement."

## 5. CHAMPS
### Nom technique (non modifiable)
| Élément | Valeur |
|---------|--------|
| Label | Nom technique (non modifiable) |
| Valeur | vps-abc12345.vps.ovh.net |
| Style | Texte gris |

### Nom d'affichage
| Élément | Valeur |
|---------|--------|
| Label | Nom d'affichage |
| Valeur exemple | Serveur Production Web |

## 6. BOUTONS
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Enregistrer | Primary |

## 7. APIS & ENDPOINTS
- PUT /vps/{serviceName}
  - Body: { displayName: string }
