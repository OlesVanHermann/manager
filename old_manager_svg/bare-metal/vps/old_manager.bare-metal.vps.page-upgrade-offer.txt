# Cahier des Charges : bare-metal/vps - Page Upgrade Offer

## 1. VUE D'ENSEMBLE
- **Type** : Page changement d'offre
- **Contexte** : VPS
- **Objectif** : Migrer vers une offre VPS supérieure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Cartes offres
- **OdsBadge** : Sélection
- **OdsButton** : Actions
- **OdsMessage** : Récapitulatif
