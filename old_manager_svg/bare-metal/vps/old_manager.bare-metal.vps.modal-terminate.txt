# Cahier des Charges : bare-metal/vps - Modal Résilier

## 1. VUE D'ENSEMBLE
- **Modal** : Résilier mon VPS
- **Type** : Confirmation dangereuse
- **Objectif** : Résilier définitivement le VPS

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bannière warning (jaune)
- Sélection raison
- Info date résiliation
- Boutons Annuler / Résilier (danger)

## 3. COMPOSANTS UI
- **OdsModal** : Container (600x400px)
- **OdsMessage** : Warning banner (fond jaune)
- **OdsSelect** : Raison résiliation
- **OdsText** : Informations
- **OdsButton** : Annuler (outline), Résilier (danger)

## 4. BANNIÈRE WARNING
| Style | Contenu |
|-------|---------|
| Fond jaune | ⚠️ Attention : la résiliation de votre VPS est définitive. |
| | Aucun remboursement ne sera effectué. |

## 5. SÉLECTION RAISON
| Élément | Valeur |
|---------|--------|
| Label | Veuillez sélectionner la raison de votre résiliation : |
| Options | Je n'ai plus besoin de ce service |
| | Le prix est trop élevé |
| | J'ai trouvé une meilleure offre |
| | Problèmes techniques non résolus |
| | Autre |

## 6. INFORMATIONS
- "Votre VPS sera résilié à la fin de la période de facturation en cours."
- "Date de résiliation prévue : 15/01/2025"

## 7. BOUTONS
| Bouton | Style | Couleur |
|--------|-------|---------|
| Annuler | Outline | Gris |
| Résilier | Primary | Rouge (danger) |

## 8. APIS & ENDPOINTS
- DELETE /vps/{serviceName}
  - Body: { reason: string }
  - Résiliation du VPS
- GET /vps/{serviceName}/serviceInfos
  - Date fin facturation
