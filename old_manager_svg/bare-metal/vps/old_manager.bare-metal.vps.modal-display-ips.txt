# Cahier des Charges : bare-metal/vps - Modal Adresses IP

## 1. VUE D'ENSEMBLE
- **Modal** : Adresses IP du VPS
- **Type** : Liste avec copie
- **Objectif** : Afficher et copier les IPs du VPS

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Tableau des IPs
- Boutons copier
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Container (600x420px)
- **OdsDatagrid** : Liste IPs
- **OdsButton** : Copier (outline bleu), Fermer (outline)
- **OdsClipboard** : Copie
