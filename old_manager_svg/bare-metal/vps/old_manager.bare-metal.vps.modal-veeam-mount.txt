---

## 6. APIS & ENDPOINTS

### 6.1 Monter le backup

```
POST /vps/{serviceName}/automatedBackup/restore/{restorePoint}/mount
```

### 6.2 Vérifier si un backup est déjà monté

```
GET /vps/{serviceName}/automatedBackup/attachedBackup
```

### 6.3 Démonter le backup actuel

```
POST /vps/{serviceName}/automatedBackup/detachBackup
```
