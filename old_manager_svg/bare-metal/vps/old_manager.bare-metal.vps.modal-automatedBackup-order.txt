# Cahier des Charges : bare-metal/vps - Modal Backup automatique

## 1. VUE D'ENSEMBLE
- **Modal** : Activer les sauvegardes automatiques
- **Type** : Formulaire de commande
- **Objectif** : Souscrire à l'option backup automatique

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Info banner
- Sélection formule (3 tiles)
- Heure de sauvegarde
- Features incluses
- Résumé
- Boutons Annuler / Activer

## 3. COMPOSANTS UI
- **OdsModal** : Container (700x600px)
- **OdsMessage** : Info banner
- **OdsTile** : 3 formules
- **OdsBadge** : Recommandé (vert)
- **OdsSelect** : Heure backup
- **OdsList** : Features (avec checks)
- **OdsCard** : Résumé
- **OdsButton** : Annuler (outline), Activer (primary)

## 4. INFO BANNER
"ℹ️ Les sauvegardes automatiques protègent vos données avec des copies quotidiennes. Restauration possible en quelques clics depuis votre espace client."
