## 6. APIS & ENDPOINTS

### 6.1 Configurer l'heure du backup

```
PUT /vps/{serviceName}/automatedBackup
{
  "schedule": "03:00"
}
```

### 6.2 Obtenir la configuration actuelle

```
GET /vps/{serviceName}/automatedBackup
```
