## 6. APIS & ENDPOINTS

### 6.1 Obtenir les options du VPS

```
GET /vps/{serviceName}/option
```

### 6.2 Obtenir détail option Windows

```
GET /vps/{serviceName}/option/windows
```

### 6.3 Résilier l'option Windows

```
DELETE /vps/{serviceName}/option/windows
```
