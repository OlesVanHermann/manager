## 7. APIS & ENDPOINTS

### 7.1 Obtenir les offres de migration

```
GET /vps/{serviceName}/migration
```

### 7.2 Lancer la migration

```
POST /vps/{serviceName}/migration
{
  "targetPlanCode": "vps-ssd-2"
}
```
