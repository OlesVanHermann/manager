# Cahier des Charges : bare-metal/vps - Page Snapshot Management

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion snapshot
- **Contexte** : VPS
- **Objectif** : Gérer les snapshots du VPS (sauvegardes instantanées)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 450px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Carte snapshot
- **OdsBadge** : Statut snapshot
- **OdsButton** : Actions
- **OdsMessage** : Bloc information

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Snapshot" |
| Description | "Sauvegardez l'état actuel de votre VPS pour pouvoir le restaurer ultérieurement." |

## 5. CARTE SNAPSHOT ACTUEL
| Propriété | Valeur |
|-----------|--------|
| Largeur | 940px |
| Hauteur | 200px |
| Sous-titre | "Snapshot actuel" |

## 6. INFORMATIONS SNAPSHOT
| Champ | Largeur | Exemple |
|-------|---------|---------|
| Date de création | 250px | "28/12/2024 à 14:30:00" |

| Statut | 150px | Badge "Disponible" |

## 8. BOUTONS ACTIONS
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Restaurer | Fond #0050D7, texte blanc | 120x40px |

|-----------|--------|

| Icône | ℹ |
| Texte | "Un seul snapshot peut être conservé à la fois. La création d'un nouveau snapshot remplacera l'existant." |
| Largeur | 940px |
| Hauteur | 80px |

## 10. APIS & ENDPOINTS
- GET /vps/{serviceName}/snapshot - Récupérer snapshot actuel
- POST /vps/{serviceName}/createSnapshot - Créer un nouveau snapshot
- POST /vps/{serviceName}/snapshot/revert - Restaurer le snapshot
- DELETE /vps/{serviceName}/snapshot - Supprimer le snapshot
