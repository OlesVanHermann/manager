# Cahier des Charges : bare-metal/vps - Dashboard VPS

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard VPS
- **Type** : Page détail avec onglets et graphiques
- **Objectif** : Vue complète du VPS sélectionné

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb (Bare Metal Cloud > VPS > vps-abc123)
- Titre + badge état (Démarré)
- Onglets navigation
- Cartes informations (3 colonnes)
- Graphiques performances (CPU, RAM)
- Bande passante

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 6 onglets
- **OdsBadge** : État (Démarré - success)
- **OdsCard** : 3 cartes infos + graphiques
- **OdsButton** : Redémarrer (primary), Arrêter, Réinstaller, KVM, Réinitialiser MDP
- **OdsChart** : CPU, RAM
- **OdsProgressBar** : Quota bande passante

## 10. APIS & ENDPOINTS
- GET /vps/{serviceName}
  - Informations VPS
- GET /vps/{serviceName}/status
  - État du VPS
- GET /vps/{serviceName}/monitoring
  - Métriques CPU/RAM
- POST /vps/{serviceName}/reboot
  - Redémarrer le VPS
- POST /vps/{serviceName}/stop
  - Arrêter le VPS
