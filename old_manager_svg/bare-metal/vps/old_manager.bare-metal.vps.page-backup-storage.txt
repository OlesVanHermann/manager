# Cahier des Charges : bare-metal/vps - Page Backup Storage

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion backup storage
- **Contexte** : VPS
- **Objectif** : Gérer l'espace de stockage de sauvegarde

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Tuiles informations
- **OdsProgressBar** : Usage stockage
- **OdsBadge** : Protocoles
- **OdsSwitch** : Toggles protocoles
- **OdsDatagrid** : Tableau accès
- **OdsButton** : Actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Backup Storage" |
| Description | "Gérez l'espace de stockage de sauvegarde de votre VPS." |

## 5. LAYOUT TUILES
| Propriété | Valeur |
|-----------|--------|

| Propriété | Valeur |
|-----------|--------|
| Texte | "Révoquer" |
| Fond | #FEE2E2 |
| Texte | #DC2626 |

## 15. APIS & ENDPOINTS
- GET /vps/{serviceName}/backupftp - Statut backup storage
- GET /vps/{serviceName}/backupftp/access - Liste accès
- POST /vps/{serviceName}/backupftp/access - Ajouter accès
- PUT /vps/{serviceName}/backupftp/access/{ipBlock} - Modifier accès
- DELETE /vps/{serviceName}/backupftp/access/{ipBlock} - Révoquer accès
