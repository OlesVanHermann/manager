# Cahier des Charges : bare-metal/vps - Outperform (Surclassement gratuit)

## 1. VUE D'ENSEMBLE
- **Page** : Surclasser mon VPS
- **Type** : Page de promotion/upgrade gratuit
- **Objectif** : Bénéficier d'un surclassement gratuit

## 2. STRUCTURE PAGE
- Titre "Surclasser mon VPS"
- Sous-titre (avantage gratuit)
- Bandeau warning interruption
- Comparaison avant/après
- Boutons action

## 3. COMPOSANTS UI
