## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline primary | Ferme la modal |
| Confirmer | Primary | Applique les modifications |

---

## 6. APIS & ENDPOINTS

### 6.1 Modifier un accès

```
PUT /vps/{serviceName}/backupftp/access/{ipBlock}
{
  "cifs": true,
  "ftp": true,
  "nfs": false
}
```
