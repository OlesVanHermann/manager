## 4. CONTENU

### 4.1 Description

| Texte |
|-------|
| Sélectionnez les adresses IP à autoriser pour accéder au backup storage. |

### 4.2 Sélection des IPs

| Champ | Type | Description |
|-------|------|-------------|
| Adresses IP | Select multiple | Liste des IPs à autoriser |

| Aide |
|------|
| Maintenez Ctrl pour sélection multiple |

### 4.3 Note explicative

| Texte |
|-------|
| Les IP seront autorisées à accéder à votre backup storage via les protocoles sélectionnés. |

### 4.4 Protocoles d'accès

| Protocole | État | Description |
|-----------|------|-------------|
| CIFS (Windows) | Coché | Protocole Windows/Samba |
| FTP | Coché | File Transfer Protocol |
| NFS (Linux) | Non coché | Network File System |

---

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline primary | Ferme la modal |
| Confirmer | Primary | Ajoute les accès |

---

## 6. APIS & ENDPOINTS

### 6.1 Ajouter un accès

```
POST /vps/{serviceName}/backupftp/access
{
  "ipBlock": "192.168.1.0/24",
  "cifs": true,
  "ftp": true,
  "nfs": false
}
```

### 6.2 Liste des accès existants

```
GET /vps/{serviceName}/backupftp/access
```
