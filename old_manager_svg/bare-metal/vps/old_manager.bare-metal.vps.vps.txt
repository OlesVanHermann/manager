## 7. APIS & ENDPOINTS

```
# Informations VPS
GET /vps/{serviceName}
→ {
  name: "vps-abc123",
  displayName: "vps-abc123.vps.ovh.net",
  model: {
    name: "VPS Starter",
    vcore: 2,
    memory: 4,
    disk: 80,
    diskType: "nvme"
  },
  zone: "rbx",
  state: "running"
}

# OS et accès
GET /vps/{serviceName}/distribution
→ {
  name: "Ubuntu",
  version: "22.04 LTS"
}

GET /vps/{serviceName}/ips
→ {
  ipv4: "51.38.123.45",
  ipv6: "2001:41d0:xxx::1",
  reverse: "vps-abc123.vps.ovh.net"
}

# État
GET /vps/{serviceName}/status
→ {
  state: "running",
  uptime: 1314300,  // secondes
  renewDate: "2026-01-15"
}

# Monitoring
GET /vps/{serviceName}/monitoring
Query: { period: "hour", type: "cpu" }
→ {
  values: [{ timestamp: "...", value: 23 }, ...],
  average: 23
}

# Bande passante
GET /vps/{serviceName}/bandwidth
Query: { period: "month" }
→ {
  quota: 1024,  // GB
  used: 350,
  incoming: 125,
  outgoing: 225,
  resetDate: "2026-01-01"
}

# Actions
POST /vps/{serviceName}/reboot
POST /vps/{serviceName}/stop
POST /vps/{serviceName}/start
```
