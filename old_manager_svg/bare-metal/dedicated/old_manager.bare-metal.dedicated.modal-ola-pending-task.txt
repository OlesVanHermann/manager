# Cahier des Charges : bare-metal/dedicated - Modal OLA Pending Task

## 1. VUE D'ENSEMBLE
- **Type** : État de chargement
- **Contexte** : Configuration OLA
- **Objectif** : Indiquer qu'une tâche OLA est en cours

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 220px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsMessage** : Bannière info
- **OdsSpinner** : Indicateur de chargement

## 4. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |

## 5. SPINNER
| Propriété | Valeur |
|-----------|--------|
| Type | Cercle animé |
| Rayon | 35px |

| Couleur active | #0050D7 |
| Épaisseur | 6px |
| Texte | "Veuillez patienter" |
| Texte couleur | #6B7280 |

## 6. COMPORTEMENT
| État | Action |
|------|--------|
| Tâche en cours | Afficher ce modal |
| Tâche terminée | Fermer et rafraîchir |
| Erreur | Afficher message d'erreur |

## 7. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/task - Vérifier statut tâche
