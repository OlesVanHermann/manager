# Cahier des Charges : bare-metal/dedicated - Tile Tagging

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les étiquettes (tags) attribuées au serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsChip** : Tags
- **OdsLink** : Gérer les étiquettes

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Étiquettes" |

## 5. SECTION TAGS
| Propriété | Valeur |
|-----------|--------|
| Label | "Attribuées" |
| Disposition | Flex wrap horizontal |

## 6. STYLE CHIPS
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | #0050D7 |
| Padding | 8px 12px |
| Border radius | 4px |

### Tags exemple
- production
- europe
- critical
- team-a

## 7. ACTION
| Propriété | Valeur |
|-----------|--------|
| Lien | "Gérer les étiquettes ›" |
| Couleur | Bleu #0050D7 |
| Navigation | Page gestion tags |

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/tag - Liste tags serveur
- GET /tag - Liste tous les tags disponibles
- POST /dedicated/server/{serviceName}/tag - Attribuer tag
- DELETE /dedicated/server/{serviceName}/tag/{tagKey} - Retirer tag
