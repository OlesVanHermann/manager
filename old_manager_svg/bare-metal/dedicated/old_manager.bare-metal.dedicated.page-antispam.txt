## 4. APIS & ENDPOINTS

### 4.1 Récupérer statut anti-spam

```
GET /ip/{ip}/spam/{ipSpamming}
```

### 4.2 Historique des événements

```
GET /ip/{ip}/spam/{ipSpamming}/stats?from={date}&to={date}
```

### 4.3 Débloquer l'IP

```
POST /ip/{ip}/spam/{ipSpamming}/unblock
```
