## 4. APIS & ENDPOINTS

### 4.1 Liste des événements de mitigation

```
GET /ip/mitigation
```

### 4.2 Détails d'un événement

```
GET /ip/{ip}/mitigation/{ipOnMitigation}
```

### 4.3 Statistiques

```
GET /ip/{ip}/mitigation/{ipOnMitigation}/stats?from={date}&to={date}
```
