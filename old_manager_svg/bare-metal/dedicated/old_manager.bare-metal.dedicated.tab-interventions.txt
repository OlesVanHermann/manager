# Cahier des Charges : bare-metal/dedicated - Onglet Interventions (Détaillé)

## 1. VUE D'ENSEMBLE
- **Onglet** : Interventions (historique détaillé)
- **Type** : Page tableau avec plus de colonnes
- **Objectif** : Afficher l'historique complet des interventions physiques

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre serveur + badge état
- Onglets navigation
- Titre + description
- Datagrid interventions

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets
- **OdsBadge** : État (Normal)
- **OdsDatagrid** : Liste interventions
- **OdsText** : Message fin de liste

## 4. COLONNES TABLEAU
| Colonne | Largeur |
|---------|---------|
| ID | 100px |
| Type | 200px |
| Date | 150px |
| Description | 350px |
| Status | 100px |
