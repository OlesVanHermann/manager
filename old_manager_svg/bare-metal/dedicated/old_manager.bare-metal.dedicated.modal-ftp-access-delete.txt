# Cahier des Charges : dedicated - Modal supprimer accès FTP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : FTP Backup Storage
- **Objectif** : Supprimer un accès FTP pour un bloc IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 140px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'accès" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Voulez-vous supprimer l'accès pour le bloc {ipBlock} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Danger (rouge) | Supprime l'accès |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/server/{serviceName}/features/backupFTP/access/{ipBlock} - Supprimer accès
