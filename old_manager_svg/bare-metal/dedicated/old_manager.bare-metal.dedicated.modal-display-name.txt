# Cahier des Charges : bare-metal/dedicated - Modal Modifier Nom d'Affichage

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le nom d'affichage
- **Type** : Modal formulaire simple
- **Objectif** : Personnaliser le nom affiché du serveur

## 2. STRUCTURE MODAL
- Modal blanche (600x300px)
- Header avec titre + bouton fermer
- Bandeau info
- Nom technique (lecture seule)
- Input nom d'affichage
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu)
- **OdsText** : Nom technique (monospace)
- **OdsInput** : Nom d'affichage
- **OdsButton** : Annuler (outline), Enregistrer (primary)

## 4. HEADER
- **Titre** : "Modifier le nom d'affichage"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
"ℹ️ Ce nom sera affiché dans votre espace client uniquement."

## 6. NOM TECHNIQUE
- **Label** : "Nom technique (non modifiable)"
- **Valeur** : ns1234567.ip-192-168-1.eu (monospace, lecture seule)

## 7. NOM D'AFFICHAGE
- **Label** : "Nom d'affichage"
- **Input** : "Production DB Master"
- **Largeur** : 540px

## 8. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Enregistrer** : Primary, sauvegarde le nom

## 9. APIS & ENDPOINTS
- PUT /dedicated/server/{serviceName}
  - Body: { displayName: "Production DB Master" }
  - Met à jour le nom d'affichage
