# Cahier des Charges : bare-metal/dedicated - Modal Détacher Interface vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Gestion réseau serveur dédié
- **Objectif** : Déconnecter une interface réseau d'un vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Détacher

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détacher du réseau privé" |
| Fond | Warning (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Êtes-vous sûr de vouloir détacher l'interface" |
| Ligne 2 | "{interface_name} du vRack {vrack_name} ?" |
| Noms | Bleu (#0050D7), font-weight 500 |

### Warning
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠ |
| Texte | "L'interface perdra sa connectivité au réseau privé." |
| Couleur texte | #92400E |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Détacher | Warning (orange #F59E0B) | Déconnecte du vRack |

## 7. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/dedicatedServerInterface/{serverInterface} - Détacher interface
