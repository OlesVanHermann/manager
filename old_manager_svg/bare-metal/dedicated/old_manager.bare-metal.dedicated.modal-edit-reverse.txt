# Cahier des Charges : bare-metal/dedicated - Modal Modifier Reverse DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le reverse DNS
- **Type** : Modal formulaire
- **Objectif** : Associer un nom de domaine à une IP

## 2. STRUCTURE MODAL
- Modal blanche (600x380px)
- Header avec titre + bouton fermer
- Bandeau info
- Adresse IP (lecture seule)
- Input reverse DNS
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu)
- **OdsText** : IP (monospace, lecture seule)
- **OdsInput** : Reverse DNS (FQDN)
- **OdsButton** : Annuler (outline), Enregistrer (primary)

## 4. HEADER
- **Titre** : "Modifier le reverse DNS"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
- "ℹ️ Le reverse DNS permet d'associer un nom de domaine à une IP."
- "Le domaine doit pointer vers cette IP (enregistrement A)."

## 6. FORMULAIRE

### Adresse IP
- **Label** : "Adresse IP"
- **Valeur** : 192.168.1.123 (monospace, fond gris, lecture seule)
- **Largeur** : 250px

### Reverse DNS
- **Label** : "Reverse DNS *"
- **Description** : "Nom de domaine complet (FQDN)"
- **Valeur** : "mail.example.com" (monospace)
- **Largeur** : 540px
- **Requis** : Oui

## 7. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Enregistrer** : Primary, sauvegarde le reverse

## 8. APIS & ENDPOINTS
- PUT /ip/{ip}/reverse
  - Body: { reverse: "mail.example.com" }
  - Met à jour le reverse DNS
