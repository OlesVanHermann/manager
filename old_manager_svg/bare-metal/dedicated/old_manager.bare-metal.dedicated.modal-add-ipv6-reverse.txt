## 6. APIS & ENDPOINTS

### 6.1 Ajouter reverse IPv6

```
POST /ip/{ip}/reverse
{
  "ipReverse": "2001:41d0::1234",
  "reverse": "example.com"
}
```

### 6.2 Vérifier le bloc IP

```
GET /ip/{ip}
```
