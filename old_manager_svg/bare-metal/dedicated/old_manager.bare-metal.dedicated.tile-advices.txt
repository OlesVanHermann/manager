# Cahier des Charges : bare-metal/dedicated - Tile Advices

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard recommandations
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les recommandations et conseils personnalisés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsMessage** : Cartes recommandation
- **OdsText** : Contenu
- **OdsLink** : Actions "Voir ›"

## 4. HEADER TUILE

|-----------|--------|
| Titre | "Recommandations" |
