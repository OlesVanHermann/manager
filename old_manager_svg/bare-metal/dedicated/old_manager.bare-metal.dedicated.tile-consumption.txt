# Cahier des Charges : bare-metal/dedicated - Tile Consumption

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher la consommation trafic et gérer les quotas

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 330px |
| Tuile hauteur | 230px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsProgressBar** : Barre consommation
- **OdsText** : Labels et valeurs
- **OdsButton** : Actions quota
- **OdsPopover** : Menu actions

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Consommation" |

## 5. TRAFIC SORTANT
| Propriété | Valeur |
|-----------|--------|
| Label | "Trafic sortant" |
| Barre fond | #E5E7EB |
| Barre remplie | #0050D7 |
| Valeur affichée | "2.5 TB" (blanc sur barre) |
| Total | "/ 10 TB inclus" |

## 6. PÉRIODE FACTURATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Période de facturation" |
| Format | "01/12/2024 - 31/12/2024" |

## 7. BOUTONS ACTIONS
| Bouton | Largeur | Action |
|--------|---------|--------|
| Commander quota | 135px | Ouvre modal commande |
| Annuler quota | 135px | Ouvre modal résiliation |

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/statistics/traffic - Consommation
- GET /order/dedicated/server/{serviceName}/traffic - Options trafic
- POST /order/dedicated/server/{serviceName}/traffic - Commander quota
- DELETE /dedicated/server/{serviceName}/option/TRAFFIC - Annuler quota
