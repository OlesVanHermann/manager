# Cahier des Charges : bare-metal/dedicated - Modal Résiliation

## 1. VUE D'ENSEMBLE
- **Modal** : Résilier le serveur
- **Type** : Modal destructive avec confirmation
- **Objectif** : Résilier définitivement le serveur dédié

## 2. STRUCTURE MODAL
- Modal blanche (600x420px)
- Header rouge avec titre + bouton fermer
- Message danger
- Info serveur
- Input confirmation texte
- Checkbox confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Danger (fond rouge)
- **OdsText** : Info serveur
- **OdsInput** : Confirmation "RESILIER"
- **OdsCheckbox** : Confirmation irréversible
- **OdsButton** : Annuler (outline), Résilier (danger)

## 4. HEADER
- **Fond** : Rouge clair
- **Titre** : "Résilier le serveur"
- **Bouton fermer** : ✕

## 5. MESSAGE DANGER
Fond rouge avec bordure :
- "⚠ Attention : Cette action est définitive !"
- "Le serveur sera supprimé à la fin de la période de facturation en cours."
- "Toutes les données seront définitivement effacées."

## 6. INFO SERVEUR
| Champ | Valeur |
|-------|--------|
| Serveur concerné | ns1234567.ip-123-45-67.eu |
| Date d'expiration | 15/01/2025 |

## 7. CONFIRMATION
- **Label** : "Pour confirmer, tapez \"RESILIER\""
- **Input** : Placeholder "Entrez RESILIER..."
- **Checkbox** : ☐ "Je comprends que cette action est irréversible."

## 8. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Résilier** : Danger (rouge), désactivé tant que confirmation incomplète

## 9. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/terminate
  - Déclenche la résiliation
  - Prend effet à la fin de la période de facturation
