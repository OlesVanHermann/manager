# Cahier des Charges : bare-metal/dedicated - Tile Service Status

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher l'état du serveur et options monitoring

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 330px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsBadge** : Statuts
- **OdsPopover** : Menu actions
- **OdsLink** : Liens externes

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "État des services" |

## 5. STATUT SERVEUR
| Propriété | Valeur |
|-----------|--------|
| Label | "Statut" |
| Badge OK | Fond #DCFCE7, texte #16A34A "OK" |

## 6. MENU ACTIONS (⋮)
| Action | Style |
|--------|-------|
| Redémarrer | Texte noir |
| Résilier | Texte rouge #DC2626 |

## 7. MONITORING
| Propriété | Valeur |
|-----------|--------|
| Label | "Monitoring" |
| Badge | Fond #DCFCE7, texte #16A34A "Proactif avec intervention" |
| Lien | "Modifier les options ›" (bleu) |

## 8. LIENS UTILES
| Lien | URL |
|------|-----|
| Weathermap ↗ | Lien externe status réseau |
| VMS ↗ | Lien externe système virtualization |
| Travaux en cours ↗ | Lien externe travaux |

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName} - Statut serveur
- POST /dedicated/server/{serviceName}/reboot - Redémarrer
- POST /dedicated/server/{serviceName}/terminate - Résilier
- GET /dedicated/server/{serviceName}/serviceMonitoring - Options monitoring
- PUT /dedicated/server/{serviceName}/serviceMonitoring - Modifier monitoring
