# Cahier des Charges : bare-metal/dedicated - Wizard Commander du Trafic

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de commande (3 étapes)
- **Contexte** : Gestion trafic serveur dédié
- **Objectif** : Commander du trafic supplémentaire pour le serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes
- **OdsRadio** : Sélection type de trafic
- **OdsButton** : Annuler, Suivant
- **OdsText** : Descriptions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Breadcrumb | "Serveur dédié › Commander du trafic" |
| Titre | "Commander du trafic" |

## 5. CONTENU

### Indicateur d'étapes
| Étape | Label | État |
|-------|-------|------|
| 1 | Type de trafic | En cours (cercle bleu) |
| 2 | Durée | À venir (cercle gris) |
| 3 | Confirmation | À venir (cercle gris) |

### Titre étape 1
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sélectionnez le type de trafic" |
| Style | 16px, font-weight 600 |

| Propriété | Valeur |
|-----------|--------|

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Suivant | Droite | Primary (#0050D7) | Passe à l'étape durée |

## 7. ÉTAPES SUIVANTES (aperçu)

### Étape 2 - Durée
- Sélection de la durée d'engagement
- Options : 1 mois, 3 mois, 6 mois, 12 mois
- Calcul du prix total

### Étape 3 - Confirmation
- Récapitulatif de la commande
- Détail du prix
- Validation du paiement

## 8. APIS & ENDPOINTS
- GET /order/dedicated/server/{serviceName}/traffic - Options de trafic disponibles
- GET /order/dedicated/server/{serviceName}/traffic/{duration} - Prix selon durée
- POST /order/dedicated/server/{serviceName}/traffic/{duration} - Valider la commande
