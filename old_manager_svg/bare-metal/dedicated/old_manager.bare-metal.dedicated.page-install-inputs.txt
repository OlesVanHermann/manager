# Cahier des Charges : bare-metal/dedicated - Page Install Inputs

## 1. VUE D'ENSEMBLE
- **Type** : Formulaire wizard
- **Contexte** : Installation OS serveur dédié
- **Objectif** : Configurer les paramètres d'installation

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 800px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titre et labels
- **OdsInput** : Champs texte
- **OdsTextarea** : Clé SSH
- **OdsSelect** : Sélecteurs
- **OdsSwitch** : Toggle RAID
- **OdsMessage** : Info OS
- **OdsLink** : Import clé SSH

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Paramètres d'installation" |

## 7. APIS & ENDPOINTS
- GET /dedicated/installationTemplate - Templates disponibles
- GET /dedicated/server/{serviceName}/install/templateCapabilities - Capacités template
- GET /me/sshKey - Clés SSH utilisateur
- POST /dedicated/server/{serviceName}/install/start - Lancer installation
