## 4. APIS & ENDPOINTS

### 4.1 Liste des règles

```
GET /ip/{ip}/game/{ipOnGame}/rule
```

### 4.2 Ajouter une règle

```
POST /ip/{ip}/game/{ipOnGame}/rule
{
  "protocol": "udp",
  "ports": {"from": 27015, "to": 27015}
}
```

### 4.3 Supprimer une règle

```
DELETE /ip/{ip}/game/{ipOnGame}/rule/{id}
```

### 4.4 Activer/désactiver politique

```
PUT /ip/{ip}/game/{ipOnGame}
{
  "firewallModeEnabled": true
}
```
