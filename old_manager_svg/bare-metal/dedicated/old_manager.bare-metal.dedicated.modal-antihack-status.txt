## 6. APIS & ENDPOINTS

### 6.1 Récupérer statut Anti-Hack

```
GET /ip/{ip}/antihack/{ipBlocked}
```

**Réponse :**
```json
{
  "ipBlocked": "123.45.67.89",
  "blockedSince": "2026-01-02T14:32:15+01:00",
  "state": "blocked",
  "time": 7200
}
```

### 6.2 Débloquer une IP

```
DELETE /ip/{ip}/antihack/{ipBlocked}
```
