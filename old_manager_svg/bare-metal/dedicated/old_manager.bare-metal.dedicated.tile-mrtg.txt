# Cahier des Charges : bare-metal/dedicated - Tile MRTG

## 1. VUE D'ENSEMBLE
- **Type** : Tuile graphique statistiques
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les statistiques réseau (MRTG)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 630px |
| Tuile hauteur | 380px |
| Zone graphique | 590x220px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsSelect** : Filtres (réseau, type, période)
- **OdsChart** : Graphique linéaire
- **OdsText** : Labels axes et légende

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Statistiques réseau (MRTG)" |
| Sous-titre | "Consultez le trafic réseau de votre serveur" |

## 5. FILTRES

### Sélecteur Réseau
| Propriété | Valeur |
|-----------|--------|
| Label | "Réseau" |
| Largeur | 150px |
| Options | eth0 (public), eth1 (privé), vrack... |

### Sélecteur Type
| Propriété | Valeur |
|-----------|--------|
| Label | "Type" |
| Largeur | 150px |
| Options | Trafic, Erreurs, Paquets |

### Sélecteur Période
| Propriété | Valeur |
|-----------|--------|
| Label | "Période" |
| Largeur | 150px |
| Options | Dernière journée, Dernière semaine, Dernier mois, Dernière année |

## 6. GRAPHIQUE
| Propriété | Valeur |
|-----------|--------|
| Type | Line chart |
| Fond | #F9FAFB |
| Ligne entrant | #0050D7 (bleu) |
| Ligne sortant | #10B981 (vert) |
| Axe Y | 0-100 Mb/s (pas de 25) |
| Axe X | Heures (00:00, 06:00, 12:00, 18:00) |

## 7. LÉGENDE
| Couleur | Label |
|---------|-------|
| #0050D7 (carré bleu) | Entrant |
| #10B981 (carré vert) | Sortant |

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/mrtg - Données MRTG
- Paramètres: period (daily, weekly, monthly, yearly), type (traffic, errors, packets)
