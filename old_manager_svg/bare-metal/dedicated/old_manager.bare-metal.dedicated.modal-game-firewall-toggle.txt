## 6. APIS & ENDPOINTS

### 6.1 Activer/désactiver la politique par défaut

```
PUT /ip/{ip}/game/{ipOnGame}
{
  "firewallModeEnabled": true
}
```

### 6.2 Récupérer l'état du Game Firewall

```
GET /ip/{ip}/game/{ipOnGame}
```

**Réponse :**
```json
{
  "ipOnGame": "123.45.67.89",
  "firewallModeEnabled": false,
  "state": "ok"
}
```
