# Cahier des Charges : bare-metal/dedicated - Page Network Interfaces

## 1. VUE D'ENSEMBLE
- **Type** : Page listing interfaces réseau
- **Contexte** : Serveur dédié
- **Objectif** : Visualiser et gérer les interfaces réseau du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsDatagrid** : Tableau interfaces
- **OdsBadge** : Type interface
- **OdsPopover** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Interfaces réseau" |
| Description | "Gérez les interfaces réseau de votre serveur." |

## 5. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Interface | 120px | Nom (eth0, eth1...) |
| Adresse MAC | 180px | MAC address |
| Type | 120px | Badge type |
| Bande passante | 150px | Vitesse (Gbps) |
| État | 100px | Badge statut |
| Actions | 80px | Menu ⋮ |

## 6. DONNÉES EXEMPLE
| Interface | MAC | Type | Bande passante |
|-----------|-----|------|----------------|
| eth0 | aa:bb:cc:dd:ee:ff | Public | 1 Gbps |
| eth1 | aa:bb:cc:dd:ee:f0 | vRack | 10 Gbps |
| ipmi | aa:bb:cc:dd:ee:f1 | IPMI | 100 Mbps |
| eth2 | aa:bb:cc:dd:ee:f2 | vRack | 10 Gbps |

## 7. BADGES TYPE
| Type | Fond | Texte |
|------|------|-------|
| Public | #EFF6FF | #0050D7 |
| vRack | #F3E8FF | #7C3AED |
| IPMI | #FEF3C7 | #92400E |

## 8. BADGES ÉTAT
| État | Fond | Texte |
|------|------|-------|
| Actif | #DCFCE7 | #16A34A |
| Inactif | #F3F4F6 | #6B7280 |
| Erreur | #FEE2E2 | #DC2626 |

## 9. MENU ACTIONS
| Action | Style |
|--------|-------|
| Configurer | Texte #1F2937 |
| Voir le trafic | Texte #1F2937 |
| Désactiver | Texte #DC2626 |

## 10. SECTION INFO AGRÉGATION
| Propriété | Valeur |
|-----------|--------|
| Titre | "Agrégation de liens" |
| Contenu | Mode LACP si configuré |

## 11. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/networkInterfaceController - Liste interfaces
- GET /dedicated/server/{serviceName}/networkInterfaceController/{mac} - Détail interface
- PUT /dedicated/server/{serviceName}/networkInterfaceController/{mac} - Modifier
