# Cahier des Charges : bare-metal/dedicated - Modal Diagnostic Matériel

## 1. VUE D'ENSEMBLE
- **Modal** : Diagnostic matériel
- **Type** : Modal affichage résultats
- **Objectif** : Afficher l'état de santé des composants

## 2. STRUCTURE MODAL
- Modal blanche (800x650px)
- Header avec titre + bouton fermer
- Bandeau info + date dernier diagnostic
- État général (banner vert)
- Grille composants (6 cartes)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu)
- **OdsCard** : État général + composants
- **OdsBadge** : OK (vert), Attention (jaune)
- **OdsProgressBar** : Santé disques
- **OdsButton** : Télécharger PDF (outline), Lancer un diagnostic (primary)

## 4. HEADER
- **Titre** : "Diagnostic matériel"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
- "ℹ️ Le diagnostic matériel vérifie l'état de santé des composants de votre serveur."
- "Dernier diagnostic : 26 décembre 2025 à 14:30"

## 6. ÉTAT GÉNÉRAL
Banner vert avec bordure :
"✓ Tous les composants fonctionnent correctement"

## 8. BOUTONS
- **Télécharger PDF** : Outline, exporte le rapport
- **Lancer un diagnostic** : Primary, lance un nouveau diagnostic

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/diagnostic
  - Récupère le dernier diagnostic
- POST /dedicated/server/{serviceName}/diagnostic
  - Lance un nouveau diagnostic
