# Cahier des Charges : bare-metal/dedicated - Modal vRack Attach

## 1. VUE D'ENSEMBLE
- **Modal** : Attacher au vRack
- **Type** : Modal formulaire simple
- **Objectif** : Attacher une interface au vRack

## 2. STRUCTURE MODAL
- Modal blanche (600x340px)
- Header gris avec titre + bouton fermer
- Texte explicatif
- Sélection vRack
- Note d'information
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Description
- **OdsSelect** : vRack disponibles
- **OdsButton** : Annuler (outline), Attacher (primary)

## 4. HEADER
- **Titre** : "Attacher au vRack"
- **Bouton fermer** : ✕

## 5. CONTENU
- **Interface** : eth1
- **Description** : "Sélectionnez le vRack auquel attacher l'interface eth1"

## 6. SÉLECTION VRACK
- **Label** : "vRack disponibles"
- **Select** : pn-123456 (Europe) ▼

## 7. NOTE
- "Le vRack permet de créer un réseau privé entre vos services OVHcloud."
- "La configuration réseau de votre serveur devra être adaptée."

## 8. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Attacher** : Primary, attache l'interface au vRack

## 9. APIS & ENDPOINTS
- GET /vrack
  - Liste des vRacks disponibles
- POST /vrack/{serviceName}/dedicatedServer
  - Body: { dedicatedServer }
  - Attache le serveur au vRack
