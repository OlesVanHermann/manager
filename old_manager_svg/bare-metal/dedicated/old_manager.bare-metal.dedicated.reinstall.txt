# Cahier des Charges : dedicated - Onglet Reinstall

## 1. VUE D'ENSEMBLE
- **Page** : Réinstallation serveur dédié
- **Type** : Formulaire multi-étapes
- **Objectif** : Réinstaller l'OS du serveur

## 2. STRUCTURE PAGE
- Bandeau danger warning
- Section choix OS (cartes radio)
- Section options installation
- Bouton réinstaller

## 3. COMPOSANTS UI
- **OdsMessage** : Danger warning
- **OdsRadio** : Sélection OS (cartes)
- **OdsSelect** : Options RAID, partition, SSH
- **OdsButton** : Réinstaller
- **OdsLink** : Ajouter clé SSH

## 4. MESSAGE DANGER
| Type | Message |
|------|---------|
| Danger rouge | ⚠️ Attention : la réinstallation effacera toutes les données. Assurez-vous d'avoir sauvegardé vos données avant de procéder. |

## 5. CHOIX OS (CARTES RADIO)
| OS | Version | Sélection |
|----|---------|-----------|
| Debian | 12 (Bookworm) - 64 bits | ● Sélectionné |
| Ubuntu | 24.04 LTS - 64 bits | ○ |
| Rocky Linux | 9 - 64 bits | ○ |
| Windows Server | 2022 Standard (licence requise) | ○ |

## 6. OPTIONS D'INSTALLATION
| Option | Type | Valeur par défaut |
|--------|------|-------------------|
| Configuration RAID | Select | RAID 1 (Software) |
| Schéma de partition | Select | Par défaut (recommandé) |
| Clé SSH | Select | Ma clé SSH principale |

## 7. LIEN SSH
| Action | Type |
|--------|------|
| + Ajouter une nouvelle clé SSH | Link |

## 8. BOUTON ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Réinstaller | Primary | Lance réinstallation |

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/install/compatibleTemplates - OS disponibles
- GET /me/sshKey - Clés SSH
- POST /dedicated/server/{serviceName}/install/start - Lancer installation
- GET /dedicated/server/{serviceName}/install/status - Statut installation
