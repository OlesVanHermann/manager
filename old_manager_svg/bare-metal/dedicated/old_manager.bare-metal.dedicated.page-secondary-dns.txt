# Cahier des Charges : bare-metal/dedicated - Page Secondary DNS

## 1. VUE D'ENSEMBLE
- **Type** : Page listing DNS secondaire
- **Contexte** : Serveur dédié
- **Objectif** : Gérer les serveurs DNS secondaires pour les domaines

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajout domaine
- **OdsDatagrid** : Tableau domaines
- **OdsPagination** : Navigation pages

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "DNS Secondaire" |
| Description | "Gérez les serveurs DNS secondaires pour vos domaines." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter un domaine" |
| Style | Fond #0050D7, texte blanc |
| Dimensions | 200x40px |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Domaine | 330px | Nom du domaine |
| IP du serveur DNS | 250px | Adresse IP serveur maître |
| Date de création | 250px | Date ajout |
| Actions | 160px | Bouton supprimer |

## 7. BOUTON SUPPRIMER
| Propriété | Valeur |
|-----------|--------|
| Texte | "Supprimer" |
| Style | Fond blanc, bordure #DC2626, texte #DC2626 |
| Dimensions | 80x32px |

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "1 - 3 sur 3" |
| Position | Bas gauche |

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/secondaryDnsDomains - Liste domaines
- POST /dedicated/server/{serviceName}/secondaryDnsDomains - Ajouter domaine
- DELETE /dedicated/server/{serviceName}/secondaryDnsDomains/{domain} - Supprimer
