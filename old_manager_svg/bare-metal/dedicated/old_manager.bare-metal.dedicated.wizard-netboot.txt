# Cahier des Charges : bare-metal/dedicated - Wizard Netboot

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes
- **Contexte** : Configuration netboot serveur dédié
- **Objectif** : Configurer le mode de démarrage (disque, rescue, réseau)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 650px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Navigation retour
- **OdsStepper** : Étapes wizard
- **OdsCard** : Options de boot
- **OdsRadio** : Sélection options
- **OdsSelect** : Kernel rescue
- **OdsInput** : Email notification
- **OdsMessage** : Info rescue

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/boot - Liste boots disponibles

- PUT /dedicated/server/{serviceName} - Modifier netboot
- POST /dedicated/server/{serviceName}/reboot - Redémarrer
