## 5. DONNEES & COLONNES

### 5.1 Structure intervention

```typescript
interface Intervention {
  date: Date;             // Date de l'intervention
  type: string;           // Type d'intervention
  description?: string;   // Description optionnelle
}
```

### 5.2 Types d'interventions courants

| Type | Description |
|------|-------------|
| Remplacement disque dur | Changement d'un disque défaillant |
| Maintenance préventive | Contrôle périodique du matériel |
| Remplacement alimentation | Changement PSU |
| Diagnostic réseau | Vérification connexions réseau |
| Remplacement mémoire RAM | Changement de barrettes RAM |

## 7. APIS & ENDPOINTS

### 7.1 Liste des interventions

```
GET /dedicated/server/{serverName}/intervention
```

**Paramètres de pagination :**
```
?pageSize=5&page=1
```

**Réponse :**
```json
[123456, 123457, 123458]  // IDs des interventions
```

### 7.2 Détails d'une intervention

```
GET /dedicated/server/{serverName}/intervention/{interventionId}
```

**Réponse :**
```json
{
  "interventionId": 123456,
  "date": "2024-12-15T14:32:10+01:00",
  "type": "Remplacement disque dur"
}
```
