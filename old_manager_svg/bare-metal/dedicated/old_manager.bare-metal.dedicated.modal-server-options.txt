# Cahier des Charges : bare-metal/dedicated - Modal Options Serveur

## 1. VUE D'ENSEMBLE
- **Modal** : Options du serveur
- **Type** : Modal vue d'ensemble avec toggles
- **Objectif** : Gérer toutes les options du serveur

## 2. STRUCTURE MODAL
- Modal blanche (800x600px)
- Header avec titre + bouton fermer
- Sections par catégorie
- Toggles pour chaque option
- Bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsCard** : Options (6)
- **OdsBadge** : État (Actif/Inactif)
- **OdsToggle** : Activation options
- **OdsText** : Prix
- **OdsButton** : Fermer (primary)

## 4. HEADER
- **Titre** : "Options du serveur"
- **Bouton fermer** : ×

## 5. SECTION MONITORING
| Option | État | Prix | Toggle |
|--------|------|------|--------|
| OVH Monitoring | Actif ✅ | Inclus | ON |

- **Description** : "Surveillance proactive et alertes automatiques"

## 9. BOUTONS
- **Fermer** : Primary, ferme la modal

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/option
  - Liste des options actives
- PUT /dedicated/server/{serviceName}/serviceMonitoring/{monitoringId}
  - Active/désactive le monitoring
- POST /dedicated/server/{serviceName}/option
  - Body: { option }
  - Active une option
- DELETE /dedicated/server/{serviceName}/option/{option}
  - Désactive une option
