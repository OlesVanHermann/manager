# Cahier des Charges : bare-metal/dedicated - Wizard Mot de Passe FTP Backup

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de confirmation (1 étape)
- **Contexte** : Gestion backup serveur dédié
- **Objectif** : Demander un nouveau mot de passe pour le Backup FTP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 700px |
| Page hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étape (1 seule)
- **OdsButton** : Annuler, Envoyer
- **OdsText** : Instructions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mot de passe oublié - Backup FTP" |

## 5. CONTENU

### Indicateur d'étape
| Étape | Label | État |
|-------|-------|------|
| 1 | Confirmation | En cours (cercle bleu) |

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|
| Container | Blanc, border #E5E7EB, radius 8px |
| Ligne 1 | "Voulez-vous recevoir un nouveau mot de passe pour votre" |
| Ligne 2 | "Backup FTP {ftp_hostname} ?" |
| Hostname | Bleu (#0050D7), gras |

### Exemple hostname
| Propriété | Valeur |
|-----------|--------|
| Format | ftpback-{dc}-{id}.ovh.net |
| Exemple | ftpback-rbx1-123.ovh.net |

### Note d'information
| Propriété | Valeur |
|-----------|--------|
| Texte | "Un email contenant le nouveau mot de passe vous sera envoyé." |
| Couleur | Gris (#6B7280) |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Envoyer | Droite | Primary (#0050D7) | Envoie le nouveau mot de passe |

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/features/backupFTP/password - Demander nouveau mot de passe
