# Cahier des Charges : bare-metal/dedicated - Housing Dashboard

## 1. VUE D'ENSEMBLE
- **Type** : Page dashboard avec tuiles
- **Contexte** : Gestion housing (colocation)
- **Objectif** : Afficher les informations d'un housing

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Tableau de bord / Backup FTP / Tâches
- **OdsCard** : 4 tuiles informations

## 4. BREADCRUMB
| Niveau | Texte | Lien |
|--------|-------|------|
| 1 | Bare Metal Cloud | Oui |
| 2 | Housing | Oui |
| 3 | {housingName} | Non |

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Tableau de bord | Actif |
| Backup FTP | Inactif |
| Tâches | Inactif |
