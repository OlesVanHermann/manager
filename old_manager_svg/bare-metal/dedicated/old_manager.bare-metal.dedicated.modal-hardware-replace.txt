# Cahier des Charges : bare-metal/dedicated - Modal Demande Remplacement Matériel

## 1. VUE D'ENSEMBLE
- **Modal** : Demande de remplacement matériel
- **Type** : Modal formulaire avec sélection
- **Objectif** : Soumettre une demande d'intervention matérielle

## 2. STRUCTURE MODAL
- Modal blanche (700x600px)
- Header avec titre + bouton fermer
- Warning banner
- Sélection composant (3 options)
- Détails composant (si disque)
- Description problème
- Checkboxes confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (fond jaune)
- **OdsTile** : Sélection composant (3)
- **OdsSelect** : Disque concerné
- **OdsTextarea** : Description problème
- **OdsCheckbox** : Confirmations
- **OdsButton** : Annuler (outline), Soumettre la demande (primary)

## 4. HEADER
- **Titre** : "Demande de remplacement matériel"
- **Bouton fermer** : ×

## 5. WARNING BANNER
Fond jaune avec bordure orange :
- "⚠️ Le remplacement nécessite une intervention en datacenter."
- "Temps d'intervention estimé : 2-4 heures. Une interruption de service est à prévoir."

## 6. SÉLECTION COMPOSANT
| Composant | Description | État |
|-----------|-------------|------|
| Disque | SSD/HDD défaillant | ✅ Sélectionné |
| Mémoire RAM | Barrette défaillante | - |
| Autre | PSU, carte réseau... | - |

## 7. DÉTAILS COMPOSANT
Si Disque sélectionné :
- **Label** : "Disque concerné"
- **Select** : "Disque 2 - /dev/nvme1n1" ▼

## 8. DESCRIPTION PROBLÈME
- **Label** : "Description du problème"
- **Textarea** (640x100px) :
  - "Le disque 2 présente des signes de défaillance avec un"
  - "taux de santé SMART à 72%. Secteurs défectueux détectés."
  - "Merci de procéder au remplacement préventif."

## 9. CHECKBOXES
| Checkbox | Texte | État |
|----------|-------|------|
| ✅ | J'ai sauvegardé mes données importantes | Coché |
| ☐ | Me contacter avant l'intervention | Non coché |

## 10. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Soumettre la demande** : Primary, crée l'intervention

## 11. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/intervention
  - Body: { component, disk, description, contactBefore }
  - Crée une demande d'intervention
