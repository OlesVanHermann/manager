# Cahier des Charges : bare-metal/dedicated - Onglet Bande Passante

## 1. VUE D'ENSEMBLE
- **Onglet** : Bande passante
- **Type** : Page statistiques avec graphique
- **Objectif** : Visualiser le trafic réseau du serveur

## 2. STRUCTURE PAGE
- Titre + sélecteur de période
- 4 cartes statistiques
- Graphique trafic réseau
- Cartes interfaces réseau

## 3. COMPOSANTS UI
- **OdsButton** : Sélecteur période (24h, 7j, 30j)
- **OdsCard** : Statistiques (4)
- **OdsChart** : Graphique ligne avec zones
- **OdsBadge** : État (Illimité, Up)
- **OdsProgressBar** : Usage bande passante

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/statistics
  - Params: period (day, week, month)
  - Statistiques de trafic
- GET /dedicated/server/{serviceName}/bandwidth
  - Informations bande passante
- GET /dedicated/server/{serviceName}/networkInterfaceController
