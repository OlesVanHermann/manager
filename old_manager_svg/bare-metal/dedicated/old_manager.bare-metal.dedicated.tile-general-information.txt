# Cahier des Charges : bare-metal/dedicated - Tile General Information

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard principale
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les informations générales du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsText** : Labels et valeurs
- **OdsPopover** : Menus actions
- **OdsFlag** : Drapeau datacenter

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Informations générales" |
