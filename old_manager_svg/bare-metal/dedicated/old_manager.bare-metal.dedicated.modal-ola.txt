# Cahier des Charges : bare-metal/dedicated - Modal OLA Simple

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer l'agrégation réseau (OLA)
- **Type** : Modal configuration simple
- **Objectif** : Activer l'agrégation de liens

## 2. STRUCTURE MODAL
- Modal blanche (600x420px)
- Header gris avec titre + bouton fermer
- Bandeau info
- Select mode agrégation
- Checkboxes interfaces
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu clair)
- **OdsSelect** : Mode d'agrégation
- **OdsCheckbox** : Interfaces (2)
- **OdsButton** : Annuler (outline), Configurer (primary)

## 4. HEADER
- **Titre** : "Configurer l'agrégation réseau (OLA)"
- **Bouton fermer** : ✕

## 5. BANDEAU INFO
Fond bleu clair avec bordure :
- "ℹ OVH Link Aggregation permet d'agréger plusieurs interfaces"
- "réseau pour augmenter la bande passante."

## 6. MODE D'AGRÉGATION
- **Label** : "Mode d'agrégation *"
- **Select** : LACP (802.3ad) ▼

## 7. INTERFACES À AGRÉGER
- **Label** : "Interfaces à agréger"
- **Description** : "Sélectionnez les interfaces réseau à agréger."

| Interface | État |
|-----------|------|
| ✅ eth0 - Public (1 Gbps) | Coché |
| ✅ eth1 - Public (1 Gbps) | Coché |

## 8. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Configurer** : Primary, applique l'agrégation

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/virtualNetworkInterface
  - Liste des interfaces disponibles
- POST /dedicated/server/{serviceName}/ola/group
  - Body: { mode, interfaces[] }
  - Active l'agrégation
