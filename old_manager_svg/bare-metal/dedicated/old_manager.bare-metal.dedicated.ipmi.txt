# Cahier des Charges : dedicated - Onglet IPMI

## 1. VUE D'ENSEMBLE
- **Page** : Console IPMI/KVM serveur dédié
- **Type** : Onglet avec actions
- **Objectif** : Accéder à la console hors-bande du serveur

## 2. STRUCTURE PAGE
- Bandeau warning information
- Carte Console KVM/IPMI
- Carte Actions de contrôle

## 3. COMPOSANTS UI
- **OdsMessage** : Warning information
- **OdsCard** : Cartes sections
- **OdsBadge** : Statut IPMI
- **OdsButton** : Actions KVM
- **OdsSelect** : Configuration netboot

## 4. MESSAGE WARNING
| Type | Message |
|------|---------|
| Warning orange | L'accès IPMI nécessite Java ou une connexion HTML5. Les sessions expirent après 20 minutes d'inactivité. |

## 5. CARTE CONSOLE KVM
| Info | Valeur |
|------|--------|
| Statut IPMI | Badge Disponible (vert) |
| Type d'accès | iLO 5 (HPE) |
| Dernière connexion | 26/12/2025 10:32 |

## 6. BOUTONS KVM
| Bouton | Style | Action |
|--------|-------|--------|
| Lancer KVM (HTML5) | Primary | Ouvre console HTML5 |
| Lancer KVM (Java) | Outline | Ouvre console Java |
| Télécharger JNLP | Link | Download fichier JNLP |

## 7. CARTE ACTIONS DE CONTRÔLE
| Action | Style | Description |
|--------|-------|-------------|
| Redémarrer | Outline | Reboot serveur |
| Éteindre (soft) | Outline | Shutdown propre |
| Éteindre (hard) | Danger (rouge) | Coupure alimentation |

| Message |

## 9. BOOT RÉSEAU
| Élément | Type |
|---------|------|
| Label | Boot réseau |
| Select | Configurer netboot |

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/features/ipmi - Statut IPMI
- POST /dedicated/server/{serviceName}/features/ipmi/access - Accès KVM
- POST /dedicated/server/{serviceName}/reboot - Redémarrer
- POST /dedicated/server/{serviceName}/boot - Boot configuration
