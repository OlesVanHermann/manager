# Cahier des Charges : bare-metal/dedicated - Tile KVM Information

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les informations KVM (IPMI over IP)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 230px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsText** : Labels et valeurs
- **OdsBadge** : État actif

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Informations KVM" |

## 5. CHAMPS AFFICHÉS

### Nom
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom" |
| Valeur | {kvmName} "kvm-ns123456" |

### Adresse IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP" |
| Valeur | {kvmIp} "10.200.45.67" |

### Expiration
| Propriété | Valeur |
|-----------|--------|
| Label | "Expiration" |
| Valeur | {expirationDate} "31 décembre 2025" |
| Badge | Fond #DCFCE7, texte #16A34A "Actif" |

## 6. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/features/ipmi - Informations IPMI
- GET /dedicated/server/{serviceName}/features/kvm - Informations KVM
