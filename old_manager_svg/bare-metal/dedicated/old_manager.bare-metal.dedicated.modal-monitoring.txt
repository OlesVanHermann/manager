# Cahier des Charges : bare-metal/dedicated - Modal Monitoring Simple

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer le monitoring
- **Type** : Modal configuration simple
- **Objectif** : Activer/désactiver la surveillance OVH

## 2. STRUCTURE MODAL
- Modal blanche (500x330px)
- Header gris avec titre + bouton fermer
- Description
- Toggle monitoring
- Toggle alertes SMS
- Email notification (lecture seule)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Description
- **OdsToggle** : Monitoring (2)
- **OdsButton** : Fermer (outline), Enregistrer (primary)

## 4. HEADER
- **Titre** : "Configurer le monitoring"
- **Bouton fermer** : ✕

## 5. DESCRIPTION
"Le monitoring permet de surveiller l'état de votre serveur et de recevoir des alertes en cas de problème."

## 6. TOGGLES
| Option | Description | État |
|--------|-------------|------|
| Activer le monitoring | - | ✅ ON |
| Alertes SMS | Recevez des alertes par SMS en cas d'indisponibilité. | ⭘ OFF |

## 7. EMAIL NOTIFICATION
- **Label** : "Email de notification"
- **Valeur** : admin@example.com (compte OVHcloud)

## 8. BOUTONS
- **Fermer** : Outline, ferme la modal
- **Enregistrer** : Primary, sauvegarde les paramètres

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/serviceMonitoring
  - Configuration actuelle
- PUT /dedicated/server/{serviceName}/serviceMonitoring
  - Body: { monitoring, smsAlert }
  - Active/désactive le monitoring
