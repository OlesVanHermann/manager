# Cahier des Charges : bare-metal/dedicated - Modal Modifier Nom Cluster

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Gestion cluster serveurs dédiés
- **Objectif** : Modifier le nom d'affichage du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champ nom d'affichage
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le nom du cluster" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Description | "Personnalisez le nom affiché pour ce cluster." |
| Label champ | "Nom d'affichage" |
| Type input | Text |
| Largeur input | 350px |
| État focus | Bordure #0050D7 (2px) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary #0050D7 | Enregistre le nom |

## 7. APIS & ENDPOINTS
- PUT /dedicated/cluster/{serviceName} - Modifier displayName
