# Cahier des Charges : bare-metal/dedicated - Modal Renommer Interface

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification simple
- **Contexte** : Gestion réseau serveur dédié
- **Objectif** : Renommer une interface réseau pour faciliter son identification

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champ nom
- **OdsText** : Labels et aide
- **OdsButton** : Annuler, Renommer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Renommer l'interface" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Nom de l'interface | OdsInput | Nom personnalisé |

### Style champ focus
| Propriété | Valeur |
|-----------|--------|
| Bordure | Bleue (#0050D7) 2px |

### Texte d'aide
| Propriété | Valeur |
|-----------|--------|
| Texte | "Le nom permet d'identifier facilement vos interfaces réseau." |
| Couleur | Gris (#6B7280) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Renommer | Primary (bleu #0050D7) | Enregistre le nom |

## 7. APIS & ENDPOINTS
- PUT /dedicated/server/{serviceName}/virtualNetworkInterface/{uuid} - Modifier interface
