# Cahier des Charges : bare-metal/dedicated - Modal Redémarrer Serveur

## 1. VUE D'ENSEMBLE
- **Modal** : Redémarrer le serveur
- **Type** : Modal confirmation avec options
- **Objectif** : Redémarrer le serveur (soft ou hard)

## 2. STRUCTURE MODAL
- Modal blanche (600x480px)
- Header avec titre + bouton fermer
- Warning banner
- Options radio (soft/hard reboot)
- Info serveur
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (fond jaune)
- **OdsRadio** : Type de redémarrage (2)
- **OdsCard** : Info serveur
- **OdsButton** : Annuler (outline), Redémarrer (danger/rouge)

## 4. HEADER
- **Titre** : "Redémarrer le serveur"
- **Bouton fermer** : ×

## 5. WARNING BANNER
Fond jaune avec bordure orange :
- "⚠️ Le redémarrage interrompra tous les services en cours."
- "Assurez-vous d'avoir sauvegardé vos données."

## 6. TYPE DE REDÉMARRAGE
| Type | Description | État |
|------|-------------|------|
| Redémarrage logiciel (soft) | Envoie une commande de redémarrage propre au système. Recommandé pour la plupart des cas. | ✅ Sélectionné |
| Redémarrage matériel (hard) | Coupe l'alimentation puis redémarre le serveur. À utiliser si le serveur ne répond plus. | - |

## 7. INFO SERVEUR
Fond gris :
- **Label** : "Serveur"
- **Valeur** : ns1234567.ip-192-168-1.eu

## 8. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Redémarrer** : Danger (rouge), exécute le redémarrage

## 9. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/reboot
  - Body: { type: "soft" | "hard" }
  - Redémarre le serveur
