# Cahier des Charges : bare-metal/dedicated - Page IPMI Console

## 1. VUE D'ENSEMBLE
- **Type** : Page console IPMI
- **Contexte** : Serveur dédié
- **Objectif** : Accéder à la gestion distante du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

- **OdsCard** : Statut, options accès, actions

- POST /dedicated/server/{serviceName}/features/ipmi/access - Demander accès KVM/SOL

- GET /dedicated/server/{serviceName}/features/ipmi/access - URL session
