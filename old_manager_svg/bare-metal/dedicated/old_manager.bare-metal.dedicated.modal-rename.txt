# Cahier des Charges : bare-metal/dedicated - Modal Renommer Serveur

## 1. VUE D'ENSEMBLE
- **Modal** : Renommer le serveur
- **Type** : Modal formulaire simple
- **Objectif** : Modifier le nom d'affichage du serveur

## 2. STRUCTURE MODAL
- Modal blanche (600x280px)
- Header gris avec titre + bouton fermer
- Formulaire simple
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom d'affichage
- **OdsButton** : Annuler (outline), Renommer (primary)

## 4. HEADER
- **Titre** : "Renommer le serveur"
- **Bouton fermer** : ✕

## 5. FORMULAIRE
- **Label** : "Nom d'affichage"
- **Description** : "Ce nom est utilisé uniquement dans l'interface manager."
- **Input** : "Serveur Production Web"
- **Largeur** : 552px

## 6. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Renommer** : Primary, applique le nouveau nom

## 7. APIS & ENDPOINTS
- PUT /dedicated/server/{serviceName}
  - Body: { displayName: "Serveur Production Web" }
  - Met à jour le nom d'affichage
