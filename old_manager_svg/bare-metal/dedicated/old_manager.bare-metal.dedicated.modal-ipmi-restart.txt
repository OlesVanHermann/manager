# Cahier des Charges : bare-metal/dedicated - Modal IPMI Restart

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion serveur dédié
- **Objectif** : Confirmer la réinitialisation des interfaces IPMI

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 180px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Container modal
- **OdsText** : Contenu
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Hauteur | 55px |
| Titre | "Réinitialiser les interfaces IPMI" |
| Bouton fermer | × (gris #6B7280) |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Texte L1 | "Cette action va réinitialiser les interfaces IPMI de votre" |
| Texte L2 | "serveur. L'opération peut prendre quelques minutes." |
| Couleur texte | #1F2937 |

## 6. BOUTONS ACTION
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | Fond blanc, bordure #E5E7EB, texte #6B7280 | 90x36px |
| Confirmer | Fond #0050D7, texte blanc | 90x36px |

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/features/ipmi/resetInterface - Réinitialiser IPMI
