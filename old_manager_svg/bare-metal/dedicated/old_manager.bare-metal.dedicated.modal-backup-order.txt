# Cahier des Charges : bare-metal/dedicated - Modal Commander Backup Storage

## 1. VUE D'ENSEMBLE
- **Modal** : Commander Backup Storage
- **Type** : Modal commande avec sélection
- **Objectif** : Commander un espace de stockage backup

## 2. STRUCTURE MODAL
- Modal blanche (700x610px)
- Header avec titre + bouton fermer
- Bandeau info
- Sélection capacité (4 options)
- Liste fonctionnalités
- Récapitulatif prix
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu)
- **OdsTile** : Cartes capacité (4)
- **OdsList** : Fonctionnalités (checkmarks)
- **OdsCard** : Récapitulatif
- **OdsButton** : Annuler (outline), Commander (primary)

## 4. HEADER
- **Titre** : "Commander Backup Storage"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
- "ℹ️ Le Backup Storage est un espace de stockage accessible via FTP/NFS/CIFS."
- "Idéal pour sauvegarder vos données serveur."

## 6. SÉLECTION CAPACITÉ
| Capacité | Supplément | Prix |
|----------|------------|------|
| 500 Go | Inclus | Gratuit (vert) |
| 1 To | +500 Go | +9,99 €/mois ✅ Sélectionné |
| 5 To | +4,5 To | +39,99 €/mois |
| 10 To | +9,5 To | +69,99 €/mois |

## 7. FONCTIONNALITÉS INCLUSES
Deux colonnes :
| Colonne 1 | Colonne 2 |
|-----------|-----------|
| ✓ Accès FTP, FTPS, NFS, CIFS | ✓ Chiffrement des données |
| ✓ Réplication dans le même datacenter | ✓ Bande passante illimitée |
| ✓ Gestion des accès par IP | ✓ Support 24/7 |
| ✓ Snapshots quotidiens automatiques | |

## 8. RÉCAPITULATIF
Fond gris :
- "Backup Storage 1 To pour ns1234567.ip-192-168-1.eu"
- "Total mensuel : 9,99 € HT"

## 9. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Commander** : Primary, passe commande

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/features/backupCloud/availability
  - Vérifie disponibilité du backup
- POST /order/dedicated/server/{serviceName}/backupStorage
  - Body: { capacity: "1000" }
  - Commande le backup storage
