# Cahier des Charges : bare-metal/dedicated - Modal Renouvellement

## 1. VUE D'ENSEMBLE
- **Modal** : Renouveler votre serveur
- **Type** : Modal sélection durée avec pricing
- **Objectif** : Renouveler l'abonnement du serveur dédié

## 2. STRUCTURE MODAL
- Modal blanche (800x530px)
- Header avec titre + bouton fermer
- Bandeau info serveur
- Sélection durée (4 options)
- Options incluses (checkboxes)
- Résumé renouvellement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info serveur (fond bleu)
- **OdsTile** : Options durée (4)
- **OdsRadio** : Sélection durée
- **OdsBadge** : Remises (-10%, -20%, -30%)
- **OdsCheckbox** : Options incluses
- **OdsCard** : Résumé
- **OdsButton** : Annuler (outline), Payer (primary)

## 4. HEADER
- **Titre** : "Renouveler votre serveur"
- **Bouton fermer** : ×
