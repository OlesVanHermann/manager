# Cahier des Charges : bare-metal/dedicated - Page Tag Manager

## 1. VUE D'ENSEMBLE
- **Type** : Page listing avec tableau
- **Contexte** : Gestion serveur dédié
- **Objectif** : Gérer les étiquettes (tags) du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Tableau largeur | 870px |

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsButton** : Toolbar actions
- **OdsDatagrid** : Liste tags
- **OdsCheckbox** : Sélection multiple

## 4. NAVIGATION
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "‹ Retour au dashboard" |

## 5. TOOLBAR
| Bouton | Style | État |
|--------|-------|------|
| + Attribuer une étiquette | Secondary | Actif |
| Retirer l'étiquette | Disabled (#F3F4F6) | Grisé si aucune sélection |
| Guides | Secondary | Actif |

## 6. COLONNES DATAGRID
| Colonne | Largeur | Type |
|---------|---------|------|
| Checkbox | 40px | Sélection |
| Étiquette | 360px | Texte |
| Type | 420px | Texte + info ⓘ |
| Actions | 50px | Menu ⋮ |

## 7. TYPES D'ÉTIQUETTES
| Type | Description |
|------|-------------|
| IAM | Étiquette IAM système |
| Personnalisé | Étiquette créée par l'utilisateur |

## 8. MENU ACTIONS (⋮)
| Action |
|--------|
| Modifier |
| Supprimer |

## 9. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format gauche | "X résultats" |
| Format droite | "10 par page" |

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/tag - Liste tags serveur
- POST /dedicated/server/{serviceName}/tag - Ajouter tag
- DELETE /dedicated/server/{serviceName}/tag/{tagKey} - Supprimer tag
