# Cahier des Charges : bare-metal/dedicated - Onglet OLA

## 1. VUE D'ENSEMBLE
- **Onglet** : OVH Link Aggregation
- **Type** : Page configuration réseau avancée
- **Objectif** : Gérer l'agrégation de liens réseau

## 2. STRUCTURE PAGE
- Titre
- Bannière info
- Carte Configuration OLA
- Carte Connectivité vRack
- Cartes Interfaces membres

## 3. COMPOSANTS UI
- **OdsMessage** : Info (fond bleu)
- **OdsCard** : Sections (3)
- **OdsBadge** : États (Activé, Actif, VLAN IDs)
- **OdsLink** : Documentation, Configurer

## 4. BANNIÈRE INFO
Fond bleu clair :
- "ℹ️ OLA permet d'agréger les interfaces réseau de votre serveur pour augmenter la bande passante"
- "et la redondance. Cette fonctionnalité nécessite un serveur compatible avec plusieurs interfaces réseau."
- Lien : "En savoir plus sur OLA"

## 5. CARTE CONFIGURATION OLA
| Champ | Valeur |
|-------|--------|
| Statut | Activé ✅ |
| Mode | LACP (802.3ad) |
| Interfaces | eth0, eth1 |
| Bande passante | 2 × 1 Gbit/s = 2 Gbit/s |

## 6. CARTE CONNECTIVITÉ VRACK
| Champ | Valeur |
|-------|--------|
| vRack | pn-123456 |
| VLAN natif | Non taggé |
| VLANs configurés | 100, 200, 300 (badges bleus) |

- **Lien** : "Configurer"

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/ola
  - Configuration OLA actuelle
- GET /dedicated/server/{serviceName}/ola/group
  - Groupes d'agrégation
- PUT /dedicated/server/{serviceName}/ola/group/{name}
  - Modifie un groupe
- DELETE /dedicated/server/{serviceName}/ola/group/{name}
  - Supprime un groupe
