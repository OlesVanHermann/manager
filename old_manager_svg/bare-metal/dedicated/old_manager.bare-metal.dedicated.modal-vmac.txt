# Cahier des Charges : bare-metal/dedicated - Modal vMAC

## 1. VUE D'ENSEMBLE
- **Modal** : Créer une adresse MAC virtuelle
- **Type** : Modal formulaire configuration
- **Objectif** : Créer une vMAC associée à une IP Failover

## 2. STRUCTURE MODAL
- Modal blanche (700x540px)
- Header avec titre + bouton fermer
- Bandeau info explicatif
- Sélection IP Failover
- Sélection type vMAC (2 options)
- Nom VM optionnel
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu)
- **OdsSelect** : IP Failover
- **OdsTile** : Types vMAC (2)
- **OdsInput** : Nom VM
- **OdsButton** : Annuler (outline), Créer (primary)

## 4. HEADER
- **Titre** : "Créer une adresse MAC virtuelle"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
- "ℹ️ Une adresse MAC virtuelle (vMAC) est associée à une IP Failover."
- "Elle permet de déplacer l'IP entre serveurs sans reconfiguration."

## 6. IP FAILOVER
- **Label** : "IP Failover"
- **Select** : 192.168.1.200 ▼

## 7. TYPES DE vMAC
| Type | Description | Info | État |
|------|-------------|------|------|
| OVH | Préfixe 02:00:00 | Recommandé | ✅ Sélectionné |
| VMware | Préfixe 00:50:56 | Pour ESXi | - |

## 8. NOM VM
- **Label** : "Nom de la machine virtuelle"
- **Description** : "Optionnel - pour identifier l'usage"
- **Placeholder** : "Ex: vm-web-prod-01"

## 9. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Créer la vMAC** : Primary, crée l'adresse MAC virtuelle

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/virtualMac
  - Liste des vMAC existantes
- POST /dedicated/server/{serviceName}/virtualMac
  - Body: { ipAddress, type, virtualMachineName }
  - Crée une nouvelle vMAC
