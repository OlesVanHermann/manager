## 3. COMPOSANTS UI

### 3.1 Message d'avertissement

| Type | Contenu |
|------|---------|
| Warning (#FEF3C7) | La période sélectionnée peut affecter la précision des données. Par défaut, les dernières 24 heures sont affichées. |

### 3.2 Filtres

| Filtre | Type | Valeur par défaut |
|--------|------|-------------------|
| Filtrer par | Select | Tous les services |
| IP | Select | Sélectionner une IP |
| Période | Select | 24 heures |
| Timezone | Label | UTC+1 |

### 3.3 KPIs

| KPI | Valeur exemple | Description |
|-----|----------------|-------------|
| Nombre d'événements | 12 | Événements détectés |
| Paquets supprimés | 1.2M | Paquets filtrés |
| Bande passante nettoyée | 45 Gbps | Trafic mitigé |
| Attaques détectées | 3 | Nombre d'attaques |

### 3.4 Contrôles graphique

| Contrôle | Type | Description |
|----------|------|-------------|
| Actualiser | Bouton ↻ | Rafraîchit les données |
| Empiler les séries | Checkbox | Active le mode stacked |
| Mode | Toggle bps/pps | Bits ou paquets par seconde |

### 3.5 Graphique

| Élément | Description |
|---------|-------------|
| Type | Line chart avec aire remplie |
| Axe Y | 0 - 100 Gbps |
| Axe X | Heures (00:00 - 00:00) |
| Couleur | Bleu primary #0050D7 |

## 4. APIS & ENDPOINTS

### 4.1 Statistiques de trafic

```
GET /ip/{ip}/mitigation/{ipOnMitigation}/stats
?from={timestamp}
&to={timestamp}
&scale=5m
```

### 4.2 Métriques agrégées

```
GET /ip/{ip}/mitigation/stats
?period=day
```
