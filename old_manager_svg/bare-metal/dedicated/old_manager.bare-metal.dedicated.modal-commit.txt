# Cahier des Charges : bare-metal/dedicated - Modal Commit

## 1. VUE D'ENSEMBLE
- **Type** : Modal sélection radio avec réductions
- **Contexte** : Gestion serveur dédié
- **Objectif** : Souscrire un engagement avec économies

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsRadio** : Options engagement
- **OdsBadge** : Badge recommandé
- **OdsButton** : Annuler, S'engager

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "S'engager et économiser" |
| Fond | #F9FAFB |

## 5. INSTRUCTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Choisissez votre durée d'engagement pour bénéficier de réductions." |

## 6. OPTIONS ENGAGEMENT
| Durée | Économie | Style | Badge |
|-------|----------|-------|-------|
| 12 mois | -10% (vert #16A34A) | Fond blanc, bordure #E5E7EB | - |
| 24 mois | -20% (vert #16A34A) | Fond #EFF6FF, bordure #0050D7 | "Recommandé" |
| 36 mois | -30% (vert #16A34A) | Fond blanc, bordure #E5E7EB | - |

### Style option
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 70px |
| Radio sélectionné | Cercle bleu avec point central |
| Description | "Économisez X% sur votre abonnement" |

### Badge Recommandé
| Propriété | Valeur |
|-----------|--------|
| Fond | #DCFCE7 |
| Texte | #16A34A "Recommandé" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary, 90x36px | Ferme modal |
| S'engager | Primary #0050D7, 90x36px | Souscrit engagement |

## 8. APIS & ENDPOINTS
- GET /services/{serviceId}/billing/engagement/available - Options disponibles
- POST /services/{serviceId}/billing/engagement - Souscrire engagement
