# Cahier des Charges : bare-metal/dedicated - Page BIOS Settings

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration BIOS
- **Contexte** : Serveur dédié
- **Objectif** : Configurer les paramètres BIOS du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titres et descriptions
- **OdsCard** : Sections configuration
- **OdsRadio** : Mode de boot
- **OdsSwitch** : Toggles options
- **OdsButton** : Appliquer

## 4. NAVIGATION
| Propriété | Valeur |
|-----------|--------|
| Lien | "← Retour au serveur" |
| Couleur | #0050D7 |

## 5. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration BIOS" |
| Description | "Modifiez les paramètres BIOS de votre serveur." |

## 6. SECTION MODE DE BOOT
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mode de démarrage" |
| Fond | Blanc avec bordure |

## 7. OPTIONS RADIO BOOT
| Option | Description |
|--------|-------------|
| UEFI | "Mode UEFI (recommandé)" |
| Legacy | "Mode Legacy BIOS" |

## 8. STYLE RADIO
| État | Style |
|------|-------|
| Selected | Cercle bleu #0050D7 avec point |
| Default | Cercle blanc, bordure grise |

## 9. SECTION OPTIONS AVANCÉES
| Propriété | Valeur |
|-----------|--------|
| Titre | "Options avancées" |
| Fond | Blanc avec bordure |

## 10. TOGGLES OPTIONS
| Option | Description | État par défaut |
|--------|-------------|-----------------|
| C-States | "Activer les états C (économie d'énergie)" | OFF |
| Hyper-Threading | "Activer l'Hyper-Threading" | ON |
| Virtualization | "Activer la virtualisation (VT-x/AMD-V)" | ON |
| SGX | "Activer Intel SGX" | OFF |

## 11. STYLE TOGGLE
| État | Style |
|------|-------|
| ON | Fond #0050D7, cercle droite |
| OFF | Fond #E5E7EB, cercle gauche |

## 12. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Icône | ⚠️ |
| Texte | "La modification des paramètres BIOS nécessite un redémarrage du serveur." |

## 13. FOOTER ACTIONS
| Bouton | Style |
|--------|-------|
| Annuler | Fond blanc, bordure grise |
| Appliquer | Fond #0050D7, texte blanc |

## 14. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/biosSettings - Configuration actuelle
- PUT /dedicated/server/{serviceName}/biosSettings - Modifier configuration
- POST /dedicated/server/{serviceName}/reboot - Redémarrer après changement
