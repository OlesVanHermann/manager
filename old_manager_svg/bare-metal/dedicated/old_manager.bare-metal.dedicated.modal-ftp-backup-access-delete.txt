# Cahier des Charges : bare-metal/dedicated - Modal Supprimer Accès FTP Backup

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion du stockage FTP Backup
- **Objectif** : Supprimer un accès IP au stockage FTP Backup du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'accès" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Êtes-vous sûr de vouloir supprimer l'accès FTP Backup" |
| Ligne 2 | "pour le bloc IP {ip_block} ?" |
| IP/CIDR | Bleu (#0050D7), font-weight 500 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Danger (rouge #DC2626) | Supprime l'accès |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/server/{serviceName}/features/backupFTP/access/{ipBlock} - Supprimer accès
