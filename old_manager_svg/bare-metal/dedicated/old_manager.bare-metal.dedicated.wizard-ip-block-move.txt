## 7. APIS & ENDPOINTS

### 7.1 Liste des services compatibles

```
GET /ip/{ip}/destination
```

### 7.2 Déplacer le bloc IP

```
POST /ip/{ip}/move
{
  "to": "ns123456.ip-192-168-1.eu",
  "nexthop": "192.168.1.1"
}
```

### 7.3 Mettre en parking

```
POST /ip/{ip}/park
```
