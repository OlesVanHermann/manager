# Cahier des Charges : bare-metal/housing - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Housing - Récapitulatif
- **Type** : Dashboard détail
- **Objectif** : Afficher les informations d'un serveur housing

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb
- Titre (nom du serveur)
- Onglets (Récapitulatif, Stockage backup, Tâches)
- 3 cartes en ligne (Général, Réseau, Routage)
- Carte Abonnement

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud › Housing
- **OdsTabs** : 3 onglets
- **OdsCard** : Général, Réseau, Routage, Abonnement
- **OdsBadge** : Normal (vert)
- **OdsActionMenu** : Menu actions par carte
- **OdsText** : Labels et valeurs

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Récapitulatif | Vue d'ensemble du housing |
| Stockage backup | Gestion du backup FTP |
| Tâches | Historique des tâches |
