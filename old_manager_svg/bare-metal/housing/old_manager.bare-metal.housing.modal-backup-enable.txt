# Cahier des Charges : bare-metal/housing - Activer backup

## 1. VUE D'ENSEMBLE
- **Page** : Activer le backup storage
- **Type** : Modal confirmation
- **Objectif** : Activer le service de backup storage

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Icône info + question
- Message explicatif
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsText** : Question de confirmation
- **OdsMessage** : Info (bleu)
- **OdsButton** : Annuler (outline), Activer (primary)

## 4. QUESTION
Êtes-vous certain(e) de vouloir activer le service de backup storage pour ce serveur ?

## 5. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte 1 | Le backup storage sera activé dans quelques instants après confirmation. |
| Texte 2 | Un email vous sera envoyé avec les informations de connexion. |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Activer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/housing/{serviceName}/features/backupFTP
  - Activer le backup storage
