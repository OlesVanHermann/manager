# Cahier des Charges : bare-metal/housing - Supprimer accès backup

## 1. VUE D'ENSEMBLE
- **Page** : Suppression d'un accès
- **Type** : Modal confirmation
- **Objectif** : Supprimer un accès IP du backup storage

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Icône corbeille + question
- Message info délai
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal danger
- **OdsText** : Question (avec bloc IP)
- **OdsText** : Info délai
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. QUESTION
Êtes-vous certain(e) de vouloir supprimer l'accès du block IP **123.45.67.0/24** ?

## 5. MESSAGE INFO
Cette action peut prendre quelques minutes à être appliquée.

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Supprimer | Danger (rouge) | Droite |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/housing/{serviceName}/features/backupFTP/access/{ipBlock}
  - Supprimer l'accès IP
