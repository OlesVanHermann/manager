# Cahier des Charges : bare-metal/housing - Page Bandwidth

## 1. VUE D'ENSEMBLE
- **Type** : Page consommation bande passante
- **Contexte** : Housing datacenter
- **Objectif** : Visualiser l'utilisation réseau de la baie

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et valeurs
- **OdsTabs** : Sélecteur période
- **OdsCard** : Statistiques
- **OdsProgressBar** : Quota
- **OdsChart** : Graphique trafic

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Consommation bande passante" |
| Description | "Suivez la consommation réseau de votre baie." |
