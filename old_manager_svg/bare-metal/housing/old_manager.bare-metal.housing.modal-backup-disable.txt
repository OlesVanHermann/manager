# Cahier des Charges : bare-metal/housing - Supprimer backup

## 1. VUE D'ENSEMBLE
- **Page** : Supprimer le backup storage
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer le service de backup storage

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Icône warning + question
- Message d'avertissement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal danger
- **OdsText** : Question (avec nom backup)
- **OdsMessage** : Warning irréversible (rouge)
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. QUESTION
Êtes-vous certain(e) de vouloir supprimer le backup storage **ftpback-rbx1-123** ?
