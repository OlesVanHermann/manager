# Cahier des Charges : bare-metal/ip - Modal supprimer règle Game Firewall

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Game Firewall (protection jeux)
- **Objectif** : Confirmer la suppression d'une règle Game Firewall

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsCard** : Détails de la règle
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la règle" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message
| Propriété | Valeur |
|-----------|--------|
| Texte | "Êtes-vous sûr de vouloir supprimer la règle :" |

### Détails de la règle
| Propriété | Valeur |
|-----------|--------|
| Fond | Gris (#F3F4F6) |
| Format | "{protocole} ports {port_debut} - {port_fin}" |
| Exemple | "UDP ports 27015 - 27030" |
| Protocole | Gras |
| Ports | Gras |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime la règle |

## 7. APIS & ENDPOINTS
- DELETE /ip/{ip}/game/{gameFirewallRule} - Supprimer la règle
