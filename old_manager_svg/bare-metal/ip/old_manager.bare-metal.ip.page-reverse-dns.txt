# Cahier des Charges : bare-metal/ip - Page Reverse DNS

## 1. VUE D'ENSEMBLE
- **Type** : Page listing reverse DNS
- **Contexte** : Gestion des IPs
- **Objectif** : Gérer les enregistrements PTR des adresses IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsMessage** : Bannière info
- **OdsDatagrid** : Tableau reverse DNS
- **OdsBadge** : Type IP, état
- **OdsButton** : Actions
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Reverse DNS" |
| Description | "Gérez les enregistrements PTR de vos adresses IP." |

## 5. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Information |
| Fond | #EFF6FF |
| Texte | "Le reverse DNS permet d'associer un nom de domaine à une adresse IP. Utile pour l'envoi d'emails et l'identification des serveurs." |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Adresse IP | 200px | IP (bold) |
| Type | 80px | Badge IPv4/IPv6 |
| Reverse actuel | 300px | Domaine ou "Non configuré" |
| État | 120px | Badge statut |
| Actions | 120px | Bouton action |

| Type | Fond | Texte |
|------|------|-------|

| État | Fond | Texte |

| État | Style |
|------|-------|
| Configuré | Texte #1F2937 normal |
| Non configuré | Texte #6B7280 italique "Non configuré" |

## 10. BOUTONS ACTION
| État | Bouton | Style |
|------|--------|-------|
| Non configuré | "Configurer" | Fond #0050D7, texte blanc |
| Configuré | "Modifier" | Fond #F9FAFB, bordure grise |

## 11. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "X - Y sur Z adresses IP" |

## 12. APIS & ENDPOINTS
- GET /ip/{ip}/reverse - Liste reverse DNS
- PUT /ip/{ip}/reverse - Modifier reverse
- DELETE /ip/{ip}/reverse - Supprimer reverse
