# Cahier des Charges : bare-metal/ip - Modal informations bloc ARIN

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative
- **Contexte** : Gestion des blocs IP ARIN
- **Objectif** : Afficher les informations d'un bloc IP enregistré auprès d'ARIN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsLink** : Email abuse contact
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Informations bloc ARIN" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

| Champ | Label | Exemple |

| Bloc IP | "Bloc IP" | "198.51.100.0/24" (gras) |

## 6. FOOTER MODAL
| Bouton | Style | Action |

## 7. APIS & ENDPOINTS
