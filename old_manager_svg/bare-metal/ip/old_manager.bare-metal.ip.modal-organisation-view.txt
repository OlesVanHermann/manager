# Cahier des Charges : bare-metal/ip - Modal détails organisation

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative
- **Contexte** : Gestion des organisations IP
- **Objectif** : Afficher les détails complets d'une organisation

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détails de l'organisation" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |
