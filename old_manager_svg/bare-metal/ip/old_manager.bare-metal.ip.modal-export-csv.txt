# Cahier des Charges : bare-metal/ip - Modal exporter CSV

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion des adresses IP
- **Objectif** : Exporter la liste des adresses IP au format CSV

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsMessage** : Question
- **OdsButton** : Annuler, Exporter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Exporter en CSV" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message principal
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Cette action va exporter toutes vos adresses IP" |
| Ligne 2 | "au format CSV." |

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Texte | "Voulez-vous continuer ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Exporter | Primary (bleu #0050D7) | Lance l'export CSV |

## 7. APIS & ENDPOINTS
- GET /ip/export/csv - Exporter les IPs en CSV
