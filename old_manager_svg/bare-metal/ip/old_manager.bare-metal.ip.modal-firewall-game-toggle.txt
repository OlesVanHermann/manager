# Cahier des Charges : bare-metal/ip - Modal activer politique Game Firewall

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Game Firewall (protection jeux)
- **Objectif** : Activer/Désactiver la politique par défaut du Game Firewall

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Explication
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer la politique par défaut" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Explication
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Lorsque la politique par défaut est activée, tout le trafic" |
| Ligne 2 | "non autorisé par une règle sera bloqué." |

## 6. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Ligne 1 | "Assurez-vous d'avoir configuré les règles nécessaires avant" |
| Ligne 2 | "d'activer cette politique." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Activer | Primary (bleu #0050D7) | Active la politique |

## 8. APIS & ENDPOINTS
- PUT /ip/{ip}/game - Activer/Désactiver la politique
