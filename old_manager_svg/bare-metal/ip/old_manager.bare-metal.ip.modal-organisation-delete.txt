# Cahier des Charges : bare-metal/ip - Modal supprimer organisation

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation de suppression
- **Contexte** : Gestion des adresses IP - Organisations RIPE
- **Objectif** : Confirmer la suppression d'une organisation RIPE

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement danger
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'organisation" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (rouge) |
| Fond | #FEE2E2 |
| Titre | "Attention !" |
| Message | "Êtes-vous sûr de vouloir supprimer l'organisation {orgId}" |
| Format orgId | Gras |
| Exemple | "ORG-EXAMPLE-RIPE" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime l'organisation |

## 7. APIS & ENDPOINTS
- DELETE /ip/organization/{organizationId} - Supprimer l'organisation RIPE
