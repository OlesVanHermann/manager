# Cahier des Charges : bare-metal/ip - Modal modifier description

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Modifier la description d'une adresse IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Label
- **OdsTextarea** : Description
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la description" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champ Description
| Propriété | Valeur |
|-----------|--------|
| Label | "Description" |
| Type | OdsTextarea |
| Hauteur | 100px |
| Placeholder | - |
| Exemple | "Serveur de production - Web frontend" |
| Max caractères | 255 |

### Compteur de caractères
| Propriété | Valeur |
|-----------|--------|
| Format | "{current}/{max}" |
| Exemple | "220/255" |
| Position | Bas droite |
| Couleur | Gris (#6B7280) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary (bleu #0050D7) | Sauvegarde |

## 7. APIS & ENDPOINTS
- PUT /ip/{ip} - Modifier la description
