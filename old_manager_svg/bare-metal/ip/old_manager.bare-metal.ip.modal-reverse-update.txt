# Cahier des Charges : bare-metal/ip - Modal modifier reverse DNS

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Modifier l'enregistrement reverse DNS d'une adresse IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsInput** : IP (désactivé), reverse DNS
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le reverse" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Texte | "Le reverse DNS doit pointer vers une IP valide." |

## 6. FORMULAIRE

### Champ Adresse IP (lecture seule)
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP" |
| Type | OdsInput (disabled) |
| Fond | Gris (#F3F4F6) |
| Exemple | "51.210.xxx.100" |

### Champ Reverse DNS *
| Propriété | Valeur |
|-----------|--------|
| Label | "Reverse DNS *" |
| Type | OdsInput |
| Bordure active | Bleu (#0050D7) 2px |
| Exemple | "server01.example.com" |
| Requis | Oui |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Applique le reverse |

## 8. APIS & ENDPOINTS
- PUT /ip/{ip}/reverse - Modifier le reverse DNS
