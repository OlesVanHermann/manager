# Cahier des Charges : bare-metal/ip - BYOIP

## 1. VUE D'ENSEMBLE
- **Page** : Bring Your Own IP (BYOIP)
- **Type** : Page liste avec infos
- **Objectif** : Gérer les blocs IP personnels importés

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb
- Titre
- Encart explicatif
- Boutons actions
- Tableau des blocs importés
- Section prérequis

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : IP › BYOIP
- **OdsMessage** : Info explicatif (bleu)
- **OdsButton** : Importer un bloc IP (primary), Documentation (outline)
- **OdsDatagrid** : Tableau blocs BYOIP
- **OdsBadge** : États (Actif, En attente)
- **OdsActionMenu** : Menu actions par ligne
- **OdsCard** : Prérequis

## 4. ENCART EXPLICATIF
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Titre | Qu'est-ce que BYOIP ? |
| Texte 1 | BYOIP vous permet d'utiliser vos propres plages d'adresses IP sur l'infrastructure OVHcloud. |
| Texte 2 | Vous devez être le propriétaire légitime des blocs IP et les avoir déclarés auprès d'un RIR (RIPE, ARIN, etc.). |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Bloc IP | Link | Bloc importé |
| RIR | Text | RIPE, ARIN, APNIC, etc. |
| Service associé | Text | Serveur de destination |
| État | Badge | Actif (vert), En attente (jaune) |
| Date d'import | Date | Format JJ/MM/AAAA |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTION MENU
| Option | Style |
|--------|-------|
| Voir détails | Normal |
| Diviser le bloc | Normal |
| Agréger les blocs | Normal |
| Retirer | Rouge (danger) |

## 7. PRÉREQUIS BYOIP
- Posséder un bloc IP /24 minimum (IPv4) ou /48 (IPv6)
- Être le propriétaire légitime enregistré auprès d'un RIR
- Créer une autorisation ROA (Route Origin Authorization)
- Configurer les objets route dans la base RIPE/ARIN

## 8. APIS & ENDPOINTS
- GET /ip/byoip
  - Liste des blocs BYOIP
- POST /ip/byoip
  - Importer un nouveau bloc
- DELETE /ip/byoip/{block}
  - Retirer un bloc
