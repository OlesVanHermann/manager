# Cahier des Charges : bare-metal/ip - Modal ajouter MAC virtuelle

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Créer une adresse MAC virtuelle pour une IP failover

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, informations
- **OdsSelect** : Type de MAC
- **OdsInput** : Nom de la VM
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une MAC virtuelle" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS CONTEXTE
| Champ | Valeur |
|-------|--------|
| Label | "Adresse IP" |
| Valeur | IP failover concernée |
| Exemple | "51.210.xxx.100" |

## 6. FORMULAIRE

### Champ Type de MAC
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de MAC" |
| Type | OdsSelect |
| Options | OVH, VMware |
| Défaut | "OVH" |

### Champ Nom de la machine virtuelle
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom de la machine virtuelle" |
| Type | OdsInput |
| Placeholder | "Ex: vm-web-01" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary (bleu) | Crée la MAC virtuelle |

## 8. APIS & ENDPOINTS
- POST /ip/{ip}/virtualMac - Créer MAC virtuelle
- GET /ip/{ip}/virtualMac - Liste des MAC virtuelles
