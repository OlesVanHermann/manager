# Cahier des Charges : bare-metal/ip - Wizard migration IP

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (3 étapes)
- **Contexte** : Migration des IPs Failover
- **Objectif** : Migrer les IPs Failover vers Additional IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : Indicateur d'étapes
- **OdsMessage** : Information
- **OdsCheckbox** : Sélection des IPs
- **OdsText** : Labels, compteur
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Migrer vers Additional IP" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. STEPPER

### Étapes
| Étape | Label | État initial |
|-------|-------|--------------|
| 1 | "Sélection" | Active (bleu) |
| 2 | "Confirmation" | Inactive |
| 3 | "Terminé" | Inactive |

### Style étape active
| Propriété | Valeur |
|-----------|--------|
| Cercle | Bleu (#0050D7) |
| Numéro | Blanc |
| Label | Bleu |

### Style étape inactive
| Propriété | Valeur |
|-----------|--------|
| Cercle | Blanc avec bordure grise |
| Numéro | Gris |
| Label | Gris |

## 6. CONTENU ÉTAPE 1 (SÉLECTION)

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Ligne 1 | "Cette migration convertit vos IPs Failover en Additional IP." |
| Ligne 2 | "Les fonctionnalités et tarifs restent identiques." |

### Sélection des IPs
| Propriété | Valeur |
|-----------|--------|
| Titre | "Sélectionnez les IPs à migrer" |
| Type | Liste de checkboxes |
| Format | Checkbox + IP address |

### Compteur
| Propriété | Valeur |
|-----------|--------|
| Format | "{count} IPs sélectionnées" |
| Couleur | Gris (#6B7280) |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary (bleu #0050D7) | Étape suivante |

## 8. APIS & ENDPOINTS
- GET /ip?type=failover - Liste des IPs Failover
- POST /ip/migrate - Lancer la migration
