# Cahier des Charges : bare-metal/ip - Page commander IPs

## 1. VUE D'ENSEMBLE
- **Type** : Page de commande avec sélection
- **Contexte** : Commander des adresses IP additionnelles
- **Objectif** : Permettre la commande d'adresses IPv4 ou IPv6

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsText** : Titres, descriptions
- **OdsLink** : Retour
- **OdsCard** : Cartes de sélection IPv4/IPv6
- **OdsRadio** : Sélection type IP

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander des IPs additionnelles" |
| Fond | Blanc avec bordure |

## 5. NAVIGATION
| Élément | Style | Action |
|---------|-------|--------|
| Retour | "← Retour" (bleu) | Revient à la liste |

| Titre | "Sélectionnez le type d'IP" |
| Description | "Choisissez entre IPv4 et IPv6 selon vos besoins." |
