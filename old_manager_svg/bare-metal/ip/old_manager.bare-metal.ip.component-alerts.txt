# Cahier des Charges : bare-metal/ip - Composant alertes

## 1. VUE D'ENSEMBLE
- **Type** : Composant réutilisable (bannière d'alertes)
- **Contexte** : Gestion des adresses IP
- **Objectif** : Afficher les alertes (antispam, firewall, antihack) sur les IPs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% (900px max) |
| Hauteur | Variable selon état |

## 3. COMPOSANTS UI
- **OdsMessage** : Bannière d'alerte
- **OdsText** : Labels, compteurs
- **OdsLink** : Lien "Charger les détails"

## 4. ÉTATS DU COMPOSANT

### État Warning (alertes détectées)
| Propriété | Valeur |
|-----------|--------|
| Fond | Jaune (#FEF3C7) |
| Couleur texte | #92400E |
| Message | "Des anomalies ont été détectées sur vos adresses IP." |
| Lien | "Charger les détails ▼" (bleu #0050D7) |
| Détail | "{N} alertes détectées : • X alertes antispam • Y alerte firewall" |

### État Success (aucune alerte)
| Propriété | Valeur |
|-----------|--------|
| Fond | Vert (#DCFCE7) |
| Couleur texte | #166534 |
| Message | "Aucune alerte détectée sur vos adresses IP." |

### État Error (erreur de chargement)
| Propriété | Valeur |
|-----------|--------|
| Fond | Rouge (#FEE2E2) |
| Couleur texte | #DC2626 |
| Message | "Erreur lors du chargement des alertes. Veuillez réessayer." |

## 5. APIS & ENDPOINTS
- GET /ip/alerts - Liste des alertes actives
- GET /ip/{ip}/antispam - Alertes antispam
- GET /ip/{ip}/firewall - Alertes firewall
