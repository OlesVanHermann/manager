# Cahier des Charges : bare-metal/nasha - Modal suppression partition

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer une partition
- **Type** : Confirmation suppression
- **Objectif** : Supprimer une partition et ses données

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message de confirmation
- Warning irréversible
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x250px)
- **OdsText** : Messages
- **OdsButton** : Fermer (outline), Supprimer la partition (danger)

## 4. CONTENU
| Élément | Valeur |
|---------|--------|
| Question | Êtes-vous certain(e) de vouloir supprimer la partition |
| Partition | partition-data (bold) |
| Warning | Cette action est irréversible. Toutes les données seront perdues. |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline | Gauche |
| Supprimer la partition | Danger (rouge) | Droite |

## 6. APIS & ENDPOINTS
- DELETE /dedicated/nasha/{serviceName}/partition/{partitionName}
  - Supprime la partition
