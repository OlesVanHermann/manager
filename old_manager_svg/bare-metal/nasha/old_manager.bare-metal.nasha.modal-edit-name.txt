# Cahier des Charges : bare-metal/nasha - Modal modifier nom

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le nom du service
- **Type** : Formulaire édition
- **Objectif** : Personnaliser le nom d'affichage du NAS

## 2. STRUCTURE MODAL
- Header avec titre (incluant l'ID)
- Séparateur
- Champ input
- Helper text
- Séparateur
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x320px)
- **OdsInput** : Nom du service
- **OdsText** : Helper
- **OdsButton** : Fermer (outline), Modifier le nom (primary)

## 4. CONTENU HEADER
| Élément | Valeur |
|---------|--------|
| Titre | Modifier le nom du service zpool-123456 |

## 5. CHAMPS FORMULAIRE

### Nom du service
| Propriété | Valeur |
|-----------|--------|
| Label | Nom du service zpool-123456 |
| Valeur | NAS Production |
| Aide | Les caractères < et > ne sont pas autorisés |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline | Gauche |
| Modifier le nom | Primary | Droite |

## 7. APIS & ENDPOINTS
- PUT /dedicated/nasha/{serviceName}
  - customName: string
