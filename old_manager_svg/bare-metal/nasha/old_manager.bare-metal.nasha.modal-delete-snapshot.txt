# Cahier des Charges : bare-metal/nasha - Modal suppression snapshot

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un snapshot
- **Type** : Confirmation suppression
- **Objectif** : Supprimer un snapshot existant

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message de confirmation
- Warning irréversible
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x230px)
- **OdsText** : Messages
- **OdsButton** : Fermer (outline), Supprimer le snapshot (danger)

## 4. CONTENU
| Élément | Valeur |
|---------|--------|
| Question | Êtes-vous certain(e) de vouloir supprimer le snapshot |
| Snapshot | partition-web-2024-03-15 (bold) |
| Warning | Cette action est irréversible. |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline | Gauche |
| Supprimer le snapshot | Danger (rouge) | Droite |

## 6. APIS & ENDPOINTS
- DELETE /dedicated/nasha/{serviceName}/partition/{partitionName}/customSnapshot/{name}
  - Supprime le snapshot
