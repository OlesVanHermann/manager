# Cahier des Charges : bare-metal/nasha - Détail partition

## 1. VUE D'ENSEMBLE
- **Page** : Détail partition NAS-HA
- **Type** : Page détail
- **Objectif** : Afficher les infos d'une partition

## 2. STRUCTURE PAGE
- Lien retour
- Titre partition + ID parent
- Onglets navigation
- Tuile informations

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour aux partitions
- **OdsTabs** : 3 onglets
- **OdsTile** : Informations
- **OdsProgressBar** : Quota utilisé
- **OdsActionMenu** : Menu ⋮
