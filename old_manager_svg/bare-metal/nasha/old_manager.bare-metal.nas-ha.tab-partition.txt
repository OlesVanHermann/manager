# Cahier des Charges : bare-metal/nas-ha - Onglet Partitions

## 1. VUE D'ENSEMBLE
- **Page** : Liste des partitions NAS-HA
- **Type** : Onglet tableau
- **Objectif** : Gérer les partitions du NAS

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Partitions"
- Onglets navigation
- Bouton création
- Tableau partitions
- Bandeau commandes montage

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Créer une partition
- **OdsDatagrid** : Tableau partitions
- **OdsBadge** : Statut
- **OdsActionMenu** : Actions par ligne
- **OdsMessage** : Bandeau commandes

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link | Nom partition (cliquable) |
| Chemin | Monospace | Chemin /partition |
| Taille | Text | Capacité |
| Utilisé | Text | Espace + pourcentage |
| Protocoles | Text | NFS, CIFS, FTP |
| ACL | Text | Nombre de règles |
| État | Badge | Actif (vert) |
| Actions | ActionMenu | Menu ⋮ |

## 5. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Modifier la taille | Normal | Redimensionner |
| Gérer les ACL | Normal | Ouvre modal ACL |
| Créer un snapshot | Normal | Snapshot manuel |
| Configurer protocoles | Normal | Config NFS/CIFS/FTP |
| Supprimer | Danger | Supprime la partition |

## 6. BANDEAU COMMANDES MONTAGE
| Propriété | Valeur |
|-----------|--------|
| Background | Bleu clair |
| Titre | Commandes de montage |
| NFS | mount -t nfs zpool-12345.rbx.nas.ovh.net:/backup /mnt/backup |
| CIFS | mount -t cifs //zpool-12345.rbx.nas.ovh.net/www /mnt/www -o user=... |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Créer une partition | Primary | Haut |

## 8. APIS & ENDPOINTS
- GET /dedicated/nasha/{serviceName}/partition
  - Liste des partitions
- GET /dedicated/nasha/{serviceName}/partition/{partitionName}
  - Détails partition
- POST /dedicated/nasha/{serviceName}/partition
  - Créer partition
- DELETE /dedicated/nasha/{serviceName}/partition/{partitionName}
  - Supprimer partition
