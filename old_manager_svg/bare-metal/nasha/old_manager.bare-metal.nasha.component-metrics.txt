# Cahier des Charges : bare-metal/nasha - Component Metrics

## 1. VUE D'ENSEMBLE
- **Type** : Composant graphiques métriques
- **Contexte** : NAS-HA (stockage haute disponibilité)
- **Objectif** : Visualiser les performances IOPS et débit

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1000px |
| Hauteur | 550px |
| Background | #F9FAFB |

- **OdsText** : Titres et labels
- **OdsToggleButton** : Sélecteur période
- **OdsCard** : Conteneurs graphiques
- **Chart.js/D3** : Graphiques linéaires
