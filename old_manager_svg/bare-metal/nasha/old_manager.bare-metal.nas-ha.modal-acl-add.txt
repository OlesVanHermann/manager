# Cahier des Charges : bare-metal/nas-ha - Ajouter ACL

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter une règle d'accès
- **Type** : Modal formulaire
- **Objectif** : Ajouter une ACL à une partition

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ partition (lecture seule)
- Champ IP/réseau
- Sélection permissions
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Partition (disabled), IP/réseau
- **OdsRadio** : Permissions
- **OdsButton** : Annuler, Ajouter

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Partition | Input disabled | - | Nom partition (backup) |
| IP/Réseau | Input | Oui | Adresse IP ou CIDR (192.168.1.0/24) |
| Permissions | Radio | Oui | Lecture seule / Lecture-Écriture |

## 5. OPTIONS PERMISSIONS
| Option | Style | Description |
|--------|-------|-------------|
| Lecture seule | Normal | Accès en lecture uniquement |
| Lecture/Écriture | Sélectionné | Accès complet |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/nasha/{serviceName}/partition/{partitionName}/access
  - Paramètres : ip (string), type (readonly/readwrite)
