# Cahier des Charges : bare-metal/nas-ha - Créer partition (simple)

## 1. VUE D'ENSEMBLE
- **Page** : Création de partition (formulaire simplifié)
- **Type** : Modal formulaire
- **Objectif** : Créer une nouvelle partition NAS-HA

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Formulaire simplifié
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom, description
- **OdsQuantity** : Taille
- **OdsRadio** : Protocole
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom | Input | Oui | Nom de la partition (ex: backup) |
| Taille | Quantity + unité | Oui | Taille en Go |
| Protocole | Radio | Oui | NFS / CIFS / Les deux |
| Description | Input | Non | Description optionnelle |

## 5. OPTIONS PROTOCOLE (Radio)
| Option | Description |
|--------|-------------|
| NFS | Network File System (Linux) |
| CIFS (SMB) | Common Internet File System (Windows) |
| Les deux | NFS + CIFS |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/nasha/{serviceName}/partition
  - Paramètres : partitionName, size, protocol, description
