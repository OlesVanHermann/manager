# Cahier des Charges : bare-metal/nas-ha - Modal Snapshot Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création snapshot
- **Contexte** : Stockage NAS-HA
- **Objectif** : Créer un snapshot manuel d'une partition

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Titres et labels
- **OdsCard** : Info partition
- **OdsInput** : Nom snapshot
- **OdsTextarea** : Description
- **OdsButton** : Actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Créer un snapshot" |
| Bouton fermer | × |

## 5. INFO PARTITION
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Bordure | #E5E7EB |
| Label | "Partition" |
| Valeur | "{partitionPath}" (bold) |

## 6. CHAMP NOM SNAPSHOT
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du snapshot *" |
| Type | Input text |
| Placeholder | "snapshot-YYYY-MM-DD" |
| Obligatoire | Oui |

## 7. VALIDATION NOM
| Propriété | Valeur |
|-----------|--------|
| Texte aide | "Caractères alphanumériques et tirets uniquement." |
| Couleur | #6B7280 |

## 8. CHAMP DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Label | "Description (optionnel)" |
| Type | Textarea |
| Placeholder | "Snapshot avant mise à jour..." |
| Rows | 3 |
| Obligatoire | Non |

## 9. FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Fond blanc, bordure grise |
| Créer | Fond #0050D7, texte blanc |

## 10. VALIDATION
| Condition | Comportement |
|-----------|--------------|
| Nom vide | Bouton "Créer" désactivé |
| Nom invalide | Message erreur rouge |
| Nom valide | Bouton "Créer" actif |

## 11. APIS & ENDPOINTS
- POST /dedicated/nasha/{serviceName}/partition/{partitionName}/customSnapshot - Créer snapshot
- GET /dedicated/nasha/{serviceName}/partition/{partitionName}/customSnapshot - Liste snapshots
