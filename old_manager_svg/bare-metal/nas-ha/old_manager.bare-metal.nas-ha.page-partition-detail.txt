# Cahier des Charges : bare-metal/nas-ha - Page Partition Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail partition NAS-HA
- **Contexte** : Stockage NAS-HA
- **Objectif** : Visualiser et configurer une partition

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titres et valeurs
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tuiles informations
- **OdsButton** : Actions

## 13. APIS & ENDPOINTS
- GET /dedicated/nasha/{serviceName}/partition/{partitionName} - Détails partition
- PUT /dedicated/nasha/{serviceName}/partition/{partitionName} - Modifier partition
- GET /dedicated/nasha/{serviceName}/partition/{partitionName}/snapshot - Snapshots
- PUT /dedicated/nasha/{serviceName}/partition/{partitionName}/snapshot - Configurer snapshots
