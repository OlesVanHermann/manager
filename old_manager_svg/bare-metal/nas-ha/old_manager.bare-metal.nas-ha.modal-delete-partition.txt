# Cahier des Charges : bare-metal/nas-ha - Modal Supprimer Partition

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Gestion NAS-HA
- **Objectif** : Supprimer une partition du NAS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsButton** : Fermer, Supprimer partition

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer une partition" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |
| Bouton fermer | × |

## 5. CONTENU

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|

| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Fermer | Secondary | Bordure grise | Ferme modal |
