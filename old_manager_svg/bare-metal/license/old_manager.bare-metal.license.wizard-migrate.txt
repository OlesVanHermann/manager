# Cahier des Charges : bare-metal/license - Wizard migrer licence

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard multi-étapes
- **Contexte** : Licences Dedicated
- **Objectif** : Migrer une licence vers une nouvelle adresse IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 650px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, labels, question
- **OdsStepper** : Indicateur 2 étapes
- **OdsSelect** : Sélection bloc IP
- **OdsInput** : Adresse IP cible
- **OdsMessage** : Vérification compatibilité (succès)
- **OdsCard** : Informations licence
- **OdsButton** : Annuler, Suivant

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Migrer la licence" |

## 5. INDICATEUR D'ÉTAPES
| Étape | Description |
|-------|-------------|
| 1 | Sélection IP (active) |
| 2 | Confirmation |

## 6. ÉTAPE 1 : SÉLECTION IP

### Question
| Texte |
|-------|
| "Sur quelle nouvelle IP souhaitez-vous migrer cette licence ?" |

### Champ Bloc IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Bloc IP" |
| Type | OdsSelect |
| Format | "{serviceName} - {ipBlock}" |

### Champ Adresse IP cible
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP cible" |
| Type | OdsInput |
| Exemple | 51.91.0.5 |

## 7. VÉRIFICATION COMPATIBILITÉ
| Type | Contenu |
|------|---------|
| Succès (vert) | "✓ Cette IP est compatible avec la migration" |
| Fond | Vert clair (#D1FAE5) |
| Icône | Cercle vert avec checkmark |

## 8. INFORMATIONS LICENCE

### Carte récapitulative
| Champ | Exemple |
|-------|---------|
| Type | Windows Server 2022 Standard |
| IP actuelle | 51.91.0.2 |

## 9. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Suivant | Primary (bleu) | Étape suivante |

## 10. APIS & ENDPOINTS
- GET /license/{licenseType}/{serviceName} - Infos licence
- GET /ip/service - Blocs IP disponibles
- POST /license/{licenseType}/{serviceName}/changeIp - Migrer licence
