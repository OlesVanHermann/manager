# Cahier des Charges : bare-metal/vrack - Page Private Network

## 1. VUE D'ENSEMBLE
- **Type** : Page listing réseaux privés VLAN
- **Contexte** : vRack
- **Objectif** : Gérer les réseaux privés virtuels (VLANs)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
