# Cahier des Charges : bare-metal/vrack - Liste des vRack

## 1. VUE D'ENSEMBLE
- **Type** : Page listing avec datagrid
- **Contexte** : Gestion réseau privé vRack
- **Objectif** : Afficher et gérer tous les vRack du compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 550px |
| Datagrid largeur | 1040px |

## 3. COMPOSANTS UI
- **OdsDatagrid** : Tableau des vRack
- **OdsButton** : Commander
- **OdsBadge** : Statuts
- **OdsDropdown** : Menu guides, Actions
- **OdsLink** : Liens vers détails

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "vRack" |
| Bouton droit | "Guides ▾" (dropdown) |

## 5. TOPBAR DATAGRID
| Élément | Style | Action |
|---------|-------|--------|
| Commander | Primary #0050D7 | Ouvre wizard commande |

## 6. COLONNES DATAGRID
| Colonne | Largeur | Type | Description |
|---------|---------|------|-------------|
| SERVICE | 225px | Lien | Identifiant vRack (pn-XXXXX) |
| NOM | 250px | Texte | Nom personnalisé |
| DESCRIPTION | 300px | Texte | Description du vRack |
| ÉTAT | 100px | Badge | Statut du vRack |
| ACTIONS | 90px | Menu | Actions contextuelles |

## 7. STATUTS
| Statut | Fond | Couleur texte |
|--------|------|---------------|
| Actif | #DCFCE7 | #16A34A |
| En création | #EFF6FF | #0050D7 |
| En erreur | #FEE2E2 | #DC2626 |

## 8. MENU ACTIONS (⋮)
| Option | Action |
|--------|--------|
| Modifier le nom | Ouvre modal édition |
| Gérer les services | Navigue vers détails |
| Résilier | Ouvre modal résiliation |

## 9. FOOTER DATAGRID
| Propriété | Valeur |
|-----------|--------|
| Texte | "{N} vRack" |
| Style | 12px, gris #6B7280 |

## 10. ÉTAT VIDE
| Propriété | Valeur |
|-----------|--------|
| Redirection | Page onboarding |

## 11. APIS & ENDPOINTS
- GET /vrack - Liste des vRack
- POST /vrack - Commander un vRack
- DELETE /vrack/{serviceName} - Résilier un vRack
