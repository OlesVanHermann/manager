# Cahier des Charges : bare-metal/vrack - Modal Delete Service

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation suppression
- **Contexte** : vRack
- **Objectif** : Confirmer le retrait d'un service du vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container modal
- **OdsText** : Messages
- **OdsButton** : Actions footer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Retirer un service du vRack" |
| Fond | #FEF3C7 (warning) |
| Texte | #92400E |
| Bouton fermer | × |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir retirer le service suivant du vRack ?" |

## 6. BOX SERVICE
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Bordure | #F59E0B (50% opacity) |
| Label | Type de service (ex: "Serveur dédié") |
| Valeur | Nom du service |

## 7. WARNING
| Propriété | Valeur |
|-----------|--------|
| Icône | ⚠ |
| Message | "Le service perdra sa connectivité au réseau privé." |
| Couleur | #92400E |

## 8. FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Fond blanc, bordure grise, texte gris |
| Retirer | Fond #DC2626, texte blanc |

## 9. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/dedicatedServer/{dedicatedServer} - Retirer serveur
- DELETE /vrack/{serviceName}/cloudProject/{project} - Retirer projet cloud
- DELETE /vrack/{serviceName}/ipLoadbalancing/{ipLoadbalancing} - Retirer LB
