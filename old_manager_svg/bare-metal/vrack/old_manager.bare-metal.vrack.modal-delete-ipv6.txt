# Cahier des Charges : bare-metal/vrack - Modal Retirer IPv6

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Gestion vRack
- **Objectif** : Retirer un bloc IPv6 du vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bouger l'IP additionnelle" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |
| Bouton fermer | × |

## 5. CONTENU

### Description action
| Propriété | Valeur |
|-----------|--------|
| Texte | "Vous allez supprimer {ipBlock} de votre vRack." |
| IP Block | Bleu #0050D7, font-weight 500 |

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Texte | "Tous les sous-réseaux configurés avec ce bloc seront supprimés." |

### Confirmation
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voulez-vous continuer ?" |

## 6. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|

| Confirmer | Primary warning | #F59E0B | Retire le bloc IPv6 |

## 7. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/ipv6/{ipBlock} - Retirer bloc IPv6
