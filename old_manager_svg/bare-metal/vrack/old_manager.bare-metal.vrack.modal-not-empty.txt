# Cahier des Charges : bare-metal/vrack - Modal vRack Non Vide

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information bloquante
- **Contexte** : Gestion vRack
- **Objectif** : Informer que le vRack ne peut être résilié car il contient des services

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement warning
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "vRack {vrackId}" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. CONTENU

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Titre | "Attention" |
| Ligne 1 | "Ce vRack contient encore des services associés." |
| Ligne 2 | "Veuillez les retirer avant de résilier." |
| Couleur texte | #92400E |

## 6. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Fermer | Primary | #0050D7 | Ferme modal |

## 7. COMPORTEMENT
- Ce modal apparaît quand l'utilisateur tente de résilier un vRack non vide
- Aucune action de résiliation n'est possible depuis ce modal
- L'utilisateur doit d'abord retirer tous les services du vRack

## 8. APIS & ENDPOINTS
- GET /vrack/{serviceName}/allowedServices - Vérifier services associés
