# Cahier des Charges : bare-metal/network-security - Modal détail attaque

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative
- **Contexte** : Protection DDoS Network Security
- **Objectif** : Afficher les détails complets d'une attaque DDoS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsBadge** : Statut (En cours, Bloquée)
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détail de l'attaque DDoS" |
| Fond | Orange clair (#FEF3C7) |
| Couleur texte | Orange (#92400E) |
| Badge statut | "En cours" (orange) ou "Bloquée" (vert) |
| Bouton fermer | × |

## 5. INFORMATIONS ATTAQUE

### Détails généraux
| Champ | Label | Exemple |
|-------|-------|---------|
| IP ciblée | "IP ciblée" | "51.210.xxx.12" |
| Type d'attaque | "Type d'attaque" | "UDP Flood" |
| Début | "Début" | "29/12/2024 14:23:45" |
| Durée | "Durée" | "15 minutes (en cours)" |
| Volume maximum | "Volume maximum" | "8.5 Gbps" (rouge si élevé) |
| Paquets/seconde | "Paquets/seconde" | "2.3 Mpps" |

### Types d'attaques
- UDP Flood
- SYN Flood
- DNS Amplification
- NTP Amplification
- HTTP Flood

## 6. SECTION MITIGATION

### Informations mitigation
| Champ | Label | Exemple |
|-------|-------|---------|
| Statut | "Statut" | Badge "Active" (vert #DCFCE7) |
| Trafic bloqué | "Trafic bloqué" | "99.2%" (vert) |
| Scrubbing center | "Scrubbing center" | "Paris (PAR)" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu) | Ferme modal |

## 8. APIS & ENDPOINTS
- GET /ip/{ip}/antihack/{attackId} - Détails de l'attaque
