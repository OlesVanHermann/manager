# Cahier des Charges : bare-metal/network-security - Modal configurer protection

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Protection DDoS Network Security
- **Objectif** : Configurer le mode de protection DDoS pour une IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, descriptions
- **OdsSelect** : Sélection IP
- **OdsRadio** : Mode de protection (cards)
- **OdsButton** : Annuler, Appliquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configurer la protection DDoS" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champ Adresse IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP" |
| Type | OdsSelect |
| Exemple | "51.210.xxx.xxx" |

### Champ Mode de protection
| Propriété | Valeur |
|-----------|--------|
| Label | "Mode de protection" |
| Type | Radio cards |

#### Options
| Option | Description | État |
|--------|-------------|------|
| Automatique | "Protection déclenchée automatiquement lors d'une attaque" | Sélectionné (bordure bleue) |
| Permanente | "Protection activée en permanence" | Non sélectionné |

### Style carte sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Fond | Bleu clair (#EFF6FF) |
| Bordure | Bleue (#0050D7) 2px |
| Radio | Rempli bleu |

### Style carte non sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc |
| Bordure | Grise (#E5E7EB) |
| Radio | Vide avec bordure |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Appliquer | Primary (bleu) | Applique la configuration |

## 7. APIS & ENDPOINTS
- GET /ip/{ip}/mitigation - Mode actuel
- PUT /ip/{ip}/mitigation - Modifier mode de protection
