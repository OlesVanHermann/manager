# Cahier des Charges : bare-metal/nutanix - Page Network Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail configuration réseau
- **Contexte** : Nutanix cluster
- **Objectif** : Visualiser la configuration réseau du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Lien retour
- **OdsText** : Titres et valeurs
- **OdsCard** : Tuiles informations
- **OdsDatagrid** : Tableau nœuds
- **OdsBadge** : Statut connexion
