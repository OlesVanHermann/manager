# Cahier des Charges : bare-metal/nutanix - Onglet Tâches Nœud

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau de données
- **Contexte** : Gestion nœud Nutanix
- **Objectif** : Historique des tâches exécutées sur le nœud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Datagrid largeur | 940px |
| Datagrid hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Tableau tâches
- **OdsBadge** : Statuts
- **OdsProgress** : Barre de progression

## 4. BREADCRUMB
| Niveau | Texte | Lien |
|--------|-------|------|
| 1 | Nutanix | Oui (bleu) |
| 2 | {clusterId} | Oui (bleu) |
| 3 | {nodeId} | Non (noir) |

## 5. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Nœud - {nodeId}" |

## 6. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | Non sélectionné |
| IPMI | Non sélectionné |
| Interventions | Non sélectionné |
| Tâches | Sélectionné (souligné bleu) |

## 7. COLONNES DATAGRID
| Colonne | Largeur | Type | Description |
|---------|---------|------|-------------|
| ID | 125px | Texte | Identifiant tâche (task-XXXXXX) |
| TYPE | 200px | Texte | Nature de la tâche |
| STATUT | 100px | Badge | État de la tâche |
| PROGRESSION | 200px | Progress + % | Avancement de la tâche |
| DATE | 150px | Date | Timestamp de la tâche |

## 8. TYPES DE TÂCHES
| Type | Description |
|------|-------------|
| Installation AOS | Installation Acropolis Operating System |
| Redémarrage | Reboot du nœud |
| Configuration réseau | Modification paramètres réseau |
| Mise à jour AHV | Update de l'hyperviseur |
| Synchronisation | Sync avec le cluster |
