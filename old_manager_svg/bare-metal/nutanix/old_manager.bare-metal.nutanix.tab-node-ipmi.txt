# Cahier des Charges : bare-metal/nutanix - Onglet IPMI Nœud

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec informations et actions
- **Contexte** : Gestion nœud Nutanix
- **Objectif** : Accéder à la console IPMI et afficher les identifiants

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |

- **OdsCard** : Container informations
