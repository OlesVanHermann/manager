# Cahier des Charges : bare-metal/nutanix - Modal Résilier Nœud

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger avec saisie obligatoire
- **Contexte** : Gestion cluster Nutanix
- **Objectif** : Résilier définitivement un nœud du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement danger
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Résilier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résilier le nœud" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Danger |
| Fond | #FEE2E2 |
| Border | #FECACA |
| Icône | ⚠ |
| Titre | "Action irréversible" |
| Ligne 1 | "Cette action va résilier définitivement le nœud." |
| Ligne 2 | "Les données seront définitivement perdues." |
| Couleur texte | #DC2626 |

## 6. TEXTE DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir résilier le nœud" |
| Node ID | {nodeId}.ip-x-x-x.eu (bleu #0050D7) |

## 7. CHAMP DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Pour confirmer, saisissez \"TERMINATE\"" |
| Type | OdsInput |
| Largeur | 450px |
| Placeholder | TERMINATE |
| Validation | Doit être exactement "TERMINATE" |

## 8. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Résilier | Primary danger | #DC2626 | Résilie le nœud |

### État bouton Résilier
| Condition | État |
|-----------|------|
| Input vide | Désactivé |
| Input != "TERMINATE" | Désactivé |
| Input == "TERMINATE" | Activé |

## 9. APIS & ENDPOINTS
- DELETE /nutanix/{serviceName}/node/{nodeId} - Résilier le nœud
