# Cahier des Charges : bare-metal/nutanix - Modal Cluster Reboot

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation redémarrage
- **Contexte** : Cluster Nutanix
- **Objectif** : Confirmer le redémarrage du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Titres et messages
- **OdsCard** : Info cluster
- **OdsMessage** : Avertissement
- **OdsButton** : Actions
