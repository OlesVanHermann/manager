# Cahier des Charges : bare-metal/nutanix - Onglet Noeuds

## 1. VUE D'ENSEMBLE
- **Page** : Liste des noeuds du cluster
- **Type** : Onglet tableau
- **Objectif** : Gérer les noeuds Nutanix

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre cluster
- Onglets navigation
- Bouton ajouter
- Tableau noeuds

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Ajouter des noeuds
- **OdsDatagrid** : Tableau noeuds
- **OdsBadge** : Statut (Installé, À installer)
- **OdsLink** : Liens vers détail noeud
- **OdsActionMenu** : Menu par ligne

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom du noeud | Link | Lien vers détail |
| Serveur | Text | Modèle serveur |
| Statut | Badge | Installé / À installer |
| Localisation | Text | DC + Baie |
| OS installé | Text | Version AHV |
| IP du noeud | Text | IP AHV |
| IP CVM | Text | IP Controller VM |
| Actions | ActionMenu | Menu ⋮ |

## 5. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Installé | Vert |
| À installer | Orange |

## 6. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Allumer | Normal | Power on |
| Eteindre | Normal | Power off |
| Installer | Normal | Déploie AHV |
| Ré-installer | Normal | Réinstalle AHV |
| Désinstaller | Normal | Supprime config |
| Résilier | Danger | Résilie le noeud |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Ajouter des noeuds | Primary | Haut |

## 8. APIS & ENDPOINTS
- GET /nutanix/{serviceName}/node
  - Liste des noeuds
- POST /nutanix/{serviceName}/node
  - Ajouter noeud
