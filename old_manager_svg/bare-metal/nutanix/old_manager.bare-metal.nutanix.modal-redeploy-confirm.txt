# Cahier des Charges : bare-metal/nutanix - Modal Confirmation Redéploiement

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning avec saisie obligatoire
- **Contexte** : Gestion cluster Nutanix
- **Objectif** : Confirmer le redéploiement du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement warning
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Redéploiement du cluster" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Vous allez redéployer l'ensemble de votre cluster." |
| Style | 13px, #1F2937 |

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Titre | "Attention" |
| Ligne 1 | "Le redéploiement de votre cluster entrainera la suppression" |
| Ligne 2 | "de toutes les données du cluster." |
| Couleur texte | #92400E |

## 6. CHAMP DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Entrez \"REDEPLOY\" pour confirmer votre choix" |
| Type | OdsInput |
| Largeur | 450px |
| Placeholder | REDEPLOY |
| Validation | Doit être exactement "REDEPLOY" |

## 7. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Confirmer | Primary warning | #F59E0B (orange) | Lance le redéploiement |

### État bouton Confirmer
| Condition | État |
|-----------|------|
| Input vide | Désactivé |
| Input != "REDEPLOY" | Désactivé |
| Input == "REDEPLOY" | Activé |

## 8. APIS & ENDPOINTS
- POST /nutanix/{serviceName}/redeploy - Lancer le redéploiement
