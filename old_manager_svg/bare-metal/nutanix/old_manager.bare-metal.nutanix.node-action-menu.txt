# Cahier des Charges : bare-metal/nutanix - Menu d'actions Nœud

## 1. VUE D'ENSEMBLE
- **Type** : Menu déroulant contextuel (ActionMenu)
- **Contexte** : Gestion nœud Nutanix
- **Objectif** : Actions rapides sur un nœud du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Bouton déclencheur | 32x32px |
| Menu largeur | 200px |
| Menu hauteur | ~200px |

## 3. COMPOSANTS UI
- **OdsButton** : Bouton icône ⋮
- **OdsPopover** : Container menu déroulant
- **OdsLink** : Options du menu

## 4. BOUTON DÉCLENCHEUR
| Propriété | Valeur |
|-----------|--------|
| Icône | ⋮ (trois points verticaux) |
| Fond | Blanc avec bordure #E5E7EB |
| Taille | 32x32px |
| Border radius | 4px |

## 5. MENU DÉROULANT
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc |
| Border | #E5E7EB |
| Border radius | 8px |
| Shadow | Drop shadow (blur 8px, opacity 0.15) |

## 6. OPTIONS DU MENU

### Section 1 : Alimentation
| Option | Couleur | Action |
|--------|---------|--------|
| Allumer | #1F2937 | Démarre le nœud |
| Éteindre | #1F2937 | Arrête le nœud |

### Séparateur 1
| Propriété | Valeur |
|-----------|--------|
| Type | Ligne horizontale |
| Couleur | #E5E7EB |

### Section 2 : Installation
| Option | Couleur | Action |
|--------|---------|--------|
| Installer | #1F2937 | Lance l'installation Nutanix |
| Réinstaller | #1F2937 | Réinstalle Nutanix |
| Désinstaller | #DC2626 (rouge) | Ouvre modal désinstaller |

### Séparateur 2
| Propriété | Valeur |
|-----------|--------|
| Type | Ligne horizontale |
| Couleur | #E5E7EB |

### Section 3 : Résiliation
| Option | Couleur | Action |
|--------|---------|--------|
| Résilier | #DC2626 (rouge) | Ouvre modal résilier |

## 7. ÉTATS DES OPTIONS
| État | Style |
|------|-------|
| Normal | Fond transparent |
| Hover | Fond #F3F4F6 |
| Désactivé | Gris #9CA3AF, curseur not-allowed |

## 8. APIS & ENDPOINTS
- POST /nutanix/{serviceName}/node/{nodeId}/power - Allumer/Éteindre
- POST /nutanix/{serviceName}/node/{nodeId}/install - Installer
- POST /nutanix/{serviceName}/node/{nodeId}/reinstall - Réinstaller
- DELETE /nutanix/{serviceName}/node/{nodeId}/uninstall - Désinstaller
- DELETE /nutanix/{serviceName}/node/{nodeId} - Résilier
