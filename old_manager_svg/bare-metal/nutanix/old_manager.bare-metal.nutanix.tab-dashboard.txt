# Cahier des Charges : bare-metal/nutanix - Dashboard Cluster

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard avec tuiles d'information et actions
- **Contexte** : Gestion cluster Nutanix
- **Objectif** : Vue d'ensemble du cluster avec accès rapide aux actions

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Zone contenu | 940px largeur |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsMessage** : Bannière d'alerte
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tuiles d'information
- **OdsBadge** : Statuts
- **OdsButton** : Actions
- **OdsDropdown** : Menu guides
