# Cahier des Charges : bare-metal/cda - Supprimer pool

## 1. VUE D'ENSEMBLE
- **Page** : Supprimer un pool
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'un pool

## 2. STRUCTURE MODAL
- Bandeau danger (rouge)
- Header avec icône + titre + bouton fermer
- Récapitulatif pool
- Question de confirmation
- Warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal danger
- **OdsText** : Nom du pool, question
- **OdsMessage** : Warning irréversible (rouge)
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. RÉCAPITULATIF
| Champ | Valeur exemple |
|-------|----------------|
| Nom | data-pool-01 |

## 5. QUESTION
Êtes-vous certain de vouloir supprimer ce pool?

## 6. WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Error (rouge) |
| Icône | ⚠ |
| Texte | Cette action est irréversible. Toutes les données seront perdues. |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Supprimer | Danger (rouge) | Droite |

## 8. APIS & ENDPOINTS
- DELETE /dedicated/ceph/{serviceName}/pool/{poolName}
  - Supprimer le pool
