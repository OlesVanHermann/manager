# Cahier des Charges : bare-metal/cda - Pools

## 1. VUE D'ENSEMBLE
- **Page** : Cloud Disk Array - Pools
- **Type** : Page liste onglet
- **Objectif** : Gérer les pools de stockage CEPH

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb + titre
- Onglets (Pools actif)
- Encart info
- Bouton ajouter
- Tableau des pools

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud / Cloud Disk Array
- **OdsTabs** : Pools actif
- **OdsMessage** : Info explicatif (bleu)
- **OdsButton** : Ajouter un pool (primary)
- **OdsDatagrid** : Tableau des pools
- **OdsActionMenu** : Menu actions par ligne

## 4. ENCART INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte 1 | Vous pouvez créer vos pools ici. |
| Texte 2 | Les pools sont des "containers" pour vos données. L'accès aux pools peut être contrôlé par utilisateur. |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Text | Nom du pool |
| Type | Text | replicated, erasure, etc. |
| Nombre de répliques | Number | Ex: 3 |
| Min. répliques actives | Number | Ex: 2 |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTION MENU
| Option | Style |
|--------|-------|
| Supprimer | Rouge (danger) |

## 7. BOUTON
| Bouton | Style |
|--------|-------|
| Ajouter un pool | Primary |

## 8. APIS & ENDPOINTS
- GET /dedicated/ceph/{serviceName}/pool
  - Liste des pools
- POST /dedicated/ceph/{serviceName}/pool
  - Créer un pool
- DELETE /dedicated/ceph/{serviceName}/pool/{poolName}
  - Supprimer un pool
