# Cahier des Charges : bare-metal/cda - Onglet Santé du cluster

## 1. VUE D'ENSEMBLE
- **Type** : Page tableau de bord
- **Contexte** : Ceph Distributed Architecture (CDA)
- **Objectif** : Visualiser l'état de santé du cluster Ceph

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsText** : Titres, labels, valeurs
- **OdsCard** : Cartes de statut
- **OdsBadge** : États (Sain, INFO, etc.)
- **OdsProgressBar** : Utilisation stockage
- **OdsDatagrid** : Événements récents

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Santé du cluster" |

## 5. CARTES DE STATUT

### Carte État global
| Propriété | Valeur |
|-----------|--------|
| Label | "État global" |
| Badge | "Sain" (vert #DCFCE7) |

### Carte OSDs
| Propriété | Valeur |
|-----------|--------|
| Label | "OSDs" |
| Format | "{actifs} / {total}" |
| Exemple | "24 / 24" |
| Sous-texte | "Tous actifs" (vert) |

### Carte Monitors
| Propriété | Valeur |
|-----------|--------|
| Label | "Monitors" |
| Format | "{actifs} / {total}" |
| Exemple | "3 / 3" |
| Sous-texte | "Quorum OK" (vert) |

### Carte Placement Groups
| Propriété | Valeur |
|-----------|--------|
| Label | "Placement Groups" |
| Format | "{count}" |
| Exemple | "256" |
| Sous-texte | "100% active+clean" (vert) |

## 6. UTILISATION DU STOCKAGE

### Barre de progression
| Propriété | Valeur |
|-----------|--------|
| Titre | "Utilisation du stockage" |
| Format | "Utilisé: {used} / {total} ({percent}%)" |
| Exemple | "Utilisé: 18.4 TB / 46 TB (40%)" |
| Sous-texte | "Disponible: {available}" |

## 7. ÉVÉNEMENTS RÉCENTS

### Colonnes du tableau
| Colonne | Type | Description |
|---------|------|-------------|
| Date | Date | Format "DD/MM/YYYY HH:mm" |
| Type | Badge | INFO (vert), WARN (jaune), ERROR (rouge) |
| Message | Texte | Description de l'événement |

### Exemples d'événements
- "Scrubbing completed on OSD.5"
- "Deep scrubbing started on pool 'data'"

## 8. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/ceph/health - Santé du cluster
- GET /dedicatedCloud/{serviceName}/ceph/osd - Liste des OSDs
- GET /dedicatedCloud/{serviceName}/ceph/mon - Liste des monitors
- GET /dedicatedCloud/{serviceName}/ceph/pool - Liste des pools
- GET /dedicatedCloud/{serviceName}/ceph/log - Événements récents
