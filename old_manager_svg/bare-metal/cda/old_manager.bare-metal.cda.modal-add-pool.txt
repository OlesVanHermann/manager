# Cahier des Charges : bare-metal/cda - Ajouter pool

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un pool
- **Type** : Modal formulaire
- **Objectif** : Créer un nouveau pool de stockage

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ nom
- Texte d'aide
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom du pool
- **OdsText** : Texte d'aide
- **OdsButton** : Annuler (outline), Ajouter (primary)

## 4. CHAMP NOM
| Propriété | Valeur |
|-----------|--------|
| Label | Nom * |
| Type | Input text |
| Placeholder | my-pool-name |
| Requis | Oui |

## 5. RÈGLES DE VALIDATION
- Le nom ne peut contenir d'espaces
- Le nom doit avoir une longueur maximale de 50 caractères

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/ceph/{serviceName}/pool
  - Paramètre : poolName (string)
