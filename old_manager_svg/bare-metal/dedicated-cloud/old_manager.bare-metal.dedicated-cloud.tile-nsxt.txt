# Cahier des Charges : bare-metal/dedicated-cloud - Tile NSX-T

## 1. VUE D'ENSEMBLE
- **Type** : Tuile options VMware
- **Contexte** : Dashboard Dedicated Cloud
- **Objectif** : Afficher l'état NSX-T par datacenter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsBadge** : États activation
- **OdsLink** : Voir détails
- **OdsCollapsible** : Liste par datacenter

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Options VMware" |

## 5. NSX-T
| Propriété | Valeur |
|-----------|--------|
| Label | "Virtualisation des réseaux et de la sécurité (NSX-T)" |

### État global
| Propriété | Valeur |
|-----------|--------|
| Badge activé | Fond #DCFCE7, texte #16A34A "Activé" |
| Lien | "Voir par datacenter ›" (bleu) |

### Liste par datacenter (expandable)
| Datacenter | État |
|------------|------|
| dc-rbx-1 | Activé (vert #DCFCE7) |
| dc-rbx-2 | Désactivé (rouge #FEE2E2) |

## 6. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter - Liste datacenters
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsxt - Statut NSX-T
