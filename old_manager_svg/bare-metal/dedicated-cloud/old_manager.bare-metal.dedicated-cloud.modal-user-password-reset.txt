# Cahier des Charges : bare-metal/dedicated-cloud - Modal User Password Reset

## 1. VUE D'ENSEMBLE

**Type:** Modal de formulaire
**Fonction:** Réinitialiser le mot de passe d'un utilisateur vSphere
**Contexte:** Accessible depuis le menu actions d'un utilisateur

## 2. DIMENSIONS

- Modal: 600x390px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal
- Background: #F9FAFB
- Titre: "Réinitialiser le mot de passe"
- Close button: ×

### Info utilisateur
- Label: "Utilisateur :"
- Valeur: "admin@vsphere.local" (font-weight: 600)

### Formulaire

#### Champ Nouveau mot de passe
- Label: "Nouveau mot de passe *"
- Input: 520x40px, type password
- Affichage: ••••••••••••

#### Champ Confirmation
- Label: "Confirmer le mot de passe *"
- Input: 520x40px, type password

### Bloc info politique
- Background: #EFF6FF
- Border: 1px solid #BFDBFE
- Texte: "Le mot de passe doit contenir au minimum 8 caractères, une majuscule, une minuscule, un chiffre et un caractère spécial."

## 4. HEADER MODAL

- Hauteur: 60px
- Titre: "Réinitialiser le mot de passe"

## 5. CONTENU

### Formulaire
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Nouveau mot de passe | password | Oui |
| Confirmer le mot de passe | password | Oui |

### Politique de mot de passe
```
Le mot de passe doit contenir au minimum 8 caractères,
une majuscule, une minuscule, un chiffre et un caractère spécial.
```

## 6. FOOTER MODAL

- Séparateur: ligne 1px
- Bouton Annuler: btn-secondary
- Bouton Confirmer: btn-primary

## 7. APIS & ENDPOINTS

```
PUT /dedicatedCloud/{serviceName}/user/{userId}/password
Body: {
  password: "newPassword123!"
}
→ Réinitialise le mot de passe de l'utilisateur
```

## 8. VALIDATION

- Mot de passe minimum 8 caractères
- Au moins une majuscule
- Au moins une minuscule
- Au moins un chiffre
- Au moins un caractère spécial
- Confirmation identique au mot de passe
