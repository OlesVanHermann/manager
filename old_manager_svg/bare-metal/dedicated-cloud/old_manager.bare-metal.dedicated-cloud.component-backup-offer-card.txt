# Cahier des Charges : bare-metal/dedicated-cloud - Component Backup Offer Card

## 1. VUE D'ENSEMBLE

**Type:** Composant de sélection (Card)
**Fonction:** Afficher une offre Veeam Backup avec ses caractéristiques
**Contexte:** Sélection d'offre de backup dans les wizards

## 2. DIMENSIONS

- Card: 200x260px
- Border-radius: 8px
- Background: white ou #EFF6FF (sélectionné)

## 3. STRUCTURE DE LA CARD

### Header
- Titre: "Veeam Backup [Niveau]" (15px, font-weight: 600)
- Prix: "À partir de X,XX€/VM" (text-primary, 13px)

### Badge "Offre actuelle" (optionnel)
- Background: #0050D7
- Text: white, 11px
- Dimensions: 90x22px
- Border-radius: 4px

### Séparateur
- Ligne #E5E7EB

### Liste des fonctionnalités
Chaque item:
- Cercle indicateur (3px radius)
  - Vert (#16A34A): Inclus
  - Gris (#6B7280): Non inclus
- Texte 12px
- Si non inclus: text-decoration: line-through

## 4. NIVEAUX D'OFFRE

### Standard
- 14 points de rétention
- 1 sauvegarde complète
- ✗ Rapport personnalisable
- ✗ Chiffrement
- ✗ Réplication hors site

### Advanced
- 28 points de rétention
- 2 sauvegardes complètes
- ✓ Rapport personnalisable
- ✓ Sauvegarde chiffrée
- ✗ Réplication hors site

### Premium
- Min. 28 points rétention
- Min. 2 sauvegardes complètes
- ✓ Rapport personnalisable
- ✓ Sauvegarde chiffrée
- ✓ Réplication hors site
- ✓ Rétention annuelle

## 5. ÉTATS

| État | Border | Background |
|------|--------|------------|
| Normal | #E5E7EB (1px) | white |
| Sélectionné | #0050D7 (2px) | #EFF6FF |

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/offerCapabilities
```
