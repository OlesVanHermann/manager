# Cahier des Charges : bare-metal/dedicated-cloud - Modal VM License Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation danger
**Fonction:** Supprimer/Désactiver une licence d'une machine virtuelle
**Contexte:** Accessible depuis le menu actions d'une VM avec licence active

## 2. DIMENSIONS

- Modal: 600x240px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal (Danger)
- Background: #FEE2E2
- Icône: Cercle rouge #DC2626 avec "!"
- Titre: "Supprimer la licence" (color: #991B1B)
- Close button: × (color: #991B1B)

### Contenu
- Question: "Souhaitez-vous vraiment supprimer la licence de cette machine virtuelle ?"
- Info VM: "VM : vm-webapp-prod-01"

## 4. HEADER MODAL

- Hauteur: 60px
- Background: #FEE2E2
- Icône warning + titre

## 5. CONTENU

### Message de confirmation
```
Souhaitez-vous vraiment supprimer la licence de cette
machine virtuelle ?

VM : vm-webapp-prod-01
```

## 6. FOOTER MODAL

- Séparateur: ligne 1px
- Bouton Annuler: btn-secondary (100x36px)
- Bouton Confirmer: btn-danger (100x36px)

## 7. APIS & ENDPOINTS

```
DELETE /dedicatedCloud/{serviceName}/vm/{vmId}/license
→ Supprime la licence de la VM
```
