# Cahier des Charges : bare-metal/dedicated-cloud - Page Service Pack Picker

## 1. VUE D'ENSEMBLE

**Type:** Page de sélection (Cards)
**Fonction:** Sélectionner/changer le Service Pack de l'infrastructure
**Contexte:** Accessible depuis les paramètres du Dedicated Cloud

## 2. DIMENSIONS

- Container principal: 1360x650px
- Cartes: 400x300px chacune (3 cartes en ligne)

## 3. LAYOUT & STRUCTURE

### Header OVHcloud
- Background: #000E9C
- Logo OVHcloud + "Bare Metal Cloud"

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > **Service Pack**

### Zone principale
1. Titre + description
2. 3 cartes Service Pack en ligne
3. Bouton d'action

## 4. COMPOSANTS UI

### Titre
- Text: "Sélectionnez votre Service Pack"
- Font-size: 20px, font-weight: 600

### Description
- Text: "Choisissez le pack de services adapté à vos besoins"

### Cartes Service Pack

#### Carte sélectionnée (active)
- Border: 2px solid #0050D7
- Header background: #EFF6FF
- Radio: checked (cercle bleu)
- Badge "Option actuelle" si applicable

#### Carte non sélectionnée
- Border: 1px solid #E5E7EB
- Header background: #F9FAFB
- Radio: unchecked

## 5. DONNÉES & COLONNES

### Pack Basic
- Support standard inclus
- Monitoring basique
- Accès vSphere Web Client
- Backup basique
- **Prix: +0,00 € HT/mois**

### Pack Premier
- Support prioritaire 24/7
- Monitoring avancé
- Accès vSphere + vRealize
- Backup Veeam inclus
- **Prix: +150,00 € HT/mois**

### Pack Enterprise
- Support dédié avec TAM
- Monitoring temps réel
- Suite complète VMware
- Disaster Recovery inclus
- **Prix: +450,00 € HT/mois**

## 6. ACTIONS & INTERACTIONS

### Sélection carte
- Click sur carte: Sélectionne le pack (change le style + radio)
- Seule 1 carte sélectionnable à la fois

### Bouton "Changer de pack"
- Position: Bas droite
- Style: btn-primary (180x40px)
- Désactivé si pack actuel déjà sélectionné
- Action: Ouvre modal de confirmation avec récapitulatif

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/servicePack
→ Récupère le pack actuel

GET /dedicatedCloud/{serviceName}/servicePack/available
→ Liste les packs disponibles avec prix

PUT /dedicatedCloud/{serviceName}/servicePack
Body: { servicePack: "premier" }
→ Change le service pack
```
