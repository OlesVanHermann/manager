# Cahier des Charges : bare-metal/dedicated-cloud - Modal Move NSX-T Edge

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion NSX-T Dedicated Cloud
- **Objectif** : Déplacer un edge NSX-T vers un autre datastore

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection edge et datastore
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Déplacer l'edge NSX-T" |
| Fond | #F9FAFB |

## 5. CHAMPS

### Edge NSX-T
| Propriété | Valeur |
|-----------|--------|
| Label | "Edge NSX-T" |
| Type | Select |
| Valeur exemple | "edge-medium-01" |

### Datastore destination
| Propriété | Valeur |
|-----------|--------|
| Label | "Datastore de destination" |
| Type | Select |
| Placeholder | "Sélectionner un datastore" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary #0050D7 | Déplace edge |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsxt/edges - Liste edges
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer - Liste datastores
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsxt/edge/{edgeId}/move
