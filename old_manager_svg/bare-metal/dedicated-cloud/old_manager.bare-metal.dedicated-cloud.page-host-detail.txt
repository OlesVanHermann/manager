# Cahier des Charges : bare-metal/dedicated-cloud - Page Host Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail host ESXi
- **Contexte** : Dedicated Cloud (VMware)
- **Objectif** : Visualiser et gérer un host ESXi

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titres et valeurs
- **OdsBadge** : Statut
- **OdsCard** : Tuiles informations
- **OdsProgressBar** : Utilisation ressources
- **OdsButton** : Actions
- **OdsDatagrid** : Tâches récentes

## 4. NAVIGATION
| Propriété | Valeur |
|-----------|--------|
| Lien | "← Retour aux hosts" |
| Couleur | #0050D7 |

## 5. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "{hostName}" |
| Badge | "Actif" (vert) |

## 6. LAYOUT TUILES
| Propriété | Valeur |
|-----------|--------|
| Disposition | 3 colonnes |
| Dimensions | 340x180px |

## 7. TUILE 1 : MATÉRIEL
| Champ | Valeur |
|-------|--------|
| Profil | "{profileName}" (ex: L1-128) |
| CPU | "{cpuType}" |
| RAM | "{ramSize}" |
| Cœurs | "{coreCount}" |

## 8. TUILE 2 : UTILISATION
| Champ | Type |
|-------|------|
| CPU | Progress bar + % |
| RAM | Progress bar + % |
| VMs actives | Nombre |

## 9. TUILE 3 : RÉSEAU
| Champ | Valeur |
|-------|--------|
| IP Management | "{ipMgmt}" |
| vMotion | "{ipVmotion}" |
| VSAN | "{ipVsan}" |

## 10. SECTION ACTIONS
| Bouton | Style |
|--------|-------|
| Maintenance on | Fond blanc, bordure grise |
| Ajouter une licence | Fond blanc, bordure grise |
| Reboot | Fond blanc, bordure grise |

## 11. SECTION TÂCHES RÉCENTES
| Colonne | Contenu |
|---------|---------|
| Date | Date/heure |
| Opération | Description |
| Statut | Badge statut |

## 12. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/host/{hostId} - Détails host
- PUT /dedicatedCloud/{serviceName}/datacenter/{dcId}/host/{hostId} - Modifier
- POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/host/{hostId}/maintenanceMode - Maintenance
