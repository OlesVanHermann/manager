# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Ajouter Règle Sécurité

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes
- **Contexte** : Gestion de la politique de sécurité Dedicated Cloud
- **Objectif** : Ajouter une entrée IP/CIDR à la liste d'accès autorisé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 900px |
| Padding | 60px |

## 3. COMPOSANTS UI
- **OdsStepper** : Navigation étapes
- **OdsInput** : Champs du formulaire
- **OdsButton** : Navigation

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une entrée de sécurité" |

## 5. STEPPER
| Étape | Label | État exemple |
|-------|-------|--------------|
| 1 | Configuration | Actif (bleu #0050D7) |
| 2 | Confirmation | À venir (gris) |

## 6. ÉTAPE 1 : CONFIGURATION

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Réseau autorisé | OdsInput | Oui | Adresse IP ou bloc CIDR |
| Description | OdsInput | Non | Identifier cette règle |

### Labels et helper text
| Champ | Label | Helper |
|-------|-------|--------|
| Réseau | "Réseau autorisé *" | "Entrez l'adresse IP ou le bloc CIDR à autoriser" |
| Description | "Description (optionnel)" | "Ajoutez une description pour identifier cette règle" |

### Style champ focus
| Propriété | Valeur |
|-----------|--------|
| Bordure | Bleue (#0050D7) 2px |

### Exemples de valeurs
| Champ | Exemple |
|-------|---------|
| Réseau | 192.168.1.0/24 |
| Description | Bureau Paris |
