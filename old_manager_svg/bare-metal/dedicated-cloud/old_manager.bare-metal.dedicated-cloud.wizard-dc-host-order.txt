# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Commander Host

## 1. VUE D'ENSEMBLE
- **Type** : Page de commande avec sélection
- **Contexte** : Commande de ressources Dedicated Cloud
- **Objectif** : Commander un ou plusieurs hosts ESXi pour un datacenter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 1100px |
| Padding | 80px |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsMessage** : Information tarifs
- **OdsDatagrid** : Liste des hosts
- **OdsRadio** : Sélection host
- **OdsQuantity** : Sélecteur quantité
- **OdsButton** : Commander

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Navigation | "← Commander un host" (lien retour) |
