# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Upgrade Service Pack

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (3 étapes)
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Commander un upgrade du service pack (Standard → Premier/Enterprise)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes
- **OdsRadio** : Sélection pack (cartes)
- **OdsMessage** : Info NSX required
- **OdsButton** : Annuler, Retour, Suivant
- **OdsBadge** : Pack actuel
- **OdsText** : Descriptions, prix

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour au dashboard" |
| Couleur lien | Bleu (#0050D7) |

| Pack | Sélectionné | Description | Prix | Badge |
|------|-------------|-------------|------|-------|

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Suivant | Droite | Primary (#0050D7) | Passe à l'étape 2 |

## 7. ÉTAPES SUIVANTES (aperçu)

### Étape 2 - Configuration utilisateurs
- Assignation des droits PCI-DSS/HDS aux utilisateurs
- Configuration des validateurs de token

### Étape 3 - Validation
- Récapitulatif de la commande
- Conditions générales
- Validation finale

## 8. APIS & ENDPOINTS
- GET /order/dedicatedCloud/{serviceName}/servicePack - Packs disponibles
- POST /order/dedicatedCloud/{serviceName}/servicePack - Commander le pack
