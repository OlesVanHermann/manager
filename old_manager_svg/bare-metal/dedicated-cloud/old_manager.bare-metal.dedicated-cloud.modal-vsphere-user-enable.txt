# Cahier des Charges : bare-metal/dedicated-cloud - Modal vSphere User Enable

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation avec warning
**Fonction:** Activer un utilisateur vSphere
**Contexte:** Réactivation utilisateur (notamment support temporaire)

## 2. DIMENSIONS

- Modal: 450x240px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. HEADER

- Titre: "Activer un utilisateur" (18px, font-weight: 600)
- Bouton fermer: × (20px, muted)
- Séparateur: #E5E7EB

## 4. WARNING BANNER

- Dimensions: 400x50px
- Background: #FEF3C7
- Border: #FDE68A
- Message: "Cet utilisateur est un administrateur temporaire de votre infrastructure. Il sera désactivé dans 2h."

## 5. CONTENU

### Message principal
- "Etes-vous certain(e) de vouloir activer l'utilisateur"
- "**support-temp** ?" (bold)

### Boutons
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | Secondary (border) | 90x36px |
