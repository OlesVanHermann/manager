# Cahier des Charges : bare-metal/dedicated-cloud - Wizard VMware Option Disable

## 1. VUE D'ENSEMBLE

**Type:** Wizard single-step
**Fonction:** Désactiver une option VMware (ex: vRealize Operations)
**Contexte:** Gestion des options VMware sur le Dedicated Cloud

## 2. DIMENSIONS

- Container principal: 1440x750px
- Zone wizard: 1100x600px (centrée)

## 3. LAYOUT & STRUCTURE

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > **Désactiver vRealize Operations**

### Header wizard
- Background: #F9FAFB
- Titre: "Désactiver vRealize Operations"
- Hauteur: 70px

## 4. CONTENU PRINCIPAL

### Description
- Ligne 1: "Vous êtes sur le point de désactiver l'option vRealize Operations."
- Ligne 2: "Les prix suivants seront appliqués après la désactivation :"

### Tableau impact prix

**Colonnes:**
| Colonne | Largeur | Alignement |
|---------|---------|------------|
| Nom de l'hôte | 220px | Gauche |
| Datacenter | 230px | Gauche |
| Prix actuel | Auto | Droite |
| Prix après désactivation | Auto | Droite |

**En-tête:**
- Background: #F3F4F6
- Font-weight: 600

### Message warning

- Border: #FECACA
- Dimensions: 1020x50px

### Alternance lignes tableau
- Impaires: bg-white

### Message danger
- Border-radius: 6px

## 6. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x40px)
- Position: gauche
- Action: Retour à la page options

### Bouton Désactiver
- Style: btn-danger #DC2626 (130x40px)
- Position: droite
- Texte blanc
- Action: Déclenche la désactivation de l'option

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/vmwareOption/{optionId}
→ Récupère les détails de l'option

GET /dedicatedCloud/{serviceName}/vmwareOption/{optionId}/priceImpact
→ Calcule l'impact sur les prix par host

POST /dedicatedCloud/{serviceName}/vmwareOption/{optionId}/disable
→ Désactive l'option VMware
```
