# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Host Order Legacy

## 1. VUE D'ENSEMBLE

**Type:** Wizard 2 étapes
**Fonction:** Commander un hôte pour le datacenter
**Contexte:** Commande de ressources depuis un datacenter existant

## 2. DIMENSIONS

- Container principal: 1440x850px
- Zone wizard: 1360x650px

## 3. LAYOUT & STRUCTURE

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > dc-paris-01 > **Commander un host**

### Lien retour
- "← Commander un host" (text-primary)

### Stepper (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Sélection | Active (bleu) |
| 2 | Récapitulatif | Inactive (gris) |

## 4. ÉTAPE 1 : SÉLECTION

## 6. ACTIONS & INTERACTIONS

### Sélection host
- Click sur ligne → sélectionne le radio button
- Un seul host sélectionnable à la fois
- Si vSAN sélectionné, affiche le message info

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Retour à la page précédente

### Bouton Suivant
- Style: btn-primary (100x36px)
- Action: Passe à l'étape 2 (Récapitulatif)
- Disabled si aucun host sélectionné

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/orderableHostProfiles
→ Liste les profils d'hôte commandables

POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/orderHosts
Body: {
  profile: "Host L",
  quantity: 1
}
→ Génère un bon de commande
```
