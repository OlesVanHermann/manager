# Cahier des Charges : bare-metal/dedicated-cloud - Tab DC VMs Actions

## 1. VUE D'ENSEMBLE

**Type:** Page Listing / Datagrid avec menu actions
**Fonction:** Gérer les machines virtuelles et leurs licences/sauvegardes
**Contexte:** Onglet Machines virtuelles d'un datacenter

## 2. DIMENSIONS

- Container principal: 1360x600px
- Datagrid: largeur complète

## 3. LAYOUT & STRUCTURE

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > dc-paris-01 > **Machines virtuelles**

### Zone principale
1. Message info (conditionnel)
2. Titre
3. Datagrid des VMs
4. Pagination

## 4. COMPOSANTS UI

### Message info
- Background: #EFF6FF
- Icône: i (cercle bleu)
- Texte: "Pour activer les licences, vous devez d'abord activer l'option SPLA dans les paramètres."

### Titre
- Text: "Machines virtuelles"
- Font-size: 18px

| Action | Style |
|--------|-------|
| Activer la licence | Normal |

## 7. PAGINATION

- Format: "Affichage de 1 à 4 sur 24 machines virtuelles"

## 8. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm
→ Liste les VMs du datacenter

GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}
→ Détails d'une VM

POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/license
→ Active une licence

DELETE /dedicatedCloud/{serviceName}/datacenter/{dcId}/vm/{vmId}/license
→ Supprime une licence
```
