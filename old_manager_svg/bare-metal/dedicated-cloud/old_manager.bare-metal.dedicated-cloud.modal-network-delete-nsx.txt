# Cahier des Charges : bare-metal/dedicated-cloud - Modal Network Delete NSX

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation danger avec warning
- **Contexte** : Gestion réseau NSX Dedicated Cloud
- **Objectif** : Supprimer un réseau NSX

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 420px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le réseau NSX" |
| Fond | #F9FAFB |

## 5. AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Ligne 1 | "⚠ Cette action supprimera définitivement le réseau NSX et" |
| Ligne 2 | "toutes ses configurations associées." |

## 6. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Voulez-vous continuer la suppression du réseau NSX ?" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Danger #DC2626 | Supprime réseau |

## 8. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsxt/network/{networkId}
