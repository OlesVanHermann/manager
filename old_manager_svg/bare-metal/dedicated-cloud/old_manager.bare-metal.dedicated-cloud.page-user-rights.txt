# Cahier des Charges : bare-metal/dedicated-cloud - Page User Rights

## 1. VUE D'ENSEMBLE

**Type:** Page Listing / Datagrid
**Fonction:** Afficher et gérer les droits d'un utilisateur par datacenter
**Contexte:** Accessible depuis la fiche utilisateur, onglet Droits

## 2. DIMENSIONS

- Container principal: 1360x550px
- Datagrid: largeur complète

## 3. LAYOUT & STRUCTURE

### Header OVHcloud
- Background: #000E9C
- Logo OVHcloud + "Bare Metal Cloud"

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > Utilisateurs > **Droits par datacenter**

### Zone principale
1. Lien retour
2. Titre avec nom utilisateur
3. Datagrid des droits par datacenter
4. Pagination

## 4. COMPOSANTS UI

### Lien retour
- Text: "← Retour"
- Style: text-primary

### Titre
- Text: "Gestion des droits par datacenter pour **admin@vsphere.local**"
- Font-size: 18px, font-weight: 600

## 6. MENU ACTIONS

| Action | Description |
|--------|-------------|
| Modifier les droits | Ouvre modal modal-user-rights-edit |

## 7. PAGINATION

- Format: "Affichage de 1 à 3 sur 3 datacenters"

## 8. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/user/{userId}/rights
→ Liste les droits par datacenter

GET /dedicatedCloud/{serviceName}/datacenter
→ Liste des datacenters

PUT /dedicatedCloud/{serviceName}/user/{userId}/rights/{datacenterId}
→ Modifie les droits
```
