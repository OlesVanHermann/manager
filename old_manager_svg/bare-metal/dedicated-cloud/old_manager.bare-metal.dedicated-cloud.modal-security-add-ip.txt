# Cahier des Charges : bare-metal/dedicated-cloud - Modal Security Add IP

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter une IP à la liste d'accès autorisée
**Contexte:** Gestion sécurité Private Cloud

## 2. DIMENSIONS

- Modal: 450x280px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. HEADER

- Titre: "Ajouter une plage d'adresses IP" (18px, font-weight: 600)
- Bouton fermer: × (20px, muted)
- Séparateur: #E5E7EB

## 4. FORMULAIRE

### Champ 1 - IP/CIDR (requis)
- Label: "Adresse IP ou bloc CIDR *" (font-weight: 500)
- Input: 400x40px, border #D1D5DB
- Placeholder/Value: "192.168.1.0/24"

### Champ 2 - Description (optionnel)
- Label: "Description" (font-weight: 500)
- Input: 400x40px, border #D1D5DB

### Boutons
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | Secondary (border) | 90x36px |

## 6. VALIDATION

- IP/CIDR requis, format IPv4 ou IPv6 avec masque
- Description optionnelle, max 255 caractères
