# Cahier des Charges : bare-metal/dedicated-cloud - Modal Zerto Datacenter Selection

## 1. VUE D'ENSEMBLE

**Type:** Modal de sélection
**Fonction:** Sélectionner le datacenter pour la configuration Zerto DRP
**Contexte:** Étape du wizard de configuration Zerto

## 2. DIMENSIONS

- Modal: 600x250px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal
- Background: #F9FAFB
- Titre: "Sélection du datacenter"
- Close button: ×

### Formulaire

#### Champ Datacenter
- Label: "Datacenter pour Zerto DRP"
- Type: Select dropdown
- Dimensions: 520x44px
- Valeur exemple: "dc-paris-01 - Datacenter principal"

- Hauteur: 60px

- Séparateur: ligne 1px

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter
→ Liste les datacenters disponibles

GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/zerto/eligibility
→ Vérifie l'éligibilité du datacenter pour Zerto
```
