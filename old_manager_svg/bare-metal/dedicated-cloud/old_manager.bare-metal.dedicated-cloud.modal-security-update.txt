# Cahier des Charges : bare-metal/dedicated-cloud - Modal Security Update

## 1. VUE D'ENSEMBLE

**Type:** Modal d'édition
**Fonction:** Modifier une règle de sécurité IP existante
**Contexte:** Accessible depuis le menu actions d'une règle dans l'onglet Sécurité

## 2. DIMENSIONS

- Modal: 600x350px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal
- Background: #F9FAFB
- Titre: "Modification de la plage IP"
- Close button: ×

### Formulaire

#### Champ IP
- Label: "IP :"
- Input: 520x40px
- Valeur: IP existante (ex: "192.168.1.0/24")
- État: Lecture seule ou éditable selon contexte

#### Champ Description
- Label: "Description :"
- Input: 520x40px
- Valeur: Description existante

## 4. HEADER MODAL

- Hauteur: 60px
- Background: #F9FAFB
- Titre: "Modification de la plage IP"

## 5. CONTENU

### Formulaire d'édition
| Champ | Type | Valeur initiale | Éditable |
|-------|------|-----------------|----------|
| IP | text | 192.168.1.0/24 | Non (ou selon règles) |
| Description | text | Réseau local bureau Paris | Oui |

### Pré-remplissage
- Les champs sont pré-remplis avec les valeurs actuelles
- L'IP peut être en lecture seule (modification = suppression + création)

## 6. FOOTER MODAL

- Séparateur: ligne 1px #E5E7EB
- Bouton Annuler: btn-secondary (100x36px)
- Bouton Modifier: btn-primary (100x36px)

## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/security/ip/{ipBlock}
→ Récupère les détails de la règle

PUT /dedicatedCloud/{serviceName}/security/ip/{ipBlock}
Body: {
  description: "Nouvelle description"
}
→ Met à jour la règle de sécurité
```

## 8. VALIDATION

- Description: max 255 caractères
- Si IP modifiable: format CIDR valide
