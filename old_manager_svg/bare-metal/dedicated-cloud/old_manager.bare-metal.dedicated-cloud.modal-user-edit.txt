# Cahier des Charges : bare-metal/dedicated-cloud - Modal User Edit

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Modifier les informations et droits d'un utilisateur
**Contexte:** Gestion utilisateurs VMware Private Cloud

## 2. DIMENSIONS

- Modal: 450x580px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Titre | "Modifier l'utilisateur" (18px, font-weight: 600) |
| Close button | × (text-muted, 20px) |
| Séparateur | ligne #E5E7EB |

## 4. FORMULAIRE

### Champs texte
| Champ | Type | Placeholder |
|-------|------|-------------|
| Prénom | Input text | Jean |
| Nom | Input text | Dupont |
| Email | Input email | jean.dupont@example.com |
| Téléphone | Input tel | +33 6 12 34 56 78 |

### Style inputs
- Width: 390px
- Height: 40px
- Border: #D1D5DB
- Border-radius: 4px
- Padding: 10px

### Toggle switches (droits)
| Droit | Description | Default |
|-------|-------------|---------|
| Token validator | Peut valider les tokens | ON |
| Gérer les IPs | Peut gérer les adresses IP | ON |
| Gérer les IP failover | Peut gérer les IPs failover | OFF |
| Droit NSX | Accès aux fonctionnalités NSX | ON |

### Style toggles
- Active: #0050D7, cercle blanc à droite
- Inactive: #E5E7EB, cercle blanc à gauche
- Dimensions: 50x26px

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary (border) | Fermer modal |
| Confirmer | Primary (#0050D7) | Sauvegarder |

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/user/{userId}
PUT /dedicatedCloud/{serviceName}/user/{userId}
PUT /dedicatedCloud/{serviceName}/user/{userId}/right
```
