# Cahier des Charges : bare-metal/dedicated-cloud - Onglet Datacenters

## 1. VUE D'ENSEMBLE
- **Onglet** : Datacenters
- **Type** : Page liste datacenters virtuels
- **Objectif** : Gérer les datacenters d'un Dedicated Cloud

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre Dedicated Cloud
- Onglets navigation
- Titre section + bouton Commander
- Tableau datacenters
- Cartes résumé ressources

## 3. COMPOSANTS UI
- **OdsTabs** : 5 onglets
- **OdsButton** : Commander un datacenter (primary)
- **OdsDatagrid** : Tableau
- **OdsBadge** : Actif (success)
- **OdsActionMenu** : Menu ⋮
- **OdsCard** : 4 cartes résumé

## 9. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter
  - Liste datacenters
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}
  - Détails datacenter
- POST /order/dedicatedCloud/{serviceName}/datacenter
  - Commander datacenter
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}
  - Supprimer datacenter
