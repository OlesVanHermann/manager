- **Type** : Onglet de gestion réseau
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Gérer les NSX Edges pour le réseau software-defined

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 700px |

## 6. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx - Liste NSX Edges
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx - Créer NSX Edge
- PUT /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx/{nsxId} - Modifier NSX
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx/{nsxId} - Supprimer NSX
