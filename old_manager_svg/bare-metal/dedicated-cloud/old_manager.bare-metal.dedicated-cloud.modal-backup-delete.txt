# Cahier des Charges : bare-metal/dedicated-cloud - Modal Backup Delete

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation warning
- **Contexte** : Gestion Dedicated Cloud
- **Objectif** : Confirmer la suppression d'un backup

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 420px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement irréversible
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le backup" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |

## 5. AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Texte | "⚠ Cette action est irréversible." |

## 6. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir supprimer le backup du datacenter" |
| Datacenter | {datacenterName} en gras |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger #DC2626 | Supprime backup |

## 8. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup
