# Cahier des Charges : bare-metal/dedicated-cloud - Hosts datacenter

## 1. VUE D'ENSEMBLE
- **Page** : Hosts datacenter
- **Type** : Page gestion hosts ESXi
- **Objectif** : Gérer les hosts d'un datacenter VMware

## 2. STRUCTURE PAGE
- Breadcrumb complet
- Titre (dc-X - Hosts)
- Onglets navigation datacenter
- Titre section + bouton Ajouter
- Tableau hosts
- Résumé ressources

## 3. COMPOSANTS UI
- **OdsTabs** : 3 onglets
- **OdsButton** : Ajouter un host (primary)
- **OdsDatagrid** : Tableau
- **OdsBadge** : Connecté (success), Maintenance (warning)
- **OdsProgressBar** : CPU et RAM (vert, jaune)
- **OdsActionMenu** : Menu ⋮

## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host
  - Liste hosts
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/{hostId}/maintenance
  - Mode maintenance
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/{hostId}/reboot
  - Redémarrer
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/host/{hostId}
  - Supprimer
