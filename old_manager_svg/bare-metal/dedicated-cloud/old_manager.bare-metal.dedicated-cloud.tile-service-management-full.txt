# Cahier des Charges : bare-metal/dedicated-cloud - Tile Service Management Full

## 1. VUE D'ENSEMBLE

**Type:** Tile gestion service
**Fonction:** Afficher mailing list, renouvellement, résiliation
**Contexte:** Composant dashboard

## 2. LAYOUT

- Tile: 450x280px
- Background: #FFFFFF
- Border: #E5E7EB, 1px
- Border-radius: 8px

## 3. CONTENU

### Titre
- "Gestion du service" (16px, font-weight: 600)

### Mailing list
| Élément | Type |
|---------|------|
| Label | "Mailing list" |
| Valeur | email@domain.com |
| Action | Menu ⋮ |

### Menu Mailing list
| Option |
|--------|
| S'inscrire à la mailing-list |

### Renouvellement automatique
| Élément | Type |
|---------|------|
| Label | "Renouvellement automatique" |
| Date | JJ/MM/AAAA |
| Badge | "Activé" (vert) ou "Désactivé" |
| Action | Menu ⋮ |

### Séparateur
- Ligne horizontale #E5E7EB

### Bouton résiliation
| Élément | Style |
|---------|-------|
| Label | "Résilier le service" |
| Couleur texte | #DC2626 |
| Border | #E5E7EB, 1px |
| Background | #FFFFFF |
| Dimensions | Full width, 40px height |

## 4. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/serviceInfos
PUT /dedicatedCloud/{serviceName}/serviceInfos
POST /dedicatedCloud/{serviceName}/terminate
GET /me/mailingList
POST /me/mailingList/subscribe
```
