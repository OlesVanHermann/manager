# Cahier des Charges : bare-metal/dedicated-cloud - Modal Modifier Utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification avec formulaire
- **Contexte** : Gestion des utilisateurs vSphere
- **Objectif** : Modifier les informations et permissions d'un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 590px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs texte (prénom, nom, email, téléphone)
- **OdsToggle** : Permissions utilisateur
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'utilisateur" |
| Fond | Gris clair (#F9FAFB) |
| Couleur texte | Gris foncé (#1F2937) |
| Bouton fermer | × |

## 5. CONTENU

### Champs de formulaire
| Champ | Type | Obligatoire | Valeur exemple |
|-------|------|-------------|----------------|
| Prénom | OdsInput | Non | Jean |
| Nom | OdsInput | Non | Dupont |
| Email | OdsInput | Oui | jean.dupont@example.com |
| Téléphone | OdsInput | Non | +33 1 23 45 67 89 |

### Permissions (Toggles)
| Permission | Description | État |
|------------|-------------|------|
| Validateur de token | Peut valider les tokens | Off (gris) |
| Gestion réseau | Accès gestion réseau | On (bleu #0050D7) |
| Gestion IP Failover | Gestion des IP failover | Off (gris) |
| NSX | Accès NSX | On (bleu) |
| Chiffrement | Accès au chiffrement | Off (gris) |

### Style des toggles
| État | Couleur fond | Position cercle |
|------|--------------|-----------------|
| Off | Gris (#E5E7EB) | Gauche |
| On | Bleu (#0050D7) | Droite |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Enregistre les modifications |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/user/{userId} - Détails utilisateur
- PUT /dedicatedCloud/{serviceName}/user/{userId} - Modifier utilisateur
