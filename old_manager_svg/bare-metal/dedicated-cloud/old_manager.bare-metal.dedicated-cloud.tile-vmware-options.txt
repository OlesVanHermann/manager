# Cahier des Charges : bare-metal/dedicated-cloud - Tile VMware Options

## 1. VUE D'ENSEMBLE
- **Type** : Tuile options VMware
- **Contexte** : Dashboard Dedicated Cloud
- **Objectif** : Afficher et gérer les options VMware (Zerto, NSX, vROps)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsText** : Labels et valeurs
- **OdsBadge** : États activation
- **OdsPopover** : Menus actions
- **OdsLink** : Liens documentation

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Options" |

## 5. OPTIONS AFFICHÉES

### Zerto (DRP)
| Propriété | Valeur |
|-----------|--------|
| Label | "Zerto (DRP)" |
| Badge actif | Fond #DCFCE7, texte #16A34A "Actif" |
| Actions menu | Configurer VPN, Désactiver (rouge) |

### NSX
| Propriété | Valeur |
|-----------|--------|
| Label | "NSX" |
| Valeur | "Activé" |
| Lien | "Découvrir NSX ↗" |
| Actions | Menu ⋮ |

### vRealize Operations (vROps)
| Propriété | Valeur |
|-----------|--------|
| Label | "vRealize Operations (vROps)" |
| Valeur | "Désactivé" (gris #9CA3AF) |
| Lien | "Découvrir vROps ↗" |
| Actions | Menu ⋮ |

## 6. MENU ACTIONS ZERTO (OUVERT)
| Action | Style |
|--------|-------|
| Configurer VPN | Texte #1F2937 |
| Désactiver | Texte #DC2626 (rouge) |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/zerto - Statut Zerto
- PUT /dedicatedCloud/{serviceName}/zerto/vpn - Configurer VPN
- DELETE /dedicatedCloud/{serviceName}/zerto - Désactiver Zerto
- GET /dedicatedCloud/{serviceName}/nsx - Statut NSX
- GET /dedicatedCloud/{serviceName}/vrops - Statut vROps
- POST /dedicatedCloud/{serviceName}/vrops/enable - Activer vROps
