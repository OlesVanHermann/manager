# Cahier des Charges : bare-metal/dedicated-cloud - Component Service Pack Picker

## 1. VUE D'ENSEMBLE

**Type:** Composant de sélection (Radio cards)
**Fonction:** Permettre la sélection d'un Service Pack
**Contexte:** Wizard de configuration ou upgrade

## 2. DIMENSIONS

- Container: 1100x300px
- Cards: 320x260px chacune
- Espacement: 20px

## 3. STRUCTURE DES CARDS

### Radio button
| État | Style |
|------|-------|
| Sélectionné | Cercle #0050D7, centre rempli |
| Non sélectionné | Cercle vide, border #D1D5DB |

### Badge "Option actuelle"
- Background: #DCFCE7
- Text: #16A34A, 12px
- Dimensions: 110x22px
- Border-radius: 4px

### Contenu
- Titre: Nom du pack (14px, font-weight: 600)
- Description: 2 lignes (muted, 13px)

### Prix
- Séparateur #E5E7EB
- Format: "+X,XX € HT/mois" (13px)

## 4. SERVICE PACKS DISPONIBLES

### Essentials (Sélectionné)
- Description: "Pack de base avec les fonctionnalités essentielles pour démarrer."
- Prix: +0,00 € HT/mois
- Badge: "Option actuelle"

### Standard
- Description: "Pack standard avec options de backup et monitoring avancés."
- Prix: +50,00 € HT/mois

### Premier
- Description: "Pack complet avec toutes les fonctionnalités et support prioritaire."
- Prix: +150,00 € HT/mois

## 5. ÉTATS DES CARDS

| État | Border |
|------|--------|
| Non sélectionnée | #E5E7EB (1px) |
| Sélectionnée | #0050D7 (2px) |

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/servicePacks
POST /dedicatedCloud/{serviceName}/upgradeServicePack
```
