## 10. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/user
  - Liste utilisateurs
- POST /dedicatedCloud/{serviceName}/user
  - Créer utilisateur
- PUT /dedicatedCloud/{serviceName}/user/{userId}
  - Modifier utilisateur
- POST /dedicatedCloud/{serviceName}/user/{userId}/password
  - Réinitialiser mot de passe
- POST /dedicatedCloud/{serviceName}/user/{userId}/disable
  - Désactiver utilisateur
