# Cahier des Charges : bare-metal/dedicated-cloud - Modal License Enable SPLA

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation
**Fonction:** Activer la licence Windows SPLA sur l'infrastructure
**Contexte:** Déclenché depuis les options de licence du Dedicated Cloud

## 2. DIMENSIONS

- Modal: 600x390px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)
- Shadow: drop-shadow(0 25px 50px rgba(0,0,0,0.25))

## 3. COMPOSANTS UI

### Header Modal
- Background: #F9FAFB
- Titre: "Activation de la licence Windows spla"
- Font-size: 20px, font-weight: 600
- Bouton fermeture: × (coin droit)

### Contenu
- Texte intro: "Veuillez valider les contrats de licences :"

### Checkbox validation
- État: Checked (bleu #0050D7)
- Label: "Je reconnais avoir bien pris connaissance des conditions suivantes :"

### Liste des contrats (liens)
- Contrat de licence Microsoft SPLA ↗
- Conditions Générales d'Utilisation OVHcloud ↗
- Contrat de Services VMware ↗
- Style: text-primary avec icône lien externe

### Bloc info
- Background: #EFF6FF
- Border: 1px solid #BFDBFE
- Texte: Explication de la licence SPLA et facturation

## 4. HEADER MODAL

- Hauteur: 60px
- Background: #F9FAFB
- Titre centré verticalement
- Close button: × à droite

## 5. CONTENU

### Checkbox obligatoire
```
☑ Je reconnais avoir bien pris connaissance des conditions suivantes :
  • Contrat de licence Microsoft SPLA
  • Conditions Générales d'Utilisation OVHcloud
  • Contrat de Services VMware
```

### Message informatif
```
La licence SPLA permet d'utiliser Windows Server sur vos
machines virtuelles. La facturation est basée sur la consommation.
```

## 6. FOOTER MODAL

- Séparateur: ligne 1px #E5E7EB
- Bouton Annuler: btn-secondary (100x36px)
- Bouton Confirmer: btn-primary (100x36px)

## 7. APIS & ENDPOINTS

```
POST /dedicatedCloud/{serviceName}/license/spla/enable
Body: { acceptedContracts: true }
→ Active la licence SPLA

GET /dedicatedCloud/{serviceName}/license/spla/contracts
→ Récupère les URLs des contrats à accepter
```

## 8. VALIDATION

- Le bouton "Confirmer" est désactivé tant que la checkbox n'est pas cochée
- Après confirmation: fermeture modal + notification succès
