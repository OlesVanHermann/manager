# Cahier des Charges : bare-metal/dedicated-cloud - Modal Session Timeout

## 1. VUE D'ENSEMBLE

**Type:** Wizard modal (2 étapes)
**Fonction:** Configurer l'expiration de session utilisateur
**Contexte:** Sécurité VMware Private Cloud

## 2. DIMENSIONS

- Container: 570x320px
- Background: #F9FAFB
- Border-radius: 8px

## 3. STEPPER

| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Active (#0050D7) |
| 2 | Confirmation | Inactive (#E5E7EB) |

## 4. ÉTAPE 1 - CONFIGURATION

### Header
- Titre: "Modifier l'expiration de session" (18px, font-weight: 600)

### Champs
| Champ | Type | Dimensions |
|-------|------|------------|
| Session n'expire jamais | Checkbox | 18x18px |
| Timeout (minutes) | Input number | 200x40px |

### Valeur par défaut
- Timeout: 30 minutes

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (border) | Fermer modal |
| Suivant | Primary (#0050D7) | Étape 2 |

## 6. APIS & ENDPOINTS

```
PUT /dedicatedCloud/{serviceName}/user/{userId}
{
  "sessionTimeout": 30,
  "sessionNeverExpire": false
}
```
