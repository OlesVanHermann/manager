# Cahier des Charges : bare-metal/dedicated-cloud - User Rights

## 1. VUE D'ENSEMBLE

**Type:** Page listing avec datagrid
**Fonction:** Afficher les droits d'un utilisateur par datacenter
**Contexte:** Gestion permissions utilisateurs VMware Private Cloud

## 2. LAYOUT

- Page: 1100x450px
- Background: #F9FAFB

## 3. NAVIGATION

### Bouton retour
- Format: "← Droits de **{username}** par datacenter"
- Color: #0050D7 (text-link)

## 4. DATAGRID

### Conteneur
- Width: 1020px
- Height: 350px
- Background: #FFFFFF
- Border: #E5E7EB
- Border-radius: 4px

### Header
- Background: #F3F4F6
- Height: 40px
- Font-size: 12px, font-weight: 600

### Colonnes
| Colonne | Largeur | Type |
|---------|---------|------|
| Datacenter | 140px | Texte |
| Accès vSphere | 180px | Texte |
| Ajout ressource | 180px | Texte (Oui/Non) |
| Rôle VM Network | 180px | Texte |
| Rôle Network | 180px | Texte |
| Actions | 160px | Menu ⋮ |

### Valeurs Accès vSphere / Rôles
| Valeur | Description |
|--------|-------------|
| Lecture/écriture | Accès complet |
| Lecture seule | Consultation uniquement |
| Aucun accès | Pas de droit |

### Menu Actions
| Option | Description |
|--------|-------------|
| Modifier les droits | Ouvre modal de modification |

### Pagination
- Format: "1 - X sur Y"
- Font-size: 12px
- Color: text-muted

## 5. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/user/{userId}/right
PUT /dedicatedCloud/{serviceName}/user/{userId}/right/{rightId}
```
