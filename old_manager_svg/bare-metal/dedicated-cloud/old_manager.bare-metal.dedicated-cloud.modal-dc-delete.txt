# Cahier des Charges : bare-metal/dedicated-cloud - Modal DC Delete

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation danger
- **Contexte** : Gestion Dedicated Cloud
- **Objectif** : Supprimer un datacenter virtuel

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 170px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le datacenter" |
| Fond | #F9FAFB |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir supprimer le datacenter" |
| Datacenter | {datacenterName} en gras |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger #DC2626 | Supprime datacenter |

## 7. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}
