# Cahier des Charges : bare-metal/dedicated-cloud - Tile Service Management

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard Dedicated Cloud
- **Objectif** : Gérer les paramètres du service (mailing list, expiration, résiliation)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsText** : Labels et valeurs
- **OdsBadge** : État renouvellement
- **OdsPopover** : Menu actions
- **OdsLink** : Lien commande
- **OdsButton** : Bouton résiliation

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Gestion du service" |

## 5. CHAMPS AFFICHÉS

### Mailing list
| Propriété | Valeur |
|-----------|--------|
| Label | "Mailing list" |
| Valeur | {email} "user@example.com" |
| Actions | Menu ⋮ (S'abonner à la ML) |

### Expiration du service
| Propriété | Valeur |
|-----------|--------|
| Label | "Expiration du service" |
| Valeur | {expirationDate} "15/03/2025" |
| Badge | Fond #DCFCE7, texte #16A34A "Renouvellement auto" |
| Actions | Menu ⋮ (Gérer le renouvellement) |

## 6. LIENS
| Propriété | Valeur |
|-----------|--------|
| Lien commande | "Commander des licences ↗" |

## 7. BOUTON RÉSILIATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Résilier le service" |
| Style | Bordure #DC2626 (rouge) |
| Texte couleur | #DC2626 |
| Action | Ouvre modal de confirmation |

## 8. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName} - Infos service
- GET /dedicatedCloud/{serviceName}/serviceInfos - Date expiration
- POST /dedicatedCloud/{serviceName}/mailinglist/subscribe - S'abonner ML
- DELETE /dedicatedCloud/{serviceName}/terminate - Résilier
