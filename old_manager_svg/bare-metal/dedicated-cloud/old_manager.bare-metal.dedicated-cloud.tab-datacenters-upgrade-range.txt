# Cahier des Charges : bare-metal/dedicated-cloud - Tab Datacenters Upgrade Range

## 1. VUE D'ENSEMBLE

**Type:** Page de mise à niveau
**Fonction:** Migrer les hôtes vers une gamme supérieure
**Contexte:** Accessible depuis l'onglet Datacenters du Dedicated Cloud

## 2. DIMENSIONS

- Container principal: 1360x550px
- Cartes gamme: 400x100px

## 3. LAYOUT & STRUCTURE

### Breadcrumb
- Hosted Private Cloud VMware > pcc-192-0-2-1 > Datacenters > **Mise à niveau gamme**

### Zone principale
1. Titre + description
2. Bannière avertissement
3. Comparaison gamme actuelle → gamme cible
4. Impact facturation
5. Boutons action

## 4. COMPOSANTS UI

### Titre
- Text: "Mise à niveau de la gamme de vos hôtes"
- Description: "Mettez à niveau vos hôtes vers une gamme supérieure..."

### Bannière avertissement
- Background: #FEF3C7
- Icône: ⚠
- Messages:
  - "Cette opération nécessite une migration des machines virtuelles..."
  - "Planifiez cette opération pendant une fenêtre de maintenance."

### Carte Gamme actuelle
- Border: 1px solid #E5E7EB
- Contenu:
  - Nom: "Standard-L"
  - Specs: "64 vCPU - 256 Go RAM"
  - Info: "4 hôtes dans cette gamme"

### Flèche transition
- Icône: → (bleu primary)

### Carte Gamme cible (sélectionnée)
- Border: 2px solid #0050D7
- Background: #EFF6FF
- Radio: checked
- Contenu:
  - Nom: "Standard-XL" (text-primary)
  - Specs: "128 vCPU - 512 Go RAM"
  - Bonus: "+50% de performances"

### Impact facturation
- Background: #F9FAFB
- Texte: "Coût mensuel supplémentaire : +400,00 € HT/mois (pour 4 hôtes)"

## 5. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary
- Action: Retour page précédente

### Bouton "Lancer la migration"
- Style: btn-primary (180x40px)
- Action: Déclenche la migration avec confirmation

## 6. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/datacenter/{dcId}/host/ranges
→ Liste les gammes disponibles

POST /dedicatedCloud/{serviceName}/datacenter/{dcId}/host/upgrade
Body: { targetRange: "standard-xl" }
→ Lance la migration vers la nouvelle gamme
```
