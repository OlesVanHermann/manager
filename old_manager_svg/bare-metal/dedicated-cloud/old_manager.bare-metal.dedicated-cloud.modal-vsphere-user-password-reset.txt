# Cahier des Charges : bare-metal/dedicated-cloud - Modal vSphere User Password Reset

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Changer le mot de passe d'un utilisateur vSphere
**Contexte:** Reset password utilisateur

## 2. DIMENSIONS

- Modal: 450x360px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. HEADER

- Titre: "Changer le mot de passe" (18px, font-weight: 600)
- Bouton fermer: × (20px, muted)
- Séparateur: #E5E7EB

## 4. CONTENU

### Texte info
- "Vous allez modifier le mot de passe de l'utilisateur **admin-user**"

### Formulaire
| Champ | Type | Dimensions |
|-------|------|------------|
| Nouveau mot de passe | Password | 400x40px |
| Confirmer le mot de passe | Password | 400x40px |

### Règles password (info)
- Icône: Cercle info bleu (#EFF6FF, #0050D7)
- Message: "Le mot de passe doit contenir au moins 8 caractères, une majuscule, une minuscule et un chiffre."
- Font-size: 12px, muted

## 5. FOOTER

### Boutons
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | Secondary (border) | 90x36px |
| Confirmer | Primary (#0050D7) | 90x36px |

## 6. VALIDATION

- Minimum 8 caractères
- Au moins une majuscule
- Au moins une minuscule
- Au moins un chiffre
- Les deux champs doivent correspondre

## 7. APIS & ENDPOINTS

```
POST /dedicatedCloud/{serviceName}/user/{userId}/changePassword
GET /dedicatedCloud/{serviceName}/user/{userId}
```
