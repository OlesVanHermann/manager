# Cahier des Charges : bare-metal/dedicated-cloud - Modal Convertir Datastore Global

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion des datastores Dedicated Cloud
- **Objectif** : Convertir un datastore local en datastore global (accessible depuis tous les datacenters)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 170px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Convertir en datastore global" |
| Fond | Gris clair (#F9FAFB) |
| Couleur texte | Gris foncé (#1F2937) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Confirmez-vous la conversion du datastore" |
| Ligne 2 | "{datastore_name} en global ?" |
| Nom datastore | Bleu (#0050D7), font-weight 500 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Convertit le datastore |

## 7. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/datastore/{datastoreId}/convertToGlobal - Convertir en global
