## 7. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/task
Query: { state: "doing|done|todo" }
→ Liste les opérations avec filtre

GET /dedicatedCloud/{serviceName}/task/{taskId}
→ Détails d'une opération
```
