# Cahier des Charges : bare-metal/dedicated-cloud - Modal VM Set License

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire avec select
- **Contexte** : Gestion licences SPLA Dedicated Cloud
- **Objectif** : Attribuer une licence à une VM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 420px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection licence
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Définir la licence" |
| Fond | #F9FAFB |

## 5. CHAMP LICENCE
| Propriété | Valeur |
|-----------|--------|
| Label | "Licence" |
| Type | Select avec dropdown |
| Placeholder | "Sélectionner une licence..." |

### Options disponibles
| Licence |
|---------|
| WINDOWS_SERVER_2016_STANDARD |
| WINDOWS_SERVER_2019_STANDARD |
| WINDOWS_SERVER_2022_DATACENTER |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary #0050D7 | Attribue licence |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId}/spla/availableLicenses
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId}/spla
