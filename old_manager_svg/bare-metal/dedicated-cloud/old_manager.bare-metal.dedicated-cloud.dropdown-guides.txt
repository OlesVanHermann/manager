# Cahier des Charges : bare-metal/dedicated-cloud - Dropdown Guides

## 1. VUE D'ENSEMBLE

**Type:** Dropdown Menu (Menu déroulant)
**Fonction:** Proposer des liens rapides vers la documentation et les guides
**Contexte:** Accessible depuis un bouton "Guides" dans le header ou dashboard

## 2. DIMENSIONS

- Bouton trigger: 100x36px
- Menu dropdown: 350x165px
- Border-radius: 6px
- Shadow: drop-shadow(0 4px 6px rgba(0,0,0,0.1))

## 3. COMPOSANTS UI

### Bouton Trigger
- Label: "Guides"
- Icône: ▼ (chevron down)
- Style: btn-secondary (fond blanc, border gris)

### Menu Dropdown
- Background: #FFFFFF
- Border: 1px solid #E5E7EB
- Position: Sous le bouton, aligné à gauche

### Items du menu
3 entrées avec:
- Padding vertical: ~50px par item
- Hover: Background #F9FAFB (alternance)
- Icône externe: ↗ (lien externe)

## 4. DONNÉES & COLONNES

| Item | Label | Action |
|------|-------|--------|
| 1 | Découvrir l'espace client | Lien externe documentation |
| 2 | Accès à vSphere | Lien externe guide vSphere |
| 3 | Activer et utiliser Veeam Backup | Lien externe guide Veeam |

## 5. ACTIONS & INTERACTIONS

### Ouverture menu
- Click sur bouton "Guides" → Toggle visibilité dropdown
- Click extérieur → Ferme le dropdown
- Escape → Ferme le dropdown

### Click sur item
- Action: Ouvre lien externe (target="_blank")
- Icône ↗ indique ouverture nouvel onglet

## 6. APIS & ENDPOINTS

Aucune API requise - liens statiques vers documentation:
```
https://docs.ovh.com/fr/dedicated-cloud/
https://docs.ovh.com/fr/dedicated-cloud/vsphere-interface/
https://docs.ovh.com/fr/dedicated-cloud/veeam-backup/
```
