# Cahier des Charges : bare-metal/dedicated-cloud - Modal Datastore Convert Global

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation simple
- **Contexte** : Gestion Dedicated Cloud
- **Objectif** : Convertir un datastore en datastore global

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 190px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Convertir en datastore global" |
| Fond | #F9FAFB |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir convertir le datastore" |
| Datastore | {datastoreName} en gras |
| Suite | "en datastore global ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary #0050D7 | Convertit datastore |

## 7. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/{filerId}/convertToGlobal
