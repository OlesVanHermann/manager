# Cahier des Charges : bare-metal/dedicated-cloud - Modal VM Delete License

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation danger
- **Contexte** : Gestion licences VM Dedicated Cloud
- **Objectif** : Désactiver la licence d'une machine virtuelle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Désactiver

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Désactiver la licence" |
| Fond | #F9FAFB |

## 5. CONTENU
| Ligne | Texte |
|-------|-------|
| 1 | "Êtes-vous sûr de vouloir désactiver la licence de cette" |
| 2 | "machine virtuelle ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Désactiver | Danger #DC2626 | Désactive licence |

- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/vm/{vmId}/spla
