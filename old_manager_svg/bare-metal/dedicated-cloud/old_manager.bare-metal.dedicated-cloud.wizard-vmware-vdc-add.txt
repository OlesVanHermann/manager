## 1. VUE D'ENSEMBLE

**Type:** Wizard 2 étapes
**Fonction:** Ajouter un nouveau datacenter virtuel (VDC) au Private Cloud
**Contexte:** Extension de l'infrastructure avec un nouveau VDC

- Container principal: 1440x800px
- Zone wizard: 1360x650px

## 6. ACTIONS & INTERACTIONS

### Sélection carte
- Click sur carte → sélectionne le type
- Mise à jour du message info selon le type sélectionné

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Position: bas gauche
- Action: Retour au dashboard

### Bouton Suivant
- Style: btn-primary (100x36px)
- Position: bas droite
- Action: Passe à l'étape 2 (Profil d'hôte)

## 7. ÉTAPE 2 : PROFIL D'HÔTE (non visible)

### Contenu
- Sélection du profil d'hôte initial
- Configuration des ressources
- Récapitulatif et validation

## 8. APIS & ENDPOINTS

```
GET /dedicatedCloud/{serviceName}/availableVdcTypes
→ Liste les types de VDC disponibles

GET /dedicatedCloud/{serviceName}/availableHostProfiles
→ Liste les profils d'hôte pour le type sélectionné

POST /dedicatedCloud/{serviceName}/datacenter
Body: {
  type: "NSX-T" | "NSX-V" | "vRack",
  hostProfile: "Host L",
  quantity: 2
}
→ Crée un nouveau datacenter virtuel
```
