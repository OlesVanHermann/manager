# Cahier des Charges : bare-metal/dedicated-cloud - Modal Terminate Confirm

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation avec questionnaire
**Fonction:** Recueillir le motif de résiliation avant confirmation finale
**Contexte:** Deuxième étape du processus de résiliation du Dedicated Cloud

## 2. DIMENSIONS

- Modal: 700x600px
- Border-radius: 8px
- Overlay: rgba(0,0,0,0.5)

## 3. COMPOSANTS UI

### Header Modal (Danger)
- Background: #FEE2E2 (rouge clair)
- Titre: "Confirmation de la fermeture du service"
- Color titre: #991B1B
- Close button: × (couleur #991B1B)

### Question principale
- Texte: "Pourrions-nous savoir quelle est la principale raison qui vous amène à fermer ce service ?"
- Font-weight: 500

### Options Radio (8 choix)
1. ● Je n'utilise plus le service (sélectionné par défaut)
2. ○ Les outils sont trop complexes
3. ○ Les performances du système sont insuffisantes
4. ○ Le service manque de fiabilité
5. ○ Le service coûte trop cher
6. ○ J'utilise un autre service OVH
7. ○ J'utilise le service d'un concurrent
8. ○ Autres...

### Zone de commentaire
- Label: "Aimeriez-vous nous donner plus de détails ou nous faire une suggestion ?"
- Textarea: 620x60px
- Placeholder: "Écrivez votre commentaire ici..."
- Compteur: "0/450"

## 4. HEADER MODAL

- Hauteur: 60px
- Background: #FEE2E2 (danger light)
- Border-radius: 8px (top)
- Titre: color #991B1B

## 5. CONTENU

### Questionnaire de feedback
| Option | Valeur |
|--------|--------|
| no_longer_use | Je n'utilise plus le service |
| too_complex | Les outils sont trop complexes |
| poor_performance | Les performances du système sont insuffisantes |
| unreliable | Le service manque de fiabilité |
| too_expensive | Le service coûte trop cher |
| other_ovh_service | J'utilise un autre service OVH |
| competitor | J'utilise le service d'un concurrent |
| other | Autres... |

### Textarea feedback
- Max length: 450 caractères
- Compteur en temps réel

## 6. FOOTER MODAL

- Séparateur: ligne 1px
- Bouton "J'ai changé d'avis": btn-secondary (170x36px)
- Bouton "Je confirme vouloir": btn-danger #DC2626 (140x36px)

## 7. APIS & ENDPOINTS

```
POST /dedicatedCloud/{serviceName}/terminate/confirm
Body: {
  reason: "no_longer_use",
  feedback: "Commentaire optionnel..."
}
→ Confirme la résiliation avec le motif

DELETE /dedicatedCloud/{serviceName}
→ Déclenche effectivement la résiliation
```

## 8. VALIDATION

- Au moins une option radio sélectionnée
- Commentaire optionnel (max 450 caractères)
