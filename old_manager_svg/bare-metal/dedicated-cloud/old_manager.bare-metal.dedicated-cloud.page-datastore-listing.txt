# Cahier des Charges : bare-metal/dedicated-cloud - Page Datastore Listing

## 1. VUE D'ENSEMBLE
- **Type** : Page listing datastores
- **Contexte** : Dedicated Cloud (VMware)
- **Objectif** : Gérer les datastores de l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajouter datastore
- **OdsDatagrid** : Tableau datastores
- **OdsBadge** : Type et état
- **OdsProgressBar** : Utilisation
- **OdsPopover** : Menu actions
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Datastores" |
| Description | "Gérez les datastores de votre infrastructure VMware." |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter un datastore" |
| Style | Fond #0050D7, texte blanc |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Nom | 180px | Nom datastore (bold) |
| Type | 100px | Badge type |
| Datacenter | 150px | Nom DC |

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/datastore - Liste datastores
- POST /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/orderNewDatastore - Commander
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/datastore/{datastoreId} - Supprimer
