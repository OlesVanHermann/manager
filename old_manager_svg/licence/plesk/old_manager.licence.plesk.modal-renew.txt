# Cahier des Charges : licence/plesk - Modal Renew

## 1. VUE D'ENSEMBLE
- **Type** : Modal renouvellement licence Plesk
- **Contexte** : Licences Plesk
- **Objectif** : Renouveler licence et extensions ensemble

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 340px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsRadio** : Sélection durée
- **OdsButton** : Annuler, Renouveler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Renouveler la licence Plesk" |
| Bouton fermer | × |

- GET /license/plesk/{serviceName} - Détail licence
