# Cahier des Charges : licence/windows - Page Activation

## 1. VUE D'ENSEMBLE
- **Type** : Page guide d'activation Windows Server
- **Contexte** : Licences Windows Server
- **Objectif** : Guider l'utilisateur pour activer sa licence

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et instructions
- **OdsLink** : Retour et aide
- **OdsCard** : Cartes étapes
- **OdsClipboard** : Copie clé et commandes
- **OdsBadge** : Statut activation
- **OdsMessage** : Notes informatives

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour au détail de la licence" |
| Titre | "Guide d'activation Windows Server" |
| Sous-titre | Service name (monospace) |

| Propriété | Valeur |
|-----------|--------|
| Largeur carte | 500px |
| Titre | "Étape 2 : Activer via PowerShell" |

### Commandes
```powershell
# Installer la clé
slmgr.vbs /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
# Activer Windows
slmgr.vbs /ato
```
