# Cahier des Charges : licence/cPanel - Page History

## 1. VUE D'ENSEMBLE
- **Type** : Page historique licence cPanel
- **Contexte** : Licences cPanel
- **Objectif** : Consulter l'historique des événements d'une licence

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et breadcrumb
- **OdsLink** : Retour au détail
- **OdsDatagrid** : Liste événements
- **OdsBadge** : Type événement
- **OdsPagination** : Navigation pages

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour au détail de la licence" |
| Titre | "Historique de la licence" |
| Sous-titre | Service name (monospace) |

## 5. DATAGRID ÉVÉNEMENTS
| Colonne | Description |
|---------|-------------|
| Date | Format "DD MMM YYYY HH:MM" |
| Type | Badge coloré |
| Détails | Description événement |
| IP | Adresse IP associée (monospace) |

## 6. TYPES ÉVÉNEMENTS (BADGES)
| Type | Fond | Texte |
|------|------|-------|
| Renouvellement | #DCFCE7 | #16A34A |
| Changement IP | #EFF6FF | #0050D7 |
| Upgrade | #F3E8FF | #7C3AED |
| Création | #0050D7 | white |

## 7. ÉVÉNEMENTS TRACÉS
| Événement | Détails affichés |
|-----------|------------------|
| Création | "Nouvelle licence {édition} commandée" |
| Renouvellement | "Renouvellement automatique - {durée}" |
| Changement IP | "{ancienne IP} → {nouvelle IP}" |
| Upgrade | "{ancienne édition} → {nouvelle édition}" |

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Page X sur Y" |
| Boutons | ‹ Précédent, Numéros, Suivant › |

## 9. APIS & ENDPOINTS
- GET /license/cpanel/{serviceName}/tasks - Historique tâches
- GET /license/cpanel/{serviceName}/serviceInfos - Infos service
