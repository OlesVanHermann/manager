# Cahier des Charges : licence/cPanel - Modal Change IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification adresse IP
- **Contexte** : Licences cPanel
- **Objectif** : Modifier l'adresse IP associée à une licence cPanel

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 300px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsInput** : Nouvelle IP
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Modifier l'adresse IP" |
| Bouton fermer | × |

|-----------|--------|

| Propriété | Valeur |
|-----------|--------|
| Label | "IP actuelle" |
| Fond | #F9FAFB |
| Valeur | Adresse IP (monospace, lecture seule) |
