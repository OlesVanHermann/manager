# Cahier des Charges : licence/cloudlinux - Modal Renew

## 1. VUE D'ENSEMBLE
- **Type** : Modal renouvellement licence CloudLinux
- **Contexte** : Licences CloudLinux
- **Objectif** : Renouveler licence et add-ons ensemble

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 320px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsRadio** : Sélection durée
- **OdsButton** : Annuler, Renouveler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Renouveler CloudLinux" |
| Bouton fermer | × |

## 5. RÉCAPITULATIF LICENCE + ADD-ONS
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Label | "Licence et add-ons" |

### Détail tarification
| Élément | Affichage |
|---------|-----------|
| Licence | "CloudLinux {version}" avec prix |
| Add-on | "+ {Nom}" avec prix |
| Séparateur | Ligne #E5E7EB |
| Total | "Total" avec somme (bold) |

## 6. SÉLECTION DURÉE
| Durée | Description |
|-------|-------------|
| 1 mois | Sélectionné par défaut |
| 6 mois | Option moyenne durée |
| 12 mois | Option longue durée |

### Style sélection
| État | Style |
|------|-------|
| Sélectionné | Border #0050D7 2px |
| Non sélectionné | Border #E5E7EB |

## 7. TOTAL PRIX
| Propriété | Valeur |
|-----------|--------|

| Label | "Protection continue garantie" |

| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Renouveler | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- GET /license/cloudLinux/{serviceName} - Détail licence
- GET /license/cloudLinux/{serviceName}/options - Add-ons actifs
- POST /order/license/cloudLinux/{serviceName}/renew - Renouveler
