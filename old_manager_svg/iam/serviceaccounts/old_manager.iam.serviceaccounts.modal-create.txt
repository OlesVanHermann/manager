# Cahier des Charges : iam/serviceaccounts - Modal Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création compte de service
- **Contexte** : IAM Service Accounts
- **Objectif** : Créer un compte pour automatisation/CI/CD

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 460px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : Nom, description
- **OdsSelect** : Politiques d'accès (multi)
- **OdsCheckbox** : Options
- **OdsMessage** : Avertissement credentials
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Créer un compte de service" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT NOM COMPTE
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du compte" |
| Largeur | 450px |
| Placeholder | "terraform-staging" |

## 6. INPUT DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Label | "Description" |
| Largeur | 450px |
| Placeholder | "Compte pour infrastructure as code" |

## 7. SELECT POLITIQUES
| Propriété | Valeur |
|-----------|--------|
| Label | "Politiques d'accès" |
| Largeur | 450px |
| Type | Multi-select |
| Valeur exemple | "ComputeAdmin, StorageReadWrite" |

## 8. CHECKBOXES OPTIONS
| Option | État défaut |
|--------|-------------|
| Générer des credentials (App Key/Secret) | Coché |
| Activer les logs d'audit détaillés | Non coché |

## 9. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠️ |
| Texte | "Les credentials ne seront affichés qu'une seule fois après création." |

## 10. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Créer | Primary #0050D7 | 100px |

## 11. APIS & ENDPOINTS
- POST /me/identity/serviceAccount - Créer compte
- GET /me/iam/policy - Liste politiques disponibles
