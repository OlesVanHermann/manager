# Cahier des Charges : iam/apikeys - Modal Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création clé API
- **Contexte** : IAM API Keys
- **Objectif** : Créer un nouveau token d'accès API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 540px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsInput** : Nom, description, IPs
- **OdsCheckbox** : Permissions
- **OdsButton** : Annuler, Créer
- **Cards sélection** : Durée validité
