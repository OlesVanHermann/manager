# Cahier des Charges : iam/apikeys - Page Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail clé API
- **Contexte** : IAM API Keys
- **Objectif** : Consulter les détails et l'utilisation d'une clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsButton** : Éditer, Révoquer
- **OdsCard** : Cartes informations, permissions, utilisation
- **OdsCheckbox** : Permissions (lecture seule)
- **OdsDatagrid** : Historique utilisation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | Nom de l'application (ex: "CI/CD Pipeline") |
| Description | Description de la clé |

### Boutons actions
| Bouton | Style | Largeur |
|--------|-------|---------|
| Éditer | Outline | 80px |
| Révoquer | Danger outline (#FEE2E2) | 80px |

## 5. CARTE INFORMATIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 510px |
| Hauteur | 200px |
| Titre | "Informations" |

### Champs
| Champ | Contenu |
|-------|---------|
| Application Key | Clé + bouton copier |
| Créée le | Date création |
| Expire le | Date expiration |
| État | Badge (Actif, Expiré, Révoqué) |

## 6. CARTE PERMISSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 510px |
| Hauteur | 200px |
| Titre | "Permissions" |

### Checkboxes permissions
| Permission | État |
|------------|------|
| GET (lecture) | ✓/#16A34A ou ✗/#6B7280 |
| POST (création) | ✓/#16A34A ou ✗/#6B7280 |
| PUT (modification) | ✓/#16A34A ou ✗/#6B7280 |
| DELETE (suppression) | ✓/#16A34A ou ✗/#6B7280 |

### IP autorisées
| Propriété | Valeur |
|-----------|--------|
| Format | Liste CIDR séparée par virgules |

## 7. CARTE UTILISATION RÉCENTE
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1040px |
| Hauteur | 200px |
| Titre | "Utilisation récente" |

### Colonnes tableau
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Date | 170px | Timestamp |
| Endpoint | 360px | Path API (monospace) |
| Méthode | 120px | Badge GET/POST/PUT/DELETE |
| IP Source | 160px | Adresse IP |
| Status | 100px | Badge code HTTP |

## 8. BADGES MÉTHODE HTTP
| Méthode | Fond | Texte |
|---------|------|-------|
| GET | #DBEAFE | #1E40AF |
| POST | #DCFCE7 | #16A34A |
| PUT | #FEF3C7 | #92400E |
| DELETE | #FEE2E2 | #DC2626 |

## 9. APIS & ENDPOINTS
- GET /me/api/application/{applicationId} - Détails application
- GET /me/api/credential/{credentialId} - Détails credential
- GET /me/api/logs?credentialId={id} - Historique utilisation
