# Cahier des Charges : iam/apikeys - Modal Rotate

## 1. VUE D'ENSEMBLE
- **Type** : Modal rotation clé API
- **Contexte** : IAM API Keys
- **Objectif** : Générer nouvelle clé avec période de grâce

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 360px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et descriptions
- **OdsRadio** : Sélection période de grâce
- **OdsMessage** : Info mise à jour
- **OdsButton** : Annuler, Rotation

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #FEF3C7 (warning) |
| Titre | "Rotation de la clé API" |
| Couleur titre | #92400E |
| Bouton fermer | × en #92400E |

## 5. INFO CLÉ ACTUELLE
| Propriété | Valeur |
|-----------|--------|
| Question | "Générer une nouvelle clé et révoquer l'ancienne ?" |
| Fond carte | #F9FAFB |
| Label | "Clé actuelle" |
| Clé | Tronquée + nom (monospace) |

## 6. PÉRIODE DE GRÂCE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Période de grâce" |
| Description | "Temps pendant lequel les deux clés seront valides." |

### Options
| Option | Largeur | Description |
|--------|---------|-------------|
| 0 min | 90px | Révocation immédiate |
| 1 heure | 90px | Défaut (sélectionné) |
| 24 heures | 90px | Un jour |
| 7 jours | 90px | Une semaine |

### Style options
| État | Style |
|------|-------|
| Sélectionné | #0050D7 fond, blanc texte |
| Non sélectionné | Blanc, bordure #E5E7EB |

## 7. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Icône | ℹ️ |
| Texte | "Après la rotation, vous devrez mettre à jour vos applications avec la nouvelle clé pendant la période de grâce." |

## 8. BOUTONS FOOTER
| Bouton | Style | Couleur |
|--------|-------|---------|
| Annuler | Outline | #E5E7EB |
| Rotation | Primary | #F59E0B (warning) |

## 9. APIS & ENDPOINTS
- POST /me/api/application/{appId}/key/rotate - Rotation
- GET /me/api/application/{appId}/key - Clés actuelles
