# Cahier des Charges : iam/kms - Modal désactivation clé

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action sur une clé de service
- **Objectif** : Désactiver temporairement une clé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Désactiver

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Désactiver la clé" |
| Warning | "La clé {nom} ne pourra plus être utilisée pour les opérations cryptographiques." |
| Info | "Vous pourrez la réactiver ultérieurement." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Désactiver | Warning | Désactive clé |

## 6. APIS & ENDPOINTS
- POST /kms/{kmsId}/serviceKeys/{keyId}/deactivate
