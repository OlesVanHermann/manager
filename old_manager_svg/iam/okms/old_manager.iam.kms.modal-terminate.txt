# Cahier des Charges : iam/kms - Modal résiliation KMS

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec saisie
- **Contexte** : Action sur un service KMS
- **Objectif** : Résilier définitivement le service KMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 380px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning danger
- **OdsInput** : Confirmation
- **OdsButton** : Annuler, Résilier

## 4. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (rouge) |
| Messages | 
- "Cette action est irréversible."
- "Toutes les clés et credentials seront supprimés."
- "Les données chiffrées avec ces clés seront irrécupérables." |

## 5. CHAMP CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Tapez TERMINATE pour confirmer" |
| Validation | Doit être "TERMINATE" |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Résilier | Danger | Résilie si TERMINATE saisi |

## 7. APIS & ENDPOINTS
- DELETE /kms/{kmsId}/terminate
