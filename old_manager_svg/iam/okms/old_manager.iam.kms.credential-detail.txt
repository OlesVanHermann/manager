# Cahier des Charges : iam/kms - Détail credential

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/kms/{kmsId}/credentials/{credId}
- **Objectif** : Afficher les informations d'un credential

## 2. COMPOSANTS UI
- **OdsBreadcrumb** : Accueil / IAM / KMS / {kms} / Credentials / {cred}
- **OdsBadge** : Statut
- **OdsCard** : Informations
- **OdsButton** : Supprimer

## 3. CARD INFORMATIONS
| Champ | Type |
|-------|------|
| Nom | Texte |
| Access Key ID | Code copiable |
| Description | Texte |
| Statut | Badge |
| Date création | Datetime |

## 4. NOTE SÉCURITÉ
Le Secret Access Key n'est affiché qu'une seule fois lors de la création.

## 5. APIS & ENDPOINTS
- GET /kms/{kmsId}/credentials/{credId} - Détail credential
- DELETE /kms/{kmsId}/credentials/{credId} - Supprimer
