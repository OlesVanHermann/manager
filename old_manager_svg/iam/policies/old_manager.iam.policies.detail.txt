# Cahier des Charges : iam/policies - Détail politique

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/policies/{policyId}
- **Objectif** : Afficher les informations détaillées d'une politique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Politiques / {nom}
- **OdsBadge** : Type (Personnalisé)
- **OdsButton** : Modifier, Supprimer
- **OdsCard** : Cards informations

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | Nom de la politique |
| Badge | Type (Système/Personnalisé) |
| Actions | Modifier (secondary), Supprimer (danger) |

## 5. CARD INFORMATIONS
| Propriété | Valeur |
|-----------|--------|
| Taille | 540x180px |

### Champs affichés
| Champ | Type | Exemple |
|-------|------|---------|
| Description | Texte | Administration des instances compute Public Cloud |
| Créé le | Datetime | 15 décembre 2025 à 14:30 |
| Modifié le | Datetime | 20 décembre 2025 à 10:15 |

## 6. CARD IDENTITÉS ASSOCIÉES
| Propriété | Valeur |
|-----------|--------|
| Taille | 540x180px |
| Titre | "Identités associées (X)" |

### Contenu
- Liste de liens vers les identités (utilisateurs, groupes, services)
- Bouton "+ Ajouter une identité"

## 7. CARD DÉFINITION POLITIQUE (JSON)
| Propriété | Valeur |
|-----------|--------|
| Taille | 1100x380px |
| Titre | "Définition de la politique (JSON)" |

### Éditeur JSON
| Propriété | Valeur |
|-----------|--------|
| Background | #1F2937 |
| Font | Monaco, monospace |
| Size | 11px |
| Syntax highlighting | Oui |

### Couleurs syntaxe
| Élément | Couleur |
|---------|---------|
| Clés | #93C5FD (bleu clair) |
| Strings | #86EFAC (vert clair) |
| Default | #E5E7EB (gris clair) |

## 8. STRUCTURE JSON POLITIQUE
```json
{
  "permissions": [
    {
      "allow": ["action:*"],
      "except": ["action:terminate"],
      "resource": "urn:v1:eu:resource:*"
    }
  ]
}
```

## 9. APIS & ENDPOINTS
- GET /iam/policies/{policyId} - Détail politique
- PUT /iam/policies/{policyId} - Modifier politique
- DELETE /iam/policies/{policyId} - Supprimer politique
- GET /iam/policies/{policyId}/identities - Liste identités attachées
- POST /iam/policies/{policyId}/identities - Attacher identité
