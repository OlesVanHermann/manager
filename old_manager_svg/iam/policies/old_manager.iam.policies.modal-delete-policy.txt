# Cahier des Charges : iam/policies - Modal suppression politique

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec saisie
- **Contexte** : Action menu sur une politique personnalisée
- **Objectif** : Supprimer définitivement une politique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 380px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning box
- **OdsInput** : Champ confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer la politique" |
| Bouton fermer | X |

## 5. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Message | "Cette action est irréversible. Toutes les identités utilisant cette politique perdront les permissions associées." |

## 6. INFORMATIONS
| Élément | Description |
|---------|-------------|
| Label | "Politique à supprimer :" |
| Nom | {nom de la politique} (gras) |
| Identités | "X identités utilisent cette politique" |

## 7. CHAMP CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Tapez DELETE pour confirmer" |
| Validation | Doit être "DELETE" |

## 8. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime si DELETE saisi |

## 9. APIS & ENDPOINTS
- DELETE /iam/policies/{policyId} - Supprimer politique
