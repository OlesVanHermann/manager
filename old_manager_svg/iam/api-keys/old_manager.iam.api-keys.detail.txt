# Cahier des Charges : iam/api-keys - Détail clé API

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/api-keys/{keyId}
- **Objectif** : Afficher les informations détaillées d'une clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Clés API / {nom}
- **OdsBadge** : Statut Actif
- **OdsButton** : Modifier, Supprimer
- **OdsCard** : Cards informations

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | Nom de la clé API |
| Badge | Statut (Actif/Expiré) |
| Actions | Modifier (secondary), Supprimer (danger) |

## 8. APIS & ENDPOINTS
- GET /iam/api-keys/{keyId} - Détail clé API
- PUT /iam/api-keys/{keyId} - Modifier clé API
- DELETE /iam/api-keys/{keyId} - Supprimer clé API
- GET /iam/api-keys/{keyId}/certificates - Liste certificats
- POST /iam/api-keys/{keyId}/certificates - Ajouter certificat
- GET /iam/api-keys/{keyId}/policies - Liste politiques attachées
- POST /iam/api-keys/{keyId}/policies - Attacher politique
