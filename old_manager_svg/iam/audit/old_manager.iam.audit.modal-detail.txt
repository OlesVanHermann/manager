# Cahier des Charges : iam/audit - Modal Detail

## 1. VUE D'ENSEMBLE
- **Type** : Modal détail événement audit
- **Contexte** : IAM Audit Logs
- **Objectif** : Afficher détails complets d'un événement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 510px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et valeurs
- **OdsBadge** : Statut (réussi/échoué)
- **OdsAvatar** : Utilisateur
- **Terminal** : JSON requête
- **OdsButton** : Fermer

## 4. HEADER MODAL
