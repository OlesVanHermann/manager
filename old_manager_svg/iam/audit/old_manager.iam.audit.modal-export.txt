# Cahier des Charges : iam/audit - Modal Export

## 1. VUE D'ENSEMBLE
- **Type** : Modal export logs d'audit
- **Contexte** : IAM Audit
- **Objectif** : Exporter les logs d'audit dans différents formats

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 360px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

- **Cards sélection** : Période, format
- **OdsCheckbox** : Filtres inclusion
