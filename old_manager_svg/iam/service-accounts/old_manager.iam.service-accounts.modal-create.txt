# Cahier des Charges : iam/service-accounts - Modal création

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page comptes de service
- **Objectif** : Créer un nouveau compte de service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom
- **OdsTextarea** : Description
- **OdsSelect** : Politiques
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |
| Description | Textarea | Non |
| Politiques | Multi-select | Non |

## 5. POST-CRÉATION
| Élément | Description |
|---------|-------------|
| Message | "Compte créé avec succès" |
| Action suivante | Proposer de créer une clé API |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée compte |

## 7. APIS & ENDPOINTS
- POST /iam/service-accounts - Créer compte
- GET /iam/policies - Liste politiques
