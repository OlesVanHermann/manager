# Cahier des Charges : iam/groups - Modal Rename

## 1. VUE D'ENSEMBLE
- **Type** : Modal renommer groupe
- **Contexte** : IAM Groups
- **Objectif** : Modifier le nom d'un groupe

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 240px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : Champ nouveau nom
- **OdsButton** : Annuler, Renommer
