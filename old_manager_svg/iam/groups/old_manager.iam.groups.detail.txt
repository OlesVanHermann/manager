# Cahier des Charges : iam/groups - Détail groupe

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/groups/{groupId}
- **Objectif** : Afficher et gérer un groupe

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Accueil / IAM / Groupes / {nom}
- **OdsCard** : Cards informations
- **OdsDatagrid** : Listes membres et politiques
- **OdsButton** : Modifier, Supprimer

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Icône | Initiale colorée |
| Titre | Nom du groupe |
| Description | Sous-titre |
| Actions | Modifier, Supprimer |

## 5. CARD INFORMATIONS
| Champ | Type |
|-------|------|
| Nom | Texte |
| Description | Texte |
| Date création | Datetime |
| Nombre membres | Badge compteur |

## 6. CARD MEMBRES
| Élément | Description |
|---------|-------------|
| Tableau | Liste des utilisateurs du groupe |
| Colonnes | Avatar, Nom, Email, Statut, Actions |
| Action | + Ajouter des membres |
| Action ligne | Retirer du groupe |

## 7. CARD POLITIQUES
| Élément | Description |
|---------|-------------|
| Tableau | Liste des politiques attachées |
| Colonnes | Nom, Type, Description, Actions |
| Action | + Attacher une politique |
| Action ligne | Détacher politique |

## 8. APIS & ENDPOINTS
- GET /iam/groups/{groupId} - Détail groupe
- PUT /iam/groups/{groupId} - Modifier
- DELETE /iam/groups/{groupId} - Supprimer
- GET /iam/groups/{groupId}/users - Membres
- POST /iam/groups/{groupId}/users - Ajouter membre
- DELETE /iam/groups/{groupId}/users/{userId} - Retirer membre
- GET /iam/groups/{groupId}/policies - Politiques
- POST /iam/groups/{groupId}/policies - Attacher politique
