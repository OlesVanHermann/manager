# Cahier des Charges : iam/groups - Page List

## 1. VUE D'ENSEMBLE
- **Type** : Page liste groupes (cartes)
- **Contexte** : IAM Groups
- **Objectif** : Gérer les groupes d'utilisateurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer groupe
- **OdsCard** : Cartes groupes
- **OdsAvatar** : Initiale groupe

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Groupes" |
| Description | "Regroupement d'utilisateurs pour une gestion simplifiée." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Créer groupe" |
| Style | Primary #0050D7 |
| Largeur | 140px |

## 6. CARTE GROUPE
| Propriété | Valeur |
|-----------|--------|
| Largeur | 330px |
| Hauteur | 180px |
| Background | white |
| Bordure | #E5E7EB |

### Contenu carte
| Élément | Description |
|---------|-------------|
| Avatar | Initiale avec couleur |

| Membres | Nombre |
| Politiques | Nombre |

## 9. APIS & ENDPOINTS
- GET /me/identity/group - Liste groupes
- POST /me/identity/group - Créer groupe
- PUT /me/identity/group/{id} - Modifier
- DELETE /me/identity/group/{id} - Supprimer
