# Cahier des Charges : iam/groups - Modal création groupe

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page liste groupes
- **Objectif** : Créer un nouveau groupe d'utilisateurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom du groupe
- **OdsTextarea** : Description
- **OdsSelect** : Membres initiaux
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom du groupe | Input | Oui * |
| Description | Textarea | Non |
| Membres | Multi-select | Non |

## 5. VALIDATION
| Champ | Règle |
|-------|-------|
| Nom | Unique, 3-64 caractères |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée groupe |

## 7. APIS & ENDPOINTS
- POST /iam/groups - Créer groupe
- GET /iam/users - Liste utilisateurs disponibles
