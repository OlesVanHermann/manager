# Cahier des Charges : iam/components - Error State

## 1. VUE D'ENSEMBLE
- **Type** : Composant état d'erreur
- **Usage** : Afficher quand une page/section échoue à charger

## 2. COMPOSANT ODS
- **OdsMessage** : Type error avec actions

## 3. LAYOUT
| Propriété | Valeur |
|-----------|--------|
| Position | Centré dans container |
| Padding | 48px |

## 4. ÉLÉMENTS
| Élément | Description |
|---------|-------------|
| Icône | Cercle rouge avec ! |
| Titre | "Une erreur est survenue" |
| Message | Description de l'erreur |
| Bouton | "Réessayer" |

## 5. STYLE
| Propriété | Valeur |
|-----------|--------|
| Icône taille | 64px |
| Icône couleur | #EF4444 |
| Titre | 18px, font-weight 600 |
| Message | 14px, gris |

## 6. EXEMPLES MESSAGES
| Contexte | Message |
|----------|---------|
| Réseau | "Impossible de se connecter au serveur" |
| Timeout | "La requête a pris trop de temps" |
| Permission | "Vous n'êtes pas autorisé à voir cette page" |
