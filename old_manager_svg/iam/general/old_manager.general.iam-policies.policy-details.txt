## 8. APIS & ENDPOINTS

### 8.1 Obtenir les détails de la politique

```
GET /me/iam/policy/{policyId}
```

Response:
```json
{
  "id": "policy-123",
  "name": "Admin-Full-Access",
  "description": "Politique d'accès complet pour les administrateurs",
  "active": true,
  "createdAt": "2024-06-15T10:00:00Z",
  "updatedAt": "2026-01-02T14:30:00Z"
}
```

### 8.2 Obtenir les identités de la politique

```
GET /me/iam/policy/{policyId}/identity
```

### 8.3 Obtenir les permissions de la politique

```
GET /me/iam/policy/{policyId}/permission
```
