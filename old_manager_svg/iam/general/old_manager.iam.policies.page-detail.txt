# Cahier des Charges : iam/policies - Page Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail politique
- **Contexte** : IAM Policies
- **Objectif** : Consulter et gérer une politique d'accès

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
