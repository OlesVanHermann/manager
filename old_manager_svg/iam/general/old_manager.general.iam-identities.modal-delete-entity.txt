# Cahier des Charges : home/iam - Modal supprimer utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive avec saisie
- **Contexte** : Page identités IAM
- **Objectif** : Supprimer définitivement un utilisateur IAM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Messages d'avertissement
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "⚠️ Supprimer l'utilisateur" |
| Fond | Jaune warning (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Message warning
| Texte |
|-------|
| "Cette action est irréversible." (rouge) |

### Informations entité
| Champ | Valeur exemple |
|-------|----------------|
| Nom de l'utilisateur | john.doe@example.com |

### Champ de confirmation
| Champ | Description |
|-------|-------------|
| Label | 'Pour confirmer, tapez "DELETE" ci-dessous' |
| Placeholder | DELETE |

## 6. FOOTER MODAL
| Bouton | Style | État |
|--------|-------|------|
| Annuler | Secondary | Actif |
| Supprimer | Danger (rouge) | Désactivé tant que "DELETE" non saisi |

## 7. APIS & ENDPOINTS
- DELETE /me/identity/user/{userId} - Supprimer utilisateur IAM
