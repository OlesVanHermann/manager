# Cahier des Charges : iam/policies - Modal Duplicate

## 1. VUE D'ENSEMBLE
- **Type** : Modal duplication politique
- **Contexte** : IAM Policies
- **Objectif** : Créer une copie d'une politique existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 300px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : Nom de la copie
- **OdsCheckbox** : Option éditeur
- **OdsButton** : Annuler, Dupliquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Dupliquer la politique" |
| Bouton fermer | × en #6B7280 |

## 5. INFO POLITIQUE SOURCE
| Propriété | Valeur |
|-----------|--------|
| Label | "Politique source" |
| Fond | #F9FAFB |
| Hauteur | 45px |
| Nom | Nom politique (bold) |
| Détails | "X règles de permission" |

## 6. INPUT NOM COPIE
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom de la copie *" |
| Largeur | 350px |
| Défaut | "{Nom original} (copie)" |
| Bordure focus | 2px #0050D7 |

## 7. CHECKBOX OPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Ouvrir l'éditeur après la duplication" |
| État défaut | Coché |

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Dupliquer | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- POST /me/iam/policy/{policyId}/duplicate - Dupliquer
