## 9. APIS & ENDPOINTS

### 9.1 Lister les politiques OVH

```
GET /me/iam/policy?policyType=builtin
```

Response:
```json
[
  {
    "id": "ovhAccountOwner",
    "name": "ovhAccountOwner",
    "description": "Accès complet au compte (propriétaire)",
    "identitiesCount": 1,
    "actionsCount": null,
    "policyType": "builtin",
    "readOnly": true
  }
]
```

### 9.2 Obtenir les détails d'une politique

```
GET /me/iam/policy/{policyId}
```
