# Cahier des Charges : iam/policies - Création groupe de ressources

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire
- **URL** : /iam/resource-groups/create
- **Objectif** : Créer un nouveau groupe de ressources

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsCard** : Container formulaire
- **OdsInput** : Nom du groupe
- **OdsTextarea** : Description
- **OdsSelect** : Type de produit
- **OdsDatagrid** : Sélection ressources
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Helper |
|-------|------|--------|--------|
| Nom du groupe | Input | Oui * | Nom unique pour identifier le groupe |
| Description | Textarea | Non | - |
| Type de produit | Select | Oui * | Filtrer par type de ressource |
| Ressources | Multi-select | Oui * | Sélectionner les ressources à grouper |

## 5. TABLEAU SÉLECTION RESSOURCES
| Colonne | Type |
|---------|------|
| Checkbox | Sélection |
| Nom | Texte |
| Type | Badge |
| URN | Code |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Créer le groupe | Primary | Soumet formulaire |

## 7. APIS & ENDPOINTS
- POST /iam/resource-groups - Créer groupe
- GET /iam/products - Types de produits
- GET /iam/resources?type={type} - Ressources par type
