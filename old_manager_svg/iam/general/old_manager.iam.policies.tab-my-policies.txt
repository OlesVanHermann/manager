# Cahier des Charges : iam/policies - Onglet Mes politiques

## 1. VUE D'ENSEMBLE
- **Type** : Onglet dans page politiques
- **URL** : /iam/policies?tab=my-policies
- **Objectif** : Afficher les politiques personnalisées

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Tableau politiques
- **OdsBadge** : Type Personnalisé

## 3. ONGLETS
| Onglet | État | Filtre |
|--------|------|--------|
| Mes politiques | Actif | type=custom |
| Politiques OVH | Inactif | type=system |
| Groupes de ressources | Inactif | - |

## 4. CONTENU
- Affiche uniquement les politiques avec type=Personnalisé
- Toutes les actions disponibles (modifier, supprimer)

## 5. APIS & ENDPOINTS
- GET /iam/policies?type=custom - Politiques personnalisées
