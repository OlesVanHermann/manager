## 6. APIS & ENDPOINTS

### 6.1 Obtenir les détails du groupe SSO

```
GET /me/identity/group/{groupId}
```

### 6.2 Modifier le groupe SSO

```
PUT /me/identity/group/{groupId}
```

Body:
```json
{
  "description": "Groupe des développeurs avec accès aux environnements de dev",
  "role": "RESTRICTED_ADMIN"
}
```

### 6.3 Rôles disponibles

| Valeur API | Label FR |
|------------|----------|
| NONE | Aucun |
| READONLY | Lecture seule |
| RESTRICTED_ADMIN | Administrateur restreint |
| ADMIN | Administrateur |
