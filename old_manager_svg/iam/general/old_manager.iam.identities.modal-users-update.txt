## 6. APIS & ENDPOINTS

### 6.1 Obtenir les détails de l'utilisateur

```
GET /me/identity/user/{login}
```

Response:
```json
{
  "login": "developer-1",
  "email": "developer@enterprise.com",
  "group": "developers",
  "description": "Développeur frontend de l'équipe produit",
  "status": "ok",
  "creation": "2025-06-15T10:00:00Z",
  "lastUpdate": "2025-12-01T14:30:00Z"
}
```

### 6.2 Modifier l'utilisateur

```
PUT /me/identity/user/{login}
```

Body:
```json
{
  "email": "developer@enterprise.com",
  "group": "developers",
  "description": "Développeur frontend de l'équipe produit"
}
```

### 6.3 Lister les groupes disponibles

```
GET /me/identity/group
```
