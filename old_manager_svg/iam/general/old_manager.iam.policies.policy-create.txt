## 7. APIS & ENDPOINTS

### 7.1 Créer une politique

```
POST /me/iam/policy
```

Body:
```json
{
  "name": "developers-read-only",
  "description": "Description de la politique",
  "permissionType": "allow",
  "identities": [...],
  "resources": [...],
  "actions": [...],
  "conditions": [...]
}
```
