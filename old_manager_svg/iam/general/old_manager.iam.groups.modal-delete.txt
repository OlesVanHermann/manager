# Cahier des Charges : iam/groups - Modal Delete

- **Type** : Modal confirmation suppression groupe
- **Contexte** : IAM Groups
- **Objectif** : Confirmer la suppression d'un groupe

| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 260px |
