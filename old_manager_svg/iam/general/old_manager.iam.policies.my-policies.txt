## 10. APIS & ENDPOINTS

### 10.1 Lister les politiques utilisateur

```
GET /me/iam/policy
```

Response:
```json
[
  {
    "id": "policy-123",
    "name": "production-access",
    "description": "Accès aux ressources de production",
    "identitiesCount": 8,
    "resourcesCount": 25,
    "actionsCount": 56
  }
]
```

### 10.2 Créer une politique

```
POST /me/iam/policy
```

### 10.3 Recherche avancée

```
GET /me/iam/policy?search={query}&sort={field}&order={asc|desc}
```
