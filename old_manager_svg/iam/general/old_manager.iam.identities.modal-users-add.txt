## 6. APIS & ENDPOINTS

### 6.1 Lister les groupes disponibles

```
GET /me/identity/group
```

Response:
```json
[
  {
    "name": "DEFAULT",
    "description": "Default group"
  },
  {
    "name": "developers",
    "description": "Development team"
  }
]
```

### 6.2 Créer un utilisateur

```
POST /me/identity/user
```

Body:
```json
{
  "login": "new-user",
  "email": "newuser@enterprise.com",
  "password": "SecureP@ss123",
  "group": "DEFAULT",
  "description": "Description optionnelle"
}
```
