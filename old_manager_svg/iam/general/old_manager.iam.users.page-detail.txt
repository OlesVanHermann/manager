# Cahier des Charges : iam/users - Page Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail utilisateur
- **Contexte** : IAM Users
- **Objectif** : Consulter et gérer un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsAvatar** : Photo/initiales utilisateur
- **OdsBadge** : Rôle (Admin)
- **OdsCard** : Cartes infos
- **OdsButton** : Éditer, Désactiver
- **OdsDatagrid** : Activité récente

## 4. HEADER UTILISATEUR
| Propriété | Valeur |

| Nom | 22px bold |
| Email | 13px #6B7280 |

| Bouton | Style |
|--------|-------|

## 9. APIS & ENDPOINTS
- GET /me/identity/user/{id} - Détails utilisateur
- PUT /me/identity/user/{id} - Modifier
- GET /me/identity/user/{id}/group - Groupes
- GET /me/identity/user/{id}/activity - Activité
- POST /me/identity/user/{id}/disable - Désactiver
