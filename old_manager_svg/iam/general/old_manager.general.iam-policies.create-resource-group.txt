# Cahier des Charges : home/iam - Créer Groupe Ressources

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire création groupe
- **URL** : /iam/policies/resource-groups/create
- **Objectif** : Créer groupe de ressources

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre + description
- Formulaire carte blanche
- Footer actions

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur formulaire
- **OdsInput** : Nom groupe, recherche
- **OdsSelect** : Type produit
- **OdsCheckbox** : Sélection ressources
- **OdsButton** : Annuler, Créer

## 4. SECTIONS FORMULAIRE
| Section | Description |
|---------|-------------|
| Nom du groupe | Input texte requis |
| Types de produits | Select catégorie |
| Ressources | Liste avec checkboxes |

## 5. CHAMP NOM
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du groupe de ressources" |
| Requis | Oui (*) |
| Largeur | 400px |
| Placeholder | "mon-groupe-de-ressources" |

## 6. SÉLECTION RESSOURCES
| Propriété | Valeur |
|-----------|--------|
| Container | 832x180px fond #F9FAFB |
| Recherche | Input 300px |
| Liste | Checkboxes avec nom + type |
| Compteur | "X ressources sélectionnées" |

## 7. FORMAT RESSOURCE
| Colonne | Exemple |
|---------|---------|
| Checkbox | ☑/☐ |
| Nom | ns1234567.ip-123-45-67.eu |
| Type | Dedicated Server |

## 8. TYPES PRODUITS
| Type | Description |
|------|-------------|
| dedicatedServer | Serveurs dédiés |
| vps | VPS |
| cloud | Public Cloud |
| domain | Noms de domaine |
| hosting | Hébergement Web |

## 9. FOOTER ACTIONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Créer | Primary | "Créer le groupe de ressources" |

## 10. APIS & ENDPOINTS
- GET /services - Liste toutes ressources
- GET /services?type={type} - Filtrer par type
- POST /me/iam/resourceGroup - Créer groupe
