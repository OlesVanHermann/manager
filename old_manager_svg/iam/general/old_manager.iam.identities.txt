# Cahier des Charges : iam/identities - Liste des identités

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec onglets
- **URL** : /iam/identities
- **Objectif** : Gérer toutes les identités IAM (utilisateurs, groupes, SSO)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Identités
- **OdsTabs** : Utilisateurs / Groupes / SSO
- **OdsButton** : + Ajouter
- **OdsCard** : Container tableau
- **OdsDatagrid** : Tableau identités

## 4. ONGLETS
| Onglet | Badge | URL |
|--------|-------|-----|
| Utilisateurs locaux | (12) | /iam/identities?tab=users |
| Groupes d'utilisateurs | (4) | /iam/identities?tab=groups |
| SSO | - | /iam/identities?tab=sso |

## 5. BOUTON ACTION PRINCIPALE
| Élément | Description |
|---------|-------------|
| Texte | "+ Ajouter" |
| Style | Primary (bleu) |
| Position | Haut droite |
| Action | Dépend de l'onglet actif |

## 6. APIS & ENDPOINTS
- GET /iam/users - Utilisateurs
- GET /iam/groups - Groupes
- GET /iam/sso - Configuration SSO
