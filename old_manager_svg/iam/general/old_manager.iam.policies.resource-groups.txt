## 10. APIS & ENDPOINTS

### 10.1 Lister les groupes de ressources

```
GET /me/iam/resourceGroup
```

Response:
```json
[
  {
    "id": "prod-servers-123",
    "name": "production-servers",
    "resourcesCount": 15,
    "createdAt": "2025-11-15T10:32:45Z",
    "updatedAt": "2025-12-28T14:22:11Z"
  }
]
```

### 10.2 Créer un groupe de ressources

```
POST /me/iam/resourceGroup
```

### 10.3 Supprimer un groupe de ressources

```
DELETE /me/iam/resourceGroup/{resourceGroupId}
```
