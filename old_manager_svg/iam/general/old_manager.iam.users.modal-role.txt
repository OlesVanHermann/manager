# Cahier des Charges : iam/users - Modal Role

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification rôle utilisateur
- **Contexte** : IAM Users
- **Objectif** : Changer le rôle d'un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 390px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsAvatar** : Utilisateur
- **OdsRadio** : Sélection rôle
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Modifier le rôle" |
| Bouton fermer | × en #6B7280 |

## 5. INFO UTILISATEUR
| Propriété | Valeur |
|-----------|--------|
| Avatar | Cercle 40px avec initiales |
| Nom | Nom complet (bold 14px) |
| Email | Email utilisateur |

## 6. SÉLECTION RÔLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Sélectionner un rôle" |
| Type | Radio buttons |

### Carte rôle
| Propriété | Valeur |
|-----------|--------|
| Largeur | 350px |
| Hauteur | 50px |
| Border radius | 4px |

### Carte sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Bordure | 2px #0050D7 |
| Radio | Rempli #0050D7 |

### Carte non sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Fond | white |
| Bordure | #E5E7EB |
| Radio | Vide |

## 7. RÔLES DISPONIBLES
| Rôle | Description |
|------|-------------|
| Viewer | Lecture seule sur toutes les ressources |
| Developer | Accès étendu aux ressources cloud |
| Admin | Accès complet, gestion des utilisateurs |

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Enregistrer | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- PUT /me/identity/user/{userId}/role - Modifier rôle
- GET /me/identity/role - Liste rôles disponibles
