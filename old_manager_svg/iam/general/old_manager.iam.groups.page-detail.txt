# Cahier des Charges : iam/groups - Page Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail groupe
- **Contexte** : IAM Groups
- **Objectif** : Voir et gérer les membres et politiques d'un groupe

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsAvatar** : Initiale groupe et membres
- **OdsButton** : Éditer, Supprimer, Ajouter, Retirer
- **OdsCard** : Cartes membres et politiques

## 9. APIS & ENDPOINTS
- GET /me/identity/group/{groupId} - Détails groupe
- GET /me/identity/group/{groupId}/user - Membres
- GET /me/identity/group/{groupId}/policy - Politiques
- DELETE /me/identity/group/{groupId}/user/{userId} - Retirer membre
- DELETE /me/identity/group/{groupId} - Supprimer groupe
