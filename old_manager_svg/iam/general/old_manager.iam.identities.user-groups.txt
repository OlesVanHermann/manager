## 10. APIS & ENDPOINTS

### 10.1 Lister les groupes d'utilisateurs

```
GET /me/identity/group
```

Response:
```json
[
  {
    "name": "DEFAULT",
    "role": "NONE",
    "description": "Groupe par défaut des utilisateurs",
    "defaultGroup": true,
    "creation": "2024-01-01T00:00:00Z",
    "lastUpdate": "2024-01-01T00:00:00Z"
  },
  {
    "name": "developers",
    "role": "REGULAR",
    "description": "Équipe développement",
    "defaultGroup": false,
    "creation": "2025-03-15T10:00:00Z",
    "lastUpdate": "2025-12-28T14:30:00Z"
  }
]
```

### 10.2 Créer un groupe

```
POST /me/identity/group
```

### 10.3 Supprimer un groupe

```
DELETE /me/identity/group/{group}
```
