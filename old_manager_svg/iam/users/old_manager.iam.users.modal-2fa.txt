# Cahier des Charges : iam/users - Modal 2FA

## 1. VUE D'ENSEMBLE
- **Type** : Modal activation double authentification
- **Contexte** : IAM Users
- **Objectif** : Activer la 2FA via application authenticator

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 440px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Instructions
- **QR Code** : Image générée
- **OdsInput** : Code 6 chiffres
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Activer la double authentification" |
| Bouton fermer | × en #6B7280 |

## 5. ÉTAPE 1 - QR CODE
| Propriété | Valeur |
|-----------|--------|
| Instruction | "1. Scannez ce QR code avec votre application" |
| QR code taille | 200x200px |
| QR background | #1F2937 |
| Info apps | "Google Authenticator, Authy..." |

## 6. ÉTAPE 2 - CODE VÉRIFICATION
| Propriété | Valeur |
|-----------|--------|
| Instruction | "2. Entrez le code de vérification" |
| Nombre inputs | 6 |
| Taille input | 50x50px |
| Police | 18px bold |

### Style input
| État | Bordure |
|------|---------|
| Vide | #E5E7EB |
| Focus | 2px #0050D7 |
| Rempli | #E5E7EB |

## 7. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Activer | Primary #0050D7 | 100px |

## 8. APIS & ENDPOINTS
- POST /me/identity/user/{id}/2fa/setup - Initier setup
- GET /me/identity/user/{id}/2fa/qrcode - QR code
- POST /me/identity/user/{id}/2fa/verify - Vérifier code
- DELETE /me/identity/user/{id}/2fa - Désactiver 2FA
