# Cahier des Charges : iam/users - Modal Delete

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation suppression utilisateur
- **Contexte** : IAM Users
- **Objectif** : Confirmer la suppression d'un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 270px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Question et infos
- **OdsAvatar** : Utilisateur
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #FEE2E2 (danger) |
| Titre | "Supprimer l'utilisateur" |
| Couleur titre | #DC2626 |
| Bouton fermer | × en #DC2626 |

## 5. QUESTION CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Êtes-vous sûr de vouloir supprimer cet utilisateur ?" |

## 6. INFO UTILISATEUR
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Hauteur | 50px |
| Avatar | Cercle 30px avec initiales |
| Nom | Nom complet (bold) |
| Email | Email utilisateur |

## 7. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠️ |
| Ligne 1 | "Cette action est irréversible. L'utilisateur perdra" |
| Ligne 2 | "immédiatement l'accès à toutes les ressources." |

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Supprimer | Danger #DC2626 | 100px |

## 9. APIS & ENDPOINTS
- DELETE /me/identity/user/{userId} - Supprimer utilisateur
