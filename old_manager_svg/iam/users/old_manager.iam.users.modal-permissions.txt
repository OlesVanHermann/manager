# Cahier des Charges : iam/users - Modal Permissions

## 1. VUE D'ENSEMBLE
- **Type** : Modal édition permissions utilisateur
- **Contexte** : IAM Users
- **Objectif** : Gérer groupes et politiques d'un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 510px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et sections
- **OdsCheckbox** : Sélection groupes/politiques
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Modifier les permissions - {Nom}" |
| Bouton fermer | × en #6B7280 |

## 5. SECTION GROUPES
| Propriété | Valeur |
|-----------|--------|
| Titre | "Groupes" |
| Liste | Checkboxes avec noms groupes |

### Groupes exemples
| Groupe | État |
|--------|------|
| Administrators | Coché |
| Developers | Coché |
| Support | Non coché |
| Billing | Non coché |

## 6. SECTION POLITIQUES INDIVIDUELLES
| Propriété | Valeur |
|-----------|--------|
| Titre | "Politiques individuelles" |
| Liste | Checkboxes avec noms + source |

### Indication source
| Élément | Description |
|---------|-------------|
| Politique héritée | "(via {NomGroupe})" en #6B7280 |
| Politique directe | Pas d'indication |

### Politiques exemples
| Politique | Source |
|-----------|--------|
| FullAdminAccess | (via Administrators) |
| ComputeReadOnly | (via Developers) |
| BillingViewer | - |
| DNSManagement | - |

## 7. SECTION OPTIONS COMPTE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Options du compte" |
| Disposition | 2 colonnes |

### Options
| Option | Description |
|--------|-------------|
| Accès Manager | Accès interface web |
| Accès API | Accès programmatique |

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Enregistrer | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- GET /me/identity/user/{id}/group - Groupes actuels
- PUT /me/identity/user/{id}/group - Modifier groupes
- GET /me/identity/user/{id}/policy - Politiques
- PUT /me/identity/user/{id}/policy - Modifier politiques
