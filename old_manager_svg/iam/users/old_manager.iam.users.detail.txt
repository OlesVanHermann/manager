# Cahier des Charges : iam/users - Détail utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/users/{userId}
- **Objectif** : Afficher et gérer un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Accueil / IAM / Utilisateurs / {nom}
- **OdsAvatar** : Initiales utilisateur
- **OdsBadge** : Statut
- **OdsCard** : Cards informations
- **OdsButton** : Modifier, Désactiver, Supprimer

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Avatar | Initiales colorées |
| Titre | Nom complet |
| Badge | Statut (Actif/Désactivé) |
| Actions | Modifier, Reset MDP, Désactiver, Supprimer |

## 5. CARD INFORMATIONS PERSONNELLES
| Champ | Type |
|-------|------|
| Email | Texte |
| Login | Code |
| Description | Texte |
| Date création | Datetime |
| Dernière connexion | Datetime relatif |

## 6. CARD GROUPES
| Élément | Description |
|---------|-------------|
| Liste | Chips des groupes |
| Action | + Ajouter à un groupe |

## 7. CARD POLITIQUES
| Élément | Description |
|---------|-------------|
| Liste | Politiques directes + via groupes |
| Indication | Source (Direct/Groupe X) |

## 8. CARD SESSIONS ACTIVES
| Colonne | Type |
|---------|------|
| Appareil | Texte |
| IP | Code |
| Dernière activité | Datetime |
| Action | Révoquer |

## 9. APIS & ENDPOINTS
- GET /iam/users/{userId} - Détail utilisateur
- PUT /iam/users/{userId} - Modifier
- POST /iam/users/{userId}/disable - Désactiver
- DELETE /iam/users/{userId} - Supprimer
- GET /iam/users/{userId}/sessions - Sessions actives
