# Cahier des Charges : iam/users - Modal Add

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout utilisateur
- **Contexte** : IAM Users
- **Objectif** : Inviter un nouvel utilisateur au compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 460px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : Email, prénom, nom
- **OdsSelect** : Groupe
- **OdsCheckbox** : Permissions
- **OdsButton** : Annuler, Inviter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Ajouter un utilisateur" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT EMAIL
| Propriété | Valeur |
|-----------|--------|
| Label | "Email" |
| Largeur | 450px |
| Placeholder | "utilisateur@company.com" |

## 6. INPUTS IDENTITÉ (côte à côte)
| Champ | Largeur | Placeholder |
|-------|---------|-------------|
| Prénom | 215px | "Jean" |
| Nom | 215px | "Dupont" |

## 7. SELECT GROUPE
| Propriété | Valeur |
|-----------|--------|
| Label | "Groupe" |
| Largeur | 450px |
| Valeur exemple | "Developers" |

## 8. CHECKBOXES PERMISSIONS
| Permission | État défaut |
|------------|-------------|
| Accès Manager | ✓ Coché |
| Accès API | ✓ Coché |
| Droits administrateur | Non coché |

## 9. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Inviter | Primary #0050D7 | 100px |

## 10. APIS & ENDPOINTS
- POST /me/identity/user - Inviter utilisateur
- GET /me/identity/group - Liste groupes (pour select)
