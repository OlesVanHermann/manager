# Cahier des Charges : iam/secret-manager - Création secret

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire
- **URL** : /iam/secret-manager/create
- **Objectif** : Créer un nouveau secret

## 2. COMPOSANTS UI
- **OdsCard** : Container formulaire
- **OdsInput** : Nom
- **OdsSelect** : Type
- **OdsTextarea** : Description, Valeur
- **OdsButton** : Annuler, Créer

## 3. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |
| Type | Select | Oui * |
| Description | Textarea | Non |
| Valeur du secret | Textarea (masqué) | Oui * |

## 4. TYPES
| Type | Description |
|------|-------------|
| API Key | Clé d'API |
| Password | Mot de passe |
| Certificate | Certificat PEM |
| Generic | Valeur libre |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Créer | Primary | Crée le secret |

## 6. APIS & ENDPOINTS
- POST /secret-manager/secrets - Créer secret
