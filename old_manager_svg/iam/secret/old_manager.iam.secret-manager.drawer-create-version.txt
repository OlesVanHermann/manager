# Cahier des Charges : iam/secret-manager - Drawer nouvelle version

## 1. VUE D'ENSEMBLE
- **Type** : Drawer (panneau latéral)
- **Contexte** : Création d'une nouvelle version
- **Objectif** : Ajouter une nouvelle valeur au secret

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Position | Droite |

## 3. COMPOSANTS UI
- **OdsDrawer** : Container principal
- **OdsTextarea** : Nouvelle valeur
- **OdsToggle** : Activer immédiatement
- **OdsButton** : Annuler, Créer

## 4. HEADER
| Élément | Description |
|---------|-------------|
| Titre | "Nouvelle version" |
| Sous-titre | "Secret: {nom}" |

## 5. CHAMPS
| Champ | Type | Description |
|-------|------|-------------|
| Nouvelle valeur | Textarea (masqué) | Requis |
| Activer immédiatement | Toggle | Défaut: Oui |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme drawer |
| Créer | Primary | Crée version |

## 7. APIS & ENDPOINTS
- POST /secret-manager/secrets/{secretId}/versions
