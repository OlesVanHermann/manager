# Cahier des Charges : iam/api-keys - Modal création clé API

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Accessible depuis liste clés API
- **Objectif** : Créer une nouvelle clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 700px |
| Modal hauteur | 620px |
| Position | Centrée verticalement |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Info box (bleu)
- **OdsInput** : Nom de la clé
- **OdsTextarea** : Description
- **OdsSelect** : Durée de validité
- **OdsSelect** : Politiques
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Créer une clé API" |
| Bouton fermer | X en haut droite |

## 5. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Information (bleu) |
| Background | #EFF6FF |
| Icône | i (cercle bleu) |
| Texte | "Le secret de la clé ne sera affiché qu'une seule fois après création." |

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Requis | Helper |
|-------|------|--------|--------|
| Nom de la clé API | Input texte | Oui * | "Utilisez un nom descriptif pour identifier cette clé." |
| Description | Textarea | Non | - |
| Durée de validité | Select | Oui | "La clé expirera automatiquement après cette durée." |
| Politiques | Select multi | Non | "Associez des politiques pour définir les permissions de cette clé." |

## 7. OPTIONS DURÉE DE VALIDITÉ
| Option | Valeur |
|--------|--------|
| 1 mois | 1 month |
| 3 mois | 3 months |
| 6 mois | 6 months (défaut) |
| 1 an | 1 year |
| Sans expiration | never |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Soumet formulaire |

## 9. VALIDATION
| Champ | Règle |
|-------|-------|
| Nom | Requis, 3-64 caractères |
| Description | Max 256 caractères |

## 10. APIS & ENDPOINTS
- POST /iam/api-keys - Créer clé API
- GET /iam/policies - Liste politiques disponibles
