# Cahier des Charges : iam/conditions - Liste des conditions

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **URL** : /iam/conditions
- **Objectif** : Gérer les conditions d'accès IAM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Conditions
- **OdsButton** : + Créer une condition
- **OdsDatagrid** : Tableau conditions
- **OdsBadge** : Type

## 4. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien | 250px |
| Type | Badge | 150px |
| Description | Texte | 300px |
| Politiques | Badge compteur | 120px |
| Actions | Menu ⋮ | 50px |

## 5. TYPES DE CONDITIONS
| Type | Description |
|------|-------------|
| IP | Restriction par adresse IP |
| Time | Restriction horaire |
| MFA | Exigence MFA |
| Region | Restriction géographique |

## 6. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Éditer la condition |
| Supprimer | Supprimer la condition |

## 7. APIS & ENDPOINTS
- GET /iam/conditions - Liste conditions
- POST /iam/conditions - Créer condition
- PUT /iam/conditions/{condId} - Modifier
- DELETE /iam/conditions/{condId} - Supprimer
