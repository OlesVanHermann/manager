# Cahier des Charges : iam/apikeys - Modal Revoke

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation révocation clé API
- **Contexte** : IAM API Keys
- **Objectif** : Confirmer la révocation d'une clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 240px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Message confirmation
- **OdsMessage** : Avertissement irréversibilité
- **OdsButton** : Annuler, Révoquer

## 4. HEADER MODAL (DANGER)
| Propriété | Valeur |
|-----------|--------|
| Background | #FEE2E2 |
| Titre | "Révoquer la clé API" |
| Couleur titre | #DC2626 |
| Bouton fermer | × en #DC2626 |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir révoquer cette clé API ?" |
| Info clé | Nom + clé tronquée (fond gris) |
| Message warning | Icône ⚠️ + texte irréversibilité |

## 6. INFO CLÉ À RÉVOQUER
| Propriété | Valeur |
|-----------|--------|
| Largeur | 350px |
| Hauteur | 40px |
| Background | #F9FAFB |
| Contenu | Nom (bold) + clé tronquée (monospace) |

## 7. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠️ |
| Texte | "Cette action est irréversible. Les applications utilisant cette clé ne pourront plus accéder à l'API." |

## 8. BOUTONS FOOTER
| Bouton | Style | Couleur |
|--------|-------|---------|
| Annuler | Outline | #E5E7EB |
| Révoquer | Danger solid | #DC2626 |

## 9. APIS & ENDPOINTS
- DELETE /me/api/credential/{credentialId} - Révoquer clé
