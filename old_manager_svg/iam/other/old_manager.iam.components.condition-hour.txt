## 6. APIS & ENDPOINTS

### 6.1 Ajouter une condition heure

```
POST /me/iam/policy/{policyId}/condition
```

Body:
```json
{
  "type": "hour",
  "operator": "gte",
  "values": ["08:00"],
  "timezone": "Europe/Paris"
}
```

### 6.2 Opérateurs heure

| Valeur API | Label FR |
|------------|----------|
| gte | Supérieur ou égal à |
| lte | Inférieur ou égal à |
| between | Entre |
| eq | Égal à |
