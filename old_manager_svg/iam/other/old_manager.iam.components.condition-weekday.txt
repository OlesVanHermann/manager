## 6. APIS & ENDPOINTS

### 6.1 Ajouter une condition jour de la semaine

```
POST /me/iam/policy/{policyId}/condition
```

Body:
```json
{
  "type": "dayOfWeek",
  "operator": "in",
  "values": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "timezone": "Europe/Paris"
}
```

### 6.2 Jours disponibles

| Valeur API | Label FR |
|------------|----------|

| tuesday | Mardi |
| wednesday | Mercredi |
