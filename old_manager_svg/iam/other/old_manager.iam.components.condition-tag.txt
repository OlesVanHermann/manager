## 6. APIS & ENDPOINTS

### 6.1 Ajouter une condition tag

```
POST /me/iam/policy/{policyId}/condition
```

Body:
```json
{
  "type": "tag",
  "operator": "eq",
  "key": "environment",
  "value": "production"
}
```

### 6.2 Opérateurs disponibles

| Valeur API | Label FR |
|------------|----------|
| eq | Égal à |
| neq | Non égal à |
| exists | Existe |
| notExists | N'existe pas |

---

## 7. VALIDATION

| Champ | Règle |
|-------|-------|
| Tag | Obligatoire, alphanumeric et tirets |
| Valeur | Obligatoire si opérateur = eq ou neq |
