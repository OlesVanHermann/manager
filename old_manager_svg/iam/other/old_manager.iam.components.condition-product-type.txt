## 6. APIS & ENDPOINTS

### 6.1 Ajouter une condition type de produit

```
POST /me/iam/policy/{policyId}/condition
```

Body:
```json
{
  "type": "productType",
  "operator": "in",
  "values": ["cloudDB", "hosting"]
}
```

### 6.2 Lister les types de produits

```
GET /me/iam/resource/productType
```

Response:
```json
[
  "cloudDB",
  "dedicatedCloud",
  "domain",
  "hosting",
  "ip",
  "license",
  "vps"
]
```
