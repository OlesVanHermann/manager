## 7. APIS & ENDPOINTS

### 7.1 Créer un groupe de ressources

```
POST /me/iam/resourceGroup
```

Body:
```json
{
  "name": "groupe-production",
  "resources": [
    "urn:v1:eu:resource:hosting:production-website.ovh.net",
    "urn:v1:eu:resource:cloudDB:db-prod-001.clouddb.ovh.net"
  ]
}
```

### 7.2 Lister les types de produits

```
GET /me/iam/resourceGroup/productType
```

### 7.3 Rechercher des ressources

```
GET /me/iam/resource?productType=hosting&search=production
```
