## 8. APIS & ENDPOINTS

### 8.1 Lister les utilisateurs

```
GET /me/identity/user
```

### 8.2 Lister les groupes

```
GET /me/identity/group
```

### 8.3 Lister les comptes de service

```
GET /me/api/credential
```

### 8.4 Format URN identité

```
urn:v1:eu:identity:{type}:{account}/{identifier}
```

| Type | Description |
|------|-------------|
| user | Utilisateur local |
| group | Groupe d'utilisateurs |
| account | Compte OVHcloud |
