# Cahier des Charges : iam/api-keys - Modal suppression clé API

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec saisie
- **Contexte** : Action menu sur une clé API
- **Objectif** : Supprimer définitivement une clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 380px |
| Position | Centrée |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning box (orange)
- **OdsInput** : Champ confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer une clé API" |
| Bouton fermer | X en haut droite |

## 5. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Background | #FEF3C7 |
| Icône | ! (cercle orange) |

### Messages d'avertissement
| Ligne | Texte |
|-------|-------|
| 1 | "Avertissement : cette action supprimera la clé API et" |
| 2 | "révoquera tous les certificats associés à cette clé API." |

## 6. INFORMATIONS
| Élément | Description |
|---------|-------------|
| Label | "Suppression de la clé API :" |
| Nom | {nom de la clé} (gras) |

## 7. CHAMP CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Veuillez entrer le mot \"TERMINATE\" pour confirmer la suppression" |
| Placeholder | TERMINATE |
| Validation | Doit être exactement "TERMINATE" |

## 8. FOOTER MODAL
| Bouton | Style | Action | Condition |
|--------|-------|--------|-----------|
| Annuler | Secondary | Ferme modal | - |
| Supprimer | Danger (rouge) | Supprime la clé | Actif si TERMINATE saisi |

## 9. APIS & ENDPOINTS
- DELETE /iam/api-keys/{keyId} - Supprimer clé API
