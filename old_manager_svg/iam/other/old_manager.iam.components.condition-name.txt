## 6. APIS & ENDPOINTS

### 6.1 Ajouter une condition de nom

```
POST /me/iam/policy/{policyId}/condition
```

Body:
```json
{
  "type": "name",
  "operator": "startsWith",
  "value": "prod-"
}
```
