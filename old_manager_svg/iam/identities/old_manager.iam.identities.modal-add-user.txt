# Cahier des Charges : iam/identities - Modal ajout utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Onglet Utilisateurs
- **Objectif** : Créer un nouvel utilisateur local

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs texte
- **OdsSelect** : Groupes
- **OdsToggle** : Statut actif
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Prénom | Input | Oui * |
| Nom | Input | Oui * |
| Email | Input email | Oui * |
| Login | Input | Oui * (généré auto) |
| Description | Textarea | Non |
| Groupes | Multi-select | Non |
| Actif | Toggle | Oui (défaut: actif) |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée utilisateur |

## 6. POST-CRÉATION
- Email envoyé avec lien de création de mot de passe
- Utilisateur apparaît dans la liste

## 7. APIS & ENDPOINTS
- POST /iam/users - Créer utilisateur
- GET /iam/groups - Liste groupes disponibles
