# Cahier des Charges : iam/identities - Onglet Groupes utilisateurs

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau
- **URL** : /iam/identities?tab=groups
- **Objectif** : Gérer les groupes d'utilisateurs IAM

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsSearchField** : Recherche groupe
- **OdsDatagrid** : Tableau groupes
- **OdsBadge** : Compteurs

## 3. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Groupe | Icône + Nom | 326px |
| Description | Texte | 350px |
| Membres | Badge compteur | 150px |
| Politiques | Badge compteur | 150px |
| Créé le | Date | 150px |
| Actions | Menu ⋮ | 50px |

## 4. ICÔNE GROUPE
| Propriété | Valeur |
|-----------|--------|
| Taille | 32x32px |
| Border radius | 6px |
| Texte | Initiale du groupe |
| Couleur | Dynamique par groupe |

## 5. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Éditer le groupe |
| Gérer les membres | Ajouter/retirer utilisateurs |
| Gérer les politiques | Attacher/détacher politiques |
| Supprimer | Supprimer le groupe |

## 6. APIS & ENDPOINTS
- GET /iam/groups - Liste groupes
- POST /iam/groups - Créer groupe
