# Cahier des Charges : iam/identities - Modal activation utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu utilisateur désactivé
- **Objectif** : Réactiver un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsButton** : Annuler, Activer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Activer l'utilisateur" |
| Message | "L'utilisateur {nom} pourra de nouveau se connecter." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Activer | Primary | Active utilisateur |

## 6. APIS & ENDPOINTS
- POST /iam/users/{userId}/enable - Activer utilisateur
