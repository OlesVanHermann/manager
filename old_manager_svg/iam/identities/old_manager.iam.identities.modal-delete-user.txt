# Cahier des Charges : iam/identities - Modal suppression utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu utilisateur
- **Objectif** : Supprimer un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Supprimer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir supprimer l'utilisateur {nom} ?" |
| Warning | "Cette action est irréversible." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime utilisateur |

## 6. APIS & ENDPOINTS
- DELETE /iam/users/{userId} - Supprimer utilisateur
