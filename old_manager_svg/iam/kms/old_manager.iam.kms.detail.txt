# Cahier des Charges : iam/kms - Détail service KMS

## 1. VUE D'ENSEMBLE
- **Type** : Page détail avec onglets
- **URL** : /iam/kms/{kmsId}
- **Objectif** : Gérer un service KMS spécifique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / KMS / {nom}
- **OdsBadge** : Statut
- **OdsTabs** : Clés de service / Credentials
- **OdsCard** : Cards informations
- **OdsButton** : Actions

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | Nom du service KMS |
| Badge | Statut |
| Actions | Renommer, Résilier |

## 5. CARD INFORMATIONS
| Champ | Type |
|-------|------|
| ID du service | Code |
| Région | Texte |
| Date de création | Datetime |
| Endpoint | URL copiable |

## 6. ONGLETS
| Onglet | Contenu |
|--------|---------|
| Clés de service | Liste des clés cryptographiques |
| Credentials | Identifiants d'accès |

## 7. APIS & ENDPOINTS
- GET /kms/{kmsId} - Détail service
- PUT /kms/{kmsId} - Modifier service
- DELETE /kms/{kmsId}/terminate - Résilier
