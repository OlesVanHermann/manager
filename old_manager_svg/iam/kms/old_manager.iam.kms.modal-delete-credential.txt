# Cahier des Charges : iam/kms - Modal suppression credential

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action sur un credential
- **Objectif** : Supprimer un credential

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Supprimer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer le credential" |
| Warning | "Cette action est irréversible." |
| Info | "Toutes les applications utilisant ce credential perdront l'accès au KMS." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime credential |

## 6. APIS & ENDPOINTS
- DELETE /kms/{kmsId}/credentials/{credId}
