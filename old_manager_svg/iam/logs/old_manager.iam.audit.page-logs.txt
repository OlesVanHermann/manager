# Cahier des Charges : iam/audit - Page Logs

## 1. VUE D'ENSEMBLE
- **Type** : Page journal d'audit
- **Contexte** : IAM Audit
- **Objectif** : Visualiser l'historique des actions

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsInput** : Recherche
- **OdsSelect** : Filtres (utilisateur, action, période)
- **OdsButton** : Exporter
- **OdsDatagrid** : Liste événements
- **OdsBadge** : Action, résultat

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Journal d'audit" |
| Description | "Historique des actions et connexions." |

## 5. BARRE FILTRES
| Élément | Largeur | Contenu |
|---------|---------|---------|
| Recherche | 250px | Input texte |
| Utilisateurs | 150px | Select |
| Actions | 150px | Select |
| Période | 150px | Select (7j, 30j, 90j) |
| Exporter | 100px | Bouton outline |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Date/Heure | 160px | Timestamp |
| Utilisateur | 170px | Email ou API key |
| Action | 200px | Badge action |
| Ressource | 230px | Nom ressource |
| IP Source | 140px | Adresse IP |
| Résultat | 140px | Badge succès/échec |

## 7. BADGES ACTION
| Action | Fond | Texte |
|--------|------|-------|
| LOGIN | #DBEAFE | #1E40AF |
| CREATE | #DCFCE7 | #16A34A |
| UPDATE | #FEF3C7 | #92400E |
| DELETE | #FEE2E2 | #DC2626 |

## 8. BADGES RÉSULTAT
| Résultat | Fond | Texte |
|----------|------|-------|
| Succès | #DCFCE7 | #16A34A |
| Échec | #FEE2E2 | #DC2626 |

## 9. EXEMPLES ÉVÉNEMENTS
| Utilisateur | Action | Ressource | Résultat |
|-------------|--------|-----------|----------|
| john.doe@... | LOGIN | - | Succès |
| alice.martin@... | UPDATE | instance-web-01 | Succès |
| api-key:ci-cd | CREATE | instance-app-03 | Succès |
| unknown | LOGIN | - | Échec |
| pierre.dupont@... | DELETE | volume-backup-old | Succès |

## 10. APIS & ENDPOINTS
- GET /me/api/logs - Logs d'audit
- GET /me/api/logs?action={action} - Filtrer par action
- GET /me/api/logs?user={user} - Filtrer par utilisateur
