# Cahier des Charges : iam/logs - Logs d'audit IAM

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec filtres avancés
- **URL** : /iam/logs
- **Objectif** : Consulter les logs d'audit des actions IAM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Logs d'audit
- **OdsDateRange** : Sélecteur période
- **OdsSelect** : Filtres
- **OdsDatagrid** : Tableau logs
- **OdsBadge** : Type d'action

## 4. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Période | Date range | Dernières 24h, 7j, 30j, Personnalisé |
| Type d'action | Select | Tous, Connexion, Modification, Suppression |
| Utilisateur | Select/Input | Liste ou recherche |
| Ressource | Select | Type de ressource |

## 5. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Date/Heure | Datetime | 180px |
| Action | Badge | 150px |
| Utilisateur | Lien | 200px |
| Ressource | Texte | 250px |
| IP Source | Code | 150px |
| Statut | Badge | 100px |
| Actions | Icône | 50px |

## 6. BADGES ACTION
| Action | Couleur |
|--------|---------|
| Connexion | Vert |
| Création | Bleu |
| Modification | Orange |
| Suppression | Rouge |
| Consultation | Gris |

## 7. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Succès | Vert |
| Échec | Rouge |

## 8. APIS & ENDPOINTS
- GET /iam/logs - Liste logs avec filtres
- GET /iam/logs/{logId} - Détail log
