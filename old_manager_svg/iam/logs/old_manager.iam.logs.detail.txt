# Cahier des Charges : iam/logs - Détail log

## 1. VUE D'ENSEMBLE
- **Type** : Drawer ou Modal détail
- **Contexte** : Clic sur icône détail dans tableau
- **Objectif** : Afficher tous les détails d'un événement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Position | Droite (drawer) |

## 3. COMPOSANTS UI
- **OdsDrawer** : Container principal
- **OdsBadge** : Action, Statut
- **OdsCode** : Payload JSON

## 4. INFORMATIONS AFFICHÉES
| Champ | Type |
|-------|------|
| ID événement | Code |
| Date/Heure | Datetime précis |
| Action | Badge + description |
| Utilisateur | Lien vers profil |
| Email | Texte |
| IP Source | Code |
| User Agent | Texte |
| Ressource | URN |
| Statut | Badge |
| Message | Texte (si erreur) |
| Payload | JSON formaté |

## 5. ACTIONS
| Bouton | Description |
|--------|-------------|
| Copier ID | Copie ID événement |
| Fermer | Ferme drawer |

## 6. APIS & ENDPOINTS
- GET /iam/logs/{logId} - Détail complet
