# Cahier des Charges : iam/secret-manager - Drawer valeur secret

## 1. VUE D'ENSEMBLE
- **Type** : Drawer (panneau latéral)
- **Contexte** : Visualisation sécurisée de la valeur
- **Objectif** : Afficher la valeur d'une version de secret

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Position | Droite |

## 3. COMPOSANTS UI
- **OdsDrawer** : Container principal
- **OdsButton** : Copier, Fermer
- **OdsToggle** : Afficher/Masquer valeur

## 4. HEADER
| Élément | Description |
|---------|-------------|
| Titre | "Valeur du secret" |
| Sous-titre | "Version {n}" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Toggle | Afficher la valeur |
| Valeur | Masquée par défaut (•••••) |
| Bouton | Copier dans le presse-papier |

## 6. SÉCURITÉ
- Valeur masquée par défaut
- Copie sans affichage possible
- Fermeture auto après 60s

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary | Ferme drawer |

## 8. APIS & ENDPOINTS
- GET /secret-manager/secrets/{secretId}/versions/{versionId}/value
