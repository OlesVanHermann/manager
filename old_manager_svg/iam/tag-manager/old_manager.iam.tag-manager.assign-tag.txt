# Cahier des Charges : iam/tag-manager - Assigner tag

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'assignation
- **Contexte** : Depuis page détail ressource ou tag
- **Objectif** : Assigner un tag à une ressource

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection tag existant
- **OdsInput** : Ou créer nouveau tag
- **OdsButton** : Annuler, Assigner

## 4. OPTIONS
| Option | Description |
|--------|-------------|
| Tag existant | Sélectionner dans la liste |
| Nouveau tag | Créer un nouveau tag |

## 5. CHAMPS NOUVEAU TAG
| Champ | Type | Requis |
|-------|------|--------|
| Clé | Input | Oui |
| Valeur | Input | Oui |
| Couleur | Color picker | Non |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Assigner | Primary | Assigne tag |

## 7. APIS & ENDPOINTS
- GET /iam/tags - Liste tags
- POST /iam/tags/{tagId}/resources - Assigner tag
- POST /iam/tags - Créer et assigner
