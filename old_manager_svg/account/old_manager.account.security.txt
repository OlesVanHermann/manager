# Cahier des Charges : account - Sécurité du compte

## 1. VUE D'ENSEMBLE
- **Page** : Sécurité
- **Type** : Dashboard paramètres sécurité
- **Objectif** : Gérer la sécurité du compte OVHcloud

## 2. STRUCTURE PAGE
- Header avec titre
- Section mot de passe
- Section double authentification (2FA)
- Section sessions actives
- Section restrictions IP

## 3. COMPOSANTS UI
- **OdsCard** : Conteneurs sections
- **OdsButton** : Actions
- **OdsBadge** : Statuts sécurité
- **OdsToggle** : Activation options
- **OdsDatagrid** : Sessions, IPs

## 4. SECTION MOT DE PASSE
| Élément | Type | Description |
|---------|------|-------------|
| Dernière modification | Text | Date dernier changement |
| Force du MDP | Badge | Faible/Moyen/Fort |
| Bouton Modifier | Button | Ouvre modal changement |

## 5. SECTION 2FA
| Méthode | Statut | Actions |
|---------|--------|---------|
| SMS | Badge Actif/Inactif | Configurer, Désactiver |
| TOTP (App) | Badge Actif/Inactif | Configurer, Désactiver |
| Clé de sécurité | Badge Actif/Inactif | Ajouter, Supprimer |
| Codes backup | Text | X codes restants, Régénérer |

## 6. SECTION SESSIONS
| Colonne | Type | Description |
|---------|------|-------------|
| Appareil | Text | Navigateur/OS |
| IP | Text | Adresse IP |
| Localisation | Text | Ville, Pays |
| Dernière activité | Date | Horodatage |
| Actions | Button | Révoquer |

## 7. SECTION RESTRICTIONS IP
| Colonne | Type | Description |
|---------|------|-------------|
| IP/Plage | Text | Adresse ou CIDR |
| Description | Text | Note utilisateur |
| Statut | Badge | Actif/Inactif |
| Actions | ActionMenu | Modifier, Supprimer |

## 8. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Actif | Vert |
| Inactif | Gris |
| Non configuré | Orange |
| Fort | Vert |
| Moyen | Orange |
| Faible | Rouge |

## 9. APIS & ENDPOINTS
- GET /me/security - Infos sécurité
- PUT /me/changePassword - Changer MDP
- GET /me/accessRestriction/sms - 2FA SMS
- POST /me/accessRestriction/sms - Activer SMS
- DELETE /me/accessRestriction/sms/{id} - Désactiver SMS
- GET /me/accessRestriction/totp - 2FA TOTP
- POST /me/accessRestriction/totp - Activer TOTP
- GET /me/accessRestriction/backupCode - Codes backup
- POST /me/accessRestriction/backupCode - Régénérer codes
- GET /me/accessRestriction/ip - Restrictions IP
- POST /me/accessRestriction/ip - Ajouter restriction
- DELETE /me/accessRestriction/ip/{id} - Supprimer restriction
