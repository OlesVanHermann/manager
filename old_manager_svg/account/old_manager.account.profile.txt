# Cahier des Charges : account - Profil utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Mon profil
- **Type** : Formulaire multi-sections
- **Objectif** : Consulter et modifier les informations personnelles

## 2. STRUCTURE PAGE
- Header avec titre
- Section informations personnelles
- Section adresse
- Section entreprise (si applicable)
- Section préférences

## 3. COMPOSANTS UI
- **OdsCard** : Conteneurs sections
- **OdsInput** : Champs texte
- **OdsSelect** : Pays, civilité
- **OdsButton** : Modifier, Sauvegarder
- **OdsMessage** : Alertes validation

## 4. SECTION INFORMATIONS PERSONNELLES
| Champ | Type | Modifiable | Requis |
|-------|------|------------|--------|
| Civilité | Select | Oui | Oui |
| Prénom | Input | Oui | Oui |
| Nom | Input | Oui | Oui |
| Email principal | Input | Non | Oui |
| Téléphone | Input | Oui | Non |
| Date de naissance | Date | Non | - |
| NIC Handle | Text | Non | - |

## 5. SECTION ADRESSE
| Champ | Type | Modifiable | Requis |
|-------|------|------------|--------|

| Bouton | Style | Action |
|--------|-------|--------|

## 9. APIS & ENDPOINTS
- GET /me - Informations profil
- PUT /me - Modifier profil
- GET /me/contact - Infos contact
- PUT /me/contact - Modifier contact
