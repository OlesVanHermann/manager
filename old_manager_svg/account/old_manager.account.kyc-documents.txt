# Cahier des Charges : account - Documents KYC

## 1. VUE D'ENSEMBLE
- **Type** : Page de téléversement de documents
- **Contexte** : Validation de compte
- **Objectif** : Permettre l'envoi de documents justificatifs pour valider un compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 900px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, description, mentions légales
- **OdsMessage** : Bannière d'information (bleu)
- **OdsLink** : Contacter le support
- **OdsDropzone** : Zone de glisser-déposer
- **OdsButton** : Parcourir, Envoyer les documents

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Documents justificatifs" |
| Description | "Pour valider votre compte, veuillez fournir les documents demandés." |

## 5. BANNIÈRE D'INFORMATION
| Type | Contenu |
|------|---------|
| Info (bleu) | "Votre dossier est en cours d'examen. Pour toute question, contactez le support." |
| Lien | "Contacter le support" |

## 6. SECTION DOCUMENTS REQUIS

### Titre
| Texte |
|-------|
| "Documents requis" |

### Liste des documents (particulier)
| Document | Obligatoire |
|----------|-------------|
| Pièce d'identité (recto/verso) | Oui |
| Justificatif de domicile de moins de 3 mois | Oui |
| RIB | Non (optionnel) |

## 7. ZONE DE TÉLÉVERSEMENT

### Contraintes
| Propriété | Valeur |
|-----------|--------|
| Taille max par fichier | 5 Mo |
| Nombre max de documents | 10 |
| Formats acceptés | pdf, jpeg, png, jpg |

### Zone de dépôt
| Élément | Description |
|---------|-------------|
| Style | Bordure pointillée, fond blanc |
| Texte principal | "Glissez-déposez vos fichiers" |
| Texte secondaire | "ou" |
| Bouton | "Parcourir" (bleu) |
