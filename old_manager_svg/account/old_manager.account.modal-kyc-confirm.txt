# Cahier des Charges : account - Modal confirmation envoi KYC

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page documents KYC
- **Objectif** : Confirmer l'envoi des documents justificatifs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 140px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Envoyer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Confirmer l'envoi" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir envoyer ces documents ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Envoyer | Primary (bleu) | Envoie les documents |

## 7. APIS & ENDPOINTS
- POST /me/document - Soumettre les documents
