# Cahier des Charges : network/security - Modal Configure Alerting

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Configurer les alertes de sécurité réseau
**Contexte:** Monitoring sécurité

## 2. DIMENSIONS

- Modal: 450x320px
- Overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Titre | "Configurer les alertes" (18px, font-weight: 600) |
| Bouton fermer | × (20px, #6B7280) |

## 4. CONTENU

### Description
- "Configurez les notifications pour être alerté en cas d'événement de sécurité sur votre infrastructure."

### Toggle notifications
| Élément | Style |
|---------|-------|
| Label | "Notifications par email" (font-weight: 500) |
| Toggle | Switch ON (#0050D7) |

### Champ email
| Élément | Style |
|---------|-------|
| Label | "Adresse email" (font-weight: 500) |
| Input | 402x40px |
| Valeur | "admin@example.com" |

### Seuil d'alerte
| Élément | Style |
|---------|-------|
| Label | "Seuil d'alerte (paquets/sec)" (font-weight: 500) |
| Helper | "Déclenche une alerte si le trafic dépasse ce seuil." (12px) |
| Input | 200x40px |
| Valeur | "10000" |

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Ghost (#0050D7 text) | Fermer modal |
| Enregistrer | Primary (#0050D7) | PUT API |

## 6. APIS & ENDPOINTS

```
GET /ip/{ip}/mitigation/alert
PUT /ip/{ip}/mitigation/alert
{
  "emailEnabled": true,
  "email": "admin@example.com",
  "threshold": 10000
}
```
