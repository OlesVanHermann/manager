# Cahier des Charges : network/security - Onglet Attaques

## 1. VUE D'ENSEMBLE
- **Page** : Historique des attaques d'un bloc IP
- **URL** : /network/security/{ip}?tab=attacks
- **Objectif** : Afficher l'historique des attaques DDoS

## 2. STRUCTURE PAGE
- Breadcrumb et titre
- Onglets avec "Attaques" actif
- Bouton Rafraîchir
- Liste des attaques (cartes)

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : Rafraîchir (outline)
- **OdsCard** : Carte attaque (colorée selon statut)
- **OdsBadge** : Statut (En cours/Mitigé)
- **OdsIcon** : Warning/Check selon statut

## 4. ONGLETS
| Onglet | Actif |
|--------|-------|
| Vue d'ensemble | - |
| Attaques | ✓ |
| Firewall | - |

| Type | Description |
|------|-------------|
| UDP Flood | Saturation UDP |
| SYN Flood | Saturation SYN TCP |
| DNS Amplification | Amplification DNS |
| ICMP Flood | Ping flood |
| NTP Amplification | Amplification NTP |
