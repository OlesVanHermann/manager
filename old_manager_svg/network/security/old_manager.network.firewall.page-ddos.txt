# Cahier des Charges : network/firewall - Page DDoS

## 1. VUE D'ENSEMBLE
- **Type** : Page Protection DDoS
- **Contexte** : Network Firewall
- **Objectif** : Configurer et monitorer la protection anti-DDoS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : KPIs, attaques, config
- **OdsSelect** : Mode, sensibilité
- **OdsCheckbox** : Notifications

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Protection DDoS" |
| Description | "Configuration anti-DDoS pour vos services." |

## 5. CARTES KPI (3 colonnes)
| KPI | Valeur | Info |
|-----|--------|------|
| Protection active | VAC Enabled | Badge "Permanent" |
| Attaques bloquées (24h) | 3 | Volume: 12.5 Gbps mitigé |
| Capacité mitigation | 480 Tbps | Réseau mondial VAC |

## 6. SECTION ATTAQUES RÉCENTES
| Propriété | Valeur |
|-----------|--------|
| Largeur | 510px |
| Hauteur | 350px |
| Titre | "Attaques récentes" |

## 7. CARTE ATTAQUE
| Propriété | Valeur |
|-----------|--------|
| Type | SYN Flood, UDP Amp, HTTP Flood |
| Statut | Mitigated |
| Infos | Date, volume Gbps, durée |
| Cible | IP et nom serveur |

## 9. SECTION CONFIGURATION
| Propriété | Valeur |
|-----------|--------|

## 10. OPTIONS CONFIG
| Option | Valeurs |
|--------|---------|
| Mode protection | Permanent, Auto, Manuel |

| Notifications | Email, Webhook API |

## 11. APIS & ENDPOINTS
- GET /ip/{ip}/antihack - Statut anti-DDoS
- PUT /ip/{ip}/antihack - Configurer mode
- GET /ip/{ip}/antihack/attack - Historique attaques
- PUT /ip/{ip}/antihack/notification - Notifications
