**Type:** Page tableau de bord avec filtres
**Fonction:** Journal d'activité du centre de nettoyage Anti-DDoS
**Contexte:** Surveillance des attaques et de la protection DDoS

## 2. DIMENSIONS

- Page: 1200x700px
- Tableau: 1100x280px

## 3. LAYOUT & STRUCTURE

- Logo OVHcloud

### Breadcrumb

### Titre page
"Tableau de bord Anti-DDoS"

### Boutons guides
- "Changelog" (btn-outline)
- "Guides" (btn-outline avec dropdown)

### Description
"En surveillant constamment le trafic réseau, nous observons un certain nombre d'activités suspectes.
Pour chacune d'entre elles, le trafic réseau entrant est redirigé de manière transparente via nos centres de nettoyage."

## 8. APIS & ENDPOINTS

```
GET /ip/{ip}/antihack
→ Statut protection anti-hack

GET /ip/{ip}/mitigation
→ Historique des mitigations

GET /ip/{ip}/mitigation/{ipOnMitigation}/stats
→ Statistiques de mitigation

GET /ip/service/{serviceName}/mitigationProfiles
→ Profils de mitigation configurés
```
