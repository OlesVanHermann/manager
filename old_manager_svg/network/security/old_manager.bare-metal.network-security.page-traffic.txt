# Cahier des Charges : bare-metal/network-security - Page Traffic

## 1. VUE D'ENSEMBLE
- **Type** : Page analyse trafic
- **Contexte** : Network Security (Anti-DDoS)
- **Objectif** : Visualiser et analyser le trafic réseau et les attaques

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 700px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres
- **OdsMessage** : Bannière warning
- **OdsSelect** : Filtres
- **OdsInput** : Recherche IP
- **OdsCard** : Tuiles stats
- **OdsToggle** : Switch bps/pps
- **Chart.js** : Graphique trafic

| Propriété | Valeur |

| Titre | "Analyse du trafic" |
