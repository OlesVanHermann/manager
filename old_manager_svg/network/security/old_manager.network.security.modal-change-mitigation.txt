# Cahier des Charges : network/security - Modal Modifier Mitigation

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Déclencheur** : Action "Changer mitigation" ou menu IP
- **Objectif** : Modifier le mode de mitigation Anti-DDoS

## 2. STRUCTURE MODAL
- Overlay sombre (40% opacité)
- Modal centrée (600x540px)
- Header avec titre et bouton fermer
- Info bloc IP
- Options de mitigation (radio cards)
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Bloc IP concerné
- **OdsRadio** : Options de mitigation (cartes)
- **OdsButton** : Annuler/Appliquer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Modifier la mitigation" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. INFO BLOC IP
| Élément | Valeur |
|---------|--------|
| Label | "Bloc IP concerné" |
| Valeur | 192.168.1.0/24 (monospace) |

## 7. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Appliquer | Primary bleu | Droite |

## 8. MODES DE MITIGATION
| Mode | Description | Recommandé |
|------|-------------|------------|
| Automatique | Active à la demande | ✓ |
| Permanent | Toujours active | Pour services critiques |
| Désactivé | Pas de protection | ✕ |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/mitigation - Mode actuel
- PUT /ip/{ip}/mitigation - Modifier mode
