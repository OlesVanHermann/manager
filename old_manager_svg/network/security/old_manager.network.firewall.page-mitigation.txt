# Cahier des Charges : network/firewall - Page Mitigation

## 1. VUE D'ENSEMBLE
- **Type** : Page statistiques mitigation DDoS
- **Contexte** : Network Firewall
- **Objectif** : Visualiser les statistiques de protection DDoS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Cartes métriques et graphiques
- **OdsChart** : Graphique barres trafic
- **OdsProgress** : Barres progression attaques
