# Cahier des Charges : network/firewall - Modal Add Rule

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout règle firewall
- **Contexte** : Network Firewall
- **Objectif** : Créer une nouvelle règle de pare-feu

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 650x580px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Sélection Action
- **OdsSelect** : Protocol, Destination IP
- **OdsInput** : Source IP, Port, Description
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Ajouter une règle" |
| Bouton fermer | × |

## 5. CARTES ACTION
| Option | Style sélectionné |
|--------|-------------------|
| ALLOW | Border #16A34A, texte vert |
| DENY | Border #E5E7EB |

| Propriété | Valeur |
|-----------|--------|
| Label | "Protocol" |
| Options | TCP, UDP, ICMP, GRE, ANY |

## 12. APIS & ENDPOINTS
- GET /ip/{ip}/firewall/{ipBlock}/rule - Liste règles
- POST /ip/{ip}/firewall/{ipBlock}/rule - Créer règle
- DELETE /ip/{ip}/firewall/{ipBlock}/rule/{sequence} - Supprimer
