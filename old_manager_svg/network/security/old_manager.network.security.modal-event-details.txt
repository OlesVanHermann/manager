# Cahier des Charges : network/security - Modal Event Details

## 1. VUE D'ENSEMBLE

**Type:** Modal informatif
**Fonction:** Afficher les détails d'un événement de scrubbing/DDoS
**Contexte:** Monitoring sécurité

## 2. DIMENSIONS

- Modal: 550x420px
- Overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Titre | "Détails de l'événement de scrubbing" (18px, font-weight: 600) |
| Bouton fermer | × (20px, #6B7280) |

## 4. INFORMATIONS ÉVÉNEMENT

| Champ | Valeur exemple |
|-------|----------------|
| IP de destination | 192.168.1.100/32 |
| Heure de détection | 2024-12-30 14:25:00 |
| Heure de fin | 2024-12-30 14:45:00 |
| Statut | Badge "Attaque DDoS" (#FEE2E2, text #DC2626) |

## 5. VECTEURS D'ATTAQUE

### Zone code

- Background: #F3F4F6
- Font: Source Code Pro, 12px
