## 7. VARIANTE DÉSACTIVATION

Pour la désactivation, les éléments suivants changent :

| Élément | Valeur activation | Valeur désactivation |
|---------|-------------------|----------------------|
| Header background | #DBEAFE | #FEF3C7 |
| Titre | Activer le firewall | Désactiver le firewall |
| Couleur titre | #1E40AF | #D97706 |
| Question | Voulez-vous activer... | Voulez-vous désactiver... |
| Description | règles appliquées | règles ne seront plus appliquées |
| Bouton action | Activer | Désactiver |

---

## 8. APIS & ENDPOINTS

### 8.1 Activer le firewall

```
POST /ip/{ipAddress}/firewall/enable
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "pending"
}
```

### 8.2 Désactiver le firewall

```
POST /ip/{ipAddress}/firewall/disable
```
