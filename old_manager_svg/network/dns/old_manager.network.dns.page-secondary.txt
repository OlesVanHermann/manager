# Cahier des Charges : network/dns - Page Secondary

## 1. VUE D'ENSEMBLE
- **Type** : Page DNS Secondaire
- **Contexte** : Network DNS
- **Objectif** : Configurer les zones DNS secondaires (slave)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajouter zone slave
- **OdsDatagrid** : Liste zones
- **OdsBadge** : État synchronisation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "DNS Secondaire" |
| Description | "Configuration des zones DNS secondaires." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter zone slave" |
| Style | Primary #0050D7 |
| Largeur | 180px |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Zone | 230px | Nom zone (bold) |
| Master IP | 170px | IP serveur maître |
| Dernière synchro | 170px | Date/heure |
| Serial | 130px | Numéro série |
| Records | 100px | Nombre |
| État | 140px | Dot + texte |
| Actions | 60px | Menu ⋮ |

## 7. ÉTATS SYNCHRONISATION
| État | Couleur dot | Texte |
|------|-------------|-------|
| Sync | #16A34A | "Sync" |
| Stale | #F59E0B | "Stale" |
| Erreur | #DC2626 | "Erreur" |

## 8. MENU ACTIONS (⋮)
| Action | Style |
|--------|-------|
| Forcer synchro | Normal |
| Voir records | Normal |
| Éditer master | Normal |
| --- | Séparateur |
| Supprimer | Danger (#DC2626) |

## 9. APIS & ENDPOINTS
- GET /domain/zone/slave - Liste zones secondaires
- POST /domain/zone/slave - Ajouter zone slave
- POST /domain/zone/slave/{zone}/sync - Forcer synchro
- DELETE /domain/zone/slave/{zone} - Supprimer
