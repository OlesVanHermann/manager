## 10. APIS & ENDPOINTS
- GET /domain/zone/template - Liste templates
- POST /domain/zone/template - Créer template
- POST /domain/zone/{zoneName}/template/{templateId} - Appliquer template
- PUT /domain/zone/template/{id} - Modifier template
- DELETE /domain/zone/template/{id} - Supprimer template
