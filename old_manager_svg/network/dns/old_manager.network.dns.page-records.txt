## 1. VUE D'ENSEMBLE
- **Type** : Page listing enregistrements DNS
- **Contexte** : Gestion DNS
- **Objectif** : Gérer les records d'une zone DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Ajout record
- **OdsInput** : Recherche/filtre
- **OdsSelect** : Filtre par type
- **OdsDatagrid** : Liste des records
- **OdsBadge** : Type de record

## 10. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/record - Liste records
- POST /domain/zone/{zoneName}/record - Créer record
- PUT /domain/zone/{zoneName}/record/{id} - Modifier record
- DELETE /domain/zone/{zoneName}/record/{id} - Supprimer record
