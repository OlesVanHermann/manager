# Cahier des Charges : network/dns - Modal AXFR

## 1. VUE D'ENSEMBLE
- **Type** : Modal configuration transfert AXFR
- **Contexte** : Network DNS
- **Objectif** : Configurer le transfert de zone DNS secondaire

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsMessage** : Info AXFR
- **OdsTextarea** : Liste serveurs
- **OdsCheckbox** : Option NOTIFY
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Configurer transfert AXFR" |
| Bouton fermer | × en #6B7280 |

## 5. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |
| Icône | ℹ️ |
| Texte | "Le transfert AXFR permet à un serveur DNS secondaire de récupérer automatiquement les enregistrements de zone." |

## 6. TEXTAREA SERVEURS AUTORISÉS
| Propriété | Valeur |
|-----------|--------|
| Label | "Serveurs autorisés à recevoir le transfert" |
| Largeur | 450px |
| Hauteur | 100px |
| Police | Monospace 11px |
| Helper | "Une entrée par ligne (IP ou hostname)" |

### Exemple contenu
```
ns1.secondary-dns.com
ns2.secondary-dns.com
203.0.113.53
```

## 7. CHECKBOX NOTIFY
| Propriété | Valeur |
|-----------|--------|
| Label | "Activer les notifications (NOTIFY)" |
| État défaut | Coché |
| Description | Notifie les serveurs secondaires des changements |

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Enregistrer | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/dnssec - Config DNSSEC/AXFR
- PUT /domain/zone/{zoneName}/dnssec - Modifier config
- POST /domain/zone/{zoneName}/refresh - Forcer transfert
