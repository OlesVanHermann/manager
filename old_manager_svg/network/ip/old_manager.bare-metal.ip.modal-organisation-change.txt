# Cahier des Charges : bare-metal/ip - Modal modifier organisation

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des blocs IP
- **Objectif** : Changer l'organisation associée à un bloc IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description, bloc IP
- **OdsSelect** : Organisation
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'organisation" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Changer l'organisation pour le bloc" |
| Alignement | Centré |

### Bloc IP
| Propriété | Valeur |
|-----------|--------|
| Format | IP en bleu (#0050D7), gras, centré |
| Exemple | "51.210.xxx.0/24" |

## 6. FORMULAIRE

### Champ Organisation
| Propriété | Valeur |
|-----------|--------|
| Label | "Organisation" |
| Type | OdsSelect |
| Format | "{org_id} - {nom}" |
| Exemple | "ORG-ACME-RIPE - Jean Dupont" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary (bleu #0050D7) | Applique le changement |

## 8. APIS & ENDPOINTS
- GET /ip/organisation - Liste des organisations
- PUT /ip/{ip}/organisation - Modifier l'organisation
