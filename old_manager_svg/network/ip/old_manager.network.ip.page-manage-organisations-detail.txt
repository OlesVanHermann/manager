## 10. APIS & ENDPOINTS

### 10.1 Lister les organisations

```
GET /ip/organisation
```

Response:
```json
[
  {
    "organisationId": "ORG-OVH-FR1",
    "type": "RIPE",
    "name": "Jean Dupont",
    "abuse": "abuse@example.com",
    "phone": "+33 1 23 45 67 89",
    "address": "123 Rue Example, 75001 Paris, France"
  }
]
```

### 10.2 Obtenir une organisation

```
GET /ip/organisation/{organisationId}
```

Response:
```json
{
  "organisationId": "ORG-OVH-FR1",
  "type": "RIPE",
  "name": "Jean Dupont",
  "firstname": "Jean",
  "lastname": "Dupont",
  "abuse": "abuse@example.com",
  "phone": "+33 1 23 45 67 89",
  "address": "123 Rue Example",
  "city": "Paris",
  "zip": "75001",
  "country": "FR"
}
```

### 10.3 Créer une organisation

```
POST /ip/organisation
```

Body:
```json
{
  "firstname": "Jean",
  "lastname": "Dupont",
  "abuse": "abuse@example.com",
  "phone": "+33 1 23 45 67 89",
  "address": "123 Rue Example",
  "city": "Paris",
  "zip": "75001",
  "country": "FR"
}
```

### 10.4 Modifier une organisation

```
PUT /ip/organisation/{organisationId}
```

### 10.5 Supprimer une organisation

```
DELETE /ip/organisation/{organisationId}
```
