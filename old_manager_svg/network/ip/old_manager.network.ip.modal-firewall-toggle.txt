# Cahier des Charges : network/ip - Modal Firewall Toggle

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation wizard 2 étapes
**Fonction:** Activer/Désactiver le firewall pour une IP
**Contexte:** Gestion du Edge Network Firewall

## 2. DIMENSIONS

- Modal: 500x270px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Activer le firewall"
- Bouton fermer: ×

### Stepper (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Confirmation | Active |
| 2 | Traitement | Inactive |

### Question
- "Souhaitez-vous activer le firewall pour cette IP ?"
- Font-weight: 600

### Description
- "Le firewall protégera votre IP contre les attaques réseau."
- "Des règles pourront être configurées après activation."
- Couleur: text-muted

## 4. COMPOSANTS UI

### Stepper compact
- Cercle actif: bleu #0050D7
- Cercle inactif: blanc avec bordure grise
- Ligne connexion: #E5E7EB

## 5. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Ferme le modal

### Bouton Activer
- Style: btn-primary (100x36px)
- Action: Active le firewall, passe à l'étape 2

## 6. VARIANTE DÉSACTIVATION

### Titre alternatif
- "Désactiver le firewall"

### Question alternative
- "Souhaitez-vous désactiver le firewall pour cette IP ?"

### Description alternative
- "La désactivation supprimera la protection mais conservera vos règles."

## 7. APIS & ENDPOINTS

```
PUT /ip/{ip}/firewall/{ipOnFirewall}
Body: { enabled: true }
→ Active le firewall

PUT /ip/{ip}/firewall/{ipOnFirewall}
Body: { enabled: false }
→ Désactive le firewall
```
