# Cahier des Charges : bare-metal/ip - Wizard Block Move

## 1. VUE D'ENSEMBLE
- **Type** : Wizard modal multi-étapes
- **Contexte** : Déplacement bloc IP
- **Objectif** : Déplacer un bloc IP vers un autre service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Container modal
- **OdsStepper** : Navigation étapes
- **OdsText** : Labels et descriptions
- **OdsSelect** : Sélecteurs service et nexthop
- **OdsButton** : Actions
