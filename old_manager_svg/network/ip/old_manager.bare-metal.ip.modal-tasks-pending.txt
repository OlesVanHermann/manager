# Cahier des Charges : bare-metal/ip - Modal tâches en cours

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec liste
- **Contexte** : Gestion des adresses IP
- **Objectif** : Afficher les tâches en cours sur les adresses IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Carte de tâche
- **OdsText** : Titre, description
- **OdsBadge** : Statut
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Tâches en cours" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. LISTE DES TÂCHES

### Structure d'une carte de tâche
| Élément | Description |
|---------|-------------|
| Indicateur | Barre verticale colorée (4px) |
| Titre | Nom de la tâche (gras) |
| Description | Détails de la tâche |
| Badge | Statut de la tâche |

### Statuts possibles

#### En cours
| Propriété | Valeur |
|-----------|--------|
| Indicateur | Orange (#F59E0B) |
| Badge fond | #FEF3C7 |
| Badge texte | #92400E |
| Label | "En cours" |

#### Planifiée
| Propriété | Valeur |
|-----------|--------|
| Indicateur | Bleu (#0050D7) |
| Badge fond | #DBEAFE |
| Badge texte | #1E40AF |
| Label | "Planifiée" |

#### Terminée
| Propriété | Valeur |
|-----------|--------|
| Indicateur | Vert (#10B981) |
| Badge fond | #DCFCE7 |

| Label | "Terminée" |

### Exemples de tâches
| Tâche | Description |
|-------|-------------|
| Migration IP Failover | "51.210.xxx.100 → Additional IP" |
| Attribution reverse DNS | "51.210.xxx.101 → mail.example.com" |
| Génération certificat SSL | "server.example.com" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /ip/task - Liste des tâches en cours
