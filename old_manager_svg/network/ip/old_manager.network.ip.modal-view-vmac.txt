# Cahier des Charges : network/ip - Modal Voir MAC Virtuelle

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information (lecture seule)
- **Déclencheur** : Action "Voir la MAC virtuelle" dans le menu IP
- **Objectif** : Afficher les détails d'une MAC virtuelle existante

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x380px)
- Header avec titre et bouton fermer
- Tableau de détails key-value
- Footer avec bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et valeurs
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Détails de la MAC virtuelle" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. CONTENU

### Introduction
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voici le détail de la MAC virtuelle:" |
| Position | Haut du contenu |

### Détails (Key-Value)
| Label | Valeur | Style label |
|-------|--------|-------------|
| Adresse IP | 192.168.1.100 | Bold |
| MAC virtuelle | 02:00:00:12:34:56 | Bold |
| Type | ovh | Bold |
| Nom de la machine virtuelle | vm-web-01 | Bold |

## 6. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline (bordure grise) | Droite |

## 7. TYPES DE MAC
| Type | Préfixe | Description |
|------|---------|-------------|
| ovh | 02:00:00 | MAC générée par OVHcloud |
| vmware | 00:50:56 | MAC compatible VMware |

## 8. FORMAT MAC
| Propriété | Valeur |
|-----------|--------|
| Format | XX:XX:XX:XX:XX:XX |
| Séparateur | : (deux-points) |
| Case | Minuscules |
| Longueur | 17 caractères |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/virtualMac - Liste des MAC virtuelles
- GET /ip/{ip}/virtualMac/{macAddress} - Détails d'une MAC
