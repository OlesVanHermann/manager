## 9. APIS & ENDPOINTS

### 9.1 Lister les services éligibles par type

```
GET /ip/order/eligibleServices
```

Query params:
```
?type=dedicated|vps|pcc|parking
```

Response:
```json
{
  "dedicated": [
    { "serviceName": "ns123456.ip-123-45-67.eu", "displayName": "ns123456.ip-123-45-67.eu" }
  ],
  "vps": [
    { "serviceName": "vps-xxxxx.ovh.net", "displayName": "vps-xxxxx.ovh.net" }
  ],
  "pcc": [
    { "serviceName": "pcc-xxx-xxx-xxx-xxx", "displayName": "pcc-xxx-xxx-xxx-xxx" }
  ]
}
```
