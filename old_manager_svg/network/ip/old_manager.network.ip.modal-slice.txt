# Cahier des Charges : network/ip - Modal Slice

## 1. VUE D'ENSEMBLE
- **Type** : Modal segmentation bloc IP
- **Contexte** : Gestion IP
- **Objectif** : Segmenter un bloc IP en sous-blocs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 600x480px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Taille des sous-blocs
- **OdsCard** : Prévisualisation blocs créés
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Segmenter le bloc {bloc}" |
| Bouton fermer | × |

## 5. INSTRUCTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Veuillez indiquer la taille des blocs enfants désirée" |

## 6. SELECT TAILLE
| Propriété | Valeur |
|-----------|--------|
| Options | /24, /25, /26, /27, /28 |
| Largeur | 100% |

## 7. SECTION PRÉVISUALISATION
| Propriété | Valeur |
|-----------|--------|
| Fond | #F3F4F6 |
| Titre | "Cette configuration entraînera la création des blocs suivants" |
| Liste | Puces + CIDR blocs enfants |

## 8. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Confirmer | Primary |

## 9. APIS & ENDPOINTS
- GET /ip/{bloc}/slice/preview - Prévisualisation
- POST /ip/{bloc}/slice - Segmenter bloc
