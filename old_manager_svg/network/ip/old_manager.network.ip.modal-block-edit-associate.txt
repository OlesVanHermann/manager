# Cahier des Charges : network/ip - Modal Block Edit Associate

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire avec onglets
**Fonction:** Éditer les informations réseau ou associer à une organisation
**Contexte:** Action depuis la liste des blocs IP

## 2. DIMENSIONS

- Modal: 500x350px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #000E9C
- Titre: "Éditer / Associer le bloc IP" (15px, blanc, bold)
- Bouton fermer: × (18px, blanc)

## 4. DESCRIPTION

"Vous pouvez éditer les informations du bloc ou l'associer à une organisation." (12px, muted)

## 5. ONGLETS

| Onglet | État | Style |
|--------|------|-------|
| Réseau | Actif | #0050D7, border-bottom 2px |
| Organisation | Inactif | Muted |

### Champ Description

## 9. APIS & ENDPOINTS

```
# Informations bloc IP
GET /ip/{ip}/ripe
→ {
  netname: "OVH-DEDICATED-BLOCK",
  description: "Production servers - Web cluster"
}

# Modifier bloc
PUT /ip/{ip}/ripe
Body: {
  netname: "OVH-DEDICATED-BLOCK",
  description: "Production servers - Web cluster"
}

# Associer à organisation
POST /ip/{ip}/organisation
Body: {
  organisationId: "ORG-ABC123"
}
```
