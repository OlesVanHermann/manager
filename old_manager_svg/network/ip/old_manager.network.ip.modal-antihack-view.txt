# Cahier des Charges : network/ip - Modal Antihack View

## 1. VUE D'ENSEMBLE

**Type:** Modal information
**Fonction:** Afficher le statut de blocage antihack et les logs
**Contexte:** Action depuis la liste IP

## 2. DIMENSIONS

- Modal: 500x350px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #000E9C
- Titre: "Antihack" (15px, blanc, bold)
- Bouton fermer: × (18px, blanc)

## 4. STEPPER

- 1 étape active (#0050D7)

## 5. CONTENU

### Informations statut

| Champ | Valeur |
|-------|--------|
| Statut | Badge "Bloqué" (#FEE2E2, #DC2626) |
| Bloqué depuis | 28/12/2025 09:15:22 |
| Temps restant | 18h 22m 08s |

### Section Logs
- Label: "Logs :" (12px, muted, bold)
- Container: 440x80px, background #F3F4F6

## 6. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Débloquer antihack | btn-primary | 150x36px |
| Fermer | btn-outline | 100x36px |

## 7. APIS & ENDPOINTS

```
# Statut antihack
GET /ip/{ip}/antihack/{ipBlocked}
→ {
  ipBlocked: "51.178.42.123",
  state: "blocked",
  blockedSince: "2025-12-28T09:15:22Z",
  time: 66128  // secondes restantes
}

# Logs antihack
GET /ip/{ip}/antihack/{ipBlocked}/logs
→ [{
  message: "Intrusion attempt detected",
  target: "51.178.42.123:22",
  type: "SSH",
  attemptsBlocked: 8456
}]

# Débloquer
POST /ip/{ip}/antihack/{ipBlocked}/unblock
```
