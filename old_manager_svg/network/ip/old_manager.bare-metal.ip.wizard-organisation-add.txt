# Cahier des Charges : bare-metal/ip - Wizard ajouter organisation

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (2 étapes)
- **Contexte** : Gestion des adresses IP - Organisations RIPE
- **Objectif** : Créer une nouvelle organisation pour l'enregistrement WHOIS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 660px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : Indicateur d'étapes
- **OdsMessage** : Information
- **OdsText** : Labels
- **OdsSelect** : Type, Pays
- **OdsInput** : Tous les champs de formulaire
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une organisation" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |
