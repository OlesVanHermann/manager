# Cahier des Charges : network/ip - Modal Firewall

## 1. VUE D'ENSEMBLE
- **Type** : Modal configuration firewall
- **Contexte** : Gestion IP - Sécurité
- **Objectif** : Configurer les règles firewall d'une IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 800x650px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSwitch** : Toggle activation firewall
- **OdsButton** : Ajouter règle
- **OdsDatagrid** : Liste des règles
- **OdsBadge** : ACCEPT/DROP, statut

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configurer le firewall - {ip}" |
| Bouton fermer | × |

## 5. TOGGLE FIREWALL
| Propriété | Valeur |
|-----------|--------|
| Label | "État du firewall" |
| États | Activé (#0050D7) / Désactivé (#E5E7EB) |

## 6. COLONNES TABLEAU RÈGLES
| Colonne | Contenu |
|---------|---------|
| # | Numéro ordre |
| Action | Badge ACCEPT/DROP |
| Protocole | TCP/UDP/ICMP |
| Source | IP/CIDR |
| Port | Port(s) |
| Statut | Badge Actif |
| Actions | Modifier/Supprimer |

## 7. BADGES ACTION
| Action | Fond | Texte |
|--------|------|-------|
| ACCEPT | #DEF7EC | #03543F |
| DROP | #FEE2E2 | #991B1B |

## 8. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Fermer | Outline |
| Appliquer les règles | Primary |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/firewall - État firewall
- PUT /ip/{ip}/firewall - Activer/désactiver
- GET /ip/{ip}/firewall/rule - Liste règles
- POST /ip/{ip}/firewall/rule - Ajouter règle
- DELETE /ip/{ip}/firewall/rule/{ruleId} - Supprimer
