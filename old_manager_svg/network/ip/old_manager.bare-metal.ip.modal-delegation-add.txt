# Cahier des Charges : bare-metal/ip - Modal ajouter délégation DNS

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des blocs IP (IPv6)
- **Objectif** : Ajouter une délégation DNS pour gérer les enregistrements reverse

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, bloc IP
- **OdsInput** : Serveur DNS cible
- **OdsMessage** : Information délégation
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une délégation DNS" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS AFFICHÉES
| Champ | Label | Exemple |
|-------|-------|---------|
| Bloc | "Bloc IP" | "2001:41d0:xxxx::/56" (gras) |

## 6. FORMULAIRE

### Champ Serveur DNS cible *
| Propriété | Valeur |
|-----------|--------|
| Label | "Serveur DNS cible *" |
| Type | OdsInput |
| Placeholder | "ns1.example.com" |
| Requis | Oui |

## 7. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Ligne 1 | "La délégation DNS permet de gérer les enregistrements" |
| Ligne 2 | "reverse sur votre propre serveur DNS." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary (bleu #0050D7) | Ajoute la délégation |

## 9. APIS & ENDPOINTS
- POST /ip/{ip}/delegation - Ajouter une délégation DNS
