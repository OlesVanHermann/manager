# Cahier des Charges : bare-metal/ip - Modal assigner IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Assigner une adresse IP à un service (serveur dédié)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, IP
- **OdsSelect** : Service destination
- **OdsInput** : Nexthop (optionnel)
- **OdsButton** : Annuler, Assigner

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Assigner l'adresse IP" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS AFFICHÉES
| Champ | Label | Exemple |
|-------|-------|---------|
| IP | "Adresse IP" | "51.210.xxx.102" (gras) |

## 6. FORMULAIRE

### Champ Service de destination *
| Propriété | Valeur |
|-----------|--------|
| Label | "Service de destination *" |
| Type | OdsSelect |
| Exemple | "ns1234570.ip-51-210-xxx.eu" |
| Requis | Oui |

### Champ Nexthop (optionnel)
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse de nexthop (optionnel)" |
| Type | OdsInput |
| Placeholder | "IP du service cible" |
| Requis | Non |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Assigner | Primary (bleu #0050D7) | Assigne l'IP |

## 8. APIS & ENDPOINTS
- GET /dedicated/server - Liste des serveurs disponibles
- POST /ip/{ip}/move - Assigner l'IP à un service
