# Cahier des Charges : network/ip - Order Type

## 1. VUE D'ENSEMBLE

**Type:** Page de sélection
**Fonction:** Choisir le type d'IP à commander (IPv4 ou IPv6)
**Contexte:** Première étape du processus de commande IP

## 2. DIMENSIONS

- Page: 1200x550px
- Cartes: 350x220px chacune

## 3. LAYOUT & STRUCTURE

### Header OVHcloud
- Background: #000E9C
- Logo OVHcloud

### Breadcrumb
Network > IP > Commander Additional IP

### Lien retour
"< Retour" vers listing IP

### Titre page
"Commander Additional IP"

### Sous-titre
"Sélectionnez le type d'adresse IP que vous souhaitez commander."

## 5. COMPOSANTS UI

### Carte sélectionnable
- Border-radius: 8px
- Background: white
- Border selected: #0050D7, 2px
- Border default: #E5E7EB, 1px

### Indicateur sélection
- Position: top-right (320, 30)
- Cercle: r=12, fill #0050D7
- Checkmark: blanc, stroke-width 2

### Prix display
- Font-size: 20px
- Font-weight: 600
- Color: #0050D7 (payant) ou #16A34A (gratuit)

## 6. ACTIONS & INTERACTIONS

### Sélection carte
- Click: toggle la sélection
- Une seule carte sélectionnable

### Bouton Continuer
- Style: btn-primary (150x40px)
- Position: bas gauche
- Action:
  - IPv4: redirige vers wizard commande IPv4
  - IPv6: redirige vers page IPv6 (généralement déjà inclus)

## 7. APIS & ENDPOINTS

```
GET /order/catalog/public/ip
→ Catalogue avec prix IPv4/IPv6

GET /ip?type=failover
→ Vérifie les IP existantes

GET /me/order/cart
→ Panier en cours
```
