# Cahier des Charges : bare-metal/ip - Page Anti-spam

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion anti-spam
- **Contexte** : Gestion IP
- **Objectif** : Gérer les IPs bloquées pour spam

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsMessage** : Bannière warning
- **OdsDatagrid** : Tableau IPs
- **OdsBadge** : Statut blocage
- **OdsButton** : Action déblocage
- **OdsPagination** : Navigation pages

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Anti-spam" |
| Description | "Gérez les IPs bloquées pour spam." |

## 5. BANNIÈRE WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Texte principal | "⚠ Attention : X IP est actuellement bloquée pour spam." |
| Texte secondaire | "Débloquez-la après avoir résolu le problème à l'origine du spam." |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| IP | 180px | Adresse IP |
| Date de blocage | 200px | Date/heure blocage |
| Date de déblocage | 200px | Date/heure ou "-" |
| Statut | 200px | Badge statut |
| Actions | 200px | Bouton déblocage ou "-" |

## 7. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Bloquée | #FEE2E2 | #DC2626 |
| Débloquée | #DCFCE7 | #16A34A |

## 8. BOUTON DÉBLOQUER
| Propriété | Valeur |
|-----------|--------|
| Texte | "Débloquer" |
| Style | Fond #0050D7, texte blanc |
| Dimensions | 100x32px |
| Visible | Uniquement si statut "Bloquée" |

## 9. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "1 - 4 sur 4" |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/spam - Liste IPs spam
- POST /ip/{ip}/spam/{ipSpamming}/unblock - Débloquer IP
