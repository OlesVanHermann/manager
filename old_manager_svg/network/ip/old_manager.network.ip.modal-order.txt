# Cahier des Charges : network/ip - Modal Order

## 1. VUE D'ENSEMBLE
- **Type** : Modal commande IP
- **Contexte** : Gestion IP
- **Objectif** : Commander de nouvelles adresses IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 800x700px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsRadio** : Type d'IP (cartes)
- **OdsSelect** : Service, pays, quantité
- **OdsCard** : Récapitulatif commande
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander des adresses IP" |
| Bouton fermer | ✕ |

## 5. TYPES D'IP (CARTES)
| Type | Description |
|------|-------------|
| IP Failover | Adresse individuelle basculable |
| Bloc IP RIPE | Bloc d'adresses IP |
| BYOIP | Importez vos propres IP |

## 6. STYLE CARTES TYPE
| État | Fond | Bordure |
|------|------|---------|
| Sélectionné | #E8F0FE | #0050D7, 2px |
| Non sélectionné | Blanc | #E5E7EB |

## 7. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Service de destination | Select | Oui |
| Pays de géolocalisation | Select | Oui |
| Quantité | Select | Oui |

## 8. RÉCAPITULATIF COMMANDE
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Ligne produit | Description + prix |
| Total | Montant HT/mois |

## 9. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Commander | Primary |

## 10. APIS & ENDPOINTS
- GET /ip/order/eligibility - Services éligibles
- GET /ip/order/countries - Pays disponibles
- POST /ip/order - Créer commande
