# Cahier des Charges : bare-metal/ip - Modal générer certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Générer un certificat SSL Let's Encrypt pour le reverse DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsMessage** : Information
- **OdsSelect** : Type de certificat
- **OdsButton** : Annuler, Générer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Générer un certificat SSL" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS AFFICHÉES
| Champ | Label | Exemple |
|-------|-------|---------|
| IP | "Adresse IP" | "51.210.xxx.100" (gras) |
| Reverse | "Reverse DNS" | "server.example.com" |

## 6. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Ligne 1 | "Un certificat SSL Let's Encrypt sera généré" |
| Ligne 2 | "automatiquement pour le reverse DNS configuré." |

## 7. FORMULAIRE

### Champ Type de certificat
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de certificat" |
| Type | OdsSelect |
| Options | "Let's Encrypt (gratuit)" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Générer | Primary (bleu #0050D7) | Génère le certificat |

## 9. APIS & ENDPOINTS
- POST /ip/{ip}/ssl - Générer un certificat SSL
