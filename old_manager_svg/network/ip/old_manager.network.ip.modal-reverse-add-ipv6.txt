# Cahier des Charges : network/ip - Modal Reverse Add IPv6

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Configurer le reverse DNS d'une adresse IPv6
**Contexte:** Gestion DNS inverse pour bloc IPv6

## 2. DIMENSIONS

- Modal: 500x370px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Configurer le reverse d'une IPv6"
- Bouton fermer: ×

### Message info
- Background: #EFF6FF
- Border: #BFDBFE
- Texte (2 lignes):
  - "Insérez une IPv6 valide contenue dans ce bloc,"
  - "et cliquez sur « Valider »."

## 4. FORMULAIRE

| Champ | Type | État | Valeur exemple |
|-------|------|------|----------------|
| Bloc IP | Input disabled | Readonly | 2001:41d0:302:2100::/56 |
| Adresse IP | Input | Editable | 2001:41d0:302:2100::1 |
| Reverse | Input | Editable | mail.example.com |

### Champ Bloc IP
- Background: #F3F4F6 (disabled)
- Affiche le bloc IPv6 parent

### Champ Adresse IP
- IPv6 complète à renseigner
- Doit appartenir au bloc

### Champ Reverse
- FQDN valide

## 5. COMPOSANTS UI

### Input disabled
- Background: #F3F4F6
- Text color: text-muted

### Input enabled
- Background: white
- Border: #E5E7EB

## 6. ACTIONS & INTERACTIONS

### Validation
- Adresse IP: IPv6 valide dans le bloc
- Reverse: FQDN valide (avec résolution)

### Bouton Annuler
- Style: btn-secondary (100x36px)

### Bouton Valider
- Style: btn-primary (100x36px)
- Action: Configure le reverse

## 7. APIS & ENDPOINTS

```
GET /ip/{ip}/reverse
→ Récupère le reverse actuel

POST /ip/{ip}/reverse
Body: {
  ipReverse: "2001:41d0:302:2100::1",
  reverse: "mail.example.com"
}
→ Configure le reverse IPv6
```
