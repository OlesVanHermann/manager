# Cahier des Charges : bare-metal/ip - Modal débloquer IP (Antihack)

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Protection Antihack
- **Objectif** : Débloquer une adresse IP bloquée suite à une activité malveillante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Débloquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "IP bloquée - Antihack" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. INFORMATIONS AFFICHÉES

| Champ | Label | Exemple |
|-------|-------|---------|
| IP | "Adresse IP" | 51.210.xxx.100 (rouge, gras) |
| Raison | "Raison" | "Activité malveillante détectée" |
| Date | "Bloqué depuis" | "29/12/2024 10:15:32" |

## 6. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (rouge) |
| Fond | #FEE2E2 |
| Texte | "Assurez-vous d'avoir corrigé le problème avant de débloquer l'adresse IP." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Débloquer | Danger (rouge #DC2626) | Débloque l'IP |

## 8. APIS & ENDPOINTS
- GET /ip/{ip}/antihack - Détails du blocage
- POST /ip/{ip}/antihack/unblock - Débloquer l'IP
