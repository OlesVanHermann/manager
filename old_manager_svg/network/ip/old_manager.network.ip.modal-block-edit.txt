# Cahier des Charges : network/ip - Modal Block Edit

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire avec onglets
**Fonction:** Modifier les informations WHOIS d'un bloc IP
**Contexte:** Personnalisation des informations publiques du bloc

## 2. DIMENSIONS

- Modal: 500x370px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Voir/Modifier les informations sur le bloc IP"
- Bouton fermer: ×

### Description
- "Cette section vous permet de personnaliser les informations publiques sur votre bloc IP (informations présentes sur la base whois)."

### Onglets
| Onglet | État |
|--------|------|
| Réseau | Actif (souligné bleu) |
| Organisation | Inactif |

## 4. ONGLET RÉSEAU (actif)

### Champ Netname
- Label: "Nom du réseau (Netname)"
- Type: text input
- Dimensions: 450x40px
- Valeur: "OVH-CUSTOMER-NET"

### Champ Description
- Label: "Description"
- Type: textarea
- Dimensions: 450x80px
- Valeur exemple:
  - "Web hosting infrastructure"
  - "Production servers"

## 5. ONGLET ORGANISATION (non visible)

### Contenu attendu
- Nom de l'organisation
- Adresse
- Contact technique
- Contact abuse

## 6. COMPOSANTS UI

### Onglets
- Actif: bg-white, bordure bleue en bas, text-link
- Inactif: bg-muted #F3F4F6, text-muted

### Inputs
- Border: #E5E7EB
- Border-radius: 4px
- Padding: 15px

## 7. ACTIONS & INTERACTIONS

### Navigation onglets
- Click sur onglet → affiche le contenu

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Ferme sans sauvegarder

### Bouton Éditer
- Style: btn-primary (100x36px)
- Action: Sauvegarde les modifications

## 8. APIS & ENDPOINTS

```
GET /ip/{ip}/block
→ Récupère les informations du bloc

PUT /ip/{ip}/block
Body: {
  netname: "OVH-CUSTOMER-NET",
  description: "Web hosting infrastructure"
}
→ Met à jour les informations
```
