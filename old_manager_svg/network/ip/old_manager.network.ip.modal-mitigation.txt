# Cahier des Charges : network/ip - Modal Mitigation

## 1. VUE D'ENSEMBLE
- **Type** : Modal configuration Anti-DDoS
- **Contexte** : Gestion IP - Sécurité
- **Objectif** : Gérer la mitigation Anti-DDoS d'une IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 600x580px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsRadio** : Options mode mitigation (cartes)
- **OdsBadge** : Statut IP
- **OdsMessage** : Info et warning
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Gérer la mitigation Anti-DDoS" |
| Bouton fermer | ✕ |

## 5. INFO IP
| Propriété | Valeur |
|-----------|--------|
| Fond | #E8F0FE |
| Label | "Adresse IP" |
| Badge statut | "Statut: Normal" (vert) |

## 6. MODES MITIGATION
| Mode | Description |
|------|-------------|
| Automatique (recommandé) | S'active automatiquement lors d'une attaque |
| Permanent | Toujours active, peut augmenter la latence |
| Forcer maintenant | Activation immédiate pendant 1 heure |

## 7. STYLE CARTES OPTIONS
| État | Fond | Bordure |
|------|------|---------|
| Sélectionné | #E8F0FE | #0050D7, 2px |
| Non sélectionné | Blanc | #E5E7EB |

## 8. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Texte | #92400E |
| Message | Impact performances mode permanent |

## 9. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Appliquer | Primary |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/mitigation - État mitigation
- PUT /ip/{ip}/mitigation - Modifier mode mitigation
- POST /ip/{ip}/mitigation/force - Forcer activation
