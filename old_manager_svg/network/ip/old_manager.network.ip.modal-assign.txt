# Cahier des Charges : network/ip - Modal Assign

## 1. VUE D'ENSEMBLE
- **Type** : Modal assignation IP Failover
- **Contexte** : Network IP
- **Objectif** : Assigner une IP Failover à un serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : IP (readonly), MAC virtuelle
- **OdsSelect** : Serveur destination
- **OdsMessage** : Info propagation
- **OdsButton** : Annuler, Assigner

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Assigner une IP Failover" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT IP (READONLY)
| Propriété | Valeur |
|-----------|--------|
| Label | "IP à assigner" |
| Largeur | 450px |
| Background | #F9FAFB |
| Valeur exemple | "141.94.200.15" |

## 6. SELECT SERVEUR DESTINATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Serveur de destination" |
| Largeur | 450px |
| Valeur exemple | "ns123456.ip-xxx-xxx.eu" |

## 7. INPUT MAC VIRTUELLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse MAC virtuelle (optionnel)" |
| Largeur | 450px |
| Placeholder | "02:00:00:xx:xx:xx (auto-généré)" |

## 8. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |
| Icône | ℹ️ |
| Texte | "L'assignation peut prendre quelques minutes à se propager." |

## 9. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Assigner | Primary #0050D7 | 100px |

## 10. APIS & ENDPOINTS
- POST /ip/{ip}/move - Déplacer IP vers serveur
- GET /dedicated/server - Liste serveurs disponibles
- POST /ip/{ip}/virtualMac - Créer MAC virtuelle
