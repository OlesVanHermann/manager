# Cahier des Charges : bare-metal/ip - Page List

## 1. VUE D'ENSEMBLE
- **Type** : Page listing IPs
- **Contexte** : Gestion IP
- **Objectif** : Lister et gérer toutes les IPs du compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsChip** : Filtre IP actif
- **OdsButton** : Actions (settings, export)
- **OdsSwitch** : Mode avancé
- **OdsSelect** : Filtre type IP
- **OdsDatagrid** : Tableau IPs
- **OdsPagination** : Navigation pages
