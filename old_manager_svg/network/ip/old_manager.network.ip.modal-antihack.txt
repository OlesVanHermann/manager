# Cahier des Charges : network/ip - Modal Anti-hack

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information et action
- **Déclencheur** : Action "Anti-hack" dans le menu IP (IP bloquée)
- **Objectif** : Afficher le statut de blocage et permettre le déblocage

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x480px)
- Header avec titre et bouton fermer
- Informations de blocage (tableau key-value)
- Zone logs (monospace)
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et valeurs
- **OdsCode** : Zone logs monospace
- **OdsButton** : Fermer/Débloquer l'IP

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Anti-hack" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. INFORMATIONS BLOCAGE
| Label | Valeur | Style |
|-------|--------|-------|
| Status | blocked | Text body |
| Bloquée depuis | 27 décembre 2025 à 10:30:00 | Text body |
| Temps restant (s) | 3600 | Text body |

## 6. ZONE LOGS
| Propriété | Valeur |
|-----------|--------|
| Fond | Gris (#F3F4F6) |
| Dimensions | 540x180px |
| Font | Monospace 12px |
| Bordure | Gris #D1D5DB |

### Exemple de logs
```
[2025-12-27 10:28:15] Suspicious activity detected
[2025-12-27 10:28:16] Multiple failed SSH attempts
[2025-12-27 10:28:17] Source: 203.0.113.50
[2025-12-27 10:28:18] IP blocked by anti-hack system
[2025-12-27 10:30:00] Block confirmed
--- End of logs ---
```

## 7. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline (bordure grise) | Gauche |
| Débloquer l'IP | Primary (bleu) | Droite |

## 8. TYPES DE BLOCAGE
| Type | Description |
|------|-------------|
| SSH bruteforce | Tentatives SSH multiples échouées |
| SMTP abuse | Envoi massif de mails |
| Scan ports | Scan de ports détecté |
| DDoS source | Participation à une attaque |

## 9. STATUTS POSSIBLES
| Status | Description |
|--------|-------------|
| blocked | IP actuellement bloquée |
| released | IP débloquée manuellement |
| expired | Blocage expiré automatiquement |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/antihack/{ipBlocked} - Détails blocage
- DELETE /ip/{ip}/antihack/{ipBlocked} - Débloquer l'IP
