# Cahier des Charges : network/ip - Modal Organisation

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire organisation
- **Contexte** : Gestion IP - RIPE
- **Objectif** : Ajouter une organisation pour les IP RIPE

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 700x820px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type (RIR), Pays
- **OdsInput** : Champs texte
- **OdsPhoneNumber** : Téléphone avec indicatif
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une organisation" |
| Bouton fermer | × |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Largeur |
|-------|------|---------|
| Type | Select (RIPE, ARIN...) | 100% |
| Nom | Input | 50% |
| Prénom | Input | 50% |
| Pays | Select | 100% |
| Adresse E-mail | Input | 100% |
| Adresse | Input | 100% |
| Ville | Input | 50% |
| Code postal | Input | 50% |
| Téléphone | PhoneNumber | 100% |

## 6. STYLE INPUTS
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc |
| Bordure | #D1D5DB |
| Hauteur | 44px |

| Élément | Style |
|---------|-------|
| Indicatif pays | Badge #F3F4F6 avec drapeau |
| Numéro | Input texte |

## 8. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Confirmer | Primary |

## 9. APIS & ENDPOINTS
- GET /ip/organisation/country - Liste pays
- POST /ip/organisation - Créer organisation
