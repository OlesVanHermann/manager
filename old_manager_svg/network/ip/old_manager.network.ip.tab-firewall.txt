# Cahier des Charges : network/ip - Onglet Firewall

## 1. VUE D'ENSEMBLE
- **Page** : Vue d'ensemble Firewall
- **URL** : /network/ip?tab=firewall
- **Objectif** : Gérer le firewall de toutes les IP

## 2. STRUCTURE PAGE
- Sidebar Network
- Titre "IP"
- Onglets navigation
- 3 cartes statistiques
- Datagrid des IP avec statut firewall
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Toutes les IP / IP Failover / Blocs RIPE / BYOIP / Firewall (actif)
- **OdsCard** : 3 cartes stats
- **OdsDatagrid** : Liste IP avec firewall
- **OdsToggle** : Activer/désactiver firewall
- **OdsBadge** : Statut (Actif/Désactivé)
- **OdsPagination** : Navigation pages

## 4. ONGLETS
| Onglet | Actif |
|--------|-------|
| Toutes les IP | - |
| IP Failover | - |
| Blocs RIPE | - |
| BYOIP | - |
| Firewall | ✓ |

## 6. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Adresse IP | Link | ✓ | IP concernée |

| Règles actives | Text | ✓ | Nombre de règles |
| Dernière modification | Date | ✓ | Timestamp |
| Statut | Badge | ✓ | Actif/Désactivé |
| Actions | Link | - | Configurer/Activer |

## 7. TOGGLE FIREWALL
| État | Style | Action label |
|------|-------|--------------|
| ON | Bleu (#0050D7) cercle droite | Configurer |
| OFF | Gris (#E5E7EB) cercle gauche | Activer |

## 10. APIS & ENDPOINTS
- GET /ip - Liste IP avec firewall
- GET /ip/{ip}/firewall - Statut firewall
- PUT /ip/{ip}/firewall/{ipOnFirewall} - Activer/désactiver
- GET /ip/{ip}/firewall/{ipOnFirewall}/rule - Liste règles
