# Cahier des Charges : bare-metal/ip - Liste des IPs

## 1. VUE D'ENSEMBLE
- **Page** : Gérer les IPs
- **Type** : Dashboard liste
- **Objectif** : Afficher et gérer tous les blocs IP

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb
- Titre + bouton Commander
- Bandeau alertes (badges)
- Onglets (IP / IP Load Balancing)
- Barre de filtre
- Tableau des IPs
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud › IP
- **OdsButton** : Commander des IPs (primary)
- **OdsBadge** : États alertes (success, warning)
- **OdsTabs** : IP, IP Load Balancing
- **OdsSelect** : Filtre type de service
- **OdsInput** : Recherche
- **OdsLink** : Exporter CSV
- **OdsDatagrid** : Tableau des IPs
- **OdsActionMenu** : Menu actions par ligne
- **OdsPagination** : Navigation résultats

## 4. BANDEAU ALERTES
| Badge | Couleur | Exemple |
|-------|---------|---------|
| Aucune alerte | Vert (success) | |
| 0 IP bloquée (Spam) | Jaune (warning) | |
| 0 IP bloquée (Anti-hack) | Jaune (warning) | |
| 0 IP attaquée (Mitigation) | Vert (success) | |

## 5. BARRE DE FILTRE
| Élément | Type | Valeur par défaut |
|---------|------|-------------------|
| Filtre par type de service | Select | Tous les types de service |
| Recherche | Input | Recherche... |
| Exporter CSV | Link | |

## 6. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Bloc IP | Link | Lien vers détail du bloc |
| Service associé | Text | Nom du serveur/service |
| Type | Text | Serveur dédié, VPS, HPC, etc. |
| Pays | Text + Flag | Emoji drapeau + nom pays |
| Reverse DNS | Text | Nom DNS ou "-" |
| Actions | ActionMenu | Menu ⋮ |

## 7. TYPES DE SERVICES
- Serveur dédié
- VPS
- Hosted Private Cloud
- Public Cloud
- Additional IP (parking)

## 8. ACTION MENU
| Option | Style |
|--------|-------|
| Voir détails | Normal |
| Configurer reverse | Normal |
| Configurer firewall | Normal |
| Déplacer | Normal |
| Supprimer | Rouge (danger) |

## 9. APIS & ENDPOINTS
- GET /ip
  - Liste des blocs IP
- GET /ip/{ip}
  - Détails d'un bloc IP
- GET /ip/{ip}/reverse
  - Reverse DNS
