# Cahier des Charges : network/ip - Modal Game Firewall Toggle

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation wizard 2 étapes
**Fonction:** Activer la stratégie "Refus par défaut" du GAME firewall
**Contexte:** Configuration de la politique de sécurité GAME

## 2. DIMENSIONS

- Modal: 550x300px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Activation de la stratégie « Refus par défaut »"
- Bouton fermer: ×

### Stepper (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Confirmation | Active |
| 2 | Traitement | Inactive |

### Description
- Ligne 1: "Tous les ports de jeu qui ont été définis sont protégés par le GAME firewall"
- Ligne 2: "et désormais, tout le reste du trafic sera supprimé."

### Question confirmation
- "Voulez-vous continuer?"
- Font-weight: 600

## 4. COMPOSANTS UI

### Stepper compact
- Cercle actif: bleu #0050D7
- Cercle inactif: blanc avec bordure grise

## 5. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Ferme le modal

### Bouton Valider
- Style: btn-primary (100x36px)
- Action: Active la stratégie, passe à l'étape 2

## 6. IMPACT DE L'ACTIVATION

- Tout trafic non explicitement autorisé sera bloqué
- Seuls les ports définis dans les règles GAME seront accessibles
- Protection maximale contre les attaques DDoS

## 7. APIS & ENDPOINTS

```
PUT /ip/{ip}/game
Body: { firewallModeEnabled: true }
→ Active la stratégie "Refus par défaut"

PUT /ip/{ip}/game
Body: { firewallModeEnabled: false }
→ Désactive la stratégie
```
