# Cahier des Charges : network/ip - Modal Export

## 1. VUE D'ENSEMBLE

**Type:** Modal wizard 2 étapes
**Fonction:** Exporter les adresses IP au format CSV
**Contexte:** Action depuis la page IP

## 2. DIMENSIONS

- Modal: 500x270px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #000E9C
- Titre: "Exporter mes adresses IP dans un fichier CSV" (14px, blanc, bold)
- Bouton fermer: × (18px, blanc)

## 4. STEPPER

| Étape | Nom | État |
|-------|-----|------|
| 1 | Confirmation | Actif (#0050D7) |
| 2 | Téléchargement | Inactif |

## 5. CONTENU

### Message
"L'export de vos IP dure quelques secondes" (13px)
"(en fonction du nombre d'IP que vous possédez)."

### Question
"Êtes-vous sûr de vouloir exporter les IP en format CSV ?" (13px)

## 6. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Exporter | btn-primary | 100x36px |

## 7. APIS & ENDPOINTS

```
# Exporter IP en CSV
POST /ip/export/csv
→ {
  taskId: "task-export-123",
  status: "pending"
}

# Télécharger export
GET /ip/export/{taskId}/download
→ Binary (CSV file)
```
