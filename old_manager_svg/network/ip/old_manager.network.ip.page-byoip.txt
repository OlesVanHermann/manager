# Cahier des Charges : network/ip - Page BYOIP

## 1. VUE D'ENSEMBLE
- **Type** : Page Bring Your Own IP
- **Contexte** : Network IP
- **Objectif** : Gérer les blocs IP RIPE/ARIN importés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Importer un bloc
- **OdsDatagrid** : Liste blocs IP
- **OdsBadge** : RIR, état
- **ActionMenu** : Menu ⋮

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bring Your Own IP (BYOIP)" |
| Description | "Importer vos propres blocs IP RIPE/ARIN." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Importer un bloc" |

## 10. MENU ACTIONS (⋮)
| Action | Style |
|--------|-------|
| Voir détails | Normal |
| Modifier ASN | Normal |
| Vérifier annonce | Normal |
| --- | Séparateur |
| Retirer bloc | Danger (#DC2626) |

## 11. APIS & ENDPOINTS
- GET /ip/byoip - Liste blocs BYOIP
- POST /ip/byoip - Importer bloc
- PUT /ip/byoip/{id} - Modifier
- DELETE /ip/byoip/{id} - Retirer bloc
- POST /ip/byoip/{id}/announce - Forcer annonce
