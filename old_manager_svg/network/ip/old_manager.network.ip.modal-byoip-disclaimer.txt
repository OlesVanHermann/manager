# Cahier des Charges : network/ip - Modal BYOIP Disclaimer

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation/information
**Fonction:** Déclaration de propriété BYOIP (Bring Your Own IP)
**Contexte:** Processus d'import d'IP externe

## 2. DIMENSIONS

- Modal: 550x400px
- Border radius: 8px

## 3. HEADER MODAL (TYPE INFO)

- Background: #EFF6FF
- Border: 1px solid #BFDBFE

- Bouton fermer: × (#1E40AF)

## 4. CONTENU

### Déclaration
"Je déclare :" (13px, bold)

### Points à accepter (12px)
1. "Être le propriétaire de la plage d'adresses IP renseignée."
2. "M'engager à faire en sorte que la plage d'adresses IP renseignée ne soit pas annoncée sur Internet par plusieurs réseaux simultanément (multi-homing)"
3. "Avoir une réputation saine sur cette plage d'adresses IP"

- Container: 490x50px

- Border: 1px solid #BFDBFE
- Texte: "Veuillez noter que ce processus peut prendre approximativement 3 semaines. Nous vous prions de nous excuser pour ce désagrément."
