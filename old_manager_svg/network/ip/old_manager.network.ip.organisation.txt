# Cahier des Charges : network/ip - Organisation

## 1. VUE D'ENSEMBLE

**Type:** Page listing avec tableau expansible
**Fonction:** Gérer les organisations WHOIS pour les blocs IP
**Contexte:** Gestion des déclarations RIPE/ARIN/APNIC

## 2. DIMENSIONS

- Page: 1200x700px
- Tableau: 1100x380px

## 3. LAYOUT & STRUCTURE

### Header OVHcloud
- Background: #000E9C
- Logo OVHcloud

### Breadcrumb
Network > IP > Gérer mes organisations

### Lien retour
"← IP" vers listing IP principal

### Titre page
"Gérer mes organisations"

### Bouton action principal
"+ Ajouter une organisation"

## 7. ACTIONS & INTERACTIONS

### Ajouter organisation
- Click: ouvre modal-organisation-add
- Formulaire wizard 2 étapes

### Expand/Collapse ligne
- Click sur chevron: toggle les détails
- Affiche téléphone et adresse

### Menu actions
- Modifier: ouvre modal d'édition

## 8. APIS & ENDPOINTS

```
GET /me/ipOrganisation
→ Liste toutes les organisations

GET /me/ipOrganisation/{organisationId}
→ Détails d'une organisation

POST /me/ipOrganisation
→ Crée une nouvelle organisation

PUT /me/ipOrganisation/{organisationId}
→ Modifie une organisation

DELETE /me/ipOrganisation/{organisationId}
→ Supprime une organisation
```
