# Cahier des Charges : network/ip - Modal Import SYS

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard multi-étapes
- **Déclencheur** : Action "Importer depuis SYS"
- **Objectif** : Migrer des IP de l'infrastructure SYS vers OVHcloud

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x520px)
- Header avec titre, indicateur étape et bouton fermer
- Indicateur d'étapes (dots)
- Formulaire étape 1
- Footer avec boutons navigation

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsStepper** : Indicateur 5 étapes (dots)
- **OdsInput** : IP ou bloc IP, Token
- **OdsText** : Instructions et labels
- **OdsButton** : Annuler/Suivant

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Importer mes adresses IP de SYS vers OVHcloud" |
| Indicateur | "Étape 1/5" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. INDICATEUR D'ÉTAPES
| Propriété | Valeur |
|-----------|--------|
| Type | Dots reliés par lignes |
| Étape actuelle | Bleu (#0050D7) |
| Étapes futures | Gris (#D1D5DB) |
| Nombre d'étapes | 5 |

### Instructions

## 7. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Suivant → | Primary (bleu) | Droite |

## 8. ÉTAPES DU WIZARD
| Étape | Description |
|-------|-------------|
| 1 | Saisie IP et token |
| 2 | Vérification du token |
| 3 | Choix du service destination |
| 4 | Résumé de la migration |
| 5 | Confirmation et import |

## 9. APIS & ENDPOINTS
- POST /ip/{ip}/migrate/tokenVerify - Vérifier le token
- POST /ip/{ip}/migrate - Lancer la migration
