# Cahier des Charges : network/ip - Modal Import SYS

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard multi-étapes
- **Contexte** : Gestion IP - Migration
- **Objectif** : Importer des IP depuis SYS vers OVHcloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 600x520px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : Indicateur d'étapes (dots)
- **OdsInput** : Champs IP et token
- **OdsButton** : Navigation

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Importer mes adresses IP de SYS vers OVHcloud" |
| Indicateur étape | "Étape X/5" |
| Bouton fermer | × |

## 5. INDICATEUR ÉTAPES
| Propriété | Valeur |
|-----------|--------|
| Nombre étapes | 5 |
| Dot actif | #0050D7 |
| Dot inactif | #D1D5DB |

## 6. CHAMPS FORMULAIRE
| Champ | Placeholder |
|-------|-------------|
| IP ou block IP | "Ex: 192.168.1.0/24" |
| Token | "Entrez le token fourni par SYS" |

## 7. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Suivant → | Primary |

## 8. APIS & ENDPOINTS
- POST /ip/import/sys/validate - Valider token
- POST /ip/import/sys - Lancer import
