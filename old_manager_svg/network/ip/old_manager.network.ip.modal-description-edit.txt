# Cahier des Charges : network/ip - Modal Description Edit

## 1. VUE D'ENSEMBLE

**Type:** Modal wizard 2 étapes
**Fonction:** Éditer la description d'une IP
**Contexte:** Action depuis la liste IP

## 2. DIMENSIONS

- Modal: 500x300px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #000E9C
- Titre: "Éditer la description" (15px, blanc, bold)
- Bouton fermer: × (18px, blanc)

## 4. STEPPER

| Étape | Nom | État |
|-------|-----|------|
| 1 | Saisie | Actif (#0050D7) |
| 2 | Confirmation | Inactif |

## 5. CONTENU

### Champ Description
- Label: "Description :" (13px, bold)
- Textarea: 450x80px
- Placeholder/Valeur: "Serveur de production - Mail"

### Compteur caractères
- Position: Aligné à droite
- Texte: "255 caractères max" (11px, muted)

## 6. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Valider | btn-primary | 100x36px |

## 7. APIS & ENDPOINTS

```
# Modifier description IP
PUT /ip/{ip}/description
Body: {
  description: "Serveur de production - Mail"
}
→ {
  ip: "51.178.42.123",
  description: "Serveur de production - Mail"
}
```
