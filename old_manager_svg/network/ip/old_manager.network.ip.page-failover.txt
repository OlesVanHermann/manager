# Cahier des Charges : network/ip - Page Failover

## 1. VUE D'ENSEMBLE
- **Type** : Page listing IP Failover
- **Contexte** : Gestion IP réseau
- **Objectif** : Gérer les IP publiques routables

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Commander, Importer
- **OdsDatagrid** : Liste des IP
- **OdsBadge** : État IP
- **OdsPopover** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Failover IP" |
| Description | "IP publiques routables entre vos services." |

## 5. BOUTONS ACTIONS
| Bouton | Style |
|--------|-------|
| + Commander IP | Primary |
| Importer IP | Outline |

## 6. COLONNES TABLEAU
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Adresse IP | 180px | IP (bold) |
| Pays | 80px | Drapeau + code |
| Reverse DNS | 220px | Nom ou "Non configuré" |
| Service attaché | 220px | Nom service + DC |
| État | 130px | Badge avec dot |
| Actions | 160px | Bouton + menu ⋮ |

## 7. ÉTATS IP
| État | Couleur dot | Texte |
|------|-------------|-------|
| Attachée | #16A34A (vert) | "Attachée" |
| Libre | #6B7280 (gris) | "Libre" |

## 8. BOUTONS PAR LIGNE
| État IP | Bouton principal |
|---------|------------------|
| Attachée | "Déplacer" (outline bleu) |
| Libre | "Attacher" (primary) |

## 9. MENU ACTIONS (⋮)
| Action | Style |
|--------|-------|
| Modifier reverse | Normal |
| Configurer firewall | Normal |
| Gérer mitigation | Normal |
| --- | Séparateur |
| Résilier | Danger (#DC2626) |

## 10. APIS & ENDPOINTS
- GET /ip/failover - Liste IP failover
- POST /ip/failover/order - Commander IP
- POST /ip/{ip}/attach - Attacher à service
- POST /ip/{ip}/move - Déplacer IP
