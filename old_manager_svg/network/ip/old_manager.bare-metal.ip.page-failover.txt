# Cahier des Charges : bare-metal/ip - Page IP Failover

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **Contexte** : Gestion des IPs Failover
- **Objectif** : Gérer les adresses IP Failover assignées aux serveurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsMessage** : Bannière d'alertes
- **OdsToggle** : Filtre IPs non utilisées
- **OdsDatagrid** : Tableau des IPs
- **OdsBadge** : Statut
- **OdsActionMenu** : Actions par ligne
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "IP Failover" |
| Fond | Blanc avec bordure |

## 5. BANNIÈRE D'ALERTES
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Texte | "{N} alertes détectées sur vos adresses IP." |
| Lien | "Voir les détails ▼" (bleu, souligné) |

## 6. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Gérez vos adresses IP Failover assignées à vos serveurs." |

## 7. FILTRE TOGGLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Afficher uniquement les IPs non utilisées" |
| Type | OdsToggle |
| État par défaut | Désactivé |

## 8. TABLEAU DES IPS FAILOVER

### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Adresse IP | Texte | Adresse IPv4 |
| Service associé | Lien/Texte | Nom du serveur ou "Non assignée" |
| Reverse | Texte | Enregistrement reverse ou "-" |
| Statut | Badge | Actif, Non utilisée |
| Actions | Menu | ⋮ |

### Badge Statut Actif
| Propriété | Valeur |
|-----------|--------|
| Fond | #DCFCE7 |
| Texte | #166534 |
| Label | "Actif" |

### Badge Statut Non utilisée
| Propriété | Valeur |
|-----------|--------|
| Fond | #F3F4F6 |
| Texte | #6B7280 |
| Label | "Non utilisée" |

### Ligne Non assignée
| Propriété | Valeur |
|-----------|--------|
| Service | "Non assignée" (gris #9CA3AF) |
| Reverse | "-" (gris) |

## 9. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "{count} IPs sur {total}" |

## 10. APIS & ENDPOINTS
- GET /ip?type=failover - Liste des IPs Failover
- GET /ip/alerts - Alertes sur les IPs
