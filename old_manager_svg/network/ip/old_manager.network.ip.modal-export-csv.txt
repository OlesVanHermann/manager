# Cahier des Charges : network/ip - Modal Export CSV

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Déclencheur** : Bouton "Exporter CSV" ou action menu
- **Objectif** : Exporter la liste des IP en fichier CSV

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée compacte (500x300px)
- Header avec titre et bouton fermer
- Message d'information
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Messages explicatifs
- **OdsButton** : Annuler/Valider

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Exporter mes adresses IP dans un fichier CSV" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. CONTENU
| Ligne | Texte |
|-------|-------|
| L1 | "L'export de vos IP dure quelques secondes" |
| L2 | "(en fonction du nombre d'IP que vous possédez)." |
| L3 | "Êtes-vous sûr de vouloir exporter les IP en format CSV ?" |

## 6. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Valider | Primary (bleu) | Droite |

## 7. COLONNES CSV EXPORTÉES
| Colonne | Description |
|---------|-------------|
| IP | Adresse IP |
| Type | additional/failover/byoip |
| Service | Service associé |
| Reverse DNS | PTR record |
| Description | Description personnalisée |
| Datacenter | Localisation |
| Mitigation | État Anti-DDoS |
| Firewall | État firewall |

## 8. COMPORTEMENT EXPORT
| Étape | Action |
|-------|--------|
| 1 | Clic Valider |
| 2 | Spinner loading |
| 3 | Génération côté serveur |
| 4 | Téléchargement automatique |
| 5 | Fermeture modal |

## 9. FORMAT FICHIER
| Propriété | Valeur |
|-----------|--------|
| Format | CSV (Comma Separated Values) |
| Encodage | UTF-8 |
| Séparateur | Virgule (,) |
| Nom fichier | ip_export_YYYY-MM-DD.csv |

## 10. APIS & ENDPOINTS
- GET /ip/export - Déclencher l'export CSV
- GET /ip/export/{taskId} - Récupérer le fichier généré
