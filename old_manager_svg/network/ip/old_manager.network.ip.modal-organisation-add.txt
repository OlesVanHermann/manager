# Cahier des Charges : network/ip - Modal Organisation Add

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire wizard 2 étapes
**Fonction:** Déclarer une nouvelle organisation pour les blocs IP (WHOIS)
**Contexte:** Gestion des informations RIPE/ARIN/APNIC

## 2. DIMENSIONS

- Modal: 600x700px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Ajout d'une organisation"
- Bouton fermer: ×

### Stepper (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Formulaire | Active |
| 2 | Confirmation | Inactive |

### Message info
- Background: #EFF6FF
- Border: #BFDBFE
- Texte: "Veuillez remplir ce formulaire afin de déclarer une nouvelle organisation :"

## 4. FORMULAIRE

| Champ | Type | Largeur | Exemple |
|-------|------|---------|---------|
| Type | Select | 540px | RIPE |
| Prénom | Input | 540px | John |
| Nom | Input | 540px | Doe |
| Pays | Select | 540px | France |
| Email | Input | 540px | abuse@example.com |
| Adresse | Input | 540px | 2 rue Kellermann |
| Ville | Input | 350px | Roubaix |
| Code postal | Input | 170px | 59100 |
| Téléphone | Input | 540px | +33 1 23 45 67 89 |

### Options Type
- RIPE (Europe)
- ARIN (Amérique du Nord)
- APNIC (Asie-Pacifique)

## 5. COMPOSANTS UI

### Champs input
- Border: #E5E7EB
- Border-radius: 4px
- Hauteur: 36px

### Ligne Ville/Code postal
- 2 colonnes: 350px + 170px

## 6. ACTIONS & INTERACTIONS

### Validation formulaire
- Tous les champs requis
- Email: format valide
- Téléphone: format international

### Bouton Annuler
- Style: btn-secondary (100x36px)

### Bouton Suivant
- Style: btn-primary (100x36px)
- Action: Valide et passe à l'étape 2

## 7. APIS & ENDPOINTS

```
GET /me/ipOrganisation
→ Liste les organisations existantes

POST /me/ipOrganisation
Body: {
  type: "RIPE",
  firstname: "John",
  lastname: "Doe",
  country: "FR",
  email: "abuse@example.com",
  address: "2 rue Kellermann",
  city: "Roubaix",
  zip: "59100",
  phone: "+33123456789"
}
→ Crée une nouvelle organisation
```
