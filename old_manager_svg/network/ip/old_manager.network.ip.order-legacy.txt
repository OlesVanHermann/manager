# Cahier des Charges : network/ip - Order Legacy

## 1. VUE D'ENSEMBLE

**Type:** Modal wizard 4 étapes
**Fonction:** Commander des Additional IP (interface legacy)
**Contexte:** Ancien formulaire de commande IP dans modal

## 2. DIMENSIONS

- Modal: 550x600px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Commander Additional IP"
- Bouton fermer: ×

### Breadcrumb steps
1. Service
2. Configuration
3. Durée
4. Résumé

Format: "1. Service > 2. Configuration > 3. Durée > 4. Résumé"

## 4. ÉTAPE 1 - SÉLECTION SERVICE

### Description
"Sélectionnez le service pour lequel vous désirez acheter des Additional IP"

### Champ Service
- Label: "Service"
- Type: Select dropdown
- Largeur: 490px
- Exemple: "ns123456.ip-51-178-0.eu"

### Informations capacité
"Votre serveur peut supporter jusqu'à :"
- 256 IP de blocs Additional IP
- 16 Additional IP individuelles (blocs de 1 IP)

### Message succès
- Couleur: #16A34A (vert)
- Texte: "Cliquez sur 'Suivant' pour accéder à la commande."

## 5. RÉSUMÉ COMMANDE

### Titre section
"Résumé de votre commande"

### Détail commande
| Élément | Valeur |
|---------|--------|
| Additional IP x 5 | 10,00 EUR |
| Prix total HT | 10,00 EUR |
| Prix total TTC | 12,00 EUR |

### Styles prix
- Prix HT/TTC: text-danger (#DC2626), font-weight: 600

### Message validation
"Cliquez sur le bouton 'Valider' afin de confirmer votre commande."
"Un nouvel onglet s'ouvrira et vous serez automatiquement redirigé(e) vers l'interface de paiement."

## 6. COMPOSANTS UI

### Select service
- Height: 40px
- Border: #E5E7EB
- Chevron dropdown

### Liste capacité
- Bullets: astérisques (*)
- Font-size: 12px
- Couleur: text-muted

### Card résumé
- Dimensions: 490x150px
- Border: #E5E7EB
- Séparateur horizontal

## 7. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Ferme le modal

### Bouton Précédent
- Style: btn-secondary (100x36px)
- Visible: étapes 2, 3, 4

### Bouton Suivant
- Style: btn-primary (100x36px)
- Action: Passe à l'étape suivante

### Bouton Valider
- Style: btn-primary (100x36px)
- Visible: étape 4 (résumé)
- Action: Crée le bon de commande, redirige vers paiement

## 8. APIS & ENDPOINTS

```
GET /dedicated/server
→ Liste des serveurs éligibles

GET /dedicated/server/{serviceName}/specifications/ip
→ Capacité IP du serveur

POST /order/dedicated/server/{serviceName}/ip
Body: {
  blockSize: 1,
  quantity: 5,
  country: "fr"
}
→ Commande les Additional IP

GET /order/dedicated/server/{serviceName}/ip/{duration}
→ Prix selon durée d'engagement
```
