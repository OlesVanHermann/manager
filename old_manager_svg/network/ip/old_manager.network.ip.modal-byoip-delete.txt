# Cahier des Charges : network/ip - Modal BYOIP Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal informatif (warning)
**Fonction:** Informer que la suppression BYOIP se fait via la résiliation
**Contexte:** Tentative de suppression d'un bloc BYOIP

## 2. DIMENSIONS

- Modal: 500x230px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal (warning style)
- Background: #FEF3C7
- Border: #FDE68A
- Icône: ⚠ (cercle orange avec !)
- Titre: "Supprimer le bloc"
- Bouton fermer: × (couleur warning)

### Contenu
- Texte explicatif sur 3 lignes:
  - "Ce bloc d'IP fait partie d'un plus grand bloc BYOIP."
  - "Veuillez vous rendre sur [mes services] et choisir « résilier »"
  - "si vous souhaitez annuler votre service BYOIP."

### Lien
- "mes services" → lien vers la page de gestion des services

## 4. COMPOSANTS UI

### Header warning
- Icône cercle orange avec point d'exclamation
- Texte couleur: #92400E

### Lien inline
- Couleur: text-link #0050D7
- Souligné au hover

## 5. ACTIONS & INTERACTIONS

### Lien "mes services"
- Redirige vers la page de gestion des services

### Bouton Fermer
- Style: btn-secondary (100x32px)
- Position: footer droite
- Action: Ferme le modal

## 6. APIS & ENDPOINTS

```
(Aucun endpoint - modal informatif uniquement)

Pour résilier:
DELETE /services/{serviceId}/terminate
```
