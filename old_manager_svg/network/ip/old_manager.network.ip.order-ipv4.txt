## 6. ACTIONS & INTERACTIONS

### Bouton Précédent
- Style: btn-secondary (120x40px)
- Action: Retour à l'étape 1

### Bouton Suivant
- Style: btn-primary (120x40px)
- Action: Vers étape 3 (Localisation)

### Sélection offre
- Click sur carte: sélectionne l'offre
- Recalcul prix dynamique

### Modification quantité
- Click [-]: décrémente (min 1)
- Click [+]: incrémente (max selon quota)

## 7. APIS & ENDPOINTS

```
GET /order/cart
→ Récupère le panier en cours

POST /order/cart/{cartId}/ip
Body: {
  country: "fr",
  offer: "additionalIp",
  quantity: 5
}
→ Ajoute des IP au panier

GET /ip/service/{serviceName}/availableQuantity
→ Quota disponible pour le service

GET /order/catalog/public/ip
→ Catalogue des offres IP avec prix
```
