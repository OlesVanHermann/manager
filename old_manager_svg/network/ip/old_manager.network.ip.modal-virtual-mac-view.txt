# Cahier des Charges : network/ip - Modal Virtual MAC View

## 1. VUE D'ENSEMBLE

**Type:** Modal information
**Fonction:** Afficher les détails d'une adresse MAC virtuelle
**Contexte:** Action depuis la gestion des MAC virtuelles

## 2. DIMENSIONS

- Modal: 450x300px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #000E9C
- Titre: "Adresse MAC virtuelle" (15px, blanc, bold)
- Bouton fermer: × (18px, blanc)

## 4. STEPPER

- 1 étape active (#0050D7)

## 5. CONTENU

### Instruction
"Voici les informations de l'adresse MAC virtuelle :" (12px, muted)

### Champs affichés

| Label | Valeur exemple | Position |
|-------|----------------|----------|
| Adresse IP | 51.178.42.123 | x=200 |
| Adresse MAC | 02:00:00:a1:b2:c3 | x=200 |
| Type | ovh | x=200 |
| Nom de la machine | vm-production-01 | x=200 |

### Style labels
- Police: 12px, #6B7280, bold
- Espacement vertical: 30px

## 6. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Fermer | btn-primary | 100x36px |

## 7. APIS & ENDPOINTS

```
# Détails MAC virtuelle
GET /ip/{ip}/virtualMac/{macAddress}
→ {
  ip: "51.178.42.123",
  macAddress: "02:00:00:a1:b2:c3",
  type: "ovh",
  virtualMachineName: "vm-production-01"
}
```
