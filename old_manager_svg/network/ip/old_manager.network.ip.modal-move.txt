# Cahier des Charges : network/ip - Modal Move

## 1. VUE D'ENSEMBLE
- **Type** : Modal déplacement IP
- **Contexte** : Gestion IP Failover
- **Objectif** : Déplacer une IP failover vers un autre service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 600x480px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning interruption
- **OdsInput** : Champs readonly
- **OdsSelect** : Service destination (dropdown ouvert)
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Déplacer l'IP" |
| Bouton fermer | ✕ |

## 5. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Titre | "⚠️ Attention" |
| Message | Courte interruption de service |

## 6. CHAMPS FORMULAIRE
| Champ | Type | État |
|-------|------|------|
| IP à déplacer | Input | Readonly (fond gris) |
| Service actuel | Input | Readonly (fond gris) |
| Service de destination | Select | Éditable |

## 7. DROPDOWN OPTIONS
| Option | Style |
|--------|-------|
| Sélectionnée | Fond #E8F0FE, texte #0050D7 |
| Non sélectionnée | Fond blanc, texte #1F2937 |

## 8. BOUTONS FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Déplacer | Primary |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/destination - Services disponibles
- POST /ip/{ip}/move - Déplacer IP
