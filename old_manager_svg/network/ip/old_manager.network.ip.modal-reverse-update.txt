# Cahier des Charges : network/ip - Modal Reverse Update

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire simple
**Fonction:** Modifier le reverse DNS d'une adresse IP
**Contexte:** Gestion DNS inverse pour IPv4/IPv6

## 2. DIMENSIONS

- Modal: 500x330px
- Overlay: fond noir 50%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #000E9C
- Titre: "Modifier le reverse"
- Bouton fermer: ×

### Message info
- Background: #EFF6FF
- Border: #BFDBFE
- Texte: "Insérez un reverse valide pour cette IP et cliquez sur 'Valider'."

## 4. FORMULAIRE

| Champ | Type | État | Valeur exemple |
|-------|------|------|----------------|
| Adresse IP | Input | Disabled | 51.178.42.123 |
| Reverse DNS | Input | Editable | mail.example.com |

### Champ Adresse IP
- Background: #F3F4F6 (disabled)
- Affiche l'IP en lecture seule

### Champ Reverse DNS
- Border: #0050D7, 2px (focus)
- FQDN à configurer

## 5. COMPOSANTS UI

### Input disabled
- Background: #F3F4F6
- Text color: text-muted

### Input focus
- Border: #0050D7, 2px
- Indique le champ actif

## 6. ACTIONS & INTERACTIONS

### Validation Reverse
- Format FQDN valide
- Résolution DNS vers l'IP requise

### Bouton Annuler
- Style: btn-secondary (100x36px)

### Bouton Valider
- Style: btn-primary (100x36px)
- Action: Met à jour le reverse

## 7. APIS & ENDPOINTS

```
GET /ip/{ip}/reverse
→ Récupère le reverse actuel

PUT /ip/{ip}/reverse
Body: {
  reverse: "mail.example.com"
}
→ Met à jour le reverse DNS
```
