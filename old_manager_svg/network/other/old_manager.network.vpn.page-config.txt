# Cahier des Charges : network/vpn - Page Config

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration VPN
- **Contexte** : Network VPN
- **Objectif** : Configurer les paramètres IPSec du tunnel

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Sections Phase 1, Phase 2, Réseaux
- **OdsSelect** : Algorithmes, groupes
- **OdsInput** : Réseaux CIDR
- **OdsCheckbox** : Options
- **OdsButton** : Sauvegarder

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration VPN" |
| Sous-titre | "vpn-aws-prod - Paramètres IPSec" |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sauvegarder" |
| Style | Primary #0050D7 |
| Position | Haut droit |

## 6. SECTION PHASE 1 (IKE)
| Paramètre | Options |
|-----------|---------|
| Encryption | AES-128, AES-256, 3DES |
| Hash | SHA-1, SHA-256, SHA-384 |
| DH Group | Group 2, 14, 19, 20 |
| Lifetime | 28800s (8h) par défaut |

## 7. SECTION PHASE 2 (ESP)
| Paramètre | Options |
|-----------|---------|
| Encryption | AES-128, AES-256, 3DES |
| Hash | SHA-1, SHA-256, SHA-384 |
| PFS Group | Disabled, Group 2, 14, 19 |
| Lifetime | 3600s (1h) par défaut |

## 8. SECTION RÉSEAUX
| Champ | Exemple |
|-------|---------|
| Réseau local | 192.168.0.0/16 |
| Réseau distant | 10.0.0.0/8 |

## 9. OPTIONS AVANCÉES
| Option | État défaut |
|--------|-------------|
| NAT-Traversal | Activé |
| Dead Peer Detection | Activé |
| Force UDP encapsulation | Désactivé |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/ipsec/{tunnelId} - Config tunnel
- PUT /ip/{ip}/ipsec/{tunnelId} - Modifier config
- PUT /ip/{ip}/ipsec/{tunnelId}/phase1 - Config Phase 1
- PUT /ip/{ip}/ipsec/{tunnelId}/phase2 - Config Phase 2
