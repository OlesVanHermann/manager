# Cahier des Charges : network/vpn - Page Logs

## 1. VUE D'ENSEMBLE
- **Type** : Page logs VPN
- **Contexte** : Network VPN
- **Objectif** : Visualiser l'historique des connexions VPN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsInput** : Recherche IP
- **OdsSelect** : Filtres (événement, période)
- **OdsButton** : Exporter
- **Terminal** : Affichage logs (fond sombre)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Logs VPN" |
| Sous-titre | "vpn-paris-01 - Historique des connexions." |

## 5. FILTRES
| Filtre | Largeur | Options |
|--------|---------|---------|
| Recherche IP | 200px | Input |
| Événements | 150px | Tous, Connected, Failed |
| Période | 140px | 1h, 24h, 7j |
| Exporter | 100px | Bouton outline |

## 6. TERMINAL LOGS
| Propriété | Valeur |
|-----------|--------|
| Fond | #1F2937 |
| Largeur | 1040px |
| Hauteur | 370px |
| Police | Monospace, 10px |

| Info | Exemple |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/ipsec/{tunnelId}/log - Logs tunnel
- GET /ip/{ip}/ipsec/{tunnelId}/statistics - Statistiques
