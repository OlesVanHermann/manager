# Cahier des Charges : network/vpn - Modal Peer

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout peer VPN
- **Contexte** : Network VPN
- **Objectif** : Configurer un tunnel VPN site-to-site

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 540px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et helper text
- **OdsInput** : Nom, IP, réseaux, PSK
- **Cards sélection** : Version IKE
- **OdsButton** : Générer PSK, Annuler, Ajouter
- **OdsMessage** : Info configuration
