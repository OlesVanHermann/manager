# Cahier des Charges : network/vrack - Modal ajouter sous-réseau IPv6

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page IPv6 vRack
- **Objectif** : Ajouter un sous-réseau IPv6 routé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description
- **OdsInput** : Sous-réseau, Adresse passerelle
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un sous-réseau" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Configurez un nouveau sous-réseau IPv6 routé." |

## 6. FORMULAIRE

### Champs
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Sous-réseau * | Input | 2001:db8:0:1::/64 | Oui |
| Adresse passerelle * | Input | 2001:db8::1 | Oui |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary | Crée le sous-réseau |

## 8. APIS & ENDPOINTS
- POST /vrack/{serviceName}/ipv6/{ipv6}/routedSubrange - Ajouter sous-réseau routé
