# Cahier des Charges : network/vrack - Page Topology

## 1. VUE D'ENSEMBLE
- **Type** : Page vue topologie vRack
- **Contexte** : Network vRack
- **Objectif** : Visualiser graphiquement les ressources connectées au vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Zone graphique
- **Schéma graphique** : Nœuds et connexions SVG
- **Légende** : Types de ressources

## 12. APIS & ENDPOINTS
- GET /vrack/{serviceName}/service - Liste services connectés
- GET /vrack/{serviceName}/cloudProject - Projets Public Cloud
- GET /vrack/{serviceName}/dedicatedServer - Serveurs dédiés
- GET /vrack/{serviceName}/kubernetes - Clusters K8s
- GET /vrack/{serviceName}/ipLoadbalancing - Load Balancers
