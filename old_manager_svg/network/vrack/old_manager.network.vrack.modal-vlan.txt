# Cahier des Charges : network/vrack - Modal VLAN

## 1. VUE D'ENSEMBLE
- **Type** : Modal création VLAN
- **Contexte** : Network vRack
- **Objectif** : Créer un VLAN dans le vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et helper text
- **OdsInput** : Nom, VLAN ID, sous-réseau
- **OdsSelect** : Région
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Créer un VLAN" |
| Bouton fermer | × en #6B7280 |

## 10. APIS & ENDPOINTS
- POST /vrack/{serviceName}/vlan - Créer VLAN
- GET /vrack/{serviceName}/vlan - Liste VLANs
- DELETE /vrack/{serviceName}/vlan/{vlanId} - Supprimer VLAN
