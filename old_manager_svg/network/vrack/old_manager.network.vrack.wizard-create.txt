## 7. COMPORTEMENT

| Action | Résultat |
|--------|----------|
| Clic Annuler | Retour à la liste des vRack |
| Clic lien retour | Retour à la liste des vRack |
| Clic lien contrat | Ouvrir le contrat (nouvel onglet) |
| Clic checkbox | Toggle acceptation conditions |
| Clic Commander | Commander le vRack (si conditions acceptées) |

## 9. APIS & ENDPOINTS

### 9.1 Commander un vRack

```
POST /order/vrack/new
```

Body:
```json
{
  "acceptedContracts": true
}
```

Response:
```json
{
  "orderId": "order-123456",
  "serviceName": "pn-987654",
  "status": "pending"
}
```
