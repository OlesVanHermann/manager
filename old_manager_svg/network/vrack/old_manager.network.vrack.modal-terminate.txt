# Cahier des Charges : network/vrack - Modal Résiliation

## 1. VUE D'ENSEMBLE
- **Modal** : Résiliation du vRack
- **Déclencheur** : Action "Résilier" dans ActionMenu
- **Objectif** : Confirmer la résiliation du vRack

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 420px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsIcon** : Warning (triangle rouge)
- **OdsMessage** : Bandeau erreur si services attachés
- **OdsText** : Message confirmation
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler / Résilier

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Résilier le vRack" |
| Body | Message + conditions |
| Input | Confirmation par saisie |
| Footer | Boutons Annuler / Résilier |

## 5. CAS NORMAL (aucun service)
| Élément | Valeur |
|---------|--------|
| Titre | "Résilier le vRack" |
| Message | "Êtes-vous sûr(e) de vouloir résilier ce vRack ?" |
| Confirmation | Saisir "RESILIER" pour confirmer |

## 6. CAS ERREUR (services attachés)
| Élément | Valeur |
|---------|--------|
| Bandeau | Error (rouge) |
| Message | "Ce vRack contient encore X services" |
| Action | "Veuillez retirer tous les services avant de résilier" |
| Bouton | Désactivé tant que services présents |

## 7. BOUTONS
| Bouton | Style | Condition |
|--------|-------|-----------|
| Annuler | Secondary | Toujours actif |
| Résilier | Danger | Actif si aucun service |

## 8. APIS & ENDPOINTS
- GET /vrack/{serviceName} - Vérifier services attachés
- DELETE /vrack/{serviceName}/terminate - Résilier vRack
