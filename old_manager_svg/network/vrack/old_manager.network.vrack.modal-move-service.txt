# Cahier des Charges : network/vrack - Modal Déplacer service

## 1. VUE D'ENSEMBLE
- **Modal** : Déplacement d'un service Private Cloud
- **Déclencheur** : Action "Déplacer" sur datacenter Private Cloud
- **Objectif** : Transférer un datacenter vers un autre vRack

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 450px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Explication
- **OdsMessage** : Warning interruption service
- **OdsSelect** : Sélecteur vRack destination
- **OdsButton** : Annuler / Déplacer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Déplacement d'un datacentre Private Cloud" |
| Explication | Texte règles |
| Warning | Interruption 5 min |
| Formulaire | Sélecteur destination |
| Footer | Boutons |

## 5. TEXTE EXPLICATIF
| Ligne | Contenu |
|-------|---------|
| 1 | "Vous avez sélectionné un service de type datacentre Private Cloud." |
| 2 | "Ce service ne peut être orphelin et doit être transféré obligatoirement vers un autre vRack." |

## 6. MESSAGE WARNING
| Propriété | Valeur |
|-----------|--------|
| Style | Warning (jaune #FEF3C7) |
| Icône | ⚠️ |
| Texte | "Cette opération entraînera une interruption de service d'environ 5 minutes." |

## 7. SÉLECTEUR VRACK
| Propriété | Valeur |
|-----------|--------|
| Label | "vRack de destination :" |
| Type | OdsSelect |
| Placeholder | "Sélectionnez un vRack" |
| Options | Liste des vRack disponibles |
| Format option | "pn-XXXXXX - Description" |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary outline | Fermer modal |
| Déplacer | Primary | Exécuter déplacement |

## 9. APIS & ENDPOINTS
- GET /vrack - Liste vRack disponibles
- POST /vrack/{serviceName}/dedicatedCloud/{dcId}/move - Déplacer datacenter
- GET /vrack/{serviceName}/task - Suivre tâche
