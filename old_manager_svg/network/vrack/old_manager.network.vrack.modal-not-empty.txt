# Cahier des Charges : network/vrack - Modal impossible de résilier

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information warning
- **Contexte** : Dashboard vRack
- **Objectif** : Informer que le vRack ne peut pas être résilié

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning avec icône
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résilier {vrackName}" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ Afin de résilier votre vRack vous devez au préalable déconnecter tous les services rattachés." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Warning (orange) | Ferme modal |

## 7. COMPORTEMENT
- Ce modal s'affiche lorsque l'utilisateur tente de résilier un vRack non vide
- Seul bouton disponible : Fermer
- L'utilisateur doit d'abord retirer tous les services du vRack

## 8. APIS & ENDPOINTS
- GET /vrack/{serviceName}/allowedServices - Vérifier services attachés
