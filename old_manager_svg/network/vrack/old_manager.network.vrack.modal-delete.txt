# Cahier des Charges : network/vrack - Modal Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal de confirmation destructive
**Fonction:** Résilier un vRack
**Contexte:** Suppression définitive d'un réseau privé

## 2. DIMENSIONS

- Modal: 600x400px
- Overlay: fond noir 40%

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #FEE2E2 (danger light)
- Titre: "Résilier le vRack"
- Couleur titre: #DC2626
- Bouton fermer: × (rouge)
- Height: 60px

### Contenu
- Description: "Vous êtes sur le point de résilier le vRack suivant :"

## 4. INFORMATIONS vRACK

### Card info
- Background: #F9FAFB
- Dimensions: 550x80px
- Border-radius: 4px

### Champs affichés
| Label | Valeur exemple |
|-------|----------------|
| Nom | Production Network |
| ID | pn-123456 |
| Services associés | 12 |

## 5. MESSAGE D'AVERTISSEMENT

### Warning box
- Background: #FEF3C7
- Border-left: 4px #F59E0B
- Dimensions: 550x90px

### Contenu warning
- Titre: "Attention" (font-weight 600)
- Ligne 1: "Cette action est irréversible. Tous les services associés"
- Ligne 2: "seront déconnectés du réseau privé."

## 6. CHECKBOX CONFIRMATION

### Checkbox obligatoire
- Label: "Je comprends que cette action est irréversible"
- Must be checked pour activer le bouton Résilier

### Style checkbox
- Size: 20x20px
- Unchecked: border #E5E7EB
- Checked: bg #DC2626 avec ✓

## 7. ACTIONS & INTERACTIONS

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Ferme le modal

### Bouton Résilier
- Style: btn-danger (110x36px)
- Background: #DC2626
- État désactivé si checkbox non cochée
- Action: Lance la résiliation

### Validation
- Checkbox doit être cochée
- Confirmation de l'action destructive

## 8. APIS & ENDPOINTS

```
DELETE /vrack/{serviceName}
→ Résilie le vRack

GET /vrack/{serviceName}/serviceInfos
→ Informations de service pour résiliation

POST /vrack/{serviceName}/terminate
→ Alternative: demande de résiliation
```
