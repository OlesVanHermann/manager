## 7. ACTIONS & INTERACTIONS

### Commander
- Click: redirige vers page de commande vRack

### Recherche
- Filtre en temps réel sur nom, ID, description

### Clic sur nom
- Lien vers page détail du vRack

### Menu actions
- Gérer: page de configuration du vRack
- Résilier: ouvre modal de confirmation

## 8. APIS & ENDPOINTS

```
GET /vrack
→ Liste tous les vRacks

GET /vrack/{serviceName}
→ Détails d'un vRack

GET /vrack/{serviceName}/allowedServices
→ Services pouvant être ajoutés

GET /vrack/{serviceName}/dedicatedServer
→ Serveurs dédiés dans le vRack

GET /vrack/{serviceName}/cloudProject
→ Projets Public Cloud dans le vRack

GET /vrack/{serviceName}/dedicatedCloud
→ Dedicated Cloud dans le vRack

PUT /vrack/{serviceName}
Body: { name: "...", description: "..." }
→ Modifie le vRack
```
