# Cahier des Charges : network/vrack - Liste vRack

## 1. VUE D'ENSEMBLE
- **Page** : Liste des vRack
- **URL** : /network/vrack
- **Objectif** : Afficher et gérer tous les vRack du compte

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation "Network"
- Sidebar avec menu Network (vRack sélectionné)
- Zone principale avec titre + boutons + datagrid
- Pagination en bas

## 3. COMPOSANTS UI
- **OdsButton** : "Commander un vRack" (primary)
- **OdsButton** : "Guides" (secondary, icône livre)
- **OdsDatagrid** : Liste des vRack
- **OdsBadge** : Statut (Actif = vert)
- **OdsActionMenu** : Menu contextuel par ligne
- **OdsPagination** : Navigation pages

## 4. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Nom | Link bleu | ✓ | Identifiant vRack (pn-XXXXXX) |
| ID | Text | ✓ | ID technique (vrack-xxx) |
| Description | Text muted | ✓ | Description libre |
| Statut | Badge | ✓ | Actif (vert) |
| Services | Text | ✓ | Nombre de services |
| Actions | ActionMenu | - | Menu ⋮ |

## 5. ACTION MENU
| Action | Style | Description |
|--------|-------|-------------|
| Résilier | Danger (rouge) | Résilier le vRack |

## 6. BOUTONS HEADER
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Commander un vRack | Primary | Gauche | Ouvre modal/page commande |
| Guides | Secondary outline | Droite | Ouvre documentation |

## 7. APIS & ENDPOINTS
- GET /vrack - Liste des vRack
- GET /vrack/{serviceName} - Détails d'un vRack
- DELETE /vrack/{serviceName}/terminate - Résiliation
