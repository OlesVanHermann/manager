# Cahier des Charges : network/vrack - Modal Add IPv6

## 1. VUE D'ENSEMBLE

**Type:** Modal info
**Fonction:** Activer la connectivité IPv6 sur le vRack
**Contexte:** Première activation IPv6 du vRack

## 2. DIMENSIONS

- Modal: 450x250px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #EFF6FF (info)
- Bordure: #BFDBFE
- Icône: Cercle bleu avec "i"
- Titre: "Activer IPv6 sur le vRack" (15px, bold, #1E40AF)
- Bouton fermer: × à droite

## 4. CONTENU

### Description
"Activez la connectivite IPv6 pour votre reseau prive vRack." (12px, muted)

### Champ bloc IPv6

| Élément | Description |
|---------|-------------|
| Label | "Bloc IPv6 disponible" (13px, bold) |
| Select | Dropdown 400x40px |
| Valeur | Bloc IPv6 sélectionné (ex: 2001:db8::/56) |

### Note
"Le bloc IPv6 sera route vers votre vRack." (11px, muted)

## 5. FOOTER MODAL

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Activer | btn-primary (#0050D7) | 100x36px |

## 6. APIS & ENDPOINTS

```
# Blocs IPv6 disponibles
GET /ip?type=ipv6
Query: { state: "available", routedTo: null }
→ ["2001:db8::/56", "2001:db8:1::/56"]

# Activer IPv6 sur vRack
POST /vrack/{serviceName}/ipv6
Body: {
  block: "2001:db8::/56"
}
→ {
  taskId: "task-123",
  status: "pending"
}
```
