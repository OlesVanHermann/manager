# Cahier des Charges : network/vrack - Modal Add

## 1. VUE D'ENSEMBLE

**Type:** Modal avec liste sélectionnable
**Fonction:** Ajouter des services au vRack
**Contexte:** Association de services au réseau privé

## 2. DIMENSIONS

- Modal: 600x500px
- Overlay: fond noir 40%
- Liste services: 550x280px

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #F9FAFB
- Titre: "Ajouter un service au vRack"
- Bouton fermer: × (top-right)
- Height: 60px

### Contenu
- Description: "Sélectionnez les services à ajouter au vRack :"
- Info vRack: "pn-123456 - Production Network"

## 4. LISTE DES SERVICES

### Barre de recherche
- Placeholder: "Rechercher un service..."
- Largeur: 530px
- Height: 36px

### Items services
| Propriété | Valeur |
|-----------|--------|
| Height item | 40px |
| Checkbox | 20x20px à gauche |
| Nom service | texte principal |
| Type service | texte secondaire à droite |

### États des items
| État | Style |
|------|-------|
| Non sélectionné | Border #E5E7EB, bg white |
| Sélectionné | Border #0050D7, bg #EFF6FF |
| Checkbox checked | Fill #0050D7 avec ✓ blanc |

### Types de services
- Serveur dédié
- Public Cloud
- Load Balancer
- Dedicated Cloud
- IP Load Balancing

### Exemples affichés
| Service | Type |
|---------|------|
| ns123456.ip-192-0-2.eu | Serveur dédié |
| ns789012.ip-198-51-100.eu | Serveur dédié |
| pci-project-abc | Public Cloud |
| loadbalancer-xyz | Load Balancer |

### Compteur sélection
"2 services sélectionnés"

## 5. COMPOSANTS UI

### Checkbox
- Size: 20x20px
- Border-radius: 2px
- Unchecked: border #E5E7EB, bg white
- Checked: bg #0050D7, checkmark blanc

### Item service sélectionné
- Background: #EFF6FF
- Border: #0050D7
- Border-radius: 4px

## 6. ACTIONS & INTERACTIONS

### Recherche
- Filtre la liste en temps réel
- Cherche dans nom et type

### Sélection multiple
- Click sur item: toggle sélection
- Checkbox: toggle sélection
- Plusieurs services sélectionnables

### Bouton Annuler
- Style: btn-secondary (100x36px)
- Action: Ferme le modal

### Bouton Ajouter
- Style: btn-primary (110x36px)
- Action: Ajoute les services sélectionnés
- Désactivé si aucune sélection

## 7. APIS & ENDPOINTS

```
GET /vrack/{serviceName}/allowedServices
→ Liste les services pouvant être ajoutés

POST /vrack/{serviceName}/dedicatedServer
Body: { server: "ns123456.ip-192-0-2.eu" }
→ Ajoute un serveur dédié

POST /vrack/{serviceName}/cloudProject
Body: { project: "pci-project-abc" }
→ Ajoute un projet Public Cloud

POST /vrack/{serviceName}/ipLoadbalancing
Body: { ipLoadbalancing: "loadbalancer-xyz" }
→ Ajoute un IP Load Balancing
```
