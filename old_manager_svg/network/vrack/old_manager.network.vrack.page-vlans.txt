- **Type** : Page configuration VLANs
- **Contexte** : Network vRack
- **Objectif** : Créer et gérer les VLANs du vRack

## 9. APIS & ENDPOINTS
- GET /vrack/{serviceName}/vlan - Liste VLANs
- POST /vrack/{serviceName}/vlan - Créer VLAN
- PUT /vrack/{serviceName}/vlan/{vlanId} - Modifier
- DELETE /vrack/{serviceName}/vlan/{vlanId} - Supprimer
