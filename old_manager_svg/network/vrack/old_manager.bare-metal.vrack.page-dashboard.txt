# Cahier des Charges : bare-metal/vrack - Page Dashboard

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard principal vRack
- **Contexte** : Réseau privé vRack
- **Objectif** : Gérer les services associés au vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 700px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsInput** : Recherche
- **OdsCheckbox** : Sélection services
- **OdsButton** : Actions (Ajouter, Retirer, Guides)
- **OdsLink** : Édition

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "{vrackName}" + icône ✎ édition |
| Sous-titre | "{serviceName}" (ex: pn-123456) |
| Menu Guides | Dropdown "Guides ▼" |

## 5. SECTION DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Titre | "Description" |
| Texte | Description libre |
| Action | "✎ Modifier" (lien bleu) |

## 6. SECTION INTRODUCTION
| Propriété | Valeur |
|-----------|--------|
| Titre | "Introduction" |
| Texte | Explication du vRack |

## 7. MAPPER SERVICES
Interface à deux colonnes pour associer/dissocier des services.

### Panel Gauche : Services disponibles
| Propriété | Valeur |
|-----------|--------|
| Titre | "Services disponibles" |
| Largeur | 400px |
| Hauteur | 380px |
| Recherche | Input "Rechercher..." |

### Panel Droite : Services associés
| Propriété | Valeur |
|-----------|--------|
| Titre | "Services associés au vRack" |
| Largeur | 400px |
| Hauteur | 380px |
| Recherche | Input "Rechercher..." |

## 8. LISTE SERVICES
| Propriété | Valeur |
|-----------|--------|
| Item hauteur | 40px |
| Checkbox | 16x16px |
| Selected | Fond #EFF6FF, bordure #0050D7, checkbox cochée |
| Normal | Fond blanc, bordure grise, checkbox vide |

## 9. TYPES DE SERVICES
- Serveur dédié
- Public Cloud
- IP Load Balancer
- Bloc IP
- Dedicated Cloud

## 10. BOUTONS ACTIONS (centre)
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter → | Fond #0050D7, texte blanc | Associer sélection |
| ← Retirer | Fond blanc, bordure grise | Dissocier sélection |

## 11. SECTION RÉSILIATION
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résiliation" |
| Action | "Résilier ce vRack" (texte rouge #DC2626) |

## 12. APIS & ENDPOINTS
- GET /vrack/{serviceName} - Infos vRack
- GET /vrack/{serviceName}/allowedServices - Services associables
- GET /vrack/{serviceName}/dedicatedServer - Serveurs associés
- GET /vrack/{serviceName}/cloudProject - Projets cloud associés
- POST /vrack/{serviceName}/dedicatedServer - Associer serveur
- DELETE /vrack/{serviceName}/dedicatedServer/{server} - Dissocier
- PUT /vrack/{serviceName} - Modifier nom/description
