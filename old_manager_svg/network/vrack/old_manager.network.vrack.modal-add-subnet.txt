# Cahier des Charges : network/vrack - Modal Add Subnet

## 1. VUE D'ENSEMBLE

**Type:** Modal info avec formulaire
**Fonction:** Ajouter un sous-réseau IPv6 routé au vRack
**Contexte:** Accessible depuis la gestion IPv6

## 2. DIMENSIONS

- Modal: 450x300px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #EFF6FF (info)
- Bordure: #BFDBFE
- Icône: Cercle bleu avec "i"
- Titre: "Ajouter un sous-reseau IPv6" (15px, bold, #1E40AF)
- Bouton fermer: × à droite

## 4. CONTENU

### Description
"Configurez un nouveau sous-reseau IPv6 route pour votre vRack." (12px, muted)

### Champ sous-réseau

| Élément | Description |
|---------|-------------|
| Label | "Sous-reseau IPv6 *" (13px, bold) |
| Input | Champ texte 400x40px |
| Placeholder | "2001:db8::/64" |

### Champ nexthop

| Élément | Description |
|---------|-------------|
| Label | "Adresse de routage (nexthop) *" (13px, bold) |
| Input | Champ texte 400x40px |
| Placeholder | "2001:db8::1" |

## 5. FOOTER MODAL

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Confirmer | btn-primary (#0050D7) | 100x36px |

## 6. VALIDATION

- Sous-réseau: Format CIDR IPv6 valide
- Nexthop: Adresse IPv6 valide appartenant au vRack
- Champs requis: Les deux

## 7. APIS & ENDPOINTS

```
# Ajouter sous-réseau routé
POST /vrack/{serviceName}/ipv6
Body: {
  subnet: "2001:db8:2::/64",
  nexthop: "2001:db8::1",
  type: "routed"
}
→ {
  taskId: "task-456",
  status: "pending"
}
```
