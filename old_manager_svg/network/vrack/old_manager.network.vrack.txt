# Cahier des Charges : network/vrack - Dashboard principal

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard vRack
- **URL** : /network/vrack
- **Objectif** : Vue d'ensemble des vRack avec liste et actions

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation "Network"
- Sidebar avec menu Network
- Zone principale avec datagrid
- Bouton Commander en haut

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste des vRack
- **OdsButton** : Commander un vRack
- **OdsBadge** : Statuts
- **OdsActionMenu** : Actions par ligne

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link | Nom du vRack |
| ID | Text | Identifiant technique |
| Description | Text muted | Description |
| Statut | Badge | État (Actif) |
| Services | Number | Nombre services |
| Actions | Menu ⋮ | Actions contextuelles |

## 5. APIS & ENDPOINTS
- GET /vrack - Liste des vRack
- GET /vrack/{serviceName} - Détails
