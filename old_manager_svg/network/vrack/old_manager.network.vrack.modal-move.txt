# Cahier des Charges : network/vrack - Modal Déplacer datacenter

## 1. VUE D'ENSEMBLE
- **Modal** : Déplacement datacenter Private Cloud (variante)
- **Déclencheur** : Action sur service Private Cloud
- **Objectif** : Transférer vers un autre vRack

## 2. DIMENSIONS
- Largeur : 650px
- Hauteur : 450px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Explications
- **OdsMessage** : Warning interruption
- **OdsSelect** : Sélecteur vRack
- **OdsText** : Infos service actuel
- **OdsButton** : Annuler / Déplacer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Déplacement d'un datacentre Private Cloud" |
| Explication | Règle service non-orphelin |
| Warning | Interruption service |
| Sélecteur | vRack destination |
| Infos | Service et vRack actuel |
| Footer | Boutons |

## 5. TEXTE EXPLICATIF
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Vous avez sélectionné un service de type datacentre Private Cloud." |
| Ligne 2 | "Ce service ne peut être orphelin et doit être transféré obligatoirement" |
| Ligne 3 | "vers un autre vRack." |

## 6. MESSAGE WARNING
| Propriété | Valeur |
|-----------|--------|
| Style | Warning (jaune #FEF3C7) |
| Icône | ⚠️ |
| Texte | "Cette opération entraînera une interruption de service d'environ 5 minutes." |

## 7. SÉLECTEUR DESTINATION
| Propriété | Valeur |
|-----------|--------|
| Label | "vRack de destination :" |
| Type | OdsSelect |
| Largeur | 400px |
| Placeholder | "Sélectionnez un vRack" |

## 8. INFORMATIONS SERVICE
| Champ | Valeur exemple |
|-------|----------------|
| Service à déplacer | pcc-192-0-2-15 (Private Cloud Datacenter) |
| vRack actuel | pn-123456 |

## 9. BOUTONS
| Bouton | Style | Largeur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | 120px | Fermer |
| Déplacer | Primary | 120px | Exécuter |

## 10. APIS & ENDPOINTS
- GET /vrack - Liste vRack pour sélecteur
- POST /vrack/{serviceName}/dedicatedCloud/{dcId}/move - Déplacer
- GET /vrack/{serviceName}/task/{taskId} - Suivi tâche
