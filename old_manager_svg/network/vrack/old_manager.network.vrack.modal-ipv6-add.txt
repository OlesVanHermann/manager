# Cahier des Charges : network/vrack - Modal ajouter IPv6 au vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec warning
- **Contexte** : Page IPv6 vRack
- **Objectif** : Ajouter un bloc IPv6 au vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Informations et description
- **OdsMessage** : Warning conflit
- **OdsLink** : Documentation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter IPv6 au vRack" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Informations affichées
| Élément | Description |
|---------|-------------|
| Bloc IPv6 | Adresse du bloc (ex: 2001:db8::/48) |
| vRack | Nom du vRack (ex: pn-123456) |
| Région | Localisation (ex: GRA) |
| Délai | "Cette opération peut prendre quelques minutes." |
| Documentation | Lien "En savoir plus" |

### Message warning (conditionnel)
| Type | Contenu |
|------|---------|
| Warning (jaune) | "Ce vRack contient déjà un bloc IPv6 dans cette région." |
| | "Cette configuration peut créer des conflits." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary | Ajoute le bloc IPv6 |

## 7. APIS & ENDPOINTS
- POST /vrack/{serviceName}/ipv6 - Ajouter bloc IPv6
