# Cahier des Charges : network/vrack - Modal supprimer IPv6 du vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Page IPv6 vRack
- **Objectif** : Supprimer un bloc IPv6 du vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 190px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer IPv6 du vRack" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Vous allez supprimer le bloc IPv6 suivant :" |
| Bloc IPv6 | Adresse affichée en gras (ex: 2001:db8::/48) |
| Question | "Êtes-vous sûr de vouloir continuer ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Warning (orange) | Supprime le bloc IPv6 |

## 7. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/ipv6/{ipv6} - Supprimer bloc IPv6
