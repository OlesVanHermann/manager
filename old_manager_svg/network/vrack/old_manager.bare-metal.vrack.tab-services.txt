# Cahier des Charges : bare-metal/vrack - Onglet Services

## 1. VUE D'ENSEMBLE
- **Page** : Liste des services connectés au vRack
- **Type** : Onglet tableau
- **Objectif** : Gérer les services du réseau privé

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Services"
- Onglets navigation
- Toolbar (bouton + filtre)
- Tableau services
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Ajouter un service
- **OdsSelect** : Filtrer par type
- **OdsDatagrid** : Tableau services
- **OdsBadge** : État (Connecté)
- **OdsActionMenu** : Menu par ligne
- **OdsPagination** : Navigation pages

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Service | Link | Nom/ID du service |
| Type | Text | Serveur dédié, HPC, VPS, IP Block, NAS-HA |
| Datacenter | Text | rbx, sbg, gra, bhs |
| VLANs | Text | VLANs attribués |
| État | Badge | Connecté (vert) |
| Actions | ActionMenu | Menu ⋮ |

## 5. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Configurer VLANs | Normal | Attribuer/retirer VLANs |
| Voir le service | Normal | Ouvre le service |
| Retirer du vRack | Danger | Déconnecte le service |

## 6. FILTRE TYPE
| Option | Description |
|--------|-------------|
| Tous les types | Par défaut |
| Serveur dédié | Dedicated servers |
| VPS | Virtual Private Server |
| Hosted Private Cloud | VMware/Nutanix |
| IP Block | Blocs IP |
| NAS-HA | Stockage HA |

## 7. APIS & ENDPOINTS
- GET /vrack/{serviceName}/dedicatedServer
- GET /vrack/{serviceName}/cloudProject
- GET /vrack/{serviceName}/ip
- POST /vrack/{serviceName}/dedicatedServer
- DELETE /vrack/{serviceName}/dedicatedServer/{dedicatedServer}
