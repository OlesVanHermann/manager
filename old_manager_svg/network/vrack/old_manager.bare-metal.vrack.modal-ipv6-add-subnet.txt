---

## 8. APIS & ENDPOINTS

### 8.1 Ajouter un sous-réseau IPv6

```
POST /vrack/{serviceName}/ipv6
```

Body:
```json
{
  "subnet": "2001:db8::/64",
  "address": "2001:db8::1"
}
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "pending"
}
```
