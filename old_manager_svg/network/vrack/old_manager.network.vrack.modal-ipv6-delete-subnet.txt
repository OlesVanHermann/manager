# Cahier des Charges : network/vrack - Modal supprimer sous-réseau IPv6

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page IPv6 vRack
- **Objectif** : Supprimer un sous-réseau IPv6 routé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le sous-réseau" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Voulez-vous supprimer ce sous-réseau IPv6 routé ?" |
| Warning | "Cette action est irréversible." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Danger (rouge) | Supprime le sous-réseau |

## 7. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/ipv6/{ipv6}/routedSubrange/{subrangeId} - Supprimer sous-réseau
