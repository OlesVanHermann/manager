# Cahier des Charges : network/cloud-connect - Onglet Interfaces

## 1. VUE D'ENSEMBLE
- **Page** : Liste des interfaces réseau
- **URL** : /network/cloud-connect/{serviceName}?tab=interfaces
- **Objectif** : Surveiller l'état des interfaces du service

## 2. STRUCTURE PAGE
- Breadcrumb et titre
- Onglets avec "Interfaces" actif
- Carte conteneur
- Tableau des interfaces

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Conteneur
- **OdsTable** : Liste des interfaces
- **OdsBadge** : Statut (active/standby/down)
- **OdsIcon** : Signal optique (emoji)
- **OdsButton** : "Statistiques" par interface

## 4. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| ID | Text | Identifiant interface |
| Statut | Badge | active/standby/down |
| Signal optique | Icon + Text | up/idle/no signal |
| Actions | Button | Statistiques |

## 5. BADGES STATUT
| Statut | Badge | Couleur | Description |
|--------|-------|---------|-------------|
| active | success | Vert | Interface active |
| standby | warning | Jaune | En attente |
| down | error | Rouge | Interface inactive |

## 6. SIGNAL OPTIQUE
| État | Icône | Description |
|------|-------|-------------|
| up | | Signal nominal |
| idle | | En veille |
| no signal | | Pas de signal |

## 7. DONNÉES EXEMPLE
| ID | Statut | Signal | Actions |
|----|--------|--------|---------|
| Interface #1 | active | up | Statistiques |
| Interface #2 | active | up | Statistiques |
| Interface #3 | standby | idle | Statistiques |
| Interface #4 | down | no signal | Statistiques |

## 8. APIS & ENDPOINTS
- GET /ovhCloudConnect/{serviceName}/interface - Liste interfaces
- GET /ovhCloudConnect/{serviceName}/interface/{interfaceId} - Détail interface
- GET /ovhCloudConnect/{serviceName}/interface/{interfaceId}/statistics - Stats
