# Cahier des Charges : network/cloud-connect - Onglet Vue Générale

## 1. VUE D'ENSEMBLE
- **Page** : Vue générale avec diagramme réseau
- **URL** : /network/cloud-connect/{serviceName}?tab=overview
- **Objectif** : Visualiser l'architecture réseau du service

## 2. STRUCTURE PAGE
- Onglets avec "Vue générale" actif
- Carte diagramme réseau
- Indicateurs de statut
- Actions rapides

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Conteneur diagramme
- **OdsIcon** : Indicateurs de statut (cercles colorés)
- **OdsButton** : Actions rapides (outline)

## 4. DIAGRAMME RÉSEAU
| Élément | Description | Style |
|---------|-------------|-------|
| On-Premise | "Votre infrastructure" | Fond gris, bordure grise |
| Connexion 1 | Lien vers PoP | Ligne bleue, label "1 Gbps" |
| PoP | "PoP PAR-TH2 - Point of Presence" | Fond bleu clair, bordure bleue |
| Connexion 2 | Backbone OVH | Ligne bleue, label "Backbone OVH" |
| Datacenter | "DC Gravelines - GRA" | Fond vert clair, bordure verte |
| Connexion 3 | Lien vers vRack | Ligne pointillée bleue |
| vRack | "vRack pn-123456" | Fond blanc, bordure bleue |

## 5. INDICATEURS STATUT
| Indicateur | Couleur | Label |
|------------|---------|-------|
| ● | Vert | PoP configuré |
| ● | Vert | DC configuré |
| ● | Vert | vRack associé |

## 6. ACTIONS RAPIDES
| Action | Style | Description |
|--------|-------|-------------|
| Télécharger LOA | Outline bleu | Letter of Authorization |
| Ajouter un datacenter | Outline bleu | Ajouter un DC |
| Modifier le vRack | Outline bleu | Changer l'association |
| Lancer diagnostic | Outline bleu | Test de connectivité |

## 7. APIS & ENDPOINTS
- GET /ovhCloudConnect/{serviceName} - Infos service
- GET /ovhCloudConnect/{serviceName}/pop/configuration - Config PoP
- GET /ovhCloudConnect/{serviceName}/datacenter - Liste DC
- GET /vrack/{serviceName} - Infos vRack
