# Cahier des Charges : network/cloud-connect - Liste

## 1. VUE D'ENSEMBLE
- **Page** : Liste des services OVHcloud Connect
- **URL** : /network/cloud-connect
- **Objectif** : Lister et gérer les connexions hybrides OVHcloud Connect

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation Network
- Sidebar avec menu Network (vRack Services, vRack, OVHcloud Connect actif, Load Balancer, IP)
- Breadcrumb (Network > OVHcloud Connect)
- Titre + description
- Bouton Commander
- Datagrid des services
- ActionMenu contextuel

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation contextuelle
- **OdsText** : Titre et description
- **OdsButton** : "Commander" (primary)
- **OdsDatagrid** : Liste des services OVHcloud Connect
- **OdsBadge** : Statut (actif/installation)
- **OdsStepper** : Suivi de commande (4 étapes)
- **OdsActionMenu** : Actions par service

## 4. DESCRIPTION PAGE
"Solution d'interconnexion hybride permettant de relier votre infrastructure on-premise aux datacentres OVHcloud."

## 5. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Nom | Link | ✓ | Nom du service OCC |
| Description | Text | ✓ | Description personnalisée |
| Port | Text | ✓ | Bande passante (1/10/100 Gbps) |
| Statut | Badge | ✓ | actif/installation |
| Suivi de commande | Stepper | - | 4 étapes de provisioning |
| Actions | Menu ⋮ | - | Menu contextuel |

## 6. BADGES STATUT
| Statut | Badge | Couleur |
|--------|-------|---------|
| actif | success | Vert |
| installation | warning | Jaune |
| erreur | error | Rouge |

## 7. STEPPER SUIVI COMMANDE
| Étape | Description |
|-------|-------------|
| 1 | Commande créée |
| 2 | Configuration PoP |
| 3 | Configuration DC |
| 4 | Service actif |

## 8. ACTION MENU
| Action | Style | Description |
|--------|-------|-------------|
| Accéder au service | Normal | Ouvre le détail |
| --- | Separator | |
