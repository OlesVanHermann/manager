# Cahier des Charges : network/cloud-connect - Modal supprimer configuration POP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page interfaces Cloud Connect
- **Objectif** : Supprimer la configuration d'un port POP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la configuration POP" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir supprimer la configuration du port {portName} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime la configuration |

## 7. APIS & ENDPOINTS
- DELETE /ovhCloudConnect/{serviceName}/pop/{id} - Supprimer configuration POP
