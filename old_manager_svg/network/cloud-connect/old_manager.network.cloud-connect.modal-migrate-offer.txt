# Cahier des Charges : network/cloud-connect - Modal mettre à jour offre

## 1. VUE D'ENSEMBLE
- **Type** : Modal de migration avec warning
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Mettre à jour l'offre (upgrade bande passante)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description et informations
- **OdsMessage** : Warning actions requises
- **OdsButton** : Annuler, Appliquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mettre à jour l'offre" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Informations
| Texte |
|-------|
| "La bande passante sera augmentée de 1 Gbps à 10 Gbps." |
| "Votre configuration actuelle sera conservée." |
| "Un email de confirmation vous sera envoyé." |

### Message warning
| Type | Contenu |
|------|---------|
| Warning (jaune) | **Attention** |
| | "Certaines actions sont requises de votre part après la migration." |
| | "Consultez la documentation pour plus d'informations." |

### Confirmation
| Texte |
|-------|
| "Souhaitez-vous lancer le processus de migration ?" |
| "Le prix et l'engagement seront modifiés en conséquence." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Appliquer | Primary | Lance la migration |

## 7. APIS & ENDPOINTS
- POST /ovhCloudConnect/{serviceName}/migrate - Lancer migration offre
