# Cahier des Charges : network - Error Page

## 1. VUE D'ENSEMBLE
- **Type** : Page d'erreur
- **URL** : /network/* (en cas d'erreur)
- **Objectif** : Afficher une erreur et proposer des actions

## 2. STRUCTURE PAGE
- Header OVHcloud avec Network actif
- Sidebar avec menu complet
- Zone centrale avec état d'erreur

## 3. COMPOSANTS UI
- **OdsEmptyState** : Conteneur erreur
- **OdsIcon** : Icône avertissement ⚠️
- **OdsText** : Messages d'erreur
- **OdsButton** : Réessayer (primary), Retour accueil (outline)
- **OdsLink** : Liens d'aide

## 4. SIDEBAR MENU
| Item | Lien |
|------|------|
| vRack Services | /network/vrack-services |
| vRack | /network/vrack |
| OVHcloud Connect | /network/cloud-connect |
| Load Balancer | /network/load-balancer |
| IP | /network/ip |
| Network Security | /network/security |

## 5. CONTENU ERREUR

### Icône
| Propriété | Valeur |
|-----------|--------|
| Fond | Cercle rouge clair #FEE2E2 |
| Rayon | 80px |
| Icône | ⚠️ (64px) |
| Couleur | Rouge #DC2626 |

### Messages
| Élément | Texte |
|---------|-------|
| Titre | "Une erreur est survenue" |
| L1 | "Nous n'avons pas pu charger cette page." |
| L2 | "Veuillez réessayer dans quelques instants." |
| Code | "Code erreur: ERR_500_INTERNAL_SERVER_ERROR" |

## 6. BOUTONS ACTION
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Réessayer | Primary bleu | 140x44px |
| Retour accueil | Outline bleu | 140x44px |

## 7. LIENS AIDE
| Lien | Action |
|------|--------|
| Consulter le statut des services | status.ovhcloud.com |
| Contacter le support | /support |

## 8. CODES ERREUR POSSIBLES
| Code | Description |
|------|-------------|
| ERR_500_INTERNAL_SERVER_ERROR | Erreur serveur |
| ERR_404_NOT_FOUND | Ressource non trouvée |
| ERR_403_FORBIDDEN | Accès refusé |
| ERR_503_SERVICE_UNAVAILABLE | Service indisponible |
