# Cahier des Charges : network/vrack-services - Liste endpoints

## 1. VUE D'ENSEMBLE
- **Page** : Liste des Service Endpoints
- **URL** : /network/vrack-services/{id}/endpoints
- **Objectif** : Gérer les endpoints existants

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre service
- Onglets (Service Endpoints actif)
- Bouton création
- Datagrid

## 3. COMPOSANTS UI
- **OdsTabs** : Vue d'ensemble / Sous-réseaux / Service Endpoints
- **OdsButton** : "+ Créer un Service Endpoint"
- **OdsDatagrid** : Liste endpoints
- **OdsActionMenu** : Actions

## 4. COLONNES DATAGRID
| Colonne | Type | Exemple |
|---------|------|---------|
| Nom du service | Text | efs-prod-storage-01 |
| Type de service | Text | Enterprise File Storage |
| Adresse IP | Text | 10.0.0.5 |
| Sous-réseau | CIDR | 10.0.0.0/24 |
| Actions | Menu ⋮ | Actions |
