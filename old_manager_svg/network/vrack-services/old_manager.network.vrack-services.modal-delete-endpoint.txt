# Cahier des Charges : network/vrack-services - Modal Supprimer endpoint

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un Service Endpoint
- **Déclencheur** : Action "Supprimer le Service Endpoint"
- **Objectif** : Supprimer un endpoint

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 250px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Message d'avertissement
- **OdsButton** : Annuler / Supprimer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Supprimer le Service Endpoint" |
| Body | Message conséquence |
| Footer | Boutons |

## 5. MESSAGE
```
Si vous supprimez ce Service Endpoint, vous perdrez la
connectivité au service associé.
```

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer |
| Supprimer | Danger | Exécuter |

## 7. APIS & ENDPOINTS
- DELETE /vrackServices/{id}/endpoints/{endpointId} - Supprimer
