# Cahier des Charges : network/vrack-services - Modal Associer (variante)

## 1. VUE D'ENSEMBLE
- **Modal** : Association vRack existant
- **Déclencheur** : Configuration initiale
- **Objectif** : Associer un vRack au service

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 380px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Explication
- **OdsSelect** : Dropdown avec liste vRack
- **OdsButton** : Annuler / Associer mon vRack

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Associer un vRack existant à vRack Services" |
| Body | Explication + sélecteur avec dropdown |
| Footer | Boutons |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Message L1 | "Pour que vRack Services fonctionne, vous devez l'associer" |
| Message L2 | "à l'un de vos vRack existants." |
| Label | "Sélectionner un vRack" |
| Placeholder | "Sélectionner un vRack" |

## 6. DROPDOWN (ouvert)
| Option | État |
|--------|------|
| pn-123456 | Sélectionné (bleu) |
| pn-789012 | - |
| pn-456789 | - |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer |
| Associer mon vRack | Primary | Exécuter |

## 8. APIS & ENDPOINTS
- GET /vrack - Liste vRack
- POST /vrackServices/{id}/vrack/associate - Associer
