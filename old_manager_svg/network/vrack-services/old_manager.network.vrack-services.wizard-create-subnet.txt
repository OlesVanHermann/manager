# Cahier des Charges : network/vrack-services - Wizard Create Subnet

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire
**Fonction:** Créer un sous-réseau vRack Services
**Contexte:** Configuration vRack Services

## 2. DIMENSIONS

- Page: 900x680px
- Background: #F9FAFB

## 3. NAVIGATION

### Breadcrumb
- vRack Services > vs-abc123 > Sous-réseaux > Créer

### Titre page
- "Créer un sous-réseau" (24px, font-weight: 600)

## 4. FORMULAIRE

### Card principale
- Dimensions: 500x520px
- Background: #FFFFFF
- Border: #E5E7EB

### Champs
| Champ | Type | Obligatoire | Dimensions |
|-------|------|-------------|------------|
| Nom du sous-réseau | Input text | Non (facultatif) | 436x40px |
| Plage d'adresses du sous-réseau (CIDR) | Input text | Oui | 436x40px |
| Plage d'adresses de service (CIDR) | Input text | Oui | 436x40px |
| VLAN | Radio buttons | Non | - |

### Helpers
- CIDR sous-réseau: "La plage d'adresses du sous-réseau doit être en /24."
- CIDR service: "La plage d'adresses de service sera utilisée par les services du vRack Services. Taille entre /27 et /29."
- VLAN: "Vous pouvez exposer ce sous-réseau sur un VLAN spécifique."

### Options VLAN
| Option | État par défaut |
|--------|-----------------|
| Pas de VLAN | Sélectionné |
| Sélectionnez un VLAN | Non sélectionné |

### Valeurs par défaut
- Nom: "Mon sous-réseau"
- CIDR: "10.0.0.0/24"
- Service: "10.0.0.0/28"

## 5. CARD INFO

### Dimensions
- 300x140px
- Background: #FFFFFF
- Border: #E5E7EB

### Contenu
- Titre: "Information"
- "Pour que vRack Services fonctionne, vous devez créer au moins un sous-réseau."

## 6. FOOTER

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Ghost (#0050D7 text) | Retour liste |
| Créer | Primary (#0050D7) | POST API |

```

```
