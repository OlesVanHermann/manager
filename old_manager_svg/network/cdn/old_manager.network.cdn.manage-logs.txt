# Cahier des Charges : network/cdn - Manage Logs

## 1. VUE D'ENSEMBLE

**Type:** Page console logs
**Fonction:** Visualiser les logs CDN en temps réel
**Contexte:** Page de gestion des logs CDN

## 2. NAVIGATION

### NAV1 (Header)
OVHcloud (logo)

## 3. LAYOUT

- Background page: #F9FAFB
- Dimensions: 1200x500px

## 4. HEADER PAGE

### Titre
"Logs CDN" (22px, bold)

### Sous-titre
"Logs en temps reel de votre service CDN" (13px, muted)

## 5. CONSOLE (1120x330px)

### Style
- Background: #1F2937 (dark)
- Border radius: 8px
- Font: Monaco/Courier New monospace 11px

### Couleurs logs
- Succès (2xx): #4ADE80 (vert)
- Warning (3xx): #FBBF24 (jaune)
- Erreur (4xx, 5xx): #F87171 (rouge)

### Format
```
{IP} - {user} [{timestamp}] "{method} {url} {protocol}" {status} {bytes} "{user-agent}" {cache}
```

## 6. APIS & ENDPOINTS

```
# Logs temps réel
GET /cdn/dedicated/{serviceName}/logs
Query: {
  stream: true,
  domain: "cdn.example.com"
}

# Télécharger logs
GET /cdn/dedicated/{serviceName}/logs/download
Query: { from: "...", to: "...", format: "csv" }
```
