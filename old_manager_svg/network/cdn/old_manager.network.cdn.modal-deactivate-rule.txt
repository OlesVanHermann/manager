# Cahier des Charges : network/cdn - Modal désactiver règle de cache

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page règles de cache CDN
- **Objectif** : Désactiver temporairement une règle de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Désactiver

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Désactiver la règle de cache" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir désactiver cette règle ?" |
| Règle | ".jpg, .png, .gif" (gras) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Désactiver | Warning (orange) | Désactive la règle |

## 7. APIS & ENDPOINTS
- PUT /cdn/{serviceName}/domains/{domain}/cacheRules/{ruleId} - Désactiver règle (status: off)
