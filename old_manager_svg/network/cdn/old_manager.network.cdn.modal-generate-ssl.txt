# Cahier des Charges : network/cdn - Modal Generate SSL

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire avec warning
**Fonction:** Générer un nouveau certificat SSL Let's Encrypt
**Contexte:** Action depuis la gestion SSL

## 2. DIMENSIONS

- Modal: 500x300px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #000E9C (OVHcloud)
- Titre: "Generer un certificat SSL" (15px, bold, blanc)
- Bouton fermer: × blanc à droite

## 4. CONTENU

### Encart warning (450x50px)
- Background: #FEF3C7
- Bordure: #FDE68A
- Texte 1: "Un certificat SSL existe deja."
- Texte 2: "Cette action le remplacera definitivement."

### Champ nom

| Élément | Description |
|---------|-------------|
| Label | "Nom du certificat *" (13px, bold) |
| Input | Champ texte 450x40px |
| Placeholder | "ssl-certificate-prod" |

### Contraintes
"Le nom ne doit pas contenir d'espaces" (11px, muted)

## 5. FOOTER MODAL

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Generer | btn-primary (#0050D7) | 100x36px |

## 6. APIS & ENDPOINTS

```
# Générer certificat Let's Encrypt
POST /cdn/dedicated/{serviceName}/ssl/letsencrypt
Body: {
  name: "ssl-certificate-prod"
}
→ {
  taskId: "task-ssl-123",
  status: "generating",
  estimatedTime: 120
}
```
