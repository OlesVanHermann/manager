# Cahier des Charges : network/cdn - Page Origins

## 1. VUE D'ENSEMBLE
- **Type** : Page liste origines CDN
- **Contexte** : Network CDN
- **Objectif** : Gérer les serveurs d'origine backend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

- **OdsButton** : Ajouter origine
- **OdsDatagrid** : Liste origines
- **OdsBadge** : Type, état health
- **ActionMenu** : Menu ⋮

## 10. MENU ACTIONS (⋮)
| Action | Style |
|--------|-------|
| Modifier | Normal |
| Tester connectivité | Normal |
| Définir comme principal | Normal |
| --- | Séparateur |
| Supprimer | Danger (#DC2626) |

## 11. APIS & ENDPOINTS
- GET /cdn/{serviceName}/domain/{domain}/origin - Liste origines
- POST /cdn/{serviceName}/domain/{domain}/origin - Ajouter origine
- PUT /cdn/{serviceName}/domain/{domain}/origin/{id} - Modifier
- DELETE /cdn/{serviceName}/domain/{domain}/origin/{id} - Supprimer
