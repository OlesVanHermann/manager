## 6. DATAGRID (1120x270px)

### Colonnes

| Colonne | Largeur | Description |
|---------|---------|-------------|
| Nom | 270px | Nom du certificat (lien) |
| CN (Common Name) | 300px | Domaine couvert |
| Expiration | 250px | Date d'expiration |
| Statut | 200px | Badge Valide/Expire |

### Badges statut
- Valide: #DCFCE7, texte #16A34A

### Pagination

## 7. MENU ACTIONS (⋮)

| Option | Action |
|--------|--------|
| Voir details | Affiche infos certificat |
| Mettre a jour | Ouvre modal update |
| Renouveler | Lance renouvellement |
| Supprimer | Modal confirmation |

## 8. APIS & ENDPOINTS

```
# Liste des certificats
GET /cdn/dedicated/{serviceName}/ssl
→ [{
  name: "cert-wildcard-prod",
  cn: "*.example.com",
  expiration: "2025-06-15",
  status: "valid"
}]

# Détails certificat
GET /cdn/dedicated/{serviceName}/ssl/{name}
→ {
  name: "cert-wildcard-prod",
  cn: "*.example.com",
  expiration: "2025-06-15",
  status: "valid",
  issuer: "Let's Encrypt",
  san: ["*.example.com", "example.com"]
}

# Supprimer certificat
DELETE /cdn/dedicated/{serviceName}/ssl/{name}
```
