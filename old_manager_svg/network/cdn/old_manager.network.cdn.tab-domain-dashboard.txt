# Cahier des Charges : network/cdn - En-tête domaine

## 1. VUE D'ENSEMBLE
- **Type** : Header de page avec navigation
- **Contexte** : Vue détail d'un domaine CDN
- **Objectif** : Afficher les informations et navigation du domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% |
| Hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsHeader** : Barre de navigation OVHcloud
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsText** : Titre du domaine
- **OdsBadge** : Statut ON/OFF
- **OdsTabs** : Navigation onglets

## 4. HEADER OVHCLOUD
| Élément | Valeur |
|---------|--------|
| Logo | OVHcloud |
| Breadcrumb | Network › CDN › {serviceName} › {domainName} |
| Fond | Bleu OVH (#000E9C) |

## 5. SECTION TITRE
| Élément | Description |
|---------|-------------|
| Nom domaine | Titre principal (22px, gras) |
| Badge statut | ON (vert) ou OFF (rouge) |

## 6. NAVIGATION ONGLETS
| Onglet | Description |
|--------|-------------|
| Statistiques | Graphiques de trafic (actif) |
| Règles de cache | Gestion des règles |
| Logs | Logs en temps réel |

## 7. APIS & ENDPOINTS
- GET /cdn/{serviceName}/domains/{domain} - Récupérer infos domaine
