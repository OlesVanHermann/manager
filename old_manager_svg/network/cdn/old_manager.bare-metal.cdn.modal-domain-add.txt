# Cahier des Charges : bare-metal/cdn - Ajouter domaine

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un domaine au CDN
- **Type** : Modal formulaire
- **Objectif** : Configurer un nouveau domaine

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Formulaire configuration
- Message info DNS
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Domaine, backend, port
- **OdsRadio** : Options SSL
- **OdsMessage** : Info DNS
- **OdsButton** : Annuler, Ajouter

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom de domaine | Input | Oui | www.example.com |
| Serveur backend | Input | Oui | backend.example.com |
| Port du backend | Input | Non | 443 (défaut) |
| Certificat SSL | Radio | Oui | Choix SSL |

## 5. OPTIONS SSL
| Option | Sélection | Description |
|--------|-----------|-------------|
| Let's Encrypt | Sélectionné | Gratuit, automatique |
| Importer certificat | - | Certificat custom |
| Pas de SSL | - | HTTP uniquement |

## 6. HELPER TEXTS
| Champ | Message |
|-------|---------|
| Domaine | Le domaine doit pointer vers l'IP Anycast du CDN (46.105.200.1) |
| Backend | Adresse du serveur d'origine qui héberge votre contenu |

## 7. MESSAGE INFO DNS
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ |
| Titre | Configuration DNS requise |
| Texte | Ajoutez un enregistrement CNAME pointant vers cdn-abc123.cdn.ovh.net |

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 9. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains
  - Paramètres : domain, backend, backendPort, ssl
