# Cahier des Charges : network/cdn - Wizard Add Rule

## 1. VUE D'ENSEMBLE

**Type:** Wizard 2 étapes
**Fonction:** Créer une nouvelle règle de cache
**Contexte:** Accessible depuis la liste des règles

## 2. DIMENSIONS

- Container: 600x440px
- Border radius: 8px

## 3. HEADER WIZARD

- Background: #F9FAFB
- Titre: "Creer une regle de cache" (16px, bold)

## 4. STEPPER (2 étapes)

| Étape | Nom | État |
|-------|-----|------|
| 1 | Configuration | Actif (#0050D7) |
| 2 | Confirmation | Inactif |

## 5. ÉTAPE 1 - CONFIGURATION

### Instruction
"Configurez votre nouvelle regle de cache :" (13px)

### Champ type de fichier

| Élément | Description |
|---------|-------------|
| Label | "Type de fichier" (12px, muted) |
| Select | Dropdown 250x36px |
| Options | Extension, Dossier, URI |

### Champ motif

| Élément | Description |
|---------|-------------|
| Label | "Motif de correspondance" (12px, muted) |
| Input | Champ texte 400x36px |

| Élément | Description |
|---------|-------------|

## 7. APIS & ENDPOINTS

```
# Créer règle
POST /cdn/dedicated/{serviceName}/domains/{domain}/rules
Body: {
  type: "extension",  // extension, folder, uri
  pattern: ".jpg, .png, .gif",
  cacheType: "cache",  // cache, noCache
  ttl: 86400
}
→ {
  ruleId: "rule-123",
  status: "active"
}
```
