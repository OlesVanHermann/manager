# Cahier des Charges : bare-metal/cdn - Modal Purge Cache

## 1. VUE D'ENSEMBLE
- **Type** : Modal purge cache
- **Contexte** : CDN Dedicated
- **Objectif** : Purger le cache d'un domaine CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsMessage** : Info message
- **OdsRadio** : Options purge
- **OdsButton** : Actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Purger le cache CDN" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Information |
| Fond | #EFF6FF |
| Message | "La purge du cache peut prendre quelques minutes pour se propager sur tous les points de présence." |

## 6. OPTIONS RADIO
| Option | État | Description |
|--------|------|-------------|
| Purge complète | Selected | "Supprime tout le cache du domaine" |
| Purge par URL | Default | "Supprime uniquement les URLs spécifiées" |

## 7. STYLE RADIO
| État | Style |
|------|-------|
| Selected | Cercle bleu #0050D7 avec point interne |
| Default | Cercle blanc, bordure grise |

## 8. INFO DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB avec bordure |
| Label | "Domaine" |
| Valeur | "{domainName}" |

## 9. FOOTER
| Bouton | Style |
|--------|-------|
| Annuler | Fond blanc, bordure grise |
| Purger | Fond #0050D7, texte blanc |

## 10. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domain}/purge - Purge complète
- POST /cdn/dedicated/{serviceName}/domains/{domain}/purge - Purge par URL (body: urls[])
