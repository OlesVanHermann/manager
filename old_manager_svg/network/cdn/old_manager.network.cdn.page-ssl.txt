## 10. APIS & ENDPOINTS
- GET /cdn/dedicated/{id}/ssl - Liste certificats
- POST /cdn/dedicated/{id}/ssl - Ajouter certificat
- POST /cdn/dedicated/{id}/ssl/{sslId}/renew - Renouveler
- DELETE /cdn/dedicated/{id}/ssl/{sslId} - Supprimer
