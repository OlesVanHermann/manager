# Cahier des Charges : network/cdn - Modal générer certificat Let's Encrypt

## 1. VUE D'ENSEMBLE
- **Type** : Modal de génération
- **Contexte** : Page SSL CDN
- **Objectif** : Générer automatiquement un certificat Let's Encrypt

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning remplacement
- **OdsInput** : Nom du certificat
- **OdsButton** : Annuler, Générer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Générer un certificat Let's Encrypt" |
| Fond | Vert (#D1FAE5) |
| Couleur texte | Vert (#10B981) |
| Bouton fermer | × |

## 5. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠️ Un certificat existe déjà. Il sera remplacé." |

## 6. FORMULAIRE

### Champ
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Nom du certificat * | Input | "letsencrypt-cdn" |

### Info
| Texte |
|-------|
| "Le certificat sera généré automatiquement via Let's Encrypt." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Générer | Success (vert) | Génère le certificat |

## 8. APIS & ENDPOINTS
- POST /cdn/{serviceName}/ssl/letsencrypt - Générer certificat Let's Encrypt
