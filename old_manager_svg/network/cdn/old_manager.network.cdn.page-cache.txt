# Cahier des Charges : network/cdn - Page Cache

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration cache CDN
- **Contexte** : Gestion CDN
- **Objectif** : Configurer les règles de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Purge, Préchargement
- **OdsCard** : Sections configuration
- **OdsInput** : Champs TTL
- **OdsSelect** : Mode de cache
- **OdsCheckbox** : Options cache
- **OdsBadge** : Extensions cachées

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Cache Settings" |
| Sous-titre | "cdn.example.com - Configuration du cache" |

## 5. BOUTONS ACTIONS
| Bouton | Style | Largeur |
|--------|-------|---------|
| Purger le cache | Danger #DC2626 | 160px |
| Précharger URL | Outline | 160px |

## 6. SECTION RÈGLES DE CACHE
| Champ | Type | Valeur par défaut |
|-------|------|-------------------|
| TTL par défaut | Input number | 86400 (24h) |
| Browser cache TTL | Input number | 3600 (1h) |
| Mode de cache | Select | Standard |

## 7. OPTIONS CACHE
| Option | Type | État |
|--------|------|------|
| Ignorer query string | Checkbox | Activé (✓) |
| Servir contenu expiré | Checkbox | Désactivé |

## 8. SECTION EXTENSIONS CACHÉES
| Propriété | Valeur |
|-----------|--------|
| Fond badge | #DCFCE7 |
| Texte badge | #16A34A |
| Extensions | .jpg, .png, .gif, .webp, .css, .js, .woff2, .svg |

## 9. SECTION CHEMINS EXCLUS
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Format | Monospace |
| Exemples | /api/*, /admin/*, /user/profile, /checkout/* |

## 10. APIS & ENDPOINTS
- GET /cdn/{serviceName}/cache - Paramètres cache
- PUT /cdn/{serviceName}/cache - Modifier paramètres
- POST /cdn/{serviceName}/cache/purge - Purger cache
- POST /cdn/{serviceName}/cache/preload - Précharger URL
