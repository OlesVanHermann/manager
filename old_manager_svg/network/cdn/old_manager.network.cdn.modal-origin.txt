# Cahier des Charges : network/cdn - Modal Origin

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout origine CDN
- **Contexte** : Network CDN
- **Objectif** : Configurer un serveur d'origine backend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 460px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : Nom, adresse, port
- **Cards sélection** : Type origine, protocole
- **OdsCheckbox** : Option backup
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Ajouter une origine" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT NOM ORIGINE
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom de l'origine" |
| Largeur | 450px |
| Placeholder | "origin-backup" |

## 6. SÉLECTION TYPE ORIGINE
| Option | Largeur | Sélectionné |
|--------|---------|-------------|
| Serveur IP | 140px | Oui (bleu) |
| Domaine | 140px | Non |
| S3 Storage | 140px | Non |

### Style sélection
| État | Bordure |
|------|---------|
| Sélectionné | #0050D7 (2px) |
| Non sélectionné | #E5E7EB (1px) |

## 7. INPUT ADRESSE + PORT
| Champ | Largeur | Placeholder |
|-------|---------|-------------|
| Adresse IP/hostname | 350px | "141.94.100.20" |
| Port | 85px | "443" |

## 8. SÉLECTION PROTOCOLE
| Option | Largeur | Sélectionné |
|--------|---------|-------------|
| HTTPS | 120px | Oui |
| HTTP | 120px | Non |

## 9. CHECKBOX BACKUP
| Propriété | Valeur |
|-----------|--------|
| Label | "Utiliser comme backup (failover)" |
| État défaut | Non coché |

## 10. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Ajouter | Primary #0050D7 | 100px |

## 11. APIS & ENDPOINTS
- POST /cdn/{serviceName}/domain/{domain}/origin - Ajouter origine
- GET /cdn/{serviceName}/domain/{domain}/origin - Liste origines
