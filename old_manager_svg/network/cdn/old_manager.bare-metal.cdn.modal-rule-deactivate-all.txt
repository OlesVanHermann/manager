---

## 7. APIS & ENDPOINTS

### 7.1 Désactiver toutes les règles

```
PUT /cdn/{serviceName}/domain/{domainName}/rules/deactivateAll
```

Body:
```json
{
  "pattern": "*.jpg"
}
```

Response:
```json
{
  "taskId": "task-123456",
  "rulesAffected": 5,
  "status": "pending"
}
```
