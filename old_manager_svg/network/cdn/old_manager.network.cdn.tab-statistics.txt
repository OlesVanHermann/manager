# Cahier des Charges : network/cdn - Onglet Statistiques

## 1. VUE D'ENSEMBLE
- **Page** : Statistiques CDN
- **URL** : /network/cdn/{serviceName}?tab=statistics
- **Objectif** : Visualiser les métriques de performance

## 2. STRUCTURE PAGE
- Header avec titre service
- Onglets avec Statistiques actif
- Sélecteur de période (Jour/Semaine/Mois)
- Bouton Rafraîchir
- Cartes métriques (3)
- Graphique évolution trafic

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Conteneur principal
- **OdsButtonGroup** : Sélecteur période
- **OdsButton** : Rafraîchir (outline)
- **OdsTile** : Métriques (3 cartes)
- **OdsChart** : Graphique barres

## 6. CARTES MÉTRIQUES
| Métrique | Valeur | Description |
|----------|--------|-------------|
| Requêtes | 1.2M | Nombre total de requêtes |
| Bande passante | 2.4 TB | Volume de données servies |
| Taux de cache | 94.5% | Cache hit ratio |

## 7. GRAPHIQUE TRAFIC
| Propriété | Valeur |
|-----------|--------|
| Type | Bar chart |
| Titre | "Évolution du trafic" |

## 8. ACTIONS
| Bouton | Style | Position |

## 9. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/statistics - Métriques globales
- GET /cdn/dedicated/{serviceName}/statistics/bandwidth - Bande passante
- GET /cdn/dedicated/{serviceName}/statistics/requests - Requêtes
