## 7. APIS & ENDPOINTS

### 7.1 Vider le cache du domaine

```
POST /cdn/{serviceName}/domain/{domainName}/flush
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "pending",
  "domain": "cdn.example.com"
}
```

### 7.2 Vérifier le statut du flush

```
GET /cdn/{serviceName}/task/{taskId}
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "done",
  "completedAt": "2026-01-02T14:35:00Z"
}
```
