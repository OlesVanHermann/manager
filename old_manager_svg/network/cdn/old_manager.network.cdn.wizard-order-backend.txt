## 7. APIS & ENDPOINTS

```
# Prix backends
GET /cdn/dedicated/{serviceName}/options/backend/price
Query: { quantity: 5 }
→ {
  unitPrice: 10.00,
  currency: "EUR",
  totalHT: 50.00,
  totalTTC: 60.00
}

# Commander backends
POST /cdn/dedicated/{serviceName}/options/backend/order
Body: {
  quantity: 5,
  duration: "P1M"  // ISO 8601 duration
}
→ {
  orderId: "order-123",
  url: "https://www.ovh.com/order/..."
}
```
