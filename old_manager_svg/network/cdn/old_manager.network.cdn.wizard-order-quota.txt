# Cahier des Charges : network/cdn - Wizard Order Quota

## 1. VUE D'ENSEMBLE

**Type:** Wizard commande 3 étapes
**Fonction:** Commander du quota CDN supplémentaire
**Contexte:** Accessible depuis les options CDN

## 2. DIMENSIONS

- Container: 650x440px
- Border radius: 8px

## 3. HEADER WIZARD

- Background: #F9FAFB
- Titre: "Commander du quota CDN" (16px, bold)

## 4. STEPPER (3 étapes)

| Étape | Nom | État |
|-------|-----|------|
| 1 | Tarif | Actif (#0050D7) |
| 2 | Contrats | Inactif |
| 3 | Paiement | Inactif |

## 5. ÉTAPE 1 - TARIF

### Instruction
"Selectionnez le quota supplementaire :" (13px)

### Options radio

| Option | Prix | État |
|--------|------|------|
| 10 To | 50,00 EUR HT | Sélectionné |
| 100 To | 400,00 EUR HT | Non sélectionné |
| 1000 To | 3500,00 EUR HT | Non sélectionné |

### Style radio
- Sélectionné: #0050D7, cercle blanc intérieur
- Non sélectionné: Blanc, bordure #D1D5DB

### Résumé prix
- Total HT: "50,00 EUR" (16px, bold, #DC2626)
- Total TTC: "60,00 EUR" (16px, bold, #DC2626)

## 6. FOOTER WIZARD

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Suivant → | btn-primary | 100x36px |

## 7. APIS & ENDPOINTS

```
# Options quota disponibles
GET /cdn/dedicated/{serviceName}/options/quota
→ [{
  size: 10,
  unit: "TB",
  priceHT: 50.00,
  priceTTC: 60.00
}, ...]

# Commander quota
POST /cdn/dedicated/{serviceName}/options/quota/order
Body: {
  quotaSize: 10  // TB
}
→ {
  orderId: "order-456",
  url: "https://www.ovh.com/order/..."
}
```
