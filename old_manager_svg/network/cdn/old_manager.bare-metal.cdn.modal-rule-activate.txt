# Cahier des Charges : bare-metal/cdn - Modal Activer Règle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Confirmer l'activation d'une règle de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 600px |
| Page hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsText** : Question de confirmation
- **OdsButton** : Annuler, Activer

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer la règle de cache" |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir activer la règle" |
| Nom règle | "{ruleName}" (bleu #0050D7, font-weight 500) |

## 6. FOOTER
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Activer | Primary | #0050D7 | Active la règle |

## 7. APIS & ENDPOINTS
- PUT /cdn/dedicated/{serviceName}/domains/{domainName}/cacheRules/{ruleId} - Activer règle (status: on)
