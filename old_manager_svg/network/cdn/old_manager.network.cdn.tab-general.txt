# Cahier des Charges : network/cdn - Onglet Général

## 1. VUE D'ENSEMBLE
- **Page** : Informations générales CDN
- **URL** : /network/cdn/{serviceName}?tab=general
- **Objectif** : Afficher les infos et configuration du service

## 2. STRUCTURE PAGE
- Header avec titre service
- Onglets avec Général actif
- Section informations du service (4 tuiles)
- Section configuration
- Section actions

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Conteneur principal
- **OdsTile** : Informations (2x2 grille)
- **OdsBadge** : Statut ok (vert), Activé (vert)
- **OdsButton** : Rafraîchir, Vider le cache

## 4. ONGLETS
| Onglet | Actif |
|--------|-------|
| Général | ✓ |
| Domaines | - |
| Statistiques | - |
| Tâches | - |

## 5. INFORMATIONS DU SERVICE
| Label | Valeur | Style |
|-------|--------|-------|
| Nom du service | cdn-abc123.ovh.net | Monospace |
| Anycast | 1.2.3.4 | Monospace |
| Offre | CDN Infrastructure | Normal |
| Statut | ok | Badge vert |

## 6. CONFIGURATION
| Option | Description | Badge |
|--------|-------------|-------|
| Règle de cache | Permet de définir des règles de mise en cache personnalisées | Activé (vert) |

## 7. ACTIONS
| Bouton | Style | Taille |
|--------|-------|--------|
| Rafraîchir | Primary bleu | 140x44px |
| Vider le cache | Outline gris | 140x44px |

## 8. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName} - Infos service
- PUT /cdn/dedicated/{serviceName} - Modifier config
- POST /cdn/dedicated/{serviceName}/flush - Purge globale
