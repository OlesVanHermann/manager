## 7. APIS & ENDPOINTS

```
# Backends disponibles
GET /cdn/dedicated/{serviceName}/backends
→ ["192.168.1.100", "192.168.1.101", "backend.example.com"]

# Modifier backend
PUT /cdn/dedicated/{serviceName}/domains/{domain}/backend
Body: {
  backend: "192.168.1.101"
}
→ {
  taskId: "task-123",
  status: "updating"
}
```
