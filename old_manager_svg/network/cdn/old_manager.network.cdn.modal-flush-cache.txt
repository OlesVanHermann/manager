# Cahier des Charges : network/cdn - Modal Flush Cache

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation warning
**Fonction:** Purger le cache CDN d'un domaine
**Contexte:** Action depuis la gestion du domaine

## 2. DIMENSIONS

- Modal: 450x230px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #FEF3C7 (warning)
- Bordure: #FDE68A
- Icône: Cercle orange avec "!"
- Titre: "Purger le cache CDN" (15px, bold, #92400E)
- Bouton fermer: × à droite

## 4. CONTENU

### Question
"Etes-vous sur de vouloir purger le cache du domaine ?" (13px)

### Domaine
"example.com" (13px, bold)

### Encart warning (400x50px)
- Background: #FEF3C7
- Bordure: #FDE68A
- Texte 1: "Cette action invalidera tous les fichiers en cache."
- Texte 2: "Le cache sera reconstruit lors des prochaines requetes."

## 5. FOOTER MODAL

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x36px |
| Purger | btn-warning (#F59E0B) | 100x36px |

## 6. APIS & ENDPOINTS

```
# Purger le cache
POST /cdn/dedicated/{serviceName}/domains/{domain}/flush
→ {
  taskId: "task-789",
  status: "flushing"
}

# Vérifier statut
GET /cdn/dedicated/{serviceName}/domains/{domain}/flush/{taskId}
→ {
  taskId: "task-789",
  status: "done",
  filesInvalidated: 15420
}
```
