# Cahier des Charges : network/cdn - Onglet Tâches

## 1. VUE D'ENSEMBLE
- **Page** : Historique des tâches CDN
- **URL** : /network/cdn/{serviceName}?tab=tasks
- **Objectif** : Suivre les opérations du service

## 2. STRUCTURE PAGE
- Header avec titre service
- Onglets avec Tâches actif
- Bouton Rafraîchir
- Tableau des tâches

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Conteneur principal
- **OdsButton** : Rafraîchir (outline)
- **OdsTable** : Historique tâches
- **OdsBadge** : done (vert), running (jaune)

## 4. ONGLETS
| Onglet | Actif |
|--------|-------|
| Général | - |
| Domaines | - |
| Statistiques | - |
| Tâches | ✓ |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Fonction | Text | Type d'opération |
| Statut | Badge | done/running |
| Démarré | Datetime | Date de début |
| Terminé | Datetime | Date de fin (ou -) |

## 6. BADGES STATUT
| Statut | Badge | Couleur |
|--------|-------|---------|
| done | success | Vert |
| running | warning | Jaune |
| error | error | Rouge |

## 7. TYPES DE TÂCHES
| Fonction | Description |
|----------|-------------|
| flush | Purge du cache |
| addDomain | Ajout d'un domaine |
| configureDomain | Configuration domaine |
| deleteDomain | Suppression domaine |
