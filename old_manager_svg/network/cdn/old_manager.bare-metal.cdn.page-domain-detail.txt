# Cahier des Charges : bare-metal/cdn - Page Domain Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail domaine CDN
- **Contexte** : CDN Dedicated
- **Objectif** : Gérer la configuration d'un domaine CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsLink** : Lien retour
- **OdsText** : Titres et valeurs
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tuiles informations
- **OdsBadge** : Statut
- **OdsButton** : Actions

## 10. BOUTONS ACTIONS
| Bouton | Style | Largeur |
|--------|-------|---------|
| Purger le cache | Fond #0050D7, texte blanc | 140px |
| Mode bypass | Fond blanc, bordure grise | 140px |

| Supprimer | Fond blanc, bordure #DC2626, texte rouge | 120px |

## 11. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/domains/{domain} - Détails domaine
- PUT /cdn/dedicated/{serviceName}/domains/{domain} - Modifier config
- POST /cdn/dedicated/{serviceName}/domains/{domain}/purge - Purger cache
- DELETE /cdn/dedicated/{serviceName}/domains/{domain} - Supprimer
- GET /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules - Règles cache
