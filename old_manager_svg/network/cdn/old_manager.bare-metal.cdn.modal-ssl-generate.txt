# Cahier des Charges : bare-metal/cdn - Modal Générer Let's Encrypt

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire avec avertissement
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Générer un certificat Let's Encrypt automatique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement remplacement
- **OdsInput** : Nom du certificat
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Générer un certificat Let's Encrypt" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. MESSAGE D'AVERTISSEMENT (si certificat existant)
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Titre | "Attention" |
| Texte | "Un certificat existe déjà. Il sera remplacé par le nouveau." |
| Couleur texte | #92400E |

## 6. CONTENU

### Nom du certificat
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du certificat" |
| Type | OdsInput |
| Largeur | 400px |
| Exemple | letsencrypt-2024 |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (#0050D7) | Lance la génération |

## 8. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/ssl/letsencrypt - Générer certificat Let's Encrypt
