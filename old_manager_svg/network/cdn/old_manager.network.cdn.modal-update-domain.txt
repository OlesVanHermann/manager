# Cahier des Charges : network/cdn - Modal activer/désactiver bypass

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page domaines CDN
- **Objectif** : Activer ou désactiver le mode bypass pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message et info
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le mode bypass" / "Désactiver le mode bypass" |
| Fond | Bleu (#DBEAFE) |
| Couleur texte | Bleu (#0050D7) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir activer/désactiver le mode bypass pour ce domaine ?" |
| Domaine | Nom du domaine affiché (gras) |
| Info | "Le mode bypass désactive temporairement le CDN pour ce domaine." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary | Active/désactive le bypass |

## 7. APIS & ENDPOINTS
- PUT /cdn/{serviceName}/domains/{domain} - Modifier bypass (status: on/off)
