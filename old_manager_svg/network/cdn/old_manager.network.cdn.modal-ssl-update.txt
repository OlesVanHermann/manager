# Cahier des Charges : network/cdn - Modal mettre à jour certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de mise à jour
- **Contexte** : Page SSL CDN
- **Objectif** : Mettre à jour un certificat SSL existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description
- **OdsTextarea** : Certificat, Clé privée, Chaîne
- **OdsButton** : Annuler, Mettre à jour

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mettre à jour le certificat SSL" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Mettez à jour votre certificat SSL existant." |

## 6. FORMULAIRE

### Champs
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nouveau certificat SSL * | Textarea (100px) | "-----BEGIN CERTIFICATE-----..." | Oui |
| Nouvelle clé privée * | Textarea (80px) | "-----BEGIN PRIVATE KEY-----..." | Oui |
| Chaîne de certificats | Textarea (40px) | "Certificats intermédiaires..." | Non |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Mettre à jour | Primary | Met à jour le certificat |

## 8. APIS & ENDPOINTS
- PUT /cdn/{serviceName}/ssl/{sslId} - Mettre à jour certificat SSL
