## 9. APIS & ENDPOINTS

### 9.1 Créer une règle de cache

```
POST /cdn/{serviceName}/domain/{domainName}/rule
```

Body:
```json
{
  "fileType": "extension",
  "fileMatch": ".jpg,.png,.gif,.webp",
  "cacheType": "enabled",
  "ttl": 3600
}
```

Response:
```json
{
  "ruleId": "rule-123456",
  "taskId": "task-789",
  "status": "pending"
}
```

### 9.2 Types de fichiers disponibles

```
GET /cdn/{serviceName}/domain/{domainName}/rule/types
```

Response:
```json
[
  { "type": "extension", "label": "Extension" },
  { "type": "folder", "label": "Dossier" },
  { "type": "url", "label": "URL exacte" }
]
```
