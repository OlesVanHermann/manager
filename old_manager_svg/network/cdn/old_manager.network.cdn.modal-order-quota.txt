# Cahier des Charges : network/cdn - Modal commander quota

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard commande (3 étapes)
- **Contexte** : Dashboard CDN
- **Objectif** : Commander du quota de bande passante supplémentaire

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : 3 étapes
- **OdsRadio** : Options quota
- **OdsMessage** : Info
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander du quota CDN" |
| Bouton fermer | × |

## 5. STEPPER (3 étapes)
| Étape | État |
|-------|------|
| 1 | Actif |
| 2 | Inactif |
| 3 | Inactif |

## 6. ÉTAPE 1 - SÉLECTION QUOTA

### Question
| Texte |
|-------|
| "Sélectionnez le quota à commander :" |

### Options
| Option | Prix |
|--------|------|
| 100 Go | 9,99 € HT (sélectionné) |
| 500 Go | 39,99 € HT |
| 1 To | 69,99 € HT |
| 10 To | 499,99 € HT |

### Message info
| Type | Contenu |
|------|---------|
| Info (bleu) | "ℹ️ Le quota sera ajouté à votre CDN immédiatement après paiement." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary | Passe à l'étape suivante |

## 8. APIS & ENDPOINTS
- GET /cdn/{serviceName}/quota - Quota disponible
- POST /order/cdn/{serviceName}/quota - Commander quota
