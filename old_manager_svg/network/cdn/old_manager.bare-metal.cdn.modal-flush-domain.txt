# Cahier des Charges : bare-metal/cdn - Modal purger le cache

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation
- **Contexte** : CDN Dedicated
- **Objectif** : Purger le cache d'un domaine CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question, info
- **OdsButton** : Annuler, Purger

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Purger le cache" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Voulez-vous purger le cache du domaine {domain} ?" |

## 6. INFORMATION
| Texte |
|-------|
| "Cette opération supprimera tous les fichiers en cache pour ce domaine." |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Purger | Warning (orange #F59E0B) | Purge le cache |

## 8. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domain}/flush - Purger le cache
