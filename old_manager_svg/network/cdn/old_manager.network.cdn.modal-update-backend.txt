# Cahier des Charges : network/cdn - Modal modifier backend

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard (2 étapes)
- **Contexte** : Page domaines CDN
- **Objectif** : Modifier le serveur backend d'un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : 2 étapes
- **OdsSelect** : Backend existant
- **OdsInput** : Nouveau backend (IP)
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le serveur backend" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. STEPPER (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Actif |
| 2 | Confirmation | Inactif |

## 6. ÉTAPE 1 - SÉLECTION

### Question
| Texte |
|-------|
| "Quel backend souhaitez-vous utiliser pour {domainName} ?" |

### Informations affichées
| Élément | Description |
|---------|-------------|
| Backend actuel | Adresse IP du backend en cours |

### Champs
| Champ | Type | Description |
|-------|------|-------------|
| Backend existant | Select | Liste des backends disponibles |
| Ou nouveau backend | Input | Adresse IP du nouveau backend |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary | Passe à l'étape de confirmation |

## 8. APIS & ENDPOINTS
- GET /cdn/{serviceName}/domains/{domain} - Récupérer le domaine
- GET /cdn/{serviceName}/option/backends - Liste des backends
- PUT /cdn/{serviceName}/domains/{domain} - Modifier backend
