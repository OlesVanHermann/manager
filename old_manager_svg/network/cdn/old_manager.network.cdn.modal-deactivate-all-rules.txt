# Cahier des Charges : network/cdn - Modal Deactivate All Rules

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation warning
**Fonction:** Désactiver toutes les règles de cache
**Contexte:** Action depuis la page des règles

## 2. DIMENSIONS

- Modal: 450x150px
- Border radius: 8px

## 3. HEADER MODAL

- Background: #FEF3C7 (warning)
- Titre: "Desactiver toutes les regles" (15px, bold, #92400E)
- Bouton fermer: × à droite

## 4. CONTENU

### Question
"Voulez-vous desactiver toutes les regles de cache ?" (13px)

## 5. FOOTER MODAL

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x32px |
| Desactiver | btn-warning (#92400E) | 100x32px |

## 6. APIS & ENDPOINTS

```
# Désactiver toutes les règles
POST /cdn/dedicated/{serviceName}/domains/{domain}/rules/deactivateAll
→ {
  taskId: "task-456",
  rulesDeactivated: 15
}
```
