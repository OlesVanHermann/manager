# Cahier des Charges : network/cdn - Modal ajouter règle de cache

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard (2 étapes)
- **Contexte** : Page règles de cache CDN
- **Objectif** : Créer une nouvelle règle de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : Indicateur étapes (1-2)
- **OdsSelect** : Type de fichier
- **OdsInput** : Correspondance, TTL
- **OdsRadio** : Type de cache
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer une règle de cache" |
| Bouton fermer | × |

## 5. STEPPER
| Étape | État |
|-------|------|
| 1 | Actif (bleu) |
| 2 | Inactif (gris) |

## 6. FORMULAIRE ÉTAPE 1

### Description
| Texte |
|-------|
| "Configurez les paramètres de la règle de cache :" |

### Champs
| Champ | Type | Options/Placeholder |
|-------|------|---------------------|
| Type de fichier | Select | Extension, URI, Header... |
| Correspondance | Input | "Ex: .jpg, .png, .css" |
| Type de cache | Radio | Forcer le cache (défaut), Pas de cache |
| TTL (durée de vie) | Input + suffix | 3600 secondes |

## 7. OPTIONS TYPE DE CACHE
| Option | Description |
|--------|-------------|
| Forcer le cache | Met en cache selon le TTL défini |
| Pas de cache | Désactive le cache pour ces fichiers |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary | Passe à l'étape 2 |

## 9. APIS & ENDPOINTS
- POST /cdn/{serviceName}/domains/{domain}/cacheRules - Créer règle de cache
