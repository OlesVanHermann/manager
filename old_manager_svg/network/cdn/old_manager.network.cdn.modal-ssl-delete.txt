# Cahier des Charges : network/cdn - Modal supprimer certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page SSL CDN
- **Objectif** : Supprimer un certificat SSL

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message warning
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le certificat SSL" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir supprimer ce certificat ?" |
| Warning | "HTTPS sera désactivé sur votre CDN." (rouge) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le certificat |

## 7. APIS & ENDPOINTS
- DELETE /cdn/{serviceName}/ssl/{sslId} - Supprimer certificat SSL
