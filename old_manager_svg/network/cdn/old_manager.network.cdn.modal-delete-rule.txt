# Cahier des Charges : network/cdn - Modal supprimer règle de cache

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page règles de cache CDN
- **Objectif** : Supprimer définitivement une règle de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message warning
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la règle de cache" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir supprimer cette règle ?" (rouge, gras) |
| Règle | ".jpg, .png, .gif" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime la règle |

## 7. APIS & ENDPOINTS
- DELETE /cdn/{serviceName}/domains/{domain}/cacheRules/{ruleId} - Supprimer règle
