## 7. APIS & ENDPOINTS

### 7.1 Réactiver le cache CDN

```
POST /cdn/{serviceName}/domain/{domainName}/enable
```

Response:
```json
{
  "taskId": "task-123456",
  "status": "pending"
}
```
