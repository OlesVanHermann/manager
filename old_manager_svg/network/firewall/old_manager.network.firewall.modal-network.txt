# Cahier des Charges : network/firewall - Modal Network

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout réseau au firewall
- **Contexte** : Network Firewall
- **Objectif** : Ajouter un réseau à la liste blanche/noire

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : IP/réseau, description
- **Cards sélection** : Action (autoriser/bloquer)
- **OdsMessage** : Info application
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Ajouter un réseau" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT ADRESSE IP/RÉSEAU
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP ou réseau" |
| Largeur | 450px |
| Placeholder | "192.168.1.0/24 ou 10.0.0.1" |

## 6. INPUT DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom/Description" |
| Largeur | 450px |
| Placeholder | "Réseau interne production" |

## 7. SÉLECTION ACTION
| Option | Largeur | Description |
|--------|---------|-------------|
| Autoriser | 140px | Liste blanche |
| Bloquer | 140px | Liste noire |

## 8. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |
| Icône | ℹ️ |
| Texte | "Ce réseau sera ajouté à la liste blanche/noire globale et appliqué à toutes les IP du firewall." |

## 9. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Ajouter | Primary #0050D7 | 100px |

## 10. APIS & ENDPOINTS
- POST /ip/{ip}/firewall/network - Ajouter réseau
- GET /ip/{ip}/firewall/network - Liste réseaux
- DELETE /ip/{ip}/firewall/network/{networkId} - Supprimer
