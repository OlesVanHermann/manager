# Cahier des Charges : network/load-balancer - Onglet Live Tail

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet logs temps réel
- **Contexte** : Dashboard Load Balancer - Logs
- **Objectif** : Visualiser les logs d'accès en temps réel

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation principale
- **OdsButton** : Sous-navigation, Pause, Effacer
- **OdsIndicator** : Point vert "En direct"
- **Terminal** : Container logs (fond sombre)

## 5. BARRE DE CONTRÔLE

### Contrôles
| Élément | Type | Action |
|---------|------|--------|
| Indicateur live | Badge | Point vert + "En direct" |
| Pause | Button | Pause le flux |
| Effacer | Button | Vide le terminal |

## 6. TERMINAL LOGS

### Container
| Propriété | Valeur |
|-----------|--------|
| Fond | Sombre (#1F2937) |
| Bordure | Arrondie (8px) |
| Police | Monospace |
| Taille | 12px |
| Hauteur | 330px |

### Format des logs
```
{timestamp} {client_ip}:{port} → {backend_ip}:{port} {status} {message} {duration} {method} {path}
```

### Code couleur
| Code HTTP | Couleur | Description |
|-----------|---------|-------------|
| 2xx | Bleu (#60A5FA) | Success |
| 4xx | Jaune (#FBBF24) | Client Error |
| 5xx | Rouge (#F87171) | Server Error |

## 7. LÉGENDE
| Couleur | Label |
|---------|-------|
| Bleu | 2xx Success |
| Jaune | 4xx Client Error |
| Rouge | 5xx Server Error |

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/logs/live - Logs temps réel (WebSocket)
