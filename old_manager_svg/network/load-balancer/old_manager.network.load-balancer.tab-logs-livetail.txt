# Cahier des Charges : network/load-balancer - Tab Logs Live Tail

## 1. VUE D'ENSEMBLE

**Type:** Onglet avec console logs temps réel
**Fonction:** Afficher les logs en streaming depuis l'onglet Logs
**Contexte:** Dashboard LB > Onglet "Logs" > Sous-onglet "Live Tail"

## 2. NAVIGATION

### Onglets principaux
| Onglet | État |
|--------|------|
| Accueil | Inactif |
| Fermes | Inactif |
| Frontends | Inactif |
| Graphiques | Inactif |
| Logs | Actif (#0050D7 avec underline) |
| Tâches | Inactif |

### Sous-onglets Logs
| Sous-onglet | Style |
|-------------|-------|
| Data Streams | btn-outline (white) |
| Live Tail | btn-primary (#0050D7) |

## 3. BARRE DE CONTRÔLE

### Indicateur live
- Cercle vert (6px radius, #10B981)
- Texte: "En direct" (#10B981, 12px)

### Boutons de contrôle
| Bouton | Dimensions | Style |
|--------|------------|-------|
| Pause | 80x36px | btn-outline |
| Effacer | 80x36px | btn-outline |

## 4. CONSOLE LOGS

### Container
- Dimensions: 1100x330px
- Background: #1F2937 (dark)
- Border-radius: 8px
- Font: monospace, 12px

### Format des entrées
```
YYYY-MM-DD HH:MM:SS IP:port → IP:port STATUS message latency METHOD /path
```

### Couleurs des logs
| Type | Couleur |
|------|---------|
| 2xx Success | #60A5FA (bleu) |
| 4xx Client Error | #FBBF24 (jaune) |
| 5xx Server Error | #F87171 (rouge) |

### Curseur
- Rectangle vert clignotant (#10B981, opacity 0.8)
- Dimensions: 10x18px

## 5. LÉGENDE

### Position: Bas de la page
| Indicateur | Couleur | Label |
|------------|---------|-------|
| Cercle | #60A5FA | 2xx Success |
| Cercle | #FBBF24 | 4xx Client Error |
| Cercle | #F87171 | 5xx Server Error |

```
WSS /ipLoadbalancing/{serviceName}/logs/stream
```
