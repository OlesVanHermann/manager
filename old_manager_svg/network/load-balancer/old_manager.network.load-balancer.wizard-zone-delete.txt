# Cahier des Charges : network/load-balancer - Wizard Zone Delete

## 1. VUE D'ENSEMBLE

**Type:** Formulaire wizard avec checkboxes
**Fonction:** Supprimer des zones de déploiement du Load Balancer
**Contexte:** Réduction de la couverture géographique

## 2. DIMENSIONS

- Container: 820x290px
- Background: white
- Border-radius: 8px
- Border: #E5E7EB

## 3. NAVIGATION

### Lien retour
- "← Supprimer des zones" (14px, text-body)

## 4. CONTENU

### Description
- "Vous avez X zones configurées. Sélectionnez les zones à supprimer."
- Font-size: 13px

### Titre section
- "Zones disponibles" (13px, font-weight: 500)

### Checkbox
| État | Style |
|------|-------|
| Checked | Background #0050D7, ✓ blanc |
| Unchecked | Background white, border #D1D5DB |
