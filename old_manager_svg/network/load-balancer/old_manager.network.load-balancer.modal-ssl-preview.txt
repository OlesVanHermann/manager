## 6. APIS & ENDPOINTS

### 6.1 Obtenir les détails d'un certificat SSL

```
GET /ipLoadbalancing/{serviceName}/ssl/{sslId}
```

Response:
```json
{
  "sslId": 789012,
  "displayName": "ssl-certificate-example",
  "serial": "A1:B2:C3:D4:E5:F6:78:90",
  "subject": "CN=*.example.com",
  "fingerprint": "SHA256:AB:CD:EF:12:34:56:78:90...",
  "type": "custom",
  "expireDate": "2026-12-31T23:59:59Z"
}
```
