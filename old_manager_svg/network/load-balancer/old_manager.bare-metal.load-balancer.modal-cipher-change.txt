# Cahier des Charges : bare-metal/load-balancer - Modal Modifier Cipher SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Configuration Load Balancer SSL
- **Objectif** : Modifier le niveau de cipher SSL pour le load balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 350px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsRadio** : Options cipher
- **OdsButton** : Annuler, Modifier
- **OdsText** : Descriptions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le cipher SSL" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Options Cipher
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Intermediate | ○ | Recommandé pour la compatibilité avec la plupart des navigateurs |
| Modern | ● | TLS 1.3 uniquement, sécurité maximale mais compatibilité réduite |
| Old | ○ | Pour la compatibilité avec les anciens navigateurs (TLS 1.0+) |

### Style Radio
| État | Apparence |
|------|-----------|
| Sélectionné | Cercle bleu #0050D7 avec point blanc |
| Non sélectionné | Cercle blanc avec border #E5E7EB |

### Format option
| Propriété | Valeur |
|-----------|--------|
| Titre | 14px, font-weight 500, noir |
| Description | 12px, gris #6B7280 |
| Espacement vertical | 70px entre options |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Modifier | Primary (#0050D7) | Applique le changement |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/ssl/cipher - Cipher actuel
- PUT /ipLoadbalancing/{serviceName}/ssl/cipher - Modifier cipher
