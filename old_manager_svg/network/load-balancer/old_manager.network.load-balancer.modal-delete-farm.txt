# Cahier des Charges : network/load-balancer - Modal Supprimer Ferme

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Déclencheur** : Action "Supprimer" dans le menu ferme
- **Objectif** : Confirmer la suppression d'une ferme

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée compacte (500x250px)
- Header avec titre et bouton fermer
- Message de confirmation
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titre et message
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Supprimer la ferme" |
| Bouton fermer | × (coin droit) |
| Séparateur | Ligne grise |

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Êtes-vous sûr de vouloir supprimer la ferme" |
| Nom ferme | "farm-web-prod (78901)" (gras) |
| Ponctuation | "?" |

## 6. FOOTER MODAL
| Bouton | Style | Position | Dimensions |
|--------|-------|----------|------------|
| Annuler | Outline (bordure grise) | Centre-droite | 80x40px |
| Supprimer | Danger (rouge #DC2626) | Droite | 100x40px |

## 7. COMPORTEMENT
| Action | Résultat |
|--------|----------|
| Clic Annuler | Ferme la modal sans action |
| Clic Supprimer | Supprime la ferme et ses serveurs |
| Clic × | Ferme la modal sans action |
| Clic overlay | Ferme la modal sans action |

## 8. AVERTISSEMENTS
- La suppression est **irréversible**
- Tous les serveurs de la ferme seront également supprimés
- Les frontends liés à cette ferme perdront leur configuration

## 9. APIS & ENDPOINTS
- DELETE /ipLoadbalancing/{serviceName}/http/farm/{farmId} - Supprimer ferme HTTP
- DELETE /ipLoadbalancing/{serviceName}/tcp/farm/{farmId} - Supprimer ferme TCP
- DELETE /ipLoadbalancing/{serviceName}/udp/farm/{farmId} - Supprimer ferme UDP
