# Cahier des Charges : network/load-balancer - Wizard Add Farm

## 1. VUE D'ENSEMBLE

**Type:** Page wizard/formulaire
**Fonction:** Créer une nouvelle ferme de serveurs
**Contexte:** Load Balancer OVHcloud

## 2. DIMENSIONS

- Page: 950x700px
- Background: #F9FAFB
- Card: 870x530px

## 3. NAVIGATION

### Lien retour
- "← Retour aux fermes" (lien bleu #0050D7)

### Titre page
- "Ajouter une ferme de serveurs" (24px, font-weight: 600)

### Description
- "La ferme de serveurs est la partie de votre Load Balancer qui sert à répartir le trafic sur les serveurs."

## 4. FORMULAIRE PRINCIPAL

### Champs de base
| Champ | Type | Largeur | Valeur |
|-------|------|---------|--------|
| Nom (facultatif) | Input | 380px | Placeholder: "Nom de la ferme" |
| Protocole | Select | 200px | HTTPS |
| Port | Input | 180px | 443 |
| Datacenter | Select | 200px | rbx |
| Réseau privé | Select | 200px | pn-123456 |

### Helpers
- Protocole: "Non modifiable après la création"
- Port: "Port attaché à la ferme (1 à 65535)"

## 5. SECTION AVANCÉE

### Header
- "Paramètres avancés" (16px, font-weight: 600)
- Toggle: "Afficher ▼" (lien bleu)

### Info box
- Background: #EFF6FF
- Border: #BFDBFE
- Texte: "Les valeurs par défaut les plus courantes sont préréglées. Vous pouvez les modifier à tout moment."

### Champs avancés
| Champ | Type | Largeur | Valeur |
|-------|------|---------|--------|
| Mode de répartition | Select | 250px | Round-robin |
| Suivi de session | Select | 250px | Cookie |
| Sonde de disponibilité | Select | 200px | HTTP |

### Options Mode de répartition
- Round-robin
- Least connections
- Source IP
- URI
- First

### Options Suivi de session
- Cookie
- Source IP
- Aucun

### Bouton sonde
- "Paramètres sonde" (outline bleu)

## 6. ACTIONS

| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter | Primary (#0050D7) | Créer ferme |
| Annuler | Ghost (texte bleu) | Retour liste |

## 7. APIS & ENDPOINTS

```
POST /ipLoadbalancing/{serviceName}/http/farm
GET /ipLoadbalancing/{serviceName}/zone
GET /ipLoadbalancing/{serviceName}/vrack/network
```
