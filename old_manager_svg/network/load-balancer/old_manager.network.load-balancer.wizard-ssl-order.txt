# Cahier des Charges : network/load-balancer - Wizard SSL Order

## 1. VUE D'ENSEMBLE

**Type:** Page wizard/formulaire
**Fonction:** Commander un certificat SSL pour le load balancer
**Contexte:** Load Balancer OVHcloud

## 2. DIMENSIONS

- Page: 900x650px
- Background: #F9FAFB
- Card: 820x500px

## 3. NAVIGATION

### Lien retour
- "← Retour aux certificats" (lien bleu #0050D7)

### Titre page
- "Commander un certificat SSL" (24px, font-weight: 600)
