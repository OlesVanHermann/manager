# Cahier des Charges : network/load-balancer - Onglet Certificats SSL

## 1. VUE D'ENSEMBLE
- **Page** : Liste des certificats SSL
- **URL** : /network/load-balancer/{serviceName}/ssl
- **Objectif** : Gérer les certificats SSL du Load Balancer

## 2. STRUCTURE PAGE
- Sidebar Network
- Breadcrumb
- Onglets navigation
- Description explicative
- Bouton d'ajout
- Tableau des certificats

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Accueil/Frontends/Fermes de serveurs/Certificats SSL (actif)
- **OdsText** : Titre et description
- **OdsButton** : "Ajouter un certificat SSL" (outline)
- **OdsDatagrid** : Liste des certificats
- **OdsBadge** : Type de certificat
- **OdsActionMenu** : Actions par certificat

## 4. EN-TÊTE SECTION
| Élément | Contenu |
|---------|---------|
| Titre | "Certificats SSL" |
| Description | "Les certificats configurés sont automatiquement disponibles pour l'ensemble des frontends sur lesquels l'option SSL est activée." |
| Bouton | "+ Ajouter un certificat SSL" (180x36px, outline) |

## 5. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Certificat | Text | ✓ | Nom du certificat |
| Sujet | Text | ✓ | Domaines couverts |
| Type | Badge | ✓ | Custom / Let's Encrypt |
| Actions | Menu ⋮ | - | Actions contextuelles |

## 6. DONNÉES EXEMPLE
| Certificat | Sujet | Type |
|------------|-------|------|
| cert-main-domain | *.example.com, example.com | Custom (bleu) |
| cert-api | api.example.com | Let's Encrypt (bleu) |

## 7. TYPES DE CERTIFICAT
| Type | Badge | Description |
|------|-------|-------------|
| Custom | Bleu info | Certificat importé manuellement |
| Let's Encrypt | Bleu info | Certificat automatique gratuit |

## 8. ACTION MENU (ouvert)
| Action | Style |
|--------|-------|
| Voir l'aperçu | Normal |
| --- | Séparateur |
| Modifier | Normal |
| Mettre à jour | Normal |
| --- | Séparateur |
| Supprimer | Danger (rouge) |

## 9. INFORMATIONS CERTIFICAT (aperçu)
| Champ | Description |
|-------|-------------|
| CN | Common Name |
| SAN | Subject Alternative Names |
| Émetteur | Autorité de certification |
| Validité | Date début - Date fin |
| Empreinte | SHA-256 fingerprint |

## 10. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/ssl - Liste certificats
- GET /ipLoadbalancing/{serviceName}/ssl/{id} - Détails certificat
- POST /ipLoadbalancing/{serviceName}/ssl - Ajouter certificat
- PUT /ipLoadbalancing/{serviceName}/ssl/{id} - Modifier certificat
- DELETE /ipLoadbalancing/{serviceName}/ssl/{id} - Supprimer certificat
- POST /ipLoadbalancing/{serviceName}/ssl/{id}/regenerate - Renouveler Let's Encrypt
