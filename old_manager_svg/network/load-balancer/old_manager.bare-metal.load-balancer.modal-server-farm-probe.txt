# Cahier des Charges : bare-metal/load-balancer - Modal Configurer Health Check

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Gestion Load Balancer
- **Objectif** : Configurer le health check (probe) d'une ferme de serveurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 520px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : URL, Pattern
- **OdsCheckbox** : Force SSL
- **OdsQuantity** : Port, Intervalle
- **OdsButtonGroup** : Méthode, Correspondance
- **OdsButton** : Annuler, OK

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configurer le health check" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Champ URL
| Propriété | Valeur |
|-----------|--------|
| Label | "URL" |
| Type | OdsInput |
| Largeur | 450px |
| Exemple | /health |

### Checkbox Force SSL
| Propriété | Valeur |
|-----------|--------|
| Label | "Forcer SSL" |
| État par défaut | Non coché |

### Champ Port
| Propriété | Valeur |
|-----------|--------|
| Label | "Port" |
| Type | OdsQuantity |
| Largeur | 150px |
| Boutons | − / + |
| Valeur par défaut | 8080 |

### Sélection Méthode
| Propriété | Valeur |
|-----------|--------|
| Label | "Méthode" |
| Type | ButtonGroup |
| Options | GET (sélectionné), HEAD |
| Style sélectionné | Fond #0050D7, texte blanc |

### Sélection Correspondance
| Propriété | Valeur |
|-----------|--------|
| Label | "Correspondance" |
| Type | ButtonGroup |
| Options | status (sélectionné), contains, default |
| Style sélectionné | Fond #0050D7, texte blanc |

### Champ Pattern
| Propriété | Valeur |
|-----------|--------|
| Label | "Pattern" |
| Type | OdsInput |
| Largeur | 450px |
| Placeholder | "Regex ou valeur attendue" |

### Champ Intervalle
| Propriété | Valeur |
|-----------|--------|
| Label | "Intervalle (secondes)" |
| Type | OdsQuantity |
| Largeur | 150px |
| Boutons | − / + |
| Valeur par défaut | 30 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| OK | Primary (#0050D7) | Enregistre la configuration |

## 7. APIS & ENDPOINTS
- PUT /ipLoadbalancing/{serviceName}/farm/{farmId} - Modifier probe de la ferme
