## 6. APIS & ENDPOINTS

### 6.1 Obtenir les détails d'un serveur

```
GET /ipLoadbalancing/{serviceName}/farm/{farmId}/server/{serverId}
```

Response:
```json
{
  "serverId": 123456,
  "displayName": "backend-server-01",
  "address": "192.168.1.100",
  "port": 8080,
  "backup": false,
  "probe": true,
  "ssl": true,
  "cookie": "server-01-cookie",
  "chain": null,
  "proxyProtocolVersion": "v2",
  "weight": 100,
  "status": "active"
}
```
