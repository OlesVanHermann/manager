# Cahier des Charges : network/load-balancer - Modal Edit Probe

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Configurer la sonde de health check
**Contexte:** Configuration ferme backend

## 2. DIMENSIONS

- Modal: 500x450px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Background | #000E9C |
| Titre | "Configurer la sonde" (15px, font-weight: 600, white) |
| Close button | × (white) |

## 4. FORMULAIRE

### Champs
| Champ | Type | Default | Description |
|-------|------|---------|-------------|
| URL de la sonde | Input text | /health | Chemin HTTP à interroger |
| Forcer SSL | Checkbox | checked | Force connexion HTTPS |
| Port | Input number | 443 | Port de la sonde |
| Méthode | Radio group | GET | GET / HEAD / OPTIONS |
| Correspondance | Radio group | status | status / contains / matches |
| Intervalle (sec) | Input number | 30 | Intervalle entre checks |

### Styles checkbox/radio
- Selected: #0050D7 avec checkmark/point blanc
- Unselected: border #D1D5DB

## 5. FOOTER MODAL

| Bouton | Style |
|--------|-------|
| Annuler | Secondary (border) |
| Valider | Primary (#0050D7) |

## 6. APIS & ENDPOINTS

```
GET /ipLoadbalancing/{serviceName}/http/farm/{farmId}
PUT /ipLoadbalancing/{serviceName}/http/farm/{farmId}
```
