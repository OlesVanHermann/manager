# Cahier des Charges : network/load-balancer - Modal SSL Rename

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire simple
**Fonction:** Renommer un certificat SSL
**Contexte:** Gestion certificats HTTPS

## 2. DIMENSIONS

- Modal: 450x200px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Background | #000E9C |
| Titre | "Renommer le certificat SSL" (15px, font-weight: 600, white) |
| Close button | × (white) |

## 4. FORMULAIRE

| Champ | Type | Description |
|-------|------|-------------|
| Nom d'affichage | Input text | Nouveau nom du certificat |

### Style input
- Height: 36px
- Border: #D1D5DB
- Border-radius: 4px
- Padding: 10px

## 5. FOOTER MODAL

| Bouton | Style |
|--------|-------|
| Annuler | Secondary (border) |
| Confirmer | Primary (#0050D7) |

## 6. APIS & ENDPOINTS

```
PUT /ipLoadbalancing/{serviceName}/ssl/{sslId}
```
