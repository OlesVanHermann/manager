# Cahier des Charges : network/load-balancer - Modal Add Frontend

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter un frontend
**Contexte:** Action depuis l'onglet Frontends

## 2. DIMENSIONS

- Modal: 700x700px
- Border radius: 12px

## 3. HEADER MODAL

- Titre: "Ajouter un frontend" (20px, bold)
- Bouton fermer: ✕

## 4. FORMULAIRE

### Champs
- Nom du frontend * (input 640x44px)
- Protocole * (radio: HTTP, HTTPS, TCP, UDP)
- Port * (input 200x44px)
- Zone * (select 320x44px)
- Ferme de serveurs par défaut (select)
- Certificat SSL (select)
- + Options avancées

## 5. FOOTER MODAL

| Bouton | Style |
|--------|-------|
| Annuler | btn-outline |
| Ajouter | btn-primary |

## 6. APIS & ENDPOINTS

```
POST /ipLoadbalancing/{serviceName}/http/frontend
POST /ipLoadbalancing/{serviceName}/tcp/frontend
POST /ipLoadbalancing/{serviceName}/udp/frontend
```
