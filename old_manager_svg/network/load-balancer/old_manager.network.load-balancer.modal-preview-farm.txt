# Cahier des Charges : network/load-balancer - Modal Preview Farm

## 1. VUE D'ENSEMBLE

**Type:** Modal preview (lecture seule)
**Fonction:** Afficher les détails d'une ferme backend
**Contexte:** Visualisation configuration load balancer

## 2. DIMENSIONS

- Modal: 500x500px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Background | #000E9C |
| Titre | "Aperçu de la ferme" (15px, font-weight: 600, white) |
| Close button | × (white) |
