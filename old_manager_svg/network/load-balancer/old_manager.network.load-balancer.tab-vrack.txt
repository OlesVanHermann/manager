# Cahier des Charges : network/load-balancer - Onglet vRack

## 1. VUE D'ENSEMBLE
- **Page** : Configuration vRack
- **URL** : /network/load-balancer/{serviceName}/vrack
- **Objectif** : Gérer l'intégration réseau privé vRack

## 2. STRUCTURE PAGE
- Sidebar Network
- Breadcrumb
- Titre service
- Onglets navigation
- Message d'information
- 2 cartes côte à côte

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec vRack actif
- **OdsMessage** : Information vRack (info)
- **OdsCard** : Association vRack
- **OdsCard** : Serveurs réseau privé
- **OdsBadge** : Statuts (Connecté, OK)
- **OdsButton** : "Modifier l'association" (outline)

## 9. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/vrack/status - Statut vRack
- GET /ipLoadbalancing/{serviceName}/vrack/network - Réseau privé
- GET /ipLoadbalancing/{serviceName}/vrack/networkCreationRules - Règles création
- POST /ipLoadbalancing/{serviceName}/vrack/network - Créer réseau
- DELETE /ipLoadbalancing/{serviceName}/vrack/network/{vrackNetworkId} - Supprimer
