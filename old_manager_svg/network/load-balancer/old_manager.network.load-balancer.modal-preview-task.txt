# Cahier des Charges : network/load-balancer - Modal Preview Task

## 1. VUE D'ENSEMBLE

**Type:** Modal preview (lecture seule)
**Fonction:** Afficher les détails d'une tâche en cours
**Contexte:** Suivi opérations load balancer

## 2. DIMENSIONS

- Modal: 450x270px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Background | #000E9C |
| Titre | "Aperçu de la tâche" (15px, font-weight: 600, white) |
| Close button | × (white) |

## 4. CONTENU

### Informations tâche
| Champ | Exemple |
|-------|---------|
| Date de création | 2025-12-30 14:30:00 |
| Tâche | addServer |
| Progression | 75% (barre de progression) |
| Statut | En cours (badge warning) |
| ID | task-456789 |

### Barre de progression
- Background: #E5E7EB
- Fill: #0050D7
- Height: 8px
- Border-radius: 4px
- Label: "{percent}%" à droite

### Badge statut
| État | Background | Text |
|------|------------|------|
| En cours | #FEF3C7 | #92400E |
| Terminé | #DCFCE7 | #16A34A |
| Erreur | #FEE2E2 | #DC2626 |

## 5. APIS & ENDPOINTS

```
GET /ipLoadbalancing/{serviceName}/task/{taskId}
```
