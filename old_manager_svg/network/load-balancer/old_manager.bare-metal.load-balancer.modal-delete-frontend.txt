# Cahier des Charges : bare-metal/load-balancer - Modal supprimer frontend

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Load Balancer Dedicated
- **Objectif** : Supprimer un frontend du load balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le frontend" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Voulez-vous supprimer le frontend {frontendName} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le frontend |

## 7. APIS & ENDPOINTS
- DELETE /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Supprimer frontend HTTP
- DELETE /ipLoadbalancing/{serviceName}/tcp/frontend/{frontendId} - Supprimer frontend TCP
