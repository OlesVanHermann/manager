# Cahier des Charges : network/load-balancer - Modal Add Farm

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter une ferme de serveurs
**Contexte:** Action depuis l'onglet Fermes

## 2. DIMENSIONS

- Modal: 700x750px
- Border radius: 12px

## 3. HEADER MODAL

- Titre: "Ajouter une ferme de serveurs" (20px, bold)
- Bouton fermer: ✕

## 4. FORMULAIRE

### Champs
- Nom de la ferme * (input 640x44px)
- Protocole * (radio: HTTP, TCP, UDP)
- Port * (input 200x44px)
- Zone * (select 320x44px)
- Méthode de répartition * (select: roundrobin, leastconn, etc.)
- Persistance de session (select)
- Type de probe (select + chemin)
- + Options avancées

## 5. FOOTER MODAL

| Bouton | Style |
|--------|-------|
| Annuler | btn-outline |
| Ajouter | btn-primary |

## 6. APIS & ENDPOINTS

```
POST /ipLoadbalancing/{serviceName}/http/farm
POST /ipLoadbalancing/{serviceName}/tcp/farm
POST /ipLoadbalancing/{serviceName}/udp/farm
```
