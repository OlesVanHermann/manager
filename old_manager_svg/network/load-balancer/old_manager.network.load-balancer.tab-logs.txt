# Cahier des Charges : network/load-balancer - Tab Logs

## 1. VUE D'ENSEMBLE

**Type:** Onglet avec console logs
**Fonction:** Afficher les logs en temps réel (Live Tail)
**Contexte:** Onglet "Logs"

## 2. NAVIGATION

### Sous-onglets
| Sous-onglet | État |
|-------------|------|
| Live Tail | Actif |
| Data Streams | Inactif |

## 3. CONTRÔLES

- Sélecteur type de logs (haproxy)
- Bouton Pause
- Bouton Effacer

## 4. CONSOLE LOGS

- Container dark (#1F2937)
- Format: [timestamp] [level] [source] message

### Couleurs par niveau
- INFO: #10B981 (vert)
- WARN: #FBBF24 (jaune)
- Timestamp: #9CA3AF (gris)
- Source: #06B6D4 (cyan)

## 5. APIS & ENDPOINTS

```
WSS /ipLoadbalancing/{serviceName}/logs/stream
```
