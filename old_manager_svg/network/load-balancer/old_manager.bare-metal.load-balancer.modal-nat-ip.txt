# Cahier des Charges : bare-metal/load-balancer - Modal IP NAT

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information (lecture seule)
- **Contexte** : Configuration Load Balancer
- **Objectif** : Afficher les adresses IP NAT utilisées par le load balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsTable** : Liste IP NAT par zone
- **OdsText** : Description

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "IP NAT du service" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Ligne | Contenu |
|-------|---------|
| 1 | "Les adresses IP NAT sont utilisées par le load balancer pour contacter vos serveurs backend." |
| 2 | "Assurez-vous que ces IPs sont autorisées dans vos pare-feux." |
| Couleur | Gris (#6B7280), 13px |

### Tableau IP NAT
| Propriété | Valeur |
|-----------|--------|
| Container | Fond #F9FAFB, border #E5E7EB |
| Colonnes | Zone, IP NAT |

### Exemple données
| Zone | IP NAT |
|------|--------|
| gra | 10.100.0.1 |
| rbx | 10.100.0.2 |
| sbg | 10.100.0.3 |
| waw | 10.100.0.4 |

### Format tableau
| Propriété | Valeur |
|-----------|--------|
| Header | 11px, gris #6B7280 |
| Données | 12px, noir #1F2937 |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/natIp - Liste des IP NAT par zone
