# Cahier des Charges : bare-metal/load-balancer - Modal Supprimer Ferme

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion Load Balancer
- **Objectif** : Supprimer une ferme de serveurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI

- **OdsButton** : Annuler, Supprimer
- **OdsText** : Question de confirmation

| Propriété | Valeur |
