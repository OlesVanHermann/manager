# Cahier des Charges : network/load-balancer - Liste

## 1. VUE D'ENSEMBLE
- **Page** : Liste des Load Balancers
- **URL** : /network/load-balancer
- **Objectif** : Afficher et gérer les Load Balancers

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation Network
- Sidebar avec Load Balancer sélectionné
- Breadcrumb + Titre + Guides
- Bouton Commander
- Datagrid des services

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Network / Load Balancer
- **OdsText** : Titre "Load Balancer"
- **OdsSelect** : Dropdown "Guides"
- **OdsButton** : "Commander" (outline)
- **OdsDatagrid** : Liste des LB
- **OdsBadge** : Statut (actif)
- **OdsActionMenu** : Menu contextuel

## 4. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Nom du service | Link | ✓ | loadbalancer-XXXXX |
| Nom | Text | ✓ | Nom personnalisé |
| Offre | Text | ✓ | L/M/S |
| Configuration SSL | Text | ✓ | intermediate/modern |
| Statut | Badge | ✓ | actif (vert) |
| Éligible vRack | Text | ✓ | Oui/Non |
| Actions | Menu ⋮ | - | Actions |

## 5. ACTION MENU (ouvert)
| Action | Style |
|--------|-------|
| Voir les détails de mon service | Normal |
| Résilier mon service | Danger |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing - Liste des LB
- GET /ipLoadbalancing/{serviceName} - Détails
- DELETE /ipLoadbalancing/{serviceName}/terminate - Résilier
