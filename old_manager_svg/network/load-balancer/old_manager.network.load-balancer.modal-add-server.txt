# Cahier des Charges : network/load-balancer - Modal Add Server

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter un serveur à une ferme
**Contexte:** Action depuis la gestion d'une ferme

## 2. DIMENSIONS

- Modal: 600x620px
- Border radius: 12px

## 3. HEADER MODAL

- Titre: "Ajouter un serveur" (20px, bold)
- Bouton fermer: ✕

## 4. FORMULAIRE

### Info box ferme
- Background: #E8F0FE
- Ferme de destination: "farm-web-prod (HTTP:80 - GRA)"

### Champs
- Nom du serveur (optionnel)
- Adresse IP *
- Port
- Poids (weight) 1-256

### Options (checkboxes)
- ✓ Activer le serveur immédiatement
- ☐ Serveur de backup
- ☐ Utiliser une connexion SSL vers le serveur

## 5. FOOTER MODAL

| Bouton | Style |
|--------|-------|
| Annuler | btn-outline |
| Ajouter | btn-primary |

## 6. APIS & ENDPOINTS

```
POST /ipLoadbalancing/{serviceName}/http/farm/{farmId}/server
POST /ipLoadbalancing/{serviceName}/tcp/farm/{farmId}/server
```
