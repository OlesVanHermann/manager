# Cahier des Charges : network/load-balancer - Logs Live Tail

## 1. VUE D'ENSEMBLE

**Type:** Page avec console logs temps réel
**Fonction:** Visualiser les logs en streaming
**Contexte:** Monitoring en direct du Load Balancer

## 2. NAVIGATION

### Lien retour
- "← Retour" (text-link #0050D7)

### Titre
- "Logs - Live Tail" (22px, font-weight: 600)
- Sous-titre: "Visualisez les logs en temps réel de votre Load Balancer"

## 3. SÉLECTEUR DE TYPE DE LOGS

### Dropdown
- Dimensions: 200x36px
- Background: white
- Border: #E5E7EB
- Border-radius: 4px
- Valeur par défaut: "HTTP Access Logs"
- Icône: ▼ à droite

### Options disponibles
- HTTP Access Logs
- TCP Connection Logs
- Health Check Logs

## 4. CONSOLE LOGS

### Container
- Dimensions: 1120x300px
- Background: #1F2937 (dark)
- Border-radius: 8px

### Format des logs
```
[timestamp] IP - METHOD /path HTTP/version STATUS size "user-agent" - latency
```

### Couleurs par code HTTP
| Code | Couleur |
|------|---------|
| 2xx Success | #4ADE80 (vert) |
| 3xx Redirect | #FBBF24 (jaune) |
| 4xx Client Error | #FBBF24 (jaune) |
| 5xx Server Error | #F87171 (rouge) |

### Indicateur de streaming
- Cercle vert animé (5px radius, #4ADE80)
- Texte: "Streaming en direct..." (#E5E7EB, 10px)
- Position: Bas gauche de la console

## 5. APIS & ENDPOINTS

```
WSS /ipLoadbalancing/{serviceName}/logs/stream
GET /ipLoadbalancing/{serviceName}/http/frontend/{frontendId}/log
```
