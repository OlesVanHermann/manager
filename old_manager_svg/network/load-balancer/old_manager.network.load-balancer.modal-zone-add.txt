# Cahier des Charges : network/load-balancer - Page ajouter zone

## 1. VUE D'ENSEMBLE
- **Type** : Page de sélection
- **Contexte** : Dashboard Load Balancer
- **Objectif** : Ajouter des zones de disponibilité au Load Balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation retour
- **OdsText** : Description
- **OdsCard** : Cartes de zone (checkbox)
- **OdsButton** : Ajouter, Annuler

## 4. HEADER PAGE
| Élément | Valeur |
|---------|--------|
| Retour | "← Ajouter une zone" |

## 5. DESCRIPTION
| Texte |
|-------|
| "Sélectionnez les zones de disponibilité à ajouter à votre Load Balancer." |

## 6. SÉLECTION DES ZONES

### Format des cartes
| Élément | Description |
|---------|-------------|
| Checkbox | Sélection de la zone |
| Nom | Code et nom de la zone |
| Pays | Localisation |
| Statut | Disponible / Déjà configuré |

### Zones disponibles
| Zone | Nom | Pays | Statut |
|------|-----|------|--------|
| GRA | Gravelines | France | Disponible |
| SBG | Strasbourg | France | Disponible |
| RBX | Roubaix | France | Disponible |
| BHS | Beauharnois | Canada | Disponible |
| WAW | Varsovie | Pologne | Déjà configuré (grisé) |

### États visuels
| État | Bordure | Checkbox |
|------|---------|----------|
| Non sélectionné | Grise | Vide |
| Sélectionné | Bleue | Coché bleu |
| Déjà configuré | Grise, fond gris | Grisé |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter | Primary | Ajoute les zones sélectionnées |
| Annuler | Secondary | Retour au dashboard |

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/availableZones - Zones disponibles
- POST /ipLoadbalancing/{serviceName}/zone - Ajouter zone
