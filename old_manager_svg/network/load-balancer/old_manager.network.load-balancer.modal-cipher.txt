# Cahier des Charges : network/load-balancer - Modal Cipher

## 1. VUE D'ENSEMBLE

**Type:** Modal configuration
**Fonction:** Modifier le cipher SSL du load balancer
**Contexte:** Configuration sécurité frontend HTTPS

## 2. DIMENSIONS

- Modal: 500x350px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Background | #000E9C |
| Titre | "Modifier le cipher SSL" (15px, font-weight: 600, white) |
| Close button | × (white) |

## 4. CONTENU

### Options Radio
| Option | Description | État |
|--------|-------------|------|
| Compatible | Compatibilité maximale avec anciens navigateurs | Selected (radio filled) |
| Modern | TLS 1.2+ uniquement, sécurité renforcée | Unselected |
| Intermediate | Équilibre entre compatibilité et sécurité | Unselected |
| Custom | Configuration personnalisée | Unselected |

### Style Radio
- Selected: circle #0050D7 avec point blanc au centre
- Unselected: circle border #D1D5DB

### Style texte
- Label: font-weight: 500, 14px
- Description: text-muted, 12px

## 5. FOOTER MODAL

| Bouton | Style |
|--------|-------|
| Annuler | Secondary (border #E5E7EB) |
| Modifier | Primary (#0050D7) |

## 6. APIS & ENDPOINTS

```
GET /ipLoadbalancing/{serviceName}/ssl
PUT /ipLoadbalancing/{serviceName}/ssl
GET /ipLoadbalancing/{serviceName}/availableSslCiphers
```
