# Cahier des Charges : network/load-balancer - Onglet Frontends

## 1. VUE D'ENSEMBLE
- **Page** : Liste des frontends
- **URL** : /network/load-balancer/{serviceName}/frontends
- **Objectif** : Gérer les points d'entrée du Load Balancer

## 2. STRUCTURE PAGE
- Sidebar Network
- Breadcrumb
- Onglets navigation
- Description explicative
- Bouton d'ajout
- Tableau des frontends

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Accueil/Frontends (actif)/Fermes de serveurs/Certificats SSL
- **OdsText** : Titre et description
- **OdsButton** : "Ajouter un frontend" (outline)
- **OdsDatagrid** : Liste des frontends
- **OdsToggle** : Actif/Inactif
- **OdsActionMenu** : Actions par frontend

## 10. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/http/frontend - Liste frontends HTTP
- GET /ipLoadbalancing/{serviceName}/tcp/frontend - Liste frontends TCP
- GET /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Détails frontend
- POST /ipLoadbalancing/{serviceName}/http/frontend - Créer frontend
- PUT /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Modifier frontend
- DELETE /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Supprimer frontend
