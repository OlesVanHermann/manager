# Cahier des Charges : network/load-balancer - Modal IP Failover

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information
- **Contexte** : Dashboard Load Balancer
- **Objectif** : Afficher la liste des IP Failover disponibles

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsList** : Liste des IP
- **OdsButton** : Bouton copie par IP
- **OdsClipboard** : Copier dans le presse-papier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "IP Failover" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Liste des IP
| Élément | Description |
|---------|-------------|
| Format | Liste d'adresses IP avec bouton copie |
| Exemple | 203.0.113.10, 203.0.113.11, 203.0.113.12 |

### Actions par ligne
| Action | Description |
|--------|-------------|
| Copy | Copie l'IP dans le presse-papier |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/failover - Liste des IP Failover
