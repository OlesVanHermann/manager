# Cahier des Charges : bare-metal/load-balancer - Page Commander Certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Page de formulaire de commande
- **Contexte** : Gestion Load Balancer
- **Objectif** : Commander un certificat SSL (Let's Encrypt, DV ou EV)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 750px |

## 3. COMPOSANTS UI
- **OdsButtonGroup** : Type de certificat
- **OdsInput** : FQDN, Email, Organisation, Unité, Adresse
- **OdsSelect** : Catégorie d'activité
- **OdsButton** : Commander, Annuler

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Header standard | OVHcloud (bleu #000E9C) |
| Lien retour | "← Ajouter un certificat" |

## 6. FOOTER PAGE
| Bouton | Style | Action |
|--------|-------|--------|
| Commander | Primary (#0050D7) | Lance la commande du certificat |
| Annuler | Secondary (bordure grise) | Retour à la liste |

## 7. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/ssl/letsencrypt - Commander Let's Encrypt
- POST /order/ipLoadbalancing/{serviceName}/certificate/dv - Commander DV
- POST /order/ipLoadbalancing/{serviceName}/certificate/ev - Commander EV
