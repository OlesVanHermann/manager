# Cahier des Charges : network/load-balancer - Zone Add

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire avec liste de sélection
**Fonction:** Ajouter des datacenters/zones au Load Balancer
**Contexte:** Déploiement multi-zones

## 2. DIMENSIONS

- Page width: 800px
- Container zones: 720x260px

## 3. NAVIGATION

### Lien retour
- "← Ajouter une zone" (text-link #0050D7)

## 4. DESCRIPTION

- "Sélectionnez les datacenters où déployer votre Load Balancer"
- Font-size: 13px

## 5. SÉLECTEUR DE ZONES

### Container
- Dimensions: 720x260px
- Background: white
- Border: #E5E7EB
- Border-radius: 8px

### Liste des zones

Chaque zone:
| Élément | Description |
|---------|-------------|
| Checkbox | 20x20px, border-radius: 4px |
| Nom | "Datacenter (CODE)" |
| Pays | Texte muted à droite |

### Zones disponibles
| Code | Datacenter | Pays |
|------|------------|------|
| GRA | Gravelines | France |
| RBX | Roubaix | France |
| SBG | Strasbourg | France |
| BHS | Beauharnois | Canada |
| WAW | Varsovie | Pologne |

### Checkbox styles
| État | Style |
|------|-------|
| Checked | Background #0050D7, ✓ blanc (#FFFFFF) |
| Unchecked | Background white, border #D1D5DB (2px) |

## 6. BOUTONS D'ACTION

### Position
- Bas de page, sous le container

### Boutons
| Bouton | Style | Dimensions |
|--------|-------|------------|
| Ajouter | btn-primary (#0050D7) | 100x40px |
| Annuler | btn-outline | 100x40px |

POST /ipLoadbalancing/{serviceName}/zone
