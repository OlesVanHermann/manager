# Cahier des Charges : network/load-balancer - Modal Preview Frontend

## 1. VUE D'ENSEMBLE

**Type:** Modal preview (lecture seule)
**Fonction:** Afficher les détails d'un frontend
**Contexte:** Visualisation configuration load balancer

## 2. DIMENSIONS

- Modal: 500x450px
- Background overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Background | #000E9C |
| Titre | "Aperçu du frontend" (15px, font-weight: 600, white) |
| Close button | × (white) |

## 4. CONTENU

### Informations principales
| Champ | Exemple |
|-------|---------|
| Nom | frontend-https-443 |
| Protocole | HTTPS |
| Port | 443 |
| Datacenter | GRA (Gravelines) |
| Ferme par défaut | farm-backend-123456 |
| Certificat par défaut | ssl-cert-789012 |
| IP Failover dédiées | 192.168.1.10, 192.168.1.11 |
| HSTS | true |
| ID | 654321 |

### Badge Actif
| État | Background | Text |
|------|------------|------|
| true | #DCFCE7 | #16A34A |
| false | #FEE2E2 | #DC2626 |

### Style
- Labels: text-muted, 12px
- Values: text-body, 13px
- IP list: one per line

## 5. APIS & ENDPOINTS

```
GET /ipLoadbalancing/{serviceName}/http/frontend/{frontendId}
GET /ipLoadbalancing/{serviceName}/https/frontend/{frontendId}
```
