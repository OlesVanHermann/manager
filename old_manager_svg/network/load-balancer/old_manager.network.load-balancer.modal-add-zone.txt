# Cahier des Charges : network/load-balancer - Modal Add Zone

## 1. VUE D'ENSEMBLE

**Type:** Modal de sélection
**Fonction:** Ajouter une zone géographique au load balancer
**Contexte:** Configuration Load Balancer

## 2. DIMENSIONS

- Modal: 450x300px
- Overlay: rgba(0,0,0,0.5)
- Border-radius: 8px

## 3. HEADER MODAL

| Élément | Style |
|---------|-------|
| Titre | "Ajouter une zone" (18px, font-weight: 600) |
| Bouton fermer | × (20px, #6B7280) |

## 4. CONTENU

### Description
- "Sélectionnez une zone géographique pour votre load balancer."

### Sélection zone
| Élément | Style |
|---------|-------|
| Label | "Zone" (font-weight: 500) |
| Dropdown | 402x40px |
| Placeholder | "Sélectionner une zone" |

### Zones disponibles
- rbx, sbg, gra, bhs, waw

## 5. FOOTER MODAL

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Ghost (#0050D7 text) | Fermer modal |
| Ajouter | Primary (#0050D7) | POST API |

## 6. APIS & ENDPOINTS

```
GET /ipLoadbalancing/{serviceName}/availableZones
POST /ipLoadbalancing/{serviceName}/zone
{
  "zone": "rbx"
}
```
