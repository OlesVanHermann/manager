# Cahier des Charges : network/load-balancer - Modal supprimer serveur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page fermes Load Balancer
- **Objectif** : Supprimer un serveur d'une ferme

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le serveur" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Voulez-vous supprimer le serveur {serverName} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le serveur |

## 7. APIS & ENDPOINTS
- DELETE /ipLoadbalancing/{serviceName}/http/farm/{farmId}/server/{serverId} - Supprimer serveur HTTP
- DELETE /ipLoadbalancing/{serviceName}/tcp/farm/{farmId}/server/{serverId} - Supprimer serveur TCP
