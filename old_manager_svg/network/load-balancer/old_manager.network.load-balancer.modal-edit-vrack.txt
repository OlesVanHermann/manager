## 8. APIS & ENDPOINTS

### 8.1 Modifier le réseau privé

```
PUT /ipLoadbalancing/{serviceName}/vrack/network/{vrackNetworkId}
```

Body:
```json
{
  "displayName": "Mon réseau privé",
  "vlan": 100,
  "subnet": "10.0.0.0/8",
  "natIp": "10.0.0.0/24",
  "farmId": [123456]
}
```
