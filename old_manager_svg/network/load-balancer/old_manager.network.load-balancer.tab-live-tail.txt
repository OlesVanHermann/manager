# Cahier des Charges : network/load-balancer - Tab Live Tail

## 1. VUE D'ENSEMBLE

**Type:** Page console temps réel
**Fonction:** Visualiser les logs du load balancer en temps réel
**Contexte:** Monitoring load balancer

## 2. LAYOUT

- Page: 1100x500px
- Background: #F9FAFB
- Header OVHcloud: 56px, #000E9C

## 3. HEADER PAGE

| Élément | Style |
|---------|-------|
| Titre | "Logs en temps réel" (18px, font-weight: 600) |
| Sous-titre | "Visualisez les logs de votre Load Balancer en temps réel" (text-muted, 13px) |

## 4. CONTRÔLES

### Sélecteur type de logs
| Élément | Style |
|---------|-------|
| Label | "Type de logs" (text-muted, 12px) |
| Dropdown | 200x36px, border #D1D5DB |
| Options | Accès HTTP, Erreurs, Connexions TCP |

### Boutons
| Bouton | Style | Action |
|--------|-------|--------|
| ▶ Démarrer | Primary (#0050D7) | Lancer le stream |
| ⏸ Pause | Primary | Mettre en pause |
| Effacer | Secondary (border) | Vider la console |

## 5. CONSOLE

### Conteneur
- Width: 1020px
- Height: 270px
- Background: #1F2937 (dark)
- Border-radius: 8px

### Header console (macOS style)
- Trois cercles: rouge (#EF4444), jaune (#F59E0B), vert (#10B981)
- Diamètre: 12px

### Lignes de log
| Élément | Style |
|---------|-------|
| Timestamp | #9CA3AF (gris), Monaco/Consolas 12px |
| Contenu normal | #10B981 (vert), Monaco/Consolas 12px |
| Contenu warning | #F59E0B (jaune) - temps > 1s |
| Contenu error | #EF4444 (rouge) - codes 4xx/5xx |

### Format log
```
[{timestamp}] {method} {path} HTTP/1.1 {status} {statusText} {duration}ms
```

### Curseur
- Rectangle 8x14px
- Fill: #10B981
- Clignotant

## 6. APIS & ENDPOINTS

```
GET /ipLoadbalancing/{serviceName}/logs (WebSocket ou SSE)
GET /ipLoadbalancing/{serviceName}/logs/tail
```
