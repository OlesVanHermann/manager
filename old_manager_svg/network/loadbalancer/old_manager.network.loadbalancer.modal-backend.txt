# Cahier des Charges : network/loadbalancer - Modal Backend

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout serveur backend
- **Contexte** : Network Load Balancer
- **Objectif** : Ajouter un serveur au pool backend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 510px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels et helper text
- **OdsInput** : Nom, IP, port, poids
- **Cards sélection** : État initial
- **OdsCheckbox** : Vérification connectivité
- **OdsMessage** : Avertissement accessibilité
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Ajouter un serveur backend" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT NOM SERVEUR
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du serveur" |
| Largeur | 500px |
| Placeholder | "web-server-05" |

## 6. INPUTS IP + PORT
| Champ | Largeur | Placeholder |
|-------|---------|-------------|
| Adresse IP | 350px | "10.0.0.15" |
| Port | 130px | "8080" |

## 7. INPUT POIDS
| Propriété | Valeur |
|-----------|--------|
| Label | "Poids (1-256)" |
| Largeur | 150px |
| Valeur défaut | 100 |
| Helper | "Plus le poids est élevé, plus le serveur reçoit de trafic" |

## 8. SÉLECTION ÉTAT INITIAL
| Option | Largeur | Description |
|--------|---------|-------------|
| Actif | 155px | Reçoit du trafic |
| Backup | 155px | Utilisé si autres down |
| Drain | 155px | Sessions existantes seulement |

## 9. CHECKBOX VÉRIFICATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Vérifier la connectivité avant ajout" |
| État défaut | Coché |

## 10. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠️ |
| Texte | "Le serveur doit être accessible depuis le Load Balancer sur le port spécifié." |

## 11. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Ajouter | Primary #0050D7 | 100px |

## 12. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/http/farm/{farmId}/server - Ajouter serveur
- GET /ipLoadbalancing/{serviceName}/http/farm - Liste farms
