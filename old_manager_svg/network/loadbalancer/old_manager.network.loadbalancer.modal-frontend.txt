# Cahier des Charges : network/loadbalancer - Modal Frontend

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout frontend
- **Contexte** : Network Load Balancer
- **Objectif** : Configurer un point d'entrée frontend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 510px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **OdsInput** : Nom, port
- **Cards sélection** : Protocole
- **OdsSelect** : Backend, certificat SSL
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Ajouter un frontend" |
| Bouton fermer | × en #6B7280 |

## 5. INPUT NOM FRONTEND
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du frontend" |
| Largeur | 500px |
| Placeholder | "https-frontend" |

## 6. SÉLECTION PROTOCOLE
| Option | Largeur | Description |
|--------|---------|-------------|
| HTTPS | 120px | Trafic chiffré |
| HTTP | 120px | Trafic non chiffré |
| TCP | 120px | Couche transport |

## 7. INPUT PORT
| Propriété | Valeur |
|-----------|--------|
| Label | "Port d'écoute" |
| Largeur | 150px |
| Valeur défaut | 443 (HTTPS) / 80 (HTTP) |

## 8. SELECT BACKEND
| Propriété | Valeur |
|-----------|--------|
| Label | "Backend par défaut" |
| Largeur | 500px |
| Valeur exemple | "web-backend (4 serveurs)" |

## 9. SELECT CERTIFICAT SSL
| Propriété | Valeur |
|-----------|--------|
| Label | "Certificat SSL" |
| Largeur | 500px |
| Valeur exemple | "example.com (Let's Encrypt)" |
| Condition | Affiché si protocole HTTPS |

## 10. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Ajouter | Primary #0050D7 | 100px |

## 11. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/http/frontend - Créer frontend
- GET /ipLoadbalancing/{serviceName}/http/farm - Liste backends
- GET /ipLoadbalancing/{serviceName}/ssl - Liste certificats
