## 10. APIS & ENDPOINTS
- GET /loadbalancer/{id}/backend - Liste backends

- PUT /loadbalancer/{id}/backend/{backendId} - Modifier backend
- DELETE /loadbalancer/{id}/backend/{backendId} - Supprimer backend
