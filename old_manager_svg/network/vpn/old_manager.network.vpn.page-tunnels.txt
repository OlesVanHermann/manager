# Cahier des Charges : network/vpn - Page Tunnels

## 1. VUE D'ENSEMBLE
- **Type** : Page listing tunnels VPN
- **Contexte** : Gestion réseau
- **Objectif** : Lister et gérer les tunnels VPN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

- **OdsPopover** : Menu actions

## 10. APIS & ENDPOINTS
- GET /vpn - Liste tunnels VPN
- GET /vpn/{id} - Détail tunnel
- GET /vpn/{id}/statistics - Statistiques trafic
- POST /vpn/{id}/restart - Redémarrer tunnel
- DELETE /vpn/{id} - Supprimer tunnel
