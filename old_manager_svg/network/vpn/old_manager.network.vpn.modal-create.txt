# Cahier des Charges : network/vpn - Modal Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création tunnel VPN
- **Contexte** : Gestion réseau
- **Objectif** : Créer un nouveau tunnel VPN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 550x540px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom, Endpoint distant, PSK
- **OdsCard** : Sélection type VPN
- **OdsSelect** : Endpoint local
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Créer un tunnel VPN" |
| Bouton fermer | × |

## 12. APIS & ENDPOINTS
- GET /vpn/availableEndpoints - Endpoints locaux disponibles

- POST /vpn/generatePSK - Générer Pre-shared Key
