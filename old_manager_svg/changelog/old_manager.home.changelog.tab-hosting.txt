# Cahier des Charges : home/changelog - Onglet Hébergement

## 1. VUE D'ENSEMBLE
- **Page** : Changelog - Onglet Hébergement
- **Type** : Liste filtrée
- **Objectif** : Voir les nouveautés Web & Hébergement

## 2. STRUCTURE PAGE
- Header avec titre
- Onglets (Hébergement actif)
- Filtre par produit
- Liste entrées Hébergement

## 3. COMPOSANTS UI
- **OdsTabs** : Hébergement sélectionné
- **OdsSelect** : Filtre produit
- **OdsCard** : Entrées changelog
- **OdsBadge** : Type modification

## 4. FILTRE PRODUIT HÉBERGEMENT
| Option | Description |
|--------|-------------|
| Tous | Tous les produits Web |
| Hébergement Web | Hosting mutualisé |
| Domaines | Noms de domaine |
| Email | MX Plan, Exchange |
| WordPress | Managed WordPress |
| Base de données | Web Cloud Databases |

## 5. ENTRÉES AFFICHÉES
| Type | Exemples |
|------|----------|
| Hosting | Nouvelles versions PHP |
| Email | Nouvelles fonctionnalités Exchange |
| Domaines | Nouvelles extensions |
| WordPress | Versions WP, plugins |

## 6. APIS & ENDPOINTS
- GET /changelog?category=hosting - Changelog Hosting
- GET /changelog?category=hosting&product={id} - Filtré par produit
