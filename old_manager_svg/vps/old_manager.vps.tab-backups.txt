# Cahier des Charges : vps - Onglet Sauvegardes

## 1. VUE D'ENSEMBLE
- **Page** : Sauvegardes automatiques VPS
- **Type** : Onglet avec liste sauvegardes
- **Objectif** : Lister et gérer les backups

## 2. STRUCTURE PAGE
- Onglet "Sauvegardes" actif
- Titre section
- Bandeau info (option activée)
- Bouton création manuelle
- Datagrid des sauvegardes

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsMessage** : Info bleue
- **OdsButton** : Créer une sauvegarde
- **OdsDatagrid** : Liste backups
- **OdsBadge** : Statut

## 4. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Style | Info (bleu clair) |
| Icône | ℹ |
| Texte | "L'option Automated Backup est activée. Une sauvegarde est créée automatiquement chaque jour." |

## 5. BOUTON HEADER
| Propriété | Valeur |
|-----------|--------|
| Label | "+ Créer une sauvegarde" |
| Style | Primary |
| Action | Création manuelle |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Date | DateTime | Date/heure création |
| Type | Text | Automatique / Manuelle |
| Taille | Size | Taille en Go |
| Status | Badge | Disponible (vert) |
| Actions | Links | Restaurer | Supprimer |

## 7. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Disponible | Vert |
| En cours | Bleu |
| Échec | Rouge |

## 8. EXEMPLES DONNÉES
| Date | Type | Taille | Status |
|------|------|--------|--------|

## 9. ACTIONS PAR LIGNE
| Action | Description |
|--------|-------------|
| Restaurer | Restaurer depuis ce backup |
| Supprimer | Supprimer ce backup |

## 10. APIS & ENDPOINTS
- GET /vps/{serviceName}/backups - Liste sauvegardes
- POST /vps/{serviceName}/backups - Créer backup manuel
- POST /vps/{serviceName}/backups/{backupId}/restore - Restaurer
- DELETE /vps/{serviceName}/backups/{backupId} - Supprimer
