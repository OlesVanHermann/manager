# Cahier des Charges : vps - Backup Automatique

## 1. VUE D'ENSEMBLE
- **Page** : Option backup automatique VPS
- **Type** : Onglet configuration et liste
- **Objectif** : Gérer les backups automatiques

## 2. STRUCTURE PAGE
- Onglet "Backup automatique" actif
- Carte option backup avec toggle
- Carte planification
- Tableau points de restauration

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Configuration
- **OdsToggle** : Activation option
- **OdsBadge** : Statut Activé
- **OdsDatagrid** : Points restauration

## 4. CARTE OPTION BACKUP
| Info | Valeur |
|------|--------|
| Statut | Badge "Activé" + Toggle ON |
| Prix | 2,00 € HT/mois |
| Rétention | 7 points de restauration |

## 5. CARTE PLANIFICATION
| Info | Valeur |
|------|--------|
| Fréquence | Quotidien |
| Heure | 03:00 (UTC+1) |
| Dernier backup | 27/12/2025 03:00 |

## 6. COLONNES POINTS RESTAURATION
| Colonne | Type | Description |
|---------|------|-------------|
| Date | DateTime | Date/heure backup |
| Taille | Size | Taille en Go |
| Statut | Badge | Disponible (vert) |
| Actions | Links | Restaurer | Monter |

## 7. EXEMPLES DONNÉES
| Date | Taille | Statut |
|------|--------|--------|
| 27/12/2025 03:00 | 15.2 Go | Disponible |

| 25/12/2025 03:00 | 14.9 Go | Disponible |

## 8. ACTIONS DISPONIBLES
| Action | Description |
|--------|-------------|
| Restaurer | Restaurer le VPS depuis ce point |
| Monter | Monter le backup comme disque |

## 9. APIS & ENDPOINTS
- GET /vps/{serviceName}/automatedBackup - Statut option
- PUT /vps/{serviceName}/automatedBackup - Activer/désactiver
- GET /vps/{serviceName}/automatedBackup/restorePoints - Points restauration
- POST /vps/{serviceName}/automatedBackup/restore - Restaurer
- POST /vps/{serviceName}/automatedBackup/attachedBackup - Monter
