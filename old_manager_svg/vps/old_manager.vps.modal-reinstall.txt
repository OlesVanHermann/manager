# Cahier des Charges : vps - Modal Réinstallation

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation réinstallation VPS
- **Type** : Modal de confirmation danger
- **Objectif** : Confirmer la réinstallation

## 2. STRUCTURE MODAL
- Dimensions : 700x520px
- Header avec titre
- Avertissement danger
- Formulaire sélection
- Checkbox confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsMessage** : Warning danger
- **OdsSelect** : Image, Clé SSH, Langue
- **OdsCheckbox** : Confirmation obligatoire
- **OdsButton** : Annuler, Réinstaller

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Titre | "Réinstaller le VPS" |
| Fond | Gris clair |
| Bouton fermer | ✕ |

## 5. AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Style | Danger (rouge) |
| Icône | ⚠ |
| Titre | "Attention : Toutes les données seront supprimées !" |
| Message | "Pensez à sauvegarder vos données avant de continuer." |

## 6. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Image à installer | Select | ✓ |
| Clé SSH | Select | Non |
| Langue | Select | Non |

### Options Image
- Debian 12 (Bookworm)
- Ubuntu 24.04 LTS
- Rocky Linux 9
- ...

### Options Langue
- Français
- English
- Deutsch
- ...

## 7. CHECKBOX CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Je confirme vouloir réinstaller mon VPS" |
| Obligatoire | Oui (pour activer bouton) |

## 8. FOOTER
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Default (outline) | Gauche |
| Réinstaller | Danger (rouge) | Droite |

## 9. VALIDATION
- Bouton "Réinstaller" désactivé tant que checkbox non cochée
- Image obligatoire sélectionnée

## 10. APIS & ENDPOINTS
- GET /vps/{serviceName}/templates - Images disponibles
- GET /me/sshKey - Clés SSH
- POST /vps/{serviceName}/reinstall - Lancer réinstallation
