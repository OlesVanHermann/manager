# Cahier des Charges : vps - Modal console KVM

## 1. VUE D'ENSEMBLE
- **Type** : Modal avec console interactive
- **Contexte** : Dashboard VPS
- **Objectif** : Accéder à la console KVM du VPS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 490px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **iframe** : Console KVM embarquée
- **OdsText** : Note clavier
- **OdsLink** : Ouvrir dans nouvelle fenêtre

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Console KVM" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. ZONE CONSOLE

### Container console
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 380px |
| Fond | Noir (#1a1a2e) |
| Bordure | #E5E7EB |

### Contenu terminal
| Élément | Description |
|---------|-------------|

## 7. LIEN EXTERNE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Ouvrir dans une nouvelle fenêtre ↗" |
| Style | Lien bleu souligné |
| Action | Ouvre la console dans un nouvel onglet |

## 8. APIS & ENDPOINTS
- GET /vps/{serviceName}/getConsoleUrl - URL de la console KVM
- POST /vps/{serviceName}/openConsoleAccess - Ouvrir accès console
