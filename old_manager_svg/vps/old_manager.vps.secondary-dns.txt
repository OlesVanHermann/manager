# Cahier des Charges : vps - DNS Secondaire

## 1. VUE D'ENSEMBLE
- **Page** : DNS secondaire VPS
- **Type** : Onglet gestion domaines
- **Objectif** : Configurer OVHcloud comme DNS secondaire

## 2. STRUCTURE PAGE
- Onglet "DNS secondaire" actif
- Carte explicative avec bouton ajout
- Tableau domaines configurés
- Carte serveurs DNS à utiliser

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Sections
- **OdsButton** : Ajouter un domaine
- **OdsDatagrid** : Domaines
- **OdsBadge** : Statut synchro
- **OdsCode** : Domaines, IPs

## 4. CARTE HEADER
| Élément | Valeur |
|---------|--------|
| Titre | "Configuration DNS secondaire OVHcloud" |
| Description | "OVHcloud peut agir comme serveur DNS secondaire pour vos domaines hébergés sur ce VPS." |
| Bouton | "Ajouter un domaine" (primary) |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Code | Nom de domaine |
| IP du serveur DNS primaire | Code | IP du VPS |
| Dernière synchro | DateTime | Date/heure synchro |
| Statut | Badge | Synchronisé (vert) |
| Actions | Links | Forcer synchro | Supprimer |

## 6. EXEMPLES DONNÉES
| Domaine | IP Primaire | Statut |
|---------|-------------|--------|
| monsite.fr | 51.178.123.45 | Synchronisé |
| monappli.io | 51.178.123.45 | Synchronisé |

## 7. ACTIONS PAR LIGNE
| Action | Style | Description |
|--------|-------|-------------|
| Forcer synchro | Link bleu | Force synchronisation |
| Supprimer | Link rouge | Retire le domaine |

## 8. CARTE SERVEURS DNS
| Serveur | Usage |
|---------|-------|
| sdns1.ovh.net | DNS secondaire 1 |
| sdns2.ovh.net | DNS secondaire 2 |

### Note
"Ajoutez ces serveurs dans la configuration DNS de votre registrar."

## 9. APIS & ENDPOINTS
- GET /vps/{serviceName}/secondaryDns - Liste domaines
- POST /vps/{serviceName}/secondaryDns - Ajouter domaine
- DELETE /vps/{serviceName}/secondaryDns/{domain} - Supprimer
- POST /vps/{serviceName}/secondaryDns/{domain}/sync - Forcer synchro
