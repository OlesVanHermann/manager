# Cahier des Charges : vps - Snapshots

## 1. VUE D'ENSEMBLE
- **Page** : Gestion des snapshots VPS
- **Type** : Onglet avec cards et historique
- **Objectif** : Créer et gérer les snapshots

## 2. STRUCTURE PAGE
- Onglet Snapshots actif
- Message info (limitation 1 snapshot)
- Carte snapshot actuel
- Carte création nouveau
- Tableau historique opérations

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsMessage** : Info bleue
- **OdsCard** : Snapshot actuel, création
- **OdsDatagrid** : Historique
- **OdsBadge** : Statut
- **OdsButton** : Restaurer, Supprimer, Créer

## 4. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Style | Info (bleu) |
| Icône | ℹ️ |
| Texte | "Un snapshot est une image instantanée de votre VPS. Vous pouvez en conserver 1 à la fois." |

## 5. CARTE SNAPSHOT ACTUEL
| Info | Valeur |
|------|--------|
| Nom | snapshot-20251225-1430 |
| Date de création | 25/12/2025 à 14:30 |
| Taille | 12.5 Go |
| Actions | Boutons Restaurer, Supprimer |

### Boutons actions
| Bouton | Style |
|--------|-------|
| Restaurer | Primary |
| Supprimer | Danger (outline) |

## 6. CARTE CRÉATION
| Info | Valeur |
|------|--------|
| Avertissement | "La création d'un nouveau snapshot remplacera l'actuel." |
| Info | "Le VPS sera redémarré pendant l'opération." |
| Durée estimée | 5-15 minutes |
| Bouton | "Créer un snapshot" (primary) |

## 7. COLONNES HISTORIQUE
| Colonne | Type | Description |
|---------|------|-------------|
| Date | DateTime | Date/heure opération |
| Action | Text | Création/Restauration |
| Durée | Duration | Temps d'exécution |
| Statut | Badge | Réussi (vert) |

## 8. EXEMPLES HISTORIQUE
| Date | Action | Durée | Statut |
|------|--------|-------|--------|
| 25/12/2025 14:30 | Création de snapshot | 8 min 32 sec | Réussi |
| 20/12/2025 09:15 | Restauration de snapshot | 12 min 45 sec | Réussi |

## 9. APIS & ENDPOINTS
- GET /vps/{serviceName}/snapshot - Snapshot actuel
- POST /vps/{serviceName}/snapshot - Créer snapshot
- DELETE /vps/{serviceName}/snapshot - Supprimer
- POST /vps/{serviceName}/snapshot/restore - Restaurer
- GET /vps/{serviceName}/tasks - Historique opérations
