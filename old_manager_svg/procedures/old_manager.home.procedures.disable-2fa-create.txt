# Cahier des Charges : home/procedures - Créer demande désactivation 2FA

## 1. VUE D'ENSEMBLE
- **Page** : Upload documents désactivation 2FA
- **Type** : Formulaire upload
- **Objectif** : Soumettre les documents pour désactiver la 2FA

## 2. STRUCTURE PAGE
- Stepper (3 étapes)
- Zone upload documents
- Récapitulatif
- Boutons action

## 3. COMPOSANTS UI
- **OdsStepper** : Progression (Identité, Documents, Confirmation)
- **OdsFileUpload** : Upload documents
- **OdsMessage** : Instructions
- **OdsButton** : Précédent, Suivant, Envoyer

## 4. STEPPER
| Étape | Nom | Description |
|-------|-----|-------------|
| 1 | Identité | Vérification email |
| 2 | Documents | Upload pièces |
| 3 | Confirmation | Validation |

## 5. ZONE UPLOAD
| Document | Type | Max | Obligatoire |
|----------|------|-----|-------------|
| Pièce d'identité (recto) | Image/PDF | 5 Mo | Oui |
| Pièce d'identité (verso) | Image/PDF | 5 Mo | Oui |
| Justificatif domicile | Image/PDF | 5 Mo | Oui |

## 6. COMPOSANT FILE UPLOAD
| Élément | Description |
|---------|-------------|
| Zone drop | Zone drag & drop |
| Bouton | Parcourir fichiers |
| Preview | Miniature du fichier |
| Supprimer | Bouton X |
| Taille | Affichage taille fichier |

## 7. INSTRUCTIONS
| Message | Contenu |
|---------|---------|
| Formats | Formats acceptés : JPG, PNG, PDF |
| Taille | Taille maximale : 5 Mo par fichier |
| Qualité | Les documents doivent être lisibles |

## 8. BOUTONS
| Bouton | Position | Action |
|--------|----------|--------|
| Précédent | Gauche | Étape -1 |
| Suivant | Droite | Étape +1 |
| Envoyer | Droite (étape 3) | Soumettre demande |

## 9. APIS & ENDPOINTS
- POST /me/accessRestriction/disable2fa/request - Créer demande
- POST /me/accessRestriction/disable2fa/request/{id}/document - Upload document
