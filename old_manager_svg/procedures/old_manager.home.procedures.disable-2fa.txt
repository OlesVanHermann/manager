# Cahier des Charges : home/procedures - Désactivation 2FA

## 1. VUE D'ENSEMBLE
- **Page** : Désactivation double authentification
- **Type** : Page informative procédure
- **Objectif** : Informer sur la procédure de désactivation 2FA

## 2. STRUCTURE PAGE
- Header avec titre
- Bandeau warning
- Explication procédure
- Documents requis
- Bouton démarrer

## 3. COMPOSANTS UI
- **OdsMessage** : Warning sécurité
- **OdsText** : Explications
- **OdsList** : Documents requis
- **OdsButton** : Démarrer procédure
- **OdsCard** : Container principal

## 4. MESSAGE WARNING
| Type | Couleur | Message |
|------|---------|---------|
| Warning | Orange | Cette procédure est réservée aux cas où vous n'avez plus accès à votre second facteur d'authentification. |

## 5. EXPLICATION
| Section | Contenu |
|---------|---------|
| Introduction | Si vous ne pouvez plus vous connecter à cause de la double authentification... |
| Délai | Le traitement prend généralement 24 à 48h ouvrées. |
| Sécurité | Des documents d'identité seront demandés pour vérifier votre identité. |

## 6. DOCUMENTS REQUIS
| Document | Obligatoire | Format |
|----------|-------------|--------|
| Pièce d'identité recto | Oui | JPG, PNG, PDF |
| Pièce d'identité verso | Oui | JPG, PNG, PDF |
| Justificatif de domicile | Oui | JPG, PNG, PDF |

## 7. BOUTON ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Démarrer la procédure | Primary | Vers formulaire upload |

## 8. APIS & ENDPOINTS
- GET /me/accessRestriction/disable2fa/info - Infos procédure
