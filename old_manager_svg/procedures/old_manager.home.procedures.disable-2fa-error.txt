# Cahier des Charges : home/procedures - Erreur demande 2FA

## 1. VUE D'ENSEMBLE
- **Page** : Erreur demande désactivation 2FA
- **Type** : Page erreur
- **Objectif** : Informer d'une erreur lors de la demande

## 2. STRUCTURE PAGE
- Icône erreur
- Titre erreur
- Message explicatif
- Actions possibles

## 3. COMPOSANTS UI
- **OdsIcon** : Icône erreur (X rouge)
- **OdsText** : Titre, message
- **OdsButton** : Actions
- **OdsMessage** : Info supplémentaire

## 4. ICÔNE ERREUR
| Propriété | Valeur |
|-----------|--------|
| Cercle | Rouge clair (#FEE2E2) |
| Icône | X rouge (#DC2626) |
| Taille | 100px |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Titre | Une erreur est survenue |
| Message | Impossible de traiter votre demande pour le moment. |
| Sous-message | Veuillez réessayer ultérieurement. |

## 6. CAUSES POSSIBLES
| Code | Message |
|------|---------|
| INVALID_DOCS | Documents non conformes |
| EXPIRED_SESSION | Session expirée |
| ALREADY_EXISTS | Demande déjà en cours |
| SERVER_ERROR | Erreur serveur |

## 7. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Réessayer | Primary | Reprendre formulaire |
| Contacter le support | Outline | Vers support |
| Retour accueil | Ghost | Vers dashboard |

## 8. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info | Si le problème persiste, contactez notre support avec le code d'erreur. |

## 9. APIS & ENDPOINTS
- Aucun (page statique d'erreur)
