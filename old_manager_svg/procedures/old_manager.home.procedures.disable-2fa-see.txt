# Cahier des Charges : home/procedures - Suivi demande désactivation 2FA

## 1. VUE D'ENSEMBLE
- **Page** : Suivi de ma demande
- **Type** : Page suivi avec timeline
- **Objectif** : Suivre l'avancement de la demande de désactivation 2FA

## 2. STRUCTURE PAGE
- Header avec numéro demande
- Badge statut
- Timeline étapes
- Détails demande

## 3. COMPOSANTS UI
- **OdsBadge** : Statut demande
- **OdsTimeline** : Étapes traitement
- **OdsCard** : Détails demande
- **OdsText** : Informations

## 4. HEADER
| Élément | Type | Description |
|---------|------|-------------|
| Titre | H1 | Suivi de votre demande |
| Numéro | Text | #2FA-2024-123456 |
| Statut | Badge | En cours / Validé / Refusé |

## 5. TIMELINE ÉTAPES
| Étape | Icône | Description | Statut |
|-------|-------|-------------|--------|
| Demande reçue | ✓ | Documents reçus le JJ/MM/AAAA | Complété |
| Vérification | ⏳ | En cours de traitement | En cours |
| Validation | ○ | En attente | Pending |
| 2FA désactivée | ○ | Accès restauré | Pending |

## 6. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| En cours | Orange | Traitement en cours |
| Validé | Vert | Demande acceptée |
| Refusé | Rouge | Demande refusée |
| Info requise | Bleu | Documents supplémentaires |

## 7. DÉTAILS DEMANDE
| Info | Valeur |
|------|--------|
| Date demande | JJ/MM/AAAA HH:MM |
| Email | user@example.com |
| Documents | 3 fichiers |

| Bouton | Style | Action |

- GET /me/accessRestriction/disable2fa/request/{id} - Statut demande
- GET /me/accessRestriction/disable2fa/requests - Liste demandes
