# Cahier des Charges : public/storage - Modal Volume Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création volume
- **Contexte** : Public Cloud Block Storage
- **Objectif** : Créer un nouveau volume de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titres et labels
- **OdsInput** : Nom, taille
- **OdsSelect** : Région
- **OdsButton** : Annuler, Créer
- **Cards sélection** : Type volume

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Créer un volume" |
| Bouton fermer | × en #6B7280 |

| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du volume" |
| Largeur | 400px |
| Placeholder | "Ex: vol-data-prod" |

## 6. SELECT RÉGION
| Propriété | Valeur |
|-----------|--------|
| Label | "Région" |
| Largeur | 400px |

## 7. SÉLECTION TYPE VOLUME
| Propriété | Valeur |
|-----------|--------|

## 8. INPUT TAILLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Taille (Go)" |
| Largeur | 150px |

## 9. BOUTONS FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline #E5E7EB | Fermer modal |
| Créer | Primary #0050D7 | POST création |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/region - Régions disponibles
- POST /cloud/project/{id}/volume - Créer volume
