# Cahier des Charges : public/storage - Page Volumes

## 1. VUE D'ENSEMBLE
- **Type** : Page liste volumes
- **Contexte** : Public Cloud Block Storage
- **Objectif** : Gérer les volumes de stockage persistant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer volume
- **OdsDatagrid** : Liste volumes
- **OdsBadge** : Type, état
- **ActionMenu** : Menu ⋮

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Block Storage" |
| Description | "Volumes de stockage persistant pour vos instances." |

## 5. BOUTON ACTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Créer volume" |
| Style | Primary #0050D7 |
| Largeur | 150px |

## 10. MENU ACTIONS (⋮)
| Action | Style |
|--------|-------|
| Attacher | Normal |
| Détacher | Normal |
| Redimensionner | Normal |
| Créer snapshot | Normal |
| --- | Séparateur |
| Supprimer | Danger (#DC2626) |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/volume - Liste volumes
- POST /cloud/project/{id}/volume - Créer volume
- POST /cloud/project/{id}/volume/{volumeId}/attach - Attacher
- POST /cloud/project/{id}/volume/{volumeId}/detach - Détacher
- DELETE /cloud/project/{id}/volume/{volumeId} - Supprimer
