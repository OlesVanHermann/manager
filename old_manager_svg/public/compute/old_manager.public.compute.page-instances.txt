- **Type** : Page liste instances Public Cloud
- **Contexte** : Public Cloud Compute
- **Objectif** : Gérer les machines virtuelles

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

- **OdsInput** : Recherche
- **OdsSelect** : Filtres (région, état)
- **OdsButton** : Créer instance
- **OdsDatagrid** : Liste instances

## 10. APIS & ENDPOINTS
- GET /cloud/project/{id}/instance - Liste instances
- POST /cloud/project/{id}/instance - Créer
- POST /cloud/project/{id}/instance/{instanceId}/reboot - Redémarrer
- POST /cloud/project/{id}/instance/{instanceId}/stop - Arrêter
- DELETE /cloud/project/{id}/instance/{instanceId} - Supprimer
