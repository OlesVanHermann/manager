# Cahier des Charges : public/compute - Page Flavors

## 1. VUE D'ENSEMBLE
- **Type** : Page catalogue flavors
- **Contexte** : Public Cloud Compute
- **Objectif** : Afficher les configurations d'instances disponibles

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsTabs** : Catégories flavors
- **OdsDatagrid** : Liste flavors
- **OdsBadge** : Type flavor

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Flavors disponibles" |
| Description | "Configurations de ressources pour vos instances." |

## 5. ONGLETS CATÉGORIES

|--------|-------------|
| Général | Workloads standards |
| CPU | Calcul intensif |
