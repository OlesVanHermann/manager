# Cahier des Charges : public/compute - Page Images

## 1. VUE D'ENSEMBLE
- **Type** : Page catalogue images système
- **Contexte** : Public Cloud Compute
- **Objectif** : Afficher les images disponibles pour créer des instances

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsTabs** : Catégories images
- **OdsDatagrid** : Liste images
- **OdsBadge** : Type et statut
- **OdsAvatar** : Icône OS (cercle coloré)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Images système" |
| Description | "Images disponibles pour créer des instances." |

|--------|-------------|
