- **Type** : Page notebooks AI
- **Contexte** : Public Cloud AI
- **Objectif** : Gérer les environnements Jupyter

## 10. APIS & ENDPOINTS
- GET /ai/notebook - Liste notebooks
- POST /ai/notebook - Créer notebook
- POST /ai/notebook/{id}/start - Démarrer
- POST /ai/notebook/{id}/stop - Arrêter
- DELETE /ai/notebook/{id} - Supprimer
