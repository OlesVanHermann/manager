## 7. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}/ai/endpoint
→ Liste des endpoints

GET /cloud/project/{projectId}/ai/endpoint/{endpointId}
→ Détails d'un endpoint

GET /cloud/project/{projectId}/ai/endpoint/{endpointId}/metrics
→ Métriques (IOPS, requêtes)

POST /cloud/project/{projectId}/ai/endpoint
Body: { name, model, region, gpu, replicas }
→ Déploie un endpoint

DELETE /cloud/project/{projectId}/ai/endpoint/{endpointId}
→ Supprime l'endpoint
```
