**Type:** Page monitoring avec console de logs
**Fonction:** Visualiser les logs d'une fonction serverless
**Contexte:** Debugging et monitoring FaaS

## 6. CONSOLE LOGS

### Container
- Background: #1F2937 (dark)
- Border-radius: 4px
- Dimensions: 1240x450px

### Format entrée log
```
[timestamp] [LEVEL] message
```

### Styles par niveau
| Niveau | Couleur |
|--------|---------|
| INFO | #10B981 (vert) |
| WARN | #F59E0B (orange) |
| ERROR | #EF4444 (rouge) |
| timestamp | #6B7280 (gris) |

### Logs exemple
```
2024-01-15T14:32:45.123Z [INFO]  Function started - Request ID: abc-123-def
2024-01-15T14:32:45.125Z [INFO]  Processing incoming request...

2024-01-15T14:32:45.145Z [INFO]  Query executed: SELECT * FROM users WHERE id = ?
2024-01-15T14:32:45.180Z [INFO]  Response sent - Status: 200 - Duration: 57ms
2024-01-15T14:32:50.015Z [WARN]  Deprecated API endpoint called: /api/v1/legacy
2024-01-15T14:33:01.240Z [ERROR] Failed to parse JSON body: Unexpected token
```

## 7. PAGINATION

- Texte: "Affichage de 15 logs sur 1,234 (page 1/83)"
- Lien: "Charger plus..."

## 8. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}/function/{functionId}/logs
Query: { period, level, search, page }
→ Logs de la fonction

GET /cloud/project/{projectId}/function/{functionId}/logs/stream
→ WebSocket pour logs temps réel

GET /cloud/project/{projectId}/function/{functionId}
→ Informations fonction (runtime, etc.)
```
