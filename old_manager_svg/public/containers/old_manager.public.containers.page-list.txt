## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/container - Liste containers
- POST /cloud/project/{id}/container - Créer container
- POST /cloud/project/{id}/container/{containerId}/restart - Redémarrer
- DELETE /cloud/project/{id}/container/{containerId} - Supprimer
