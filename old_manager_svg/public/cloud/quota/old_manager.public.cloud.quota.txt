## 7. APIS & ENDPOINTS

```
GET /cloud/project/{projectId}/quota
→ Quotas globaux du projet

GET /cloud/project/{projectId}/region/{regionId}/quota
→ Quotas par région

PUT /cloud/project/{projectId}/quota
Body: { autoScaling: true }
→ Active/désactive quota automatique

POST /cloud/project/{projectId}/quota/increase
Body: { region, resource, amount }
→ Demande d'augmentation
```
