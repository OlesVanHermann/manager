# Cahier des Charges : public/cloud - Modal vRack Add

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout service au vRack
- **Contexte** : Public Cloud / vRack
- **Objectif** : Ajouter des instances au réseau privé vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 600x480px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

- **OdsModal** : Container principal
- **OdsSelect** : Type service, VLAN
- **OdsCheckbox** : Sélection services
- **OdsButton** : Actions
