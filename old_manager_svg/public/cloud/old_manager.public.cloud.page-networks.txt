# Cahier des Charges : public/cloud - Page Networks

## 1. VUE D'ENSEMBLE
- **Type** : Page Private Networks
- **Contexte** : Public Cloud / vRack
- **Objectif** : Gérer les réseaux privés pour connecter les instances

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer réseau
- **OdsDatagrid** : Liste réseaux
- **OdsBadge** : État DHCP

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Private Networks" |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{id}/network/private - Liste réseaux
- POST /cloud/project/{id}/network/private - Créer réseau
- GET /cloud/project/{id}/network/private/{networkId}/subnet - Subnets
- DELETE /cloud/project/{id}/network/private/{networkId} - Supprimer
