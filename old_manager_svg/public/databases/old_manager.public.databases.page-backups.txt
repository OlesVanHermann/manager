# Cahier des Charges : public/databases - Page Backups

## 1. VUE D'ENSEMBLE
- **Type** : Page liste backups
- **Contexte** : Public Cloud Databases
- **Objectif** : Gérer les sauvegardes des bases de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |

- **ActionMenu** : Menu ⋮

## 10. MENU ACTIONS (⋮)

|--------|-------|
| Restaurer | Normal |

| --- | Séparateur |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/database/{dbId}/backup - Liste backups
- POST /cloud/project/{id}/database/{dbId}/backup - Créer backup
- POST /cloud/project/{id}/database/{dbId}/backup/{backupId}/restore - Restaurer
- DELETE /cloud/project/{id}/database/{dbId}/backup/{backupId} - Supprimer
