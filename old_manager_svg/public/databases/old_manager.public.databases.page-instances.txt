# Cahier des Charges : public/databases - Page Instances

## 1. VUE D'ENSEMBLE
- **Type** : Page liste databases (cartes)
- **Contexte** : Public Cloud Databases
- **Objectif** : Afficher les bases de données managées

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

- **OdsBadge** : Moteur, état

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/database - Liste databases
- POST /cloud/project/{id}/database - Créer
- GET /cloud/project/{id}/database/{dbId} - Détails
- DELETE /cloud/project/{id}/database/{dbId} - Supprimer
