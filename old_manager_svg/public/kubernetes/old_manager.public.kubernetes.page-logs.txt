# Cahier des Charges : public/kubernetes - Page Logs

## 1. VUE D'ENSEMBLE
- **Type** : Page logs cluster
- **Contexte** : Public Cloud Kubernetes
- **Objectif** : Visualiser les logs des pods et services

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsInput** : Recherche/filtre
- **OdsSelect** : Filtres (pod, namespace, période)
- **OdsButton** : Rafraîchir, Live
- **Terminal** : Affichage logs (fond sombre)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Logs du Cluster" |
| Sous-titre | "k8s-production | v1.28.3" |

## 5. BARRE FILTRES
| Élément | Largeur | Contenu |
|---------|---------|---------|
| Recherche | 300px | Input filtre texte |
| Pods | 150px | Select tous/pod spécifique |
| Namespaces | 150px | Select tous/namespace |
| Période | 150px | Select (1h, 24h, 7j) |

## 6. BOUTONS ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| ⟳ | Outline | Rafraîchir |
| ⤓ | Outline | Télécharger |
| Live | Primary #0050D7 | Auto-refresh |

## 7. TERMINAL LOGS
| Propriété | Valeur |

| Largeur | 1040px |

| Élément | Couleur | Exemple |

| Timestamp | #6B7280 | 14:32:15.123 |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/kubernetes/{clusterId}/log - Logs cluster
- GET /cloud/project/{id}/kubernetes/{clusterId}/log?pod={pod} - Logs pod
