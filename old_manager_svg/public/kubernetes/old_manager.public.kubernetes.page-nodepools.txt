- **Type** : Page node pools
- **Contexte** : Public Cloud Kubernetes
- **Objectif** : Gérer les groupes de nodes du cluster

## 12. APIS & ENDPOINTS
- GET /cloud/project/{id}/kubernetes/{clusterId}/nodepool - Liste pools
- POST /cloud/project/{id}/kubernetes/{clusterId}/nodepool - Créer
- PUT /cloud/project/{id}/kubernetes/{clusterId}/nodepool/{poolId} - Modifier
- DELETE /cloud/project/{id}/kubernetes/{clusterId}/nodepool/{poolId} - Supprimer
