# Cahier des Charges : public/kubernetes - Page Addons

## 1. VUE D'ENSEMBLE
- **Type** : Page addons/extensions
- **Contexte** : Public Cloud Kubernetes
- **Objectif** : Gérer les extensions du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et descriptions
- **OdsCard** : Cartes addons
- **OdsBadge** : État installation
- **OdsButton** : Installer, Configurer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Addons Kubernetes" |
| Description | "Extensions et intégrations pour votre cluster." |

## 5. SECTIONS
| Section | Description |
|---------|-------------|
| Addons installés | Extensions actives |
| Addons disponibles | Extensions à installer |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{id}/kubernetes/{clusterId}/addon - Liste addons
- POST /cloud/project/{id}/kubernetes/{clusterId}/addon/{addonId} - Installer
- PUT /cloud/project/{id}/kubernetes/{clusterId}/addon/{addonId} - Configurer
- DELETE /cloud/project/{id}/kubernetes/{clusterId}/addon/{addonId} - Désinstaller
