# Cahier des Charges : public/kubernetes - Page Clusters

## 1. VUE D'ENSEMBLE
- **Type** : Page liste clusters
- **Contexte** : Public Cloud Kubernetes
- **Objectif** : Afficher les clusters Kubernetes managés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

- **OdsCard** : Cartes clusters
- **OdsSummary** : Résumé ressources
