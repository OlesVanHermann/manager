# Cahier des Charges : hub - IAM Tag Détail

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'un tag
- **Type** : Liste ressources taguées
- **Objectif** : Voir et gérer les ressources d'un tag

## 2. STRUCTURE PAGE
- Lien retour "← Retour au Tag Manager"
- Titre avec valeur du tag
- Bouton action "Assigner à des ressources"
- Sous-titre avec compteur
- Toolbar avec action bulk
- Datagrid ressources sélectionnables

## 3. COMPOSANTS UI
- **OdsLink** : Retour navigation
- **OdsButton** : Actions (Assigner, Désassigner)
- **OdsDatagrid** : Tableau ressources
- **OdsBadge** : Tags sur ressources
- **OdsCheckbox** : Sélection lignes
- **OdsActionMenu** : Actions par ligne

## 4. HEADER
| Élément | Exemple |
|---------|---------|
| Lien retour | "← Retour au Tag Manager" |
| Titre | "Détail pour environment:production" |
| Bouton | "Assigner à des ressources" (primary) |

## 5. TOOLBAR SÉLECTION
| Élément | Description |
|---------|-------------|
| Bouton | "Désassigner (X)" - X = nombre sélectionnés |
| Style | Default (outline) |
| Condition | Visible si sélection > 0 |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Checkbox | Checkbox | Sélection multiple |
| Nom | Link | Nom de la ressource |
| Type de ressource | Text | dedicatedServer, vps, cloudProject... |
| Tags | Badges | Tous les tags de la ressource |
| Actions | ActionMenu | ⋮ |

## 7. TYPES DE RESSOURCES
| Type | Description |
|------|-------------|
| dedicatedServer | Serveur dédié |
| vps | Serveur VPS |
| cloudProject | Projet Public Cloud |
| domain | Nom de domaine |
| ipFailover | IP Failover |

## 8. BADGES TAGS
- Fond bleu clair (#DBEAFE)
- Texte bleu foncé (#1E40AF)
- Format : clé:valeur
- Plusieurs tags par ressource possibles

## 9. ACTION MENU (par ligne)
| Action | Description |
|--------|-------------|
| Désassigner le tag | Retire le tag de cette ressource |

## 10. ÉTATS SÉLECTION
| État | Style ligne |
|------|-------------|
| Non sélectionné | Fond blanc/gris alterné |
| Sélectionné | Fond bleu clair (#E8F0FE) |

## 11. PAGINATION
- Format : "Affichés X - Y sur Z éléments"

## 12. APIS & ENDPOINTS
- GET /iam/tag/{tagKey}/{tagValue}/resource - Ressources du tag
- DELETE /iam/tag/{tagKey}/{tagValue}/resource/{resourceId} - Désassigner
- POST /iam/tag/{tagKey}/{tagValue}/resource - Assigner ressource
