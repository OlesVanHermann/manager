# Cahier des Charges : hub - IAM Modal Désassigner Tag

## 1. VUE D'ENSEMBLE
- **Modal** : Désassigner un tag
- **Type** : Confirmation avec warning
- **Objectif** : Retirer un tag de ressources sélectionnées

## 2. STRUCTURE MODAL
- Dimensions : 600x380px
- Header avec titre
- Contenu avec warning
- Liste ressources concernées
- Message d'avertissement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsBadge** : Tag concerné
- **OdsMessage** : Warning box
- **OdsButton** : Annuler, Désassigner
- **OdsText** : Question, liste

## 4. HEADER
| Élément | Style |
|---------|-------|
| Titre | "Désassigner un tag" (20px, bold) |
| Séparateur | Ligne horizontale |

## 5. CONTENU PRINCIPAL
### Question
| Élément | Texte |
|---------|-------|
| Icône | ⚠️ dans cercle orange |
| Texte | "Êtes-vous sûr de vouloir retirer le tag" |
| Badge | [clé:valeur] (bleu) |
| Suite | "des ressources ?" |

### Liste ressources
| Label | "Ressources concernées :" |
|-------|---------------------------|
| Format | Liste à puces (•) |
| Exemple | • ns1234567.ip-51-210.eu |
| | • vps-prod-001.vps.ovh.net |

### Warning message
| Propriété | Valeur |
|-----------|--------|
| Fond | Orange clair (#FEF3C7) |
| Icône | ⚠️ |
| Texte | "Cette action retirera le tag de toutes les ressources sélectionnées." |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Default (outline) | Fermer modal |
| Désassigner | Danger (rouge #DC2626) | Confirmer suppression |

## 7. DISPOSITION BOUTONS
- Alignés à droite
- Annuler puis Désassigner

## 8. APIS & ENDPOINTS
- DELETE /iam/tag/{tagKey}/{tagValue}/resource - Désassigner
- Body : { resourceIds: [...] }
