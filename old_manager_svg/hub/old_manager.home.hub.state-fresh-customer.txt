# Cahier des Charges : home - Hub Nouveau Client

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard nouveau client
- **Type** : Empty state / Onboarding
- **Objectif** : Guider vers premier achat

## 2. STRUCTURE PAGE
- Header avec bienvenue personnalisé
- Section "Découvrez nos produits"
- Grille de cartes catalogue (6 produits)
- Bandeau aide en bas

## 3. COMPOSANTS UI
- **OdsCard** : Cartes produits
- **OdsButton** : Boutons "Commander"
- **OdsText** : Titres, descriptions
- **OdsMessage** : Bandeau aide

## 4. HEADER
| Élément | Valeur |
|---------|--------|
| Titre | "Bienvenue [Nom] !" |
| Sous-titre | Zone de confiance |
| Section | "Découvrez nos produits" |
| Message | "Vous n'avez pas encore de services. Découvrez notre catalogue :" |

## 5. GRILLE PRODUITS
### Disposition
- 4 colonnes sur desktop
- 2 colonnes sur tablette
- 1 colonne sur mobile

### Cartes produits
| Produit | Icône | Description |
|---------|-------|-------------|
| VPS | S | Serveurs virtuels performants pour vos projets web |
| Serveur Dédié | D | Puissance et contrôle total sur votre infrastructure |
| Hébergement Web | W | Hébergez vos sites web facilement et rapidement |
| Public Cloud | C | Infrastructure cloud flexible à la demande |
| Nom de domaine | @ | Réservez votre nom de domaine idéal |
| Email Pro | E | Messagerie professionnelle sécurisée |

### Structure carte
- Icône dans carré bleu clair
- Nom produit (bold)
- Description (2 lignes, muted)
- Bouton "Commander" (primary)

## 6. BOUTONS COMMANDER
| Produit | Action |
|---------|--------|
| VPS | → /order/vps |
| Serveur Dédié | → /order/dedicated |
| Hébergement Web | → /order/hosting |
| Public Cloud | → /order/cloud |
| Nom de domaine | → /order/domain |
| Email Pro | → /order/email-pro |
