# Cahier des Charges : home - Hub Widget Statuts Paiement

## 1. VUE D'ENSEMBLE
- **Widget** : Derniers statuts de paiement des services
- **Type** : Tableau récapitulatif
- **Objectif** : Surveiller les échéances services

## 2. STRUCTURE WIDGET
- Carte blanche avec bordure
- Header avec titre et lien "Voir tout"
- Tableau 3 colonnes
- Lignes alternées

## 3. COMPOSANTS UI
- **OdsCard** : Container
- **OdsDatagrid** : Tableau services
- **OdsBadge** : Statuts
- **OdsLink** : Header "Voir tout"

## 4. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Service | Text | Nom du service (ex: vps-abc123.ovh.net) |
| Statut | Badge | Actif / À renouveler |
| Échéance | Text muted | "avant le DD/MM/YYYY" ou "Immédiatement" |

## 5. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| Actif | Vert (#DEF7EC) | Service payé |
| À renouveler | Orange (#FEF3C7) | Échéance proche ou dépassée |

## 6. EXEMPLES DONNÉES
| Service | Statut | Échéance |
|---------|--------|----------|
| vps-abc123.ovh.net | Actif | avant le 15/01/2025 |
| ns123456.ip-192-168-1.eu | Actif | avant le 20/01/2025 |
| pcc-192-0-2-1.ovh.com | À renouveler | Immédiatement |
| domain-example.com | Actif | avant le 01/03/2025 |

## 7. ACTIONS
- **Voir tout** : Navigation vers /billing/services

## 8. APIS & ENDPOINTS
- GET /services?sortBy=expirationDate - Services avec échéance
