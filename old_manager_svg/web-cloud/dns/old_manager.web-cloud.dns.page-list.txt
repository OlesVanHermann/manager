# Cahier des Charges : web-cloud/dns - Page List

## 1. VUE D'ENSEMBLE
- **Type** : Page liste zones DNS
- **Contexte** : Web Cloud DNS
- **Objectif** : Vue consolidée de toutes les zones DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et sous-titres
- **OdsButton** : CTA créer zone
- **OdsCard** : Cartes métriques
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste zones
- **OdsBadge** : Statut
- **OdsActionMenu** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Zones DNS" |
| Sous-titre | "Gérez les enregistrements DNS de vos domaines" |
| Bouton | "+ Créer une zone DNS" |

## 5. CARTES MÉTRIQUES (3 colonnes)
| Carte | Label | Détail |
|-------|-------|--------|
| Zones DNS | Nombre | "Toutes actives" (vert) |
| Enregistrements | Nombre total | "A, AAAA, CNAME, MX, TXT..." |
| Anycast | Nombre | "Zones avec Anycast activé" |

## 6. FILTRES
| Élément | Type |
|---------|------|
| Recherche | Input avec placeholder "Rechercher une zone..." |

## 7. DATAGRID
| Colonne | Description |
|---------|-------------|
| Zone | Nom de la zone (lien bleu) |
| Enregistrements | "XX enregistrements" |
| DNSSEC | Badge Actif/Inactif |
| Anycast | Badge Activé/Désactivé |
| Dernière modif. | "Il y a X heures/jours" ou "En cours" |
| Statut | Badge Propagé/Propagation |
| Actions | Menu ⋮ |

## 8. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Propagé | #DCFCE7 | #16A34A |
| Propagation | #FEF3C7 | #92400E |
| Actif | #DCFCE7 | #16A34A |
| Inactif | #F9FAFB | #6B7280 |
| Activé | #DCFCE7 | #16A34A |
| Désactivé | #F9FAFB | #6B7280 |

## 9. APIS & ENDPOINTS
- GET /domain/zone - Liste zones DNS
- GET /domain/zone/{zoneName} - Détail zone
- GET /domain/zone/{zoneName}/record - Enregistrements
