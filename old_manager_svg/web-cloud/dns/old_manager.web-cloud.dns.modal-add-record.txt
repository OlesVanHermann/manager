# Cahier des Charges : web-cloud/dns - Modal Add Record

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout enregistrement DNS
- **Contexte** : Web Cloud DNS Management
- **Objectif** : Ajouter un nouvel enregistrement DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsButton** : Types d'enregistrement (toggle group)
- **OdsInput** : Sous-domaine et cible
- **OdsRadio** : Sélection TTL
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un enregistrement DNS" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. TYPES D'ENREGISTREMENT
| Type | Couleur sélectionné |
|------|---------------------|
| A | #16A34A (par défaut) |
| AAAA | default |
| CNAME | default |
| MX | default |
| TXT | default |
| SRV | default |

## 6. FORMULAIRE
| Champ | Type | Format |
|-------|------|--------|
| Sous-domaine * | Input + Suffixe | "[www].example.com" |
| Cible (IPv4) * | Input | "192.0.2.1" |

### Label cible dynamique par type
| Type | Label cible |
|------|-------------|
| A | "Cible (IPv4)" |
| AAAA | "Cible (IPv6)" |
| CNAME | "Cible (hostname)" |
| MX | "Serveur mail" |
| TXT | "Valeur" |

## 7. SÉLECTION TTL
| TTL | Description |
|-----|-------------|
| 60 | (1m) |
| 300 | (5m) |
| 3600 | (1h) - sélectionné |
| 86400 | (1j) |

## 8. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Ajouter | Primary #0050D7 | Droite |

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record - Créer enregistrement
- POST /domain/zone/{zoneName}/refresh - Rafraîchir zone
