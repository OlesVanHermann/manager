## 8. APIS & ENDPOINTS

```
# Informations du numéro
GET /telephony/{billingAccount}/number/{serviceName}
→ {
  featureType: "easyHunting",
  serviceName: "+33123456789",
  description: "Mon numéro principal",
  country: "fr"
}

# Résiliation programmée
GET /telephony/{billingAccount}/number/{serviceName}/pendingTermination
→ { terminationDate: "2025-02-15" }

# Annuler la résiliation
DELETE /telephony/{billingAccount}/number/{serviceName}/pendingTermination

# Statistiques temps réel (pour CCS)
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/liveStatistics
→ {
  callsInQueue: 0,
  callsInProgress: 2,
  activeAgents: 5
}

# Groupe associé
GET /telephony/{billingAccount}
→ { billingAccount: "ab12345-ovh-1", description: "..." }
```
