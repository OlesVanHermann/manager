## 1. RESUME EXECUTIF

Modal import liste numeros abreges depuis fichier CSV.
Validation format/contenu, apercu donnees, import avec progression.

--------------------------------------------------------------------------------
## 2. ARCHITECTURE
--------------------------------------------------------------------------------

Source : telecom-universe-components/abbreviatedNumbers/import/
- Template : telecom-telephony-abbreviated-numbers-import.modal.html
- Controller : telecom-telephony-abbreviated-numbers-import.modal.js

Dependances : CSV-JS, tuc-input-file

--------------------------------------------------------------------------------
## 3. FORMAT CSV
--------------------------------------------------------------------------------

Sans entete, 4 colonnes :
abbreviatedNumber,destinationNumber,name,surname
2000,0033123456789,Jean,Dupont

--------------------------------------------------------------------------------
## 4. VALIDATION
--------------------------------------------------------------------------------

Patterns :
- abbreviatedNumber : /^\d{3,4}$/ (injecte via pattern.regexp)
- destinationNumber : /^00\d{2,3}[\s\d]+$/
- name/surname : /^[a-zA-Z0-9\s]+$/

Fichier : .csv obligatoire, max 50ko

--------------------------------------------------------------------------------
## 5. ETAPES
--------------------------------------------------------------------------------

1. Selection fichier (tuc-input-file)
2. Apercu donnees (oui-datagrid) - lignes invalides en rouge
3. Import (progression) - par chunks de 10, delai 100ms
4. Resultat - imported[], rejected count
