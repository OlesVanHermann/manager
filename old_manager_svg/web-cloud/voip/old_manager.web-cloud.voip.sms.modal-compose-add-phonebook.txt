# Cahier des Charges : telecom/sms - Modal ajouter contacts du carnet

## 1. VUE D'ENSEMBLE
- **Type** : Modal de sélection
- **Contexte** : Page composition SMS
- **Objectif** : Ajouter des contacts du carnet d'adresses comme destinataires

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 650px |
| Modal hauteur | 540px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Info sélection
- **OdsSelect** : Carnet d'adresses, Type de numéro
- **OdsDatagrid** : Tableau des contacts
- **OdsCheckbox** : Sélection contacts
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter des contacts du carnet d'adresses" |

## 5. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Sélectionnez les contacts à ajouter au message" |

## 6. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Carnet d'adresses | Select | Liste des carnets |
| Type de numéro | Select | Tous, Mobile pro, Mobile perso, Fixe |

## 7. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Checkbox | Checkbox | Sélection |
| Nom | Texte | Nom de famille |
| Prénom | Texte | Prénom |
| Numéro | Téléphone | Format international |
| Type | Badge | Type de numéro |

## 8. COMPTEUR SÉLECTION
| Format |
|--------|
| "X contact(s) sélectionné(s)" |

## 9. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Ajoute contacts sélectionnés |

## 10. APIS & ENDPOINTS
- GET /sms/{serviceName}/phonebook - Liste carnets d'adresses
- GET /sms/{serviceName}/phonebook/{bookId}/contact - Liste contacts
