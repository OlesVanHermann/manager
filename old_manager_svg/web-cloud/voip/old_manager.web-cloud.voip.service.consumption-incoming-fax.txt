## 7. APIS & ENDPOINTS

### 7.1 Lister les fax reçus

```
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns?type=incoming
```

Response:
```json
{
  "totalCount": 8,
  "totalPages": 24,
  "data": [
    {
      "id": "fax-001",
      "date": "2025-12-28T14:32:15Z",
      "callerNumber": "+33123456789",
      "pages": 3,
      "type": "incoming"
    }
  ]
}
```

### 7.2 Télécharger un fax

```
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}/document
```
