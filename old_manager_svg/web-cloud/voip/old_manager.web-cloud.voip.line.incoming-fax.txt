## 6. APIS & ENDPOINTS

```
# Historique fax entrants
GET /telephony/{billingAccount}/line/{serviceName}/consumption/incomingFax
Query: ?dateFrom=2024-12-01&dateTo=2024-12-31&page=1&pageSize=10
→ faxes: [{ date, number, pages, nature }], totalFaxes, totalPages

# Télécharger un fax
GET /telephony/{billingAccount}/line/{serviceName}/fax/{faxId}/download
→ URL ou fichier PDF
```
