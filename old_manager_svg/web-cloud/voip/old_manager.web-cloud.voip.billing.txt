# Cahier des Charges : telecom/billing - Facturation

## 1. RESUME EXECUTIF

### Perimetre
Page principale de facturation d'un groupe de telephonie.

### Objectif fonctionnel
- Acceder aux differentes sections de facturation
- Voir les informations du compte
- Gerer depot et seuil de credit

- Depot = garantie consommations hors forfait
- Seuil = limite avant blocage appels
- Historique disponible 12 mois
- Factures telechargeables PDF/CSV
