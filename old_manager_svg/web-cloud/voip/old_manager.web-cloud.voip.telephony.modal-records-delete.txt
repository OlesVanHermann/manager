# Cahier des Charges : telecom/telephony - Modal supprimer les enregistrements

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page enregistrements d'appels
- **Objectif** : Confirmer la suppression d'enregistrements sélectionnés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 190px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsIcon** : Icône warning
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Confirmer

## 4. CONTENU MODAL

### Icône Warning
| Propriété | Valeur |
|-----------|--------|
| Icône | ⚠ |
| Fond | Jaune (#FEF3C7) |
| Taille | 50px (cercle) |

### Titre
| Texte |
|-------|
| "Supprimer les enregistrements" |

### Message
| Texte |
|-------|
| "Êtes-vous sûr de vouloir supprimer ces {n} enregistrements d'appels ?" |

## 5. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Warning (jaune) | Supprime les enregistrements |

## 6. APIS & ENDPOINTS
- DELETE /telephony/{billingAccount}/line/{serviceName}/records - Supprimer enregistrements (bulk)
- DELETE /telephony/{billingAccount}/line/{serviceName}/records/{recordId} - Supprimer un enregistrement
