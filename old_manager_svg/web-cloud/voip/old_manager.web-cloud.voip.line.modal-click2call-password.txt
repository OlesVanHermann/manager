# Cahier des Charges : telecom/line - Modal mot de passe Click2Call

## 1. RESUME EXECUTIF

### Perimetre
Modal de changement de mot de passe d'un utilisateur Click2Call.

### Objectif fonctionnel
- Changer le mot de passe d'un utilisateur
- Valider la force du mot de passe

## 3. COMPOSANTS UI

### Header page
| Element | Description |
|---------|-------------|
| Retour | ‚Üê Retour a Click2Call |
| Titre page | Click2Call |

### Carte formulaire
| Element | Description |
|---------|-------------|
| Titre | Changer le mot de passe - {username} |

### Champ mot de passe
| Element | Description |
|---------|-------------|
| Label | Nouveau mot de passe * |
| Type | Password avec toggle visibilite |
| Icone | Oeil pour afficher |

### Indicateur force
| Niveau | Couleur | Segments |
|--------|---------|----------|
| Faible | Rouge | 1/4 |
| Moyenne | Orange | 2/4 |
| Bonne | Vert | 3/4 |
| Forte | Vert | 4/4 |

### Texte force
| Element | Description |
|---------|-------------|
| Texte | Force: {niveau} |
| Couleur | Meme que segments |

### Boutons
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondary | Ferme modal |
| Valider | Primary | Sauvegarde |

## 4. APIS & DONNEES

### Endpoints
```
PUT /telephony/{billingAccount}/line/{serviceName}/click2Call/user/{id}/password
```

### Modele
```typescript
interface Click2CallPasswordChange {
  password: string;
}

interface PasswordStrength {
  level: 'weak' | 'medium' | 'good' | 'strong';
  score: number;
}
```
