# Cahier des Charges : telecom/telephony - Convertir en numéro alias

## 1. VUE D'ENSEMBLE
- **Type** : Page de conversion
- **URL** : /telephony/{billingAccount}/line/{serviceName}/management/convert
- **Objectif** : Convertir une ligne VoIP en numéro alias (redirection)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne
- **OdsLink** : Retour au tableau de bord
- **OdsCard** : Informations + Action
- **OdsButton** : Convertir

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour au tableau de bord de la ligne" |
| Titre | "Convertir en numéro alias" |

| Propriété | Valeur |
|-----------|--------|
| Titre | "Informations" |
