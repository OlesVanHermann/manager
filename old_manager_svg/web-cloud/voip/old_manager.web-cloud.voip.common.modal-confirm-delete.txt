# Cahier des Charges : telecom/common - Modal Confirmation de suppression

## 1. RESUME EXECUTIF

### Perimetre
Composant modal de confirmation de suppression.

### Objectif fonctionnel
- Confirmer une action de suppression
- Afficher l'element a supprimer
- Proposer confirmation ou annulation

---

## 2. LAYOUT

### Structure modal centree avec icone danger

---

## 3. COMPOSANTS UI

### Modal Container
| Element | Description |
|---------|-------------|
| Overlay | Fond semi-transparent |
| Modal | Boite blanche avec shadow |
| Dimensions | 400x220px |

### Icone
| Element | Description |
|---------|-------------|
| Type | Warning (triangle) |
| Couleur | Rouge sur fond rouge clair |

### Titre
| Element | Description |
|---------|-------------|
| Texte | Suppression |
| Style | 18px, gras, centre |

### Message
| Element | Description |
|---------|-------------|
| Format | Question de confirmation |
| Exemple | "Etes-vous certain de vouloir supprimer cette plage horaire ?" |

### Boutons
| Bouton | Type | Action |
|--------|------|--------|
| Confirmer | Danger (rouge) | Execute la suppression |
| Annuler | Default | Ferme la modal |

---

## 4. APIS & DONNEES

### Props
| Prop | Type | Description |
|------|------|-------------|
| title | string | Titre de la modal |
| message | string | Message de confirmation |
| itemName | string | Nom de l'element a supprimer |
| onConfirm | function | Callback de confirmation |
| onCancel | function | Callback d'annulation |

---

## 6. REGLES METIER

- Icone warning obligatoire
- Bouton confirm rouge (action destructive)
- Focus par defaut sur Annuler
- Echap ferme la modal
