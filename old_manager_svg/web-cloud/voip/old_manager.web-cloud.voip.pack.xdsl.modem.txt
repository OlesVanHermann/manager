## 1. RÉSUMÉ EXÉCUTIF

Page principale de configuration du modem xDSL permettant d'accéder aux
différentes sections de paramétrage : WiFi, réseau, pare-feu, etc.

### Utilisateurs cibles
- Clients disposant d'un accès xDSL avec modem géré par OVHcloud

### Objectif fonctionnel
- Centraliser l'accès à toutes les configurations modem
- Afficher l'état général du modem
- Alerter si le modem n'est pas configurable ou en mode bridge
