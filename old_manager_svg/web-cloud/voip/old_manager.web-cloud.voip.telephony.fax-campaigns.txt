# CAHIER DES CHARGES - old_manager.telecom.telephony.fax-campaigns
# Page campagnes de fax

## 1. RESUME EXECUTIF

### Perimetre
Gestion des envois de fax en masse (campagnes).

### Objectif fonctionnel
Permettre l'envoi de fax a de multiples destinataires avec suivi.

## 4. COMPOSANTS UI

### Formulaire Nouvelle Campagne
| Champ | Description |
|-------|-------------|
| Nom | Nom de la campagne |
| Document | PDF a faxer |
| Destinataires | Liste numeros |
| Date envoi | Immediat ou programme |

### Datagrid Campagnes
| Colonne | Description |
|---------|-------------|
| ID | Identifiant |
| Nom | Nom campagne |
| Total | Nombre destinataires |
| Envoyes | Nombre succes |
| Echecs | Nombre echecs |
| Statut | En cours/Termine |

## 5. APIs & DONNEES

### Endpoints

```
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns/{id}
DELETE /telephony/{billingAccount}/fax/{serviceName}/campaigns/{id}
POST /telephony/{billingAccount}/fax/{serviceName}/settings/sendFax
Body: { pdfUrl: string, recipients: Array<string>, dateSchedule: string }
```

## 7. REGLES METIER

### Limites
- Maximum 200 destinataires par campagne
- Format PDF uniquement
- Taille max: 5 Mo

### Statuts campagne
| Statut | Description |
|--------|-------------|
| todo | Programmee |
| doing | En cours |
| done | Terminee |
| failed | Echecs partiels |
