# Cahier des Charges : telecom/sms - Modal supprimer modèle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page modèles SMS
- **Objectif** : Supprimer un modèle SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Détails du modèle
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le modèle SMS" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Confirmez-vous la suppression de ce modèle ?" |

## 6. DÉTAILS MODÈLE
| Champ | Description |
|-------|-------------|
| Nom | Identifiant du modèle |
| Date de création | DD/MM/YYYY |
| Activité | Secteur d'activité |
| Description | Texte tronqué |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le modèle |

## 8. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/templates/{templateId} - Supprimer modèle
