# Cahier des Charges : web-cloud/voip/alias - Config Redirect

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire simple
**Fonction:** Configurer la redirection d'appels vers une ligne
**Contexte:** Configuration redirect du numéro alias (type DDI)

## 2. NAVIGATION

### Breadcrumb
Télécom › Groupe VoIP › Numéro alias 0033972XXXXXX › Configuration Redirection

## 3. DIMENSIONS

- Carte principale: 800x380px
- Hauteur page: 550px

## 4. SECTION APPELS ENTRANTS

### Titre section
"Vos appels entrants"

### Ligne de redirection
- Label: "Ligne vers laquelle vos appels seront redirigés"
- Valeur affichée: "Poste 7001 - +33 9 72 10 00 01"

### Bouton modification
- "+ Modifier la ligne" (btn-outline, 180x36px)

### Avertissement légal
- Texte: "Veuillez noter que la législation interdit les redirections vers les numéros surtaxés."
- Style: text-muted, font-size 12px

### Note aide
- Ligne 1: "Pour faire une redirection vers une ligne externe, il vous faut configurer votre numéro en"
- Ligne 2: "File d'appels" (lien bleu)

## 5. SECTION APPELS SORTANTS (facultatif)

### Titre section
"Vos appels sortants" + "(facultatif)" en gris

### Checkbox présentation
- ☐ "Utiliser ce numéro comme identifiant lors de vos appels sortants"
- Note: "Cette option permet d'afficher le numéro alias lors des appels sortants depuis la ligne de redirection."

## 6. ACTIONS

- Bouton "Annuler" (btn-outline, 100x36px)
- Bouton "Valider" (btn-primary, 100x36px)

## 7. APIS & ENDPOINTS

```
# Configuration redirection
GET /telephony/{billingAccount}/number/{serviceName}/redirect
→ destination, outgoingPresentation

PUT /telephony/{billingAccount}/number/{serviceName}/redirect
Body: {
  destination: "+33972100001",
  outgoingPresentation: true
}

# Changer la destination
PUT /telephony/{billingAccount}/number/{serviceName}/changeDestination
Body: {
  destination: "+33972100002"
}

# Lignes disponibles
GET /telephony/{billingAccount}/line
→ Liste des lignes pouvant recevoir la redirection
```
