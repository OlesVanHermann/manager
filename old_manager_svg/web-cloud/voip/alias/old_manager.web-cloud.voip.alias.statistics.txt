# Cahier des Charges : web-cloud/voip/alias - Statistics

## 1. VUE D'ENSEMBLE

**Type:** Dashboard statistiques avec KPIs et datagrids
**Fonction:** Statistiques d'appels en temps réel de la file d'appels
**Contexte:** Configuration file d'appels du numéro alias

## 2. NAVIGATION

### Breadcrumb
Télécom › Groupe VoIP › Numéro alias 0033972XXXXXX › Statistiques

## 3. HEADER

- Titre: "Statistiques d'appels"
- Sous-titre: "Consultez en temps réel les statistiques ainsi que l'état de votre file d'appels"
- Note: "Les données sont réinitialisées tous les jours à 00h heure française" (text-muted)

## 7. APIS & ENDPOINTS

```
# Statistiques globales
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/statistics
→ {
  qos: 87,
  totalCalls: 156,
  answeredCalls: 136,
  lostCalls: 20,
  avgWaitTime: 45,
  avgCallDuration: 272
}

# Appels en cours
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/liveCalls
→ Liste des appels en cours

# Agents de la file
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/agent
→ Liste des agents avec statuts

# Actions sur appel
POST /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/liveCalls/{id}/eavesdrop
POST /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/liveCalls/{id}/transfer
POST /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/liveCalls/{id}/hangup
```
