## 6. APIS & ENDPOINTS

```
# Lister les agents
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent
→ Liste des IDs agents

# Détails agent
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}
→ number, description, status, timeout, wrapUpTime, simultaneousLines

# Ajouter un agent
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent
Body: {
  number: "+33612345678",
  description: "Nouvel agent",
  timeout: 20,
  wrapUpTime: 5,
  simultaneousLines: 1
}

# Modifier un agent
PUT /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}
Body: { description: "Nouveau nom", timeout: 25 }

# Supprimer un agent
DELETE /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}

# Statut agent (connexion/déconnexion)
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}/logOn
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/agent/{agentId}/logOff
```
