**Fonction:** Gérer les synthèses vocales (Text-to-Speech) du numéro alias

## 8. APIS & ENDPOINTS

```
# Lister les TTS
GET /telephony/{billingAccount}/tts
→ Liste des synthèses vocales

# Détails TTS
GET /telephony/{billingAccount}/tts/{id}
→ voice, text, creationDate

# Créer un TTS
POST /telephony/{billingAccount}/tts
Body: {
  voice: "fr-FR-Claire",
  text: "Bienvenue chez OVHcloud..."
}

# Supprimer un TTS
DELETE /telephony/{billingAccount}/tts/{id}

# Prévisualiser (générer audio)
POST /telephony/{billingAccount}/tts/{id}/preview
→ audioUrl

# Voix disponibles
GET /telephony/tts/availableVoices
→ Liste des voix par langue
```
