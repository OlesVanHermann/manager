## 6. APIS & ENDPOINTS

```
# Demander la résiliation
POST /telephony/{billingAccount}/number/{serviceName}/terminate
Body: {
  reason: "cessation"
}

# Annuler la résiliation (avant facturation)
DELETE /telephony/{billingAccount}/number/{serviceName}/terminate

# Vérifier le statut de résiliation
GET /telephony/{billingAccount}/number/{serviceName}/terminate
→ status, requestDate, effectiveDate

# KYC / Coordonnées
GET /telephony/{billingAccount}/contact
PUT /telephony/{billingAccount}/contact
```
