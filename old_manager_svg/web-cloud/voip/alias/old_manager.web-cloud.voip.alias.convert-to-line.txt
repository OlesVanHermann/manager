## 6. APIS & ENDPOINTS

```
# Convertir numéro en ligne
POST /telephony/{billingAccount}/number/{serviceName}/convertToLine
Body: {
  offer: "voip-decouverte",
  contracts: ["cgv", "cps"]
}

# Offres disponibles
GET /telephony/{billingAccount}/number/{serviceName}/convertToLineOffers
→ Liste des offres de conversion disponibles

# Annuler la conversion (avant facturation)
DELETE /telephony/{billingAccount}/number/{serviceName}/convertToLine
```
