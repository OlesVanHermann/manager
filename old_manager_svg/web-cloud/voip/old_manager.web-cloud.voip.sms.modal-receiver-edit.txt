## 1. INFORMATIONS GENERALES

| Champ | Valeur |
|-------|--------|
| Type | Modal formulaire |
| Parent | receivers |
| Declencheur | Action "Editer" sur une liste |
| Fichier source | receivers/edit/telecom-sms-receivers-edit.html |

--------------------------------------------------------------------------------
## 2. CONTEXTE FONCTIONNEL
--------------------------------------------------------------------------------

Permet de modifier les proprietes d une liste de destinataires existante :
- Renommer la liste
- Modifier la description
- Activer/desactiver la mise a jour automatique

--------------------------------------------------------------------------------
## 3. FORMULAIRE
--------------------------------------------------------------------------------

| Champ | Type | Validation |
|-------|------|------------|
| Description | Input text | Optionnel |
| Mise a jour automatique | Toggle/Checkbox | - |

**Note:** Le slot ID ne peut pas etre modifie.

--------------------------------------------------------------------------------
## 4. BOUTONS
--------------------------------------------------------------------------------

- Valider / Enregistrer (primary)
- Annuler (secondary)

--------------------------------------------------------------------------------
## 5. API
--------------------------------------------------------------------------------

| Methode | Endpoint | Description |
|---------|----------|-------------|
| PUT | /sms/{serviceName}/receivers/{slotId} | Modifier liste |

**Body:**
```json
{
  "description": "string",
  "autoUpdate": true/false
}
```

--------------------------------------------------------------------------------
## 6. MESSAGES RETOUR
--------------------------------------------------------------------------------

- Loading : "Mise a jour de la liste des destinataires..."
- Succes : "Liste des destinataires mise a jour !"
- Erreur : "Impossible d editer la liste de destinataires. [error]"
