# Cahier des Charges : telecom/line - Detacher le telephone

## 1. RESUME EXECUTIF

### Perimetre
Detachement d'un telephone Plug & Phone de sa ligne pour reassociation.

### Objectif fonctionnel
- Dissocier un telephone de la ligne actuelle
- Permettre la reassociation sur une autre ligne
- Recuperer l'adresse IP pour la procedure

## 3. COMPOSANTS UI

### Zone information
| Element | Description |
|---------|-------------|
| Texte | Explication du detachement |
| Modele | Cisco SPA504G (gras) |
| MAC | 00:1A:2B:3C:4D:5E (gras) |
| Consequence | Pourra etre associe a une autre ligne |

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Adresse IP | Input | IP actuelle du telephone |
| Placeholder | Ex: 192.168.1.100 |

### Bouton
| Bouton | Type | Action |
|--------|------|--------|
| Detacher le telephone | Primary | Execute detachement |

## 4. APIS & DONNEES

### Endpoints

GET /telephony/{billingAccount}/line/{serviceName}/phone
