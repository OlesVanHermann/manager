# Cahier des Charges : telecom/telephony - Transfert depot de garantie

## 1. RESUME EXECUTIF

### Perimetre
Transfert de depot de garantie entre groupes VoIP.

### Objectif fonctionnel
- Transferer tout ou partie du depot
- Selectionner groupe source et destination
- Visualiser les montants avant/apres

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony
    Response: [billingAccount1, ...]

GET /telephony/{billingAccount}
    Response: { securityDeposit }

POST /telephony/{billingAccount}/transferSecurityDeposit
    Body: { amount, billingAccountDestination }
```
