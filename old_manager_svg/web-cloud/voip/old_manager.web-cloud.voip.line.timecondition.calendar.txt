# Cahier des Charges : telecom/line - Calendrier plages horaires

## 1. RESUME EXECUTIF

### Perimetre
Vue calendrier hebdomadaire des plages horaires configurees sur la ligne.

### Objectif fonctionnel
- Visualiser les plages horaires de la semaine
- Naviguer entre les semaines
- Identifier les creneaux par couleur
- Cliquer pour modifier un creneau

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/timeCondition
    Response: [{
      id, day, timeFrom, timeTo,
      slot: "slot1" | "slot2" | "slot3" | "available" | "unavailable",
      policy
    }]

GET /telephony/{billingAccount}/line/{serviceName}/timeCondition/slots
    Response: {
      slot1: { number, type },
      slot2: { number, type },
      slot3: { number, type }
    }
```

### Jours de la semaine (enum)
```
monday, tuesday, wednesday, thursday, friday, saturday, sunday
```
