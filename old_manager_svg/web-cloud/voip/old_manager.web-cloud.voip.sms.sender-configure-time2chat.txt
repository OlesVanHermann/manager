# Cahier des Charges : telecom/sms - Page configurer Time2Chat

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /sms/{serviceName}/senders/time2chat/configure
- **Objectif** : Configurer un expéditeur Time2Chat existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 800px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / SMS
- **OdsLink** : Retour aux expéditeurs
- **OdsCard** : Cards configuration
- **OdsInput** : Champs formulaire
- **OdsButton** : Modifier

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour aux expéditeurs" |
| Titre | "Configurer Time2Chat" |

## 5. LAYOUT 2 COLONNES
| Colonne | Contenu |
|---------|---------|
| Gauche | Configuration réponse automatique |
| Droite | Informations KYC |

## 6. CARD RÉPONSE AUTOMATIQUE
| Section | Champs |
|---------|--------|
| Informations de contact | Nom commercial, Numéro, Site web |
| Action | Bouton "Modifier" |

## 7. CHAMPS CONTACT
| Champ | Type | Requis |
|-------|------|--------|
| Nom commercial | Input | Oui * |
| Numéro de téléphone | Input tel | Oui * |
| Site web | Input URL | Oui * |

## 8. CARD INFORMATIONS KYC
| Champ | Type | Requis |
|-------|------|--------|
| Nom de l'entreprise | Input | Oui * |
| Nom de la marque | Input | Oui * |
| Cas d'usage principal | Input | Oui * |
| Nom du contact opérationnel | Input | Oui * |
| E-mail du contact | Input email | Oui * |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/sender/time2chat - Configuration actuelle
- PUT /sms/{serviceName}/sender/time2chat - Modifier configuration
