## 1. RÉSUMÉ EXÉCUTIF

### Périmètre
Page de liste des services Carrier SIP de l'utilisateur.

### Objectif fonctionnel
Afficher tous les trunks Carrier SIP avec navigation vers le dashboard de chaque service.

### Utilisateurs cibles
Clients OVHcloud disposant d'un ou plusieurs services Carrier SIP.

### Note
Le code source de la liste n'est pas fourni dans le tar. Cette spécification est basée sur les patterns standards OVH.
