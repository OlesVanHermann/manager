## 9. APIS & ENDPOINTS

```
# Récupérer configuration transferts
GET /telephony/{billingAccount}/line/{serviceName}/forward
→ {
  unconditional: {
    enabled: false,
    number: null
  },
  busy: {
    enabled: true,
    number: "+33612345678"
  },
  noAnswer: {
    enabled: true,
    number: "+33612345678",
    timeout: 20
  }
}

# Activer/désactiver renvoi inconditionnel
PUT /telephony/{billingAccount}/line/{serviceName}/forward/unconditional
Body: {
  enabled: true,
  number: "+33612345678"
}

# Configurer renvoi sur occupation
PUT /telephony/{billingAccount}/line/{serviceName}/forward/busy
Body: {
  enabled: true,
  number: "+33612345678"
}

# Configurer renvoi sur non-réponse
PUT /telephony/{billingAccount}/line/{serviceName}/forward/noAnswer
Body: {
  enabled: true,
  number: "+33612345678",
  timeout: 20
}
```
