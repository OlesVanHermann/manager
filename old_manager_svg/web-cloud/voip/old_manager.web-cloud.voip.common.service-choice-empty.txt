# Cahier des Charges : telecom/common - Service Choice Etat vide

## 1. RESUME EXECUTIF

### Perimetre
Etat vide de la modal de choix de service (aucun resultat).

### Objectif fonctionnel
- Afficher un message quand aucun resultat
- Proposer d'effacer la recherche
- Donner des conseils de recherche

---

## 2. LAYOUT

### Structure modal avec zone vide + conseils

---

## 3. COMPOSANTS UI

### Recherche active
| Element | Description |
|---------|-------------|
| Input | Avec terme recherche |
| Focus | Bordure bleue |
| Clear | Bouton X pour effacer |

### Zone vide
| Element | Description |
|---------|-------------|
| Illustration | Icone loupe grise |
| Titre | Aucun resultat trouve |
| Message | Aucun service ne correspond a votre recherche |
| Terme | "xyz123456" (en gras) |
| Action | Lien "Effacer la recherche" |

### Conseils
| Element | Description |
|---------|-------------|
| Fond | Bleu clair (#EFF6FF) |
| Icone | Ampoule |
| Titre | Conseils de recherche |
| Tips | Liste de suggestions |

### Conseils de recherche
| Conseil |
|---------|
| Verifiez l'orthographe de votre recherche |
| Essayez avec un numero de telephone ou un nom de service |

### Footer
| Bouton | Type | Etat |
|--------|------|------|
| Valider | Disabled | Grise (pas de selection) |
| Annuler | Default | Actif |

---

## 4. APIS & DONNEES

### Props
| Prop | Type | Description |
|------|------|-------------|
| searchTerm | string | Terme recherche |
| onClear | function | Callback effacement |
