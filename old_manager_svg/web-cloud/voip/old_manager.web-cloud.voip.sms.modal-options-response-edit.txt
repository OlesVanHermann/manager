# Cahier des Charges : telecom/sms - Modal modifier suivi réponses

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification
- **Contexte** : Page options SMS
- **Objectif** : Modifier une option de suivi des réponses SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type de média, Numéro expéditeur
- **OdsInput** : Numéro destinataire
- **OdsMessage** : Info
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'option de suivi" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Type de média | Select | Oui |
| Numéro expéditeur | Select | Oui * |
| Numéro destinataire | Input | Oui * |

## 6. OPTIONS MÉDIA
| Option | Description |
|--------|-------------|
| E-mail | Transfert par email |
| SMS | Transfert par SMS |
| HTTP | Callback HTTP |

## 7. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Les réponses seront transférées par SMS au numéro spécifié." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre modifications |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/smsResponse/{hash} - Détail suivi
- PUT /sms/{serviceName}/smsResponse/{hash} - Modifier suivi
