## 6. RÈGLES MÉTIER

| Règle | Description |
|-------|-------------|
| Date d'effet | Fin du mois en cours |
| Reversements SVA | Perdus si non réclamés |
| Portabilité sortante | Impossible après résiliation |
| Annulation | Possible jusqu'à la date d'effet |

## 7. APIS & ENDPOINTS

```
# Vérifier si le numéro est SVA
GET /telephony/{billingAccount}/number/{serviceName}
→ { featureType: "...", isSva: true }

# Solde SVA (si applicable)
GET /telephony/{billingAccount}/svaWallet/{serviceName}/summary
→ { availableBalance: 245.50 }

# Résilier le numéro
POST /telephony/{billingAccount}/number/{serviceName}/terminate
Body: {
  reason: "endOfActivity",
  details: "Fermeture de la société"
}
→ { terminationDate: "2025-01-31" }

# Annuler la résiliation
DELETE /telephony/{billingAccount}/number/{serviceName}/pendingTermination

# Obtenir la date d'effet
GET /telephony/{billingAccount}/number/{serviceName}/terminationDate
→ { effectiveDate: "2025-01-31" }
```
