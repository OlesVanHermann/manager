## 8. APIS & ENDPOINTS

### 8.1 Lancer un diagnostic

```
POST /telephony/{billingAccount}/line/{serviceName}/diagnostic/run
```

Response:
```json
{
  "diagnosticId": "diag-12345",
  "status": "running",
  "startedAt": "2026-01-02T14:30:00Z"
}
```

### 8.2 Obtenir l'historique des diagnostics

```
GET /telephony/{billingAccount}/line/{serviceName}/diagnostic/history
```

Response:
```json
{
  "diagnostics": [
    {
      "id": "diag-12345",
      "date": "2026-01-02T14:30:00Z",
      "status": "completed",
      "result": "ok"
    }
  ]
}
```

### 8.3 Créer un ticket de support

Redirection vers:
```
https://www.ovh.com/manager/dedicated/#/support/tickets/new
```

### 8.4 Voir les tickets

Redirection vers:
```
https://www.ovh.com/manager/dedicated/#/support/tickets
```
