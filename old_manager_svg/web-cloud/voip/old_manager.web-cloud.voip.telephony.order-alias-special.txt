# Cahier des Charges : telecom/telephony - Commander numero special (SVA)

## 1. RESUME EXECUTIF

### Perimetre
Formulaire de commande d'un numero special a valeur ajoutee (08xx).

### Objectif fonctionnel
- Commander un numero SVA (08)
- Choisir la tranche tarifaire
- Renseigner les informations legales SVA
- Declarer le type de service

### Titre

|---------|-------------|

### Carte Configuration
| Champ | Type | Options/Description |
|-------|------|---------------------|
| Nombre de numeros | Select | 1 numero, 2 numeros, ... |
| Typologie d'usage | Select | Service client / SAV, Hotline technique, Jeux / Concours, Information, Autre |

| Code | Description | Tarif appelant |
|------|-------------|----------------|
| 0800 | Numero vert | Gratuit |
| 0805 | Numero vert | Gratuit |

| 0809 | Tarif local | Prix appel local |
| 0810 | Numero azur | ~0,06 EUR/appel |
| 0811 | Numero azur | ~0,06 EUR/min |
| 0820 | Numero indigo | ~0,12 EUR/min |
| 0825 | Numero indigo | ~0,15 EUR/min |
| 0890 | Numero majoré | ~0,80 EUR/min |
| 0891 | Numero majoré | ~0,30 EUR/appel |
| 0892 | Numero majoré | ~0,40 EUR/min |
| 0899 | Numero majoré | ~1,35 EUR/appel + 0,34 EUR/min |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/number/sva/ranges
    Response: [{ range, description, callerCost, hostCost }]

GET /telephony/{billingAccount}/number/sva/typologies
    Response: [{ code, label }]

POST /telephony/{billingAccount}/number/sva/order
    Body: {
      quantity: 1,
      range: "0806",
      typology: "sav",
      company: {
        name: "Ma Societe SARL",
        siret: "12345678901234",
        ape: "6201Z",
        address: "...",
        serviceDescription: "..."
      },
      acceptSvaCgv: true
    }
```

## 6. REGLES METIER

- SIRET obligatoire (entreprises uniquement)
- Declaration ARCEP obligatoire
- Description du service = affichage annuaire
- Revenus reverses selon tranche
- Tarification particuliere selon appels
