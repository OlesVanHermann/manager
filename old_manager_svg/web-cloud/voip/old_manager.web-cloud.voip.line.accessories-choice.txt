# Cahier des Charges : telecom/line - Choix accessoires

## 1. RESUME EXECUTIF

### Perimetre
Page de selection des accessoires a commander.

### Objectif fonctionnel
- Parcourir le catalogue accessoires
- Filtrer par marque et prix
- Selectionner les quantites

## 2. LAYOUT

### Structure page avec filtres + grille de cartes accessoires

## 3. COMPOSANTS UI

### Header page
| Element | Description |
|---------|-------------|
| Retour | ← Retour au telephone |
| Titre | Commander des accessoires |
| Section | Choisir vos accessoires |

### Info box
| Element | Description |
|---------|-------------|
| Fond | Bleu clair |
| Texte | Maximum 5 exemplaires par accessoire |

### Onglets marques
| Onglet | Etat |
|--------|------|
| Cisco | Active (bleu) |
| Yealink | Inactif |
| Polycom | Inactif |
| Jabra | Inactif |

### Filtres dropdown
| Filtre | Options |
|--------|---------|
| Prix | Croissant/Decroissant |
| Compatibilite | Par modele telephone |

### Grille accessoires (4 colonnes)
| Element | Description |
|---------|-------------|
| Image | Zone grise emoji |
| Nom | Nom accessoire |
| Prix | XX,XX € HT (vert) |
| Lien | Voir les details |
| Stepper | - / quantite / + |

### Carte accessoire
| Element | Description |
|---------|-------------|
| Zone image | 140px hauteur |
| Zone infos | Nom + prix + lien |
| Zone quantite | Stepper centré |

### Stepper quantite
| Element | Description |
|---------|-------------|
| Bouton - | Decrementer |
| Valeur | 0-5 |
| Bouton + | Incrementer |

### Bouton continuer
| Element | Description |
|---------|-------------|
| Type | Primary |
| Texte | Continuer |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone/accessories
GET /telephony/{billingAccount}/line/{serviceName}/phone/accessories?brand={brand}
```

### Modele
```typescript
interface Accessory {
  id: string;
  name: string;
  brand: string;
  priceHT: number;
  description: string;
  compatible: string[];
  image?: string;
}

interface CartItem {
  accessoryId: string;
  quantity: number;
}
```
