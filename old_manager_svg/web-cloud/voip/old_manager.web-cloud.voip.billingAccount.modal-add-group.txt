## 8. APIS & ENDPOINTS

### 8.1 Commander un nouveau groupe

```
POST /telephony/order
```

Body:
```json
{
  "acceptedContracts": ["cgv-telephony"]
}
```

Response:
```json
{
  "orderId": "order-12345",
  "billingAccount": "ab12345-ovh-4",
  "status": "pending"
}
```

### 8.2 Obtenir les contrats à accepter

```
GET /telephony/contracts
```

Response:
```json
{
  "contracts": [
    {
      "id": "cgv-telephony",
      "name": "Conditions Générales de Vente OVHcloud",
      "url": "https://www.ovh.com/fr/support/documents_legaux/",
      "mandatory": true
    }
  ]
}
```
