# CAHIER DES CHARGES - old_manager.telecom.telephony.alias-dashboard
# Dashboard d'un numero alias

## 1. RESUME EXECUTIF

### Perimetre
Page d'accueil et vue synthetique d'un numero alias.

### Objectif fonctionnel
Presenter les informations cles et acces rapides pour un numero.
