# Cahier des Charges : telecom/telephony - Modal ajouter une file d'attente

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page files d'attente (EasyHunting)
- **Objectif** : Créer une nouvelle file d'attente téléphonique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom, temps, appelants max
- **OdsSelect** : Stratégie, Musique
- **OdsCheckbox** : Annonce position
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une file d'attente" |
| Bouton fermer | × |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Valeur défaut |
|-------|------|--------|---------------|
| Nom de la file | Input | Oui * | - |
| Stratégie de distribution | Select | Oui * | Séquentielle |
| Temps d'attente max (sec) | Input number | Non | 300 |
| Appelants max en attente | Input number | Non | 10 |
| Annonce position | Checkbox | Non | Activé |
| Musique d'attente | Select | Non | Par défaut |

## 6. OPTIONS STRATÉGIE
| Option | Description |
|--------|-------------|
| Séquentielle | Appelle les agents dans l'ordre |
| Simultanée | Appelle tous les agents en même temps |
| Round-robin | Alterne entre les agents |
| Moins d'appels | Priorité à l'agent ayant le moins d'appels |

## 7. OPTIONS MUSIQUE
| Option |
|--------|
| Par défaut |
| Aucune |
| Personnalisée |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Crée la file |

## 9. APIS & ENDPOINTS
- POST /telephony/{billingAccount}/easyHunting/{serviceName}/queue - Créer file
- GET /telephony/{billingAccount}/easyHunting/{serviceName}/sound - Sons disponibles
