## 9. APIS & ENDPOINTS

```
# Récupérer le logo actuel
GET /telephony/{billingAccount}/fax/{serviceName}/logo
→ {
  filename: "logo-entreprise.png",
  width: 400,
  height: 80,
  size: 24576,
  updatedAt: "2024-11-15T10:30:00",
  url: "https://..."
}

# Téléverser un nouveau logo
POST /telephony/{billingAccount}/fax/{serviceName}/logo
Content-Type: multipart/form-data
Body: { file: <binary> }
→ { success: true, filename: "nouveau-logo.png" }

# Supprimer le logo
DELETE /telephony/{billingAccount}/fax/{serviceName}/logo

# Télécharger le logo
GET /telephony/{billingAccount}/fax/{serviceName}/logo/download
→ Binary file
```
