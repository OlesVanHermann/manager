## 6. APIS & ENDPOINTS

### 6.1 Lister les services convertibles

```
GET /telephony/{billingAccount}/fax/convertible
```

Response:
```json
{
  "services": [
    {
      "serviceName": "+33972123456",
      "description": "Ecofax perso",
      "canConvert": true
    }
  ]
}
```

### 6.2 Convertir un service

```
POST /telephony/{billingAccount}/fax/{serviceName}/convertToVoicefax
```

Body:
```json
{
  "acceptedContracts": true
}
```

Response:
```json
{
  "orderId": "order-12345",
  "orderUrl": "https://www.ovh.com/order/12345",
  "status": "pending"
}
```
