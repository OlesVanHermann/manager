## 3. FORMULAIRE
--------------------------------------------------------------------------------

| Champ | Type | Validation |
|-------|------|------------|
| Identifiant | Input text | Requis, max 40 car |
| Mot de passe | Input password | Requis, 8 car alphanumeriques |

**Erreurs Identifiant:**
- Requis : "Le champ identifiant est obligatoire."
- Maxlength : "Le champ identifiant ne doit pas depasser les 40 caracteres."

**Erreurs Mot de passe:**
- Requis : "Le champ mot de passe est obligatoire."
- Pattern : "Le champ mot de passe doit etre compose de 8 caracteres alphanumeriques."
