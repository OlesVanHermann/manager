# Cahier des Charges : telecom/line - Calendrier week-end

## 1. RESUME EXECUTIF

### Perimetre
Vue calendrier filtree sur le week-end (samedi/dimanche).

### Objectif fonctionnel
- Visualiser les plages du week-end uniquement
- Basculer entre vue semaine et week-end
- Configurer les horaires specifiques week-end

## 3. COMPOSANTS UI

### Navigation
| Element | Description |
|---------|-------------|
| Bouton precedent | < Precedent |
| Label date | Samedi XX - Dimanche XX Mois |
| Bouton suivant | Suivant > |

### Toggle vue
| Element | Description |
|---------|-------------|
| Bouton Semaine | Vue 7 jours (inactif) |
| Bouton Week-end | Vue 2 jours (actif, bleu) |

### Grille calendrier week-end
| Element | Description |
|---------|-------------|
| Colonne 1 | SAMEDI XX (largeur 50%) |
| Colonne 2 | DIMANCHE XX (largeur 50%) |
| Fond | Rose pale (#FEF2F2) |

### Affichage creneaux
| Element | Exemple |
|---------|---------|
| Creneau actif | Rectangle colore avec label |
| Label creneau | Creneau 2 : 10h - 14h |
| Jour ferme | Rectangle rouge avec "FERME" |
| Sous-texte | Redirection vers repondeur |

### Info box
| Element | Description |
|---------|-------------|
| Type | Information (fond rose) |
| Message | Vue week-end : Affiche uniquement samedi et dimanche |

### Legende
| Couleur | Signification |
|---------|---------------|
| Vert | Horaires d'ouverture |
| Rouge | Horaires de fermeture |
| Bleu | Creneau 1 |
| Jaune | Creneau 2 |
| Rose | Creneau 3 |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/timeCondition?day=saturday,sunday
    Response: [{ id, day, timeFrom, timeTo, slot }]
```

### Filtrage
| Parametre | Valeur |
|-----------|--------|
| day | saturday,sunday |
