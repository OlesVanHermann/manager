# Cahier des Charges : telecom/alias - PABX Text-to-Speech (TTS)

## 1. RESUME EXECUTIF

### Perimetre
Gestion des messages TTS (synthese vocale) du PABX.

### Objectif fonctionnel
- Creer des messages vocaux synthetiques
- Gerer les differentes voix disponibles
- Previsualiser les messages

## 3. COMPOSANTS UI

### Header
| Element | Description |
|---------|-------------|
| Titre | Text-to-Speech |
| Sous-titre | Gerez vos messages vocaux synthetiques |

### Liste TTS
| Colonne | Description |
|---------|-------------|
| Nom | Identifiant du message |
| Texte | Contenu du message (tronque) |
| Voix | Voix utilisee |
| Actions | Menu contextuel |

### Formulaire TTS
| Champ | Type | Description |
|-------|------|-------------|
| Nom | Input text | Identifiant |
| Texte | Textarea | Contenu a synthetiser |
| Voix | Select | Choix de la voix |

### Voix disponibles
| Voix | Langue |
|------|--------|
| Alice | Francais feminin |
| Thomas | Francais masculin |
| Emma | Anglais feminin |
| James | Anglais masculin |

### Bouton previsualisation
| Element | Description |
|---------|-------------|
| Icone | Play |
| Action | Genere et joue le TTS |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Creer | Primary | Enregistre le TTS |
| Annuler | Default | Annule |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/tts
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/tts/{id}
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/tts
DELETE /telephony/{billingAccount}/ovhPabx/{serviceName}/tts/{id}
```

### Modele TTS
```typescript
interface OvhPabxTts {
  id: number;
  text: string;
  voice: string;
}
```

## 6. REGLES METIER

- Texte limite a 500 caracteres
- Previsualisation obligatoire avant creation
- Voix par defaut selon langue du compte
- Suppression avec confirmation
