## 2. CONTEXTE FONCTIONNEL
--------------------------------------------------------------------------------

Permet de recharger automatiquement le compte SMS quand le solde
franchit un certain seuil.

**Conditions:**
- Service avec minimum 2 mois d anciennete
- Moyen de paiement SEPA valide sur le compte OVHcloud
- Prelevement sur compte client OVHcloud

## 4. AFFICHAGE CONFIGURATION
--------------------------------------------------------------------------------

| Champ | Format |
|-------|--------|
| Seuil minimum | XX.XX credits |
| Quantite a recharger | XX credits (prix) OU "Ne pas recharger" |

--------------------------------------------------------------------------------
## 5. MODAL CONFIGURATION
--------------------------------------------------------------------------------

### Titre : Gerer la recharge automatique

| Champ | Type | Validation |
|-------|------|------------|
| Seuil minimum | Input number | Requis, min 0 |
| Quantite a recharger | Select | Options predefinies (100, 500, 1000...) |

**Boutons:**
- Valider
- Annuler

--------------------------------------------------------------------------------
## 6. APIS UTILISEES
--------------------------------------------------------------------------------

| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName} | Recuperer config actuelle |
| PUT | /sms/{serviceName} | Mettre a jour config |
| GET | /sms/{serviceName}/seeOffers | Offres credits disponibles |

**Champs API:**
- creditThresholdForAutomaticRecredit : Seuil minimum
- automaticRecreditAmount : Quantite a recharger

--------------------------------------------------------------------------------
