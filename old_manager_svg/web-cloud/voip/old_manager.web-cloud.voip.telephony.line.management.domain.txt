# Cahier des Charges : telecom/telephony - Domaine SIP

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/management/domain
- **Objectif** : Configurer le domaine SIP de la ligne et du compte client

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne
- **OdsLink** : Retour tableau de bord
- **OdsCard** : Domaine ligne + Domaines compte
- **OdsMessage** : Info domaine
- **OdsSelect** : Choix domaine
- **OdsButton** : Enregistrer, Annuler

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour au tableau de bord de la ligne" |
| Titre | "Domaine SIP" |

## 8. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/line/{serviceName} - Infos ligne
- PUT /telephony/{billingAccount}/line/{serviceName} - Modifier domaine ligne
- GET /telephony/domains - Liste domaines disponibles
- PUT /telephony/domains/{country} - Modifier domaine par pays
