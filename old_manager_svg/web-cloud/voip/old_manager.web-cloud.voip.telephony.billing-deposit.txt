# Cahier des Charges : telecom/telephony - Depot de garantie

## 1. RESUME EXECUTIF

### Perimetre
Gestion du depot de garantie du groupe VoIP.

### Objectif fonctionnel
- Voir le depot actuel
- Commander un depot supplementaire
- Accepter les conditions

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}
    Response: { securityDeposit }

POST /telephony/{billingAccount}/securityDeposit
    Body: { amount }
```

### Montants disponibles
| Montant | Description |
|---------|-------------|
| 50 | 50,00 euros HT |
| 100 | 100,00 euros HT |
| 200 | 200,00 euros HT |
