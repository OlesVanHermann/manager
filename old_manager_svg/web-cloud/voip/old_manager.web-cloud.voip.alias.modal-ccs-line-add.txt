# Cahier des Charges : telecom/alias - Modal Ajouter une ligne CCS

## 1. RESUME EXECUTIF

### Perimetre
Modal d'ajout de ligne au Contact Center Solution.

### Objectif fonctionnel
- Ajouter des lignes au CCS
- Configurer les parametres
- Informer des frais eventuels

---

## 2. LAYOUT

### Structure modal avec formulaire multi-lignes

## 3. COMPOSANTS UI

### Header modal
| Element | Description |
|---------|-------------|
| Lien | ← Ajouter une ligne |
| Titre | Implicite |

### Champ numeros
| Element | Description |
|---------|-------------|
| Label | Numero |
| Info | Ajoutez les numeros |
| Input | Champ numero |
| Rechercher | Bouton |
| Supprimer | Bouton |

### Bouton ajout
| Element | Description |
|---------|-------------|
| Texte | + Ajouter une ligne |
| Action | Nouveau champ |

### Timeout
| Element | Description |
|---------|-------------|
| Label | Delai de sonnerie (timeout) |
| Help | Temps avant ligne suivante |
| Input | Number avec fleches |
| Defaut | 20 secondes |

### Wrap-up
| Element | Description |
|---------|-------------|
| Label | Temps de repos (wrap-up) |
| Help | Delai avant nouvel appel |
| Input | Number avec fleches |
| Defaut | 5 secondes |

### Appels simultanes
| Element | Description |
|---------|-------------|
| Label | Appels simultanes |
| Input | Number avec fleches |
| Defaut | 1 |

### Informations tarif
| Element | Description |
|---------|-------------|
| Ligne 1 | Ligne externe = frais |
| Ligne 2 | Meme groupe facturation |
| Lien | Grille tarifaire → |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Ajouter | Primary | Valide |
| Annuler | Default | Ferme |

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/agent
```

### Parametres
```typescript
interface AddLineParams {
  number: string;
  timeout: number;
  wrapUpTime: number;
  simultaneousLines: number;
}
```
