## 7. APIS & ENDPOINTS

```
# Résumé global
GET /telephony/{billingAccount}/statistics/summary
Params: { from, to }
→ {
  totalCalls: 1247,
  incomingCalls: 523,
  outgoingCalls: 689,
  transferCalls: 35
}

# Détails par service
GET /telephony/{billingAccount}/statistics/byService
Params: { from, to, page, pageSize }
→ [{
  serviceName: "...",
  serviceNumber: "+33172100001",
  description: "Ligne principale",
  totalCalls: 456,
  incomingCalls: 198,
  outgoingCalls: 245,
  transferCalls: 13
}]

# Export
GET /telephony/{billingAccount}/statistics/export
Params: { from, to, format: "csv" }
```
