## 7. APIS & ENDPOINTS

```
# Obtenir les informations du seuil
GET /telephony/{billingAccount}/credit
→ {
  currentThreshold: 150.00,
  allowedOutOfPlan: 50.00,
  currentOutOfPlan: 12.50,
  currency: "EUR"
}

# Obtenir les seuils disponibles
GET /telephony/{billingAccount}/credit/thresholds
→ [50, 100, 150, 200, 300, 500, 1000]

# Modifier le seuil
PUT /telephony/{billingAccount}/credit
Body: {
  threshold: 200.00,
  acceptTerms: true
}

# Obtenir les CGV
GET /telephony/contracts
→ [{ name, url }]
```
