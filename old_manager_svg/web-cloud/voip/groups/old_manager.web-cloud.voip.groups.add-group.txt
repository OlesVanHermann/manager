| CGV | Oui |
| Conditions particulières VoIP | Oui |

## 6. APIS & ENDPOINTS

```
# Obtenir les contrats à signer
GET /telephony/contracts
→ [{
  name: "Conditions générales de vente",
  url: "https://...",
  required: true
}, {
  name: "Conditions particulières du service",
  url: "https://...",
  required: true
}]

# Créer un nouveau groupe
POST /telephony
Body: {
  acceptCgv: true,
  acceptParticular: true
}
→ { billingAccount: "ab12345-ovh-2", orderId: "..." }

# Suivre la commande
GET /order/{orderId}
→ { status, deliveryDate }
```
