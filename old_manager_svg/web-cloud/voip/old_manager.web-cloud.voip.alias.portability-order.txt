## 7. APIS & ENDPOINTS

```
# Créer une demande de portabilité
POST /telephony/{billingAccount}/portability
Body: {
  type: "isolated",
  number: "+33123456789",
  rio: "ABC123456789",
  desiredDate: "2025-01-15",
  country: "fr"
}

# Valider le RIO
POST /telephony/portability/validateRio
Body: { rio: "ABC123456789", number: "+33123456789" }

# Opérateurs disponibles
GET /telephony/portability/operators/{country}
→ Liste des opérateurs par pays

# Dates disponibles
GET /telephony/portability/availableDates/{country}
→ Prochaines dates de portabilité possibles
```
