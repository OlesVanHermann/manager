# Cahier des Charges : telecom/line - Selection type redirection

## 1. RESUME EXECUTIF

### Perimetre
Ecran de selection du type de redirection pour un creneau.

### Objectif fonctionnel
- Afficher les 3 types de redirection
- Permettre la selection
- Retour automatique apres selection

---

## 2. LAYOUT

### Modal 480x700 avec options radio

---

## 3. COMPOSANTS UI

### En-tete modal
| Element | Description |
|---------|-------------|
| Bouton retour | < (fleche gauche) |
| Titre | Rediriger l'appel vers |

### Options radio
| Option | Description | Etat |
|--------|-------------|------|
| Un repondeur | Renvoyer vers le repondeur sans faire sonner | Non selectionnee |
| Un numero externe | Renvoyer vers un numero hors OVHcloud | Selectionnee |
| Une ligne OVHcloud | Renvoyer vers une ligne/numero OVHcloud | Non selectionnee |

### Indicateur selection
| Element | Style |
|---------|-------|
| Ligne selectionnee | Fond bleu pale (#EFF6FF) |
| Badge | Vert "Selectionne" |

### Section informations
| Element | Description |
|---------|-------------|
| Titre | INFORMATIONS |
| Liste | 3 points explicatifs |

### Liste informations
| Point | Description |
|-------|-------------|
| 1 | Le repondeur permet de laisser un message vocal |
| 2 | Un numero externe peut etre n'importe quel numero hors OVHcloud |
| 3 | Une ligne OVHcloud fait partie de votre parc telephonique |

### Resume selection
| Element | Description |
|---------|-------------|
| Type | Info box (fond bleu) |
| Label | Selection actuelle : |
| Valeur | Un numero externe |

### Note pied
| Element | Description |
|---------|-------------|
| Texte | Cliquez sur une option pour la selectionner et revenir a l'ecran precedent |

---

## 4. APIS & DONNEES

### Types disponibles
| Type | Valeur API |
|------|------------|
| Repondeur | voicemail |
| Numero externe | external |
| Ligne OVHcloud | ovhLine |
