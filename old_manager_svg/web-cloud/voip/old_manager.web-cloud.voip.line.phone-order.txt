# Cahier des Charges : telecom/line - Commander un telephone

## 1. RESUME EXECUTIF

### Perimetre
Commande d'un telephone IP Plug & Phone avec systeme de caution.

### Objectif fonctionnel
- Choisir un modele de telephone parmi le catalogue
- Visualiser les caracteristiques et prix
- Commander avec versement de caution

---

## 2. LAYOUT

### 2 sections verticales

| Haut | Selection telephone (grille de cartes) |
| Bas | Informations sur la procedure |
