# Cahier des Charges : telecom/alias - Modal Niveau sonore participant

## 1. RESUME EXECUTIF

### Perimetre
Modal de gestion du niveau sonore d'un participant conference.

### Objectif fonctionnel
- Ajuster le niveau sonore (0-4)
- Gerer le mute/unmute
- Exclure le participant
- Rendre sourd (ne peut plus entendre)

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/conference/{serviceName}/participants/{participantId}/kick
POST /telephony/{billingAccount}/conference/{serviceName}/participants/{participantId}/deaf
POST /telephony/{billingAccount}/conference/{serviceName}/participants/{participantId}/mute
PUT /telephony/{billingAccount}/conference/{serviceName}/participants/{participantId}/energy
```

### Parametres
```typescript
interface ParticipantAction {
  participantId: number;
  energy?: number; // 0-4
}
```

## 6. REGLES METIER

- Niveau 0: minimum (presque mute)
- Niveau 4: maximum
- Deaf = n'entend plus mais peut parler
- Mute = ne peut plus parler mais entend
- Kick = deconnexion immediate
