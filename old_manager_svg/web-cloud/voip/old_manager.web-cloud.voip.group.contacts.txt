## 9. APIS & ENDPOINTS

```
# Liste des contacts du groupe
GET /telephony/{billingAccount}/serviceInfos
→ { admin, tech, billing }

# Liste des contacts par service
GET /telephony/{billingAccount}/service/{serviceName}/serviceInfos
→ { admin, tech, billing }

# Modifier le contact technique
PUT /telephony/{billingAccount}/service/{serviceName}/serviceInfos
Body: { tech: "new-nic-handle" }

# Vérifier un NIC handle
GET /me/contact/{nicHandle}
→ { ... } ou 404
```
