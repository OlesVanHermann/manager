## 1. RÉSUMÉ EXÉCUTIF

### Périmètre
Page dashboard d'un compte SMS individuel avec informations et navigation.

### Objectif fonctionnel
Afficher les informations générales d'un compte SMS et permettre l'accès aux différentes fonctionnalités.

### Note
Le code source du dashboard n'est pas fourni. Spécification basée sur les références dans le routing.

| Propriété | Valeur |
|-----------|--------|
| State | sms.service.dashboard |

### Paramètres
| Param | Type | Description |
|-------|------|-------------|
| serviceName | string | Identifiant du compte SMS |
