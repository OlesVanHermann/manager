# Cahier des Charges : telecom/telephony - Mise en attente / Intercom

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/calls/callWaiting
- **Objectif** : Configurer le signal d'appel et le mode intercom

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne / Appels
- **OdsLink** : Retour aux appels
- **OdsCard** : Container principal
- **OdsMessage** : Infos MGCP et Intercom
- **OdsToggle** : Activation signal d'appel
- **OdsCheckbox** : Activation intercom
- **OdsButton** : Enregistrer, Annuler

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour aux appels" |
| Titre | "Mise en attente / Intercom" |

| Propriété | Valeur |
|-----------|--------|
| Titre | "Signal d'appel (MGCP)" |

### Message Info
| Type | Contenu |
|------|---------|

| Label | État |
|-------|------|
