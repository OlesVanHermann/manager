# Cahier des Charges : telecom/telephony - Reversements SVA

## 1. RESUME EXECUTIF

### Perimetre
Page d'historique des reversements SVA (numeros surtaxes).

### Objectif fonctionnel
- Consulter l'historique des appels SVA
- Suivre les montants reverses
- Filtrer par periode et statut

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/svaWallet/repayments
GET /telephony/{billingAccount}/svaWallet/repayments/summary
```

### Modele Repayment
```typescript
interface SvaRepayment {
  id: string;
  calledNumber: string;
  callingNumber: string;
  date: string;
  duration: number;
  amount: number;
  status: 'paid' | 'pending';
}

interface RepaymentSummary {
  pendingTotal: number;
  pendingCount: number;
  paidThisMonth: number;
  paidCount: number;
  nextPaymentDate: string;
}
```
