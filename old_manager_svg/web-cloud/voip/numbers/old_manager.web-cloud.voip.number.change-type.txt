# Cahier des Charges : web-cloud/voip/number - Change Type

## 1. VUE D'ENSEMBLE

**Type:** Page sélection avec cartes radio
**Fonction:** Changer le type de configuration d'un numéro
**Contexte:** Conversion entre Redirection, CCS, Cloud IVR, Conférence

## 2. NAVIGATION

### Breadcrumb

## 7. TYPES DE CONFIGURATION

| Type | Code API | Description |
|------|----------|-------------|
| Redirection | redirect | Renvoi simple vers une ligne |
| Contact Center | easyHunting | Centre d'appels avec files et agents |
| Serveur vocal | cloudIvr | Menu interactif avec touches DTMF |
| Conférence | conference | Salle de conférence audio |

## 8. LIEN ACTION GROUPÉE

- Texte: "Appliquer à plusieurs numéros..."
- Position: sous les cartes
- Fonction: Ouvre une sélection multi-numéros

## 9. BOUTON VALIDATION (200x44px)

- Texte: "Appliquer la configuration"
- Style: btn-primary

## 10. APIS & ENDPOINTS

```
# Informations du numéro actuel
GET /telephony/{billingAccount}/number/{serviceName}
→ { featureType: "easyHunting", ... }

# Types disponibles
GET /telephony/{billingAccount}/number/{serviceName}/availableFeatureTypes
→ ["redirect", "easyHunting", "cloudIvr", "conference"]

# Changer le type
POST /telephony/{billingAccount}/number/{serviceName}/changeFeatureType
Body: { featureType: "conference" }

# Changer le type pour plusieurs numéros (bulk)
POST /telephony/{billingAccount}/number/changeFeatureType
Body: {
  serviceNames: ["+33123456789", "+33123456790"],
  featureType: "redirect"
}
```
