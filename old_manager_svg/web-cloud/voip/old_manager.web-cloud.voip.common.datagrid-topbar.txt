## 1. RESUME EXECUTIF

Composant ODS pour barre d'outils au-dessus d'un tableau datagrid.
Contient actions globales, menus, filtres.

--------------------------------------------------------------------------------
## 2. UTILISATION
--------------------------------------------------------------------------------

<oui-datagrid data-rows="$ctrl.items">
    <oui-datagrid-topbar>
        <oui-action-menu text="Actions">
            <oui-action-menu-item data-on-click="$ctrl.add()">Ajouter</oui-action-menu-item>
        </oui-action-menu>
        <oui-button data-on-click="$ctrl.export()">Exporter</oui-button>
    </oui-datagrid-topbar>
    <oui-datagrid-column ...></oui-datagrid-column>
</oui-datagrid>

--------------------------------------------------------------------------------
## 3. POSITION
--------------------------------------------------------------------------------

Toujours en premier enfant de oui-datagrid, avant les colonnes.

--------------------------------------------------------------------------------
## 4. CONTENU TYPIQUE
--------------------------------------------------------------------------------

- oui-action-menu : Menu deroulant actions
- oui-button : Boutons action
- Texte informatif : Compteurs, statuts
- oui-search : Recherche additionnelle
