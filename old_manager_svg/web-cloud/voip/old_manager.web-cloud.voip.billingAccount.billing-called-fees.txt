## 7. APIS & ENDPOINTS

### 7.1 Obtenir les coûts SVA

```
GET /telephony/{billingAccount}/historyCalled?type=sva
```

Response:
```json
{
  "total": {
    "amount": 127.50,
    "calls": 45
  },
  "details": [
    {
      "number": "+33899123456",
      "operators": [
        {"name": "Orange", "calls": 12, "seconds": 180, "amount": 45.00},
        {"name": "SFR", "calls": 8, "seconds": 95, "amount": 28.50}
      ],
      "total": {"calls": 20, "seconds": 275, "amount": 73.50}
    }
  ]
}
```
