## 1. IDENTIFICATION
- Composant: VoipTimeConditionSlot
- Type: Factory AngularJS
- Fichier source: slot.factory.js
- Usage: Représente un créneau horaire de redirection

## 2. DESCRIPTION FONCTIONNELLE
Un slot représente une destination de redirection pour les appels.
Chaque TimeCondition peut avoir jusqu'à 5 slots :
- available : Horaires d'ouverture (appels normaux)
- slot1, slot2, slot3 : Créneaux personnalisés
- unavailable : Horaires de fermeture

## 3. PROPRIÉTÉS
| Propriété | Type | Obligatoire | Description |
|-----------|------|-------------|-------------|
| name | string | Oui | Nom du slot (available, slot1-3, unavailable) |
| serviceName | string | Oui | Nom du service téléphonique |
| type | string | Non | Type de redirection: 'number' ou 'voicemail' |
| number | string | Non | Numéro de destination |
| inEdition | boolean | - | État d'édition actuel |
| saveForEdition | object | - | Sauvegarde pour annulation |
| originalSave | object | - | Sauvegarde originale |

## 4. MÉTHODES
| Méthode | Retour | Description |
|---------|--------|-------------|
| setOptions(opts) | self | Configure type et number |
| startEdition() | self | Démarre l'édition, sauvegarde état |
| stopEdition(cancel, cancelToOriginal, reset) | self | Termine l'édition |
| hasChange(property, fromOriginal) | boolean | Détecte les changements |
