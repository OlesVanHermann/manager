# Cahier des Charges : telecom/sms - Modal modèles notification utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Page options SMS
- **Objectif** : Configurer les modèles de notification personnalisés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 680px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Info variables disponibles
- **OdsCheckbox** : Activation modes
- **OdsInput** : Champs email et SMS
- **OdsTextarea** : Corps des messages
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modèles de notification utilisateur" |

## 5. VARIABLES DISPONIBLES
| Variable | Description |
|----------|-------------|
| {user} | Identifiant utilisateur |
| {credits} | Crédits restants |
| {quota} | Quota utilisé |
| {date} | Date de notification |
| {service} | Nom du service SMS |

## 6. MODE E-MAIL PERSONNALISÉ
| Champ | Type | Requis |
|-------|------|--------|
| Checkbox activation | Toggle | - |
| E-mail expéditeur | Input email | Oui * |
| Objet de l'e-mail | Input | Oui * |
| Corps de l'e-mail | Textarea | Oui * |

## 7. MODE SMS PERSONNALISÉ
| Champ | Type | Requis |
|-------|------|--------|
| Checkbox activation | Toggle | - |
| Corps du SMS | Textarea | Oui * |
| Compteur caractères | Text | "X / 160" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde configuration |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/users/{userId}/templates - Templates actuels
- PUT /sms/{serviceName}/users/{userId}/templates - Modifier templates
