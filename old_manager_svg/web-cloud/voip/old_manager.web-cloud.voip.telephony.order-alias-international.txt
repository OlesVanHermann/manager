# Cahier des Charges : telecom/telephony - Commander numero international

## 1. RESUME EXECUTIF

### Perimetre
Formulaire de commande d'un numero international.

### Objectif fonctionnel
- Commander un numero dans un pays etranger
- Choisir le pays et la zone geographique
- Fournir une adresse locale valide
- Valider et passer commande

## 4. APIS & DONNEES

### Pays disponibles (exemples)
| Pays | Code | Prefixe |
|------|------|---------|
| Belgique | BE | +32 |
| Suisse | CH | +41 |
| Allemagne | DE | +49 |
| Espagne | ES | +34 |
| Royaume-Uni | GB | +44 |
| Italie | IT | +39 |
| Pays-Bas | NL | +31 |
| Portugal | PT | +351 |

### Endpoints
```
GET /telephony/number/international/countries
    Response: [{ country, code, prefix, available }]

GET /telephony/number/international/{country}/zones
    Response: [{ zone, city, prefix }]

GET /telephony/number/international/prices
    Query: { country: "BE" }
    Response: { monthly, setup }

POST /telephony/{billingAccount}/number/international/order
    Body: {
      quantity: 1,
      country: "BE",
      zone: "brussels",
      address: {
        name: "Ma Societe",
        street: "Rue de la Loi 42",
        zipCode: "1000",
        city: "Bruxelles",
        country: "BE"
      },
      acceptRetractation: true
    }
```

## 6. REGLES METIER

- Adresse DOIT etre dans le pays du numero
- Verification d'adresse selon reglementation locale
- Tarification variable selon pays
- Certains pays necessitent justificatifs
- CGV et retractation obligatoires
