## 6. APIS & ENDPOINTS

### 6.1 Supprimer une campagne

```
DELETE /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}
```

Response:
```json
{
  "status": "deleted",
  "campaignId": "camp-12345"
}
```

---

## 7. COMPORTEMENT

### 7.1 Actions

| Action | Résultat |
|--------|----------|
| Clic Annuler | Fermer la modal |
| Clic × | Fermer la modal |
| Clic Supprimer | Supprimer la campagne + Toast succès |

### 7.2 Post-suppression

| Action | Description |
|--------|-------------|
| Rafraîchir liste | Retirer la campagne de la liste |
| Toast | "Campagne supprimée avec succès" |
