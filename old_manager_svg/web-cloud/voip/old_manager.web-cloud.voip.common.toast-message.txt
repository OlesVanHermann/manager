================================================================================
CAHIER DES CHARGES - old_manager.telecom.common.toast-message
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/common
Composant  : Messages toast (notifications)
Type       : Composant notification
Framework  : AngularJS 1.7.x
Module     : telecom-universe-components

2. DESCRIPTION FONCTIONNELLE
----------------------------
Composant affichant des messages de notification (toasts) en haut de page.
Supporte differents types : succes, erreur, warning, info.
Les messages peuvent etre fermes manuellement.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| [SUCCESS] Operation effectuee avec succes !              [x]     |
+------------------------------------------------------------------+

+------------------------------------------------------------------+
| [ERROR] Une erreur est survenue lors de l operation.     [x]     |
+------------------------------------------------------------------+

+------------------------------------------------------------------+
| [WARNING] Attention : cette action est irreversible.     [x]     |
+------------------------------------------------------------------+

+------------------------------------------------------------------+
| [INFO] Information : nouveau service disponible.         [x]     |
+------------------------------------------------------------------+

4. TYPES DE MESSAGES
--------------------
| Type    | Classe CSS      | Usage                           |
|---------|-----------------|--------------------------------|
| success | alert-success   | Confirmation action reussie     |
| error   | alert-danger    | Erreur survenue                 |
| warning | alert-warning   | Avertissement important         |
| info    | alert-info      | Information generale            |

5. ATTRIBUTS SERVICE TOAST
--------------------------
| Methode              | Description                         |
|----------------------|-------------------------------------|
| toast.success(msg)   | Affiche message succes              |
| toast.error(msg)     | Affiche message erreur              |
| toast.warning(msg)   | Affiche message warning             |
| toast.info(msg)      | Affiche message info                |
| toast.clearMessage() | Supprime un message                 |
| toast.getAllMessages()| Recupere tous les messages         |

6. STRUCTURE MESSAGE
--------------------
{
  type: "success|error|warning|info",
  content: "<p>Contenu HTML du message</p>"
}

7. COMPOSANTS UTILISES
----------------------
- oui-icon-close : Bouton fermer
- tuc-toast-message-scroller : Scroll vers nouveau message

8. TRADUCTIONS FR
-----------------
toaster_close: "Fermer"

9. STYLES
---------
- alert alert-xs : Style alerte compact
- alert-danger : Rouge pour erreurs
- alert-warning : Orange pour warnings
- alert-info : Bleu pour infos
- alert-success : Vert pour succes

10. ACCESSIBILITE
-----------------
- role="alert" pour lecteurs d ecran
- title sur bouton fermer
- aria-hidden sur scroller decoratif

11. EXEMPLE D UTILISATION
-------------------------
// Dans un controleur
TucToast.success('Fichier envoye avec succes !');
TucToast.error('Erreur lors de l envoi du fichier.');

// Dans le template (une seule fois par page)
<tuc-toast-message></tuc-toast-message>
