# Cahier des Charges : telecom/svaWallet - KYC Documents justificatifs

## 1. RESUME EXECUTIF

### Perimetre
Page de telechargement des documents KYC pour portefeuille SVA.

### Objectif fonctionnel
- Lister les documents requis
- Telecharger les justificatifs
- Suivre le statut de validation
- Remplacer les documents refuses

## 4. APIS & DONNEES

### Documents requis
| Type | Description |
|------|-------------|
| ID_CARD | Piece d'identite (CNI/Passeport) |
| KBIS | Extrait Kbis |
| RIB | Releve d'identite bancaire |
| PROOF_ADDRESS | Justificatif de domicile |

### Endpoints
```
GET /svaWallet/{walletId}/kyc/documents
    Response: [
      { type: "ID_CARD", status: "validated", uploadedAt, validatedAt },
      { type: "KBIS", status: "pending", uploadedAt },
      { type: "RIB", status: "not_provided" },
      { type: "PROOF_ADDRESS", status: "not_provided" }
    ]

POST /svaWallet/{walletId}/kyc/documents
    Body: FormData { type, file }
    Response: { documentId, status: "pending" }

GET /svaWallet/{walletId}/kyc/documents/{documentId}/download
    Response: Binary file

DELETE /svaWallet/{walletId}/kyc/documents/{documentId}
```

### Contraintes fichiers
| Contrainte | Valeur |
|------------|--------|
| Taille max | 5 Mo |
| Formats | PDF, JPG, PNG |
| Resolution min | 300 DPI |
