## 9. APIS & ENDPOINTS

### 9.1 Lister les campagnes

```
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns
```

Response:
```json
{
  "campaigns": [
    {
      "id": "camp-001",
      "name": "Campagne Noël 2025",
      "state": "done",
      "status": "success",
      "createdAt": "2025-12-15T10:00:00Z"
    }
  ],
  "total": 4
}
```

### 9.2 Lancer une campagne

```
POST /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}/start
```

### 9.3 Stopper une campagne

```
POST /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}/stop
```

### 9.4 Supprimer une campagne

```
DELETE /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}
```
