# Cahier des Charges : telecom/sms - Modal annuler envoi différé

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page envois programmés
- **Objectif** : Annuler un envoi SMS différé/programmé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Détails de l'envoi
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Annuler l'envoi" |

## 5. QUESTION CONFIRMATION
| Élément | Texte |
|---------|-------|
| Question | "Êtes-vous sûr de vouloir annuler cet envoi différé ?" |

## 6. DÉTAILS ENVOI
| Champ | Description |
|-------|-------------|
| Nom | Nom de la campagne |
| Date | Date et heure programmées |
| Expéditeur | Sender ID |
| Destinataires | Liste des numéros |

## 7. MESSAGE WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Error (rouge) |
| Icône | ⚠ |
| Texte | "Cette action est irréversible. L'envoi ne pourra pas être rétabli." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Danger (rouge) | Annule l'envoi |

## 9. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/jobs/{jobId} - Annuler envoi programmé
