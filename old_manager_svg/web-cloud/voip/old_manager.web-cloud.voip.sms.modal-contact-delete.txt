# Cahier des Charges : telecom/sms - Modal supprimer contact

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Page carnet d'adresses
- **Objectif** : Supprimer un contact du carnet

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le contact" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir supprimer le contact {nom} ?" |
| Nom contact | En gras dans la question |

## 6. MESSAGE WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Error (rouge) |
| Icône | ⚠ |
| Texte | "Cette action est irréversible. Le contact sera définitivement supprimé du carnet d'adresses." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le contact |

## 8. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/phonebook/{bookId}/contact/{contactId} - Supprimer contact
