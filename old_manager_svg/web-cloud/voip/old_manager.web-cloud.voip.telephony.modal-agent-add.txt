# Cahier des Charges : telecom/telephony - Modal ajouter un agent

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page agents de file d'attente
- **Objectif** : Ajouter un agent à une file d'attente téléphonique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection ligne agent
- **OdsInput** : Paramètres numériques
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un agent" |
| Bouton fermer | × |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Ligne agent | Select | Oui * | Sélectionner une ligne du groupe |
| Temps de sonnerie | Input number | Non | En secondes (défaut: 20) |
| Délai post-appel | Input number | Non | Wrap-up time en secondes (défaut: 5) |
| Appels simultanés | Input number | Non | Nombre max (défaut: 1) |

## 6. OPTIONS LIGNE AGENT
| Source |
|--------|
| Lignes disponibles du groupe de facturation |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Ajoute l'agent |

## 8. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/line - Liste lignes disponibles
- POST /telephony/{billingAccount}/easyHunting/{serviceName}/agent - Ajouter agent
