## 2. CONTENU MODAL

### Titre

--------------------------------------------------------------------------------
## 3. FORMULAIRE
--------------------------------------------------------------------------------

| Champ | Type | Validation |
|-------|------|------------|
| Fichier de contact | File input | Requis, formats supportes |

**Formats acceptes:**
- CSV
- VCF (vCard)
- XLS/XLSX

**Erreur format:**
"Format de fichier non supporte."

--------------------------------------------------------------------------------
## 4. BOUTONS
--------------------------------------------------------------------------------

- Importer (primary)
- Annuler (secondary)

--------------------------------------------------------------------------------
## 5. API
--------------------------------------------------------------------------------

| Methode | Endpoint | Description |
|---------|----------|-------------|
| POST | /sms/{serviceName}/phonebooks/{bookKey}/import | Importer fichier |

**Content-Type:** multipart/form-data

--------------------------------------------------------------------------------
## 6. MESSAGES RETOUR
--------------------------------------------------------------------------------

- Loading : "Importation des contacts en cours..."
- Succes : "Importation des contacts terminee !"
- Erreur : "Impossible d importer vos contacts. [error]"

## 8. EXPORT ASSOCIE

Bouton "Exporter" disponible sur la meme page :
- Telecharge le carnet au format CSV
- Message succes : "Exportation des contacts terminee !"
