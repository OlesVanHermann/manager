# Cahier des Charges : telecom/line - Modal edition contact

## 1. RESUME EXECUTIF

### Perimetre
Modal de modification d'un contact du carnet d'adresses.

### Objectif fonctionnel
- Modifier les informations d'un contact
- Changer le groupe
- Supprimer le contact

## 3. COMPOSANTS UI

### Modal
| Element | Description |
|---------|-------------|
| Titre | Modifier le contact |
| Bouton fermer | X en haut droite |
| Overlay | Fond sombre |

### Champs identite
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Text input | Oui |
| Prenom | Text input | Oui |

### Champ groupe
| Element | Description |
|---------|-------------|
| Input | Text avec valeur actuelle |
| Bouton | Choisir (dropdown) |

### Dropdown groupe
| Option | Description |
|--------|-------------|

|---------|-------------|

## 4. APIS & DONNEES

### Endpoints
```
PUT /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/{id}
DELETE /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/{id}
```

### Modele
```typescript
interface PhonebookContactUpdate {
  surname: string;
  name: string;
  group?: string;
  homePhone?: string;
  homeMobile?: string;
  workPhone?: string;
  workMobile?: string;
}
```
