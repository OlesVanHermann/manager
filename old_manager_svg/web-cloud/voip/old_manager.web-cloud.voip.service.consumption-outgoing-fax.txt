## 7. APIS & ENDPOINTS

### 7.1 Lister les fax émis

```
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns?type=outgoing
```

Response:
```json
{
  "totalCount": 12,
  "totalPages": 38,
  "totalPrice": 4.56,
  "data": [
    {
      "id": "fax-001",
      "date": "2025-12-28T14:32:15Z",
      "destinationNumber": "+33123456789",
      "pages": 2,
      "type": "outgoing",
      "price": 0.38
    }
  ]
}
```

### 7.2 Télécharger un fax

```
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}/document
```
