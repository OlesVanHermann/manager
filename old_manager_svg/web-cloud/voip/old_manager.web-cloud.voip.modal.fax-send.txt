# Cahier des Charges : web-cloud/voip - Modal Fax Send

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire avec upload
**Fonction:** Envoyer un fax depuis une ligne fax virtuelle
**Contexte:** Accessible depuis le dashboard d'une ligne fax

## 2. DIMENSIONS

- Modal: 600x550px

## 3. HEADER MODAL (60px)

- Background: #F9FAFB
- Titre: "Envoyer un fax" (16px, bold)
- Bouton fermer: "x"

## 4. CONTENU

### Champ Destinataire
- Label: "Numero de fax destinataire *"
- Input: 552x40px

### Zone upload document
- Label: "Document a envoyer *"

## 6. APIS & ENDPOINTS

```
# Envoyer un fax
POST /telephony/{billingAccount}/fax/{serviceName}/send
Body: FormData {
  recipient: "+33123456789",
  document: File (PDF),
  quality: "standard",  // ou "high"
  includeCoverPage: false,
  subject: "Objet du fax"
}
→ {
  faxId: "fax-12345",
  status: "sending",
  pageCount: 3
}

# Vérifier statut d'envoi
GET /telephony/{billingAccount}/fax/{serviceName}/send/{faxId}
→ {
  faxId: "fax-12345",
  status: "sent",  // pending, sending, sent, failed
  sentAt: "2024-12-30T15:30:00",
  pageCount: 3,
  duration: 45
}
```
