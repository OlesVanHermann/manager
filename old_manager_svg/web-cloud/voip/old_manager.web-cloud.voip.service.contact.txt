## 8. APIS & ENDPOINTS

### 8.1 Obtenir les coordonnées

```
GET /telephony/{billingAccount}/line/{serviceName}/contact
```

Response:
```json
{
  "legalForm": "individual",
  "lastName": "Dupont",
  "firstName": "Jean",
  "email": "jean.dupont@example.com",
  "address": {
    "country": "FR",
    "zipCode": "75001",
    "city": "Paris",
    "streetNumber": "15",
    "streetNumberExtra": null,
    "streetName": "Rue de Rivoli"
  }
}
```

### 8.2 Modifier les coordonnées

```
PUT /telephony/{billingAccount}/line/{serviceName}/contact
```

Body:
```json
{
  "legalForm": "individual",
  "lastName": "Dupont",
  "firstName": "Jean",
  "email": "jean.dupont@example.com",
  "address": {
    "zipCode": "75001",
    "city": "Paris",
    "streetNumber": "15",
    "streetNumberExtra": "",
    "streetName": "Rue de Rivoli"
  }
}
```
