# Cahier des Charges : telecom/sms - Modal supprimer SMS reçu

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page SMS reçus
- **Objectif** : Supprimer un SMS reçu

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Détails du SMS
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le SMS reçu" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Confirmez-vous la suppression de ce SMS ?" |

## 6. DÉTAILS SMS
| Champ | Format |
|-------|--------|
| Date | DD/MM/YYYY HH:mm:ss |
| Expéditeur | Numéro international |
| Message | Tronqué avec "..." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le SMS |

## 8. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/incoming/{id} - Supprimer SMS reçu
