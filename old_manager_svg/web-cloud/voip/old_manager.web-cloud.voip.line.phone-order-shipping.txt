# Cahier des Charges : telecom/line - Wizard livraison telephone

## 1. RESUME EXECUTIF

### Perimetre
Etape de selection de l'adresse de livraison dans le wizard commande.

### Objectif fonctionnel
- Selectionner le contact de livraison
- Choisir le mode de livraison
- Ajouter un nouveau contact

## 4. APIS & DONNEES

### Endpoints
```
GET /me/contact
POST /me/contact
GET /telephony/mondialRelay?zipCode={code}
```

### Modele
```typescript
interface Contact {
  id: string;
  firstName: string;
  lastName: string;
  address: string;
  zipCode: string;
  city: string;
  country: string;
  phone: string;
}

interface ShippingMode {
  type: 'home' | 'relay';
  delay: string;
  price: number;
}
```
