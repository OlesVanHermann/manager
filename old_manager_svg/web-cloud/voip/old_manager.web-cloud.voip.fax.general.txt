## 10. APIS & ENDPOINTS

```
# Informations fax
GET /telephony/{billingAccount}/fax/{serviceName}
→ {
  serviceName: "0033972999888",
  description: "Fax Bureau",
  offer: "Fax Pro",
  status: "active"
}

# Paramètres actuels
GET /telephony/{billingAccount}/fax/{serviceName}/settings
→ {
  quality: "normal",
  retries: 3,
  emailFormat: "pdf",
  notificationEmail: "contact@entreprise.fr"
}

# Consommation du mois
GET /telephony/{billingAccount}/fax/{serviceName}/consumption/summary
Query: { month: "current" }
→ {
  sentCount: 23,
  receivedCount: 45
}

# Email fax
GET /telephony/{billingAccount}/fax/{serviceName}/emailAddress
→ { email: "0033972999888@fax.ovh.net" }
```
