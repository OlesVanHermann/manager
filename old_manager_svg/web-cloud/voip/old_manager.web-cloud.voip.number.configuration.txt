## 7. FONCTIONNALITÉS CCS

| Fonctionnalité | Code | Description |
|----------------|------|-------------|
| Lignes | hunting/queue/agent | Gestion des agents de la file |
| Sons | sound | Musiques et annonces audio |
| Plages horaires | timeCondition | Horaires d'ouverture/fermeture |
| Filtrage | screenList | Listes blanches et noires |
| Planificateur | scheduler | Jours fériés et exceptions |
| Enregistrements | records | Enregistrement des conversations |

## 8. APIS & ENDPOINTS

```
# Informations du numéro
GET /telephony/{billingAccount}/number/{serviceName}
→ { featureType: "easyHunting", ... }

# Configuration CCS
GET /telephony/{billingAccount}/easyHunting/{serviceName}
→ {
  featureType: "easyHunting",
  description: "...",
  ...
}

# Sous-ressources CCS
GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue
→ [queueId1, queueId2]

GET /telephony/{billingAccount}/easyHunting/{serviceName}/hunting/queue/{queueId}/agent
→ [agentId1, agentId2, ...]

GET /telephony/{billingAccount}/easyHunting/{serviceName}/sound
→ [soundId1, soundId2, ...]

GET /telephony/{billingAccount}/easyHunting/{serviceName}/timeConditions
→ [conditionId1, conditionId2, ...]

GET /telephony/{billingAccount}/easyHunting/{serviceName}/screenListConditions
→ [screenListId1, screenListId2, ...]

GET /telephony/{billingAccount}/easyHunting/{serviceName}/records
→ [recordId1, recordId2, ...]
```
