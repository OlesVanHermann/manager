```
# Statistiques reversements
GET /telephony/{billingAccount}/repayment/statistics
→ {
  currentBalance: 347.82,
  currentMonthRevenue: 89.50,
  total12Months: 1248.60,
  nextPaymentDate: "2025-01-15"
}

# Historique des reversements
GET /telephony/{billingAccount}/repayment/history
Query: { limit, offset }
→ [{
  period: "2024-12",
  number: "+33892123456",
  type: "perCall",
  rate: 0.80,
  calls: 156,
  minutes: 423,

  status: "pending"

→ {
  number: "+33892123456",
  type: "perCall",

  amountHT: 124.80,
  details: [...]
}
```
