# Cahier des Charges : telecom/alias - CCS Configuration des sons

## 1. RESUME EXECUTIF

### Perimetre
Configuration des sons du Contact Center Solution.

### Objectif fonctionnel
- Configurer les annonces d'attente
- Gerer le debordement
- Configurer la fermeture

---

## 2. LAYOUT

### Structure page avec 3 sections sons

---

## 3. COMPOSANTS UI

### Header
| Element | Description |
|---------|-------------|
| Titre | Configuration des sons |
| Sous-titre | Contact Center Solution |

### Section Attente
| Element | Description |
|---------|-------------|
| Titre | Configuration de l'attente |
| Annonce ouverture | Select son |
| Musique attente | Select son |
| Bouton + | Ajouter un son |

### Section Debordement
| Element | Description |
|---------|-------------|
| Titre | Configuration du debordement |
| Max appels | Number input |
| Action | Radio (Messagerie/Son) |
| Ligne messagerie | Select ligne |

### Actions debordement
| Option | Description |
|--------|-------------|
| Messagerie vocale | Vers boite vocale |
| Jouer son | Son puis raccroche |

### Section Fermeture
| Element | Description |
|---------|-------------|
| Titre | Configuration de la fermeture |
| Delai max | Number (secondes) |
| Action | Radio (Messagerie/Son) |
| Son fermeture | Select son |

### Selecteur son
| Element | Description |
|---------|-------------|
| Dropdown | Liste sons disponibles |
| Bouton + | Ajouter nouveau son |
| Preview | Bouton lecture |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Valider | Primary | Sauvegarde |
| Annuler | Default | Annule |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/easyHunting/{serviceName}
PUT /telephony/{billingAccount}/easyHunting/{serviceName}
GET /telephony/{billingAccount}/easyHunting/{serviceName}/sound
POST /telephony/{billingAccount}/easyHunting/{serviceName}/sound
```

### Modele Configuration
```typescript
interface EasyHuntingSounds {
  onHoldSound: number;
  toneOnOpening: number;
  maxWaitTime: number;
  queueOverflowMaxWaitTime: number;
  voicemail: string;
  toneOnClosing: number;
}
```

- Debordement obligatoire si file pleine
