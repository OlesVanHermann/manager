## 4. COMPOSANTS UI

### Creation campagne
- Nom campagne
- Upload PDF
- Liste destinataires (CSV ou saisie)
- Planification date/heure

### Composants
- oui-stepper : Wizard creation
- oui-file : Upload PDF/CSV

  recipientsList: string[],
  sendDate: string
}

POST /telephony/:billingAccount/fax/:serviceName/campaigns/:id/start
POST /telephony/:billingAccount/fax/:serviceName/campaigns/:id/stop
