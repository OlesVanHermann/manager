# Cahier des Charges : telecom/telephony - Renvoi d'appels

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/calls/forward
- **Objectif** : Configurer les règles de renvoi pour les appels entrants

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 750px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne / Appels
- **OdsLink** : Retour aux appels
- **OdsMessage** : Info renvois
- **OdsCard** : Configuration des renvois
- **OdsCheckbox** : Activation des types de renvoi
- **OdsInput** : Délai, Numéro destination
- **OdsSelect** : Type de destination
- **OdsButton** : Enregistrer, Annuler

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour aux appels" |
| Titre | "Renvoi d'appels" |

## 5. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info (bleu) | "Configurez les règles de renvoi pour vos appels entrants. Vous pouvez renvoyer les appels vers un autre numéro de votre groupe ou vers un numéro externe." |

## 6. TYPES DE RENVOI

### Renvoi sur Non-réponse
| Champ | Type | Description |
|-------|------|-------------|
| Checkbox | Activé (✓) | "Renvoi sur non-réponse" |
| Délai | Input number | "Renvoyer après X secondes" (défaut: 20) |
| Type destination | Select | "Ligne du groupe" / "Numéro externe" / "Messagerie vocale" |
| Numéro | Input tel | "+33 9 12 34 56 79" |

### Renvoi sur Occupation
| Champ | Type | Description |
|-------|------|-------------|
| Checkbox | Activé (✓) | "Renvoi sur occupation" |
| Type destination | Select | "Messagerie vocale" |

### Renvoi Inconditionnel
| Champ | Type | Description |
|-------|------|-------------|
| Checkbox | Non activé | "Renvoi inconditionnel" |
| Note | Text | "Tous les appels seront renvoyés immédiatement." |

## 7. OPTIONS DE DESTINATION
| Option |
|--------|
| Ligne du groupe |
| Numéro externe |
| Messagerie vocale |
| Numéro SDA |

## 8. ACTION GROUPÉE
| Lien | Action |
|------|--------|
| "Appliquer à plusieurs lignes..." | Multi-sélection |

## 9. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Enregistrer | Primary | Sauvegarde configuration |
| Annuler | Secondary | Annule modifications |

## 10. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/line/{serviceName}/options - Options actuelles
- PUT /telephony/{billingAccount}/line/{serviceName}/options - Modifier renvois
