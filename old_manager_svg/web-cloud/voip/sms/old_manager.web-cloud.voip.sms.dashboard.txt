## 9. APIS & ENDPOINTS

```
# Informations du service
GET /sms/{serviceName}
→ {
  name: "sms-abc123-1",
  status: "active",
  creditsLeft: 8542,
  accountType: "standard"
}

# Statistiques
GET /sms/{serviceName}/statistics
Query: { period: "today" | "week" | "month" }
→ {
  campaignsCount: 3,
  sentCount: 1245,
  deliveredCount: 1198,
  stoplistedCount: 12
}

# Envoi rapide
POST /sms/{serviceName}/jobs
Body: {
  message: "...",
  receivers: ["+33..."]
}
```
