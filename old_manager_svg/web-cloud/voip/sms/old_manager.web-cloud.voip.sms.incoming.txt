# Cahier des Charges : web-cloud/voip/sms - Incoming

## 1. VUE D'ENSEMBLE

**Type:** Page liste avec datagrid
**Fonction:** Consulter les SMS reçus
**Contexte:** Historique des messages entrants

## 2. NAVIGATION

### Breadcrumb
Télécom > SMS > sms-xxxxxxxx-1 > SMS reçus

### Retour
← Retour aux SMS

## 3. HEADER

- Titre: "SMS reçus"

## 4. BARRE D'ACTIONS

### Actions dropdown (120x36px)
- Texte: "Actions ▼"
- Options: Télécharger l'historique, Actualiser

### Bouton suppression (230x36px)
- État: Désactivé (grisé)
- Texte: "Supprimer le SMS sélectionné"
- Activé quand: sélection d'une ligne

### Bouton filtre (100x36px)
- Position: droite
- Texte: "Filtrer"

## 5. DATAGRID SMS REÇUS (1120x380px)

### Colonnes

| Colonne | Largeur | Description |

## 6. APIS & ENDPOINTS

```
# Liste des SMS reçus
GET /sms/{serviceName}/incoming
Query: { limit, offset }
→ [{
  id: "12345",
  creationDatetime: "2024-12-28T14:32:15",
  sender: "+33612345678",
  message: "Bonjour, bien reçu votre..."
}]

# Détails d'un SMS reçu
GET /sms/{serviceName}/incoming/{id}

# Supprimer un SMS reçu
DELETE /sms/{serviceName}/incoming/{id}

# Télécharger l'historique
GET /sms/{serviceName}/incoming/export
→ CSV file
```
