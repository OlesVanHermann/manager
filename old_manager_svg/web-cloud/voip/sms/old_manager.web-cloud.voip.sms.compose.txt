## 10. APIS & ENDPOINTS

```
# Expéditeurs
GET /sms/{serviceName}/senders
→ [{ sender, status }]

# Solde crédits
GET /sms/{serviceName}
→ { creditsLeft: 245.50 }

# Envoyer un SMS
POST /sms/{serviceName}/jobs
Body: {
  sender: "MonEntreprise",
  message: "Votre message",
  receivers: ["+33601020304"],
  noStopClause: true
}

# Estimation
POST /sms/{serviceName}/estimateCost
Body: { message, receiversCount: 1 }
→ { smsCount: 1, creditCost: 0.065 }
```
