# Cahier des Charges : web-cloud/voip/sms - Blacklist

## 1. VUE D'ENSEMBLE

**Type:** Page liste avec datagrid
**Fonction:** Gérer les destinataires blacklistés (STOP)
**Contexte:** Numéros ayant répondu STOP aux SMS

## 2. NAVIGATION

### Breadcrumb
Télécom > SMS > sms-xxxxxxxx-1 > Options > Destinataires blacklistés

### Retour
← Retour aux options

## 3. HEADER

- Titre: "Gérer les destinataires blacklistés"

## 4. CARTE INFORMATIONS (1120x100px)

### Titre
"Informations"

### Contenu
- Ligne 1: "Liste de vos destinataires qui ne souhaitent plus recevoir de SMS de votre part."
- Ligne 2: "Si vous supprimez un numéro de la liste, la réception de SMS en provenance de vos expéditeurs sera rétablie vers ce destinataire."

## 5. BARRE D'ACTIONS

### Bouton supprimer (disabled)
- Texte: "Supprimer le numéro sélectionné" (260x36px)
- État: désactivé (grisé) si aucune sélection

## 6. DATAGRID BLACKLIST (1120x280px)

### Colonnes

| Colonne | Largeur | Description |
|---------|---------|-------------|
| Checkbox | 40px | Sélection multiple |
| Date | 290px | Date du STOP |
| Numéro | 550px | Numéro blacklisté |
| Actions | 120px | Bouton Supprimer |

### Données exemple

| ☐ | Date | Numéro | Actions |
|---|------|--------|---------|
| ☐ | 15/12/2024 | +33 6 12 34 56 78 | Supprimer |
| ☐ | 10/12/2024 | +33 7 98 76 54 32 | Supprimer |

### Pagination
"Affichage 1-2 sur 2"
Navigation: ‹ [1] ›

## 7. APIS & ENDPOINTS

```
# Liste des blacklistés
GET /sms/{serviceName}/blacklists
→ [{
  number: "+33612345678",
  creationDatetime: "2024-12-15T10:30:00"
}]

# Supprimer de la blacklist
DELETE /sms/{serviceName}/blacklists/{number}

# Supprimer plusieurs (bulk)
POST /sms/{serviceName}/blacklists/delete
Body: { numbers: ["+33612345678", "+33798765432"] }
```
