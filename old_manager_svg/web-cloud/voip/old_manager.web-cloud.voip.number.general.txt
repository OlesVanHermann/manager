## 11. APIS & ENDPOINTS

```
# Informations numéro
GET /telephony/{billingAccount}/number/{serviceName}
→ {
  serviceName: "0033912345680",
  description: "Accueil principal",
  country: "fr",
  countryCode: "+33",
  status: "active",
  featureType: "easyHunting"
}

# Modifier informations
PUT /telephony/{billingAccount}/number/{serviceName}
Body: { description: "Nouveau nom" }

# Configuration actuelle
GET /telephony/{billingAccount}/number/{serviceName}/feature
→ {
  featureType: "easyHunting",
  agentsCount: 5,
  queuesCount: 2
}

# Changer type configuration
PUT /telephony/{billingAccount}/number/{serviceName}/changeFeatureType
Body: { featureType: "redirect" | "easyHunting" | "conference" | "svi" }

# Convertir en ligne
POST /telephony/{billingAccount}/number/{serviceName}/convertToLine

# Résilier numéro
DELETE /telephony/{billingAccount}/number/{serviceName}/terminate
```
