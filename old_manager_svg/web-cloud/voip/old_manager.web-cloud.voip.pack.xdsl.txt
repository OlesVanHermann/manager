## 1. RÉSUMÉ EXÉCUTIF

Page principale d'un accès xDSL dans un pack, servant de conteneur pour les
différents onglets de gestion (informations, modem, accès, tâches).

### Utilisateurs cibles
- Clients disposant d'un accès xDSL/VDSL/SDSL

### Objectif fonctionnel
- Centraliser la navigation vers les sous-pages xDSL
- Afficher le titre et identifiant de l'accès
- Permettre le renommage de l'accès
