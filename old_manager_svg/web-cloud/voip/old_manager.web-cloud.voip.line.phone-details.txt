# Cahier des Charges : telecom/line - Informations telephone

## 1. RESUME EXECUTIF

### Perimetre
Page d'informations detaillees sur le telephone associe.

### Objectif fonctionnel
- Voir les caracteristiques du telephone
- Voir les lignes associees
- Gerer la configuration

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone
```

### Modele
```typescript
interface PhoneDetails {
  model: string;
  brand: string;
  macAddress: string;
  ipAddress: string;
  firmwareVersion: string;
  protocol: string;
  maxLines: number;
  usedLines: number;
  associatedLines: AssociatedLine[];
}

interface AssociatedLine {
  number: string;
  description: string;
  keyNumber: number;
}
```
