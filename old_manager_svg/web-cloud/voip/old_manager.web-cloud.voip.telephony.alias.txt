# CAHIER DES CHARGES - old_manager.telecom.telephony.alias
# Module principal Alias (numeros virtuels)

## 1. RESUME EXECUTIF

### Perimetre
Module de gestion des numeros alias (numeros virtuels sans forfait).

### Objectif fonctionnel
Configurer les numeros pour: redirection, file d'appels, conference, SVI.
