## 8. APIS & ENDPOINTS

### 8.1 Obtenir les informations du groupe

```
GET /telephony/{billingAccount}/lines/count
```

Response:
```json
{
  "linesCount": 5,
  "numbersCount": 12
}
```

### 8.2 Lister les autres groupes

```
GET /telephony
```

Response:
```json
{
  "billingAccounts": [
    {
      "billingAccount": "ab12345-ovh-2",
      "description": "Groupe secondaire"
    },
    {
      "billingAccount": "ab12345-ovh-3",
      "description": "Groupe test"
    }
  ]
}
```

### 8.3 Attacher des services d'un autre groupe

```
POST /telephony/{billingAccount}/attachFromOtherBillingAccount
```

Body:
```json
{
  "sourceBillingAccount": "ab12345-ovh-2",
  "serviceIds": ["line1", "line2", "number1"]
}
```
