# Cahier des Charges : telecom/telephony - Changer d'offre

## 1. VUE D'ENSEMBLE
- **Type** : Page de changement d'offre
- **URL** : /telephony/{billingAccount}/line/{serviceName}/management/offerChange
- **Objectif** : Changer l'offre tarifaire de la ligne VoIP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne
- **OdsLink** : Retour tableau de bord
- **OdsCard** : Informations et changement
- **OdsSelect** : Choix nouvelle offre
- **OdsButton** : Valider, Annuler

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour au tableau de bord de la ligne" |
| Titre | "Changer d'offre" |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Valider | Primary | Confirme le changement |
| Annuler | Secondary | Annule |

## 8. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/line/{serviceName} - Infos ligne
- GET /telephony/{billingAccount}/line/{serviceName}/offers - Offres disponibles
- POST /telephony/{billingAccount}/line/{serviceName}/changeOffer - Changer offre
