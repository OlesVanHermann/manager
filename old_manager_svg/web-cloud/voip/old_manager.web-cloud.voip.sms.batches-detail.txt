================================================================================
CAHIER DES CHARGES - old_manager.telecom.sms.batches-detail
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/sms
Composant  : Detail campagne SMS
Type       : Page detail avec tableau
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Page affichant le detail d une campagne SMS : liste des destinataires
avec leur statut de livraison (delivre, en attente, echec).

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| < Retour                                                         |
| Detail de la campagne: Promo Noel                                |
+------------------------------------------------------------------+
|                                                                  |
| +--------------------------------------------------------------+ |
| | Destinataire           | Delivre                             | |
| |------------------------|-------------------------------------| |
| | +33612345678           | Oui                                 | |
| | +33698765432           | Oui                                 | |
| | +33601020304           | Non                                 | |
| | +33607080910           | En attente                          | |
| +--------------------------------------------------------------+ |
|                                                                  |
| [Telecharger CSV]                                                |
|                                                                  |
+------------------------------------------------------------------+

4. COLONNES TABLEAU
-------------------
| Colonne      | Description                         |
|--------------|-------------------------------------|
| Destinataire | Numero de telephone du destinataire |
| Delivre      | Statut de livraison du SMS          |

5. STATUTS LIVRAISON
--------------------
| Statut     | Description                    |
|------------|--------------------------------|
| Oui        | SMS delivre avec succes        |
| Non        | Echec de livraison             |
| En attente | Livraison en cours             |

6. ETATS
--------
- Chargement : Spinner + "Telechargement en cours..."
- Affichage : Tableau des destinataires
- Export : Generation fichier CSV

7. ENDPOINTS API
----------------
| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName}/batches/{id} | Details campagne |
| GET | /sms/{serviceName}/batches/{id}/statistics | Statistiques |

8. COMPOSANTS UTILISES
----------------------
- oui-datagrid : Tableau de donnees
- oui-spinner : Chargement
- oui-button : Export CSV

9. TRADUCTIONS FR
-----------------
sms_batches_details_title: "Detail de la campagne: {{name}}"
sms_batches_details_receiver: "Destinataire"
sms_batches_details_delivered: "Delivre"
sms_batches_details_downloading: "Telechargement en cours..."

10. ACTIONS UTILISATEUR
-----------------------
- Retour : Revient a la liste des campagnes
- Telecharger CSV : Exporte les resultats

FIN DU CAHIER DES CHARGES
