# Cahier des Charges : telecom/sms - Modal détails SMS programmé

## 1. VUE D'ENSEMBLE
- **Type** : Modal de lecture
- **Contexte** : Page SMS programmés
- **Objectif** : Afficher les détails d'un SMS programmé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Informations
- **OdsCard** : Message complet
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détails du SMS programmé" |

## 5. INFORMATIONS AFFICHÉES
| Champ | Format |
|-------|--------|
| Date de création | DD/MM/YYYY HH:mm:ss |
| Date d'envoi prévue | DD/MM/YYYY HH:mm:ss (bold) |
| Expéditeur | Sender ID |
| Destinataire | Numéro international |
| Taille | "X SMS (Y crédit)" |

## 6. ZONE MESSAGE
| Propriété | Valeur |
|-----------|--------|
| Label | "Message complet" |
| Background | Gris clair (#F9FAFB) |
| Contenu | Texte intégral du SMS |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary | Ferme modal |

## 8. APIS & ENDPOINTS
- GET /sms/{serviceName}/jobs/{jobId} - Détail SMS programmé
