# Cahier des Charges : telecom/telephony - Modal ajouter un contact

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page annuaire (phonebook)
- **Objectif** : Ajouter un contact à l'annuaire de la ligne

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 610px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs identité et téléphones
- **OdsSelect** : Groupe
- **OdsMessage** : Info format numéro
- **OdsButton** : Annuler, Valider

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un contact" |
| Bouton fermer | × |

## 5. CHAMPS IDENTITÉ
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nom | Input | Oui * | "Nom de famille" |
| Prénom | Input | Oui * | "Prénom" |
| Groupe | Select | Non | Liste des groupes |

## 6. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info (bleu) | "Au moins un numéro de téléphone est requis au format international (+33...)" |

## 7. CHAMPS TÉLÉPHONES (grille 2x2)
| Champ | Type | Placeholder |
|-------|------|-------------|
| Téléphone domicile | Input tel | "+33..." |
| Mobile domicile | Input tel | "+33..." |
| Téléphone bureau | Input tel | "+33..." |
| Mobile bureau | Input tel | "+33..." |

## 8. VALIDATION
| Règle |
|-------|
| Au moins un numéro requis |
| Format international obligatoire (+33...) |

## 9. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Valider | Primary | Ajoute le contact |

## 10. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/phonebook/{phonebookId}/phonebookContact/group - Liste groupes
- POST /telephony/{billingAccount}/phonebook/{phonebookId}/phonebookContact - Ajouter contact
