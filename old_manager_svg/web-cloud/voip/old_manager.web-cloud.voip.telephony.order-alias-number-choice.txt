# Cahier des Charges : telecom/telephony - Commande numero choix

## 1. RESUME EXECUTIF

### Perimetre
Etape de selection du numero lors de la commande d'un alias.

### Objectif fonctionnel
- Choisir entre numeros classiques et premium
- Selectionner un numero dans la liste
- Voir les tarifs

---

## 2. LAYOUT

### Structure wizard avec stepper et cartes choix

---

## 3. COMPOSANTS UI

### Progress stepper
| Etape | Etat | Nom |
|-------|------|-----|
| 1 | Done (check vert) | Type |
| 2 | Active (bleu) | Numero |
| 3 | Todo (gris) | Retractation |
| 4 | Todo (gris) | Confirmation |

### Header page
| Element | Description |
|---------|-------------|
| Titre | Choisissez votre numero |

### Carte Numeros classiques (selected)
| Element | Description |
|---------|-------------|
| Header | Fond bleu clair, bordure bleue |
| Titre | Numeros classiques |
| Radio | Selectionne (cercle plein) |
| Liste | 3 numeros au choix |
| Prix | 1,00 € HT par mois |

### Carte Numeros premium
| Element | Description |
|---------|-------------|
| Header | Fond gris |
| Titre | Numeros premium |
| Radio | Non selectionne |
| Liste | 3 numeros faciles (00, 11, 22) |
| Prix | 9,99 € HT par mois |

### Ligne numero (dans liste)
| Etat | Style |
|------|-------|
| Selected | Fond bleu, bordure bleue, radio plein |
| Normal | Fond blanc, bordure grise, radio vide |

### Radio button numero
| Etat | Style |
|------|-------|
| Selected | Cercle bleu avec point blanc |
| Unselected | Cercle blanc bordure grise |

### Navigation buttons
| Bouton | Type | Position |
|--------|------|----------|
| ← Retour | Secondary | Gauche |
| Suivant → | Primary | Droite |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/number/available
```

### Modele
```typescript
interface AvailableNumber {
  number: string;
  type: 'standard' | 'premium';
  priceHT: number;
}
```
