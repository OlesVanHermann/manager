## 3. COMPOSANTS UI

### Header
| Element | Description |
|---------|-------------|
| Titre | Bibliotheque de sons |
| Bouton | + Ajouter un son |

### Zone upload
| Element | Description |
|---------|-------------|
| Drop zone | Glisser-deposer |
| Formats | WAV, MP3, OGG |
| Taille max | 10 Mo |

### Liste sons
| Colonne | Description |
|---------|-------------|
| Nom | Nom du fichier |
| Duree | Duree en secondes |
| Taille | Taille du fichier |
| Actions | Ecouter / Telecharger / Supprimer |

### Lecteur audio
| Element | Description |
|---------|-------------|
| Bouton play | Lecture/Pause |
| Barre progression | Position |
| Temps | Temps ecoule/total |

### Formulaire upload
| Champ | Type | Description |
|-------|------|-------------|
| Fichier | File input | Selection fichier |
| Nom | Input text | Nom personnalise |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Uploader | Primary | Envoie fichier |
| Annuler | Default | Ferme |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/sound
GET /telephony/{billingAccount}/ovhPabx/{serviceName}/sound/{soundId}
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/sound
DELETE /telephony/{billingAccount}/ovhPabx/{serviceName}/sound/{soundId}
```

### Modele Sound
```typescript
interface OvhPabxSound {
  soundId: number;
  name: string;
}
```

## 6. REGLES METIER

- Formats: WAV, MP3, OGG uniquement
- Taille max: 10 Mo par fichier
- Duree max recommandee: 5 minutes
- Conversion automatique cote serveur
