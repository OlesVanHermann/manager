## 11. APIS & ENDPOINTS

```
# Informations groupe
GET /telephony/{billingAccount}
→ {
  billingAccount: "group-abc123",
  description: "Groupe Principal",
  creationDate: "2022-03-15",
  status: "active"
}

# Compteurs services
GET /telephony/{billingAccount}/serviceCount
→ {
  lines: 5,
  numbers: 3,
  fax: 2,
  sms: 1,
  trunk: 1,
  total: 12
}

# Facturation
GET /telephony/{billingAccount}/billing
→ {
  billingCycle: "monthly",
  nextBillDate: "2026-01-01",
  estimatedAmount: 156.78
}

# Consommation du mois
GET /telephony/{billingAccount}/consumption/summary
Query: { month: "current" }
→ {
  outgoingCalls: 2547,
  outgoingDuration: "127:23:00",
  smsSent: 847,
  faxSent: 23
}
```
