## 8. APIS & ENDPOINTS

```
# Récupérer les options
GET /telephony/{billingAccount}/line/{serviceName}/options
→ {
  callerIdDisplay: true,
  callWaiting: true,
  callPickup: false,
  doNotDisturb: false,
  click2Call: true
}

# Modifier une option
PUT /telephony/{billingAccount}/line/{serviceName}/options
Body: {
  callerIdDisplay: true,
  callWaiting: true,
  ...
}

# Récupérer les restrictions
GET /telephony/{billingAccount}/line/{serviceName}/restrictions
→ {
  international: true,
  premium: false,
  mobile: true,
  ...
}

# Modifier les restrictions
PUT /telephony/{billingAccount}/line/{serviceName}/restrictions
Body: {
  international: false,
  premium: false,
  mobile: true
}
```
