# Cahier des Charges : telecom/alias - Serveur Vocal Interactif (SVI)

## 1. RESUME EXECUTIF

### Perimetre
Configuration d'un SVI externe.

### Objectif fonctionnel
- Configurer l'URL du serveur SVI
- Gerer les enregistrements
- Visualiser le flux d'appels

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/svi/{serviceName}
PUT /telephony/{billingAccount}/svi/{serviceName}
```

### Modele SVI
```typescript
interface TelephonySvi {
  serviceName: string;
  url: string;
  urlRecord: string;
}
```
