# Cahier des Charges : web-cloud/voip/number - Conference

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire avec panneau latéral
**Fonction:** Configurer une salle de conférence téléphonique
**Contexte:** Configuration du numéro en mode conférence

## 2. NAVIGATION

### Breadcrumb
Télécom > Groupe VoIP > Numéro > Configuration Conférence

## 3. HEADER

- Titre: "Configuration de la conférence"

## 4. LAYOUT PRINCIPAL

- Colonne gauche: Formulaire configuration (800x640px)
- Colonne droite: Panneau URLs (400x300px)

## 5. FORMULAIRE CONFIGURATION (800x640px)

### Section Configuration

#### Code PIN (200x40px)
- Label: "Code PIN d'accès (optionnel)"
- Input text: 4 chiffres
- Exemple: "1234"

#### Annonce personnalisée
- Label: "Annonce personnalisée (fichier audio)"
- Statut: "Aucun fichier"
- Bouton: "Télécharger" (btn-secondary, 150x32px)

#### Enregistrer la conférence (toggle buttons)
- Label: "Enregistrer la conférence"
- Option 1: "Activé" (sélectionné, bleu)
- Option 2: "Désactivé"

#### Micro des participants (toggle buttons)
- Label: "Micro des participants à l'entrée"
- Option 1: "Actif" (sélectionné)
- Option 2: "Muet"

#### Appels anonymes (toggle buttons)
- Label: "Appels anonymes"
- Option 1: "Acceptés" (sélectionné)
- Option 2: "Rejetés"

#### Langue (select 200x40px)
- Label: "Langue de la conférence"
- Options: Français, English, Deutsch, Español, etc.

### Section Administration

#### Rapport email (select 300x40px)
- Label: "Recevoir un rapport par e-mail"
- Options: À chaque conférence, Hebdomadaire, Mensuel, Jamais

### Boutons
- "Annuler" (btn-secondary, 100x36px)
- "Valider" (btn-primary, 100x36px)

## 6. PANNEAU URLS (400x300px)

### Titre
"URLs de suivi et contrôle"

### Bouton génération
"Générer les URLs" (btn-secondary, 180x36px)

### URL de suivi
- Label: "URL de suivi"
- Champ lecture seule: https://conference.ovh.net/follow/...

### URL de contrôle

## 9. APIS & ENDPOINTS

```
# Configuration de la conférence
GET /telephony/{billingAccount}/conference/{serviceName}
→ {
  pin: "1234",
  recordConference: true,
  enterMuted: false,
  anonymousCall: true,
  language: "fr",
  reportEmail: "everyConference"
}

# Modifier la configuration
PUT /telephony/{billingAccount}/conference/{serviceName}
Body: {
  pin: "5678",
  recordConference: false,
  enterMuted: true,
  anonymousCall: false,
  language: "en",
  reportEmail: "never"
}

# Uploader une annonce
POST /telephony/{billingAccount}/conference/{serviceName}/announcements
Body: { file: <audio_file> }

# Générer les URLs de suivi/contrôle
POST /telephony/{billingAccount}/conference/{serviceName}/webAccess
→ {
  followUrl: "https://conference.ovh.net/follow/...",
  controlUrl: "https://conference.ovh.net/control/..."
}

# Obtenir les URLs existantes
GET /telephony/{billingAccount}/conference/{serviceName}/webAccess
→ [{ type: "follow", url: "..." }, { type: "control", url: "..." }]
```
