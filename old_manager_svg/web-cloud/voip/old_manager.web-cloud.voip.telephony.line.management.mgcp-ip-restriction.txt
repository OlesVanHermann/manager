# Cahier des Charges : telecom/telephony - Restriction IP MGCP

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/management/mgcpIpRestriction
- **Objectif** : Configurer les restrictions IP pour les lignes MGCP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne
- **OdsLink** : Retour tableau de bord
- **OdsCard** : Restriction ligne + Restriction compte
- **OdsMessage** : Info restrictions
- **OdsInput** : Saisie IP/CIDR
- **OdsButton** : Valider

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour au tableau de bord de la ligne" |
| Titre | "Restriction IP MGCP" |

## 5. LAYOUT 2 COLONNES
| Colonne | Contenu |
|---------|---------|
| Gauche (540px) | Restriction de la ligne |
| Droite (540px) | Restriction du compte client |

## 6. CARD RESTRICTION DE LA LIGNE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Restriction de la ligne" |

### Message Info
| Type | Contenu |
|------|---------|
| Info (bleu) | "Restriction IP actuelle : 192.168.1.1. Seule cette IP pourra utiliser la ligne MGCP." |

### Champ
| Champ | Type | Placeholder |
|-------|------|-------------|
| Nouvelle restriction IP | Input | "Ex: 192.168.1.0/24" |

### Action Groupée
| Lien | Action |
|------|--------|
| "Appliquer à plusieurs lignes..." | Multi-sélection |

### Bouton
| Bouton | Style |
|--------|-------|
| Valider | Primary |

## 7. CARD RESTRICTION DU COMPTE CLIENT
| Propriété | Valeur |
|-----------|--------|
| Titre | "Restriction du compte client" |

### Message Info
| Type | Contenu |
|------|---------|
| Info (bleu) | "Cette restriction s'applique à toutes les lignes MGCP de votre compte par défaut." |

### Champ
| Champ | Type | Placeholder |
|-------|------|-------------|
| Restriction IP par défaut | Input | "Ex: 10.0.0.0/8" |

### Bouton
| Bouton | Style |
|--------|-------|
| Valider | Primary |

## 8. APIS & ENDPOINTS
- GET /telephony/{billingAccount}/line/{serviceName} - Infos ligne
- PUT /telephony/{billingAccount}/line/{serviceName} - Modifier restriction ligne
- GET /telephony/settings - Paramètres compte
- PUT /telephony/settings - Modifier restriction compte
