# Cahier des Charges : telecom/line - Modal Edition contact

## 1. RESUME EXECUTIF

### Perimetre
Modal de modification d'un contact existant dans le carnet d'adresses.

### Objectif fonctionnel
- Charger les donnees du contact
- Modifier les informations
- Sauvegarder les changements

---

## 2. LAYOUT

### Modal 600x650 avec formulaire pre-rempli

---

## 3. COMPOSANTS UI

### En-tete modal
| Element | Description |
|---------|-------------|
| Titre | Modifier le contact |
| Bouton fermer | X en haut a droite |

### Formulaire - Ligne 1 (2 colonnes)
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Nom | Input text | Dupont |
| Prenom | Input text | Jean |

### Formulaire - Ligne 2
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Groupe | Input + Select | Famille |

### Message info
| Element | Description |
|---------|-------------|
| Info | Au moins un numero de telephone est requis |

### Formulaire - Ligne 3 (2 colonnes)
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Tel. domicile | Input tel | 01 23 45 67 89 |
| Mobile perso | Input tel | 06 12 34 56 78 |

### Formulaire - Ligne 4 (2 colonnes)
| Champ | Type | Placeholder |
|-------|------|-------------|
| Tel. travail | Input tel | Telephone professionnel |
| Mobile pro | Input tel | Mobile professionnel |

### Footer modal
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Default | Ferme sans sauvegarder |
| Valider | Primary | Sauvegarde les modifications |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/{bookId}/phonebookContact/{contactId}
    Response: { name, surname, group, homePhone, homeMobile, workPhone, workMobile }

PUT /telephony/{billingAccount}/line/{serviceName}/phone/phonebook/{bookId}/phonebookContact/{contactId}
    Body: { name, surname, group, homePhone, homeMobile, workPhone, workMobile }
```
