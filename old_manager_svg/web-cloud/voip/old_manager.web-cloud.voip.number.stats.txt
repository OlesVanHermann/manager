## 11. APIS & ENDPOINTS

```
# Statistiques globales
GET /telephony/{billingAccount}/easyHunting/{serviceName}/statistics
Query: { period: "2025-12" }
→ {
  callsReceived: 1247,
  callsAnswered: 1089,
  callsLost: 158,
  avgWaitTime: 42,
  avgTalkTime: 263,
  availableAgents: 3,
  totalAgents: 5,
  serviceLevel: 78,
  serviceLevelTarget: 85,
  avgCallsPerHour: 52
}

# Volume par jour
GET /telephony/{billingAccount}/easyHunting/{serviceName}/statistics/daily
Query: { period: "2025-12" }
→ [{
  date: "2025-12-01",
  callsReceived: 45,
  callsAnswered: 40,
  callsLost: 5
}]

# Top agents
GET /telephony/{billingAccount}/easyHunting/{serviceName}/statistics/agents
Query: { period: "2025-12", limit: 5 }
→ [{
  agentId: "agent-001",
  name: "Jean Dupont",
  callsCount: 342
}]

# Export CSV
GET /telephony/{billingAccount}/easyHunting/{serviceName}/statistics/export
Query: { period: "2025-12", format: "csv" }
→ CSV file

# Rapport détaillé
GET /telephony/{billingAccount}/easyHunting/{serviceName}/statistics/report
Query: { period: "2025-12" }
→ PDF file
```
