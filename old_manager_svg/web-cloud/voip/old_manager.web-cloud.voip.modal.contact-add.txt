# Cahier des Charges : web-cloud/voip - Modal Contact Add

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter un contact à l'annuaire téléphonique
**Contexte:** Accessible depuis la gestion des contacts/annuaires

## 2. DIMENSIONS

- Modal: 550x520px

## 3. HEADER MODAL (60px)

- Background: #F9FAFB
- Titre: "Ajouter un contact" (16px, bold)
- Bouton fermer: "x"

## 4. CONTENU

### Champs du formulaire

| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Nom | Input text | Oui * | Entrez le nom du contact... |
| Prenom | Input text | Non | Entrez le prenom du contact... |
| Telephone principal | Input tel | Oui * | +33 ... |
| Telephone secondaire | Input tel | Non | Optionnel |
| Email | Input email | Non | contact@exemple.com |

### Dimensions inputs
- Tous les inputs: 502x40px
- Espacement vertical: 25px entre labels

### Note champs obligatoires
- Position: Après le dernier champ
- Texte: "* Champs obligatoires"
- Style: text-muted, font-size 10px

## 5. FOOTER MODAL

- Séparateur: Ligne #E5E7EB
- Position boutons: Gauche

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Ajouter | btn-primary | 120x36px |
| Annuler | btn-outline | 100x36px |

## 6. APIS & ENDPOINTS

```
# Ajouter un contact à l'annuaire
POST /telephony/{billingAccount}/phonebook/{phonebookId}/contact
Body: {
  lastName: "Dupont",
  firstName: "Jean",
  primaryPhone: "+33612345678",
  secondaryPhone: "+33198765432",
  email: "jean.dupont@exemple.com"
}
→ {
  contactId: "contact-123",
  lastName: "Dupont",
  firstName: "Jean"
}

# Valider format téléphone
POST /telephony/validatePhoneNumber
Body: { phoneNumber: "+33612345678" }
→ { valid: true, formatted: "+33 6 12 34 56 78" }
```
