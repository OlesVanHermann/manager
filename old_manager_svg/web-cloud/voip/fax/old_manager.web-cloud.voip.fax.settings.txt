## 10. APIS & ENDPOINTS

```
# Obtenir les paramètres
GET /telephony/{billingAccount}/fax/{serviceName}/settings
→ {
  quality: "high",
  maxTries: 3,
  header: "...",
  senderName: "OVH Fax",
  senderEmail: "noreply@ovh.com",
  fileFormat: "pdf"
}

# Modifier les paramètres
PUT /telephony/{billingAccount}/fax/{serviceName}/settings
Body: {
  quality: "high",
  maxTries: 3,
  header: "...",
  senderName: "...",
  senderEmail: "...",
  fileFormat: "pdf"
}

# Adresses de notification
GET /telephony/{billingAccount}/fax/{serviceName}/notifications/emails
POST /telephony/{billingAccount}/fax/{serviceName}/notifications/emails
Body: { email: "..." }

# Appliquer à plusieurs services
POST /telephony/{billingAccount}/fax/batchSettings
Body: {
  serviceNames: ["fax1", "fax2"],
  settings: { ... }
}
```
