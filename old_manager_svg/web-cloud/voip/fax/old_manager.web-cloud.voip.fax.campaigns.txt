# Cahier des Charges : web-cloud/voip/fax - Campaigns

## 1. VUE D'ENSEMBLE

**Type:** Page liste avec tableau et pagination
**Fonction:** Gérer les campagnes de fax en masse
**Contexte:** Envois de fax programmés ou terminés

## 2. NAVIGATION

### Breadcrumb
Télécom > Groupe VoIP > Fax {numéro} > Gestion du fax > Campagnes de fax

### Retour
← Gestion du fax

## 9. APIS & ENDPOINTS

```
# Liste des campagnes
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns
→ [{
  id, name, createdAt, status,
  faxSent, faxError, scheduledAt
}]

# Détail d'une campagne
GET /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}
→ { id, name, recipients: [...], status, stats: {...} }

# Créer une campagne
POST /telephony/{billingAccount}/fax/{serviceName}/campaigns
Body: {
  name: "...",
  recipients: ["0612345678", ...],
  documentUrl: "...",
  scheduledAt: "2024-12-28T09:00:00Z"
}

# Modifier une campagne (si programmée)
PUT /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}
Body: { name, scheduledAt, recipients }

# Annuler une campagne
DELETE /telephony/{billingAccount}/fax/{serviceName}/campaigns/{campaignId}
```
