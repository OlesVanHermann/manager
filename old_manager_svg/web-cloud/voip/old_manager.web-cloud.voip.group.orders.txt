## 8. APIS & ENDPOINTS

```
# Liste des commandes
GET /telephony/{billingAccount}/order
Query: { type, status, search, limit, offset }
→ [{
  orderId: "ORD-2024-001234",
  type: "line",
  description: "3 nouvelles lignes SIP",
  status: "pending",
  date: "2024-12-28"
}]

# Détail d'une commande
GET /telephony/{billingAccount}/order/{orderId}
→ {
  orderId: "ORD-2024-001234",
  type: "line",
  description: "3 nouvelles lignes SIP",
  status: "pending",
  date: "2024-12-28",
  items: [...],
  total: 45.00
}

# Créer une commande
POST /telephony/{billingAccount}/order
Body: {
  type: "line",
  quantity: 3,
  options: {...}
}

# Annuler une commande
DELETE /telephony/{billingAccount}/order/{orderId}
```
