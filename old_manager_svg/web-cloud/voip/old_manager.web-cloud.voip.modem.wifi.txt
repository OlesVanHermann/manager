## 5. APIs & DONNEES

GET/PUT /xdsl/:serviceName/modem/wifi/:wifiName

### Structure
{
  SSID: string,
  enabled: boolean,
  securityType: "None" | "WPA2" | "WPA2andWPA3" | "WPA3",
  securityKey: string,
  channel: number,
  SSIDAdvertisementEnabled: boolean,
  frequency: "2.4GHz" | "5GHz"
}

## 6. TRADUCTIONS (FR)

--------------------------------------------------------------------------------

## 7. REGLES METIER

- SSID : 2-32 caracteres, sans accents
- Cle : min 8 caracteres
- Securite : WPA2andWPA3 recommande
