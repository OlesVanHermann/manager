# Cahier des Charges : telecom/xdsl - Dashboard acces

## 1. RESUME EXECUTIF

### Perimetre
Tableau de bord simplifie d'un acces xDSL.

### Objectif fonctionnel
- Vue rapide des informations essentielles
- Acces aux actions rapides
- Alertes et notifications

## 2. LAYOUT

### Structure avec onglets et cartes

---

## 3. COMPOSANTS UI

### Lien retour
| Element | Description |
|---------|-------------|
| Link | Detail du pack |

### Titre
| Element | Description |
|---------|-------------|
| Titre | xdsl-ab12345-1 |
| Sous-titre | SDSL 2M - Ligne associee : 0033123456789 |

### Onglets
| Onglet | Etat |
|--------|------|
| Mon acces | Actif (souligne bleu) |
| Mon modem | Inactif |
| Taches | Inactif |

### Carte Informations generales
| Champ | Exemple |
|-------|---------|
| Identifiant | xdsl-ab12345-1 |
| Type | SDSL |
| Debit | 2 Mbps / 2 Mbps |
| Statut | Actif (badge vert) |
| Adresse IP | 203.0.113.42 |
| IPv6 | Desactive (badge jaune) |
| Lien | Configurer IPv6 |

### Carte Actions rapides
| Action | Description |
|--------|-------------|
| Voir les statistiques | Lien vers stats |
| Diagnostic de ligne | Lien vers diag |
| Gerer les notifications | Lien vers alertes |
| Changer de profil | Lien vers profil |

### Banniere mode bridge
| Element | Description |
|---------|-------------|
| Type | Warning (fond jaune) |
| Message | Le mode bridge est active sur votre modem |
| Lien | Configurer le mode bridge |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /xdsl/{serviceName}
    Response: { status, type, description }

GET /xdsl/{serviceName}/lines/{number}
    Response: { debitUp, debitDown }

GET /xdsl/{serviceName}/modem
    Response: { bridgeMode }
```
