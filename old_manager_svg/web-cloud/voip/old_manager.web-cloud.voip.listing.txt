**Fonction:** Lister et voir les lignes d'un compte

## 7. APIS & ENDPOINTS

```
# Liste des lignes d'un compte
GET /telephony/{billingAccount}/line
→ [{
  serviceName: "0033912345678",
  description: "Poste Direction",
  status: "active",
  phone: {
    model: "Yealink T46U",
    macAddress: "AA:BB:CC:DD:EE:01"
  }
}]

# Détails d'une ligne
GET /telephony/{billingAccount}/line/{serviceName}
→ {
  serviceName: "0033912345678",
  description: "Poste Direction",
  offer: "VoIP Entreprise",
  status: "active",
  sipUsername: "0033912345678@sip.ovh.fr"
}

# Téléphone associé
GET /telephony/{billingAccount}/line/{serviceName}/phone
→ {
  model: "Yealink T46U",
  macAddress: "AA:BB:CC:DD:EE:01",
  ipAddress: "192.168.1.100",
  firmware: "86.86.0.15"
}

# Abonnement
GET /telephony/{billingAccount}/line/{serviceName}/subscription
→ {
  creationDate: "2020-01-15",
  expirationDate: "2025-01-15",
  autoRenew: true
}
```
