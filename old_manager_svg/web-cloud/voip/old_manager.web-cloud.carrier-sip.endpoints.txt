# Cahier des Charges : web-cloud/carrier-sip - Endpoints

## 1. VUE D'ENSEMBLE
- **Type** : Page liste endpoints
- **URL** : /carrier-sip/{serviceName}/endpoints
- **Objectif** : Gérer les endpoints SIP (IP et configurations)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsText** : Titres sections
- **OdsDatagrid** : Tables IP et endpoints

## 4. SECTION LISTE DES IP
| Colonne | Description |
|---------|-------------|
| IP SIP | Adresse IP du serveur SIP |

### Données exemple
| IP SIP |
|--------|
| 91.121.128.5 |
| 91.121.128.6 |
| 213.186.32.10 |

## 5. SECTION ENDPOINTS AVEC APPELS ENTRANTS
| Colonne | Type | Description |
|---------|------|-------------|
| ID | String | Identifiant endpoint |
| Protocole | String | UDP, TCP, TLS |
| IP SIP | String | Adresse IP |
| Port | Number | Port SIP (5060, 5061) |
| Priorité | Number | Priorité de routage |
| Poids | Number | Poids pour load balancing |

### Données exemple
| ID | Protocole | IP SIP | Port | Priorité | Poids |
|----|-----------|--------|------|----------|-------|
| endpoint-001 | UDP | 91.121.128.5 | 5060 | 10 | 100 |
| endpoint-002 | TCP | 91.121.128.6 | 5060 | 20 | 50 |
| endpoint-003 | TLS | 213.186.32.10 | 5061 | 30 | 25 |

## 6. APIS & ENDPOINTS
- GET /telephony/trunks/{serviceName}/endpoints - Liste endpoints
- GET /telephony/trunks/{serviceName}/ips - Liste IP SIP
