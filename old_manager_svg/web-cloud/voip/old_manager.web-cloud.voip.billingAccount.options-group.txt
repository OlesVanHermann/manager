## 7. APIS & ENDPOINTS

### 7.1 Obtenir les options du groupe

```
GET /telephony/{billingAccount}
```

Response:
```json
{
  "hideCallsOnConsumption": false,
  "separateInternalExternalDisplay": true,
  "showLineDescriptionOnCalls": false
}
```

### 7.2 Modifier les options du groupe

```
PUT /telephony/{billingAccount}
```

Body:
```json
{
  "hideCallsOnConsumption": false,
  "separateInternalExternalDisplay": true,
  "showLineDescriptionOnCalls": false
}
```
