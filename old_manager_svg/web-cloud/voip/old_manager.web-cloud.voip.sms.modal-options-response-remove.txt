# Cahier des Charges : telecom/sms - Modal supprimer suivi réponses

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page options SMS
- **Objectif** : Supprimer une option de suivi des réponses

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Détails de l'option
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'option de suivi" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Confirmez-vous la suppression de cette option de suivi ?" |

## 6. DÉTAILS OPTION
| Champ | Description |
|-------|-------------|
| Média | E-mail, SMS ou HTTP |
| Expéditeur | Adresse email ou numéro |
| Destinataire | Adresse email ou numéro |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime l'option |

## 8. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/smsResponse/{hash} - Supprimer suivi
