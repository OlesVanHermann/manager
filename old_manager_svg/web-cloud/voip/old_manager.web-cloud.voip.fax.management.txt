## 1. RESUME EXECUTIF

Page centrale gestion fax associe a ligne VoIP. Acces aux configurations :
mot de passe, parametres, domaines marque blanche (VIP), filtrage, campagnes,
conversion Voicefax.

--------------------------------------------------------------------------------
## 2. ARCHITECTURE
--------------------------------------------------------------------------------

Fichiers : line/fax/fax.controller.js, fax.html
Sous-modules : password/, settings/, customDomains/, filtering/, campaigns/, convertToVoicefax/

--------------------------------------------------------------------------------
## 3. NAVIGATION
--------------------------------------------------------------------------------

Route : telecom.telephony.billingAccount.line.dashboard.fax
URL : /telecom/telephony/{billingAccount}/line/{serviceName}/fax

Sous-routes : .password, .settings, .customDomains, .filtering, .campaigns, .convertToVoicefax

--------------------------------------------------------------------------------
## 4. ACTIONS
--------------------------------------------------------------------------------

1. Mot de passe fax (toujours actif)
2. Configuration du fax (toujours actif)
3. Domaines Ecofax Marque Blanche (disabled si !TelecomMediator.isVip)
4. Filtrage des fax (toujours actif)
5. Campagnes de Fax (toujours actif)
6. Convertir un Ecofax perso (toujours actif)
