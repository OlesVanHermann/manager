# Cahier des Charges : telecom/sms - Modal annuler SMS programmé

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page SMS programmés
- **Objectif** : Annuler un SMS programmé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCard** : Détails du SMS
- **OdsMessage** : Info crédits
- **OdsButton** : Fermer, Annuler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Annuler le SMS programmé" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Confirmez-vous l'annulation de ce SMS programmé ?" |

## 6. DÉTAILS SMS
| Champ | Format |
|-------|--------|
| Date de création | DD/MM/YYYY HH:mm:ss |
| Expéditeur | Sender ID |
| Destinataire | Numéro international |
| Message | Texte tronqué |

## 7. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Icône | ⚠ |
| Texte | "Les crédits réservés seront restitués sur votre compte." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary | Ferme modal |
| Annuler | Danger (rouge) | Annule le SMS programmé |

## 9. APIS & ENDPOINTS
- DELETE /sms/{serviceName}/jobs/{jobId} - Annuler SMS programmé
