## 1. RÉSUMÉ EXÉCUTIF

### Périmètre
Écran de confirmation après génération d'un bon de commande de crédits fax.

### Objectif fonctionnel
Afficher le récapitulatif de la commande et les liens pour finaliser le paiement.

### Note
Le code source n'est pas fourni. Spécification basée sur les traductions.

---

## 2. ARCHITECTURE

### Contexte
État affiché après succès de la génération du bon de commande dans le module credit.

---

## 3. PAGES & NAVIGATION

### Intégration
Section/État dans le module credit du dashboard FreeFax.

### Navigation
| Action | Destination |
|--------|-------------|
| Régler commande | URL de paiement OVH |
| Version PDF | URL du bon de commande PDF |

## 4. COMPOSANTS UI

### Message de confirmation
"Votre bon de commande de {{amount}} fax, soit {{quantity}} unités, pour un montant de {{total}} TTC est disponible. Cliquez sur le lien ci-dessous pour valider votre commande."

### Liens/Boutons
| Élément | Label | Action |
|---------|-------|--------|
| Lien PDF | "Version PDF" | Téléchargement BC |
| Bouton principal | "Régler votre commande" | Redirection paiement |

## 5. APIs & DONNÉES

### Données reçues
```typescript
interface CreditOrderResult {
  amount: number;      // Nombre de fax commandés
  quantity: number;    // Unités
  total: string;       // Prix TTC formaté
  orderId: string;     // ID de la commande
  pdfUrl: string;      // URL du PDF
  paymentUrl: string;  // URL de paiement
}
```
