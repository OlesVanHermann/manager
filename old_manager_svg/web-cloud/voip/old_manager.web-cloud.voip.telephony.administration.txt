## 1. RÉSUMÉ EXÉCUTIF

Page d'administration du groupe de facturation téléphonie permettant d'accéder
aux différentes fonctions d'administration : options, regroupement de lignes,
gestion des groupes, portabilités, manager marque blanche.

### Utilisateurs cibles
- Administrateurs de groupes de facturation VoIP

### Objectif fonctionnel
- Centraliser les actions d'administration du groupe
- Accéder aux options de facturation
- Gérer les regroupements de lignes
- Créer/supprimer des groupes
- Suivre les portabilités
- Gérer le manager marque blanche
