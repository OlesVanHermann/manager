# Cahier des Charges : telecom/line - Code secret messagerie

## 1. RESUME EXECUTIF

### Perimetre
Page de changement du code secret de la messagerie vocale.

### Objectif fonctionnel
- Changer le code secret
- Forcer le changement au premier acces
- Voir les numeros d'acces

## 4. APIS & DONNEES

### Endpoints
```
PUT /telephony/{billingAccount}/line/{serviceName}/voicemail/password
GET /telephony/{billingAccount}/line/{serviceName}/voicemail/accessNumbers
```

### Modele
```typescript
interface VoicemailPassword {
  password: string;
  forceChange: boolean;
}

interface VoicemailAccessNumbers {
  internalNumber: string;
  externalNumber: string;
}
```
