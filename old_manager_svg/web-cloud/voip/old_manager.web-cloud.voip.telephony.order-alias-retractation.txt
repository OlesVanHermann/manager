# Cahier des Charges : telecom/telephony - Commande numero retractation

## 1. RESUME EXECUTIF

### Perimetre
Etape de choix du droit de retractation lors de la commande d'un numero.

### Objectif fonctionnel
- Informer sur le droit de retractation (14 jours)
- Permettre de renoncer pour activation immediate
- Valider le choix utilisateur
