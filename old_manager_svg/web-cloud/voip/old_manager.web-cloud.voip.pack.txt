## 1. RÉSUMÉ EXÉCUTIF

Page de liste des packs xDSL et dashboard d'un pack individuel avec ses
services associés (accès internet, téléphonie, email, etc.).

### Utilisateurs cibles
- Clients disposant de packs xDSL/Fibre OVHcloud

### Objectif fonctionnel
- Lister tous les packs du compte
- Accéder au détail d'un pack
- Gérer les services inclus dans le pack
