# Cahier des Charges : telecom/xdsl - Configuration WiFi

## 1. RESUME EXECUTIF

### Perimetre
Configuration des reseaux WiFi 2.4 GHz et 5 GHz du modem.

### Objectif fonctionnel
- Activer/desactiver le WiFi
- Configurer les SSID et mots de passe
- Choisir les parametres de securite

## 4. APIS & DONNEES

### Endpoints
```
GET /xdsl/{serviceName}/modem/wifi
    Response: {
      enabled, wifi24: { ssid, security, channel },
      wifi5: { ssid, security, channel },
      options: { defaultEnabled, broadcastSsid }
    }

PUT /xdsl/{serviceName}/modem/wifi
    Body: { enabled, wifi24, wifi5, options }
```

### Types de securite
| Type | Description |
|------|-------------|
| open | Pas de securite |
| WEP | Obsolete |
| WPA-PSK | WPA personnel |
| WPA2-PSK | WPA2 personnel (recommande) |
| WPA3 | WPA3 (si supporte) |

### Canaux
| Bande | Canaux |
|-------|--------|
| 2.4 GHz | 1-13, Auto |
| 5 GHz | 36-165, Auto |

## 6. REGLES METIER

- Mode bridge desactive WiFi
- WPA2-PSK minimum recommande
- Canal auto optimise par modem
- SSID cache si broadcast off
