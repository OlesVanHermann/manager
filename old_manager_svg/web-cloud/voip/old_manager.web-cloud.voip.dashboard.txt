## 11. APIS & ENDPOINTS

```
# Informations groupe
GET /telephony/{billingAccount}
→ {
  billingAccount: "voip-ab12345",
  description: "Telephonie Siege",
  status: "active",
  currentOutplan: 12.50,
  credit: 50.00
}

# Compteurs services
GET /telephony/{billingAccount}/service
→ {
  lines: 3,
  numbers: 2,
  fax: 1
}

# Consommation du mois
GET /telephony/{billingAccount}/consumption
Query: { month: "current" }
→ {
  usagePercent: 72,
  outOfPlan: 12.50
}

# Derniers relevés
GET /telephony/{billingAccount}/bill
Query: { limit: 5 }
→ [{
  date: "2025-12-01",
  amountHT: 45.00,
  paid: true
}]

# Équipements
GET /telephony/{billingAccount}/equipment
→ [{
  model: "Yealink T46U",
  line: "0033912345678",
  mac: "AA:BB:CC:DD:EE:01"
}]
```
