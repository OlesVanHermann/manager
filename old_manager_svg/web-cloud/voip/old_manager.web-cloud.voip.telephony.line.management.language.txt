# Cahier des Charges : telecom/telephony - Langue de la ligne

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **URL** : /telephony/{billingAccount}/line/{serviceName}/management/language
- **Objectif** : Configurer la langue des messages vocaux du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Télécom / Téléphonie / Ligne
- **OdsLink** : Retour tableau de bord
- **OdsCard** : Configuration langue
- **OdsSelect** : Choix langue
- **OdsButton** : Valider

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Retour au tableau de bord de la ligne" |
| Titre | "Langue de la ligne" |

| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration de la langue" |

### Description
| Texte |
