# Cahier des Charges : telecom/line - Creneau ligne OVHcloud

## 1. RESUME EXECUTIF

### Perimetre
Configuration d'un creneau avec redirection vers une ligne OVHcloud.

### Objectif fonctionnel
- Selectionner le type "ligne OVHcloud"
- Choisir parmi les lignes disponibles
- Rediriger vers une ligne du parc

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line
    Response: [{ serviceName, description, number }]

PUT /telephony/{billingAccount}/line/{serviceName}/timeCondition/slots/{slotId}
    Body: {
      type: "ovhLine",
      number: "0033177889900",
      serviceName: "support-technique"
    }
```

### Types de lignes OVH
| Type | Description |
|------|-------------|
| sip | Ligne SIP standard |
| trunk | Ligne SIP Trunk |
| alias | Numero alias |
