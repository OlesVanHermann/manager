# Cahier des Charges : web-cloud/voip - Modal Phonebook Add

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire simple
**Fonction:** Créer un nouvel annuaire téléphonique
**Contexte:** Accessible depuis la gestion des annuaires

## 2. DIMENSIONS

- Modal: 500x300px

## 3. HEADER MODAL (60px)

- Background: #F9FAFB
- Titre: "Creer un annuaire" (16px, bold)
- Bouton fermer: "x"

## 4. CONTENU

### Nom de l'annuaire
- Label: "Nom de l'annuaire *" (12px, bold)
- Input: 452x40px
- Placeholder: "Ex: Annuaire commercial, Contacts support..."

### Description
- Label: "Description" (12px, bold)
- Textarea: 452x60px
- Placeholder: "Description optionnelle..."

## 5. FOOTER MODAL

- Séparateur: Ligne #E5E7EB
- Position boutons: Gauche

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Creer | btn-primary | 120x36px |
| Annuler | btn-outline | 100x36px |

## 6. APIS & ENDPOINTS

```
# Créer un annuaire
POST /telephony/{billingAccount}/phonebook
Body: {
  name: "Annuaire commercial",
  description: "Contacts de l'equipe commerciale"
}
→ {
  phonebookId: "pb-12345",
  name: "Annuaire commercial",
  description: "Contacts de l'equipe commerciale",
  contactCount: 0
}

# Liste des annuaires
GET /telephony/{billingAccount}/phonebook
→ [{
  phonebookId: "pb-12345",
  name: "Annuaire commercial",
  description: "Contacts de l'equipe commerciale",
  contactCount: 42
}]
```
