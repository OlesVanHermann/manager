# Cahier des Charges : telecom/line - Creneau repondeur

## 1. RESUME EXECUTIF

### Perimetre
Configuration d'un creneau avec redirection vers un repondeur.

### Objectif fonctionnel
- Selectionner le type "repondeur"
- Choisir le repondeur de destination
- Configurer la redirection vers messagerie vocale

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line
    Response: [{ serviceName, description, number, voicemailEnabled }]

PUT /telephony/{billingAccount}/line/{serviceName}/timeCondition/slots/{slotId}
    Body: {
      type: "voicemail",
      number: "0033123456789",
      serviceName: "bureau-principal"
    }
```

### Lignes avec repondeur actif
| Condition | Affichage |
|-----------|-----------|
| voicemailEnabled: true | Disponible dans liste |
| voicemailEnabled: false | Non affiche |
