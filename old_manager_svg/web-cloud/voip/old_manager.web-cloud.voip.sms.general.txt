## 11. APIS & ENDPOINTS

```
# Informations compte SMS
GET /sms/{serviceName}
→ {
  serviceName: "sms-abc123",
  description: "SMS Marketing",
  status: "active",
  creditsLeft: 1500,
  defaultSender: "MonEntreprise"
}

# Statistiques du mois
GET /sms/{serviceName}/statistics
Query: { period: "month" }
→ {
  sentCount: 847,
  receivedCount: 123,
  creditsUsed: 847
}

# Liste des expéditeurs
GET /sms/{serviceName}/senders
→ ["MonEntreprise", "Support", "Info"]

# Configuration réception
GET /sms/{serviceName}/receivers
→ [{
  number: "+33600000000",
  status: "active",
  webhookUrl: "https://..."
}]

# Acheter des crédits
POST /sms/{serviceName}/credits/buy
Body: { quantity: 1000 }
```
