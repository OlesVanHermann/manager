CAHIER DES CHARGES - old_manager.telecom.fax.modal-read-campaign
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/fax
Composant  : Modal consultation campagne fax
Type       : Modal dialog (lecture seule)
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Modal affichant les details d une campagne de fax existante.
Permet de consulter l etat, les statistiques et la liste des destinataires
avec leur statut (en attente, succes, echec).

4. INFORMATIONS AFFICHEES
-------------------------
| Champ          | Format     | Description                       |
|----------------|------------|-----------------------------------|
| name           | Texte      | Nom de la campagne                |
| reference      | Date/heure | Horodatage creation               |
| success        | Texte      | Succes/Succes partiel/N/A/Echec   |
| status         | Texte      | todo/doing/stop/error/done        |
| message        | Texte      | Message erreur (si erreur)        |
| dateStart      | Date/heure | Date lancement                    |
| dateEnd        | Date/heure | Date fin                          |
| countTotal     | Nombre     | Total destinataires               |
| details.todo   | Liste      | Numeros en attente                |
| details.success| Liste      | Numeros envoyes avec succes       |
| details.failed | Liste      | Numeros en echec                  |

5. STATUTS CAMPAGNE
-------------------
| Status     | Label FR                     |
|------------|------------------------------|
| todo       | Planifie                     |
| doing      | En cours d execution         |
| stop       | En cours de synchronisation  |
| stopTodo   | Lancement manuel requis      |
| stopDoing  | Stoppe en cours              |
| error      | En erreur                    |
| done       | Termine                      |

6. ETATS UI
-----------
- Chargement : oui-spinner + "Chargement..."
- Affichage : Liste details avec sections depliables

7. ENDPOINTS API
----------------

| GET | /telephony/{billingAccount}/fax/{serviceName}/campaigns/{id}/detail | Details destinataires |
