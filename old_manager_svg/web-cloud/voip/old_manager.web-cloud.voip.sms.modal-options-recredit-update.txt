# Cahier des Charges : telecom/sms - Modal recrédit automatique

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Page options SMS
- **Objectif** : Configurer le recrédit automatique de crédits SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Seuil de déclenchement
- **OdsSelect** : Quantité à recréditer
- **OdsMessage** : Info fonctionnement
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Recrédit automatique" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Seuil de déclenchement | Input numérique | Oui * | En crédits |
| Quantité à recréditer | Select | Oui | Options avec prix HT |

## 6. OPTIONS QUANTITÉ
| Quantité | Prix |
|----------|------|
| 1000 crédits | X,XX € HT |
| 5000 crédits | X,XX € HT |
| 10000 crédits | X,XX € HT |

## 7. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | Explication du fonctionnement automatique avec prélèvement sur moyen de paiement par défaut |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre configuration |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName} - Configuration actuelle
- PUT /sms/{serviceName} - Modifier (automaticRecreditAmount)
