# Cahier des Charges : telecom/alias - Modal Annuler portabilite

## 1. RESUME EXECUTIF

### Perimetre
Modal de confirmation d'annulation d'une portabilite.

### Objectif fonctionnel
- Confirmer l'annulation
- Avertir de l'irreversibilite
- Annuler la demande de portabilite
