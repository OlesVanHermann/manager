# Cahier des Charges : web-cloud/voip - Modal Trunk Credentials

## 1. VUE D'ENSEMBLE

**Type:** Modal affichage informations sensibles
**Fonction:** Afficher les identifiants SIP d'un trunk
**Contexte:** Accessible depuis la gestion d'un trunk SIP

## 2. DIMENSIONS

- Modal: 550x400px

## 3. HEADER MODAL (60px)

- Background: #F9FAFB
- Titre: "Identifiants SIP" (16px, bold)
- Bouton fermer: "x"

## 4. CONTENU

### Information trunk
- Label: "Trunk SIP" (text-muted, 11px)
- Valeur: "{trunk-name}" (14px, bold)
- Exemple: "trunk-abc123"

### Séparateur
Ligne horizontale #E5E7EB

### Champs credentials (avec boutons copier)

| Champ | Valeur exemple | Actions |
|-------|----------------|---------|
| Serveur SIP | sip.ovh.net | Copier |
| Identifiant | trunk-abc123@sip.ovh.net | Copier |
| Mot de passe | **************** | Révéler + Copier |

### Format champs
- Zone lecture: 400x36px (background #F9FAFB)
- Bouton copier: 92x36px (btn-outline)
- Icône révéler mot de passe: "O" (position droite dans le champ)

### Bannière warning (502x40px)
- Background: #FEF3C7
- Texte: "Ne partagez jamais ces identifiants. Ils permettent l'acces complet au trunk SIP."
- Couleur texte: #D97706

## 5. FOOTER MODAL

- Séparateur: Ligne #E5E7EB
- Position boutons: Espacés

### Boutons

| Bouton | Style | Dimensions | Position |
|--------|-------|------------|----------|
| Regenerer le mot de passe | btn-outline | 180x28px | Gauche |
| Fermer | btn-primary | 120x28px | Droite |

## 6. APIS & ENDPOINTS

```
# Récupérer identifiants trunk
GET /telephony/{billingAccount}/trunk/{serviceName}/credentials
→ {
  server: "sip.ovh.net",
  username: "trunk-abc123@sip.ovh.net",
  port: 5060
}

# Révéler mot de passe
POST /telephony/{billingAccount}/trunk/{serviceName}/credentials/password/reveal
→ {
  password: "xY9#kL2mN..."
}

# Régénérer mot de passe
POST /telephony/{billingAccount}/trunk/{serviceName}/credentials/password/regenerate
→ {
  success: true,
  newPassword: "aB3$fG7hJ...",
  message: "Nouveau mot de passe genere"
}
```
