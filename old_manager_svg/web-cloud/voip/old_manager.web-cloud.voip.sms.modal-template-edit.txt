# Cahier des Charges : telecom/sms - Modal modifier modèle SMS

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification
- **Contexte** : Page modèles SMS
- **Objectif** : Modifier un modèle SMS existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 540px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom (readonly), Description, Message
- **OdsSelect** : Secteur d'activité
- **OdsCheckbox** : Activer le modèle
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le modèle SMS" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Note |
|-------|------|--------|------|
| Nom du modèle | Input (readonly) | - | Non modifiable |
| Secteur d'activité | Select | Oui * | |
| Description | Textarea | Oui * | 31-249 caractères |
| Message | Textarea | Oui * | 31-249 caractères |

## 6. SECTEURS D'ACTIVITÉ
| Option |
|--------|
| E-commerce |
| Banque/Finance |
| Santé |
| Services |
| Marketing |

## 7. CHECKBOX ACTIVATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Activer ce modèle" |
| État | Coché par défaut |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre modifications |

## 9. APIS & ENDPOINTS
- GET /sms/{serviceName}/templates/{templateId} - Détail modèle
- PUT /sms/{serviceName}/templates/{templateId} - Modifier modèle
