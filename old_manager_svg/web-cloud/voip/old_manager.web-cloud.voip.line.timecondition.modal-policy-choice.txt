# Cahier des Charges : telecom/line - Choix du creneau

## 1. RESUME EXECUTIF

### Perimetre
Modal de selection du creneau vers lequel rediriger les appels.

### Objectif fonctionnel
- Lister les creneaux disponibles
- Afficher leur configuration actuelle
- Permettre la selection

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/timeCondition/slots
    Response: {
      available: { type, queue },
      slot1: { type, number },
      slot2: { type, number },
      slot3: { type, number }
    }
```

### Structure slot
| Champ | Type | Description |
|-------|------|-------------|
| type | string | voicemail, external, ovhLine, queue |
| number | string | Numero de destination |
| configured | boolean | Si configure ou non |

## 6. REGLES METIER

- 4 options maximum (ouverture + 3 creneaux)
- Creneaux non configures utilisables mais sans effet
- Selection obligatoire pour valider
- Navigation retour conserve selection precedente
