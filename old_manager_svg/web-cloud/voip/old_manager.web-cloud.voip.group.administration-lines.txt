## 9. APIS & ENDPOINTS

### 9.1 Lister les groupes de téléphonie

```
GET /telephony
```

### 9.2 Lister les lignes d'un groupe

```
GET /telephony/{billingAccount}/line
```

### 9.3 Lister les alias d'un groupe

```
GET /telephony/{billingAccount}/number
```

### 9.4 Attacher des services à un groupe

```
POST /telephony/{billingAccount}/transferSecurityDeposit
```

Body:
```json
{
  "targetBillingAccount": "ab12345-ovh-1",
  "services": [
    "+33198765432",
    "+33198765434"
  ]
}
```
