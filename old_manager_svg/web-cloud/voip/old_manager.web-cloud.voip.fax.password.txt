# Cahier des Charges : web-cloud/voip/fax - Password

**Type:** Page formulaire simple
**Fonction:** Modifier le mot de passe d'accès au fax
**Contexte:** Sécurité du service fax

## 2. NAVIGATION

### Breadcrumb
Télécom > Groupe VoIP > Fax {numéro} > Gestion du fax > Mot de passe fax

### Retour
← Gestion du fax

- Titre: "Mot de passe fax"

### Bouton validation

## 5. RÈGLES DE VALIDATION

| Règle | Condition |
|-------|-----------|
| Longueur minimum | 6 caractères |
| Longueur maximum | 12 caractères |
| Correspondance | password === confirmation |
| Caractères | Alphanumériques uniquement |

## 6. APIS & ENDPOINTS

```
# Modifier le mot de passe
PUT /telephony/{billingAccount}/fax/{serviceName}/settings/password
Body: {
  password: "newpassword123"
}

# Appliquer à plusieurs services
POST /telephony/{billingAccount}/fax/batchPassword
Body: {
  serviceNames: ["fax1", "fax2"],
  password: "newpassword123"
}
```
