# Cahier des Charges : web-cloud/voip/line - Phone Configuration

## 1. VUE D'ENSEMBLE

**Type:** Page dashboard multi-sections en 2 colonnes
**Fonction:** Configurer les paramètres du téléphone
**Contexte:** Configuration avancée du téléphone SIP

## 2. NAVIGATION

### Retour
< Téléphone

## 3. HEADER

- Titre: "Configuration du téléphone"

## 4. TOGGLE MODE

- "Standard" | Switch | "Expert"
- Bouton: "Modifier les paramètres" (btn-primary, 180x36px)

## 6. PARAMÈTRES EXPERT (mode expert)

Paramètres supplémentaires affichés en mode expert :
- SIP Transport (UDP/TCP/TLS)
- DTMF Mode
- NAT Keep-Alive
- Provisioning Server

## 7. APIS & ENDPOINTS

```
# Configuration téléphone
GET /telephony/{billingAccount}/line/{serviceName}/phone/configuration
→ {
  ui: { language, timezone, dateFormat, ringtone, ringVolume },
  network: { ipMode, vlan, qos },
  audio: { earVolume, speakerVolume, noiseReduction },
  features: { autoAnswer, dnd, callForward }
}

PUT /telephony/{billingAccount}/line/{serviceName}/phone/configuration
Body: { ... }

# Redémarrer le téléphone
POST /telephony/{billingAccount}/line/{serviceName}/phone/reboot
```
