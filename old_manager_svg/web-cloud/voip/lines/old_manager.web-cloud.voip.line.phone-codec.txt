## 1. VUE D'ENSEMBLE

**Type:** Page formulaire avec options radio
**Fonction:** Configurer le codec audio de la ligne
**Contexte:** Configuration avancée du téléphone SIP

## 7. CODECS DISPONIBLES

| Codec | Bande passante | Qualité | Usage |
|-------|----------------|---------|-------|
| G711a | 64 kbps | Standard | Europe |
| G711u | 64 kbps | Standard | USA |
| G722 | 64 kbps | HD | Audio HD |

## 8. APIS & ENDPOINTS

```
# Configuration codec
GET /telephony/{billingAccount}/line/{serviceName}/codec
→ codecList, autoMode

PUT /telephony/{billingAccount}/line/{serviceName}/codec
Body: {
  codecOrder: ["G711a", "G711u", "G722"],
  autoMode: false
}

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchCodec
Body: {
  serviceNames: ["line1", "line2"],
  codecOrder: ["G722", "G711a", "G711u"]
}
```
