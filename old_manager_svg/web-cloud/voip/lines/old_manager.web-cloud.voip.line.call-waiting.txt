# Cahier des Charges : web-cloud/voip/line - Call Waiting

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire avec toggles
**Fonction:** Configurer le double appel et l'interphone
**Contexte:** Gestion des appels de la ligne VoIP

## 2. NAVIGATION

### Retour
< Appels

## 3. HEADER

- Titre: "Mise en attente"

- Border: #BFDBFE
- Texte ligne 1: "Le double appel permet de recevoir un second appel pendant une communication."

- Label: "Activer le double appel"
- Switch: ON (bleu #0050D7, 50x26px)

### Titre section

### Bannière info (750x100px)

## 6. ACTIONS

### Lien bulk

### Boutons

- "Annuler" (text-link)

## 7. APIS & ENDPOINTS

```
# Options d'appel
GET /telephony/{billingAccount}/line/{serviceName}/options
→ callWaiting, intercom

PUT /telephony/{billingAccount}/line/{serviceName}/options
Body: {
  callWaiting: true,
  intercom: true
}

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchOptions
Body: {
  serviceNames: ["line1", "line2"],
  callWaiting: true,
  intercom: true
}
```
