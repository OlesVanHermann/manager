## 5. APIS & ENDPOINTS

```
# Informations offre
GET /telephony/{billingAccount}/line/{serviceName}/offer
→ group, nextBilling, currentOffer, currentPrice, pendingChange

# Offres disponibles
GET /telephony/{billingAccount}/line/{serviceName}/availableOffers
→ Liste des offres avec prix

# Modifier offre
PUT /telephony/{billingAccount}/line/{serviceName}/offer
Body: {
  offer: "voip-business-pro"
}

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchOffer
Body: {
  serviceNames: ["line1", "line2"],
  offer: "voip-business-pro"
}
```
