# Cahier des Charges : web-cloud/voip/line - Modal Phonebook Add

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Ajouter un contact à l'annuaire
**Contexte:** Gestion de l'annuaire de la ligne VoIP

## 2. DIMENSIONS

- Modal: 600x700px
- Overlay: rgba(0, 0, 0, 0.5)

## 3. HEADER MODAL

- Titre: "Ajouter un contact"
- Bouton fermer: × (coin supérieur droit)

## 4. FORMULAIRE

### Ligne 1 - Identité (2 colonnes)

#### Nom *
- Input: 260x40px
- Placeholder: "Entrez le nom..."
- Obligatoire

#### Prénom *
- Input: 260x40px
- Placeholder: "Entrez le prénom..."
- Obligatoire

### Ligne 2 - Groupe

#### Groupe
- Input: 400x40px
- Placeholder: "Entrez ou sélectionnez un groupe..."
- Bouton: "Sélectionner" (130x40px)

### Bannière info (550x40px)
- Background: #EFF6FF
- Border: #BFDBFE
- Texte: "Au moins un numéro de téléphone doit être renseigné"

### Numéros de téléphone (2 colonnes)

#### Colonne gauche
| Champ | Placeholder |
|-------|-------------|
| Tél. domicile | +33... |
| Tél. professionnel | +33... |

#### Colonne droite
| Champ | Placeholder |
|-------|-------------|
| Mobile personnel | +33... |
| Mobile professionnel | +33... |

- Inputs: 260x40px chacun

## 5. FOOTER MODAL

- Séparateur: ligne horizontale
- Bouton "Annuler" (btn-outline, 90x40px)
- Bouton "Ajouter" (btn-primary, 90x40px)

## 6. CHAMPS DU FORMULAIRE

| Champ | Type | Obligatoire | Validation |
|-------|------|-------------|------------|
| lastName | text | Oui | Non vide |
| firstName | text | Oui | Non vide |
| group | text/select | Non | - |
| homePhone | tel | Non* | Format E.164 |
| personalMobile | tel | Non* | Format E.164 |
| workPhone | tel | Non* | Format E.164 |
| workMobile | tel | Non* | Format E.164 |

*Au moins un numéro requis

## 7. APIS & ENDPOINTS

```
# Ajouter un contact
POST /telephony/{billingAccount}/line/{serviceName}/phonebook/contact
Body: {
  lastName: "Dupont",
  firstName: "Jean",
  group: "Famille",
  homePhone: "+33123456789",
  personalMobile: "+33612345678",
  workPhone: "+33172100002",
  workMobile: null
}

# Groupes existants
GET /telephony/{billingAccount}/line/{serviceName}/phonebook/groups
→ Liste des groupes
```
