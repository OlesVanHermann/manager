## 6. APIS & ENDPOINTS

```
# Historique fax sortants
GET /telephony/{billingAccount}/line/{serviceName}/consumption/outgoingFax
Query: ?dateFrom=2024-12-01&dateTo=2024-12-31&page=1&pageSize=10
→ {
  faxes: [{ date, callee, pages, nature, price }],
  totalFaxes: 5,
  totalPages: 12
}

# Renvoyer un fax
POST /telephony/{billingAccount}/line/{serviceName}/fax/{faxId}/resend

# Export CSV
GET /telephony/{billingAccount}/line/{serviceName}/consumption/outgoingFax/export
Query: ?format=csv
```
