# Cahier des Charges : web-cloud/voip/line - Lock Out Call

## 1. VUE D'ENSEMBLE

**Type:** Page formulaire avec panneau aide
**Fonction:** Configurer le verrouillage des appels sortants par code PIN
**Contexte:** Sécurité de la ligne VoIP

## 2. NAVIGATION

### Retour
< Appels

## 3. HEADER

- Titre: "Verrouillage appels sortants"

## 4. FORMULAIRE PRINCIPAL (600x500px)

### Bannière info (550x80px)
- Background: #EFF6FF
- Border: #BFDBFE
- Texte: "Le verrouillage des appels sortants permet d'empêcher les appels sortants sur cette ligne. Un code PIN est requis pour passer des appels lorsque cette option est activée."

### Checkbox activation
- Case: Cochée (#0050D7)
- Label: "Activer le verrouillage des appels sortants"

### Champ PIN
- Label: "Code PIN (4 chiffres)"
- Input: 200x45px, type password
- Valeur: ●●●●
- Note: "Le code PIN doit contenir exactement 4 chiffres."

### Bannière avertissement (550x60px)
- Background: #FEE2E2
- Border: #FECACA
- Texte ligne 1: "Attention : une fois activé, vous devrez entrer ce code"
- Texte ligne 2: "avant chaque appel sortant depuis cette ligne."

### Lien bulk
"Appliquer à plusieurs lignes..." (text-link)

### Boutons
- "Valider" (btn-primary, 100x40px)
- "Annuler" (text-link)

## 5. PANNEAU AIDE (400x300px)

### Titre
"Comment ça fonctionne ?"

### Instructions
1. Activez le verrouillage et définissez un code PIN
2. Lorsque vous passez un appel sortant :
   - Décrochez et composez le numéro
   - Un message vous demande le code PIN
   - Entrez votre code à 4 chiffres
   - L'appel est connecté

### Utilisation
- Éviter les appels non autorisés
- Contrôler l'usage d'une ligne partagée
- Sécuriser les lignes en accès public

## 6. APIS & ENDPOINTS

```
# Configuration verrouillage
GET /telephony/{billingAccount}/line/{serviceName}/options
→ outgoingCallLock, outgoingCallPin

PUT /telephony/{billingAccount}/line/{serviceName}/options
Body: {
  outgoingCallLock: true,
  outgoingCallPin: "1234"
}

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchOptions
Body: {
  serviceNames: ["line1", "line2"],
  outgoingCallLock: true,
  outgoingCallPin: "1234"
}
```
