## 8. APIS & ENDPOINTS

```
# Accessoires compatibles
GET /telephony/{billingAccount}/line/{serviceName}/phone/accessories
→ Liste des accessoires avec prix et compatibilité

# Ajouter au panier
POST /telephony/{billingAccount}/cart
Body: {
  lineServiceName: "...",
  items: [
    { accessory: "exp40", quantity: 1 }
  ]
}

# Passer commande
POST /telephony/{billingAccount}/order/accessories
Body: {
  cartId: "...",
  shippingAddress: {...}
}
```
