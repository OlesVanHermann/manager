## 6. APIS & ENDPOINTS

```
# Historique appels entrants
GET /telephony/{billingAccount}/line/{serviceName}/consumption/incomingCalls
Query: ?dateFrom=2024-12-01&dateTo=2024-12-31&page=1&pageSize=10
→ calls: [{ date, caller, callee, duration, nature }], total, totalDuration

# Export CSV
GET /telephony/{billingAccount}/line/{serviceName}/consumption/incomingCalls/export
Query: ?dateFrom=2024-12-01&dateTo=2024-12-31&format=csv
```
