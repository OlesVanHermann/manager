## 5. TYPES DE RMA

| Type | Code | Description |
|------|------|-------------|
| Échange standard | exchange | Remplacement par équipement identique |
| Retour définitif | return | Retour sans remplacement |
| Upgrade | upgrade | Remplacement par modèle supérieur |

## 6. STATUTS ÉTAPES

| Statut | Couleur | Description |
|--------|---------|-------------|
| done | #16A34A | Étape complétée |
| pending | #E5E7EB | En attente |
| error | #DC2626 | Problème |

## 7. APIS & ENDPOINTS

```
# Détails RMA
GET /telephony/{billingAccount}/line/{serviceName}/phone/rma/{rmaId}
→ {
  id, type, equipment, newEquipment,
  shippingAddress: { name, street, city, email, phone },
  steps: [{ name, status, date }]
}

# Annuler RMA
DELETE /telephony/{billingAccount}/line/{serviceName}/phone/rma/{rmaId}

# Télécharger PDF
GET /telephony/{billingAccount}/line/{serviceName}/phone/rma/{rmaId}/document
→ URL du PDF
```
