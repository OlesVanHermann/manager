## 8. APIS & ENDPOINTS

```
# Informations offre actuelle
GET /telephony/{billingAccount}/line/{serviceName}/offer
→ offer, price, billingGroup, nextBilling, pendingChange

# Offres disponibles
GET /telephony/{billingAccount}/line/{serviceName}/availableOffers
→ Liste des offres avec prix

# Changer d'offre
POST /telephony/{billingAccount}/line/{serviceName}/changeOffer
Body: {
  offer: "business"
}

# Annuler changement en attente
DELETE /telephony/{billingAccount}/line/{serviceName}/changeOffer

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchChangeOffer
Body: {
  serviceNames: ["line1", "line2"],
  offer: "business"
}
```
