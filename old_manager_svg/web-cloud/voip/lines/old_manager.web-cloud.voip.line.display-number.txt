## 8. APIS & ENDPOINTS

```
# Configuration actuelle
GET /telephony/{billingAccount}/line/{serviceName}/options
→ displayNumber, displayNumberType

# Modifier le numéro affiché
PUT /telephony/{billingAccount}/line/{serviceName}/options
Body: {
  displayNumber: "+33172100050",
  displayNumberType: "alias"
}

# Alias disponibles
GET /telephony/{billingAccount}/alias
→ Liste des alias du groupe

# Valider numéro externe
POST /telephony/{billingAccount}/line/{serviceName}/validateExternalNumber
Body: { number: "+33612345678" }

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchDisplayNumber
Body: {
  serviceNames: ["line1", "line2"],
  displayNumber: "+33172100050",
  displayNumberType: "alias"
}
```
