## 1. VUE D'ENSEMBLE

**Type:** Page formulaire avec panneau info
**Fonction:** Modifier le mot de passe d'accès au répondeur
**Contexte:** Sécurité du répondeur de la ligne VoIP

## 7. RÈGLES DE VALIDATION

| Règle | Condition |
|-------|-----------|
| Format | 4 chiffres uniquement |
| Caractères | 0-9 |
| Correspondance | password === confirmation |
| Séquences interdites | 0000, 1234, 1111, etc. |

## 8. APIS & ENDPOINTS

```
# Obtenir les infos du répondeur
GET /telephony/{billingAccount}/line/{serviceName}/voicemail
→ {
  accessNumber: "123",
  externalAccessNumber: "+33972...",
  passwordRequired: true
}

# Modifier le mot de passe
PUT /telephony/{billingAccount}/line/{serviceName}/voicemail/password
Body: {
  password: "1234",
  passwordRequired: true
}

# Appliquer à plusieurs lignes
POST /telephony/{billingAccount}/line/batchVoicemailPassword
Body: {
  serviceNames: ["line1", "line2"],
  password: "1234",
  passwordRequired: true
}
```
