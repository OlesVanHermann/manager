## 8. APIS & ENDPOINTS

```
# Lister les événements
GET /telephony/{billingAccount}/line/{serviceName}/events
→ Liste des événements avec dates et types

# Ajouter un événement
POST /telephony/{billingAccount}/line/{serviceName}/events
Body: {
  name: "Fermeture annuelle",
  type: "closure",
  startDate: "2025-01-13",
  endDate: "2025-01-17",
  allDay: true
}

# Modifier un événement
PUT /telephony/{billingAccount}/line/{serviceName}/events/{id}
Body: {
  name: "...",
  startDate: "...",
  endDate: "..."
}

# Supprimer un événement
DELETE /telephony/{billingAccount}/line/{serviceName}/events/{id}

# Jours fériés prédéfinis (France)
GET /telephony/holidays/FR
→ Liste des jours fériés
```
