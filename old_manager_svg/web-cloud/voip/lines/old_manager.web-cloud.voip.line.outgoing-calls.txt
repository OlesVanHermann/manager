## 7. APIS & ENDPOINTS

```
# Historique appels sortants
GET /telephony/{billingAccount}/line/{serviceName}/consumption/outgoingCalls
Query: ?dateFrom=2024-12-01&dateTo=2024-12-31&page=1&pageSize=10
→ {
  calls: [{ date, callee, duration, nature, destination, price }],
  summary: {
    includedCalls: 38,
    includedDuration: "01:45:22",
    outOfPlanCalls: 4,
    outOfPlanDuration: "00:18:33",
    outOfPlanAmount: 2.45
  }
}

# Export CSV
GET /telephony/{billingAccount}/line/{serviceName}/consumption/outgoingCalls/export
Query: ?dateFrom=2024-12-01&dateTo=2024-12-31&format=csv
```
