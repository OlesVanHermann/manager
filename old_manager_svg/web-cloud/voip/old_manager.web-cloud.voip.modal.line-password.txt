# Cahier des Charges : web-cloud/voip - Modal Line Password

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire
**Fonction:** Afficher et modifier le mot de passe SIP d'une ligne
**Contexte:** Accessible depuis les actions rapides ou la gestion de la ligne

## 2. DIMENSIONS

- Modal: 500x380px

## 3. HEADER MODAL (60px)

- Background: #F9FAFB
- Titre: "Mot de passe SIP" (16px, bold)
- Bouton fermer: "x"

## 4. CONTENU

### Information ligne
- Label: "Ligne concernee" (text-muted, 11px)
- Valeur: "{numero} - {description}" (13px, bold)
- Exemple: "0033912345678 - Ligne Accueil"

### Séparateur
Ligne horizontale #E5E7EB

### Mot de passe actuel
- Label: "Mot de passe actuel" (12px, bold)
- Champ lecture: 350x40px (background #F9FAFB)
- Contenu: "****************"
- Icône révéler: "O" (toggle visibility)
- Bouton: "Copier" (92x40px, btn-outline)

### Nouveau mot de passe
- Label: "Nouveau mot de passe" (12px, bold)
- Input: 350x40px
- Placeholder: "Entrez un nouveau mot de passe..."
- Bouton: "Generer" (92x40px, btn-outline)

### Règles mot de passe
- Texte: "Minimum 8 caracteres, 1 majuscule, 1 chiffre, 1 caractere special"
- Style: text-muted, font-size 10px

## 5. FOOTER MODAL

- Séparateur: Ligne #E5E7EB
- Position boutons: Gauche

### Boutons

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Enregistrer | btn-primary | 120x36px |
| Annuler | btn-outline | 100x36px |

## 6. APIS & ENDPOINTS

```
# Récupérer mot de passe actuel (reveal)
POST /telephony/{billingAccount}/line/{serviceName}/sip/password/reveal
→ {
  password: "xY9#kL2mN..."
}

# Générer un nouveau mot de passe
POST /telephony/{billingAccount}/line/{serviceName}/sip/password/generate
→ {
  generatedPassword: "aB3$fG7hJ..."
}

# Modifier le mot de passe
PUT /telephony/{billingAccount}/line/{serviceName}/sip/password
Body: {
  newPassword: "aB3$fG7hJ..."
}
→ {
  success: true,
  message: "Mot de passe mis a jour"
}

# Règles de validation mot de passe
# - Minimum 8 caractères
# - Au moins 1 majuscule
# - Au moins 1 chiffre
# - Au moins 1 caractère spécial (!@#$%^&*)
```
