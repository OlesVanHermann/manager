## 2. LAYOUT ET STRUCTURE
--------------------------------------------------------------------------------

Page de commande de credits SMS avec formulaire en plusieurs etapes.

--------------------------------------------------------------------------------
## 3. FORMULAIRE COMMANDE
--------------------------------------------------------------------------------

### 3.1 Selection du compte

**Champ Compte (select):**
- Label : Selectionnez le compte a crediter
- Options : Liste des comptes SMS existants + "Nouveau Compte"

### 3.2 Format de compte (si nouveau compte)

**Radio buttons:**
- Standard : Compte classique (envoi unitaire ou campagne via Manager/API)
- SMPP marketing : Canal dedie envoi SMS commercial via SMPP
  * Note : Soumis reglementation pays destination (heures ouvrees)
- SMPP transactionnel : Canal dedie SMS non commercial (2FA, notifications)
  * Note : Reglementation moins restrictive (envoi toute heure possible)

### 3.3 Nombre de credits

**Radio buttons avec options predefinies:**
- Options : 100, 500, 1000, 5000, etc.
- Option "Autre" avec input quantite personnalisee
