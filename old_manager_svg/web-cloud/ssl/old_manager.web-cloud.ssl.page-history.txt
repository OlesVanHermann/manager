# Cahier des Charges : web-cloud/ssl - Page History

## 1. VUE D'ENSEMBLE
- **Type** : Page historique certificats
- **Contexte** : Web Cloud SSL
- **Objectif** : Consulter tous les certificats émis pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour, Détails
- **OdsCard** : Certificat actif + historique
- **OdsDatagrid** : Tableau historique
- **OdsBadge** : Type + Statut
- **OdsButton** : Télécharger, Renouveler

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour à {domain}" |
| Titre | "Historique des certificats" |
| Sous-titre | "{domain} • Tous les certificats SSL émis" |

## 5. CERTIFICAT ACTIF
| Propriété | Valeur |
|-----------|--------|
| Bordure | #16A34A 2px |
| Barre latérale | 5px #16A34A |
| Label | "CERTIFICAT ACTIF" (vert) |
| Titre | "{provider} - {domain}" |
| Détails | Émis le / Expire le |
| SHA256 | Empreinte tronquée |
| Actions | Télécharger, Renouveler |

## 6. DATAGRID HISTORIQUE
| Colonne | Description |
|---------|-------------|
| Type | Badge fournisseur coloré |
| Domaine | Nom domaine |
| Émis le | Date émission |
| Expiré le | Date expiration |
| Statut | Badge Actif/Expiré |
| Actions | Lien Détails |

## 7. BADGES TYPE
| Type | Fond | Texte |
|------|------|-------|
| Let's Encrypt | #DCFCE7 | #16A34A |
| Sectigo | #7C3AED | white |

## 8. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Actif | #DCFCE7 | #16A34A |
| Expiré | #F9FAFB | #6B7280 |

## 9. APIS & ENDPOINTS
- GET /ssl/{serviceName}/certificate - Liste certificats
- GET /ssl/{serviceName}/certificate/{id} - Détails certificat
- GET /ssl/{serviceName}/certificate/{id}/download - Télécharger
