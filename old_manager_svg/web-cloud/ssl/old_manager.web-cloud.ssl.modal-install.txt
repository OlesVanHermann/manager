# Cahier des Charges : web-cloud/ssl - Modal Install

## 1. VUE D'ENSEMBLE
- **Type** : Modal installation certificat SSL
- **Contexte** : Web Cloud SSL
- **Objectif** : Installer un certificat SSL externe

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 460px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsMessage** : Info + Warning
- **OdsTextarea** : Certificat, Clé, Chaîne
- **OdsButton** : Annuler, Installer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Installer un certificat SSL" |

|-----------|--------|
| Fond | #EFF6FF |
| Texte | "ℹ️ Installez un certificat SSL acheté auprès d'une autorité externe." |

| Champ | Label | Obligatoire | Placeholder |

| chain | Chaîne de certification | Non | Certificats intermédiaires... |
