# Cahier des Charges : web-cloud/ssl - Modal Generate CSR

## 1. VUE D'ENSEMBLE
- **Type** : Modal génération CSR
- **Contexte** : Web Cloud SSL
- **Objectif** : Générer une demande de signature de certificat

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 460px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsMessage** : Info CSR
- **OdsInput** : Champs texte
- **OdsSelect** : Pays
- **OdsRadio** : Taille clé (style bouton)
- **OdsButton** : Annuler, Générer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Générer une CSR" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. INFO
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | "ℹ️ La CSR est requise pour commander un certificat SSL externe." |

## 6. CHAMPS FORMULAIRE
| Champ | Label | Obligatoire |
|-------|-------|-------------|
| CN | Nom de domaine (CN) * | Oui |
| O | Organisation (O) * | Oui |
| C | Pays (C) * | Oui (select) |
| L | Ville (L) * | Oui |
| Email | Email de contact | Non |

## 7. TAILLE DE CLÉ
| Option | Style si sélectionné |
|--------|---------------------|
| 2048 bits | Bordure grise |
| 4096 bits | Bordure bleue 2px |

## 8. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Générer | Primary #0050D7 | Droite |

## 9. APIS & ENDPOINTS
- POST /ssl/{serviceName}/generateCsr - Générer CSR
