# Cahier des Charges : web-cloud/email-exchange - Pieds de page (Disclaimer)

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des pieds de page email
- **Type** : Page liste avec tableau
- **Objectif** : Configurer les signatures/disclaimers par domaine

## 2. LAYOUT
- Header OVHcloud
- Sidebar Exchange (Pieds de page actif)
- Titre
- Tableau des domaines avec leurs disclaimers

## 3. COMPOSANTS UI
- **OdsTable** : Tableau des disclaimers par domaine
- **OdsBadge** : Statuts (OK, Tâche en cours, Non configuré)
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. TITRE
"Pieds de page"

## 5. TABLEAU
| Colonne | Description |
|---------|-------------|
| Domaine associé | Nom du domaine |
| Contenu | Aperçu du disclaimer |
| Externe uniquement | Oui/Non |
| Statut | Badge état |
| Actions | Menu ⋮ |

## 6. BADGES STATUT
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| OK | Vert (#10B981) | Blanc |
| Tâche en cours | Bleu (#3B82F6) | Blanc |
| Non configuré | Jaune (#F59E0B) | Blanc |

## 7. EXEMPLES DE DONNÉES
| Domaine | Contenu | Externe | Statut |
|---------|---------|---------|--------|
| example.com | Ce message est confidentiel... | Oui | OK |
| company.fr | Signature entreprise... | Non | Tâche en cours |
| test-domain.org | Non configuré | - | Non configuré |

## 8. ACTION MENU (Configuré)
| Option | Couleur | Action |
|--------|---------|--------|
| Modifier | Normal | Ouvre modal modification |
| Supprimer | Rouge (#DC2626) | Supprime le disclaimer |

## 9. ACTION MENU (Non configuré)
| Option | Action |
|--------|--------|
| Configurer | Ouvre modal création |

## 10. PAGINATION
"Affichage 1-3 sur 3 domaines"

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/domain : Lister domaines
- GET /email/exchange/{org}/service/{service}/domain/{domain}/disclaimer : Récupérer disclaimer
- PUT /email/exchange/{org}/service/{service}/domain/{domain}/disclaimer : Modifier disclaimer
- DELETE /email/exchange/{org}/service/{service}/domain/{domain}/disclaimer : Supprimer disclaimer
