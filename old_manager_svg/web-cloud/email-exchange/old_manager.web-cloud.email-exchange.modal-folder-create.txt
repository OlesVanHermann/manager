# Cahier des Charges : web-cloud/email-exchange - Modal Créer un dossier

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un dossier partagé
- **Type** : Formulaire de création
- **Objectif** : Créer un nouveau dossier dans la structure Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Nom du dossier
- Sélection du dossier parent
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x270)
- **OdsInput** : Nom du dossier
- **OdsSelect** : Dossier parent
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un dossier"

## 5. CHAMPS
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Nom du dossier | Input text | Oui (*) | Ex: Projets |
| Dossier parent | Select dropdown | Non | Boîte de réception |

## 6. OPTIONS DOSSIER PARENT
| Dossier | Description |
|---------|-------------|
| Boîte de réception | Dossier par défaut |
| Éléments envoyés | Mails envoyés |
| Brouillons | Brouillons |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le dossier |

## 8. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/publicFolder : Créer dossier
