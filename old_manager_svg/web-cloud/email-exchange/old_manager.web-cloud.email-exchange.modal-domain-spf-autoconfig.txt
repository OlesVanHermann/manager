# Cahier des Charges : web-cloud/email-exchange - Modal Configuration SPF automatique

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique SPF
- **Type** : Modal de confirmation avec preview
- **Objectif** : Configurer automatiquement l'enregistrement SPF

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine concerné
- Bandeau explicatif
- SPF actuel
- Nouvel SPF
- Boutons Annuler / Configurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsMessage** : Info bleu
- **OdsInput** : SPF actuel (gris), nouveau SPF (vert)
- **OdsButton** : Annuler, Configurer

## 4. TITRE
"Configuration SPF automatique"

## 5. DOMAINE
Affiché en gras : "example.com"

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) :
"L'enregistrement SPF protège contre l'usurpation d'identité et améliore la délivrabilité des emails."

## 7. SPF ACTUEL
| Champ | Style | Valeur |
|-------|-------|--------|
| Label | "Enregistrement SPF actuel" | |
| Valeur | Fond gris (#F3F4F6) | v=spf1 include:mx.ovh.com ~all |

## 8. NOUVEL SPF
| Champ | Style | Valeur |
|-------|-------|--------|
| Label | "Nouvel enregistrement" | |
| Valeur | Fond vert (#DCFCE7) | v=spf1 include:spf.mail.ovh.net ~all |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Configurer | Primaire bleu | Configure le SPF |

## 10. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/domain/{domain}/updateDns : Configurer SPF
