# Cahier des Charges : web-cloud/email-exchange - Modal Migration

## 1. VUE D'ENSEMBLE
- **Modal** : Migrer depuis un autre service mail
- **Type** : Assistant de migration
- **Objectif** : Importer des emails depuis un serveur IMAP ou Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Type de service source (IMAP/Exchange)
- Serveur source
- Port et option SSL
- Identifiants source
- Boutons Annuler / Migrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x490)
- **OdsRadio** : Type de source (IMAP/Exchange)
- **OdsInput** : Serveur, Port, Email, Mot de passe
- **OdsCheckbox** : SSL/TLS
- **OdsButton** : Annuler, Migrer

## 4. TITRE
"Migrer depuis un autre service"

## 5. TYPE DE SERVICE SOURCE
| Option | État | Description |
|--------|------|-------------|
| IMAP | Sélectionné (défaut) | Serveur mail standard |
| Exchange | Non sélectionné | Microsoft Exchange |

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Obligatoire |
|-------|------|-------------|-------------|
| Serveur source * | Input text | imap.ancien-provider.com | Oui |
| Port | Input number | 993 | Non |
| SSL/TLS | Checkbox | ☑ | - |
| Email source * | Input email | ancien@email.com | Oui |
| Mot de passe * | Input password | •••••••••••• | Oui |

## 7. CONFIGURATION PAR DÉFAUT
| Service | Port | SSL |
|---------|------|-----|
| IMAP | 993 | Oui |
| Exchange | 443 | Oui |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Migrer | Primaire bleu | Lance la migration |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/migrate : Lancer migration
