# Cahier des Charges : web-cloud/email-exchange - Modal Créer une signature

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une signature email (disclaimer)
- **Type** : Formulaire de création
- **Objectif** : Créer un pied de page automatique pour un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Sélection du domaine
- Zone de texte pour le contenu
- Aide sur les variables
- Options
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x460)
- **OdsSelect** : Domaine
- **OdsTextarea** : Contenu de la signature
- **OdsCheckbox** : Options
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer une signature (disclaimer)"

## 5. CHAMPS
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Domaine | Select dropdown | Oui (*) |
| Contenu de la signature | Textarea | Oui (*) |
| Externes uniquement | Checkbox | Non |

## 6. CONTENU DE LA SIGNATURE
Zone de texte avec variables dynamiques :
```
--
%%cn%%
%%title%%
%%company%%
Tél: %%phone%%
```

## 7. VARIABLES DISPONIBLES
| Variable | Description |
|----------|-------------|
| %%cn%% | Nom complet |
| %%title%% | Titre/Fonction |
| %%company%% | Nom de l'entreprise |
| %%phone%% | Numéro de téléphone |
| %%email%% | Adresse email |

## 8. OPTIONS
☑ "Appliquer aux e-mails sortants externes uniquement"

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée la signature |

## 10. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/domain/{domain}/disclaimer : Créer signature
- GET /email/exchange/{org}/service/{service}/domain : Lister domaines
