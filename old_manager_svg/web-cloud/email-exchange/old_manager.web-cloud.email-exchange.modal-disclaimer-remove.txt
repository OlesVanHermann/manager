# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer une signature

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de signature
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer un disclaimer/signature d'un domaine

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Domaine concerné
- Message informatif
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (380x180)
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer la signature"
- Header fond rouge (#FEF2F2)
- Texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer cette signature ?" |
| Domaine | "Domaine: example.com" (gras) |
| Info | "Les e-mails ne contiendront plus cette signature." |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime la signature |

## 7. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/domain/{domain}/disclaimer : Supprimer signature
