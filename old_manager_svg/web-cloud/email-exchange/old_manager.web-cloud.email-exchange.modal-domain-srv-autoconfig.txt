# Cahier des Charges : web-cloud/email-exchange - Modal Configuration SRV automatique

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique SRV
- **Type** : Modal de confirmation
- **Objectif** : Créer l'enregistrement SRV Autodiscover

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine concerné
- Bandeau explicatif
- Statut actuel
- Boutons Annuler / Configurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x320)
- **OdsMessage** : Info bleu
- **OdsBadge** : Statut
- **OdsButton** : Annuler, Configurer

## 4. TITRE
"Configuration SRV automatique"

## 5. DOMAINE
Affiché en gras : "example.com"

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les enregistrements SRV permettent la configuration automatique de Outlook et des clients mail.
_autodiscover._tcp.example.com"

## 7. STATUT ACTUEL
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Non configuré | #FEE2E2 (rouge clair) | #DC2626 |
| Configuré | #DCFCE7 (vert clair) | #166534 |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Configurer | Primaire bleu | Configure le SRV |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/domain/{domain}/updateDns : Configurer SRV
