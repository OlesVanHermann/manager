# Cahier des Charges : web-cloud/email-exchange - Modal Licence Outlook

## 1. VUE D'ENSEMBLE
- **Modal** : Activation de la licence Outlook
- **Type** : Formulaire de souscription
- **Objectif** : Activer une licence Outlook sur un compte Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (lecture seule)
- Statut actuel
- Bandeau info avec tarification
- Checkbox d'acceptation
- Boutons Annuler / Activer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsInput** : Compte (disabled)
- **OdsBadge** : Statut actuel
- **OdsMessage** : Bandeau info bleu
- **OdsCheckbox** : Acceptation conditions
- **OdsButton** : Annuler, Activer

## 4. TITRE
"Licence Outlook"

## 5. CHAMPS
| Champ | Type | Valeur |
|-------|------|--------|
| Compte | Input disabled | john.doe@example.com |
| Statut actuel | Badge | "Non activée" (rouge) |

## 6. BADGE STATUT
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Non activée | #FEE2E2 (rouge clair) | #DC2626 |
| Activée | #DCFCE7 (vert clair) | #166534 |

## 7. BANDEAU INFO
Fond bleu (#EFF6FF) :
"La licence Outlook permet d'utiliser Microsoft Outlook avec votre compte Exchange.
**Coût: +2,00 € HT/mois**"

## 8. CHECKBOX ACCEPTATION
☐ "J'accepte le surcoût mensuel de la licence Outlook"

## 9. BOUTONS
| Bouton | Type | Action | État |
|--------|------|--------|------|
| Annuler | Secondaire | Ferme la modal | Toujours actif |
| Activer | Primaire bleu | Active la licence | Actif si checkbox coché |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/outlookLicense : Statut licence
- POST /email/exchange/{org}/service/{service}/account/{email}/outlookLicense : Activer licence
- DELETE /email/exchange/{org}/service/{service}/account/{email}/outlookLicense : Désactiver licence
