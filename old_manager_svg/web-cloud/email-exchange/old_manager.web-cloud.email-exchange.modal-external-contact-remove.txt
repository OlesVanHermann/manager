# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer un contact externe

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de contact externe
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer un contact externe de l'annuaire Exchange

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Contact concerné
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (380x180)
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer le contact externe"

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer ce contact externe ?" |
| Contact | contact@external-company.com (gras) |
| Info | "Cette action est irréversible." |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le contact |

## 7. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/externalContact/{id} : Supprimer contact
