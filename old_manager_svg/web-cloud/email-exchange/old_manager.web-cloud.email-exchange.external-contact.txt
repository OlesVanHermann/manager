# Cahier des Charges : web-cloud/email-exchange - Contacts externes

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des contacts externes Exchange
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les contacts dans l'annuaire global

## 2. LAYOUT
- Header OVHcloud
- Sidebar Exchange (Contacts externes actif)
- Titre + menu Actions
- Barre de recherche
- Tableau des contacts

## 3. COMPOSANTS UI
- **OdsButton** : Menu Actions (dropdown)
- **OdsInput** : Recherche adresse
- **OdsTable** : Liste des contacts
- **OdsBadge** : Statuts
- **OdsIcon** : Indicateur annuaire (✓ vert / ✗ rouge)
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. TITRE
"Contacts externes"

## 5. MENU ACTIONS
Dropdown avec options :
| Option | Action |
|--------|--------|
| Ajouter un contact | Ouvre modal création |
| Exporter en CSV | Télécharge la liste |

## 6. RECHERCHE
Placeholder : "Rechercher une adresse..."

## 7. TABLEAU
| Colonne | Description |
|---------|-------------|
| Adresse | Email du contact externe |
| Nom affiché | Nom dans l'annuaire |
| Créé le | Date de création |
| Annuaire | Visible (✓ vert) / Masqué (✗ rouge) |
| Statut | Badge état |
| Actions | Menu ⋮ |

## 8. ICÔNE ANNUAIRE
| État | Icône | Couleur |
|------|-------|---------|
| Visible | ✓ (checkmark) | Vert (#10B981) |
| Masqué | ✗ (croix) | Rouge (#EF4444) |

## 9. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| OK | Vert (#10B981) |
| En création | Bleu (#3B82F6) |

## 10. EXEMPLES DE DONNÉES
| Adresse | Nom | Date | Annuaire | Statut |
|---------|-----|------|----------|--------|
| fournisseur@external.com | Jean Fournisseur | 15 déc. 2024 | ✓ | OK |
| partenaire@partner.org | Marie Partenaire | 10 déc. 2024 | ✗ | OK |
| client@client-company.fr | Paul Client | 5 déc. 2024 | ✓ | En création |

## 11. ACTION MENU
| Option | Couleur | Action |
|--------|---------|--------|
| Modifier | Normal | Ouvre modal modification |
| Supprimer | Rouge (#DC2626) | Supprime le contact |

## 12. PAGINATION
"Affichage 1-3 sur 3 contacts"

## 13. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/externalContact : Lister contacts
- POST /email/exchange/{org}/service/{service}/externalContact : Créer contact
- PUT /email/exchange/{org}/service/{service}/externalContact/{id} : Modifier contact
- DELETE /email/exchange/{org}/service/{service}/externalContact/{id} : Supprimer contact
