# Cahier des Charges : web-cloud/email-exchange - Modal Partager le calendrier

## 1. VUE D'ENSEMBLE
- **Modal** : Partager le calendrier d'un compte
- **Type** : Formulaire de paramètres
- **Objectif** : Définir les permissions de partage du calendrier

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (readonly)
- Sélection du destinataire
- Niveau de permission
- Boutons Annuler / Partager

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsInput** : Compte (readonly)
- **OdsSelect** : Destinataire du partage
- **OdsRadio** : Niveaux de permission
- **OdsButton** : Annuler, Partager

## 4. TITRE
"Partager le calendrier"

## 5. COMPTE CONCERNÉ
| Champ | État | Valeur exemple |
|-------|------|----------------|
| Compte | Readonly grisé | john.doe@example.com |

## 6. DESTINATAIRE
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Partager avec * | Select | assistant@example.com |

## 7. NIVEAUX DE PERMISSION
| Option | État | Description |
|--------|------|-------------|
| Disponibilité uniquement | ○ | Voit créneaux libres/occupés |
| Détails limités (sujet, lieu) | ● Sélectionné | Voit sujet et lieu |
| Tous les détails | ○ | Voit tout le contenu |
| Éditeur (peut modifier) | ○ | Peut créer/modifier événements |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Partager | Primaire bleu | Applique le partage |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account : Liste des comptes
- POST /email/exchange/{org}/service/{service}/account/{email}/calendar/permission : Ajouter partage
