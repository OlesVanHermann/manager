# Cahier des Charges : web-cloud/managed-wordpress - Modal Ajouter domaine

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un domaine
- **Type** : Modal formulaire
- **Objectif** : Associer un nom de domaine personnalisé

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message info
- Champ domaine
- Choix certificat SSL
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Info bleu
- **OdsInput** : Nom de domaine
- **OdsRadio** : Type certificat
- **OdsButton** : Annuler, Ajouter

## 4. MESSAGE INFO
| Type | Couleur | Lignes |
|------|---------|--------|
| Info | Bleu | Associez un nom de domaine personnalisé à votre |
| | | site WordPress managé. |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom de domaine | Input | Oui | www.mondomaine.com |

## 6. OPTIONS CERTIFICAT SSL
| Option | Sélection | Description |
|--------|-----------|-------------|
| Let's Encrypt (gratuit) | ● Sélectionné | Certificat automatique |
| Certificat personnalisé | ○ | Import certificat |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /hosting/wordpress/{service}/website/{id}/domain - Ajouter domaine
