# Cahier des Charges : web-cloud/managed-wordpress - Liste services

## 1. VUE D'ENSEMBLE
- **Page** : WordPress managé - Liste des services
- **Type** : Page liste principale
- **Objectif** : Afficher tous les services WordPress managé

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar navigation (WEB CLOUD > WordPress managé)
- Titre + description
- Datagrid des services

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste des services
- **OdsBadge** : Status (Activé, En cours)
- **OdsActionMenu** : Menu contextuel

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| RESSOURCE | Link | Identifiant du service |
| OFFRE | Text | Type d'offre (Pro, Starter) |
| STATUT | Badge | Activé, En cours |
| SITES INSTALLÉS | Text | Quota (3 / 5) |
| ACTIONS | ActionMenu | Menu ⋮ |

## 5. BADGES STATUT
| Badge | Couleur | Description |
|-------|---------|-------------|
| Activé | Vert (success) | Service opérationnel |
| En cours | Bleu (info) | Création en cours |

## 6. NAVIGATION
- Clic sur ressource → Dashboard du service

## 7. APIS & ENDPOINTS
- GET /hosting/wordpress - Liste des services WordPress managé
