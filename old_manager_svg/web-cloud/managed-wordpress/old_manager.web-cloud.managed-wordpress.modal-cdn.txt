# Cahier des Charges : web-cloud/managed-wordpress - Modal CDN

## 1. VUE D'ENSEMBLE
- **Page** : Configuration CDN
- **Type** : Modal configuration
- **Objectif** : Configurer le CDN du site

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message info
- Checkbox activer CDN
- Options de cache
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Info bleu
- **OdsCheckbox** : Activer CDN, options cache
- **OdsButton** : Annuler, Enregistrer

## 4. MESSAGE INFO
| Type | Couleur | Lignes |
|------|---------|--------|
| Info | Bleu | Le CDN améliore les performances de votre site |
| | | en distribuant le contenu à travers le monde. |

## 5. ACTIVATION CDN
| Option | État par défaut |
|--------|-----------------|
| Activer le CDN | ✓ Coché |

## 6. OPTIONS DE CACHE
| Option | État par défaut | Description |
|--------|-----------------|-------------|
| Mise en cache des images | ✓ Coché | Images statiques |
| Mise en cache CSS/JS | ✓ Coché | Fichiers statiques |
| Mise en cache des pages (avancé) | Non coché | Pages HTML |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Enregistrer | Primary | Droite |

## 8. APIS & ENDPOINTS
- GET /hosting/wordpress/{service}/website/{id}/cdn - Config CDN
- PUT /hosting/wordpress/{service}/website/{id}/cdn - Modifier CDN
