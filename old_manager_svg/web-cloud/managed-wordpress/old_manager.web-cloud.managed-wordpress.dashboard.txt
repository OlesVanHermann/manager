# Cahier des Charges : web-cloud/managed-wordpress - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : WordPress managé - Dashboard du service
- **Type** : Page dashboard avec onglets
- **Objectif** : Gérer un service WordPress managé

## 2. STRUCTURE PAGE
- Lien retour (← Retour)
- Titre du service (wp-hosting-abc123)
- Onglets (Mes sites, Tâches)
- Boutons d'action (Créer, Importer, Gérer)
- Quota sites (Sites : 3/5)
- Datagrid des sites

## 3. COMPOSANTS UI
- **OdsTabs** : Mes sites, Tâches
- **OdsButton** : Créer (primary), Importer, Gérer (outline)
- **OdsDatagrid** : Liste des sites
- **OdsBadge** : Status (Actif)
- **OdsCheckbox** : Sélection multiple
- **OdsActionMenu** : Supprimer

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Mes sites | Liste des sites WordPress |
| Tâches | Historique des opérations |

## 5. BOUTONS ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Créer | Primary | Page création site |
| Importer | Outline | Page import site |
| Gérer | Outline | Lien externe gestion |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| ☐ | Checkbox | Sélection |
| NOM DU SITE | Text | Nom du site |
| URL | Link | FQDN (monblog.ovh.net) |
| STATUT | Badge | Actif |
| ACTIONS | ActionMenu | Supprimer |

## 7. ACTIONMENU
| Option | Action |
|--------|--------|
| Supprimer | Modal delete |

## 8. APIS & ENDPOINTS
- GET /hosting/wordpress/{service} - Détails service
- GET /hosting/wordpress/{service}/website - Liste sites
- DELETE /hosting/wordpress/{service}/website/{id} - Supprimer site
