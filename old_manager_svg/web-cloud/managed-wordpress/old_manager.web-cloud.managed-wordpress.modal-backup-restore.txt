# Cahier des Charges : web-cloud/managed-wordpress - Modal Restaurer

## 1. VUE D'ENSEMBLE
- **Page** : Restaurer une sauvegarde
- **Type** : Modal formulaire warning
- **Objectif** : Restaurer un site depuis une sauvegarde

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Warning (écrasement données)
- Site concerné (readonly)
- Select sauvegarde
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Warning orange
- **OdsInput** : Site (disabled)
- **OdsSelect** : Sauvegarde à restaurer
- **OdsButton** : Annuler, Restaurer

## 4. MESSAGE WARNING
| Type | Couleur | Lignes |
|------|---------|--------|
| Warning | Orange | La restauration écrasera les données actuelles. |
| | | Une sauvegarde de l'état actuel sera créée. |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Site WordPress | Input disabled | - | monsite.cluster.ovh.net |
| Sauvegarde à restaurer | Select | Oui | Date + type + taille |

## 6. OPTIONS SAUVEGARDE
| Format option |
|---------------|
| 29/12/2024 02:00 - Automatique (132 Mo) |
| 28/12/2024 02:00 - Automatique (128 Mo) |
| 27/12/2024 14:30 - Manuelle (125 Mo) |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Restaurer | Warning (orange) | Droite |

## 8. APIS & ENDPOINTS
- POST /hosting/wordpress/{service}/website/{id}/restore - Restaurer
