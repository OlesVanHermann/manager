# Cahier des Charges : web-cloud/managed-wordpress - Liste services

## 1. VUE D'ENSEMBLE
- **Page** : Managed WordPress - Liste services
- **Type** : Page liste principale
- **Objectif** : Afficher tous les services WordPress managé

## 2. STRUCTURE PAGE
- Breadcrumb (Hébergements > Managed WordPress)
- Titre
- Datagrid des services

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsDatagrid** : Liste services
- **OdsBadge** : Status
- **OdsLink** : Lien vers détails

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Ressource | Link | Identifiant (mwp-abc123) |
| Offre | Text | 5 sites, 10 sites |
| Statut | Badge | READY, CREATING |
| Sites installés | Text | Quota (3 / 5) |

## 5. BADGES STATUT
| Badge | Couleur | Description |
|-------|---------|-------------|
| READY | Vert (success) | Service actif |
| CREATING | Orange (warning) | En création |

## 6. PAGINATION
- Affichage du nombre de résultats

## 7. APIS & ENDPOINTS
- GET /hosting/wordpress - Liste services
