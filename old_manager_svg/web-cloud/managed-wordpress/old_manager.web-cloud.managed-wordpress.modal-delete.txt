# Cahier des Charges : web-cloud/managed-wordpress - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Page** : Supprimer mon site
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un site WordPress

## 2. STRUCTURE MODAL
- Barre rouge en haut
- Icône warning dans cercle rouge
- Titre + bouton fermer
- Question de confirmation
- Nom du site (fond gris)
- Warning irréversible
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsText** : Titre, confirmation, warning
- **OdsButton** : Annuler, Supprimer (danger)

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Barre supérieure | Rouge (#DC2626) |
| Icône | ⚠ dans cercle rouge |
| Titre | Supprimer mon site |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | Êtes-vous sûr de vouloir supprimer le site suivant ? |
| Site | www.monsite-wordpress.com (fond #F3F4F6) |
| Warning | ⚠ Cette action est irréversible. Toutes les données seront supprimées. |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Supprimer | Danger (rouge) | Droite |

## 7. APIS & ENDPOINTS
- DELETE /hosting/wordpress/{service}/website/{id} - Supprimer site
