# Cahier des Charges : web-cloud/managed-wordpress - Modal SSL

## 1. VUE D'ENSEMBLE
- **Page** : Certificat SSL
- **Type** : Modal informations SSL
- **Objectif** : Gérer le certificat SSL du site

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message statut (succès)
- Domaine (readonly)
- Date expiration
- Checkbox forcer HTTPS
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Succès vert
- **OdsInput** : Domaine (disabled)
- **OdsText** : Date expiration
- **OdsCheckbox** : Forcer HTTPS
- **OdsButton** : Régénérer, Enregistrer

## 4. MESSAGE STATUT
| Type | Couleur | Lignes |
|------|---------|--------|
| Succès | Vert | ✓ Certificat Let's Encrypt actif |
| | | Renouvellement automatique activé |

## 5. INFORMATIONS
| Champ | Type | Valeur |
|-------|------|--------|
| Domaine | Input disabled | monsite.cluster.ovh.net |
| Expire le | Text bold | 28/03/2025 |

## 6. OPTIONS
| Option | État | Description |
|--------|------|-------------|
| Forcer HTTPS | ✓ Coché | Redirection HTTP→HTTPS |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Régénérer | Outline | Gauche |
| Enregistrer | Primary | Droite |

## 8. APIS & ENDPOINTS
- GET /hosting/wordpress/{service}/website/{id}/ssl - Info SSL
- POST /hosting/wordpress/{service}/website/{id}/ssl/regenerate - Régénérer
- PUT /hosting/wordpress/{service}/website/{id}/ssl - Modifier
