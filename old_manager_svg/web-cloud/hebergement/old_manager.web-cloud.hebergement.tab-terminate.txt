# Cahier des Charges : web-cloud/hebergement - Tab Suppression

## 1. RESUME EXECUTIF

### Perimetre
Onglet de résiliation/suppression de l'hébergement.

### Objectif fonctionnel
- Afficher les avertissements
- Récapituler le service
- Confirmer la suppression

---

## 2. LAYOUT

### Structure message danger + récap + warning + boutons

---

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}
DELETE /hosting/web/{serviceName}/terminate
POST /hosting/web/{serviceName}/confirmTermination
```

### Modele
```typescript
interface TerminationInfo {
  serviceName: string;
  offer: string;
  creationDate: string;
  expirationDate: string;
}

interface TerminationRequest {
  confirmation: boolean;
}
```
