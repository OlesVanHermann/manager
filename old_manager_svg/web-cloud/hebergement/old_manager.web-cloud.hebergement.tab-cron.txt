# Cahier des Charges : web-cloud/hebergement - Tab Cron

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des tâches planifiées (Cron).

### Objectif fonctionnel
- Voir les tâches cron configurées
- Ajouter de nouvelles tâches
- Activer/désactiver des tâches

|---------|-------------|

### Toolbar
| Element | Description |
|---------|-------------|
| Bouton | + Ajouter une tâche Cron (primary) |

### Tableau cron
| Colonne | Description |
|---------|-------------|
| Commande | Chemin script (monospace) |
| Fréquence | Expression cron ou texte |
| Langage | PHP version / Autre |
| E-mail | Email notification |
| Description | Description libre |
| État | Badge Actif / Désactivé |
| Actions | Menu ⋮ |

### Menu actions (⋮)
| Option | Description |
|--------|-------------|
| Modifier | Edit |
| Activer/Désactiver | Toggle |
| Supprimer | Destructif (rouge) |

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/cron
POST /hosting/web/{serviceName}/cron
PUT /hosting/web/{serviceName}/cron/{id}
DELETE /hosting/web/{serviceName}/cron/{id}
```

### Modele
```typescript
interface HostingCron {
  id: string;
  command: string;
  frequency: string;
  language: string;
  email?: string;
  description?: string;
  state: 'enabled' | 'disabled';
}
```
