# Cahier des Charges : web-cloud/hebergement - Tab Multi domaines indépendants

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des domaines indépendants (fonctionnalité legacy).

### Objectif fonctionnel
- Gérer les domaines avec IP dédiée
- Ajouter/modifier/supprimer des domaines
- Voir l'état des domaines

---

## 2. LAYOUT

### Structure warning legacy + bouton ajout + tableau

---

## 3. COMPOSANTS UI

### Message warning (legacy)
| Element | Description |
|---------|-------------|
| Type | Warning (jaune) |
| Ligne 1 | Cette fonctionnalité est disponible uniquement pour les offres legacy |
| Ligne 2 | Pour les nouvelles offres, utilisez la fonctionnalité Multisite |

### Bouton ajout
| Element | Description |
|---------|-------------|
| Type | Primary |
| Texte | + Ajouter un domaine |

### Tableau domaines
| Colonne | Description |
|---------|-------------|
| Domaine | Nom du domaine |
| Dossier racine | Chemin du répertoire |
| IPv4 | Adresse IP dédiée |
| État | Badge Actif |
| Actions | Menu ⋮ |

### Menu actions (⋮)
| Option | Description |
|--------|-------------|
| Modifier | Éditer la configuration |
| Suspendre | Désactiver temporairement |
| Réouvrir | Réactiver après suspension |
| Supprimer | Destructif (rouge) |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/indy
POST /hosting/web/{serviceName}/indy
PUT /hosting/web/{serviceName}/indy/{domain}
DELETE /hosting/web/{serviceName}/indy/{domain}
```

### Modele
```typescript
interface IndyDomain {
  domain: string;
  path: string;
  ipv4: string;
  state: 'active' | 'suspended';
}
```
