# Cahier des Charges : web-cloud/hosting - Page Paramètres CDN

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration complète
- **Contexte** : Gestion du CDN partagé
- **Objectif** : Configurer les options de performance, cache et sécurité du CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 1100px |
| Padding | 30px |

## 3. COMPOSANTS UI
- **OdsText** : Titres de sections
- **OdsToggle** : Options on/off
- **OdsSelect** : Menus déroulants
- **OdsDatagrid** : Liste des règles de cache
- **OdsButton** : Actions et sauvegarde
- **OdsActionMenu** : Actions par règle

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Paramètres CDN - {domaine}" |
| Fond | Blanc |
| Bouton | ← Retour |

## 5. CONTENU

### Section 1 : Options de performance
| Option | Type | Description | État exemple |
|--------|------|-------------|--------------|
| HTTP/2 | OdsToggle | Améliore les performances de chargement | Activé (vert #10B981) |
| Compression Brotli | OdsToggle | Compression optimisée des ressources | Activé |
| Always Online | OdsToggle | Cache en cas d'indisponibilité | Activé |
| Mode développeur | OdsToggle | Désactive le cache temporairement | Désactivé (gris #E5E7EB) |

### Section 2 : Règles de cache
| Colonne | Description |
|---------|-------------|
| Nom | Nom de la règle (ex: Default rule) |
| Type | Standard, Aggressive, Bypass |
| TTL | Durée de cache (ex: 24 heures) |
| Statut | Badge Actif/Inactif |
| Actions | Menu ⋮ (Modifier, Supprimer) |

#### Bouton d'action
| Bouton | Style | Action |
|--------|-------|--------|
| + Ajouter une règle | Primary (bleu #0050D7) | Ouvre modal création |

### Section 3 : Options de sécurité
| Option | Type | Description | Valeur/État |
|--------|------|-------------|-------------|
| HTTPS Redirect | OdsSelect | Redirection HTTP vers HTTPS | 301, 302, Off |
| HSTS | OdsToggle | Strict Transport Security | Activé |
| WAF | OdsToggle | Pare-feu applicatif | Activé |
| CORS | OdsButton | Cross-Origin Resource Sharing | Bouton "Configurer" |

### Styles des badges
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Actif | #DCFCE7 | #166534 |
| Inactif | #F3F4F6 | #6B7280 |

## 6. FOOTER PAGE
| Bouton | Style | Action |
|--------|-------|--------|
| Enregistrer | Primary (bleu #0050D7) | Sauvegarde les modifications |

## 7. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/domains/{domain}/options - Options CDN
- PUT /cdn/dedicated/{serviceName}/domains/{domain}/options - Modifier options
- GET /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules - Règles de cache
- POST /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules - Créer règle
- PUT /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules/{id} - Modifier règle
- DELETE /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules/{id} - Supprimer règle
