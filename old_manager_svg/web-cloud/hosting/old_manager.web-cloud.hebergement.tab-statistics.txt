# Cahier des Charges : web-cloud/hebergement - Tab Statistiques

## 1. RESUME EXECUTIF

### Perimetre
Onglet de visualisation des statistiques de fréquentation.

### Objectif fonctionnel
- Voir l'aperçu rapide des métriques
- Accéder aux statistiques détaillées (Urchin/AWStats)
- Visualiser le trafic

---

## 2. LAYOUT

### Structure message info + 4 KPI cards + graphique

---

## 3. COMPOSANTS UI

### Message info
| Element | Description |
|---------|-------------|
| Type | Info (bleu) |
| Titre | Accédez à vos statistiques détaillées |
| Description | Les statistiques sont disponibles via Urchin/AWStats |
| Bouton | Accéder aux statistiques (primary) |

### Cartes KPI (4 colonnes)
| KPI | Description |
|-----|-------------|
| Visiteurs uniques | Nombre + évolution % |
| Pages vues | Nombre + évolution % |
| Bande passante | Valeur en Go + évolution % |
| Erreurs 404 | Nombre + évolution % |

### Indicateurs évolution
| Couleur | Signification |
|---------|---------------|
| Vert (#10B981) | Évolution positive |
| Rouge (#DC2626) | Évolution négative |

### Graphique trafic
| Element | Description |
|---------|-------------|
| Titre | Trafic des 7 derniers jours |
| Type | Bar chart |
| Axes | Jours de la semaine / Nombre visites |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/statistics
GET /hosting/web/{serviceName}/statistics/chart
```

### Modele
```typescript
interface HostingStatistics {
  period: '30days' | '7days';
  uniqueVisitors: number;
  pageViews: number;
  bandwidth: number;
  errors404: number;
  trends: {
    visitors: number;
    pageViews: number;
    bandwidth: number;
    errors: number;
  };
}

interface StatisticsChart {
  labels: string[];
  values: number[];
}
```
