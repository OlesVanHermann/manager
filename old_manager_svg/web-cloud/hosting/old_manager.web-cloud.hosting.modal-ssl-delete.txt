# Cahier des Charges : web-cloud/hosting - Modal Supprimer certificat SSL

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression du certificat SSL
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer le certificat SSL de l'hébergement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Question de confirmation
- Bandeau warning jaune
- Note informative
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. QUESTION
"Êtes-vous sûr de vouloir supprimer votre certificat SSL ?"

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron (#92400E) :
- "⚠ Attention : Votre certificat Let's Encrypt sera supprimé."
- "Vos sites ne seront plus accessibles en HTTPS."

## 6. NOTE INFORMATIVE
"Après suppression, vous pourrez commander un nouveau certificat."

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/ssl : Supprimer certificat SSL
