# Cahier des Charges : web-cloud/hosting - Modal Ajouter domaine OVHcloud

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout multisite
- **URL** : N/A (composant modal)
- **Objectif** : Attacher un domaine OVHcloud (avec sous-domaine) à l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 650px |
| Hauteur | 540px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Sous-domaine
- **OdsSelect** : Domaine parent
- **OdsCheckbox** : Option www, SSL, CDN, Firewall, IP géolocalisée, Logs séparés
- **OdsInput** : Dossier racine
- **OdsSelect** : Moteur d'exécution
- **OdsButton** : Actions

## 4. FORMULAIRE NOM DE DOMAINE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Sous-domaine | Input text | Non | Préfixe optionnel |
| Domaine parent | Select (readonly) | Oui | .mondomaine.com |
| Ajouter www | Checkbox | Non | Coché par défaut |

## 5. FORMULAIRE CONFIGURATION
| Champ | Type | Requis | Default |
|-------|------|--------|---------|
| Dossier racine | Input text | Non | ./www |
| Moteur d'exécution | Select dropdown | Non | PHP 8.2 |

## 6. OPTIONS CHECKBOXES
| Option | État par défaut |
|--------|-----------------|
| SSL | Coché (✓) |
| CDN | Coché (✓) |
| Firewall | Non coché |
| IP géolocalisée | Non coché |
| Logs séparés | Non coché |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Ajouter | Primary | Ajouter domaine OVHcloud |

## 8. APIS & ENDPOINTS
- GET /domain - Liste domaines OVHcloud du compte
- POST /hosting/web/{serviceName}/attachedDomain - Ajouter domaine
