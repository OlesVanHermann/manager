# Cahier des Charges : web-cloud/hosting - Modal Supprimer base de données

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation suppression base de données
- **Type** : Modal de confirmation danger
- **Objectif** : Confirmer la suppression définitive

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Question de confirmation
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. QUESTION
"Êtes-vous sûr de vouloir supprimer la base de données **mydb_hosting123** ?"

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron (#92400E) :
- "⚠ Attention : Cette action est irréversible."
- "Toutes les données seront définitivement supprimées."
- "Pensez à effectuer une sauvegarde avant de continuer."

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Supprimer (danger rouge, à droite)

## 7. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/database/{name} : Supprimer base
