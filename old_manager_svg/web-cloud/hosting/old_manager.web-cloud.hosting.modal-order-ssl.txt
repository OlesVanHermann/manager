# Cahier des Charges : web/hosting - Modal commander certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire avec choix de type
- **Contexte** : Hébergement web - SSL
- **Objectif** : Commander ou importer un certificat SSL

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 650px |
| Hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, descriptions
- **OdsRadio** : Choix type de certificat
- **OdsCard** : Options de certificat
- **OdsSelect** : Domaine (pour Sectigo)
- **OdsMessage** : Warning et info
- **OdsButton** : Annuler, Suivant

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander un certificat SSL" |

## 5. CHOIX DU TYPE DE CERTIFICAT

### Option 1 : Gratuit (Let's Encrypt) - Sélectionnée
| Propriété | Valeur |
|-----------|--------|
| Style | Carte avec bordure bleue (sélectionnée) |
| Radio | Sélectionné |
| Titre | "Gratuit (Let's Encrypt)" |
| Description | "Certificat DV automatique, renouvellement automatique tous les 90 jours" |

### Option 2 : Payant (Sectigo)
| Propriété | Valeur |
|-----------|--------|
| Style | Carte avec bordure grise |
| Radio | Non sélectionné |
| Titre | "Payant (Sectigo)" |
| Description | "Certificat DV avec validation étendue, support et garantie inclus" |
| Sous-option | Sélecteur de domaine (visible si sélectionné) |

### Option 3 : Importer
| Propriété | Valeur |
|-----------|--------|
| Style | Carte avec bordure grise |
| Radio | Non sélectionné |
| Titre | "Importer un certificat" |
| Description | "Importez votre propre certificat SSL avec sa clé privée" |

## 6. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ L'activation du SSL peut prendre jusqu'à 48 heures. Votre site sera accessible en HTTPS." |

## 7. MESSAGE INFO
| Type | Contenu |
|------|---------|
| Info (bleu) | "ℹ Les certificats gratuits ne couvrent que les domaines de moins de 64 caractères." |

## 8. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Suivant | Primary (bleu) | Étape suivante |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ssl - Certificat actuel
- POST /hosting/web/{serviceName}/ssl - Commander certificat Let's Encrypt
- POST /hosting/web/{serviceName}/ssl/import - Importer certificat
