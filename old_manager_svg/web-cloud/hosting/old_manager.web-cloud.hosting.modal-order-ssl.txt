# Cahier des Charges : web-cloud/hosting - Modal Commander SSL

## 1. VUE D'ENSEMBLE
- **Modal** : Commande de certificat SSL
- **Type** : Formulaire de sélection
- **Objectif** : Choisir et commander un certificat SSL

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Description
- Grille 3 cartes offres
- Bandeau info
- Section domaines couverts
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsCard** : Cartes offres (sélection bordure bleue)
- **OdsRadio** : Sélection offre
- **OdsBadge** : "Recommandé" sur Let's Encrypt
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Commander

## 4. DESCRIPTION
"Sélectionnez le type de certificat SSL que vous souhaitez commander pour votre hébergement."

## 5. OFFRES SSL
### Let's Encrypt (Gratuit) - Sélectionné
- Type : DV
- Renouvellement automatique
- Multi-domaines inclus
- Installation automatique
- Badge : "Recommandé"

### Sectigo DV (49,99 € HT/an)
- Type : DV
- Garantie 10 000 €
- Support prioritaire
- Sceau de confiance

### Sectigo EV (99,99 € HT/an)
- Type : EV (barre verte)
- Garantie 1 000 000 €
- Validation entreprise
- Sceau de confiance premium

## 6. BANDEAU INFO
"ℹ️ À propos de Let's Encrypt"
- "Let's Encrypt est une autorité de certification gratuite et automatisée."
- "Le certificat sera automatiquement renouvelé tous les 90 jours et couvrira tous vos multisites."

## 7. DOMAINES COUVERTS
Le certificat Let's Encrypt couvrira automatiquement :
- www.example.com
- example.com
- blog.example.com
- + 2 autres domaines

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Commander (primaire)

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ssl/available : Offres disponibles
- POST /hosting/web/{serviceName}/ssl : Commander certificat
