# Cahier des Charges : web-cloud/hosting - Modal Ajouter domaine externe

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout multisite
- **URL** : N/A (composant modal)
- **Objectif** : Attacher un domaine externe (non OVHcloud) à l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Nom de domaine
- **OdsCheckbox** : Option www
- **OdsInput** : Dossier racine
- **OdsSelect** : Moteur d'exécution
- **OdsMessage** : Info DNS (bleu)
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nom de domaine | Input text | Oui | exemple.com |
| Ajouter www | Checkbox | Non | Coché par défaut |
| Dossier racine | Input text | Non | ./www |
| Moteur d'exécution | Select dropdown | Non | PHP 8.2 |

## 5. INFO DNS
| Type | Message |
|------|---------|
| Info | "ℹ Vous devrez configurer les DNS de ce domaine pour pointer vers l'hébergement." |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Ajouter | Primary | Ajouter domaine externe |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/attachedDomain - Ajouter domaine externe
