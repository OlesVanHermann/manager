# Cahier des Charges : web-cloud/hosting - Modal Commander SSL

## 1. VUE D'ENSEMBLE
- **Modal** : Commande de certificat SSL
- **Type** : Formulaire avec sélection cartes
- **Objectif** : Choisir et commander un certificat SSL

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Description
- Grille 2 cartes offres
- Sélection domaines (checkboxes)
- Bandeau info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsCard** : 2 cartes offres (sélection bordure bleue)
- **OdsBadge** : "GRATUIT" (vert), "PAYANT" (bleu)
- **OdsCheckbox** : Sélection domaines
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Commander

## 4. DESCRIPTION
"Choisissez le type de certificat SSL pour votre hébergement"

## 5. OFFRES SSL

### Let's Encrypt (Gratuit) - Sélectionné
- Badge : "GRATUIT" (fond vert #DEF7EC)
- Caractéristiques :
  - Certificat SSL gratuit
  - Validation de domaine (DV)
  - Renouvellement automatique
  - Compatible tous navigateurs
  - Délai: quelques minutes
- Prix : **0,00 € HT**
- Radio : ● Sélectionné (coche bleue)

### Sectigo DV (Payant)
- Badge : "PAYANT" (fond bleu #DBEAFE)
- Caractéristiques :
  - Certificat SSL professionnel
  - Validation de domaine (DV)
  - Garantie 10 000 €
  - Support prioritaire
  - Délai: 1-2 jours
- Prix : **49,99 € HT/an**
- Radio : ○ Non sélectionné

## 6. DOMAINES À SÉCURISER
| Domaine | Sélectionné |
|---------|-------------|
| monsite.example.com | ✓ Oui |
| www.monsite.example.com | ✓ Oui |
| blog.monsite.example.com | ○ Non |

## 7. BANDEAU INFO
Fond bleu (#E8F0FE) :
"ℹ️ Le certificat Let's Encrypt sera généré automatiquement après validation DNS."

## 8. BOUTONS FOOTER
- Annuler (secondaire gris)
- Commander (primaire bleu)

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain : Liste domaines
- POST /hosting/web/{serviceName}/ssl : Commander certificat SSL
  - Body : { provider, domains }
