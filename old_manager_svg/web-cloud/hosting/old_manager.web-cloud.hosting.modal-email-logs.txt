# Cahier des Charges : web-cloud/hosting - Modal Logs emails automatisés

## 1. VUE D'ENSEMBLE
- **Type** : Modal tableau avec filtres
- **Contexte** : Emails automatisés
- **Objectif** : Consulter l'historique des emails envoyés par l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Filtres (statut, période)
- **OdsDatagrid** : Tableau des emails
- **OdsBadge** : Statut
- **OdsPagination** : Navigation

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Logs d'emails automatisés" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Filtres
| Filtre | Type | Options |
|--------|------|---------|
| Statut | OdsSelect | Tous les statuts, Envoyé, Échec, En attente |
| Période | OdsSelect | Dernières 24h, 7 jours, 30 jours |

### Tableau des emails
| Colonne | Type | Description |
|---------|------|-------------|
| Date | Texte | Format: DD/MM HH:mm |
| Destinataire | Texte | Adresse email |
| Sujet | Texte | Sujet de l'email |
| Statut | Badge | État de l'envoi |

### Badges statut
| État | Fond | Texte |
|------|------|-------|
| Envoyé | #DCFCE7 | #166534 |
| Échec | #FEE2E2 | #DC2626 |
| En attente | #FEF3C7 | #92400E |

### Pagination
| Propriété | Valeur |
|-----------|--------|
| Format | "{total} emails | Page {current}/{total}" |
| Navigation | ← Précédent | Suivant → |

## 6. FOOTER MODAL
Pas de footer (pagination intégrée au contenu)

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/email - Liste des emails avec filtres
- GET /hosting/web/{serviceName}/email/bounces - Emails en échec
