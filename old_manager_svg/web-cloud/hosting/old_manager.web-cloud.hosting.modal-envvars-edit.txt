# Cahier des Charges : web-cloud/hosting - Modal Modifier variable environnement

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification variable
- **URL** : N/A (composant modal)
- **Objectif** : Modifier la valeur d'une variable d'environnement existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Info variable (lecture seule)
- **OdsInput** : Nouvelle valeur
- **OdsButton** : Actions

## 4. INFORMATIONS AFFICHÉES (lecture seule)
| Champ | Exemple |
|-------|---------|
| Variable | APP_ENV |
| Type | (string) |

## 5. FORMULAIRE
| Champ | Type | Requis | Valeur actuelle |
|-------|------|--------|-----------------|
| Valeur | Input text | Oui | production |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer modification |

## 7. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/envVar/{key} - Modifier variable
