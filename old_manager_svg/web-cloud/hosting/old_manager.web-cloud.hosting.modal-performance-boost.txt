# Cahier des Charges : web-cloud/hosting - Modal Boost performance

## 1. VUE D'ENSEMBLE
- **Type** : Modal de commande avec sélection
- **Contexte** : Gestion des ressources
- **Objectif** : Activer un boost temporaire des ressources (CPU/RAM)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Explication du boost
- **OdsCard** : Ressources actuelles
- **OdsRadio** : Niveaux de boost (x2, x4)
- **OdsSelect** : Durée
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le boost de performance" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Le boost multiplie temporairement les ressources de votre" |
| Ligne 2 | "hébergement pour gérer un pic de trafic." |

### Ressources actuelles
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Ressources actuelles" |
| Fond cartes | #F9FAFB |

| Ressource | Valeur |
|-----------|--------|
| CPU | 2 vCPU |
| RAM | 4 Go |

### Niveaux de boost
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Choisir le niveau de boost" |

| Niveau | Ressources | Prix |
|--------|------------|------|
| Boost x2 | 4 vCPU, 8 Go RAM | +9,99€/jour |
| Boost x4 | 8 vCPU, 16 Go RAM | +19,99€/jour |

### Style sélection
| Propriété | Valeur |
|-----------|--------|
| Sélectionné | Bordure bleue (#0050D7), radio rempli |
| Non sélectionné | Bordure grise, radio vide |

### Durée du boost
| Champ | Type | Options |
|-------|------|---------|
| Durée du boost | OdsSelect | 1 jour, 3 jours, 7 jours, 14 jours, 30 jours |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Activer | Primary (bleu #0050D7) | Active le boost |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/boostHistory - Historique des boosts
- POST /order/cart/{cartId}/webHosting/boost - Commander un boost
