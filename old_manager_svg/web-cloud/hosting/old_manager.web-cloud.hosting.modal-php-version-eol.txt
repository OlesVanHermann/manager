# Cahier des Charges : web-cloud/hosting - Modal PHP version obsolète

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'alerte avec action
- **Contexte** : Sécurité et maintenance
- **Objectif** : Alerter sur une version PHP obsolète et proposer la mise à jour

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement de sécurité
- **OdsText** : Informations version
- **OdsBadge** : Version actuelle (danger)
- **OdsSelect** : Nouvelle version
- **OdsCheckbox** : Confirmation test
- **OdsButton** : Plus tard, Mettre à jour

## 4. HEADER MODAL (DANGER)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Version PHP obsolète" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'alerte
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEE2E2 |
| Icône | ⚠ |
| Ligne 1 | "Votre hébergement utilise PHP 7.4 qui n'est plus supporté." |
| Ligne 2 | "Cette version ne reçoit plus de mises à jour de sécurité." |
| Ligne 3 | "Nous vous recommandons fortement de migrer vers PHP 8.2." |

### Informations version
| Champ | Valeur | Style |
|-------|--------|-------|
| Version actuelle | PHP 7.4 | Badge rouge (#FEE2E2/#DC2626) |
| Fin de support | 28 novembre 2022 | Texte normal |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Nouvelle version recommandée | OdsSelect | Oui | PHP 8.2 (Recommandé) |

### Confirmation
| Propriété | Valeur |
|-----------|--------|
| Type | OdsCheckbox |
| Label | "J'ai testé la compatibilité de mon site" |
| Requis | Non mais recommandé |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Plus tard | Secondary (bordure grise) | Ferme modal |
| Mettre à jour | Primary (bleu #0050D7) | Lance la mise à jour |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ovhConfig - Version PHP actuelle
- PUT /hosting/web/{serviceName}/ovhConfig - Mettre à jour PHP
