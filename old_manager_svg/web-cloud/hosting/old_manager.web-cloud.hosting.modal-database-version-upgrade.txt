# Cahier des Charges : web-cloud/hosting - Modal Upgrade version BDD

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'action irréversible
- **Contexte** : Gestion des bases de données
- **Objectif** : Mettre à jour la version MySQL/MariaDB de la base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsText** : Labels, valeurs
- **OdsSelect** : Nouvelle version
- **OdsCheckbox** : Confirmation sauvegarde
- **OdsButton** : Annuler, Mettre à jour

## 4. HEADER MODAL (WARNING)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mettre à jour la version MySQL" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Ligne 1 | "Cette opération est irréversible. Assurez-vous d'avoir" |
| Ligne 2 | "effectué une sauvegarde de vos données avant de continuer." |

### Informations actuelles
| Champ | Valeur |
|-------|--------|
| Base de données | example_db |
| Version actuelle | MySQL 5.7 |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Nouvelle version | OdsSelect | Oui | Version cible |
| Confirmation | OdsCheckbox | Oui | Confirmer la sauvegarde |

### Versions disponibles
- MySQL 8.0
- MariaDB 10.6
- MariaDB 10.11

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Mettre à jour | Warning (orange #F59E0B) | Lance l'upgrade |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/database/{name} - Info base actuelle
- GET /hosting/web/{serviceName}/database/{name}/availableVersion - Versions disponibles
- POST /hosting/web/{serviceName}/database/{name}/changeVersion - Changer version
