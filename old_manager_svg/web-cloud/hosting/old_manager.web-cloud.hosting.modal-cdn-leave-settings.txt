# Cahier des Charges : web-cloud/hosting - Modal Quitter paramètres CDN

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation (unsaved changes)
- **Contexte** : Configuration CDN
- **Objectif** : Avertir l'utilisateur des modifications non enregistrées avant de quitter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Question, labels, valeurs
- **OdsButton** : Annuler, Quitter

## 4. HEADER MODAL (WARNING)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifications non enregistrées" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Question
| Propriété | Valeur |
|-----------|--------|
| Texte | "Vous avez des modifications non enregistrées. Voulez-vous vraiment quitter ?" |

### Récapitulatif des paramètres
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Bordure | #E5E7EB |

| Option | Valeur | Couleur |
|--------|--------|---------|
| Always online | Activé | Vert (#10B981) |
| HTTP/2 | Activé | Vert (#10B981) |
| Devmode | Désactivé | Rouge (#DC2626) |
| Brotli | Activé | Vert (#10B981) |
| Cache rule | Default rule | Noir |
| CORS | Désactivé | Rouge (#DC2626) |
| HTTPS-Redirect | 301 | Noir |
| WAF | Activé | Vert (#10B981) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Reste sur la page |
| Quitter | Warning (orange #F59E0B) | Quitte sans sauvegarder |

## 7. APIS & ENDPOINTS
Aucun appel API (modal de confirmation côté client)
