# Cahier des Charges : web-cloud/hosting - Modal Exporter statistiques

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'export avec options
- **Contexte** : Statistiques d'hébergement
- **Objectif** : Exporter les statistiques de fréquentation en différents formats

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsDatepicker** : Sélection période
- **OdsRadio** : Format d'export
- **OdsCheckbox** : Données à inclure
- **OdsButton** : Annuler, Exporter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Exporter les statistiques" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Période
| Champ | Type | Description |
|-------|------|-------------|
| Date début | OdsDatepicker | Date de début (ex: 01/01/2024) |
| Date fin | OdsDatepicker | Date de fin (ex: 15/01/2024) |

### Format d'export
| Option | Style sélectionné |
|--------|-------------------|
| CSV | Bordure bleue (#0050D7), radio rempli |
| Excel | Bordure grise si non sélectionné |
| PDF | Bordure grise si non sélectionné |

### Données à inclure
| Option | État par défaut |
|--------|-----------------|
| Visites et pages vues | ✓ Coché |
| Bande passante | ✓ Coché |
| Erreurs HTTP | □ Non coché |
| Sources de trafic | □ Non coché |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Exporter | Primary (bleu #0050D7) | Génère le fichier |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/statistics - Récupérer statistiques
- GET /hosting/web/{serviceName}/statistics/export - Export statistiques
