# Cahier des Charges : web-cloud/hosting - Modal Modifier utilisateur logs

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 2 étapes
- **URL** : N/A (composant modal)
- **Objectif** : Modifier les informations d'un utilisateur statistiques

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Navigation 2 étapes
- **OdsInput** : Identifiant (lecture seule)
- **OdsTextarea** : Description
- **OdsButton** : Navigation wizard

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Modification | Active (bleu) |
| 2 | Confirmation | Inactive |

## 5. FORMULAIRE
| Champ | Type | Requis | Modifiable |
|-------|------|--------|------------|
| Identifiant | Input (disabled) | - | Non |
| Description | Textarea | Non | Oui |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant | Primary | Passer à confirmation |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/userLogs/{login} - Obtenir infos utilisateur
- PUT /hosting/web/{serviceName}/userLogs/{login} - Modifier utilisateur
