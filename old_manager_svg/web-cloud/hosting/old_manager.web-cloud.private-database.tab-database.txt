# Cahier des Charges : web-cloud/private-database - Tab Bases de données

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Bases de données
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les bases de données du serveur

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- Bouton "Créer une base de données" + Refresh
- Tableau des bases
- Menu actions ouvert
- Texte espace total utilisé

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsTabs** : Onglets (Bases de données actif)
- **OdsButton** : Créer une base (outline)
- **OdsIcon** : Refresh (↻)
- **OdsTable** : Liste des bases
- **OdsBadge** : Nombre de sauvegardes
- **OdsActionMenu** : Menu ⋮ ouvert

## 4. BOUTONS D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Créer une base de données | Outline bleu | Ouvre modal création |
| ↻ | Icône | Rafraîchit la liste |

## 5. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Nom de la base | Nom de la DB | wordpress_prod |
| Date de création | Format DD MMM YYYY | 15 déc. 2024 |
| Espace utilisé | Taille en Mo | 125 Mo |
| Sauvegardes | Badge nombre | 3 (bleu) |
| Actions | Menu ⋮ | Voir ci-dessous |

## 6. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Créer un dump | Exporter la base |
| Restaurer un dump | Depuis sauvegarde |
| Importer un fichier | Import SQL |
| Gérer les utilisateurs | Droits utilisateurs |
| Voir les sauvegardes | Liste dumps |
| --- | Séparateur |
| Supprimer | Rouge - Suppression |

## 7. RÉSUMÉ
"L'espace total utilisé est : 226 Mo"

## 8. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/database : Lister bases
- POST /hosting/privateDatabase/{serviceName}/database : Créer
- DELETE /hosting/privateDatabase/{serviceName}/database/{name} : Supprimer
- POST /hosting/privateDatabase/{serviceName}/database/{name}/dump : Exporter
- GET /hosting/privateDatabase/{serviceName}/database/{name}/dump : Sauvegardes
