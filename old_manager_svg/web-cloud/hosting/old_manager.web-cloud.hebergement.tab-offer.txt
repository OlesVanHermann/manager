# Cahier des Charges : web-cloud/hebergement - Tab Changer d'offre

## 1. RESUME EXECUTIF

### Perimetre
Onglet de changement d'offre d'hébergement.

### Objectif fonctionnel
- Voir l'offre actuelle
- Comparer les offres disponibles
- Sélectionner une nouvelle offre

---

## 2. LAYOUT

### Structure carte offre actuelle + grille 4 offres

---

## 3. COMPOSANTS UI

### Carte offre actuelle
| Element | Description |
|---------|-------------|
| Label | Offre actuelle |
| Nom | Performance 1 |
| Badge | Active (vert) |

### Cartes offres (grille 4 colonnes)
| Element | Description |
|---------|-------------|
| Titre | Nom de l'offre |
| Description | Cible (vitrines, pro, e-commerce) |
| Prix | XX,XX EUR/mois HT |
| Séparateur | Ligne horizontale |
| Features | Liste des caractéristiques |
| Bouton | Sélectionner |

### Offres disponibles
| Offre | Prix | Caractéristiques |
|-------|------|------------------|
| Perso | 2,99 EUR | 100 Go, 1 BDD, SSL |
| Performance 1 | 9,99 EUR | 250 Go, BDD illimitées, SSL + CDN |
| Performance 2 | 19,99 EUR | 500 Go, 4 vCPU, 4 Go RAM |
| Performance 3 | 39,99 EUR | 1 To, 8 vCPU, 8 Go RAM |

### États des cartes
| État | Style |
|------|-------|
| Offre actuelle | Badge bleu "OFFRE ACTUELLE" en haut |
| Sélectionnée | Bordure bleue + fond bleu clair |
| Autre | Bordure grise + fond blanc |

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}
GET /order/hosting/web/upgrade/{serviceName}
POST /order/hosting/web/upgrade/{serviceName}
```

### Modele
```typescript
interface CurrentOffer {
  name: string;
  displayName: string;
  state: 'active' | 'suspended';
}

interface AvailableOffer {
  name: string;
  displayName: string;
  description: string;
  price: number;
  diskSpace: string;
  databases: string;
  vcpu?: number;
  ram?: string;
  features: string[];
}
```
