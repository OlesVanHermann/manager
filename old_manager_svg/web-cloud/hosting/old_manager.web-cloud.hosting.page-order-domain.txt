# Cahier des Charges : web-cloud/hosting - Page Commande Étape Domaine

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard (étape 2/4)
- **Contexte** : Commande d'hébergement web
- **Objectif** : Associer un domaine à l'hébergement commandé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 1100px |
| Padding | 30px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étapes
- **OdsRadio** : Options de domaine
- **OdsInput** : Recherche de domaine
- **OdsButton** : Navigation et recherche
- **OdsCard** : Résultats de recherche

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander un hébergement web" |
| Fond | Blanc |

## 5. STEPPER
| Étape | Label | État |
|-------|-------|------|
| 1 | Offre | ✓ Complété (vert #10B981) |
| 2 | Domaine | Actif (bleu #0050D7) |
| 3 | Module | À venir (gris #6B7280) |
| 4 | Confirmation | À venir (gris) |

## 6. CONTENU

### Titre section
| Propriété | Valeur |
|-----------|--------|
| Texte | "Associez un domaine à votre hébergement" |

### Options de domaine
| Option | Description | Style sélectionné |
|--------|-------------|-------------------|
| Commander un nouveau domaine | Enregistrez votre nom de domaine avec l'hébergement | Bordure bleue (#0050D7), radio rempli |
| Utiliser un domaine existant | Associez un domaine que vous possédez déjà chez OVHcloud | Bordure grise si non sélectionné |
| Utiliser un domaine externe | Associez un domaine géré chez un autre registrar | Bordure grise si non sélectionné |

### Recherche de domaine (Option 1)
| Champ | Type | Description |
|-------|------|-------------|
| Recherche | OdsInput | Placeholder: "mon-site.com" |
| Bouton | OdsButton | "Chercher" (bleu #0050D7) |

### Résultats de recherche
| Type | Style | Contenu |
|------|-------|---------|
| Disponible | Fond vert (#DCFCE7), bordure verte (#10B981) | ✓ {domaine} est disponible ! - Prix/an |
| Alternatives | Fond blanc, bordure grise | {domaine} - Prix/an |

### Exemple de résultats
| Domaine | Prix | Disponibilité |
|---------|------|---------------|
| mon-site.com | 9,99€/an | ✓ Disponible (mis en avant) |
| mon-site.fr | 7,99€/an | Disponible |
| mon-site.net | 11,99€/an | Disponible |

## 7. NAVIGATION
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| ← Retour | Gauche | Secondary | Étape précédente |
| Continuer → | Droite | Primary (bleu #0050D7) | Étape suivante |

## 8. APIS & ENDPOINTS
- GET /domain/check - Vérifier disponibilité domaine
- GET /order/cart/{cartId}/domain - Options de domaine
- POST /order/cart/{cartId}/domain - Ajouter domaine au panier
- GET /domain - Lister domaines existants du client
