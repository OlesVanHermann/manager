# Cahier des Charges : web-cloud/hosting - Modal Importer fichier

## 1. VUE D'ENSEMBLE
- **Modal** : Import de fichier SQL
- **Type** : Wizard 3 étapes
- **Objectif** : Importer des données dans la base

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 3 étapes
- Zone formulaire
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsStepper** : 3 étapes
- **OdsRadio** : Mode d'import
- **OdsInput** : Nom du fichier
- **OdsDropzone** : Zone drag-and-drop
- **OdsButton** : Envoyer le fichier, Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Mode | Actif (cercle bleu) |
| 2 | Fichier | À venir (cercle gris) |
| 3 | Options | À venir (cercle gris) |

## 5. FORMULAIRE ÉTAPE 1
### Question
"Quelle action souhaitez-vous effectuer ?"

### Options
| Option | État |
|--------|------|
| Envoyer un nouveau fichier | ● Sélectionné |
| Utiliser un fichier existant | ○ Non sélectionné |

### Champ nom fichier
- Label : "Nom du fichier"
- Input : backup_2024-12-27.sql

### Zone upload
- Zone bordure pointillée
- Texte : "Glisser-déposer ou"
- Lien : "parcourir les fichiers"

### Bouton
- "Envoyer le fichier" (bordure bleue)

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Suivant (primaire, à droite)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/import : Lancer import
- POST /hosting/web/{serviceName}/database/{name}/import/file : Upload fichier
- GET /hosting/web/{serviceName}/database/{name}/import/available : Fichiers existants
