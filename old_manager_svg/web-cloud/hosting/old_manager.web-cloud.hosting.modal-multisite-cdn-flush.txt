# Cahier des Charges : web-cloud/hosting - Modal Purger cache CDN

## 1. VUE D'ENSEMBLE
- **Type** : Modal action CDN
- **URL** : N/A (composant modal)
- **Objectif** : Purger le cache CDN d'un domaine multisite

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Domaine concerné
- **OdsRadio** : Type de purge (4 options)
- **OdsMessage** : Info propagation (bleu)
- **OdsButton** : Actions

## 4. INFO DOMAINE
| Champ | Exemple |
|-------|---------|
| Domaine | www.mondomaine.com |

## 5. OPTIONS DE PURGE
| Option | Sélection | Description |
|--------|-----------|-------------|
| Purger tout le cache | Par défaut (●) | Vide tout le cache CDN |
| Purger une extension de fichier | ○ | Ex: .css, .js, .jpg |
| Purger un dossier | ○ | Ex: /images/ |
| Purger une URL spécifique | ○ | URL complète à purger |

## 6. INFO
| Type | Message |
|------|---------|
| Info | "ℹ La purge du cache peut prendre quelques minutes à se propager sur tous les PoP." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Purger | Primary | Exécuter purge |

## 8. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/attachedDomain/{domain}/purgeCache - Purger cache CDN
