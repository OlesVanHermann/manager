# Cahier des Charges : web-cloud/hosting - Sites indépendants

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des sites indépendants (Indy)
- **Objectif** : Gérer les hébergements liés à l'offre principale
- **Contexte** : Multi-hébergement avec ressources partagées

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Hébergements › monsite.example.com › Sites indépendants
- Onglets de navigation
- Bandeau info explicatif
- Tableau des sites indépendants
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Onglets navigation (Sites indépendants actif)
- **OdsMessage** : Bandeau info bleu
- **OdsTable** : Liste des sites
- **OdsBadge** : Statut (Actif vert, Suspendu jaune)
- **OdsActionMenu** : Menu ⋮
- **OdsLink** : Domaine associé

## 4. ONGLETS
- Informations générales
- Multisite
- FTP - SSH
- Bases de données
- **Sites indépendants** (actif)

## 5. BANDEAU INFO
- "ℹ️ Les sites indépendants sont des hébergements liés à votre offre principale."
- "Ils partagent les ressources mais disposent de leur propre espace FTP et configuration."

## 6. TABLEAU SITES INDÉPENDANTS
| Colonne | Description |
|---------|-------------|
| Nom du site | Identifiant du site |
| Chemin racine | Répertoire FTP |
| État | Badge Actif/Suspendu |
| Domaine associé | Lien vers le domaine |
| Actions | Menu ⋮ |

## 7. EXEMPLES DONNÉES
| Nom | Chemin | État | Domaine |

## 8. MENU ACTIONS (⋮)
- Accéder au FTP
- Modifier la configuration
- Supprimer (rouge)

## 9. PAGINATION
- "Affichage 1-3 sur 3 résultats"

## 10. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/indy : Liste sites indépendants
- GET /hosting/web/{serviceName}/indy/{name} : Détails
- PUT /hosting/web/{serviceName}/indy/{name} : Modifier
- DELETE /hosting/web/{serviceName}/indy/{name} : Supprimer
