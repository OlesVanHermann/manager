# Cahier des Charges : web-cloud/hosting - Modal Copier base de données

## 1. VUE D'ENSEMBLE
- **Modal** : Copie d'une base de données vers une autre
- **Type** : Formulaire avec confirmation
- **Objectif** : Dupliquer les données d'une BDD

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Bandeau warning
- Source (lecture seule)
- Destination (sélection)
- Checkbox confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsMessage** : Bandeau warning jaune
- **OdsCard** : Zone source (lecture seule, fond grisé)
- **OdsRadio** : Type de destination
- **OdsSelect** : Sélection base cible
- **OdsCheckbox** : Confirmation écrasement
- **OdsButton** : Annuler, Copier la base

## 4. BANDEAU WARNING
- "⚠️ Attention"
- "La copie écrasera toutes les données de la base de destination."

## 5. BASE SOURCE (lecture seule)
- Nom : mydatabase_db1
- Info : MySQL 8.0 • 45.2 Mo
- Fond grisé non-modifiable

## 6. SECTION DESTINATION
### Type de destination
| Option | État |
|--------|------|
| Base de données existante | ● Sélectionné |
| Nouvelle base de données | ○ Non sélectionné |

### Sélecteur base cible
- Select : mydatabase_db2

### Info base sélectionnée
- Base sélectionnée : mydatabase_db2 (MySQL 8.0)
- Taille actuelle : 12.5 Mo (sera écrasée)

## 7. CHECKBOX CONFIRMATION
- ✓ "Je confirme vouloir écraser les données de la base de destination"
- Obligatoire pour activer le bouton

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Copier la base (primaire)

## 9. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/copy : Copier base
- GET /hosting/web/{serviceName}/database : Liste bases disponibles
