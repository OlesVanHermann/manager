# Cahier des Charges : web-cloud/hosting - Modal Créer sauvegarde BDD

## 1. VUE D'ENSEMBLE
- **Modal** : Création de sauvegarde de base de données
- **Type** : Formulaire avec options radio
- **Objectif** : Générer un dump de la base de données

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Info base de données
- Options de sauvegarde (radio)
- Options de compression
- Bandeau info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsRadio** : 3 options de sauvegarde
- **OdsSelect** : Format de compression
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Sauvegarder

## 4. INFO BASE DE DONNÉES
| Champ | Valeur |
|-------|--------|
| Nom | monsiteex_wordpress |
| Moteur | MySQL 8.0 |
| Taille | 285 Mo |

## 5. TYPE DE SAUVEGARDE (Radio)
| Option | Description | Sélectionné |
|--------|-------------|-------------|
| Sauvegarde immédiate | Créer un dump maintenant et le stocker sur le serveur | ● Oui |
| Sauvegarde planifiée quotidienne | Programmer une sauvegarde automatique chaque jour | ○ Non |
| Télécharger le dump | Générer et télécharger directement le fichier .sql.gz | ○ Non |

## 6. OPTIONS DE COMPRESSION
- Type : Select dropdown
- Valeur : "GZIP (.sql.gz)"
- Autres options possibles : ZIP, non compressé

## 7. BANDEAU INFO
Fond bleu (#E8F0FE) :
- "ℹ️ La sauvegarde sera disponible dans l'onglet «Sauvegardes»"
- "pendant 30 jours."

## 8. BOUTONS FOOTER
- Annuler (secondaire gris)
- Sauvegarder (primaire bleu)

## 9. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/dump : Créer dump
  - Body : { type, compression, sendEmail }
- GET /hosting/web/{serviceName}/database/{name}/dump : Télécharger dump
