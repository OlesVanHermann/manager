# Cahier des Charges : web-cloud/hosting - Modal Détails tâche

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec progression
- **Contexte** : Gestion des tâches d'hébergement
- **Objectif** : Afficher les détails et la progression d'une tâche en cours

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels et valeurs
- **OdsBadge** : Statut de la tâche
- **OdsProgressBar** : Barre de progression
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détails de la tâche" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Informations de la tâche
| Champ | Valeur exemple |
|-------|----------------|
| ID de la tâche | #123456789 |
| Type | Régénération certificat SSL |
| Statut | Badge "En cours" (jaune #FEF3C7/#92400E) |
| Date de création | 15/01/2024 14:32:00 |
| Dernière mise à jour | 15/01/2024 14:35:22 |

### Statuts possibles
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| En attente | #F3F4F6 | #6B7280 |
| En cours | #FEF3C7 | #92400E |
| Terminée | #DCFCE7 | #166534 |
| Erreur | #FEE2E2 | #DC2626 |

### Progression
| Propriété | Valeur |
|-----------|--------|
| Barre de progression | OdsProgressBar |
| Pourcentage | 70% |
| Message | "Vérification du domaine..." |
| Couleur barre | Bleu (#0050D7) |
| Couleur fond | Gris (#E5E7EB) |

### Derniers logs
| Propriété | Valeur |
|-----------|--------|
| Fond | Terminal (#1F2937) |
| Police | Monospace |
| Couleur texte | Vert (#10B981) |
| Format | [HH:MM:SS] Message |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/tasks/{id} - Détails de la tâche
- GET /hosting/web/{serviceName}/tasks/{id}/logs - Logs de la tâche
