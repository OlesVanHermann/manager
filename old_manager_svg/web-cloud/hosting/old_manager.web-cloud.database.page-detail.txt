# Cahier des Charges : web-cloud/database - Page Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail base de données
- **Contexte** : Web Cloud Databases
- **Objectif** : Gérer une base de données spécifique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour
- **OdsBadge** : Type DB et statut
- **OdsButton** : Actions
- **OdsCard** : Cartes métriques et infos
- **OdsClipboard** : Copier infos connexion
- **OdsDatagrid** : Liste sauvegardes
- **OdsMessage** : Note informative

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour aux bases de données" |
| Titre | Nom de la base |
| Badges | Type (MySQL 8.0) + Statut (Actif) |
| Boutons | phpMyAdmin, Exporter, Supprimer (rouge) |

## 5. CARTES MÉTRIQUES (4 colonnes)
| Carte | Label | Valeur |
|-------|-------|--------|
| Taille | "X.X Go" | "sur Y Go max" |
| Tables | Nombre | "tables dans la base" |
| Connexions | Nombre | "actives actuellement" |
| Uptime | "XX.XX%" | "30 derniers jours" (vert) |

## 6. CARTE INFORMATIONS DE CONNEXION
| Champ | Format | Actions |
|-------|--------|---------|
| Serveur | Hostname monospace, fond dark | Bouton "Copier" |
| Port | "3306" | - |
| Nom de la base | Nom monospace | - |
| Utilisateur | Username monospace | - |
| Mot de passe | Masqué "•••" | Bouton "Réinitialiser" |

### Note informative
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Icône | ℹ️ |
| Texte | Info sur connexions externes et tunnel SSH |

## 7. CARTE SAUVEGARDES AUTOMATIQUES
| Colonne | Description |
|---------|-------------|
| Date | Format "DD MMM YYYY HH:MM" |
| Taille | "X.X Go" |
| Statut | Badge "Complète" |
| Actions | "Restaurer | Télécharger" (liens bleus) |

### Paramètres sauvegarde
| Champ | Valeur |
|-------|--------|
| Fréquence | "Quotidienne à HH:MM" |
| Rétention | "X jours" |
| Action | Bouton "Modifier" |

### Bouton création
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Créer sauvegarde" |
| Style | Primary #0050D7 |

## 8. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName} - Détail base CloudDB
- GET /hosting/web/{serviceName}/database/{name} - Détail base hébergement
- POST /hosting/privateDatabase/{serviceName}/dump - Créer sauvegarde
- GET /hosting/privateDatabase/{serviceName}/dump - Liste sauvegardes
- POST /hosting/privateDatabase/{serviceName}/dump/{id}/restore - Restaurer
