# Cahier des Charges : web-cloud/hosting - Modal Restaurer snapshot FTP

## 1. VUE D'ENSEMBLE
- **Modal** : Sélection et restauration d'un snapshot
- **Type** : Wizard 2 étapes
- **Objectif** : Restaurer les fichiers FTP à partir d'un snapshot

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 2 étapes
- Zone sélection snapshot
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsStepper** : 2 étapes
- **OdsSelect** : Sélection snapshot avec dropdown
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Sélection | Actif (cercle bleu) |
| 2 | Confirmation | À venir (cercle gris) |

## 5. FORMULAIRE ÉTAPE 1
### Instruction
"Sélectionnez le snapshot à restaurer."

### Select instance de snapshot
| Option | Description |
|--------|-------------|
| snapshot.week.1 | Semaine dernière (sélectionné) |
| snapshot.week.2 | Il y a 2 semaines |
| snapshot.week.3 | Il y a 3 semaines |
| snapshot.day.1 | Hier |

## 6. DROPDOWN OUVERT
- Options affichées dans un menu avec shadow
- Option sélectionnée surlignée en bleu (#E8F0FE)
- Séparateurs entre les options

## 7. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Suivant (primaire, à droite)

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/snapshot : Liste des snapshots
- POST /hosting/web/{serviceName}/snapshot/{name}/restore : Restaurer
