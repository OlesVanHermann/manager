# Cahier des Charges : web-cloud/hosting - Modal Déployer Git

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation du déploiement Git
- **Type** : Modal de confirmation
- **Objectif** : Lancer le pull/déploiement depuis le dépôt distant

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Message de confirmation
- Checkbox option force
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsCheckbox** : Option force push
- **OdsButton** : Annuler, Confirmer

## 4. MESSAGE
"Vous êtes sur le point de déployer ou mettre à jour le contenu de votre dépôt distant sur votre hébergement."

## 5. CHECKBOX OPTION
- □ "En cas de conflit, forcer les modifications distantes sur votre dépôt local."
- Non coché par défaut

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Confirmer (primaire, à droite)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/runtime/attachedDomain/{domain}/git/pull : Déployer
