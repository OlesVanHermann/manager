# Cahier des Charges : web-cloud/hosting - Modal Chemin multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification
- **Contexte** : Configuration multisite
- **Objectif** : Modifier le répertoire racine d'un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, chemin actuel
- **OdsInput** : Nouveau chemin
- **OdsMessage** : Information
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le répertoire racine" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Informations domaine
| Champ | Valeur |
|-------|--------|
| Domaine | shop.example.com |
| Chemin actuel | /www/shop |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Nouveau répertoire racine | OdsInput | Oui | Chemin absolu |

### Aide
| Propriété | Valeur |
|-----------|--------|
| Texte | "Le chemin doit commencer par /www" |

### Information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Assurez-vous que le dossier existe et contient" |
| Ligne 2 | "les fichiers de votre site web." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Modifier | Primary (bleu #0050D7) | Modifie le chemin |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Config actuelle
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier chemin
