# Cahier des Charges : web-cloud/hosting - Utilisateurs logs

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion des utilisateurs autorisés aux statistiques.

### Objectif fonctionnel
- Accéder aux statistiques
- Accéder aux logs bruts
- Gérer les utilisateurs autorisés

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/userLogs
POST /hosting/web/{serviceName}/userLogs
PUT /hosting/web/{serviceName}/userLogs/{login}
DELETE /hosting/web/{serviceName}/userLogs/{login}
POST /hosting/web/{serviceName}/userLogs/{login}/changePassword
```

### Modele
```typescript
interface UserLogs {
  login: string;
  description?: string;
  createdAt: string;
  isPrimary: boolean;
}
```

## 6. REGLES METIER

- Utilisateur principal non supprimable
- Accès protégé par htpasswd
- Statistiques = Urchin/AWStats
- Logs = access.log/error.log Apache
