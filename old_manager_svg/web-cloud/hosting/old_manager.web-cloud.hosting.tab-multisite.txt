# Cahier des Charges : web - Onglet Multisite

## 1. VUE D'ENSEMBLE
- **Page** : Gestion multisites
- **Type** : Onglet avec datagrid
- **Objectif** : Gérer les domaines attachés à l'hébergement

## 2. STRUCTURE PAGE
- Onglet "Multisite" actif
- Titre + description
- Bouton ajout
- Datagrid des domaines

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Ajouter un domaine
- **OdsDatagrid** : Liste domaines
- **OdsActionMenu** : Actions par ligne

## 4. HEADER SECTION
| Élément | Valeur |
|---------|--------|
| Titre | "Multisite" |
| Description | "Gérez les domaines et sous-domaines attachés à votre hébergement." |
| Bouton | "+ Ajouter un domaine" (primary) |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Link bleu | Nom du domaine |
| Dossier racine | Text | Chemin (/www, /blog...) |
| SSL | Text | Actif (vert) / Inactif |
| Firewall | Text | Actif (vert) / Inactif |
| CDN | Text | Actif / Inactif |
| IP pays | Text | - ou pays |
| Actions | Menu ⋮ | ActionMenu |

## 6. EXEMPLES DONNÉES
| Domaine | Dossier | SSL | Firewall | CDN |
|---------|---------|-----|----------|-----|
| monsite.fr | /www | Actif | Actif | Inactif |
| www.monsite.fr | /www | Actif | Actif | Inactif |
| blog.monsite.fr | /blog | Actif | Inactif | Inactif |
| api.monsite.fr | /api | Actif | Actif | Actif |

## 7. ACTIONMENU OPTIONS
| Option | Style | Description |
|--------|-------|-------------|
| Modifier | Normal | Modifier configuration |
| Activer le CDN | Normal | Activer CDN |
| Activer le firewall | Normal | Activer firewall |
| Régénérer SSL | Normal | Régénérer certificat |
| --- | Séparateur | |
| Supprimer | Rouge | Détacher le domaine |

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain - Liste domaines
- POST /hosting/web/{serviceName}/attachedDomain - Attacher domaine
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Détails
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier
- DELETE /hosting/web/{serviceName}/attachedDomain/{domain} - Supprimer
- POST /hosting/web/{serviceName}/attachedDomain/{domain}/regenerateSsl - Régénérer SSL
