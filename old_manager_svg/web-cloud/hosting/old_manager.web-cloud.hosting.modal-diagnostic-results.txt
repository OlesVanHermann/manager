# Cahier des Charges : web-cloud/hosting - Modal Résultats diagnostic

## 1. VUE D'ENSEMBLE
- **Type** : Modal de rapport avec liste de tests
- **Contexte** : Diagnostic de l'hébergement
- **Objectif** : Afficher les résultats d'un diagnostic complet de l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Résumé (succès/warning/error)
- **OdsText** : Titres, labels
- **OdsBadge** : Statut de chaque test
- **OdsButton** : Relancer test, Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résultats du diagnostic" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Résumé global
| Propriété | Valeur |
|-----------|--------|
| Fond | Vert (#DCFCE7) |
| Icône | ✓ |
| Titre | "Diagnostic terminé" |
| Détails | "{success} tests réussis, {warnings} avertissement, {errors} erreurs" |

### Liste des tests
| Test | Description | Statut possible |
|------|-------------|-----------------|
| Configuration DNS | Vérification des enregistrements | OK / Erreur |
| Certificat SSL | Validité du certificat | Valide jusqu'au {date} / Expiré |
| Version PHP | Version actuelle | {version} (à jour) / Obsolète |
| Performance BDD | Requêtes lentes | OK / Requêtes lentes détectées |
| Accès FTP | Connexion FTP | Fonctionnel / Erreur |
| Envoi d'emails | Test SMTP | Fonctionnel / Bloqué |

### Styles des badges
| État | Fond | Texte | Bordure |
|------|------|-------|---------|
| OK | #DCFCE7 | #166534 | - |
| Warning | #FEF3C7 | #92400E | #F59E0B |
| Error | #FEE2E2 | #DC2626 | #DC2626 |

### Icônes état
| État | Icône | Couleur fond |
|------|-------|--------------|
| Success | ✓ | #DCFCE7 |
| Warning | ! | #FEF3C7 |
| Error | × | #FEE2E2 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Relancer test | Secondary (bordure grise) | Relance le diagnostic |
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/diagnostic - Lancer un diagnostic
- GET /hosting/web/{serviceName}/diagnostic/{id} - Résultats du diagnostic
