# Cahier des Charges : web-cloud/hosting - Modal Quota emails automatisés

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec jauge
- **Contexte** : Emails automatisés
- **Objectif** : Afficher le quota d'envoi d'emails journalier et l'utilisation actuelle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsProgressBar** : Jauge d'utilisation
- **OdsText** : Statistiques
- **OdsMessage** : Avertissement
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Quota d'emails automatisés" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Utilisation actuelle
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Utilisation actuelle" |
| Type | Barre de progression |
| Couleur remplie | Bleu (#0050D7) |
| Couleur vide | Gris (#E5E7EB) |
| Pourcentage | Affiché au centre |

### Compteur
| Propriété | Valeur |
|-----------|--------|
| Format | "{utilisés} / {max} emails envoyés aujourd'hui" |

### Statistiques
| Champ | Valeur |
|-------|--------|
| Quota journalier | 500 emails |
| Réinitialisation | Tous les jours à 00:00 UTC |

### Avertissement (si > 40%)
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Texte | Avertissement sur l'utilisation du quota |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/email/quota - Quota email
- GET /hosting/web/{serviceName}/email/statistics - Statistiques d'envoi
