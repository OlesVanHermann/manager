# Cahier des Charges : web-cloud/private-database - Dashboard détaillé

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard Web Cloud Database avec métriques
- **Type** : Page dashboard avec tuiles et statistiques
- **Objectif** : Vue complète du service avec bases et utilisateurs

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation Web Cloud
- Sidebar avec liste des services
- Breadcrumb de navigation
- Titre du service
- Onglets (Informations générales actif)
- 3 tuiles ligne 1 + 2 tuiles ligne 2
- Action menu ouvert

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Liste des Web Cloud Databases
- **OdsTabs** : Navigation onglets
- **OdsCard** : 5 tuiles
- **OdsBadge** : État "Démarré"
- **OdsProgressBar** : Espace disque
- **OdsActionMenu** : Menu ouvert

## 10. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName} : Infos générales
- GET /hosting/privateDatabase/{serviceName}/database : Liste bases
- GET /hosting/privateDatabase/{serviceName}/user : Liste utilisateurs
