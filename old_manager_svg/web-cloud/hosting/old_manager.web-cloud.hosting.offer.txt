# Cahier des Charges : web-cloud/hosting - Changer d'offre

## 1. RESUME EXECUTIF

### Perimetre
Wizard de changement d'offre d'hébergement.

### Objectif fonctionnel
- Comparer les offres
- Sélectionner une nouvelle offre
- Procéder au changement

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}
GET /order/hosting/web/upgrade/{serviceName}
POST /order/hosting/web/upgrade/{serviceName}
```

### Modele
```typescript
interface OfferComparison {
  offers: OfferDetails[];
  currentOffer: string;
  recommendedOffer: string;
}

interface OfferDetails {
  name: string;
  displayName: string;
  diskSpace: string;
  websites: string;
  databases: number;
  emails: number;
  ram?: string;
  price: number;
  isCurrent: boolean;
  isRecommended: boolean;
}
```
