# Cahier des Charges : web-cloud/hosting - Modal Désactiver SSL

## 1. VUE D'ENSEMBLE
- **Modal** : Désactivation/suppression du SSL
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer le SSL de l'hébergement

## 2. STRUCTURE MODAL
- Barre rouge en haut (indicateur danger)
- Header avec icône warning et titre
- Bouton fermer (×)
- Info domaine
- Message de confirmation
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsIcon** : Icône warning dans cercle rouge
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Valider (danger rouge)

## 4. HEADER MODAL
- Barre rouge 4px en haut (indicateur danger)
- Icône ⚠ dans cercle rose (#FEE2E2)
- Titre : "Suppression SSL"

## 5. INFORMATION DOMAINE
| Champ | Valeur |
|-------|--------|
| Nom du domaine | www.monsite.com |

## 6. MESSAGE DE CONFIRMATION
"Êtes-vous sûr de vouloir supprimer le SSL sur votre hébergement ?"

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron :
- "⚠ Cette action est irréversible. Votre site ne sera plus accessible en HTTPS"
- "jusqu'à ce qu'un nouveau certificat soit installé."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Valider (danger rouge)

## 9. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/ssl : Supprimer certificat SSL
