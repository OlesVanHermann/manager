# Cahier des Charges : web-cloud/hosting - Modal SSL multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec radio
- **Contexte** : Configuration multisite
- **Objectif** : Gérer l'inclusion/exclusion d'un domaine dans le certificat SSL

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels
- **OdsBadge** : Statut SSL actuel
- **OdsRadio** : Options activer/désactiver
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Gérer le certificat SSL" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Informations domaine
| Champ | Valeur |
|-------|--------|
| Domaine | blog.example.com |
| Statut SSL actuel | Badge "Activé" (vert #DCFCE7/#166534) |

### Options SSL
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Options SSL" |

### Option Activer SSL
| Propriété | Valeur |
|-----------|--------|
| Label | "Activer SSL" |
| Description | "Inclure ce domaine dans le certificat" |
| Bordure si sélectionné | Bleue (#0050D7) |

### Option Désactiver SSL
| Propriété | Valeur |
|-----------|--------|
| Label | "Désactiver SSL" |
| Description | "Exclure ce domaine du certificat" |
| Bordure si sélectionné | Bleue (#0050D7) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Applique le changement SSL |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Config actuelle
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier SSL
- POST /hosting/web/{serviceName}/ssl/regenerate - Régénérer certificat
