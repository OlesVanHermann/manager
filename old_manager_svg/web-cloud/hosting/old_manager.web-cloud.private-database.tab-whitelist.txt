# Cahier des Charges : web-cloud/private-database - Tab IPs autorisées

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet IPs autorisées (whitelist)
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les adresses IP autorisées à se connecter

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- Bandeau warning (si vide)
- Toggle accès réseau OVH
- Section ajout IP
- Tableau des IPs
- Menu actions ouvert

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsTabs** : Onglets (IPs autorisées actif)
- **OdsMessage** : Warning orange
- **OdsToggle** : Accès réseau hébergement
- **OdsButton** : Ajouter une IP
- **OdsTable** : Liste des IPs
- **OdsIcon** : Checkmarks (✓)
- **OdsActionMenu** : Menu ⋮ ouvert

## 4. BANDEAU WARNING
Fond orange (#FFFBEB) avec bordure (#F59E0B) :
"⚠ Aucune IP n'est autorisée. Ajoutez une IP pour pouvoir vous connecter à votre base de données."

## 5. TOGGLE ACCÈS RÉSEAU
| Élément | Type | État |
|---------|------|------|
| Toggle | Switch ON/OFF | ON (bleu) |
| Label | Texte | "Autoriser l'accès depuis le réseau d'hébergement web OVH" |

## 6. BOUTON D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter une IP | Secondaire gris | Ouvre modal ajout |

## 7. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| IP / Réseau | Adresse ou CIDR | 192.168.1.0/24 |
| Service | Checkmark ✓ | ✓ vert |
| SFTP | Checkmark ou "—" | ✓ ou — |
| Statut | Description libre | Bureau Paris |
| Actions | Menu ⋮ | Modifier, Supprimer |

## 8. CHECKMARKS
| État | Affichage | Couleur |
|------|-----------|---------|
| Activé | ✓ | Vert (#10B981) |
| Désactivé | — | Gris |

## 9. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Éditer l'IP |
| --- | Séparateur |
| Supprimer | Rouge - Supprimer |

## 10. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/whitelist : Lister IPs
- POST /hosting/privateDatabase/{serviceName}/whitelist : Ajouter IP
  - ip: string (CIDR)
  - name: string
  - service: boolean
  - sftp: boolean
- PUT /hosting/privateDatabase/{serviceName}/whitelist/{ip} : Modifier
- DELETE /hosting/privateDatabase/{serviceName}/whitelist/{ip} : Supprimer
- PUT /hosting/privateDatabase/{serviceName}/oom : Toggle accès OVH
