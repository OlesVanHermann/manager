# Cahier des Charges : web-cloud/hosting - Modal Commander base de données

## 1. VUE D'ENSEMBLE
- **Modal** : Commande d'une nouvelle base de données
- **Type** : Formulaire avec sélection d'offre
- **Objectif** : Ajouter une BDD supplémentaire

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Bandeau info
- Grille 3 offres
- Formulaire configuration
- Zone récapitulatif
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsMessage** : Bandeau info bleu
- **OdsCard** : Cartes offres (sélection bordure bleue)
- **OdsInput** : Nom base avec préfixe
- **OdsSelect** : Version moteur
- **OdsToggleGroup** : Durée engagement
- **OdsButton** : Annuler, Commander

## 4. BANDEAU INFO
"ℹ️ Vous pouvez ajouter jusqu'à 5 bases de données supplémentaires à votre hébergement."

## 5. OFFRES DISPONIBLES
| Offre | Taille | Prix |
|-------|--------|------|
| Start SQL | 100 Mo | Inclus |
| Performance SQL | 1 Go | 2,99 €/mois (sélectionné) |
| SQL Privé | Serveur dédié | À partir de 5,99 € |

## 6. FORMULAIRE
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Nom de la base | Préfixe + Input | monprefixe_nouvelle_base |
| Version du moteur | Select | MySQL 8.0 (recommandé) |
| Durée d'engagement | Toggle buttons | 1 mois, 12 mois (sélectionné), 24 mois |

## 7. RÉDUCTIONS
- -10% sur 12 mois

## 8. ZONE RÉCAPITULATIF
- "Performance SQL 1 Go - MySQL 8.0 - 12 mois"
- Prix : "32,29 € HT/an"

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Commander (primaire)

## 10. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/databaseAvailableOffer : Offres disponibles
- POST /hosting/web/{serviceName}/database/order : Commander base
- GET /hosting/web/{serviceName}/databaseAvailableVersion : Versions disponibles
