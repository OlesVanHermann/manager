# Cahier des Charges : web-cloud/hosting - Modal Vider le cache CDN

## 1. VUE D'ENSEMBLE
- **Modal** : Purge du cache CDN
- **Type** : Wizard 2 étapes
- **Objectif** : Vider le cache CDN partiellement ou totalement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 2 étapes
- Zone contenu
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsStepper** : 2 étapes (Options, Confirmation)
- **OdsRadio** : Sélection portée du flush
- **OdsInput** : Valeur du pattern (si applicable)
- **OdsButton** : Annuler, Valider

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Options | Actif (cercle bleu) |
| 2 | Confirmation | À venir (cercle gris) |

## 5. CONTENU ÉTAPE 1
### Question
"Êtes-vous certain·e de vouloir vider le cache de votre CDN ?"

### Information domaine
- **Nom du domaine** : www.monsite.com

### Options de portée
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Tout le site | ● (sélectionné) | Vider tout le cache |
| Dossier | ○ | Ex: /assets/images/ |
| URI | ○ | Chemin spécifique |
| Extension | ○ | Ex: .css, .js |
| Expression régulière | ○ | Pattern regex |

### Champ valeur (grisé si "Tout le site")
- Label : "Valeur"
- Input : Placeholder "/assets/images/"
- Désactivé si "Tout le site" sélectionné

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Valider (primaire, à droite)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/cdn/{domain}/flush : Vider cache
- POST /hosting/web/{serviceName}/cdn/{domain}/flush : Avec paramètres pattern
