# Cahier des Charges : web-cloud/hosting - Modal Informations connexion BDD

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec actions
- **Contexte** : Gestion des bases de données
- **Objectif** : Afficher les informations de connexion à la base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsText** : Labels, valeurs
- **OdsClipboard** : Boutons copier
- **OdsLink** : Modifier mot de passe, accès phpMyAdmin
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Informations de connexion" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | "Utilisez ces informations pour vous connecter à votre base de données." |

### Informations de connexion
| Champ | Valeur | Action |
|-------|--------|--------|
| Nom de la base | example_db | Copier |
| Serveur | example.mysql.db.ovh.net | Copier |
| Port | 3306 | - |
| Utilisateur | example_user | Copier |
| Mot de passe | •••••••••••• | Modifier (lien) |

### Accès phpMyAdmin
| Propriété | Valeur |
|-----------|--------|
| Type | Lien cliquable |
| Fond | #F9FAFB |
| Texte | "Accéder à phpMyAdmin" |
| Icône | → |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme la modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/database/{name} - Informations de connexion
- GET /hosting/web/{serviceName}/database/{name}/privateNetwork - Infos réseau privé
