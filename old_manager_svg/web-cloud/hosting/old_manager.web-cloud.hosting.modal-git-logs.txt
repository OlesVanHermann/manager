# Cahier des Charges : web-cloud/hosting - Modal Logs Git

## 1. VUE D'ENSEMBLE
- **Modal** : Affichage des logs de déploiement Git
- **Type** : Modal informative (lecture seule)
- **Objectif** : Consulter les logs du dernier déploiement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Zone console (fond sombre)
- Footer avec bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsCard** : Zone console noire (#1F2937)
- **OdsButton** : Fermer

## 4. ZONE CONSOLE
- Fond sombre (#1F2937)
- Police monospace
- Couleurs :
  - Timestamps : Gris (#9CA3AF)
  - Messages info : Blanc
  - Messages succès : Vert (#10B981)
