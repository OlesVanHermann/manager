# Cahier des Charges : web-cloud/hosting - Modal Détacher domaine multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression
- **URL** : N/A (composant modal)
- **Objectif** : Détacher un domaine multisite de l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container (header rouge/danger)
- **OdsText** : Informations domaine
- **OdsCheckbox** : Détacher www, Supprimer DNS
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Détacher le domaine" |
| Message | "Vous allez détacher le domaine suivant de votre hébergement :" |

## 5. DÉTAILS AFFICHÉS
| Champ | Exemple |
|-------|---------|
| Nom de domaine | blog.example.com |
| Répertoire racine | /www/blog |

## 6. OPTIONS CHECKBOXES
| Option | État par défaut | Description |
|--------|-----------------|-------------|
| Détacher www | Non coché | Détacher également www.blog.example.com |
| Supprimer DNS | Coché | Les enregistrements DNS pointant vers cet hébergement seront supprimés |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Détacher | Danger (rouge) | Confirmer détachement |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/attachedDomain/{domain} - Détacher domaine
