# Cahier des Charges : web-cloud/hosting - Modal Activer SQL Privé

## 1. VUE D'ENSEMBLE
- **Modal** : Activation d'un serveur SQL privé
- **Type** : Wizard 3 étapes
- **Objectif** : Commander un serveur de base de données dédié

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Stepper horizontal 3 étapes
- Description et avantages
- Sélection RAM (4 cartes)
- Sélection moteur BDD
- Zone récapitulatif
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsStepper** : 3 étapes
- **OdsMessage** : Bandeau avantages vert
- **OdsCard** : Cartes RAM (sélection bordure bleue)
- **OdsBadge** : "POPULAIRE" sur offre recommandée
- **OdsToggleGroup** : Sélection moteur BDD
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Configuration | Actif |
| 2 | Confirmation | À venir |
| 3 | Paiement | À venir |
