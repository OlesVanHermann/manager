# Cahier des Charges : web-cloud/hosting - Modal Supprimer tâche planifiée

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression CRON
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'une tâche planifiée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container (header rouge/danger)
- **OdsText** : Question confirmation
- **OdsText** : Détails de la tâche (5 lignes)
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir supprimer cette tâche planifiée ?" |

## 5. DÉTAILS AFFICHÉS
| Champ | Exemple |
|-------|---------|
| Commande | /www/cron/backup.php |
| Langage | PHP 8.1 |
| Fréquence | 0 3 * * * (tous les jours à 3h) |
| Email | admin@example.com |
| Description | Sauvegarde quotidienne |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Supprimer tâche CRON |

## 7. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/cron/{id} - Supprimer tâche planifiée
