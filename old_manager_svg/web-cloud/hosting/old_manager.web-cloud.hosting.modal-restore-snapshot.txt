# Cahier des Charges : web-cloud/hosting - Modal Restaurer snapshot (FTP)

## 1. VUE D'ENSEMBLE
- **Modal** : Restauration de fichiers depuis un snapshot
- **Type** : Wizard 2 étapes
- **Objectif** : Restaurer l'espace FTP à partir d'un point de sauvegarde

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 2 étapes
- Zone sélection snapshot
- Info snapshots disponibles
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsStepper** : 2 étapes
- **OdsSelect** : Sélection instance snapshot
- **OdsCard** : Zone info snapshots disponibles
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Suivant →

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Sélection | Actif (cercle bleu) |
| 2 | Confirmation | À venir (cercle gris) |

## 5. SÉLECTION
"Sélectionnez le snapshot que vous souhaitez restaurer :"

### Select instance
- Dropdown : "Snapshot de la veille (J-1)"

## 6. SNAPSHOTS DISPONIBLES
| Snapshot | Date |
|----------|------|
| Veille (J-1) | 26 décembre 2025 à 03:00 |
| Semaine (J-7) | 20 décembre 2025 à 03:00 |
| Mois (J-14) | 13 décembre 2025 à 03:00 |

## 7. BANDEAU WARNING
Fond jaune avec bordure orange :
- "⚠️ Attention"
- "La restauration écrasera les fichiers actuels de votre espace FTP."
- "Cette opération est irréversible. Assurez-vous d'avoir sauvegardé les fichiers importants avant de continuer."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Suivant → (primaire)

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/snapshot : Liste snapshots
- POST /hosting/web/{serviceName}/snapshot/{name}/restore : Restaurer snapshot
