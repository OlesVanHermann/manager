# Cahier des Charges : web-cloud/private-database - Commander CloudDB

## 1. VUE D'ENSEMBLE
- **Écran** : Wizard de commande Web Cloud Databases
- **Type** : Page formulaire avec récapitulatif
- **Objectif** : Configurer et commander une base de données

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Zone principale avec étapes
- Étape 1 : Choix solution (version, RAM)
- Étape 2 : Choix durée
- Encadré récapitulatif avec prix
- Bouton Payer

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsText** : Titres d'étapes
- **OdsSelect** : Choix version
- **OdsCard** : Cartes RAM sélectionnables
- **OdsRadio** : Sélection durée
- **OdsCard** : Récapitulatif
- **OdsButton** : Payer

## 4. ÉTAPE 1 : CHOIX SOLUTION
### Version du serveur
| Option | Description |
|--------|-------------|
| MariaDB 10.11 | Version par défaut |
| MySQL 8.0 | Alternative |
| PostgreSQL 15 | Base avancée |

### RAM (cartes sélectionnables)
| RAM | Prix | État |
|-----|------|------|
| 512 Mo | À partir de 7.99 EUR/mois | - |
| 1 Go | À partir de 15.99 EUR/mois | Sélectionné ✓ |
| 2 Go | À partir de 29.99 EUR/mois | - |

## 5. ÉTAPE 2 : CHOIX DURÉE
| Durée | Description |
|-------|-------------|
| 1 mois | Mensuel |
| 12 mois | Annuel (économie) |
