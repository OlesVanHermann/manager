# Cahier des Charges : web-cloud/hosting - Modal Configuration domaine multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification
- **URL** : N/A (composant modal)
- **Objectif** : Modifier la configuration d'un domaine multisite

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 560px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Dossier racine
- **OdsSelect** : Moteur d'exécution, Logs séparés
- **OdsCheckbox** : SSL, CDN, IP géolocalisée, Firewall WAF, Logs séparés
- **OdsTooltip** : Aide sur chaque option (?)
- **OdsMessage** : Warning Git (jaune)
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis | Valeur exemple |
|-------|------|--------|----------------|
| Dossier racine | Input text | Oui | ./www |
| Moteur d'exécution | Select dropdown | Non | PHP 8.2 |

## 5. OPTIONS CHECKBOXES
| Option | État | Tooltip |
|--------|------|---------|
| SSL | Coché (✓) | Activer le certificat SSL |
| CDN | Coché (✓) | Activer le CDN |
| IP géolocalisée | Non coché | Utiliser une IP locale |
| Firewall applicatif (WAF) | Non coché | Activer la protection WAF |
| Logs séparés | Coché (✓) | Générer des logs distincts |

## 6. LOGS SÉPARÉS (si activé)
| Champ | Type |
|-------|------|
| Domaine des logs | Select dropdown (ex: www.mondomaine.com) |

## 7. WARNING GIT
| Type | Message |
|------|---------|
| Warning | "⚠ Le dossier racine ne peut pas être modifié car une intégration Git est active." |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Enregistrer | Primary | Sauvegarder modifications |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Obtenir config actuelle
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier configuration
