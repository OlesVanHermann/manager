# Cahier des Charges : web-cloud/hosting - Activer l'offre e-mail

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard multi-étapes
- **URL** : /hosting/{serviceName}/email/activate
- **Objectif** : Activer l'adresse e-mail incluse Start10M

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 900px |
| Hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsBackLink** : Retour navigation
- **OdsText** : Description contextuelle
- **OdsCard** : Container principal
- **OdsStepper** : Indicateur étapes (2 étapes)
- **OdsSelect** : Choix domaine
- **OdsMessage** : Info offre (bleu)
- **OdsButton** : Navigation wizard

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Activer l'adresse e-mail incluse" |
| Description | "Activez l'adresse e-mail incluse avec votre hébergement." |

## 5. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Choix du domaine | Active (bleu) |
| 2 | Confirmation | Inactive (gris) |

## 6. FORMULAIRE (ÉTAPE 1)
| Champ | Type | Requis |
|-------|------|--------|
| Domaine associé | Select dropdown | Oui |

## 7. INFO OFFRE
| Message | Description |
|---------|-------------|
| Info | "ℹ L'offre e-mail Start10M vous permet de créer jusqu'à 10 adresses e-mail de 5 Go chacune, incluses gratuitement avec votre hébergement." |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Annuler activation |
| Suivant | Primary | Étape suivante |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain - Liste domaines associés
- POST /hosting/web/{serviceName}/email/activation - Activer e-mail Start10M
