# Cahier des Charges : web-cloud/hosting - Modal Logs déploiement Git

## 1. VUE D'ENSEMBLE
- **Type** : Modal de visualisation des logs
- **Contexte** : Déploiement Git
- **Objectif** : Afficher les logs du dernier déploiement Git

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **Code block** : Console de logs (fond sombre)
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Logs du dernier déploiement" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Zone des logs
| Propriété | Valeur |
|-----------|--------|
| Fond | Sombre (#1F2937) |
| Police | Monospace |
| Taille | 11px |

### Format des logs
| Élément | Format | Couleur |
|---------|--------|---------|
| Timestamp | [YYYY-MM-DD HH:mm:ss] | Vert (#10B981) |
| Message normal | Texte | Gris clair (#E5E7EB) |
| Message succès | Texte | Bleu clair (#60A5FA) |
| Commit hash | Short hash | Gris clair |

### Étapes de déploiement typiques
1. Starting deployment...
2. Cloning repository from origin/main
3. Running pre-deploy hooks...
4. Installing dependencies (npm install)
5. Building application (npm run build)
6. Copying files to www directory...
7. Running post-deploy hooks...
8. Deployment completed successfully
9. Commit: {hash} - "{message}"

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/git/deployment/{id}/logs - Logs du déploiement
- GET /hosting/web/{serviceName}/git/deployment - Liste des déploiements
