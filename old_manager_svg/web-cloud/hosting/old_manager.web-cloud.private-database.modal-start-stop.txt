# Cahier des Charges : web-cloud/private-database - Modal Arrêter le serveur (simple)

## 1. VUE D'ENSEMBLE
- **Modal** : Arrêt du serveur (confirmation simple)
- **Type** : Modal confirmation
- **Objectif** : Arrêter le serveur CloudDB

## 2. STRUCTURE MODAL
- Bande orange en haut
- Icône pause dans cercle orange
- Header avec titre et bouton fermer (×)
- Message de confirmation
- Description impact
- Boutons Annuler / Arrêter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x240)
- **OdsIcon** : || (pause)
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Arrêter

## 4. TITRE
"Arrêter le serveur"

## 5. ICÔNE
- Cercle fond jaune clair (#FEF3C7)
- Icône || (pause)
- Couleur: Orange (#F59E0B)

## 6. MESSAGE
"Êtes-vous sûr de vouloir arrêter le serveur de base de données ?"

## 7. DESCRIPTION
Texte muted (#6B7280) :
"Vos sites et applications ne pourront plus accéder à la base de données."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Arrêter | Warning orange | Arrête le serveur |

## 9. APIS & ENDPOINTS
- POST /hosting/privateDatabase/{serviceName}/stop
  - Retourne: task object
