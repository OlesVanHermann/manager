# Cahier des Charges : web-cloud/hosting - Modal Alerte OOM base de données

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'alerte critique
- **Contexte** : Monitoring base de données
- **Objectif** : Alerter l'utilisateur sur un dépassement mémoire de la base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Détails de l'erreur
- **OdsText** : Informations, recommandations
- **OdsButton** : Fermer, Voir les offres

## 4. HEADER MODAL (DANGER)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Alerte Out of Memory" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Détails de l'erreur
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEE2E2 |
| Ligne 1 | "Votre base de données a dépassé la limite mémoire allouée." |
| Ligne 2 | "Cela peut causer des erreurs sur votre site web." |
| Ligne 3 | "Dernière erreur: {date} {heure}" |

### Informations de la base
| Champ | Valeur | Couleur |
|-------|--------|---------|
| Base de données | example_db | Noir |
| RAM utilisée | 512 Mo / 512 Mo (100%) | Rouge (#DC2626) |
| Occurrences (24h) | 12 erreurs | Noir |

### Recommandations
| Propriété | Valeur |
|-----------|--------|
| Titre | "Recommandations" |
| Liste | • Optimisez vos requêtes SQL |
| | • Ajoutez des index sur les tables fréquemment consultées |
| | • Envisagez une offre avec plus de RAM |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary (bordure grise) | Ferme modal |
| Voir les offres | Primary (bleu #0050D7) | Redirige vers upgrade |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/database/{name}/statistics - Statistiques mémoire
- GET /hosting/web/{serviceName}/database/{name}/oomKiller - Historique OOM
