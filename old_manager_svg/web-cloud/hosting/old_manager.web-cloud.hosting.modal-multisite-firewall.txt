# Cahier des Charges : web-cloud/hosting - Modal Firewall multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec toggles
- **Contexte** : Configuration multisite
- **Objectif** : Configurer le firewall applicatif pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, descriptions
- **OdsToggle** : Activation firewall, restriction IP
- **OdsInput** : Liste des IPs (désactivé si toggle off)
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration firewall" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Domaine
| Champ | Valeur |
|-------|--------|
| Domaine | api.example.com |

### Option Firewall applicatif
| Propriété | Valeur |
|-----------|--------|
| Label | "Firewall applicatif" |
| Description | "Protection contre les attaques web courantes" |
| Type | OdsToggle |
| Couleur activé | Vert (#10B981) |

### Option Restriction IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Restriction IP" |
| Description | "Limiter l'accès à certaines adresses IP" |
| Type | OdsToggle |
| Couleur désactivé | Gris (#E5E7EB) |

### IPs autorisées (conditionnel)
| Propriété | Valeur |
|-----------|--------|
| Label | "IPs autorisées" |
| Type | OdsInput |
| État | Désactivé si toggle Restriction IP = off |
| Placeholder | "Activez la restriction IP pour configurer" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Enregistrer | Primary (bleu #0050D7) | Sauvegarde configuration |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Config actuelle
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier config
