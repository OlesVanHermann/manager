# Cahier des Charges : web-cloud/hosting - Modal Politique de sauvegarde

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative (lecture seule)
- **Contexte** : Gestion des hébergements web
- **Objectif** : Afficher la politique de sauvegarde automatique OVHcloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information sur les sauvegardes
- **OdsText** : Titres, labels, valeurs
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Politique de sauvegarde automatique" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "OVHcloud effectue des sauvegardes automatiques de vos fichiers" |
| Ligne 2 | "et bases de données selon la politique ci-dessous." |

### Section Snapshots FTP
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Snapshots FTP" |
| Fond | #F9FAFB |

| Type snapshot | Rétention |
|--------------|-----------|
| Snapshot quotidien | 1 jour |
| Snapshot hebdomadaire | 7 jours |
| Snapshot mensuel | 30 jours |

### Section Sauvegardes BDD
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Sauvegardes BDD" |
| Fond | #F9FAFB |

| Type | Valeur |
|------|--------|
| Dump automatique quotidien | Rétention: 7 jours |
| Points de restauration manuels | Max: 5 par base |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme la modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/snapshot - Info snapshots FTP
- GET /hosting/web/{serviceName}/database/{name}/dump - Info dumps BDD
