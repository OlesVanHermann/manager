# Cahier des Charges : web-cloud/hosting - Modal Préchauffer cache CDN

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec détails
- **Contexte** : Configuration CDN
- **Objectif** : Lancer le préchauffage du cache CDN pour les URLs configurées

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Explication du préchauffage
- **OdsText** : Labels, valeurs, liste URLs
- **OdsButton** : Annuler, Préchauffer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Préchauffer le cache CDN" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Le préchauffage charge vos ressources dans le cache CDN" |
| Ligne 2 | "pour accélérer le premier chargement par vos visiteurs." |

### Informations
| Champ | Valeur |
|-------|--------|
| Domaine | www.example.com |
| URLs à préchauffer | Titre section |
| Compteur | "{count} URLs configurées" |

### Liste des URLs
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Format | Liste des chemins relatifs |

### Note
| Propriété | Valeur |
|-----------|--------|
| Texte | "Cette opération peut prendre quelques minutes." |
| Couleur | Gris (#6B7280) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Préchauffer | Primary (bleu #0050D7) | Lance le préchauffage |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/cdn/prewarm/start - Lancer le préchauffage
- GET /hosting/web/{serviceName}/cdn/prewarm - Liste des URLs configurées
