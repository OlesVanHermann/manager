# Cahier des Charges : web-cloud/private-database - Tab Métriques

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Métriques
- **Type** : Page graphiques
- **Objectif** : Visualiser les métriques de performance du serveur

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- 2 graphiques empilés verticalement

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsTabs** : Onglets (Métriques actif)
- **OdsCard** : 2 cartes graphiques
- **OdsChart** : Graphiques ligne + area

## 7. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/metrics : Récupérer métriques
  - Params: period (24h, 7d, 30d), type (memory, connections, queries, cpu)
  - Retourne: timestamps[], values[]
