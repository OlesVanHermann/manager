# Cahier des Charges : web-cloud/hosting - Modal Éditeur .htaccess

## 1. VUE D'ENSEMBLE
- **Type** : Modal éditeur de code
- **Contexte** : Configuration Apache
- **Objectif** : Modifier le fichier .htaccess via interface web

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 490px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsText** : Chemin du fichier
- **Code editor** : Zone d'édition avec coloration syntaxique
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Éditeur .htaccess" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Texte | "Attention : une erreur dans le .htaccess peut rendre votre site inaccessible." |

### Zone d'édition
| Propriété | Valeur |
|-----------|--------|
| Titre | "Contenu du fichier .htaccess" |
| Chemin affiché | /www/.htaccess |
| Fond éditeur | Sombre (#1F2937) |
| Fond numéros ligne | Gris foncé (#374151) |
| Police | Monospace |

### Coloration syntaxique
| Type | Couleur |
|------|---------|
| Directives (RewriteEngine, RewriteCond, etc.) | Orange (#F59E0B) |
| Commentaires (#) | Gris (#9CA3AF) |
| Noms de modules | Bleu (#60A5FA) |
| Texte standard | Gris clair (#E5E7EB) |

### Numéros de ligne
| Propriété | Valeur |
|-----------|--------|
| Fond | #374151 |
| Couleur | Gris (#6B7280) |
| Alignement | Droite |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme sans sauver |
| Enregistrer | Primary (bleu #0050D7) | Sauvegarde le fichier |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ftp/file?path=/.htaccess - Lire le fichier
- PUT /hosting/web/{serviceName}/ftp/file?path=/.htaccess - Sauvegarder le fichier
