# Cahier des Charges : web-cloud/hosting - Migration d'offre

## 1. VUE D'ENSEMBLE
- **Écran** : Sélection nouvelle offre pour migration
- **Objectif** : Migrer vers une offre supérieure ou inférieure
- **Contexte** : Évolution des besoins d'hébergement

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Hébergements › monsite.example.com › Migration
- Titre : "Migrer votre hébergement"
- Sous-titre : "Choisissez la nouvelle offre..."
- Bandeau offre actuelle
- Grille 4 cartes d'offres
- Bandeau warning
- Boutons Annuler / Suivant

## 3. COMPOSANTS UI
- **OdsCard** : Cartes offres (sélection bordure bleue épaisse)
- **OdsBadge** : "ACTUEL" (badge bleu sur offre courante)
- **OdsButton** : "Sélectionner" (primaire), "Non disponible" (grisé)
- **OdsMessage** : Bandeau info (offre actuelle), bandeau warning
- **OdsButton** : Footer "Annuler" (secondaire), "Suivant" (primaire)

## 4. BANDEAU OFFRE ACTUELLE
- "Offre actuelle : Perso"
- "100 Go d'espace disque • 10 bases de données • 100 comptes email"

## 5. GRILLE OFFRES
| Offre | Espace | Prix | Caractéristiques | Action |
|-------|--------|------|------------------|--------|
| Starter | 100 Mo | 1,99 €/mois | 1 site, 1 BDD, 10 emails | Non disponible |
| **Perso** | 100 Go | 3,99 €/mois | Illimités, 10 BDD, 100 emails | ACTUEL (bordure bleue) |
| Pro | 250 Go | 7,99 €/mois | Illimités, 25 BDD, 500 emails + WCD | Sélectionner |
| Performance 1 | 500 Go | 15,99 €/mois | Illimités, 50 BDD, 1000 emails + Ressources dédiées | Sélectionner |

## 6. BANDEAU WARNING
- "⚠️ Attention"
- "La migration vers une offre inférieure peut entraîner la perte de certaines fonctionnalités."
- "Vérifiez que votre nouvelle offre correspond à vos besoins avant de valider."

## 7. BOUTONS FOOTER
- Annuler (gris, à gauche)
- Suivant (primaire bleu, à droite)

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/offer : Offre actuelle
- GET /hosting/web/offer : Liste offres disponibles
- POST /hosting/web/{serviceName}/changeOffer : Initier migration
