# Cahier des Charges : web/hosting - Modal détacher domaine multisite

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration avec options
- **Contexte** : Hébergement web - Multisite
- **Objectif** : Détacher un domaine de l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsCard** : Détails du domaine
- **OdsCheckbox** : Options de détachement
- **OdsMessage** : Warning fichiers
- **OdsButton** : Annuler, Détacher

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détacher un domaine" |

## 5. DESCRIPTION
| Texte |
|-------|
| "Vous êtes sur le point de détacher ce domaine de votre hébergement." |

## 6. CARTE DÉTAILS DOMAINE
| Champ | Exemple |
|-------|---------|
| Domaine | blog.example.com |
| Répertoire racine | /www/blog |

## 7. OPTIONS

### Option www
| Propriété | Valeur |
|-----------|--------|
| Checkbox | Coché par défaut |
| Label | "Détacher également www.{domain}" |

### Option DNS
| Propriété | Valeur |
|-----------|--------|
| Checkbox | Non coché par défaut |
| Label | "Supprimer également l'entrée DNS" |
| Description | "Retire l'enregistrement A/AAAA de la zone DNS de ce domaine." |

## 8. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ Cette action ne supprime pas les fichiers du répertoire associé." |

## 9. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Détacher | Danger (rouge) | Détache le domaine |

## 10. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/attachedDomain/{domain} - Détacher domaine
