# Cahier des Charges : web-cloud/ssl - Modal Order

## 1. VUE D'ENSEMBLE
- **Type** : Modal commande certificat SSL
- **Contexte** : Web Cloud SSL Certificates
- **Objectif** : Commander un nouveau certificat SSL payant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 490px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsCard** : Sélection type certificat
- **OdsBadge** : DV, OV, EV
- **OdsInput** : Domaine principal
- **OdsCheckbox** : Options www et wildcard
- **OdsRadio** : Durée
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander un certificat SSL" |
| Bouton fermer | × |
| Fond | #F9FAFB |
