# Cahier des Charges : web-cloud/hosting - Modal Détacher e-mail

## 1. VUE D'ENSEMBLE
- **Type** : Page détachement option e-mail
- **URL** : /hosting/{serviceName}/email/detach
- **Objectif** : Détacher l'option e-mail Start10M vers MX Plan

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 900px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsBackLink** : Retour navigation
- **OdsText** : Description contextuelle
- **OdsMessage** : Warning irréversible (rouge)
- **OdsCard** : Container offres
- **OdsRadio** : Choix offre
- **OdsButton** : Actions

| Élément | Description |
|---------|-------------|
| Titre | "Détacher l'option e-mail" |

|------|---------|

## 6. OFFRES DISPONIBLES
| Offre | Sélection | Description | Prix |
|-------|-----------|-------------|------|
| MX Plan | Par défaut (●) | Conservez vos e-mails avec une offre MX Plan indépendante | 0,00 € HT/mois |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour page précédente |
