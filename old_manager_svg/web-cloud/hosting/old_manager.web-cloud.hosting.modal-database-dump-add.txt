# Cahier des Charges : web-cloud/hosting - Modal Créer sauvegarde base

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une sauvegarde manuelle
- **Type** : Wizard 2 étapes
- **Objectif** : Créer un dump de la base de données

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 2 étapes
- Formulaire sélection date
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsStepper** : 2 étapes
- **OdsSelect** : Sélection date avec dropdown
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Date de sauvegarde | Actif (cercle bleu) |
| 2 | Confirmation | À venir (cercle gris) |

## 5. FORMULAIRE
### Question
"Quelle sauvegarde souhaitez-vous créer pour la base de données **mydb_hosting123** ?"

### Select date
| Option | Description |
|--------|-------------|
| Maintenant | Sauvegarde en temps réel (sélectionné) |
| Hier | Point de restauration J-1 |
| Semaine dernière | Point de restauration J-7 |

### Dropdown ouvert
- Options affichées dans un menu déroulant avec shadow
- Option sélectionnée surlignée en bleu (#E8F0FE)

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Suivant (primaire, à droite)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/dump : Créer dump
- GET /hosting/web/{serviceName}/database/{name}/dump/available : Dates disponibles
