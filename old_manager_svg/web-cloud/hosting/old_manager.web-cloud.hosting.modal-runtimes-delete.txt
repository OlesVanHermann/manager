# Cahier des Charges : web-cloud/hosting - Modal Supprimer runtime

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'un runtime Cloud Web

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container (header rouge/danger)
- **OdsText** : Question et détails runtime
- **OdsMessage** : Warning impact (jaune)
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer le runtime" |
| Question | "Êtes-vous sûr de vouloir supprimer ce runtime ?" |

## 5. DÉTAILS AFFICHÉS
| Champ | Exemple |
|-------|---------|
| Nom | my-nodejs-app |
| Langage | nodejs-18 |
| Répertoire public | public |
| Environnement | production |
| Fichier de démarrage | app.js |

## 6. AVERTISSEMENT
| Type | Message |
|------|---------|
| Warning | "⚠ Les sites utilisant ce runtime ne fonctionneront plus." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Supprimer runtime |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/runtime/{id} - Supprimer runtime
