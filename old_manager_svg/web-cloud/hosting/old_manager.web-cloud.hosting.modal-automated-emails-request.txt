# Cahier des Charges : web-cloud/hosting - Modal Bloquer emails automatisés

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation blocage emails
- **URL** : N/A (composant modal)
- **Objectif** : Bloquer/débloquer envoi emails automatisés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Message confirmation
- **OdsMessage** : Info conséquences
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir bloquer l'envoi des e-mails automatisés depuis votre hébergement ?" |

## 5. INFO
| Message | Description |
|---------|-------------|
| Info | "Les scripts de votre hébergement ne pourront plus envoyer d'e-mails automatiquement tant que le blocage est actif." |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Bloquer | Danger (rouge) | Confirmer blocage |

## 7. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/automatedEmail - Modifier état blocage
