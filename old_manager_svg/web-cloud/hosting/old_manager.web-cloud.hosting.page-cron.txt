# Cahier des Charges : web-cloud/hosting - Page Cron

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion tâches planifiées
- **Contexte** : Web Cloud Hosting
- **Objectif** : Créer et gérer des cron jobs PHP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour, Voir logs
- **OdsButton** : Créer une tâche
- **OdsCard** : Stats + liste
- **OdsDatagrid** : Tableau cron
- **OdsBadge** : Statut exécution
- **OdsActionMenu** : Menu actions (⋮)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour à {serviceName}" |
| Titre | "Tâches planifiées (Cron)" |
| Sous-titre | "{serviceName} • Automatisez vos scripts PHP" |
| Bouton | "+ Créer une tâche" |

## 5. CARTES STATISTIQUES (3 colonnes)
| Carte | Valeur | Détail |
|-------|--------|--------|
| Tâches actives | X / 10 disponibles | Nombre |
| Dernière exécution | Il y a X min | Nom script (succès) |
| Erreurs (24H) | X | Lien "Voir les logs →" |

## 6. DATAGRID CRON
| Colonne | Description |
|---------|-------------|
| Commande | Chemin script (monospace) |
| Fréquence | Toutes les X, Tous les jours à Xh |
| Dernière exec. | Date/heure relative |
| Statut | Badge Succès/Erreur |
| Actions | Modifier (bleu), Menu ⋮ |

## 7. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Succès | #DCFCE7 | #16A34A |
| Erreur | #FEE2E2 | #DC2626 |

## 8. MENU ACTIONS (⋮)
| Option | Description |
|--------|-------------|
| Exécuter maintenant | Lancer immédiatement |
| Voir les logs | Afficher historique |
| Désactiver | Suspendre la tâche |
| Supprimer | Supprimer définitivement |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cron - Liste cron jobs
- POST /hosting/web/{serviceName}/cron - Créer cron
- PUT /hosting/web/{serviceName}/cron/{id} - Modifier
- DELETE /hosting/web/{serviceName}/cron/{id} - Supprimer
- POST /hosting/web/{serviceName}/cron/{id}/run - Exécuter
