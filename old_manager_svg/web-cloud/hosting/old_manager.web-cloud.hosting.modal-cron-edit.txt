# Cahier des Charges : web-cloud/hosting - Modal Modifier tâche planifiée

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire d'édition
- **Contexte** : Gestion des tâches CRON
- **Objectif** : Modifier une tâche planifiée existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 490px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Commande, expression cron, email, description
- **OdsSelect** : Langage, fréquence
- **OdsTextarea** : Description
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la tâche planifiée" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champs
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Commande | OdsInput | Oui | Chemin du script à exécuter |
| Langage | OdsSelect | Non | Version PHP (8.2, 8.1, 8.0, 7.4...) |
| Fréquence | OdsSelect | Oui | Préréglages de fréquence |
| Expression cron | OdsInput | Non | Expression cron avancée |
| Email de notification | OdsInput | Non | Email pour les alertes |
| Description | OdsTextarea | Non | Description de la tâche |

### Options de fréquence
- Chaque minute
- Toutes les 5 minutes
- Toutes les 15 minutes
- Toutes les heures
- Chaque jour à 03:00
- Chaque semaine
- Chaque mois

### Options de langage
- PHP 8.2
- PHP 8.1
- PHP 8.0
- PHP 7.4
- Node.js 18
- Node.js 16
- Python 3
- Bash

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Modifier | Primary (bleu #0050D7) | Enregistre les modifications |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cron/{id} - Détails d'une tâche
- PUT /hosting/web/{serviceName}/cron/{id} - Modifier une tâche
- GET /hosting/web/{serviceName}/runtime - Liste des runtimes disponibles
