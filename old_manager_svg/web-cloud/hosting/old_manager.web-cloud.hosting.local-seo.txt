# Cahier des Charges : web-cloud/hosting - Visibilité Pro (SEO Local)

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion Visibilité Pro (référencement local)
- **Objectif** : Gérer les informations d'entreprise pour le SEO local
- **Contexte** : Présence multi-annuaires

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Hébergements > monsite.com > Visibilité Pro
- Onglets de navigation (Visibilité Pro actif avec soulignement)
- Description du service
- Lien vérification référencement
- Bouton Commander
- Tableau des entreprises

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : "Commander" (primaire)
- **OdsLink** : "Vérifiez votre référencement local ↗"
- **OdsTable** : Liste des entreprises
- **OdsBadge** : Statut (OK vert, Création en cours jaune, À définir bleu)
- **OdsActionMenu** : Menu ⋮

## 4. ONGLETS
- Informations générales
- Multisite
- Certificats SSL
- Modules en 1 clic
- FTP - SSH
- Bases de données
- **Visibilité Pro** (actif, souligné bleu)

## 5. DESCRIPTION
- "Soyez présents là où vos clients vous cherchent, mais aussi là où ils parlent de vous."
- "Cette solution de référencement local vous permet de gérer, via une interface unique et intuitive, vos informations et coordonnées d'entreprise."

## 6. TABLEAU ENTREPRISES
| Colonne | Description |
|---------|-------------|
| Nom de l'entreprise | Raison sociale |
| Adresse | Adresse complète ou badge "À définir" |
| Compte Email | Email de contact |
| Statut | Badge OK/Création en cours |
| Actions | Menu ⋮ |

## 7. EXEMPLES DONNÉES
| Entreprise | Adresse | Email | Statut |
|------------|---------|-------|--------|
| Ma Boutique Paris | 12 rue de la Paix, 75001 Paris | contact@maboutique.fr | OK |
| Agence Lyon | À définir (badge bleu) | lyon@maboutique.fr | Création en cours |

## 8. MENU ACTIONS (⋮)
- Accéder à l'interface
- Supprimer l'abonnement (rouge)

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/localSeo : Liste abonnements
- POST /hosting/web/{serviceName}/localSeo : Commander
- DELETE /hosting/web/{serviceName}/localSeo/{id} : Supprimer
- GET /hosting/web/{serviceName}/localSeo/{id} : Détails entreprise
