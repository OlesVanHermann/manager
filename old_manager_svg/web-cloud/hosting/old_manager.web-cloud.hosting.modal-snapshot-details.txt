# Cahier des Charges : web-cloud/hosting - Modal Détails snapshot FTP

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec action
- **Contexte** : Gestion des sauvegardes FTP
- **Objectif** : Afficher les détails d'un snapshot et permettre la restauration

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsText** : Détails du snapshot
- **OdsBadge** : Statut
- **OdsButton** : Fermer, Restaurer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détails du snapshot" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Les snapshots permettent de restaurer vos fichiers FTP" |
| Ligne 2 | "à une date antérieure." |

### Informations du snapshot
| Champ | Valeur exemple |
|-------|----------------|
| Type | Snapshot hebdomadaire |
| Date de création | 15/01/2024 03:00:00 |
| Taille | 2.4 Go |
| Statut | Badge "Disponible" (vert #DCFCE7/#166534) |
| Expiration | 22/01/2024 |

### Types de snapshot
- Snapshot quotidien
- Snapshot hebdomadaire
- Snapshot mensuel

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary (bordure grise) | Ferme modal |
| Restaurer | Primary (bleu #0050D7) | Lance la restauration |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/snapshot - Liste des snapshots
- POST /hosting/web/{serviceName}/snapshot/{id}/restore - Restaurer un snapshot
