# Cahier des Charges : web-cloud/hosting - Modal Modifier mot de passe logs

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification mot de passe
- **URL** : N/A (composant modal)
- **Objectif** : Changer le mot de passe d'un utilisateur statistiques

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Description utilisateur
- **OdsPassword** : Nouveau mot de passe et confirmation
- **OdsFormField** : Wrapper avec helper text
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Modifier le mot de passe" |
| Description | "Modifiez le mot de passe de l'utilisateur {nom}" |

## 5. FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nouveau mot de passe | Password | Oui | •••••••• |
| Confirmation | Password | Oui | •••••••• |

## 6. CONTRAINTES MOT DE PASSE
| Règle | Description |
|-------|-------------|
| Longueur | Entre 8 et 30 caractères |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer nouveau mot de passe |

## 8. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/userLogs/{login}/changePassword - Changer mot de passe
