# Cahier des Charges : web-cloud/hebergement - Tab Tasks

## 1. RESUME EXECUTIF

### Perimetre
Onglet de suivi des tâches en cours sur l'hébergement.

### Objectif fonctionnel
- Voir l'état des opérations
- Suivre les tâches planifiées
- Consulter l'historique

## 3. COMPOSANTS UI

### Tableau tâches
| Colonne | Description |
|---------|-------------|
| Tâche | Description de l'opération |
| Statut | Badge couleur selon état |
| Date de création | Date/heure début |
| Date de fin | Date/heure fin ou - |

### États de tâche
| État | Badge |
|------|-------|
| En cours | Bleu (#DBEAFE) |
| Terminée | Vert (#DEF7EC) |
| En erreur | Rouge (#FEE2E2) |
| Planifiée | Jaune (#FEF3C7) |
| Annulée | Gris (#F3F4F6) |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/task
GET /hosting/web/{serviceName}/task/{id}
```

### Modele
```typescript
interface HostingTask {
  id: string;
  function: string;
  status: 'doing' | 'done' | 'error' | 'todo' | 'cancelled';
  startDate: string;
  doneDate?: string;
}
```
