# Cahier des Charges : web-cloud/hosting - Modal Résilier e-mail

## 1. VUE D'ENSEMBLE
- **Type** : Modal résiliation e-mail
- **URL** : N/A (composant modal)
- **Objectif** : Résilier l'option e-mail associée à un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Question confirmation
- **OdsSelect** : Choix domaine
- **OdsMessage** : Warning suppression (rouge)
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Résilier l'option e-mail" |
| Question | "Souhaitez-vous résilier l'option e-mail associée au domaine suivant ?" |

## 5. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Domaine | Select dropdown | Oui |

## 6. AVERTISSEMENT
| Type | Message |
|------|---------|
| Danger | "⚠ Cette action supprimera définitivement toutes les adresses e-mail et les données associées à ce domaine." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Résilier | Danger (rouge) | Confirmer résiliation |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/email - Résilier option e-mail
