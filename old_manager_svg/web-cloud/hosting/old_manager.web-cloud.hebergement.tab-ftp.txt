# Cahier des Charges : web-cloud/hebergement - Tab FTP

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des accès FTP et SSH.

### Objectif fonctionnel
- Voir les informations de connexion FTP
- Gérer les utilisateurs FTP
- Activer/désactiver SSH

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/user
POST /hosting/web/{serviceName}/user
PUT /hosting/web/{serviceName}/user/{login}
DELETE /hosting/web/{serviceName}/user/{login}
POST /hosting/web/{serviceName}/user/{login}/changePassword
```

### Modele
```typescript
interface FtpInfo {
  ftpServer: string;
  ftpPort: number;
  sftpPort: number;
  homeDirectory: string;
}

interface FtpUser {
  login: string;
  home: string;
  state: 'ok' | 'off';
  sshState: 'active' | 'none';
  isPrimary: boolean;
}
```

## 6. REGLES METIER

- Utilisateur principal non supprimable
- SSH disponible selon offre
- Mot de passe principal = mot de passe OVH
- Utilisateurs secondaires avec droits limités
