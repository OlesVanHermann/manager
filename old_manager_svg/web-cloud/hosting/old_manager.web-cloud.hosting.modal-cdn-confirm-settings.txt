# Cahier des Charges : web-cloud/hosting - Modal Confirmer paramètres CDN

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Configuration CDN
- **Objectif** : Récapituler et confirmer les modifications des paramètres CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 490px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres, labels, valeurs
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Confirmer les modifications" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |
