# Cahier des Charges : web-cloud/hosting - Modal Ajouter runtime

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout runtime (Cloud Web)
- **URL** : N/A (composant modal)
- **Objectif** : Créer un nouveau moteur d'exécution pour Cloud Web

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Labels et indication champs obligatoires
- **OdsInput** : Nom, Répertoire public, Environnement, Fichier démarrage
- **OdsSelect** : Type runtime
- **OdsTooltip** : Aide sur chaque champ (?)
- **OdsButton** : Actions

| Répertoire public | Input text | Oui | public | ? |
| Environnement | Input text | Oui | production | ? |
