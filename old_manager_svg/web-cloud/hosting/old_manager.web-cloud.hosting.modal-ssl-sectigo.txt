# Cahier des Charges : web-cloud/hosting - Modal Commander SSL Sectigo

## 1. VUE D'ENSEMBLE
- **Modal** : Commande de certificat SSL Sectigo
- **Type** : Formulaire simple avec sélection
- **Objectif** : Commander un certificat SSL payant Sectigo

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Select domaine avec dropdown
- Bandeau warning
- Note informative
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsSelect** : Sélection du domaine
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Valider

## 4. SELECT DOMAINE
Dropdown avec liste des domaines multisites :
- www.monsite.com (sélectionné, fond bleu)
- blog.monsite.com
- shop.monsite.com

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron :
- "⚠ Si vous avez ajouté des hébergements multisites il y a moins de deux heures,"
- "il est possible qu'ils ne soient pas inclus dans votre certificat SSL."

## 6. NOTE INFORMATIVE
"En cliquant sur Valider, vous allez ouvrir un bon de commande dans un nouvel"
"onglet pour votre certificat SSL."

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Valider (primaire)

## 8. COMPORTEMENT
- Au clic sur Valider : ouverture bon de commande dans nouvel onglet

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain : Liste domaines disponibles
- POST /hosting/web/{serviceName}/ssl/order : Commander certificat Sectigo
