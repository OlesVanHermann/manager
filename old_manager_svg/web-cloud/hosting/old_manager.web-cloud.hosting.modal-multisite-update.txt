# Cahier des Charges : web-cloud/hosting - Modal Modifier domaine multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 2 étapes
- **URL** : N/A (composant modal)
- **Objectif** : Modifier la configuration d'un domaine multisite

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 650px |
| Hauteur | 560px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Navigation 2 étapes
- **OdsInput** : Dossier racine (disabled pour nom domaine)
- **OdsCheckbox** : SSL, CDN, Firewall, IP géolocalisée, Logs séparés, www
- **OdsSelect** : Domaine logs
- **OdsButton** : Navigation wizard

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Configuration | Active (bleu) |
| 2 | Récapitulatif | Inactive |

## 5. FORMULAIRE
| Champ | Type | Requis | Modifiable |
|-------|------|--------|------------|
| Nom de domaine | Input (disabled) | - | Non |
| Dossier racine | Input text | Oui | Oui |

## 6. OPTIONS CHECKBOXES
| Option | État par défaut |
|--------|-----------------|
| SSL | Coché (✓) |
| CDN | Coché (✓) |
| Firewall | Non coché |
| IP géolocalisée | Non coché |
| Logs séparés | Coché (✓) |
| Ajouter www | Non coché |

## 7. LOGS SÉPARÉS (si activé)
| Champ | Type |
|-------|------|
| Domaine des logs | Select dropdown |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant | Primary | Passer au récapitulatif |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Obtenir config
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier config
