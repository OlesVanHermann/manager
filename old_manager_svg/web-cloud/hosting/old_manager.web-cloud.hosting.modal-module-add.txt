# Cahier des Charges : web-cloud/hosting - Modal Installer module CMS

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 4 étapes
- **URL** : N/A (composant modal)
- **Objectif** : Installer un module CMS (WordPress, Joomla, Drupal, PrestaShop)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 610px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Navigation 4 étapes
- **OdsCard** : Grille de sélection modules
- **OdsSelect** : Choix domaine
- **OdsCheckbox** : Mode avancé
- **OdsMessage** : Warning BDD (jaune), Info mode avancé (bleu)
- **OdsButton** : Actions

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Module | Active (bleu) |
| 2 | Base de données | Inactive |
| 3 | Configuration | Inactive |
| 4 | Confirmation | Inactive |

## 5. MODULES DISPONIBLES (ÉTAPE 1)
| Module | Version | Logo |
|--------|---------|------|
| WordPress | 6.4 | Bleu #21759B |
| Joomla | 5.0 | Orange #F28D1A |
| Drupal | 10.2 | Bleu #0678BE |
| PrestaShop | 8.1 | Violet #96588A |

## 6. FORMULAIRE
| Champ | Type | Requis | Default |
|-------|------|--------|---------|
| Domaine | Select dropdown | Oui | Premier domaine |
| Mode avancé | Checkbox | Non | Non coché |

## 7. INFO CHEMIN
| Message | Description |
|---------|-------------|
| Helper | "Chemin d'installation par défaut : /www" |

## 8. WARNING BDD
| Type | Message |
|------|---------|
| Warning | "⚠ Une base de données est requise. Vérifiez que vous disposez d'une base de données disponible ou commandez-en une." |

## 9. INFO MODE AVANCÉ
| Message | Description |
|---------|-------------|
| Info | "Le mode avancé permet de : Configurer une base de données externe, Personnaliser le chemin d'installation" |

## 10. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Installer | Primary | Lancer installation |

## 11. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain - Liste domaines
- GET /hosting/web/moduleList - Liste modules disponibles
- POST /hosting/web/{serviceName}/module - Installer module
