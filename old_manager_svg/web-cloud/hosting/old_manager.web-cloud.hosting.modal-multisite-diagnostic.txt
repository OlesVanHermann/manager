# Cahier des Charges : web-cloud/hosting - Modal Diagnostic DNS multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative
- **URL** : N/A (composant modal)
- **Objectif** : Afficher les informations DNS requises pour pointer le domaine vers l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Instructions
- **OdsCard** : Info DNS (fond gris)
- **OdsLink** : Lien modification zone DNS
- **OdsButton** : Fermer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Diagnostic DNS" |
| Instructions | "Pour que votre domaine pointe vers l'hébergement, vous devez modifier l'enregistrement A de votre zone DNS :" |

## 5. INFORMATIONS DNS
| Champ | Valeur exemple |
|-------|----------------|
| Type d'enregistrement | A |
| IP cible | 213.186.33.5 |
| Domaine | www.mondomaine.com |

## 6. LIEN
| Action | Description |
|--------|-------------|
| Modifier zone DNS | "→ Modifier l'enregistrement A dans ma zone DNS" |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary | Fermer modal |

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain}/diagnostic - Obtenir diagnostic DNS
