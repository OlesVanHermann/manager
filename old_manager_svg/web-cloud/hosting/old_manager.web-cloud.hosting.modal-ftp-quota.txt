# Cahier des Charges : web-cloud/hosting - Modal Quota FTP / Espace disque

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec graphique
- **Contexte** : Gestion de l'espace disque
- **Objectif** : Visualiser l'utilisation de l'espace de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **Cercle de progression** : Jauge circulaire
- **OdsText** : Statistiques
- **Légende colorée** : Répartition
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Espace disque" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Cercle de progression
| Propriété | Valeur |
|-----------|--------|
| Type | Cercle SVG |
| Couleur utilisé | Bleu (#0050D7) |
| Couleur disponible | Gris (#E5E7EB) |
| Texte central | Pourcentage + "utilisé" |

### Détails d'espace
| Champ | Valeur | Couleur |
|-------|--------|---------|
| Espace total | 250 Go | Noir |
| Espace utilisé | 150 Go | Noir |
| Espace disponible | 100 Go | Vert (#10B981) |

### Répartition
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Répartition" |
| Fond | #F9FAFB |

### Catégories
| Type | Couleur | Exemple |
|------|---------|---------|
| Fichiers web | Bleu (#0050D7) | 120 Go |
| Bases de données | Vert (#10B981) | 25 Go |
| Emails | Orange (#F59E0B) | 5 Go |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/quota - Quota global
- GET /hosting/web/{serviceName}/database/quota - Quota BDD
- GET /hosting/web/{serviceName}/email/quota - Quota email
