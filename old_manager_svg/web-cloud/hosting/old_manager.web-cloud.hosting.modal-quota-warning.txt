# Cahier des Charges : web-cloud/hosting - Modal Alerte quota disque

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'alerte avec statistiques
- **Contexte** : Monitoring espace disque
- **Objectif** : Alerter sur un espace disque presque plein

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsProgressBar** : Jauge d'utilisation
- **OdsText** : Statistiques, répartition
- **OdsButton** : Fermer, Augmenter offre

## 4. HEADER MODAL (WARNING)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Espace disque presque plein" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'alerte
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Ligne 1 | "Vous avez utilisé plus de 90% de votre espace disque." |
| Ligne 2 | "Libérez de l'espace ou augmentez votre offre." |

### Utilisation du disque
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Utilisation du disque" |

### Barre de progression
| Propriété | Valeur |
|-----------|--------|
| Couleur remplie | Orange (#F59E0B) |
| Couleur vide | Gris (#E5E7EB) |
| Pourcentage | 90% (affiché au centre) |

### Compteur
| Propriété | Valeur |
|-----------|--------|
| Format | "{used} Go / {total} Go utilisés" |

### Répartition
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |

| Type | Espace |
|------|--------|
| Fichiers FTP | 7.2 Go |
| Bases de données | 1.5 Go |
| Emails | 0.3 Go |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary (bordure grise) | Ferme modal |
| Augmenter offre | Primary (bleu #0050D7) | Redirige vers upgrade |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/quota - Quota global
- GET /order/upgrade/webHosting/{serviceName} - Options d'upgrade
