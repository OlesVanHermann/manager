## 8. APIS & ENDPOINTS

```
# Informations hébergement
GET /hosting/web/{serviceName}
→ cluster, filer, datacenter, offer, state

# Statistiques stockage
GET /hosting/web/{serviceName}/quota
→ used, available, inodes

# Configuration
GET /hosting/web/{serviceName}/ovhConfig
→ phpVersion, environment

# SSL
GET /hosting/web/{serviceName}/ssl
→ status, provider, expiration

# CDN
GET /hosting/web/{serviceName}/cdn
→ enabled, type

# Firewall
GET /hosting/web/{serviceName}/firewall
→ enabled

# Informations abonnement
GET /hosting/web/{serviceName}/serviceInfos
→ creation, expiration, renew, contactAdmin, contactTech, contactBilling

# Domaines attachés
GET /hosting/web/{serviceName}/attachedDomain
→ count

# Modules installés
GET /hosting/web/{serviceName}/module
→ count
```
