# Cahier des Charges : web-cloud/hosting - Statistiques base de données

## 1. VUE D'ENSEMBLE
- **Écran** : Statistiques d'une base de données
- **Objectif** : Visualiser l'utilisation et les performances
- **Contexte** : Monitoring et optimisation

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Hébergements › monsite.example.com › Bases de données › Statistiques
- Titre : "Statistiques des bases de données"
- Sélecteur de base + filtres période
- 2 graphiques côte à côte
- Tuile résumé avec 5 KPIs
- Bandeau info (mise à jour horaire)

## 3. COMPOSANTS UI
- **OdsSelect** : Sélecteur de base de données
- **OdsButtonGroup** : Filtres période (7 jours, 30 jours, 1 an)
- **OdsCard** : Conteneurs graphiques et résumé
- **OdsChart** : Graphiques linéaires avec aire
- **OdsMessage** : Bandeau info bleu

## 4. SÉLECTEURS
- **Base de données** : Dropdown avec toutes les bases (ex: mydatabase_wp)
- **Période** : Boutons radio (7 jours actif, 30 jours, 1 an)

## 5. GRAPHIQUE 1 : CONNEXIONS SIMULTANÉES
- Type : Ligne avec aire remplie
- Axe X : Jours (Lun - Sam)
- Axe Y : 10 à 50 connexions
- Couleur : Bleu primaire

## 6. GRAPHIQUE 2 : ESPACE DISQUE UTILISÉ
- Type : Ligne avec aire remplie
- Axe X : Jours
- Axe Y : 100Mo à 500Mo
- Couleur : Bleu primaire

## 7. TUILE RÉSUMÉ (5 KPIs)
| Métrique | Valeur |
|----------|--------|
| Connexions moyennes | 32 |
| Connexions max | 47 |
| Espace utilisé | 285 Mo |
| Quota disponible | 715 Mo |
| Tables | 52 |

## 8. BANDEAU INFO
- "ℹ️ Les statistiques sont mises à jour toutes les heures. Les données affichées peuvent avoir jusqu'à 60 minutes de retard."

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/database/{name}/statistics : Stats complètes
- GET /hosting/web/{serviceName}/database/{name}/statistics/connections : Connexions
- GET /hosting/web/{serviceName}/database/{name}/statistics/disk : Espace disque
- GET /hosting/web/{serviceName}/database/{name} : Infos base (tables, quota)
