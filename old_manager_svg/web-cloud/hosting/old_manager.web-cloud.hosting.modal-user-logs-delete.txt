# Cahier des Charges : web-cloud/hosting - Modal Supprimer utilisateur statistiques

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'un utilisateur statistiques

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container (header rouge/danger)
- **OdsText** : Question confirmation
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer l'utilisateur" |
| Question | "Êtes-vous sûr de vouloir supprimer l'utilisateur {nom} ?" |
| Confirmation | "Voulez-vous continuer ?" |

## 5. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Supprimer utilisateur |

## 6. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/userLogs/{login} - Supprimer utilisateur
