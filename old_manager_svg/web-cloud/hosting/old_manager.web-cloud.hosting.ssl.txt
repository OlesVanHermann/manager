# Cahier des Charges : web-cloud/hosting - Certificat SSL

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion du certificat SSL.

### Objectif fonctionnel
- Voir l'état du certificat
- Régénérer le certificat
- Commander un certificat payant
- Voir les domaines couverts

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/ssl
POST /hosting/web/{serviceName}/ssl/regenerate
DELETE /hosting/web/{serviceName}/ssl
POST /order/hosting/web/{serviceName}/ssl
```

### Modele
```typescript
interface SslCertificate {
  type: 'letsencrypt' | 'sectigo' | 'custom';
  provider: string;
  status: 'active' | 'creating' | 'error';
  creationDate: string;
  expirationDate: string;
  isAutoRenew: boolean;
  domains: string[];
}
```

## 6. REGLES METIER

- Let's Encrypt = gratuit, 90 jours
- Renouvellement auto 30 jours avant expiration
- Régénération = ajout nouveaux domaines
- Sectigo = certificat payant avec EV
