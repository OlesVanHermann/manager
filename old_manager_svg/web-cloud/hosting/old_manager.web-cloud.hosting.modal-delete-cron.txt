# Cahier des Charges : web/hosting - Modal supprimer tâche CRON

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation avec détails
- **Contexte** : Hébergement web - Tâches planifiées
- **Objectif** : Supprimer une tâche CRON

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 350px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question, labels
- **OdsCard** : Détails de la tâche
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la tâche planifiée" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Voulez-vous supprimer cette tâche CRON ?" |

## 6. CARTE DÉTAILS

### Informations affichées
| Champ | Exemple |
|-------|---------|
| Commande | /www/scripts/backup.php |
| Langage | PHP 8.1 |
| Fréquence | 0 2 * * * (Tous les jours à 2h) |
| Email | admin@example.com |
| Description | Sauvegarde quotidienne |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Supprimer | Danger (rouge) | Supprime la tâche |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/cron/{id} - Supprimer tâche CRON
