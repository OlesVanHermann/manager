# Cahier des Charges : web-cloud/hosting - Informations générales

## 1. RESUME EXECUTIF

### Perimetre
Page détaillée des informations générales de l'hébergement.

### Objectif fonctionnel
- Afficher toutes les informations du service
- Gérer la configuration
- Accéder aux liens utiles

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}
GET /hosting/web/{serviceName}/serviceInfos
GET /hosting/web/{serviceName}/ssl
GET /hosting/web/{serviceName}/ovhConfig
```

### Modele
```typescript
interface GeneralInformations {
  state: 'active' | 'suspended';
  ipv4: string;
  ipv6: string;
  diskUsage: { used: number; total: number; };
  datacenter: string;
  cluster: string;
  phpVersion: string;
  ssl: { type: string; state: string; };
  email: { state: string; };
  cdn: { enabled: boolean; type: string; };
  offer: string;
  renewal: { mode: string; date: string; };
  boost: boolean;
  contacts: { admin: string; tech: string; billing: string; };
  creationDate: string;
}
```
