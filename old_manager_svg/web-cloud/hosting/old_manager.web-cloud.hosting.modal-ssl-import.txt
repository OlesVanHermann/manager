# Cahier des Charges : web-cloud/hosting - Modal Importer certificat SSL

## 1. VUE D'ENSEMBLE
- **Modal** : Import d'un certificat SSL personnalisé
- **Type** : Formulaire multi-champs
- **Objectif** : Importer un certificat SSL externe (RSA)

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- 3 zones de saisie textarea
- Note informative
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsTextarea** : 3 zones de saisie (certificat, clé, chaîne)
- **OdsButton** : Annuler, Valider

## 4. CHAMPS DU FORMULAIRE

### Certificat (obligatoire)
- Label : "Copier le contenu de votre certificat (RSA uniquement) :"
- Textarea 120px de hauteur
- Placeholder PEM :
```
-----BEGIN CERTIFICATE-----
MIIDXTCCAkWgAwIBAgIJAJC1HiIAZAiUMA0Gcm...
Lf7UVAV8kcePf5ZVqN...
-----END CERTIFICATE-----
```

### Clé privée (obligatoire)
- Label : "Copier le contenu de votre clé privée (non chiffrée) :"
- Textarea 120px de hauteur
- Placeholder PEM :
```
-----BEGIN RSA PRIVATE KEY-----
MIIEowIBAAKCAQEA0Z3VS5JJcds3xfn/ygWyf...
kdjfhKJHDF98734hkjdf...
-----END RSA PRIVATE KEY-----
```

### Chaîne de certificats (optionnel)
- Label : "Copier le contenu de votre chaîne de certificats :"
- Textarea 120px de hauteur
- Placeholder PEM :
```
-----BEGIN CERTIFICATE-----
MIIFYjCCBEqgAwIBAgIQd70NbNs2+RrqIQ/E8F...
jdfg98734kjhKJHDFkjhdf98...
-----END CERTIFICATE-----
```

## 5. NOTE INFORMATIVE
"La chaîne de certificats est optionnelle mais recommandée pour une meilleure compatibilité."

## 6. BOUTONS FOOTER
- Annuler (secondaire)
- Valider (primaire)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/ssl : Importer certificat SSL
  - Body : { certificate, key, chain }
