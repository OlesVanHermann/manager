# Cahier des Charges : web-cloud/hosting - Modal Modifier runtime

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification runtime (Cloud Web)
- **URL** : N/A (composant modal)
- **Objectif** : Modifier les paramètres d'un runtime existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 480px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Labels et indication champs obligatoires
- **OdsInput** : Nom, Répertoire public, Environnement, Fichier démarrage
- **OdsSelect** : Type runtime
- **OdsTooltip** : Aide sur chaque champ (?)
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis | Valeur exemple | Tooltip |
|-------|------|--------|----------------|---------|
| Nom | Input text | Oui | Mon Runtime Node.js | - |
| Type | Select dropdown | Oui | nodejs22 | - |
| Répertoire public | Input text | Oui | public | ? |
| Environnement | Input text | Oui | production | ? |
| Fichier de démarrage | Input text | Oui | app.js | ? |

## 5. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer modifications |

## 6. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/runtime/{id} - Obtenir config runtime
- PUT /hosting/web/{serviceName}/runtime/{id} - Modifier runtime
