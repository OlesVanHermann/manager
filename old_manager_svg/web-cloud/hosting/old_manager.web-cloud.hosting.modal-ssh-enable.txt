# Cahier des Charges : web-cloud/hosting - Modal Activer SSH

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec sélection
- **Contexte** : Gestion des accès FTP/SSH
- **Objectif** : Activer l'accès SSH pour un utilisateur FTP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsSelect** : Utilisateur FTP
- **OdsRadio** : Type d'accès
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer l'accès SSH" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "L'accès SSH vous permet de vous connecter à votre" |
| Ligne 2 | "hébergement en ligne de commande." |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Utilisateur FTP | OdsSelect | Oui | Liste des utilisateurs FTP |

### Type d'accès
| Propriété | Valeur |
|-----------|--------|
| Label | "Type d'accès" |
| Type | OdsRadio (groupe) |

### Options d'accès
| Option | Description | Bordure si sélectionné |
|--------|-------------|------------------------|
| SFTP + SSH | Accès complet en ligne de commande | Bleue (#0050D7) |
| SFTP uniquement | Transfert de fichiers sécurisé | Bleue si sélectionné |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Activer | Primary (bleu #0050D7) | Active SSH |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/user - Liste des utilisateurs FTP
- PUT /hosting/web/{serviceName}/user/{login} - Modifier accès SSH
