# Cahier des Charges : web-cloud/hosting - Modal Créer utilisateur statistiques

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 3 étapes
- **URL** : N/A (composant modal)
- **Objectif** : Créer un utilisateur pour accéder aux statistiques OVHcloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Navigation 3 étapes
- **OdsInput** : Identifiant
- **OdsTextarea** : Description
- **OdsFormField** : Wrapper avec helper text
- **OdsButton** : Navigation wizard

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Utilisateur | Active (bleu) |
| 2 | Mot de passe | Inactive |
| 3 | Confirmation | Inactive |

## 5. FORMULAIRE (ÉTAPE 1)
| Champ | Type | Requis | Placeholder | Helper |
|-------|------|--------|-------------|--------|
| Identifiant | Input text | Oui | stats-user | Entre 2 et 255 caractères, uniquement lettres, chiffres, points et tirets |
| Description | Textarea | Non | Accès aux statistiques pour le marketing... | - |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant | Primary | Passer à l'étape mot de passe |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/userLogs - Créer utilisateur statistiques
