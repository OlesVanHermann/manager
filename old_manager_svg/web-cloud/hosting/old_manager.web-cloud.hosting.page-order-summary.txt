# Cahier des Charges : web-cloud/hosting - Page Commande Récapitulatif

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard (étape 4/4) - Récapitulatif et paiement
- **Contexte** : Commande d'hébergement web
- **Objectif** : Valider la commande et procéder au paiement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 1100px |
| Padding | 30px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étapes
- **OdsDatagrid** : Récapitulatif commande
- **OdsRadio** : Sélection mode paiement
- **OdsCheckbox** : Acceptation CGV
- **OdsBadge** : Prix gratuit
- **OdsButton** : Navigation et validation

## 8. APIS & ENDPOINTS
- GET /order/cart/{cartId} - Contenu du panier
- GET /me/paymentMean - Moyens de paiement
- POST /order/cart/{cartId}/checkout - Valider la commande
- POST /me/order/{orderId}/payWithRegisteredPaymentMean - Payer
