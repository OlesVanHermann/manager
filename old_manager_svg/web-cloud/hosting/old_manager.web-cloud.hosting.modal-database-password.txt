# Cahier des Charges : web-cloud/hosting - Modal Modifier mot de passe BDD

## 1. VUE D'ENSEMBLE
- **Modal** : Changement de mot de passe base de données
- **Type** : Formulaire simple
- **Objectif** : Modifier les credentials de la BDD

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Zone formulaire
- Règles de validation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsInput** : Mot de passe avec icône œil
- **OdsButton** : Annuler, Confirmer

## 4. FORMULAIRE
### Information
"Modifier le mot de passe de la base de données **mydb_hosting123**"

### Champs
| Champ | Type | Description |
|-------|------|-------------|
| Nouveau mot de passe | Password avec œil | Obligatoire |
| Confirmation | Password avec œil | Obligatoire |

## 5. RÈGLES DE VALIDATION
- Minimum 8 caractères
- Maximum 30 caractères
- Au moins une lettre majuscule
- Au moins une lettre minuscule
- Au moins un chiffre

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Confirmer (primaire, à droite)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/changePassword : Modifier MDP
