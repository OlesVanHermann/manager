# Cahier des Charges : web-cloud/hosting - Modal Supprimer sauvegarde BDD

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression dump
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'une sauvegarde de base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Question confirmation avec date
- **OdsMessage** : Warning irréversible (rouge)
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer la sauvegarde" |
| Question | "Êtes-vous sûr de vouloir supprimer la sauvegarde créée le {date} à {heure} ?" |

## 5. AVERTISSEMENT
| Type | Message |
|------|---------|
| Danger | "⚠ Cette action est irréversible. La sauvegarde sera définitivement supprimée." |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Supprimer la sauvegarde |

## 7. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/database/{database}/dump/{id} - Supprimer dump
