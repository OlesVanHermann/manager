# Cahier des Charges : web-cloud/hosting - Modal Logs tâche planifiée

## 1. VUE D'ENSEMBLE
- **Type** : Modal de visualisation des logs
- **Contexte** : Gestion des tâches CRON
- **Objectif** : Afficher les logs d'exécution d'une tâche planifiée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Informations, résumé
- **OdsBadge** : Statut d'exécution
- **Code block** : Zone des logs (fond sombre)

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Logs de la tâche planifiée" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Informations de la tâche
| Champ | Valeur |
|-------|--------|
| Commande | /www/scripts/backup.php |
| Fréquence | Chaque jour à 03:00 |

### Zone des logs
| Propriété | Valeur |
|-----------|--------|
| Fond | Sombre (#1F2937) |
| Police | Monospace |
| Taille | 10px |

### Couleurs des logs
| Type | Couleur |
|------|---------|
| Démarrage | Vert (#10B981) |
| Normal | Gris clair (#E5E7EB) |
| Succès | Bleu clair (#60A5FA) |
| Erreur | Rouge |

### Résumé d'exécution
| Propriété | Valeur |
|-----------|--------|
| Badge succès | "✓ Dernière exécution réussie" (vert #DCFCE7/#166534) |
| Durée | "Durée: 2min 31s" |
| Exit code | "Exit code: 0" |

## 6. FOOTER MODAL
Pas de boutons de footer (fermeture via × uniquement)

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cron/{id}/logs - Logs de la dernière exécution
- GET /hosting/web/{serviceName}/cron/{id} - Détails de la tâche
