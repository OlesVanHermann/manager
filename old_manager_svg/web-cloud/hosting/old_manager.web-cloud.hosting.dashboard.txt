# Cahier des Charges : web-cloud/hosting - Dashboard

## 1. RESUME EXECUTIF

### Perimetre
Vue dashboard avec informations générales organisées en 4 tuiles.

### Objectif fonctionnel
- Afficher les informations générales
- Montrer la configuration
- Résumer les ressources
- Gérer l'abonnement

---

## 2. LAYOUT

### Structure breadcrumb + titre + grille 2x2 tuiles

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}
GET /hosting/web/{serviceName}/serviceInfos
GET /hosting/web/{serviceName}/attachedDomain
GET /hosting/web/{serviceName}/database
```

### Modele
```typescript
interface DashboardSummary {
  info: {
    state: string;
    ipv4: string;
    ipv6: string;
    diskUsage: { used: number; total: number; percent: number; };
    datacenter: string;
    cluster: string;
  };
  config: {
    phpVersion: string;
    offer: string;
    ssl: { active: boolean; type: string; };
    cdn: { active: boolean; };
  };
  summary: {
    websites: { count: number; recommended: number; };
    databases: { count: number; max: number; };
    modules: number;
    cloudDatabases: number;
  };
  subscription: {
    creationDate: string;
    expirationDate: string;
    renewalMode: 'automatic' | 'manual';
  };
}
```
