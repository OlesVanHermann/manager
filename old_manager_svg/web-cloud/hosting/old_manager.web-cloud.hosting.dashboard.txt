# Cahier des Charges : web-cloud/hosting - Dashboard

## 1. VUE D'ENSEMBLE

**Type:** Page dashboard avec tiles
**Fonction:** Afficher un résumé de l'hébergement web
**Contexte:** Hébergement Web OVHcloud

## 2. DIMENSIONS

- Page: 1440x900px
- Background: #F9FAFB
- Sidebar: 240px

## 3. NAVIGATION

### Breadcrumb
- Hébergements / mywebsite.com

### Titre page
- "Informations générales" (24px, font-weight: 600)
