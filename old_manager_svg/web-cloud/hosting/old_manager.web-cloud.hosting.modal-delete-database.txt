# Cahier des Charges : web-cloud/hosting - Modal Delete Database

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation danger avec saisie
**Fonction:** Supprimer une base de données avec confirmation textuelle
**Contexte:** Action depuis l'onglet BDD

## 2. DIMENSIONS

- Modal: 500x280px
- Overlay: rgba(0,0,0,0.5)

## 3. HEADER MODAL

- Background: #FEE2E2 (rouge pâle)
- Hauteur: 56px
- Titre: "Supprimer la base de données" (couleur #DC2626)
- Bouton fermer: × (position droite)

## 4. CONTENU

### Message principal
- Texte: "Vous êtes sur le point de supprimer la base de données :"
- Nom BDD: "monsite_prod (MySQL 8.0)" (font-weight 600)

### Bannière warning (452x50px)
- Background: #FEF3C7 (jaune)
- Icône: !
- Texte ligne 1: "Cette action est irréversible. Toutes les données seront"
- Texte ligne 2: "définitivement supprimées."
- Couleur texte: #D97706

### Champ confirmation
- Label: "Tapez \"SUPPRIMER\" pour confirmer :"
- Input: 452x36px, background #F9FAFB

## 5. FOOTER MODAL

- Bouton gauche: "Annuler" (btn-outline gris, 100x36px)
- Bouton droit: "Supprimer" (btn-disabled #E5E7EB → btn-danger quand SUPPRIMER saisi)

## 6. VALIDATION

| Condition | État bouton |
|-----------|-------------|
| Champ vide | Désactivé (gris) |
| Texte != "SUPPRIMER" | Désactivé (gris) |
| Texte == "SUPPRIMER" | Activé (rouge #DC2626) |

## 7. APIS & ENDPOINTS

```
# Supprimer base de données
DELETE /hosting/web/{serviceName}/database/{name}
→ Tâche de suppression

# Vérifier dépendances avant suppression
GET /hosting/web/{serviceName}/database/{name}/dump
→ Proposer sauvegarde avant suppression
```
