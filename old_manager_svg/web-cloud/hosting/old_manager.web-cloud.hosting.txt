# Cahier des Charges : web - Dashboard Hébergement

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard hébergement web
- **Type** : Page avec onglets et tuiles
- **Objectif** : Vue synthétique de l'hébergement

## 2. STRUCTURE PAGE
- Sidebar gauche "HÉBERGEMENTS"
- Breadcrumb : "Web Cloud / Hébergements Web / monsite.fr"
- Titre avec badge statut
- Barre 6 onglets
- 2 rangées de 3 tuiles

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation (6 onglets)
- **OdsCard** : Tuiles informatives
- **OdsBadge** : Statut "Actif"
- **OdsProgress** : Barres quota
- **OdsButton** : Actions
