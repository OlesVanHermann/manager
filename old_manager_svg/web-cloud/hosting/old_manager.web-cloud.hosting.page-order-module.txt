# Cahier des Charges : web-cloud/hosting - Page Commande Étape Module

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard (étape 3/4)
- **Contexte** : Commande d'hébergement web
- **Objectif** : Choisir un module CMS à installer automatiquement (optionnel)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur page | 100% |
| Max-width | 1100px |
| Padding | 30px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étapes
- **OdsRadio** : Option de saut et sélection module
- **OdsCard** : Cartes des modules
- **OdsBadge** : Label Recommandé
- **OdsButton** : Navigation
