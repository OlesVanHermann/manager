# Cahier des Charges : web-cloud/private-database - Modal Supprimer base

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de base de données
- **Type** : Modal confirmation destructive
- **Objectif** : Confirmer la suppression irréversible d'une base

## 2. STRUCTURE MODAL
- Barre rouge en haut (danger)
- Icône warning rouge
- Titre et bouton fermer
- Question de confirmation
- Nom de la base (lecture seule)
- Avertissement irréversibilité
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x280)
- **OdsIcon** : Cercle rouge avec "!"
- **OdsText** : Question + warning
- **OdsCard** : Nom base (fond gris)
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer la base de données"

## 5. BARRE DANGER
Barre rouge (#DC2626) de 4px en haut de la modal

## 6. ICÔNE
Cercle rose (#FEE2E2) avec "!" rouge (#DC2626)

## 7. CONTENU
- Question: "Êtes-vous sûr de vouloir supprimer cette base de données ?"
- Nom affiché: "mydb_production" (fond gris #F3F4F6)
- Warning: "⚠ Cette action est irréversible. Toutes les données seront perdues."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Supprimer | Rouge (#DC2626) | Supprime la base |

## 9. APIS & ENDPOINTS
- DELETE /hosting/privateDatabase/{serviceName}/database/{databaseName}
