# Cahier des Charges : web-cloud/hosting - Modal Sélecteur hébergement BDD

## 1. VUE D'ENSEMBLE
- **Type** : Modal de sélection
- **Contexte** : Commande base de données
- **Objectif** : Sélectionner l'hébergement auquel associer une nouvelle base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titre, description
- **OdsSelect** : Sélection de l'hébergement
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander une base de données" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Titre | "Sélectionner un hébergement" |
| Texte | "Choisissez l'hébergement auquel associer la base de données" |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Hébergement | OdsSelect | Oui | Liste des hébergements |

### Format du select
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | Domaine (www.example.com) |
| Ligne 2 | Identifiant technique (hosting123456.ovh.net) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Commander | Primary (bleu #0050D7) | Lance la commande |

## 7. APIS & ENDPOINTS
- GET /hosting/web - Liste des hébergements
- POST /order/cart/{cartId}/webHosting - Commander base de données
