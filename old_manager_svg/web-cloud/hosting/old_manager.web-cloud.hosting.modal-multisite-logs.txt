# Cahier des Charges : web-cloud/hosting - Modal Logs multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec toggles
- **Contexte** : Configuration multisite
- **Objectif** : Configurer les logs et statistiques pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, descriptions
- **OdsToggle** : Logs séparés, OWstats
- **OdsMessage** : Information
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration des logs" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Domaine
| Champ | Valeur |
|-------|--------|
| Domaine | shop.example.com |

### Option Logs séparés
| Propriété | Valeur |
|-----------|--------|
| Label | "Logs séparés" |
| Description | "Activer des logs dédiés pour ce domaine" |
| Type | OdsToggle |
| Couleur activé | Vert (#10B981) |

### Option Statistiques OWstats
| Propriété | Valeur |
|-----------|--------|
| Label | "Statistiques OWstats" |
| Description | "Activer les statistiques de visite" |
| Type | OdsToggle |
| Couleur activé | Vert (#10B981) |

### Information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | "Les logs seront disponibles sous logs.example.com" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Enregistrer | Primary (bleu #0050D7) | Sauvegarde configuration |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Config actuelle
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier config logs
