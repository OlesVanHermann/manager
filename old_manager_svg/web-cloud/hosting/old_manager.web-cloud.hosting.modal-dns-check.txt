# Cahier des Charges : web-cloud/hosting - Modal Vérification DNS

## 1. VUE D'ENSEMBLE
- **Type** : Modal de diagnostic DNS
- **Contexte** : Configuration du domaine
- **Objectif** : Vérifier la configuration DNS d'un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsBadge** : Statut de chaque enregistrement
- **OdsMessage** : Recommandation
- **OdsButton** : Relancer, Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Vérification de la configuration DNS" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Information domaine
| Champ | Valeur |
|-------|--------|
| Domaine | www.example.com |

### Titre section
| Propriété | Valeur |
|-----------|--------|
| Texte | "Enregistrements DNS détectés" |
| Style | 14px, gras |

### Liste des enregistrements
| Type | Valeur | Statut |
|------|--------|--------|
| A | 51.210.xxx.xxx | Correct (vert) |
| AAAA | 2001:41d0:xxx::xxx | Correct (vert) |
| CNAME | www → example.com | Manquant (orange) |
| MX | mx1.mail.ovh.net (priority: 1) | Correct (vert) |

### Styles des badges
| État | Fond | Texte |
|------|------|-------|
| Correct | #DCFCE7 | #166534 |
| Manquant | #FEF3C7 | #92400E |
| Erreur | #FEE2E2 | #DC2626 |

### Icônes état
| État | Icône | Fond cercle |
|------|-------|-------------|
| OK | ✓ | #DCFCE7 |
| Warning | ! | #FEF3C7 |
| Error | × | #FEE2E2 |

### Recommandation
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Texte | "Ajoutez un enregistrement CNAME pour www vers example.com" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Relancer | Secondary (bordure grise) | Relance le check |
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain/{domain}/dnsStatus - Vérification DNS
- GET /domain/{domain}/zone/record - Enregistrements DNS
