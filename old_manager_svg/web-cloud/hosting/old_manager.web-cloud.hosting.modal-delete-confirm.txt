# Cahier des Charges : web-cloud/hosting - Modal Delete Confirm

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation danger
**Fonction:** Confirmer la suppression d'une ressource générique
**Contexte:** Action de suppression depuis divers onglets

## 2. DIMENSIONS

- Modal: 500x280px
- Overlay: rgba(0,0,0,0.5)

## 3. HEADER MODAL

- Background: #FEE2E2 (rouge pâle)
- Hauteur: 56px
- Titre: "Confirmer la suppression" (couleur #DC2626)
- Bouton fermer: × (position droite)

## 4. CONTENU

### Message principal
- Texte: "Êtes-vous sûr de vouloir supprimer cette ressource ?"

### Détail ressource
- Label: "RESSOURCE" (muted, font-weight 600)
- Valeur: "Base de données : monsite_wordpress" (font-weight 500)

### Bannière warning (452x50px)
- Background: #FEE2E2
- Texte ligne 1: "Cette action est irréversible. Toutes les données seront"
- Texte ligne 2: "définitivement supprimées."
- Couleur texte: #DC2626

## 5. FOOTER MODAL

- Bouton gauche: "Annuler" (btn-outline, 100x36px)
- Bouton droit: "Supprimer" (btn-danger #DC2626, 100x36px)

## 6. VARIANTES

Ce modal générique s'adapte selon le contexte :

| Ressource | Texte spécifique |
|-----------|------------------|
| Base de données | Base de données : {name} |
| Utilisateur FTP | Utilisateur FTP : {name} |
| Module | Module : {name} |
| Tâche cron | Tâche cron : {name} |
| Variable env | Variable : {name} |

## 7. APIS & ENDPOINTS

```
# Suppression base de données
DELETE /hosting/web/{serviceName}/database/{name}

# Suppression utilisateur FTP
DELETE /hosting/web/{serviceName}/user/{login}

# Suppression module
DELETE /hosting/web/{serviceName}/module/{id}

# Suppression tâche cron
DELETE /hosting/web/{serviceName}/cron/{id}

# Suppression variable environnement
DELETE /hosting/web/{serviceName}/envVar/{key}
```
