# Cahier des Charges : web-cloud/hosting - Modal PHP Version

## 1. VUE D'ENSEMBLE
- **Type** : Modal changement version PHP
- **Contexte** : Web Cloud Hosting
- **Objectif** : Modifier la version PHP de l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsCard** : Sélection version (style bouton)
- **OdsBadge** : Statut version
- **OdsMessage** : Warning + Info
- **OdsButton** : Annuler, Appliquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Changer la version PHP" |
| Bouton fermer | × |
