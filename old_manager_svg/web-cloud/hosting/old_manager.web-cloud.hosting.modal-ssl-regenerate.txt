# Cahier des Charges : web-cloud/hosting - Modal Régénérer certificat SSL

## 1. VUE D'ENSEMBLE
- **Modal** : Régénération du certificat SSL
- **Type** : Modal de confirmation
- **Objectif** : Régénérer le certificat SSL (Let's Encrypt)

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Question de confirmation
- Bandeau warning jaune
- Bandeau info bleu
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsMessage** : Bandeau warning jaune
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Régénérer

## 4. QUESTION
"Souhaitez-vous régénérer votre certificat SSL ?"

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron :
- "⚠ La régénération peut prendre quelques minutes"
- "durant lesquelles le HTTPS sera indisponible."

## 6. BANDEAU INFO
Fond bleu (#DBEAFE), texte bleu foncé :
- "ℹ La régénération est utile si :"
- "• Vous avez ajouté de nouveaux domaines"
- "• Vous souhaitez mettre à jour le certificat"

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Régénérer (primaire)

## 8. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/ssl/regenerate : Régénérer certificat SSL
