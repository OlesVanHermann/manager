# Cahier des Charges : web-cloud/hosting - Modal Détacher BDD privée

## 1. VUE D'ENSEMBLE
- **Type** : Page détachement base de données
- **URL** : /hosting/{serviceName}/database/detach
- **Objectif** : Détacher une base de données SQL privée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 900px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsBackLink** : Retour navigation
- **OdsText** : Description contextuelle
- **OdsMessage** : Warning irréversible (rouge)
- **OdsCard** : Container options
- **OdsRadio** : Choix détachement
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Détacher la base de données SQL privée" |
| Description | "Vous allez détacher la base de données privée {nom} de l'hébergement {hébergement}" |

## 5. AVERTISSEMENT
| Type | Message |
|------|---------|
| Danger | "⚠ Attention : cette action est irréversible !" |

## 6. OPTIONS DE DÉTACHEMENT
| Option | Sélection | Description |
|--------|-----------|-------------|
| Conserver en service indépendant | Par défaut (●) | La base de données sera convertie en service Web Cloud Databases |
| Supprimer définitivement | ○ | La base de données et toutes ses données seront supprimées |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour page précédente |
| Détacher | Primary | Confirmer détachement |

## 8. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/privateDatabases/{name}/detach - Détacher BDD privée
- DELETE /hosting/web/{serviceName}/privateDatabases/{name} - Supprimer BDD privée
