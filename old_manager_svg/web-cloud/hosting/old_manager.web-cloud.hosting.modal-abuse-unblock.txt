# Cahier des Charges : web-cloud/hosting - Modal Débloquer après abus

## 1. VUE D'ENSEMBLE
- **Type** : Modal déblocage hébergement
- **URL** : N/A (composant modal)
- **Objectif** : Débloquer hébergement après signalement d'abus

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container avec header warning
- **OdsMessage** : Alerte danger (motif blocage)
- **OdsText** : Instructions étapes
- **OdsLink** : Guide aide
- **OdsCheckbox** : Confirmation nettoyage
- **OdsButton** : Actions

| Propriété | Valeur |
