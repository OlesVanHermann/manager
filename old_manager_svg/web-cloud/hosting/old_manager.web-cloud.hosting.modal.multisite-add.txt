# Cahier des Charges : web-cloud/hosting - Modal Ajouter domaine (Multisite)

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un domaine multisite
- **Type** : Wizard 3 étapes
- **Objectif** : Configurer un nouveau domaine sur l'hébergement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 3 étapes
- Formulaire principal
- Zone options (checkboxes)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsStepper** : 3 étapes horizontales
- **OdsSelect** : Type de domaine
- **OdsInput** : Nom de domaine, dossier racine
- **OdsCheckbox** : Options SSL, CDN, Firewall, www
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Domaine | Actif (cercle bleu) |
| 2 | Configuration | À venir (cercle gris) |
| 3 | Validation | À venir (cercle gris) |

## 5. CHAMPS DU FORMULAIRE

### Type de domaine
- Type : Select dropdown
- Options : "Domaine déjà enregistré", "Sous-domaine", "Domaine externe"

### Nom de domaine
- Type : Input texte
- Placeholder : "www.nouveau-site.com"

### Dossier racine
- Type : Input texte
- Exemple : "/www/nouveau-site"

## 6. OPTIONS (Checkboxes)
| Option | Coché |
|--------|-------|
| Activer SSL (Let's Encrypt) | ✓ Oui |
| Activer le CDN | ○ Non |
| Activer le firewall | ○ Non |
| Créer également www.nouveau-site.com | ✓ Oui |

## 7. BOUTONS FOOTER
- Annuler (secondaire gris)
- Suivant (primaire bleu)

## 8. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/attachedDomain : Ajouter domaine multisite
  - Body : { domain, path, ssl, cdn, firewall }
- GET /domain : Liste domaines disponibles
