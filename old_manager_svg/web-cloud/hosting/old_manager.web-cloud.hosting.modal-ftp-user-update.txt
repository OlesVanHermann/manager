# Cahier des Charges : web-cloud/hosting - Modal Modifier utilisateur FTP

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un utilisateur FTP existant
- **Type** : Formulaire simple
- **Objectif** : Modifier les paramètres d'accès FTP

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Zone formulaire
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsInput** : Dossier racine avec préfixe ./
- **OdsRadio** : Protocole d'accès
- **OdsToggle** : État de l'utilisateur
- **OdsButton** : Annuler, Confirmer

## 4. INFORMATION
"Modifiez les paramètres de l'utilisateur FTP **hosting123-webmaster**"

## 5. FORMULAIRE
### Dossier racine
- Préfixe : "./" (non-éditable, fond grisé)
- Input : "www"

### Protocole d'accès
| Option | État |
|--------|------|
| FTP uniquement | ○ Non sélectionné |
| FTP et SFTP | ● Sélectionné |
| FTP, SFTP et SSH | ○ Non sélectionné |

### État de l'utilisateur
- Toggle switch activé (bleu)
- Label : "Activé"

## 6. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Confirmer (primaire, à droite)

## 7. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/user/{login} : Modifier utilisateur
- GET /hosting/web/{serviceName}/user/{login} : Détails utilisateur
