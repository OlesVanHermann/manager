# Cahier des Charges : web-cloud/hosting - Environnements d'exécution

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des runtimes
- **Objectif** : Configurer environnements PHP/Node.js
- **Contexte** : Configuration versions langages

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud > Hébergements > hosting123 > Environnements d'exécution
- Titre + bouton Actions ▼
- Tableau des environnements

## 3. COMPOSANTS UI
- **OdsButton** : "Actions ▼" (dropdown)
- **OdsTable** : Liste environnements
- **OdsCheckbox** : Colonne "Défaut" (coché = runtime par défaut)
- **OdsBadge** : Statut (Créé, Suppression)
- **OdsActionMenu** : Menu ⋮

## 4. TABLEAU ENVIRONNEMENTS
| Colonne | Description |
|---------|-------------|
| Défaut | Checkbox (un seul coché) |
| Nom | Identifiant du runtime |
| Langage | PHP 8.2, Node.js 18, etc. |
| Multisites associés | Nombre de sites utilisant ce runtime |
| Statut | Badge Créé (vert) / Suppression (rouge) |
| Actions | Menu ⋮ |

## 5. EXEMPLES DONNÉES
| Défaut | Nom | Langage | Sites | Statut |
|--------|-----|---------|-------|--------|
| ✓ | php8.2-default | PHP 8.2 | 5 | Créé |
| ○ | php8.1-legacy | PHP 8.1 | 3 | Créé |
| ○ | nodejs18 | Node.js 18 | 1 | Créé |
| ○ | php7.4-old | PHP 7.4 | 0 | Suppression |

## 6. MENU ACTIONS (⋮)
- Modifier
- Définir par défaut (grisé si déjà défaut)
- Supprimer

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/runtime : Liste runtimes
- POST /hosting/web/{serviceName}/runtime : Créer runtime
- PUT /hosting/web/{serviceName}/runtime/{id} : Modifier
- DELETE /hosting/web/{serviceName}/runtime/{id} : Supprimer
- PUT /hosting/web/{serviceName}/runtime/{id}/setDefault : Définir par défaut
