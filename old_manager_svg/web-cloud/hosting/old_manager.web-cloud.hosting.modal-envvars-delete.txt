# Cahier des Charges : web-cloud/hosting - Modal Supprimer variable environnement

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression variable
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'une variable d'environnement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container (header rouge/danger)
- **OdsText** : Question confirmation
- **OdsText** : Détails variable (3 lignes)
- **OdsMessage** : Warning impact (jaune)
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer la variable" |
| Question | "Êtes-vous sûr de vouloir supprimer cette variable ?" |

## 5. DÉTAILS AFFICHÉS
| Champ | Exemple |
|-------|---------|
| Clé | DATABASE_URL |
| Type | string |
| Valeur | mysql://user:***@host/db |

## 6. AVERTISSEMENT
| Type | Message |
|------|---------|
| Warning | "⚠ Cette suppression peut affecter le fonctionnement de votre site." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Supprimer variable |

## 8. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/envVar/{key} - Supprimer variable
