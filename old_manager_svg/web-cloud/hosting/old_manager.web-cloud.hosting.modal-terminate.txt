# Cahier des Charges : web-cloud/hosting - Modal Supprimer hébergement

## 1. VUE D'ENSEMBLE
- **Modal** : Résiliation/suppression de l'hébergement
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer définitivement l'hébergement web

## 2. STRUCTURE MODAL
- Header gris avec titre et bouton fermer (×)
- Bandeau warning jaune
- Description
- Question de confirmation
- Bandeau info bleu
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsMessage** : Bandeau warning jaune
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange, texte marron :
- "⚠️ Attention"
- "La suppression d'un hébergement entraîne l'effacement définitif de toutes les"
- "données correspondantes. L'action de supprimer un hébergement n'octroie aucun remboursement."

## 5. DESCRIPTION
"Un e-mail contenant la procédure de fermeture de votre hébergement vous sera envoyé."

## 6. QUESTION DE CONFIRMATION
"**Voulez-vous continuer ?**"

## 7. BANDEAU INFO
Fond bleu (#E8F0FE) :
- "ℹ️ Cette action est irréversible. Assurez-vous d'avoir sauvegardé vos données avant de continuer."
- "Vos fichiers FTP, bases de données et emails seront définitivement supprimés."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 9. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/terminate : Résilier hébergement
