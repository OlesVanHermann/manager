# Cahier des Charges : web-cloud/hebergement - Tab Multisite

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des domaines attachés à l'hébergement.

### Objectif fonctionnel
- Lister les domaines configurés
- Ajouter de nouveaux domaines
- Configurer SSL, CDN, firewall par domaine

## 3. COMPOSANTS UI

### Toolbar
| Element | Description |
|---------|-------------|
| Bouton Actions | Dropdown primary |

### Menu Actions
| Option | Description |
|--------|-------------|

### Tableau multisite
| Colonne | Description |
|---------|-------------|
| Domaine | Nom de domaine (link) |
| Dossier racine | Chemin relatif |
| SSL | Badge Oui/Non |
| Firewall | Activé/Désactivé |
| CDN | Activé/- |
| Logs | Lien Voir |
| IPv6 | Activé/Désactivé |
| Actions | Menu ⋮ |

### Menu actions ligne (⋮)
| Option | Description |
|--------|-------------|
| Modifier le domaine | Edit |
| Modifier le dossier racine | Edit path |
| Activer/Désactiver SSL | Toggle SSL |
| Vider le cache CDN | Purge cache |
| Supprimer | Destructif (rouge) |

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/attachedDomain
POST /hosting/web/{serviceName}/attachedDomain
PUT /hosting/web/{serviceName}/attachedDomain/{domain}
DELETE /hosting/web/{serviceName}/attachedDomain/{domain}
```

### Modele
```typescript
interface AttachedDomain {
  domain: string;
  path: string;
  ssl: boolean;
  firewall: boolean;
  cdn: boolean;
  ipv6: boolean;
  ownLog: string;
}
```

- CDN optionnel selon offre

- IPv6 optionnel par domaine
