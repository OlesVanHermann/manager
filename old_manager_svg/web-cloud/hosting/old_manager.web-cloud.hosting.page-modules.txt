# Cahier des Charges : web-cloud/hosting - Page Modules

## 1. VUE D'ENSEMBLE
- **Type** : Page modules CMS en 1 clic
- **Contexte** : Web Cloud Hosting
- **Objectif** : Installer et gérer des CMS (WordPress, PrestaShop...)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour, Gérer
- **OdsCard** : Modules installés + disponibles
- **OdsBadge** : Statut mise à jour
- **OdsButton** : Installer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour à {serviceName}" |
| Titre | "Modules en 1 clic" |
| Sous-titre | "{serviceName} • Installez facilement WordPress, PrestaShop, Drupal..." |

## 5. MODULES INSTALLÉS
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modules installés (X)" |
| Format carte | Logo + Nom version + URL + Badge + Gérer |

## 6. CARTE MODULE INSTALLÉ
| Élément | Description |
|---------|-------------|
| Logo | Carré coloré avec initiale |
| Nom | Nom CMS + version |
| URL | Chemin installation |
| Badge | Statut mise à jour |
| Action | Lien "Gérer" bleu |

## 7. BADGES MISE À JOUR
| Statut | Fond | Texte |
|--------|------|-------|
| À jour | #DCFCE7 | #16A34A |
| MAJ dispo | #FEF3C7 | #F59E0B |

## 8. MODULES DISPONIBLES (grille)
| Module | Couleur logo | Catégorie |
|--------|--------------|-----------|
| WordPress | #21759B | CMS / Blog |
| PrestaShop | #DF0067 | E-commerce |
| Drupal | #0678BE | CMS avancé |

## 9. CARTE MODULE DISPONIBLE
| Élément | Description |
|---------|-------------|
| Logo | Carré coloré 50x50 |
| Nom | Nom CMS (gras) |
| Catégorie | Type (gris) |
| Bouton | "Installer" Primary |

## 10. VOIR PLUS
| Propriété | Valeur |
|-----------|--------|
| Style | Carte dashed |
| Icône | + |
| Texte | "Voir tous les modules" |

## 11. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/module - Modules installés
- GET /hosting/web/moduleList - Modules disponibles
- POST /hosting/web/{serviceName}/module - Installer module
- DELETE /hosting/web/{serviceName}/module/{id} - Supprimer
