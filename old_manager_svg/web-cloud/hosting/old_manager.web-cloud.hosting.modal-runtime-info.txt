# Cahier des Charges : web-cloud/hosting - Modal Informations runtime

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative (lecture seule)
- **Contexte** : Gestion des runtimes
- **Objectif** : Afficher les détails d'un runtime configuré

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsBadge** : Statut (Par défaut)
- **OdsButton** : Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Informations du runtime" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Informations du runtime
| Champ | Valeur exemple |
|-------|----------------|
| Nom | PHP 8.2 Production |
| Langage | PHP |
| Version | 8.2 |
| Chemin public | /www/public |
| Répertoire d'upload | /www |
| Statut | Badge "Par défaut" (vert #DCFCE7/#166534) |

### Domaines attachés
| Propriété | Valeur |
|-----------|--------|
| Titre section | "Domaines attachés" |
| Fond | #F9FAFB |
| Format | Liste des domaines utilisant ce runtime |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary (bleu #0050D7) | Ferme modal |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/runtime/{id} - Détails du runtime
- GET /hosting/web/{serviceName}/attachedDomain?runtime={id} - Domaines attachés
