# Cahier des Charges : web-cloud/hosting - Modal Ajouter variable environnement

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout variable
- **URL** : N/A (composant modal)
- **Objectif** : Ajouter une nouvelle variable d'environnement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Labels et indication champs obligatoires
- **OdsInput** : Clé et Valeur
- **OdsSelect** : Type variable
- **OdsFormField** : Wrapper avec helper text
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis | Placeholder | Helper |
|-------|------|--------|-------------|--------|
| Clé | Input text | Oui | MY_VARIABLE | Caractères alphanumériques et _ uniquement |
| Type | Select dropdown | Oui | string | - |
| Valeur | Input text | Oui | production | - |

## 5. OPTIONS TYPE
| Type | Description |
|------|-------------|
| string | Chaîne de caractères |
| integer | Nombre entier |
| password | Valeur masquée |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Ajouter | Primary | Créer variable |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/envVar - Créer variable d'environnement
