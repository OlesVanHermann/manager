# Cahier des Charges : web-cloud/hosting - Page FTP

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion comptes FTP
- **Contexte** : Web Cloud Hosting FTP
- **Objectif** : Gérer les accès FTP/SFTP de l'hébergement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour
- **OdsButton** : Créer compte
- **OdsCard** : Cartes informations
- **OdsBadge** : Protocoles, Principal
- **OdsProgressBar** : Quota disque
- **OdsDatagrid** : Liste comptes

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour à {serviceName}" |
| Titre | "Comptes FTP" |
| Sous-titre | "{serviceName} • Gérez les accès FTP/SFTP" |
| Bouton | "+ Créer un compte FTP" |

## 5. CARTE INFORMATIONS DE CONNEXION
| Champ | Format |
|-------|--------|
| Serveur FTP | Monospace "ftp.cluster001.hosting.ovh.net" |
| Port | "21 (FTP) / 22 (SFTP)" |
| Protocoles supportés | Badges FTP, FTPS, SFTP (tous verts) |

## 6. CARTE QUOTA DISQUE
| Propriété | Valeur |
|-----------|--------|
| Espace utilisé | "XXX Go" gros |
| Espace total | "/ YYY Go" |
| Barre | OdsProgressBar |
| Détails | "XX% utilisé • YYY Go disponibles" |

## 7. DATAGRID COMPTES FTP
| Colonne | Description |
|---------|-------------|
| Utilisateur | Nom + badge "Principal" (si applicable) |
| Répertoire | Chemin monospace |
| Quota | "Illimité" ou "XX Go" |
| Dernière connexion | "Il y a X heures" ou "Jamais" |
| Actions | Mot de passe, Modifier, Supprimer |

## 8. BADGE PRINCIPAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #0050D7 |
| Texte | "Principal" white |

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ftp - Liste comptes FTP
- POST /hosting/web/{serviceName}/ftp - Créer compte
- PUT /hosting/web/{serviceName}/ftp/{login} - Modifier
- DELETE /hosting/web/{serviceName}/ftp/{login} - Supprimer
- POST /hosting/web/{serviceName}/ftp/{login}/password - Changer mot de passe
