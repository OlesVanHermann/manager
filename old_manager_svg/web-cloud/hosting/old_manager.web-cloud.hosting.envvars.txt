# Cahier des Charges : web-cloud/hosting - Variables d'environnement

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion variables d'environnement
- **Objectif** : Configurer les variables d'env de l'hébergement
- **Contexte** : Configuration applicative sécurisée

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud > Hébergements > hosting123 > Variables d'environnement
- Titre + bouton Actions ▼
- Tableau des variables

## 3. COMPOSANTS UI
- **OdsButton** : "Actions ▼" (dropdown)
- **OdsTable** : Liste variables
- **OdsBadge** : Statut (Créée, En création)
- **OdsActionMenu** : Menu ⋮ (Modifier, Supprimer)

## 4. TABLEAU VARIABLES
| Colonne | Description |
|---------|-------------|
| Clé | Nom de la variable |
| Type | string, password, boolean |
| Valeur | Valeur ou masqué (••••) pour password |
| Statut | Badge Créée (vert) / En création (jaune) |
| Actions | Menu ⋮ |

## 5. EXEMPLES DONNÉES
| Clé | Type | Valeur | Statut |
|-----|------|--------|--------|
| DATABASE_URL | string | mysql://user:***@host/db | Créée |
| APP_ENV | string | production | Créée |
| API_KEY | password | •••••••••••• | Créée |
| DEBUG_MODE | boolean | false | En création |
| SMTP_HOST | string | ssl0.ovh.net | Créée |

## 6. MENU ACTIONS (⋮)
- Modifier
- Supprimer

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/envVar : Liste variables
- POST /hosting/web/{serviceName}/envVar : Créer variable
- PUT /hosting/web/{serviceName}/envVar/{key} : Modifier
- DELETE /hosting/web/{serviceName}/envVar/{key} : Supprimer
