# Cahier des Charges : web-cloud/hosting - Modal Supprimer module

## 1. VUE D'ENSEMBLE
- **Type** : Modal suppression
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'un module CMS installé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container (header rouge/danger)
- **OdsText** : Question confirmation
- **OdsMessage** : Warning conséquences (jaune)
- **OdsLink** : Lien vers gestion BDD
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer le module" |
| Question | "Êtes-vous sûr de vouloir supprimer le module {nom} ?" |

## 5. AVERTISSEMENT
| Type | Message |
|------|---------|
| Warning | "⚠ Les fichiers du module seront supprimés définitivement. La base de données associée ne sera pas supprimée." |
| Lien | "Gérer mes bases de données ↗" |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Supprimer module |

## 7. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/module/{id} - Supprimer module
