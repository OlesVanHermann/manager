# Cahier des Charges : web-cloud/private-database - Modal Ajouter IP whitelist

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'une adresse IP à la whitelist
- **Type** : Modal formulaire
- **Objectif** : Autoriser une nouvelle IP à se connecter

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Champ adresse IP (obligatoire)
- Champ nom (optionnel)
- Checkbox accès SFTP
- Bandeau info
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x420)
- **OdsInput** : Adresse IP, Nom
- **OdsCheckbox** : Accès SFTP
- **OdsMessage** : Info bleu
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter une IP à la whitelist"

## 5. CHAMPS
### Adresse IP *
- Type: Input obligatoire
- Placeholder/Exemple: "192.168.1.100"
- Validation: Format IP valide (IPv4 ou IPv6)

### Nom (optionnel)
- Type: Input optionnel
- Placeholder/Exemple: "Serveur production"
- Usage: Label descriptif pour identifier l'IP

### Accès SFTP
- Type: Checkbox
- État: Non coché par défaut
- Label: "Autoriser accès au service SFTP"

## 6. BANDEAU INFO
Fond bleu (#DBEAFE) :
- "ℹ L'adresse IP sera autorisée à se connecter à votre base de données."
- "La propagation peut prendre quelques minutes."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Ajouter | Primaire bleu | Ajoute l'IP |

## 8. APIS & ENDPOINTS
- POST /hosting/privateDatabase/{serviceName}/whitelist
  - ip: string (adresse IP)
  - name: string (optionnel)
  - sftp: boolean (accès SFTP)
