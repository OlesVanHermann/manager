# Cahier des Charges : web-cloud/hebergement - Tab General

## 1. RESUME EXECUTIF

### Perimetre
Onglet informations générales de l'hébergement web.

### Objectif fonctionnel
- Afficher les informations techniques
- Consulter la configuration
- Gérer l'offre et options

---

## 2. LAYOUT

### Structure 3 cartes horizontales

---

## 3. COMPOSANTS UI

### Carte Informations générales
| Champ | Type | Description |
|-------|------|-------------|
| État du service | Badge | Actif (vert) / Suspendu (rouge) |
| IPv4 | Clipboard | Adresse IP + copie |
| IPv6 | Clipboard | Adresse IPv6 + copie |
| Espace disque | Progress bar | Usage en Go et % |
| Datacenter | Texte | Localisation |
| Cluster | Texte | Numéro cluster |
| Filer | Texte | Numéro filer |
| Sites web | Texte | X / Y recommandés |

### Carte Configuration
| Champ | Type | Description |
|-------|------|-------------|
| Version PHP | Badge | Version + supportée |
| Moteur | Texte | php |
| Environnement | Texte | production |
| Application Firewall | Texte | Activé |
| Bouton | Outline | Modifier la configuration |

### Carte Offre et options
| Champ | Type | Description |
|-------|------|-------------|
| Offre actuelle | Texte bold | Performance 1/2/3 |
| Boost | Texte | Activé/Désactivé |
| CDN | Texte | CDN Basic |
| Certificat SSL | Badge | Let's Encrypt (vert) |
| Liens | Link | Changer d'offre, Commander SSL, Gérer CDN |
| Danger | Link rouge | Supprimer |

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}
GET /hosting/web/{serviceName}/ovhConfig
```

### Modele
```typescript
interface HostingGeneral {
  state: 'active' | 'suspended';
  ipv4: string;
  ipv6: string;
  quotaUsed: number;
  quotaSize: number;
  datacenter: string;
  cluster: string;
  filer: string;
  multisiteCount: number;
  multisiteLimit: number;
}

interface HostingConfig {
  engineName: string;
  engineVersion: string;
  environment: string;
  firewall: boolean;
}
```

## 6. REGLES METIER

- IPv4 et IPv6 copiables dans le presse-papier
- Progress bar colorée selon usage (vert < 70%, jaune < 90%, rouge >= 90%)
- Configuration modifiable via fichier .ovhconfig
