# Cahier des Charges : web-cloud/hosting - Modal Ajouter tâche planifiée (Cron)

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une tâche cron
- **Type** : Formulaire complet
- **Objectif** : Planifier l'exécution automatique d'un script

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire multi-champs
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsInput** : Champs description, commande, expression cron, email
- **OdsSelect** : Sélection langage, fréquence
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Créer

## 4. CHAMPS DU FORMULAIRE

### Description
- Type : Input texte
- Placeholder : "Ex: Nettoyage cache quotidien"

### Commande à exécuter
- Type : Input texte
- Exemple : "/www/scripts/cleanup.php"

### Langage
- Type : Select dropdown
- Options : PHP 8.2, Node.js, Python, etc.
- Note : "ou Node.js, Python, etc."

### Fréquence d'exécution
- Type : Select dropdown
- Exemple : "Tous les jours à 03:00"

### Expression Cron (avancé)
- Type : Input texte monospace
- Exemple : "0 3 * * *"
- Note : "Format: minute heure jour mois jour_semaine"

### Notification email (optionnel)
- Type : Input email
- Placeholder : "admin@example.com"

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7) :
"⚠️ Les logs d'exécution seront disponibles dans l'onglet «Logs»."

## 6. BOUTONS FOOTER
- Annuler (secondaire gris)
- Créer (primaire bleu)

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/cron : Créer tâche cron
  - Body : { command, language, frequency, email, description }
