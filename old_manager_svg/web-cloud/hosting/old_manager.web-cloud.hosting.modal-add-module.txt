## 8. APIS & ENDPOINTS

```
# Lister les modules disponibles
GET /hosting/web/moduleList
→ Liste des CMS installables (WordPress, PrestaShop, Joomla, Drupal)

# Installer un module
POST /hosting/web/{serviceName}/module
Body: {
  moduleId: "wordpress",
  domain: "monsite.example.com",
  path: "/www/wordpress",
  adminName: "admin",
  adminPassword: "***",
  adminMail: "admin@example.com",
  language: "fr"
}
→ Tâche d'installation

# Vérifier prérequis
GET /hosting/web/{serviceName}/module/check
Query: { moduleId: "wordpress" }
→ Vérification espace disque, PHP version, etc.
```

## 9. MODULES DISPONIBLES
