# Cahier des Charges : web-cloud/hosting - Modal Ajouter/Commander domaine

## 1. VUE D'ENSEMBLE
- **Type** : Modal choix initial
- **URL** : N/A (composant modal)
- **Objectif** : Choisir entre commander ou attacher un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Question
- **OdsCard** : Options radio (2 cartes)
- **OdsRadio** : Sélection option
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Ajouter un domaine" |
| Question | "Que souhaitez-vous faire ?" |

## 5. OPTIONS (RADIO CARDS)
| Option | Sélection | Titre | Description |
|--------|-----------|-------|-------------|
| Commander | Par défaut (●) | "Commander un nouveau domaine" | "Enregistrez un nouveau nom de domaine et associez-le à cet hébergement" |
| Attacher | ○ | "Attacher un domaine existant" | "Associez un domaine que vous possédez déjà à cet hébergement" |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant | Primary | Passer à l'étape suivante |

## 7. NAVIGATION
- Si "Commander" → Redirection vers page commande domaine
- Si "Attacher" → Ouvre modal ajout domaine OVHcloud ou externe
