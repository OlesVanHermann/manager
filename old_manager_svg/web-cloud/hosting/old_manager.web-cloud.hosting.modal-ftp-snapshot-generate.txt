# Cahier des Charges : web-cloud/hosting - Modal Générer snapshot FTP

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 2 étapes
- **URL** : N/A (composant modal)
- **Objectif** : Générer un snapshot FTP d'une instance

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Indicateur 2 étapes
- **OdsSelect** : Choix instance
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Navigation wizard

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Configuration | Active (bleu) |
| 2 | Confirmation | Inactive (gris) |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Instruction | "Sélectionnez l'instance de snapshot à générer." |

## 6. FORMULAIRE (ÉTAPE 1)
| Champ | Type | Requis | Options |
|-------|------|--------|---------|
| Instance | Select dropdown | Oui | snapshot.week.1, snapshot.week.2, etc. |

## 7. AVERTISSEMENT
| Type | Message |
|------|---------|
| Warning | "⚠ Attention : La génération d'un snapshot peut prendre plusieurs minutes. L'ancien snapshot de cette instance sera remplacé." |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant | Primary | Passer à confirmation |

## 9. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/request - Générer snapshot (type: snapshot)
