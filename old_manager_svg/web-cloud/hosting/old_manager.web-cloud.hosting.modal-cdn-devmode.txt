# Cahier des Charges : web-cloud/hosting - Modal CDN mode développeur

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'action avec avertissement
- **Contexte** : Configuration CDN
- **Objectif** : Activer le mode développeur (bypass du cache CDN)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsText** : Labels, valeurs
- **OdsBadge** : Statut
- **OdsSelect** : Durée d'activation
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL (WARNING)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le mode développeur" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Ligne 1 | "Le mode développeur désactive le cache CDN." |
| Ligne 2 | "Cela peut impacter les performances de votre site." |

### Informations
| Champ | Valeur |
|-------|--------|
| Domaine | www.example.com |
| Statut actuel | Badge "Désactivé" (vert #DCFCE7/#166534) |

### Formulaire
| Champ | Type | Valeur |
|-------|------|--------|
| Durée d'activation | OdsSelect | 3 heures |

| Propriété | Valeur |
|-----------|--------|
| Note | "Le mode développeur sera désactivé automatiquement après cette durée." |

### Options de durée
- 1 heure
- 3 heures (défaut)
- 6 heures
- 12 heures
- 24 heures

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Activer | Warning (orange #F59E0B) | Active le mode dev |

## 7. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/cdn/devmode - Activer/désactiver mode dev
- GET /hosting/web/{serviceName}/cdn - État actuel CDN
