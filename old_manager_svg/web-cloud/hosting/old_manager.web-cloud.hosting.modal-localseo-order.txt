# Cahier des Charges : web-cloud/hosting - Modal Commander Local SEO

## 1. VUE D'ENSEMBLE
- **Type** : Modal de commande avec sélection de plan
- **Contexte** : Services additionnels
- **Objectif** : Commander le service Local SEO pour améliorer la visibilité locale

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description, labels
- **OdsCard** : Plans (Basic, Premium)
- **OdsBadge** : Label "Populaire"
- **OdsInput** : Nom entreprise, adresse
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander Local SEO" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |
