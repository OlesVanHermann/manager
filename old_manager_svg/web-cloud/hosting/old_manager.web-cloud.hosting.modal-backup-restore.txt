# Cahier des Charges : web-cloud/hosting - Modal Backup Restore

## 1. VUE D'ENSEMBLE
- **Type** : Modal restauration sauvegarde
- **Contexte** : Web Cloud Hosting
- **Objectif** : Restaurer l'hébergement depuis une sauvegarde

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 440px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsMessage** : Warning + Info
- **OdsRadio** : Sélection sauvegarde (style card)
- **OdsCheckbox** : Options restauration
- **OdsButton** : Annuler, Restaurer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Restaurer une sauvegarde" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. WARNING
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠️ |
| Titre | "Attention" |
| Message | "La restauration écrasera les fichiers existants." |

## 6. SÉLECTION SAUVEGARDE
| Option | Description | Taille |
|--------|-------------|--------|
| Hier (date) | Sauvegarde automatique | X.X Go |
| Il y a 3 jours (date) | Sauvegarde automatique | X.X Go |
| Il y a 1 semaine (date) | Sauvegarde automatique | X.X Go |

## 7. STYLE RADIO CARD
| État | Bordure | Radio |
|------|---------|-------|
| Sélectionné | #0050D7 2px | Rempli bleu |
| Non sélectionné | #E5E7EB | Cercle vide |

## 8. OPTIONS RESTAURATION
| Checkbox | État par défaut |
|----------|-----------------|
| Restaurer les fichiers | ✓ Coché |
| Restaurer les bases de données | Non coché |

## 9. INFO
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | "ℹ️ La restauration peut prendre plusieurs minutes." |

## 10. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Restaurer | Danger #DC2626 | Droite |

## 11. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/dump - Liste sauvegardes
- POST /hosting/web/{serviceName}/restoreSnapshot - Restaurer
