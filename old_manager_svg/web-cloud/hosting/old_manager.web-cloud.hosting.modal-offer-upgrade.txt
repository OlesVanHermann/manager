# Cahier des Charges : web-cloud/hosting - Page Changer d'offre

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard 4 étapes
- **URL** : /hosting/{serviceName}/upgrade
- **Objectif** : Changer d'offre d'hébergement vers une offre supérieure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1000px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsBackLink** : Retour navigation
- **OdsText** : Titre et description
- **OdsCard** : Container principal
- **OdsStepper** : Navigation 4 étapes
- **OdsCard** : Cartes offres (4 offres)
- **OdsMessage** : Info prorata (bleu)
- **OdsButton** : Navigation wizard

## 4. STEPPER
| Étape | Nom | État |
|-------|-----|------|
| 1 | Offre | Active (bleu) |
| 2 | Durée | Inactive |
| 3 | Contrats | Inactive |
| 4 | Récapitulatif | Inactive |

## 5. OFFRES DISPONIBLES
| Offre | Prix | Stockage | Sites | BDD |
|-------|------|----------|-------|-----|
| PERSO (actuel) | 4,99 € HT/mois | 100 Go | 10 | 1 |
| PRO | 9,99 € HT/mois | 250 Go | Illimité | 5 |
| PERFORMANCE | 19,99 € HT/mois | 500 Go | Ressources dédiées | Illimitées |
| CLOUD WEB | 29,99 € HT/mois | Variable | Node.js, Python | - |

## 6. INFO FACTURATION
| Type | Message |
|------|---------|
| Info | "ℹ Le changement d'offre sera effectif immédiatement. Le montant sera calculé au prorata de votre période de facturation." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Annuler changement |
| Suivant | Primary | Étape suivante |

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName} - Obtenir offre actuelle
- GET /hosting/web/availableOffer - Liste offres disponibles
- POST /hosting/web/{serviceName}/changeOffer - Changer d'offre
