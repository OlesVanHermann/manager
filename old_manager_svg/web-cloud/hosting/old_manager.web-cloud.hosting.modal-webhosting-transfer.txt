# Cahier des Charges : web-cloud/hosting - Modal Transférer hébergement

## 1. VUE D'ENSEMBLE
- **Type** : Modal de transfert avec formulaire
- **Contexte** : Gestion de l'hébergement
- **Objectif** : Transférer la gestion de l'hébergement vers un autre compte client

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information et avertissement
- **OdsText** : Labels et valeurs
- **OdsInput** : Identifiant destinataire, code validation
- **OdsButton** : Annuler, Transférer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Transférer l'hébergement" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Transférez la gestion de cet hébergement vers un autre" |
| Ligne 2 | "compte client OVHcloud." |

### Informations actuelles
| Champ | Valeur exemple |
|-------|----------------|
| Hébergement | hosting123456.ovh.net |
| Offre | Pro |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Identifiant client destinataire | OdsInput | Oui | NIC-handle OVHcloud (ex: ab12345-ovh) |
| Code de validation | OdsInput | Oui | Code reçu par email |

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEF3C7 |
| Icône | ⚠ |
| Ligne 1 | "Cette action est irréversible. L'hébergement sera transféré" |
| Ligne 2 | "avec tous ses domaines et bases de données associés." |
| Couleur texte | #92400E |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Transférer | Primary (bleu #0050D7) | Lance le transfert |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/serviceInfos - Informations du service
- POST /hosting/web/{serviceName}/changeContact - Initier le transfert
- PUT /me/task/contactChange/{id}/accept - Accepter le transfert (destinataire)
