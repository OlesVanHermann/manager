# Cahier des Charges : web-cloud/private-database - IPs autorisées

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des IPs autorisées (whitelist)
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les adresses IP autorisées à se connecter

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb de navigation
- Titre "clouddb-abc123"
- Onglets (IPs autorisées actif)
- Section titre "IPs autorisées"
- Bouton "Ajouter une IP"
- Tableau des IPs
- Menu actions ouvert

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsTabs** : Onglets navigation
- **OdsButton** : Ajouter une IP
- **OdsTable** : Liste des IPs
- **OdsBadge** : Statut "Actif"
- **OdsActionMenu** : Menu ⋮ ouvert

## 4. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | - |
| Utilisateurs | - |
| Bases de données | - |
| **IPs autorisées** | Actif (bleu) |
| Configuration | - |
| Tâches | - |

## 5. BOUTON D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| + Ajouter une IP | Primaire bleu | Ouvre modal ajout |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Adresse IP | Format CIDR ou IP | 192.168.1.100 |
| Nom | Description libre | Serveur production |
| Service SFTP | Oui/Non | Oui |
| Statut | Badge état | Actif (vert) |
| Actions | Menu ⋮ | Modifier, Supprimer |

## 7. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| Actif | Vert (#DEF7EC) | IP autorisée |

## 8. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Éditer nom/SFTP |
| Supprimer | Rouge - Supprimer IP |

## 9. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/whitelist : Lister IPs
- POST /hosting/privateDatabase/{serviceName}/whitelist : Ajouter IP
- PUT /hosting/privateDatabase/{serviceName}/whitelist/{ip} : Modifier
- DELETE /hosting/privateDatabase/{serviceName}/whitelist/{ip} : Supprimer
