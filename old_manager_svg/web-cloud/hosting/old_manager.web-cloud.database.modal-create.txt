# Cahier des Charges : web-cloud/database - Modal Create

## 1. VUE D'ENSEMBLE
- **Type** : Modal création base de données
- **Contexte** : Web Cloud Databases
- **Objectif** : Créer une nouvelle base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 460px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsCard** : Sélection type DB
- **OdsInput** : Nom, utilisateur, mot de passe
- **OdsRadio** : Version MySQL
- **OdsMessage** : Note informative
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Nouvelle base de données" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. SÉLECTION TYPE BASE
| Type | Couleur badge |
|------|---------------|
| MySQL | #F59E0B (sélectionné) |
| PostgreSQL | #336791 |
| Redis | #DC382D |

## 6. FORMULAIRE
| Champ | Type | Placeholder |
|-------|------|-------------|
| Nom de la base * | Input | "ma_base_de_donnees" |
| Utilisateur * | Input | "db_user" |
| Mot de passe * | Input password | "••••••••" |

## 7. VERSION MySQL
| Version | État |
|---------|------|
| 5.7 | Non sélectionné |
| 8.0 | Sélectionné |

## 8. NOTE INFORMATIVE
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Icône | ℹ️ |
| Texte | "La base sera créée sur le serveur de votre hébergement." |

## 9. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Créer | Primary #0050D7 | Droite |

## 10. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database - Créer base hébergement
- POST /hosting/privateDatabase/{serviceName}/database - Créer base CloudDB
