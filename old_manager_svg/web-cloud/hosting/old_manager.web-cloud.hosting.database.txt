# Cahier des Charges : web-cloud/hosting - Database Listing

## 1. VUE D'ENSEMBLE

**Type:** Page listing avec tableau
**Fonction:** Lister et gérer les bases de données de l'hébergement
**Contexte:** Hébergement Web

## 2. DIMENSIONS

- Page: 1440x900px
- Background: #F9FAFB
- Table: 880x400px

## 3. NAVIGATION

### Breadcrumb
- Hébergements / mywebsite.com

### Titre page
- "Bases de données" (24px, font-weight: 600)

## 4. ACTIONS HEADER

| Bouton | Style | Action |
|--------|-------|--------|
| Créer une base de données | Primary (#0050D7) | Ouvrir modal création |
| Commander une base de données | Secondary outline | Rediriger vers commande |

## 5. DATAGRID

### Colonnes
| Colonne | Contenu |
|---------|---------|
| Nom de la base | mywebsitec_db1 |
| Nom d'utilisateur | mywebsitec |
| Adresse du serveur | mywebsitec.mysql.db |
| Version | MySQL 8.0 |
| Taille | Progress bar + texte |
| État | Badge Actif |
| Actions | Menu ⋮ |

### Progress bar Taille
- Background: #E5E7EB
- Barre normale: #10B981 (vert)
- Barre warning (>80%): #F59E0B (orange)
- Format: "156 Mo / 400 Mo"

### Badge État
| État | Fond | Texte |
|------|------|-------|
| Actif | #DEF7EC | #03543F |

### Menu Actions (⋮)
1. Changer le mot de passe
2. Créer une sauvegarde
3. Restaurer une sauvegarde
4. Importer un fichier
5. Accéder à phpMyAdmin
6. Supprimer la base (texte rouge #DC2626)

## 6. FOOTER INFO

- "2 bases de données sur 5 disponibles"

## 7. APIS & ENDPOINTS

```
GET /hosting/web/{serviceName}/database
POST /hosting/web/{serviceName}/database
DELETE /hosting/web/{serviceName}/database/{name}
POST /hosting/web/{serviceName}/database/{name}/changePassword
POST /hosting/web/{serviceName}/database/{name}/dump
POST /hosting/web/{serviceName}/database/{name}/dump/{id}/restore
POST /hosting/web/{serviceName}/database/{name}/import
```
