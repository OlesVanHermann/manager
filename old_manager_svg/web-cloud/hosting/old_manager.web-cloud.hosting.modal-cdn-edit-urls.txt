# Cahier des Charges : web-cloud/hosting - Modal Modifier URLs CDN

## 1. VUE D'ENSEMBLE
- **Type** : Modal de gestion avec liste éditable
- **Contexte** : Configuration CDN prewarm
- **Objectif** : Gérer les URLs à précharger dans le cache CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Titres, labels, compteur
- **OdsSelect** : Protocole
- **OdsInput** : Ressource (chemin)
- **OdsButton** : Ajouter, Supprimer, Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier les URLs à précharger" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Ajoutez des URLs à précharger dans le cache CDN." |

### Formulaire d'ajout
| Champ | Type | Largeur | Description |
|-------|------|---------|-------------|
| Protocole | OdsSelect | 100px | https://, http:// |
| Domaine | OdsInput (readonly) | 160px | Domaine fixe |
| Ressource | OdsInput | 150px | Chemin de la ressource |

| Propriété | Valeur |
|-----------|--------|
| Prévisualisation | URL complète en gris |
| Bouton ajouter | "+ Ajouter l'URL" (bleu) |

### Liste des URLs
| Propriété | Valeur |
|-----------|--------|
| Titre | "URLs à précharger" |
| Compteur | "{count} / 100 URLs maximum" |
| Style liste | Bordure grise, fond blanc |

| Fonctionnalité | Description |
|----------------|-------------|
| Sélection | Click sur une URL (highlight bleu) |
| Suppression | Bouton "Supprimer" pour URL sélectionnée |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Enregistre les URLs |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cdn/prewarm - Liste des URLs préchargées
- POST /hosting/web/{serviceName}/cdn/prewarm - Ajouter une URL
- DELETE /hosting/web/{serviceName}/cdn/prewarm/{url} - Supprimer une URL
