# Cahier des Charges : web-cloud/hosting - Modal Rapport erreurs emails

## 1. VUE D'ENSEMBLE
- **Type** : Modal configuration rapport erreurs
- **URL** : N/A (composant modal)
- **Objectif** : Configurer email pour rapports d'erreurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Adresse email
- **OdsMessage** : Info
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Adresse e-mail | Input email | Oui |

## 5. INFO
| Message | Description |
|---------|-------------|
| Info | "Un rapport sera envoyé à cette adresse en cas d'échec d'envoi d'e-mail." |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Enregistrer | Primary | Sauvegarder config |

## 7. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/automatedEmail - Configurer rapport
