# Cahier des Charges : web-cloud/hosting - Boost

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion du Boost temporaire
- **Objectif** : Activer boost performance temporaire
- **Contexte** : Pic de trafic anticipé

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud > Hébergements > hosting123 > Boost
- Titre section
- Bandeau info (offres Performance uniquement)
- Bouton activer
- Historique des boosts

## 3. COMPOSANTS UI
- **OdsMessage** : Info bleu (disponible Performance seulement)
- **OdsButton** : "Changer d'offre" (primaire), "Activer le boost" (secondaire)
- **OdsTable** : Historique boosts
- **OdsBadge** : Statut (Désactivé)

## 4. BANDEAU INFO
- "ℹ️ Le boost n'est disponible que pour les offres Performance."
- "Migrez vers une offre Performance pour bénéficier de cette fonctionnalité."
- Bouton : "Changer d'offre" (bleu)

## 5. TABLEAU HISTORIQUE BOOSTS
| Colonne | Description |
|---------|-------------|
| Compte | Nom hébergement |
| Offre | Offre actuelle |
| Offre Boost | Performance 1/2 ou "Désactivé" |
| Date | Date/heure action |

## 6. EXEMPLES DONNÉES
| Compte | Offre | Offre Boost | Date |
|--------|-------|-------------|------|
| hosting123 | Pro 2024 | Désactivé | 15 déc. 2025 14:30 |
| hosting123 | Pro 2024 | Performance 2 | 10 déc. 2025 09:15 |
| hosting123 | Pro 2024 | Performance 1 | 01 déc. 2025 16:45 |
| hosting123 | Pro 2024 | Désactivé | 15 nov. 2025 11:00 |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/boost : Statut boost
- POST /hosting/web/{serviceName}/boost : Activer boost
- DELETE /hosting/web/{serviceName}/boost : Désactiver boost
- GET /hosting/web/{serviceName}/boost/history : Historique
