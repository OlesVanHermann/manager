# Cahier des Charges : web-cloud/hosting - Modal Réinstaller module

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Gestion des modules CMS
- **Objectif** : Réinstaller un module en version vierge (supprime les données)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement critique
- **OdsText** : Informations module
- **OdsCheckbox** : Confirmation sauvegarde
- **OdsButton** : Annuler, Réinstaller

## 4. HEADER MODAL (DANGER)
| Propriété | Valeur |
|-----------|--------|
| Titre | "Réinstaller le module" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Fond | #FEE2E2 |
| Icône | ⚠ |
| Ligne 1 | "Attention : Cette action supprimera tous les fichiers" |
| Ligne 2 | "du module actuel et réinstallera une version vierge." |

### Informations du module
| Champ | Valeur |
|-------|--------|
| Module | WordPress |
| URL | https://www.example.com/blog |
| Version actuelle | 6.4.2 |

### Confirmation obligatoire
| Propriété | Valeur |
|-----------|--------|
| Type | OdsCheckbox |
| Label | "J'ai effectué une sauvegarde de mes données" |
| Requis | Oui |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Réinstaller | Danger (rouge #DC2626) | Lance la réinstallation |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/module/{id}/reinstall - Réinstaller le module
- GET /hosting/web/{serviceName}/module/{id} - Infos du module
