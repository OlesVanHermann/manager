# Cahier des Charges : web-cloud/hosting - Modal Supprimer association Git

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression de la configuration Git
- **Type** : Modal de confirmation warning
- **Objectif** : Dissocier le dépôt Git de l'hébergement

## 2. STRUCTURE MODAL
- Header jaune avec icône warning (⚠)
- Message de confirmation
- Checkbox option suppression fichiers
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsIcon** : ⚠ dans le header
- **OdsCheckbox** : Option vider le répertoire
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Suivant (warning orange)

## 4. HEADER
- Fond jaune (#FEF3C7)
- Icône ⚠
- Titre : "Supprimer l'association de Git"

## 5. MESSAGE
- "Vous êtes sur le point de supprimer l'association de Git."
- "**La suppression s'appliquera sur l'ensemble des domaines.**"

## 6. CHECKBOX OPTION
- □ "Souhaitez-vous vider le contenu du répertoire **/www** ?"
- Non coché par défaut

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron :
- "⚠ Cette action est irréversible."
- "Les fichiers supprimés ne pourront pas être récupérés."

## 8. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Suivant (warning orange, à droite)

## 9. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/runtime/attachedDomain/{domain}/git : Supprimer association
