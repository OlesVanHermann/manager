# Cahier des Charges : web-cloud/hosting - Modal Commander SSL DV

## 1. VUE D'ENSEMBLE
- **Type** : Modal de commande avec sélection
- **Contexte** : Gestion des certificats SSL
- **Objectif** : Commander un certificat SSL DV (Domain Validation)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Explication SSL DV
- **OdsRadio** : Type de certificat
- **OdsInput** : Domaines
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander un certificat SSL DV" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Le certificat SSL DV (Domain Validation) sécurise votre site" |
| Ligne 2 | "avec une validation automatique du domaine." |

### Type de certificat
| Propriété | Valeur |
|-----------|--------|
| Label section | "Type de certificat" |

### Options de certificat
| Option | Description | Prix |
|--------|-------------|------|
| Let's Encrypt (gratuit) | Renouvellement automatique tous les 90 jours | Gratuit (vert #10B981) |
| Sectigo DV | Certificat commercial avec garantie | 49,99€/an |

### Style sélection
| Propriété | Valeur |
|-----------|--------|
| Sélectionné | Bordure bleue (#0050D7), radio rempli |
| Non sélectionné | Bordure grise, radio vide |

### Domaines à sécuriser
| Champ | Type | Description |
|-------|------|-------------|
| Domaines à sécuriser | OdsInput | Liste séparée par virgules |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Commander | Primary (bleu #0050D7) | Lance la commande |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/ssl - Générer certificat Let's Encrypt
- POST /order/cart/{cartId}/ssl - Commander certificat Sectigo
