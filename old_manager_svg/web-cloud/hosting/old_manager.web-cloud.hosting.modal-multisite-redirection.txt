# Cahier des Charges : web-cloud/hosting - Modal Redirection multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Configuration multisite
- **Objectif** : Configurer une redirection HTTP pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels
- **OdsRadio** : Type de redirection
- **OdsInput** : URL de destination
- **OdsCheckbox** : Conserver le chemin
- **OdsButton** : Annuler, Configurer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configurer une redirection" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Domaine source
| Champ | Valeur |
|-------|--------|
| Domaine source | old.example.com |

### Type de redirection
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de redirection" |
| Type | OdsRadio (groupe) |

### Options de redirection
| Option | Code | Description | Sélection |
|--------|------|-------------|-----------|
| Redirection permanente | 301 | Recommandé pour le SEO | Sélectionné (bordure bleue) |
| Redirection temporaire | 302 | Pour des redirections ponctuelles | Non sélectionné |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| URL de destination | OdsInput | Oui | URL complète (https://) |

### Option conserver le chemin
| Propriété | Valeur |
|-----------|--------|
| Type | OdsCheckbox |
| Label | "Conserver le chemin (ex: /page → /page)" |
| État | Coché par défaut |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Configurer | Primary (bleu #0050D7) | Active la redirection |

## 7. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Configurer redirection
