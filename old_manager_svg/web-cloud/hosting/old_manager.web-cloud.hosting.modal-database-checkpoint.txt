# Cahier des Charges : web-cloud/hosting - Modal Point de restauration BDD

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Gestion des bases de données
- **Objectif** : Créer un point de restauration manuel pour une base de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsText** : Labels, valeurs
- **OdsInput** : Nom du checkpoint
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un point de restauration" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Créer | Primary (bleu #0050D7) | Crée le point de restauration |

## 7. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/dump - Créer un dump
- GET /hosting/web/{serviceName}/database/{name}/dump - Liste des dumps existants
- GET /hosting/web/{serviceName}/database/{name} - Infos base de données
