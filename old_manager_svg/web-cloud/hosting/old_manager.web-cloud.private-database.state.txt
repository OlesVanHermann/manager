# Cahier des Charges : web-cloud/private-database - État du service

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion de l'état du service SQL
- **Type** : Page actions avec historique
- **Objectif** : Démarrer, arrêter ou redémarrer le serveur

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Titre "État du service"
- Carte état actuel
- 3 cartes actions (Arrêter, Redémarrer, Démarrer)
- Tableau historique des actions

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsCard** : État actuel
- **OdsBadge** : "Démarré" (vert)
- **OdsCard** : 3 cartes actions
- **OdsButton** : Arrêter (rouge), Redémarrer (orange), Démarrer (gris)
- **OdsTable** : Historique

## 4. CARTE ÉTAT ACTUEL
| Champ | Valeur exemple |
|-------|----------------|
| État actuel | Badge "Démarré" (vert) |
| Uptime | 15 jours, 4 heures, 23 minutes |
| Dernière action | Redémarrage le 12/12/2024 à 03:15 |

## 5. CARTE ARRÊTER LE SERVICE
| Élément | Valeur |
|---------|--------|
| Titre | Arrêter le service |
| Description | Arrête le serveur MySQL. Toutes les connexions seront interrompues. |
| Bouton | "Arrêter" (rouge #EF4444) |

## 6. CARTE REDÉMARRER LE SERVICE
| Élément | Valeur |
|---------|--------|
| Titre | Redémarrer le service |
| Description | Effectue un arrêt suivi d'un démarrage. Connexions interrompues quelques secondes. |
| Bouton | "Redémarrer" (orange #F59E0B) |

## 7. CARTE DÉMARRER LE SERVICE
| Élément | Valeur |
|---------|--------|
| Titre | Démarrer le service |
| Description | Démarre le serveur MySQL s'il est arrêté. |
| Bouton | "Démarrer" (gris désactivé si déjà démarré) |
| Note | "(déjà démarré)" |

## 8. HISTORIQUE DES ACTIONS
| Colonne | Description |
|---------|-------------|
| Date | Date/heure de l'action |
| Action | Type (Redémarrage, Démarrage, Arrêt) |
| Durée | Temps de l'opération |
| Statut | Badge Succès (vert) |

## 9. EXEMPLES HISTORIQUE
| Date | Action | Durée | Statut |
|------|--------|-------|--------|
| 12/12/2024 03:15 | Redémarrage | 12 secondes | Succès |
| 01/12/2024 14:30 | Redémarrage | 8 secondes | Succès |
| 15/11/2024 09:00 | Démarrage | 25 secondes | Succès |

## 10. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName} : État actuel
- POST /hosting/privateDatabase/{serviceName}/start : Démarrer
- POST /hosting/privateDatabase/{serviceName}/stop : Arrêter
- POST /hosting/privateDatabase/{serviceName}/restart : Redémarrer
- GET /hosting/privateDatabase/{serviceName}/tasks : Historique actions
