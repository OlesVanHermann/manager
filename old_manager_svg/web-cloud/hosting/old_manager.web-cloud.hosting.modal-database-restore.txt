# Cahier des Charges : web-cloud/hosting - Modal Restaurer sauvegarde

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation restauration d'une sauvegarde
- **Type** : Modal de confirmation
- **Objectif** : Restaurer les données à partir d'un dump

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Question de confirmation
- Bandeau warning
- Note sur la durée
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay sombre
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Restaurer

## 4. QUESTION
"Êtes-vous sûr de vouloir restaurer la sauvegarde du **27/12/2024 03:00** pour la base de données **mydb_hosting123** ?"

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7), texte marron (#92400E) :
- "⚠ Attention : Cette action va écraser toutes les données actuelles de votre base de données par les données de la sauvegarde."

## 6. NOTE
"Cette opération peut prendre plusieurs minutes."

## 7. BOUTONS FOOTER
- Annuler (secondaire, à gauche)
- Restaurer (primaire, à droite)

## 8. APIS & ENDPOINTS
- POST /hosting/web/{serviceName}/database/{name}/dump/{id}/restore : Restaurer dump
