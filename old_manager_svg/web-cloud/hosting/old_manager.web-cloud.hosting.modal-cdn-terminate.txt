# Cahier des Charges : web-cloud/hosting - Modal Résilier CDN

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation résiliation CDN
- **Type** : Modal de confirmation danger
- **Objectif** : Confirmer la résiliation du service CDN

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Zone contenu avec icône warning
- Bandeau danger
- Détails de la résiliation
- Texte de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsIcon** : ⚠️ dans cercle rouge
- **OdsMessage** : Bandeau warning rouge
- **OdsButton** : Annuler, Confirmer la résiliation (danger rouge)

## 4. ICÔNE WARNING
- Grande icône ⚠️
- Cercle de fond rouge clair (#FEE2E2)
- Centrée en haut

## 5. BANDEAU DANGER
- Fond : Rouge clair (#FEE2E2)
- Texte : "Attention : cette action est irréversible"
- "La résiliation prendra effet à la fin de la période de facturation en cours."

## 6. DÉTAILS RÉSILIATION
| Champ | Valeur |
|-------|--------|
| Service | CDN Advanced |
| Hébergement | mon-hebergement.cluster051 |
| Fin de service | 31 janvier 2026 (gras) |

## 7. TEXTE CONFIRMATION
"En cliquant sur 'Confirmer la résiliation', vous acceptez de mettre fin à votre abonnement CDN."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Confirmer la résiliation (danger rouge)

## 9. APIS & ENDPOINTS
- DELETE /hosting/web/{serviceName}/cdn : Résilier CDN
- GET /hosting/web/{serviceName}/cdn : Info service actuel
