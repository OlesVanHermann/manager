# Cahier des Charges : web-cloud/hosting - Modal Commander CDN

## 1. VUE D'ENSEMBLE
- **Modal** : Souscription à une offre CDN
- **Type** : Wizard 3 étapes
- **Objectif** : Commander un service CDN pour l'hébergement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Stepper horizontal 3 étapes
- Grille de 3 cartes offres
- Récapitulatif
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsStepper** : 3 étapes
- **OdsCard** : Cartes offres (sélection bordure bleue épaisse)
- **OdsBadge** : "POPULAIRE" sur offre recommandée
- **OdsButton** : Sélectionner, Sélectionné ✓
- **OdsMessage** : Zone récapitulatif

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Choisir l'offre | Actif |
| 2 | Durée | À venir |
| 3 | Confirmation | À venir |
