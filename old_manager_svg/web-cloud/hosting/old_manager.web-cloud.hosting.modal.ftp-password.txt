# Cahier des Charges : web-cloud/hosting - Modal Modifier mot de passe FTP

## 1. VUE D'ENSEMBLE
- **Modal** : Changement de mot de passe FTP
- **Type** : Formulaire de sécurité
- **Objectif** : Modifier le mot de passe d'un utilisateur FTP

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Info utilisateur
- Champs mot de passe avec indicateur force
- Zone exigences mot de passe
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsInput** : Champs password avec toggle afficher
- **OdsProgressBar** : Indicateur force mot de passe
- **OdsCard** : Zone exigences (fond gris)
- **OdsButton** : Annuler, Valider

## 4. INFO UTILISATEUR FTP
| Champ | Valeur |
|-------|--------|
| Utilisateur FTP | monsiteex-admin |

## 5. CHAMPS MOT DE PASSE

### Nouveau mot de passe
- Type : Input password
- Bouton "Afficher" (toggle)
- Indicateur force :
  - Barre de progression (rouge/orange/vert)
  - Label : "Fort" (vert #10B981)

### Confirmer le mot de passe
- Type : Input password
- Validation : doit correspondre au premier champ

## 6. EXIGENCES MOT DE PASSE
Zone fond gris (#F9FAFB) :
- "Le mot de passe doit contenir :"
- "✓ Au moins 8 caractères" (vert)
- "✓ Une majuscule, une minuscule, un chiffre" (vert)

## 7. INDICATEUR FORCE
| Niveau | Couleur | Largeur |
|--------|---------|---------|
| Faible | Rouge (#EF4444) | 25% |
| Moyen | Orange (#F59E0B) | 50% |
| Fort | Vert (#10B981) | 75% |

## 8. BOUTONS FOOTER
- Annuler (secondaire gris)
- Valider (primaire bleu)

## 9. APIS & ENDPOINTS
- PUT /hosting/web/{serviceName}/user/{login}/password : Modifier mot de passe
  - Body : { password }
