# Cahier des Charges : web-cloud/hosting - Statistiques

## 1. VUE D'ENSEMBLE
- **Écran** : Statistiques de l'hébergement
- **Objectif** : Visualiser métriques de performance
- **Contexte** : Monitoring et analyse trafic

## 2. LAYOUT & STRUCTURE
- Titre : "Statistiques"
- Filtres (Type, Période, Grouper)
- Graphique ligne principal
- 3 tuiles résumé (Moyenne, Maximum, Minimum)

## 3. COMPOSANTS UI
- **OdsSelect** : Type, Période, Grouper
- **OdsCard** : Graphique + tuiles résumé
- **OdsChart** : Graphique ligne avec aire

## 4. FILTRES
| Filtre | Valeur exemple |
|--------|----------------|
| Type | Requêtes HTTP |
| Période | Semaine |
| Grouper | Toutes les requêtes |

## 5. GRAPHIQUE
- Titre : "Requêtes HTTP"
- Sous-titre : "hits / min"
- Type : Ligne avec aire (fill opacity 0.1)
- Couleur : Bleu OVH (#0050D7)
- Axe X : Dates (21/12 - 28/12)
- Axe Y : Valeurs (0 - 500)

## 6. TUILES RÉSUMÉ
| Métrique | Valeur | Unité |
|----------|--------|-------|
| Moyenne | 287 | hits/min |
| Maximum | 482 | hits/min |
| Minimum | 98 | hits/min |

## 7. STYLE GRAPHIQUE
- Points de données : Cercles bleus (r=4)
- Ligne : stroke-width=2
- Grille : Lignes horizontales grises claires
- Labels Y : 500, 375, 250, 125, 0

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/statistics : Données stats
- Paramètres : type, period, aggregateBy
