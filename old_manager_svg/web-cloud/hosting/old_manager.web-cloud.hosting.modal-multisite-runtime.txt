# Cahier des Charges : web-cloud/hosting - Modal Runtime multisite

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification
- **Contexte** : Configuration multisite
- **Objectif** : Modifier le runtime (version PHP/Node) d'un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, runtime actuel
- **OdsSelect** : Liste des runtimes disponibles
- **OdsMessage** : Information
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le runtime" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Informations domaine
| Champ | Valeur |
|-------|--------|
| Domaine | api.example.com |
| Runtime actuel | PHP 8.1 Production |

### Formulaire
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Nouveau runtime | OdsSelect | Oui | Runtime à appliquer |

### Format du select
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | Nom du runtime (PHP 8.2 Production) |
| Ligne 2 | Chemin racine associé (/www/public) |

### Options de runtime disponibles
- PHP 8.2 Production
- PHP 8.1 Production
- PHP 8.0 Production
- PHP 7.4 Production
- Node.js 18
- Node.js 16

### Information
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Ligne 1 | "Le changement de runtime sera appliqué" |
| Ligne 2 | "immédiatement sur ce domaine." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Modifier | Primary (bleu #0050D7) | Applique le runtime |

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/runtime - Liste des runtimes
- GET /hosting/web/{serviceName}/attachedDomain/{domain} - Config actuelle
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} - Modifier runtime
