# Cahier des Charges : web-cloud/zimbra - Comptes email

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des comptes email Zimbra.

### Objectif fonctionnel
- Voir les comptes configures
- Creer de nouveaux comptes
- Commander des comptes supplementaires

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/account
POST /zimbra/platform/{platformId}/account
DELETE /zimbra/platform/{platformId}/account/{accountId}
PUT /zimbra/platform/{platformId}/account/{accountId}
```

### Modele
```typescript
interface ZimbraAccount {
  id: string;
  email: string;
  organization: string;
  offer: 'starter' | 'pro';
  quota: number;
  state: 'ok' | 'suspended';
  renewal: 'automatic' | 'manual';
}
```
