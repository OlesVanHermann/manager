# Cahier des Charges : web-cloud/zimbra - Modal suppression redirection

## 1. RESUME EXECUTIF

### Perimetre
Modal de confirmation de suppression d'une redirection.

### Objectif fonctionnel
- Afficher les details de la redirection
- Confirmer la suppression
- Supprimer la redirection

---

## 2. LAYOUT

### Structure modale avec header danger + contenu + footer

---

## 3. COMPOSANTS UI

### Modal header
| Element | Description |
|---------|-------------|
| Icone | Warning rouge |
| Titre | Supprimer la redirection |
| Bouton fermer | X |

### Contenu
| Element | Description |
|---------|-------------|
| Question | Souhaitez-vous supprimer la redirection ? |
| Detail De | De: {from} |
| Detail Vers | Vers: {to} |

### Footer
| Element | Description |
|---------|-------------|
| Bouton annuler | Secondary |
| Bouton supprimer | Danger (rouge) |

---

## 4. APIS & DONNEES

### Endpoints
```
DELETE /zimbra/platform/{platformId}/redirection/{redirectionId}
```

### Modele
```typescript
interface DeleteRedirectionResponse {
  taskId: string;
  status: 'pending' | 'done' | 'error';
}
```
