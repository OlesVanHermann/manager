# Cahier des Charges : web-cloud/zimbra - Modal ajout organisation

## 1. RESUME EXECUTIF

### Perimetre
Modal de creation d'une nouvelle organisation Zimbra.

### Objectif fonctionnel
- Definir le nom technique
- Definir le label affiche
- Creer l'organisation

---

## 2. LAYOUT

### Structure modale avec header + description + formulaire + footer

---

## 3. COMPOSANTS UI

### Modal header
| Element | Description |
|---------|-------------|
| Titre | Ajouter une organisation |
| Bouton fermer | X |

### Description
| Element | Description |
|---------|-------------|
| Ligne 1 | L'organisation represente un groupement logique de domaines et de boites mails |
| Ligne 2 | Elle permet de definir des politiques de gestion groupees |
| Ligne 3 | Elle definit l'isolation de vos boites mails (limite de l'annuaire d'entreprise) |

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Nom * | Input | Nom technique de l'organisation |
| Label * | Input + tooltip | Label affiche dans l'interface |

### Footer
| Element | Description |
|---------|-------------|
| Bouton annuler | Secondary |
| Bouton confirmer | Primary |

---

## 4. APIS & DONNEES

### Endpoints
```
POST /zimbra/platform/{platformId}/organization
```

### Modele
```typescript
interface CreateOrganization {
  name: string;
  label: string;
}
```
