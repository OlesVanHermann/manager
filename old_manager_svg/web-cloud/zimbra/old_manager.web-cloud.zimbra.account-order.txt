# Cahier des Charges : web-cloud/zimbra - Commander comptes

## 1. RESUME EXECUTIF

### Perimetre
Formulaire de commande de comptes Zimbra supplementaires.

### Objectif fonctionnel
- Selectionner une offre (Starter/Pro)
- Definir le nombre de comptes
- Choisir la duree d'engagement
- Voir le recapitulatif et valider

|---------|-------------|
| Breadcrumb | Zimbra > Comptes email > Commander |

### En-tete
| Element | Description |
|---------|-------------|
| Titre | Commander vos comptes Zimbra |
| Description | Veuillez choisir le nombre de comptes puis le type de paiement a commander |

### Cartes offres
| Offre | Details |
|-------|---------|
| Zimbra Starter | 15 Go, Partage calendriers, ActiveSync - 1,99 EUR HT/mois |
| Zimbra Pro | 50 Go, Partage illimite, Stockage fichiers, Visio - 5,99 EUR HT/mois |

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Nombre de comptes | Input number | Quantite a commander |
| Duree engagement | Radio | 1 mois / 12 mois |
| CGV | Checkbox | Acceptation conditions |

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/offer
POST /order/zimbra/{platformId}/account
GET /order/cart/{cartId}
POST /order/cart/{cartId}/checkout
```

### Modele
```typescript
interface ZimbraOrder {
  offer: 'starter' | 'pro';
  quantity: number;
  duration: 1 | 12; // months
}

interface ZimbraOffer {
  name: string;
  storage: number;
  pricePerMonth: number;
  features: string[];
}
```
