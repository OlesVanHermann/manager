# Cahier des Charges : web-cloud/email-mxplan - Modal Ajouter compte

## 1. VUE D'ENSEMBLE
- **Page** : Créer un compte email
- **Type** : Modal formulaire
- **Objectif** : Créer un compte MXplan

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ adresse email + domaine
- Champ nom d'affichage
- Champ mot de passe
- Select quota
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Email, nom affichage, mot de passe
- **OdsSelect** : Quota
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Adresse email | Input + suffix | Oui | contact + @domain.com |
| Nom d'affichage | Input | Non | Jean Dupont |
| Mot de passe | Password + toggle | Oui | ••••••••••• (avec icône oeil) |
| Quota | Select | Non | 5 Go (défaut) |

## 5. OPTIONS QUOTA
| Valeur | Description |
|--------|-------------|
| 5 Go | Standard |
| 10 Go | Extended |
| 25 Go | Large |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /email/mxplan/{service}/account - Créer compte
