# Cahier des Charges : web-cloud/email-mxplan - Dashboard

## 1. RESUME EXECUTIF

### Perimetre
Dashboard du service MX Plan avec vue d'ensemble.

### Objectif fonctionnel
- Voir les informations generales du service
- Consulter la configuration email
- Acceder au webmail
- Naviguer vers les onglets

---

## 2. LAYOUT

### Structure page avec header + onglets + 3 cartes

## 3. COMPOSANTS UI

### Header service
| Element | Description |
|---------|-------------|
| Titre | MX Plan - {domaine} |
| Sous-titre | Date expiration |

### Onglets navigation
| Onglet | Description |
|--------|-------------|
| Informations generales | Vue d'ensemble (actif) |

## 4. APIS & DONNEES

### Endpoints
```
GET /email/domain/{domain}
GET /email/domain/{domain}/account
GET /email/domain/{domain}/redirection
GET /email/domain/{domain}/mailingList
```

### Modele
```typescript
interface MXPlanService {
  domain: string;
  offer: 'MX Plan 1' | 'MX Plan 5' | 'MX Plan 100';
  maxAccounts: number;
  currentAccounts: number;
  quota: {
    used: number;
    total: number;
  };
  expiration: string;
  state: 'ok' | 'suspended';
}

interface MXPlanConfiguration {
  imapServer: string;
  imapPort: number;
  pop3Server: string;
  pop3Port: number;
  smtpServer: string;
  smtpPort: number;
}
```

## 6. REGLES METIER

- MX Plan = offre email incluse avec domaine
- Quota global partage entre comptes
- Webmail = Roundcube
- IMAP/POP3/SMTP avec SSL obligatoire
- Anti-spam et antivirus actives par defaut
