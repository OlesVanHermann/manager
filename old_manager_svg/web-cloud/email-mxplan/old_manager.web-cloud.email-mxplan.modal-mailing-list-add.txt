# Cahier des Charges : web-cloud/email-mxplan - Modal Créer liste de diffusion

## 1. VUE D'ENSEMBLE
- **Page** : Créer une liste de diffusion
- **Type** : Modal formulaire
- **Objectif** : Créer une mailing list

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ nom de la liste + domaine
- Select propriétaire
- Select langue
- Checkboxes modération
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom liste
- **OdsSelect** : Propriétaire, langue
- **OdsCheckbox** : Options modération
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom de la liste | Input + suffix | Oui | equipe + @domain.com |
| Propriétaire | Select | Oui | admin@domain.com |
| Langue | Select | Non | Français |

## 5. OPTIONS MODÉRATION
| Option | État par défaut | Description |
|--------|-----------------|-------------|
| Modérer les messages | Non coché | Validation manuelle |
| Seuls les membres peuvent poster | ✓ Coché | Restriction posting |

## 6. OPTIONS LANGUE
| Langue |

| Français |
| English |

| Español |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /email/mxplan/{service}/mailingList - Créer liste
