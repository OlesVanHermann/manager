# Cahier des Charges : web-cloud/email-pro - Modal Ajouter une redirection

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une redirection email
- **Type** : Modal formulaire
- **Objectif** : Rediriger les emails vers une autre adresse

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Adresse source (De)
- Adresse destination (Vers)
- Option copie locale
- Bandeau info
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsInput** : Partie locale source, destination
- **OdsSelect** : Domaine source
- **OdsCheckbox** : Copie locale
- **OdsMessage** : Info bleu
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter une redirection"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| De | Input + @ + Select domaine | Oui (*) |
| Vers | Input (email complet) | Oui (*) |

## 6. OPTION COPIE LOCALE
| Checkbox | État | Description |
|----------|------|-------------|
| ☑ | Coché | Conserver une copie locale |

## 7. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Si activé, l'e-mail sera également conservé dans la boîte d'origine."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Ajouter | Primaire bleu | Crée la redirection |

## 9. APIS & ENDPOINTS
- POST /email/pro/{service}/redirection : Créer redirection
  - from: string
  - to: string
  - localCopy: boolean
