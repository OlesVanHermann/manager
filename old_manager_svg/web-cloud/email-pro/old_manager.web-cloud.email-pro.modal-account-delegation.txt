# Cahier des Charges : web-cloud/email-pro - Modal Délégations

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des délégations d'un compte Email Pro
- **Type** : Modal formulaire avec onglets
- **Objectif** : Définir les droits d'envoi et d'accès pour d'autres comptes

## 2. STRUCTURE MODAL
- Header avec email du compte et bouton fermer (×)
- Onglets types de délégation
- Bandeau info
- Tableau des comptes avec checkboxes
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x440)
- **OdsTabs** : Onglets délégation
- **OdsMessage** : Info bleu
- **OdsTable** : Liste des comptes
- **OdsCheckbox** : Autorisation par compte
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Délégations - {email}"
Exemple : "Délégations - admin@example.com"

## 5. ONGLETS
| Onglet | État | Description |
|--------|------|-------------|
| Envoyer en tant que | Actif (bleu) | Envoyer au nom du compte |
| Accès complet | Inactif | Ouvrir et gérer la boîte |

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Sélectionnez les comptes autorisés à envoyer des e-mails au nom de ce compte."

## 7. COLONNES DU TABLEAU
| Colonne | Description |
|---------|-------------|
| Compte | Email du compte |
| Autoriser | Checkbox |

## 8. EXEMPLES DE DONNÉES
| Compte | Autoriser |
|--------|-----------|
| contact@example.com | ☑ (bleu) |
| support@example.com | ☐ |
| info@example.com | ☐ |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Enregistrer | Primaire bleu | Sauvegarde les délégations |

## 10. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email}/sendAs : Lister "envoyer en tant que"
- GET /email/pro/{service}/account/{email}/fullAccess : Lister accès complets
- POST /email/pro/{service}/account/{email}/sendAs/{delegate} : Ajouter
- DELETE /email/pro/{service}/account/{email}/sendAs/{delegate} : Supprimer
