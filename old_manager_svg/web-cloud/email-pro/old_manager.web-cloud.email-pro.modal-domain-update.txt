# Cahier des Charges : web-cloud/email-pro - Modal Modifier un domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Modification du type de configuration d'un domaine
- **Type** : Modal formulaire
- **Objectif** : Changer entre autoritatif et non-autoritatif

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine (lecture seule)
- Type de configuration (radio)
- Bandeau info
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x290)
- **OdsText** : Domaine (lecture seule)
- **OdsRadio** : Type de configuration
- **OdsMessage** : Info bleu
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le domaine"

## 5. DOMAINE (LECTURE SEULE)
| Label | Valeur |
|-------|--------|
| Domaine | example.com |

## 6. TYPE DE CONFIGURATION
| Option | État | Description |
|--------|------|-------------|
| ○ Autoritatif | Sélectionné | OVH gère tous les emails |
| ○ Non autoritatif | - | Coexistence avec autre serveur |

## 7. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Le changement peut prendre quelques minutes."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Modifier | Primaire bleu | Applique la modification |

## 9. APIS & ENDPOINTS
- PUT /email/pro/{service}/domain/{domain} : Modifier domaine
  - type: "authoritative" | "nonAuthoritative"
