# Cahier des Charges : web-cloud/email-pro - Modal Ajouter un compte

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un nouveau compte Email Pro
- **Type** : Modal formulaire
- **Objectif** : Créer un compte email avec mot de passe

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire en 4 champs
- Bandeau info mot de passe
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x400)
- **OdsInput** : Champs texte
- **OdsSelect** : Sélection domaine
- **OdsMessage** : Info bleu
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Ajouter un compte"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Adresse e-mail | Input + Select | Oui | utilisateur @ domaine |
| Nom affiché | Input | Non | Prénom Nom |
| Mot de passe | Password | Oui | Min 8 caractères |
| Confirmer | Password | Oui | Confirmation |

## 6. STRUCTURE ADRESSE EMAIL
| Partie | Composant | Exemple |
|--------|-----------|---------|
| Utilisateur | OdsInput | utilisateur |
| @ | Texte | @ |
| Domaine | OdsSelect | example.com ▼ |

## 7. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Le mot de passe doit contenir au moins 8 caractères."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le compte |

## 9. VALIDATION
| Champ | Règle |
|-------|-------|
| Adresse e-mail | Format email valide |
| Mot de passe | Minimum 8 caractères |
| Confirmer | Identique au mot de passe |

## 10. APIS & ENDPOINTS
- POST /email/pro/{service}/account : Créer compte
  - email: string
  - displayName: string (optionnel)
  - password: string
  - domain: string
