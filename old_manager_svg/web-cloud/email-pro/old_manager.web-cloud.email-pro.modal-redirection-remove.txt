# Cahier des Charges : web-cloud/email-pro - Modal Supprimer une redirection

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'une redirection
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer une redirection email

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec flux
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x180)
- **OdsText** : Message confirmation, flux
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer la redirection"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Supprimer cette redirection ? |
| Flux | {source} → {destination} |

Exemple : "contact@example.com → admin@example.com"

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime la redirection |

## 7. APIS & ENDPOINTS
- DELETE /email/pro/{service}/redirection/{id} : Supprimer redirection
