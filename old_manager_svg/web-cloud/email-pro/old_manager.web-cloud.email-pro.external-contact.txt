# Cahier des Charges : web-cloud/email-pro - Contacts externes

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des contacts externes Email Pro
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les contacts externes visibles dans l'annuaire

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Email Pro (menu)
- Titre "Contacts externes"
- Dropdown Actions
- Barre de recherche
- Tableau des contacts
- Pagination

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Email Pro
- **OdsSelect** : Dropdown Actions
- **OdsInput** : Recherche contact
- **OdsTable** : Liste des contacts
- **OdsBadge** : Statut
- **OdsIcon** : Icône annuaire (✓ / ✗)
- **OdsActionMenu** : Menu actions (⋮)

## 4. TITRE
"Contacts externes"

## 5. DROPDOWN ACTIONS
| Action | Description |
|--------|-------------|
| Ajouter un contact | Ouvre modal création |
| Exporter en CSV | Télécharge le fichier CSV |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Adresse | Email du contact | fournisseur@external.com |
| Nom affiché | Nom complet | Jean Fournisseur |
| Créé le | Date de création | 15 déc. 2024 |
| Annuaire | Visible dans l'annuaire | ✓ vert / ✗ rouge |
| Statut | État du contact | OK, En création |
| Actions | Menu ⋮ | Modifier, Supprimer |

## 7. ICÔNE ANNUAIRE
| État | Icône | Couleur |
|------|-------|---------|
| Visible | ✓ | Vert (#10B981) |
| Masqué | ✗ | Rouge (#EF4444) |

## 8. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| OK | Vert (#10B981) | Contact actif |

## 12. APIS & ENDPOINTS
- GET /email/pro/{service}/externalContact : Lister contacts
- POST /email/pro/{service}/externalContact : Créer contact
- PUT /email/pro/{service}/externalContact/{id} : Modifier
- DELETE /email/pro/{service}/externalContact/{id} : Supprimer
- GET /email/pro/{service}/externalContact/export : Exporter CSV
