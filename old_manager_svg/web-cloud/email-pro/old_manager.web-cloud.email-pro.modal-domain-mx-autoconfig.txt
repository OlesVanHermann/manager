# Cahier des Charges : web-cloud/email-pro - Modal Configuration MX

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique des enregistrements MX
- **Type** : Modal action avec erreur
- **Objectif** : Configurer les MX pour la réception d'emails

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Bandeau erreur rouge
- Domaine concerné
- Configuration requise
- Boutons Fermer / Configurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x340)
- **OdsMessage** : Alerte erreur rouge
- **OdsText** : Domaine, configuration
- **OdsButton** : Fermer, Configurer

## 4. TITRE
"Configuration MX"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. BANDEAU ERREUR
Fond rouge (#FEF2F2), texte rouge (#DC2626) :
"⚠ Les enregistrements MX ne sont pas correctement configurés."

## 6. INFORMATIONS
| Élément | Valeur |
|---------|--------|
| Domaine concerné | example.com |

## 7. CONFIGURATION REQUISE
| Champ | Valeur |
|-------|--------|
| Type | MX |
| Priorité | 1 |
| Cible | mx1.mail.ovh.net |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |
| Configurer | Primaire bleu | Configure MX automatiquement |

## 9. APIS & ENDPOINTS
- GET /email/pro/{service}/domain/{domain}/diagnostic : Récupérer diagnostic
- POST /domain/{domain}/zone/record : Créer enregistrement MX
