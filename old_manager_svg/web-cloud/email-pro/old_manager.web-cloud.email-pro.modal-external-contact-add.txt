# Cahier des Charges : web-cloud/email-pro - Modal Ajouter un contact externe

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un contact externe
- **Type** : Modal formulaire
- **Objectif** : Ajouter un contact visible dans l'annuaire

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Champs du formulaire
- Checkbox annuaire
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x360)
- **OdsInput** : Email, nom affiché, prénom, nom
- **OdsCheckbox** : Afficher dans l'annuaire
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un contact externe"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Adresse e-mail externe | Input | Oui (*) | contact@external.com |
| Nom affiché | Input | Oui (*) | Prénom Nom |
| Prénom | Input | Non | - |
| Nom | Input | Non | - |

## 6. OPTION ANNUAIRE
| Checkbox | Description |
|----------|-------------|
| ☐ | Afficher dans l'annuaire |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Ajouter | Primaire bleu | Crée le contact |

## 8. VALIDATION
| Champ | Règle |
|-------|-------|
| Email | Format email valide |
| Nom affiché | Non vide |

## 9. APIS & ENDPOINTS
- POST /email/pro/{service}/externalContact : Créer contact
  - externalEmailAddress: string
  - displayName: string
  - firstName: string (optionnel)
  - lastName: string (optionnel)
  - hiddenFromGAL: boolean
