# Cahier des Charges : web-cloud/email-pro - Modal Supprimer une liste de diffusion

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'une liste
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer une liste de diffusion

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec email
- Info nombre de membres
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x180)
- **OdsText** : Message confirmation, email, info membres
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer la liste"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Supprimer cette liste de diffusion ? |
| Email | {email} (en gras) |
| Info | X membres seront retirés. |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime la liste |

## 7. APIS & ENDPOINTS
- DELETE /email/pro/{service}/mailingList/{email} : Supprimer liste
