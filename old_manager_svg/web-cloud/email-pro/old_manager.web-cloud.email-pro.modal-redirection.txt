# Cahier des Charges : web-cloud/email-pro - Modal Configurer la redirection

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration d'une redirection pour un compte
- **Type** : Modal formulaire
- **Objectif** : Configurer la redirection d'un compte spécifique

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte source (lecture seule)
- Adresse de destination
- Options (copie locale, congés)
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsInput** : Compte (désactivé), destination
- **OdsCheckbox** : Options
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Configurer la redirection"

## 5. COMPTE SOURCE
Champ désactivé avec l'email du compte

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Adresse de destination | Input | Oui (*) | destination@autre-domaine.com |

## 7. OPTIONS
| Checkbox | État | Description |
|----------|------|-------------|
| ☑ | Coché | Conserver une copie locale |
| ☐ | Non coché | Activer uniquement pendant les congés |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Enregistrer | Primaire bleu | Sauvegarde la redirection |

## 9. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email}/redirection : Récupérer config
- PUT /email/pro/{service}/account/{email}/redirection : Configurer
  - to: string
  - localCopy: boolean
  - vacationOnly: boolean
