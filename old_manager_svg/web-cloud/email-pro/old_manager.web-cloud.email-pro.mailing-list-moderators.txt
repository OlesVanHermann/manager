# Cahier des Charges : web-cloud/email-pro - Modérateurs de liste

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des modérateurs d'une liste de diffusion
- **Type** : Modal liste avec tableau
- **Objectif** : Ajouter/supprimer des modérateurs pouvant approuver les messages

## 2. STRUCTURE MODAL
- Header avec email de la liste et bouton fermer (×)
- Bandeau info
- Bouton ajouter modérateur
- Tableau des modérateurs
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x390)
- **OdsMessage** : Info bleu
- **OdsButton** : Ajouter modérateur
- **OdsTable** : Liste des modérateurs
- **OdsButton** : Supprimer (×) par ligne
- **OdsButton** : Fermer (secondaire)

## 4. TITRE
"Modérateurs - {email}"
Exemple : "Modérateurs - marketing@example.com"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les modérateurs peuvent approuver ou rejeter les messages envoyés à cette liste."

## 6. BOUTON D'ACTION
| Bouton | Position | Action |
|--------|----------|--------|
| Ajouter modérateur | Haut droite | Ouvre modal sélection |

## 7. COLONNES DU TABLEAU
| Colonne | Description |
|---------|-------------|
| E-mail du modérateur | Adresse email |
| - | Bouton supprimer (× rouge) |

## 8. EXEMPLES DE DONNÉES
| E-mail du modérateur |
|---------------------|
| admin@example.com |
| manager@example.com |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 10. APIS & ENDPOINTS
- GET /email/pro/{service}/mailingList/{email}/moderator : Lister modérateurs
- POST /email/pro/{service}/mailingList/{email}/moderator : Ajouter
- DELETE /email/pro/{service}/mailingList/{email}/moderator/{moderator} : Supprimer
