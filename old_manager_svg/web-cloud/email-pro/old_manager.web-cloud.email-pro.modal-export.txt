# Cahier des Charges : web-cloud/email-pro - Modal Export

## 1. VUE D'ENSEMBLE
- **Modal** : Export des données d'un compte email
- **Type** : Modal formulaire
- **Objectif** : Générer un fichier PST avec les données du compte

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau info
- Compte à exporter (lecture seule)
- Format d'export
- Avertissement durée
- Boutons Annuler / Lancer export

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x320)
- **OdsMessage** : Info bleu, warning orange
- **OdsInput** : Compte (désactivé)
- **OdsSelect** : Format d'export
- **OdsButton** : Annuler, Lancer export

## 4. TITRE
"Exporter les données"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"L'export génère un fichier PST contenant tous les emails, contacts et calendriers du compte."

## 6. CHAMPS
| Champ | Type | Description |
|-------|------|-------------|
| Compte à exporter | Input désactivé | Email du compte |
| Format d'export | Select | PST (Outlook) |

## 7. FORMATS DISPONIBLES
| Format | Description |
|--------|-------------|
| PST (Outlook) | Format Microsoft Outlook |

## 8. AVERTISSEMENT
Fond orange (#FEF3C7) :
"⚠ L'export peut prendre plusieurs heures selon la taille."

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Lancer export | Primaire bleu | Démarre l'export |

## 10. APIS & ENDPOINTS
- POST /email/pro/{service}/account/{email}/export : Lancer export
  - format: "pst"
