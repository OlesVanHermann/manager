# Cahier des Charges : web-cloud/email-pro - Modal Configuration Outlook

## 1. VUE D'ENSEMBLE
- **Modal** : Affichage des paramètres de configuration
- **Type** : Modal information
- **Objectif** : Afficher les paramètres IMAP/SMTP pour Outlook

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau info
- Compte (lecture seule)
- Paramètres IMAP
- Paramètres SMTP
- Boutons Fermer / Copier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsMessage** : Info bleu
- **OdsInput** : Champs lecture seule
- **OdsButton** : Fermer, Copier

## 4. TITRE
"Configuration Outlook"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Utilisez ces paramètres pour configurer votre client de messagerie Outlook ou tout autre client IMAP/SMTP."

## 6. COMPTE
Champ désactivé avec l'email du compte

## 7. SERVEUR ENTRANT (IMAP)
| Paramètre | Valeur |
|-----------|--------|
| Serveur | imap.mail.ovh.net |
| Port | 993 (SSL) |

## 8. SERVEUR SORTANT (SMTP)
| Paramètre | Valeur |
|-----------|--------|
| Serveur | smtp.mail.ovh.net |
| Port | 587 (TLS) |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |
| Copier | Primaire bleu | Copie les paramètres |

## 10. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email} : Récupérer config
  → Retourne les paramètres serveur
