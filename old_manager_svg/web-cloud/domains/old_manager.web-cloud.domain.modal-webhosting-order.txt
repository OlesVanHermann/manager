# Cahier des Charges : web-cloud/domain - Modal Commander hébergement

## 1. VUE D'ENSEMBLE
- **Modal** : Assistant de commande d'hébergement web
- **Type** : Wizard multi-étapes
- **Objectif** : Commander un hébergement web pour le domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Stepper 4 étapes
- Description
- Cartes de sélection d'offre
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Grande modal (900x650)
- **OdsStepper** : 4 étapes
- **OdsCard** : Cartes d'offre (4 offres)
- **OdsButton** : Annuler, Suivant

## 4. TITRE
"Commander un hébergement web"

## 5. STEPPER (4 ÉTAPES)

|-------|-------|------|
