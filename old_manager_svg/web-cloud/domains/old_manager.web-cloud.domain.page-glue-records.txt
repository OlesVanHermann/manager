- **Type** : Page gestion Glue Records
- **Contexte** : Web Cloud Domain DNS
- **Objectif** : Configurer ses propres serveurs DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour
- **OdsButton** : Ajouter Glue
- **OdsMessage** : Info explication
- **OdsCard** : Liste + guide
- **OdsDatagrid** : Glue Records
- **OdsBadge** : Statut

## 5. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Titre | "ℹ️ Qu'est-ce qu'un Glue Record ?" |
| Texte | "Un Glue Record permet d'associer une adresse IP à un nom de serveur DNS..." |
| Exemple | "ns1.example.com → 203.0.113.10" |

## 6. DATAGRID GLUE RECORDS
| Colonne | Description |
|---------|-------------|
| Nom du serveur | Hostname monospace (ns1.domain.com) |
| Adresse IPv4 | IP monospace |
| Adresse IPv6 | IPv6 monospace |
| Statut | Badge Propagé/En attente |
| Actions | Modifier, Supprimer |

## 7. GUIDE UTILISATION
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |

## 8. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Propagé | #DCFCE7 | #16A34A |
| En attente | #FEF3C7 | #F59E0B |

## 9. APIS & ENDPOINTS
- GET /domain/{domainName}/glueRecord - Liste Glue Records
- POST /domain/{domainName}/glueRecord - Créer
- PUT /domain/{domainName}/glueRecord/{host} - Modifier
- DELETE /domain/{domainName}/glueRecord/{host} - Supprimer
