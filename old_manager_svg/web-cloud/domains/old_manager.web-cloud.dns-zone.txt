# Cahier des Charges : web-cloud/dns-zone - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Informations générales de la zone DNS
- **Type** : Dashboard avec 3 cartes d'information
- **Objectif** : Vue synthétique de la zone DNS et de sa configuration

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec menu "Noms de domaine" > "Zone DNS" actif
- Breadcrumb : Zone DNS - {domaine}
- Onglets : Informations générales (actif) | Zone DNS | Historique | DNSSEC
- 3 cartes d'information côte à côte

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation entre sections
- **OdsCard** : 3 cartes (Informations générales, Serveurs DNS, DNSSEC)
- **OdsBadge** : Statut (Actif, Activé)
- **OdsLink** : "Modifier les serveurs DNS →", "Gérer DNSSEC →"

## 4. TITRE
"Zone DNS - {domaine}"

## 5. DESCRIPTION
"Gérez les enregistrements DNS de votre domaine"

## 6. ONGLETS
| Onglet | Description |
|--------|-------------|
| Informations générales | Vue d'ensemble (actif) |
| Zone DNS | Liste des enregistrements |
| Historique | Modifications |
| DNSSEC | Sécurité |

## 10. APIS & ENDPOINTS
- GET /domain/zone/{zoneName} : Infos zone
- GET /domain/zone/{zoneName}/record : Liste enregistrements
- GET /domain/zone/{zoneName}/dnssec : Statut DNSSEC
- GET /domain/zone/{zoneName}/nameServer : Serveurs DNS
