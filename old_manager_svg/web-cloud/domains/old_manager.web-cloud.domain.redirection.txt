# Cahier des Charges : web-cloud/domain - Redirection

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des redirections du domaine
- **Type** : Page liste avec tableau
- **Objectif** : Créer, modifier, supprimer des redirections web

## 2. LAYOUT
- Header OVHcloud
- Breadcrumb : Noms de domaine / example.com
- Titre et description
- Boutons d'action
- Filtre de recherche
- Tableau des redirections

## 3. COMPOSANTS UI
- **OdsButton** : Ajouter une redirection, Exporter en CSV
- **OdsInput** : Filtre sous-domaine
- **OdsTable** : Tableau des redirections
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. TITRE
"Redirection"

## 5. DESCRIPTION
"Gérer ici les redirections de vos domaines et sous-domaines vers d'autres adresses web ou vers des serveurs."

## 6. BOUTONS D'ACTION
| Bouton | Type | Action |
|--------|------|--------|
| Ajouter une redirection | Primaire | Ouvre modal ajout redirection |
| Exporter en CSV | Secondaire | Télécharge les redirections en CSV |

## 7. FILTRE
- Placeholder : "Filtre sous-domaine…"
- Largeur : 300px

## 8. TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Domaine | Sous-domaine source | example.com |
| Type | Type de redirection | Redirection visible, vers un domain (CNAME), vers un serveur (A) |
| Cible | Destination | https://www.example.com |
| Visibilité | Comportement | temporaire (302), permanent (301), iframe |
| Actions | Menu ⋮ | Modifier, Supprimer |

## 10. ACTION MENU
| Option | Action |
|--------|--------|
| Modifier | Ouvre modal modification |
| Supprimer | Ouvre modal confirmation suppression (rouge) |

## 11. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/redirection : Lister les redirections
- POST /domain/zone/{zoneName}/redirection : Créer une redirection
- PUT /domain/zone/{zoneName}/redirection/{id} : Modifier une redirection
- DELETE /domain/zone/{zoneName}/redirection/{id} : Supprimer une redirection
