# Cahier des Charges : web-cloud/domain - Modal Ajouter DynHost

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DynHost
- **Type** : Formulaire de création
- **Objectif** : Configurer un DynHost pour IP dynamique

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau info explicatif
- Formulaire 4 champs
- Info URL mise à jour
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x600)
- **OdsMessage** : Bandeau info bleu
- **OdsInput** : Sous-domaine, IP, Identifiant, Mot de passe
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Ajouter un DynHost"

## 5. BANDEAU INFO
Fond bleu (#E8F0FE) :
- "ℹ️ Le DynHost permet de faire pointer un sous-domaine vers"
- "une adresse IP dynamique (connexion internet résidentielle)."
- "L'IP sera mise à jour automatiquement via un client DynDNS."

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "home" |
| Suffixe | ".example.com" |

## 7. CHAMP IP ACTUELLE
| Élément | Valeur |
|---------|--------|
| Label | "Adresse IP actuelle" |
| Placeholder | "Ex: 90.123.45.67" |

## 8. CHAMP IDENTIFIANT
| Élément | Valeur |
|---------|--------|
| Label | "Identifiant DynHost" |
| Valeur | "example.com-home" |
| Note | Généré automatiquement |

## 9. CHAMP MOT DE PASSE
| Élément | Valeur |
|---------|--------|
| Label | "Mot de passe DynHost" |
| Type | Password (masqué) |
| Action | Lien "Générer" |

## 10. INFO URL MISE À JOUR
Fond gris (#F9FAFB) :
"URL de mise à jour: https://www.ovh.com/nic/update?system=dyndns&hostname=..."

## 11. BOUTONS FOOTER
- Annuler (secondaire)
- Créer (primaire)

## 12. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/dynHost/record : Créer DynHost
  - subDomain: string
  - ip: string
- POST /domain/zone/{zoneName}/dynHost/login : Créer login DynHost
