## 11. APIS & ENDPOINTS
- GET /domain/zone - Liste zones
- POST /domain/zone - Créer zone
- GET /domain/zone/{zoneName} - Détail zone
- DELETE /domain/zone/{zoneName} - Supprimer zone
- POST /domain/zone/{zoneName}/dnssec - Activer DNSSEC
