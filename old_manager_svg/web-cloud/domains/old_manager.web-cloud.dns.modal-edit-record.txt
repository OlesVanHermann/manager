# Cahier des Charges : web-cloud/dns - Modal Edit Record

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification enregistrement DNS
- **Contexte** : Web Cloud DNS Management
- **Objectif** : Modifier ou supprimer un enregistrement DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 390px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsBadge** : Type (readonly)
- **OdsInput** : Sous-domaine et cible
- **OdsRadio** : TTL
- **OdsButton** : Annuler, Supprimer, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'enregistrement DNS" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. TYPE (READONLY)
| Propriété | Valeur |
|-----------|--------|
| Label | "Type" |
| Affichage | Badge dans zone grise |
| État | Non modifiable |

## 6. FORMULAIRE
| Champ | Type | Éditable |
|-------|------|----------|
| Sous-domaine | Input + suffixe zone | Oui |
| Cible (IPv4) | Input avec focus bleu | Oui |

## 7. SÉLECTION TTL
| TTL | Description |
|-----|-------------|
| 60 | (1m) |
| 300 | (5m) |
| 3600 | (1h) - sélectionné |

## 8. ACTIONS (3 boutons)
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Supprimer | Danger #DC2626 | Centre |
| Modifier | Primary #0050D7 | Droite |

## 9. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName}/record/{id} - Modifier
- DELETE /domain/zone/{zoneName}/record/{id} - Supprimer
- POST /domain/zone/{zoneName}/refresh - Rafraîchir zone
