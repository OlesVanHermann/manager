# Cahier des Charges : network/dns - Modal Import

## 1. VUE D'ENSEMBLE
- **Type** : Modal import zone DNS
- **Contexte** : Network DNS
- **Objectif** : Importer une zone DNS depuis un fichier ou transfert AXFR

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |
| Border radius | 8px |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Labels
- **Cards sélection** : Format d'import
- **OdsTextarea** : Contenu zone file
- **OdsCheckbox** : Option écrasement
- **OdsButton** : Annuler, Importer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Background | #F9FAFB |
| Titre | "Importer une zone DNS" |
| Bouton fermer | × en #6B7280 |

## 5. SÉLECTION FORMAT IMPORT
| Option | Largeur | Description |
|--------|---------|-------------|
| Zone file (BIND) | 140px | Fichier texte format BIND |
| Transfert AXFR | 140px | Depuis serveur DNS |

## 6. TEXTAREA ZONE FILE
| Propriété | Valeur |
|-----------|--------|
| Label | "Contenu du fichier zone" |
| Largeur | 450px |
| Hauteur | 150px |
| Police | Monospace 10px |

### Exemple contenu
```
$TTL 3600
@    IN  SOA  ns1.ovh.net. hostmaster.example.com. (
         2024122901 ; serial
         3600 ; refresh
@    IN  A    141.94.100.1
www  IN  CNAME @
```

## 7. CHECKBOX ÉCRASEMENT
| Propriété | Valeur |
|-----------|--------|
| Label | "Écraser les records existants" |
| État défaut | Non coché |

## 8. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline #E5E7EB | 100px |
| Importer | Primary #0050D7 | 100px |

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/import - Importer zone
- POST /domain/zone/{zoneName}/refresh - Refresh après import
