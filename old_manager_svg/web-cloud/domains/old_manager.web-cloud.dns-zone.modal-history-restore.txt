# Cahier des Charges : web-cloud/dns-zone - Modal Restaurer version historique

## 1. VUE D'ENSEMBLE
- **Modal** : Restauration depuis l'historique
- **Type** : Modal confirmation
- **Objectif** : Confirmer la restauration d'une version de zone

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Description avec nom zone et date
- Encadré info (date + nb enregistrements)
- Bandeau warning orange
- Boutons Annuler / Restaurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x330)
- **OdsCard** : Info version (fond gris)
- **OdsMessage** : Warning orange
- **OdsButton** : Annuler, Restaurer

## 4. TITRE (Header bleu)
"Restaurer une version de zone DNS"

## 5. DESCRIPTION
"Vous allez restaurer la zone DNS {domaine} à l'état du :"

## 6. ENCADRÉ INFO
Fond gris (#F9FAFB), bordure (#E5E7EB) :
- Ligne 1 (bold): "15 janvier 2024 à 14:30:00"
- Ligne 2 (muted): "Version créée il y a 3 jours - 12 enregistrements"

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure (#F59E0B) :
- "⚠ Attention : Cette action va remplacer tous les enregistrements actuels"
- "de votre zone DNS par ceux de la version sélectionnée."
- "Cette opération peut prendre quelques minutes à se propager."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Restaurer | Primaire bleu | Lance la restauration |

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/restore
  - date: string (ISO date)
