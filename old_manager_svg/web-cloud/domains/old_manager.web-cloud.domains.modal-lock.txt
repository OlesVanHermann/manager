# Cahier des Charges : web-cloud/domains - Modal Verrouillage

## 1. VUE D'ENSEMBLE
- **Page** : Verrouillage du domaine
- **Type** : Modal formulaire simple
- **Objectif** : Activer/désactiver le TransferLock

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau info explicatif
- Champ domaine (lecture seule)
- Checkbox TransferLock
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Bandeau info (bleu)
- **OdsInput** : Domaine (disabled)
- **OdsCheckbox** : Activer verrouillage
- **OdsButton** : Annuler, Appliquer

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | Le verrouillage protège votre domaine contre les transferts non autorisés |

## 5. CHAMPS
| Champ | Type | Description |
|-------|------|-------------|
| Domaine | Input disabled | Nom du domaine |
| TransferLock | Checkbox | Activer le verrouillage (TransferLock) |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Appliquer | Primary | Droite |

## 7. APIS & ENDPOINTS
- PUT /domain/{serviceName}/lock - Modifier le verrouillage
