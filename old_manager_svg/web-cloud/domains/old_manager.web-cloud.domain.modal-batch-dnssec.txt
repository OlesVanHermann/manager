# Cahier des Charges : web-cloud/domain - Modal Batch DNSSEC

## 1. VUE D'ENSEMBLE

**Type:** Modal action batch
**Fonction:** Activer/désactiver DNSSEC sur plusieurs domaines
**Contexte:** Action groupée depuis liste des domaines

## 2. DIMENSIONS

- Modal: 700x620px
- Position: centrée avec overlay

## 3. STRUCTURE MODAL

### Header
- Background: #FFFFFF
- Titre: "Gestion DNSSEC (Batch)"
- Bouton fermer: × (coin droit)

### Content
- Padding: 24px

### Footer
- Séparateur: ligne #E5E7EB

## 4. CONTENU

### Info banner
- Background: #DBEAFE
- Dimensions: 652x50px
- Titre: "DNSSEC - Sécurité DNS"
- Description: "DNSSEC protège contre les attaques de type DNS spoofing en signant cryptographiquement les enregistrements."

### Action sélecteur
- Label: "Action à appliquer :"
- Dropdown: 220x36px
- Options: "Activer DNSSEC" / "Désactiver DNSSEC"

### Liste domaines
- Titre: "Domaines sélectionnés (4)"
- Colonnes:
  - Checkbox
  - DOMAINE
  - STATUT DNSSEC

### Badges statut
- Actif: Background #DEF7EC, text #03543F
- Inactif: Background #F3F4F6, text #6B7280

### Exemples données

| Domaine | Statut |
|---------|--------|
| example.com | Inactif |
| example.fr | Actif |
| example.eu | Inactif |
| example.org | Inactif |

### Warning banner
- Background: #FEF3C7
- Dimensions: 652x60px
- Titre: "Important"
- Messages:
  - "L'activation de DNSSEC nécessite que vos serveurs DNS supportent DNSSEC."
  - "Les domaines utilisant les DNS OVH seront automatiquement configurés."

## 5. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 100x34px |
| Appliquer (4) | btn-primary | 150x34px |

## 6. COMPOSANTS UI

### Dropdown action
- Background: #F9FAFB
- Border: #E5E7EB
- Border-radius: 4px

### Badge statut DNSSEC
- Border-radius: 9px
- Padding: 4px 12px
- Font-size: 9px

## 7. APIS & ENDPOINTS

```
GET /domain/{serviceName}/dnssec
→ Statut DNSSEC actuel

POST /domain/{serviceName}/dnssec
→ Active DNSSEC

DELETE /domain/{serviceName}/dnssec
→ Désactive DNSSEC

# Pour chaque domaine sélectionné:
POST /domain/{domain}/dnssec   # si activation
DELETE /domain/{domain}/dnssec # si désactivation
```
