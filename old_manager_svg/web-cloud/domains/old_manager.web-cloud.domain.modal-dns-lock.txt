# Cahier des Charges : web-cloud/domain - Modal Réinitialisation DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Réinitialisation des serveurs DNS
- **Type** : Modal de confirmation danger
- **Objectif** : Réinitialiser les DNS vers les serveurs OVH d'origine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation
- Description de l'action
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x340)
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Réinitialiser

## 4. TITRE
"Réinitialisation des serveurs DNS"

## 5. QUESTION DE CONFIRMATION
"Voulez-vous vraiment réinitialiser la configuration des serveurs DNS ?"

## 6. DESCRIPTION
"Cette action réinitialisera la configuration actuelle et entraînera le basculement vers les serveurs OVH d'origine."

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
- "⚠️ Cette action est irréversible."
- "Tous vos serveurs DNS personnalisés seront supprimés."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Réinitialiser (primaire)

## 9. APIS & ENDPOINTS
- POST /domain/{serviceName}/nameServer/reset : Réinitialiser serveurs DNS
