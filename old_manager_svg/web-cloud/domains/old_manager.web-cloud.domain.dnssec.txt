## 6. TABLEAU DS RECORDS
| Colonne | Description |
|---------|-------------|
| Tag | Identifiant numérique |
| Drapeau | Type de clé (256/257) |
| Algorithme | Algorithme cryptographique |
| Clé publique | Valeur encodée (tronquée) |

### Drapeaux
| Code | Signification |
|------|---------------|
| 257 | Zone Signing Key (ZSK) |
| 256 | Key Signing Key (KSK) |

### Algorithmes
| Code | Nom |
|------|-----|
| 13 | ECDSAP256SHA256 |
| 8 | RSASHA256 |

### Exemple données
| Tag | Drapeau | Algorithme | Clé publique |
|-----|---------|------------|--------------|
| 12345 | 257 - Zone Signing Key | 13 - ECDSAP256SHA256 | AwEAAb2bj...KLM= |
| 67890 | 256 - Key Signing Key | 8 - RSASHA256 | AwEAAcNE...XYZ= |
| 11111 | 257 - Zone Signing Key | 13 - ECDSAP256SHA256 | AwEAAd4R...ABC= |
