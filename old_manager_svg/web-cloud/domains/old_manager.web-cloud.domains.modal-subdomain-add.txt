# Cahier des Charges : web-cloud/domains - Modal Subdomain Add

## 1. VUE D'ENSEMBLE
- **Type** : Modal création sous-domaine
- **URL** : N/A (composant modal)
- **Objectif** : Créer nouveau sous-domaine

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Description
- Formulaire
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 380px |

## 4. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Nom du sous-domaine | Input + suffixe | Oui |
| Hébergement | Radio buttons | Oui |
| Hébergement de destination | Select | Si OVHcloud |

## 5. NOM SOUS-DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Placeholder | "blog" |
| Suffixe | ".exemple.com" (readonly) |

## 6. OPTIONS HÉBERGEMENT
| Option | Label |
|--------|-------|
| OVH | "Utiliser mon hébergement OVHcloud" (défaut) |
| Externe | "Pointer vers une adresse externe" |

## 7. SÉLECTEUR HÉBERGEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Select dropdown |
| Options | Liste hébergements du compte |
| Exemple | "monsite.cluster001.hosting.ovh.net" |

## 8. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Créer | Primary | "Créer" |

## 9. APIS & ENDPOINTS
- GET /hosting/web - Liste hébergements
- POST /domain/{domain}/zone/record - Créer enregistrement
