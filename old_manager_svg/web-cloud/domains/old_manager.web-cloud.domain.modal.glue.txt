## 6. UTILISATION

Les enregistrements GLUE sont nécessaires quand:
- Vous utilisez des serveurs DNS sous votre propre domaine
- Exemple: ns1.exemple.com doit avoir un GLUE si exemple.com utilise ns1.exemple.com comme DNS

Cela évite les références circulaires dans la résolution DNS.

## 7. APIS & ENDPOINTS

```
POST /domain/{serviceName}/glueRecord
Body: {
  host: "ns1",
  ips: ["203.0.113.10", "2001:db8::1"]
}
→ Crée un enregistrement GLUE

GET /domain/{serviceName}/glueRecord
→ Liste des enregistrements GLUE

DELETE /domain/{serviceName}/glueRecord/{host}
→ Supprime un enregistrement GLUE

PUT /domain/{serviceName}/glueRecord/{host}
Body: {
  ips: ["203.0.113.11"]
}
→ Modifie un enregistrement GLUE
```
