# Cahier des Charges : web - Zone DNS

## 1. VUE D'ENSEMBLE
- **Page** : Gestion zone DNS
- **Type** : Onglet avec datagrid d'enregistrements
- **Objectif** : Gérer les enregistrements DNS du domaine

## 2. STRUCTURE PAGE
- Onglet "Zone DNS" actif
- Titre + description
- Filtres et recherche
- Bouton ajout
- Datagrid des enregistrements

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsSelect** : Filtrer par type
- **OdsInput** : Recherche
- **OdsButton** : Ajouter une entrée
- **OdsDatagrid** : Liste enregistrements
- **OdsBadge** : Type d'enregistrement

## 4. FILTRES
| Filtre | Type | Placeholder |
|--------|------|-------------|
| Type | Select | "Filtrer par type..." |
| Recherche | Input | "Rechercher..." |
| Bouton | Primary | "+ Ajouter une entrée" |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Type | Badge coloré | A, AAAA, MX, CNAME, TXT... |
| Sous-domaine | Text | @, www, blog, api... |
| Cible | Text | IP, domaine, valeur |
| TTL | Number | Temps en secondes |
| Actions | Links | Modifier | Supprimer |

## 6. BADGES TYPE
| Type | Couleur fond | Couleur texte |
|------|--------------|---------------|
| A | Bleu clair (#DBEAFE) | Bleu (#0050D7) |
| AAAA | Jaune (#FEF3C7) | Marron (#92400E) |
| MX | Vert (#D1FAE5) | Vert (#059669) |
| CNAME | Indigo (#E0E7FF) | Indigo (#4338CA) |
| TXT | Violet (#F3E8FF) | Violet (#7C3AED) |

## 7. EXEMPLES DONNÉES
| Type | Sous-domaine | Cible | TTL |
|------|--------------|-------|-----|
| A | @ | 123.45.67.89 | 3600 |
| A | www | 123.45.67.89 | 3600 |
| AAAA | @ | 2001:db8::1 | 3600 |
| MX | @ | 10 mx1.mail.ovh.net. | 3600 |
| CNAME | blog | monsite.fr. | 3600 |
| TXT | @ | "v=spf1 include:mx.ovh.com ~all" | 3600 |

## 8. ACTIONS PAR LIGNE
| Action | Style | Description |
|--------|-------|-------------|
| Modifier | Link bleu | Éditer l'enregistrement |
| Supprimer | Link bleu | Supprimer l'enregistrement |

## 9. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/record - Liste enregistrements
- GET /domain/zone/{zoneName}/record/{id} - Détail
- POST /domain/zone/{zoneName}/record - Créer
- PUT /domain/zone/{zoneName}/record/{id} - Modifier
- DELETE /domain/zone/{zoneName}/record/{id} - Supprimer
- POST /domain/zone/{zoneName}/refresh - Rafraîchir zone
