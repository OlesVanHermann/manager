# Cahier des Charges : web-cloud/domain - Tâches récentes

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des tâches sur le domaine
- **Type** : Page liste avec tableau
- **Objectif** : Visualiser l'historique des opérations

## 2. LAYOUT
- Header OVHcloud
- Breadcrumb : Web Cloud > Noms de domaine > example.com > Tâches récentes
- Titre
- Tableau des tâches

## 3. COMPOSANTS UI
- **OdsTable** : Tableau des tâches
- **OdsBadge** : Statuts (Terminée, En cours, Planifiée, A traiter, Échouée)
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. TITRE
"Tâches récentes"

## 5. TABLEAU
| Colonne | Description |
|---------|-------------|
| Tâche | Nom technique de l'opération |
| Date de création | Date au format FR |
| Détail technique | Description de l'opération |
| Statut | Badge coloré selon l'état |
| Actions | Menu ⋮ |

## 6. BADGES STATUT
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Terminée | Vert (#DEF7EC) | Vert foncé (#03543F) |
| En cours | Bleu (#E8F0FE) | Bleu (#0050D7) |
| Planifiée | Bleu (#E8F0FE) | Bleu (#0050D7) |
| A traiter | Jaune (#FEF3C7) | Orange (#92400E) |
| Échouée | Rouge (#FEE2E2) | Rouge (#991B1B) |

## 7. EXEMPLES DE TÂCHES

## 8. ACTION MENU
| Option | Action |
|--------|--------|
| Modifier | Voir détails / Relancer |

## 9. APIS & ENDPOINTS
- GET /domain/{serviceName}/task : Lister les tâches
- GET /domain/{serviceName}/task/{id} : Détails d'une tâche
