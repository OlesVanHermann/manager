# Cahier des Charges : web-cloud/domain - Page List

## 1. VUE D'ENSEMBLE
- **Type** : Page liste noms de domaine
- **Contexte** : Web Cloud Domains
- **Objectif** : Vue consolidée de tous les domaines

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et sous-titres
- **OdsButton** : CTA commander
- **OdsCard** : Cartes métriques
- **OdsInput** : Recherche
- **OdsSelect** : Filtre extension
- **OdsDatagrid** : Liste domaines
- **OdsBadge** : Statut domaine
- **OdsActionMenu** : Menu actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Noms de domaine" |
| Sous-titre | "Gérez vos domaines et DNS" |
| Bouton | "+ Commander un domaine" |

## 5. CARTES MÉTRIQUES (3 colonnes)
| Carte | Label | Valeur | Couleur |
|-------|-------|--------|---------|
| Total | "TOTAL DOMAINES" | Nombre | Vert pour actifs |
| Transfert | "EN TRANSFERT" | Nombre | Orange |
| Expiration | "EXPIRENT BIENTÔT" | Nombre | Rouge |

## 6. FILTRES
| Élément | Type |
|---------|------|
| Recherche | Input avec placeholder "Rechercher un domaine..." |
| Extension | Select (com, fr, eu, org...) |

## 7. DATAGRID
| Colonne | Description |
|---------|-------------|
| Domaine | Nom du domaine (lien bleu) |
| DNS | "OVHcloud" ou "Externe" |
| DNSSEC | Badge Actif/Inactif |
| Propriétaire | Nom du contact owner |
| Expiration | Date (rouge si proche) |
| Statut | Badge coloré |
| Actions | Menu ⋮ |

## 8. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Actif | #DCFCE7 | #16A34A |
| Expire bientôt | #FEF3C7 | #92400E |
| Transfert | #EFF6FF | #0050D7 |
| Inactif | #F9FAFB | #6B7280 |

## 9. APIS & ENDPOINTS
- GET /domain - Liste domaines
- GET /domain/{domainName} - Détail domaine
- GET /domain/{domainName}/dsRecord - DNSSEC
