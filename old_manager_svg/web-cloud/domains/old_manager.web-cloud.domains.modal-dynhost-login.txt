# Cahier des Charges : web-cloud/domains - Modal DynHost Login

## 1. VUE D'ENSEMBLE
- **Type** : Modal gestion identifiants DynHost
- **URL** : N/A (composant modal)
- **Objectif** : Gérer identifiants DynHost

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Info explicative
- Tableau identifiants existants
- Formulaire création nouvel identifiant
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 390px |

## 4. TABLEAU IDENTIFIANTS
| Colonne | Type |
|---------|------|
| Identifiant | Texte |
| Sous-domaine | Texte |
| Supprimer | Icône poubelle rouge |

## 5. SECTION CRÉATION
| Champ | Type |
|-------|------|
| Suffixe d'identifiant | Input + prévisualisation |
| Sous-domaine associé | Select dropdown |

## 6. PRÉVISUALISATION IDENTIFIANT
| Propriété | Valeur |
|-----------|--------|
| Format | "mondomaine.com-{suffixe}" |
| Exemple | "mondomaine.com-nas" |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Fermer | Secondary | "Fermer" |
| Créer | Primary | "Créer" |

## 8. APIS & ENDPOINTS
- GET /domain/{domain}/dynHost/login - Liste identifiants
- POST /domain/{domain}/dynHost/login - Créer identifiant
- DELETE /domain/{domain}/dynHost/login/{login} - Supprimer
