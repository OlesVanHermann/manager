# Cahier des Charges : web-cloud/domain - Modal WHOIS

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification WHOIS
- **Contexte** : Web Cloud Domain Contacts
- **Objectif** : Modifier les informations WHOIS du domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 540px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsTabs** : Types contact
- **OdsInput** : Champs formulaire
- **OdsSelect** : Pays
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier les informations WHOIS" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. INFO DOMAINE
| Champ | Valeur |
|-------|--------|
| Label | "DOMAINE" |
| Nom | Nom du domaine |

## 6. ONGLETS CONTACT
| Onglet | Style sélectionné |
|--------|-------------------|
| Propriétaire | Primary #0050D7 |
| Admin | Default |

## 7. FORMULAIRE
| Champ | Largeur | Obligatoire |
|-------|---------|-------------|
| Nom | 50% | Oui |
| Prénom | 50% | Oui |
| Organisation | 100% | Non |
| Adresse | 100% | Oui |
| Code postal | 30% | Oui |

| Pays | 35% (Select) | Oui |

| Téléphone | 50% | Oui |

## 8. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Enregistrer | Primary #0050D7 | Droite |

## 9. APIS & ENDPOINTS
- GET /domain/{domainName}/contacts - Contacts actuels
- PUT /domain/{domainName}/contacts - Modifier contacts
