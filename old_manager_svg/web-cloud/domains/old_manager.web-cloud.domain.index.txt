## 7. APIS & ENDPOINTS

```
GET /domain/{serviceName}
→ Informations générales (extension, dates, registrar)

GET /domain/{serviceName}/serviceInfos
→ Infos service (renouvellement, offre)

GET /domain/zone/{zoneName}
→ Informations zone DNS

GET /domain/{serviceName}/dnssec
→ Statut DNSSEC

GET /domain/{serviceName}/options/anycast
→ Statut Anycast DNS

GET /domain/{serviceName}/authInfo
→ Auth code (transfert sortant)

GET /domain/{serviceName}/transferLock
→ Statut verrouillage transfert

GET /domain/{serviceName}/owo
→ Statut protection WHOIS

GET /domain/{serviceName}/contacts
→ Liste des contacts
```
