# Cahier des Charges : web-cloud/domain - Modal Tag transfert UK

## 1. VUE D'ENSEMBLE
- **Modal** : Tag de transfert sortant pour domaines UK
- **Type** : Formulaire de saisie
- **Objectif** : Définir le tag IPS pour transfert domaine .co.uk

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Description avec nom de domaine
- Champ de saisie du tag
- Texte d'aide
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x320)
- **OdsInput** : Tag de transfert
- **OdsButton** : Annuler, Envoyer

## 4. TITRE
"Tag de transfert sortant"

## 5. DESCRIPTION
"Veuillez saisir le tag de transfert sortant pour le domaine **exemple.co.uk**"

## 6. CHAMP TAG
| Élément | Valeur |
|---------|--------|
| Label | "Tag de transfert sortant *" |
| Obligatoire | Oui |
| Input | "GANDI" |
| Placeholder | Nom du registrar cible |

## 7. AIDE CONTEXTUELLE
- "Le tag ne doit comporter que des lettres, chiffres et tirets (-)"
- "Maximum 50 caractères"

## 8. VALIDATION
- Caractères autorisés : A-Z, 0-9, -
- Longueur max : 50 caractères

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Envoyer (primaire)

## 10. APIS & ENDPOINTS
- POST /domain/{serviceName}/ukOutgoingTransfer : Initier transfert sortant UK
  - tag: string (IPS tag du registrar cible)

## 11. NOTES
- Spécifique aux domaines .uk, .co.uk, .org.uk, .me.uk
- Le tag IPS identifie le registrar vers lequel transférer
