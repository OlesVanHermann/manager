# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement A

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type A
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement A (IPv4) dans la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Bandeau info explicatif
- Champ sous-domaine
- Champ adresse IPv4
- Champ TTL
- Aperçu de l'enregistrement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x600)
- **OdsMessage** : Bandeau info bleu
- **OdsInput** : Sous-domaine, IPv4
- **OdsSelect** : TTL
- **OdsCard** : Aperçu (fond gris)
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un enregistrement A"

## 5. BANDEAU INFO
Fond bleu (#E8F0FE) :
- **Titre** : "ℹ️ Enregistrement A (Address)"
- "Associe un nom de domaine à une adresse IPv4."

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Aide | "Laissez vide pour le domaine principal (@)" |
| Input | "www" |
| Suffixe (addon gris) | ".example-domain.com" |

## 7. CHAMP ADRESSE IPV4
| Élément | Valeur |
|---------|--------|
| Label | "Adresse IPv4 cible *" |
| Aide | "L'adresse IP vers laquelle pointer" |
| Obligatoire | Oui |
| Input | "203.0.113.50" |

## 8. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL (Time To Live)" |
| Aide | "Durée de mise en cache de l'enregistrement" |
| Type | Select dropdown |
| Valeur | "3600 (1 heure)" |

## 9. APERÇU
Fond gris (#F9FAFB), police monospace :
`www.example-domain.com.  3600  IN  A  203.0.113.50`

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter (primaire)

## 11. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement A
  - fieldType: "A"
  - subDomain: string
  - target: IPv4
  - ttl: number
