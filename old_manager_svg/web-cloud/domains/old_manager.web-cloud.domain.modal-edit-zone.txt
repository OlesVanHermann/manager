# Cahier des Charges : web-cloud/domain - Modal Modifier entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un enregistrement DNS
- **Type** : Wizard 2 étapes
- **Objectif** : Éditer un enregistrement DNS existant

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 2 étapes
- Formulaire modification
- Panneau aide latéral
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Grande modal (900x690)
- **OdsStepper** : 2 étapes
- **OdsInput** : Sous-domaine, TTL, Cible
- **OdsCard** : Panneau aide
- **OdsMessage** : Info valeur actuelle
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Modification | Actif |
| 2 | Résumé | À venir |
