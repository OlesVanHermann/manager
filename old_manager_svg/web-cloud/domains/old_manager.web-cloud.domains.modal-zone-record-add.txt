# Cahier des Charges : web-cloud/domains - Modal Zone Record Add

## 1. VUE D'ENSEMBLE
- **Type** : Modal création entrée DNS
- **URL** : N/A (composant modal)
- **Objectif** : Ajouter enregistrement zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Sélection type enregistrement
- Formulaire configuration
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 560px |

## 4. SECTION TYPES - POINTAGE
| Type | Description |
|------|-------------|
| A | IPv4 (sélectionné) |
| AAAA | IPv6 |
| NS | Serveur de noms |
| CNAME | Alias canonique |
| DNAME | Alias de domaine |

## 5. SECTION TYPES - AVANCÉS
| Type | Description |
|------|-------------|
| CAA | Autorité certificat |
| TXT | Texte |
| NAPTR | Réécriture URI |
| SRV | Service |
| LOC | Localisation |

## 6. SECTION TYPES - MAIL
| Type | Description |
|------|-------------|
| MX | Serveur mail |
| SPF | Sender Policy Framework |
| DKIM | DomainKeys |
| DMARC | Authentication mail |

## 7. FORMULAIRE ENREGISTREMENT A
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Sous-domaine | Input + suffixe | Non |
| Cible (IPv4) | Input | Oui |
| TTL | Input number | Non |

## 8. VALEURS TTL
| Propriété | Valeur |
|-----------|--------|
| Défaut | 3600 |
| Unité | secondes |

## 9. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Créer | Primary | "Créer" |

## 10. APIS & ENDPOINTS
- POST /domain/zone/{domain}/record - Créer enregistrement
