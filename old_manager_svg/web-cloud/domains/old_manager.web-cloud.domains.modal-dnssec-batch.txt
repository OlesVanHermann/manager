# Cahier des Charges : web-cloud/domains - Modal DNSSEC Batch

## 1. VUE D'ENSEMBLE
- **Page** : Configuration DNSSEC en lot
- **Type** : Modal formulaire avec tableau
- **Objectif** : Activer/désactiver DNSSEC sur plusieurs domaines

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau explicatif DNSSEC
- Sélection action (radio)
- Tableau domaines sélectionnés
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal large
- **OdsMessage** : Bandeau info DNSSEC
- **OdsRadio** : Choix action
- **OdsDatagrid** : Tableau domaines
- **OdsBadge** : États (Activé, Désactivé)
- **OdsButton** : Annuler, Appliquer

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte | DNSSEC (Domain Name System Security Extensions) permet de sécuriser les réponses DNS contre les attaques de type "man-in-the-middle" |

## 5. OPTIONS RADIO
| Option | Description |
|--------|-------------|
| Activer DNSSEC | Protège vos domaines contre les attaques DNS |
| Désactiver DNSSEC | Retire la protection DNSSEC de vos domaines |

## 6. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom de domaine | Text | Nom du domaine |
| Statut actuel | Badge | Activé (vert) / Désactivé (rouge) |
| Nouveau statut | Badge/Text | Activé / Désactivé / Aucun changement |

## 7. BADGES
| État | Couleur |
|------|---------|
| Activé | Vert |
| Désactivé | Rouge |

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Appliquer | Primary | Droite |

## 9. APIS & ENDPOINTS
- POST /domain/batch/dnssec - Configuration DNSSEC en lot
