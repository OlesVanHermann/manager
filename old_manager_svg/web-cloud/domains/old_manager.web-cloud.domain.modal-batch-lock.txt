# Cahier des Charges : web-cloud/domain - Modal Batch Lock

## 1. VUE D'ENSEMBLE

**Type:** Modal action batch
**Fonction:** Verrouiller/déverrouiller le transfert de plusieurs domaines
**Contexte:** Action groupée depuis liste des domaines

## 2. DIMENSIONS

- Modal: 700x620px
- Position: centrée avec overlay

## 3. STRUCTURE MODAL

### Header
- Background: #FFFFFF
- Titre: "Verrouillage transfert (Batch)"
- Bouton fermer: × (coin droit)

### Content
- Padding: 24px

### Footer
- Séparateur: ligne #E5E7EB

## 4. CONTENU

### Info banner
- Background: #DBEAFE
- Dimensions: 652x50px
- Titre: "Protection contre le transfert"
- Description: "Le verrouillage empêche le transfert non autorisé de vos domaines vers un autre registrar."

### Action sélecteur
- Label: "Action à appliquer :"
- Dropdown: 200x36px
- Options: "Verrouiller" / "Déverrouiller"

### Liste domaines
- Titre: "Domaines sélectionnés (5)"
- Colonnes:
  - Checkbox
  - DOMAINE
  - STATUT ACTUEL

### Badges statut

| Badge | Background | Text color |
|-------|------------|------------|
| Verrouillé | #DEF7EC | #03543F |
| Déverrouillé | #FEF3C7 | #92400E |

### Exemples données

| Domaine | Statut | Checkbox |
|---------|--------|----------|
| example.com | Déverrouillé | ✓ |
| example.fr | Déverrouillé | ✓ |
| example.eu | Verrouillé | ✓ |
| example.net | Non supporté (.net externe) | ☐ (disabled) |
| example.org | Déverrouillé | ✓ |

### Summary banner
- Background: #FEF3C7
- Dimensions: 652x45px
- Titre: "Résumé de l'opération"
- Message: "4 domaines seront verrouillés. 1 domaine ignoré (non supporté)."

## 7. APIS & ENDPOINTS

```
GET /domain/{serviceName}/transferLock
→ Statut verrouillage actuel

POST /domain/{serviceName}/lock
→ Verrouille le domaine

DELETE /domain/{serviceName}/lock
→ Déverrouille le domaine

# Pour chaque domaine sélectionné:
POST /domain/{domain}/lock     # si verrouillage
DELETE /domain/{domain}/lock   # si déverrouillage
```
