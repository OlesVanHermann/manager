# Cahier des Charges : web-cloud/domains - Modal DynHost Login Add

## 1. VUE D'ENSEMBLE
- **Type** : Modal création identifiant DynHost
- **URL** : N/A (composant modal)
- **Objectif** : Créer nouvel identifiant DynHost

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Description
- Formulaire
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 480px |

## 4. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Suffixe de l'identifiant | Input avec préfixe readonly | Oui |
| Sous-domaine autorisé | Input (* ou spécifique) | Oui |
| Mot de passe | Password | Oui |
| Confirmer le mot de passe | Password | Oui |

## 5. SUFFIXE IDENTIFIANT
| Propriété | Valeur |
|-----------|--------|
| Préfixe | "exemple.com -" (readonly) |
| Placeholder | "dyn_user" |

## 6. SOUS-DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Placeholder | "* (tous) ou dyn" |
| Suffixe | ".exemple.com" |
| Aide | "Utilisez * pour autoriser tous les sous-domaines" |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Créer | Primary | "Créer" |

## 8. APIS & ENDPOINTS
- POST /domain/{domain}/dynHost/login - Créer identifiant
