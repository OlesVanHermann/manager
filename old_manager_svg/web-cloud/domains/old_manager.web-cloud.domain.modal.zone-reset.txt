# Cahier des Charges : web-cloud/domain - Modal Réinitialiser zone DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Réinitialisation de la zone DNS
- **Type** : Modal danger avec confirmation
- **Objectif** : Supprimer tous les enregistrements et réinitialiser

## 2. STRUCTURE MODAL
- Header avec icône warning rouge
- Titre
- Bandeau danger
- Options de réinitialisation (3 radios)
- Champ confirmation par saisie domaine
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x560)
- **OdsIcon** : Point d'exclamation rouge sur cercle rouge clair
- **OdsMessage** : Bandeau danger rouge
- **OdsRadio** : 3 options de réinitialisation
- **OdsInput** : Confirmation par saisie domaine
- **OdsButton** : Annuler, Réinitialiser (danger rouge)

## 4. ICÔNE WARNING
- Cercle rouge clair (#FEE2E2)
- Point d'exclamation rouge (#991B1B)

## 5. TITRE
"Réinitialiser la zone DNS"

## 6. BANDEAU DANGER
Fond rouge (#FEE2E2) :
- "⚠️ Cette action est irréversible ! Tous les enregistrements"
- "DNS personnalisés seront supprimés."

## 7. OPTIONS DE RÉINITIALISATION
Label : "Type de réinitialisation"

| Option | Description |
|--------|-------------|
| ● Réinitialisation minimale | Crée uniquement les enregistrements NS de base |
| ○ Réinitialisation avec redirection OVH | Pointe le domaine vers les serveurs OVH (A, MX, CNAME) |
| ○ Réinitialisation avec hébergement web | Configure la zone pour votre hébergement OVH existant |

## 8. CONFIRMATION
- Label : "Confirmez en tapant le nom du domaine"
- Input : "example.com"
- Validation : Doit correspondre exactement au nom du domaine

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Réinitialiser (danger rouge)

## 10. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/reset : Réinitialiser la zone
  - minimized: boolean (réinitialisation minimale)
  - DnsRecords: array (enregistrements à recréer)
