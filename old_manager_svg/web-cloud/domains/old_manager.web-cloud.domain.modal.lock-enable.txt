# Cahier des Charges : web-cloud/domain - Modal Activer verrouillage

## 1. VUE D'ENSEMBLE
- **Modal** : Activation du verrouillage transfert
- **Type** : Modal de confirmation avec informations
- **Objectif** : Verrouiller le domaine contre les transferts

## 2. STRUCTURE MODAL
- Header avec icône cadenas bleu
- Titre
- Description
- Bandeau avantages (vert)
- Domaine concerné
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsIcon** : Cadenas bleu sur cercle bleu clair
- **OdsMessage** : Bandeau succès vert
- **OdsButton** : Annuler, Verrouiller

## 4. ICÔNE HEADER
- Cercle bleu clair (#E8F0FE)
- Icône cadenas bleu (#0050D7)

## 5. TITRE
"Activer le verrouillage du transfert"

## 6. DESCRIPTION
- "Le verrouillage du transfert empêche tout transfert"
- "non autorisé de votre nom de domaine vers un"
- "autre registrar."

## 7. BANDEAU AVANTAGES
Fond vert (#DEF7EC), texte vert foncé :
- ✓ Recommandé pour la sécurité
- ✓ Peut être désactivé à tout moment

## 8. DOMAINE CONCERNÉ
- Label : "Domaine concerné :"
- Valeur : **example.com**

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Verrouiller (primaire)

## 10. APIS & ENDPOINTS
- PUT /domain/{serviceName} : Modifier verrouillage
  - transferLockStatus: "locked"

## 11. NOTES
- Protection contre le vol de domaine
- Empêche les transferts non autorisés
- Peut être désactivé pour transférer le domaine
