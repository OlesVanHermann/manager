# Cahier des Charges : web-cloud/domains - Liste des domaines

## 1. VUE D'ENSEMBLE
- **Page** : Liste des noms de domaine
- **Type** : Page liste avec tableau
- **Objectif** : Gérer tous les domaines du compte

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar avec navigation
- Titre "Noms de domaine"
- Barre de recherche
- Boutons actions groupées
- Tableau des domaines
- Pagination

## 3. COMPOSANTS UI
- **OdsInput** : Recherche domaine
- **OdsButton** : Actions groupées (Renouveler, DNSSEC, Export, Propriétaire)
- **OdsDatagrid** : Tableau domaines
- **OdsCheckbox** : Sélection multiple
- **OdsBadge** : États (Configurable, Activé, Désactivé, Non supporté)
- **OdsActionMenu** : Menu actions par ligne
- **OdsPagination** : Navigation pages

## 4. SIDEBAR NAVIGATION
| Item | État |
|------|------|
| Tous mes domaines | Actif (sélectionné) |
| Opérations en cours | Normal |

## 5. BOUTONS ACTIONS GROUPÉES
| Bouton | Action |
|--------|--------|
| Renouveler | Renouvellement en lot |
| DNSSEC | Configuration DNSSEC en lot |
| Exporter en CSV | Export liste domaines |
| Changer le propriétaire | Lien externe ↗ |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| ☐ | Checkbox | Sélection multiple |
| Nom | Link | Nom du domaine (cliquable) |
| Expiration | Date | Date d'expiration |
| WHOIS | Badge | Configurable / Non supporté |
| DNSSEC | Badge | Activé / Désactivé |
| DNS Anycast | Badge | Activé / Désactivé / Non dispo. |
| Actions | ActionMenu | Menu ⋮ |

## 7. BADGES ÉTATS
| État | Couleur |
|------|---------|
| Configurable | Vert |
| Activé | Vert |
| Désactivé | Rouge |
| Non supporté | Orange |
| Non dispo. | Orange |

## 8. ACTIONMENU (⋮)
| Action | Description |
|--------|-------------|
| Renouveler | Renouveler le domaine |
| Configurer WHOIS | Modifier infos WHOIS |
| Activer DNSSEC | Activer/désactiver DNSSEC |

## 9. PAGINATION
| Élément | Description |
|---------|-------------|
| Info | Affichage de X à Y sur Z domaines |
| Boutons | ‹ 1 2 3 › |

## 10. APIS & ENDPOINTS
- GET /domain - Liste des domaines
- GET /domain/{serviceName} - Détails domaine
