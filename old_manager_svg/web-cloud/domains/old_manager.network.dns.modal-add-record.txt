# Cahier des Charges : network/dns - Modal Add Record

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout enregistrement DNS
- **Contexte** : Network DNS
- **Objectif** : Ajouter un nouvel enregistrement à la zone DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal | 600x480px |
| Background | Blanc |
| Overlay | Noir 50% opacité |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type d'enregistrement
- **OdsInput** : Sous-domaine, Valeur, TTL
- **OdsButton** : Actions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Titre | "Ajouter un enregistrement" |
| Bouton fermer | × |

## 5. SELECT TYPE
| Propriété | Valeur |
|-----------|--------|
| Label | "Type" |
| Options | A, AAAA, CNAME, MX, TXT, NS, SRV... |
| Valeur défaut | "A" |
| Largeur | 450px |

## 6. CHAMP SOUS-DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Label | "Sous-domaine" |
| Placeholder | "@ pour le domaine racine" |
| Largeur | 450px |

## 7. CHAMP VALEUR
| Propriété | Valeur |
|-----------|--------|
| Label | "Valeur" |
| Placeholder | "Adresse IP ou valeur..." |
| Largeur | 450px |

## 8. CHAMP TTL
| Propriété | Valeur |
|-----------|--------|
| Label | "TTL (secondes)" |
| Valeur défaut | 3600 |
| Largeur | 200px |
| Aide | "Recommandé: 3600 (1 heure)" |

## 9. BOUTONS FOOTER
| Bouton | Style | Largeur |
|--------|-------|---------|
| Annuler | Outline | 100px |
| Ajouter | Primary #0050D7 | 100px |

## 10. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/record/type - Types disponibles
- POST /domain/zone/{zoneName}/record - Créer enregistrement
- POST /domain/zone/{zoneName}/refresh - Rafraîchir zone
