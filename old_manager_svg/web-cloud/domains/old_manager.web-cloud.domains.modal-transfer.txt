# Cahier des Charges : web-cloud/domains - Modal Transfert

## 1. VUE D'ENSEMBLE
- **Page** : Transférer un domaine
- **Type** : Modal formulaire
- **Objectif** : Transfert entrant d'un domaine

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau info (AuthCode)
- Input nom de domaine
- Input AuthCode
- Options (checkboxes)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Info (bleu)
- **OdsInput** : Domaine, AuthCode
- **OdsCheckbox** : Options transfert
- **OdsButton** : Annuler, Transférer

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | Le transfert entrant nécessite le code d'autorisation (AuthCode) de votre registrar actuel |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom de domaine | Input | Oui | example.com |
| AuthCode | Input | Oui | XXXX-XXXX-XXXX-XXXX |

## 6. OPTIONS (Checkboxes)
| Option | État par défaut | Description |
|--------|-----------------|-------------|
| Renouveler pour 1 an | ✓ Coché | Renouvellement automatique |
| Activer DNSSEC | ✓ Coché | Protection DNS |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Transférer | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /domain/transfer - Initier transfert entrant
