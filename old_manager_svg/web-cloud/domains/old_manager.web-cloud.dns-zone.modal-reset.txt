# Cahier des Charges : web-cloud/dns-zone - Modal Réinitialiser zone

## 1. VUE D'ENSEMBLE
- **Modal** : Réinitialisation complète de la zone DNS
- **Type** : Modal danger avec confirmation
- **Objectif** : Supprimer tous les enregistrements et restaurer les valeurs par défaut

## 2. STRUCTURE MODAL
- Header rouge avec titre et bouton fermer (×)
- Bandeau warning rouge
- Info zone DNS
- Checkbox confirmation
- Boutons Annuler / Réinitialiser

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x260)
- **OdsMessage** : Warning rouge
- **OdsText** : Info zone
- **OdsCheckbox** : Confirmation
- **OdsButton** : Annuler, Réinitialiser (danger)

## 4. TITRE
"Réinitialiser la zone DNS"
(Header fond rouge #FEE2E2, texte #DC2626)

## 5. BANDEAU WARNING
Fond rouge (#FEE2E2) :
- "⚠ ATTENTION"
- "Cette action supprimera TOUS les enregistrements"
- "et restaurera les valeurs par défaut."

## 6. INFO ZONE
- Label: "Zone DNS :"
- Valeur: "domain.com" (bold)

## 7. CHECKBOX CONFIRMATION
- Label: "Je comprends les conséquences"
- État: Non coché par défaut
- Le bouton Réinitialiser est désactivé tant que non coché

## 8. BOUTONS
| Bouton | Style | État | Action |
|--------|-------|------|--------|
| Annuler | Outline | Toujours actif | Ferme la modal |
| Réinitialiser | Danger rouge | Actif si checkbox coché | Réinitialise la zone |

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/reset
  - minimized: boolean (true = entrées minimales SOA+NS)
- POST /domain/zone/{zoneName}/refresh : Appliquer les changements
