# Cahier des Charges : web-cloud/domains - Modal Zone Reset

## 1. VUE D'ENSEMBLE
- **Type** : Modal réinitialisation zone DNS
- **URL** : N/A (composant modal)
- **Objectif** : Réinitialiser zone DNS par défaut

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Warning
- Question confirmation
- Sélecteur type réinitialisation
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 280px |

## 4. WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (rouge) |
| Fond | #FEE2E2 |
| Ligne 1 | "⚠ Attention : cette action supprimera toutes les entrées DNS" |
| Ligne 2 | "personnalisées et rétablira la configuration par défaut." |

## 5. CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Êtes-vous sûr de vouloir réinitialiser la zone DNS de" |
| Domaine | "exemple.com" (bold) |

## 6. TYPE RÉINITIALISATION
| Propriété | Valeur |
|-----------|--------|
| Type | Select dropdown |
| Options | "Configuration minimale", "Configuration recommandée" |
| Défaut | "Configuration minimale" |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Réinitialiser | Danger (rouge) | "Réinitialiser" |

## 8. APIS & ENDPOINTS
- POST /domain/zone/{domain}/reset - Réinitialiser zone
