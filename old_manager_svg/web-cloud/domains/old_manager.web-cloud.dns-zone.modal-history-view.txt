# Cahier des Charges : web-cloud/dns-zone - Modal Voir version historique

## 1. VUE D'ENSEMBLE
- **Modal** : Visualisation d'une version de zone historique
- **Type** : Modal lecture seule avec contenu fichier zone
- **Objectif** : Consulter le contenu de la zone à une date précise

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Encadré info (domaine + date + nb enregistrements)
- Label "Contenu de la zone à cette date"
- Zone texte monospace (contenu fichier zone)
- Boutons Fermer / Restaurer cette version

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x510)
- **OdsCard** : Info version (fond gris)
- **OdsTextarea** : Contenu zone (readonly, monospace)
- **OdsButton** : Fermer, Restaurer cette version

## 4. TITRE (Header bleu)
"Visualiser l'historique de zone DNS"

## 5. ENCADRÉ INFO
Fond gris (#F9FAFB), bordure (#E5E7EB) :
- Ligne 1: "{domaine}" (bold) - "Version du {date} à {heure}"
- Ligne 2: "{nb} enregistrements • Créée automatiquement avant modification"

## 6. LABEL
"Contenu de la zone à cette date :"

## 7. ZONE TEXTE
Fond gris (#F3F4F6), police monospace, lecture seule :
```
$TTL 3600
@       IN    SOA   dns.exemple.com. admin.exemple.com. (
                    2024011501 ; Serial
                    86400      ; Refresh
                    7200       ; Retry
                    3600000    ; Expire
                    86400 )    ; Minimum TTL

@       IN    NS    dns10.ovh.net.
@       IN    NS    ns10.ovh.net.
@       IN    A     192.168.1.100
www     IN    CNAME @
mail    IN    A     192.168.1.101
@       IN    MX    10 mail.exemple.com.
```

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Outline | Ferme la modal |
| Restaurer cette version | Primaire bleu | Ouvre modal-history-restore |

## 9. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/history/{date}/content
  - Retourne: string (contenu fichier zone)
