# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement MX

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type MX
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement MX (serveur de messagerie)

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Bandeau info explicatif
- Champ sous-domaine
- Champ priorité
- Champ serveur de messagerie
- Champ TTL
- Aperçu de l'enregistrement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x660)
- **OdsMessage** : Bandeau info bleu
- **OdsInput** : Sous-domaine, Priorité, Serveur mail
- **OdsSelect** : TTL
- **OdsCard** : Aperçu (fond gris)
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un enregistrement MX"

## 5. BANDEAU INFO
Fond bleu (#E8F0FE) :
- **Titre** : "ℹ️ Enregistrement MX (Mail Exchange)"
- "Indique les serveurs de messagerie pour votre domaine."

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Aide | "Généralement vide pour les emails du domaine principal" |
| Input | "@ (domaine principal)" (placeholder) |
| Suffixe (addon gris) | ".example-domain.com" |

## 7. CHAMP PRIORITÉ
| Élément | Valeur |
|---------|--------|
| Label | "Priorité *" |
| Aide | "Plus le nombre est bas, plus la priorité est haute (0-65535)" |
| Obligatoire | Oui |
| Input | 10 |

## 8. CHAMP SERVEUR DE MESSAGERIE
| Élément | Valeur |
|---------|--------|
| Label | "Serveur de messagerie *" |
| Aide | "Nom du serveur mail (doit se terminer par un point)" |
| Obligatoire | Oui |
| Input | "mx1.mail.ovh.net." |

## 9. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL (Time To Live)" |
| Type | Select dropdown |
| Valeur | "3600 (1 heure)" |

## 10. APERÇU
Fond gris (#F9FAFB), police monospace :
`@  3600  IN  MX  10 mx1.mail.ovh.net.`

## 11. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter (primaire)

## 12. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement MX
  - fieldType: "MX"
  - subDomain: string
  - target: "priority mailserver"
  - ttl: number
