# Cahier des Charges : web-cloud/domain - Serveurs DNS (vue détaillée)

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des serveurs DNS
- **Type** : Page avec tableau
- **Objectif** : Gérer les serveurs DNS du domaine

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar navigation
- Breadcrumb
- Titre
- Boutons d'action
- Tableau serveurs DNS
- Section informations

## 3. COMPOSANTS UI
- **OdsButton** : Ajouter serveur (primaire), Réinitialiser (secondaire)
- **OdsTable** : Liste des serveurs DNS
- **OdsBadge** : Statut Actif (vert)
- **OdsActionMenu** : Menu contextuel (⋮)
- **OdsCard** : Section informations (fond bleu)

## 4. BOUTONS D'ACTION
| Bouton | Style |
|--------|-------|
| Ajouter un serveur DNS | Primaire bleu |
| Réinitialiser les serveurs DNS | Secondaire outline |

## 5. TABLEAU SERVEURS DNS
| Colonne | Description |
|---------|-------------|
| Serveur DNS | Nom FQDN du serveur |
| IP associée | Adresse IP |
| Statut | Badge Actif |
| Actions | Menu ⋮ |

### Exemple données
| Serveur DNS | IP Associée | Statut |
|-------------|-------------|--------|
| dns100.ovh.net | 213.186.33.99 | Actif |
| ns100.ovh.net | 213.251.128.100 | Actif |

## 6. MENU ACTIONS (⋮)
- Supprimer (texte rouge danger)

## 7. SECTION INFORMATIONS
Fond bleu (#E8F0FE) :
- "**Informations sur vos serveurs DNS**"
- "Vous utilisez actuellement les serveurs DNS OVH recommandés pour votre domaine :"
- • dns100.ovh.net
- • ns100.ovh.net

## 8. APIS & ENDPOINTS
- GET /domain/{serviceName}/nameServer : Liste serveurs DNS
- POST /domain/{serviceName}/nameServer : Ajouter serveur
- DELETE /domain/{serviceName}/nameServer/{id} : Supprimer
- POST /domain/{serviceName}/nameServer/reset : Réinitialiser
