# Cahier des Charges : web-cloud/dns-zone - Modal Modifier enregistrement

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un enregistrement DNS
- **Type** : Modal formulaire
- **Objectif** : Modifier la cible et le TTL d'un enregistrement existant

## 2. STRUCTURE MODAL
- Header gris avec titre et bouton fermer (×)
- Type (lecture seule)
- Sous-domaine (lecture seule)
- Champ cible (modifiable)
- Champ TTL (modifiable)
- Bandeau warning propagation
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsInput** : Type (disabled), Sous-domaine (disabled), Cible, TTL
- **OdsMessage** : Warning orange
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier l'enregistrement"

## 5. CHAMPS
### Type (lecture seule)
- Type: Input disabled
- Fond: Gris (#F3F4F6)
- Valeur: "A"

### Sous-domaine (lecture seule)
- Type: Input disabled
- Fond: Gris (#F3F4F6)
- Valeur: "www.domain.com"

### Cible *
- Type: Input modifiable
- Valeur: "192.168.1.100"

### TTL
- Type: Select
- Valeur: "3600"
- Suffixe: secondes

## 6. BANDEAU WARNING
Fond jaune (#FEF3C7) :
"⚠ Les modifications DNS peuvent prendre jusqu'à 24h pour se propager."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Modifier | Primaire bleu | Modifie l'enregistrement |

## 8. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName}/record/{id}
  - target: string
  - ttl: number
- POST /domain/zone/{zoneName}/refresh : Rafraîchir zone
