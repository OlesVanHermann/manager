# Cahier des Charges : web-cloud/domain - Modal Modifier GLUE

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un enregistrement GLUE
- **Type** : Formulaire de modification
- **Objectif** : Éditer les IPs d'un serveur de noms personnalisé

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Description
- Champ hôte (lecture seule)
- Champ IPs (éditable)
- Aide contextuelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x420)
- **OdsInput** : Champ hôte (disabled), Champ IP
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier l'enregistrement GLUE"

## 5. DESCRIPTION
"Veuillez renseigner les champs suivants :"

## 6. CHAMP HÔTE
| Élément | Valeur |
|---------|--------|
| Label | "Nom du hôte" |
| État | Désactivé (fond gris) |
| Valeur | ns1.example.com |

## 7. CHAMP IPS
| Élément | Valeur |
|---------|--------|
| Label | "IP(s) de destination" |
| Obligatoire | Oui (*) |
| Valeur exemple | 203.0.113.10, 203.0.113.11 |

## 8. AIDE CONTEXTUELLE
- "Vous pouvez spécifier plusieurs IPs, séparées par une virgule."
- "Vous pouvez spécifier des IPv4 et des IPv6."

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Modifier (primaire)

## 10. APIS & ENDPOINTS
- PUT /domain/{serviceName}/glueRecord/{host} : Modifier GLUE
