# Cahier des Charges : web-cloud/dns-zone - Créer une zone DNS (modal)

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une nouvelle zone DNS
- **Type** : Modal formulaire
- **Objectif** : Créer une zone DNS pour un domaine externe

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Bandeau info bleu
- Champ nom de zone (obligatoire)
- Champ IP par défaut (optionnel)
- Checkboxes options
- Boutons Annuler / Créer la zone

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x440)
- **OdsMessage** : Info bleu
- **OdsInput** : Nom de la zone, Adresse IP
- **OdsCheckbox** : Options création
- **OdsButton** : Annuler, Créer la zone

## 4. TITRE (Header bleu)
"Créer une zone DNS"

## 5. BANDEAU INFO
Fond bleu (#DBEAFE) :
- "Créez une zone DNS pour un domaine externe ou un sous-domaine."
- "Vous pourrez ensuite gérer les enregistrements DNS depuis OVH."

## 6. CHAMPS
### Nom de la zone *
- Type: Input obligatoire
- Placeholder: "mon-domaine-externe.com"
- Aide: "Entrez un nom de domaine complet (FQDN) ou un sous-domaine"

### Adresse IP par défaut (optionnel)
- Type: Input optionnel
- Placeholder: "ex: 192.168.1.1"
- Aide: "Cette IP sera utilisée pour créer un enregistrement A par défaut"

## 7. OPTIONS (CHECKBOXES)
| Checkbox | État par défaut | Description |
|----------|-----------------|-------------|
| ☑ | Coché | Créer les entrées DNS minimales (SOA, NS) |
| ☑ | Coché | Utiliser les serveurs DNS OVH (dns*.ovh.net) |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Créer la zone | Primaire bleu | Crée la zone DNS |

## 9. APIS & ENDPOINTS
- POST /domain/zone
  - zoneName: string (nom de domaine)
  - template: string ("minimized", "standard")
