# Cahier des Charges : web-cloud/domain - Modal DNSSEC Disable

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation danger
**Fonction:** Désactiver DNSSEC sur un domaine
**Contexte:** Action depuis configuration DNSSEC

## 2. DIMENSIONS

- Modal: 550x400px
- Position: centrée avec overlay

## 3. STRUCTURE MODAL

### Header
- Background: #FEE2E2 (error)
- Titre: "Désactiver DNSSEC" (text-error, centré)
- Bouton fermer: × (coin droit)

### Content
- Padding: 30px

### Footer
- Séparateur: ligne #E5E7EB

## 4. CONTENU

### Warning banner
- Background: #FEF3C7
- Dimensions: 490x60px
- Titre: "Réduction de la sécurité"
- Description: "La désactivation de DNSSEC supprimera les signatures cryptographiques de votre zone DNS."

### Texte confirmation
- Question: "Êtes-vous sûr de vouloir désactiver DNSSEC pour"
- Domaine: "exemple.com" (13px, font-weight 600)

### Conséquences
- Titre: "Conséquences :"
- Liste (text-muted, 10px):
  - Les enregistrements DS seront supprimés du registre
  - La validation DNSSEC ne sera plus possible pour ce domaine
  - La propagation peut prendre jusqu'à 48h

## 5. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 110x34px |
| Désactiver | btn-danger (#DC2626) | 130x34px |

## 6. COMPOSANTS UI

### Header error
- Background: #FEE2E2
- Border-radius: 8px 8px 0 0

### Bouton danger
- Background: #DC2626
- Text: white
- Hover: #B91C1C

### Liste conséquences
- Bullet: -
- Color: text-muted
- Font-size: 10px

## 7. APIS & ENDPOINTS

```
DELETE /domain/{serviceName}/dnssec
→ Désactive DNSSEC

# Conséquence automatique:
# Les enregistrements DS sont supprimés du registre parent
```
