# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement LOC

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type LOC
- **Type** : Formulaire avancé de création
- **Objectif** : Ajouter un enregistrement LOC (localisation géographique)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge LOC et bouton fermer (✕)
- Champ sous-domaine
- Champ TTL
- Section Latitude (degrés, minutes, secondes, direction)
- Section Longitude (degrés, minutes, secondes, direction)
- Altitude, Taille, Précisions
- Aide contextuelle
- Aperçu
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Grande modal (700x620)
- **OdsInput** : Multiples champs numériques
- **OdsCard** : Aperçu (fond gris)
- **OdsBadge** : Type LOC (vert)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - LOC" avec badge [LOC]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "datacenter" |
| Suffixe (addon gris) | ".exemple.com" |

## 6. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL" |
| Input | 3600 |

## 7. SECTION LATITUDE
| Champ | Largeur | Valeur |
|-------|---------|--------|
| Degrés * | 70px | 48 |
| Minutes | 70px | 51 |
| Secondes | 70px | 24.0 |
| Direction | 50px | N |

## 8. SECTION LONGITUDE
| Champ | Largeur | Valeur |
|-------|---------|--------|
| Degrés * | 70px | 2 |
| Minutes | 70px | 21 |
| Secondes | 70px | 7.0 |
| Direction | 50px | E |

## 9. PARAMÈTRES COMPLÉMENTAIRES
| Champ | Largeur | Valeur |
|-------|---------|--------|
| Altitude (mètres) * | 150px | 35.0 |
| Taille (mètres) | 120px | 1.0 |
| Précision horizontale | 120px | 10000.0 |
| Précision verticale | 120px | 10.0 |

## 10. AIDE CONTEXTUELLE
- "Exemple: Coordonnées GPS de Paris (Tour Eiffel): 48°51'24"N 2°21'07"E"
- "L'enregistrement LOC permet de spécifier la localisation géographique d'un hôte."

## 11. APERÇU
Fond gris (#F9FAFB) :
- **Label** : "Aperçu de l'enregistrement:"
- `datacenter.exemple.com. 3600 IN LOC 48 51 24.000 N 2 21 7.000 E 35.00m 1.00m 10000.00m 10.00m`

## 12. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 13. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement LOC
  - fieldType: "LOC"
  - subDomain: string
  - target: format LOC
  - ttl: number
