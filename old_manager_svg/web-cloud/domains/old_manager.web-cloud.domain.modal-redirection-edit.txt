# Cahier des Charges : web-cloud/domain - Modal Éditer redirection

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'une redirection web
- **Type** : Formulaire de modification
- **Objectif** : Éditer une redirection HTTP existante

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Description actuelle
- Champ adresse web
- Section Options (Titre, Mots clés, Description)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x550)
- **OdsInput** : Adresse web, Titre, Mots clés
- **OdsTextarea** : Description
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Édition de la redirection"

## 5. DESCRIPTION ACTUELLE
"Édition de la redirection du sous-domaine **www.example.com** vers **https://target.example.com**"

## 6. CHAMP ADRESSE WEB
| Élément | Valeur |
|---------|--------|
| Label | "Adresse web" |
| Obligatoire | Oui (*) |
| Input | "https://target.example.com" |

## 7. SECTION OPTIONS

### Champ Titre
| Élément | Valeur |
|---------|--------|
| Label | "Titre" |
| Placeholder | "Titre de la page (optionnel)" |

### Champ Mots clés
| Élément | Valeur |
|---------|--------|
| Label | "Mots clés" |
| Placeholder | "Mots clés séparés par des virgules (optionnel)" |

### Champ Description
| Élément | Valeur |
|---------|--------|
| Label | "Description" |
| Type | Textarea (60px) |
| Placeholder | "Description de la page (optionnel)" |

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Modifier (primaire)

## 9. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName}/redirection/{id} : Modifier redirection
