# Cahier des Charges : web-cloud/domains - Modal Serveurs DNS

## 1. VUE D'ENSEMBLE
- **Page** : Modifier les serveurs DNS
- **Type** : Modal formulaire avec options
- **Objectif** : Configurer les serveurs DNS du domaine

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ domaine (lecture seule)
- Radio mode DNS
- Champs NS personnalisés (conditionnels)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Domaine (disabled), NS1, NS2
- **OdsRadio** : Mode DNS
- **OdsButton** : Annuler, Enregistrer

## 4. CHAMP DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Label | Domaine |
| Type | Input disabled |
| Valeur | example.com |

## 5. MODE DNS (Radio)
| Option | Description |
|--------|-------------|
| DNS OVHcloud (recommandé) | Utilise ns.ovh.net |
| Serveurs DNS personnalisés | NS1/NS2 manuels |

## 6. CHAMPS NAMESERVERS
| Champ | État | Description |
|-------|------|-------------|
| NS1 | Disabled si OVHcloud | Serveur primaire |
| NS2 | Disabled si OVHcloud | Serveur secondaire |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Enregistrer | Primary | Droite |

## 8. APIS & ENDPOINTS
- PUT /domain/{serviceName}/nameServer - Modifier les NS
