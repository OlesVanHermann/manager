# Cahier des Charges : web-cloud/domain - Enregistrements GLUE

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des enregistrements GLUE
- **Type** : Page avec tableau
- **Objectif** : Créer des serveurs DNS personnalisés avec le nom de domaine

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar navigation
- Breadcrumb
- Titre
- Description
- Bouton ajouter
- Tableau GLUE records
- Section guides

## 3. COMPOSANTS UI
- **OdsButton** : Ajouter un GLUE (outline)
- **OdsTable** : Liste des enregistrements GLUE
- **OdsActionMenu** : Menu ⋮
- **OdsCard** : Section guides

## 4. BREADCRUMB
Web Cloud › Noms de domaine › example.com › **GLUE**

## 5. TITRE
**Enregistrement GLUE**

## 6. DESCRIPTION
- "Personnalisez ici vos serveurs DNS avec votre nom de domaine afin qu'ils s'affichent dans les bases WHOIS."
- "Vous pouvez créer vos propres serveurs DNS en IPv4."

## 7. BOUTON
"Ajouter un GLUE" (outline gris, à droite)

## 8. TABLEAU GLUE RECORDS
| Colonne | Description |
|---------|-------------|
| Hôte | Nom du serveur personnalisé |
| Cible(s) | Adresse(s) IP (IPv4 et/ou IPv6) |
| Actions | Menu ⋮ |

### Exemple données
| Hôte | Cible(s) |
|------|----------|
| ns1.example.com | 213.186.33.5 |
| ns2.example.com | 213.186.33.6 |
| ns3.example.com | 213.186.33.7, 2001:41d0:1:1b00::5 |

## 9. MENU ACTIONS (⋮)
- Modifier
- Supprimer

## 10. SECTION GUIDES
- › Créer un enregistrement GLUE
- › Configurer ses serveurs DNS

## 11. APIS & ENDPOINTS
- GET /domain/{serviceName}/glueRecord : Liste GLUE
- POST /domain/{serviceName}/glueRecord : Créer GLUE
- PUT /domain/{serviceName}/glueRecord/{host} : Modifier
- DELETE /domain/{serviceName}/glueRecord/{host} : Supprimer
