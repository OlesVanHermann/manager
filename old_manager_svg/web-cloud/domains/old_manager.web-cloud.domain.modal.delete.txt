# Cahier des Charges : web-cloud/domain - Modal Delete

## 1. VUE D'ENSEMBLE

**Type:** Modal confirmation danger
**Fonction:** Résilier définitivement un nom de domaine
**Contexte:** Action depuis la page informations générales

## 2. DIMENSIONS

- Modal: 500x340px
- Position: centrée avec overlay

## 3. STRUCTURE MODAL

### Design
- Pas de header classique
- Icône d'avertissement en haut
- Contenu centré

### Footer
- Séparateur: ligne #E5E7EB

## 4. CONTENU

### Icône warning
- Cercle: 70px diamètre, background #FEE2E2
- Icône: "!" (35px, color #DC2626)
- Position: centré en haut

### Titre
- Texte: "Résilier exemple.com ?"
- Font-size: 16px, font-weight 600
- Centré

### Description
- Texte: "Cette action est irréversible. Le domaine sera libéré et pourra être enregistré par quelqu'un d'autre."
- Font-size: 11px, text-muted
- Centré, 2 lignes

### Confirmation input
- Label: "Tapez \"exemple.com\" pour confirmer"
- Input: 452x36px
- Placeholder: curseur clignotant

## 5. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 180x34px |
| Résilier définitivement | btn-danger (#DC2626) | 180x34px |

## 6. VALIDATION

Le bouton "Résilier définitivement" est désactivé tant que:
- L'input ne contient pas exactement le nom du domaine
- Le texte doit correspondre caractère par caractère

## 7. APIS & ENDPOINTS

```
DELETE /domain/{serviceName}/terminate
→ Lance la résiliation

# Conséquences:
# - Le domaine est libéré à expiration
# - La zone DNS est supprimée
# - Les services associés sont résiliés
# - Action irréversible

GET /domain/{serviceName}/serviceInfos
→ Vérifie le statut avant résiliation
```
