# Cahier des Charges : web-cloud/domain - Modal Supprimer accès DynHost

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un identifiant d'accès DynHost
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer un compte d'accès DynHost

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation avec identifiant
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (500x280)
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. TITRE
"Supprimer un identifiant d'accès au Dynhost"

## 5. QUESTION
"Êtes-vous sûr de vouloir supprimer l'identifiant d'accès **example.com-admin** ?"

## 6. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
- "⚠️ Cette action est irréversible."
- "L'accès DynHost ne pourra plus être utilisé."

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 8. APIS & ENDPOINTS
- DELETE /domain/zone/{zoneName}/dynHost/login/{login} : Supprimer accès
