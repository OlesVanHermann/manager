# Cahier des Charges : web-cloud/domain - Informations générales

## 1. VUE D'ENSEMBLE
- **Écran** : Page principale des informations du domaine
- **Type** : Dashboard avec onglets et tuiles
- **Objectif** : Vue d'ensemble complète du domaine

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar avec liste domaines
- Breadcrumb
- Titre domaine
- Onglets navigation
- Grille de tuiles (3 colonnes + 2 colonnes)

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation principale
- **OdsCard** : Tuiles informations
- **OdsBadge** : Statuts (Actif, Activé, Verrouillé)
- **OdsActionMenu** : Menus ⋮ par section
- **OdsLink** : Actions et liens
