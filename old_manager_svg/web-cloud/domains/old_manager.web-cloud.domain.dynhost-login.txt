# Cahier des Charges : web-cloud/domain - Gestion accès DynHost

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des accès DynHost
- **Type** : Page avec tableau
- **Objectif** : Gérer les identifiants de connexion DynHost

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar navigation
- Breadcrumb
- Bouton retour
- Titre
- Bouton créer accès
- Tableau des accès

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsButton** : Créer un accès (primaire)
- **OdsTable** : Liste des accès DynHost
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. BREADCRUMB
Web Cloud › Noms de domaine › example.com › **DynHost**

## 5. NAVIGATION
← Retour (lien bleu)

## 6. TITRE
**Gestion des accès DynHost**
Bouton : "+ Créer un accès" (primaire, à droite)

## 7. TABLEAU ACCÈS DYNHOST
| Colonne | Description |
|---------|-------------|
| IDENTIFIANT | Login de connexion |
| SOUS-DOMAINE | Sous-domaine associé |
| DERNIÈRE MISE À JOUR | Date/heure dernière MAJ |
| ACTIONS | Menu ⋮ |

### Exemple données
| Identifiant | Sous-domaine | Dernière mise à jour |
|-------------|--------------|---------------------|
| example.com-admin | dyn.example.com | 2024-01-15 14:32 |
| example.com-home | home.example.com | 2024-01-14 09:15 |
| example.com-office | office.example.com | 2024-01-10 16:45 |

## 8. MENU ACTIONS (⋮)
- Modifier
- Changer le mot de passe
- Supprimer (rouge)

## 9. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/dynHost/login : Liste accès
- POST /domain/zone/{zoneName}/dynHost/login : Créer accès
- PUT /domain/zone/{zoneName}/dynHost/login/{login} : Modifier
- POST /domain/zone/{zoneName}/dynHost/login/{login}/changePassword : Changer MDP
- DELETE /domain/zone/{zoneName}/dynHost/login/{login} : Supprimer
