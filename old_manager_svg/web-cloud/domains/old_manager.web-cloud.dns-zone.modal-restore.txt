# Cahier des Charges : web-cloud/dns-zone - Modal Restaurer zone

## 1. VUE D'ENSEMBLE
- **Modal** : Restauration de la zone DNS depuis l'historique
- **Type** : Modal confirmation avec warnings
- **Objectif** : Restaurer la zone à un état antérieur

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Message confirmation avec date
- Bandeau info bleu (sauvegarde auto)
- Bandeau warning orange (conséquences)
- Boutons Annuler / Restaurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x440)
- **OdsText** : Message confirmation
- **OdsMessage** : Info bleu, Warning orange
- **OdsButton** : Annuler, Restaurer (primaire)

## 4. TITRE
"Restaurer la zone DNS"

## 5. MESSAGE CONFIRMATION
"Souhaitez-vous restaurer la zone DNS à l'état du"
"15/12/2025 à 14:30 ?" (bold)

## 6. BANDEAU INFO (BLEU)
Fond bleu (#E8F0FE), bordure (#0050D7) :
- "ℹ️ La configuration actuelle de votre zone DNS sera sauvegardée"
- "dans l'historique avant la restauration."

## 7. BANDEAU WARNING (ORANGE)
Fond jaune (#FEF3C7), bordure (#92400E) :
- "⚠️ Attention : cette action remplacera l'ensemble des enregistrements"
- "DNS actuels. La propagation peut prendre jusqu'à 24 heures."
- "Vérifiez le contenu avant de confirmer."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Restaurer | Primaire bleu | Restaure la zone |

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/restore
  - date: string (ISO date de la version à restaurer)
