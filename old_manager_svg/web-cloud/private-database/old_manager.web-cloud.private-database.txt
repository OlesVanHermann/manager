# Cahier des Charges : web-cloud/private-database - Dashboard

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard principal Web Cloud Database
- **Type** : Page dashboard avec tuiles
- **Objectif** : Afficher les informations essentielles du serveur SQL privé

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases (menu)
- Titre service avec expiration
- Bouton Actions avec dropdown
- Onglets de navigation
- 3 tuiles d'informations

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Web Cloud Databases
- **OdsTabs** : Navigation onglets
- **OdsCard** : 3 tuiles informations
- **OdsBadge** : État du service
- **OdsProgressBar** : Espace disque
- **OdsButton** : Actions, Accès phpMyAdmin
- **OdsActionMenu** : Menu déroulant

## 4. ONGLETS

## 8. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName} : Informations service
- GET /hosting/privateDatabase/{serviceName}/serviceInfos : Infos abonnement
- POST /hosting/privateDatabase/{serviceName}/restart : Redémarrer
- POST /hosting/privateDatabase/{serviceName}/stop : Arrêter
- POST /hosting/privateDatabase/{serviceName}/start : Démarrer
