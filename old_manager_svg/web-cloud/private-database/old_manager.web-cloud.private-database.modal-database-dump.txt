# Cahier des Charges : web-cloud/private-database - Modal Créer sauvegarde

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une sauvegarde (dump)
- **Type** : Modal formulaire
- **Objectif** : Exporter une base de données en dump SQL

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Nom de la base (lecture seule)
- Type de sauvegarde (select)
- Options (checkboxes)
- Bandeau info
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x480)
- **OdsInput** : Base de données (disabled)
- **OdsSelect** : Type de sauvegarde
- **OdsCheckbox** : Options
- **OdsMessage** : Info bleu
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer une sauvegarde"

## 5. CHAMPS
### Base de données
- Type: Input disabled
- Valeur: mydb_production
- Fond: Gris (#F3F4F6)

### Type de sauvegarde
| Option | Description |
|--------|-------------|
| Sauvegarde quotidienne | Par défaut |
| Sauvegarde hebdomadaire | Planifiée |
| Sauvegarde manuelle | Immédiate |

## 6. OPTIONS
| Checkbox | État | Description |
|----------|------|-------------|
| ☑ | Coché | Envoyer par email une fois terminé |
| ☐ | Non coché | Compresser la sauvegarde (gzip) |

## 7. BANDEAU INFO
Fond bleu (#DBEAFE) :
- "ℹ La sauvegarde sera disponible dans l'onglet Sauvegardes une fois terminée."
- "Cette opération peut prendre plusieurs minutes."

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Créer | Primaire bleu | Lance la sauvegarde |

## 9. APIS & ENDPOINTS
- POST /hosting/privateDatabase/{serviceName}/database/{databaseName}/dump
  - sendEmail: boolean
  - compress: boolean
