# Cahier des Charges : web-cloud/private-database - Modal Créer base

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une nouvelle base de données
- **Type** : Modal formulaire
- **Objectif** : Créer une base de données sur le serveur SQL

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Indication champs obligatoires (*)
- Champ nom de base
- Bandeau info (règles de nommage)
- Select utilisateur associé (optionnel)
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x450)
- **OdsText** : Indication obligatoire
- **OdsInput** : Nom de la base
- **OdsMessage** : Bandeau info bleu
- **OdsSelect** : Utilisateur associé
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer une base de données"

## 5. CHAMPS
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Nom de la base de données | Input | Oui (*) | ex: wordpress_prod |
| Associer un utilisateur | Select | Non | Sélectionner un utilisateur |

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) avec bordure (#3B82F6) :
- "ℹ Le nom doit contenir entre 1 et 64 caractères."
- "Caractères autorisés : lettres, chiffres, underscore (_)."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Créer | Primaire bleu | Crée la base |

## 8. APIS & ENDPOINTS
- POST /hosting/privateDatabase/{serviceName}/database
  - databaseName: string (1-64 chars, alphanumeric + _)
  - user: string (optionnel)
