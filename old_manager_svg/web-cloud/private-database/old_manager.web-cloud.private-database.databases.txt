# Cahier des Charges : web-cloud/private-database - Bases de données

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des bases de données
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les bases de données du serveur SQL

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Titre service avec onglets
- Bouton "Créer une base de données"
- Tableau des bases
- Menu actions ouvert
- Résumé espace utilisé

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsTabs** : Onglets (Bases de données actif)
- **OdsButton** : Créer une base de données
- **OdsTable** : Liste des bases
- **OdsProgressBar** : Taille par base
- **OdsActionMenu** : Menu ⋮ ouvert

## 4. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | - |
| Utilisateurs et droits | - |
| **Bases de données** | Actif (bleu) |
| IPs autorisées | - |
| Logs | - |
| Configuration | - |

## 5. BOUTON D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Créer une base de données | Primaire bleu | Ouvre modal création |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Nom de la base | Nom cliquable (lien) | production_db |
| Taille | ProgressBar + texte | 1.2 Go / 2 Go (60%) |
| Utilisateurs | Liste users associés | admin_user, app_user |
| Date de création | Format DD/MM/YYYY | 15/03/2024 |
| Actions | Menu ⋮ | Voir ci-dessous |

## 7. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Importer un fichier | Import SQL |
| Exporter (dump) | Export .sql.gz |
| --- | Séparateur |
| Restaurer un dump | Depuis sauvegarde |
| Voir les sauvegardes | Liste dumps |
| --- | Séparateur |
| Supprimer | Rouge - Suppression |

## 8. RÉSUMÉ
"2 base(s) de données • Espace total utilisé: 1.5 Go / 5 Go"

## 9. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/database : Lister bases
- POST /hosting/privateDatabase/{serviceName}/database : Créer base
- DELETE /hosting/privateDatabase/{serviceName}/database/{databaseName} : Supprimer
- POST /hosting/privateDatabase/{serviceName}/database/{databaseName}/dump : Exporter
- POST /hosting/privateDatabase/{serviceName}/database/{databaseName}/import : Importer
