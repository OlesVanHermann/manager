# Cahier des Charges : web-cloud/private-database - Modal Supprimer IP whitelist

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'une IP de la whitelist
- **Type** : Modal confirmation danger
- **Objectif** : Retirer l'autorisation d'accès d'une IP

## 2. STRUCTURE MODAL
- Bande rouge en haut
- Icône ! dans cercle rouge
- Header avec titre et bouton fermer (×)
- Message de confirmation
- Info IP à supprimer (fond gris)
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x280)
- **OdsIcon** : ! (warning)
- **OdsCard** : Info IP (fond gris)
- **OdsButton** : Annuler, Supprimer (danger)

## 4. TITRE
"Supprimer IP de la whitelist"

## 5. ICÔNE
- Cercle fond rose (#FEE2E2)
- Point d'exclamation ! rouge (#DC2626)

## 6. MESSAGE
"Êtes-vous sûr de vouloir supprimer cette IP de la whitelist ?"

## 7. INFO IP
Fond gris (#F3F4F6) :
- Format: "{IP} - {Nom}"
- Exemple: "192.168.1.100 - Serveur production"

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Supprimer | Danger rouge | Supprime l'IP |

## 9. APIS & ENDPOINTS
- DELETE /hosting/privateDatabase/{serviceName}/whitelist/{ip}
