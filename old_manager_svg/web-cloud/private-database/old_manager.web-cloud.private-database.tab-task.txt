# Cahier des Charges : web-cloud/private-database - Tab Tâches

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Tâches
- **Type** : Page liste avec tableau
- **Objectif** : Suivre les tâches en cours et passées

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- Bouton refresh
- Tableau des tâches

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsTabs** : Onglets (Tâches actif)
- **OdsIcon** : Refresh (↻)
- **OdsTable** : Liste des tâches
- **OdsBadge** : Statut

## 4. BOUTON REFRESH
| Bouton | Style | Action |
|--------|-------|--------|
| ↻ | Icône grise | Rafraîchit la liste |

## 5. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Nom de la tâche | Type d'opération | databaseBackup |
| Date de début | Format complet | 23 déc. 2024 10:15:00 |
| Dernière mise à jour | Format complet | 23 déc. 2024 10:16:30 |
| Date de fin | Format complet ou "—" | 23 déc. 2024 10:16:30 |
| Statut | Badge | Terminé, En cours |

## 6. BADGES STATUT
| Statut | Couleur | Code |
|--------|---------|------|
| Terminé | Vert | #DEF7EC / #03543F |
| En cours | Orange | #FEF3C7 / #92400E |
| Erreur | Rouge | #FEE2E2 / #991B1B |

## 7. TYPES DE TÂCHES
| Nom | Description |
|-----|-------------|
| databaseBackup | Sauvegarde de base |
| restart | Redémarrage serveur |
| changeConfiguration | Modification config |
| databaseImport | Import de données |
| databaseRestore | Restauration dump |
| createUser | Création utilisateur |
| deleteUser | Suppression utilisateur |

## 8. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/tasks : Lister tâches
  - Params: status (doing, done, error), function
- GET /hosting/privateDatabase/{serviceName}/tasks/{id} : Détail tâche
