## 9. APIS & ENDPOINTS

```
# Exchange - Groupes de distribution
GET /email/exchange/{organization}/service/{serviceName}/mailingList
→ Liste des groupes

POST /email/exchange/{organization}/service/{serviceName}/mailingList
Body: {
  displayName: "Équipe",
  mailingListAddress: "equipe@example.com",
  senderAuthentification: true
}
→ Crée un groupe

GET /email/exchange/{organization}/service/{serviceName}/mailingList/{name}/member
→ Membres du groupe

POST /email/exchange/{organization}/service/{serviceName}/mailingList/{name}/member/account
Body: { email: "jean@example.com" }
→ Ajoute un membre

DELETE /email/exchange/{organization}/service/{serviceName}/mailingList/{name}
→ Supprime le groupe

# Email Pro - Mailing lists
GET /email/pro/{service}/mailingList
POST /email/pro/{service}/mailingList
DELETE /email/pro/{service}/mailingList/{name}

# MX Plan - Mailing lists
GET /email/domain/{domain}/mailingList
POST /email/domain/{domain}/mailingList
DELETE /email/domain/{domain}/mailingList/{name}

GET /email/domain/{domain}/mailingList/{name}/subscriber
→ Abonnés de la liste

POST /email/domain/{domain}/mailingList/{name}/subscriber
→ Ajoute un abonné
```
