## 7. APIS & ENDPOINTS

### 7.1 Obtenir les comptes éligibles

```
GET /email/exchange/{organizationName}/service/{exchangeService}/account?outlookEligible=true
```

Response:
```json
{
  "eligibleCount": 25,
  "accounts": [
    {
      "primaryEmailAddress": "john.doe@example.com",
      "hasOutlook": false
    }
  ]
}
```

### 7.2 Obtenir les tarifs

```
GET /order/cart/{cartId}/outlookLicense
```

Response:
```json
{
  "prices": [
    {
      "duration": "P1M",
      "price": 2.99,
      "discount": 0
    },
    {
      "duration": "P12M",
      "price": 32.29,
      "discount": 10
    },
    {
      "duration": "P24M",
      "price": 57.42,
      "discount": 20
    }
  ]
}
```

### 7.3 Commander les licences

```
POST /order/cart/{cartId}/outlookLicense
```

Body:
```json
{
  "exchangeService": "exchange-xxxxx",
  "duration": "P12M",
  "quantity": 5
}
```
