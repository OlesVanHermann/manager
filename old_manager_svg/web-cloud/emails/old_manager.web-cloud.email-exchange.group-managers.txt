# Cahier des Charges : web-cloud/email-exchange - Gestionnaires du groupe

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des gestionnaires de groupe
- **Type** : Modal liste avec actions
- **Objectif** : Ajouter/supprimer des gestionnaires de groupe Exchange

## 2. STRUCTURE MODAL
- Header avec email du groupe et bouton fermer (×)
- Bandeau info explicatif
- Bouton ajouter
- Tableau des gestionnaires
- Bouton Fermer en bas

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x390)
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Ajouter gestionnaire (secondaire), Fermer
- **OdsTable** : Liste des gestionnaires
- **OdsIcon** : Bouton supprimer (×) rouge

## 4. TITRE
"Gestionnaires - {email_groupe}"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les gestionnaires peuvent ajouter/supprimer des membres et modifier les paramètres du groupe."

## 6. BOUTON AJOUTER
| Élément | Valeur |
|---------|--------|
| Label | "Ajouter gestionnaire" |
| Type | Secondaire (bordure) |
| Position | Haut droite |

## 7. TABLEAU
| Colonne | Description |
|---------|-------------|
| Gestionnaire | Email du gestionnaire |
| Actions | Bouton supprimer (×) |

## 8. EXEMPLES DE DONNÉES
| Gestionnaire |
|--------------|
| admin@example.com |
| manager@example.com |

## 9. ACTION SUPPRIMER
| Élément | Valeur |
|---------|--------|
| Icône | × (croix) |
| Couleur | Rouge (#DC2626) |
| Action | Supprime le gestionnaire |

## 10. BOUTON FERMER
| Élément | Valeur |
|---------|--------|
| Label | "Fermer" |
| Type | Secondaire (bordure) |
| Position | Bas droite |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/mailingList/{mailingList}/manager : Lister gestionnaires
- POST /email/exchange/{org}/service/{service}/mailingList/{mailingList}/manager : Ajouter gestionnaire
- DELETE /email/exchange/{org}/service/{service}/mailingList/{mailingList}/manager/{id} : Supprimer gestionnaire
