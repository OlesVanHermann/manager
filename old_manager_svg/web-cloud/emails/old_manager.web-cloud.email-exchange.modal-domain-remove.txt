# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer un domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de domaine
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer un domaine de la plateforme Exchange

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Domaine concerné
- Avertissement
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x220)
- **OdsMessage** : Avertissement rouge
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer le domaine"
- Header fond rouge (#FEF2F2)
- Texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer ce domaine de la plateforme Exchange ?" |
| Domaine | example.com (gras) |

## 6. AVERTISSEMENT
Fond rouge (#FEF2F2) :
"Tous les comptes associés à ce domaine seront supprimés. Cette action est irréversible."

## 7. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le domaine |

## 8. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/domain/{domain} : Supprimer domaine
