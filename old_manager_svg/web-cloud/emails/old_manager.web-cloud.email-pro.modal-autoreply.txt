# Cahier des Charges : web-cloud/email-pro - Modal Réponse automatique

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration de la réponse automatique (OOO)
- **Type** : Modal formulaire
- **Objectif** : Configurer un message d'absence automatique

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte (lecture seule)
- Checkbox activation
- Sélection période (dates)
- Zone de texte message
- Option contacts connus
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x460)
- **OdsInput** : Compte (désactivé)
- **OdsCheckbox** : Activation, contacts connus
- **OdsDatepicker** : Dates début/fin
- **OdsTextarea** : Message de réponse
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Réponse automatique"

## 5. COMPTE (LECTURE SEULE)
Champ désactivé avec l'email du compte

## 6. ACTIVATION
| Élément | Description |
|---------|-------------|
| Checkbox | Activer la réponse automatique |
| État | ☑ coché (bleu) |

## 7. PÉRIODE
| Champ | Type | Format |
|-------|------|--------|
| Date début | Datepicker | JJ/MM/AAAA |
| au | Texte | - |
| Date fin | Datepicker | JJ/MM/AAAA |

## 8. MESSAGE
| Élément | Description |
|---------|-------------|
| Label | Message |
| Composant | Textarea |
| Contenu exemple | Bonjour, Je suis absent jusqu'au... |

## 9. OPTIONS
| Option | Description |
|--------|-------------|
| Checkbox | Répondre uniquement aux contacts connus |

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Enregistrer | Primaire bleu | Sauvegarde la configuration |

## 11. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email}/autoReply : Récupérer config
- PUT /email/pro/{service}/account/{email}/autoReply : Configurer
  - enabled: boolean
  - from: date
  - to: date
  - message: string
  - contactOnly: boolean
