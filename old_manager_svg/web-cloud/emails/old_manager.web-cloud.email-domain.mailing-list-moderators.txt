# Cahier des Charges : web-cloud/email-domain - Modérateurs liste

## 1. VUE D'ENSEMBLE
- **Type** : Page liste modérateurs
- **URL** : /email/domain/{domain}/mailingList/{name}/moderator
- **Objectif** : Gérer les modérateurs d'une mailing list

## 2. STRUCTURE PAGE
- Bouton retour
- Titre avec nom liste et compteur
- Bouton suppression sélection
- Barre recherche
- Tableau modérateurs
- Sidebar actions

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre
- **OdsButton** : Supprimer sélectionnés
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste modérateurs
- **OdsCheckbox** : Sélection multiple

## 4. TABLEAU MODÉRATEURS
| Colonne | Type |
|---------|------|
| Checkbox | Sélection |
| Email du modérateur | Texte |
| Refresh | Icône action |

## 5. SIDEBAR
| Bouton | Action |
|--------|--------|
| Ajouter | Ouvrir modal ajout |
| Supprimer (fichier) | Import fichier suppression |

## 6. PAGINATION
| Élément | Format |
|---------|--------|
| Page | "Page X sur Y" |

## 7. APIS & ENDPOINTS
- GET /email/domain/{domain}/mailingList/{name}/moderator - Liste modérateurs
- POST /email/domain/{domain}/mailingList/{name}/moderator - Ajouter
- DELETE /email/domain/{domain}/mailingList/{name}/moderator/{email} - Supprimer
