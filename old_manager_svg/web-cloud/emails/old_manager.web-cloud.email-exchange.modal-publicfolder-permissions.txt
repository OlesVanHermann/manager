# Cahier des Charges : web-cloud/email-exchange - Modal Permissions dossier public

## 1. VUE D'ENSEMBLE
- **Modal** : Gérer les permissions d'un dossier public
- **Type** : Modal liste éditable
- **Objectif** : Définir qui peut accéder au dossier et avec quels droits

## 2. STRUCTURE MODAL
- Header avec titre du dossier et bouton fermer (×)
- Bouton Ajouter permission
- Tableau des permissions
- Légende des niveaux
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x440)
- **OdsButton** : Ajouter permission
- **OdsTable** : Liste des permissions
- **OdsSelect** : Niveau de permission (dans chaque ligne)
- **OdsButton** : Supprimer (×) par ligne
- **OdsText** : Légende

## 4. TITRE
"Permissions - {nom_dossier}"
Exemple: "Permissions - Documents partagés"

## 5. BOUTON D'ACTION
| Bouton | Position | Action |
|--------|----------|--------|
| Ajouter permission | En haut à droite | Ouvre modal d'ajout |

## 6. TABLEAU DES PERMISSIONS
| Colonne | Description |
|---------|-------------|
| Utilisateur | Email ou groupe |
| Permission | Select avec niveau |
| - | Bouton supprimer (×) |

## 7. NIVEAUX DE PERMISSION
| Niveau | Code | Description |
|--------|------|-------------|
| Lecteur | Lecteur | Lecture seule |
| Éditeur | Éditeur | Lecture + Écriture |
| Propriétaire | Propriétaire | Contrôle total |

## 8. EXEMPLES DE DONNÉES
| Utilisateur | Permission |
|-------------|------------|
| Tous les utilisateurs | Lecteur |
| admin@example.com | Propriétaire |
| marketing@example.com | Éditeur |

## 9. LÉGENDE
"Lecteur: Lecture seule | Éditeur: Lecture + Écriture | Propriétaire: Contrôle total"

## 10. BOUTON
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/publicFolder/{path}/permission : Lister permissions
- POST /email/exchange/{org}/service/{service}/publicFolder/{path}/permission : Ajouter permission
- PUT /email/exchange/{org}/service/{service}/publicFolder/{path}/permission/{id} : Modifier permission
- DELETE /email/exchange/{org}/service/{service}/publicFolder/{path}/permission/{id} : Supprimer permission
