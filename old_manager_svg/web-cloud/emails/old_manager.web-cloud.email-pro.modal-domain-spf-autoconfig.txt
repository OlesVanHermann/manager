# Cahier des Charges : web-cloud/email-pro - Modal Configuration SPF

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique de l'enregistrement SPF
- **Type** : Modal action avec erreur
- **Objectif** : Configurer SPF pour l'authentification des emails

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Bandeau erreur rouge
- Domaine concerné
- Configuration requise
- Boutons Fermer / Configurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x320)
- **OdsMessage** : Alerte erreur rouge
- **OdsText** : Domaine, configuration
- **OdsButton** : Fermer, Configurer

## 4. TITRE
"Configuration SPF"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. BANDEAU ERREUR
Fond rouge (#FEF2F2), texte rouge (#DC2626) :
"⚠ L'enregistrement SPF n'est pas configuré."

## 6. INFORMATIONS
| Élément | Valeur |
|---------|--------|
| Domaine concerné | example.com |

## 7. CONFIGURATION REQUISE
| Champ | Valeur |
|-------|--------|
| Type | TXT |
| Valeur | v=spf1 include:mx.ovh.com ~all |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |
| Configurer | Primaire bleu | Configure SPF automatiquement |

## 9. APIS & ENDPOINTS
- GET /email/pro/{service}/domain/{domain}/diagnostic : Récupérer diagnostic
- POST /domain/{domain}/zone/record : Créer enregistrement TXT SPF
