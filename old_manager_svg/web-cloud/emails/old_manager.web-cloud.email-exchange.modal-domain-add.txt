# Cahier des Charges : web-cloud/email-exchange - Modal Ajouter un domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un domaine au service Exchange
- **Type** : Formulaire de création
- **Objectif** : Associer un nouveau domaine à la plateforme Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Type de domaine (radio)
- Sélection du domaine
- Options de configuration DNS
- Avertissement propagation
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsRadio** : Type de domaine
- **OdsSelect** : Sélection domaine
- **OdsCheckbox** : Configuration MX/SRV
- **OdsMessage** : Warning propagation
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un domaine"

## 5. TYPE DE DOMAINE
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Domaine OVHcloud | ● (par défaut) | Domaine géré par OVHcloud |
| Domaine externe (non-OVHcloud) | ○ | Domaine externe |

## 6. SÉLECTION DOMAINE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Sélectionner un domaine | Select dropdown | Oui (*) |

## 7. CONFIGURATION MX
☑ "Configurer automatiquement les enregistrements MX"

## 8. CONFIGURATION SRV
☑ "Configurer automatiquement les enregistrements SRV"

## 9. AVERTISSEMENT
Fond jaune (#FEF3C7) :
"La configuration DNS peut prendre jusqu'à 24h pour se propager."

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Ajouter | Primaire bleu | Ajoute le domaine |

## 11. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/domain : Ajouter domaine
- GET /domain : Lister domaines OVHcloud disponibles
