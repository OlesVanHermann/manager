# Cahier des Charges : web-cloud/email-exchange - Modal Modifier un domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Modification des paramètres d'un domaine
- **Type** : Formulaire de modification
- **Objectif** : Modifier le type et les options d'un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine (lecture seule)
- Type de domaine (radio)
- Option domaine principal
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsInput** : Domaine (disabled)
- **OdsRadio** : Type de domaine
- **OdsCheckbox** : Domaine principal
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le domaine"

## 5. DOMAINE (lecture seule)
Affiché en champ grisé : "example.com"

## 6. TYPE DE DOMAINE
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Autoritatif | ● (par défaut) | Tous les e-mails sont gérés par Exchange |
| Non-autoritatif | ○ | Les e-mails non trouvés sont relayés vers un autre serveur |

## 7. OPTION DOMAINE PRINCIPAL
☑ "Définir comme domaine principal"

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Modifier | Primaire bleu | Enregistre les modifications |

## 9. APIS & ENDPOINTS
- PUT /email/exchange/{org}/service/{service}/domain/{domain} : Modifier domaine
