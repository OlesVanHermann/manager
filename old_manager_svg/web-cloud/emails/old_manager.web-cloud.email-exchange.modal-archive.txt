# Cahier des Charges : web-cloud/email-exchange - Modal Archivage

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration de l'archivage email
- **Type** : Formulaire de paramètres
- **Objectif** : Activer et configurer l'archivage pour un compte

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (lecture seule)
- Statut archivage
- Options d'archivage
- Boutons Annuler / Activer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsInput** : Compte (disabled)
- **OdsBadge** : Statut archivage
- **OdsCheckbox** : Activer archivage automatique
- **OdsSelect** : Quota archive
- **OdsButton** : Annuler, Activer

## 4. TITRE
"Configurer l'archivage"

## 5. CHAMPS
| Champ | Type | Valeur |
|-------|------|--------|
| Compte | Input disabled | john.doe@example.com |
| Statut archivage | Badge | "Désactivé" (rouge) |

## 6. BADGE STATUT ARCHIVAGE
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Désactivé | #FEE2E2 (rouge clair) | #DC2626 |
| Activé | #DCFCE7 (vert clair) | #166534 |

## 7. OPTIONS D'ARCHIVAGE
| Option | Type | Description |
|--------|------|-------------|
| Activer l'archivage automatique | Checkbox | Active l'archive sur le compte |
| Quota archive | Select dropdown | Taille de l'archive (50 Go, 100 Go, etc.) |

## 8. QUOTAS DISPONIBLES
| Quota | Description |
|-------|-------------|
| 50 Go | Standard |
| 100 Go | Étendu |
| 200 Go | Maximum |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Activer | Primaire bleu | Active l'archivage |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/archive : Récupérer statut
- POST /email/exchange/{org}/service/{service}/account/{email}/archive : Activer archive
- PUT /email/exchange/{org}/service/{service}/account/{email}/archive : Modifier quota
- DELETE /email/exchange/{org}/service/{service}/account/{email}/archive : Désactiver
