# Cahier des Charges : web-cloud/web-office - Modal Mot de passe

## 1. VUE D'ENSEMBLE
- **Page** : Modifier le mot de passe
- **Type** : Modal formulaire
- **Objectif** : Changer le mot de passe d'un utilisateur

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Info utilisateur (readonly)
- Options radio (aléatoire/personnalisé)
- Email réception
- Champs mot de passe (conditionnels)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Email, mot de passe
- **OdsRadio** : Type génération
- **OdsMessage** : Info email
- **OdsButton** : Annuler, Confirmer

## 4. INFO UTILISATEUR
| Champ | Valeur |
|-------|--------|
| Fond | Gris |
| Texte | Utilisateur: jean.dupont@monentreprise.onmicrosoft.com |

## 5. OPTIONS RADIO
| Option | Sélection | Description |
|--------|-----------|-------------|
| Générer aléatoire | ● Sélectionné | Mot de passe auto |
| Personnaliser | ○ | Saisir manuellement |

## 6. CHAMPS FORMULAIRE
| Champ | Type | Requis | Condition |
|-------|------|--------|-----------|
| Email réception | Input email | Oui | Toujours |
| Nouveau mot de passe | Password | Conditionnel | Si personnalisé |
| Confirmation | Password | Conditionnel | Si personnalisé |

## 7. RÈGLES MOT DE PASSE
- Minimum 8 caractères
- Au moins une majuscule et une minuscule
- Ne doit pas contenir l'identifiant

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Confirmer | Primary | Droite |

## 9. APIS & ENDPOINTS
- POST /license/office/{service}/user/{id}/changePassword - Changer MDP
