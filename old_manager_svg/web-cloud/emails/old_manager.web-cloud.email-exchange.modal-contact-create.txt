# Cahier des Charges : web-cloud/email-exchange - Modal Créer un contact

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un contact externe
- **Type** : Formulaire de création
- **Objectif** : Ajouter un contact externe à l'annuaire Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire avec informations contact
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x460)
- **OdsInput** : Champs texte
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un contact"

## 5. FORMULAIRE
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Prénom | Input text | Oui (*) | Jean |
| Nom | Input text | Oui (*) | Dupont |
| Adresse email | Input email | Oui (*) | jean.dupont@example.com |
| Société | Input text | Non | Acme Corporation |
| Téléphone | Input tel | Non | +33 1 23 45 67 89 |
| Mobile | Input tel | Non | +33 6 12 34 56 78 |
| Notes | Input text | Non | Contact principal pour le projet X |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le contact |

## 8. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/externalContact : Créer contact
- GET /email/exchange/{org}/service/{service}/externalContact : Lister contacts
