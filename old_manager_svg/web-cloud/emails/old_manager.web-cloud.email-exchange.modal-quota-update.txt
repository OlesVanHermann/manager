# Cahier des Charges : web-cloud/email-exchange - Modal Modifier le quota

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le quota d'un compte Exchange
- **Type** : Formulaire de modification
- **Objectif** : Augmenter ou réduire l'espace de stockage d'un compte

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (readonly)
- Utilisation actuelle (barre de progression)
- Nouveau quota
- Avertissement
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsInput** : Compte (readonly)
- **OdsProgressBar** : Utilisation actuelle
- **OdsText** : Détail utilisation
- **OdsInput** : Nouveau quota (number)
- **OdsText** : Avertissement, limite max
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le quota"

## 5. COMPTE CONCERNÉ
| Champ | État | Valeur exemple |
|-------|------|----------------|
| Compte | Readonly grisé | john.doe@example.com |

## 6. UTILISATION ACTUELLE
| Élément | Valeur |
|---------|--------|
| Barre de progression | 50% |
| Texte | 25 Go utilisés sur 50 Go (50%) |

## 7. NOUVEAU QUOTA
| Champ | Type | Valeur | Limite |
|-------|------|--------|--------|
| Nouveau quota (Go) | Input number | 100 | Max: 300 Go |

## 8. AVERTISSEMENT
"L'augmentation du quota peut entraîner des frais supplémentaires."

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Modifier | Primaire bleu | Applique le nouveau quota |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email} : Récupérer quota actuel
- PUT /email/exchange/{org}/service/{service}/account/{email} : Modifier quota
