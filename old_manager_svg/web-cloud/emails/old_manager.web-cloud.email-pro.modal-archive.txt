# Cahier des Charges : web-cloud/email-pro - Modal Archivage

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration de l'archivage automatique
- **Type** : Modal formulaire
- **Objectif** : Activer et configurer l'archivage des emails

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte (lecture seule)
- Checkbox activation
- Sélection rétention
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x290)
- **OdsInput** : Compte (désactivé)
- **OdsCheckbox** : Activer l'archivage
- **OdsSelect** : Durée de rétention
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Configurer l'archivage"

## 5. COMPTE (LECTURE SEULE)
Champ désactivé avec l'email du compte

## 6. ACTIVATION
| Élément | Description |
|---------|-------------|
| Checkbox | Activer l'archivage automatique |
| État | ☑ coché (bleu) |

## 7. RÉTENTION
| Label | Composant | Valeurs |
|-------|-----------|---------|
| Archiver les emails de plus de | OdsSelect | 6 mois, 12 mois, 24 mois |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Enregistrer | Primaire bleu | Active l'archivage |

## 9. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email}/archive : Récupérer config
- PUT /email/pro/{service}/account/{email}/archive : Configurer archivage
  - enabled: boolean
  - retention: number (mois)
