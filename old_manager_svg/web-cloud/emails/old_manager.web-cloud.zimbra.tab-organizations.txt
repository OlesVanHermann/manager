# Cahier des Charges : web-cloud/zimbra - Organisations

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des organisations Zimbra.

### Objectif fonctionnel
- Voir les organisations
- Creer de nouvelles organisations
- Gerer les labels

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/organization
POST /zimbra/platform/{platformId}/organization
DELETE /zimbra/platform/{platformId}/organization/{orgId}
PUT /zimbra/platform/{platformId}/organization/{orgId}
```

### Modele
```typescript
interface ZimbraOrganization {
  id: string;
  name: string;
  label: 'production' | 'development' | 'test';
  accountCount: number;
  state: 'ok' | 'suspended';
}
```
