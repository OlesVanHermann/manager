# Cahier des Charges : web-cloud/web-office - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Mes comptes Microsoft 365
- **Type** : Page dashboard avec onglets
- **Objectif** : Gérer un service Microsoft 365

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre
- Carte info service
- Onglets (Utilisateurs, Licences, Consommation)
- Bouton ajouter utilisateur
- Datagrid des utilisateurs
- Bandeau info installation

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsCard** : Info service
- **OdsTabs** : Utilisateurs, Licences, Consommation
- **OdsButton** : Ajouter un utilisateur
- **OdsDatagrid** : Liste utilisateurs
- **OdsBadge** : Status
- **OdsMessage** : Info installation

## 4. CARTE INFO SERVICE
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| ID | Text bold | office365-abc123 |
| Nom du groupe | Text | Mon entreprise - Microsoft 365 |
| Prochaine échéance | Text | 15/02/2025 |
| Type | Text | Paiement à l'utilisation |
| Contacts | Link | Gérer les contacts |

## 5. ONGLETS
| Onglet | Description |
|--------|-------------|
| Utilisateurs | Liste des utilisateurs |
| Licences | Gestion des licences |
| Consommation | Facturation |

## 6. COLONNES DATAGRID UTILISATEURS
| Colonne | Type | Description |
|---------|------|-------------|
| Email d'activation | Text | Email utilisateur |
| Licences | Text | Type licence |
| Statut | Badge | Créé, En création |
| Actions | ActionMenu | ⋮ |

## 7. BADGES STATUT
| Badge | Couleur | Description |
|-------|---------|-------------|
| Créé | Vert (success) | Utilisateur actif |
| En création | Orange (warning) | En cours |

## 8. MESSAGE INFO
- Installation PC/MAC sur site Microsoft
- Smartphones via App Store

## 9. APIS & ENDPOINTS
- GET /license/office/{service} - Détails service
- GET /license/office/{service}/user - Liste utilisateurs
