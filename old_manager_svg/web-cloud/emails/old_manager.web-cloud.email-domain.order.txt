# Cahier des Charges : web-cloud/email-domain - Commander MX Plan

## 1. VUE D'ENSEMBLE
- **Type** : Wizard commande MX Plan
- **URL** : /email/order
- **Objectif** : Commander une offre email MX Plan

## 2. STRUCTURE PAGE
- Header OVHcloud
- Titre page
- Stepper 3 étapes
- Sélection domaine
- Cartes offres
- Bouton navigation

## 3. COMPOSANTS UI
