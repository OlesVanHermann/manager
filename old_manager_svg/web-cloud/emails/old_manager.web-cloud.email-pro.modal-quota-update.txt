# Cahier des Charges : web-cloud/email-pro - Modal Modifier le quota

## 1. VUE D'ENSEMBLE
- **Modal** : Modification du quota d'un compte
- **Type** : Modal formulaire
- **Objectif** : Ajuster l'espace disque alloué à un compte

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte (lecture seule)
- Utilisation actuelle (progress bar)
- Nouveau quota
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsInput** : Compte (désactivé), quota
- **OdsProgressBar** : Utilisation actuelle
- **OdsText** : Informations quota
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le quota"

## 5. COMPTE
Champ désactivé avec l'email du compte

## 6. UTILISATION ACTUELLE
| Élément | Description |
|---------|-------------|
| Progress bar | Barre bleue de progression |
| Texte | X Go utilisés sur Y Go (Z%) |

## 7. NOUVEAU QUOTA
| Élément | Description |
|---------|-------------|
| Input | Nombre en Go |
| Info | Max: 50 Go |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Modifier | Primaire bleu | Applique le nouveau quota |

## 9. VALIDATION
| Règle | Description |
|-------|-------------|
| Min | Supérieur à l'utilisation actuelle |
| Max | 50 Go (selon offre) |

## 10. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email} : Récupérer quota actuel
- PUT /email/pro/{service}/account/{email} : Modifier quota
  - quota: number (en octets ou Go)
