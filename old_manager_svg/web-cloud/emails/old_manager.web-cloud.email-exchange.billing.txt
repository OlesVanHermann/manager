# Cahier des Charges : web-cloud/email-exchange - Facturation

## 1. VUE D'ENSEMBLE
- **Écran** : Page facturation Exchange
- **Type** : Dashboard avec cartes et tableau
- **Objectif** : Visualiser l'abonnement et l'historique de facturation

## 2. LAYOUT
- Header OVHcloud
- Sidebar Exchange (Facturation actif)
- Titre
- 2 cartes (Abonnement, Coût mensuel)
- Historique des factures (tableau)

## 3. COMPOSANTS UI
- **OdsCard** : Cartes informations
- **OdsBadge** : Statuts (Automatique, Payée)
- **OdsTable** : Historique factures
- **OdsLink** : Liens téléchargement

## 4. TITRE
"Facturation"
