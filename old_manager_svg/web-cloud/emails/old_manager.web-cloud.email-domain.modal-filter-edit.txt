# Cahier des Charges : web-cloud/email-domain - Modal Modifier filtre

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification filtre email
- **URL** : N/A (composant modal)
- **Objectif** : Modifier une règle de filtrage existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 560px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Valeur règle
- **OdsSelect** : Priorité, en-tête, opérateur, action
- **OdsCheckbox** : Activer filtre
- **OdsButton** : Actions, ajout/suppression règles
- **OdsMessage** : Info priorité

## 4. SECTION INFORMATIONS
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Priorité | Input nombre | 2 |
| Activer le filtre | Checkbox | Coché |

## 5. SECTION RÈGLES
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| En-tête | Select | From |
| Opérateur | Select | contient |
| Valeur | Input | spam@ |
| Actions | Boutons +/- | Ajouter/supprimer |

## 6. SECTION ACTION
| Champ | Type | Valeur |
|-------|------|--------|
| Action | Select | Supprimer |

## 7. MESSAGE INFO
| Type | Texte |
|------|-------|
| Info | "La priorité détermine l'ordre d'exécution des filtres (1 = haute priorité)" |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer modifications |

## 9. APIS & ENDPOINTS
- PUT /email/domain/{domain}/account/{account}/filter/{name} - Modifier filtre
