# Cahier des Charges : web-cloud/email-domain - Délégations

## 1. VUE D'ENSEMBLE
- **Type** : Page liste comptes email délégués
- **URL** : /email/domain/{domain}/email
- **Objectif** : Gérer comptes email du domaine

## 2. STRUCTURE PAGE
- Titre domaine + changelog
- Barre de recherche
- Tableau comptes
- Sidebar actions

## 3. COMPOSANTS UI
- **OdsText** : Titre domaine
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste comptes
- **OdsBadge** : Statuts
- **OdsProgressBar** : Taille utilisée
- **OdsActionMenu** : Actions par compte

## 4. TABLEAU COMPTES
| Colonne | Type |
|---------|------|
| Nom du compte | Texte |
| Description | Texte |
| Taille | ProgressBar + texte |
| Bloqué | Badge (Actif/Bloqué) |
| Filtres | Icône engrenage |
| Actions | ActionMenu |

## 5. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Actif | Vert #DEF7EC |
| Bloqué | Rouge #FEE2E2 |

## 6. MENU ACTIONS
| Option | Action |
|--------|--------|
| Modifier le compte | Ouvrir modal edit |
| Changer le mot de passe | Ouvrir modal password |
| Débloquer le compte | Débloquer (grisé si actif) |

## 7. SIDEBAR
| Bouton | Action |
|--------|--------|
| Gérer les répondeurs | Navigation |
| Exporter CSV | Téléchargement |

## 8. APIS & ENDPOINTS
- GET /email/domain/{domain}/account - Liste comptes
- PUT /email/domain/{domain}/account/{account} - Modifier
- POST /email/domain/{domain}/account/{account}/changePassword - Password
