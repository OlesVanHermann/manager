# Cahier des Charges : web-cloud/email-exchange - Modal Réponse automatique

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration réponse automatique (out of office)
- **Type** : Formulaire de paramètres
- **Objectif** : Configurer un message d'absence automatique

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (lecture seule)
- Activation
- Période (dates début/fin)
- Message
- Options
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x490)
- **OdsInput** : Compte (disabled)
- **OdsCheckbox** : Activation, options
- **OdsDatepicker** : Dates début et fin
- **OdsTextarea** : Message
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Réponse automatique"

## 5. CHAMPS
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Compte | Input disabled | - |
| Activer | Checkbox | - |
| Date début | Datepicker | Oui |
| Date fin | Datepicker | Oui |
| Message | Textarea | Oui |
| Contacts internes | Checkbox | - |

## 6. ACTIVATION
☑ "Activer la réponse automatique"

## 7. PÉRIODE
| Champ | Format | Exemple |
|-------|--------|---------|
| Début | DD/MM/YYYY | 23/12/2024 |
| Fin | DD/MM/YYYY | 02/01/2025 |

## 8. MESSAGE
Zone de texte multilignes avec exemple :
```
Bonjour,
Je suis absent du bureau jusqu'au 2 janvier.
Je répondrai à votre message dès mon retour.
Cordialement
```

## 9. OPTIONS
☐ "Répondre uniquement aux contacts internes"

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde la configuration |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/autoReply : Récupérer config
- PUT /email/exchange/{org}/service/{service}/account/{email}/autoReply : Modifier config
