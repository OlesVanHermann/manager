# Cahier des Charges : web-cloud/email-mxplan - Modal Supprimer compte

## 1. VUE D'ENSEMBLE
- **Page** : Supprimer le compte
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un compte email

## 2. STRUCTURE MODAL
- Header rouge avec titre + bouton fermer
- Bandeau warning (irréversible)
- Info compte à supprimer
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Warning (orange)
- **OdsText** : Info compte
- **OdsButton** : Annuler, Supprimer (danger)

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | Rouge (#FEE2E2) |
| Titre | Supprimer le compte (rouge) |

## 5. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠ |
| Texte ligne 1 | Cette action est irréversible |
| Texte ligne 2 | Tous les emails seront supprimés |

## 6. INFO COMPTE
| Élément | Description |
|---------|-------------|
| Label | Compte à supprimer : |
| Valeur | user@domain.com (bold) |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Supprimer | Danger (rouge) | Droite |

## 8. APIS & ENDPOINTS
- DELETE /email/mxplan/{service}/account/{email} - Supprimer compte
