# Cahier des Charges : web-cloud/email-domain - Modal Actualiser quota

## 1. VUE D'ENSEMBLE
- **Type** : Modal actualisation quota
- **URL** : N/A (composant modal)
- **Objectif** : Demander actualisation utilisation quota

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Messages
- **OdsMessage** : Warning si déjà en cours
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Actualiser le quota" |
| Message | "Demander une actualisation de l'utilisation du quota pour ce compte email." |
| Dernière MAJ | Date et heure formatées |

## 5. AVERTISSEMENT
| Condition | Message |
|-----------|---------|
| Actualisation en cours | "Une actualisation est déjà en cours." (jaune) |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Actualiser | Primary | Lancer actualisation |

## 7. APIS & ENDPOINTS
- POST /email/domain/{domain}/account/{account}/usage/calculate - Actualiser quota
