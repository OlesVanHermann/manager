# Cahier des Charges : web-cloud/email-exchange - Dashboard

## 1. RESUME EXECUTIF

### Perimetre
Dashboard du service Microsoft Exchange avec vue d'ensemble.

### Objectif fonctionnel
- Voir les informations generales du service
- Consulter la configuration Exchange
- Verifier les parametres de securite
- Acceder aux differentes sections via onglets

---

## 2. LAYOUT

### Structure page avec header + onglets + 3 cartes

## 3. COMPOSANTS UI

### Header service
| Element | Description |
|---------|-------------|
| Badge | Exchange (bleu) |
| Titre | Nom du service (exchange-abc123) |
| Sous-titre | Date expiration + domaine principal |
| Bouton Actions | Dropdown menu |

### Onglets navigation
| Onglet | Description |
|--------|-------------|
| Informations generales | Vue d'ensemble (actif) |
| Comptes email | Liste des comptes |
| Groupes | Groupes de distribution |
| Boites partagees | Shared mailboxes |
| Contacts | Contacts partages |
| Ressources | Salles, equipements |
| Taches | Historique operations |

## 4. APIS & DONNEES

### Endpoints
```
GET /email/exchange/{organization}/{service}
GET /email/exchange/{organization}/{service}/domain
GET /email/exchange/{organization}/{service}/account
GET /email/exchange/{organization}/{service}/sharedAccount
GET /email/exchange/{organization}/{service}/mailingList
```

### Modele
```typescript
interface ExchangeService {
  displayName: string;
  domain: string;
  offer: 'hosted' | 'provider' | 'dedicated';
  maxAccounts: number;
  currentAccounts: number;
  quota: {
    used: number;
    total: number;
  };
  expiration: string;
  state: 'ok' | 'inMaintenance' | 'suspended';
}

interface ExchangeConfiguration {
  autodiscoverUrl: string;
  owaUrl: string;
  ewsUrl: string;
}

interface ExchangeSecurity {
  antispamEnabled: boolean;
  antivirusEnabled: boolean;
  dkimEnabled: boolean;
}
```
