# Cahier des Charges : web-cloud/email-exchange - Modal Créer un groupe

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un groupe de distribution
- **Type** : Formulaire de création
- **Objectif** : Créer un nouveau groupe Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire avec nom, email, gestionnaire
- Option visibilité
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsInput** : Nom, nom affiché
- **OdsSelect** : Domaine, gestionnaire
- **OdsCheckbox** : Cacher dans l'annuaire
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un groupe"

## 5. CHAMPS
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Nom du groupe | Input text | Oui (*) | Ex: Équipe Marketing |
| Adresse e-mail | Input + Select | Oui (*) | marketing@example.com |
| Nom affiché | Input text | Non | Marketing Team |
| Gestionnaire du groupe | Select dropdown | Oui (*) | admin@example.com |

## 6. CHAMP ADRESSE EMAIL
Composé de :
- Input texte (placeholder: "marketing")
- Séparateur "@"
- Select dropdown pour le domaine

## 7. OPTIONS
☐ "Cacher dans l'annuaire"

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le groupe |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/mailingList : Créer groupe
- GET /email/exchange/{org}/service/{service}/account : Lister comptes pour gestionnaire
