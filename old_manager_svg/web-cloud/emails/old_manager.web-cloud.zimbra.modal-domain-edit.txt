# Cahier des Charges : web-cloud/zimbra - Modal edition domaine

## 1. RESUME EXECUTIF

### Perimetre
Modal de modification de la configuration d'un domaine.

### Objectif fonctionnel
- Changer l'organisation associee au domaine
- Mettre a jour la configuration

---

## 2. LAYOUT

### Structure modale avec header + formulaire + footer

---

## 3. COMPOSANTS UI

### Modal header
| Element | Description |
|---------|-------------|
| Titre | Configuration du domaine |
| Bouton fermer | X |

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Domaine * | Input disabled | Non modifiable |
| Organisation * | Select | Changer d'organisation |

### Footer
| Element | Description |
|---------|-------------|
| Bouton annuler | Secondary |
| Bouton confirmer | Primary |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/organization
PUT /zimbra/platform/{platformId}/domain/{domainId}
```

### Modele
```typescript
interface UpdateDomain {
  organization: string;
}
```

- Nom de domaine non modifiable
- Organisation modifiable
- Changement d'organisation impacte les comptes associes
