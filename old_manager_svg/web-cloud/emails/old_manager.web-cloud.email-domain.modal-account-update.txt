# Cahier des Charges : web-cloud/email-domain - Modal Modifier compte

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification compte email
- **URL** : N/A (composant modal)
- **Objectif** : Modifier paramètres d'un compte email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Labels
- **OdsInput** : Description
- **OdsSelect** : Taille boîte
- **OdsButton** : Actions

## 4. EN-TÊTE
| Élément | Valeur |
|---------|--------|
| Titre | "Modifier le compte" |
| Compte | contact@example.com |
| Sous-titre | "Modifiez les paramètres du compte email" |

## 5. FORMULAIRE
| Champ | Type | Contraintes |
|-------|------|-------------|
| Description | Input texte | 50 caractères max |
| Taille de la boîte | Select dropdown | Options : 1 Go, 5 Go, 10 Go |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer modifications |

## 7. APIS & ENDPOINTS
- PUT /email/domain/{domain}/account/{account} - Modifier compte
