# Cahier des Charges : web-cloud/email-exchange - Modal Diagnostic DKIM

## 1. VUE D'ENSEMBLE
- **Modal** : Résultat diagnostic DKIM
- **Type** : Modal informative avec action
- **Objectif** : Afficher le statut DKIM et permettre son activation

## 2. STRUCTURE MODAL
- Header jaune (warning)
- Domaine concerné
- Message d'avertissement
- Sélecteur DKIM disponible
- Boutons Fermer / Activer DKIM

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x320)
- **OdsMessage** : Avertissement jaune
- **OdsInput** : Sélecteur DKIM (readonly)
- **OdsButton** : Fermer, Activer DKIM

## 4. TITRE
"DKIM non activé"
- Header fond jaune (#FEF3C7)
- Texte orange (#92400E)

## 5. DOMAINE
Affiché en gras : "example.com"

## 6. MESSAGE D'AVERTISSEMENT
Fond jaune (#FEF3C7) avec icône :
| Élément | Valeur |
|---------|--------|
| Icône | ! (cercle jaune) |
| Titre | "Signature DKIM désactivée" |
| Message | "DKIM permet de signer cryptographiquement vos emails et améliore la délivrabilité." |

## 7. SÉLECTEUR DKIM
| Champ | Valeur exemple |
|-------|----------------|
| Label | "Sélecteur DKIM disponible" |
| Valeur | ovhex1234-selector1._domainkey.example.com |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |
| Activer DKIM | Primaire bleu | Active la signature DKIM |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/domain/{domain}/dkim : Récupérer statut DKIM
- POST /email/exchange/{org}/service/{service}/domain/{domain}/dkim : Activer DKIM
