# Cahier des Charges : web-cloud/email-exchange - Modal Créer un connecteur

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un connecteur SMTP
- **Type** : Formulaire de création
- **Objectif** : Créer un connecteur pour router les emails

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau info explicatif
- Formulaire de configuration
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x460)
- **OdsMessage** : Bandeau info bleu
- **OdsInput** : Nom, serveur, port
- **OdsRadio** : Direction
- **OdsCheckbox** : TLS
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un connecteur"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les connecteurs permettent d'acheminer les emails vers des serveurs externes ou partenaires."

## 6. FORMULAIRE
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Nom du connecteur | Input text | Oui (*) | Partenaire externe |
| Direction | Radio | Oui | Sortant / Entrant |
| Serveur de destination | Input text | Oui (*) | smtp.partner.com |
| Port | Input number | Non | 25 |
| Forcer TLS | Checkbox | Non | ☑ |

## 7. DIRECTION
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Sortant (vers serveur externe) | ● (par défaut) | Emails envoyés vers l'extérieur |
| Entrant (depuis serveur externe) | ○ | Emails reçus depuis l'extérieur |

## 8. OPTIONS TLS
☑ "Forcer TLS" - Force le chiffrement de la connexion

## 9. PORTS STANDARDS
| Port | Usage |
|------|-------|
| 25 | SMTP standard |
| 465 | SMTPS (SSL) |
| 587 | Submission (STARTTLS) |

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le connecteur |

## 11. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/connector : Créer connecteur
- GET /email/exchange/{org}/service/{service}/connector : Lister connecteurs
