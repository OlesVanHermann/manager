# Cahier des Charges : web-cloud/email-exchange - Partages (Dossiers publics)

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des dossiers partagés Exchange
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les dossiers publics et leurs permissions

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Exchange (menu)
- Titre "Partages"
- Bouton ajout
- Barre de recherche
- Tableau des dossiers
- Pagination

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Exchange
- **OdsButton** : Ajouter un dossier partagé
- **OdsInput** : Recherche par chemin
- **OdsTable** : Liste des dossiers
- **OdsBadge** : Accès (nombre), Statut
- **OdsActionMenu** : Menu actions (⚙)
- **OdsText** : Pagination

## 4. TITRE
"Partages"

## 5. BOUTON D'ACTION PRINCIPAL
| Bouton | Position | Action |
|--------|----------|--------|
| Ajouter un dossier partagé | En haut à droite | Ouvre modal création |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Chemin | Chemin du dossier | /Documents communs |
| Type | Type de contenu | Courrier, Calendrier, Contact, Note |
| Quota | Taille allouée | 500 Mo |
| Accès | Badge nombre d'utilisateurs | 12 (rouge) |
| Statut | État | OK, Création, En cours |
| Actions | Menu ⚙ | Modifier, Permissions, Supprimer |

## 7. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| OK | Vert (#10B981) | Actif |
| Création | Bleu (#3B82F6) | En cours de création |
| En cours | Icône ⏳ | Tâche en traitement |

## 8. BADGE ACCÈS
| Élément | Couleur | Description |
|---------|---------|-------------|
| Nombre | Rouge (#EF4444) | Nombre d'utilisateurs avec accès |

## 9. MENU ACTIONS (⚙)
| Action | Description |
|--------|-------------|
| Modifier | Ouvre modal modification |
| Gérer les permissions | Ouvre modal permissions |
| Supprimer | Ouvre modal suppression (rouge) |

## 12. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/publicFolder : Lister dossiers
- POST /email/exchange/{org}/service/{service}/publicFolder : Créer
- PUT /email/exchange/{org}/service/{service}/publicFolder/{path} : Modifier
- DELETE /email/exchange/{org}/service/{service}/publicFolder/{path} : Supprimer
