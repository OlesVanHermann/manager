# Cahier des Charges : web-cloud/email - Modal Add Account

## 1. VUE D'ENSEMBLE
- **Type** : Modal création compte email
- **Contexte** : Web Cloud Email Services
- **Objectif** : Créer un nouveau compte email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 440px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsInput** : Champs email, nom, mot de passe
- **OdsSelect** : Choix quota (boutons radio style)
- **OdsCheckbox** : Envoyer identifiants
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un compte email" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. FORMULAIRE
| Champ | Type | Format |
|-------|------|--------|
| Adresse email * | Input + Suffixe | "[utilisateur]@company.com" |
| Nom d'affichage | Input | "John Doe" |
| Mot de passe * | Input password | Masqué avec œil |

## 6. SÉLECTION QUOTA
| Quota | Style |
|-------|-------|
| 5 Go | Non sélectionné |
| 10 Go | Sélectionné (bordure bleue) |
| 25 Go | Non sélectionné |

## 7. OPTIONS
| Option | État |
|--------|------|
| Envoyer les identifiants par email | ✓ Coché |

## 8. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Créer | Primary #0050D7 | Droite |

## 9. APIS & ENDPOINTS
- POST /email/{type}/{serviceName}/account - Créer compte
- GET /email/{type}/{serviceName}/account - Vérifier disponibilité
