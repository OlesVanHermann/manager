# Cahier des Charges : web-cloud/email-domain - Filtres Délégation

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion filtres email
- **URL** : /email/domain/{domain}/email/{account}/filter
- **Objectif** : Gérer filtres d'un compte email

## 2. STRUCTURE PAGE
- Bouton retour
- Titre avec adresse email
- Tableau filtres
- Sidebar création

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre, email
- **OdsDatagrid** : Liste filtres
- **OdsBadge** : Statut
- **OdsActionMenu** : Actions

## 4. TABLEAU FILTRES
| Colonne | Type |
|---------|------|
| Nom ↑ | Texte (triable) |
| Priorité | Nombre |
| Statut | Badge |
| Actions | ActionMenu |

## 5. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Actif | Vert #DEF7EC |
| Inactif | Rouge #FEE2E2 |

## 6. MENU ACTIONS
| Option | Couleur |
|--------|---------|
| Modifier le filtre | Standard |
| Supprimer le filtre | Rouge |

## 7. SIDEBAR
| Bouton | Action |
|--------|--------|
| Créer un filtre | Ouvrir modal création |

## 8. APIS & ENDPOINTS
- GET /email/domain/{domain}/account/{account}/filter - Liste filtres
- POST /email/domain/{domain}/account/{account}/filter - Créer
- PUT /email/domain/{domain}/account/{account}/filter/{name} - Modifier
- DELETE /email/domain/{domain}/account/{account}/filter/{name} - Supprimer
