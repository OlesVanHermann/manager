# Cahier des Charges : web-cloud/email-exchange - Modal Liste blanche / noire

## 1. VUE D'ENSEMBLE
- **Modal** : Gérer les listes blanche et noire
- **Type** : Modal liste éditable avec onglets
- **Objectif** : Autoriser ou bloquer des expéditeurs

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Onglets Liste blanche / Liste noire
- Formulaire d'ajout
- Liste des entrées
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsTabs** : Onglets blanche/noire
- **OdsInput** : Champ d'ajout
- **OdsButton** : Ajouter (+)
- **OdsTable** : Liste des entrées
- **OdsButton** : Supprimer (×) par ligne

## 4. TITRE
"Liste blanche / noire"

## 5. ONGLETS
| Onglet | État | Description |
|--------|------|-------------|
| Liste blanche | Actif | Expéditeurs autorisés |
| Liste noire | Inactif | Expéditeurs bloqués |

## 6. FORMULAIRE D'AJOUT
| Champ | Type | Placeholder | Action |
|-------|------|-------------|--------|
| Expéditeur | Input text | email@domain.com ou @domain.com | + Ajouter |

## 7. TABLEAU DES ENTRÉES
| Colonne | Description |
|---------|-------------|
| Expéditeur | Email ou domaine |
| - | Bouton supprimer (×) |

## 8. EXEMPLES D'ENTRÉES
| Expéditeur |
|------------|
| @trusted-company.com |
| partner@external.com |
| @newsletter.service.com |

## 9. BOUTON
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/filter : Lister filtres
- POST /email/exchange/{org}/service/{service}/account/{email}/filter : Ajouter filtre
- DELETE /email/exchange/{org}/service/{service}/account/{email}/filter/{id} : Supprimer filtre
