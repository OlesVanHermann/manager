# Cahier des Charges : web-cloud/email-domain - Dashboard principal

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard MX Plan
- **URL** : /email/domain/{domain}
- **Objectif** : Gérer service email MX Plan

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar navigation (Email Pro, Exchange, MX Plan)
- Breadcrumb
- Titre service
- Carte info service
- Onglets
- Tableau comptes email
- Pagination

## 3. COMPOSANTS UI
- **OdsText** : Titres, labels
- **OdsBadge** : Statut Actif
- **OdsTabs** : Navigation onglets
- **OdsButton** : Créer un compte
- **OdsDatagrid** : Tableau comptes
- **OdsActionMenu** : Actions par ligne
- **OdsPagination** : Navigation pages

| Élément | État |
