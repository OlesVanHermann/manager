# Cahier des Charges : web-cloud/email-exchange - Modal ActiveSync

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration ActiveSync
- **Type** : Formulaire de paramètres
- **Objectif** : Configurer les paramètres de synchronisation mobile

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (lecture seule)
- Statut ActiveSync
- Politiques de sécurité (checkboxes)
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x360)
- **OdsInput** : Compte (disabled)
- **OdsBadge** : Statut ActiveSync
- **OdsCheckbox** : Politiques de sécurité
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Paramètres ActiveSync"

## 5. CHAMPS
| Champ | Type | Valeur |
|-------|------|--------|
| Compte | Input disabled | john.doe@example.com |
| ActiveSync | Badge | "Activé" (vert) |

## 6. BADGE STATUT ACTIVESYNC
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Activé | #DCFCE7 (vert clair) | #166534 |
| Désactivé | #FEE2E2 (rouge clair) | #DC2626 |

## 7. POLITIQUES DE SÉCURITÉ
| Politique | État par défaut | Description |
|-----------|-----------------|-------------|
| Exiger un mot de passe sur l'appareil | ☑ | Force le verrouillage par mot de passe |
| Autoriser l'effacement à distance | ☐ | Permet le wipe à distance |
| Bloquer les appareils non conformes | ☐ | Refuse les appareils sans sécurité |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde les paramètres |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/activeSync : Récupérer paramètres
- PUT /email/exchange/{org}/service/{service}/account/{email}/activeSync : Modifier paramètres
