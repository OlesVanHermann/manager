# Cahier des Charges : web-cloud/email-exchange - Modal Modifier un compte partagé

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier un compte partagé
- **Type** : Formulaire de modification
- **Objectif** : Mettre à jour les informations d'une boîte mail partagée

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Adresse email (readonly)
- Nom affiché
- Prénom / Nom
- Quota
- Option visibilité annuaire
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x460)
- **OdsInput** : Email (readonly), Nom affiché, Prénom, Nom, Quota
- **OdsCheckbox** : Cacher dans l'annuaire
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le compte partagé"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur exemple |
|-------|------|------|----------------|
| Adresse e-mail | Input readonly | Grisé | partage@example.com |
| Nom affiché * | Input text | Éditable | Boîte partagée Marketing |
| Prénom | Input text | Éditable | Boîte |
| Nom | Input text | Éditable | Marketing |
| Quota (Go) | Input number | Éditable | 50 |

## 6. LIMITE QUOTA
| Élément | Valeur |
|---------|--------|
| Max | 300 Go |

## 7. OPTION VISIBILITÉ
☐ "Cacher dans l'annuaire"

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Modifier | Primaire bleu | Sauvegarde les modifications |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/sharedAccount/{email} : Récupérer compte
- PUT /email/exchange/{org}/service/{service}/sharedAccount/{email} : Modifier compte
