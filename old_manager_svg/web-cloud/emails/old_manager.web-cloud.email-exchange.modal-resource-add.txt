# Cahier des Charges : web-cloud/email-exchange - Modal Créer une ressource

## 1. VUE D'ENSEMBLE
- **Modal** : Créer une ressource Exchange (salle/équipement)
- **Type** : Formulaire de création
- **Objectif** : Ajouter une salle de réunion ou un équipement réservable

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Type de ressource
- Nom de la ressource
- Adresse email
- Capacité et emplacement
- Option acceptation automatique
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x460)
- **OdsRadio** : Type de ressource
- **OdsInput** : Nom, Email, Capacité, Emplacement
- **OdsSelect** : Domaine
- **OdsCheckbox** : Acceptation auto
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer une ressource"

## 5. TYPE DE RESSOURCE
| Option | État | Description |
|--------|------|-------------|
| Salle de réunion | ● Sélectionné | Espace réservable |
| Équipement | ○ | Matériel réservable |

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Obligatoire |
|-------|------|-------------|-------------|
| Nom de la ressource * | Input text | Ex: Salle Everest | Oui |
| Préfixe email * | Input text | salle-everest | Oui |
| @ | Texte fixe | @ | - |
| Domaine | Select | example.com | Oui |
| Capacité | Input number | 10 | Non |
| Emplacement | Input text | Bâtiment A, 2ème étage | Non |

## 7. OPTION ACCEPTATION
☐ "Accepter automatiquement les réservations"

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Créer | Primaire bleu | Crée la ressource |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/domain : Liste des domaines
- POST /email/exchange/{org}/service/{service}/resourceAccount : Créer ressource
