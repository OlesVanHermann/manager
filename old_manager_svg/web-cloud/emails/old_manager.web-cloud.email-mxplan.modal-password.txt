# Cahier des Charges : web-cloud/email-mxplan - Modal Mot de passe

## 1. VUE D'ENSEMBLE
- **Page** : Changer le mot de passe
- **Type** : Modal formulaire
- **Objectif** : Modifier le mot de passe d'un compte MXplan

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ compte (lecture seule)
- Champ nouveau mot de passe
- Champ confirmation
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Compte (disabled), mot de passe
- **OdsButton** : Annuler, Modifier

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Compte | Input disabled | - | user@domain.com (readonly) |
| Nouveau mot de passe | Password | Oui | Nouveau MDP |
| Confirmer | Password | Oui | Confirmation |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Modifier | Primary | Droite |

## 6. APIS & ENDPOINTS
- POST /email/mxplan/{service}/account/{email}/changePassword - Changer MDP
