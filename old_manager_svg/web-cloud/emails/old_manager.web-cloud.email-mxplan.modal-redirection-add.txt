# Cahier des Charges : web-cloud/email-mxplan - Modal Ajouter redirection

## 1. VUE D'ENSEMBLE
- **Page** : Créer une redirection
- **Type** : Modal formulaire
- **Objectif** : Créer une redirection email

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ adresse source + domaine
- Champ destination
- Checkbox copie locale
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Source, destination
- **OdsCheckbox** : Conserver copie locale
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Adresse source | Input + suffix | Oui | contact + @domain.com |
| Destination | Input email | Oui | destination@autre-domaine.com |
| Copie locale | Checkbox | Non | Conserver une copie locale (coché) |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 6. APIS & ENDPOINTS
- POST /email/mxplan/{service}/redirection - Créer redirection
