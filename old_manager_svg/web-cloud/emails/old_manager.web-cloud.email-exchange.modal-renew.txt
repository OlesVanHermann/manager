# Cahier des Charges : web-cloud/email-exchange - Modal Renouvellement

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer le renouvellement du service
- **Type** : Formulaire de paramètres
- **Objectif** : Modifier le mode et la période de renouvellement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Renouvellement actuel (info)
- Mode de renouvellement (automatique/manuel)
- Période de renouvellement
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsCard** : Renouvellement actuel
- **OdsRadio** : Mode automatique/manuel
- **OdsSelect** : Période
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Renouvellement"

## 5. RENOUVELLEMENT ACTUEL
| Élément | Valeur exemple |
|---------|----------------|
| Mode | Automatique - Mensuel |
| Prochaine échéance | 15/01/2025 |

## 6. MODE DE RENOUVELLEMENT
| Option | État | Description |
|--------|------|-------------|
| Automatique | ● Sélectionné | Renouvellement automatique |
| Manuel | ○ | Rappel avant expiration |

## 7. PÉRIODE DE RENOUVELLEMENT
| Option | Réduction |
|--------|-----------|
| 1 mois | - |
| 3 mois | -5% |
| 6 mois | -8% |
| 12 mois | -10% |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Modifier | Primaire bleu | Applique les changements |

## 9. APIS & ENDPOINTS
- GET /service/{serviceId}/renew : Récupérer config renouvellement
- PUT /service/{serviceId}/renew : Modifier renouvellement
