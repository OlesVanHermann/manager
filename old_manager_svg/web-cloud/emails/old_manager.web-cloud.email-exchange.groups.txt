# Cahier des Charges : web-cloud/email-exchange - Groupes

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des groupes de distribution Exchange.

### Objectif fonctionnel
- Voir la liste des groupes
- Creer de nouveaux groupes
- Gerer les membres
- Modifier les proprietes

## 4. APIS & DONNEES

### Endpoints
```
GET /email/exchange/{organization}/{service}/mailingList
POST /email/exchange/{organization}/{service}/mailingList
DELETE /email/exchange/{organization}/{service}/mailingList/{name}
PUT /email/exchange/{organization}/{service}/mailingList/{name}
GET /email/exchange/{organization}/{service}/mailingList/{name}/member/account
POST /email/exchange/{organization}/{service}/mailingList/{name}/member/account
DELETE /email/exchange/{organization}/{service}/mailingList/{name}/member/account/{member}
```

### Modele
```typescript
interface ExchangeMailingList {
  mailingListAddress: string;
  displayName: string;
  mailingListType: 'distribution' | 'security';
  memberCount: number;
  managedByAccount: string;
  state: 'ok' | 'inCreation';
  hiddenFromAddressBook: boolean;
  senderAuthentification: boolean;
}

interface MailingListMember {
  memberAccount: string;
  memberType: 'account' | 'contact' | 'group';
}
```

## 6. REGLES METIER

- Distribution = diffusion emails a tous les membres
- Securite = groupe avec permissions (plus rare)
- Gestionnaire = peut ajouter/supprimer membres
- Membres = comptes, contacts ou autres groupes
- Authentification expediteur optionnelle
