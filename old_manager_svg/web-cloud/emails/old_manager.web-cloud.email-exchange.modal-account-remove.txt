# Cahier des Charges : web-cloud/email-exchange - Modal Réinitialiser un compte

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de réinitialisation de compte
- **Type** : Modal de confirmation destructive
- **Objectif** : Réinitialiser un compte Exchange (supprime toutes les données)

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Message de confirmation
- Compte concerné
- Avertissement de perte de données
- Boutons Annuler / Réinitialiser

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x200)
- **OdsMessage** : Avertissement rouge
- **OdsButton** : Annuler (secondaire), Réinitialiser (danger rouge)

## 4. TITRE
"Réinitialiser le compte"
- Header fond rouge (#FEF2F2)
- Texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Êtes-vous sûr de vouloir réinitialiser ce compte ?" |
| Compte | admin@example.com (gras) |

## 6. AVERTISSEMENT
Fond rouge (#FEF2F2) :
"⚠ Toutes les données (e-mails, contacts, calendrier) seront supprimées."

## 7. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Réinitialiser | Danger | Rouge (#DC2626) | Réinitialise le compte |

## 8. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/reset : Réinitialiser compte
