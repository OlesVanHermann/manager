# Cahier des Charges : web-cloud/email-pro - Modal Supprimer un alias

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'un alias
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un alias email

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec alias
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x170)
- **OdsText** : Message confirmation, alias
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer l'alias"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Voulez-vous supprimer cet alias ? |
| Alias | {alias} (en gras) |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime l'alias |

## 7. APIS & ENDPOINTS
- DELETE /email/pro/{service}/account/{email}/alias/{alias} : Supprimer alias
