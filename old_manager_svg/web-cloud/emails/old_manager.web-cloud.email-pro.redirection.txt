# Cahier des Charges : web-cloud/email-pro - Redirections

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des redirections Email Pro
- **Type** : Page liste avec tableau
- **Objectif** : Créer et gérer des redirections email

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Email Pro (menu)
- Titre "Redirections"
- Boutons d'action (refresh, ajouter)
- Barre de recherche
- Tableau des redirections
- Pagination
- Bandeau info

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Email Pro
- **OdsButton** : Refresh, Ajouter une redirection
- **OdsInput** : Recherche
- **OdsTable** : Liste des redirections
- **OdsIcon** : Copie locale (✓/✗)
- **OdsBadge** : Statut
- **OdsActionMenu** : Menu actions (⋮)
- **OdsMessage** : Bandeau info bleu

## 4. TITRE
"Redirections"

## 5. BOUTONS D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| ↻ | Icône | Rafraîchir la liste |
| Ajouter une redirection | Secondaire | Ouvre modal création |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| De | Adresse source | contact@example.com |
| Vers | Adresse destination | admin@example.com |
| Copie locale | Icône ✓/✗ | ✓ vert / ✗ rouge |
| Statut | Badge | OK, En création |
| Actions | Menu ⋮ | Modifier, Supprimer |

## 7. ICÔNE COPIE LOCALE
| État | Icône | Couleur |
|------|-------|---------|
| Activée | ✓ | Vert (#10B981) |
| Désactivée | ✗ | Rouge (#EF4444) |

## 8. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| OK | Vert (#10B981) | Redirection active |
| En création | Bleu (#3B82F6) | Création en cours |

## 9. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Ouvre modal modification |
| Supprimer | Ouvre modal suppression (rouge) |

## 13. APIS & ENDPOINTS
- GET /email/pro/{service}/redirection : Lister redirections
- POST /email/pro/{service}/redirection : Créer
- PUT /email/pro/{service}/redirection/{id} : Modifier
- DELETE /email/pro/{service}/redirection/{id} : Supprimer
