# Cahier des Charges : web-cloud/email-exchange - Modal Modifier un compte

## 1. VUE D'ENSEMBLE
- **Modal** : Modification des informations d'un compte
- **Type** : Formulaire de modification
- **Objectif** : Modifier les propriétés d'un compte Exchange existant

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Informations compte (lecture seule)
- Champs modifiables
- Options checkboxes
- Liens vers autres actions
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x560)
- **OdsInput** : Champs texte modifiables
- **OdsCheckbox** : Options (annuaire, Outlook)
- **OdsLink** : Liens vers autres modals
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Modifier le compte"

## 5. INFORMATIONS COMPTE (lecture seule)
| Champ | Valeur |
|-------|--------|
| Compte | admin@example.com |

## 6. CHAMPS MODIFIABLES
| Champ | Type | Exemple |
|-------|------|---------|
| Nom affiché | Input text | Jean Dupont |
| Prénom | Input text | Jean |
| Nom | Input text | Dupont |
| Initiales | Input text | JD |
| Société | Input text | OVHcloud |
| Quota (Go) | Input number | 50 |

## 7. QUOTA
| Élément | Valeur |
|---------|--------|
| Valeur actuelle | 50 Go |
| Maximum | 300 Go max |
| Format | Input + texte info |

## 8. OPTIONS CHECKBOX
| Option | État | Description |
|--------|------|-------------|
| Cacher dans l'annuaire | ☐ | Masque le compte de l'annuaire global |
| Activer Outlook | ☑ | Active la licence Outlook |

## 9. LIENS D'ACTION
| Lien | Action |
|------|--------|
| Modifier le mot de passe → | Ouvre modal mot de passe |
| Gérer les alias → | Ouvre modal alias |

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde les modifications |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email} : Récupérer compte
- PUT /email/exchange/{org}/service/{service}/account/{email} : Modifier compte
- PUT /email/exchange/{org}/service/{service}/account/{email}/changeQuota : Modifier quota
