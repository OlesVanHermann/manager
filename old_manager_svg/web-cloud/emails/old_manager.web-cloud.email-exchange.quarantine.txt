# Cahier des Charges : web-cloud/email-exchange - Quarantaine

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des emails en quarantaine
- **Type** : Page liste avec actions en masse
- **Objectif** : Consulter et gérer les messages bloqués

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Exchange (menu)
- Titre "Quarantaine"
- Bandeau info rétention
- Barre de recherche et filtres
- Tableau des messages
- Actions en masse
- Compteur

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Exchange
- **OdsMessage** : Info bleu (rétention 30j)
- **OdsInput** : Recherche
- **OdsSelect** : Filtre type
- **OdsTable** : Liste des messages
- **OdsCheckbox** : Sélection
- **OdsBadge** : Type de menace
- **OdsButton** : Libérer, Supprimer (par ligne et en masse)

## 4. TITRE
"Quarantaine"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les messages mis en quarantaine sont conservés pendant 30 jours avant suppression automatique."

## 6. FILTRES
| Élément | Type | Options |
|---------|------|---------|
| Recherche | Input | Placeholder: Rechercher... |
| Type | Select | Tous les types, Spam, Malware, Phishing |

## 7. COLONNES DU TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| ☐ | Checkbox | Sélection multiple |
| Date | Date + heure | 29/12/2024 09:15 |
| Expéditeur | Email | spam@unknown.com |
| Destinataire | Email | contact@domain.com |
| Raison | Badge | Malware, Spam, Phishing |
| Actions | Liens | Libérer, Supprimer |

## 8. BADGES RAISON
| Type | Couleur fond | Couleur texte |
|------|--------------|---------------|
| Malware | #FEE2E2 | #DC2626 |
| Spam | #FEF3C7 | #D97706 |
| Phishing | #FEE2E2 | #DC2626 |

## 9. ACTIONS PAR LIGNE
| Action | Couleur | Description |
|--------|---------|-------------|
| Libérer | Bleu (#0050D7) | Débloque le message |
| Supprimer | Rouge (#DC2626) | Supprime définitivement |

## 10. ACTIONS EN MASSE
| Bouton | Style | Action |
|--------|-------|--------|
| Libérer sélection | Secondaire | Débloque les messages sélectionnés |
| Supprimer sélection | Bordure rouge | Supprime les messages sélectionnés |

## 11. COMPTEUR
"X messages en quarantaine"

## 12. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/quarantine : Lister messages
- POST /email/exchange/{org}/service/{service}/quarantine/{id}/release : Libérer
- DELETE /email/exchange/{org}/service/{service}/quarantine/{id} : Supprimer
