# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer une ressource

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de ressource
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer une salle ou un équipement

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Ressource concernée
- Avertissement
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x180)
- **OdsText** : Question, email ressource, avertissement
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer la ressource"

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer cette ressource ?" |
| Ressource | salle-everest@example.com (gras) |
| Avertissement | "Cette action est irréversible." |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime la ressource |

## 7. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/resourceAccount/{email} : Supprimer ressource
