# Cahier des Charges : web-cloud/email-domain - Modal Créer compte

## 1. VUE D'ENSEMBLE
- **Type** : Modal création compte email
- **URL** : N/A (composant modal)
- **Objectif** : Créer un nouveau compte email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Nom compte, description, mot de passe
- **OdsSelect** : Espace disque
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom du compte | Input + @domain fixe | Oui |
| Description | Input texte | Non |
| Mot de passe | Password input | Oui |
| Espace disque | Select dropdown | Non (défaut: 5 Go) |

## 5. CHAMP NOM COMPTE
| Élément | Format |
|---------|--------|
| Input | Partie locale (ex: contact) |
| Suffixe | @domain.com (fixe) |

## 6. OPTIONS ESPACE DISQUE
| Valeur | Description |
|--------|-------------|
| 1 Go | Petite boîte |
| 5 Go | Standard (défaut) |
| 10 Go | Grande boîte |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Créer | Primary | Créer compte |

## 8. APIS & ENDPOINTS
- POST /email/domain/{domain}/account - Créer compte
