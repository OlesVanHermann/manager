# Cahier des Charges : web-cloud/zimbra - Creation compte

## 1. RESUME EXECUTIF

### Perimetre
Formulaire de creation d'un compte Zimbra.

### Objectif fonctionnel
- Creer un nouveau compte email
- Definir les informations utilisateur
- Selectionner l'offre

## 3. COMPOSANTS UI

### Navigation
| Element | Description |
|---------|-------------|
| Breadcrumb | Zimbra > Comptes email > Creer un compte |
| Lien retour | Retour vers mes comptes emails |

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Domaine * | Select | Selectionner un domaine |
| Compte * | Input | Nom du compte |
| Nom | Input | Nom de famille |
| Prenom | Input | Prenom |
| Nom a afficher | Input | Display name |
| Cacher dans annuaire | Checkbox | Hide from GAL |
| Mot de passe * | Password + dice | Avec generateur |
| Forcer changement | Checkbox (coche) | Force reset |
| Offre * | Select | Starter (3) |

### Message info
| Element | Description |
|---------|-------------|
| Type | Info (bleu) |
| Texte | Votre compte fera partie de l'organisation Production |

### Exigences mot de passe
- 10 caracteres minimum
- 1 chiffre et 1 caractere special
- 1 lettre majuscule

### Bouton
| Element | Description |
|---------|-------------|
| Type | Primary |
| Texte | Confirmer |

---

## 4. APIS & DONNEES

### Endpoints
```
POST /zimbra/platform/{platformId}/account
GET /zimbra/platform/{platformId}/domain
GET /zimbra/platform/{platformId}/slot
```

### Modele
```typescript
interface CreateZimbraAccount {
  domain: string;
  name: string;
  lastName?: string;
  firstName?: string;
  displayName?: string;
  hideFromGAL: boolean;
  password: string;
  forcePasswordChange: boolean;
  slotId: string;
}
```
