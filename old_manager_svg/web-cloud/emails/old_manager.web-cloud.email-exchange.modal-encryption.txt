# Cahier des Charges : web-cloud/email-exchange - Modal Chiffrement des messages

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration du chiffrement email
- **Type** : Formulaire de paramètres
- **Objectif** : Configurer les options de chiffrement TLS/S/MIME

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau explicatif
- Options de chiffrement (checkboxes)
- Domaines TLS forcé
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsMessage** : Info bleu
- **OdsCheckbox** : Options de chiffrement
- **OdsInput** : Domaines TLS
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Chiffrement des messages"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Le chiffrement permet de protéger les emails sensibles. Seul le destinataire peut lire le contenu."

## 6. OPTIONS DE CHIFFREMENT
| Option | État | Description |
|--------|------|-------------|
| Activer le chiffrement TLS | ☑ | Chiffre les connexions SMTP |
| Forcer TLS pour les domaines partenaires | ☐ | TLS obligatoire avec certains domaines |
| Activer S/MIME | ☐ | Signature et chiffrement de bout en bout |

## 7. DOMAINES TLS FORCÉ
| Champ | Type | Placeholder |
|-------|------|-------------|
| Domaines TLS forcé | Input text | partner1.com, partner2.com |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde les paramètres |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/encryption : Récupérer paramètres
- PUT /email/exchange/{org}/service/{service}/encryption : Modifier paramètres
