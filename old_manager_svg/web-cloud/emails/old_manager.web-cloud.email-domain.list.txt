# Cahier des Charges : web-cloud/email-domain - Liste des services

## 1. RESUME EXECUTIF

### Perimetre
Liste de tous les services MX Plan du compte.

### Objectif fonctionnel
- Voir tous les domaines avec email MX Plan

## 4. APIS & DONNEES

### Endpoints
```
GET /email/domain
GET /email/domain/{domain}
```

### Modele
```typescript
interface EmailDomainListItem {
  domain: string;
  offer: string;
  accountsUsed: number;
  accountsMax: number;
  spaceUsed: number;
  spaceMax: number;
  status: 'ok' | 'suspended';
}
```
