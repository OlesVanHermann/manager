# Cahier des Charges : web-cloud/email-exchange - Modal Redirection

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer la redirection d'emails
- **Type** : Formulaire de paramètres
- **Objectif** : Rediriger les emails vers une autre adresse

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (readonly)
- Adresse de redirection
- Options de redirection
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x360)
- **OdsInput** : Compte (readonly), Adresse de redirection
- **OdsCheckbox** : Options de redirection
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Configurer la redirection"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur exemple |
|-------|------|------|----------------|
| Compte | Input readonly | Grisé | john.doe@example.com |
| Rediriger vers * | Input email | Requis | backup@autre-domaine.com |

## 6. OPTIONS
| Option | État | Description |
|--------|------|-------------|
| Conserver une copie locale | ☑ Coché | Garde une copie dans la boîte |
| Rediriger uniquement les nouveaux messages | ☐ Décoché | N'affecte pas l'historique |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde la redirection |

## 8. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email} : Récupérer config redirection
- PUT /email/exchange/{org}/service/{service}/account/{email} : Modifier redirection
