# Cahier des Charges : web-cloud/email-pro - Modal Ajouter un alias

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un alias à un compte Email Pro
- **Type** : Modal formulaire
- **Objectif** : Créer une adresse email alias

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Champ alias avec sélection domaine
- Info quota
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x240)
- **OdsInput** : Partie locale de l'alias
- **OdsSelect** : Sélection domaine
- **OdsText** : Info quota
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un alias"

## 5. CHAMP ALIAS
| Partie | Composant | Exemple |
|--------|-----------|---------|
| Alias | OdsInput | alias |
| @ | Texte statique | @ |
| Domaine | OdsSelect | example.com ▼ |

## 6. INFO QUOTA
"Quota : X/Y alias utilisés"

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Ajouter | Primaire bleu | Crée l'alias |

## 8. VALIDATION
| Règle | Description |
|-------|-------------|
| Format | alias@domaine valide |
| Quota | Vérifier disponibilité |
| Unicité | Alias non existant |

## 9. APIS & ENDPOINTS
- POST /email/pro/{service}/account/{email}/alias : Créer alias
  - alias: string
