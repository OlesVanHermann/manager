Onglet de gestion des repondeurs automatiques Zimbra.

- Voir les repondeurs configures

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/autoReply
POST /zimbra/platform/{platformId}/autoReply
DELETE /zimbra/platform/{platformId}/autoReply/{autoReplyId}
PUT /zimbra/platform/{platformId}/autoReply/{autoReplyId}
```

### Modele
```typescript
interface ZimbraAutoReply {
  id: string;
  account: string;
  from: string; // date
  to: string; // date
  copyTo?: string;
  message: string;
  state: 'ok' | 'error';
}
```
