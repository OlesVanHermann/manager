# Cahier des Charges : web-cloud/email-pro - Modal Supprimer un domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'un domaine
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un domaine associé

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec domaine
- Avertissement suppression comptes
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x200)
- **OdsText** : Message confirmation, domaine
- **OdsMessage** : Avertissement rouge
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer le domaine"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Voulez-vous supprimer ce domaine ? |
| Domaine | {domaine} (en gras) |

## 6. AVERTISSEMENT
Fond rouge (#FEF2F2) :
"⚠ Tous les comptes liés seront également supprimés."

## 7. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le domaine |

## 8. APIS & ENDPOINTS
- DELETE /email/pro/{service}/domain/{domain} : Supprimer domaine
