# Cahier des Charges : web-cloud/email-exchange - Modal Alias du compte

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des alias d'un compte email
- **Type** : Modal liste avec tableau
- **Objectif** : Visualiser et gérer les alias d'un compte Exchange

## 2. STRUCTURE MODAL
- Header avec titre (email du compte) et bouton fermer (×)
- Bandeau info explicatif
- Bouton ajouter alias
- Tableau des alias
- Compteur et bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x440)
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Ajouter alias, Fermer
- **OdsTable** : Liste des alias
- **OdsIcon** : Icône supprimer (× rouge)

## 4. TITRE
"Alias - john.doe@example.com"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les alias permettent de recevoir des emails sur plusieurs adresses dans la même boîte."

## 6. BOUTON AJOUTER
| Élément | Valeur |
|---------|--------|
| Position | En haut à droite |
| Label | "Ajouter alias" |
| Type | Secondaire |

## 7. TABLEAU
| Colonne | Description |
|---------|-------------|
| Alias | Partie locale de l'adresse |
| Domaine | Domaine de l'alias |
| Date de création | Date au format FR |
| Action | Bouton supprimer (× rouge) |

## 8. EXEMPLES DE DONNÉES
| Alias | Domaine | Date |
|-------|---------|------|
| jdoe | example.com | 15/12/2024 |
| john | example.com | 10/12/2024 |
| contact | example.org | 01/12/2024 |

## 9. COMPTEUR
"3 alias"

## 10. BOUTON FERMER
| Élément | Valeur |
|---------|--------|
| Position | Footer droit |
| Label | "Fermer" |
| Type | Secondaire |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{organizationName}/service/{exchangeService}/account/{primaryEmailAddress}/alias : Lister les alias
- POST /email/exchange/{organizationName}/service/{exchangeService}/account/{primaryEmailAddress}/alias : Ajouter un alias
- DELETE /email/exchange/{organizationName}/service/{exchangeService}/account/{primaryEmailAddress}/alias/{alias} : Supprimer un alias
