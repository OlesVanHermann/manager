## 6. APIS & ENDPOINTS

### 6.1 Exporter les comptes en CSV

```
GET /email/exchange/{organizationName}/service/{exchangeService}/account/export?format=csv
```

Response Headers:
```
Content-Type: text/csv
Content-Disposition: attachment; filename="exchange-accounts-2026-01-02.csv"
```

Response Body (CSV):
```csv
email,alias,quota_used_mb,status
john.doe@example.com,"alias1@example.com,alias2@example.com",2048,active
jane.smith@example.com,,512,active
```
