# Cahier des Charges : web-cloud/email-pro - Modal Ajouter un domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Assistant d'ajout de domaine Email Pro
- **Type** : Modal formulaire multi-étapes
- **Objectif** : Associer un domaine au service Email Pro

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Indicateur d'étape
- Sélection source domaine (radio)
- Select domaine OVH ou input externe
- Type de configuration (radio)
- Boutons Annuler / Suivant

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsText** : Indicateur étape
- **OdsRadio** : Source domaine, type config
- **OdsSelect** : Domaine OVH
- **OdsInput** : Domaine externe
- **OdsButton** : Annuler, Suivant

## 4. TITRE
"Ajouter un domaine"

## 5. INDICATEUR D'ÉTAPE
"Étape 1/2 - Sélection du domaine"

## 6. SOURCE DU DOMAINE
| Option | Description | État |
|--------|-------------|------|
| ○ Domaine OVHcloud | Domaine géré par OVH | Sélectionné |
| ○ Domaine externe | Non géré par OVHcloud | - |

## 7. SÉLECTION DOMAINE OVH
| Composant | Description |
|-----------|-------------|
| OdsSelect | Liste des domaines disponibles |
| Exemple | example.com ▼ |

## 8. TYPE DE CONFIGURATION
| Option | Description | Recommandé |
|--------|-------------|------------|
| ○ Autoritatif | OVH gère tous les emails | Oui |
| ○ Non autoritatif | Coexistence avec autre serveur | Non |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Suivant | Primaire bleu | Passe à l'étape 2 |

## 10. APIS & ENDPOINTS
- GET /domain : Lister domaines OVH
- POST /email/pro/{service}/domain : Ajouter domaine
  - name: string
  - type: "authoritative" | "nonAuthoritative"
