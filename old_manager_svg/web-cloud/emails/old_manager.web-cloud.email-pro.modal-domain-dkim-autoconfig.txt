# Cahier des Charges : web-cloud/email-pro - Modal Configuration DKIM

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique DKIM
- **Type** : Modal action avec statut
- **Objectif** : Activer la signature DKIM pour un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau statut (warning orange)
- Domaine concerné
- Sélecteur DKIM
- Instructions DNS
- Boutons Fermer / Activer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x360)
- **OdsMessage** : Alerte warning orange
- **OdsText** : Domaine, sélecteur
- **OdsInput** : Sélecteur DKIM (lecture seule)
- **OdsButton** : Fermer, Activer

## 4. TITRE
"Configuration DKIM"

## 5. BANDEAU STATUT
Fond orange (#FEF3C7), texte orange (#92400E) :
"⚠ DKIM non activé pour ce domaine."

## 6. INFORMATIONS
| Élément | Valeur |
|---------|--------|
| Domaine concerné | example.com |
| Sélecteur DKIM | ovhex1234-selector1 |

## 7. INSTRUCTIONS
"Ajoutez l'enregistrement CNAME dans votre zone DNS."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |
| Activer | Primaire bleu | Active DKIM automatiquement |

## 9. APIS & ENDPOINTS
- GET /email/pro/{service}/domain/{domain}/dkim : Récupérer config DKIM
- POST /email/pro/{service}/domain/{domain}/dkim : Activer DKIM
