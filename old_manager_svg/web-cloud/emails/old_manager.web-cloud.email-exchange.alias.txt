# Cahier des Charges : web-cloud/email-exchange - Page Gestion des alias

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des alias d'un compte email
- **Type** : Page liste avec tableau
- **Objectif** : Visualiser, ajouter et supprimer les alias

## 2. LAYOUT
- Header OVHcloud avec navigation
- Sidebar Exchange
- Lien retour vers compte
- Titre + boutons (refresh, ajouter)
- Tableau des alias
- Bandeau info quota

## 3. COMPOSANTS UI
- **OdsButton** : Ajouter un alias, Rafraîchir
- **OdsTable** : Liste des alias
- **OdsActionMenu** : Menu contextuel (⋮)
- **OdsMessage** : Bandeau info quota
- **OdsBadge** : Statut tâche en cours

## 4. NAVIGATION
- Lien retour : "← Gestion des alias pour user@example.com"

## 5. TITRE
"Alias"

## 6. BOUTONS D'ACTION
| Bouton | Type | Action |
|--------|------|--------|
| ↻ | Secondaire (icône) | Rafraîchir la liste |
| Ajouter un alias | Secondaire | Ouvre modal ajout alias |

## 7. TABLEAU
| Colonne | Description |
|---------|-------------|
| Alias | Adresse email complète |
| Statut | État de l'alias ou tâche en cours |
| Actions | Menu ⋮ |

## 8. EXEMPLES DE DONNÉES
| Alias | Statut |
|-------|--------|
| alias1@example.com | - |
| contact@example.com | - |
| info@example.com | Tâche planifiée ou en cours (bleu) |
| X500:/O=EXCHANGE/... | - |

## 9. ACTION MENU
| Option | Couleur | Action |
|--------|---------|--------|
| Supprimer cet alias | Rouge (#DC2626) | Supprime l'alias |

## 10. BANDEAU QUOTA
Fond bleu (#EFF6FF), bordure bleue :
"Chaque compte Exchange acheté ouvre le droit à 5 alias. Vous avez utilisé 4 sur 5 alias disponibles."

## 11. PAGINATION
"Affichage 1-4 sur 4 alias"

## 12. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/alias : Lister alias
- POST /email/exchange/{org}/service/{service}/account/{email}/alias : Ajouter alias
- DELETE /email/exchange/{org}/service/{service}/account/{email}/alias/{alias} : Supprimer alias
