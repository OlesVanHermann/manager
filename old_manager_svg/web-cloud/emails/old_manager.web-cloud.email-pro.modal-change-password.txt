# Cahier des Charges : web-cloud/email-pro - Modal changement mot de passe

## 1. RESUME EXECUTIF

### Perimetre
Modal de modification du mot de passe d'un compte Email Pro.

### Objectif fonctionnel
- Changer le mot de passe d'un compte
- Valider la complexite
- Optionnellement envoyer par email

## 4. APIS & DONNEES

### Endpoints
```
POST /email/pro/{service}/account/{email}/changePassword
```

### Modele
```typescript
interface ChangePasswordForm {
  password: string;
  sendByEmail: boolean;
}

interface PasswordStrength {
  score: number;
  level: 'weak' | 'medium' | 'strong';
  requirements: {
    minLength: boolean;
    hasUppercase: boolean;
    hasLowercase: boolean;
    hasDigit: boolean;
    hasSpecial: boolean;
  };
}
```
