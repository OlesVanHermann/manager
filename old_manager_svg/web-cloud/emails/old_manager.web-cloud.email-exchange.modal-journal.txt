# Cahier des Charges : web-cloud/email-exchange - Modal Journalisation

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration de la journalisation email
- **Type** : Formulaire de paramètres
- **Objectif** : Configurer l'archivage automatique des emails

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau explicatif
- Activation journalisation
- Adresse de destination
- Portée (internes/externes)
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x360)
- **OdsMessage** : Info bleu
- **OdsCheckbox** : Activation, portée
- **OdsInput** : Adresse de journalisation
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Journalisation des emails"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"La journalisation conserve une copie de tous les emails envoyés et reçus pour conformité réglementaire."

## 6. ACTIVATION
☐ "Activer la journalisation"

## 7. ADRESSE DE JOURNALISATION
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Adresse de journalisation | Input email | Oui (*) |

Placeholder: "archive@example.com"

## 8. PORTÉE
| Option | État | Description |
|--------|------|-------------|
| Emails internes | ☑ | Archive les mails internes |
| Emails externes | ☑ | Archive les mails externes |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde la configuration |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/journal : Récupérer config
- PUT /email/exchange/{org}/service/{service}/journal : Modifier config
