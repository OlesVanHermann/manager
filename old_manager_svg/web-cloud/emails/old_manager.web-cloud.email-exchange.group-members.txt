# Cahier des Charges : web-cloud/email-exchange - Membres du groupe

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des membres de groupe
- **Type** : Modal liste avec actions et pagination
- **Objectif** : Gérer les membres d'un groupe de distribution Exchange

## 2. STRUCTURE MODAL
- Header avec email du groupe et bouton fermer (×)
- Bouton ajouter
- Tableau des membres avec type et date
- Pagination
- Bouton Fermer en bas

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x490)
- **OdsButton** : Ajouter membre (secondaire), Fermer
- **OdsTable** : Liste des membres
- **OdsIcon** : Bouton supprimer (×) rouge
- **OdsPagination** : Navigation pages

## 4. TITRE
"Membres - {email_groupe}"

## 5. BOUTON AJOUTER
| Élément | Valeur |
|---------|--------|
| Label | "Ajouter membre" |
| Type | Secondaire (bordure) |
| Position | Haut droite |

## 6. TABLEAU
| Colonne | Description |
|---------|-------------|
| Membre | Email du membre |
| Type | Compte / Contact / Groupe |
| Ajouté le | Date d'ajout au groupe |
| Actions | Bouton supprimer (×) |

## 7. TYPES DE MEMBRES
| Type | Description |
|------|-------------|
| Compte | Compte Exchange interne |
| Contact | Contact externe |
| Groupe | Autre groupe de distribution |

## 8. EXEMPLES DE DONNÉES
| Membre | Type | Ajouté le |
|--------|------|-----------|
| user1@example.com | Compte | 15/12/2024 |
| user2@example.com | Compte | 14/12/2024 |
| contact@external.com | Contact | 10/12/2024 |
| support@example.com | Groupe | 05/12/2024 |

## 9. ACTION SUPPRIMER
| Élément | Valeur |
|---------|--------|
| Icône | × (croix) |
| Couleur | Rouge (#DC2626) |
| Action | Supprime le membre du groupe |

## 10. PAGINATION
"1-4 sur 8 membres"

## 11. BOUTON FERMER
| Élément | Valeur |
|---------|--------|
| Label | "Fermer" |
| Type | Secondaire (bordure) |
| Position | Bas droite |

## 12. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/mailingList/{mailingList}/member : Lister membres
- POST /email/exchange/{org}/service/{service}/mailingList/{mailingList}/member : Ajouter membre
- DELETE /email/exchange/{org}/service/{service}/mailingList/{mailingList}/member/{id} : Supprimer membre
