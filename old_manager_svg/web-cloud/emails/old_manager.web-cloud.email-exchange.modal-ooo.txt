# Cahier des Charges : web-cloud/email-exchange - Modal Absence du bureau

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration de l'absence du bureau (Out of Office)
- **Type** : Formulaire de paramètres
- **Objectif** : Configurer la réponse automatique d'absence

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Statut (au bureau / absent)
- Option période
- Plage de dates
- Message interne
- Option expéditeurs externes
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsRadio** : Statut présent/absent
- **OdsCheckbox** : Définir période, Expéditeurs externes
- **OdsInput** : Dates de début/fin
- **OdsTextarea** : Message d'absence
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Absence du bureau"

## 5. STATUT
| Option | État | Description |
|--------|------|-------------|
| Je suis au bureau | Non sélectionné | Désactive la réponse auto |
| Je suis absent | Sélectionné | Active la réponse auto |

## 6. PÉRIODE
☑ "Définir une période"

| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Du | Date picker | 23/12/2024 |
| au | Date picker | 02/01/2025 |

## 7. MESSAGE INTERNE
| Champ | Type | Contenu exemple |
|-------|------|-----------------|
| Message (interne) | Textarea | Je suis en congés jusqu'au 2 janvier. Contactez Jane pour les urgences. |

## 8. OPTION EXTERNE
☐ "Envoyer aussi aux expéditeurs externes"

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde la configuration |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/outOfOffice : Récupérer config
- PUT /email/exchange/{org}/service/{service}/account/{email}/outOfOffice : Modifier config
