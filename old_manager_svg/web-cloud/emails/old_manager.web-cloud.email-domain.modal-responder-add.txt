# Cahier des Charges : web-cloud/email-domain - Modal Créer répondeur

## 1. VUE D'ENSEMBLE
- **Type** : Modal création répondeur automatique
- **URL** : N/A (composant modal)
- **Objectif** : Configurer une réponse automatique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsSelect** : Compte email
- **OdsInput** : Dates (datepicker)
- **OdsTextarea** : Message réponse
- **OdsCheckbox** : Copie messages
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Compte | Select dropdown | Oui |
| Date de début | Datepicker | Oui |
| Date de fin | Datepicker | Oui |
| Message de réponse | Textarea | Oui |
| Conserver une copie des messages reçus | Checkbox | Non (coché par défaut) |

## 5. DATES
| Format | Exemple |
|--------|---------|
| Date | 01/01/2025 |
| Icône | calendrier |

## 6. MESSAGE
| Propriété | Valeur |
|-----------|--------|
| Type | Textarea multi-lignes |
| Placeholder | Message type absence |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Créer | Primary | Créer répondeur |

## 8. APIS & ENDPOINTS
- POST /email/domain/{domain}/responder - Créer répondeur
