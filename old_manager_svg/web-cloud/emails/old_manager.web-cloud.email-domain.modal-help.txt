# Cahier des Charges : web-cloud/email-domain - Modal Aide

## 1. VUE D'ENSEMBLE
- **Type** : Modal aide contextuelle
- **URL** : N/A (composant modal)
- **Objectif** : Guider utilisateur sans emails configurés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Explication
- **OdsButton** : Actions rapides
- **OdsCheckbox** : Options masquage

## 4. CONTENU
| Élément | Texte |
|---------|-------|
| Explication | "Vous n'avez pas encore d'emails configurés. Que souhaitez-vous faire ?" |

## 5. ACTIONS RAPIDES
| Bouton | Action |
|--------|--------|
| Créer un compte email | Ouvrir modal création compte |
| Créer une redirection | Ouvrir modal création redirection |
| Créer une mailing list | Ouvrir modal création liste |
| Créer un répondeur | Ouvrir modal création répondeur |

## 6. OPTIONS MASQUAGE
| Checkbox | Label |
|----------|-------|
| Option 1 | Ne plus afficher cette aide pour ce domaine |
| Option 2 | Ne plus afficher cette aide pour tous les domaines |

## 7. BOUTON
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary | Fermer modal |

## 8. APIS & ENDPOINTS
- POST /me/preferences - Sauvegarder préférence masquage
