# Cahier des Charges : web-cloud/email-exchange - Modal Historique des connexions

## 1. VUE D'ENSEMBLE
- **Modal** : Affichage de l'historique des connexions
- **Type** : Modal liste avec tableau
- **Objectif** : Consulter l'historique des connexions d'un compte

## 2. STRUCTURE MODAL
- Header avec email du compte et bouton fermer (×)
- Filtre par période
- Tableau des connexions
- Pagination
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x490)
- **OdsSelect** : Filtre période
- **OdsTable** : Liste des connexions
- **OdsBadge** : Statut (Succès/Échec)
- **OdsPagination** : Navigation pages
- **OdsButton** : Fermer

## 4. TITRE
"Historique des connexions - {email}"

## 5. FILTRE PÉRIODE
| Option | Description |
|--------|-------------|
| 7 derniers jours | Par défaut |
| 30 derniers jours | Un mois |
| 90 derniers jours | Trois mois |

## 6. TABLEAU
| Colonne | Description |
|---------|-------------|
| Date | Date et heure de la connexion |
| IP | Adresse IP (masquée partiellement) |
| Localisation | Ville, Pays |
| Client | Type de client (Outlook, iPhone, Webmail) |
| Statut | Badge Succès/Échec |

## 7. BADGES STATUT
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Succès | #DCFCE7 (vert clair) | #166534 |
| Échec | #FEE2E2 (rouge clair) | #DC2626 |

## 8. EXEMPLES DE DONNÉES
| Date | IP | Localisation | Client | Statut |
|------|-----|--------------|--------|--------|
| 28/12/2024 14:32 | 92.184.xxx.xxx | Paris, FR | Outlook | Succès |
| 28/12/2024 10:15 | 92.184.xxx.xxx | Paris, FR | iPhone | Succès |

## 9. PAGINATION
"1-4 sur 47 connexions"

## 10. BOUTON FERMER
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/loginHistory : Récupérer historique
