# Cahier des Charges : web-cloud/email-exchange - Délégation compte partagé

## 1. VUE D'ENSEMBLE
- **Modal** : Gérer les délégations d'un compte partagé
- **Type** : Modal liste avec onglets
- **Objectif** : Définir les droits d'accès à une boîte partagée

## 2. STRUCTURE MODAL
- Header avec email du compte et bouton fermer (×)
- Onglets types de délégation
- Bandeau info
- Bouton Ajouter délégation
- Tableau des délégués
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x490)
- **OdsTabs** : Onglets types de délégation
- **OdsMessage** : Info bleu
- **OdsButton** : Ajouter délégation
- **OdsTable** : Liste des délégués
- **OdsButton** : Supprimer (×)

## 4. TITRE
"Délégation - {email}"
Exemple: "Délégation - partage@example.com"

## 5. ONGLETS
| Onglet | État | Description |
|--------|------|-------------|
| Accès complet | Actif | Ouvrir et gérer la boîte |
| Envoyer en tant que | Inactif | Envoyer au nom du compte |

## 6. BANDEAU INFO (ACCÈS COMPLET)
Fond bleu (#EFF6FF) :
"L'accès complet permet à l'utilisateur d'ouvrir et gérer la boîte partagée."

## 7. BOUTON D'ACTION
| Bouton | Position | Action |
|--------|----------|--------|
| Ajouter délégation | En haut à droite | Ouvre modal d'ajout |

## 8. COLONNES DU TABLEAU
| Colonne | Description |
|---------|-------------|
| Compte | Email du délégué |
| Date d'ajout | Date de création |
| - | Bouton supprimer (×) |

## 9. EXEMPLES DE DONNÉES
| Compte | Date d'ajout |
|--------|--------------|
| admin@example.com | 15/12/2024 |
| marketing@example.com | 10/12/2024 |
| support@example.com | 05/12/2024 |

## 10. BOUTON
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/sharedAccount/{email}/fullAccess : Lister accès complets
- GET /email/exchange/{org}/service/{service}/sharedAccount/{email}/sendAs : Lister envoi en tant que
- POST /email/exchange/{org}/service/{service}/sharedAccount/{email}/fullAccess/{delegate} : Ajouter
- DELETE /email/exchange/{org}/service/{service}/sharedAccount/{email}/fullAccess/{delegate} : Supprimer
