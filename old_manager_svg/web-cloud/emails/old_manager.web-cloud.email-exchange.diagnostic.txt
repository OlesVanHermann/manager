# Cahier des Charges : web-cloud/email-exchange - Diagnostics

## 1. VUE D'ENSEMBLE
- **Écran** : Outil de diagnostic Exchange
- **Type** : Page formulaire + résultats
- **Objectif** : Diagnostiquer les problèmes d'un compte email

## 2. LAYOUT
- Header OVHcloud
- Sidebar Exchange (Diagnostics actif)
- Titre + description
- Formulaire de diagnostic
- Bandeau warning
- Section résultats

## 3. COMPOSANTS UI
- **OdsSelect** : Sélection compte
- **OdsInput** : Mot de passe (type password)
- **OdsButton** : Lancer le diagnostic, Nouveau diagnostic, Créer un ticket
- **OdsMessage** : Bandeau warning jaune
- **OdsCard** : Section résultats
- **OdsIcon** : Indicateurs OK/Warning

## 4. TITRE
"Diagnostics"

## 5. DESCRIPTION
"Le diagnostic d'un compte email permet de valider plusieurs éléments à savoir si le compte est bloqué, la validité de l'enregistrement MX et SRV et de l'abonnement au compte, l'authentification au webmail ainsi que l'envoi et la réception par ce compte. Le processus peut prendre entre 3 et 10 minutes."

## 6. FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Compte email à diagnostiquer | Select dropdown | Oui (*) |
| Mot de passe | Input password | Oui (*) |

## 7. BOUTON LANCEMENT
| Élément | Valeur |
|---------|--------|
| Label | "Lancer le diagnostic" |
| Type | Primaire bleu |

## 8. BANDEAU WARNING
Fond jaune (#FEF3C7) :
"Si vous quittez la section web de l'espace client, le résultat du diagnostic ne pourra pas être conservé."

## 9. SECTION RÉSULTATS
Titre : "Résultat du diagnostic"
Sous-titre : "Compte analysé : user@example.com"

| Test | Résultat | Icône |
|------|----------|-------|
| Le compte n'est pas bloqué pour envoi de spam | OK | ✓ vert |
| Enregistrement MX valide | OK | ✓ vert |
| Enregistrement SRV valide | OK | ✓ vert |
| Authentification webmail | Warning | ⚠ jaune |

## 10. MESSAGE WARNING WEBMAIL
"Vérifiez les identifiants du compte"

## 11. BOUTONS RÉSULTATS
| Bouton | Type | Action |
|--------|------|--------|
| Nouveau diagnostic | Secondaire (bordure bleue) | Réinitialise le formulaire |
| Créer un ticket | Primaire bleu | Ouvre formulaire support |

## 12. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/diagnostic : Lancer diagnostic
