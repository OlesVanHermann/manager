# Cahier des Charges : web-cloud/email-pro - Modal Réinitialiser un compte

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de réinitialisation d'un compte
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression de toutes les données du compte

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec email
- Avertissement de suppression
- Boutons Annuler / Réinitialiser

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x200)
- **OdsText** : Message confirmation
- **OdsMessage** : Avertissement rouge
- **OdsButton** : Annuler, Réinitialiser (rouge)

## 4. TITRE
"Réinitialiser le compte"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Êtes-vous sûr de vouloir réinitialiser ce compte ? |
| Email | {email} (en gras) |
| Avertissement | ⚠ Toutes les données seront supprimées. |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Réinitialiser | Danger | Rouge (#DC2626) | Réinitialise le compte |

## 7. APIS & ENDPOINTS
- DELETE /email/pro/{service}/account/{email} : Réinitialiser compte
