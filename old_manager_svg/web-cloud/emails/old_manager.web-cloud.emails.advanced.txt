## 6. APIS & ENDPOINTS

```
# Ressources partagées
GET /email/exchange/{organization}/service/{serviceName}/resourceAccount
→ Liste des ressources (salles, équipements)

# Dossiers publics
GET /email/exchange/{organization}/service/{serviceName}/publicFolder
→ Liste des dossiers publics

# Groupes de distribution
GET /email/exchange/{organization}/service/{serviceName}/mailingList
→ Liste des groupes

# Délégations
GET /email/exchange/{organization}/service/{serviceName}/account/{email}/fullAccess
GET /email/exchange/{organization}/service/{serviceName}/account/{email}/sendAs
GET /email/exchange/{organization}/service/{serviceName}/account/{email}/sendOnBehalfTo
→ Droits de délégation

# Politiques
GET /email/exchange/{organization}/service/{serviceName}/policy
→ Politiques de rétention/archivage

# Quotas
GET /email/exchange/{organization}/service/{serviceName}
→ Inclut les informations de quota global
```
