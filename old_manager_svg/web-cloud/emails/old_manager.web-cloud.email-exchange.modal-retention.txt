# Cahier des Charges : web-cloud/email-exchange - Modal Politique de rétention

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer la politique de rétention
- **Type** : Formulaire de paramètres
- **Objectif** : Définir la durée de conservation des emails

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau explicatif
- Paramètres de rétention par type
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x390)
- **OdsMessage** : Info bleu
- **OdsSelect** : Durée par type de dossier
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Politique de rétention"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"La politique de rétention définit la durée de conservation des emails et leur suppression automatique."

## 6. PARAMÈTRES DE RÉTENTION
| Dossier | Options disponibles | Défaut |
|---------|---------------------|--------|
| Éléments supprimés | 7/14/30/60/90 jours | 30 jours |
| Spam | 7/14/30 jours | 14 jours |
| Brouillons | 30/60/90 jours/Jamais | Jamais |

## 7. OPTIONS DE DURÉE
| Option | Description |
|--------|-------------|
| 7 jours | Suppression après 1 semaine |
| 14 jours | Suppression après 2 semaines |
| 30 jours | Suppression après 1 mois |
| 60 jours | Suppression après 2 mois |
| 90 jours | Suppression après 3 mois |
| Jamais | Conservation permanente |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Applique la politique |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/retention : Récupérer politique
- PUT /email/exchange/{org}/service/{service}/retention : Modifier politique
