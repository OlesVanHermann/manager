# Cahier des Charges : web-cloud/email-exchange - Modal Configuration mobile

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration d'un compte Exchange sur mobile
- **Type** : Modal informative avec QR code
- **Objectif** : Faciliter la configuration sur iPhone/Android

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (readonly)
- QR code de configuration
- Paramètres manuels
- Instructions
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x490)
- **OdsInput** : Compte (readonly)
- **OdsImage** : QR code
- **OdsText** : Paramètres serveur
- **OdsMessage** : Instructions (info bleu)
- **OdsButton** : Fermer

## 4. TITRE
"Configuration mobile"

## 5. COMPTE CONCERNÉ
| Champ | État | Valeur exemple |
|-------|------|----------------|
| Compte | Readonly grisé | john.doe@example.com |

## 6. QR CODE
Zone 150x150 px avec QR code pour configuration automatique

## 7. PARAMÈTRES MANUELS
| Paramètre | Valeur exemple |
|-----------|----------------|
| Serveur | ex.mail.ovh.net |
| Utilisateur | john.doe@example.com |
| Domaine | example.com |

## 8. INSTRUCTIONS
Fond bleu (#EFF6FF) :
```
Instructions:
1. Ouvrez l'app Mail sur votre iPhone/Android
2. Ajoutez un compte Exchange et scannez le QR code
```

## 9. BOUTON
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/mobileConfig : Générer QR code/config
