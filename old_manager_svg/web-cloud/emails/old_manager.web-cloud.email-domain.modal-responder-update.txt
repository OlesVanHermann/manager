# Cahier des Charges : web-cloud/email-domain - Modal Modifier répondeur

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification répondeur
- **URL** : N/A (composant modal)
- **Objectif** : Modifier un répondeur automatique existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 510px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsRadio** : Durée (Temporaire/Permanent)
- **OdsInput** : Dates (datepicker)
- **OdsTextarea** : Message réponse
- **OdsButton** : Actions

## 4. EN-TÊTE
| Élément | Description |
|---------|-------------|
| Compte | Affiche email concerné |
| Sous-titre | "Configurez les paramètres du répondeur automatique." |

## 5. FORMULAIRE
| Champ | Type | Options |
|-------|------|---------|
| Durée | Radio group | Temporaire (défaut), Permanent |
| Date de début | Datepicker | Format: YYYY-MM-DD HH:mm |
| Date de fin | Datepicker | Format: YYYY-MM-DD HH:mm |
| Message | Textarea | Texte libre |

## 6. AIDE
| Type | Texte |
|------|-------|
| Info | "Le message ne doit pas contenir de code HTML." |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer modifications |

## 8. APIS & ENDPOINTS
- PUT /email/domain/{domain}/responder/{account} - Modifier répondeur
