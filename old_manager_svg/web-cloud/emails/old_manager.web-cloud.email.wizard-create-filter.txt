# Cahier des Charges : web/email - Wizard créer un filtre

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire avec règles dynamiques
- **Contexte** : Gestion email - Filtres
- **Objectif** : Créer un filtre de tri automatique des emails

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 800px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, labels, aide
- **OdsInput** : Nom du filtre, valeurs
- **OdsSelect** : Header, opérande, action
- **OdsCheckbox** : Activer le filtre
- **OdsCard** : Panneau d'aide
- **OdsButton** : Ajouter règle (+), Annuler, Créer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un filtre email" |
| Description | "Créez un filtre pour trier automatiquement vos emails entrants." |

## 5. SECTION INFORMATIONS GÉNÉRALES

### Champ Nom du filtre
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du filtre *" |
| Type | OdsInput |
| Exemple | "Filtre spam" |

### Champ Priorité
| Propriété | Valeur |
|-----------|--------|
| Label | "Priorité *" |
| Type | OdsInput (number) |
| Exemple | 1 |

### Checkbox Activer
| Propriété | Valeur |
|-----------|--------|
| Label | "Activer le filtre" |
| État | Coché par défaut |

## 6. SECTION RÈGLES

### Titre section
| Texte |
|-------|
| "Règles *" |

### Ligne de règle
| Élément | Type | Options |
|---------|------|---------|
| Header | OdsSelect | De (expéditeur), À (destinataire), Objet, etc. |
| Opérande | OdsSelect | contient, ne contient pas, est égal à, etc. |
| Valeur | OdsInput | Texte libre |
| Bouton + | Rond bleu | Ajouter une règle |

## 7. SECTION ACTION

### Sélection action
| Propriété | Valeur |
|-----------|--------|
| Label | "Action *" |
| Type | OdsSelect |
| Options | Supprimer le message, Déplacer vers dossier, Marquer comme lu, Transférer à |

## 8. PANNEAU D'AIDE
| Section | Contenu |
|---------|---------|
| Priorité | "Un nombre plus petit = priorité plus haute." |
| Règles | "Toutes les règles doivent correspondre pour que l'action soit exécutée." |

## 9. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Créer | Primary (bleu) | Crée le filtre |

## 10. APIS & ENDPOINTS
- GET /email/domain/{domain}/account/{accountName}/filter - Liste des filtres
- POST /email/domain/{domain}/account/{accountName}/filter - Créer filtre
- GET /email/domain/{domain}/account/{accountName}/filter/{name}/rule - Règles du filtre
