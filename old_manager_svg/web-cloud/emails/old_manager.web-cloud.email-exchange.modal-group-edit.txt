# Cahier des Charges : web-cloud/email-exchange - Modal Modifier un groupe

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un groupe de distribution
- **Type** : Formulaire de modification
- **Objectif** : Modifier les paramètres d'un groupe Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Email du groupe (lecture seule)
- Formulaire avec nom et options
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsInput** : Nom, nom affiché
- **OdsCheckbox** : Options d'adhésion
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le groupe"

## 5. CHAMPS
| Champ | Type | Modifiable |
|-------|------|------------|
| Adresse e-mail du groupe | Input disabled | Non |
| Nom du groupe | Input text | Oui (*) |
| Nom affiché | Input text | Oui |

## 6. OPTIONS D'ADHÉSION
| Option | Description |
|--------|-------------|
| Adhésion sur demande | Les utilisateurs peuvent demander à rejoindre |
| Cacher dans l'annuaire | Masque le groupe dans l'annuaire global |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Modifier | Primaire bleu | Sauvegarde les modifications |

## 8. APIS & ENDPOINTS
- PUT /email/exchange/{org}/service/{service}/mailingList/{mailingList} : Modifier groupe
