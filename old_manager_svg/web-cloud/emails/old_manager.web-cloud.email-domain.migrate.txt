# Cahier des Charges : web-cloud/email-domain - Migration

## 1. VUE D'ENSEMBLE
- **Type** : Wizard migration compte email
- **URL** : /email/domain/{domain}/account/{account}/migrate
- **Objectif** : Migrer un compte vers Email Pro ou Exchange

## 2. STRUCTURE PAGE
- Bouton retour
- Lien guides
- Info compte email
- Stepper 3 étapes
- Zone formulaire

## 3. COMPOSANTS UI
- **OdsLink** : Retour, guides
- **OdsText** : Labels
- **OdsStepper** : Étapes migration
- **OdsCard** : Sélection service
- **OdsSelect** : Choix service destination
- **OdsButton** : Navigation

## 4. STEPPER
| Étape | Label | État |
|-------|-------|------|
| 1 | Destination | Active (bleu) |
| 2 | Informations | Inactive |
| 3 | Confirmation | Inactive |

## 5. SERVICES DESTINATION
| Service | Description |
|---------|-------------|
| EMAIL PRO | Carte sélectionnable |
| HOSTED EXCHANGE | Carte sélectionnable |
| PRIVATE EXCHANGE | Carte sélectionnable |

## 6. FORMULAIRE STEP 1
| Champ | Type |
|-------|------|
| Type service | Cards radio |
| Service | Select dropdown |

## 7. BOUTONS NAVIGATION
| Bouton | Position | Style |
|--------|----------|-------|
| Annuler | Gauche | Secondary |
| Suivant | Droite | Primary |

## 8. APIS & ENDPOINTS
- GET /email/domain/{domain}/account/{account} - Info compte
- GET /email/pro - Liste services Email Pro
- GET /email/exchange - Liste services Exchange
- POST /email/domain/{domain}/account/{account}/migrate - Lancer migration
