**Type:** Modal wizard (étapes)
**Fonction:** Créer un nouveau compte email
**Contexte:** Action depuis la toolbar de l'onglet Boîtes mail

- Position: centrée avec overlay

## 6. APIS & ENDPOINTS

```
# Vérifier licences disponibles
GET /email/exchange/{organization}/service/{serviceName}/account
→ Nombre de comptes vs quota

GET /email/exchange/{organization}/service/{serviceName}
→ Infos service (maxReceiveSize, maxSendSize, quota)

# Créer compte Exchange
POST /email/exchange/{organization}/service/{serviceName}/account
Body: {
  login: "jean",
  displayName: "Jean Dupont",
  password: "SecureP@ss123",
  company: "ACME Corp"
}

# Créer compte Email Pro
POST /email/pro/{service}/account
Body: {
  primaryEmailAddress: "jean@example.com",
  password: "SecureP@ss123",
  displayName: "Jean Dupont"
}

# Créer compte Zimbra
POST /email/zimbra/{serviceName}/account
Body: {
  email: "jean@example.com",
  password: "SecureP@ss123"
}

# Commander licence supplémentaire
POST /order/cart
POST /order/cart/{cartId}/exchange
POST /order/cart/{cartId}/checkout
```
