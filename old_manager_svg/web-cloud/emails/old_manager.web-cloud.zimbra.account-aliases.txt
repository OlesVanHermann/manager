# Cahier des Charges : web-cloud/zimbra - Alias compte

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion des alias d'un compte Zimbra.

### Objectif fonctionnel
- Voir la liste des alias du compte
- Ajouter de nouveaux alias
- Supprimer des alias existants

## 3. COMPOSANTS UI

### Toolbar
| Element | Description |

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/account/{accountId}/alias
POST /zimbra/platform/{platformId}/account/{accountId}/alias
DELETE /zimbra/platform/{platformId}/account/{accountId}/alias/{aliasId}
```

### Modele
```typescript
interface ZimbraAlias {
  id: string;
  alias: string;
  state: 'ok' | 'suspended';
}
```
