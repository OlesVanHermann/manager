# Cahier des Charges : web-cloud/web-office - Modal Changer licence

## 1. VUE D'ENSEMBLE
- **Page** : Changer de licence
- **Type** : Modal formulaire
- **Objectif** : Modifier la licence d'un utilisateur

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Info utilisateur (readonly)
- Licence actuelle
- Options radio nouvelles licences
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Utilisateur (disabled)
- **OdsText** : Licence actuelle
- **OdsRadio** : Options licence
- **OdsButton** : Annuler, Confirmer

## 4. INFO UTILISATEUR
| Champ | Type | Valeur |
|-------|------|--------|
| Label | Text | Utilisateur |
| Champ | Input disabled | user@domain.com |

## 5. LICENCE ACTUELLE
| Élément | Valeur exemple |
|---------|----------------|
| Label | Licence actuelle : |
| Valeur | Microsoft 365 Business Basic |

## 6. OPTIONS LICENCE
| Option | Prix | Sélection |
|--------|------|-----------|
| Business Basic | 5,60€/mois | ○ |
| Business Standard | 11,70€/mois | ● Sélectionné |
| Business Premium | 20,60€/mois | ○ |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Confirmer | Primary | Droite |

## 8. APIS & ENDPOINTS
- PUT /license/office/{service}/user/{id}/changeLicense - Changer licence
