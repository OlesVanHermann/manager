## 8. APIS & ENDPOINTS

```
# Exchange - Disclaimer
GET /email/exchange/{organization}/service/{serviceName}/domain/{domain}/disclaimer
→ Configuration disclaimer actuelle

PUT /email/exchange/{organization}/service/{serviceName}/domain/{domain}/disclaimer
Body: {
  outsideMessageOnly: false,
  content: "<html>...</html>"
}
→ Met à jour la signature

# Désactiver
DELETE /email/exchange/{organization}/service/{serviceName}/domain/{domain}/disclaimer

# Prévisualisation
POST /email/exchange/{organization}/service/{serviceName}/domain/{domain}/disclaimer/preview
Body: {
  content: "Texte avec %%DISPLAY_NAME%%...",
  accountEmail: "jean@example.com"
}
→ Renvoie le HTML avec variables remplacées
```
