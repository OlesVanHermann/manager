# Cahier des Charges : web-cloud/email-pro - Modal Import

## 1. VUE D'ENSEMBLE
- **Modal** : Import de données depuis un fichier
- **Type** : Modal formulaire avec upload
- **Objectif** : Importer des emails depuis un fichier PST/MBOX

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau info
- Compte de destination (lecture seule)
- Type de source
- Zone upload fichier
- Boutons Annuler / Importer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x360)
- **OdsMessage** : Info bleu
- **OdsInput** : Compte (désactivé)
- **OdsSelect** : Type de source
- **OdsUpload** : Zone glisser-déposer
- **OdsButton** : Annuler, Importer

## 4. TITRE
"Importer des données"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Importez un fichier PST ou MBOX pour récupérer vos emails depuis un autre service."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Description |
|-------|------|-------------|
| Compte de destination | Input désactivé | Email cible |
| Type de source | Select | PST (Outlook), MBOX |

## 7. ZONE UPLOAD
| Élément | Description |
|---------|-------------|
| Style | Bordure pointillée |
| Texte | Glisser-déposer ou cliquer pour sélectionner |
| Actions | Drag & drop, click to browse |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Importer | Primaire bleu | Lance l'import |

## 9. APIS & ENDPOINTS
- POST /email/pro/{service}/account/{email}/import : Lancer import
  - file: binary
  - format: "pst" | "mbox"
