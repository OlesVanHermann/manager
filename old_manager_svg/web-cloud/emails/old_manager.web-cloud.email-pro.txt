# Cahier des Charges : web-cloud/email-pro - Dashboard

## 1. RESUME EXECUTIF

### Perimetre
Dashboard principal d'un service Email Pro.

### Objectif fonctionnel
- Voir les informations du service
- Gerer les comptes email
- Acceder aux onglets (domaines, taches)

## 4. APIS & DONNEES

### Endpoints
```
GET /email/pro/{service}
GET /email/pro/{service}/account
POST /email/pro/{service}/account
DELETE /email/pro/{service}/account/{email}
```

### Modele
```typescript
interface EmailProService {
  domain: string;
  hostname: string;
  offer: string;
  maxAccounts: number;
  state: string;
}

interface EmailProAccount {
  primaryEmailAddress: string;
  displayName: string;
  quota: number;
  currentUsage: number;
  lastLogonDate: string;
  state: 'ok' | 'suspended';
}
```
