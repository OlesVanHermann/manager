# Cahier des Charges : web-cloud/email-exchange - Modal Restriction IP

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration des restrictions IP
- **Type** : Formulaire de paramètres
- **Objectif** : Limiter l'accès Exchange à certaines IP

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau explicatif
- Activation de la restriction
- Liste des IP autorisées
- Formulaire d'ajout
- Avertissement
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsMessage** : Info bleu, warning jaune
- **OdsCheckbox** : Activer restriction
- **OdsInput** : Adresses IP
- **OdsButton** : Ajouter (+), Supprimer (×), Annuler, Enregistrer

## 4. TITRE
"Restriction IP"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Limitez l'accès au compte Exchange à certaines adresses IP pour renforcer la sécurité."

## 6. ACTIVATION
☑ "Activer la restriction IP"

## 7. ADRESSES IP AUTORISÉES
| IP/CIDR | Action |
|---------|--------|
| 192.168.1.0/24 | × (supprimer) |
| 10.0.0.50 | × (supprimer) |

## 8. FORMULAIRE D'AJOUT
| Champ | Type | Placeholder |
|-------|------|-------------|
| IP ou plage CIDR | Input text | Ajouter une IP ou plage CIDR |
| Ajouter | Bouton + | - |

## 9. AVERTISSEMENT
Fond jaune (#FEF3C7) :
"Attention: Vous pourriez vous bloquer si votre IP n'est pas incluse."

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde les restrictions |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/ipRestriction : Lister restrictions
- POST /email/exchange/{org}/service/{service}/ipRestriction : Ajouter IP
- DELETE /email/exchange/{org}/service/{service}/ipRestriction/{ip} : Supprimer IP
