# Cahier des Charges : web-cloud/email-exchange - Modal Configuration MX automatique

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique des MX
- **Type** : Modal de confirmation avec preview
- **Objectif** : Configurer automatiquement les enregistrements MX

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine concerné
- Bandeau explicatif
- MX actuels
- Boutons Annuler / Configurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x320)
- **OdsMessage** : Info bleu
- **OdsCard** : MX actuels
- **OdsButton** : Annuler, Configurer

## 4. TITRE
"Configuration MX automatique"

## 5. DOMAINE
Affiché en gras : "example.com"

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Cette action va configurer automatiquement les enregistrements MX pour pointer vers Exchange."

## 7. ENREGISTREMENTS MX ACTUELS
| Priorité | Serveur |
|----------|---------|
| 10 | mx1.mail.ovh.net |
| 20 | mx2.mail.ovh.net |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Configurer | Primaire bleu | Configure les MX |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/domain/{domain}/updateDns : Configurer MX
