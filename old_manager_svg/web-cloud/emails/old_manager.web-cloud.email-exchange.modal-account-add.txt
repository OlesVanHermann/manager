# Cahier des Charges : web-cloud/email-exchange - Modal Ajouter un compte

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un compte Exchange
- **Type** : Formulaire de création
- **Objectif** : Créer un nouveau compte email Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire de création avec champs
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x490)
- **OdsInput** : Champs texte et password
- **OdsSelect** : Domaine, Type de licence
- **OdsButton** : Annuler (secondaire), Créer (primaire)

## 4. TITRE
"Ajouter un compte Exchange"

## 5. FORMULAIRE
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Adresse e-mail | Input + Select | Oui (*) | Partie locale + domaine |
| Nom affiché | Input text | Oui (*) | Nom visible dans les mails |
| Prénom | Input text | Non | Prénom de l'utilisateur |
| Nom | Input text | Non | Nom de l'utilisateur |
| Mot de passe | Input password | Oui (*) | Mot de passe initial |
| Confirmer | Input password | Oui (*) | Confirmation mot de passe |
| Type de licence | Select dropdown | Non | Standard par défaut |

## 6. CHAMP ADRESSE EMAIL
Composé de :
- Input texte pour la partie locale (placeholder: "utilisateur")
- Séparateur "@"
- Select dropdown pour le domaine (ex: example.com)

## 7. TYPES DE LICENCE
| Type | Description |
|------|-------------|
| Standard | Licence Exchange standard |
| Basic | Licence Exchange basique |

## 8. PLACEHOLDERS
| Champ | Placeholder |
|-------|-------------|
| Adresse | utilisateur |
| Nom affiché | Prénom Nom |
| Mot de passe | •••••••• |
| Confirmer | •••••••• |

## 9. BOUTONS
| Bouton | Type | Position | Action |
|--------|------|----------|--------|
| Annuler | Secondaire (bordure) | Bas gauche | Ferme la modal |
| Créer | Primaire bleu | Bas droite | Crée le compte |

## 10. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account : Créer compte
- GET /email/exchange/{org}/service/{service}/domain : Lister domaines disponibles
