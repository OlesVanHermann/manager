# Cahier des Charges : web-cloud/email-pro - Modal Modifier un compte

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un compte Email Pro
- **Type** : Modal formulaire
- **Objectif** : Modifier les propriétés d'un compte existant

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte (lecture seule)
- Formulaire de modification
- Lien mot de passe
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x490)
- **OdsText** : Label compte (lecture seule)
- **OdsInput** : Champs texte
- **OdsCheckbox** : Option annuaire
- **OdsLink** : Modifier le mot de passe
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Modifier le compte"

## 5. COMPTE (LECTURE SEULE)
| Label | Valeur |
|-------|--------|
| Compte | admin@example.com |

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Description |
|-------|------|-------------|
| Nom affiché | Input | Nom complet |
| Prénom | Input | Prénom |
| Nom | Input | Nom de famille |
| Quota (Go) | Input number | Quota en Go |
| Cacher dans l'annuaire | Checkbox | Option visibility |

## 7. QUOTA
| Élément | Description |
|---------|-------------|
| Input | Valeur actuelle (ex: 5) |
| Texte info | / 50 Go disponibles |

## 8. LIEN MOT DE PASSE
"Modifier le mot de passe →"
Lien bleu vers modal changement mot de passe

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Enregistrer | Primaire bleu | Sauvegarde les modifications |

## 10. APIS & ENDPOINTS
- GET /email/pro/{service}/account/{email} : Récupérer compte
- PUT /email/pro/{service}/account/{email} : Modifier compte
  - displayName: string
  - firstName: string
  - lastName: string
  - quota: number
  - hiddenFromGAL: boolean
