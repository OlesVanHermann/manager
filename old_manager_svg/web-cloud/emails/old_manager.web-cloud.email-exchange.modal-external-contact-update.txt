# Cahier des Charges : web-cloud/email-exchange - Modal Modifier un contact externe

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un contact externe
- **Type** : Formulaire de modification
- **Objectif** : Modifier les informations d'un contact externe

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Email (lecture seule)
- Formulaire avec champs modifiables
- Option visibilité
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsInput** : Champs texte
- **OdsCheckbox** : Cacher dans l'annuaire
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le contact externe"

## 5. CHAMPS
| Champ | Type | Modifiable |
|-------|------|------------|
| Adresse e-mail externe | Input disabled | Non |
| Nom affiché | Input text | Oui (*) |
| Prénom | Input text | Oui |
| Nom | Input text | Oui |
| Organisation | Input text | Oui |

## 6. OPTIONS
☐ "Cacher dans l'annuaire"

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Modifier | Primaire bleu | Sauvegarde les modifications |

## 8. APIS & ENDPOINTS
- PUT /email/exchange/{org}/service/{service}/externalContact/{id} : Modifier contact
