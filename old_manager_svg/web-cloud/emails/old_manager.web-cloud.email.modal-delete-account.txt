# Cahier des Charges : web/email - Modal supprimer compte email

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation avec warning
- **Contexte** : Gestion email
- **Objectif** : Supprimer un compte email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, question
- **OdsMessage** : Warning perte de données
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le compte email" |

## 5. QUESTION DE CONFIRMATION
| Texte |
|-------|
| "Voulez-vous supprimer le compte {email} ?" |

## 6. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠ Attention" |
| | "Tous les emails de ce compte seront supprimés définitivement." |
| | "Pensez à sauvegarder vos messages avant de continuer." |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Supprimer | Danger (rouge) | Supprime le compte |

## 8. APIS & ENDPOINTS
- DELETE /email/domain/{domain}/account/{accountName} - Supprimer compte email
