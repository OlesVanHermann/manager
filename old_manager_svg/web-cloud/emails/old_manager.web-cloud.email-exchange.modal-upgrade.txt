# Cahier des Charges : web-cloud/email-exchange - Modal Changer d'offre

## 1. VUE D'ENSEMBLE
- **Modal** : Changer d'offre Exchange
- **Type** : Formulaire de sélection
- **Objectif** : Upgrader vers une offre supérieure

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Offre actuelle (info)
- Sélection nouvelle offre
- Avertissement délai
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsCard** : Offre actuelle
- **OdsRadio** : Options d'offres (cartes)
- **OdsMessage** : Warning délai
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Changer d'offre"

## 5. OFFRE ACTUELLE
| Élément | Valeur exemple |
|---------|----------------|
| Nom | Exchange Hosted |
| Détails | 3 comptes • 150 Go de stockage total |

## 6. NOUVELLES OFFRES
| Offre | Sélection | Description | Prix |
|-------|-----------|-------------|------|
| Exchange Provider | ● Sélectionné | Comptes illimités • API complète • White label | +15€/mois |
| Exchange Dedicated | ○ | Serveur dédié • Personnalisation complète | +50€/mois |

## 7. AVERTISSEMENT
Fond jaune (#FEF3C7) :
"Le changement d'offre sera effectif sous 24h."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Confirmer | Primaire bleu | Confirme le changement |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/upgrade : Offres disponibles
- POST /email/exchange/{org}/service/{service}/upgrade : Commander upgrade
