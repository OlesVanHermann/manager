# Cahier des Charges : web-cloud/email-domain - Onglet Redirections

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des redirections email.

### Objectif fonctionnel
- Creer des redirections automatiques
- Choisir de conserver ou non une copie
- Gerer les redirections actives

## 4. APIS & DONNEES

### Endpoints
```
GET /email/domain/{domain}/redirection
GET /email/domain/{domain}/redirection/{id}
POST /email/domain/{domain}/redirection
DELETE /email/domain/{domain}/redirection/{id}
```

## 6. REGLES METIER

- Redirection = transfert automatique
- Copie locale = garde une copie sur le serveur
- Sans copie = email uniquement sur destination
- Destination interne ou externe
