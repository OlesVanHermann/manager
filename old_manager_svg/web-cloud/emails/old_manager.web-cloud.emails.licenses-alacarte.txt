## 6. APIS & ENDPOINTS

```
# Catalogue produits
GET /order/catalog/formatted/email
→ Liste des offres email disponibles

# Commande licence
POST /order/cart
Body: { ovhSubsidiary: "FR" }
→ Crée un panier

POST /order/cart/{cartId}/exchange
Body: {
  domain: "example.com",
  offer: "exchange",
  duration: "P1M"
}
→ Ajoute une licence Exchange

POST /order/cart/{cartId}/emailPro
→ Ajoute une licence Email Pro

POST /order/cart/{cartId}/zimbra
→ Ajoute une licence Zimbra

# Options
POST /email/exchange/{organization}/service/{serviceName}/account/{email}/archive
→ Active l'archivage

POST /order/cart/{cartId}/emailDomain/options
Body: { option: "antispam" }
→ Commande Anti-spam Pro
```
