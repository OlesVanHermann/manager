# Cahier des Charges : web-cloud/web-office - Onglet Consommation

## 1. VUE D'ENSEMBLE
- **Page** : Microsoft 365 - Onglet Consommation
- **Type** : Onglet avec statistiques
- **Objectif** : Visualiser la consommation et facturation

## 2. STRUCTURE PAGE
- Carte info service
- Onglets (Consommation actif)
- Sélecteur période
