# Cahier des Charges : web-cloud/web-office - Modal Supprimer utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Suppression d'un utilisateur
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un utilisateur Microsoft 365

## 2. STRUCTURE MODAL
- Icône warning dans cercle rouge
- Titre
- Question confirmation
- Email utilisateur (rouge bold)
- Bandeau warning irréversible
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsText** : Titre, confirmation
- **OdsMessage** : Warning irréversible
- **OdsButton** : Annuler, Supprimer

## 4. ICÔNE WARNING
| Propriété | Valeur |
|-----------|--------|
| Cercle | Rouge clair (#FEE2E2) |
| Icône | ⚠ rouge |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Titre | Suppression d'un utilisateur |
| Question | Êtes-vous sûr de vouloir supprimer l'utilisateur : |
| Email | jean.dupont@monentreprise.onmicrosoft.com (rouge bold) |

## 6. MESSAGE WARNING
| Type | Couleur | Lignes |
|------|---------|--------|
| Warning | Orange | Cette action est irréversible. Toutes les données associées |
| | | à cet utilisateur seront définitivement supprimées. |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Supprimer | Danger (rouge) | Droite |

## 8. APIS & ENDPOINTS
- DELETE /license/office/{service}/user/{id} - Supprimer utilisateur
