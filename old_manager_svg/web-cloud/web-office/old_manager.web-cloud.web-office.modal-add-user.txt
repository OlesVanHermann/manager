# Cahier des Charges : web-cloud/web-office - Modal Ajouter utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un utilisateur
- **Type** : Modal formulaire
- **Objectif** : Créer un nouvel utilisateur Microsoft 365

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau warning facturation
- Formulaire (identifiant, email, licence, domaine)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Warning facturation
- **OdsInput** : Identifiant, email
- **OdsSelect** : Licence, domaine
- **OdsButton** : Annuler, Ajouter

## 4. MESSAGE WARNING
| Type | Couleur | Message |
|------|---------|---------|
| Warning | Orange | Vous serez facturé sur le nombre de licences activées dans le mois en cours. |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Identifiant | Input | Oui | jean.dupont (3-20 car.) |
| Email d'activation | Input email | Oui | exemple@domaine.com |
| Licence | Select | Oui | Microsoft 365 Business Basic |
| Domaine | Select | Non | @monentreprise.onmicrosoft.com |

## 6. RÈGLES IDENTIFIANT
- 3 à 20 caractères
- Lettres, chiffres, - et _ autorisés

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /license/office/{service}/user - Créer utilisateur
