# Cahier des Charges : web-cloud/database - Page List

## 1. VUE D'ENSEMBLE
- **Type** : Page liste bases de données
- **Contexte** : Web Cloud Databases
- **Objectif** : Vue consolidée de toutes les bases de données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et sous-titres
- **OdsButton** : CTA nouvelle base
- **OdsCard** : Cartes métriques
- **OdsInput** : Recherche
- **OdsSelect** : Filtre type
- **OdsDatagrid** : Liste bases
- **OdsBadge** : Type DB et statut

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bases de données" |
| Sous-titre | "MySQL, MariaDB, PostgreSQL, Redis" |
| Bouton | "+ Nouvelle base de données" |

## 5. CARTES MÉTRIQUES (4 colonnes)
| Carte | Label | Valeur |
|-------|-------|--------|
| MySQL/MariaDB | Nombre | "XX bases de données" |
| PostgreSQL | Nombre | "XX bases de données" |
| Redis/Cache | Nombre | "XX instances" |
| Taille totale | "XX Go" | "sur XX Go alloués" |

## 6. FILTRES
| Élément | Type |
|---------|------|
| Recherche | Input avec placeholder "Rechercher une base..." |
| Type | Select avec options (MySQL, PostgreSQL, Redis...) |

## 7. DATAGRID
| Colonne | Description |
|---------|-------------|
| Nom | Nom de la base (lien bleu) |
| Type | Badge coloré selon moteur |
| Serveur | Hostname (monospace) |
| Taille | Utilisation disque |
| Version | Version du moteur |
| Hébergement | Service associé |
| Statut | Badge Actif/Inactif |

## 8. BADGES TYPE DATABASE
| Type | Fond | Texte |
|------|------|-------|
| MySQL | #F59E0B | white |
| PostgreSQL | #336791 | white |
| Redis | #DC382D | white |

## 9. APIS & ENDPOINTS
- GET /hosting/privateDatabase - Liste CloudDB
- GET /hosting/web/{serviceName}/database - DBs hébergement
