# Cahier des Charges : web-cloud/email - Modal Créer liste de diffusion

## 1. VUE D'ENSEMBLE
- **Page** : Créer une liste de diffusion
- **Type** : Modal formulaire
- **Objectif** : Créer une mailing list

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ nom de la liste + domaine
- Select administrateur
- Select réponse vers
- Checkboxes options
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom liste
- **OdsSelect** : Administrateur, réponse vers
- **OdsCheckbox** : Options modération
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom de la liste | Input + suffix | Oui | equipe + @domain.com |
| Administrateur | Select | Oui | admin@domain.com |
| Réponse vers | Select | Non | La liste |

## 5. OPTIONS RÉPONSE
| Option |
|--------|
| La liste |
| L'expéditeur |
| L'administrateur |

## 6. OPTIONS MODÉRATION
| Option | État par défaut | Description |
|--------|-----------------|-------------|
| Modération activée | Non coché | Validation manuelle |
| Restreinte aux membres | ✓ Coché | Seuls les membres postent |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /email/domain/{domain}/mailingList - Créer liste
