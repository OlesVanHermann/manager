# Cahier des Charges : web-cloud/email - Modal Créer compte

## 1. VUE D'ENSEMBLE
- **Page** : Créer un compte email
- **Type** : Modal formulaire
- **Objectif** : Créer un compte email sur domaine

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ adresse email + domaine
- Champ description
- Champ mot de passe
- Select taille boîte
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Email, description, mot de passe
- **OdsSelect** : Taille boîte
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Adresse email | Input + suffix | Oui | contact + @domain.com |
| Description | Input | Non | Service commercial |
| Mot de passe | Password | Oui | ••••••••••• |
| Taille boîte | Select | Non | 5 Go (défaut) |

## 5. OPTIONS TAILLE
| Valeur |
|--------|
| 5 Go |
| 10 Go |
| 25 Go |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /email/domain/{domain}/account - Créer compte
