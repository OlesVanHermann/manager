# Cahier des Charges : web-cloud/email - Modal Créer filtre

## 1. VUE D'ENSEMBLE
- **Page** : Créer un filtre
- **Type** : Modal formulaire
- **Objectif** : Créer une règle de filtrage email

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ nom du filtre
- Section condition (critère + opérateur + valeur)
- Section action (radio buttons)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom filtre, valeur
- **OdsSelect** : Critère, opérateur, dossier
- **OdsRadio** : Actions
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom du filtre | Input | Oui | Newsletter |
| Critère | Select | Oui | Sujet |
| Opérateur | Select | Oui | contient |
| Valeur | Input | Oui | newsletter |

## 5. OPTIONS CRITÈRE
| Critère |
|---------|
| Sujet |
| De |
| À |
| Corps |

## 6. OPTIONS OPÉRATEUR
| Opérateur |
|-----------|
| contient |
| ne contient pas |
| est égal à |
| commence par |

## 7. ACTIONS DISPONIBLES
| Action | Sélectionnée | Description |
|--------|--------------|-------------|
| Déplacer vers un dossier | ● | Select dossier (Newsletters) |
| Supprimer le message | ○ | - |
| Rediriger vers une adresse | ○ | - |

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 9. APIS & ENDPOINTS
- POST /email/domain/{domain}/account/{email}/filter - Créer filtre
