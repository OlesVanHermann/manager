# Cahier des Charges : web-cloud/email - Page Filters

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion filtres email
- **Contexte** : Web Cloud Email Services
- **Objectif** : Créer et gérer des règles de filtrage automatique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour
- **OdsButton** : Créer un filtre
- **OdsMessage** : Info drag & drop
- **OdsCard** : Liste filtres
- **OdsDatagrid** : Tableau filtres
- **OdsBadge** : Action, Statut

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour à {email}" |
| Titre | "Filtres email" |
| Sous-titre | "{email} • Gérez vos règles de filtrage" |
| Bouton | "+ Créer un filtre" |

## 5. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Texte | "ℹ️ Les filtres sont exécutés dans l'ordre affiché. Glissez pour réorganiser." |

## 6. DATAGRID FILTRES
| Colonne | Description |
|---------|-------------|
| Priorité | Icône drag (⋮⋮) + numéro |
| Nom du filtre | Nom texte (gras) |
| Condition | Règle (Sujet contient, De contient...) |
| Action | Badge couleur (Supprimer, Déplacer, Marquer) |
| Statut | Badge Actif/Inactif |
| Actions | Lien Modifier |

## 7. BADGES ACTION
| Action | Fond | Texte |
|--------|------|-------|
| Supprimer | #FEE2E2 | #DC2626 |
| Déplacer → Dossier | #EFF6FF | #0050D7 |
| Marquer important | #FEF3C7 | #F59E0B |

## 8. BADGES STATUT
| Statut | Fond | Texte |
|--------|------|-------|
| Actif | #DCFCE7 | #16A34A |
| Inactif | #F9FAFB | #6B7280 |

## 9. DRAG & DROP
| Propriété | Valeur |
|-----------|--------|
| Icône | ⋮⋮ (6 points) |
| Comportement | Réorganiser par glisser-déposer |
| Mise à jour | Priorité automatique |

## 10. APIS & ENDPOINTS
- GET /email/{type}/{serviceName}/account/{email}/filter - Liste filtres
- POST /email/{type}/{serviceName}/account/{email}/filter - Créer filtre
- PUT /email/{type}/{serviceName}/account/{email}/filter/{id} - Modifier
- PUT /email/{type}/{serviceName}/account/{email}/filter/changePriority - Réordonner
- DELETE /email/{type}/{serviceName}/account/{email}/filter/{id} - Supprimer
