# Cahier des Charges : web-cloud/email - Modal Quota

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification quota
- **Contexte** : Web Cloud Email Services
- **Objectif** : Modifier le quota d'un compte email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 340px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsProgressBar** : Utilisation actuelle
- **OdsRadio** : Sélection quota (style bouton)
- **OdsMessage** : Info quota disponible
- **OdsButton** : Annuler, Appliquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le quota" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. INFO COMPTE
| Champ | Valeur |
|-------|--------|
| Label | "COMPTE" |
| Email | Adresse email |

## 6. UTILISATION ACTUELLE
| Propriété | Valeur |
|-----------|--------|
| Label | "UTILISATION ACTUELLE" |
| Barre progression | Couleur selon % (orange >50%) |
| Format | "X.X Go / Y.Y Go (XX%)" |

## 7. SÉLECTION NOUVEAU QUOTA
| Quota | Style si sélectionné |
|-------|---------------------|
| 1 Go | Bordure bleue |
| 5 Go | Default |
| 10 Go | Bordure bleue épaisse |
| 25 Go | Default |

## 8. INFO QUOTA DISPONIBLE
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Icône | ℹ️ |
| Ligne 1 | "Quota total disponible : XX Go" |
| Ligne 2 | "Actuellement alloué : XX Go (XX Go restants)" |

## 9. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Appliquer | Primary #0050D7 | Droite |

## 10. APIS & ENDPOINTS
- PUT /email/{type}/{serviceName}/account/{email} - Modifier quota
