# Cahier des Charges : web-cloud/email - Modal Créer redirection

## 1. VUE D'ENSEMBLE
- **Page** : Créer une redirection
- **Type** : Modal formulaire
- **Objectif** : Créer une redirection email

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ adresse source + domaine
- Champ destination
- Checkbox copie locale
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Source, destination
- **OdsCheckbox** : Garder copie locale
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Adresse à rediriger | Input + suffix | Oui | source + @domain.com |
| Vers | Input email | Oui | destination@external.com |
| Copie locale | Checkbox | Non | Garder une copie locale (coché) |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 6. APIS & ENDPOINTS
- POST /email/domain/{domain}/redirection - Créer redirection
