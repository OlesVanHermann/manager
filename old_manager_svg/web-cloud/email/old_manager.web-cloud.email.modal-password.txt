# Cahier des Charges : web-cloud/email - Modal Mot de passe

## 1. VUE D'ENSEMBLE
- **Page** : Modifier le mot de passe
- **Type** : Modal formulaire
- **Objectif** : Changer le mot de passe d'un compte email

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ compte (lecture seule)
- Champ nouveau mot de passe
- Champ confirmation
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Compte (disabled), mot de passe
- **OdsButton** : Annuler, Modifier

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Compte | Input disabled | - | user@domain.com (readonly) |
| Nouveau mot de passe | Password | Oui | Nouveau MDP |
| Confirmation | Password | Oui | Confirmation MDP |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Modifier | Primary | Droite |

## 6. APIS & ENDPOINTS
- POST /email/domain/{domain}/account/{email}/changePassword - Changer MDP
