# Cahier des Charges : web-cloud/email - Page Detail

## 1. VUE D'ENSEMBLE
- **Type** : Page détail service email
- **Contexte** : Web Cloud Email Services
- **Objectif** : Gérer un service email spécifique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |
| Background | #F9FAFB |

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsLink** : Retour
- **OdsBadge** : Type et statut
- **OdsButton** : Actions
- **OdsCard** : Cartes métriques
- **OdsDatagrid** : Liste comptes
- **OdsProgressBar** : Espace utilisé
- **OdsPagination** : Navigation

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Retour | "← Retour aux services email" |
| Titre | Nom du service |
| Badges | Type (Exchange/Email Pro/MX Plan) + Statut |
| Boutons | Configurer, + Ajouter compte |

## 5. CARTES MÉTRIQUES (4 colonnes)
| Carte | Label | Affichage |
|-------|-------|-----------|
| Comptes | "XX / YY" | "boîtes actives" |
| Espace | "XXX / YYY Go" | Barre de progression |
| Domaine | Nom domaine | "✓ Configuré" |
| Expiration | Date | "Renouvellement auto" |

## 6. DATAGRID COMPTES
| Colonne | Description |
|---------|-------------|
| Adresse | Email complet (lien bleu) |
| Nom | Nom du propriétaire |
| Quota | "X.X/Y Go" (orange si >80%) |
| Dernière activité | "Il y a X min/h" |
| Statut | Badge Actif/Envoi seul |
| Actions | Menu ⋮ |

## 7. BADGES TYPE EMAIL
| Type | Fond | Texte |
|------|------|-------|
| Exchange | #7C3AED | white |
| Email Pro | #0050D7 | white |
| MX Plan | #16A34A | white |

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Affichage X-Y sur Z" |
| Boutons | ‹ Précédent, Numéros, Suivant › |

## 9. APIS & ENDPOINTS
- GET /email/{type}/{serviceName} - Détail service
- GET /email/{type}/{serviceName}/account - Liste comptes
- POST /email/{type}/{serviceName}/account - Créer compte
- DELETE /email/{type}/{serviceName}/account/{email} - Supprimer
