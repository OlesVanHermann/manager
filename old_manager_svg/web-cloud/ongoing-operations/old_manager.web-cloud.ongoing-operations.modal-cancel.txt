# Cahier des Charges : web-cloud/ongoing-operations - Modal Annuler

## 1. VUE D'ENSEMBLE
- **Page** : Annuler l'opération
- **Type** : Modal confirmation warning
- **Objectif** : Annuler une opération en cours

## 2. STRUCTURE MODAL
- Header warning orange + bouton fermer
- Message confirmation
- Info opération
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Warning
- **OdsText** : Info opération
- **OdsButton** : Conserver, Annuler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | Orange (#FEF3C7) |
| Titre | Annuler l'opération |

## 5. MESSAGE WARNING
| Type | Couleur | Lignes |
|------|---------|--------|
| Warning | Orange | Êtes-vous sûr de vouloir annuler cette opération ? |
| | | Cette action peut être irréversible. |

## 6. INFO OPÉRATION
| Élément | Valeur exemple |
|---------|----------------|
| Label | Opération : |
| Valeur | Transfert domain.com → OVHcloud |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Conserver | Outline | Gauche |
| Annuler | Danger (rouge) | Droite |
