# Cahier des Charges : web-cloud/ongoing-operations - Onglet DNS

## 1. VUE D'ENSEMBLE
- **Page** : Opérations en cours - DNS
- **Type** : Onglet liste
- **Objectif** : Afficher les opérations sur les zones DNS

## 2. STRUCTURE PAGE
- Titre + onglets (DNS actif)
- Filtre recherche
- Datagrid des opérations

## 3. COMPOSANTS UI
- **OdsTabs** : Domaine, DNS (actif), Alldom
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste opérations
- **OdsBadge** : Status
- **OdsActionMenu** : Actions

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| ZONE DNS | Link | Nom de zone |
| OPÉRATION | Text | Type opération DNS |
| ÉTAT | Badge | Statut |
| DATE | Text | Date |
| ACTIONS | ActionMenu | ⋮ |

## 5. TYPES OPÉRATIONS DNS
| Type | Description |
|------|-------------|
| Création zone DNS | Nouvelle zone |
| Activation DNSSEC | Sécurisation |
| Importation zone | Import depuis fichier |
| Modification entrées | Mise à jour records |

## 6. APIS & ENDPOINTS
- GET /domain/zone/operations - Liste opérations DNS
