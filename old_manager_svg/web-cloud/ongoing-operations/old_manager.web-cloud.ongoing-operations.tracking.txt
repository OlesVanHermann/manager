# Cahier des Charges : web-cloud/ongoing-operations - Suivi transfert

## 1. VUE D'ENSEMBLE
- **Page** : Suivi du transfert
- **Type** : Page détail avec stepper
- **Objectif** : Suivre les étapes d'un transfert de domaine

## 2. STRUCTURE PAGE
- Lien retour (← Retour aux opérations)
- Titre + nom de domaine
- Stepper vertical avec 4 étapes
- Message warning contextuel

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsStepper** : Étapes du transfert
- **OdsMessage** : Warning validation

## 4. ÉTAPES DU TRANSFERT
| Étape | Titre | Description |
|-------|-------|-------------|
| 1 | Création de l'opération | ✓ Demande de transfert initiée |
| 2 | Vérification des contacts | En attente de validation par email |
| 3 | Confirmation du registrar actuel | En attente |
| 4 | Finalisation du transfert | En attente |

## 5. STATUTS ÉTAPES
| Statut | Couleur | Description |
|--------|---------|-------------|
| Complété | Vert (✓) | Étape terminée |
| Actif | Bleu (2) | Étape en cours |
| En attente | Gris (3,4) | À venir |

## 6. MESSAGE WARNING
| Type | Couleur | Message |
|------|---------|---------|
| Warning | Orange | Un email de confirmation a été envoyé. |
| | | Veuillez valider le transfert dans les 5 jours. |

## 7. APIS & ENDPOINTS
- GET /domain/{domain}/operation/{id}/tracking - Suivi opération
