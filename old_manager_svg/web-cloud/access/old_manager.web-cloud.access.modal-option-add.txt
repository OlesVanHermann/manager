**Fonction:** Souscrire une option supplémentaire
**Contexte:** Enrichissement de l'offre accès Internet

## 9. APIS & ENDPOINTS

```
GET /xdsl/{serviceName}/availableOptions
→ Liste des options disponibles avec prix

POST /xdsl/{serviceName}/options
Body: {
  option: "staticIp" | "ipv6" | "ipFailover" | "gtr"
}
→ Souscrit l'option

GET /order/cart/{cartId}/xdslOption
→ Récupère le bon de commande
```
