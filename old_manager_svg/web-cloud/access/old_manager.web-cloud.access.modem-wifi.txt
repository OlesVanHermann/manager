## 8. APIS & ENDPOINTS

```
GET /xdsl/{serviceName}/modem/wifi
→ Configuration WiFi (par bande)

PUT /xdsl/{serviceName}/modem/wifi
Body: {
  band: "2.4GHz",
  enabled: true,
  ssid: "MonReseau",
  security: "WPA3",
  key: "password123",
  channel: "auto"
}
→ Applique la configuration

GET /xdsl/{serviceName}/modem/wifi/password
→ Affiche le mot de passe

POST /xdsl/{serviceName}/modem/wifi/password/generate
→ Génère un nouveau mot de passe aléatoire

GET /xdsl/{serviceName}/modem/connectedDevices?band=2.4GHz
→ Liste des appareils connectés sur cette bande
```
