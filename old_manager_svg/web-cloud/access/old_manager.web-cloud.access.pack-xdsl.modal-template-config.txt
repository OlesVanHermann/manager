# Cahier des Charges : web-cloud/access/pack-xdsl - Modal Template Config

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation suppression template
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression template modem

## 2. STRUCTURE MODAL
- Icône warning
- Message confirmation
- Nom du template
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal
- **OdsIcon** : Warning/Trash
- **OdsText** : Message, nom template
- **OdsButton** : Annuler, Supprimer

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | Auto |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le template" |
| Message | "Êtes-vous sûr de vouloir supprimer ce template ?" |
| Template | "template-bureau" (exemple) |
| Warning | "Cette action est irréversible." |

## 6. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary outline | "Annuler" |
| Supprimer | Danger (rouge) | "Supprimer" |

## 7. APIS & ENDPOINTS
- DELETE /xdsl/templateModem/{templateName} - Supprimer template
