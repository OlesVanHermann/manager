# Cahier des Charges : web-cloud/access/pack-xdsl - Line Diagnostic

## 1. VUE D'ENSEMBLE
- **Type** : Page diagnostic ligne étape par étape
- **URL** : /pack/{packId}/xdsl/{serviceName}/lineDiagnostic
- **Objectif** : Diagnostiquer problèmes de connexion

## 2. STRUCTURE PAGE
- En-tête avec titre
- Barre de progression étapes
- Zone questions/réponses
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsStepper** : Progression étapes
- **OdsText** : Questions et instructions
- **OdsRadio** : Choix réponses Oui/Non
- **OdsButton** : Précédent, Suivant
- **OdsSpinner** : Chargement diagnostic

## 4. BARRE PROGRESSION
| Propriété | Valeur |
|-----------|--------|
| Étape actuelle | 2/5 |
| Labels | "Modem", "Connexion", "Équipements", "Diagnostic", "Résultat" |
| Style actif | Bleu #0050D7 |

## 5. QUESTION COURANTE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Vérification de la connexion" |
| Question | "La LED DSL de votre modem est-elle allumée ?" |

## 6. OPTIONS RÉPONSE
| Option | Action |
|--------|--------|
| Oui | Passer à l'étape suivante |
| Non | Afficher conseils de dépannage |

## 7. BOUTONS NAVIGATION
| Bouton | Style | Label | État |
|--------|-------|-------|------|
| Précédent | Secondary outline | "Précédent" | Actif si étape > 1 |
| Suivant | Primary | "Suivant" | Actif après réponse |

## 8. ÉTAT DIAGNOSTIC EN COURS
| Propriété | Valeur |
|-----------|--------|
| Icône | OdsSpinner |
| Message | "Diagnostic en cours..." |
| Durée estimée | "Cette opération peut durer quelques minutes" |

## 9. APIS & ENDPOINTS
- POST /xdsl/{serviceName}/diagnostic - Lancer diagnostic
- GET /xdsl/{serviceName}/diagnostic - État diagnostic
- GET /xdsl/{serviceName}/diagnostic/result - Résultat
