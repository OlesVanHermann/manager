# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Reboot

## 1. VUE D'ENSEMBLE
- **Écran** : Redémarrage du modem
- **Objectif** : Redémarrer le modem à distance
- **Contexte** : Résolution problèmes de connectivité

## 2. LAYOUT & STRUCTURE
- Carte principale : Description + bouton
- Modale confirmation
- Indicateur redémarrage en cours

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsButton** : "Redémarrer" (primaire)
- **OdsMessage** : Warning interruption (jaune)
- **OdsModal** : Confirmation
- Spinner : Progression

## 4. SECTION PRINCIPALE
- Titre : "Redémarrer le modem"
- Description : "Un redémarrage peut résoudre certains problèmes de connectivité temporaires"
- Bouton : "Redémarrer" (bleu)

## 5. WARNING
- Message ligne 1 : "Votre connexion Internet sera interrompue"
- Message ligne 2 : "pendant environ 2-3 minutes"

## 6. MODALE CONFIRMATION
- Header : "Confirmer le redémarrage" + bouton fermer (×)
- Question : "Êtes-vous sûr de vouloir redémarrer le modem ?"
- Info : "La connexion sera interrompue pendant quelques minutes"
- Boutons : OK (primaire), Annuler (secondaire)

## 7. ÉTAT REDÉMARRAGE EN COURS
- Titre : "Redémarrage en cours..."
- Spinner + texte : "En attente du modem..."
- Sous-texte : "Veuillez patienter"

## 8. APIS & ENDPOINTS
- POST /xdsl/{serviceName}/modem/reboot : Lancer redémarrage
