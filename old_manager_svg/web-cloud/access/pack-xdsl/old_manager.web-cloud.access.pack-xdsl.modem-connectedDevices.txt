# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Connected Devices

## 1. VUE D'ENSEMBLE
- **Type** : Tableau équipements connectés
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem/connectedDevices
- **Objectif** : Visualiser appareils connectés au modem

## 2. STRUCTURE PAGE
- Titre section
- Bouton rafraîchir
- Tableau équipements

## 3. COMPOSANTS UI
- **OdsText** : Titre
- **OdsButton** : Rafraîchir
- **OdsDatagrid** : Tableau équipements
- **OdsBadge** : Type connexion

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Équipements connectés" |
| Style | Section heading |

## 5. BOUTON RAFRAÎCHIR
| Propriété | Valeur |
|-----------|--------|
| Label | "Actualiser" |
| Icône | Refresh |
| Style | Secondary |

## 6. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Texte | 200px |
| Adresse MAC | Texte mono | 180px |
| Adresse IP | Texte mono | 150px |
| Type | Badge | 100px |
| Connexion | Texte | 100px |

## 7. TYPES CONNEXION
| Type | Badge | Couleur |
|------|-------|---------|
| WiFi 2.4GHz | "2.4 GHz" | Bleu |
| WiFi 5GHz | "5 GHz" | Vert |
| Ethernet | "LAN" | Gris |

## 8. DONNÉES EXEMPLE
| Nom | MAC | IP | Type | Connexion |
|-----|-----|----|----|-----------|
| iPhone-Jean | AA:BB:CC:DD:EE:FF | 192.168.1.101 | 5 GHz | WiFi |
| PC-Bureau | 00:1A:2B:3C:4D:5E | 192.168.1.102 | LAN | Ethernet |

## 9. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/connectedDevices - Liste équipements
- POST /xdsl/{serviceName}/modem/connectedDevices/refresh - Actualiser liste
