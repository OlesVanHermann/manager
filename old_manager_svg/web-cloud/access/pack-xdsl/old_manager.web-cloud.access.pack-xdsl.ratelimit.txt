# Cahier des Charges : web-cloud/access/pack-xdsl - Ratelimit

## 1. VUE D'ENSEMBLE
- **Type** : Composant optimisation débit LNS
- **URL** : /pack/{packId}/xdsl/{serviceName}
- **Objectif** : Ajuster limite de débit au niveau LNS

## 2. STRUCTURE COMPOSANT
- Titre section
- Description
- Slider de débit
- Checkbox suppression limite
- Warning
- Bouton validation

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsRange** : Slider pourcentage
- **OdsCheckbox** : Supprimer limite
- **OdsMessage** : Warning interruption
- **OdsButton** : Valider

| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
