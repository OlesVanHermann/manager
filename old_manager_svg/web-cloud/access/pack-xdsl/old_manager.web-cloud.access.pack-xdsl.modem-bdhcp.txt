# Cahier des Charges : web-cloud/access/pack-xdsl - Modem BDHCP

## 1. VUE D'ENSEMBLE
- **Type** : Tableau baux DHCP
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem/dhcp/bdhcp
- **Objectif** : Gérer baux DHCP statiques

## 2. STRUCTURE PAGE
- Titre section
- Bouton ajout bail
- Tableau baux existants

## 3. COMPOSANTS UI
- **OdsText** : Titre
- **OdsButton** : Ajouter bail
- **OdsDatagrid** : Tableau baux
- **OdsActionMenu** : Actions par ligne

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Baux DHCP statiques" |
| Style | Section heading |

## 5. BOUTON AJOUT
| Propriété | Valeur |
|-----------|--------|
| Label | "Ajouter un bail" |
| Style | Primary |
| Icône | + |

## 6. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Texte | 200px |
| Adresse MAC | Texte mono | 180px |
| Adresse IP | Texte mono | 150px |
| Actions | ActionMenu | 80px |

## 7. DONNÉES EXEMPLE
| Nom | Adresse MAC | Adresse IP |
|-----|-------------|------------|
| Serveur NAS | 00:1A:2B:3C:4D:5E | 192.168.1.50 |
| Imprimante | 00:11:22:33:44:55 | 192.168.1.60 |

## 8. MENU ACTIONS
| Option | Icône | Action |
|--------|-------|--------|
| Modifier | Crayon | Ouvrir modal édition |
| Supprimer | Poubelle rouge | Confirmer suppression |

## 9. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/lan/{lanName}/dhcp/DHCPStaticAddresses - Liste baux
- POST /xdsl/{serviceName}/modem/lan/{lanName}/dhcp/DHCPStaticAddresses - Créer bail
- PUT /xdsl/{serviceName}/modem/lan/{lanName}/dhcp/DHCPStaticAddresses/{macAddress} - Modifier
- DELETE /xdsl/{serviceName}/modem/lan/{lanName}/dhcp/DHCPStaticAddresses/{macAddress} - Supprimer
