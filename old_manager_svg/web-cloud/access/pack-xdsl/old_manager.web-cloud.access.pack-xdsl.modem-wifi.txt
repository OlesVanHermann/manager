# Cahier des Charges : web-cloud/access/pack-xdsl - Modem WiFi

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des réseaux WiFi du modem
- **Objectif** : Afficher et gérer les réseaux WiFi configurés
- **Contexte** : Configuration WiFi du modem xDSL

## 2. LAYOUT & STRUCTURE
- Carte blanche avec titre
- Tableau des réseaux WiFi
- Bandeau info en bas

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsTable** : Liste des réseaux WiFi
- **OdsBadge** : État (Activé/Désactivé)
- **OdsActionMenu** : Menu ⋮ avec options
- **OdsMessage** : Bandeau info bleu

## 4. DONNÉES & COLONNES
| Colonne | Type | Description |
|---------|------|-------------|
| SSID | text (bold) | Nom du réseau WiFi |
| Visibilité | text | Visible / Masqué |
| Sécurité | text | WPA2-PSK, WPA3, Open |
| Canal | text | Auto (X) ou numéro fixe |
| Fréquence | text | 2.4 GHz / 5 GHz |
| Invité | text | Oui / Non |
| État | badge | Activé (vert) / Désactivé (rouge) |
| Actions | menu | Menu contextuel ⋮ |

## 5. MENU ACTIONS (⋮)
- Modifier → pack-xdsl.modem-wifi-config
- QR Code → Affiche QR pour connexion mobile

## 6. EXEMPLES RÉSEAUX
| SSID | Fréquence | Sécurité | Invité | État |
|------|-----------|----------|--------|------|
| MaBox-WiFi | 2.4 GHz | WPA2-PSK | Non | Activé |
| MaBox-WiFi-5G | 5 GHz | WPA3 | Non | Activé |
| MaBox-Guest | 2.4 GHz | WPA2-PSK | Oui | Désactivé |

## 7. BANDEAU INFO
- Message : "Cliquez sur le menu d'actions (⋮) pour modifier les paramètres ou afficher le QR Code de connexion."

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/wifi : Liste réseaux
- PUT /xdsl/{serviceName}/modem/wifi/{ssid} : Modifier réseau
