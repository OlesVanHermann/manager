# Cahier des Charges : web-cloud/access/pack-xdsl - Dashboard

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard principal d'un accès xDSL
- **Objectif** : Afficher les informations et navigation d'un accès Internet
- **Contexte** : Accès après sélection dans la liste des packs

## 2. LAYOUT & STRUCTURE
- Lien retour "← Retour au pack"
- Titre : référence xDSL + description
- Barre d'onglets (Ligne, Modem, Tâches)
- Zone contenu avec infos générales + actions rapides

## 3. COMPOSANTS UI
- **OdsLink** : Retour au pack
- **OdsText** : Titre avec référence et description
- **OdsTabs** : Navigation onglets
- **OdsCard** : Container contenu
- **OdsBadge** : Statut (Actif)
- **OdsButton** : Actions rapides (liens)

## 4. DONNÉES - Informations générales
| Champ | Type | Exemple |
|-------|------|---------|
| Référence | text | xdsl-abc123-1 |
| Type d'accès | text | VDSL |
| Statut | badge | Actif (vert) |
| Débit synchronisé | text | ↓ 45.2 Mbps / ↑ 8.5 Mbps |

## 5. ACTIONS RAPIDES
- Lancer un diagnostic → pack-xdsl.diagnostic
- Voir les statistiques → pack-xdsl.statistics
- Gérer les IP → pack-xdsl.ip

## 6. NAVIGATION ONGLETS
- Ligne (actif) → pack-xdsl.access
- Modem → pack-xdsl.modem
- Tâches → pack-xdsl.tasks

## 7. APIS & ENDPOINTS
- GET /xdsl/{serviceName} : Infos accès
