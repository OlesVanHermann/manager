# Cahier des Charges : web-cloud/access - Index

## 1. VUE D'ENSEMBLE

**Type:** Page index (alias de General)
**Fonction:** Point d'entrée des accès internet
**Contexte:** Vue par défaut lors de la sélection d'une connexion

## 2. DIMENSIONS

- Identique à general.svg
- Page: 1440x900px

## 3. COMPORTEMENT

Cette page est identique à `general.svg`. Elle sert de point d'entrée lorsqu'un utilisateur sélectionne une connexion dans la liste.

### Redirection
- URL: `/web-cloud/access/{serviceName}`
- Redirige vers: `/web-cloud/access/{serviceName}/general`
