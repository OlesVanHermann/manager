# Cahier des Charges : web-cloud/access/pack-xdsl - Modem MTU

## 1. VUE D'ENSEMBLE
- **Type** : Composant configuration MTU
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem
- **Objectif** : Configurer valeur MTU du modem

## 2. STRUCTURE COMPOSANT
- Titre section
- Description
- Sélecteur MTU
- Bouton validation

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsSelect** : Sélecteur valeur MTU
- **OdsButton** : Appliquer

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Configuration MTU" |
| Style | Section heading |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Le MTU (Maximum Transmission Unit) définit la taille maximale des paquets réseau. Modifiez cette valeur uniquement si vous rencontrez des problèmes de connexion." |

## 6. SÉLECTEUR MTU
| Propriété | Valeur |
|-----------|--------|
| Label | "Valeur MTU" |
| Type | OdsSelect |
| Options | 1432, 1456, 1480, 1492, 1500 |
| Défaut | 1492 |
| Largeur | 150px |

## 7. VALEURS MTU
| Valeur | Usage |
|--------|-------|
| 1432 | PPPoE sur ATM (ADSL) |
| 1492 | PPPoE standard |
| 1500 | Ethernet standard |

## 8. BOUTON
| Propriété | Valeur |
|-----------|--------|
| Label | "Appliquer" |
| Style | Primary |

## 9. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem - Valeur MTU actuelle
- PUT /xdsl/{serviceName}/modem - Modifier MTU
