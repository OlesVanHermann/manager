## 10. APIS & ENDPOINTS

```
GET /xdsl/{serviceName}/lines/{number}
→ Informations ligne (statut, débit)

GET /xdsl/{serviceName}/lines/{number}/statistics
→ Statistiques synchronisation

GET /xdsl/{serviceName}/ip
→ Adresses IP

POST /xdsl/{serviceName}/lines/{number}/resetDslStats
→ Resynchronise la ligne

POST /xdsl/{serviceName}/requestTotalDeconsolidation
→ Réinitialise la connexion
```
