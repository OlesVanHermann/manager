# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Templates Config

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion templates modem
- **URL** : /xdsl/templateModem
- **Objectif** : Gérer et appliquer templates configuration

## 2. STRUCTURE PAGE
- Liste des templates (haut)
- Détail template sélectionné (bas)
- Sections : DHCP, WiFi, DMZ/MTU, Ports

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste templates
- **OdsActionMenu** : Modifier, Supprimer
- **OdsCheckbox** : Ignorer sections
- **OdsText** : Labels et valeurs

## 4. TABLEAU TEMPLATES
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom du template | Lien | 300px |
| Date de création | Date | 200px |

## 5. MENU ACTIONS
| Option | Couleur | Action |
|--------|---------|--------|
| Modifier | Standard | Éditer template |
| Supprimer | Rouge | Confirmer suppression |

## 10. APIS & ENDPOINTS
- GET /xdsl/templateModem - Liste templates
- GET /xdsl/templateModem/{templateName} - Détail template
- POST /xdsl/templateModem - Créer template
- PUT /xdsl/templateModem/{templateName} - Modifier template
- DELETE /xdsl/templateModem/{templateName} - Supprimer template
- POST /xdsl/{serviceName}/modem/applyTemplate - Appliquer template
