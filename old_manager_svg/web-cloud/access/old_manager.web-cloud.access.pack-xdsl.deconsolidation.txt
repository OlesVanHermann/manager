# Cahier des Charges : web-cloud/access/pack-xdsl - Déconsolidation

## 1. VUE D'ENSEMBLE
- **Écran** : Déconsolidation de la ligne xDSL
- **Objectif** : Passer en dégroupage total
- **Contexte** : Amélioration débit et tarifs

## 2. LAYOUT & STRUCTURE
- Carte principale : Formulaire déconsolidation
- Carte état alternatif (déjà en dégroupage total)

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsMessage** : Warning offre historique (jaune)
- **OdsMessage** : Info importante (bleu)
- **OdsInput** : Code RIO
- **OdsButton** : "Valider la demande" (primaire)

## 4. WARNING OFFRE HISTORIQUE
- Titre : "⚠ Offre historique"
- Message : "Pour passer en dégroupage total, vous devez d'abord migrer vers une offre récente."

## 5. SECTION CODE RIO
- Titre : "Code RIO"
- Input : "Entrez votre code RIO"
- Aide : "Pour obtenir votre RIO, appelez le 3179 depuis votre ligne fixe."

## 6. BANDEAU INFO
- Titre : "ℹ Information importante"
- Point 1 : "La déconsolidation totale permet de bénéficier d'un meilleur débit"
- Point 2 : "et de tarifs plus avantageux. Votre numéro de téléphone sera conservé."
- Point 3 : "Délai de traitement : environ 10 jours ouvrés."

## 7. ÉTAT DÉJÀ DÉGROUPÉ
- Titre : "État : Dégroupage total"
- Warning : "⚠ Ligne déjà en dégroupage total"
- Message : "Aucune action de déconsolidation n'est disponible."

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/eligibility : Éligibilité déconsolidation
- POST /xdsl/{serviceName}/totalDeconsolidation : Lancer déconsolidation
