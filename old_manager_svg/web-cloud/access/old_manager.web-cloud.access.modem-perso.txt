## 10. APIS & ENDPOINTS

```
GET /xdsl/{serviceName}/modem/personal
→ Informations modem déclaré

PUT /xdsl/{serviceName}/modem/personal
Body: {
  brand: "Ubiquiti",
  model: "UniFi Dream Machine Pro",
  macAddress: "AA:BB:CC:DD:EE:FF",
  serialNumber: "UDM-PRO-12345678"
}
→ Met à jour les informations

DELETE /xdsl/{serviceName}/modem/personal
→ Supprime la déclaration (retour modem OVH)

GET /xdsl/{serviceName}/pppLogin
→ Identifiant PPPoE

GET /xdsl/{serviceName}/pppPassword
→ Affiche le mot de passe PPPoE

POST /xdsl/{serviceName}/pppPassword/reset
→ Régénère le mot de passe

GET /xdsl/{serviceName}/vlanConfiguration
→ Configuration VLAN (Internet, TV, VoIP)
```
