# Cahier des Charges : web-cloud/access/overthebox - Configure

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration équipement
- **URL** : /overthebox/configure
- **Objectif** : Rattacher équipement après redémarrage ou nouvelle commande

## 2. STRUCTURE PAGE
- Carte blanche avec bordure
- Titre principal
- Deux sections de configuration

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur principal
- **OdsMessage** : Sections info gris
- **OdsLink** : Liens d'action

## 4. SECTION 1: REDÉMARRAGE
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Dimensions | 740x140px |
| Titre | "Vous avez redémarré votre overTheBox" |
| Message | "Votre équipement a changé son identifiant..." |
| Action | Lien vers service existant (ex: otb-abc123-xyz) |

## 5. SECTION 2: NOUVELLE COMMANDE
| Propriété | Valeur |
|-----------|--------|
| Fond | #F9FAFB |
| Dimensions | 740x140px |
| Titre | "Vous avez reçu votre nouvelle overTheBox" |
| Message | "Vous avez fait l'acquisition d'un nouvel équipement" |
| Action | Lien "souscrire à un nouveau service" |

## 6. LIENS D'ACTION
| Lien | Couleur | Destination |
|------|---------|-------------|
| Service existant | #0050D7 | /overthebox/{serviceId} |
| Nouveau service | #0050D7 | /overthebox/order |

## 7. APIS & ENDPOINTS
- GET /overthebox/services - Liste services disponibles
- POST /overthebox/{serviceId}/link - Rattacher équipement
