# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Firewall

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration du niveau de firewall
- **Objectif** : Définir le niveau de protection du modem
- **Contexte** : Sécurité réseau

## 2. LAYOUT & STRUCTURE
- 2 colonnes : Sélecteur niveau | Détails niveaux
- Dropdown avec descriptions

## 3. COMPOSANTS UI
- **OdsCard** : 2 cartes
- **OdsSelect** : Niveau firewall (avec descriptions)
- Cartes colorées : Descriptions par niveau

## 4. NIVEAUX DE FIREWALL
| Niveau | Couleur | Description |
|--------|---------|-------------|
| Désactivé | Rouge #FEE2E2 | Aucune protection (non recommandé) |
| Normal | Vert #D1FAE5 | Protection standard avec NAT/PAT (recommandé) |
| Élevé | Jaune #FEF3C7 | Bloque toutes les connexions entrantes |

## 5. SÉLECTEUR DROPDOWN
- Option sélectionnée : fond bleu, checkmark
- Chaque option : Titre + description courte

## 6. DÉTAILS DES NIVEAUX

### Désactivé (rouge)
- Toutes connexions entrantes autorisées
- Exposition directe à Internet

### Normal - recommandé (vert)
- Protection NAT/PAT active
- Redirections de ports possibles
- Équilibre sécurité/fonctionnalités

### Élevé (jaune)
- Toutes connexions entrantes bloquées
- Redirections de ports désactivées

## 7. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/firewall : Niveau actuel
- PUT /xdsl/{serviceName}/modem/firewall : Modifier niveau
