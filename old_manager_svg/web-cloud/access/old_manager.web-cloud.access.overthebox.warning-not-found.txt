# Cahier des Charges : web-cloud/access/overthebox - Warning Not Found

## 1. VUE D'ENSEMBLE
- **Écran** : Alerte aucun appareil détecté
- **Objectif** : Guider l'utilisateur pour résoudre le problème de détection
- **Contexte** : Affiché quand aucun appareil OTB n'est trouvé sur le réseau

## 2. LAYOUT & STRUCTURE
- Alerte warning jaune (fond #FEF3C7, bordure #F59E0B)
- Icône warning + titre
- Checklist de vérification (3 points)
- Lien vers guide d'installation
- Section additionnelle "Appareil non activé" (info bleue)

## 3. COMPOSANTS UI
- **OdsMessage** : Alerte warning principale
- Checklist numérotée (badges 1, 2, 3)
- **OdsLink** : Lien guide d'installation
- **OdsMessage** : Alerte info (appareil non activé)

## 4. CONTENU ALERTE WARNING
- Icône : ⚠ (cercle jaune)
- Titre : "Aucun appareil OverTheBox détecté"
- Description : "Nous n'avons pas pu détecter d'appareil"

## 5. CHECKLIST VÉRIFICATION
| # | Vérification |
|---|--------------|
| 1 | Votre boîtier OverTheBox est connecté à Internet |
| 2 | L'appareil est bien allumé et les voyants sont actifs |
| 3 | Votre ordinateur est connecté au même réseau que le boîtier |

## 6. AIDE SUPPLÉMENTAIRE
- Texte : "Si le problème persiste, consultez notre..."
- Lien : "guide d'installation" (bleu, souligné)

## 7. SECTION APPAREIL NON ACTIVÉ
- Type : info (fond bleu #EFF6FF)
- Titre : "Appareil non activé"
- Message : Appareil détecté mais pas activé
- Action : Connexion à http://overthebox.lan

## 8. APIS & ENDPOINTS
- GET /overthebox/detected-devices : Vérifier détection
- Polling régulier pour détecter nouvel appareil
