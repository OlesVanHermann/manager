# Cahier des Charges : web-cloud/access/overthebox - Migration Résumé

## 1. VUE D'ENSEMBLE
- **Type** : Page récapitulatif commande migration
- **URL** : /overthebox/{id}/migration/summary
- **Objectif** : Confirmer migration avant paiement

## 2. STRUCTURE PAGE
- Titre + description transition
- Tableau récapitulatif commande
- Section conditions contractuelles
- Boutons action

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur principal
- **OdsDatagrid** : Tableau récapitulatif
- **OdsCheckbox** : Acceptation CGV
- **OdsLink** : Liens vers contrats
- **OdsButton** : Annuler, Confirmer
