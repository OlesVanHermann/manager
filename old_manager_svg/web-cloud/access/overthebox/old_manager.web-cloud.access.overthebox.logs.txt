# Cahier des Charges : web-cloud/access/overthebox - Logs

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Logs (Journaux système) d'une OverTheBox
- **Objectif** : Afficher les logs système en temps réel de l'appareil
- **Contexte** : Diagnostic et monitoring de l'OverTheBox

## 2. LAYOUT & STRUCTURE
- Carte blanche avec titre "Journaux système"
- En-tête de colonnes (fond gris clair)
- Zone de logs (fond sombre #1F2937 style terminal)
- Légende des couleurs en bas

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsTable** : En-tête des colonnes
- Zone terminal : Fond sombre avec texte monospace

## 4. DONNÉES & COLONNES
| Colonne | Type | Description |
|---------|------|-------------|
| IP | text | Adresse IP source |
| Date | datetime | Format DD/MM/YYYY HH:mm:ss |
| Programme | text | Nom du service (mwan3, glorytun, otb-daemon...) |
| Message | text | Contenu du log |

## 5. NIVEAUX DE LOG (couleurs)
| Niveau | Couleur | Code |
|--------|---------|------|
| Info | Bleu | #60A5FA |
| Warning | Jaune | #FBBF24 |
| Erreur | Rouge | #F87171 |
| Debug | Gris | #6B7280 |

## 6. PROGRAMMES SYSTÈME
- mwan3 : Gestion multi-WAN
- glorytun : Tunnel VPN
- otb-daemon : Daemon principal OTB
- system : Système général
- network : Configuration réseau

## 7. APIS & ENDPOINTS
- GET /overthebox/{serviceName}/device/logs : Récupérer logs
- Paramètres : limit, level, since
- WebSocket possible pour temps réel
