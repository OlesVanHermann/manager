# Cahier des Charges : web-cloud/access/overthebox - Accès distant

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Accès distant d'une OverTheBox
- **Objectif** : Gérer les autorisations d'accès distant au support OVHcloud
- **Contexte** : Permettre au support d'intervenir sur l'appareil

## 2. LAYOUT & STRUCTURE
- Carte blanche avec titre "Configuration des accès distants"
- Tableau des accès avec lignes extensibles
- Bannière info en bas

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsTable** : Liste des accès distants
- **OdsBadge** : Statut (Actif, En attente)
- **OdsButton** : Autoriser (primaire), Révoquer/Refuser (danger)
- **OdsMessage** : Bannière info bleue
- Ligne extensible (▶) pour détails

## 4. DONNÉES & COLONNES
| Colonne | Type | Description |
|---------|------|-------------|
| ID | link | Identifiant accès (remote-xxxxx), extensible |
| Statut | badge | Actif (vert), En attente (jaune/spinner) |
| Autorisé par | text | Email de l'utilisateur autorisé |
| Expiration | date | Date d'expiration |
| Actions | buttons | Autoriser/Révoquer/Refuser |

## 5. DÉTAILS (ligne étendue)
| Champ | Description |
|-------|-------------|
| Port exposé | Port et protocole (22 SSH) |
| IP source | Domaine/IP autorisé |
| Connexion | Commande SSH complète |
| Créé le | Date/heure création |

## 6. ACTIONS & INTERACTIONS
- Clic ▶ → Étendre/réduire détails
- Bouton "Autoriser" → Accepter demande d'accès
- Bouton "Refuser" → Refuser demande d'accès
- Bouton "Révoquer" → Supprimer accès actif

## 7. APIS & ENDPOINTS
- GET /overthebox/{serviceName}/remoteAccesses : Liste accès
- POST /overthebox/{serviceName}/remoteAccesses/{id}/authorize : Autoriser
- DELETE /overthebox/{serviceName}/remoteAccesses/{id} : Révoquer/Refuser
