# Cahier des Charges : web-cloud/access/overthebox - Dashboard

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard principal d'un service OverTheBox
- **Objectif** : Afficher les informations et onglets de navigation d'une OverTheBox spécifique
- **Contexte** : Accès après sélection d'une OTB dans la liste

## 2. LAYOUT & STRUCTURE
- Header OVHcloud (fond bleu #000E9C)
- Titre éditable avec nom personnalisé + identifiant service (otb-xxx123)
- Barre d'onglets horizontale

## 3. COMPOSANTS UI
- **OdsText** : Titre éditable avec icône crayon
- **OdsTabs** : Navigation par onglets
  - Détails (actif par défaut)
  - Accès distant
  - Tâches
  - Actions
  - Logs

## 4. DONNÉES
- serviceName : Nom personnalisé éditable
- serviceId : Identifiant technique (otb-xxx)

## 5. ACTIONS & INTERACTIONS
- Clic sur crayon → Édition du nom
- Clic sur onglet → Navigation vers section

## 6. APIS & ENDPOINTS
- GET /overthebox/{serviceName} : Infos service
- PUT /overthebox/{serviceName} : Modifier nom

## 7. NAVIGATION
- Onglet Détails → overthebox.details
- Onglet Accès distant → overthebox.remote
- Onglet Tâches → overthebox.tasks
- Onglet Actions → overthebox.actions
- Onglet Logs → overthebox.logs
