# Cahier des Charges : web-cloud/access/pack-xdsl - Modal DHCP Warning

## 1. VUE D'ENSEMBLE
- **Type** : Modal alerte DNS invalide
- **URL** : N/A (composant modal)
- **Objectif** : Alerter IP DNS invalide

## 2. STRUCTURE MODAL
- Icône warning
- Message d'alerte
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal
- **OdsIcon** : Warning triangle
- **OdsText** : Message alerte
- **OdsButton** : Corriger, Ignorer

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | Auto |

## 5. ICÔNE
| Propriété | Valeur |
|-----------|--------|
| Type | Triangle warning |
| Couleur | #F59E0B (amber) |
| Taille | 48px |

## 6. MESSAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Adresse IP DNS invalide" |
| Description | "L'adresse IP DNS que vous avez saisie semble invalide. Voulez-vous la corriger ?" |
| Couleur | #92400E |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Corriger | Primary | "Corriger" |
| Ignorer | Secondary outline | "Ignorer et continuer" |
