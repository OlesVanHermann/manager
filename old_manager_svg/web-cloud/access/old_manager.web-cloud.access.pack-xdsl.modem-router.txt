# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Router

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration réseau complète
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem/router
- **Objectif** : Configurer LAN, DHCP, ports

## 2. STRUCTURE PAGE
- Section Configuration LAN
- Section Configuration DHCP
- Section Redirections de ports
- Bouton enregistrer global

## 3. COMPOSANTS UI
- **OdsCard** : Conteneurs sections
- **OdsInput** : Champs IP, masque
- **OdsSwitch** : Activation DHCP
- **OdsDatagrid** : Tableau ports
- **OdsButton** : Ajouter, Enregistrer

## 4. SECTION LAN
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Adresse IP | Input | 192.168.1.1 |
| Masque | Select | 255.255.255.0 (/24) |

## 9. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/lan - Config LAN
- PUT /xdsl/{serviceName}/modem/lan/{lanName} - Modifier LAN
- GET /xdsl/{serviceName}/modem/lan/{lanName}/dhcp - Config DHCP
- PUT /xdsl/{serviceName}/modem/lan/{lanName}/dhcp - Modifier DHCP
- GET /xdsl/{serviceName}/modem/portMappings - Liste ports
- POST /xdsl/{serviceName}/modem/portMappings - Créer redirection
- DELETE /xdsl/{serviceName}/modem/portMappings/{name} - Supprimer
