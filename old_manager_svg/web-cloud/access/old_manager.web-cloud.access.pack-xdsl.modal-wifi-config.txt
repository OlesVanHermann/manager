# Cahier des Charges : web-cloud/access/pack-xdsl - Modal WiFi Config

## 1. VUE D'ENSEMBLE
- **Type** : Modal QR code connexion WiFi
- **URL** : N/A (composant modal)
- **Objectif** : Afficher QR code pour connexion facile

## 2. STRUCTURE MODAL
- Titre
- QR Code centré
- Informations réseau
- Bouton fermeture

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal
- **OdsText** : Titre, infos réseau
- **QRCode** : Image QR code
- **OdsButton** : Fermer

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 350px |
| Hauteur | Auto |

## 5. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Connexion WiFi" |
| Style | 18px bold centré |

## 6. QR CODE
| Propriété | Valeur |
|-----------|--------|
| Taille | 200x200px |
| Format | WIFI:T:WPA;S:{SSID};P:{password};; |
| Position | Centré |
| Bordure | 1px solid #E5E7EB |

## 7. INFORMATIONS RÉSEAU
| Champ | Valeur exemple |
|-------|----------------|
| Nom du réseau | "MonWifi-5GHz" |
| Sécurité | "WPA2" |

## 8. INSTRUCTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Scannez ce QR code avec votre téléphone pour vous connecter automatiquement." |
| Style | 12px gris centré |

## 9. BOUTON
| Bouton | Style | Label |
|--------|-------|-------|
| Fermer | Secondary | "Fermer" |

## 10. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/wifi - Config WiFi pour générer QR
