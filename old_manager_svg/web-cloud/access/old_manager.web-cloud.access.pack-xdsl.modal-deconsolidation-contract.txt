# Cahier des Charges : web-cloud/access/pack-xdsl - Modal Deconsolidation Contract

## 1. VUE D'ENSEMBLE
- **Type** : Modal acceptation contrat dégroupage
- **URL** : N/A (composant modal)
- **Objectif** : Accepter conditions dégroupage total

## 2. STRUCTURE MODAL
- Titre
- Texte explicatif
- Zone contrat scrollable
- Checkboxes acceptation
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal
- **OdsText** : Titre, description, contenu contrat
- **OdsCheckbox** : Acceptation CGV
- **OdsButton** : Annuler, Accepter

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur max | 500px |

## 5. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Conditions du dégroupage total" |
| Style | 18px bold |

## 6. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Veuillez lire et accepter les conditions suivantes pour procéder au dégroupage total de votre ligne." |

## 7. ZONE CONTRAT
| Propriété | Valeur |
|-----------|--------|
| Hauteur | 200px |
| Scroll | Vertical |
| Fond | #F9FAFB |
| Contenu | Texte contrat complet |

## 8. CHECKBOXES
| Checkbox | Label |
|----------|-------|
| CGV | "J'accepte les conditions générales de vente" |
| Résiliation | "Je comprends que ma ligne France Telecom sera résiliée" |

## 9. BOUTONS
| Bouton | Style | Label | État |
|--------|-------|-------|------|
| Annuler | Secondary outline | "Annuler" | Toujours actif |
| Accepter | Primary | "Accepter et continuer" | Actif si checkboxes cochées |

## 10. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/deconsolidation/contract - Texte contrat
- POST /xdsl/{serviceName}/deconsolidation/accept - Accepter contrat
