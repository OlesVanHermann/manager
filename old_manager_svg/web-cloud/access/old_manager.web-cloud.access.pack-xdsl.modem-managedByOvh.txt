# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Managed By OVH

## 1. VUE D'ENSEMBLE
- **Type** : Composant toggle gestion OVH
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem
- **Objectif** : Activer/désactiver gestion à distance OVH

## 2. STRUCTURE COMPOSANT
- Titre section
- Description
- Toggle activation
- Warning info

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsSwitch** : Toggle activation
- **OdsMessage** : Warning info

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Gestion à distance par OVH" |
| Style | Section heading |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Autorisez OVH à configurer votre modem à distance pour simplifier la maintenance et les interventions." |

## 6. TOGGLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Activer la gestion à distance" |
| Type | OdsSwitch |
| État | On/Off |

## 7. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "En activant cette option, OVH pourra modifier la configuration de votre modem en cas de besoin." |

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem - État managedByOvh
- PUT /xdsl/{serviceName}/modem - Modifier managedByOvh
