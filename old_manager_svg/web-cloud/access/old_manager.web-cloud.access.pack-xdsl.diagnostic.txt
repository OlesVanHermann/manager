# Cahier des Charges : web-cloud/access/pack-xdsl - Diagnostic

## 1. VUE D'ENSEMBLE
- **Écran** : Diagnostic de ligne xDSL
- **Objectif** : Analyser et diagnostiquer les problèmes de ligne
- **Contexte** : Dépannage et optimisation connexion

## 2. LAYOUT & STRUCTURE
- Lien retour "← Retour à la ligne"
- Titre "Diagnostic de ligne"
- Carte avec étapes diagnostic (gauche) + résumé (droite)
- Section recommandations
- Boutons actions

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsCard** : Container diagnostic
- **OdsStepper** : Étapes avec connecteurs
- **OdsBadge** : Statut étapes (✓, !, ✗)
- **OdsCard** : Résumé diagnostic
- **OdsMessage** : Recommandations (info bleue)
- **OdsButton** : Relancer, Contacter support

## 4. ÉTAPES DIAGNOSTIC
| # | Étape | Statut | Message |
|---|-------|--------|---------|
| 1 | Vérification statut ligne | ✓ vert | Ligne active et synchronisée |
| 2 | Qualité du signal | ✓ vert | SNR : 12 dB (Bon) - Atténuation : 28 dB |
| 3 | État du modem | ✓ vert | Modem connecté et fonctionnel |
| 4 | Test de débit | ! jaune | Débit inférieur aux performances attendues |
| 5 | Résolution DNS | ✓ vert | Serveurs DNS accessibles |

## 5. RÉSUMÉ DIAGNOSTIC
| Champ | Valeur |
|-------|--------|
| Numéro de ligne | 01 23 45 67 89 |
| Type d'accès | VDSL2 |
| Débit synchronisé | ↓ 45.2 Mbps / ↑ 8.5 Mbps |
| Débit théorique | ↓ 70 Mbps / ↑ 15 Mbps |
| Alertes | Badge "1 alerte" (jaune) |

## 6. RECOMMANDATIONS
- Vérifiez les branchements de votre installation
- Évitez les rallonges téléphoniques
- Un technicien peut être envoyé si le problème persiste

## 7. ACTIONS
- Bouton "Relancer le diagnostic" (primaire)
- Bouton "Contacter le support" (secondaire)

## 8. APIS & ENDPOINTS
- POST /xdsl/{serviceName}/diagnostic/run : Lancer diagnostic
- GET /xdsl/{serviceName}/diagnostic : Résultats
