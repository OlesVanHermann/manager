# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Service

## 1. VUE D'ENSEMBLE
- **Type** : Page toggles services modem
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem/service
- **Objectif** : Activer/désactiver services réseau

## 2. STRUCTURE PAGE
- Titre principal
- Grille de services avec toggles
- Descriptions par service

## 3. COMPOSANTS UI
- **OdsText** : Titre, descriptions
- **OdsSwitch** : Toggles par service
- **OdsCard** : Conteneur services

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Services du modem" |
| Style | Page heading |

## 5. GRILLE SERVICES
### Layout
| Propriété | Valeur |
|-----------|--------|
| Type | Grid 2 colonnes |
| Gap | 20px |

### Services disponibles
| Service | Description | État |
|---------|-------------|------|
| Bloc d'IP | Attribution bloc IP publiques | Toggle |
| SIP ALG | Aide à la traversée NAT pour VoIP | Toggle |
| UPnP | Configuration automatique des ports | Toggle |
| Partage de contenu | Serveur média DLNA | Toggle |
| Firewall | Protection pare-feu intégré | Toggle |
| SMTP | Relais mail sortant | Toggle |

## 6. FORMAT SERVICE
| Élément | Style |
|---------|-------|
| Nom service | 14px bold |
| Description | 12px gris |
| Toggle | À droite |

## 7. DONNÉES EXEMPLE
| Service | État |
|---------|------|
| Bloc d'IP | Off |
| SIP ALG | On |
| UPnP | On |
| Partage de contenu | Off |
| Firewall | On |
| SMTP | Off |

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/availableWLANChannel - Services disponibles
- GET /xdsl/{serviceName}/modem - États services
- PUT /xdsl/{serviceName}/modem - Modifier services
