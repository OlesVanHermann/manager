# Cahier des Charges : web-cloud/access/pack-xdsl - Access (Ligne)

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Ligne d'un accès xDSL
- **Objectif** : Afficher détails techniques et statistiques de la ligne
- **Contexte** : Onglet par défaut du dashboard xDSL

## 2. LAYOUT & STRUCTURE
- 2 colonnes : Détails ligne | Informations techniques
- Section pleine largeur : Statistiques (graphiques)

## 3. COMPOSANTS UI
- **OdsCard** : 3 cartes (détails, infos tech, stats)
- **OdsBadge** : Statut (Actif, IPv6 Active)
- **OdsToggle** : Monitoring on/off
- **OdsButton** : Diagnostic, Résilier
- **OdsLink** : Voir détails, Modifier, Guide
- **OdsSelect** : Période graphiques
- Graphiques : Synchronisation, Trafic

## 4. DONNÉES - Détails ligne
| Champ | Description |
|-------|-------------|
| Statut | Actif (badge vert) + lien détails |
| Diagnostic | Bouton "Lancer un diagnostic" |
| Envoi de mail | Désactivé + lien Modifier |
| LNS | lns-th2-105 |
| Monitoring | Toggle on/off |

## 5. DONNÉES - Informations techniques
| Champ | Exemple |
|-------|---------|
| Type d'accès | FTTH |
| Adresse | 12 rue de la Paix, 75001 Paris |
| Opérateur de collecte | Orange |
| Login PPP | fti/abc123@fti |
| Bloc IP | 91.134.xxx.0/29 |
| IPv6 | Active (badge vert) |
| NRA/NRO | PAR01 |
| Fin d'engagement | 15/03/2025 |

## 6. STATISTIQUES
### Graphique Synchronisation
- Sélecteur période (Dernières 24h)
- Courbes Download/Upload

### Graphique Trafic
- Sélecteur période (Dernière semaine)
- Courbes Download/Upload

## 7. ACTIONS
- Bouton "Résilier" → Modal résiliation

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName} : Infos ligne
- GET /xdsl/{serviceName}/statistics : Statistiques
- PUT /xdsl/{serviceName}/monitoring : Toggle monitoring
