# Cahier des Charges : web-cloud/access/overthebox - Tâches

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Tâches d'une OverTheBox
- **Objectif** : Afficher l'historique des tâches système
- **Contexte** : Suivi des opérations internes de l'appareil

## 2. LAYOUT & STRUCTURE
- Carte blanche avec titre "Liste des tâches"
- Tableau des tâches avec pagination

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsTable** : Liste des tâches
- **OdsBadge** : Statut tâche
- **OdsPagination** : Navigation pages

## 4. DONNÉES & COLONNES
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | text | Type de tâche |
| ID | text | Identifiant (task-xxxxx) |
| Statut | badge | État de la tâche |

## 5. STATUTS POSSIBLES
| Statut | Couleur | Fond |
|--------|---------|------|
| Terminé | Vert #065F46 | #D1FAE5 |
| En cours | Bleu #1E40AF + spinner | #DBEAFE |
| En attente | Jaune #92400E | #FEF3C7 |
| Erreur | Rouge #991B1B | #FEE2E2 |

## 6. TYPES DE TÂCHES
- refreshDevice : Rafraîchir infos appareil
- updateConfiguration : Mettre à jour config
- restartService : Redémarrer service
- backupConfiguration : Sauvegarder config
- syncInterfaces : Synchroniser interfaces

## 7. APIS & ENDPOINTS
- GET /overthebox/{serviceName}/tasks : Liste tâches
- Paramètres : page, pageSize, status
- Pagination : 1 - 5 sur 12
