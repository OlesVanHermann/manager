# Cahier des Charges : web-cloud/dns-zone - Modal Ajouter entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un enregistrement DNS
- **Type** : Modal formulaire
- **Objectif** : Créer un nouvel enregistrement DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Champ sous-domaine + suffixe domaine
- Select type d'enregistrement
- Champ TTL
- Champ cible/valeur
- Aide contextuelle
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x520)
- **OdsInput** : Sous-domaine, TTL, Cible
- **OdsSelect** : Type d'enregistrement
- **OdsText** : Aide contextuelle
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Ajouter une entrée DNS"

## 5. CHAMPS
### Sous-domaine
- Type: Input
- Placeholder: "www"
- Suffixe: ".mondomaine.fr" (affiché à droite)

### Type d'enregistrement
- Type: Select
- Options: A - Adresse IPv4, AAAA - Adresse IPv6, CNAME, MX, TXT, SRV, etc.
- Exemple sélectionné: "A - Adresse IPv4"

### TTL (Time To Live)
- Type: Input number
- Valeur: "3600"
- Suffixe: "secondes"

### Cible
- Type: Input
- Placeholder: "213.186.33.5"
- Aide: "L'adresse IPv4 vers laquelle le sous-domaine pointera."

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Confirmer | Primaire bleu | Crée l'enregistrement |

## 7. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record
  - fieldType: string (A, AAAA, CNAME, MX, TXT, SRV...)
  - subDomain: string
  - target: string
  - ttl: number (optionnel, défaut 3600)
