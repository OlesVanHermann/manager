# Cahier des Charges : web-cloud/dns-zone - Historique

## 1. VUE D'ENSEMBLE
- **Page** : Historique des modifications de la zone DNS
- **Type** : Tableau chronologique
- **Objectif** : Consulter l'historique des changements DNS

## 2. STRUCTURE PAGE
- Header avec titre
- Onglet "Historique" actif
- Tableau des modifications
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation (Historique actif)
- **OdsTable** : Liste modifications
- **OdsBadge** : Type d'action (Ajout, Modification, Suppression)

## 4. TITRE
"Zone DNS - {domaine}"

## 5. COLONNES DU TABLEAU
| Colonne | Type | Exemple |
|---------|------|---------|
| Date | Texte | 26/12/2025 14:32 |
| Action | Badge | Ajout, Modification, Suppression |
| Type | Texte | TXT, A, CNAME, MX |
| Sous-domaine | Monospace | @, www, _dmarc |
| Ancienne valeur | Monospace | 213.186.33.4 ou "-" |
| Nouvelle valeur | Monospace | 213.186.33.5 ou "-" |

## 6. BADGES ACTION
| Action | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Ajout | Vert (#D1FAE5) | #065F46 |
| Modification | Bleu (#DBEAFE) | #1E40AF |
| Suppression | Rouge (#FEE2E2) | #991B1B |

## 7. EXEMPLES DONNÉES
| Date | Action | Type | Sous-domaine | Ancienne | Nouvelle |
|------|--------|------|--------------|----------|----------|
| 26/12/2025 14:32 | Ajout | TXT | _dmarc | - | "v=DMARC1; p=none" |
| 25/12/2025 10:15 | Modification | A | @ | 213.186.33.4 | 213.186.33.5 |
| 24/12/2025 16:45 | Suppression | CNAME | old | legacy.example.com. | - |
| 20/12/2025 09:30 | Ajout | MX | @ | - | 10 mx2.mail.ovh.net. |

## 8. PAGINATION
"Affichage de 1-4 sur 15 modifications"

## 9. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/history : Historique modifications
  - fieldType: filter par type
  - dateFrom: date début
  - dateTo: date fin
