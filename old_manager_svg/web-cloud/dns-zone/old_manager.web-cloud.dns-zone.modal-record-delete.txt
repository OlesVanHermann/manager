# Cahier des Charges : web-cloud/dns-zone - Modal Supprimer enregistrement

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un enregistrement DNS
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un enregistrement de la zone DNS

## 2. STRUCTURE MODAL
- Header rouge avec titre et bouton fermer (×)
- Bandeau warning orange
- Info enregistrement à supprimer
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x220)
- **OdsMessage** : Warning orange
- **OdsText** : Info enregistrement
- **OdsButton** : Annuler, Supprimer (danger)

## 4. TITRE
"Supprimer l'enregistrement"
(Header fond rouge #FEE2E2, texte #DC2626)

## 5. BANDEAU WARNING
Fond jaune (#FEF3C7) :
- "⚠ Cette action est irréversible."
- "Le service associé pourrait être impacté."

## 6. INFO ENREGISTREMENT
- Label: "Enregistrement :"
- Format: "{type} - {subdomain}.{domain} → {target}"
- Exemple: "A - www.domain.com → 192.168.1.1"
- Style: Texte bold

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Supprimer | Danger rouge | Supprime l'enregistrement |

## 8. APIS & ENDPOINTS
- DELETE /domain/zone/{zoneName}/record/{id}
- POST /domain/zone/{zoneName}/refresh : Rafraîchir zone après suppression
