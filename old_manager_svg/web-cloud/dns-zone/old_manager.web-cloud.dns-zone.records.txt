# Cahier des Charges : web-cloud/dns-zone - Zone DNS (Records)

## 1. VUE D'ENSEMBLE
- **Page** : Liste des enregistrements DNS
- **Type** : Tableau avec actions
- **Objectif** : Gérer les enregistrements DNS de la zone

## 2. STRUCTURE PAGE
- Header avec titre "Zone DNS - {domaine}"
- Onglets (Zone DNS actif)
- Bouton "Ajouter une entrée"
- Filtre par type
- Tableau des enregistrements
- Menu actions (⋮)

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : "Ajouter une entrée" (primaire)
- **OdsSelect** : "Filtrer par type"
- **OdsTable** : Liste enregistrements
- **OdsBadge** : Type (A, CNAME, MX, TXT)
- **OdsActionMenu** : Modifier, Supprimer

## 4. TITRE
"Zone DNS - {domaine}"

## 5. BOUTONS D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter une entrée | Primaire bleu | Ouvre modal ajout |
| Filtrer par type ▼ | Outline | Dropdown filtre |

## 6. COLONNES DU TABLEAU
| Colonne | Type | Exemple |
|---------|------|---------|
| Sous-domaine | Monospace | @, www, mail |
| Type | Badge couleur | A, CNAME, MX, TXT |
| TTL | Texte gris | 3600 |
| Cible / Valeur | Monospace | 213.186.33.5 |
| ⋮ | ActionMenu | Modifier, Supprimer |

## 7. BADGES TYPE
| Type | Couleur fond | Couleur texte |
|------|--------------|---------------|
| A | Bleu (#DBEAFE) | #1E40AF |
| CNAME | Vert (#D1FAE5) | #065F46 |
| MX | Jaune (#FEF3C7) | #92400E |
| TXT | Violet (#E0E7FF) | #3730A3 |

## 8. ACTIONMENU (⋮)
| Option | Action |
|--------|--------|
| Modifier | Ouvre modal-record-edit |
| Supprimer | Ouvre modal-record-delete |

## 9. PAGINATION
"Affichage de 1-5 sur 24 enregistrements"

## 10. APIS & ENDPOINTS
- GET /domain/zone/{zoneName}/record : Lister enregistrements
  - fieldType: filter par type (A, AAAA, CNAME, MX, TXT, etc.)
  - subDomain: filter par sous-domaine
