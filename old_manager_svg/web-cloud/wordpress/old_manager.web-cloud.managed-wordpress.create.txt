# Cahier des Charges : web-cloud/managed-wordpress - Créer site

## 1. VUE D'ENSEMBLE
- **Page** : Créer un site WordPress
- **Type** : Formulaire de création
- **Objectif** : Créer un nouveau site WordPress managé

## 2. STRUCTURE PAGE
- Lien retour (← Retour)
- Titre (Créer un site WordPress)
- Formulaire dans carte blanche

## 3. COMPOSANTS UI
- **OdsCard** : Container formulaire
- **OdsSelect** : Langue, Version PHP
- **OdsInput** : Email admin, Mot de passe
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Langue de l'administration | Select | Oui | Français (défaut) |
| Version PHP | Select | Oui | 8.2 (défaut) |
| Email administrateur | Input email | Oui | admin@example.com |
| Mot de passe | Password | Oui | ••••••••••• |
