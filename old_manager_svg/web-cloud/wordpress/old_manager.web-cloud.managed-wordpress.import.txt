# Cahier des Charges : web-cloud/managed-wordpress - Importer site

## 1. VUE D'ENSEMBLE
- **Page** : Importer un site WordPress
- **Type** : Formulaire d'import
- **Objectif** : Importer un site WordPress existant

## 2. STRUCTURE PAGE
- Lien retour (← Retour)
- Titre (Importer un site WordPress)
- Formulaire dans carte blanche

## 3. COMPOSANTS UI
- **OdsCard** : Container formulaire
- **OdsMessage** : Info (PHP 8.2), Warning (maintenance)
- **OdsInput** : URL, Login, Mot de passe
- **OdsLink** : Aide mot de passe app
- **OdsButton** : Annuler, Importer

## 4. MESSAGES INFORMATIFS
| Type | Couleur | Message |
|------|---------|---------|
| Info | Bleu | Votre site sera importé avec PHP 8.2 |
| Warning | Orange | Votre site sera mis en mode maintenance pendant l'import |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| URL du site à importer | Input URL | Oui | https://monsite.com |
| Login administrateur WordPress | Input | Oui | admin |
| Mot de passe d'application WordPress | Password | Oui | ••••••••••• |

## 6. AIDE
- Lien "Comment générer un mot de passe d'application ?"

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Importer | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /hosting/wordpress/{service}/import - Importer site
