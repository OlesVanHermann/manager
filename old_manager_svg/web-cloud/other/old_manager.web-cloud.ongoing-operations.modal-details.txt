# Cahier des Charges : web-cloud/ongoing-operations - Modal Détails

## 1. VUE D'ENSEMBLE
- **Page** : Détails de l'opération
- **Type** : Modal information
- **Objectif** : Afficher les détails d'une opération

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Informations détaillées en grille
- Barre de progression
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsText** : Labels et valeurs
- **OdsBadge** : Statut
- **OdsProgress** : Progression
- **OdsButton** : Annuler, Fermer

## 4. INFORMATIONS AFFICHÉES
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| ID de l'opération | Text | OP-2024-12-29-001234 |
| Statut | Badge | En attente (orange) |
| Type d'opération | Text | Transfert entrant de domaine |
| Service concerné | Text | domain.com |
| Créée le | Text | 29/12/2024 10:30 |
| Dernière mise à jour | Text | 29/12/2024 14:15 |

## 5. PROGRESSION
| Élément | Valeur |
|---------|--------|
| Barre | 50% remplie |
| Label | Étape 2/4 - En attente de validation |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline rouge | Gauche |
| Fermer | Primary | Droite |

## 7. APIS & ENDPOINTS
- GET /domain/operation/{id} - Détails opération
