# Cahier des Charges : web-cloud/ongoing-operations - Modal Relancer

## 1. VUE D'ENSEMBLE
- **Page** : Relancer l'opération
- **Type** : Modal confirmation info
- **Objectif** : Relancer une opération échouée

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message info
- Info opération
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Info bleu
- **OdsText** : Info opération
- **OdsButton** : Annuler, Relancer

## 4. MESSAGE INFO
| Type | Couleur | Lignes |
|------|---------|--------|
| Info | Bleu | Cette opération a échoué. Vous pouvez la relancer |
| | | après avoir corrigé le problème. |

## 5. INFO OPÉRATION
| Élément | Valeur exemple |
|---------|----------------|
| Label | Opération : |
| Valeur | Configuration DNS pour domain.com |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Relancer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /domain/operation/{id}/retry - Relancer opération
