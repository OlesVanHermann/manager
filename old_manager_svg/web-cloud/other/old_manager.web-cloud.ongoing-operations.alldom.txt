# Cahier des Charges : web-cloud/ongoing-operations - Onglet Alldom

## 1. VUE D'ENSEMBLE
- **Page** : Opérations en cours - Alldom
- **Type** : Onglet liste
- **Objectif** : Afficher les opérations sur les packs Alldom

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre + description
- Onglets (Alldom actif)
- Datagrid des opérations
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Domaines, DNS, Alldom (actif)
- **OdsDatagrid** : Liste opérations
- **OdsBadge** : Status
- **OdsPagination** : Navigation pages

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Alldom | Link | ID pack (alldom-pack-123) |
| Opération | Text | Type opération |
| Commentaire | Text | Détails |
| Date de mise à jour | Text | Date/heure |
| État | Badge | Statut |
| Actions | ActionMenu | ⋮ |

## 5. TYPES OPÉRATIONS ALLDOM
| Type | Description |
|------|-------------|
| Suppression du service Alldom | Résiliation |
| Transfert entrant | Transfert pack |
| Renouvellement du domaine | Renouvellement |

## 6. BADGES STATUT
| Badge | Couleur | Description |
|-------|---------|-------------|
| En cours | Orange (warning) | Traitement |
| À traiter | Rouge (error) | Action requise |
| Terminée | Vert (success) | Complété |

## 7. APIS & ENDPOINTS
- GET /allDom/operations - Liste opérations Alldom
