# Cahier des Charges : web-cloud/ongoing-operations - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Opérations en cours
- **Type** : Page liste principale avec onglets
- **Objectif** : Suivre les opérations en cours sur les services

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar navigation
- Titre + onglets (Domaine, DNS, Alldom)
- Filtres (recherche, type, état)
- Datagrid des opérations

## 3. COMPOSANTS UI
- **OdsTabs** : Domaine, DNS, Alldom
- **OdsInput** : Recherche
- **OdsSelect** : Type d'opération, État
- **OdsDatagrid** : Liste opérations
- **OdsBadge** : Status
- **OdsActionMenu** : Menu contextuel

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Domaine | Opérations sur les domaines |
| DNS | Opérations sur les zones DNS |
| Alldom | Opérations packs Alldom |

## 5. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Recherche | Input | Texte libre |
| Type d'opération | Select | Transfert, Création, Protection... |
| État | Select | En attente, En cours, Terminé |

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| DOMAINE | Link | Nom de domaine |
| OPÉRATION | Text | Type d'opération |
| ÉTAT | Badge | Statut |
| DATE | Text | Date création |
| ACTIONS | ActionMenu | Menu ⋮ |

## 7. BADGES STATUT
| Badge | Couleur | Description |
|-------|---------|-------------|
| En attente | Orange (warning) | Action requise |
| En cours | Bleu (info) | Traitement |
| Terminé | Vert (success) | Complété |

## 8. ACTIONMENU
| Option | Action |
|--------|--------|
| Modifier l'opération | Page update |
| Suivre le transfert | Page tracking |

## 9. APIS & ENDPOINTS
- GET /domain/operations - Liste opérations domaine
- GET /dns/operations - Liste opérations DNS
