# Cahier des Charges : web-cloud/ongoing-operations - Mettre à jour

## 1. VUE D'ENSEMBLE
- **Page** : Mettre à jour l'opération
- **Type** : Page formulaire avec actions
- **Objectif** : Modifier une opération en cours

## 2. STRUCTURE PAGE
- Lien retour (← Retour aux opérations)
- Titre + description opération
- Formulaire dans carte blanche
- Boutons d'action

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsCard** : Container formulaire
- **OdsInput** : Code AuthInfo
- **OdsMessage** : Warning accélération
- **OdsButton** : Relancer, Accélérer, Annuler, Retour

## 4. CHAMPS FORMULAIRE
| Champ | Type | Description |
|-------|------|-------------|
| Code AuthInfo | Password | Code d'autorisation du registrar |

## 5. MESSAGE WARNING
| Type | Couleur | Lignes |
|------|---------|--------|
| Warning | Orange | Accélérer le transfert |
| | | L'accélération peut échouer si le registrar actuel refuse. |

## 6. BOUTONS ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Relancer | Primary bleu | Relancer l'opération |
| Accélérer | Warning orange | Accélérer le transfert |
| Annuler | Danger rouge | Annuler l'opération |
| Retour | Outline | Retour liste |

## 7. APIS & ENDPOINTS
- PUT /domain/operation/{id}/authinfo - Modifier AuthInfo
- POST /domain/operation/{id}/retry - Relancer
- POST /domain/operation/{id}/accelerate - Accélérer
- DELETE /domain/operation/{id} - Annuler
