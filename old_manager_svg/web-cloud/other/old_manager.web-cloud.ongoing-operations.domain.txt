# Cahier des Charges : web-cloud/ongoing-operations - Onglet Domaine

## 1. VUE D'ENSEMBLE
- **Page** : Opérations en cours - Domaines
- **Type** : Onglet liste
- **Objectif** : Afficher les opérations sur les domaines

## 2. STRUCTURE PAGE
- Titre + onglets (Domaine actif)
- Filtres (recherche, type, état)
- Datagrid des opérations

## 3. COMPOSANTS UI
- **OdsTabs** : Domaine (actif), DNS, Alldom
- **OdsInput** : Recherche
- **OdsSelect** : Type, État
- **OdsDatagrid** : Liste opérations
- **OdsBadge** : Status
- **OdsActionMenu** : Actions

## 4. FILTRES
| Filtre | Valeur exemple |
|--------|----------------|
| Type | Transfert entrant |
| État | En attente |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| DOMAINE | Link | Nom de domaine |
| OPÉRATION | Text | Transfert entrant, Création, Protection WHOIS |
| ÉTAT | Badge | En attente, En cours, Terminé |
| DATE CRÉATION | Text | 27/12/2024 10:30 |
| ACTIONS | ActionMenu | ⋮ |

## 6. TYPES OPÉRATIONS
| Type | Description |
|------|-------------|
| Transfert entrant | Transfert vers OVHcloud |
| Création | Enregistrement domaine |
| Protection WHOIS | Activation protection |
| Renouvellement | Renouvellement domaine |

## 7. APIS & ENDPOINTS
- GET /domain/operations?type=transfer - Opérations transfert
