## 1. RÉSUMÉ EXÉCUTIF

### Périmètre
Section de commande de crédits fax intégrée au dashboard FreeFax.

### Objectif fonctionnel
Permettre à l'utilisateur de commander des crédits fax supplémentaires.

### Note
Le code source du module credit n'est pas fourni. Spécification basée sur les traductions disponibles.

## 2. ARCHITECTURE

### Référence dans le code
```javascript
import credit from './credit';
```

---

## 3. PAGES & NAVIGATION

### Intégration
Section intégrée dans le dashboard FreeFax (freefaxes.freefax).

---

## 4. COMPOSANTS UI

### Tile Commande
| Élément | Contenu |
|---------|---------|
| Titre | "Commander des fax" (freefax_general_credit) |
| Input quantité | Sélection du nombre de fax |
| Bouton | "Générer le bon de commande" |

### Résultat commande
| Élément | Contenu |
|---------|---------|
| Résumé | Montant, quantité, prix TTC |
| Lien PDF | Version PDF du bon de commande |
| Bouton paiement | "Régler votre commande" |

---

## 5. APIs & DONNÉES

### Endpoints probables
| Méthode | Endpoint | Description |
|---------|----------|-------------|
| GET | /freefax/{serviceName}/credits | Crédits disponibles |
| POST | /order/freefax/{serviceName}/credit | Génération bon de commande |

## 7. RÈGLES MÉTIER

### Flux de commande
1. Sélection de la quantité de fax
2. Clic sur "Générer le bon de commande"
3. Affichage du résumé avec montant TTC
4. Téléchargement PDF possible
5. Clic sur "Régler votre commande" → redirection paiement
