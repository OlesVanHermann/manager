# Cahier des Charges : web-cloud/email-domain - Onglet Repondeurs

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des repondeurs automatiques (absence).

### Objectif fonctionnel
- Creer des reponses automatiques
- Planifier les periodes d'absence
- Configurer les messages

## 4. APIS & DONNEES

### Endpoints
```
GET /email/domain/{domain}/responder
GET /email/domain/{domain}/responder/{account}
POST /email/domain/{domain}/responder
PUT /email/domain/{domain}/responder/{account}
DELETE /email/domain/{domain}/responder/{account}
```

### Modele
```typescript
interface EmailResponder {
  account: string;
  from: string;
  to: string;
  copy: string;
  content: string;
}

interface ResponderStatus {
  account: string;
  startDate: string;
  endDate: string;
  copyTo?: string;
  status: 'active' | 'scheduled' | 'inactive';
}
```
