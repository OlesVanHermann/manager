# Cahier des Charges : web-cloud/email-domain - Modal Créer redirection

## 1. VUE D'ENSEMBLE
- **Type** : Modal création redirection email
- **URL** : N/A (composant modal)
- **Objectif** : Créer une redirection email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Adresse source, destination
- **OdsCheckbox** : Copie locale
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Adresse source | Input + domaine fixe | Oui |
| Destination | Input email complet | Oui |
| Conserver une copie locale | Checkbox | Non |

## 5. CHAMP SOURCE
| Élément | Format |
|---------|--------|
| Input | Partie locale (ex: info) |
| Suffixe | @domain.com (fixe) |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Créer | Primary | Créer redirection |

## 7. APIS & ENDPOINTS
- POST /email/domain/{domain}/redirection - Créer redirection
