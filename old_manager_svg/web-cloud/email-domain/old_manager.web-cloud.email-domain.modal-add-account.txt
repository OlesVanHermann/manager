# Cahier des Charges : web-cloud/email-domain - Modal creation compte

## 1. RESUME EXECUTIF

### Perimetre
Modal de creation d'un nouveau compte email.

### Objectif fonctionnel
- Creer une nouvelle adresse email
- Definir le mot de passe
- Configurer le quota

## 4. APIS & DONNEES

### Endpoints
```
POST /email/domain/{domain}/account
GET /email/domain/{domain}/quota
```

### Modele
```typescript
interface CreateAccountForm {
  accountName: string;
  description?: string;
  password: string;
  size: number;
}

interface QuotaInfo {
  accountsUsed: number;
  accountsMax: number;
  spaceUsed: number;
  spaceMax: number;
}
```

- Nom du compte unique sur le domaine

- Creation asynchrone (tache)
