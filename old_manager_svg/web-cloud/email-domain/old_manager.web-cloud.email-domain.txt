# Cahier des Charges : web-cloud/email-domain - Dashboard MX Plan

## 1. RESUME EXECUTIF

### Perimetre
Dashboard principal d'un service email MX Plan.

### Objectif fonctionnel
- Voir les informations du service email
- Gerer les comptes email
- Acceder aux onglets (redirections, listes, filtres, repondeurs, taches)

## 4. APIS & DONNEES

### Endpoints
```
GET /email/domain/{domain}
GET /email/domain/{domain}/account
GET /email/domain/{domain}/account/{accountName}
POST /email/domain/{domain}/account
DELETE /email/domain/{domain}/account/{accountName}
```

### Modele
```typescript
interface EmailDomainService {
  domain: string;
  offer: string;
  status: string;
  accountsUsed: number;
  accountsMax: number;
}

interface EmailAccount {
  accountName: string;
  email: string;
  description: string;
  size: number;
  quota: number;
  isBlocked: boolean;
}
```
