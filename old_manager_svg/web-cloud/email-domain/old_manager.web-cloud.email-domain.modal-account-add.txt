# Cahier des Charges : web-cloud/email-domain - Modal Créer compte

## 1. VUE D'ENSEMBLE
- **Page** : Créer un compte email
- **Type** : Modal formulaire
- **Objectif** : Créer une nouvelle adresse email

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ nom du compte + domaine
- Champ description
- Champ mot de passe
- Select espace disque
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom compte, description, mot de passe
- **OdsSelect** : Espace disque
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Nom du compte | Input + suffix | Oui | contact + @domain.com |
| Description | Input | Non | Ex: Service commercial |
| Mot de passe | Password | Oui | ••••••••••• |
| Espace disque | Select | Non | 5 Go (défaut) |

## 5. OPTIONS ESPACE DISQUE
| Valeur | Description |
|--------|-------------|
| 5 Go | Standard |
| 10 Go | Extended |
| 25 Go | Large |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /email/domain/{domain}/account - Créer compte
