# Cahier des Charges : web-cloud/email-domain - Modal Modifier redirection

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification redirection
- **URL** : N/A (composant modal)
- **Objectif** : Modifier la destination d'une redirection

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Labels, source (lecture seule)
- **OdsInput** : Nouvelle destination
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Modifier la destination de la redirection" |
| Sous-titre | "Définissez l'adresse email vers laquelle rediriger les messages." |

## 5. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| De | Texte (lecture seule) | - |
| Vers | Input email | Oui |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Modifier | Primary | Enregistrer modification |

## 7. APIS & ENDPOINTS
- PUT /email/domain/{domain}/redirection/{id} - Modifier redirection
