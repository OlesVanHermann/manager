# Cahier des Charges : web-cloud/email-domain - Modal Supprimer redirection

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation suppression
- **URL** : N/A (composant modal)
- **Objectif** : Confirmer suppression d'une redirection

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container danger
- **OdsText** : Message confirmation
- **OdsButton** : Actions

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Couleur fond | Rouge #FEF2F2 |
| Titre | "Supprimer la redirection" |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir supprimer la redirection ?" |
| Info | Affiche source → destination (ex: contact@example.com → backup@gmail.com) |
| Confirmation | "Voulez-vous continuer ?" |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Supprimer | Danger (rouge) | Confirmer suppression |

## 7. APIS & ENDPOINTS
- DELETE /email/domain/{domain}/redirection/{id} - Supprimer redirection
