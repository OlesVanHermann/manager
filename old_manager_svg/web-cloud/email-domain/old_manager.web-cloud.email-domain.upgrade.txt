# Cahier des Charges : web-cloud/email-domain - Changer offre

## 1. VUE D'ENSEMBLE
- **Type** : Wizard upgrade MX Plan
- **URL** : /email/domain/{domain}/upgrade
- **Objectif** : Changer vers une offre MX Plan supérieure

## 2. STRUCTURE PAGE
- Bouton retour
- Stepper 2 étapes
- Description
- Cartes offres (dont actuelle)
- Récapitulatif prix
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsStepper** : Étapes
- **OdsCard** : Offres avec radio et badge "Actuel"
- **OdsBadge** : Offre actuelle
- **OdsText** : Récapitulatif prix
- **OdsButton** : Actions

## 4. STEPPER
| Étape | Label | État |
|-------|-------|------|
| 1 | Offre | Active |
| 2 | Validation | Inactive |

## 5. OFFRES DISPONIBLES
| Offre | Comptes | Stockage | Prix |
|-------|---------|----------|------|
| MX Plan 25 | 25 | 5 Go/compte | - (Actuel) |
| MX Plan 100 | 100 | 5 Go/compte | 29,99 € HT/an |
| MX Plan 500 | 500 | 5 Go/compte | 99,99 € HT/an |

## 6. BADGE OFFRE ACTUELLE
| Propriété | Valeur |
|-----------|--------|
| Label | "Actuel" |
| Style | Gris désactivé |

## 7. RÉCAPITULATIF PRIX
| Ligne | Description |
|-------|-------------|
| Prix HT | Montant HT |
| Prix TTC | Montant TTC |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Suivant | Primary | Étape validation |

## 9. APIS & ENDPOINTS
- GET /email/domain/{domain} - Infos offre actuelle
- POST /order/upgrade/emailDomain - Commander upgrade
