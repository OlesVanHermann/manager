# Cahier des Charges : web-cloud/email-domain - Modal Créer filtre

## 1. VUE D'ENSEMBLE
- **Type** : Modal création filtre email
- **URL** : N/A (composant modal)
- **Objectif** : Créer une règle de filtrage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 610px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Nom filtre, valeur règle, email redirection
- **OdsSelect** : Priorité, en-tête, opérateur, action
- **OdsCheckbox** : Activer filtre
- **OdsButton** : Actions, ajout/suppression règles
- **OdsMessage** : Info priorité

## 4. SECTION INFORMATIONS
| Champ | Type | Requis |
|-------|------|--------|
| Nom du filtre | Input texte | Oui |
| Priorité | Input nombre | Oui |
| Activer le filtre | Checkbox | Non (coché par défaut) |

## 5. SECTION RÈGLES
| Champ | Type | Options |
|-------|------|---------|
| En-tête | Select | From, To, Subject, Cc... |
| Opérateur | Select | Contient, Ne contient pas, Égal... |
| Valeur | Input texte | Libre |
| Actions | Boutons +/- | Ajouter/supprimer règle |

## 6. SECTION ACTION
| Champ | Type | Options |
|-------|------|---------|
| Action | Select | Supprimer, Rediriger, Déplacer... |
| Rediriger vers | Input email | Conditionnel selon action |

## 7. MESSAGE INFO
| Type | Texte |
|------|-------|
| Info | "La priorité détermine l'ordre d'exécution des filtres (1 = haute priorité)" |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Créer | Primary | Créer filtre |

## 9. APIS & ENDPOINTS
- POST /email/domain/{domain}/account/{account}/filter - Créer filtre
