# Cahier des Charges : web-cloud/email-domain - Abonnés liste

## 1. VUE D'ENSEMBLE
- **Type** : Page liste abonnés
- **URL** : /email/domain/{domain}/mailingList/{name}/subscriber
- **Objectif** : Gérer les abonnés d'une mailing list

## 2. STRUCTURE PAGE
- Bouton retour
- Titre avec nom liste et quota (X / Y)
- Bouton suppression sélection
- Barre recherche
- Tableau abonnés
- Sidebar actions

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre avec quota
- **OdsButton** : Supprimer sélectionnés
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste abonnés
- **OdsCheckbox** : Sélection multiple

## 4. TABLEAU ABONNÉS
| Colonne | Type |
|---------|------|
| Checkbox | Sélection |
| Email de l'abonné | Texte |
| Refresh | Icône action |

## 5. SIDEBAR
| Bouton | Action |
|--------|--------|
| Ajouter | Ouvrir modal ajout |
| Supprimer (fichier) | Import fichier suppression |
| Exporter CSV | Téléchargement liste |

## 6. PAGINATION
| Élément | Format |
|---------|--------|
| Page | "Page X sur Y" |

## 7. APIS & ENDPOINTS
- GET /email/domain/{domain}/mailingList/{name}/subscriber - Liste abonnés
- POST /email/domain/{domain}/mailingList/{name}/subscriber - Ajouter
- DELETE /email/domain/{domain}/mailingList/{name}/subscriber/{email} - Supprimer
