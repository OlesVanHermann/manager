# Cahier des Charges : web-cloud/email-domain - Modal Créer répondeur

## 1. VUE D'ENSEMBLE
- **Page** : Créer un répondeur automatique
- **Type** : Modal formulaire
- **Objectif** : Configurer une réponse automatique

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Select compte email
- Date de début + fin
- Textarea message
- Checkbox copie messages
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsSelect** : Compte email
- **OdsInput** : Dates (date picker)
- **OdsTextarea** : Message de réponse
- **OdsCheckbox** : Conserver copie
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| Compte | Select | Oui | contact@domain.com |
| Date de début | DatePicker | Oui | 01/01/2025 |
| Date de fin | DatePicker | Oui | 15/01/2025 |
| Message | Textarea | Oui | Message d'absence |
| Copie messages | Checkbox | Non | Conserver une copie des messages reçus |

## 5. EXEMPLE MESSAGE
```
Bonjour,
Je suis actuellement absent et ne pourrai pas
répondre à votre message avant mon retour.
Cordialement
```

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /email/domain/{domain}/responder - Créer répondeur
