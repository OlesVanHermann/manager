# Cahier des Charges : web-cloud/email-domain - Répondeurs Délégation

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion répondeurs automatiques
- **URL** : /email/domain/{domain}/responder
- **Objectif** : Configurer réponses automatiques

## 2. STRUCTURE PAGE
- Bouton retour
- Titre page
- Tableau répondeurs
- Sidebar création et guides

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre
- **OdsDatagrid** : Liste répondeurs
- **OdsBadge** : Statut
- **OdsActionMenu** : Actions

## 4. TABLEAU RÉPONDEURS
| Colonne | Type |
|---------|------|
| Nom | Texte |
| Date de début | Date |
| Date de fin | Date |
| Statut | Badge |
| Copie vers | Email ou "-" |
| Actions | ActionMenu |

## 5. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Actif | Vert #DEF7EC |
| Expiré | Rouge #FEE2E2 |
| permanent | Texte gris |

## 6. MENU ACTIONS
| Option | Couleur |
|--------|---------|
| Modifier le répondeur | Standard |
| Supprimer le répondeur | Rouge |

## 7. SIDEBAR
| Élément | Action |
|---------|--------|
| Créer un répondeur | Bouton modal |
| Configurer un répondeur email | Lien guide |

## 8. APIS & ENDPOINTS
- GET /email/domain/{domain}/responder - Liste répondeurs
- POST /email/domain/{domain}/responder - Créer
- PUT /email/domain/{domain}/responder/{account} - Modifier
- DELETE /email/domain/{domain}/responder/{account} - Supprimer
