# Cahier des Charges : web-cloud/email-domain - Listes de diffusion

## 1. VUE D'ENSEMBLE
- **Type** : Page liste des mailing lists
- **URL** : /email/domain/{domain}/mailingList
- **Objectif** : Afficher et gérer les listes de diffusion

## 2. STRUCTURE PAGE
- Info quota
- Barre recherche
- Tableau listes
- Sidebar création

## 3. COMPOSANTS UI
- **OdsText** : Quota (X / Y)
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste des listes
- **OdsButton** : Icônes modérateurs/abonnés
- **OdsActionMenu** : Actions par liste

## 4. TABLEAU LISTES
| Colonne | Type |
|---------|------|
| Nom | Email liste |
| Propriétaire | Email |
| Langue | Texte |
| Répondre à | Texte (À la liste / Expéditeur) |
| Modérateurs | Icône cliquable |
| Abonnés | Icône cliquable |
| Actions | ActionMenu ⋮ |

## 5. MENU ACTIONS
| Option | Style |
|--------|-------|
| Modifier | Standard |
| Supprimer | Rouge |
| Envoyer la liste par email | Standard |

## 6. SIDEBAR
| Bouton | Action |
|--------|--------|
| Créer une liste | Ouvrir modal création |

## 7. PAGINATION
| Élément | Format |
|---------|--------|
| Page | "Page X sur Y" |

## 8. APIS & ENDPOINTS
- GET /email/domain/{domain}/mailingList - Liste des listes
- PUT /email/domain/{domain}/mailingList/{name} - Modifier
- DELETE /email/domain/{domain}/mailingList/{name} - Supprimer
