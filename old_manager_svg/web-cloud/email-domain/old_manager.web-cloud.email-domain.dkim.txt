# Cahier des Charges : web-cloud/email-domain - Modal DKIM

## 1. VUE D'ENSEMBLE
- **Type** : Modal configuration DKIM
- **URL** : N/A (composant modal)
- **Objectif** : Afficher et activer la signature DKIM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsBadge** : Statut configuration
- **OdsText** : Informations DNS
- **OdsLink** : Lien guide
- **OdsButton** : Actions
