# Cahier des Charges : web-cloud/email-domain - Informations générales

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard informations domaine email
- **URL** : /email/domain/{domain}
- **Objectif** : Afficher synthèse du service email

## 2. STRUCTURE PAGE
- Header avec breadcrumb
- Titre "Informations générales"
- 3 tuiles principales
- Section guides

## 3. COMPOSANTS UI
- **OdsText** : Titres et labels
- **OdsBadge** : Statuts (OK, DKIM)
- **OdsProgressBar** : Quotas utilisation
- **OdsLink** : Liens webmail et guides
- **OdsCard** : Tuiles informations
- **OdsActionMenu** : Actions sur offre et contacts

## 4. TUILE INFORMATIONS GÉNÉRALES
| Élément | Type |
|---------|------|
| Statut | Badge OK (vert) |
| Filer | Texte (filerz-001) |
| Enregistrements MX | Liste (mx1, mx2) |
| Diagnostic | Badge avertissement DKIM |

## 5. TUILE QUOTAS
| Quota | Format |
|-------|--------|
| Comptes email | X / Y + ProgressBar |
| Répondeurs | X / Y + ProgressBar |
| Redirections | X / Y + ProgressBar |

## 6. TUILE OFFRE
| Élément | Type |
|---------|------|
| Webmail | Lien Roundcube ↗ |
| Offre | Texte (MXPlan) + ActionMenu |
| Contacts | Liste Admin/Tech/Billing + ActionMenu |

## 7. GUIDES
| Lien | Action |
|------|--------|
| Premiers pas avec MXPlan | Ouvrir guide ↗ |
| Configurer votre client email | Ouvrir guide ↗ |
| Créer des filtres email | Ouvrir guide ↗ |

## 8. APIS & ENDPOINTS
- GET /email/domain/{domain} - Infos domaine
- GET /email/domain/{domain}/account - Comptes
- GET /email/domain/{domain}/responder - Répondeurs
- GET /email/domain/{domain}/redirection - Redirections
