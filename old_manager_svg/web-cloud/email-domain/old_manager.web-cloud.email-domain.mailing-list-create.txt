# Cahier des Charges : web-cloud/email-domain - Créer liste diffusion

## 1. VUE D'ENSEMBLE
- **Type** : Modal création liste de diffusion
- **URL** : N/A (composant modal)
- **Objectif** : Créer une nouvelle mailing list

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : 2 étapes
- **OdsInput** : Champs texte
- **OdsSelect** : Domaine, langue
- **OdsRadio** : Options réponse/modération
- **OdsButton** : Actions

## 4. STEPPER
| Étape | Label |
|-------|-------|
| 1 | Configuration (active) |
| 2 | Confirmation |

## 5. FORMULAIRE ÉTAPE 1
| Champ | Type | Requis |
|-------|------|--------|
| Nom de la liste | Input + Select domaine | Oui |
| Email du propriétaire | Input | Oui |
| Répondre à | Radio group | Non |
| Langue | Select | Oui |
| Modération | Radio group | Non |

## 6. OPTIONS RÉPONSE
| Option | Valeur |
|--------|--------|
| À la liste de diffusion | Sélectionnée par défaut |
| Au dernier expéditeur | - |
| À une adresse spécifique | Affiche input email |

## 7. OPTIONS MODÉRATION
| Option | Valeur |
|--------|--------|
| Pas de modération | Sélectionnée par défaut |
| Modération messages | - |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant | Primary | Étape 2 |

## 9. APIS & ENDPOINTS
- POST /email/domain/{domain}/mailingList - Créer liste
