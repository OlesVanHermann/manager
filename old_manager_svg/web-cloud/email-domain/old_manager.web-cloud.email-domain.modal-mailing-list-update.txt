# Cahier des Charges : web-cloud/email-domain - Modal Modifier liste

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification liste de diffusion
- **URL** : N/A (composant modal)
- **Objectif** : Modifier paramètres d'une mailing list

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Email propriétaire
- **OdsRadio** : Options réponse, modération
- **OdsSelect** : Langue
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Email du propriétaire | Input email | Oui |
| Répondre à | Radio group | Non |
| Langue | Select | Oui |
| Modération | Radio group | Non |

## 5. OPTIONS RÉPONSE
| Option | Description |
|--------|-------------|
| À la liste de diffusion | Défaut |
| Au dernier expéditeur | - |
| À une adresse spécifique | Affiche input |

## 6. OPTIONS MODÉRATION
| Option | Description |
|--------|-------------|
| Pas de modération | Défaut |
| Modération des messages | - |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Valider | Primary | Enregistrer modifications |

## 8. APIS & ENDPOINTS
- PUT /email/domain/{domain}/mailingList/{name} - Modifier liste
