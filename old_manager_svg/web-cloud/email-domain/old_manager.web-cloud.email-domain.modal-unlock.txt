# Cahier des Charges : web-cloud/email-domain - Modal Débloquer compte

## 1. VUE D'ENSEMBLE
- **Type** : Modal déblocage compte email
- **URL** : N/A (composant modal)
- **Objectif** : Débloquer un compte email bloqué

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Messages
- **OdsInput** : Mot de passe, confirmation
- **OdsMessage** : Info règles mot de passe
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Texte |
|---------|-------|
| Info | "Le compte contact@example.com est bloqué." |
| Instruction | "Pour le débloquer, définissez un nouveau mot de passe." |
| Question | "Voulez-vous continuer ?" |

## 5. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nouveau mot de passe | Password input | Oui |
| Confirmer le mot de passe | Password input | Oui |

## 6. RÈGLES MOT DE PASSE
| Règle | Description |
|-------|-------------|
| Longueur | Entre 9 et 30 caractères |
| Contenu | Au moins une majuscule, une minuscule et un chiffre |

## 7. MESSAGE INFO
| Type | Couleur |
|------|---------|
| Info | Bleu #EFF6FF |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Débloquer | Primary | Débloquer compte |

## 9. APIS & ENDPOINTS
- POST /email/domain/{domain}/account/{account}/changePassword - Changer mot de passe et débloquer
