# Cahier des Charges : web-cloud/email-domain - Modal Changer mot de passe

## 1. VUE D'ENSEMBLE
- **Type** : Modal changement mot de passe
- **URL** : N/A (composant modal)
- **Objectif** : Changer le mot de passe d'un compte email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Messages
- **OdsInput** : Mots de passe
- **OdsMessage** : Info règles
- **OdsButton** : Actions

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Changer le mot de passe" |
| Instruction | "Définissez un nouveau mot de passe pour ce compte email." |
| Question | "Voulez-vous continuer ?" |

## 5. FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nouveau mot de passe | Password input | Oui |
| Confirmer le mot de passe | Password input | Oui |

## 6. RÈGLES MOT DE PASSE
| Règle | Description |
|-------|-------------|
| Longueur | Entre 9 et 30 caractères |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Valider | Primary | Changer mot de passe |

## 8. APIS & ENDPOINTS
- POST /email/domain/{domain}/account/{account}/changePassword - Changer mot de passe
