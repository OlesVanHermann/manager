# Cahier des Charges : web-cloud/email-domain - ACL

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion ACL (droits d'accès)
- **URL** : /email/domain/{domain}/acl
- **Objectif** : Gérer accès collaborateurs aux emails

## 2. STRUCTURE PAGE
- Bouton retour
- Titre page
- Description
- Tableau ACL avec formulaire inline

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre, description
- **OdsDatagrid** : Liste ACL
- **OdsInput** : Formulaire ajout inline
- **OdsButton** : Valider, Annuler, Supprimer

## 4. FORMULAIRE AJOUT INLINE
| Élément | Type |
|---------|------|
| Input | Identifiant NIC |
| Bouton ✓ | Primary - Valider |
| Bouton × | Secondary - Annuler |

## 5. TABLEAU ACL
| Colonne | Type |
|---------|------|
| Identifiant NIC | Texte |
| Supprimer | Icône poubelle rouge |

## 6. SIDEBAR
| Bouton | Action |
|--------|--------|
| Créer un compte NIC ↗ | Lien externe |

## 7. APIS & ENDPOINTS
- GET /email/domain/{domain}/acl - Liste ACL
- POST /email/domain/{domain}/acl - Créer ACL
- DELETE /email/domain/{domain}/acl/{accountId} - Supprimer ACL
