## 7. APIS & ENDPOINTS

```
GET /domain/zone/{zoneName}/export
→ Exporte la zone au format BIND

# Le fichier est téléchargé avec le nom:
# {zoneName}.zone ou {zoneName}.txt

# Format BIND retourné:
$TTL 3600
@    IN    SOA   dns1.ovh.net. tech.ovh.net. (
                  2024010101 ; serial
                  86400      ; refresh
                  3600       ; retry
                  3600000    ; expire
                  3600 )     ; minimum
@    IN    A     213.186.33.5
www  IN    CNAME @
...
```
