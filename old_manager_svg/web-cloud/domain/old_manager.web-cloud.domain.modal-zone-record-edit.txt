# Cahier des Charges : web-cloud/domain - Modal Modifier entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un enregistrement DNS
- **Type** : Formulaire de modification
- **Objectif** : Éditer une entrée existante de la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Type de champ (readonly avec badge)
- Champ sous-domaine
- Section TTL (défaut/personnalisé)
- Champ cible
- Note sur propagation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x520)
- **OdsBadge** : Type de champ (bleu)
- **OdsInput** : Sous-domaine, TTL personnalisé, Cible
- **OdsRadio** : TTL (Par défaut / Personnalisé)
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier l'entrée DNS"

## 5. CHAMP TYPE (READONLY)
| Élément | Valeur |
|---------|--------|
| Label | "Type de champ" |
| Input | Désactivé (fond gris) |
| Badge | [A] (bleu #E8F0FE) |
| Note | Le type n'est pas modifiable |

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "www" |
| Suffixe (addon gris) | ".example.com" |

## 7. SECTION TTL
| Option | État | Valeur |
|--------|------|--------|
| Par défaut | ○ | - |
| Personnalisé | ● Sélectionné | 3600 secondes |

Input personnalisé :
- Largeur : 120px
- Valeur : 3600
- Suffixe : "secondes"

## 8. CHAMP CIBLE
| Élément | Valeur |
|---------|--------|
| Label | "Cible *" |
| Obligatoire | Oui |
| Input | "203.0.113.50" |

## 9. NOTE PROPAGATION
"La modification sera immédiate dans la zone DNS, mais veuillez prendre en compte le temps de propagation (maximum 24h)."

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Modifier (primaire)

## 11. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName}/record/{id} : Modifier enregistrement DNS
  - subDomain: string (optionnel)
  - target: string
  - ttl: number (optionnel)
