# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement NS

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type NS
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement NS (serveur de noms)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge NS et bouton fermer (✕)
- Champ sous-domaine
- Champ TTL
- Champ cible
- Aide contextuelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x420)
- **OdsInput** : Sous-domaine, TTL, Cible
- **OdsBadge** : Type NS (jaune)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - NS" avec badge [NS]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "subdomain" |
| Suffixe (addon gris) | ".exemple.com" |

## 6. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL" |
| Input | 86400 |
| Unité | "secondes" |

## 7. CHAMP CIBLE
| Élément | Valeur |
|---------|--------|
| Label | "Cible" |
| Obligatoire | Oui (*) |
| Input | "ns1.exemple.com." |
| Aide | "Serveur DNS autoritaire pour cette zone (FQDN terminé par un point)" |
| Icône | ? (tooltip) |

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement NS
  - fieldType: "NS"
  - subDomain: string
  - target: FQDN du serveur DNS
  - ttl: number
