# Cahier des Charges : web-cloud/domain - Modal Transfer

## 1. VUE D'ENSEMBLE
- **Type** : Modal transfert domaine
- **Contexte** : Web Cloud Domain Transfer
- **Objectif** : Transférer un domaine vers OVHcloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsInput** : Champs domaine et authcode
- **OdsMessage** : Note informative
- **OdsRadio** : Options DNS
- **OdsButton** : Annuler, Transférer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Transférer un domaine" |
| Bouton fermer | × |
| Fond | #F9FAFB |

## 5. FORMULAIRE
| Champ | Type | Placeholder |
|-------|------|-------------|
| Nom de domaine * | Input | "example.com" |
| Code d'autorisation (AuthCode) * | Input | "Entrez le code fourni par votre registrar..." |

## 6. NOTE INFORMATIVE
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Icône | ℹ️ |
| Ligne 1 | "Le transfert peut prendre jusqu'à 5 jours." |
| Ligne 2 | "Le domaine doit être déverrouillé chez l'ancien registrar." |

## 7. OPTIONS DNS
| Option | État par défaut |
|--------|-----------------|
| Conserver mes DNS actuels | ✓ Coché |
| Utiliser les DNS OVHcloud | Non coché |

## 8. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Transférer | Primary #0050D7 | Droite |

## 9. APIS & ENDPOINTS
- POST /domain/transfer - Initier transfert
- GET /domain/{domainName}/transferLock - Vérifier verrouillage
