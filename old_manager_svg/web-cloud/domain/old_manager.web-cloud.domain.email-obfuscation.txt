# Cahier des Charges : web-cloud/domain - Obfuscation e-mails

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration de l'obfuscation des e-mails WHOIS
- **Type** : Page avec tableau et toggles
- **Objectif** : Masquer les adresses e-mail des contacts dans le WHOIS

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar navigation
- Breadcrumb
- Lien retour
- Titre
- Description
- Tableau contacts avec toggles
- Bouton sauvegarde

## 3. COMPOSANTS UI
- **OdsLink** : Retour (régénérer adresses)
- **OdsTable** : Liste contacts
- **OdsToggle** : Activer/désactiver obfuscation
- **OdsButton** : Enregistrer les modifications

## 4. BREADCRUMB
Web Cloud › Noms de domaine › **example.com**

## 5. LIEN RETOUR
← Régénérer les adresses e-mail Anti-spam

## 6. TITRE
**Obfuscation des e-mails**

## 7. DESCRIPTION
"Sélectionner les contacts pour lesquels vous souhaitez régénérer les adresses e-mail Anti-spam :"

## 8. TABLEAU CONTACTS
| Colonne | Description |
|---------|-------------|
| CONTACT | Type de contact |
| E-MAIL PERSONNELLE | Adresse réelle |
| STATUT | Toggle on/off |
| ACTION | Lien Regénérer |

### États toggle
| État | Label |
|------|-------|
| ON (bleu) | Obfusquer |
| OFF (gris) | Afficher en clair |

### Exemple données
| Contact | E-mail | Toggle | Statut |
|---------|--------|--------|--------|
| Contact propriétaire | owner@example.com | ON | Obfusquer |
| Contact administrateur | admin@example.com | OFF | Afficher en clair |
| Contact technique | tech@example.com | ON | Obfusquer |

## 9. BOUTON SAUVEGARDE
"Enregistrer les modifications" (primaire bleu)

## 10. APIS & ENDPOINTS
- GET /domain/{serviceName}/owo : État obfuscation
- PUT /domain/{serviceName}/owo : Modifier obfuscation
