# Cahier des Charges : web-cloud/domain - Modal Supprimer DynHost

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'une entrée DynHost
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer un enregistrement DynHost

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Question de confirmation avec détails
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x280)
- **OdsButton** : Annuler, Supprimer

## 4. TITRE
"Supprimer un DynHost"

## 5. QUESTION DE CONFIRMATION
"Êtes-vous sûr de vouloir supprimer l'entrée DynHost du sous-domaine **www.exemple.com** vers **192.168.1.100** ?"

## 6. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (primaire)

## 7. APIS & ENDPOINTS
- DELETE /domain/zone/{zoneName}/dynHost/record/{id} : Supprimer DynHost
