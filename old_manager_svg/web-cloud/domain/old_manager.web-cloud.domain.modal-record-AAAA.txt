# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement AAAA

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type AAAA
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement AAAA (IPv6) dans la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Bandeau info explicatif
- Champ sous-domaine
- Champ adresse IPv6
- Champ TTL
- Aperçu de l'enregistrement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x600)
- **OdsMessage** : Bandeau info bleu
- **OdsInput** : Sous-domaine, IPv6
- **OdsSelect** : TTL
- **OdsCard** : Aperçu (fond gris)
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un enregistrement AAAA"

## 5. BANDEAU INFO
Fond bleu (#E8F0FE) :
- **Titre** : "ℹ️ Enregistrement AAAA (IPv6 Address)"
- "Associe un nom de domaine à une adresse IPv6."

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Aide | "Laissez vide pour le domaine principal (@)" |
| Input | "www" |
| Suffixe (addon gris) | ".example-domain.com" |

## 7. CHAMP ADRESSE IPV6
| Élément | Valeur |
|---------|--------|
| Label | "Adresse IPv6 cible *" |
| Aide | "L'adresse IPv6 vers laquelle pointer" |
| Obligatoire | Oui |
| Input | "2001:0db8:85a3:0000:0000:8a2e:0370:7334" |

## 8. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL (Time To Live)" |
| Aide | "Durée de mise en cache de l'enregistrement" |
| Type | Select dropdown |
| Valeur | "3600 (1 heure)" |

## 9. APERÇU
Fond gris (#F9FAFB), police monospace :
`www  3600  IN  AAAA  2001:0db8:85a3::8a2e:0370:7334`

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter (primaire)

## 11. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement AAAA
  - fieldType: "AAAA"
  - subDomain: string
  - target: IPv6
  - ttl: number
