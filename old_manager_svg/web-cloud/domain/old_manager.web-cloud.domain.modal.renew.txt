**Type:** Modal formulaire
**Fonction:** Renouveler un nom de domaine
**Contexte:** Action depuis la page informations générales

## 6. LOGIQUE MÉTIER

- Les prix varient selon l'extension du domaine
- La nouvelle date = date actuelle + durée sélectionnée
- Le toggle renouvellement auto peut être modifié
- Les domaines ont une durée max de renouvellement (10 ans)

## 7. APIS & ENDPOINTS

```
GET /domain/{serviceName}/serviceInfos
→ Obtient la date d'expiration et le status de renouvellement

GET /order/cart/{cartId}/domain
Body: { domain: "exemple.com", duration: "P1Y" }
→ Calcule le prix de renouvellement

POST /order/cart/{cartId}/checkout
→ Finalise la commande

POST /domain/{serviceName}/renew
Body: {
  renew: {
    automatic: true,
    period: 12 // mois
  }
}
→ Configure le renouvellement automatique

# Durées disponibles
# P1Y = 1 an
# P2Y = 2 ans
# P5Y = 5 ans
# P10Y = 10 ans
```
