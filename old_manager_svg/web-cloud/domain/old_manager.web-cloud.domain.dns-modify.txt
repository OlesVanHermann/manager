# Cahier des Charges : web-cloud/domain - Modal Modifier entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'un enregistrement DNS
- **Type** : Formulaire modal
- **Objectif** : Éditer une entrée de la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire multi-champs
- Bandeau info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsInput** : Sous-domaine, TTL, Cible
- **OdsSelect** : Type d'enregistrement
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Enregistrer

## 4. CHAMPS DU FORMULAIRE

### Sous-domaine
- Type : Input texte
- Exemple : "www"

### Type d'enregistrement
- Type : Select dropdown
- Options : A, AAAA, CNAME, MX, TXT, NS, SRV, CAA, etc.
- Exemple sélectionné : "CNAME"

### TTL (Time To Live)
- Type : Input numérique
- Exemple : "3600"
- Note : "secondes (1 heure par défaut)"

### Cible
- Type : Input texte
- Exemple : "example.com."
- Note : Le point final est important pour les CNAME

## 5. BANDEAU INFO
Fond bleu (#E8F0FE) :
"ℹ️ Les modifications DNS peuvent prendre jusqu'à 24h pour se propager sur l'ensemble des serveurs DNS mondiaux."

## 6. BOUTONS FOOTER
- Annuler (secondaire gris)
- Enregistrer (primaire bleu)

## 7. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName}/record/{id} : Modifier enregistrement DNS
- POST /domain/zone/{zoneName}/refresh : Rafraîchir zone DNS
