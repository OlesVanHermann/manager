# Cahier des Charges : web-cloud/domain - Modal Créer accès DynHost

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un identifiant d'accès DynHost
- **Type** : Formulaire de création
- **Objectif** : Créer un compte pour mettre à jour dynamiquement une entrée DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Champ suffixe identifiant
- Champ sous-domaine
- Champ mot de passe
- Aide mot de passe
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x400)
- **OdsInput** : Suffixe, Sous-domaine, Mot de passe
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un accès DynHost"

## 5. CHAMP SUFFIXE IDENTIFIANT
| Élément | Valeur |
|---------|--------|
| Label | "Suffixe de l'identifiant" |
| Obligatoire | Oui (*) |
| Préfixe (addon gris) | "example.com-" |
| Input | "suffixe" (placeholder) |

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Obligatoire | Oui (*) |
| Input | "sous-domaine" (placeholder) |
| Suffixe (addon gris) | ".example.com" |

## 7. CHAMP MOT DE PASSE
| Élément | Valeur |
|---------|--------|
| Label | "Mot de passe" |
| Obligatoire | Oui (*) |
| Type | password (••••••••••••) |
| Aide | "Le mot de passe doit contenir au moins 8 caractères" |

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Créer (primaire)

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/dynHost/login : Créer accès DynHost
