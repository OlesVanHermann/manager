# Cahier des Charges : web-cloud/domain - Modal Activer zone DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Activation d'une zone DNS
- **Type** : Wizard de commande (2 étapes)
- **Objectif** : Activer une zone DNS pour un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Stepper 2 étapes
- Sélection de durée
- Checkbox contrats
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (800x560)
- **OdsStepper** : 2 étapes
- **OdsRadio** : Options de durée
- **OdsCheckbox** : Acceptation contrats
- **OdsButton** : Annuler, Activer

## 4. TITRE
"Activer la zone DNS"

## 5. STEPPER (2 ÉTAPES)
| Étape | Label | État |
|-------|-------|------|
| 1 | Durée | Active |
| 2 | Paiement | Inactive |

## 6. DESCRIPTION
"Sélectionnez la durée de votre zone DNS :"

## 7. OPTIONS DE DURÉE
| Option | Prix | État |
|--------|------|------|
| 12 mois | 0,00 € HT | ● Sélectionnée |
| 24 mois | 0,00 € HT | ○ |
| 36 mois | 0,00 € HT | ○ |

## 8. CHECKBOX CONTRATS
☐ "J'ai lu et j'accepte les contrats ci-dessus"

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Activer (primaire)

## 10. APIS & ENDPOINTS
- GET /order/cart/{cartId}/dnsZone : Options de zone DNS
- POST /order/cart/{cartId}/dnsZone : Ajouter zone au panier
- POST /order/cart/{cartId}/checkout : Finaliser la commande

## 11. NOTES
- Zone DNS gratuite (0,00 €)
- Nécessaire si le domaine n'a pas de zone DNS OVH
- Permet de gérer les enregistrements DNS depuis OVHcloud
