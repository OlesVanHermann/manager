# Cahier des Charges : web-cloud/domain - Modal DNS Change

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification serveurs DNS
- **Contexte** : Web Cloud Domain DNS
- **Objectif** : Changer les serveurs DNS du domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 420px |
| Overlay | rgba(0,0,0,0.5) |

- **OdsModal** : Conteneur
- **OdsText** : Titres et labels
- **OdsMessage** : Avertissement propagation
- **OdsRadio** : Type configuration

- **OdsMessage** : Info
- **OdsButton** : Annuler, Appliquer
