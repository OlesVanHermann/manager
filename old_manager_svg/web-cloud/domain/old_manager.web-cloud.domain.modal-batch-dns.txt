# Cahier des Charges : web-cloud/domain - Modal Batch DNS

## 1. VUE D'ENSEMBLE

**Type:** Modal action batch
**Fonction:** Changer les serveurs DNS de plusieurs domaines simultanément
**Contexte:** Action groupée depuis liste des domaines

## 2. DIMENSIONS

- Modal: 800x680px
- Position: centrée avec overlay

## 3. STRUCTURE MODAL

### Header
- Background: #FFFFFF
- Titre: "Changement Serveurs DNS (Batch)"
- Bouton fermer: × (coin droit)

### Content
- Padding: 24px

### Footer
- Séparateur: ligne #E5E7EB
- Boutons: Annuler (gauche), Appliquer (droite)

## 4. CONTENU

### Info banner
- Background: #DBEAFE
- Dimensions: 752x50px
- Titre: "Serveurs DNS"
- Description: "Modifiez les serveurs DNS utilisés par les domaines sélectionnés. La propagation peut prendre jusqu'à 48h."

### Formulaire DNS

| Champ | Type | Dimensions | Obligatoire |
|-------|------|------------|-------------|

### Liste domaines
- Titre: "Domaines sélectionnés (3)"
- Colonnes:

## 7. APIS & ENDPOINTS

```
PUT /domain/{serviceName}/nameServers
Body: {
  nameServers: [
    { host: "dns1.ovh.net" },
    { host: "ns1.ovh.net" },
    { host: "dns3.ovh.net" },
    { host: "ns3.ovh.net" }
  ]
}
→ Change les serveurs DNS d'un domaine

# Pour chaque domaine sélectionné, appeler:
PUT /domain/{domain}/nameServers
→ Application en batch
```
