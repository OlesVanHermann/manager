## 8. APIS & ENDPOINTS

```
GET /domain/{serviceName}/contacts
→ Liste des 4 contacts

PUT /domain/{serviceName}/contacts/{contactType}
Body: { nichandle: "ab12345-ovh" }
→ Modifie un contact (owner, admin, tech, billing)

GET /me/contact
→ Liste des contacts du compte

POST /domain/{serviceName}/changeContact
Body: {
  contactAdmin: "new-nic",
  contactTech: "new-nic",
  contactBilling: "new-nic"
}
→ Réattribue les contacts

GET /domain/{serviceName}/owo
→ Configuration OWO (Owner Whois Obfuscation)

PUT /domain/{serviceName}/owo
→ Active/désactive l'obfuscation WHOIS
```
