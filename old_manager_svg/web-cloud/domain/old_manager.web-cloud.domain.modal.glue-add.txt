# Cahier des Charges : web-cloud/domain - Modal Ajouter Glue Record

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un Glue Record
- **Type** : Formulaire de création
- **Objectif** : Définir l'IP d'un serveur DNS sur le domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau info explicatif
- Formulaire 3 champs
- Bandeau warning propagation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x520)
- **OdsMessage** : Bandeau info bleu, Bandeau warning jaune
- **OdsInput** : Nom serveur, IPv4, IPv6
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un Glue Record"

## 5. BANDEAU INFO
Fond bleu (#E8F0FE) :
- "ℹ️ Les Glue Records permettent de définir l'adresse IP d'un"
- "serveur DNS hébergé sur votre propre domaine."

## 6. CHAMP NOM SERVEUR DNS
| Élément | Valeur |
|---------|--------|
| Label | "Nom du serveur DNS" |
| Input | "ns1" |
| Suffixe | ".example.com" |

## 7. CHAMP IPv4
| Élément | Valeur |
|---------|--------|
| Label | "Adresse IPv4" |
| Placeholder | "Ex: 203.0.113.10" |
| Obligatoire | Oui |

## 8. CHAMP IPv6
| Élément | Valeur |
|---------|--------|
| Label | "Adresse IPv6 (optionnel)" |
| Placeholder | "Ex: 2001:db8::1" |
| Obligatoire | Non |

## 9. BANDEAU WARNING
Fond jaune (#FEF3C7) :
"⚠️ La propagation peut prendre jusqu'à 48h."

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter (primaire)

## 11. APIS & ENDPOINTS
- POST /domain/{serviceName}/glueRecord : Créer Glue Record
  - host: string (ex: "ns1")
  - ips: string[] (IPv4 et/ou IPv6)

## 12. NOTES
- Utilisé pour définir ns1.example.com, ns2.example.com
- Nécessaire si vous hébergez vos propres DNS sur votre domaine
- IPv4 obligatoire, IPv6 optionnel
