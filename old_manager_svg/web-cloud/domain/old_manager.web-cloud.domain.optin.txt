# Cahier des Charges : web-cloud/domain - Configuration opt-in/opt-out

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration de la visibilité WHOIS
- **Type** : Page de paramètres avec toggles
- **Objectif** : Gérer les données visibles dans le WHOIS

## 2. LAYOUT
- Header OVHcloud
- Breadcrumb : Noms de domaine > example.com > Configuration opt-in
- Titre et description
- Bandeau info RGPD
- 2 cartes de configuration
- Légende des toggles
- Bandeau warning propagation
- Boutons d'action

## 3. COMPOSANTS UI
- **OdsMessage** : Bandeau info bleu, Bandeau warning jaune
- **OdsCard** : 2 cartes (Propriétaire, Technique)
- **OdsToggle** : Toggles on/off pour chaque donnée
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Configuration opt-in / opt-out"

## 5. DESCRIPTION
"Gérez la visibilité de vos informations dans le WHOIS"

## 6. BANDEAU INFO RGPD
Fond bleu (#E8F0FE) :
- Titre : "ℹ️ Protection des données WHOIS"
- "Le règlement RGPD permet de masquer vos informations personnelles dans le WHOIS."
- "Vous pouvez choisir de rendre certaines informations publiques (opt-in) ou de les garder privées (opt-out)."

## 7. CARTE PROPRIÉTAIRE
Titre : "Informations du propriétaire"

| Donnée | État |
|--------|------|
| Nom et prénom / Raison sociale | OFF (masqué) |
| Adresse postale | OFF (masqué) |
| Numéro de téléphone | OFF (masqué) |
| Adresse email | ON (visible) |

## 8. CARTE TECHNIQUE
Titre : "Informations techniques"

| Donnée | État |
|--------|------|
| Nom du contact technique | OFF (masqué) |
| Email du contact technique | ON (visible) |
| Téléphone du contact technique | OFF (masqué) |

## 9. LÉGENDE
| Toggle | Signification |
|--------|---------------|
| OFF (gris) | Données masquées (opt-out) |
| ON (vert) | Données visibles (opt-in) |

## 10. BANDEAU WARNING
Fond jaune (#FEF3C7) :
"⚠️ Attention : Les modifications peuvent prendre jusqu'à 24h pour être appliquées dans les bases WHOIS mondiales."

## 11. BOUTONS
- Annuler (secondaire)
- Enregistrer (primaire)

## 12. APIS & ENDPOINTS
- GET /domain/{serviceName}/owo : Récupérer configuration opt-in
- PUT /domain/{serviceName}/owo : Modifier configuration opt-in
