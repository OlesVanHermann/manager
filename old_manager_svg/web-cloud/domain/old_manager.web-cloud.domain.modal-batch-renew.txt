## 1. VUE D'ENSEMBLE

**Type:** Modal commande batch
**Fonction:** Renouveler plusieurs domaines simultanément
**Contexte:** Action groupée depuis liste des domaines

## 7. APIS & ENDPOINTS

```
GET /domain/{serviceName}/serviceInfos
→ Date d'expiration actuelle

GET /order/catalog/formatted?productName=domain
→ Prix de renouvellement par extension

POST /order/cart
→ Crée un panier

POST /order/cart/{cartId}/domain
Body: {
  domain: "example.com",
  duration: "P1Y",
  planCode: "domain-renew"
}
→ Ajoute le renouvellement au panier

POST /order/cart/{cartId}/checkout
Body: { autoPayWithPreferredPaymentMethod: true }
→ Valide la commande
```
