# Cahier des Charges : web-cloud/domain - Modal Supprimer redirection

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'une redirection web
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer une redirection HTTP

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation avec détails
- Checkbox suppression associée
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x320)
- **OdsCheckbox** : Option suppression associée
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. TITRE
"Suppression d'une redirection"

## 5. QUESTION
"Êtes-vous sûr de vouloir supprimer la redirection du domaine **www.example.com** vers **https://target.com** ?"

## 6. CHECKBOX SUPPRESSION ASSOCIÉE
☐ "Supprimer également la redirection du sous-domaine **example.com** vers **https://target.com**"

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
"⚠️ Cette action est irréversible."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 9. APIS & ENDPOINTS
- DELETE /domain/zone/{zoneName}/redirection/{id} : Supprimer redirection
