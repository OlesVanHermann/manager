# Cahier des Charges : web-cloud/domain - Modal Lier sous-domaine hébergement

## 1. VUE D'ENSEMBLE
- **Modal** : Liaison d'un sous-domaine à un hébergement
- **Type** : Formulaire de configuration
- **Objectif** : Configurer un multisite sur un hébergement web

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Champ sous-domaine
- Sélection hébergement
- Champ dossier racine
- Sélection SSL
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x450)
- **OdsInput** : Sous-domaine, Dossier racine
- **OdsSelect** : Hébergement, SSL
- **OdsButton** : Annuler, Lier

## 4. TITRE
"Lier un sous-domaine à un hébergement"

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine *" |
| Obligatoire | Oui |
| Input | "blog" |
| Suffixe (addon gris) | ".example.com" |

## 6. CHAMP HÉBERGEMENT
| Élément | Valeur |
|---------|--------|
| Label | "Hébergement *" |
| Obligatoire | Oui |
| Type | Select dropdown |
| Valeur | "myhosting.cluster023.hosting.ovh.net" |

## 7. CHAMP DOSSIER RACINE
| Élément | Valeur |
|---------|--------|
| Label | "Dossier racine *" |
| Obligatoire | Oui |
| Input | "/blog" |
| Aide | "Chemin du dossier sur l'hébergement où pointeront les requêtes." |

## 8. CHAMP SSL
| Élément | Valeur |
|---------|--------|
| Label | "SSL" |
| Type | Select dropdown |
| Valeur par défaut | "Let's Encrypt (recommandé)" |
| Options | Aucun, Let's Encrypt, Certificat importé |

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Lier (primaire)

## 10. APIS & ENDPOINTS
- GET /hosting/web : Lister les hébergements du compte
- POST /hosting/web/{serviceName}/attachedDomain : Ajouter un multisite
