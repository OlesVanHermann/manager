## 6. APIS & ENDPOINTS

```
GET /domain/{serviceName}/options/anycast
→ Statut actuel de l'option

POST /domain/{serviceName}/options/anycast
→ Active DNS Anycast

GET /order/catalog/formatted?productName=dnsAnycast
→ Prix de l'option

POST /order/cart
Body: { description: "DNS Anycast" }
→ Crée un panier

POST /order/cart/{cartId}/domain
Body: { domain, planCode: "dns-anycast" }
→ Ajoute au panier
```
