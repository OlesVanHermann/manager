# Cahier des Charges : web-cloud/domain - Modal Supprimer entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un enregistrement DNS
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer une entrée de la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation
- Récapitulatif de l'enregistrement (fond gris)
- Bandeau info propagation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x380)
- **OdsCard** : Récapitulatif (fond gris #F9FAFB)
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. TITRE
"Supprimer l'entrée de la zone DNS"

## 5. QUESTION
"Voulez-vous réellement supprimer l'entrée suivante de la zone DNS du domaine *example.com* ?"

## 6. RÉCAPITULATIF ENREGISTREMENT
Fond gris (#F9FAFB), 3 lignes :
| Champ | Valeur |
|-------|--------|
| Type de champ : | A |
| Sous-domaine : | www |
| Cible : | 203.0.113.50 |

## 7. BANDEAU INFO
Fond bleu (#E8F0FE) :
"ℹ️ La suppression sera immédiate dans la zone DNS, mais veuillez prendre en compte le temps de propagation (maximum 24h)."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 9. APIS & ENDPOINTS
- DELETE /domain/zone/{zoneName}/record/{id} : Supprimer enregistrement
