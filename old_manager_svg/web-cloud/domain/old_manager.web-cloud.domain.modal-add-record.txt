# Cahier des Charges : web-cloud/domain - Modal Ajouter entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS
- **Type** : Wizard 3 étapes
- **Objectif** : Ajouter un enregistrement dans la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Stepper horizontal 3 étapes
- Zone sélection type
- Panneau aide
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsStepper** : 3 étapes (Type, Configuration, Résumé)
- **OdsToggleGroup** : Boutons sélection type
- **OdsCard** : Panneau aide
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | État |
|-------|-----|------|
| 1 | Type d'enregistrement | Actif |
| 2 | Configuration | À venir |
| 3 | Résumé | À venir |

## 5. DESCRIPTION
"Sélectionnez le type d'enregistrement DNS que vous souhaitez créer."

## 6. TYPES D'ENREGISTREMENTS

### Enregistrements de base
| Type | Description |
|------|-------------|
| A | IPv4 (sélectionné par défaut) |
| AAAA | IPv6 |
| NS | Serveur de noms |
| CNAME | Alias |
| DNAME | Alias de zone |

### Enregistrements avancés
| Type | Description |
|------|-------------|
| CAA | Autorité de certification |
| TXT | Texte |
| NAPTR | Réécriture URI |
| SRV | Service |
| LOC | Localisation |
| SSHFP | Empreinte SSH |
| TLSA | Certificat TLS |
| RP | Responsable |
| SVCB | Service Binding |
| HTTPS | HTTPS Binding |

### Enregistrements email
| Type | Description |
|------|-------------|
| MX | Serveur mail |
| SPF | Sender Policy |
| DKIM | DomainKeys |
| DMARC | Authentification |

## 7. PANNEAU AIDE
Fond gris (#F9FAFB) avec descriptions :
- **Enregistrement A** : Associe un nom de domaine à une adresse IPv4
- **Enregistrement AAAA** : Associe un nom de domaine à une adresse IPv6
- **Enregistrement CNAME** : Crée un alias vers un autre nom de domaine
- **Enregistrement MX** : Définit les serveurs de messagerie pour votre domaine

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Suivant → (primaire)

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement
