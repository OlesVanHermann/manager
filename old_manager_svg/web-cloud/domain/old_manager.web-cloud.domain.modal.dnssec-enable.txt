# Cahier des Charges : web-cloud/domain - Modal Activer DNSSEC

## 1. VUE D'ENSEMBLE
- **Modal** : Activation de DNSSEC sur le domaine
- **Type** : Modal de confirmation avec informations
- **Objectif** : Activer la protection DNSSEC

## 2. STRUCTURE MODAL
- Header avec icône bouclier vert
- Titre
- Bandeau info explicatif
- Liste des fonctionnalités
- Bandeau warning propagation
- Domaine concerné
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x580)
- **OdsIcon** : Bouclier vert avec checkmark
- **OdsMessage** : Bandeau info bleu, Bandeau warning jaune
- **OdsList** : Liste fonctionnalités (✓)
- **OdsButton** : Annuler, Activer DNSSEC

## 4. ICÔNE HEADER
- Cercle vert (#DEF7EC)
- Icône bouclier vert (#10B981) avec checkmark

## 5. TITRE
"Activer DNSSEC"

## 6. BANDEAU INFO
Fond bleu (#E8F0FE) :
- "ℹ️ DNSSEC (Domain Name System Security Extensions)"
- "protège votre domaine contre les attaques de type"
- "\"cache poisoning\" et \"man-in-the-middle\"."

## 7. FONCTIONNALITÉS
Liste avec checkmarks verts :
- ✓ Signature cryptographique des enregistrements DNS
- ✓ Validation de l'authenticité des réponses DNS
- ✓ Protection contre le détournement de domaine
- ✓ Compatible avec les navigateurs modernes

## 8. BANDEAU WARNING
Fond jaune (#FEF3C7) :
- "⚠️ L'activation de DNSSEC peut prendre jusqu'à 24h."
- "La zone DNS doit être hébergée chez OVH."

## 9. DOMAINE CONCERNÉ
"Domaine concerné : **example.com**"

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Activer DNSSEC (primaire)

## 11. APIS & ENDPOINTS
- POST /domain/{serviceName}/activateDnssec : Activer DNSSEC
