# Cahier des Charges : web-cloud/domain - Modal OWO

## 1. VUE D'ENSEMBLE

**Type:** Modal configuration
**Fonction:** Configurer la protection WHOIS (OWO - Owner Whois Obfuscation)
**Contexte:** Action depuis la page contacts ou sécurité

## 2. DIMENSIONS

- Modal: 650x540px
- Position: centrée avec overlay

## 3. STRUCTURE MODAL

### Header
- Background: #F9FAFB
- Titre: "Protection WHOIS (OWO)" (centré)
- Bouton fermer: × (coin droit)

### Content
- Padding: 30px

### Footer
- Séparateur: ligne #E5E7EB

## 4. CONTENU

### Info banner
- Background: #DBEAFE
- Dimensions: 590x50px
- Titre: "Protection des données personnelles"
- Description: "Masquez vos informations personnelles dans les résultats WHOIS publics."

### Domaine
- Label: "Domaine"
- Valeur: "exemple.com" (14px, font-weight 600)

### Niveau de protection (radio buttons)

| Option | Sélecteur | Description |
|--------|-----------|-------------|
| Désactivé | ○ | Toutes vos informations sont visibles publiquement |
| Email masqué | ● (sélectionné) | Seul l'email est remplacé par une adresse OVH anonyme |
| Protection complète | ○ | Nom, adresse, téléphone et email sont masqués |

### Success banner
- Background: #DEF7EC
- Dimensions: 590x40px
- Texte: "La protection WHOIS est gratuite et applicable immédiatement."

### Informations masquées
- Titre: "Informations masquées selon le niveau :"
- Liste:
  - Email masqué : uniquement l'adresse email
  - Protection complète : nom, adresse postale, téléphone, email

## 5. FOOTER MODAL

| Bouton | Style | Dimensions |
|--------|-------|------------|
| Annuler | btn-outline | 110x34px |
| Enregistrer | btn-primary | 120x34px |

## 6. COMPOSANTS UI

### Radio button
- Non sélectionné: cercle blanc avec border #D1D5DB
- Sélectionné: cercle #0050D7 avec point blanc au centre
- Diamètre: 16px

### Option item
- Spacing: 50px entre options
- Titre: 11px, font-weight 500
- Description: 10px, text-muted

## 7. APIS & ENDPOINTS

```
GET /domain/{serviceName}/owo
→ Configuration OWO actuelle (fields[], enabled)

PUT /domain/{serviceName}/owo
Body: {
  fields: ["email"] // ou ["name", "address", "phone", "email"]
}
→ Met à jour la protection

# Options de protection:
# - [] : désactivé
# - ["email"] : email masqué
# - ["name", "address", "phone", "email"] : protection complète
```
