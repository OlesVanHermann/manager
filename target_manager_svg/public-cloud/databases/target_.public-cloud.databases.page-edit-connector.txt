# Cahier des Charges : public-cloud/databases - Page Modifier connector (Kafka)

## 1. VUE D'ENSEMBLE
- **Page** : Modifier le connector
- **Type** : Page complète avec éditeur
- **Objectif** : Modifier la configuration JSON d'un connector existant

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Bases de données actif
- Breadcrumb avec "Edition"
- Titre "Modifier le connector"
- Lien retour "← Retour aux connectors"
- Info box bleue
- Éditeur de code JSON pleine largeur
- Bouton "Mettre à jour le connector"

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsLink** : Retour aux connectors
- **OdsMessage** : Info box bleue
- **OdsCodeEditor** : Éditeur JSON pleine largeur
- **OdsButton** : "Mettre à jour le connector" (primary)

## 4. TITRE
"Modifier le connector"

## 5. LIEN RETOUR
"← Retour aux connectors"

## 6. INFO BOX
- Fond: Bleu clair (#DBEAFE)
- Instructions:
  - "Utilisez l'éditeur ci-dessous pour modifier la configuration JSON de votre connecteur :"
  - "• Écrire ou coller directement votre configuration JSON"
  - "• Utiliser **Ctrl + espace** pour afficher les suggestions d'autocomplétion"
  - "• Survoler les propriétés pour afficher des descriptions détaillées"
  - "• Consulter la documentation officielle pour les détails"

## 7. ÉDITEUR JSON
### Dimensions
- Largeur: 1120px
- Hauteur: 400px

### Coloration syntaxique
- Fond: Noir (#1F2937)
- Numéros de ligne: Gris (#6B7280)
- Clés: Bleu (#60A5FA)
- Valeurs string: Violet (#A78BFA)
- Valeurs booléennes: Vert (#34D399)
- Texte: Blanc (#F9FAFB)

### Exemple de contenu
```json
{
  "name": "my-jdbc-sink-connector",
  "connector.class": "io.confluent.connect.jdbc.JdbcSinkConnector",
  "topics": "orders,payments",
  "connection.url": "jdbc:postgresql://db.example.com:5432/mydb",
  "connection.user": "admin",
  "auto.create": true,
  "auto.evolve": true,
  "insert.mode": "upsert",
  "pk.mode": "record_key"
}
```

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Mettre à jour le connector | Primary bleu | Sous l'éditeur | PUT connector |

## 9. FONCTIONNALITÉS ÉDITEUR
- Autocomplétion (Ctrl+Space)
- Validation JSON en temps réel
- Numéros de ligne
- Coloration syntaxique
- Indentation automatique

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}
  - Configuration actuelle du connector
- PUT /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}
  - Body: { configuration }
  - Mettre à jour la configuration
