- **Page** : Databases - Onglet Topics (Kafka)
- **Type** : Page onglet listing
- **Objectif** : Gérer les topics Kafka

## 6. TABLEAU TOPICS
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Topic | Texte | Nom du topic |
| Partitions | Nombre | Nombre de partitions |
| Réplication | Nombre | Facteur de réplication |
| Min in-sync | Nombre | Réplicas min sync |
| Rétention (h) | Texte | Durée ou "illimité" |
| Politique | Texte | delete / compact |
| Actions | Menu ⋮ | ActionMenu |

### Données exemple
| Topic | Partitions | Réplication | Min in-sync | Rétention | Politique |
|-------|------------|-------------|-------------|-----------|-----------|
| orders-events | 12 | 3 | 2 | 168 | delete |
| user-activities | 6 | 3 | 2 | illimité | compact |
| notifications | 3 | 3 | 1 | 24 | delete |

## 7. ACTION MENU (ouvert)
| Action | Type |
|--------|------|
| Modifier | Standard |
| Supprimer | Danger (rouge) |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/kafka/{serviceId}/topic
  - Liste des topics
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/topic
  - Body: { name, partitions, replication, minInsyncReplicas, retentionHours, cleanupPolicy }
  - Créer un topic
- PUT /cloud/project/{projectId}/database/kafka/{serviceId}/topic/{topicId}
  - Modifier un topic
- DELETE /cloud/project/{projectId}/database/kafka/{serviceId}/topic/{topicId}
  - Supprimer un topic
