# Cahier des Charges : public-cloud/databases - Modal Modifier utilisateur

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier un utilisateur
- **Type** : Modal formulaire
- **Objectif** : Modifier les propriétés d'un utilisateur existant

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x400px
- Header avec titre
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom d'utilisateur (disabled)
- **OdsInput** : Groupe
- **OdsSelect** : Rôles
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier un utilisateur"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur |
|-------|------|------|--------|
| Nom d'utilisateur | Input | Disabled | admin_user |
| Groupe | Input | Éditable | default_group |
| Rôles | Select | Éditable | readWrite@mydb |

### Notes
- Le nom d'utilisateur n'est pas modifiable
- Fond grisé pour le champ disabled
- Les rôles peuvent être modifiés selon l'engine

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Modifier | Primary | Droite | PUT user |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/database/{engine}/{serviceId}/user/{userId}
  - Body: { group?, roles? }
  - Modifier un utilisateur
