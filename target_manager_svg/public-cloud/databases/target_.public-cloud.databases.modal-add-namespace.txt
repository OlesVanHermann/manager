# Cahier des Charges : public-cloud/databases - Modal Ajouter namespace (M3DB)

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un namespace
- **Type** : Modal formulaire
- **Objectif** : Créer un nouveau namespace M3DB

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 550x460px
- Header avec titre et description
- Formulaire avec 4 champs + accordéon
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom, Durée rétention, Résolution
- **OdsRadio** : Type (Agrégé)
- **OdsAccordion** : Configurations avancées
- **OdsButton** : Annuler, Créer un namespace

## 4. TITRE
"Ajouter un namespace"

## 5. DESCRIPTION
"Créer un nouveau namespace sur votre service."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nom du namespace | Input | name | Oui |
| Type | Radio | Agrégé (sélectionné) | Oui |
| Durée de rétention | Input | Ex: 48H | Oui |
| Résolution | Input | Ex: 1D | Oui |

### Types de namespace
| Type | Description |
|------|-------------|
| Agrégé | Données agrégées selon résolution |
| Non agrégé | Données brutes |

### Format durées
- H : heures (ex: 48H)
- D : jours (ex: 7D)
- M : mois (ex: 1M)

### Configurations avancées
- writesToCommitLogEnabled
- snapshotEnabled
- Autres paramètres M3DB

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Créer un namespace | Primary | Droite | POST namespace |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/m3db/{serviceId}/namespace
  - Body: { name, type, retention, resolution, config? }
  - Créer un namespace
