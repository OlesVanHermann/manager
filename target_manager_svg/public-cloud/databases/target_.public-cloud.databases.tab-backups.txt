# Cahier des Charges : public-cloud/databases - Onglet Sauvegardes

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Sauvegardes
- **Type** : Page onglet listing
- **Objectif** : Gérer les backups d'un service DB

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Databases actif
- Breadcrumb
- Header service avec badges
- Onglets (Sauvegardes actif)
- Titre avec compteur
- Bouton créer backup
- Tableau des backups

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Sauvegardes actif
- **OdsBadge** : Compteur (5)
- **OdsButton** : "Créer une sauvegarde" (primaire)
- **OdsTable** : Liste des backups
- **OdsBadge** : Statut (READY)
- **OdsButton** : "Restaurer" (outline)
- **OdsButton** : "Supprimer" (danger)

## 4. TITRE
"Sauvegardes" avec badge compteur (5)

## 5. ONGLETS
| Onglet | État |
|--------|------|
| General | Inactif |
| Utilisateurs | Inactif |
| Sauvegardes | Actif (souligné bleu) |
| Metriques | Inactif |

## 6. TABLEAU BACKUPS
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Date | Date heure | Timestamp du backup |
| Taille | Texte | Taille en GB |
| Statut | Badge | READY, IN_PROGRESS |
| Actions | Boutons | Restaurer / Supprimer |

### Données exemple
| Date | Taille | Statut | Actions |
|------|--------|--------|---------|
| 27/12/2024 03:00 | 2.4 GB | READY | [Restaurer] [Supprimer] |
| 26/12/2024 03:00 | 2.3 GB | READY | [Restaurer] [Supprimer] |
| 25/12/2024 03:00 | 2.2 GB | READY | [Restaurer] [Supprimer] |

### Badge Statut
- READY: Vert (#DEF7EC, #03543F)

### Boutons Actions
- Restaurer: Outline blanc avec bordure
- Supprimer: Rouge clair (#FEE2E2, #991B1B)

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/service/{serviceId}/backup
  - Liste des backups
- POST /cloud/project/{projectId}/database/service/{serviceId}/backup
  - Créer un backup manuel
- POST /cloud/project/{projectId}/database/service/{serviceId}/backup/{backupId}/restore
  - Restaurer un backup
- DELETE /cloud/project/{projectId}/database/service/{serviceId}/backup/{backupId}
  - Supprimer un backup
