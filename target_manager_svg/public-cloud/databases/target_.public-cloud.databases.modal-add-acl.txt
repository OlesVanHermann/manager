# Cahier des Charges : public-cloud/databases - Modal Créer ACL (Kafka)

## 1. VUE D'ENSEMBLE
- **Modal** : Créer une ACL
- **Type** : Modal formulaire
- **Objectif** : Créer une Access Control List pour un utilisateur Kafka

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x340px
- Header avec titre et description
- Formulaire avec 3 champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom d'utilisateur, Topic
- **OdsSelect** : Permission
- **OdsButton** : Annuler, Créer une ACL

## 4. TITRE
"Créer une ACL"

## 5. DESCRIPTION
"Créez une ACL pour un utilisateur sur un topic spécifique."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nom d'utilisateur | Input | admin | Oui |
| Topic | Input | my_topic | Oui |
| Permission | Select | read | Oui |

### Options Permission
| Permission | Description |
|------------|-------------|
| read | Lecture du topic |
| write | Écriture dans le topic |
| admin | Administration du topic |

### Wildcards
- Le topic peut utiliser des wildcards (*) pour matcher plusieurs topics
- Exemple: "orders_*" matche tous les topics commençant par "orders_"

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Créer une ACL | Primary | Droite | POST ACL |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/acl
  - Body: { username, topic, permission }
  - Créer une ACL
