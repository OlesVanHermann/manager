# Cahier des Charges : public-cloud/databases - Modal Supprimer connector (Kafka)

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un connector
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'un Kafka connector

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x220px
- Header avec titre et bouton fermer
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer (danger)

## 4. TITRE
"Supprimer un connector"

## 5. MESSAGE
"Vous allez supprimer votre connector **{connector_name}**.

Souhaitez-vous continuer ?"

### Variables
- `{connector_name}` : Nom du connector (ex: my-jdbc-sink-connector)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Confirmer | Danger rouge | Droite | DELETE connector |

## 7. AVERTISSEMENTS
- Le connector sera immédiatement arrêté
- Les tâches en cours seront interrompues
- Action irréversible

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}
  - Supprimer un connector
