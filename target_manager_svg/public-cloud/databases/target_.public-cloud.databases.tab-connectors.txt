# Cahier des Charges : public-cloud/databases - Onglet Connectors (Kafka)

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Connectors (Kafka Connect)
- **Type** : Page onglet listing
- **Objectif** : Gérer les Kafka Connectors (source/sink)

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec service Kafka actif
- Breadcrumb avec "Connectors"
- Onglets (Connectors actif)
- Titre "Connectors"
- Bouton ajouter connector
- Tableau des connectors
- ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Connectors actif
- **OdsButton** : "+ Ajouter un connector" (outline)
- **OdsTable** : Liste des connectors
- **OdsBadge** : Statut (RUNNING, PAUSED)
- **OdsIcon** : Icône direction (↓ sink, ↑ source)
- **OdsActionMenu** : Actions

## 4. TITRE
"Connectors"

## 5. TABLEAU CONNECTORS
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Texte | Nom du connector |
| Connector | Icône + infos | Type, version, auteur |
| Statut | Badge pill | RUNNING / PAUSED |
| Actions | Menu ⋮ | ActionMenu |

### Icônes Direction
- Sink (↓): Cercle bleu avec flèche bas
- Source (↑): Cercle bleu avec flèche haut

### Badges Statut
| Statut | Couleur |
|--------|---------|
| RUNNING | Vert (#DEF7EC, #03543F) |
| PAUSED | Jaune (#FEF3C7, #92400E) |

### Données exemple
| Nom | Connector | Statut |
|-----|-----------|--------|
| my-sink-connector | ↓ Elasticsearch Sink (v14.0.8, Aiven) | RUNNING |
| my-source-connector | ↑ Debezium MySQL Source (v2.4.1, Debezium) | PAUSED |

## 6. ACTION MENU (ouvert)
| Action | Type |
|--------|------|
| Gérer les tâches | Standard |
| Mettre en pause | Standard |
| Redémarrer | Standard |
| Modifier | Standard |
| Supprimer | Danger (rouge) |

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/kafka/{serviceId}/connector
  - Liste des connectors
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/connector
  - Créer un connector
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}/pause
  - Mettre en pause
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}/resume
  - Reprendre
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}/restart
  - Redémarrer
- DELETE /cloud/project/{projectId}/database/kafka/{serviceId}/connector/{connectorId}
  - Supprimer
