# Cahier des Charges : public-cloud/databases - Modal Regénérer mot de passe (succès)

## 1. VUE D'ENSEMBLE
- **Modal** : Regénérer le mot de passe
- **Type** : Modal résultat (état succès)
- **Objectif** : Afficher le nouveau mot de passe généré

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x340px
- Header avec titre et bouton fermer
- Zone succès avec message et mot de passe
- Footer avec bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Zone succès verte
- **OdsCode** : Affichage du mot de passe
- **OdsClipboard** : Copier le mot de passe
- **OdsButton** : Fermer

## 4. TITRE
"Regénérer le mot de passe"

## 5. ZONE SUCCÈS
### Structure
- Fond: Vert clair (#ECFDF5)
- Bordure: Vert (#059669)
- Icône: ✓ Succès

### Messages
- "Le mot de passe de l'utilisateur a été régénéré avec succès."
- "Veuillez le noter car il ne pourra pas être récupéré par la suite."

### Affichage du mot de passe
- Fond: Gris (#F3F4F6)
- Police: Monospace
- Mot de passe: xK9#mP2$vL5@nQ8!wR3&jT6*
- Bouton copier: Bleu avec icône clipboard

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Copier | Primary (dans le bloc) | À droite du mot de passe | Copier |
| Fermer | Outline | Footer droite | Fermer modal |

## 7. SÉCURITÉ
- Le mot de passe n'est affiché qu'une seule fois
- Recommander de stocker dans un gestionnaire de mots de passe
- Ne pas partager en clair

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/{engine}/{serviceId}/user/{userId}/credentials/reset
  - Retourne: { password }
