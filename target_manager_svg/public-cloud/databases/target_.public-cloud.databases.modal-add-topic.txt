# Cahier des Charges : public-cloud/databases - Modal Créer topic (Kafka)

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un topic
- **Type** : Modal formulaire
- **Objectif** : Créer un nouveau topic Kafka

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x420px
- Header avec titre et description
- Formulaire avec 3 champs + accordéon
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom, Réplication, Partitions
- **OdsAccordion** : Configuration avancée
- **OdsButton** : Annuler, Créer un topic

## 4. TITRE
"Créer un topic"

## 5. DESCRIPTION
"Créez un nouveau topic Kafka pour gérer les flux de données entre les services."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Valeur défaut | Requis |
|-------|------|---------------|--------|
| Nom du topic | Input | my_topic | Oui |
| Réplication | Input (number) | 3 | Oui |
| Partitions | Input (number) | 1 | Oui |

### Configuration avancée (accordéon)
- retention.ms
- retention.bytes
- cleanup.policy
- min.insync.replicas
- Et autres paramètres Kafka

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Créer un topic | Primary | Droite | POST topic |

## 8. VALIDATION
- Nom: caractères alphanumériques, tirets, underscores
- Réplication: max = nombre de brokers
- Partitions: min 1

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/kafka/{serviceId}/topic
  - Body: { name, replication, partitions, config? }
  - Créer un topic
