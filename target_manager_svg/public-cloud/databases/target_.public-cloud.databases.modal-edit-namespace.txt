# Cahier des Charges : public-cloud/databases - Modal Modifier namespace (M3DB)

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier un namespace
- **Type** : Modal formulaire
- **Objectif** : Modifier les propriétés d'un namespace M3DB existant

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 550x420px
- Header avec titre
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom (disabled), Durée rétention, Résolution
- **OdsRadio** : Type (disabled)
- **OdsButton** : Annuler, Modifier ce namespace

## 4. TITRE
"Modifier un namespace"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur |
|-------|------|------|--------|
| Nom du namespace | Input | Disabled | metrics_1d |
| Type | Radio | Disabled | Agrégé |
| Durée de rétention | Input | Éditable | 48H |
| Résolution | Input | Éditable | 1D |

### Notes
- Le nom et le type ne sont pas modifiables après création
- Fond grisé pour les champs disabled

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Modifier ce namespace | Primary | Droite | PUT namespace |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/database/m3db/{serviceId}/namespace/{namespaceId}
  - Body: { retention?, resolution? }
  - Modifier un namespace
