# Cahier des Charges : public-cloud/databases - Modal Supprimer le service

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le service
- **Type** : Modal confirmation critique
- **Objectif** : Confirmer la suppression définitive d'un service de base de données

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x320px
- Header avec fond jaune warning
- Message de confirmation
- Champ de saisie TERMINATE
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Header warning (fond jaune)
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. TITRE
"Supprimer le service"

## 5. HEADER WARNING
- Fond: Jaune (#FEF3C7)
- Indique une action dangereuse

## 6. MESSAGE
"Etes-vous sur de vouloir supprimer le service {service_name} ?"

### Variables
- `{service_name}` : Nom du service (ex: my-postgresql-db)

## 7. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Validation |
|-------|------|-------------|------------|
| Confirmation | Input | TERMINATE | Doit être exactement "TERMINATE" |

### Label
"Entrez « TERMINATE » pour confirmer votre choix."

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Supprimer | Primary | Droite | DELETE service |

### État du bouton
- Désactivé tant que "TERMINATE" n'est pas saisi correctement

## 9. AVERTISSEMENTS
- Action irréversible
- Toutes les données seront perdues
- Les sauvegardes ne sont pas supprimées automatiquement

## 10. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/service/{serviceId}
  - Supprimer le service
