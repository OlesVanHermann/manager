# Cahier des Charges : public-cloud/databases - Modal Mettre à jour la version

## 1. VUE D'ENSEMBLE
- **Modal** : Mettre à jour la version
- **Type** : Modal sélection avec avertissements
- **Objectif** : Migrer vers une version plus récente de l'engine

## 2. STRUCTURE MODAL
- Overlay sombre avec fond page visible
- Conteneur blanc 700x580px
- Header avec titre et bouton fermer
- Affichage version actuelle
- Sélection des versions disponibles (radio)
- Warning box avec impacts
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur large
- **OdsCard** : Version actuelle (gris)
- **OdsRadio** : Sélection version
- **OdsBadge** : "Actuel", "Nouveau"
- **OdsMessage** : Warning impacts
- **OdsButton** : Annuler, Mettre à jour

## 4. TITRE
"Mettre à jour la version"

## 5. VERSION ACTUELLE
- Fond: Gris (#F3F4F6)
- Label: "Version actuelle"
- Valeur: "MySQL 8.0.32"
- Badge: "Actuel" (vert)

## 6. SÉLECTION NOUVELLE VERSION
### Structure
| Version | Description | Badge |
|---------|-------------|-------|
| MySQL 8.0.35 | Correctifs de sécurité et améliorations | Nouveau |
| MySQL 8.0.36 (sélectionné) | Dernière version stable avec nouvelles fonctionnalités | Nouveau |
| MySQL 8.4.0 LTS | Version Long Term Support - recommandée pour la production | - |

### Visuel sélection
- Non sélectionné: Bordure grise, fond blanc
- Sélectionné: Bordure bleue, fond bleu clair

## 7. WARNING BOX
- Fond: Jaune (#FEF3C7)
- Titre: "⚠️ Important"
- Points:
  - "La mise à jour nécessitera un redémarrage du service (~5-10 min)"
  - "Nous recommandons de faire une sauvegarde avant la mise à jour"
  - "La rétrogradation vers une version antérieure n'est pas possible"

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Mettre à jour | Primary bleu | Droite | POST update version |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/capabilities/{engine}/version
  - Liste des versions disponibles
- POST /cloud/project/{projectId}/database/service/{serviceId}/update
  - Body: { version }
  - Lancer la mise à jour
