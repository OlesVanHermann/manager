# Cahier des Charges : public-cloud/databases - Modal Supprimer ACL (Kafka)

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer une ACL
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'une Access Control List

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 420x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. TITRE
"Supprimer une ACL"

## 5. MESSAGE
"Voulez-vous vraiment supprimer l'ACL {acl_name} ?"

### Variables
- `{acl_name}` : Nom de l'ACL (ex: admin)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Supprimer | Primary | Droite | DELETE ACL |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/kafka/{serviceId}/acl/{aclId}
  - Supprimer une ACL
