# Cahier des Charges : public-cloud/databases - Onglet Métriques

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Métriques
- **Type** : Page onglet monitoring
- **Objectif** : Afficher les métriques de performance d'un service DB

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Databases actif
- Breadcrumb
- Header service avec badges
- Onglets (Métriques actif)
- Titre "Métriques"
- Carte métriques actuelles (KPIs)
- Carte historique (graphiques)

- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Metriques actif
- **OdsCard** : 2 cartes
- **OdsText** : Valeurs KPI (28px, bold)
- **OdsChart** : Graphique historique (placeholder)
