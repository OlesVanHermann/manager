# Cahier des Charges : public-cloud/databases - Modal Supprimer namespace (M3DB)

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un namespace
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'un namespace M3DB

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 420x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Supprimer un namespace"

## 5. MESSAGE
"Voulez-vous vraiment supprimer le namespace {namespace_name} ?"

### Variables
- `{namespace_name}` : Nom du namespace (ex: metrics_1d)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Confirmer | Primary | Droite | DELETE namespace |

## 7. AVERTISSEMENT
- Toutes les métriques du namespace seront perdues
- Action irréversible

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/m3db/{serviceId}/namespace/{namespaceId}
  - Supprimer un namespace
