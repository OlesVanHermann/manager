# Cahier des Charges : public-cloud/databases - Modal Créer base de données

## 1. VUE D'ENSEMBLE
- **Modal** : Créer une base de données
- **Type** : Modal formulaire
- **Objectif** : Ajouter une base de données au service

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x220px
- Header avec titre et description
- Formulaire avec 1 champ
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom de la base de données
- **OdsButton** : Annuler, Créer la base de données

## 4. TITRE
"Créer une base de données"

## 5. DESCRIPTION
"Ajouter une base de données à votre service"

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nom de la base de données | Input | my_database | Oui |

### Validation
- Caractères alphanumériques et underscores
- Longueur maximale selon l'engine

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Créer la base de données | Primary | Droite | POST database |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/database/{engine}/{serviceId}/database
  - Body: { name }
  - Créer une base de données
