# Cahier des Charges : public-cloud/databases - Modal Supprimer utilisateur

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un utilisateur
- **Type** : Modal confirmation
- **Objectif** : Confirmer la suppression d'un utilisateur

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x160px
- Header avec titre
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Supprimer un utilisateur"

## 5. MESSAGE
"Voulez-vous vraiment supprimer l'utilisateur {username} ?"

### Variables
- `{username}` : Nom de l'utilisateur Ã  supprimer (ex: admin_user)

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline | Gauche | Fermer modal |
| Confirmer | Primary | Droite | DELETE user |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/database/{engine}/{serviceId}/user/{userId}
  - Supprimer un utilisateur
