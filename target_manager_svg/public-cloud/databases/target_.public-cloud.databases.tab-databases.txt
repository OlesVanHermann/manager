# Cahier des Charges : public-cloud/databases - Onglet Bases de données

## 1. VUE D'ENSEMBLE
- **Page** : Databases - Onglet Bases de données (PostgreSQL/MySQL)
- **Type** : Page onglet listing
- **Objectif** : Gérer les bases de données d'un service

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec service actif
- Breadcrumb avec "Bases de données"
- Onglets (Bases de données actif)
- Titre "Bases de données"
- Bouton créer base
- Tableau des bases
- ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets avec Bases de données actif
- **OdsButton** : "+ Créer une base de données" (outline)
- **OdsTable** : Liste des bases
- **OdsActionMenu** : Actions

## 4. TITRE
"Bases de données"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | Inactif |
| Utilisateurs | Inactif |
| Bases de données | Actif (souligné bleu) |
| Logs | Inactif |
| Métriques | Inactif |
| Sauvegardes | Inactif |
| Paramètres | Inactif |

## 6. TABLEAU BASES DE DONNÉES
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Texte | Nom de la base |
| Actions | Menu ⋮ | ActionMenu |

### Données exemple
| Nom | Note |
|-----|------|
| defaultdb | (par défaut) - grisé |
| app_production | - |
| app_staging | - |
| analytics | - |

## 7. ACTION MENU (ouvert)
| Action | Type |
|--------|------|
| Supprimer | Danger (rouge) |

Note: La base "defaultdb" ne peut pas être supprimée.

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/database/{engine}/{serviceId}/database
  - Liste des bases de données
- POST /cloud/project/{projectId}/database/{engine}/{serviceId}/database
  - Body: { name }
  - Créer une base
- DELETE /cloud/project/{projectId}/database/{engine}/{serviceId}/database/{databaseId}
  - Supprimer une base
