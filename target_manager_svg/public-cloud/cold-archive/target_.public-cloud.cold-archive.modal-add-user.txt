# Cahier des Charges : public-cloud/cold-archive - Modal Ajouter Utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal ajouter utilisateur à un conteneur
- **Type** : Modal formulaire avec dropdown
- **Objectif** : Associer un utilisateur S3 existant à un conteneur

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 700x500px centrée
- Header avec titre et bouton fermer
- Description
- Select dropdown avec liste utilisateurs
- Badges statut credentials
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (700x500px)
- **OdsText** : Description
- **OdsSelect** : Sélection utilisateur (dropdown ouvert)
- **OdsBadge** : Statut credentials (vert/rouge)
- **OdsButton** : Annuler (outline)
- **OdsButton** : Suivant (primaire)

## 4. TITRE
"Ajouter un utilisateur à mon conteneur"

## 5. DESCRIPTION
"Sélectionnez l'utilisateur à ajouter à votre conteneur. S'il n'apparaît pas dans cette liste, veuillez le créer dans la section dédiée."

## 6. SÉLECTEUR UTILISATEUR
### Label
"Sélectionner un utilisateur"

### Options (dropdown ouvert)
| Utilisateur | Badge statut |
|-------------|--------------|
| user-cold-archive-prod | Credentials générés (vert) |
| admin-storage | Credentials générés (vert) |
| backup-service | Credentials générés (vert) |
| new-user-test | Pas de credentials (rouge) |

### Badges
| Statut | Couleur |
|--------|---------|
| Credentials générés | Vert (#DEF7EC, #03543F) |
| Pas de credentials | Rouge (#FEE2E2, #991B1B) |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Suivant | Primaire bleu | Droite |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/user
  - Liste des utilisateurs avec statut credentials
- POST /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/user
  - Body: { userId: string }
  - Ajouter l'utilisateur au conteneur
