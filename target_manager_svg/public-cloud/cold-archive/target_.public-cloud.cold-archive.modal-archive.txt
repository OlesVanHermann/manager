# Cahier des Charges : public-cloud/cold-archive - Modal Archiver

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal archiver conteneur
- **Type** : Modal formulaire
- **Objectif** : Archiver un conteneur avec option de rétention

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 700x580px centrée
- Header avec titre et bouton fermer
- Description
- Options radio (rétention ou non)
- Input durée de rétention
- Message warning avec date fin
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (700x580px)
- **OdsText** : Description
- **OdsRadio** : 2 options (rétention ou non)
- **OdsQuantity** : Nombre de jours (−/+)
- **OdsMessage** : Warning avec date fin (jaune)
- **OdsButton** : Annuler (outline)
- **OdsButton** : Archiver (primaire)

## 4. TITRE
"Archiver votre conteneur"

## 5. DESCRIPTION
"Vous êtes sur le point d'archiver votre conteneur **backup-archives-2024**."
"En configurant une durée de rétention, votre archive sera verrouillée et vous ne pourrez plus supprimer vos données pendant toute la période de rétention."

## 6. OPTIONS RADIO
| Option | État | Description |
|--------|------|-------------|
| Planifier une durée de rétention | Sélectionné | Affiche le champ de durée |
| Ne pas planifier de durée de rétention | Non sélectionné | Archive sans verrouillage |

## 7. CHAMP DURÉE (si option 1 sélectionnée)
- Label: "Ajouter le nombre de jours de rétention"
- Type: OdsQuantity
- Boutons: − / +
- Valeur: 90 jours

## 8. MESSAGE WARNING
- Fond: Jaune (#FEF3C7)
- Bordure: Orange (#F59E0B)
- Contenu:
  - "Votre durée de rétention prendra fin le **27/03/2025**."
  - "Vous ne pourrez procéder à la suppression de vos données qu'une fois cette date passée."

## 9. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Archiver | Primaire bleu | Droite |

## 10. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionId}/coldArchive/container/{containerId}/archive
  - Body: { retentionDays?: number }
  - Archive le conteneur avec ou sans rétention
