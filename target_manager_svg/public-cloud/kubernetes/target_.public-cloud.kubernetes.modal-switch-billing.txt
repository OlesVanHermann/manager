# Cahier des Charges : public-cloud/kubernetes - Modal Facturation Mensuelle

## 1. VUE D'ENSEMBLE
- **Modal** : Passer à la facturation mensuelle
- **Type** : Modal confirmation simple
- **Objectif** : Basculer un nœud de facturation horaire vers mensuelle

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x240px
- Titre + bouton fermer
- Question de confirmation
- Info sur la facturation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsText** : Question et information
- **OdsButton** : Annuler (outline bleu), Confirmer (primary bleu)

## 4. TITRE
"Passer à la facturation mensuelle"

## 5. CONTENU

### Question
- "Êtes-vous sûr de vouloir basculer la facturation du nœud"
- "'node-pool-abc-123' vers la facturation mensuelle ?"

### Information
- "Vous serez immédiatement facturé au prorata des jours"
- "restants du mois en cours."

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Confirmer | Primary bleu | Droite | POST switch billing |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/kube/{kubeId}/nodepool/{poolId}/nodes/{nodeId}/switchToMonthlyBilling
  - Passer en facturation mensuelle
