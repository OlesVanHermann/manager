# Cahier des Charges : public-cloud/kubernetes - Modal Supprimer Cluster

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le cluster
- **Type** : Modal danger avec confirmation TERMINATE
- **Objectif** : Supprimer définitivement un cluster Kubernetes

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 700x420px
- Icône danger rouge + titre + bouton fermer
- Question de confirmation
- Banner warning
- Champ confirmation "TERMINATE"
- Info cluster (gris)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Warning rouge (#FEE2E2)
- **OdsMessage** : Warning (jaune)
- **OdsInput** : Confirmation
- **OdsCard** : Info cluster (gris)
- **OdsButton** : Annuler, Confirmer (danger)

## 4. TITRE
"Supprimer le cluster"

## 5. QUESTION
"Êtes-vous sûr(e) de vouloir supprimer le cluster **my-kubernetes-cluster** ?"

## 6. BANNER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- Icône ⚠
- "La suppression entraînera l'effacement définitif de toutes les données correspondantes. L'action de suppression n'octroie aucun remboursement."

## 7. CHAMP CONFIRMATION
- Label : "Entrez 'TERMINATE' dans le champ ci-dessous pour confirmer"
- Input : TERMINATE

## 8. INFO CLUSTER
- Fond gris (#F3F4F6)
- "Cluster : **my-kubernetes-cluster** • GRA7 • Version 1.29"

## 9. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Confirmer | Danger rouge | Droite | DELETE cluster |

## 10. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/kube/{kubeId}
  - Supprimer le cluster
