# Cahier des Charges : public-cloud/kubernetes - Modal Renommer Cluster

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le nom
- **Type** : Modal formulaire simple
- **Objectif** : Renommer le cluster

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x320px
- Titre + bouton fermer
- Info nom par défaut (UUID)
- Champ nom personnalisé
- Texte d'aide (règles de nommage)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Info et aide
- **OdsInput** : Nom personnalisé (focus bleu)
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le nom"

## 5. NOM PAR DÉFAUT
- Label : "Nom par défaut"
- Valeur : `a1b2c3d4-e5f6-7890-abcd-ef1234567890` (monospace)

## 6. CHAMP NOM PERSONNALISÉ
| Champ | Type | Valeur |
|-------|------|--------|
| Nom personnalisé | Input | my-production-cluster |

## 7. TEXTE D'AIDE
"Le nom du cluster doit contenir entre 2 et 64 caractères, commencer par une lettre, se terminer par un caractère alphanumérique et ne contenir que des lettres, chiffres, tirets ou underscores."

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Modifier | Primary bleu | Droite | PUT rename |

## 9. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/kube/{kubeId}
  - Body: { name }
  - Renommer le cluster
