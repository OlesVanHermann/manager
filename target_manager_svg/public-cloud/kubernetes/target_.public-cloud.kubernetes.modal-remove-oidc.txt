# Cahier des Charges : public-cloud/kubernetes - Modal Supprimer OIDC

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le fournisseur OIDC
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer la configuration OIDC du cluster

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x200px avec barre orange en haut
- Titre + bouton fermer
- Question de confirmation avec nom du provider
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact avec accent orange
- **OdsText** : Question de confirmation
- **OdsButton** : Annuler (outline bleu), Confirmer (primary bleu)

## 4. HEADER WARNING
- Barre orange (#F59E0B) en haut du modal (6px)
- Titre : "Supprimer le fournisseur OIDC"
- Bouton fermer (×)

## 5. QUESTION
- Ligne 1 : "Etes-vous sûr de supprimer le provider OIDC"
- Ligne 2 (bold) : "cluster-kubernetes-production ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Confirmer | Primary bleu | Droite | DELETE OIDC |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/kube/{kubeId}/openIdConnect
  - Supprimer la configuration OIDC
