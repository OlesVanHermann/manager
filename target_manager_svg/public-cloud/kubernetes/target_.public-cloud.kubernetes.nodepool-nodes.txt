# Cahier des Charges : public-cloud/kubernetes - Page Node Pool Nodes

## 1. VUE D'ENSEMBLE
- **Page** : Liste des nœuds d'un pool
- **Type** : Page détail avec datagrid
- **Objectif** : Afficher et gérer les nœuds d'un pool

## 2. STRUCTURE PAGE
- Header avec breadcrumb
- Titre du pool + lien retour
- Onglets : Service, Node pools (actif), Restrictions API, Logs
- Card info pool (modèle, nœuds, autoscaling, facturation)
- Bouton "Redimensionner"
- Titre "Nœuds" + bouton "+ Ajouter un nœud"
- Tableau des nœuds

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsLink** : Retour au cluster
- **OdsTabs** : Onglets
- **OdsCard** : Infos du pool
- **OdsButton** : Redimensionner, + Ajouter un nœud
- **OdsDatagrid** : Tableau des nœuds
- **OdsBadge** : Ok (vert)

## 4. TITRE
"nodepool-1"

| Info | Valeur |

## 6. SECTION NŒUDS

- GET /cloud/project/{projectId}/kube/{kubeId}/nodepool/{poolId}/nodes
