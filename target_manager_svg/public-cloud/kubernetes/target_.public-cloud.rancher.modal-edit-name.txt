# Cahier des Charges : public-cloud/rancher - Modal Modifier Nom

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le nom
- **Type** : Modal formulaire simple
- **Objectif** : Renommer le service Rancher

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x300px)
- Titre avec close
- Champ de saisie
- Helper text
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom du service
- **OdsButton** : Annuler (secondary), Modifier (primary)

## 4. HEADER
"Modifier le nom" + bouton ×

## 5. CHAMP
- **Label** : "Nom du service"
- **Input** : Champ texte pré-rempli
- **Valeur** : "rancher-prod-01"

## 6. HELPER TEXT
"Le nom doit comporter entre 3 et 63 caractères, commencer et se terminer par des caractères alphanumériques minuscules (a-z, 0-9) et peut contenir: _ . -"

## 7. BOUTONS
- Annuler (secondary)
- Modifier (primary)

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/rancher/{rancherId}
  - Body: { name }
  - Modifie le nom du service
