# Cahier des Charges : public-cloud/kubernetes - Liste des Clusters

## 1. VUE D'ENSEMBLE
- **Page** : Liste des clusters Kubernetes
- **Type** : Datagrid avec ActionMenu
- **Objectif** : Afficher et gérer les clusters Kubernetes

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation
- Sidebar avec menu (Kubernetes actif)
- Breadcrumb : Public Cloud / Mon Projet Cloud / Kubernetes
- Titre "Kubernetes" + bouton création
- Barre de recherche
- Tableau des clusters
- ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsButton** : "+ Créer un cluster Kubernetes"
- **OdsInput** : Recherche
- **OdsDatagrid** : Tableau des clusters
- **OdsBadge** : Plan (Free/Standard), Statut (OK vert)
- **OdsActionMenu** : Actions contextuelles

## 4. TITRE
"Kubernetes"

## 5. COLONNES DU TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom du cluster | Link bleu | Nom cliquable vers détail |
| ID | Text small | UUID tronqué |
| Plan | Badge | Free (bleu), Standard (jaune) |
| Région | Text | GRA9, SBG5, etc. |
| Réseau attaché | Text | Nom vRack ou "Public" |
| Version | Text | Version K8s (1.28, 1.29) |
| Statut | Badge pill | OK (vert) |
| Actions | Menu ⋮ | ActionMenu |
