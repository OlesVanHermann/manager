# Cahier des Charges : public-cloud/rancher - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer votre Managed Rancher Service
- **Type** : Modal confirmation danger avec saisie
- **Objectif** : Confirmer la suppression définitive du service

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x540px)
- Barre rouge en haut (6px)
- Titre avec close
- Message d'avertissement
- Warning box
- Champ de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsInput** : Champ de confirmation TERMINATE
- **OdsButton** : Annuler (secondary), Supprimer (danger)

## 4. HEADER
Barre rouge (6px) + "Supprimer votre Managed Rancher Service" + bouton ×

## 5. MESSAGE PRINCIPAL
"Vous ne pourrez pas annuler cette action. Nous détruirons définitivement votre déploiement de Rancher **rancher-prod-01** et l'ensemble de sa configuration."

## 6. WARNING
Fond jaune :
"⚠️ Attention

Nous vous conseillons fortement de détruire et/ou désenregistrer tous les clusters « downstream » avant de détruire votre Managed Rancher Service."

## 7. CONFIRMATION
- **Label** : "Entrez « TERMINATE » pour confirmer votre choix."
- **Input** : Champ texte
- **Valeur attendue** : TERMINATE

## 8. BOUTONS
- Annuler (secondary)
- Supprimer (danger/rouge)

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/rancher/{rancherId}
  - Supprime le service Rancher
