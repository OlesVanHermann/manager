# Cahier des Charges : public-cloud/kubernetes - Détail Cluster (Aperçu)

## 1. VUE D'ENSEMBLE
- **Page** : Détail cluster Kubernetes - Onglet Aperçu
- **Type** : Dashboard avec tiles d'informations
- **Objectif** : Vue d'ensemble du cluster

## 2. STRUCTURE PAGE
- Header avec nom cluster + badges (OK, Free)
- Onglets : Aperçu (actif), Node Pools, Accès APIServer, Logs
- Grille de tiles 2 colonnes

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsBadge** : OK (vert pill), Free (bleu)
- **OdsActionMenu** : Menu ⋮
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tiles d'informations
- **OdsProgressBar** : Utilisation ETCD
- **OdsButton** : Télécharger, Ajouter pool, etc.
