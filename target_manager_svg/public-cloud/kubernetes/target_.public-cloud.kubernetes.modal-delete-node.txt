# Cahier des Charges : public-cloud/kubernetes - Modal Supprimer Node

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le nœud
- **Type** : Modal danger avec confirmation DELETE
- **Objectif** : Supprimer un nœud individuel

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x380px
- Header jaune (warning)
- Titre warning + bouton fermer
- Question de confirmation avec nom nœud
- Banner danger rouge
- Champ confirmation "DELETE"
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Danger (rouge clair)
- **OdsInput** : Confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- Titre orange : "Supprimer le nœud"

## 5. QUESTION
"Êtes-vous sûr de supprimer **node-abc123def456** ?"

## 6. BANNER DANGER
- Fond rouge clair (#FEF2F2)
- Bordure rouge (#FCA5A5)
- Icône ⚠ rouge
- "Cette action est irréversible."
- "Le nœud et ses conteneurs seront supprimés définitivement."

## 7. CHAMP CONFIRMATION
- Label : "Entrez 'DELETE' dans le champ ci-dessous pour confirmer"
- Input : (curseur clignotant)

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Primary bleu | Droite | DELETE node |

## 9. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/kube/{kubeId}/nodepool/{poolId}/nodes/{nodeId}
  - Supprimer le nœud
