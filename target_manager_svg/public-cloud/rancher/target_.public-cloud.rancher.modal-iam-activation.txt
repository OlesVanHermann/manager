# Cahier des Charges : public-cloud/rancher - Modal Activation IAM

## 1. VUE D'ENSEMBLE
- **Modal** : Activer l'authentification via OVHcloud IAM
- **Type** : Modal confirmation avec informations
- **Objectif** : Activer l'authentification SSO via OVHcloud IAM

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (550x500px)
- Titre avec close
- Description
- Lien documentation
- Liste de points
- Warning box
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsLink** : Lien vers documentation
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler (ghost), Activer (primary)

## 4. HEADER
"Activer l'authentification via OVHcloud IAM" + bouton ×

## 5. DESCRIPTION
"La fonctionnalité d'authentification OVHcloud IAM permet à Rancher d'utiliser les identités OVHcloud IAM pour gérer l'accès. Ainsi, les utilisateurs se connectent à Rancher via leur compte SSO OVHcloud IAM."

- **Lien** : "Documentation OVHcloud IAM ↗"

## 6. POINTS D'INFORMATION
"Lors de l'activation de l'**authentification OVHcloud IAM** dans Rancher :"

- Les utilisateurs locaux resteront toujours fonctionnels.
- Vous pourrez donc continuer à vous connecter avec vos comptes Rancher habituels.

## 7. WARNING
Fond jaune avec bordure :
"**Limitation connue :**

En cas de régénération du mot de passe « admin » alors que l'authentification IAM est activée, cela aura pour effet de désactiver temporairement la possibilité de se connecter avec l'utilisateur root IAM.

Pour corriger, connectez-vous avec l'utilisateur local « admin »."

## 8. BOUTONS
- Annuler (ghost/outline)
- Activer (primary)

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/rancher/{rancherId}/iam
  - Active l'authentification IAM
