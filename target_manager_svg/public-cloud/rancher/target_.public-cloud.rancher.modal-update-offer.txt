# Cahier des Charges : public-cloud/rancher - Modal Modifier Offre

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier l'offre
- **Type** : Modal sélection avec cartes
- **Objectif** : Changer l'offre du service Rancher

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x460px)
- Titre avec close
- Description
- Cartes offres
- Info text
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsTile** : Cartes offres avec radio
- **OdsRadio** : Sélection offre
- **OdsButton** : Annuler (secondary), Confirmer (primary)

## 4. HEADER
"Modifier l'offre" + bouton ×

## 5. DESCRIPTION
"Sélectionnez l'offre de votre service Rancher."

## 6. OFFRES

### Standard (non sélectionné)
- **Titre** : "Standard"
- **Description** : "Offre d'entrée de gamme"
- **Feature** : "Jusqu'à 15 clusters downstream"
- **Prix** : Gratuit

### OVHcloud Edition (sélectionné)
- **Titre** : "OVHcloud Edition"
- **Description** : "Pour les environnements de production"
- **Feature** : "Clusters illimités, SLA 99.9%"
- **Prix** : 0,05 €/vCPU/heure

## 7. INFO TEXT
"Le changement d'offre sera effectif immédiatement."

## 8. BOUTONS
- Annuler (secondary)
- Confirmer (primary)

## 9. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/rancher/{rancherId}/plan
  - Body: { plan: "standard" | "ovhcloud-edition" }
  - Change l'offre du service
