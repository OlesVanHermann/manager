# Cahier des Charges : public-cloud/ai-tools - Modal Ajouter Datastore

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer un Datastore S3
- **Type** : Modal formulaire
- **Objectif** : Ajouter un datastore S3 pour les produits AI

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal centrée (600x650px)
- Header avec titre
- Formulaire 6 champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Alias, Endpoint, Access Key, Secret Key, Région S3
- **OdsSelect** : Région
- **OdsButton** : Annuler, Ajouter

## 4. HEADER
- **Titre** : "Configurer un Datastore S3"
- **Bouton fermer** : ✕

## 5. CHAMPS FORMULAIRE

| Champ | Type | Placeholder | Position |
|-------|------|-------------|----------|
| Alias | Input | alias | Full width |
| Endpoint | Input | https://s3.bhs.io.cloud.ovh.net/ | Full width |
| Région | Select | GRA (Gravelines) ▼ | Full width |
| Access Key | Input (focus) | AKIAIOSFODNN7EXAMPLE | Full width |
| Secret Key | Input password | ••••••••••••••••••• | Full width |
| Région compatible S3 | Input | gra | Full width |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer modal |
| Ajouter | Primary bleu | POST datastore |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/data/region/{region}/alias
  - Body: { alias, endpoint, accessKey, secretKey, s3Region }
