# Cahier des Charges : public-cloud/ai-apps - Modal Arrêter

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation arrêt application
- **Type** : Modal confirmation warning
- **Objectif** : Confirmer l'arrêt d'une app en cours

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal centrée (500x220px)
- Header avec titre
- Corps avec message
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, message
- **OdsButton** : Annuler (outline), Arrêter (warning orange)

## 4. HEADER
- **Titre** : "Arrêter une application"
- **Bouton fermer** : ✕

## 5. CONTENU
"Êtes-vous sûr de vouloir arrêter l'application **my-llm-app** ?"

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer modal |
| Arrêter | Warning orange | PUT stop |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/ai/app/{appId}/stop
  - Arrêter l'application
