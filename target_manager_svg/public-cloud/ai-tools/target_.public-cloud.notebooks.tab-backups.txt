# Cahier des Charges : public-cloud/notebooks - Onglet Sauvegardes

## 1. VUE D'ENSEMBLE
- **Page** : Sauvegardes d'un notebook
- **Type** : Onglet avec datagrid
- **Objectif** : Lister et gérer les sauvegardes

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Notebooks actif)
- Breadcrumb
- Onglets navigation
- Titre + description
- Datagrid sauvegardes

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 5 onglets
- **OdsDatagrid** : Liste sauvegardes
- **OdsActionMenu** : Menu contextuel (Fork)
