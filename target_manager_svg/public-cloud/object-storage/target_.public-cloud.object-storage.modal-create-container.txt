# Cahier des Charges : public-cloud/object-storage - Modal Créer Conteneur

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un conteneur
- **Type** : Modal formulaire complet
- **Objectif** : Créer un nouveau conteneur Object Storage

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (800x640px)
- Titre + bouton fermer
- Formulaire multi-sections
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsTile** : Cartes radio type conteneur
- **OdsRadio** : Sélections
- **OdsSelect** : Région, Classe stockage
- **OdsInput** : Nom conteneur
- **OdsCheckbox** : Options (versioning, chiffrement)
- **OdsButton** : Annuler (secondary), Créer (primary)

## 4. HEADER
- **Titre** : "Créer un conteneur"
- **Bouton fermer** : ×

## 5. FORMULAIRE

### Type de conteneur
| Type | Sélectionné | Description |
|------|-------------|-------------|
| S3 - Standard | ✅ | Compatible API S3, recommandé |
| Swift | - | Compatible OpenStack Swift |

### Région
- **Select** : GRA - Gravelines (dropdown)

### Classe de stockage
- **Select** : Standard (dropdown)

### Nom du conteneur
- **Input** : "prod-assets"
- **Helper** : "Le nom doit être unique, entre 3 et 63 caractères"

### Options
| Option | Actif | Description |
|--------|-------|-------------|
| Activer le versioning | ☐ | Gestion des versions |
| Chiffrement des objets | ☑ | Chiffrement SSE |

## 6. BOUTONS
- Annuler (secondary)
- Créer (primary)

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/storage
  - Body: { name, region, type, storageClass, versioning, encryption }
  - Crée le conteneur
- GET /cloud/project/{projectId}/region
  - Liste des régions disponibles
