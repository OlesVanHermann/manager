# Cahier des Charges : public-cloud/object-storage - Modal Activer Versioning

## 1. VUE D'ENSEMBLE
- **Modal** : Activer le versioning
- **Type** : Modal confirmation simple
- **Objectif** : Activer la gestion des versions sur un conteneur

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (500x250px)
- Barre bleue à gauche
- Icône info
- Titre
- Description explicative
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : ℹ dans cercle bleu
- **OdsText** : Description
- **OdsButton** : Annuler (secondary), Confirmer (primary)

## 4. HEADER
- **Icône** : ℹ dans cercle bleu
- **Titre** : "Activer le versioning"

## 5. DESCRIPTION
"Automatise la création de versions d'un objet pour pallier la suppression accidentelle de ce dernier, ou simplement quand un nouvel objet est uploadé."

## 6. BOUTONS
- Annuler (secondary)
- Confirmer (primary)

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/storage/{containerId}/versioning
  - Body: { enabled: true }
  - Active le versioning sur le conteneur
