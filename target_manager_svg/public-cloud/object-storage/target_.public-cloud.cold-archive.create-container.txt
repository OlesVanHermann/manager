# Cahier des Charges : public-cloud/cold-archive - Créer un conteneur

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Créer un conteneur
- **Type** : Page wizard/stepper
- **Objectif** : Créer un nouveau conteneur Cold Archive

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Cold Archive actif
- Breadcrumb avec "Créer un conteneur"
- Titre "Créer un conteneur Cold Archive"
- Stepper horizontal 3 étapes
- Sections accordéon pour chaque étape
- Boutons action

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : My Project / Cold Archive / Créer un conteneur
- **OdsStepper** : 3 étapes horizontales
- **OdsCard** : Sections d'étape (accordéon)
- **OdsInput** : Nom du conteneur
- **OdsSelect** : Sélection utilisateur (étape 1)
- **OdsText** : Région (lecture seule)
- **OdsButton** : "Créer →" (primaire)
- **OdsButton** : "Annuler" (outline)

## 4. TITRE
"Créer un conteneur Cold Archive"

## 5. STEPPER - 3 ÉTAPES

### Étape 1 : Lier un utilisateur (✓ complété)
- État: Done (cercle vert avec ✓)
- Section collapsed (fond vert clair #F0FDF4)
- Texte: "Utilisateur sélectionné : user-cold-archive-prod"
- Lien: "Modifier"

### Étape 2 : Nommer le conteneur (actif)
- État: Active (cercle bleu avec "2")
- Section étendue (fond blanc)
- Contenu:
  - Label: "Nom du conteneur *"
  - Input focus: "backup-2024-q4"
  - Helper: "Le nom doit contenir entre 3 et 63 caractères alphanumériques ou tirets."
  - Région (lecture seule): "GRA (Gravelines) - rbx-archive"

### Étape 3 : Confirmation (en attente)
- État: Inactive (cercle gris avec "3")
- Section collapsed (fond gris)

## 6. VALIDATION NOM
- Min: 3 caractères
- Max: 63 caractères
- Caractères autorisés: alphanumériques, tirets

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Créer → | Primaire bleu | Créer le conteneur |
| Annuler | Outline bleu | Retour à la liste |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/user
  - Liste des utilisateurs S3 pour étape 1
- GET /cloud/project/{projectId}/region
  - Régions disponibles pour Cold Archive
- POST /cloud/project/{projectId}/region/{regionId}/coldArchive/container
  - Body: { name: string, userId: string }
  - Création du conteneur
