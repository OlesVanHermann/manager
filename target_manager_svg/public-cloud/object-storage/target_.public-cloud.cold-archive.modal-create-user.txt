# Cahier des Charges : public-cloud/cold-archive - Modal Créer Utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Cold Archive - Modal créer utilisateur Object Storage
- **Type** : Modal formulaire avec radio
- **Objectif** : Créer ou sélectionner un utilisateur S3

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 700x480px centrée
- Header avec titre et bouton fermer
- Description
- Options radio (existant ou nouveau)
- Input description utilisateur (si nouveau)
- Note sur credentials
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal (700x480px)
- **OdsText** : Description
- **OdsRadio** : 2 options (existant/nouveau)
- **OdsInput** : Description utilisateur
- **OdsText** : Note informative
- **OdsButton** : Annuler (outline)
- **OdsButton** : Créer (primaire)

## 4. TITRE
"Ajouter un utilisateur Object Storage"

## 5. DESCRIPTION
"Vous avez la possibilité de définir l'accès à vos archives à vos utilisateurs prédéfinis ou créer de nouveaux utilisateurs et obtenir vos clés d'accès."

## 6. OPTIONS RADIO
| Option | État | Description |
|--------|------|-------------|
| Sélectionner un utilisateur existant | Non sélectionné | Affiche un dropdown |
| Créer un nouvel utilisateur | Sélectionné | Affiche le formulaire |

## 7. FORMULAIRE CRÉATION (si option 2)
### Champ Description
- Label: "Description de l'utilisateur"
- Type: OdsInput text
- Valeur: "backup-archive-user"

### Note
"Les credentials S3 seront générés automatiquement après la création." (12px, gris)

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Centre-droite |
| Créer | Primaire bleu | Droite |

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/user
  - Body: { description: string }
  - Créer un nouvel utilisateur
- POST /cloud/project/{projectId}/user/{userId}/s3Credentials
  - Générer les credentials S3 automatiquement
