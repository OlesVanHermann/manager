# Cahier des Charges : public-cloud/object-storage - Dashboard S3

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard conteneur S3
- **Type** : Page détail avec métriques et configuration
- **Objectif** : Afficher aperçu et configuration d'un bucket S3

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Object Storage actif)
- Breadcrumb
- Header conteneur (nom, badge S3, région)
- Onglets (Dashboard actif, Objets, Réplication)
- Carte aperçu du bucket
- Cartes configuration (3 colonnes)
- Carte tags
- Cartes autres paramètres (3 colonnes)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation

- **OdsText** : Valeurs métriques (28px gras)
- **OdsChip** : Tags avec fond bleu clair
- **OdsBadge** : Statuts Activé (vert), Désactivé (jaune)
- **OdsButton** : Copier, Activer, Ajouter
- **OdsLink** : "En savoir plus"

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/storage/{containerId}
  - Détails du conteneur
- GET /cloud/project/{projectId}/storage/{containerId}/metrics
  - Métriques (taille, nombre objets)
- PUT /cloud/project/{projectId}/storage/{containerId}/versioning
  - Configurer le versioning
- PUT /cloud/project/{projectId}/storage/{containerId}/encryption
  - Configurer le chiffrement
- GET /cloud/project/{projectId}/storage/{containerId}/tags
  - Liste des tags
- PUT /cloud/project/{projectId}/storage/{containerId}/tags
  - Modifier les tags
