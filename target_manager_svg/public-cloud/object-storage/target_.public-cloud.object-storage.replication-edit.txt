# Cahier des Charges : public-cloud/object-storage - Modifier Règle Réplication

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier une règle de réplication
- **Type** : Modal formulaire édition
- **Objectif** : Modifier une règle de réplication existante

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x640px)
- Barre orange en haut (modification)
- Titre + bouton fermer
- Formulaire avec valeurs existantes
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Identifiant, priorité
- **OdsSelect** : Conteneur destination, classe stockage
- **OdsToggle** : Statut, Delete markers
- **OdsButton** : Annuler (outline), Modifier (primary)

## 4. HEADER
- **Barre orange** : Indicateur édition
- **Titre** : "Modifier une règle de réplication"
- **Bouton fermer** : ×

## 5. FORMULAIRE

### Identifiant de la règle
- **Input** : "replication-rule-001"
- **Helper** : "Un identifiant unique pour cette règle de réplication (3-255 caractères)"

### Conteneur de destination
- **Select** : "backup-container-gra"
- **Helper** : "Le conteneur de destination où les objets seront répliqués"

### Priorité
- **Input** : 100 (120px)
- **Helper** : "La priorité de la règle (1-999)"

### Section : Options de réplication supplémentaires

#### Classe de stockage en destination
- **Select** : "Standard"

#### Statut
- **Toggle** : ● Activé (bleu)

#### Réplication des marqueurs de suppression
- **Toggle** : ○ Désactivé (gris)

## 6. BOUTONS
- Annuler (outline primary)
- Modifier (primary)

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/storage/{containerId}/replication/{ruleId}
  - Body: { destination, priority, storageClass, status, deleteMarkerReplication }
  - Modifie la règle de réplication
- GET /cloud/project/{projectId}/storage/{containerId}/replication/{ruleId}
  - Récupère les détails de la règle
