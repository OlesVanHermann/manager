# Cahier des Charges : public-cloud/quantum - Créer Notebook

## 1. VUE D'ENSEMBLE
- **Page** : Créer un notebook quantique
- **Type** : Formulaire wizard avec récapitulatif
- **Objectif** : Créer un notebook pour accéder aux QPUs

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Quantum actif)
- Breadcrumb
- Titre "Créer un notebook"
- Formulaire (colonne gauche)
- Récapitulatif commande (colonne droite)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsChip** : Sélection région
- **OdsTile** : Cartes QPU, Framework, Ressources
- **OdsRadio** : Sélection QPU
- **OdsInput** : Nom du notebook
- **OdsCard** : Récapitulatif commande
- **OdsButton** : Commander (primary)

## 6. BOUTONS
- Commander (primary)

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/qpu
  - Liste des QPUs disponibles
- GET /cloud/project/{projectId}/ai/framework
  - Liste des frameworks
- GET /cloud/project/{projectId}/ai/flavor
  - Liste des ressources
- POST /cloud/project/{projectId}/ai/notebook
  - Body: { name, region, qpu, framework, flavor }
  - Crée le notebook quantique
