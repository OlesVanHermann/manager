# Cahier des Charges : public-cloud/load-balancer - Modal Créer L7 Rule

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter une L7 rule
- **Type** : Modal formulaire
- **Objectif** : Créer une règle L7 pour une policy

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x440px
- Titre
- Formulaire avec champs
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, labels
- **OdsInput** : Clé, Valeur
- **OdsSelect** : Type, Type de comparaison
- **OdsCheckbox** : Inverser
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter une L7 rule"

## 5. CHAMPS

| Champ | Type | Placeholder |
|-------|------|-------------|
| Type | Select | - |
| Type de comparaison | Select | - |
| Clé | Input | - |
| Valeur | Input | - |
| Inverser | Checkbox | - |

## 6. TYPES DE RÈGLE
- COOKIE
- FILE_TYPE
- HEADER
- HOST_NAME
- PATH
- SSL_CONN_HAS_CERT
- SSL_DN_FIELD
- SSL_VERIFY_RESULT

## 7. TYPES DE COMPARAISON
- CONTAINS
- ENDS_WITH
- EQUAL_TO
- REGEX
- STARTS_WITH

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Ajouter | Primary bleu | Droite | POST L7 rule |

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy/{l7PolicyId}/l7Rule
  - Body: { type, compareType, key?, value, invert }
  - Créer la L7 rule
