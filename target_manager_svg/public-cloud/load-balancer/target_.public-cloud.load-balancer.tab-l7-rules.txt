# Cahier des Charges : public-cloud/load-balancer - Onglet L7 Rules

## 1. VUE D'ENSEMBLE
- **Page** : Liste des L7 Rules d'une policy
- **Type** : Page datagrid avec actions
- **Objectif** : Gérer les règles L7 d'une policy

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Titre "L7 Rules"
- Bouton ajouter (outline)
- Datagrid

## 3. COMPOSANTS UI
- **OdsText** : Titre
- **OdsButton** : Ajouter une L7 rule (outline bleu)
- **OdsDatagrid** : Liste des rules
- **OdsBadge** : Statut (ACTIVE)

## 4. TITRE
"L7 Rules"

## 5. BOUTON AJOUTER
- **Label** : "+ Ajouter une L7 rule"
- **Style** : Outline bleu

## 6. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Type | Text | PATH, HEADER, HOST_NAME... |
| Comparaison | Text | STARTS_WITH, CONTAINS... |
| Clé | Text | Nom du header ou "-" |
| Valeur | Text | Pattern de match |
| Statut | Badge success | ACTIVE |

## 7. DONNÉES EXEMPLE

| Type | Comparaison | Clé | Valeur | Statut |
|------|-------------|-----|--------|--------|
| PATH | STARTS_WITH | - | /api/v1 | ACTIVE |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy/{l7PolicyId}/l7Rule
  - Liste des L7 rules
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/l7Policy/{l7PolicyId}/l7Rule
  - Créer une L7 rule
