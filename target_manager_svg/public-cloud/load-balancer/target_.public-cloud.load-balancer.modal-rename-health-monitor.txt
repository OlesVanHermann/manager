# Cahier des Charges : public-cloud/load-balancer - Modal Renommer Health Monitor

## 1. VUE D'ENSEMBLE
- **Modal** : Renommer le health monitor
- **Type** : Modal formulaire simple
- **Objectif** : Modifier le nom d'un health monitor

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x220px
- Titre
- Champ nom
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsInput** : Nom
- **OdsButton** : Annuler, Renommer

## 4. TITRE
"Renommer le health monitor"

## 5. CHAMP
| Champ | Type | Valeur |
|-------|------|--------|
| Nom | Input | health-monitor-http |

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline bleu | Gauche | Fermer modal |
| Renommer | Primary bleu | Droite | PUT health monitor |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor/{healthMonitorId}
  - Body: { name }
  - Renommer le health monitor
