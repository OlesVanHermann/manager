# Cahier des Charges : public-cloud/gateway - Modal Supprimer Gateway

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer la Gateway
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'une Gateway

## 2. STRUCTURE MODAL
- Overlay sombre avec fond page visible
- Conteneur blanc 500x220px centré
- Icône warning rouge
- Titre avec nom action
- Message de confirmation avec nom gateway
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Warning rouge (cercle rouge avec ⚠)
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer (danger)

## 4. TITRE
"Supprimer la Gateway"

## 5. MESSAGE
"Souhaitez-vous réellement supprimer la Gateway **gateway-gra-001** ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Confirmer | Danger rouge | Droite | DELETE gateway |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/gateway/{gatewayId}
  - Supprimer la Gateway
