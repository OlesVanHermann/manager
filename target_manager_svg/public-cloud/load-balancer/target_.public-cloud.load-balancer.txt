# Cahier des Charges : public-cloud/load-balancer - Liste des Load Balancers

## 1. VUE D'ENSEMBLE
- **Page** : Liste des Load Balancers
- **Type** : Datagrid avec ActionMenu
- **Objectif** : Lister et gérer les Load Balancers du projet

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar avec Load Balancer actif
- Breadcrumb
- Titre + bouton Ajouter
- Barre de filtres/recherche
- Datagrid avec données

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre
- **OdsButton** : Ajouter un Load Balancer
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste des LB
- **OdsBadge** : Taille (S/M/L), Statut (online/dégradé)
- **OdsActionMenu** : Menu contextuel par ligne

## 4. BREADCRUMB
"Public Cloud / Mon Projet Cloud / Load Balancer"

## 5. HEADER PAGE
- **Titre** : "Load Balancers"
- **Bouton** : "+ Ajouter un Load Balancer" (primary)

## 6. FILTRES
- Champ recherche (placeholder: "Rechercher...")

## 7. COLONNES DATAGRID

| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Nom | Link bleu | Oui | Lien vers détail |
| Région | Text | Oui | Code région (GRA9, SBG5...) |
| Taille | Badge info | Oui | S, M, L, XL |
| IP Privée | Monospace | Non | Adresse IP interne |
| Floating IP | Monospace | Non | IP publique ou "-" |
| Date de création | Date | Oui | Format DD/MM/YYYY |
| Statut | Badge | Oui | online (vert), dégradé (orange) |
| Actions | ActionMenu | Non | ⋮ |

| Action | Style | Navigation |
|--------|-------|------------|
| Détail du Load Balancer | Normal | Navigate to detail |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/loadbalancer
  - Liste des Load Balancers
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/loadbalancer/{loadBalancerId}
  - Supprimer un LB
