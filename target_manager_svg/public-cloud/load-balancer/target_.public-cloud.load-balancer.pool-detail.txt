# Cahier des Charges : public-cloud/load-balancer - Détail Pool

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'un pool
- **Type** : Page détail avec tabs et tiles
- **Objectif** : Afficher et gérer un pool

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Lien retour vers liste des pools
- Titre (nom du pool)
- Barre d'onglets
- Tiles d'information

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre, labels, valeurs
- **OdsTabs** : Informations générales, Health monitor, Membres
- **OdsCard** : Tiles
- **OdsBadge** : Statut (ACTIVE)

## 4. LIEN RETOUR
"‹ liste des pools"

## 5. TITRE
"pool-backend-api"

## 6. ONGLETS
| Onglet | Actif |
|--------|-------|
| Informations générales | ✓ (souligné bleu) |
| Health monitor | - |
| Membres | - |

## 7. TILE INFORMATIONS
| Propriété | Valeur |
|-----------|--------|
| Nom | pool-backend-api |
| Statut | Badge "ACTIVE" (vert) |

## 8. TILE CONFIGURATION
| Propriété | Valeur |
|-----------|--------|
| Protocole | HTTP |
| Algorithme | ROUND_ROBIN |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}
  - Détails du pool
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member
  - Liste des membres
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor?poolId={poolId}
  - Health monitor associé
