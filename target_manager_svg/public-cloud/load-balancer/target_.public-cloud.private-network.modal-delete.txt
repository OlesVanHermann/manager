# Cahier des Charges : public-cloud/private-network - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le réseau privé
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'un réseau privé

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (500x220px)
- Icône danger (cercle rouge avec ⚠)
- Titre avec close button
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsButton** : Annuler (outline), Confirmer (danger)

## 4. HEADER
Icône danger rouge + "Supprimer le réseau privé" + bouton ×

## 5. MESSAGE
"Souhaitez-vous réellement supprimer le réseau privé **my-private-network** ?"

## 6. BOUTONS
- Annuler (outline)
- Confirmer (danger/rouge)

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/network/private/{networkId}
  - Supprime le réseau privé
