# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer Health Monitor

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le health monitor
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer un health monitor d'un pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 450x180px
- Header avec icône warning + titre
- Question de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsIcon** : Warning (⚠) orange dans cercle jaune
- **OdsText** : Question
- **OdsButton** : Annuler (outline gris), Supprimer (danger rouge)

## 4. HEADER
- Icône ⚠ orange dans cercle jaune (#FEF3C7)
- Titre : "Supprimer le health monitor"

## 5. QUESTION
"Supprimer "health-monitor-http" du pool ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge (#DC2626) | Droite | DELETE health monitor |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor/{healthMonitorId}
  - Supprimer le health monitor
