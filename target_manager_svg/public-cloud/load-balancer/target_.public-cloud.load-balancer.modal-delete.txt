# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer Load Balancer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un Load Balancer
- **Type** : Modal danger avec confirmation
- **Objectif** : Supprimer un Load Balancer et toutes ses ressources

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x220px
- Header avec icône danger + titre + bouton fermer
- Question avec nom du LB
- Message d'avertissement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Danger (⚠) rouge dans cercle rose
- **OdsText** : Question, avertissement
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. HEADER
- Icône ⚠ rouge dans cercle rose (#FEE2E2)
- Titre : "Supprimer un Load Balancer"
- Bouton fermer (×)

## 5. QUESTION
- Ligne 1 : "Souhaitez-vous vraiment supprimer le load balancer"
- Ligne 2 (bold) : "my-load-balancer" ?

## 6. MESSAGE AVERTISSEMENT
"Cette action est irréversible. Tous les listeners et pools associés seront supprimés."

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge (#DC2626) | Droite | DELETE LB |

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/loadbalancer/{loadBalancerId}
  - Supprimer le Load Balancer
