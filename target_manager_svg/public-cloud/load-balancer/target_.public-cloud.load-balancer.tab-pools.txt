# Cahier des Charges : public-cloud/load-balancer - Onglet Pools

## 1. VUE D'ENSEMBLE
- **Page** : Liste des pools d'un Load Balancer
- **Type** : Datagrid avec ActionMenu
- **Objectif** : Gérer les pools backend du LB

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Breadcrumb
- Titre LB
- Barre d'onglets (Pools actif)
- Titre section + description
- Bouton ajouter
- Datagrid

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titres, description
- **OdsTabs** : Navigation onglets
- **OdsButton** : Ajouter un pool
- **OdsDatagrid** : Liste des pools
- **OdsBadge** : Statut (online)
- **OdsActionMenu** : Menu contextuel

## 4. BREADCRUMB
"Mon Projet / Load Balancer / my-load-balancer"

## 5. ONGLETS
| Onglet | Actif |
|--------|-------|
| Aperçu | - |
| Listeners | - |
| Pools | ✓ (souligné bleu) |
| Statistiques | - |
| Certificats | - |
| Logs | - |

## 6. SECTION POOLS
- **Titre** : "Pools"
- **Description** : "Un pool représente un groupe de membres en charge du backend partageant une configuration commune, telle que l'algorithme du Load Balancer, la sonde ou la configuration de sessions permanentes."

## 7. BOUTON AJOUTER
- **Label** : "+ Ajouter un pool"
- **Style** : Primary bleu

## 8. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link bleu | Lien vers détail pool |
| Protocole | Text | HTTP, HTTPS, TCP... |
| Algorithme | Text | Round Robin, Least Connections... |
| Membres | Link bleu | Lien vers liste membres |
| Statut | Badge success | online |
| Health Monitor | Text | Type de HM ou "-" |
| Actions | ActionMenu | ⋮ |

## 9. DONNÉES EXEMPLE

| Nom | Protocole | Algorithme | Membres | Statut | HM |
|-----|-----------|------------|---------|--------|-----|
| pool-http-backend | HTTP | Round Robin | 3 membres | online | HTTP |
| pool-https-backend | HTTPS | Least Connections | 2 membres | online | HTTPS |

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool?loadbalancerId={loadBalancerId}
  - Liste des pools
- POST /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool
  - Créer un pool
