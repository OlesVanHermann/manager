# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer Pool

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un Pool
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer un pool de backends

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x180px
- Header avec icône warning + titre
- Question de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsIcon** : Warning (⚠) orange
- **OdsText** : Question
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
- Icône ⚠ orange dans cercle jaune
- Titre : "Supprimer un Pool"

## 5. QUESTION
"Supprimer "pool-backend-api" ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE pool |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}
  - Supprimer le pool
