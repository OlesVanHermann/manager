# Cahier des Charges : public-cloud/load-balancer - Modal Modifier Pool

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le pool
- **Type** : Modal formulaire avec sections
- **Objectif** : Modifier les paramètres d'un pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 700x550px avec barre bleue gauche
- Icône + titre + bouton fermer
- Section nom
- Section algorithme (radio cards)
- Section persistance de session
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec accent bleu
- **OdsIcon** : Engrenage (⚙)
- **OdsInput** : Nom, Nom du cookie
- **OdsRadio** : Algorithmes (cards)
- **OdsCheckbox** : Activer persistance
- **OdsSelect** : Type de persistance
- **OdsButton** : Annuler, Modifier

## 4. HEADER
- Icône engrenage dans cercle bleu clair
- Titre : "Modifier le pool"
- Bouton fermer (×)

## 5. NOM DU POOL
| Champ | Type | Valeur |
|-------|------|--------|
| Nom du pool | Input (required) | backend-web-servers |

## 6. ALGORITHME DE RÉPARTITION (Radio cards)
| Algorithme | Sélection |
|------------|-----------|
| Round Robin | Sélectionné (bordure bleue) |
| Least Connections | Non |
| Source IP | Non |

## 7. PERSISTANCE DE SESSION
- **Checkbox** : ✓ Activer la persistance de session

| Champ | Type | Valeur | Position |
|-------|------|--------|----------|
| Type de persistance | Select | Cookie applicatif | Gauche (300px) |
| Nom du cookie | Input | SERVERID | Droite (330px) |

## 8. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Modifier | Primary bleu | Droite | PUT pool |

## 9. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}
  - Body: { name?, algorithm?, sessionPersistence? }
  - Modifier le pool
