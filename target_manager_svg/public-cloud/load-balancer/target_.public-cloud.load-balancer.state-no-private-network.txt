# Cahier des Charges : public-cloud/load-balancer - État Sans Réseau Privé

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration requise
- **Type** : Modal d'avertissement avec CTA
- **Objectif** : Informer qu'un réseau privé est nécessaire

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 500x340px
- Titre
- Banner warning
- Texte explicatif
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Warning (jaune)
- **OdsText** : Explication
- **OdsButton** : Annuler, Créer un réseau privé

## 4. TITRE
"Configuration requise"

## 5. BANNER WARNING
- Fond jaune (#FEF3C7)
- Icône ⚠
- "Un réseau privé est nécessaire pour utiliser le Load Balancer."

## 6. TEXTE EXPLICATIF
- "Le Load Balancer nécessite un réseau privé pour communiquer avec vos instances backend."
- "Créez un réseau privé avant de configurer votre Load Balancer."

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Créer un réseau privé | Primary bleu | Droite | Navigate to Private Network |

## 8. NAVIGATION
Redirection vers : /public-cloud/project/{projectId}/private-network/new
