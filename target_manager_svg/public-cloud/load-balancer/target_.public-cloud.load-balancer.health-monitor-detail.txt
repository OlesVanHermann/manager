# Cahier des Charges : public-cloud/load-balancer - Détail Health Monitor

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'un Health Monitor
- **Type** : Page détail avec tiles
- **Objectif** : Afficher et gérer un Health Monitor de pool

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar avec sous-menu Pools > Health Monitor actif
- Breadcrumb : Load Balancer > my-lb-prod > Pool pool-web > Health Monitor
- Grille de 3 tiles

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsCard** : Tiles d'information
- **OdsText** : Labels et valeurs
- **OdsLink** : Actions (Editer, Supprimer)
- **OdsBadge** : Statuts (OK, Actif)

## 4. BREADCRUMB
"Load Balancer > my-lb-prod > Pool pool-web > Health Monitor"

## 5. TILE GESTION
| Action | Type |
|--------|------|
| Editer | Lien bleu |
| Supprimer | Lien bleu |

## 6. TILE INFORMATIONS
- **Style** : Fond gris (#F9FAFB)

| Propriété | Valeur |
|-----------|--------|
| Nom | hm-web-check |
| Type | HTTP |
| URL Path | /health |
| Code attendu | 200 |

## 7. TILE STATUT
- **Style** : Fond gris (#F9FAFB)

| Propriété | Valeur |
|-----------|--------|
| Statut opérationnel | Badge "OK" (vert) |
| Statut provisionnement | Badge "Actif" (vert) |
| Interval | 30s |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor/{healthMonitorId}
  - Détails du health monitor
- PUT /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor/{healthMonitorId}
  - Modifier le health monitor
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/healthMonitor/{healthMonitorId}
  - Supprimer le health monitor
