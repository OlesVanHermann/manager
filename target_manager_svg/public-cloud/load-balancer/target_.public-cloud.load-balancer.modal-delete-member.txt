# Cahier des Charges : public-cloud/load-balancer - Modal Supprimer Membre

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un membre
- **Type** : Modal warning avec confirmation
- **Objectif** : Supprimer un membre d'un pool

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 400x160px
- Header avec icône warning + titre
- Question de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact
- **OdsIcon** : Warning (⚠) orange
- **OdsText** : Question
- **OdsButton** : Annuler, Supprimer

## 4. HEADER
- Icône ⚠ orange dans cercle jaune
- Titre : "Supprimer un membre"

## 5. QUESTION
"Supprimer "member-web-01" ?"

## 6. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Supprimer | Danger rouge | Droite | DELETE membre |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/region/{regionName}/loadbalancing/pool/{poolId}/member/{memberId}
  - Supprimer le membre
