# Cahier des Charges : public-cloud/instances - Onglet Snapshots

## 1. VUE D'ENSEMBLE
- **Page** : Onglet Snapshots du détail instance
- **Type** : Vue avec tableau des snapshots
- **Objectif** : Gérer les snapshots de l'instance

## 2. STRUCTURE PAGE
- Header avec breadcrumb et infos instance
- Onglets : General, Réseau, Snapshots (actif), Console
- Titre "Snapshots" + compteur badge + bouton création
- Tableau des snapshots

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsBadge** : Compteur de snapshots (2)
- **OdsButton** : "Créer un snapshot" (Primary bleu)
- **OdsTable** : Tableau des snapshots
- **OdsBadge** : Statut "active" (vert)

## 4. EN-TÊTE INSTANCE
- Breadcrumb : Public Cloud > Mon Projet Prod > {nom}
- Titre : {nom_instance}
- Info : {région} | {modèle} | ACTIVE (vert)

## 5. ONGLETS
- General
- Réseau
- **Snapshots** (actif, souligné bleu)
- Console

## 6. HEADER SNAPSHOTS
- Titre : "Snapshots" + Badge "2" (bleu clair)
- Bouton : "Créer un snapshot" (Primary bleu, à droite)

## 7. COLONNES DU TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Text | Nom du snapshot |
| Taille | Text | Taille en GB |
| Statut | Badge | active (vert) |
| Créé le | Date | Date de création |
| Actions | Button | "Créer instance" |

## 8. DONNÉES EXEMPLE
| Nom | Taille | Statut | Créé le | Actions |
|-----|--------|--------|---------|---------|
| snap-web-server-01-2024-12-27 | 20 GB | active | 27/12/2024 | Créer instance |
| snap-web-server-01-weekly | 18 GB | active | 20/12/2024 | Créer instance |

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance/{instanceId}/snapshot
  - Liste des snapshots de l'instance
- POST /cloud/project/{projectId}/instance/{instanceId}/snapshot
  - Créer un snapshot
- POST /cloud/project/{projectId}/instance
  - Body: { imageId: snapshotId }
  - Créer une instance depuis un snapshot
