# Cahier des Charges : public-cloud/instances - Détail Instance

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'une instance
- **Type** : Page détail avec tiles d'informations
- **Objectif** : Afficher toutes les informations d'une instance

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar avec menu (Instances actif)
- Breadcrumb : Public Cloud / Mon Projet Cloud / Instances / {nom}
- Titre avec nom instance + badge statut
- Boutons d'action (Éteindre, Créer un backup, menu ⋮)
- Onglets (Informations générales actif, Console VNC)
- Grille de tiles 2 colonnes

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsBadge** : Statut "Activée" (vert)
- **OdsButton** : Éteindre (outline), Créer un backup (primary)
- **OdsActionMenu** : Menu ⋮
- **OdsTabs** : Navigation onglets
- **OdsCard** : Tiles d'informations

## 4. TITRE
"{nom_instance}" + Badge Activée

## 5. ONGLETS
- Informations générales (actif)
- Console VNC
