# Cahier des Charges : public-cloud/instances - Liste des Instances

## 1. VUE D'ENSEMBLE
- **Page** : Liste des instances Public Cloud
- **Type** : Datagrid avec ActionMenu
- **Objectif** : Afficher et gérer les instances compute

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation
- Sidebar avec menu (Instances actif)
- Breadcrumb : Public Cloud / Mon Projet Cloud / Instances
- Titre "Instances" + bouton création
- Barre de recherche
- Tableau des instances avec colonnes
- ActionMenu ouvert sur la 1ère ligne

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation fil d'Ariane
- **OdsButton** : "+ Créer une instance"
- **OdsInput** : Recherche
- **OdsDatagrid** : Tableau des instances
- **OdsBadge** : Statut (Activée vert, Éteinte orange)
- **OdsActionMenu** : Actions contextuelles

## 4. TITRE
"Instances"

## 5. COLONNES DU TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom/ID | Link + small text | Nom cliquable + UUID tronqué |
| Région | Text | Code région (GRA11, SBG5) |
| Modèle | Text | Flavor (b2-15, c2-30, r2-60) |
| Image | Text | OS (Ubuntu 22.04, Debian 12) |
| IP publique(s) | Text | Adresse IPv4 |
| Statut | Badge | Activée (vert) / Éteinte (jaune) |
| Facturation | Text | Mensuel / Horaire |
| Actions | ActionMenu | Menu ⋮ |

## 6. DONNÉES EXEMPLE
| Nom | Région | Modèle | Image | IP | Statut | Facturation |
|-----|--------|--------|-------|-----|--------|-------------|

## 7. ACTION MENU
| Action | Type | Description |
|--------|------|-------------|
| Modifier | Standard | Éditer les propriétés |
| Éteindre | Standard | Arrêter l'instance |
| Redémarrer à chaud (soft) | Standard | Soft reboot |
| Redémarrer à froid (hard) | Standard | Hard reboot |
| --- | Séparateur | --- |
| Mode rescue | Standard | Démarrer en rescue |
| Réinstaller | Standard | Réinstaller l'OS |
| Créer un backup | Standard | Snapshot |
| --- | Séparateur | --- |
| Supprimer | Danger (rouge) | Supprimer l'instance |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/instance
  - Liste des instances
- DELETE /cloud/project/{projectId}/instance/{instanceId}
  - Supprimer une instance
- POST /cloud/project/{projectId}/instance/{instanceId}/start
  - Démarrer
- POST /cloud/project/{projectId}/instance/{instanceId}/stop
  - Éteindre
- POST /cloud/project/{projectId}/instance/{instanceId}/reboot
  - Redémarrer
