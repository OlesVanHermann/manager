# Cahier des Charges : public-cloud/instances - Modal Redémarrer (Hard)

## 1. VUE D'ENSEMBLE
- **Modal** : Redémarrer à froid (hard)
- **Type** : Modal confirmation avec warning
- **Objectif** : Forcer le redémarrage de l'instance

## 2. STRUCTURE MODAL
- Overlay sombre
- Conteneur blanc 600x280px
- Icône warning orange
- Titre + bouton fermer
- Message de confirmation
- Banner warning jaune
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Refresh (cercle orange avec ⟳)
- **OdsText** : Titre et messages
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler, Confirmer (orange)

## 4. TITRE
"Redémarrer à froid (hard)"

## 5. MESSAGE
"Vous allez redémarrer votre instance **my-instance-01**."

## 6. BANNER WARNING
- Fond jaune (#FEF3C7)
- Bordure orange (#F59E0B)
- Icône ⚠
- "Un redémarrage à froid (hard reboot) peut entraîner une perte de données non sauvegardées."

## 7. BOUTONS
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Outline gris | Gauche | Fermer modal |
| Confirmer | Warning orange | Droite | POST reboot hard |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/reboot
  - Body: { type: "hard" }
  - Forcer le redémarrage (risque perte données)
