# Cahier des Charges : public-cloud/instances - Onboarding

## 1. VUE D'ENSEMBLE
- **Page** : Onboarding Instances (empty state)
- **Type** : Page d'accueil vide avec CTA
- **Objectif** : Présenter les instances et inciter à la création

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation
- Sidebar avec menu (Instances actif)
- Breadcrumb
- Contenu centré : illustration, titre, descriptions
- Bouton CTA
- Section tutoriels (3 cartes)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsIcon** : Illustration cloud ☁️
- **OdsText** : Titres et descriptions
- **OdsButton** : Créer une instance (primary)
- **OdsCard** : Cartes tutoriels
