# Cahier des Charges : public-cloud/instances - Onglet Console

## 1. VUE D'ENSEMBLE
- **Page** : Onglet Console du détail instance
- **Type** : Vue avec options de console
- **Objectif** : Accéder aux consoles VNC et série

## 2. STRUCTURE PAGE
- Header avec breadcrumb et infos instance
- Onglets : General, Réseau, Snapshots, Console (actif)
- Titre "Console"
- Deux cartes verticales :
  - Console VNC
  - Console série

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes d'options
- **OdsButton** : Ouvrir la console VNC (primary), Ouvrir la console série (outline)

## 4. EN-TÊTE INSTANCE
- Breadcrumb : Public Cloud > Mon Projet Prod > {nom}
- Titre : {nom_instance}
- Info : {région} | {modèle} | ACTIVE (vert)

## 5. ONGLETS
- General
- Réseau
- Snapshots
- **Console** (actif, souligné bleu)

## 6. CARTE CONSOLE VNC
- Titre : "Console VNC"
- Description : "Accédez à la console graphique de votre instance via VNC."
- Bouton : **"Ouvrir la console VNC"** (Primary bleu)

## 7. CARTE CONSOLE SÉRIE
- Titre : "Console série"
- Description : "Accédez à la console série pour le debug."
- Bouton : "Ouvrir la console série" (Outline gris)

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/vnc
  - Obtenir l'URL de la console VNC
- GET /cloud/project/{projectId}/instance/{instanceId}/console
  - Obtenir les logs de console série
