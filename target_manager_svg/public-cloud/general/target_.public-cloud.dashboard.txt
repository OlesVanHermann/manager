# Cahier des Charges : public-cloud/dashboard - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Public Cloud - Dashboard principal
- **Type** : Page dashboard avec statistiques
- **Objectif** : Vue d'ensemble des ressources Public Cloud

## 2. STRUCTURE PAGE
- Header OVHcloud Manager
- Sidebar navigation univers
- Titre et description
- Cartes statistiques (4)
- Cartes catégories de services (3)

## 3. COMPOSANTS UI
- **OdsText** : Titre, description, labels
- **OdsCard** : Cartes statistiques et catégories
- **OdsIcon** : Icônes dans cercles colorés

## 4. SIDEBAR NAVIGATION
Menu univers (sidebar gauche) :
| Univers | État |
|---------|------|
| Public Cloud | ✅ Actif (fond bleu clair) |
| Bare Metal Cloud | - |
| Web Cloud | - |
| Network | - |

## 5. HEADER PAGE
- **Titre** : "Public Cloud"
- **Description** : "Infrastructure cloud OpenStack"

## 6. CARTES STATISTIQUES

### Layout
- 4 cartes en ligne (260px chacune)
- Icône circulaire colorée à gauche
- Nombre statistique + label à droite

### Statistiques
| Stat | Valeur | Couleur icône |
|------|--------|---------------|
| Projets | 3 | Bleu (#E8F0FE) |
| Instances | 12 | Violet (#EDE9FE) |
| Volumes | 8 | Vert (#D1FAE5) |
| Conteneurs | 5 | Jaune (#FEF3C7) |

## 7. CARTES CATÉGORIES

### Layout
- 3 cartes en ligne (350px chacune)
- Grande icône circulaire à gauche
- Titre + description à droite
- Cliquables vers la section correspondante

### Catégories
| Catégorie | Description | Couleur |
|-----------|-------------|---------|
| Compute | Instances, images, SSH keys | Bleu |
| Stockage | Volumes, snapshots, object storage | Jaune |
| Réseau | Réseaux privés, sous-réseaux | Vert |

## 8. APIS & ENDPOINTS
- GET /cloud/project
  - Liste des projets Public Cloud
- GET /cloud/project/{projectId}/instance
  - Compte les instances
- GET /cloud/project/{projectId}/volume
  - Compte les volumes
- GET /cloud/project/{projectId}/storage/container
  - Compte les conteneurs Object Storage
