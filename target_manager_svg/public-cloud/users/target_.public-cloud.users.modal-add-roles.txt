# Cahier des Charges : public-cloud/users - Modal Ajouter Rôles

## 1. VUE D'ENSEMBLE
- **Modal** : Rôles de l'utilisateur - Étape 2
- **Type** : Modal sélection multiple (wizard step 2)
- **Objectif** : Sélectionner les rôles à attribuer

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x540px)
- Header avec titre
- Grille de rôles (checkboxes)
- Compteur rôles sélectionnés
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsCheckbox** : Sélection rôles
- **OdsCard** : Encarts rôles
- **OdsButton** : Annuler, Créer

## 4. HEADER
"Rôles de l'utilisateur"

## 5. RÔLES DISPONIBLES (grille 3 colonnes)

| Rôle | État exemple |
|------|--------------|
| Administrator | ✓ sélectionné |
| Compute Operator | ✓ sélectionné |
| Network Operator | non sélectionné |
| ObjectStore Operator | ✓ sélectionné |
| Volume Operator | non sélectionné |
| Image Operator | non sélectionné |
| AI Training Operator | non sélectionné |
| Backup Operator | non sélectionné |

## 6. COMPTEUR
"3 rôles sélectionnés"

## 7. BOUTONS
- Annuler (outline)
- Créer (primary)

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/user
  - Body: { description, roles: [...] }
