# Cahier des Charges : public-cloud/users - Modal Télécharger rclone

## 1. VUE D'ENSEMBLE
- **Modal** : Télécharger un fichier de configuration rclone
- **Type** : Modal formulaire
- **Objectif** : Générer un fichier de config rclone

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x480px)
- Header avec titre
- Sélection type stockage (radio)
- Sélection région (select)
- Info text
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsRadio** : Type stockage
- **OdsSelect** : Région
- **OdsButton** : Annuler, Télécharger

## 4. HEADER
"Télécharger un fichier de configuration rclone"

## 5. MESSAGE
"Générez un fichier de configuration rclone pour l'utilisateur **user-admin-abc123**"

## 6. TYPE DE STOCKAGE (Radio)
- **Object Storage Swift** (sélectionné)
- **Object Storage S3**

## 7. RÉGION (Select)
- Dropdown : "GRA - Gravelines"

## 8. INFO TEXT
"Le fichier de configuration sera téléchargé automatiquement.
Vous pourrez l'utiliser avec rclone pour gérer votre Object Storage."

## 9. BOUTONS
- Annuler (outline)
- Télécharger (primary)

## 10. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/user/{userId}/rclone?region={region}&type={swift|s3}
  - Télécharge le fichier de configuration
