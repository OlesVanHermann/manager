# Cahier des Charges : public-cloud/users - Modal Ajouter

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un utilisateur - Étape 1
- **Type** : Modal formulaire (wizard step 1)
- **Objectif** : Saisir la description de l'utilisateur

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x280px)
- Header avec titre
- Champ description
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Description
- **OdsButton** : Annuler, Suivant

## 4. HEADER
"Ajouter un utilisateur"

## 5. CHAMPS
- **Label** : "Description de l'utilisateur"
- **Placeholder** : "Ex: Utilisateur API production"
- **Note** : "Cette description vous permettra d'identifier facilement l'utilisateur."

## 6. BOUTONS
- Annuler (outline)
- Suivant (primary) → va à l'étape 2 (sélection rôles)

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/user
  - Body: { description, roles[] }
