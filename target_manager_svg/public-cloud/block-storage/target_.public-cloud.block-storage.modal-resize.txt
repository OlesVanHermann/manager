# Cahier des Charges : public-cloud/block-storage - Modal Redimensionner

## 1. VUE D'ENSEMBLE
- **Modal** : Redimensionner le volume
- **Type** : Modal formulaire
- **Objectif** : Augmenter la taille d'un volume

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal blanche centrée (400x290px)
- Header gris avec titre + fermeture
- Info volume (read-only)
- Input nouvelle taille
- Warning message
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Volume (disabled), Nouvelle taille
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Redim.

## 4. CHAMPS
- Volume : my-volume (50 Go) - disabled
- Nouvelle taille (Go) : 100

## 5. WARNING
"Le volume ne peut pas être réduit après extension"

## 6. BOUTONS
- Annuler (outline)
- Redim. (primary)

## 7. VALIDATION
- Nouvelle taille > taille actuelle
- Nouvelle taille <= 4 To

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/volume/{volumeId}/upsize
  - Body: { size }
