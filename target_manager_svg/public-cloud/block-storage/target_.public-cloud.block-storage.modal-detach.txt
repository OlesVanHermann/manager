# Cahier des Charges : public-cloud/block-storage - Modal Détacher

## 1. VUE D'ENSEMBLE
- **Modal** : Détacher le volume d'une instance
- **Type** : Modal confirmation avec select
- **Objectif** : Détacher un volume de son instance

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal blanche centrée (600x280px)
- Header avec titre + fermeture
- Message confirmation
- Select instance
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message
- **OdsSelect** : Sélection instance
- **OdsButton** : Annuler, Confirmer

## 4. MESSAGE
"Détacher le Volume volume-gra-001 de l'instance my-instance-01"

## 5. SELECT
- Label : Sélectionner l'instance
- Valeur : my-instance-01

## 6. BOUTONS
- Annuler (outline)
- Confirmer (primary)

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/volume/{volumeId}/detach
