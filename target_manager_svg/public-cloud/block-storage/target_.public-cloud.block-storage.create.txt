# Cahier des Charges : public-cloud/block-storage - Créer un Volume

## 1. VUE D'ENSEMBLE
- **Page** : Créer un volume Block Storage
- **Type** : Wizard multi-étapes avec stepper
- **Objectif** : Configurer et créer un nouveau volume

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Block Storage actif)
- Breadcrumb
- Titre page
- Wizard avec 5 étapes
- Footer avec estimation et boutons

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsStepper** : Indicateur d'étapes
- **OdsTile** : Sélection visuelle
- **OdsInput** : Champs saisie
- **OdsCard** : Encart estimation
- **OdsButton** : Annuler, Créer

## 4. BREADCRUMB
"Public Cloud / Mon Projet Cloud / Block Storage / Créer un volume"

## 5. ÉTAPES WIZARD

### Étape 1 : Mode de déploiement
Options : Région 1-AZ (sélectionné), Région 3-AZ

### Étape 2 : Localisation
Options tiles : GRA9 (sélectionné), SBG5, BHS5, DE1

### Étape 3 : Type de volume
- Classique : 0,04€/Go/mois, 250 IOPS max
- Haute performance (sélectionné) : 0,08€/Go/mois, 20000 IOPS max

### Étape 4 : Capacité
Input : 100 Go, Bande passante : 400 Mo/s, IOPS : 3000

### Étape 5 : Nom
Input : "volume-data-prod"

## 6. FOOTER ESTIMATION
Montant estimé : 8,00€ HT/mois
Boutons : Annuler, Créer le volume

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/region
- GET /cloud/project/{projectId}/volume/type
- POST /cloud/project/{projectId}/volume
