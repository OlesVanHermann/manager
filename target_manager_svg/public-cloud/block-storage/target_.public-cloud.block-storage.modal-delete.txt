# Cahier des Charges : public-cloud/block-storage - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un volume
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression définitive d'un volume

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal blanche centrée (600x360px)
- Header avec titre + fermeture
- Icône warning
- Message confirmation
- Warning banner orange
- Infos volume
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : Warning
- **OdsMessage** : Warning banner
- **OdsButton** : Annuler, Supprimer

## 4. MESSAGE
"Supprimer définitivement le volume backup-volume"

## 5. WARNING BANNER
"Toutes les données sur le volume seront perdues.
Cette action est irréversible."

## 6. INFOS VOLUME
"Volume : backup-volume • 1 To • Classic • SBG5"

## 7. BOUTONS
- Annuler (outline)
- Supprimer (danger rouge)

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/volume/{volumeId}
