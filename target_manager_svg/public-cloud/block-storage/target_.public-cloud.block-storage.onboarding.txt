# Cahier des Charges : public-cloud/block-storage - Onboarding

## 1. VUE D'ENSEMBLE
- **Page** : Onboarding Block Storage
- **Type** : Page promotionnelle (empty state)
- **Objectif** : Présenter le service et inciter à créer un volume

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Block Storage actif)
- Breadcrumb
- Contenu centré avec icône, titre, description
- Liste des fonctionnalités
- Bouton CTA
- Cartes tutoriels

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsEmptyState** : Zone centrale
- **OdsButton** : CTA principal
- **OdsCard** : Cartes tutoriels

## 4. FONCTIONNALITÉS
- Volumes de 10 Go jusqu'à 4 To
- Attachement à chaud aux instances
- Triple réplication des données
- Type Classic (économique)
- Type High-speed (performance)
- Déplacement entre instances

## 5. BOUTON CTA
Label : "Créer un volume" (primary bleu)

## 6. TUTORIELS
- Créer et attacher un volume
- Configurer un volume
- Augmenter la taille

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/volume (retourne vide)
