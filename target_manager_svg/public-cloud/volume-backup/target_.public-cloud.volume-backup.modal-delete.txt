# Cahier des Charges : public-cloud/volume-backup - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer la sauvegarde de volume
- **Type** : Modal confirmation danger avec saisie
- **Objectif** : Confirmer la suppression définitive d'une sauvegarde

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x400px)
- Barre rouge en haut
- Titre avec close
- Message d'avertissement
- Champ de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Champ de confirmation DELETE
- **OdsButton** : Annuler (secondary), Supprimer (danger)

## 4. HEADER
Barre rouge (6px) + "Supprimer la sauvegarde de volume" + bouton ×

## 5. MESSAGE
- "Vous êtes sur le point de supprimer définitivement la sauvegarde **backup-prod-2024**."
- "Cette action est irréversible. Toutes les données de cette sauvegarde seront perdues."

## 6. CONFIRMATION
- **Label** : "Entrez « DELETE » pour confirmer votre choix."
- **Input** : Champ texte avec valeur "DELETE"

## 7. BOUTONS
- Annuler (secondary)
- Supprimer (danger/rouge)

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/volume/backup/{backupId}
  - Supprime la sauvegarde
