# Cahier des Charges : public-cloud/billing - Modal Forfait Mensuel

## 1. VUE D'ENSEMBLE
- **Page** : Facturation - Modal passage au forfait mensuel
- **Type** : Modal confirmation
- **Objectif** : Convertir une instance horaire en forfait mensuel

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 600x340px centrée
- Header avec titre et bouton fermer
- Corps avec explication et warning
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal
- **OdsText** : Description avantages
- **OdsLink** : "Voir les prix"
- **OdsMessage** : Warning prorata (jaune)
- **OdsText** : Instance concernée
- **OdsButton** : Annuler (secondaire)
- **OdsButton** : Confirmer (primaire)

## 4. TITRE
"Passer au forfait mensuel"

## 5. CONTENU
### Description
"Réduisez le montant de votre facture en passant votre instance au forfait mensuel."

### Lien
- "Voir les prix" (lien bleu souligné)

### Message Warning
- Fond: Jaune clair (#FEF3C7)
- Bordure: Orange (#F59E0B)
- Icône: ⚠ (orange)
- Texte: "Une facture au prorata des jours restants du mois va être immédiatement éditée."

### Instance concernée
- Label: "Instance concernée :"
- Valeur: "my-instance-01 (b2-15)" (gras)

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Secondaire outline | Gauche |
| Confirmer | Primaire bleu | Droite |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/instance/{instanceId}/activeMonthlyBilling
  - Active la facturation mensuelle
- GET /cloud/project/{projectId}/instance/{instanceId}
  - Récupère les détails de l'instance
