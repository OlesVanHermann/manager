# Cahier des Charges : public-cloud/notebooks - Modal Fork Backup

## 1. VUE D'ENSEMBLE
- **Modal** : Commander un notebook depuis un backup
- **Type** : Modal confirmation
- **Objectif** : Créer un nouveau notebook à partir d'une sauvegarde

## 2. STRUCTURE MODAL
- Overlay sombre (40% opacité)
- Modal blanche centrée
- Header avec titre + fermeture
- Corps avec détails backup
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Informations backup
- **OdsButton** : Annuler, Commander

## 4. DIMENSIONS
- Largeur : 500px
- Hauteur : ~220px

## 5. HEADER MODAL
- **Titre** : "Commander un notebook depuis un backup"
- **Fermeture** : × (coin droit)

## 6. CORPS MODAL
- **Question** : "Utiliser le backup"
- **ID backup** : "bkp-a1b2c3d4-e5f6" (bold)
- **Date** : "créé le 15 janv. 2024 10:30"
- **Suite** : "pour commander un nouveau notebook?"

## 7. BOUTONS FOOTER

| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline gris | Ferme la modal |
| Commander | Primary bleu | Lance la création |

## 8. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/notebook
  - Body: { ...notebookConfig, backupId: "bkp-xxx" }
  - Crée un notebook depuis un backup
