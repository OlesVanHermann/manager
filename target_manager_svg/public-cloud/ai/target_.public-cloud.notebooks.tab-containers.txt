# Cahier des Charges : public-cloud/notebooks - Onglet Conteneurs de Données

## 1. VUE D'ENSEMBLE
- **Page** : Conteneurs de données d'un notebook
- **Type** : Onglet avec datagrid
- **Objectif** : Gérer les volumes de données attachés

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Notebooks actif)
- Breadcrumb
- Onglets navigation
- Titre
- Info box explicative
- Bouton ajouter
- Datagrid conteneurs

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 5 onglets
- **OdsMessage** : Info box (info)
- **OdsButton** : Ajouter conteneur
- **OdsDatagrid** : Liste conteneurs
- **OdsBadge** : Permission, Cache
- **OdsActionMenu** : Menu contextuel

## 12. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/notebook/{notebookId}/volume
  - Liste des volumes attachés
- POST /cloud/project/{projectId}/ai/notebook/{notebookId}/volume
  - Attacher un volume
- DELETE /cloud/project/{projectId}/ai/notebook/{notebookId}/volume/{volumeId}
  - Détacher un volume
- POST /cloud/project/{projectId}/ai/notebook/{notebookId}/volume/{volumeId}/sync
  - Synchroniser les données
