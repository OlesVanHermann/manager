# Cahier des Charges : public-cloud/ai - Onglet Apps

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard AI Platform - Onglet Apps
- **Type** : Page tableau de bord avec tableau
- **Objectif** : Afficher les applications AI déployées

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar
- Breadcrumb
- Titre "AI Platform"
- Onglets (Notebooks, Jobs, Apps actif)
- Tableau des apps déployées

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 3 onglets
- **OdsButton** : Déployer une app (primaire)
- **OdsTable** : Liste des apps
- **OdsBadge** : Statuts (RUNNING, SCALING)
- **OdsLink** : URL de l'app

## 4. TITRE
"AI Platform"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Notebooks | Inactif |
| Jobs | Inactif |
| Apps | Actif (souligné bleu) |

## 6. SECTION APPS
- Titre: "AI Deploy Apps" avec badge compteur (3)
- Bouton: "Déployer une app" (primaire bleu)

## 7. TABLEAU APPS
### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Texte | Nom de l'application |
| Image | Texte mono | Image Docker utilisée |
| Replicas | Texte | Nombre de replicas actifs/total |
| Statut | Badge | RUNNING (vert), SCALING (jaune) |
| URL | Lien | URL d'accès (tronquée) |
| Actions | Bouton | Ouvrir / Scale |

### Données exemple
| Nom | Image | Replicas | Statut | URL |
|-----|-------|----------|--------|-----|
| sentiment-api | my-registry/sentiment:v2 | 3 / 3 | RUNNING | sentiment-api.ai.gra... |
| image-classifier | tensorflow/serving:2.14 | 2 / 2 | RUNNING | image-classifier.ai... |
| chatbot-llm | ollama/ollama:latest | 1 / 5 | SCALING | chatbot-llm.ai.gra... |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/app
- POST /cloud/project/{projectId}/ai/app/{appId}/scale
  - replicas: number
