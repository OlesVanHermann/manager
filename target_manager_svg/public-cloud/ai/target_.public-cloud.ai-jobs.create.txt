# Cahier des Charges : public-cloud/ai-jobs - Lancer un job

## 1. VUE D'ENSEMBLE
- **Page** : Formulaire de création d'un job AI Training
- **Type** : Page wizard avec récapitulatif
- **Objectif** : Configurer et lancer un job d'entraînement

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec AI Training actif
- Breadcrumb
- Titre "Lancer un job"
- Colonne gauche: Formulaire (sections)
- Colonne droite: Récapitulatif commande

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsCard** : Sections formulaire
- **OdsInput** : Nom, Image Docker
- **OdsRadio** : Région, Flavor, Confidentialité
- **OdsCard** : Récapitulatif (droite)
- **OdsButton** : Commander, Équivalent CLI
- **OdsLink** : Configuration avancée
