# Cahier des Charges : public-cloud/ai-apps - Onglet Logs

## 1. VUE D'ENSEMBLE
- **Page** : Logs en temps réel de l'application
- **Type** : Onglet avec terminal
- **Objectif** : Consulter les logs applicatifs

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre app + badge statut
- Onglets (Logs actif)
- Barre d'outils (Actualiser, Copier, Télécharger, Auto-refresh)
- Zone terminal (fond noir)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsBadge** : RUNNING (vert)
- **OdsTabs** : Dashboard, Conteneurs, Dépots Git, Logs
- **OdsButton** : Actualiser, Copier, Télécharger
- **OdsText** : Auto-refresh status
- **OdsTextarea** : Zone logs (style terminal)

## 4. TITRE
"{app-name}"
Badge: "RUNNING" (vert)

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Dashboard | Inactif |
| Conteneurs | Inactif |
| Dépots Git | Inactif |
| Logs | Actif (souligné bleu) |

## 6. BARRE D'OUTILS
| Bouton | Style | Action |
|--------|-------|--------|
| Actualiser | Outline | Rafraîchit les logs |
| Copier | Outline | Copie dans presse-papier |
| Télécharger | Outline | Télécharge fichier log |
| Auto-refresh: ON | Texte | Toggle auto-refresh |

## 7. ZONE TERMINAL
- Fond: Noir (#1F2937)
- Police: Monospace (Courier New)
- Couleurs logs:
  - Timestamp: Gris (#D1D5DB)
  - INFO: Vert (#10B981)
  - HTTP: Bleu (#60A5FA)
  - WARN: Jaune (#FBBF24)
  - ERROR: Rouge (#EF4444)
- Curseur clignotant en fin

## 8. FORMAT LOGS
```
[YYYY-MM-DD HH:MM:SS] LEVEL Message
```
Exemples:
- `[2024-01-15 14:30:00] INFO Application starting on port 8080...`
- `[2024-01-15 14:30:10] HTTP POST /predict - 200 OK - 45ms`
- `[2024-01-15 14:30:20] WARN GPU memory usage: 78%`
- `[2024-01-15 14:30:30] INFO Scaling event: replicas 1 → 2`

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/app/{appId}/log : Récupérer logs
  - since: timestamp (depuis quand)
  - tail: number (nb lignes)
