# Cahier des Charges : public-cloud/ai-endpoints - Liste clés API

## 1. VUE D'ENSEMBLE
- **Page** : Liste des clés API AI Endpoints
- **Type** : Page tableau
- **Objectif** : Gérer les clés d'accès aux modèles LLM

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec menu AI Endpoints actif
- Breadcrumb
- Titre "AI Endpoints"
- Onglets (Clés API actif, Métriques)
- Bouton "Créer une clé API"
- Tableau des clés
- Menu actions (⋮)
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Clés API, Métriques
- **OdsButton** : + Créer une clé API (outline bleu)
- **OdsTable** : Liste clés API
- **OdsBadge** : Statut (Actif, Expire bientôt)
- **OdsActionMenu** : Modifier, Supprimer

## 4. TITRE
"AI Endpoints"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Clés API | Actif (souligné bleu) |
| Métriques | Inactif |

## 6. BOUTON
| Bouton | Style | Action |
|--------|-------|--------|
| + Créer une clé API | Outline bleu | Ouvre modal-create |

| Colonne | Type | Exemple |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/endpoints/token : Lister clés API
- POST /cloud/project/{projectId}/ai/endpoints/token : Créer clé
- PUT /cloud/project/{projectId}/ai/endpoints/token/{tokenId} : Modifier
- DELETE /cloud/project/{projectId}/ai/endpoints/token/{tokenId} : Supprimer
