# Cahier des Charges : public-cloud/private-registry - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le registry
- **Type** : Modal confirmation destructive avec saisie DELETE
- **Objectif** : Supprimer un registre et toutes ses images

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x380px)
- Barre orange à gauche (warning)
- Icône warning
- Titre
- Message de confirmation
- Champ saisie DELETE
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsIcon** : ⚠ dans cercle orange
- **OdsText** : Message avec nom registre en rouge
- **OdsInput** : Champ saisie confirmation
- **OdsButton** : Annuler (secondary), Supprimer (danger)

## 4. HEADER
- **Icône** : ⚠ dans cercle orange
- **Titre** : "Supprimer le registry"

## 5. CONTENU

### Message
"Êtes-vous sûr de vouloir supprimer le registry"
**my-production-registry** (rouge) "?"

"Cette action est irréversible. Toutes les images Docker stockées seront définitivement supprimées."

### Confirmation
- **Label** : "Tapez DELETE pour confirmer"
- **Input** : Champ texte
- **Placeholder** : "DELETE"

## 6. BOUTONS
- Annuler (secondary)
- Supprimer (danger/rouge)

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/containerRegistry/{registryId}
  - Supprime le registre et toutes ses données
