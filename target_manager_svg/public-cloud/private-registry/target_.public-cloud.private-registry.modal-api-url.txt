# Cahier des Charges : public-cloud/private-registry - Modal API URL

## 1. VUE D'ENSEMBLE
- **Modal** : API du Registre privé
- **Type** : Modal informative avec champ copiable
- **Objectif** : Afficher l'URL API du registre

## 2. STRUCTURE MODAL
- Modal blanche (450x200px)
- Titre + bouton fermer
- Champ copiable avec URL
- Bouton annuler

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre
- **OdsClipboard** : URL API (fond gris, monospace)
- **OdsButton** : Annuler (outline)

## 4. HEADER
- **Titre** : "API du Registre privé"
- **Bouton fermer** : ×

## 5. CONTENU
- **Label** : "URL API de votre Registre privé"
- **Champ** : `https://abc123xyz.c1.gra9.container-registry.ovh.net` (fond gris, monospace, icône copie)

## 6. BOUTONS
- Annuler (outline)

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/containerRegistry/{registryId}/url
  - Récupère l'URL API du registre
