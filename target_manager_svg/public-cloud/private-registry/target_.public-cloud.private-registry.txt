# Cahier des Charges : public-cloud/private-registry - Liste Registres

## 1. VUE D'ENSEMBLE
- **Page** : Managed Private Registry - Liste
- **Type** : Page liste avec datagrid
- **Objectif** : Gérer les registres Harbor privés

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Private Registry actif)
- Breadcrumb
- Titre page
- Bouton créer
- Barre recherche
- Tableau registres

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre "Managed Private Registry"
- **OdsButton** : "+ Créer un registre privé" (primary)
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste registres
- **OdsBadge** : Plan (SMALL/MEDIUM bleu), Statut (READY vert)
- **OdsProgressBar** : Espace utilisé
- **OdsActionMenu** : Actions (⋮)

## 4. SIDEBAR
- Section : CONTAINERS & ORCHESTRATION
- Éléments :
  - Kubernetes
  - **Private Registry** (actif)
  - Workflow Management

## 5. TABLEAU REGISTRES

### Colonnes
| Colonne | Description |
|---------|-------------|
| Nom | Nom + ID tronqué |
| Région | GRA, SBG, etc. |
| Plan | Badge SMALL/MEDIUM/LARGE |
| Version Harbor | Numéro version |
| Espace utilisé | Barre de progression + ratio |
| Auth IAM | Activé/Désactivé |
| Statut | Badge READY |
| Actions | Menu ⋮ |

### Données exemple
| Nom | Région | Plan | Version | Espace | IAM | Statut |
|-----|--------|------|---------|--------|-----|--------|
| prod-registry (a1b2c3d4...) | GRA | MEDIUM | 2.10.0 | 45/200 Go | Activé | READY |
| dev-registry (e5f6g7h8...) | SBG | SMALL | 2.10.0 | 8/50 Go | Désactivé | READY |

## 6. ACTION MENU
Options (menu ouvert) :
- Interface Harbor
- API Harbor
- Générer credentials
- Changer de plan
- ---
- Supprimer (rouge)

## 7. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/containerRegistry
  - Liste des registres privés
- POST /cloud/project/{projectId}/containerRegistry
  - Crée un registre
- DELETE /cloud/project/{projectId}/containerRegistry/{registryId}
  - Supprime un registre
