# Cahier des Charges : public-cloud/ai-notebooks - Modal Supprimer volume

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression (détachement) d'un volume
- **Type** : Modal confirmation danger
- **Objectif** : Détacher un conteneur S3 du notebook

## 2. STRUCTURE MODAL
- Overlay sombre sur la page
- Modal centrée (600x320)
- Header avec titre et bouton fermer
- Icône danger + message
- Carte récapitulative du volume
- Note de rassurance
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Message de confirmation
- **OdsCard** : Récapitulatif volume (fond gris)
- **OdsButton** : Annuler (outline), Supprimer (danger)

## 4. TITRE
"Supprimer un conteneur"

## 5. CONTENU
### Icône
- Cercle rouge clair (#FEE2E2) avec "!" rouge

### Message
"Êtes-vous sûr de vouloir supprimer ce conteneur ?"

### Récapitulatif volume
- Fond gris (#F3F4F6)
- Alias: "training-data" (bold)
- Conteneur: "s3://my-ml-bucket"

### Note de rassurance
"Cette action supprimera le lien vers le conteneur."
"Les données dans le conteneur source ne seront pas supprimées."

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline gris | Ferme la modal |
| Supprimer | Danger rouge (#DC2626) | Détache le volume |

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/ai/notebook/{notebookId}/volumes/{volumeId}
