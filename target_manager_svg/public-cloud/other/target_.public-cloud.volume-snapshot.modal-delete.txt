# Cahier des Charges : public-cloud/volume-snapshot - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer le snapshot
- **Type** : Modal confirmation danger
- **Objectif** : Confirmer la suppression d'un snapshot de volume

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x250px)
- Barre rouge en haut
- Titre avec close
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsButton** : Annuler (secondary), Supprimer (danger)

## 4. HEADER
Barre rouge (6px) + "Supprimer le snapshot" + bouton ×

## 5. MESSAGE
- "Êtes-vous sûr de vouloir supprimer le snapshot **snap-staging-weekly** ?"
- "Cette action est irréversible."

## 6. BOUTONS
- Annuler (secondary)
- Supprimer (danger/rouge)

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/volume/snapshot/{snapshotId}
  - Supprime le snapshot
