# Cahier des Charges : public-cloud/billing - Modal Alerte Consommation

## 1. VUE D'ENSEMBLE
- **Page** : Billing Control - Modal configuration alerte
- **Type** : Modal formulaire
- **Objectif** : Configurer un seuil d'alerte de consommation

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal 600x380px centrée
- Header avec titre et bouton fermer
- Corps avec description et formulaire
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur principal
- **OdsInput** : Seuil en euros
- **OdsInput** : Email de notification
- **OdsButton** : Annuler (secondaire)
- **OdsButton** : Enregistrer (primaire)
- **OdsText** : Description explicative

## 4. TITRE
"Configurer l'alerte de consommation"

## 5. DESCRIPTION
"Recevez une notification par email lorsque votre consommation dépasse le seuil défini."

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Valeur exemple |
|-------|------|-------------|----------------|
| Seuil d'alerte | OdsInput number | - | 150 |
| Unité | Label | € HT par mois | - |
| Email de notification | OdsInput email | - | admin@exemple.com |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Secondaire outline | Gauche |
| Enregistrer | Primaire bleu | Droite |

## 8. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/alerting
  - Body: { threshold: number, email: string }
- GET /cloud/project/{projectId}/alerting
  - Récupère la configuration actuelle
