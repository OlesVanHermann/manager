# Cahier des Charges : public-cloud/private-registry - Modal Supprimer CIDR

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un bloc CIDR autorisé
- **Type** : Modal confirmation
- **Objectif** : Supprimer une restriction IP

## 2. STRUCTURE MODAL
- Modal blanche (400x180px)
- Barre orange en haut (warning)
- Titre + bouton fermer
- Message de confirmation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Message confirmation
- **OdsButton** : Annuler (outline), Confirmer (warning/orange)

## 4. HEADER
- **Barre orange** : Indicateur warning
- **Titre** : "Supprimer un bloc CIDR autorisé"
- **Bouton fermer** : ×

## 5. CONTENU
"Êtes vous sûr de vouloir supprimer le bloc CIDR ?"

## 6. BOUTONS
- Annuler (outline primary)
- Confirmer (warning/orange)

## 7. APIS & ENDPOINTS
- DELETE /cloud/project/{projectId}/containerRegistry/{registryId}/ipRestriction/{cidrId}
  - Supprime le bloc CIDR
