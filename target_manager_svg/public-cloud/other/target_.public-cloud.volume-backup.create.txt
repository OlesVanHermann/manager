# Cahier des Charges : public-cloud/volume-backup - Créer

## 1. VUE D'ENSEMBLE
- **Page** : Créer une sauvegarde de volume
- **Type** : Page formulaire 3 étapes
- **Objectif** : Créer une nouvelle sauvegarde de volume

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Volume Backup actif)
- Breadcrumb
- Titre et description
- Formulaire en 3 étapes
- Bouton de création

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsStepper** : Indicateur d'étapes (cercles numérotés)
- **OdsSelect** : Sélection volume
- **OdsTile** : Options type de sauvegarde
- **OdsRadio** : Sélection type
- **OdsInput** : Nom de la sauvegarde
- **OdsButton** : Créer la sauvegarde (primary)

## 4. HEADER
- **Titre** : "Créer une sauvegarde de volume"
- **Description** : "Effectuez des sauvegardes régulières pour une meilleure résilience de vos données."

## 5. ÉTAPE 1 : SÉLECTION VOLUME
- **Indicateur** : ① bleu
- **Titre** : "Sélectionnez un volume"
- **Select** : "volume-prod-01 (GRA11 - 100 GB)"

## 6. ÉTAPE 2 : TYPE DE SAUVEGARDE
- **Indicateur** : ② bleu
- **Titre** : "Type de sauvegarde"

### Options (Tiles avec radio)
| Type | Description | Prix |
|------|-------------|------|
| Volume Snapshot | Snapshot instantané du volume. Stocké dans la même région | 0,01 €/GB/mois |
| **Volume Backup** ● | Sauvegarde complète du volume. Stockage Object Storage répliqué | 0,02 €/GB/mois |

## 7. ÉTAPE 3 : NOM
- **Indicateur** : ③ bleu
- **Titre** : "Nom de la sauvegarde"
- **Input** : "backup-volume-prod-01-2024-12-27T14:30:00"

## 8. BOUTON
"Créer la sauvegarde" (primary)

## 9. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/volume
  - Liste des volumes disponibles
- POST /cloud/project/{projectId}/volume/{volumeId}/backup
  - Body: { name, type }
  - Crée la sauvegarde
