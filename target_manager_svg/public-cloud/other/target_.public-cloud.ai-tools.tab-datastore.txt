# Cahier des Charges : public-cloud/ai-tools - Onglet Datastores S3

## 1. VUE D'ENSEMBLE
- **Page** : Gestion des datastores S3
- **Type** : Onglet avec datagrid
- **Objectif** : Configurer les conteneurs S3 pour les produits AI

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Dashboard actif)
- Breadcrumb
- Titre AI Dashboard
- Onglets (Datastores S3 actif avec compteur)

- **OdsMessage** : Alert info S3
- **OdsButton** : Configurer un datastore S3
- **OdsDatagrid** : Liste datastores
- **OdsActionMenu** : Menu contextuel
