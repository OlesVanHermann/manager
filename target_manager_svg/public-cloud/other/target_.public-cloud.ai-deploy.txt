# Cahier des Charges : public-cloud/ai-deploy - Liste Déploiements

## 1. VUE D'ENSEMBLE
- **Page** : Liste des déploiements AI Deploy
- **Type** : Datagrid avec filtres
- **Objectif** : Gérer les modèles AI déployés en production

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (AI Deploy actif)
- Breadcrumb
- Titre + bouton créer
- Barre recherche
- Datagrid

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsText** : Titre
- **OdsButton** : Créer un déploiement
- **OdsInput** : Recherche
- **OdsDatagrid** : Liste déploiements
- **OdsBadge** : Statut (RUNNING, SCALING, STOPPED)

## 4. BREADCRUMB

## 8. COLONNES DATAGRID

| Colonne | Type | Description |
|---------|------|-------------|
| Déploiement | Link + ID | Nom + ID tronqué |
| Modèle | Text + version | Nom modèle + version/source |
| Ressources | Text | GPU type + VRAM |
| Endpoint | Link mono | URL du endpoint |
| Replicas | Text | N / M (actifs/total) |
| Requêtes/min | Number | Métriques temps réel |
| Statut | Badge | RUNNING/SCALING/STOPPED |

## 11. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/ai/deploy
  - Liste des déploiements
- POST /cloud/project/{projectId}/ai/deploy
  - Créer un déploiement
- DELETE /cloud/project/{projectId}/ai/deploy/{deployId}
  - Supprimer un déploiement
