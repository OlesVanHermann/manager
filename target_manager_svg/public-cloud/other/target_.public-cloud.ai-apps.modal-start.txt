# Cahier des Charges : public-cloud/ai-apps - Modal Démarrer

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation démarrage application
- **Type** : Modal confirmation
- **Objectif** : Confirmer le démarrage d'une app arrêtée

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal centrée (500x220px)
- Header avec titre
- Corps avec message
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Titre, message
- **OdsButton** : Annuler (outline), Démarrer (primary)

## 4. HEADER
- **Titre** : "Démarrer une app"
- **Bouton fermer** : ✕

## 5. CONTENU
"Êtes-vous sûr de vouloir démarrer l'application **my-llm-app** ?"

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer modal |
| Démarrer | Primary bleu | POST start |

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/ai/app/{appId}/start
  - Démarrer l'application
