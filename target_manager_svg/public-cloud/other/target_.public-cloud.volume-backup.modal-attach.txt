# Cahier des Charges : public-cloud/volume-backup - Modal attacher volume

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Volume Backup Public Cloud
- **Objectif** : Attacher un volume backup à une instance

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Attacher votre volume" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Souhaitez-vous rattacher votre {volumeName} à cette instance {instanceName} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Warning (orange) | Attache le volume |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{serviceName}/volume/{volumeId}/attach - Attacher volume
