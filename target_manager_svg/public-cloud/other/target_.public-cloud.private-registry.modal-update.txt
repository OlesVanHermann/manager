# Cahier des Charges : public-cloud/private-registry - Modal Renommer

## 1. VUE D'ENSEMBLE
- **Modal** : Renommer le Registre privé
- **Type** : Modal formulaire simple
- **Objectif** : Modifier le nom du registre

## 2. STRUCTURE MODAL
- Modal blanche (400x240px)
- Titre + bouton fermer
- Champ input nom
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nom du registre
- **OdsButton** : Annuler (outline), Confirmer (primary)

## 4. HEADER
- **Titre** : "Renommer le Registre privé"
- **Bouton fermer** : ×

## 5. FORMULAIRE
- **Label** : "Choisir un nom pour le Registre privé"
- **Input** : "my-registry-name"

## 6. BOUTONS
- Annuler (outline primary)
- Confirmer (primary)

## 7. APIS & ENDPOINTS
- PUT /cloud/project/{projectId}/containerRegistry/{registryId}
  - Body: { name: "new-name" }
  - Modifie le nom du registre
