# Cahier des Charges : public-cloud/billing - Onglet Consommation

## 1. VUE D'ENSEMBLE
- **Page** : Facturation - Ma consommation actuelle
- **Type** : Page détail facturation
- **Objectif** : Afficher le détail de la consommation du mois en cours

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Facturation actif
- Breadcrumb
- Titre "Facturation"
- Onglets (Ma consommation actuelle actif)
- 2 cartes (Facturation mensuelle, Facturation horaire)
- Tableau détail consommation

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 3 onglets
- **OdsCard** : 2 cartes montants
- **OdsTable** : Détail consommation

## 4. TITRE
"Facturation"
