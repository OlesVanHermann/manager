# Cahier des Charges : public-cloud/ai-notebooks - Modal Démarrer

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de démarrage d'un notebook
- **Type** : Modal confirmation simple
- **Objectif** : Demander confirmation avant de démarrer un notebook arrêté

## 2. STRUCTURE MODAL
- Overlay sombre sur la page
- Modal centrée (500x220)
- Header avec titre et bouton fermer
- Message de confirmation
- Boutons Annuler / Démarrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Message et nom du notebook
- **OdsButton** : Annuler (outline), Démarrer (primaire)

## 4. TITRE
"Démarrer le notebook"

## 5. CONTENU
### Message
"Êtes-vous sûr de vouloir démarrer le notebook"
"**data-exploration**" (en gras)
"?"

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline gris | Ferme la modal |
| Démarrer | Primaire bleu | Démarre le notebook |

## 7. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/ai/notebook/{notebookId}/start
