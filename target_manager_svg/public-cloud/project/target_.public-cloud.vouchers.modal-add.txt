# Cahier des Charges : public-cloud/vouchers - Modal Activer Voucher

## 1. VUE D'ENSEMBLE
- **Modal** : Activer un voucher
- **Type** : Modal formulaire
- **Objectif** : Activer un code voucher sur le projet

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (600x360px)
- Titre avec close
- Champ de saisie code
- Info message
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Code du voucher
- **OdsMessage** : Info (bleu)
- **OdsButton** : Annuler (secondary), Activer (primary)

## 4. HEADER
"Activer un voucher" + bouton ×

## 5. DESCRIPTION
"Entrez le code de votre voucher pour l'activer sur ce projet."

## 6. CHAMP
- **Label** : "Code du voucher"
- **Placeholder** : "Ex: CLOUD-2024-XXXXX"

## 7. INFO MESSAGE
Fond bleu clair : "ℹ️ Le crédit sera automatiquement appliqué à votre prochaine facture."

## 8. BOUTONS
- Annuler (secondary)
- Activer (primary)

## 9. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/credit
  - Body: { voucherCode }
  - Active le voucher
