# Cahier des Charges : public-cloud/vouchers - Modal Acheter Crédit

## 1. VUE D'ENSEMBLE
- **Modal** : Acheter du crédit cloud
- **Type** : Modal formulaire avec sélection montant
- **Objectif** : Acheter du crédit prépayé pour le projet

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x480px)
- Titre avec close
- Description
- Montants suggérés (boutons)
- Montant personnalisé (input)
- Récapitulatif prix
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsTile** : Montants prédéfinis (10€, 25€, 50€, 100€, 200€)
- **OdsInput** : Montant personnalisé
- **OdsCard** : Récapitulatif (fond bleu)
- **OdsButton** : Annuler (secondary), Payer (primary)

## 4. HEADER
"Acheter du crédit cloud" + bouton ×

## 5. DESCRIPTION
"Ajoutez du crédit prépayé à votre compte pour vos projets Public Cloud."

## 6. MONTANTS SUGGÉRÉS
Boutons de sélection rapide :
| 10 € | 25 € | **50 €** (sélectionné) | 100 € | 200 € |

## 7. MONTANT PERSONNALISÉ
- **Label** : "Ou saisissez un montant personnalisé (HT)"
- **Input** : Nombre avec suffixe "€"

## 8. RÉCAPITULATIF
Fond bleu clair :
| Ligne | Valeur |
|-------|--------|
| Montant HT | 50,00 € |
| TVA (20%) | 10,00 € |
| **Total TTC** | **60,00 €** |

## 9. BOUTONS
- Annuler (secondary)
- Payer (primary)

## 10. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/credit/order
  - Body: { amount }
  - Crée une commande de crédit
