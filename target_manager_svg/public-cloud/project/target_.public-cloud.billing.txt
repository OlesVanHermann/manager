# Cahier des Charges : public-cloud/billing - Facturation principale

## 1. VUE D'ENSEMBLE
- **Page** : Billing Control - Vue d'ensemble
- **Type** : Page dashboard facturation
- **Objectif** : Visualiser la consommation et les coûts du projet

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Billing Control actif
- Breadcrumb
- Titre "Billing Control"
- Onglets (Ma consommation actuelle actif, Estimation, Historique)
- 3 cartes KPI
- Tableau détail par service

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 3 onglets
- **OdsCard** : 3 cartes KPI
- **OdsTable** : Détail par service

## 4. TITRE
"Billing Control"

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Ma consommation actuelle | Actif (souligné bleu) |
| Estimation | Inactif |
| Historique | Inactif |

## 6. CARTES KPI
3 cartes côte à côte:

| Titre | Valeur | Note |

## 7. TABLEAU DÉTAIL PAR SERVICE
### En-tête
"Détail par service"

### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Service | Texte | Nom du service |
| Quantité | Texte | Quantité utilisée |
| Montant HT | Montant | Prix |

### Données exemple
| Service | Quantité | Montant |
|---------|----------|---------|
| Instances (Compute) | 3 instances | 89.50 € |
| Block Storage | 250 GB | 18.75 € |
| Object Storage | 1.2 TB | 12.40 € |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/usage/current
  - Consommation actuelle par service
- GET /cloud/project/{projectId}/usage/forecast
  - Estimation de fin de mois
- GET /cloud/project/{projectId}/credit
  - Crédit disponible
