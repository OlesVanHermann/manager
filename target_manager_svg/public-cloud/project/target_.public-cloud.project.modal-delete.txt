# Cahier des Charges : public-cloud/project - Modal résilier le projet

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Dashboard projet Public Cloud
- **Objectif** : Résilier un projet Public Cloud et toutes ses ressources

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message d'avertissement
- **OdsButton** : Non refuser, Oui résilier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résilier le projet" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Avertissement ligne 1 | "Il est important de noter que tous vos services, ressources," |
| Avertissement ligne 2 | "paramètres et données seront détruits, sans possibilité de" |
| Avertissement ligne 3 | "restauration ultérieure, à l'exception des IP Failovers," |
| Avertissement ligne 4 | "qui seront parkées." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Non, refuser | Secondary | Ferme modal |
| Oui, résilier | Danger (rouge) | Résilie le projet |

## 7. COMPORTEMENT
- Destruction irréversible de toutes les ressources
- Les IP Failover sont conservées (parkées)
- Facturation finale générée

## 8. APIS & ENDPOINTS
- DELETE /cloud/project/{serviceName}/terminate - Résilier le projet
