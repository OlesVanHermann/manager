# Cahier des Charges : public-cloud/quota - Modal Augmenter

## 1. VUE D'ENSEMBLE
- **Modal** : Augmenter mes quotas
- **Type** : Modal formulaire avec aperçu
- **Objectif** : Acheter du crédit pour augmenter les quotas

## 2. STRUCTURE MODAL
- Overlay sombre
- Modal blanche (700x560px)
- Header avec titre
- Texte explicatif
- Sélection montant crédit
- Aperçu nouveaux quotas
- Info message
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsSelect** : Montant du crédit
- **OdsCard** : Aperçu quotas (fond bleu clair)
- **OdsMessage** : Info (bleu)
- **OdsButton** : Annuler (secondary), Acheter (primary)

## 4. HEADER
"Augmenter mes quotas" + bouton ×

## 5. DESCRIPTION
- "Pour augmenter vos quotas, vous pouvez acheter du crédit prépayé."
- "Les quotas seront automatiquement ajustés en fonction de votre crédit."

## 6. SÉLECTION CRÉDIT
- **Label** : "Montant du crédit à acheter"
- **Dropdown** : "100 € - Quotas x2"

## 7. APERÇU QUOTAS
Fond bleu clair :
| Ressource | Avant | Après |
|-----------|-------|-------|
| Serveurs | 20 | **40** |
| vCPU | 40 | **80** |
| RAM | 80 GB | **160 GB** |
| Disques | 20 | **40** |
| Stockage | 1 TB | **2 TB** |

## 8. INFO MESSAGE
Fond bleu clair : "ℹ️ Le crédit sera déduit de vos prochaines factures jusqu'à épuisement."

## 9. BOUTONS
- Annuler (secondary)
- Acheter 100 € (primary)

## 10. APIS & ENDPOINTS
- POST /cloud/project/{projectId}/credit
  - Body: { amount }
  - Achète du crédit prépayé
