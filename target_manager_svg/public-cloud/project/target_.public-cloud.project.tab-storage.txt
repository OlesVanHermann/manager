# Cahier des Charges : public-cloud/project - Onglet Object Storage

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard projet - Onglet Object Storage
- **Type** : Page liste avec datagrid
- **Objectif** : Afficher les conteneurs Object Storage du projet

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar
- Breadcrumb
- Titre projet
- Onglets (Object Storage actif)
- Tableau conteneurs

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Onglets ressources (Object Storage actif)
- **OdsChip** : Compteur (3)
- **OdsDatagrid** : Liste conteneurs

## 5. TABLEAU

### Colonnes
| Colonne | Description |
|---------|-------------|
| Nom | Nom du conteneur |
| Objets | Nombre d'objets |
| Taille | Taille totale |
| Région | Code région |

### Données exemple
| Nom | Objets | Taille | Région |
|-----|--------|--------|--------|
| assets-production | 1,234 | 45.2 GB | GRA |
| backups-daily | 89 | 120.5 GB | SBG |
| logs-archive | 5,678 | 890.0 GB | GRA |

## 6. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/storage
  - Liste des conteneurs Object Storage
- GET /cloud/project/{projectId}/storage/{containerId}
  - Détails d'un conteneur
