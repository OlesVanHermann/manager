# Cahier des Charges : public-cloud/project - Onglet contacts & droits

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet gestion des contacts
- **Contexte** : Dashboard projet Public Cloud
- **Objectif** : Gérer les contacts et droits d'accès au projet

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, descriptions
- **OdsCard** : Informations propriétaire/facturation
- **OdsLink** : Modifier les contacts
- **OdsButton** : Ajouter, Guides
- **OdsDatagrid** : Liste des droits
- **OdsActionMenu** : Actions par ligne

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Contacts & droits" |
| Bouton | "Guides ▼" (dropdown) |

## 5. CARTE PROPRIÉTAIRE/FACTURATION

### Informations affichées
| Champ | Valeur | Action |
|-------|--------|--------|
| Propriétaire | ab12345-ovh | Modifier ↗ |
| Contact facturation | ab12345-ovh | Modifier ↗ |

## 6. SECTION DROITS

### Description
| Texte |
|-------|
| "Les personnes ayant des droits sur le projet peuvent visualiser et/ou administrer votre projet." |
| "Remarque : Toute demande de changement de droits passe par un processus de validation." |

### Bouton d'ajout
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter" |
| Style | Secondary (bordure bleue) |

## 7. TABLEAU DES DROITS

### Colonnes
| Colonne | Description |
|---------|-------------|
| NIC | Identifiant client OVH |
| DROITS | Niveau d'accès |
| ⋮ | Menu d'actions |

### Types de droits
| Droit | Description |
|-------|-------------|
| Lecture seule | Visualisation uniquement |
| Lecture/Écriture | Administration complète |

### ActionMenu (⋮)
| Option | Style |
|--------|-------|
| Supprimer | Rouge (danger) |

## 8. APIS & ENDPOINTS
- GET /cloud/project/{serviceName}/acl - Liste des droits
- POST /cloud/project/{serviceName}/acl - Ajouter un droit
- DELETE /cloud/project/{serviceName}/acl/{accountId} - Supprimer un droit
- GET /cloud/project/{serviceName}/serviceInfos - Infos contacts
