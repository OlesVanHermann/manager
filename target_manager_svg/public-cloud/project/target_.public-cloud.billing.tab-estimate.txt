# Cahier des Charges : public-cloud/billing - Onglet Estimation

## 1. VUE D'ENSEMBLE
- **Page** : Billing Control - Estimation
- **Type** : Page estimation facturation
- **Objectif** : Afficher l'estimation de la prochaine facture

## 2. STRUCTURE PAGE
- Header OVH
- Sidebar avec Billing Control actif
- Titre "Billing Control"
- Onglets (Estimation actif)
- Carte estimation
- Carte alerte consommation (si seuil atteint)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 3 onglets
- **OdsCard** : Estimation, Alerte
- **OdsButton** : Modifier l'alerte
- **OdsMessage** : Alerte warning

## 4. TITRE
"Billing Control"

|--------|------|
| Ma consommation actuelle | Inactif |
