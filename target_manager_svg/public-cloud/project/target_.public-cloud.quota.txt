# Cahier des Charges : public-cloud/quota - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Quota and Regions
- **Type** : Page avec tableau et progress bars
- **Objectif** : Visualiser et gérer les quotas par localisation

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar (Quota and Regions actif sous Project Management)
- Breadcrumb
- Titre "Quota and Regions"
- Onglets (Quota actif, Localisations)
- Info box avec CTA
- Toggle autoscaling
- Tableau des quotas par région

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Quota | Localisations
- **OdsMessage** : Info box bleue
- **OdsButton** : Augmenter mes quotas (primary)
- **OdsLink** : Comprendre les quotas, Contacter le support
- **OdsToggle** : Quota autoscaling (activé)
- **OdsDatagrid** : Quotas par région
- **OdsProgressBar** : Utilisation des ressources

## 4. INFO BOX
Fond bleu clair :
- **Titre** : "Protégez votre projet"
- **Description** : "Augmentez vos quotas pour bénéficier de plus de ressources sur votre projet."
- **CTA** : "Augmenter mes quotas" (primary)
- **Liens** : "Comprendre les quotas →" | "Contacter le support →"

## 5. TOGGLE AUTOSCALING
- **Label** : "Quota autoscaling"
- **État** : Activé (vert)

## 6. TABLEAU QUOTAS

### Colonnes
| Colonne | Description |
|---------|-------------|
| Localisation | Région (ex: Gravelines GRA11) |
| Serveurs | Usage X / Y avec progress bar |
| vCPU | Usage X / Y avec progress bar |
| RAM | Usage X / Y GB avec progress bar |
| Disques | Usage X / Y avec progress bar |
| Stockage (GB) | Usage X / Y avec progress bar |

### Données exemple
| Localisation | Serveurs | vCPU | RAM | Disques | Stockage |
|--------------|----------|------|-----|---------|----------|
| Gravelines (GRA11) | 3/20 | 12/40 | 24/80 GB | 5/20 | 250/1000 |
| Strasbourg (SBG5) | 15/20 ⚠ | 35/40 ⚠ | 70/80 GB ⚠ | 12/20 | 800/1000 ⚠ |
| Varsovie (WAW1) | 0/20 | 0/40 | 0/80 GB | 0/20 | 0/1000 |

### Progress bar couleurs
- Bleu : usage normal (<75%)
- Orange/Warning : usage élevé (>75%)

## 7. PAGINATION
"3 localisations" | "Page 1 sur 1"

## 8. APIS & ENDPOINTS
- GET /cloud/project/{projectId}/quota
  - Quotas par région
- PUT /cloud/project/{projectId}/quota/autoscaling
  - Activer/désactiver autoscaling
