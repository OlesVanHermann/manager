# Cahier des Charges : home/billing - Modal remboursement compte prépayé

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page compte prépayé OVH
- **Objectif** : Demander le remboursement du solde du compte prépayé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Montant, informations
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Demander un remboursement" |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Montant
| Texte |
|-------|
| "Vous souhaitez demander le remboursement de" |
| "**25,50 €** de votre compte prépayé." |

### Informations
| Texte |
|-------|
| "Le remboursement sera effectué sur votre moyen de paiement par défaut dans un délai de 5 à 10 jours ouvrés." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary | Demande le remboursement |

## 7. APIS & ENDPOINTS
- GET /me/ovhAccount - Solde compte prépayé
- POST /me/ovhAccount/refund - Demander remboursement
