# Cahier des Charges : home/billing - Mouvements des bons

## 1. VUE D'ENSEMBLE
- **Type** : Page de détail avec historique
- **URL** : /billing/vouchers/{voucherId}/movements
- **Objectif** : Consulter l'historique des mouvements d'un bon de réduction

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Facturation / Bons / Mouvements
- **OdsLink** : Retour
- **OdsCard** : Container tableau
- **OdsDatagrid** : Liste des mouvements
- **OdsButton** : Rafraîchir
- **OdsMessage** : Solde actuel
- **OdsPagination** : Navigation

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Mouvements du bon VOUCHER-123ABC" |
