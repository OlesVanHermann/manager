# Cahier des Charges : home/billing - Bons de commande

## 1. VUE D'ENSEMBLE
- **Type** : Page de liste avec gestion
- **URL** : /billing/purchase-orders
- **Objectif** : Gérer les bons de commande (Purchase Orders) pour achats entreprise

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Facturation / Bons de commande
- **OdsMessage** : Warning validation en cours
- **OdsCard** : Container tableau
- **OdsButton** : Ajouter une référence
- **OdsDatagrid** : Liste des bons de commande
- **OdsBadge** : Statuts (Actif, Inactif, Désactivé)
- **OdsActionMenu** : Actions par ligne
- **OdsPagination** : Navigation

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | "Bons de commande" |
| Description | "Gérez vos bons de commande pour les achats de services OVHcloud." |

## 5. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠️ Un bon de commande est en cours de validation. Vous ne pouvez pas en créer un nouveau pour le moment." |

## 6. ACTION PRINCIPALE
| Bouton | Action |
|--------|--------|
| + Ajouter une référence | Ouvre modal de création |

## 9. APIS & ENDPOINTS
- GET /me/billing/purchaseOrder - Liste des bons de commande
- POST /me/billing/purchaseOrder - Créer bon de commande
- PUT /me/billing/purchaseOrder/{id} - Modifier bon de commande
- DELETE /me/billing/purchaseOrder/{id} - Désactiver bon de commande
