# Cahier des Charges : home/billing - Services (Renouvellement)

## 1. VUE D'ENSEMBLE
- **Type** : Gestion renouvellement services
- **URL** : /services
- **Objectif** : Gérer le renouvellement automatique

## 2. STRUCTURE PAGE
- Sidebar avec Mes services actif
- Filtres (Actions groupées, Type, Renouvellement)
- Recherche
- Datagrid des services
- Pagination

## 3. COMPOSANTS UI
- **OdsSelect** : Actions groupées, Type, Renouvellement
- **OdsInput** : Recherche service
- **OdsCheckbox** : Sélection multiple
- **OdsDatagrid** : Liste services
- **OdsBadge** : Statuts renouvellement
- **OdsActionMenu** : Menu contextuel

## 4. FILTRES
| Filtre | Options |
|--------|---------|
| Actions groupées | Activer auto, Désactiver auto, Résilier |
| Type | Tous, Domaine, VPS, Serveur dédié, Email... |
| Renouvellement | Tous, Automatique, Manuel |

## 5. COLONNES DATAGRID
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Checkbox | 35px | Sélection |
| Service | 280px | Nom (link) |
| Type | 120px | Type produit |
| Date de renouvellement | 150px | DD/MM/YYYY |
| Renouvellement | 110px | Badge (Automatique/Manuel) |
| Statut | 60px | Badge Actif |
| Actions | 50px | Menu ⋮ |

## 6. ACTION MENU
| Option | Action |
|--------|--------|
| Modifier le renouvellement | Modal config |
| Anticiper le paiement | Payer maintenant |
| Configurer le service | Vers config |
| --- | --- |
| Résilier à échéance | Modal résiliation |
| Résilier immédiatement | Modal résiliation (rouge) |

## 7. BADGES RENOUVELLEMENT
| Type | Badge | Couleur |
|------|-------|---------|
| Automatique | Vert | #DEF7EC |
| Manuel | Jaune | #FEF3C7 |

## 8. APIS & ENDPOINTS
- GET /services - Liste services
- PUT /services/{id}/renew - Config renouvellement
- DELETE /services/{id} - Résilier
