# Cahier des Charges : home/billing - Modal Résiliation Service

## 1. VUE D'ENSEMBLE
- **Type** : Modal résiliation avec choix type
- **Déclencheur** : Action menu "Résilier"
- **Objectif** : Choisir entre résiliation immédiate ou à échéance

## 2. STRUCTURE MODAL
- Header warning (rouge)
- Service concerné
- Box attention
- Radio buttons type résiliation
- Footer actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x450px
- **OdsInput** : Service (readonly)
- **OdsMessage** : Attention (rouge)
- **OdsRadio** : Type résiliation
- **OdsButton** : Annuler, Résilier (rouge)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "⚠️ Résilier un service" |
| Background | #FEE2E2 |
| Couleur texte | #991B1B |

## 5. BOX ATTENTION
| Propriété | Valeur |
|-----------|--------|
| Type | Error (rouge) |
| Titre | "Attention !" |
| L1 | "Cette action est irréversible." |
| L2 | "Toutes les données seront définitivement supprimées à la date d'échéance." |

## 6. TYPE RÉSILIATION (Radio)
| Option | Description | Style |
|--------|-------------|-------|
| À échéance (DD/MM/YYYY) | Résiliation programmée | Normal |
| Immédiatement | Résiliation immédiate | Normal |

## 7. RADIO BUTTONS STYLE
| Propriété | Valeur |
|-----------|--------|
| Dimensions | 270x50px chacun |
| Layout | 2 colonnes |
| Sélectionné | Border bleu, cercle plein |
| Non sélectionné | Border gris |

## 8. ACTIONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Résilier | Danger rouge #DC2626 | "Résilier" |

## 9. APIS & ENDPOINTS
- POST /services/{id}/terminate - Résilier à échéance
- DELETE /services/{id} - Résilier immédiatement
