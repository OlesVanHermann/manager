# Cahier des Charges : home/billing - Factures avec Alerte

## 1. VUE D'ENSEMBLE
- **Type** : Liste factures avec dette
- **URL** : /billing/invoices (avec solde dû)
- **Objectif** : Afficher les factures avec bannière de dette

## 2. STRUCTURE PAGE
- Bannière alerte dette
- Tabs
- Actions de masse + Recherche
- Datagrid factures
- Pagination

## 3. COMPOSANTS UI
- **OdsMessage** : Bannière warning (jaune)
- **OdsButton** : "Régler mon solde maintenant"
- **OdsDatagrid** : Liste factures
- **OdsBadge** : Statuts

## 4. BANNIÈRE DETTE
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Icône | ⚠️ |
| Message | "Un montant de 99,90 € est dû pour le renouvellement" |
| CTA | "Régler mon solde maintenant" (outline) |
| Dimensions | 1120x60px |

| Colonne | Contenu |
|---------|---------|

## 6. APIS & ENDPOINTS
- GET /me/bill - Liste factures
- GET /me/debtAccount - Solde dû
- POST /me/debtAccount/pay - Régler dette
