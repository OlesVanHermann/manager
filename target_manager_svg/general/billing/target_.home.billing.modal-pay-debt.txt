# Cahier des Charges : home/billing - Modal Régler Solde

## 1. VUE D'ENSEMBLE
- **Type** : Modal paiement dette
- **Déclencheur** : Clic "Régler mon solde" dans bannière
- **Objectif** : Régler le solde impayé

## 2. STRUCTURE MODAL
- Header avec titre
- Message explicatif
- Box montant (warning)
- Select moyen de paiement
- Checkbox CGV
- Footer actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x400px
- **OdsMessage** : Box montant (warning jaune)
- **OdsSelect** : Moyen de paiement
- **OdsCheckbox** : Accepter CGV
- **OdsButton** : Annuler, Payer

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Régler mon solde" |
| Background | #F3F4F6 |

## 5. BOX MONTANT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Label | "Montant à régler" |
| Montant | 99,90 € (24px bold) |
| Background | #FEF3C7 |

## 6. SELECT MOYEN DE PAIEMENT
| Propriété | Valeur |
|-----------|--------|
| Label | "Moyen de paiement" |
| Défaut | Carte par défaut |
| Options | Liste moyens enregistrés |

## 7. CHECKBOX CGV
| Propriété | Valeur |
|-----------|--------|
| Label | "J'accepte les conditions générales de vente" |
| État initial | Checked |
| Requis | Oui |

## 8. ACTIONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Payer | Primary bleu | "Payer 99,90 €" |

## 9. APIS & ENDPOINTS
- POST /me/debtAccount/pay - Régler dette
- GET /me/paymentMethod - Moyens disponibles
