# Cahier des Charges : home/account - Modal Ajouter restriction IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal ajout restriction IP
- **URL** : N/A (composant modal)
- **Objectif** : Définir une règle de restriction d'accès par IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 450px |
| Hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Description
- **OdsInput** : Adresse IP ou plage CIDR
- **OdsCheckbox** : Alerte connexion
- **OdsSelect** : Type de règle
- **OdsBadge** : Indicateurs règles
- **OdsButton** : Actions

## 4. FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Adresse IP ou plage | Input text | Oui | 192.168.1.0/24 |
| Recevoir alerte | Checkbox | Non | Connexion depuis cette IP |
| Règle | Select dropdown | Oui | Autoriser |

## 5. OPTIONS RÈGLE
| Règle | Badge | Couleur |
|-------|-------|---------|
| Autoriser (ACCEPT) | Vert | #D1FAE5 |
| Bloquer (DENY) | Rouge | #FEE2E2 |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Ajouter | Primary | Créer restriction |

## 7. APIS & ENDPOINTS
- POST /me/accessRestriction/ip - Ajouter restriction IP
