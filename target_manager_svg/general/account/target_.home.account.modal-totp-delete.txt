# Cahier des Charges : home/account - Modal supprimer 2FA TOTP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de suppression avec vérification
- **Contexte** : Page sécurité du compte (2FA)
- **Objectif** : Supprimer une application TOTP (Google Authenticator, etc.)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Messages informatifs
- **OdsInput** : Code de vérification TOTP
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'application TOTP" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Titre section
| Texte |
|-------|
| "Supprimer cette application d'authentification" |

### Information
| Texte |
|-------|
| "Cette application ne sera plus utilisée pour générer les codes de vérification." |

### Champ de vérification
| Champ | Type | Placeholder |
|-------|------|-------------|
| Code de vérification * | Input | "Code à 6 chiffres de votre application" |

### Aide
| Texte |
|-------|
| "Entrez le code actuellement affiché dans votre application d'authentification." |

## 6. FOOTER MODAL
| Bouton | Style | État |
|--------|-------|------|
| Annuler | Secondary | Actif |
| Supprimer | Danger (rouge) | Désactivé tant que code non saisi |

## 7. APIS & ENDPOINTS
- DELETE /me/accessRestriction/totp/{id} - Supprimer 2FA TOTP (avec code de validation)
