# Cahier des Charges : home/account - Modal 2FA SMS

## 1. VUE D'ENSEMBLE
- **Type** : Modal activation 2FA
- **Déclencheur** : Clic "Activer" sur carte SMS
- **Objectif** : Configurer l'authentification par SMS

## 2. STRUCTURE MODAL
- Header avec titre
- Étape 1 : Saisie numéro téléphone
- Étape 2 : Vérification code
- Description optionnelle
- Footer actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x480px
- **OdsInput** : Numéro, Code, Description
- **OdsButton** : Envoyer code, Annuler, Activer

## 4. ÉTAPE 1 : NUMÉRO TÉLÉPHONE
| Élément | Contenu |
|---------|---------|
| Titre | "Étape 1 : Saisir votre numéro de téléphone" |
| Label | "Numéro de téléphone (format international)" |
| Input | 400x44px |
| Placeholder | "+33 6 12 34 56 78" |
| Bouton | "Envoyer code" (primary 130x44px) |

## 5. ÉTAPE 2 : CODE VÉRIFICATION
| Élément | Contenu |
|---------|---------|
| Titre | "Étape 2 : Saisir le code reçu par SMS" |
| Label | "Code à 6 chiffres" |
| Input | 200x44px |
| Placeholder | "• • • • • •" |

## 6. DESCRIPTION
| Élément | Contenu |
|---------|---------|
| Label | "Description (facultatif)" |
| Input | 552x44px |
| Placeholder | "Ex: Mon téléphone personnel" |

## 7. ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Secondary | Droite |
| Activer | Primary bleu | Extrême droite |

## 8. APIS & ENDPOINTS
- POST /me/auth/2fa/sms - Initier activation
- POST /me/auth/2fa/sms/validate - Valider code
