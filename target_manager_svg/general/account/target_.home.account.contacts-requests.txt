# Cahier des Charges : home/account - Demandes de changement de contact

## 1. VUE D'ENSEMBLE
- **Type** : Page liste demandes contacts
- **URL** : /account/contacts/requests
- **Objectif** : Gérer les demandes de changement de contact reçues et envoyées

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsText** : Titre page
- **OdsTabs** : Navigation (Mes services / Demandes de contact)
- **OdsCard** : Container principal
- **OdsSelect** : Filtre par état
- **OdsButton** : Refresh
- **OdsDatagrid** : Tableau demandes
- **OdsBadge** : États (En attente, Terminé, Refusé)
- **OdsActionMenu** : Actions par ligne

## 4. ONGLETS
| Onglet | État | Description |
|--------|------|-------------|
| Mes services | Inactif | Liste des services |
| Demandes de contact | Actif (bleu) | Demandes en cours |

## 5. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| État | Select dropdown | Tous les états, En attente, Terminé, Refusé |

## 6. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| ID / Date | String + Date | Numéro demande et date |
| De | String | NIC émetteur |
| Vers | String | NIC destinataire |
| Types | String | Type de contact (Admin, Tech, Billing) |
| Service | String | Nom du service |
| État | Badge | Statut de la demande |
| Actions | ActionMenu | Actions disponibles |

## 7. ÉTATS DEMANDES
| État | Couleur | Description |
|------|---------|-------------|
| En attente (vous) | Jaune #FEF3C7 | Requiert votre action |
| Terminé | Vert #D1FAE5 | Demande acceptée |
| Refusé | Rouge #FEE2E2 | Demande refusée |

## 8. ACTIONS MENU
| Action | Icône | Condition |
|--------|-------|-----------|
| Accepter | ✓ (vert) | Si en attente |
| Refuser | ✗ (rouge) | Si en attente |
| Renvoyer | ↻ (bleu) | Si en attente |

## 9. APIS & ENDPOINTS
- GET /me/contact/request - Liste demandes
- POST /me/contact/request/{id}/accept - Accepter
- POST /me/contact/request/{id}/refuse - Refuser
- POST /me/contact/request/{id}/resend - Renvoyer
