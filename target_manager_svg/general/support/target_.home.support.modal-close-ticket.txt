# Cahier des Charges : home/support - Modal Clore Ticket

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation
- **Déclencheur** : Bouton "Clore" dans détail ticket
- **Objectif** : Confirmer la clôture d'un ticket

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x350px)
- Header avec titre et bouton fermer
- Infos ticket
- Message confirmation
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Infos ticket
- **OdsMessage** : Confirmation (success vert)
- **OdsButton** : Annuler/Clore le ticket

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Fond | Gris #F3F4F6 |
| Titre | "Clore le ticket" |
| Bouton fermer | × (coin droit) |

## 5. TICKET CONCERNÉ
| Propriété | Valeur |
|-----------|--------|
| Label | "Ticket concerné" |
| Fond | Gris #F3F4F6 |
| Format | "TKT-2024-12456 - Problème de connexion SSH" |

## 6. MESSAGE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Success (vert) |

| Message | "En clôturant ce ticket, vous confirmez que votre demande a été traitée." |

## 8. FOOTER MODAL

|--------|-------|---------|

## 9. APIS & ENDPOINTS
