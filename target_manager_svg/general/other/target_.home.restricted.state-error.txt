# Cahier des Charges : home/restricted - État Erreur Détaillé

## 1. VUE D'ENSEMBLE
- **Page** : Accès restreint avec permissions manquantes
- **URL** : /restricted (avec détail erreurs)
- **Objectif** : Afficher les permissions IAM manquantes

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar avec infos sous-utilisateur
- Zone centrale avec erreur
- Encart permissions manquantes (détaillé)
- Cartes de guides tutoriels

## 3. COMPOSANTS UI
- **OdsEmptyState** : État d'erreur
- **OdsIcon** : Avertissement (rouge)
- **OdsMessage** : Encart permissions (error)
- **OdsCard** : Cartes tutoriels (3)
- **OdsBadge** : "Tutoriel"

## 4. MESSAGE ERREUR
| Élément | Valeur |
|---------|--------|
| Icône | ! (cercle rouge #EF4444) |
| Titre | "Accès restreint" |
| L1 | "L'accès à l'espace client OVHcloud nécessite 3 droits nécessaires sur votre compte" |

## 5. ENCART PERMISSIONS MANQUANTES
| Propriété | Valeur |
|-----------|--------|
| Fond | Rouge clair #FEE2E2 |
| Titre | "Permissions manquantes :" |

### Permissions requises
| Permission | Description |
|------------|-------------|
| account:apiovh:me/get | Lecture des informations du compte |
| account:apiovh:me/accessRestriction/*/get | Lecture des restrictions d'accès |
| iam:apiovh:me/identity/*/get | Lecture des identités IAM |

## 6. STYLE PERMISSION
| Propriété | Valeur |
|-----------|--------|
| Fond item | #FEF2F2 |
| Texte | Rouge #991B1B |
| Format | • {permission} - {description} |
