# Cahier des Charges : home/hub - Bannière KYC Inde

- **Type** : Bannière info réglementaire
- **Déclencheur** : Client Inde sans KYC
- **Objectif** : Informer obligation légale documents

## 2. STRUCTURE BANNIÈRE
- Icône info gauche
- Message explicatif

- Bouton fermer
