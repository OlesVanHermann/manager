# Cahier des Charges : home/procedures - RGPD

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire exercice droits RGPD
- **URL** : /procedures/gdpr
- **Objectif** : Exercer droits sur données personnelles

## 2. STRUCTURE PAGE
- Logo OVHcloud centré
- Carte blanche principale (800x1150px)
- Titre + introduction
- Formulaire complet
- Section upload documents
- Bouton soumission

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur principal
- **OdsInput** : Champs texte
- **OdsSelect** : Sélecteur type demande
- **OdsTextarea** : Description demande
- **OdsInput[type=file]** : Upload documents
- **OdsButton** : Submit
