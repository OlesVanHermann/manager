# Cahier des Charges : bare-metal/network-security - Onglet Trafic

## 1. VUE D'ENSEMBLE
- **Page** : Analyse du trafic réseau
- **Type** : Onglet avec graphiques et statistiques
- **Objectif** : Visualiser le trafic et les attaques filtrées

## 2. STRUCTURE PAGE
- Breadcrumb (Network Security > Trafic)
- Titre "Analyse du trafic"
- Onglets navigation
- Sélecteurs période et IP
- Cartes statistiques (4 KPIs)
- Graphique trafic réseau
- Répartition par protocole
- IP les plus ciblées

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsSelect** : Période, filtre IP
- **OdsCard** : Cartes statistiques
- **OdsChart** : Graphique trafic (line chart)
- **OdsProgressBar** : Barres répartition protocoles
- **OdsLink** : Liens vers IP

## 4. SÉLECTEURS FILTRES
| Sélecteur | Options |
|-----------|---------|
| Période | Dernières 24h, 7 jours, 30 jours |
| IP | Toutes les IP, IP spécifique |
