# Cahier des Charges : bare-metal/nutanix - Modal éteindre le noeud

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning avec saisie
- **Contexte** : Dashboard Nutanix
- **Objectif** : Éteindre un noeud du cluster Nutanix

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning avec impact
- **OdsLink** : Accès au portail Nutanix
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Éteindre

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Éteindre le noeud" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "L'extinction d'un noeud Nutanix peut avoir un impact sur votre cluster, vérifiez les pré-requis sur le portail de Nutanix." |
| Lien | "Accéder au portail →" |

## 6. CONFIRMATION TEXTE
| Élément | Description |
|---------|-------------|
| Question | "Voulez-vous vraiment éteindre votre noeud {nodeName} ?" |

## 7. CHAMP DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Saisissez \"POWEROFF\" pour confirmer" |
| Placeholder | "POWEROFF" |
| Validation | Doit correspondre exactement à "POWEROFF" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Éteindre | Warning (orange) | Éteint le noeud |

## 9. APIS & ENDPOINTS
- POST /nutanix/{serviceName}/node/{nodeId}/powerOff - Éteindre le noeud
