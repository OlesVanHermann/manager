# Cahier des Charges : bare-metal/nutanix - Ajouter noeuds

## 1. VUE D'ENSEMBLE
- **Page** : Commander des noeuds supplémentaires
- **Type** : Modal formulaire
- **Objectif** : Étendre le cluster avec des noeuds

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Section configuration actuelle
- Section informations prix
- Input nombre de noeuds
- Warning si limite
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsCard** : Configuration actuelle
- **OdsMessage** : Info prix (bleu)
- **OdsQuantity** : Nombre de noeuds
- **OdsMessage** : Warning limite (orange)
- **OdsButton** : Annuler, Commander
