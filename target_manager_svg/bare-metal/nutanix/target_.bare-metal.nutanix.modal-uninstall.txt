# Cahier des Charges : bare-metal/nutanix - Modal Désinstaller Nœud

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger avec saisie
- **Contexte** : Gestion cluster Nutanix
- **Objectif** : Désinstaller un nœud du cluster avec confirmation

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement danger
- **OdsInput** : Champ de confirmation
- **OdsLink** : Lien vers guide
- **OdsButton** : Annuler, Désinstaller

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Désinstaller le nœud" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Danger |
| Fond | #FEE2E2 |
| Border | #FECACA |
| Icône | ⚠ |
| Titre | "Action dangereuse" |
| Texte | "Le nœud {nodeId}.ip-x-x-x.eu sera désinstallé du cluster." |
| Lien | "Consulter le guide →" (bleu #0050D7) |
| Couleur texte | #DC2626 |

## 6. CHAMP DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Pour confirmer, saisissez \"UNINSTALL\"" |
| Type | OdsInput |
| Largeur | 450px |
| Placeholder | UNINSTALL |

### Note
| Propriété | Valeur |
|-----------|--------|
| Texte | "Cette action est irréversible. Le nœud sera retiré du cluster." |
| Style | 12px, gris #6B7280 |

## 7. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Désinstaller | Primary danger | #DC2626 | Désinstalle le nœud |

## 8. APIS & ENDPOINTS
- DELETE /nutanix/{serviceName}/node/{nodeId} - Désinstaller nœud
