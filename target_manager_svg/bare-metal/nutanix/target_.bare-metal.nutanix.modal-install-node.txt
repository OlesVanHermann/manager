# Cahier des Charges : bare-metal/nutanix - Modal installer nœud

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Nutanix Hyperconverged Infrastructure
- **Objectif** : Installer un nouveau nœud dans le cluster Nutanix

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsText** : Labels, description
- **OdsInput** : Champs IP
- **OdsSelect** : Version AOS
- **OdsButton** : Annuler, Installer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Installer un nœud" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Le nœud sera installé avec les paramètres suivants." |

## 6. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Bordure | #0050D7 |
| Texte | "Les IP doivent être dans le même sous-réseau que la gateway." |

## 7. FORMULAIRE

### Champ IP Hyperviseur AHV *
| Propriété | Valeur |
|-----------|--------|
| Label | "IP Hyperviseur AHV *" |

| Placeholder | "Ex: 192.168.1.10" |
| Requis | Oui |

### Champ IP CVM *
| Propriété | Valeur |
|-----------|--------|
| Label | "IP CVM *" |
| Type | OdsInput |

| Propriété | Valeur |
|-----------|--------|
| Label | "Version AOS *" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Installer | Primary (bleu) | Lance l'installation |

## 9. APIS & ENDPOINTS
- POST /nutanix/{serviceName}/node - Installer un nœud
