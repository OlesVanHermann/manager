# Cahier des Charges : bare-metal/dedicated - Housing Tab Tâches

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet avec tableau
- **Contexte** : Gestion housing
- **Objectif** : Afficher l'historique des tâches

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Tableau largeur | 940px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Liste des tâches
- **OdsBadge** : Statut tâche

## 4. ONGLETS
| Onglet | État |
|--------|------|
| Tableau de bord | Inactif |
| Backup FTP | Inactif |
| Tâches | Actif |

## 5. COLONNES DATAGRID
| Colonne | Largeur | Type |
|---------|---------|------|
| Dernière action | 185px | Date/heure |
| Fonction | 250px | Texte (code) |
| Commentaire | 300px | Texte |
| Statut | 200px | Badge |

## 6. STATUTS TÂCHES
| Statut | Style |
|--------|-------|
| done | Badge vert #DCFCE7 |
| doing | Badge bleu #EFF6FF |
| error | Badge rouge #FEE2E2 |

## 7. FONCTIONS EXEMPLE
| Fonction | Description |
|----------|-------------|
| hardReboot | Redémarrage électrique demandé |
| ftpBackupActivation | Activation du backup FTP |
| addBackupAccess | Ajout accès IP |
| passwordReset | Réinitialisation mot de passe FTP |

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "1-4 sur 4 résultats" |

## 9. APIS & ENDPOINTS
- GET /dedicated/housing/{serviceName}/task - Liste des tâches
- GET /dedicated/housing/{serviceName}/task/{taskId} - Détails tâche
