# Cahier des Charges : bare-metal/housing - Redémarrer

## 1. VUE D'ENSEMBLE
- **Page** : Redémarrer le serveur
- **Type** : Modal formulaire
- **Objectif** : Redémarrer le serveur housing

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Zone de saisie raison (optionnel)
- Message info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsTextarea** : Raison du redémarrage
- **OdsText** : Message info
- **OdsButton** : Annuler (outline), Redémarrer (primary)

## 4. CHAMP RAISON
| Propriété | Valeur |
|-----------|--------|
| Label | Raison du redémarrage (optionnel) ? |
| Type | Textarea |
| Placeholder | Saisissez la raison du redémarrage... |
| Requis | Non |

## 5. MESSAGE INFO
Pour accéder aux options de redémarrage, il est nécessaire de procéder à l'acquisition d'un module complémentaire.

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Redémarrer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/housing/{serviceName}/reboot
  - Paramètre optionnel : reason (string)
