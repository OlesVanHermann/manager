# Cahier des Charges : bare-metal/housing - Tâches

## 1. VUE D'ENSEMBLE
- **Page** : Housing - Tâches
- **Type** : Page onglet
- **Objectif** : Afficher l'historique des tâches du housing

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb + titre
- Onglets (Tâches actif)
- Titre section
- Tableau des tâches
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud › Housing
- **OdsTabs** : Tâches actif
- **OdsDatagrid** : Tableau des tâches
- **OdsBadge** : Terminée (vert), Initialisation (bleu), En erreur (rouge)
- **OdsPagination** : Navigation résultats

## 4. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Dernière action | DateTime | Format JJ/MM/AAAA HH:MM |
| Tâche | Text | Nom de l'opération |
| Détail technique | Text | Description technique |
| Statut | Badge | État de la tâche |

## 5. TYPES DE TÂCHES
- Redémarrage
- Modification du backup storage
- Création d'un backup storage
- Changement de mot de passe backup storage
- Déplacement d'une IP failover

## 6. STATUTS
| Statut | Badge | Couleur |
|--------|-------|---------|
| Terminée | success | Vert |
| Initialisation | info | Bleu |
| En cours | warning | Jaune |
| En erreur | error | Rouge |

## 7. APIS & ENDPOINTS
- GET /dedicated/housing/{serviceName}/task
  - Liste des tâches
- GET /dedicated/housing/{serviceName}/task/{taskId}
  - Détails d'une tâche
