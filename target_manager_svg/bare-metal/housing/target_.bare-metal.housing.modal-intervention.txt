# Cahier des Charges : bare-metal/housing - Modal demande d'intervention

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Colocation (Housing)
- **Objectif** : Créer une demande d'intervention datacenter sur une baie

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, informations
- **OdsSelect** : Type d'intervention
- **OdsInput** : Équipement concerné
- **OdsTextarea** : Description détaillée
- **OdsButton** : Groupe niveau urgence, Annuler, Envoyer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Demande d'intervention" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS CONTEXTE
| Champ | Valeur |
|-------|--------|
| Label | "Baie" |
| Format | "{datacenter}-{hall}-{rack}-{bay}" |
| Exemple | "RBX-H1-R42-B08" |

## 6. FORMULAIRE

### Champ Type d'intervention *
| Propriété | Valeur |
|-----------|--------|
| Label | "Type d'intervention *" |
| Type | OdsSelect |
| Options | Redémarrage d'un équipement, Remplacement de composant, Câblage, Vérification physique, Autre |
| Requis | Oui |

### Champ Équipement concerné *
| Propriété | Valeur |
|-----------|--------|
| Label | "Équipement concerné *" |
| Type | OdsInput |
| Placeholder | "Ex: Serveur U12-U14, Switch principal..." |
| Requis | Oui |

### Champ Description détaillée *
| Propriété | Valeur |
|-----------|--------|
| Label | "Description détaillée *" |
| Type | OdsTextarea |
| Placeholder | "Décrivez précisément l'intervention souhaitée..." |
| Hauteur | 100px |
| Requis | Oui |

### Champ Niveau d'urgence
| Propriété | Valeur |
|-----------|--------|
| Label | "Niveau d'urgence" |
| Type | Button group |

#### Options
| Niveau | Style | Description |
|--------|-------|-------------|
| Normal | Selected (bleu) | Traitement sous 24-48h |
| Urgent | Default | Traitement prioritaire |
| Critique | Default | Traitement immédiat (surcoût) |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Envoyer la demande | Primary (bleu) | Soumet la demande |

## 8. APIS & ENDPOINTS
- POST /dedicated/housing/{serviceName}/task - Créer intervention
- GET /dedicated/housing/{serviceName}/task - Liste des interventions
