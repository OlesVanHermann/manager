# Cahier des Charges : bare-metal/housing - Mot de passe backup

## 1. VUE D'ENSEMBLE
- **Page** : Demande de nouveau mot de passe
- **Type** : Modal confirmation
- **Objectif** : Réinitialiser le mot de passe du backup storage

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Icône clé + question
- Message info email
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsText** : Question (avec nom backup)
- **OdsMessage** : Info email (bleu)
- **OdsButton** : Annuler (outline), Réinitialiser (primary)
