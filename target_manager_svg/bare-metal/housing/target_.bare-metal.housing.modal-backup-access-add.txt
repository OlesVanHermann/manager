# Cahier des Charges : bare-metal/dedicated - Modal Ajouter Accès Backup

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard 2 étapes
- **Contexte** : Gestion backup FTP housing
- **Objectif** : Ajouter un accès FTP pour une IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : Étapes 1-2
- **OdsCheckbox** : Sélection IP et protocoles
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un accès FTP backup" |
| Fond | #F9FAFB |

## 5. STEPPER
| Étape | Libellé | État |
|-------|---------|------|
| 1 | Configuration | Actif (bleu) |
| 2 | Confirmation | Inactif (gris) |

## 6. CONTENU ÉTAPE 1

### Sélection IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresses IP" |
| Type | Liste checkboxes |
| Options | IPs disponibles du housing |

### Protocoles
| Propriété | Valeur |
|-----------|--------|
| Label | "Protocoles autorisés" |
| Options | CIFS, FTP, NFS (checkboxes) |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary #0050D7 | Passe à étape 2 |

## 8. APIS & ENDPOINTS
- GET /dedicated/housing/{serviceName}/features/backupFTP/authorizableBlocks - IPs disponibles
- POST /dedicated/housing/{serviceName}/features/backupFTP/access - Créer accès
