# Cahier des Charges : bare-metal/license - Migrer licence

## 1. VUE D'ENSEMBLE
- **Page** : Migrer ma licence
- **Type** : Formulaire étapes
- **Objectif** : Migrer une licence vers une nouvelle IP

## 2. STRUCTURE PAGE
- Titre + lien retour
- Étape 1 : Sélection IP
- Étape 2 : Confirmation
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour à la gestion des licenses
- **OdsSelect** : Ip Block, Ip de destination
- **OdsMessage** : Info (bleu)
- **OdsText** : Récapitulatif
- **OdsButton** : Annuler (outline), Migrer la licence (primary)

## 4. ÉTAPE 1 : SÉLECTION IP
### Ip Block
| Propriété | Valeur |
|-----------|--------|
| Label | Ip Block |
| Type | Select |
| Exemple | 51.210.0.0/24 |

### Ip de destination
| Propriété | Valeur |
|-----------|--------|
| Label | Ip de destination |
| Type | Select |
| Exemple | 51.210.xxx.yyy |

### Info Banner
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte | La licence sera migrée vers la nouvelle IP. Cette action est irréversible. |

## 5. ÉTAPE 2 : CONFIRMATION
| Champ | Valeur exemple |
|-------|----------------|
| Ip actuelle | 51.210.xxx.xxx |
| Nouvelle Ip | 51.210.xxx.yyy |
| Licence | Plesk - Edition Web Pro (PLESK-12345) |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Migrer la licence | Primary | Droite |

## 7. APIS & ENDPOINTS
- GET /ip
  - Liste des blocs IP
- GET /ip/{ipBlock}
  - IPs du bloc
- POST /license/{type}/{serviceName}/changeIp
  - Paramètre : destinationIp (string)
