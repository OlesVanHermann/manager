# Cahier des Charges : bare-metal/license - Wizard ajouter licence SPLA

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (3 étapes)
- **Contexte** : Gestion des licences Microsoft SPLA
- **Objectif** : Commander une licence SPLA pour un serveur dédié

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étapes
- **OdsText** : Labels, descriptions
- **OdsSelect** : Sélection serveur
- **OdsButton** : Annuler, Suivant

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une licence SPLA" |

## 5. STEPPER

### Étapes
| Étape | Label | État |
|-------|-------|------|
| 1 | Serveur | Active (bleu) |
| 2 | Type | Inactive (gris) |
| 3 | Confirmation | Inactive (gris) |

## 6. FORMULAIRE - ÉTAPE 1 (SERVEUR)

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sélectionnez le serveur pour la licence SPLA" |

### Champ Serveur
| Propriété | Valeur |
|-----------|--------|
| Label | "Serveur" |
| Type | OdsSelect avec liste déroulante |
| Placeholder | "Sélectionner un serveur..." |

### Liste des serveurs
| Propriété | Valeur |
|-----------|--------|
| Format | "{serverName}" |
| Exemples | "ns123456.ip-xxx-xxx-xxx.eu" |
| Sélection | Highlight bleu sur l'élément sélectionné |

## 7. ÉTAPE 2 (TYPE) - Non visible
| Champ | Description |
|-------|-------------|
| Type de licence | Windows Server, SQL Server, etc. |
| Nombre | Quantité de licences |

## 8. ÉTAPE 3 (CONFIRMATION) - Non visible
| Élément | Description |
|---------|-------------|
| Récapitulatif | Serveur, type, prix |
| Action | Commander |

## 9. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Suivant | Primary (bleu) | Passe à l'étape suivante |

## 10. APIS & ENDPOINTS
- GET /dedicated/server - Liste des serveurs éligibles
- GET /license/spla - Types de licences disponibles
- POST /order/cart/{cartId}/license - Commander licence
