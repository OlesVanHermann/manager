# Cahier des Charges : bare-metal/license - Révoquer licence SPLA

## 1. VUE D'ENSEMBLE
- **Page** : Révoquer une licence SPLA
- **Type** : Page confirmation
- **Objectif** : Supprimer une licence SPLA d'un serveur

## 2. STRUCTURE PAGE
- Titre + lien retour
- Banner warning
- Récapitulatif SPLA
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour à la gestion des licenses
- **OdsMessage** : Warning (jaune)
- **OdsCard** : Récapitulatif licence
- **OdsButton** : Annuler (outline), Révoquer la licence SPLA (danger)

## 4. WARNING BANNER
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Icône | ⚠️ |
| Texte | Attention : Cette action est définitive. La licence SPLA sera supprimée. |

## 5. RÉCAPITULATIF LICENCE
| Champ | Valeur exemple |
|-------|----------------|
| Serveur | ns1234567.ip-51-210-xxx.eu |
| Type | Windows Server OS |
| Numéro de série | XXXXX-XXXXX-XXXXX-XXXXX-XXXXX |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Révoquer la licence SPLA | Danger (rouge) | Droite |

## 7. APIS & ENDPOINTS
- DELETE /license/spla/{id}
  - Révoquer la licence SPLA
