# Cahier des Charges : bare-metal/dedicated-cloud - Modal Upgrade Offre Backup

## 1. VUE D'ENSEMBLE
- **Type** : Modal de changement d'offre
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Changer l'offre de backup vers une formule supérieure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsCheckbox** : Acceptation conditions
- **OdsButton** : Annuler, Changer l'offre
- **OdsText** : Descriptions

| Propriété | Valeur |

| Titre | "Changer l'offre de backup" |

- GET /dedicatedCloud/{serviceName}/allowedBackupStorageCapacities - Capacités disponibles
