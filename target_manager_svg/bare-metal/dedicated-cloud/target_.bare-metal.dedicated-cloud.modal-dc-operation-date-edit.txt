# Cahier des Charges : bare-metal/dedicated-cloud - Modal Modifier Date Opération

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec calendrier
- **Contexte** : Gestion opérations Dedicated Cloud (VMware)
- **Objectif** : Modifier la date et l'heure d'exécution d'une opération planifiée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsDatepicker** : Calendrier
- **OdsTimepicker** : Sélecteur d'heure
- **OdsButton** : Annuler, Modifier
- **OdsText** : Labels

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la date d'exécution" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Calendrier
| Propriété | Valeur |
|-----------|--------|
| Largeur | 280px |
| Hauteur | 240px |
| Navigation | ◀ Mois ▶ |
| Format mois | "Janvier 2025" |

### En-têtes jours
| Lu | Ma | Me | Je | Ve | Sa | Di |
|----|----|----|----|----|----|----|

### Style jours
| État | Apparence |
|------|-----------|
| Jour sélectionné | Cercle bleu #0050D7, texte blanc |
| Jours passés | Gris clair (#6B7280) |
| Jours disponibles | Noir (#1F2937) |

### Sélection heure
| Propriété | Valeur |
|-----------|--------|
| Label | "Heure :" |
| Format | HH:MM (24h) |
| Champs séparés | Heures (60px) : Minutes (60px) |
| Séparateur | ":" |
| Exemple | 03:00 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Modifier | Primary (#0050D7) | Enregistre la nouvelle date |

## 7. APIS & ENDPOINTS
- PUT /dedicatedCloud/{serviceName}/task/{taskId} - Modifier la date d'exécution
