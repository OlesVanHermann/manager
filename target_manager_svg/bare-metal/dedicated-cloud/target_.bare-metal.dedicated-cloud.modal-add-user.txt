# Cahier des Charges : bare-metal/dedicated-cloud - Modal création utilisateur

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un utilisateur
- **Type** : Formulaire création
- **Objectif** : Créer un utilisateur vSphere

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Formulaire 4 champs
- Section permissions (checkboxes)
- Message info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x620px)
- **OdsInput** : Login, Nom, Email, Téléphone
- **OdsCheckbox** : 4 permissions
- **OdsMessage** : Info (bleu)
- **OdsButton** : Annuler (outline), Créer (primary)

## 4. CHAMPS FORMULAIRE

### Login
| Propriété | Valeur |
|-----------|--------|
| Label | Login * |
| Placeholder | - |
| Valeur exemple | dev-user |
| Requis | Oui |

### Nom complet
| Propriété | Valeur |
|-----------|--------|
| Label | Nom complet * |
| Placeholder | Nom affiché dans vSphere |
| Requis | Oui |

### Email
| Propriété | Valeur |
|-----------|--------|
| Label | Email * |
| Placeholder | email@example.com |
| Requis | Oui |

### Téléphone
| Propriété | Valeur |
|-----------|--------|
| Label | Téléphone |
| Placeholder | +33 6 XX XX XX XX |
| Requis | Non |

## 5. PERMISSIONS
| Permission | Description | Défaut |
|------------|-------------|--------|
| Accès NSX | Gestion du réseau virtuel | ✅ Coché |
| Accès IP | Gestion des blocs IP | ✅ Coché |
| Accès VM Network | Réseau des VMs | ⬜ Non coché |
| Validation par token | Requis pour la sécurité | ✅ Coché |

## 6. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | ℹ Un email avec les identifiants sera envoyé à l'utilisateur. |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/user
  - login: string
  - name: string
  - email: string
  - phone: string (optionnel)
  - nsxRight: boolean
  - ipRight: boolean
  - vmNetworkRight: boolean
  - tokenValidator: boolean
