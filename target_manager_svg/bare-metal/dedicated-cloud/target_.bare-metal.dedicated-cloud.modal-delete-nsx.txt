# Cahier des Charges : bare-metal/dedicated-cloud - Modal supprimer NSX

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation de suppression
- **Contexte** : VMware Dedicated Cloud (Private Cloud)
- **Objectif** : Confirmer la suppression de la configuration NSX d'un datacenter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer NSX" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voulez-vous vraiment supprimer la configuration NSX du datacenter {dcName} ?" |
| Format dcName | Gras |
| Exemple | "dc-rbx-1" |

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Icône | ⚠️ |
| Message | "Cette action supprimera toutes les règles de firewall distribuées configurées." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime NSX |

## 7. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/nsx - Supprimer NSX
