# Cahier des Charges : bare-metal/dedicated-cloud - Modal Mise à Jour VMware

## 1. VUE D'ENSEMBLE
- **Type** : Modal de mise à jour
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Mettre à jour la version VMware de l'infrastructure

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Info
- **OdsButton** : Annuler, Mettre à jour
- **OdsText** : Informations de version

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mettre à jour la version VMware" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Informations de version
| Label | Valeur | Style |
|-------|--------|-------|
| Version actuelle | vSphere 7.0 Update 3 | Noir (#1F2937), gras |
| Version cible | vSphere 8.0 Update 1 | Bleu (#0050D7), gras |

### Description
| Ligne | Contenu |
|-------|---------|
| 1 | "La mise à jour de votre infrastructure vers une nouvelle version" |
| 2 | "de VMware apporte de nouvelles fonctionnalités et améliorations." |

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |
| Icône | ℹ |
| Ligne 1 | "La mise à jour sera effectuée datacenter par datacenter." |
| Ligne 2 | "Vos services resteront disponibles pendant l'opération." |
| Ligne 3 | "Durée estimée : 2-4 heures par datacenter." |
| Couleur texte | #1E40AF |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Mettre à jour | Primary (#0050D7) | Lance la mise à jour |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/upgradeVersion - Versions disponibles
- POST /dedicatedCloud/{serviceName}/upgradeVersion - Lancer la mise à jour
