# Cahier des Charges : bare-metal/dedicated-cloud - Datastores

## 1. VUE D'ENSEMBLE
- **Page** : Datastores datacenter
- **Type** : Page gestion datastores
- **Objectif** : Gérer les datastores d'un datacenter VMware

## 2. STRUCTURE PAGE
- Breadcrumb complet
- Titre (dc-X - Datastores)
- Onglets navigation datacenter
- Titre section + bouton Commander
- Tableau datastores
- Résumé utilisation

## 3. COMPOSANTS UI
- **OdsTabs** : 3 onglets
- **OdsButton** : Commander un datastore (primary)
- **OdsDatagrid** : Tableau
- **OdsBadge** : Type (SSD NVMe, NFS, vSAN), Statut (Monté)
- **OdsProgressBar** : Utilisation (vert, jaune, rouge)
- **OdsActionMenu** : Menu ⋮

## 11. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer
  - Liste datastores
- POST /order/dedicatedCloud/{serviceName}/filer
  - Commander datastore
- PUT /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/{filerId}
  - Modifier datastore
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/filer/{filerId}
  - Supprimer datastore
