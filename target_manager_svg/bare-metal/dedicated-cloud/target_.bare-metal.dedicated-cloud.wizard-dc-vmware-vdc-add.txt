# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Ajouter Datacenter VDC

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (2 étapes)
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Ajouter un nouveau datacenter virtuel (VDC) au service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsStepper** : 2 étapes
- **OdsRadio** : Sélection type VDC (cartes)
- **OdsMessage** : Info NSX
- **OdsButton** : Annuler, Suivant
- **OdsText** : Descriptions

| Propriété | Valeur |

| Titre | "Ajouter un datacenter" |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Suivant | Droite | Primary (#0050D7) | Passe à l'étape 2 |

## 7. ÉTAPE 2 (aperçu) - Gamme de hosts
- Sélection de la gamme de hosts (Entry, Standard, Premium)
- Configuration du nombre de hosts initial
- Choix du storage

## 8. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/allowedDatacenters - Types VDC disponibles
- POST /dedicatedCloud/{serviceName}/datacenter - Créer le datacenter
- GET /order/dedicatedCloud/{serviceName}/datacenter - Options de commande
