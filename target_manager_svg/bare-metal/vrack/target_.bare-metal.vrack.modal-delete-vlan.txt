# Cahier des Charges : bare-metal/vrack - Modal supprimer VLAN

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : vRack (réseau privé)
- **Objectif** : Confirmer la suppression d'un VLAN du vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le VLAN" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voulez-vous vraiment supprimer le VLAN" |
| Détail | "{VLAN_ID} ({nom})" - en gras |
| Exemple | "VLAN 100 (Production) ?" |

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Texte | "Cette action est irréversible." |
| Couleur | Rouge (#DC2626) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime le VLAN |

## 7. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/vlanId/{vlanId} - Supprimer un VLAN
