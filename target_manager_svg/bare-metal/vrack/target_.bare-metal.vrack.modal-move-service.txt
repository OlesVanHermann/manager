# Cahier des Charges : bare-metal/vrack - Modal déplacer service

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : vRack (réseau privé)
- **Objectif** : Déplacer un service vers un autre vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description, labels
- **OdsMessage** : Avertissement interruption
- **OdsSelect** : Sélection vRack destination
- **OdsButton** : Annuler, Déplacer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Déplacer vers un autre vRack" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Déplacer ce service vers un autre vRack." |

## 6. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Icône | ⚠️ |
| Texte | "Attention : cette opération peut entraîner une interruption de service." |
| Couleur texte | #92400E |

## 7. FORMULAIRE

### Champ vRack destination
| Propriété | Valeur |
|-----------|--------|
| Label | "Choisir un vRack de destination" |
| Type | OdsSelect |
| Placeholder | "Sélectionner un vRack..." |
| Options | Liste des vRacks disponibles |
| Requis | Oui |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Déplacer | Primary (bleu #0050D7) | Lance le déplacement |

## 9. APIS & ENDPOINTS
- GET /vrack - Liste des vRacks disponibles
- POST /vrack/{serviceName}/move - Déplacer le service vers un autre vRack
