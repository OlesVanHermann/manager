# Cahier des Charges : bare-metal/vrack - Modal Sélectionner vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Association service au vRack
- **Objectif** : Confirmer l'association d'un service IPv6 à un vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Confirmation" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Vous allez associer ce service IPv6 au vRack sélectionné." |
| Ligne 2 | "Cette action va créer les ressources nécessaires dans votre réseau privé." |
| Style | 13px, #1F2937 |

## 6. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Confirmer | Primary warning | #F59E0B | Associe le service |

## 7. APIS & ENDPOINTS
- POST /vrack/{serviceName}/ipv6/{ipBlock} - Associer IPv6 au vRack
