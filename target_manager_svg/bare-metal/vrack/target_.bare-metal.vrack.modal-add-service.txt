# Cahier des Charges : bare-metal/vrack - Ajouter service

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un service au vRack
- **Type** : Modal formulaire
- **Objectif** : Connecter des services au réseau privé

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Sélection type de service
- Liste services disponibles avec checkboxes
- Attribution VLANs optionnelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsSelect** : Type de service
- **OdsInput** : Recherche
- **OdsCheckbox** : Sélection services
- **OdsSelect** : Attribution VLANs
- **OdsButton** : Annuler, Ajouter

## 4. CHAMPS FORMULAIRE
| Champ | Type | Description |
|-------|------|-------------|
| Type de service | Select | Serveur dédié, VPS, HPC, etc. |
| Recherche | Input | Filtrer services |
| Services | Checkbox list | Liste multi-sélection |
| VLANs | Select multi | Attribution optionnelle |

## 5. TYPES DE SERVICES
| Type | Description |
|------|-------------|
| Serveur dédié | Dedicated servers |
| VPS | Virtual Private Server |
| Hosted Private Cloud | VMware, Nutanix |
| Public Cloud | Instances OpenStack |
| IP Block | Blocs IP |
| NAS-HA | Stockage HA |

## 6. LISTE SERVICES
| Colonne | Description |
|---------|-------------|
| Checkbox | Sélection |
| Nom | Identifiant service |
| Datacenter | Localisation |

Compteur: "X services sélectionnés sur Y disponibles"

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 8. APIS & ENDPOINTS
- GET /vrack/{serviceName}/allowedServices
  - Liste services connectables
- POST /vrack/{serviceName}/dedicatedServer
  - Ajouter serveur dédié
