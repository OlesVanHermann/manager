# Cahier des Charges : bare-metal/vrack - Onglet VLAN

## 1. VUE D'ENSEMBLE
- **Page** : Liste des VLANs du vRack
- **Type** : Onglet tableau
- **Objectif** : Gérer la segmentation réseau

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "VLAN"
- Onglets navigation
- Bouton création
- Bandeau info VLANs
- Tableau VLANs
- Résumé utilisation

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Créer un VLAN
- **OdsMessage** : Info VLANs
- **OdsDatagrid** : Tableau VLANs
- **OdsBadge** : État (Actif)
- **OdsActionMenu** : Menu par ligne

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Titre | À propos des VLANs |
| Texte | Les VLANs permettent de segmenter votre réseau privé. Chaque VLAN peut être attribué à plusieurs services. |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| VLAN ID | Text (bold) | Identifiant numérique |
| Nom | Link | Nom du VLAN |
| Sous-réseau | Monospace | CIDR notation |
| Services | Text | Nombre de services |
| État | Badge | Actif (vert) |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Modifier | Normal | Édite le VLAN |
| Voir les services | Normal | Filtre services |
| Dupliquer | Normal | Clone le VLAN |
| Supprimer | Danger | Supprime le VLAN |
