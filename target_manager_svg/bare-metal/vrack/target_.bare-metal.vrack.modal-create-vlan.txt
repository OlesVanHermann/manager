# Cahier des Charges : bare-metal/vrack - Créer VLAN (complet)

## 1. VUE D'ENSEMBLE
- **Page** : Création d'un VLAN (formulaire complet)
- **Type** : Modal formulaire
- **Objectif** : Créer un segment réseau

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Formulaire complet
- Message info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : VLAN ID, nom, sous-réseau, description
- **OdsSelect** : Masque CIDR
- **OdsTextarea** : Description
- **OdsMessage** : Info
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| VLAN ID | Input number | Oui | 1-4095 |
| Nom | Input | Oui | Nom du VLAN |
| Sous-réseau | Input | Non | Adresse IP réseau |
| Masque | Select | Non | /8 à /30 |
| Description | Textarea | Non | Description optionnelle |

## 5. VALIDATION VLAN ID
| Règle | Message |
|-------|---------|
| Min | 1 |
| Max | 4095 |
| Unique | Doit être unique dans le vRack |

## 6. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ |
| Texte | Le VLAN sera créé immédiatement et pourra être attribué aux services du vRack. |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /vrack/{serviceName}/nasha/{nasha}/virtualNetworkInterface
  - Créer interface VLAN
