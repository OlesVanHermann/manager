# Cahier des Charges : bare-metal/dedicated-cloud - Onglet Zerto DRP

## 1. VUE D'ENSEMBLE
- **Type** : Onglet de configuration Plan de reprise d'activité
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Configurer la réplication Zerto pour le disaster recovery

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsRadio** : Sélection type de plan (cartes)
- **OdsMessage** : Warning hosts manquants
- **OdsButton** : Activer Zerto DRP
- **OdsText** : Descriptions

- GET /dedicatedCloud/{serviceName}/zerto - Statut Zerto
