# Cahier des Charges : bare-metal/ip - Modal débloquer IP (Antispam)

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Protection Antispam
- **Objectif** : Débloquer une adresse IP bloquée pour spam

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message, IP
- **OdsMessage** : Information délai
- **OdsButton** : Annuler, Débloquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Débloquer l'IP" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message principal
| Propriété | Valeur |
|-----------|--------|
| Texte | "L'adresse IP suivante est actuellement bloquée :" |

### IP affichée
| Propriété | Valeur |
|-----------|--------|
| Format | IP en rouge (#DC2626), gras |
| Exemple | "51.210.xxx.100" |

## 6. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Ligne 1 | "Raison : Activité de spam détectée." |
| Ligne 2 | "Débloquer l'IP peut prendre jusqu'à 30 minutes." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Débloquer | Warning (orange #F59E0B) | Débloque l'IP |

## 8. APIS & ENDPOINTS
- GET /ip/{ip}/antispam - Statut antispam
- POST /ip/{ip}/antispam/unblock - Débloquer l'IP
