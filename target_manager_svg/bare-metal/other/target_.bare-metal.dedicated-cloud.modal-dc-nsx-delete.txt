# Cahier des Charges : bare-metal/dedicated-cloud - Modal Supprimer NSX Edge

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion réseau Dedicated Cloud (VMware)
- **Objectif** : Supprimer un NSX Edge

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Supprimer
- **OdsText** : Description

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer NSX Edge" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠ |

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|

| Bouton | Style | Action |
