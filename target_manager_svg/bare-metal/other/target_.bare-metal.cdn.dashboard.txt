# Cahier des Charges : bare-metal/cdn - Tableau de bord

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard principal
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Vue synthétique des statistiques et raccourcis du service CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 750px |

## 3. COMPOSANTS UI
- **OdsCard** : Tuiles raccourcis, statistiques, informations
- **OdsButton** : Liens raccourcis
- **OdsSelect** : Filtres période et type de données
- **ActionMenu** : Menu contextuel (⋮)
- **OdsChart** : Graphique statistiques (barres)

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "CDN Dédié - {cdnId}" |

## 5. TUILES INFORMATIONS (3 colonnes)

### Tuile Raccourcis
| Propriété | Valeur |
|-----------|--------|
| Largeur | 330px |
| Hauteur | 200px |
| Titre | "Raccourcis" |

| Raccourci | Action |
|-----------|--------|
| + Ajouter un domaine | Ouvre wizard ajout domaine |
| + Commander des règles | Ouvre wizard commande règles |
| + Commander du quota | Ouvre wizard commande quota |

### Tuile Statistiques
| Propriété | Valeur |
|-----------|--------|
| Largeur | 330px |
| Hauteur | 200px |
| Titre | "Statistiques" |

| Métrique | Exemple |
|----------|---------|
| Activité | 1 250 pages vues |
| Économies bande passante | 85% |
| Économies requêtes | 92% |
| Quota restant | 450 Go |

### Tuile Informations
| Propriété | Valeur |
|-----------|--------|
| Largeur | 330px |
| Hauteur | 200px |
| Titre | "Informations" |

| Information | Exemple |
|-------------|---------|
| Domaines | 3 domaines configurés |
| IP Anycast | 213.186.33.5 |
| Backends | 2 backends utilisés |

### ActionMenu tuile Informations
| Option | Action |
|--------|--------|
| Ajouter un domaine | Ouvre wizard ajout domaine |

## 6. SECTION GRAPHIQUE STATISTIQUES
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1040px |
| Hauteur | 350px |
| Type graphique | Barres verticales |

### Filtres
| Filtre | Options |
|--------|---------|
| Visualiser | Bande passante, Requêtes, Pages vues |
| Depuis | Dernier jour, Dernière semaine, Dernier mois |

### Axes
| Axe | Contenu |
|-----|---------|
| Y | Valeurs (Go, Mo, requêtes) |
| X | Périodes (Sem 1, Sem 2, Sem 3, Sem 4) |

## 7. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName} - Détails service CDN
- GET /cdn/dedicated/{serviceName}/domains - Liste domaines
- GET /cdn/dedicated/{serviceName}/statistics - Statistiques globales
- GET /cdn/dedicated/{serviceName}/quota - Quota restant
