# Cahier des Charges : bare-metal/load-balancer - Modal détails frontend

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative (lecture seule)
- **Contexte** : Load Balancer Dedicated
- **Objectif** : Afficher les détails complets d'un frontend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 460px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsBadge** : Statut

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Détails du frontend" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

- Valeurs texte : Noir (#1F2937)
- Labels : Gris (#6B7280)

- Badge actif : Fond vert clair (#DCFCE7), texte vert (#166534)

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Détails frontend
