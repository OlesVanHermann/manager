# Cahier des Charges : bare-metal/vrack - Créer VLAN (simple)

## 1. VUE D'ENSEMBLE
- **Page** : Création d'un VLAN (formulaire simplifié)
- **Type** : Modal formulaire
- **Objectif** : Créer un segment réseau

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Formulaire simplifié
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : VLAN ID, nom, passerelle
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| ID du VLAN | Input number | Oui | 1-4094 |
| Nom du VLAN | Input | Non | Nom descriptif |
| Passerelle | Input | Non | IP passerelle (optionnel) |

## 5. VALIDATION VLAN ID
| Propriété | Valeur |
|-----------|--------|
| Min | 1 |
| Max | 4094 |
| Helper text | (1-4094) |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /vrack/{serviceName}/nasha/{nasha}/virtualNetworkInterface
  - Paramètres : vlanId, name, gateway (optionnel)
