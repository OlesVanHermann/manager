# Cahier des Charges : bare-metal/dedicated-cloud - Modal Activer Licence SPLA

## 1. VUE D'ENSEMBLE
- **Type** : Modal de commande avec sélection
- **Contexte** : Gestion des licences Dedicated Cloud
- **Objectif** : Commander une licence SPLA (Windows, SQL Server, etc.)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 440px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsDatagrid** : Liste des licences disponibles
- **OdsRadio** : Sélection de licence
- **OdsCheckbox** : Acceptation CGV
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer une licence SPLA" |
| Fond | Gris clair (#F9FAFB) |
| Couleur texte | Gris foncé (#1F2937) |
| Bouton fermer | × |

## 5. CONTENU

### Tableau des licences
| Colonne | Description |
|---------|-------------|
| Type de licence | Nom du produit (Windows, SQL, etc.) |
| Prix | Montant HT/mois + unité de facturation |

### Licences disponibles
| Licence | Prix | Unité |
|---------|------|-------|
| Windows Server Standard 2022 | 15,00 € HT/mois | Par VM |
| Windows Server Datacenter 2022 | 45,00 € HT/mois | Par VM |
| SQL Server Standard 2022 | 200,00 € HT/mois | Par cœur |
| ... | ... | ... |

### Style sélection
| État | Style |
|------|-------|
| Sélectionné | Fond bleu clair (#EFF6FF), radio rempli (#0050D7) |
| Non sélectionné | Fond blanc, radio vide |

### Conditions générales
| Propriété | Valeur |
|-----------|--------|
| Type | Checkbox |

| Obligatoire | Oui |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Commander | Primary (bleu #0050D7) | Lance la commande |

## 7. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/license - Licences disponibles
- POST /order/cart/{cartId}/privateCloud - Commander licence SPLA
