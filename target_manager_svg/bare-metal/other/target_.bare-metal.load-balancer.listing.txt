# Cahier des Charges : bare-metal/load-balancer - Liste des Load Balancers

## 1. VUE D'ENSEMBLE
- **Type** : Page listing
- **Contexte** : Gestion Load Balancer IP
- **Objectif** : Afficher la liste de tous les load balancers du compte

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste des load balancers
- **OdsButton** : Commander
- **OdsLink** : Guides
- **OdsBadge** : Zones, État
- **OdsActionMenu** : Actions par ligne

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Load Balancer IP" |
| Bouton Guides | Dropdown avec liens documentation |

## 5. CONTENU

### Bouton Commander
| Propriété | Valeur |
|-----------|--------|
| Label | "+ Commander" |
| Style | Secondary (bordure grise) |
| Action | Ouvre page de commande |

### Datagrid
| Colonne | Type | Largeur | Triable |
|---------|------|---------|---------|
| Service | Lien bleu | 170px | Oui |
| Nom | Texte | 200px | Oui |
| IP | Texte | 150px | Non |
| Zones | Badges multiples | 150px | Non |
| État | Badge | 100px | Oui |
| Actions | ActionMenu | 80px | Non |

### Badges Zones
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Couleur texte | #1E40AF |
| Border radius | 11px |
| Exemples | gra, rbx, sbg, waw |

### Badges État
| État | Fond | Couleur texte |
|------|------|---------------|
| ok | #DCFCE7 | #16A34A |
| deploying | #FEF3C7 | #92400E |
| error | #FEE2E2 | #DC2626 |

### Exemple données
| Service | Nom | IP | Zones | État |
|---------|-----|------|-------|------|
| loadbalancer-abc123 | lb-production | 1.2.3.4 | gra, rbx | ok |
| loadbalancer-def456 | lb-staging | 5.6.7.8 | sbg | ok |
| loadbalancer-ghi789 | lb-dev | 9.10.11.12 | waw | deploying |

### ActionMenu options
| Action | Couleur |
|--------|---------|
| Voir les détails | Noir |
| Résilier | Rouge (#DC2626) |

### Pagination
| Propriété | Valeur |
|-----------|--------|
| Texte | "{n} load balancers" |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing - Liste des load balancers
- GET /ipLoadbalancing/{serviceName} - Détails d'un load balancer
