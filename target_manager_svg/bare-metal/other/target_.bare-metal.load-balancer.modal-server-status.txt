# Cahier des Charges : bare-metal/load-balancer - Modal Statut Serveur

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information (lecture seule)
- **Contexte** : Gestion Load Balancer
- **Objectif** : Afficher le statut de santé d'un serveur sur chaque instance/zone

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 370px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsBadge** : Statut UP/DOWN
- **OdsText** : Informations de check

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Statut du serveur" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Structure par instance
Pour chaque zone/instance du load balancer :

| Section | Contenu |
|---------|---------|
| Titre | "Instance {zone}-{id}" (ex: rbx-1) |
| Container | Fond #F9FAFB, border radius 4px |

### Informations par instance
| Label | Valeur exemple |
|-------|----------------|
| Statut | Badge UP ou DOWN |
| Code check | 200, 503, etc. |
| Statut check | L7OK, L7STS, etc. |
| Dernier check | OK, Service unavailable, etc. |
| Heure check | 29/12 15:45 |

### Badges Statut
| État | Fond | Couleur texte |
|------|------|---------------|
| UP | #DCFCE7 | #16A34A |
| DOWN | #FEE2E2 | #DC2626 |

### Format affichage
| Propriété | Valeur |
|-----------|--------|
| Titre instance | 14px, font-weight 600 |
| Label | 11px, gris #6B7280 |
| Valeur | 11px, noir #1F2937 |
| Container hauteur | 130px |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/farm/{farmId}/server/{serverId}/status - Statut serveur
