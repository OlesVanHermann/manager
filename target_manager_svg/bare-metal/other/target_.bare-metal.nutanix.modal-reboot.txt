# Cahier des Charges : bare-metal/nutanix - Modal redémarrer le serveur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Dashboard Nutanix
- **Objectif** : Redémarrer un serveur du cluster Nutanix

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 140px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Valider

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Redémarrer le serveur" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr(e) de vouloir redémarrer votre serveur {nodeName} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Valider | Primary (bleu) | Redémarre le serveur |

## 7. APIS & ENDPOINTS
- POST /nutanix/{serviceName}/node/{nodeId}/reboot - Redémarrer le serveur
