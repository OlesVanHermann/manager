# Cahier des Charges : bare-metal/dedicated-cloud - Modal Résilier Service

## 1. VUE D'ENSEMBLE
- **Type** : Modal de résiliation danger
- **Contexte** : Gestion Dedicated Cloud (VMware)
- **Objectif** : Résilier définitivement le service Dedicated Cloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 350px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning danger
- **OdsSelect** : Type de résiliation
- **OdsButton** : Annuler, Confirmer
- **OdsText** : Descriptions

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résilier le service" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × (rouge) |

## 5. CONTENU

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (#FEE2E2 avec border #FECACA) |
| Titre | "⚠ Attention !" |
| Texte | "Cette action est irréversible et entraînera la perte définitive de vos données." |
| Couleur texte | Rouge (#DC2626) |

### Sélection type de résiliation
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de résiliation" |
| Type | OdsSelect avec dropdown |

### Options de résiliation
| Option | Description |
|--------|-------------|
| Résilier immédiatement | Suppression immédiate |
| Résilier à expiration | Suppression à la fin de la période |

### Description conséquences
| Ligne | Contenu |
|-------|---------|
| 1 | "La résiliation entraîne la suppression de toutes les machines virtuelles," |
| 2 | "datastores et configurations associées à ce service." |
| 3 | "Êtes-vous sûr de vouloir continuer ?" (noir) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Danger (#DC2626) | Lance la résiliation |

## 7. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/terminate - Résilier immédiatement
- PUT /dedicatedCloud/{serviceName}/serviceInfos - Résilier à expiration
