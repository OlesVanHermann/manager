# Cahier des Charges : bare-metal/ip - Page statistiques trafic

## 1. VUE D'ENSEMBLE
- **Type** : Page statistiques avec graphique
- **Contexte** : Monitoring du trafic IP
- **Objectif** : Visualiser les statistiques de trafic d'une adresse IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsText** : Titres, labels
- **OdsLink** : Retour
- **OdsSelect** : Période
- **OdsCard** : Statistiques
- **OdsChart** : Graphique de trafic

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour" (bleu) |
| Titre | "Statistiques - {ip}" |

## 5. SÉLECTEUR DE PÉRIODE
| Propriété | Valeur |
|-----------|--------|
| Label | "Période :" |
| Type | OdsSelect |
| Options | Dernières 24h, 7 jours, 30 jours |
| Défaut | "Dernières 24h" |
