# Cahier des Charges : bare-metal/license - Upgrader licence

## 1. VUE D'ENSEMBLE
- **Page** : Upgrader ma licence
- **Type** : Formulaire étapes
- **Objectif** : Mettre à niveau une licence vers une version supérieure

## 2. STRUCTURE PAGE
- Titre + lien retour
- Récapitulatif licence actuelle
- Étape 1 : Options
- Étape 2 : Durée
- Étape 3 : Validation
- Bouton commande

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour à la gestion des licenses
- **OdsCard** : Récapitulatif actuel
- **OdsSelect** : Version, Domaines, Antivirus
- **OdsRadio** : Durée
- **OdsText** : Prix à payer
- **OdsButton** : Générer le bon de commande (primary)

## 4. RÉCAPITULATIF LICENCE ACTUELLE
| Champ | Valeur exemple |
|-------|----------------|
| Version | Plesk - Edition Web Admin |
| Nombre de domaines | 10 |
