# Cahier des Charges : bare-metal/nas-ha - Modal modifier taille partition

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : NAS-HA (Storage haute disponibilité)
- **Objectif** : Modifier la taille d'une partition NAS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, aide
- **OdsInput** : Champ numérique avec stepper
- **OdsButton** : -, +, Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la taille de \"{partitionName}\"" |
| Exemple | "Modifier la taille de \"data\"" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champ Nouvelle taille
| Propriété | Valeur |
|-----------|--------|
| Label | "Nouvelle taille pour \"{partitionName}\" (Go)" |
| Aide | "Min: 10 Go - Max: 5000 Go" |
| Type | Input numérique avec stepper |

### Composant Stepper
| Élément | Description |
|---------|-------------|
| Bouton - | Décrémente la valeur |
| Input | Valeur numérique (ex: 500) |
| Bouton + | Incrémente la valeur |
| Unité | "Go" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary (bleu) | Applique la nouvelle taille |

## 7. APIS & ENDPOINTS
- PUT /dedicated/nasha/{serviceName}/partition/{partitionName} - Modifier taille partition
