# Cahier des Charges : bare-metal/dedicated-cloud - Modal Supprimer Entrée Sécurité

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion de la politique de sécurité Dedicated Cloud
- **Objectif** : Supprimer une ou plusieurs entrées IP de la liste d'accès

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsMessage** : Warning si aucune sélection
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'entrée de sécurité" |
| Fond | Gris clair (#F9FAFB) |
| Couleur texte | Gris foncé (#1F2937) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation (une entrée)
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Êtes-vous sûr de vouloir supprimer l'entrée" |
| Ligne 2 | "{ip_range} de la politique de sécurité ?" |
| IP/CIDR | Bleu (#0050D7), font-weight 500 |

### Message de confirmation (plusieurs entrées)
| Propriété | Valeur |
|-----------|--------|
| Texte | "Êtes-vous sûr de vouloir supprimer {count} entrées de la politique de sécurité ?" |

### Warning aucune sélection (conditionnel)
| Propriété | Valeur |
|-----------|--------|
| Fond | Warning (#FEF3C7) |
| Texte | "⚠ Aucune entrée sélectionnée" |
| Couleur texte | #92400E |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Supprime l'entrée |

## 7. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/allowedNetwork/{networkId} - Supprimer entrée sécurité
