# Cahier des Charges : bare-metal/load-balancer - Modal Détails IP Failover

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information (lecture seule)
- **Contexte** : Configuration Load Balancer
- **Objectif** : Afficher les détails d'une IP Failover

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsBadge** : Statut
- **OdsLink** : Frontend associé
- **OdsText** : Informations

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "IP Failover" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU

### Informations affichées (2 colonnes)
| Label | Valeur exemple |
|-------|----------------|
| Adresse IP | 203.0.113.10 |
| Statut | Badge "actif" (vert) |
| Zone | gra |
| Type | failover |
| Frontend associé | frontend-web-443 (lien bleu) |
| Date d'attribution | 15/12/2024 10:30 |

### Format affichage
| Propriété | Valeur |
|-----------|--------|
| Label | 12px, gris #6B7280 |
| Valeur | 14px, noir #1F2937, font-weight 500 |
| Lien | 14px, bleu #0050D7 |

### Badge Statut
| État | Fond | Couleur texte |
|------|------|---------------|
| actif | #DCFCE7 | #16A34A |
| inactif | #FEE2E2 | #DC2626 |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/failover - Liste IPs failover
- GET /ip/{ip}/game - Détails IP
