# Cahier des Charges : bare-metal/nas-ha - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard NAS-HA
- **Type** : Page détail avec onglets
- **Objectif** : Gérer un service NAS haute disponibilité

## 2. STRUCTURE PAGE
- Breadcrumb Bare Metal Cloud > NAS-HA > nom
- Titre avec badge statut
- Onglets : Tableau de bord | Partitions | Snapshots | Tâches
- Cartes informations
- Tableau partitions résumé
- Actions rapides

- **OdsCard** : Cartes informations
- **OdsProgressBar** : Utilisation stockage
- **OdsDatagrid** : Tableau partitions
- **OdsButton** : Actions
- **OdsActionMenu** : Menu partitions
