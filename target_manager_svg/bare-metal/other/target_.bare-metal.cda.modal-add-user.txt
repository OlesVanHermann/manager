# Cahier des Charges : bare-metal/cda - Ajouter utilisateur

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un utilisateur
- **Type** : Modal formulaire
- **Objectif** : Créer un nouvel utilisateur CEPH

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ nom
- Texte d'aide
- Message info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : Nom de l'utilisateur
- **OdsText** : Texte d'aide
- **OdsMessage** : Info clé générée (bleu)
- **OdsButton** : Annuler (outline), Ajouter (primary)

## 4. CHAMP NOM
| Propriété | Valeur |
|-----------|--------|
| Label | Nom * |
| Type | Input text |
| Placeholder | admin-user |
| Requis | Oui |

## 5. RÈGLES DE VALIDATION
- Le nom ne peut contenir d'espaces
- Le nom doit avoir une longueur maximale de 50 caractères

## 6. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte 1 | Une clé d'accès sera générée automatiquement. |
| Texte 2 | Vous pourrez configurer les permissions après création. |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /dedicated/ceph/{serviceName}/user
  - Paramètre : userName (string)
