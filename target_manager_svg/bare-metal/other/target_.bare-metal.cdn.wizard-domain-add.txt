# Cahier des Charges : bare-metal/cdn - Wizard Ajouter Domaine

## 1. VUE D'ENSEMBLE
- **Type** : Wizard 3 étapes
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Ajouter un nouveau domaine au service CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 800px |
| Page hauteur | 580px |

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes
- **OdsSelect** : Backend existant
- **OdsInput** : Nouveau backend (IPv4)
- **OdsText** : Informations
- **OdsButton** : Précédent, Suivant

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un domaine" |

## 5. STEPPER (3 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Domaine | Complété (✓) |
| 2 | Backend | Active |
| 3 | Confirmation | À venir |

## 6. CONTENU ÉTAPE 2 - BACKEND

| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Précédent | Secondary | Gauche | Retour étape 1 |
| Suivant | Primary (#0050D7) | Droite | Passe à la confirmation |

## 8. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains - Ajouter domaine
- GET /cdn/dedicated/{serviceName}/backends - Liste des backends
