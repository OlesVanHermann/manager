# Cahier des Charges : bare-metal/nutanix - Détail noeud

## 1. VUE D'ENSEMBLE
- **Page** : Détail d'un noeud Nutanix
- **Type** : Page détail avec sous-onglets
- **Objectif** : Voir configuration d'un noeud

## 2. STRUCTURE PAGE
- Breadcrumb complet (cluster > Noeuds)
- Titre noeud + bouton Guides
- Sous-titre ID serveur
- Sous-onglets : Informations générales | Interventions | IPMI | Tâches
- Grille de cartes 2x2

## 3. COMPOSANTS UI
- **OdsTabs** : Sous-onglets
- **OdsCard** : Cartes informations
- **OdsBadge** : Statut (Installé)
- **OdsButton** : Guides
- **OdsLink** : Liens vRack
