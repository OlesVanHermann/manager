# Cahier des Charges : bare-metal/network-security - Onglet Scrubbing Center

## 1. VUE D'ENSEMBLE
- **Page** : Configuration Scrubbing Center
- **Type** : Onglet avec configuration et tableau
- **Objectif** : Configurer le filtrage avancé des IP

## 2. STRUCTURE PAGE
- Breadcrumb (Network Security > Scrubbing Center)
- Titre "Scrubbing Center"
- Onglets navigation
- Bandeau explicatif
- Carte configuration globale
- Tableau IP protégées

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsMessage** : Bandeau informatif
- **OdsCard** : Carte configuration
- **OdsToggle** : Toggles options
- **OdsDatagrid** : Tableau IP
- **OdsBadge** : État mitigation/firewall
- **OdsActionMenu** : Menu actions par IP
- **OdsPagination** : Pagination tableau

## 4. BANDEAU EXPLICATIF
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte ligne 1 | Le Scrubbing Center permet de configurer des règles de filtrage avancées pour vos IP |
| Texte ligne 2 | Le trafic suspect est analysé et filtré avant d'atteindre vos serveurs |

## 5. CONFIGURATION GLOBALE

### Options avec Toggle
| Option | État | Description |
|--------|------|-------------|
| Mitigation permanente | ON | Le trafic est toujours filtré par le Scrubbing Center |
| Mitigation automatique | ON | Activation automatique en cas d'attaque détectée |
| Firewall Network | ON | Règles de filtrage au niveau réseau |

## 6. COLONNES TABLEAU IP
| Colonne | Type | Description |
|---------|------|-------------|
| Adresse IP | Link | IP protégée (cliquable) |
| Service | Text | Nom du service associé |
| Mitigation | Badge | Permanent (vert), Automatique (orange) |
| Firewall | Badge | Actif (vert), Inactif (orange) |
| Actions | ActionMenu | Menu ⋮ |

## 7. BADGES MITIGATION
| Type | Couleur | Description |
|------|---------|-------------|
| Permanent | Vert | Filtrage continu |
| Automatique | Orange | Activation sur détection |

## 8. BADGES FIREWALL
| État | Couleur |
|------|---------|
| Actif | Vert |
| Inactif | Orange |

## 9. ACTIONMENU (⋮)
| Action | Description |
|--------|-------------|
| Configurer mitigation | Paramètres mitigation pour cette IP |
| Gérer firewall | Règles firewall pour cette IP |
| Voir statistiques | Stats de trafic/attaques pour cette IP |

## 10. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | Affichage X-Y sur Z IP |
| Position | Bas du tableau |

## 11. APIS & ENDPOINTS
- GET /ip/service - Liste des services IP
- GET /ip/{ip}/mitigation - Configuration mitigation
- PUT /ip/{ip}/mitigation - Modifier mitigation
- GET /ip/{ip}/firewall/{ipOnFirewall} - Règles firewall
- PUT /ip/{ip}/firewall/{ipOnFirewall} - Modifier firewall
- GET /ip/{ip}/mitigation/stats - Statistiques
