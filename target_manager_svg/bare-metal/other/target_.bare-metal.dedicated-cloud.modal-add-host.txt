# Cahier des Charges : bare-metal/dedicated-cloud - Modal ajout host

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un host
- **Type** : Formulaire commande
- **Objectif** : Commander un nouveau host VMware

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Select datacenter
- Sélection type de host (cartes)
- Sélection quantité
- Récapitulatif commande
- Checkbox CGV
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (800x700px)
- **OdsSelect** : Datacenter
- **OdsCard** : 3 cartes host (sélection)
- **OdsQuantity** : Sélecteur quantité
- **OdsMessage** : Récapitulatif (gris)
- **OdsCheckbox** : CGV
- **OdsButton** : Annuler (outline), Commander (primary)
