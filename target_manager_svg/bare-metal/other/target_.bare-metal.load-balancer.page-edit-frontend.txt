# Cahier des Charges : bare-metal/load-balancer - Page modifier frontend

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire d'édition
- **Contexte** : Load Balancer Dedicated
- **Objectif** : Modifier la configuration d'un frontend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 900px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, labels, aides
- **OdsLink** : Retour, options avancées
- **OdsInput** : Nom, port
- **OdsButton** : Sélection protocole
- **OdsSelect** : Datacenter, ferme, certificat
- **OdsButton** : Modifier, Annuler

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour" |
| Titre | "Modifier le frontend" |

## 5. FORMULAIRE

### Champ Nom d'affichage
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom d'affichage" |
| Type | OdsInput |
| Exemple | "frontend-https-production" |

### Sélection Protocole
| Propriété | Valeur |
|-----------|--------|
| Label | "Protocole" |
| Type | Groupe de boutons |
| Options | HTTP, HTTPS (sélectionné), TCP, UDP |

### Champ Port
| Propriété | Valeur |
|-----------|--------|
| Label | "Port" |
| Type | OdsInput |
| Exemple | 443 |
| Aide | "Port ou plage de ports (ex: 80, 443, 8080-8090). Max: 65535" |

### Champ Datacenter
| Propriété | Valeur |
|-----------|--------|
| Label | "Datacenter" |
| Type | OdsSelect |
| Exemple | "GRA (Gravelines)" |

### Champ Ferme par défaut
| Propriété | Valeur |
|-----------|--------|
| Label | "Ferme par défaut" |
| Type | OdsSelect |
| Format | "{farmName} ({farmId})" |
| Exemple | "farm-production-web (123456)" |

### Champ Certificat SSL
| Propriété | Valeur |
|-----------|--------|
| Label | "Certificat SSL par défaut" |
| Type | OdsSelect |
| Exemple | "*.example.com (Let's Encrypt)" |

## 6. OPTIONS AVANCÉES
| Élément | Description |
|---------|-------------|
| Lien | "▼ Options avancées" |
| Sous-texte | "IP failover, restrictions, headers..." |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Modifier | Primary (bleu) | Enregistre les modifications |
| Annuler | Secondary | Retour |

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Détails frontend
- PUT /ipLoadbalancing/{serviceName}/http/frontend/{frontendId} - Modifier frontend
- GET /ipLoadbalancing/{serviceName}/zone - Zones disponibles
- GET /ipLoadbalancing/{serviceName}/http/farm - Fermes disponibles
- GET /ipLoadbalancing/{serviceName}/ssl - Certificats disponibles
