# Cahier des Charges : bare-metal/vrack - Wizard Commander vRack

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard (étape unique)
- **Contexte** : Commande nouveau vRack
- **Objectif** : Commander un vRack gratuit avec acceptation des CGU

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 900px |
| Page hauteur | 550px |
| Container contenu | 780x320px |

## 3. COMPOSANTS UI
- **OdsMessage** : Info gratuité
- **OdsLink** : Liens contrats
- **OdsCheckbox** : Acceptation CGU
- **OdsButton** : Annuler, Commander

## 4. NAVIGATION
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour" (bleu #0050D7) |

## 5. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander un vRack" |

## 6. CONTENU

### Message gratuité
| Propriété | Valeur |
|-----------|--------|
| Type | Info |
| Fond | #EFF6FF |
| Border | #BFDBFE |
| Icône | ℹ |
| Texte | "La commande d'un vRack est entièrement gratuite" |
| Couleur texte | #1E40AF |

### Section Contrats
| Propriété | Valeur |
|-----------|--------|
| Titre | "Acceptation des conditions d'utilisation" |
| Description | "Pour continuer la commande de votre vRack, merci de lire et d'accepter le(s) contrat(s) ci-dessous :" |

### Liens contrats
| Contrat | Style |
|---------|-------|
| Conditions générales d'utilisation ↗ | Lien bleu #0050D7 |
| Conditions particulières vRack ↗ | Lien bleu #0050D7 |

### Checkbox acceptation
| Propriété | Valeur |
|-----------|--------|
| Type | OdsCheckbox |
| Label | "J'ai pris connaissance et j'accepte les termes des contrats suivants :" |
| Requis | Oui |

## 7. FOOTER
| Bouton | Style | État | Action |
|--------|-------|------|--------|
| Annuler | Secondary | Toujours actif | Retour à la liste |
| Commander | Primary | Désactivé si checkbox non cochée | Lance la commande |

### État bouton Commander
| Condition | État |
|-----------|------|
| Checkbox non cochée | Désactivé (opacity 0.5) |
| Checkbox cochée | Activé |

## 8. FLUX
```
Page commande → [Commander] → Création vRack → Redirection listing
```

## 9. APIS & ENDPOINTS
- POST /order/vrack/new - Commander un vRack
- GET /order/vrack/new - Récupérer les contrats à accepter
