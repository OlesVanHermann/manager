# Cahier des Charges : bare-metal/dedicated-cloud - Modal Désactiver Backup

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion des backups Dedicated Cloud
- **Objectif** : Désactiver le service de backup pour un datacenter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Désactiver le backup" |
| Fond | Warning (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Êtes-vous sûr de vouloir désactiver le backup" |
| Ligne 2 | "pour le datacenter {datacenter_name} ?" |
| Nom datacenter | Bleu (#0050D7), font-weight 500 |

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Texte | "Toutes vos sauvegardes seront définitivement supprimées." |
| Couleur | Rouge danger (#DC2626) |
| Font-weight | 500 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Warning (orange #F59E0B) | Désactive le backup |

## 7. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/datacenter/{datacenterId}/backup/disable - Désactiver backup
