# Cahier des Charges : bare-metal/nutanix - Onglet Interventions Nœud

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau de données
- **Contexte** : Gestion nœud Nutanix
- **Objectif** : Historique des interventions physiques sur le nœud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |
| Datagrid largeur | 940px |
| Datagrid hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Tableau interventions
- **OdsBadge** : Statuts

## 4. BREADCRUMB
| Niveau | Texte | Lien |
|--------|-------|------|
| 1 | Nutanix | Oui (bleu) |
| 2 | {clusterId} | Oui (bleu) |
| 3 | {nodeId} | Non (noir) |

## 5. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Nœud - {nodeId}" |

## 6. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | Non sélectionné |
| IPMI | Non sélectionné |
| Interventions | Sélectionné (souligné bleu) |
| Tâches | Non sélectionné |

## 7. COLONNES DATAGRID
| Colonne | Largeur | Type | Description |
|---------|---------|------|-------------|
| ID | 125px | Texte | Identifiant intervention (INT-XXXX-XXXXX) |
| TYPE | 250px | Texte | Nature de l'intervention |
| STATUT | 100px | Badge | État de l'intervention |
| DATE DÉBUT | 150px | Date | Timestamp début intervention |
| DATE FIN | 150px | Date | Timestamp fin intervention |

## 8. TYPES D'INTERVENTION
| Type | Description |
|------|-------------|
| Remplacement disque NVMe | Changement de disque défaillant |
| Maintenance préventive | Vérification et entretien planifié |
| Remplacement mémoire | Changement de barrette RAM |
| Remplacement carte réseau | Changement de NIC |
| Mise à jour firmware | Flash du BIOS/BMC |

## 9. STATUTS
| Statut | Fond | Couleur texte |
|--------|------|---------------|
| Terminée | #DCFCE7 | #16A34A |
| En cours | #EFF6FF | #0050D7 |

| Annulée | #F3F4F6 | #6B7280 |

## 10. FORMAT DATES
| Propriété | Valeur |
|-----------|--------|
| Format | DD/MM/YYYY HH:mm |
| Exemple | 15/12/2024 09:00 |

## 11. FOOTER DATAGRID
| Propriété | Valeur |
|-----------|--------|
| Texte | "{N} interventions" |

| Propriété | Valeur |
|-----------|--------|

## 13. APIS & ENDPOINTS
- GET /nutanix/{serviceName}/node/{nodeId}/intervention - Liste interventions
