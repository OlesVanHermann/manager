# Cahier des Charges : bare-metal/ip - Modal supprimer délégation reverse

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation de suppression
- **Contexte** : Gestion des adresses IP
- **Objectif** : Confirmer la suppression d'une délégation reverse DNS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la délégation reverse" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Message | "Êtes-vous sûr de vouloir supprimer la délégation reverse pour le bloc {ipBlock} ?" |
| Format ipBlock | Gras |
| Exemple | "2001:41d0:xxxx::/56" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime la délégation |

## 7. APIS & ENDPOINTS
- DELETE /ip/{ip}/delegation/{target} - Supprimer délégation reverse
