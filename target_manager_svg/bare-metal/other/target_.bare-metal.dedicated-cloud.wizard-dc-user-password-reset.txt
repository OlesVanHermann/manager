# Cahier des Charges : bare-metal/dedicated-cloud - Wizard Changer Mot de Passe

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de formulaire
- **Contexte** : Gestion utilisateurs Dedicated Cloud (VMware)
- **Objectif** : Changer le mot de passe d'un utilisateur vSphere

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 800px |
| Page hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsInput** : Nouveau mot de passe, Confirmation
- **OdsButton** : Annuler, Confirmer
- **OdsText** : Règles de validation

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Changer le mot de passe" |

## 5. CONTENU

### Information utilisateur
| Propriété | Valeur |
|-----------|--------|
| Texte | "Changer le mot de passe de l'utilisateur {username}" |

| Propriété | Valeur |
|-----------|--------|

| Règle | État valide | État invalide |
|-------|-------------|---------------|

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Confirmer | Droite | Primary (#0050D7) | Change le mot de passe |

## 7. VALIDATION
| Règle | Message d'erreur |
|-------|------------------|
| Mot de passe vide | "Le mot de passe est requis" |
| Règles non respectées | "Le mot de passe ne respecte pas les critères" |
| Confirmation différente | "Les mots de passe ne correspondent pas" |

## 8. APIS & ENDPOINTS
- POST /dedicatedCloud/{serviceName}/user/{userId}/changePassword - Changer mot de passe
