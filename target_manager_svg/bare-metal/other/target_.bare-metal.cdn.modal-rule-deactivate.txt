# Cahier des Charges : bare-metal/cdn - Modal Désactiver Règle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation (danger)
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Confirmer la désactivation d'une règle de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 600px |
| Page hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsText** : Question de confirmation
- **OdsButton** : Annuler, Désactiver

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Désactiver la règle de cache" |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir désactiver la règle" |
| Nom règle | "{ruleName}" (bleu #0050D7, font-weight 500) |

## 6. FOOTER
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Désactiver | Primary danger | #DC2626 | Désactive la règle |

## 7. APIS & ENDPOINTS
- PUT /cdn/dedicated/{serviceName}/domains/{domainName}/cacheRules/{ruleId} - Désactiver règle (status: off)
