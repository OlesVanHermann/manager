# Cahier des Charges : bare-metal/license - Détail licence

## 1. VUE D'ENSEMBLE
- **Page** : Détail licence
- **Type** : Page détail
- **Objectif** : Afficher les informations d'une licence

## 2. STRUCTURE PAGE
- Titre + lien retour
- Icône licence
- Infos licence
- Clé de licence
- Boutons actions

## 3. COMPOSANTS UI
- **OdsLink** : ← Retour à la gestion des licenses
- **OdsText** : Labels et valeurs
- **OdsBadge** : Automatique (success)
- **OdsTextarea** : Clé de licence (readonly)
- **OdsButton** : Tout sélectionner, Modifier, Migrer la licence, Mettre à niveau

## 4. INFOS LICENCE
| Champ | Valeur exemple |
|-------|----------------|
| Ip | 51.210.xxx.xxx |
| Service | ns1234567.ip-51-210-xxx.eu |
| Renouvellement | Automatique (badge) + 15/01/2025 |
| Version | Plesk - Edition Web Pro |
| Nombre de domaines | Illimité |
| Licence ID | PLESK-12345678 |
| Date de création | 01/06/2023 |
| Résilier à l'expiration | Non + bouton Modifier |

## 5. CLÉ DE LICENCE
| Propriété | Valeur |
|-----------|--------|
| Type | Textarea readonly |
| Format | Monospace |
| Contenu | Clé multi-lignes XXXXX-XXXXX-... |
| Action | Tout sélectionner (bouton) |

## 6. BOUTONS ACTIONS
| Bouton | Style | Position |
|--------|-------|----------|
| Migrer la licence | Outline | Droite |
| Mettre à niveau | Outline | Droite |

## 7. APIS & ENDPOINTS
- GET /license/{type}/{serviceName}
  - Détails licence
- GET /license/{type}/{serviceName}/tasks
  - Tâches en cours
