# Cahier des Charges : bare-metal/ip - Modal découper bloc BYOIP

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire avec prévisualisation
- **Contexte** : BYOIP (Bring Your Own IP)
- **Objectif** : Découper un bloc BYOIP en sous-blocs plus petits

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, descriptions
- **OdsSelect** : Taille blocs enfants
- **OdsCard** : Zone prévisualisation
- **OdsButton** : Annuler, Découper

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Découper le bloc {bloc_ip}" |
| Exemple | "Découper le bloc 192.0.2.0/23" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sélectionnez la taille des blocs enfants :" |

## 6. FORMULAIRE

### Champ Taille des blocs
| Propriété | Valeur |
|-----------|--------|
| Type | OdsSelect |
| Options | /24, /25, /26, /27, /28 |
| Exemple | "/24" |

## 7. PRÉVISUALISATION
| Propriété | Valeur |
|-----------|--------|
| Fond | Gris (#F3F4F6) |
| Titre | "Configuration résultante :" (gras) |
| Sous-titre | "Blocs enfants qui seront créés :" |

### Liste des blocs
| Propriété | Valeur |
|-----------|--------|
| Icône | Cercle vert (#10B981) |
| Format | Adresse du bloc |
| Exemple | "192.0.2.0/24", "192.0.3.0/24" |

### Note
| Propriété | Valeur |
|-----------|--------|
| Texte | "Le bloc parent sera découpé en {N} sous-blocs." |
| Couleur | Gris (#6B7280) |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Découper | Primary (bleu #0050D7) | Lance le découpage |

## 9. APIS & ENDPOINTS
- GET /ip/byoip/{ip} - Informations bloc BYOIP
- POST /ip/byoip/{ip}/slice - Découper le bloc
