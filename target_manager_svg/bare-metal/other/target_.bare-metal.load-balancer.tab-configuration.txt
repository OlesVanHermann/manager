# Cahier des Charges : bare-metal/load-balancer - Onglet Configuration

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec datagrid
- **Contexte** : Gestion Load Balancer
- **Objectif** : Appliquer les modifications de configuration sur les datacenters

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste des datacenters
- **OdsCheckbox** : Sélection datacenter
- **OdsProgress** : Barre de progression
- **OdsBadge** : Statut modifications
- **OdsButton** : Appliquer la configuration
- **ActionMenu** : Menu contextuel (⋮)

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration" |
| Description | "Appliquez les modifications de configuration sur vos datacenters." |

## 5. BOUTON PRINCIPAL
| Propriété | Valeur |
|-----------|--------|
| Label | "Appliquer la configuration" |
| Style | Secondary (bordure grise) |
| Largeur | 180px |
| Hauteur | 40px |
| Position | Au-dessus du datagrid |

## 6. DATAGRID
| Propriété | Valeur |
|-----------|--------|
| Largeur | 920px |
| Hauteur | 320px |

### Colonnes
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Checkbox | 40px | Sélection pour appliquer |
| Datacenter | 200px | Code zone (gra, rbx, sbg) |
| Progression | 200px | Barre de progression ou "—" |
| État | 150px | Badge nombre modifications |
| Actions | 100px | Menu ⋮ |

### Badges État
| État | Fond | Couleur texte | Texte |
|------|------|---------------|-------|
| 0 modification | #DCFCE7 | #16A34A | "0 modification" |
| N modifications | #FEF3C7 | #92400E | "N modifications" |
| Configuration... | #EFF6FF | #1E40AF | "Configuration..." |

### Barre de progression
| Propriété | Valeur |
|-----------|--------|
| Largeur | 180px |
| Hauteur | 16px |
| Fond | #E5E7EB |
| Remplissage | #0050D7 |
| Border radius | 8px |
| Label | Pourcentage (ex: "70%") |

### ActionMenu (menu ⋮)
| Option | Action |
|--------|--------|
| Appliquer la configuration | Applique sur ce datacenter |

## 7. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "{n} datacenters" |
| Position | Bas du datagrid |

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/zone - Liste des zones
- GET /ipLoadbalancing/{serviceName}/zone/{name}/pendingChanges - Modifications en attente
- POST /ipLoadbalancing/{serviceName}/refresh - Appliquer configuration
