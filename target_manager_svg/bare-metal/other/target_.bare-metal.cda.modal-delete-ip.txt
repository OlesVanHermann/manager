# Cahier des Charges : bare-metal/cda - Modal supprimer IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation de suppression
- **Contexte** : Ceph Distributed Architecture (CDA)
- **Objectif** : Confirmer la suppression d'une adresse IP du cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'adresse IP" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Information IP
| Champ | Valeur |
|-------|--------|
| Label | "Adresse IP" |
| Format | "{ip}/{cidr}" |
| Exemple | "192.168.1.0/24" |

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Texte | "Confirmez-vous la suppression de cette IP ?" |
| Style | Gras |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime l'IP |

## 7. APIS & ENDPOINTS
- DELETE /dedicatedCloud/{serviceName}/ceph/acl/{aclId} - Supprimer l'ACL IP
