# Cahier des Charges : bare-metal/cdn - Onglet Statistiques Domaine

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tuiles et graphique
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Afficher les statistiques détaillées d'un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1200px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsCard** : Tuiles statistiques et réseau
- **OdsSelect** : Filtres période et type
- **OdsChart** : Graphique linéaire
- **ActionMenu** : Menu contextuel (⋮)

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Statistiques - {domainName}" |

## 5. TUILES INFORMATIONS (2 colonnes)

### Tuile Statistiques
| Propriété | Valeur |
|-----------|--------|
| Largeur | 340px |
| Hauteur | 120px |
| Titre | "Statistiques" |

| Information | Exemple |
|-------------|---------|
| CNAME | cdn-12345.pssl.anycast.me |

### Tuile Réseau
| Propriété | Valeur |
|-----------|--------|
| Largeur | 340px |
| Hauteur | 120px |
| Titre | "Réseau" |

| Information | Exemple |
|-------------|---------|
| Domaine | www.example.com |
| IPv4 | 213.186.33.5 |
| Backend | 192.168.1.10 |

### ActionMenu tuile Réseau
| Option | Couleur | Action |
|--------|---------|--------|
| Bypass CDN | Noir | Active le bypass |
| Supprimer le domaine | Rouge #DC2626 | Ouvre modal suppression |

## 6. SECTION GRAPHIQUE STATISTIQUES
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1000px |
| Hauteur | 380px |
| Type graphique | Ligne avec points |

### Filtres
| Filtre | Options |
|--------|---------|
| Visualiser | Bande passante, Requêtes |
| Depuis | Dernier jour, Dernière semaine, Dernier mois |

### Axes
| Axe | Contenu |
|-----|---------|
| Y | Valeurs (500 MB, 400 MB, 300 MB, 200 MB, 100 MB) |
| X | Jours (Lun, Mar, Mer, Jeu, Ven, Sam, Dim) |

### Style graphique
| Propriété | Valeur |
|-----------|--------|
| Couleur ligne | #0050D7 |
| Épaisseur ligne | 2px |
| Points | Cercles 4px radius |

## 7. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/domains/{domainName}/statistics - Statistiques domaine
- GET /cdn/dedicated/{serviceName}/domains/{domainName} - Détails domaine
