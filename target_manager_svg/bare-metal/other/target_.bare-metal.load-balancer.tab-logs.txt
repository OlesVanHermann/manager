# Cahier des Charges : bare-metal/load-balancer - Onglet Logs

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec sous-onglets et cartes
- **Contexte** : Gestion Load Balancer
- **Objectif** : Configurer la redirection des logs vers Logs Data Platform

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsTabs** : Data streams / Live tail
- **OdsCard** : Cartes de stream
- **OdsBadge** : Statut stream
- **OdsMessage** : Info Logs Data Platform
- **OdsButton** : Gérer, Ajouter un stream
- **OdsLink** : En savoir plus

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/logs - Liste des streams
- POST /ipLoadbalancing/{serviceName}/logs - Créer un stream
- GET /dbaas/logs - Configuration Logs Data Platform
- POST /dbaas/logs/input - Configurer input LDP
