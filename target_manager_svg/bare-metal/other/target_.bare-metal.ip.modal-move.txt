# Cahier des Charges : bare-metal/ip - Déplacer IP

## 1. VUE D'ENSEMBLE
- **Page** : Déplacer le bloc IP
- **Type** : Modal formulaire
- **Objectif** : Déplacer un bloc IP vers un autre service

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Récapitulatif du bloc
- Sélection destination
- Warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsText** : Infos bloc actuel
- **OdsSelect** : Service de destination
- **OdsMessage** : Warning (jaune)
- **OdsButton** : Annuler (outline), Déplacer (primary)

## 4. RÉCAPITULATIF BLOC
| Propriété | Valeur |
|-----------|--------|
| Label | Bloc IP à déplacer |
| Bloc | 192.168.1.0/28 |
| Actuellement sur | ns1234567.ip-192-168-1.eu |

## 5. SÉLECTION DESTINATION
| Propriété | Valeur |
|-----------|--------|
| Label | Service de destination * |
| Description | Sélectionnez le service vers lequel déplacer ce bloc IP. |
| Type | Select |
| Options | Liste des serveurs compatibles, IP Parking |

## 6. OPTIONS DE DESTINATION
- ns9876543.ip-10-0-0.eu
- ns5555555.ip-172-16-0.eu
- vps-abc123.vps.ovh.net
- IP Parking (non associé)

## 7. WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Icône | ⚠️ |
| Titre | Attention |
| Texte | Le déplacement peut entraîner une courte interruption de service. |

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Déplacer | Primary | Droite |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/destinationService
  - Liste des services de destination compatibles
- POST /ip/{ip}/move
  - Paramètre : nexthop (string), to (string)
