# Cahier des Charges : bare-metal/cda - Ajouter contrôle d'accès

## 1. VUE D'ENSEMBLE
- **Page** : Ajouter un contrôle d'accès
- **Type** : Modal formulaire
- **Objectif** : Ajouter une IP autorisée au cluster

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Champ IP
- Texte d'aide (format)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsInput** : IP avec masque
- **OdsText** : Helper format
- **OdsButton** : Annuler (outline), Ajouter (primary)

## 4. CHAMP IP
| Propriété | Valeur |
|-----------|--------|
| Label | IP avec masque de sous-réseau * |
| Type | Input text |
| Placeholder | 192.168.1.0/24 |
| Requis | Oui |

## 5. AIDE FORMAT
Format : 192.168.1.0/24

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Ajouter | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/ceph/{serviceName}/acl
  - Paramètre : aclList (array contenant l'IP CIDR)
