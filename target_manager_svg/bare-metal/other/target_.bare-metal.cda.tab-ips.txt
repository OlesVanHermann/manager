# Cahier des Charges : bare-metal/cda - Contrôle d'accès IP

## 1. VUE D'ENSEMBLE
- **Page** : Cloud Disk Array - Contrôle d'accès IP
- **Type** : Page liste onglet
- **Objectif** : Gérer les restrictions d'accès IP au cluster

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb + titre
- Onglets (Contrôle d'accès IP actif)
- Encart info
- Bouton ajouter
- Tableau des IPs autorisées

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Bare Metal Cloud / Cloud Disk Array
- **OdsTabs** : Contrôle d'accès IP actif
- **OdsMessage** : Info explicatif (bleu)
- **OdsButton** : Ajouter un contrôle d'accès (primary)
- **OdsDatagrid** : Tableau des IPs
- **OdsActionMenu** : Menu actions par ligne

## 4. ENCART INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte 1 | Vous pouvez créer vos contrôles d'accès IP ici. |
| Texte 2 | Votre cluster acceptera seulement le trafic réseau concordant avec les accès créés. |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| IP | Text | IP avec masque (CIDR) |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTION MENU
| Option | Style |
|--------|-------|
| Supprimer | Rouge (danger) |

## 7. BOUTON
| Bouton | Style |
|--------|-------|
| Ajouter un contrôle d'accès | Primary |

## 8. APIS & ENDPOINTS
- GET /dedicated/ceph/{serviceName}/acl
  - Liste des ACL IP
- POST /dedicated/ceph/{serviceName}/acl
  - Ajouter une ACL
  - Paramètre : aclList (array d'IPs CIDR)
- DELETE /dedicated/ceph/{serviceName}/acl/{aclId}
  - Supprimer une ACL
