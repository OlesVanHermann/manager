# Cahier des Charges : bare-metal/ip - Modal supprimer règle firewall

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation de suppression
- **Contexte** : Gestion des adresses IP - Firewall
- **Objectif** : Confirmer la suppression d'une règle de firewall

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer une règle" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voulez-vous vraiment supprimer la règle en position {sequence} ?" |
| Format sequence | Gras |
| Exemple | "3" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime la règle |

## 7. APIS & ENDPOINTS
- DELETE /ip/{ip}/firewall/{ipOnFirewall}/rule/{sequence} - Supprimer la règle
