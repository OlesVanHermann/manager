# Cahier des Charges : bare-metal/cdn - Wizard Vider Cache Domaine

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation simple
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Confirmer le vidage complet du cache d'un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 700px |
| Page hauteur | 380px |

## 3. COMPOSANTS UI
- **OdsMessage** : Avertissement de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Vider le cache du domaine" |

## 5. MESSAGE DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Largeur | 530px |
| Hauteur | 80px |
| Fond | #FEF3C7 |
| Border | #FDE68A |

### Contenu message
| Élément | Valeur |
|---------|--------|
| Titre | "⚠ Confirmation requise" |
| Question | "Êtes-vous sûr de vouloir vider le cache du domaine {domainName} ?" |
| Style domaine | Font-weight 600 |
| Couleur texte | #92400E |

## 6. FOOTER
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Secondary | Gauche | Annule et ferme |
| Confirmer | Primary (#0050D7) | Droite | Vide le cache du domaine |

## 7. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domainName}/flush - Vider cache domaine
