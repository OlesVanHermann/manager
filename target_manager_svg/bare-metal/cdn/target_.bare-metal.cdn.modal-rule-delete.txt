# Cahier des Charges : bare-metal/cdn - Modal supprimer règle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation de suppression
- **Contexte** : CDN Infrastructure
- **Objectif** : Confirmer la suppression d'une règle de cache

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la règle" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voulez-vous vraiment supprimer cette règle de cache ?" |

### Détail de la règle
| Propriété | Valeur |
|-----------|--------|
| Format | "Pattern: {pattern} - TTL: {ttl}s" |
| Exemple | "Pattern: *.jpg, *.png - TTL: 3600s" |
| Style | Gris (#6B7280) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime la règle |

## 7. APIS & ENDPOINTS
- DELETE /cdn/dedicated/{serviceName}/domains/{domain}/cacheRules/{cacheRuleId} - Supprimer la règle
