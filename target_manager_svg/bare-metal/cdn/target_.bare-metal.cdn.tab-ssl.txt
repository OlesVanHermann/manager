# Cahier des Charges : bare-metal/cdn - Onglet SSL

## 1. VUE D'ENSEMBLE
- **Page** : Certificats SSL du CDN
- **Type** : Onglet tableau
- **Objectif** : Gérer les certificats SSL des domaines

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Certificats SSL"
- Onglets navigation
- Bandeau info Let's Encrypt
- Bouton import
- Tableau certificats

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsMessage** : Info Let's Encrypt
- **OdsButton** : Importer un certificat
- **OdsDatagrid** : Tableau certificats
- **OdsBadge** : État (Actif, Expire bientôt, Expiré)
- **OdsActionMenu** : Menu par ligne

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ |
| Texte 1 | Les certificats SSL Let's Encrypt sont automatiquement générés et renouvelés pour vos domaines. |
| Texte 2 | Vous pouvez également importer vos propres certificats si nécessaire. |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Link | Nom du domaine |
| Type | Text | Let's Encrypt / Custom |
| Émetteur | Text | R3, DigiCert, etc. |
| Expiration | Date | Date d'expiration |
| État | Badge | Actif / Expire bientôt / Expiré |
| Actions | ActionMenu | Menu ⋮ |

## 6. BADGES ÉTAT
| État | Couleur | Condition |
|------|---------|-----------|
| Actif | Vert | > 30 jours avant expiration |
| Expire bientôt | Orange | < 30 jours avant expiration |
| Expiré | Rouge | Date passée |

## 7. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Voir le certificat | Normal | Détails du cert |
| Renouveler | Normal | Force renouvellement |
| Supprimer | Danger | Supprime le cert |

## 8. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/ssl
  - Liste des certificats
- POST /cdn/dedicated/{serviceName}/ssl
  - Importer certificat
- DELETE /cdn/dedicated/{serviceName}/ssl/{id}
  - Supprimer certificat
