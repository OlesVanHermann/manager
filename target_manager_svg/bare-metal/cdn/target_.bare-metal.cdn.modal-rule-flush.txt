# Cahier des Charges : bare-metal/cdn - Modal Vider Cache Règle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Confirmer le vidage du cache pour une règle spécifique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 600px |
| Page hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsText** : Question de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Vider le cache de la règle" |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir vider le cache pour la règle" |
| Nom règle | "{ruleName}" (bleu #0050D7, font-weight 500) |

## 6. FOOTER
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Confirmer | Primary | #0050D7 | Vide le cache de la règle |

## 7. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domainName}/cacheRules/{ruleId}/flush - Vider cache règle
