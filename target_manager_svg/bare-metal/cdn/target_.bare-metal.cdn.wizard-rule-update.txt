# Cahier des Charges : bare-metal/cdn - Wizard Modifier TTL Règle

## 1. VUE D'ENSEMBLE
- **Type** : Wizard 2 étapes
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Modifier le TTL (durée de cache) d'une règle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 800px |
| Page hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsStepper** : 2 étapes
- **OdsInput** : Nouveau TTL
- **OdsText** : Informations TTL actuel
- **OdsButton** : Annuler, Suivant

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la règle de cache" |

## 5. STEPPER (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Active |
| 2 | Confirmation | À venir |

## 6. CONTENU ÉTAPE 1 - CONFIGURATION
| Propriété | Valeur |
|-----------|--------|
| Largeur container | 680px |
| Hauteur | 210px |

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Modifier le TTL pour la règle {ruleName}" |
| Style règle | Bleu #0050D7, font-weight 500 |

### Note explicative
| Propriété | Valeur |
|-----------|--------|
| Texte | "Le TTL définit la durée de mise en cache d'un fichier sur les serveurs CDN." |
| Style | 12px, italique, gris |

### TTL actuel
| Propriété | Valeur |
|-----------|--------|
| Label | "TTL actuel" |
| Valeur | "{ttl} secondes ({formaté})" |
| Exemple | "3600 secondes (1 heure)" |

### Nouveau TTL
| Propriété | Valeur |
|-----------|--------|
| Label | "Nouveau TTL (secondes)" |
| Type | OdsInput (number) |
| Largeur | 200px |
| Exemple | 86400 |
| Info complémentaire | "= 1 jour" |

## 7. FOOTER
| Bouton | Style | Position | Action |
|--------|-------|----------|--------|
| Annuler | Secondary | Gauche | Annule et ferme |
| Suivant | Primary (#0050D7) | Droite | Passe à la confirmation |

## 8. APIS & ENDPOINTS
- PUT /cdn/dedicated/{serviceName}/domains/{domainName}/cacheRules/{ruleId} - Modifier règle
