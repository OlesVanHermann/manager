# Cahier des Charges : bare-metal/cdn - Onglet Domaines

## 1. VUE D'ENSEMBLE
- **Page** : Liste des domaines du CDN
- **Type** : Onglet tableau
- **Objectif** : Gérer les domaines servis par le CDN

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Domaines"
- Onglets navigation
- Bouton ajout
- Tableau domaines
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Ajouter un domaine
- **OdsDatagrid** : Tableau domaines
- **OdsBadge** : Statut SSL, État
- **OdsActionMenu** : Menu par ligne
- **OdsPagination** : Navigation

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Link | Nom du domaine |
| Backend | Text | Serveur origin:port |
| SSL | Badge | Actif (vert), En attente (orange) |
| Règles cache | Text | Nombre de règles |
| État | Badge | Actif, Validation |
| Actions | ActionMenu | Menu ⋮ |

## 5. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Voir statistiques | Normal | Stats du domaine |
| Gérer les règles | Normal | Règles de cache |
| Modifier le backend | Normal | Changer origin |
| Purger le cache | Normal | Vider le cache |
| Supprimer | Danger | Retire le domaine |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Ajouter un domaine | Primary | Haut |

## 7. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/domains
  - Liste des domaines
- POST /cdn/dedicated/{serviceName}/domains
  - Ajouter domaine
- DELETE /cdn/dedicated/{serviceName}/domains/{domain}
  - Supprimer domaine
