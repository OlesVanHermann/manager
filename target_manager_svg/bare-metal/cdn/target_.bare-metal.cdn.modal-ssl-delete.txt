# Cahier des Charges : bare-metal/cdn - Modal Supprimer Certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation (danger)
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Confirmer la suppression d'un certificat SSL

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement opération en cours
- **OdsText** : Question de confirmation
- **OdsButton** : Non, Oui

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le certificat SSL" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. MESSAGE D'AVERTISSEMENT (optionnel)
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Titre | "Attention" |
| Texte | "Une opération est en cours sur ce certificat." |
| Couleur texte | #92400E |

## 6. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Êtes-vous sûr de vouloir supprimer le certificat" |
| Nom certificat | "{certificateName}" (bleu #0050D7, font-weight 500) |

## 7. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Non | Secondary | Bordure grise | Ferme modal |
| Oui | Primary danger | #DC2626 | Supprime le certificat |

## 8. APIS & ENDPOINTS
- DELETE /cdn/dedicated/{serviceName}/ssl - Supprimer certificat SSL
