# Cahier des Charges : bare-metal/cdn - Onglet Logs

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec console de logs en temps réel
- **Contexte** : Gestion CDN Dédié
- **Objectif** : Afficher les logs d'accès en temps réel pour un domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsText** : Console de logs (fond sombre)
- **OdsIndicator** : Statut connexion
- **Légende** : Codes couleur des statuts

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Logs en temps réel - {domainName}" |

## 5. CONSOLE DE LOGS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 920px |
| Hauteur | 380px |
| Fond | #1F2937 (gris foncé) |
| Border radius | 4px |
| Police | monospace, 11px |

### Format des entrées de log
```
{IP} - {user} [{date}] "{method} {url} {protocol}" {status} {size} "{referer}" "{userAgent}" {cacheStatus}
```

### Couleurs par statut HTTP
| Statut | Couleur |
|--------|---------|
| 2xx (succès) | #10B981 (vert) |
| 3xx (redirection) | #F59E0B (orange) |
| 5xx (erreur) | #EF4444 (rouge) |

### Statuts de cache
| Statut | Description |
|--------|-------------|
| HIT | Servi depuis le cache |
| MISS | Non en cache |
| BYPASS | Bypassé |
| ERROR | Erreur |

### Scrollbar
| Propriété | Valeur |
|-----------|--------|
| Fond piste | #374151 |
| Fond curseur | #6B7280 |
| Largeur | 8px |

### Légende (bas de console)
| Couleur | Texte |
|---------|-------|
| Vert #10B981 | ● 2xx |
| Orange #F59E0B | ● 3xx |
| Rouge #EF4444 | ● 5xx |

## 6. BARRE DE STATUT
| Propriété | Valeur |
|-----------|--------|
| Indicateur | Cercle vert #10B981 |
| Texte | "Connecté - Flux en direct" |
| Compteur | "{n} requêtes affichées" |

## 7. APIS & ENDPOINTS
- WebSocket /cdn/dedicated/{serviceName}/domains/{domainName}/logs - Flux de logs en temps réel
