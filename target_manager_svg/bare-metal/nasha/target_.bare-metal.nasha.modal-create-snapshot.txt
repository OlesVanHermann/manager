# Cahier des Charges : bare-metal/nasha - Modal création snapshot

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un snapshot
- **Type** : Formulaire création
- **Objectif** : Créer un snapshot manuel d'une partition

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Description avec nom partition
- Champ nom snapshot
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x300px)
- **OdsInput** : Nom snapshot
- **OdsText** : Description
- **OdsButton** : Annuler (outline), Créer (primary)

## 4. CONTENU
| Élément | Valeur |
|---------|--------|
| Texte | Créer un snapshot manuel de la partition |
| Partition | partition-web (bold) |

## 5. CHAMPS FORMULAIRE

### Nom du snapshot
| Propriété | Valeur |
|-----------|--------|
| Label | Nom du snapshot (optionnel) |
| Aide | Laissez vide pour utiliser la date/heure actuelle. |
| Placeholder | Ex: before-migration-2024 |
| Requis | Non |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/nasha/{serviceName}/partition/{partitionName}/customSnapshot
  - name: string (optionnel)
