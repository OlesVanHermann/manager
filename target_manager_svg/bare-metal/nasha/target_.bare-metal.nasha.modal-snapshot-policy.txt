# Cahier des Charges : bare-metal/nasha - Modal snapshot policies

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer les snapshot policies
- **Type** : Configuration
- **Objectif** : Activer/désactiver les snapshots automatiques

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- 3 toggles de fréquence
- Select de rétention
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x480px)
- **OdsToggle** : 3 toggles
- **OdsSelect** : Rétention
- **OdsButton** : Annuler (outline), Enregistrer (primary)

## 4. TOGGLES SNAPSHOTS

### Snapshots quotidiens
| Propriété | Valeur |
|-----------|--------|
| Label | Snapshots quotidiens |
| Description | Créé chaque jour à 01:00 UTC |
| État | Activé (ON) |

### Snapshots hebdomadaires
| Propriété | Valeur |
|-----------|--------|
| Label | Snapshots hebdomadaires |
| Description | Créé chaque dimanche à 01:00 UTC |
| État | Activé (ON) |

### Snapshots mensuels
| Propriété | Valeur |
|-----------|--------|
| Label | Snapshots mensuels |
| Description | Créé le 1er de chaque mois à 01:00 UTC |
| État | Désactivé (OFF) |

## 5. RÉTENTION
| Propriété | Valeur |
|-----------|--------|
| Label | Rétention des snapshots |
| Type | Select |
| Valeur | 7 jours |
| Options | 1 jour, 3 jours, 7 jours, 14 jours, 30 jours |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Enregistrer | Primary | Droite |

## 7. APIS & ENDPOINTS
- PUT /dedicated/nasha/{serviceName}/partition/{partitionName}/snapshot
  - snapshotType: enum (day-1, day-2, day-3, day-7, week-1, month-1)
