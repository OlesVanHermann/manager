# Cahier des Charges : bare-metal/nasha - Liste détaillée

## 1. VUE D'ENSEMBLE
- **Page** : Liste NAS-HA détaillée
- **Type** : Page listing avec bouton commande
- **Objectif** : Liste complète des NAS-HA avec infos création partition

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre + bouton Commander
- Tableau détaillé
- Pagination

## 3. COMPOSANTS UI
- **OdsDatagrid** : Tableau
- **OdsButton** : Commander un NAS-HA (primary)
- **OdsBadge** : Autorisée (success), Non autorisée (error)
- **OdsLink** : ID cliquable
- **OdsPagination** : Navigation pages
