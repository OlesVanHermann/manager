# Cahier des Charges : bare-metal/nasha - Modal suppression accès

## 1. VUE D'ENSEMBLE
- **Modal** : Supprimer un accès
- **Type** : Confirmation suppression
- **Objectif** : Révoquer l'accès d'une IP à la partition

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Message de confirmation
- Warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x230px)
- **OdsText** : Messages
- **OdsButton** : Fermer (outline), Supprimer l'accès (danger)

## 4. CONTENU
| Élément | Valeur |
|---------|--------|
| Question | Êtes-vous certain(e) de vouloir supprimer l'accès pour l'IP |
| IP | 10.0.0.1/32 (bold) |
| Warning | Cette IP ne pourra plus accéder à la partition. |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline | Gauche |
| Supprimer l'accès | Danger (rouge) | Droite |

## 6. APIS & ENDPOINTS
- DELETE /dedicated/nasha/{serviceName}/partition/{partitionName}/access/{ip}
  - Supprime l'accès
