# Cahier des Charges : bare-metal/nasha - Onglet Partitions

## 1. VUE D'ENSEMBLE
- **Onglet** : Partitions
- **Type** : Page liste partitions
- **Objectif** : Gérer les partitions du NAS-HA

## 2. STRUCTURE PAGE
- Titre NAS + ID
- Onglets navigation
- Section métriques
- Section partitions avec tableau
