# Cahier des Charges : bare-metal/nasha - Modal modifier description

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier la description
- **Type** : Formulaire édition
- **Objectif** : Modifier la description d'une partition

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Textarea description
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x300px)
- **OdsTextarea** : Description (80px hauteur)
- **OdsButton** : Annuler (outline), Enregistrer (primary)

## 4. CHAMPS FORMULAIRE

### Description
| Propriété | Valeur |
|-----------|--------|
| Label | Description de la partition |
| Aide | 255 caractères maximum |
| Type | Textarea |
| Valeur | Partition de données pour le site web principal |

## 5. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Enregistrer | Primary | Droite |

## 6. APIS & ENDPOINTS
- PUT /dedicated/nasha/{serviceName}/partition/{partitionName}
  - description: string
