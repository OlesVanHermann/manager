# Cahier des Charges : bare-metal/nasha - Liste simple

## 1. VUE D'ENSEMBLE
- **Page** : Liste NAS-HA
- **Type** : Page listing simplifié
- **Objectif** : Liste des NAS-HA avec utilisation

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre + description
- Tableau avec barres de progression

## 3. COMPOSANTS UI
- **OdsDatagrid** : Tableau
- **OdsProgressBar** : Utilisation
- **OdsBadge** : Statut (Actif)
- **OdsLink** : ID cliquable
