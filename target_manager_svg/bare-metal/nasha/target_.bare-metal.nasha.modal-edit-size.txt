# Cahier des Charges : bare-metal/nasha - Modal modifier taille

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier la taille de la partition
- **Type** : Formulaire édition
- **Objectif** : Redimensionner une partition

## 2. STRUCTURE MODAL
- Header avec titre (incluant nom partition)
- Champ taille avec spinner
- Infos contextuelles
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x330px)
- **OdsInput** : Taille (number avec unité)
- **OdsSpinner** : Boutons ▲/▼
- **OdsText** : Infos contextuelles
- **OdsButton** : Fermer (outline), Modifier la taille (primary)

## 4. CONTENU HEADER
| Élément | Valeur |
|---------|--------|
| Titre | Modifier la taille de la partition partition-web |

## 5. CHAMPS FORMULAIRE

### Taille
| Propriété | Valeur |
|-----------|--------|
| Label | Taille de la partition partition-web |
| Aide | Taille autorisée : min 10GB, max 2500GB |
| Type | Number avec spinner |
| Unité | GB |
| Valeur | 1000 |

## 6. INFOS CONTEXTUELLES
| Info | Valeur |
|------|--------|
| Taille actuelle | 500 GB |
| Espace disponible | 2000 GB |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Fermer | Outline | Gauche |
| Modifier la taille | Primary | Droite |

## 8. APIS & ENDPOINTS
- PUT /dedicated/nasha/{serviceName}/partition/{partitionName}
  - size: number (GB)
