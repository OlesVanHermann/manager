# Cahier des Charges : bare-metal/nasha - Modal création partition

## 1. VUE D'ENSEMBLE
- **Modal** : Créer une partition
- **Type** : Formulaire création
- **Objectif** : Créer une nouvelle partition sur le NAS

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Séparateur
- Formulaire 4 champs
- Séparateur
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x630px)
- **OdsInput** : Nom, taille, description
- **OdsRadio** : Protocole
- **OdsButton** : Annuler (outline), Créer (primary)

## 4. CHAMPS FORMULAIRE

### Nom de la partition
| Propriété | Valeur |
|-----------|--------|
| Label | Nom de la partition |
| Aide | Entre 1 et 50 caractères alphanumériques, tirets et underscores autorisés |
| Placeholder | ma-nouvelle-partition |
| Requis | Oui |

### Taille
| Propriété | Valeur |
|-----------|--------|
| Label | Taille (Go) |
| Aide | Entre 10 Go et 1500 Go disponibles |
| Type | Number |
| Unité | Go |
| Exemple | 100 |
| Requis | Oui |

### Description
| Propriété | Valeur |
|-----------|--------|
| Label | Description (optionnel) |
| Aide | Maximum 255 caractères |
| Placeholder | Description de la partition... |
| Requis | Non |

### Protocole
| Propriété | Valeur |
|-----------|--------|
| Label | Protocole |
| Type | Radio |
| Options | NFS, CIFS, NFS + CIFS |
| Par défaut | NFS |

## 5. OPTIONS PROTOCOLE
### NFS
| Champ | Valeur |
|-------|--------|
| Label | NFS |
| Description | Protocole natif Linux/Unix. Recommandé pour les environnements homogènes Unix. |

### CIFS
| Champ | Valeur |
|-------|--------|
| Label | CIFS |
| Description | Protocole compatible Windows. Recommandé pour les environnements hétérogènes ou Windows. |

### NFS + CIFS
| Champ | Valeur |
|-------|--------|
| Label | NFS + CIFS |
| Description | Les deux protocoles activés. Flexibilité maximale. |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Créer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /dedicated/nasha/{serviceName}/partition
  - partitionName: string
  - size: number (Go)
  - description: string
  - protocol: enum (NFS, CIFS, NFS_CIFS)
