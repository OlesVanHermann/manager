# Cahier des Charges : bare-metal/load-balancer - Onglet vRack

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec statut et datagrid
- **Contexte** : Gestion Load Balancer
- **Objectif** : Gérer la connexion au réseau privé vRack et les réseaux privés

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsCard** : Statut vRack
- **OdsDatagrid** : Liste des réseaux privés
- **OdsBadge** : Fermes associées
- **OdsButton** : Désactiver vRack, Ajouter un réseau
- **ActionMenu** : Menu contextuel (⋮)

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Réseau privé (vRack)" |
| Description ligne 1 | "Connectez votre load balancer à votre réseau privé vRack." |
| Description ligne 2 | "Il vous reste {n} réseaux privés disponibles." |

## 5. CARTE STATUT VRACK
| Propriété | Valeur |
|-----------|--------|
| Largeur | 920px |
| Hauteur | 80px |
| Border radius | 8px |
| Border | #E5E7EB |

### Contenu carte
| Élément | Valeur |
|---------|--------|
| Label | "Statut vRack" |
| Valeur | "Actif" ou "Inactif" |
| ID | "(pn-12345)" en gris |
| Bouton | "Désactiver vRack" (bordure grise) |

## 6. BOUTON AJOUTER RÉSEAU
| Propriété | Valeur |
|-----------|--------|
| Label | "+ Ajouter un réseau" |
| Style | Secondary (bordure grise) |
| Largeur | 180px |
| Hauteur | 40px |
| Position | Au-dessus du datagrid |

## 7. DATAGRID RÉSEAUX PRIVÉS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 920px |
| Hauteur | 280px |

### Colonnes
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Réseau privé | 170px | Nom du réseau |
| VLAN ID | 80px | Identifiant VLAN |
| Sous-réseau | 150px | CIDR (ex: 10.0.1.0/24) |
| IP NAT | 120px | Adresse IP NAT |
| Fermes associées | 200px | Badges des fermes |
| Actions | 80px | Menu ⋮ |

### Badges fermes associées
| Propriété | Valeur |
|-----------|--------|
| Fond | #EFF6FF |
| Couleur texte | #1E40AF |
| Border radius | 11px |
| Hauteur | 22px |
| Multiple | Affichage horizontal des badges |

### ActionMenu (menu ⋮)
| Option | Couleur | Action |
|--------|---------|--------|
| Modifier | Noir #1F2937 | Ouvre la modal de modification |
| Supprimer | Rouge #DC2626 | Ouvre la modal de confirmation |

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "{n} réseaux privés" |
| Position | Bas du datagrid |

## 9. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/vrack/status - Statut vRack
- GET /ipLoadbalancing/{serviceName}/vrack/network - Liste réseaux privés
- POST /ipLoadbalancing/{serviceName}/vrack/network - Ajouter réseau
- PUT /ipLoadbalancing/{serviceName}/vrack/network/{vrackNetworkId} - Modifier réseau
- DELETE /ipLoadbalancing/{serviceName}/vrack/network/{vrackNetworkId} - Supprimer réseau
- POST /ipLoadbalancing/{serviceName}/vrack/attach - Activer vRack
- POST /ipLoadbalancing/{serviceName}/vrack/detach - Désactiver vRack
