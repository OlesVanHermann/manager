# Cahier des Charges : bare-metal/load-balancer - Onglet Certificats SSL

## 1. VUE D'ENSEMBLE
- **Page** : Certificats SSL du Load Balancer
- **Type** : Onglet tableau
- **Objectif** : Gérer les certificats SSL

## 2. STRUCTURE PAGE
- Breadcrumb avec retour
- Titre "Certificats SSL"
- Onglets navigation
- Boutons actions
- Tableau certificats
- Bandeau info Let's Encrypt

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsButton** : Ajouter, Commander Let's Encrypt
- **OdsDatagrid** : Tableau certificats
- **OdsBadge** : État (Actif, Expire bientôt)
- **OdsActionMenu** : Menu par ligne
- **OdsMessage** : Info Let's Encrypt

## 4. BOUTONS TOOLBAR
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter un certificat | Primary | Import manuel |
| Commander Let's Encrypt | Outline | Auto Let's Encrypt |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Link | CN du certificat |
| Type | Text | Wildcard, Standard |
| Émetteur | Text | Let's Encrypt, DigiCert |
| Expiration | Date | Date d'expiration |
| Frontends | Text | Frontends utilisant ce cert |
| État | Badge | Actif / Expire bientôt |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTIONMENU (⋮)
| Action | Style | Description |
|--------|-------|-------------|
| Voir les détails | Normal | Détails certificat |
| Renouveler | Normal | Force renouvellement |
| Télécharger | Normal | Export PEM |
| Supprimer | Danger | Supprime le certificat |

## 7. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ |
| Titre | Certificats Let's Encrypt |
| Texte | Les certificats Let's Encrypt sont renouvelés automatiquement 30 jours avant leur expiration. |
| Lien | En savoir plus sur la gestion SSL → |

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/ssl
- POST /ipLoadbalancing/{serviceName}/ssl
- DELETE /ipLoadbalancing/{serviceName}/ssl/{id}
