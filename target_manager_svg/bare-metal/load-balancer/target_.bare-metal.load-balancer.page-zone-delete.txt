# Cahier des Charges : bare-metal/load-balancer - Page Supprimer Zone

## 1. VUE D'ENSEMBLE
- **Type** : Page de sélection multi-zones (suppression)
- **Contexte** : Gestion Load Balancer
- **Objectif** : Supprimer des zones de déploiement du load balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 900px |
| Page hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsCard** : Cartes de sélection zone (checkbox intégrée)
- **OdsCheckbox** : Sélection zone
- **OdsMessage** : Avertissement zone principale
- **OdsButton** : Supprimer, Annuler

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Header standard | OVHcloud (bleu #000E9C) |
| Lien retour | "← Supprimer une zone" |
| Description | "Vous avez {n} zones actives. Sélectionnez les zones à supprimer." |

## 5. CONTENU

### Grille de zones
| Propriété | Valeur |
|-----------|--------|
| Disposition | Grille 3-4 colonnes |
| Espacement | 20px |

### Carte zone (format)
| Propriété | Valeur |
|-----------|--------|
| Largeur | 150px |
| Hauteur | 100px |
| Border radius | 6px |

### États des cartes (suppression)
| État | Border | Checkbox | Statut texte | Couleur texte |
|------|--------|----------|--------------|---------------|
| Supprimable | #E5E7EB | Vide | "Supprimable" | Vert #16A34A |
| Sélectionné | #DC2626 2px | ✓ rouge | "À supprimer" | Rouge #DC2626 |
| Zone principale | #E5E7EB | Gris désactivé | "Zone principale" | Gris #6B7280 |

### Avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | OdsMessage (warning) |
| Largeur | 500px |
| Hauteur | 50px |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Texte | "Au moins une zone doit rester active. La zone principale ne peut pas être supprimée." |
| Couleur texte | #92400E |

### Résumé sélection
| Propriété | Valeur |
|-----------|--------|
| Position | Sous l'avertissement |
| Texte | "{n} zone(s) à supprimer" |
| Style | 13px, font-weight 500, rouge #DC2626 |

## 6. FOOTER PAGE
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Supprimer | Primary danger | #DC2626 | Supprime les zones sélectionnées |
| Annuler | Secondary | Bordure grise | Retour au dashboard |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/zone - Zones actives
- DELETE /ipLoadbalancing/{serviceName}/zone/{name} - Supprimer une zone
