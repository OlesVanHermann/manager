# Cahier des Charges : bare-metal/load-balancer - Modal modifier alerte quota

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Load Balancer Dedicated
- **Objectif** : Modifier le seuil d'alerte de quota

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsInput** : Seuil d'alerte
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'alerte de quota" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champ Quota inclus (lecture seule)
| Propriété | Valeur |
|-----------|--------|
| Label | "Quota inclus" |
| Type | Lecture seule |
| Exemple | "5 TB" |

### Champ Seuil d'alerte
| Propriété | Valeur |
|-----------|--------|
| Label | "Seuil d'alerte (Go)" |
| Type | OdsInput (number) |
| Exemple | 4500 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary (bleu) | Enregistre le seuil |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/quota - Quota actuel
- PUT /ipLoadbalancing/{serviceName}/quota/alert - Modifier seuil d'alerte
