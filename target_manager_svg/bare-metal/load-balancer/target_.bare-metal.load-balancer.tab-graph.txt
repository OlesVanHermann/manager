# Cahier des Charges : bare-metal/load-balancer - Onglet Graphiques

## 1. VUE D'ENSEMBLE
- **Type** : Page tableau de bord avec graphiques
- **Contexte** : Load Balancer Dedicated
- **Objectif** : Visualiser les métriques de performance du load balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsText** : Titres, labels
- **OdsButton** : Sélecteur de période (group)
- **OdsCard** : Container graphiques
- **OdsChart** : Graphiques ligne et barres

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Graphiques" |

## 5. SÉLECTEUR DE PÉRIODE

### Boutons de période
| Période | État |
|---------|------|
| Jour | Sélectionné (bleu) |
| Semaine | Non sélectionné |
| Mois | Non sélectionné |
| Année | Non sélectionné |

## 6. GRAPHIQUE 1 - CONNEXIONS

### Configuration
| Propriété | Valeur |
|-----------|--------|
| Titre | "Connexions" |
| Type | Line chart |
| Couleur | Bleu (#0050D7) |

### Axes
| Axe | Valeurs |
|-----|---------|
| Y | 0, 250, 500, 750, 1000 |
| X | 00:00, 06:00, 12:00, 18:00, 24:00 |

### Configuration
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bande passante (Mbps)" |
| Type | Line chart (2 séries) |

### Séries
| Série | Couleur | Label |
|-------|---------|-------|

| Sortant | Orange (#F59E0B) | "Sortant" |

### Axes
| Axe | Valeurs |
|-----|---------|
| Y | 0, 25, 50, 75, 100 |

## 8. GRAPHIQUE 3 - REQUÊTES

### Configuration
| Propriété | Valeur |
|-----------|--------|

| Type | Bar chart |
| Couleur | Bleu (#0050D7, opacity 0.7) |

## 9. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/stats - Statistiques générales
- GET /ipLoadbalancing/{serviceName}/stats/connections - Graphique connexions
- GET /ipLoadbalancing/{serviceName}/stats/bandwidth - Graphique bande passante
- GET /ipLoadbalancing/{serviceName}/stats/requests - Graphique requêtes
