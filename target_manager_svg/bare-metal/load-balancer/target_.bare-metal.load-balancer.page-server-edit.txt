# Cahier des Charges : bare-metal/load-balancer - Page Modifier Serveur

## 1. VUE D'ENSEMBLE
- **Type** : Page de formulaire complet
- **Contexte** : Gestion Load Balancer
- **Objectif** : Modifier la configuration d'un serveur backend

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1000px |
| Page hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsInput** : Nom, Adresse IP, Cookie, Chain SSL
- **OdsQuantity** : Port, Poids
- **OdsCheckbox** : Options avancées
- **OdsSelect** : Proxy Protocol
- **OdsTextarea** : Chaîne certificats
- **OdsAccordion** : Options avancées
- **OdsButton** : Modifier, Annuler

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le serveur {serverId}" |

## 5. CONTENU

### Champs principaux

#### Nom d'affichage
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom d'affichage" |
| Type | OdsInput |
| Largeur | 400px |
| Exemple | server-web-01 |

#### Adresse IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP" |
| Type | OdsInput |
| Largeur | 400px |
| Exemple | 10.0.1.10 |

#### Port
| Propriété | Valeur |
|-----------|--------|
| Label | "Port" |
| Type | OdsInput |

| Propriété | Valeur |
|-----------|--------|

| Option | État par défaut |

| Propriété | Valeur |
|-----------|--------|
| Label | "Version Proxy Protocol" |
| Type | OdsSelect |
| Largeur | 200px |
| Options | v1, v2, v2-ssl, v2-ssl-cn |

## 6. FOOTER PAGE
| Bouton | Style | Action |
|--------|-------|--------|
| Modifier | Primary (#0050D7) | Enregistre les modifications |
| Annuler | Secondary (bordure grise) | Retour à la liste |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/farm/{farmId}/server/{serverId} - Détails serveur
- PUT /ipLoadbalancing/{serviceName}/farm/{farmId}/server/{serverId} - Modifier serveur
