# Cahier des Charges : bare-metal/load-balancer - Onglet Tâches

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec datagrid
- **Contexte** : Gestion Load Balancer
- **Objectif** : Afficher l'historique des tâches effectuées

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsDatagrid** : Liste des tâches
- **OdsProgress** : Barre de progression
- **OdsText** : Statut coloré
- **ActionMenu** : Menu contextuel (⋮)

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Tâches" |
| Description | "Historique des tâches effectuées sur votre load balancer." |

## 5. DATAGRID
| Propriété | Valeur |
|-----------|--------|
| Largeur | 920px |
| Hauteur | 320px |

### Colonnes
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Date de création | 170px | Format DD/MM/YYYY HH:mm:ss |
| Tâche | 200px | Nom de l'action (ex: addFrontend) |
| Progression | 180px | Barre de progression avec % |
| Statut | 100px | Texte coloré selon état |
| Actions | 80px | Menu ⋮ |

### Barre de progression
| Propriété | Valeur |
|-----------|--------|
| Largeur | 180px |
| Hauteur | 16px |
| Fond | #E5E7EB |
| Border radius | 8px |
| Label | Pourcentage à droite |

### Couleurs progression par statut
| Statut | Couleur barre | Fond |
|--------|---------------|------|
| done | #16A34A | #DCFCE7 |
| running | #0050D7 | #E5E7EB |
| error | #DC2626 | #FEE2E2 |

### Statuts tâche
| Statut | Couleur | Texte |
|--------|---------|-------|
| done | #16A34A | "done" |
| running | #0050D7 | "running" |
| error | #DC2626 | "error" |
| pending | #6B7280 | "pending" |
| cancelled | #6B7280 | "cancelled" |

### Types de tâches
| Tâche | Description |
|-------|-------------|
| addFrontend | Ajout d'un frontend |
| addFarm | Ajout d'une ferme |
| addServer | Ajout d'un serveur |
| deleteServer | Suppression d'un serveur |
| applyConfiguration | Application de la configuration |
| orderSsl | Commande certificat SSL |
| addZone | Ajout d'une zone |

### ActionMenu (menu ⋮)
| Option | Action |
|--------|--------|
| Voir les détails | Affiche le détail de la tâche |

## 6. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "{n} tâches" |
| Position | Bas du datagrid |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/task - Liste des tâches
- GET /ipLoadbalancing/{serviceName}/task/{taskId} - Détails d'une tâche
