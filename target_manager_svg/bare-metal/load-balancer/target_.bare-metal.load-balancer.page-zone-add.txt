# Cahier des Charges : bare-metal/load-balancer - Page Ajouter Zone

## 1. VUE D'ENSEMBLE
- **Type** : Page de sélection multi-zones
- **Contexte** : Gestion Load Balancer
- **Objectif** : Ajouter des zones de déploiement au load balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 900px |
| Page hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsCard** : Cartes de sélection zone (checkbox intégrée)
- **OdsCheckbox** : Sélection zone
- **OdsText** : Nom zone, ville, statut
- **OdsButton** : Ajouter, Annuler

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Header standard | OVHcloud (bleu #000E9C) |
| Lien retour | "← Ajouter une zone" |
| Description | "Sélectionnez les zones où déployer votre load balancer." |

## 5. CONTENU

### Grille de zones
| Propriété | Valeur |
|-----------|--------|
| Disposition | Grille 4 colonnes |
| Espacement | 20px |
| Max lignes | 2 |

### Carte zone (format)
| Propriété | Valeur |
|-----------|--------|
| Largeur | 150px |
| Hauteur | 100px |
| Border radius | 6px |
| Border | #E5E7EB (normal), #0050D7 2px (sélectionné) |

### Éléments par carte
| Élément | Position | Style |
|---------|----------|-------|
| Checkbox | Haut gauche (10,10) | 16x16px, coché = #0050D7 |
| Code zone | Centre | 14px, font-weight 500 |
| Ville | Centre (sous code) | 11px, gris #6B7280 |
| Statut | Bas centre | 10px, vert #16A34A ou gris |

### États des cartes
| État | Border | Checkbox | Statut texte |
|------|--------|----------|--------------|
| Disponible | #E5E7EB | Vide | "Disponible" (vert) |
| Sélectionné | #0050D7 2px | ✓ bleu | "Disponible" (vert) |
| Déjà actif | #E5E7EB | Gris désactivé | "Déjà actif" (gris) |

### Zones exemples
| Code | Ville |
|------|-------|
| GRA | Gravelines |
| RBX | Roubaix |
| SBG | Strasbourg |
| WAW | Varsovie |
| BHS | Beauharnois |
| SGP | Singapour |

### Résumé sélection
| Propriété | Valeur |
|-----------|--------|
| Position | Sous la grille |
| Texte | "{n} zones sélectionnées" |
| Style | 13px, noir #1F2937 |

## 6. FOOTER PAGE
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter | Primary (#0050D7) | Ajoute les zones sélectionnées |
| Annuler | Secondary (bordure grise) | Retour au dashboard |

## 7. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/availableZones - Zones disponibles
- GET /ipLoadbalancing/{serviceName}/zone - Zones actives actuelles
- POST /ipLoadbalancing/{serviceName}/zone - Ajouter une zone
