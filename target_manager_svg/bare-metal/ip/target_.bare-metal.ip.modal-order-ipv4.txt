# Cahier des Charges : bare-metal/ip - Commander IPv4

## 1. VUE D'ENSEMBLE
- **Page** : Commander des adresses IPv4
- **Type** : Modal wizard étapes
- **Objectif** : Commander des IPs failover

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Indicateur d'étapes (1-4)
- Contenu dynamique selon étape
- Footer avec boutons navigation

## 3. COMPOSANTS UI
- **OdsModal** : Modal grande
- **OdsStepper** : Indicateur 4 étapes
- **OdsSelect** : Type de service, Service
- **OdsMessage** : Info (bleu)
- **OdsButton** : Annuler (outline), Suivant (primary)

## 4. ÉTAPES
| Étape | Nom | Description |
|-------|-----|-------------|
| 1 | Service | Sélection du service de destination |
| 2 | Quantité | Nombre d'IPs à commander |
| 3 | Pays | Géolocalisation des IPs |
| 4 | Résumé | Récapitulatif et validation |

## 5. ÉTAPE 1 : SERVICE
### Type de service
| Propriété | Valeur |
|-----------|--------|
| Label | Type de service * |
| Type | Select |
| Options | Serveurs dédiés, VPS, Hosted Private Cloud, Public Cloud, Additional IP (parking) |

### Service
| Propriété | Valeur |
|-----------|--------|
| Label | Service * |
| Type | Select |
| Placeholder | Sélectionnez un service... |

### Info
| Type | Info (bleu) |
|------|-------------|
| Icône | ℹ️ |
| Texte | Les IP failover peuvent être déplacées entre services compatibles. |

## 6. ÉTAPE 2 : QUANTITÉ
| Propriété | Valeur |
|-----------|--------|
| Label | Nombre d'adresses IP |
| Type | Quantity (1-256) |

## 7. ÉTAPE 3 : PAYS
| Propriété | Valeur |
|-----------|--------|
| Label | Pays de géolocalisation |
| Type | Select avec drapeaux |
| Options | France, Allemagne, UK, etc. |

## 8. ÉTAPE 4 : RÉSUMÉ
| Champ | Valeur exemple |
|-------|----------------|
| Service | ns1234567.ip-192-168-1.eu |
| Quantité | 4 IPs |
| Pays | France |
| Prix | XX,XX € HT/mois |

## 9. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Suivant / Commander | Primary | Droite |

## 10. APIS & ENDPOINTS
- GET /order/cart/{cartId}/ip
  - Options de commande IP
- POST /order/cart/{cartId}/ip
  - Ajouter IP au panier
- POST /order/cart/{cartId}/checkout
  - Finaliser la commande
