# Cahier des Charges : bare-metal/ip - Modal modifier bloc IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal avec onglets
- **Contexte** : Gestion des blocs IP
- **Objectif** : Modifier les informations d'un bloc IP ou son organisation

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsTabs** : Navigation entre onglets
- **OdsText** : Labels, description
- **OdsInput** : Nom du bloc
- **OdsTextarea** : Description
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | Blanc avec bordure grise |
| Description | "Modifier les informations du bloc IP ou l'associer à une organisation." |
| Bouton fermer | × |

## 5. ONGLETS

### Liste des onglets
| Onglet | Label | État |
|--------|-------|------|
| 1 | "Bloc réseau" | Actif (bleu) |
| 2 | "Organisation" | Inactif |

### Style onglet actif
| Propriété | Valeur |
|-----------|--------|
| Couleur texte | Bleu (#0050D7) |
| Indicateur | Barre bleue 4px dessous |
| Fond | Blanc |

### Style onglet inactif
| Propriété | Valeur |
|-----------|--------|
| Couleur texte | Gris (#6B7280) |
| Fond | Gris clair (#F9FAFB) |

## 6. ONGLET BLOC RÉSEAU

### Champ Nom du bloc
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du bloc" |
| Type | OdsInput |
| Exemple | "PRODUCTION-WEB" |

### Champ Description
| Propriété | Valeur |
|-----------|--------|
| Label | "Description" |
| Type | OdsTextarea |
| Hauteur | 100px |
| Exemple | "Serveurs web de production pour le site principal et les API clients." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary (bleu #0050D7) | Sauvegarde |

## 8. APIS & ENDPOINTS
- GET /ip/{ip}/block - Informations du bloc
- PUT /ip/{ip}/block - Modifier le bloc
- GET /ip/organisation - Liste des organisations
- PUT /ip/{ip}/organisation - Associer organisation
