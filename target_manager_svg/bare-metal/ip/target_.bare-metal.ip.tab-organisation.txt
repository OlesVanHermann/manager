# Cahier des Charges : bare-metal/ip - Organisations RIPE

## 1. VUE D'ENSEMBLE
- **Page** : Organisations RIPE
- **Type** : Page liste
- **Objectif** : Gérer les organisations RIPE associées aux IPs

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb (IP › Organisations RIPE)
- Titre
- Encart info
- Bouton créer
- Tableau des organisations
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : IP › Organisations RIPE
- **OdsMessage** : Info explicatif (bleu)
- **OdsButton** : Créer une organisation (primary)
- **OdsDatagrid** : Tableau des organisations
- **OdsBadge** : Actif (vert)
- **OdsActionMenu** : Menu actions par ligne
- **OdsPagination** : Navigation résultats

## 4. ENCART INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte 1 | Les organisations RIPE permettent d'associer vos coordonnées aux blocs IP dans la base RIPE. |
| Texte 2 | Ces informations sont visibles publiquement via les outils whois. |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| ID Organisation | Link | ORG-XXXXX-RIPE |
| Nom | Text | Nom de l'organisation |
| Adresse | Text | Adresse postale |
| Pays | Text + Flag | Emoji drapeau + nom pays |
| État | Badge | Actif (vert) |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTION MENU
| Option | Style |
|--------|-------|
| Voir détails | Normal |
| Modifier | Normal |
| Supprimer | Rouge (danger) |

## 7. BOUTON
| Bouton | Style |
|--------|-------|
| Créer une organisation | Primary |

## 8. APIS & ENDPOINTS
- GET /me/ipOrganisation
  - Liste des organisations RIPE
- POST /me/ipOrganisation
  - Créer une organisation
- GET /me/ipOrganisation/{organisationId}
  - Détails d'une organisation
- PUT /me/ipOrganisation/{organisationId}
  - Modifier une organisation
- DELETE /me/ipOrganisation/{organisationId}
  - Supprimer une organisation
