# Cahier des Charges : bare-metal/ip - Modal agréger bloc BYOIP

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire avec prévisualisation
- **Contexte** : BYOIP (Bring Your Own IP)
- **Objectif** : Agréger plusieurs blocs IP en un bloc parent plus grand

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, descriptions
- **OdsSelect** : Taille bloc parent
- **OdsCard** : Zone prévisualisation
- **OdsButton** : Annuler, Agréger

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Agréger le bloc {bloc_ip}" |
| Exemple | "Agréger le bloc 192.0.2.0/24" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sélectionnez la taille du bloc parent :" |

## 6. FORMULAIRE

### Champ Bloc parent
| Propriété | Valeur |
|-----------|--------|
| Type | OdsSelect |
| Exemple | "192.0.2.0/23" |

## 7. PRÉVISUALISATION
| Propriété | Valeur |
|-----------|--------|
| Fond | Gris (#F3F4F6) |
| Titre | "Configuration résultante :" (gras) |
| Sous-titre | "Blocs enfants qui seront agrégés :" |

### Liste des blocs
| Propriété | Valeur |
|-----------|--------|
| Icône | Cercle bleu (#0050D7) |
| Format | Adresse du bloc (ex: "192.0.2.0/24") |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Agréger | Primary (bleu #0050D7) | Lance l'agrégation |

## 9. APIS & ENDPOINTS
- GET /ip/byoip/{ip} - Informations bloc BYOIP
- POST /ip/byoip/{ip}/aggregate - Agréger les blocs
