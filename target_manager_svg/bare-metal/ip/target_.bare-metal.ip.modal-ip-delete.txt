# Cahier des Charges : bare-metal/ip - Modal supprimer IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion des adresses IP
- **Objectif** : Confirmer la suppression définitive d'une adresse IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement danger
- **OdsText** : IP
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'adresse IP" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (rouge) |
| Fond | #FEE2E2 |
| Titre | "Attention !" (gras) |
| Texte | "Cette action est irréversible. L'adresse IP" |
| IP | "{ip}" (gras) |
| Suite | "sera définitivement supprimée." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime l'IP |

## 7. APIS & ENDPOINTS
- DELETE /ip/{ip} - Supprimer l'adresse IP
