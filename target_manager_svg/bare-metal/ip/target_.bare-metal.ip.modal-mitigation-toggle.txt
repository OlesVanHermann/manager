# Cahier des Charges : bare-metal/ip - Modal activer mitigation

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Protection Anti-DDoS
- **Objectif** : Activer/Désactiver la mitigation permanente sur une IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Question, IP
- **OdsMessage** : Information mitigation
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer la mitigation permanente" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Question
| Propriété | Valeur |
|-----------|--------|
| Texte | "Activer la mitigation permanente pour l'IP :" |

### IP affichée
| Propriété | Valeur |
|-----------|--------|
| Format | IP en bleu (#0050D7), gras |
| Exemple | "51.210.xxx.100" |

## 6. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Ligne 1 | "La mitigation permanente protège votre IP contre les" |
| Ligne 2 | "attaques DDoS de manière continue." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Activer | Primary (bleu #0050D7) | Active la mitigation |

## 8. APIS & ENDPOINTS
- PUT /ip/{ip}/mitigation - Activer/Désactiver la mitigation
