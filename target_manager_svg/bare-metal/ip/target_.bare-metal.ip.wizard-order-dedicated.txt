# Cahier des Charges : bare-metal/ip - Wizard commander IP dédiées

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire de commande
- **Contexte** : Gestion IP - Serveurs dédiés
- **Objectif** : Commander un bloc d'adresses IP pour un serveur dédié

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, labels
- **OdsSelect** : Nombre d'IP, pays, organisation
- **OdsCard** : Récapitulatif prix
- **OdsButton** : Annuler, Commander

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander des adresses IP" |

## 5. FORMULAIRE

### Champ Nombre d'IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Nombre d'IP" |
| Type | OdsSelect |
| Options | 1 IP (/32), 4 IP (/30), 8 IP (/29), 16 IP (/28), etc. |
| Exemple | "8 adresses IP (/29)" |

### Champ Pays de géolocalisation
| Propriété | Valeur |
|-----------|--------|
| Label | "Pays de géolocalisation" |
| Type | OdsSelect |
| Options | France, Allemagne, Royaume-Uni, etc. |
| Exemple | "France" |

### Champ Organisation RIPE
| Propriété | Valeur |
|-----------|--------|
| Label | "Organisation RIPE" |
| Type | OdsSelect |
| Format | "{orgId} ({orgName})" |
| Exemple | "ORG-MA123 (Mon Organisation)" |

## 6. RÉCAPITULATIF

### Carte de prix
| Champ | Valeur |
|-------|--------|
| Titre | "Récapitulatif" |
| Description | "{nbIp} adresses IP (bloc /{cidr}) - {country}" |
| Prix | "{prix} € HT/mois" |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour |
| Commander | Primary (bleu) | Passe la commande |

## 8. APIS & ENDPOINTS
- GET /ip/organization - Liste des organisations RIPE
- GET /order/catalog/formatted/ip - Catalogue des blocs IP
- POST /order/cart/{cartId}/ip - Ajouter au panier
