# Cahier des Charges : bare-metal/ip - Modal ajouter reverse IPv6

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Ajouter un enregistrement reverse DNS pour une adresse IPv6

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Information
- **OdsText** : Labels
- **OdsInput** : Adresse IPv6, Reverse DNS
- **OdsButton** : Annuler, Valider

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un reverse IPv6" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Texte | "Entrez l'adresse IPv6 et le reverse DNS associé." |

## 6. FORMULAIRE

### Champ Bloc IP (lecture seule)
| Propriété | Valeur |
|-----------|--------|
| Label | "Bloc IP" |
| Type | OdsInput disabled |
| Fond | Gris (#F3F4F6) |
| Exemple | "2001:41d0:xxxx::/56" |

### Champ Adresse IPv6 *
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IPv6 *" |
| Type | OdsInput |
| Placeholder | "2001:41d0:xxxx::1" |
| Requis | Oui |

### Champ Reverse DNS *
| Propriété | Valeur |
|-----------|--------|
| Label | "Reverse DNS *" |
| Type | OdsInput |
| Placeholder | "mail.example.com" |
| Requis | Oui |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Valider | Primary (bleu) | Crée le reverse |

## 8. APIS & ENDPOINTS
- POST /ip/{ip}/reverse - Créer reverse DNS IPv6
