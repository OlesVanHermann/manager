# Cahier des Charges : bare-metal/ip - Modal parker IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Gestion des adresses IP
- **Objectif** : Dissocier une adresse IP de son service actuel (parking)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description, labels, valeurs
- **OdsMessage** : Information
- **OdsButton** : Annuler, Parker

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Parker l'adresse IP" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Cette action va dissocier l'IP de son service actuel." |

### Informations affichées
| Champ | Label | Exemple |
|-------|-------|---------|
| IP | "Adresse IP" | "51.210.xxx.100" (gras) |
| Service | "Service actuel" | "ns1234567.ip-51-210-xxx.eu" (bleu) |

## 6. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Texte | "L'IP pourra être réassignée à un autre service ultérieurement." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Parker | Primary (bleu #0050D7) | Parke l'IP |

## 8. APIS & ENDPOINTS
- POST /ip/{ip}/park - Parker l'IP
