# Cahier des Charges : bare-metal/ip - Reverse DNS

## 1. VUE D'ENSEMBLE
- **Page** : Reverse DNS - Bloc IP
- **Type** : Page détail onglet
- **Objectif** : Configurer les reverse DNS des IPs

## 2. STRUCTURE PAGE
- Header OVHcloud + sidebar
- Breadcrumb (IP › Bloc › Reverse DNS)
- Titre
- Encart info
- Bouton ajouter
- Tableau des reverses
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : IP › 192.168.1.0/24 › Reverse DNS
- **OdsMessage** : Info explicatif (bleu)
- **OdsButton** : Ajouter un reverse DNS (primary)
- **OdsDatagrid** : Tableau des reverses
- **OdsBadge** : Configuré (vert), Non configuré (jaune)
- **OdsActionMenu** : Menu actions par ligne
- **OdsPagination** : Navigation résultats

## 4. ENCART INFO

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Adresse IP | Link | Adresse IP |
| Reverse DNS | Text | Nom de domaine ou "Non configuré" |
| État | Badge | Configuré (vert), Non configuré (jaune) |
| Actions | ActionMenu | Menu ⋮ |

## 6. ACTION MENU
| Option | Style |
|--------|-------|
| Modifier | Normal |
| Supprimer | Rouge (danger) |

## 7. BOUTON
| Bouton | Style |
|--------|-------|
| Ajouter un reverse DNS | Primary |

## 8. APIS & ENDPOINTS
- GET /ip/{ip}/reverse
  - Liste des reverse DNS
- POST /ip/{ip}/reverse
  - Ajouter reverse DNS
  - Paramètres : ipReverse (string), reverse (string)
- DELETE /ip/{ip}/reverse/{ipReverse}
  - Supprimer reverse DNS
