# Cahier des Charges : bare-metal/ip - Modal supprimer MAC virtuelle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion des adresses IP
- **Objectif** : Confirmer la suppression d'une adresse MAC virtuelle

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Confirmation danger
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la MAC virtuelle" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. MESSAGE DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (rouge) |
| Fond | #FEE2E2 |
| Ligne 1 | "Êtes-vous sûr de vouloir supprimer la MAC virtuelle" |
| MAC | "{mac_address}" (gras) |
| Ligne 2 | "associée à l'IP {ip} ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime la MAC |

## 7. APIS & ENDPOINTS
- DELETE /ip/{ip}/virtualMac/{macAddress} - Supprimer la MAC virtuelle
