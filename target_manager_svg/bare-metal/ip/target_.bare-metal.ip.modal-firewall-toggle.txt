# Cahier des Charges : bare-metal/ip - Modal activer firewall

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Firewall IP
- **Objectif** : Activer/Désactiver le firewall sur une adresse IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Question, IP
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer le firewall" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Question
| Propriété | Valeur |
|-----------|--------|
| Texte | "Voulez-vous activer le firewall pour l'adresse IP :" |

### IP affichée
| Propriété | Valeur |
|-----------|--------|
| Format | IP en bleu (#0050D7), gras |
| Exemple | "51.210.xxx.100" |

## 6. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Ligne 1 | "Une fois activé, vous devrez configurer les règles de filtrage" |
| Ligne 2 | "pour autoriser le trafic souhaité." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Activer | Primary (bleu #0050D7) | Active le firewall |

## 8. APIS & ENDPOINTS
- PUT /ip/{ip}/firewall - Activer/Désactiver le firewall
