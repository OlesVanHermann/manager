# Cahier des Charges : bare-metal/ip - Modal commander bloc IPv6

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Commande de blocs IP
- **Objectif** : Commander un bloc IPv6 pour un service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels
- **OdsSelect** : Service, taille bloc
- **OdsMessage** : Information gratuit
- **OdsButton** : Annuler, Commander

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander un bloc IPv6" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champ Service associé *
| Propriété | Valeur |
|-----------|--------|
| Label | "Service associé *" |
| Type | OdsSelect |
| Exemple | "ns1234567.ip-51-210-xxx.eu" |
| Requis | Oui |

### Champ Taille du bloc
| Propriété | Valeur |
|-----------|--------|
| Label | "Taille du bloc" |
| Type | OdsSelect |
| Options | /64, /56, /48 |
| Défaut | /64 |

## 6. MESSAGE D'INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Success (vert) |
| Fond | #DCFCE7 |
| Titre | "Gratuit" (gras) |
| Texte | "Un bloc /64 IPv6 offre 18 446 744 073 709 551 616 adresses disponibles." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Commander | Primary (bleu #0050D7) | Commande le bloc |

## 8. APIS & ENDPOINTS
- GET /dedicated/server - Liste des services
- POST /ip/ipv6/order - Commander un bloc IPv6
