# Cahier des Charges : bare-metal/ip - Page IPs vRack

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **Contexte** : Gestion des IPs vRack
- **Objectif** : Gérer les blocs IP associés aux vRacks

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 480px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsSelect** : Sélection vRack
- **OdsDatagrid** : Tableau des blocs IP
- **OdsActionMenu** : Actions par ligne
- **OdsPagination** : Pagination

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "IPs dans vRack" |
| Fond | Blanc avec bordure |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Gérez les blocs IP associés à vos vRacks pour la communication privée entre services." |

## 6. SÉLECTEUR VRACK
| Propriété | Valeur |
|-----------|--------|
| Label | "vRack :" |
| Type | OdsSelect |
| Exemple | "pn-123456" |

## 7. TABLEAU DES BLOCS IP

### Colonnes
| Colonne | Type | Description |
|---------|------|-------------|
| Bloc IP | Texte | CIDR (ex: 10.0.0.0/24) |
| Zone | Texte | Datacenter (rbx, gra, sbg) |
| Gateway | Texte | IP de la gateway |
| VLAN | Texte | ID du VLAN (0-4095) |
| Actions | Menu | ⋮ |

### Actions disponibles (menu ⋮)
- Modifier
- Déplacer vers un autre vRack
- Supprimer

## 8. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Format | "{count} blocs IP" |

## 9. APIS & ENDPOINTS
- GET /vrack - Liste des vRacks
- GET /vrack/{serviceName}/ip - Liste des blocs IP d'un vRack
- PUT /vrack/{serviceName}/ip/{ip} - Modifier un bloc
- DELETE /vrack/{serviceName}/ip/{ip} - Supprimer un bloc
