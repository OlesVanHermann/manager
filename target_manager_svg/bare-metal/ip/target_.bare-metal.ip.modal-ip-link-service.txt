# Cahier des Charges : bare-metal/ip - Modal lier IP à un service

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des adresses IP
- **Objectif** : Lier une adresse IP à un service (serveur dédié, VPS, etc.)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, IP
- **OdsSelect** : Type de service, service
- **OdsButton** : Annuler, Lier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Lier l'IP à un service" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS AFFICHÉES
| Champ | Label | Exemple |
|-------|-------|---------|
| IP | "Adresse IP" | "51.210.xxx.100" (gras) |

## 6. FORMULAIRE

### Champ Type de service *
| Propriété | Valeur |
|-----------|--------|
| Label | "Type de service *" |
| Type | OdsSelect |
| Options | Serveur dédié, VPS, Public Cloud |
| Requis | Oui |

### Champ Service *
| Propriété | Valeur |
|-----------|--------|
| Label | "Service *" |
| Type | OdsSelect |
| Exemple | "ns1234567.ip-51-210-xxx.eu" |
| Requis | Oui |
| Note | Liste filtrée selon le type sélectionné |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Lier | Primary (bleu #0050D7) | Lie l'IP au service |

## 8. APIS & ENDPOINTS
- GET /dedicated/server - Liste des serveurs dédiés
- GET /vps - Liste des VPS
- POST /ip/{ip}/move - Lier l'IP au service
