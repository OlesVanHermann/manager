# Cahier des Charges : bare-metal/ip - Modal déplacer vers vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion des blocs IP vRack
- **Objectif** : Déplacer un bloc IP vers un autre vRack

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs
- **OdsSelect** : vRack destination
- **OdsMessage** : Avertissement
- **OdsButton** : Annuler, Déplacer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Déplacer vers un vRack" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS AFFICHÉES
| Champ | Label | Exemple |
|-------|-------|---------|
| Bloc | "Bloc IP" | "10.0.0.0/24" (gras) |
| vRack actuel | "vRack actuel" | "pn-123456" |

## 6. FORMULAIRE

### Champ vRack de destination *
| Propriété | Valeur |
|-----------|--------|
| Label | "vRack de destination *" |
| Type | OdsSelect |
| Exemple | "pn-789012" |
| Requis | Oui |

## 7. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Texte | "Le déplacement peut prendre quelques minutes." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Déplacer | Primary (bleu #0050D7) | Lance le déplacement |

## 9. APIS & ENDPOINTS
- GET /vrack - Liste des vRacks disponibles
- POST /ip/{ip}/move/vrack - Déplacer vers un vRack
