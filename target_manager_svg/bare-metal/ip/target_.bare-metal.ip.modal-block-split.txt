# Cahier des Charges : bare-metal/ip - Modal découper bloc IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire avec prévisualisation
- **Contexte** : Gestion des blocs IP
- **Objectif** : Découper un bloc IP en sous-blocs plus petits

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, informations
- **OdsSelect** : Taille des sous-blocs
- **OdsCard** : Zone prévisualisation
- **OdsButton** : Annuler, Découper

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Découper le bloc IP" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INFORMATIONS BLOC SOURCE
| Champ | Label | Exemple |
|-------|-------|---------|
| Bloc | "Bloc source" | "51.210.xxx.0/24" (gras) |
| IPs | "IPs disponibles" | "256 adresses" |

## 6. FORMULAIRE

### Champ Taille des sous-blocs *
| Propriété | Valeur |
|-----------|--------|
| Label | "Taille des sous-blocs *" |
| Type | OdsSelect |
| Options | /25 (128 IPs), /26 (64 IPs), /27 (32 IPs), /28 (16 IPs) |
| Requis | Oui |

## 7. PRÉVISUALISATION
| Propriété | Valeur |
|-----------|--------|
| Fond | Gris (#F3F4F6) |
| Titre | "Résultat :" (gras) |
| Format | Liste des sous-blocs avec taille |
| Exemple | "• 51.210.xxx.0/25 (128 IPs)" |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Découper | Primary (bleu #0050D7) | Lance le découpage |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/block - Informations bloc source
- POST /ip/{ip}/block/split - Découper le bloc
