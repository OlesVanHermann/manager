# Cahier des Charges : bare-metal/ip - Modal avertissement BYOIP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec conditions
- **Contexte** : BYOIP (Bring Your Own IP)
- **Objectif** : Afficher les conditions d'utilisation du service BYOIP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 360px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Introduction, conditions
- **OdsMessage** : Avertissement délai
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bring Your Own IP" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. INTRODUCTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "En utilisant le service BYOIP, vous acceptez les conditions suivantes :" |

## 6. CONDITIONS (LISTE À PUCES)

| # | Condition | Icône |
|---|-----------|-------|
| 1 | "Vous devez être propriétaire légitime des blocs IP annoncés." | Cercle bleu |
| 2 | "Les blocs IP doivent être enregistrés auprès d'un RIR (RIPE, ARIN, etc.)." | Cercle bleu |
| 3 | "Vous êtes responsable de la configuration du routage BGP." | Cercle bleu |

## 7. MESSAGE D'AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Titre | "Délai d'activation" (gras) |
| Texte | "L'activation peut prendre jusqu'à 48h selon les vérifications nécessaires." |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Accepte les conditions |

## 9. APIS & ENDPOINTS
- POST /ip/byoip/accept - Accepter les conditions BYOIP
