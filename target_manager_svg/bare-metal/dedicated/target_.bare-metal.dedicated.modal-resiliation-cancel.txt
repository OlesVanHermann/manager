# Cahier des Charges : bare-metal/dedicated - Modal Resiliation Cancel

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation simple
- **Contexte** : Gestion serveur dédié
- **Objectif** : Annuler une résiliation en cours

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 180px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Annuler la résiliation" |
| Fond | #F9FAFB |

## 5. CONTENU
| Propriété | Valeur |
|-----------|--------|
| Question | "Voulez-vous vraiment annuler la résiliation du service" |
| Service | {serviceName} en bleu #0050D7 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary, 90x36px | Ferme modal |
| Confirmer | Primary #0050D7, 90x36px | Annule la résiliation |

## 7. APIS & ENDPOINTS
- POST /services/{serviceId}/terminate/cancel - Annuler résiliation
