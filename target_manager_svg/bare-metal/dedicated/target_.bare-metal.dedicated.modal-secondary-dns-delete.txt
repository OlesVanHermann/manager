# Cahier des Charges : bare-metal/dedicated - Modal Supprimer DNS Secondaire

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion DNS serveur dédié
- **Objectif** : Supprimer un domaine du DNS secondaire

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le DNS secondaire" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Ligne | Contenu |
|-------|---------|
| 1 | "Êtes-vous sûr de vouloir supprimer le DNS secondaire" |
| 2 | "pour le domaine {domain} ?" |
| Domaine | Bleu (#0050D7), font-weight 500 |
| 3 | "Cette action est irréversible." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Danger (rouge #DC2626) | Supprime le DNS secondaire |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/server/{serviceName}/secondaryDnsDomains/{domain} - Supprimer DNS secondaire
