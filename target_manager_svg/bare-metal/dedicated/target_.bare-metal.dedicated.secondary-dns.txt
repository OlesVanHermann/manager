# Cahier des Charges : dedicated - Onglet DNS secondaire

## 1. VUE D'ENSEMBLE
- **Page** : DNS secondaire serveur dédié
- **Type** : Gestion domaines DNS
- **Objectif** : Configurer OVHcloud comme DNS secondaire

## 2. STRUCTURE PAGE
- Carte explication avec bouton ajout
- Tableau domaines configurés

## 3. COMPOSANTS UI
- **OdsCard** : Container sections
- **OdsButton** : Ajouter un domaine
- **OdsDatagrid** : Tableau domaines
- **OdsBadge** : Statut synchronisation

## 4. HEADER SECTION
| Élément | Valeur |
|---------|--------|
| Titre | Configuration DNS secondaire OVHcloud |
| Description | Ajoutez des domaines pour lesquels OVHcloud agira comme serveur DNS secondaire. |
| Bouton | Ajouter un domaine |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Code | Nom de domaine |
| IP du DNS primaire | Code | IP du serveur DNS maître |
| Dernière synchro | DateTime | Date/heure dernière sync |
| Statut | Badge | Synchronisé / Erreur |
| Actions | Links | Forcer synchro | Supprimer |

## 6. EXEMPLES DOMAINES
| Domaine | IP Primaire | Statut |
|---------|-------------|--------|
| mondomaine.fr | 1.2.3.4 | Synchronisé (vert) |
| exemple.com | 1.2.3.4 | Synchronisé (vert) |
| autresite.org | 5.6.7.8 | Erreur (rouge) |

## 7. BADGES STATUT
| Statut | Couleur |
|--------|---------|
| Synchronisé | Vert |
| Erreur | Rouge |
| En attente | Orange |

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/secondaryDns - Liste domaines
- POST /dedicated/server/{serviceName}/secondaryDns - Ajouter domaine
- DELETE /dedicated/server/{serviceName}/secondaryDns/{domain} - Supprimer
- POST /dedicated/server/{serviceName}/secondaryDns/{domain}/sync - Forcer sync
