# Cahier des Charges : bare-metal/dedicated - Tile Subscription

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les informations d'abonnement et engagement

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 330px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsText** : Labels et valeurs
- **OdsBadge** : Statut renouvellement
- **OdsLink** : Actions engagement

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Abonnement" |

## 10. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/serviceInfos - Infos service
- PUT /dedicated/server/{serviceName}/serviceInfos - Modifier renouvellement
- GET /services/{serviceId}/billing/engagement - Engagement
- POST /services/{serviceId}/billing/engagement/endRule - Résilier engagement
