# Cahier des Charges : bare-metal/dedicated - Modal Tags Unassign

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation danger
- **Contexte** : Gestion serveur dédié
- **Objectif** : Retirer des étiquettes du serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message et liste
- **OdsButton** : Annuler, Retirer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Retirer les étiquettes" |
| Fond | #FEE2E2 (danger) |
| Couleur titre | #DC2626 |

## 5. CONTENU

### Question
| Propriété | Valeur |
|-----------|--------|
| Texte | "Êtes-vous sûr de vouloir retirer ces étiquettes de" |
| Service | {serviceName} en bleu #0050D7 |

### Liste tags à retirer
| Format | Valeur |
|--------|--------|
| Style | Liste à puces (•) |
| Exemple | • production, • europe, • critical |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary, 90x36px | Ferme modal |
| Retirer | Danger #DC2626, 90x36px | Supprime les tags |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/server/{serviceName}/tag/{tagKey} - Retirer tag
