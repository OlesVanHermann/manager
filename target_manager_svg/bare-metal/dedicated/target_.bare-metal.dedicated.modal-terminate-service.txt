# Cahier des Charges : bare-metal/dedicated - Modal Résilier Service

## 1. VUE D'ENSEMBLE
- **Type** : Modal de résiliation danger
- **Contexte** : Gestion serveur dédié
- **Objectif** : Résilier définitivement le service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning données
- **OdsSelect** : Méthode de résiliation
- **OdsText** : Explications
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Résilier le service" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Warning données
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠ |
| Ligne 1 | "Attention ! Toutes vos données seront supprimées." |
| Ligne 2 | "Cette action est irréversible et aucun remboursement ne sera effectué." |
| Couleur texte | #92400E |

### Formulaire
| Champ | Type | Options |
|-------|------|---------|
| Méthode de résiliation | OdsSelect | Voir ci-dessous |

### Options de résiliation
| Option | Description |
|--------|-------------|
| Résilier immédiatement | Suppression immédiate |
| Résilier à la fin de la période de facturation | Suppression à échéance |

### Message d'avertissement
| Ligne | Couleur | Contenu |
|-------|---------|---------|
| 1 | Rouge (#DC2626) | "Cette action supprimera définitivement votre serveur dédié." |
| 2 | Noir | "Êtes-vous sûr de vouloir continuer ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Danger (rouge #DC2626) | Lance la résiliation |

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/terminate - Résilier immédiatement
- PUT /dedicated/server/{serviceName}/serviceInfos - Résilier à échéance
