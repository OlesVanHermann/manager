# Cahier des Charges : bare-metal/dedicated - Modal Modifier Accès FTP Backup

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification avec checkboxes
- **Contexte** : Gestion du stockage FTP Backup
- **Objectif** : Modifier les protocoles autorisés pour un bloc IP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels et informations
- **OdsCheckbox** : Sélection protocoles
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'accès" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Modifier les protocoles autorisés pour le bloc IP :" |
| IP/CIDR | Bleu (#0050D7), font-weight 500 |

### Protocoles (checkboxes)
| Protocole | Label | Description |
|-----------|-------|-------------|
| CIFS | CIFS (Windows) | Accès Windows/SMB |
| FTP | FTP | Protocole FTP standard |
| NFS | NFS (Linux/Unix) | Accès Linux/Unix |

### Style checkbox
| État | Style |
|------|-------|
| Coché | Fond bleu (#0050D7), icône ✓ blanc |
| Non coché | Fond blanc, bordure grise |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Primary (bleu #0050D7) | Enregistre les modifications |

## 7. APIS & ENDPOINTS
- PUT /dedicated/server/{serviceName}/features/backupFTP/access/{ipBlock} - Modifier accès
