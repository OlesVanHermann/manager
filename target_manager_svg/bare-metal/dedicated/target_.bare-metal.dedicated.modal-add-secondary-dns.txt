# Cahier des Charges : bare-metal/dedicated - Modal Ajouter DNS Secondaire

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un domaine DNS secondaire
- **Type** : Modal formulaire
- **Objectif** : Configurer un DNS secondaire sur le serveur

## 2. STRUCTURE MODAL
- Modal blanche (600x340px)
- Header gris avec titre + bouton fermer
- Formulaire (domaine + IP DNS primaire)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Domaine, IP DNS primaire
- **OdsButton** : Annuler (outline), Ajouter (primary)

- **Titre** : "Ajouter un domaine DNS secondaire"

## 5. FORMULAIRE

### Champ Domaine
- **Label** : "Domaine *"
- **Input** : "example.com"
- **Largeur** : 552px
- **Requis** : Oui

### Champ IP DNS primaire
- **Label** : "IP du DNS primaire *"
- **Input** : "123.45.67.89"
- **Largeur** : 300px
- **Requis** : Oui

## 6. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Ajouter** : Primary, soumet le formulaire

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/secondaryDnsDomains
  - Body: { domain, ip }
  - Ajoute un domaine DNS secondaire
