# Cahier des Charges : bare-metal/dedicated - Onglet Réseau

## 1. VUE D'ENSEMBLE
- **Onglet** : Réseau
- **Type** : Page complète réseau
- **Objectif** : Gérer la configuration réseau du serveur

## 2. STRUCTURE PAGE
- Breadcrumb + titre
- Onglets navigation
- Carte Interfaces réseau
- Carte Adresses IP
- Carte vRack

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Sections (3)
- **OdsDatagrid** : Tableaux (interfaces, IPs)
- **OdsBadge** : États (Actif)
- **OdsButton** : Commander IP (primary), Configurer vRack, Retirer du vRack
- **OdsActionMenu** : Actions par IP

## 7. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/networkInterfaceController
  - Liste des interfaces
- GET /ip?type=dedicated&serviceName={serviceName}
  - Liste des IPs associées
- GET /vrack/{serviceName}/dedicatedServer
  - Serveurs dans le vRack
- POST /ip/{ip}/move
  - Déplace une IP failover
