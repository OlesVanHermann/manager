# Cahier des Charges : bare-metal/dedicated - Modal Supprimer Reverse DNS

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation danger
- **Contexte** : Gestion DNS serveur dédié
- **Objectif** : Supprimer un reverse DNS personnalisé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le reverse DNS" |
| Fond | Rouge clair (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Ligne | Contenu |
|-------|---------|
| 1 | "Êtes-vous sûr de vouloir supprimer le reverse DNS :" |
| 2 | "{reverse_dns}" (bleu #0050D7, font-weight 500) |
| 3 | "Le reverse DNS sera réinitialisé à sa valeur par défaut." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Supprimer | Danger (rouge #DC2626) | Supprime le reverse |

## 7. APIS & ENDPOINTS
- DELETE /ip/{ip}/reverse/{ipReverse} - Supprimer reverse DNS
