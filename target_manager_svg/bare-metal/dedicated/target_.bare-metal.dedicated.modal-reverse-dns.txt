# Cahier des Charges : bare-metal/dedicated - Modal Reverse DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le Reverse DNS
- **Type** : Modal formulaire simple
- **Objectif** : Configurer le reverse DNS d'une IP

## 2. STRUCTURE MODAL
- Modal blanche (600x300px)
- Header gris avec titre + bouton fermer
- Affichage IP (lecture seule)
- Input reverse DNS
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Adresse IP
- **OdsInput** : Reverse DNS
- **OdsButton** : Annuler (outline), Enregistrer (primary)

## 4. HEADER
- **Titre** : "Modifier le Reverse DNS"
- **Bouton fermer** : âœ•

## 5. FORMULAIRE
| Champ | Type | Valeur |
|-------|------|--------|
| Adresse IP | Texte (lecture seule) | 123.45.67.89 |
| Reverse DNS * | Input | mon-serveur.example.com |

- **Note** : "Entrez un nom de domaine valide pointant vers cette IP."

## 6. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Enregistrer** : Primary, applique le reverse DNS

## 7. APIS & ENDPOINTS
- GET /ip/{ip}/reverse
  - Reverse DNS actuel
- POST /ip/{ip}/reverse
  - Body: { ipReverse, reverse: "mon-serveur.example.com" }
  - Configure le reverse DNS
