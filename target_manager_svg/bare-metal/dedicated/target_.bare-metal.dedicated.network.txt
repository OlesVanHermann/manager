# Cahier des Charges : dedicated - Onglet Réseau

## 1. VUE D'ENSEMBLE
- **Page** : Configuration réseau serveur dédié
- **Type** : Onglet avec tableaux
- **Objectif** : Gérer les IPs et configuration réseau

## 2. STRUCTURE PAGE
- Onglets (Réseau actif)
- Carte Adresses IP avec tableau
- Carte Configuration réseau
- Carte Trafic réseau

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Cartes sections
- **OdsDatagrid** : Tableau IPs
- **OdsBadge** : Type IP (IPv4/IPv6)
- **OdsButton** : Commander des IPs
- **OdsChart** : Graphique trafic

## 4. TABLEAU ADRESSES IP
| Colonne | Type | Description |
|---------|------|-------------|
| Adresse IP | Code | IP format CIDR |
| Type | Badge | IPv4 (bleu) / IPv6 (vert) |
| Reverse | Text | Nom DNS inverse |

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/ips - Liste IPs
- POST /order/dedicated/server/{serviceName}/ip - Commander IP
- GET /dedicated/server/{serviceName}/networkInterfaceController - Config réseau
- GET /dedicated/server/{serviceName}/statistics - Statistiques trafic
