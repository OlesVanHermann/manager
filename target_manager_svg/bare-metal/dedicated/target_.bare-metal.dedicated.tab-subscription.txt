# Cahier des Charges : bare-metal/dedicated - Onglet Abonnement

## 1. VUE D'ENSEMBLE
- **Onglet** : Abonnement et facturation
- **Type** : Page gestion abonnement
- **Objectif** : Gérer l'abonnement et consulter la facturation

## 2. STRUCTURE PAGE
- Onglets navigation
- Titre
- Carte Abonnement
- Carte Coût mensuel
- Tableau Historique factures
- Boutons actions

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Abonnement, Prix, Factures
- **OdsBadge** : État (Actif, Payé)
- **OdsDatagrid** : Liste factures
- **OdsButton** : Renouveler (primary), Changer d'offre (outline), Résilier (danger outline)
- **OdsLink** : Modifier renouvellement, Télécharger PDF
