# Cahier des Charges : bare-metal/dedicated - Modal Attacher Interface vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration avec sélecteur
- **Contexte** : Gestion réseau serveur dédié
- **Objectif** : Connecter une interface réseau à un vRack (réseau privé)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection du vRack
- **OdsText** : Labels et aide
- **OdsButton** : Annuler, Attacher

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Attacher au réseau privé" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Sélecteur vRack
| Propriété | Valeur |
|-----------|--------|
| Label | "Sélectionner un vRack" |
| Type | OdsSelect (dropdown) |
| Exemple | pn-123456 |

### Texte d'aide
| Ligne | Contenu |
|-------|---------|
| 1 | "L'interface réseau sera connectée au vRack sélectionné." |
| 2 | "Vous pourrez ensuite configurer le VLAN si nécessaire." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Attacher | Primary (bleu #0050D7) | Connecte au vRack |

## 7. APIS & ENDPOINTS
- GET /vrack - Liste des vRacks du compte
- POST /vrack/{serviceName}/dedicatedServerInterface - Attacher interface
