# Cahier des Charges : bare-metal/dedicated - Modal Redémarrer Serveur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Gestion serveur dédié
- **Objectif** : Redémarrer (reboot) le serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 200px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Redémarrer le serveur" |
| Fond | Warning (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Ligne | Contenu |
|-------|---------|
| 1 | "Êtes-vous sûr de vouloir redémarrer le serveur" |
| 2 | "{server_name} ?" |
| Nom serveur | Bleu (#0050D7), font-weight 500 |
| 3 | "Le serveur sera temporairement indisponible pendant le redémarrage." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Warning (orange #F59E0B) | Redémarre le serveur |

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/reboot - Redémarrer serveur
