# Cahier des Charges : bare-metal/dedicated - Modal Changer Mot de Passe FTP

## 1. VUE D'ENSEMBLE
- **Modal** : Changer le mot de passe FTP
- **Type** : Modal formulaire simple
- **Objectif** : Modifier le mot de passe d'accès au backup

## 2. STRUCTURE MODAL
- Modal blanche (600x320px)
- Header gris avec titre + bouton fermer
- Formulaire (nouveau + confirmation)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Nouveau mot de passe (type password)
- **OdsInput** : Confirmation (type password)
- **OdsButton** : Annuler (outline), Changer (primary)

## 4. HEADER
- **Titre** : "Changer le mot de passe FTP"
- **Bouton fermer** : ✕

## 5. FORMULAIRE

### Nouveau mot de passe
- **Label** : "Nouveau mot de passe *"
- **Input** : Type password, masqué
- **Requis** : Oui

### Confirmation
- **Label** : "Confirmer le mot de passe *"
- **Input** : Type password, masqué
- **Requis** : Oui

## 6. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Changer** : Primary, change le mot de passe

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/features/backupCloud/password
  - Body: { password }
  - Change le mot de passe FTP du backup
