# Cahier des Charges : bare-metal/dedicated - Modal Ajouter DNS Secondaire

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un domaine DNS secondaire
- **Type** : Modal formulaire
- **Objectif** : Configurer un DNS secondaire sur le serveur

## 2. STRUCTURE MODAL
- Modal blanche (600x400px)
- Header avec titre + bouton fermer
- Bandeau info avec IP
- Formulaire (domaine + IP primaire)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info avec IP (fond bleu)
- **OdsInput** : Nom de domaine
- **OdsInput** : IP DNS primaire
- **OdsButton** : Annuler (outline), Ajouter (primary)

## 4. HEADER
- **Titre** : "Ajouter un domaine DNS secondaire"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
- "ℹ️ Assurez-vous que votre DNS primaire autorise les transferts vers :"
- "213.251.188.141 (ns.ovh.net)" (monospace)

## 6. FORMULAIRE

### Nom de domaine
- **Label** : "Nom de domaine *"
- **Placeholder** : "Ex: example.com"
- **Largeur** : 540px
- **Requis** : Oui

### IP DNS primaire
- **Label** : "IP du serveur DNS primaire *"
- **Description** : "L'IP de votre serveur qui héberge la zone DNS"
- **Valeur** : "192.168.1.123" (monospace)
- **Largeur** : 300px
- **Requis** : Oui

## 7. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Ajouter** : Primary, ajoute le domaine

## 8. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/secondaryDnsDomains
  - Body: { domain, ip }
  - Ajoute un domaine DNS secondaire
