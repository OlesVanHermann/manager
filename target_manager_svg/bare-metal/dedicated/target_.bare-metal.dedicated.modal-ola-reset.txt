# Cahier des Charges : bare-metal/dedicated - Modal Réinitialiser OLA

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Configuration réseau OVHcloud Link Aggregation
- **Objectif** : Réinitialiser la configuration OLA (agrégation de liens)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Réinitialiser OLA" |
| Fond | Warning (#FEF3C7) |
| Couleur texte | Orange foncé (#92400E) |
| Bouton fermer | × |

## 5. CONTENU

### Message de confirmation
| Ligne | Contenu |
|-------|---------|
| 1 | "Êtes-vous sûr de vouloir réinitialiser la configuration OLA" |
| 2 | "(OVHcloud Link Aggregation) de ce serveur ?" |

### Warning
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (#FEF3C7) |
| Icône | ⚠ |
| Ligne 1 | "Cette action est irréversible et supprimera toute" |
| Ligne 2 | "la configuration d'agrégation actuelle." |
| Couleur texte | #92400E |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Ferme modal |
| Confirmer | Warning (orange #F59E0B) | Réinitialise OLA |

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/ola/reset - Réinitialiser OLA
