# Cahier des Charges : bare-metal/dedicated - Page Install Progress

## 1. VUE D'ENSEMBLE
- **Type** : Modal progression avec barre animée
- **Contexte** : Installation OS serveur dédié
- **Objectif** : Afficher la progression de l'installation

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsProgressBar** : Barre de progression
- **OdsText** : Informations étapes
- **OdsButton** : Annuler l'installation, Fermer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Installation en cours" |
| Fond | #F9FAFB |

## 5. CONTENU

### Information
| Propriété | Valeur |
|-----------|--------|
| Texte principal | "L'installation de votre système d'exploitation est en cours." |
| Texte secondaire | "Cette opération peut prendre plusieurs minutes." (gris) |

### Progression
| Propriété | Valeur |
|-----------|--------|
| Label étape | "Étape 3/5 : Installation des paquets" |
| Format | "Étape X/Y : {description}" |

### Barre de progression
| Propriété | Valeur |
|-----------|--------|
| Largeur | 500px |
| Hauteur | 20px |
| Fond | #E5E7EB |
| Remplissage | #0050D7 |
| Pourcentage | Affiché à droite "60%" |

### Opération courante
| Propriété | Valeur |
|-----------|--------|
| Texte | "Configuration du réseau..." (gris) |

## 6. BOUTON ANNULER INSTALLATION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Annuler l'installation" |
| Style | Danger #DC2626, 150x40px |
| Position | Centré |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Primary #0050D7, 100x36px | Ferme modal (arrière-plan) |

## 8. ÉTAPES INSTALLATION (exemple)
| Étape | Description |
|-------|-------------|
| 1/5 | Préparation du serveur |
| 2/5 | Formatage des disques |
| 3/5 | Installation des paquets |
| 4/5 | Configuration du réseau |
| 5/5 | Finalisation |

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/install/status - Statut installation
- DELETE /dedicated/server/{serviceName}/install - Annuler installation
