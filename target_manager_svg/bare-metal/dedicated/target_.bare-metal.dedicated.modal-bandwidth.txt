# Cahier des Charges : bare-metal/dedicated - Modal Commander Bande Passante

## 1. VUE D'ENSEMBLE
- **Modal** : Commander de la bande passante
- **Type** : Modal commande avec sélection
- **Objectif** : Commander une option de bande passante garantie

## 2. STRUCTURE MODAL
- Modal blanche (700x560px)
- Header avec titre + bouton fermer
- Description
- Grille options (4 cartes)
- Récapitulatif commande
- Checkbox paiement auto
- Footer gris avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsText** : Description
- **OdsTile** : Cartes options (4)
- **OdsCard** : Récapitulatif
- **OdsCheckbox** : Paiement auto
- **OdsButton** : Annuler (outline), Commander (primary)

## 4. HEADER
- **Titre** : "Commander de la bande passante"
- **Bouton fermer** : ×

## 5. DESCRIPTION
"Les offres Premium, Platinum et Ultimate garantissent la qualité de la bande passante sur une zone géographique plus ou moins étendue."

## 6. OPTIONS BANDE PASSANTE
| Option | Description | Prix |
|--------|-------------|------|
| 1 Gbps | Bande passante publique garantie, Trafic illimité | 49,99 €/mois ✅ Sélectionné |
| 2 Gbps | Bande passante publique garantie, Trafic illimité | 99,99 €/mois |
| 3 Gbps | Option Ultimate, Trafic illimité | 149,99 €/mois |

## 7. RÉCAPITULATIF
Fond gris :
- **Titre** : "Récapitulatif de la commande"
- "1 Gbps de bande passante publique garantie - Trafic illimité"
- **Prix** : 49,99 € HT/mois

## 8. CHECKBOX PAIEMENT
✓ "Payer avec le moyen de paiement par défaut"

## 9. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Commander** : Primary, passe commande

## 10. APIS & ENDPOINTS
- GET /order/catalog/formatted/bandwidth
  - Liste des options disponibles
- POST /order/dedicated/server/{serviceName}/bandwidth
  - Body: { bandwidth: 1000, type: "public" }
  - Commande la bande passante
