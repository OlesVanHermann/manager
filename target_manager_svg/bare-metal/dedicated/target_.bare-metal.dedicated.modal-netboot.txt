# Cahier des Charges : bare-metal/dedicated - Modal Configuration Netboot

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration du Netboot
- **Type** : Modal sélection radio
- **Objectif** : Définir le mode de démarrage du serveur

## 2. STRUCTURE MODAL
- Modal blanche (600x500px)
- Header avec titre + bouton fermer
- Bandeau info
- Options radio (4 modes)
- Statut actuel
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsMessage** : Info (fond bleu)
- **OdsRadio** : Options de démarrage (4)
- **OdsCard** : Mode actuel
- **OdsButton** : Annuler (outline), Appliquer (primary)

## 4. HEADER
- **Titre** : "Configuration du Netboot"
- **Bouton fermer** : ×

## 5. BANDEAU INFO
Fond bleu clair :
- "ℹ️ Le Netboot définit comment votre serveur démarre."
- "Les changements prennent effet au prochain redémarrage."

## 6. MODES DE DÉMARRAGE
| Mode | Description | État |
|------|-------------|------|
| Disque dur | Démarrage normal sur le système installé | ✅ Sélectionné |
| Mode rescue | Environnement de dépannage Linux | - |
| Réseau (PXE) | Démarrage via le réseau pour installation | - |
| Script iPXE personnalisé | Utiliser une URL de script iPXE | - |

## 7. STATUT ACTUEL
Fond gris :
- "Mode actuel : **Disque dur**"

## 8. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Appliquer** : Primary, applique le changement

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/boot
  - Récupère le mode de boot actuel
- PUT /dedicated/server/{serviceName}/boot
  - Body: { bootId } (harddisk, rescue, network, ipxeScript)
  - Change le mode de démarrage
