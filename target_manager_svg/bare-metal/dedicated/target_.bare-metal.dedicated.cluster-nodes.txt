# Cahier des Charges : bare-metal/dedicated - Cluster Noeuds

## 1. VUE D'ENSEMBLE
- **Type** : Page listing avec tableau
- **Contexte** : Gestion cluster serveurs dédiés
- **Objectif** : Lister les noeuds d'un cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |
| Tableau largeur | 940px |

## 3. COMPOSANTS UI

- **OdsBadge** : Statut noeud
