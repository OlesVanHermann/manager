# Cahier des Charges : bare-metal/dedicated - Modal Tags Assign

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire avec sélection
- **Contexte** : Gestion serveur dédié
- **Objectif** : Attribuer des étiquettes (tags) au serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection clé/valeur (autocomplete)
- **OdsChip** : Tags sélectionnés
- **OdsButton** : Ajouter, Annuler, Attribuer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Attribuer des étiquettes" |
| Fond | #F9FAFB |

## 5. CHAMPS

### Clé
| Propriété | Valeur |
|-----------|--------|
| Label | "Clé" |
| Placeholder | "Sélectionner ou créer une clé..." |
| Type | Select avec création possible |

### Valeur
| Propriété | Valeur |
|-----------|--------|
| Label | "Valeur" |
| Placeholder | "Sélectionner ou créer une valeur..." |
| Type | Select avec création possible |
| État initial | Fond #F9FAFB (désactivé tant que clé non sélectionnée) |

## 6. BOUTON AJOUTER
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter" |
| Style | Secondary, 100x36px |

## 7. ÉTIQUETTES SÉLECTIONNÉES
| Propriété | Valeur |
|-----------|--------|
| Label section | "Étiquettes sélectionnées" |
| Style chip | Fond #F3F4F6, texte noir |
| Format | "key:value" avec bouton × |

## 8. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary, 90x36px | Ferme modal |
| Attribuer | Primary #0050D7, 90x36px | Attribue les tags |

## 9. APIS & ENDPOINTS
- GET /tag - Liste tous les tags disponibles
- POST /dedicated/server/{serviceName}/tag - Attribuer tag
- Payload: { key: string, value: string }
