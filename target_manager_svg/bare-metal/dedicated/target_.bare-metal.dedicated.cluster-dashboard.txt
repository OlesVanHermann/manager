# Cahier des Charges : bare-metal/dedicated - Cluster Dashboard

## 1. VUE D'ENSEMBLE
- **Type** : Page dashboard avec tuiles
- **Contexte** : Gestion cluster serveurs dédiés
- **Objectif** : Afficher les informations générales d'un cluster

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Tableau de bord / Noeuds
- **OdsCard** : 3 tuiles informations
- **OdsButton** : Actions support

## 4. BREADCRUMB
| Niveau | Texte | Lien |
|--------|-------|------|
| 1 | Bare Metal Cloud | Oui |
| 2 | Serveurs dédiés | Oui |
| 3 | Clusters | Oui |
| 4 | {clusterName} | Non |

## 5. ONGLETS
| Onglet | État |
|--------|------|
| Tableau de bord | Actif (souligné bleu) |
| Noeuds | Inactif |

## 6. TUILES (3 colonnes)

### Tuile 1 : Informations générales
| Champ | Valeur exemple |
|-------|----------------|
| Nom d'affichage | cluster-prod-01 (avec menu ⋮) |
| Nom commercial | Scale-3 HGR-HCI |
| Nombre de noeuds | 4 |
| Datacenter | rbx (Roubaix) |

### Tuile 2 : Abonnement
| Champ | Valeur exemple |
|-------|----------------|
| Date de création | 15/03/2024 |
| Prochaine facturation | 15/01/2025 |
| Renouvellement | Badge "Automatique" vert |
| Contacts | admin-nic (Admin, Tech, Billing) |

### Tuile 3 : Support
| Champ | Valeur exemple |
|-------|----------------|
| Tickets ouverts | 0 |
| Travaux en cours | Aucun |
| Bouton primaire | "Créer un ticket" |
| Bouton secondaire | "Voir les travaux" |

## 7. APIS & ENDPOINTS
- GET /dedicated/cluster - Liste des clusters
- GET /dedicated/cluster/{serviceName} - Détails cluster
- GET /dedicated/cluster/{serviceName}/serviceInfos - Infos abonnement
