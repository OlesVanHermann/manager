# Cahier des Charges : bare-metal/dedicated - Wizard Supprimer FTP Backup

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de confirmation danger (1 étape)
- **Contexte** : Gestion backup serveur dédié
- **Objectif** : Supprimer définitivement le service Backup FTP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 700px |
| Page hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étape (rouge)
- **OdsButton** : Annuler, Supprimer
- **OdsMessage** : Warning danger
- **OdsText** : Instructions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le Backup FTP" |

## 5. CONTENU

### Indicateur d'étape
| Étape | Label | État |
|-------|-------|------|
| 1 | Confirmation | En cours (cercle rouge #DC2626) |

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|
| Container | Blanc, border #DC2626, radius 8px |
| Ligne 1 | "Voulez-vous vraiment supprimer le Backup FTP" |
| Ligne 2 | "{ftp_hostname}" (bleu #0050D7, gras) |

### Exemple hostname
| Propriété | Valeur |
|-----------|--------|
| Format | ftpback-{dc}-{id}.ovh.net |
| Exemple | ftpback-rbx1-123.ovh.net |

### Warning danger
| Propriété | Valeur |
|-----------|--------|
| Type | Danger (#FEE2E2) |
| Icône | ⚠ |
| Texte | "Toutes les données seront définitivement supprimées !" |
| Couleur texte | Rouge (#DC2626), gras |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Supprimer | Droite | Danger (#DC2626) | Supprime le backup FTP |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/server/{serviceName}/features/backupFTP - Supprimer le backup FTP
