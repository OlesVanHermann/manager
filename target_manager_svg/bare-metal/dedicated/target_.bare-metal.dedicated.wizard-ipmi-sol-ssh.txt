# Cahier des Charges : bare-metal/dedicated - Wizard Session SOL via SSH

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de configuration (1 étape)
- **Contexte** : Accès IPMI serveur dédié
- **Objectif** : Démarrer une session SOL (Serial Over LAN) via SSH avec clé SSH

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 800px |
| Page hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étape
- **OdsSelect** : Sélection clé SSH
- **OdsButton** : Annuler, Démarrer
- **OdsLink** : Ajouter nouvelle clé
- **OdsText** : Descriptions, preview clé

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Démarrer une session SOL via SSH" |

## 5. CONTENU

### Indicateur d'étape
| Étape | Label | État |
|-------|-------|------|
| 1 | Ajouter une clé SSH | En cours (cercle bleu) |

### Instructions
| Propriété | Valeur |
|-----------|--------|
| Titre | "Sélectionnez ou ajoutez une clé SSH publique" |
| Sous-titre | "Cette clé sera utilisée pour authentifier votre session SOL (Serial Over LAN)." |

### Sélection clé SSH
| Propriété | Valeur |
|-----------|--------|
| Label | "Clé SSH *" |
| Type | OdsSelect |
| Largeur | 400px |
| Required | Oui |
| Exemple valeur | my-server-key |

### Lien ajouter clé
| Propriété | Valeur |
|-----------|--------|
| Texte | "+ Ajouter une nouvelle clé" |
| Couleur | Bleu (#0050D7) |
| Position | À droite du select |

### Preview clé
| Propriété | Valeur |
|-----------|--------|
| Format | ssh-rsa AAAAB3NzaC1yc2EAAAADAQABAAABAQ... ({key_name}) |
| Couleur | Gris (#6B7280) |
| Taille | 12px |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Démarrer | Droite | Primary (#0050D7) | Démarre la session SOL |

## 7. APIS & ENDPOINTS
- GET /me/sshKey - Liste des clés SSH du compte
- POST /me/sshKey - Créer une nouvelle clé SSH
- POST /dedicated/server/{serviceName}/features/ipmi/access - Démarrer session IPMI/SOL
