# Cahier des Charges : bare-metal/dedicated - Modal Ajouter Accès Backup

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un accès backup
- **Type** : Modal formulaire
- **Objectif** : Configurer un accès au stockage backup

## 2. STRUCTURE MODAL
- Modal blanche (600x420px)
- Header gris avec titre + bouton fermer
- Formulaire (IP + protocoles)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur
- **OdsInput** : Adresse IP
- **OdsCheckbox** : Sélection protocoles
- **OdsButton** : Annuler (outline), Ajouter (primary)

## 4. HEADER
- **Titre** : "Ajouter un accès backup"
- **Bouton fermer** : ✕

## 5. FORMULAIRE

### Champ IP
- **Label** : "Adresse IP *"
- **Input** : "10.0.0.100"
- **Requis** : Oui

### Protocoles autorisés
- **Label** : "Protocoles autorisés"
- **Checkboxes** :
  | Protocole | État |
  |-----------|------|
  | FTP | ✅ Coché |
  | NFS | ✅ Coché |
  | CIFS | ☐ Non coché |

## 6. BOUTONS
- **Annuler** : Outline, ferme la modal
- **Ajouter** : Primary, soumet le formulaire

## 7. APIS & ENDPOINTS
- POST /dedicated/server/{serviceName}/features/backupCloud/access
  - Body: { ip, ftp, nfs, cifs }
  - Ajoute un accès au backup
