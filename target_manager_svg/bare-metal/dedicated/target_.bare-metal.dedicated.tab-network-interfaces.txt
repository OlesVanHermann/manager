# Cahier des Charges : bare-metal/dedicated - Onglet Interfaces Réseau (Détaillé)

## 1. VUE D'ENSEMBLE
- **Onglet** : Interfaces réseau
- **Type** : Page détaillée par interface
- **Objectif** : Voir et gérer chaque interface en détail

## 2. STRUCTURE PAGE
- Onglets navigation
- Titre section
- Cartes par interface (eth0, eth1, ipmi0)

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsCard** : Interface cards (3)
- **OdsBadge** : État (Up)
- **OdsProgressBar** : Bande passante utilisée
- **OdsActionMenu** : Actions par interface
- **OdsLink** : Actions (Configurer OLA, Voir trafic, Gérer VLANs)

## 4. CARTE eth0 - Interface publique
| Champ | Valeur |
|-------|--------|
| Titre | eth0 - Interface publique |
| État | Up ✅ |
| Adresse MAC | 00:1a:2b:3c:4d:5e |
| Mode | Public (Default) |
| Vitesse | 1 Gbit/s |
| Duplex | Full |
| Bande passante utilisée | 30% (300 Mbit/s) |

**Liens :**
- Configurer le mode OLA
- Voir le trafic

**Menu actions :**
- Configurer OLA
- Voir le trafic
- Éditer le mode

## 7. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/networkInterfaceController
  - Liste des interfaces
- GET /dedicated/server/{serviceName}/networkInterfaceController/{mac}
  - Détails d'une interface
- GET /dedicated/server/{serviceName}/statistics?type=traffic
  - Statistiques de trafic
- GET /dedicated/server/{serviceName}/ola
  - Configuration OLA
