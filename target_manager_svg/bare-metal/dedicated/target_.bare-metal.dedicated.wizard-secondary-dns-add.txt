# Cahier des Charges : bare-metal/dedicated - Wizard Ajouter DNS Secondaire

## 1. VUE D'ENSEMBLE
- **Type** : Wizard multi-étapes (3 étapes)
- **Contexte** : Gestion DNS serveur dédié
- **Objectif** : Ajouter un domaine au DNS secondaire OVHcloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 900px |
| Page hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes
- **OdsSelect** : Adresse IP
- **OdsInput** : Domaine
- **OdsButton** : Annuler, Suivant
- **OdsMessage** : Info
- **OdsText** : Descriptions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un DNS secondaire" |

## 5. CONTENU

### Indicateur d'étapes
| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | En cours (cercle bleu) |
| 2 | Vérification | À venir (cercle gris) |
| 3 | Confirmation | À venir (cercle gris) |

### Instructions
| Propriété | Valeur |
|-----------|--------|
| Texte | "Configurez le DNS secondaire en sélectionnant l'IP et le domaine associé." |

### Champ Adresse IP
| Propriété | Valeur |
|-----------|--------|
| Label | "Adresse IP" |
| Type | OdsSelect |
| Largeur | 350px |
| Exemple valeur | 51.210.xxx.xxx |

### Champ Domaine
| Propriété | Valeur |
|-----------|--------|
| Label | "Domaine" |
| Type | OdsInput |
| Largeur | 350px |
| Exemple valeur | example.com |
| Border focus | #0050D7 (2px) |

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |
| Hauteur | 70px |
| Ligne 1 | "ℹ Le DNS secondaire OVHcloud agira comme serveur de noms esclave pour votre domaine." |
| Ligne 2 | "Vous devrez vérifier la propriété du domaine à l'étape suivante." |
| Couleur texte | #1E40AF |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Suivant | Droite | Primary (#0050D7) | Passe à l'étape vérification |

## 7. ÉTAPES SUIVANTES (aperçu)

### Étape 2 - Vérification
- Vérification propriété du domaine
- Ajout enregistrement TXT ou fichier de vérification

### Étape 3 - Confirmation
- Récapitulatif de la configuration
- Validation finale

## 8. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/secondaryDnsDomains - Liste domaines DNS secondaires
- POST /dedicated/server/{serviceName}/secondaryDnsDomains - Ajouter DNS secondaire
- POST /dedicated/server/{serviceName}/secondaryDnsDomains/{domain}/dnsServer - Créer serveur DNS
