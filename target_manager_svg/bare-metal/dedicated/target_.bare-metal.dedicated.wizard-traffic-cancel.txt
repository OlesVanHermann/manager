# Cahier des Charges : bare-metal/dedicated - Wizard Annuler Trafic Supplémentaire

## 1. VUE D'ENSEMBLE
- **Type** : Wizard de confirmation (1 étape)
- **Contexte** : Gestion trafic serveur dédié
- **Objectif** : Annuler l'option de trafic supplémentaire

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 700px |
| Page hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsStepper** : Indicateur d'étape (1 seule)
- **OdsButton** : Annuler, Confirmer
- **OdsText** : Instructions, email

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Annuler le trafic supplémentaire" |

## 5. CONTENU

### Indicateur d'étape
| Étape | Label | État |
|-------|-------|------|
| 1 | Confirmation | En cours (cercle bleu) |

### Question de confirmation
| Propriété | Valeur |
|-----------|--------|
| Container | Blanc, border #E5E7EB, radius 8px |
| Texte | "Êtes-vous sûr de vouloir annuler votre option de trafic supplémentaire ?" |

### Notification email
| Propriété | Valeur |
|-----------|--------|
| Label | "Un email de confirmation sera envoyé à l'adresse :" |
| Email | {user_email} (bleu #0050D7, gras) |
| Exemple | admin@example.com |

## 6. FOOTER PAGE
| Bouton | Position | Style | Action |
|--------|----------|-------|--------|
| Annuler | Gauche | Secondary (bordure grise) | Ferme le wizard |
| Confirmer | Droite | Primary (#0050D7) | Annule l'option trafic |

## 7. APIS & ENDPOINTS
- DELETE /dedicated/server/{serviceName}/option/TRAFFIC - Annuler option trafic
