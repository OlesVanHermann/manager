# Cahier des Charges : bare-metal/dedicated - Tile Support

## 1. VUE D'ENSEMBLE
- **Type** : Tuile dashboard
- **Contexte** : Dashboard serveur dédié
- **Objectif** : Afficher les informations support et tickets

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Tuile largeur | 360px |
| Tuile hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsCard** : Container tuile
- **OdsText** : Labels et valeurs
- **OdsLink** : Actions support
- **OdsBadge** : Compteur interventions

## 4. HEADER TUILE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Support" |

## 5. NIVEAU DE SUPPORT
| Propriété | Valeur |
|-----------|--------|
| Label | "Niveau de support" |
| Valeur | "Standard" |

## 6. TICKETS OUVERTS
| Propriété | Valeur |
|-----------|--------|
| Label | "Tickets ouverts" |
| Compteur | Nombre bleu #0050D7 grand |
| Lien | "Créer un ticket ›" (bleu) |

## 7. INTERVENTIONS
| Propriété | Valeur |
|-----------|--------|
| Label | "Interventions" |
| Badge | Fond #F9FAFB, bordure #E5E7EB |

## 8. AIDE

|-----------|--------|
| Question | "Un problème avec votre produit ?" |
| Lien | "Consulter les travaux ↗" (bleu, externe) |

## 9. APIS & ENDPOINTS
- GET /support/tickets - Liste tickets
- POST /support/tickets - Créer ticket
- GET /dedicated/server/{serviceName}/intervention - Interventions
- GET /support/levels - Niveau de support
