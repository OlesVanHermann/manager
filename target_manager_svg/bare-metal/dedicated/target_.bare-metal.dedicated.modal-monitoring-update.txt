# Cahier des Charges : bare-metal/dedicated - Modal Monitoring

## 1. VUE D'ENSEMBLE
- **Type** : Modal sélection radio
- **Contexte** : Gestion serveur dédié
- **Objectif** : Configurer le niveau de monitoring

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsRadio** : Options monitoring
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Configuration du monitoring" |
| Fond | #F9FAFB |

## 5. OPTIONS MONITORING
| Option | Description | Style sélectionné |
|--------|-------------|-------------------|
| Proactif avec intervention | OVHcloud surveille et intervient automatiquement | Fond #EFF6FF, bordure #0050D7 |
| Proactif sans intervention | OVHcloud surveille et vous notifie uniquement | Fond blanc, bordure #E5E7EB |
| Désactivé | Aucune surveillance OVHcloud | Fond blanc, bordure #E5E7EB |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary #0050D7 | Applique config |

## 7. APIS & ENDPOINTS
- PUT /dedicated/server/{serviceName} - Modifier monitoring
