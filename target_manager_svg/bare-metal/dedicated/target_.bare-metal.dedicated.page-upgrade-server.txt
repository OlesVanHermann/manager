# Cahier des Charges : bare-metal/dedicated - Page Upgrade Serveur

## 1. VUE D'ENSEMBLE
- **Type** : Page wizard (upgrade)
- **Contexte** : Gestion serveur dédié
- **Objectif** : Commander un upgrade RAM pour le serveur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 800px |

## 3. COMPOSANTS UI
- **OdsStepper** : Étapes du wizard
- **OdsRadio** : Sélection option RAM
- **OdsButton** : Annuler, Commander
- **OdsMessage** : Info
- **OdsText** : Descriptions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour au serveur" |
| Titre | "Upgrade RAM" |
| Sous-titre | "Augmentez la mémoire RAM de votre serveur dédié." |

## 5. CONTENU

| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |

| Texte | "L'upgrade nécessite une intervention physique sur votre serveur. Une interruption de service est à prévoir." |

|-------|-------|------|

## 6. FOOTER PAGE
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary (bordure grise) | Retour à la page serveur |
| Commander | Primary (#0050D7) | Passe à l'étape 2 / Confirmation |

## 7. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/specifications/hardware - Specs actuelles
- GET /order/dedicated/server/{serviceName}/upgrade - Options d'upgrade disponibles
- POST /order/dedicated/server/{serviceName}/upgrade - Commander l'upgrade
