# Cahier des Charges : bare-metal/dedicated - Modal Cancel Commit

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation warning
- **Contexte** : Gestion serveur dédié
- **Objectif** : Annuler un engagement en attente

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Conserver, Annuler

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Annuler l'engagement en attente" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |

## 5. CONTENU
| Ligne | Texte |
|-------|-------|
| 1 | "Vous avez un engagement en attente de 24 mois qui" |
| 2 | "prendra effet à la prochaine date de facturation." |
| 3 | "Voulez-vous vraiment annuler cet engagement ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Conserver | Secondary, 90x36px | Ferme modal |
| Annuler | Danger #DC2626, 90x36px | Annule engagement |

## 7. APIS & ENDPOINTS
- DELETE /services/{serviceId}/billing/engagement/request - Annuler engagement en attente
