# Cahier des Charges : bare-metal/dedicated - Page Gérer Intel SGX

## 1. VUE D'ENSEMBLE
- **Type** : Page de configuration
- **Contexte** : Gestion serveur dédié
- **Objectif** : Configurer le mode d'activation Intel SGX (Software Guard Extensions)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Page largeur | 1100px |
| Page hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsButton** : Retour, Confirmer
- **OdsRadio** : Sélection mode (cartes)
- **OdsSelect** : Taille PRMRR
- **OdsMessage** : Info
- **OdsText** : Descriptions

## 4. HEADER PAGE
| Propriété | Valeur |
|-----------|--------|
| Lien retour | "← Retour" |
| Titre | "Gérer Intel SGX" |
| Sous-titre | "Configurez le mode d'activation SGX pour votre serveur dédié." |

## 5. CONTENU

### Sélection Mode d'activation
| Propriété | Valeur |
|-----------|--------|
| Label | "Mode d'activation" |
| Type | Radio cards |
| Layout | 3 colonnes |

### Options disponibles
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Désactivé | ☐ | "SGX est désactivé sur ce serveur" |
| Contrôlé par logiciel | ● | "SGX est activé et géré par le système d'exploitation" |
| Manuel | ☐ | "SGX est activé avec une taille PRMRR fixe" |

### Cartes de sélection
| Propriété | Valeur |
|-----------|--------|
| Largeur carte | 280px |
| Hauteur carte | 100px |
| Border radius | 8px |
| Border sélectionné | #0050D7 (2px) |
| Border non sélectionné | #E5E7EB |

### Sélection Taille PRMRR
| Propriété | Valeur |
|-----------|--------|
| Label | "Taille PRMRR" |
| Type | OdsSelect |
| Valeur | 64 MB |
| Largeur | 280px |
| Visibilité | Affiché si mode ≠ Désactivé |

### Options PRMRR (dropdown)
| Valeur |
|--------|
| 32 MB |
| 64 MB |
| 128 MB |

### Message d'information
| Propriété | Valeur |
|-----------|--------|
| Type | Info (#EFF6FF) |
| Icône | ℹ |
| Texte | "La modification du mode SGX nécessite un redémarrage du serveur. Les changements seront appliqués après le prochain reboot." |
| Couleur texte | #1E40AF |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Confirmer | Primary (#0050D7) | Applique la configuration SGX |

## 7. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/biosSettings/sgx - Récupérer config SGX
- PUT /dedicated/server/{serviceName}/biosSettings/sgx - Modifier config SGX
