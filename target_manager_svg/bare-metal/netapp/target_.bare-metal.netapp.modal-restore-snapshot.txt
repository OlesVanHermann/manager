# Cahier des Charges : bare-metal/netapp - Modal Restaurer Snapshot

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning avec saisie
- **Contexte** : Gestion volumes NetApp
- **Objectif** : Restaurer un volume à partir d'un snapshot

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 420px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Avertissement warning
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Restaurer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Restaurer le dernier snapshot" |
| Fond | #FEF3C7 (warning) |
| Couleur titre | #92400E |
| Bouton fermer | × |

## 5. CONTENU

### Question
| Propriété | Valeur |
|-----------|--------|
| Texte | "Êtes-vous sûr de vouloir restaurer le volume suivant ?" |

### Informations volume
| Champ | Valeur exemple |
|-------|----------------|
| Volume | vol-production-01 |
| Nom du snapshot | backup-2024-12-28 |

### Message d'avertissement
| Propriété | Valeur |
|-----------|--------|
| Type | Warning |
| Fond | #FEF3C7 |
| Border | #FDE68A |
| Icône | ⚠ |
| Titre | "Attention" |
| Ligne 1 | "Cette action va restaurer les données sur le volume {volumeName} à l'état du {snapshotDate}" |
| Ligne 2 | "Les données modifiées après cette date seront supprimées." |
| Couleur texte | #92400E |

### Champ de confirmation
| Propriété | Valeur |
|-----------|--------|
| Label | "Confirmer le nom du volume" |
| Type | OdsInput |
| Largeur | 450px |
| Placeholder | {volumeName} |
| Validation | Doit correspondre au nom du volume |

## 6. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Restaurer | Primary warning | #F59E0B | Restaure le snapshot |

## 7. APIS & ENDPOINTS
- POST /storage/netapp/{serviceName}/volume/{volumeId}/snapshot/{snapshotId}/restore - Restaurer snapshot
