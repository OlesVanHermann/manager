# Cahier des Charges : bare-metal/netapp - Modal supprimer réseau

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive avec saisie
- **Contexte** : NetApp Dedicated
- **Objectif** : Supprimer un réseau VLAN du NetApp

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message, warning
- **OdsInput** : Champ de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le réseau" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#92400E) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Question
| Texte |
|-------|
| "Voulez-vous supprimer le réseau {networkName} ?" |

### Warning
| Texte |
|-------|
| "Cette action est irréversible." |

## 6. CHAMP DE CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Pour confirmer, tapez TERMINATE" |
| Placeholder | "TERMINATE" |
| Validation | Doit correspondre exactement à "TERMINATE" |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge, désactivé tant que non confirmé) | Supprime le réseau |

## 8. COMPORTEMENT
- Le bouton Supprimer est désactivé (opacité 50%) tant que "TERMINATE" n'est pas saisi
- Validation case-sensitive

## 9. APIS & ENDPOINTS
- DELETE /storage/netapp/{serviceName}/network/{networkId} - Supprimer réseau
