# Cahier des Charges : bare-metal/netapp - Modal Créer Snapshot

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Contexte** : Gestion volumes NetApp
- **Objectif** : Créer un snapshot manuel d'un volume

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs nom et description
- **OdsButton** : Annuler, Créer snapshot

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Créer un snapshot" |
| Fond | #F9FAFB |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Propriété | Valeur |
|-----------|--------|
| Texte | "Vous allez créer un snapshot manuel pour {volumeName}" |
| Volume | Bleu #0050D7 |

### Champ Nom
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom du snapshot (optionnel)" |
| Type | OdsInput |
| Largeur | 400px |
| Placeholder | "Ex: backup-2024-12-29" |

### Champ Description
| Propriété | Valeur |
|-----------|--------|
| Label | "Description (optionnel)" |
| Type | OdsInput |
| Largeur | 400px |
| Placeholder | "Description du snapshot" |

## 6. FOOTER MODAL
| Bouton | Style | Couleur | Action |
|--------|-------|---------|--------|
| Annuler | Secondary | Bordure grise | Ferme modal |
| Créer snapshot | Primary | #0050D7 | Crée le snapshot |

## 7. APIS & ENDPOINTS
- POST /storage/netapp/{serviceName}/volume/{volumeId}/snapshot - Créer snapshot
