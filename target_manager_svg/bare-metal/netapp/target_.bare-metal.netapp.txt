# Cahier des Charges : bare-metal/netapp - Liste

## 1. VUE D'ENSEMBLE
- **Page** : Liste des services Enterprise File Storage (NetApp)
- **Type** : Dashboard liste
- **Objectif** : Afficher tous les NetApp du compte

## 2. STRUCTURE PAGE
- Header avec breadcrumb Bare Metal > Enterprise File Storage
- Zone de recherche/filtres
- Tableau des NetApp
- Pagination

## 3. COMPOSANTS UI
- **OdsDatagrid** : Tableau liste NetApp
- **OdsInput** : Recherche
- **OdsBadge** : Statut service
- **OdsLink** : Navigation vers détail
- **OdsPagination** : Navigation pages

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link | Nom du NetApp (cliquable) |
| ID | Text | Identifiant technique |
| Région | Text | Localisation |
| Capacité | Text | Stockage total |
| Utilisation | Text | Stockage utilisé |
| Statut | Badge | État du service |
| Actions | ActionMenu | Menu contextuel |

## 5. ACTIONMENU (⋮)
| Action | Description |
|--------|-------------|
| Voir les détails | Ouvre le dashboard |
| Gérer les volumes | Ouvre l'onglet volumes |
| Résilier | Ouvre modal résiliation |

## 6. APIS & ENDPOINTS
- GET /storage/netapp
  - Liste des services NetApp
