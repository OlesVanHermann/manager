# Cahier des Charges : bare-metal/cda - Onglet Utilisation de l'espace

## 1. VUE D'ENSEMBLE
- **Type** : Page tableau de bord
- **Contexte** : Ceph Distributed Architecture (CDA)
- **Objectif** : Visualiser l'utilisation de l'espace de stockage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsText** : Titres, labels, valeurs
- **OdsCard** : Cartes d'information
- **OdsProgressBar** : Barres de progression par pool
- **OdsBadge** : Alertes
- **OdsChart** : Graphique circulaire (pourcentage)

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Utilisation de l'espace" |

## 5. CARTE ESPACE TOTAL

### Graphique circulaire
| Propriété | Valeur |
|-----------|--------|
| Type | Donut chart |
| Valeur centrale | "{percent}% utilisé" |
| Exemple | "40% utilisé" |

### Légende
| Élément | Couleur | Valeur |
|---------|---------|--------|
| Utilisé | Bleu (#0050D7) | "18.4 TB" |
| Disponible | Gris (#E5E7EB) | "27.6 TB" |
| Capacité totale | - | "46 TB" |

## 6. CARTE PAR POOL

### Barres de progression par pool
| Pool | Couleur | Pourcentage |
|------|---------|-------------|
| data | Bleu (#0050D7) | 70% |
| backup | Vert (#10B981) | 30% |
| archive | Orange (#F59E0B) | 10% |

### Format
- Nom du pool
- Barre de progression colorée
- Pourcentage à droite

## 7. SECTION ALERTES D'ESPACE

### Format alerte
| Propriété | Valeur |
|-----------|--------|
| Badge | "Attention" (jaune #FEF3C7) |
| Message | Description de l'alerte |
| Exemple | "Le pool "data" approche de 80% de capacité. Pensez à ajouter de l'espace." |

### Niveaux d'alerte
| Niveau | Seuil | Couleur |
|--------|-------|---------|
| Info | < 60% | Vert |
| Attention | 60-80% | Jaune |
| Critique | > 80% | Rouge |

## 8. APIS & ENDPOINTS
- GET /dedicatedCloud/{serviceName}/ceph/pool - Liste des pools avec usage
- GET /dedicatedCloud/{serviceName}/ceph/space - Espace total
- GET /dedicatedCloud/{serviceName}/ceph/alert - Alertes d'espace
