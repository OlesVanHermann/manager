# Cahier des Charges : bare-metal/vps - Migration VPS

## 1. VUE D'ENSEMBLE
- **Page** : Migration VPS
- **Type** : Page formulaire avec programmation
- **Objectif** : Migrer vers la nouvelle gamme VPS

## 2. STRUCTURE PAGE
- Titre "Migration VPS"
- Sous-titre explicatif
- Bandeau info (avantages)
- Bandeau warning (interruption)
- Détails VPS après migration
- Programmation date/heure
- Boutons action

## 3. COMPOSANTS UI
- **OdsMessage** : Bandeau info (bleu)
- **OdsMessage** : Bandeau warning (orange)
- **OdsCard** : Détails nouvelle configuration
- **OdsInput** : Date (date picker)
- **OdsSelect** : Heure
- **OdsButton** : Annuler, Programmer

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Titre | Information importante |
| Texte | Votre VPS actuel va être remplacé par une machine de notre nouvelle gamme, à un prix inférieur ou équivalent |

## 5. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠️ |
| Texte | La migration rendra votre service indisponible entre 1 et 4 heures |
| Sous-texte | L'historique des sauvegardes ne sera pas conservé |

## 6. DÉTAILS NOUVELLE CONFIGURATION
| Propriété | Valeur |
|-----------|--------|
| Modèle | VPS Value (Nouvelle gamme) |
| CPU | 2 vCores |
| Mémoire | 4 Go |
| Stockage | 80 Go SSD NVMe |

## 7. PROGRAMMATION
| Champ | Type | Description |
|-------|------|-------------|
| Date | DatePicker | Sélection date migration |
| Heure | Select | Créneau horaire (02:00 recommandé) |

### Helper text
- La migration sera effectuée automatiquement à l'heure choisie

## 8. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Outline | Annuler |
| Programmer | Primary | Programmer la migration |

## 9. APIS & ENDPOINTS
- GET /vps/{serviceName} - Infos VPS actuel
- GET /vps/{serviceName}/migration - Disponibilité migration
- POST /vps/{serviceName}/migration - Programmer la migration
