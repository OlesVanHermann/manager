# Cahier des Charges : bare-metal/vps - Modal Reverse DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Modifier le Reverse DNS
- **Type** : Formulaire
- **Objectif** : Configurer le reverse DNS d'une IP

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Info box explicative
- Champs IP et reverse
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Container (600x380px)
- **OdsMessage** : Info box (fond bleu)
- **OdsInput** : IP (lecture seule), Reverse DNS
- **OdsText** : Note aide
- **OdsButton** : Annuler (outline), Modifier (primary)

## 4. INFO BOX
"Le reverse DNS permet d'associer un nom de domaine à votre adresse IP (important pour les serveurs mail)."

## 5. CHAMPS FORMULAIRE
### Adresse IP
| Élément | Valeur |
|---------|--------|
| Label | Adresse IP |
| Valeur | 51.210.123.45 |
| Style | Lecture seule (fond gris) |

### Reverse DNS
| Élément | Valeur |
|---------|--------|
| Label | Reverse DNS |
| Valeur exemple | mail.mondomaine.com |

### Note aide
"Le nom de domaine doit pointer vers cette IP (enregistrement A)."

## 6. BOUTONS
| Bouton | Style |
|--------|-------|
| Annuler | Outline |
| Modifier | Primary |

## 7. APIS & ENDPOINTS
- GET /vps/{serviceName}/ips
  - Liste IPs
- PUT /ip/{ip}/reverse
  - Body: { reverse: string }
