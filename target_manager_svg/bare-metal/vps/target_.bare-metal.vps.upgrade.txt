# Cahier des Charges : bare-metal/vps - Upgrade VPS

## 1. VUE D'ENSEMBLE
- **Page** : Faire évoluer mon VPS
- **Type** : Wizard multi-étapes
- **Objectif** : Changer de gamme VPS (upgrade)

## 2. STRUCTURE PAGE
- Breadcrumb (VPS > nom > Faire évoluer)
- Titre "Faire évoluer mon VPS"
- Stepper 3 étapes
- Sélection modèle (cartes)
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes
- **OdsCard** : Cartes modèles VPS
- **OdsRadio** : Sélection modèle
- **OdsBadge** : "Modèle actuel"
- **OdsButton** : Annuler, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | Description |
|-------|-----|-------------|
| 1 | Modèle | Choix du nouveau modèle |
| 2 | Conditions | Termes et conditions |
| 3 | Récapitulatif | Confirmation et paiement |
