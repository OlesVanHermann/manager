# Cahier des Charges : bare-metal/vps - Wizard Mode Rescue

## 1. VUE D'ENSEMBLE
- **Page** : Assistant mode rescue
- **Type** : Wizard multi-étapes
- **Objectif** : Redémarrer le VPS en mode rescue

## 2. STRUCTURE PAGE
- Breadcrumb (VPS > nom > Mode rescue)
- Titre "Redémarrer en mode rescue"
- Stepper 3 étapes
- Bandeau warning
- Sélection clé SSH
- Informations de connexion
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes (Type rescue, Clé SSH, Confirmation)
- **OdsMessage** : Warning interruption service
- **OdsCard** : Cartes clés SSH (sélectionnables)
- **OdsRadio** : Sélection clé
- **OdsLink** : Ajouter nouvelle clé
- **OdsButton** : Retour, Suivant

## 4. ÉTAPES WIZARD
| Étape | Nom | Description |
|-------|-----|-------------|
| 1 | Type de rescue | Choix du système rescue |
| 2 | Clé SSH | Sélection clé d'accès |
| 3 | Confirmation | Récapitulatif et validation |

## 5. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠️ |
| Texte | Le mode rescue redémarrera votre VPS sur un système temporaire |
| Sous-texte | Vos données seront accessibles mais les services ne démarreront pas automatiquement |

## 6. CARTE CLÉ SSH
| Élément | Description |
|---------|-------------|
| Radio | Sélection unique |
| Nom | Nom de la clé (bold) |
| Fingerprint | SHA256:xxx... (monospace, tronqué) |
| Date | Ajoutée le DD MMM YYYY |

## 7. ÉTATS CARTE
| État | Style |
|------|-------|
| Sélectionné | Fond bleu clair, bordure primary |
| Non sélectionné | Fond blanc, bordure grise |

## 8. INFORMATIONS CONNEXION
| Propriété | Valeur |
|-----------|--------|
| Titre | Informations de connexion |
| Message | Les identifiants seront envoyés à : |
| Email | admin@example.com (monospace) |

## 9. BOUTONS NAVIGATION
| Bouton | Style | Position |
|--------|-------|----------|
| Retour | Outline | Gauche |
| Suivant | Primary | Droite |

## 10. APIS & ENDPOINTS
- GET /vps/{serviceName}/sshKey - Liste des clés SSH
- POST /vps/{serviceName}/reboot - Reboot en mode rescue
- PUT /vps/{serviceName}/rescue - Configuration rescue
