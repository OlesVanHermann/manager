# Cahier des Charges : bare-metal/vps - Modal KVM

## 1. VUE D'ENSEMBLE
- **Modal** : Console KVM (noVNC)
- **Type** : Terminal intégré
- **Objectif** : Accéder à la console du VPS via KVM

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Zone terminal (fond noir)
- Avertissement clavier
- Lien ouvrir nouvelle fenêtre

## 3. COMPOSANTS UI
- **OdsModal** : Container (700x520px)
- **Zone terminal** : Fond sombre avec texte vert/orange
- **OdsMessage** : Avertissement
- **OdsLink** : Ouvrir nouvelle fenêtre

## 4. TERMINAL
### Style
- Fond : #1F2937 (gris sombre)
- Texte principal : #10B981 (vert)
- Prompt : #F59E0B (orange/jaune)
- Police : monospace (Consolas, Monaco)

### Contenu exemple
```
Ubuntu 22.04.3 LTS vps-abc123def tty1

vps-abc123def login: root
Password:
Last login: Thu Dec 26 10:15:32 2024 from 192.168.1.1

root@vps-abc123def:~# █
```

## 5. MESSAGES
### Avertissement
"⚠️ Attention : la disposition du clavier peut différer selon votre configuration locale"

### Action
"↗ Ouvrir dans une nouvelle fenêtre" (lien bleu)

## 6. FONCTIONNALITÉS KVM
- noVNC pour accès web
- Pas de plugin Java requis
- Session temps réel
- Support copier-coller

## 7. APIS & ENDPOINTS
- GET /vps/{serviceName}/console
  - Génère URL console KVM
- POST /vps/{serviceName}/openConsole
  - Ouvre session console
