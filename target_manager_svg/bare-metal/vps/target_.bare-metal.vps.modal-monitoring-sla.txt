# Cahier des Charges : bare-metal/vps - Modal supervision SLA

## 1. VUE D'ENSEMBLE
- **Type** : Modal informative avec confirmation
- **Contexte** : VPS (Virtual Private Server)
- **Objectif** : Afficher et gérer la supervision SLA du VPS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Labels, valeurs, statut
- **OdsMessage** : Information IPs supervisées
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supervision SLA" |
| Fond | Bleu clair (#EFF6FF) |
| Couleur texte | Bleu (#1E40AF) |
| Bouton fermer | × |

## 5. CONTENU

### Statut supervision
| Propriété | Valeur |
|-----------|--------|
| Label | "Supervision SLA :" |
| Valeur | "Activée" / "Désactivée" |
| Couleur activée | Vert (#10B981), gras |
| Couleur désactivée | Rouge (#DC2626), gras |

### Encadré IPs supervisées
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Fond | #EFF6FF |
| Bordure | #0050D7 |
| Titre | "Adresses IP supervisées :" |
| Format | Liste à puces |

### Exemple IPs
- • 51.210.xxx.xxx (IPv4)
- • 2001:41d0:xxx:xxxx::xxxx (IPv6)

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary (bleu) | Confirme l'action |

## 7. APIS & ENDPOINTS
- GET /vps/{serviceName}/monitoring - Statut monitoring SLA
- PUT /vps/{serviceName}/monitoring - Activer/Désactiver monitoring
