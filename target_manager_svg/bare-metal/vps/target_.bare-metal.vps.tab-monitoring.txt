# Cahier des Charges : bare-metal/vps - Onglet Monitoring

## 1. VUE D'ENSEMBLE
- **Onglet** : Monitoring
- **Type** : Page graphiques performances
- **Objectif** : Surveiller les performances du VPS

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre
- Onglets navigation
- Carte état surveillance
- Sélecteur période
- 4 graphiques (CPU, RAM, Réseau IN, Réseau OUT)

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : 6 onglets
- **OdsCard** : État surveillance
- **OdsToggle** : Surveillance active
- **OdsBadge** : En ligne (success)
- **OdsSelect** : Période
- **OdsChart** : 4 graphiques ligne

## 4. ONGLETS
| Onglet | État |
