# Cahier des Charges : bare-metal/vps - Modal Redémarrer

## 1. VUE D'ENSEMBLE
- **Modal** : Redémarrer le VPS
- **Type** : Confirmation avec option
- **Objectif** : Confirmer le redémarrage du VPS

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Corps avec message confirmation
- Option mode rescue (checkbox)
- Boutons Annuler / Confirmer

## 3. COMPOSANTS UI
- **OdsModal** : Container (600x380px)
- **OdsText** : Message confirmation
- **OdsCheckbox** : Mode rescue
- **OdsButton** : Annuler (outline), Confirmer (primary)

## 4. CONTENU
### Message principal
"Êtes-vous sûr de vouloir redémarrer votre VPS ?"

### Notes (italique)
- "Le redémarrage peut prendre quelques minutes."
- "Votre VPS sera temporairement indisponible."

## 5. OPTION MODE RESCUE
| Élément | Valeur |
|---------|--------|
| Checkbox | ☐ Non coché par défaut |
| Label | Redémarrer en mode rescue |
| Description | Le mode rescue vous permet de diagnostiquer et réparer votre VPS en cas de problème. |

## 6. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Confirmer | Primary | Droite |

## 7. APIS & ENDPOINTS
- POST /vps/{serviceName}/reboot
  - Body: { type: "soft" | "hard" | "rescue" }
