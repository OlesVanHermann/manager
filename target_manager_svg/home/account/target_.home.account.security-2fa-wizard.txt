# Cahier des Charges : home/account - Assistant activation 2FA

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard multi-étapes
- **URL** : N/A (composant modal)
- **Objectif** : Guider l'utilisateur pour activer la double authentification

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 540px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsStepper** : Indicateur 4 étapes (cercles connectés)
- **OdsRadio** : Sélection méthode 2FA
- **OdsText** : Descriptions
- **OdsLink** : Aide externe
- **OdsButton** : Navigation

## 4. STEPPER
| Étape | Nom | Description |
|-------|-----|-------------|
| 1 | Choix | Sélection méthode |
| 2 | Configuration | Setup selon méthode |
| 3 | Vérification | Validation code |
| 4 | Confirmation | Récapitulatif |

## 5. ÉTAPE 1 : CHOIX DE MÉTHODE
| Option | Label | Description | Sélectionné |
|--------|-------|-------------|-------------|
| SMS | SMS | Recevez un code de vérification par SMS sur votre téléphone. | Par défaut |
| TOTP | Application mobile (TOTP) | Utilisez Google Authenticator, Authy ou une application similaire. | - |
| U2F | Clé de sécurité (U2F/FIDO2) | Utilisez une clé physique comme YubiKey pour vous authentifier. | - |

## 6. MISE EN FORME OPTIONS
| Propriété | Valeur |
|-----------|--------|
| Type | Cards sélectionnables |
| Bordure sélectionnée | Bleu #0050D7 (2px) |
| Bordure non sélectionnée | Gris #E5E7EB |
| Hauteur card | 80px |
| Espacement | 15px entre cards |

## 7. LIEN AIDE
| Élément | Description |
|---------|-------------|
| Texte | "En savoir plus sur la double authentification" |
| Style | Lien bleu souligné |
| Action | Ouvre documentation externe |

## 8. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Suivant → | Primary | Passer à l'étape 2 |

## 9. COMPORTEMENT
- Radio buttons mutuellement exclusifs
- Card entière cliquable pour sélection
- Bouton "Suivant" toujours actif (option par défaut sélectionnée)

## 10. APIS & ENDPOINTS
Dépend de la méthode choisie :
- SMS : POST /me/accessRestriction/sms
- TOTP : POST /me/accessRestriction/totp
- U2F : POST /me/accessRestriction/u2f
