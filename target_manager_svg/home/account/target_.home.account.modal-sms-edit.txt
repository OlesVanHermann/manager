# Cahier des Charges : home/account - Modal modifier 2FA SMS

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Page sécurité du compte (2FA)
- **Objectif** : Modifier la description d'un numéro de téléphone 2FA

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Description
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'authentification SMS" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Titre section
| Texte |
|-------|
| "Modifier la description" |

### Champ
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Description | Input | "Mon téléphone principal" |

### Aide
| Texte |
|-------|
| "Maximum 100 caractères" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre la modification |

## 7. APIS & ENDPOINTS
- PUT /me/accessRestriction/sms/{id} - Modifier description 2FA SMS
