# Cahier des Charges : home/account - Modal supprimer 2FA SMS

## 1. VUE D'ENSEMBLE
- **Type** : Modal de suppression avec vérification
- **Contexte** : Page sécurité du compte (2FA)
- **Objectif** : Supprimer un numéro de téléphone utilisé pour l'authentification SMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Messages informatifs
- **OdsMessage** : Warning désactivation 2FA
- **OdsInput** : Code de vérification
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer l'authentification SMS" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Titre section
| Texte |
|-------|
| "Supprimer ce numéro de téléphone" |

### Information
| Texte |
|-------|
| "Ce numéro ne sera plus utilisé pour l'authentification à deux facteurs." |

### Message Warning
| Type | Contenu |
|------|---------|
| Warning (rouge) | "⚠️ Si aucune autre méthode 2FA n'est configurée, la double authentification sera désactivée." |

### Champ de vérification
| Champ | Type | Placeholder |
|-------|------|-------------|
| Code de vérification * | Input | "Entrez le code reçu par SMS" |

### Aide
| Texte |
|-------|
| "Un code de vérification a été envoyé à votre numéro de téléphone." |

## 6. FOOTER MODAL
| Bouton | Style | État |
|--------|-------|------|
| Annuler | Secondary | Actif |
| Supprimer | Danger (rouge) | Désactivé tant que code non saisi |

## 7. APIS & ENDPOINTS
- POST /me/accessRestriction/sms/{id}/sendCode - Envoyer code SMS
- DELETE /me/accessRestriction/sms/{id} - Supprimer 2FA SMS (avec code)
