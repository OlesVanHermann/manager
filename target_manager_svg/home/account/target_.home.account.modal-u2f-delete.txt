# Cahier des Charges : home/account - Modal supprimer clé U2F

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page sécurité du compte (2FA)
- **Objectif** : Supprimer une clé de sécurité U2F/FIDO2

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 170px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Messages informatifs
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer la clé de sécurité" |
| Fond | Rouge (#FEE2E2) |
| Couleur texte | Rouge (#DC2626) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Titre section
| Texte |
|-------|
| "Supprimer cette clé de sécurité U2F" |

### Information
| Texte |
|-------|
| "Cette clé ne pourra plus être utilisée pour l'authentification." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime la clé U2F |

## 7. APIS & ENDPOINTS
- DELETE /me/accessRestriction/u2f/{id} - Supprimer clé U2F
