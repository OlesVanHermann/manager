# Cahier des Charges : home/account - Sécurité

## 1. VUE D'ENSEMBLE
- **Type** : Page paramètres sécurité
- **URL** : /account/security
- **Objectif** : Gérer la sécurité du compte (2FA, mot de passe)

## 2. STRUCTURE PAGE
- Tabs avec Sécurité actif
- Bannière d'avertissement
- Section Mot de passe
- Section Double authentification (3 méthodes)
- Section Codes de secours

## 3. COMPOSANTS UI
- **OdsMessage** : Bannière warning
- **OdsCard** : Méthodes 2FA
- **OdsBadge** : Statut (Activé, Non activé)
- **OdsButton** : Gérer, Activer

## 4. BANNIÈRE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Icône | ⚠️ |
| Message | "Les actions modifient les paramètres de sécurité" |

## 5. SECTION MOT DE PASSE
| Élément | Contenu |
|---------|---------|
| Affichage | ••••••••••••• |
| CTA | "Changer mon mot de passe" |

## 6. MÉTHODES 2FA (3 cartes)
| Méthode | Statut | Description |
|---------|--------|-------------|
| SMS | Activé (vert) | Code reçu par SMS |
| Application Mobile (TOTP) | Non activé (jaune) | Google Authenticator, Authy |
| Clé de sécurité (U2F) | Non activé (jaune) | YubiKey, Titan |

## 7. CARTE 2FA
| Élément | Contenu |
|---------|---------|
| Dimensions | 350x200px |
| Titre | Nom méthode |
| Description | Explication |
| Badge | Statut |
| Info | Numéro masqué si SMS |
| CTA | "Gérer" ou "Activer" |

## 8. SECTION CODES SECOURS
| Élément | Contenu |
|---------|---------|
| Titre | "Codes de secours" |
| Description | Codes pour accès compte |
| Info | "8 codes restants" |
| CTA | "Regénérer les codes" |

## 9. APIS & ENDPOINTS
- GET /me/auth/2fa - Liste méthodes 2FA
- POST /me/auth/2fa/sms - Activer SMS
- POST /me/auth/2fa/totp - Activer TOTP
- POST /me/auth/2fa/u2f - Activer U2F
- GET /me/auth/backupCodes - Codes secours
