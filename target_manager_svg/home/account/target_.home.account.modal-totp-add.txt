# Cahier des Charges : home/account - Modal Ajouter TOTP

## 1. VUE D'ENSEMBLE
- **Type** : Modal configuration TOTP
- **URL** : N/A (composant modal)
- **Objectif** : Configurer l'authentification TOTP (Google Authenticator, Authy, etc.)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 490px |

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Instructions
- **QR Code** : Image QR code à scanner
- **OdsInput** : Clé secrète (masquée), Code vérification, Description
- **OdsButton** : Afficher clé, Actions
- **OdsMessage** : Info (bleu)

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Ajouter une application TOTP" |
| Instructions | "Scannez ce QR code avec votre application" |
| Apps supportées | "Google Authenticator, Authy, Microsoft Authenticator..." |

## 5. QR CODE
| Propriété | Valeur |
|-----------|--------|
| Taille | 180x180px |
| Contenu | otpauth://totp/... |

## 6. CLÉ SECRÈTE
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Clé secrète | Code masqué | JBSWY3DPEHPK3PXP |
| Bouton | Afficher/Masquer | Toggle visibilité |

## 7. FORMULAIRE VÉRIFICATION
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Code de vérification | Input text | Oui | Entrez le code à 6 chiffres |
| Description | Input text | Non | Ex: Mon iPhone |

## 8. INFO
| Type | Message |
|------|---------|
| Info | "ℹ Le code change toutes les 30 secondes. Entrez le code actuel pour valider." |

## 9. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Fermer modal |
| Valider | Primary | Vérifier et activer TOTP |

## 10. APIS & ENDPOINTS
- POST /me/accessRestriction/totp - Créer secret TOTP
- POST /me/accessRestriction/totp/{id}/validate - Valider code
