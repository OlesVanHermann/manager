# Cahier des Charges : home/account - Historique des emails

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **URL** : /account/emails
- **Objectif** : Consulter l'historique des emails envoyés par OVHcloud

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Historique des emails
- **OdsCard** : Container principal
- **OdsSearchField** : Recherche
- **OdsSelect** : Filtre par type
- **OdsDatagrid** : Tableau des emails
- **OdsPagination** : Navigation pages

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | "Historique des emails" |
| Description | "Consultez les emails envoyés par OVHcloud sur votre compte." |

## 5. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Recherche | Input | Texte libre |
| Type | Select | Tous les types, Facturation, Sécurité, Support, Services |

## 6. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Date | Datetime | Format DD/MM/YYYY HH:mm |
| Sujet | Texte | Objet de l'email |
| Type | Badge | Catégorie de l'email |
| Destinataire | Email masqué | Format je***@email.com |
| Actions | Lien | "Voir" |

## 7. BADGES TYPE
| Type | Couleur background | Couleur texte |
|------|--------------------|---------------|
| Facturation | #DBEAFE | #0050D7 |
| Sécurité | #FEF3C7 | #D97706 |
| Support | #D1FAE5 | #10B981 |
| Services | #E9D5FF | #7C3AED |

## 8. PAGINATION
| Élément | Description |
|---------|-------------|
| Compteur | "Affichage X-Y sur Z emails" |
| Navigation | Page X sur Y |

## 9. APIS & ENDPOINTS
- GET /me/document - Liste des emails envoyés
- GET /me/document/{documentId} - Détail d'un email
