# Cahier des Charges : home/hub - Widget Support

## 1. VUE D'ENSEMBLE
- **Type** : Widget liste tickets support
- **Parent** : Dashboard hub
- **Objectif** : Accès rapide aux tickets récents

## 2. STRUCTURE WIDGET
- Header avec titre et lien
- Liste 3 derniers tickets
- Statuts colorés
- Lien création ticket

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur blanc
- **OdsList** : Liste tickets
- **OdsBadge** : Statuts
- **OdsLink** : Navigation
- **OdsButton** : Créer ticket

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 240px |
| Border radius | 12px |
| Padding | 24px |

## 5. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Support" |
| Style | 18px semibold #0050D7 |
| Lien | "Tous les tickets →" |

## 6. LISTE TICKETS
| Colonne | Style |
|---------|-------|
| ID | #0050D7 lien |
| Sujet | 14px bold tronqué |
| Date | 12px muted |
| Statut | Badge coloré |

## 7. STATUTS TICKET
| Statut | Label | Couleur |
|--------|-------|---------|
| open | Ouvert | Vert #DEF7EC |
| closed | Fermé | Gris #F3F4F6 |
| pending | En attente | Jaune #FEF3C7 |

## 8. LIGNE TICKET
| Propriété | Valeur |
|-----------|--------|
| Hauteur | 56px |
| Padding | 12px |
| Border bottom | 1px #E5E7EB |
| Hover | Background #F9FAFB |

## 9. ACTION CTA
| Label | Style | Destination |
|-------|-------|-------------|
| "Créer un ticket" | Primary outline | /support/new |

## 10. ÉTAT VIDE
Si aucun ticket :
- Message : "Aucun ticket en cours"
- CTA : "Créer un ticket"

## 11. APIS & ENDPOINTS
- GET /support/tickets - Liste tickets
- GET /support/tickets?status=open - Tickets ouverts
- POST /support/tickets - Créer ticket
