# Cahier des Charges : home/hub - Dashboard Principal

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard client avec widgets
- **URL** : /hub
- **Objectif** : Vue d'ensemble des services, facturation, support

## 2. STRUCTURE PAGE
- Bannières contextuelles (top)
- Message de bienvenue personnalisé
- Grid widgets 2 colonnes responsive
- Layout adaptatif mobile

## 3. COMPOSANTS UI
- **OdsCard** : Conteneurs widgets
- **OdsMessage** : Bannières alertes
- **OdsBadge** : Compteurs
- **OdsButton** : Actions CTA
- **OdsSkeleton** : État loading

## 4. HEADER BIENVENUE
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bonjour {prénom}" |
| Sous-titre | "Bienvenue sur votre espace client OVHcloud" |
| Background | #FFFFFF |

## 5. LAYOUT WIDGETS
| Position | Widget | Dimensions |
|----------|--------|------------|
| Gauche haut | Facturation | 400x280px |
| Droite haut | Mes produits | 400x280px |
| Gauche milieu | Notifications | 400x200px |
| Droite milieu | Support | 400x240px |
| Bas | Suivi commandes | 800x200px |

## 6. RESPONSIVE
| Breakpoint | Colonnes |
|------------|----------|
| Desktop >1024px | 2 colonnes |
| Tablette 768-1024px | 2 colonnes |
| Mobile <768px | 1 colonne |

## 7. APIS & ENDPOINTS
- GET /me - Informations client
- GET /me/bill - Factures récentes
- GET /services - Liste services
- GET /me/notification - Notifications
- GET /support/tickets - Tickets support
