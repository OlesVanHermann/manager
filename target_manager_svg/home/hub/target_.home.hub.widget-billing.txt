# Cahier des Charges : home/hub - Widget Facturation

## 1. VUE D'ENSEMBLE
- **Type** : Widget carte avec gradient bleu
- **Parent** : Dashboard hub
- **Objectif** : Résumé facturation rapide

## 2. STRUCTURE WIDGET
- Fond gradient bleu OVHcloud
- Montant total prominent
- Sélecteur période
- Indicateur paiement

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur avec gradient
- **OdsText** : Montant, labels
- **OdsSelect** : Période
- **OdsChip** : Statut paiement
- **OdsLink** : Voir détails

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 280px |
| Border radius | 12px |
| Padding | 24px |

## 5. GRADIENT BACKGROUND
| Propriété | Valeur |
|-----------|--------|
| Type | linear-gradient |
| Direction | 135deg |
| Couleur 1 | #000E9C |
| Couleur 2 | #0050D7 |

## 6. CONTENU
| Élément | Style | Exemple |
|---------|-------|---------|
| Titre | 24px bold blanc | "Mes factures" |
| Montant | 48px bold blanc | "1 234,56 €" |
| Période | Select blanc outline | "Ce mois" |
| Statut | Chip vert/jaune | "Payé" / "En attente" |

## 7. SÉLECTEUR PÉRIODE
| Option | Valeur |
|--------|--------|
| Ce mois | current_month |
| Mois dernier | last_month |
| 3 derniers mois | last_3_months |
| Cette année | current_year |

## 8. STATUT PAIEMENT
| Statut | Couleur |
|--------|---------|
| Payé | Vert #DEF7EC |
| En attente | Jaune #FEF3C7 |
| Impayé | Rouge #FEE2E2 |

## 9. LIEN
| Label | Destination |
|-------|-------------|
| "Voir mes factures →" | /billing/history |

## 10. APIS & ENDPOINTS
- GET /me/bill - Liste factures
- GET /me/bill/debt - Solde impayé
