# Cahier des Charges : home/hub - Widget Notifications

## 1. VUE D'ENSEMBLE
- **Type** : Widget carrousel notifications
- **Parent** : Dashboard hub
- **Objectif** : Alertes et informations importantes

## 2. STRUCTURE WIDGET
- Header avec titre
- Carrousel messages
- Navigation dots
- Types colorés

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur blanc
- **OdsMessage** : Notifications colorées
- **OdsCarousel** : Navigation
- **OdsDots** : Indicateurs page

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400px |
| Hauteur | 200px |
| Border radius | 12px |
| Padding | 20px |

## 5. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Notifications" |
| Badge compteur | Nombre non lues |
| Style badge | Background bleu |

## 6. TYPES NOTIFICATIONS
| Type | Background | Border | Icône |
|------|------------|--------|-------|
| Info | #E8F4FD | #0050D7 | ℹ |
| Warning | #FEF3C7 | #F59E0B | ⚠ |
| Success | #DEF7EC | #059669 | ✓ |
| Error | #FEE2E2 | #DC2626 | ✕ |

## 7. STRUCTURE MESSAGE
| Élément | Style |
|---------|-------|
| Icône | 24px gauche |
| Titre | 14px semibold |
| Description | 13px muted |
| Date | 12px muted italic |

## 8. NAVIGATION CARROUSEL
| Élément | Style |
|---------|-------|
| Dots | Cercles 8px |
| Actif | #0050D7 plein |
| Inactif | #E5E7EB outline |
| Flèches | Chevron gauche/droite |

## 9. ACTIONS
| Action | Résultat |
|--------|----------|
| Clic notification | Navigue vers détail |
| Marquer lu | Supprime badge |
| Tout voir | /account/notifications |

## 10. APIS & ENDPOINTS
- GET /me/notification - Liste notifications
- PUT /me/notification/{id}/read - Marquer lue
- DELETE /me/notification/{id} - Supprimer
