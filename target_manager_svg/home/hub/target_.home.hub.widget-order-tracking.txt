# Cahier des Charges : home/hub - Widget Suivi Commande

## 1. VUE D'ENSEMBLE
- **Type** : Widget stepper progression
- **Parent** : Dashboard hub
- **Objectif** : Suivre l'avancement des commandes

## 2. STRUCTURE WIDGET
- Header avec ID commande
- Stepper horizontal 4 étapes
- Détails étape courante
- Lien détails complets

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur blanc
- **OdsStepper** : Progression étapes
- **OdsChip** : ID commande
- **OdsLink** : Navigation

## 4. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 800px (full width) |
| Hauteur | 200px |
| Border radius | 12px |
| Padding | 24px |

## 5. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Suivi de commande" |
| Chip ID | "#12345678" |
| Lien | "Voir détails →" |

## 6. STEPPER 4 ÉTAPES
| Étape | Label | Icône |
|-------|-------|-------|
| 1 | Commande reçue | ✓ cart |
| 2 | Paiement validé | ✓ credit-card |
| 3 | En préparation | ⟳ cog |
| 4 | Livré | ○ check |

## 7. ÉTATS ÉTAPE
| État | Style cercle | Style ligne |
|------|--------------|-------------|
| Complété | Vert plein | Vert |
| En cours | Bleu animé | Gris |
| À venir | Gris outline | Gris |

## 8. STYLE STEPPER
| Propriété | Valeur |
|-----------|--------|
| Cercle taille | 40px |
| Ligne hauteur | 4px |
| Espacement | 160px |
| Animation | Pulse pour étape active |

## 9. DÉTAIL ÉTAPE COURANTE
| Élément | Valeur |
|---------|--------|
| Label | "En préparation" |
| Description | "Votre serveur est en cours de configuration" |
| Temps estimé | "~15 minutes restantes" |

## 10. MASQUAGE
Widget masqué si aucune commande en cours

## 11. APIS & ENDPOINTS
- GET /me/order - Commandes récentes
- GET /me/order/{id}/status - Statut commande
- GET /me/order/{id}/details - Détails commande
