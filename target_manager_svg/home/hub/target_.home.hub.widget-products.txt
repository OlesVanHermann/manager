# Cahier des Charges : home/hub - Widget Mes Produits

## 1. VUE D'ENSEMBLE
- **Type** : Widget grille produits
- **Parent** : Dashboard hub
- **Objectif** : Accès rapide aux services actifs

## 2. STRUCTURE WIDGET
- Header avec titre et lien
- Grille icônes produits
- Compteur par catégorie
- Lien détails

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur blanc
- **OdsIcon** : Icônes catégories
- **OdsBadge** : Compteurs
- **OdsLink** : Navigation
