# Cahier des Charges : home/support - Liste Tickets

## 1. VUE D'ENSEMBLE
- **Page** : Mes demandes d'assistance
- **URL** : /support
- **Objectif** : Gérer les tickets de support

## 2. STRUCTURE PAGE
- Sidebar avec menu actif sur Support
- Breadcrumb
- Titre et sous-titre
- Actions (créer ticket, filtres)
- Tableau des tickets
- Pagination

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsButton** : "+ Créer une demande d'assistance"
- **OdsCheckbox** : "Afficher les tickets archivés"
- **OdsInput** : Recherche
- **OdsTable** : Liste tickets
- **OdsBadge** : Statut, Intervenant
- **OdsActionMenu** : Actions par ticket
- **OdsPagination** : Navigation pages

## 4. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| N° ticket | Link | Identifiant TKT-YYYY-XXXXX |
| Service | Text | Nom du service concerné |
| Sujet | Text | Titre du ticket |
| Dernier intervenant | Badge | Support OVH / Vous |
| Date création | Date | JJ/MM/AAAA |
| Dernière MAJ | Date | JJ/MM/AAAA |
| État | Badge | En cours / Clos |
| Actions | Menu ⋮ | Voir le détail |
