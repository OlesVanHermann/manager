# Cahier des Charges : home/changelog - Tab Hosting

## 1. VUE D'ENSEMBLE
- **Type** : Onglet changelog Hosting
- **URL** : /changelog?tab=hosting
- **Objectif** : Afficher les changements produits Hosting/Web

## 2. STRUCTURE PAGE
- Tabs avec Hosting actif
- Filtres par produit
- Datagrid des changements Hosting
- Pagination

## 3. COMPOSANTS UI
- **OdsTabs** : Cloud | Hosting (actif)
- **OdsSelect** : Filtre produit
- **OdsDatagrid** : Liste changements
- **OdsBadge** : Tags produits (jaune/orange)
