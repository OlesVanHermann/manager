# Cahier des Charges : home/procedures - Modal Confirm

## 1. VUE D'ENSEMBLE
- **Type** : Modal confirmation envoi documents
- **URL** : N/A (overlay sur page MFA)
- **Objectif** : Confirmer avant soumission définitive

## 2. STRUCTURE MODAL
- Overlay semi-transparent
- Carte modale blanche centrée
- Bouton fermer (X)
- Titre bleu
- Message avertissement
- Boutons Non/Oui

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsButton** : Fermer, Non, Oui
- **OdsText** : Message confirmation
