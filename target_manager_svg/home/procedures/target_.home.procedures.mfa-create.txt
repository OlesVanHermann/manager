# Cahier des Charges : home/procedures - MFA Create

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire KYC pour désactiver MFA
- **URL** : /procedures/mfa/create
- **Objectif** : Soumettre documents d'identité pour désactiver 2FA

## 2. STRUCTURE PAGE
- Logo OVHcloud centré
- Carte blanche centrée (800x950px)
- Titre explicatif bleu
- Instructions avec bullet points
- Formulaire upload documents
- Bouton soumission

## 3. COMPOSANTS UI
- **OdsCard** : Conteneur principal blanc avec ombre
- **OdsMessage** : Avertissement jaune
- **OdsInput[type=file]** : Upload documents
- **OdsButton** : Soumettre

## 4. TITRE ET INSTRUCTIONS
| Élément | Contenu |
|---------|---------|
| Titre L1 | "La vérification de votre identité est une étape" |
| Titre L2 | "essentielle de sécurisation de votre compte." |
| Instruction 1 | Masquer informations sensibles (numéros) |
| Instruction 2 | Vérifier visibilité nom, adresse, autorité émettrice |

## 5. AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Background | #FEF3C7 (jaune) |
| Texte couleur | #92400E |

## 6. FORMAT FICHIERS
| Propriété | Valeur |
|-----------|--------|
| Formats | jpg, jpeg, pdf, png |
| Taille max | 10 Mo par document |

## 7. CHAMPS UPLOAD
| Champ | Requis | Description |
|-------|--------|-------------|
| Pièce d'identité (recto) | Oui (*) | CNI, passeport, carte de séjour, permis |
| Pièce d'identité (verso) | Oui (*) | Verso du document |
| Justificatif de domicile | Oui (*) | Moins de 3 mois, adresse correspondante |
| Autres documents | Non | Max 5 fichiers supplémentaires |
