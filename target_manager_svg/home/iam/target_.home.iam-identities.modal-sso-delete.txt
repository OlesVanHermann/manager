# Cahier des Charges : home/iam - Modal supprimer fournisseur SSO

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Contexte** : Page SSO/SAML IAM
- **Objectif** : Supprimer la configuration d'un fournisseur SSO

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Messages d'avertissement
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "⚠️ Supprimer le fournisseur SSO" |
| Fond | Jaune warning (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL

### Message
| Texte |
|-------|
| "Êtes-vous sûr de vouloir supprimer la configuration SSO ?" (rouge) |
| "Les utilisateurs SSO ne pourront plus se connecter via ce fournisseur." (rouge) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger (rouge) | Supprime le fournisseur |

## 7. APIS & ENDPOINTS
- DELETE /me/identity/provider/saml2/{providerId} - Supprimer fournisseur SAML
