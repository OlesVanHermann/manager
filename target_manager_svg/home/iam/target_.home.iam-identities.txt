# Cahier des Charges : home/iam - Identités Utilisateurs

## 1. VUE D'ENSEMBLE
- **Type** : Page liste utilisateurs locaux
- **URL** : /iam/identities
- **Objectif** : Gérer utilisateurs du compte

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar IAM
- Onglets (Utilisateurs / Groupes / SSO)
- Bannière info connexion
- Bouton ajouter
- Tableau utilisateurs

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsMessage** : Info connexion
- **OdsDatagrid** : Liste utilisateurs
- **OdsBadge** : Statuts Activé/Désactivé
- **OdsButton** : Ajouter utilisateur
- **OdsActionMenu** : Actions

## 4. ONGLETS
| Onglet | Active | Route |
|--------|--------|-------|
| Utilisateurs locaux | Oui | /iam/identities |
| Groupes d'utilisateurs | Non | /iam/identities/groups |
| SSO | Non | /iam/identities/sso |

## 5. BANNIÈRE INFO CONNEXION
| Propriété | Valeur |
|-----------|--------|
| Background | #E8F0FE |
| Message L1 | "Afin de se connecter à l'interface, l'utilisateur devra renseigner votre NIC" |
| Message L2 | "{NIC}-ovh et son nom d'utilisateur séparé par un '/' ainsi que son mot de passe." |

## 6. COLONNES TABLEAU
| Colonne | Type |
|---------|------|
| Nom d'utilisateur | Lien + description |
| Groupe | Texte |
| Type | Texte (local) |
| Création | Date |
| Changement MDP | Date |
| Statut | Badge |
| Actions | Menu ⋮ |

## 7. ACTIONMENU OPTIONS
| Option | Couleur |
|--------|---------|
| Modifier l'utilisateur | Normal |
| Désactiver l'utilisateur | Normal |
| Supprimer l'utilisateur | Rouge |

## 8. APIS & ENDPOINTS
- GET /me/identity/user - Liste utilisateurs
- POST /me/identity/user - Créer utilisateur
- PUT /me/identity/user/{id} - Modifier
- DELETE /me/identity/user/{id} - Supprimer
- PUT /me/identity/user/{id}/disable - Désactiver
