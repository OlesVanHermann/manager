# Cahier des Charges : home/iam - Modal modifier fournisseur SSO

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Page SSO/SAML IAM
- **Objectif** : Modifier les paramètres d'un fournisseur d'identité SAML

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsCheckbox** : Utilisateurs locaux
- **OdsInput** : Attributs SAML
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le fournisseur SSO" |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Modifiez les paramètres de votre fournisseur d'identité SAML." |

## 6. FORMULAIRE

### Champs
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Activer également les utilisateurs locaux | Checkbox (coché) | ✓ |
| Attribut utilisateur | Input | http://schemas.xmlsoap.org/ws/2005/identity |
| Attribut groupe | Input | http://schemas.xmlsoap.org/claims/Group |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre les modifications |

## 8. APIS & ENDPOINTS
- PUT /me/identity/provider/saml2/{providerId} - Modifier fournisseur SAML
