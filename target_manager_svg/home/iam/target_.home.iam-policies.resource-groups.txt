# Cahier des Charges : home/iam - Groupes de Ressources

## 1. VUE D'ENSEMBLE
- **Type** : Page liste groupes ressources
- **URL** : /iam/policies/resource-groups
- **Objectif** : Regrouper ressources pour politiques

## 2. STRUCTURE PAGE
- Onglets (actif: Groupes de ressources)
- Bouton créer groupe
- Tableau groupes

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Liste groupes
- **OdsButton** : Créer groupe
- **OdsActionMenu** : Actions

## 4. ONGLETS
| Onglet | Active |
|--------|--------|
| Mes politiques | Non |
| Politiques OVHcloud | Non |
| Groupes de ressources | Oui |

## 5. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom du groupe | Lien bleu | 200px |
| Nombre de ressources | Nombre | 150px |
| Date de création | Date complète | 180px |
| Dernière modification | Date complète | 180px |
| Actions | Menu ⋮ | 50px |

## 7. ACTIONMENU OPTIONS
| Option | Couleur |
|--------|---------|
| Modifier le groupe de ressources | Normal |
| Supprimer le groupe de ressources | Rouge |

## 8. APIS & ENDPOINTS
- GET /me/iam/resourceGroup - Liste groupes
- POST /me/iam/resourceGroup - Créer groupe
- PUT /me/iam/resourceGroup/{id} - Modifier
- DELETE /me/iam/resourceGroup/{id} - Supprimer
