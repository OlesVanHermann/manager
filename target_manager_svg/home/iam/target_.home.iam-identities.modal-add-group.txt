# Cahier des Charges : home/iam - Modal Déclarer Groupe

## 1. VUE D'ENSEMBLE
- **Type** : Modal création groupe
- **Déclencheur** : Clic "Déclarer un groupe"
- **Objectif** : Créer groupe pour SSO/IAM

## 2. STRUCTURE MODAL
- Header avec titre
- Bannière info
- Formulaire 3 champs
- Footer actions

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur 600x460px
- **OdsMessage** : Info SSO
- **OdsInput** : Nom groupe
- **OdsSelect** : Rôle
- **OdsTextarea** : Description
- **OdsButton** : Annuler, Valider

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Déclarer un groupe" |
| Bouton fermer | × (top right) |

## 5. BANNIÈRE INFO
| Propriété | Valeur |
|-----------|--------|
| Background | #E8F0FE |
| L1 | "Afin que l'identité récupérée via l'identity provider soit liée à un rôle" |
| L2 | "OVHcloud, il est nécessaire de déclarer les groupes." |

## 6. FORMULAIRE
| Champ | Type | Requis | Placeholder |
|-------|------|--------|-------------|
| Nom du groupe | Input | Oui | "mon-groupe" |
| Rôle | Select | Oui | Options ci-dessous |
| Description | Textarea | Non | "Description optionnelle..." |

## 7. RÔLES DISPONIBLES
| Valeur | Label |
|--------|-------|
| ADMIN | Administrateur |
| REGULAR | Administrateur restreint |
| UNPRIVILEGED | Lecture seule |

## 8. ACTIONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Valider | Primary | "Valider" |

## 9. APIS & ENDPOINTS
- POST /me/identity/group - Créer groupe
- GET /me/identity/group/role - Rôles disponibles
