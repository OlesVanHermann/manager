# Cahier des Charges : home/billing - Désactiver renouvellement automatique

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation
- **URL** : /billing/autorenew/disable
- **Objectif** : Désactiver le renouvellement automatique d'un service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Facturation / Renouvellement
- **OdsLink** : Retour
- **OdsMessage** : Warning conséquences
- **OdsCard** : Informations service
- **OdsButton** : Annuler, Désactiver

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Désactiver le renouvellement automatique" |

## 5. MESSAGE WARNING
| Type | Contenu |
|------|---------|
| Warning (jaune) | "⚠️ Attention" |
| | "La désactivation du renouvellement automatique entraîne la suspension du service à la date d'expiration." |
| | "Vous devrez renouveler manuellement pour éviter toute interruption." |
| | "**Vos données peuvent être supprimées définitivement après la date d'expiration.**" |

## 6. CARD SERVICE
| Élément | Description |
|---------|-------------|
| Titre | "1 service sélectionné" |
| Description | "Le renouvellement automatique sera désactivé pour ce service :" |

### Détails service
| Champ | Valeur exemple |
|-------|----------------|
| Nom | example-domain.com |
| Type | Nom de domaine |
| Expiration | Expire le 28/01/2025 (rouge) |

## 7. MESSAGE DANGER
| Type | Contenu |
|------|---------|
| Danger (rouge) | "⚠️ Ce service ne sera plus renouvelé automatiquement" |

## 8. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Désactiver | Danger (rouge) | Désactive le renouvellement |

## 9. APIS & ENDPOINTS
- DELETE /services/{serviceId}/autoRenew - Désactiver renouvellement
- GET /services/{serviceId} - Informations service
