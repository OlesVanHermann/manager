# Cahier des Charges : home/billing - Détails du contrat

## 1. VUE D'ENSEMBLE
- **Type** : Page de consultation et acceptation
- **URL** : /billing/agreements/{agreementId}
- **Objectif** : Consulter et accepter les conditions générales de vente

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 750px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Facturation / Contrats / Détails
- **OdsLink** : Retour, Annexes
- **OdsCard** : Contenu du contrat (scrollable)
- **OdsCheckbox** : Acceptation des CGV
- **OdsButton** : Accepter le contrat
- **OdsBadge** : Déjà accepté (vert)

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Détails du contrat" |
| Titre | "Conditions Générales de Vente" |
| Description | "Ce document définit les conditions générales de vente applicables à tous les services OVHcloud." |

## 5. ZONE CONTRAT
| Propriété | Valeur |
|-----------|--------|
| Dimensions | 1100x380px |
| Fond | Blanc |
| Bordure | Grise |
| Scroll | Vertical avec indicateur |

### Structure du contrat
| Section | Contenu |
|---------|---------|
| ARTICLE 1 - OBJET | Définition des CGV et relation OVHcloud/Client |
| ARTICLE 2 - DÉFINITIONS | Compte Client, Services |
| ARTICLE 3 - COMMANDES | Processus de commande |
| ... | Suite du document |

## 6. SECTION ACCEPTATION
| Élément | Description |
|---------|-------------|
| Checkbox | "J'ai lu et j'accepte les Conditions Générales de Vente" |
| Lien | "Consulter les annexes du contrat" |

## 7. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Accepter le contrat | Primary | Valide l'acceptation |

### Badge statut
| État | Badge |
|------|-------|
| Déjà accepté | Vert "✓ Déjà accepté" |

## 8. APIS & ENDPOINTS
- GET /me/agreements - Liste des contrats
- GET /me/agreements/{id} - Détails d'un contrat
- POST /me/agreements/{id}/accept - Accepter le contrat
