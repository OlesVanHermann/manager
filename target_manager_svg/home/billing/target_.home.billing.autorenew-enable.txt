# Cahier des Charges : home/billing - Activer renouvellement automatique

## 1. VUE D'ENSEMBLE
- **Type** : Page de confirmation
- **URL** : /billing/autorenew/enable
- **Objectif** : Activer le renouvellement automatique d'un service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1200px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsBreadcrumb** : Mon compte / Facturation / Renouvellement
- **OdsLink** : Retour
- **OdsMessage** : Info moyen de paiement
- **OdsCard** : Informations service
- **OdsButton** : Annuler, Activer

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Lien retour | "← Activer le renouvellement automatique" |

| | "Le moyen de paiement utilisé sera : Carte bancaire - Visa •••• 4242" |
| | "Il s'agit de votre moyen de paiement par défaut." |
| | "Le prélèvement sera effectué automatiquement selon la fréquence de renouvellement du service." |

|---------|-------------|

| Champ | Valeur exemple |
|-------|----------------|
| Nom | ns1234567.ip-123-45-67.eu |
| Type | Serveur dédié |

## 7. MESSAGE SUCCÈS
| Type | Contenu |
|------|---------|
| Success (vert) | "✓ Ce service sera renouvelé automatiquement" |

## 8. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Activer | Primary | Active le renouvellement |

## 9. APIS & ENDPOINTS
- PUT /services/{serviceId}/autoRenew - Activer renouvellement
- GET /services/{serviceId} - Informations service
- GET /me/paymentMean/default - Moyen de paiement par défaut
