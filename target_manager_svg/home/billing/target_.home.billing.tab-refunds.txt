# Cahier des Charges : home/billing - Avoirs

## 1. VUE D'ENSEMBLE
- **Type** : Liste des avoirs
- **URL** : /billing/refunds
- **Objectif** : Consulter et utiliser les avoirs

## 2. STRUCTURE PAGE
- Tabs avec Mes avoirs actif
- Datagrid des avoirs
- Carte solde disponible

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation
- **OdsDatagrid** : Liste avoirs
- **OdsBadge** : Statuts (Utilisé, Disponible)
- **OdsCard** : Solde avoirs
- **OdsPagination** : Navigation
