# Cahier des Charges : network/cdn - Logs en temps réel

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet logs
- **Contexte** : Dashboard CDN
- **Objectif** : Afficher les logs en temps réel du CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% |
| Hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsText** : Titre et indicateur live
- **OdsIndicator** : Point vert "En direct"
- **Terminal** : Container logs (fond sombre)

## 4. NAVIGATION ONGLETS
| Onglet | État |
|--------|------|
| Statistiques | Inactif |
| SSL | Inactif |
| Logs | Actif |
| Domaines | Inactif |

## 5. HEADER SECTION
| Élément | Description |
|---------|-------------|
| Titre | "Logs en temps réel" (18px, gras) |
| Indicateur | Point vert + "En direct" |

## 6. TERMINAL LOGS

### Container
| Propriété | Valeur |
|-----------|--------|
| Fond | Sombre (#1F2937) |
| Bordure | Arrondie (8px) |
| Police | Monospace |
| Taille | 12px |

### Format des logs
```
{IP} - - [{date}] "{method} {url} HTTP/1.1" {status} {size} "-" "{user-agent}" {cache-status}
```

### Code couleur
| Code HTTP | Couleur |
|-----------|---------|
| 2xx (Success) | Bleu (#60A5FA) |
| 3xx (Redirect) | Jaune (#FBBF24) |
| 4xx (Client Error) | Jaune (#FBBF24) |
| 5xx (Server Error) | Rouge (#F87171) |

### Légende
| Statut | Couleur | Label |
|--------|---------|-------|
| 2xx | Bleu | Success |
| 3xx/4xx | Jaune | Redirect/Client |
| 5xx | Rouge | Server Error |

## 7. APIS & ENDPOINTS
- GET /cdn/{serviceName}/logs - Récupérer logs temps réel (WebSocket ou polling)
