# Cahier des Charges : network/cdn - Modal Purge

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'action
- **Déclencheur** : Bouton "Purge" ou "Vider le cache"
- **Objectif** : Vider le cache pour un domaine

## 2. STRUCTURE MODAL
- Overlay sombre (40% opacité)
- Modal centrée (500x400px)
- Header avec titre et bouton fermer
- Message d'avertissement
- Info domaine
- Options de purge
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsMessage** : Avertissement (warning jaune)
- **OdsText** : Domaine concerné
- **OdsRadio** : Type de purge
- **OdsButton** : Annuler/Purger

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Purger le cache" |
| Bouton fermer | × (coin droit) |
| Séparateur | Ligne grise |

## 5. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Icône | ⚠️ |
| Titre | "Attention" |
| Message | "Cette action videra le cache pour le domaine sélectionné." |

## 6. INFO DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Label | "Domaine concerné" |
| Valeur | www.example.com |
| Style | Monospace, fond gris |

## 7. OPTIONS DE PURGE
| Option | Sélection | Description |
|--------|-----------|-------------|
| Purge complète | ● Sélectionné | Tout le cache du domaine |
| Purge sélective | ○ Non sélectionné | Par URL spécifique |

## 8. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Purger | Warning orange | Droite |

## 9. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains/{domain}/flush - Purge complète
- POST /cdn/dedicated/{serviceName}/domains/{domain}/flush - Purge sélective
  - body: { urls: [...] }
