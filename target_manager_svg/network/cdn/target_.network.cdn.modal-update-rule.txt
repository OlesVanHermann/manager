# Cahier des Charges : network/cdn - Modal modifier TTL règle

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard (2 étapes)
- **Contexte** : Page règles de cache CDN
- **Objectif** : Modifier le TTL d'une règle de cache existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 290px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : 2 étapes
- **OdsText** : Informations règle et TTL actuel
- **OdsInput** : Nouveau TTL
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le TTL" |
| Fond | Gris (#F9FAFB) |
| Bouton fermer | × |

## 5. STEPPER (2 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Configuration | Actif |
| 2 | Confirmation | Inactif |

## 6. ÉTAPE 1 - MODIFICATION

### Informations affichées
| Élément | Description |
|---------|-------------|
| Règle concernée | Extensions ou pattern de la règle |
| Explication | "Le TTL définit la durée de mise en cache des fichiers." |
| TTL actuel | Valeur en secondes |

### Champ
| Champ | Type | Description |
|-------|------|-------------|
| Nouveau TTL (secondes) * | Input number | Nouvelle valeur TTL |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary | Passe à l'étape de confirmation |

## 8. APIS & ENDPOINTS
- PUT /cdn/{serviceName}/domains/{domain}/cacheRules/{ruleId} - Modifier TTL
