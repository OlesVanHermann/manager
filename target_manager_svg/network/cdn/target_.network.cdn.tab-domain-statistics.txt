# Cahier des Charges : network/cdn - Statistiques domaine

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet statistiques
- **Contexte** : Vue détail d'un domaine CDN
- **Objectif** : Afficher les statistiques de trafic du domaine

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% |
| Hauteur | 700px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsTile** : Cartes d'informations
- **OdsSelect** : Filtres (métrique, période)
- **OdsChart** : Graphique de trafic
- **OdsActionMenu** : Menu actions réseau

## 4. NAVIGATION ONGLETS
| Onglet | État |
|--------|------|
| Général | Inactif |
| Règles de cache | Inactif |
| Statistiques | Actif |
| Logs | Inactif |

## 5. TUILES INFORMATIONS

### Tuile Statistiques
| Élément | Valeur |
|---------|--------|
| Titre | "Statistiques" |
| CNAME | cdn-{id}.cdn-ovh.net |

### Tuile Réseau
| Élément | Valeur |
|---------|--------|
| Titre | "Réseau" |
| Domaine | Nom du domaine |
| IPv4 | Adresse IP |
| Actions | Menu ⋮ |

## 6. SECTION GRAPHIQUE

### Filtres
| Filtre | Type | Options |
|--------|------|---------|
| Visualiser | Select | Bande passante, Requêtes, Hit rate |
| Depuis | Select | Dernière heure, 24h, semaine, mois |

### Graphique
| Propriété | Valeur |
|-----------|--------|
| Type | Line chart avec aire |
| Axe Y | Valeur en Mbps/Gbps |
| Axe X | Timeline (jours/heures) |
| Couleur | Bleu OVH (#0050D7) |

## 7. APIS & ENDPOINTS
- GET /cdn/{serviceName}/domains/{domain}/statistics - Récupérer statistiques
