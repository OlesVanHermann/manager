# Cahier des Charges : network/cdn - Modal vider cache règle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page règles de cache CDN
- **Objectif** : Purger le cache pour une règle spécifique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Vider le cache

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Vider le cache de la règle" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir vider le cache pour cette règle ?" |
| Règle | ".jpg, .png, .gif" (gras) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Vider le cache | Warning (orange) | Purge le cache de la règle |

## 7. APIS & ENDPOINTS
- POST /cdn/{serviceName}/domains/{domain}/cacheRules/{ruleId}/flush - Purger cache règle
