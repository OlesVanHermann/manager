# Cahier des Charges : network/cdn - Modal Ajouter Domaine

## 1. VUE D'ENSEMBLE
- **Type** : Modal formulaire
- **Déclencheur** : Bouton "+ Ajouter" dans onglet Domaines
- **Objectif** : Ajouter un domaine au CDN

## 2. STRUCTURE MODAL
- Overlay sombre (40% opacité)
- Modal centrée (600x500px)
- Header avec titre et bouton fermer
- Formulaire (3 champs)
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsInput** : Domaine, Backend
- **OdsSelect** : Certificat SSL
- **OdsFormField** : Labels et helpers
- **OdsButton** : Annuler/Ajouter

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Ajouter un domaine" |
| Bouton fermer | × (coin droit) |
| Séparateur | Ligne grise |

## 5. CHAMPS FORMULAIRE

### Champ 1 - Domaine
| Propriété | Valeur |
|-----------|--------|
| Label | "Domaine *" |
| Type | Input text |
| Placeholder | "www.example.com" |
| Helper | "Entrez le domaine que vous souhaitez ajouter au CDN" |
| Required | Oui |

### Champ 2 - Backend
| Propriété | Valeur |
|-----------|--------|
| Label | "Backend (origine) *" |
| Type | Input text |
| Placeholder | "origin.example.com" |
| Helper | "Serveur d'origine vers lequel le CDN redirigera les requêtes" |
| Required | Oui |

### Champ 3 - Certificat SSL
| Propriété | Valeur |
|-----------|--------|
| Label | "Certificat SSL" |
| Type | Select dropdown |
| Valeur par défaut | "Let's Encrypt (automatique)" |
| Required | Non |

## 6. OPTIONS SSL
| Option | Description |
|--------|-------------|
| Let's Encrypt (automatique) | Certificat gratuit auto-renouvelé |
| Certificat personnalisé | Import de certificat |
| Aucun | HTTP uniquement |

## 7. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline gris | Gauche |
| Ajouter | Primary bleu | Droite |

## 8. APIS & ENDPOINTS
- POST /cdn/dedicated/{serviceName}/domains - Créer domaine
  - body: { domain, backend, ssl }
