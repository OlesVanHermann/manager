# Cahier des Charges : network/cdn - Onglet Domaines

## 1. VUE D'ENSEMBLE
- **Page** : Gestion des domaines CDN
- **URL** : /network/cdn/{serviceName}?tab=domains
- **Objectif** : Gérer les domaines associés au CDN

## 2. STRUCTURE PAGE
- Header avec titre service
- Onglets avec Domaines actif
- Bouton "+ Ajouter"
- Tableau des domaines

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Conteneur principal
- **OdsButton** : "+ Ajouter" (primary)
- **OdsTable** : Liste des domaines
- **OdsBadge** : active (vert), configuring (jaune)
- **OdsButton** : Configurer, Purge, Supprimer

## 4. ONGLETS
| Onglet | Actif |
|--------|-------|
| Général | - |
| Domaines | ✓ |
| Statistiques | - |
| Tâches | - |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Domaine | Text bold | Nom du domaine |
| CNAME | Monospace muted | Cible CDN |
| Statut | Badge | active/configuring |
| Actions | Buttons | Configurer, Purge, Supprimer |

## 6. BADGES STATUT
| Statut | Badge | Couleur |
|--------|-------|---------|
| active | success | Vert |
| configuring | warning | Jaune |

## 7. BOUTONS ACTIONS LIGNE
| Bouton | Style | Action |
|--------|-------|--------|
| Configurer | Outline gris | Configuration avancée |
| Purge | Outline gris | Ouvre modal purge |
| Supprimer | Danger rouge | Supprime le domaine |

## 8. DONNÉES EXEMPLE
| Domaine | CNAME | Statut |
|---------|-------|--------|
| www.example.com | cdn-abc123.ovh.net | active |
| static.example.com | cdn-abc123.ovh.net | active |
| api.example.com | cdn-abc123.ovh.net | configuring |

## 9. APIS & ENDPOINTS
- GET /cdn/dedicated/{serviceName}/domains - Liste domaines
- POST /cdn/dedicated/{serviceName}/domains - Ajouter domaine
- DELETE /cdn/dedicated/{serviceName}/domains/{domain} - Supprimer
- POST /cdn/dedicated/{serviceName}/domains/{domain}/flush - Purge
