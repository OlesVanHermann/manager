# Cahier des Charges : network/cdn - Statistiques générales

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet statistiques
- **Contexte** : Dashboard CDN principal
- **Objectif** : Afficher les statistiques globales du service CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% |
| Hauteur | 650px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsTile** : Cartes métriques
- **OdsSelect** : Filtres graphique
- **OdsChart** : Graphique de trafic

## 4. NAVIGATION ONGLETS
| Onglet | État |
|--------|------|
| Statistiques | Actif |
| SSL | Inactif |
| Logs | Inactif |
| Domaines | Inactif |

## 5. TUILES MÉTRIQUES

### Disposition
| Position | Largeur |
|----------|---------|
| 4 tuiles | 260px chacune |

### Contenu des tuiles
| Tuile | Label | Valeur exemple |
|-------|-------|----------------|
| 1 | Bande passante utilisée | 1.2 TB |
| 2 | Requêtes totales | 45.2 M |
| 3 | Taux de cache | 94.5% (vert) |
| 4 | Domaines actifs | 8 |

## 6. SECTION GRAPHIQUE

### Titre
| Texte |
|-------|
| "Trafic" |

### Filtres
| Filtre | Type | Options |
|--------|------|---------|
| Visualiser | Select | Bande passante, Requêtes, Hit rate |
| Période | Select | Dernière heure, 24h, semaine, mois |

### Graphique
| Propriété | Valeur |
|-----------|--------|
| Type | Line chart avec aire |
| Axe Y | Valeur en Mbps |
| Axe X | Timeline avec dates |
| Couleur | Bleu OVH (#0050D7) |
| Hauteur | 370px |

## 7. APIS & ENDPOINTS
- GET /cdn/{serviceName}/statistics - Récupérer statistiques globales
- GET /cdn/{serviceName}/quota - Quota bande passante
