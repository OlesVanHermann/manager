# Cahier des Charges : network/cdn - Modal ajouter certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page SSL CDN
- **Objectif** : Importer un certificat SSL personnalisé

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 490px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom du certificat
- **OdsTextarea** : Certificat, Clé privée, Chaîne
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter un certificat SSL" |
| Bouton fermer | × |

## 5. DESCRIPTION
| Texte |
|-------|
| "Importez votre certificat SSL pour activer HTTPS sur votre CDN." |

## 6. FORMULAIRE

### Champs
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nom du certificat * | Input | "Ex: mon-certificat" | Oui |
| Certificat SSL | Textarea (80px) | "-----BEGIN CERTIFICATE-----..." | Oui |
| Clé privée | Textarea (80px) | "-----BEGIN PRIVATE KEY-----..." | Oui |
| Chaîne de certificats | Textarea (40px) | "Certificats intermédiaires..." | Non |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Importe le certificat |

## 8. APIS & ENDPOINTS
- POST /cdn/{serviceName}/ssl - Ajouter certificat SSL
