# Cahier des Charges : network/cdn - Modal commander backends

## 1. VUE D'ENSEMBLE
- **Type** : Modal wizard commande (4 étapes)
- **Contexte** : Dashboard CDN
- **Objectif** : Commander des backends supplémentaires pour le CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsStepper** : 4 étapes (Quantité, Durée, Confirmation, Paiement)
- **OdsInput** : Nombre de backends
- **OdsMessage** : Info tarif
- **OdsButton** : Annuler, Suivant

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Commander des backends" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. STEPPER (4 étapes)
| Étape | Label | État |
|-------|-------|------|
| 1 | Quantité | Actif |
| 2 | Durée | Inactif |
| 3 | Confirmation | Inactif |
| 4 | Paiement | Inactif |

## 6. ÉTAPE 1 - QUANTITÉ

### Question
| Texte |
|-------|
| "Combien de backends supplémentaires souhaitez-vous commander ?" |

### Champ
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Nombre de backends | Input number | 5 |

### Info tarif
| Type | Contenu |
|------|---------|
| Info (bleu clair) | "Tarif unitaire :" |
| | "**4,99 € HT / mois / backend**" (rouge) |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Suivant | Primary | Passe à l'étape suivante |

## 8. APIS & ENDPOINTS
- GET /cdn/{serviceName}/option/backends - Options backends disponibles
- POST /order/cdn/{serviceName}/option/backends - Commander backends
