# Cahier des Charges : network/cdn - Modal activer règle de cache

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page règles de cache CDN
- **Objectif** : Activer une règle de cache désactivée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Activer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Activer la règle de cache" |
| Fond | Vert (#D1FAE5) |
| Couleur texte | Vert (#10B981) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir activer cette règle ?" |
| Règle | ".jpg, .png, .gif" (gras) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Activer | Success (vert) | Active la règle |

## 7. APIS & ENDPOINTS
- PUT /cdn/{serviceName}/domains/{domain}/cacheRules/{ruleId} - Activer règle (status: on)
