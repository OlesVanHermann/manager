# Cahier des Charges : network/cdn - Modal vider cache domaine

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page domaines CDN
- **Objectif** : Purger le cache pour un domaine entier

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message confirmation
- **OdsButton** : Annuler, Vider le cache

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Vider le cache" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir vider le cache pour ce domaine ?" |
| Domaine | cdn.example.com (gras) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Vider le cache | Warning (orange) | Purge le cache |

## 7. APIS & ENDPOINTS
- POST /cdn/{serviceName}/domains/{domain}/flush - Purger cache domaine
