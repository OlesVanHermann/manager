# Cahier des Charges : network/cdn - Gestion SSL

## 1. VUE D'ENSEMBLE
- **Type** : Page onglet SSL
- **Contexte** : Dashboard CDN principal
- **Objectif** : Gérer le certificat SSL du CDN

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsTile** : Carte informations certificat
- **OdsText** : Labels et valeurs
- **OdsBadge** : Statut certificat
- **OdsButton** : Actions SSL

## 4. NAVIGATION ONGLETS
| Onglet | État |
|--------|------|
| Statistiques | Inactif |
| SSL | Actif |
| Logs | Inactif |
| Domaines | Inactif |

## 5. TUILE CERTIFICAT SSL

### Header tuile
| Élément | Valeur |
|---------|--------|
| Titre | "Certificat SSL" |
| Largeur | 750px |
| Hauteur | 280px |

### Informations affichées
| Champ | Valeur exemple |
|-------|----------------|
| Nom | letsencrypt-cdn |
| Common Name (CN) | cdn.example.com |
| Statut | Badge "Actif" (vert) |
| Valide depuis | 15 janvier 2025 |
| Valide jusqu'à | 15 avril 2025 |
| Fournisseur | Let's Encrypt |

## 6. BOUTONS ACTIONS

### Disposition
| Position | Description |
|----------|-------------|
| Colonne droite | À côté de la tuile |

### Actions disponibles
| Bouton | Style | Action |
|--------|-------|--------|
| Régénérer le certificat | Secondary | Ouvre modal-ssl-generate |
| Mettre à jour | Secondary | Ouvre modal-ssl-update |
| Supprimer | Danger (texte rouge) | Ouvre modal-ssl-delete |

## 7. ÉTATS POSSIBLES

### Avec certificat
- Affiche les informations du certificat
- Toutes les actions disponibles

### Sans certificat
- Message "Aucun certificat SSL configuré"
- Boutons "Générer Let's Encrypt" ou "Importer un certificat"

## 8. APIS & ENDPOINTS
- GET /cdn/{serviceName}/ssl - Récupérer certificat SSL
- POST /cdn/{serviceName}/ssl - Ajouter certificat SSL
- POST /cdn/{serviceName}/ssl/letsencrypt - Générer Let's Encrypt
- PUT /cdn/{serviceName}/ssl/{sslId} - Mettre à jour certificat
- DELETE /cdn/{serviceName}/ssl/{sslId} - Supprimer certificat
