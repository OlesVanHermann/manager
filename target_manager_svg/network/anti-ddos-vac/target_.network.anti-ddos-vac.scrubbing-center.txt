# Cahier des Charges : network/security - Scrubbing Center

## 1. VUE D'ENSEMBLE
- **Page** : Configuration Scrubbing Center
- **URL** : /network/security/scrubbing-center
- **Objectif** : Gérer la mitigation Anti-DDoS par IP

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre
- Onglets avec "Scrubbing Center" actif
- Message d'information
- Datagrid des IP
- ActionMenu par IP

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Vue d'ensemble, Scrubbing Center (actif), Trafic
- **OdsMessage** : Information (info bleu)
- **OdsDatagrid** : Liste des IP
- **OdsBadge** : Mitigation (Active), Mode, Statut
- **OdsActionMenu** : Actions par IP

## 4. MESSAGE INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| L1 | "Le Scrubbing Center filtre le trafic malveillant lors d'une attaque DDoS." |
| L2 | "Vos IP sont automatiquement protégées. Vous pouvez personnaliser la mitigation pour chaque IP." |

## 5. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| IP | Text | Adresse IP |
| Service | Text | Serveur associé |
| Mitigation | Badge | Active (vert) |
| Mode | Text | Automatique/Permanent |
| Statut actuel | Badge | Normal/Mitigation |
| Actions | Menu ⋮ | Menu contextuel |

## 6. BADGES
| Type | Badge | Couleur |
|------|-------|---------|
| Active | success | Vert |
| Normal | success | Vert |
| Mitigation | warning | Jaune |

## 7. ACTION MENU
| Action | Description |
|--------|-------------|
| Voir les statistiques | Graphiques de l'IP |
| Mode permanent | Mitigation toujours active |
| Mode automatique | Mitigation à la demande |
| Forcer la mitigation | Active immédiatement |

## 8. DONNÉES EXEMPLE
| IP | Service | Mitigation | Mode | Statut |
|----|---------|------------|------|--------|
| 123.45.67.89 | ns1234567.ip-123-45-67.eu | Active | Automatique | Normal |
| 98.76.54.32 | ns7654321.ip-98-76-54.eu | Active | Permanent | Mitigation |
| 192.168.100.1 | pn-123456 (vRack) | Active | Automatique | Normal |

## 9. APIS & ENDPOINTS
- GET /ip - Liste des IP
- GET /ip/{ip}/mitigation - Mode mitigation
- PUT /ip/{ip}/mitigation - Modifier mode
- POST /ip/{ip}/mitigation/force - Forcer mitigation
