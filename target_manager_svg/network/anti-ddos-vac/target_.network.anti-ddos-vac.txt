# Cahier des Charges : network/security - Dashboard Anti-DDoS

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard Network Security Anti-DDoS
- **URL** : /network/security
- **Objectif** : Vue d'ensemble de la protection Anti-DDoS

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation Network
- Sidebar avec Network Security actif
- Breadcrumb (Network > Network Security)
- Titre
- Onglets navigation
- Cartes statistiques (4)
- Timeline des attaques
- Liens rapides

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation contextuelle
- **OdsText** : Titre "Network Security - Anti-DDoS"
- **OdsTabs** : Vue d'ensemble (actif), Scrubbing Center, Trafic
- **OdsCard** : Cartes statistiques (4)
- **OdsIcon** : Indicateurs statut (cercles colorés)
- **OdsTimeline** : Historique attaques (7 jours)
- **OdsButton** : Liens rapides (outline)

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Vue d'ensemble | Dashboard principal (actif) |
| Scrubbing Center | Configuration mitigation par IP |
