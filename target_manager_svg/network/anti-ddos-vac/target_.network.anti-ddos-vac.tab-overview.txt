# Cahier des Charges : network/security - Onglet Vue d'ensemble

## 1. VUE D'ENSEMBLE
- **Page** : Vue d'ensemble d'un bloc IP
- **URL** : /network/security/{ip}?tab=overview
- **Objectif** : Afficher les informations de sécurité d'un bloc IP

## 2. STRUCTURE PAGE
- Breadcrumb avec IP
- Titre avec badge mode
- Onglets (Vue d'ensemble actif, Attaques, Firewall)
- Informations bloc IP
- Zone Protection Anti-DDoS
- Zone Actions

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation avec IP
- **OdsText** : Titre avec IP
- **OdsBadge** : Mode "auto" (vert)
- **OdsTabs** : Navigation onglets
- **OdsButton** : Rafraîchir (outline)
- **OdsCard** : Zones d'information
- **OdsIcon** : Indicateur statut protection
