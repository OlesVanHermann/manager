# Cahier des Charges : network/security - Onglet Firewall

## 1. VUE D'ENSEMBLE
- **Page** : Règles de firewall d'un bloc IP
- **URL** : /network/security/{ip}?tab=firewall
- **Objectif** : Gérer les règles de firewall Network

## 2. STRUCTURE PAGE
- Breadcrumb et titre
- Onglets avec "Firewall" actif
- Bouton "+ Ajouter une règle"
- Tableau des règles

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : "+ Ajouter une règle" (primary)
- **OdsTable** : Liste des règles
- **OdsBadge** : Action (permit/deny)
- **OdsButton** : Supprimer (danger)

## 4. ONGLETS
| Onglet | Actif |
|--------|-------|
| Vue d'ensemble | - |
| Attaques | - |
| Firewall | ✓ |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| # | Number | Numéro de séquence |
| Action | Badge | permit/deny |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/firewall/{ipOnFirewall}/rule - Liste règles
- POST /ip/{ip}/firewall/{ipOnFirewall}/rule - Créer règle
- DELETE /ip/{ip}/firewall/{ipOnFirewall}/rule/{sequence} - Supprimer
