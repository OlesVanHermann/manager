# Cahier des Charges : network/vrack-services - Onglet Endpoints

## 1. VUE D'ENSEMBLE
- **Page** : Gestion des Service Endpoints
- **URL** : /network/vrack-services/{id}/endpoints
- **Objectif** : Créer et gérer les endpoints

## 2. STRUCTURE PAGE
- Onglets (Service Endpoints actif)
- Bouton création
- Datagrid endpoints

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : "+ Créer un Service Endpoint"
- **OdsDatagrid** : Liste endpoints
- **OdsActionMenu** : Actions par ligne

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Adresse IP | Text | 10.0.0.5 |
| Sous-réseau | Text | 10.0.0.0/24 |
| Nom du service | Link | efs-storage-prod-01 |
| Type de service | Text | Enterprise File Storage |
| Actions | Menu ⋮ | Actions |

## 5. ACTION MENU
| Action | Style |
|--------|-------|
| Supprimer le Service Endpoint | Danger |

## 6. TYPES DE SERVICES SUPPORTÉS
| Type | Description |
|------|-------------|
| Enterprise File Storage | Stockage fichiers |
| Netapp | Stockage Netapp |

## 7. APIS & ENDPOINTS
- GET /vrackServices/{id}/endpoints - Liste
- POST /vrackServices/{id}/endpoints - Créer
- DELETE /vrackServices/{id}/endpoints/{endpointId} - Supprimer
