# Cahier des Charges : network/vrack-services - Modal Supprimer

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression du vRack Services
- **Déclencheur** : Action "Supprimer"
- **Objectif** : Supprimer définitivement le service

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 400px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsIcon** : Warning (cercle rouge)
- **OdsText** : Messages
- **OdsInput** : Champ confirmation
- **OdsButton** : Annuler / Supprimer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Supprimer le vRack Services" |
| Body | Warning + messages + input |
