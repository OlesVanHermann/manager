# Cahier des Charges : network/vrack-services - Modal Supprimer sous-réseau

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un sous-réseau
- **Déclencheur** : Action "Supprimer le sous-réseau"
- **Objectif** : Supprimer un sous-réseau

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 250px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Message d'avertissement
- **OdsButton** : Annuler / Supprimer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Supprimer le sous-réseau" |
| Body | Message condition |
| Footer | Boutons |

## 5. MESSAGE
```
Pour être supprimé, un sous-réseau ne doit pas contenir
de Service Endpoint associé.
```

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer |
| Supprimer | Danger | Exécuter |

## 7. CONDITION
- Le sous-réseau ne peut être supprimé que s'il n'a aucun endpoint
- Si endpoints présents : bouton Supprimer désactivé

## 8. APIS & ENDPOINTS
- GET /vrackServices/{id}/subnets/{subnetId}/endpoints - Vérifier endpoints
- DELETE /vrackServices/{id}/subnets/{subnetId} - Supprimer
