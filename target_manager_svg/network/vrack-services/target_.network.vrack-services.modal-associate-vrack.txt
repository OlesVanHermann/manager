# Cahier des Charges : network/vrack-services - Modal Associer vRack

## 1. VUE D'ENSEMBLE
- **Modal** : Association à un vRack
- **Déclencheur** : Action "Associer à un vRack"
- **Objectif** : Lier le vRack Services à un vRack

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 300px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Message explicatif
- **OdsSelect** : Sélecteur vRack
- **OdsButton** : Annuler / Associer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Associer à un vRack" |
| Body | Message + sélecteur |
| Footer | Boutons |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Message | "Sélectionnez un vRack pour y associer votre vRack Services." |
| Label | "vRack" |
| Select | Liste des vRack disponibles |
| Format option | "pn-XXXXXX - Description" |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer |
| Associer | Primary | Exécuter |

## 7. APIS & ENDPOINTS
- GET /vrack - Liste des vRack disponibles
- POST /vrackServices/{id}/vrack/associate - Associer
