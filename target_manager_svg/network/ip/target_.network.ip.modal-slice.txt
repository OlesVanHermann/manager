# Cahier des Charges : network/ip - Modal Segmenter Bloc

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'action
- **Déclencheur** : Action "Segmenter le bloc" dans le menu IP
- **Objectif** : Diviser un bloc IP en sous-réseaux plus petits

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x480px)
- Header avec titre dynamique et bouton fermer
- Select dropdown pour taille des blocs enfants
- Zone d'information sur les blocs créés
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsSelect** : Choix de la taille CIDR
- **OdsList** : Liste des blocs qui seront créés
- **OdsText** : Labels et descriptions
- **OdsButton** : Annuler/Confirmer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Segmenter le bloc {IP}/{CIDR}" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. FORMULAIRE

### Instructions
| Propriété | Valeur |
