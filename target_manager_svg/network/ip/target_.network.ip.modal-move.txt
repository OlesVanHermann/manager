# Cahier des Charges : network/ip - Modal Déplacer IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'action
- **Déclencheur** : Action "Déplacer l'IP" dans le menu
- **Objectif** : Déplacer une IP failover vers un autre service

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x480px)
- Header avec titre et bouton fermer
- Avertissement
- Formulaire
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsMessage** : Avertissement (warning)
- **OdsInput** : IP à déplacer (readonly), Service actuel (readonly)
- **OdsSelect** : Service de destination
- **OdsButton** : Annuler/Déplacer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Déplacer l'IP" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Icône | ⚠️ |
| Titre | "Attention" |
| Texte | "Le déplacement d'une IP failover peut entraîner une courte interruption de service (quelques secondes)." |

### Service de destination
| Propriété | Valeur |

## 8. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Déplacer | Primary (bleu) | Droite |

## 9. SERVICES COMPATIBLES
| Type | Description |
|------|-------------|
| Serveur dédié | Serveurs dans le même datacenter |
| Private Cloud | VMware Hosted |
| VPS | Serveurs virtuels OVHcloud |
| Public Cloud | Instances Public Cloud |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/destination - Services compatibles
- POST /ip/{ip}/move - Déplacer l'IP
