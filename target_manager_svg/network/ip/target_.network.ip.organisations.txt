# Cahier des Charges : network/ip - Organisations

## 1. VUE D'ENSEMBLE
- **Page** : Liste des organisations IP (RIPE/ARIN)
- **URL** : /network/ip/organisations
- **Objectif** : Gérer les informations WHOIS des blocs IP

## 2. STRUCTURE PAGE
- Sidebar Network
- Onglets (Toutes mes IP / Mes organisations)
- Titre + description
- Bouton ajouter
- Datagrid des organisations

## 3. COMPOSANTS UI
- **OdsTabs** : "Toutes mes IP" / "Mes organisations" (actif)
- **OdsText** : Titre et descriptions
- **OdsButton** : "Ajouter une organisation" (outline)
- **OdsDatagrid** : Liste des organisations
- **OdsActionMenu** : Actions par organisation

## 11. APIS & ENDPOINTS
- GET /me/ipOrganisation - Liste organisations
- GET /me/ipOrganisation/{organisationId} - Détails
- POST /me/ipOrganisation - Créer organisation
- PUT /me/ipOrganisation/{organisationId} - Modifier
- DELETE /me/ipOrganisation/{organisationId} - Supprimer
