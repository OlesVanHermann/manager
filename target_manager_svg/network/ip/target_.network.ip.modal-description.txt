# Cahier des Charges : network/ip - Modal Modifier Description

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Déclencheur** : Action "Modifier la description" dans le menu IP
- **Objectif** : Ajouter ou modifier une description personnalisée pour l'IP

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x380px)
- Header avec titre et bouton fermer
- Textarea avec compteur de caractères
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsTextarea** : Zone de texte description
- **OdsText** : Label et compteur
- **OdsButton** : Annuler/Valider

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Modifier la description" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. FORMULAIRE

### Label avec compteur
| Élément | Position | Style |
|---------|----------|-------|
| Label | "Description:" | Gauche, semi-bold |
| Compteur | "45/250" | Droite, gris, 12px |

### Textarea
| Propriété | Valeur |
|-----------|--------|
| Largeur | 540px |
| Hauteur | 120px |
| Bordure | Grise #D1D5DB |
| Fond | Blanc |
| Placeholder | - |
| Max caractères | 250 |

### Exemple de contenu
```
Serveur web de production - Frontend
Application e-commerce
```

## 6. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Valider | Primary (bleu) | Droite |

## 7. VALIDATION
| Règle | Description |
|-------|-------------|
| Longueur max | 250 caractères |
| Caractères autorisés | Alphanumériques, espaces, ponctuation |
| Optionnel | Peut être vide |

## 8. USAGE DESCRIPTION
| Usage | Exemple |
|-------|---------|
| Identification serveur | "Serveur web production" |
| Projet | "Projet e-commerce v2" |
| Environnement | "Staging - Tests QA" |
| Contact | "Équipe DevOps - support@" |

## 9. APIS & ENDPOINTS
- GET /ip/{ip} - Récupérer description actuelle
- PUT /ip/{ip} - Modifier description (body: { description })
