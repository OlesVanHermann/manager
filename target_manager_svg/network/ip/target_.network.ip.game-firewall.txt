# Cahier des Charges : network/ip - GAME Firewall

## 1. VUE D'ENSEMBLE
- **Page** : Configuration GAME Firewall Anti-DDoS
- **URL** : /network/ip/{ip}/game-firewall
- **Objectif** : Configurer la protection DDoS pour serveurs de jeux

## 2. STRUCTURE PAGE
- Sidebar Network
- Lien retour
- Breadcrumb
- Titre + bouton guides
- Description détaillée
- Barre d'action avec toggle
- Datagrid des règles

## 3. COMPOSANTS UI
- **OdsLink** : "← Retour à \"IP\""
- **OdsBreadcrumb** : {bloc} › {IP} › Configurer le GAME firewall
- **OdsText** : Titre et descriptions
- **OdsButton** : "Guides" (outline), "Ajouter une règle" (outline)
- **OdsToggle** : Stratégie "Refus par défaut"
- **OdsDatagrid** : Liste des règles de jeu
- **OdsBadge** : Statut (Protégé)
- **OdsActionMenu** : Actions par règle

## 4. DESCRIPTION
| Ligne | Contenu |
|-------|---------|
| L1 | "Ceci est la page de configuration de la protection DDoS GAME pour l'adresse IP sélectionnée." |
| L2 | "Plusieurs règles de pare-feu peuvent être configurées pour protéger au mieux toutes les applications" |
| L3 | "hébergées sur l'adresse IP sélectionnée." |
| Astuce | "vous pouvez choisir « autre » protocole pour autoriser un trafic arbitraire vers un ou plusieurs ports spécifiques. Nous suggérons d'appliquer la politique « Refus par défaut »." |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/game/{ipOnGame} - Détails protection
- GET /ip/{ip}/game/{ipOnGame}/rule - Liste règles
- POST /ip/{ip}/game/{ipOnGame}/rule - Créer règle
- PUT /ip/{ip}/game/{ipOnGame}/rule/{id} - Modifier règle
- DELETE /ip/{ip}/game/{ipOnGame}/rule/{id} - Supprimer règle
