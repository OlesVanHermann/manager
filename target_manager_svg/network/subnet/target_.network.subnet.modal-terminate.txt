# Cahier des Charges : network/ip - Modal Résilier IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation destructive
- **Déclencheur** : Action "Résilier" dans le menu IP
- **Objectif** : Confirmer la résiliation d'une adresse IP

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée compacte (500x280px)
- Icône d'avertissement
- Titre et message de confirmation
- Boutons d'action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsIcon** : Warning dans cercle rouge
- **OdsText** : Titre, message, note
- **OdsButton** : Annuler/Résilier (danger)

## 4. ICÔNE WARNING
| Propriété | Valeur |
|-----------|--------|
| Conteneur | Cercle 60px |
| Fond | Rouge clair (#FEE2E2) |
| Icône | ! (exclamation) |
| Couleur icône | Rouge (#DC2626) |
| Position | Centré en haut |

## 5. CONTENU
| Élément | Valeur | Style |
|---------|--------|-------|
| Titre | "Résilier ce service ?" | Heading, centré |
| Message L1 | "Êtes-vous sûr de vouloir résilier l'adresse IP" | Body, centré |
| IP | "192.168.1.100 ?" | Bold, centré |
| Avertissement | "Cette action est irréversible." | Muted 13px, centré |

## 6. BOUTONS
| Bouton | Style | Position | Largeur |
|--------|-------|----------|---------|
| Annuler | Outline (bordure grise) | Gauche | 140px |
| Résilier | Danger (rouge #DC2626) | Droite | 140px |

## 7. FERMETURE MODAL
| Élément | Position |
|---------|----------|
| Bouton ✕ | Coin supérieur droit |

## 8. COMPORTEMENT
| Action | Résultat |
|--------|----------|
| Clic Annuler | Ferme la modal |
| Clic Résilier | Lance résiliation + toast confirmation |
| Clic ✕ | Ferme la modal |
| Clic overlay | Ferme la modal |

## 9. CONSÉQUENCES RÉSILIATION
| Impact | Description |
|--------|-------------|
| IP libérée | L'IP retourne dans le pool OVHcloud |
| Reverse DNS | Supprimé |
| Configuration firewall | Perdue |
| Facturation | Arrêtée à la fin du mois |

## 10. APIS & ENDPOINTS
- DELETE /ip/{ip}/terminate - Résilier l'IP
- POST /services/{serviceName}/terminate - Alternative via services
