# Cahier des Charges : network/ip - Modal Gérer Mitigation

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Déclencheur** : Action "Gérer la mitigation" dans le menu IP
- **Objectif** : Configurer le mode de mitigation Anti-DDoS

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x580px)
- Header avec titre et bouton fermer
- Info box IP + statut
- Options de mitigation (radio cards)
- Avertissement
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsMessage** : Info IP (info)
- **OdsBadge** : Statut (Normal)
- **OdsRadio** : Options (cartes cliquables)
- **OdsMessage** : Avertissement (warning)
- **OdsButton** : Annuler/Appliquer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Gérer la mitigation Anti-DDoS" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. INFO BOX IP
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Label | "Adresse IP" |
| Valeur | 123.45.67.89 (bold) |
| Badge | "Statut: Normal" (vert) |

## 6. MODES DE MITIGATION

### Option 1 - Automatique (recommandé)
| Propriété | Valeur |
|-----------|--------|
| Sélectionné | ● |
| Fond | Bleu clair (#E8F0FE) |
| Bordure | Bleu 2px |
| Titre | "Automatique (recommandé)" |
| Description | "La mitigation s'active automatiquement lors d'une attaque détectée." |

### Option 2 - Permanent
| Propriété | Valeur |
|-----------|--------|
| Sélectionné | ○ |
| Fond | Blanc |
| Bordure | Grise |
| Titre | "Permanent" |
| Description | "La mitigation est toujours active. Peut augmenter légèrement la latence." |

### Option 3 - Forcer maintenant
| Propriété | Valeur |
|-----------|--------|
| Sélectionné | ○ |
| Fond | Blanc |
| Bordure | Grise |
| Titre | "Forcer maintenant" |
| Description | "Force l'activation immédiate de la mitigation pendant 1 heure." |

## 7. MESSAGE AVERTISSEMENT
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (jaune) |
| Fond | #FEF3C7 |
| Bordure | #F59E0B |
| Icône | ⚠️ |
| Texte | "Le mode permanent peut impacter les performances de certaines applications sensibles à la latence." |

## 8. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Appliquer | Primary (bleu) | Droite |

## 9. STATUTS MITIGATION
| Statut | Badge | Description |
|--------|-------|-------------|
| Normal | Vert | Pas d'attaque détectée |
| Under attack | Rouge | Attaque en cours |
| Protected | Bleu | Mitigation active |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/mitigation/{ipOnMitigation} - Statut mitigation
- PUT /ip/{ip}/mitigation/{ipOnMitigation} - Modifier mode
- POST /ip/{ip}/mitigation/{ipOnMitigation}/stats - Statistiques
