# Cahier des Charges : network/ip - Liste des IP

## 1. VUE D'ENSEMBLE
- **Page** : Liste des adresses IP
- **URL** : /network/ip
- **Objectif** : Afficher et gérer toutes les IP

## 2. STRUCTURE PAGE
- Sidebar Network avec IP sélectionné
- Breadcrumb
- Titre + boutons action
- Filtres de recherche
- Datagrid des IP

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Network / IP
- **OdsText** : Titre "IP"
- **OdsButton** : "Commander des IP" (primary), "Exporter en CSV" (outline)
- **OdsSearchBar** : Recherche IP
- **OdsSelect** : Filtres service et type
- **OdsDatagrid** : Liste des IP
- **OdsBadge** : Type et statut
- **OdsActionMenu** : Actions par IP

## 4. BOUTONS ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Commander des IP | Primary | Ouvre wizard commande |
| Exporter en CSV | Outline | Export CSV |

## 5. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| Recherche | SearchBar | "Rechercher une IP..." |
| Service | Select | "Tous les services" + liste |
| Type | Select | "Tous les types" + Failover/vRack/BYOIP |

## 6. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| Bloc IP | Link | ✓ | Adresse IP ou bloc CIDR |
| Type | Badge | ✓ | Failover/vRack/BYOIP |
| Service associé | Text | ✓ | Nom du service |
| Localisation | Text + flag | ✓ | Pays avec emoji drapeau |
| Firewall | Badge | ✓ | Actif/Désactivé |
| Statut | Badge | ✓ | OK/Warning/Error |
| Actions | Menu ⋮ | - | Actions contextuelles |

## 7. TYPES D'IP (badges)
| Type | Couleur | Description |
|------|---------|-------------|
| Failover | Bleu info | IP basculable |
| vRack | Violet | IP réseau privé |
| BYOIP | Vert | Bring Your Own IP |
| RIPE | Gris | Bloc RIPE |

## 8. STATUTS FIREWALL
| Statut | Badge |
|--------|-------|
| Actif | Vert (success) |
| Désactivé | Jaune (warning) |
| — | Tiret (non applicable) |

## 10. APIS & ENDPOINTS
- GET /ip - Liste des IP
- GET /ip/{ip} - Détails IP
- GET /ip/{ip}/reverse - Reverse DNS
- GET /dedicated/server/{serviceName}/ips - IP d'un serveur
