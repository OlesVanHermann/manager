# Cahier des Charges : network/ip - Commander des IP

## 1. VUE D'ENSEMBLE
- **Page** : Wizard de commande IP
- **URL** : /network/ip/order
- **Objectif** : Commander des Additional IP (failover)

## 2. STRUCTURE PAGE
- Sidebar Network
- Lien retour
- Breadcrumb
- Titre
- Formulaire en étapes
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsLink** : "← Retour"
- **OdsBreadcrumb** : IP › Commander des Additional IP
- **OdsText** : Titre et descriptions
- **OdsCard** : Cartes de sélection (radio visuel)
- **OdsSelect** : Sélection service
- **OdsButton** : Retour / Continuer ma commande

| Élément | Contenu |

## 8. BOUTONS NAVIGATION
| Bouton | Style | État |
|--------|-------|------|
| Retour | Outline gris | Toujours actif |
| Continuer ma commande | Primary (disabled gris) | Actif si formulaire valide |

## 9. WORKFLOW
1. Sélection version IP (IPv4/IPv6)
2. Sélection service destination
3. Sélection région (si applicable)
4. Choix offre (nombre IP, bloc)
5. Récapitulatif et paiement

## 10. APIS & ENDPOINTS
- GET /order/cart - Créer panier
- POST /order/cart/{cartId}/ip - Ajouter IP au panier
- GET /order/catalog/formatted/ip - Catalogue IP
- POST /order/cart/{cartId}/checkout - Finaliser
