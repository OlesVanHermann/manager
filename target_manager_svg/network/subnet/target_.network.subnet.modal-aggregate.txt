# Cahier des Charges : network/ip - Modal Agréger Bloc IP

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'action
- **Déclencheur** : Action "Agréger le bloc" dans le menu IP
- **Objectif** : Fusionner plusieurs sous-réseaux en un bloc parent

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x450px)
- Header avec titre dynamique et bouton fermer
- Select dropdown pour bloc parent
- Zone d'information sur les sous-réseaux impactés
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsSelect** : Choix du bloc parent
- **OdsList** : Liste des sous-réseaux à agréger
- **OdsText** : Labels et descriptions
- **OdsButton** : Annuler/Confirmer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Agréger le bloc {IP}/{CIDR}" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. FORMULAIRE

### Instructions
| Propriété | Valeur |
|-----------|--------|
| Texte | "Veuillez spécifier le bloc parent désiré" |
| Position | Haut du formulaire |

### Select Bloc Parent
| Propriété | Valeur |
|-----------|--------|

| Largeur | 540px |
| Exemple | 192.168.0.0/22 |

## 7. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Confirmer | Primary (bleu) | Droite |

## 8. RÈGLES AGRÉGATION
| Règle | Description |
|-------|-------------|
| CIDR contigu | Les blocs doivent être contigus |
| Même datacenter | Dans le même datacenter |
| Même propriétaire | Tous du même compte |

## 9. APIS & ENDPOINTS
- GET /ip/{ip}/aggregatedChildren - Liste sous-réseaux agrégables
- POST /ip/{ip}/aggregate - Agréger les blocs
