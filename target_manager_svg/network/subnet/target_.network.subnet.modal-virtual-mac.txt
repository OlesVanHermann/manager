# Cahier des Charges : network/ip - Modal Configurer MAC Virtuelle

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Déclencheur** : Action "Ajouter une MAC virtuelle" dans le menu IP
- **Objectif** : Créer une adresse MAC virtuelle pour une IP failover

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x520px)
- Header avec titre et bouton fermer
- Message d'information
- Formulaire de configuration
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsMessage** : Information (info bleu)
- **OdsInput** : IP (readonly), Nom VM (optionnel)
- **OdsRadio** : Type de MAC (ovh/vmware)
- **OdsButton** : Annuler/Créer

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Configurer la MAC virtuelle" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Créer | Primary (bleu) | Droite |

## 8. TYPES DE MAC DISPONIBLES
| Type | Préfixe | Usage |
|------|---------|-------|
| ovh | 02:00:00 | Usage général, VM Linux/Windows |
| vmware | 00:50:56 | Environnements VMware ESXi |

## 9. CAS D'USAGE
| Contexte | Description |
|----------|-------------|
| VM Windows | Licence liée à la MAC |
| Haute disponibilité | MAC persistante entre failovers |
| VMware | Compatibilité vSphere |
| Réseaux legacy | MAC fixe requise |

## 10. APIS & ENDPOINTS
- GET /ip/{ip}/virtualMac/availableTypes - Types disponibles
- POST /ip/{ip}/virtualMac - Créer MAC virtuelle
- DELETE /ip/{ip}/virtualMac/{macAddress} - Supprimer MAC
