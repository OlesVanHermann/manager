# Cahier des Charges : network/load-balancer - Modal modifier chiffrement SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Dashboard Load Balancer
- **Objectif** : Modifier la politique de chiffrement SSL

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsRadio** : Options de chiffrement
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier le chiffrement SSL" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. OPTIONS DE CHIFFREMENT

### Options radio
| Option | Description | État |
|--------|-------------|------|
| Modern | TLS 1.3 uniquement - Sécurité maximale | Sélectionné |
| Intermediate | TLS 1.2+ - Compatibilité avec les navigateurs récents | - |
| Old | TLS 1.0+ - Compatibilité maximale (déconseillé) | - |
| Personnalisé | Configuration manuelle des suites de chiffrement | - |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Applique le chiffrement |

## 7. APIS & ENDPOINTS
- PUT /ipLoadbalancing/{serviceName}/ssl/cipher - Modifier politique chiffrement
