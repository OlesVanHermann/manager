# Cahier des Charges : network/load-balancer - Modal Ajouter Frontend

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Déclencheur** : Bouton "Ajouter un frontend"
- **Objectif** : Créer un nouveau frontend

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (700x700px)
- Header avec titre et bouton fermer
- Formulaire avec champs
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsText** : Titre
- **OdsInput** : Champs texte
- **OdsSelect** : Dropdowns
- **OdsRadio** : Sélection protocole
- **OdsButton** : Annuler/Ajouter
- **OdsLink** : Options avancées

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Ajouter un frontend" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. CHAMPS DU FORMULAIRE

### Nom du frontend *
| Propriété | Valeur |
|-----------|--------|
| Type | Input text |
| Placeholder | frontend-https-prod |
| Largeur | 640px |
| Obligatoire | Oui |

### Protocole *
| Option | Sélectionné |
|--------|-------------|

| HTTPS | ● (actif) |
| TCP | ○ |
| UDP | ○ |

Type : Radio buttons

| Propriété | Valeur |

## 6. OPTIONS AVANCÉES
- **Style** : Lien bleu avec "+"
- **Texte** : "+ Options avancées"
- **Action** : Affiche champs additionnels

### Champs avancés (masqués par défaut)
| Champ | Type | Description |
|-------|------|-------------|
| IP Failover | Select | IP dédiée au frontend |
| Restriction IP | Input | CIDR pour restreindre accès |
| HSTS | Checkbox | Activer Strict Transport Security |
| Redirect HTTP | Checkbox | Redirection automatique vers HTTPS |

## 7. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Ajouter | Primary (bleu) | Droite |

## 8. PROTOCOLES DISPONIBLES
| Protocole | Port par défaut | SSL |
|-----------|-----------------|-----|
| HTTP | 80 | Non |
| HTTPS | 443 | Oui |
| TCP | Variable | Non |
| TCP SSL | Variable | Oui |
| UDP | Variable | Non |

## 9. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/http/frontend - Créer frontend HTTP
- POST /ipLoadbalancing/{serviceName}/tcp/frontend - Créer frontend TCP
- POST /ipLoadbalancing/{serviceName}/udp/frontend - Créer frontend UDP
