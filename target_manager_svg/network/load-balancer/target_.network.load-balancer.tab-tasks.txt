# Cahier des Charges : network/load-balancer - Onglet Tâches

## 1. VUE D'ENSEMBLE
- **Page** : Historique des tâches
- **URL** : /network/load-balancer/{serviceName}/tasks
- **Objectif** : Suivre les opérations en cours et passées

## 2. STRUCTURE PAGE
- Sidebar Network
- Breadcrumb
- Titre service
- Onglets navigation
- Tableau des tâches

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Accueil/Fermes/Frontends/SSL/Tâches (actif)/Graphiques
- **OdsDatagrid** : Liste des tâches
- **OdsBadge** : Statut (En cours/En attente/Terminé/Erreur)
- **OdsLink** : "Détails" pour chaque tâche

|--------|-------------|
| Ajout d'un serveur | Nouveau serveur dans une ferme |
| Modification frontend | Changement config frontend |
| Renouvellement certificat SSL | Mise à jour certificat |
| Création ferme de serveurs | Nouvelle ferme |
| Suppression serveur | Retrait d'un serveur |
| Modification ferme | Changement config ferme |
| Suppression frontend | Retrait d'un frontend |
