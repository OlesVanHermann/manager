# Cahier des Charges : network/load-balancer - Page supprimer zone

## 1. VUE D'ENSEMBLE
- **Type** : Page de sélection destructive
- **Contexte** : Dashboard Load Balancer
- **Objectif** : Supprimer des zones de disponibilité du Load Balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation retour (rouge)
- **OdsText** : Description avec contrainte
- **OdsCard** : Cartes de zone (checkbox)
- **OdsButton** : Supprimer, Annuler

## 4. HEADER PAGE
| Élément | Valeur |
|---------|--------|
| Retour | "← Supprimer une zone" (rouge) |

## 5. DESCRIPTION
| Texte |
|-------|
| "Vous pouvez supprimer X zone(s). Au moins une zone doit rester active." |

## 6. SÉLECTION DES ZONES

### Format des cartes
| Élément | Description |
|---------|-------------|
| Checkbox | Sélection pour suppression |
| Nom | Code et nom de la zone |
| Pays | Localisation |
| Statut | Supprimable / Sélectionné / Zone principale |

### États visuels
| État | Bordure | Checkbox | Label |
|------|---------|----------|-------|
| Supprimable | Grise | Vide | "Supprimable" |
| Sélectionné | Rouge | Coché rouge | "Sélectionné" (rouge) |
| Zone principale | Grise, fond gris | Grisé | "Zone principale" (gris) |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Supprimer | Danger (rouge) | Supprime les zones sélectionnées |
| Annuler | Secondary | Retour au dashboard |

## 8. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/zone - Liste des zones actives
- DELETE /ipLoadbalancing/{serviceName}/zone/{zoneName} - Supprimer zone
