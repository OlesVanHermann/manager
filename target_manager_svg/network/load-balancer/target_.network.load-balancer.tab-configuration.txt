# Cahier des Charges : network/load-balancer - Onglet Configuration

## 1. VUE D'ENSEMBLE
- **Page** : Application des modifications
- **URL** : /network/load-balancer/{serviceName}/configuration
- **Objectif** : Appliquer les changements de config par datacenter

## 2. STRUCTURE PAGE
- Sidebar Network
- Onglets navigation
- Titre section
- Description
- Bouton appliquer (disabled si 0 modif)
- Tableau des datacenters

## 3. COMPOSANTS UI
- **OdsTabs** : Vue d'ensemble/Frontends/Fermes/SSL/Configuration (actif)/Logs
- **OdsText** : Titre "Configuration" et description
- **OdsButton** : "Appliquer les modifications" (disabled/primary)
- **OdsDatagrid** : Liste datacenters
- **OdsCheckbox** : Sélection datacenters
- **OdsProgressBar** : Progression application
- **OdsBadge** : Nombre de modifications

## 4. EN-TÊTE SECTION
| Élément | Contenu |
|---------|---------|
| Titre | "Configuration" |
| Description | "Appliquez les modifications de configuration sur vos datacenters." |

## 5. BOUTON APPLIQUER
| État | Style | Texte |
|------|-------|-------|
| Disabled | Gris (#F3F4F6), texte gris | "Appliquer les modifications" |
| Active | Primary bleu | "Appliquer les modifications" |

Le bouton est actif uniquement si au moins un datacenter avec modifications est sélectionné.

## 6. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Checkbox | Checkbox | Sélection |
| Datacenter | Text | Nom + code |
| Progression | ProgressBar | État application |
| Statut | Badge | Nombre modifications |
| Actions | Menu ⋮ | Actions |

## 7. DONNÉES EXEMPLE
| ☐ | Datacenter | Progression | Statut |
|---|------------|-------------|--------|
| ☐ | Gravelines (GRA) | - | 0 modification (vert) |
| ☑ | Francfort (FRA) | 70% | 3 modifications (jaune) |
| ☐ | Roubaix (RBX) | Modifications appliquées | 0 modification (vert) |

## 8. STATUTS MODIFICATIONS
| Statut | Badge | Signification |
|--------|-------|---------------|
| 0 modification | Vert (success) | Tout est à jour |
| X modifications | Jaune (warning) | Changements en attente |

## 9. BARRE DE PROGRESSION
| Propriété | Valeur |
|-----------|--------|
| Fond | #E5E7EB |
| Remplissage | #0050D7 |
| Hauteur | 8px |
| Largeur | 200px |
| Arrondi | 4px |
| Label | Pourcentage à droite |

## 10. CHECKBOX
| État | Style |
|------|-------|
| Non sélectionné | Rectangle blanc, bordure grise |
| Sélectionné | Rectangle bleu, checkmark blanc |

## 11. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/zone - Liste des zones
- GET /ipLoadbalancing/{serviceName}/pending - Modifications en attente
- POST /ipLoadbalancing/{serviceName}/refresh - Appliquer modifications
- GET /ipLoadbalancing/{serviceName}/task - Suivi application
