# Cahier des Charges : network/load-balancer - Onglet Graphiques

## 1. VUE D'ENSEMBLE
- **Page** : Graphiques de performance
- **URL** : /network/load-balancer/{serviceName}/graphs
- **Objectif** : Visualiser les métriques du Load Balancer

## 2. STRUCTURE PAGE
- Sidebar Network
- Breadcrumb
- Titre service
- Onglets navigation
- Sélecteur de période
- 2 graphiques empilés verticalement

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation
- **OdsTabs** : Accueil/Fermes/Frontends/SSL/Tâches/Graphiques (actif)
- **OdsSelect** : Sélecteur de période
- **OdsCard** : Conteneur pour chaque graphique
- **OdsChart** : Graphiques en ligne (Line chart)

|----------|-------------|

## 11. APIS & ENDPOINTS

- GET /ipLoadbalancing/{serviceName}/status - Métriques temps réel
