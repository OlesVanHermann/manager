# Cahier des Charges : network/load-balancer - Page ajouter réseau privé vRack

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire
- **Contexte** : Page vRack Load Balancer
- **Objectif** : Configurer un réseau privé vRack pour le Load Balancer

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 700px |
| Hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation retour
- **OdsInput** : Nom, VLAN, Sous-réseau, IP NAT
- **OdsSelect** : Ferme associée
- **OdsButton** : Ajouter ferme, Ajouter, Annuler

## 4. HEADER PAGE
| Élément | Valeur |
|---------|--------|
| Retour | "← Ajouter un réseau privé" |

## 5. FORMULAIRE

### Champs
| Champ | Type | Description | Aide |
|-------|------|-------------|------|
| Nom d'affichage | Input | Nom du réseau privé | - |
| VLAN | Input number | Identifiant VLAN | "Identifiant VLAN (0-4095)" |
| Sous-réseau | Input | Plage IP privée | "Plage d'adresses IP privées" |
| IP NAT | Input | Adresse pour le NAT | "Adresse IP pour le NAT" |
| Ferme associée | Select + Button | Sélection de ferme | Bouton "+" pour ajouter |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter | Primary | Crée le réseau privé |
| Annuler | Secondary | Retour à la liste |

## 7. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/vrack/network - Créer réseau privé
- GET /ipLoadbalancing/{serviceName}/http/farm - Liste des fermes HTTP
- GET /ipLoadbalancing/{serviceName}/tcp/farm - Liste des fermes TCP
