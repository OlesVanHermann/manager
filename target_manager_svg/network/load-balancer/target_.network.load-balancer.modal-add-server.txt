# Cahier des Charges : network/load-balancer - Modal Ajouter Serveur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Déclencheur** : Bouton "Ajouter un serveur" dans une ferme
- **Objectif** : Ajouter un serveur backend à une ferme

## 2. STRUCTURE MODAL
- Overlay sombre (50% opacité)
- Modal centrée (600x620px)
- Header avec titre et bouton fermer
- Info box ferme destination
- Formulaire avec champs
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur modal
- **OdsMessage** : Info ferme destination
- **OdsInput** : Champs texte/number
- **OdsCheckbox** : Options
- **OdsButton** : Annuler/Ajouter

## 4. HEADER MODAL
| Élément | Valeur |
|---------|--------|
| Titre | "Ajouter un serveur" |
| Bouton fermer | ✕ (coin droit) |
| Séparateur | Ligne grise |

## 5. INFO BOX FERME
| Propriété | Valeur |
|-----------|--------|
| Type | Message info (bleu) |
| Fond | #E8F0FE |
| Bordure | #0050D7 |
| Label | "Ferme de destination" |
| Valeur | "farm-web-prod (HTTP:80 - GRA)" |

## 6. CHAMPS DU FORMULAIRE

### Nom du serveur (optionnel)
| Propriété | Valeur |
|-----------|--------|
| Type | Input text |
| Placeholder | web-server-03 |
| Largeur | 540px |
| Obligatoire | Non |

### Adresse IP *
| Propriété | Valeur |
|-----------|--------|
| Type | Input text |
| Exemple | 10.0.0.13 |
| Largeur | 350px |
| Obligatoire | Oui |

### Port
| Propriété | Valeur |
|-----------|--------|
| Type | Input number |
| Placeholder | 80 (défaut) |
| Largeur | 170px |
| Obligatoire | Non (hérite de la ferme) |

### Poids (weight)
| Propriété | Valeur |
|-----------|--------|
| Type | Input number |
| Défaut | 1 |
| Largeur | 170px |
| Plage | 1 à 256 |

## 7. OPTIONS (checkboxes)

| Option | État défaut | Description |
|--------|-------------|-------------|
| Activer le serveur immédiatement | ☑ Coché | Serveur actif dès création |
| Serveur de backup | ☐ Non coché | Utilisé si tous les autres sont HS |
| Utiliser une connexion SSL | ☐ Non coché | Connexion SSL vers le serveur |

## 8. FOOTER MODAL
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline (bordure grise) | Gauche |
| Ajouter | Primary (bleu) | Droite |

## 9. VALIDATION
| Champ | Règle |
|-------|-------|
| Adresse IP | Requis, format IPv4 ou IPv6 valide |
| Port | Optionnel, 1-65535 |
| Poids | 1-256 |

## 10. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/http/farm/{farmId}/server - Ajouter serveur HTTP
- POST /ipLoadbalancing/{serviceName}/tcp/farm/{farmId}/server - Ajouter serveur TCP
- POST /ipLoadbalancing/{serviceName}/udp/farm/{farmId}/server - Ajouter serveur UDP
