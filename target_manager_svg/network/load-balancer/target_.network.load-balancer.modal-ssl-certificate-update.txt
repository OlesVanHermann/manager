# Cahier des Charges : network/load-balancer - Modal mettre à jour certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Modal de mise à jour
- **Contexte** : Page SSL Load Balancer
- **Objectif** : Mettre à jour un certificat SSL existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 340px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Nom du certificat
- **OdsTextarea** : Nouveau certificat
- **OdsMessage** : Info clé privée
- **OdsButton** : Annuler, Mettre à jour

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Mettre à jour le certificat" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Informations affichées
| Élément | Description |
|---------|-------------|
| Nom du certificat | Nom actuel (lecture seule) |

### Champ
| Champ | Type | Placeholder | Requis |
|-------|------|-------------|--------|
| Nouveau certificat * | Textarea (100px) | -----BEGIN CERTIFICATE----- | Oui |

### Message info
| Type | Contenu |
|------|---------|
| Info (bleu) | "La clé privée reste inchangée lors de la mise à jour." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Mettre à jour | Primary | Met à jour le certificat |

## 7. APIS & ENDPOINTS
- PUT /ipLoadbalancing/{serviceName}/ssl/{sslId} - Mettre à jour certificat
