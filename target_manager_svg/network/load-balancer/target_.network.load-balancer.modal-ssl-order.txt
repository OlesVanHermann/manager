# Cahier des Charges : network/load-balancer - Page Commander Certificat SSL

## 1. VUE D'ENSEMBLE
- **Type** : Page de formulaire (pas une modal)
- **URL** : /network/load-balancer/{serviceName}/ssl/order
- **Objectif** : Commander un nouveau certificat SSL

## 2. STRUCTURE PAGE
- Sidebar Network
- Lien retour
- Formulaire de commande
- Message d'information
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : "← Commander un certificat SSL"
- **OdsButtonGroup** : Type de certificat
- **OdsInput** : FQDN
- **OdsMessage** : Information Let's Encrypt
- **OdsButton** : Commander / Annuler

## 4. LIEN RETOUR
- **Texte** : "← Commander un certificat SSL"
- **Style** : Lien bleu primaire
- **Action** : Retour à la liste des certificats

## 5. CHAMPS DU FORMULAIRE

### Type de certificat
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Gratuit | ● (actif) | Let's Encrypt |
| Comodo DV | ○ | Domain Validation |
| Comodo EV | ○ | Extended Validation |

Style : ButtonGroup avec sélection exclusive

### FQDN (Fully Qualified Domain Name)
| Propriété | Valeur |
|-----------|--------|
| Type | Input text |
| Exemple | www.example.com |
| Largeur | 500px |
| Aide | "example.com, www.example.com" |
| Obligatoire | Oui |

## 6. MESSAGE INFORMATION (Let's Encrypt)
| Propriété | Valeur |
|-----------|--------|
| Type | Message info (bleu) |
| Fond | #DBEAFE |
| Bordure | #93C5FD |
| Icône | ℹ️ |
| Texte L1 | "Le certificat gratuit est fourni par Let's Encrypt." |
| Texte L2 | "Il sera automatiquement renouvelé avant expiration." |

Visible uniquement si type "Gratuit" sélectionné.

## 7. TYPES DE CERTIFICATS

### Gratuit (Let's Encrypt)
| Caractéristique | Valeur |
|-----------------|--------|
| Fournisseur | Let's Encrypt |
| Validation | Domain Validation (DV) |
| Durée | 90 jours |
| Renouvellement | Automatique |
| Prix | Gratuit |
| Wildcard | Non |

### Comodo DV
| Caractéristique | Valeur |
|-----------------|--------|
| Fournisseur | Sectigo (ex-Comodo) |
| Validation | Domain Validation |
| Durée | 1 an |
| Renouvellement | Manuel |
| Prix | Payant |
| Wildcard | Oui |

### Comodo EV
| Caractéristique | Valeur |
|-----------------|--------|
| Fournisseur | Sectigo (ex-Comodo) |
| Validation | Extended Validation |
| Durée | 1 an |
| Renouvellement | Manuel |
| Prix | Payant |
| Wildcard | Non |
| Barre verte | Oui |

## 8. FOOTER (boutons)
| Bouton | Style | Position |
|--------|-------|----------|
| Commander | Primary (bleu) | Gauche |
| Annuler | Outline (gris) | À droite du précédent |

## 9. APIS & ENDPOINTS
- POST /ipLoadbalancing/{serviceName}/ssl - Commander certificat
- GET /order/cart - Créer panier commande
- POST /order/cart/{cartId}/assign - Assigner au compte
