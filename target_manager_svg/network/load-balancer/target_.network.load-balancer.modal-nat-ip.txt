# Cahier des Charges : network/load-balancer - Modal IP NAT

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'information
- **Contexte** : Dashboard Load Balancer
- **Objectif** : Afficher les IP NAT par zone de disponibilité

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description et labels zones
- **OdsList** : Liste des IP par zone
- **OdsClipboard** : Copier dans le presse-papier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "IP NAT" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Texte |
|-------|
| "Adresses IP NAT par zone de disponibilité" |

### Liste par zone
| Zone | Label | Exemple IP |
|------|-------|------------|
| GRA | Gravelines | 10.0.0.1 |
| SBG | Strasbourg | 10.0.1.1 |
| RBX | Roubaix | 10.0.2.1 |

### Actions par ligne
| Action | Description |
|--------|-------------|
| Copy | Copie l'IP dans le presse-papier |

## 6. APIS & ENDPOINTS
- GET /ipLoadbalancing/{serviceName}/natIp - Liste des IP NAT par zone
