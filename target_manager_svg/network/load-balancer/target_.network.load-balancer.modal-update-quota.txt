# Cahier des Charges : network/load-balancer - Modal modifier alerte quota

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Dashboard Load Balancer
- **Objectif** : Configurer le seuil d'alerte d'utilisation des données

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 260px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Données incluses (readonly), Seuil d'alerte
- **OdsButton** : Annuler, Modifier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier l'alerte d'utilisation" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champs
| Champ | Type | Description | Modifiable |
|-------|------|-------------|------------|
| Données incluses | Input (readonly) | Quota inclus dans l'offre (ex: 500 GB) | Non |
| Seuil d'alerte (GB) | Input number | Seuil déclenchant une alerte | Oui |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Modifier | Primary | Enregistre le seuil |

## 7. APIS & ENDPOINTS
- PUT /ipLoadbalancing/{serviceName}/quota - Modifier seuil d'alerte
