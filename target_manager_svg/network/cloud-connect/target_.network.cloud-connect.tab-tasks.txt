# Cahier des Charges : network/cloud-connect - Onglet Tâches

## 1. VUE D'ENSEMBLE
- **Page** : Historique des tâches
- **URL** : /network/cloud-connect/{serviceName}?tab=tasks
- **Objectif** : Suivre les opérations effectuées sur le service

## 2. STRUCTURE PAGE
- Onglets avec "Tâches" actif
- Datagrid des tâches

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Liste des tâches
- **OdsBadge** : Statut (en cours/en attente/terminé)
- **OdsLink** : Détails de chaque tâche

## 4. COLONNES DATAGRID
| Colonne | Type | Tri | Description |
|---------|------|-----|-------------|
| ID | Text | ✓ | Identifiant tâche |
| Type | Text | ✓ | Type d'opération |
| Date de création | DateTime | ✓ | Début de la tâche |
| Date de fin | DateTime | ✓ | Fin (ou —) |
| Statut | Badge | ✓ | en cours/en attente/terminé |
| Actions | Link | - | Lien "Détails" |

## 5. BADGES STATUT
| Statut | Badge | Couleur |
|--------|-------|---------|
| en cours | warning | Jaune |
| en attente | info | Bleu |
| terminé | success | Vert |
| erreur | error | Rouge |

## 6. TYPES DE TÂCHES
| Type | Description |
|------|-------------|
| Création du service | Provisioning initial |
| Génération clé de service | Création d'une clé |
| Configuration PoP | Configuration Point of Presence |
| Configuration datacenter | Ajout d'un datacenter |
| Association vRack | Liaison avec un vRack |
| Modification | Changement de configuration |
| Résiliation | Suppression du service |
