# Cahier des Charges : network/cloud-connect - Modal dissocier vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Dissocier un vRack de la connexion OVHcloud Connect

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Dissocier

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Dissocier le vRack" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir dissocier le vRack de cette connexion OVHcloud Connect ?" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Dissocier | Warning (orange) | Dissocie le vRack |

## 7. APIS & ENDPOINTS
- DELETE /ovhCloudConnect/{serviceName}/vrack/{vrackId} - Dissocier vRack
