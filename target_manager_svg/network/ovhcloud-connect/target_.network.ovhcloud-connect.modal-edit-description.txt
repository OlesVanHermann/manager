# Cahier des Charges : network/cloud-connect - Modal modifier description

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition simple
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Modifier la description du service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champ description
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la description" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Champ
| Champ | Type | Exemple |
|-------|------|---------|
| Description | Input | "Ma connexion OVHcloud Connect principale" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary | Enregistre la description |

## 7. APIS & ENDPOINTS
- PUT /ovhCloudConnect/{serviceName} - Modifier description
