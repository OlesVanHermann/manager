# Cahier des Charges : network/cloud-connect - Modal gérer notifications

## 1. VUE D'ENSEMBLE
- **Type** : Modal de configuration
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Gérer les préférences de notifications

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : UUID info
- **OdsCheckbox** : Options de notification
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Gérer les notifications" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Information
| Élément | Description |
|---------|-------------|
| UUID | Identifiant du service (affiché en gris) |

### Options de notification
| Option | Description | État défaut |
|--------|-------------|-------------|
| Alertes de connectivité | Recevoir des alertes en cas de problème de connexion | Coché |
| Alertes de maintenance | Recevoir des notifications pour les maintenances planifiées | Non coché |
| Alertes de performance | Recevoir des alertes en cas de dégradation des performances | Coché |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Primary | Enregistre les préférences |

## 7. APIS & ENDPOINTS
- GET /ovhCloudConnect/{serviceName}/notifications - Récupérer préférences
- PUT /ovhCloudConnect/{serviceName}/notifications - Modifier préférences
