# Cahier des Charges : network/cloud-connect - Modal débloquer port

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Page interfaces Cloud Connect
- **Objectif** : Débloquer un port pour rétablir le trafic

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Débloquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Débloquer le port" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir débloquer le port {portName} ?" |
| Info | "Le trafic sera rétabli sur ce port." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Débloquer | Warning (orange) | Débloque le port |

## 7. APIS & ENDPOINTS
- PUT /ovhCloudConnect/{serviceName}/interface/{id}/unlock - Débloquer port
