# Cahier des Charges : network/cloud-connect - Modal ajouter configuration de routage

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page datacenter Cloud Connect
- **Objectif** : Ajouter une configuration de routage BGP supplémentaire

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 390px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type de routage
- **OdsInput** : ASN, IP du voisin BGP
- **OdsLink** : Documentation
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Ajouter une configuration de routage" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. FORMULAIRE

### Champs
| Champ | Type | Description | Requis |
|-------|------|-------------|--------|
| Type de routage | Select | BGP, Static | Oui |
| ASN du voisin BGP | Input | Numéro de système autonome (ex: 65000) | Oui |
| Adresse IP du voisin BGP | Input | Adresse IPv4 | Oui |

### Aide contextuelle
| Champ | Texte aide |
|-------|------------|
| Type de routage | Lien "Consultez la documentation pour choisir le type adapté" |
| ASN | "Numéro de système autonome du voisin BGP" |
| IP | "Adresse IPv4 du voisin BGP" |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée la configuration |

## 7. APIS & ENDPOINTS
- POST /ovhCloudConnect/{serviceName}/datacenter/{id}/extra - Ajouter configuration routage
