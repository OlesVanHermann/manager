# Cahier des Charges : network/cloud-connect - Modal associer vRack

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'association
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Associer un vRack existant à la connexion OVHcloud Connect

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 240px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Description
- **OdsLink** : Commander nouveau vRack
- **OdsSelect** : Liste des vRacks disponibles
- **OdsButton** : Annuler, Associer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Associer un vRack" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Description
| Texte |
|-------|
| "Sélectionnez un vRack à associer à cette connexion OVHcloud Connect." |

### Lien
| Label | Action |
|-------|--------|
| "Commander un nouveau vRack" | Redirige vers commande vRack |

### Champ
| Champ | Type | Exemple |
|-------|------|---------|
| vRack | Select | pn-123456 |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Associer | Primary | Associe le vRack |

## 7. APIS & ENDPOINTS
- GET /vrack - Liste des vRacks disponibles
- POST /ovhCloudConnect/{serviceName}/vrack - Associer vRack
