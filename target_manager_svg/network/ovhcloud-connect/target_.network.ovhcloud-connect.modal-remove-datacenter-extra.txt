# Cahier des Charges : network/cloud-connect - Modal supprimer configuration routage

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Page datacenter Cloud Connect
- **Objectif** : Supprimer une configuration de routage

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le routage" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir supprimer cette configuration de routage ?" |
| Warning | "Le trafic sera interrompu." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Warning (orange) | Supprime la configuration |

## 7. APIS & ENDPOINTS
- DELETE /ovhCloudConnect/{serviceName}/datacenter/{id}/extra/{extraId} - Supprimer configuration
