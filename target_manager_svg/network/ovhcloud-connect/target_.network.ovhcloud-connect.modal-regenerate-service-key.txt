# Cahier des Charges : network/cloud-connect - Modal régénérer clé de service

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Page clés de service Cloud Connect
- **Objectif** : Régénérer une nouvelle clé de service

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Fermer, Régénérer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Régénérer la clé de service" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir régénérer la clé de service ?" |
| Info | "L'ancienne clé ne sera plus valide." (gris) |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Fermer | Secondary | Ferme modal |
| Régénérer | Primary | Régénère la clé |

## 7. APIS & ENDPOINTS
- POST /ovhCloudConnect/{serviceName}/serviceKey/regenerate - Régénérer clé
