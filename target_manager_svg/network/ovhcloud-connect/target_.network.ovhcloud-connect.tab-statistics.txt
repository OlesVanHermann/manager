# Cahier des Charges : network/cloud-connect - Onglet Statistiques

## 1. VUE D'ENSEMBLE
- **Page** : Statistiques de performance
- **URL** : /network/cloud-connect/{serviceName}?tab=statistics
- **Objectif** : Visualiser les métriques de bande passante et latence

## 2. STRUCTURE PAGE
- Onglets avec "Statistiques" actif
- Sélecteur de période
- Cartes métriques (4)
- Graphique bande passante

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsSelect** : Sélecteur période
- **OdsCard** : Cartes métriques (4)
- **OdsChart** : Graphique ligne/aire

## 4. SÉLECTEUR PÉRIODE
| Propriété | Valeur |
|-----------|--------|
| Type | Select dropdown |
| Options | Dernières 24h, 7 jours, 30 jours, 1 an |
| Défaut | Dernières 24 heures |

## 5. CARTES MÉTRIQUES
| Métrique | Valeur | Tendance | Couleur |
|----------|--------|----------|---------|
| Bande passante entrante | 456 Mbps | ↑ 12% | Vert |
| Bande passante sortante | 312 Mbps | ↓ 5% | Rouge |
| Latence moyenne | 2.3 ms | Stable | Vert |

## 6. GRAPHIQUE BANDE PASSANTE
| Propriété | Valeur |
|-----------|--------|
| Type | Line chart avec area |

| Axe X | Temps (00:00 à 24:00) |

## 8. APIS & ENDPOINTS
- GET /ovhCloudConnect/{serviceName}/statistics - Statistiques globales
- GET /ovhCloudConnect/{serviceName}/statistics/bandwidth - Bande passante
- GET /ovhCloudConnect/{serviceName}/statistics/latency - Latence
