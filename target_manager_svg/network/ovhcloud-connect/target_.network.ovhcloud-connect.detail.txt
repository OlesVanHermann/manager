# Cahier des Charges : network/cloud-connect - Détail Service

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard d'un service OVHcloud Connect
- **URL** : /network/cloud-connect/{serviceName}
- **Objectif** : Vue détaillée d'un service OVHcloud Connect

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation Network
- Sidebar avec OVHcloud Connect actif
- Breadcrumb (Network > OVHcloud Connect > {nom})
- Titre avec description
- Onglets de navigation
- Dashboard avec tiles d'information

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Navigation contextuelle
- **OdsText** : Titre et description
- **OdsTabs** : Navigation (Vue générale, Clés de service, Tâches, Diagnostics, Statistiques)
- **OdsCard** : Tiles d'information (4)
- **OdsBadge** : Statuts
- **OdsButton** : Actions (Télécharger LOA, Ajouter datacenter, Modifier vRack)

## 4. ONGLETS
| Onglet | Description |
|--------|-------------|
| Vue générale | Dashboard principal (actif) |
| Clés de service | Gestion des clés d'authentification |
| Tâches | Historique des opérations |
| Diagnostics | Tests de connectivité |
| Statistiques | Graphiques de performance |
