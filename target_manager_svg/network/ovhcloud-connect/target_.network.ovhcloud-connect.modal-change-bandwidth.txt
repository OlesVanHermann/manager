# Cahier des Charges : network/cloud-connect - Modal modifier bande passante

## 1. VUE D'ENSEMBLE
- **Type** : Modal de modification avec warning
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Modifier la bande passante de l'offre

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Sélection offre
- **OdsMessage** : Warning frais supplémentaires
- **OdsButton** : Annuler, Confirmer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Modifier la bande passante" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU

### Champ
| Champ | Type | Options exemple |
|-------|------|-----------------|
| Sélectionner une offre | Select | 1 Gbps, 10 Gbps, 100 Gbps |

### Message warning
| Type | Contenu |
|------|---------|
| Warning (jaune) | "Cette modification peut entraîner des frais supplémentaires." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Confirmer | Warning (orange) | Applique la modification |

## 7. APIS & ENDPOINTS
- GET /ovhCloudConnect/{serviceName}/availableOffers - Offres disponibles
- PUT /ovhCloudConnect/{serviceName}/bandwidth - Modifier bande passante
