# Cahier des Charges : network/cloud-connect - Modal bloquer port

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Page interfaces Cloud Connect
- **Objectif** : Bloquer un port pour interrompre le trafic

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Bloquer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Bloquer le port" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir bloquer le port {portName} ?" |
| Warning | "Le trafic sera interrompu sur ce port." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Bloquer | Warning (orange) | Bloque le port |

## 7. APIS & ENDPOINTS
- PUT /ovhCloudConnect/{serviceName}/interface/{id}/lock - Bloquer port
