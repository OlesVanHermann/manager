# Cahier des Charges : network/cloud-connect - Modal supprimer datacenter

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation warning
- **Contexte** : Dashboard Cloud Connect
- **Objectif** : Supprimer la configuration d'un datacenter

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 160px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsText** : Message de confirmation
- **OdsButton** : Annuler, Supprimer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Supprimer le datacenter" |
| Fond | Jaune (#FEF3C7) |
| Couleur texte | Orange (#D97706) |
| Bouton fermer | × |

## 5. CONTENU MODAL
| Élément | Description |
|---------|-------------|
| Message | "Êtes-vous sûr de vouloir supprimer la configuration de ce datacenter ?" |
| Warning | "Cette action est irréversible." |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Warning (orange) | Supprime le datacenter |

## 7. APIS & ENDPOINTS
- DELETE /ovhCloudConnect/{serviceName}/datacenter/{id} - Supprimer datacenter
