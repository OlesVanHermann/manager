# Cahier des Charges : network/cloud-connect - Modal envoyer clé de service

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'envoi
- **Contexte** : Page clés de service Cloud Connect
- **Objectif** : Envoyer la clé de service par email

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Adresse email
- **OdsButton** : Annuler, Envoyer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Titre | "Envoyer la clé de service" |
| Fond | Bleu (#0050D7) |
| Couleur texte | Blanc |
| Bouton fermer | × |

## 5. CONTENU

### Champ
| Champ | Type | Placeholder |
|-------|------|-------------|
| Adresse email * | Input email | exemple@domaine.com |

## 6. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Envoyer | Primary | Envoie la clé par email |

## 7. APIS & ENDPOINTS
- POST /ovhCloudConnect/{serviceName}/serviceKey/send - Envoyer clé par email
