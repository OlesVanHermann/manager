# Cahier des Charges : network/vrack - Onglet Tâches

## 1. VUE D'ENSEMBLE
- **Page** : Historique des tâches vRack
- **URL** : /network/vrack/{serviceName}/tasks
- **Objectif** : Suivre les opérations en cours et passées

## 2. STRUCTURE PAGE
- Header avec titre vRack
- Onglets de navigation
- Datagrid des tâches

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Liste des tâches
- **OdsBadge** : Statuts (done/running/error)
- **OdsSpinner** : Tâches en cours

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| ID | Text | Identifiant tâche |
| Type | Text | Type d'opération |
| Service | Link | Service concerné |
| Statut | Badge | État de la tâche |
| Date début | DateTime | Démarrage |
| Date fin | DateTime | Terminaison |

## 5. STATUTS TÂCHES
| Statut | Badge | Couleur |
|--------|-------|---------|
| done | Terminé | Vert |
| running | En cours | Bleu + spinner |
| error | Erreur | Rouge |
| pending | En attente | Gris |
| cancelled | Annulé | Gris |

## 6. TYPES DE TÂCHES
| Type | Description |
|------|-------------|
| addService | Ajout d'un service |
| removeService | Retrait d'un service |
| moveService | Déplacement vers autre vRack |

## 7. APIS & ENDPOINTS
- GET /vrack/{serviceName}/task - Liste des tâches
- GET /vrack/{serviceName}/task/{taskId} - Détail tâche
