# Cahier des Charges : network/vrack - Modal Retirer service

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation retrait service
- **Déclencheur** : Action "Retirer" dans menu contextuel
- **Objectif** : Confirmer le retrait d'un service du vRack

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 380px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsIcon** : Warning (triangle jaune)
- **OdsText** : Message confirmation
- **OdsButton** : Annuler / Confirmer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | Icône warning + "Retirer un service" |
| Body | Message + nom du service |
| Footer | Boutons Annuler / Confirmer |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Icône | ⚠️ (warning jaune) |
| Question | "Êtes-vous sûr(e) de vouloir retirer ce service de votre vRack ?" |
| Service | Nom du service concerné (bold) |
| Note | "Le service ne sera plus connecté au réseau privé" |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary outline | Fermer modal |
| Confirmer | Danger (rouge) | Exécuter retrait |

## 7. APIS & ENDPOINTS
- DELETE /vrack/{serviceName}/dedicatedServer/{server} - Retirer serveur
- DELETE /vrack/{serviceName}/cloudProject/{project} - Retirer projet
- DELETE /vrack/{serviceName}/ipLoadbalancing/{lb} - Retirer LB
