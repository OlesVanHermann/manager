# Cahier des Charges : network/vrack - Modal Commander

## 1. VUE D'ENSEMBLE
- **Modal** : Commande d'un nouveau vRack
- **Déclencheur** : Bouton "Commander un vRack"
- **Objectif** : Commander un vRack gratuitement

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 450px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsMessage** : Bandeau info gratuit (vert)
- **OdsText** : Conditions d'utilisation
- **OdsLink** : Liens vers contrats
- **OdsCheckbox** : Acceptation contrats
- **OdsButton** : Annuler / Commander

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "← Commander un vRack" |
| Info | Bandeau "Gratuit" |
| Contrats | Liens + checkbox |
| Footer | Boutons Annuler / Commander |

## 5. BANDEAU INFORMATION
| Propriété | Valeur |
|-----------|--------|
| Style | Success (vert #DEF7EC) |
| Icône | ✓ |
| Texte | "La commande d'un vRack est entièrement gratuite" |

## 6. CONTRATS
| Contrat | Type |
|---------|------|
| Conditions particulières vRack | Lien externe ↗ |

## 7. CHECKBOX ACCEPTATION
| Propriété | Valeur |
|-----------|--------|
| Label | "J'ai pris connaissance et j'accepte les termes des contrats" |
| Requis | Oui |

## 8. BOUTONS
| Bouton | Style | Condition |
|--------|-------|-----------|
| Annuler | Secondary | Toujours actif |
| Commander | Primary | Actif si checkbox coché |

## 9. APIS & ENDPOINTS
- GET /order/vrack/new - Récupérer bon de commande
- POST /order/vrack/new - Valider commande
- GET /me/order - Historique commandes
