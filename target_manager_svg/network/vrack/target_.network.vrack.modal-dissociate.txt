# Cahier des Charges : network/vrack-services - Modal Dissocier

## 1. VUE D'ENSEMBLE
- **Modal** : Dissociation du vRack
- **Déclencheur** : Action "Dissocier"
- **Objectif** : Retirer l'association avec un vRack

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 300px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsMessage** : Warning box jaune
- **OdsText** : Informations
- **OdsButton** : Annuler / Dissocier

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Dissocier le vRack Services de votre vRack" |
| Body | Warning + infos |
| Footer | Boutons |

## 5. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Style | Warning (jaune #FEF3C7) |
| Titre | "⚠️ Attention" |
| Message | "Êtes-vous sûr de vouloir dissocier ce vRack Services de votre vRack ?" |

## 6. INFORMATIONS
| Champ | Valeur |
|-------|--------|
| vRack Services | vs-abc123-prod |
| vRack actuel | pn-123456 |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer |
| Dissocier | Danger | Exécuter |

## 8. APIS & ENDPOINTS
- POST /vrackServices/{id}/vrack/dissociate - Dissocier
