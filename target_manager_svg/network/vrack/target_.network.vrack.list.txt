# Cahier des Charges : network/vrack-services - Liste

## 1. VUE D'ENSEMBLE
- **Page** : Liste des vRack Services
- **URL** : /network/vrack-services
- **Objectif** : Afficher et gérer tous les vRack Services

## 2. STRUCTURE PAGE
- Header OVHcloud avec navigation Network
- Sidebar avec vRack Services sélectionné
- Titre + description du service
- Bouton d'activation
- Datagrid des services

## 3. COMPOSANTS UI
- **OdsText** : Titre "vRack Services"
- **OdsText** : Description multilignes
- **OdsButton** : "+ Activer vRack Services" (outline)
- **OdsDatagrid** : Liste des services
- **OdsBadge** : Statuts (Actif/Brouillon/Updating)
- **OdsActionMenu** : Menu contextuel

## 4. DESCRIPTION SERVICE
```
Le produit vRack Services vous permet de bénéficier de services réseau sur le vRack.
En créant un Service Endpoint, vous pouvez exposer, avec une adresse IP privée,
un service managé par OVHcloud sur votre vRack.
```

- GET /vrackServices - Liste des vRack Services
