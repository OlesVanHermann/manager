# Cahier des Charges : network/vrack-services - Modal Créer endpoint

## 1. VUE D'ENSEMBLE
- **Page** : Page création Service Endpoint
- **URL** : /network/vrack-services/{id}/endpoints/create
- **Objectif** : Créer un nouveau endpoint

## 2. STRUCTURE PAGE
- Lien retour
- Titre
- Description explicative
- Formulaire
- Boutons

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre et description
- **OdsSelect** : Type service / Nom service / Sous-réseau
- **OdsButton** : Créer / Annuler

## 4. DESCRIPTION
```
Le Service Endpoint vous permet d'exposer vos services managés par OVHcloud sur le vRack,
via une ou plusieurs adresses IP, selon le type de service. L'adresse IP du Service Endpoint
appartient à la plage d'adresses de service définie dans le sous-réseau sélectionné.
```

## 5. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Type du service managé | Select | Oui | Enterprise File Storage |
| Nom du service managé | Select | Oui | efs-prod-storage-01 |
| Sous-réseau | Select | Oui | subnet-prod - 10.0.0.0/24 |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Créer un Service Endpoint | Primary | Valider |
| Annuler | Outline | Retour |

## 7. APIS & ENDPOINTS
- GET /vrackServices/{id}/availableServices - Services éligibles
- GET /vrackServices/{id}/subnets - Sous-réseaux disponibles
- POST /vrackServices/{id}/endpoints - Créer endpoint
