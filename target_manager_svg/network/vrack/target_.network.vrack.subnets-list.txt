# Cahier des Charges : network/vrack-services - Liste sous-réseaux

## 1. VUE D'ENSEMBLE
- **Page** : Liste des sous-réseaux (variante)
- **URL** : /network/vrack-services/{id}/subnets
- **Objectif** : Gérer les sous-réseaux existants

## 2. STRUCTURE PAGE
- Breadcrumb (vRack Services › {id} › Sous-réseaux)
- Titre service
- Onglets
- Message info
- Bouton création (disabled si max atteint)
- Datagrid

## 3. COMPOSANTS UI
- **OdsTabs** : Vue d'ensemble / Sous-réseaux / Service Endpoints
- **OdsMessage** : Info limitation (bleu)
- **OdsButton** : "+ Créer un sous-réseau" (disabled si 1 existe)
- **OdsDatagrid** : Liste sous-réseaux
- **OdsActionMenu** : Actions

## 4. MESSAGE INFO
