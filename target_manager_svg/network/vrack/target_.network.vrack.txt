# Cahier des Charges : network/vrack-services - Dashboard

## 1. VUE D'ENSEMBLE
- **Page** : Dashboard vRack Services
- **URL** : /network/vrack-services
- **Objectif** : Vue principale avec liste et actions

## 2. STRUCTURE PAGE
- Header OVHcloud avec breadcrumb
- Sidebar Network
- Description du service
- Bouton activation
- Datagrid avec ActionMenu ouvert

## 3. COMPOSANTS UI
- **OdsBreadcrumb** : Network / vRack Services
- **OdsText** : Titre et description
- **OdsButton** : "+ Activer vRack Services"
- **OdsDatagrid** : Liste services
- **OdsBadge** : Statuts
- **OdsActionMenu** : Menu contextuel (ouvert)

## 4. COLONNES DATAGRID
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Link bleu | vs-xxx-yyy |
| Statut | Badge | ACTIVE/UPDATING |
| Région | Text | France, Gravelines (GRA) |
| vRack | Link/muted | pn-xxxxx ou "Non associé" |
| Date de création | Date | DD/MM/YYYY |
| Actions | Menu ⋮ | Menu contextuel |

## 5. ACTION MENU (ouvert)
| Action | Style |
|--------|-------|
| Voir les détails | Normal |
| Modifier le nom | Normal |
| Associer à un vRack | Normal |
| Dissocier | Normal |
| Supprimer | Danger (rouge) |

## 6. APIS & ENDPOINTS
- GET /vrackServices - Liste
- PUT /vrackServices/{id} - Modifier nom
- POST /vrackServices/{id}/associate - Associer vRack
- POST /vrackServices/{id}/dissociate - Dissocier
- DELETE /vrackServices/{id} - Supprimer
