# Cahier des Charges : network/vrack - Onglet Informations générales

## 1. VUE D'ENSEMBLE
- **Page** : Informations générales du vRack
- **URL** : /network/vrack/{serviceName}/general
- **Objectif** : Afficher les stats et infos du vRack

## 2. STRUCTURE PAGE
- Header avec titre et badge statut
- Cartes statistiques
- Informations détaillées

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes stats
- **OdsBadge** : Statut
- **OdsTile** : Informations

## 4. STATISTIQUES SERVICES
| Carte | Valeur exemple | Description |
|-------|----------------|-------------|
| Total services | 12 | Nombre total |
| Serveurs dédiés | 5 | Type dedicated |
| Public Cloud | 3 | Projets cloud |
| Autres | 4 | IP, LB, etc. |

## 5. INFORMATIONS VRACK
| Champ | Type | Exemple |
|-------|------|---------|
| Nom | Text | pn-123456 |
| ID | Text | vrack-abc123 |
| Description | Editable | "Production network" |
| Date création | Date | 15/01/2024 |
| Statut | Badge | Actif (vert) |

## 6. ACTIONS DISPONIBLES
| Action | Type | Description |
|--------|------|-------------|
| Modifier description | Edit inline | Changer la description |
| Modifier nom | Edit inline | Renommer le vRack |

## 7. APIS & ENDPOINTS
- GET /vrack/{serviceName} - Informations vRack
- PUT /vrack/{serviceName} - Modifier description
- GET /vrack/{serviceName}/serviceInfos - Infos service
