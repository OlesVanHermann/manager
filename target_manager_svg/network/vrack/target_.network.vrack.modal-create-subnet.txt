# Cahier des Charges : network/vrack-services - Modal Créer sous-réseau

## 1. VUE D'ENSEMBLE
- **Page** : Page création sous-réseau
- **URL** : /network/vrack-services/{id}/subnets/create
- **Objectif** : Créer un nouveau sous-réseau

## 2. STRUCTURE PAGE
- Lien retour "← Créer un sous-réseau"
- Titre
- Formulaire complet
- Boutons action

## 3. COMPOSANTS UI
- **OdsLink** : Retour
- **OdsText** : Titre
- **OdsInput** : Champs texte
- **OdsRadio** : Options VLAN
- **OdsButton** : Créer / Annuler

## 4. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire | Exemple |
|-------|------|-------------|---------|
| Nom du sous-réseau | Input | Non | Mon sous-réseau |
| Plage d'adresses (CIDR) | Input | Oui | 10.0.0.0/24 |
| Plage d'adresses de service | Input | Oui | 10.0.0.0/28 |
| VLAN | Radio | Non | Pas de VLAN / Sélectionner |

## 5. AIDE CONTEXTUELLE
| Champ | Helper text |
|-------|-------------|
| Plage CIDR | "La plage d'adresses du sous-réseau doit être en /24." |
| Plage service | "La plage d'adresse de service doit être un sous-ensemble de la plage du sous-réseau, et sa taille doit être comprise entre /27 et /29." |

## 6. OPTIONS VLAN
| Option | Sélectionné |
|--------|-------------|
| Pas de VLAN | ✓ (défaut) |
| Sélectionnez un VLAN | - |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Créer un sous-réseau | Primary | Valider |
| Annuler | Outline | Retour |

## 8. APIS & ENDPOINTS
- POST /vrackServices/{id}/subnets - Créer sous-réseau
