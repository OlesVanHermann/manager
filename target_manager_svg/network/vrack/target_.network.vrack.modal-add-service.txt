# Cahier des Charges : network/vrack - Modal Ajouter services

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation ajout services
- **Déclencheur** : Bouton "Ajouter" ou flèche →
- **Objectif** : Confirmer l'ajout de services au vRack

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 380px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsText** : Message confirmation
- **OdsList** : Services sélectionnés
- **OdsButton** : Annuler / Confirmer

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Ajouter des services" |
| Body | Liste des services sélectionnés |
| Footer | Boutons Annuler / Confirmer |

## 5. CONTENU
| Élément | Valeur |
|---------|--------|
| Question | "Êtes-vous sûr(e) de vouloir ajouter ces services à votre vRack ?" |
| Liste | Services sélectionnés (noms) |
| Note | "Cette opération peut prendre quelques minutes" |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary outline | Fermer modal |
| Confirmer | Primary | Exécuter ajout |

## 7. APIS & ENDPOINTS
- POST /vrack/{serviceName}/dedicatedServer - Ajouter serveur
- POST /vrack/{serviceName}/cloudProject - Ajouter projet cloud
- POST /vrack/{serviceName}/ipLoadbalancing - Ajouter LB
