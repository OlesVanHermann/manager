# Cahier des Charges : network/vrack-services - Modal Modifier nom affichage

## 1. VUE D'ENSEMBLE
- **Modal** : Modification du nom d'affichage
- **Déclencheur** : Icône ✏️ sur le nom
- **Objectif** : Changer le nom d'affichage

## 2. DIMENSIONS
- Largeur : 600px
- Hauteur : 260px

## 3. COMPOSANTS UI
- **OdsModal** : Container
- **OdsInput** : Champ nom
- **OdsButton** : Annuler / Modifier

## 4. STRUCTURE MODAL
| Section | Contenu |
|---------|---------|
| Header | "Modifier le nom" |
| Body | Input avec valeur actuelle |
| Footer | Boutons |

## 5. CHAMP INPUT
| Propriété | Valeur |
|-----------|--------|
| Label | "Nom d'affichage" |
| Type | Input text |
| Valeur | vs-production-01 |
| Max length | 64 caractères |

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Fermer |
| Modifier | Primary | Sauvegarder |

## 7. APIS & ENDPOINTS
- PUT /vrackServices/{id} - Modifier nom
