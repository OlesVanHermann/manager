# Cahier des Charges : network/security - Trafic

## 1. VUE D'ENSEMBLE
- **Page** : Statistiques de trafic réseau
- **URL** : /network/security/traffic
- **Objectif** : Visualiser le trafic légitime vs filtré

## 2. STRUCTURE PAGE
- Titre
- Onglets avec "Trafic" actif
- Filtres (IP, Période)
- Cartes statistiques (4)
- Graphique de trafic
- Bouton export

## 3. COMPOSANTS UI
- **OdsTabs** : Vue d'ensemble, Scrubbing Center, Trafic (actif)
- **OdsSelect** : Filtre IP, Filtre période
- **OdsCard** : Cartes métriques (4)
- **OdsChart** : Graphique ligne/aire
- **OdsButton** : Exporter les données (outline)

## 4. FILTRES
| Filtre | Type | Options |
|--------|------|---------|
| IP | Select | Toutes les IP / IP spécifique |
| Période | Select | Dernières 24h, 7j, 30j |

## 5. CARTES STATISTIQUES
| Métrique | Valeur | Couleur |
|----------|--------|---------|
| Trafic total entrant | 1.2 Tbps | Normal |
| Trafic légitime | 1.15 Tbps | Vert |
| Trafic filtré (attaque) | 50 Gbps | Rouge |
| Efficacité filtrage | 99.2% | Bleu |

## 6. GRAPHIQUE TRAFIC
| Propriété | Valeur |
|-----------|--------|
| Type | Line chart avec area |
| Titre | "Trafic réseau (bps)" |
| Axe Y | 0 - 2 Tbps |
| Axe X | 00:00 - 24:00 |
| Ligne verte | Trafic légitime |
| Zone rouge | Trafic filtré (attaque) |

| Couleur | Label |
|---------|-------|

## 8. BOUTON EXPORT
| Propriété | Valeur |
|-----------|--------|

## 9. APIS & ENDPOINTS
- GET /ip/antiddos/traffic - Statistiques trafic
- GET /ip/{ip}/antiddos/traffic - Trafic par IP
- GET /ip/antiddos/traffic/export - Export données
