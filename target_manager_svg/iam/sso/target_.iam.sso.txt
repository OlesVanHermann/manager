# Cahier des Charges : iam/sso - Configuration SSO

## 1. VUE D'ENSEMBLE
- **Type** : Page configuration
- **URL** : /iam/sso
- **Objectif** : Configurer l'authentification SSO

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / SSO
- **OdsToggle** : Activer SSO
- **OdsButton** : + Ajouter un IdP
- **OdsDatagrid** : Liste IdP
- **OdsBadge** : Type, Statut

## 4. SECTION ACTIVATION
| Élément | Description |
|---------|-------------|
| Toggle | Activer/Désactiver SSO global |
| Description | "Permet aux utilisateurs de se connecter via un fournisseur d'identité externe." |

## 5. COLONNES TABLEAU IdP
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien | 200px |
| Type | Badge | 100px |
| Entity ID / Issuer | Code | 300px |
| Statut | Badge | 100px |
| Actions | Menu ⋮ | 50px |

## 6. TYPES IdP
| Type | Description |
|------|-------------|
| SAML 2.0 | Protocole SAML |
| OIDC | OpenID Connect |

## 7. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir détails | Page détail IdP |
| Modifier | Éditer configuration |
| Tester | Tester connexion |
| Supprimer | Supprimer IdP |

## 8. APIS & ENDPOINTS
- GET /iam/sso - Configuration SSO
- PUT /iam/sso - Modifier config globale
- GET /iam/sso/idp - Liste IdP
- POST /iam/sso/idp - Ajouter IdP
