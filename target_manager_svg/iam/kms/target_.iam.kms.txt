# Cahier des Charges : iam/kms - Key Management Service

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **URL** : /iam/kms
- **Objectif** : Gérer les services KMS (Key Management Service)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / KMS
- **OdsButton** : + Commander un KMS
- **OdsCard** : Container tableau
- **OdsDatagrid** : Tableau services KMS
- **OdsBadge** : Statut

## 4. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien | 250px |
| Région | Texte | 120px |
| Clés de service | Badge compteur | 150px |
| Credentials | Badge compteur | 150px |
| Statut | Badge | 100px |
| Créé le | Date | 120px |
| Actions | Menu ⋮ | 50px |

## 5. BADGES STATUT
| Badge | Couleur fond | Couleur texte |
|-------|--------------|---------------|
| Actif | #DEF7EC | #03543F |
| Suspendu | #FEF3C7 | #D97706 |
| En suppression | #FEE2E2 | #991B1B |

## 6. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir détails | Ouvre page détail |
| Renommer | Modifier le nom |
| Résilier | Supprimer le service |

## 7. APIS & ENDPOINTS
- GET /kms - Liste services KMS
- POST /kms/order - Commander KMS
- DELETE /kms/{kmsId}/terminate - Résilier KMS
