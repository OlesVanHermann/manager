# Cahier des Charges : iam/policies - Modification politique

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire édition
- **URL** : /iam/policies/{policyId}/edit
- **Objectif** : Modifier une politique personnalisée existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 1200px (scrollable) |

## 3. COMPOSANTS UI
- Identique à la page de création
- Champs pré-remplis avec les données existantes

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | "Modifier la politique" |
| Sous-titre | Nom de la politique |

## 5. CHAMPS MODIFIABLES
| Champ | Modifiable |
|-------|------------|
| Nom | Oui |
| Description | Oui |
| Identités | Oui |
| Types de produits | Oui |
| Ressources | Oui |
| Groupes de ressources | Oui |
| Conditions | Oui |
| Actions | Oui |

## 6. CHAMPS NON MODIFIABLES
| Champ | Raison |
|-------|--------|
| ID politique | Identifiant unique |
| Date création | Historique |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour détail |
| Enregistrer | Primary | Sauvegarde modifications |

## 8. APIS & ENDPOINTS
- GET /iam/policies/{policyId} - Récupérer données existantes
- PUT /iam/policies/{policyId} - Enregistrer modifications
