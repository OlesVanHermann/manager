# Cahier des Charges : iam/identities - Modal suppression SSO IdP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu IdP
- **Objectif** : Supprimer un IdP SSO

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 320px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsInput** : Confirmation
- **OdsButton** : Annuler, Supprimer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer la configuration SSO" |
| Warning | "Les utilisateurs SSO liés à cet IdP ne pourront plus se connecter." |
| Confirmation | Saisir le nom de l'IdP |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime IdP |

## 6. APIS & ENDPOINTS
- DELETE /iam/sso/idp/{idpId} - Supprimer IdP
