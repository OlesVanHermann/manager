# Cahier des Charges : iam/identities - Modal modification utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Action menu utilisateur
- **Objectif** : Modifier les informations d'un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs texte
- **OdsSelect** : Groupes
- **OdsButton** : Annuler, Enregistrer

## 4. CHAMPS MODIFIABLES
| Champ | Type | Requis |
|-------|------|--------|
| Prénom | Input | Oui * |
| Nom | Input | Oui * |
| Description | Textarea | Non |
| Groupes | Multi-select | Non |

## 5. CHAMPS NON MODIFIABLES
| Champ | Raison |
|-------|--------|
| Email | Identifiant unique |
| Login | Généré à la création |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde modifications |

## 7. APIS & ENDPOINTS
- GET /iam/users/{userId} - Récupérer données
- PUT /iam/users/{userId} - Modifier utilisateur
