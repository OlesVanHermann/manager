# Cahier des Charges : iam/identities - Modal suppression groupe

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu groupe
- **Objectif** : Supprimer un groupe

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 300px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Supprimer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer le groupe" |
| Question | "Êtes-vous sûr de vouloir supprimer le groupe {nom} ?" |
| Warning | "Les utilisateurs du groupe ne seront pas supprimés." |
| Info | "Les politiques attachées à ce groupe seront détachées." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime groupe |

## 6. APIS & ENDPOINTS
- DELETE /iam/groups/{groupId} - Supprimer groupe
