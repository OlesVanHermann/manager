# Cahier des Charges : iam/identities - Onglet Utilisateurs

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau
- **URL** : /iam/identities?tab=users
- **Objectif** : Gérer les utilisateurs locaux IAM

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsSearchField** : Recherche utilisateur
- **OdsDatagrid** : Tableau utilisateurs
- **OdsBadge** : Statut (Actif/Désactivé)
- **OdsAvatar** : Initiales utilisateur

## 3. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Utilisateur | Avatar + Nom + Description | 276px |
| Email | Texte | 250px |
| Groupes | Liens | 150px |
| Statut | Badge | 100px |
| Dernière connexion | Texte relatif | 150px |
| Actions | Menu ⋮ | 50px |

## 4. BADGES STATUT
| Badge | Couleur fond | Couleur texte |
|-------|--------------|---------------|
| Actif | #DEF7EC | #03543F |
| Désactivé | #FEE2E2 | #991B1B |

## 5. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Éditer les informations |
| Réinitialiser mot de passe | Envoyer email reset |
| Désactiver/Activer | Toggle statut |
| Supprimer | Supprimer utilisateur |

## 6. APIS & ENDPOINTS
- GET /iam/users - Liste utilisateurs
- POST /iam/users - Créer utilisateur
