# Cahier des Charges : iam/identities - Modal modification groupe

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Action menu groupe
- **Objectif** : Modifier un groupe existant

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom du groupe
- **OdsTextarea** : Description
- **OdsSelect** : Membres
- **OdsButton** : Annuler, Enregistrer

## 4. CHAMPS MODIFIABLES
| Champ | Type | Requis |
|-------|------|--------|
| Nom du groupe | Input | Oui * |
| Description | Textarea | Non |
| Membres | Multi-select | Non |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde modifications |

## 6. APIS & ENDPOINTS
- GET /iam/groups/{groupId} - Récupérer données
- PUT /iam/groups/{groupId} - Modifier groupe
