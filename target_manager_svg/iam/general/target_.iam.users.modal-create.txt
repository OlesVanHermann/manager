# Cahier des Charges : iam/users - Modal création utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page liste utilisateurs
- **Objectif** : Créer un nouvel utilisateur local

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champs texte
- **OdsSelect** : Groupes
- **OdsToggle** : Statut actif
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Prénom | Input | Oui * |
| Nom | Input | Oui * |
| Email | Input email | Oui * |
| Login | Input | Auto-généré |
| Description | Textarea | Non |
| Groupes | Multi-select | Non |
| Actif | Toggle | Défaut: Oui |

## 5. VALIDATION
| Champ | Règle |
|-------|-------|
| Email | Format email valide, unique |
| Login | Généré depuis email, modifiable |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée utilisateur |

## 7. POST-CRÉATION
- Email envoyé avec lien création mot de passe

## 8. APIS & ENDPOINTS
- POST /iam/users - Créer utilisateur
- GET /iam/groups - Liste groupes
