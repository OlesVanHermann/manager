# Cahier des Charges : iam/components - Search Results

## 1. VUE D'ENSEMBLE
- **Type** : Composant résultats de recherche
- **Usage** : Afficher les résultats d'une recherche

## 2. COMPOSANT ODS
- **OdsDatagrid** : Tableau filtré
- **OdsMessage** : Empty state

## 3. ÉTATS
| État | Affichage |
|------|-----------|
| Loading | Spinner centré |
| Résultats | Tableau avec highlight |
| Vide | Message "Aucun résultat" |

## 4. HIGHLIGHT
| Propriété | Valeur |
|-----------|--------|
| Background | #FEF3C7 |
| Text | Bold sur terme recherché |

## 5. EMPTY STATE
| Élément | Contenu |
|---------|---------|
| Icône | Loupe |
| Titre | "Aucun résultat" |
| Message | "Aucun élément ne correspond à '{terme}'" |
| Action | "Effacer la recherche" |

## 6. COMPTEUR
| Format | Exemple |
|--------|---------|
| Résultats | "X résultats pour '{terme}'" |
