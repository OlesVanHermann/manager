# Cahier des Charges : iam/conditions - Modal modification

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Action menu condition
- **Objectif** : Modifier une condition existante

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 500px (variable) |

## 3. COMPOSANTS UI
- Identique à modal création
- Champs pré-remplis avec valeurs existantes

## 4. CHAMPS NON MODIFIABLES
| Champ | Raison |
|-------|--------|
| Type | Définit la structure |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde modifications |

## 6. APIS & ENDPOINTS
- GET /iam/conditions/{condId} - Récupérer données
- PUT /iam/conditions/{condId} - Modifier condition
