# Cahier des Charges : iam/components - Confirmation Modal

## 1. VUE D'ENSEMBLE
- **Type** : Composant modal générique
- **Usage** : Demander confirmation avant action

## 2. COMPOSANT ODS
- **OdsModal** : Modal de confirmation

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 400-500px |
| Position | Centré |

## 4. ÉLÉMENTS
| Élément | Description |
|---------|-------------|
| Titre | Question/Action |
| Message | Description des conséquences |
| Boutons | Annuler, Confirmer |

## 5. VARIANTES
| Type | Bouton confirmer |
|------|------------------|
| Info | Primary (bleu) |
| Warning | Warning (orange) |
| Danger | Danger (rouge) |

## 6. AVEC SAISIE
| Propriété | Valeur |
|-----------|--------|
| Champ | Input de confirmation |
| Mot | "DELETE" ou "TERMINATE" |
| Bouton | Désactivé jusqu'à saisie correcte |
