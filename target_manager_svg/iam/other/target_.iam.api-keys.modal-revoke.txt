# Cahier des Charges : iam/api-keys - Modal révocation clé API

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu sur une clé API
- **Objectif** : Révoquer (désactiver) une clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 320px |
| Position | Centrée |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning box (rouge)
- **OdsButton** : Annuler, Révoquer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Révoquer la clé API" |
| Bouton fermer | × en haut droite |

## 5. CONTENU
| Élément | Description |
|---------|-------------|
| Question | "Êtes-vous sûr de vouloir révoquer la clé API" |
| Nom clé | « {nom} » (gras) |
| Client ID | ({clientId}) |

## 6. WARNING BOX
| Propriété | Valeur |
|-----------|--------|
| Background | #FEE2E2 |
| Border | #DC2626 |
| Icône | ⚠️ |

### Messages d'avertissement
| Ligne | Texte |
|-------|-------|
| 1 | "Attention : Cette action est irréversible." |
| 2 | "Toutes les applications utilisant cette clé cesseront" |
| 3 | "immédiatement de fonctionner." |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Révoquer | Danger (rouge) | Révoque la clé |

## 8. APIS & ENDPOINTS
- POST /iam/api-keys/{keyId}/revoke - Révoquer clé API
