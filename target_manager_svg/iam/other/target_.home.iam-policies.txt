# Cahier des Charges : home/iam - Politiques

## 1. VUE D'ENSEMBLE
- **Type** : Page liste politiques IAM
- **URL** : /iam/policies
- **Objectif** : Gérer politiques d'accès

## 2. STRUCTURE PAGE
- Titre + bouton Guides
- Onglets politiques
- Description
- Boutons création + recherche
- Tableau politiques

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Liste politiques
- **OdsButton** : Créer, Recherche avancée
- **OdsActionMenu** : Actions

## 4. ONGLETS
| Onglet | Active | Description |
|--------|--------|-------------|
| Mes politiques | Oui | Politiques personnalisées |
| Politiques OVHcloud | Non | Templates OVH |
| Groupes de ressources | Non | Regroupements |

## 5. DESCRIPTION
"Gérez les identités de vos utilisateurs et de vos applications, ainsi que leurs droits d'accès dans une interface unique pour tous vos services."

## 6. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Lien bleu | Nom politique |
| Description | Muted | Description courte |
| Identités | Nombre | Nb identités liées |
| Ressources | Nombre | Nb ressources |
| Actions | Nombre | Nb actions autorisées |
| Menu | ⋮ | ActionMenu |

## 7. BOUTONS
| Label | Style | Action |
|-------|-------|--------|
| Créer une politique | Primary | Ouvre formulaire |
| Recherche avancée | Secondary outline | Filtre avancé |

## 8. ACTIONMENU OPTIONS
| Option | Couleur |
|--------|---------|
| Modifier la politique | Normal |
| Supprimer la politique | Rouge |

## 9. APIS & ENDPOINTS
- GET /me/iam/policy - Liste politiques
- POST /me/iam/policy - Créer politique
- PUT /me/iam/policy/{id} - Modifier
- DELETE /me/iam/policy/{id} - Supprimer
