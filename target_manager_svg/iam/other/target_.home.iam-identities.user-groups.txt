# Cahier des Charges : home/iam - Groupes Utilisateurs

## 1. VUE D'ENSEMBLE
- **Type** : Page liste groupes
- **URL** : /iam/identities/groups
- **Objectif** : Gérer groupes d'utilisateurs

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar IAM
- Onglets (actif: Groupes)
- Bouton déclarer groupe
- Tableau groupes

## 3. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Liste groupes
- **OdsButton** : Déclarer groupe
- **OdsActionMenu** : Actions

## 4. ONGLETS
| Onglet | Active |
|--------|--------|
| Utilisateurs locaux | Non |
| Groupes d'utilisateurs | Oui |
| SSO | Non |

## 5. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien bold | 160px |
| Rôle | Texte | 180px |
| Description | Muted | 200px |
| Création | Date | 130px |
| Dernière modification | Date | 160px |
| Actions | Menu ⋮ | 50px |

## 6. RÔLES DISPONIBLES
| Rôle | Description |
|------|-------------|
| Administrateur | Accès complet |
| Administrateur restreint | Accès limité |
| Lecture seule | Consultation uniquement |

## 7. ACTIONMENU OPTIONS
| Option | Couleur |
|--------|---------|
| Modifier le groupe | Normal |
| Supprimer le groupe | Rouge |

## 8. APIS & ENDPOINTS
- GET /me/identity/group - Liste groupes
- POST /me/identity/group - Créer groupe
- PUT /me/identity/group/{id} - Modifier
- DELETE /me/identity/group/{id} - Supprimer
