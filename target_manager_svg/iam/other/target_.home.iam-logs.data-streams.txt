# Cahier des Charges : home/iam - Data Streams

## 1. VUE D'ENSEMBLE
- **Type** : Page gestion abonnements logs
- **URL** : /iam/logs/data-streams
- **Objectif** : Configurer export vers LDP

## 2. STRUCTURE PAGE
- Lien retour
- Titre "Data Streams"
- Bannière info
- Bouton créer abonnement
- Tableau abonnements

## 3. COMPOSANTS UI
- **OdsLink** : Retour aux logs
- **OdsMessage** : Info LDP
- **OdsDatagrid** : Abonnements
- **OdsBadge** : Statut Actif
- **OdsButton** : Créer abonnement
- **OdsActionMenu** : Actions

## 4. BANNIÈRE INFO LDP
| Ligne | Contenu |
|-------|---------|
| L1 | "En vous abonnant aux logs de sécurité de votre compte client, ils seront transférés" |
| L2 | "vers la Logs Data Platform. Cette plateforme vous donne accès à des fonctionnalités" |
| L3 | "avancées telles que la gestion des alertes, l'exécution de recherches ou la gestion des rétentions." |

## 8. APIS & ENDPOINTS
- GET /me/iam/logs/subscription - Liste abonnements
- POST /me/iam/logs/subscription - Créer abonnement
- DELETE /me/iam/logs/subscription/{id} - Supprimer
- GET /dbaas/logs - Services LDP disponibles
