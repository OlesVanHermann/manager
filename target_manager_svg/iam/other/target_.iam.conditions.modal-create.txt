# Cahier des Charges : iam/conditions - Modal création

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Page conditions
- **Objectif** : Créer une nouvelle condition

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 500px (variable) |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom
- **OdsSelect** : Type
- **OdsTextarea** : Description
- **OdsForm** : Champs dynamiques par type
- **OdsButton** : Annuler, Créer

## 4. CHAMPS COMMUNS
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |
| Type | Select | Oui * |
| Description | Textarea | Non |

## 5. CHAMPS PAR TYPE
### IP
| Champ | Description |
|-------|-------------|
| Adresses IP | Liste d'IPs/CIDR autorisées |
| Mode | Whitelist/Blacklist |

### Time
| Champ | Description |
|-------|-------------|
| Jours | Jours de la semaine |
| Heures | Plage horaire (début-fin) |
| Timezone | Fuseau horaire |

### MFA
| Champ | Description |
|-------|-------------|
| Méthodes | TOTP, U2F, SMS |

### Region
| Champ | Description |
|-------|-------------|
| Pays | Liste de pays |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée condition |

## 7. APIS & ENDPOINTS
- POST /iam/conditions - Créer condition
