# Cahier des Charges : iam/dashboard - Onboarding IAM

## 1. VUE D'ENSEMBLE
- **Type** : Page onboarding wizard
- **URL** : /iam (premier accès)
- **Objectif** : Guider l'utilisateur dans la configuration IAM

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsCard** : Container principal
- **OdsStepper** : Étapes verticales avec connecteurs
- **OdsButton** : Actions par étape
- **OdsLink** : Passer l'introduction

## 4. HEADER
| Élément | Description |
|---------|-------------|
| Titre | "Bienvenue dans IAM" |
| Sous-titre | "Configurez la gestion des accès pour votre compte OVHcloud" |

## 5. STEPPER VERTICAL
| Propriété | Valeur |
|-----------|--------|
| Taille container | 1100x600px |
| Connecteur | Ligne verticale 2px #E5E7EB |

### Étapes
| Étape | État | Titre | Description | Action |
|-------|------|-------|-------------|--------|
| 1 | Actif (bleu) | Créer votre premier utilisateur | Ajoutez un utilisateur local pour déléguer des accès | Ajouter → |
| 2 | Pending | Définir des politiques d'accès | Créez des politiques pour contrôler les permissions | Créer une politique |
| 3 | Pending | Configurer les groupes | Organisez vos utilisateurs en groupes logiques | Créer un groupe |
| 4 | Pending | Générer des clés API | Créez des clés pour l'accès programmatique | Générer une clé API |

### Style cercle étape
| État | Background | Border | Texte |
|------|------------|--------|-------|
| Actif | #0050D7 | - | Blanc |
| Pending | #FFFFFF | #D1D5DB | Gris |

### Style bouton
| État | Style |
|------|-------|
| Actif | Primary #0050D7 |
| Pending | Outline #D1D5DB |

## 6. LIEN SKIP
| Élément | Description |
|---------|-------------|
| Texte | "Passer l'introduction →" |
| Position | Bas droite |
| Style | Lien bleu |

## 7. COMPORTEMENT
- Étape 1 active par défaut
- Compléter une étape débloque la suivante
- Option skip disponible à tout moment

## 8. APIS & ENDPOINTS
- POST /iam/users - Créer utilisateur
- POST /iam/policies - Créer politique
- POST /iam/groups - Créer groupe
- POST /iam/apikeys - Créer clé API
