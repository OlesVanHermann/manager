# Cahier des Charges : iam/tag-manager - Gestionnaire de tags

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **URL** : /iam/tags
- **Objectif** : Gérer les tags pour organiser les ressources

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Tags
- **OdsButton** : + Créer un tag
- **OdsDatagrid** : Tableau tags
- **OdsBadge** : Compteur ressources

## 4. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Tag | Lien avec couleur | 200px |
| Description | Texte | 300px |
| Ressources | Badge compteur | 150px |
| Créé le | Date | 150px |
| Actions | Menu ⋮ | 50px |

## 5. STYLE TAG
| Propriété | Valeur |
|-----------|--------|
| Format | clé:valeur |
| Couleur | Personnalisable |
| Border radius | 4px |

## 6. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir détails | Page détail tag |
| Modifier | Éditer le tag |
| Supprimer | Supprimer le tag |

## 7. APIS & ENDPOINTS
- GET /iam/tags - Liste tags
- POST /iam/tags - Créer tag
- PUT /iam/tags/{tagId} - Modifier tag
- DELETE /iam/tags/{tagId} - Supprimer tag
