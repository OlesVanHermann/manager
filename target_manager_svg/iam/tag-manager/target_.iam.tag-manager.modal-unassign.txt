# Cahier des Charges : iam/tag-manager - Modal retirer tag

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action sur ressource taggée
- **Objectif** : Retirer un tag d'une ressource

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 400px |
| Modal hauteur | 220px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsButton** : Annuler, Retirer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Retirer le tag" |
| Message | "Voulez-vous retirer le tag {tag} de la ressource {ressource} ?" |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Retirer | Primary | Retire tag |

## 6. APIS & ENDPOINTS
- DELETE /iam/tags/{tagId}/resources/{resourceId}
