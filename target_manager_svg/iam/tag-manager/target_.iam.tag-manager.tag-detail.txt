# Cahier des Charges : iam/tag-manager - Détail tag

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/tags/{tagId}
- **Objectif** : Afficher les ressources utilisant ce tag

## 2. COMPOSANTS UI
- **OdsBreadcrumb** : Accueil / IAM / Tags / {tag}
- **OdsCard** : Informations tag
- **OdsDatagrid** : Liste ressources taggées
- **OdsButton** : Modifier, Supprimer

## 3. CARD INFORMATIONS
| Champ | Type |
|-------|------|
| Clé | Texte |
| Valeur | Texte |
| Description | Texte |
| Couleur | Color picker |
| Date création | Datetime |

## 4. TABLEAU RESSOURCES
| Colonne | Type |
|---------|------|
| Ressource | Lien |
| Type | Badge |
| Assigné le | Date |
| Actions | Retirer tag |

## 5. APIS & ENDPOINTS
- GET /iam/tags/{tagId} - Détail tag
- GET /iam/tags/{tagId}/resources - Ressources taggées
- DELETE /iam/tags/{tagId}/resources/{resourceId} - Retirer tag
