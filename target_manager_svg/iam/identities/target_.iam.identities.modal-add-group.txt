# Cahier des Charges : iam/identities - Modal création groupe

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Onglet Groupes
- **Objectif** : Créer un nouveau groupe d'utilisateurs

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 450px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom du groupe
- **OdsTextarea** : Description
- **OdsSelect** : Membres
- **OdsButton** : Annuler, Créer

## 4. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom du groupe | Input | Oui * |
| Description | Textarea | Non |
| Membres | Multi-select | Non |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée groupe |

## 6. APIS & ENDPOINTS
- POST /iam/groups - Créer groupe
- GET /iam/users - Liste utilisateurs disponibles
