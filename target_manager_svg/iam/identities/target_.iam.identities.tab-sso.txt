# Cahier des Charges : iam/identities - Onglet SSO

## 1. VUE D'ENSEMBLE
- **Type** : Onglet configuration SSO
- **URL** : /iam/identities?tab=sso
- **Objectif** : Configurer l'authentification SSO (SAML, OIDC)

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cards configuration
- **OdsToggle** : Activer/Désactiver SSO
- **OdsDatagrid** : Liste des IdP configurés

## 3. SECTION SSO STATUS
| Élément | Description |
|---------|-------------|
| Toggle | Activer/Désactiver SSO globalement |
| État | Actif/Inactif |

## 4. COLONNES TABLEAU IdP
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Texte | 200px |
| Type | Badge (SAML/OIDC) | 100px |
| Entity ID | Code | 300px |
| Statut | Badge | 100px |
| Actions | Menu ⋮ | 50px |

## 5. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Éditer configuration |
| Tester | Tester la connexion |
| Désactiver | Désactiver l'IdP |
| Supprimer | Supprimer l'IdP |

## 6. APIS & ENDPOINTS
- GET /iam/sso - Configuration SSO
- POST /iam/sso/idp - Ajouter IdP
- PUT /iam/sso/idp/{idpId} - Modifier IdP
- DELETE /iam/sso/idp/{idpId} - Supprimer IdP
