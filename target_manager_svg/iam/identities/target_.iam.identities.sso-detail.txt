# Cahier des Charges : iam/identities - Détail SSO IdP

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/sso/{idpId}
- **Objectif** : Afficher la configuration détaillée d'un IdP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Identités / SSO / {nom}
- **OdsBadge** : Type (SAML/OIDC), Statut
- **OdsCard** : Cards configuration
- **OdsButton** : Modifier, Tester, Supprimer

## 4. HEADER PAGE
| Élément | Description |
|---------|-------------|
| Titre | Nom de l'IdP |
| Badges | Type + Statut |
| Actions | Modifier, Tester connexion, Supprimer |

## 5. CARD CONFIGURATION
### SAML
| Champ | Description |
|-------|-------------|
| Entity ID | Identifiant SAML |
| SSO URL | URL connexion |
| SLO URL | URL déconnexion |
| Certificat | Validité et empreinte |

### OIDC
| Champ | Description |
|-------|-------------|
| Issuer URL | URL émetteur |
| Client ID | ID application |
| Scopes | Permissions demandées |
| Authorization URL | URL autorisation |
| Token URL | URL token |

## 6. CARD ENDPOINTS OVHcloud
| Champ | Valeur |
|-------|--------|
| ACS URL | https://ovhcloud.com/auth/saml/acs |
| Entity ID | https://ovhcloud.com/auth/saml/sp |
| Callback URL (OIDC) | https://ovhcloud.com/auth/oidc/callback |

## 7. CARD UTILISATEURS SSO
| Colonne | Type |
|---------|------|
| Utilisateur | Nom + Email |
| Dernière connexion | Date |
| Actions | Menu |

## 8. APIS & ENDPOINTS
- GET /iam/sso/idp/{idpId} - Détail IdP
- GET /iam/sso/idp/{idpId}/users - Utilisateurs SSO
- POST /iam/sso/idp/{idpId}/test - Tester connexion
