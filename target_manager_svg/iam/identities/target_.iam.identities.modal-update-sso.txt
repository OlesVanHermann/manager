# Cahier des Charges : iam/identities - Modal modification SSO IdP

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Action menu IdP
- **Objectif** : Modifier la configuration d'un IdP SSO

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 700px |
| Modal hauteur | 600px |

## 3. COMPOSANTS UI
- Identique à modal ajout SSO
- Champs pré-remplis

## 4. CHAMPS NON MODIFIABLES
| Champ | Raison |
|-------|--------|
| Type | Définit la structure |
| Entity ID (SAML) | Identifiant unique |
| Issuer URL (OIDC) | Identifiant unique |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde modifications |

## 6. APIS & ENDPOINTS
- GET /iam/sso/idp/{idpId} - Récupérer configuration
- PUT /iam/sso/idp/{idpId} - Modifier configuration
