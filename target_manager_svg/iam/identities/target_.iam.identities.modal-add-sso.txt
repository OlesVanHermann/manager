# Cahier des Charges : iam/identities - Modal ajout SSO IdP

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création
- **Contexte** : Onglet SSO
- **Objectif** : Configurer un nouveau fournisseur d'identité SSO

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 700px |
| Modal hauteur | 600px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsSelect** : Type (SAML/OIDC)
- **OdsInput** : Champs configuration
- **OdsTextarea** : Certificat/Metadata
- **OdsButton** : Annuler, Configurer

## 4. CHAMPS COMMUNS
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |
| Type | Select (SAML/OIDC) | Oui * |

## 5. CHAMPS SAML
| Champ | Type | Requis |
|-------|------|--------|
| Entity ID | Input | Oui * |
| SSO URL | Input URL | Oui * |
| Certificat X.509 | Textarea | Oui * |
| OU | Metadata XML | Textarea | Alternative |

## 6. CHAMPS OIDC
| Champ | Type | Requis |
|-------|------|--------|
| Issuer URL | Input URL | Oui * |
| Client ID | Input | Oui * |
| Client Secret | Input password | Oui * |
| Scopes | Input | Oui (défaut: openid profile email) |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Configurer | Primary | Configure IdP |

## 8. APIS & ENDPOINTS
- POST /iam/sso/idp - Créer configuration IdP
- POST /iam/sso/idp/validate - Valider configuration
