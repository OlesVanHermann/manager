# Cahier des Charges : iam/identities - Modal désactivation utilisateur

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu utilisateur actif
- **Objectif** : Désactiver un utilisateur

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 250px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Désactiver

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Désactiver l'utilisateur" |
| Warning | "L'utilisateur {nom} ne pourra plus se connecter." |
| Info | "Vous pourrez le réactiver ultérieurement." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Désactiver | Warning | Désactive utilisateur |

## 6. APIS & ENDPOINTS
- POST /iam/users/{userId}/disable - Désactiver utilisateur
