# Cahier des Charges : iam/api-keys - Modal modification clé API

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition
- **Contexte** : Action menu ou page détail
- **Objectif** : Modifier les informations d'une clé API

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 400px |
| Position | Centrée |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom de la clé
- **OdsTextarea** : Description
- **OdsButton** : Annuler, Enregistrer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Modifier la clé API" |
| Bouton fermer | X en haut droite |

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Requis | Valeur initiale |
|-------|------|--------|-----------------|
| Nom de la clé API | Input texte | Oui * | {nom actuel} |
| Description | Textarea | Non | {description actuelle} |

## 6. INFORMATIONS NON MODIFIABLES
| Champ | Affichage |
|-------|-----------|
| Client ID | Lecture seule (code) |
| Date de création | Lecture seule |
| Date d'expiration | Lecture seule |

## 7. FOOTER MODAL
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde modifications |

## 8. VALIDATION
| Champ | Règle |
|-------|-------|
| Nom | Requis, 3-64 caractères |
| Description | Max 256 caractères |

## 9. APIS & ENDPOINTS
- GET /iam/api-keys/{keyId} - Récupérer données actuelles
- PUT /iam/api-keys/{keyId} - Modifier clé API
