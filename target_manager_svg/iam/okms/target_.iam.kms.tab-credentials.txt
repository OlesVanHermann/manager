# Cahier des Charges : iam/kms - Onglet Credentials

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau
- **URL** : /iam/kms/{kmsId}?tab=credentials
- **Objectif** : Gérer les identifiants d'accès au KMS

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : + Créer un credential
- **OdsDatagrid** : Tableau credentials
- **OdsBadge** : Statut

## 3. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien | 200px |
| Access Key ID | Code | 250px |
| Description | Texte | 200px |
| Statut | Badge | 100px |
| Créé le | Date | 120px |
| Actions | Menu ⋮ | 50px |

## 4. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir détails | Détail du credential |
| Supprimer | Supprimer credential |

## 5. APIS & ENDPOINTS
- GET /kms/{kmsId}/credentials - Liste credentials
- POST /kms/{kmsId}/credentials - Créer credential
- DELETE /kms/{kmsId}/credentials/{credId} - Supprimer
