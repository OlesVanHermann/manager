# Cahier des Charges : iam/kms - Création clé de service

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire
- **URL** : /iam/kms/{kmsId}/servicekeys/create
- **Objectif** : Créer une nouvelle clé cryptographique

## 2. COMPOSANTS UI
- **OdsCard** : Container formulaire
- **OdsInput** : Nom
- **OdsSelect** : Type, Algorithme, Taille
- **OdsButton** : Annuler, Créer

## 3. CHAMPS FORMULAIRE
| Champ | Type | Options |
|-------|------|---------|
| Nom | Input | Requis |
| Type | Select | AES, RSA, EC |
| Algorithme | Select | Dépend du type |
| Taille | Select | Dépend du type |

## 4. OPTIONS PAR TYPE
### AES
- Tailles: 128, 192, 256 bits

### RSA
- Tailles: 2048, 3072, 4096 bits
- Algorithmes: RSA-OAEP, RSA-PSS

### EC
- Courbes: P-256, P-384, P-521

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Créer | Primary | Crée la clé |

## 6. APIS & ENDPOINTS
- POST /kms/{kmsId}/serviceKeys - Créer clé
