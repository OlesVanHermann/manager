# Cahier des Charges : iam/kms - Création credential

## 1. VUE D'ENSEMBLE
- **Type** : Page formulaire
- **URL** : /iam/kms/{kmsId}/credentials/create
- **Objectif** : Créer un nouveau credential d'accès

## 2. COMPOSANTS UI
- **OdsCard** : Container formulaire
- **OdsInput** : Nom, Description
- **OdsMessage** : Info sécurité
- **OdsButton** : Annuler, Créer

## 3. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |
| Description | Textarea | Non |

## 4. MESSAGE INFO
"Le Secret Access Key ne sera affiché qu'une seule fois après la création. Conservez-le précieusement."

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Retour liste |
| Créer | Primary | Crée credential |

## 6. POST-CRÉATION
Modal affichant:
- Access Key ID
- Secret Access Key (à copier impérativement)

## 7. APIS & ENDPOINTS
- POST /kms/{kmsId}/credentials - Créer credential
