# Cahier des Charges : iam/kms - Modal modification nom

## 1. VUE D'ENSEMBLE
- **Type** : Modal d'édition simple
- **Contexte** : Action sur un service KMS
- **Objectif** : Renommer le service KMS

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 250px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nouveau nom
- **OdsButton** : Annuler, Enregistrer

## 4. CHAMP
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Enregistrer | Primary | Sauvegarde |

## 6. APIS & ENDPOINTS
- PUT /kms/{kmsId} - Modifier KMS
