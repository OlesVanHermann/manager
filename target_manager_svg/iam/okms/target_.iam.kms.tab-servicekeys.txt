# Cahier des Charges : iam/kms - Onglet Clés de service

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau
- **URL** : /iam/kms/{kmsId}?tab=servicekeys
- **Objectif** : Gérer les clés cryptographiques

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : + Créer une clé
- **OdsDatagrid** : Tableau clés
- **OdsBadge** : Statut, Type

## 3. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien | 200px |
| ID | Code | 200px |
| Type | Badge | 100px |
| Algorithme | Texte | 150px |
| Statut | Badge | 100px |
| Créé le | Date | 120px |
| Actions | Menu ⋮ | 50px |

## 4. TYPES DE CLÉS
| Type | Description |
|------|-------------|
| AES | Chiffrement symétrique |
| RSA | Chiffrement asymétrique |
| EC | Courbes elliptiques |

## 5. STATUTS
| Badge | Description |
|-------|-------------|
| Active | Clé utilisable |
| Désactivée | Clé inactive temporaire |
| En attente suppression | Suppression programmée |

## 6. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir détails | Détail de la clé |
| Désactiver | Désactive la clé |
| Supprimer | Programme suppression |

## 7. APIS & ENDPOINTS
- GET /kms/{kmsId}/serviceKeys - Liste clés
- POST /kms/{kmsId}/serviceKeys - Créer clé
- DELETE /kms/{kmsId}/serviceKeys/{keyId} - Supprimer clé
