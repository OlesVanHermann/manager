# Cahier des Charges : iam/kms - Détail clé de service

## 1. VUE D'ENSEMBLE
- **Type** : Page détail
- **URL** : /iam/kms/{kmsId}/servicekeys/{keyId}
- **Objectif** : Afficher les informations d'une clé

## 2. COMPOSANTS UI
- **OdsBreadcrumb** : Accueil / IAM / KMS / {kms} / {clé}
- **OdsBadge** : Type, Statut
- **OdsCard** : Informations clé
- **OdsButton** : Désactiver, Supprimer

## 3. CARD INFORMATIONS
| Champ | Type |
|-------|------|
| Nom | Texte |
| ID | Code copiable |
| Type | Badge |
| Algorithme | Texte |
| Taille | Texte (bits) |
| Statut | Badge |
| Date création | Datetime |

## 4. ACTIONS
| Bouton | Style | Description |
|--------|-------|-------------|
| Désactiver/Activer | Warning | Toggle statut |
| Supprimer | Danger | Programme suppression |

## 5. APIS & ENDPOINTS
- GET /kms/{kmsId}/serviceKeys/{keyId} - Détail clé
- POST /kms/{kmsId}/serviceKeys/{keyId}/deactivate - Désactiver
- POST /kms/{kmsId}/serviceKeys/{keyId}/activate - Activer
- DELETE /kms/{kmsId}/serviceKeys/{keyId} - Supprimer
