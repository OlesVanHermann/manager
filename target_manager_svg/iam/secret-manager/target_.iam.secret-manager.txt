# Cahier des Charges : iam/secret-manager - Liste des secrets

## 1. VUE D'ENSEMBLE
- **Type** : Page liste avec tableau
- **URL** : /iam/secret-manager
- **Objectif** : Gérer les secrets (clés API, mots de passe, certificats)

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1440px |
| Hauteur | 900px |

## 3. COMPOSANTS UI
- **OdsHeader** : En-tête OVHcloud
- **OdsSidebar** : Navigation
- **OdsBreadcrumb** : Accueil / IAM / Secret Manager
- **OdsButton** : + Créer un secret
- **OdsDatagrid** : Tableau secrets
- **OdsBadge** : Type, Versions

## 4. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Nom | Lien | 250px |
| Type | Badge | 120px |
| Description | Texte | 250px |
| Versions | Badge compteur | 100px |
| Dernière MAJ | Date | 150px |
| Actions | Menu ⋮ | 50px |

## 5. TYPES DE SECRETS
| Type | Badge couleur |
|------|---------------|
| API Key | Bleu |
| Password | Vert |
| Certificate | Orange |
| Generic | Gris |

## 6. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir détails | Page détail |
| Nouvelle version | Créer version |
| Supprimer | Supprimer secret |

## 7. APIS & ENDPOINTS
- GET /secret-manager/secrets - Liste secrets
- POST /secret-manager/secrets - Créer secret
- DELETE /secret-manager/secrets/{secretId} - Supprimer
