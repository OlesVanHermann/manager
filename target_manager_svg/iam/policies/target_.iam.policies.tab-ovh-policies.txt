# Cahier des Charges : iam/policies - Onglet Politiques OVH

## 1. VUE D'ENSEMBLE
- **Type** : Onglet dans page politiques
- **URL** : /iam/policies?tab=ovh-policies
- **Objectif** : Afficher les politiques système OVHcloud

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsDatagrid** : Tableau politiques
- **OdsBadge** : Type Système

## 3. ONGLETS
| Onglet | État | Filtre |
|--------|------|--------|
| Mes politiques | Inactif | type=custom |
| Politiques OVH | Actif | type=system |
| Groupes de ressources | Inactif | - |

## 4. CONTENU
- Affiche uniquement les politiques avec type=Système
- Actions limitées (voir, dupliquer - pas modifier/supprimer)

## 5. NOTE
Les politiques système sont prédéfinies par OVHcloud et ne peuvent pas être modifiées.

## 6. APIS & ENDPOINTS
- GET /iam/policies?type=system - Politiques système
