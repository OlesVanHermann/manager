# Cahier des Charges : iam/policies - Modal ajout compte

## 1. VUE D'ENSEMBLE
- **Type** : Modal de sélection
- **Contexte** : Page détail/création politique
- **Objectif** : Ajouter un compte externe à la politique

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 550px |
| Modal hauteur | 400px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Saisie NIC handle
- **OdsButton** : Annuler, Ajouter

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Ajouter un compte" |
| Bouton fermer | X |

## 5. DESCRIPTION
"Ajoutez un compte OVHcloud externe pour lui donner accès à vos ressources via cette politique."

## 6. CHAMPS
| Champ | Type | Description |
|-------|------|-------------|
| NIC Handle | Input | Format xx00000-ovh |
| Validation | Pattern | ^[a-z]{2}\d+-[a-z]+$ |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Ajouter | Primary | Ajoute le compte |

## 8. APIS & ENDPOINTS
- POST /iam/policies/{policyId}/accounts - Ajouter compte externe
- GET /me/identity/group - Vérifier compte
