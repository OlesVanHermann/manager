# Cahier des Charges : iam/policies - Modal création rapide

## 1. VUE D'ENSEMBLE
- **Type** : Modal de création rapide
- **Contexte** : Alternative à la page complète de création
- **Objectif** : Créer rapidement une politique simple

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 650px |
| Modal hauteur | 550px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Nom
- **OdsTextarea** : Description
- **OdsSelect** : Template prédéfini
- **OdsButton** : Annuler, Créer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Créer une politique" |
| Bouton fermer | X |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Requis |
|-------|------|--------|
| Nom | Input | Oui * |
| Description | Textarea | Non |
| Template | Select | Non |

## 6. TEMPLATES PRÉDÉFINIS
| Template | Description |
|----------|-------------|
| Lecture seule | Permissions de lecture uniquement |
| Administrateur | Toutes les permissions |
| Support | Lecture + création tickets |
| Billing | Accès facturation |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Créer | Primary | Crée la politique |
| Lien | "Créer une politique avancée →" | Redirige vers page complète |

## 8. APIS & ENDPOINTS
- POST /iam/policies - Créer politique
- GET /iam/policies/templates - Liste templates
