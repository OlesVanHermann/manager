# Cahier des Charges : iam/policies - Modal recherche avancée

## 1. VUE D'ENSEMBLE
- **Type** : Modal de filtrage avancé
- **Contexte** : Page liste politiques
- **Objectif** : Filtrer les politiques avec critères multiples

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 600px |
| Modal hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsInput** : Champ recherche texte
- **OdsSelect** : Filtres sélection
- **OdsCheckbox** : Options filtrage
- **OdsButton** : Réinitialiser, Appliquer

## 4. HEADER MODAL
| Élément | Description |
|---------|-------------|
| Titre | "Recherche avancée" |
| Bouton fermer | X |

## 5. FILTRES DISPONIBLES
| Filtre | Type | Options |
|--------|------|---------|
| Nom contient | Input | Texte libre |
| Type | Select | Système, Personnalisé, Tous |
| Type de produit | Select | Liste produits OVHcloud |
| Créé après | Date picker | Date |
| Créé avant | Date picker | Date |
| Avec identités | Checkbox | Oui/Non |

## 6. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Réinitialiser | Secondary | Efface filtres |
| Appliquer | Primary | Applique filtres |

## 7. APIS & ENDPOINTS
- GET /iam/policies?name={name}&type={type}&productType={type}&createdAfter={date}&createdBefore={date}
