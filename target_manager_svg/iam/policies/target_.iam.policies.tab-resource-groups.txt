# Cahier des Charges : iam/policies - Onglet Groupes de ressources

## 1. VUE D'ENSEMBLE
- **Type** : Onglet dans page politiques
- **URL** : /iam/policies?tab=resource-groups
- **Objectif** : Gérer les groupes de ressources

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : + Créer un groupe
- **OdsDatagrid** : Tableau groupes
- **OdsSearchField** : Recherche

## 3. ONGLETS
| Onglet | État |
|--------|------|
| Mes politiques | Inactif |
| Politiques OVH | Inactif |
| Groupes de ressources | Actif |

## 4. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom | Lien | Nom du groupe |
| Description | Texte | Description |
| Ressources | Badge compteur | Nombre de ressources |
| Politiques | Badge compteur | Politiques utilisant ce groupe |
| Créé le | Date | Date création |
| Actions | Menu ⋮ | - |

## 5. ACTION MENU
| Action | Description |
|--------|-------------|
| Modifier | Éditer le groupe |
| Supprimer | Supprimer le groupe |

## 6. APIS & ENDPOINTS
- GET /iam/resource-groups - Liste groupes de ressources
- POST /iam/resource-groups - Créer groupe
- PUT /iam/resource-groups/{groupId} - Modifier groupe
- DELETE /iam/resource-groups/{groupId} - Supprimer groupe
