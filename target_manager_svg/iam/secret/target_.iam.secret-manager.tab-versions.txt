# Cahier des Charges : iam/secret-manager - Onglet Versions

## 1. VUE D'ENSEMBLE
- **Type** : Onglet avec tableau
- **URL** : /iam/secret-manager/{secretId}?tab=versions
- **Objectif** : Gérer les versions d'un secret

## 2. COMPOSANTS UI
- **OdsTabs** : Navigation onglets
- **OdsButton** : + Nouvelle version
- **OdsDatagrid** : Tableau versions
- **OdsBadge** : Statut

## 3. COLONNES TABLEAU
| Colonne | Type | Largeur |
|---------|------|---------|
| Version | Numéro | 100px |
| Statut | Badge | 100px |
| Créé le | Datetime | 200px |
| Actions | Menu ⋮ | 50px |

## 4. STATUTS
| Statut | Description |
|--------|-------------|
| Active | Version courante |
| Archivée | Version précédente |
| En suppression | Suppression programmée |

## 5. ACTION MENU (⋮)
| Action | Description |
|--------|-------------|
| Voir la valeur | Affiche drawer valeur |
| Définir active | Active cette version |
| Archiver | Archive la version |

## 6. APIS & ENDPOINTS
- GET /secret-manager/secrets/{secretId}/versions - Liste versions
- POST /secret-manager/secrets/{secretId}/versions - Créer version
- PUT /secret-manager/secrets/{secretId}/versions/{versionId} - Modifier statut
