# Cahier des Charges : iam/secret-manager - Modal suppression secret

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation avec saisie
- **Contexte** : Action sur un secret
- **Objectif** : Supprimer définitivement un secret

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 500px |
| Modal hauteur | 350px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning danger
- **OdsInput** : Confirmation
- **OdsButton** : Annuler, Supprimer

## 4. WARNING BOX
| Type | Danger (rouge) |
| Messages |
- "Cette action est irréversible."
- "Le secret et toutes ses versions seront supprimés." |

## 5. INFORMATIONS
| Élément | Description |
|---------|-------------|
| Label | "Secret à supprimer :" |
| Nom | {nom du secret} (gras) |
| Versions | "X versions seront supprimées" |

## 6. CHAMP CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Label | "Tapez DELETE pour confirmer" |
| Validation | Doit être "DELETE" |

## 7. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime si DELETE saisi |

## 8. APIS & ENDPOINTS
- DELETE /secret-manager/secrets/{secretId}
