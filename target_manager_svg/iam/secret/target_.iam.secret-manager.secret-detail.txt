# Cahier des Charges : iam/secret-manager - Détail secret

## 1. VUE D'ENSEMBLE
- **Type** : Page détail avec onglets
- **URL** : /iam/secret-manager/{secretId}
- **Objectif** : Afficher et gérer un secret

## 2. COMPOSANTS UI
- **OdsBreadcrumb** : Accueil / IAM / Secret Manager / {nom}
- **OdsBadge** : Type
- **OdsTabs** : Informations / Versions
- **OdsCard** : Informations
- **OdsButton** : Nouvelle version, Supprimer

## 3. CARD INFORMATIONS
| Champ | Type |
|-------|------|
| Nom | Texte |
| Type | Badge |
| Description | Texte |
| ID | Code copiable |
| Date création | Datetime |
| Dernière MAJ | Datetime |

## 4. ONGLET VERSIONS
Liste des versions du secret avec:
- Numéro de version
- Date création
- Statut (Active/Archivée)
- Actions (Voir valeur, Archiver)

## 5. APIS & ENDPOINTS
- GET /secret-manager/secrets/{secretId} - Détail
- POST /secret-manager/secrets/{secretId}/versions - Nouvelle version
- DELETE /secret-manager/secrets/{secretId} - Supprimer
