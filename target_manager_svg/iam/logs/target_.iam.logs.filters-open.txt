# Cahier des Charges : iam/logs - Panneau filtres ouvert

## 1. VUE D'ENSEMBLE
- **Type** : Panneau de filtres expandable
- **Contexte** : Page logs d'audit
- **Objectif** : Filtrer les logs avec critères multiples

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 100% de la card |
| Hauteur | ~200px (expandable) |

## 3. COMPOSANTS UI
- **OdsCollapsible** : Container filtres
- **OdsDateRange** : Période
- **OdsSelect** : Filtres sélection
- **OdsInput** : Recherche texte
- **OdsButton** : Appliquer, Réinitialiser

## 4. FILTRES DISPONIBLES
| Filtre | Type | Options |
|--------|------|---------|
| Période | DateRange | Prédéfini ou personnalisé |
| Type d'action | Multi-select | Connexion, Création, Modification, Suppression, Consultation |
| Utilisateur | Autocomplete | Recherche par nom/email |
| Ressource | Select | Type de ressource |
| Statut | Select | Succès, Échec |
| IP Source | Input | Adresse IP |

## 5. PÉRIODES PRÉDÉFINIES
| Option | Valeur |
|--------|--------|
| Dernières 24 heures | 24h |
| 7 derniers jours | 7d |
| 30 derniers jours | 30d |
| Personnalisé | Date picker |

## 6. ACTIONS
| Bouton | Style | Action |
|--------|-------|--------|
| Réinitialiser | Secondary | Efface filtres |
| Appliquer | Primary | Applique filtres |

## 7. APIS & ENDPOINTS
- GET /iam/logs?dateFrom={date}&dateTo={date}&action={type}&user={id}&status={status}
