# Cahier des Charges : iam/logs - Composant Datagrid logs

## 1. VUE D'ENSEMBLE
- **Type** : Composant tableau
- **Contexte** : Page logs d'audit
- **Objectif** : Afficher les logs de manière structurée

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1120px |
| Hauteur ligne | 48px |

## 3. COLONNES
| Colonne | Type | Sortable |
|---------|------|----------|
| Date/Heure | Datetime | Oui (défaut desc) |
| Action | Badge | Oui |
| Utilisateur | Lien | Oui |
| Ressource | Texte | Non |
| IP Source | Code | Non |
| Statut | Badge | Oui |
| Détails | Icône œil | Non |

## 4. STYLE LIGNE
| État | Background |
|------|------------|
| Normal | Blanc |
| Hover | #F3F4F6 |
| Échec | #FEF2F2 (fond rouge léger) |

## 5. PAGINATION
| Propriété | Valeur |
|-----------|--------|
| Taille page | 25, 50, 100 |
| Défaut | 50 |

## 6. EXPORT
| Format | Description |
|--------|-------------|
| CSV | Export filtres actuels |
| JSON | Export brut |
