# Cahier des Charges : iam/conditions - Modal suppression

## 1. VUE D'ENSEMBLE
- **Type** : Modal de confirmation
- **Contexte** : Action menu condition
- **Objectif** : Supprimer une condition

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Modal largeur | 450px |
| Modal hauteur | 280px |

## 3. COMPOSANTS UI
- **OdsModal** : Container principal
- **OdsMessage** : Warning
- **OdsButton** : Annuler, Supprimer

## 4. CONTENU
| Élément | Description |
|---------|-------------|
| Titre | "Supprimer la condition" |
| Question | "Êtes-vous sûr de vouloir supprimer la condition {nom} ?" |
| Warning | "Les politiques utilisant cette condition ne seront plus restreintes." |

## 5. FOOTER
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Secondary | Ferme modal |
| Supprimer | Danger | Supprime condition |

## 6. APIS & ENDPOINTS
- DELETE /iam/conditions/{condId}
