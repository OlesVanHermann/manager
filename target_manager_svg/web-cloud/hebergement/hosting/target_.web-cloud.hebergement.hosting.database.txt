# Cahier des Charges : web-cloud/hosting - Bases de données

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des bases de données
- **Objectif** : Créer et gérer les BDD MySQL
- **Contexte** : Stockage données applications

## 2. LAYOUT & STRUCTURE
- Titre : "Bases de données"
- 2 boutons action
- Tableau des bases
- Info quota en bas

## 3. COMPOSANTS UI
- **OdsButton** : "Créer une base de données" (primaire)
- **OdsButton** : "Commander une base de données" (secondaire)
- **OdsTable** : Liste bases
- **OdsBadge** : État (Actif)
- **OdsProgressBar** : Taille utilisée
- **OdsActionMenu** : Menu ⋮

## 4. BOUTONS ACTION
- "Créer une base de données" (bleu primaire)
- "Commander une base de données" (bordure grise)

## 5. TABLEAU BASES DE DONNÉES
| Colonne | Description |
|---------|-------------|
| Nom de la base | Identifiant BDD |
| Nom d'utilisateur | Login connexion |
| Adresse du serveur | Hostname MySQL |
| Version | MySQL 8.0, etc. |
| Taille | Usage avec barre progression |
| État | Badge Actif |
| Actions | Menu ⋮ |

## 6. EXEMPLES DONNÉES
| Base | Utilisateur | Serveur | Version | Taille |
|------|------------|---------|---------|--------|
| mywebsitec_db1 | mywebsitec | mywebsitec.mysql.db | MySQL 8.0 | 156 Mo / 400 Mo |
| mywebsitec_db2 | mywebsitec | mywebsitec.mysql.db | MySQL 8.0 | 340 Mo / 400 Mo (warning) |

## 7. MENU ACTIONS (⋮)
- Changer le mot de passe
- Créer une sauvegarde
- Restaurer une sauvegarde
- Importer un fichier
- Accéder à phpMyAdmin
- Supprimer la base (rouge)

## 8. INFO QUOTA
- "2 bases de données sur 5 disponibles"

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/database : Liste BDD
- POST /hosting/web/{serviceName}/database : Créer BDD
- DELETE /hosting/web/{serviceName}/database/{name} : Supprimer
- POST /hosting/web/{serviceName}/database/{name}/changePassword : Changer MDP
- POST /hosting/web/{serviceName}/database/{name}/dump : Créer sauvegarde
- POST /hosting/web/{serviceName}/database/{name}/import : Importer
- POST /hosting/web/{serviceName}/database/{name}/restore : Restaurer
