# Cahier des Charges : web-cloud/hosting - FTP - SSH

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion accès FTP/SSH
- **Objectif** : Gérer utilisateurs et accès
- **Contexte** : Transfert fichiers et accès console

## 2. LAYOUT & STRUCTURE
- Titre : "FTP - SSH"
- Tuile infos connexion
- 3 boutons action
- Tableau utilisateurs

## 3. COMPOSANTS UI
- **OdsCard** : Infos FTP
- **OdsButton** : "Créer un utilisateur" (primaire)
- **OdsButton** : "Restaurer une sauvegarde" (secondaire)
- **OdsButton** : "Générer une sauvegarde" (secondaire)
- **OdsTable** : Liste utilisateurs
- **OdsBadge** : FTP, SFTP, SSH, État (Activé/Désactivé)
- **OdsActionMenu** : Menu ⋮

## 4. TUILE INFORMATIONS FTP
| Champ | Valeur |
|-------|--------|
| Serveur FTP et SFTP | ftp.cluster027.hosting.ovh.net |
| Serveur SSH | ssh.cluster027.hosting.ovh.net |
| Login principal | mywebsitec |
| Port FTP | 21 |
| Port SFTP | 22 |
| Port SSH | 22 |

## 5. BOUTONS ACTION
- "Créer un utilisateur" (bleu primaire)
- "Restaurer une sauvegarde" (bordure grise)
- "Générer une sauvegarde" (bordure grise)

## 6. TABLEAU UTILISATEURS
| Colonne | Description |
|---------|-------------|
| Login | Nom utilisateur |
| Répertoire cible | Chemin home |
| FTP | Badge Activé/Désactivé |
| SFTP | Badge Activé/Désactivé |
| SSH | Badge Activé/Désactivé |
| État | Badge Activé/Désactivé |
| Actions | Menu ⋮ |

## 7. EXEMPLES DONNÉES
| Login | Répertoire | FTP | SFTP | SSH | État |
|-------|-----------|-----|------|-----|------|
| mywebsitec | /home/mywebsitec | Activé | Activé | Activé | Activé |
| mywebsitec-dev | /home/mywebsitec/dev | Activé | Activé | Désactivé | Activé |

## 8. MENU ACTIONS (⋮)
- Changer le mot de passe
- Modifier
- Supprimer (rouge)

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/user : Liste utilisateurs
- POST /hosting/web/{serviceName}/user : Créer utilisateur
- PUT /hosting/web/{serviceName}/user/{login} : Modifier
- DELETE /hosting/web/{serviceName}/user/{login} : Supprimer
- POST /hosting/web/{serviceName}/user/{login}/changePassword : Changer MDP
- GET /hosting/web/{serviceName}/dump : Liste sauvegardes
- POST /hosting/web/{serviceName}/dump : Créer sauvegarde
- POST /hosting/web/{serviceName}/restoreSnapshot : Restaurer
