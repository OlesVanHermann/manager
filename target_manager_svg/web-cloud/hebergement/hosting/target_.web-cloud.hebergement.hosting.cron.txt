# Cahier des Charges : web-cloud/hosting - Tâches planifiées (Cron)

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des tâches planifiées
- **Objectif** : Configurer les cron jobs
- **Contexte** : Automatisation scripts

## 2. LAYOUT & STRUCTURE
- Titre : "Tâches planifiées - Cron"
- Bouton ajouter
- Tableau des crons
- Lien aide en bas

## 3. COMPOSANTS UI
- **OdsButton** : "Ajouter une planification" (primaire)
- **OdsTable** : Liste des crons
- **OdsBadge** : État (Activé/Désactivé)
- **OdsActionMenu** : Menu ⋮ (Modifier, Supprimer)

## 4. TABLEAU CRON JOBS
| Colonne | Description |
|---------|-------------|
| Commande | Script à exécuter (monospace) |
| Langage | PHP 8.2, Autre |
| Fréquence | Format cron (ex: 0 2 * * *) |
| E-mail | Adresse notification ou "Non" |
| État | Badge Activé/Désactivé |
| Description | Note courte |
| Actions | Menu ⋮ |

## 5. EXEMPLES DONNÉES
| Commande | Langage | Fréquence | E-mail | État | Description |
|----------|---------|-----------|--------|------|-------------|
| cron/backup.php | PHP 8.2 | 0 2 * * * | admin@mywebsite.com | Activé | Backup quotidien |
| scripts/cleanup.php | PHP 8.2 | */15 * * * * | Non | Activé | Nettoyage cache |
| cron/newsletter.php | PHP 8.2 | 0 8 * * 1 | marketing@mywebsite.com | Désactivé | Newsletter hebdo |
| bin/sitemap.sh | Autre | 0 4 * * * | Non | Activé | Génération sitemap |

## 6. MENU ACTIONS (⋮)
- Modifier
- Supprimer (rouge)

## 7. LIEN AIDE
- "Besoin d'aide pour configurer vos tâches planifiées ?"
- "Consultez nos guides en ligne."

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cron : Liste crons
- POST /hosting/web/{serviceName}/cron : Créer cron
- PUT /hosting/web/{serviceName}/cron/{id} : Modifier
- DELETE /hosting/web/{serviceName}/cron/{id} : Supprimer
