# Cahier des Charges : web-cloud/hebergement - Tab Boost

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion du boost temporaire de l'hébergement.

### Objectif fonctionnel
- Activer/désactiver le boost
- Changer le niveau de boost
- Consulter l'historique

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/boostHistory
POST /hosting/web/{serviceName}/boost
DELETE /hosting/web/{serviceName}/boost
```

### Modele
```typescript
interface HostingBoost {
  currentOffer: string;
  boostedOffer?: string;
  isActive: boolean;
  pricePerDay: number;
}

interface BoostOption {
  offer: string;
  vcpu: number;
  ram: number;
  pricePerDay: number;
}

interface BoostHistory {
  serviceName: string;
  originOffer: string;
  boostedOffer: string;
  date: string;
}
```
