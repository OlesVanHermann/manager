# Cahier des Charges : web-cloud/hebergement - Tab Visibilité Pro

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion du référencement local (Local SEO).

### Objectif fonctionnel
- Gérer les fiches entreprise
- Accéder à l'interface de gestion
- Commander de nouveaux abonnements

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/localSeo/location
POST /hosting/web/{serviceName}/localSeo/order
DELETE /hosting/web/{serviceName}/localSeo/location/{id}
```

### Modele
```typescript
interface LocalSeoLocation {
  id: string;
  name: string;
  email: string;
  address?: string;
  status: 'ok' | 'creating' | 'error';
  interfaceUrl: string;
}
```
