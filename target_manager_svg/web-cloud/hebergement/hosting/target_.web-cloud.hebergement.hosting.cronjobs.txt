# Cahier des Charges : web-cloud/hosting - Tâches planifiées (CRON)

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des tâches CRON
- **Objectif** : Gérer les tâches planifiées de l'hébergement
- **Contexte** : Automatisation des scripts

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud › Hébergements › hosting123 › Tâches planifiées (CRON)
- Titre : "Tâches planifiées (CRON)"
- Lien guide externe
- Bouton ajouter (droite)
- Tableau des tâches CRON

## 3. COMPOSANTS UI
- **OdsLink** : Lien guide externe "Configurer une tâche CRON ↗"
- **OdsButton** : "Ajouter une tâche CRON" (secondaire bordure bleue)
- **OdsTable** : Liste des CRON
- **OdsBadge** : Statut (Activé vert, Désactivé rouge, Suspendu jaune)
- **OdsActionMenu** : Menu ⋮ avec options

## 4. TABLEAU CRON
| Colonne | Description |
|---------|-------------|
| Commande | Chemin du script |
| Description | Texte descriptif |
| Fréquence | Expression cron (monospace) |
| Langage | PHP 8.2, Node 18, etc. |
| Statut | Badge Activé/Désactivé/Suspendu |
| E-mail | Destinataire logs (Admin, Tech, Autre, Non) |
| Actions | Menu ⋮ |

## 5. EXEMPLES DONNÉES
| Commande | Description | Fréquence | Langage | Statut | E-mail |
|----------|-------------|-----------|---------|--------|--------|
| ./www/cron/backup.php | Sauvegarde quotidienne | 0 2 * * * | PHP 8.2 | Activé | Admin |
| ./www/cron/newsletter.php | Envoi newsletter | 0 8 * * 1 | PHP 8.2 | Activé | Autre |
| ./www/cron/cleanup.php | Nettoyage cache | */30 * * * * | PHP 8.1 | Désactivé | Non |
| ./www/cron/sync.sh | Synchronisation externe | 0 */4 * * * | Node 18 | Suspendu | Tech |
| ./www/cron/sitemap.php | Génération sitemap | 0 3 * * 0 | PHP 8.2 | Activé | Admin |

## 6. MENU ACTIONS (⋮)
- Modifier
- Supprimer

## 7. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cron : Liste des CRON
- POST /hosting/web/{serviceName}/cron : Créer CRON
- PUT /hosting/web/{serviceName}/cron/{id} : Modifier
- DELETE /hosting/web/{serviceName}/cron/{id} : Supprimer
