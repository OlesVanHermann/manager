# Cahier des Charges : web-cloud/hosting - Certificat SSL

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion certificat SSL
- **Objectif** : Visualiser et gérer le SSL
- **Contexte** : Sécurité HTTPS du site

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud > Hébergements > hosting123 > Certificat SSL
- Titre : "Certificat SSL"
- Carte état du certificat
- Boutons actions (droite)
- Section domaines couverts
- Panneau info (droite)

## 3. COMPOSANTS UI
- **OdsCard** : État certificat, domaines couverts
- **OdsBadge** : Statut "Actif" (vert)
- **OdsButton** : "Régénérer le SSL" (secondaire bordure bleue)
- **OdsButton** : "Commander un SSL" (secondaire)
- **OdsButton** : "Supprimer le SSL" (danger rouge)
- **OdsMessage** : Panneau info bleu

## 4. ÉTAT DU CERTIFICAT
| Champ | Valeur |
|-------|--------|
| Type | Let's Encrypt |
| Fournisseur | Let's Encrypt Authority X3 |
| Création | 15 décembre 2025 à 10:30 |
| Expiration | 15 mars 2026 à 10:30 |
| Renouvellement | Automatique |

## 5. BOUTONS ACTIONS
- "Régénérer le SSL" (bordure bleue)
- "Commander un SSL" (bordure grise)
- "Supprimer le SSL" (fond rouge)

## 6. DOMAINES COUVERTS
Liste avec indicateur vert :
- www.example.com
- example.com
- blog.example.com
- shop.example.com
- api.example.com

## 7. PANNEAU INFO
- Fond bleu clair
- Titre : "ℹ️ Informations"
- Texte : "Le certificat Let's Encrypt est gratuit et renouvelé automatiquement tous les 90 jours. Pour un certificat payant avec validation étendue (EV), utilisez 'Commander un SSL'."

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ssl : État SSL
- POST /hosting/web/{serviceName}/ssl/regenerate : Regénérer
- DELETE /hosting/web/{serviceName}/ssl : Supprimer
- POST /hosting/web/{serviceName}/ssl : Commander nouveau
