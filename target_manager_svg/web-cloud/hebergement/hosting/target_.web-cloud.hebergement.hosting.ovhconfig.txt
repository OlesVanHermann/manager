# Cahier des Charges : web-cloud/hosting - Configuration PHP (.ovhconfig)

## 1. VUE D'ENSEMBLE
- **Écran** : Modale configuration PHP
- **Objectif** : Modifier paramètres PHP de l'hébergement
- **Contexte** : Configuration technique avancée

## 2. LAYOUT & STRUCTURE
- Modale plein écran avec overlay sombre
- Header avec titre + bouton fermer
- Stepper 2 étapes (Action, Configuration)
- Radio buttons pour choix action
- Formulaire config PHP
- Panneau aide (droite)
- Warning en bas
- Footer avec Annuler/Suivant

## 3. COMPOSANTS UI
- **OdsModal** : Container plein écran
- **OdsStepper** : 2 étapes
- **OdsRadio** : 2 options (Restaurer, Modifier)
- **OdsSelect** : Version PHP, Environnement, Moteur, etc.
- **OdsMessage** : Warning jaune
- **OdsButton** : Annuler, Suivant

## 4. STEPPER
1. Action (actif, bleu)
2. Configuration (gris)

## 5. OPTIONS ACTION
| Option | Description |
|--------|-------------|
| Restaurer une configuration précédente | Revenir à une version antérieure |
| Modifier la configuration actuelle (sélectionné) | Personnaliser les paramètres PHP |

## 6. FORMULAIRE CONFIGURATION PHP
| Paramètre | Valeur par défaut | Options |
|-----------|-------------------|---------|
| Version PHP | 8.2 | Dropdown versions |
| Environnement d'exécution | stable64 | stable64, legacy |
| Moteur | php | php, phpcgi |
| Environnement | production | production, development |
| Firewall HTTP | none | none, security |

## 7. PANNEAU AIDE
- **Versions PHP** : Version recommandée : PHP 8.2
- **Environnement** : stable64 (64 bits recommandé), legacy (compatibilité anciennes apps)
- **Firewall HTTP** : Active les protections contre les attaques web courantes

## 8. WARNING
- "⚠️ La modification de ces paramètres peut impacter le fonctionnement de votre site."

## 9. FOOTER
- "Annuler" (bouton secondaire)
- "Suivant →" (bouton primaire)

## 10. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ovhConfig : Config actuelle
- PUT /hosting/web/{serviceName}/ovhConfig : Modifier config
- GET /hosting/web/{serviceName}/ovhConfig/histories : Historique
- POST /hosting/web/{serviceName}/ovhConfig/rollback : Restaurer
