# Cahier des Charges : web-cloud/hosting - Modules en 1 clic

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion modules CMS
- **Objectif** : Installer/gérer WordPress, PrestaShop, etc.
- **Contexte** : Installation simplifiée CMS

## 2. LAYOUT & STRUCTURE
- Titre : "Modules en 1 clic"
- Description
- Bouton ajouter
- Tableau modules installés
- Grille modules disponibles

## 3. COMPOSANTS UI
- **OdsButton** : "Ajouter un module" (primaire)
- **OdsTable** : Modules installés
- **OdsBadge** : État (Actif)
- **OdsActionMenu** : Menu ⋮
- **OdsCard** : Cartes modules disponibles

## 4. DESCRIPTION
"Installez des modules populaires comme WordPress, PrestaShop, Joomla! en quelques clics."

## 5. TABLEAU MODULES INSTALLÉS
| Colonne | Description |
|---------|-------------|
| Module | Icône + nom (WordPress, PrestaShop) |
| Version | Numéro version |
| Chemin | Lien vers dossier (/www, /shop) |
| Login | Identifiant admin |
| Base de données | BDD associée |
| État | Badge Actif |
| Actions | Menu ⋮ |

## 6. EXEMPLES DONNÉES
| Module | Version | Chemin | Login | BDD | État |
|--------|---------|--------|-------|-----|------|
| WordPress (bleu #21759B) | 6.4.2 | /www | admin | mywebsitec_db1 | Actif |
| PrestaShop (rose #DF0067) | 8.1.2 | /shop | admin_shop | mywebsitec_db2 | Actif |

## 7. MENU ACTIONS (⋮)
- Accéder au site
- Interface d'administration
- Changer le mot de passe
- Supprimer le module (rouge)

## 8. MODULES DISPONIBLES
Cartes avec icône colorée :
- WordPress (bleu #21759B)
- PrestaShop (rose #DF0067)
- Joomla! (bleu #5091CD)
- Drupal (bleu #0678BE)

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/module : Liste modules
- POST /hosting/web/{serviceName}/module : Installer module
- DELETE /hosting/web/{serviceName}/module/{id} : Supprimer
- POST /hosting/web/{serviceName}/module/{id}/changePassword : Changer MDP
