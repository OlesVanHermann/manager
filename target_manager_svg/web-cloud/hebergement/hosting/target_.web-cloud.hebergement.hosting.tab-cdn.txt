# Cahier des Charges : web-cloud/hebergement - Tab CDN

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion du CDN (Content Delivery Network).

### Objectif fonctionnel
- Voir l'état du CDN
- Consulter les statistiques de cache
- Gérer le CDN par domaine
- Vider le cache

---

## 2. LAYOUT

### Structure 2 cartes + tableau domaines

---

## 3. COMPOSANTS UI

### Carte État du CDN
| Champ | Description |
|-------|-------------|
| Type de CDN | CDN Basic / Advanced |
| État | Badge Actif (vert) |
| Domaines couverts | Nombre de domaines |
| Bouton | Vider le cache CDN (outline) |
| Bouton | Changer d'offre (outline) |

### Carte Statistiques CDN
| Champ | Description |
|-------|-------------|
| Requêtes servies (24h) | Nombre total |
| Taux de cache hit | Pourcentage |
| Bande passante économisée | Valeur en Go |

### Tableau domaines CDN
| Colonne | Description |
|---------|-------------|
| Domaine | Nom du domaine |
| CDN | Badge Activé/Désactivé |
| Cache | Type de cache |
| Actions | Menu ⋮ |

### Menu actions (⋮)
| Option | Description |
|--------|-------------|
| Vider le cache | Purger cache domaine |
| Configurer le cache | Règles de cache |
| Désactiver CDN | Désactiver pour ce domaine |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web/{serviceName}/cdn
GET /hosting/web/{serviceName}/cdn/domain
POST /hosting/web/{serviceName}/cdn/flush
PUT /hosting/web/{serviceName}/cdn/domain/{domain}
```

### Modele
```typescript
interface HostingCdn {
  type: 'basic' | 'advanced';
  state: 'active' | 'inactive';
  domainsCount: number;
}

interface CdnStats {
  requests24h: number;
  cacheHitRate: number;
  bandwidthSaved: number;
}

interface CdnDomain {
  domain: string;
  enabled: boolean;
  cacheType: 'standard' | 'custom';
}
```
