# Cahier des Charges : web-cloud/hosting - Dashboard

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard hébergement web
- **Objectif** : Vue d'ensemble de l'hébergement
- **Contexte** : Page d'accueil du service

## 2. LAYOUT & STRUCTURE
- Sidebar : Navigation WEB CLOUD > Hébergements
- Breadcrumb : "Hébergements / mywebsite.com"
- Titre page : "Informations générales"
- 4 tuiles en grille 2x2

## 3. COMPOSANTS UI
- **OdsSidebar** : Navigation gauche
- **OdsCard** : 4 tuiles informations
- **OdsBadge** : Statut service (Actif)
- **OdsProgressBar** : Espace disque
- **OdsLink** : Actions (Modifier config, Changer offre, etc.)
