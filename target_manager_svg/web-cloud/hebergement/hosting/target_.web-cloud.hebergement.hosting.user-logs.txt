# Cahier des Charges : web-cloud/hosting - Logs et statistiques (User Logs)

## 1. VUE D'ENSEMBLE
- **Écran** : Accès logs et gestion utilisateurs stats
- **Objectif** : Consulter logs et gérer accès statistiques
- **Contexte** : Monitoring et partage accès

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud > Hébergements > hosting123 > Logs et statistiques
- Bandeau info avec lien guide
- Section Statistiques (bouton)
- Section Logs (bouton)
- Section Utilisateurs logs (tableau)

## 3. COMPOSANTS UI
- **OdsMessage** : Bandeau info bleu + lien guide
- **OdsButton** : "Voir les statistiques →", "Voir les logs →"
- **OdsButton** : "+ Créer un utilisateur" (secondaire bordure bleue)
- **OdsTable** : Liste utilisateurs logs
- **OdsActionMenu** : Menu ⋮ (Modifier, Mot de passe, Supprimer)

## 4. BANDEAU INFO
- "ℹ️ Consultez notre guide pour comprendre les statistiques de votre hébergement."
- Lien : "Consulter le guide ↗"

- Bouton : "Voir les statistiques →"

### Logs
- "Accédez aux logs bruts de votre hébergement."
- Bouton : "Voir les logs →"

| Colonne | Description |
|---------|-------------|

## 8. MENU ACTIONS (⋮)
- Modifier
- Mot de passe
- Supprimer

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/ownLogs/user : Liste utilisateurs
- POST /hosting/web/{serviceName}/ownLogs/user : Créer utilisateur
- PUT /hosting/web/{serviceName}/ownLogs/user/{login} : Modifier
- DELETE /hosting/web/{serviceName}/ownLogs/user/{login} : Supprimer
- POST /hosting/web/{serviceName}/ownLogs/user/{login}/changePassword : Changer MDP
