# Cahier des Charges : web-cloud/hosting - CDN

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration CDN
- **Objectif** : Gérer le CDN de l'hébergement
- **Contexte** : Optimisation performances et cache

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Web Cloud > Hébergements > hosting123 > CDN
- Tuile statut CDN
- Boutons actions (droite)
- Tableau configuration par multisite

## 3. COMPOSANTS UI
- **OdsCard** : Tuile statut CDN
- **OdsBadge** : Statuts (Actif/Désactivé)
- **OdsButton** : Vider cache, Configurer, Désactiver
- **OdsTable** : Config par domaine
- **OdsActionMenu** : Menu ⋮ (Configurer, Désactiver)

## 4. TUILE STATUT CDN
| Champ | Valeur |
|-------|--------|
| Service | Badge "Actif" (vert) |
| Type | CDN Basic |
| Backend | hosting123.cluster042.hosting.ovh.net |

## 5. BOUTONS ACTIONS
- "Vider le cache" (secondaire)
- "Configurer le CDN" (secondaire)
- "Désactiver le CDN" (secondaire)

## 6. TABLEAU CONFIGURATION PAR MULTISITE
| Colonne | Description |
|---------|-------------|
| Domaine | Nom du domaine |
| CDN | Badge Actif/Désactivé |
| Firewall | Badge Actif/Désactivé |
| Cache | Durée cache (24h, 1h) |
| HTTP/2 | Badge Actif/Désactivé |
| Actions | Menu ⋮ |

## 7. EXEMPLES DONNÉES
| Domaine | CDN | Firewall | Cache | HTTP/2 |
|---------|-----|----------|-------|--------|
| www.example.com | Actif | Actif | 24h | Actif |
| blog.example.com | Actif | Désactivé | 1h | Actif |
| api.example.com | Désactivé | - | - | - |

## 8. MENU ACTIONS (⋮)
- Configurer
- Désactiver

## 9. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/cdn : Statut CDN
- POST /hosting/web/{serviceName}/cdn/flush : Vider cache
- PUT /hosting/web/{serviceName}/cdn : Configurer
- DELETE /hosting/web/{serviceName}/cdn : Désactiver
- PUT /hosting/web/{serviceName}/attachedDomain/{domain}/cdn : Config par domaine
