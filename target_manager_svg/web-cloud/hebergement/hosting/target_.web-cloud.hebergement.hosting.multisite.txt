# Cahier des Charges : web-cloud/hosting - Multisite

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des sites multiples
- **Objectif** : Configurer les domaines/sous-domaines
- **Contexte** : Multisite sur hébergement

## 2. LAYOUT & STRUCTURE
- Titre : "Multisite"
- 2 boutons action
- Champ recherche
- Tableau des domaines

## 3. COMPOSANTS UI
- **OdsButton** : "Ajouter un domaine ou sous-domaine" (primaire)
- **OdsButton** : "Régénérer le certificat SSL" (secondaire)
- **OdsInput** : Recherche
- **OdsTable** : Liste domaines
- **OdsBadge** : SSL, CDN, Firewall, Logs (Activé/Désactivé)
- **OdsActionMenu** : Menu ⋮ par ligne

## 4. BOUTONS ACTION
- "Ajouter un domaine ou sous-domaine" (bleu primaire)
- "Régénérer le certificat SSL" (bordure grise)

## 5. TABLEAU MULTISITE
| Colonne | Description |
|---------|-------------|
| Domaine | Nom du domaine (lien) |
| Dossier racine | Chemin du répertoire |
| SSL | Badge Activé/Désactivé |
| CDN | Badge Activé/Désactivé |
| Firewall | Badge Activé/Désactivé |
| Logs séparés | Badge Activé/Désactivé |
| Diagnostic | Cercle vert (OK) |
| Actions | Menu ⋮ |

## 6. EXEMPLES DONNÉES
| Domaine | Dossier | SSL | CDN | Firewall | Logs |
|---------|---------|-----|-----|----------|------|
| mywebsite.com | /www | Activé | Activé | Activé | Désactivé |
| www.mywebsite.com | /www | Activé | Activé | Activé | Désactivé |
| blog.mywebsite.com | /blog | Activé | Désactivé | Activé | Activé |

## 7. MENU ACTIONS (⋮)
- Modifier le domaine
- Accéder aux logs
- Purger le CDN
- Associer Git
- Détacher le domaine (rouge)

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/attachedDomain : Liste domaines
- POST /hosting/web/{serviceName}/attachedDomain : Ajouter domaine
- PUT /hosting/web/{serviceName}/attachedDomain/{domain} : Modifier
- DELETE /hosting/web/{serviceName}/attachedDomain/{domain} : Supprimer
- POST /hosting/web/{serviceName}/ssl/regenerate : Regénérer SSL
