# Cahier des Charges : web-cloud/hosting - Changer d'offre

## 1. VUE D'ENSEMBLE
- **Écran** : Changement d'offre hébergement
- **Objectif** : Upgrade/downgrade vers nouvelle offre
- **Contexte** : Évolution besoins client

## 2. LAYOUT & STRUCTURE
- Breadcrumb : Hébergements > monsite > Changer d'offre
- Stepper 3 étapes (Offre, Configuration, Paiement)
- Tableau comparatif des offres
- Boutons Annuler/Continuer

## 3. COMPOSANTS UI
- **OdsStepper** : 3 étapes numérotées
- **OdsTable** : Comparatif offres
- **OdsBadge** : "Offre actuelle", "Recommandée"
- **OdsButton** : "Choisir" (primaire/secondaire), Annuler, Continuer

## 4. STEPPER
1. Offre (actif, bleu)
2. Configuration (gris)
3. Paiement (gris)

## 5. TABLEAU COMPARATIF
| Caractéristique | Perso (actuelle) | Pro (recommandée) | Performance 1 | Performance 2 |
|-----------------|------------------|-------------------|---------------|---------------|
| Espace disque | 100 Go | 250 Go | 500 Go | 1 To |
| Sites web | Illimités | Illimités | Illimités | Illimités |
| Bases de données | 10 | 25 | 50 | 100 |
| Comptes email | 100 | 500 | 1000 | 2000 |
| RAM garantie | - | - | 2 Go | 4 Go |
| Prix HT/mois | 3,99 € | 7,99 € | 15,99 € | 29,99 € |

## 6. BADGES OFFRES
- Offre actuelle : Fond gris + texte "Offre actuelle"
- Recommandée : Fond bleu + texte blanc "Recommandée"

## 7. BOUTONS ACTION
Par offre :
- "Offre actuelle" (grisé) pour offre en cours
- "Choisir" (bleu plein) pour recommandée
- "Choisir" (bordure bleue) pour autres

Footer :
- "Annuler" (gris)
- "Continuer" (bleu primaire)

## 8. APIS & ENDPOINTS
- GET /hosting/web/{serviceName}/availableOffer : Offres disponibles
- POST /hosting/web/{serviceName}/changeOffer : Changer offre
