# Cahier des Charges : web-cloud/hebergement - Liste des hébergements

## 1. RESUME EXECUTIF

### Perimetre
Page de listing de tous les hébergements du compte.

### Objectif fonctionnel
- Lister tous les hébergements
- Rechercher un hébergement
- Accéder à la gestion
- Commander un nouvel hébergement

---

## 2. LAYOUT

### Structure sidebar + titre + barre recherche + tableau

## 3. COMPOSANTS UI

### Sidebar navigation
| Element | Description |
|---------|-------------|
| Hébergements | Sélectionné (bleu) |
| Domaines | Lien |
| Emails | Lien |

### En-tête
| Element | Description |
|---------|-------------|
| Titre | Hébergements web |
| Recherche | Champ avec icône loupe |
| Bouton | + Commander (primary) |

### Tableau hébergements
| Colonne | Description |
|---------|-------------|
| Nom | Nom du service (lien bleu) |
| Offre | Perso/Performance 1/2/3 |
| Datacenter | Localisation (gra3, rbx7...) |
| État | Badge Actif/Expire bientôt |
| Expiration | Date (rouge si proche) |
| Actions | Menu ⋮ |

### États
| État | Badge |
|------|-------|
| Actif | Vert (#DEF7EC) |
| Expire bientôt | Rouge (#FEE2E2) |

### Menu actions (⋮)
| Option | Description |
|--------|-------------|
| Gérer | Accéder au dashboard |
| Renouveler | Prolonger l'abonnement |
| Changer d'offre | Upgrade/downgrade |
| Résilier | Destructif (rouge) |

### Pagination
| Element | Description |
|---------|-------------|
| Texte | 1-4 sur 4 hébergements |

---

## 4. APIS & DONNEES

### Endpoints
```
GET /hosting/web
GET /hosting/web/{serviceName}
GET /hosting/web/{serviceName}/serviceInfos
```

### Modele
```typescript
interface HostingListItem {
  serviceName: string;
  offer: string;
  datacenter: string;
  state: 'active' | 'expired' | 'expiring';
  expirationDate: string;
}
```
