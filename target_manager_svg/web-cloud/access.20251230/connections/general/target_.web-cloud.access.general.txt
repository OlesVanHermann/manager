STRUCTURE PAGE
1. NAV3: General (actif) | Ligne | Modem | Services | VoIP | Options | Taches
2. Card statut connexion
   - Indicateur statut (vert=connecte, rouge=deconnecte)
   - Uptime, Technologie, Offre, Adresse, NRO
   - Debits descendant/montant
3. Card equipement
   - Modele modem, MAC, Firmware, Statut WiFi
   - Badge OVH si modem gere
4. Card adresses IP
   - IPv4 WAN, IPv6, Gateway, DNS
   - Lien gestion IPs
5. Card services inclus
   - VoIP, TV, IPv6, Reverse DNS
   - Option commander IPs failover
6. Banniere info statut
7. Actions rapides
   - Diagnostic ligne, Redemarrer modem, Modifier mot de passe

NAV3 CONNEXIONS
| Tab      | Route     | Description              |

API ENDPOINTS
GET  /access/connections/{id}         -> Details connexion
GET  /access/connections/{id}/modem   -> Infos modem
GET  /access/connections/{id}/ips     -> Adresses IP
POST /access/connections/{id}/diagnostic -> Lancer diagnostic
