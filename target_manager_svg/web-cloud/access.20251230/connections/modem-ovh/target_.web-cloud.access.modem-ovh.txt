STRUCTURE PAGE
1. NAV3: General | Ligne | Modem | Services | VoIP | Options | Taches
2. NAV4: Dashboard | WiFi | DHCP | NAT/Ports | Routeur
3. Header modem
   - Modele, Firmware, Uptime
   - Boutons: Redemarrer, Mise a jour
4. Cards WiFi (2.4 GHz et 5 GHz)
   - Statut, SSID, Canal, Securite, Appareils
5. Cards Reseau local et Mode
   - IP LAN, DHCP, Baux actifs
   - Mode routeur/bridge, DMZ, Firewall

GET  /access/connections/{id}/modem/wifi    -> Config WiFi
PUT  /access/connections/{id}/modem/wifi    -> Modifier WiFi
