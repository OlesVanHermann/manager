STRUCTURE PAGE
1. NAV3: General | Ligne | Modem (actif) | Services | VoIP | Options | Taches
2. NAV4: Dashboard | WiFi | DHCP | NAT/Ports | Routeur (actif)
3. Card mode de fonctionnement
   - Mode Routeur (recommande, selectionne par defaut)
   - Mode Bridge (pour routeur externe)
4. Card Firewall
   - Toggle activation
   - Niveau protection (Faible/Standard/Strict)
   - Toggle ping WAN
5. Card IPv6
   - Toggle support IPv6
   - Prefixe IPv6
   - Mode (SLAAC, DHCPv6, Statique)
6. Card MTU
   - Valeur MTU (defaut 1500)
7. Banniere warning modifications

MODES FONCTIONNEMENT
| Mode     | NAT | DHCP | WiFi | Firewall | Usage                    |
|----------|-----|------|------|----------|--------------------------|
| Routeur  | Oui | Oui  | Oui  | Oui      | Usage standard           |
| Bridge   | Non | Non  | Non  | Non      | Avec routeur externe     |

NIVEAUX FIREWALL
| Niveau   | Description                                      |
|----------|--------------------------------------------------|
| Faible   | Bloque ports dangereux uniquement                |
| Standard | Protection equilibree (recommande)               |
| Strict   | Bloque tout sauf regles explicites               |

MODES IPv6
| Mode           | Description                                |
|----------------|-----------------------------------------------|
| SLAAC          | Auto-configuration sans etat                  |
| DHCPv6         | Attribution par serveur DHCP                  |
| SLAAC+DHCPv6   | Hybride (recommande)                          |
| Statique       | Configuration manuelle                        |

API ENDPOINTS
GET  /access/connections/{id}/modem/router    -> Config routeur
PUT  /access/connections/{id}/modem/router    -> Modifier mode
PUT  /access/connections/{id}/modem/firewall  -> Config firewall
PUT  /access/connections/{id}/modem/ipv6      -> Config IPv6
