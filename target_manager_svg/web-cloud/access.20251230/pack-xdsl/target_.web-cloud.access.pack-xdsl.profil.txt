# STRUCTURE PAGE
1. NAV3: Général | Ligne | Modem | Services | VoIP | [Options] | Tâches
2. Bannière info: Explication profil de ligne
3. Bloc PROFIL ACTUEL: Profil, débit max, SNR, interleave
4. Bloc CHANGER DE PROFIL: 3 options radio
5. Bloc HISTORIQUE: Lien vers historique

# PROFILS DISPONIBLES
- Standard: équilibre débit/stabilité (SNR 6dB)
- Performance: max débit (SNR 3dB, moins stable)
- Stabilité: max stabilité (SNR 9dB, débit réduit)

# INTERACTIONS
- Clic radio profil → Sélection
- Clic "Appliquer" → PUT profil + resync automatique
- Clic "Voir historique" → Liste modifications

# API ENDPOINTS
GET  /access/connections/{id}/profile           - Profil actuel
GET  /access/connections/{id}/profile/available - Profils dispo
PUT  /access/connections/{id}/profile           - Changer profil
GET  /access/connections/{id}/profile/history   - Historique
