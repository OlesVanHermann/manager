# STRUCTURE MODAL
1. Icône warning (cercle jaune)
2. Titre: Question confirmation
3. Warning box: Avertissement impact
4. Boutons: Annuler + Confirmer

# CONTEXTE
- Apparaît lors de l'activation/désactivation d'un service
- Avertit que l'action affecte tous les partages liés
- Requiert confirmation explicite

# SERVICES CONCERNÉS
- Partage de contenu (DLNA, UPnP, etc.)
- Partage de fichiers
- Services multimédia

# INTERACTIONS
- Clic "Annuler" → Ferme modal, aucune action
- Clic "Confirmer" → PUT service, toggle état

# API ENDPOINTS
PUT /access/connections/{id}/modem/services/{serviceId}
Body: { "enabled": true/false }
