# STRUCTURE PAGE
1. NAV3: ... | [Options] | ...
2. Bloc PORTABILITÉ: Options radio + champ RIO
3. Info box: Aide obtention RIO
4. Bouton validation

# OPTIONS PORTABILITÉ
1. S'affranchir du numéro - Ne pas conserver le numéro existant
2. Conserver le numéro - Requiert code RIO

# CODE RIO
- Relevé d'Identité Opérateur
- 12 caractères alphanumériques
- Obtenu en appelant le 3179 (gratuit)

# INTERACTIONS
- Sélection radio → Active/désactive champ RIO
- Clic "?" → Affiche aide contextuelle
- Clic "Valider" → POST portabilité avec RIO

# VALIDATION
- RIO requis si conservation numéro
- Format: 12 caractères [A-Z0-9]
- Vérification validité côté API

# API ENDPOINTS
GET  /access/connections/{id}/portability        - Infos portabilité
POST /access/connections/{id}/portability        - Valider portabilité
GET  /access/connections/{id}/portability/status - Statut demande
