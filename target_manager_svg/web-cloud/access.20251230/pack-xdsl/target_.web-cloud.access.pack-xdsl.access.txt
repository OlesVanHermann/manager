# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section Web Cloud (Accès actif)
3. SOUS-NAV - Toggle [Connexions] | [OverTheBox]
4. LEFT PANEL - Liste des connexions
5. RIGHT PANEL
   - Header: Nom connexion
   - NAV3: Général | [Ligne] | Modem | Services | VoIP | Options | Tâches
   - NAV4: [Statut] | Diagnostic | Statistiques | Alertes
   - Bloc STATUS: Connecté/Déconnecté + uptime + bouton Resynchroniser
   - Bloc DÉBIT + QUALITÉ LIGNE (2 colonnes)
   - Bloc ADRESSES IP + INSTALLATION (2 colonnes)

# INTERACTIONS
- Clic NAV4 "Diagnostic" → Vue diagnostic
- Clic NAV4 "Statistiques" → Vue graphiques
- Clic NAV4 "Alertes" → Gestion alertes
- Clic "Resynchroniser" → Action resync ligne
- Clic icône copie IP → Copie dans presse-papier

# API ENDPOINTS
GET  /access/connections/{id}/line           - Statut ligne complet
POST /access/connections/{id}/line/resync    - Resynchroniser
POST /access/connections/{id}/line/reset     - Réinitialiser
