# STRUCTURE PAGE
1. NAV3: Général | Ligne | [Modem] | Services | VoIP | Options | Tâches
2. NAV4: [Dashboard] | WiFi | DHCP | NAT/Ports | Routeur
3. Bloc STATUS MODEM: Modèle, firmware, uptime + actions
4. Bloc WIFI 2.4GHz + 5GHz: SSID, canal, sécurité, appareils
5. Bloc RÉSEAU LOCAL + MODE: IP LAN, DHCP, routeur/bridge

# INTERACTIONS
- Clic "Redémarrer" → Confirmation + reboot modem
- Clic "Mise à jour" → Update firmware si disponible
- Clic "Configurer" WiFi → Modal config WiFi
- Clic "DHCP" → Navigation vers /modem-dhcp
- Clic radio mode → Changement routeur/bridge

# API ENDPOINTS
GET  /access/connections/{id}/modem         - Infos modem
POST /access/connections/{id}/modem/reboot  - Redémarrer
POST /access/connections/{id}/modem/update  - Mise à jour
GET  /access/connections/{id}/modem/wifi    - Config WiFi
