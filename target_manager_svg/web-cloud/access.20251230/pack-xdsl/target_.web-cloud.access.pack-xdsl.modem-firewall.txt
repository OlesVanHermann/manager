# STRUCTURE PAGE
1. NAV4: Dashboard | WiFi | DHCP | NAT/Ports | [Routeur]
2. Sous-nav: LAN | [Firewall] | Bridge
3. Bloc NIVEAU DE PROTECTION: Dropdown sélection niveau
4. Bloc DÉTAIL DES NIVEAUX: Explications colorées (Désactivé, Normal, Élevé)
5. Bloc IMPACT SUR FONCTIONNALITÉS: Tableau comparatif

# NIVEAUX FIREWALL
- Désactivé: Aucune protection (bg-error)
- Normal: Protection standard avec NAT/PAT (bg-success, recommandé)
- Élevé: Toutes connexions entrantes bloquées (bg-warning)

# INTERACTIONS
- Dropdown niveau → Sélection du niveau
- Changement niveau → PUT + confirmation
- Si "Élevé" → Avertissement redirections ports désactivées

# API ENDPOINTS
GET /access/connections/{id}/modem/firewall       - Niveau actuel
PUT /access/connections/{id}/modem/firewall       - Modifier niveau
