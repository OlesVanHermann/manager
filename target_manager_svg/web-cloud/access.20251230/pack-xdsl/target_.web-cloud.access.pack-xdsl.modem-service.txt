# STRUCTURE PAGE
1. NAV3: ... | [Modem] | ...
2. Bloc SERVICES: Grille de toggles
3. Info box: Avertissement dépendances

# SERVICES DISPONIBLES

| Service | Description |
|---------|-------------|
| Bloc d'IP | Allocation d'adresses IP supplémentaires |
| Mise en attente | Mise en attente d'appel téléphonique |
| Partage de contenu | DLNA, partage multimédia |
| FTP | Serveur FTP intégré |
| IPsec ALG | Application Layer Gateway pour VPN IPsec |
| SIP ALG | Application Layer Gateway pour VoIP |
| UPnP | Universal Plug and Play (ouverture ports auto) |

# DÉPENDANCES
- Partage de contenu → Lie tous les partages (DLNA, etc.)
- Désactiver un parent désactive les enfants

# INTERACTIONS
- Clic toggle → Active/désactive le service
- Changement instantané (pas de bouton valider)
- Modal confirmation pour services liés

# API ENDPOINTS
GET  /access/connections/{id}/modem/services          - Liste services
PUT  /access/connections/{id}/modem/services/{sid}    - Toggle service
Body: { "enabled": true/false }
