# STRUCTURE PAGE
1. NAV3: Général | [Ligne] | Modem | Services...
2. NAV4: Statut | Diagnostic | Statistiques | [Alertes]
3. Toolbar: Refresh + Ajouter alerte
4. Bloc ALERTES ACTIVES: Table des alertes configurées
5. Bloc TYPES DISPONIBLES: Cards des types d'alertes

# TYPES D'ALERTES
- Déconnexion ligne: down > X minutes
- Dégradation qualité: SNR/atténuation
- Rapport périodique: stats quotidien/hebdo/mensuel
- Resynchronisation: notif à chaque resync

# INTERACTIONS
- Clic "Ajouter alerte" → Modal configuration
- Clic ✎ → Modifier alerte
- Clic × → Supprimer alerte
- Clic "Configurer" card → Modal pré-rempli

# API ENDPOINTS
GET    /access/connections/{id}/line/alerts        - Liste alertes
POST   /access/connections/{id}/line/alerts        - Ajouter
PUT    /access/connections/{id}/line/alerts/{aid}  - Modifier
DELETE /access/connections/{id}/line/alerts/{aid}  - Supprimer
