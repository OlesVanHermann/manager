# STRUCTURE MODALE
1. Header: Titre + bouton fermer
2. Connexion concernée (info)
3. Type d'IP: Radio buttons (Failover, /29, /28)
4. Affectation: Champ texte optionnel
5. Récapitulatif: Prix HT et TTC
6. Actions: Annuler | Commander

# INTERACTIONS
- Clic radio → Sélection type IP + mise à jour prix
- Clic × → Ferme modal
- Clic Annuler → Ferme modal
- Clic Commander → POST API + ferme modal

# API ENDPOINTS
GET  /access/connections/{id}/ips/available  - Types et prix disponibles
POST /access/connections/{id}/ips            - Commander l'IP
