2. Bloc MTU: Description + info + dropdown + bouton

# MTU (Maximum Transmission Unit)
- Taille maximale des paquets réseau
- Valeur par défaut: 1492 (xDSL) ou 1500 (Fibre)
- Réduire peut résoudre des problèmes de fragmentation

# VALEURS COURANTES
| Valeur | Usage |
|--------|-------|
| 1500 | Standard Ethernet/Fibre |
| 1492 | xDSL (PPPoE overhead) |
| 1472 | VPN commun |
| 1400 | Tunnels/Double NAT |

# CAS D'USAGE RÉDUCTION
- Problèmes de navigation sur certains sites
- Difficultés FTP/SFTP
- VPN qui ne fonctionne pas
- Fragmentation excessive

# INTERACTIONS
- Sélection dropdown → Change valeur
- Clic "Valider" → PUT nouvelle valeur

# API ENDPOINTS
GET  /access/connections/{id}/modem/mtu        - MTU actuel
PUT  /access/connections/{id}/modem/mtu        - Modifier MTU
GET  /access/connections/{id}/modem/mtu/values - Valeurs disponibles
