# STRUCTURE BANNIÈRE
1. Icône info (i)
2. Titre "Équipement non activé"
3. Message explicatif
4. Liste des étapes à suivre
5. Lien guide d'installation

# STYLE
- Fond bleu clair (#EFF6FF)
- Bordure bleue (#0050D7)
- Texte bleu foncé (#1E40AF)

# CAS D'AFFICHAGE
Bannière affichée quand:
- Un appareil OTB est détecté
- L'appareil a un abonnement associé
- Mais l'appareil n'a pas encore été activé

# ÉTAPES D'ACTIVATION
1. Se connecter à l'interface de gestion OTB
2. Suivre les instructions d'activation

# INTERACTIONS
- Clic guide → Documentation externe

# PROCESSUS D'ACTIVATION
1. Client reçoit le boîtier
2. Connecte le boîtier au réseau
3. Accède à http://overthebox.lan
4. Suit l'assistant d'activation
5. L'appareil se synchronise avec l'API OVH
6. Le statut passe à "Actif"

# API ENDPOINTS
GET  /overthebox/{id}         - Vérifie status activation
