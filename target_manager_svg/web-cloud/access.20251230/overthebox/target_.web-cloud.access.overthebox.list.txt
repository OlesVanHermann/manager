# STRUCTURE PAGE
1. Titre "OverTheBox" + bouton Changelog
2. Card principale avec toolbar + table
3. Pagination

# TOOLBAR
- Bouton "+ Commander" → Commander nouvelle OTB
- Champ recherche → Filtrer par nom/description

# COLONNES TABLE

| Colonne | Description |
|---------|-------------|
| Nom | serviceName (lien cliquable) |
| Description | Nom personnalisé |
| Statut | Actif / Création / Inactif |
| Actions | Menu contextuel |

# STATUTS

| Statut | Couleur | Description |
|--------|---------|-------------|
| Actif | Vert | OTB opérationnelle |
| Création | Orange | En cours d'activation |
| Inactif | Gris | Suspendue/résiliée |

# INTERACTIONS
- Clic nom → Détails OTB
- Clic ⋮ → Menu: Renommer, Résilier
- Clic Commander → Tunnel achat

# API ENDPOINTS
GET    /overthebox           - Liste des OTB
POST   /overthebox/order     - Commander nouvelle OTB
DELETE /overthebox/{id}      - Résilier OTB
