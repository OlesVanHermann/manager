# STRUCTURE MODAL
1. Titre "Contact de livraison"
2. Description
3. Grille de contacts sélectionnables
4. Lien ajouter contact
5. Boutons Annuler/Sélectionner

# CONTACTS
- Affichage en grille (2 colonnes)
- Contact sélectionné = bordure bleue + badge check
- Contact non sélectionné = bordure grise

# INFORMATIONS CONTACT

| Champ | Description |
|-------|-------------|
| Nom | Nom du contact |
| Adresse | Ligne 1, 2 |
| Ville | Code postal + ville |
| Pays | Pays |
| Téléphone | Numéro |
| Email | Adresse email |

# INTERACTIONS
- Clic contact → Sélection (bordure bleue)
- Clic "Ajouter" → Modal création contact
- Clic Annuler → Retour étape précédente
- Clic Sélectionner → Étape suivante (résumé)

# TUNNEL MIGRATION
1. Choix offre
2. **Contact livraison (cette page)**
3. Résumé + validation

# API ENDPOINTS
GET  /me/contacts                     - Liste contacts
POST /me/contacts                     - Créer contact
