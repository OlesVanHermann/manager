# STRUCTURE PAGE
1. Titre + bouton souscrire
2. Formulaire options (gauche)
3. Récapitulatif prix (droite)
4. Warning appareil non trouvé

# FORMULAIRE OPTIONS

| Champ | Type | Description |
|-------|------|-------------|
| Type d'offre | Select | OTB IT v1, Plus, etc. |
| Durée | Select | 1 mois, 12 mois... |
| Matériel | Checkbox | Commander boîtier |
| Description | Input | Nom personnalisé |

# RÉCAPITULATIF

| Ligne | Valeur |
|-------|--------|
| Offre mensuelle | 9,99 €/mois |
| Boîtier (si coché) | 99,00 € une fois |
| Total HT | Somme |
| Total TTC | Avec TVA |

# WARNING
Affiché si aucun appareil OTB détecté sur le réseau.
Propose de commander ou configurer un appareil existant.

# INTERACTIONS
- Select offre → Recalcul prix
- Toggle matériel → Ajoute/retire prix boîtier
- Clic Valider → POST commande

# API ENDPOINTS
GET  /overthebox/offers           - Liste offres disponibles
POST /overthebox/order            - Créer commande
Body: { offer, duration, hardware, description }
