STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - NAV3: 10 tabs (Zone DNS actif)
   - Title + Description
   - Toolbar: Ajouter, Importer, Exporter, Reinitialiser, Filtre
   - Table enregistrements DNS
   - Pagination

TOOLBAR ACTIONS
| Bouton              | Style    | Action                    |
|---------------------|----------|---------------------------|
| + Ajouter une entree| Primaire | Modal ajout enregistrement|
| Importer            | Outline  | Modal import zone         |
| Exporter            | Outline  | Telecharger fichier zone  |
| Reinitialiser       | Outline  | Modal confirmation reset  |

TABLE COLONNES
| Colonne     | Description                          |
|-------------|--------------------------------------|
| Sous-domaine| @ ou nom du sous-domaine             |

| TTL         | Duree en secondes                    |
| Cible       | Valeur de l'enregistrement           |
| Actions     | Menu contextuel (⋮)                  |

API ENDPOINTS
GET    /domain/zone/{zoneName}/record          -> Liste enregistrements
GET    /domain/zone/{zoneName}/record/{id}     -> Detail enregistrement
POST   /domain/zone/{zoneName}/record          -> Ajouter enregistrement
PUT    /domain/zone/{zoneName}/record/{id}     -> Modifier enregistrement
DELETE /domain/zone/{zoneName}/record/{id}     -> Supprimer enregistrement
GET    /domain/zone/{zoneName}/export          -> Exporter zone (fichier)
POST   /domain/zone/{zoneName}/import          -> Importer zone
POST   /domain/zone/{zoneName}/reset           -> Reinitialiser zone
