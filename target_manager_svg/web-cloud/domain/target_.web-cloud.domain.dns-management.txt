# Cahier des Charges : web-cloud/domain - Serveurs DNS

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des serveurs DNS
- **Type** : Page avec tableau
- **Objectif** : Gérer les serveurs DNS du domaine

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar navigation
- Breadcrumb
- Titre
- Bandeau warning (si moins de 2 serveurs)
- Boutons d'action
- Tableau serveurs DNS

## 3. COMPOSANTS UI
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Ajouter serveur, Réinitialiser
- **OdsTable** : Liste des serveurs DNS
- **OdsBadge** : Statut (Actif)
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. BREADCRUMB
Web Cloud › Noms de domaine › **example.com**

## 5. TITRE
**Serveurs DNS**

## 6. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
- "⚠️ Attention, il est fortement déconseillé de n'avoir qu'un seul serveur DNS actif."
- "Ajoutez un serveur DNS secondaire pour assurer la disponibilité de votre domaine."

## 7. BOUTONS D'ACTION
| Bouton | Style |
|--------|-------|
| + Ajouter un serveur DNS | Primaire bleu |
| Réinitialiser les serveurs DNS | Secondaire outline |

## 8. TABLEAU SERVEURS DNS
| Colonne | Description |
|---------|-------------|
| SERVEUR DNS | Nom du serveur |
| IP ASSOCIÉE | Adresse IP |
| STATUT | Badge Actif/Inactif |
| ACTIONS | Menu ⋮ |

### Exemple données
| Serveur DNS | IP Associée | Statut |
|-------------|-------------|--------|
| dns1.ovh.net | 213.186.33.99 | Actif |
| ns1.ovh.net | 213.251.128.136 | Actif |
| ns2.ovh.net | 213.251.188.136 | Actif |

## 9. MENU ACTIONS (⋮)
- Supprimer (texte rouge danger)

## 10. APIS & ENDPOINTS
- GET /domain/{serviceName}/nameServer : Liste serveurs DNS
- POST /domain/{serviceName}/nameServer : Ajouter serveur DNS
- DELETE /domain/{serviceName}/nameServer/{id} : Supprimer serveur
- POST /domain/{serviceName}/nameServer/reset : Réinitialiser
