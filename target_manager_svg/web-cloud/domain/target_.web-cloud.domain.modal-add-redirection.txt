STRUCTURE MODAL
1. Header: Titre + bouton fermer (x)
2. Formulaire
   - Source (depuis)
   - Destination (vers)
   - Type de redirection (radio)
   - Options avancees (checkbox)
3. Banniere info
4. Footer: Annuler + Enregistrer

CHAMPS FORMULAIRE
| Champ         | Type     | Validation           | Exemple                      |
|---------------|----------|----------------------|------------------------------|
| Source        | text     | Required, URL valide | http://example.com/          |
| Destination   | text     | Required, URL valide | https://www.example.com/page |
| Type          | radio    | Required             | 301, iframe, 302             |
| Wildcard      | checkbox | Optional             | *.example.com                |

TYPES DE REDIRECTION
| Type          | Code | Description                                 |
|---------------|------|---------------------------------------------|
| Visible       | 301  | Redirection permanente - recommandee SEO    |
| Invisible     | -    | Contenu dans iframe, URL inchangee          |
| Temporaire    | 302  | Redirection temporaire - maintenances       |

OPTIONS AVANCEES
| Option    | Description                               |
|-----------|-------------------------------------------|
| Wildcard  | Redirige tous les sous-domaines           |

BOUTONS
| Bouton     | Style     | Action                    |
|------------|-----------|---------------------------|
| Annuler    | Secondary | Fermer modal sans sauver  |
| Enregistrer| Primaire  | POST + fermer modal       |

API ENDPOINT
POST /domain/{serviceName}/redirection -> Ajouter redirection
