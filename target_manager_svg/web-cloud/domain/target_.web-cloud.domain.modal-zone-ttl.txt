# Cahier des Charges : web-cloud/domain - Modal Modifier TTL par défaut

## 1. VUE D'ENSEMBLE
- **Modal** : Modification du TTL par défaut de la zone
- **Type** : Formulaire simple
- **Objectif** : Changer le TTL global de la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Description de l'action
- Explication du TTL
- Champ TTL
- Conseils
- Bandeau info propagation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x400)
- **OdsInput** : TTL
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier le TTL par défaut"

## 5. DESCRIPTION
"Vous êtes sur le point de modifier la valeur du TTL par défaut de la zone **example.com**"

## 6. EXPLICATION TTL
"Le TTL correspond au temps maximum (en secondes) de propagation de mise à jour de la zone DNS."

## 7. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL par défaut" |
| Input | 3600 |
| Suffixe | "secondes" |

## 8. CONSEILS
- "Valeur conseillée en cas de changements fréquents : 3600 secondes."
- "Sinon : 86400 secondes (1 jour)."

## 9. BANDEAU INFO
Fond bleu (#E8F0FE) :
"ℹ️ La modification sera effective après un temps de propagation de max 24h."

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Modifier (primaire)

## 11. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName} : Modifier TTL par défaut
  - ttl: number
