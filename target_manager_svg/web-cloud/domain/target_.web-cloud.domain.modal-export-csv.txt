# Cahier des Charges : web-cloud/domain - Modal Export CSV

## 1. VUE D'ENSEMBLE
- **Modal** : Export des domaines en CSV
- **Type** : Formulaire de sélection colonnes
- **Objectif** : Exporter la liste des domaines avec les colonnes choisies

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Description
- Section "Informations du domaine"
- Section "Options de sécurité"
- Section "Contacts"
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x580)
- **OdsCheckbox** : Sélection des colonnes
- **OdsButton** : Annuler, Exporter

## 4. TITRE
"Exporter les domaines en CSV"

## 5. DESCRIPTION
"Sélectionnez les colonnes à inclure dans l'export :"

## 6. SECTION INFORMATIONS DU DOMAINE
| Checkbox | État |
|----------|------|
| ☑ Nom du domaine ASCII | Coché |
| ☑ Nom du domaine UTF-8 | Coché |
| ☑ Date de création | Coché |
| ☑ Date d'expiration | Coché |

## 7. SECTION OPTIONS DE SÉCURITÉ
| Checkbox | État | Description |
|----------|------|-------------|
| ☐ DNSSEC | Non coché | Inclure l'état de la délégation sécurisée |
| ☐ DNS Anycast | Non coché | Inclure l'état d'activation de la solution DNS Anycast |

## 8. SECTION CONTACTS
| Checkbox | État |
|----------|------|
| ☐ Contact de facturation | Non coché |
| ☐ Contact technique | Non coché |

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Exporter (primaire)

## 10. APIS & ENDPOINTS
- GET /domain/export/csv : Exporter en CSV avec paramètres colonnes
