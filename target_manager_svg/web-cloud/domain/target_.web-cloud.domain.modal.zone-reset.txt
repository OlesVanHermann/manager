STRUCTURE MODAL
1. Icone warning (danger)
2. Header: Titre
3. Banniere danger (avertissement)
4. Options de reinitialisation (radio)
5. Champ confirmation texte
6. Footer: Annuler + Reinitialiser

TYPES DE REINITIALISATION
| Type        | Description                                      |
|-------------|--------------------------------------------------|
| Minimale    | Cree uniquement les enregistrements NS de base   |
| Normale     | Cree NS + A/AAAA vers les serveurs OVH           |
| Avec MX     | Cree NS + A/AAAA + MX vers les serveurs OVH      |

CONFIRMATION
| Champ        | Type | Validation                   |
|--------------|------|------------------------------|
| Confirmation | text | Required, doit etre "REINITIALISER" |

BOUTONS
| Bouton       | Style   | Action                    |
|--------------|---------|---------------------------|
| Annuler      | Secondary| Fermer modal sans action  |
| Reinitialiser| Danger  | POST + fermer modal       |

API ENDPOINT
POST /domain/zone/{zoneName}/reset -> Reinitialiser zone DNS
