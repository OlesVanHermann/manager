STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
   - Recherche
   - Filtre
   - Items avec icone type (domain/zone)
   - Badge extension (.com, .fr)
   - Badge statut (Actif, Expire bientot, Expire)
   - Pagination
4. RIGHT PANEL
   - Header: Nom domaine + badges securite
   - NAV3: 10 tabs (Infos generales actif)
   - Layout 3 colonnes:
     * DOMAINE: Nom, Extension, Statut, Serveurs DNS, Sous-domaines
     * SECURITE: Verrou, DNSSEC, AUTH/INFO, WHOIS
     * ABONNEMENT: Creation, Expiration, Renouveler, Contacts
   - Row 2: GUIDES, OPTIONS, CONSEILS

API ENDPOINTS
GET    /domain                              -> Liste domaines
GET    /domain/zone                         -> Liste zones
GET    /domain/{domain}                     -> Details domaine
GET    /domain/{domain}/serviceInfos        -> Infos facturation
GET    /domain/{domain}/nameServer          -> Serveurs DNS
GET    /domain/{domain}/authInfo            -> Code AUTH
POST   /domain/{domain}/lock                -> Activer verrou
DELETE /domain/{domain}/lock                -> Desactiver verrou
