# Cahier des Charges : web-cloud/domain - Modal Ajouter entrée DNS

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un enregistrement DNS générique
- **Type** : Formulaire de création
- **Objectif** : Ajouter une entrée dans la zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Sélection type de champ
- Champ sous-domaine avec suffixe
- Section TTL (défaut/personnalisé)
- Champ cible
- Note sur propagation
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x520)
- **OdsSelect** : Type de champ
- **OdsInput** : Sous-domaine, TTL personnalisé, Cible
- **OdsRadio** : TTL (Par défaut / Personnalisé)
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter une entrée DNS"

## 5. CHAMP TYPE
| Élément | Valeur |
|---------|--------|
| Label | "Type de champ *" |
| Obligatoire | Oui |
| Type | Select dropdown |
| Valeur | "A" |
| Options | A, AAAA, CNAME, MX, TXT, NS, SRV, etc. |

## 6. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | Texte optionnel |
| Placeholder | "sous-domaine (optionnel)" |
| Suffixe (addon gris) | ".example.com" |

## 7. SECTION TTL
| Option | État | Valeur |
|--------|------|--------|
| Par défaut | ● Sélectionné | - |
| Personnalisé | ○ | Input + "secondes" |

Input personnalisé :
- Largeur : 120px
- Valeur par défaut : 3600
- Suffixe : "secondes"

## 8. CHAMP CIBLE
| Élément | Valeur |
|---------|--------|
| Label | "Cible *" |
| Obligatoire | Oui |
| Placeholder | "Adresse IP (ex: 203.0.113.50)" |

## 9. NOTE PROPAGATION
"La modification sera immédiate dans la zone DNS, mais veuillez prendre en compte le temps de propagation (maximum 24h)."

## 10. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter (primaire)

## 11. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement DNS
  - fieldType: string
  - subDomain: string (optionnel)
  - target: string
  - ttl: number (optionnel)
