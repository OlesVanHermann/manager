API ENDPOINTS
GET    /domain/{domain}/nameServer         -> Liste serveurs DNS
POST   /domain/{domain}/nameServer         -> Modifier serveurs
GET    /domain/{domain}/nameServer/status  -> Statut serveurs
GET    /domain/{domain}/option/dnsAnycast  -> État Anycast
POST   /order/domain/zone/{zone}/dnsAnycast -> Commander Anycast
DELETE /domain/{domain}/option/dnsAnycast  -> Résilier Anycast
