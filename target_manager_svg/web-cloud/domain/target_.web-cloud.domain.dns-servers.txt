STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - NAV3: 10 tabs (Serveurs DNS actif)
   - Toolbar: Ajouter serveur, Reinitialiser OVH, DNS Anycast
   - Table serveurs DNS
   - Info box serveurs recommandes

TOOLBAR ACTIONS
| Bouton            | Action                          |
|-------------------|---------------------------------|
| + Ajouter serveur | Modal ajout serveur DNS         |
| Reinitialiser OVH | Reset vers serveurs OVH         |
| DNS Anycast       | Page/Modal configuration Anycast|

TABLE COLONNES
| Colonne      | Description                     |
|--------------|---------------------------------|
| SERVEUR DNS  | Nom FQDN du serveur             |
| IP ASSOCIEE  | Adresse IP (si GLUE)            |
| STATUT       | Badge Actif/Inactif             |
| ACTIONS      | Supprimer                       |

API ENDPOINTS
GET    /domain/{domain}/nameServer                -> Liste serveurs DNS
POST   /domain/{domain}/nameServer                -> Ajouter serveur
DELETE /domain/{domain}/nameServer/{id}           -> Supprimer serveur
POST   /domain/{domain}/nameServer/reset          -> Reset OVH
GET    /domain/{domain}/option/dnsAnycast         -> Statut Anycast
POST   /domain/{domain}/option/dnsAnycast/enable  -> Activer Anycast
