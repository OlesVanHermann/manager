# Cahier des Charges : web-cloud/domain - Modal Supprimer GLUE

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'un enregistrement GLUE
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer un serveur de noms personnalisé (GLUE record)

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation
- Détails de l'enregistrement (hôte + IP)
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (500x280)
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. TITRE
"Supprimer un enregistrement GLUE"

## 5. QUESTION
"Êtes-vous sûr de vouloir supprimer cet enregistrement GLUE ?"

## 6. DÉTAILS AFFICHÉS
| Élément | Valeur |
|---------|--------|
| Hôte | **ns1.example.com** |
| IP(s) | **203.0.113.10** |

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
"⚠️ La suppression sera effective d'ici quelques minutes."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 9. APIS & ENDPOINTS
- DELETE /domain/{serviceName}/glueRecord/{host} : Supprimer GLUE
