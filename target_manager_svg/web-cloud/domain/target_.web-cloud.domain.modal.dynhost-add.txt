2. Banniere info (explication DynHost)
3. Formulaire

   - Adresse IP actuelle
   - Identifiant DynHost
   - Mot de passe
   - Confirmation mot de passe
4. Footer: Annuler + Creer

CHAMPS FORMULAIRE
| Champ               | Type     | Validation           | Exemple          |
|---------------------|----------|----------------------|------------------|
| Sous-domaine        | text     | Required, alphanum   | home             |
| Adresse IP actuelle | text     | Required, IPv4 valide| 90.123.45.67     |
| Identifiant DynHost | text     | Auto-genere          | example.com-home |
| Mot de passe        | password | Required, min 8 chars| ••••••••         |
| Confirmation        | password | Required, match      | ••••••••         |
