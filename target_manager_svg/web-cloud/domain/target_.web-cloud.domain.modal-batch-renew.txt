STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière explicative
3. Dropdown durée (1 an, 2 ans, etc.)
4. Table domaines sélectionnés:
   - Checkbox
   - Domaine
   - Expiration actuelle
   - Prix renouvellement
   - Nouvelle expiration
5. Récapitulatif avec totaux
6. Footer: Annuler | Commander

DURÉES DISPONIBLES
| Durée  | Disponibilité                          |
|--------|----------------------------------------|
| 1 an   | Toutes extensions                      |
| 2 ans  | Selon extension                        |
| 3 ans  | Selon extension                        |
| 5 ans  | Selon extension                        |
| 10 ans | Selon extension                        |

CALCUL PRIX
- Chaque domaine a son propre prix selon l'extension
- Total HT = Somme des prix individuels
- TVA = Total HT × 20%
- Total TTC = Total HT + TVA

API ENDPOINTS
GET  /order/catalog/formatted/domain         -> Prix renouvellement
POST /order/cart/{cartId}/domain/{domain}    -> Ajouter au panier
  Body: { "action": "renew", "duration": "1y" }
POST /order/cart/{cartId}/checkout           -> Valider commande

ACTIONS
- Click checkbox -> Toggle sélection domaine
- Click dropdown durée -> Modifier durée
- Click "Annuler" -> Fermer modal
- Click "Commander" -> Créer panier + checkout
