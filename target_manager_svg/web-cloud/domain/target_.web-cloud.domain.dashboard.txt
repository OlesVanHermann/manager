STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - Grille 2x2 de tuiles informatives

TUILES (2x2)
| Tuile                    | Contenu                              |
|--------------------------|--------------------------------------|
| Informations generales   | Statut, DNS, dates, protection       |
| Securite                 | Transfert, DNSSEC, AUTH, WHOIS       |
| Hebergement Web/Email    | Hosting associe, multisites          |
| Resume des options       | DNS Anycast, date effet              |

TUILE 1: INFORMATIONS GENERALES
| Champ             | Type    | Valeur exemple    |
|-------------------|---------|-------------------|
| Statut            | Badge   | Actif             |
| Serveurs DNS      | Badge   | Actifs            |
| Date creation     | Text    | 15/01/2020        |
| Date expiration   | Text    | 15/01/2025        |
| Protection        | Badge   | Activee           |
| DNSSEC            | Badge   | Active            |

TUILE 2: SECURITE
| Element           | Statut  | Action            |
|-------------------|---------|-------------------|
| Protection transf | Activee | Modifier          |
| DNSSEC            | Active  | Modifier          |
| Code AUTH/INFO    | -       | Afficher          |
| Affichage WHOIS   | -       | Configurer        |
| Adresses Anti-spam| -       | Gerer             |

API ENDPOINTS
GET  /domain/{domain}                  -> Details domaine
GET  /domain/{domain}/serviceInfos     -> Infos facturation
GET  /domain/{domain}/dnssec           -> Etat DNSSEC
GET  /domain/{domain}/option           -> Options actives
