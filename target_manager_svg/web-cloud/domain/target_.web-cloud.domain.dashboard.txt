# Cahier des Charges : web-cloud/domain - Dashboard

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard principal du domaine
- **Type** : Page avec grille de tuiles
- **Objectif** : Vue d'ensemble du domaine et accès rapide aux fonctionnalités

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud
- Sidebar navigation avec liste domaines
- Breadcrumb
- Titre domaine avec badge statut
- Grille tuiles (2x2)

## 3. COMPOSANTS UI
- **OdsCard** : 4 tuiles informatives
- **OdsBadge** : Statut (Actif, Activé, Activée)
- **OdsLink** : Actions (Modifier, Afficher, Configurer, Gérer, Résilier)

## 4. SIDEBAR
- WEB CLOUD
- **Noms de domaine** (sélectionné)
  - example.com (actif)
  - monsite.fr
- Hébergements
- E-mails
