# TYPES DE REDIRECTIONS
- Redirection visible: L'URL change dans le navigateur
  - Permanente (301): Pour un changement définitif
  - Temporaire (302): Pour un changement temporaire
- Redirection invisible (iframe): L'URL reste la même
- vers un domaine (CNAME): Alias DNS
- vers un serveur (A/AAAA): Pointe vers une IP

# ACTIONS
- + Ajouter une redirection → modal-redirection-add
- Exporter CSV → Télécharge la liste des redirections
- ⋮ → Modifier / Supprimer
