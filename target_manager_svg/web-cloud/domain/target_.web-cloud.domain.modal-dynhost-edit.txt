# Cahier des Charges : web-cloud/domain - Modal Modifier DynHost

## 1. VUE D'ENSEMBLE
- **Modal** : Modification d'une entrée DynHost
- **Type** : Formulaire de modification
- **Objectif** : Éditer un enregistrement DynHost existant

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Description
- Indication champs obligatoires
- Champ sous-domaine
- Champ IP
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x400)
- **OdsInput** : Sous-domaine, IP
- **OdsButton** : Annuler, Valider

## 4. TITRE
"Modifier un DynHost"

## 5. DESCRIPTION
"Modifier une entrée DynHost dans la zone DNS"

## 6. INDICATION OBLIGATOIRE
"* Champs obligatoires"

## 7. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine :" |
| Input | "www" |
| Suffixe (addon gris) | ".exemple.com" |

## 8. CHAMP IP
| Élément | Valeur |
|---------|--------|
| Label | "Entrez l'IP actuelle de la destination :" |
| Obligatoire | Oui (*) |
| Input | "192.168.1.100" |

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Valider (primaire)

## 10. APIS & ENDPOINTS
- PUT /domain/zone/{zoneName}/dynHost/record/{id} : Modifier DynHost
