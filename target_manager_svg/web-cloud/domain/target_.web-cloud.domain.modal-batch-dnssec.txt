STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière explicative DNSSEC
3. Dropdown action (Activer / Désactiver)
4. Table domaines sélectionnés:
   - Checkbox
   - Domaine
   - Statut DNSSEC actuel
5. Avertissement (DNS compatible)
6. Footer: Annuler | Appliquer (N)

ACTIONS DNSSEC
| Action     | Effet                                    |
|------------|------------------------------------------|
| Activer    | Active DNSSEC et génère les clés DS     |
| Désactiver | Supprime les clés DS et désactive       |

PRÉ-REQUIS ACTIVATION
- Serveurs DNS supportant DNSSEC
- Pour DNS OVH: configuration automatique
- Pour DNS externes: clés DS à déclarer manuellement

API ENDPOINTS
PUT /domain/{domain}
  Body: { "dnssecSupported": true | false }

Note: L'API est appelée pour chaque domaine sélectionné

ACTIONS
- Click checkbox -> Toggle sélection domaine
- Click dropdown -> Choisir Activer/Désactiver
- Click "Annuler" -> Fermer modal
- Click "Appliquer" -> Exécuter l'opération batch
