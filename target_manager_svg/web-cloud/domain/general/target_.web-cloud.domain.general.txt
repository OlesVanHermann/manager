# STRUCTURE PAGE
1. LEFT PANEL - Domaine + NAV3 (10 onglets)
2. RIGHT PANEL:
   - Titre "Informations générales"
   - BLOC Configuration: nom, statut, dates, renouvellement, contacts
   - BLOC Protection: Owo, Transfer Lock, DNSSEC (toggles)
   - BLOC Actions: Transférer, Changer propriétaire, Résilier

# API ENDPOINTS
GET /domain/{domainName} - Informations générales
PUT /domain/{domainName} - Modifier configuration
PUT /domain/{domainName}/owo - Activer/Désactiver Owo
PUT /domain/{domainName}/lock - Activer/Désactiver transfert lock
