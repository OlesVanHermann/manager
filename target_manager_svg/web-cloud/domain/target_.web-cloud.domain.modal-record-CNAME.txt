# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement CNAME

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type CNAME
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement CNAME (alias) dans la zone DNS

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge CNAME et bouton fermer (✕)
- Champ sous-domaine
- Champ TTL
- Champ cible
- Aide contextuelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x420)
- **OdsInput** : Sous-domaine, TTL, Cible
- **OdsBadge** : Type CNAME (bleu)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - CNAME" avec badge [CNAME]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "blog" |
| Suffixe (addon gris) | ".exemple.com" |

## 6. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL" |
| Input | 3600 |
| Unité | "secondes" |

## 7. CHAMP CIBLE
| Élément | Valeur |
|---------|--------|
| Label | "Cible" |
| Obligatoire | Oui (*) |
| Input | "www.exemple.com." |
| Aide | "La cible doit être un nom d'hôte valide (FQDN terminé par un point)" |
| Icône | ? (tooltip) |

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement CNAME
  - fieldType: "CNAME"
  - subDomain: string
  - target: FQDN
  - ttl: number
