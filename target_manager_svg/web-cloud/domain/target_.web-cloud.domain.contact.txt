3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - NAV3: 10 tabs (Contacts actif)
   - Title + Description
   - Toolbar: Reattribuer les contacts
   - 4 Contact Cards (2x2 grid)

TOOLBAR ACTIONS
| Bouton                   | Action                          |
|--------------------------|---------------------------------|
| Reattribuer les contacts | Modal changement contacts       |

| Technique     | NIC, Nom, Email                        |

| Role          | Description                                    |

API ENDPOINTS
