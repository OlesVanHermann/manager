# STRUCTURE MODAL
1. Header: Titre avec nom domaine + fermer
2. Warning banner: Avertissement irréversible
3. Type transfert: 2 cards radio
   - Sortant (vers autre registrar)
   - Entre comptes OVH
4. AuthCode: Champ lecture seule + bouton Copier
5. Checklist: 3 vérifications pré-transfert
6. Footer: Annuler + Obtenir le code (rouge)

# DIMENSIONS
- Largeur: 700px
- Hauteur: 560px
- Position: Centrée

# PRÉREQUIS TRANSFERT
- Domaine > 60 jours d'ancienneté
- Transfer Lock désactivé
- Pas en période de rédemption
- Pas de litige en cours

# API ENDPOINTS
GET /domain/{domainName}/authInfo - Obtenir code transfert
POST /domain/{domainName}/transfer/unlock - Débloquer transfert
