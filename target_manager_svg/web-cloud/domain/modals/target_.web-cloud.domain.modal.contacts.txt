# STRUCTURE MODAL
1. Header: Titre dynamique selon type contact + fermer
2. Contact actuel: Affichage NIC + nom + email
3. Nouveau NIC: Input pour saisir NIC handle
4. Séparateur "ou"
5. Bouton créer nouveau contact
6. Info banner: Confirmation par email
7. Footer: Annuler + Modifier

# DIMENSIONS
- Largeur: 600px
- Hauteur: 450px
- Position: Centrée

# TYPES DE CONTACTS MODIFIABLES
- Administrateur (nicAdmin)
- Technique (nicTech)
- Facturation (nicBilling)
- Propriétaire: modal différent (changeOwner)

# API ENDPOINTS
PUT /domain/{domainName}/contacts - Modifier contacts
POST /me/contact - Créer nouveau contact
