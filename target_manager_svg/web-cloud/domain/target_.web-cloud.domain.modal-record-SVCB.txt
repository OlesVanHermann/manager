# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement SVCB

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type SVCB
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement SVCB (Service Binding)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge SVCB et bouton fermer (✕)
- Champ sous-domaine
- Champs TTL et Priorité
- Champ cible
- Champ paramètres de service
- Aide contextuelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x520)
- **OdsInput** : Sous-domaine, TTL, Priorité, Cible
- **OdsTextarea** : Paramètres de service
- **OdsBadge** : Type SVCB (vert)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - SVCB" avec badge [SVCB]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "_dns" |
| Suffixe (addon gris) | ".exemple.com" |

## 6. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL" |
| Input | 3600 |

## 7. CHAMP PRIORITÉ
| Élément | Valeur |
|---------|--------|
| Label | "Priorité (SvcPriority)" |
| Obligatoire | Oui (*) |
| Input | 1 |
| Aide | "0=alias" |

## 8. CHAMP CIBLE
| Élément | Valeur |
|---------|--------|
| Label | "Cible (TargetName)" |
| Obligatoire | Oui (*) |
| Input | "dns.exemple.com." |

## 9. CHAMP PARAMÈTRES DE SERVICE
| Élément | Valeur |
|---------|--------|
| Label | "Paramètres de service (SvcParams)" |
| Type | Textarea |
| Valeur exemple | alpn="dot,doq" port=853 ipv4hint=192.0.2.1 ipv6hint=2001:db8::1 |

## 10. AIDE CONTEXTUELLE
- "SVCB (Service Binding) permet de publier des informations sur les services"
- "disponibles pour un nom de domaine (ports, protocoles, adresses IP alternatives)."

## 11. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 12. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement SVCB
  - fieldType: "SVCB"
  - subDomain: string
  - target: "priority target svcparams"
  - ttl: number
