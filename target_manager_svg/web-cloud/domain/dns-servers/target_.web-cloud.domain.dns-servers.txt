# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Domaines et DNS actif
3. LEFT PANEL - Liste domaines (280px)
4. RIGHT PANEL (1096px)
   - Header: Serveurs DNS
   - NAV3: 10 tabs, Serveurs DNS actif
   - TOOLBAR: Modifier les serveurs, Réinitialiser vers OVH
   - TABLE: Serveur DNS, IP associée, Statut, Actions
   - INFO BANNER: Informations serveurs actuels
   - HELP LINK

# TYPES DE SERVEURS
- OVH (hosted): dns*.ovh.net, ns*.ovh.net
- Externes: Serveurs tiers configurés par l'utilisateur

# ACTIONS
- Modifier les serveurs DNS → modal-dns-modify
- Réinitialiser vers OVH → Restaure les DNS OVH par défaut
- ⋮ → Supprimer (si plus de 2 serveurs)

# STATUTS
- Actif (vert): Serveur répond correctement
- Erreur (rouge): Serveur ne répond pas
- En cours (gris): Propagation DNS en cours

# API ENDPOINTS
GET /domain/{domainName}/nameServer
GET /domain/{domainName}/nameServer/status
POST /domain/{domainName}/nameServer
PUT /domain/{domainName}/nameServer
DELETE /domain/{domainName}/nameServer/{id}
