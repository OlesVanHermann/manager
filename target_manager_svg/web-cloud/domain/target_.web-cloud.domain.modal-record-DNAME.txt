# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement DNAME

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type DNAME
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement DNAME (alias de zone)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge DNAME et bouton fermer (✕)
- Champ sous-domaine
- Champ TTL
- Champ cible
- Aide contextuelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x420)
- **OdsInput** : Sous-domaine, TTL, Cible
- **OdsBadge** : Type DNAME (rouge)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - DNAME" avec badge [DNAME]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "alias" |
| Suffixe (addon gris) | ".exemple.com" |

## 6. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL" |
| Input | 3600 |
| Unité | "secondes" |

## 7. CHAMP CIBLE
| Élément | Valeur |
|---------|--------|
| Label | "Cible" |
| Obligatoire | Oui (*) |
| Input | "autre-domaine.com." |
| Aide | "DNAME redirige tout le sous-arbre DNS vers un autre domaine (alias de zone)" |
| Icône | ? (tooltip) |

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 9. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement DNAME
  - fieldType: "DNAME"
  - subDomain: string
  - target: FQDN
  - ttl: number
