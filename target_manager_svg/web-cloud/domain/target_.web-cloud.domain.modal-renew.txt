# Cahier des Charges : web-cloud/domain - Modal Renouveler domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Renouvellement d'un nom de domaine
- **Type** : Modal de confirmation avec info
- **Objectif** : Générer un bon de commande pour le renouvellement

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation
- Description du processus
- Bandeau info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x380)
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Commander

## 4. TITRE
"Renouveler le nom de domaine"

## 5. QUESTION
"Voulez-vous renouveler le nom de domaine **example.com** ?"

## 6. DESCRIPTION
- "Cette action va générer un bon de commande."
- "Le renouvellement sera effectif après paiement du bon de commande."

## 7. BANDEAU INFO
Fond bleu (#E8F0FE), bordure bleue :
- "ℹ️ Le renouvellement prend effet à la date d'expiration actuelle du domaine."
- "La nouvelle date d'expiration sera calculée à partir de cette date."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Commander (primaire)

## 9. APIS & ENDPOINTS
- POST /order/domain/{serviceName}/renew : Générer bon de commande renouvellement
