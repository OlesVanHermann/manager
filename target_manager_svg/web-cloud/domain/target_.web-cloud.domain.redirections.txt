2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - NAV3: 10 tabs (Redirections actif)
   - Toolbar: Ajouter, Exporter CSV
   - Filtre sous-domaine
   - Table redirections
   - Pagination
   - Info box types

API ENDPOINTS
GET    /domain/zone/{zone}/redirection              -> Liste redirections
GET    /domain/zone/{zone}/redirection/{id}         -> Details redirection
POST   /domain/zone/{zone}/redirection              -> Ajouter redirection
PUT    /domain/zone/{zone}/redirection/{id}         -> Modifier redirection
DELETE /domain/zone/{zone}/redirection/{id}         -> Supprimer redirection
