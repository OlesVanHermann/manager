STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Sous-menu (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: domainName
   - NAV3: 10 tabs (Redirections actif)
   - Toolbar: Ajouter + Refresh
   - Table: SOURCE, TYPE, DESTINATION, TITRE, ACTIONS
   - Bannière info

API ENDPOINTS
GET    /domain/{domain}/redirection          -> Liste redirections
GET    /domain/{domain}/redirection/{id}     -> Détails redirection
POST   /domain/{domain}/redirection          -> Créer redirection
PUT    /domain/{domain}/redirection/{id}     -> Modifier redirection
DELETE /domain/{domain}/redirection/{id}     -> Supprimer redirection

TYPES REDIRECTION
| Type          | Code | Description                           |
|---------------|------|---------------------------------------|
| Permanent     | 301  | Redirection SEO-friendly              |
| Temporaire    | 302  | Redirection temporaire                |
| Frame         | -    | Affiche dans iframe, URL visible      |

NAVIGATION
- Click "+ Ajouter" -> Modal création
- Click ✎ -> Modal modification
- Click × -> Modal confirmation suppression
