# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement SRV

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type SRV
- **Type** : Formulaire de création
- **Objectif** : Ajouter un enregistrement SRV (Service)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge SRV et bouton fermer (✕)
- Champ sous-domaine
- Champ TTL
- Champs Priorité et Poids
- Champ Port
- Champ Cible
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x550)
- **OdsInput** : Multiples champs
- **OdsBadge** : Type SRV (rouge)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - SRV" avec badge [SRV]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "_sip._tcp" |
| Suffixe (addon gris) | ".exemple.com" |

| Élément | Valeur |

## 11. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 12. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement SRV
  - fieldType: "SRV"
  - subDomain: "_service._protocol"
  - target: "priority weight port target"
  - ttl: number
