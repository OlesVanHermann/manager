# Cahier des Charges : web-cloud/domain - Modal Modifier zone texte

## 1. VUE D'ENSEMBLE
- **Modal** : Modification de la zone DNS en mode textuel
- **Type** : Éditeur de texte
- **Objectif** : Éditer la zone DNS au format BIND

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Label et description
- Zone de texte avec format BIND
- Avertissement
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (750x600)
- **OdsTextarea** : Zone DNS (font monospace)
- **OdsButton** : Annuler, Valider

## 4. TITRE
"Modifier en mode textuel"

## 5. DESCRIPTION
- Label : "Zone DNS en format texte"
- Aide : "Modifiez directement le contenu de votre zone DNS au format BIND."

## 6. ZONE TEXTAREA
| Élément | Valeur |
|---------|--------|
| Type | Textarea (702x380) |
| Police | Monospace (Consolas, Monaco) |
| Contenu exemple | Zone BIND complète avec SOA, NS, A, CNAME, MX |

## 7. CONTENU EXEMPLE
```
$TTL 3600
@    IN    SOA   dns.example.com. hostmaster.example.com. (
              2024011501 ; serial
              86400      ; refresh
              3600       ; retry
              604800     ; expire
              3600 )     ; minimum

@    IN    NS    dns.example.com.
@    IN    A     203.0.113.50
www  IN    CNAME @
@    IN    MX    10 mx1.mail.ovh.net.
```

## 8. AVERTISSEMENT
⚠️ "La zone doit commencer par $TTL suivi d'une valeur numérique."

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Valider (primaire)

## 10. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/import : Importer zone au format texte
