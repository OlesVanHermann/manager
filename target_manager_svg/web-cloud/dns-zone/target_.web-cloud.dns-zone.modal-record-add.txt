# Cahier des Charges : web-cloud/dns-zone - Modal Ajouter enregistrement

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un enregistrement DNS (variante compacte)
- **Type** : Modal formulaire
- **Objectif** : Créer un nouvel enregistrement DNS

## 2. STRUCTURE MODAL
- Header gris avec titre et bouton fermer (×)
- Select type d'enregistrement
- Champ sous-domaine + suffixe
- Champ cible
- Champ TTL avec suffixe
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsSelect** : Type d'enregistrement
- **OdsInput** : Sous-domaine, Cible, TTL
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un enregistrement DNS"

## 5. CHAMPS
### Type d'enregistrement *
- Type: Select obligatoire
- Valeur: "A"
- Options: A, AAAA, CNAME, MX, TXT, SRV, NS, CAA...

### Sous-domaine
- Type: Input
- Placeholder: "www"
- Suffixe: ".domain.com"

### Cible *
- Type: Input obligatoire
- Placeholder: "192.168.1.1"

### TTL
- Type: Select/Input
- Valeur: "3600"
- Suffixe: "secondes"

## 6. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Ajouter | Primaire bleu | Crée l'enregistrement |

## 7. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record
  - fieldType: string
  - subDomain: string
  - target: string
  - ttl: number
