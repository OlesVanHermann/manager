# Cahier des Charges : web-cloud/domain - Modal Résilier DNS Anycast

## 1. VUE D'ENSEMBLE
- **Modal** : Résiliation de l'option DNS Anycast
- **Type** : Modal de confirmation danger
- **Objectif** : Résilier l'option DNS Anycast

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Question de confirmation
- Description fonctionnalité
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x340)
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Résilier (danger rouge)

## 4. TITRE
"Résilier DNS Anycast"

## 5. QUESTION
"Voulez-vous vraiment résilier l'option DNS Anycast pour le domaine **example.com** ?"

## 6. DESCRIPTION
"L'option DNS Anycast permet d'accélérer le fonctionnement des serveurs DNS pour fournir des temps de réponse plus courts dans le monde entier."

## 7. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
- "⚠️ La résiliation prendra effet à la fin de la période de facturation."
- "Vos DNS reviendront en mode classique."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Résilier (danger rouge)

## 9. APIS & ENDPOINTS
- DELETE /domain/{serviceName}/option/anycast : Résilier Anycast
