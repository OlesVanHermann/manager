# Cahier des Charges : web-cloud/domain - Modal Rattacher domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Rattachement d'un sous-domaine
- **Type** : Modal formulaire simple
- **Objectif** : Rattacher un sous-domaine à un domaine principal

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Description
- Select dropdown
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur avec overlay
- **OdsSelect** : Sélection du sous-domaine
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Rattacher un domaine"

## 5. DESCRIPTION
"Veuillez sélectionner le domaine à rattacher au domaine **example.com**"

## 6. SELECT DOMAINE
- Label : "Sélectionner un domaine rattaché"
- Valeur exemple : "sub.example.com"
- Dropdown avec flèche

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Confirmer (primaire)

## 8. APIS & ENDPOINTS
- POST /domain/{serviceName}/attach : Rattacher domaine
