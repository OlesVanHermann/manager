# STRUCTURE PAGE
1. LEFT PANEL - Domaine + NAV3 (Zone DNS active)
2. RIGHT PANEL:
   - Breadcrumb: Zone DNS > Historique
   - Titre "Historique des modifications"
   - TOOLBAR: Refresh, Filtre période
   - TIMELINE: Entrées chronologiques avec:
     - Point bleu (récent) ou gris (ancien)
     - Action (Ajout/Modification/Suppression)
     - Détails de l'enregistrement
     - Auteur (NIC handle)
     - Date/heure
     - Bouton Restaurer

# API ENDPOINTS
GET /domain/zone/{zoneName}/history - Historique modifications
POST /domain/zone/{zoneName}/history/{id}/restore - Restaurer état
