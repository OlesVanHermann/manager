# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Domaines et DNS actif
3. LEFT PANEL - Liste domaines (280px)
4. RIGHT PANEL (1096px)
   - Header: Zone DNS de {domainName}
   - NAV3: 10 tabs, Zone DNS actif
   - TOOLBAR: + Ajouter, Mode textuel, TTL, Reset, Historique
   - SEARCH + FILTER par type
   - TABLE: Sous-domaine, Type (badge couleur), TTL, Cible, Actions

- AAAA: #0050D7 (bleu)
- CNAME: #9333EA (violet)
- MX: #059669 (vert)
- TXT: #6B7280 (gris)
- NS: #DC2626 (rouge)
- SRV: #F59E0B (orange)
- CAA: #14B8A6 (teal)
- DKIM: #8B5CF6 (violet clair)
