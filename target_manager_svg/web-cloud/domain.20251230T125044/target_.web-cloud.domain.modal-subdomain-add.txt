# Cahier des Charges : web-cloud/domain - Modal Ajouter sous-domaine

## 1. VUE D'ENSEMBLE
- **Modal** : Assistant d'ajout d'un sous-domaine
- **Type** : Formulaire avec choix multiples
- **Objectif** : Choisir l'action pour un sous-domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Champ sous-domaine avec suffixe
- Section choix d'action (3 options radio)
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x520)
- **OdsInput** : Sous-domaine avec addon
- **OdsRadio** : 3 options d'action
- **OdsButton** : Annuler, Suivant

## 4. TITRE
"Ajouter un sous-domaine"

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "sous-domaine" |
| Suffixe (addon gris) | ".example.com" |
| Icône aide | ⓘ |
| Aide | "Si vous laissez ce champ vide, la configuration s'appliquera directement au nom de domaine" |

## 6. SECTION CHOIX "JE VEUX :"

### Option 1 : Redirection
| Élément | Valeur |
|---------|--------|
| Radio | ○ |
| Label | "Mettre en place une redirection" |
| Description | "Exemple : test.montest.com redirige vers test.montest.fr" |

### Option 2 : Entrée DNS (sélectionnée)
| Élément | Valeur |
|---------|--------|
| Radio | ● (sélectionné) |
| Label | "Ajouter une entrée dans la zone DNS (A, CNAME, MX, …)" |
| Description | "Exemple : test.montest.com 3600 IN A 1.2.3.4" |

### Option 3 : Hébergement mutualisé
| Élément | Valeur |
|---------|--------|
| Radio | ○ |
| Label | "Ajouter un domaine, sous-domaine à mon hébergement mutualisé" |

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Suivant (primaire)

## 8. COMPORTEMENT
- Selon le choix, "Suivant" ouvre :
  - Option 1 → Modal redirection
  - Option 2 → Modal type d'enregistrement DNS
  - Option 3 → Modal hébergement

## 9. APIS & ENDPOINTS
- Pas d'API directe, navigation vers le formulaire correspondant
