STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - NAV3: 10 tabs (Serveurs DNS actif)
   - Title + Description
   - Warning banner (si moins de 2 serveurs)
   - Toolbar: Ajouter un serveur
   - Table serveurs DNS
   - Info block

WARNING BANNER
Condition: Affiche si moins de 2 serveurs DNS actifs
Message: Il est fortement deconseille de n'avoir qu'un seul serveur DNS.

TABLE COLONNES
| Colonne     | Description                   |
|-------------|-------------------------------|
| Serveur DNS | Hostname du serveur           |
| Adresse IP  | IP du serveur                 |
| Statut      | Badge Actif/Inactif           |
| Actions     | Menu contextuel (⋮)           |

MENU ACTIONS (⋮)
| Action    | Description              |
|-----------|--------------------------|
| Modifier  | Modal edition serveur    |
| Supprimer | Modal confirmation       |

API ENDPOINTS
GET    /domain/{serviceName}/nameServer       -> Liste serveurs DNS
POST   /domain/{serviceName}/nameServer       -> Ajouter serveur
PUT    /domain/{serviceName}/nameServer/{id}  -> Modifier serveur
DELETE /domain/{serviceName}/nameServer/{id}  -> Supprimer serveur
