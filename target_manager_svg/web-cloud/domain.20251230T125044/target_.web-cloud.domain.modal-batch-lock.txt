STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière explicative
3. Dropdown action (Verrouiller / Déverrouiller)
4. Table domaines sélectionnés:
   - Checkbox
   - Domaine
   - Statut actuel
5. Résumé opération (avertissement)
6. Footer: Annuler | Appliquer (N)

STATUTS TRANSFERT
| Statut        | Badge         | Description                    |
|---------------|---------------|--------------------------------|
| locked        | #DEF7EC       | Transfert bloqué               |
| unlocked      | #FEF3C7       | Transfert possible             |
| locking       | #DBEAFE       | Verrouillage en cours          |
| unlocking     | #DBEAFE       | Déverrouillage en cours        |

CAS D'EXCLUSION
- Domaines avec DNS externes (non gérés par OVH)
- Extensions ne supportant pas le verrouillage
- Domaines en cours de transfert

API ENDPOINTS
PUT /domain/{domain}
  Body: { "transferLockStatus": "locked" | "unlocked" }

Note: L'API est appelée pour chaque domaine sélectionné

ACTIONS
- Click checkbox -> Toggle sélection domaine
- Click dropdown -> Choisir Verrouiller/Déverrouiller
- Click "Annuler" -> Fermer modal
- Click "Appliquer" -> Exécuter l'opération batch
