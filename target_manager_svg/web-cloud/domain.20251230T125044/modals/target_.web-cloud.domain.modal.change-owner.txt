# STRUCTURE MODAL
1. Header: Titre avec nom domaine + fermer
2. Warning banner: Procédure officielle
3. Propriétaire actuel: Affichage NIC + nom + email
4. Type nouveau proprio: 3 cards (Particulier/Entreprise/Association)
5. NIC Handle: Input ou bouton créer contact
6. Justificatifs: Zone drag & drop fichiers
7. Footer: Annuler + Initier la procédure

# DIMENSIONS
- Largeur: 700px
- Hauteur: 600px
- Position: Centrée

# TYPES DE PROPRIÉTAIRE
| Type         | Justificatifs possibles        |
|--------------|--------------------------------|
| Particulier  | Pièce d'identité               |
| Entreprise   | KBIS, statuts                  |
| Association  | Récépissé, statuts             |

# API ENDPOINTS
POST /domain/{domainName}/changeOwner - Initier changement propriétaire
