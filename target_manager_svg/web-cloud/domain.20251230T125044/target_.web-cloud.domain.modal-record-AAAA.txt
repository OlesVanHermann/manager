DESCRIPTION
-----------
Modal de création d'un enregistrement DNS de type AAAA.
Enregistrement AAAA = Associe un nom de domaine à une adresse IPv6.

⚠️ DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                          |
|--------------------------|-------------------------------------|
| {domain}                 | Contexte - domaine sélectionné      |
| Sous-domaine             | Input utilisateur                   |
| TTL (3600)               | Input - défaut 3600 secondes        |
| Adresse IPv6             | Input utilisateur (format IPv6)     |

================================================================================

STRUCTURE MODALE
----------------
1. OVERLAY - Fond semi-transparent rgba(0,0,0,0.5)
2. HEADER MODAL - Badge AAAA (#991B1B sur fond #FEE2E2) + titre
3. BANNIÈRE INFO - Explication du type d'enregistrement
4. FORMULAIRE:
   - Sous-domaine (input texte, largeur 350px)
   - TTL (input numérique, largeur 150px, défaut 3600)
   - Adresse IPv6 (input texte, largeur 520px, requis)
5. ACTIONS - Boutons Annuler / Créer

COULEURS SPÉCIFIQUES
--------------------
- Header modal: #FEE2E2 (rouge clair)
- Badge AAAA: #991B1B (rouge foncé)

CHAMPS FORMULAIRE
-----------------
| Champ          | Type   | Requis | Placeholder                        |
|----------------|--------|--------|------------------------------------|
| Sous-domaine   | text   | Non    | @ (racine) ou sous-domaine         |
| TTL            | number | Non    | 3600                               |
| Adresse IPv6   | text   | Oui    | Ex: 2001:0db8:85a3::8a2e:0370:7334 |

API ENDPOINT
------------
POST /domain/zone/{zoneName}/record
Body: {
  fieldType: "AAAA",
  subDomain: string,
  ttl: number,
  target: string (IPv6)
}
