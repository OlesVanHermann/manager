# Cahier des Charges : web-cloud/domain - Modal Ajouter enregistrement NAPTR

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un enregistrement DNS de type NAPTR
- **Type** : Formulaire avancé de création
- **Objectif** : Ajouter un enregistrement NAPTR (réécriture de domaines)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre, badge NAPTR et bouton fermer (✕)
- Champ sous-domaine
- Champs TTL, Ordre, Préférence
- Champs Flags, Service
- Champ Expression régulière
- Champ Remplacement
- Aide contextuelle
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x580)
- **OdsInput** : Multiples champs
- **OdsBadge** : Type NAPTR (violet)
- **OdsButton** : Annuler, Ajouter l'entrée

## 4. TITRE
"Ajouter une entrée DNS - NAPTR" avec badge [NAPTR]

## 5. CHAMP SOUS-DOMAINE
| Élément | Valeur |
|---------|--------|
| Label | "Sous-domaine" |
| Input | "sip" |
| Suffixe (addon gris) | ".exemple.com" |

## 6. CHAMP TTL
| Élément | Valeur |
|---------|--------|
| Label | "TTL" |
| Input | 3600 |

## 7. CHAMP ORDRE
| Élément | Valeur |
|---------|--------|
| Label | "Ordre" |
| Obligatoire | Oui (*) |
| Input | 100 |

## 8. CHAMP PRÉFÉRENCE
| Élément | Valeur |
|---------|--------|
| Label | "Préférence" |
| Obligatoire | Oui (*) |
| Input | 10 |

## 9. CHAMP FLAGS
| Élément | Valeur |
|---------|--------|
| Label | "Flags" |
| Obligatoire | Oui (*) |
| Input | "s" |
| Aide | "s, a, u, p" |

## 10. CHAMP SERVICE
| Élément | Valeur |
|---------|--------|
| Label | "Service" |
| Obligatoire | Oui (*) |
| Input | "SIP+D2U" |

## 11. CHAMP EXPRESSION RÉGULIÈRE
| Élément | Valeur |
|---------|--------|
| Label | "Expression régulière (Regexp)" |
| Input | "!^.*$!sip:info@exemple.com!" |

## 12. CHAMP REMPLACEMENT
| Élément | Valeur |
|---------|--------|
| Label | "Remplacement (Replacement)" |
| Obligatoire | Oui (*) |
| Input | "_sip._udp.exemple.com." |

## 13. AIDE CONTEXTUELLE
- "NAPTR (Naming Authority Pointer) est utilisé pour la réécriture de domaines,"
- "notamment pour les services ENUM et SIP."

## 14. BOUTONS FOOTER
- Annuler (secondaire)
- Ajouter l'entrée (primaire)

## 15. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/record : Créer enregistrement NAPTR
  - fieldType: "NAPTR"
  - subDomain: string
  - target: "order preference flags service regexp replacement"
  - ttl: number
