TYPES D'ENREGISTREMENT (grille 5x2)
| Type  | Description                    | Champs specifiques          |
|-------|--------------------------------|-----------------------------|
| A     | Adresse IPv4                   | Cible: IPv4                 |
| AAAA  | Adresse IPv6                   | Cible: IPv6                 |
| CNAME | Alias vers un autre domaine    | Cible: FQDN                 |
| MX    | Serveur mail                   | Priorite + Serveur          |
| TXT   | Texte libre (SPF, DKIM...)     | Cible: texte                |
| NS    | Serveur DNS                    | Cible: hostname             |
| SRV   | Service                        | Priority/Weight/Port/Target |
| CAA   | Certification Authority        | Flags/Tag/Value             |
| DKIM  | DomainKeys (emails)            | Selecteur + Cle             |
| SPF   | Sender Policy Framework        | Generateur SPF              |

CHAMPS FORMULAIRE
| Champ         | Type   | Validation             | Exemple      |
|---------------|--------|------------------------|--------------|
| Sous-domaine  | text   | Optionnel, @ si vide   | www          |
| TTL           | number | Min: 60, Default: 3600 | 3600         |
| Cible         | text   | Required, format valide| 213.186.33.5 |

BOUTONS
| Bouton   | Style     | Action                    |
|----------|-----------|---------------------------|
| Annuler  | Secondary | Fermer modal sans sauver  |
| Suivant  | Primaire  | Passer a l'etape suivante |

API ENDPOINT
