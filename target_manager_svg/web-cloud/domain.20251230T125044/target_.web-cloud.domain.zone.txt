4. RIGHT PANEL
   - Header: Nom domaine + badge statut

   - Banner info
   - Toolbar: Ajouter, Mode textuel, TTL, Reinitialiser, Historique
   - Recherche + Filtre par type

TOOLBAR ACTIONS
| Bouton            | Action                          |
|-------------------|---------------------------------|
| + Ajouter entree  | Modal ajout record              |
| Mode textuel      | Edition zone format texte       |
| Modifier TTL      | Modal TTL par defaut            |
| Reinitialiser     | Modal reset zone                |
| Historique        | Page historique zone            |

TABLE COLONNES
| Colonne      | Description                     |
|--------------|---------------------------------|
| SOUS-DOMAINE | @ pour racine, sinon prefix     |
| TYPE         | Badge colore (A, AAAA, CNAME, MX, TXT, NS, SRV, CAA) |
| TTL          | Duree en secondes               |
| CIBLE        | Valeur de l'enregistrement      |
| ACTIONS      | Edit, Delete                    |

API ENDPOINTS
GET    /domain/zone/{zone}/record              -> Liste enregistrements
GET    /domain/zone/{zone}/record/{id}         -> Details record
POST   /domain/zone/{zone}/record              -> Ajouter record
PUT    /domain/zone/{zone}/record/{id}         -> Modifier record
DELETE /domain/zone/{zone}/record/{id}         -> Supprimer record
POST   /domain/zone/{zone}/refresh             -> Rafraichir zone
POST   /domain/zone/{zone}/reset               -> Reinitialiser zone
GET    /domain/zone/{zone}/export              -> Exporter zone
POST   /domain/zone/{zone}/import              -> Importer zone
GET    /domain/zone/{zone}/history             -> Historique zone
