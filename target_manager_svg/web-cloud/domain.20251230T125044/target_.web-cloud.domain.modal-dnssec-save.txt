# Cahier des Charges : web-cloud/domain - Modal Configurer DNSSEC

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration DNSSEC
- **Type** : Modal avec choix radio
- **Objectif** : Activer ou désactiver DNSSEC sur le domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Description avec nom de domaine
- Boutons radio (activer/désactiver)
- Bandeau info
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x380)
- **OdsRadio** : Choix activer/désactiver
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Annuler, Confirmer

## 4. TITRE
"Configurer DNSSEC"

## 5. DESCRIPTION
"Veuillez choisir si DNSSEC doit être activé ou désactivé pour le domaine **example.com**"

## 6. OPTIONS RADIO
| Option | État |
|--------|------|
| ● Activer DNSSEC sur ce domaine | Sélectionné |
| ○ Désactiver DNSSEC sur ce domaine | Non sélectionné |

## 7. BANDEAU INFO
Fond bleu (#E8F0FE), bordure bleue :
"ℹ️ Le protocole DNSSEC sécurise les échanges DNS grâce au mécanisme de la signature cryptographique asymétrique."

## 8. BOUTONS FOOTER
- Annuler (secondaire)
- Confirmer (primaire)

## 9. APIS & ENDPOINTS
- PUT /domain/{serviceName}/dnssec : Activer/désactiver DNSSEC
