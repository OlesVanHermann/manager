# STRUCTURE PAGE
1. LEFT PANEL - Domaine + NAV3 (AllDom = dernier onglet)
2. RIGHT PANEL:
   - Titre "AllDom - Gestion multi-extensions"
   - INFO BANNER: Explication AllDom
   - TABLE avec checkboxes: Domaine, Statut, Expiration, Prix
   - ACTIONS groupées: Renouveler, Modifier DNS, Modifier contacts
   - TOTAL: Prix de renouvellement de la sélection

# FONCTIONNALITÉ
AllDom permet de:
- Voir toutes les extensions d'un même nom
- Sélectionner plusieurs domaines
- Effectuer des actions groupées (renouvellement, DNS, contacts)

# API ENDPOINTS
GET /allDom - Liste des offres AllDom
GET /allDom/{allDom}/domain - Domaines liés à une offre AllDom
POST /allDom/{allDom}/renew - Renouveler les domaines sélectionnés
