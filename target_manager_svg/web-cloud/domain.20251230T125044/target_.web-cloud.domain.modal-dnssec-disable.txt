STRUCTURE MODAL
1. Header: Titre avec icône warning + bouton fermer
2. Warning box (rouge) expliquant les risques
3. Nom du domaine concerné
4. Checkbox de confirmation obligatoire
5. Footer: Annuler | Désactiver (rouge)

COMPORTEMENT
- Le bouton "Désactiver" est disabled tant que la checkbox n'est pas cochée
- Après confirmation, le DNSSEC est désactivé et les clés DS supprimées

API ENDPOINTS
DELETE /domain/{domain}/dsRecord/{id}     -> Supprimer clé DS
PUT    /domain/{domain}                   -> Désactiver DNSSEC
  Body: { "dnssecSupported": false }

ACTIONS
- Click checkbox -> Active le bouton Désactiver
- Click "Annuler" -> Fermer modal
- Click "Désactiver" -> Exécuter la désactivation
