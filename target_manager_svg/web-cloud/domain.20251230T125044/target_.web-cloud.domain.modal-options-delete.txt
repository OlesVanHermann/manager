# Cahier des Charges : web-cloud/domain - Modal Supprimer option

## 1. VUE D'ENSEMBLE
- **Modal** : Suppression d'une option de domaine
- **Type** : Modal de confirmation danger
- **Objectif** : Supprimer une option (DNS Anycast, etc.)

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Question de confirmation
- Bandeau warning
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur compact (550x300)
- **OdsMessage** : Bandeau warning jaune
- **OdsButton** : Annuler, Supprimer (danger rouge)

## 4. TITRE
"Supprimer une option"

## 5. QUESTION
"Êtes-vous sûr de vouloir supprimer l'option **DNS Anycast** pour le domaine **exemple.com** ?"

## 6. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
"⚠ Cette action est irréversible. L'option ne sera plus active après la suppression."

## 7. BOUTONS FOOTER
- Annuler (secondaire)
- Supprimer (danger rouge)

## 8. APIS & ENDPOINTS
- DELETE /domain/{serviceName}/option/{optionId} : Supprimer option
