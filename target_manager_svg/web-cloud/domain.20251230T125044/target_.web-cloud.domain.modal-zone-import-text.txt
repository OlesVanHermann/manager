# Cahier des Charges : web-cloud/domain - Modal Modifier zone texte (détaillé)

## 1. VUE D'ENSEMBLE
- **Modal** : Modification de la zone DNS en mode textuel
- **Type** : Éditeur avancé avec alertes
- **Objectif** : Éditer la zone DNS au format BIND avec validation

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre et bouton fermer (✕)
- Bandeau info explicatif
- Label "Contenu de la zone DNS"
- Zone de texte avec fond gris
- Bandeau warning danger
- Footer avec boutons

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x570)
- **OdsMessage** : Bandeau info bleu, Bandeau warning jaune
- **OdsTextarea** : Zone DNS (fond gris, font monospace)
- **OdsButton** : Annuler, Valider les modifications

## 4. TITRE
"Modifier en mode textuel"

## 5. BANDEAU INFO
Fond bleu (#DBEAFE) :
- "Ce mode vous permet d'éditer votre zone DNS au format texte standard BIND."
- "Les modifications seront validées avant d'être appliquées."

## 6. ZONE TEXTAREA
| Élément | Valeur |
|---------|--------|
| Label | "Contenu de la zone DNS" |
| Type | Textarea (660x250) |
| Fond | Gris (#F3F4F6) |
| Police | Monospace |

## 7. CONTENU EXEMPLE
```
$TTL 3600
@    IN    SOA   dns.exemple.com. admin.exemple.com. (
                 2024010101 ; Serial
                 86400      ; Refresh
                 7200       ; Retry
                 3600000    ; Expire
                 86400 )    ; Minimum TTL
@    IN    NS    dns.exemple.com.
@    IN    A     192.168.1.1
www  IN    CNAME @
mail IN    A     192.168.1.2
@    IN    MX    10 mail.exemple.com.
```

## 8. BANDEAU WARNING
Fond jaune (#FEF3C7), bordure orange :
- "⚠ Attention : Cette action remplacera l'intégralité de votre zone DNS."
- "Assurez-vous de vérifier la syntaxe avant de valider. Une erreur peut rendre votre domaine inaccessible."

## 9. BOUTONS FOOTER
- Annuler (secondaire)
- Valider les modifications (primaire)

## 10. APIS & ENDPOINTS
- POST /domain/zone/{zoneName}/import : Importer zone au format texte
- GET /domain/zone/{zoneName}/export : Exporter zone actuelle
