STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Domaines et DNS actif)
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: Nom domaine + badge statut
   - NAV3: 10 tabs (Redirections actif)
   - Title + Description
   - Toolbar: Ajouter, Exporter, Filtre
   - Table redirections
   - Pagination

TOOLBAR ACTIONS
| Bouton                 | Style    | Action                    |
|------------------------|----------|---------------------------|
| + Ajouter une redirection| Primaire| Modal ajout redirection  |
| Exporter CSV           | Outline  | Telecharger fichier CSV   |

TABLE COLONNES
| Colonne    | Description                              |
|------------|------------------------------------------|
| Domaine    | Sous-domaine source                      |
| Type       | Type de redirection                      |
| Cible      | URL ou adresse destination               |
| Visibilite | 302, 301, iframe, -                      |
| Actions    | Menu contextuel (⋮)                      |

TYPES DE REDIRECTION
| Type                    | Description                       |
|-------------------------|-----------------------------------|
| Redirection visible     | Redirection HTTP avec changement URL|
| Redirection invisible   | Iframe (URL reste la meme)        |
| vers un domaine (CNAME) | Enregistrement DNS CNAME          |
| vers un serveur (A)     | Enregistrement DNS A              |

VISIBILITE
| Code      | Description              |
|-----------|--------------------------|
| 301       | Redirection permanente   |
| 302       | Redirection temporaire   |
| iframe    | Cadre invisible          |
| -         | Non applicable (DNS)     |

MENU ACTIONS (⋮)
| Action    | Description              |
|-----------|--------------------------|
| Modifier  | Modal edition            |
| Supprimer | Modal confirmation (rouge)|

API ENDPOINTS
GET    /domain/zone/{zoneName}/redirection          -> Liste redirections
GET    /domain/zone/{zoneName}/redirection/{id}     -> Detail redirection
POST   /domain/zone/{zoneName}/redirection          -> Ajouter redirection
PUT    /domain/zone/{zoneName}/redirection/{id}     -> Modifier redirection
DELETE /domain/zone/{zoneName}/redirection/{id}     -> Supprimer redirection
