# STRUCTURE PAGE
1. LEFT PANEL - Service email + NAV3 (Listes actif)
2. RIGHT PANEL:
   - TITRE "Listes de diffusion"
   - TOOLBAR: Refresh, Recherche, "Créer une liste"
   - TABLE: Nom, Abonnés, Modération, Dernière activité, Actions
   - BANNIÈRE INFO

# TYPES MODÉRATION
| Type    | Badge  | Description                          |
|---------|--------|--------------------------------------|
| Ouvert  | Vert   | Tout le monde peut envoyer           |
| Modéré  | Orange | Les messages sont validés            |
| Fermé   | Rouge  | Seuls les modérateurs peuvent poster |

# ACTIONS
- Gérer abonnés → page dédiée
- ✕ Supprimer la liste

# API ENDPOINTS
GET /email/domain/{domain}/mailingList - Liste des ML
POST /email/domain/{domain}/mailingList - Créer ML
DELETE /email/domain/{domain}/mailingList/{name} - Supprimer
