# STRUCTURE MODAL
1. HEADER - "Créer une liste de diffusion" + close
2. FORMULAIRE:
   - Nom de la liste * (input + domaine fixe)
   - Description
   - Modération (3 cards: Ouvert/Modéré/Fermé)
   - Email du propriétaire *
   - Répondre à (radio: liste/expéditeur)
3. BANNIÈRE INFO
4. ACTIONS: Annuler, Créer

# MODES MODÉRATION
| Mode   | Comportement                    |
|--------|----------------------------------|
| Ouvert | Tout le monde peut poster       |
| Modéré | Messages validés par modérateur |
| Fermé  | Seuls les modérateurs postent   |

# API ENDPOINTS
POST /email/domain/{domain}/mailingList - Créer liste
