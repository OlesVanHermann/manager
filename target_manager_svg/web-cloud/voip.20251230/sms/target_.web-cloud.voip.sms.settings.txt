API ENDPOINTS
GET    /sms/{account}                     -> Detail compte
PUT    /sms/{account}                     -> Modifier parametres
GET    /sms/{account}/credentials         -> Cle API
POST   /sms/{account}/credentials/refresh -> Regenerer cle
GET    /sms/{account}/alert               -> Parametres alerte
PUT    /sms/{account}/alert               -> Modifier alerte
DELETE /sms/{account}                     -> Supprimer compte
