STRUCTURE PAGE
1. NAV3: General | SMS envoyes | SMS recus | Campagnes (actif) | Modeles | Expediteurs | Parametres
2. Left panel: Liste comptes SMS
3. Header: Nom compte + button nouvelle campagne
4. Filters: Recherche + statut
5. Table campagnes
6. Pagination

STATUTS CAMPAGNE
| Statut     | Couleur   | Description              |
|------------|-----------|--------------------------|
| Brouillon  | #F3F4F6   | Non encore programme     |
| Programmee | #DBEAFE   | En attente execution     |
| En cours   | #FEF3C7   | Envoi en progression     |
| Terminee   | #DEF7EC   | Tous SMS envoyes         |
| Partiel    | #FEE2E2   | Termine avec echecs      |
| Annulee    | #F3F4F6   | Campagne annulee         |

COLONNES TABLE
| Colonne       | Description                    |
|---------------|--------------------------------|
| Nom           | Nom campagne                   |
| Destinataires | Nombre total                   |
| Programme     | Date/heure envoi               |
| Envoyes       | Nombre envoyes                 |
| Delivres      | Nb delivres + taux             |
| Credits       | Credits consommes              |
| Statut        | Badge etat                     |
| Actions       | Selon statut                   |

ACTIONS PAR STATUT
| Statut     | Actions disponibles            |
|------------|--------------------------------|
| Brouillon  | Editer, Supprimer              |
| Programmee | Modifier, Supprimer            |
| En cours   | Arreter                        |
| Terminee   | Rapport                        |
| Partiel    | Rapport, Relancer echecs       |

CREATION CAMPAGNE (wizard multi-etapes)
| Etape | Champs                           |
|-------|----------------------------------|
| 1     | Nom, Description                 |
| 2     | Message ou Modele                |
| 3     | Destinataires (CSV, liste, API)  |
| 4     | Expediteur                       |
| 5     | Programmation (immediate/date)   |
| 6     | Resume + Confirmation            |

API ENDPOINTS
GET    /sms/{account}/batches                    -> Liste campagnes
POST   /sms/{account}/batches                    -> Creer campagne
GET    /sms/{account}/batches/{id}               -> Detail
PUT    /sms/{account}/batches/{id}               -> Modifier
DELETE /sms/{account}/batches/{id}               -> Supprimer
POST   /sms/{account}/batches/{id}/start         -> Demarrer
POST   /sms/{account}/batches/{id}/stop          -> Arreter
POST   /sms/{account}/batches/{id}/retry         -> Relancer echecs
GET    /sms/{account}/batches/{id}/report        -> Rapport detaille
