TYPES EXPEDITEUR
| Type           | Badge     | Description                |
|----------------|-----------|----------------------------|
| Alphanumerique | #DBEAFE   | Nom (11 car. max)          |
| Numerique      | #F3E8FF   | Numero telephone           |
| Court          | #DEF7EC   | Numero court (5 chiffres)  |

STATUTS
| Statut        | Couleur   | Description              |
|---------------|-----------|--------------------------|
| Valide        | #DEF7EC   | Utilisable               |
| En validation | #FEF3C7   | En cours de validation   |
| Rejete        | #FEE2E2   | Validation refusee       |

COLONNES TABLE
| Colonne     | Description                    |
|-------------|--------------------------------|
| Expediteur  | Nom ou numero                  |
| Type        | Badge type                     |
| Statut      | Badge validation               |
| Description | Texte libre                    |
| Par defaut  | Badge si defaut                |
| Actions     | Selon statut                   |

ACTIONS PAR STATUT
| Statut        | Actions disponibles           |
|---------------|-------------------------------|
| Valide        | Defaut, Modifier, Supprimer   |
| En validation | Annuler                       |
| Rejete        | Relancer, Supprimer           |

FORMULAIRE CREATION (modal)
| Champ       | Type    | Requis | Contraintes            |
|-------------|---------|--------|------------------------|
| Expediteur  | text    | Oui    | 11 car. max (alpha)    |
| Type        | select  | Oui    | Alpha/Numerique        |
| Description | text    | Non    | 100 car.               |
| Justificatif| file    | Non    | Pour alphanumerique    |

API ENDPOINTS
GET    /sms/{account}/senders                -> Liste expediteurs
POST   /sms/{account}/senders                -> Creer expediteur
GET    /sms/{account}/senders/{id}           -> Detail
PUT    /sms/{account}/senders/{id}           -> Modifier
DELETE /sms/{account}/senders/{id}           -> Supprimer
PUT    /sms/{account}/senders/{id}/default   -> Definir par defaut
POST   /sms/{account}/senders/{id}/validate  -> Relancer validation
