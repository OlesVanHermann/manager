STRUCTURE PAGE
1. SOUS-NAV: Lignes | Numeros | Fax (actif) | Abreges
2. Left panel: Liste fax du groupe
3. Right panel header: Numero + description + actions
4. NAV3: General (actif) | Parametres | Notifications | Logo | Fax envoyes | Fax recus | Consommation | Campagnes
5. Cards:
   - Service fax (numero, description, email, statut, qualite)
   - Notifications (reception, echec, confirmation)
   - Abonnement (dates)
   - Statistiques du mois
   - Actions rapides (banniere)
