# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Back link: Retour au téléphone
   - Titre: Informations du téléphone
   - Card info téléphone avec image
   - Table lignes associées
   - Actions rapides

# INFORMATIONS TÉLÉPHONE
- Photo/icône du modèle
- Modèle + badge statut connexion
- Adresse MAC
- Adresse IP
- Version firmware
- Protocole (SIP)
- Lignes disponibles (x/y)
- Dernière synchronisation
- Uptime

| Colonne | Contenu |
|---------|---------|
| Ligne | Numéro de téléphone |
| Description | Label personnalisé |
| Touche | Numéro touche (1-4) |
| Statut | Badge Actif/Inactif |

- Codecs → Page phone-codec
- Touches prog. → Page phone-programmable-keys
