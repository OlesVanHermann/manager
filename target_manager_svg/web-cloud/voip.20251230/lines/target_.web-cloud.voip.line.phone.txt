# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. SOUS-NAV - Lignes actif
4. LEFT PANEL - Liste lignes
5. RIGHT PANEL
   - Header: Numéro + Description
   - NAV3: Téléphone actif
   - Bloc Téléphone associé (infos + actions)
   - Table Touches programmables
   - Bloc Codecs audio

# NAV3 TAB ACTIF
Tab 2: Téléphone (x=64, w=75)

# BLOCS CONTENU
| Bloc                    | Données                              |
|-------------------------|--------------------------------------|
| Téléphone associé       | Marque, modèle, MAC, IP, firmware    |
| Touches programmables   | Liste touches BLF/SpeedDial/Line     |
| Codecs audio            | Ordre préférence G.722, G.711        |

# ACTIONS
| Action            | Type    | API                                  |
|-------------------|---------|--------------------------------------|
| Reboot            | Bouton  | POST /telephony/{ba}/line/{sn}/phone/reboot |
| Configuration     | Bouton  | Navigation vers config               |
| MDP admin         | Bouton  | POST /telephony/{ba}/line/{sn}/phone/changePhoneConfiguration |
| Ajouter touche    | Bouton  | POST /telephony/{ba}/line/{sn}/phone/functionKey |
| ✎ touche          | Icône   | PUT /telephony/{ba}/line/{sn}/phone/functionKey/{keyNum} |
| × touche          | Icône   | DELETE /telephony/{ba}/line/{sn}/phone/functionKey/{keyNum} |

# TYPES TOUCHES
| Type      | Badge    | Description                   |
|-----------|----------|-------------------------------|
| SpeedDial | Bleu     | Numérotation rapide           |
| BLF       | Vert     | Busy Lamp Field (supervision) |
| Line      | Orange   | Ligne supplémentaire          |
| Park      | Gris     | Parcage d'appel               |
| Intercom  | Violet   | Intercom                      |
| Transfer  | Rose     | Transfert                     |

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/phone - Infos téléphone
POST /telephony/{ba}/line/{sn}/phone/reboot - Redémarrer
GET /telephony/{ba}/line/{sn}/phone/functionKey - Liste touches
POST /telephony/{ba}/line/{sn}/phone/functionKey - Créer touche
PUT /telephony/{ba}/line/{sn}/phone/functionKey/{keyNum} - Modifier
DELETE /telephony/{ba}/line/{sn}/phone/functionKey/{keyNum} - Supprimer
