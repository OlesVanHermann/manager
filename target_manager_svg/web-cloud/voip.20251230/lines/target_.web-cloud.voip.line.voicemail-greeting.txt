| Type          | Description                          |

# ACTIONS
| Action            | Type    | API                                      |
|-------------------|---------|------------------------------------------|
| Écouter           | Player  | GET /telephony/{ba}/voicemail/{sn}/settings/downloadGreeting |
| Télécharger       | Bouton  | GET /telephony/{ba}/voicemail/{sn}/settings/downloadGreeting |
| Annonce défaut    | Bouton  | PUT /telephony/{ba}/voicemail/{sn}/settings → greetingType=default |

- Maximum 2 minutes

POST /telephony/{ba}/voicemail/{sn}/settings/uploadGreeting - Envoyer
