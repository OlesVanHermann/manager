# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif, x=510)
2. NAV2 - Web Cloud (VoIP actif, x=500)
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Header: Numéro + description + modèle téléphone
   - NAV3: 7 tabs (Général, Téléphone, Appels, **Transferts**, Messagerie, Consommation, Options)
   - NAV4: 3 sous-tabs (**Inconditionnel**, Occupation, Non-réponse)
   - Bannière warning: Alerte renvoi inconditionnel
   - Bloc gauche: Configuration renvoi (toggle, type, destination)
   - Bloc droite: Explication types de renvoi
   - Bannière info: Info combinaison renvois

# INTERACTIONS
- Clic NAV3 tab → Navigation vers autre section ligne
- Clic NAV4 sous-tab → Bascule Inconditionnel/Occupation/Non-réponse
- Toggle Activé/Désactivé → Active/désactive le renvoi
- Select type destination → Change vers Ligne interne/Externe/Messagerie
- Input numéro → Saisie destination
- Bouton Valider → Sauvegarde configuration
- Lien "Appliquer à plusieurs lignes" → Modal sélection lignes

# API ENDPOINTS
GET  /telephony/{ba}/line/{sn}/options                     # Options ligne
PUT  /telephony/{ba}/line/{sn}/options                     # Modifier options

# OPTIONS RENVOI (dans /options)
- forwardUnconditional: boolean
- forwardUnconditionalNumber: string
- forwardUnconditionalNature: enum (line, external, voicemail)
- forwardNoReply: boolean
- forwardNoReplyNumber: string
- forwardNoReplyDelay: number (5-60)
- forwardBusy: boolean
- forwardBusyNumber: string
- forwardBackup: boolean
- forwardBackupNumber: string
