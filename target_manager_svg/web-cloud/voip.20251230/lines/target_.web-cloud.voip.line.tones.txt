# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Back link: Retour à la gestion
   - Titre: Gestion des tonalités
   - Sections: Tonalité retour, Musique attente, Sons personnalisés

# SECTIONS
1. Tonalité de retour d'appel
   - Dropdown sélection tonalité
   - Bouton Écouter (preview)
   - Bouton Uploader (fichier personnalisé)

2. Musique d'attente
   - Dropdown sélection musique
   - Bouton Écouter (preview)
   - Bouton Uploader (fichier personnalisé)

3. Sons personnalisés
   - Table des sons uploadés (nom, type, actions)
   - Bouton Ajouter un son

# INTERACTIONS
- Sélection dropdown → Change tonalité/musique
- "Écouter" → Lecture audio preview
- "Uploader" → Modal upload fichier
- "Supprimer" → Supprime son personnalisé
- "Ajouter un son" → Modal upload
