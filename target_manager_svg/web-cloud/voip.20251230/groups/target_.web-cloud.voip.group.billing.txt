1. Left panel: Liste groupes VoIP
2. NAV3: Dashboard | Services | Repertoire | Facturation (actif) | Administration
3. Cards: Credit, Consommation mois, Abonnements
4. Table historique factures
5. Banniere info prelevement

| Type            | Description                    |
|-----------------|--------------------------------|
| Abonnements     | Forfaits mensuels lignes       |
| Consommation    | Appels hors forfait            |

API ENDPOINTS
GET  /telephony/{ba}                    -> Infos credit
GET  /telephony/{ba}/serviceInfos       -> Infos abonnement
GET  /telephony/{ba}/historyConsumption -> Consommation
POST /telephony/{ba}/credit             -> Recharger credit
PUT  /telephony/{ba}                    -> Modifier seuil
GET  /me/bill?orderId={ba}              -> Liste factures
GET  /me/bill/{id}/download             -> Telecharger PDF
