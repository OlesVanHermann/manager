STRUCTURE PAGE
1. SOUS-NAV: Lignes | Numeros (actif) | Fax | Abreges
2. Left panel: Liste numeros du groupe
3. NAV3: General | Configuration | Statistiques (actif) | Portabilite | Enregistrements
4. Period selector + Granularity + Export
5. Stats cards (5 KPIs)
6. Volume chart (line chart)
7. Repartition horaire (heatmap/bars)
8. Performance agents (table)

GRAPHIQUE VOLUME
- Type: Line chart double
- Lignes: Recus (bleu), Repondus (vert)
- Periode: Selon selecteur (jour/semaine/mois)
- Axes: Date (X), Nombre appels (Y)

- Type: Horizontal bars (heatmap style)
- Heures: 8h-18h
- Couleurs: Intensite proportionnelle au volume
