# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. LEFT PANEL - Liste des numéros du groupe
4. RIGHT PANEL
   - Titre: Configuration des sons
   - Sous-titre: Numéro + description
   - NAV3 tabs
   - Config attente
   - Config débordement
   - Config fermeture
   - Boutons action

| Tab | Route |

# CONFIG ATTENTE
| Champ | Type | Description |
|-------|------|-------------|
| Annonce d'ouverture | Dropdown | Son joué à l'entrée |
| Musique d'attente | Dropdown | Musique pendant attente |

# CONFIG DÉBORDEMENT
| Champ | Type | Description |
|-------|------|-------------|
| Max appels | Number | Nombre max en file |
| Action débordement | Radio | Voicemail ou son |

# CONFIG FERMETURE
| Champ | Type | Description |
|-------|------|-------------|
| Délai max | Number | Timeout en secondes |
| Action fermeture | Radio | Voicemail ou son |
| Son fermeture | Dropdown | Si action = son |

# INTERACTIONS
- Dropdowns → Sélection sons
- "+ Ajouter" → Modal upload son
- Radio buttons → Toggle actions
- "Annuler" → Annule modifications
- "Valider" → PUT API
