STRUCTURE PAGE
1. Left panel: Liste trunks SIP
2. Header: Nom trunk + statut
3. NAV3: General | Parametres | Routage | CDR | Statistiques (actif)
4. Period selector + export
5. Stats cards (5 KPIs)
6. Volume chart (bar chart)
7. Repartition direction (pie)
8. Metriques qualite

STATS CARDS
| Carte          | Donnee              | Comparaison     |
|----------------|---------------------|-----------------|
| Appels totaux  | Compteur            | vs mois prec.   |
| Appels reussis | Compteur + taux     | -               |
| Duree totale   | Heures + moyenne    | -               |
| Qualite SIP    | % ASR               | Niveau          |
| Pic simultane  | Max / Capacite      | Date/heure      |

GRAPHIQUE VOLUME
- Type: Bar chart
- Periode: Selon selecteur (jour/semaine)
- Axes: Date (X), Nombre appels (Y)
- Couleur: #0050D7 (primary)

REPARTITION DIRECTION
- Type: Pie chart
- Segments: Sortants / Entrants
- Details: Compteur + % + duree

METRIQUES QUALITE
| Metrique | Description                | Ideal        |
|----------|----------------------------|--------------|
| ASR      | Answer Seizure Ratio       | > 85%        |
| ACD      | Average Call Duration      | Variable     |
| PDD      | Post Dial Delay            | < 2s         |
| MOS      | Mean Opinion Score         | > 4.0 / 5    |

API ENDPOINTS
GET /telephony/trunk/{trunk}/statistics               -> Stats globales
GET /telephony/trunk/{trunk}/statistics/daily         -> Stats journalieres
GET /telephony/trunk/{trunk}/statistics/quality       -> Metriques qualite
GET /telephony/trunk/{trunk}/statistics/export        -> Export CSV
