COMPOSANTS
- Card IP SIP avec bouton copier
- Table avec colonnes: ID, Protocole, IP, Port, Priorite, Poids, Statut, Actions
- Badges protocole colores (UDP=bleu, TCP=orange, TLS=vert)
- Badges statut (Actif=vert, Inactif=gris)
- Bouton + Ajouter en header
- Pagination

INTERACTIONS
- Clic + Ajouter → Modal ajout endpoint
- Clic Modifier → Edition inline ou modal
- Clic × → Confirmation suppression
- Clic Copier → Copie IP presse-papier
- Clic tab NAV3 → Change vue

API ENDPOINTS

POST /telephony/trunks/{sn}/endpoints          # Ajouter endpoint
PUT /telephony/trunks/{sn}/endpoints/{id}      # Modifier endpoint
DELETE /telephony/trunks/{sn}/endpoints/{id}   # Supprimer endpoint
