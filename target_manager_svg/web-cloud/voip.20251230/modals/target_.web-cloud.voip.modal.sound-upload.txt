# STRUCTURE MODAL
1. Header: "Ajouter un son" + bouton fermer
2. Zone upload drag & drop
3. Fichier sélectionné (état)
4. Nom du son
5. Description
6. Bannière info formats
7. Footer: Annuler / Téléverser

# FORMATS ACCEPTÉS
| Format | Specs |
|--------|-------|
| MP3 | 128kbps mono/stéréo |
| WAV | 16-bit PCM 8kHz/16kHz |
| OGG | Vorbis |

# LIMITES
| Limite | Valeur |
|--------|--------|
| Taille max | 10 Mo |
| Durée max | 3 minutes |
| Durée min recommandée | 5 secondes |

# CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Fichier | File upload | Oui |
| Nom | Text | Oui |
| Description | Textarea | Non |

# ZONE UPLOAD
- Drag & drop fichier audio
- Clic pour parcourir
- Affichage info fichier (nom, taille, durée)

# ÉTATS FICHIER
| État | Affichage |
|------|-----------|
| Sélectionné | Fond vert + ✓ |
| Erreur format | Fond rouge + message |
| En cours | Barre progression |

# INTERACTIONS
- Drag & drop → Upload fichier
- Clic zone → Ouvrir explorateur
- ✓ → Fichier validé
- × → Supprimer fichier
- "Annuler" → Ferme modal
- "Téléverser" → POST API + ferme modal

# API ENDPOINTS
POST /telephony/{ba}/sound (multipart/form-data)
