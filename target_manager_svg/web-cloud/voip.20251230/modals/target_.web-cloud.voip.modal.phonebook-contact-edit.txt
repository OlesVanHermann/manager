# STRUCTURE MODAL
1. Header: "Modifier le contact" + bouton fermer
2. Formulaire pré-rempli
3. Footer: Supprimer / Annuler / Enregistrer

# CHAMPS FORMULAIRE (pré-remplis)
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Nom | Text | Oui |
| Prénom | Text | Oui |
| Groupe | Dropdown | Non |
| Tél. domicile | Tel | Min 1 requis |
| Mobile personnel | Tel | Min 1 requis |
| Tél. bureau | Tel | Min 1 requis |
| Mobile professionnel | Tel | Min 1 requis |
| Email | Email | Non |
| Notes | Textarea | Non |

# ACTIONS FOOTER
- "Supprimer" (rouge) → Confirmation puis DELETE
- "Annuler" → Ferme modal sans sauvegarder
- "Enregistrer" → PUT API + ferme modal

# VALIDATION
- Mêmes règles que l'ajout
- Nom et Prénom obligatoires
- Au moins 1 numéro de téléphone

# INTERACTIONS
- Champs → Modification des valeurs
- Dropdown groupe → Sélection groupe
- "×" → Ferme modal
- "Supprimer" → Modal confirmation puis DELETE
- "Annuler" → Ferme modal
- "Enregistrer" → PUT contact

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/phone/phonebook/contacts/{id}
PUT /telephony/{ba}/line/{sn}/phone/phonebook/contacts/{id}
DELETE /telephony/{ba}/line/{sn}/phone/phonebook/contacts/{id}
