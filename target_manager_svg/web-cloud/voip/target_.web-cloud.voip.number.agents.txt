INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic numéro → Charge les détails du numéro
- Clic NAV4 → Navigation vers l'onglet
- Clic ↻ → Actualiser la liste
- Clic + Ajouter agent → Modal ajout agent
- Clic ✎ → Modal édition agent
- Clic × → Modal confirmation suppression
- Clic header colonne → Tri

================================================================================

API ENDPOINTS

GET /telephony/{ba}/number → Liste numéros
GET /telephony/{ba}/easyHunting/{sn} → Configuration CCS
GET /telephony/{ba}/easyHunting/{sn}/agent → Liste agents
POST /telephony/{ba}/easyHunting/{sn}/agent → Ajouter agent
GET /telephony/{ba}/easyHunting/{sn}/agent/{id} → Détails agent
PUT /telephony/{ba}/easyHunting/{sn}/agent/{id} → Modifier agent
DELETE /telephony/{ba}/easyHunting/{sn}/agent/{id} → Supprimer agent
GET /telephony/{ba}/easyHunting/{sn}/agent/{id}/liveStatus → Statut temps réel
