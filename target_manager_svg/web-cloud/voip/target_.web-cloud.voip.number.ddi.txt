INTERACTIONS

- Dropdown "Destination par défaut" → Sélection SVI, ligne, file...
- Dropdown "Timeout" → 10, 20, 30, 45, 60 secondes
- Dropdown "Renvoi si occupé" → Messagerie, numéro externe, autre ligne
- Clic "+ Ajouter redirection" → Modal création extension
- Clic ✎ → Modifier redirection existante
- Clic × → Supprimer redirection (confirmation)

================================================================================

API ENDPOINTS

GET /telephony/{billingAccount}/ddi/{serviceName}
  → Configuration DDI (destination par défaut, timeouts)

PUT /telephony/{billingAccount}/ddi/{serviceName}
  → Modifier configuration DDI

GET /telephony/{billingAccount}/ddi/{serviceName}/entries
  → Liste des redirections

POST /telephony/{billingAccount}/ddi/{serviceName}/entries
  → Créer une redirection

PUT /telephony/{billingAccount}/ddi/{serviceName}/entries/{id}
  → Modifier une redirection

DELETE /telephony/{billingAccount}/ddi/{serviceName}/entries/{id}
  → Supprimer une redirection
