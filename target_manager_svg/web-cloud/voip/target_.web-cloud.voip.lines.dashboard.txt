# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Header: Numéro + description + badge statut
   - NAV3: [Dashboard] | Général | Téléphone | Appels | Transferts | Messagerie | ...
   - Row 1: 4 tiles stats (Statut, Appels, Durée, Messages)
   - Row 2: Derniers appels (tableau) + Config rapide (toggles)
   - Row 3: Téléphone + Consommation + Alertes

# TILES STATISTIQUES
- Statut ligne: Badge + dernière activité
- Appels aujourd'hui: Compteur + évolution
- Durée moyenne: Format mm:ss
- Messages vocaux: Compteur non lus

# CONFIGURATION RAPIDE
- Toggle Transfert d'appels (activé/désactivé)
- Toggle Ne pas déranger
- Toggle Messagerie vocale
- Bouton "Configurer tout"

# INTERACTIONS
- Clic tile → Navigation vers onglet correspondant
- Clic "Voir tout" appels → Onglet Appels
- Clic toggle → Active/désactive la fonction
- Clic "Configurer tout" → Modal configuration
- Clic alerte → Action contextuelle

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}              → Détails ligne
GET /telephony/{ba}/line/{sn}/serviceInfos → Infos statut
GET /telephony/{ba}/line/{sn}/calls        → Historique appels
GET /telephony/{ba}/line/{sn}/phone        → Infos téléphone
GET /telephony/{ba}/line/{sn}/options      → Options (transferts, DND)
GET /telephony/{ba}/voicemail/{sn}/messages→ Messages vocaux
GET /telephony/{ba}/line/{sn}/consumption  → Consommation
