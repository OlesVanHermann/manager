4. RIGHT PANEL
   - Header: Numéro + description + badge status
   - NAV4: [Général][Consommation][Appels][Téléphone][Options]

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic ligne → Charge les détails de la ligne
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la fonctionnalité
- Clic card restriction → Modal info (readonly)
- Clic + Bloquer numéro → Modal ajout liste noire
- Clic × → Supprimer de la liste noire

API ENDPOINTS

GET /telephony/{ba}/line → Liste lignes
GET /telephony/{ba}/line/{sn} → Détails ligne
GET /telephony/{ba}/line/{sn}/options → Options restrictions sortantes
GET /telephony/{ba}/line/{sn}/phone/phonebook → Liste noire
POST /telephony/{ba}/line/{sn}/phone/phonebook → Ajouter à liste noire
DELETE /telephony/{ba}/line/{sn}/phone/phonebook/{id} → Retirer de liste noire
