INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic groupe → Charge les détails du groupe
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la section
- Clic tuile Portabilités → Page portabilité
- Clic tuile Groupes de lignes → Page groupes
- Clic tuile White Label → Configuration personnalisation
- Clic tuile Gestion contacts → Modal modification contacts
- Clic tuile Supprimer → Modal confirmation suppression
- Clic tuile Options → Page paramètres avancés
- Clic tuile Ajouter → Modal rattachement services
- Clic tuile Renommer → Modal renommage
- Clic tuile Sécurité → Page restrictions/plafonds

================================================================================

API ENDPOINTS

GET /telephony → Liste billingAccounts
GET /telephony/{ba} → Détails groupe
PUT /telephony/{ba} → Modifier groupe (description)
DELETE /telephony/{ba} → Supprimer groupe
GET /telephony/{ba}/portability → Liste portabilités
GET /telephony/{ba}/line/options → Options du groupe
POST /telephony/{ba}/service → Ajouter service au groupe
