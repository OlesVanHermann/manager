# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des numéros du groupe
4. RIGHT PANEL
   - Header: Numéro + description + badge type (File d'attente/SVI/etc.)
   - NAV3: [Dashboard] | Général | Configuration | Agents | ...
   - Row 1: 4 tiles stats (Type, Appels en cours, En attente, Appels jour)
   - Row 2: Liste agents (tableau) + Stats temps réel (graphique)
   - Row 3: Actions rapides

# TILES STATISTIQUES (FILE D'ATTENTE)
- Type de numéro: Badge + nombre agents
- Appels en cours: Compteur + agents disponibles
- En attente: Compteur + temps moyen
- Appels aujourd'hui: Compteur + évolution

# TABLEAU AGENTS
- Avatar initiales
- Nom + poste
- Statut (Disponible/En appel/Pause)
- Nombre appels traités
- Durée moyenne

# STATS TEMPS RÉEL
- Graphique appels/heure
- Légende: Répondus/Manqués/Abandonnés
- Temps d'attente moyen

# ACTIONS RAPIDES
- Voir statistiques détaillées
- Gérer agents
- Musique d'attente
- Planifier horaires
- Configuration complète

# INTERACTIONS
- Clic tile → Navigation onglet correspondant
- Clic "Gérer" agents → Onglet Agents
- Clic action rapide → Navigation/Modal

# API ENDPOINTS
GET /telephony/{ba}/number/{sn}                        → Détails numéro
GET /telephony/{ba}/easyHunting/{sn}/hunting/agent     → Liste agents
GET /telephony/{ba}/easyHunting/{sn}/hunting/queue     → File d'attente temps réel
GET /telephony/{ba}/easyHunting/{sn}/records           → Historique appels
GET /telephony/{ba}/easyHunting/{sn}/screenListConditions → Conditions
