INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic groupe → Charge les détails du groupe
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la section
- Clic ✎ nom annuaire → Modal édition nom
- Clic checkbox → Sélection/désélection contact
- Clic checkbox header → Tout sélectionner/désélectionner
- Clic Actions → Menu contextuel (Importer CSV, Exporter CSV)
- Clic Supprimer (n) → Modal confirmation suppression sélection
- Clic Ajouter → Modal ajout contact
- Clic ⋯ ligne → Menu actions contact (Modifier, Supprimer)
- Clic pagination → Change de page

================================================================================

API ENDPOINTS

GET /telephony → Liste billingAccounts
GET /telephony/{ba} → Détails groupe
GET /telephony/{ba}/phonebook → Liste annuaires
GET /telephony/{ba}/phonebook/{id} → Détail annuaire
PUT /telephony/{ba}/phonebook/{id} → Modifier annuaire (nom)
DELETE /telephony/{ba}/phonebook/{id} → Supprimer annuaire
GET /telephony/{ba}/phonebook/{id}/phonebookContact → Liste contacts
POST /telephony/{ba}/phonebook/{id}/phonebookContact → Créer contact
PUT /telephony/{ba}/phonebook/{id}/phonebookContact/{contactId} → Modifier contact
DELETE /telephony/{ba}/phonebook/{id}/phonebookContact/{contactId} → Supprimer contact
POST /telephony/{ba}/phonebook/{id}/import → Import CSV
