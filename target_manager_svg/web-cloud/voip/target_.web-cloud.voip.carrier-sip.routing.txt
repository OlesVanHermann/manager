1. NAV1 - Header OVH (Web Cloud actif)

   - NAV3: [SIP][SMS][FAX][Trunk] - Trunk actif
   - Recherche
   - Liste des trunks SIP
4. RIGHT PANEL
   - Header: ID trunk + description + badge status
   - NAV4: [Général][Paramètres][Routage][CDR]
   - Titre: Routage des appels
   - Bannière info (ordre des règles)
   - Toolbar: Refresh, + Nouvelle règle
   - Section Routage entrant (DDI):
     * Table: Priorité, Numéro, Destination, Description, Statut, Actions
     * Actions: Monter/Descendre, Éditer, Supprimer
   - Section Routage sortant:
     * Table: Priorité, Préfixe, Caller ID, Route, Description, Statut, Actions

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic trunk → Charge les détails du trunk
- Clic NAV4 → Navigation vers l'onglet
- Clic ↻ → Actualiser règles
- Clic Nouvelle règle → Modal création
- Clic Gérer les DDI → Navigation page DDI
- Clic ↑ → Monter priorité
- Clic ↓ → Descendre priorité
- Clic ✎ → Modal édition règle
- Clic × → Modal confirmation suppression

================================================================================

API ENDPOINTS

GET /telephony/{ba}/carrierSip → Liste trunks
GET /telephony/{ba}/carrierSip/{id}/ddi → Numéros DDI
GET /telephony/{ba}/carrierSip/{id}/rules/inbound → Règles entrantes
GET /telephony/{ba}/carrierSip/{id}/rules/outbound → Règles sortantes
POST /telephony/{ba}/carrierSip/{id}/rules → Créer règle
PUT /telephony/{ba}/carrierSip/{id}/rules/{ruleId} → Modifier règle
PUT /telephony/{ba}/carrierSip/{id}/rules/{ruleId}/priority → Changer ordre
DELETE /telephony/{ba}/carrierSip/{id}/rules/{ruleId} → Supprimer règle
