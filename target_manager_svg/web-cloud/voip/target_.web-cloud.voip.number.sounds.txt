STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL
   - NAV3: [SIP][SMS][FAX][Trunk]
   - Recherche
   - Liste des numéros
4. RIGHT PANEL
   - Header: Numéro + description + badge status
   - NAV4: [Général][Agents][Sons][Horaires][Filtrage]
   - Titre: Sons et musique d'attente
   - Bannière info formats
   - Cards sons (6 slots):
     * Musique d'attente (holdMusic)
     * Message d'accueil (welcome)
     * Message file d'attente (queue)
     * Message hors horaires (closed)
     * Message débordement (overflow)
     * Annonce position (position)
   - Chaque card: Nom fichier, durée, Play, Supprimer, Modifier/Ajouter

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic numéro → Charge les détails du numéro
- Clic NAV4 → Navigation vers l'onglet
- Clic ▶ → Lecture du fichier audio
- Clic × → Supprimer le fichier
- Clic Modifier → Modal upload nouveau fichier
- Clic Ajouter → Modal upload fichier
- Clic Configurer → Modal paramètres annonce

================================================================================

API ENDPOINTS

GET /telephony/{ba}/number → Liste numéros
GET /telephony/{ba}/easyHunting/{sn} → Configuration CCS
GET /telephony/{ba}/easyHunting/{sn}/sound → Liste sons
GET /telephony/{ba}/easyHunting/{sn}/sound/{slot} → Détails son
POST /telephony/{ba}/easyHunting/{sn}/sound/upload → Upload fichier
DELETE /telephony/{ba}/easyHunting/{sn}/sound/{slot} → Supprimer son
PUT /telephony/{ba}/easyHunting/{sn} → Modifier config (announcePosition, etc.)
