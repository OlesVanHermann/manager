1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL
   - NAV3: [SIP][SMS][FAX][Trunk] - FAX actif
   - Recherche
   - Liste des services fax
4. RIGHT PANEL
   - Header: Numéro + description + badge status
   - NAV4: [Général][Envoyés][Reçus][Paramètres]
   - Card infos service:
     * Numéro, Description, Statut
     * Email réception, Format, Logo
   - Card statistiques mois:
     * Fax envoyés, reçus, pages totales
     * Taux réussite, crédits
   - Tiles actions rapides:
     * Envoyer un fax
     * Fax reçus (avec badge compteur)
     * Paramètres
     * Logo et en-tête
     * Notifications
     * Supprimer (danger)

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic fax → Charge les détails du fax
- Clic NAV4 → Navigation vers l'onglet
- Clic ✎ description → Modal édition
- Clic ✎ email → Modal édition email
- Clic tile → Navigation vers la fonctionnalité
- Clic Supprimer → Modal confirmation

================================================================================

API ENDPOINTS

GET /telephony/{ba}/fax → Liste services fax
GET /telephony/{ba}/fax/{sn} → Détails fax
PUT /telephony/{ba}/fax/{sn} → Modifier fax
GET /telephony/{ba}/fax/{sn}/campaigns → Statistiques
DELETE /telephony/{ba}/fax/{sn}/terminate → Supprimer service
