# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des lignes
4. RIGHT PANEL
   - Header: Numéro + breadcrumb /Options
   - NAV3: Général | Téléphone | Appels | Transferts | Messagerie | Consommation | [Options]
   - Section "Fonctionnalités de la ligne"
     * Présentation du numéro (toggle)
     * Signal d'appel (toggle)
     * Interception d'appels (toggle)
     * Ne pas déranger (toggle)
   - Section "Sécurité"
     * Restrictions d'appels (lien config)
     * Click2Call (toggle)
   - Bannière warning options payantes

# OPTIONS DISPONIBLES
| Option               | Type   | API Field            | Description                    |
|---------------------|--------|----------------------|--------------------------------|
| Présentation numéro | toggle | identificationRestriction | Masquer/afficher CLID      |
| Signal d'appel      | toggle | callWaiting          | Notification appel en cours    |
| Interception        | toggle | intercom             | Prendre appel autre ligne      |
| Ne pas déranger     | toggle | doNotDisturb         | Rejeter tous appels            |
| Click2Call          | toggle | click2CallUser       | Autoriser appels depuis web    |

# INTERACTIONS
- Toggle ON/OFF → Active/désactive l'option (PUT immédiat)
- Clic "Configurer les restrictions →" → Navigation page restrictions
- Clic "Voir détails" → Modal détail restrictions

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/options              → Toutes les options
PUT /telephony/{ba}/line/{sn}/options              → Modifier options
GET /telephony/{ba}/line/{sn}/options/restrictions → Restrictions appels
PUT /telephony/{ba}/line/{sn}/options/restrictions → Modifier restrictions

# RESTRICTIONS DISPONIBLES (Page dédiée)
| Type                  | Description                        |
|-----------------------|------------------------------------|
| outgoingInternational | Appels internationaux              |
| outgoingPremiumRate   | Numéros surtaxés                   |
| outgoingMobile        | Appels vers mobiles                |
| incomingAnonymous     | Appels entrants anonymes           |
