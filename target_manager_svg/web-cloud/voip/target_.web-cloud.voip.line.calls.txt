INTERACTIONS

- Clic ligne → Charge les détails de la ligne

- Clic tuile → Navigation vers la fonctionnalité

API ENDPOINTS

GET /telephony/{ba}/line → Liste lignes
GET /telephony/{ba}/line/{sn} → Détails ligne
GET /telephony/{ba}/line/{sn}/options → Options de la ligne
GET /telephony/{ba}/line/{sn}/options/forward → Renvoi d'appels
GET /telephony/{ba}/line/{sn}/options/screening → Filtrage
GET /telephony/{ba}/line/{sn}/options/timeCondition → Plages horaires
GET /telephony/{ba}/line/{sn}/options/simultaneousLines → Appels simultanés
GET /telephony/{ba}/line/{sn}/click2Call → Click-to-Call
GET /telephony/{ba}/line/{sn}/options/intercept → Interception
GET /telephony/{ba}/line/{sn}/options/displayNumber → Présentation numéro
GET /telephony/{ba}/line/{sn}/options/restriction → Restrictions
GET /telephony/{ba}/line/{sn}/options/lock → Verrouillage
GET /telephony/{ba}/line/{sn}/voicemail → Messagerie vocale
GET /telephony/{ba}/line/{sn}/options/callWaiting → Double appel
GET /telephony/{ba}/line/{sn}/calls → Logs d'appels
GET /telephony/{ba}/line/{sn}/options/record → Enregistrement
