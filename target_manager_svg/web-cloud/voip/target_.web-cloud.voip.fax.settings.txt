STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL
   - NAV3: [SIP][SMS][FAX][Trunk] - FAX actif
   - Recherche
   - Liste des services fax
4. RIGHT PANEL
   - Header: Numéro + description + badge status
   - NAV4: [Général][Envoyés][Reçus][Paramètres]
   - Titre: Paramètres du fax
   - Section Réception:
     * Email de réception (modifiable)
     * Format de réception (dropdown)
   - Section Envoi:
     * Qualité d'envoi (dropdown)
     * Tentatives en cas d'échec (dropdown)
     * Notification d'envoi (toggle)
   - Section Personnalisation:
     * Logo et en-tête (preview + modifier/supprimer)
     * Informations d'en-tête (société, adresse, tel)

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic fax → Charge les détails du fax
- Clic NAV4 → Navigation vers l'onglet
- Clic Modifier email → Modal édition email
- Clic dropdown → Sélection option
- Clic toggle ON/OFF → Basculer paramètre
- Clic Modifier logo → Modal upload logo
- Clic Supprimer logo → Modal confirmation
- Clic Modifier en-tête → Modal édition texte

================================================================================

API ENDPOINTS

GET /telephony/{ba}/fax → Liste services fax
GET /telephony/{ba}/fax/{sn} → Détails fax
PUT /telephony/{ba}/fax/{sn} → Modifier paramètres
POST /telephony/{ba}/fax/{sn}/logo/upload → Upload logo
DELETE /telephony/{ba}/fax/{sn}/logo → Supprimer logo
PUT /telephony/{ba}/fax/{sn}/header → Modifier en-tête
