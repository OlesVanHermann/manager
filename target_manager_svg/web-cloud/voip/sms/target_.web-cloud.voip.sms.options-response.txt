## API ENDPOINTS

### Config réponses
GET /sms/{serviceName}/response
Response: { enabled, dedicatedNumber, receiveMode, stopHandling }

### Modifier config
PUT /sms/{serviceName}/response
Body: { enabled?, receiveMode?, stopHandling? }

### Liste réponses
GET /sms/{serviceName}/incoming
Query: { limit, page }
Response: [{ date, sender, originalMessage, response }]

### Liste noire STOP
GET /sms/{serviceName}/blacklist
Response: [{ number, date, reason }]

================================================================================

## REGLES METIER

- Numéro dédié attribué automatiquement si activé
- Modes: webhook, email
- Réponse STOP = désinscription automatique
- Liste noire permanente sauf suppression manuelle
- Conservation réponses: 90 jours
