# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des comptes SMS
4. RIGHT PANEL
   - Header: Compte + crédits
   - NAV3: Général | Envoyer | Campagnes | Entrants | [Sortants] | Expéditeurs | Templates
   - Toolbar: Rafraîchir, Exporter, Filtres
   - Stats summary (4 tiles)
   - Tableau SMS sortants
   - Pagination

# TOOLBAR
- Rafraîchir: Recharge les données
- Exporter CSV: Télécharge historique
- Filtre dates: Du / Au
- Filtre statut: Tous / Délivré / En attente / Échoué

# STATS SUMMARY
4 tiles horizontales:
- SMS envoyés: Total count
- Délivrés: Count + % (fond vert)
- En attente: Count + % (fond orange)
- Échoués: Count + % (fond rouge)

# COLONNES TABLEAU
- DATE: Date et heure d'envoi
- DESTINATAIRE: Numéro du destinataire
- EXPÉDITEUR: Nom ou numéro expéditeur
- MESSAGE: Aperçu du contenu (tronqué)
- CRÉDITS: Nombre de crédits utilisés
- STATUT: Délivré/En attente/Échoué (badges)
- ACTIONS: Voir détails

# STATUTS SMS
- Délivré: Badge vert #DEF7EC
- En attente: Badge orange #FEF3C7
- Échoué: Badge rouge #FEE2E2

# INTERACTIONS
- Clic "Rafraîchir" → Recharge liste
- Clic "Exporter" → Téléchargement CSV
- Clic filtres → Applique filtres
- Clic "Détails" → Modal détail SMS

# API ENDPOINTS
GET /sms/{sn}                    → Infos compte
GET /sms/{sn}/outgoing           → Liste SMS sortants
GET /sms/{sn}/outgoing/{id}      → Détail SMS
