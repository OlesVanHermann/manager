## API ENDPOINTS

### Liste SMS sortants
GET /sms/{serviceName}/outgoing
Query: { startDate?, endDate?, status?, search?, limit, offset }
Response: [{ id, receiver, message, sender, sentAt, status, credits }]

### Détail SMS
GET /sms/{serviceName}/outgoing/{smsId}
Response: { id, receiver, message, sender, sentAt, status, credits, deliveredAt?, errorCode? }

### Export CSV
GET /sms/{serviceName}/outgoing/export
Query: { startDate, endDate, status? }
Response: CSV file

### Stats
GET /sms/{serviceName}/outgoing/stats
Response: { todayCount, deliveryRate, monthlyCredits }

================================================================================

## REGLES METIER

- Historique conservé 1 an
- Message tronqué à 50 caractères dans la liste
- Statuts: sent (envoyé), delivered (délivré), failed (échec)
- SMS long compte plusieurs crédits (>160 car)
- Export max 100 000 lignes
