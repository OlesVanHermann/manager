STRUCTURE PAGE
1. NAV3: General | SMS envoyes (actif) | SMS recus | Campagnes | Modeles | Expediteurs | Parametres
2. Left panel: Liste comptes SMS
3. Header: Nom compte + button envoyer
4. Filters: Recherche + periode + statut + export
5. Table SMS envoyes
6. Pagination

STATUTS SMS
| Statut     | Couleur   | Description              |
|------------|-----------|--------------------------|
| Delivre    | #DEF7EC   | SMS recu par destinataire|
| En attente | #FEF3C7   | En cours d'envoi         |
| Echec      | #FEE2E2   | Echec livraison          |
| Expire     | #F3F4F6   | Non delivre (timeout)    |

COLONNES TABLE
| Colonne      | Description                    |
|--------------|--------------------------------|
| Date         | Date/heure envoi               |
| Destinataire | Numero telephone               |
| Expediteur   | Sender ID utilise              |
| Message      | Apercu du contenu (tronque)    |
| Credits      | Nombre credits consommes       |
| Statut       | Badge etat livraison           |
| Actions      | Renvoyer / Annuler             |

ACTIONS
| Statut     | Actions disponibles  |
|------------|----------------------|
| Delivre    | Renvoyer             |
| En attente | Annuler              |
| Echec      | Renvoyer             |
| Expire     | Renvoyer             |

MODAL ENVOI SMS
| Champ       | Type      | Requis |
|-------------|-----------|--------|
| Destinataire| tel       | Oui    |
| Expediteur  | select    | Oui    |
| Message     | textarea  | Oui    |
| Programmer  | datetime  | Non    |

API ENDPOINTS
GET    /sms/{account}/outgoing                -> Liste SMS envoyes
GET    /sms/{account}/outgoing/{id}           -> Detail SMS
POST   /sms/{account}/outgoing                -> Envoyer SMS
POST   /sms/{account}/outgoing/{id}/resend    -> Renvoyer
DELETE /sms/{account}/outgoing/{id}           -> Annuler (si en attente)
