## DESCRIPTION

Configuration des webhooks (callback URLs) pour recevoir les notifications
SMS : accusés de réception, réponses entrantes, erreurs.

## API ENDPOINTS

### Config callbacks
GET /sms/{serviceName}/callbacks
Response: { delivery: { enabled, url, method }, response: { enabled, url, method } }

### Modifier callbacks
PUT /sms/{serviceName}/callbacks
Body: { delivery?: { enabled, url, method }, response?: { enabled, url, method } }

### Tester callback
POST /sms/{serviceName}/callbacks/test
Body: { type: 'delivery' | 'response' }
Response: { success, statusCode, responseTime }

================================================================================

## REGLES METIER

- URL: HTTPS obligatoire en production
- Méthodes: GET, POST
- Timeout callback: 10 secondes
- Retry: 3 tentatives en cas d'échec
- Format payload: JSON
- Signature HMAC disponible
