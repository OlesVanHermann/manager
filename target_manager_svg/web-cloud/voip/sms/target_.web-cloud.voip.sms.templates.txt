# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des comptes SMS
4. RIGHT PANEL
   - Header: Nom compte + breadcrumb /Modèles
   - NAV3: Général | Envoyer | Campagnes | Entrants | Historique | [Modèles]
   - Toolbar: Refresh, Recherche, + Créer modèle
   - Bannière info: Explication variables
   - Grille de modèles: Cards avec aperçu contenu

# STRUCTURE CARD MODÈLE
| Zone | Contenu |
|------|---------|
| Header | Nom + Badge type |
| Body | Aperçu message (3 lignes max) |
| Footer | Stats (caractères, SMS) + Actions |

# TYPES DE MODÈLES
| Type | Badge | Usage |
|------|-------|-------|
| Transactionnel | Vert DEF7EC | Confirmations, codes, RDV |
| Marketing | Bleu DBEAFE | Promos, newsletters |

# VARIABLES DISPONIBLES
| Variable | Description |
|----------|-------------|
| {NOM} | Nom du destinataire |
| {PRENOM} | Prénom du destinataire |
| {DATE} | Date formatée |
| {HEURE} | Heure formatée |
| {CODE} | Code de vérification |
| {REF} | Référence commande/colis |
| {URL} | Lien court |
| {REDUCTION} | Pourcentage de réduction |

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic ↻ → Rafraîchir liste
- Clic "+ Créer modèle" → Modal création
- Clic "Utiliser" → Ouvre page Envoyer avec modèle pré-rempli
- Clic ✎ → Modal édition modèle
- Clic ✕ → Supprimer modèle (confirmation)
- Clic carte création → Modal création nouveau modèle

# API ENDPOINTS
GET /sms/{sn}/templates                    → Liste modèles
POST /sms/{sn}/templates                   → Créer modèle
GET /sms/{sn}/templates/{id}               → Détails modèle
PUT /sms/{sn}/templates/{id}               → Modifier modèle
DELETE /sms/{sn}/templates/{id}            → Supprimer modèle
