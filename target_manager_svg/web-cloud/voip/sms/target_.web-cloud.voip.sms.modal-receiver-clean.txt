## 2. CONTENU MODAL
--------------------------------------------------------------------------------

### Titre
Nettoyer

### Message confirmation
"Etes-vous sur de vouloir nettoyer les contacts ?"

### Options disponibles
"Deux options sont disponibles :"

--------------------------------------------------------------------------------
## 3. OPTIONS DE NETTOYAGE
--------------------------------------------------------------------------------

### Option 1 : Freemium
- **Label:** Freemium
- **Description:** Dedoublonnage et verifications syntaxiques
- **Cout:** Gratuit

### Option 2 : Premium
- **Label:** Premium
- **Description:** Dedoublonnage et validation de la validite des contacts via une requete HLR
- **Cout:** Facture X.XX credits (calcule selon nombre de contacts)

--------------------------------------------------------------------------------
## 4. BOUTONS
--------------------------------------------------------------------------------

- Nettoyer (primary)
- Annuler (secondary)

--------------------------------------------------------------------------------
## 5. API
--------------------------------------------------------------------------------

| Methode | Endpoint | Description |
|---------|----------|-------------|
| POST | /sms/{serviceName}/receivers/{slotId}/clean | Lancer nettoyage |

**Body:**
```json
{
  "freemium": true/false
}
```

--------------------------------------------------------------------------------
## 6. MESSAGES RETOUR
--------------------------------------------------------------------------------

- Loading : "Nettoyage de la liste des destinataires..."
- Succes : "Tache de nettoyage ajoutee !"
- Erreur : "Impossible de nettoyer la liste des destinataires. [error]"

--------------------------------------------------------------------------------
## 7. BLOCAGE SI NETTOYAGE EN COURS
--------------------------------------------------------------------------------

Si un nettoyage est deja en cours sur une liste :
"Un nettoyage est en cours sur l une de vos listes de contacts..."

Le bouton Nettoyer est desactive.
