# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des comptes SMS
4. RIGHT PANEL
   - Header: Nom compte + badge statut + description
   - NAV3: [Général] | Envoyer | Campagnes | Entrants | Historique | Modèles
   - Card "Crédits SMS": Solde + bouton achat
   - Card "Ce mois-ci": Stats envoyés/reçus
   - Card "Informations": Nom, description, expéditeur par défaut
   - Section "Actions rapides": Envoyer, Campagne, Modèles
   - Section "Configuration": Expéditeurs, Réception
   - Bannière info API

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Acheter des crédits" → Modal achat crédits
- Clic "Voir l'historique →" → Page historique
- Clic action rapide → Navigation page correspondante
- Clic "Gérer les expéditeurs →" → Page gestion senders
- Clic "Configurer la réception →" → Page config receivers

# API ENDPOINTS
GET /sms/{sn}                       → Détails compte SMS
GET /sms/{sn}/statistics            → Statistiques
GET /sms/{sn}/senders               → Liste expéditeurs
GET /sms/{sn}/receivers             → Numéros de réception
POST /sms/{sn}/creditsOrder         → Commander crédits
GET /sms/{sn}/jobs                  → Liste envois
