## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche comptes SMS
   - Liste des comptes avec crédits

2. RIGHT PANEL (1096px)
   - Header: Compte + Description
   - NAV3 tabs: [Général] | Envoyer | Campagnes | Reçus | Modèles

   - STATS CARDS (4 cards):
     * Crédits disponibles
     * SMS envoyés (mois)
     * Taux de délivrabilité
     * Campagnes actives

   - BLOC: Informations du compte (1040x140)
     * Identifiant + date création
     * Émetteur par défaut + callback URL
     * Offre + statut

   - BLOC: Graphique envois 7 jours (1040x210)
     * Bar chart par jour de la semaine

   - BLOC: Actions rapides (4 boutons)
     * Envoyer SMS
     * Nouvelle campagne
     * Acheter crédits
     * Paramètres

## API ENDPOINTS

### Informations compte
GET /sms/{serviceName}
Response: {
  serviceName, status, creditsLeft, offerName,
  callbackUrl, creationDate
}

### Statistiques
GET /sms/{serviceName}/statistics
Query: { dateTime.from, dateTime.to }
Response: {
  sentCount, receivedCount, deliveryRate,
  dailyStats: [{ date, sent, delivered, failed }]
}

### Émetteurs
GET /sms/{serviceName}/senders
Response: [{ name, status, isDefault }]

### Crédits
GET /sms/{serviceName}/credits
Response: { available, used, pending }

### Acheter crédits
POST /sms/{serviceName}/credits/buy
Body: { amount, paymentMethod }

## REGLES METIER

- Crédits sans expiration
- Alerte automatique à 10% de crédits
- Taux de délivrabilité calculé sur 30 jours
- Callback URL pour accusés de réception
- Max 5 émetteurs par compte
