## DESCRIPTION

Vérification HLR (Home Location Register) des numéros de téléphone.
Permet de vérifier la validité, l'opérateur et le type de numéro avant envoi.

## API ENDPOINTS

### Vérification unitaire
POST /sms/{serviceName}/hlr
Body: { number }
Response: { number, status, operator, country, type, cost }

### Vérification en masse
POST /sms/{serviceName}/hlr/batch
Body: { numbers[] } ou FormData avec fichier CSV
Response: { batchId, count }

### Résultat batch
GET /sms/{serviceName}/hlr/batch/{batchId}
Response: { status, results[], validCount, invalidCount }

### Historique
GET /sms/{serviceName}/hlr/history
Query: { startDate?, endDate?, limit, offset }
Response: [{ number, status, operator, country, type, checkedAt, cost }]

### Stats
GET /sms/{serviceName}/hlr/stats
Response: { monthlyCreditsUsed, monthlyRequestCount }

## REGLES METIER

- Coût: 0.01 crédit par requête
- Résultat mis en cache 24h
- Batch max 10 000 numéros
- Format international requis (+XX...)
- Résultats: valid, invalid, unknown
