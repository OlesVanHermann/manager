# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des comptes SMS
4. RIGHT PANEL
   - Header: Compte
   - NAV3: ... | [Expéditeurs] | Templates
   - Bannière info (validation)
   - Toolbar: Ajouter, Rafraîchir
   - Tableau expéditeurs
   - Card Règles + Card Expéditeur par défaut

# BANNIÈRE INFO
- Information sur le délai de validation (24-48h)

# TOOLBAR
- Bouton "+ Ajouter expéditeur" (primaire)
- Bouton "Rafraîchir"

# COLONNES TABLEAU
- EXPÉDITEUR: Nom ou numéro
- TYPE: Alphanumérique ou Numérique
- DESCRIPTION: Usage prévu
- STATUT: Validé/En attente (badges)
- DATE CRÉATION: Date d'ajout
- ACTIONS: Modifier, Supprimer

# TYPES D'EXPÉDITEURS
- Alphanumérique: Texte 3-11 caractères (ex: "MonEntreprise")
- Numérique: Numéro de téléphone (ex: "+33912345678")

# STATUTS
- Validé: Badge vert #DEF7EC
- En attente: Badge orange #FEF3C7
- Refusé: Badge rouge #FEE2E2

# CARD RÈGLES
- Contraintes sur les noms d'expéditeurs
- Format alphanumérique
- Délai de validation

# CARD EXPÉDITEUR PAR DÉFAUT
- Dropdown sélection expéditeur
- Bouton Enregistrer

# INTERACTIONS
- Clic "+ Ajouter" → Modal création
- Clic "Modifier" → Modal édition
- Clic "Supprimer" → Modal confirmation
- Changement dropdown → Sélection défaut
- Clic "Enregistrer" → Sauvegarde défaut

# API ENDPOINTS
GET /sms/{sn}/senders                    → Liste expéditeurs
GET /sms/{sn}/senders/{sender}           → Détail expéditeur
POST /sms/{sn}/senders                   → Créer expéditeur
DELETE /sms/{sn}/senders/{sender}        → Supprimer expéditeur
