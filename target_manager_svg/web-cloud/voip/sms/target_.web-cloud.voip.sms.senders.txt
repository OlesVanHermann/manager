## API ENDPOINTS

### Liste expéditeurs
GET /sms/{serviceName}/senders
Response: [{ sender, type, status, createdAt, sentCount, isDefault }]

### Créer expéditeur
POST /sms/{serviceName}/senders
Body: { sender, description?, referer? }
Response: { sender, status: 'pending' }

### Définir par défaut
PUT /sms/{serviceName}/senders/{sender}/default

### Supprimer expéditeur
DELETE /sms/{serviceName}/senders/{sender}

### Vérifier disponibilité
GET /sms/{serviceName}/senders/check
Query: { sender }
Response: { available: boolean, reason? }

================================================================================

## REGLES METIER

- Alphanumérique: 3-11 car, lettres/chiffres, pas ressembler à tel
- Numérique: numéro court ou numéro long validé
- Validation alphanumérique: 24-48h ouvrées
- Maximum 50 expéditeurs par compte
- 1 seul expéditeur par défaut
- Refus possible si marque déposée/trompeur
