Gestion des envois SMS par lots. Historique des campagnes, suivi des
envois groupés et création de nouveaux lots.

## API ENDPOINTS

### Liste lots
GET /sms/{serviceName}/batches
Query: { status?, startDate?, endDate?, limit, offset }
Response: [{ id, name, scheduledDate, recipientCount, sentCount, failedCount, status }]

### Détail lot
GET /sms/{serviceName}/batches/{batchId}
Response: { id, name, message, sender, scheduledDate, recipientCount, sentCount, failedCount, status, createdAt }

### Créer lot
POST /sms/{serviceName}/batches
Body: { name, message, sender, recipients[], scheduledDate? }

### Annuler lot planifié
DELETE /sms/{serviceName}/batches/{batchId}

### Rapport lot
GET /sms/{serviceName}/batches/{batchId}/report
Response: { deliveredCount, failedCount, pendingCount, details[] }

### Stats globales
GET /sms/{serviceName}/batches/stats
Response: { totalSent, deliveryRate, totalFailed }

## REGLES METIER

- Lot planifié modifiable jusqu'à 5min avant envoi
- Lot en cours peut être stoppé
- Lot terminé en lecture seule
- Rapport téléchargeable CSV
- Maximum 50 000 destinataires par lot
