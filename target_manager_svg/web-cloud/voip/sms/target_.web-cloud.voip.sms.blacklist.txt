désabonnement (STOP), ajout manuel ou import externe.

## API ENDPOINTS

### Liste noire
GET /sms/{serviceName}/blacklists
Query: { source?, search?, limit, offset }
Response: [{ number, source, createdAt, reason }]

### Ajouter numéro
POST /sms/{serviceName}/blacklists
Body: { number, reason? }

### Retirer numéro
DELETE /sms/{serviceName}/blacklists/{number}

### Import liste
POST /sms/{serviceName}/blacklists/import
Body: { numbers[], reason? }

### Export liste
GET /sms/{serviceName}/blacklists/export
Response: CSV file
