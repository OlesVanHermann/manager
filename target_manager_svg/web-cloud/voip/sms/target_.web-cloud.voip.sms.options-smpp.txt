--------------------------------------------------------------------------------
## 2. CONTEXTE FONCTIONNEL
--------------------------------------------------------------------------------

Configuration des parametres SMPP pour les comptes de type SMPP
(marketing ou transactionnel).

Permet de gerer les adresses IP autorisees a se connecter au service SMPP.

--------------------------------------------------------------------------------
## 3. LAYOUT ET STRUCTURE
--------------------------------------------------------------------------------

### Section IPs autorisees
- Liste des adresses IP configurees
- Bouton ajout nouvelle IP
- Actions par IP (supprimer)

--------------------------------------------------------------------------------
## 4. LISTE DES IPS
--------------------------------------------------------------------------------

Affichage des IPs autorisees avec possibilite de :
- Visualiser toutes les IPs configurees
- Supprimer une IP existante
- Ajouter une nouvelle IP

Support IPv4 et IPv6.

--------------------------------------------------------------------------------
## 5. MODAL AJOUT IP
--------------------------------------------------------------------------------

### Titre : Ajouter une adresse IP

| Champ | Type | Validation |
|-------|------|------------|
| IP | Input | Format IPv4 ou IPv6 valide |

**Tooltip:** L adresse IP peut etre de type IPv4 ou IPv6

**Boutons:**
- Envoyer (primary)
- Annuler (secondary)

--------------------------------------------------------------------------------
## 6. MODAL SUPPRESSION IP
--------------------------------------------------------------------------------

### Titre : Supprimer une adresse IP

**Message:** Vous etes sur le point de supprimer : [IP]

**Boutons:**
- Supprimer (danger)
- Annuler (secondary)

--------------------------------------------------------------------------------
## 7. APIS UTILISEES
--------------------------------------------------------------------------------

| Methode | Endpoint | Description |
|---------|----------|-------------|
| GET | /sms/{serviceName} | Config SMPP (allowedIPs) |
| PUT | /sms/{serviceName} | Modifier liste IPs |

**Champ API:**
- smpp.allowedIPs : Liste des IPs autorisees
