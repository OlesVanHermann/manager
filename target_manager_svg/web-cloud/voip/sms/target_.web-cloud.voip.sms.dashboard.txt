STRUCTURE PAGE
1. NAV3: General (actif) | SMS envoyes | SMS recus | Campagnes | Modeles | Expediteurs | Parametres
2. Left panel: Liste comptes SMS
3. Header: Nom compte + badges + actions
4. Stats cards (3 KPIs)
5. Expediteurs configures
6. Activite recente
7. Actions rapides
8. Info banner API

STATS CARDS
| Carte              | Donnee                | Unite          |
|--------------------|-----------------------|----------------|
| Credits disponibles| Solde actuel          | Credits        |
| Usage ce mois      | SMS envoyes           | SMS + delta    |
| Taux de succes     | % delivres            | % + details    |

EXPEDITEURS
| Type           | Description              |
|----------------|--------------------------|
| Alphanumerique | Nom entreprise (11 car.) |
| Numerique      | Numero telephone         |
| Court          | Numero court (5 chiffres)|

ACTIONS RAPIDES
| Action             | Description              |
|--------------------|--------------------------|
| Envoyer un SMS     | Message individuel       |
| Nouvelle campagne  | Envoi en masse           |
| Creer un modele    | Message reutilisable     |
| Acheter des credits| Recharger le compte      |

ACTIVITE RECENTE
| Type      | Indicateur | Description            |
|-----------|------------|------------------------|
| SMS envoye| vert       | Envoye + statut delivre|
| Campagne  | vert       | Terminee + stats       |
| SMS recu  | orange     | Recu + statut lu/non lu|

API ENDPOINTS
GET /sms                                      -> Liste comptes
GET /sms/{account}                            -> Detail compte
GET /sms/{account}/consumption                -> Stats consommation
GET /sms/{account}/senders                    -> Liste expediteurs
GET /sms/{account}/outgoing                   -> SMS envoyes recents
POST /sms/{account}/credits/buy               -> Acheter credits
