## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche comptes SMS
   - Liste des comptes

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - STATS CARDS (4):
     * Crédit disponible (vert) + valeur €
     * Envoyés ce mois + trend
     * Taux délivrabilité + note
     * Réponses reçues

   - BLOC INFORMATIONS COMPTE:
     * Identifiant compte
     * Statut (badge)
     * Expéditeur par défaut
     * Date création
     * Dernier envoi

   - BLOC ACTIONS RAPIDES (4):
     * Envoyer un SMS (principal)
     * Campagnes
     * Recharger
     * Historique

   - TABLE DERNIERS SMS:
     * Date
     * Destinataire
     * Message (tronqué)
     * Crédits utilisés
     * Statut (badge)

## API ENDPOINTS

### Info compte
GET /sms/{serviceName}
Response: { name, status, creditsLeft, creditPrice, defaultSender, creationDate }

### Stats mois
GET /sms/{serviceName}/statistics
Query: { period: 'month' }
Response: { sentThisMonth, deliveryRate, responsesReceived, trends }

### Derniers SMS
GET /sms/{serviceName}/outgoing
Query: { limit: 5 }
Response: [{ date, receiver, message, credits, status }]

================================================================================

## REGLES METIER

- 1 SMS = 160 caractères (70 en Unicode)
- Crédit jamais négatif
- Taux délivrabilité = délivrés / envoyés
- Statuts: Délivré, En cours, Échec
- Réponses nécessitent option activée
