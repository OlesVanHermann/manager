## DESCRIPTION

Configuration générale du compte SMS. Paramètres d'envoi, limites,
notifications et sécurité.

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche comptes SMS
   - Liste des comptes SMS

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - BLOC PARAMETRES GENERAUX:
     * Nom du compte (éditable)
     * Callback URL
     * SMS entrants (toggle)
     * Accusé de réception (toggle)

   - BLOC EXPEDITEUR PAR DEFAUT:
     * Sélection expéditeur
     * Numéro de réponse

   - BLOC LIMITES D'ENVOI:
     * Limite quotidienne
     * Limite par minute
     * Seuil alerte crédit bas

   - BLOC NOTIFICATIONS:
     * Email notification
     * Toggle alerte crédit
     * Toggle rapport quotidien

   - BLOC SECURITE:
     * Token API (masqué + copie)
     * IPs autorisées

   - ACTIONS: Enregistrer, Annuler

================================================================================

## API ENDPOINTS

### Info compte
GET /sms/{serviceName}
Response: { name, callbackUrl, smsResponse, deliveryReceipt, defaultSender, ... }

### Modifier options
PUT /sms/{serviceName}
Body: { name?, callbackUrl?, smsResponse?, ... }

### Régénérer token
POST /sms/{serviceName}/regenerateToken
Response: { token }

### Modifier IPs autorisées
PUT /sms/{serviceName}/allowedIps
Body: { ips[] }

================================================================================

## REGLES METIER

- Callback URL doit être HTTPS
- Limite quotidienne max: 100 000
- Limite par minute max: 500
- Token API régénérable (invalidation immédiate)
- IPs au format CIDR ou adresse unique
