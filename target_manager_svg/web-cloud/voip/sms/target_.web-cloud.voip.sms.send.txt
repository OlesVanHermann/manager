# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des comptes SMS
4. RIGHT PANEL
   - Header: Nom compte + breadcrumb /Envoyer
   - NAV3: Général | [Envoyer] | Campagnes | Entrants | Historique | Modèles
   - Formulaire envoi:
     * Expéditeur (dropdown)
     * Destinataires (textarea + import CSV)
     * Message (textarea avec compteur)
     * Options (checkboxes)
     * Boutons Envoyer / Aperçu
   - Panel "Résumé": Calcul crédits
   - Panel "Modèles récents": Accès rapide

# CALCUL CRÉDITS
| Caractères | SMS | Crédits |
|------------|-----|---------|
| 0-160      | 1   | 1       |
| 161-306    | 2   | 2       |
| 307-459    | 3   | 3       |
| ...        | ... | ...     |

# INTERACTIONS
- Dropdown expéditeur → Sélectionner sender
- Textarea destinataires → Saisie numéros
- "Importer CSV" → Modal import fichier
- Textarea message → Saisie avec compteur temps réel
- Variables {NOM} etc → Insertion auto
- Checkbox planifier → Affiche date/heure
- Checkbox modèle → Affiche dropdown modèles
- Clic modèle récent → Remplit le message
- Clic "Aperçu" → Modal preview
- Clic "Envoyer" → POST API + confirmation

# API ENDPOINTS
GET /sms/{sn}/senders                  → Liste expéditeurs
GET /sms/{sn}/templates                → Liste modèles
POST /sms/{sn}/jobs                    → Envoyer SMS
POST /sms/{sn}/jobs/estimate           → Estimer coût
