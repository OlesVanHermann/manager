INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic ligne → Charge les détails de la ligne
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la fonctionnalité
- Clic ↻ → Actualiser les événements
- Clic filtre Type → Dropdown types d'événements
- Clic filtre Période → Dropdown périodes
- Clic pagination → Navigation pages

================================================================================

API ENDPOINTS

GET /telephony/{ba}/line → Liste lignes
GET /telephony/{ba}/line/{sn} → Détails ligne
GET /telephony/{ba}/line/{sn}/events → Liste événements
GET /telephony/{ba}/line/{sn}/events?type={type} → Filtrer par type
GET /telephony/{ba}/line/{sn}/events?dateStart={date}&dateEnd={date} → Filtrer par période
