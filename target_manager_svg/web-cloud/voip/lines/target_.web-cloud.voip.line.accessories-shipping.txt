# Cahier des Charges : telecom/line - Livraison accessoires

## 1. RESUME EXECUTIF

### Perimetre
Etape de selection de l'adresse et du mode de livraison pour les accessoires.

### Objectif fonctionnel
- Selectionner le contact de livraison
- Choisir le mode de livraison
- Ajouter un nouveau contact

## 4. APIS & DONNEES

### Endpoints
```
GET /me/contact
POST /me/contact
```

### Modele
```typescript
interface Contact {
  id: string;
  firstName: string;
  lastName: string;
  address: string;
  zipCode: string;
  city: string;
  country: string;
  phone: string;
}

interface ShippingOption {
  type: 'home' | 'relay' | 'express';
  delay: string;
  priceHT: number;
}
```
