Configuration avancée de la messagerie vocale. Paramètres généraux,
notifications, messages d'accueil et gestion du stockage.

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche lignes
   - Liste des lignes

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - BLOC PARAMETRES GENERAUX:
     * Nombre sonneries (dropdown)
     * Durée max message (input)
     * Langue répondeur (dropdown)
     * Toggle activer répondeur

   - BLOC NOTIFICATIONS:
     * Toggle email
     * Adresse email (input)
     * Toggle pièce jointe audio
     * Toggle SMS

   - BLOC MESSAGE ACCUEIL:
     * Radio: par défaut / personnalisé
     * Description

   - BLOC MESSAGE ABSENCE:
     * Toggle activer
     * Date début
     * Date fin

   - BLOC STOCKAGE:
     * Barre progression espace
     * Nombre messages

   - ACTIONS: Enregistrer, Annuler

## API ENDPOINTS

### Config messagerie
GET /telephony/{billingAccount}/line/{serviceName}/voicemail
Response: { enabled, ringCount, maxDuration, language, notificationEmail, attachAudio, usedSpace, messageCount }

### Modifier config
PUT /telephony/{billingAccount}/line/{serviceName}/voicemail
Body: { ringCount?, maxDuration?, language?, notificationEmail?, attachAudio? }

### Message personnalisé
POST /telephony/{billingAccount}/line/{serviceName}/voicemail/greeting
Body: FormData avec fichier audio

### Message absence
PUT /telephony/{billingAccount}/line/{serviceName}/voicemail/absence
Body: { enabled, startDate?, endDate? }

================================================================================

## REGLES METIER

- Sonneries: 1-10
- Durée max: 30-300 secondes
- Langues: fr, en, de, es, it
- Audio: WAV/MP3, max 5 MB
- Stockage: 50 MB par boîte
- Messages conservés 30 jours
