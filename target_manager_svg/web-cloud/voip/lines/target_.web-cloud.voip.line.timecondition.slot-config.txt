# Cahier des Charges : telecom/line - Configuration des creneaux

## 1. RESUME EXECUTIF

### Perimetre
Page de configuration des differents types de creneaux horaires.

### Objectif fonctionnel
- Visualiser tous les creneaux disponibles
- Configurer chaque creneau individuellement
- Definir les redirections par creneau

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/timeCondition/slots
    Response: {
      available: { enabled: true, queue: "..." },
      unavailable: { type: "voicemail", number: "..." },
      slot1: { type: "external", number: "+33612345678" },
      slot2: { configured: false },
      slot3: { configured: false }
    }
```
