| Ligne SIP         | Numéro, description, offre, statut, SIP  |

# API ENDPOINTS
GET /telephony/{ba}/line/{sn} - Détails ligne
GET /telephony/{ba}/line/{sn}/phone - Infos téléphone
GET /telephony/{ba}/line/{sn}/serviceInfos - Infos abonnement
PUT /telephony/{ba}/line/{sn} - Modifier ligne
POST /telephony/{ba}/line/{sn}/changePassword - Changer MDP SIP
POST /telephony/{ba}/line/{sn}/phone/reboot - Redémarrer téléphone
DELETE /telephony/{ba}/line/{sn} - Résilier ligne
