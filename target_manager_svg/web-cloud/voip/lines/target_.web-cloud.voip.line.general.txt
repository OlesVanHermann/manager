STRUCTURE PAGE
1. SOUS-NAV: Lignes (actif) | Numeros | Fax | Abreges
2. Left panel: Liste lignes du groupe
3. Right panel header: Numero + description + actions
4. NAV3: General (actif) | Telephone | Appels | Transferts | Messagerie | Consommation | Options
5. Cards:
   - Ligne SIP (numero, description, offre, statut, identifiant)
   - Telephone associe (modele, MAC, IP, firmware, actions)
   - Abonnement (dates, renouvellement)
   - Authentification SIP (mot de passe)

NAV3 LIGNES (7 tabs)
| Tab          | Route         | Description                    |
|--------------|---------------|--------------------------------|
| General      | /general      | Infos ligne + telephone        |
| Telephone    | /phone        | Config telephone detaillee     |
| Appels       | /calls        | Historique + filtrage          |
| Transferts   | /forward      | Renvois d'appels               |
| Messagerie   | /voicemail    | Messagerie vocale              |
| Consommation | /consumption  | Stats appels/fax               |
| Options      | /options      | Options ligne (call waiting...) |

ACTIONS HEADER
| Action        | Comportement                               |
|---------------|-------------------------------------------|
| MDP SIP       | Modal changement mot de passe SIP         |
| Changer offre | Modal upgrade/downgrade offre             |
| Resilier      | Modal confirmation resiliation            |

ACTIONS TELEPHONE
| Action     | Comportement                               |
|------------|-------------------------------------------|
| Reboot     | Redemarrage distant du telephone          |
| Configurer | Modal configuration avancee               |

API ENDPOINTS
GET  /telephony/{ba}/line/{sn}              -> Details ligne
GET  /telephony/{ba}/line/{sn}/serviceInfos -> Infos abonnement
GET  /telephony/{ba}/line/{sn}/phone        -> Infos telephone
PUT  /telephony/{ba}/line/{sn}              -> Modifier description
POST /telephony/{ba}/line/{sn}/changePassword -> Changer MDP SIP
POST /telephony/{ba}/line/{sn}/phone/reboot -> Redemarrer telephone
DELETE /telephony/{ba}/line/{sn}            -> Resilier ligne
