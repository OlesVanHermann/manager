## DESCRIPTION

Configuration de l'ordre de priorité des codecs audio pour les appels.
Permet d'activer/désactiver et réordonner les codecs par drag & drop.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| G.722                    | GET /telephony/.../codecs → activeCodecs[].name         |
| HD Voice                 | codec.quality                                           |
| 64 kbps                  | codec.bitrate                                           |
| 1                        | codec.priority                                          |
| Opus                     | availableCodecs[].name                                  |

================================================================================

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche lignes
   - Liste des lignes

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - BANNIERE INFO:
     * Explication négociation codecs

   - BLOC CODECS ACTIVES:
     * Liste ordonnée drag & drop
     * Numéro priorité
     * Nom codec
     * Badge qualité (HD/Standard/Faible bande)
     * Caractéristiques (bitrate, fréquence)
     * Lien réinitialiser ordre

   - BLOC CODECS DISPONIBLES:
     * Codecs non activés
     * Bouton activer chacun

   - ACTIONS: Enregistrer, Annuler

================================================================================

## API ENDPOINTS

### Liste codecs
GET /telephony/{billingAccount}/line/{serviceName}/codecs
Response: { activeCodecs: [{ name, priority, bitrate, quality }], availableCodecs: [...] }

### Modifier ordre
PUT /telephony/{billingAccount}/line/{serviceName}/codecs
Body: { codecs: [{ name, priority }] }

### Activer codec
POST /telephony/{billingAccount}/line/{serviceName}/codecs/{codecName}/activate

### Désactiver codec
DELETE /telephony/{billingAccount}/line/{serviceName}/codecs/{codecName}

================================================================================

## REGLES METIER

- Minimum 1 codec actif obligatoire
- G.722 = HD Voice (recommandé)
- G.711 = Standard téléphonie classique
- G.729 = Faible bande passante (ADSL)
- L'ordre détermine la négociation
- Reprovisionnement après modification
