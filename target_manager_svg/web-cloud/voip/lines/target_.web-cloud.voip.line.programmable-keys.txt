## DESCRIPTION

Configuration des touches programmables du téléphone. Assignation de fonctions
(appel rapide, BLF, transfert, etc.) aux touches physiques du poste.

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche lignes
   - Liste des lignes avec modèle téléphone

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - INFO BANNER: Modèle téléphone + nombre de touches

   - REPRÉSENTATION TÉLÉPHONE:
     * Schéma visuel du téléphone
     * Touches numérotées 1-10

   - TABLE CONFIGURATION:
     * N° touche
     * Libellé

     * Paramètre (numéro, etc.)
     * Actions: Éditer, Supprimer

   - PAGINATION TOUCHES: Page 1-5 | 6-10

   - ACTIONS: Enregistrer, Réinitialiser

## API ENDPOINTS

### Configuration téléphone
GET /telephony/{billingAccount}/line/{serviceName}/phone
Response: {
  model, functionKeysCount, macAddress
}

### Liste touches programmables
GET /telephony/{billingAccount}/line/{serviceName}/phone/functionKey
Response: [{
  keyNum, label, function, parameter
}]

### Configurer touche
PUT /telephony/{billingAccount}/line/{serviceName}/phone/functionKey/{keyNum}
Body: { label, function, parameter }

### Supprimer configuration touche
DELETE /telephony/{billingAccount}/line/{serviceName}/phone/functionKey/{keyNum}

### Réinitialiser toutes les touches
POST /telephony/{billingAccount}/line/{serviceName}/phone/functionKey/reset

## FONCTIONS DISPONIBLES

| Fonction      | Paramètre              | Description                  |
|---------------|------------------------|------------------------------|
| speedDial     | Numéro                 | Appel rapide                 |
| blf           | Ligne à superviser     | Supervision (BLF)            |
| transfer      | -                      | Transfert d'appel            |
| conference    | -                      | Conférence                   |
| pickup        | Groupe interception    | Interception d'appel         |
| dnd           | -                      | Ne pas déranger              |
| voicemail     | -                      | Messagerie vocale            |
| hold          | -                      | Mise en attente              |
| park          | Position               | Parcage d'appel              |

## REGLES METIER

- Nombre de touches selon modèle téléphone
- BLF: supervision lignes du même groupe
- Libellé: max 12 caractères
- Synchro auto après enregistrement
- Redémarrage téléphone si nécessaire
