STRUCTURE PAGE
1. SOUS-NAV: Lignes (actif) | Numeros | Fax | Abreges
2. Left panel: Liste lignes du groupe
3. NAV3: General | Telephone | Appels | Transferts (actif) | Messagerie | Consommation | Options
4. Cards:
   - Renvoi inconditionnel (toggle + destination)
   - Renvoi sur non-reponse (toggle + destination + delai)
   - Renvoi sur occupation (toggle + destination)
   - Renvoi sur injoignable (toggle + destination)
   - Banniere info priorite
   - Filtrage appels entrants (mode + liste)

TYPES DE RENVOI
| Type           | API Field                      | Priorite |
|----------------|--------------------------------|----------|
| Inconditionnel | forwardUnconditional           | 1        |
| Occupation     | forwardBusy                    | 2        |
| Non-reponse    | forwardNoReply                 | 3        |
| Injoignable    | forwardBackup                  | 4        |

DESTINATIONS POSSIBLES
| Type             | Description                    |
|------------------|--------------------------------|
| Numero externe   | +33... ou 00...                |
| Ligne interne    | Autre ligne du groupe          |
| Messagerie       | Voicemail de la ligne          |
| Numero special   | File attente, SVI...           |

MODES FILTRAGE
| Mode           | API Value    | Description                     |
|----------------|--------------|--------------------------------|
| Aucun          | none         | Tous appels acceptes           |
| Liste noire    | blacklist    | Numeros listes rejetes         |
| Liste blanche  | whitelist    | Seuls numeros listes acceptes  |

API ENDPOINTS
GET  /telephony/{ba}/line/{sn}/options              -> Config renvois
PUT  /telephony/{ba}/line/{sn}/options              -> Modifier renvois
GET  /telephony/{ba}/line/{sn}/callsFiltering       -> Liste filtrage
POST /telephony/{ba}/line/{sn}/callsFiltering       -> Ajouter filtre
DELETE /telephony/{ba}/line/{sn}/callsFiltering/{n} -> Supprimer filtre
