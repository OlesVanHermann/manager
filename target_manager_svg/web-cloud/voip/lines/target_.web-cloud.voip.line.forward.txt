# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des lignes
4. RIGHT PANEL
   - Header: Numéro + breadcrumb /Transferts
   - NAV3: Général | Téléphone | Appels | [Transferts] | Messagerie | Consommation | Options
   - NAV4: [Inconditionnel] | Occupation | Non-réponse
   - Card "Renvoi inconditionnel": Toggle + numéro + bouton Activer
   - Bannière info statut
   - Cards preview: Occupation + Non-réponse (avec liens vers config)
   - Bannière info modifications

# TYPES DE RENVOI
| Type           | Description                                   |
|----------------|-----------------------------------------------|
| Inconditionnel | Tous les appels renvoyés, téléphone ne sonne pas |
| Occupation     | Renvoi si ligne occupée                       |
| Non-réponse    | Renvoi après X secondes sans décrocher        |

# INTERACTIONS
- Clic NAV4 sous-tab → Navigation entre types de renvoi
- Toggle ON/OFF → Active/désactive le renvoi
- Input numéro → Saisie numéro destination
- Clic "Activer" → Activation du renvoi
- Clic "Configurer →" → Navigation vers config détaillée

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/options              → Options ligne (forwards)
PUT /telephony/{ba}/line/{sn}/options              → Modifier options
POST /telephony/{ba}/line/{sn}/options/forward     → Configurer renvoi

# OPTIONS FORWARD (API)
| Champ              | Type    | Description                    |
|--------------------|---------|--------------------------------|
| forwardUnconditional | boolean | Renvoi inconditionnel actif  |
| forwardUnconditionalNumber | string | Numéro destination       |
| forwardBusy        | boolean | Renvoi sur occupation actif    |
| forwardBusyNumber  | string  | Numéro destination occupation  |
| forwardNoReply     | boolean | Renvoi sur non-réponse actif   |
| forwardNoReplyNumber | string | Numéro destination non-réponse |
| forwardNoReplyDelay | integer | Délai avant renvoi (secondes) |
