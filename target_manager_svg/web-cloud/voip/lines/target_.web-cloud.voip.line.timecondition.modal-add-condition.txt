# Cahier des Charges : telecom/line - Modal ajout plage horaire

## 1. RESUME EXECUTIF

### Perimetre
Modal de creation d'une nouvelle plage horaire.

### Objectif fonctionnel
- Selectionner le jour
- Definir la repetition
- Choisir le creneau de redirection
- Definir l'heure de debut/fin

---

## 2. LAYOUT

### Modal popover 480x600

---

## 3. COMPOSANTS UI

### En-tete modal
| Element | Description |
|---------|-------------|
| Titre | Ajouter une plage horaire |
| Separateur | Ligne horizontale |

### Champ jour
| Element | Description |
|---------|-------------|
| Label | Jour de la semaine |
| Valeur | Lundi (texte statique) |

### Champ repetition (cliquable)
| Element | Description |
|---------|-------------|
| Label | Repeter |
| Valeur | Tous les lundis |
| Indicateur | Fleche > |
| Fond | Gris pale (#FAFAFA) |

### Champ redirection (cliquable)
| Element | Description |
|---------|-------------|
| Label | Rediriger les appels vers |
| Pastille couleur | Carre colore du creneau |
| Nom creneau | Creneau 1 |
| Detail | Rediriger vers le numero : +33 1 23 45 67 89 |
| Indicateur | Fleche > |

### Champs plage horaire
| Element | Description |
|---------|-------------|
| Label | Plage horaire |
| Input De | Select heure debut |
| Input A | Select heure fin |

### Legende creneaux disponibles
| Element | Description |
|---------|-------------|
| Titre | CRENEAUX DISPONIBLES |
| Creneau vert | Horaires d'ouverture |
| Creneau bleu | Creneau 1 - +33 1 23 45 67 89 |
| Creneau jaune | Creneau 2 - Non configure |
| Creneau rose | Creneau 3 - Non configure |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Ajouter | Primary | Cree la plage |
| Annuler | Default | Ferme la modal |

---

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/line/{serviceName}/timeCondition
    Body: {
      day: "monday",
      timeFrom: "09:00",
      timeTo: "12:00",
      slot: "slot1",
      repeat: "weekly" | "once"
    }
```

### Options repetition
| Option | Description |
|--------|-------------|
| once | Une seule fois |
| weekly | Toutes les semaines |
| weekdays | Jours ouvres |
| custom | Personnalise |
