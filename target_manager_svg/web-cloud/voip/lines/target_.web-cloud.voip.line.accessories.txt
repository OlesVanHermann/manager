## DESCRIPTION

Wizard de commande d'accessoires téléphoniques (casques, alimentations,
modules d'extension). Interface en 4 étapes avec grille de produits.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| Jabra Evolve2 65         | GET /telephony/accessories → name                       |
| 249,00 € HT              | accessories[].price                                     |
| Casque sans fil...       | accessories[].description                               |
| Jabra, Plantronics...    | accessories[].brand                                     |
| 1                        | cart.quantity (state local)                             |
| 249,00 €                 | cart.total (calculé)                                    |

================================================================================

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche lignes
   - Liste des lignes avec téléphone

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - WIZARD STEPS (4 étapes):
     * 1. Sélection (actif)
     * 2. Livraison
     * 3. Résumé
     * 4. Finalisation

   - FILTRES MARQUE:
     * Toutes | Jabra | Plantronics | Cisco | ...

   - GRILLE ACCESSOIRES (4 colonnes):
     * Image produit
     * Nom + description
     * Prix HT
     * Spinner quantité (0-99)
     * Sélection visuelle (bordure bleue)

   - RÉCAPITULATIF PANIER:
     * Nombre d'articles
     * Liste sélection
     * Total HT
     * Bouton Continuer

================================================================================

## API ENDPOINTS

### Liste accessoires
GET /telephony/accessories
Query: { brand, compatible }
Response: [{
  id, name, description, brand,
  price, image, compatible: []
}]

### Commander accessoires
POST /order/telephony/accessories
Body: {
  accessories: [{ id, quantity }],
  shippingAddress: { ... }
}

### Suivre commande
GET /telephony/accessories/orders/{orderId}

================================================================================

## ÉTAPES WIZARD

| Étape | Description                              |
|-------|------------------------------------------|
| 1     | Sélection des accessoires                |
| 2     | Adresse de livraison                     |
| 3     | Résumé de commande                       |
| 4     | Paiement et confirmation                 |

================================================================================

## REGLES METIER

- Quantité max: 99 par article
- Au moins 1 accessoire pour continuer
- Filtrage par compatibilité téléphone
- Prix HT, TVA calculée à l'étape 3
- Livraison sous 3-5 jours ouvrés
