## DESCRIPTION

Configuration du nombre d'appels simultanés (trunk SIP). Affiche l'usage
actuel, l'historique et permet de modifier la capacité.

## API ENDPOINTS

### Config trunk
GET /telephony/{billingAccount}/line/{serviceName}/trunk
Response: { simultaneousCalls, currentCalls, peakCalls, peakDate, includedCalls, pricePerExtraCall }

### Usage historique
GET /telephony/{billingAccount}/line/{serviceName}/trunk/usage
Query: { period: '7d' | '30d' }
Response: [{ date, maxCalls }]

### Modifier nombre
PUT /telephony/{billingAccount}/line/{serviceName}/trunk
Body: { simultaneousCalls: number }

### Calculer prix
GET /telephony/{billingAccount}/line/{serviceName}/trunk/pricing
Query: { simultaneousCalls: number }
Response: { includedCalls, extraCalls, monthlyPrice }

================================================================================

## REGLES METIER

- Minimum: 1 appel simultané
- Maximum: 50 par ligne (peut varier selon offre)
- 2 appels inclus dans offre de base
- Facturation mensuelle des supplémentaires
- Modification effective immédiatement
- Réduction possible en fin de mois
