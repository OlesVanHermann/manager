## DESCRIPTION

Page de redémarrage à distance du téléphone IP. Permet un redémarrage
simple ou avec reprovisionnement, avec historique des redémarrages.

================================================================================

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche lignes
   - Liste des lignes

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - BANNIERE WARNING:
     * Avertissement interruption appels
     * Durée estimée

   - BLOC ETAT TELEPHONE:
     * Modèle
     * Adresse MAC
     * Statut (badge)
     * Dernier redémarrage

   - BLOC OPTIONS:
     * Radio: Redémarrage simple
     * Radio: Avec reprovisionnement

   - TABLE HISTORIQUE:
     * Date
     * Type
     * Durée
     * Statut (badge)

   - ACTIONS: Redémarrer, Annuler

================================================================================

## API ENDPOINTS

### Info téléphone
GET /telephony/{billingAccount}/line/{serviceName}/phone
Response: { model, macAddress, status, lastReboot }

### Redémarrer
POST /telephony/{billingAccount}/line/{serviceName}/phone/reboot
Body: { withProvision: boolean }

### Historique redémarrages
GET /telephony/{billingAccount}/line/{serviceName}/phone/reboot/history
Response: [{ date, type, duration, status }]

================================================================================

## REGLES METIER

- Redémarrage simple: ~1 minute
- Avec reprovisionnement: ~2 minutes
- Interrompt les appels en cours
- Maximum 5 redémarrages par heure
- Historique conservé 30 jours
