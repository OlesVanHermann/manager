# Cahier des Charges : telecom/line - Fax entrants

## 1. RESUME EXECUTIF

### Perimetre
Page de consultation des fax entrants d'une ligne.

### Objectif fonctionnel
- Consulter l'historique des fax recus
- Voir le nombre de pages
- Filtrer par periode

### Resume periode
| Element | Description |

| Colonne | Description |
|---------|-------------|

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/statistics/fax/incoming
```

### Modele
```typescript
interface IncomingFax {
  id: string;
  date: string;
  sender: string;
  pages: number;
}

interface FaxIncomingSummary {
  totalFax: number;
  totalPages: number;
  period: { start: string; end: string };
}
```
