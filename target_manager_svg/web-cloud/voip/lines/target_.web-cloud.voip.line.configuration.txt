# STRUCTURE PAGE
1. LEFT PANEL - Ligne + NAV3 (Configuration actif)
2. RIGHT PANEL:
   - TITRE "Configuration de la ligne"
   - IDENTITE: Description, CallerID, Nom présenté
   - OPTIONS: Toggles appel en attente, double appel, etc.
   - CODECS: Liste ordonnée draggable
   - SIP: Infos connexion SIP

# OPTIONS D'APPEL
| Option               | Default |
|----------------------|---------|
| Appel en attente     | ON      |
| Présentation numéro  | ON      |
| Mode intercom        | OFF     |
| Double appel         | ON      |
| Enregistrement       | OFF     |

# CODECS DISPONIBLES
G.722 (HD), G.711a, G.711u, G.729, Opus

# API ENDPOINTS
GET /telephony/{ba}/line/{serviceName} - Config ligne
PUT /telephony/{ba}/line/{serviceName} - Modifier config
POST /telephony/{ba}/line/{serviceName}/password - Regen password
