## API ENDPOINTS

### Liste événements
GET /telephony/{billingAccount}/line/{serviceName}/events
Query: { type, dateTime.from, dateTime.to }
Response: [{
  id, dateTime, type, description, origin, status, details
}]

### Détail événement
GET /telephony/{billingAccount}/line/{serviceName}/events/{id}
Response: { ... + fullDetails }

### Statistiques événements
GET /telephony/{billingAccount}/line/{serviceName}/events/stats
Query: { period: '7d' }
Response: {
  incomingCalls, outgoingCalls, configurations, errors, security
}
