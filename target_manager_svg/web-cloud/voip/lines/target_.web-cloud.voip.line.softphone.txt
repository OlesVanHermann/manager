# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Header: Numéro + description
   - NAV3: Tabs (Softphone actif)
   - Contenu: Configuration softphone

# BLOCS
1. Toggle activation softphone
2. Liens de téléchargement (App Store, Google Play, lien direct)
3. Configuration appareils (table avec ID, Nom, OS, User Agent)
4. Personnalisation thème (5 couleurs)
5. Logo personnalisé (upload)

# INTERACTIONS
- Toggle → Active/désactive le softphone
- "Générer un jeton" → Crée nouveau token pour appareil
- "Supprimer" → Supprime un appareil autorisé
- Clic couleur thème → Sélectionne couleur
- Upload logo → Téléverse image personnalisée
