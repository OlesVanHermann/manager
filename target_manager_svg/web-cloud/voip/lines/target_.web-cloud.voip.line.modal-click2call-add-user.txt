# Cahier des Charges : telecom/line - Modal ajout utilisateur Click2Call

## 1. RESUME EXECUTIF

### Perimetre
Modal d'ajout d'un utilisateur Click2Call.

### Objectif fonctionnel
- Creer un nouvel utilisateur Click2Call
- Definir identifiant et mot de passe
- Appliquer a plusieurs lignes

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/line/{serviceName}/click2Call/user
```

### Modele
```typescript
interface Click2CallUser {
  login: string;
  password: string;
}
```

## 6. REGLES METIER

- Identifiant unique par ligne
- Format identifiant: alphanum sans espaces
- Mot de passe 8 caracteres minimum
- Application multi-lignes possible
