# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des lignes du groupe avec sélection
4. RIGHT PANEL
   - Header: Numéro + description + badge statut + actions (MDP SIP, Résilier)
   - NAV3: [Général] | Téléphone | Appels | Transferts | Messagerie | Consommation | Options
   - Card "Ligne SIP": Numéro, description, offre, statut, identifiant SIP
   - Card "Téléphone associé": Modèle, MAC, IP, Firmware, boutons Reboot/Configurer
   - Card "Abonnement": Dates création/expiration, renouvellement, boutons
   - Card "Actions rapides": MDP SIP, Domaine, Softphone, Convertir
   - Bannière info

# INTERACTIONS
- Clic ligne LEFT PANEL → Sélection
- Clic tab NAV3 → Navigation onglet
- Clic "MDP SIP" → Modal changement mot de passe
- Clic "Résilier" → Modal confirmation résiliation
- Clic "Reboot" → Action redémarrage téléphone
- Clic "Configurer" → Page configuration téléphone
- Clic action rapide → Modal/page correspondante

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}           → Détails ligne
GET /telephony/{ba}/line/{sn}/serviceInfos → Infos abonnement
GET /telephony/{ba}/line/{sn}/phone     → Infos téléphone
GET /telephony/{ba}/line/{sn}/sipAccounts → Comptes SIP
POST /telephony/{ba}/line/{sn}/changePassword → Changer MDP
POST /telephony/{ba}/line/{sn}/phone/reboot → Redémarrer téléphone
DELETE /telephony/{ba}/line/{sn}        → Résilier
