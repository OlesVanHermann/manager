# Cahier des Charges : telecom/line - Modal suppression softphones

## 1. RESUME EXECUTIF

### Perimetre
Modal de confirmation de suppression de tous les appareils softphone.

### Objectif fonctionnel
- Confirmer la suppression de tous les appareils
- Avertir de l'invalidation des tokens
