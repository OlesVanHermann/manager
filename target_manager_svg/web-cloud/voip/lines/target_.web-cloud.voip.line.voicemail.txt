STRUCTURE PAGE
1. SOUS-NAV: Lignes (actif) | Numeros | Fax | Abreges
2. Left panel: Liste lignes du groupe
3. NAV3: General | Telephone | Appels | Transferts | Messagerie (actif) | Consommation | Options
4. Cards:
   - Messagerie vocale (toggle, PIN, email, format, joindre)
   - Annonce personnalisee (upload, ecouter, enregistrer)
   - Messages vocaux (table)

FORMATS AUDIO
| Format | Extension | Description          |
|--------|-----------|----------------------|
| MP3    | .mp3      | Compresse, leger     |
| WAV    | .wav      | Haute qualite        |
| OGG    | .ogg      | Libre, compresse     |

STATUTS MESSAGE
| Statut | Couleur   | Description          |
|--------|-----------|----------------------|
| Non lu | #DBEAFE   | Message non ecoute   |

ACTIONS MESSAGE
| Action      | Comportement                    |
|-------------|---------------------------------|
| Ecouter     | Player audio inline             |
| Telecharger | Download fichier audio          |
| Supprimer   | Suppression avec confirmation   |

API ENDPOINTS
GET  /telephony/{ba}/line/{sn}/options              -> Config voicemail
PUT  /telephony/{ba}/line/{sn}/options              -> Modifier config
GET  /telephony/{ba}/voicemail/{vm}/greetings       -> Annonce
POST /telephony/{ba}/voicemail/{vm}/greetings       -> Upload annonce
GET  /telephony/{ba}/voicemail/{vm}/directories     -> Messages
GET  /telephony/{ba}/voicemail/{vm}/directories/{m} -> Detail message
DELETE /telephony/{ba}/voicemail/{vm}/directories/{m} -> Supprimer
