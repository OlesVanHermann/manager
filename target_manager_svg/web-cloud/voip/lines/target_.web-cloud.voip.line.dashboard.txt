## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche lignes
   - Liste des lignes

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - STATUS CARD:
     * État ligne (badge coloré)
     * Durée enregistrement

   - STATS CARDS (4):
     * Appels aujourd'hui (entrants/sortants)
     * Durée totale (moyenne/max)
     * Messages vocaux (non lus)

   - BLOC INFORMATIONS:
     * Numéro
     * Offre
     * Téléphone + statut connexion
     * Adresse IP
     * Groupe facturation

   - BLOC ACTIONS RAPIDES (6 boutons):
     * Click2Call
     * Changer mot de passe
     * Voir consommation
     * Configurer renvois
     * Messagerie vocale
     * Configurer téléphone

   - DERNIERS APPELS (3 lignes):
     * Direction + numéro
     * Nom contact
     * Date/heure
     * Durée
     * Statut (badge)
     * Lien "Voir tout"

## API ENDPOINTS

### Info ligne
GET /telephony/{billingAccount}/line/{serviceName}
Response: { number, offer, status, lastRegistrationIp, phone }

### Stats aujourd'hui
GET /telephony/{billingAccount}/line/{serviceName}/statistics
Query: { period: 'day' }
Response: { incomingCalls, outgoingCalls, totalDuration, avgDuration }

### Messagerie vocale
GET /telephony/{billingAccount}/line/{serviceName}/voicemail
Response: { unreadCount, totalCount }

### Derniers appels
GET /telephony/{billingAccount}/line/{serviceName}/calls
Query: { limit: 3 }
Response: [{ number, contactName, direction, date, duration, status }]
