## DESCRIPTION

Configuration du nombre d'appels simultanés autorisés sur une ligne SIP Trunk.
Gestion des canaux voix et facturation associée.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| 10                       | GET /telephony/.../line → simultaneousCalls             |
| 2 appels                 | line.includedCalls                                      |
| 8 appels                 | line.additionalCalls                                    |
| 2,00 €                   | pricing.additionalCallPrice                             |

## API ENDPOINTS

### Configuration actuelle
GET /telephony/{billingAccount}/line/{serviceName}
Response: {
  simultaneousCalls, includedCalls, additionalCalls
}

### Tarification
GET /telephony/{billingAccount}/line/{serviceName}/simultaneousCalls/pricing
Response: {
  includedCallsPrice: 0,
  additionalCallPrice: 2.00,
  currency: 'EUR'
}

### Modifier appels simultanés
PUT /telephony/{billingAccount}/line/{serviceName}/simultaneousCalls
Body: { quantity: 10 }
