## DESCRIPTION

Conversion d'une ligne téléphonique en numéro alias. Opération irréversible
qui transforme la ligne en simple redirection vers une autre ligne.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| +33 9 12 34 56 78        | GET /telephony/.../line → number                        |
| Ligne SIP                | line.type                                               |
| VoIP Entreprise          | line.offer                                              |
| Cisco 8845               | line.phone.model                                        |

================================================================================

## STRUCTURE PAGE

1. LEFT PANEL (280px)

2. RIGHT PANEL (1096px)

   - WARNING BANNER: Opération irréversible

   - BLOC INFORMATIONS ACTUELLES:
     * Numéro

     * Téléphone associé

   - BLOC DESTINATION:
     * Dropdown sélection ligne destination

   - CHECKBOX CONFIRMATION

   - ACTIONS: Convertir (rouge), Annuler

================================================================================

## API ENDPOINTS

### Informations ligne
GET /telephony/{billingAccount}/line/{serviceName}
Response: {
  number, type, offer, phone: { model }
}

### Liste lignes destination possibles
GET /telephony/{billingAccount}/line
Response: [{ serviceName, description }]

### Convertir en alias
POST /telephony/{billingAccount}/line/{serviceName}/convertToNumber
Body: {
  destinationLine: serviceName
}

================================================================================

## REGLES METIER

- Opération irréversible
- Nécessite une ligne destination dans le même groupe
- Supprime: téléphone, messagerie, options ligne
- Conserve: le numéro en tant que redirection
- Facturation: passage au tarif alias
- Délai: effectif sous 24h

================================================================================

## CONSEQUENCES CONVERSION

| Avant (Ligne)        | Après (Alias)              |
|----------------------|----------------------------|
| Téléphone associé    | ❌ Supprimé                |
| Messagerie vocale    | ❌ Supprimée               |
| Renvois d'appels     | → Redirection simple       |
| Plages horaires      | ❌ Supprimées              |
| Facturation ligne    | → Facturation alias        |
