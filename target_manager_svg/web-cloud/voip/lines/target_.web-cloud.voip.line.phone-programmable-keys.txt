# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Back link: Retour au téléphone
   - Titre: Touches programmables - {modèle}
   - Lien bulk action
   - Table des touches
   - Warning L1
   - Boutons action

# TYPES DE TOUCHES
| Fonction | Description |
|----------|-------------|
| Ligne principale | Touche L1 réservée |
| Transfert supervisé | Transfert avec annonce |
| Supervision BLF | Voir statut autre ligne |
| Appel rapide | Numéro prédéfini |
| Parking d'appel | Mise en attente partagée |
| Non configuré | Touche libre |

# TABLE COLONNES
| Colonne | Contenu |
|---------|---------|
| Touche | Badge L1-Lx |
| Fonction | Type de touche |
| Paramètre | Numéro associé |
| Libellé | Texte affiché |
| Actions | ✎ Modifier, × Supprimer, + Configurer |

# SPÉCIFICITÉS
- L1 (jaune) = Ligne principale, modif risquée
- Touches vides = "+ Configurer"
- Touches configurées = "✎ ×"

# INTERACTIONS
- "+ Configurer" → Modal configuration
- "✎" → Modal édition
- "×" → Suppression avec confirmation
- "Enregistrer" → Sauvegarde + reboot téléphone
- "Réinitialiser" → Valeurs par défaut
- "Appliquer à plusieurs lignes" → Bulk modal

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/phone
GET /telephony/{ba}/line/{sn}/phone/functionKeys
PUT /telephony/{ba}/line/{sn}/phone/functionKeys/{keyNum}
DELETE /telephony/{ba}/line/{sn}/phone/functionKeys/{keyNum}
POST /telephony/{ba}/line/{sn}/phone/reboot
