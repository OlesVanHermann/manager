## DESCRIPTION

Modification du mot de passe SIP de la ligne. Utilisé pour l'authentification
des téléphones et softphones sur le serveur SIP OVHcloud.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| 0033912345678            | GET /telephony/.../line → sipUsername                   |
| sip.ovh.fr:5060          | line.sipServer                                          |

## STRUCTURE PAGE

1. LEFT PANEL (280px)

   - BLOC REGLES:
     * Liste des exigences

   - ACTIONS: Modifier, Annuler

================================================================================

## API ENDPOINTS

### Informations authentification
GET /telephony/{billingAccount}/line/{serviceName}
Response: {
  sipUsername, sipServer, sipPort
}

### Modifier mot de passe
POST /telephony/{billingAccount}/line/{serviceName}/changePassword
Body: { password }

### Générer mot de passe aléatoire
POST /telephony/{billingAccount}/line/{serviceName}/generatePassword
Response: { password }

================================================================================

## REGLES MOT DE PASSE

| Règle              | Valeur                    |
|--------------------|---------------------------|
| Longueur min       | 8 caractères              |
| Majuscule          | Au moins 1                |
| Minuscule          | Au moins 1                |
| Chiffre            | Au moins 1                |
| Caractère spécial  | Au moins 1 (!@#$%^&*)     |

================================================================================

## REGLES METIER

- Changement déconnecte tous les appareils
- Nouveau mdp effectif immédiatement
- Pas d'historique des anciens mots de passe
- Notification email envoyée
