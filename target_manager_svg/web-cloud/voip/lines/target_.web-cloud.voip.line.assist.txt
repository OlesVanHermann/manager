Gestion des demandes d'assistance pour une ligne téléphonique. Création de
tickets et suivi de l'historique des interventions.

## API ENDPOINTS

### Liste tickets pour service
GET /support/tickets?serviceName={serviceName}
Response: [{
  ticketId, subject, status, creationDate, lastUpdate, category
}]

### Créer ticket
POST /support/tickets
Body: {
  serviceName, category, subject, body
}

### Détail ticket
GET /support/tickets/{ticketId}
Response: {
  ticketId, subject, status, messages: [{ date, body, from }]
}

### Répondre ticket
POST /support/tickets/{ticketId}/reply
Body: { body }

### Fermer ticket
POST /support/tickets/{ticketId}/close
