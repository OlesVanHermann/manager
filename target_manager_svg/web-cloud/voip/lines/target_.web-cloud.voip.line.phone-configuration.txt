# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - VoIP actif
3. LEFT PANEL - Liste des lignes du groupe
4. RIGHT PANEL
   - Back link: Retour au téléphone
   - Titre: Configuration du téléphone
   - Sections: Réseau, Affichage, Audio

# SECTIONS
1. Paramètres réseau
   - Mode réseau (DHCP/Statique)
   - Serveur NTP
   - VLAN ID
   - QoS (toggle)

2. Affichage
   - Langue interface
   - Fuseau horaire
   - Format heure (12h/24h)
   - Rétroéclairage

3. Audio
   - Volume sonnerie (slider)
   - Volume combiné (slider)
   - Haut-parleur (toggle)

# INTERACTIONS
- Dropdowns → Sélection options
- Toggles → Active/désactive
- Sliders → Ajuste volume
- "Appliquer et redémarrer" → Sauvegarde + reboot
- "Annuler" → Annule modifications

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/phone
PUT /telephony/{ba}/line/{sn}/phone
POST /telephony/{ba}/line/{sn}/phone/reboot
