## API ENDPOINTS

### Info téléphone
GET /telephony/{billingAccount}/line/{serviceName}/phone
Response: { model, macAddress, firmware, status, ipAddress }

### Config téléphone
GET /telephony/{billingAccount}/line/{serviceName}/phone/config
Response: { ringVolume, speakerVolume, ringtone, backlight, language, codecs[] }

### Modifier config
PUT /telephony/{billingAccount}/line/{serviceName}/phone/config
Body: { ringVolume?, speakerVolume?, ringtone?, backlight?, language? }

### Redémarrer
POST /telephony/{billingAccount}/line/{serviceName}/phone/reboot

### Reprovisionner
POST /telephony/{billingAccount}/line/{serviceName}/phone/provision

### MAJ Firmware
POST /telephony/{billingAccount}/line/{serviceName}/phone/firmware

### Reset usine
POST /telephony/{billingAccount}/line/{serviceName}/phone/factoryReset

================================================================================

## REGLES METIER

- Redémarrage: téléphone indisponible ~1 min
- Reprovisionnement: récupère config serveur
- Firmware: téléchargement + redémarrage auto
- Reset usine: efface config locale
