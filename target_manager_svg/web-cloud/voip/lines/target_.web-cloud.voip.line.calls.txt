API ENDPOINTS
GET /telephony/{ba}/line/{sn}/automaticCall                -> Liste appels
GET /telephony/{ba}/line/{sn}/automaticCall?direction=incoming -> Entrants
GET /telephony/{ba}/line/{sn}/automaticCall?direction=outgoing -> Sortants
GET /telephony/{ba}/line/{sn}/automaticCall/{id}           -> Detail appel
