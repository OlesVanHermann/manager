## DESCRIPTION

Configuration des options avancées de la ligne VoIP. Présentation du numéro,
appels simultanés, Click-to-Call, interception et paramètres techniques.

## BLOCS OPTIONS

### Présentation du numéro (CLIP)
| Option          | Type    | Valeurs                    |
|-----------------|---------|----------------------------|
| Afficher numéro | Toggle  | Activé/Désactivé           |
| Numéro présenté | Select  | Liste numéros du groupe    |

### Appels simultanés
| Option          | Type    | Valeurs                    |
|-----------------|---------|----------------------------|
| Nb max          | Select  | 1 à 10                     |
| Double appel    | Toggle  | Activé/Désactivé           |

### Click-to-Call
| Option          | Type    | Valeurs                    |
|-----------------|---------|----------------------------|
| Activer         | Toggle  | Activé/Désactivé           |
| URL callback    | Text    | URL API (copie)            |

### Interception d'appel
| Option          | Type    | Description                |
|-----------------|---------|----------------------------|
| Autoriser       | Toggle  | Permet *8 + numéro         |

### Musique d'attente
| Option          | Type    | Valeurs                    |
|-----------------|---------|----------------------------|
| Musique         | Select  | Classique, Jazz, Pop, Custom |
| Écouter         | Bouton  | Preview audio              |
| Personnaliser   | Bouton  | Upload fichier audio       |

## CODES TÉLÉPHONE

| Code   | Fonction                    |
|--------|-----------------------------|
| *8     | Interception (+ numéro)     |
| *21*   | Renvoi inconditionnel       |
| #21#   | Annuler renvoi incond.      |
| *67*   | Renvoi sur occupation       |
| *61*   | Renvoi sur non-réponse      |
