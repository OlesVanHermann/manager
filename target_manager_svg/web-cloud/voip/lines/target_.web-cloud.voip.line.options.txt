STRUCTURE PAGE
1. SOUS-NAV: Lignes (actif) | Numeros | Fax | Abreges
2. Left panel: Liste lignes du groupe
3. NAV3: General | Telephone | Appels | Transferts | Messagerie | Consommation | Options (actif)
4. Cards:
   - Options d'appel (double appel, CLIP, CLIR, intercom)
   - Restrictions d'appels (internationaux, mobiles, surtaxes)
   - Numero affiche (choix numero present)
   - Langue et region (langue annonces, timezone)

OPTIONS D'APPEL
| Option              | API Field                  | Description                |
|---------------------|----------------------------|----------------------------|
| Double appel        | callWaiting                | Signal appel entrant       |
| Presentation numero | displayNumber              | Affiche numero sortant     |
| Numero masque       | identificationRestriction  | Cache numero sortant       |
| Interphone          | intercom                   | Auto-reponse appels int.   |
