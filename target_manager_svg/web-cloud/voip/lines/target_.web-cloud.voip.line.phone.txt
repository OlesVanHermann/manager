- Clic "Mise à jour firmware" → Modal mise à jour
- Clic "Touches programmables" → Page configuration touches

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/phone             → Infos téléphone
POST /telephony/{ba}/line/{sn}/phone/reboot     → Redémarrer
PUT /telephony/{ba}/line/{sn}/phone             → Modifier config
GET /telephony/{ba}/line/{sn}/phone/functionKey → Touches programmables
PUT /telephony/{ba}/line/{sn}/phone/functionKey/{id} → Modifier touche
POST /telephony/{ba}/line/{sn}/attachDevice     → Rattacher
POST /telephony/{ba}/line/{sn}/detachDevice     → Détacher
