STRUCTURE PAGE
1. SOUS-NAV: Lignes (actif) | Numeros | Fax | Abreges
2. Left panel: Liste lignes du groupe
3. NAV3: General | Telephone (actif) | Appels | Transferts | Messagerie | Consommation | Options
4. Cards:
   - Telephone (infos materiel + actions)
   - Codecs audio (liste avec priorite)
   - Touches programmables (table config)

CODECS DISPONIBLES
| Codec    | Description         | Bande passante |
|----------|---------------------|----------------|
| G.711a   | PCMA, HD quality    | 64 kbps        |
| G.711u   | PCMU, standard      | 64 kbps        |
| G.729    | Compresse           | 8 kbps         |
| G.722    | HD Voice            | 64 kbps        |
| OPUS     | Moderne, adaptif    | Variable       |

TYPES TOUCHES PROGRAMMABLES
| Type       | Description                          |
|------------|--------------------------------------|
| BLF        | Supervision presence (voyant)        |
| Raccourci  | Appel rapide numero                  |
| Transfert  | Code de transfert                    |
| DTMF       | Sequence DTMF                        |
| Park       | Parking appel                        |
| Intercom   | Interphone direct                    |

ACTIONS TELEPHONE
| Action       | Comportement                               |
|--------------|--------------------------------------------|
| Reboot       | Redemarrage distant                        |
| Reconfigurer | Reprovisionning config OVH                 |
| Detacher     | Dissocier telephone de la ligne            |

API ENDPOINTS
GET  /telephony/{ba}/line/{sn}/phone                -> Details telephone
POST /telephony/{ba}/line/{sn}/phone/reboot         -> Redemarrer
POST /telephony/{ba}/line/{sn}/phone/resetConfig    -> Reconfigurer
DELETE /telephony/{ba}/line/{sn}/phone              -> Detacher
GET  /telephony/{ba}/line/{sn}/options              -> Codecs
PUT  /telephony/{ba}/line/{sn}/options              -> Modifier codecs
GET  /telephony/{ba}/line/{sn}/phone/functionKey    -> Touches
PUT  /telephony/{ba}/line/{sn}/phone/functionKey/{k}-> Modifier touche
