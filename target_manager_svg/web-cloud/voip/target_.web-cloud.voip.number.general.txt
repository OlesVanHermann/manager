INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic numéro → Charge les détails du numéro
- Clic NAV4 → Navigation vers l'onglet
- Clic ✎ description → Modal édition description
- Clic tile → Navigation vers la fonctionnalité
- Clic Supprimer → Modal confirmation suppression

API ENDPOINTS

GET /telephony/{ba}/number/{sn} → Détails numéro

GET /telephony/{ba}/easyHunting/{sn} → Config CCS
GET /telephony/{ba}/easyHunting/{sn}/agent → Liste agents
DELETE /telephony/{ba}/number/{sn}/changeFeatureType → Supprimer config
