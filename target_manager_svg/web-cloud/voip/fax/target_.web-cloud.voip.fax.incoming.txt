## API ENDPOINTS

### Liste fax recus
GET /telephony/{billingAccount}/fax/{serviceName}/incoming
Query: { dateTime.from, dateTime.to }
Response: [{
  id, dateTime, caller, pages, duration, quality, status, size
}]

### Telecharger fax
GET /telephony/{billingAccount}/fax/{serviceName}/incoming/{id}/download
Response: PDF/TIFF file

### Supprimer fax
DELETE /telephony/{billingAccount}/fax/{serviceName}/incoming/{id}

### Export liste
GET /telephony/{billingAccount}/fax/{serviceName}/incoming/export
Query: { dateTime.from, dateTime.to, format: 'csv' }
