# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des lignes fax
4. RIGHT PANEL
   - Header: Numéro fax + badge + breadcrumb /Logo
   - NAV3: Général | Paramètres | [Logo] | Notifications | Campagnes
   - Bannière info: Explication logo fax
   - Section logo actuel (prévisualisation + infos)
   - Section téléversement (drag & drop)
   - Section spécifications techniques

# SECTIONS DE LA PAGE

## 1. Logo actuel
- Zone de prévisualisation du logo
- Informations: nom fichier, dimensions, taille, date
- Actions: Changer, Supprimer, Télécharger

## 2. Téléverser un nouveau logo
- Zone drag & drop
- Bouton parcourir fichiers
- Support multi-format

## 3. Spécifications techniques
- Formats acceptés: PNG, JPEG, GIF
- Dimensions max: 500 x 100 pixels
- Taille max: 100 KB
- Recommandation: fond blanc/transparent

# UTILISATION DU LOGO
Le logo configuré apparaît :
- En en-tête de chaque page de fax envoyé
- Sur la page de garde (si configurée)
- Dans le récapitulatif de transmission

# BONNES PRATIQUES
1. Utiliser un logo vectoriel converti en PNG
2. Privilégier un fond blanc ou transparent
3. Garder une résolution de 300 DPI pour impression
4. Éviter les logos trop détaillés (perte de qualité fax)
5. Tester le rendu en envoyant un fax test

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Changer le logo" → Ouvrir sélecteur fichier
- Drag & drop zone → Upload fichier
- Clic "Supprimer" → Supprimer logo (confirmation)
- Clic "Télécharger" → Download logo actuel

# API ENDPOINTS
GET /telephony/{ba}/fax/{sn}/logo → Récupérer logo actuel
PUT /telephony/{ba}/fax/{sn}/logo → Mettre à jour logo
DELETE /telephony/{ba}/fax/{sn}/logo → Supprimer logo
POST /telephony/{ba}/fax/{sn}/logo/upload → Upload nouveau logo

# FORMATS SUPPORTÉS
| Format | Extension | Notes |
|--------|-----------|-------|

| JPEG | .jpg, .jpeg | Bon pour photos |
| GIF | .gif | Support basique |
