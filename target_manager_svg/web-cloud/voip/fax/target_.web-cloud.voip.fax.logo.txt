## API ENDPOINTS

### Logo actuel
GET /telephony/{billingAccount}/fax/{serviceName}/logo
Response: {
  hasLogo: boolean,
  filename, format, dimensions: { width, height }, size,
  url (preview)
}

### Upload logo
POST /telephony/{billingAccount}/fax/{serviceName}/logo
Body: multipart (image file)
Response: { success, filename, ... }

### Supprimer logo
DELETE /telephony/{billingAccount}/fax/{serviceName}/logo

### Preview en-tête
GET /telephony/{billingAccount}/fax/{serviceName}/logo/preview
Response: Image (preview de l'en-tête complet)
