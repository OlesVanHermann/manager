================================================================================
CAHIER DES CHARGES - old_manager.telecom.fax.modal-remove-campaign
================================================================================

1. IDENTIFICATION
-----------------
Chemin     : telecom/fax
Composant  : Modal suppression campagne fax
Type       : Modal confirmation
Framework  : AngularJS 1.7.x

2. DESCRIPTION FONCTIONNELLE
----------------------------
Modal de confirmation pour supprimer une campagne de fax.
Demande confirmation avant suppression definitive.

3. STRUCTURE UI
---------------
+------------------------------------------------------------------+
| [X] Fermer                                                       |
+------------------------------------------------------------------+
| Supprimer une campagne                                           |
|                                                                  |
| Voulez-vous vraiment supprimer la campagne                       |
| "Nom de la campagne" ?                                           |
|                                                                  |
+------------------------------------------------------------------+
| [Annuler]                                    [Valider]           |
+------------------------------------------------------------------+

Etat suppression :
+------------------------------------------------------------------+
|                        [Spinner]                                 |
|               Suppression de la campagne...                      |
+------------------------------------------------------------------+

Etat succes :
+------------------------------------------------------------------+
|                     [Check anime]                                |
|                   Campagne supprimee                             |
+------------------------------------------------------------------+

4. ETATS
--------
1. Confirmation : Affiche message de confirmation
2. Suppression : oui-spinner + "Suppression de la campagne..."
3. Succes : tuc-success-drawing-check + "Campagne supprimee"

5. ENDPOINTS API
----------------
| Methode | Endpoint | Description |
|---------|----------|-------------|
| DELETE | /telephony/{billingAccount}/fax/{serviceName}/campaigns/{id} | Supprimer campagne |

6. COMPOSANTS UTILISES
----------------------
- oui-spinner : Indicateur suppression
- tuc-success-drawing-check : Animation succes

7. TRADUCTIONS FR
-----------------
telephony_service_fax_campaigns_remove: "Supprimer une campagne"
telephony_service_fax_campaigns_remove_campaign_confirm: "Voulez-vous vraiment supprimer la campagne {{campaign}} ?"
telephony_service_fax_campaigns_removing: "Suppression de la campagne..."
telephony_service_fax_campaigns_removed: "Campagne supprimee"

8. ACTIONS UTILISATEUR
----------------------
- Annuler : Ferme modal sans action (disabled pendant suppression)
- Valider : Lance suppression (disabled si en cours ou termine)

FIN DU CAHIER DES CHARGES
