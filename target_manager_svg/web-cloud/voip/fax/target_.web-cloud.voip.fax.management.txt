## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche fax
   - Liste des fax

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - BLOC INFORMATIONS:
     * Numéro
     * Offre
     * Statut (badge)
     * Date activation
     * Groupe facturation

   - BLOC DESCRIPTION:
     * Description éditable

   - BLOC STATISTIQUES (3 cards):
     * Fax envoyés
     * Fax reçus
     * Échecs

   - BLOC ACTIONS RAPIDES:
     * Changer mot de passe
     * Voir historique
     * Convertir en ligne
     * Résilier (rouge)

## API ENDPOINTS

### Info fax
GET /telephony/{billingAccount}/fax/{serviceName}
Response: {
  number, offer, status, activationDate, billingAccount, description
}

### Stats mois
GET /telephony/{billingAccount}/fax/{serviceName}/stats
Query: { period: 'month' }
Response: { sentCount, receivedCount, failedCount }

### Modifier description
PUT /telephony/{billingAccount}/fax/{serviceName}
Body: { description }
