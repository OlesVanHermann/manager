## DESCRIPTION

Gestion des demandes d'assistance pour un service fax. Création de
tickets et suivi de l'historique des interventions.

### Liste tickets
GET /support/tickets?serviceName={serviceName}&category=fax

### Créer ticket
POST /support/tickets
Body: { serviceName, category: 'fax', subject, body }
