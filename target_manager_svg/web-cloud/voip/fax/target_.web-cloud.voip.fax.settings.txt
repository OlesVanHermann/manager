# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des fax
4. RIGHT PANEL
   - Header: Numéro + breadcrumb /Paramètres
   - NAV3: Général | [Paramètres] | Campagnes | Consommation | Filtrage | Logo | Notifications
   - Section "Paramètres d'envoi": Qualité, Tentatives, En-tête, Délai
   - Section "Paramètres de réception": Format PJ, Taille max, Email
   - Section "Sécurité": MDP, Domaines autorisés
   - Boutons: Enregistrer / Annuler

# OPTIONS QUALITÉ
| Valeur    | Description                    |
|-----------|--------------------------------|
| Normale   | Résolution standard (200x100)  |
| Haute     | Résolution fine (200x200)      |
| Très haute| Résolution super-fine (200x400)|

# OPTIONS FORMAT
| Valeur | Description            |
|--------|------------------------|
| PDF    | Document PDF           |
| TIFF   | Image TIFF             |
| Both   | PDF + TIFF             |

# INTERACTIONS
- Dropdown qualité → Sélectionner qualité
- Dropdown tentatives → Nombre 1-5
- Input en-tête → Texte libre
- Dropdown format → PDF/TIFF/Both
- Input email → Email valide
- Clic "Modifier" MDP → Modal changement MDP
- Clic "Gérer →" domaines → Modal gestion domaines
- Clic "Enregistrer" → PUT API
- Clic "Annuler" → Reset formulaire

# API ENDPOINTS
GET /telephony/{ba}/fax/{sn}/settings              → Lire paramètres
PUT /telephony/{ba}/fax/{sn}/settings              → Modifier paramètres
POST /telephony/{ba}/fax/{sn}/settings/changePassword → Changer MDP
