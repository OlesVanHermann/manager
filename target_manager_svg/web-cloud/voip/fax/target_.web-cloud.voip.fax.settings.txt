STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif avec indicateur bleu)
3. LEFT PANEL - Liste des fax (280px)
   - Recherche
   - Compteur (3 services fax)
   - Items services avec badge statut
4. RIGHT PANEL (1096px)
   - Header: Numero fax + description
   - NAV3: General | Envoi | Campagnes | Recus | Envoyes | [Parametres] | Notifications
   - Form Card: Configuration fax
   - Help Card: Variables en-tete

COMPOSANTS
- Formulaire avec champs label + input
- Select dropdown (qualite, nb essais)
- Bouton Reinitialiser mot de passe
- Card aide variables en-tete

FORM FIELDS
| Champ              | Type     | Description                    |
|--------------------|----------|--------------------------------|
| Expediteur         | text     | Nom entreprise                 |
| Email expediteur   | email    | Email pour reponses            |
| Qualite d'envoi    | select   | Normal/Haute/Optimale          |
| Nombre d'essais    | select   | 1 a 5                          |
| Mot de passe       | button   | Reinitialiser                  |
| En-tete fax        | text     | Variables %%n, %%p, %%t, etc.  |

VARIABLES EN-TETE
| Variable | Description            |
|----------|------------------------|
| |        | Separateur colonne     |
| %%i      | Identifiant tache      |
| %%j      | Nom piece jointe       |
| %%l      | ID emission            |
| %%m      | Email expediteur       |
| %%n      | Numero de fax          |
| %%p      | Page session           |
| %%t      | Total pages session    |

INTERACTIONS
- Clic fax left panel → Charge details
- Clic tab NAV3 → Change vue
- Clic Reinitialiser → Modal confirmation
- Clic Enregistrer → PUT API
- Clic Annuler → Restaure valeurs

API ENDPOINTS
GET  /telephony/{ba}/fax/{sn}              # Details fax
PUT  /telephony/{ba}/fax/{sn}              # Modifier parametres
POST /telephony/{ba}/fax/{sn}/changePassword # Reinitialiser MDP
