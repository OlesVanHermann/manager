# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des fax
4. RIGHT PANEL
   - Header: Numéro + badge statut + description
   - NAV3: [Général] | Paramètres | Campagnes | Consommation | Filtrage | Logo | Notifications
   - Card "Informations": Numéro, Offre, Description, État
   - Card "Paramètres actuels": Qualité, Tentatives, Format, Notifications
   - Card "Consommation du mois": Fax envoyés/reçus
   - Card "Actions rapides": Liens vers fonctionnalités

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Modifier →" → Modal édition description
- Clic "Configurer →" → Page paramètres
- Clic "Voir le détail →" → Page consommation
- Clic action rapide → Navigation page correspondante

# API ENDPOINTS
GET /telephony/{ba}/fax/{sn}                    → Détails fax
PUT /telephony/{ba}/fax/{sn}                    → Modifier description
GET /telephony/{ba}/fax/{sn}/serviceInfos       → Infos abonnement
GET /telephony/{ba}/fax/{sn}/settings           → Paramètres
GET /telephony/{ba}/fax/{sn}/consumption        → Consommation
