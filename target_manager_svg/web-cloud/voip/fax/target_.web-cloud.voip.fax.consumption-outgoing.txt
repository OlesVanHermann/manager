## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche fax
   - Liste des fax

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - TABS: Fax envoyés / Fax reçus

   - TOOLBAR:
     * Refresh, recherche
     * Filtre période (dropdown)
     * Filtre statut (dropdown)
     * Bouton envoyer fax

   - STATS SUMMARY (4 cards):
     * Total envoyés
     * Pages envoyées
     * Réussis (vert)
     * Échecs (rouge)

   - TABLE FAX ENVOYES:
     * Date
     * Destinataire
     * Objet
     * Pages
     * Durée
     * Statut (badge)
     * Actions (télécharger/renvoyer)

   - PAGINATION

## API ENDPOINTS

### Liste fax envoyés
GET /telephony/{billingAccount}/fax/{serviceName}/sent
Query: { period, status, page, limit }
Response: { items: [...], totalCount, stats }

### Télécharger fax
GET /telephony/{billingAccount}/fax/{serviceName}/sent/{faxId}/download
Response: PDF file

### Renvoyer fax échoué
POST /telephony/{billingAccount}/fax/{serviceName}/sent/{faxId}/resend

### Stats période
GET /telephony/{billingAccount}/fax/{serviceName}/sent/stats
Query: { period }
Response: { totalCount, totalPages, successCount, failedCount }

================================================================================

## REGLES METIER

- Périodes: 7j, 30j, 90j, personnalisée
- Statuts: Tous, Réussis, Échecs, En cours
- Renvoi disponible uniquement si échec
- PDF téléchargeable 90 jours
- Échec = ligne occupée, pas de réponse, etc.
