## 1. RÉSUMÉ EXÉCUTIF

### Périmètre
Modal de confirmation de suppression d'une notification email.

### Objectif fonctionnel
Demander confirmation avant de supprimer une notification configurée.

### Note
Le code source du modal n'est pas fourni. Spécification basée sur les traductions.

---

## 2. ARCHITECTURE

### Déclencheur
Action "Supprimer cette notification" (freefax_notification_delete) dans le menu contextuel.

---

## 3. PAGES & NAVIGATION

### Type
Modal de confirmation (Confirm Dialog)

---

## 4. COMPOSANTS UI

### Message de confirmation
"Voulez-vous vraiment supprimer la notification <strong>{{email}}</strong> ?"

### Boutons
| Bouton | Label | Variante | Action |
|--------|-------|----------|--------|
| Confirmer | freefax_ok | Primary/Danger | Suppression |
| Annuler | freefax_cancel | Secondary | Fermer modal |

---

## 5. APIs & DONNÉES

### Endpoint
| Méthode | Endpoint | Description |
|---------|----------|-------------|
| DELETE | /freefax/{serviceName}/notifications/{notificationId} | Suppression |
