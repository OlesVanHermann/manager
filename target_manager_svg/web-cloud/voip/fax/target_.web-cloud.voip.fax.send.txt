| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Numéro(s) | Textarea | Oui | Numéros fax destinataires |
| Document | File upload | Oui | PDF/DOC/TXT max 10Mo |
| Qualité | Dropdown | Non | Normale/Haute/Meilleure |
| Tentatives | Dropdown | Non | 1-5 (défaut: 3) |
| Différé | DateTime | Non | Envoi programmé |
