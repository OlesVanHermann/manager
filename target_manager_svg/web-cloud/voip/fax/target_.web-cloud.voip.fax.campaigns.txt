STRUCTURE PAGE
1. SOUS-NAV: Lignes | Numeros | Fax (actif) | Abreges
2. Left panel: Liste fax du groupe
3. NAV3: General | Parametres | Notifications | Logo | Fax envoyes | Fax recus | Consommation | Campagnes (actif)
4. Info banner: Explication campagnes
5. Filters: Recherche + statut
6. Table campagnes avec progression

STATUTS CAMPAGNE
| Statut     | Couleur   | Description              |
|------------|-----------|--------------------------|
| Programmee | #DBEAFE   | En attente execution     |
| En cours   | #FEF3C7   | Envoi en progression     |
| Terminee   | #DEF7EC   | Tous envois termines     |
| Partiel    | #FEE2E2   | Termine avec echecs      |
| Annulee    | #F3F4F6   | Campagne annulee         |

COLONNES TABLE
| Colonne       | Description                    |
|---------------|--------------------------------|
| Nom           | Nom campagne                   |
| Document      | Fichier PDF source             |
| Destinataires | Nombre contacts                |
| Programme     | Date/heure execution           |
| Statut        | Badge etat                     |
| Progression   | Compteur + barre progression   |
| Actions       | Selon statut                   |

ACTIONS PAR STATUT
| Statut     | Actions disponibles                |
|------------|-------------------------------------|
| Programmee | Modifier, Supprimer                 |
| En cours   | Arreter                             |
| Terminee   | Rapport                             |
| Partiel    | Rapport, Relancer echecs            |
| Annulee    | Supprimer                           |

CREATION CAMPAGNE (modal)
| Champ            | Type         | Requis |
|------------------|--------------|--------|
| Nom              | text         | Oui    |
| Document PDF     | file upload  | Oui    |
| Destinataires    | CSV/liste    | Oui    |
| Date envoi       | datetime     | Oui    |
| Priorite         | select       | Non    |

FORMAT CSV DESTINATAIRES
```
numero;nom

+33142567890;Client B
```

API ENDPOINTS
GET    /telephony/{ba}/fax/{sn}/campaigns                   -> Liste campagnes
POST   /telephony/{ba}/fax/{sn}/campaigns                   -> Creer campagne
GET    /telephony/{ba}/fax/{sn}/campaigns/{id}              -> Detail
PUT    /telephony/{ba}/fax/{sn}/campaigns/{id}              -> Modifier
DELETE /telephony/{ba}/fax/{sn}/campaigns/{id}              -> Supprimer
POST   /telephony/{ba}/fax/{sn}/campaigns/{id}/stop         -> Arreter
POST   /telephony/{ba}/fax/{sn}/campaigns/{id}/retry        -> Relancer echecs
GET    /telephony/{ba}/fax/{sn}/campaigns/{id}/report       -> Telecharger rapport
