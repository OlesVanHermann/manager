## API ENDPOINTS

### Consommation fax
GET /telephony/{billingAccount}/fax/{serviceName}/consumption
Query: { dateFrom, dateTo }
Response: {
  faxSent, pagesSent, faxReceived, pagesReceived,
  creditsUsed, creditsRemaining, cost
}

### Historique mensuel
GET /telephony/{billingAccount}/fax/{serviceName}/consumption/history
Query: { months: 6 }
Response: [{
  month, faxSent, pagesSent, faxReceived, pagesReceived, credits, cost
}]

### Export
GET /telephony/{billingAccount}/fax/{serviceName}/consumption/export
Query: { dateFrom, dateTo, format: 'csv' }
Response: CSV download

## REGLES METIER

- Historique disponible sur 24 mois
- Credits non utilises reportes mois suivant
- Taux reussite = fax aboutis / fax tentes
- Reception gratuite (incluse dans offre)
- Cout = credits * prix unitaire credit
