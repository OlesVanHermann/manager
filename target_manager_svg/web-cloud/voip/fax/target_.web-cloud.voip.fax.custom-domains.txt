## DESCRIPTION

Configuration des domaines personnalisés pour l'envoi de fax par email.
Permet d'utiliser une adresse email professionnelle au lieu de l'adresse
par défaut @fax.ovh.net.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| 0033912345678@fax.ovh.net| GET /telephony/.../fax → defaultEmail                   |
| entreprise.com           | domains[].domain                                        |
| fax@entreprise.com       | domains[].email                                         |
| OK / À configurer        | domains[].dnsStatus                                     |

## API ENDPOINTS

### Info fax
GET /telephony/{billingAccount}/fax/{serviceName}
Response: { defaultEmail }

### Liste domaines
GET /telephony/{billingAccount}/fax/{serviceName}/customDomains
Response: [{ id, domain, email, status, dnsStatus }]

### Ajouter domaine
POST /telephony/{billingAccount}/fax/{serviceName}/customDomains
Body: { domain, email }

### Vérifier DNS
POST /telephony/{billingAccount}/fax/{serviceName}/customDomains/{id}/checkDns

### Supprimer domaine
DELETE /telephony/{billingAccount}/fax/{serviceName}/customDomains/{id}

## STATUTS DNS

| Statut       | Couleur | Description          |
|--------------|---------|----------------------|
| ok           | Vert    | DNS configuré        |
| pending      | Jaune   | En vérification      |
| notConfigured| Rouge   | À configurer         |

## REGLES METIER

- Domaine: validation propriété via DNS
- Propagation: 24-48h
- Max 5 domaines par fax
- Email: format valide
