## API ENDPOINTS

### Stats fax
GET /telephony/{billingAccount}/fax/{serviceName}/stats
Query: { period: '30d' }
Response: { sentCount, receivedCount, pendingCount, failedCount, trends }

### Info service
GET /telephony/{billingAccount}/fax/{serviceName}
Response: { number, status, offer, remainingCredit, renewalDate }

### Derniers fax
GET /telephony/{billingAccount}/fax/{serviceName}/history
Query: { limit: 5 }
Response: [{ date, type, recipient, pages, status }]

### Télécharger fax
GET /telephony/{billingAccount}/fax/{serviceName}/history/{faxId}/download
Response: PDF file

### Renvoyer fax
POST /telephony/{billingAccount}/fax/{serviceName}/history/{faxId}/resend
