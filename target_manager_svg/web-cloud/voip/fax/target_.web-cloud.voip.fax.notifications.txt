INTERACTIONS
- Clic fax left panel → Charge details
- Clic tab NAV3 → Change vue
- Toggle → Active/desactive notification
- Clic + Ajouter email → Modal ajout
- Clic Supprimer → Confirmation suppression
- Clic Enregistrer → PUT API settings

API ENDPOINTS

PUT    /telephony/{ba}/fax/{sn}/settings              # Modifier parametres
GET    /telephony/{ba}/fax/{sn}/notifications         # Destinataires supp.
POST   /telephony/{ba}/fax/{sn}/notifications         # Ajouter destinataire
DELETE /telephony/{ba}/fax/{sn}/notifications/{id}    # Supprimer destinataire
