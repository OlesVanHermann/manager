## DESCRIPTION

Conversion d'un service fax en ligne téléphonique. Opération irréversible
qui supprime les fonctionnalités fax pour activer les fonctions téléphonie.

================================================================================

## DONNEES DYNAMIQUES

|--------------------------|---------------------------------------------------------|
| +33 9 12 34 56 78        | GET /telephony/.../fax → number                         |
| Fax Pro                  | fax.offer                                               |
| 1 234 fax                | fax.totalSentCount                                      |

## API ENDPOINTS

### Info fax
GET /telephony/{billingAccount}/fax/{serviceName}
Response: { number, offer, totalSentCount }

### Offres lignes
GET /order/{billingAccount}/fax/{serviceName}/lineOffers
Response: [{ offerId, name, price }]

### Convertir
POST /telephony/{billingAccount}/fax/{serviceName}/convertToLine
Body: { lineOfferId }
