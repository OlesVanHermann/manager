## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche fax
   - Liste des fax

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - TABS: Fax envoyés / Fax reçus

   - TOOLBAR:
     * Refresh, recherche
     * Filtre période (dropdown)
     * Filtre lecture (dropdown)
     * Bouton télécharger tout (ZIP)

   - STATS SUMMARY (4 cards):
     * Total reçus
     * Pages reçues
     * Non lus (bleu)
     * Lus

   - TABLE FAX RECUS:
     * Indicateur non lu (point bleu)
     * Date
     * Émetteur
     * Pages
     * Taille
     * Qualité (badge)
     * Statut lecture (badge)
     * Actions (voir/télécharger/supprimer)

   - PAGINATION

## API ENDPOINTS

### Liste fax reçus
GET /telephony/{billingAccount}/fax/{serviceName}/received
Query: { period, readStatus, page, limit }
Response: { items: [...], totalCount, stats }

### Marquer comme lu
PUT /telephony/{billingAccount}/fax/{serviceName}/received/{faxId}/read

### Télécharger fax
GET /telephony/{billingAccount}/fax/{serviceName}/received/{faxId}/download
Response: PDF file

### Télécharger tout (ZIP)
GET /telephony/{billingAccount}/fax/{serviceName}/received/download
Query: { period }
Response: ZIP file

### Supprimer fax
DELETE /telephony/{billingAccount}/fax/{serviceName}/received/{faxId}

================================================================================

## REGLES METIER

- Fax non lus en surbrillance bleue
- Qualités: Fine (200 dpi), Standard (100 dpi)
- Conservation 90 jours
- ZIP limité à 100 fax max
- Suppression définitive
