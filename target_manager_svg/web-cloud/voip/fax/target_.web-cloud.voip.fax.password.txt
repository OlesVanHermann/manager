## DESCRIPTION

Modification du mot de passe pour l'envoi de fax par email via SMTP.
Affiche également les informations de connexion SMTP.

================================================================================

|--------------------------|---------------------------------------------------------|
| 0033912345678@fax.ovh.net| GET /telephony/.../fax → email                          |
| smtp.fax.ovh.net:587     | fax.smtpServer                                          |

   - ACTIONS: Modifier, Annuler

================================================================================

## API ENDPOINTS

### Info fax
GET /telephony/{billingAccount}/fax/{serviceName}
Response: { email, smtpServer, smtpPort }

### Modifier mot de passe
POST /telephony/{billingAccount}/fax/{serviceName}/changePassword
Body: { password }

### Générer mot de passe
POST /telephony/{billingAccount}/fax/{serviceName}/generatePassword
Response: { password }
