================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic fax → Charge les détails du fax
- Clic NAV4 → Navigation vers l'onglet
- Clic ↻ → Actualiser la liste
- Clic Envoyer fax → Modal envoi fax
- Clic filtre Statut → Dropdown filtres
- Clic ↓ → Télécharger le document
- Clic ↻ → Renvoyer le fax
- Clic × → Annuler l'envoi (si en cours)

================================================================================

API ENDPOINTS

GET /telephony/{ba}/fax → Liste services fax
GET /telephony/{ba}/fax/{sn}/campaigns → Liste campagnes/envois
GET /telephony/{ba}/fax/{sn}/campaigns?status={status} → Filtrer par statut
POST /telephony/{ba}/fax/{sn}/campaigns → Envoyer un fax
GET /telephony/{ba}/fax/{sn}/campaigns/{id}/download → Télécharger document
POST /telephony/{ba}/fax/{sn}/campaigns/{id}/resend → Renvoyer
DELETE /telephony/{ba}/fax/{sn}/campaigns/{id} → Annuler envoi
