INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic numéro → Charge les détails du numéro
- Clic NAV4 → Navigation vers l'onglet
- Clic ↻ → Actualiser les données
- Clic + Ajouter plage → Modal ajout plage horaire
- Clic ✎ jour → Modal édition horaires du jour
- Clic ✎ exception → Modal édition exception
- Clic × exception → Modal confirmation suppression
- Clic + Ajouter exception → Modal ajout jour férié
