# SECTIONS DE LA PAGE

## 1. Protection anti-fraude
Configurez les seuils de protection :
- Limite appels sortants (€ HT / mois)
- Toggle appels internationaux
- Gestion des pays autorisés

## 2. Adresses IP autorisées
Restriction de l'accès SIP par IP :
- Liste blanche d'IP/CIDR
- Description pour chaque entrée
- Actions: modifier, supprimer

## 3. Tentatives suspectes
Monitoring sécurité en temps réel :
- Tentatives bloquées (IP non autorisée)
- Tentatives échouées (mauvais mot de passe)
- Historique 24h/7j

# TYPES D'ALERTES
| Type | Couleur | Description |
|------|---------|-------------|
| Bloquée (IP) | Rouge #FEE2E2 | IP non dans whitelist |
| Échouée (MDP) | Orange #FEF3C7 | Mauvais mot de passe |
| Suspecte | Orange #FEF3C7 | Pattern anormal détecté |

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Modifier" limite → Modal édition limite
- Toggle international → Activer/désactiver
- Clic "Gérer pays" → Modal liste pays autorisés
- Clic "+ Ajouter IP" → Modal ajout IP
- Clic ✎ sur IP → Modal édition
- Clic ✕ sur IP → Supprimer (confirmation)

# API ENDPOINTS
GET /telephony/{ba}/security → Configuration sécurité
PUT /telephony/{ba}/security → Modifier config
GET /telephony/{ba}/security/allowedIp → Liste IP autorisées
POST /telephony/{ba}/security/allowedIp → Ajouter IP
DELETE /telephony/{ba}/security/allowedIp/{id} → Supprimer IP
GET /telephony/{ba}/security/alerts → Alertes récentes
GET /telephony/{ba}/security/countries → Pays internationaux

# BONNES PRATIQUES SÉCURITÉ
1. Définir une limite de consommation réaliste
2. Restreindre les appels internationaux si non utilisés
3. Configurer une whitelist IP si environnement fixe
4. Surveiller régulièrement les alertes
5. Changer les mots de passe SIP régulièrement
