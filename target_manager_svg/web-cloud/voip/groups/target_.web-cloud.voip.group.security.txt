   - Bannière score sécurité
   - Bloc: Paramètres de sécurité (toggles)
   - Bloc: Liste blanche IP
   - Bloc: Activité récente

# PARAMÈTRES SÉCURITÉ
| Paramètre | Description |
|-----------|-------------|
| Restriction IP | Limiter connexions SIP à IPs autorisées |
| 2FA | Double authentification actions sensibles |
| Politique MDP | Exigences de complexité des mots de passe |

# SCORE SÉCURITÉ
| Niveau | Score | Couleur |
|--------|-------|---------|
| Critique | 0-25% | Rouge |
| Faible | 26-50% | Orange |
| Moyen | 51-75% | Jaune |
| Bon | 76-100% | Vert |

# TABLE IP WHITELIST
| Colonne | Description |
|---------|-------------|
| Adresse IP/CIDR | IP ou plage autorisée |
| Description | Note descriptive |
| Actions | Supprimer |

# TABLE ACTIVITÉ
| Colonne | Description |
|---------|-------------|
| Date/Heure | Timestamp événement |
| Action | Type d'opération |
| Utilisateur | Compte concerné |
| IP source | Origine connexion |
| Résultat | OK / Bloqué / Échec |

# INTERACTIONS
- Toggles → Active/désactive paramètres
- "+ Ajouter une IP" → Modal ajout IP
- "✕ Supprimer" → Retire IP de whitelist
- "Voir tout l'historique" → Page logs complets
