## API ENDPOINTS

### Reversements
GET /telephony/{billingAccount}/repayments
Response: {
  availableBalance, currentMonth: { amount, calls },
  yearTotal, iban, minimumThreshold,
  history: [{
    id, date, period, calls, grossAmount, commission, netAmount, status
  }]
}

### Demander versement
POST /telephony/{billingAccount}/repayments/request
Body: { amount } // optionnel, tout le solde par défaut
Response: { requestId, status }

### Configurer IBAN
PUT /telephony/{billingAccount}/repayments/iban
Body: { iban, bic }

## CALCUL REVERSEMENT

| Élément      | Description                        |
|--------------|------------------------------------|
| Montant brut | Total des appels surtaxés          |
| Commission   | 10% retenu par OVH                 |
| Montant net  | Montant versé au client            |

================================================================================

## REGLES METIER

- Seuil minimum: 100 € pour demander versement
- Versements mensuels automatiques si > 100 €
- Commission OVH: 10%
- IBAN obligatoire pour recevoir les fonds
- Délai versement: 15 jours
- Applicable uniquement aux numéros 08XX surtaxés
