## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/bill
    Response: [{ date, amount, paid }]

GET /telephony/{billingAccount}/bill/{billId}/document
    Response: PDF blob

GET /telephony/{billingAccount}/bill/{billId}/callsCsv?type=outgoing
GET /telephony/{billingAccount}/bill/{billId}/callsCsv?type=incoming
```
