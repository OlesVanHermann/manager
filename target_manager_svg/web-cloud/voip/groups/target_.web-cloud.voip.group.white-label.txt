## API ENDPOINTS

### Config marque blanche
GET /telephony/{billingAccount}/whiteLabel
Response: { logo, primaryColor, secondaryColor, bgColor, customDomain, sslStatus, supportEmail, supportPhone }

### Upload logo
POST /telephony/{billingAccount}/whiteLabel/logo
Body: FormData avec fichier image

### Modifier config
PUT /telephony/{billingAccount}/whiteLabel
Body: { primaryColor?, secondaryColor?, bgColor?, supportEmail?, supportPhone? }

### Configurer domaine
POST /telephony/{billingAccount}/whiteLabel/domain
Body: { domain }

### Vérifier SSL
GET /telephony/{billingAccount}/whiteLabel/ssl
Response: { status, issuedAt, expiresAt }

### Réinitialiser
DELETE /telephony/{billingAccount}/whiteLabel

## REGLES METIER

- Option payante (activation via commande)
- Logo: PNG/SVG, max 500KB, 200x60px recommandé
- Domaine: CNAME vers ovh requis
- SSL: Let's Encrypt automatique
- Délai propagation domaine: 24-48h
