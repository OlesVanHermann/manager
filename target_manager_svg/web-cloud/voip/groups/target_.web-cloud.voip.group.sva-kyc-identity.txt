# Cahier des Charges : telecom/svaWallet - KYC Identite entreprise

## 1. RESUME EXECUTIF

### Perimetre
Formulaire de verification d'identite (KYC) pour portefeuille SVA.

### Objectif fonctionnel
- Collecter les informations legales de l'entreprise
- Renseigner l'adresse du siege social
- Identifier le representant legal
- Valider les donnees avant soumission

## 4. APIS & DONNEES

### Formes juridiques
| Code | Label |
|------|-------|
| SARL | Societe a responsabilite limitee |
| SAS | Societe par actions simplifiee |
| SA | Societe anonyme |
| EURL | Entreprise unipersonnelle |
| EI | Entreprise individuelle |
| SCI | Societe civile immobiliere |
| SASU | SAS unipersonnelle |

### Endpoints
```
GET /svaWallet/{walletId}/kyc
    Response: { status, identity, address, representative }

PUT /svaWallet/{walletId}/kyc/identity
    Body: {
      businessType: "professional",
      legalName: "MA SOCIETE SAS",
      siret: "12345678901234",
      registrationDate: "2020-01-15",
      legalForm: "SAS",
      activityDescription: "Services telecom",
      isCrowdfunding: false,
      isMarketplace: false
    }

PUT /svaWallet/{walletId}/kyc/address
    Body: {
      streetNumber: "10",
      streetType: "rue",
      streetName: "de la Paix",
      postalCode: "75001",
      city: "Paris",
      country: "FR"
    }

PUT /svaWallet/{walletId}/kyc/representative
    Body: {
      lastName: "DUPONT",
      firstName: "Jean",
      birthDate: "1975-03-15",
      email: "jean.dupont@masociete.fr"
    }
```

### Validation SIRET
| Regle | Description |
|-------|-------------|
| Format | 14 chiffres |
| Algorithme | Luhn |
| Verification | API INSEE |

## 6. REGLES METIER

- SIRET verifie via API INSEE
- Adresse doit correspondre au Kbis
- Representant legal = signataire
- Certification obligatoire pour valider
- Donnees conformes a la reglementation LCB-FT
