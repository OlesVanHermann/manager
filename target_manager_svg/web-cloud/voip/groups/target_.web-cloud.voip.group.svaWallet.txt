# CAHIER DES CHARGES - old_manager.telecom.telephony.svaWallet
# Module portefeuille SVA (Services a Valeur Ajoutee)

## 1. RESUME EXECUTIF

### Perimetre
Gestion du portefeuille SVA pour les numeros surtaxes avec reversements.

### Objectif fonctionnel
Permettre l'onboarding KYC, la gestion des documents, et le suivi des reversements.

## 7. REGLES METIER

### Processus KYC
1. Creation profil (particulier/entreprise)
2. Upload documents obligatoires
3. Verification manuelle (1-3 jours)
4. Activation compte

### Reversements
- Seuil minimum: 10 EUR
- Frequence: mensuelle
- Virement vers IBAN valide
