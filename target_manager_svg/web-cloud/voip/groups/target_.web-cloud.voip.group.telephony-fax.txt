# CAHIER DES CHARGES - old_manager.telecom.telephony.fax
# Module principal Fax (Ecofax, Plug&Fax)

## 1. RESUME EXECUTIF

### Perimetre
Module de gestion des services fax: Ecofax, Plug&Fax, Voicefax.

### Objectif fonctionnel
Configurer et utiliser les services de fax virtuel OVHcloud.
