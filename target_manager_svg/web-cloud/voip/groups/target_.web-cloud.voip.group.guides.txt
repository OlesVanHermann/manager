## API ENDPOINTS

### Liste catégories
GET /telephony/guides/categories
Response: [{ id, name, icon, color, guidesCount }]

### Guides par catégorie
GET /telephony/guides/categories/{categoryId}
Response: [{ id, title, url, type, views }]

### Recherche guides
GET /telephony/guides/search
Query: { q }
Response: [{ id, title, excerpt, categoryId, url }]

### FAQ
GET /telephony/guides/faq
Response: [{ question, answer, categoryId }]
