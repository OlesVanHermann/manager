## DESCRIPTION

Page de facturation du groupe VoIP avec vue sur le crédit, la consommation
mensuelle et le détail par type d'appel.

================================================================================

## STRUCTURE PAGE

### NAV4 Tabs
- Dashboard | Services | Répertoire | [Facturation] | Administration

### NAV5 Sous-tabs
- [Consommation] | Factures | Crédit

### Contenu (tab Consommation)
- Card crédit disponible
- Card consommation du mois
- Card évolution mensuelle
- Table détail consommation
- Actions export

================================================================================

## DONNÉES DYNAMIQUES

| Élément SVG          | Source API                                |
|----------------------|-------------------------------------------|
| Crédit disponible    | GET /telephony/{ba} → currentOutplan      |
| Seuil alerte         | GET /telephony/{ba} → creditThreshold     |
| Conso mois           | GET /telephony/{ba}/historyConsumption    |
| Détail par type      | GET /telephony/{ba}/historyConsumption    |
| Évolution            | Calcul depuis historyConsumption          |

================================================================================

## CARDS HEADER

### Card Crédit (fond bleu info)
- Montant crédit disponible
- Seuil d'alerte paramétré
- Bouton "+ Recharger"

### Card Consommation mois
- Montant consommé
- Budget mensuel
- Barre progression %

### Card Évolution
- % variation vs mois précédent
- Mini graphique tendance

================================================================================

## TABLE DÉTAIL CONSOMMATION

| Colonne      | Description                    |
|--------------|--------------------------------|
| Type         | Nationaux fixe, Mobiles, International |
| Appels       | Nombre total appels            |
| Durée totale | HH:mm format                   |
| Coût HT      | Montant hors taxes             |
| Coût TTC     | Montant TTC                    |
| % du total   | Barre proportion + %           |

### Ligne Total
- Somme de toutes les colonnes
- Fond gris
