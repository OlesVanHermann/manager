# QU'EST-CE QUE LA PORTABILITÉ ?
La portabilité des numéros permet de :
- Conserver ses numéros lors d'un changement d'opérateur
- Importer des numéros d'autres opérateurs vers OVH (entrante)
- Exporter des numéros OVH vers d'autres opérateurs (sortante)

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic ↻ → Rafraîchir liste
- Clic filtre Direction → Entrante/Sortante/Tous
- Clic filtre Statut → Filtrer par statut
- Clic "+ Demander portabilité" → Assistant portabilité
- Clic "Détails" → Page détail portabilité
- Clic "Annuler" → Annuler portabilité (si possible)

# API ENDPOINTS
GET /telephony/{ba}/portability → Liste portabilités
GET /telephony/{ba}/portability/{id} → Détail portabilité
POST /telephony/{ba}/portability → Nouvelle demande
DELETE /telephony/{ba}/portability/{id} → Annuler portabilité
GET /telephony/{ba}/portability/{id}/document → Documents requis

# CODE RIO
Le RIO (Relevé d'Identité Opérateur) est un code à 12 caractères
nécessaire pour toute portabilité. Il s'obtient en appelant le 3179
depuis la ligne concernée.
