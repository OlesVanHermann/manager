# Cahier des Charges : telecom/telephony - Options du groupe

## 1. RESUME EXECUTIF

### Perimetre
Configuration des options globales du groupe VoIP.

### Objectif fonctionnel
- Configurer les options de presentation
- Definir les parametres utilisateur
- Appliquer a toutes les lignes du groupe

| Option | Description |
|--------|-------------|

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}
    Response: { hideCallerNumber, forceCallerNumber, showCallerDescription }

PUT /telephony/{billingAccount}
    Body: { hideCallerNumber, forceCallerNumber, showCallerDescription }
```
