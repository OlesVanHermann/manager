# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des groupes de facturation
4. RIGHT PANEL
   - Header: Identifiant groupe + badge statut + description
   - NAV3: [Général] | Services | Consommation | Factures | Sécurité
   - Card "Informations groupe": ID, Description, Date création, Statut
   - Card "Résumé services": Compteurs par type de service
   - Card "Facturation": Cycle, Prochaine facture, Montant estimé
   - Card "Consommation mois": Appels, SMS, Fax
   - Card "Actions rapides": Commander, Portabilité, Sécurité, Export

# COMPTEURS SERVICES
| Type | Endpoint |
|------|----------|
| Lignes SIP | GET /telephony/{ba}/line |
| Numéros | GET /telephony/{ba}/number |
| Fax | GET /telephony/{ba}/fax |
| SMS | /sms (filtré par billingAccount) |
| Trunk SIP | GET /telephony/{ba}/trunk |

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Modifier" → Modal édition description
- Clic "Voir tous les services" → Page Services
- Clic "Voir les factures" → Page Factures
- Clic "Voir le détail" → Page Consommation
- Clic action rapide → Navigation/Modal correspondant

# API ENDPOINTS
GET /telephony/{ba}                        → Détails groupe
PUT /telephony/{ba}                        → Modifier description
GET /telephony/{ba}/serviceInfos           → Infos abonnement
GET /telephony/{ba}/line                   → Liste lignes
GET /telephony/{ba}/number                 → Liste numéros
GET /telephony/{ba}/fax                    → Liste fax
GET /telephony/{ba}/trunk                  → Liste trunks
GET /telephony/{ba}/billing/estimate       → Estimation facture
GET /telephony/{ba}/statistics             → Statistiques consommation
