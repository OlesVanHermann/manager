## DESCRIPTION

Gestion des numéros abrégés du groupe de facturation. Permet de créer
des raccourcis de numérotation interne (2-4 chiffres).

================================================================================

## API ENDPOINTS

### Liste numéros abrégés
GET /telephony/{billingAccount}/abbreviatedNumber
Response: [abbreviatedNumber, ...]

### Détail numéro abrégé
GET /telephony/{billingAccount}/abbreviatedNumber/{abbreviatedNumber}
Response: { abbreviatedNumber, destinationNumber, name, destinationType }

### Créer numéro abrégé
POST /telephony/{billingAccount}/abbreviatedNumber
Body: { abbreviatedNumber, destinationNumber, name }

### Modifier numéro abrégé
PUT /telephony/{billingAccount}/abbreviatedNumber/{abbreviatedNumber}
Body: { destinationNumber?, name? }

### Supprimer numéro abrégé
DELETE /telephony/{billingAccount}/abbreviatedNumber/{abbreviatedNumber}

### Stats disponibilité
GET /telephony/{billingAccount}/abbreviatedNumber/availability
Response: { range2digits, range3digits, range4digits }

================================================================================

## REGLES METIER

- Formats: 2 chiffres (10-99), 3 chiffres (100-999), 4 chiffres (1000-9999)
- Maximum 1000 numéros abrégés par groupe
- Destination: ligne du groupe, numéro alias, ou numéro externe
- Numéro abrégé unique dans le groupe
- Appel depuis poste du groupe uniquement
