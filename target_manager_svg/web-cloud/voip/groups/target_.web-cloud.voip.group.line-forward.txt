# Cahier des Charges : telecom/telephony - Renvois d'appels

## 1. RESUME EXECUTIF

### Perimetre
Page de configuration des renvois d'appels d'une ligne.

### Objectif fonctionnel
- Configurer le renvoi inconditionnel
- Configurer le renvoi sur non-reponse
- Configurer le renvoi sur occupation

|---------|-------------|
| Breadcrumb | Telephonie / grp-X / Ligne Y / Renvois |

| Onglet | Etat |

### Carte renvoi inconditionnel
| Element | Description |
|---------|-------------|
| Titre | Renvoi inconditionnel |

| Element | Description |
|---------|-------------|

| Input numero | Numero destination |
| Bouton | Modifier |

### Carte renvoi occupation
| Element | Description |

| Toggle | On/Off |

| Bouton | Modifier |

### Toggle switch
| Etat | Style |
|------|-------|
| Off | Fond gris, cercle gauche |

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/options
PUT /telephony/{billingAccount}/line/{serviceName}/options
```

### Modele Options
```typescript
interface LineForwardOptions {
  forwardUnconditional: {
    enabled: boolean;
    number: string;
  };
  forwardNoReply: {
    enabled: boolean;
    number: string;
    delay: number;
  };
  forwardBusy: {
    enabled: boolean;
    number: string;
  };
}
```

## 6. REGLES METIER

- Format E.164 pour numeros
- Delai non-reponse: 5-60 secondes
- Renvoi inconditionnel prioritaire
- Application immediate sans confirmation
