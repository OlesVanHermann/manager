# Cahier des Charges : telecom/telephony - Recapitulatif

## 1. RESUME EXECUTIF

### Perimetre
Recapitulatif de consommation telephonique du groupe.

### Objectif fonctionnel
- Vue globale des appels
- Detail par service
- Statistiques entrants/sortants

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/historyConsumption
    Response: {
      summary: { total, incoming, outgoing, transfer },
      byService: [{
        serviceName, description, number,
        total, incoming, outgoing, transfer
      }]
    }
```
