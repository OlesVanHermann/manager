## DESCRIPTION

Vue récapitulative de tous les services du groupe VoIP : lignes, numéros et fax.
Permet de voir rapidement l'ensemble du parc téléphonique.

================================================================================

## STRUCTURE PAGE

### NAV4 Tabs
- Dashboard | [Services] | Répertoire | Facturation | Administration

### Contenu
- Toolbar: Refresh + Commander
- Section Lignes téléphoniques (table)
- Section Numéros (table)
- Section Fax (table)

================================================================================

## DONNÉES DYNAMIQUES

| Élément SVG      | Source API                         |
|------------------|------------------------------------|
| Lignes           | GET /telephony/{ba}/line           |
| Détail ligne     | GET /telephony/{ba}/line/{sn}      |
| Numéros          | GET /telephony/{ba}/number         |
| Détail numéro    | GET /telephony/{ba}/number/{sn}    |
| Fax              | GET /telephony/{ba}/fax            |
| Détail fax       | GET /telephony/{ba}/fax/{sn}       |

================================================================================

## TABLE LIGNES

| Colonne     | Source                                    |
|-------------|-------------------------------------------|
| Numéro      | serviceName                               |
| Description | description                               |
| Offre       | offerType (VoIP Pro, VoIP Entreprise...)  |
| Téléphone   | phone.brand + phone.model                 |
| Statut      | serviceInfos.status                       |
| Actions     | Lien vers détail ligne                    |

## TABLE NUMÉROS

| Colonne     | Source                              |
|-------------|-------------------------------------|
| Numéro      | serviceName                         |
| Description | description                         |
| Type        | featureType (SVI, File, Redirect)   |
| Destination | Configuration selon type            |
| Statut      | status                              |

## TABLE FAX

| Colonne           | Source                        |
|-------------------|-------------------------------|
| Numéro            | serviceName                   |
| Description       | description                   |
| Email notification| notifications.email           |
| Statut            | status                        |
