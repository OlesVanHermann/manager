1. NAV1 - Header OVH (Web Cloud actif)

3. LEFT PANEL - Liste groupes avec sélection
4. RIGHT PANEL
   - Header: Nom groupe + breadcrumb /Services
   - NAV3: Dashboard | [Services] | Répertoire | Facturation | Administration
   - Filtres: Tous | Lignes (3) | Numéros (2) | Fax (1)
   - Recherche + Bouton Ajouter
   - Table services:
     * Service (numéro)
     * Description
     * Type (badge Ligne/Numéro/Fax)
     * Configuration (Ligne SIP, File d'attente, SVI, etc.)
     * Statut (badge Actif/Inactif)
     * Actions (lien Détail)

# INTERACTIONS

- Clic "Détail" → Navigation vers page service
- Clic "Ajouter" → Modal ajout service
- Recherche → Filtrage live

# API ENDPOINTS
GET /telephony/{ba}/line           → Liste lignes
GET /telephony/{ba}/number         → Liste numéros
GET /telephony/{ba}/fax            → Liste fax
GET /telephony/{ba}/line/{sn}      → Détails ligne
GET /telephony/{ba}/number/{sn}    → Détails numéro
GET /telephony/{ba}/fax/{sn}       → Détails fax
