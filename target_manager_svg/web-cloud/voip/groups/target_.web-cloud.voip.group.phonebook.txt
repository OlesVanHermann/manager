## DESCRIPTION

Répertoire partagé du groupe VoIP. Gestion des contacts accessibles depuis
tous les téléphones du groupe.

================================================================================

## STRUCTURE PAGE

### NAV4 Tabs
- Dashboard | Services | [Répertoire] | Facturation | Administration

### Contenu
- Toolbar: Refresh, Recherche, Import, Export, Ajouter
- Compteur contacts
- Table contacts avec checkbox
- Pagination

## TABLE CONTACTS

| Colonne    | Source                  | Type     |
|------------|-------------------------|----------|
| Checkbox   | Sélection multiple      | UI       |
| Nom        | contact.name            | text     |
| Prénom     | contact.surname         | text     |
| Société    | contact.group           | text     |
| Téléphone  | contact.phoneNumber     | text     |
| Mobile     | contact.phoneMobile     | text     |
| Actions    | ✎ Edit, ✕ Delete        | buttons  |

## ACTIONS

| Action     | Déclencheur    | API                                    |
|------------|----------------|----------------------------------------|
| Refresh    | Bouton ↻       | Recharger liste                        |
| Rechercher | Input search   | Filtrage client-side                   |
| Importer   | Bouton Import  | Modal + POST /telephony/{ba}/phonebook |
| Exporter   | Bouton Export  | GET /telephony/{ba}/phonebook/export   |
| Ajouter    | Bouton +       | Modal + POST /telephony/{ba}/phonebook |
| Modifier   | Icône ✎        | Modal + PUT /telephony/{ba}/phonebook/{id} |
| Supprimer  | Icône ✕        | Confirm + DELETE /telephony/{ba}/phonebook/{id} |

## IMPORT/EXPORT

### Format CSV
```
nom;prenom;societe;telephone;mobile;email
Dupont;Jean;ACME Corp;0123456789;0612345678;jean@acme.com
```

### Import
- Upload fichier CSV
- Mapping colonnes automatique
- Preview avant validation

### Export
- Téléchargement CSV complet
- Encodage UTF-8
