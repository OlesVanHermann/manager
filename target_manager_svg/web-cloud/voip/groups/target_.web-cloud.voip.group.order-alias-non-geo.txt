# Cahier des Charges : telecom/telephony - Commander numero non geographique

## 1. RESUME EXECUTIF

### Perimetre
Formulaire de commande d'un numero non geographique (09).

### Objectif fonctionnel
- Commander un ou plusieurs numeros 09
- Choisir entre numero standard ou personnalise
- Renseigner les informations du titulaire
- Valider et passer commande

## 3. COMPOSANTS UI

### Lien retour
| Element | Description |
|---------|-------------|
| Link | <- Retour aux commandes |

### Titre
| Element | Description |
|---------|-------------|
| Titre | Commander un numero non geographique |

### Carte Quantite
| Champ | Type | Options |
|-------|------|---------|
| Nombre de numeros | Select | 1 numero, 2 numeros, ..., 10 numeros |
| Type d'offre | Radio | Numero alias (redirection), SDA uniquement |

### Carte Choix du numero
| Option | Description |
|--------|-------------|
| Numero standard | 09 XX XX XX XX (attribue automatiquement) |
| Numero personnalise | Choix du suffixe |

| Element | Description |
|---------|-------------|
| Numero attribue | +33 9 72 XX XX XX (preview) |

### Carte Adresse du titulaire
| Champ | Type | Description |
|-------|------|-------------|
| Nom / Raison sociale | Input | Nom ou entreprise |
| Adresse | Input | Adresse complete avec ville et CP |

### Carte Validation
| Element | Description |
|---------|-------------|
| Checkbox | Je renonce au droit de retractation et j'accepte les conditions generales |
| Bouton | Commander (0,00 EUR HT/mois) |

## 4. APIS & DONNEES

### Caracteristiques 09
| Propriete | Valeur |
|-----------|--------|
| Prefixe | 09 |
| Type | Non geographique |
| Cout appel | Inclus dans forfaits |
| Zone | France entiere |

### Endpoints
```
GET /telephony/{billingAccount}/number/nongeo/prices
    Response: { standard, premium }

POST /telephony/{billingAccount}/number/nongeo/order
    Body: {
      quantity: 1,
      type: "alias",
      address: { name, fullAddress },
      acceptRetractation: true
    }
```

- Tarification identique partout
- Ideal pour entreprises multi-sites
- CGV et retractation obligatoires
