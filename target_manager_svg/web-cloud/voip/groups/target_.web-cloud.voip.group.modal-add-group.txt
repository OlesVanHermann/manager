# Cahier des Charges : telecom/telephony - Ajouter un groupe

## 1. RESUME EXECUTIF

### Perimetre
Formulaire d'ajout d'un nouveau groupe de telephonie.

### Objectif fonctionnel
- Accepter les conditions contractuelles
- Creer un nouveau groupe de facturation
- Commander le service (gratuit)

## 4. APIS & DONNEES

### Endpoints
```
POST /order/telephony/new
    Body: { contracts: [accepted] }
    Response: { orderId, url }
```
