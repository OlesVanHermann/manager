# Cahier des Charges : telecom/svaWallet - KYC Beneficiaires effectifs

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion des beneficiaires effectifs pour portefeuille SVA.

### Objectif fonctionnel
- Declarer les beneficiaires effectifs (>25% capital)
- Identifier le representant legal
- Gerer les beneficiaires (ajout, modification, suppression)

## 4. APIS & DONNEES

### Endpoints
```
GET /svaWallet/{walletId}/kyc/beneficiaries
    Response: [
      {
        id: "ben_123",
        lastName: "DUPONT",
        firstName: "Jean",
        birthDate: "1975-03-15",
        nationality: "FR",
        isLegalRepresentative: false
      },
      ...
    ]

POST /svaWallet/{walletId}/kyc/beneficiaries
    Body: {
      lastName: "MARTIN",
      firstName: "Marie",
      birthDate: "1980-08-22",
      nationality: "FR"
    }

PUT /svaWallet/{walletId}/kyc/beneficiaries/{beneficiaryId}
    Body: { lastName, firstName, birthDate, nationality }

DELETE /svaWallet/{walletId}/kyc/beneficiaries/{beneficiaryId}
```

### Contraintes
| Contrainte | Valeur |
|------------|--------|
| Nombre max | 4 beneficiaires |
| Representant legal | Non supprimable |
| Part minimum | 25% du capital |

## 6. REGLES METIER

- Beneficiaire effectif = >25% capital/droits de vote
- Representant legal auto-ajoute (non supprimable)
- Maximum 4 beneficiaires
- Obligation LCB-FT/5AMLD
- Nationalite = code ISO 2 lettres
