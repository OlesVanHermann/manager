## API ENDPOINTS

### Liste commandes
GET /telephony/{billingAccount}/orders
Query: { status }
Response: [{
  orderId, date, type, description, amount, currency, status
}]

### Détail commande
GET /telephony/{billingAccount}/orders/{orderId}
Response: { ... + items[], timeline[] }

### Nouvelle commande
POST /telephony/{billingAccount}/orders
Body: { type, items: [...] }
Response: { orderId, paymentUrl }

### Statistiques commandes
GET /telephony/{billingAccount}/orders/stats
Response: { completed, pending, awaitingPayment }

| Type           | Description                    |
|----------------|--------------------------------|
