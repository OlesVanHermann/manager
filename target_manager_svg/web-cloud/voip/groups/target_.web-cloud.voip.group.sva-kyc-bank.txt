# Cahier des Charges : telecom/svaWallet - KYC Compte bancaire

## 1. RESUME EXECUTIF

### Perimetre
Modal de modification du compte bancaire pour portefeuille SVA.

### Objectif fonctionnel
- Modifier les coordonnees bancaires
- Valider le format IBAN
- Associer le compte a l'entreprise

## 3. COMPOSANTS UI

### Modal Header
| Element | Description |
|---------|-------------|
| Titre | Modifier le compte bancaire |
| Bouton fermer | X en haut a droite |

### Formulaire
| Champ | Type | Description |
|-------|------|-------------|
| Titulaire du compte | Input | Nom identique au Kbis |
| IBAN | Input | Format IBAN international |

### Boutons
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondary | Ferme la modal |
| Confirmer | Primary | Enregistre les modifications |

## 4. APIS & DONNEES

### Validation IBAN
| Pays | Format |
|------|--------|
| France | FR76 XXXX XXXX XXXX XXXX XXXX XXX |
| Belgique | BE68 XXXX XXXX XXXX |
| Allemagne | DE89 XXXX XXXX XXXX XXXX XX |

### Endpoints
```
GET /svaWallet/{walletId}/kyc/bankAccount
    Response: {
      accountHolder: "MA SOCIETE SAS",
      iban: "FR76123456789012345678901xx"
    }

PUT /svaWallet/{walletId}/kyc/bankAccount
    Body: {
      accountHolder: "MA SOCIETE SAS",
      iban: "FR76123456789012345678901xx"
    }
```

### Validation
| Champ | Regle |
|-------|-------|
| accountHolder | Doit correspondre a la raison sociale |
| iban | Format IBAN valide (checksum mod 97) |

## 6. REGLES METIER

- Titulaire = raison sociale exacte
- IBAN verifie par checksum
- Modification necessite nouveau RIB
- Compte utilise pour les reversements SVA
- Uniquement comptes europeens (SEPA)
