## API ENDPOINTS

### Liste contacts
GET /telephony/{billingAccount}/phonebook
Query: { letter?, search?, limit, offset }
Response: [{ id, name, title, phone, mobile, email, company }]

### Créer contact
POST /telephony/{billingAccount}/phonebook
Body: { name, title?, phone?, mobile?, email?, company? }

### Modifier contact
PUT /telephony/{billingAccount}/phonebook/{contactId}
Body: { name?, title?, phone?, mobile?, email?, company? }

### Supprimer contact
DELETE /telephony/{billingAccount}/phonebook/{contactId}

### Import CSV
POST /telephony/{billingAccount}/phonebook/import
Body: FormData avec fichier CSV

### Export
GET /telephony/{billingAccount}/phonebook/export
Response: CSV file

## REGLES METIER

- Maximum 5000 contacts par groupe
- Import CSV: colonnes name, phone, mobile, email, company
- Synchronisation avec téléphones IP compatibles
- Doublons détectés sur nom+téléphone
- Avatar généré à partir des initiales
