# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic ↻ → Rafraîchir liste
- Clic filtre Type → Filtrer par type de commande
- Clic filtre Statut → Filtrer par statut
- Clic "+ Nouvelle commande" → Assistant commande
- Clic "Détails →" → Page détail commande

# API ENDPOINTS
GET /telephony/{ba}/order → Liste des commandes
GET /telephony/{ba}/order/{id} → Détail commande
POST /telephony/{ba}/order → Nouvelle commande
GET /telephony/{ba}/order/{id}/status → Statut commande
