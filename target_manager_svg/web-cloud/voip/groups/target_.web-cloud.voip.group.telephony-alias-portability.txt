# CAHIER DES CHARGES - old_manager.telecom.telephony.alias-portability
# Page portabilite des numeros

## 1. RESUME EXECUTIF

### Perimetre
Gestion des demandes de portabilite de numeros vers OVHcloud.

### Objectif fonctionnel
Suivre et gerer les portabilites en cours: statut, documents, relance, annulation.
