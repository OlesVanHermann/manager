# Cahier des Charges : telecom/telephony - Facturation groupe

## 1. RESUME EXECUTIF

### Perimetre
Page de facturation et releves d'un groupe de telephonie.

### Objectif fonctionnel
- Consulter les releves de consommation
- Gerer le depot de garantie
- Acceder aux coordonnees bancaires
- Consulter les couts SVA

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/bill
GET /telephony/{billingAccount}/bill/{id}
GET /telephony/{billingAccount}/bill/{id}/document
GET /telephony/{billingAccount}/historyConsumption
```

### Modele Bill
```typescript
interface Bill {
  date: string;
  amountHT: number;
  paid: boolean;
  pdfUrl: string;
  csvOutboundUrl: string;
  csvInboundUrl: string;
}
```
