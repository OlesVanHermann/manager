# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste groupes avec sélection
4. RIGHT PANEL
   - Header: Nom groupe + bouton Commander
   - NAV3: [Dashboard] | Services | Répertoire | Facturation | Administration
   - Tiles row 1:
     * "Je veux..." (4 actions rapides)
     * "Informations générales" (état, compteurs, crédit)
     * "Consommation du mois" (graphique circulaire)
   - Table "Mes 5 derniers relevés" (date, montant, statut, actions)
   - Section "Mon parc VoIP" (cartes téléphones avec pagination)

# INTERACTIONS
- Clic groupe LEFT PANEL → Sélection
- Clic action rapide → Navigation
- Clic tab NAV3 → Navigation section
- Clic PDF/CSV → Téléchargement
- Clic "Voir tout" parc → Liste complète

# API ENDPOINTS
GET /telephony/{ba}                     → Détails groupe
GET /telephony/{ba}/serviceInfos        → Infos facturation
GET /telephony/{ba}/line                → Liste lignes
GET /telephony/{ba}/line/{sn}/phone     → Détails téléphone
GET /telephony/{ba}/historyConsumption  → Historique conso
