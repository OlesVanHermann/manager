STRUCTURE PAGE
1. Left panel: Liste groupes VoIP
2. Right panel header: Nom groupe + billingAccount
3. NAV3: Dashboard (actif) | Services | Repertoire | Facturation | Administration
4. Stats cards: 4 cards (Lignes, Numeros, Fax, Credit)
5. Actions rapides: 3 boutons
6. Activite recente: Table derniers evenements
7. Consommation du mois: 2 progress bars

NAV3 GROUPE (5 tabs)
| Tab           | Route          | Description                  |
|---------------|----------------|------------------------------|
| Dashboard     | /dashboard     | Vue d'ensemble groupe        |
| Services      | /services      | Liste services detaillee     |
| Repertoire    | /phonebook     | Contacts partages            |
| Facturation   | /billing       | Conso, factures, credit      |
| Administration| /admin         | Params, contacts, securite   |

TYPES EVENEMENTS
| Type               | Description                    |
|--------------------|--------------------------------|
| Appel entrant      | Appel recu sur une ligne       |
| Appel sortant      | Appel emis depuis une ligne    |
| Config modifiee    | Changement parametres          |
| Fax envoye         | Fax transmis                   |
| Credit recharge    | Ajout credit prepaye           |

ACTIONS RAPIDES
| Action           | Destination                           |
|------------------|---------------------------------------|
| Gerer services   | /web-cloud/voip/{ba}/services         |
| Commander        | /web-cloud/voip/{ba}/order            |
| Recharger credit | /web-cloud/voip/{ba}/billing/credit   |

API ENDPOINTS
GET  /telephony                         -> Liste groupes
GET  /telephony/{ba}                    -> Details groupe
GET  /telephony/{ba}/serviceInfos       -> Infos facturation
GET  /telephony/{ba}/line               -> Liste lignes
GET  /telephony/{ba}/number             -> Liste numeros
GET  /telephony/{ba}/fax                -> Liste fax
GET  /telephony/{ba}/historyConsumption -> Historique conso
