# Cahier des Charges : telecom/telephony - Modal Nettoyage numeros abreges

## 1. RESUME EXECUTIF

### Perimetre
Modal de confirmation de suppression de tous les numeros abreges.

### Objectif fonctionnel
- Confirmer la suppression massive
- Avertir de l'irreversibilite de l'action

---

## 2. LAYOUT

### Structure modal compacte avec warning

---

## 3. COMPOSANTS UI

### Modal Header
| Element | Description |
|---------|-------------|
| Titre | Nettoyage de la liste des numeros abreges |
| Icone | Warning (triangle jaune) |
| Bouton fermer | X en haut a droite |

### Contenu
| Element | Description |
|---------|-------------|
| Message | L'ensemble de vos numeros abreges va etre supprime. |
| Question | Voulez-vous continuer ? |

### Boutons
| Bouton | Type | Action |
|--------|------|--------|
| Confirmer | Danger (rouge) | Supprime tous les numeros |
| Annuler | Secondary | Ferme la modal |

---

## 4. APIS & DONNEES

### Endpoint
```
DELETE /telephony/{billingAccount}/abbreviatedNumber/batch
    ou
POST /telephony/{billingAccount}/abbreviatedNumber/removeAll
```

## 6. REGLES METIER

- Action irreversible
- Supprime TOUS les numeros abreges du groupe
- Confirmation obligatoire
- Pas de restauration possible
