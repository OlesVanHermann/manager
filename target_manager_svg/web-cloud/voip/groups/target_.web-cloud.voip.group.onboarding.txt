# CAHIER DES CHARGES - old_manager.telecom.telephony.onboarding
# Page d'accueil vide (empty state) VoIP

## 1. RESUME EXECUTIF

### Perimetre
Page affichee lorsque l'utilisateur n'a aucun service VoIP actif.

### Objectif fonctionnel
Presenter les offres VoIP OVHcloud et inciter a la commande.

### Users cibles
- Nouveaux clients sans service VoIP
- Clients ayant resilie tous leurs services
