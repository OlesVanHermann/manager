# Cahier des Charges : telecom/telephony - Telephone ligne

## 1. RESUME EXECUTIF

### Perimetre
Page de gestion du telephone associe a une ligne VoIP.

### Objectif fonctionnel
- Voir les informations du telephone
- Configurer les options (touches, codecs, contacts)
- Commander accessoires
- Rattacher/detacher le telephone

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/line/{serviceName}/phone
PUT /telephony/{billingAccount}/line/{serviceName}/phone
POST /telephony/{billingAccount}/line/{serviceName}/phone/reboot
POST /telephony/{billingAccount}/line/{serviceName}/phone/changePhoneConfiguration
GET /telephony/{billingAccount}/line/{serviceName}/phone/functionKey
```

### Modele Phone
```typescript
interface Phone {
  model: string;
  macAddress: string;
  ipAddress: string;
  firmwareVersion: string;
  status: 'connected' | 'disconnected';
}
```
