# Cahier des Charges : telecom/telephony - Options groupe

## 1. RESUME EXECUTIF

### Perimetre
Page de configuration des options globales d'un groupe de telephonie.

### Objectif fonctionnel
- Configurer le masquage numero sortant
- Autoriser la substitution numero
- Afficher description sur appels entrants

## 4. APIS & DONNEES

### Endpoints
```
GET /telephony/{billingAccount}/options
PUT /telephony/{billingAccount}/options
```

### Modele Options
```typescript
interface GroupOptions {
  hideOutboundNumber: boolean;
  allowNumberSubstitution: boolean;
  showDescriptionOnInbound: boolean;
}
```
