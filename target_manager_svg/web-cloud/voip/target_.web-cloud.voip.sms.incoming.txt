================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic compte → Charge les détails du compte
- Clic NAV4 → Navigation vers l'onglet
- Clic ↻ → Actualiser liste
- Clic Configurer → Modal config numéro réception
- Clic checkbox → Sélection multiple
- Clic ↩ → Répondre au SMS
- Clic × → Supprimer SMS
- Clic Exporter CSV → Téléchargement fichier
- Clic ligne → Marquer comme lu

================================================================================

API ENDPOINTS

GET /sms → Liste comptes SMS
GET /sms/{account}/receivers → Numéros de réception
GET /sms/{account}/incoming → Liste SMS reçus
GET /sms/{account}/incoming/{id} → Détails SMS
PUT /sms/{account}/incoming/{id} → Marquer lu
DELETE /sms/{account}/incoming/{id} → Supprimer SMS
POST /sms/{account}/jobs → Répondre (créer SMS sortant)
