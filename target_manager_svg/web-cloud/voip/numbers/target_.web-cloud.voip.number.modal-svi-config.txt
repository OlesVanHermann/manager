# Cahier des Charges : telecom/alias - Modal Configuration SVI

## 1. RESUME EXECUTIF

### Perimetre
Modal de configuration du Serveur Vocal Interactif.

### Objectif fonctionnel
- Configurer l'URL du handler
- Configurer l'URL d'enregistrement
- Valider les URLs

---

## 2. LAYOUT

### Structure modal popover avec formulaire

---

## 3. COMPOSANTS UI

### Header modal
| Element | Description |
|---------|-------------|
| Titre | Configuration du SVI |
| Bouton X | Fermer modal |

### Champ URL
| Element | Description |
|---------|-------------|
| Label | URL * |
| Protocole | Select (https://) |
| Input | Champ URL |

### Protocoles disponibles
| Protocole | Usage |
|-----------|-------|
| https:// | URL handler (recommande) |
| http:// | URL handler (non secure) |
| ftp:// | URL enregistrement |
| mailto: | Email (rare) |

### Champ URL Record
| Element | Description |
|---------|-------------|
| Label | URL d'enregistrement |
| Protocole | Select (ftp://) |
| Input | Champ URL |
| Help | Texte aide optionnel |

### Dropdown protocole
| Etat | Style |
|------|-------|
| Ferme | Bordure normale |
| Ouvert | Liste deroulante |
| Selectionne | Fond bleu |

### Message erreur
| Element | Description |
|---------|-------------|
| Fond | Rouge clair |
| Icone | Warning |
| Ligne 1 | URL doit etre valide |
| Ligne 2 | Format attendu |

### Footer
| Bouton | Type | Action |
|--------|------|--------|
| Enregistrer | Primary | Sauvegarde |
| Annuler | Default | Ferme |

## 4. APIS & DONNEES

### Endpoints
```
PUT /telephony/{billingAccount}/svi/{serviceName}
```

### Parametres
```typescript
interface SviConfig {
  url: string;
  urlRecord: string;
}
```
