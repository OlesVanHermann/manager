## API ENDPOINTS

### Liste portabilités
GET /telephony/{billingAccount}/portability
Query: { status, page, limit }
Response: { items: [...], totalCount, stats }

### Détails portabilité
GET /telephony/{billingAccount}/portability/{reference}
Response: { reference, numbers, operator, requestDate, portingDate, status, documents }

### Nouvelle portabilité
POST /telephony/{billingAccount}/portability
Body: { numbers: [], operator, rio? }

### Ajouter document
POST /telephony/{billingAccount}/portability/{reference}/documents
Body: FormData avec fichier

================================================================================

## REGLES METIER

- Délai portabilité: 7-30 jours selon opérateur
- Documents requis: RIO, facture, mandat
- Statuts: pending, docs_required, in_progress, completed, rejected
- Numéros géographiques ou mobiles
- Portage groupé possible (même opérateur)
