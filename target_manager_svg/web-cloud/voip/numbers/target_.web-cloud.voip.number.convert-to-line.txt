## DESCRIPTION

Conversion d'un numéro alias en ligne téléphonique complète. Sélection
de l'offre de ligne et transformation du numéro.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| 0820 123 456             | GET /telephony/.../number → serviceName                 |
| Alias géographique       | number.featureType                                      |
| +33 9 12 34 56 78        | number.destination                                      |
| 9,99 € HT/mois           | GET /order/.../line/offers → prices                     |

================================================================================

## STRUCTURE PAGE

1. LEFT PANEL (280px)

   - ACTIONS: Convertir, Annuler

## API ENDPOINTS

### Informations numéro
GET /telephony/{billingAccount}/number/{serviceName}
Response: {
  serviceName, featureType, destination
}

### Offres lignes disponibles
GET /order/{billingAccount}/number/{serviceName}/lineOffers
Response: [{
  offerId, name, description, features: [], monthlyPrice
}]

### Convertir en ligne
POST /telephony/{billingAccount}/number/{serviceName}/convertToLine
Body: { lineOfferId }
Response: { lineServiceName, estimatedActivation }
