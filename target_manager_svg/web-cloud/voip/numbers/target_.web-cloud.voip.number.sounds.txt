4. RIGHT PANEL
   - Header: Numéro + badge type + breadcrumb /Sons
   - NAV3: Général | Configuration | Agents | Statistiques | [Sons] | Horaires

   - Toolbar: Refresh, Recherche, + Ajouter son

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic ↻ → Rafraîchir liste
- Clic "+ Ajouter un son" → Modal upload audio
- Clic ▶ → Lire le fichier audio
- Clic ⬇ → Télécharger le fichier
- Clic ✎ → Modal édition (nom, type)
- Clic ✕ → Supprimer son (confirmation)

# API ENDPOINTS
GET /telephony/{ba}/number/{sn}/easyHunting/sound → Liste sons
POST /telephony/{ba}/number/{sn}/easyHunting/sound → Upload son
GET /telephony/{ba}/number/{sn}/easyHunting/sound/{id} → Détails son
PUT /telephony/{ba}/number/{sn}/easyHunting/sound/{id} → Modifier son
DELETE /telephony/{ba}/number/{sn}/easyHunting/sound/{id} → Supprimer son
GET /telephony/{ba}/number/{sn}/easyHunting/sound/{id}/file → Download fichier

# FORMATS AUDIO SUPPORTÉS
- WAV (recommandé): 16-bit, 8kHz mono
- MP3: Compatible avec conversion
- OGG: Compatible avec conversion
- Taille max: 10 MB par fichier
