STRUCTURE PAGE
1. SOUS-NAV: Lignes | Numeros (actif) | Fax | Abreges
2. Left panel: Liste numeros du groupe
3. NAV3: General | Configuration | Statistiques | Portabilite | Enregistrements (actif)
4. Recording status card (statut + retention + stockage)
5. Filters: Recherche + periode + agent + export
6. Table enregistrements avec lecteur audio
7. Pagination

PARAMETRES ENREGISTREMENT (modal)
| Option     | Type      | Valeurs              |
|------------|-----------|----------------------|
| Activer    | toggle    | Oui/Non              |
| Retention  | select    | 30/60/90/180 jours   |
| Format     | select    | MP3/WAV              |
| Annonce    | toggle    | Informer appelant    |

API ENDPOINTS

GET    /telephony/{ba}/ovhPabx/{sn}/records/{id}           -> Detail
GET    /telephony/{ba}/ovhPabx/{sn}/records/{id}/audio     -> Telecharger audio
DELETE /telephony/{ba}/ovhPabx/{sn}/records/{id}           -> Supprimer
GET    /telephony/{ba}/ovhPabx/{sn}/records/settings       -> Parametres
PUT    /telephony/{ba}/ovhPabx/{sn}/records/settings       -> Modifier parametres
