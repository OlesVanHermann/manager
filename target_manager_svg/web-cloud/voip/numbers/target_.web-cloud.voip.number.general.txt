## DESCRIPTION

Page générale d'un numéro VoIP (alias). Affiche les informations du numéro,
sa configuration actuelle et les types de configuration disponibles.

================================================================================

## STRUCTURE PAGE

### NAV3 (Left Panel)
- Lignes | [Numéros] | Fax | Abrégés

### NAV4 Tabs (Right Panel)
- [Général] | Configuration | Statistiques | Options

### Contenu
- Breadcrumb vers groupe
- Header: numéro + description + badge type + actions
- Bloc Informations du numéro
- Bloc Configuration actuelle (schéma visuel)
- Bloc Types de configuration
- Zone danger (résiliation)

================================================================================

## DONNÉES DYNAMIQUES

| Élément SVG          | Source API                                 |
|----------------------|--------------------------------------------|
| Numéro               | serviceName                                |
| Description          | GET /telephony/{ba}/number/{sn} → description |
| Type configuration   | number.featureType                         |
| Pays                 | number.country                             |
| Date activation      | serviceInfos.creation                      |
| Engagement           | serviceInfos.engagedUpTo                   |
| Config redirection   | GET /telephony/{ba}/number/{sn}/redirect   |
| Config queue         | GET /telephony/{ba}/number/{sn}/queue      |
| Config IVR           | GET /telephony/{ba}/number/{sn}/ovhPabx    |

================================================================================

## TYPES DE CONFIGURATION (featureType)

| Type       | Badge     | Description                        |
|------------|-----------|-----------------------------------|
| redirect   | Bleu      | Redirection simple vers numéro    |
| ddi        | Gris      | SDA vers ligne                    |
| queue      | Jaune     | File d'attente                    |
| ivr        | Violet    | Serveur vocal interactif          |
| conference | Vert      | Salle de conférence               |
| voicemail  | Orange    | Messagerie vocale                 |

================================================================================

## BLOC CONFIGURATION ACTUELLE

Schéma visuel montrant le flux des appels:
- Box "Appel entrant" (bleu)
- Flèche direction
- Box "Destination" (vert) - variable selon type

================================================================================

## ACTIONS HEADER

| Action       | Bouton        | API                              |
|--------------|---------------|----------------------------------|
| Renommer     | ✎ Renommer    | PUT /telephony/{ba}/number/{sn}  |
| Changer type | ⇄ Changer     | POST /telephony/{ba}/number/{sn}/changeFeatureType |
