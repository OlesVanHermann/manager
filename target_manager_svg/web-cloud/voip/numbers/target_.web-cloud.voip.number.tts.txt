## DESCRIPTION

Création de messages vocaux par synthèse vocale (Text-To-Speech). Conversion
de texte en fichiers audio utilisables dans les menus SVI.

================================================================================

| Exemple SVG              | Source API                                              |

| 0:12                     | sounds[].duration                                       |
| 28/12/2024               | sounds[].createdAt                                      |

## STRUCTURE PAGE

1. LEFT PANEL (280px)
   - Recherche numéros
   - Liste des numéros

2. RIGHT PANEL (1096px)
   - Header: Titre + description

   - BLOC CREATION:
     * Textarea texte (obligatoire)
     * Dropdown voix
     * Dropdown vitesse
     * Dropdown tonalité
     * Boutons: Prévisualiser, Créer

   - TABLE SONS TTS:
     * Nom
     * Texte (aperçu)
     * Voix
     * Durée
     * Date création
     * Actions: Play, Download, Delete

================================================================================

## API ENDPOINTS

### Liste sons TTS
GET /telephony/{billingAccount}/number/{serviceName}/tts
Response: [{
  id, name, text, voice, speed, duration, createdAt
}]

### Créer son TTS
POST /telephony/{billingAccount}/number/{serviceName}/tts
Body: { text, voice, speed, pitch, name }
Response: { id, audioUrl }

### Prévisualiser (sans sauvegarder)
POST /telephony/{billingAccount}/number/{serviceName}/tts/preview
Body: { text, voice, speed, pitch }
Response: { previewUrl }

### Télécharger son
GET /telephony/{billingAccount}/number/{serviceName}/tts/{id}/download

### Supprimer son
DELETE /telephony/{billingAccount}/number/{serviceName}/tts/{id}

## REGLES METIER

- Texte: max 1000 caractères
- Durée max: 5 minutes
- Max 50 sons par numéro
- Formats: MP3, WAV
