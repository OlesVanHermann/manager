# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des numéros du groupe
4. RIGHT PANEL
   - Header: Numéro + description
   - NAV3: Dashboard | ... | [Appels] | Consommation | ...
   - Toolbar: Rafraîchir, Exporter CSV, Filtres dates
   - Tableau historique appels
   - Pagination

# TOOLBAR
- Bouton Rafraîchir
- Bouton Exporter CSV
- Filtre date début
- Filtre date fin
- Bouton Filtrer

# COLONNES TABLEAU
- DATE: Date et heure de l'appel
- APPELANT: Numéro de l'appelant
- AGENT: Agent ayant répondu (ou "--" si non répondu)
- ATTENTE: Temps d'attente avant réponse
- DURÉE: Durée de communication
- STATUT: Répondu/Manqué/Abandonné (badges couleur)
- ENREGISTREMENT: Lien écouter (si enregistré)
- ACTIONS: Bouton détails

# STATUTS
- Répondu: Badge vert #DEF7EC
- Manqué: Badge rouge #FEE2E2
- Abandonné: Badge orange #FEF3C7

# PAGINATION
- Affichage X-Y sur Z appels
- Navigation pages: ‹ 1 2 3 ... ›

# INTERACTIONS
- Clic "Rafraîchir" → Recharge données
- Clic "Exporter" → Téléchargement CSV
- Clic "Filtrer" → Applique filtres dates
- Clic "Écouter" → Lecture enregistrement
- Clic "Détails" → Modal détails appel
- Clic pagination → Change de page

# API ENDPOINTS
GET /telephony/{ba}/easyHunting/{sn}/records              → Historique
GET /telephony/{ba}/easyHunting/{sn}/records/{id}         → Détail appel
GET /telephony/{ba}/easyHunting/{sn}/records/{id}/file    → Enregistrement audio
