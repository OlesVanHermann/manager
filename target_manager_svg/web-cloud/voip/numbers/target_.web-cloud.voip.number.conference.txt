## DESCRIPTION

Configuration d'une salle de conférence audio VoIP. Paramètres d'accès,
codes PIN et options de la conférence.

================================================================================

## STRUCTURE PAGE

### NAV4 Tabs
- Général | [Configuration] | Statistiques | Options

### Contenu
- Cards statut temps réel (3)
- Bloc Accès à la conférence
- Bloc Paramètres de la conférence
- Bloc Invitation

================================================================================

## ACCÈS À LA CONFÉRENCE

| Champ              | Format        | Copiable |
|--------------------|---------------|----------|
| Numéro d'accès     | +33 X XX XX XX| Oui      |
| Code participant   | 4 chiffres    | Oui      |
| Code modérateur    | 4 chiffres    | Oui      |

### Actions codes PIN
- Régénérer: POST /telephony/{ba}/number/{sn}/conference/generatePin
- Le code modérateur donne des droits supplémentaires (mute, kick)

================================================================================

## PARAMÈTRES CONFÉRENCE

| Paramètre                   | Type    | Valeurs                 |
|-----------------------------|---------|-------------------------|
| Annonce entrée/sortie       | Toggle  | Activé/Désactivé        |
| Type d'annonce              | Select  | Bip, Nom, Aucun         |
| Musique d'attente           | Toggle  | Activé/Désactivé        |
| Musique                     | Select  | Classique, Jazz, Custom |
| Enregistrement auto         | Toggle  | Activé/Désactivé (payant)|
| Modérateur requis           | Toggle  | Oui/Non                 |

================================================================================

## INVITATION

Bloc pré-formaté avec:
- Numéro d'accès
- Code participant
- Code modérateur
- Capacité

Action: Bouton "Envoyer invitation" ouvre modal email

## API

| Action           | Méthode | Endpoint                                 |
|------------------|---------|------------------------------------------|
| Config conf      | GET     | /telephony/{ba}/number/{sn}/conference   |
| Modifier         | PUT     | /telephony/{ba}/number/{sn}/conference   |
| Générer PIN      | POST    | .../conference/generatePin               |
| Participants     | GET     | .../conference/participants              |
| Kick participant | DELETE  | .../conference/participants/{id}         |
| Mute participant | PUT     | .../conference/participants/{id}/mute    |
