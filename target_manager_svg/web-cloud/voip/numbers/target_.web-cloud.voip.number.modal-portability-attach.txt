# Cahier des Charges : telecom/alias - Modal Joindre document portabilite

## 1. RESUME EXECUTIF

### Perimetre
Modal d'upload de document pour une portabilite.

### Objectif fonctionnel
- Joindre le formulaire client
- Joindre le mandat de portabilite
- Ajouter des pieces justificatives

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/portability/{id}/document
```

### Parametres
```typescript
interface DocumentUpload {
  file: File;
  type: 'mandat' | 'formulaire' | 'justificatif';
}
```
