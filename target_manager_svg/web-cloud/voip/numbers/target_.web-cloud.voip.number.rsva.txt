## DESCRIPTION

Suivi des reversements pour les numéros à valeur ajoutée (SVA/surtaxés).
Gestion du solde, configuration IBAN et historique des versements.

## API ENDPOINTS

### Résumé RSVA
GET /telephony/{billingAccount}/number/{serviceName}/rsva
Response: {
  balance, monthlyEarnings, monthlyCallsCount,
  ratePerMinute, palier
}

### Configuration versement
GET /telephony/{billingAccount}/number/{serviceName}/rsva/config
Response: { iban, threshold, frequency }

### Modifier configuration
PUT /telephony/{billingAccount}/number/{serviceName}/rsva/config
Body: { iban, threshold, frequency }

### Historique versements
GET /telephony/{billingAccount}/number/{serviceName}/rsva/payments
Response: [{
  id, date, period, callsCount, totalDuration, amount, status, invoiceUrl
}]

### Demander versement
POST /telephony/{billingAccount}/number/{serviceName}/rsva/payout
Response: { payoutId, estimatedDate }
