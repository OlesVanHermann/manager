# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des numéros
4. RIGHT PANEL
   - Header: Numéro + badge + breadcrumb /Statistiques
   - NAV3: Général | Configuration | Consommation | [Statistiques]
   - Sélecteur période
   - KPI Cards Row 1: Appels reçus | Traités | Perdus | Temps attente
   - KPI Cards Row 2: Temps conversation | Agents dispo | Niveau service | Appels/heure
   - Graphique volume appels par jour
   - Top agents
   - Boutons Export CSV + Rapport détaillé

# KPI METRICS
| Métrique              | Calcul                              |
|-----------------------|-------------------------------------|
| Appels reçus          | Total entrants sur la période       |
| Appels traités        | Appels décrochés / Total * 100      |
| Appels perdus         | Appels non décrochés (abandonné, timeout) |
| Temps attente moyen   | Somme temps file / Nb appels        |
| Temps conversation    | Somme durée appels / Nb appels      |
| Niveau de service     | % appels répondus < seuil (ex: 20s) |
| Appels/heure          | Moyenne horaire sur période         |

# INTERACTIONS
- Sélecteur période → Changer plage temporelle
- Clic graphique → Détail journée
- Clic "Voir toutes les stats agents →" → Page détail agents
- Clic "Exporter CSV" → Téléchargement données brutes
- Clic "Rapport détaillé" → Génération rapport PDF

# API ENDPOINTS
GET /telephony/{ba}/easyHunting/{sn}/statistics            → Stats globales
GET /telephony/{ba}/easyHunting/{sn}/statistics?from=&to=  → Stats période
GET /telephony/{ba}/easyHunting/{sn}/statistics/byDay      → Stats par jour
GET /telephony/{ba}/easyHunting/{sn}/statistics/byHour     → Stats par heure
GET /telephony/{ba}/easyHunting/{sn}/agent/statistics      → Stats par agent
