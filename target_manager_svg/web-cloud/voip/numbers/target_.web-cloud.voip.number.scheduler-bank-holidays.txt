# Cahier des Charges : telecom/scheduler - Ajouter jours feries

## 1. RESUME EXECUTIF

### Perimetre
Modal d'ajout des jours feries au calendrier des plages horaires.

### Objectif fonctionnel
- Selectionner un pays
- Choisir une annee
- Cocher les jours feries a ajouter

## 4. APIS & DONNEES

### Endpoints
```
GET /me/availablePublicHolidays?country={country}&year={year}
POST /telephony/{billingAccount}/timeCondition/{serviceName}/condition (batch)
```

### Modele
```typescript
interface PublicHoliday {
  name: string;
  date: string;
  country: string;
}

interface BankHolidaySelection {
  country: string;
  year: number;
  holidays: PublicHoliday[];
}

interface AddHolidaysResult {
  added: number;
  skipped: number;
  errors: string[];
}
```
