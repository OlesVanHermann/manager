STRUCTURE PAGE
1. SOUS-NAV: Lignes | Numeros (actif) | Fax | Abreges
2. Left panel: Liste numeros du groupe
3. NAV3: General | Configuration | Statistiques | Portabilite (actif) | Enregistrements
4. Info banner: Explication portabilite
5. Button: Nouvelle portabilite
6. Table historique portabilites
7. Timeline etapes (pour portabilite en cours)

STATUTS PORTABILITE
| Statut           | Couleur   | Description                  |
|------------------|-----------|------------------------------|
| En attente RIO   | #DBEAFE   | RIO non encore saisi         |
| En cours         | #FEF3C7   | Portabilite en traitement    |
| Termine          | #DEF7EC   | Portage effectue             |
| Rejete           | #FEE2E2   | Demande refusee              |
| Annule           | #F3F4F6   | Annule par utilisateur       |

COLONNES TABLE
| Colonne      | Description                    |
|--------------|--------------------------------|
| Numero       | Numero a porter                |
| Operateur    | Operateur cedant               |
| Date demande | Date creation demande          |
| Date portage | Date effective du portage      |
| Statut       | Badge etat                     |

| Actions      | Selon statut                   |

ETAPES TIMELINE
| Etape | Description                  |
|-------|------------------------------|
| 1     | Demande envoyee              |
| 2     | RIO valide                   |
| 3     | En attente accord operateur  |
| 4     | Portage effectif             |

ACTIONS PAR STATUT
| Statut           | Actions disponibles            |
|------------------|--------------------------------|
| En attente RIO   | Saisir RIO, Annuler            |
| En cours         | Annuler                        |
| Termine          | Details                        |
| Rejete           | Relancer, Details              |

FORMULAIRE NOUVELLE PORTABILITE (modal)
| Champ           | Type    | Requis |
|-----------------|---------|--------|
| Numero a porter | text    | Oui    |
| Operateur       | select  | Oui    |
| RIO             | text    | Oui    |
| Date souhaitee  | date    | Non    |

API ENDPOINTS
GET    /telephony/{ba}/portability                    -> Liste portabilites
POST   /telephony/{ba}/portability                    -> Creer demande
GET    /telephony/{ba}/portability/{id}               -> Detail
PUT    /telephony/{ba}/portability/{id}               -> Modifier (RIO)
DELETE /telephony/{ba}/portability/{id}               -> Annuler
POST   /telephony/{ba}/portability/{id}/retry         -> Relancer
