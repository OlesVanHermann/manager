## STATUTS PORTABILITÉ

| Statut      | Badge     | Description                    |
|-------------|-----------|--------------------------------|
| Terminé     | Vert      | Portage effectué avec succès   |
| En cours    | Jaune     | Portage en attente             |

## CODE RIO

Format: 2 lettres + 12 chiffres (ex: 01XXXXXXXXXXXX)
- Permet d'identifier le numéro auprès de l'opérateur
- Nécessaire pour toute demande de portabilité sortante
- Valable 30 jours après génération
