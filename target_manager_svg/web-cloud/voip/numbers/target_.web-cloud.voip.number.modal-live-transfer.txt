# Cahier des Charges : telecom/alias - Modal Transferer l'appel

## 1. RESUME EXECUTIF

### Perimetre
Modal de transfert d'un appel en cours.

### Objectif fonctionnel
- Rediriger un appel vers un autre numero
- Afficher les infos de l'appel
- Confirmer le transfert

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/ovhPabx/{serviceName}/hunting/queue/{queueId}/liveCalls/{id}/transfer
```

### Parametres
```typescript
interface TransferParams {
  number: string;
}
```
