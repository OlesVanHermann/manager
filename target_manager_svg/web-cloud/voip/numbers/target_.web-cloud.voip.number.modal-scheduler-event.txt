# Cahier des Charges : telecom/alias - Modal Evenement calendrier

## 1. RESUME EXECUTIF

### Perimetre
Modal de creation/edition d'evenement calendrier.

### Objectif fonctionnel
- Creer des periodes speciales
- Definir dates et heures
- Categoriser les evenements

## 4. APIS & DONNEES

### Endpoints
```
POST /telephony/{billingAccount}/scheduler/{serviceName}/events
PUT /telephony/{billingAccount}/scheduler/{serviceName}/events/{eventId}
DELETE /telephony/{billingAccount}/scheduler/{serviceName}/events/{eventId}
```

### Modele Event
```typescript
interface SchedulerEvent {
  uid: string;
  title: string;
  dateStart: string;
  dateEnd: string;
  categories: string;
  description: string;
}
```
