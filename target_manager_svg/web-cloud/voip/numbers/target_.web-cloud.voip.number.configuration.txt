STRUCTURE PAGE
1. SOUS-NAV: Lignes | Numeros (actif) | Fax | Abreges
2. Left panel: Liste numeros du groupe
3. NAV3: General | Configuration (actif) | Portabilite | Statistiques
4. Cards:
   - Strategie de sonnerie (mode, timeout membre, timeout global)
   - Musique d'attente (son actuel, upload)
   - Membres de la file (table avec ordre, destination, type, timeout, statut)

STRATEGIES DISPONIBLES
| Strategy        | API Value         | Description                     |
|-----------------|-------------------|---------------------------------|
| Sequentielle    | ringAll           | Sonne membres un par un         |
| Simultanee      | memoryRing        | Sonne tous en meme temps        |
| Aleatoire       | random            | Ordre aleatoire                 |
| Moins recente   | leastRecent       | Membre ayant recu moins d'appels|
| Round Robin     | roundRobin        | Rotation cyclique               |

TYPES DE MEMBRES
| Type            | Description                     |
|-----------------|---------------------------------|
| Ligne interne   | Ligne SIP du groupe             |
| Numero externe  | Numero hors groupe (mobile...)  |
| Voicemail       | Messagerie vocale               |
| Alias           | Autre numero du groupe          |

ACTIONS MEMBRES
| Action     | Comportement                               |
|------------|--------------------------------------------|
| Editer     | Modal modification timeout, statut         |
| Supprimer  | Suppression avec confirmation              |
| Reordonner | Drag & drop pour changer ordre sonnerie    |

API ENDPOINTS
GET  /telephony/{ba}/ovhPabx/{sn}/hunting              -> Config file
PUT  /telephony/{ba}/ovhPabx/{sn}/hunting              -> Modifier strategie
GET  /telephony/{ba}/ovhPabx/{sn}/hunting/agent        -> Liste membres
POST /telephony/{ba}/ovhPabx/{sn}/hunting/agent        -> Ajouter membre
PUT  /telephony/{ba}/ovhPabx/{sn}/hunting/agent/{id}   -> Modifier membre
DELETE /telephony/{ba}/ovhPabx/{sn}/hunting/agent/{id} -> Supprimer membre
GET  /telephony/{ba}/ovhPabx/{sn}/sound                -> Son attente
POST /telephony/{ba}/ovhPabx/{sn}/sound                -> Upload son
