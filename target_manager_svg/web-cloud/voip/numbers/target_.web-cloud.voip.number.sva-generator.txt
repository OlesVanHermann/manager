## DESCRIPTION

Générateur de mentions légales obligatoires pour les numéros à valeur ajoutée.
Génère les mentions textuelles, audio et pictogrammes réglementaires.

================================================================================

## DONNEES DYNAMIQUES

| Exemple SVG              | Source API                                              |
|--------------------------|---------------------------------------------------------|
| 0891 123 456             | GET /telephony/.../number → serviceName                 |
| 0,80 €/min               | number.rate                                             |
| Standard (0891)          | number.palier                                           |

     * Mention courte (copyable)
     * Mention complète (copyable)
     * Annonce audio (écouter/télécharger)
     * Pictogramme SVA (PNG/SVG)

   - BANNIERE INFO REGLEMENTATION

## API ENDPOINTS

### Informations numéro SVA
GET /telephony/{billingAccount}/number/{serviceName}/sva
Response: {
  number, rate, palier, legalName
}

### Générer mentions
GET /telephony/{billingAccount}/number/{serviceName}/sva/mentions
Response: {
  shortMention, fullMention, audioUrl, pictogramPng, pictogramSvg
}

### Télécharger annonce audio
GET /telephony/{billingAccount}/number/{serviceName}/sva/audio

### Télécharger pictogramme
GET /telephony/{billingAccount}/number/{serviceName}/sva/pictogram
Query: { format: 'png' | 'svg' }
