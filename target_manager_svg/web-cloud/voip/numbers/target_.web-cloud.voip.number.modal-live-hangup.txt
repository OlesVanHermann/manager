# Cahier des Charges : telecom/alias - Modal Raccrocher appel

## 1. RESUME EXECUTIF

### Perimetre
Modal de confirmation pour raccrocher un appel en cours.

### Objectif fonctionnel
- Confirmer la fin d'appel
- Informer de la deconnexion des parties
- Mettre fin a l'appel
