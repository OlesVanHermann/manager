# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des numéros du groupe
4. RIGHT PANEL
   - Header: Numéro + description
   - NAV3: ... | [Consommation] | ...
   - Sélecteur période
   - 3 cards résumé (Total appels, Durée, Montant)
   - Graphique évolution
   - Répartition par type + Top agents

# SÉLECTEUR PÉRIODE
- Ce mois (actif par défaut)
- Mois dernier
- 3 derniers mois
- Personnalisé (ouvre date picker)

# CARDS RÉSUMÉ
1. Total appels: Nombre + évolution vs période précédente
2. Durée totale: Format Xh Xmin + moyenne par appel
3. Montant estimé: Format XX,XX € HT + info forfait

# GRAPHIQUE ÉVOLUTION
- Type: Barres ou ligne
- Données: Appels par jour
- Légende: Entrants / Répondus

# RÉPARTITION PAR TYPE
- Barres de progression horizontales
- Répondus (vert), Manqués (rouge), Abandonnés (orange)
- Pourcentages

# TOP AGENTS
- Classement par nombre d'appels traités
- Colonnes: Rang, Nom, Appels, Durée, %

# INTERACTIONS
- Clic période → Change les données affichées
- Clic "Voir détails" → Navigation Appels
- Clic "Voir tous les agents" → Navigation Agents

# API ENDPOINTS
GET /telephony/{ba}/easyHunting/{sn}/records              → Historique appels
GET /telephony/{ba}/easyHunting/{sn}/hunting/agent        → Liste agents
GET /telephony/{ba}/serviceInfos                          → Infos facturation
