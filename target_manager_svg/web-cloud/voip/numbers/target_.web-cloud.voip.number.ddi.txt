# QU'EST-CE QUE LE DDI ?
DDI (Direct Dial-In) permet de créer des extensions internes.
L'appelant compose le numéro principal + une extension pour joindre
directement un agent, une file ou un menu SVI sans passer par l'accueil.

Exemple: +33 800 123 456 + ext 101 → Service commercial
