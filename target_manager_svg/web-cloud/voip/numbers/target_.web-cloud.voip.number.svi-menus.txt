# ARBORESCENCE VISUELLE
- Vue graphique des menus imbriqués
- Nœud racine : Menu principal
- Nœuds enfants : Options DTMF
- Lignes de connexion entre menus

# TABLEAU MENUS
| Colonne | Description |
|---------|-------------|
| Menu | Nom du menu |
| Son d'accueil | Fichier audio |
| Options | Touches DTMF configurées |
| Timeout | Délai avant action par défaut |
| Actions | Edit, Delete |

# OPTIONS DTMF
| Touche | Action possible |
|--------|----------------|
| 0-9 | Renvoi, sous-menu, raccrocher |
| * | Répéter message |
| # | Retour menu parent |

# TYPES D'ACTIONS
| Type | Description |
|------|-------------|
| Transfer | Renvoi vers numéro |
| Queue | Renvoi vers file d'attente |
| Menu | Aller vers sous-menu |
| Hangup | Raccrocher |
| Playback | Jouer son puis action |

# INTERACTIONS
- "+ Ajouter menu" → Modal création menu
- ✎ → Modal édition menu/options
- × → Confirmation suppression
- "Éditer →" sur nœud → Édition options
