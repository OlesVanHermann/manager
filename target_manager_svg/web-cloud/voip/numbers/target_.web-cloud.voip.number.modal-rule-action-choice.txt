# Cahier des Charges : telecom/alias - Modal Choix d'action

## 1. RESUME EXECUTIF

### Perimetre
Modal de selection du type d'action pour le dialplan.

### Objectif fonctionnel
- Lister toutes les actions disponibles
- Decrire chaque action
- Selectionner une action

## 2. LAYOUT

### Structure modal avec liste radio

## 3. COMPOSANTS UI

### Header modal
| Element | Description |
|---------|-------------|
| Retour | Lien ‚Üê Retour |
| Titre | Choisir une action |
| Bouton X | Fermer modal |

### Liste actions
| Element | Description |
|---------|-------------|
| Radio | Selection exclusive |
| Titre | Nom de l'action |
| Description | Explication courte |

### Actions disponibles
| Action | Description |
|--------|-------------|
| Transferer | Redirige vers numero externe |
| File d'attente | Route vers file d'agents |
| Raccrocher | Termine immediatement |
| Menu IVR | Menu avec choix numeriques |
| Jouer son | Diffuse fichier audio |
| Lire DTMF | Capture touches appelant |
| Nom appelant | Modifie nom affiche |
| Pause | Attend avant continuer |
| TTS | Lit texte synthetique |
| Messagerie | Vers boite vocale |

### Etat selection
| Etat | Style |
|------|-------|
| Non selectionne | Bordure grise |
| Selectionne | Fond bleu, bordure bleue |

### Radio button
| Etat | Style |
|------|-------|
| Inactif | Cercle vide |
| Actif | Cercle bleu + point blanc |
