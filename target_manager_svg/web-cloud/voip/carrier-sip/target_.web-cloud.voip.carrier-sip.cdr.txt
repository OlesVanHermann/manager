# TABLE CDR
| Colonne | Description |
|---------|-------------|
| Date/Heure | Timestamp de l'appel |
| Source | Numéro appelant (CLI) |
| Destination | Numéro appelé (DNIS) |
| Endpoint | Point de terminaison utilisé |
| Durée | Durée de l'appel |
| Code SIP | Code réponse SIP (200, 486, etc) |
| Statut | OK / Busy / Timeout / Error |
| Call-ID | Identifiant unique SIP |

# CODES SIP COMMUNS
| Code | Statut | Badge |
|------|--------|-------|
| 200 | OK | Vert |
| 486 | Busy | Rouge |
| 487 | Cancelled | Jaune |
| 503 | Service unavailable | Rouge |
