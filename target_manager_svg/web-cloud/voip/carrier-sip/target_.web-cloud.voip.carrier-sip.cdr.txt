# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des trunks
4. RIGHT PANEL
   - Header: Identifiant trunk + badge statut + description
   - NAV3: Général | Paramètres | Routage | [CDR] | Consommation
   - Toolbar filtres: Refresh, Période, Direction, Statut, Export CSV
   - Cards stats: Total appels, Durée totale, Réussis, Échoués
   - Table CDR: Date, Direction, Source, Dest, Durée, Statut, Code SIP, Gateway, Coût
   - Pagination

# COLONNES TABLE
| Colonne | Largeur | Contenu |
|---------|---------|---------|
| Date/Heure | 130px | Format DD/MM HH:MM:SS |
| Direction | 80px | ↗ Sortant / ↙ Entrant |
| Source | 150px | Numéro appelant |
| Destination | 150px | Numéro appelé |
| Durée | 90px | Format Xmin Xs |
| Statut | 100px | Badge Réussi/Échec |
| Code SIP | 100px | Code réponse SIP |
| Gateway | 130px | Nom gateway utilisée |
| Coût | 86px | Montant € |

# FILTRES DISPONIBLES
- Période: Date début → Date fin (calendrier)
- Direction: Tous / Entrant / Sortant
- Statut: Tous / Réussi / Échec

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic ↻ → Rafraîchir données
- Dropdown période → Sélectionner dates
- Dropdown direction → Filtrer entrant/sortant
- Dropdown statut → Filtrer réussi/échec
- Clic "Exporter CSV" → Téléchargement fichier
- Clic ligne → Détails appel (modal)
- Clic pagination → Naviguer pages

# CODES SIP COURANTS
| Code | Signification |
|------|---------------|
| 200 OK | Appel réussi |
| 403 Forbidden | Appel refusé |
| 404 Not Found | Numéro inexistant |
| 480 Unavailable | Temporairement indisponible |
| 486 Busy | Occupé |
| 487 Cancelled | Annulé |
| 503 Service Unavailable | Service indisponible |

# API ENDPOINTS
GET /telephony/{ba}/trunk/{sn}/cdr                 → Liste CDR (paginée)
GET /telephony/{ba}/trunk/{sn}/cdr/{id}            → Détails appel
GET /telephony/{ba}/trunk/{sn}/cdr/export          → Export CSV
GET /telephony/{ba}/trunk/{sn}/cdr/statistics      → Stats agrégées

# PARAMÈTRES PAGINATION
- limit: 50 (défaut)
- offset: 0, 50, 100...
- sort: datetime DESC (défaut)
