# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des trunks SIP
4. RIGHT PANEL
   - Header: Trunk + description + badge statut
   - NAV3: Général | CDR | [Endpoints] | Routing | Settings
   - Toolbar: Ajouter, Rafraîchir, Voir credentials
   - Stats (3 tiles)
   - Tableau endpoints
   - Card Infos SIP

# TOOLBAR
- Bouton "+ Ajouter endpoint" (primaire)
- Bouton "Rafraîchir"
- Bouton "Voir credentials" (modal)

# STATS TILES
- Endpoints actifs: X / Y (fond vert)
- Appels simultanés max: Limite du trunk
- Appels en cours: Temps réel

# COLONNES TABLEAU
- ENDPOINT: Nom de l'endpoint
- TYPE: Inbound / Outbound / Bidirectional
- IP / FQDN: Adresse de destination
- PORT: Port SIP
- PROTOCOLE: UDP / TCP / TLS
- STATUT: Actif / Inactif (badges)
- APPELS: Nombre d'appels en cours
- ACTIONS: Modifier, Supprimer

# TYPES D'ENDPOINTS
- Inbound: Réception d'appels vers le trunk
- Outbound: Émission d'appels depuis le trunk
- Bidirectional: Les deux directions

# PROTOCOLES
- UDP: Protocole standard (port 5060)
- TCP: Pour traversée NAT
- TLS: Chiffré (port 5061)

# CARD INFOS SIP
- Domaine SIP du trunk
- Adresse IP du serveur
- Ports disponibles

# INTERACTIONS
- Clic "+ Ajouter" → Modal création endpoint
- Clic "Voir credentials" → Modal avec login/password SIP
- Clic "Modifier" → Modal édition endpoint
- Clic "Supprimer" → Modal confirmation

# API ENDPOINTS
GET /telephony/{ba}/carrierSip/{sn}                    → Détails trunk
GET /telephony/{ba}/carrierSip/{sn}/endpoints          → Liste endpoints
GET /telephony/{ba}/carrierSip/{sn}/endpoints/{id}     → Détail endpoint
POST /telephony/{ba}/carrierSip/{sn}/endpoints         → Créer endpoint
PUT /telephony/{ba}/carrierSip/{sn}/endpoints/{id}     → Modifier endpoint
DELETE /telephony/{ba}/carrierSip/{sn}/endpoints/{id}  → Supprimer endpoint
GET /telephony/{ba}/carrierSip/{sn}/settings           → Paramètres SIP
