STRUCTURE PAGE
1. Left panel: Liste trunks SIP
2. Header: Nom trunk + statut + label
3. NAV3: General (actif) | Parametres | Routage | CDR | Statistiques
4. Stats cards (4 KPIs)
5. Configuration SIP
6. Endpoints IP autorises
7. Appels recents

STATS CARDS
| Carte           | Donnee              | Indicateur     |
|-----------------|---------------------|----------------|
| Canaux actifs   | Utilises / Max      | Disponibles    |
| Appels auj.     | Compteur            | Delta vs hier  |
| Duree moyenne   | mm:ss               | Total          |
| Qualite SIP     | % reussite          | Niveau         |

CONFIGURATION SIP
| Champ        | Description                    |
|--------------|--------------------------------|
| Serveur SIP  | Hostname serveur OVH           |
| Port         | Port SIP (5060)                |
| Transport    | UDP/TCP                        |
| Identifiant  | user@domain (copiable)         |
| Mot de passe | Masque (affichable)            |
| Codecs       | Liste codecs supportes         |
| DTMF         | Mode DTMF (RFC 2833)           |

ENDPOINTS IP
| Colonne | Description                    |
|---------|--------------------------------|
| IP      | Adresse IP autorisee           |
| Label   | Description (IPBX, etc.)       |
| Statut  | Actif/Inactif                  |
| Actions | Supprimer                      |

APPELS RECENTS (preview CDR)
| Colonne     | Description                    |
|-------------|--------------------------------|
| Date        | Date/heure appel               |
| Source      | Numero appelant                |
| Destination | Numero appele                  |
| Duree       | Duree appel                    |
| Statut      | Repondu/Echec/...              |
| Direction   | Entrant/Sortant                |
| Qualite     | Score qualite                  |

API ENDPOINTS
GET /telephony/trunk                           -> Liste trunks
GET /telephony/trunk/{trunk}                   -> Detail trunk
GET /telephony/trunk/{trunk}/statistics        -> Stats
GET /telephony/trunk/{trunk}/endpoints         -> Liste IPs
POST /telephony/trunk/{trunk}/endpoints        -> Ajouter IP
DELETE /telephony/trunk/{trunk}/endpoints/{ip} -> Supprimer IP
GET /telephony/trunk/{trunk}/cdr               -> CDR recents
POST /telephony/trunk/{trunk}/regenerate       -> Regenerer credentials
