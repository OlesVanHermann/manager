# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des trunks
4. RIGHT PANEL
   - Header: Identifiant trunk + badge statut + description
   - NAV3: [Général] | Paramètres | Routage | CDR | Consommation
   - Card "Informations trunk": ID, Description, Canaux, État
   - Card "Connexion SIP": Serveur, Identifiant, Protocole, Codecs
   - Card "Statistiques temps réel": Appels en cours, entrants, sortants
   - Card "Consommation du mois": Appels, Durée, Coût
   - Card "Actions rapides": CDR, Routage, Export

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Modifier →" → Modal édition description
- Clic "Voir les identifiants →" → Modal credentials SIP
- Clic "Configurer →" → Page paramètres SIP
- Clic "Voir le monitoring →" → Dashboard temps réel
- Clic action rapide → Navigation page correspondante

# API ENDPOINTS
GET /telephony/{ba}/trunk/{sn}                     → Détails trunk
PUT /telephony/{ba}/trunk/{sn}                     → Modifier description
GET /telephony/{ba}/trunk/{sn}/serviceInfos        → Infos abonnement
GET /telephony/{ba}/trunk/{sn}/statistics          → Statistiques
GET /telephony/{ba}/trunk/{sn}/cdr                 → Call Detail Records
GET /telephony/{ba}/trunk/{sn}/externalDisplayedNumber → DIDs associés
