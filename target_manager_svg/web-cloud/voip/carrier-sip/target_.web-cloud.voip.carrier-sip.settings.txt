# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des trunks
4. RIGHT PANEL
   - Header: Identifiant trunk + badge statut + description
   - NAV3: Général | [Paramètres] | Routage | CDR | Consommation
   - Card "Paramètres de connexion SIP": Serveur, Port, Protocole, Credentials
   - Card "Codecs audio": Liste drag&drop avec priorité
   - Card "Sécurité": TLS, SRTP, Filtrage IP
   - Card "Options avancées": Session Timers, Max Forwards, DTMF
   - Boutons: Enregistrer / Annuler

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Dropdown protocole → Sélectionner UDP/TCP/TLS
- Clic "Afficher" mot de passe → Révéler credentials
- Drag&drop codecs → Réorganiser priorité
- Toggle TLS/SRTP → Activer/désactiver sécurité
- Clic "+ Ajouter IP" → Modal ajout IP whitelist
- Dropdown DTMF Mode → Sélectionner mode (RFC 2833, SIP INFO, In-band)
- Clic "Enregistrer" → PUT API + confirmation
- Clic "Annuler" → Reset formulaire

# API ENDPOINTS
GET /telephony/{ba}/trunk/{sn}                     → Détails trunk
PUT /telephony/{ba}/trunk/{sn}                     → Modifier paramètres
GET /telephony/{ba}/trunk/{sn}/codecs              → Liste codecs disponibles
PUT /telephony/{ba}/trunk/{sn}/codecs              → Modifier ordre codecs
GET /telephony/{ba}/trunk/{sn}/security            → Paramètres sécurité
PUT /telephony/{ba}/trunk/{sn}/security            → Modifier sécurité
GET /telephony/{ba}/trunk/{sn}/ipRestrictions      → Liste IPs autorisées
POST /telephony/{ba}/trunk/{sn}/ipRestrictions     → Ajouter IP
DELETE /telephony/{ba}/trunk/{sn}/ipRestrictions   → Supprimer IP
