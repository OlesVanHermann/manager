   - Bloc: Options SIP
   - Bannière info
   - Boutons: Annuler / Appliquer

# BLOCS CONFIGURATION
| Bloc | Champs |
|------|--------|
| Infos générales | ID, Description, Statut, Canaux max |
| Authentification | Méthode, Username, Password, Realm |
| Codecs | G.711 A-law, G.711 µ-law, G.729, Opus |
| Options | DTMF, T.38, SRTP, NAT Keepalive |

# CODECS SUPPORTÉS
| Codec | Bande passante | Qualité |
|-------|----------------|---------|
| G.711 A-law | 64 kbps | Excellente |
| G.711 µ-law | 64 kbps | Excellente |
| G.729 | 8 kbps | Bonne |
| Opus | Variable | Excellente |

# OPTIONS SIP
| Option | Valeurs |
|--------|---------|
| DTMF | RFC 2833 / SIP INFO / Inband |
| T.38 | Activé / Désactivé |
| SRTP | Activé / Désactivé |
| NAT Keepalive | 30s / 60s / Désactivé |

# INTERACTIONS
- Toggle codecs → Checkbox on/off
- Toggle options → Switch on/off
- "Appliquer" → PUT API + reload endpoints

# API ENDPOINTS
GET /telephony/{ba}/trunk/{sn}
PUT /telephony/{ba}/trunk/{sn}
PUT /telephony/{ba}/trunk/{sn}/codecs
PUT /telephony/{ba}/trunk/{sn}/options
