# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des lignes
4. RIGHT PANEL
   - Header: Numéro + breadcrumb /Appels
   - NAV3: Général | Téléphone | [Appels] | Transferts | Messagerie | Consommation | Options
   - NAV4: [Historique] | Filtrage entrant | Filtrage sortant
   - Toolbar: Refresh + Export CSV
   - Table historique appels:
     * Date/heure
     * Type (entrant/sortant avec icône)
     * Numéro (lien cliquable)
     * Durée (mm:ss)
     * Statut (badge couleur)
     * Destination (type)
   - Pagination

# STATUTS APPELS
| Statut   | Badge Color | Description           |
|----------|-------------|-----------------------|
| Répondu  | #DEF7EC     | Appel entrant pris    |
| Abouti   | #DEF7EC     | Appel sortant abouti  |
| Manqué   | #FEE2E2     | Appel entrant manqué  |
| Rejeté   | #FEF3C7     | Appel rejeté/filtré   |

# INTERACTIONS
- Clic NAV4 sous-tab → Navigation Historique/Filtrage
- Clic numéro → Appel Click2Call ou détails
- Clic Export CSV → Téléchargement fichier
- Pagination → Navigation pages

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/calls             → Historique appels
GET /telephony/{ba}/line/{sn}/calls?limit=10    → Pagination
GET /telephony/{ba}/line/{sn}/calls/filtering   → Règles filtrage
