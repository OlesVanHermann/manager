# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des numéros
4. RIGHT PANEL
   - Header: Numéro + badge statut + description
   - NAV3: [Général] | Configuration | Consommation | Statistiques
   - Bannière info "Qu'est-ce qu'un numéro"
   - Card "Informations": Numéro, Pays, Description, État, bouton modifier
   - Card "Configuration actuelle": Type, Agents, Files, liens gestion
   - Section "Types de configuration": 4 cards (Redirection, File, Conférence, SVI)
   - Section "Actions": Convertir en ligne, Résilier

# TYPES DE CONFIGURATION
| Type         | featureType    | Description                       |
|--------------|----------------|-----------------------------------|
| Redirection  | redirect       | Renvoi simple vers une ligne      |
| File d'appels| easyHunting    | Contact Center Solution           |
| Conférence   | conference     | Appels multi-participants         |
| SVI          | svi            | Serveur Vocal Interactif          |

# INTERACTIONS
- Clic NAV3 tab → Navigation onglet
- Clic "Modifier les informations →" → Modal édition description
- Clic "Gérer les agents →" → Page agents file d'appels
- Clic "Tableau de bord →" → Dashboard file d'appels
- Clic "Modifier le type →" → Modal changement type
- Clic card type → Configure ce type
- Clic "Convertir en ligne" → Modal conversion
- Clic "Résilier" → Modal confirmation résiliation

# API ENDPOINTS
GET /telephony/{ba}/number/{sn}                    → Détails numéro
PUT /telephony/{ba}/number/{sn}                    → Modifier description
GET /telephony/{ba}/number/{sn}/serviceInfos       → Infos abonnement
POST /telephony/{ba}/number/{sn}/changeFeatureType → Changer type
GET /telephony/{ba}/easyHunting/{sn}               → Config file d'appels
GET /telephony/{ba}/easyHunting/{sn}/agent         → Liste agents
DELETE /telephony/{ba}/number/{sn}                 → Résilier
