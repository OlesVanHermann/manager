INTERACTIONS

- Clic NAV3 → Change le type de service affiché (SIP/SMS/FAX/Trunk)
- Clic groupe (left panel) → Charge les détails du groupe
- Clic NAV4 tab → Navigation vers l'onglet correspondant
- Clic action rapide → Ouvre la fonctionnalité
- Clic téléphone → Détails de la ligne
- Clic facture → Télécharge le PDF

================================================================================

API ENDPOINTS

Groupes VoIP:
- GET /telephony → Liste billingAccounts
- GET /telephony/{ba} → Détails groupe
- GET /telephony/{ba}/serviceInfos → Statut facturation

Services du groupe:
- GET /telephony/{ba}/line → Liste lignes
- GET /telephony/{ba}/number → Liste numéros
- GET /telephony/{ba}/fax → Liste fax

Consommation:
- GET /telephony/{ba}/historyConsumption → Historique

Téléphones:
- GET /telephony/{ba}/line/{sn}/phone → Infos téléphone
