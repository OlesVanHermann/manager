3. LEFT PANEL - Liste des groupes VoIP avec recherche
4. RIGHT PANEL
   - Header: Nom groupe + bouton Commander
   - NAV3: [Dashboard] | Services | Répertoire | Facturation | Administration
   - Contenu:
     * Tile "Je veux..." (actions rapides)
     * Tile "Informations générales" (état, compteurs)
     * Tile "Consommation du mois" (graphique circulaire)
     * Table "Mes 5 derniers relevés"

# INTERACTIONS
- Clic groupe LEFT PANEL → Sélection et affichage détails
- Clic tab NAV3 → Navigation vers section
- Clic action rapide → Navigation vers fonctionnalité
- Clic PDF/CSV → Téléchargement document

# API ENDPOINTS
GET /telephony                          → Liste groupes
GET /telephony/{ba}                     → Détails groupe
GET /telephony/{ba}/serviceInfos        → Infos facturation
GET /telephony/{ba}/line                → Liste lignes
GET /telephony/{ba}/number              → Liste numéros
GET /telephony/{ba}/fax                 → Liste fax
GET /telephony/{ba}/historyConsumption  → Historique conso
