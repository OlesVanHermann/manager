================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic ligne → Charge les détails de la ligne
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la fonctionnalité
- Clic ✎ paramètre → Modal configuration
- Clic ▶ → Lecture du message
- Clic ↓ → Télécharger le message
- Clic × → Supprimer le message
- Clic Vider messagerie → Modal confirmation
- Clic checkbox → Sélection multiple

================================================================================

API ENDPOINTS

GET /telephony/{ba}/line → Liste lignes
GET /telephony/{ba}/voicemail/{vm} → Configuration messagerie
PUT /telephony/{ba}/voicemail/{vm} → Modifier configuration
GET /telephony/{ba}/voicemail/{vm}/directories → Liste dossiers
GET /telephony/{ba}/voicemail/{vm}/directories/{dir} → Liste messages
GET /telephony/{ba}/voicemail/{vm}/directories/{dir}/{id}/download → Télécharger
DELETE /telephony/{ba}/voicemail/{vm}/directories/{dir}/{id} → Supprimer
POST /telephony/{ba}/voicemail/{vm}/greeting → Upload annonce
