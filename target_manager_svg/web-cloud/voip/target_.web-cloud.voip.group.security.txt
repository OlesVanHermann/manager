INTERACTIONS

- Clic groupe → Charge les détails du groupe
- Clic NAV4 → Navigation vers l'onglet
- Clic Activer 2FA → Modal configuration 2FA
- Clic Gérer tokens → Page gestion tokens API
- Clic ✎ (IP) → Modal édition whitelist IP
- Clic Configurer (horaires) → Modal configuration plages horaires
- Clic Modifier (plafond) → Modal modification limite
- Clic Modifier (alerte) → Modal modification seuil alerte
- Clic Bloquer (international) → Modal confirmation blocage
- Clic Voir tout → Page journal sécurité complet

API ENDPOINTS

GET /telephony → Liste groupes facturation
GET /telephony/{ba} → Détails groupe
GET /telephony/{ba}/security → Paramètres sécurité
PUT /telephony/{ba}/security → Modifier paramètres
GET /telephony/{ba}/security/tokens → Liste tokens API
POST /telephony/{ba}/security/tokens → Créer token
DELETE /telephony/{ba}/security/tokens/{id} → Révoquer token
GET /telephony/{ba}/security/ipWhitelist → Liste IP autorisées
PUT /telephony/{ba}/security/ipWhitelist → Modifier whitelist
GET /telephony/{ba}/security/logs → Journal de sécurité
