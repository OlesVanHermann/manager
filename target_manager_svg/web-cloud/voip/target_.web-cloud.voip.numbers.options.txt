# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL - Liste des numéros du groupe
4. RIGHT PANEL
   - Header: Numéro
   - NAV3: ... | [Options]
   - Card Options générales (toggles)
   - Card Affichage numéro
   - Card Sécurité
   - Card Intégrations

# CARD OPTIONS GÉNÉRALES
Toggles activables/désactivables:
- Enregistrement des appels (actif)
- Annonce d'enregistrement (actif)
- Filtrage des appels (inactif)
- Bouton "Enregistrer"

# CARD AFFICHAGE NUMÉRO
- Numéro présenté aux appelés (modifiable)
- Options radio:
  - Afficher numéro du groupe
  - Afficher numéro de l'agent
  - Masquer le numéro (anonyme)

# CARD SÉCURITÉ
- Plafond de consommation mensuel (dropdown montant)
- Toggle alertes dépassement à 80%

# CARD INTÉGRATIONS
- CRM Salesforce: Statut connecté/non connecté
- Webhook: Configuration URL de notification

# TOGGLES
- Activé: Fond vert #059669, cercle à droite
- Désactivé: Fond gris #E5E7EB, cercle à gauche

# INTERACTIONS
- Clic toggle → Bascule état
- Clic "Modifier" numéro → Modal édition
- Clic radio → Sélection exclusive
- Clic "Enregistrer" → Sauvegarde options
- Clic "Configurer" intégration → Modal configuration

# API ENDPOINTS
GET /telephony/{ba}/easyHunting/{sn}              → Options générales
PUT /telephony/{ba}/easyHunting/{sn}              → Modifier options
GET /telephony/{ba}/number/{sn}/options           → Options numéro
PUT /telephony/{ba}/number/{sn}/options           → Modifier options numéro
GET /telephony/{ba}/serviceInfos                  → Infos facturation/plafond
