STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (VoIP actif)
3. LEFT PANEL
   - NAV3: [SIP][SMS][FAX][Trunk]
   - Recherche
   - Liste des lignes
4. RIGHT PANEL
   - Header: Numéro + description + badge status
   - NAV4: [Général][Consommation][Appels][Téléphone][Options]
   - Sous-tabs: Renvoi (actif), Filtrage, Messagerie, Click-to-Call
   - Titre: Renvoi d'appel
   - Cards renvois (6 types):
     * Renvoi inconditionnel
     * Renvoi si occupé
     * Renvoi si non réponse
     * Renvoi de secours
     * Renvoi vers messagerie
     * Renvoi horaire
   - Section paramètres rapides:
     * Présentation du numéro
     * Appel en attente
     * Intercom
     * Ne pas déranger (DND)

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic ligne → Charge les détails de la ligne
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la fonctionnalité
- Clic ✎ renvoi → Modal configuration renvoi
- Clic ✎ paramètre → Toggle ou modal configuration

================================================================================

API ENDPOINTS

GET /telephony/{ba}/line → Liste lignes
GET /telephony/{ba}/line/{sn} → Détails ligne
GET /telephony/{ba}/line/{sn}/options → Options de la ligne
PUT /telephony/{ba}/line/{sn}/options → Modifier options
GET /telephony/{ba}/line/{sn}/phone → Info téléphone
PUT /telephony/{ba}/line/{sn}/options/forwardUnconditional → Config renvoi
PUT /telephony/{ba}/line/{sn}/options/forwardBusy → Config renvoi si occupé
PUT /telephony/{ba}/line/{sn}/options/forwardNoReply → Config renvoi non réponse
PUT /telephony/{ba}/line/{sn}/options/forwardBackup → Config renvoi secours
