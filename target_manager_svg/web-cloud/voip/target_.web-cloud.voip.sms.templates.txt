4. RIGHT PANEL
   - Header: Identifiant compte + badge status
   - NAV4: [Général][Envoyer][Campagnes][Reçus][Modèles]

================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic compte → Charge les détails du compte
- Clic NAV4 → Navigation vers l'onglet
- Clic ↻ → Actualiser liste
- Clic Nouveau modèle → Modal création
- Clic Utiliser → Pré-remplit formulaire envoi SMS
- Clic Modifier → Modal édition
- Clic × → Modal confirmation suppression
- Clic card dashed → Modal création

================================================================================

API ENDPOINTS

GET /sms → Liste comptes SMS
GET /sms/{account}/templatesControl → Liste modèles
GET /sms/{account}/templatesControl/{name} → Détails modèle
POST /sms/{account}/templatesControl → Créer modèle
PUT /sms/{account}/templatesControl/{name} → Modifier modèle
DELETE /sms/{account}/templatesControl/{name} → Supprimer modèle
