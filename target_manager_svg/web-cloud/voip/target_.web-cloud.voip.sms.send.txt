================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic compte → Charge les détails du compte
- Clic NAV4 → Navigation vers l'onglet
- Clic dropdown Expéditeur → Liste senders
- Clic Importer CSV → Modal upload fichier
- Clic Coller liste → Mode paste
- Clic Modèles → Modal sélection template
- Toggle options → Active/désactive option
- Clic Envoyer → Envoi SMS + confirmation

================================================================================

API ENDPOINTS

GET /sms → Liste comptes SMS
GET /sms/{account} → Détails compte (crédits)
GET /sms/{account}/senders → Liste expéditeurs
GET /sms/{account}/templatesControl → Modèles
POST /sms/{account}/jobs → Envoyer SMS
GET /sms/{account}/outgoing → Vérifier statut
