# STRUCTURE MODAL
1. HEADER - Titre + bouton fermer
2. CONTENT
   - Sélecteur type d'agent (2 cards)
   - Dropdown sélection ligne (si interne)
   - Champ numéro (si externe)
   - Champ nom agent
   - Champ priorité (spinner)
   - Options (checkboxes)
3. FOOTER - Boutons Ajouter / Annuler

# DIMENSIONS
- Width: 550px
- Height: 480px
- Padding: 24px

# TYPES D'AGENT
| Type | Description |
|------|-------------|
| Ligne interne | Ligne SIP appartenant au groupe |
| Numéro externe | N'importe quel numéro de téléphone |

# PRIORITÉ
- 1 = plus haute priorité
- Appelé en premier si stratégie séquentielle
- Les agents de même priorité sont appelés selon la stratégie

# OPTIONS
| Option | Description |
|--------|-------------|
| Actif immédiatement | L'agent reçoit les appels dès l'ajout |
| Débordement uniquement | N'est appelé qu'en cas de timeout |

# INTERACTIONS
- Clic ✕ → Fermer modal
- Clic type agent → Change le champ ligne/numéro
- Dropdown ligne → Sélectionner ligne interne
- Spinner priorité → Incrémenter/décrémenter
- Checkbox → Activer/désactiver option
- Clic "Ajouter" → POST API + fermer

# API ENDPOINTS
GET /telephony/{ba}/line                   → Liste lignes disponibles
POST /telephony/{ba}/number/{sn}/easyHunting/hunting/agent → Ajouter agent

# PAYLOAD ATTENDU
```json
{
  "agentNumber": "0033912345678",
  "agentName": "Jean Dupont",
  "priority": 1,
  "enabled": true,
  "overflowOnly": false
}
```
