# STRUCTURE MODAL
1. HEADER - Titre + bouton fermer
2. CONTENT
   - Champ Nom (obligatoire)
   - Champ Prénom (optionnel)
   - Champ Téléphone principal (obligatoire)
   - Champ Téléphone secondaire (optionnel)
   - Champ Email (optionnel)
   - Note champs obligatoires
3. FOOTER - Boutons Ajouter / Annuler

# DIMENSIONS
- Width: 550px
- Height: 520px
- Padding: 24px

# VALIDATION
| Champ | Règle |
|-------|-------|
| Nom | Obligatoire, 2-50 caractères |
| Prénom | Optionnel, max 50 caractères |
| Téléphone principal | Obligatoire, format international |
| Téléphone secondaire | Optionnel, format international |
| Email | Optionnel, format email valide |

# INTERACTIONS
- Clic ✕ → Fermer modal
- Saisie champs → Validation en temps réel
- Clic "Ajouter" → POST API + fermer modal si succès
- Clic "Annuler" → Fermer modal sans sauvegarder
- Erreur validation → Afficher message sous le champ

# API ENDPOINTS
POST /telephony/{ba}/phonebook/{bookKey}/phonebookContact → Créer contact
