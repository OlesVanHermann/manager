# STRUCTURE MODAL
1. HEADER - Titre + bouton fermer
2. CONTENT
   - Sélecteur type d'annonce (2 cards)
   - Zone upload fichier audio (drag & drop)
   - Info formats acceptés
   - Section annonce actuelle avec lecture
3. FOOTER - Boutons Enregistrer / Annuler

# DIMENSIONS
- Width: 550px
- Height: 420px
- Padding: 24px

# TYPES D'ANNONCE
| Type | Description |
|------|-------------|
| Défaut | Message standard OVH |
| Personnalisée | Fichier audio uploadé |

# FORMATS AUDIO ACCEPTÉS
| Format | Extension |
|--------|-----------|
| WAV | .wav |
| MP3 | .mp3 |

# CONTRAINTES
- Taille max : 5 Mo
- Durée max : 2 minutes
- Encodage : PCM 16 bits recommandé

# INTERACTIONS
- Clic ✕ → Fermer modal
- Clic type annonce → Sélectionner
- Drag & drop fichier → Upload
- Clic "Parcourir" → Ouvrir file picker
- Clic "▶ Écouter" → Lecture audio
- Clic "Enregistrer" → PUT API + fermer modal
- Clic "Annuler" → Fermer modal

# API ENDPOINTS
GET /telephony/{ba}/line/{sn}/voicemail          → Config messagerie
PUT /telephony/{ba}/line/{sn}/voicemail/greeting → Upload annonce
DELETE /telephony/{ba}/line/{sn}/voicemail/greeting → Revenir au défaut
