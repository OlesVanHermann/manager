# STRUCTURE MODAL
1. HEADER - Titre + bouton fermer
2. CONTENT
   - Info numéro (bandeau)
   - Sélecteur type de configuration (3 cards)
   - Dropdown Stratégie de distribution
   - Dropdown Musique d'attente
   - Options (checkboxes)
3. FOOTER - Boutons Enregistrer / Annuler

# DIMENSIONS
- Width: 600px
- Height: 550px
- Padding: 24px

# TYPES DE CONFIGURATION
| Type | Description |
|------|-------------|
| File d'attente | Distribution des appels aux agents |
| SVI Menu | Menu vocal interactif avec choix |
| Renvoi direct | Redirection vers une ligne unique |

# STRATÉGIES DE DISTRIBUTION
| Stratégie | Description |
|-----------|-------------|
| Séquentielle | Les agents sont appelés l'un après l'autre |
| Simultanée | Tous les agents sonnent en même temps |
| Cyclique | Round-robin entre les agents |
| Aléatoire | Un agent choisi au hasard |
| Moins d'appels | L'agent ayant pris le moins d'appels |

# OPTIONS
| Option | Description |
|--------|-------------|
| Position file | Annonce la position d'attente |
| Enregistrement | Enregistre tous les appels |
| Débordement | Redirige vers messagerie si timeout |
| Filtrage horaire | Différents comportements selon l'heure |

# INTERACTIONS
- Clic ✕ → Fermer modal
- Clic type config → Sélectionner
- Dropdown stratégie → Choisir stratégie
- Clic "Changer" musique → Modal upload audio
- Checkbox → Activer/désactiver option
- Clic "Enregistrer" → PUT API + fermer

# API ENDPOINTS
GET /telephony/{ba}/number/{sn}              → Config numéro
PUT /telephony/{ba}/number/{sn}              → Modifier config
GET /telephony/{ba}/number/{sn}/easyHunting  → Config easyHunting
