# STRUCTURE MODAL
1. HEADER - Titre + bouton fermer
2. CONTENT
   - Champ Nom du modèle (obligatoire)
   - Dropdown Type (obligatoire)
   - Textarea Message avec compteur
   - Boutons variables à insérer
   - Bannière info
3. FOOTER - Boutons Créer / Annuler

# DIMENSIONS
- Width: 550px
- Height: 500px
- Padding: 24px

# TYPES DE MODÈLE
| Type | Usage |
|------|-------|
| Transactionnel | Confirmations, codes, notifications |
| Marketing | Promotions, newsletters |

# VARIABLES DISPONIBLES
| Variable | Description |
|----------|-------------|
| {NOM} | Nom du destinataire |
| {PRENOM} | Prénom du destinataire |
| {DATE} | Date formatée |
| {CODE} | Code de vérification |
| {REF} | Référence commande/colis |

# COMPTEUR SMS
| Caractères | SMS |
|------------|-----|
| 0-160 | 1 |
| 161-306 | 2 |
| 307-459 | 3 |

# INTERACTIONS
- Clic ✕ → Fermer modal
- Clic variable → Insérer dans message
- Saisie message → Compteur temps réel
- Clic "Créer" → POST API + fermer modal
- Clic "Annuler" → Fermer modal

# API ENDPOINTS
POST /sms/{sn}/templates → Créer modèle

# PAYLOAD ATTENDU
```json
{
  "name": "Confirmation commande",
  "type": "transactional",
  "content": "Bonjour {NOM}, votre commande {REF} est confirmée."
}
```
