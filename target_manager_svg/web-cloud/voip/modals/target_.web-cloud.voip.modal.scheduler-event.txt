# STRUCTURE MODAL
1. Header: "Ajouter un créneau horaire" + bouton fermer
2. Sélection jour(s)
3. Créneaux horaires
4. Actions ouverture/fermeture
5. Bannière info
6. Footer: Annuler / Enregistrer

# CHAMPS FORMULAIRE
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Jours | Multi-checkbox | Oui (min 1) | Jours concernés |
| Créneaux | Time range | Oui (min 1) | Heures début/fin |
| Action ouverture | Dropdown | Oui | Normal/Custom |
| Action fermeture | Dropdown | Oui | Voicemail/Son/Redirect |

# JOURS
| Checkbox | Label |
|----------|-------|
| ☑️ | Lundi |
| ☑️ | Mardi |
| ☑️ | Mercredi |
| ☑️ | Jeudi |
| ☑️ | Vendredi |
| ☐ | Samedi |
| ☐ | Dimanche |

# CRÉNEAUX
- Format : HH:MM
- Possibilité d'ajouter plusieurs créneaux par jour
- Exemple : 09:00-12:00 + 14:00-18:00

# ACTIONS DISPONIBLES
| Type | Description |
|------|-------------|
| Normal | Traitement normal des appels |
| Voicemail | Renvoi vers messagerie |
| Playback | Jouer un son puis raccrocher |
| Redirect | Renvoyer vers autre numéro |

# INTERACTIONS
- Checkboxes jours → Toggle multi-sélection
- Time inputs → Saisie heures
- × → Supprime un créneau
- "+" → Ajoute un créneau
- Dropdowns → Sélection actions
- "Annuler" → Ferme modal
- "Enregistrer" → POST/PUT API + ferme modal

# API ENDPOINTS
POST /telephony/{ba}/number/{sn}/schedule
PUT /telephony/{ba}/number/{sn}/schedule/{day}
