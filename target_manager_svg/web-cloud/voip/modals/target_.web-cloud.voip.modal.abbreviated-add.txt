# STRUCTURE MODAL
1. Header: "Ajouter un numéro abrégé" + bouton fermer
2. Formulaire
3. Bannière info
4. Footer: Annuler / Ajouter

# CHAMPS FORMULAIRE
| Champ | Type | Obligatoire | Validation |
|-------|------|-------------|------------|
| Numéro abrégé | Text | Oui | 3-4 chiffres, commence par 2 |
| Numéro cible | Tel | Oui | Format international (0033...) |
| Nom | Text | Non | - |
| Prénom | Text | Non | - |

# RÈGLES VALIDATION
- Numéro abrégé : regex ^2[0-9]{2,3}$
- Numéro cible : regex ^00[0-9]{10,14}$
- Unicité du numéro abrégé dans le groupe

# INTERACTIONS
- Saisie champs → Validation en temps réel
- "×" → Ferme modal
- "Annuler" → Ferme modal
- "Ajouter" → POST API + ferme modal

# API ENDPOINTS
POST /telephony/{ba}/abbreviatedNumber

# PAYLOAD
```json
{
  "abbreviatedNumber": "2001",
  "destinationNumber": "0033612345678",
  "name": "Dupont",
  "surname": "Jean"
}
```
