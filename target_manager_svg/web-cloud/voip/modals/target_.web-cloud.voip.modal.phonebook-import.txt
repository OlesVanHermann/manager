# STRUCTURE MODAL
1. Header: "Importer des contacts" + bouton fermer
2. Bannière info formats
3. Zone upload drag & drop
4. Fichier sélectionné (état)
5. Options d'import
6. Footer: Annuler / Importer

# FORMATS ACCEPTÉS
- CSV (séparateur virgule ou point-virgule)
- TXT (tabulations)
- vCard (.vcf)

# COLONNES ATTENDUES
| Colonne | Obligatoire | Format |
|---------|-------------|--------|
| Nom | Oui | Texte |
| Prénom | Oui | Texte |
| Groupe | Non | Texte |
| Tél. domicile | Non | E.164 |
| Mobile | Non | E.164 |
| Tél. bureau | Non | E.164 |
| Mobile pro | Non | E.164 |

# ZONE UPLOAD
- Drag & drop fichier
- Clic pour parcourir
- Taille max : 2 MB
- Affichage nom fichier + nb contacts détectés

# OPTIONS D'IMPORT
- Remplacer contacts existants (même nom)
- Ignorer doublons (défaut activé)

# ÉTATS
1. Vide - Zone upload visible
2. Fichier sélectionné - Preview avec count
3. Erreur - Message erreur format

# INTERACTIONS
- Drag & drop → Upload fichier
- Clic zone → Ouvrir explorateur
- "×" sur fichier → Supprimer sélection
- Checkboxes → Toggle options
- "Annuler" → Ferme modal
- "Importer" → POST API + ferme modal

# API ENDPOINTS
POST /telephony/{ba}/line/{sn}/phone/phonebook/import
