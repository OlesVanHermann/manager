# STRUCTURE MODAL
1. Header: "Ajouter un contact" + bouton fermer
2. Formulaire champs
3. Footer: Annuler / Ajouter

# CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Nom | Text | Oui |
| Prénom | Text | Oui |
| Groupe | Dropdown | Non |
| Tél. domicile | Tel | Min 1 requis |
| Mobile personnel | Tel | Min 1 requis |
| Tél. bureau | Tel | Min 1 requis |
| Mobile professionnel | Tel | Min 1 requis |
| Email | Email | Non |
| Notes | Textarea | Non |

# VALIDATION
- Nom et Prénom obligatoires
- Au moins 1 numéro de téléphone
- Format numéro E.164 recommandé

# GROUPES PRÉDÉFINIS
- Travail
- Famille
- Amis
- Autre

# INTERACTIONS
- "×" → Ferme modal sans sauvegarder
- "Annuler" → Ferme modal sans sauvegarder
- "Ajouter" → Validation + POST API + ferme modal

# API ENDPOINTS
POST /telephony/{ba}/line/{sn}/phone/phonebook/contacts
