# STRUCTURE MODAL
1. HEADER - Titre + bouton fermer
2. CONTENT
   - Champ numéro destinataire (obligatoire)
   - Zone upload document PDF (drag & drop)
   - Info formats acceptés
   - Sélecteur qualité d'envoi
   - Checkbox page de garde
   - Champ objet (si page de garde)
3. FOOTER - Boutons Envoyer / Annuler

# DIMENSIONS
- Width: 600px
- Height: 550px
- Padding: 24px

# QUALITÉS D'ENVOI
| Qualité | Description |
|---------|-------------|
| Standard | 200 dpi, envoi rapide |
| Haute définition | 300+ dpi, meilleure qualité |

# CONTRAINTES DOCUMENT
- Format : PDF uniquement
- Taille max : 10 Mo
- Pages max : 50 pages

# INTERACTIONS
- Clic ✕ → Fermer modal
- Drag & drop → Upload document
- Clic "Parcourir" → File picker
- Clic qualité → Sélectionner
- Checkbox page garde → Affiche/cache objet
- Clic "Envoyer" → POST API + fermer

# API ENDPOINTS
POST /telephony/{ba}/fax/{sn}/jobs → Envoyer fax

# PAYLOAD ATTENDU
```json
{
  "destinationNumber": "+33123456789",
  "document": "<base64_pdf>",
  "quality": "standard",
  "coverPage": true,
  "subject": "Objet du fax"
}
```
