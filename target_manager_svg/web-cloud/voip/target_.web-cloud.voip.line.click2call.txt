================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic ligne → Charge les détails de la ligne
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la fonctionnalité
- Clic Appeler → Lance l'appel via API
- Clic Ajouter → Modal ajout utilisateur
- Clic Modifier → Modal édition utilisateur
- Clic Supprimer → Modal confirmation suppression

================================================================================

API ENDPOINTS

GET /telephony/{ba}/line → Liste lignes
GET /telephony/{ba}/line/{sn} → Détails ligne
GET /telephony/{ba}/line/{sn}/click2CallUser → Liste utilisateurs autorisés
POST /telephony/{ba}/line/{sn}/click2CallUser → Ajouter utilisateur
DELETE /telephony/{ba}/line/{sn}/click2CallUser/{id} → Supprimer utilisateur
POST /telephony/{ba}/line/{sn}/click2Call → Lancer un appel
