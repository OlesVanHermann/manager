INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic groupe → Charge les détails du groupe
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la section
- Clic Ajouter → Modal ajout numéro abrégé
- Clic Import CSV → Modal import
- Clic Export CSV → Téléchargement CSV
- Clic ✎ → Modal édition numéro abrégé
- Clic × → Modal confirmation suppression

================================================================================

API ENDPOINTS

GET /telephony → Liste billingAccounts
GET /telephony/{ba} → Détails groupe
GET /telephony/{ba}/abbreviatedNumber → Liste numéros abrégés
GET /telephony/{ba}/abbreviatedNumber/{abbr} → Détail numéro abrégé
POST /telephony/{ba}/abbreviatedNumber → Créer numéro abrégé
PUT /telephony/{ba}/abbreviatedNumber/{abbr} → Modifier numéro abrégé
DELETE /telephony/{ba}/abbreviatedNumber/{abbr} → Supprimer numéro abrégé
