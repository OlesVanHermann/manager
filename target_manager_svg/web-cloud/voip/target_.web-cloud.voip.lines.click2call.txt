# FONCTIONNALITÉ CLICK2CALL
Le Click to Call permet de lancer un appel depuis l'interface web:
1. L'utilisateur entre le numéro à appeler
2. Clique sur "Appeler"
3. Le téléphone physique associé à la ligne sonne
4. L'utilisateur décroche
5. L'appel est lancé vers le destinataire

# CONTACTS FAVORIS
- Liste de contacts fréquents
- Avatar avec initiales
- Nom + numéro
- Bouton appel rapide (vert)
- Possibilité d'ajouter des favoris

# HISTORIQUE
- Date/heure de l'appel
- Numéro appelé
- Durée
- Statut (Abouti/Échoué)
- Action rappeler

# INTERACTIONS
- Saisie numéro dans input
- Clic "Appeler" → POST click2call
- Clic "Rappeler" → Pré-remplit l'input
- Clic contact favori → Lance l'appel
- Clic "Ajouter favori" → Modal ajout

# API ENDPOINTS
POST /telephony/{ba}/line/{sn}/click2CallUser          → Lancer appel
GET  /telephony/{ba}/line/{sn}/click2CallUser          → Historique
GET  /telephony/{ba}/phonebook                         → Annuaire
POST /telephony/{ba}/phonebook                         → Ajouter contact
