STRUCTURE PAGE
1. Left panel
   - Liste groupes VoIP du client
   - Badge statut par groupe
   - Compteurs services par groupe
2. Right panel header
   - Nom groupe + billingAccount
3. NAV3: Dashboard (actif) | Services | Repertoire | Facturation | Administration
4. Dashboard content
   - 4 cards stats (Lignes, Numeros, Fax, Credit)
   - Actions rapides (3 boutons)
   - Activite recente (table)
   - Consommation du mois (2 cards)
