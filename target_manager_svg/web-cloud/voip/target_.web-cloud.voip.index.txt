4. PANEL - Table des comptes de facturation
   - Barre recherche
   - Header: Compte | Description | Nb services | Hors forfait | Caution | Statut | Actions

# INTERACTIONS
- Input recherche → Filtre comptes

- ... actions → Menu (Voir, Modifier, Résilier)
- + Commander → Modal/Page commande nouveau groupe
- Pagination → Navigation pages
