INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic ligne → Charge les détails de la ligne
- Clic NAV4 → Navigation vers l'onglet
- Clic ✎ description → Modal édition description
- Clic Configurer téléphone → Page configuration téléphone
- Clic action rapide → Navigation vers la fonctionnalité
