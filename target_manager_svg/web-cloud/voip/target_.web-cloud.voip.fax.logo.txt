================================================================================

INTERACTIONS

- Clic NAV3 → Change le type de service (SIP/SMS/FAX/Trunk)
- Clic fax → Charge les détails du fax
- Clic NAV4 → Navigation vers l'onglet
- Clic sous-tab → Navigation vers la fonctionnalité
- Clic Changer le logo → Modal upload fichier
- Clic Supprimer → Modal confirmation suppression
- Clic Enregistrer → Sauvegarde infos d'en-tête
- Modification champs → Mise à jour aperçu temps réel

================================================================================

API ENDPOINTS

GET /telephony/{ba}/fax → Liste services fax
GET /telephony/{ba}/fax/{sn} → Détails fax avec logo
POST /telephony/{ba}/fax/{sn}/logo/upload → Upload nouveau logo
DELETE /telephony/{ba}/fax/{sn}/logo → Supprimer logo
PUT /telephony/{ba}/fax/{sn}/header → Modifier informations en-tête
GET /telephony/{ba}/fax/{sn}/header/preview → Aperçu en-tête
