# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer un compte partagé

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de compte partagé
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer une boîte mail partagée

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Compte concerné
- Avertissement
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x180)
- **OdsText** : Question, email, avertissement
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer le compte partagé"

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer ce compte partagé ?" |
| Compte | partage@example.com (gras) |
| Avertissement | "Cette action est irréversible." |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le compte |

## 7. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/sharedAccount/{email} : Supprimer compte partagé
