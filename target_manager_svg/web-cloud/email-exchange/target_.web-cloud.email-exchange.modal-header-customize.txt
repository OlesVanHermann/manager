# Cahier des Charges : web-cloud/email-exchange - Modal En-têtes personnalisés

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration des en-têtes X-Custom
- **Type** : Formulaire de paramètres
- **Objectif** : Ajouter des en-têtes personnalisés aux emails

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Bandeau explicatif
- Liste des en-têtes configurés
- Formulaire d'ajout
- Avertissement
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x360)
- **OdsMessage** : Info bleu, warning jaune
- **OdsInput** : Nom d'en-tête, valeur
- **OdsButton** : Ajouter (+), Supprimer (×), Annuler, Enregistrer

## 4. TITRE
"En-têtes personnalisés"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Ajoutez des en-têtes X-Custom aux emails sortants."

## 6. EN-TÊTES CONFIGURÉS
| Nom | Valeur | Action |
|-----|--------|--------|
| X-Company-ID | ACME-123 | × (supprimer) |

## 7. FORMULAIRE D'AJOUT
| Champ | Type | Placeholder |
|-------|------|-------------|
| Nom de l'en-tête | Input text | X-Header-Name |
| Valeur | Input text | Valeur |
| Ajouter | Bouton + | - |

## 8. AVERTISSEMENT
Fond jaune (#FEF3C7) :
"Les en-têtes doivent commencer par X- pour être acceptés."

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde les en-têtes |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/customHeader : Lister en-têtes
- POST /email/exchange/{org}/service/{service}/customHeader : Ajouter en-tête
- DELETE /email/exchange/{org}/service/{service}/customHeader/{id} : Supprimer en-tête
