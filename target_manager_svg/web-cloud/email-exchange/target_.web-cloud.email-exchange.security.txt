# Cahier des Charges : web-cloud/email-exchange - Sécurité

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration de la sécurité Exchange
- **Type** : Page formulaire avec sections
- **Objectif** : Configurer MFA, verrouillage de compte, politique de mot de passe

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Exchange (menu)
- Titre "Sécurité"
- Formulaire en sections
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Exchange
- **OdsCard** : Conteneur formulaire
- **OdsToggle** : Options Oui/Non
- **OdsInput** : Champs numériques avec stepper
- **OdsCheckbox** : Options binaires
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Sécurité"
