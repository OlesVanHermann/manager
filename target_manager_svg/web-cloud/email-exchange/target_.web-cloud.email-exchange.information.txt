# Cahier des Charges : web-cloud/email-exchange - Informations générales

## 1. VUE D'ENSEMBLE
- **Écran** : Dashboard informations générales Exchange
- **Type** : Page tableau de bord avec tiles
- **Objectif** : Vue d'ensemble du service Exchange et statistiques

## 2. LAYOUT
- Header OVHcloud avec breadcrumb
- Sidebar navigation Exchange (Informations générales actif)
- Zone principale avec tiles d'information

## 3. COMPOSANTS UI
- **OdsCard** : Tiles Résumé, Statistiques, Connexion, Sauvegarde
- **OdsBadge** : Diagnostics DNS (A, AAAA, PTR, PTRv6)
- **OdsProgressBar** : Espace disque utilisé
- **OdsActionMenu** : Menu contextuel (⋮)
- **OdsLink** : Liens externes (webmail, guides)
