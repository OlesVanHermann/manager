# Cahier des Charges : web-cloud/email-exchange - Connecteurs

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des connecteurs Exchange
- **Type** : Page liste avec tableau
- **Objectif** : Configurer les connecteurs de messagerie

## 2. LAYOUT
- Header OVHcloud
- Sidebar Exchange (Connecteurs actif)
- Titre
- Bandeau info explicatif
- Bouton nouveau connecteur
- Tableau des connecteurs

## 3. COMPOSANTS UI
- **OdsMessage** : Bandeau info bleu
- **OdsButton** : Nouveau connecteur (primaire)
- **OdsTable** : Liste des connecteurs
- **OdsBadge** : Direction (Sortant), État (Actif)
- **OdsActionMenu** : Menu contextuel (⋮)

## 4. TITRE
"Connecteurs"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les connecteurs permettent d'acheminer les emails vers ou depuis des serveurs externes."

## 6. BOUTON AJOUTER
| Élément | Valeur |
|---------|--------|
| Label | "+ Nouveau connecteur" |
| Type | Primaire bleu |

## 7. TABLEAU
| Colonne | Description |
|---------|-------------|
| Nom | Nom du connecteur |
| Direction | Badge (Sortant/Entrant) |
| Serveur | Adresse et port SMTP |
| État | Badge (Actif/Inactif) |
| Actions | Menu ⋮ |

## 8. BADGES
| Badge | Couleur fond | Couleur texte |
|-------|--------------|---------------|
| Sortant | Bleu (#EFF6FF) | Bleu (#1E40AF) |
| Entrant | Bleu (#EFF6FF) | Bleu (#1E40AF) |
| Actif | Vert (#DCFCE7) | Vert (#166534) |
| Inactif | Gris (#F3F4F6) | Gris (#6B7280) |

## 9. EXEMPLES DE DONNÉES
| Nom | Direction | Serveur | État |
|-----|-----------|---------|------|
| Partenaire CRM | Sortant | smtp.crm.com:25 | Actif |
| Archivage légal | Sortant | smtp.archive.fr:587 | Actif |

## 10. ACTION MENU
| Option | Couleur | Action |
|--------|---------|--------|
| Modifier | Normal | Ouvre modal modification |
| Tester la connexion | Normal | Lance test SMTP |
| Supprimer | Rouge (#DC2626) | Supprime le connecteur |

## 11. PAGINATION
"2 connecteurs"

## 12. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/connector : Lister connecteurs
- POST /email/exchange/{org}/service/{service}/connector : Créer connecteur
- PUT /email/exchange/{org}/service/{service}/connector/{id} : Modifier connecteur
- DELETE /email/exchange/{org}/service/{service}/connector/{id} : Supprimer connecteur
