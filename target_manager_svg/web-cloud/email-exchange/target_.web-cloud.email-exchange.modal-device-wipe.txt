# Cahier des Charges : web-cloud/email-exchange - Modal Effacer l'appareil

## 1. VUE D'ENSEMBLE
- **Modal** : Effacement à distance d'un appareil
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer les données Exchange d'un appareil mobile

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Avertissement avec icône
- Informations appareil
- Boutons Annuler / Effacer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (400x240)
- **OdsMessage** : Avertissement rouge
- **OdsIcon** : Point d'exclamation
- **OdsButton** : Annuler (secondaire), Effacer (danger rouge)

## 4. TITRE
"Effacer l'appareil à distance"
- Header fond rouge (#FEF2F2)
- Texte rouge (#DC2626)

## 5. AVERTISSEMENT
Fond rouge (#FEF2F2) avec icône :
| Élément | Valeur |
|---------|--------|
| Icône | ! (cercle rouge) |
| Titre | "Action irréversible" |
| Message | "Toutes les données Exchange seront supprimées de l'appareil." |

## 6. INFORMATIONS APPAREIL
| Champ | Exemple |
|-------|---------|
| Appareil | iPhone 15 Pro - John |
| Dernière synchronisation | il y a 2 heures |

## 7. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Effacer | Danger | Rouge (#DC2626) | Lance l'effacement |

## 8. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/device/{deviceId}/wipe : Effacer appareil
