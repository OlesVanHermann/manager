# Cahier des Charges : web-cloud/email-exchange - Modal Configuration DKIM

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration automatique DKIM
- **Type** : Modal informative avec action
- **Objectif** : Activer la signature DKIM sur un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine concerné
- Bandeau explicatif
- Sélecteur DKIM
- Statut
- Avertissement
- Boutons Annuler / Activer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x390)
- **OdsMessage** : Info bleu, warning jaune
- **OdsInput** : Sélecteur DKIM (readonly)
- **OdsBadge** : Statut
- **OdsButton** : Annuler, Activer

## 4. TITRE
"Configuration DKIM"

## 5. DOMAINE
Affiché en gras : "example.com"

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) :
"DKIM (DomainKeys Identified Mail) permet de signer cryptographiquement vos emails sortants."

## 7. SÉLECTEUR DKIM
| Champ | Valeur exemple |
|-------|----------------|
| Label | "Sélecteur DKIM" |
| Valeur | ovhex1234-selector1 |

## 8. BADGE STATUT
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Désactivé | #FEE2E2 (rouge clair) | #DC2626 |
| Activé | #DCFCE7 (vert clair) | #166534 |

## 9. AVERTISSEMENT
Fond jaune (#FEF3C7) :
"L'activation de DKIM peut prendre jusqu'à 48h pour être effective."

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Activer | Primaire bleu | Active DKIM |

## 11. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/domain/{domain}/dkim : Activer DKIM
