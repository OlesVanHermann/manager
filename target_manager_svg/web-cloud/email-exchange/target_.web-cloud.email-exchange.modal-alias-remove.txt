# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer un alias

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'alias
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer un alias d'un compte Exchange

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Alias à supprimer
- Compte associé
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x180)
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer l'alias"
- Header fond rouge (#FEF2F2)
- Texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer cet alias ?" |
| Alias | jdoe@example.com (gras) |
| Compte associé | "du compte john.doe@example.com" |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime l'alias |

## 7. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/account/{email}/alias/{alias} : Supprimer alias
