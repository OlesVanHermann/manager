# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Wordpress actif)
3. LEFT PANEL
   - Recherche
   - Liste services
4. RIGHT PANEL
   - Header: serviceName
   - NAV4: [Mes sites] [Taches] (Taches actif)
   - Toolbar: Refresh + Search
   - Table: Historique taches

# NAV4 TABS
| Tab       | Description                    |
|-----------|--------------------------------|
| Mes sites | Liste des sites WordPress      |
| Taches    | Historique operations (ACTIF)  |

# COLONNES TABLE
| Colonne | Description                    |
|---------|--------------------------------|
| ID      | Identifiant tache              |
| TYPE    | Creation, Import, Backup, SSL  |
| FQDN    | Site concerne                  |
| STATUT  | DONE, PENDING, ERROR           |
| DATE    | Date/heure operation           |

# STATUTS TACHES
| Statut  | Couleur | Description            |
|---------|---------|------------------------|
| DONE    | Vert    | Tache terminee         |
| PENDING | Jaune   | En cours               |
| ERROR   | Rouge   | Echec                  |

# INTERACTIONS
- Clic tab Mes sites → Retour liste sites
- Clic refresh → Recharge liste taches
- Recherche → Filtre taches

# API ENDPOINTS
GET /hosting/wordpress/{serviceName}/task - Liste taches
GET /hosting/wordpress/{serviceName}/task/{id} - Details tache
