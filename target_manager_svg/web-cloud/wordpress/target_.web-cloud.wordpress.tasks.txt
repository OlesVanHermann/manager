STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif x=510)
2. NAV2 - Web Cloud (Wordpress actif x=345)
3. LEFT PANEL - Liste des sites WordPress
4. RIGHT PANEL
   - Header: Nom site + version + offre
   - NAV3: 6 tabs (Taches actif)
   - Contenu Taches:
     * Header "Taches" + description + bouton "Rafraichir"
     * Table taches:
       - Colonne DATE
       - Colonne TYPE (Sauvegarde, MAJ plugin, SSL, etc.)
       - Colonne DESCRIPTION
       - Colonne STATUT (badge)
     * Pagination

TYPES DE TACHES
- Sauvegarde (automatique ou manuelle)
- Mise a jour plugin
- Mise a jour theme
- Mise a jour WordPress
- Configuration SSL
- Restauration
- Installation
- Import

STATUTS
- En cours: badge jaune (#FEF3C7), texte orange
- Termine: badge vert (#DEF7EC), texte vert
- Echec: badge rouge (#FEE2E2), texte rouge

INTERACTIONS
- Clic "Rafraichir" → GET /tasks (refresh liste)
- Pagination → Navigation pages

API ENDPOINTS
GET /managedCMS/resource/{sn}/tasks       - Liste taches
GET /managedCMS/resource/{sn}/tasks/{id}  - Details tache
