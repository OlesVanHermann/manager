STRUCTURE PAGE
1. NAV4 sous-tabs: [Themes] | Plugins

2. Header "THEMES INSTALLES (N)"

3. Grille de cards themes (3 colonnes)
   Chaque card:
   - Apercu visuel (placeholder)
   - Nom du theme
   - Description courte
   - Version
   - Badge etat (Actif / MAJ dispo)
   - Boutons action

ETATS THEME
| Etat       | Badge     | Actions disponibles        |
|------------|-----------|----------------------------|
| Actif      | Vert      | Personnaliser              |
| Inactif    | -         | Activer                    |
| MAJ dispo  | Orange    | Mettre a jour, Activer     |

API ENDPOINTS
GET    /webWordpress/{serviceName}/theme        -> Liste themes
POST   /webWordpress/{serviceName}/theme/{name}/activate -> Activer
POST   /webWordpress/{serviceName}/theme/{name}/update   -> Mettre a jour

ACTIONS
- Click "Themes" -> Route /extensions (NAV4)
- Click "Plugins" -> Route /extensions-plugins (NAV4)
- Click "Personnaliser" -> Ouvre customizer WP (externe)
- Click "Activer" -> POST activate
- Click "Mettre a jour" -> POST update
