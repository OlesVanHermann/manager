# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 500x520
- Header: Titre + bouton fermer
- Banniere info (bleue): Description import
- Champs:
  - URL site source (input, required)
  - Hote FTP (input, required)
  - Port (input, default 21)
  - Utilisateur FTP (input, required)
  - Mot de passe (input password, required)
  - Chemin WordPress (input, default /www)
- Footer: Annuler + Importer

# CHAMPS FORMULAIRE
| Champ           | Type     | Validation       |
|-----------------|----------|------------------|
| URL source      | Input    | Required, URL    |
| Hote FTP        | Input    | Required         |
| Port            | Input    | Default: 21      |
| Utilisateur FTP | Input    | Required         |
| Mot de passe    | Password | Required         |
| Chemin WP       | Input    | Default: /www    |

# INTERACTIONS
- Clic Annuler → Ferme modal
- Clic Importer → POST API import site
- Clic X → Ferme modal

# API ENDPOINTS
POST /hosting/wordpress/{serviceName}/website/import
Payload: { sourceUrl, ftpHost, ftpPort, ftpUser, ftpPassword, wpPath }
