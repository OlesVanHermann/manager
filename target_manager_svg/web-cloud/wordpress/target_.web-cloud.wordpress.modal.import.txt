STRUCTURE MODAL
1. HEADER - Titre "Importer un site WordPress existant" + bouton fermer (x)
2. FORMULAIRE:
   - URL du site actuel: Input text
   - Section "Acces FTP/SFTP":
     * Hote + Port
     * Utilisateur + Mot de passe
   - Section "Acces base de donnees":
     * Hote + Port
     * Utilisateur + Mot de passe
     * Nom de la base
   - Offre destination: Dropdown
   - Datacenter: Dropdown
3. BANNIERE WARNING - Avertissement duree import
4. FOOTER - Boutons Annuler + Lancer l'import
