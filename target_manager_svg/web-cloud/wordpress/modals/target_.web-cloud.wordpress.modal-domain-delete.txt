DESCRIPTION
Modal de confirmation pour supprimer un domaine alias du site WordPress.
Le domaine principal ne peut pas être supprimé.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                          |
|--------------------------|-----------------------------------------------------|
| monblog.com              | GET /managedCMS/resource/{sn}/domains → domain      |
| Alias                    | GET /managedCMS/resource/{sn}/domains → type        |
| mon-blog.fr              | GET /managedCMS/resource/{sn}/domains → redirectTo  |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière error: Avertissement
3. Domaine concerné: Nom + type + redirection
4. Informations: Conséquences de la suppression
5. Footer: Annuler | Supprimer

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Supprimer" → DELETE /managedCMS/resource/{sn}/domains/{domain}

================================================================================

API ENDPOINTS
DELETE /managedCMS/resource/{serviceName}/domains/{domain}
  Response: { taskId, status: "pending" }
