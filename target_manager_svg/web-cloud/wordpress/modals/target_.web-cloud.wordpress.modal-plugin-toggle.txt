DESCRIPTION
Modal de confirmation pour activer ou désactiver un plugin WordPress.
Affiche un avertissement sur l'impact potentiel sur le site.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                          |
|--------------------------|-----------------------------------------------------|
| WooCommerce              | GET /managedCMS/resource/{sn}/plugins → name        |
| E-commerce               | GET /managedCMS/resource/{sn}/plugins → category    |
| 8.4.0                    | GET /managedCMS/resource/{sn}/plugins → version     |
| Actif                    | GET /managedCMS/resource/{sn}/plugins → active      |

================================================================================

STRUCTURE MODAL
1. Header: Titre dynamique (Activer/Désactiver)
2. Bannière warning: Impact potentiel
3. Plugin concerné: Icône + nom + catégorie + version + statut
4. Footer: Annuler | Activer/Désactiver

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Désactiver" → POST /managedCMS/resource/{sn}/plugins/{name}/deactivate
- Clic "Activer" → POST /managedCMS/resource/{sn}/plugins/{name}/activate

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/plugins/{pluginName}/activate
POST /managedCMS/resource/{serviceName}/plugins/{pluginName}/deactivate
  Response: { taskId, status: "pending" }
