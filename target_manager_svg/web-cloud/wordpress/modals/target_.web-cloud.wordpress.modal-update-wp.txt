DESCRIPTION
Modal de mise à jour de WordPress vers la dernière version stable.
Propose de créer une sauvegarde avant la mise à jour.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                    |
|--------------------------|-----------------------------------------------|
| mon-blog.fr              | GET /managedCMS/resource/{sn} → displayName   |
| 6.3.1                    | GET /managedCMS/resource/{sn} → wordpressVersion |
| 6.4.2                    | GET /managedCMS/resource/{sn} → availableVersion |
| PHP 8.2                  | GET /managedCMS/resource/{sn} → phpVersion    |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière warning: Recommandation sauvegarde
3. Site concerné: Nom + version actuelle
4. Versions: Actuelle vs nouvelle
5. Checkbox: Option sauvegarde
6. Bannière info: Durée indisponibilité
7. Footer: Annuler | Mettre à jour

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Toggle checkbox → Active/désactive sauvegarde préalable
- Clic "Mettre à jour" → POST /managedCMS/resource/{sn}/update

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/update
  Body: { createBackup: true }
  Response: { taskId, status: "pending" }
