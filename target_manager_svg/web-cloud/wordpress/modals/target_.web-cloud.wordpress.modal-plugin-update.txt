DESCRIPTION
Modal de confirmation pour mettre à jour un plugin WordPress.
Affiche les informations du plugin et les versions disponibles.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                          |
|--------------------------|-----------------------------------------------------|
| Yoast SEO                | GET /managedCMS/resource/{sn}/plugins → name        |
| Actif                    | GET /managedCMS/resource/{sn}/plugins → active      |
| 21.6                     | GET /managedCMS/resource/{sn}/plugins → version     |
| 21.7                     | GET /managedCMS/resource/{sn}/plugins → newVersion  |
| SEO                      | GET /managedCMS/resource/{sn}/plugins → category    |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière info: Sauvegarde automatique
3. Plugin concerné: Icône + nom + catégorie + statut
4. Versions: Actuelle vs nouvelle
5. Footer: Annuler | Mettre à jour

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Mettre à jour" → POST /managedCMS/resource/{sn}/plugins/{name}/update

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/plugins/{pluginName}/update
  Response: { taskId, status: "pending" }
