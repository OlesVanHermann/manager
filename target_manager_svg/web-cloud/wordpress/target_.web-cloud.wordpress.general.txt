# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Wordpress actif)
3. LEFT PANEL
   - Recherche
   - Liste services
4. RIGHT PANEL
   - Header: serviceName
   - NAV4: [Mes sites] [Taches]
   - Description
   - Actions: Creer, Importer, Gerer
   - Table: Liste sites

# NAV4 TABS
| Tab       | Description                    |
|-----------|--------------------------------|
| Mes sites | Liste des sites WordPress      |
| Taches    | Historique operations          |

# COLONNES TABLE
| Colonne | Description                    |
|---------|--------------------------------|
| FQDN    | Nom de domaine du site         |
| STATUT  | READY, ERROR, CREATING         |
| ACTIONS | Menu contextuel                |

# ACTIONS BOUTONS
| Bouton        | Action                       |
|---------------|------------------------------|
| + Creer       | Modal creation site          |
| Importer      | Modal import site existant   |
| Gerer mes sites | Lien externe gestionnaire  |

# INTERACTIONS
- Clic tab → Change onglet actif
- Clic Creer → Modal modal.create
- Clic Importer → Modal modal.import
- Clic menu actions → Dropdown (SSL, CDN, Backup, Delete)

# API ENDPOINTS
GET /hosting/wordpress/{serviceName} - Info service
GET /hosting/wordpress/{serviceName}/website - Liste sites
POST /hosting/wordpress/{serviceName}/website - Creer site
DELETE /hosting/wordpress/{serviceName}/website/{id} - Supprimer
