STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif x=510)
2. NAV2 - Web Cloud (Wordpress actif x=345)
3. LEFT PANEL - Liste des sites WordPress
   - Recherche
   - Compteur sites
   - Items sites (site selectionne en bleu)
   - Bouton "Nouveau site"
4. RIGHT PANEL
   - Header: Nom site + version + offre
   - Quick actions header (Admin WP, Visiter)
   - NAV3: 6 tabs (General actif)
   - Contenu General:
     * Actions rapides (4 boutons inline)
     * Bloc WORDPRESS (infos site WP)
     * Bloc TECHNIQUE (infos serveur/offre)
     * Zone Danger (suppression)

INTERACTIONS
- Clic "Admin WP" → window.open(adminUrl)
- Clic "Visiter le site" → window.open(url)
- Clic "Vider le cache" → POST flushCache
- Clic "Reinitialiser mot de passe" → Modal reset password
- Clic "Mettre a jour WP" → POST update (si MAJ dispo)
- Clic "Supprimer" → Modal confirmation suppression

API ENDPOINTS
GET    /managedCMS/resource/{serviceName}           - Details site
GET    /managedCMS/resource/{serviceName}/serviceInfos - Infos abonnement
POST   /managedCMS/resource/{sn}/flushCache         - Vider cache
POST   /managedCMS/resource/{sn}/resetAdminPassword - Reset MDP
POST   /managedCMS/resource/{sn}/update             - MAJ WordPress
DELETE /managedCMS/resource/{serviceName}           - Supprimer
