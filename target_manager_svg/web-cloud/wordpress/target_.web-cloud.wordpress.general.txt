# INTERACTIONS
- Clic tab NAV4 → Navigation vers tab correspondant
- Clic Acceder admin → Lien externe /wp-admin
- Clic Visiter site → Lien externe URL site
- Clic Vider cache → POST /managedCMS/resource/{id}/flushCache
- Clic Reset mot de passe → POST /managedCMS/resource/{id}/resetAdminPassword
- Clic Supprimer → Modal DeleteWebsite

# API ENDPOINTS
GET /managedCMS/resource - Liste resources
GET /managedCMS/resource/{id} - Details resource
GET /managedCMS/resource/{id}/website - Liste websites
GET /managedCMS/resource/{id}/website/{wid} - Details website
POST /managedCMS/resource/{id}/flushCache - Vider cache
POST /managedCMS/resource/{id}/resetAdminPassword - Reset MDP
POST /managedCMS/resource/{id}/update - MAJ WordPress
DELETE /managedCMS/resource/{id}/website/{wid} - Supprimer
