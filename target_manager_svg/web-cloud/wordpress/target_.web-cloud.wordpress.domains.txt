STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif x=510)
2. NAV2 - Web Cloud (Wordpress actif x=345)
3. LEFT PANEL - Liste des sites WordPress
4. RIGHT PANEL
   - Header: Nom site + version + offre
   - NAV3: 6 tabs (Domaines actif)
   - Contenu Domaines:
     * Header "Domaines" + bouton "Ajouter domaine"
     * Table des domaines:
       - Colonne DOMAINE (nom + redirect info)
       - Colonne TYPE (Principal / Alias / Alias auto)
       - Colonne SSL (statut certificat)
       - Colonne STATUT DNS (OK / Erreur)
       - Colonne ACTIONS (SSL config, supprimer)
     * Box "Configuration DNS requise"

BADGES TYPE DOMAINE
- Principal: fond bleu (#DBEAFE), texte bleu
- Alias: fond gris (#F3F4F6), texte gris
- Alias auto: fond gris (#F3F4F6), texte gris (www.*)

ETATS SSL
- Let's Encrypt: cercle vert + texte
- Inclus: cercle vert + texte muted (pour www auto)
- Non configure: cercle rouge + texte erreur

ETATS DNS
- OK: cercle vert
- Erreur DNS: cercle rouge + message warning jaune

INTERACTIONS
- Clic "Ajouter domaine" → Modal ajout domaine
- Clic "SSL" → Modal configuration SSL
- Clic "x" → Confirmation suppression alias
- Clic "Copier configuration" → Copie DNS dans clipboard

API ENDPOINTS
GET    /managedCMS/resource/{sn}/domains           - Liste domaines
POST   /managedCMS/resource/{sn}/domains           - Ajouter domaine
DELETE /managedCMS/resource/{sn}/domains/{domain}  - Supprimer alias
GET    /managedCMS/resource/{sn}/domains/{d}/ssl   - Etat SSL
POST   /managedCMS/resource/{sn}/domains/{d}/ssl   - Activer SSL
