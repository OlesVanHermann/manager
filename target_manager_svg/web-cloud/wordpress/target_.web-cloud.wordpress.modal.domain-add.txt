# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 500x440
- Header: Titre + bouton fermer
- Banniere info (bleue): Description association domaine
- Champs:
  - Nom de domaine (input, required)
  - Certificat SSL (radio buttons)
    - Let's Encrypt (gratuit) - default
    - Certificat personnalise
- Footer: Annuler + Ajouter

# CHAMPS FORMULAIRE
| Champ              | Type   | Validation       |
|--------------------|--------|------------------|
| Nom de domaine     | Input  | Required, FQDN   |
| Certificat SSL     | Radio  | Required         |

# OPTIONS SSL
| Option               | Description                    |
|----------------------|--------------------------------|
| Let's Encrypt        | Certificat gratuit automatique |
| Personnalise         | Import certificat custom       |

# INTERACTIONS
- Clic radio → Change selection SSL
- Clic Annuler → Ferme modal
- Clic Ajouter → POST API add domain
- Clic X → Ferme modal

# API ENDPOINTS
POST /hosting/wordpress/{serviceName}/website/{id}/domain
Payload: { domain, sslType }
