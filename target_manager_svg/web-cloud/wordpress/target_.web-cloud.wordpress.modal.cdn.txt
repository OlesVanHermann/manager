# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 500x420
- Header: Titre + bouton fermer
- Banniere info (bleue): Description CDN
- Checkbox: Activer le CDN
- Section Options de cache:
  - Mise en cache des images (checked)
  - Mise en cache CSS/JS (checked)
  - Mise en cache des pages (unchecked, avance)
- Footer: Annuler + Enregistrer

# OPTIONS CDN
| Option              | Default | Description                    |
|---------------------|---------|--------------------------------|
| Activer CDN         | true    | Active/desactive CDN           |
| Cache images        | true    | Cache images statiques         |
| Cache CSS/JS        | true    | Cache fichiers CSS et JS       |
| Cache pages         | false   | Cache pages HTML (avance)      |

# INTERACTIONS
- Toggle checkbox → Change valeur option
- Clic Annuler → Ferme modal
- Clic Enregistrer → PUT API update CDN config
- Clic X → Ferme modal

# API ENDPOINTS
GET /hosting/wordpress/{serviceName}/website/{id}/cdn - Get CDN config
PUT /hosting/wordpress/{serviceName}/website/{id}/cdn - Update CDN config
Payload: { enabled, cacheImages, cacheCssJs, cachePages }
