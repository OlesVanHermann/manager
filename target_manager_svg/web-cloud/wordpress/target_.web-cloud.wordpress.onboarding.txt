CONDITION D'AFFICHAGE
Cette page s'affiche uniquement si:
- GET /webWordpress retourne une liste vide
- L'utilisateur n'a aucun site WordPress

STRUCTURE PAGE
1. Header centre
   - Titre "Bienvenue sur WordPress Manage"
   - Sous-titre explicatif

2. Deux cards cote a cote:
   - Card "Creer un nouveau site" (gauche)
     - Icone +
     - Description
     - Liste features
     - Bouton primaire "Creer un site"

   - Card "Importer un site existant" (droite)
     - Icone refresh
     - Description
     - Liste features
     - Bouton outline "Importer un site"

3. Banniere info en bas
   - Lien vers guides
   - Lien vers support

ACTIONS
- Click "Creer un site" -> Ouvre modal-create
- Click "Importer un site" -> Ouvre modal-import
- Click "Guides" -> Navigation vers /guides

API ENDPOINTS
GET /webWordpress -> Liste vide = afficher onboarding
