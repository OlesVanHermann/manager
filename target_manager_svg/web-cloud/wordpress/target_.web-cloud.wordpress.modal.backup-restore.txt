# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 600x450
- Header: Titre + bouton fermer
- Banniere warning (jaune): Avertissement irreversible
- Info site
- Liste sauvegardes (radio buttons)
- Checkboxes elements a restaurer
- Footer: Annuler + Restaurer

# LISTE SAUVEGARDES
| Champ     | Description                    |
|-----------|--------------------------------|
| Date      | Date/heure sauvegarde          |
| Type      | Automatique ou Manuelle        |
| Taille    | Taille en MB                   |

# OPTIONS RESTAURATION
| Option           | Description                    |
|------------------|--------------------------------|
| Fichiers         | Restaurer fichiers WP          |
| Base de donnees  | Restaurer BDD MySQL            |

# INTERACTIONS
- Clic radio → Selectionne sauvegarde
- Clic checkbox → Toggle option
- Clic Annuler → Ferme modal
- Clic Restaurer → POST API restore

# API ENDPOINTS
GET /hosting/wordpress/{serviceName}/website/{id}/backup - Liste sauvegardes
POST /hosting/wordpress/{serviceName}/website/{id}/backup/{backupId}/restore
