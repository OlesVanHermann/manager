# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 450x380
- Header: Titre + bouton fermer
- Banniere succes (verte): Statut certificat
- Champs:
  - Domaine (readonly, fond gris)
  - Date expiration
  - Checkbox Forcer HTTPS
- Footer: Regenerer + Enregistrer

# INFORMATIONS SSL
| Champ        | Description                    |
|--------------|--------------------------------|
| Domaine      | FQDN du certificat             |
| Expire le    | Date expiration certificat     |
| Forcer HTTPS | Redirection automatique HTTP→HTTPS |

# STATUTS SSL
| Statut              | Couleur | Description            |
|---------------------|---------|------------------------|
| Actif               | Vert    | Certificat valide      |
| Expiring soon       | Jaune   | Expire dans < 30j      |
| Expired             | Rouge   | Certificat expire      |

# INTERACTIONS
- Toggle Forcer HTTPS → Change configuration
- Clic Regenerer → POST API regenerate cert
- Clic Enregistrer → PUT API update SSL config
- Clic X → Ferme modal

# API ENDPOINTS
GET /hosting/wordpress/{serviceName}/website/{id}/ssl - Get SSL info
PUT /hosting/wordpress/{serviceName}/website/{id}/ssl - Update config
POST /hosting/wordpress/{serviceName}/website/{id}/ssl/regenerate - Regenerate
