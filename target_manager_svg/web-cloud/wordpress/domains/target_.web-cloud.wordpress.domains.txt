STRUCTURE PAGE
1. Header + bouton "Ajouter domaine"

2. Table domaines
   - Colonnes: DOMAINE, TYPE, SSL, DNS, ACTIONS
   - Domaine principal avec alias www auto
   - Domaines alias avec redirection
   - Indicateurs visuels pour erreurs DNS/SSL

3. Bloc aide DNS
   - Configuration requise (A, CNAME)
   - Bouton copier config

TYPES DOMAINE
| Type      | Badge      | Description                    |
|-----------|------------|--------------------------------|
| Principal | Vert       | Domaine principal du site      |
| Alias     | Gris       | Redirige vers le principal     |
| Multisite | Bleu       | Installation WP separee        |

ETATS SSL
| Etat         | Couleur  | Description                    |
|--------------|----------|--------------------------------|
| Let's Encrypt| #10B981  | SSL actif et valide            |
| Non configure| #DC2626  | Pas de certificat              |
| En cours     | #F59E0B  | Generation en cours            |
| Expire       | #DC2626  | Certificat expire              |

ETATS DNS
| Etat         | Couleur  | Description                    |
|--------------|----------|--------------------------------|
| Configure    | #10B981  | DNS pointe vers serveur        |
| Erreur DNS   | #DC2626  | DNS mal configure              |
| En propagation| #F59E0B | Propagation en cours           |

API ENDPOINTS
GET    /webWordpress/{serviceName}/domain        -> Liste domaines
POST   /webWordpress/{serviceName}/domain        -> Ajouter domaine
DELETE /webWordpress/{serviceName}/domain/{name} -> Supprimer alias
PUT    /webWordpress/{serviceName}/domain/{name}/ssl -> Config SSL

ACTIONS
- "+ Ajouter domaine" -> Modal domain-add
- "SSL" -> Modal ssl-config
- "x" (supprimer) -> Modal domain-delete
- "Copier config" -> Copie DNS dans presse-papier
