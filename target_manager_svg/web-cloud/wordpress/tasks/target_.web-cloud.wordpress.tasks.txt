STRUCTURE PAGE
1. Header + bouton "Rafraichir"

2. Table taches
   - Colonnes: DATE, TYPE, DESCRIPTION, STATUT
   - Badge type colore
   - Badge statut colore
   - Taches en cours en surbrillance

TYPES TACHE
| Type        | Couleur  | Description                          |
|-------------|----------|--------------------------------------|
| Installation| Vert     | Installation WP ou plugin            |
| Mise a jour | Vert     | MAJ WP, theme ou plugin              |
| Sauvegarde  | Bleu     | Sauvegarde auto ou manuelle          |
| Restauration| Bleu     | Restauration depuis backup           |
| Cache       | Rouge    | Purge cache serveur ou CDN           |
| Domaine     | Gris     | Ajout/suppression domaine            |
| SSL         | Gris     | Generation certificat SSL            |

ETATS TACHE
| Etat      | Badge   | Couleur  | Description          |
|-----------|---------|----------|----------------------|
| En cours  | Orange  | #FEF3C7  | Operation en cours   |
| Termine   | Vert    | #DEF7EC  | Operation reussie    |
| Echec     | Rouge   | #FEE2E2  | Operation echouee    |

API ENDPOINTS
GET /webWordpress/{serviceName}/task -> Liste des taches

ACTIONS
- "Rafraichir" -> GET task (refresh)
- Click sur tache en echec -> Modal avec details erreur
