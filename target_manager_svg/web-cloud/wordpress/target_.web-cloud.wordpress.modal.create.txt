# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 500x480
- Header: Titre + bouton fermer
- Banniere info (bleue): Description creation
- Champs:
  - Nom du site (input, required)
  - Domaine (dropdown, cluster par defaut)
  - Email administrateur (input, required)
  - Langue (dropdown)
- Footer: Annuler + Creer

# CHAMPS FORMULAIRE
| Champ              | Type     | Validation       |
|--------------------|----------|------------------|
| Nom du site        | Input    | Required         |
| Domaine            | Dropdown | Default: cluster |
| Email admin        | Input    | Required, Email  |
| Langue             | Dropdown | Default: FR      |

# INTERACTIONS
- Clic Annuler → Ferme modal
- Clic Creer → POST API creation site
- Clic X → Ferme modal

# API ENDPOINTS
POST /hosting/wordpress/{serviceName}/website
Payload: { name, domain, adminEmail, language }
