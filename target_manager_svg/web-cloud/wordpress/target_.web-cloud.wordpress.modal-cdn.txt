STRUCTURE MODAL
1. HEADER - Titre "Configurer le CDN" + bouton fermer (x)
2. CONTENU:
   - Statut CDN: Badge actif/inactif + toggle ON/OFF
   - Options de cache:
     * Cache fichiers statiques (toggle) - recommande
     * Cache pages HTML (toggle) - attention contenu dynamique
     * Compression Brotli (toggle)
   - Duree de cache (TTL): Dropdown
   - Purger le cache: Bouton + derniere purge
3. BANNIERE INFO - Explication CDN
4. FOOTER - Boutons Annuler + Enregistrer

OPTIONS CDN
| Option           | Defaut | Description                              |
|------------------|--------|------------------------------------------|

| Cache HTML       | OFF    | Pages completes (attention dynamique)    |
