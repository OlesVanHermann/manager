# STRUCTURE MODAL
- Overlay semi-transparent
- Modal centree 600x320
- Barre rouge en haut (danger)
- Header: Icone warning + Titre + bouton fermer
- Contenu:
  - Question de confirmation
  - FQDN du site (readonly, fond gris)
  - Warning texte irreversible
- Footer: Annuler + Supprimer (rouge)

# DESIGN DANGER
| Element        | Style                      |
|----------------|----------------------------|
| Top stripe     | 4px, #DC2626 (rouge)       |
| Icon circle    | #FEE2E2 (rouge clair)      |
| Icon           | &#x26A0; rouge             |
| Bouton delete  | #DC2626 (rouge)            |

# INTERACTIONS
- Clic Annuler → Ferme modal
- Clic Supprimer → DELETE API suppression site
- Clic X → Ferme modal

# API ENDPOINTS
DELETE /hosting/wordpress/{serviceName}/website/{websiteId}
