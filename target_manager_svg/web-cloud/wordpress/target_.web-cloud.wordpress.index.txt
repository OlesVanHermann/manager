STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif x=510)
2. NAV2 - Web Cloud (Wordpress actif x=345)
3. LEFT PANEL - Liste des sites WordPress
   - Recherche
   - Compteur sites
   - Items sites (88px height chacun)
   - Bouton "Nouveau site"
4. RIGHT PANEL
   - Header: Nom site + version + offre
   - Quick actions header (Admin WP, Visiter)
   - NAV3: 6 tabs (General actif)
   - Contenu General:
     * Actions rapides (4 boutons)
     * Bloc WORDPRESS (infos site)
     * Bloc TECHNIQUE (infos serveur)
     * Zone Danger (suppression)

INTERACTIONS
- Clic item left panel → Selection site
- Clic tab NAV3 → Navigation vers tab
- Clic "Admin WP" → Ouvre /wp-admin
- Clic "Visiter le site" → Ouvre URL site
- Clic "Vider le cache" → POST flushCache
- Clic "Supprimer" → Modal confirmation

API ENDPOINTS
GET    /managedCMS/resource                    - Liste sites
GET    /managedCMS/resource/{serviceName}      - Details site
POST   /managedCMS/resource/{sn}/flushCache    - Vider cache
POST   /managedCMS/resource/{sn}/resetAdminPassword - Reset MDP
DELETE /managedCMS/resource/{serviceName}      - Supprimer site
