# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Wordpress actif)
3. LEFT PANEL
   - Recherche
   - Liste services WordPress
4. RIGHT PANEL
   - Header: Titre + description
   - Toolbar: Refresh + Commander
   - Table: Liste ressources
   - Pagination

# COLONNES TABLE
| Colonne         | Description                    |
|-----------------|--------------------------------|
| RESSOURCE       | serviceName (lien)             |
| OFFRE           | Quota max sites                |
| STATUT          | READY, CREATING, ERROR         |
| SITES INSTALLES | Nb sites / quota max           |
| ACTIONS         | Menu contextuel                |

# STATUTS
| Statut   | Couleur | Description            |
|----------|---------|------------------------|
| READY    | Vert    | Service operationnel   |
| CREATING | Jaune   | Creation en cours      |
| ERROR    | Rouge   | Erreur                 |

# INTERACTIONS
- Clic service LEFT PANEL → Charge detail
- Clic ressource table → Navigation vers general
- Clic Commander → Modal commande
- Clic menu actions → Dropdown

# API ENDPOINTS
GET /hosting/wordpress - Liste des services
GET /hosting/wordpress/{serviceName} - Details service
