STRUCTURE PAGE
1. LEFT PANEL (280px)
   - Header "Sites WordPress"
   - Barre recherche
   - Liste des sites avec:
     - Icone WordPress
     - Nom du site (displayName)
     - Version WordPress
     - Etat (circle color + label)
     - Offre + Datacenter

2. RIGHT PANEL (1096px)
   - Header avec nom site + boutons rapides
   - NAV3: 6 tabs (General, Domaines, Performances, Extensions, Sauvegardes, Taches)
   - Contenu tab General (defaut):
     - Quick Actions (4 boutons)
     - Bloc WordPress (etat, version, PHP, URL, admin)
     - Bloc Technique (offre, datacenter, SSL, CDN, auto-update, creation)
     - Zone Danger (suppression)

API ENDPOINTS
GET    /webWordpress                          -> Liste des sites
GET    /webWordpress/{serviceName}            -> Detail d'un site
POST   /webWordpress/{serviceName}/purgeCache -> Vider cache
POST   /webWordpress/{serviceName}/resetPassword -> Reset MDP admin
PUT    /webWordpress/{serviceName}            -> Mettre a jour config
DELETE /webWordpress/{serviceName}            -> Supprimer site

ACTIONS
- Click site left panel -> Selectionner et afficher details
- Click "Admin WP" -> Ouvrir wp-admin (nouvelle fenetre)
- Click "Visiter" -> Ouvrir site (nouvelle fenetre)
- Click "Vider le cache" -> POST purgeCache
- Click "Reinitialiser mot de passe" -> Modal reset password
- Click "Mettre a jour WP" -> Modal confirmation MAJ
- Click "Supprimer le site" -> Modal confirmation suppression
