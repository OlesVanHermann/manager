STRUCTURE PAGE
1. Header + bouton "Creer une sauvegarde"
2. Sous-titre avec info retention

3. Table sauvegardes
   - Colonnes: DATE, TYPE, TAILLE, STATUT, ACTIONS
   - Badge Auto (gris) / Manuel (bleu)
   - Note optionnelle pour sauvegardes manuelles
   - Bouton Restaurer pour toutes
   - Bouton Supprimer pour manuelles uniquement

4. Banniere info
   - Regles de retention
   - Espace utilise

TYPES SAUVEGARDE
| Type   | Badge  | Retention        | Suppression possible |
|--------|--------|------------------|----------------------|
| Auto   | Gris   | 14 jours         | Non                  |
| Manuel | Bleu   | Jusqu'a suppression | Oui               |

API ENDPOINTS
GET    /webWordpress/{serviceName}/backup        -> Liste sauvegardes
POST   /webWordpress/{serviceName}/backup        -> Creer sauvegarde manuelle
POST   /webWordpress/{serviceName}/backup/{id}/restore -> Restaurer
DELETE /webWordpress/{serviceName}/backup/{id}   -> Supprimer (manuel only)
GET    /webWordpress/{serviceName}/backup/quota  -> Espace utilise

ACTIONS
- "+ Creer une sauvegarde" -> Modal backup-create
- "Restaurer" -> Modal backup-restore
- "Supprimer" -> Modal backup-delete
