STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif x=510)
2. NAV2 - Web Cloud (Wordpress actif x=345)
3. LEFT PANEL - Liste des sites WordPress
4. RIGHT PANEL
   - Header: Nom site + version + offre
   - NAV3: 6 tabs (Extensions actif)
   - NAV4: Themes (actif) | Plugins
   - Contenu Themes:
     * Header "THEMES INSTALLES (n)"
     * Grid de cards themes (3 colonnes, 320px chaque)
     * Chaque card:
       - Preview placeholder (image theme)
       - Nom theme
       - Version
       - Badge si actif ou MAJ dispo
       - Bouton "Mettre a jour" si MAJ

BADGES THEMES
- Actif: fond vert (#DEF7EC), texte vert fonce
- MAJ disponible: fond jaune (#FEF3C7), texte orange

INTERACTIONS
- Clic NAV4 "Plugins" → Navigation vers extensions-plugins
- Clic "Mettre a jour" → POST /themes/{name}/update

API ENDPOINTS
GET  /managedCMS/resource/{sn}/themes              - Liste themes
POST /managedCMS/resource/{sn}/themes/{name}/update - MAJ theme
