STRUCTURE MODAL
1. Overlay sombre
2. Container 560x680px centre
3. Header: Creer un compte email
4. Body:
   - Nom du compte (local + @domain)
   - Description (optionnel)
   - Mot de passe
   - Confirmer mot de passe
   - Regles mot de passe
   - Taille du compte (select)
   - Info disponibilite (comptes/espace)
5. Footer: Annuler/Creer

CHAMPS FORMULAIRE
| Champ               | Type     | Required |
|---------------------|----------|----------|
| Nom du compte       | Input    | Oui      |
| Description         | Input    | Non      |
| Mot de passe        | Password | Oui      |
| Confirmer           | Password | Oui      |
| Taille (Go)         | Select   | Oui      |

VALIDATION
- 9 a 30 caracteres
- Au moins 1 chiffre, 1 majuscule, 1 minuscule
- Aucun caractere accentue
- Confirmation doit correspondre

API ENDPOINT
POST /email/domain/{domain}/account
Body: { accountName, password, description, size }
