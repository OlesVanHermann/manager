## 9. APIS & ENDPOINTS

```
# Vérifier domaines avec MX Plan inclus
GET /email/domain
→ Liste des domaines avec email inclus

# Obtenir catalogue prix
GET /order/catalog/formatted/exchange
GET /order/catalog/formatted/emailPro
GET /order/catalog/formatted/zimbra
→ Prix actuels des offres

# Commander
POST /order/cart
POST /order/cart/{cartId}/exchange
POST /order/cart/{cartId}/emailPro
POST /order/cart/{cartId}/checkout
```
