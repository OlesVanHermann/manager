STRUCTURE MODAL
1. Overlay sombre
2. Container 540x580px centre
3. Header: Modifier le repondeur
4. Body:
   - Affichage compte (readonly)
   - Radio duree (Temporaire/Permanent)
   - Date debut + Date fin (si temporaire)
   - Textarea message
   - Info HTML non autorise
   - Checkbox copie messages
5. Footer: Annuler/Modifier

CHAMPS FORMULAIRE
| Champ               | Type       | Editable |
|---------------------|------------|----------|
| Compte              | Display    | Non      |
| Duree               | Radio      | Oui      |
| Date de debut       | Datepicker | Oui      |
| Date de fin         | Datepicker | Oui      |
| Message             | Textarea   | Oui      |
| Copie messages      | Checkbox   | Oui      |

VALIDATION
- Dates obligatoires si mode temporaire
- Message ne doit pas contenir HTML
- Date fin > Date debut

API ENDPOINT
PUT /email/domain/{domain}/responder/{account}
Body: { from, to, content, copy }
