INTERACTIONS

- Clic [+ Ajouter contact] → Modal creation contact externe

- Clic (...) → Menu contextuel (supprimer)

API ENDPOINTS
GET /email/exchange/{service}/externalContact → Liste contacts externes
POST /email/exchange/{service}/externalContact → Creer contact
PUT /email/exchange/{service}/externalContact/{email} → Modifier contact
DELETE /email/exchange/{service}/externalContact/{email} → Supprimer contact
