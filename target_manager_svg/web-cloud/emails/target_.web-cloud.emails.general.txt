STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Emails actif)
3. LEFT PANEL (280px)
   - NAV3 Groupement: [MX Plan] [Email Pro] [Exchange] [Zimbra]
   - Recherche domaines
   - Liste domaines avec statut
   - Pagination
4. RIGHT PANEL (1096px)
   - Header: Nom domaine + Badge statut
   - NAV4 Fonction: [Comptes] [Redirections] [Listes] [Filtres] [Repondeurs] [Taches]
   - Banniere info offre
   - Toolbar: Refresh + Search + Bouton creer
   - Tableau comptes email
   - Pagination

TABLEAU COMPTES
| Colonne           | Type              | Description            |
|-------------------|-------------------|------------------------|
| Compte            | Lien primary      | Adresse email          |
| Description       | Texte             | Description compte     |
| Quota             | Texte             | X Go / Y Go            |
| Statut            | Badge             | Actif/Inactif          |
| Derniere connexion| Date              | Format DD/MM/YYYY HH:mm|
| Actions           | Menu              | Edit, Delete, etc.     |

INTERACTIONS
- Clic NAV3 groupe -> Filtre type service (MX Plan, Email Pro, Exchange, Zimbra)
- Clic domaine LEFT PANEL -> Charge details RIGHT PANEL
- Clic NAV4 tab -> Navigation onglet
- Clic bouton Creer -> Modal creation compte
- Clic menu Actions -> Menu contextuel

API ENDPOINTS
GET  /email/domain                    - Liste domaines MX Plan
GET  /email/domain/{domain}           - Details domaine
GET  /email/domain/{domain}/account   - Liste comptes
POST /email/domain/{domain}/account   - Creer compte
GET  /email/pro/{service}             - Details Email Pro
GET  /email/exchange/{org}/{exchange} - Details Exchange
GET  /zimbra/{service}                - Details Zimbra
