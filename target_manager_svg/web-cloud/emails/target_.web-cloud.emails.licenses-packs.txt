STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section Web Cloud (Emails actif)
3. LEFT PANEL (280px)
   - NAV3 Toggle: [General] [Packs] - Packs actif
   - Resume abonnements (nb packs, nb licences)
   - Detail par type offre
4. RIGHT PANEL (1096px)
   - Header: Titre "Mes packs email"
   - NAV4: 3 tabs Mode Packs (Mes packs actif)
   - Toolbar: [+ Commander pack]
   - Liste packs avec details
   - Total mensuel

NAV4 TABS MODE PACKS
| Tab         | Route              | Description                    |
|-------------|-------------------|--------------------------------|
| Mes packs   | /packs            | Liste packs actifs (ACTIF)     |
| A la carte  | /packs/alacarte   | Commander licences             |
| Historique  | /packs/history    | Historique commandes           |

STRUCTURE CARD PACK
| Element           | Description                              |
|-------------------|------------------------------------------|
| Barre laterale    | Couleur selon offre (8px)                |
| Titre             | Nom du pack (ex: Exchange Business 10)   |
| Badge statut      | Actif/Suspendu                           |
| Renouvellement    | Date prochain paiement                   |
| Scope             | Multi-domaine ou domaine specifique      |
| Progress bar      | Utilisation licences (used/max)          |
| Prix              | Mensuel HT                               |
| Actions           | Gerer, Upgrade                           |

INTERACTIONS
- Clic NAV3 General → Switch mode General
- Clic NAV4 tab → Navigation vers tab Packs
- Clic [+ Commander pack] → Modal choix pack
- Clic [Gerer] → Modal gestion pack (ajouter licences, resilier)
- Clic [Upgrade] → Modal upgrade pack

API ENDPOINTS
GET /email/exchange → Liste services Exchange
GET /email/pro → Liste services Email Pro
GET /email/zimbra → Liste services Zimbra
GET /services/{service} → Details service
GET /services/{service}/billing → Facturation
