# CARDS DNS

| Card  | Statut possible       | Actions                    |
|-------|-----------------------|----------------------------|
| MX    | Valide/Invalide       | Auto-configurer            |
| SPF   | Valide/Invalide       | Auto-configurer            |
| DKIM  | Actif/Inactif         | Régénérer                  |
| DMARC | Valide/Non configuré  | Configurer                 |
| SRV   | Valide/Invalide       | Auto-configurer            |

# INTERACTIONS

- Clic "Diagnostiquer" → Lance diagnostic complet
- Clic "Auto-configurer X" → Modal confirmation + action
- Clic "Régénérer DKIM" → Modal confirmation + action

# API ENDPOINTS

GET  /email/domains/{domain}/dns/diagnostic → Diagnostic complet
GET  /email/domains/{domain}/dns/mx        → État MX
POST /email/domains/{domain}/dns/mx        → Auto-configurer MX
GET  /email/domains/{domain}/dns/spf       → État SPF
POST /email/domains/{domain}/dns/spf       → Auto-configurer SPF
GET  /email/domains/{domain}/dns/dkim      → État DKIM
POST /email/domains/{domain}/dns/dkim/regenerate → Régénérer DKIM
GET  /email/domains/{domain}/dns/dmarc     → État DMARC
POST /email/domains/{domain}/dns/dmarc     → Configurer DMARC
