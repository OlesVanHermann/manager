# API ENDPOINTS

## Ressources
GET    /email/exchange/{org}/{domain}/resources        → Liste ressources
POST   /email/exchange/{org}/{domain}/resources        → Créer ressource
DELETE /email/exchange/{org}/{d}/resources/{id}        → Supprimer

## Dossiers publics
GET    /email/exchange/{org}/{domain}/publicFolders    → Liste dossiers
POST   /email/exchange/{org}/{domain}/publicFolders    → Créer dossier
PUT    /email/exchange/{org}/{d}/publicFolders/{id}    → Modifier
DELETE /email/exchange/{org}/{d}/publicFolders/{id}    → Supprimer

## Groupes
GET    /email/exchange/{org}/{domain}/groups           → Liste groupes
POST   /email/exchange/{org}/{domain}/groups           → Créer groupe
GET    /email/exchange/{org}/{d}/groups/{id}/members   → Membres groupe

## Délégations
GET    /email/exchange/{org}/{d}/accounts/{a}/delegations → Délégations compte
POST   /email/exchange/{org}/{d}/accounts/{a}/delegations → Ajouter délégation

## Quotas
GET    /email/exchange/{org}/{domain}/quota            → Quotas globaux
