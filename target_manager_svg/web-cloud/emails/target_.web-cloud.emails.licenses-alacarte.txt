INTERACTIONS
- Clic NAV3 General → Switch mode General
- Clic NAV4 tab → Navigation vers tab Packs
- Clic carte offre → Selection offre (highlight)
- Clic +/- quantite → Ajuste nombre licences
- Clic [Commander] → Processus commande (panier OVH)

API ENDPOINTS
GET /order/catalog/formatted/emailpro → Catalogue Email Pro
GET /order/catalog/formatted/exchange → Catalogue Exchange
GET /order/catalog/formatted/zimbra → Catalogue Zimbra
POST /order/cart → Creer panier
POST /order/cart/{cartId}/email → Ajouter au panier
POST /order/cart/{cartId}/checkout → Finaliser commande
