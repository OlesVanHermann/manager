# INTERACTIONS

- Clic "+ Ajouter contact" → Modal création contact
- Clic "↑ Importer" → Modal import CSV/vCard
- Clic "↓ Exporter" → Téléchargement CSV/vCard
- Clic ✎ → Modal modifier contact
- Clic × → Modal confirmer suppression

# RESTRICTIONS

⚠️ Les contacts externes Exchange sont UNIQUEMENT disponibles pour :
- Exchange (toutes versions)

Les contacts apparaissent dans le carnet d'adresses global (GAL) de tous les utilisateurs Exchange du domaine.

# API ENDPOINTS

GET    /email/exchange/{org}/{domain}/contacts       → Liste contacts
POST   /email/exchange/{org}/{domain}/contacts       → Créer contact
GET    /email/exchange/{org}/{d}/contacts/{id}       → Détails contact
PUT    /email/exchange/{org}/{d}/contacts/{id}       → Modifier contact
DELETE /email/exchange/{org}/{d}/contacts/{id}       → Supprimer contact
POST   /email/exchange/{org}/{domain}/contacts/import → Import CSV/vCard
GET    /email/exchange/{org}/{domain}/contacts/export → Export CSV/vCard
