TYPES D'APPAREILS
| Type      | Couleur   | Description                     |
|-----------|-----------|----------------------------------|
| Mobile    | #E5E7EB   | Smartphones iOS/Android         |
| Desktop   | #DBEAFE   | Outlook desktop, Mac Mail       |
| Tablette  | #FEF3C7   | iPad, tablettes Android         |
| Web       | #DEF7EC   | OWA, Webmail                    |

STATUTS APPAREILS
| Statut    | Couleur   | Critere                         |
|-----------|-----------|----------------------------------|
| Actif     | #DEF7EC   | Sync < 7 jours                  |
| Inactif   | #FEF3C7   | Sync 7-30 jours                 |
| Obsolete  | #FEE2E2   | Sync > 30 jours                 |

ACTIONS SECURITE
| Action    | Effet                                      |
|-----------|--------------------------------------------|
| Wipe      | Efface donnees Exchange de l'appareil      |
| Bloquer   | Empeche synchronisation future             |
| Supprimer | Retire de la liste (si inactif)            |

POLITIQUE MDM EXCHANGE
- Peut imposer code PIN
- Peut imposer chiffrement
- Peut limiter applications
- Configurable par compte ou globalement

API ENDPOINTS
GET  /email/domains/{domain}/devices              -> Liste appareils
GET  /email/domains/{domain}/devices/{id}         -> Details appareil
POST /email/domains/{domain}/devices/{id}/wipe    -> Wipe distant
POST /email/domains/{domain}/devices/{id}/block   -> Bloquer
DELETE /email/domains/{domain}/devices/{id}       -> Supprimer
GET  /email/domains/{domain}/devices/policy       -> Politique MDM
PUT  /email/domains/{domain}/devices/policy       -> Modifier politique
