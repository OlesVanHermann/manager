ACTIONS PAR RESSOURCE
| Action     | Comportement                               |
|------------|--------------------------------------------|
| Calendrier | Voir disponibilites de la ressource        |
| Modifier   | Modal edition nom, capacite, etc.          |
| Suppr      | Suppression avec confirmation              |

API ENDPOINTS
GET  /email/domains/{domain}/resources            -> Liste ressources
POST /email/domains/{domain}/resources            -> Creer ressource
GET  /email/domains/{domain}/resources/{id}       -> Details
PUT  /email/domains/{domain}/resources/{id}       -> Modifier
DELETE /email/domains/{domain}/resources/{id}     -> Supprimer
GET  /email/domains/{domain}/resources/{id}/calendar -> Calendrier
