STRUCTURE PAGE
1. Left panel avec liste domaines
2. Right panel header (domaine)
3. NAV3: 8 tabs (Avance actif)
4. NAV4: Ressources | Contacts | Audit (actif) | Appareils
5. Filtres (periode, compte, export)
6. Table journal audit
7. Pagination
8. Cartes statistiques

TYPES D'ACTIONS AUDIT
| Action                    | Description                        |
|---------------------------|------------------------------------|
| Connexion                 | Login OWA/Outlook                  |
| Deconnexion               | Logout                             |
| Envoi email               | Email envoye                       |
| Reception email           | Email recu                         |
| Modification mot de passe | Changement password                |
| Ajout delegue             | Partage calendrier/boite           |
| Modification parametres   | Config compte modifiee             |
| Suppression email         | Message supprime                   |

RESULTATS POSSIBLES
| Resultat | Couleur   | Description                     |
|----------|-----------|----------------------------------|
| Succes   | #DEF7EC   | Action completee                |
| Echec    | #FEE2E2   | Erreur ou refus                 |
| Warning  | #FEF3C7   | Complete avec avertissement     |

FILTRES DISPONIBLES
| Filtre    | Options                                    |
|-----------|--------------------------------------------|
| Periode   | 7j, 30j, 90j, Personnalise                 |
| Compte    | Tous, Liste comptes specifiques            |
| Action    | Toutes, Connexions, Emails, Parametres     |
| Resultat  | Tous, Succes, Echecs                       |

RETENTION LOGS

API ENDPOINTS
GET  /email/domains/{domain}/audit             -> Liste evenements
GET  /email/domains/{domain}/audit/stats       -> Statistiques
GET  /email/domains/{domain}/audit/{id}        -> Details evenement
GET  /email/domains/{domain}/audit/export      -> Export CSV
