STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section Web Cloud (Emails actif)
3. LEFT PANEL (280px)
   - NAV3 Toggle: [General] [Packs] - General actif
   - Recherche + Liste domaines (Exchange only pour ce tab)
4. RIGHT PANEL (1096px)
   - Header: Nom domaine + info
   - NAV4: 7 tabs Mode General (Avance actif)
   - NAV5 Sub-tabs: [Ressources] [Contacts ext.] [Audit] [Appareils] - Audit actif
   - Filtres: Periode, Utilisateur, Action
   - Bouton Exporter
   - Table logs audit
   - Pagination

NOTE: Tab "Avance" et sous-tabs visibles uniquement pour domaines avec Exchange.

NAV5 SUB-TABS (advanced)
| Sub-tab      | Route              | Contenu                        |
|--------------|--------------------|--------------------------------|
| Ressources   | /advanced          | Salles, equipements, partagees |
| Contacts ext.| /advanced-contacts | Contacts externes Exchange     |
| Audit        | /advanced-audit    | Logs activites utilisateurs    |
| Appareils    | /advanced-devices  | Appareils mobiles connectes    |

FILTRES DISPONIBLES
| Filtre       | Options                                        |
|--------------|------------------------------------------------|
| Periode      | 24h, 7 jours, 30 jours, 90 jours, Personnalise |
| Utilisateur  | Tous, Liste comptes Exchange                   |
| Action       | Toutes, Connexion, Envoi, Suppression, etc.    |

TYPES D'ACTIONS LOGGEES
| Action           | Description                    | Couleur badge |
|------------------|--------------------------------|---------------|
| Connexion        | Login OWA/ActiveSync/IMAP/etc  | #DEF7EC (OK)  |
| Connexion echouee| Echec authentification         | #FEF3C7       |
| Envoi message    | Email envoye                   | #DEF7EC       |
| Suppression      | Element supprime               | #FEF3C7       |
| Modification     | Parametre modifie              | #DBEAFE       |

INTERACTIONS
- Clic sub-tab NAV5 → Navigation sous-section
- Clic filtre Periode → Dropdown selection periode
- Clic filtre Utilisateur → Dropdown liste comptes
- Clic filtre Action → Dropdown types actions
- Clic [Exporter] → Telecharge CSV logs
- Clic pagination → Navigation pages

API ENDPOINTS
GET /email/exchange/{service}/log → Liste logs audit
GET /email/exchange/{service}/log?startDate=X&endDate=Y → Filtre periode
GET /email/exchange/{service}/log?user=X → Filtre utilisateur
GET /email/exchange/{service}/log?action=X → Filtre action
