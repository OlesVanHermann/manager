# Cahier des Charges : web-cloud/email-pro - Pieds de page (Disclaimer)

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des pieds de page Email Pro par domaine
- **Type** : Page liste avec tableau
- **Objectif** : Configurer les signatures automatiques par domaine

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Email Pro (menu)
- Titre "Pieds de page"
- Bandeau info (si aucun domaine)
- Tableau des domaines avec leur pied de page
- Pagination

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Email Pro
- **OdsMessage** : Alerte info bleue
- **OdsTable** : Liste des domaines
- **OdsBadge** : Statut
- **OdsActionMenu** : Menu actions (⋮)
- **OdsButton** : Configurer

## 4. TITRE
"Pieds de page"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Vous devez associer un domaine à votre offre Email Pro pour pouvoir configurer un pied de page."

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Domaine associé | Nom du domaine | example.com |
| Contenu | Aperçu du pied de page | Ce message est confidentiel... |
| Externe uniquement | Application externe | Oui / Non |
| Statut | État configuration | - / ⏳ / Non configuré |
| Actions | Menu ⋮ ou bouton | Configurer |

## 7. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| - | Gris | Configuré et actif |
| ⏳ | Bleu (#3B82F6) | En cours de traitement |
| Non configuré | Orange (#F59E0B) | Aucun pied de page |

## 8. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Ouvre modal modification |
| Supprimer | Ouvre modal suppression (rouge) |
| Configurer | Configure le pied de page |

## 9. BOUTON CONFIGURER
Pour les domaines non configurés, un bouton "Configurer" remplace le menu actions.

## 10. PAGINATION
"Affichage X-Y sur Z domaines"

## 11. SIDEBAR MENU
| Élément | État |
|---------|------|
| Informations générales | - |
| Comptes e-mail | - |
| Domaines associés | - |
| Contacts externes | - |
| Redirections | - |
| Listes de diffusion | - |
| **Pieds de page** | Actif (bleu) |
| Tâches récentes | - |

## 12. APIS & ENDPOINTS
- GET /email/pro/{service}/domain : Lister domaines
- GET /email/pro/{service}/domain/{domain}/disclaimer : Récupérer pied de page
- PUT /email/pro/{service}/domain/{domain}/disclaimer : Modifier
- DELETE /email/pro/{service}/domain/{domain}/disclaimer : Supprimer
