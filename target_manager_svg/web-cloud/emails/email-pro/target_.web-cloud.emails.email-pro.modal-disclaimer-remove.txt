# Cahier des Charges : web-cloud/email-pro - Modal Supprimer pied de page

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'un pied de page
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer le pied de page d'un domaine

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec domaine
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x160)
- **OdsText** : Message confirmation, domaine
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer le pied de page"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Supprimer ce pied de page ? |
| Domaine | {domaine} (en gras) |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le pied de page |

## 7. APIS & ENDPOINTS
- DELETE /email/pro/{service}/domain/{domain}/disclaimer : Supprimer
