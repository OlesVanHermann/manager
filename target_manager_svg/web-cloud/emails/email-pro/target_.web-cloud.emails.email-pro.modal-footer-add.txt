# Cahier des Charges : web-cloud/email-pro - Modal Créer un pied de page

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un pied de page (disclaimer)
- **Type** : Modal formulaire
- **Objectif** : Créer une signature automatique pour un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Sélection domaine
- Zone de texte contenu
- Options d'application
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x460)
- **OdsSelect** : Choix du domaine
- **OdsTextarea** : Contenu du pied de page
- **OdsCheckbox** : Options (externe, interne)
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un pied de page"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Domaine | Select | Oui (*) |
| Contenu du pied de page | Textarea | Oui (*) |

## 6. PLACEHOLDER CONTENU
```
--
Ce message est confidentiel.
Si vous n'êtes pas le destinataire,
veuillez supprimer ce message.
```

## 7. OPTIONS D'APPLICATION
| Checkbox | État | Description |
|----------|------|-------------|
| ☑ | Coché | Appliquer aux emails sortants externes |
| ☐ | Non coché | Appliquer aux emails internes |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le pied de page |

## 9. APIS & ENDPOINTS
- POST /email/pro/{service}/domain/{domain}/disclaimer : Créer
  - content: string (HTML)
  - externalOnly: boolean
