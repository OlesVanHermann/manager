# Cahier des Charges : web-cloud/email-pro - Liste services

## 1. RESUME EXECUTIF

### Perimetre
Liste de tous les services Email Pro du compte.

### Objectif fonctionnel
- Voir tous les services Email Pro
- Rechercher un service
- Commander un nouveau service
