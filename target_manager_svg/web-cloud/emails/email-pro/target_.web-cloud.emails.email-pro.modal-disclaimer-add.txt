# Cahier des Charges : web-cloud/email-pro - Modal Configurer pied de page

## 1. VUE D'ENSEMBLE
- **Modal** : Configuration d'un pied de page (disclaimer)
- **Type** : Modal formulaire
- **Objectif** : Définir une signature automatique pour un domaine

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Domaine (lecture seule)
- Zone de texte contenu
- Info format
- Checkbox externe uniquement
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x440)
- **OdsText** : Domaine (lecture seule)
- **OdsTextarea** : Contenu du pied de page
- **OdsText** : Info format
- **OdsCheckbox** : Externe uniquement
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Configurer le pied de page"

## 5. DOMAINE (LECTURE SEULE)
| Label | Valeur |
|-------|--------|
| Domaine | example.com |

## 6. CONTENU
| Élément | Description |
|---------|-------------|
| Label | Contenu du pied de page * |
| Composant | Textarea (150px hauteur) |
| Placeholder | Ce message et ses pièces jointes... |

## 7. INFO FORMAT
"Format HTML accepté. Maximum 5000 caractères."

## 8. OPTION EXTERNE
| Checkbox | Description |
|----------|-------------|
| ☑ | Appliquer uniquement aux e-mails externes |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Enregistrer | Primaire bleu | Sauvegarde le pied de page |

## 10. APIS & ENDPOINTS
- PUT /email/pro/{service}/domain/{domain}/disclaimer : Configurer
  - content: string (HTML)
  - externalOnly: boolean
