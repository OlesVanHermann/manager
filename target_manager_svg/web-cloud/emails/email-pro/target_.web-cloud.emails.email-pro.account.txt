# Cahier des Charges : web-cloud/email-pro - Comptes e-mail

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des comptes email Email Pro
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les comptes email du service

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Email Pro (menu)
- Titre "Comptes e-mail"
- Info webmail et compteur
- Boutons d'action
- Barre de recherche
- Tableau des comptes
- Pagination

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Email Pro
- **OdsButton** : Ajouter un compte, Migrer (OMM)
- **OdsInput** : Recherche par e-mail
- **OdsTable** : Liste des comptes
- **OdsProgressBar** : Barre quota
- **OdsBadge** : Alias, Statut
- **OdsActionMenu** : Menu actions (⋮)
- **OdsText** : Pagination

## 4. TITRE
"Comptes e-mail"

## 5. INFORMATIONS HEADER
| Label | Valeur | Exemple |
|-------|--------|---------|
| Webmail | URL cliquable | https://pro1.mail.ovh.net |
| E-mails | Compteur | 5 / 10 |

## 6. BOUTONS D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter un compte | Secondaire | Ouvre modal création |
| Migrer (OMM) | Secondaire + lien externe | Ouvre OMM |

## 7. COLONNES DU TABLEAU
| Colonne | Description | Type |
|---------|-------------|------|
| E-mail | Adresse du compte | Texte |
| Quota | Barre progression + texte | ProgressBar |
| Alias | Badge nombre | Badge vert |
| Statut | État du compte | Badge/Texte |
| Actions | Menu ⋮ | ActionMenu |

## 8. BARRE QUOTA
| Seuil | Couleur | Description |
|-------|---------|-------------|
| 0-50% | Vert (#10B981) | Utilisation normale |
| 50-80% | Orange (#F59E0B) | Attention |
| 80-100% | Rouge (#EF4444) | Critique |

## 9. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| - | Vert | Compte actif (pas d'alerte) |
| En création | Bleu (#3B82F6) | Création en cours |
| SPAM | Rouge (#EF4444) | Spammer détecté |
| Suspendu | Orange (#F59E0B) | Compte suspendu |

## 10. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Modifier | Ouvre modal modification |
| Gérer les alias | Navigue vers page alias |
| Délégations | Ouvre modal délégations |
| Réinitialiser | Action en rouge |

## 13. APIS & ENDPOINTS
- GET /email/pro/{service}/account : Lister comptes
- POST /email/pro/{service}/account : Créer compte
- PUT /email/pro/{service}/account/{email} : Modifier
- DELETE /email/pro/{service}/account/{email} : Supprimer
