# Cahier des Charges : web-cloud/email-pro - Membres de liste de diffusion

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des membres d'une liste de diffusion
- **Type** : Modal liste avec tableau paginé
- **Objectif** : Ajouter/supprimer des membres de la liste

## 2. STRUCTURE MODAL
- Header avec email de la liste et bouton fermer (×)
- Bouton ajouter membre
- Tableau des membres
- Pagination
- Bouton Fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x490)
- **OdsButton** : Ajouter membre
- **OdsTable** : Liste des membres
- **OdsButton** : Supprimer (×) par ligne
- **OdsPagination** : Pagination
- **OdsButton** : Fermer (secondaire)

## 4. TITRE
"Membres - {email}"
Exemple : "Membres - marketing@example.com"

## 5. BOUTON D'ACTION
| Bouton | Position | Action |
|--------|----------|--------|
| Ajouter membre | Haut droite | Ouvre modal ajout |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| E-mail | Adresse du membre | user1@example.com |
| Ajouté le | Date d'ajout | 15/12/2024 |
| - | Bouton supprimer (× rouge) | - |

## 7. EXEMPLES DE DONNÉES
| E-mail | Ajouté le |
|--------|-----------|
| user1@example.com | 15/12/2024 |
| user2@example.com | 14/12/2024 |
| user3@example.com | 10/12/2024 |
| user4@example.com | 08/12/2024 |

## 8. PAGINATION
"X-Y sur Z membres"

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Fermer | Secondaire | Ferme la modal |

## 10. APIS & ENDPOINTS
- GET /email/pro/{service}/mailingList/{email}/subscriber : Lister membres
- POST /email/pro/{service}/mailingList/{email}/subscriber : Ajouter
- DELETE /email/pro/{service}/mailingList/{email}/subscriber/{member} : Supprimer
