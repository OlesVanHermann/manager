# Cahier des Charges : web-cloud/email-pro - Modal Créer une liste de diffusion

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une liste de diffusion
- **Type** : Modal formulaire
- **Objectif** : Créer une nouvelle liste avec propriétaire

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Champs du formulaire
- Options
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsInput** : Nom, partie locale email
- **OdsSelect** : Domaine, propriétaire, langue
- **OdsCheckbox** : Modération
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer une liste de diffusion"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Nom de la liste | Input | Oui (*) |
| Adresse e-mail | Input + @ + Select domaine | Oui (*) |
| Propriétaire | Select | Oui (*) |
| Langue | Select | Non |

## 6. PLACEHOLDERS
| Champ | Placeholder |
|-------|-------------|
| Nom de la liste | Ex: Équipe Marketing |
| Email | marketing |

## 7. LANGUES DISPONIBLES
| Valeur | Label |
|--------|-------|
| fr | Français |
| en | English |
| de | Deutsch |
| es | Español |

## 8. OPTIONS
| Checkbox | État | Description |
|----------|------|-------------|
| ☑ | Coché | Modération des messages |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée la liste |

## 10. APIS & ENDPOINTS
- POST /email/pro/{service}/mailingList : Créer liste
  - name: string
  - address: string
  - owner: string
  - language: string
  - moderationEnabled: boolean
