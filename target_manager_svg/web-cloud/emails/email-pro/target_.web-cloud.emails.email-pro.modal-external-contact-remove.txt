# Cahier des Charges : web-cloud/email-pro - Modal Supprimer un contact externe

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression d'un contact externe
- **Type** : Modal confirmation danger
- **Objectif** : Supprimer un contact de l'annuaire

## 2. STRUCTURE MODAL
- Header rouge avec titre
- Message de confirmation avec nom et email
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x160)
- **OdsText** : Message confirmation, contact
- **OdsButton** : Annuler, Supprimer (rouge)

## 4. TITRE
"Supprimer le contact"
Header fond rouge (#FEF2F2), texte rouge (#DC2626)

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | Supprimer ce contact externe ? |
| Contact | {nom} ({email}) en gras |

Exemple : "Jean Fournisseur (fournisseur@external.com)"

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le contact |

## 7. APIS & ENDPOINTS
- DELETE /email/pro/{service}/externalContact/{id} : Supprimer contact
