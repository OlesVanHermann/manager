# Cahier des Charges : web-cloud/email-pro - Listes de diffusion

## 1. VUE D'ENSEMBLE
- **Écran** : Gestion des listes de diffusion Email Pro
- **Type** : Page liste avec tableau
- **Objectif** : Créer et gérer des listes de diffusion

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Email Pro (menu)
- Lien retour
- Titre "Listes de diffusion"
- Bouton créer une liste
- Barre de quota
- Barre de recherche
- Tableau des listes
- Pagination
- Bandeau info

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Email Pro
- **OdsLink** : Retour vers compte
- **OdsButton** : Créer une liste
- **OdsProgressBar** : Quota utilisé
- **OdsInput** : Recherche liste
- **OdsTable** : Liste des listes
- **OdsBadge** : Statut
- **OdsActionMenu** : Menu actions (⋮)
- **OdsMessage** : Bandeau info bleu

## 4. TITRE
"Listes de diffusion"

## 5. LIEN RETOUR
"← Listes de diffusion pour {email}"

## 6. BOUTON D'ACTION PRINCIPAL
| Bouton | Position | Action |
|--------|----------|--------|
| Créer une liste | Haut droite | Ouvre modal création |

## 7. BARRE DE QUOTA
| Élément | Description |
|---------|-------------|
| Label | Quota utilisé : |
| Barre | Progression bleue |
| Texte | X / Y listes |

## 8. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Nom | Nom de la liste | Équipe Marketing |
| Adresse | Email de la liste | marketing@example.com |
| Membres | Nombre de membres | 12 membres |
| Statut | État de la liste | OK, En création |
| Actions | Menu ⋮ | Gérer, Modifier, Supprimer |

## 9. BADGES STATUT
| Statut | Couleur | Description |
|--------|---------|-------------|
| OK | Vert (#10B981) | Liste active |
| En création | Bleu (#3B82F6) | Création en cours |

## 10. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Gérer les membres | Ouvre modal membres |
| Modérateurs | Ouvre modal modérateurs |
| Modifier | Ouvre modal modification |
| Supprimer | Ouvre modal suppression (rouge) |

## 11. PAGINATION
"Affichage X-Y sur Z listes"

## 12. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Les listes de diffusion permettent de contacter plusieurs destinataires en une seule adresse. Chaque liste peut avoir ses propres modérateurs."

## 13. SIDEBAR MENU
| Élément | État |
|---------|------|
| Informations générales | - |
| Comptes e-mail | - |
| Domaines associés | - |
| Contacts externes | - |
| Redirections | - |
| **Listes de diffusion** | Actif (bleu) |
| Tâches récentes | - |

## 14. APIS & ENDPOINTS
- GET /email/pro/{service}/mailingList : Lister listes
- POST /email/pro/{service}/mailingList : Créer liste
- PUT /email/pro/{service}/mailingList/{email} : Modifier
- DELETE /email/pro/{service}/mailingList/{email} : Supprimer
- GET /email/pro/{service}/mailingList/{email}/subscriber : Lister membres
- GET /email/pro/{service}/mailingList/{email}/moderator : Lister modérateurs
