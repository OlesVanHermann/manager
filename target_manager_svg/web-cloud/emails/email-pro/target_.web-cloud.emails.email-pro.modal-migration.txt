# Cahier des Charges : web-cloud/email-pro - Modal Migration

## 1. VUE D'ENSEMBLE
- **Modal** : Migration depuis un autre service email
- **Type** : Modal formulaire
- **Objectif** : Migrer les emails depuis un serveur IMAP/POP3

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Type de service source (cards radio)
- Paramètres serveur
- Identifiants source
- Boutons Annuler / Migrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x440)
- **OdsCard** : Cards radio IMAP/POP3
- **OdsInput** : Serveur, port, email, mot de passe
- **OdsCheckbox** : SSL/TLS
- **OdsButton** : Annuler, Migrer

## 4. TITRE
"Migrer depuis un autre service"

## 5. TYPE DE SERVICE SOURCE
| Option | Style | Description |
|--------|-------|-------------|
| IMAP | Card sélectionnée (bordure bleue) | Serveur mail standard |
| POP3 | Card non sélectionnée | Téléchargement local |

## 6. CHAMPS DU FORMULAIRE
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Serveur source | Input | Oui (*) | imap.ancien-provider.com |
| Port | Input number | Oui | 993 |
| SSL/TLS | Checkbox | - | ☑ coché |
| Email | Input | Oui (*) | ancien@email.com |
| Mot de passe | Password | Oui (*) | ••••••••••• |

## 7. PORTS PAR DÉFAUT
| Protocole | Port SSL | Port standard |
|-----------|----------|---------------|
| IMAP | 993 | 143 |
| POP3 | 995 | 110 |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Migrer | Primaire bleu | Lance la migration |

## 9. APIS & ENDPOINTS
- POST /email/pro/{service}/account/{email}/migrate : Lancer migration
  - protocol: "imap" | "pop3"
  - server: string
  - port: number
  - ssl: boolean
  - sourceEmail: string
  - sourcePassword: string
