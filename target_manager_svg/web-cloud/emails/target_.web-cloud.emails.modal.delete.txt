# STRUCTURE MODAL

- Dimensions: 500x380px
- Type: Modal de confirmation dangereuse
- Icône: Warning rouge
- Champ confirmation: Saisie "SUPPRIMER"

# ÉLÉMENTS

| Élément          | Description                           |
|------------------|---------------------------------------|
| Icône warning    | Cercle rouge avec ⚠                   |
| Titre            | "Supprimer le compte ?"               |
| Email cible      | Adresse du compte à supprimer         |
| Warning box      | Conséquences irréversibles            |
| Champ confirm    | Input pour taper "SUPPRIMER"          |
| Bouton Annuler   | Ferme la modal                        |
| Bouton Supprimer | Désactivé jusqu'à confirmation        |

# CONSÉQUENCES AFFICHÉES

- Tous les emails seront définitivement supprimés
- Les redirections vers ce compte seront supprimées
- Les alias associés seront supprimés
- La licence restera disponible pour un autre compte

# INTERACTIONS

- Saisie "SUPPRIMER" → Active le bouton
- Clic "Supprimer" → Exécute suppression
- Clic "Annuler" → Ferme modal
- Clic × → Ferme modal

# VALIDATION

Le bouton "Supprimer" ne devient actif que si :
- Le champ contient exactement "SUPPRIMER" (case sensitive)

# API ENDPOINTS

DELETE /email/domains/{domain}/accounts/{account}

## Réponse succès
```json
{
  "taskId": "T-2025123001",
  "status": "pending",
  "message": "Suppression en cours"
}
```
