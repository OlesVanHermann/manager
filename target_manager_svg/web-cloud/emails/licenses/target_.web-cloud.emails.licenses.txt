STRUCTURE PAGE
1. Left panel avec liste domaines
2. Right panel header (domaine)
3. NAV3: 8 tabs (Licences actif)
4. NAV4: Packs (actif) | A la carte | Historique
5. Toolbar: Ajouter pack, Modifier allocation
6. Cards packs actifs
   - Header colore par offre
   - Jauge licences utilisees
   - Stockage consomme
   - Date renouvellement
7. Resume global des licences
8. Banniere upsell

API ENDPOINTS
GET  /email/domains/{domain}/packs                -> Liste packs
POST /email/domains/{domain}/packs                -> Commander pack
GET  /email/domains/{domain}/packs/{id}           -> Details pack
PUT  /email/domains/{domain}/packs/{id}           -> Modifier pack
GET  /email/domains/{domain}/licenses/cost        -> Cout total
GET  /email/domains/{domain}/licenses/available   -> Licences dispo
