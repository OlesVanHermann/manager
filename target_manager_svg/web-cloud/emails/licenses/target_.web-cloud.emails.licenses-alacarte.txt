# INTERACTIONS

- Clic "Commander" (offre) → Modal confirmation + paiement
- Clic "Ajouter" (espace) → Modal sélection compte + quota
- Clic "Activer" (option) → Modal configuration option

# API ENDPOINTS

GET  /email/offers                    → Liste offres disponibles
GET  /email/offers/{offer}/prices     → Prix par offre
POST /email/domains/{domain}/order    → Commander licence
GET  /email/options                   → Liste options
POST /email/domains/{d}/options/{opt} → Activer option
GET  /email/quota/upgrades            → Options d'upgrade quota
