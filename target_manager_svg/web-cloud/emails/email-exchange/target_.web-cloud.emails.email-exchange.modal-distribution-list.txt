# Cahier des Charges : web-cloud/email-exchange - Modal Créer une liste de distribution

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'une liste de distribution (mailing list)
- **Type** : Formulaire de création
- **Objectif** : Créer un groupe de distribution Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire avec nom, email, options
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsInput** : Nom, partie locale email, réponse
- **OdsSelect** : Domaine
- **OdsCheckbox** : Options (modération, externes)
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer une liste de distribution"

## 5. CHAMPS
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Nom de la liste | Input text | Oui (*) | Ex: Tous les commerciaux |
| Adresse e-mail | Input + Select | Oui (*) | commerciaux@example.com |
| Adresse de réponse | Input email | Non | responsable@example.com (optionnel) |

## 6. CHAMP ADRESSE EMAIL
Composé de :
- Input texte pour la partie locale (placeholder: "commerciaux")
- Séparateur "@"
- Select dropdown pour le domaine (ex: example.com)

## 7. OPTIONS
| Option | État par défaut | Description |
|--------|-----------------|-------------|
| Modération des messages | ☐ | Valider avant envoi |
| Autoriser les expéditeurs externes | ☑ | Accepter les emails hors organisation |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée la liste |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/mailingList : Créer liste
- GET /email/exchange/{org}/service/{service}/domain : Lister domaines
