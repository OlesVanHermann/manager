# Cahier des Charges : web-cloud/email-exchange - Modal Créer un compte partagé

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un compte partagé (shared mailbox)
- **Type** : Formulaire de création
- **Objectif** : Ajouter une boîte mail partagée entre plusieurs utilisateurs

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Adresse email
- Nom affiché
- Prénom / Nom
- Quota
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsInput** : Préfixe email, Nom affiché, Prénom, Nom, Quota
- **OdsSelect** : Domaine
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un compte partagé"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Obligatoire |
|-------|------|-------------|-------------|
| Préfixe email * | Input text | partage | Oui |
| @ | Texte fixe | @ | - |
| Domaine | Select | example.com | Oui |
| Nom affiché * | Input text | Boîte partagée Marketing | Oui |
| Prénom | Input text | - | Non |
| Nom | Input text | - | Non |
| Quota (Go) | Input number | 50 | Non |

## 6. LIMITE QUOTA
| Élément | Valeur |
|---------|--------|
| Quota par défaut | 50 Go |
| Quota maximum | 300 Go |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Créer | Primaire bleu | Crée le compte partagé |

## 8. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/domain : Liste des domaines
- POST /email/exchange/{org}/service/{service}/sharedAccount : Créer compte partagé
