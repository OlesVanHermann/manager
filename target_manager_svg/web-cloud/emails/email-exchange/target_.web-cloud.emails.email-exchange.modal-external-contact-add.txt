# Cahier des Charges : web-cloud/email-exchange - Modal Créer un contact externe

## 1. VUE D'ENSEMBLE
- **Modal** : Création d'un contact externe
- **Type** : Formulaire de création
- **Objectif** : Ajouter un contact externe à l'annuaire Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Formulaire avec informations du contact
- Option visibilité annuaire
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsInput** : Champs texte
- **OdsCheckbox** : Cacher dans l'annuaire
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un contact externe"

## 5. CHAMPS
| Champ | Type | Obligatoire | Placeholder |
|-------|------|-------------|-------------|
| Adresse e-mail externe | Input email | Oui (*) | contact@external-company.com |
| Nom affiché | Input text | Oui (*) | John Smith (External Company) |
| Prénom | Input text | Non | John |
| Nom | Input text | Non | Smith |
| Organisation | Input text | Non | External Company Inc. |

## 6. DISPOSITION DES CHAMPS
### Ligne 1
| Adresse e-mail externe (445px) |

### Ligne 2
| Nom affiché (445px) |

### Ligne 3
| Prénom (210px) | Nom (210px) |

### Ligne 4
| Organisation (445px) |

## 7. OPTIONS
☐ "Cacher dans l'annuaire"

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Créer | Primaire bleu | Crée le contact |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/externalContact : Créer contact externe
