# Cahier des Charges : web-cloud/email-exchange - Modal Appareils connectés

## 1. VUE D'ENSEMBLE
- **Modal** : Liste des appareils synchronisés ActiveSync
- **Type** : Modal liste avec actions
- **Objectif** : Gérer les appareils connectés au compte

## 2. STRUCTURE MODAL
- Header avec titre (email) et bouton fermer (×)
- Bandeau info explicatif
- Tableau des appareils
- Compteur et bouton fermer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (650x440)
- **OdsMessage** : Bandeau info bleu
- **OdsTable** : Liste des appareils
- **OdsButton** : Effacer (danger), Fermer

## 4. TITRE
"Appareils connectés - john.doe@example.com"

## 5. BANDEAU INFO
Fond bleu (#EFF6FF) :
"Appareils synchronisés via ActiveSync. Vous pouvez les bloquer ou les effacer à distance."

## 6. TABLEAU
| Colonne | Description |
|---------|-------------|
| Appareil | Nom/modèle de l'appareil |
| Type | Système d'exploitation |
| Dernière sync | Temps écoulé depuis la dernière synchronisation |
| Actions | Bouton "Effacer" (rouge) |

## 7. EXEMPLES DE DONNÉES
| Appareil | Type | Dernière sync |
|----------|------|---------------|
| iPhone 15 Pro | iOS 17.2 | Il y a 5 min |
| MacBook Pro | macOS 14.2 | Il y a 1 heure |
| Outlook Windows | Windows 11 | Il y a 3 heures |

## 8. BOUTON EFFACER
| Élément | Valeur |
|---------|--------|
| Label | "Effacer" |
| Couleur | Rouge (#FEF2F2 fond, #DC2626 texte) |
| Action | Efface à distance les données Exchange |

## 9. COMPTEUR
"3 appareils connectés"

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/device : Lister appareils
- DELETE /email/exchange/{org}/service/{service}/account/{email}/device/{deviceId}/wipe : Effacer appareil
