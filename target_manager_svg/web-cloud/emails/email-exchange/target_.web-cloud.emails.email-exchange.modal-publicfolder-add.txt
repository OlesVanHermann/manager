# Cahier des Charges : web-cloud/email-exchange - Modal Créer un dossier public

## 1. VUE D'ENSEMBLE
- **Modal** : Créer un dossier public Exchange
- **Type** : Formulaire de création
- **Objectif** : Ajouter un dossier partagé accessible à tous

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Nom du dossier
- Type de dossier
- Adresse email du dossier
- Quota
- Boutons Annuler / Créer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x420)
- **OdsInput** : Nom du dossier
- **OdsRadio** : Type de dossier
- **OdsInput** : Préfixe email
- **OdsSelect** : Domaine
- **OdsInput** : Quota (number)
- **OdsButton** : Annuler, Créer

## 4. TITRE
"Créer un dossier public"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | Placeholder | Obligatoire |
|-------|------|-------------|-------------|
| Nom du dossier * | Input text | Documents partagés | Oui |

## 6. TYPE DE DOSSIER
| Option | État | Description |
|--------|------|-------------|
| Dossier mail | ● Sélectionné | Dossier pour emails |
| Dossier calendrier | ○ | Calendrier partagé |
| Dossier contacts | ○ | Carnet d'adresses partagé |

## 7. ADRESSE EMAIL DU DOSSIER
| Partie | Type | Valeur exemple |
|--------|------|----------------|
| Préfixe | Input text | documents |
| @ | Texte fixe | @ |
| Domaine | Select | example.com |

## 8. QUOTA
| Champ | Type | Valeur | Unité |
|-------|------|--------|-------|
| Quota | Input number | 10 | Go |

## 9. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Créer | Primaire bleu | Crée le dossier public |

## 10. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/domain : Liste des domaines
- POST /email/exchange/{org}/service/{service}/publicFolder : Créer dossier public
