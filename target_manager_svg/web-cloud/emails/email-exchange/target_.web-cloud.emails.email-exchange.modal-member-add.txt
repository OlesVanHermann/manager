# Cahier des Charges : web-cloud/email-exchange - Modal Ajouter un membre

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un membre à un groupe
- **Type** : Formulaire de sélection
- **Objectif** : Ajouter un utilisateur comme membre d'un groupe de distribution

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Groupe concerné (readonly)
- Sélection du membre
- Type de membre
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x320)
- **OdsInput** : Groupe (readonly, fond gris)
- **OdsSelect** : Liste des utilisateurs
- **OdsSelect** : Type de membre
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un membre"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur exemple |
|-------|------|------|----------------|
| Groupe | Input readonly | Grisé | marketing@example.com |
| Sélectionner un membre * | Select | Requis | user@example.com |
| Type | Select | Optionnel | Compte Exchange |

## 6. OPTIONS TYPE DE MEMBRE
| Option | Description |
|--------|-------------|
| Compte Exchange | Compte interne Exchange |
| Contact externe | Contact hors organisation |
| Groupe de distribution | Autre groupe |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Ajouter | Primaire bleu | Ajoute le membre |

## 8. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account : Liste des comptes
- GET /email/exchange/{org}/service/{service}/externalContact : Liste contacts externes
- POST /email/exchange/{org}/service/{service}/mailingList/{mailingList}/member/account/{email} : Ajouter membre
