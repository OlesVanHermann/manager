# Cahier des Charges : web-cloud/email-exchange - Modal Ajouter un gestionnaire

## 1. VUE D'ENSEMBLE
- **Modal** : Ajouter un gestionnaire de groupe
- **Type** : Formulaire de sélection
- **Objectif** : Désigner un utilisateur comme gestionnaire d'un groupe

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Groupe concerné (readonly)
- Sélection du gestionnaire
- Note d'information
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x290)
- **OdsInput** : Groupe (readonly, fond gris)
- **OdsSelect** : Liste des utilisateurs disponibles
- **OdsText** : Note explicative
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un gestionnaire"

## 5. CHAMPS DU FORMULAIRE
| Champ | Type | État | Valeur exemple |
|-------|------|------|----------------|
| Groupe | Input readonly | Grisé | marketing@example.com |
| Sélectionner un gestionnaire * | Select | Requis | admin@example.com |

## 6. NOTE D'INFORMATION
"Le gestionnaire pourra ajouter/supprimer des membres du groupe."

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Ajouter | Primaire bleu | Ajoute le gestionnaire |

## 8. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account : Liste des comptes disponibles
- POST /email/exchange/{org}/service/{service}/mailingList/{mailingList}/manager/{email} : Ajouter gestionnaire
