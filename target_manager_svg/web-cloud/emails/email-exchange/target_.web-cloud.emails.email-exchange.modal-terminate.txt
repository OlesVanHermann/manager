# Cahier des Charges : web-cloud/email-exchange - Modal Résilier le service

## 1. VUE D'ENSEMBLE
- **Modal** : Résilier le service Exchange
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer définitivement le service et toutes ses données

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Avertissement danger
- Informations du service
- Confirmation checkbox
- Confirmation texte
- Boutons Annuler / Résilier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x360)
- **OdsMessage** : Avertissement rouge
- **OdsCard** : Info service
- **OdsCheckbox** : Confirmation irréversible
- **OdsInput** : Confirmation texte
- **OdsButton** : Annuler, Résilier (danger)

## 4. TITRE
"Résilier le service"

## 5. AVERTISSEMENT
Fond rouge (#FEF2F2) avec icône :
```
⚠ Attention : action irréversible
Tous les comptes, données et configurations
seront définitivement supprimés.
```

## 6. INFORMATIONS DU SERVICE
| Élément | Valeur exemple |
|---------|----------------|
| Service | Exchange Hosted - exchange-123456 |
| Détails | 5 comptes • 250 Go utilisés |

## 7. CONFIRMATIONS
| Type | Texte |
|------|-------|
| Checkbox | ☐ "Je comprends que cette action est irréversible" |
| Input | Tapez "RESILIER" pour confirmer |

## 8. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris | Ferme la modal |
| Résilier | Danger | Rouge (#DC2626) | Résilie le service |

## 9. VALIDATION
- Checkbox doit être cochée
- Le texte "RESILIER" doit correspondre exactement

## 10. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/terminate : Résilier service
