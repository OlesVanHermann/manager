# Cahier des Charges : web-cloud/email-exchange - Modal Commander des comptes

## 1. VUE D'ENSEMBLE
- **Modal** : Commander des comptes Exchange supplémentaires
- **Type** : Formulaire de commande
- **Objectif** : Acheter des licences Exchange additionnelles

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Sélection type de compte (Standard/Basic)
- Nombre de comptes
- Durée d'engagement
- Récapitulatif avec prix
- Note
- Boutons Annuler / Commander

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (550x490)
- **OdsRadio** : Type de compte (cartes radio)
- **OdsInput** : Quantité (number avec stepper)
- **OdsSelect** : Durée d'engagement
- **OdsCard** : Récapitulatif
- **OdsText** : Note
- **OdsButton** : Annuler, Commander

## 4. TITRE
"Commander des comptes Exchange"

## 5. TYPES DE COMPTE
| Type | Sélection | Description | Prix |
|------|-----------|-------------|------|
| Standard | ● (sélectionné) | 50 Go, Exchange complet | 4,00 €/mois HT |
| Basic | ○ | 5 Go, webmail uniquement | 1,00 €/mois HT |

## 6. QUANTITÉ ET DURÉE
| Champ | Type | Valeur exemple |
|-------|------|----------------|
| Nombre de comptes | Number stepper | 5 |
| Durée d'engagement | Select | 12 mois |

## 7. OPTIONS DURÉE
| Durée | Description |
|-------|-------------|
| 1 mois | Sans engagement |
| 12 mois | Engagement annuel |
| 24 mois | Engagement 2 ans |

## 8. RÉCAPITULATIF
| Élément | Détail | Prix |
|---------|--------|------|
| 5 x Compte Standard | - | 20,00 €/mois |
| **Total HT** | - | **20,00 €/mois** |

## 9. NOTE
"En cliquant sur Commander, vous serez redirigé vers le bon de commande."

## 10. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Commander | Primaire bleu | Redirige vers commande |

## 11. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/orderableAccount : Types de comptes disponibles
- POST /order/email/exchange/{org}/service/{service}/account : Créer bon de commande
