# Cahier des Charges : web-cloud/email-exchange - Modal Activer MFA

## 1. VUE D'ENSEMBLE
- **Modal** : Activation de l'authentification multi-facteurs
- **Type** : Formulaire de configuration
- **Objectif** : Activer MFA sur un compte Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Information du compte
- Bandeau explicatif
- Choix de la méthode
- Boutons Annuler / Activer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x340)
- **OdsMessage** : Bandeau info bleu
- **OdsRadio** : Choix méthode d'authentification
- **OdsButton** : Annuler, Activer

## 4. TITRE
"Activer l'authentification MFA"

## 5. INFORMATION COMPTE
| Champ | Valeur |
|-------|--------|
| Compte | admin@example.com |

## 6. BANDEAU INFO
Fond bleu (#EFF6FF) :
"L'authentification multi-facteurs (MFA) ajoute une couche de sécurité supplémentaire à votre compte Exchange. Un code sera demandé lors de la connexion OWA."

## 7. MÉTHODES D'AUTHENTIFICATION
| Méthode | Sélectionné | Description |
|---------|-------------|-------------|
| Application d'authentification (TOTP) | ● (par défaut) | Google Authenticator, etc. |
| SMS | ○ | Code envoyé par SMS |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire (bordure) | Ferme la modal |
| Activer | Primaire bleu | Active le MFA |

## 9. STYLES RADIO
| État | Apparence |
|------|-----------|
| Sélectionné | Cercle bleu (#0050D7) avec point central |
| Non sélectionné | Cercle vide avec bordure grise |

## 10. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/mfa : Activer MFA
- GET /email/exchange/{org}/service/{service}/account/{email}/mfa : Statut MFA
- DELETE /email/exchange/{org}/service/{service}/account/{email}/mfa : Désactiver MFA
