# Cahier des Charges : web-cloud/email-exchange - Modal Ajouter une délégation

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'une délégation de compte
- **Type** : Formulaire de création
- **Objectif** : Accorder des permissions à un autre compte

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte source (lecture seule)
- Sélection du délégué
- Type de permission (radio)
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x360)
- **OdsInput** : Compte (disabled)
- **OdsSelect** : Déléguer à
- **OdsRadio** : Type de permission
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter une délégation"

## 5. CHAMPS
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Compte | Input disabled | - |
| Déléguer à | Select dropdown | Oui (*) |
| Type de permission | Radio | Oui |

## 6. TYPES DE PERMISSION
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| Accès complet | ● (par défaut) | Accès total à la boîte mail |
| Envoyer en tant que | ○ | Peut envoyer comme ce compte |

## 7. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Ajouter | Primaire bleu | Crée la délégation |

## 8. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/fullAccess : Ajouter accès complet
- POST /email/exchange/{org}/service/{service}/account/{email}/sendAs : Ajouter "envoyer en tant que"
- GET /email/exchange/{org}/service/{service}/account : Liste des comptes pour le select
