# Cahier des Charges : web-cloud/email-exchange - Modal Export PST

## 1. VUE D'ENSEMBLE
- **Modal** : Export de boîte mail
- **Type** : Formulaire d'export
- **Objectif** : Exporter les données d'un compte en PST ou MBOX

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (lecture seule)
- Format d'export (radio)
- Avertissement durée
- Boutons Annuler / Exporter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x320)
- **OdsInput** : Compte (disabled)
- **OdsRadio** : Format d'export
- **OdsMessage** : Warning jaune
- **OdsButton** : Annuler, Exporter

## 4. TITRE
"Exporter la boîte mail"

## 5. COMPTE
Affiché en champ grisé : "john.doe@example.com"

## 6. FORMAT D'EXPORT
| Option | Sélectionné | Description |
|--------|-------------|-------------|
| PST (Microsoft Outlook) | ● (par défaut) | Format natif Outlook |
| MBOX (Standard) | ○ | Format universel |

## 7. AVERTISSEMENT
Fond jaune (#FEF3C7) :
"L'export peut prendre plusieurs heures selon la taille de la boîte."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Exporter | Primaire bleu | Lance l'export |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/export : Lancer export
