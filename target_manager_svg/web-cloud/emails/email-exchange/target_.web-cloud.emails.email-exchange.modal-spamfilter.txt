# Cahier des Charges : web-cloud/email-exchange - Modal Filtre anti-spam

## 1. VUE D'ENSEMBLE
- **Modal** : Configurer le filtre anti-spam
- **Type** : Formulaire de paramètres
- **Objectif** : Ajuster le niveau de filtrage spam pour un compte

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte concerné (readonly)
- Niveau de filtrage
- Action sur les spams
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (500x440)
- **OdsInput** : Compte (readonly)
- **OdsRadio** : Niveaux de filtrage
- **OdsSelect** : Action sur spam
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Filtre anti-spam"

## 5. COMPTE CONCERNÉ
| Champ | État | Valeur exemple |
|-------|------|----------------|
| Compte | Readonly grisé | john.doe@example.com |

## 6. NIVEAUX DE FILTRAGE
| Option | État | Description |
|--------|------|-------------|
| Désactivé | ○ | Aucun filtrage |
| Faible | ○ | Moins de faux positifs |
| Normal | ● Sélectionné | Équilibré (recommandé) |
| Agressif | ○ | Plus de faux positifs possibles |

## 7. ACTION SUR LES SPAMS
| Option | Description |
|--------|-------------|
| Déplacer dans le dossier Spam | Dossier spam (défaut) |
| Marquer comme spam | Préfixe [SPAM] |
| Supprimer | Suppression directe |

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme sans sauvegarder |
| Enregistrer | Primaire bleu | Sauvegarde la configuration |

## 9. APIS & ENDPOINTS
- GET /email/exchange/{org}/service/{service}/account/{email}/spamFilter : Récupérer config
- PUT /email/exchange/{org}/service/{service}/account/{email}/spamFilter : Modifier config
