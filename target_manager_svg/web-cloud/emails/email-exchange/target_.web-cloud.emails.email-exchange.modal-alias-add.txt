# Cahier des Charges : web-cloud/email-exchange - Modal Ajouter un alias

## 1. VUE D'ENSEMBLE
- **Modal** : Ajout d'un alias email
- **Type** : Formulaire de création
- **Objectif** : Ajouter une adresse alias à un compte Exchange

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Compte principal (lecture seule)
- Champ alias (partie locale + domaine)
- Message informatif
- Boutons Annuler / Ajouter

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (450x290)
- **OdsInput** : Compte (disabled), partie locale alias
- **OdsSelect** : Domaine
- **OdsButton** : Annuler, Ajouter

## 4. TITRE
"Ajouter un alias"

## 5. CHAMPS
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Compte | Input disabled | - |
| Adresse alias | Input + Select | Oui (*) |

## 6. CHAMP ADRESSE ALIAS
Composé de :
- Input texte pour la partie locale (placeholder: "jdoe")
- Séparateur "@"
- Select dropdown pour le domaine (ex: example.com)

## 7. MESSAGE INFORMATIF
"Les e-mails envoyés à l'alias seront reçus par le compte principal."

## 8. BOUTONS
| Bouton | Type | Action |
|--------|------|--------|
| Annuler | Secondaire | Ferme la modal |
| Ajouter | Primaire bleu | Crée l'alias |

## 9. APIS & ENDPOINTS
- POST /email/exchange/{org}/service/{service}/account/{email}/alias : Créer alias
- GET /email/exchange/{org}/service/{service}/domain : Lister domaines disponibles
