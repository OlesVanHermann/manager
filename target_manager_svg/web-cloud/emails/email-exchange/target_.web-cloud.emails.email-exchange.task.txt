# Cahier des Charges : web-cloud/email-exchange - Tâches récentes

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des tâches en cours et passées
- **Type** : Page liste avec tableau paginé
- **Objectif** : Suivre l'état des opérations sur le service Exchange

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Exchange (menu)
- Titre "Tâches récentes"
- Tableau des tâches
- Pagination

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu Exchange
- **OdsTable** : Liste des tâches
- **OdsBadge** : Statut avec icône
- **OdsPagination** : Navigation pages

## 4. TITRE
"Tâches récentes"

## 5. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Fonction | Type d'opération | Création de compte |
| Statut | État avec icône | En cours, Terminée, Erreur |
| Date de début | Date/heure début | 28/12/2024 10:30 |
| Date de fin | Date/heure fin ou "-" | 28/12/2024 10:31 |

## 6. BADGES STATUT
| Statut | Icône | Couleur | Description |
|--------|-------|---------|-------------|
| En cours | ⏳ | Bleu (#3B82F6) | Tâche active |
| Terminée | ✓ | Vert (#10B981) | Tâche réussie |
| Erreur | ✗ | Rouge (#EF4444) | Tâche échouée |
| Planifiée | ⏳ | Bleu (#3B82F6) | Tâche programmée |
| Annulée | ⊘ | Rouge (#EF4444) | Tâche annulée |

## 7. TYPES DE TÂCHES
| Fonction | Description |
|----------|-------------|
| Création de compte | Nouveau compte email |
| Modification de compte | Mise à jour compte |
| Suppression de compte | Suppression compte |
| Ajout d'alias | Création d'alias |
| Suppression de groupe | Suppression groupe |
| Modification de domaine | Mise à jour domaine |
