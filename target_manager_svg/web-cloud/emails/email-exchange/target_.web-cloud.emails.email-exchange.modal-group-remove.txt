# Cahier des Charges : web-cloud/email-exchange - Modal Supprimer un groupe

## 1. VUE D'ENSEMBLE
- **Modal** : Confirmation de suppression de groupe
- **Type** : Modal de confirmation destructive
- **Objectif** : Supprimer un groupe de distribution Exchange

## 2. STRUCTURE MODAL
- Header rouge (action destructive)
- Question de confirmation
- Groupe concerné
- Information sur les membres
- Boutons Annuler / Supprimer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (350x180)
- **OdsButton** : Annuler (secondaire), Supprimer (danger rouge)

## 4. TITRE
"Supprimer le groupe"

## 5. CONTENU
| Élément | Texte |
|---------|-------|
| Question | "Supprimer ce groupe ?" |
| Groupe | marketing@example.com (gras) |
| Info | "Tous les membres seront retirés du groupe." |

## 6. BOUTONS
| Bouton | Type | Couleur | Action |
|--------|------|---------|--------|
| Annuler | Secondaire | Gris bordure | Ferme la modal |
| Supprimer | Danger | Rouge (#DC2626) | Supprime le groupe |

## 7. APIS & ENDPOINTS
- DELETE /email/exchange/{org}/service/{service}/mailingList/{mailingList} : Supprimer groupe
