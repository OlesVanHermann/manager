# NIVEAUX PROTECTION ANTISPAM

| Niveau     | Description                              |
|------------|------------------------------------------|
| Désactivé  | Pas de filtrage antispam                 |
| Faible     | Filtrage minimal, peu de faux positifs   |
| Modéré     | Équilibre filtrage/faux positifs         |
| Strict     | Filtrage agressif, plus de faux positifs |

# COLONNES TABLE RÈGLES

|------------|---------|--------------------------------|

| Actions    | 150px   | ✎ Modifier, × Supprimer        |

# INTERACTIONS

- Clic niveau protection → Modal changement niveau
- Clic "+ Ajouter règle" → Modal création règle
- Clic "Voir quarantaine" → Liste emails en quarantaine
- Clic ✎ → Modal modifier règle
- Clic × → Modal confirmer suppression

# API ENDPOINTS

GET    /email/domains/{domain}/antispam          → Config antispam
PUT    /email/domains/{domain}/antispam          → Modifier niveau
GET    /email/domains/{domain}/antispam/rules    → Liste règles
POST   /email/domains/{domain}/antispam/rules    → Créer règle
PUT    /email/domains/{d}/antispam/rules/{id}    → Modifier règle
DELETE /email/domains/{d}/antispam/rules/{id}    → Supprimer règle
GET    /email/domains/{domain}/antispam/quarantine → Liste quarantaine
POST   /email/domains/{d}/antispam/quarantine/{id}/release → Libérer email
DELETE /email/domains/{d}/antispam/quarantine/{id} → Supprimer email
