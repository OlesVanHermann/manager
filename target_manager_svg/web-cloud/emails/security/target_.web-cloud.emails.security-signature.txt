# NAV4 - SOUS-TABS SÉCURITÉ

| # | Tab        | Route               | Contenu                    |
|---|------------|---------------------|----------------------------|
| 1 | DNS Config | /security           | MX, SPF, DKIM, DMARC, SRV  |
| 2 | Antispam   | /security-antispam  | Règles, quarantaine        |
| 3 | Signature  | /security-signature | Disclaimer entreprise      |

# CARD SIGNATURE

| Section            | Description                              |
|--------------------|------------------------------------------|
| Toggle activation  | Activer/désactiver signature globale     |
| Position           | Avant message / Après message / Réponses |
| Éditeur WYSIWYG    | Formatage riche (gras, italique, liens)  |
| Variables          | Insertion données utilisateur dynamiques |

# VARIABLES DISPONIBLES

| Variable          | Description                  |
|-------------------|------------------------------|
| %%DISPLAY_NAME%%  | Nom complet affiché          |
| %%FIRST_NAME%%    | Prénom utilisateur           |
| %%LAST_NAME%%     | Nom de famille               |
| %%TITLE%%         | Fonction/Titre               |
| %%DEPARTMENT%%    | Service/Département          |
| %%PHONE%%         | Numéro de téléphone          |
| %%MOBILE%%        | Numéro mobile                |
| %%EMAIL%%         | Adresse email                |

# POSITIONS SIGNATURE

| Position    | Description                                |
|-------------|-------------------------------------------|
| after       | Après le corps du message (défaut)        |
| before      | Avant le corps du message                 |
| reply       | Uniquement sur les réponses               |

# INTERACTIONS

- Toggle ON/OFF → Active/désactive signature
- Sélecteur position → Change emplacement
- Bouton "Variables" → Insère variable dans éditeur
- Bouton "Aperçu" → Modal prévisualisation
- Bouton "Enregistrer" → Sauvegarde signature

# RESTRICTIONS

⚠️ La signature entreprise est disponible UNIQUEMENT pour :
- Exchange (toutes versions)

NON disponible pour :
- Email Pro
- Zimbra
- MX Plan

# API ENDPOINTS

GET  /email/exchange/{org}/{domain}/signature → Config signature
PUT  /email/exchange/{org}/{domain}/signature → Modifier signature
POST /email/exchange/{org}/{domain}/signature/preview → Prévisualisation
