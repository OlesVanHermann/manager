# Cahier des Charges : web-cloud/zimbra - Redirections

## 1. RESUME EXECUTIF

### Perimetre
Onglet de gestion des redirections Zimbra.

### Objectif fonctionnel
- Voir les redirections
- Creer de nouvelles redirections
- Supprimer des redirections

## 4. APIS & DONNEES

### Endpoints
```
GET /zimbra/platform/{platformId}/redirection
POST /zimbra/platform/{platformId}/redirection
DELETE /zimbra/platform/{platformId}/redirection/{redirectionId}
```

### Modele
```typescript
interface ZimbraRedirection {
  id: string;
  from: string;
  to: string;
  organization: string;
  state: 'ok' | 'suspended';
}
```
