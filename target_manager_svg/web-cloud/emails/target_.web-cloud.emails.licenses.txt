## 7. APIS & ENDPOINTS

```
# Liste des services email
GET /email/exchange/{organization}/service
→ Liste des services Exchange

GET /email/pro
→ Liste des services Email Pro

GET /email/zimbra
→ Liste des services Zimbra

GET /email/domain
→ Liste des domaines MX Plan

# Détails service
GET /email/exchange/{organization}/service/{serviceName}
→ Détails (quota, nb comptes, etc.)

# Comptes par service
GET /email/exchange/{organization}/service/{serviceName}/account
GET /email/pro/{service}/account
GET /email/zimbra/{serviceName}/account
GET /email/domain/{domain}/account
→ Liste des comptes

# Informations service (renouvellement)
GET /email/exchange/{organization}/service/{serviceName}/serviceInfos
→ Date expiration, renouvellement auto

# Prix
GET /order/catalog/formatted/exchange
GET /order/catalog/formatted/emailPro
→ Prix des offres
```
