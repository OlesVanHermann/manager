STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section Web Cloud (Emails actif)
3. LEFT PANEL (280px)
   - NAV3 Toggle: [General] [Packs] - General actif
   - Recherche domaine
   - Liste domaines (aucun selectionne)
   - Bouton [+ Ajouter domaine]
4. RIGHT PANEL (1096px)
   - Header: Titre "Emails" + description
   - Banner info: Message de bienvenue
   - Stats cards: 4 cartes avec totaux par offre
   - Actions rapides: 3 cartes liens
   - Activite recente: Table dernieres actions

NOTE: Cette page s'affiche quand AUCUN domaine n'est selectionne.
      Equivalent d'un "hub" ou dashboard section.

STATS CARDS
| Card            | Contenu                              |
|-----------------|--------------------------------------|
| Total comptes   | Nombre total + nb domaines           |
| Exchange        | Nb comptes + stockage total          |
| Email Pro       | Nb comptes + stockage total          |
| MX Plan+Zimbra  | Nb comptes + stockage total          |

ACTIONS RAPIDES
| Action                | Lien vers                    |
|-----------------------|------------------------------|
| Commander licences    | Mode Packs → A la carte      |
| Gerer les packs       | Mode Packs → Mes packs       |
| Configurer securite   | Premier domaine → Securite   |

INTERACTIONS
- Clic domaine left panel → Charge page domaine (accounts)
- Clic NAV3 Packs → Switch vers mode Packs
- Clic action rapide → Navigation vers section
- Clic ligne activite → Navigation vers domaine concerne
- Clic [+ Ajouter domaine] → Modal/workflow ajout domaine

API ENDPOINTS
GET /email/domain → Liste domaines
GET /email/{type}/{service}/account → Comptes par service
GET /email/domain/{domain}/task → Activite recente

Types: exchange, emailpro, mxplan, zimbra
