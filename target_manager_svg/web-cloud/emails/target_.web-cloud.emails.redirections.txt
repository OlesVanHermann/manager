COLONNES TABLE
| Colonne       | Description                              |
|---------------|------------------------------------------|
| Source        | Adresse email source (alias)             |
| Destination   | Adresse(s) de destination                |
| Copie locale  | Garde copie dans boite source            |
| Statut        | Active ou inactive                       |
| Actions       | Edit, supprimer                          |

TYPES DE REDIRECTION
| Type        | Copie locale | Description                    |
|-------------|--------------|--------------------------------|
| Alias       | Oui          | Forward + garde copie locale   |
| Redirection | Non          | Forward seul, pas de copie     |

INTERACTIONS
- Clic NAV4 tab → Navigation vers tab
- Clic [+ Creer redirection] → Modal creation redirection
- Clic edit (✎) → Modal edition redirection
- Clic (...) → Menu contextuel (supprimer)

API ENDPOINTS
GET /email/domain/{domain}/redirection → Liste redirections
POST /email/domain/{domain}/redirection → Creer redirection
DELETE /email/domain/{domain}/redirection/{id} → Supprimer
PUT /email/domain/{domain}/redirection/{id} → Modifier

Note: Exchange utilise /email/exchange/{service}/account/{email}/alias
