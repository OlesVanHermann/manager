STRUCTURE PAGE
1. Left panel
   - Toggle vue: Par domaine (actif) / Par licence
   - Liste domaines avec stats (nb boites, offres)
   - Domain selectionne en surbrillance
2. Right panel header
   - Nom domaine + compteur boites
3. NAV3: 8 tabs (Boites mail actif)
4. Toolbar
   - Bouton "Creer boite" (primary)
   - Bouton "Importer" (outline)
   - Bouton "Exporter" (outline)
5. Table comptes email
   - Colonnes: Email, Offre, Licence, Quota, Statut, Actions
   - Badge couleur par offre
   - Actions contextuelles par offre
6. Pagination

ACTIONS TOOLBAR
| Action    | Comportement                                    |
|-----------|-------------------------------------------------|
| Creer     | Modal creation compte avec choix offre          |
| Importer  | Modal import CSV/migration                      |
| Exporter  | Telecharge CSV liste comptes                    |

API ENDPOINTS
GET  /email/domains/{domain}/accounts           -> Liste comptes
POST /email/domains/{domain}/accounts           -> Creer compte
GET  /email/domains/{domain}/accounts/{id}      -> Details compte
PUT  /email/domains/{domain}/accounts/{id}      -> Modifier compte
DELETE /email/domains/{domain}/accounts/{id}    -> Supprimer compte
