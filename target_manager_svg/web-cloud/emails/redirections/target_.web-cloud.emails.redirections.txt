   - Colonnes: Source, Destination, Type, Statut, Actions

6. Pagination
7. Banniere info types redirections

TYPES DE REDIRECTIONS
| Type      | Couleur   | Badge     | Description                     |
|-----------|-----------|-----------|----------------------------------|
| Locale    | #DBEAFE   | bg-info   | Vers une boite du meme domaine  |
| Externe   | #FEF3C7   | bg-warning| Vers une adresse externe        |
| Multiple  | #E5E7EB   | bg-neutral| Vers plusieurs destinations     |
| Catch-all | #F3E8FF   | violet    | Capture tous les non-attribues  |

ACTIONS PAR LIGNE
| Action    | Comportement                                    |
|-----------|-------------------------------------------------|
| Modifier  | Modal edition source/destination                |
| Suppr     | Confirmation puis suppression                   |

API ENDPOINTS
GET  /email/domains/{domain}/redirections          -> Liste redirections
POST /email/domains/{domain}/redirections          -> Creer redirection
PUT  /email/domains/{domain}/redirections/{id}     -> Modifier
DELETE /email/domains/{domain}/redirections/{id}   -> Supprimer
