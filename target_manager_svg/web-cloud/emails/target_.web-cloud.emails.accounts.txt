   - NAV3 Toggle: [General] [Packs] - General actif
   - Recherche domaine
   - Filtre par offre
   - Liste domaines avec badges offres
4. RIGHT PANEL (1096px)
   - Header: Nom domaine + stats + bouton Webmail
   - NAV4: 7 tabs Mode General (Comptes actif)
   - Toolbar: [+ Creer] [Import] [Export] [Recherche] [Refresh]

NAV3 MODES
| Mode    | Tabs NAV4                                              |
|---------|--------------------------------------------------------|
| General | Comptes, Redirections, Repondeurs, Listes, Securite, Avance, Taches |
| Packs   | Mes packs, A la carte, Historique                      |

ACTIONS PAR LIGNE (selon offre)
| Offre    | Actions disponibles                    |
|----------|----------------------------------------|
| Exchange | edit, password, alias, delegation, mfa |
| Pro      | edit, password, alias, delegation      |
| Zimbra   | edit, password, alias                  |
| MX Plan  | edit, password                         |

INTERACTIONS
- Clic NAV3 toggle → Change mode (General/Packs)
- Clic domaine left panel → Charge comptes du domaine
- Clic NAV4 tab → Navigation vers tab
- Clic [+ Creer] → Modal creation compte
- Clic [Import] → Modal import CSV
- Clic [Export] → Telecharge CSV
- Clic filtre offre → Filtre table
- Clic actions ligne → Actions sur compte

API ENDPOINTS
GET /email/domain → Liste domaines
GET /email/{type}/{service}/account → Liste comptes
POST /email/{type}/{service}/account → Creer compte
PUT /email/{type}/{service}/account/{email} → Modifier compte
DELETE /email/{type}/{service}/account/{email} → Supprimer compte

Types: exchange, emailpro, mxplan, zimbra
