STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Emails actif)
3. LEFT PANEL - Liste domaines
4. RIGHT PANEL
   - Header: Domaine
   - NAV3: 8 tabs (Listes actif)
   - Toolbar: Creer liste, Filtrer
   - Table listes:
     Liste | Type | Membres | Moderateurs | Options | Actions
   - Pagination

DISPONIBILITE PAR OFFRE
| Offre     | Mailing List | Groupe Distribution |
|-----------|--------------|---------------------|
| Exchange  | Non          | Oui                 |
| Email Pro | Oui          | Non                 |
| Zimbra    | Non          | Non                 |
| MX Plan   | Oui          | Non                 |

COLONNES TABLE
| Colonne     | Description                            |
|-------------|----------------------------------------|
| Liste       | Adresse + nom descriptif               |
| Type        | Badge offre (Exchange/Pro/MX Plan)     |
| Membres     | Nombre de membres                      |
| Moderateurs | Nombre de moderateurs                  |
| Options     | Modere/Ouvert, Reply-to, Archivage     |
| Actions     | Membres, Edit, Supprimer               |

API ENDPOINTS
GET    /email/domains/{domain}/lists              - Liste mailing lists
POST   /email/domains/{domain}/lists              - Creer liste
GET    /email/domains/{d}/lists/{l}               - Details liste
PUT    /email/domains/{d}/lists/{l}               - Modifier liste
DELETE /email/domains/{d}/lists/{l}               - Supprimer liste
GET    /email/domains/{d}/lists/{l}/members       - Membres
POST   /email/domains/{d}/lists/{l}/members       - Ajouter membre
DELETE /email/domains/{d}/lists/{l}/members/{m}   - Retirer membre
GET    /email/domains/{d}/lists/{l}/moderators    - Moderateurs
