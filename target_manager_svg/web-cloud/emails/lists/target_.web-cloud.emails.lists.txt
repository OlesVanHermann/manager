STRUCTURE PAGE
1. Left panel avec liste domaines
2. Right panel header (domaine + compteur)
3. NAV3: 8 tabs (Listes actif)
4. Toolbar
   - Bouton "Creer liste" (primary)
   - Recherche
5. Table listes de diffusion
   - Colonnes: Liste, Membres, Moderateurs, Mode, Statut, Actions
6. Banniere info modes

MODES DE LISTE
| Mode      | Couleur   | Qui peut envoyer               |
|-----------|-----------|--------------------------------|
| Ouvert    | #DEF7EC   | Tout le monde (public)         |
| Restreint | #DBEAFE   | Seuls les membres              |
| Annonce   | #E5E7EB   | Seuls les moderateurs          |

DISPONIBILITE PAR OFFRE
| Offre     | Listes    | Features                       |
|-----------|-----------|--------------------------------|
| MX Plan   | Basique   | Listes simples                 |
| Email Pro | Avance    | Moderation, archives           |
| Exchange  | Avance    | Groupes dynamiques, calendriers|
| Zimbra    | Avance    | Listes collaboratives          |

ACTIONS PAR LIGNE
| Action    | Comportement                                    |
|-----------|-------------------------------------------------|
| Membres   | Modal gestion membres (ajouter/retirer)         |
| Params    | Modal parametres (mode, moderation, archivage)  |
| Suppr     | Confirmation puis suppression                   |

API ENDPOINTS
GET  /email/domains/{domain}/lists                -> Liste listes
POST /email/domains/{domain}/lists                -> Creer liste
GET  /email/domains/{domain}/lists/{id}           -> Details
PUT  /email/domains/{domain}/lists/{id}           -> Modifier
DELETE /email/domains/{domain}/lists/{id}         -> Supprimer
GET  /email/domains/{domain}/lists/{id}/members   -> Membres
POST /email/domains/{domain}/lists/{id}/members   -> Ajouter membre
