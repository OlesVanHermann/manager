STRUCTURE MODAL
1. Overlay sombre
2. Container 540x560px centre
3. Header: Creer un repondeur automatique
4. Body:
   - Select compte email
   - Radio duree (Temporaire/Permanent)
   - Date debut + Date fin (si temporaire)
   - Textarea message
   - Info HTML non autorise
   - Checkbox copie messages
5. Footer: Annuler/Creer

CHAMPS FORMULAIRE
| Champ               | Type       | Required |
|---------------------|------------|----------|
| Compte email        | Select     | Oui      |
| Duree               | Radio      | Oui      |
| Date de debut       | Datepicker | Si temp  |
| Date de fin         | Datepicker | Si temp  |
| Message             | Textarea   | Oui      |
| Copie messages      | Checkbox   | Non      |

VALIDATION
- Dates obligatoires si mode temporaire
- Message ne doit pas contenir HTML
- Date fin > Date debut

API ENDPOINT
POST /email/domain/{domain}/responder
Body: { account, from, to, content, copy }
