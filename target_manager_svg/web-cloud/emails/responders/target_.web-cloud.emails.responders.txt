STRUCTURE PAGE
1. Left panel avec liste domaines
2. Right panel header (domaine + compteur)
3. NAV3: 8 tabs (Repondeurs actif)
4. Toolbar: Bouton "Creer repondeur"
5. Liste cards repondeurs
   - Badge statut
   - Email associe
   - Titre + dates
   - Apercu message
   - Actions (Modifier, Activer/Desactiver, Supprimer)
6. Banniere info

STATUTS REPONDEUR
| Statut    | Couleur   | Description                           |
|-----------|-----------|---------------------------------------|
| Actif     | #DEF7EC   | Envoie actuellement des reponses      |
| Planifie  | #FEF3C7   | Programme pour dates futures          |
| Inactif   | #F3F4F6   | Desactive, ne repond pas              |

CONFIGURATION REPONDEUR
| Champ       | Type     | Description                           |
|-------------|----------|---------------------------------------|
| email       | select   | Boite mail associee                   |
| title       | text     | Nom du repondeur (interne)            |
| startDate   | date     | Date debut (optionnel)                |
| endDate     | date     | Date fin (optionnel)                  |
| message     | textarea | Corps du message automatique          |
| copyTo      | email    | Copie vers (optionnel)                |
| frequency   | select   | Une fois/jour/toujours par expediteur |

API ENDPOINTS
GET  /email/domains/{domain}/responders           -> Liste repondeurs
POST /email/domains/{domain}/responders           -> Creer repondeur
GET  /email/domains/{domain}/responders/{id}      -> Details
PUT  /email/domains/{domain}/responders/{id}      -> Modifier
DELETE /email/domains/{domain}/responders/{id}    -> Supprimer
POST /email/domains/{domain}/responders/{id}/toggle -> Activer/Desactiver
