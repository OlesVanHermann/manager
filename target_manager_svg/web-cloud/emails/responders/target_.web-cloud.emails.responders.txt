STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Emails actif)
3. LEFT PANEL - Liste domaines
4. RIGHT PANEL
   - Header: Domaine
   - NAV3: 8 tabs (Repondeurs actif)
   - Toolbar: Configurer repondeur, Filtrer
   - Table repondeurs:
     Compte | Statut | Periode | Message | Actions
   - Pagination

COLONNES TABLE
| Colonne  | Description                               |
|----------|-------------------------------------------|
| Compte   | Adresse email du compte                   |
| Statut   | Actif (vert) / Inactif (gris)             |
| Periode  | Dates debut-fin ou "Permanent"            |
| Message  | Apercu du message de reponse automatique  |
| Actions  | Configurer, Modifier, Desactiver          |

API ENDPOINTS
GET    /email/domains/{d}/accounts/{e}/responder   - Etat repondeur
PUT    /email/domains/{d}/accounts/{e}/responder   - Configurer
DELETE /email/domains/{d}/accounts/{e}/responder   - Desactiver
