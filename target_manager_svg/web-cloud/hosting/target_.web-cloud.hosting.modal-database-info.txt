DONNEES DYNAMIQUES

|------------------------------|-----------------------------------------------|

| Type/Version                 | database → server + version                   |
| Serveur                      | database → server                             |
| Port                         | database → port                               |
| Utilisateur                  | database → user                               |
| Quota                        | database → quotaSize                          |
| Utilisation                  | database → quotaUsed                          |
| Nombre tables                | database → tableCount                         |
| Nombre lignes                | database → rowCount                           |
| Date creation                | database → creationDate                       |
| Derniere sauvegarde          | GET /hosting/web/{sn}/dump (filter)           |

STRUCTURE MODAL
1. Header avec titre
2. Banniere statut (active/inactive)
3. Informations generales (nom, serveur, user, etc.)
4. Section utilisation avec barre progression
5. Chaine de connexion (copiable)
6. Lien acces phpMyAdmin
7. Info sauvegardes
8. Footer

INFORMATIONS GENERALES
| Champ       | Description                         | Copiable |
|-------------|-------------------------------------|----------|
| Nom         | Nom de la base                      | Oui      |
| Type        | MySQL/MariaDB + version             | Non      |
| Serveur     | Hostname du serveur                 | Oui      |
| Port        | Port de connexion (3306)            | Non      |
| Utilisateur | Nom utilisateur BDD                 | Oui      |
| Mot de passe| Masque avec boutons voir/copier     | Oui      |
| Creation    | Date de creation                    | Non      |
| Quota       | Espace alloue                       | Non      |

UTILISATION
- Barre de progression (vert < 70%, orange 70-90%, rouge > 90%)
- Espace utilise / Total
- Nombre de tables
- Nombre approximatif de lignes

CHAINE CONNEXION
Format: mysql://user:PASSWORD@host:port/database
Affichee en style terminal avec bouton copie

ACCES PHPMYADMIN
Bouton ouvrant phpMyAdmin dans nouvel onglet
Connexion automatique via token temporaire

INTERACTIONS
- Boutons copie → Copier dans presse-papier
- Bouton oeil → Afficher/masquer mot de passe
- Bouton phpMyAdmin → Ouvrir interface
- Lien sauvegardes → Voir liste dumps

API ENDPOINTS
GET /hosting/web/{sn}/database/{name}                  → Details BDD
GET /hosting/web/{sn}/dump                             → Sauvegardes
