# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, Logs actif
   - NAV4: Logs bruts | Statistiques (actif) | Utilisateurs
   - FILTRES: Période (dropdown) + Site (dropdown)
   - GRAPHIQUE: Visites et pages vues (ligne)
   - STATS CARDS: Visites, Pages vues, Bande passante, Visiteurs uniques
   - TABLE: Pages les plus visitées

# FILTRES
- Période: 7 derniers jours, 30 derniers jours, 90 derniers jours, Année
- Site: Liste des multisites

# GRAPHIQUE
- Type: Line chart
- Séries: Visites (bleu), Pages vues (vert)
- X-axis: Jours de la semaine
- Y-axis: Nombre de visites/pages

# CARDS STATS
- Total visites + variation %
- Pages vues + variation %
- Bande passante + variation %
- Visiteurs uniques + variation %

# API ENDPOINTS
GET /hosting/web/{serviceName}/statistics
GET /hosting/web/{serviceName}/statistics/pages
GET /hosting/web/{serviceName}/statistics?period={period}
GET /hosting/web/{serviceName}/attachedDomain (pour filtre site)
