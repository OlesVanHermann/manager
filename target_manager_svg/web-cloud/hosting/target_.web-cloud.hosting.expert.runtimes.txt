# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Runtimes actif)
   - TOOLBAR: Refresh | Recherche | + Ajouter environnement
   - TABLE: Défaut, Nom, Langage, Multisites, Statut, Actions
   - PAGINATION
   - BANNIÈRE INFO

# COLONNES TABLE
| Colonne            | Largeur | Contenu                          |
|--------------------|---------|----------------------------------|
| Défaut             | 70px    | Radio button (●/○)               |
| Nom                | 200px   | Identifiant du runtime           |
| Langage            | 190px   | Badge PHP/Node.js + version      |
| Multisites associés| 220px   | Nombre de sites utilisant        |
| Statut             | 120px   | Badge Créé/Suppression           |
| Actions            | 56px    | Menu ⋮                           |

# LANGAGES SUPPORTÉS
| Langage     | Badge couleur | Versions         |
|-------------|---------------|------------------|
| PHP         | Bleu #DBEAFE  | 7.4, 8.0, 8.1, 8.2, 8.3 |
| Node.js     | Vert #DEF7EC  | 16, 18, 20       |

# ÉTATS POSSIBLES
| État        | Couleur badge | Description              |
|-------------|---------------|--------------------------|
| Créé        | Vert #DEF7EC  | Runtime opérationnel     |
| En création | Jaune #FEF3C7 | Création en cours        |
| Suppression | Rouge #FEE2E2 | Suppression en cours     |

# INDICATEUR DÉFAUT
- Radio bleu (#0050D7) = Runtime par défaut
- Radio gris (#E5E7EB) = Autre runtime
- Un seul runtime peut être défaut

# INTERACTIONS
- Clic ↻ → Rafraîchir liste
- Clic "+ Ajouter environnement" → Modal création
- Clic radio ○ → Modal définir par défaut
- Clic ⋮ → Menu (Modifier, Définir par défaut, Supprimer)

# API ENDPOINTS
GET /hosting/web/{sn}/runtime              - Liste runtimes
GET /hosting/web/{sn}/runtime/{id}         - Détails runtime
POST /hosting/web/{sn}/runtime             - Créer runtime
PUT /hosting/web/{sn}/runtime/{id}         - Modifier runtime
PUT /hosting/web/{sn}/runtime/{id}/setDefault - Définir par défaut
DELETE /hosting/web/{sn}/runtime/{id}      - Supprimer runtime
