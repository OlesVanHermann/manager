# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Multisite actif)
   - TOOLBAR: + Ajouter domaine | Régénérer SSL | Recherche
   - TABLE: Domaine, Dossier, SSL, CDN, Firewall, Logs, Runtime, Git, Diagnostic, Actions
   - PAGINATION

# COLONNES TABLE
| Colonne      | Largeur | Contenu                        |
|--------------|---------|--------------------------------|
| Domaine      | 200px   | Lien vers config domaine       |
| Dossier      | 140px   | Chemin relatif                 |
| SSL          | 70px    | Badge Actif/Inactif            |
| CDN          | 70px    | Badge Actif/Désactivé          |
| Firewall     | 80px    | Badge Actif/Inactif            |
| Logs séparés | 100px   | Badge Actif/Désactivé          |
| Runtime      | 100px   | Nom runtime (php-8.x)          |
| Git          | 70px    | Badge Oui/Non                  |
| Diagnostic   | 80px    | Indicateur ● vert/orange/rouge |
| Actions      | 56px    | Menu ⋮                         |

# INTERACTIONS
- Clic "Ajouter domaine" → Modal ajout domaine OVH/externe
- Clic "Régénérer SSL" → Modal confirmation régénération
- Clic domaine → Détails domaine
- Clic ⋮ → Menu contextuel (Modifier, Logs, Purger CDN, Git, Détacher)

# API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain              - Liste domaines
GET /hosting/web/{sn}/attachedDomain/{domain}     - Détails domaine
POST /hosting/web/{sn}/attachedDomain             - Ajouter domaine
PUT /hosting/web/{sn}/attachedDomain/{domain}     - Modifier domaine
DELETE /hosting/web/{sn}/attachedDomain/{domain}  - Détacher domaine
POST /hosting/web/{sn}/ssl/regenerate             - Régénérer SSL
