STRUCTURE MODAL
1. Header avec titre
2. Info domaine + date analyse
3. Liste checks avec statut
4. Footer avec boutons

CHECKS DIAGNOSTIC
| Check             | Description                          |
|-------------------|--------------------------------------|
| DNS               | Enregistrements A/AAAA/CNAME         |
| Certificat SSL    | Validite et expiration               |
| Reponse HTTP      | Temps de reponse + code              |
| Dossier racine    | Existence + fichier index            |
| Configuration PHP | Version + parametres                 |
| Firewall WAF      | Statut + attaques bloquees           |

ETATS POSSIBLES
| Etat      | Couleur | Icone | Description           |
|-----------|---------|-------|-----------------------|
| OK        | Vert    | ✓     | Tout est correct      |
| Attention | Orange  | !     | Amelioration possible |
| Erreur    | Rouge   | ✗     | Action requise        |

CODES COULEUR
| Element          | Hex       |
|------------------|-----------|
| Succes bg        | #DEF7EC   |
| Succes text      | #059669   |
| Warning bg       | #FEF3C7   |
| Warning text     | #D97706   |
| Error bg         | #FEE2E2   |
| Error text       | #DC2626   |

BOUTONS FOOTER
| Bouton           | Action                    |
|------------------|---------------------------|
| Relancer analyse | Nouveau diagnostic        |
| Fermer           | Fermer modal              |

INTERACTIONS
- Clic Relancer → POST diagnostic
- Clic check erreur → Details + solution

API ENDPOINTS
POST /hosting/web/{sn}/attachedDomain/{domain}/diagnostic → Lancer analyse
GET /hosting/web/{sn}/attachedDomain/{domain}/diagnostic  → Dernier resultat
