# STRUCTURE MODAL
1. OVERLAY sombre sur fond
2. MODAL plein écran (1200x800px)
   - HEADER: Titre + bouton fermer
   - STEPPER: 2 étapes (Action, Configuration)
   - OPTIONS RADIO: Restaurer / Modifier
   - FORMULAIRE: 5 selects PHP
   - PANNEAU AIDE: Explications
   - WARNING: Avertissement impact
   - FOOTER: Annuler / Suivant

# STEPPER
| Étape | Label         | État        |
|-------|---------------|-------------|
| 1     | Action        | Actif (bleu)|
| 2     | Configuration | Grisé       |

# OPTIONS RADIO
| Option                              | Description                    |
|-------------------------------------|--------------------------------|
| Restaurer config précédente         | Rollback vers historique       |
| Modifier configuration (sélectionné)| Personnaliser paramètres PHP   |

# FORMULAIRE CONFIGURATION
| Paramètre               | Valeur défaut | Options                  |
|-------------------------|---------------|--------------------------|
| Version PHP             | 8.2           | 7.4, 8.0, 8.1, 8.2, 8.3  |
| Environnement exécution | stable64      | stable64, legacy         |
| Moteur                  | php           | php, phpcgi              |
| Environnement           | production    | production, development  |
| Firewall HTTP           | none          | none, security           |

# PANNEAU AIDE
- Versions PHP recommandées
- Différences stable64 vs legacy
- Utilité du firewall HTTP

# INTERACTIONS
- Clic "✕" → Ferme modal
- Clic radio → Sélection option
- Clic select → Dropdown valeurs
- Clic "Annuler" → Ferme modal
- Clic "Suivant" → Étape 2 / Confirmation

# API ENDPOINTS
GET /hosting/web/{sn}/ovhConfig              - Config actuelle
PUT /hosting/web/{sn}/ovhConfig              - Modifier config
GET /hosting/web/{sn}/ovhConfig/histories    - Historique configs
POST /hosting/web/{sn}/ovhConfig/rollback    - Restaurer version
