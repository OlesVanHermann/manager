TYPES DE PURGE
| Type       | Description                           |
|------------|---------------------------------------|
| Complete   | Vider tout le cache du domaine        |
| Selective  | Purger des URLs specifiques           |

QUOTA
- 5 purges par jour par domaine
- Reset a minuit UTC

INTERACTIONS
- Clic card type → Choisir type de purge
- Dropdown domaine → Selectionner domaine
- Clic Purger → Lancer la purge

API ENDPOINTS
GET /hosting/web/{sn}/cdn                         → Info CDN
GET /hosting/web/{sn}/cdn/domain                  → Domaines CDN
POST /hosting/web/{sn}/cdn/flush                  → Purger cache
