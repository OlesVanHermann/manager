STRUCTURE MODAL
1. Header avec titre
2. Affichage domaine concerne
3. Runtime actuel (lecture seule)
4. Selection nouveau runtime (cartes)
5. Lien creer runtime
6. Banniere info
7. Footer avec boutons

CARTES RUNTIME
| Element        | Description                    |
|----------------|--------------------------------|
| Nom            | Header de la carte             |
| Technologie    | PHP/Node.js/Python + version   |
| Chemin         | Application path               |
| Badge Actuel   | Si runtime actif               |

TYPES RUNTIME DISPONIBLES
| Type     | Versions supportees          |
|----------|------------------------------|
| PHP      | 7.4, 8.0, 8.1, 8.2, 8.3      |
| Node.js  | 16, 18, 20                   |
| Python   | 3.9, 3.10, 3.11              |

BOUTONS FOOTER
| Bouton   | Action                        |
|----------|-------------------------------|
| Annuler  | Fermer sans modifier          |
| Associer | Appliquer nouveau runtime     |

INTERACTIONS
- Clic carte → Selectionner runtime
- Clic "Creer un runtime" → Modal creation
- Clic Associer → PUT runtime

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}     → Info domaine
GET /hosting/web/{sn}/runtime                     → Liste runtimes
PUT /hosting/web/{sn}/attachedDomain/{domain}     → Modifier runtime
