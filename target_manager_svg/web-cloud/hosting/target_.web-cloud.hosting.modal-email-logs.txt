STRUCTURE MODAL (large)
1. Header avec titre
2. Barre de filtres (dates, statut, recherche)
3. Résumé statistiques (4 cartes)
4. Tableau des logs paginé
5. Pagination
6. Footer

FILTRES DISPONIBLES
| Filtre      | Type       | Options                        |
|-------------|------------|--------------------------------|
| Période     | Date range | Sélecteur dates                |
| Statut      | Dropdown   | Tous, Envoyé, Erreur, En attente|
| Recherche   | Text       | Email, sujet, script           |

COLONNES TABLEAU
| Colonne      | Largeur | Description               |
|--------------|---------|---------------------------|
| Date         | 140px   | Format JJ/MM HH:MM:SS     |
| Destinataire | 240px   | Adresse email             |
| Sujet        | 280px   | Sujet de l'email          |
| Script       | 170px   | Chemin script PHP         |
| Statut       | 110px   | Badge coloré              |
| Actions      | 80px    | Lien détails              |

STATUTS EMAIL
| Statut     | Badge    | Couleur  |
|------------|----------|----------|
| Envoyé     | success  | Vert     |
| Erreur     | error    | Rouge    |
| En attente | warning  | Orange   |

STATISTIQUES
| Métrique   | Description              |
|------------|--------------------------|
| Envoyés    | Emails délivrés          |
| Erreurs    | Emails en échec          |
| En attente | En cours d'envoi         |
| Total      | Nombre total             |

BOUTONS
| Bouton     | Action                     |
|------------|----------------------------|
| Refresh    | Actualiser les logs        |
| Export CSV | Télécharger les données    |
| Fermer     | Fermer modal               |

INTERACTIONS
- Filtres → Rafraîchir tableau
- Clic Détails → Voir détail email
- Pagination → Navigation pages
- Export → Télécharger CSV

API ENDPOINTS
GET /hosting/web/{sn}/email/log                  → Liste des logs
GET /hosting/web/{sn}/email/log/{id}             → Détail log
