# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (BDD actif)
   - NAV4: 3 sous-tabs (Liste, Statistiques, Dumps)
   - TOOLBAR: Refresh, Search, Créer BDD, Commander BDD
   - TABLE: Bases de données
     - Nom de la base
     - Utilisateur
     - Serveur
     - Version
     - Taille (progress bar)
     - État (badge)
     - Dernière modif
     - Actions (...)
   - INFO: X bases sur Y disponibles

# INTERACTIONS
- Clic "Créer une BDD" -> Modal add-database
- Clic "Commander BDD" -> Modal order-database
- Clic "..." -> Menu (Changer MDP, Dump, Restaurer, Import, phpMyAdmin, Supprimer)

# API ENDPOINTS
GET /hosting/web/{serviceName}/database
GET /hosting/web/{serviceName}/database/{name}
POST /hosting/web/{serviceName}/database
POST /hosting/web/{serviceName}/database/{name}/changePassword
POST /hosting/web/{serviceName}/database/{name}/dump
POST /hosting/web/{serviceName}/database/{name}/import
DELETE /hosting/web/{serviceName}/database/{name}
