STRUCTURE MODAL
1. Header avec titre
2. Banniere info
3. Identifiant (lecture seule)
4. Dropdown repertoire
5. Toggle acces SSH
6. Toggle etat compte
7. Section mot de passe (lien vers autre modal)
8. Info derniere connexion
9. Footer avec boutons

CHAMPS
| Champ           | Type     | Modifiable | Description                  |
|-----------------|----------|------------|------------------------------|
| Identifiant     | Text     | Non        | Login FTP                    |
| Repertoire      | Select   | Oui        | Chemin home                  |
| SSH             | Toggle   | Oui        | Acces SSH actif              |
| Etat            | Toggle   | Oui        | Compte actif/inactif         |

INTERACTIONS
- Dropdown repertoire → Changer chemin home
- Toggle SSH → Activer/desactiver acces SSH
- Toggle etat → Activer/desactiver compte
- Clic Changer MDP → Ouvrir modal mot de passe
- Clic Modifier → Sauvegarder les modifications

API ENDPOINTS
GET /hosting/web/{sn}/user/{login}                → Details utilisateur
PUT /hosting/web/{sn}/user/{login}                → Modifier utilisateur
