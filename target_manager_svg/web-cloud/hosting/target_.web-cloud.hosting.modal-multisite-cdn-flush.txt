STRUCTURE MODAL
1. Header avec titre
2. Domaine concerne
3. Type de purge (radio)
4. Zone URLs (si specifique)
5. Warning performances
6. Banniere info propagation
7. Footer avec boutons

TYPES PURGE
| Type         | Description                    |
|--------------|--------------------------------|
| Tout         | Vide tout le cache du domaine  |
| Specifique   | URLs listees uniquement        |

FORMAT URLs
- Une URL par ligne
- Peut inclure wildcards (*)
- Exemple: /images/*

IMPACT PURGE
- Temps chargement temporairement plus long
- Premiers visiteurs reloadent depuis origine
- Cache reconstruit progressivement

PROPAGATION
- Quelques minutes
- Tous les serveurs CDN
- Globalement distribue

BOUTONS FOOTER
| Bouton  | Action                     |
|---------|----------------------------|
| Annuler | Fermer sans purger         |
| Purger  | POST cdn/flush             |

INTERACTIONS
- Radio → Activer/desactiver zone URLs
- Clic Purger → POST flush

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}     → Info domaine
POST /hosting/web/{sn}/attachedDomain/{domain}/cdn/flush → Purger
