# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste des hébergements avec recherche et filtre
4. RIGHT PANEL
   - Header: Nom du service + domaines attachés
   - NAV3: 16 tabs (Home actif)
   - NAV4: 7 sous-tabs (Général actif)
   - BLOC 1: Informations générales (état, IP, stockage, datacenter)
   - BLOC 2: Configuration (PHP, SSL, CDN, email, firewall)
   - BLOC 3: Résumé (sites, BDD, modules, FTP, crons)
   - BLOC 4: Abonnement (offre, dates, contacts)
   - BLOC 5: Liens utiles

# INTERACTIONS
- Clic service LEFT PANEL -> Sélection et affichage détails
- Clic tab NAV3 -> Navigation vers section
- Clic sous-tab NAV4 -> Navigation vers sous-section
- Clic "Modifier" PHP -> Modal ovhconfig
- Clic "Changer d'offre" -> Modal upgrade
