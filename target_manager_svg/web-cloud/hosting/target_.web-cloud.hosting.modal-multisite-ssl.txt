STRUCTURE MODAL
1. Header avec titre
2. Affichage domaine concerne
3. Statut SSL actuel (banniere)
4. Toggle activation SSL
5. Toggle redirection HTTPS
6. Section HSTS (toggle + duree)
7. Informations certificat
8. Info application immediate
9. Footer avec boutons

OPTIONS SSL
| Option              | Description                              | Defaut   |
|---------------------|------------------------------------------|----------|
| Activer SSL         | Utiliser certificat pour ce domaine      | ON       |
| Redirection HTTPS   | Forcer HTTP → HTTPS                      | ON       |
| HSTS                | Strict Transport Security                | OFF      |

DUREES HSTS
| Duree      | Valeur (secondes) |
|------------|-------------------|
| 1 mois     | 2592000           |
| 6 mois     | 15552000          |
| 1 an       | 31536000          |
| 2 ans      | 63072000          |

INFO CERTIFICAT
- Type (Let's Encrypt DV, Sectigo DV, etc.)
- Dates delivrance/expiration
- Nombre domaines couverts
- Mode renouvellement

AVERTISSEMENT HSTS
HSTS est difficile a desactiver une fois active car
les navigateurs conservent la directive en cache.

INTERACTIONS
- Toggles → Activer/desactiver options
- Dropdown duree HSTS → Selectionner
- Clic Appliquer → Sauvegarder configuration

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}          → Config domaine
PUT /hosting/web/{sn}/attachedDomain/{domain}          → Modifier SSL
GET /hosting/web/{sn}/ssl                              → Info certificat
