STRUCTURE MODAL
1. Header avec titre
2. Banniere info deploiement
3. Champ URL depot Git
4. Dropdown branche
5. Champ chemin deploiement
6. Dropdown site associe
7. Section authentification (3 methodes)
8. Affichage cle SSH publique
9. Options deploiement (checkboxes)
10. Footer avec boutons

METHODES AUTHENTIFICATION
| Methode          | Description                          |
|------------------|--------------------------------------|
| Cle SSH          | Recommande, plus securise            |
| Token personnel  | GitHub/GitLab access token           |
| Identifiants     | Utilisateur et mot de passe          |

OPTIONS DEPLOIEMENT
- Deploiement automatique sur push (webhook)
- Vider repertoire destination avant deploiement

INTERACTIONS
- Clic methode auth → Selectionner
- Clic copie cle SSH → Copier dans presse-papier
- Checkbox → Activer/desactiver option
- Clic Associer → Creer association Git

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain                  → Liste sites
POST /hosting/web/{sn}/attachedDomain/{d}/git         → Associer Git
GET /hosting/web/{sn}/attachedDomain/{d}/git          → Config Git existante
