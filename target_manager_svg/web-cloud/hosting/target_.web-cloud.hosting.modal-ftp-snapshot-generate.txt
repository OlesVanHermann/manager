STRUCTURE MODAL
1. Header avec titre
2. Banniere info explicative
3. Champ nom du snapshot
4. Champ description (optionnel)
5. Info espace (taille + quota)
6. Checkbox fichiers caches
7. Footer avec boutons

CHAMPS
| Champ       | Type     | Required | Description              |
|-------------|----------|----------|--------------------------|
| Nom         | Input    | Oui      | Identifiant snapshot     |
| Description | Textarea | Non      | Note explicative         |

LIMITES SNAPSHOTS
| Offre       | Max snapshots |
|-------------|---------------|
| Start       | 2             |
| Performance | 5             |
| Pro         | 10            |

OPTIONS
| Option          | Description                   |
|-----------------|-------------------------------|
| Fichiers caches | .htaccess, .env, .git, etc.   |

BOUTONS FOOTER
| Bouton  | Action                     |
|---------|----------------------------|
| Annuler | Fermer sans creer          |
| Creer   | POST snapshot              |

INTERACTIONS
- Remplir champs → Validation
- Clic Creer → POST snapshot

API ENDPOINTS
GET /hosting/web/{sn}                             → Quota info
GET /hosting/web/{sn}/snapshot                    → Liste snapshots
POST /hosting/web/{sn}/snapshot                   → Creer snapshot
