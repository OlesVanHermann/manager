# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (BDD actif)
   - NAV4: Liste | Statistiques | Dumps
   - TOOLBAR: Créer BDD | Commander BDD | Recherche
   - TABLE: Nom, Utilisateur, Serveur, Version, Taille, Quota, État, Actions
   - PAGINATION + Info quota

# COLONNES TABLE BASES DE DONNÉES
| Colonne      | Largeur | Contenu                           |
|--------------|---------|-----------------------------------|
| Nom          | 160px   | Nom base de données               |
| Utilisateur  | 120px   | Login utilisateur BDD             |
| Serveur      | 190px   | Adresse serveur MySQL             |
| Version      | 90px    | MySQL 5.7 / 8.0                   |
| Taille       | 170px   | Taille actuelle + barre progress  |
| Quota        | 130px   | Utilisé / Max + %                 |
| État         | 120px   | Badge Actif/Suspendu              |
| Actions      | 56px    | Menu ⋮                            |

# INDICATEURS VISUELS
- Progress bar verte: < 70% quota
- Progress bar orange: >= 70% quota
- Progress bar rouge: >= 90% quota
- Texte warning orange: >= 70% quota

# INTERACTIONS
- Clic "Créer BDD" → Modal création base
- Clic "Commander BDD" → Page commande
- Clic ⋮ → Menu contextuel (Changer MDP, Dump, Restaurer, Importer, phpMyAdmin, Supprimer)
- Clic sous-tab "Statistiques" → Page statistiques BDD
- Clic sous-tab "Dumps" → Page liste dumps

# API ENDPOINTS
GET /hosting/web/{sn}/database              - Liste bases
GET /hosting/web/{sn}/database/{name}       - Détails base
POST /hosting/web/{sn}/database             - Créer base
PUT /hosting/web/{sn}/database/{name}       - Modifier base
POST /hosting/web/{sn}/database/{name}/changePassword - Changer MDP
POST /hosting/web/{sn}/database/{name}/dump - Créer dump
POST /hosting/web/{sn}/database/{name}/import - Importer SQL
POST /hosting/web/{sn}/database/{name}/restore - Restaurer
DELETE /hosting/web/{sn}/database/{name}    - Supprimer
