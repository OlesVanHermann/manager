# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: Sauvegardes de la base {dbName}
   - Breadcrumb: {serviceName} › {dbName} › Sauvegardes
   - BANNIERE INFO: Conservation 30 jours
   - TOOLBAR: + Créer une sauvegarde
   - TABLE: Date, Type, Taille, Statut, Expiration, Actions
   - PAGINATION

# TYPES DE SAUVEGARDES
- Automatique (badge bleu) - Créée par le système
- Manuel (badge jaune) - Créée par l'utilisateur

# STATUTS
- Disponible (vert) - Prête à restaurer/télécharger
- En cours (jaune) - Création en cours
- Erreur (rouge) - Échec de création

# EXPIRATION
- Date d'expiration pour automatiques (30 jours)
- "-" pour manuelles (pas d'expiration)

# ACTIONS
- + Créer une sauvegarde → Crée un dump manuel
- Restaurer → Restaure la base à cet état
- Télécharger → Télécharge le fichier .sql.gz
- ⋮ (menu) → Restaurer / Télécharger / Supprimer

# API ENDPOINTS
GET /hosting/web/{serviceName}/database/{name}
GET /hosting/web/{serviceName}/database/{name}/dump
POST /hosting/web/{serviceName}/database/{name}/dump
DELETE /hosting/web/{serviceName}/database/{name}/dump/{id}
POST /hosting/web/{serviceName}/database/{name}/dump/{id}/restore
