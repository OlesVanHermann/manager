STRUCTURE MODAL
1. Header danger (rouge)
2. Bannière warning sécurité
3. Info certificat actuel
4. Liste des conséquences
5. Checkbox confirmation
6. Footer avec boutons

CONSEQUENCES DESACTIVATION
| Impact            | Description                          |
|-------------------|--------------------------------------|
| Navigateurs       | Affichage "Non sécurisé"             |
| Données           | Transmission non chiffrée            |
| SEO               | Pénalité classement Google           |
| Confiance         | Baisse crédibilité site              |
| APIs              | Incompatibilité certaines APIs       |

CERTIFICAT ACTUEL
| Champ       | Description                |
|-------------|----------------------------|
| Type        | Let's Encrypt, Sectigo...  |
| Domaines    | Nombre domaines couverts   |
| Expiration  | Date fin validité          |
| Statut      | Actif/Inactif              |

VALIDATION
- Checkbox obligatoire pour activer bouton
- Bouton grisé par défaut

BOUTONS FOOTER
| Bouton     | État         | Action                     |
|------------|--------------|----------------------------|
| Annuler    | Toujours     | Fermer sans désactiver     |
| Désactiver | Grisé→Rouge  | DELETE ssl                 |

INTERACTIONS
- Checkbox → Active bouton Désactiver
- Clic Désactiver → DELETE ssl

API ENDPOINTS
GET /hosting/web/{sn}/ssl                        → Info SSL actuel
DELETE /hosting/web/{sn}/ssl                     → Désactiver SSL
