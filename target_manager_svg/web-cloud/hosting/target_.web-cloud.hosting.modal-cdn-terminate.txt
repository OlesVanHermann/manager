STRUCTURE MODAL
1. Header rouge (action dangereuse)
2. Icone warning
3. Question de confirmation
4. Info offre CDN actuelle
5. Liste consequences
6. Date effective de resiliation
7. Checkbox confirmation obligatoire
8. Footer avec boutons

INFORMATIONS CDN
| Champ              | Description                    |
|--------------------|--------------------------------|
| Offre              | Nom de l'offre CDN             |
| Prix               | Cout mensuel HT                |
| Domaines actifs    | Nombre de domaines avec CDN    |
| Date souscription  | Date d'activation              |

CONSEQUENCES RESILIATION
- Desactivation CDN sur tous les domaines
- Suppression du cache et configurations
- Trafic servi directement depuis l'hebergement
- Effet a la fin de la periode de facturation

DATE RESILIATION
La resiliation prend effet a la fin de la periode
de facturation en cours (pas de remboursement prorata).

REGLES VALIDATION
- Checkbox confirmation obligatoire
- Bouton Resilier desactive sans confirmation

INTERACTIONS
- Checkbox → Activer bouton Resilier
- Clic Resilier → Demander resiliation
- Clic Annuler → Fermer modal

API ENDPOINTS
DELETE /hosting/web/{sn}/cdn                           → Resilier CDN
  (ou POST /hosting/web/{sn}/cdn/terminate)
