STRUCTURE MODAL
1. Header avec titre et bouton fermer
2. Etape 1: Type de domaine
   - Domaine OVHcloud (config DNS auto)
   - Domaine externe (config DNS manuelle)
3. Etape 2: Selection domaine
   - Dropdown domaines OVH
   - Input sous-domaine optionnel
4. Etape 3: Configuration
   - Repertoire racine
   - Runtime
5. Etape 4: Options
   - SSL, Firewall, CDN, Logs
6. Banniere info DNS
7. Footer avec boutons

TYPE DE DOMAINE
| Type          | Description                       |
|---------------|-----------------------------------|
| OVHcloud      | Domaine gere chez OVH             |
|               | Config DNS automatique            |
| Externe       | Domaine gere ailleurs             |
|               | Config DNS manuelle requise       |

OPTIONS DISPONIBLES
| Option     | Description                       | Defaut |
|------------|-----------------------------------|--------|
| SSL        | Certificat Let's Encrypt          | ON     |
| Firewall   | Protection applicative            | ON     |
| CDN        | Cache et acceleration             | OFF    |
| Logs       | Logs separes par domaine          | OFF    |

INTERACTIONS
- Clic carte type → Selection type domaine
- Clic dropdown → Selection domaine
- Clic &#128193; → Parcourir repertoires
- Clic toggle → Activer/desactiver option
- Clic Annuler → Fermer sans action
- Clic Ajouter → Ajouter le multisite

VALIDATIONS
- Domaine non deja attache
- Repertoire valide
- Quota multisites non atteint

API ENDPOINTS
GET /domain                                        → Domaines OVH
GET /hosting/web/{sn}/runtime                      → Runtimes
GET /hosting/web/{sn}/attachedDomain               → Multisites existants
POST /hosting/web/{sn}/attachedDomain              → Ajouter multisite
GET /hosting/web/{sn}                              → Info CDN/firewall
