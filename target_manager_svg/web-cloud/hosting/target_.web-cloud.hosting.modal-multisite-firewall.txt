STRUCTURE MODAL
1. Header avec titre
2. Affichage domaine concerne
3. Statut firewall (toggle ON/OFF)
4. Selection niveau protection (3 cartes)
5. Liste protections actives (checkboxes)
6. Info explicative
7. Footer avec boutons

NIVEAUX DE PROTECTION
| Niveau    | Description                              |
|-----------|------------------------------------------|
| Faible    | Regles minimales, moins de faux positifs |
| Standard  | Equilibre securite/usabilite (recommande)|
| Strict    | Protection maximale, peut bloquer        |

PROTECTIONS SUPPORTEES
- Injection SQL
- Cross-Site Scripting (XSS)
- CSRF (Cross-Site Request Forgery)
- Path Traversal
- Remote File Inclusion
- Command Injection
- Bad Bots detection
- DDoS Layer 7
- Protocol Anomalies

TOGGLE FIREWALL
- ON (vert) : Firewall actif avec protections
- OFF (gris) : Firewall desactive

INTERACTIONS
- Toggle → Activer/desactiver firewall
- Clic niveau → Selectionner niveau protection
- Clic Appliquer → Sauvegarder configuration

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}          → Config actuelle
PUT /hosting/web/{sn}/attachedDomain/{domain}          → Modifier firewall
