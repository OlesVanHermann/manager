# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, CDN actif
   - Description
   - CARD: Statut CDN (service, type, backend)
   - BUTTONS: Vider cache / Configurer / Désactiver
   - TABLE: Configuration par multisite (Domaine, CDN, Firewall, Cache, HTTP/2, Actions)
   - PAGINATION

# TYPES CDN
- CDN Basic (inclus)
- CDN Advanced (option payante)

# OPTIONS PAR MULTISITE
- CDN: Actif/Désactivé
- Firewall: Actif/Désactivé
- Cache TTL: 1h, 24h, 7j, etc.
- HTTP/2: Actif/Désactivé

# ACTIONS
- Vider le cache → POST /hosting/web/{sn}/cdn/flush
- Configurer CDN → modal-cdn-settings
- Désactiver CDN → PUT /hosting/web/{sn}/cdn status=disabled
- ⋮ (row action) → Configurer / Désactiver (par domaine)

# API ENDPOINTS
GET /hosting/web/{serviceName}/cdn
PUT /hosting/web/{serviceName}/cdn
POST /hosting/web/{serviceName}/cdn/flush
GET /hosting/web/{serviceName}/attachedDomain
PUT /hosting/web/{serviceName}/attachedDomain/{domain}
