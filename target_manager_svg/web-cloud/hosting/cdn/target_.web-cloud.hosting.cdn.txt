2. RIGHT PANEL:
   - Titre "CDN"

GET /hosting/web/{serviceName}/cdn/statistics - Statistiques
POST /hosting/web/{serviceName}/cdn/purge - Purger cache global
POST /hosting/web/{serviceName}/attachedDomain/{domain}/purgeCache - Purger domaine
