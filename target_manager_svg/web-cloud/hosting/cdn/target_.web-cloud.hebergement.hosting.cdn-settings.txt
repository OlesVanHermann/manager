STRUCTURE PAGE
1. NAV4: General | Configuration
2. Header + bouton "Ajouter regle"
3. Table regles de cache
   - Colonnes: PATTERN, TYPE, TTL, PRIORITE, ACTIONS
   - Actions: Modifier, Supprimer
4. Banniere info

TYPES REGLES
| Type   | Description                    |
|--------|--------------------------------|
| Cache  | Mise en cache CDN              |
| Bypass | Pas de cache (direct origin)   |

TTL DISPONIBLES
| Valeur    | Description                    |
|-----------|--------------------------------|
| 1h        | 1 heure                        |
| 24h       | 24 heures                      |
| 7 jours   | 1 semaine                      |
| 30 jours  | 1 mois                         |
| 1 an      | 1 an (contenu statique)        |

API ENDPOINTS
GET    /hosting/web/{sn}/cdn/rules        -> Liste regles
POST   /hosting/web/{sn}/cdn/rules        -> Creer regle
PUT    /hosting/web/{sn}/cdn/rules/{id}   -> Modifier regle
DELETE /hosting/web/{sn}/cdn/rules/{id}   -> Supprimer regle
