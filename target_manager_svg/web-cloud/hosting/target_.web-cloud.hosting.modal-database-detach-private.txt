STRUCTURE MODAL
1. Header warning (orange)
2. Bannière alerte
3. Configuration actuelle
4. Nouvelle configuration
5. Liste des impacts
6. Checkbox confirmation
7. Footer avec boutons

CONFIGURATION ACTUELLE
| Champ         | Description                    |
|---------------|--------------------------------|
| Base          | Nom de la base                 |
| Serveur       | Hostname Private SQL           |
| Offre         | Niveau Private SQL             |
| Taille        | Espace utilisé                 |

NOUVELLE CONFIGURATION
| Champ         | Description                    |
|---------------|--------------------------------|
| Serveur       | Hostname MySQL mutualisé       |
| Type          | MySQL partagé                  |

IMPACTS
| Impact                  | Gravité  |
|------------------------|----------|
| Performances           | Warning  |
| Limite connexions      | Warning  |
| Quota espace           | Warning  |
| Conservation données   | Info     |

VALIDATION
- Checkbox obligatoire pour activer bouton
- Bouton grisé par défaut

BOUTONS FOOTER
| Bouton   | État         | Action                     |
|----------|--------------|----------------------------|
| Annuler  | Toujours     | Fermer sans détacher       |
| Détacher | Grisé→Orange | POST detach                |

INTERACTIONS
- Checkbox → Active bouton Détacher
- Clic Détacher → POST detach

API ENDPOINTS
GET /hosting/web/{sn}/database/{db}              → Info base
POST /hosting/web/{sn}/database/{db}/detach      → Détacher du Private SQL
