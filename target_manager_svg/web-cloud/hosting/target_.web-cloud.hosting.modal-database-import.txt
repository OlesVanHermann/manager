STRUCTURE MODAL
1. Header avec titre
2. Banniere info (formats supportes)
3. Base de donnees cible (lecture seule)
4. Zone upload drag&drop
5. Options d'import (toggles)
6. Warning recommandation
7. Footer avec boutons

FORMATS SUPPORTES
- .sql (SQL brut)
- .sql.gz (compresse gzip)
- .sql.bz2 (compresse bzip2)
- Max 500 Mo

OPTIONS
| Option                    | Defaut | Description                    |
|---------------------------|--------|--------------------------------|
| Vider base avant import   | OFF    | DROP tables existantes         |
| Notification email        | ON     | Email fin d'import             |

VALIDATION
- Fichier obligatoire pour activer bouton Importer
- Verification format et taille

INTERACTIONS
- Drag & drop fichier → Selection
- Clic zone → Ouvrir explorateur
- Toggle options → Activer/desactiver
- Clic Importer → Lancer import

API ENDPOINTS
GET /hosting/web/{sn}/database/{name}             → Details BDD
POST /hosting/web/{sn}/database/{name}/import     → Importer fichier
