STRUCTURE MODAL
1. Header avec titre
2. Liste sauvegardes (radio)
3. Options restauration (checkboxes)
4. Warning consequences
5. Checkbox confirmation
6. Banniere info duree
7. Footer avec boutons

LISTE SAUVEGARDES
| Colonne   | Description                    |
|-----------|--------------------------------|
| Selection | Radio button                   |
| Date      | Date et heure sauvegarde       |
| Tailles   | Fichiers + BDD                 |
| Statut    | Badge Complete/Partielle       |

OPTIONS RESTAURATION
| Option   | Description                    |
|----------|--------------------------------|
| Fichiers | Restaurer contenu FTP          |
| BDD      | Restaurer bases de donnees     |

STATUTS SAUVEGARDE
| Statut    | Couleur | Description           |
|-----------|---------|------------------------|
| Complete  | Vert    | Tout est sauvegarde    |
| Partielle | Orange  | Certains elements manq |
| Echec     | Rouge   | Sauvegarde echouee     |

BOUTONS FOOTER
| Bouton    | Action                     | Etat            |
|-----------|----------------------------|-----------------|
| Annuler   | Fermer sans restaurer      | Toujours actif  |
| Restaurer | POST backup/restore        | Si checkbox     |

INTERACTIONS
- Radio → Selectionner sauvegarde
- Checkboxes → Elements a restaurer
- Checkbox confirm → Active bouton

API ENDPOINTS
GET /hosting/web/{sn}/backup                      → Liste sauvegardes
POST /hosting/web/{sn}/backup/{id}/restore        → Lancer restauration
