STRUCTURE MODAL
1. Header avec titre
2. Banniere info prechargement
3. Textarea URLs a precharger
4. Options prechargement (checkboxes)
5. Section limites
6. Compteur URLs saisies
7. Footer avec boutons

LIMITES PRECHARGEMENT
| Limite                | Valeur    |
|-----------------------|-----------|
| URLs par demande      | 100 max   |
| Demandes par jour     | 5 max     |
| Taille max par URL    | 50 Mo     |

OPTIONS DISPONIBLES
| Option                         | Description                          |
|--------------------------------|--------------------------------------|
| Ressources statiques auto      | Pre-charge CSS, JS, images lies      |
| Parcours recursif              | Suit les liens (1 niveau)            |

FORMAT URLS
- Une URL complete par ligne
- Doit commencer par https://
- Doit etre sur un domaine de l'hebergement

CAS D'UTILISATION
- Mise a jour majeure du site
- Avant un evenement/pic de trafic
- Apres purge complete du cache

INTERACTIONS
- Textarea → Saisir URLs
- Checkboxes → Activer options
- Clic Precharger → Lancer prechargement

API ENDPOINTS
POST /hosting/web/{sn}/cdn/prewarm                     → Precharger URLs
  Body: { urls: [...], options: {...} }
GET /hosting/web/{sn}/cdn                              → Quota restant
