STRUCTURE MODAL
1. Header avec titre
2. Warning si quota proche limite
3. Barre progression utilisation
4. Details quotas (quotidien, horaire, taille)
5. Section upgrade offre
6. Conseils optimisation
7. Footer

QUOTAS PAR OFFRE
| Offre       | Quotidien | Horaire | Taille max |
|-------------|-----------|---------|------------|
| Starter     | 100       | 20      | 5 Mo       |
| Performance | 200       | 50      | 10 Mo      |
| Business    | 500       | 100     | 25 Mo      |
| Cloud Web   | 1000      | 200     | 50 Mo      |

BARRE DE PROGRESSION
- Vert si < 70%
- Orange si 70-90%
- Rouge si > 90%

RESET QUOTA
- Reset quotidien a minuit (UTC+1)
- Compteur heure glissante

UPGRADE
Lien vers page upgrade offre ou
commande Email Pro/Exchange

INTERACTIONS
- Clic Voir offres → Ouvrir page offres
- Clic Fermer → Fermer modal

API ENDPOINTS
GET /hosting/web/{sn}/email                            → Stats emails
GET /hosting/web/{sn}/email/quota                      → Details quota
