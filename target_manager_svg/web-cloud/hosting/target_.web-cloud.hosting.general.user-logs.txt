INTERACTIONS
- Clic Ajouter utilisateur → Modal creation compte
- Clic &#128274; (mot de passe) → Modal changement mot de passe
- Clic &#9998; (editer) → Modal modification description
- Clic &#10005; (supprimer) → Modal confirmation suppression
- Clic &#128203; (copier) → Copier URL dans presse-papier
- Clic sous-tab → Changer de vue

API ENDPOINTS
GET /hosting/web/{serviceName}/userLogs            → Liste utilisateurs
GET /hosting/web/{sn}/userLogs/{login}             → Details utilisateur
POST /hosting/web/{serviceName}/userLogs           → Creer utilisateur
PUT /hosting/web/{sn}/userLogs/{login}             → Modifier utilisateur
DELETE /hosting/web/{sn}/userLogs/{login}          → Supprimer utilisateur
POST /hosting/web/{sn}/userLogs/{login}/password   → Changer mot de passe
