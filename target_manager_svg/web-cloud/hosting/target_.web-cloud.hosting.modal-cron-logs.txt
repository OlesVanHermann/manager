STRUCTURE MODAL
1. Header avec titre
2. Info tache cron (commande, frequence, email)
3. Filtres (statut, periode, refresh, export)
4. Table historique executions
5. Zone affichage output selectionne
6. Pagination
7. Footer

COLONNES TABLE
| Colonne     | Description                    |
|-------------|--------------------------------|
| Date/Heure  | Timestamp execution            |
| Duree       | Temps d'execution              |
| Statut      | Succes/Erreur                  |
| Code retour | Exit code (0 = succes)         |
| Output      | Lien vers logs detailles       |

CODES RETOUR
| Code | Signification                    |
|------|----------------------------------|
| 0    | Execution reussie                |
| 1    | Erreur generale                  |
| 2    | Mauvaise utilisation commande    |
| 126  | Permission refusee               |
| 127  | Commande non trouvee             |
| 137  | Killed (timeout ou OOM)          |

ZONE OUTPUT
- Affichage terminal style
- Coloration syntaxique (erreurs en rouge)
- Infos execution (temps, memoire)
- Code retour final

FILTRES
- Statut: Tous / Succes / Erreur
- Periode: 24h / 7 jours / 30 jours / Tout

INTERACTIONS
- Dropdown filtres → Filtrer resultats
- Clic refresh → Actualiser
- Clic Exporter → Telecharger CSV
- Clic ligne → Afficher output
- Pagination → Naviguer

API ENDPOINTS
GET /hosting/web/{sn}/cron/{id}                        → Details cron
GET /hosting/web/{sn}/cron/{id}/logs                   → Historique
