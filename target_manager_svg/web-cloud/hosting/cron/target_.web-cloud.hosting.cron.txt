# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, Cron actif
   - Description
   - TOOLBAR: + Ajouter planification
   - TABLE: Commande, Langage, Fréquence, E-mail, État, Description, Actions
   - PAGINATION
   - HELP LINK

# LANGAGES
- PHP 7.4, 8.0, 8.1, 8.2, 8.3
- Node.js 16, 18, 20
- Autre (shell scripts)

# FRÉQUENCES (cron syntax)
- 0 2 * * * → Tous les jours à 2h
- */15 * * * * → Toutes les 15 minutes
- 0 8 * * 1 → Tous les lundis à 8h

# STATUTS
- Activé (vert) - Tâche active
- Désactivé (gris) - Tâche désactivée

# ACTIONS
- + Ajouter planification → modal-cron-add
- ⋮ (row action) → Modifier / Supprimer
