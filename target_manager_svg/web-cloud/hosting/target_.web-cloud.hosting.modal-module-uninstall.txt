STRUCTURE MODAL
1. Header danger (rouge)
2. Bannière alerte irréversible
3. Info module à supprimer
4. Liste éléments supprimés
5. Note base de données
6. Checkbox confirmation
7. Footer avec boutons

ELEMENTS SUPPRIMES
| Élément              | Impact                       |
|----------------------|------------------------------|
| Fichiers module      | Tous supprimés               |
| Thèmes/plugins       | Supprimés                    |
| Uploads/médias       | Supprimés                    |
| Configuration        | Supprimée                    |

ELEMENTS CONSERVES
| Élément              | Note                         |
|----------------------|------------------------------|
| Base de données      | Non touchée (suppr. séparée) |
| Autres fichiers      | Hors du répertoire module    |

VALIDATION
- Checkbox obligatoire pour activer bouton
- Bouton grisé par défaut

BOUTONS FOOTER
| Bouton      | État         | Action                     |
|-------------|--------------|----------------------------|
| Annuler     | Toujours     | Fermer sans supprimer      |
| Désinstaller| Grisé→Rouge  | DELETE module              |

INTERACTIONS
- Checkbox → Active bouton Désinstaller
- Clic Désinstaller → DELETE module

API ENDPOINTS
GET /hosting/web/{sn}/module/{id}                → Détail module
DELETE /hosting/web/{sn}/module/{id}             → Désinstaller
