# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (SEO actif)
   - TITRE
   - DESCRIPTION du service
   - LIEN vérification + BOUTON Commander
   - TABLE ENTREPRISES
   - PAGINATION

# DESCRIPTION
- Référencement local
- Gestion des informations d'entreprise
- Présence multi-annuaires

# TABLE ENTREPRISES
| Colonne            | Largeur | Contenu                          |
|--------------------|---------|----------------------------------|
| Nom de l'entreprise| 260px   | Raison sociale                   |
| Adresse            | 300px   | Adresse ou badge "À définir"     |
| Compte Email       | 220px   | Email de contact                 |
| Statut             | 140px   | Badge OK/Création en cours       |
| Actions            | 56px    | Menu ⋮                           |

# ÉTATS POSSIBLES
| État              | Badge couleur |
|-------------------|---------------|
| OK                | Vert #DEF7EC  |
| Création en cours | Jaune #FEF3C7 |
| À définir         | Bleu #DBEAFE  |

# MENU ACTIONS
- Accéder à l'interface
- Supprimer l'abonnement (rouge)

# INTERACTIONS
- Clic "Vérifiez votre référencement local" → Lien externe
- Clic "Commander" → Modal commande
- Clic ⋮ → Menu (Accéder, Supprimer)

# API ENDPOINTS
GET /hosting/web/{sn}/localSeo              - Liste abonnements
GET /hosting/web/{sn}/localSeo/{id}         - Détails entreprise
POST /hosting/web/{sn}/localSeo             - Commander
DELETE /hosting/web/{sn}/localSeo/{id}      - Supprimer abonnement
