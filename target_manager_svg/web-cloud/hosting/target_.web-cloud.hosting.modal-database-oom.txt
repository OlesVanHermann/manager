STRUCTURE MODAL
1. Header rouge (erreur critique)
2. Banniere alerte avec date
3. Info base de donnees
4. Barre memoire depassee (>100%)
5. Requetes consommatrices (terminal)
6. Solutions recommandees
7. Option upgrade
8. Footer avec boutons

ERREUR OOM
| Element           | Description                   |
|-------------------|-------------------------------|
| Out Of Memory     | Depassement limite RAM        |
| Consequence       | Arret automatique de la BDD   |
| Cause frequente   | Requete SQL trop volumineuse  |

REQUETES DETECTEES
| Info     | Description                    |
|----------|--------------------------------|
| Memoire  | RAM consommee par la requete   |
| Query    | Extrait de la requete SQL      |

SOLUTIONS
1. Optimisation requetes SQL
2. Ajout d'index
3. Limitation resultats (LIMIT)
4. Division requetes volumineuses
5. Upgrade memoire

BOUTONS FOOTER
| Bouton         | Action                    |
|----------------|---------------------------|
| Redemarrer BDD | POST restart              |
| J'ai compris   | Fermer modal (acknowledge)|

INTERACTIONS
- Clic Redemarrer → POST restart
- Clic Voir offres → Page upgrade

API ENDPOINTS
GET /hosting/web/{sn}/database/{name}           → Info base
GET /hosting/web/{sn}/database/{name}/oom       → Details erreur OOM
POST /hosting/web/{sn}/database/{name}/restart  → Redemarrer base
