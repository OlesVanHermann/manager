STRUCTURE MODAL
1. Header avec titre
2. Warning banniere
3. Dropdown selection sauvegarde
4. Liste sauvegardes disponibles (radio)
5. Checkboxes elements a restaurer
6. Checkbox confirmation
7. Footer avec boutons

ELEMENTS RESTAURABLES
| Element        | Description                        |
|----------------|------------------------------------|
| Fichiers FTP   | Arborescence complete              |
| Bases donnees  | Toutes les BDD                     |
| Emails         | Boites et messages                 |

TYPES SAUVEGARDE
- Quotidienne (7 derniers jours)
- Hebdomadaire (4 dernieres semaines)
- Mensuelle (12 derniers mois)

VALIDATION
- Selection sauvegarde obligatoire
- Au moins un element a restaurer
- Checkbox confirmation obligatoire

INTERACTIONS
- Radio sauvegarde → Selectionner
- Checkboxes elements → Choisir quoi restaurer
- Checkbox confirmation → Debloquer bouton
- Clic Restaurer → Lancer restauration

API ENDPOINTS
GET /hosting/web/{sn}/backup                      → Liste sauvegardes
POST /hosting/web/{sn}/backup/{id}/restore        → Restaurer
