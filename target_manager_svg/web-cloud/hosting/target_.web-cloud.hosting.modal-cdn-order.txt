STRUCTURE MODAL
1. Header avec titre
2. Banniere info
3. Selection offre (3 cards)
4. Selection domaines (checkboxes)
5. Liste fonctionnalites
6. Footer avec total + boutons

OFFRES CDN
| Offre         | Prix      | Description                    |
|---------------|-----------|--------------------------------|
| CDN Basic     | Inclus    | Cache statique, 1 domaine      |
| CDN Advanced  | 9,99 EUR  | Cache intelligent, tous domaines|
| CDN Security  | 29,99 EUR | + WAF, DDoS mitigation         |

FONCTIONNALITES
- Cache automatique ressources statiques
- Compression Gzip/Brotli
- Purge du cache a la demande
- Statistiques de trafic
- Support HTTP/2
- Points de presence mondiaux

INTERACTIONS
- Clic card offre → Selection
- Checkbox domaine → Inclure/exclure
- Clic Activer → Commander CDN

API ENDPOINTS
GET /order/catalog/public/cdn                     → Catalogue offres
GET /hosting/web/{sn}/attachedDomain              → Domaines attaches
POST /hosting/web/{sn}/cdn                        → Activer CDN
