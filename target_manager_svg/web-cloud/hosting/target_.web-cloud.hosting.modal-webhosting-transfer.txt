STRUCTURE MODAL
1. Header avec titre
2. Bannière info transfert
3. Informations hébergement actuel
4. Champ destinataire
5. Liste éléments à transférer (checkboxes)
6. Warning délai validation
7. Footer avec boutons

ELEMENTS TRANSFERABLES
| Élément           | Obligatoire | Note                      |
|-------------------|-------------|---------------------------|
| Hébergement       | Oui         | Toujours inclus           |
| Bases de données  | Non         | Par défaut sélectionné    |
| Comptes FTP       | Non         | Par défaut sélectionné    |
| Emails            | Non         | Optionnel                 |
| Domaines          | Non         | Optionnel                 |

VALIDATION
- NIC-handle valide requis
- Acceptation par destinataire
- Délai: 7 jours max

BOUTONS FOOTER
| Bouton     | Action                     |
|------------|----------------------------|
| Annuler    | Fermer sans transférer     |
| Transférer | POST transfer request      |

INTERACTIONS
- Input → Saisir NIC destinataire
- Checkboxes → Sélectionner éléments
- Clic Transférer → POST transfer

API ENDPOINTS
GET /hosting/web/{serviceName}                   → Info hébergement
POST /hosting/web/{serviceName}/changeContact    → Initier transfert
GET /hosting/web/{sn}/database                   → Liste BDD
GET /hosting/web/{sn}/user                       → Liste FTP
GET /hosting/web/{sn}/attachedDomain             → Liste domaines
