STRUCTURE MODAL
1. Header avec titre
2. Warning banniere
3. Dropdown version PHP
4. Selection moteur (php-fpm/php-cgi)
5. Selection environnement (production/development)
6. Toggle firewall WAF
7. Selection container
8. Apercu fichier .ovhconfig
9. Footer avec boutons

OPTIONS CONFIGURATION
| Parametre      | Options                           |
|----------------|-----------------------------------|
| Version PHP    | 7.4, 8.0, 8.1, 8.2               |
| Moteur         | php-fpm (rec.), php-cgi          |
| Environnement  | production, development          |
| Firewall       | security, none                    |
| Container      | stable, stable64, testing, legacy|

APERCU FICHIER
Le fichier .ovhconfig est genere dynamiquement
selon les options selectionnees.

INTERACTIONS
- Dropdown/cards → Modifier parametres
- Toggle firewall → Activer/desactiver
- Apercu → MAJ en temps reel
- Clic Appliquer → Sauvegarder config

API ENDPOINTS
GET /hosting/web/{sn}                             → Config actuelle
PUT /hosting/web/{sn}/configuration               → Modifier config
