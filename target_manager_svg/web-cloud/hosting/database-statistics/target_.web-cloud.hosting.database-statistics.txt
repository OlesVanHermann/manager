# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: Statistiques de la base de données
   - Breadcrumb: {serviceName} › {dbName}
   - FILTRES: Base (dropdown) + Période (7 jours, 30 jours, 1 an)
   - CHARTS ROW:
     - Connexions simultanées (line chart)
     - Espace disque utilisé (line chart)
   - SUMMARY CARDS: Connexions moy/max, Espace, Quota, Tables
   - BANNIERE INFO

# CHARTS
- Type: Line chart with area fill
- X-axis: Jours de la semaine
- Y-axis: Connexions / Espace Mo
- Color: Bleu OVH (#0050D7)
- Area: rgba(0,80,215,0.1)

# SUMMARY METRICS
- Connexions moyennes - Moyenne sur la période
- Connexions max - Pic maximum
- Espace utilisé - Taille actuelle en Mo
- Quota disponible - Espace restant
- Tables - Nombre de tables MySQL

# PÉRIODES
- 7 jours (par défaut)
- 30 jours
- 1 an
