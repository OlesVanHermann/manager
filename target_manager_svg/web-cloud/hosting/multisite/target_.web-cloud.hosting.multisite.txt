# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Multisite actif)
   - TOOLBAR: Refresh, Search, Ajouter domaine, Régénérer SSL
   - TABLE: Domaines attachés avec colonnes
     - Domaine
     - Dossier racine
     - SSL (badge)
     - CDN (badge)
     - Firewall (badge)
     - Logs séparés (badge)
     - Git (badge)
     - Diagnostic (indicateur)
     - Runtime
     - Actions (...)
   - PAGINATION

# INTERACTIONS
- Clic "Ajouter un domaine" -> Modal add-multisite
- Clic "..." -> Menu contextuel (Modifier, Logs, Purger CDN, Git, Détacher)
- Clic domaine -> Détail/modification
- Clic "Régénérer certificat SSL" -> Modal confirmation

# API ENDPOINTS
GET /hosting/web/{serviceName}/attachedDomain
GET /hosting/web/{serviceName}/attachedDomain/{domain}
POST /hosting/web/{serviceName}/attachedDomain
PUT /hosting/web/{serviceName}/attachedDomain/{domain}
DELETE /hosting/web/{serviceName}/attachedDomain/{domain}
