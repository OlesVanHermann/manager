# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - HEADER: Titre avec nom base + service
   - NAV3: 16 tabs (BDD actif)
   - BANDEAU INFO: Rétention 30 jours
   - TOOLBAR: Bouton créer sauvegarde
   - TABLE: Liste des sauvegardes
   - PAGINATION

# BANDEAU INFO
- Background bleu info #DBEAFE
- 2 lignes: rétention automatique + possibilité manuel

| Colonne          | Largeur | Contenu                          |
|------------------|---------|----------------------------------|
| Date de création | 200px   | Timestamp complet                |
| Type             | 130px   | Badge Automatique/Manuel         |
| Taille           | 110px   | En Mo                            |
| Statut           | 120px   | Badge Disponible                 |
| Expiration       | 140px   | Date ou "-" pour manuel          |

# ACTIONS EN LIGNE
- Restaurer (lien bleu)
- Télécharger (lien bleu)
- Menu ⋮ (options supplémentaires)

# MENU ACTIONS (⋮)
- Restaurer
- Télécharger (.sql.gz)
- Supprimer (rouge)

# PAGINATION
- Affichage X à Y sur Z sauvegardes
- Navigation par pages

# API ENDPOINTS
GET /hosting/web/{sn}/database/{name}/dump             - Liste dumps
POST /hosting/web/{sn}/database/{name}/dump            - Créer dump
GET /hosting/web/{sn}/database/{name}/dump/{id}        - Détails
DELETE /hosting/web/{sn}/database/{name}/dump/{id}     - Supprimer
POST /hosting/web/{sn}/database/{name}/dump/{id}/restore - Restaurer
GET /hosting/web/{sn}/database/{name}/dump/{id}/download - Télécharger
