# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - HEADER: Titre + sous-titre
   - STEPPER: 3 étapes (Offre actif, Configuration, Paiement)
   - TABLEAU COMPARATIF: Grille des offres
   - FOOTER: Annuler / Continuer

# STEPPER
| Étape | Label        | État        |
|-------|--------------|-------------|
| 1     | Offre        | Actif (bleu)|
| 2     | Configuration| Grisé       |
| 3     | Paiement     | Grisé       |
