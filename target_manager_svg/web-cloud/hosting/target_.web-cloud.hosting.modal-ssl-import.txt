STRUCTURE MODAL
1. Header avec titre
2. Banniere info certificat personnalise
3. Textarea certificat (PEM)
4. Textarea cle privee (PEM)
5. Textarea chaine certification (optionnel)
6. Warning important
7. Footer avec boutons

CHAMPS REQUIS
| Champ              | Format    | Obligatoire |
|--------------------|-----------|-------------|
| Certificat         | PEM       | Oui         |
| Cle privee         | PEM       | Oui         |
| Chaine CA          | PEM       | Non         |

FORMAT PEM
Les fichiers doivent etre au format PEM avec:
- -----BEGIN CERTIFICATE-----
- Contenu encode en Base64
- -----END CERTIFICATE-----

VALIDATION
Le systeme verifie:
- Format PEM valide
- Correspondance cle/certificat
- Domaines couverts vs domaines configures
- Dates de validite

AVERTISSEMENT
- Le certificat existant sera remplace
- Le certificat doit couvrir tous les domaines

INTERACTIONS
- Textarea → Coller contenu certificat
- Clic Importer → Valider et importer

API ENDPOINTS
POST /hosting/web/{sn}/ssl                             → Importer certificat
  Body: { certificate, key, chain }
