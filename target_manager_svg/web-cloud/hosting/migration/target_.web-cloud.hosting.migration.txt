# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: Migrer votre hébergement + {serviceName}
   - WIZARD STEPS: 1.Sélection offre | 2.Options | 3.Confirmation
   - BANNIERE INFO: Offre actuelle
   - TITRE: Choisissez la nouvelle offre
   - OFFER CARDS: Starter, Pro, Performance 1/2/4
   - BANNIERE WARNING: Avertissement downgrade
   - BOUTONS: Annuler / Suivant

# WIZARD STEPS
1. Sélection offre - Choix de l'offre cible
2. Options - Configuration supplémentaire
3. Confirmation - Récapitulatif et validation

# OFFER CARDS
- Starter: 1,99€/mois, 100Mo, 1 site, 1 BDD, 10 emails (non disponible si downgrade)
- Pro: 7,99€/mois, 250Go, illimité, 25 BDD, 500 emails
- Performance 1: 15,99€/mois, 500Go, illimité, 50 BDD, 1000 emails, ressources dédiées
- Performance 2: 29,99€/mois, 500Go, illimité, 100 BDD, 2000 emails
- Performance 4: 59,99€/mois, 1To, ressources premium

# ÉTATS CARDS
- Sélectionné (badge vert + border bleu) - Offre choisie
- Non disponible (grisé) - Downgrade non autorisé
- Sélectionner (bouton bleu) - Offre disponible

# ACTIONS
- Sélectionner → Met à jour l'offre sélectionnée
- Annuler → Retour dashboard
- Suivant → Étape 2 Options

# API ENDPOINTS
GET /hosting/web/{serviceName}
GET /order/hosting/web
GET /hosting/web/{serviceName}/migration/available
POST /order/hosting/web/{serviceName}/migration
GET /hosting/web/{serviceName}/migration/status
