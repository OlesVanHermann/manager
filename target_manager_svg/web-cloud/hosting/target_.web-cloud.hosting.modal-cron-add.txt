STRUCTURE MODAL
1. Header avec titre et bouton fermer
2. Banniere info
3. Formulaire creation tache CRON
   - Commande a executer
   - Langage
   - Frequence (presets + personnalise)
   - Bloc aide expression CRON
   - Email notification
   - Description
4. Warning performances
5. Footer avec boutons

CHAMPS FORMULAIRE
| Champ              | Type        | Validation                    |
|--------------------|-------------|-------------------------------|
| Commande           | Input       | Requis, chemin absolu         |
| Langage            | Select      | PHP, Node.js, Python, bash    |
| Frequence          | Presets     | Selection preset ou custom    |
| Expression CRON    | Input       | Format valide 5 champs        |
| Email              | Email       | Optionnel, format email       |
| Description        | Input       | Optionnel, max 255 chars      |

PRESETS FREQUENCE
| Preset              | Expression     | Description              |
|---------------------|----------------|--------------------------|
| Toutes les heures   | 0 * * * *      | A chaque heure ronde     |
| Tous les jours      | 0 0 * * *      | A minuit                 |
| Toutes les semaines | 0 0 * * 0      | Dimanche a minuit        |
| Personnalise        | * * * * *      | Expression libre         |

LANGAGES DISPONIBLES
| Langage  | Chemin binaire                    |
|----------|-----------------------------------|
| PHP 8.2  | /usr/local/php8.2/bin/php         |
| PHP 8.1  | /usr/local/php8.1/bin/php         |
| Node.js  | /usr/local/nodejs18/bin/node      |
| Python   | /usr/local/python3.11/bin/python3 |
| bash     | /bin/bash                         |

INTERACTIONS
- Clic preset frequence → Selection et MAJ expression
- Clic Personnalise → Mode edition libre
- Clic Annuler → Fermer sans action
- Clic Creer → Creer la tache CRON

API ENDPOINTS
GET /hosting/web/{serviceName}                     → Info home
GET /hosting/web/{sn}/cron                         → Liste crons existants
POST /hosting/web/{sn}/cron                        → Creer tache CRON
GET /hosting/web/{sn}/runtime                      → Langages disponibles
