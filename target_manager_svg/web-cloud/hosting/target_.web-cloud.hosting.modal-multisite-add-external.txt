STRUCTURE MODAL (3 etapes)
ETAPE 1 - DOMAINE (affichee)
1. Header avec titre
2. Indicateur etapes (1/3)
3. Warning configuration DNS manuelle
4. Champ saisie domaine complet
5. Champ repertoire racine
6. Options (SSL, firewall, IPv6)
7. Instructions DNS avec enregistrements a copier
8. Selection version PHP
9. Footer navigation

ETAPE 2 - CONFIGURATION
- Options avancees
- Validation du domaine

ETAPE 3 - VALIDATION
- Verification propagation DNS
- Confirmation finale

INSTRUCTIONS DNS
L'utilisateur doit configurer chez son registrar :
- Enregistrement A → IP de l'hebergement
- Enregistrement AAAA → IPv6 de l'hebergement (optionnel)

OPTIONS DISPONIBLES
| Option              | Description                          | Defaut   |
|---------------------|--------------------------------------|----------|
| SSL Let's Encrypt   | Certificat gratuit                   | Active   |
| Firewall applicatif | Protection WAF                       | Active   |
| IPv6                | Support adressage IPv6               | Desactive|

NOTE PROPAGATION
La propagation DNS peut prendre 24-48 heures.
Le certificat SSL ne peut etre genere qu'apres
validation des enregistrements DNS.

INTERACTIONS
- Input domaine → Saisir FQDN
- Input repertoire → Definir chemin
- Checkboxes → Activer/desactiver options
- Clic copie → Copier enregistrement DNS
- Clic Suivant → Etape suivante

API ENDPOINTS
GET /hosting/web/{sn}                                  → IP hebergement
POST /hosting/web/{sn}/attachedDomain                  → Ajouter multisite
