1. Header avec titre
2. Banniere info
3. Input commande
4. Selection frequence (cards)
5. Selecteur heure
6. Dropdown langage
7. Input email (optionnel)
8. Toggle etat
9. Info derniere/prochaine execution
10. Footer avec boutons

|---------------|------------------------------------|
| Quotidien     | Tous les jours                     |

INTERACTIONS
- Saisie commande → Validation chemin
- Clic card frequence → Selection
- Dropdown heure/langage → Choisir
- Toggle etat → Activer/desactiver
- Clic Modifier → Sauvegarder

API ENDPOINTS

PUT /hosting/web/{sn}/cron/{id}                   → Modifier cron
