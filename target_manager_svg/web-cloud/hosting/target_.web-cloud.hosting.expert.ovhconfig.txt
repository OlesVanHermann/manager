INTERACTIONS
- Clic version PHP -> Change version
- Clic moteur -> Change moteur
- Clic environnement -> Change mode
- Clic firewall -> Toggle firewall
- Bouton Sauvegarder -> Applique configuration
- Bouton Reinitialiser -> Remet valeurs par defaut

API ENDPOINTS
GET /hosting/web/{sn}                         # Configuration actuelle
GET /hosting/web/{sn}/availableConfigurations # Versions PHP disponibles
PUT /hosting/web/{sn}                         # Modifier configuration
POST /hosting/web/{sn}/request                # Actions speciales
