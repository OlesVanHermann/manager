STRUCTURE MODAL
1. Header rouge avec titre
2. Icone warning
3. Message de confirmation
4. Details de la ressource
5. Warning irreversibilite
6. Checkbox confirmation obligatoire
7. Footer avec boutons

ELEMENTS
| Element             | Description                       |
|---------------------|-----------------------------------|
| Icone               | Triangle warning rouge            |
| Message             | Question de confirmation          |
| Bloc details        | Infos sur la ressource            |
| Warning             | Rappel action irreversible        |
| Checkbox            | Confirmation utilisateur requise  |

INTERACTIONS
- Clic checkbox → Activer bouton Supprimer
- Clic Annuler → Fermer sans action
- Clic Supprimer → Executer suppression (si checkbox cochee)
- Clic &#10005; → Fermer sans action

UTILISATIONS
- Suppression base de donnees
- Suppression utilisateur FTP
- Suppression multisite
- Suppression tache CRON
- Suppression variable env
- Resiliation service

API ENDPOINTS
DELETE /hosting/web/{sn}/database/{name}           → Supprimer BDD
DELETE /hosting/web/{sn}/user/{login}              → Supprimer FTP user
DELETE /hosting/web/{sn}/attachedDomain/{domain}   → Supprimer multisite
DELETE /hosting/web/{sn}/cron/{id}                 → Supprimer CRON
DELETE /hosting/web/{sn}/envVar/{key}              → Supprimer envvar
