STRUCTURE MODAL
1. Header avec titre
2. Affichage base source (lecture seule)
3. Selection destination (radio)
4. Nom nouvelle base (si nouvelle)
5. Options de copie (checkboxes)
6. Banniere warning duree
7. Info espace disponible
8. Footer avec boutons

OPTIONS DESTINATION
| Option           | Description                     |
|------------------|---------------------------------|
| Nouvelle base    | Creer une copie independante    |
| Base existante   | Ecraser une base existante      |

OPTIONS COPIE
| Option           | Description                     |
|------------------|---------------------------------|
| Structure        | Tables, index, contraintes      |
| Donnees          | Contenu des tables              |
| Procedures       | Stored procedures et functions  |
| Triggers         | Declencheurs                    |

VALIDATION
- Nom base unique
- Espace suffisant
- Pas de caracteres speciaux dans nom

BOUTONS FOOTER
| Bouton   | Action                        |
|----------|-------------------------------|
| Annuler  | Fermer sans copier            |
| Copier   | Lancer la copie               |

INTERACTIONS
- Radio → Afficher/masquer champ nom
- Checkboxes → Options de copie
- Clic Copier → POST copy

API ENDPOINTS
GET /hosting/web/{sn}/database/{name}             → Info base source
GET /hosting/web/{sn}/database/quota              → Espace disponible
POST /hosting/web/{sn}/database/{name}/copy       → Lancer copie
