DONNEES DYNAMIQUES

| Domaines couverts            | ssl → subjects                                |
| Emetteur                     | ssl → issuer                                  |
| Algorithme                   | ssl → type + bits                             |
| Date emission                | ssl → creationDate                            |
| Date expiration              | ssl → expirationDate                          |
| Empreintes                   | ssl → fingerprint                             |

STRUCTURE MODAL
1. Header avec titre
2. Banniere statut (valide/invalide)
3. Informations certificat
4. Empreintes (SHA-256, SHA-1)
5. Toggle renouvellement auto
6. Boutons actions
7. Footer

INFORMATIONS AFFICHEES
| Champ              | Description                        |
|--------------------|------------------------------------|
| Type               | Let's Encrypt, Sectigo, etc.       |
| Domaine principal  | Domaine du certificat              |
| Domaines couverts  | SAN (Subject Alternative Names)    |
| Emetteur           | Autorite de certification          |
| Algorithme         | RSA 2048, ECDSA, etc.              |
| Dates              | Emission et expiration             |
| Jours restants     | Jusqu'a expiration                 |

ACTIONS
| Action                | Description                        |
|-----------------------|------------------------------------|
| Regenerer             | Forcer regeneration                |
| Exporter              | Telecharger certificat + cle       |

INTERACTIONS
- Toggle renouvellement → Activer/desactiver
- Clic Regenerer → Forcer renouvellement
- Clic Exporter → Telecharger fichiers
- Clic copie empreinte → Copier dans presse-papier

API ENDPOINTS
GET /hosting/web/{sn}/ssl                         → Details SSL
POST /hosting/web/{sn}/ssl/regenerate             → Regenerer
GET /hosting/web/{sn}/ssl/export                  → Exporter
