STRUCTURE MODAL
1. Header avec titre
2. Banniere info
3. Selection module (cards visuelles)
4. Dropdown domaine
5. Input chemin installation
6. Identifiant administrateur
7. Mot de passe administrateur
8. Email administrateur
9. Dropdown langue
10. Footer avec boutons

MODULES DISPONIBLES
| Module      | Description                        |
|-------------|------------------------------------|
| WordPress   | CMS blog/site                      |
| PrestaShop  | E-commerce                         |
| Joomla      | CMS polyvalent                     |
| Drupal      | CMS entreprise                     |

CHAMPS
| Champ           | Type     | Requis | Validation                    |
|-----------------|----------|--------|-------------------------------|
| Module          | Select   | Oui    | Un parmi la liste             |
| Domaine         | Select   | Oui    | Domaine attache               |
| Chemin          | Text     | Non    | Path valide                   |
| Identifiant     | Text     | Oui    | 3-32 chars                    |
| Mot de passe    | Password | Oui    | Min 8, complexite             |
| Email           | Email    | Oui    | Format email valide           |
| Langue          | Select   | Non    | Defaut: Francais              |

INTERACTIONS
- Clic card module → Selection du CMS
- Clic dropdown → Ouvrir selection
- Clic oeil → Afficher/masquer mot de passe
- Clic generateur → Generer mot de passe
- Clic Installer → Lancer installation

API ENDPOINTS
GET /hosting/web/moduleList                       → Liste modules disponibles
GET /hosting/web/{sn}/attachedDomain              → Domaines attaches
POST /hosting/web/{sn}/module                     → Installer module
