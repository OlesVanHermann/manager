STRUCTURE MODAL (fullscreen-like)
1. Header avec titre
2. Toolbar (chemin, navigation, actions)
3. En-tetes colonnes
4. Liste fichiers/dossiers
5. Banniere info quota
6. Zone drag & drop
7. Footer avec bouton fermer

TOOLBAR ACTIONS
| Action           | Description                 |
|------------------|-----------------------------|
| ↑ Parent         | Remonter d'un niveau        |
| ↻ Rafraichir     | Recharger contenu           |
| Nouveau dossier  | Creer dossier               |
| Upload           | Envoyer fichiers            |
| Telecharger      | Download selection          |

COLONNES FICHIERS
| Colonne     | Description                 |
|-------------|-----------------------------|
| Icone       | Type fichier (couleur)      |
| Nom         | Nom fichier/dossier         |
| Taille      | Ko/Mo/Go (- pour dossiers)  |
| Modifie     | Date derniere modification  |
| Permissions | Format Unix (rwx)           |
| Actions     | Menu contextuel             |

ICONES PAR TYPE
| Type    | Couleur | Extensions           |
|---------|---------|----------------------|
| Dossier | Orange  | -                    |
| PHP     | Violet  | .php                 |
| CSS     | Bleu    | .css                 |
| JS      | Jaune   | .js                  |
| Image   | Vert    | .jpg, .png, .gif     |
| Config  | Gris    | .htaccess, .env      |

ACTIONS CONTEXTUELLES
- Renommer
- Editer (fichiers texte)
- Copier
- Deplacer
- Permissions
- Supprimer

DRAG & DROP
- Zone pour upload fichiers
- Support multi-fichiers
- Progress bar pendant upload

BOUTONS FOOTER
| Bouton  | Action           |
|---------|------------------|
| Fermer  | Fermer modal     |

INTERACTIONS
- Double-clic dossier → Ouvrir
- Clic fichier → Selectionner
- Drag file → Zone drop → Upload
- Clic Action → Menu contextuel

API ENDPOINTS
GET /hosting/web/{sn}/ftp/browse/{path}           → Contenu dossier
POST /hosting/web/{sn}/ftp/upload                 → Upload fichier
POST /hosting/web/{sn}/ftp/mkdir                  → Creer dossier
DELETE /hosting/web/{sn}/ftp/{path}               → Supprimer
PUT /hosting/web/{sn}/ftp/{path}                  → Renommer/deplacer
