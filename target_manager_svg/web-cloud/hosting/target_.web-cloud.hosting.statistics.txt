# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Logs actif)
   - NAV4: Logs | Statistiques (actif) | Utilisateurs
   - TITRE
   - FILTRES: Type, Période, Grouper
   - GRAPHIQUE: Courbe avec aire
   - TUILES RÉSUMÉ: Moyenne, Maximum, Minimum

# NAV4 SOUS-TABS
| Tab          | Description                    |
|--------------|--------------------------------|
| Logs         | Visualisation des logs         |
| Statistiques | Statistiques de trafic (actif) |
| Utilisateurs | Gestion utilisateurs           |

# FILTRES
| Filtre  | Valeurs exemple                      |
|---------|--------------------------------------|
| Type    | Requêtes HTTP, Bande passante, etc.  |
| Période | Jour, Semaine, Mois, Année           |
| Grouper | Toutes les requêtes, Par code HTTP   |

# GRAPHIQUE
- Type: Ligne avec aire (fill opacity 0.1)
- Couleur: Bleu OVH (#0050D7)
- Axe X: Dates selon période
- Axe Y: Valeurs avec grille
- Points: Cercles r=4

# TUILES RÉSUMÉ
| Métrique | Description              |
|----------|--------------------------|
| Moyenne  | Valeur moyenne période   |
| Maximum  | Pic le plus haut         |
| Minimum  | Valeur la plus basse     |

# INTERACTIONS
- Changement filtre → Actualise graphique
- Survol point → Tooltip avec valeur exacte

# API ENDPOINTS
GET /hosting/web/{sn}/statistics?type=X&period=Y&aggregateBy=Z
