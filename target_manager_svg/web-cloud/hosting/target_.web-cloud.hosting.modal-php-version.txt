BADGES STATUT
| Badge       | Couleur | Signification                    |

| Derniere    | Bleu    | Version la plus recente          |
| Recommandee | Vert    | Equilibre stabilite/performance  |
| Actuelle    | Gris    | Version actuellement utilisee    |
| Securite    | Orange  | Support securite uniquement      |
| EOL         | Rouge   | Plus supporte, a migrer          |

DATES SUPPORT
- Support actif: nouvelles fonctionnalites + securite
- Support securite: corrections securite uniquement
- EOL: aucun support, vulnerabilites non corrigees

INTERACTIONS

- Clic Annuler → Fermer modal

GET /hosting/web/{sn}/availableConfigurations          → Versions dispo
PUT /hosting/web/{sn}                                  → Modifier version
