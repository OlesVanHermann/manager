STRUCTURE PAGE
1. Header "Emails automatises"
2. Card statut service
   - Statut (Actif/Bloque)
   - Emails envoyes (24h) / quota
   - Boutons: Purger file, Bloquer
3. Banniere warning si bounce eleve
4. Statistiques 7 jours
   - Envoyes, Delivres, Rejetes
5. Banniere info

STATUTS EMAIL
| Statut  | Description                    |
|---------|--------------------------------|
| Actif   | Service email actif            |
| Bloque  | Service bloque (spam/bounce)   |
| Purge   | File d'attente en purge        |

API ENDPOINTS
GET    /hosting/web/{sn}/email           -> Etat service email
POST   /hosting/web/{sn}/email/purge     -> Purger file d'attente
POST   /hosting/web/{sn}/email/block     -> Bloquer emails
POST   /hosting/web/{sn}/email/unblock   -> Debloquer emails
GET    /hosting/web/{sn}/email/bounces   -> Statistiques bounce
GET    /hosting/web/{sn}/email/statistics -> Stats envoi
