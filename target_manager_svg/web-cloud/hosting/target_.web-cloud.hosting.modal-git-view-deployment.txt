STRUCTURE MODAL (large)
1. Header avec titre
2. Section infos déploiement
3. Zone logs (style terminal)
4. Boutons actions
5. Footer

INFOS DEPLOIEMENT
| Champ      | Description                      |
|------------|----------------------------------|
| ID         | Identifiant unique               |
| Statut     | Succès/Échec/En cours            |
| Date       | Date et heure                    |
| Durée      | Temps total                      |
| Repository | URL du dépôt git                 |
| Branche    | Branche déployée                 |
| Commit     | Hash court + message             |

STATUTS DEPLOIEMENT
| Statut    | Badge     | Couleur  |
|-----------|-----------|----------|
| Succès    | success   | Vert     |
| Échec     | error     | Rouge    |
| En cours  | warning   | Orange   |

LOGS TERMINAL
| Couleur    | Signification           |
|------------|-------------------------|
| Vert       | Timestamps, succès      |
| Blanc      | Messages normaux        |
| Bleu       | Infos, liens            |
| Orange     | Warnings                |
| Rouge      | Erreurs                 |

ACTIONS DISPONIBLES
| Bouton          | Action                     |
|-----------------|----------------------------|
| Copier les logs | Clipboard                  |
| Télécharger     | Download logs file         |
| Relancer        | POST redeploy              |
| Fermer          | Fermer modal               |

INTERACTIONS
- Clic Copier → Copie au clipboard
- Clic Télécharger → Download fichier
- Clic Relancer → POST redeploy

API ENDPOINTS
GET /hosting/web/{sn}/git                        → Config git
GET /hosting/web/{sn}/git/deploy/{id}            → Détail déploiement
GET /hosting/web/{sn}/git/deploy/{id}/logs       → Logs
POST /hosting/web/{sn}/git/deploy                → Relancer
