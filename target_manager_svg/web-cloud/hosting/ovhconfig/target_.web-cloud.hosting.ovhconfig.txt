# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, Runtimes actif
   - TITRE: Configuration PHP (.ovhconfig)
   - DESCRIPTION + BANNIERE INFO
   - SECTION: Version PHP (radio cards)
   - SECTION: Environnement d'exécution (radio cards)
   - SECTION: Environnement (radio cards)
   - SECTION: Application Firewall (radio cards)
   - BOUTONS: Enregistrer / Annuler

# VERSIONS PHP
- PHP 7.4 - Fin de vie (deprecated)
- PHP 8.1 - LTS (Long Term Support)
- PHP 8.2 - Recommandé (par défaut)
- PHP 8.3 - Dernière version

# ENVIRONNEMENTS D'EXÉCUTION
- phpcgi - Legacy, compatibilité ancienne
- php (FPM) - FastCGI Process Manager, recommandé

# MODES ENVIRONNEMENT
- development - Erreurs PHP affichées
- production - Erreurs masquées (recommandé en production)

# APPLICATION FIREWALL
- Désactivé - Pas de protection WAF
- Activé - Protection contre attaques web (XSS, SQL injection, etc.)

# ACTIONS
- Enregistrer → PUT /hosting/web/{sn}/ovhConfig
- Annuler → Retour aux valeurs actuelles

# API ENDPOINTS
GET /hosting/web/{serviceName}/ovhConfig
PUT /hosting/web/{serviceName}/ovhConfig
GET /hosting/web/{serviceName}/ovhConfig/availableEngineVersions
