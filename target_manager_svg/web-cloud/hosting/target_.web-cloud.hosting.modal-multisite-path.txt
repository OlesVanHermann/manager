STRUCTURE MODAL
1. Header avec titre
2. Affichage domaine concerne
3. Dossier racine actuel (lecture seule)
4. Champ nouveau dossier
5. Suggestions de dossiers existants
6. Banniere info
7. Footer avec boutons

CHAMP NOUVEAU DOSSIER
- Input editable
- Autocompletion basee sur FTP
- Commence par /www

SUGGESTIONS DOSSIERS
| Dossier      | Description            |
|--------------|------------------------|
| /www         | Racine par defaut      |
| /www/public  | Convention public dir  |
| /www/web     | Convention Symfony     |
| /www/html    | Convention Apache      |

BOUTONS FOOTER
| Bouton   | Action                        |
|----------|-------------------------------|
| Annuler  | Fermer sans modifier          |
| Modifier | Appliquer nouveau dossier     |

INTERACTIONS
- Clic suggestion → Remplir champ
- Clic Modifier → PUT path

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}     → Info domaine
PUT /hosting/web/{sn}/attachedDomain/{domain}     → Modifier path
