STRUCTURE MODAL
1. Header avec titre
2. Banniere info nouveau certificat
3. Info certificat actuel
4. Liste domaines inclus (checkboxes)
5. Warning operation
6. Raisons courantes de regeneration
7. Footer avec boutons

INFORMATIONS CERTIFICAT ACTUEL
| Champ               | Description                    |
|---------------------|--------------------------------|
| Type                | Let's Encrypt, Sectigo, etc.   |
| Statut              | Valide, Expire, En erreur      |
| Date expiration     | Date limite de validite        |
| Nombre domaines     | Domaines couverts              |
| Derniere generation | Date/heure derniere generation |

DOMAINES INCLUS
- Liste tous les domaines avec SSL active
- Checkboxes pre-coches (non modifiables)
- Seuls les domaines actifs sont inclus

PROCESSUS REGENERATION
1. Validation DNS des domaines
2. Generation CSR
3. Demande certificat Let's Encrypt
4. Installation sur l'hebergement

INTERACTIONS
- Clic Regenerer → Lancer regeneration
- Clic Annuler → Fermer modal

API ENDPOINTS
GET /hosting/web/{sn}/ssl                              → Info certificat
POST /hosting/web/{sn}/ssl/regenerate                  → Regenerer
