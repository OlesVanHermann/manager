STRUCTURE MODAL
1. Header rouge (action dangereuse)
2. Icone warning
3. Question de confirmation
4. Info depot a dissocier
5. Liste consequences
6. Checkbox confirmation obligatoire
7. Footer avec boutons

CONSEQUENCES AFFICHEES
| Type     | Description                                    |
|----------|------------------------------------------------|
| Danger   | Suppression association, desactivation auto    |
| Info     | Fichiers conserves, reassociation possible     |

REGLES VALIDATION
- Checkbox confirmation obligatoire avant dissociation
- Bouton Dissocier desactive tant que non coche

INTERACTIONS
- Checkbox → Activer/desactiver bouton Dissocier
- Clic Annuler → Fermer modal
- Clic Dissocier → Supprimer association Git

API ENDPOINTS
DELETE /hosting/web/{sn}/attachedDomain/{d}/git    → Dissocier depot
