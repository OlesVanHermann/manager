2. Champ nom runtime
3. Selection moteur (PHP/Node.js/Python)
4. Dropdown version
5. Champ chemin application
6. Champ repertoire public
7. Apercu variables env associees
8. Liste sites utilisant ce runtime
9. Info impact modifications
10. Footer avec boutons

MOTEURS DISPONIBLES
| Moteur   | Versions disponibles          |
|----------|-------------------------------|
| PHP      | 7.4, 8.0, 8.1, 8.2, 8.3       |
| Node.js  | 14, 16, 18, 20                |
| Python   | 3.8, 3.9, 3.10, 3.11          |

CHEMINS
- Chemin application: repertoire racine du code
- Repertoire public: DocumentRoot servi par le web server

VARIABLES ENVIRONNEMENT
- Affichage apercu des variables associees
- Lien vers gestion complete des variables

IMPACT MODIFICATION
Les modifications s'appliquent a tous les sites
utilisant cet environnement d'execution.

INTERACTIONS
- Clic moteur → Selectionner
- Dropdown version → Choisir version
- Input chemins → Modifier
- Lien variables → Ouvrir gestion variables
- Clic Enregistrer → Sauvegarder modifications

API ENDPOINTS
GET /hosting/web/{sn}/runtime/{id}                     → Details runtime
PUT /hosting/web/{sn}/runtime/{id}                     → Modifier runtime
GET /hosting/web/{sn}/envVar                           → Variables env
