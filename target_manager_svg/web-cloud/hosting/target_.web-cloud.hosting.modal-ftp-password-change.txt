STRUCTURE MODAL
1. Header avec titre

3. Champ nouveau mot de passe
4. Bouton génération

6. Indicateur force
7. Liste exigences
8. Warning mise à jour
9. Footer avec boutons

CHAMPS MOT DE PASSE
| Champ        | Type     | Validation               |
|--------------|----------|--------------------------|
| Nouveau      | Password | Exigences complexité     |
| Confirmation | Password | Doit correspondre        |

EXIGENCES MOT DE PASSE
| Règle                | Requis  |
|---------------------|---------|
| Min 12 caractères   | Oui     |
| Une majuscule       | Oui     |
| Une minuscule       | Oui     |
| Un chiffre          | Oui     |
| Caractère spécial   | Oui     |
| Correspondance      | Oui     |

INDICATEUR FORCE
| Niveau  | Couleur | Largeur |
|---------|---------|---------|
| Faible  | Rouge   | 25%     |
| Moyen   | Orange  | 50%     |
| Bon     | Jaune   | 75%     |
| Fort    | Vert    | 100%    |

GENERATION
- Bouton génère mot de passe aléatoire
- Respecte toutes les exigences
- Copie automatique au clipboard

BOUTONS FOOTER
| Bouton     | Action                     |
|------------|----------------------------|
| Annuler    | Fermer sans modifier       |
| Enregistrer| PUT password               |

INTERACTIONS
- Input → Saisir mot de passe
- Toggle → Afficher/masquer
- Clic Générer → Génération auto
- Clic Enregistrer → PUT password

API ENDPOINTS
GET /hosting/web/{sn}/user/{login}               → Info utilisateur
PUT /hosting/web/{sn}/user/{login}/changePassword→ Modifier mot de passe
