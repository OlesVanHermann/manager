DONNEES DYNAMIQUES

|------------------------------|-----------------------------------------------|
| URL depot                    | GET /hosting/web/{sn}/attachedDomain/{d}/git  |
| Branche                      | git config → branch                           |
| Chemin deploiement           | git config → path                             |
| Site associe                 | git config → attachedDomain                   |
| Liste commits                | Fetch depuis depot Git                        |

================================================================================

STRUCTURE MODAL
1. Header avec titre
2. Warning banniere (ecrasement)
3. Info depot associe (URL, branche, chemin, site)
4. Selection type deploiement (dernier/specifique)
5. Liste derniers commits avec selection
6. Option vider repertoire
7. Footer avec boutons

TYPES DEPLOIEMENT
| Type              | Description                               |
|-------------------|-------------------------------------------|
| Dernier commit    | Deploie automatiquement HEAD de la branche|
| Commit specifique | Permet de selectionner un commit precis   |

AFFICHAGE COMMITS
- Hash court (7 caracteres)
- Message du commit
- Date relative
- Selection par radio button

INTERACTIONS
- Clic type deploiement → Selectionner
- Clic commit → Selectionner commit specifique
- Checkbox → Activer option vider repertoire
- Clic Deployer → Lancer deploiement

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{d}/git           → Config Git
POST /hosting/web/{sn}/attachedDomain/{d}/git/deploy   → Lancer deploiement
