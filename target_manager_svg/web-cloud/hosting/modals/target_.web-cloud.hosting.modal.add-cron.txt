# STRUCTURE MODAL
1. HEADER - Titre "Créer une tâche planifiée" + close
2. FORMULAIRE:
   - Commande à exécuter *
   - Description
   - Fréquence (Simple/Avancé toggle)
     - Simple: select prédéfini + heure
     - Avancé: expression cron complète
   - Notifications (checkboxes: erreur, toutes)
   - Email destination
   - Langage (PHP version)
   - État initial (actif checkbox)
3. ACTIONS: Annuler, Créer

# FRÉQUENCES SIMPLES
- Toutes les minutes
- Toutes les heures
- Tous les jours
- Toutes les semaines
- Tous les mois

# API ENDPOINTS
POST /hosting/web/{serviceName}/cron - Créer tâche CRON
GET /hosting/web/{serviceName}/cron - Liste tâches
