# STRUCTURE MODAL
1. HEADER - Rouge "Supprimer la base de données" + close
2. MESSAGE - Nom base à supprimer
3. BANNIÈRE WARNING - Action irréversible
4. CONFIRMATION - Input "SUPPRIMER"
5. ACTIONS: Annuler, Supprimer (désactivé jusqu'à confirmation)

# COMPORTEMENT
- Bouton Supprimer grisé par défaut
- S'active quand l'utilisateur tape "SUPPRIMER"
- Case-sensitive

# API ENDPOINTS
DELETE /hosting/web/{serviceName}/database/{name} - Supprimer base
