# STRUCTURE MODAL
1. HEADER - Titre "Installer un module en 1 clic" + close
2. CATÉGORIE FILTER - Tabs: Tous, CMS, E-commerce, Forum, Autre
3. MODULE GRID - Cartes modules avec logo (sélection unique)
4. FORMULAIRE CONFIGURATION (selon module sélectionné):
   - Domaine d'installation * (select)
   - Chemin d'installation
   - Identifiant admin *
   - Mot de passe admin *
   - Email admin *
   - Langue du site (select)
   - Base de données (créer auto / existante)
5. BANNIÈRE INFO
6. ACTIONS: Annuler, Installer
