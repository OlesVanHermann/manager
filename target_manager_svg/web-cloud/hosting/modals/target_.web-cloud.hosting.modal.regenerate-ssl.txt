# STRUCTURE MODAL
1. HEADER - Titre "Régénérer le certificat SSL" + close
2. LISTE DOMAINES - Checkboxes pour sélectionner domaines
   - Coché: domaines actuellement couverts
   - Décoché: domaines non résolus (DNS pas configuré)
3. BANNIÈRE WARNING - Temps de génération
4. INFO CERTIFICAT - Provider, validité, renouvellement
5. ACTIONS: Annuler, Régénérer

# ÉTATS DOMAINES
| État      | Badge     | Description              |
|-----------|-----------|--------------------------|
| Principal | Gris      | Domaine principal        |
| (vide)    | -         | Domaine valide           |
| Non résolu| Gris      | DNS pas configuré        |

# API ENDPOINTS
POST /hosting/web/{serviceName}/ssl/regenerate - Régénérer SSL
GET /hosting/web/{serviceName}/attachedDomain - Liste domaines
