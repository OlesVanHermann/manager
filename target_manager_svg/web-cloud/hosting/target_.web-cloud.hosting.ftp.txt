# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (FTP-SSH actif)
   - BLOC INFO: Serveur FTP/SFTP, Serveur SSH, Login principal, Accès web
   - TOOLBAR: Créer utilisateur | Restaurer snapshot | Générer snapshot | Recherche
   - TABLE: Login, Répertoire, FTP, SFTP, SSH, État, Dernière connexion, Actions
   - PAGINATION

# INTERACTIONS
- Clic "Créer utilisateur" → Modal création
- Clic "Restaurer snapshot" → Modal sélection snapshot
- Clic "Générer snapshot" → Modal confirmation
- Clic "Net2FTP" → Ouverture nouvel onglet
- Clic ⋮ → Menu contextuel (Changer MDP, Modifier, Supprimer)

# API ENDPOINTS
GET /hosting/web/{sn}/user              - Liste utilisateurs FTP
GET /hosting/web/{sn}/user/{login}      - Détails utilisateur
POST /hosting/web/{sn}/user             - Créer utilisateur
PUT /hosting/web/{sn}/user/{login}      - Modifier utilisateur
POST /hosting/web/{sn}/user/{login}/changePassword - Changer mot de passe
DELETE /hosting/web/{sn}/user/{login}   - Supprimer utilisateur
GET /hosting/web/{sn}/dump              - Liste snapshots
POST /hosting/web/{sn}/dump/{id}/restore - Restaurer snapshot
