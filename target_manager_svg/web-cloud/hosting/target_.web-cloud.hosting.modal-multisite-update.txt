STRUCTURE MODAL
1. Header avec titre
2. Affichage domaine (lecture seule)
3. Champ dossier racine
4. Dropdown firewall WAF
5. Badge SSL (lecture seule)
6. Toggle forcer HTTPS
7. Toggle CDN
8. Dropdown restriction pays
9. Toggle logs separes
10. Footer avec boutons

OPTIONS FIREWALL
| Valeur        | Description                    |
|---------------|--------------------------------|
| Active        | Protection activee             |
| Inactive      | Protection desactivee          |
| Maintenance   | Pages bloquees sauf admin      |

OPTIONS RESTRICTION PAYS
| Valeur          | Description                  |
|-----------------|------------------------------|
| Aucune          | Pas de restriction           |
| Europe          | Seulement pays EU            |
| France          | Seulement France             |
| Personnalise    | Liste de pays custom         |

TOGGLES
| Option         | Description                   |
|----------------|-------------------------------|
| Forcer HTTPS   | Redirection HTTP → HTTPS      |
| CDN            | Activer cache CDN             |
| Logs separes   | Fichier log dedie             |

BOUTONS FOOTER
| Bouton      | Action                         |
|-------------|--------------------------------|
| Annuler     | Fermer sans sauvegarder        |
| Enregistrer | Sauvegarder modifications      |

INTERACTIONS
- Modification champs → Activer bouton Enregistrer
- Toggle → Changement immediat visuel
- Clic Enregistrer → PUT attachedDomain

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}     → Parametres actuels
PUT /hosting/web/{sn}/attachedDomain/{domain}     → Modifier parametres
