STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hebergement actif (indicateur bleu x=235)
3. LEFT PANEL (280px)
   - NAV3 Groupement: [General][Sites][Offre][Expert] - 4 boutons
   - Recherche: Barre avec icone loupe
   - Filtre: Dropdown par statut
   - Liste services: Cards avec nom, offre, badge statut
   - Pagination: En bas du panel
4. RIGHT PANEL (1096px)
   - Header: Titre service + boutons Refresh/Commander
   - NAV4: [General][Statistiques][Logs][Modules][SEO] - 5 tabs
   - Contenu: 4 blocs (Infos generales, Configuration, Abonnement, Liens utiles)

INTERACTIONS
- Clic NAV3 [General] → Affiche tabs General (5 tabs)
- Clic NAV3 [Sites] → Affiche tabs Sites (5 tabs)
- Clic NAV3 [Offre] → Affiche tabs Offre (4 tabs)
- Clic NAV3 [Expert] → Affiche tabs Expert (10 tabs)
- Clic NAV4 General → Affiche dashboard general (cette page)
- Clic NAV4 Statistiques → Affiche statistiques
- Clic NAV4 Logs → Affiche logs
- Clic NAV4 Modules → Affiche modules
- Clic NAV4 SEO → Affiche Local SEO
- Clic service dans sidebar → Selection et chargement details
- Clic Refresh → Recharge donnees
- Clic Commander → Ouvre modal commande

API ENDPOINTS
GET /hosting/web                              → Liste hebergements
GET /hosting/web/{serviceName}                → Details hebergement
GET /hosting/web/{serviceName}/ssl            → Info SSL
GET /hosting/web/{serviceName}/cdn            → Info CDN
GET /hosting/web/{serviceName}/serviceInfos   → Info abonnement
