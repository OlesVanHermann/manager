# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (CDN actif)
   - NAV4: État | Paramètres (actif)
   - TITRE + SOUS-TITRE
   - BANDEAU INFO: Propagation 5 min
   - SECTION: Options de performance (4 cartes)
   - SECTION: Options de sécurité (2 cartes)
   - SECTION: Règles de cache (tableau)

# OPTIONS DE PERFORMANCE
| Option                    | Description                                | Toggle |
|---------------------------|--------------------------------------------|--------|
| Always Online             | Cache si serveur indisponible              | on/off |
| HTTP/2                    | Protocole HTTP/2                           | on/off |
| Compression Brotli        | Compression ressources                     | on/off |
| En-têtes géolocalisation  | Headers localisation visiteur              | on/off |

# OPTIONS DE SÉCURITÉ
| Option       | Description                           | Toggle |
|--------------|---------------------------------------|--------|
| HSTS         | Force HTTPS                           | on/off |
| Mixed Content| Réécriture HTTP→HTTPS                 | on/off |

# TABLE RÈGLES DE CACHE
| Colonne  | Largeur | Contenu                          |
|----------|---------|----------------------------------|
| Nom      | 180px   | Identifiant de la règle          |
| Type     | 120px   | Badge Extension/URI              |
| Pattern  | 300px   | Motif de correspondance          |
| TTL      | 130px   | Durée de vie du cache            |
| Actions  | 150px   | Modifier, Supprimer              |

# INTERACTIONS
- Toggle switch → PUT .../settings (activation/désactivation)
- Clic "+ Ajouter une règle" → Modal création règle
- Clic "Modifier" → Modal édition règle
- Clic "Supprimer" → Modal confirmation suppression
- Clic "Documentation ↗" → Lien externe doc

# API ENDPOINTS
GET /hosting/web/{sn}/cdn/{domain}/settings          - Configuration complète
PUT /hosting/web/{sn}/cdn/{domain}/settings          - Modifier options
GET /hosting/web/{sn}/cdn/{domain}/cacheRules        - Liste règles cache
POST /hosting/web/{sn}/cdn/{domain}/cacheRules       - Ajouter règle
PUT /hosting/web/{sn}/cdn/{domain}/cacheRules/{id}   - Modifier règle
DELETE /hosting/web/{sn}/cdn/{domain}/cacheRules/{id} - Supprimer règle
