# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (CDN actif)
   - NAV4: État | Paramètres
   - TITRE + BOUTONS ACTIONS
   - BLOC STATUT CDN
   - TABLE CONFIGURATION PAR DOMAINE
   - PAGINATION

# NAV4 SOUS-TABS
| Tab        | Description                    |
|------------|--------------------------------|
| État       | Vue d'ensemble CDN (actif)     |
| Paramètres | Configuration détaillée        |

# BOUTONS ACTIONS
| Bouton         | Style     | Action                    |
|----------------|-----------|---------------------------|
| Vider le cache | Outline   | POST .../cdn/flush        |
| Configurer CDN | Outline   | Navigation vers paramètres|
| Désactiver     | Secondary | DELETE .../cdn            |

# BLOC STATUT CDN
| Champ   | Contenu                         |
|---------|---------------------------------|
| Type    | CDN Basic / CDN Advance / etc.  |
| Backend | Adresse du cluster backend      |
| Status  | Badge Actif/Désactivé           |

# COLONNES TABLE
| Colonne  | Largeur | Contenu                          |
|----------|---------|----------------------------------|
| Domaine  | 260px   | Nom du domaine                   |
| CDN      | 100px   | Badge Actif/Désactivé            |
| Firewall | 120px   | Badge Actif/Désactivé            |
| Cache    | 120px   | Durée TTL (24h, 1h, etc.)        |
| HTTP/2   | 120px   | Badge Actif/Désactivé            |
| Actions  | 56px    | Menu ⋮                           |

# INTERACTIONS
- Clic "Vider le cache" → Modal confirmation vidage
- Clic "Configurer CDN" → Page paramètres CDN
- Clic ⋮ → Menu (Configurer, Désactiver)

# API ENDPOINTS
GET /hosting/web/{sn}/cdn                          - Statut CDN global
POST /hosting/web/{sn}/cdn/flush                   - Vider cache
PUT /hosting/web/{sn}/cdn                          - Configurer CDN
DELETE /hosting/web/{sn}/cdn                       - Désactiver CDN
GET /hosting/web/{sn}/attachedDomain/{d}/cdn       - Config CDN par domaine
PUT /hosting/web/{sn}/attachedDomain/{d}/cdn       - Modifier config domaine
