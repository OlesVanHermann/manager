STRUCTURE MODAL
1. Header avec titre
2. Bannière info SAN
3. Liste domaines avec checkboxes
4. Sélection type certificat (radio)
5. Résumé et prix
6. Warning validation
7. Footer avec boutons

CERTIFICAT SAN
| Avantage               | Description                    |
|------------------------|--------------------------------|
| Multi-domaines         | Jusqu'à 100 domaines           |
| Un seul certificat     | Gestion simplifiée             |
| Économique             | Moins cher que certificats séparés|
| Flexible               | Ajouter/supprimer domaines     |

TYPES CERTIFICAT
| Type | Validation         | Délai      | Prix    |
|------|-------------------|------------|---------|
| DV   | Domaine (DNS)     | Minutes    | 49€/an  |
| OV   | Organisation      | 2-3 jours  | 149€/an |

DOMAINES
| Statut      | Badge     | Note                    |
|-------------|-----------|-------------------------|
| Principal   | Vert      | Domaine de référence    |
| Sélectionné | Checkbox  | Inclus dans certificat  |
| Non inclus  | -         | Disponible à ajouter    |

BOUTONS FOOTER
| Bouton    | Action                     |
|-----------|----------------------------|
| Annuler   | Fermer sans commander      |
| Commander | POST ssl/san order         |

INTERACTIONS
- Checkboxes → Sélectionner domaines
- Radio → Choisir type certificat
- Clic Commander → POST order

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain             → Domaines
GET /hosting/web/ssl/availableOffers             → Offres SSL
POST /hosting/web/{sn}/ssl                       → Commander SSL SAN
