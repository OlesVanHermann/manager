STRUCTURE MODAL
1. Header rouge avec titre
2. Icone corbeille
3. Message confirmation
4. Details module (icone, nom, URL, date)
5. Warning banniere
6. Checkbox confirmation obligatoire
7. Footer avec boutons (Supprimer desactive sans checkbox)

VALIDATION
- L'utilisateur doit cocher la case pour activer le bouton Supprimer
- Bouton Supprimer grise tant que non coche

INTERACTIONS
- Clic checkbox → Active/desactive bouton Supprimer
- Clic Annuler → Fermer sans action
- Clic Supprimer → Supprimer le module (si checkbox cochee)

API ENDPOINTS
GET /hosting/web/{sn}/module/{id}                 → Details module
DELETE /hosting/web/{sn}/module/{id}              → Supprimer module
