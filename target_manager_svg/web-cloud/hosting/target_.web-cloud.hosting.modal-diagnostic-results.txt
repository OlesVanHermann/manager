STRUCTURE MODAL
1. Header avec titre
2. Banniere resume (succes/warning/erreur)
3. Liste tests avec statut
4. Resume compteurs
5. Footer avec boutons

TESTS EFFECTUES
| Test              | Description                           |
|-------------------|---------------------------------------|
| DNS               | Configuration DNS vers hebergement    |
| SSL               | Validite certificat SSL               |
| FTP               | Acces FTP operationnel                |
| Database          | Etat bases de donnees                 |
| PHP               | Version PHP supportee                 |
| Performance       | Temps de reponse                      |

STATUTS
| Statut  | Couleur | Icone | Description              |
|---------|---------|-------|--------------------------|
| Success | Vert    | ✓     | Test reussi              |
| Warning | Jaune   | !     | Avertissement            |
| Error   | Rouge   | ✗     | Echec                    |

INTERACTIONS
- Clic Nouveau diagnostic → Relancer diagnostic
- Clic Fermer → Fermer la modal

API ENDPOINTS
GET /hosting/web/{sn}/diagnostic                  → Resultats diagnostic
POST /hosting/web/{sn}/diagnostic                 → Lancer diagnostic
