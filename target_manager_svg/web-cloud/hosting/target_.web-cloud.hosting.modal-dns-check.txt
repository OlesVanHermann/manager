STRUCTURE MODAL
1. Header avec titre
2. Banniere info
3. Domaine verifie
4. Tableau enregistrements DNS avec statut
5. Configuration attendue
6. Footer avec boutons

ENREGISTREMENTS VERIFIES
| Type   | Description                        |
|--------|------------------------------------|
| A      | IPv4 vers serveur                  |
| AAAA   | IPv6 vers serveur                  |
| CNAME  | Alias www vers domaine             |
| MX     | Serveurs mail                      |

STATUTS
| Statut | Couleur | Description                    |
|--------|---------|--------------------------------|
| OK     | Vert    | Configuration correcte         |
| KO     | Rouge   | Configuration incorrecte       |

INTERACTIONS
- Clic Actualiser → Relancer verification DNS
- Clic Fermer → Fermer la modal

API ENDPOINTS
GET /hosting/web/{sn}                             → Info hebergement
GET /hosting/web/{sn}/attachedDomain/{domain}     → Config domaine
POST /hosting/web/{sn}/attachedDomain/{domain}/purgeCache → Purge DNS
