================================================================================
8. NOTES DE DESIGN
================================================================================

8.1 COHÉRENCE VISUELLE
----------------------
- Tous les panels utilisent rx=8
- Tous les boutons utilisent rx=4
- Espacement cohérent de 16-20px entre sections
- Hiérarchie typographique claire (18→14→12→11→10)

8.2 ACCESSIBILITÉ
-----------------
- Contraste suffisant texte/fond
- Icônes accompagnées de contexte visuel (colonnes nommées)
- États visuels distincts (actif/inactif, badges couleur)

8.3 RESPONSIVE NOTES
--------------------
- Design fixe 1440px (desktop)
- Left panel peut être réduit/masqué en mobile
- Tableau peut nécessiter scroll horizontal en mobile
