STRUCTURE PAGE
1. Header + bouton "Creer utilisateur"
2. Banniere info connexion FTP/SFTP/SSH
3. Table utilisateurs FTP
   - Colonnes: UTILISATEUR, CHEMIN, ETAT, SSH, ACTIONS
   - Utilisateur principal non supprimable
   - Actions: Changer MDP, Modifier, Supprimer

CONNEXION FTP
| Protocol | Serveur                        | Port |
|----------|--------------------------------|------|
| FTP      | ftp.{cluster}.hosting.ovh.net  | 21   |
| SFTP     | ftp.{cluster}.hosting.ovh.net  | 22   |
| SSH      | ssh.{cluster}.hosting.ovh.net  | 22   |

API ENDPOINTS
GET    /hosting/web/{sn}/user        -> Liste utilisateurs
POST   /hosting/web/{sn}/user        -> Creer utilisateur
PUT    /hosting/web/{sn}/user/{login}-> Modifier utilisateur
DELETE /hosting/web/{sn}/user/{login}-> Supprimer utilisateur
POST   /hosting/web/{sn}/user/{login}/changePassword -> Changer MDP

ACTIONS
- "+ Creer utilisateur" -> Modal creation
- "MDP" -> Modal changer mot de passe
- "Modifier" -> Modal modification (path, SSH)
- "x" -> Modal confirmation suppression
