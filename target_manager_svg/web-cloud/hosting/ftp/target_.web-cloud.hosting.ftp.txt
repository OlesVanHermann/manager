# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (FTP-SSH actif)
   - BLOC INFO: Serveurs FTP/SSH, Login principal, Ports
   - TOOLBAR: Refresh, Search, Créer utilisateur, Sauvegardes
   - TABLE: Utilisateurs FTP/SSH
     - Login (+ badge principal)
     - Répertoire cible
     - FTP (badge)
     - SFTP (badge)
     - SSH (badge)
     - État (badge)
     - Dernière connexion
     - Actions (...)
   - PAGINATION

# INTERACTIONS
- Clic "Créer utilisateur" -> Modal add-ftp-user
- Clic "..." -> Menu (Changer MDP, Modifier, Supprimer)
- Clic "Restaurer sauvegarde" -> Modal restore
- Clic "Générer sauvegarde" -> Modal snapshot
- Clic icône copie (C) -> Copier dans presse-papier
