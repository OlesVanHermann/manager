# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: Utilisateurs de la base {dbName}
   - Breadcrumb: {serviceName} › {dbName} › Utilisateurs
   - BANNIERE INFO
   - TOOLBAR: + Créer un utilisateur
   - TABLE: Nom, Droits, Date création, Statut, Actions
   - INFO: X utilisateurs
   - HELP LINK

# TYPES D'UTILISATEURS
- Principal (badge bleu) - Utilisateur créé avec la base
- Secondaire - Utilisateurs additionnels avec droits limités

# DROITS MySQL
- ALL PRIVILEGES - Tous les droits
- SELECT - Lecture seule
- SELECT, INSERT, UPDATE - Lecture/écriture
- SELECT, LOCK TABLES - Pour backups

# STATUTS
- Actif (vert) - Utilisateur actif
- Désactivé (gris) - Utilisateur suspendu

# ACTIONS
- + Créer un utilisateur → modal-database-user-add
- Mot de passe → Modifier le mot de passe
- ⋮ (menu) → Modifier droits / Supprimer

# API ENDPOINTS
GET /hosting/web/{serviceName}/database/{name}
GET /hosting/web/{serviceName}/database/{name}/user
POST /hosting/web/{serviceName}/database/{name}/user
PUT /hosting/web/{serviceName}/database/{name}/user/{userName}
DELETE /hosting/web/{serviceName}/database/{name}/user/{userName}
POST /hosting/web/{serviceName}/database/{name}/user/{userName}/changePassword
