DONNEES DYNAMIQUES

|------------------------------|-----------------------------------------------|

| Liste runtimes               | GET /hosting/web/{sn}/runtime                 |

STRUCTURE MODAL
1. Header avec titre
2. Warning donnee sensible
3. Champ nom variable (readonly)
4. Textarea valeur (editable)
5. Dropdown environnement d'execution
6. Exemples utilisation code
7. Info redemarrage application
8. Footer avec boutons

NOM VARIABLE
- Affiche en lecture seule (non modifiable)
- Format SCREAMING_SNAKE_CASE
- Icone cadenas indiquant immutabilite

VALEUR
- Textarea multi-lignes pour valeurs longues
- Bouton afficher/masquer (oeil)
- Support caracteres speciaux

EXEMPLES CODE
| Langage | Syntaxe                              |
|---------|--------------------------------------|
| PHP     | getenv('VARIABLE_NAME')              |
| Node.js | process.env.VARIABLE_NAME            |
| Python  | os.environ.get('VARIABLE_NAME')      |

ENVIRONNEMENT EXECUTION
- Dropdown selection runtime
- Variable accessible uniquement dans ce runtime

APPLICATION MODIFICATIONS
Les modifications necessitent un redemarrage
de l'application pour etre prises en compte.

INTERACTIONS
- Textarea → Modifier valeur
- Bouton oeil → Afficher/masquer valeur
- Dropdown runtime → Changer association
- Clic Enregistrer → Sauvegarder

API ENDPOINTS
GET /hosting/web/{sn}/envVar/{key}                     → Details variable
PUT /hosting/web/{sn}/envVar/{key}                     → Modifier variable
