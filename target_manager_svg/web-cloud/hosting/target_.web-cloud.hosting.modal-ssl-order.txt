STRUCTURE MODAL
1. Header avec titre et bouton fermer
2. Etape 1: Selection type de certificat
   - Let's Encrypt (gratuit)
   - Sectigo DV (payant)
   - Sectigo EV (payant)
3. Etape 2: Selection des domaines
   - Liste domaines attaches
   - Checkboxes selection multiple
4. Etape 3: Options
   - Redirection HTTP->HTTPS
   - Renouvellement automatique
5. Resume
6. Footer avec boutons

TYPES DE CERTIFICATS
| Type           | Prix      | Validation  | Validite | Garantie     |
|----------------|-----------|-------------|----------|--------------|
| Let's Encrypt  | Gratuit   | DV          | 90 jours | -            |
| Sectigo DV     | 49,99 EUR   | DV          | 1 an     | 250k EUR       |
| Sectigo EV     | 149,99 EUR  | EV          | 1 an     | 1.75M EUR      |

SELECTION DOMAINES
| Element     | Description                       |
|-------------|-----------------------------------|
| Principal   | Domaine principal (auto-selectionne)|
| www         | Sous-domaine www                  |
| Autres      | Autres sous-domaines attaches     |

OPTIONS
| Option                    | Description                       |
|---------------------------|-----------------------------------|
| Redirection HTTPS         | Force HTTPS via .htaccess         |
| Renouvellement auto       | Regenère avant expiration         |

INTERACTIONS
- Clic carte certificat → Selection du type
- Clic checkbox domaine → Selection/deselection
- Clic checkbox option → Activer/desactiver
- Clic Annuler → Fermer sans action
- Clic Commander → Lancer commande SSL

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain               → Domaines disponibles
GET /order/catalog/public/sslGateway               → Catalogue SSL
POST /hosting/web/{sn}/ssl                         → Commander Let's Encrypt
POST /order/cart/{id}/sslGateway                   → Commander Sectigo
POST /hosting/web/{sn}/ssl/regenerate              → Regenerer certificat
