# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (SSL actif)
   - TITRE + BOUTONS ACTIONS
   - BLOC ÉTAT: Type, Fournisseur, Dates, Renouvellement
   - BLOC DOMAINES: Liste domaines couverts
   - PANNEAU INFO: Explications Let's Encrypt

# BOUTONS ACTIONS
| Bouton          | Style     | Action                        |
|-----------------|-----------|-------------------------------|
| Régénérer SSL   | Outline   | POST .../ssl/regenerate       |
| Commander SSL   | Secondary | Redirection commande          |
| Supprimer SSL   | Danger    | DELETE .../ssl                |

# BLOC ÉTAT CERTIFICAT
| Champ          | Contenu                           |
|----------------|-----------------------------------|
| Type           | Let's Encrypt / Sectigo / etc.    |
| Fournisseur    | Autorité de certification         |
| Création       | Date/heure création               |
| Expiration     | Date/heure expiration             |
| Renouvellement | Badge Automatique / Manuel        |

# BLOC DOMAINES COUVERTS
- Liste avec indicateur vert (●)
- Tous les domaines inclus dans le certificat
- Généralement: domaine principal + sous-domaines

# INTERACTIONS
- Clic "Régénérer SSL" → Régénération du certificat
- Clic "Commander SSL" → Page commande certificat payant
- Clic "Supprimer SSL" → Modal confirmation suppression

# API ENDPOINTS
GET /hosting/web/{sn}/ssl              - État du certificat
POST /hosting/web/{sn}/ssl/regenerate  - Régénérer certificat
POST /hosting/web/{sn}/ssl             - Commander nouveau
DELETE /hosting/web/{sn}/ssl           - Supprimer certificat
