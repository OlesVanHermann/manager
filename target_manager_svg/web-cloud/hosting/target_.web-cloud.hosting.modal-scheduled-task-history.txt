STRUCTURE MODAL (large)
1. Header avec titre
2. Info tâche (nom, commande, planning)
3. 4 cartes statistiques
4. Tableau historique paginé
5. Pagination
6. Footer

INFO TACHE
| Champ     | Description                    |
|-----------|--------------------------------|
| Nom       | Identifiant tâche              |
| Commande  | Script PHP exécuté             |
| Planning  | Expression CRON + description  |

STATISTIQUES (7 JOURS)
| Métrique          | Description                    |
|-------------------|--------------------------------|
| Succès            | Nombre + pourcentage           |
| Échecs            | Nombre + pourcentage           |
| Durée moyenne     | Temps moyen d'exécution        |
| Prochaine exec    | Date prochaine exécution       |

COLONNES TABLEAU
| Colonne     | Description                    |
|-------------|--------------------------------|
| Date        | Date exécution                 |
| Heure début | Timestamp début                |
| Heure fin   | Timestamp fin                  |
| Durée       | Temps d'exécution              |
| Code retour | Exit code                      |
| Statut      | Badge Succès/Échec/Timeout     |
| Actions     | Lien vers logs                 |

STATUTS EXECUTION
| Statut  | Code | Badge   | Couleur  |
|---------|------|---------|----------|
| Succès  | 0    | success | Vert     |
| Échec   | >0   | error   | Rouge    |
| Timeout | 124  | warning | Orange   |

BOUTONS
| Bouton   | Action                     |
|----------|----------------------------|
| Voir logs| Ouvrir modal logs détail   |
| Fermer   | Fermer modal               |

INTERACTIONS
- Clic Voir logs → Modal détail
- Pagination → Navigation pages

API ENDPOINTS
GET /hosting/web/{sn}/cron/{id}                  → Info tâche
GET /hosting/web/{sn}/cron/{id}/execution        → Liste exécutions
