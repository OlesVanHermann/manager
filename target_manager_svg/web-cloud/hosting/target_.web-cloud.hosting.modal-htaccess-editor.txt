STRUCTURE MODAL (large)
1. Header avec titre
2. Warning fichier sensible
3. Selecteur fichier + actions (rafraichir, sauvegarder)
4. Editeur code avec numeros lignes
5. Actions rapides (snippets)
6. Info test apres modification
7. Footer avec boutons (Annuler, Valider, Enregistrer)

SELECTEUR FICHIER
- Dropdown avec tous les .htaccess du site
- /www/.htaccess (principal)
- /www/admin/.htaccess
- etc.

EDITEUR CODE
| Element            | Description                    |
|--------------------|--------------------------------|
| Numeros lignes     | Colonne gauche grise           |
| Coloration syntaxe | Directives, variables, strings |
| Police             | Monospace                      |
| Fond               | Sombre (#1F2937)               |

COLORATION SYNTAXE
| Type        | Couleur  | Exemple           |
|-------------|----------|-------------------|
| Commentaire | Gris     | # Comment         |
| Directive   | Rose     | RewriteEngine     |
| Variable    | Violet   | %{HTTPS}          |
| String      | Vert     | "access plus 1y"  |
| Tag         | Bleu     | <IfModule>        |

ACTIONS RAPIDES
| Snippet           | Description                    |
|-------------------|--------------------------------|
| HTTPS redirect    | Force HTTPS via RewriteRule    |
| Cache headers     | Configure expiration cache     |
| Gzip compress     | Active compression mod_deflate |
| Security headers  | X-Frame, X-XSS, etc.           |
| IP block          | Deny from IP                   |

BOUTONS FOOTER
| Bouton     | Action                              |
|------------|-------------------------------------|
| Annuler    | Fermer sans sauvegarder             |
| Valider    | Verifier syntaxe                    |
| Enregistrer| Sauvegarder modifications           |

INTERACTIONS
- Dropdown fichier → Charger contenu
- Clic snippet → Inserer code
- Clic Valider → Verifier syntaxe
- Clic Enregistrer → Sauvegarder

API ENDPOINTS
GET /hosting/web/{sn}/htaccess                         → Liste fichiers
GET /hosting/web/{sn}/htaccess/{path}                  → Lire contenu
PUT /hosting/web/{sn}/htaccess/{path}                  → Modifier
POST /hosting/web/{sn}/htaccess/{path}/validate        → Valider syntaxe
