# API ENDPOINTS

GET    /hosting/web/{serviceName}
       -> Configuration actuelle (.ovhconfig)

PUT    /hosting/web/{serviceName}
       -> Modifier la configuration
       Body: {
         phpVersion: '8.3' | '8.2' | '8.1' | '8.0' | '7.4',
         environment: 'production' | 'development',
         engineName: 'php-fpm' | 'phpcgi',
         firewall: boolean
       }

GET    /hosting/web/availablePhpVersion
       -> Versions PHP disponibles

# MODELE DONNEES

interface OvhConfig {
  phpVersion: string;
  environment: 'production' | 'development';
  engineName: 'php-fpm' | 'phpcgi';
  firewall: boolean;
  container: string;
}

# REGLES METIER

- Fichier .ovhconfig cree a la racine FTP
- Modifications effectives immediatement
- php-fpm = meilleure performance
- phpcgi = compatibilite anciens scripts
- Firewall = protection ModSecurity
- Versions PHP < 8.0 deprecies (EOL)
- Development active display_errors
