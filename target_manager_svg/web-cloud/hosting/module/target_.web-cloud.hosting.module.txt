# STRUCTURE PAGE
1. LEFT PANEL - Hébergement + NAV3 (Modules active)
2. RIGHT PANEL:
   - Titre "Modules en 1 clic"
   - NAV4 TABS: Modules installés | Catalogue
   - CARDS modules (grille 3 colonnes):
     - Logo + Nom + Version
     - Badge statut
     - Chemin + Domaine
     - Boutons Gérer, Désinstaller
   - Card "+" pour installer

# API ENDPOINTS
GET /hosting/web/{serviceName}/module - Liste modules installés
POST /hosting/web/{serviceName}/module - Installer module
DELETE /hosting/web/{serviceName}/module/{id} - Désinstaller
