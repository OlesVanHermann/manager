STRUCTURE MODAL
1. Header orange (warning)
2. Info boost actif avec duree restante
3. Warning consequences
4. Info remboursement
5. Checkbox confirmation
6. Footer avec boutons

INFO BOOST ACTIF
| Element     | Description                    |
|-------------|--------------------------------|
| Type        | Performance Boost              |
| Workers     | Nombre actuel (ex: 3)          |
| Dates       | Activation et expiration       |
| Restant     | Badge jours restants           |

CONSEQUENCES
- Retrait ressources immediat
- Retour a 1 worker
- Impact performances possible
- Pic de charge non gere

REMBOURSEMENT
- Pas de prorata
- Jours non utilises perdus
- Reactivation possible

CONFIRMATION REQUISE
- Checkbox obligatoire
- Desactive bouton si non coche

BOUTONS FOOTER
| Bouton     | Action                    |
|------------|---------------------------|
| Annuler    | Fermer sans desactiver    |
| Desactiver | DELETE boost              |

INTERACTIONS
- Checkbox → Active bouton Desactiver
- Clic Desactiver → DELETE boost

API ENDPOINTS
GET /hosting/web/{sn}/boost                       → Info boost actif
DELETE /hosting/web/{sn}/boost                    → Desactiver boost
