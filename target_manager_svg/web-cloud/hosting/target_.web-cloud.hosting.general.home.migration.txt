# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - HEADER: Titre "Migrer votre hébergement" + service
   - BANDEAU INFO: Offre actuelle avec caractéristiques
   - GRILLE 4 CARTES: Starter, Perso, Pro, Performance 1
   - BANDEAU WARNING: Avertissement migration
   - FOOTER: Annuler / Suivant

# BANDEAU OFFRE ACTUELLE
- Background bleu info #DBEAFE
- Affiche nom offre, espace disque, BDD, emails
- Permet à l'utilisateur de voir sa situation actuelle

| Offre         | Espace  | Prix       | Caractéristiques                    | État           |
|---------------|---------|------------|-------------------------------------|----------------|
| Starter       | 100 Mo  | 1,99 €/mois| 1 site, 1 BDD, 10 emails            | Non disponible |
| Perso         | 100 Go  | 3,99 €/mois| Illimités, 10 BDD, 100 emails       | ACTUEL         |
| Pro           | 250 Go  | 7,99 €/mois| Illimités, 25 BDD, 500 emails, WCD  | Sélectionner   |
| Performance 1 | 500 Go  | 15,99 €/mois| Illimités, 50 BDD, 1000 emails     | Sélectionner   |

# STYLES CARTES

|----------------|--------------------|------------------|

| Actuelle       | Bleu 3px #0050D7   | Grisé disabled   |

# BADGE ACTUEL
- Positionné en haut de la carte offre actuelle
- Background primaire #0050D7
- Texte blanc "ACTUEL"

# BANDEAU WARNING
- Background jaune #FEF3C7
- Titre "Attention" en gras
- Message d'avertissement sur les risques de downgrade

# INTERACTIONS
- Clic "Sélectionner" → Sélection offre + étape suivante
- Clic "Annuler" → Retour page précédente
- Clic "Suivant" → Étape confirmation/paiement

# API ENDPOINTS
GET /hosting/web/{sn}              - Offre actuelle
GET /hosting/web/offer             - Liste offres disponibles
POST /hosting/web/{sn}/changeOffer - Initier migration
