STRUCTURE MODAL
1. Header avec titre
2. Politique actuelle (lecture seule)
3. Types sauvegarde (toggles)
4. Frequence (radio)
5. Retention (cartes selection)
6. Notifications (checkboxes)
7. Banniere info horaire
8. Footer avec boutons

TYPES SAUVEGARDE
| Type    | Description                    |
|---------|--------------------------------|
| Fichiers| Contenu FTP complet            |
| BDD     | Toutes bases de donnees        |

FREQUENCES
| Frequence     | Description              |
|---------------|--------------------------|
| Quotidienne   | Chaque nuit (recommande) |
| Hebdomadaire  | 1x par semaine           |
| Mensuelle     | 1x par mois              |

RETENTION
| Duree   | Prix        | Description      |
|---------|-------------|------------------|
| 7 jours | Inclus      | Standard         |
| 14 jours| +5€/mois    | Etendue          |
| 30 jours| +10€/mois   | Longue           |
| 90 jours| +20€/mois   | Maximum          |

NOTIFICATIONS
| Type            | Description              |
|-----------------|--------------------------|
| Echec           | Alerte si backup echoue  |
| Rapport hebdo   | Resume par email         |

BOUTONS FOOTER
| Bouton      | Action                     |
|-------------|----------------------------|
| Annuler     | Fermer sans sauvegarder    |
| Enregistrer | PUT backup/policy          |

INTERACTIONS
- Toggle type → Activer/desactiver
- Radio frequence → Selectionner
- Carte retention → Selectionner
- Checkbox → Activer notification

API ENDPOINTS
GET /hosting/web/{sn}/backup/policy               → Politique actuelle
PUT /hosting/web/{sn}/backup/policy               → Modifier politique
GET /hosting/web/{sn}/backup/latest               → Derniere sauvegarde
