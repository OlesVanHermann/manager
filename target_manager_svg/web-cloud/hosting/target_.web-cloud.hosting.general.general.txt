1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Hebergement actif)
3. LEFT PANEL (280px)
   - NAV3 Groupement: [General] [Sites] [Offre] [Expert]
   - Recherche
   - Liste services (3 exemples)
4. RIGHT PANEL (1096px)
   - Header: Titre service
   - NAV4: [General] Statistiques, Logs, Modules, SEO
   - Contenu:
     - Bloc Informations generales (etat, IP, disque, datacenter)
     - Bloc Configuration (PHP, SSL, email, CDN)
     - Bloc Abonnement (offre, renouvellement, boost)
     - Bloc Liens utiles
     - Bloc Apercu du site

INTERACTIONS
- Clic NAV3 -> Filtre les tabs NAV4
- Clic NAV4 -> Navigation vers tab
- Clic service sidebar -> Selection service
- Clic "Modifier" -> Modal changement PHP
- Clic "Gerer" -> Navigation vers tab correspondant
- Clic "Changer" -> Modal upgrade offre
- Clic "Activer" boost -> Modal boost

API ENDPOINTS
GET /hosting/web                           # Liste hebergements
GET /hosting/web/{sn}                      # Details hebergement
GET /hosting/web/{sn}/serviceInfos         # Infos abonnement
GET /hosting/web/{sn}/ssl                  # Certificat SSL
GET /hosting/web/{sn}/cdn                  # Configuration CDN
PUT /hosting/web/{sn}                      # Modifier config
