STRUCTURE MODAL (3 etapes)
ETAPE 1 - DOMAINE (affichee)
1. Header avec titre
2. Indicateur etapes (1/3)
3. Dropdown selection domaine OVH
4. Champ sous-domaine optionnel
5. Affichage domaine final
6. Champ repertoire racine
7. Options (SSL, firewall, CDN, DNS auto)
8. Info DNS automatique
9. Selection version PHP
10. Footer navigation

ETAPE 2 - CONFIGURATION
- Resume des choix
- Options avancees

ETAPE 3 - CONFIRMATION
- Recapitulatif complet
- Bouton validation

OPTIONS DISPONIBLES
| Option                | Description                          | Defaut  |
|-----------------------|--------------------------------------|---------|
| SSL Let's Encrypt     | Certificat gratuit automatique       | Active  |
| Firewall applicatif   | Protection WAF                       | Active  |
| CDN                   | Acceleration contenu                 | Desactive|
| DNS automatique       | Modification zone DNS                | Active  |

DNS AUTOMATIQUE
Cree automatiquement:
- Enregistrement A vers IP hebergement
- Enregistrement AAAA (IPv6) si disponible

INTERACTIONS
- Dropdown domaine → Selectionner
- Input sous-domaine → Saisir
- Checkboxes → Activer/desactiver options
- Dropdown PHP → Selectionner version
- Clic Suivant → Etape suivante

API ENDPOINTS
GET /domain                                            → Liste domaines
GET /hosting/web/{sn}/availableConfigurations          → Configs disponibles
POST /hosting/web/{sn}/attachedDomain                  → Ajouter multisite
