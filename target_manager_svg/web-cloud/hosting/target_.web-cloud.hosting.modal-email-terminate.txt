STRUCTURE MODAL
1. Header rouge (danger)
2. Info service email
3. Warning consequences
4. Liste comptes concernes
5. Checkboxes confirmation (2)
6. Footer avec boutons

CONSEQUENCES RESILIATION
- Suppression tous comptes
- Perte emails definitive
- Suppression alias/redirections
- Arret reception emails

COMPTES CONCERNES
- Liste avec apercu (max 5 affiches)
- Compteur total
- Scrollable si > 5

CONFIRMATIONS REQUISES
| Checkbox          | Description                 |
|-------------------|-----------------------------|
| Sauvegarde OK     | Emails importants sauves    |
| Comprend risques  | Action irreversible         |

BOUTONS FOOTER
| Bouton   | Action                     | Etat            |
|----------|----------------------------|-----------------|
| Annuler  | Fermer sans resilier       | Toujours actif  |
| Resilier | DELETE email service       | Si 2 checkboxes |

INTERACTIONS
- 2 checkboxes → Active bouton Resilier
- Clic Resilier → DELETE email

API ENDPOINTS
GET /hosting/web/{sn}/email                       → Info service
GET /hosting/web/{sn}/email/account               → Liste comptes
DELETE /hosting/web/{sn}/email                    → Resilier service
