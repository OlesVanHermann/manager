STRUCTURE MODAL (large)
1. Header avec titre
2. Section infos exécution
3. Zone stdout (terminal)
4. Zone stderr (terminal)
5. Métriques (4 cartes)
6. Boutons actions
7. Footer

INFOS EXECUTION
| Champ       | Description                    |
|-------------|--------------------------------|
| Statut      | Succès/Échec/Timeout           |
| Tâche       | Nom de la tâche CRON           |
| Exécution   | Date et heure                  |
| Durée       | Temps total en secondes        |
| Code retour | Exit code (0 = succès)         |
| Commande    | Commande exécutée              |
| Planning    | Expression CRON                |

STATUTS EXECUTION
| Statut  | Code | Badge   | Couleur  |
|---------|------|---------|----------|
| Succès  | 0    | success | Vert     |
| Échec   | >0   | error   | Rouge    |
| Timeout | -    | warning | Orange   |

METRIQUES
| Métrique     | Description                    |
|--------------|--------------------------------|
| Mémoire      | RAM utilisée vs limite         |
| CPU          | Temps processeur               |
| E/S Disque   | Volume lu/écrit                |
| Réseau       | Nombre d'appels                |

LOGS TERMINAL
| Couleur | Signification           |
|---------|-------------------------|
| Vert    | Timestamps, succès      |
| Blanc   | Sortie normale          |
| Bleu    | Notices, infos          |
| Rouge   | Erreurs                 |

ACTIONS
| Bouton          | Action                     |
|-----------------|----------------------------|
| Copier les logs | Clipboard                  |
| Télécharger     | Download fichier           |
| Relancer        | POST run now               |

INTERACTIONS
- Clic Copier → Copie au clipboard
- Clic Télécharger → Download
- Clic Relancer → POST run

API ENDPOINTS
GET /hosting/web/{sn}/cron/{id}                  → Info tâche
GET /hosting/web/{sn}/cron/{id}/execution/{execId}→ Détail exécution
GET /hosting/web/{sn}/cron/{id}/exec/{e}/stdout  → Sortie standard
GET /hosting/web/{sn}/cron/{id}/exec/{e}/stderr  → Erreurs
POST /hosting/web/{sn}/cron/{id}/run             → Relancer
