# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Logs actif)
   - NAV4: Logs | Statistiques
   - TITRE
   - FILTRES: Type, Domaine, Date
   - LIENS EXTERNES: Urchin, Statistiques OVHcloud
   - VISUALISEUR LOGS (fond sombre)
   - LÉGENDE COULEURS

# NAV4 SOUS-TABS
| Tab          | Description                    |
|--------------|--------------------------------|
| Logs         | Visualisation des logs (actif) |
| Statistiques | Statistiques de trafic         |

# FILTRES
| Filtre      | Type       | Valeurs exemple                  |
|-------------|------------|----------------------------------|
| Type de log | Select     | Logs web, Logs erreur, etc.      |
| Domaine     | Select     | Liste des multisites             |
| Date        | Datepicker | JJ/MM/AAAA                       |

# CODE COULEUR LOGS
| Couleur | Code    | Signification                    |
|---------|---------|----------------------------------|
| Gris    | #D1D5DB | Requête OK (2xx)                 |
| Vert    | #10B981 | Info/Bot (Googlebot, etc.)       |
| Orange  | #F59E0B | Accès refusé (403)               |
| Rouge   | #EF4444 | Erreur (404, 5xx)                |

# FORMAT LOGS
```
DATE TIME - IP/BOT - METHOD URL PROTOCOL CODE SIZE
30/12/2025 10:45:23 - 203.0.113.45 - GET /index.php HTTP/1.1 200 2048
```

# VISUALISEUR
- Fond sombre (#1F2937)
- Police monospace (Monaco, Consolas)
- Scrollbar verticale
- Coloration syntaxique selon code HTTP

# INTERACTIONS
- Sélection filtre → Actualise les logs affichés
- Clic "Télécharger" → Export fichier log
- Clic "Accéder à Urchin" → Lien externe
- Clic "Accéder aux statistiques" → Lien externe

# API ENDPOINTS
GET /hosting/web/{sn}/logs                    - Liste logs disponibles
GET /hosting/web/{sn}/logs?date=X&type=Y      - Télécharger logs
GET /hosting/web/{sn}/statistics              - Statistiques trafic
