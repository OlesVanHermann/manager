STRUCTURE MODAL
1. Header avec titre
2. Banniere info explicative
3. Runtime par defaut actuel
4. Liste runtimes selectionnables
5. Footer avec boutons

RUNTIME PAR DEFAUT
- Utilise pour nouveaux domaines
- Applique si pas de config specifique
- Un seul runtime par defaut possible

LISTE RUNTIMES
| Etat           | Style                      |
|----------------|----------------------------|
| Actuel defaut  | Grise, non selectionnable  |
| Selectionne    | Bordure bleue, radio plein |
| Disponible     | Bordure grise, radio vide  |

TYPES RUNTIMES
| Type    | Description               |
|---------|---------------------------|
| PHP     | 7.4, 8.0, 8.1, 8.2, 8.3   |
| Node.js | 16, 18, 20                |
| Python  | 3.9, 3.10, 3.11           |

BOUTONS FOOTER
| Bouton    | Action                     |
|-----------|----------------------------|
| Annuler   | Fermer sans modifier       |
| Appliquer | PUT runtime/default        |

INTERACTIONS
- Clic radio → Selectionner runtime
- Clic Appliquer → PUT default

API ENDPOINTS
GET /hosting/web/{sn}/runtime                     → Liste runtimes
GET /hosting/web/{sn}/runtime/default             → Runtime defaut
PUT /hosting/web/{sn}/runtime/default             → Changer defaut
