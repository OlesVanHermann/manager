# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif (indicateur bleu x=235)
3. LEFT PANEL - Liste hébergements (service sélectionné surligné)
4. RIGHT PANEL
   - Header: Nom service + domaines attachés
   - NAV3: 16 tabs (Home actif)
   - NAV4: 7 sous-tabs (Général, Statistiques, Indy, Offre, OvhConfig, Migration, Résiliation)
   - BLOC 1: Hébergement (Stockage, FTP/SSH, BDD, Multisite, Config)
   - BLOC 2: Configuration (SSL, CDN, Boost, Firewall, Emails auto)
   - BLOC 3: Infrastructure (Datacenter, Cluster, IPv4, IPv6, Filer)
   - BLOC 4: Abonnement (Offre, Création, Expiration, Renouvellement, Contacts)

# INTERACTIONS
- Clic NAV3 tab → Navigation vers page correspondante
- Clic NAV4 sous-tab → Navigation vers sous-page
- Clic "Changer d'offre" → Modal upgrade
- Clic "Renouveler" → Modal renouvellement
- Clic "Commander" → Page commande Cloud DB

# API ENDPOINTS
GET /hosting/web/{serviceName}              - Détails hébergement
GET /hosting/web/{serviceName}/serviceInfos - Infos facturation
GET /hosting/web/{serviceName}/ovhConfig    - Configuration PHP/moteur
GET /hosting/web/{serviceName}/ssl          - État SSL
GET /hosting/web/{serviceName}/cdn          - État CDN
GET /hosting/web/{serviceName}/database     - Liste BDD
GET /hosting/web/{serviceName}/attachedDomain - Liste domaines
