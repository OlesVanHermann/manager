# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 17 tabs (Indy actif) - tab supplémentaire
   - TITRE
   - BANDEAU INFO explicatif
   - TABLE SITES INDÉPENDANTS
   - PAGINATION

# NOTE
- L'onglet "Indy" n'est pas dans les 16 tabs standard
- Il apparaît uniquement si des sites indépendants existent
- Position en fin de NAV3

# BANDEAU INFO
- Explication des sites indépendants
- Partage de ressources avec offre principale
- Espace FTP et configuration propres

# MENU ACTIONS
- Accéder au FTP
- Modifier la configuration
- Supprimer (rouge)

# INTERACTIONS
- Clic domaine associé → Navigation vers le domaine
- Clic ⋮ → Menu contextuel

# API ENDPOINTS
GET /hosting/web/{sn}/indy              - Liste sites indépendants
GET /hosting/web/{sn}/indy/{name}       - Détails site
PUT /hosting/web/{sn}/indy/{name}       - Modifier configuration
DELETE /hosting/web/{sn}/indy/{name}    - Supprimer site
