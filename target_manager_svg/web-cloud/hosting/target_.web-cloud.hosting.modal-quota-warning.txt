STRUCTURE MODAL
1. Header orange (warning)
2. Banniere alerte critique
3. Barre progression avec legende
4. Recommandations d'actions
5. Encadre upgrade option
6. Footer avec boutons

SEUILS ALERTES
| Seuil | Couleur | Action                           |
|-------|---------|----------------------------------|
| 70%   | Orange  | Notification optionnelle         |
| 85%   | Orange  | Modal warning                    |
| 95%   | Rouge   | Modal critique + email           |
| 100%  | Rouge   | Blocage upload + email urgent    |

REPARTITION ESPACE
| Categorie    | Couleur | Description                 |
|--------------|---------|------------------------------|
| Fichiers site| Bleu    | Contenu FTP                  |
| Logs         | Vert    | Logs d'acces et erreurs      |
| Emails       | Orange  | Boites mail hebergees        |
| Autre        | Gris    | Cache, temp, etc.            |

ACTIONS RECOMMANDEES
- Supprimer fichiers inutiles
- Vider/rotation logs anciens
- Nettoyer boite mail
- Compresser fichiers volumineux

OPTIONS UPGRADE
- Augmentation quota disque
- Passage offre superieure
- Lien vers page options

BOUTONS FOOTER
| Bouton      | Action                               |
|-------------|--------------------------------------|
| Me rappeler | Reporter notification 7 jours        |
| J'ai compris| Fermer modal (acknowledge)           |

INTERACTIONS
- Clic Voir les options → Page upgrade
- Clic Me rappeler → Snooze 7 jours
- Clic J'ai compris → Fermer modal

API ENDPOINTS
GET /hosting/web/{sn}                                  → Quota global
GET /hosting/web/{sn}/quota/breakdown                  → Repartition
POST /hosting/web/{sn}/quota/snooze                    → Reporter alerte
