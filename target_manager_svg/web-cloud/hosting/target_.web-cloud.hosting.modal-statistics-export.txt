1. Header avec titre

6. Footer avec boutons

| Option           | Description                       |
|------------------|-----------------------------------|
| Dernieres 24h    | Statistiques du jour              |
| 7 derniers jours | Statistiques de la semaine        |
| 30 derniers jours| Statistiques du mois              |
| Personnalisee    | Date debut/fin custom             |

FORMATS
| Format | Extension | Description                    |
|--------|-----------|--------------------------------|
| CSV    | .csv      | Fichier texte separe virgules  |
| Excel  | .xlsx     | Classeur Microsoft Excel       |
| PDF    | .pdf      | Document portable              |

- Codes de reponse HTTP
- Erreurs 4xx/5xx

INTERACTIONS
- Clic carte periode → Selection de la periode
- Clic carte format → Selection du format
- Clic checkbox → Inclure/exclure donnees
- Clic Exporter → Telecharger le fichier

API ENDPOINTS
GET /hosting/web/{sn}/statistics                  → Donnees statistiques
POST /hosting/web/{sn}/statistics/export          → Generer export
