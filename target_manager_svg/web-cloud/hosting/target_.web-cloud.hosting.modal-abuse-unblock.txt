STRUCTURE MODAL
1. Header orange avec titre
2. Banniere erreur (raison blocage)
3. Details du blocage
4. Etapes de resolution
5. Checkboxes confirmation
6. Note delai deblocage
7. Footer avec boutons

TYPES DE BLOCAGE
| Type           | Description                        |
|----------------|------------------------------------|
| Spam/Phishing  | Envoi de spam detecte              |
| Malware        | Fichiers malveillants              |
| Abuse          | Violation CGU                      |
| DDoS           | Attaque depuis hebergement         |

ETAPES DEBLOCAGE
1. Identifier et supprimer fichiers malveillants
2. Changer tous les mots de passe
3. Mettre a jour CMS et plugins
4. Demander deblocage

VALIDATION
- Les 2 checkboxes doivent etre cochees pour activer le bouton
- Bouton desactive tant que non confirme

INTERACTIONS
- Clic checkbox → Activer/desactiver
- Clic lien ticket → Ouvrir support
- Clic Demander deblocage → Soumettre demande

API ENDPOINTS
GET /hosting/web/{sn}/abuse                       → Details blocage
POST /hosting/web/{sn}/abuse/unblock              → Demander deblocage
