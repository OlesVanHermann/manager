STRUCTURE MODAL
1. Header avec titre
2. Warning banniere
3. Details module a reinstaller
4. Selection version
5. Options (backup, reset plugins)
6. Footer avec boutons

OPTIONS
| Option                           | Defaut | Description                    |
|----------------------------------|--------|--------------------------------|
| Sauvegarder fichiers avant       | ON     | Cree backup avant reinstall    |
| Reinitialiser plugins/extensions | OFF    | Supprime plugins installes     |

INTERACTIONS
- Clic dropdown version → Choisir version
- Toggle options → Activer/desactiver
- Clic Reinstaller → Lancer reinstallation

API ENDPOINTS
GET /hosting/web/{sn}/module/{id}                 → Details module
GET /hosting/web/moduleList                       → Versions disponibles
POST /hosting/web/{sn}/module/{id}/reinstall      → Reinstaller module
