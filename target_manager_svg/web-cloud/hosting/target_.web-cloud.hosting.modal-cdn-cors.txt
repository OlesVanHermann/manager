STRUCTURE MODAL
1. Header avec titre
2. Toggle activation CORS
3. Selection origines (radio + textarea)
4. Checkboxes methodes HTTP
5. Champ headers autorises
6. Champ headers exposes
7. Champ max-age preflight
8. Warning securite
9. Footer avec boutons

ORIGINES
| Option         | Description                    |
|----------------|--------------------------------|
| Toutes (*)     | Wildcard - moins securise      |
| Specifiques    | Liste whitelist                |

METHODES HTTP
| Methode | Description                    |
|---------|--------------------------------|
| GET     | Lecture                        |
| POST    | Creation                       |
| PUT     | Mise a jour complete           |
| DELETE  | Suppression                    |
| OPTIONS | Preflight CORS                 |
| PATCH   | Mise a jour partielle          |

HEADERS COURANTS
| Header            | Usage                      |
|-------------------|----------------------------|
| Content-Type      | Type de contenu            |
| Authorization     | Token auth                 |
| X-Requested-With  | Ajax requests              |

PREFLIGHT MAX-AGE
- Duree cache de la reponse OPTIONS
- Defaut: 86400 (1 jour)
- Max: 604800 (7 jours)

BOUTONS FOOTER
| Bouton      | Action                     |
|-------------|----------------------------|
| Annuler     | Fermer sans sauvegarder    |
| Enregistrer | PUT cors config            |

INTERACTIONS
- Toggle → Activer/desactiver CORS
- Radio origines → Afficher textarea
- Clic Enregistrer → PUT cors

API ENDPOINTS
GET /hosting/web/{sn}/cdn/cors                    → Config actuelle
PUT /hosting/web/{sn}/cdn/cors                    → Modifier config
