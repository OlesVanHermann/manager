# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Modules actif)
   - TOOLBAR: + Ajouter module | Recherche
   - TABLE MODULES INSTALLÉS: Module, Version, Chemin, Login, BDD, Date, État, Actions
   - SECTION MODULES DISPONIBLES: Cartes WordPress, PrestaShop, Joomla!, Drupal, SPIP

# COLONNES TABLE
| Colonne         | Largeur | Contenu                           |
|-----------------|---------|-----------------------------------|
| Module          | 180px   | Icône + Nom module                |
| Version         | 100px   | Numéro version                    |
| Chemin          | 150px   | Path d'installation               |
| Login admin     | 130px   | Nom utilisateur admin             |
| Base de données | 170px   | Nom BDD associée                  |
| Installation    | 150px   | Date d'installation               |
| État            | 100px   | Badge Actif/Inactif               |
| Actions         | 56px    | Menu ⋮                            |

# MODULES DISPONIBLES
- WordPress (bleu #21759B)
- PrestaShop (rose #DF0067)
- Joomla! (bleu clair #5091CD)
- Drupal (bleu #0678BE)
- SPIP (rouge #9F1E1E)

# INTERACTIONS
- Clic "Ajouter module" → Modal sélection module
- Clic card module disponible → Modal installation
- Clic chemin → Accès site
- Clic ⋮ → Menu (Accès site, Admin, Changer MDP, Supprimer)

# API ENDPOINTS
GET /hosting/web/{sn}/module              - Liste modules installés
GET /hosting/web/{sn}/module/{id}         - Détails module
POST /hosting/web/{sn}/module             - Installer module
POST /hosting/web/{sn}/module/{id}/changePassword - Changer MDP admin
DELETE /hosting/web/{sn}/module/{id}      - Supprimer module
