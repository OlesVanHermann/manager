STRUCTURE MODAL
1. Header avec titre
2. Warning banniere emails bloques
3. Statistiques 7 derniers jours
4. Table liste bounces
5. Explication types bounces
6. Pagination
7. Footer

STATISTIQUES AFFICHEES
| Metrique    | Description                        |
|-------------|------------------------------------|
| Envoyes     | Total emails envoyes               |
| Delivres    | Emails correctement delivres       |
| Bounces     | Emails en erreur                   |
| Taux erreur | Pourcentage bounces/envoyes        |

TYPES DE BOUNCES
| Type | Description                              | Action         |
|------|------------------------------------------|----------------|
| Hard | Erreur permanente (adresse inexistante)  | Supprimer      |
| Soft | Erreur temporaire (boite pleine, etc.)   | Retenter auto  |

RAISONS COURANTES
- Adresse inexistante
- Boite pleine
- Domaine bloque
- Timeout serveur
- Rejete par politique
- Contenu spam

COLONNES TABLE
| Colonne      | Description              |
|--------------|--------------------------|
| Date         | Date/heure du bounce     |
| Destinataire | Email destinataire       |
| Type         | Hard ou Soft bounce      |
| Raison       | Explication de l'erreur  |

INTERACTIONS
- Pagination → Naviguer dans liste
- Clic Fermer → Fermer modal

API ENDPOINTS
GET /hosting/web/{sn}/email/bounces                    → Liste bounces
GET /hosting/web/{sn}/email/bounces/statistics         → Stats
