STRUCTURE MODAL
1. Header rouge avec titre
2. Warning banniere (irreversible)
3. Info hebergement
4. Liste elements supprimes
5. Selection date resiliation
6. Dropdown raison (optionnel)
7. Checkboxes confirmation
8. Footer avec boutons (Resilier desactive sans confirmations)

DATES RESILIATION
| Option              | Description                        |
|---------------------|------------------------------------|
| Fin de periode      | Resiliation a l'echeance           |
| Immediate           | Suppression sous 48h               |

VALIDATION
- Les 2 checkboxes doivent etre cochees
- Bouton Resilier grise tant que non confirme

INTERACTIONS
- Clic card date → Selection type resiliation
- Dropdown raison → Choisir raison
- Checkboxes → Confirmer
- Clic Resilier → Lancer resiliation

API ENDPOINTS
GET /hosting/web/{sn}                             → Info hebergement
POST /hosting/web/{sn}/terminate                  → Resilier
