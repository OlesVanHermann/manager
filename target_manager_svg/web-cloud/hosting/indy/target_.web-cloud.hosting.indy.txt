# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, Multisite actif
   - NAV4: Domaines attachés | Sites indépendants (actif)
   - TITRE + DESCRIPTION
   - BANNIERE INFO (bleu)
   - TOOLBAR: + Ajouter un site indépendant
   - TABLE: Domaine, Répertoire, Utilisateur FTP, PHP, Espace, Statut, Actions
   - PAGINATION

# SITES INDÉPENDANTS (INDY)
- Espace FTP isolé par site
- Configuration PHP séparée
- Quota d'espace dédié
- Utilisateur FTP propre

# STATUTS
- Actif (vert) - Site fonctionnel
- En création (jaune) - Provisionnement en cours
- Erreur (rouge) - Problème de configuration

# ACTIONS
- + Ajouter un site indépendant → modal-indy-add
- ⋮ (row action) → Modifier / Gérer FTP / Supprimer
