- Cles SSH
- SFTP securise

INTERACTIONS
- Dropdown utilisateur → Selectionner
- Clic copie → Copier dans presse-papier
- Clic Activer → Activer SSH

API ENDPOINTS
GET /hosting/web/{sn}/user                        → Liste utilisateurs
PUT /hosting/web/{sn}/user/{login}                → Activer SSH (isSsh=true)
