STRUCTURE MODAL
1. Header avec titre
2. Domaine source (lecture seule)
3. Type redirection (radio)
4. URL destination
5. Options (checkboxes)
6. Apercu redirection
7. Banniere info
8. Footer avec boutons

TYPES REDIRECTION
| Code | Type        | Usage                        |
|------|-------------|------------------------------|
| 301  | Permanente  | SEO, transfert pagerank      |
| 302  | Temporaire  | Redirections provisoires     |

OPTIONS
| Option              | Description                   |
|---------------------|-------------------------------|
| www/non-www         | Redirige les deux variantes   |
| Conserver chemin    | Ajoute path a destination     |
| Conserver params    | Conserve query string         |

APERCU
- Montre URL source complete
- Fleche de redirection
- URL destination resultante
- Code HTTP utilise

BOUTONS FOOTER
| Bouton      | Action                     |
|-------------|----------------------------|
| Annuler     | Fermer sans sauvegarder    |
| Enregistrer | PUT redirection            |

INTERACTIONS
- Radio → Selectionner type
- Checkboxes → Modifier options
- Input → URL destination
- Apercu → Mise a jour temps reel

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}     → Info domaine
PUT /hosting/web/{sn}/attachedDomain/{domain}     → Configurer redirection
DELETE /hosting/web/{sn}/attachedDomain/{domain}/redirect → Supprimer
