DONNEES DYNAMIQUES
| Exemple SVG                  | Source API                                    |
|------------------------------|-----------------------------------------------|
| Liste bases                  | GET /hosting/web/{sn}/database                |
| Nom                          | database → name                               |
| Type (MySQL/MariaDB)         | database → type                               |
| Version                      | database → version                            |
| Serveur                      | database → server                             |
| Taille                       | database → quotaUsed                          |
| Statut                       | database → status                             |

STRUCTURE MODAL
1. Header avec titre
2. Barre recherche et filtres
3. Bannière info
4. Liste bases avec radio
5. Résumé sélection
6. Footer avec boutons

COLONNES TABLEAU
| Colonne  | Largeur | Description               |
|----------|---------|---------------------------|
| Radio    | 30px    | Sélection                 |
| Nom      | 170px   | Nom + alias               |
| Type     | 100px   | MySQL/MariaDB + version   |
| Serveur  | 180px   | Hostname                  |
| Taille   | 100px   | Espace utilisé            |
| Statut   | 70px    | Badge Actif/Inactif       |

FILTRES
| Filtre     | Options                        |
|------------|--------------------------------|
| Recherche  | Nom, alias                     |
| Type       | Toutes, MySQL, MariaDB         |

STATUTS BASE
| Statut  | Badge   | Couleur  |
|---------|---------|----------|
| Actif   | success | Vert     |
| Inactif | muted   | Gris     |

BOUTONS
| Bouton       | Action                     |
|--------------|----------------------------|
| + Nouvelle   | Ouvrir création BDD        |
| Annuler      | Fermer sans sélection      |
| Sélectionner | Retourner la sélection     |

INTERACTIONS
- Recherche → Filtrer liste
- Dropdown → Filtrer par type
- Radio → Sélectionner base
- Clic ligne → Sélectionner
- Clic Sélectionner → Confirmer

API ENDPOINTS
GET /hosting/web/{sn}/database                   → Liste bases
GET /hosting/web/{sn}/database/{name}            → Détail base
