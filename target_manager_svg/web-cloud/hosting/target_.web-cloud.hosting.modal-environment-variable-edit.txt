STRUCTURE MODAL
1. Header avec titre
2. Nom variable (lecture seule)
3. Valeur actuelle (masquée)
4. Nouvelle valeur
5. Options
6. Warning redémarrage
7. Zone suppression
8. Footer avec boutons

CHAMPS FORMULAIRE
| Champ          | Type     | Modifiable |
|----------------|----------|------------|
| Nom            | Input    | Non        |
| Valeur actuelle| Display  | Non        |
| Nouvelle valeur| Textarea | Oui        |
| Masquer        | Checkbox | Oui        |

AFFICHAGE VALEUR
- Masquée par défaut si option active
- Bouton "Afficher" pour révéler
- Sécurité pour les secrets

IMPACT MODIFICATION
- Redémarrage runtime requis
- Brève interruption possible

ZONE DANGEREUSE
| Action    | Effet                      |
|-----------|----------------------------|
| Supprimer | DELETE envVar              |

BOUTONS FOOTER
| Bouton     | Action                     |
|------------|----------------------------|
| Annuler    | Fermer sans modifier       |
| Enregistrer| PUT envVar                 |

INTERACTIONS
- Clic Afficher → Révéler valeur
- Textarea → Saisie nouvelle valeur
- Toggle → Modifier masquage
- Clic Supprimer → Confirmation
- Clic Enregistrer → PUT envVar

API ENDPOINTS
GET /hosting/web/{sn}/envVar/{key}               → Détail variable
PUT /hosting/web/{sn}/envVar/{key}               → Modifier
DELETE /hosting/web/{sn}/envVar/{key}            → Supprimer
