STRUCTURE MODAL
1. Header avec titre
2. Warning operation irreversible
3. Dropdown selection snapshot
4. Liste snapshots disponibles (selectable)
5. Options restauration (radio buttons)
6. Info disponibilite
7. Checkbox confirmation obligatoire
8. Footer avec boutons

LISTE SNAPSHOTS
| Colonne    | Description                    |
|------------|--------------------------------|
| Date/heure | Date et heure du snapshot      |
| Type       | Automatique quotidien / Manuel |
| Taille     | Espace occupe                  |
| Age        | Temps ecoule                   |

OPTIONS RESTAURATION
| Option               | Description                         |
|----------------------|-------------------------------------|
| Complete             | Remplace tout le contenu FTP        |
| Partielle            | Permet de choisir un repertoire     |

TYPES SNAPSHOTS
| Type       | Frequence | Retention |
|------------|-----------|-----------|
| Automatique| Quotidien | 7 jours   |
| Manuel     | A la demande | 30 jours |

PRECAUTIONS
- Warning operation irreversible
- Site temporairement indisponible
- Notification email a la fin
- Checkbox confirmation obligatoire

INTERACTIONS
- Clic snapshot → Selectionner
- Radio option → Choisir type restauration
- Checkbox → Confirmer comprehension
- Clic Restaurer → Lancer restauration

API ENDPOINTS
GET /hosting/web/{sn}/dump                             → Liste snapshots
POST /hosting/web/{sn}/dump/{id}/restore               → Restaurer
