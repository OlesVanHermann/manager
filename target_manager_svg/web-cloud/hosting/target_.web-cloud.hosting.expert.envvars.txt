# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements
4. RIGHT PANEL
   - Header: Nom service
   - NAV3: 16 tabs (Variables actif)
   - TOOLBAR: Refresh | Recherche | + Ajouter variable
   - TABLE: Clé, Type, Valeur, Statut, Actions
   - PAGINATION
   - BANNIÈRE INFO

# COLONNES TABLE
| Colonne  | Largeur | Contenu                          |
|----------|---------|----------------------------------|
| Clé      | 230px   | Nom de la variable               |
| Type     | 110px   | Badge string/password/boolean    |
| Valeur   | 340px   | Valeur ou •••• pour password     |
| Statut   | 120px   | Badge Créée/En création          |
| Actions  | 56px    | Menu ⋮                           |

# TYPES DE VARIABLES
| Type     | Badge couleur | Affichage valeur |
|----------|---------------|------------------|
| string   | Gris #F3F4F6  | Valeur normale   |
| password | Jaune #FEF3C7 | ••••••••         |
| boolean  | Bleu #DBEAFE  | true/false       |

# ÉTATS POSSIBLES
| État        | Couleur badge |
|-------------|---------------|
| Créée       | Vert #DEF7EC  |
| En création | Jaune #FEF3C7 |
| Suppression | Rouge #FEE2E2 |

# INTERACTIONS
- Clic ↻ → Rafraîchir liste
- Clic "+ Ajouter variable" → Modal création
- Clic ⋮ → Menu (Modifier, Supprimer)

# API ENDPOINTS
GET /hosting/web/{sn}/envVar              - Liste variables
GET /hosting/web/{sn}/envVar/{key}        - Détails variable
POST /hosting/web/{sn}/envVar             - Créer variable
PUT /hosting/web/{sn}/envVar/{key}        - Modifier variable
DELETE /hosting/web/{sn}/envVar/{key}     - Supprimer variable
