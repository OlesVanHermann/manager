# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, SSL actif
   - Description
   - CARD: État du certificat (type, fournisseur, dates)
   - BUTTONS: Régénérer / Commander / Supprimer
   - CARD: Domaines couverts (liste)
   - BANNIERE INFO: Informations Let's Encrypt

# TYPES SSL
- Let's Encrypt (gratuit, auto-renouvelé)
- DV (Domain Validation)
- EV (Extended Validation)

# STATUTS
- Actif (vert) - Certificat valide
- Expiré (rouge) - Certificat à renouveler

# ACTIONS
- Régénérer SSL → POST /hosting/web/{sn}/ssl/regenerate
- Commander SSL → Tunnel commande certificat payant
- Supprimer SSL → DELETE /hosting/web/{sn}/ssl

# API ENDPOINTS
GET /hosting/web/{serviceName}/ssl
GET /hosting/web/{serviceName}/ssl/domains
POST /hosting/web/{serviceName}/ssl/regenerate
DELETE /hosting/web/{serviceName}/ssl
