# STRUCTURE PAGE
1. LEFT PANEL - Hébergement + NAV3 (SSL active)
2. RIGHT PANEL:
   - Titre "Certificat SSL"
   - STATUS BANNER: Vert si actif, détails certificat
   - TABLE Domaines: Domaine, Statut SSL, Toggle HTTPS
   - BLOC Options: Toggle forcer HTTPS
   - ACTIONS: Importer certificat, Supprimer SSL

# TYPES SSL
| Type          | Description            |
|---------------|------------------------|
| Let's Encrypt | Gratuit, auto-renouvelé|
| Sectigo DV    | Payant, validation domaine |
| Sectigo EV    | Payant, validation étendue |
| Personnalisé  | Importé par l'utilisateur |

# API ENDPOINTS

POST /hosting/web/{serviceName}/ssl - Activer SSL
DELETE /hosting/web/{serviceName}/ssl - Supprimer SSL
POST /hosting/web/{serviceName}/ssl/regenerate - Régénérer
