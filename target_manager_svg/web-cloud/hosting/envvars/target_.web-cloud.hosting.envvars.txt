# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: {serviceName}
   - NAV3: 16 tabs, Variables actif
   - Description
   - TOOLBAR: + Ajouter variable
   - TABLE: Clé, Type, Valeur, État, Actions
   - PAGINATION

# TYPES
- string - Variable texte
- password - Variable masquée (••••)
- boolean - true/false

# STATUTS
- Créée (vert) - Variable active
- En création (jaune) - Variable en cours de déploiement

# ACTIONS
- + Ajouter variable → modal-envvar-add
- ⋮ (row action) → Modifier / Supprimer

# API ENDPOINTS
GET /hosting/web/{serviceName}/envVar
GET /hosting/web/{serviceName}/envVar/{key}
POST /hosting/web/{serviceName}/envVar
PUT /hosting/web/{serviceName}/envVar/{key}
DELETE /hosting/web/{serviceName}/envVar/{key}
