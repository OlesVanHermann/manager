# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hébergement actif
3. LEFT PANEL - Liste hébergements (280px)
4. RIGHT PANEL (1096px)
   - Header: Changer d'offre + {serviceName}
   - WIZARD STEPS: 1.Offre | 2.Configuration | 3.Confirmation
   - BANNIERE INFO: Offre actuelle
   - OFFER CARDS: Starter, Perso, Pro, Performance 1, Performance 2
   - BANNIERE WARNING: Migration vers offre inférieure
   - BOUTONS: Annuler / Suivant

# WIZARD STEPS
1. Offre - Sélection de la nouvelle offre
2. Configuration - Options supplémentaires
3. Confirmation - Récapitulatif et paiement

# OFFER CARDS
- Starter: 1,99€/mois, 100Mo, 1 site, 1 BDD, 10 emails
- Perso: 3,99€/mois, 100Go, illimité, 10 BDD, 100 emails
- Pro: 7,99€/mois, 250Go, illimité, 25 BDD, 500 emails, Web Cloud DB
- Performance 1: 15,99€/mois, 500Go, illimité, 50 BDD, 1000 emails, ressources dédiées
- Performance 2+: Offres supérieures avec plus de ressources

# ÉTATS CARDS
- ACTUEL (badge bleu) - Offre en cours
- Non disponible (grisé) - Downgrade non autorisé
- Sélectionner (bouton bleu) - Offre disponible

# ACTIONS
- Sélectionner → Étape 2 Configuration
- Annuler → Retour dashboard
- Suivant → Étape suivante du wizard

# API ENDPOINTS
GET /hosting/web/{serviceName}
GET /order/hosting/web
POST /order/hosting/web/{serviceName}/changeOffer
GET /hosting/web/{serviceName}/changeOfferAvailability
