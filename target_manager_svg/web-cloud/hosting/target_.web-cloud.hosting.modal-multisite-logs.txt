STRUCTURE MODAL (large)
1. Header avec titre
2. Barre filtres (domaine, date, type, URL)
3. Visionneur logs style terminal
4. Legende codes HTTP
5. Footer avec boutons

FILTRES DISPONIBLES
| Filtre     | Options                               |
|------------|---------------------------------------|
| Date       | Aujourd'hui, 7 jours, 30 jours, Custom|
| Type       | Tous, Access, Error, WAF              |
| URL        | Recherche textuelle                   |

CODES COULEUR HTTP
| Code | Couleur | Signification          |
|------|---------|------------------------|
| 2xx  | Vert    | Succes                 |
| 3xx  | Orange  | Redirection            |
| 4xx  | Rouge   | Erreur client (404...) |
| 5xx  | Rouge   | Erreur serveur (500...)|

FORMAT LOG
| Champ      | Description                    |
|------------|--------------------------------|
| Timestamp  | [YYYY-MM-DD HH:MM:SS]          |
| Code HTTP  | Code reponse colore            |
| Methode    | GET/POST/PUT/DELETE            |
| URL        | Chemin demande                 |
| IP         | Adresse client                 |
| User-Agent | Navigateur client              |

BOUTONS FOOTER
| Bouton          | Action                     |
|-----------------|----------------------------|
| Telecharger logs| Export fichier .log        |
| Fermer          | Fermer modal               |

INTERACTIONS
- Filtres → Actualiser logs
- Scroll → Charger plus de logs
- Clic Telecharger → Download .log

API ENDPOINTS
GET /hosting/web/{sn}/attachedDomain/{domain}/logs     → Logs access
GET /hosting/web/{sn}/attachedDomain/{domain}/logs/error → Logs erreur
GET /hosting/web/{sn}/attachedDomain/{domain}/logs/export → Export
