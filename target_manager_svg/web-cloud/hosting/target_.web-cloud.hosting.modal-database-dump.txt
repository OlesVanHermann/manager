STRUCTURE MODAL
1. Header avec titre
2. Banniere info
3. Details base de donnees
4. Selection type (maintenant / automatique)
5. Options (compression, email)
6. Bloc estimation
7. Footer avec boutons

TYPES DE DUMP
| Type        | Description                       |
|-------------|-----------------------------------|
| Maintenant  | Creation immediate du dump        |
| Automatique | Planifier sauvegardes quotidiennes|

OPTIONS
| Option      | Description                       | Defaut |
|-------------|-----------------------------------|--------|
| Compression | Compresser en gzip                | ON     |
| Email       | Notification par email            | OFF    |

INTERACTIONS
- Clic carte type → Selection du type
- Clic toggle → Activer/desactiver option
- Clic Creer → Lancer la creation du dump

API ENDPOINTS
GET /hosting/web/{sn}/database/{name}              → Details BDD
POST /hosting/web/{sn}/database/{name}/dump        → Creer dump
GET /hosting/web/{sn}/database/{name}/dump         → Liste dumps
