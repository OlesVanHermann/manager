# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste OverTheBox
4. RIGHT PANEL
   - Header: Nom OTB
   - NAV3: 6 tabs (Logs actif)
   - Contenu:
     - TOOLBAR:
       - Bouton refresh
       - Recherche
       - Filtre niveau
       - Filtre période
       - Bouton Exporter
     - CONSOLE LOGS:
       - Affichage temps réel
       - Horodatage
       - Niveau coloré
       - Message
       - Scrollbar
       - Indicateur Live + Pause

# NIVEAUX DE LOG
| Niveau | Couleur | Usage |
|--------|---------|-------|
| DEBUG | Gris | Développement |
| INFO | Vert | Informations |
| WARN | Jaune | Avertissements |
| ERROR | Rouge | Erreurs |
| FATAL | Rouge vif | Erreurs critiques |

# PÉRIODES
- Dernière heure
- Dernières 24h (défaut)
- 7 derniers jours
- 30 derniers jours
- Personnalisé

# FORMAT LOG
```
[YYYY-MM-DD HH:MM:SS] [LEVEL] Message détaillé
```

# INTERACTIONS
- ↻ Refresh → Force rechargement
- Recherche → Filtre en temps réel
- Filtres → Dropdown
- Exporter → Télécharge fichier .log
- Live → WebSocket temps réel
- Pause → Arrête le flux live

# API ENDPOINTS
GET /overthebox/{id}/logs                    - Historique logs
GET /overthebox/{id}/logs/export             - Export fichier
WS  /overthebox/{id}/logs/stream             - Flux temps réel

# WEBSOCKET
Connexion temps réel pour affichage live des logs.
Format message: JSON { timestamp, level, message }
