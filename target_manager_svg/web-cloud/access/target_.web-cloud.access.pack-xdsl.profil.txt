# Cahier des Charges : web-cloud/access/pack-xdsl - Profil DSLAM

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration profil DSLAM
- **Objectif** : Ajuster synchronisation ligne xDSL
- **Contexte** : Optimisation débit/stabilité

## 2. LAYOUT & STRUCTURE
- Carte principale : Formulaire sélection profil
- Carte état : Changement en cours (optionnel)

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsText** : Description fonctionnelle
- **OdsBadge** : Profil actuel (vert)
- **OdsSelect** : Sélecteur de profil avec dropdown
- **OdsSpinner** : État changement en cours
- **OdsMessage** : Warning interruption (jaune)

## 4. DESCRIPTION PROFIL
- Ligne 1 : "Le profil DSLAM définit les paramètres de synchronisation de votre ligne."
- Ligne 2 : "Un profil optimisé pour la stabilité réduira les déconnexions mais peut limiter le débit."

## 5. PROFIL ACTUEL
- Label : "Profil actuel"
- Badge vert : "Standard (auto)"

## 6. SÉLECTEUR PROFIL
- Label : "Changer de profil"
- Options :
  - Standard (auto) ✓ (sélectionné)
  - Performance (haut débit)
  - Stabilité (moins de coupures)

## 7. ÉTAT CHANGEMENT EN COURS
- Titre : "Changement en cours"
- Spinner + texte : "Modification du profil DSLAM"
- Sous-texte : "Cette opération peut prendre quelques minutes"
- Warning : "⚠ Votre connexion sera interrompue brièvement."

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/lines/{lineNumber}/dslamPort : Profil actuel
- PUT /xdsl/{serviceName}/lines/{lineNumber}/dslamPort : Modifier profil
- GET /xdsl/{serviceName}/lines/{lineNumber}/dslamPort/availableProfiles : Profils disponibles
