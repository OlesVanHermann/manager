# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste connexions (item sélectionné)
4. RIGHT PANEL
   - Header: Nom + adresse + badge techno (FTTH)
   - NAV3: 7 tabs (Général actif)
   - Contenu:
     - CONNEXION: Statut, débit sync, dernière sync
     - INFORMATIONS: Type, offre, adresse, NRO, GTR
     - MODEM: Modèle, badge OVH/perso, firmware, uptime, actions
     - SERVICES INCLUS: Liste domaine/emails/VoIP avec liens
     - OPTIONS ACTIVES: Badges + bouton ajouter
     - ABONNEMENT: Dates, engagement, montant
     - ACTIONS: Renommer, Contacts, Déménager, Migrer, Résilier

# INTERACTIONS
- Clic tab → Navigation
- Clic "Configurer" modem → Ouvre tab Modem
- Clic "Gérer" service → Redirige vers service
- Clic action → Ouvre modal correspondante

# API ENDPOINTS
GET    /access/connections/{id}            - Détails connexion
PUT    /access/connections/{id}            - Modifier
POST   /access/connections/{id}/rename     - Renommer
POST   /access/connections/{id}/migrate    - Migrer offre
POST   /access/connections/{id}/move       - Déménager
DELETE /access/connections/{id}            - Résilier
