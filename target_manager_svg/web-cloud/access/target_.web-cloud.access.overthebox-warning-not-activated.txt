# STRUCTURE PAGE

Page affichée quand un OverTheBox est en attente d'activation.

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. CONTENT CENTRÉ:
   - Icône attente
   - Titre "En attente d'activation"
   - Description
   - Étapes d'installation
   - Boutons d'action

# QUAND AFFICHER

Cette page s'affiche quand:
- L'OTB a été commandé récemment
- Le statut est PENDING ou DELIVERING
- L'équipement n'a pas encore été lié

# ÉTAPES D'ACTIVATION

1. **Réception** - Livraison en cours
2. **Branchement** - Connexion physique
3. **Activation** - Liaison avec le compte

# ÉTATS DES ÉTAPES

- Terminée: cercle bleu plein
- En cours: cercle bleu avec animation
- À venir: cercle gris

# INTERACTIONS

- Suivre le colis → Lien suivi transporteur
- Activer → POST /overthebox/{id}/activate

# API ENDPOINTS

GET  /overthebox/{id}                  - État de l'OTB
GET  /overthebox/{id}/tracking         - Suivi livraison
POST /overthebox/{id}/activate         - Activer l'OTB

# SUIVI LIVRAISON

Le lien de suivi est fourni par:
- GET /overthebox/{id}/tracking → trackingUrl
