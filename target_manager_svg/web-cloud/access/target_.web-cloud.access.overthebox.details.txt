# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste OverTheBox
4. RIGHT PANEL
   - Header: Nom OTB
   - NAV3: 6 tabs (Connexions actif)
   - Contenu:
     - TOOLBAR: Bouton + Lier une connexion
     - CONNEXIONS (cards détaillées):
       - Header: Nom + Statut + Priorité
       - Type, Opérateur, ID Connexion
       - Débit down/up, Latence, Perte paquets
       - IP WAN, Gateway
       - Signal (si 4G)
       - Warning (si dégradé)
       - Boutons: Configurer, Délier

# TYPES CONNEXIONS
- FTTH: Fibre optique
- VDSL2/ADSL: DSL
- SDSL: DSL symétrique
- 4G LTE / 5G: Mobile
- Satellite: Liaison satellite
- Ethernet: Connexion directe

# STATUTS CONNEXION
- Actif: Fonctionnement normal (vert)
- Dégradé: Problème détecté (jaune)
- Hors ligne: Non connecté (rouge)

# MÉTRIQUES CONNEXION
- Débit: Bande passante réelle mesurée
- Latence: Temps de réponse (ping)
- Perte paquets: % paquets perdus
- Signal: Niveau signal (4G/5G uniquement)

# INTERACTIONS
- + Lier une connexion → Modal liaison
- Configurer → Modal configuration avancée
- Délier → Confirmation + DELETE

# API ENDPOINTS
GET  /overthebox/{id}/connections           - Liste connexions
GET  /overthebox/{id}/connections/{connId}  - Détail connexion
POST /overthebox/{id}/connections           - Lier connexion
PUT  /overthebox/{id}/connections/{connId}  - Configurer connexion
DELETE /overthebox/{id}/connections/{connId} - Délier connexion
