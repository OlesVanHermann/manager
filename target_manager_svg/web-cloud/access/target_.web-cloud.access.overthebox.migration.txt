# Cahier des Charges : web-cloud/access/overthebox - Migration

## 1. VUE D'ENSEMBLE
- **Écran** : Page de migration/changement d'offre OverTheBox
- **Objectif** : Permettre l'upgrade vers une offre supérieure
- **Contexte** : Accessible depuis "Changer d'offre" dans Détails

## 2. LAYOUT & STRUCTURE
- Lien retour "← Migration d'offre"
- Carte principale avec tableau comparatif
- Lien "En savoir plus" en bas

## 3. COMPOSANTS UI
- **OdsLink** : Retour + lien info
- **OdsCard** : Container principal
- **OdsTable** : Tableau comparatif des offres
- **OdsSelect** : Choix du matériel
- **OdsButton** : "Choisir cette offre" (primaire)

## 4. TABLEAU COMPARATIF
### Colonnes
| Position | Contenu |
|----------|---------|
| 1 | Labels (critères) |
| 2 | Offre actuelle (fond bleu clair) |
| 3 | Offre Plus |
| 4 | Offre Pro |

### Lignes (critères comparés)
| Critère | IT v1 | Plus | Pro |
|---------|-------|------|-----|
| Abonnement mensuel | 9,99 € HT | 14,99 € HT | 29,99 € HT |
| Matériel | / | Boîtier Plus (+99€) | Boîtier Pro (+149€) |
| Interfaces WAN | 4 ports | 6 ports | 8 ports + 4G backup |
| Débit agrégé max | 100 Mbps | 250 Mbps | 500 Mbps |

## 5. ACTIONS & INTERACTIONS
- Clic "←" → Retour au dashboard
- Select matériel → Choix option hardware
- Bouton "Choisir cette offre" → Processus de commande
- Lien "En savoir plus" → Documentation offres

## 6. APIS & ENDPOINTS
- GET /overthebox/{serviceName} : Offre actuelle
- GET /overthebox/offers : Liste offres disponibles
- POST /overthebox/{serviceName}/migrate : Initier migration

## 7. NOTES
- Offre actuelle non sélectionnable (grisée)
- Prix HT affiché
- Matériel optionnel avec surcoût
