# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste connexions
4. RIGHT PANEL
   - Header: Nom connexion
   - NAV3: 7 tabs (VoIP actif)
   - NAV4: 2 sous-tabs (EcoFax actif)
   - Contenu:
     - ECOFAX CONFIG:
       - Numéro + Badge statut
       - Email de réception (éditable)
       - Format réception (dropdown)
       - Qualité (dropdown)
       - Bouton Enregistrer
     - ENVOYER UN FAX:
       - Numéro destinataire (input)
       - Fichier (upload drag & drop)
       - Bouton Envoyer
     - HISTORIQUE DES FAX (table):
       - Date
       - Type (Reçu/Envoyé)
       - Correspondant
       - Pages
       - Statut (Succès/Échec)
       - Actions (Télécharger/Renvoyer)
       - Pagination

# FORMATS RÉCEPTION
- PDF (recommandé)
- TIFF
- PDF + TIFF

# QUALITÉS
- Standard (100 dpi)
- Haute (200 dpi)
- Ultra (300 dpi)

# STATUTS FAX
- Succès: Transmission réussie
- Échec: Erreur de transmission
- En cours: Transmission en cours
- En attente: Dans la file d'attente

# INTERACTIONS
- ✎ Modifier email → Modal édition
- Enregistrer → PUT /voip/ecofax
- Envoyer → POST /voip/ecofax/send
- Télécharger → Download PDF/TIFF
- Renvoyer → POST /voip/ecofax/resend/{id}

# API ENDPOINTS
GET  /access/connections/{id}/voip/ecofax           - Config EcoFax
PUT  /access/connections/{id}/voip/ecofax           - Modifier config
POST /access/connections/{id}/voip/ecofax/send      - Envoyer fax
GET  /access/connections/{id}/voip/ecofax/history   - Historique fax
GET  /access/connections/{id}/voip/ecofax/{id}      - Télécharger fax
POST /access/connections/{id}/voip/ecofax/resend/{id} - Renvoyer fax
