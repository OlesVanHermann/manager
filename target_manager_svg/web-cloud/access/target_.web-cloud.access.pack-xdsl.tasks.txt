# Cahier des Charges : web-cloud/access/pack-xdsl - Tasks

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des tâches d'un accès xDSL
- **Objectif** : Suivre les opérations en cours et passées
- **Contexte** : Monitoring des modifications modem/ligne

## 2. LAYOUT & STRUCTURE
- Carte blanche avec titre + bouton refresh
- Tableau des tâches avec pagination

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsButton** : Refresh (↻)
- **OdsTable** : Liste tâches
- **OdsBadge** : Statut tâche
- **OdsPagination** : Navigation pages

## 4. DONNÉES & COLONNES
| Colonne | Type | Description |
|---------|------|-------------|
| ID | text | Numéro tâche (12847) |
| Nom de la tâche | text | Nom technique opération |
| Statut | badge | État de la tâche |
| Date de mise à jour | datetime | DD/MM/YYYY HH:mm:ss |

## 5. STATUTS POSSIBLES
| Statut | Couleur fond | Couleur texte |
|--------|--------------|---------------|
| Terminée | #D1FAE5 | #065F46 |
| En cours | #DBEAFE | #1E40AF |
| À faire | #E0E7FF | #3730A3 |
| Erreur | #FEE2E2 | #991B1B |

## 6. TYPES DE TÂCHES
- changeModemConfigWLAN : Config WiFi
- resetModem : Réinitialisation modem
- changeModemConfigFirmware : Mise à jour firmware
- changeModemConfigDMZ : Config DMZ
- dslamProfileChange : Changement profil DSLAM

## 7. PAGINATION
- Format : "1 - 5 sur 12"
- Boutons : ‹ (précédent) › (suivant)

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/tasks : Liste tâches
- Paramètres : page, pageSize
