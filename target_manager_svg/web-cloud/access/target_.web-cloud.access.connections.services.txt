# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste connexions
4. RIGHT PANEL
   - Header: Nom connexion
   - NAV3: 7 tabs (Services actif)
   - Contenu:
     - SERVICES ASSOCIÉS (titre)
     - Cards services (2x2 grid):
       - Email Pro
       - Hébergement Web
       - Nom de domaine
       - IP Failover
     - AJOUTER UN SERVICE (bannière + bouton)
     - RÉSUMÉ (compteurs)

# TYPES DE SERVICES
- Email Pro / Exchange
- Hébergement Web
- Nom de domaine
- IP Failover / RIPE Block
- VoIP (voir onglet dédié)
- CDN
- SSL

# CARD SERVICE
Chaque card affiche:
- Icône + Nom service + Badge statut
- 3 lignes d'info clé
- Bouton "Gérer →"

# INTERACTIONS
- Gérer → Redirige vers la gestion du service
- Commander → Ouvre catalogue services

# API ENDPOINTS
GET /access/connections/{id}/services           - Liste services associés
GET /access/connections/{id}/services/summary   - Résumé compteurs

# LIENS CROISÉS
Chaque service renvoie vers son manager dédié:
- Email Pro → /web-cloud/emails/{domain}
- Hébergement → /web-cloud/hosting/{id}
- Domaine → /web-cloud/domain/{domain}
- IP → /bare-metal/ip/{ip}
