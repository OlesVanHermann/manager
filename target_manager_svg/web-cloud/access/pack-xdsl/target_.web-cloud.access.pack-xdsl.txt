# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section Web Cloud (Accès actif)
3. SOUS-NAV - Toggle [Connexions] | [OverTheBox]
4. LEFT PANEL - Liste des connexions (4 items)
5. RIGHT PANEL
   - Header: Nom connexion + badges + actions (Migrer, Déménager)
   - NAV3: [Général] | Ligne | Modem | Services | VoIP | Options | Tâches
   - Bloc CONNEXION: Statut, type, offre, débit, adresse
   - Bloc MODEM + SERVICES INCLUS (2 colonnes)
   - Bloc OPTIONS ACTIVES
   - Actions: Renommer, Modifier contacts, Résilier

# INTERACTIONS
- Clic service left panel → Sélection + chargement détails
- Clic tab NAV3 → Navigation vers section
- Clic "Configurer" modem → Navigation vers /modem
- Clic "Gérer services" → Navigation vers /services
- Clic "Ajouter option" → Modal ajout option
- Clic "Résilier" → Modal confirmation résiliation

# API ENDPOINTS
GET    /access/connections                    - Liste toutes connexions
GET    /access/connections/{id}               - Détails connexion
GET    /access/connections/{id}/line          - Statut ligne
GET    /access/connections/{id}/modem         - Infos modem
GET    /access/connections/{id}/services      - Services inclus
GET    /access/connections/{id}/options       - Options actives
PUT    /access/connections/{id}               - Modifier (nom, contacts)
POST   /access/connections/{id}/migrate       - Migrer offre
POST   /access/connections/{id}/move          - Déménager
DELETE /access/connections/{id}               - Résilier
