# STRUCTURE PAGE
1. NAV3: Général | [Ligne] | Modem | Services...
2. NAV4: Statut | [Diagnostic] | Statistiques | Alertes
3. Toolbar: Lancer diagnostic + Historique
4. Bloc DERNIER DIAGNOSTIC: Date + résultat global
5. Bloc RÉSULTATS DÉTAILLÉS: 6 tests (sync, PPP, DNS, ping, qualité, CRC)
6. Actions: Contacter support, Resynchroniser

# TESTS EFFECTUÉS
- Synchronisation: débit sync
- Connectivité PPP: session
- DNS: résolution
- Ping passerelle: latence
- Qualité ligne: SNR, atténuation
- Erreurs CRC: compteur

# INTERACTIONS
- Clic "Lancer un diagnostic" → Lance tests + affiche progression
- Clic "Historique" → Liste diagnostics passés
- Clic "Contacter support" → Ouvre ticket
- Clic "Resynchroniser" → POST resync

# API ENDPOINTS
GET  /access/connections/{id}/line/diagnostic         - Dernier diagnostic
POST /access/connections/{id}/line/diagnostic         - Lancer nouveau
GET  /access/connections/{id}/line/diagnostic/history - Historique
