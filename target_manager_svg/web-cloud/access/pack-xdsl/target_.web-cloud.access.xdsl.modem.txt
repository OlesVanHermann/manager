# STRUCTURE PAGE
1. LEFT PANEL - Accès xDSL + NAV3 (Modem actif)
2. RIGHT PANEL:
   - TITRE "Configuration du modem"
   - NAV4: Général, WiFi, LAN, DHCP, Firewall, NAT
   - INFOS MODEM: Modèle, MAC, Série, Firmware, Mode
   - ACTIONS: Redémarrer, Réinitialiser, Sauvegarder, Bridge
   - CONFIG RESEAU: MTU, DNS

# NAV4 MODEM
| Tab      | Description              |
|----------|--------------------------|
| Général  | Infos et actions         |
| WiFi     | Configuration WiFi       |
| LAN      | Configuration LAN        |
| DHCP     | Serveur DHCP             |
| Firewall | Règles firewall          |
| NAT      | Port forwarding          |

# MODES MODEM
| Mode    | Description              |
|---------|--------------------------|
| Routeur | Mode par défaut          |
| Bridge  | Modem passif (propre routeur) |

# API ENDPOINTS
GET /xdsl/{serviceName}/modem - Infos modem
POST /xdsl/{serviceName}/modem/reboot - Redémarrer
POST /xdsl/{serviceName}/modem/reset - Réinitialiser
PUT /xdsl/{serviceName}/modem - Modifier config
