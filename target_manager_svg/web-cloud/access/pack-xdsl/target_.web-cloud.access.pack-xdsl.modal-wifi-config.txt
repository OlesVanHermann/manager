# STRUCTURE MODAL
1. Header: Titre "Accès WiFi" + bouton fermer
2. Zone QR Code: QR code scannable
3. Infos WiFi: SSID, nom, sécurité
4. Info box: Note sur la clé de sécurité
5. Boutons: Fermer + Copier

# QR CODE WIFI
- Format standard WIFI: pour connexion automatique
- Encode: protocole, SSID, mot de passe
- Compatible iOS/Android

# INFORMATIONS AFFICHÉES
- SSID: Identifiant du réseau
- Nom WiFi: Nom affiché (peut différer du SSID)
- Sécurité: Type d'encryption (WPA2, WPA3)

# INTERACTIONS
- Scanner QR → Connexion automatique sur mobile
- Clic "Copier" → Copie les infos de connexion
- Clic "Fermer" ou × → Ferme modal

# NOTE
Si la clé n'a pas été modifiée depuis l'installation,
le QR code physique sur le modem est utilisable.

# API ENDPOINTS
GET /access/connections/{id}/modem/wifi         - Config WiFi
GET /access/connections/{id}/modem/wifi/qrcode  - QR code image/data
