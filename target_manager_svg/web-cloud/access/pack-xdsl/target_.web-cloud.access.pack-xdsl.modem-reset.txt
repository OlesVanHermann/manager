# STRUCTURE PAGE
1. NAV3: ... | [Modem] | ...
2. 3 options de réinitialisation côte à côte
3. Warning box: Avertissements importants
4. Bloc DÉTAILS: Explications détaillées

# OPTIONS DE RÉINITIALISATION
1. **Réinitialiser** (soft reset)
   - Redémarre le modem
   - Conserve la configuration
   - Style: neutral

2. **Réinit. config OVH** (factory reset)
   - Restaure config d'usine OVH
   - EFFACE les personnalisations
   - Style: warning (orange)

3. **Reconfigurer VoIP**
   - Reconfigure téléphonie uniquement
   - Conserve autres paramètres
   - Style: neutral

# INTERACTIONS
- Clic bouton → Modal confirmation obligatoire
- Confirmation → POST action + tâche créée
- Pendant reset → Spinner + polling

# API ENDPOINTS
POST /access/connections/{id}/modem/reset        - Soft reset
POST /access/connections/{id}/modem/factoryReset - Factory reset OVH
POST /access/connections/{id}/modem/voipReset    - Reset VoIP
