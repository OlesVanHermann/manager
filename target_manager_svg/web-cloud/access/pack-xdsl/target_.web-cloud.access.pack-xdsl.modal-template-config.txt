# STRUCTURE MODAL
1. Icône warning (cercle jaune)
2. Titre: "Suppression du template..."
3. Question confirmation avec nom template
4. Boutons: Annuler + Confirmer

# CONTEXTE
- Apparaît lors de la suppression d'un template modem
- Les templates sont des configurations prédéfinies
- Action irréversible

# TEMPLATES MODEM
- Configurations sauvegardées (WiFi, DHCP, NAT, etc.)
- Peuvent être appliquées à plusieurs modems
- Utiles pour standardiser les configurations

# INTERACTIONS
- Clic "Annuler" → Ferme modal, template conservé
- Clic "Confirmer" → DELETE template

# API ENDPOINTS
DELETE /access/connections/{id}/modem/templates/{templateId}
