## 1. RÉSUMÉ EXÉCUTIF

Page de configuration WiFi du modem xDSL permettant d'activer/désactiver
le réseau sans fil et d'accéder à la configuration détaillée.

### Utilisateurs cibles
- Clients disposant d'un modem xDSL géré par OVHcloud

### Objectif fonctionnel
- Afficher l'état du WiFi (activé/désactivé)
- Permettre l'activation/désactivation rapide
- Accéder à la configuration détaillée des réseaux WiFi
