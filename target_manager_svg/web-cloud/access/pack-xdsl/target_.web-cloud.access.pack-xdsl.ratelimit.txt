# STRUCTURE PAGE
1. NAV3: ... | [Ligne] | ...
2. Bloc LIMITE DE DÉBIT: Description + slider + checkbox + warning
3. Bouton validation

# OPTIMISATION DÉBIT
- Limite haute du débit au niveau LNS (équipements OVH)
- Permet de stabiliser la connexion
- Réduit les micro-coupures sur lignes instables

# SLIDER
- 0% à 100% du débit max théorique
- Valeur en temps réel affichée à droite
- Granularité: 5%

# CHECKBOX
- "Supprimer la limite" = 100% du débit
- Désactive le slider si coché

# AVERTISSEMENT
- Opération peut prendre 10 minutes
- Connexion interrompue pendant l'opération
- Éviter pendant télétravail/visio

# CAS D'USAGE
- Ligne instable → Réduire à 80-90%
- Micro-coupures → Réduire à 70-80%
- Ligne stable → 100% (pas de limite)

# INTERACTIONS
- Drag slider → Ajuste limite
- Clic checkbox → Toggle 100%
- Clic "Valider" → PUT nouvelle limite

# API ENDPOINTS
GET  /access/connections/{id}/ratelimit        - Limite actuelle
PUT  /access/connections/{id}/ratelimit        - Modifier limite
Body: { "percent": 65 } ou { "unlimited": true }
