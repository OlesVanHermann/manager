# STRUCTURE MODAL
1. Header: Titre + bouton fermer (×)
2. Timestamp: Heure du diagnostic
3. Lignes de statut: 5 checks avec indicateurs colorés
4. Séparateur
5. Info: Diagnostics restants
6. Boutons: Relancer (outline) + Fermer (primary)

# INDICATEURS STATUT
| Couleur | Signification |
|---------|---------------|
| ● Vert (#10B981) | OK / Nominal |
| ● Orange (#F59E0B) | Warning / Dégradé |
| ● Rouge (#EF4444) | Erreur / KO |

# CHECKS EFFECTUÉS
1. Connexion DSLAM - État connexion au central
2. Synchronisation - État sync ligne
3. Signal ligne - Qualité du signal
4. Modem - État du modem
5. Ping gateway - Latence réseau

# INTERACTIONS
- Clic "Relancer" → POST nouveau diagnostic
- Clic "Fermer" → Ferme modal
- Clic × → Ferme modal

# API ENDPOINTS
GET  /access/connections/{id}/diagnostics/{diagId} - Détails diagnostic
POST /access/connections/{id}/diagnostics          - Lancer diagnostic
GET  /access/connections/{id}/diagnostics/remaining - Diagnostics restants
