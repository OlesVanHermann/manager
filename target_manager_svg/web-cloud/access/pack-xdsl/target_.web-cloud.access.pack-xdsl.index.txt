1. NAV1 - Header OVH (Web Cloud actif, x=510)
2. NAV2 - Web Cloud (Accès actif, x=550)
3. LEFT PANEL - Liste des connexions (4 exemples)
4. RIGHT PANEL
   - Header: Nom connexion + actions (Migrer, Déménager)
   - NAV3: 7 tabs (Général, Ligne, Modem, Services, VoIP, Options, Tâches)
   - Contenu: Tab Général par défaut

# BLOCS AFFICHÉS (Tab Général)
1. CONNEXION - Statut, type, offre, débit sync, adresse
2. MODEM - Modèle, gestion (OVH/perso), lien config
3. SERVICES INCLUS - Liste services du pack
4. OPTIONS ACTIVES - Badges options + bouton ajouter

# INTERACTIONS
- Clic tab → Navigation vers section
- Clic "Migrer" → Modal migration offre
- Clic "Déménager" → Modal déménagement
- Clic item left panel → Sélectionne autre connexion
- Clic "Configurer" modem → Tab Modem
- Clic "Gérer services" → Tab Services
- Clic "+ Ajouter option" → Modal options

# API ENDPOINTS
GET    /access/connections                    - Liste connexions
GET    /access/connections/{id}               - Détails connexion
PUT    /access/connections/{id}               - Modifier connexion
POST   /access/connections/{id}/migrate       - Migrer offre
POST   /access/connections/{id}/move          - Déménager
GET    /access/connections/{id}/line          - Statut ligne
GET    /access/connections/{id}/modem         - Infos modem
GET    /access/connections/{id}/services      - Services inclus
GET    /access/connections/{id}/options       - Options actives
