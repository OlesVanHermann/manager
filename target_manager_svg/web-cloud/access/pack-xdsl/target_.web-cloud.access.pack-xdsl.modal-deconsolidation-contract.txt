# STRUCTURE MODAL
1. Header: Titre + bouton fermer (×)
2. Zone CONTENU: Question confirmation + service + prix
3. Warning: Avertissement téléphone non transféré
4. Checkboxes: 3 conditions à accepter
5. Info banner: Indication validation
6. Boutons: Annuler (outline) + Valider (disabled tant que non coché)

# CHECKBOXES
1. ✓ Engagement 12 mois (coché)
2. ✓ CGV (coché)
3. ☐ Frais résiliation (non coché)

# ÉTATS BOUTON VALIDER
- Disabled (gris): Tant que toutes les cases ne sont pas cochées
- Enabled (bleu): Quand toutes les cases sont cochées

# INTERACTIONS
- Clic checkbox → Toggle état
- Clic "Annuler" → Ferme modal
- Clic "Valider" (enabled) → POST contrat + ferme modal
- Clic × → Ferme modal

# API ENDPOINTS
GET  /access/connections/{id}/deconsolidation - Infos dégroupage
POST /access/connections/{id}/deconsolidation - Valider contrat
