# STRUCTURE MODAL
1. Icône warning (cercle jaune)
2. Titre: "Adresse IP refusée"
3. Message explicatif
4. DNS valides fournis
5. Bouton "Compris"

# CONTEXTE
- Apparaît lors de la configuration DHCP
- L'utilisateur a entré une IP invalide/obsolète
- Indique les DNS OVH corrects à utiliser

# DNS OVH OFFICIELS
- IPv4: 213.186.33.99
- IPv6: 2001:41d0:3:163::1

# INTERACTIONS
- Clic "Compris" → Ferme modal, retour formulaire DHCP
- L'utilisateur peut corriger sa saisie

# VALIDATION
- Bloque les anciens DNS OVH dépréciés
- Suggère les DNS actuels
