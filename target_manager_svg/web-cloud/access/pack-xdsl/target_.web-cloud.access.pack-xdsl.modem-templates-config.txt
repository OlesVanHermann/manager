# STRUCTURE PAGE
1. NAV3: ... | [Modem] | ...
2. Table TEMPLATES: Nom + date + actions
3. Séparateur
4. Détail TEMPLATE sélectionné: DHCP + WiFi + Ports

# TEMPLATES
Configurations sauvegardées applicables à plusieurs modems.
Utile pour standardiser ou restaurer une config.

# SECTIONS DÉTAIL

## DHCP
- Serveur, passerelle, DNS, plage d'adresses

## WiFi
- SSID, sécurité, canal, fréquence

## DMZ/MTU
- IP DMZ, valeur MTU

## Redirections de ports
- Liste des règles NAT

# INTERACTIONS
- Clic template → Affiche détail
- Clic "Modifier" → Mode édition
- Clic "Supprimer" → Modal confirmation
- Clic "Appliquer" → Applique au modem actuel

# API ENDPOINTS
GET    /access/connections/{id}/modem/templates       - Liste templates
GET    /access/connections/{id}/modem/templates/{tid} - Détail template
PUT    /access/connections/{id}/modem/templates/{tid} - Modifier template
DELETE /access/connections/{id}/modem/templates/{tid} - Supprimer template
POST   /access/connections/{id}/modem/templates/{tid}/apply - Appliquer
