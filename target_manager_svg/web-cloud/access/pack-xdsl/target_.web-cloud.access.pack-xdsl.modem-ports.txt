# STRUCTURE PAGE
1. NAV4: Dashboard | WiFi | DHCP | [NAT/Ports] | Routeur
2. Bannière info: Prérequis firewall
3. Bloc RÈGLES DE REDIRECTION: Table complète avec toutes les colonnes

# COLONNES TABLE
- Nom: Identifiant de la règle
- IP source: Filtre source (Toutes ou IP spécifique)
- Port ext.: Port externe début
- Port fin: Port externe fin
- Protocole: TCP/UDP/Both
- Client interne: IP destination LAN
- Port int.: Port interne début
- Port fin int.: Port interne fin
- Statut: Actif/Inactif
- Actions: Modifier/Supprimer

# INTERACTIONS
- Clic "+ Ajouter règle" → Modal création
- Clic ✎ → Mode édition inline
- Clic × → Confirmation + DELETE
- Toggle statut → Active/désactive règle

# API ENDPOINTS
GET    /access/connections/{id}/modem/ports           - Liste règles
POST   /access/connections/{id}/modem/ports           - Créer règle
PUT    /access/connections/{id}/modem/ports/{ruleId}  - Modifier règle
DELETE /access/connections/{id}/modem/ports/{ruleId}  - Supprimer règle
