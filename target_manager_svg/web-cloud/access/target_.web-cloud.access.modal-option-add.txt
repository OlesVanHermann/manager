# STRUCTURE MODAL

Modal de souscription d'options pour une ligne d'accès.
Dimensions: 600x550px

1. HEADER:
   - Titre "Souscrire une option"
   - Bouton fermer (×)

2. LISTE OPTIONS:
   - Radio buttons pour sélection unique
   - Nom de l'option
   - Description
   - Prix mensuel

3. INFO:
   - Note sur la facturation

4. ACTIONS:
   - Annuler → Ferme modal
   - Souscrire → POST option

# OPTIONS DISPONIBLES

| Option | Prix | Description |
|--------|------|-------------|
| IP Fixe v4 | 2,99€/mois | IP publique statique |
| Bloc IPv6 /56 | Gratuit | Bloc IPv6 natif |
| IP Failover | 4,99€/mois | IP portable |
| GTR 4h | 14,99€/mois | Intervention prioritaire |
| Modem de secours | 9,99€/mois | Backup 4G |
| Reverse DNS | Gratuit | PTR personnalisé |

# ÉTATS OPTIONS

- Disponible: Sélectionnable
- Déjà souscrite: Grisée avec badge "Actif"
- Incompatible: Grisée avec message

# INTERACTIONS

- Sélection radio → Une seule option à la fois
- Annuler → Ferme sans sauvegarder
- Souscrire → POST /access/{id}/options

# API ENDPOINTS

GET  /access/{id}/options/available    - Options disponibles
GET  /access/{id}/options              - Options actives
POST /access/{id}/options              - Souscrire option
DELETE /access/{id}/options/{optionId} - Résilier option

# FACTURATION

- Première facture au prorata
- Renouvellement automatique mensuel
- Résiliation à tout moment (prorata)
