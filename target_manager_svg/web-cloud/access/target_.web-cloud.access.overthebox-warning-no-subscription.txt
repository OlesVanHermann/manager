# STRUCTURE PAGE

Page affichée quand l'utilisateur n'a aucun OverTheBox.

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. CONTENT CENTRÉ:
   - Icône package
   - Titre "Aucun OverTheBox"
   - Description
   - Liste des avantages
   - Bouton commander

# QUAND AFFICHER

Cette page s'affiche quand:
- GET /overthebox retourne une liste vide
- L'utilisateur n'a jamais commandé d'OTB
- Tous les OTB ont été résiliés

# AVANTAGES PRÉSENTÉS

- Agrégation jusqu'à 4 connexions
- Haute disponibilité (failover)
- Accès distant sécurisé

# INTERACTIONS

- Commander → /order/overthebox
- Retour → Liste accès

# API ENDPOINTS

GET /overthebox                        - Liste des OTB (vide)
POST /order/overthebox                 - Nouvelle commande
