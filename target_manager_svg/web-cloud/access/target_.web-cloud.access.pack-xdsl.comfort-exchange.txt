# Cahier des Charges : web-cloud/access/pack-xdsl - Comfort Exchange

## 1. VUE D'ENSEMBLE
- **Type** : Page processus échange modem
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem/exchange
- **Objectif** : Commander échange modem défectueux

## 2. STRUCTURE PAGE
- Titre principal
- Description du processus
- Formulaire sélection contact
- Boutons navigation

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsSelect** : Sélecteur contact livraison
- **OdsButton** : Annuler, Valider

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Échange de matériel" |
| Style | Titre principal bold |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Ligne 1 | "Vous pouvez demander l'échange de votre modem" |
| Ligne 2 | "si celui-ci est défectueux." |

## 6. SÉLECTEUR CONTACT
| Propriété | Valeur |
|-----------|--------|
| Label | "Sélectionner le contact de livraison" |
| Type | OdsSelect dropdown |
| Options | Liste contacts /me/contact |
| Valeur exemple | "Jean Dupont - 12 rue..." |
| Largeur | 400px |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary outline | "Annuler" |
| Valider | Primary | "Commander l'échange" |

## 8. APIS & ENDPOINTS
- GET /pack/xdsl/{packId}/exchangableModems - Modems éligibles
- GET /me/contact - Liste contacts
- POST /pack/xdsl/{packId}/modem/exchange - Commander échange
