# STRUCTURE MODAL

Modal d'ajout d'un accès distant (remote) sur OverTheBox.
Dimensions: 550x520px

1. HEADER:
   - Titre "Ajouter un accès distant"
   - Bouton fermer (×)

2. FORMULAIRE:
   - Nom du tunnel
   - Protocole (toggle buttons)
   - IP locale
   - Port local
   - Port distant (optionnel)

3. INFO:
   - URL d'accès générée

4. ACTIONS:
   - Annuler → Ferme modal
   - Créer → POST remote

# PROTOCOLES SUPPORTÉS

| Protocole | Port défaut | Usage |
|-----------|-------------|-------|
| SSH | 22 | Administration serveur |
| HTTP | 80 | Web non sécurisé |
| HTTPS | 443 | Web sécurisé |
| TCP | - | Générique (RDP, etc.) |

# URL D'ACCÈS

Format: {protocol}://{serviceId}.overthebox.net:{exposedPort}

Exemples:
- ssh://otb-abc123.overthebox.net:45678
- https://otb-abc123.overthebox.net:45679

# INTERACTIONS

- Sélection protocole → Toggle buttons
- Port distant vide → Attribution automatique
- Annuler → Ferme sans sauvegarder
- Créer → POST /overthebox/{id}/remotes

# API ENDPOINTS

GET  /overthebox/{id}/remotes              - Liste remotes
POST /overthebox/{id}/remotes              - Créer remote
PUT  /overthebox/{id}/remotes/{remoteId}   - Modifier remote
DELETE /overthebox/{id}/remotes/{remoteId} - Supprimer remote

# VALIDATION

- Nom: obligatoire, 1-50 caractères
- IP: format IPv4 valide dans le réseau local
- Port local: 1-65535
- Port distant: 1024-65535 (si spécifié)
