# STRUCTURE MODAL

Modal de déclaration d'un modem personnel (BYOD).
Dimensions: 600x480px

1. HEADER:
   - Titre "Déclarer un modem personnel"
   - Bouton fermer (×)

2. WARNING:
   - Avertissement support limité

3. FORMULAIRE:
   - Marque (dropdown)
   - Modèle (dropdown dépendant)
   - Adresse MAC
   - Identifiant PPPoE (lecture seule)
   - Bouton voir mot de passe

4. ACTIONS:
   - Annuler → Ferme modal
   - Déclarer → POST modem

# MARQUES COMPATIBLES

- Netgear
- TP-Link
- ASUS
- Synology
- Ubiquiti
- Autre (générique)

# INFORMATIONS PPPoE

Les identifiants PPPoE sont fournis automatiquement :
- Login: fti/xxxxxx (fourni par OVH)
- Password: généré (à demander)

# INTERACTIONS

- Sélection marque → Charge modèles compatibles
- Copier login → Clipboard
- Voir mot de passe → Modal confirmation
- Annuler → Ferme sans sauvegarder
- Déclarer → POST /access/{id}/modem/personal

# API ENDPOINTS

GET  /access/modem/brands              - Liste marques
GET  /access/modem/brands/{id}/models  - Modèles par marque
GET  /access/{id}/pppoe                - Identifiants PPPoE
POST /access/{id}/modem/personal       - Déclarer modem
DELETE /access/{id}/modem/personal     - Retirer déclaration

# VALIDATION

- Marque: obligatoire
- Modèle: obligatoire
- MAC: format XX:XX:XX:XX:XX:XX valide
