# STRUCTURE PAGE

Page d'avertissement affichée quand un OverTheBox n'est pas lié au compte.

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. CONTENT CENTRÉ:
   - Icône warning
   - Titre "Équipement non lié"
   - Description
   - Instructions étape par étape
   - Bouton d'action

# QUAND AFFICHER

Cette page s'affiche quand:
- L'OTB a été acheté mais pas encore activé
- L'OTB n'a pas été physiquement connecté
- La liaison avec le compte a échoué

# PRÉREQUIS LIAISON

1. OverTheBox branché et allumé
2. Connexion Internet active sur l'OTB
3. Voyant LED vert fixe

# INTERACTIONS

- Lier l'équipement → Lance le processus de liaison
- Retour → Liste des OverTheBox

# API ENDPOINTS

GET  /overthebox/{id}                  - État de l'OTB
POST /overthebox/{id}/link             - Lancer liaison

# PROCESSUS DE LIAISON

1. POST /link déclenche la recherche de l'équipement
2. L'OTB se manifeste auprès des serveurs OVH
3. Association automatique au compte client
4. Redirection vers la page de configuration
