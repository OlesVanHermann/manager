# STRUCTURE MODAL

Modal pour lier une connexion Internet à un OverTheBox.
Dimensions: 550x420px

1. HEADER:
   - Titre "Lier à un OverTheBox"
   - Bouton fermer (×)

2. RECHERCHE:
   - Champ de recherche

3. LISTE OTB:
   - Liste des OverTheBox disponibles
   - Sélection par radio/check

4. INFO:
   - Note sur l'agrégation

5. ACTIONS:
   - Annuler → Ferme modal
   - Lier → POST liaison

# INFORMATIONS AFFICHÉES PAR OTB

- Icône et nom
- Service ID
- Nombre de connexions
- Statut (Actif/Inactif)

# FILTRAGE

La recherche filtre sur:
- Nom de l'OTB
- Service ID
- Localisation

# INTERACTIONS

- Recherche → Filtre la liste
- Sélection OTB → Radio button
- Annuler → Ferme sans sauvegarder
- Lier → POST /access/{id}/overthebox

# API ENDPOINTS

GET  /overthebox                       - Liste des OTB
POST /access/{id}/overthebox           - Lier à OTB
DELETE /access/{id}/overthebox         - Délier de OTB

# VALIDATION

- Un OTB doit être sélectionné
- L'OTB doit être actif
- La connexion ne doit pas déjà être liée
