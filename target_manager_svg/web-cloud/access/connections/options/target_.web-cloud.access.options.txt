STRUCTURE PAGE
1. NAV3: General | Ligne | Modem | Services | VoIP | Options (actif) | Taches
2. Card informations abonnement
   - Reference, Offre, Date activation, Engagement
   - Facturation, Prochain renouvellement, Prix
3. Card nom d'affichage
   - Champ modification nom personnalise
4. Card notifications
   - Toggle alertes email
   - Toggle alertes SMS
5. Card mot de passe
   - Bouton modifier mot de passe espace client
6. Card demenagement
   - Bouton declarer demenagement
7. Zone de danger
   - Bouton resilier abonnement (irreversible)

ACTIONS DISPONIBLES
| Action               | Description                              |
|----------------------|------------------------------------------|
| Changer d'offre      | Migrer vers une offre superieure/inf.   |
| Modifier nom         | Personnaliser le nom d'affichage        |
| Alertes email        | Recevoir notifications par email        |
| Alertes SMS          | Recevoir notifications par SMS          |
| Modifier password    | Changer mot de passe espace client      |
| Demenagement         | Transferer connexion nouvelle adresse   |
| Resilier             | Mettre fin a l'abonnement               |

API ENDPOINTS
PUT    /access/connections/{id}/description    -> Modifier nom
PUT    /access/connections/{id}/notifications  -> Config notifications
POST   /access/connections/{id}/password       -> Changer mot de passe
POST   /access/connections/{id}/move           -> Declarer demenagement
DELETE /access/connections/{id}                -> Resilier abonnement
