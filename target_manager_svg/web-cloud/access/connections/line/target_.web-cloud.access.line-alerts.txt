STRUCTURE PAGE
1. NAV3: General | Ligne (actif) | Modem | Services | VoIP | Options | Taches
2. NAV4: Statut | Diagnostic | Statistiques | Alertes (actif)
3. Header avec bouton configuration
4. Resume alertes actives
   - Compteur critiques (rouge)
   - Compteur avertissements (orange)
   - Compteur informations (bleu)
   - Etat general (vert si OK)
5. Table historique alertes
   - Date, Type, Message, Statut, Actions
   - Code couleur selon severite
6. Pagination

TYPES D'ALERTES
| Type     | Couleur    | Badge      | Exemples                        |
|----------|------------|------------|----------------------------------|
| Critique | #DC2626    | bg-error   | Deconnexion, Panne materiel     |
| Warning  | #F59E0B    | bg-warning | Latence elevee, Debit degrade   |
| Info     | #0050D7    | bg-info    | Maintenance, Mise a jour, IP    |

STATUTS ALERTES
| Statut   | Couleur    | Description                      |
|----------|------------|----------------------------------|
| Active   | #F59E0B    | Probleme en cours                |
| Resolue  | #E5E7EB    | Probleme corrige automatiquement |
| Ignoree  | #6B7280    | Ignoree par l'utilisateur        |

API ENDPOINTS
GET    /access/connections/{id}/alerts              -> Liste alertes
GET    /access/connections/{id}/alerts/{alertId}    -> Details alerte
PUT    /access/connections/{id}/alerts/{alertId}    -> Ignorer alerte
GET    /access/connections/{id}/alerts/config       -> Config notifications
PUT    /access/connections/{id}/alerts/config       -> Modifier config
