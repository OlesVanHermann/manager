STRUCTURE PAGE
1. NAV3: General | Ligne | Modem (actif) | Services | VoIP | Options | Taches
2. NAV4: Dashboard | WiFi | DHCP (actif) | NAT/Ports | Routeur
3. Card configuration DHCP
   - Toggle activation
   - Plage IP (debut - fin)
   - Passerelle, DNS primaire, DNS secondaire
4. Card resume
   - Compteurs: disponibles, actifs, reservations
   - Barre visuelle utilisation
5. Table baux DHCP actifs
   - Colonnes: Nom, MAC, IP, Type, Expiration, Actions
   - Bouton ajouter reservation statique

TYPES DE BAUX
| Type       | Badge      | Expiration | Description              |
|------------|------------|------------|--------------------------|
| Dynamique  | bg-success | Variable   | Attribue automatiquement |
| Statique   | bg-info    | -          | IP fixe reservee         |

ACTIONS DISPONIBLES
| Action    | Description                            |
|-----------|----------------------------------------|
| Reserver  | Creer reservation statique pour l'IP  |
| Bloquer   | Empecher l'appareil de se connecter   |
| Modifier  | Changer IP de reservation statique    |
| Supprimer | Retirer reservation statique          |

API ENDPOINTS
GET  /access/connections/{id}/modem/dhcp              -> Config DHCP
PUT  /access/connections/{id}/modem/dhcp              -> Modifier config
GET  /access/connections/{id}/modem/dhcp/leases       -> Liste baux
POST /access/connections/{id}/modem/dhcp/reservations -> Creer reservation
DELETE /access/connections/{id}/modem/dhcp/reservations/{id} -> Supprimer
