STRUCTURE PAGE
1. NAV3: General | Ligne | Modem (actif) | Services | VoIP | Options | Taches
2. NAV4: Dashboard | WiFi (actif) | DHCP | NAT/Ports | Routeur
3. Card WiFi 2.4 GHz
   - Toggle activation
   - Champs: SSID, Mot de passe, Securite, Canal, Largeur canal
   - Compteur appareils connectes
4. Card WiFi 5 GHz
   - Meme structure que 2.4 GHz
5. Table appareils connectes
   - Colonnes: Appareil, MAC, IP, Bande, Signal, Debit
   - Indicateur qualite signal (badge couleur)

OPTIONS SECURITE
| Option    | Description                    |
|-----------|--------------------------------|
| WPA3      | Derniere norme (recommande)    |
| WPA2/WPA3 | Compatibilite anciens appareils|
| WPA2      | Compatible tous appareils      |

API ENDPOINTS
GET  /access/connections/{id}/modem/wifi          -> Config WiFi
PUT  /access/connections/{id}/modem/wifi          -> Modifier WiFi
GET  /access/connections/{id}/modem/wifi/devices  -> Appareils connectes
POST /access/connections/{id}/modem/wifi/password -> Changer mot de passe
