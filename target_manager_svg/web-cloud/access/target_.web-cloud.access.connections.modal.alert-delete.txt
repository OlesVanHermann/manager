# STRUCTURE MODAL

Modal de confirmation de suppression d'alerte.
Dimensions: 450x280px

1. HEADER:
   - Titre "Supprimer l'alerte"
   - Bouton fermer (×)

2. CONTENU:
   - Icône warning (cercle rouge)
   - Message de confirmation
   - Détail de l'alerte à supprimer (fond rouge clair)

3. ACTIONS:
   - Annuler → Ferme modal
   - Supprimer → DELETE + ferme

# ÉLÉMENTS VISUELS

- Icône warning: cercle rouge clair (#FEF2F2) avec ⚠ rouge
- Détail alerte: fond #FEE2E2, texte #DC2626
- Bouton supprimer: rouge danger (#DC2626)

# INTERACTIONS

- Annuler → Ferme sans action
- Supprimer → DELETE /alerts/{alertId}
- × (fermer) → Ferme sans action

# API ENDPOINTS

DELETE /access/{id}/alerts/{alertId}   - Supprime l'alerte

# CONFIRMATION

Cette modal est déclenchée depuis:
- Bouton supprimer dans la liste des alertes
- Menu actions sur une alerte
