# STRUCTURE MODAL

Modal de configuration WiFi du modem.
Dimensions: 600x520px

1. HEADER:
   - Titre "Configuration WiFi"
   - Bouton fermer (×)

2. FORMULAIRE:
   - Sélection bande (2.4 GHz / 5 GHz)
   - SSID (nom réseau)
   - Type de sécurité (dropdown)
   - Clé de sécurité (avec toggle visibilité)
   - Canal (dropdown)

3. ACTIONS:
   - Annuler → Ferme modal
   - Enregistrer → PUT config

# BANDES DE FRÉQUENCE

| Bande | Caractéristiques |
|-------|-----------------|
| 2.4 GHz | Meilleure portée, moins de débit |
| 5 GHz | Meilleur débit, portée réduite |

# TYPES DE SÉCURITÉ

- WPA3-Personal (le plus sécurisé)
- WPA2-PSK (AES) (recommandé)
- WPA/WPA2 Mixed
- WEP (obsolète, déconseillé)
- Open (non sécurisé)

# CANAUX

**2.4 GHz:** 1-13 ou Automatique
**5 GHz:** 36, 40, 44, 48, 52, 56, 60, 64, 100+ ou Automatique

# INTERACTIONS

- Sélection bande → Recharge config de la bande
- Toggle visibilité → Affiche/masque mot de passe
- Annuler → Ferme sans sauvegarder
- Enregistrer → PUT /modem/{id}/wifi

# API ENDPOINTS

GET  /modem/{id}/wifi              - Configuration WiFi
PUT  /modem/{id}/wifi              - Modifier WiFi
GET  /modem/{id}/wifi/channels     - Canaux disponibles

# VALIDATION

- SSID: 1-32 caractères
- Clé: 8-63 caractères (WPA2)
- Canal: valeur valide pour la bande
