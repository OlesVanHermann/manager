# Cahier des Charges : web-cloud/access/pack-xdsl - Modem DMZ

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration DMZ du modem
- **Objectif** : Configurer la zone démilitarisée
- **Contexte** : Exposition d'un appareil à Internet (tous ports)

## 2. LAYOUT & STRUCTURE
- 2 colonnes : Config DMZ | Warning sécurité
- Tableau avec édition inline
- Bannière info explicative

## 3. COMPOSANTS UI
- **OdsCard** : 2 cartes
- **OdsTable** : Config DMZ
- **OdsInput** : Adresse DMZ
- **OdsActionMenu** : Menu ⋮ pour modifier
- **OdsMessage** : Warning sécurité (jaune)

## 4. BANNIÈRE INFO
- Message ligne 1 : "La DMZ expose un appareil directement à Internet"
- Message ligne 2 : "Tous les ports entrants seront redirigés vers cette adresse IP"

## 5. TABLEAU CONFIG
| Colonne | Description |
|---------|-------------|
| Adresse modem | IP modem (192.168.1.254) - lecture seule |
| Masque réseau | Masque (255.255.255.0) - lecture seule |
| Adresse DMZ | IP exposée ou "(Aucune)" |
| Actions | Menu ⋮ → Modifier |

## 6. MODE ÉDITION
- Fond bleu clair (#F0F9FF)
- Input adresse DMZ : "192.168.1.___"
- Boutons ✓ valider, ✕ annuler

## 7. WARNING SÉCURITÉ
- Titre : "⚠ Avertissement sécurité"
- Message :
  - "L'utilisation de la DMZ expose votre appareil à tous les ports entrants d'Internet"
  - "Assurez-vous que l'appareil dispose d'un pare-feu actif"

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/dmz : Config actuelle
- PUT /xdsl/{serviceName}/modem/dmz : Modifier IP DMZ
