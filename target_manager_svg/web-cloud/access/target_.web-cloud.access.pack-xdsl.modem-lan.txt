# Cahier des Charges : web-cloud/access/pack-xdsl - Modem LAN

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration LAN du modem
- **Objectif** : Configurer l'adressage IP du réseau local
- **Contexte** : Configuration réseau avancée

## 2. LAYOUT & STRUCTURE
- 2 colonnes : Config LAN | Explication types
- Tableau éditable inline
- Bandeau info

## 3. COMPOSANTS UI
- **OdsCard** : 2 cartes
- **OdsTable** : Config LAN
- **OdsSelect** : Type d'adressage
- **OdsInput** : IP, Masque
- **OdsButton** : ✎ éditer, ✓ valider, ✕ annuler
- **OdsMessage** : Bandeau info bleu

## 4. TABLEAU CONFIG LAN
| Colonne | Description |
|---------|-------------|
| Type d'adressage ⓘ | DHCP ou Statique |
| Adresse IP ⓘ | IP du modem (192.168.1.254) |
| Masque de sous-réseau ⓘ | Masque (255.255.255.0) |
| Actions | Bouton édition |

## 5. MODE ÉDITION
- Fond bleu clair (#F0F9FF)
- Type : Dropdown DHCP/Statique
- IP : Input modifiable
- Masque : Input modifiable
- Boutons ✓/✕

## 6. BANDEAU INFO
- Ligne 1 : "Modifiez l'adresse IP du modem avec précaution"
- Ligne 2 : "Un changement peut interrompre la connexion de vos appareils"

## 7. TYPES D'ADRESSAGE
### DHCP
- Attribution automatique des adresses IP aux appareils

### Statique
- Configuration manuelle des adresses IP sur chaque appareil

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/lan : Config LAN
- PUT /xdsl/{serviceName}/modem/lan : Modifier config
