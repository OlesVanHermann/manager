# STRUCTURE MODAL

Modal de configuration DHCP du modem.
Dimensions: 600x480px

1. HEADER:
   - Titre "Configuration DHCP"
   - Bouton fermer (×)

2. FORMULAIRE:
   - Toggle activer/désactiver DHCP
   - Plage IP (début/fin)
   - Durée du bail
   - Serveurs DNS (primaire/secondaire)

3. ACTIONS:
   - Annuler → Ferme modal
   - Appliquer → PUT config

# PARAMÈTRES DHCP

| Paramètre | Défaut | Description |
|-----------|--------|-------------|
| Actif | ON | Active le serveur DHCP |
| Début | 192.168.1.10 | Première IP distribuée |
| Fin | 192.168.1.254 | Dernière IP distribuée |
| Bail | 86400s | Durée de validité (24h) |
| DNS1 | 213.186.33.99 | DNS OVH primaire |
| DNS2 | 213.186.33.98 | DNS OVH secondaire |

# INTERACTIONS

- Toggle DHCP → Active/désactive tout le formulaire
- Annuler → Ferme sans sauvegarder
- Appliquer → PUT /modem/{id}/dhcp

# API ENDPOINTS

GET  /modem/{id}/dhcp              - Configuration DHCP
PUT  /modem/{id}/dhcp              - Modifier DHCP

# VALIDATION

- IP début < IP fin
- Plage dans le même sous-réseau
- DNS valides (format IPv4)
- Bail: 300-604800 secondes (5min-7j)
