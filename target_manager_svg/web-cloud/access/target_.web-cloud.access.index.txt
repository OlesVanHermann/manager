STRUCTURE PAGE
1. Sous-nav produit: [Connexions (N)] | [OverTheBox (N)]
2. Left panel: Liste des connexions
   - Nom, Type techno, Offre
   - Statut, Debits, Modem
   - Lien OTB si applicable
3. Right panel: Details connexion selectionnee
   - NAV3: General | Ligne | Modem | Services | VoIP | Options | Taches
   - Dashboard connexion

TYPES TECHNO
| Type   | Description                    |
|--------|--------------------------------|
| ADSL   | ADSL classique                 |
| VDSL2  | Very-high-bit-rate DSL         |
| FTTH   | Fiber To The Home              |
| FTTO   | Fiber To The Office            |
| 4G/LTE | Mobile 4G                      |
| 5G     | Mobile 5G                      |

TYPES OFFRES
| Offre         | Inclus                         |
|---------------|--------------------------------|
| Acces seul    | Connexion uniquement           |
| Pack Perso    | + Domaine + Emails basiques    |
| Pack Pro      | + Domaine + Emails + VoIP      |
| Pack Business | + Tout Pack Pro + GTR + IP     |

API ENDPOINTS
GET /access/connections           -> Liste connexions
GET /access/connections/{id}      -> Details connexion
GET /overTheBox                   -> Liste OverTheBox
