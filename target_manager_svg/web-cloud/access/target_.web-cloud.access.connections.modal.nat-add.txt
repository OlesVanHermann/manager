# STRUCTURE MODAL

Modal d'ajout de règle NAT/Port Forwarding.
Dimensions: 550x520px

1. HEADER:
   - Titre "Ajouter une règle NAT"
   - Bouton fermer (×)

2. FORMULAIRE:
   - Nom de la règle
   - Protocole (dropdown)
   - Port externe
   - IP interne
   - Port interne (optionnel)

3. INFO:
   - Note sur le port interne

4. ACTIONS:
   - Annuler → Ferme modal
   - Ajouter → POST règle

# PROTOCOLES

| Protocole | Usage typique |
|-----------|---------------|
| TCP | Web (HTTP/HTTPS), SSH, FTP |
| UDP | Gaming, VoIP, streaming |
| TCP+UDP | Les deux protocoles |

# PORTS COMMUNS

| Service | Port |
|---------|------|
| HTTP | 80 |
| HTTPS | 443 |
| SSH | 22 |
| FTP | 21 |
| RDP | 3389 |
| Minecraft | 25565 |

# INTERACTIONS

- Sélection protocole → Dropdown
- Annuler → Ferme sans sauvegarder
- Ajouter → POST /modem/{id}/nat

# API ENDPOINTS

GET  /modem/{id}/nat               - Liste règles NAT
POST /modem/{id}/nat               - Créer règle
PUT  /modem/{id}/nat/{ruleId}      - Modifier règle
DELETE /modem/{id}/nat/{ruleId}    - Supprimer règle

# VALIDATION

- Nom: obligatoire, 1-50 caractères
- Port externe: 1-65535
- IP interne: format IPv4 valide
- Port interne: 1-65535 (optionnel)
- Pas de conflit avec règle existante
