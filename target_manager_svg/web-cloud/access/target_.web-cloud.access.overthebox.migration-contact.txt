# Cahier des Charges : web-cloud/access/overthebox - Migration Contact

## 1. VUE D'ENSEMBLE
- **Type** : Page sélection contact livraison
- **URL** : /overthebox/{id}/migration/contact
- **Objectif** : Choisir adresse de livraison matériel

## 2. STRUCTURE PAGE
- Titre + description
- Grille de contacts (cards)
- Lien ajout contact
- Boutons action

## 3. COMPOSANTS UI
- **OdsCard** : Cards contact sélectionnables
- **OdsRadio** : Sélection contact (implicite via card)
- **OdsButton** : Annuler, Sélectionner
- **OdsLink** : Ajouter nouveau contact

## 4. CONTACT SÉLECTIONNÉ
| Propriété | Valeur |
|-----------|--------|
| Bordure | #0050D7 (2px) |
| Indicateur | Cercle bleu avec ✓ blanc |
| Position indicateur | Haut droite de la card |

## 5. CONTACT NON SÉLECTIONNÉ
| Propriété | Valeur |
|-----------|--------|
| Bordure | #E5E7EB |
| Indicateur | Aucun |

## 6. INFORMATIONS CONTACT
| Champ | Exemple |
|-------|---------|
| Nom | Jean Dupont |
| Adresse L1 | 12 rue de Paris |
| Adresse L2 | 75001 Paris |
| Pays | France |
| Téléphone | +33 1 23 45 67 89 |
| Email | jean.dupont@email.com |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary outline | "Annuler" |
| Sélectionner | Primary | "Sélectionner" |

## 8. LIEN AJOUT
| Propriété | Valeur |
|-----------|--------|
| Label | "+ Ajouter un nouveau contact" |
| Couleur | #0050D7 |
| Action | Ouvre modal/page ajout contact |

## 9. APIS & ENDPOINTS
- GET /me/contact - Liste contacts
- POST /me/contact - Créer contact
- POST /overthebox/{id}/migration/contact - Valider sélection
