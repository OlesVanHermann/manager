# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Firmware

## 1. VUE D'ENSEMBLE
- **Écran** : Mise à jour du firmware du modem
- **Objectif** : Gérer les versions de firmware
- **Contexte** : Maintenance et mises à jour

## 2. LAYOUT & STRUCTURE
- 2 colonnes : Sélection version | État mise à jour
- Dropdown versions disponibles
- Indicateur de progression

## 3. COMPOSANTS UI
- **OdsCard** : 2 cartes
- **OdsBadge** : "À jour" (vert)
- **OdsSelect** : Sélecteur version
- **OdsButton** : Appliquer (désactivé si à jour)
- Spinner : Progression mise à jour
- **OdsMessage** : Warning redémarrage

## 4. SECTION SÉLECTION
| Élément | Description |
|---------|-------------|
| Version actuelle | v3.2.1 + badge "À jour" |
| Sélecteur | Dropdown avec versions |
| Bouton Appliquer | Grisé si à jour |

## 5. DROPDOWN VERSIONS
- Option sélectionnée : Fond bleu + checkmark
- Versions : v3.2.1 (dernière), v3.2.0, etc.

## 6. SECTION MISE À JOUR EN COURS
- Titre : "Mise à jour en cours"
- Spinner + texte : "Installation du firmware v3.2.1"
- Info : "Ne débranchez pas le modem..."
- Warning : "Le modem va redémarrer automatiquement"

## 7. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/firmware : Version actuelle
- GET /xdsl/{serviceName}/modem/firmware/available : Versions dispo
- POST /xdsl/{serviceName}/modem/firmware/update : Lancer MAJ
