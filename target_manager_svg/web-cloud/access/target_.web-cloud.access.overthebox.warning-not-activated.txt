## 8. APIS & ENDPOINTS

```
GET /overTheBox/{serviceName}
→ Status: "notActivated" = affiche cet écran

GET /overTheBox/{serviceName}/device
→ Vérifie si le device est connecté

POST /overTheBox/{serviceName}/device/activate
→ Active l'OverTheBox

GET /me/order/{orderId}/follow
→ Suivi de la livraison

GET /me/order?date.from=xxx
→ Récupère la commande récente
```
