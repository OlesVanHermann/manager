## 8. APIS & ENDPOINTS

```
GET /telephony/{billingAccount}/line
→ Liste des lignes SIP

GET /telephony/{billingAccount}/line/{serviceName}
→ Détails d'une ligne

GET /telephony/{billingAccount}/line/{serviceName}/statistics
→ Statistiques d'appels

GET /freefax
→ Liste des EcoFax du compte

GET /telephony/{billingAccount}/voiceConsumption
→ Consommation voix du mois

GET /telephony/{billingAccount}/faxConsumption
→ Consommation fax du mois

POST /order/telephony/new
→ Commander une nouvelle ligne

# Redirection manager VoIP
/telephony/{billingAccount}
→ Ouvre le manager VoIP complet
```
