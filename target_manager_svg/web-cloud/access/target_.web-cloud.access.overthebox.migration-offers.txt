# Cahier des Charges : web-cloud/access/overthebox - Migration Offres

## 1. VUE D'ENSEMBLE
- **Type** : Page comparaison offres migration
- **URL** : /overthebox/{id}/migration/offers
- **Objectif** : Choisir nouvelle offre OTB

## 2. STRUCTURE PAGE
- Titre
- Tableau comparatif des offres
- Lien externe documentation
- Boutons choix offre

## 3. COMPOSANTS UI
- **OdsDatagrid** : Tableau comparatif
- **OdsSelect** : Sélecteur matériel
- **OdsButton** : Choisir offre
- **OdsLink** : Documentation externe
