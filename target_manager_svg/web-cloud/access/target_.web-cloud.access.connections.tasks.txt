# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste connexions
4. RIGHT PANEL
   - Header: Nom connexion
   - NAV3: 7 tabs (Tâches actif)
   - Contenu:
     - TOOLBAR:
       - Bouton refresh
       - Recherche
       - Filtre statut
     - HISTORIQUE DES TÂCHES (table):
       - ID
       - Date
       - Type
       - Description
       - Statut
       - Actions (Détails/Réessayer)
       - Pagination

# TYPES DE TÂCHES
- Mise à jour firmware
- Configuration WiFi
- Configuration DHCP
- Configuration NAT
- Redémarrage modem
- Diagnostic ligne
- Sauvegarde config
- Restauration config
- Maintenance planifiée

# STATUTS TÂCHE
- Planifié: Tâche programmée (bleu)
- En cours: Exécution en cours (jaune)
- Terminé: Succès (vert)
- Échec: Erreur (rouge)
- Annulé: Annulé par l'utilisateur (gris)

# FILTRES DISPONIBLES
- Tous
- En cours
- Terminé
- Échec
- Planifié

# INTERACTIONS
- ↻ Refresh → Recharge la liste
- Recherche → Filtre par ID ou description
- Filtre statut → Dropdown
- Détails → Modal détail tâche
- Réessayer → POST /tasks/{id}/retry (si échec)

# API ENDPOINTS
GET  /access/connections/{id}/tasks           - Liste tâches
GET  /access/connections/{id}/tasks/{taskId}  - Détail tâche
POST /access/connections/{id}/tasks/{taskId}/retry - Réessayer tâche

# DÉTAIL TÂCHE (modal)
- ID + Date création
- Type + Description complète
- Statut + Progression
- Logs d'exécution
- Message d'erreur (si échec)
- Durée d'exécution
