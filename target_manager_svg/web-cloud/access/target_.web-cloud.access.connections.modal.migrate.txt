# STRUCTURE MODAL

Modal de confirmation de migration OverTheBox.
Dimensions: 550x400px

1. HEADER:
   - Titre "Confirmer la migration"
   - Bouton fermer (×)

2. RÉCAPITULATIF:
   - Source et destination
   - Éléments migrés

3. WARNING:
   - Conséquences de la migration

4. CONFIRMATION:
   - Checkbox obligatoire

5. ACTIONS:
   - Annuler → Ferme modal
   - Migrer → POST migration

# ÉLÉMENTS MIGRÉS

- Connexions avec configuration
- Accès distants (remotes)
- Configuration réseau
- Configuration QoS
- IP publique

# IMPACTS

- Coupure de service: 5-10 minutes
- Réattribution des ports distants
- Nouvelle URL pour les remotes

# INTERACTIONS

- Checkbox → Obligatoire pour activer bouton Migrer
- Annuler → Ferme sans action
- Migrer → POST /overthebox/{id}/migration

# API ENDPOINTS

POST /overthebox/{id}/migration        - Lancer migration
GET  /overthebox/{id}/migration/status - Statut migration

# VALIDATION

- Checkbox de confirmation obligatoire
- Migration préparée au préalable
