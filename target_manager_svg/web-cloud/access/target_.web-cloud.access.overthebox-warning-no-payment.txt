# STRUCTURE PAGE

Page d'avertissement affichée quand aucun moyen de paiement n'est configuré.

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. CONTENT CENTRÉ:
   - Icône carte
   - Titre "Moyen de paiement requis"
   - Description
   - Liste moyens acceptés
   - Bouton d'action

# QUAND AFFICHER

Cette page s'affiche quand:
- L'utilisateur tente d'activer un OTB sans moyen de paiement
- La commande d'un nouvel OTB requiert un paiement
- Le moyen de paiement précédent a expiré

# MOYENS ACCEPTÉS

- Carte bancaire (Visa, Mastercard, Amex)
- Prélèvement SEPA
- PayPal

# INTERACTIONS

- Ajouter un paiement → /billing/payment-methods/add
- Retour → Page précédente

# API ENDPOINTS

GET /me/paymentMean                    - Liste moyens de paiement
POST /me/paymentMean                   - Ajouter moyen paiement
