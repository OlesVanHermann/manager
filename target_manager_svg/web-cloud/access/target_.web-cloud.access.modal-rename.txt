# STRUCTURE MODAL

Modal générique de renommage de service.
Dimensions: 450x280px

1. HEADER:
   - Titre "Renommer"
   - Bouton fermer (×)

2. FORMULAIRE:
   - Champ nouveau nom

3. ACTIONS:
   - Annuler → Ferme modal
   - Renommer → PUT displayName

# UTILISATION

Cette modal est utilisée pour renommer:
- Une ligne d'accès Internet
- Un OverTheBox
- Une connexion
- Un accès distant (remote)

# INTERACTIONS

- Annuler → Ferme sans sauvegarder
- Renommer → PUT /{type}/{id}/displayName

# API ENDPOINTS

PUT /access/{id}/displayName           - Renommer ligne
PUT /overthebox/{id}/displayName       - Renommer OTB
PUT /connection/{id}/displayName       - Renommer connexion
PUT /remote/{id}/displayName           - Renommer remote

# VALIDATION

- Nom: obligatoire, 1-100 caractères
- Caractères autorisés: lettres, chiffres, espaces, tirets
