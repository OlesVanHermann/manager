# Cahier des Charges : web-cloud/access - Modal Rename

## 1. VUE D'ENSEMBLE

**Type:** Modal formulaire simple
**Fonction:** Renommer un service (ligne, OTB, etc.)
**Contexte:** Personnalisation du nom d'affichage

## 2. DIMENSIONS

- Modal: 400x220px
- Background page: overlay rgba(0,0,0,0.5)

## 3. LAYOUT & STRUCTURE

### Header modal
- Background: #F9FAFB
- Titre: "Renommer"
- Bouton fermer: ×
- Height: 50px

## 4. DESCRIPTION

- Texte: "Modifiez le nom d'affichage de ce service."
- Font-size: 12px, text-muted

## 5. FORMULAIRE

### Champ Nouveau nom
- Label: "Nouveau nom *"
- Type: Input text
- Valeur exemple: "Ligne Bureau Principal"
- Largeur: 300px
- Height: 40px

## 6. COMPOSANTS UI

### Input text
- Height: 40px
- Border: #E5E7EB
- Border-radius: 4px
- Font-size: 13px

## 7. FOOTER MODAL

### Bouton Annuler
- Style: btn-outline (100x36px)

### Bouton Renommer
- Style: btn-primary (120x36px)
- Disabled si: champ vide

## 8. VALIDATION

- Nouveau nom: obligatoire, 1-50 caractères

## 9. APIS & ENDPOINTS

```
PUT /xdsl/{serviceName}/changeContact
Body: {
  displayName: "Ligne Bureau Principal"
}
→ Met à jour le nom d'affichage

# ou pour OverTheBox
PUT /overTheBox/{serviceName}
Body: {
  displayName: "OTB Siège Paris"
}
→ Met à jour le nom d'affichage
```
