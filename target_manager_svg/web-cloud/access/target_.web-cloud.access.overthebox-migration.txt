# STRUCTURE PAGE

Page wizard en 4 étapes pour migrer un OverTheBox.

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. CONTENT CENTRÉ (wizard):
   - Header: Titre + Description
   - Steps indicator (1-2-3-4)
   - Contenu étape courante
   - Info banner (ce qui sera migré)
   - Warning banner (coupure)
   - Actions (Annuler / Suivant)

# ÉTAPES DU WIZARD

**Étape 1: Sélection**
- Affiche OTB source (infos)
- Sélection OTB destination

**Étape 2: Vérification**
- Récapitulatif des éléments à migrer
- Comparaison source/destination
- Alertes de compatibilité

**Étape 3: Migration**
- Barre de progression
- Statut en temps réel
- Logs de migration

**Étape 4: Confirmation**
- Résultat de la migration
- Récapitulatif des actions
- Prochaines étapes

# CE QUI EST MIGRÉ
- Connexions liées (avec config)
- Accès distants (remotes)
- Configuration réseau
- Configuration QoS
- IP publique

# INTERACTIONS
- Sélectionner → Modal liste OTB disponibles
- Annuler → Retour liste
- Suivant → Étape suivante (si conditions remplies)
- Précédent → Étape précédente

# API ENDPOINTS
GET  /overthebox/{id}/migration/prepare     - Prépare migration
POST /overthebox/{id}/migration             - Lance migration
GET  /overthebox/{id}/migration/status      - Statut migration
POST /overthebox/{id}/migration/cancel      - Annule migration

# DURÉE ESTIMÉE
Migration typique: 5-10 minutes
Coupure de service pendant la migration.
