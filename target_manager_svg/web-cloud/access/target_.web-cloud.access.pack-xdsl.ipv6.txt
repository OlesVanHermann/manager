# Cahier des Charges : web-cloud/access/pack-xdsl - IPv6

## 1. VUE D'ENSEMBLE
- **Écran** : Configuration IPv6 d'un accès xDSL
- **Objectif** : Activer/désactiver IPv6 sur la ligne
- **Contexte** : Configuration réseau avancée

## 2. LAYOUT & STRUCTURE
- Carte principale : Configuration IPv6
- Modale de confirmation (affichée au toggle)

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsCheckbox** : Activer IPv6
- **OdsMessage** : Warning (attention redémarrage)
- **OdsBadge** : Statut actuel (Activé)
- **OdsModal** : Confirmation désactivation
- **OdsButton** : OK, Annuler

## 4. CONTENU CARTE
| Élément | Description |
|---------|-------------|
| Checkbox | "Activer IPv6" (coché) |
| Warning | Attention: redémarrage nécessaire |
| Statut actuel | Badge "Activé" (vert) |
| Préfixe IPv6 | 2001:41d0:xxxx::/56 |

## 5. WARNING BOX
- Titre : "⚠ Attention"
- Message ligne 1 : "L'activation de l'IPv6 nécessite un redémarrage de la box."
- Message ligne 2 : "Votre connexion sera momentanément interrompue."

## 6. MODALE CONFIRMATION
- Header : "Confirmation" + bouton fermer (×)
- Question : "Voulez-vous vraiment désactiver IPv6 ?"
- Info : "Cette action nécessitera un redémarrage de votre box."
- Boutons : OK (primaire), Annuler (secondaire)

## 7. ACTIONS & INTERACTIONS
- Toggle checkbox → Affiche modale confirmation
- OK modale → POST modification + redémarrage
- Annuler → Ferme modale, annule action

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/ipv6 : Statut IPv6
- PUT /xdsl/{serviceName}/ipv6 : Activer/désactiver
