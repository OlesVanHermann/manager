# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. SOUS-NAV - Connexions / OverTheBox (OverTheBox actif)
4. CONTENT (pleine largeur):
   - TOOLBAR:
     - Bouton refresh
     - Recherche
     - Bouton + Commander
   - MES OVERTHEBOX (table):
     - Nom + Offre
     - Service ID
     - IP publique
     - Connexions (count + types)
     - Bande passante (down/up)
     - Statut
     - Actions
     - Pagination

# OFFRES OVERTHEBOX
- OverTheBox IT: Usage bureautique
- OverTheBox Plus: Usage intensif
- OverTheBox Business: Entreprise

# STATUTS
- Actif: Fonctionnement normal
- Dégradé: Une ou plusieurs connexions en panne
- Hors ligne: Appareil déconnecté
- En configuration: Provisioning en cours

# TYPES CONNEXIONS
- FTTH: Fibre optique
- ADSL/VDSL: DSL
- SDSL: DSL symétrique
- 4G/5G: Mobile
- Satellite: Liaison satellite

# INTERACTIONS
- ↻ Refresh → Recharge la liste
- Recherche → Filtre par nom ou ID
- + Commander → Tunnel de commande
- Gérer → /overthebox/{id}/general
- ••• → Menu contextuel (Renommer, Supprimer)

# API ENDPOINTS
GET  /overthebox                     - Liste OverTheBox
GET  /overthebox/{id}                - Détail OverTheBox
POST /overthebox                     - Commander OverTheBox
DELETE /overthebox/{id}              - Supprimer OverTheBox
