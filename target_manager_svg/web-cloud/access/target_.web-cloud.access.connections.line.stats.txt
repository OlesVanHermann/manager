# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste connexions
4. RIGHT PANEL
   - Header: Nom connexion
   - NAV3: 7 tabs (Ligne actif)
   - NAV4: 4 sous-tabs (Statistiques actif)
   - Contenu:
     - Toolbar: Sélecteur période + Export CSV
     - Graphique DÉBIT (courbes down/up sur 7j)
     - UPTIME (%, durée disponible, indisponibilité)
     - CONNEXIONS (resync, déconnexions, durée)
     - HISTORIQUE événements (table)

# PÉRIODES DISPONIBLES
- Dernières 24h
- 7 derniers jours
- 30 derniers jours
- 3 derniers mois
- 1 an

# INTERACTIONS
- Clic sélecteur période → Change données
- Clic "Exporter CSV" → Télécharge fichier
- Hover courbe → Affiche tooltip valeur

# API ENDPOINTS
GET /access/connections/{id}/line/stats?period=7d - Statistiques
