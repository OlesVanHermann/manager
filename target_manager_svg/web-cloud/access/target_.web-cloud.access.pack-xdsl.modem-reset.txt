# Cahier des Charges : web-cloud/access/pack-xdsl - Modem Reset

## 1. VUE D'ENSEMBLE
- **Écran** : Réinitialisation du modem
- **Objectif** : Proposer différentes options de reset
- **Contexte** : Restauration configuration

## 2. LAYOUT & STRUCTURE
- Carte avec 3 options de reset
- Warning danger en bas

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsCard** : 3 cartes options (couleurs différentes)
- **OdsButton** : Action par option
- **OdsMessage** : Warning danger (rouge)

## 4. OPTIONS DE RESET

### Option 1 : Réinitialiser (gris)
- Description : "Redémarre le modem et applique la dernière configuration connue"
- Bouton : "Réinitialiser" (bleu)

### Option 2 : Réinit. config OVH (jaune)
- Description : "Restaure la configuration OVH d'origine (perte des personnalisations)"
- Bouton : "Réinit. config OVH" (orange)

### Option 3 : Reconfigurer VoIP (gris)
- Description : "Reconfigure la téléphonie sans toucher aux autres paramètres"
- Bouton : "Reconfigurer" (bleu)

## 5. WARNING DANGER
- Titre : "⚠ Attention - Opérations irréversibles"
- Point 1 : "La réinitialisation config OVH effacera toutes vos personnalisations (WiFi, ports, etc.)"
- Point 2 : "Votre connexion sera interrompue pendant plusieurs minutes"

## 6. APIS & ENDPOINTS
- POST /xdsl/{serviceName}/modem/reset : Reset simple
- POST /xdsl/{serviceName}/modem/resetConfig : Reset config OVH
- POST /xdsl/{serviceName}/modem/reconfigureVoip : Reconfig VoIP
