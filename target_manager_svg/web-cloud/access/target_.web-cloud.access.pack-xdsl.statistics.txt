# Cahier des Charges : web-cloud/access/pack-xdsl - Statistics

## 1. VUE D'ENSEMBLE
- **Écran** : Page statistiques d'un accès xDSL
- **Objectif** : Afficher graphiques détaillés des performances
- **Contexte** : Monitoring et diagnostic de la ligne

## 2. LAYOUT & STRUCTURE
- Carte blanche pleine page
- 4 sections graphiques empilées verticalement

## 3. COMPOSANTS UI
- **OdsCard** : Container principal
- **OdsSelect** : Sélecteur période par section
- Graphiques : Line charts et bar charts

## 4. SECTION SYNCHRONISATION
- **Titre** : Synchronisation
- **Période** : Dernières 24h (select)
- **Graphique** : Line chart
- **Courbes** : Download (bleu), Upload (vert)
- **Légende** : Download, Upload

## 5. SECTION TRAFIC
- **Titre** : Trafic
- **Période** : Dernière semaine (select)
- **Graphique** : Bar chart
- **Barres** : Download (bleu), Upload (vert)
- **Axe X** : Jours (Lun, Mar, Mer, Jeu, Ven)

## 6. SECTION PING (LATENCE)
- **Titre** : Ping (latence)
- **Période** : Dernières 24h (select)
- **Graphique** : Line chart
- **Courbe** : Latence (orange)
- **Valeur indicative** : ~15ms

## 7. SECTION ERREURS
- **Titre** : Erreurs (HEC/FEC/CRC)
- **Période** : Dernières 24h (select)
- **Graphique** : Line chart ou message vide
- **État vide** : "Aucune erreur détectée"

## 8. PÉRIODES DISPONIBLES
- Dernières 24h
- Dernière semaine
- Dernier mois
- Dernière année

## 9. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/statistics/synchronization : Sync
- GET /xdsl/{serviceName}/statistics/traffic : Trafic
- GET /xdsl/{serviceName}/statistics/ping : Latence
- GET /xdsl/{serviceName}/statistics/errors : Erreurs
