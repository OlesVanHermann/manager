# Cahier des Charges : web-cloud/access/pack-xdsl - Modem ACS Backend

## 1. VUE D'ENSEMBLE
- **Type** : Composant sélection backend ACS
- **URL** : /pack/{packId}/xdsl/{serviceName}/modem
- **Objectif** : Configurer serveur ACS du modem

## 2. STRUCTURE COMPOSANT
- Titre section
- Description
- Sélecteur backend
- Bouton validation

## 3. COMPOSANTS UI
- **OdsText** : Titre, description
- **OdsSelect** : Sélecteur backend
- **OdsButton** : Appliquer

## 4. TITRE
| Propriété | Valeur |
|-----------|--------|
| Texte | "Configuration ACS" |
| Style | Section heading |

## 5. DESCRIPTION
| Propriété | Valeur |
|-----------|--------|
| Texte | "Sélectionnez le serveur ACS (Auto Configuration Server) pour la gestion à distance de votre modem." |

## 6. SÉLECTEUR BACKEND
| Propriété | Valeur |
|-----------|--------|
| Label | "Serveur ACS" |
| Options | "OVH ACS", "ACS Externe", "Désactivé" |
| Défaut | "OVH ACS" |
| Largeur | 250px |

## 7. BOUTON
| Propriété | Valeur |
|-----------|--------|
| Label | "Appliquer" |
| Style | Primary |

## 8. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/modem/acsBackend - Backend actuel
- PUT /xdsl/{serviceName}/modem/acsBackend - Modifier backend
