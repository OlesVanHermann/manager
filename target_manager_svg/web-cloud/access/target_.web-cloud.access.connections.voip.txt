# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste connexions
4. RIGHT PANEL
   - Header: Nom connexion
   - NAV3: 7 tabs (VoIP actif)
   - NAV4: 2 sous-tabs (Lignes / EcoFax)
   - Contenu:
     - LIGNES TÉLÉPHONIQUES (table):
       - Numéro
       - Description
       - Type (SIP/EcoFax)
       - Statut (En ligne/Hors ligne)
       - Appels ce mois
       - Actions (Configurer/Historique)
       - Bouton + Commander ligne
     - CONSOMMATION CE MOIS:
       - Appels entrants
       - Appels sortants
       - Durée totale
       - Fax envoyés/reçus
       - Bouton Voir détails
     - Lien vers manager VoIP complet

# TYPES DE LIGNES
- SIP: Ligne téléphonique standard
- EcoFax: Fax numérique

# STATUTS LIGNE
- En ligne: Ligne active et opérationnelle
- Hors ligne: Ligne inactive ou déconnectée
- En cours: Configuration en cours

# INTERACTIONS
- + Commander ligne → Tunnel de commande
- Configurer → Modal configuration ligne
- Historique → Détail appels de la ligne
- Voir détails → Page statistiques détaillées
- Ouvrir manager VoIP → /web-cloud/voip

# API ENDPOINTS
GET /access/connections/{id}/voip/lines       - Liste lignes VoIP
GET /access/connections/{id}/voip/lines/{num} - Détail ligne
GET /access/connections/{id}/voip/stats       - Statistiques consommation
GET /access/connections/{id}/voip/history     - Historique appels

# LIEN CROISÉ
Le manager VoIP complet (/web-cloud/voip) offre:
- Gestion des files d'attente
- SVI (Serveur Vocal Interactif)
- Renvois d'appels
- Messagerie vocale
- Configuration avancée SIP
