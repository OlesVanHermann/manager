# STRUCTURE PAGE

1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Section (Accès actif)
3. LEFT PANEL - Liste OverTheBox
4. RIGHT PANEL
   - Header: Nom OTB
   - NAV3: 6 tabs (Tâches actif)
   - Contenu:
     - TOOLBAR:
       - Bouton refresh
       - Recherche
       - Filtre statut
     - HISTORIQUE DES TÂCHES (table):
       - ID
       - Date
       - Type
       - Description
       - Statut
       - Actions
       - Pagination

# TYPES DE TÂCHES OVERTHEBOX
- Mise à jour firmware
- Ajout connexion
- Suppression connexion
- Modification priorités
- Configuration remote
- Redémarrage
- Sauvegarde config
- Maintenance système

# STATUTS TÂCHE
- Planifié: Tâche programmée (bleu)
- En cours: Exécution en cours (jaune)
- Terminé: Succès (vert)
- Échec: Erreur (rouge)
- Annulé: Annulé par l'utilisateur (gris)

# FILTRES DISPONIBLES
- Tous
- En cours
- Terminé
- Échec
- Planifié

# INTERACTIONS
- ↻ Refresh → Recharge la liste
- Recherche → Filtre par ID ou description
- Filtre statut → Dropdown
- Détails → Modal détail tâche
- Réessayer → POST /tasks/{id}/retry (si échec)
- Annuler → DELETE /tasks/{id} (si planifié)

# API ENDPOINTS
GET  /overthebox/{id}/tasks           - Liste tâches
GET  /overthebox/{id}/tasks/{taskId}  - Détail tâche
POST /overthebox/{id}/tasks/{taskId}/retry - Réessayer
DELETE /overthebox/{id}/tasks/{taskId} - Annuler tâche planifiée
