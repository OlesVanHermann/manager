# STRUCTURE PAGE
1. NAV3: Détails | Accès distant | [Tâches] | Actions | Logs
2. Table des tâches avec pagination

# COLONNES TABLE

| Colonne | Description |
|---------|-------------|
| Nom | Type de tâche (refreshDevice, etc.) |
| ID | Identifiant unique de la tâche |
| Statut | État de la tâche |

# STATUTS

| Statut | Couleur | Description |
|--------|---------|-------------|
| Terminé | Vert | Tâche complétée avec succès |
| En cours | Bleu | Tâche en exécution |
| En attente | Orange | Tâche planifiée |
| Erreur | Rouge | Tâche échouée |

# TYPES DE TÂCHES

| Tâche | Description |
|-------|-------------|
| refreshDevice | Actualisation appareil |
| updateConfiguration | MAJ configuration |
| restartService | Redémarrage service |
| backupConfiguration | Sauvegarde config |
| syncInterfaces | Sync interfaces réseau |

# DIFFÉRENCE TÂCHES vs ACTIONS
- **Tâches**: Opérations automatiques/planifiées
- **Actions**: Opérations déclenchées manuellement

# INTERACTIONS
- Pagination → Naviguer dans l'historique

# API ENDPOINTS
GET  /overthebox/{id}/tasks           - Liste des tâches
GET  /overthebox/{id}/tasks/{tid}     - Détail tâche
