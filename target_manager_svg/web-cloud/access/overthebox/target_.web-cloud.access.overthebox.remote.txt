# STRUCTURE PAGE
1. NAV3: Détails | [Accès distant] | Tâches | Actions | Logs
2. Table des accès distants
3. Détails expandables par accès
4. Bannière info

# COLONNES TABLE

| Colonne | Description |
|---------|-------------|
| ID | Identifiant de l'accès distant |
| Statut | Actif / En attente |
| Autorisé par | Email de l'autorisateur |
| Expiration | Date de fin de validité |
| Actions | Autoriser/Refuser/Révoquer |

# STATUTS

| Statut | Couleur | Description |
|--------|---------|-------------|
| Actif | Vert | Accès autorisé et utilisable |
| En attente | Orange | En attente d'autorisation |

# DÉTAILS EXPANDABLES

| Champ | Description |
|-------|-------------|
| Port exposé | Port accessible (22 SSH, etc.) |
| IP source | Adresse autorisée |
| Connexion | Commande SSH complète |
| Créé le | Date de création |

# INTERACTIONS
- Clic ▶ → Expand/collapse détails
- Clic Autoriser → Accepter l'accès
- Clic Refuser → Refuser l'accès
- Clic Révoquer → Supprimer l'accès

# API ENDPOINTS
GET    /overthebox/{id}/remotes           - Liste accès distants
POST   /overthebox/{id}/remotes/{rid}/approve  - Autoriser
POST   /overthebox/{id}/remotes/{rid}/deny     - Refuser
DELETE /overthebox/{id}/remotes/{rid}          - Révoquer
