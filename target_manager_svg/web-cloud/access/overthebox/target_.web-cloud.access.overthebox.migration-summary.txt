# STRUCTURE MODAL
1. Titre "Résumé de la commande"
2. Description transition
3. Table récapitulatif (offre + matériel)
4. Total et explications prix
5. Checkbox conditions contractuelles
6. Liens CGV
7. Boutons Annuler/Confirmer

# TABLE RÉCAPITULATIF

| Colonne | Description |
|---------|-------------|
| Désignation | Nom produit |
| Prix unitaire | Prix HT |
| Quantité | Nombre |
| Prix/mois | Total ligne |

# PRIX
- Abonnement: Mensuel récurrent
- Matériel: Unique (première facture)
- Première mensualité = Abo + Matériel

# CONDITIONS
- Checkbox obligatoire pour valider
- Liens vers CGV et conditions particulières

# INTERACTIONS
- Toggle checkbox → Active/désactive bouton
- Clic Annuler → Retour étape précédente
- Clic Confirmer → POST migration + confirmation

# TUNNEL MIGRATION
1. Choix offre
2. Contact livraison
3. **Résumé + validation (cette page)**

# API ENDPOINTS
POST /overthebox/{id}/migration/confirm   - Confirmer migration
Body: { offerId, hardwareId, contactId, acceptTerms }
