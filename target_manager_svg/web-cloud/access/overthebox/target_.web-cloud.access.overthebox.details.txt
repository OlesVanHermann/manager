# STRUCTURE PAGE
1. NAV3: [Détails] | Accès distant | Tâches | Actions | Logs
2. Bannière info mise à jour (si disponible)
3. BLOC INFORMATIONS GÉNÉRALES
4. BLOC INFORMATIONS APPAREIL
5. TABLE INTERFACES RÉSEAU

# INFORMATIONS GÉNÉRALES
- État de l'appareil (Activé/Désactivé)
- Date de facturation
- Offre souscrite
- Boutons: Résilier, Changer d'offre

# INFORMATIONS APPAREIL
- Identifiant service
- Identifiant appareil
- Version système
- IP publique
- Toggle IPv6

# INTERFACES RÉSEAU

| Colonne | Description |
|---------|-------------|
| Interface | Nom (if0, if1...) |
| Type | WAN DHCP, PPPoE, 4G... |
| IP | Adresse IP |
| Passerelle | Gateway |
| Débit | Download/Upload |
| Statut | Actif/Inactif |

# INTERACTIONS
- Toggle IPv6 → PUT config
- Clic Résilier → Modal confirmation
- Clic Changer d'offre → Tunnel migration

# API ENDPOINTS
GET  /overthebox/{id}                    - Infos générales
GET  /overthebox/{id}/device             - Infos appareil
GET  /overthebox/{id}/interfaces         - Liste interfaces
PUT  /overthebox/{id}/device/ipv6        - Toggle IPv6
POST /overthebox/{id}/terminate          - Résilier
GET  /overthebox/{id}/availableOffers    - Offres migration
