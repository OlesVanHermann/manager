1. LEFT PANEL - OverTheBox + NAV3 (Connexions actif)
2. RIGHT PANEL:
   - TITRE "Gestion des connexions"
   - BOUTON "Ajouter"
   - CONNEXION 1: FTTH Primaire (vert highlight)
   - CONNEXION 2: VDSL (standard)
   - CONNEXION 3: 4G Backup (orange)
