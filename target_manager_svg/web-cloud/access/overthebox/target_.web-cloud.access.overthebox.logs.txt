| Niveau | Couleur | Description |
|--------|---------|-------------|
| Info | Bleu (#60A5FA) | Information normale |
| Warning | Orange (#FBBF24) | Avertissement |
| Erreur | Rouge (#F87171) | Erreur critique |
| Debug | Gris (#6B7280) | Debug/trace |
