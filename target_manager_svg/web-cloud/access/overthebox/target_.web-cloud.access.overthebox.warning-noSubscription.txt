# STRUCTURE BANNIÈRE
1. Icône info (i)
2. Titre "Équipement détecté"
3. Message explicatif

# STYLE
- Fond bleu clair (#EFF6FF)
- Bordure bleue (#0050D7)
- Texte bleu foncé (#1E40AF)

# CAS D'AFFICHAGE
Bannière affichée quand:
- Un appareil OTB est détecté sur le réseau
- L'appareil n'est associé à aucun abonnement actif

# MESSAGE
Invite l'utilisateur à souscrire un abonnement pour activer l'appareil.

# PROCHAINES ÉTAPES
1. L'utilisateur voit cette bannière
2. Il peut cliquer sur "Commander" pour souscrire
3. Une fois l'abonnement actif, l'appareil est associé

# API ENDPOINTS
GET  /overthebox/detected         - Appareils détectés
POST /overthebox/order            - Commander abonnement
