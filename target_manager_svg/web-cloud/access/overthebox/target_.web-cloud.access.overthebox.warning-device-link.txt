# STRUCTURE PAGE
1. Bannière info "Appareil détecté"
2. Description + bouton association
3. Liste des services à configurer
4. Warning si plusieurs appareils

# BANNIÈRE INFO
- Fond bleu clair (#EFF6FF)
- Icône ℹ
- Message explicatif
- Bouton "Associer l'appareil"

# SERVICES À CONFIGURER
Liste des services OTB nécessitant une association avec un appareil physique.
Liens cliquables vers chaque service.

# WARNING MULTIPLE
Si plusieurs appareils OTB détectés sur le réseau:
- Bannière orange
- Message demandant de sélectionner le bon appareil

# CAS D'AFFICHAGE
Page affichée quand:
- Un appareil OTB est détecté sur le réseau local
- Le service n'est pas encore associé à un appareil

# INTERACTIONS
- Clic "Associer" → POST linkDevice
- Clic service → Détails du service

# API ENDPOINTS
GET  /overthebox/detected         - Appareils détectés
GET  /overthebox/unlinked         - Services sans appareil
POST /overthebox/{id}/linkDevice  - Associer appareil
