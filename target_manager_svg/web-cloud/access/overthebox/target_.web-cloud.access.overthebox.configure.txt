# STRUCTURE MODAL
1. Titre "Configuration OverTheBox"
2. Section rattachement (OTB redémarrée)
3. Section nouvelle souscription (nouveau matériel)

# SECTION RATTACHEMENT
Apparaît quand un appareil a changé d'identifiant (reset, remplacement).
- Liste des services sans appareil rattaché
- Bouton "Rattacher au service"

# SECTION NOUVELLE SOUSCRIPTION
Pour les nouveaux équipements sans service existant.
- Description de l'action
- Bouton "Souscrire à un service"
- Lien vers guide d'installation

# CAS D'USAGE

| Situation | Section |
|-----------|---------|
| OTB redémarrée/réinitialisée | Rattachement |
| Nouvel achat matériel | Nouvelle souscription |
| Remplacement matériel | Rattachement |

# INTERACTIONS
- Clic "Rattacher" → POST link device
- Clic "Souscrire" → Tunnel commande
- Clic ✕ → Fermer modal
- Clic guide → Nouvelle fenêtre doc

# API ENDPOINTS
GET  /overthebox/unlinked              - Services sans appareil
POST /overthebox/{id}/linkDevice       - Rattacher appareil
GET  /overthebox/order                 - Démarrer commande
