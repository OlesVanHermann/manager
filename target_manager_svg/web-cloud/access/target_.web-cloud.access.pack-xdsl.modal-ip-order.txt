# Cahier des Charges : web-cloud/access/pack-xdsl - Modal IP Order

## 1. VUE D'ENSEMBLE
- **Écran** : Modale commande d'IP supplémentaires
- **Objectif** : Commander un bloc d'adresses IP
- **Contexte** : Besoins réseau avancés

## 2. LAYOUT & STRUCTURE
- Modale centrée avec overlay sombre
- Header avec bouton fermer
- Corps avec infos commande
- Footer avec boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Container modale
- **OdsText** : Infos opération, service, prix
- **OdsCheckbox** : 2 cases à cocher
- **OdsLink** : "voir le contrat"
- **OdsButton** : "Commander" (désactivé si checkbox manquant), "Annuler"

## 4. HEADER MODALE
- Titre : "Commander des IP supplémentaires"
- Bouton fermer : × (coin droit)

## 5. INFORMATIONS COMMANDE
- Opération : "Commander 8 adresses IP"
- Service : "xdsl-abc123-1"

## 6. TARIFICATION
- Encadré gris avec :
- Prix : "5,00 EUR HT/mois"
- Info : "Activation sous 2h - Engagement minimum 1 mois"

## 7. CHECKBOXES OBLIGATOIRES
- ☑ "J'accepte l'exécution immédiate du service"
- ☐ "J'accepte les conditions générales (voir le contrat)"

## 8. BOUTONS FOOTER
- "Commander" (grisé si cases non cochées)
- "Annuler" (secondaire)

## 9. APIS & ENDPOINTS
- GET /xdsl/{serviceName}/ips/options : Options IP disponibles
- POST /xdsl/{serviceName}/ips/order : Commander IP
