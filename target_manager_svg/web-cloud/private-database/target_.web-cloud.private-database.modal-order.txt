# Cahier des Charges : web-cloud/private-database - Modal Commander CloudDB

## 1. VUE D'ENSEMBLE
- **Page** : Formulaire de commande Web Cloud Databases
- **Type** : Page wizard pleine largeur
- **Objectif** : Commander une nouvelle base de données Cloud

## 2. STRUCTURE PAGE
- Header OVH avec titre
- Sidebar avec menu "Web Cloud Databases" actif
- 3 étapes : Solution, Durée, Paiement
- Zone récapitulatif + boutons

## 3. COMPOSANTS UI
- **OdsCard** : Cartes sélectionnables RAM
- **OdsSelect** : Version serveur
- **OdsText** : Titre sections
- **OdsButton** : Annuler, Payer

## 4. TITRE
"Commander une base de données Web Cloud Databases"

## 5. DESCRIPTION
"L'offre Web Cloud Databases nécessite un service ayant accès au réseau internet public."

## 6. ÉTAPE 1 : CHOISISSEZ VOTRE SOLUTION
### Version du serveur
- Type: Select
- Exemple: "MariaDB 10.11"

### Options RAM (cartes)
| RAM | Prix HT/mois |
|-----|--------------|
| 512 Mo | 5,99 € |
| 1 Go | 9,99 € (sélectionné) |
| 2 Go | 19,99 € |
| 4 Go | 39,99 € |

Carte sélectionnée : Fond bleu (#E8F0FE), bordure bleue (#0050D7), coche ✓

## 7. ÉTAPE 2 : CHOISISSEZ LA DURÉE
| Durée | Style |
|-------|-------|
| 1 mois | Normal |
| 12 mois | Sélectionné (bleu) |
| 24 mois | Normal |

## 8. ÉTAPE 3 : PAIEMENT
### Récapitulatif
| Libellé | Valeur |
|---------|--------|
| Prix HT | 9,99 € / mois |
| Prix TTC | 11,99 € / mois |
| Durée | 12 mois |

## 9. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la page |
| Payer | Primaire bleu | Lance le paiement |

## 10. APIS & ENDPOINTS
- GET /order/cart : Créer panier
- POST /order/cart/{cartId}/privateDatabase : Ajouter CloudDB au panier
  - version: string (ex: "mariadb_10.11")
  - ram: string (ex: "ram-512", "ram-1024")
  - duration: string ("P1M", "P12M", "P24M")
- POST /order/cart/{cartId}/checkout : Valider commande
