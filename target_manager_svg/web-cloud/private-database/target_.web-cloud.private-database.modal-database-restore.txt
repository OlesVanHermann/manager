# Cahier des Charges : web-cloud/private-database - Modal Restaurer sauvegarde

## 1. VUE D'ENSEMBLE
- **Modal** : Restauration d'une sauvegarde
- **Type** : Modal formulaire avec warning
- **Objectif** : Restaurer une base depuis un dump existant

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Nom de la base (lecture seule)
- Select sauvegarde à restaurer
- Bandeau warning irréversibilité
- Boutons Annuler / Restaurer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x420)
- **OdsInput** : Base de données (disabled)
- **OdsSelect** : Sauvegarde à restaurer
- **OdsMessage** : Warning orange
- **OdsButton** : Annuler, Restaurer

## 4. TITRE
"Restaurer une sauvegarde"

## 5. CHAMPS
### Base de données
- Type: Input disabled
- Valeur: mydb_production
- Fond: Gris (#F3F4F6)

### Sauvegarde à restaurer
- Type: Select obligatoire (*)
- Format options: "YYYY-MM-DD HH:MM - Type de sauvegarde"
- Exemple: "2025-12-27 03:00 - Sauvegarde quotidienne"

## 6. BANDEAU WARNING
Fond orange (#FEF3C7) :
- "⚠ Attention : cette action va remplacer toutes les données actuelles de la base par celles de la sauvegarde sélectionnée."
- "Cette opération est irréversible."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Restaurer | Primaire bleu | Lance la restauration |

## 8. APIS & ENDPOINTS
- POST /hosting/privateDatabase/{serviceName}/database/{databaseName}/restore
  - dumpId: string (ID du dump à restaurer)
