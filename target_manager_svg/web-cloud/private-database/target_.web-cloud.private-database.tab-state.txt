# Cahier des Charges : web-cloud/private-database - Tab Informations générales

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Informations générales
- **Type** : Page dashboard avec tuiles
- **Objectif** : Afficher les informations essentielles du service

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation Web Cloud
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- 4 tuiles d'information

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsTabs** : Onglets (Informations générales actif)
- **OdsCard** : 4 tuiles
- **OdsBadge** : État, Saturation CPU, RAM
- **OdsActionMenu** : Menus ⋮

## 4. TUILE 1 : INFORMATIONS GÉNÉRALES
| Champ | Valeur | Action ⋮ |
|-------|--------|----------|
| État du service | Badge "Démarré" (vert) | Arrêter/Redémarrer |
| Version | MariaDB 10.11 | Changer la version |
| Saturation CPU | Badge "0%" (vert) + description | - |
| RAM | Badge "512 Mo" (vert) | Modifier RAM |

### Descriptions
- Saturation CPU: "Temps CPU passé en saturation sur les dernières 24h"
