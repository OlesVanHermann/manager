# Cahier des Charges : web-cloud/private-database - Utilisateurs

## 1. VUE D'ENSEMBLE
- **Écran** : Liste des utilisateurs et droits
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les utilisateurs et leurs permissions

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Titre service avec onglets
- Bouton "Créer un utilisateur"
- Tableau des utilisateurs
- Menu actions ouvert
- Pagination

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsTabs** : Onglets (Utilisateurs et droits actif)
- **OdsButton** : Créer un utilisateur
- **OdsTable** : Liste des utilisateurs
- **OdsBadge** : Type de droits
- **OdsActionMenu** : Menu ⋮ ouvert

## 4. ONGLETS
| Onglet | État |
|--------|------|
| Informations générales | - |
| **Utilisateurs et droits** | Actif (bleu) |
| Bases de données | - |
| IPs autorisées | - |
| Logs | - |
| Configuration | - |

## 5. BOUTON D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Créer un utilisateur | Primaire bleu | Ouvre modal création |

## 6. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Utilisateur | Nom utilisateur | admin_user |
| Base de données | Base associée | production_db |
| Droits | Badge coloré | Admin, Lecture/Écriture, Lecture seule |
| Date de création | Format DD/MM/YYYY | 15/03/2024 |
| Actions | Menu ⋮ | Voir ci-dessous |

| Droit | Couleur | Code couleur |

## 8. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Changer le mot de passe | Nouveau password |
| Modifier les droits | Ajuster permissions |
| --- | Séparateur |
| Accéder à phpMyAdmin | Ouvre phpMyAdmin |
| --- | Séparateur |
| Supprimer | Rouge - Suppression |

## 9. PAGINATION
"Affichage de 1-3 sur 3 utilisateurs"

## 10. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/user : Lister utilisateurs
- POST /hosting/privateDatabase/{serviceName}/user : Créer utilisateur
- PUT /hosting/privateDatabase/{serviceName}/user/{userName} : Modifier
- DELETE /hosting/privateDatabase/{serviceName}/user/{userName} : Supprimer
- POST /hosting/privateDatabase/{serviceName}/user/{userName}/changePassword : Changer MDP
- GET /hosting/privateDatabase/{serviceName}/user/{userName}/grant : Voir droits
- POST /hosting/privateDatabase/{serviceName}/user/{userName}/grant : Modifier droits
