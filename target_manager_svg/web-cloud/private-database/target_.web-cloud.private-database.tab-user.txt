# Cahier des Charges : web-cloud/private-database - Tab Utilisateurs

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Utilisateurs et droits
- **Type** : Page liste avec tableau
- **Objectif** : Gérer les utilisateurs et leurs permissions

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- Bouton "Ajouter un utilisateur"
- Tableau des utilisateurs
- Menu actions ouvert

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsTabs** : Onglets (Utilisateurs et droits actif)
- **OdsButton** : Ajouter un utilisateur (outline)
- **OdsTable** : Liste des utilisateurs
- **OdsBadge** : Nom base + type droit
- **OdsActionMenu** : Menu ⋮ ouvert

## 4. BOUTON D'ACTION
| Bouton | Style | Action |
|--------|-------|--------|
| Ajouter un utilisateur | Outline gris | Ouvre modal création |

## 5. COLONNES DU TABLEAU
| Colonne | Description | Exemple |
|---------|-------------|---------|
| Nom d'utilisateur | Login | admin_user |
| Date de création | Format DD MMM YYYY | 1 déc. 2024 |
| Droits sur les bases | Badge + type droit | wordpress_prod - Administrateur |
| Actions | Menu ⋮ | Voir ci-dessous |

## 6. BADGES DROITS
| Badge | Couleur | Type affiché |
|-------|---------|--------------|
| wordpress_prod | Bleu (#DBEAFE) | Administrateur |
| wordpress_prod | Orange (#FEF3C7) | Lecture |
| prestashop_db | Vert (#DEF7EC) | Lecture/Écriture |

## 7. MENU ACTIONS (⋮)
| Action | Description |
|--------|-------------|
| Changer le mot de passe | Nouveau password |
| Gérer les droits | Modifier permissions |
| --- | Séparateur |
| Supprimer l'utilisateur | Rouge - Suppression |

## 8. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/user : Lister utilisateurs
- POST /hosting/privateDatabase/{serviceName}/user : Créer
- DELETE /hosting/privateDatabase/{serviceName}/user/{userName} : Supprimer
- POST /hosting/privateDatabase/{serviceName}/user/{userName}/changePassword : Changer MDP
- GET /hosting/privateDatabase/{serviceName}/user/{userName}/grant/{databaseName} : Voir droits
- POST /hosting/privateDatabase/{serviceName}/user/{userName}/grant/{databaseName} : Modifier droits
