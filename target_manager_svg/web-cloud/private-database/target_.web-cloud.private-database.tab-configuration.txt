# Cahier des Charges : web-cloud/private-database - Tab Configuration

## 1. VUE D'ENSEMBLE
- **Écran** : Onglet Configuration MariaDB
- **Type** : Page formulaire avec variables
- **Objectif** : Personnaliser les paramètres MySQL/MariaDB

## 2. STRUCTURE DE LA PAGE
- Header OVHcloud avec navigation
- Sidebar Web Cloud Databases
- Breadcrumb
- Titre service avec onglets
- Bandeau warning bleu
- Lien documentation
- Formulaire variables (2 colonnes)
- Boutons Appliquer / Annuler

## 3. COMPOSANTS UI
- **OdsHeader** : Navigation principale
- **OdsSidebar** : Menu navigation
- **OdsBreadcrumb** : Fil d'Ariane
- **OdsTabs** : Onglets (Configuration actif)
- **OdsMessage** : Warning bleu
- **OdsLink** : Documentation
- **OdsInput** : Champs numériques
- **OdsSelect** : Choix tmpdir
- **OdsToggle** : ON/OFF pour event_scheduler, autocommit
- **OdsButton** : Appliquer, Annuler

## 4. BANDEAU WARNING
Fond bleu (#EFF6FF) :
"⚠ Toute modification entraîne un redémarrage du serveur."

## 5. VARIABLES COLONNE GAUCHE
| Variable | Type | Défaut | Description |
|----------|------|--------|-------------|
| max_connections | Input number | 100 | Connexions simultanées autorisées |
| max_user_connections | Input number | 25 | Connexions par utilisateur |
| max_allowed_packet | Input number + Mo | 128 | Taille max paquets (Mo) |
| innodb_buffer_pool_size | Input number + Mo | 256 | Mémoire tampon InnoDB (max 50% RAM) |
| tmpdir | Select | /dev/shm | Répertoire fichiers temporaires |

## 6. VARIABLES COLONNE DROITE
| Variable | Type | Défaut | Description |
|----------|------|--------|-------------|
| event_scheduler | Toggle ON/OFF | ON | Requêtes programmées |
| autocommit | Toggle ON/OFF | ON | Transactions automatiques |

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Appliquer | Primaire bleu | Sauvegarde et redémarre |
| Annuler | Secondaire | Annule modifications |

## 8. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/config : Récupérer config
- PUT /hosting/privateDatabase/{serviceName}/config : Modifier config
