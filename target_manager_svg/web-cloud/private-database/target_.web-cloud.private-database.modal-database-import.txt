# Cahier des Charges : web-cloud/private-database - Modal Importer fichier

## 1. VUE D'ENSEMBLE
- **Modal** : Import d'un fichier SQL
- **Type** : Modal formulaire avec upload
- **Objectif** : Importer des données depuis un fichier SQL

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- Select base de données cible
- Zone upload (drag & drop)
- Info formats acceptés
- Options (checkboxes)
- Boutons Annuler / Importer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x540)
- **OdsSelect** : Base de données cible
- **OdsUpload** : Zone drag & drop
- **OdsButton** : Parcourir...
- **OdsText** : Info formats
- **OdsCheckbox** : Options
- **OdsButton** : Annuler, Importer

## 4. TITRE
"Importer un fichier"

## 5. CHAMPS
### Base de données cible
- Type: Select obligatoire (*)
- Valeur: mydb_production

## 6. ZONE UPLOAD
- Style: Zone pointillée (#D1D5DB dashed)
- Texte: "Glisser-déposer ici"
- Ou: "Parcourir..." (bouton outline bleu)

## 7. FORMATS ACCEPTÉS
- "Formats acceptés : .sql, .sql.gz, .sql.bz2"
- "Taille maximale : 100 Mo"

## 8. OPTIONS
| Checkbox | État | Description |
|----------|------|-------------|
| ☐ | Non coché | Vider la base avant import |
| ☐ | Non coché | Envoyer email de confirmation |

## 9. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Importer | Primaire bleu | Lance l'import |

## 10. APIS & ENDPOINTS
- POST /hosting/privateDatabase/{serviceName}/database/{databaseName}/import
  - file: File (.sql, .sql.gz, .sql.bz2)
  - flushDatabase: boolean
  - sendEmail: boolean
