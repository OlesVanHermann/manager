# Cahier des Charges : web-cloud/private-database - Modal Gérer les droits utilisateur

## 1. VUE D'ENSEMBLE
- **Modal** : Gestion des privilèges SQL d'un utilisateur
- **Type** : Modal formulaire avec checkboxes
- **Objectif** : Attribuer les droits d'accès sur une base de données

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (✕)
- Info utilisateur et serveur
- Select base de données
- Bandeau info
- Grille de checkboxes privilèges
- Actions rapides (liens)
- Boutons Annuler / Enregistrer

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (700x780)
- **OdsCard** : Info utilisateur (fond gris)
- **OdsSelect** : Base de données
- **OdsMessage** : Info bleu
- **OdsCheckbox** : Privilèges (grille 4 colonnes)
- **OdsLink** : Actions rapides
- **OdsButton** : Annuler, Enregistrer

## 4. TITRE
"Gérer les droits de l'utilisateur"

## 5. INFO UTILISATEUR
Fond gris (#F9FAFB) :
- Utilisateur: "admin_user"
- Sur: "private-sql-abc123.database.cloud.ovh.net"

## 6. BASE DE DONNÉES
- Type: Select obligatoire (*)
- Exemple: "production_db"

## 7. BANDEAU INFO
Fond bleu (#E8F0FE) :
"ℹ️ Sélectionnez les privilèges à accorder pour cette base de données."

## 8. PRIVILÈGES DE DONNÉES
Grille 4 colonnes :
| Privilège | État |
|-----------|------|
| SELECT | ☑ Coché |
| INSERT | ☑ Coché |
| UPDATE | ☑ Coché |
| DELETE | ☑ Coché |
| CREATE | ☐ Non coché |
| DROP | ☐ Non coché |
| ALTER | ☐ Non coché |
| INDEX | ☐ Non coché |

## 9. PRIVILÈGES AVANCÉS
Grille 2 colonnes :
| Privilège | État |
|-----------|------|
| CREATE TEMPORARY TABLES | ☐ Non coché |
| LOCK TABLES | ☐ Non coché |
| REFERENCES | ☐ Non coché |
| TRIGGER | ☐ Non coché |
| CREATE VIEW | ☐ Non coché |
| SHOW VIEW | ☐ Non coché |
| CREATE ROUTINE | ☐ Non coché |
| ALTER ROUTINE | ☐ Non coché |
| EXECUTE | ☐ Non coché |
| EVENT | ☐ Non coché |

## 10. ACTIONS RAPIDES
Liens bleus (#0050D7) :
- "Tout sélectionner" | "Lecture seule" | "Tout désélectionner"

## 11. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Enregistrer | Primaire bleu | Sauvegarde les privilèges |

## 12. APIS & ENDPOINTS
- GET /hosting/privateDatabase/{serviceName}/user/{userName}/grant/{databaseName} : Obtenir droits actuels
- PUT /hosting/privateDatabase/{serviceName}/user/{userName}/grant/{databaseName}
  - grant: string (ex: "admin", "all", "rw", "ro", "none" ou custom)
  - Ou liste des privilèges individuels
