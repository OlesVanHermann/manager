# Cahier des Charges : web-cloud/private-database - Modal Modifier la RAM

## 1. VUE D'ENSEMBLE
- **Modal** : Modification de la RAM du serveur
- **Type** : Modal formulaire avec radio buttons
- **Objectif** : Augmenter ou réduire la RAM allouée

## 2. STRUCTURE MODAL
- Header avec titre et bouton fermer (×)
- RAM actuelle (lecture seule)
- Radio buttons nouvelle RAM
- Bandeau info facturation
- Boutons Annuler / Modifier

## 3. COMPOSANTS UI
- **OdsModal** : Conteneur (600x480)
- **OdsInput** : RAM actuelle (disabled)
- **OdsRadio** : Options de RAM
- **OdsMessage** : Info bleu
- **OdsButton** : Annuler, Modifier

## 4. TITRE
"Modifier la RAM"

## 5. CHAMPS
### RAM actuelle
- Type: Input disabled
- Valeur: "512 Mo"
- Fond: Gris (#F3F4F6)

### Nouvelle RAM *
| Option | Prix | État |
|--------|------|------|
| 512 Mo | Inclus | ○ Non sélectionné |
| 1 Go | 9.99 EUR HT/mois | ● Sélectionné |
| 2 Go | 19.99 EUR HT/mois | ○ Non sélectionné |
| 4 Go | 39.99 EUR HT/mois | ○ Non sélectionné |

## 6. BANDEAU INFO
Fond bleu (#DBEAFE) :
- "ℹ Le changement de RAM nécessite un redémarrage du serveur."
- "La facturation sera ajustée au prorata."

## 7. BOUTONS
| Bouton | Style | Action |
|--------|-------|--------|
| Annuler | Outline | Ferme la modal |
| Modifier | Primaire bleu | Applique le changement |

## 8. APIS & ENDPOINTS
- PUT /hosting/privateDatabase/{serviceName}/ram
  - ram: string ("ram-512", "ram-1024", "ram-2048", "ram-4096")
