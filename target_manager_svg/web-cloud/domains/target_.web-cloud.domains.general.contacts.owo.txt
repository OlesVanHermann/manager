# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Domaines et DNS actif
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: {domainName}
   - NAV3: 10 tabs (Contacts actif)
   - Titre: Obfuscation des e-mails
   - Description fonctionnalité
   - Table contacts
     - Colonnes: Contact, E-mail, Obfuscation (toggle), Action
     - Toggle ON/OFF pour chaque contact
     - Bouton "Régénérer" par ligne
   - Bouton "Enregistrer les modifications"

# ÉTATS TOGGLE
- ON (bleu #0050D7): E-mail obfusqué dans WHOIS
- OFF (gris #D1D5DB): E-mail affiché en clair

# API ENDPOINTS
GET /domain/{domain}/owo - État obfuscation
POST /domain/{domain}/owo - Modifier obfuscation
POST /domain/{domain}/owo/{contact}/regenerate - Régénérer adresse
