# STRUCTURE MODAL
1. HEADER - Titre "Activer DNSSEC" (fond OVH bleu)
2. CONTENT
   - Bannière info (explication DNSSEC)
   - Domaine concerné
   - Étapes de l'activation
   - Avantages
   - Note gestion automatique
3. FOOTER - Boutons Annuler / Activer DNSSEC

# MODAL TYPE
- Type: Confirmation activation
- Dimensions: 600x560px
- Header: Fond OVH bleu (#000E9C)

# PROCESSUS D'ACTIVATION
1. Génération des clés KSK (Key Signing Key) et ZSK (Zone Signing Key)
2. Signature de tous les enregistrements de la zone
3. Publication des enregistrements DS au registre
4. Activation de la chaîne de confiance

# DÉLAI
- Génération: Quelques minutes
- Propagation complète: Jusqu'à 24 heures

# PRÉREQUIS
- Domaine avec DNS hébergés chez OVH
- Zone DNS active
- Extension supportant DNSSEC

# AVANTAGES DNSSEC
- Protection contre DNS spoofing
- Protection contre cache poisoning
- Authentification des réponses DNS
- Prérequis pour DANE (TLSA records)

# GESTION AUTOMATIQUE
OVH gère automatiquement:
- Renouvellement des clés
- Rotation ZSK (tous les 3 mois)
- Rotation KSK (annuelle)

# API ENDPOINTS
POST /domain/{domain}/activateDnssec - Activer DNSSEC
GET /domain/{domain}/dsRecord - Vérifier état DNSSEC
