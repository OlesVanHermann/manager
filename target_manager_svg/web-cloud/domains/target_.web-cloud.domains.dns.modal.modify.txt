# STRUCTURE PAGE
1. Page de fond: Zone DNS avec table des enregistrements
2. Overlay semi-transparent
3. Modal "Modifier l'entrée DNS"
   - Header: Titre + bouton fermer (×)
   - Formulaire:
     - Sous-domaine (input + hint .domain.tld)
     - Type d'enregistrement (select dropdown)
     - TTL (input + hint secondes)
     - Cible (input selon type)
   - Bannière info propagation DNS
   - Boutons: Annuler, Enregistrer

# INTERACTIONS
- Clic "×" ou "Annuler" → Ferme modal
- Clic "Enregistrer" → PUT API, ferme modal, refresh table
- Sélection type → Adapte champs cible selon type
- Focus input → Border bleue

# TYPES D'ENREGISTREMENT
- A: Cible = adresse IPv4
- AAAA: Cible = adresse IPv6
- CNAME: Cible = FQDN avec point final
- MX: Priority + Cible
- TXT: Cible = texte libre
- NS: Cible = FQDN serveur DNS
- SRV: Priority, Weight, Port, Target

# API ENDPOINTS
GET /domain/zone/{zone}/record/{id} - Détails record
PUT /domain/zone/{zone}/record/{id} - Modifier record
POST /domain/zone/{zone}/refresh - Rafraîchir zone après modif
