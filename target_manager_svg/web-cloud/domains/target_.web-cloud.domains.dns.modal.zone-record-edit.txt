# STRUCTURE MODAL
1. HEADER - Titre "Modifier l'enregistrement DNS" (fond gris)
2. CONTENT
   - Type d'enregistrement (lecture seule avec badge)
   - Sous-domaine (lecture seule)
   - TTL (modifiable)
   - Cible (modifiable)
   - Valeur actuelle (affichage)
   - Nouvelle valeur (prévisualisation)
3. FOOTER - Boutons Annuler / Enregistrer

# MODAL TYPE
- Type: Formulaire édition record
- Dimensions: 600x560px
- Header: Fond gris (#F9FAFB)

# CHAMPS
| Champ | Type | Modifiable | Description |
|-------|------|------------|-------------|
| Type | badge | Non | Type d'enregistrement |
| Sous-domaine | text | Non | Préfixe |
| TTL | select | Oui | Durée de cache |
| Cible | text | Oui | Valeur de l'enregistrement |

# RESTRICTIONS
- Le type ne peut pas être modifié
- Le sous-domaine ne peut pas être modifié
- Pour changer type ou subdomain: supprimer + recréer

# CHAMPS MODIFIABLES SELON TYPE
| Type | Champs modifiables |
|------|-------------------|
| A | TTL, target (IP) |
| AAAA | TTL, target (IPv6) |
| CNAME | TTL, target (FQDN) |
| MX | TTL, priority, target |
| TXT | TTL, target (texte) |
| SRV | TTL, priority, weight, port, target |

# COMPARAISON
La modale affiche:
- Valeur actuelle (fond gris)
- Nouvelle valeur (fond bleu avec bordure)
Permet à l'utilisateur de visualiser les changements.

# API ENDPOINTS
GET /domain/zone/{zone}/record/{id} - Détails enregistrement
PUT /domain/zone/{zone}/record/{id} - Modifier enregistrement
  Body: { target, ttl }
POST /domain/zone/{zone}/refresh - Rafraîchir zone
