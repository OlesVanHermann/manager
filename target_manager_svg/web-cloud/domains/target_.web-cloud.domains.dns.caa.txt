# STRUCTURE RECORD CAA
| Champ | Description |
|-------|-------------|
| Flag  | 0 (normal) ou 128 (critical) |
| Tag   | Type de politique (issue, issuewild, iodef) |
| Value | Domaine de la CA ou email/URL |

# TAGS CAA
| Tag | Description |
|-----|-------------|
| issue | CA autorisée pour certificats standards |
| issuewild | CA autorisée pour certificats wildcard |
| iodef | URL/email pour signaler les violations |
| contactemail | Email de contact pour les CA (RFC 8657) |
| contactphone | Téléphone de contact (RFC 8657) |

# API ENDPOINTS
GET /domain/zone/{zone}/record?fieldType=CAA - Lister records CAA
POST /domain/zone/{zone}/record - Créer record CAA
PUT /domain/zone/{zone}/record/{id} - Modifier record
DELETE /domain/zone/{zone}/record/{id} - Supprimer record
POST /domain/zone/{zone}/refresh - Appliquer changements
