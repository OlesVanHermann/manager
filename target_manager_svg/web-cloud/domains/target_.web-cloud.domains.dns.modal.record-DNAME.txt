# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - DNAME" + badge DNAME (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL (secondes)
   - Cible (FQDN obligatoire)
   - Texte d'aide explicatif
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire DNAME
- Dimensions: 600x420px
- Header: Fond OVH bleu (#000E9C)
- Badge: Rouge pour type DNAME

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe à rediriger |
| TTL | number | Non | Durée de cache |
| Cible | text | Oui | FQDN du domaine cible |

# DIFFÉRENCE CNAME vs DNAME
| Type | Portée |
|------|--------|
| CNAME | Un seul nom |
| DNAME | Tout le sous-arbre DNS |

# CAS D'USAGE DNAME
- Renommage de domaine
- Alias de zone entière
- Migration de domaine

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement DNAME
POST /domain/zone/{zone}/refresh - Rafraîchir zone
