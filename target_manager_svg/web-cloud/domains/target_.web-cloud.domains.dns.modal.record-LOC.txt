# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - LOC" + badge LOC (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL
   - Latitude (degrés, minutes, secondes, direction)
   - Longitude (degrés, minutes, secondes, direction)
   - Altitude (mètres)
   - Taille, Précision horizontale, Précision verticale
   - Exemple GPS
   - Aperçu de l'enregistrement
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire LOC (localisation géographique)
- Dimensions: 700x620px
- Header: Fond OVH bleu (#000E9C)
- Badge: Vert pour type LOC

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe |
| TTL | number | Non | Durée de cache |
| Latitude deg | number | Oui | 0-90 |
| Latitude min | number | Non | 0-59 |
| Latitude sec | number | Non | 0.000-59.999 |
| Latitude dir | select | Oui | N ou S |
| Longitude deg | number | Oui | 0-180 |
| Longitude min | number | Non | 0-59 |
| Longitude sec | number | Non | 0.000-59.999 |
| Longitude dir | select | Oui | E ou W |
| Altitude | number | Oui | Mètres |
| Taille | number | Non | Mètres |
| Préc. horiz | number | Non | Mètres |
| Préc. vert | number | Non | Mètres |

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement LOC
POST /domain/zone/{zone}/refresh - Rafraîchir zone
