# Cahier des Charges : web-cloud/domains - Modal Changement propriétaire

## 1. VUE D'ENSEMBLE
- **Page** : Changement de propriétaire
- **Type** : Modal formulaire complexe
- **Objectif** : Transférer la propriété du domaine

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau warning (irréversible)
- Champ domaine (lecture seule)
- Radio type propriétaire
- Select contact existant
- Lien créer nouveau contact
- Checkbox confirmation
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Warning (orange)
- **OdsInput** : Domaine (disabled)
- **OdsRadio** : Type propriétaire
- **OdsSelect** : Contact existant
- **OdsLink** : Créer nouveau contact
- **OdsCheckbox** : Confirmation
- **OdsButton** : Annuler, Confirmer

## 4. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠ |
| Texte | Cette opération est irréversible et peut prendre jusqu'à 5 jours ouvrés |

## 5. TYPE PROPRIÉTAIRE (Radio)
| Option | Description |
|--------|-------------|
| Particulier | Personne physique |
| Entreprise | Personne morale |

## 6. CONTACT
| Élément | Description |
|---------|-------------|
| Select | Sélectionner un contact existant |
| Lien | + Créer un nouveau contact |

## 7. CONFIRMATION
| Propriété | Valeur |
|-----------|--------|
| Type | Checkbox |
| Label | Je confirme vouloir changer le propriétaire |
| Requis | Oui |

## 8. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Confirmer | Primary | Droite |

## 9. APIS & ENDPOINTS
- POST /domain/{serviceName}/changeOwner - Changement propriétaire
- GET /me/contact - Liste des contacts
