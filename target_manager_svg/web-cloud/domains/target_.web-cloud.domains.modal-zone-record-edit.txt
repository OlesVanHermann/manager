# Cahier des Charges : web-cloud/domains - Modal Zone Record Edit

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification entrée DNS
- **URL** : N/A (composant modal)
- **Objectif** : Modifier enregistrement zone DNS

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Badge type enregistrement
- Formulaire modification
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 380px |

## 4. BADGE TYPE
| Propriété | Valeur |
|-----------|--------|
| Fond | #0050D7 |
| Texte | "A" blanc |
| Description | "Enregistrement de type A" |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Modifiable |
|-------|------|------------|
| Sous-domaine | Input + suffixe | Oui |
| Cible (IPv4) | Input | Oui |
| TTL | Input number | Oui |

## 6. SOUS-DOMAINE
| Propriété | Valeur |
|-----------|--------|
| Valeur | "www" |
| Suffixe | ".exemple.com" (readonly) |

## 7. TTL

|-----------|--------|

## 8. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Modifier | Primary | "Modifier" |
