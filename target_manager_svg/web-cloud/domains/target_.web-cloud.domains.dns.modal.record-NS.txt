# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - NS" + badge NS (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL (généralement 86400s = 1 jour)
   - Cible (serveur DNS FQDN)
   - Texte d'aide
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire NS
- Dimensions: 600x420px
- Header: Fond OVH bleu (#000E9C)
- Badge: Orange pour type NS

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Zone déléguée |
| TTL | number | Non | 86400s typique |
| Cible | text | Oui | Serveur DNS FQDN |

# CAS D'USAGE NS
- Délégation de sous-zone
- Serveurs DNS secondaires
- Délégation à un prestataire tiers

# TTL RECOMMANDÉ
- 86400 secondes (1 jour) pour NS
- Les NS changent rarement

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement NS
POST /domain/zone/{zone}/refresh - Rafraîchir zone
