# STRUCTURE MODAL
1. HEADER - Titre "Ajouter un enregistrement A" + bouton fermer (×)
2. CONTENT
   - Bannière info explicative (type A)
   - Champ sous-domaine + suffixe domaine
   - Champ adresse IPv4 cible (obligatoire)
   - Sélecteur TTL
   - Aperçu de l'enregistrement (format DNS)
3. FOOTER - Boutons Annuler / Ajouter

# MODAL TYPE
- Type: Formulaire d'ajout
- Dimensions: 600x620px
- Layout: Formulaire vertical avec aperçu

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe (vide = @) |
| IPv4 cible | text | Oui | Adresse IPv4 valide |
| TTL | select | Non | Durée de cache |

# VALEURS TTL DISPONIBLES
- 60 (1 minute)
- 300 (5 minutes)
- 3600 (1 heure) - défaut
- 86400 (1 jour)
- 604800 (1 semaine)

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement A
POST /domain/zone/{zone}/refresh - Rafraîchir zone après ajout
