# MÉCANISMES SPF
| Mécanisme | Description |
|-----------|-------------|
| include   | Inclure les serveurs d'un domaine |
| a         | Autoriser les IPs du record A |
| mx        | Autoriser les serveurs MX |
| ip4       | Autoriser une IP ou plage IPv4 |
| ip6       | Autoriser une IP ou plage IPv6 |
| all       | Règle par défaut |

# QUALIFICATEURS SPF
| Qualificateur | Signification |
|---------------|---------------|
| + (pass)      | Autorisé (défaut) |
| - (fail)      | Refuser |
| ~ (softfail)  | Marquer comme spam |
| ? (neutral)   | Neutre |

# API ENDPOINTS
GET /domain/zone/{zone}/record?fieldType=SPF - Lister records SPF
GET /domain/zone/{zone}/record?fieldType=TXT&subDomain= - SPF dans TXT
POST /domain/zone/{zone}/record - Créer record SPF
PUT /domain/zone/{zone}/record/{id} - Modifier record
DELETE /domain/zone/{zone}/record/{id} - Supprimer record
POST /domain/zone/{zone}/refresh - Appliquer changements
