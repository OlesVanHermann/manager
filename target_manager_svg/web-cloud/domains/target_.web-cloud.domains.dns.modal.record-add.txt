# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS" + bouton fermer (×)
2. WIZARD STEPS - 3 étapes (Type, Configuration, Résumé)
3. CONTENT
   - Description de l'étape
   - Section "Enregistrements de base" (A, AAAA, NS, CNAME, DNAME)
   - Section "Enregistrements avancés" (CAA, TXT, NAPTR, SRV, LOC, SSHFP, TLSA, RP, SVCB, HTTPS)
   - Section "Enregistrements email" (MX, SPF, DKIM, DMARC)
4. HELP PANEL - Aide contextuelle sur les types
5. FOOTER - Boutons Annuler / Suivant

# TYPES DNS DISPONIBLES
| Type | Description |
|------|-------------|
| A | Adresse IPv4 |
| AAAA | Adresse IPv6 |
| NS | Name Server |
| CNAME | Alias canonique |
| DNAME | Délégation nom |
| CAA | Certificate Authority Authorization |
| TXT | Texte libre |
| NAPTR | Naming Authority Pointer |
| SRV | Service |
| LOC | Localisation |
| SSHFP | SSH Fingerprint |
| TLSA | TLS Authentication |
| RP | Responsible Person |
| SVCB | Service Binding |
| HTTPS | HTTPS Service |
| MX | Mail Exchange |
| SPF | Sender Policy Framework |
| DKIM | DomainKeys Identified Mail |
| DMARC | Domain-based Message Authentication |

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement
