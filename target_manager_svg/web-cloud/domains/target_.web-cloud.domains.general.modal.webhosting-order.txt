# STRUCTURE MODAL
1. HEADER - Titre "Commander un hébergement web" (fond OVH bleu)
2. CONTENT
   - Bannière info (domaine associé)
   - Offres d'hébergement (3 cartes radio)
   - Durée d'engagement (3 options radio)
   - Options incluses (checkboxes)
   - Récapitulatif de commande
3. FOOTER - Boutons Annuler / Commander

# MODAL TYPE
- Type: Formulaire de commande
- Dimensions: 800x740px
- Header: Fond OVH bleu (#000E9C)

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Offre | radio cards | Oui | Starter/Pro/Performance |
| Durée | radio | Oui | 12/24/36 mois |
| DNS auto | checkbox | Non | Configuration DNS automatique |
| SSL | checkbox | Non | Certificat Let's Encrypt |

# OFFRES HÉBERGEMENT
| Offre | Espace | BDD | Email | Prix/mois |
|-------|--------|-----|-------|-----------|
| Starter | 100 Go | 1 | 1 | 2,99 € |
| Pro | 250 Go | Illimité | 10 | 5,99 € |
| Performance | 500 Go NVMe | Illimité | 100 | 11,99 € |

# DURÉES ET RÉDUCTIONS
| Durée | Réduction | Prix Pro |
|-------|-----------|----------|
| 12 mois | - | 5,99 €/mois |
| 24 mois | -17% | 4,99 €/mois |
| 36 mois | -33% | 3,99 €/mois |

# OPTIONS INCLUSES
- Configuration DNS automatique: Crée les enregistrements A nécessaires
- SSL Let's Encrypt: Certificat gratuit, renouvellement automatique

# CALCUL PRIX
Total HT = Prix mensuel × Nombre de mois
Total TTC = Total HT × 1.20 (TVA 20%)

# API ENDPOINTS
GET /order/hosting/web - Liste des offres et prix
POST /order/hosting/web - Commander hébergement
  Body: {
    offer: "pro",
    duration: 24,
    domain: "exemple.com",
    dnsConfiguration: true,
    ssl: true
  }
