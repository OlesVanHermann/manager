# Cahier des Charges : web-cloud/domains - Serveurs DNS

## 1. VUE D'ENSEMBLE
- **Type** : Page liste serveurs DNS
- **URL** : /domain/{domain}/dns
- **Objectif** : Gérer serveurs DNS du domaine

## 2. STRUCTURE PAGE
- Messages système (success/info)
- Tableau serveurs DNS
- Sidebar boutons et guides

## 3. MESSAGE SUCCESS
| Propriété | Valeur |
|-----------|--------|
| Type | Success (vert) |
| Texte | "Vos serveurs DNS ont été mis à jour avec succès." |

## 4. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "Vérifiez la configuration de vos DNS avec l'outil ZoneCheck ↗" |
| Action | Lien externe ZoneCheck |

## 5. TABLEAU SERVEURS
| Colonne | Type |
|---------|------|
| Nom | Texte |
| IP | Texte |
| État | Badge (Activé/En cours) |
| Type | Badge (Standard/Externe) |

## 6. BADGES ÉTAT
| État | Couleur |
|------|---------|
| Activé | Vert #DEF7EC |

## 7. BADGES TYPE
| Type | Couleur |
|------|---------|
| Standard | Gris #E5E7EB |
| Externe | Bleu #DBEAFE |

## 8. SIDEBAR BOUTONS
| Bouton | Action |
|--------|--------|
| Activer DNS Anycast | Ouvrir modal anycast |
| Modifier les serveurs DNS | Ouvrir modal modification |

## 9. SIDEBAR GUIDES
| Guide | Lien |
|-------|------|
| Configurer les serveurs DNS | Documentation |
| Modifier une zone DNS | Documentation |

## 10. APIS & ENDPOINTS
- GET /domain/{domain}/nameServer - Liste serveurs
- PUT /domain/{domain}/nameServer - Modifier serveurs
- POST /domain/{domain}/option/dnsAnycast - Activer Anycast
