# STRUCTURE MODAL
1. HEADER - Titre "Modifier la redirection" (fond gris)
2. CONTENT
   - Sous-domaine source (désactivé, non modifiable)
   - URL de destination (modifiable)
   - Type de redirection (radios)
   - Options iframe (conditionnelles)
   - Configuration actuelle
3. FOOTER - Boutons Annuler / Enregistrer les modifications

# MODAL TYPE
- Type: Formulaire édition redirection
- Dimensions: 700x660px
- Header: Fond gris (#F9FAFB)

# CHAMPS
| Champ | Type | Modifiable | Description |
|-------|------|------------|-------------|
| Sous-domaine source | text | Non | Fixé à la création |
| URL de destination | url | Oui | URL complète |
| Type de redirection | radio | Oui | 301, 302, ou iframe |
| Titre | text | Oui | Pour iframe |
| Mots-clés | text | Oui | Pour iframe |
| Description | textarea | Oui | Pour iframe |

# RESTRICTIONS
- Le sous-domaine source ne peut pas être modifié
- Pour changer le sous-domaine, supprimer et recréer la redirection
- L'URL de destination peut être mise à jour à tout moment

# TYPES DE REDIRECTION
| Code | Nom | Description |
|------|-----|-------------|
| 301 | Permanente | Déménagement définitif |
| 302 | Temporaire | Redirection temporaire |
| iframe | Visible | Affichage dans frame |

# API ENDPOINTS
GET /domain/{domain}/redirection/{id} - Détails redirection
PUT /domain/{domain}/redirection/{id} - Modifier redirection
  Body: { target, type, title, description, keywords }
