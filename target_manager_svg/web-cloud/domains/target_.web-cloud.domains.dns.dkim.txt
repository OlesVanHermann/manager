# STRUCTURE RECORD DKIM
| Tag | Description |
|-----|-------------|
| v   | Version (DKIM1) |
| k   | Type de clé (rsa, ed25519) |
| p   | Clé publique (base64) |
| t   | Flags (s=strict, y=testing) |
| h   | Algorithmes de hash acceptés |

# API ENDPOINTS
GET /email/domain/{domain}/dkim - Lister sélecteurs DKIM
GET /email/domain/{domain}/dkim/{selectorName} - Détails sélecteur
POST /email/domain/{domain}/dkim - Créer sélecteur DKIM
DELETE /email/domain/{domain}/dkim/{selectorName} - Supprimer
GET /domain/zone/{zone}/record?fieldType=TXT&subDomain=*._domainkey - Records DKIM
POST /domain/zone/{zone}/record - Créer record TXT DKIM
POST /domain/zone/{zone}/refresh - Appliquer changements
