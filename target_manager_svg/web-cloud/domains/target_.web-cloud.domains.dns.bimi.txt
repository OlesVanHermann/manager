# PRÉREQUIS BIMI
1. DMARC configuré avec politique p=quarantine ou p=reject
2. Logo au format SVG Tiny 1.2 (PS/Secure)
3. Certificat VMC (optionnel mais requis pour Gmail/Apple)

# FORMAT RECORD BIMI
| Tag | Description | Obligatoire |
|-----|-------------|-------------|
| v   | Version (BIMI1) | Oui |
| l   | URL du logo SVG | Oui |
| a   | URL du certificat VMC | Non |

# SPÉCIFICATIONS LOGO
- Format: SVG Tiny 1.2 Portable/Secure
- Dimensions: Carré (1:1)
- Taille max: 32KB
- Pas de scripts/animations
- Fond transparent ou uni

# API ENDPOINTS
GET /domain/zone/{zone}/record?fieldType=TXT&subDomain=default._bimi - Record BIMI
POST /domain/zone/{zone}/record - Créer record BIMI
PUT /domain/zone/{zone}/record/{id} - Modifier record
DELETE /domain/zone/{zone}/record/{id} - Supprimer record
POST /domain/zone/{zone}/refresh - Appliquer changements
