# STRUCTURE MODAL
1. HEADER - Titre "Modifier les informations du contact" + bouton fermer (×)
2. CONTENT - Formulaire avec 4 sections:
   - Identité: Prénom*, Nom*
   - Adresse: Adresse*, Code postal*, Ville*, Pays*
   - Contact: E-mail*, Téléphone
   - Organisation (optionnel): Société, SIRET
3. FOOTER - Boutons Annuler / Enregistrer

# CHAMPS OBLIGATOIRES
Les champs marqués * sont obligatoires:
- Prénom, Nom
- Adresse, Code postal, Ville, Pays
- E-mail

# API ENDPOINTS
GET /domain/{domain}/contact/{contactType} - Récupérer contact
PUT /domain/{domain}/contact/{contactType} - Modifier contact

# TYPES DE CONTACT
- owner: Propriétaire
- admin: Administrateur
- tech: Technique
- billing: Facturation
