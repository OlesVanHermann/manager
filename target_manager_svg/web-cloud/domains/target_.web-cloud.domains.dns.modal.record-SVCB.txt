# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - SVCB" + badge SVCB (fond OVH bleu)
2. CONTENT
   - Bannière info SVCB
   - Bannière warning (type avancé)
   - Sous-domaine + suffixe
   - TTL
   - Priorité (SvcPriority) + Cible (TargetName)
   - Paramètres de service (SvcParams textarea)
   - Paramètres courants (référence)
   - Aperçu
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire SVCB
- Dimensions: 700x700px
- Header: Fond OVH bleu (#000E9C)
- Badge: Bleu pour type SVCB

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Service name |
| TTL | number | Non | Durée de cache |
| Priorité | number | Oui | 0=alias, 1+=service |
| Cible | text | Oui | FQDN ou "." |
| SvcParams | textarea | Non | Paramètres clé=valeur |

# MODES SVCB
| Priorité | Mode | Usage |
|----------|------|-------|
| 0 | AliasMode | Redirige vers un autre nom |
| 1+ | ServiceMode | Définit paramètres du service |

# PARAMÈTRES COURANTS (SvcParams)
| Param | Description | Exemple |
|-------|-------------|---------|
| alpn | Protocoles ALPN | h2, h3, dot |
| port | Port alternatif | 443, 853 |
| ipv4hint | IPv4 suggérée | 192.0.2.1 |
| ipv6hint | IPv6 suggérée | 2001:db8::1 |
| ech | Encrypted Client Hello | base64data |
| mandatory | Params obligatoires | alpn,port |
| no-default-alpn | Pas d'ALPN par défaut | (flag) |

# CAS D'USAGE SVCB
- DNS over TLS (DoT): _dns.domain
- DNS over HTTPS (DoH): _dns.domain
- Services génériques non-HTTP

# RELATION HTTPS/SVCB
- HTTPS est un alias pour SVCB avec schéma https
- Pour le web, utiliser le type HTTPS dédié
- SVCB pour autres services (DNS, générique)

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement SVCB
POST /domain/zone/{zone}/refresh - Rafraîchir zone

RFC 9460 - Service Binding and Parameter Specification
