# Cahier des Charges : web-cloud/domains - Modal Renouveler

## 1. VUE D'ENSEMBLE
- **Page** : Renouveler vos domaines
- **Type** : Modal avec tableau et pricing
- **Objectif** : Renouveler un ou plusieurs domaines

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau warning (durée registre)
- Tableau domaines sélectionnés
- Résumé tarification
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal large
- **OdsMessage** : Warning (orange)
- **OdsDatagrid** : Tableau domaines
- **OdsText** : Résumé prix
- **OdsButton** : Annuler, Renouveler

## 4. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠ |
| Titre | Attention |
| Texte | Le renouvellement sera effectué pour la durée minimale autorisée par le registre |

## 5. COLONNES TABLEAU
| Colonne | Type | Description |
|---------|------|-------------|
| Nom de domaine | Text | Nom du domaine |
| Expiration actuelle | Date | Date actuelle d'expiration |
| Nouvelle expiration | Date (bold) | Nouvelle date après renouvellement |

## 6. RÉSUMÉ TARIFICATION
| Élément | Valeur |
|---------|--------|
| Durée | 1 an |
| Total HT | 29,97 € |
| Détail | (3 × 9,99 €) |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Renouveler | Primary | Droite |

## 8. APIS & ENDPOINTS
- POST /domain/batch/renew - Renouvellement en lot
