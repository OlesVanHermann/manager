# Cahier des Charges : web-cloud/domains - Modal Restaurer et Renouveler

## 1. VUE D'ENSEMBLE
- **Page** : Restaurer et renouveler un domaine
- **Type** : Modal formulaire avec pricing
- **Objectif** : Restaurer un domaine en rédemption

## 2. STRUCTURE MODAL
- Header bleu OVH avec titre + bouton fermer
- Bandeau info (période rédemption)
- Box info domaine (nom + dates)
- Select durée renouvellement
- Box prix total
- Bandeau warning (frais élevés)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Info (bleu), Warning (orange)
- **OdsCard** : Info domaine
- **OdsSelect** : Durée renouvellement
- **OdsText** : Prix (vert)
- **OdsButton** : Annuler, Restaurer et payer

## 4. HEADER MODAL
| Propriété | Valeur |
|-----------|--------|
| Fond | Bleu OVH (#000E9C) |
| Titre | Restaurer et renouveler un domaine |

## 5. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | Votre domaine est en période de rédemption. Vous pouvez le restaurer moyennant des frais supplémentaires avant sa suppression définitive |

## 6. INFO DOMAINE
| Élément | Description |
|---------|-------------|
| Nom | exemple.com (bold) |
| Détails | Expiré le XX • Suppression prévue le YY |

## 7. DURÉE RENOUVELLEMENT
| Propriété | Valeur |
|-----------|--------|
| Label | Durée de renouvellement |
| Type | Select |
| Options | 1 an, 2 ans, etc. |

## 8. PRIX TOTAL
| Élément | Valeur |
|---------|--------|
| Label | Total à payer |
| Prix | 149,99 € HT (vert) |
| Note | (frais de restauration inclus) |

## 9. BANDEAU WARNING
| Propriété | Valeur |
|-----------|--------|
| Type | Warning (orange) |
| Icône | ⚠ |
| Texte | Les frais de restauration sont plus élevés que les frais de renouvellement standard. Cette opération ne peut pas être annulée une fois validée |

## 10. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Restaurer et payer | Primary | Droite |

## 11. APIS & ENDPOINTS
- POST /domain/{serviceName}/restore - Restauration domaine
- GET /domain/{serviceName}/redemption - Info rédemption
