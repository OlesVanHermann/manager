# Cahier des Charges : web-cloud/domains - Informations Générales

## 1. VUE D'ENSEMBLE
- **Type** : Page détails domaine
- **URL** : /domain/{domain}/information
- **Objectif** : Afficher toutes les infos du domaine

## 2. STRUCTURE PAGE
- Breadcrumb
- Titre page
- Carte informations

## 3. SECTION INFORMATIONS DOMAINE
### Colonne gauche
| Champ | Valeur exemple |
|-------|----------------|
| Nom du domaine | exemple.com |
| Statut | Badge "Actif" vert |
| Date de création | 15/03/2020 |
| Date d'expiration | 15/03/2025 |
| Registrar | OVH |

### Colonne droite
| Champ | Valeur exemple |
|-------|----------------|
| Extension | .com |
| Verrouillage transfert | Badge "Verrouillé" vert |
| DNS Zone | OVHcloud |
| DNSSEC | Badge "Actif" vert |
| Renouvellement | Automatique |

## 4. SECTION SERVEURS DNS
| Colonne | Exemple |
|---------|---------|
| Serveur | dns1.ovh.net |
| IP | 213.251.128.136 |

## 5. BOUTONS ACTION
| Bouton | Label |
|--------|-------|
| Secondary | "Modifier les DNS" |
| Secondary | "Changer le propriétaire" |
| Secondary | "Renouveler" |

## 6. APIS & ENDPOINTS
- GET /domain/{domain} - Infos domaine
- GET /domain/{domain}/nameServer - Serveurs DNS
- PUT /domain/{domain} - Modifier domaine
