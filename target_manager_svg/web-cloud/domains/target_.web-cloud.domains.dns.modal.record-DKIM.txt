# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - DKIM" + badge DKIM (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL
   - Version DKIM1 (checkbox)
   - Granularité
   - Algorithmes de hashage (SHA-1, SHA-256)
   - Algorithme de clé (RSA)
   - Notes (optionnel)
   - Clé publique base64 (obligatoire)
   - Révocation (checkbox)
   - Types de services (radio: Tous, Email, Aucun)
   - Mode test (radio: Non/Oui)
   - Mode strict (radio: Strict/Relaxed)
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire DKIM complexe
- Dimensions: 700x750px
- Header: Fond OVH bleu (#000E9C)
- Badge: Violet pour type DKIM

# CHAMPS DKIM
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | selector._domainkey |
| TTL | number | Non | Durée de cache |
| Version | checkbox | Oui | DKIM1 |
| Granularité | text | Non | * par défaut |
| Hash algorithms | checkbox | Oui | SHA-1, SHA-256 |
| Key algorithm | checkbox | Oui | RSA |
| Notes | text | Non | Description |
| Clé publique | textarea | Oui | Base64 encoded |
| Service types | radio | Non | *, email, none |
| Mode test | radio | Non | flag y |
| Mode strict | radio | Non | flag s |

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement TXT (DKIM)
POST /domain/zone/{zone}/refresh - Rafraîchir zone
