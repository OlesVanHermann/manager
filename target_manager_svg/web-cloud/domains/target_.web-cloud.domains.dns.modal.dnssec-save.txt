# DNSSEC
Le protocole DNSSEC (DNS Security Extensions) ajoute une couche
de sécurité aux requêtes DNS en utilisant des signatures
cryptographiques pour authentifier les réponses DNS.

# API ENDPOINTS
POST /domain/{domain}/activateDnssec - Activer DNSSEC
POST /domain/{domain}/deactivateDnssec - Désactiver DNSSEC
GET /domain/{domain}/dsRecord - État DNSSEC
