# STRUCTURE MODAL
1. HEADER - Titre "Importer une zone DNS (texte)" (fond OVH bleu)
2. CONTENT
   - Bannière info
   - Bannière warning (écrasement)
   - Zone de destination
   - Textarea grand format (contenu zone)
   - Options (checkboxes)
   - Aide format
3. FOOTER - Boutons Annuler / Valider / Importer

# MODAL TYPE
- Type: Import texte direct
- Dimensions: 700x700px
- Header: Fond OVH bleu (#000E9C)

# CHAMPS
| Champ | Type | Description |
|-------|------|-------------|
| Contenu zone | textarea | Texte format BIND |
| Valider syntaxe | checkbox | Vérifier avant import |
| Conserver NS | checkbox | Garder les NS actuels |

# DIFFÉRENCE AVEC modal-zone-import
- modal-zone-import: Upload de fichier
- modal-zone-import-text: Copier-coller direct

# FORMAT BIND ATTENDU
```
$TTL 3600
@    IN    SOA   dns1.ovh.net. tech.ovh.net. (
                 2024123001 ; serial
                 86400      ; refresh
                 7200       ; retry
                 3600000    ; expire
                 86400 )    ; minimum
@    IN    NS    dns1.ovh.net.
@    IN    NS    dns2.ovh.net.
@    IN    A     213.186.33.5
www  IN    CNAME exemple.com.
```

# BOUTONS
| Bouton | Action |
|--------|--------|
| Annuler | Fermer sans sauvegarder |
| Valider | Vérifier la syntaxe uniquement |
| Importer | Appliquer les changements |

# API ENDPOINTS
POST /domain/zone/{zone}/import - Importer contenu zone
  Body: { zoneFile: "contenu texte" }
POST /domain/zone/{zone}/refresh - Rafraîchir après import
