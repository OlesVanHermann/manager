# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Domaines et DNS actif
3. LEFT PANEL - Liste domaines/zones
   - Search bar
   - Filter dropdown
   - Service count
   - Service items (selected, hover states)
4. RIGHT PANEL - Dashboard domaine
   - Header: {domainName} + badge statut
   - Tile "Informations générales"
     - Statut, Serveurs DNS, Dates, Protection, DNSSEC
   - Tile "Sécurité"
     - Lock transfert, DNSSEC, AUTH/INFO, WHOIS, Anti-spam
   - Tile "Hébergement Web et e-mail"
     - Hébergements associés, Sous-domaines
   - Tile "Résumé des options"
     - DNS Anycast status
   - Actions rapides

# INTERACTIONS
- Clic service LEFT → Affiche dashboard
- Clic "Voir les détails →" → Page Infos générales
- Clic "Modifier" → Modal correspondante
- Clic "Afficher" AUTH/INFO → Modal authinfo
- Clic actions rapides → Navigation/Modal

# API ENDPOINTS
GET /domain - Liste domaines
GET /domain/{domain} - Détails domaine
GET /domain/{domain}/serviceInfos - Infos facturation
GET /domain/zone - Liste zones
