# STRUCTURE MODAL
1. HEADER - Titre "Supprimer la redirection" (fond OVH bleu)
2. CONTENT
   - Bannière warning (action irréversible)
   - Question de confirmation
   - Détails de la redirection (source, destination, type)
   - Texte explicatif
3. FOOTER - Boutons Annuler / Supprimer (rouge)

# MODAL TYPE
- Type: Confirmation de suppression
- Dimensions: 500x340px
- Header: Fond OVH bleu (#000E9C)
- Bouton action: Rouge danger (#DC2626)

# CHAMPS AFFICHÉS
| Champ | Description |
|-------|-------------|
| Source | Sous-domaine + domaine |
| Destination | URL de redirection |
| Type | 301, 302, ou iframe |

# COMPORTEMENT
- L'utilisateur doit confirmer la suppression
- Aucune saisie requise (confirmation simple)
- Bouton Supprimer en rouge pour indiquer danger

# CONSÉQUENCES
- La redirection est supprimée immédiatement
- Les visiteurs ne sont plus redirigés
- L'action est irréversible

# API ENDPOINTS
GET /domain/{domain}/redirection/{id} - Détails redirection
DELETE /domain/{domain}/redirection/{id} - Supprimer redirection
