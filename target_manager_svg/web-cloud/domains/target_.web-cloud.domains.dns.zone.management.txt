# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Domaines et DNS actif
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: {domainName}
   - NAV3: 10 tabs (Serveurs DNS actif)
   - Warning banner: Si moins de 2 serveurs DNS
   - Toolbar:
     - [+ Ajouter un serveur DNS]
     - [Réinitialiser serveurs DNS]
     - [Activer DNS Anycast]
   - Table serveurs DNS
     - Colonnes: Serveur DNS, IP associée, Statut, Actions
     - Context menu: Modifier, Supprimer

# INTERACTIONS
- Clic "+ Ajouter" → Modal ajout serveur
- Clic "Réinitialiser" → Modal confirmation reset vers OVH
- Clic "Activer DNS Anycast" → Page commande Anycast
- Clic "⋮" → Context menu (Modifier, Supprimer)
- Clic "Modifier" → Modal édition serveur
- Clic "Supprimer" → Modal confirmation suppression

# API ENDPOINTS
GET /domain/{domain}/nameServer - Liste serveurs DNS
POST /domain/{domain}/nameServer - Ajouter/Modifier serveurs
DELETE /domain/{domain}/nameServer/{id} - Supprimer serveur
GET /domain/{domain}/nameServer/status - Statut serveurs
POST /order/domain/zone/{zone}/dnsAnycast - Commander Anycast
