# STRUCTURE MODAL
1. HEADER - Titre "Rattacher un domaine" + bouton fermer (×)
2. CONTENT
   - Description avec nom du domaine principal
   - Select dropdown pour choisir le domaine à rattacher
   - Info text sur l'héritage de la zone DNS
3. FOOTER - Boutons Annuler / Confirmer

# FONCTIONNALITÉ
Permet de rattacher un sous-domaine au domaine principal
pour qu'il hérite automatiquement de la zone DNS.

# API ENDPOINTS
POST /domain/{domain}/attachedDomain - Rattacher un domaine
GET /domain/{domain}/attachedDomain - Liste domaines rattachés
DELETE /domain/{domain}/attachedDomain/{id} - Détacher
