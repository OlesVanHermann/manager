# STRUCTURE MODAL
1. HEADER - Titre "Associer un hébergement au sous-domaine" (fond OVH bleu)
2. CONTENT
   - Bannière info
   - Sous-domaine concerné (lecture seule)
   - Liste des hébergements (radios)
   - Option SSL Let's Encrypt
   - Résumé des actions
3. FOOTER - Boutons Annuler / Associer

# MODAL TYPE
- Type: Sélection hébergement + configuration
- Dimensions: 700x620px
- Header: Fond OVH bleu (#000E9C)

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Hébergement | radio | Oui | Hébergement à associer |
| SSL | checkbox | Non | Activer Let's Encrypt |

# HÉBERGEMENTS LISTÉS
- Tous les hébergements web du compte sont listés
- Badge d'état: Actif (vert), Expiré (gris)
- Les hébergements expirés sont désactivés

# ACTIONS EFFECTUÉES
1. Création enregistrement A dans la zone DNS
2. Ajout du domaine attaché sur l'hébergement
3. (Optionnel) Génération certificat SSL Let's Encrypt

# SSL LET'S ENCRYPT
- Gratuit et automatique
- Renouvelé automatiquement tous les 90 jours
- Peut prendre quelques minutes à être actif

# API ENDPOINTS
GET /hosting/web - Liste des hébergements
POST /domain/zone/{zone}/record - Créer enregistrement A
POST /hosting/web/{hosting}/attachedDomain - Ajouter domaine attaché
POST /hosting/web/{hosting}/ssl - Générer certificat SSL
