# QU'EST-CE QUE OWO ?
OWO (Owner WHOIS Obfuscation) est un service gratuit OVH qui permet
de masquer les informations personnelles du propriétaire d'un domaine
dans les bases WHOIS publiques.

# NIVEAUX DE PROTECTION
| Niveau | Email | Nom | Adresse | Téléphone |
|--------|-------|-----|---------|-----------|
| Désactivé | ✗ | ✗ | ✗ | ✗ |
| Email masqué | ✓ | ✗ | ✗ | ✗ |
| Protection complète | ✓ | ✓ | ✓ | ✓ |

✓ = masqué, ✗ = visible

# EMAIL ANONYME
Quand l'email est masqué, il est remplacé par une adresse du type :
`xxxxx@contact-whois.ovh.net`
Les emails envoyés à cette adresse sont transférés vers l'adresse
réelle du propriétaire.

# EXTENSIONS SUPPORTÉES
- La plupart des gTLD (.com, .net, .org, .info...)
- Nombreux ccTLD (.fr, .be, .eu...)
- Attention : certains registres (ex: .uk) ont leurs propres règles

# RGPD
Depuis le RGPD (2018), de nombreuses informations WHOIS sont déjà
masquées par défaut pour les personnes physiques. OWO offre une
protection supplémentaire.

# TARIF
**Gratuit** - Inclus avec tous les domaines OVH

# API ENDPOINTS
GET /domain/{domain}/owo - Récupérer niveau actuel
  Response: { level: "email" | "full" | "none" }

PUT /domain/{domain}/owo - Modifier protection
  Body: { level: "email" | "full" | "none" }

# CHAMPS WHOIS CONCERNÉS
- registrant-name
- registrant-street
- registrant-city
- registrant-phone
- registrant-email
- admin-* (mêmes champs)
- tech-* (mêmes champs)
