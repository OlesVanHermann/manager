# STRUCTURE MODAL
1. HEADER - Titre "Changement Serveurs DNS (Batch)"
2. CONTENT
   - Bannière info (propagation DNS)
   - Champs serveurs DNS (4 inputs, 2 obligatoires)
   - Tableau domaines sélectionnés avec DNS actuels
   - Bannière warning (indisponibilité possible)
3. FOOTER - Boutons Annuler / Appliquer (N)

# MODAL TYPE
- Type: Batch action sur multiple domaines
- Dimensions: 800x680px
- Contexte: Depuis la page AllDom

# CHAMPS DNS
| # | Label | Obligatoire | Placeholder |
|---|-------|-------------|-------------|
| 1 | Serveur DNS 1 | Oui | dns1.ovh.net |
| 2 | Serveur DNS 2 | Oui | ns1.ovh.net |
| 3 | Serveur DNS 3 | Non | - |
| 4 | Serveur DNS 4 | Non | - |

# TABLEAU DOMAINES
| Colonne | Description |
|---------|-------------|
| ☑️ | Checkbox sélection |
| DOMAINE | Nom du domaine |
| DNS ACTUELS | Serveurs DNS actuellement configurés |

# VALIDATIONS
- Au moins 2 serveurs DNS requis
- Format FQDN valide
- Serveurs DNS doivent répondre (vérification)

# PROPAGATION DNS
- Délai: 24h à 48h maximum
- Varie selon TTL des enregistrements NS au registre
- Pendant propagation: réponses mixtes possibles

# API ENDPOINTS
POST /domain/batch/nameServer - Changer DNS en batch
  Body: {
    domains: ["example.com", "example.fr", "example.eu"],
    nameServers: ["dns1.ovh.net", "ns1.ovh.net"]
  }

GET /domain/{domain}/nameServer - Obtenir DNS actuels
  Response: {
    nameServers: ["dns1.ovh.net", "ns1.ovh.net"]
  }
