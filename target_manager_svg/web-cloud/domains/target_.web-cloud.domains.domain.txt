# Cahier des Charges : web-cloud/domains - Dashboard Principal

## 1. VUE D'ENSEMBLE
- **Type** : Dashboard domaine
- **URL** : /domain/{domain}
- **Objectif** : Vue globale de la configuration domaine

## 2. STRUCTURE PAGE
- Header OVHcloud
- Sidebar navigation
- Zone contenu avec tuiles

## 3. SIDEBAR NAVIGATION
| Menu | Lien |
|------|------|
| Informations générales | /domain/{domain}/information |
| Zone DNS | /domain/{domain}/zone |
| Serveurs DNS | /domain/{domain}/dns |
| Redirections | /domain/{domain}/redirection |
| DynHost | /domain/{domain}/dynhost |
| GLUE | /domain/{domain}/glue |
| Tâches | /domain/{domain}/tasks |
