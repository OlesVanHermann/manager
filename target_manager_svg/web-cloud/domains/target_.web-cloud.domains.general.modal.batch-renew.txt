# STRUCTURE MODAL
1. HEADER - Titre "Renouvellement Domaines (Batch)"
2. CONTENT
   - Bannière info (renouvellement groupé)
   - Sélecteur durée (1 an, 2 ans, etc.)
   - Tableau domaines avec prix
   - Récapitulatif avec totaux
3. FOOTER - Boutons Annuler / Commander

# MODAL TYPE
- Type: Commande batch avec paiement
- Dimensions: 800x680px
- Contexte: Depuis la page AllDom

# DURÉES DISPONIBLES
- 1 an (par défaut)
- 2 ans
- 3 ans
- 5 ans (selon TLD)
- 10 ans (selon TLD)

# TABLEAU DOMAINES
| Colonne | Description |
|---------|-------------|
| ☑️ | Checkbox sélection |
| DOMAINE | Nom du domaine |
| EXPIRATION | Date expiration actuelle |
| PRIX RENOUVELLEMENT | Prix HT selon extension |
| NOUVELLE EXP. | Nouvelle date après renouvellement |

# RÉCAPITULATIF
- Nombre de domaines
- Durée sélectionnée
- Total HT
- TVA (20%)
- Total TTC (en bleu, plus grand)

# PRIX PAR EXTENSION
Exemples de prix annuels HT :
- .com : 9,99 €
- .fr : 7,99 €
- .eu : 6,99 €
- .org : 12,99 €
- .net : 12,99 €

# API ENDPOINTS
POST /domain/batch/renew - Commander renouvellement batch
  Body: {
    domains: ["example.com", "example.fr", "example.eu", "example.org"],
    duration: 1
  }
  Response: {
    orderId: "12345678",
    total: { ht: 37.96, tva: 7.59, ttc: 45.55 }
  }

GET /price/domain/renew/{tld} - Prix renouvellement par TLD
  Response: { price: "9.99", currency: "EUR" }
