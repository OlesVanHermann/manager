# Cahier des Charges : web-cloud/domains - Modal Export CSV

## 1. VUE D'ENSEMBLE
- **Page** : Export CSV des domaines
- **Type** : Modal formulaire avec options
- **Objectif** : Exporter la liste des domaines en CSV

## 2. STRUCTURE MODAL
- Header avec titre + bouton fermer
- Bandeau info explicatif
- Checkboxes colonnes à exporter
- Sélection format (radio)
- Boutons action

## 3. COMPOSANTS UI
- **OdsModal** : Modal principale
- **OdsMessage** : Bandeau info
- **OdsCheckbox** : Sélection colonnes
- **OdsRadio** : Format CSV
- **OdsButton** : Annuler, Exporter

## 4. BANDEAU INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Icône | ℹ️ |
| Texte | L'export contiendra tous vos domaines avec leurs informations principales |

## 5. COLONNES À EXPORTER (Checkboxes)
| Colonne | État par défaut |
|---------|-----------------|
| Nom de domaine | ✓ Coché |
| Date d'expiration | ✓ Coché |
| Statut DNSSEC | ✓ Coché |
| DNS Anycast | ✓ Coché |
| Serveurs DNS | Non coché |
| Contact propriétaire | Non coché |

## 6. FORMAT (Radio)
| Option | Description |
|--------|-------------|
| CSV (séparateur virgule) | Standard international |
| CSV (séparateur point-virgule) | Format Excel FR |

## 7. BOUTONS
| Bouton | Style | Position |
|--------|-------|----------|
| Annuler | Outline | Gauche |
| Exporter | Primary | Droite |

## 8. APIS & ENDPOINTS
- GET /domain/export - Export CSV des domaines
