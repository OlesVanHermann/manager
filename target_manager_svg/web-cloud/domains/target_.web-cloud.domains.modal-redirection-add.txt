# Cahier des Charges : web-cloud/domains - Modal Redirection Add

## 1. VUE D'ENSEMBLE
- **Type** : Modal création redirection
- **URL** : N/A (composant modal)
- **Objectif** : Créer nouvelle redirection

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Formulaire
- Message info
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 550px |
| Hauteur | 420px |

## 4. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Domaine à rediriger | Input + suffixe | Oui |
| Type de redirection | Radio buttons | Oui |
| URL de destination | Input URL | Oui |

## 5. DOMAINE À REDIRIGER
| Propriété | Valeur |
|-----------|--------|
| Placeholder | "www" |
| Suffixe | ".mondomaine.com" |

## 6. TYPES REDIRECTION
| Option | Label | Sélectionné |
|--------|-------|-------------|
| 301 | "Redirection visible (HTTP 301)" | Oui (défaut) |
| 302 | "Redirection visible (HTTP 302)" | Non |
| iframe | "Redirection invisible (iFrame)" | Non |

## 7. URL DESTINATION
| Propriété | Valeur |
|-----------|--------|
| Placeholder | "https://example.com/page" |

## 8. MESSAGE INFO
| Propriété | Valeur |
|-----------|--------|
| Type | Info (bleu) |
| Texte | "ℹ La redirection peut prendre quelques minutes pour être effective." |

## 9. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Ajouter | Primary | "Ajouter" |

## 10. APIS & ENDPOINTS
- POST /domain/{domain}/redirection - Créer redirection
