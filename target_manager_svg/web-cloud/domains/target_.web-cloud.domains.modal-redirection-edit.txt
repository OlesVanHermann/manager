# Cahier des Charges : web-cloud/domains - Modal Redirection Edit

## 1. VUE D'ENSEMBLE
- **Type** : Modal modification redirection
- **URL** : N/A (composant modal)
- **Objectif** : Modifier redirection existante

## 2. STRUCTURE MODAL
- Header avec titre et fermeture
- Redirection actuelle
- Formulaire modification
- Boutons action

## 3. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 600px |
| Hauteur | 480px |

## 4. REDIRECTION ACTUELLE
| Propriété | Valeur |
|-----------|--------|
| Format | "Modification de la redirection de {source} vers {cible}" |
| Source | "www.exemple.com" (bold) |
| Cible | "https://cible.com" (bold) |

## 5. CHAMPS FORMULAIRE
| Champ | Type | Obligatoire |
|-------|------|-------------|
| Cible de la redirection | Input URL | Oui |
| Titre de la page | Input | Non (invisible) |
| Mots-clés | Input | Non (invisible) |
| Description | Input | Non (invisible) |

## 6. OPTIONS REDIRECTION INVISIBLE
| Propriété | Description |
|-----------|-------------|
| Titre | Pour balise <title> |
| Mots-clés | Pour meta keywords |
| Description | Pour meta description |

## 7. BOUTONS
| Bouton | Style | Label |
|--------|-------|-------|
| Annuler | Secondary | "Annuler" |
| Modifier | Primary | "Modifier" |

## 8. APIS & ENDPOINTS
- PUT /domain/{domain}/redirection/{id} - Modifier redirection
