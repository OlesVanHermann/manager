# TAGS DMARC
| Tag | Description | Valeurs |
|-----|-------------|---------|
| v   | Version | DMARC1 (obligatoire) |
| p   | Politique domaine | none, quarantine, reject |
| sp  | Politique sous-domaines | none, quarantine, reject |
| pct | Pourcentage appliqué | 0-100 |
| rua | Rapports agrégés | mailto:email |
| ruf | Rapports forensic | mailto:email |
| adkim | Alignement DKIM | r (relaxed), s (strict) |
| aspf | Alignement SPF | r (relaxed), s (strict) |

# POLITIQUES
| Politique | Description |
|-----------|-------------|
| none      | Surveiller, ne pas agir |
| quarantine| Marquer comme spam |
| reject    | Rejeter le message |

# API ENDPOINTS
GET /domain/zone/{zone}/record?fieldType=TXT&subDomain=_dmarc - Record DMARC
POST /domain/zone/{zone}/record - Créer record DMARC
PUT /domain/zone/{zone}/record/{id} - Modifier record
DELETE /domain/zone/{zone}/record/{id} - Supprimer record
POST /domain/zone/{zone}/refresh - Appliquer changements
