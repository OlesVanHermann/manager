STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Hebergement actif
3. LEFT PANEL - Liste hebergements
4. RIGHT PANEL
   - Header: Sauvegardes de la base {dbName}
   - INFO BANNER: Retention 30 jours + sauvegardes manuelles
   - TOOLBAR: + Creer une sauvegarde
   - TABLE: Date, Type, Taille, Statut, Expiration, Actions
     - Badges: Automatique (bleu), Manuel (jaune)
     - Statut: Disponible (vert)
     - Actions: Restaurer | Telecharger | ...
   - PAGINATION: Affichage x a y sur z

API ENDPOINTS
GET    /hosting/privateDatabase/{sn}/dump          Liste sauvegardes
POST   /hosting/privateDatabase/{sn}/dump          Creer sauvegarde
GET    /hosting/pDB/{sn}/dump/{id}                 Details sauvegarde
POST   /hosting/pDB/{sn}/dump/{id}/restore         Restaurer
GET    /hosting/pDB/{sn}/dump/{id}/download        Telecharger
DELETE /hosting/pDB/{sn}/dump/{id}                 Supprimer
