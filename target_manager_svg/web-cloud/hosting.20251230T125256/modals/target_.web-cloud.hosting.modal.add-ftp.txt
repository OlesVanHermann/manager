# STRUCTURE MODAL
1. HEADER - Titre "Créer un utilisateur FTP" + close
2. FORMULAIRE:
   - Identifiant *
   - Mot de passe * + bouton Générer
   - Dossier racine (sélecteur dossier)
   - Accès SSH (checkbox, selon offre)
3. BANNIÈRE INFO - Serveur et ports
4. ACTIONS: Annuler, Créer

# VALIDATION
- Identifiant: alphanumérique + tirets, max 32 chars
- Mot de passe: min 8 chars, complexité requise
- Dossier: doit exister sur l'hébergement

# API ENDPOINTS
POST /hosting/web/{serviceName}/user - Créer utilisateur FTP
GET /hosting/web/{serviceName}/user - Liste utilisateurs
