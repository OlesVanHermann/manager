# STRUCTURE MODAL
1. HEADER - Titre "Modifier la version PHP" + close
2. VERSION ACTUELLE - Affichage PHP actuel
3. GRID VERSIONS - Cards cliquables avec badges
4. BANNIÈRE INFO - Effet immédiat, tester après
5. ACTIONS: Annuler, Appliquer

# BADGES VERSIONS
| Badge       | Couleur | Description              |
|-------------|---------|--------------------------|
| Legacy      | Orange  | Version fin de vie       |
| Stable      | Vert    | Version stable           |
| Actuel      | Bleu    | Version en cours         |
| Recommandé  | Vert    | Version recommandée      |
| Latest      | Bleu    | Dernière version         |

# API ENDPOINTS
PUT /hosting/web/{serviceName} - Modifier config PHP
GET /hosting/web/{serviceName}/availablePhp - Versions disponibles
