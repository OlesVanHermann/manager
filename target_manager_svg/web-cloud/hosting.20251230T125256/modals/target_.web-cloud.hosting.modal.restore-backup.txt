# STRUCTURE MODAL
1. HEADER - Titre "Restaurer une sauvegarde" + close
2. TYPE SELECTION - Cards: Fichiers FTP, Base de données, Complet
3. DATE SELECTION - Select avec dates disponibles
4. LISTE SAUVEGARDES - Table avec date, taille, sélection
5. BANNIÈRE ERROR - Avertissement écrasement
6. ACTIONS: Annuler, Restaurer

# TYPES RESTAURATION
| Type     | Description                    |
|----------|--------------------------------|
| FTP      | Fichiers uniquement            |
| BDD      | Base de données uniquement     |
| Complet  | Fichiers + base de données     |

# RÉTENTION
- 7 sauvegardes quotidiennes
- Créées automatiquement à 03:00

# API ENDPOINTS
GET /hosting/web/{serviceName}/dump - Liste sauvegardes
POST /hosting/web/{serviceName}/restore - Lancer restauration
