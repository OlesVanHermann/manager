# STRUCTURE MODAL
1. HEADER - Titre "Créer une base de données" + close
2. FORMULAIRE:
   - Type (MySQL / PostgreSQL toggle)
   - Nom de la base *
   - Version (select)
   - Utilisateur *
   - Mot de passe * + bouton Générer
3. BANNIÈRE INFO - Serveur de connexion
4. ACTIONS: Annuler, Créer

# VALIDATION
- Nom: alphanumérique + underscore, max 16 chars
- Utilisateur: alphanumérique, max 16 chars
- Mot de passe: min 8 chars, complexité requise

# API ENDPOINTS
POST /hosting/web/{serviceName}/database - Créer base
GET /hosting/web/{serviceName}/databaseAvailableType - Types disponibles
