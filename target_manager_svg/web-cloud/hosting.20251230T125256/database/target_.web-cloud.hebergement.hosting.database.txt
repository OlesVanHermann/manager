STRUCTURE PAGE
1. Header + bouton "Creer BDD"
2. NAV4: Liste | Statistiques | Dumps
3. Banniere info serveur MySQL
4. Table bases de donnees
   - Colonnes: NOM, UTILISATEUR, TYPE, TAILLE, QUOTA, ACTIONS
   - Actions: MDP, Import SQL, Dump, Supprimer

TYPES BDD
| Type      | Versions disponibles          |
|-----------|-------------------------------|
| MySQL     | 5.7, 8.0                      |
| MariaDB   | 10.4, 10.5, 10.6              |

API ENDPOINTS
GET    /hosting/web/{sn}/database        -> Liste BDD
POST   /hosting/web/{sn}/database        -> Creer BDD
DELETE /hosting/web/{sn}/database/{name} -> Supprimer BDD
POST   /hosting/web/{sn}/database/{name}/changePassword -> Changer MDP
POST   /hosting/web/{sn}/database/{name}/dump     -> Creer dump
POST   /hosting/web/{sn}/database/{name}/import   -> Importer SQL
POST   /hosting/web/{sn}/database/{name}/restore  -> Restaurer dump

ACTIONS
- "+ Creer BDD" -> Modal creation
- "MDP" -> Modal changer mot de passe
- "Import" -> Modal importer fichier SQL
- "Dump" -> Creer sauvegarde BDD
- "x" -> Modal confirmation suppression
