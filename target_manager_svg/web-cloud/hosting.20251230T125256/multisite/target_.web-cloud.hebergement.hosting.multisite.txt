STRUCTURE PAGE
1. Header + bouton "Ajouter domaine"
2. Table domaines attaches
   - Colonnes: DOMAINE, CHEMIN, SSL, CDN, FIREWALL, ACTIONS
   - Badge etat pour chaque feature
   - Actions: Modifier, Diagnostic, Vider CDN, Supprimer

ACTIONS PAR LIGNE
| Action    | Description                      |
|-----------|----------------------------------|
| Modifier  | Modal modification domaine       |
| Diag      | Lancer diagnostic domaine        |
| Vider CDN | Purger cache CDN du domaine      |
| x         | Supprimer domaine (si non principal)|

API ENDPOINTS
GET    /hosting/web/{sn}/attachedDomain      -> Liste domaines
POST   /hosting/web/{sn}/attachedDomain      -> Ajouter domaine
PUT    /hosting/web/{sn}/attachedDomain/{d}  -> Modifier domaine
DELETE /hosting/web/{sn}/attachedDomain/{d}  -> Supprimer domaine
POST   /hosting/web/{sn}/attachedDomain/{d}/purgeCache -> Vider CDN
