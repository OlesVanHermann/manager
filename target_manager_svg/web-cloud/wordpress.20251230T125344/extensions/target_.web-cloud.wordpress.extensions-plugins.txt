STRUCTURE PAGE
1. NAV4 sous-tabs: Themes | [Plugins]

2. Header "PLUGINS INSTALLES (N)" + bouton "Tout mettre a jour (X)"

3. Table plugins
   - Colonnes: PLUGIN, VERSION, STATUT, ACTIONS
   - Affichage MAJ disponibles en sous-ligne
   - Badge Actif/Inactif

4. Banniere warning si MAJ disponibles

ETATS PLUGIN
| Etat       | Badge     | Couleur  | Actions disponibles    |
|------------|-----------|----------|------------------------|
| Actif      | Vert      | #DEF7EC  | Desactiver, MAJ        |
| Inactif    | Gris      | #F3F4F6  | Activer, MAJ           |
| MAJ dispo  | -         | -        | Affiche sous le nom    |

API ENDPOINTS
GET    /webWordpress/{serviceName}/plugin        -> Liste plugins
POST   /webWordpress/{serviceName}/plugin/{name}/activate   -> Activer
POST   /webWordpress/{serviceName}/plugin/{name}/deactivate -> Desactiver
POST   /webWordpress/{serviceName}/plugin/{name}/update     -> Mettre a jour
POST   /webWordpress/{serviceName}/plugin/updateAll         -> Tout MAJ

ACTIONS
- "Themes" -> Route /extensions (NAV4)
- "Plugins" -> Route /extensions-plugins (NAV4)
- "Tout mettre a jour" -> POST updateAll
- "MAJ" -> POST update
- "Activer" -> POST activate
- "Desactiver" -> POST deactivate
- "Verifier maintenant" -> GET plugin (refresh)
