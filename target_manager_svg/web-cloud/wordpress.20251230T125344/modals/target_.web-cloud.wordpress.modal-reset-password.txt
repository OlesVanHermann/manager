DESCRIPTION
Modal de réinitialisation du mot de passe administrateur WordPress.
Génère un nouveau mot de passe aléatoire et l'envoie par email.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                    |
|--------------------------|-----------------------------------------------|
| mon-blog.fr              | GET /managedCMS/resource/{sn} → displayName   |
| admin                    | GET /managedCMS/resource/{sn} → adminUser     |
| contact@mon-blog.fr      | GET /me → email                               |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière info: Explication de l'action
3. Site concerné: Nom + utilisateur admin
4. Email notification: Adresse email destinataire
5. Footer: Annuler | Réinitialiser

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Réinitialiser" → POST /managedCMS/resource/{sn}/resetAdminPassword

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/resetAdminPassword
  Response: { taskId, status: "pending" }
