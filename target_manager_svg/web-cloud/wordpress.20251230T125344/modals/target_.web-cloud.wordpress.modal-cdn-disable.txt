DESCRIPTION
Modal de confirmation pour désactiver le CDN sur un site WordPress Pro/Business.
Affiche l'impact potentiel et les statistiques actuelles du CDN.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                    |
|--------------------------|-----------------------------------------------|
| mon-blog.fr              | GET /managedCMS/resource/{sn} → displayName   |
| Pro                      | GET /managedCMS/resource/{sn} → offer         |
| 12.4 Go                  | GET /managedCMS/resource/{sn}/cdn → bandwidth |
| 45 230                   | GET /managedCMS/resource/{sn}/cdn → requests  |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière warning: Impact sur performances
3. Site concerné: Nom + offre + statut CDN
4. Statistiques CDN: Bande passante + requêtes
5. Footer: Annuler | Désactiver

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Désactiver" → POST /managedCMS/resource/{sn}/cdn/disable

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/cdn/disable
  Response: { taskId, status: "pending" }
