DESCRIPTION
Modal de confirmation pour mettre à jour un thème WordPress.
Affiche les informations du thème et les versions disponibles.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                          |
|--------------------------|-----------------------------------------------------|
| flavor flavor...         | GET /managedCMS/resource/{sn}/themes → name         |
| Actif                    | GET /managedCMS/resource/{sn}/themes → active       |
| 2.0.5                    | GET /managedCMS/resource/{sn}/themes → version      |
| 2.1.0                    | GET /managedCMS/resource/{sn}/themes → newVersion   |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière info: Sauvegarde automatique
3. Thème concerné: Aperçu + nom + statut
4. Versions: Actuelle vs nouvelle
5. Footer: Annuler | Mettre à jour

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Mettre à jour" → POST /managedCMS/resource/{sn}/themes/{name}/update

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/themes/{themeName}/update
  Response: { taskId, status: "pending" }
