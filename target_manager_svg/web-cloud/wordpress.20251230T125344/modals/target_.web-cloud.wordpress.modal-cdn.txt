DESCRIPTION
Modal de configuration du CDN pour un site WordPress Pro/Business.
Permet d'activer le CDN et de configurer les options de cache.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                    |
|--------------------------|-----------------------------------------------|
| mon-blog.fr              | GET /managedCMS/resource/{sn} → displayName   |
| Pro                      | GET /managedCMS/resource/{sn} → offer         |
| OVHcloud CDN             | GET /managedCMS/resource/{sn}/cdn → provider  |
| 19 PoP                   | GET /managedCMS/resource/{sn}/cdn → popCount  |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Site concerné: Nom + offre
3. Statut CDN: Actif/Inactif avec toggle
4. Options cache: TTL, exclusions
5. Bannière info: Avantages du CDN
6. Footer: Annuler | Activer/Configurer

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Toggle CDN → Active/désactive
- Sélection TTL → Modifie durée cache
- Clic "Activer" → POST /managedCMS/resource/{sn}/cdn/enable

================================================================================

API ENDPOINTS
POST /managedCMS/resource/{serviceName}/cdn/enable
  Body: { ttl: 3600, excludePaths: ["/wp-admin/*"] }
  Response: { taskId, status: "pending" }

GET /managedCMS/resource/{serviceName}/cdn
  Response: { enabled, provider, ttl, excludePaths, stats }
