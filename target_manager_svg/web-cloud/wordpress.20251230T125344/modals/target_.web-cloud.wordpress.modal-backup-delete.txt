DESCRIPTION
Modal de confirmation pour supprimer une sauvegarde manuelle.
Seules les sauvegardes manuelles peuvent être supprimées.
Les sauvegardes automatiques expirent selon la politique de rétention.

================================================================================

DONNÉES DYNAMIQUES
| Exemple SVG              | Source API                                          |
|--------------------------|-----------------------------------------------------|
| 27/12/2025 15:42         | GET /managedCMS/resource/{sn}/backups/{id} → date   |
| Manuel                   | GET /managedCMS/resource/{sn}/backups/{id} → type   |
| 242 Mo                   | GET /managedCMS/resource/{sn}/backups/{id} → size   |
| Note: Avant mise...      | GET /managedCMS/resource/{sn}/backups/{id} → note   |

================================================================================

STRUCTURE MODAL
1. Header: Titre + bouton fermer
2. Bannière error: Avertissement action irréversible
3. Sauvegarde concernée: Date + type + taille + note
4. Footer: Annuler | Supprimer

================================================================================

INTERACTIONS
- Clic "×" → Ferme modal
- Clic "Annuler" → Ferme modal
- Clic "Supprimer" → DELETE /managedCMS/resource/{sn}/backups/{id}

================================================================================

API ENDPOINTS
DELETE /managedCMS/resource/{serviceName}/backups/{backupId}
  Response: { status: "deleted" }
