STRUCTURE PAGE
1. QUICK ACTIONS (bandeau)
   - Acceder a l'admin WordPress
   - Visiter le site
   - Vider le cache
   - Reinitialiser mot de passe

2. BLOC WORDPRESS (340px)
   - Etat (circle + label)
   - Version WordPress
   - Version PHP
   - URL du site
   - Utilisateur admin
   - Bouton "Mettre a jour WP"

3. BLOC TECHNIQUE (340px)
   - Offre (badge)
   - Datacenter
   - SSL (etat)
   - CDN (etat)
   - Auto-update (etat)
   - Dates creation/expiration

4. BLOC STOCKAGE (336px)
   - Barre de progression utilisation
   - Taille base de donnees
   - Taille fichiers
   - Taille sauvegardes

5. ZONE DANGER
   - Avertissement suppression
   - Bouton "Supprimer le site" (rouge)

API ENDPOINTS
GET    /webWordpress/{serviceName}            -> Detail site
GET    /webWordpress/{serviceName}/storage    -> Info stockage
POST   /webWordpress/{serviceName}/purgeCache -> Vider cache
POST   /webWordpress/{serviceName}/resetPassword -> Reset MDP
POST   /webWordpress/{serviceName}/update     -> Lancer MAJ WP
DELETE /webWordpress/{serviceName}            -> Supprimer site

ACTIONS
- "Acceder a l'admin" -> window.open(adminUrl)
- "Visiter le site" -> window.open(url)
- "Vider le cache" -> POST purgeCache
- "Reinitialiser mot de passe" -> Modal reset-password
- "Mettre a jour WP" -> Modal update-wp
- "Supprimer le site" -> Modal delete
