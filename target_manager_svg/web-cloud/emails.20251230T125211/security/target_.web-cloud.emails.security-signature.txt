| Scope              | Description                           |

| Defaut             | Appliquee a tous les utilisateurs     |

FORMAT SIGNATURE
- HTML supporte (images, liens, mise en forme)
- Variables disponibles: {nom}, {prenom}, {email}, {tel}, etc.
- Taille max: 10 Ko
- Images: recommande liens externes (pas base64)

API ENDPOINTS
GET    /email/domains/{domain}/signatures         - Liste signatures
POST   /email/domains/{domain}/signatures         - Creer signature
GET    /email/domains/{d}/signatures/{id}         - Details signature
PUT    /email/domains/{d}/signatures/{id}         - Modifier signature
DELETE /email/domains/{d}/signatures/{id}         - Supprimer signature
GET    /email/domains/{d}/signatures/{id}/preview - Apercu signature
