STRUCTURE PAGE
1. Left panel avec liste domaines
2. Right panel header (domaine)
3. NAV3: 8 tabs (Securite actif)
4. NAV4: Antispam | Signature | MFA (actif)
5. Banniere info Exchange only
6. Card politique globale (toggle forcer MFA)
7. Table statut MFA par compte Exchange
8. Stats globales (pourcentage, alertes)

DISPONIBILITE MFA
| Offre     | MFA disponible | Methodes                      |
|-----------|----------------|-------------------------------|
| MX Plan   | Non            | -                             |
| Email Pro | Non            | -                             |
| Exchange  | Oui            | App, SMS, Email backup        |
| Zimbra    | Non            | -                             |

METHODES MFA
| Methode           | Description                           |
|-------------------|---------------------------------------|
| App Authenticator | Google Auth, Microsoft Auth, etc.     |
| SMS               | Code par SMS sur telephone            |
| Email backup      | Code sur email secondaire             |

STATUTS MFA
| Statut    | Couleur   | Description                     |
|-----------|-----------|----------------------------------|
| Oui       | #DEF7EC   | MFA active et configuree        |
| Non       | #FEF3C7   | MFA non activee (risque)        |
| En attente| #DBEAFE   | Invitation envoyee              |

ACTIONS PAR COMPTE
| Action             | Comportement                               |
|--------------------|--------------------------------------------|
| Reinitialiser      | Supprime MFA, utilisateur doit reconfigurer|
| Desactiver         | Desactive MFA pour ce compte               |
| Envoyer invitation | Email d'activation MFA a l'utilisateur     |

API ENDPOINTS
GET  /email/domains/{domain}/mfa/stats            -> Statistiques MFA
GET  /email/domains/{domain}/mfa/policy           -> Politique globale
PUT  /email/domains/{domain}/mfa/policy           -> Modifier politique
GET  /email/domains/{domain}/mfa/accounts         -> Liste comptes MFA
POST /email/domains/{domain}/mfa/accounts/{id}/reset -> Reinitialiser
POST /email/domains/{domain}/mfa/accounts/{id}/disable -> Desactiver
POST /email/domains/{domain}/mfa/accounts/{id}/invite -> Envoyer invitation
