TYPES DE REGLES
| Type         | Badge    | Action possible        |
|--------------|----------|------------------------|
| Whitelist    | Vert     | Toujours accepter      |
| Blacklist    | Rouge    | Bloquer                |
| Filtre sujet | Orange   | Quarantaine            |

API ENDPOINTS
GET    /email/domains/{d}/antispam/rules       - Liste regles
POST   /email/domains/{d}/antispam/rules       - Ajouter regle
PUT    /email/domains/{d}/antispam/rules/{id}  - Modifier regle
DELETE /email/domains/{d}/antispam/rules/{id}  - Supprimer regle
GET    /email/domains/{d}/antispam/quarantine  - Messages en quarantaine
POST   /email/domains/{d}/antispam/quarantine/{id}/release - Liberer message
