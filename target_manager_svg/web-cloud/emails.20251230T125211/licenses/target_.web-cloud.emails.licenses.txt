STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Emails actif)
3. LEFT PANEL - Vue licence
   - Toggle: [Par domaine] / [Par licence (actif)]
   - Section PACKS avec barres utilisation
   - Section A LA CARTE
   - Boutons Commander pack / Ajouter licence
4. RIGHT PANEL
   - Header: Nom pack + stats + Gerer
   - NAV3: 8 tabs (Licences actif)
   - NAV4: [Mes packs] | A la carte | Historique
   - Details pack (offre, scope, expiration, prix)
   - Table comptes utilisant ce pack
   - Recapitulatif mensuel

TYPES LICENCES
| Type       | Description                          |
|------------|--------------------------------------|
| Pack       | Lot de licences (remise volume)      |
| A la carte | Licences individuelles               |
| Inclus     | MX Plan gratuit avec domaine         |

SCOPE PACKS
| Scope        | Description                         |
|--------------|-------------------------------------|
| Multi-domaine| Utilisable sur tous les domaines    |
| Mono-domaine | Reserve a un domaine specifique     |

API ENDPOINTS
GET    /email/licenses                      - Vue inventaire complet
GET    /email/licenses/packs                - Liste packs
GET    /email/licenses/packs/{id}           - Details pack
GET    /email/licenses/packs/{id}/accounts  - Comptes du pack
POST   /email/licenses/packs                - Commander pack
PUT    /email/licenses/packs/{id}           - Modifier pack (upgrade)
DELETE /email/licenses/packs/{id}           - Resilier pack
GET    /email/licenses/alacarte             - Licences a la carte
POST   /email/licenses/alacarte             - Ajouter licence
DELETE /email/licenses/alacarte/{id}        - Supprimer licence
