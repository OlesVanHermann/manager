STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Web Cloud (Emails actif)
3. LEFT PANEL
   - Toggle vue: [Par domaine] / [Par licence]
   - Recherche domaine
   - Liste domaines avec resume offres
   - Bouton + Ajouter domaine
4. RIGHT PANEL
   - Header: Domaine + stats + Webmail
   - NAV3: 8 tabs (Boites mail actif)
   - Toolbar: Creer, Importer, Exporter, Filtrer
   - Filtres par offre (Exchange, Pro, Zimbra, MX Plan)
   - Table comptes email
   - Pagination

INTERACTIONS
- Toggle vue -> Bascule domaine/licence dans left panel
- Clic domaine -> Selection + mise a jour right panel
- Clic tab NAV3 -> Navigation vers section
- Clic Creer boite -> Modal creation avec choix licence
- Clic filtre offre -> Filtrage table
- Clic ... actions -> Menu contextuel

API ENDPOINTS
GET    /email/domains                     - Liste domaines
GET    /email/domains/{domain}            - Details domaine
GET    /email/domains/{domain}/summary    - Resume (counts, quotas)
GET    /email/domains/{domain}/accounts   - Liste comptes
POST   /email/domains/{domain}/accounts   - Creer compte
GET    /email/licenses/packs              - Liste packs
GET    /email/licenses/alacarte           - Licences individuelles
