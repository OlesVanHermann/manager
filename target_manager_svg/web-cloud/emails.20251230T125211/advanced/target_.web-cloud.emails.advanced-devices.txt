DISPONIBILITE
EXCHANGE UNIQUEMENT - La gestion des appareils ActiveSync
permet de controler les mobiles et clients lourds connectes.

STATUTS APPAREILS
| Statut   | Badge   | Description                      |
|----------|---------|----------------------------------|
| Actif    | Vert    | Synchro recente (< 7 jours)      |
| Inactif  | Gris    | Pas de synchro recente           |
| Bloque   | Rouge   | Appareil bloque par l'admin      |

ACTIONS DISPONIBLES
| Action     | Description                              |
|------------|------------------------------------------|
| Effacer    | Remote wipe - efface donnees Exchange    |
| Bloquer    | Empeche la synchronisation               |
| Debloquer  | Retablit la synchronisation              |

API ENDPOINTS
GET    /email/domains/{domain}/devices           - Liste appareils
GET    /email/domains/{d}/devices/{id}           - Details appareil
POST   /email/domains/{d}/devices/{id}/wipe      - Remote wipe
POST   /email/domains/{d}/devices/{id}/block     - Bloquer appareil
DELETE /email/domains/{d}/devices/{id}/block     - Debloquer appareil
