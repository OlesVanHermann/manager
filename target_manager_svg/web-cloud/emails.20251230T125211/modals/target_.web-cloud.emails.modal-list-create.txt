STRUCTURE MODAL
1. Header: Creer une liste de diffusion + bouton fermer
2. Formulaire:
   - Adresse de la liste (local + domaine)
   - Nom de la liste (optionnel)
   - Proprietaire/Moderateur
   - Type de liste (Ouverte/Moderee/Privee)
   - Membres initiaux (textarea)
   - Option: Masquer membres
3. Footer: Annuler + Creer

TYPES DE LISTES
| Type     | Description                              |
|----------|------------------------------------------|
| Ouverte  | Tout le monde peut envoyer a la liste    |
| Moderee  | Messages approuves par moderateur        |
| Privee   | Seuls les membres peuvent envoyer        |

OPTIONS AVANCEES
| Option           | Description                          |
|------------------|--------------------------------------|
| Masquer membres  | BCC pour tous les destinataires      |
| Archivage        | Conserver copie des messages         |
| Reponse a        | Reponse a l'expediteur ou a la liste |

API ENDPOINTS
POST   /email/domains/{domain}/lists         - Creer liste
GET    /email/domains/{domain}/accounts     - Comptes pour moderateur
