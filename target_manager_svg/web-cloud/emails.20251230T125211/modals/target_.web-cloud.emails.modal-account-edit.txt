STRUCTURE MODAL
1. Header: Modifier le compte + bouton fermer
2. Info compte (avatar, email, nom, type)
3. Formulaire:
   - Nom complet (prenom + nom)
   - Quota de stockage
   - Alias avec possibilite ajout
   - Delegations d'acces
   - Actions rapides (mdp, repondeur)
4. Footer: Annuler + Enregistrer

DELEGATIONS
| Type          | Permissions                          |
|---------------|--------------------------------------|
| Lecture seule | Consulter emails sans modifier       |
| Lecture/envoi | Consulter et envoyer au nom de       |
| Acces complet | Gestion complete du compte           |

API ENDPOINTS
GET    /email/accounts/{email}              - Details compte
PUT    /email/accounts/{email}              - Modifier compte
GET    /email/accounts/{email}/aliases      - Liste alias
POST   /email/accounts/{email}/aliases      - Ajouter alias
DELETE /email/accounts/{email}/aliases/{a}  - Supprimer alias
GET    /email/accounts/{email}/delegations  - Liste delegations
POST   /email/accounts/{email}/delegations  - Ajouter delegation
