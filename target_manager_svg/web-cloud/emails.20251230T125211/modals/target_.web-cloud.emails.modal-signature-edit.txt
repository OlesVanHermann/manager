STRUCTURE MODAL
1. Header: Editer la signature + bouton fermer
2. Formulaire:
   - Nom de la signature
   - Appliquer a (selection utilisateurs)
   - Editeur HTML avec toolbar
   - Variables disponibles
   - Toggle activation
   - Info apercu
3. Footer: Annuler + Apercu + Enregistrer

EDITEUR HTML
| Fonction   | Description                    |
|------------|--------------------------------|
| B/I/U      | Gras, Italique, Souligne       |
| Lien       | Inserer lien hypertexte        |
| Image      | Inserer image (URL externe)    |
| Couleur    | Couleur du texte               |
| Taille     | Taille de police               |

VARIABLES
| Variable   | Remplacee par                   |
|------------|--------------------------------|
| {prenom}   | Prenom de l'utilisateur        |
| {nom}      | Nom de l'utilisateur           |
| {email}    | Adresse email                  |
| {tel}      | Telephone (si renseigne)       |
| {poste}    | Intitule de poste              |

API ENDPOINTS
GET    /email/signatures/{id}              - Details signature
PUT    /email/signatures/{id}              - Modifier signature
GET    /email/signatures/{id}/preview      - Apercu HTML
