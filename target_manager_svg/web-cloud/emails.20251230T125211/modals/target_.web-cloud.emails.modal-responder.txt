STRUCTURE MODAL
1. Header: Configurer le repondeur automatique + bouton fermer
2. Info compte (avatar + email)
3. Formulaire:
   - Toggle activation
   - Periode (dates debut/fin optionnelles)
   - Sujet de la reponse
   - Message de reponse (textarea)
   - Options (une fois par expediteur, copie, listes)
4. Footer: Annuler + Enregistrer

OPTIONS REPONDEUR
| Option              | Description                              |
|---------------------|------------------------------------------|
| Une fois/expediteur | Evite spam de reponses                   |
| Conserver copie     | Garde les messages recus                 |
| Inclure listes      | Repond aussi aux listes de diffusion     |

PERIODE
- Si vide: activation permanente
- Dates optionnelles pour vacances

API ENDPOINTS
GET    /email/responders/{email}            - Config actuelle
PUT    /email/responders/{email}            - Modifier config
DELETE /email/responders/{email}            - Desactiver
