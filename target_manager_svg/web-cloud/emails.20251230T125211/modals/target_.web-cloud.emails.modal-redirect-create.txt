STRUCTURE MODAL
1. Header: Creer une redirection + bouton fermer
2. Formulaire:
   - Adresse source (local + domaine)
   - Adresse de destination
   - Option: Conserver copie locale (checkbox)
   - Info activation immediate
3. Footer: Annuler + Creer

TYPES REDIRECTIONS
| Type            | Description                          |
|-----------------|--------------------------------------|
| Simple          | Redirige vers 1 adresse              |
| Multiple        | Redirige vers plusieurs adresses     |
| Avec copie      | Garde copie + redirige               |
| Sans copie      | Redirige uniquement                  |

API ENDPOINTS
