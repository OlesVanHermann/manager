STRUCTURE MODAL
1. Header: Ajouter un domaine + bouton fermer
2. Formulaire:
   - Nom de domaine
   - Source licences (pack existant/nouveau)
   - Selection pack (si pack existant)
   - Info configuration DNS a venir
3. Footer: Annuler + Ajouter

PREREQUIS DOMAINE
- Domaine enregistre chez OVH
- Acces a la zone DNS
- Pack avec licences disponibles

WORKFLOW AJOUT
1. Saisie domaine
2. Selection source licences
3. Ajout domaine
4. Assistant configuration DNS
5. Validation MX/SPF/DKIM

API ENDPOINTS
POST   /email/domains                       - Ajouter domaine
GET    /domains                            - Liste domaines OVH
GET    /email/licenses/packs               - Packs disponibles
