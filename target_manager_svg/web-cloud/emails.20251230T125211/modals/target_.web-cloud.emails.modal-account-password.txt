STRUCTURE MODAL
1. Header: Changer le mot de passe + bouton fermer
2. Info compte (avatar + email)
3. Formulaire:
   - Nouveau mot de passe (avec afficher/generer)
   - Indicateur de force
   - Confirmation mot de passe
   - Warning deconnexion appareils
   - Option envoi par email
4. Footer: Annuler + Changer

INDICATEUR FORCE
| Niveau     | Couleur | Barres |
|------------|---------|--------|
| Tres faible| Rouge   | 1/4    |
| Faible     | Orange  | 2/4    |
| Bonne      | Vert    | 3/4    |
| Excellente | Vert    | 4/4    |

REGLES MOT DE PASSE
- Minimum 12 caracteres
- Au moins 1 majuscule
- Au moins 1 chiffre
- Au moins 1 caractere special
- Pas de mot de passe precedent

API ENDPOINTS
PUT    /email/accounts/{email}/password     - Changer mot de passe
POST   /email/accounts/{email}/password/generate - Generer aleatoire
POST   /email/accounts/{email}/password/send    - Envoyer par email
