STRUCTURE MODAL
1. Header: Creer une boite mail + bouton fermer
2. Stepper: 3 etapes (Informations > Licence > Confirmation)
3. Formulaire etape 1:
   - Type d'offre (Exchange/Email Pro/Zimbra)
   - Adresse email (local + domaine)
   - Nom complet (prenom + nom)
   - Mot de passe (avec generateur)
   - Info licence disponible
   - Alias optionnel
4. Footer: Annuler + Suivant

WORKFLOW 3 ETAPES
| Etape | Contenu                                |
|-------|----------------------------------------|
| 1     | Infos compte (email, nom, mdp)         |
| 2     | Selection/achat licence si necessaire  |
| 3     | Resume et confirmation                 |

VALIDATION FORMULAIRE
| Champ          | Regles                              |
|----------------|-------------------------------------|
| Email          | Unique, format valide               |
| Mot de passe   | 12 car min, maj, chiffre, special   |
| Nom            | Optionnel mais recommande           |

API ENDPOINTS
POST   /email/domains/{domain}/accounts    - Creer compte
GET    /email/licenses/available           - Licences dispo
POST   /email/licenses/alacarte           - Si besoin licence
