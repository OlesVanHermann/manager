# Cahier des Charges : dedicated - FTP Backup Storage

## 1. VUE D'ENSEMBLE
- **Type** : Page de gestion des accès FTP Backup
- **Contexte** : Serveur dédié
- **Objectif** : Gérer les accès au stockage de sauvegarde FTP

## 2. DIMENSIONS
| Propriété | Valeur |
|-----------|--------|
| Largeur | 1100px |
| Hauteur | 500px |

## 3. COMPOSANTS UI
- **OdsText** : Titre, informations
- **OdsCard** : Section informations
- **OdsButton** : Ajouter un accès, Mot de passe
- **OdsDatagrid** : Liste des accès
- **OdsActionMenu** : Actions par ligne

## 4. HEADER
| Propriété | Valeur |
|-----------|--------|
| Titre | "Backup Storage FTP" |
| Bouton | "+ Ajouter un accès" (bleu) |

## 5. SECTION INFORMATIONS

### Carte d'informations
| Champ | Exemple |
|-------|---------|
| Espace de stockage | "500 Go utilisés sur 1 To" |
| Serveur FTP | "ftpback-rbx1-001.ovh.net" |
| Login | "ns123456.ip-1-2-3.eu" |

### Bouton mot de passe
| Propriété | Valeur |
|-----------|--------|
| Texte | "Mot de passe" |
| Style | Secondary |
| Action | Affiche/régénère le mot de passe |

## 6. TABLEAU DES ACCÈS

### Colonnes
| Colonne | Description |
|---------|-------------|
| BLOC IP | Adresse IP ou plage CIDR |
| CIFS | Accès CIFS activé (✓/✗) |
| FTP | Accès FTP activé (✓/✗) |
| NFS | Accès NFS activé (✓/✗) |
| ⋮ | Menu d'actions |

### Indicateurs d'état
| État | Couleur |
|------|---------|
| Activé (✓) | Vert (#10B981) |
| Désactivé (✗) | Rouge (#DC2626) |

### ActionMenu (⋮)
| Option | Style |
|--------|-------|
| Modifier | Normal |
| Supprimer | Rouge (danger) |

## 7. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/features/backupFTP - Infos backup FTP
- GET /dedicated/server/{serviceName}/features/backupFTP/access - Liste des accès
- POST /dedicated/server/{serviceName}/features/backupFTP/access - Ajouter accès
- PUT /dedicated/server/{serviceName}/features/backupFTP/access/{ipBlock} - Modifier accès
- DELETE /dedicated/server/{serviceName}/features/backupFTP/access/{ipBlock} - Supprimer accès
- POST /dedicated/server/{serviceName}/features/backupFTP/password - Régénérer mot de passe
