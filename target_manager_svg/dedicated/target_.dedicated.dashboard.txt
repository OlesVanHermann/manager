# Cahier des Charges : dedicated - Dashboard serveur dédié

## 1. VUE D'ENSEMBLE
- **Page** : Informations générales serveur dédié
- **Type** : Dashboard avec onglets
- **Objectif** : Vue globale d'un serveur dédié

## 2. STRUCTURE PAGE
- Header avec nom serveur + badge gamme
- Menu déroulant Actions
- Onglets (Informations, Réseau, IPMI, Reinstall, Backup, Firewall, Interventions)
- 3 cartes (Hardware, Système, Monitoring)

## 3. COMPOSANTS UI
- **OdsBadge** : Gamme (Advance, Eco, Scale)
- **OdsTabs** : Navigation onglets
- **OdsCard** : Cartes informations
- **OdsProgress** : Barres CPU/RAM/Disque
- **OdsSelect** : Menu Actions
