# Cahier des Charges : dedicated - Onglet Backup

## 1. VUE D'ENSEMBLE
- **Page** : Backup serveur dédié
- **Type** : Dashboard backup avec historique
- **Objectif** : Gérer les sauvegardes du serveur

## 2. STRUCTURE PAGE
- 3 cartes (Espace, Accès FTP, Planification)
- Tableau historique des sauvegardes

## 3. COMPOSANTS UI
- **OdsCard** : Cartes statistiques
- **OdsProgress** : Espace utilisé
- **OdsBadge** : Statut (Activé, Réussi)
- **OdsDatagrid** : Historique
- **OdsButton** : Lancer sauvegarde

| Info | Valeur |

|---------|------|-------------|

| Bouton | Style | Action |

| Lancer une sauvegarde | Primary | Backup manuel |

## 9. APIS & ENDPOINTS
- GET /dedicated/server/{serviceName}/backupFtp - Info backup FTP
- POST /dedicated/server/{serviceName}/backupFtp/password - Changer MDP
- GET /dedicated/server/{serviceName}/task?function=backupFtp - Historique
- POST /dedicated/server/{serviceName}/backupFtp/restore - Restaurer
