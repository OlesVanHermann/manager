# STRUCTURE MODAL
1. HEADER - Titre "Ajouter un enregistrement MX" + bouton fermer (×)
2. CONTENT
   - Bannière info explicative (type MX)
   - Sous-domaine + suffixe (généralement @)
   - Priorité (0-65535)
   - Serveur de messagerie (FQDN)
   - TTL
   - Aperçu de l'enregistrement
3. FOOTER - Boutons Annuler / Ajouter

# MODAL TYPE
- Type: Formulaire MX
- Dimensions: 600x660px
- Layout: Formulaire vertical avec aperçu

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Vide = @ |
| Priorité | number | Oui | 0-65535 |
| Serveur mail | text | Oui | FQDN terminé par . |
| TTL | select | Non | Durée de cache |

# PRIORITÉS COURANTES
| Priorité | Usage |
|----------|-------|
| 1-10 | Primaire |
| 20-50 | Secondaire |
| 100+ | Backup |

POST /domain/zone/{zone}/record - Créer enregistrement MX
