# STRUCTURE PAGE
1. NAV1 - Header OVH (Web Cloud actif)
2. NAV2 - Domaines et DNS actif
3. LEFT PANEL - Liste domaines/zones
4. RIGHT PANEL
   - Header: {domainName}
   - NAV3: 10 tabs (Infos générales actif)
   - TILES ROW 1 (3 tuiles):
     - Tile 1: Informations générales (état, extension, registrar, dates)
     - Tile 2: Configuration DNS (serveurs, DNSSEC, Anycast, zone)
     - Tile 3: Sécurité (transfert, Whois, Auth Code, OwO)
   - TILES ROW 2 (2 tuiles):
     - Tile 4: Abonnement (offre, prix, renouvellement)
     - Tile 5: Contacts (4 types de contacts)

# INTERACTIONS
- Clic "⋮" sur chaque tuile → Menu contextuel actions
- Clic "Gérer la zone DNS" → Onglet Zone DNS
- Clic "Afficher" Auth Code → Modal avec code

# API ENDPOINTS
GET /domain/{domain} - Informations domaine
GET /domain/{domain}/nameServer - Serveurs DNS
GET /domain/{domain}/dsRecord - DNSSEC
GET /domain/{domain}/contact - Contacts
GET /domain/{domain}/serviceInfos - Abonnement
