# STRUCTURE MODAL
1. HEADER - Titre "Renouveler le domaine" (fond OVH bleu)
2. CONTENT
   - Informations du domaine (nom, expiration, badge)
   - Bannière warning (si expiration proche)
   - Durées de renouvellement (radios avec prix)
   - Information paiement
   - Total
3. FOOTER - Boutons Annuler / Renouveler

# MODAL TYPE
- Type: Formulaire renouvellement
- Dimensions: 600x640px
- Header: Fond OVH bleu (#000E9C)

# CHAMPS
| Champ | Type | Description |
|-------|------|-------------|
| Durée | radio | 1, 2, 3 ou 5 ans |

# DURÉES DISPONIBLES
| Durée | Prix exemple | Économie |
|-------|--------------|----------|
| 1 an | 9,99 € HT | - |
| 2 ans | 18,99 € HT | - |
| 3 ans | 27,99 € HT | - |
| 5 ans | 44,99 € HT | 10% |

Note: Les prix varient selon l'extension du domaine

# CALCUL NOUVELLE EXPIRATION
- Si pas encore expiré: expiration actuelle + durée
- Si expiré (redemption): date du jour + durée
- Maximum 10 ans d'expiration cumulée

# PAIEMENT
- Utilise le moyen de paiement par défaut
- Paiement immédiat à la confirmation
- Facture générée automatiquement

# ÉTATS DOMAINE
| État | Badge | Action |
|------|-------|--------|
| Actif | Vert | Renouvellement normal |
| Expire bientôt (<90j) | Orange | Warning affiché |
| Expiré | Rouge | Période de grâce |
| Redemption | Rouge | Frais supplémentaires |

# API ENDPOINTS
GET /domain/{domain}/serviceInfos - Infos expiration
GET /order/domain/{domain}/renew - Prix renouvellement
POST /order/domain/{domain}/renew - Commander renouvellement
  Body: { duration: 12 | 24 | 36 | 60 }
