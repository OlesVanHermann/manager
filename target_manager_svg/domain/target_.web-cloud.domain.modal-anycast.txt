# QU'EST-CE QUE DNS ANYCAST ?
DNS Anycast est une technologie qui permet de distribuer les
requêtes DNS vers le serveur le plus proche géographiquement
de l'utilisateur, réduisant ainsi la latence.

# AVANTAGES TECHNIQUES
1. **Latence réduite** - Réponse du serveur le plus proche
2. **Haute disponibilité** - Réseau mondial redondant
3. **Protection DDoS** - Absorption des attaques distribuée
4. **Zero config** - Activation automatique, aucune action requise

# TARIFICATION
- Prix: 1,00 € HT/mois
- Facturation: Mensuelle
- Résiliation: À tout moment
- Engagement: Aucun

# PRÉREQUIS
- Domaine avec DNS hébergés chez OVH
- Zone DNS active
