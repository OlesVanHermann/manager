# STRUCTURE MODAL
1. HEADER - Titre "Verrouillage transfert (Batch)"
2. CONTENT
   - Bannière info (protection transfert)
   - Sélecteur action (Verrouiller/Déverrouiller)
   - Tableau domaines avec statut actuel
   - Bannière résumé opération
3. FOOTER - Boutons Annuler / Appliquer (N)

# MODAL TYPE
- Type: Batch action sur multiple domaines
- Dimensions: 700x620px
- Contexte: Depuis la page AllDom

# ACTIONS DISPONIBLES
| Action | Description |
|--------|-------------|
| Verrouiller | Active le transfer lock |
| Déverrouiller | Désactive le transfer lock |

# TABLEAU DOMAINES
| Colonne | Description |
|---------|-------------|
| ☑️ | Checkbox sélection |
| DOMAINE | Nom du domaine |
| STATUT ACTUEL | Badge Verrouillé/Déverrouillé/Non supporté |

# BADGES STATUT
- Verrouillé: fond vert (#DEF7EC), texte vert foncé (#03543F)
- Déverrouillé: fond ambre (#FEF3C7), texte ambre foncé (#92400E)
- Non supporté: texte gris, checkbox désactivée

# CAS NON SUPPORTÉS
Certains domaines ne supportent pas le transfer lock :
- Domaines avec DNS externes
- Certaines extensions ccTLD (.uk utilise IPS TAG)
- Domaines en période de rédemption

# RÉSUMÉ OPÉRATION
La bannière warning affiche :
- Nombre de domaines qui seront traités
- Nombre de domaines ignorés (avec raison)

# API ENDPOINTS
POST /domain/batch/lock - Verrouiller en batch
  Body: {
    domains: ["example.com", "example.fr", "example.org"],
    action: "lock"
  }

POST /domain/batch/unlock - Déverrouiller en batch
  Body: {
    domains: ["example.eu"],
    action: "unlock"
  }

GET /domain/{domain}/lock - Statut verrouillage
  Response: { locked: true, supported: true }
