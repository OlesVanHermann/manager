# STRUCTURE MODAL
1. HEADER - Titre "Ajouter un enregistrement AAAA" + bouton fermer (×)
2. CONTENT
   - Bannière info explicative (type AAAA)
   - Champ sous-domaine + suffixe domaine
   - Champ adresse IPv6 cible (obligatoire)
   - Sélecteur TTL
   - Aperçu de l'enregistrement (format DNS)
3. FOOTER - Boutons Annuler / Ajouter

# MODAL TYPE
- Type: Formulaire d'ajout
- Dimensions: 600x620px
- Layout: Formulaire vertical avec aperçu

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe (vide = @) |
| IPv6 cible | text | Oui | Adresse IPv6 valide |
| TTL | select | Non | Durée de cache |

# VALIDATION IPv6
- Format complet: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
- Format abrégé: 2001:db8:85a3::8a2e:370:7334
- 8 groupes de 4 caractères hexadécimaux

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement AAAA
POST /domain/zone/{zone}/refresh - Rafraîchir zone après ajout
