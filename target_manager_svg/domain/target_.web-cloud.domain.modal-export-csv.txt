# STRUCTURE MODAL
1. HEADER - Titre "Exporter les domaines en CSV" + bouton fermer (×)
2. CONTENT
   - Texte explicatif
   - Section "Informations du domaine" (4 checkboxes)
   - Section "Options de sécurité" (2 checkboxes avec description)
   - Section "Contacts" (2 checkboxes)
3. FOOTER - Boutons Annuler / Exporter

# MODAL TYPE
- Type: Configuration d'export
- Dimensions: 600x580px
- Layout: Sections avec checkboxes groupées

# COLONNES DISPONIBLES
## Informations du domaine (cochées par défaut)
- Nom du domaine ASCII
- Nom du domaine UTF-8
- Date de création
- Date d'expiration

## Options de sécurité (non cochées par défaut)
- DNSSEC - État de la délégation sécurisée
- DNS Anycast - État d'activation

## Contacts (non cochées par défaut)
- Contact de facturation
- Contact technique

# API ENDPOINTS
POST /domain/export - Générer l'export CSV avec colonnes sélectionnées

# VALIDATION
- Au moins une colonne doit être sélectionnée
- L'export génère un fichier téléchargeable
