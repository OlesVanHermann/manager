# STRUCTURE MODAL
1. HEADER - Titre "Désactiver la protection" (fond OVH bleu)
2. CONTENT
   - Question de confirmation avec nom domaine
   - Bannière info: Accès au code AUTH/INFO
   - Bannière warning: Avertissement DNSSEC (si actif)
3. HELP PANEL - Panneau explicatif à droite
4. FOOTER - Boutons Annuler / Valider

# MODAL TYPE
- Type: Confirmation avec aide contextuelle
- Dimensions: 700x480px
- Header: Fond OVH bleu (#000E9C)
- Layout: Content (65%) + Help panel (35%)

# HELP PANEL SECTIONS
- Protection du domaine (explication)
- Qu'est-ce que le vol ? (risque)
- Comment OVH protège ? (mesure)
- Note: désactiver pour transfert

# BANNIERES
- Info (bleu): Code AUTH/INFO disponible après désactivation
- Warning (jaune): Affiché si DNSSEC actif sur le domaine

# API ENDPOINTS
DELETE /domain/{domain}/lock - Désactiver la protection
GET /domain/{domain}/authInfo - Récupérer le code AUTH après désactivation
