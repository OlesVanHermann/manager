# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - NAPTR" + badge NAPTR (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL
   - Ordre (order)
   - Préférence (preference)
   - Flags
   - Service
   - Expression régulière (regexp)
   - Remplacement (replacement)
   - Texte d'aide
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire NAPTR
- Dimensions: 700x580px
- Header: Fond OVH bleu (#000E9C)
- Badge: Violet pour type NAPTR

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe |
| TTL | number | Non | Durée de cache |
| Ordre | number | Oui | Ordre de traitement |
| Préférence | number | Oui | Préférence |
| Flags | text | Oui | s, a, u, p |
| Service | text | Oui | Ex: SIP+D2U |
| Regexp | text | Non | Expression régulière |
| Replacement | text | Oui | FQDN de remplacement |

# FLAGS NAPTR
| Flag | Description |
|------|-------------|
| s | Lookup SRV |
| a | Lookup A/AAAA |
| u | URI final |
| p | Protocole spécifique |

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement NAPTR
POST /domain/zone/{zone}/refresh - Rafraîchir zone
