# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une redirection" + bouton fermer (×)
2. WIZARD STEPS - 4 étapes (Sous-domaine, Type, Cible, Résumé)
3. CONTENT (Étape 1)
   - Description de l'étape
   - Input sous-domaine + suffixe .example.com
   - Checkbox "Créer aussi www.www.example.com"
   - Info banner (champ vide = domaine principal)
4. HELP PANEL - Types de redirection expliqués
5. FOOTER - Boutons Annuler / Suivant

# TYPES DE REDIRECTION
| Type | Code | Description |
|------|------|-------------|
| Visible | 302 | URL change dans la barre d'adresse |
| Invisible | iframe | URL reste identique, contenu cible affiché |
| Permanente | 301 | Indique migration définitive aux moteurs |

# API ENDPOINTS
POST /domain/zone/{zone}/redirection - Créer redirection
GET /domain/zone/{zone}/redirection - Liste redirections
