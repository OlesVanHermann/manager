2. NAV2 - Domaines et DNS actif

4. RIGHT PANEL
   - Header: {domainName}
   - NAV3: 10 tabs (DynHost actif)
   - NAV4: 2 sous-tabs (Accès/Logins actif)
   - Titre: Gestion des accès DynHost
   - Bouton + Créer un accès
   - Table logins
     - Colonnes: Identifiant, Sous-domaine, Dernière MAJ, Actions
     - Context menu: Modifier, Changer MDP, Supprimer

# INTERACTIONS
- Clic tab "Entrées DynHost" → Liste entrées
- Clic tab "Accès/Logins" → Cette page
- Clic "+ Créer" → Modal création login
- Clic "⋮" → Context menu
- Clic "Modifier" → Modal édition
- Clic "Changer MDP" → Modal changement mot de passe
- Clic "Supprimer" → Modal confirmation

DELETE /domain/zone/{zone}/dynHost/login/{login} - Supprimer
