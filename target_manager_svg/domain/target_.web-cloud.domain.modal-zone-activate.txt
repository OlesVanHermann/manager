# STRUCTURE MODAL
1. HEADER - Titre "Activer la zone DNS" (fond OVH bleu)
2. CONTENT
   - Bannière info (explication)
   - Domaine concerné
   - Liste des enregistrements par défaut
   - Note gratuit
   - Estimation propagation
3. FOOTER - Boutons Annuler / Activer la zone

# MODAL TYPE
- Type: Confirmation activation
- Dimensions: 600x500px
- Header: Fond OVH bleu (#000E9C)

# CONTEXTE
Cette modale s'affiche quand:
- Le domaine n'a pas de zone DNS chez OVH
- L'utilisateur veut gérer le DNS via le Manager
- Domaine transféré ou avec DNS externe

# ENREGISTREMENTS PAR DÉFAUT
| Type | Nom | Valeur |
|------|-----|--------|
| A | @ | IP parking OVH |
| NS | @ | dns1.ovh.net, dns2.ovh.net |
| MX | @ | redirect.ovh.net |
| TXT | @ | v=spf1 include:mx.ovh.com ~all |

# CARACTÉRISTIQUES
- Gratuit (inclus avec le domaine)
- Propagation DNS: 24-48 heures
- Zone modifiable immédiatement après activation
- Possibilité de réinitialiser plus tard

# PRÉREQUIS
- Domaine actif dans le compte
- Pas de zone existante (sinon option "Réinitialiser")
- DNS pointant vers OVH (ou à modifier après)

# API ENDPOINTS
POST /domain/zone - Créer zone DNS pour le domaine
  Body: { zoneName: "exemple.com" }
POST /domain/zone/{zone}/refresh - Rafraîchir après création
