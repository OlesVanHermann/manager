# QU'EST-CE QUE LE TRANSFER LOCK ?
Le verrouillage du transfert (Transfer Lock / Domain Lock) est une
mesure de sécurité qui empêche le transfert non autorisé d'un nom
de domaine vers un autre registrar.

# AVANTAGES
- Protection contre le vol de domaine
- Protection contre les transferts accidentels
- Peut être désactivé à tout moment pour un transfert légitime

# EXTENSIONS SUPPORTANT LE LOCK
- .com, .net, .org, .info
- .fr, .eu, .be, .de
- La plupart des extensions gTLD et ccTLD

# EXTENSIONS NE SUPPORTANT PAS LE LOCK
- .uk (utilise le système IPS TAG)
- Certaines extensions ccTLD exotiques

# API ENDPOINTS
POST /domain/{domain}/lock - Activer le verrouillage
  Response: { status: "locked" }

GET /domain/{domain}/lock - Vérifier état du verrouillage
  Response: { locked: true, since: "2025-01-15" }
