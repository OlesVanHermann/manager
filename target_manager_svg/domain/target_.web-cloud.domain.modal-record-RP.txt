# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - RP" + badge RP (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL
   - Boîte mail du responsable (mbox)
   - Pointeur vers enregistrement TXT (txt-dname)
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire RP (Responsible Person)
- Dimensions: 600x450px
- Header: Fond OVH bleu (#000E9C)
- Badge: Violet pour type RP

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe |
| TTL | number | Non | Durée de cache |
| mbox | text | Oui | Email au format DNS |
| txt-dname | text | Oui | Pointeur vers TXT info |

# FORMAT EMAIL DNS
- @ est remplacé par un point
- admin@exemple.com → admin.exemple.com.
- Le premier point après le nom local devient @

# ENREGISTREMENT TXT ASSOCIÉ
L'enregistrement TXT pointé par txt-dname peut contenir:
- Nom complet du responsable
- Numéro de téléphone
- Autres informations de contact

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement RP
POST /domain/zone/{zone}/refresh - Rafraîchir zone
