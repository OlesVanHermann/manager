# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - DMARC" + badge DMARC (fond OVH bleu)
2. CONTENT
   - Sous-domaine (_dmarc) + suffixe
   - TTL
   - Version (DMARC1 fixe)
   - Politique (p): none/quarantine/reject
   - Pourcentage (pct): 0-100
   - URI de rapport agrégé (rua)
   - Politique sous-domaines (sp)
   - Alignement SPF (aspf): relaxed/strict
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire DMARC
- Dimensions: 700x580px
- Header: Fond OVH bleu (#000E9C)
- Badge: Orange pour type DMARC

# CHAMPS DMARC
| Champ | Tag | Type | Description |
|-------|-----|------|-------------|
| Sous-domaine | - | text | Toujours _dmarc |
| TTL | - | number | Durée de cache |
| Version | v | fixed | DMARC1 |
| Politique | p | select | none/quarantine/reject |
| Pourcentage | pct | number | 0-100 (%) |
| Rapport URI | rua | text | mailto:... |
| Sous-dom policy | sp | select | Optionnel |
| Alignement SPF | aspf | radio | r (relaxed) / s (strict) |

# POLITIQUES DMARC
| Valeur | Description |
|--------|-------------|
| none | Surveillance uniquement |
| quarantine | Marquer comme spam |
| reject | Rejeter le message |

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement TXT (DMARC)
POST /domain/zone/{zone}/refresh - Rafraîchir zone
