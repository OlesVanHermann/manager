# STRUCTURE MODAL
1. HEADER - Titre "Supprimer l'enregistrement DNS" (fond OVH bleu)
2. CONTENT
   - Bannière warning (action irréversible)
   - Question de confirmation
   - Détails de l'enregistrement (type, subdomain, TTL, cible, FQDN)
   - Texte d'avertissement
3. FOOTER - Boutons Annuler / Supprimer (rouge)

# MODAL TYPE
- Type: Confirmation de suppression
- Dimensions: 500x380px
- Header: Fond OVH bleu (#000E9C)
- Bouton action: Rouge danger (#DC2626)

# INFORMATIONS AFFICHÉES
| Champ | Description |
|-------|-------------|
| Type | Badge coloré (A, AAAA, CNAME...) |
| Sous-domaine | Préfixe ou @ |
| TTL | Durée en secondes |
| Cible | IP, FQDN, ou valeur selon type |
| FQDN complet | Nom complet avec le domaine |

# AVERTISSEMENTS
- Action irréversible
- Impact potentiel sur site web et emails
- Propagation DNS 24-48 heures

# API ENDPOINTS
GET /domain/zone/{zone}/record/{id} - Détails enregistrement
DELETE /domain/zone/{zone}/record/{id} - Supprimer enregistrement
POST /domain/zone/{zone}/refresh - Rafraîchir zone après suppression
