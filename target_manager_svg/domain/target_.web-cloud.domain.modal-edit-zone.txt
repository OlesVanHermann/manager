# STRUCTURE MODAL
1. HEADER - Titre "Modifier une entrée DNS" + bouton fermer (×)
2. WIZARD STEPS - Étapes 1.Modification / 2.Résumé
3. CONTENT
   - Type d'enregistrement (lecture seule)
   - Champ sous-domaine + suffixe domaine
   - Champ TTL (secondes)
   - Champ cible (selon type)
   - Bannière info avec valeur actuelle
4. HELP PANEL - Panneau aide contextuelle à droite
5. FOOTER - Boutons Annuler / Suivant

# MODAL TYPE
- Type: Wizard formulaire en 2 étapes
- Dimensions: 900x700px
- Layout: Form (60%) + Help panel (40%)

# WIZARD STEPS
- Étape 1: Modification (formulaire)
- Étape 2: Résumé et confirmation

# CHAMPS VARIABLES SELON TYPE
| Type | Champ cible |
|------|-------------|
| A | Adresse IPv4 |
| AAAA | Adresse IPv6 |
| CNAME | Nom d'hôte cible |
| MX | Serveur mail + priorité |
| TXT | Valeur texte |
| NS | Serveur de noms |

# API ENDPOINTS
PUT /domain/zone/{zone}/record/{id} - Modifier enregistrement DNS
POST /domain/zone/{zone}/refresh - Rafraîchir la zone
