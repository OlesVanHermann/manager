# STRUCTURE MODAL
1. HEADER - Titre "Transfert sortant .UK - Tag IPS" (fond OVH bleu)
2. CONTENT
   - Bannière info (explication système IPS)
   - Domaine concerné
   - Bannière warning (action irréversible)
   - Tag IPS du nouveau registrar (input)
   - Tags IPS courants (aide)
   - Tag OVH (info)
3. FOOTER - Boutons Annuler / Modifier le tag

# MODAL TYPE
- Type: Formulaire transfert UK
- Dimensions: 600x540px
- Header: Fond OVH bleu (#000E9C)

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Tag IPS | text | Oui | Tag du nouveau registrar |

# EXTENSIONS CONCERNÉES
Cette modale est spécifique aux extensions UK:
- .uk
- .co.uk
- .org.uk
- .me.uk
- .ltd.uk
- .plc.uk

# SYSTÈME IPS TAG (Nominet)
- Système spécifique au registre Nominet (.UK)
- Chaque registrar a un tag IPS unique
- Le transfert est immédiat une fois le tag modifié
- Pas de code de transfert (AUTH/INFO) pour les .UK

# TAGS IPS COURANTS
| Registrar | Tag IPS |
|-----------|---------|
| OVH | OVH |
| GoDaddy | GODADDY |
| 123-Reg | 123-REG |
| Namecheap | NAMECHEAP |
| Enom | ENOM |
| Gandi | GANDI |

# PROCESSUS DE TRANSFERT UK
1. Demander le tag IPS au nouveau registrar
2. Saisir le tag IPS dans cette modale
3. Confirmer la modification
4. Le domaine est transféré immédiatement
5. Le nouveau registrar facturera le renouvellement

# API ENDPOINTS
PUT /domain/{domain}/ukOutgoingTransfer - Modifier le tag IPS
  Body: { tag: "NOMINET" }
