# WHOIS / OWO
- OWO = OVH OwO (WHOIS Obfuscation)
- Protège les données personnelles du propriétaire
- Remplace les vraies infos par des données OVH
- Conforme RGPD

# CHAMPS PROTÉGEABLES
| Champ | Clé API |
|-------|---------|
| Nom | name |
| Email | email |
| Adresse | address |
| Téléphone | phone |

# API ENDPOINTS
GET /domain/{domain}/owo - État protection WHOIS
  Response: {
    name: true,
    email: true,
    address: true,
    phone: false
  }

PUT /domain/{domain}/owo - Modifier protection
  Body: {
    name: true,
    email: true,
    address: true,
    phone: true
  }
  Response: { success: true }

# NOTES
- Certaines extensions (.fr) n'autorisent pas la protection
- La protection peut prendre jusqu'à 24h pour être effective
- Les infos restent visibles pour les autorités compétentes
