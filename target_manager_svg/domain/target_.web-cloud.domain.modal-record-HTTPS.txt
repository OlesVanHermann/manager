# STRUCTURE MODAL
1. HEADER - Titre "Ajouter une entrée DNS - HTTPS" + badge HTTPS (fond OVH bleu)
2. CONTENT
   - Sous-domaine + suffixe
   - TTL
   - Priorité (SvcPriority) obligatoire
   - Cible (TargetName) obligatoire
   - Paramètres de service (SvcParams)
   - Texte d'aide
3. FOOTER - Boutons Annuler / Ajouter l'entrée

# MODAL TYPE
- Type: Formulaire HTTPS/SVCB
- Dimensions: 700x520px
- Header: Fond OVH bleu (#000E9C)
- Badge: Bleu pour type HTTPS

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Non | Préfixe |
| TTL | number | Non | Durée de cache |
| Priorité | number | Oui | 0=alias, 1+=endpoint |
| Cible | text | Oui | . ou FQDN |
| SvcParams | textarea | Non | Paramètres clé=valeur |

# PARAMÈTRES SVCPARAMS
| Paramètre | Description |
|-----------|-------------|
| alpn | Protocoles ALPN (h2, h3) |
| port | Port alternatif |
| ipv4hint | Adresses IPv4 |
| ipv6hint | Adresses IPv6 |
| ech | Encrypted Client Hello |

# API ENDPOINTS
POST /domain/zone/{zone}/record - Créer enregistrement HTTPS
POST /domain/zone/{zone}/refresh - Rafraîchir zone
