3. FOOTER - Boutons Annuler / Ajouter

# MODAL TYPE

# CHAMPS
| Champ | Type | Obligatoire | Description |
|-------|------|-------------|-------------|
| Sous-domaine | text | Oui | Préfixe dynamique |
| IP actuelle | text | Oui | IPv4 initiale |
| TTL | number | Non | 60s recommandé |

# FONCTIONNEMENT DYNHOST
- L'IP est mise à jour automatiquement
- Utilise le protocole DynDNS
- TTL court recommandé (60s)
- Nécessite un identifiant d'accès

# TTL RECOMMANDÉ
- 60 secondes pour IP dynamique
- Permet propagation rapide des changements

# API ENDPOINTS
POST /domain/zone/{zone}/dynHost/record - Créer entrée DynHost
GET /domain/zone/{zone}/dynHost/login - Récupérer identifiants
