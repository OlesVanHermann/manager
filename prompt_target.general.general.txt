# PART 1: MASTER DESIGN SYSTEM (NAV1=General)

## OVERVIEW

NAV1 "General" regroupe les fonctions transversales du compte OVHcloud:
- Dashboard global
- Gestion du compte utilisateur
- Facturation et paiements
- Support client
- API et tokens
- Empreinte carbone

## NAVIGATION HIERARCHY

```
NAV1: [General*] Bare Metal | Hosted Private | Public Cloud | Web Cloud | Network | IAM
       ═════════

NAV2: General | Mon compte | Facturation | Support | API | Carbon
      (6 items)
```

NAV3 is ALWAYS displayed even if only one tab (General), for consistency.

# PART 2: NAV2=General (Dashboard)

## ROUTE
/general

## NAVIGATION STATE
```
NAV1: [General*] Bare Metal | Hosted Private | Public Cloud | Web Cloud | Network | IAM
NAV2: [General*] Mon compte | Facturation | Support | API | Carbon
NAV3: (none - dashboard)
NAV4: (none)
SIDECAR: (none - full width)
```

## DESCRIPTION
Dashboard principal affichant une vue d'ensemble du compte:
- Alertes et notifications urgentes
- Grille des services actifs
- Derniere facture
- Suivi des commandes
- Tickets support ouverts
- Actions rapides

## LAYOUT
Full-width (no sidecar), widget-based grid layout.

## COMPONENTS

### 1. Alert Banner (optional)
- Position: top
- Types: warning (unpaid invoice), info (KYC pending), error (service issue)
- Action button: [Regler] [Completer] [Voir]

### 2. Services Grid
- Title: "Mes services (N)"
- Cards: icon + name + count
- Categories: Domaines, Hosting, VPS, Emails, Databases, VoIP, etc.
- Click -> navigate to universe

### 3. Last Invoice Widget
- Reference number
- Amount TTC
- Date
- Actions: [PDF] [Toutes les factures ->]

### 4. Order Tracking Widget
- Order number + product name
- Progress bar with percentage
- Estimated delivery date

### 5. Open Tickets Widget
- List of recent tickets (max 3)
- Ticket ID + subject + date
- Link: [Voir tous les tickets ->]

### 6. Quick Actions
- Buttons: [Factures] [Compte] [Support]
- Primary CTA: [+ Commander un service]

## STATES
- Default: all widgets populated
- Empty: no services -> onboarding CTA
- Alert: unpaid invoice banner visible
- Loading: skeleton placeholders

## ACTIONS
| Action | Target |
|--------|--------|
| Click service card | Navigate to universe |
| Click invoice PDF | Download PDF |
| Click ticket | Open ticket detail |
| Click "Commander" | Open order catalog |
