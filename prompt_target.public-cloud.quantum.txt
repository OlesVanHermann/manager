## 1. VUE D'ENSEMBLE

### 1.1 Positionnement
- **NAV1** : Public Cloud (Header bleu #000E9C, 50px)
- **NAV2** : Quantum (Sous-header blanc, 40px) - Actif parmi [Projet, Compute, Storage, Databases, Containers, AI/ML, Quantum, Network]
- **NAV3** : General (filtres Region + Projet dans sidecar)
- **NAV4** : Tabs contextuels dans RIGHT PANEL (detail notebook)

### 1.2 Objectif
Acces aux ordinateurs quantiques (QPU) en QaaS (Quantum as a Service) :
- Notebooks JupyterLab avec acces QPU
- Partenaires : Quandela (photonique), IBM Quantum (supraconducteur)
- Frameworks : Perceval, Qiskit, Cirq
- Facturation a la seconde pour le temps QPU

### 1.3 URL Pattern
```
/public-cloud/quantum                           -> Dashboard (Tous les services)
/public-cloud/quantum/create                    -> Wizard creation notebook
/public-cloud/quantum/{notebookId}              -> Detail notebook
/public-cloud/quantum/{notebookId}/logs         -> Logs notebook
/public-cloud/quantum/{notebookId}/metrics      -> Metriques notebook
/public-cloud/quantum/{notebookId}/settings     -> Parametres notebook
```

## 4. TYPES DE RESSOURCES QUANTUM

### 4.1 Notebooks Quantum
- **Description** : Environnement JupyterLab avec acces QPU
- **QPU disponibles** :
  - **Quandela Ascella** : 12 qubits photoniques, 0.012E/s
  - **IBM Quantum** : 127 qubits supraconducteurs, 0.025E/s
- **Frameworks** :
  - **Perceval** : Framework photonique Quandela (v0.10)
  - **Qiskit** : Framework IBM (v0.45)
  - **Cirq** : Framework Google (v1.2)
- **Etats** : RUNNING, STARTING, STOPPED, ERROR
- **Actions** : Start, Stop, Ouvrir JupyterLab, Supprimer

### 4.2 Ressources Notebook
- **cpu-small** : 2 vCPU, 4 GB RAM
- **cpu-medium** : 4 vCPU, 16 GB RAM
- **cpu-large** : 8 vCPU, 32 GB RAM

## 8. COMPORTEMENTS

### 8.1 Filtrage NAV3
- Changement Region -> Filtre liste notebooks
- Changement Projet -> Filtre liste notebooks
- Les deux filtres sont combines (AND)

### 8.2 Navigation sidecar -> RIGHT PANEL
- Clic "Tous les services" -> Dashboard notebooks
- Clic notebook -> NAV4 = [General] Logs Metrics Settings

### 8.3 Acces JupyterLab
- Bouton "Ouvrir JupyterLab" -> Nouvel onglet
- URL unique par notebook avec token temporaire

### 8.4 Facturation
- Ressources (notebook) : Facturation horaire
- Temps QPU : Facturation a la seconde
- Affichage temps cumule + cout en temps reel

---

## 9. MODALS

### 9.1 Modal Supprimer Notebook
- **Titre** : Supprimer le notebook Quantum
- **Champs** : Aucun
- **Warning** : Action irreversible, donnees et configurations supprimees
- **Boutons** : [Annuler] [Supprimer]

### 9.2 Modal Start Notebook
- **Titre** : Demarrer le notebook
- **Info** : La facturation reprendra
- **Boutons** : [Annuler] [Demarrer]

### 9.3 Modal Stop Notebook
- **Titre** : Arreter le notebook
- **Warning** : Circuits en cours seront interrompus
- **Info** : Facturation ressources s'arrete, donnees conservees
- **Boutons** : [Annuler] [Arreter]
