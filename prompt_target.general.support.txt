# PROMPT_TARGET.GENERAL.SUPPORT.TXT
# NAV1=General > NAV2=Support

## ROUTE
/general/support

## NAVIGATION STATE
```
NAV1: [General*] Bare Metal | Hosted Private | Public Cloud | Web Cloud | Network | IAM
NAV2: General | Mon compte | Facturation | [Support*] API | Carbon
NAV3: [General*] Niveau Support
NAV4: [Echange*] Chat Max | Contact Urgent
SIDECAR: YES (ticket list when NAV3=General)
```

## DESCRIPTION
Gestion des tickets support: création, suivi, échanges, options de contact
selon le niveau de support souscrit.

## LAYOUT
WITH SIDECAR (280px) when NAV3=General: displays ticket list.
NO SIDECAR when NAV3=Niveau Support.

## NAV3 TABS (2)

### 1. General (default)
- Ticket management
- SIDECAR: List of tickets with search
- RIGHT PANEL: Selected ticket detail
- NAV4: Echange | Chat Max | Contact Urgent

### 2. Niveau Support
- Support level information
- Compare and upgrade options
- NO SIDECAR
- NO NAV4

## NAV4 TABS (3) - only when NAV3=General

### 1. Echange (default)
- Ticket conversation history
- Reply form with attachments
- Ticket info (status, service, created, etc.)

### 2. Chat Max
- Live chat interface (if available with support level)
- Chat history
- Availability status

### 3. Contact Urgent
- Emergency contact options
- Phone callback request
- Urgent ticket creation

## SIDECAR CONTENT (NAV3=General)
- Title: "Support"
- Search: ticket search
- Button: [+ Créer un ticket]
- Ticket list: ID, subject, status, date
- Filters: status dropdown

## COMPONENTS

### Ticket Card (sidecar)
- Ticket ID
- Subject (truncated)
- Status badge
- Date

### Ticket Detail
- Full subject
- Status, created date, service
- Actions: [Fermer] [Relancer]
- Conversation thread
- Reply form

### Message Bubble
- Sender (You / Support OVH)
- Timestamp
- Message content
- Attachments

### Support Level Card
- Level name (Standard/Premium/Business)
- Features list
- Price
- Action button

## STATES
- Default: ticket list loaded
- Empty: no tickets
- Selected: ticket detail visible
- Creating: new ticket form
- Closed: ticket marked resolved

## MODALS
- Create ticket
- Close ticket confirmation
- Escalate ticket
- Rate support interaction
