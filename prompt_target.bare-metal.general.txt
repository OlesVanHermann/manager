| Element | Valeur |

#### Tableau services recents
| Colonne | Largeur | Type | Description |
|---------|---------|------|-------------|
| Type | 120px | Badge | Icone + label (NAS-HA, Ceph, Serveur, VPS) |
| Nom | flex | Link | Nom du service (lien cliquable) |
| Statut | 100px | Badge | Actif (vert), Inactif (gris), Erreur (rouge) |
| Region | 80px | Text | Code region (GRA, SBG, BHS) |
| Derniere activite | 180px | Text | Temps relatif (Il y a X heures) |

#### Regles
- Maximum 5 lignes affichees
- Tri par derniere activite (plus recent en premier)
- Tous types de services confondus (cross-NAV2)
- Click ligne = navigation vers service

### 3.4 Section Alertes

Layout: Liste cards alertes

#### Condition affichage
- Visible uniquement si alertes actives > 0
- Masquee completement si aucune alerte

#### Card alerte
| Element | Description |
|---------|-------------|
| Icone gauche | Warning triangle (orange) ou Error circle (rouge) |
